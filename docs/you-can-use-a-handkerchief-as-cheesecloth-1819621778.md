# 你可以用手帕当粗棉布

> 原文:[https://life hacker . com/you-can-use-a-parp-as-cheese cloth-1819621778](https://lifehacker.com/you-can-use-a-handkerchief-as-cheesecloth-1819621778)

粗棉布是我手头从来没有的东西之一，每次我想做意大利乳清干酪的时候都得重新买。虽然从技术上讲，你可以清洗并重复使用它，但我总是觉得这项任务有点令人沮丧。(当我手洗的时候，它没有完全洗干净，但是在洗衣房里就变得很脏了。)

幸运的是，如果找不到奶酪专用布，Kitchn 有三种奶酪布可供选择——其中之一是 [一块大手帕](https://www.amazon.com/Pack-Hankies-Handkerchief-Umo-Lorenzo/dp/B00188UDKU/ref=as_at/?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/you-can-use-a-handkerchief-as-cheesecloth-1819621778&asc_source=&creativeASIN=B00188UDKU&imprToken=RRWoV103di5V0N9N5WCSbg&linkCode=w61&slotNum=4&tag=kinjalifehackerlink-20) 。我喜欢这个主意，因为它们是预先切好的，非常容易洗，而且相当便宜。它们也有很好的编织，这使得它们非常适合一些软奶酪。当然，你必须确保你有一个指定的“厨房手帕”存放点，这样你就不会把它们和鼻子上用的手帕搞混了。(我只见过老人用手帕擤鼻涕，所以也许在你家这种区别是不必要的。)
