# 这张地图显示了美国每个县的平均通勤时间

> 原文：<https://lifehacker.com/this-map-shows-the-average-commute-time-in-every-u-s-c-1796559696>

地理统计网站 Overflow Data 已将人口普查数据编译成全美平均通勤时间的 [交互式地图](http://overflow.solutions/demographic-data/what-is-the-average-commute-time-in-each-u-s-county/) 放大一个州，或查看每个州内的范围。东海岸的通勤状况最糟糕；洛杉矶的交通仍然不像纽约地铁那么糟糕。最糟糕的通勤是在宾夕法尼亚州的派克县，离纽约有三个小时的车程。



溢出数据还有[性别比](http://overflow.solutions/demographic-data/what-is-the-the-sex-ratio-of-each-state-and-county-in-the-u-s/)[高校招生](http://overflow.solutions/demographic-data/what-percentage-of-the-us-population-is-enrolled-in-college/) 的分县图。
