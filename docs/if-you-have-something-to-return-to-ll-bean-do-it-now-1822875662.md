# 如果你有东西要还给 LL Bean，现在就做

> 原文：<https://lifehacker.com/if-you-have-something-to-return-to-ll-bean-do-it-now-1822875662>

去年在 LL Bean 买过东西吗？不完全满意吗？现在把它退回去。这家拥有百年历史的户外型商品目录零售商改变了其著名的慷慨退货政策:以前，你可以随时退货，不需要收据；现在你必须保留你的收据，并在一年内取回你的物品 。



正如 [NPR 报道](https://www.npr.org/sections/thetwo-way/2018/02/09/584493046/l-l-bean-scraps-legendary-lifetime-return-policy) ， [公司将变化归咎于利用终身保修的顾客](https://www.facebook.com/llbean/posts/10155636619902415) 。该公司写道:

“越来越多的少数但越来越多的客户对我们的保证的理解远远超出了其初衷。一些人认为这是一个终身产品更换计划，期望对使用多年的严重磨损产品进行退款。其他人则要求对通过第三方购买的产品进行退款，比如在庭院销售时。'

《波士顿环球报》[](https://www.bostonglobe.com/business/2018/02/09/bean-dropping-its-unlimited-returns-policy/LEc2iLzayaJxxFDkz0or8K/story.html?s_campaign=bostonglobe%3Asocialflow%3Atwitter)*总是刊登调查性报道，提供旧货拍卖行骗者的细节，还包括被退回的粗糙破旧拖鞋等物品的照片。该公司提供了其他回报的照片，这些照片肯定属于“有些过分”的胡说八道类别，比如一件挂满三年缆车通行证的儿童滑雪衣。(个人旁注:我每年都在 LL Bean 买小孩的外套。为什么我这么笨，从没想过把它们退回去换大一号的？这种可疑程度会让你怀疑，如果你只是一个多一点的骗子，你还会错过哪些黄金机会。)*

*可以预见的是，顾客们愤怒了，并把他们的生意带到了别处。但是，嗯，在哪里？对于喜欢户外活动的人来说，REI 一直是最受欢迎的，它的返回政策是 [与 LL Bean 的](https://www.rei.com/help/return-policy.html) 差不多。如果你想多花一点钱，巴塔哥尼亚似乎有 [慷慨的退货政策](http://www.patagonia.com/returns.html) :“如果你的东西不适合你，送去退货或修理，任何时候都没有麻烦，”这意味着...一辈子？但是我想这取决于那些大胆的人去测试“随时”的极限。把你恶心的旧拖鞋送回去，看看会发生什么。因为就像一个推特用户说的，LL Bean 受够了你的狗屎。*
