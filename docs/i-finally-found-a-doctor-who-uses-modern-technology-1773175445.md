# 我终于找到了一位使用现代技术的医生

> 原文：<https://lifehacker.com/i-finally-found-a-doctor-who-uses-modern-technology-1773175445>

当大多数人 [找医生](https://lifehacker.com/how-can-i-find-a-good-doctor-5959213) 时，他们会试图找一个由他们的保险覆盖的人，靠近他们的家或工作单位，也许是别人推荐的。我也在寻找这些东西，但是我真正想要的——我发现很难找到——是一个使用 21 世纪工具的医生，比如电子邮件。

那天，我丈夫的新主治医生通过电子邮件向他发送了首诊总结报告，并提出可以通过电子邮件进行讨论，这让我大吃一惊。医生使用电子邮件，实际上让他们的病人通过电子邮件向他们提问？当时，我的初级保健医师办公室甚至没有网站，更不用说发电子邮件或短信了。

现在，当我找医生时，无论是初级保健、牙医、儿科医生还是其他专家，我都坚持要找到一个使用现代技术的医生，如在线预约和支付、数字 x 光、在线访问健康记录和最新设备。

这不仅仅是方便，虽然这是一个很大的因素。现代医学技术也是关于舒适和减压的。例如，在牙医那里做 x 光检查，对我来说不仅仅是不舒服，而是彻头彻尾的痛苦。(很明显，我有一张小嘴和高度敏感的牙龈——这对咬下又大又尖的塑料东西来说是一个糟糕的组合。)我换了无数次牙医诊所，只是为了寻找一个同情这种痛苦的人，而不是简单地将我视为一个“棘手”的病人。通过查看牙医的网站和他们的照片库，我找到了一个做 [全景 x 光](http://www.dentistryiq.com/articles/2012/02/6-reasons-why-you-should-use-a-panoramic-x-ray-for-bitewings.html) 的办公室:当 x 光机在你周围旋转时，你只需要站着不动，没有任何物体切入你的牙龈。简直神奇。

同样，我需要一次核磁共振来弄清楚为什么我会突然偏头痛。由于对核磁共振成像一无所知，我只是去了医生推荐的最近的地方。他们使用了传统的机器——那种让你感觉自己被困在嘈杂的金属棺材里的机器。如果我做了调查，我就会知道有更新的替代方案，如 [开放式核磁共振成像和](http://www.mycdi.com/viewpoints/how_to_get_through_an_mri_if_youre_claustrophobic_29) 立式核磁共振成像，这样我就不会感到不舒服了。

现代科技也能让治疗更安全。拉斯蒂·霍夫曼博士[在巡诊](https://www.grandrounds.com/state-art-medical-treatment-demand/)上写道:

> 虽然没有行业范围内对最先进护理的定义，但它可以被宽泛地定义为创新、前沿，并且通常超越传统治疗的“规范”。最先进的护理可以应用于技术，实践和医生。
> 
> 例如，我的一个同事是世界上唯一一个可以通过激光移除嵌入式下腔静脉过滤器(插入心脏的医疗设备)的人。移除过滤器的替代方法既棘手又危险，因此，来自世界各地的人们都在寻求他的建议。他正在提供最先进的护理。

找一个好医生往往是徒劳的，所以找一个拥有最先进技术的医生更难。通常，我会通过查看他们的网站、查看他们的办公室照片以及提及他们使用的技术来搜索最新的医生，但这可能会有所欺骗。我去过牙医那里，他们承诺使用“无痛激光治疗”来安慰他们的病人，但发现他们仍然使用旧的折磨人的刮削工具。我曾尝试在 [健康等级](http://www.healthgrades.com/) 和 [RateMDs](http://www.ratemds.com/) 等网站上通读医生点评，但很少有人提及医生使用的设备类型。

我的父母都是退休医生，他们和我分享了寻找顶尖医生的秘诀:寻找从事前沿研究的医生。这就是他们如何为我找到一位激光眼科医生(这位医生是许多涉及我特殊眼病的国家试验的首席研究员)以及他们如何决定为我父亲做心脏手术的医生。您可以使用像[【ClinicalTrials.gov】](https://clinicaltrials.gov/)或这样的网站来查找针对您的医疗保健诊断或需求的资源。

然而，我发现的最简单的解决方法是去大型的、有声望的医院和他们的医疗网络。属于这些网络的办公室更有可能接受数字和新技术，因为它们拥有这样做的资源。正如霍夫曼博士所说:

> 如果你得到一个令人生畏的诊断，那就去那些更大更有声望的医院看看，那里已经有了最先进的医疗服务。例如，一些规模较大、声誉较好的医院能够负担得起最新、最棒的技术，比如先进的核磁共振仪——通常比附近的医院领先几年。事实上，学术医院经常与医疗设备公司合作，帮助共同开发新技术。

最近，我需要做 x 光检查，去了一个我以前从未去过的机构。我所要做的就是在平板电脑上登录，因为多亏了 NYU 庞大的医疗网络，我的记录和医生的处方已经在系统里了。他们甚至拿走了我的数字掌纹和一张我的照片，以便在未来证明我的身份(这确实有点老大哥的感觉，但我也觉得我生活在未来)。当去看医生已经是一件麻烦事的时候，这可以节省时间，减少一件麻烦事。第二天，我能够登录医疗中心的网站，阅读我的医生的信息。

现代医疗技术和工具并不一定意味着更好的护理，所以你的首要任务是找到一个你信任的、让你感到舒服的医生，这个医生对你的需求敏感，倾听你的意见，并给你实际上让你感觉更好的治疗。但是，请更进一步，找出最适合您特殊健康需求的产品。你可能会找到一个很棒的医生，他的工作也不会停留在 1899 年。
