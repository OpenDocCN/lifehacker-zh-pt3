# 使用这些应用程序为无家可归者提供即时帮助

> 原文：<https://lifehacker.com/use-these-apps-to-get-immediate-help-for-the-homeless-1823001478>

“无家可归者的应用程序”听起来可能有点奇怪。但是廉价的智能手机、外展计划以及青年无家可归意味着许多人有电话却没有家。(截至 2011 年，几乎三分之二的无家可归的年轻人 [拥有一部手机](https://www.ncbi.nlm.nih.gov/pubmed/22076445) 。)在几个城市，他们可以使用移动应用程序来寻找食物、住所和其他资源。



这些应用程序对任何曾经接触过无家可归者并希望提供更多帮助的人也很有用。正如《快公司》的阿黛尔·彼得斯指出的那样， [报警是一种极其冒险的反应](https://www.fastcompany.com/40444599/this-app-lets-you-ask-request-mental-health-help-for-a-homeless-person-in-distress)——她列举了最近几起警察枪杀流浪汉的事件。取而代之的是打电话给像 311 这样的民用服务，或者加载这些应用程序中的一个。

这个名为 [OurCalling](https://www.ourcalling.org/app/) 的应用程序列出了该国任何地方的戒毒、食物资源、庇护所等服务。该应用程序与达拉斯的一个基督教部有关，因此它也将展示该部的本地服务。

为无家可归的年轻人设计的 [街灯芝加哥](https://www.tlpchicago.org/streetlight-app/) 应用程序为收容所、诊所、紧急情况和心理健康服务提供地图和信息。它列出了收容所的营业时间、膳食和卫生服务，以及如何预订床位或探访。它还提供了一个北侧避难所的应用内预订。

在纽约，无家可归的人口在过去十年里翻了一番，纽约市 求助者应用程序列出了医疗保健、住房和法律援助、食物和衣服、就业和教育服务方面的资源。

纽约人也可以使用包罗万象的 311 应用程序 请求照顾无家可归的人。在主页底部，点击“请求”并搜索或滚动到“无家可归者援助”您可以添加位置和其他详细信息，如果需要，还可以提供您的联系信息。

在三藩市的田德隆区，[concorn](http://www.concrn.org/)是一款在不需要紧急服务时请求“富有同情心的响应者”的基本应用程序。响应者通常会打电话给用户询问更多细节，然后过来帮助无家可归者找到合适的服务。用户可以指出请求的紧急程度(选项包括“本周”、“一小时内”和“现在”)，并添加照片和文本。

在三藩市的其他地方， [Link-SF](https://link-sf.com/) 是一个精简的移动网站，列出了食物、住房、卫生、医疗保健以及互联网或计算机接入的资源。该网站是 Zendesk 和非营利组织圣安东尼基金会的合作项目。

除了像 OurCalling 这样的全国性应用程序，您所在的地区可能还有自己的本地应用程序。谷歌一下，查看你所在城市的一般城市服务应用程序，看看它们是否有无家可归者救助功能。

这些应用程序的新闻报道中的一个主题是它们没有得到充分利用，app store 中缺乏评论和评级证实了这一点。(许多应用程序甚至从未通过 [制度阻力和监管](https://www.sfgate.com/news/article/Tech-disconnect-Why-SF-homeless-apps-may-not-12348675.php) 。相反，人们不停地打电话给已经超负荷的城市服务部门，甚至警察。所以现在就下载一个应用，在你需要它之前。有一天它可能会帮助你拯救某人的生命。
