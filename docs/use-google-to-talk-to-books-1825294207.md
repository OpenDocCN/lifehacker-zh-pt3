# 使用谷歌与书籍“对话”

> 原文:[https://life hacker . com/use-Google-to-talk-to-books-1825294207](https://lifehacker.com/use-google-to-talk-to-books-1825294207)

如果你曾经想与印刷页面进行更深入的对话，或者浏览文学图书馆寻找你许多问题的答案，谷歌的 [与书籍](https://books.google.com/talktobooks/) 对话工具是一种有趣的小方法。这不是针对书籍 的 [谷歌搜索*，但它确实为你的问题提供了比传统搜索更多的对话式答案。*](https://books.google.com/)

Watch

例如，如果你打开 Talk to Books，尝试它的许多预生成的查询中的任何一个，比如“谁是哈利波特中最聪明的角色”，你不会得到一个可以用来找到问题答案的书籍列表。相反，你会从书中找到最有可能对你的问题做出自然回答的段落。

在这种情况下，下面是谷歌对其《哈利·波特》查询的最高回复:“在鲁邦和小天狼星布莱克的身份完全暴露的尖叫棚屋那场戏中，是鲁邦承认了赫敏的聪明:‘你是我见过的你这个年龄最聪明的女巫。’"

虽然这个答案并不一定会给你一个“最聪明”人物的总结，但你可以从谷歌的其他(粗体)答案中拼凑出一个像样的列表:

*   “哈利是公认的魔法神童”
*   "在成年人中，麦格教授似乎认为赫敏是一个聪明的女性，显然处于次要地位。"
*   “邓布利多仍然被视为‘无所不知’的领袖，麦格教授仍然多次要求邓布利多的认可，尽管她仍然非常聪明，并且在小说中经常表现出来。”
*   “在海格和赫敏的善良中，在赫敏为自己和他人挺身而出的意愿中，我们看到了勇气。”

除了最后一个看起来有点随机的问题，答案似乎符合哈利和邓布利多非常聪明的普遍看法。这一点，以及事实声明，你永远不要惹玛吉·史密斯。

正如谷歌 [对](https://research.google.com/semanticexperiences/about.html) 的解释:

> “与书对话更像是一种创造性的工具，而不是一种找到具体答案的方式。在这个实验中，我们不考虑这本书是权威的还是切题的。该模型只是查看每个句子与您的查询的匹配程度。有时，它会发现一些不切中要害或完全断章取义的回答。”

如果你决定尝试一下谷歌的 Talk to Books，记住这项服务是围绕语义分析设计的，所以你不应该把它当成普通的谷歌搜索。不要输入你想了解更多信息的关键词，也不要试图使用花哨的搜索操作符，比如"-伏地魔"，来删除所有提到的那个不知名的家伙。相反，用交谈的方式表达你的问题，就好像你要问一个朋友为什么他们认为佛罗多没有把戒指扔进末日火山 。你不会就这么告诉朋友“ [佛罗多](https://books.google.com/talktobooks/query?q=Frodo%20mount%20doom%20ring) ”吧？
