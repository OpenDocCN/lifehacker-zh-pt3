# 提炼家禽下脚料并获得优质烹饪脂肪

> 原文：<https://lifehacker.com/render-poultry-scraps-and-gain-excellent-cooking-fat-1821958410>

我们都知道要保存鸟骨头来做原料，但是你在屠宰一块家禽后发现自己身上多余的皮和脂肪也同样有价值。你只需付出很少的努力，就能做出一些最美味的烹饪油脂。



*这是《与克莱尔一起吃垃圾》的一部分* [*，这是一个生活黑客系列，克莱尔·劳尔说服你将你的厨余转化为可食用的美味。*](http://lifehacker.com/tag/eating-trash-with-claire)

可能需要几只鸡才能攒够渲染用的下脚料，所以只要在冰箱里放一袋，直到你有一磅左右。如果你正在屠宰一只鹅，你马上就会有足够的。有几种不同的方法可以做到这一点，而且都非常简单:

*   **在炉子上:**将脂肪和皮屑切成差不多均匀的小块，扔进一个大的无反应的锅里。用非常低的火煮，每隔五分钟左右搅拌一下，大约 45 分钟。一旦你有了一堆液体脂肪和脆脆的脆皮，把它们过滤到罐子里，储存在冰箱里。吃脆皮。
*   **在慢炖锅里:**将切好的脂肪和皮放入你最喜欢的慢炖锅里，用小火煮，直到脂肪液化，变成金黄色，大约每小时搅拌一次(这可能需要三个小时，也可能需要六个小时，取决于你的炖锅)。过滤到罐子里，储存在冰箱里。
*   **在一个真空装置中:**这是我用 [这个多肉项目](https://skillet.lifehacker.com/you-should-cook-a-sous-vide-goose-this-christmas-1821535961) 的下脚料做的，结果得到了一品脱多的鹅油。将你所有的残渣放入一个真空塑料袋中，密封，然后将袋子浸入 185 度的水中几个小时，直到袋子充满液体脂肪，皮肤看起来已经竭尽所能。这样你不会听到任何爆裂声，但你也不必去搅拌或观察任何东西。根据垃圾袋里的东西，你可能需要使用脂肪分离器才能把你的奖励装进罐子里。

一旦你有了美味的鹅油、鸭油或 schmaltz，用它做任何东西，尤其是烤蔬菜。如果有一种蔬菜比鹅油烤土豆更神圣的话，我还没有遇到过。
