# 定义你今天的最后一次重大生活变化是什么？

> 原文：<https://lifehacker.com/what-was-the-last-big-life-change-that-defines-you-toda-1825930980>

“现在”不是平滑滚动的时间窗口，总是以同样的速度滑向过去。我们时不时会改变一些重大的事情——离开一份工作，搬到一个新家，与伴侣分手——这定义了一个新的过去和一个新的现在。我们并不总是预见到这种变化的影响，或者是否会有任何影响。只有回想起来，我们才意识到我们做了一个伟大或可怕的选择。



我人生中最大的变化是一个典型的变化:那是我和妻子第一次在一起的时候，九年前。我开始用她之前和之后的事情来定义我的生活。这是我与过去许多最糟糕的决定达成妥协的一种方式——有时，不幸的是，也是我原谅它们的一种方式。我试图用“现在的我”和“过去的我”之间的区别作为一种成长的方式，而不是假装我可以完全否认我的过去。

虽然在过去的九年里我还有其他“过去的我”——从未在办公室工作过的我，独自生活的我，知道自己永远不想要孩子的我——但我在这些阶段的进展受我的关系影响最大。随着它的逐渐到来，它成了我做过的最大的承诺，这让我明白我太害怕承诺了，更喜欢“保留我的选择”所以我把它应用到我的事业、我的友谊和我对未来的想法中。这是定义“现在的我”和“过去的我”的最重要的一课——为了快乐，我需要牺牲更多直接的个人自由来支持承诺。

那么什么样的选择和事件标志着你“过去”的结束和“现在”的开始呢？你发现了什么教训或主题——更多自由、更多承诺、更多善良、更少放纵或懒惰？是什么定义了现在的你，你做了什么具体的事情来抓住它？
