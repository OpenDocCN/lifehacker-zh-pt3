# 用铅笔将酱汁冻成可以食用的份量

> 原文：<https://lifehacker.com/use-a-pencil-to-freeze-sauce-into-usable-portions-1814081647>

在冰箱里放满酱汁、汤和冰沙可以让你在繁忙的工作日晚上快速轻松地享用晚餐，但当你只需要几勺液体时，解冻一整袋食物可能会很痛苦。铅笔可以帮助你分配液体，这样你就可以抓住你需要的东西。



如果手边没有铅笔，也可以用筷子或炊具的手柄。你只是想要一个跨越包的长度，宽度窄，能经受住冷冻的东西。这有助于你将冷冻的液体分开，因为铅笔(或其他物品)在液体中形成了一个分割线，这样更容易切下一部分，而不是解冻整个袋子。

因为你要折叠袋子，你不会想把它装满酱汁。一旦你把调味汁装袋，把铅笔放在你想要分开的地方，折叠起来，放进冰箱。我晃动我的筷子，以确保它在酱汁中形成一个非常干净的分割。冷冻时，确保你的包放在一个相当平坦的表面上。冷冻后，拿掉铅笔，把你的调味汁叠起来，放在更紧凑的冰箱里。
