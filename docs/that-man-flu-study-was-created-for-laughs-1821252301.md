# 那项“人类流感”研究是为搞笑而生的

> 原文：<https://lifehacker.com/that-man-flu-study-was-created-for-laughs-1821252301>

听到美国有线电视新闻网[时间](http://time.com/5060429/what-is-man-flu/)[卫报](https://www.theguardian.com/lifeandstyle/2017/dec/11/stop-accusing-men-of-overreacting-man-flu-really-does-exist-claims-doctor) 的报道，你会认为在“男性流感”领域有了重大的医学突破，一劳永逸地回答了男人在面对普通感冒时是否是懦夫的问题。但是人类流感研究来自一个奇怪的愚人节传统，真正的科学家发表半开玩笑的论文，新闻媒体用它们作为点击诱饵。

这些论文来自 BMJ[(该杂志前身为《英国医学杂志》)的圣诞特刊。从某种意义上说，这期杂志每年都会发表真正科学的研究，但它们是为了搞笑而构建的，并不十分严谨。比如，如果你*想把人流感作为一个普遍的公共健康问题来研究，你可能会召集一个不止一个人的团队来做这件事。*](http://www.bmj.com/specialties/christmas)

这里有一个前一年的好例子:一个有趣的，无害的研究，是用真正的科学方法完成的。这是一项生存分析(通常在死亡风险高的人群中进行，例如癌症患者) [应用于医院病房](http://www.bmj.com/content/347/bmj.f7198) 的巧克力盒。存活率很低，因为员工吃了巧克力，明白吗？分析显示，最不受欢迎的巧克力“存活”的时间最长。

### 在新闻中关注这些

医学研究人员嘲笑他们的宠物话题没什么，但 BMJ [为这些研究发出新闻稿](https://www.eurekalert.org/pub_releases/2017-12/b-fm120717.php) 。它们最终出现在头条新闻中，因为它们有趣、出人意料，而且非常容易点击。报道“人流感”故事的媒体在如何提及“T4”[和“它来自哪里”](http://www.bmj.com/content/359/bmj.j5560) 方面有所不同。有些， [像商业内幕](http://www.businessinsider.com/what-is-man-flu-bmj-christmas-issue-2017-12) 一样，把重大警告放在标题里。其他的，像 CNN，直接报道了。

但是我们大多数人没有意识到它们是笑话，*尤其是*如果我们只是看到标题滚动而过。人们很容易产生这样的印象，即我们生活在一个研究人员对人类流感做出了一些突破性发现的世界。

每年这些研究都让我抓狂，因为它们是如何被报道的，但是当我在杂志上读到它们时，就像它们的创造者所希望的那样，它们是如此的令人捧腹。以下是 2017 年一期的一些值得关注的故事:

[医生失败:医师疲劳的预警信号？](http://www.bmj.com/content/359/bmj.j5503)——这是为了看起来像是对因疲劳而犯下的错误的研究。实际上，这是一个脸书医生组织收集的最搞笑的“失败医生”。例如:“我在给药房打电话续药的最后说了‘爱你’。”“我同事曾经在一次子宫颈抹片检查中放入一个窥镜，并让病人说‘啊啊’。”

[*粉红猪小妹*是否鼓励不当使用基层医疗资源？](http://www.bmj.com/content/359/bmj.j5397)——这一篇，由一位医生写的，他也是一位家长(利益冲突声明:“看起来我的孩子是由粉红猪小妹赞助的，但任何关于这种效果的说法都是假的。”)严厉批评棕熊医生浪费病人的时间和英国卫生系统的资金，用于不适当的出诊，不必要的处方，以及严重违反病人隐私，包括治疗佩德罗·波尼在幼儿园教室中间的咳嗽。

[骄者必败吗？纵向分析英国大龄成年人](http://www.bmj.com/content/359/bmj.j5451)——从字面上看，人们在摔倒受伤前的两年里，是不是感到了很多骄傲？(他们没有。)

《BMJ 圣诞特刊》很有趣，但是请——尤其是如果你写或分享新闻的话——负责任地享受它。
