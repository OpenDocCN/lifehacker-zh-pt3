# 戒掉坏习惯，用好习惯取代它

> 原文：<https://lifehacker.com/kick-a-bad-habit-by-replacing-it-with-a-good-one-1821847085>

如果你想戒掉一个坏习惯，试着在失败时奖励自己，而不是惩罚自己。习惯经常根植于我们的潜意识中，以至于我们甚至没有意识到我们正在做这些事情，这使得阻止一个对我们有害的习惯变得特别困难。

*商业内幕* [最近我采访了耶鲁大学的心理学家约翰·巴格](http://www.businessinsider.com/how-to-quit-drinking-stop-bad-habits-psychologist-simple-trick-2018-1) ，他试图改掉有自己的一个坏习惯:喝酒。为了能及时赶回家去买酒，他从田纳西高速开车到纽约，之后，巴格意识到他需要做出改变，但他不确定该如何处理。

当然，他戒掉了所有的酒，但他也开始接触其他东西:宝贝。

每当他想喝酒的时候，他就会吃棒棒糖。就这样，他没有因为想喝酒而惩罚自己，而是在喝酒的时候沉溺于一点点糖——并且创造了一个新习惯。

根植于后悔或恐惧的行为改变通常不起作用。巴格从一个朋友那里得到了棒棒糖习惯的灵感，这表明他对酒的渴望实际上是对糖的渴望。所以，他没有满足自己的欲望，而是用棒棒糖给自己的手和嘴做了别的事情。

尽管如此，他很可能只剩下吃糖的习惯，而不是喝酒，但这个想法是合理的。

南加州大学的习惯专家温迪·伍德告诉《商业内幕》,养成一个新习惯可能需要几周到 250 多天。如果你把一个习惯比如喝酒换成你喜欢的其他东西，那么你就不需要太依赖自我控制来达到你的目标，相反，就像那个坏习惯一样，新习惯会自动产生。

至于巴格和他的糖果，他说虽然它像喝酒一样成了一种习惯，但最终他不再喜欢它了，所以他戒了。然而，当他这样做时，他的饮酒习惯并没有恢复。
