# Infinity Dashboard 让您的 Mac 成为有用信息的仓库

> 原文：<https://lifehacker.com/infinity-dashboard-makes-your-mac-a-warehouse-of-useful-1826742422>

Mac: Widgets？又旧又破。让你的通知中心充满应用和服务？越塞越没用。至少，如果我需要快速浏览最有意义的数据，我不喜欢上下滚动页面。



这也是我喜欢最近发布的 Mac 版 [Infinity 仪表盘](https://fiplab.com/apps/infinity-dashboard-for-mac) 的原因。但是在我开始描述之前，我先告诉你一个坏消息:它的价格是 15 美元。这不是一个小数目，但该应用程序有七天的试用期，所以至少你可以在购买完整许可证之前看看你是否知道你会经常使用它。

Infinity Dashboard 的前提很简单:你应该能够通过点击菜单栏上的一个小按钮(一旦你在安装后通过点击菜单栏图标将应用程序设置为“dock”)来访问关于你的计算机(和生活)的大量快速信息。你不会得到又大又厚的卡片或疯狂的图形或任何东西。

相反，你只是得到文本，包括数据点，比如离某个特定事件还有多少天(离万圣节还有 4 个月零 18 天，以防你好奇)，你最喜欢的货币的当前汇率是多少(或者 [当前比特币价格](https://www.youtube.com/watch?v=uS1KcjkWdoU) )，目前有多少人喜欢你的脸书页面，以及某个特定商品在亚马逊上的价格，等等。

虽然 Infinity Dashboard 的默认布局并不那么有用，但该应用程序附带了一系列捆绑模块(总共 38 个)，您可以根据需要配置到您的仪表板中，包括从您的可用磁盘空间到当前天气，到您当前的 Fitbit 统计数据，到您的 Reddit karma。这些模块中有很多都是以服务为主题的，如果你不在乎，比如说，你的 YouTube 页面目前有多少订阅者，那也没关系。Infinity Dashboard 还可以让你安装其他用户为该应用程序创建的模块，并在其网站上进行管理。

(是的，只要你对 JavaScript 有足够的理解，你也可以自己编写。)

虽然 [社区创建的产品](https://infinitydashboard.com/) 现在有点少——在我写这篇文章的时候总共只有 10 个——我完全期待 Infinity Dashboard 随着(或者如果)它变得流行起来会有很多更有趣的黑客。在此之前，它暂时价值 15 美元，但我强烈建议尝试一下免费试用，看看它是否能很好地满足您的需求。如果没有，有很多其他方法可以把更多的信息装进你的菜单栏。
