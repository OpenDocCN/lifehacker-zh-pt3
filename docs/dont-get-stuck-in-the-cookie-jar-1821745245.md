# 不要被困在饼干罐里

> 原文：<https://lifehacker.com/dont-get-stuck-in-the-cookie-jar-1821745245>

欢迎回到 [**周中冥想**](http://lifehacker.com/tag/mid-week-meditations#_ga=2.170929957.440406047.1499703745-27973805.1434581949) ，Lifehacker 的每周一次的斯多葛派智慧之池探索，以及使用它的水域来反思和改善你的生活的指南。



本周精选来自爱比克泰德。他给那些似乎“想要一切”的人上了一课:



“当孩子们把手伸进一个狭窄的糖果罐时，他们无法伸出拳头并开始哭泣。丢几颗糖，你就会把它弄出来！抑制你的欲望——不要把你的心放在这么多事情上，你会得到你需要的。”

*话语*，3.9.22



### 这意味着什么

人们想要的太多了，以至于他们最终往往什么都得不到，然后他们一生都在抱怨。如果他们能放下一些欲望，他们就能集中精力，拥有他们真正关心的东西。基本上，你不可能拥有一切。但是你可以拥有一些东西，如果你愿意专注于它们的话。

### 从中吸取什么

我喜欢这句话，因为它的教训是如此简单，但非常有用。生活、现实、宇宙是“糖果”或饼干罐的约束。它只会让你承受(或做)这么多。如果你试图获取或做得太多，有可能你什么都得不到。你不能把自己分散得太开，也不能把宝贵的时间浪费在对你整体福祉不太重要的事情上。这些东西让你的拳头大得无法从罐子里拔出来，我们都浪费了太多时间抓住这些东西，相信有另一种方法可以把它们全部取出来。

所以，你有两个选择:砸碎罐子，把饼干送得满天飞，全部毁掉；或者干脆放弃几块饼干，用你能抓到的更少的来凑合。在你生活的每一个方面，试着“放弃一些享受”优先考虑那些能真正丰富你生活的东西——无论是工作、娱乐还是你的个人生活——然后把剩下的留在罐子里。有几块饼干总比没有好。
