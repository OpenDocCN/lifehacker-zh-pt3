# 这是一个无辜的 OK 标志还是一个白色的权力象征？

> 原文:[https://life hacker . com/is-this-an-innocent-ok-sign-or-a-white-power-symbol-1825575794](https://lifehacker.com/is-this-an-innocent-ok-sign-or-a-white-power-symbol-1825575794)

这个帖子是关于 [坎耶西的《马加周》](https://gizmodo.com/twitter-kanye-allegedly-losing-9-million-twitter-follo-1825551707) 的。抱歉。你可能已经听说了，Kanye 又回到了 Twitter 上，这次他发布了一系列奇怪且不断升级的推文，似乎支持古怪的保守派，甚至唐纳德·特朗普(Donald Trump)。 [每期](https://www.rollingstone.com/music/news/why-kanye-wests-pro-trump-tweets-are-a-real-threat-w519572) [推文](https://www.thedailybeast.com/chance-the-rappers-wildly-hypocritical-defense-of-kanye-wests-maga-turn) [激励](http://nymag.com/selectall/2018/04/why-kanye-west-endorsed-candace-owens.html) 讲解。昨天 Kanye 在推特上发了一张他自己戴着 MAGA 帽子的照片，旁边是一个做 OK 手势的男人。这令人不安，因为 OK 标志有点像 T21，是白人至上主义的象征。

如果你已经知道 OK 标志怪异愚蠢的近期历史，跳到底部。(如果你讨厌我们谈论这个，救救你自己吧！关闭选项卡！去读一下 [青西红柿](https://skillet.lifehacker.com/you-dont-have-to-fry-green-tomatoes-1825527733#_ga=2.31662013.227700041.1524575334-1297080755.1497980211) ！)但在其他方面:

2015 年左右的一段时间，OK 标志在特朗普的支持者中流行起来。知道你的热图给出了一个 [细致的历史](http://knowyourmeme.com/memes/ok-symbol-%F0%9F%91%8C) ，展示了它在迈克·塞尔诺维奇(一个 [脑丸推销员](https://splinternews.com/mike-cernovich-s-reddit-ama-backfires-spectacularly-1821538191) 相信希拉里在一家披萨店经营着一个恋童癖集团)和米洛·伊安诺普洛斯(他发誓自己不是新纳粹，但 [身边都是新纳粹](https://specialprojectsdesk.com/how-i-got-the-dsa-caught-up-in-a-two-minutes-hate-on-fo-1825542782) )等人物身上的运用。众所周知，这两个人都热衷于宣传和争议，这解释了接下来发生的许多事情。

在某个时候，很多人开始把 OK 符号视为白色权力的象征。一种解释是，三个伸出的手指组成了一个 W，圆圈和手掌组成了一个 4chan 上的一个线程(类似于 Reddit，但更糟)据说是为了传播“白色力量”的谣言，将其作为一个骗局。反诽谤联盟称 OK 标志不是白色权力的象征， [用 4chan 线作为证据](https://www.adl.org/blog/no-the-ok-gesture-is-not-a-hate-symbol) 。

但正如知道你的热图指出的那样，4chan 线程直到 2017 年才出现，这场辩论已经进行了多年。回到大选之夜，实际上毫不含糊的白人至上主义者理查德·斯潘塞(后来在电视上挨了一拳)在川普国际酒店外打出了 OK 的标志。

 [https://lifehacker.com/embed/inset/iframe?id=twitter-796132542739083264&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-796132542739083264&autosize=1) 

制造 OK 标志的右翼分子——那些声称他们不是白人至上主义者的人——喜欢这场争论，因为嘿，这只是 OK 标志！把这样一件无辜的事称为种族歧视是多么愚蠢啊！看，自由派和少数派做了这个标志！

因此，右翼分子和特朗普的支持者开始到处制作 OK 标志，然后当人们指责他们是种族主义者时大笑。“哈哈！”他们去了。“开你的玩笑！我不是种族主义者！我只是故意让你*认为*我是种族主义者！”白宫实习生杰克·布鲁尔 [在一张官方班级照片](http://www.newsweek.com/did-white-house-intern-make-white-power-symbol-764865) 中做了 OK 的手势，而照片中其他四十多个人都竖起了大拇指。当媒体询问他是否把它作为白色力量的标志时，他在推特上发了一条 [讽刺性的道歉](https://twitter.com/jjbreue/status/946627568488407040) :

> 在我们的一些实习照片中，我模仿了总统有时做的 OK 手势。那是愚蠢的。我应该更仔细地听听总司令的话，并竖起大拇指。
> 
> 我为我的犹太传统感到自豪，并强烈反对与种族主义白人权力组织相关的仇恨观点。我永远不会和他们有共同的目标。

但是 [据《每日邮报》](http://www.dailymail.co.uk/news/article-5216137/White-House-intern-flashes-white-power-sign-Trump.html) 报道，布鲁尔的 LinkedIn 称他曾为前白宫顾问、 [穆斯林禁令支持者](https://www.washingtonpost.com/politics/stephen-miller-a-key-engineer-for-trumps-america-first-agenda/2017/02/11/a70cb3f0-e809-11e6-bf6f-301b6b443624_story.html?noredirect=on&utm_term=.3960a1fd1413) 和 [职业种族主义者](https://splinternews.com/why-does-stephen-miller-sound-like-such-a-dick-a-voice-1793858636) 斯蒂芬·米勒实习。(从那以后，布鲁尔显然已经把米勒的名字从他的 LinkedIn 上抹去了。)米勒长期以来的反移民努力完全符合白人至上主义者的目标。

很自然地，米勒在白宫照片中闪现了 OK 标志——或者是某种古怪版本的 OK 标志，看起来很像白人帮派标志。巧合还是狗哨？大纲问了几个出处和[都没太定](https://theoutline.com/post/1428/the-ok-sign-is-becoming-an-alt-right-symbol)和。

 [https://lifehacker.com/embed/inset/iframe?id=twitter-896130610200104960&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-896130610200104960&autosize=1) 

但不管 OK 标志在特朗普的支持者手中是否真的意味着“我是一个白人至上主义者”，它现在意味着“我可以接受你*认为*我是一个白人至上主义者，只是为了一个玩笑。”那是一副糟糕的表情。当你试图*欺骗人们认为你是种族主义者的时候，很难坚持你不是种族主义者。* 

当然，有些人只是无辜地闪了一下 OK 信号，因为，好吧，它仍然是 OK 信号，他们从来没有听到不同的声音。知道你的热图的负责人布拉德·金通过电子邮件告诉 Lifehacker:

> OK 标志现在对具有不同迷因/媒介素养水平的人有一些不同的含义:
> 
> 1)那些得到备忘录(关于 alt-right association)的人，现在认为这是耻辱；
> 
> 2)那些得到备忘录的人，现在把它用作狗哨；(并陶醉于你所说的似是而非的否认之地)
> 
> 3)没有得到备忘录并继续使用符号表示“没事”的人；
> 
> 4)那些得到备忘录并看到符号扩散的人，现在利用它作为冲击值的触发器或重新分配的装置。

好吧，那么，Kanye 照片中的 OK 标志是白色的力量标志吗？大概不会。但也算是吧。抱歉。

制作这个符号的人是 Lyor Cohen，YouTube 的全球音乐总监。一名 YouTube 代表告诉 TMZ，科恩用这个标志来代表他创立的唱片公司， [300 娱乐](http://300ent.com/) ，该公司签下了 Fetty Wap、Hopsin、Coheed 和 Cambria 等艺术家。(科恩在 80 年代也是 [Def Jam](http://www.defjam.com/) 的负责人。)科恩亮出白色权力标志的可能性似乎极小。

当然，Kanye West 也不太可能戴 MAGA 帽子。但他就在那里，坐在科恩旁边，很可能认识到这个 OK 标志对许多右翼分子意味着什么。布拉德·金认为坎耶很清楚自己在做什么:

> 对于坎耶，我个人认为他总是对媒体以及如何操纵媒体有着敏锐的解读，他正沉浸在对他有利的 OK 姿态的模糊中。

最终结果是什么？据媒体称，这可能会 [毁掉坎耶的音乐和遗产](https://www.theringer.com/music/2018/4/25/17279292/kanye-west-donald-trump-candace-owens-scott-adams-twitter-alt-right) ，但可能 [不会改变任何事情](https://www.theverge.com/tldr/2018/4/25/17282626/kanye-west-donald-trump-twitter-love-millions-of-followers-loss-false-rumor)；这可能 [帮助](https://www.politico.com/story/2018/04/26/trump-kanye-praise-555141) 或 [伤害](https://www.washingtonexaminer.com/news/flashback-dnc-embraced-kanye-west-declaration-to-run-for-president-in-2020) 共和党；而坎耶或 [难道](https://thedailybanter.com/2018/04/kanye-west-donald-trump/) 没有机会竞选总统吗？

OK 标志并不意味着白人权力，许多使用它的人是或不是纳粹，坎耶支持或不支持唐纳德·特朗普，所有这些都无关紧要。多糟糕的一周。
