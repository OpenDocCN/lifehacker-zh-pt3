# 动作(图)打包工作区

> 原文：<https://lifehacker.com/the-action-figure-packed-workspace-1778986696>

读者 Norma 刚刚决定翻新她的工作空间，而且只花了不到 800 美元。她在 Flickr 上有很多之前和之后的照片，但我们在这里将重点放在之后的照片上，因为它们看起来很棒。工作空间、大量游戏储物空间和大量动作玩偶。



她的家庭办公室左侧的全图在这里的顶部，滑动存储被拉开，露出了她主要电脑旁边的书架和游戏书架，这是一台节省空间的 iMac。我们想象，电脑的右边有空间让另一个人放松，或者让她伸开，后面还有一台电视。这是右边的一张照片，所有的活动人偶都住在这里，储物空间都关闭了:

最后，一个体面的图片架本身。相当整洁:

就像我们提到的，如果你想看看改变之前的空间是什么样子，你可以点击下面的 Flickr 链接来浏览整个画廊，那里也有一些直接的之前/之后的对比图片。与此同时，如果你的办公室周围有一些玩具和收藏品，也许你可以从中获得一些灵感！

如果您有自己的工作空间可以炫耀，请将其添加到我们的 [**Lifehacker 工作空间展示并告诉 Flickr pool**](http://www.flickr.com/groups/lifehacker-workspace-showandtell/pool/) 来与我们分享。确保您包含的任何照片至少为 1280x720。让他们保持在 16:9 也有帮助！包括一些文字，关于你使用的材料，你是如何想出这个设计的，以及任何其他相关的细节。如果你巧妙的组织和良好的设计感吸引了我们的眼球，你可能就是下一个特色工作区！
