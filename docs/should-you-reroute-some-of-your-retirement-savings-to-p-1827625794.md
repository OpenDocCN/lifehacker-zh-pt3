# 该不该用退休储蓄买房？

如果你已经攒够了首付，你还需要多少钱来支付购房过程中出现的所有额外费用？在短期内减少你的退休供款值得吗？这就是我们这个星期在想的。

本周的问题来自 Wil:

> 我丈夫和我最终打算在一年到 18 个月内买一栋房子。我们想买的区域刚刚开始出现一些变化，所以我们不想等太久，以免它变得和周围的一切一样难以承受。
> 
> 虽然我们存了足够的首付(差不多),但我想确保我们手头有足够的流动现金用于所有的小事情，以及日常生活。我们目前向 401(k)缴纳的比例相对较高(12%)。在一年左右的时间里暂时减少这些供款，以帮助填补购房的银行账户，这是一个好主意吗？(当然，我们不会低于雇主匹配百分比！)

这是个别专家对一个对每个人都有不同影响的问题的一般看法——如果你想要个性化的建议，你应该去见理财规划师。

### 理清你的财务重点

恭喜你迈出了下一大步！听起来你已经考虑清楚了，并且倾向于减少你的退休金。这并不一定有什么错——比如，这比让自己深陷信用卡债务要好，因为你没有预见到一些费用，而且你没有足够的流动资产来支付这些费用。但是让我们检查一下你所有的选择以确保。

显然你知道如何做出牺牲——为首付存钱，同时还能为你的退休账户贡献两位数，这是一个令人印象深刻的壮举。如果你在开始将资金转移到你的房子之前，正在寻找一个特定的数字，LearnVest 的创始人 Alexa von Tobel 说:“一般来说，人们应该在退休后将退休前收入的 70%到 85%替换掉，然后再开始为首付存钱。”

也就是说，在一天结束的时候，只有这么多钱可以周转，只有你知道你的优先事项是什么。因此，如果买房是你的梦想，而且它会比提前一两年退休给你带来更多的满足感，那就值得考虑了。同样，在你把房子的一切都处理好之后，你可能会在接下来的一年左右增加你的储蓄。

“是的，为了实现一个有价值的目标，暂时减少 401(k)的缴款是可以的。购买第一套住房是一个有价值的目标。所以，确保你有未雨绸缪的储蓄来支付不可避免的房屋维修和其他房屋所有权费用，另外，正如你所说，为“一般生活”做准备，”Holden Lewis， [Nerdwallet](http://www.nerdwallet.com/) 的房屋专家说。“最终，当你知道自己有能力支付时，你可以回到以前的退休储蓄水平。与此同时，你做出的贡献足以赢得雇主匹配，这是正确的。”

另一方面，如果你对买房持观望态度，或者需要大幅减少退休储蓄来攒够钱，那么试着削减其他额外费用，同时仍然为你的退休账户贡献几个月的资金。将减少的部分存入一个单独的高收益储蓄账户，专门用于购房过程，这样就可以建立起你的安全网。

不要专注于一个任意的购买时间表，而是专注于以一种不会给你和你丈夫带来太大压力的方式逐步建立你的应急基金。毕竟，没有“正确”的购买时机。到达那里可能需要更长的时间，但是你会过得更好——无论是短期还是退休后。

你需要存多少钱？当然，这将取决于你住在哪里，抵押贷款的规模以及其他一百万个你无法预见的变量。但根据 CNN 财经 的报道，购房者往往会产生 2000 至 5000 美元的意外支出。所以，把它纳入你的预算，看看除了你为退休准备的 12%之外，是否还有其他的弹性空间。就退休储蓄而言，你处于一个很好的位置——难道你不想呆在那里吗？

如果这太难做到，那么你可以拨回你的退休金。“记住，为退休储蓄不是你的首要目标；刘易斯说:“你们的首要目标是一起过美好的生活。“对你来说，美好的生活包括拥有一栋房子，并且不用担心以后的账单。满怀热情地追求那个目标，然后你就可以回去为退休存钱了。”
