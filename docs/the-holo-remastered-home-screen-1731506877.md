# 赫萝重制了主屏幕

> 原文:[https://life hacker . com/the-holo-remastered-home-screen-1731506877](https://lifehacker.com/the-holo-remastered-home-screen-1731506877)

Android 可能已经将赫萝抛在身后，走向更光明的前景，但并不是所有人都忘记了这一点。这一设计将旧的赫萝蓝 混合在经典的深色背景中，打造出一种时尚的外观。

Watch

要在您的手机上实现这一设计，您需要:

*   [新星发射器](https://play.google.com/store/apps/details?id=com.teslacoilsw.launcher)T3】
*   [KLWP](https://play.google.com/store/apps/details?id=org.kustom.wallpaper.pro)T3】
*   [文件](https://play.google.com/store/apps/details?id=holo.klwp)
*   [指令](http://mycolorscreen.com/2015/09/13/holo-for-klwp/)

你有没有一个很棒的、被调整到遗忘的家或者你自己的锁屏，你想分享吗？把它贴在下面的评论里，或者贴在你自己的 Kinja 博客上，标签是“主屏幕展示”(没有引号)。一定要包括你是如何制作它的描述，这样我们就可以把它作为下一个特色主屏幕。
