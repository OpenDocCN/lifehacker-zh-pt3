# 你可能错过的安卓牛轧糖最酷的功能

> 原文：<https://lifehacker.com/the-coolest-features-of-android-n-google-didnt-announce-1764114846>

昨天， [谷歌为 Nexus 机主发布了安卓牛轧糖](http://lifehacker.com/android-nougat-has-arrived-rolling-out-to-recent-nexus-1785598253) 。它带有一些可爱的新功能，如多窗口支持和捆绑通知。在 [深入开发者预览](http://lifehacker.com/how-to-install-the-android-n-developer-preview-on-your-1763813851) 之后，我们发现了更多你可能已经错过的东西。

### **瞌睡现在即使在你移动的时候也能工作**

Android Marshmallow 引入了一个强大的电池节省功能，当你把手机放在桌子上时，它可以禁用大多数消耗电池的东西。现在，当你在移动中不使用手机时，你也能获得同样的能量。如果你有一段时间不碰你的手机，即使它在你的钱包里，坐在你的车里，或者在你的口袋里，瞌睡也会被激活。

### 数据保护模式可以防止你的数据溢出

数据上限很糟糕，但大多数运营商都有某种形式的计量数据。除非你想付更多的钱，否则你需要防止狡猾的应用程序花光你的钱。数据保护模式可以帮助解决这个问题。当您启用数据保护模式时，任何在后台运行的应用都将无法访问蜂窝数据(除非您将它们添加到白名单中)。如果你开始接近你的数据上限——你可以在设置>数据使用中编辑——Android 会提示你启用数据保护模式。

### **双击最近使用的按钮，在您最近使用的应用程序之间切换**

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-DgDAGV2ryGg&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-DgDAGV2ryGg&start=0) 

最近按钮已经可以很容易地 [滚动打开的应用程序列表和 Chrome 标签](http://lifehacker.com/updating-all-the-new-stuff-in-androids-l-release-prev-1595928268) 。在 Android Nougat 中，你可以双击 Recents 按钮切换回你上次打开的应用程序。

如果您需要快速跳转到您上次打开的应用程序之外的应用程序，您也可以重复轻按“最近”按钮来循环浏览您打开的所有应用程序。它就像一个粗略的 Alt-Tab，但在你的手机上。它不像双击手势那么自然，但一旦你习惯了，它会比手动滚动应用程序更容易。

### 长按通知可轻松静音

Android 有大量的选项来管理哪些应用程序可以向你发送通知，这些选项深埋在 Android 中。然而，调整它们可能会很繁琐。现在，在 Android Nougat 中，如果你看到一个你不喜欢的通知，你可以长按它来查看几个选项来摆脱它。你可以将其设置为静默显示来自该应用程序的通知，完全阻止该应用程序的所有通知，或者跳转到设置应用程序以更精确地调整它们。

### 使用系统用户界面调谐器手动设置应用程序在通知阴影中的优先级

这个功能被藏在 Android 不断变化的 [系统 UI 调谐器菜单](http://lifehacker.com/the-coolest-things-you-can-do-with-androids-hidden-syst-1773364942) 中。在内部，Android 为你的所有通知分配了一个不可见的优先级，从 0(完全阻止)到 5(显示在列表顶部，允许全屏中断)。通常，您不应该需要或想要调整这个设置，但是如果您对通知有一些非常特殊的需求，您可以调整这个设置。

### **内置的文件管理器更加强大**

[在安卓棉花糖](http://lifehacker.com/android-marshmallow-has-a-basic-file-manger-heres-how-1737296454) 中，谷歌推出了一个基本的文件管理器，允许你浏览文件夹，制作文件副本，以及...大概就是这样。现在，名为 Explorer 的文件管理器允许你移动文件，重命名文件和文件夹，并创建新的文件夹——简而言之，做一个相当强大的文件管理器应该让你做的一切。您还可以打开多个 Explorer 实例，以便在不同的文件夹之间跳转。虽然像 [Solid Explorer](https://play.google.com/store/apps/details?id=pl.solidexplorer2&hl=en) 、我们最喜欢的 Android 文件管理器 [这样的应用程序仍然更加强大，但任何只需要基本文件管理的人都会与内置工具相处得很好。](https://lifehacker.com/the-best-file-management-app-for-android-5825578#_ga=1.33784033.1347662518.1465819317)

### **您可以在系统级别阻止电话号码**

如果你厌倦了电话推销员、机器人电话和垃圾邮件发送者，Android Nougat 有一个系统级的呼叫阻止功能，将适用于整个手机。一旦某个号码被屏蔽，它就不能给你发短信或打电话了。这些设置会在重置后保持不变，所以如果你买了一部新手机或删除了当前的手机，你并不是从零开始(这也意味着被屏蔽的号码会存储在你的谷歌账户中，这很好！)此外，任何应用程序都可以访问黑名单，所以你不能阻止某人给你打电话，但不能阻止某人通过 Hangouts 给你发短信。到目前为止，Android 上有几个屏蔽来电的选项，但大多数都是折中的办法。在你的拨号器中屏蔽一个号码不会在你的 VoIP 应用中屏蔽它，屏蔽短信发送者不会屏蔽电话呼叫，屏蔽任何来电只会将他们的呼叫转到语音邮件，你稍后必须在那里处理他们的消息。这项新功能甚至会删除他们的语音邮件。

### **可将“请勿打扰”设置为闹铃响起时自动结束**

Android 的勿扰模式允许你设置静音时间，屏蔽大部分或所有通知。在 Android Nougat 中，当你早上的第一个闹钟响起时，勿扰模式可以覆盖你的安静时间设置。因此，举例来说，如果你设置免打扰在每天早上 7:00 结束，但是有一天你需要在早上 6:00 起床，你将在第一个闹钟响起时开始收到通知，而不改变你的正常规则。

### 新的图块编辑器允许您自定义快速设置图块

Android 中的快速设置阴影是启用或禁用 Wi-Fi、蓝牙或自动旋转等功能的便捷方式。现在，您可以使用新的图块编辑器挑选显示在阴影中的图块。只需拉下快速设置阴影，点击编辑按钮，删除你不需要的，拖入新的，或重组他们你认为合适的。

### 使用新的显示大小滑块调整界面的大小

在之前的 Android 版本中，如果你想更多地使用你那部大得出奇的手机，你可以调整你的文本大小。然而，你不能对滑稽的大图标做任何事情。现在，你可以用一个新的滑块来调整两者，这个滑块可以调整屏幕上界面元素的大小。

### 下班打卡时，工作模式切换可以禁用所有与工作相关的应用程序

如果你的公司使用 Android For Work 来管理你的工作电话，你就有了一个新工具来帮助你脱离办公室。新的工作模式快速设置磁贴现在可以让你一键禁用所有工作应用。只要拉下你的窗帘，轻触开关，你就可以下班了。享受你的休息时间！

### **您可以在主屏幕上添加紧急联系人和医疗信息**

在 Android Nougat 中，你可以在拨号器中添加自定义紧急联系信息，这样任何人都可以拿起你的电话，立即看到他们可以联系的人或阅读关于你的重要信息——除了拨打紧急电话，就像你一直可以做的那样。您可以添加您的全名、地址、出生日期、血型、已知过敏、药物、医疗状况、器官捐献者身份和紧急联系人。我们已经 [一直建议你这样做](http://lifehacker.com/add-emergency-contact-information-to-your-phones-lock-909275428#_ga=1.3400560.1347662518.1465819317) ，但是现在它已经被嵌入到 Android 中了。进入设置>用户>紧急信息添加您的信息。请记住，任何人都可以通过您的手机访问这些信息。它不会直接显示在锁定屏幕上——你必须挖掘紧急拨号程序才能找到它——但它仍然可以使用，无需先解锁。

### **不用再等着每次更新**
时“优化应用”

在 Android 4.4 Kit Kat 中，Android [切换到 ART 运行时](https://lifehacker.com/android-art-vs-dalvik-runtimes-effect-on-battery-life-1507264545) 后，每次 Android 手机获得重大更新时，它必须在第一次重启时经历一个“优化应用程序”阶段，这将花费*永远的时间*。现在，谷歌对运行时做了一个改变，即 [完全跳过了优化步骤](http://www.androidpolice.com/2016/03/09/android-n-feature-spotlight-updated-art-compiler-means-no-more-optimizing-apps-step/) ，从而使过程变得更快。更好的是，这意味着应用程序的安装速度也会更快。有趣的是，我发现在擦除并安装 Android N 后，重新下载我的 Nexus 6P 上的 100 多个应用程序只需要几分钟，而不是通常需要的一个小时左右。

### **测试版中被删除的功能**

自从开发者预览版首次发布以来，我们一直在使用它。正如我们所料，测试版中出现了一些不适合发布的特性。为了子孙后代(或者你可以要求谷歌明年永远把它们带回来)，下面是我们在最终版本中发现的伤亡情况:

*   夜间模式:谷歌连续第二年用夜间模式戏弄预览版用户，结果在最终版本中被抢走了。然而，还有一线希望。如果你从现有的测试版升级后没有擦你的手机，你仍然有机会从快速设置界面进入夜间模式。否则，我们都将不得不等待，并希望谷歌明年使这一点正式。
*   **手动显示器校准:**一个埋在中的设置[系统 UI 调谐器](http://lifehacker.com/the-coolest-things-you-can-do-with-androids-hidden-syst-1773364942) 允许用户 [手动校准他们显示器的 RGB 等级](http://www.androidpolice.com/2016/03/09/android-n-feature-spotlight-color-balance-calibration-now-available-in-system-ui-tuner/) 。可以理解的是，谷歌决定不给普通用户意外破坏显示的能力。

从表面上看，Android Nougat 中只有几个引人注目的功能，如分屏应用程序或即将推出的虚拟现实模式[。然而，在引擎盖下，有许多不错的小调整，让你的手机更容易日常使用。](https://vr.google.com/daydream/)
