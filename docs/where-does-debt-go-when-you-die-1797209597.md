# 你死后债务去了哪里？

> 原文：<https://lifehacker.com/where-does-debt-go-when-you-die-1797209597>

不幸的是，即使你离开了，你的债务仍然存在。虽然这不一定意味着你的孩子或配偶会继承一个财务烂摊子，但在某些情况下，他们可能要对你留下的一些债务负责。

Watch

当你死后，你通常会把你的遗产 (你拥有的所有东西，包括投资)留给你信任的人，比如你的配偶或孩子。这个人被称为遗产的*执行人*，执行人负责用你的资产来偿还账单和债务。例如，如果你有 100，000 美元的存款，欠了 5，000 美元的账单和信用卡债务，这 100，000 美元将用于支付这两者。

但是假设你除了账单和债务什么都没有。

好消息是，遗嘱执行人、你的配偶或你的孩子不会自动为你的债务承担责任。然而，在某些情况下，他们可能是负责任的，比如他们共同签署了你的贷款，或者他们是你所欠账单或债务的联名账户持有人。例如，如果你欠了几千英镑的抵押贷款，而你去世了，如果你丈夫的名字也在那笔抵押贷款上，他确实要对余额负责。

虽然 [你的配偶不能为你婚前积累的债务](http://twocents.lifehacker.com/no-getting-married-doesn-t-make-you-liable-for-your-sp-1749848886#_ga=2.162673914.719160835.1500990688-1313785359.1499701416) 负责，但根据你的居住地，配偶也可能有责任支付已故伴侣的其他债务，即使他们没有共同签署这些债务，即使他们不是共同账户持有人。如果你生活在一个共有财产的州(亚利桑那州、加利福尼亚州、爱达荷州、路易斯安那州、内华达州、新墨西哥州、得克萨斯州、华盛顿州和威斯康星州)，你的配偶可能会在你死后陷入困境，前提是你在婚姻期间承担了债务。“婚姻中任何共有财产的一半都可以用来偿还债务，” [NerdWallet 解释道](https://www.nerdwallet.com/blog/insurance/debts-after-death-life-insurance/) 。

除了这些一般规则，它可能会变得棘手，因为有具体的规则，根据不同的债务。NerdWallet 为特定类型的债务划分了规则:

抵押贷款:如果有共同房主或某人继承了房子，他们可能会负责债务。但是，联邦法律禁止贷款人在另一个共有人死后强迫另一个共有人立即还清抵押贷款。如果没有共同所有人，你的继承人可以卖掉房子来偿还抵押贷款，用任何遗产来偿还，或者申请新的住房贷款来保留房子的所有权。如果他们决定出售房屋，并且以高于抵押贷款的价格出售，他们可以将差价收入囊中。如果他们卖得更低，他们就不会上钩， [报道说会破产。](http://www.bankrate.com/finance/mortgages/what-happens-to-mortgage-after-i-die.aspx)

信用卡:据 NerdWallet 报道，如果你的遗产用光了资产来偿还余额，信用卡公司就要倒霉了。“任何联名账户持有人都要对未付账单负责。仅仅是信用卡授权用户的人不负责支付余额。”同样，在夫妻共有财产制下，夫妻双方对婚姻期间产生的债务负责，这包括信用卡债务。

学生贷款:联邦学生贷款在你死后可以免除，一些私人贷款机构(例如 Sallie Mae 和 Wells Fargo)也会在你死后免除债务。除此之外，就很棘手了。一般来说，遗产可以用来偿还私人学生贷款，但如果没有足够的钱，贷款人也无能为力，有点像信用卡债务。也就是说，有一些例外:

*   如果有人共同签署了私人学生贷款，他们要对剩余的债务负责。

*   如果学生在婚姻期间承担了债务，并生活在一个共同财产州，配偶可能要为私人学生贷款的剩余债务负责。正如一位破产律师告诉学生贷款英雄 的那样，大多数共有财产州将会为与教育相关的债务破例。您可以根据您所在的州在此查找具体规则 [。](http://statelaws.findlaw.com/)

简而言之，当你去世时，你可能会将债务传给配偶，但一般来说，你不会自动将债务传给你的孩子、受益人或管理你资产的人。然而，讨债人有权给你的家庭成员打电话询问债务情况。好消息是，我们有权要求他们停止打电话给联邦贸易委员会说，他们给 [充分的指示在这里](https://www.consumer.ftc.gov/articles/0081-debts-and-deceased-relatives) 怎么做。
