# 关于特朗普新 SNAP 提案的一些问题

> 原文：<https://lifehacker.com/some-questions-about-trumps-new-snap-proposal-1822964950>

特朗普政府的 2019 财年预算提案 中包括一项补充营养援助计划的“大胆的新方法”，该计划为低工资工作家庭、低收入老年人和残疾人提供购买食品的援助。



根据该计划，那些每月从 SNAP 获得至少 90 美元的人(绝大多数接受者)将以“美国农业部食品包”的形式获得大约一半的福利，其中包括“诸如耐储存的牛奶、即食谷物、面食、花生酱、豆类和罐头水果、蔬菜以及肉类、家禽或鱼类等物品”(换句话说，没有)。总的来说，预算建议在十年内削减 2130 亿美元。

据 [预算与政策优先中心](https://www.cbpp.org/research/policy-basics-introduction-to-the-supplemental-nutrition-assistance-program-snap) 统计，SNAP 艾滋病感染者 4000 万人，其中 70%左右是有孩子的家庭。超过 25%的参与者来自有老年人或残疾人的家庭。目前，SNAP 接收者收到一张充值的 EBT 借记卡，他们可以在杂货店和其他授权零售商处消费。接受者不能用这些钱购买酒、香烟、维生素补充剂、热食或非日常生活用品(如卫生纸或尿布)。

政府写道，这是一种“成本效益高的方法”，但很容易看出，对于任何类型的食物过敏患者，以及没有家庭或地址可以接收盒子的人来说，这可能是一场灾难——考虑到所有的包装和运输成本，这可能会增加成本。

《大西洋月刊》(The Atlantic)经济政策作家安妮·劳里(Annie Lowrey)在 Twitter 上列出了一些很好的问题，这些问题似乎是政府没有充分考虑的。例如，如果有暴风雪或飓风会发生什么？更重要的是，“穷人有权选择他们自己的食物吗？”
