# 用这个 Chrome 扩展自动静音每个网站

> 原文：<https://lifehacker.com/mute-every-website-automatically-with-this-chrome-exten-1824186542>

让你在 Chrome 中让整个网站静音可能是谷歌有史以来对浏览器的最好升级，但它还可以更好。当然，屏蔽每个带有 [烦人弹出视频](https://lifehacker.com/now-google-chrome-can-automatically-silence-annoying-au-1821369846) 的网站是很棒的，但是如果你根本不需要处理任何音频呢？多亏了 AutoMute，你再也不会听到浏览器发出任何声音了。



[AutoMute](https://chrome.google.com/webstore/detail/automute/kjcdcbhfpjkcjinohfaaihpcmpnpmpie?hl=en-US) 是一个第三方扩展，正如其名字所暗示的那样。一旦你安装了这个软件，每一个新标签都会带有谷歌浏览器的静音图标。如果你确实想听某个网站的音频(比如 YouTube，或者 Lifehacker ),你可以将整个域名列入白名单。如果你想听一些音频，但仍然不信任网站的其他部分，你也可以将特定页面列入白名单。

一个问题(也是一个小问题)是 AutoMute 没有直接与 Chrome 自带的静音功能集成，所以你不能右键单击一个标签来取消静音。相反，你必须点击 AutoMute 扩展图标，一旦安装完毕，这个图标就会出现在浏览器地址栏的右边。可能需要一点时间来适应这种差异，但这是值得的，因为只需一次点击就能让整个互联网静音，让你安心。
