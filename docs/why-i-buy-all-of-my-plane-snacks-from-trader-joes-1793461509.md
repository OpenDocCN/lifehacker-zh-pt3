# 为什么我所有的飞机零食都是从乔氏商店买的

> 原文：<https://lifehacker.com/why-i-buy-all-of-my-plane-snacks-from-trader-joes-1793461509>

当我在飞行时，我的零食有两个非常重要的作用。他们应该让我不饿，但更重要的是，他们需要娱乐和分散我对飞行的厌恶。



但是打包一个好的零食袋并不像把一堆你最喜欢的食物扔进你的随身行李那么简单。除了遵守美国运输安全管理局的规定，我还有一些自己的规则:

1.  没有易碎的，脆弱的胡说八道，不能在我混乱的背包里生存。
2.  优先考虑口味，但也要涉及一些营养。坐飞机已经让我感觉身体不好，所以除了士力架和奇多什么都不吃是个糟糕的计划。
3.  仅限手抓食品。我不想向任何人要任何东西，比如餐具(反正他们可能不会有)。
4.  没有什么令人讨厌的臭味，因为那只是粗鲁和俗气，我并不像我看起来那么俗气。
5.  我也喜欢整洁的单份小包装的东西，但这并不重要，因为有拉链袋和便当盒。

在我多年购买飞机零食的过程中，我发现我最满意的是 Trader Joe's 的选择。乔氏不是一家普通的杂货店，它是一家很酷的杂货店，他们的食物非常有趣。作为一家公司，他们似乎对小吃相当着迷，这是我的个人审美，这些是我的最爱:

### 所有的干果

我一直在吃这些芒果和菠萝片，因为它们可以买到，而且它们耐嚼、略带酸味的性质非常适合空中旅行。它们不仅让你的下巴不停地动，耳朵也不停地跳，而且酸味的东西可以帮助缓解任何与晕动病相关的恶心。如果热带水果不是 yo' thang，TJ's 有各种各样的其他干燥(和冷冻干燥)产品，如苹果、桃子、杏子和各种蓝莓。哦，如果你有点喜欢你的水果干，总有这些美女，便宜，好吃，让人想起我年轻时的水果零食。

### 许多坚果和混合坚果

乔——如果真有这样的商人的话——一定是坚果迷，因为他的同名商店充满了生坚果、烤坚果、蜜饯坚果，以及介于两者之间的所有东西。( [)哈兰胡椒](https://www.youtube.com/watch?v=KGmNqzbsuiY)*会喜欢吗*。)我偏爱蜜饯山核桃。

然后是什锦，也就是许多不同的坚果混合在一起，还有一些其他的东西。坚果和什锦沙拉是很好的飞机食物，因为它们容易吃，容易填饱肚子，而且(至少按照我的标准)对你有好处。如果你想变得贤惠，那就拿一袋 [Go Raw Trek Mix](https://www.amazon.com/Trader-Joes-Raw-Trek-Mix/dp/B00BPYJ4ZW?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/why-i-buy-all-of-my-plane-snacks-from-trader-joes-1793461509&asc_source=&tag=kinjalifehackerlink-20) ，尽管它缺少盐和巧克力片，但味道出奇地好，或者你可以像我一样，拿一些超级性感诱人的 Trek Mix，它有方便的小把。

### 简单橄榄

乍看之下，带核和盐水的橄榄似乎不是一个好的旅行伴侣，但 TJ's 已经去除了核，并将它们包装在方便的无盐水小包装中。它们不仅是完美的咸点心——特别是如果你带了一点熟食，我们马上就会谈到——而且你可能应该在你的 [航班上喝几杯血腥玛丽或马提尼](http://skillet.lifehacker.com/how-to-make-flying-or-driving-a-more-tasty-experience-1742257240) 。

### 肉类和奶酪

Trader Joe's 有相当不错的选择，我喜欢称之为“成年人的午餐”,它们超级便携和可打包。小 Citterio 肉棒又小又脆又咸，让人上瘾；意大利熏火腿包裹的马苏里拉奶酪(咳咳，串奶酪)是一个包装中的所有东西，热那亚意大利香肠/provolone 组合是意大利野餐中的一杯基安蒂酒。你可以打包饼干或面包来搭配这些熟食，但是饼干会裂开，面包会被压碎，我不喜欢面包屑。

如果你不吃肉，但想加入奶酪行动，TJ's 也有各种各样的便携式乳制品。我喜欢他们的新鲜山羊奶酪奖章，但这些对于旅行来说可能有点易碎，我绝对喜欢他们的新鲜马苏里拉奶酪棒，它们看起来像奶酪条，但不串起，尝起来新鲜，滑腻，梦幻的马苏里拉奶酪。

### 巧克力，废话

坐飞机没有糖果就不算坐飞机，乔氏百货有非常棒的糖果。我是一个喜欢巧克力的人，但他们有非常棒的口香糖和甘草糖，如果那种东西是你的包，宝贝。他们有大盆的花生酱杯子和裹有巧克力的杏仁，你可以把它们重新包装成更小的、适合旅行的部分，但也可以偷看收银台过道里你最喜欢的小包装。

我在旅行时不得不吃的非巧克力糖是姜人的姜糖。我一直都有晕车的问题，这些超级姜味糖果可以缓解你难以想象的胃部不适。

他们还需要很好的咀嚼，如果在空中有耳朵和鼻窦压力问题，这是很好的。

### 口气清新剂

除非你随身带了一把小牙刷(并且用过了)，否则当你着陆时，你的口气可能会像机舱里的空气一样污浊。幸运的是，Trader Joe's 有我最喜欢的薄荷糖品牌(Myntz)，我只能在 TJ's 或 [在线](https://www.myntz.com/Breath-Mints/Winter-Mynt-Blast.html) 找到。它们非常强壮，不含糖，并且是“牙医推荐的”，值得一提。(我不是反对牙医，但是他们中“五分之四”的人总是推荐*一些东西*。我想和那个第五个牙医谈谈。他们似乎有很高的标准。)

这些都是我登机前在乔氏超市停留的诱人理由。你能在其他杂货店买到类似的东西吗？当然，但是我知道并且喜欢这些确切的品牌，我知道我能在一次旅行中得到他们。那次旅行会被设计糟糕到不可思议的 Trader Joe 停车场破坏吗？是的，但是我愿意处理这个事实应该告诉你我是多么专注于这些飞机上的小吃。
