# 现代网络钓鱼企图看起来更加合法，但方法并没有太大变化

> 原文：<https://lifehacker.com/modern-phishing-attempts-look-more-legit-but-the-metho-1794914817>

你收到一封新邮件，看起来像是来自朋友、公司、政府官员，甚至是家庭成员。那封邮件里只有一个链接。你点击它，因为，你当然会点击。您将进入一个登录页面，在这里输入您的凭证。然后，该网站被证明是假的，并收集您的密码。恭喜你，你被钓鱼了。

Watch

网络钓鱼是一种由来已久的获取您的登录凭证或访问您的帐户的方式。网络钓鱼背后的想法非常简单:它是欺骗你交出你的信息，所以没有实际的黑客攻击是必要的。从历史上看，这意味着给你发一封电子邮件，并希望你点击它。该链接指向网络钓鱼者建立的一个网站，该网站看起来像是真实网站的登录页面。登录后，网站会保存您的用户名和密码。这种情况从上世纪 90 年代就开始了，当时, [warez 社区【](https://books.google.com/books?id=mrBBFkCaulkC&pg=PA227&lpg=PA227&dq=warez+aol+phishing&source=bl&ots=LS4xfLKJzA&sig=Tzm2QEle-Nr3QTBBSSJVhHDUbA8&hl=en&sa=X&ved=0ahUKEwj14JqhttbTAhVkwVQKHZpEDXc4ChDoAQgoMAE#v=onepage&q=warez%20aol%20phishing&f=false)】T3】利用钓鱼邮件窃取人们的 AOL 密码。

那时候，黑客可以很容易地欺骗人们点击任何东西，因为这就是互联网过去的工作方式，但现在，我们都更加警惕，对不对？

如果昨天的 [Google Docs 网络钓鱼骗局有任何迹象](https://gizmodo.com/a-huge-and-dangerously-convincing-google-docs-phishin-1794888973) ，没有。昨天的网络钓鱼企图的前提与以往一样:你收到一封看起来像是来自朋友的电子邮件，所以你点击了它。在这种情况下，它试图看起来像有人想与你分享谷歌文档。你点击这个链接，它会把你带到真正的谷歌登录页面。这是合法的谷歌页面，所以你登录。当您授予最终成为钓鱼应用程序的权限时，下一步就是钓鱼尝试。这是对常规流程的巧妙运用，即使它并不完全是在做新的事情。这就是它的作用:

尽管这一尝试听起来复杂而奇特，但有迹象表明它是假的。首先，`googledocs.g-docs.win` URL 是粗略的，但除此之外，你几乎肯定已经给了谷歌访问你账户的许可，因为它是谷歌的，所以重新授权没有意义。谢天谢地，在这种特殊情况下，您至少可以撤销对那个 [网络钓鱼程序](https://lifehacker.com/how-to-secure-your-online-accounts-by-revoking-access-f-1794631133) 的访问。

这些年来，我们已经看到了类似的版本，但是网络钓鱼企图在复制流行网站的登录页面方面变得越来越好。最近的一次尝试使用了一个 [假附件来重定向你做一个谷歌登录页面](http://lifehacker.com/beware-this-clever-fake-attachment-gmail-phishing-sca-1793264478) ，2014 年的另一次尝试做了与本周黑客 [几乎一模一样的事情，复制一个谷歌文档登录页面](http://gizmodo.com/beware-of-this-dangerously-convincing-google-docs-phish-1546278702) 。去年民主党全国委员会被黑的那次？ [钓鱼](https://www.theguardian.com/us-news/2016/dec/14/dnc-hillary-clinton-emails-hacked-russia-aide-typo-investigation-finds) 。多年来，流行的网络钓鱼目标还包括贝宝账户、易贝账户和几乎每一家主要银行。

是什么让这些现代尝试变得更加复杂？也就是说，他们越来越擅长隐藏他们正在做的事情，所以 [通常的扫描技术不像](https://lifehacker.com/how-to-boost-your-phishing-detection-skills-and-avoid-e-5873050) 那样管用。网络钓鱼企图越来越擅长复制像谷歌甚至你的银行这样的网站登录页面，他们经常使用诡计来掩盖链接的真实 URL，并且他们越来越擅长使消息看起来像是来自真人。在谷歌文档的案例中，它完全是在利用人们对网络钓鱼企图的预期，然后利用它和谷歌的 OAuth 登录系统一起让你措手不及。

* * *

那么，一个精通技术的人能做些什么呢？继续*而不是*点击你邮件中任何奇怪的链接，不管你认为是谁发的。你的朋友很可能不会只是给你发一封带有链接的电子邮件，如果他们发了，你应该要求他们改变这种奇怪的行为。

否则，如果电子邮件中的链接看起来很奇怪，但你就是无法阻止自己点击它，*不要点击链接*，而是通过手动键入 URL，然后以这种方式登录，前往有问题的网站。如果必须的话，你通常可以用 [右键单击一个网址或将鼠标悬停在它上面来查看真正的目的地](https://lifehacker.com/this-infographic-shows-the-common-ways-scammers-try-to-1787443763) ，但即使是这种扫描方法也可能救不了你。

更重要的是，如果你想善用你对网络钓鱼的警惕，并想帮助人们前进，德高望重的 [SwiftOnSecurity](https://twitter.com/SwiftOnSecurity) 已经汇集了一个 [网络钓鱼报告工具集合](https://decentsecurity.com/#/malware-web-and-phishing-investigation/) ，这样你就可以报告粗略的链接，这样其他人就不会成为那些企图的牺牲品。

请记住，不要点击公司邮件中的可疑链接。不要点击朋友和家人发来的邮件中的奇怪链接。不要点击来自你的银行、谷歌或其他任何人的电子邮件中的链接，否则你可以登录一个网站来获取该电子邮件中的信息。
