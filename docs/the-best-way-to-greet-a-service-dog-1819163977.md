# 如何问候服务犬

> 原文：<https://lifehacker.com/the-best-way-to-greet-a-service-dog-1819163977>

服务犬有一项重要的工作要做，但这并不影响它们的可爱。如果你想和一只服务犬打招呼或抚摸它，请遵循以下指导方针，以正确的方式问候它们。

要知道，狗狗可能正在工作，或者需要保持“工作状态”,所以要做好准备，不要去抚摸它们。ICAN，一个帮助训练服务犬的组织，建议你不要打断，即使狗的行为不像你期望的大多数狗那样:

> 除非主人要求，否则不要试图用命令或用身体控制皮带或项圈来控制狗。服务犬可能会做一些看似不寻常的事情。狗的主人已经接受了训练，如何处理他们的狗，以及如何在公共场合安全有效。

### 向业主登记

走近主人，问是否可以抚摸这只狗。明确你想做什么，让老板知道你真正想要的是什么。像“我可以打个招呼吗？”让主人更难预料你的行动，给你正确的指示。

让他们很容易说不，比如:

*   “对不起，我知道你的狗可能正在执行任务。我可以抚摸它们吗？”
*   “嗨！如果你的狗不能休息一下，我能理解。我可以抚摸它们吗？”

*   “嘿那里！你的狗看起来像在工作，他们能休息一下让我抚摸他们吗？”

如果主人拒绝，感谢他们，让他们继续他们的一天。如果他们说是，问是否有一个特别的地方你应该抚摸。

如果你对你的孩子或自己的狗与服务犬的互动感兴趣，你可以遵循同样的指导方针。只要根据你的情况调整上述措辞，让孩子或狗远离，直到主人允许接近。

### 记住规则

遵循主人给你的任何指示。他们可能会让你在某个地点抚摸狗狗(可能是狗狗最喜欢的地点！)或者如果狗儿离开了他们指定的位置(比如坐着或躺着)，就停止抚摸。你也应该可以在简短的问候后走开，以免过度刺激狗。

听听主人告诉你的话，如果他们要求你停止抚摸他们的狗或者离开，不要把它当成是针对你个人的。他们知道什么最适合他们的狗，什么符合狗的训练。

### 不要激怒狗

让人们抚摸或问候服务犬的一个主要问题是，这会分散狗的注意力。轻轻地抚摸狗狗，用平静、愉快的语气和它们说话，帮助它们保持平静的状态，这样它们就可以轻松地回到工作状态。如果你是那种喜欢粗暴对待狗狗或者当你和狗狗打招呼时会让它们兴奋的人，抵制你的本能，保持冷静。

如果你有食物，甚至是狗食，不要在没有主人允许的情况下喂狗。如果你真的想给狗一个奖励，先问问主人，看看他们是否更喜欢你用他们的奖励而不是你自己的。

### 要尊重

服务犬的主人正在努力工作，就像你一样，所以尽量不要占用他们太多的时间。我知道，我们可以花一整天去抚摸可爱的狗狗，但是把你的问候时间控制在几分钟，然后继续前进。

不要质疑主人对服务犬的需求或者他们的残疾。服务犬帮助所有类型的人，而不是由你来决定某人是否需要一只狗或是否残疾(许多残疾是看不见的)。大多数主人会对你询问他们的狗持开放态度(正常的问题，如他们的名字或他们是否知道任何技巧)，但不要通过询问他们为什么需要这条狗或他们有什么残疾来侵犯主人的隐私。
