# 现在，您可以在优步之旅中增加多个站点

> 原文：<https://lifehacker.com/now-you-can-add-multiple-stops-to-your-uber-ride-1819880339>

当你和朋友外出过夜回家时，分享一个优步通常很有意义。然而，到目前为止，你必须口头告诉你的司机你要停几站，然后每次有人下车时输入一个新的地址。这是一个可行的过程，但是对你和车手来说都很令人沮丧。



[据 *Mashable*](http://mashable.com/2017/10/26/uber-multiple-stop-feature/?utm_campaign=Mash-Prod-RSS-Feedburner-All-Partial&utm_cid=Mash-Prod-RSS-Feedburner-All-Partial#X3h_SVfwAZqN) 报道，从今天开始，你可以在优步应用程序中为单程旅行添加多达三个不同的站点，这样你的司机就可以知道你和你的同伴想去哪里，而无需就路线进行漫长的谈判。

添加多个停靠点非常简单和直观。当您在“收件人”中键入第一个地址时框中，您只需按下框中的+号即可添加另一个地址。

添加后，您的司机将按照添加到应用程序的顺序停靠站点。这意味着你和你的朋友应该在你开始将地址添加到应用程序之前规划出什么顺序是有意义的，就像你(希望)在普通出租车上指路一样。

当你在乘车途中时，地址也可以被添加或删除(尽管这听起来像是激怒你的司机的一种快速方式)，并将根据你的司机的驾驶方向实时更新。

就像过去一样，你也可以在应用程序中与你的朋友分摊费用，这样你们就可以一起分担费用了。

使用该功能需要最新版本的优步应用程序。如果您看不到该选项，请尝试更新您的 access 应用程序(更新将在下周推送至所有用户)。
