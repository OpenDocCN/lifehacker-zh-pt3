# 这个互动网站帮助你帮助你最喜爱的濒危动物

> 原文：<https://lifehacker.com/this-interactive-site-helps-you-help-your-favorite-enda-1796339468>

是一个互动的环境保护网站，在这里你可以了解你最喜欢的动物，是什么杀死了它们，在这里你可以捐款帮助。

Watch

动物爱好者、环保主义者或地球纪录片爱好者都明白，一些真核生物已经活不长了。根据国际自然保护联盟的数据，污染、偷猎和侵略性耕作已经将成千上万的动物推向灭绝的边缘，其中包括 474 种濒危哺乳动物。

你肯定不能把它们都保存下来，但是《碎片》是一个了解它们的好地方。由设计师布莱恩·詹姆斯创作，展示了 30 种濒临灭绝的动物，所有这些动物都被美丽地描绘成丰富多彩、移动的多边形。华莱士的鸟翼拍打着橙色和黑色的翅膀，而肯普的里德利龟推动着自己，就好像骑在一股看不见的水流上。

最酷的部分？每个动物的多边形形状完全用 CSS 编码，每个点都由詹姆斯绘制。它们甚至不时闪烁，给它们带来生命，尽管它们代表着动物数量的减少。

> 所以，本质上——每个形状都被一组新的坐标变形、移动和玩弄，由于它们始终保持为三角形，这意味着 3 个点，用 CSS 过渡来连接运动。

每种动物都有一个简短的描述，后面是威胁其生存的因素。你可以找到一些统计数据，比如人口，栖息地的丧失，以及几十年来每年人口下降的图表。一些条目的特色是 YouTube 上的动物活动视频，以及偷猎和过度捕捞等事件的后果。

不过，也不全是坏事！该项目旨在提高对生命多样性及其脆弱性的认识，并给你一个帮助的机会。每种动物都链接到一个保护页面，在那里你可以捐款来支持你选择的物种。
