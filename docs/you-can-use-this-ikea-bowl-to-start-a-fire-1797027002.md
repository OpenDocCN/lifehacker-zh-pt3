# 你可以用这个宜家碗生火

> 原文:[https://life hacker . com/you-can-use-this-IKEA-bowl-to-start-a-fire-1797027002](https://lifehacker.com/you-can-use-this-ikea-bowl-to-start-a-fire-1797027002)

意外的火灾在很大程度上被认为是一件坏事，这就是为什么当宜家的这个非常闪亮的 [BLANDA BLANK](http://www.ikea.com/us/en/catalog/products/50057254/) 上菜碗导致一些葡萄——以及后来的一些报纸——化为乌有时，一些人非常惊慌。

 [https://lifehacker.com/embed/inset/iframe?id=fb-1476448169085045&autosize=1](https://lifehacker.com/embed/inset/iframe?id=fb-1476448169085045&autosize=1) 

但是，不像 [那些毁坏昂贵汽车、杀死竹子](https://gizmodo.com/a-brief-history-of-buildings-that-melt-things-1247657178) 的闪亮建筑，我认为这只碗的点火能力可以派上用场。(如果你需要火的话，那就是。)只要把它拿出来放到夏日灿烂的阳光下，在里面装满易燃的东西( [多力多滋？](https://lifehacker.com/use-doritos-or-other-chips-to-start-a-fire-5840698#_ga=2.35325153.1701094126.1500308789-949419976.1446553382) )，看它烧。当然，如果你只是想让你的碗盛食物，而不是不小心烧焦食物，也许可以选择一个不那么闪亮的碗，或者只留这个碗在室内使用。
