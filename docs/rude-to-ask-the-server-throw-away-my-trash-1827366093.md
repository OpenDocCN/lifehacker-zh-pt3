# 问咸味女服务员:让服务员把我的垃圾扔掉是不是很没礼貌？

> 原文：<https://thetakeout.com/rude-to-ask-the-server-throw-away-my-trash-1827366093>

> 亲爱的 Salty: 我有一个朋友——一个被称为“被抛弃的共和党人”的守旧派——他坚持把早上的垃圾邮件带到餐馆或餐厅吃午饭，然后要求我们的服务员在他看完垃圾后扔掉。
> 
> 服务员是如何看待这个习惯的？这让他们生气了，还是他们觉得无所谓？
> 
> 谢谢，尴尬的用餐者

亲爱的尴尬的，

恭喜你，这可能是我开始写这个专栏以来收到的最奇怪的问题。仅仅基于这一点，我必须回答它。

Watch

这个问题的核心实际上更具有普遍性:服务器的角色在哪里开始和结束？我们是来帮你点菜和送食物的吗？还是我们不止于此。以好客的名义，我们应该走多远？

让我们从你朋友的垃圾邮件请求开始。是的，很奇怪。是的，他的态度似乎很有道理。对他的服务员来说，把几张纸扔进垃圾箱或回收箱可能并不是很难，但令人讨厌的是他的态度。“喂，亲爱的，我用完了这些垃圾——你帮我把它们扔掉。”

我觉得不是邮件的问题，而是*男*的问题。有一种请求帮忙的方式是带有歉意的:“非常抱歉，但是有没有我可以把这些扔掉的垃圾桶？”比问“嘿，介意把我的垃圾扔掉吗？”如果你的朋友是某家餐馆的常客，在那里他和工作人员是好朋友，并留下丰厚的小费，这种情况可能会有所不同。不过，从你的信来看，听起来他只是在随意的餐馆里请你帮忙，这有点过分。我敢肯定工作人员总是说是的，但他们可能会在背后给他一个白眼。一般来说，我会建议不要把垃圾带进餐厅，因为工作人员稍后会清理这些垃圾。例如，如果你想带一份报纸，那就自己把它扔进垃圾桶或者带走。

我认为在这种情况下，人们应该问自己:我是把某人当成服务员还是仆人？我让他们做的这个任务和我的饭有关吗，或者我只是喜欢发号施令？我是否利用了员工为小费而工作，不太可能对我说“不”的事实？嘿，基本的人类礼仪警告:我会希望被这样对待吗？

如果你能回答这些问题，并且对你的问题不感到尴尬，那么你的要求就是合理的。但是那些利用对 [的“款待”来让服务者](https://thetakeout.com/ask-the-salty-waitress-what-should-i-do-when-my-creepy-1825752687) 毛骨悚然的人，对他们颐指气使，或者让他们做一些有辱人格的事情不是在请求款待，而是在滥用关系。记住，好客也意味着顾客应该表现得像一个“客人”有时，客人会向主人提出要求，但他们应该总是友好和尊重地提出要求。

最后——虽然你没有问，但这是我的专栏——但我要说你朋友开邮件的恶作剧对你也很无礼。(“什么，他是在谷仓里长大的吗？”我奶奶梅布尔会说。我不知道她为什么反对谷仓。)我想，坐在那里看着他翻看地毯清洁优惠券或西尔斯的销售通告，不会是你最喜欢的活动之一。也许这是你可以对他采取的策略？“嘿，乔治，虽然我很喜欢看最新的排水沟清理传单，介意下次再做邮件分类吗？”如果他继续是一个老怪物，也许是时候找一个新的午餐伙伴了。
