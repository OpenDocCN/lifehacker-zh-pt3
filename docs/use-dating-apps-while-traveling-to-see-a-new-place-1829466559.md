# 旅行时使用约会应用程序去看看一个新地方

> 原文:[https://life hacker . com/use-dating-apps-while-travel-to-see-a-new-place-1829466559](https://lifehacker.com/use-dating-apps-while-traveling-to-see-a-new-place-1829466559)

作为一名游客，你总是怀疑你得到的是一种非常定制和商业化的体验。有时候，那些炫目的、精心制作的东西可能很有趣，但是如果你想要更多，有一个很好的方法:和当地人约会。

Watch

这个建议的灵感来自个人经历，也来自黛比·肯特(Debbie Kent)为《卫报》撰写的一篇文章 ，她在文章中建议跟随陌生人，以了解当地人一整天会选择什么路线。她认为这是一种艺术练习，并试图表明她无意吓到任何人:

> 我告诉自己，我不是在跟踪那个穿粉色裤子的男人——我不知道他的名字，也不知道关于他的任何事情，当他离开街道的公共空间进入一个私人公寓楼时，我抛弃了他，自己找到了回中央共和国广场的路。

就我个人而言，我认为在人们不知情的情况下跟随他们是令人毛骨悚然的，尽管这在艺术中出现过很多次。行为艺术家维托·阿孔奇在六十年代做了几乎与相同的事情，甚至拍下了他的印记。格兰特说她的“跟随陌生人”项目是她合作的塞尔维亚艺术家米洛什 [托米契](http://www.milostomic.net/) 的指示的一部分。我不能把它作为一种策略推荐给大众，尤其是作为一个被陌生人跟踪的女人，这些陌生人可能正在做一件艺术品，也可能没有，但他们绝对让我感到害怕。

但是我可以推荐另一种从当地人的角度看这座城市的方式:和其中一个约会吧！这是我旅行时一直在做的事情。这并不总是容易的，但这可能意味着去你不知道的地方——，从酒吧，到餐馆，到远离主要街道的美丽街道。这里有一些建议，可以让你和了解自己所在城市的人安排一对一的旅行。

### 诚实

如果我在一个约会应用上寻找一个想和我喝一杯的人，我总是会在简历上注明我只会在这个城市呆很短时间。就时间和精力而言，去约会可能是一笔大投资。如果有人在寻找认真的东西，我想让他们知道，在他们决定见面之前，我只是路过。如果我们匹配，我有时会伸出手，只是重申我什么时候离开，什么时候有空见面，这样就更清楚了。

另一个行之有效的方法是，问问我去过的地方的熟人，他们是否有单身朋友想出去玩。是的，你可以请你的朋友一起去，但是我也遇到过他们不在的时候。他们可能还认识某个喜欢出去玩的帅哥。有趣的是，大多数对遇见陌生人的想法感到兴奋的人通常更自发，更乐于冒险。

### 安全

当你在一个新城镇时，同样的规则也适用于约会应用程序，只不过更是如此。在公共场所第一次遇见某人，你可以提前确认。是的，我经常和意想不到的人一起去一些地方，但是我总是试图先在我觉得安全的地方去了解他们。如果事情有进展，我会问他们姓什么，然后把他们的照片截图发给一个朋友。我知道这听起来不吉利，但现实是陌生人会很奇怪。我很幸运，从来没有遇到过需要这些安全预防措施的情况——但我仍然会采取这些措施。

### 求教

我遇到过不能或不想见面的人，但他们非常愿意告诉我应该检查什么。如果连续几个当地人告诉你去某个地方，你应该去。一般来说，他们知道什么对游客来说是有趣的，什么是旅游陷阱。他们也完全自愿地提供这个建议。人们经常想就如何享受他们喜欢的地方提出建议。我已经找到了一些人，他们会在某个晚上带我去约会，并建议我下次约会应该做些什么。那里有一些非常慷慨的人。

### 一夜情是很好的享受——或者跳过

我告诉过人们这个练习，并让他们中的大多数人认为我和我遇到的每个人都睡过。如果我*做了*，我不会为此感到任何羞耻。然而，我确实认为有些悲哀的是，大多数人都认为这是某人会见你的唯一原因。

是的，我的应用程序中肯定有一些非常直接的信息，只提供卧室之旅。不管人们怎么想，有更多的人只是想与陌生人见面，谈论他们的生活，并喜欢通过新的视角看他们的周围。在某种程度上，和来访的人约会对你来说也像是一个小小的假期。

Aimée Lutkin 是一名自由撰稿人，他在博客上写了很多关于约会的内容。她目前正在全国旅行，每到一个城市都会去约会。
