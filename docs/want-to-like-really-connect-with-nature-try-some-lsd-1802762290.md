# 想真正接触大自然吗？试试迷幻药

> 原文：<https://lifehacker.com/want-to-like-really-connect-with-nature-try-some-lsd-1802762290>

有没有想过为什么嬉皮士这么喜欢树？根据发表在《精神药理学杂志》上的 的一项研究，这可能要归功于他们在《燃烧的人》中吃的那一把蘑菇，该研究发现，一个人的迷幻剂消费体验与他们感知的与自然的联系之间存在(或许并不令人惊讶)的关系。

不过，也可能是相反的情况:这项研究并没有排除环保人士更有可能尝试像迷幻药这样的物质的可能性。

这项研究的作者马蒂亚斯·福斯特曼和克里斯蒂娜·萨吉奥卢认为，有理由相信迷幻物质有助于人们欣赏自然。像裸盖菇素(在 200 多种蘑菇中发现的迷幻化合物)这样的迷幻药，被发现可以改变大脑网络之间的连接，也与“自我溶解”效应有关，这种效应负责与自然世界的连接感，尽管研究人员无法得出结论，即有环境意识的人更有可能吃一些蘑菇。

这项研究依赖于一项调查，该调查使用了 1400 多人的自我报告结果，询问了他们对自然的亲近程度，他们的生态行为，以及他们对迷幻物质如 LSD，mescaline 和 psilocybin 的体验。

他们在研究中说:“人们对经典迷幻药的体验越多，他们就越喜欢花时间在大自然中，他们就越把自己视为大自然的一部分。”与自然的关联性增强也意味着服用迷幻药的参与者更有环保意识，并参与更多环保活动，比如节约用水和回收利用。

并不是每种精神活性物质都与自然相关性的增加有关。像大麻这样的物质，虽然传统上与环保团体有关，但对一个人对自然的感觉没有足够大的影响。至于自然对一个人摄入迷幻物质的可能性的影响，作者建议进行一项更长期的研究，跟踪人们对自然的看法，以及迷幻剂对他们的生态行为的影响。
