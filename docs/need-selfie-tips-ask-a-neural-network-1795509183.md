# 需要自拍小技巧？问问神经网络

> 原文：<https://lifehacker.com/need-selfie-tips-ask-a-neural-network-1795509183>

在我年轻的时候，我上传了一些非常不讨人喜欢的自拍照。我是一个摄影新手，一个留着胡子的业余摄影爱好者。我很高兴地说，我自拍的结果(当然，下面分享纯粹是为了教育目的)通过经验得到了改善，但如果我有一个无情的机器人告诉我哪里出错了，事情会容易得多。幸运的是，机器学习的魔力现在就在我们身上，它在这里告诉我们如何拍出好的自拍。



多亏了斯坦福大学的博士生安德烈·卡帕西(Andrej Karpathy)和一个神经网络的帮助，你可以看到好的自拍和坏的自拍的区别，并找出为什么一些自拍流行而另一些却失败了。

卡帕西从一个  开始，这是一个旨在模仿视觉皮层的人工神经网络。然后，他向网络上传了 200 万张自拍照，根据关注者和喜欢的数量将它们分为“好”和“坏”自拍照。Karpathy 随后使用了一个新的数据集，其中有 5 万张网络看不到的自拍，并让它对它们进行相应的分类。

最佳自拍和最差自拍之间的差异非常明显，但在每一类自拍中，你都应该记住一些模式。

### **如何拍好自拍**

 [https://lifehacker.com/embed/inset/iframe?id=instagram-BSv0ugdgnoU&autosize=1](https://lifehacker.com/embed/inset/iframe?id=instagram-BSv0ugdgnoU&autosize=1) 

**让你的头发散开:**炫耀你飘逸的头发的照片一般比头发部门缺少的照片排名高。在排序后的数据集中，前 100 张自拍照中没有一张是男性的，但男性的前 100 张自拍照表明发型绝对流行。

**褪色:**与不饱和的照片相比，面部轻微褪色和过饱和的照片看起来更有吸引力，也更均匀。最好的自拍照是明亮的，而最差的自拍照总是在光线下。

滤镜，滤镜，滤镜:几乎每张排名靠前的自拍都有滤镜覆盖，大多数滤镜会降低对比度，淡化图像本身。或许是因为那怀旧的表情吧？

**去掉额头:**不管是有意还是无意，令人惊讶的是，大量“好”的自拍照漏掉了主体的一大块头部。与你想象的相反，最好的自拍并不是最淫秽的。事实上，裸露的皮肤数量和自拍质量之间几乎没有关联。

### **如何拍出糟糕的自拍**

 [https://lifehacker.com/embed/inset/iframe?id=instagram-BHM_5OLAooI&autosize=1](https://lifehacker.com/embed/inset/iframe?id=instagram-BHM_5OLAooI&autosize=1) 

**补照片:**我们拿到了，是自拍。但是如果你的头看起来像要吞没整个画面，也许把你的相机向后移一点，但是记得把你的头向前和向下倾斜一点。留点空间，哪怕只是为了让全世界知道你有脖子。

**合影:**人多就等于赞多，对吧？不完全是。只有一张团体自拍在前 100 名，而后 100 名大多数都是团体自拍。如果没有一些轻微的人体扭曲，要想获得合影的正确角度也是相当困难的。

**黑暗中拍摄:**光线不好的照片很烂，自拍也不例外。最差的照片总是很暗或曝光不足，结果充满了干扰主体的噪音。
