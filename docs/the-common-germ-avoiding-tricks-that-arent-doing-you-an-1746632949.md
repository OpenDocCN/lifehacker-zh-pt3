# 对你没有任何好处的常见细菌避免技巧

> 原文：<https://lifehacker.com/the-common-germ-avoiding-tricks-that-arent-doing-you-an-1746632949>

当有人咳嗽时，你屏住呼吸过吗？或者试着用脚冲马桶？如果你用正确的方法去做，这些技巧中的一些*可能会有帮助，但是它们通常是不必要的，并且大多是基于偏执狂。以下是日常生活中避免细菌的真相。*



### 当有人咳嗽或打喷嚏时屏住呼吸

你在医生办公室的候诊室里某个小孩决定把鼻涕喷到空气中，好像他根本不在乎，所以你的反应是闭上嘴，屏住呼吸，直到“空气变清”这有助于你避免候诊室里的人被恶心到吗？简而言之，没有。根据驻院护士梅根·加里贝(BSN 注册护士)的说法，你不可能屏住呼吸足够长的时间来避开悬在空气中的鼻涕和唾液滴。

如果有的话，当你吸第一口空气时，你要屏住呼吸足够长的时间，让水滴接近并被吸入。哎呦。也就是说，加里贝建议，如果你能屏住呼吸迅速离开该区域，你可能会得到一些保护。当然，如果你那样做，你也会看起来像一只刚刚叼走一堆坚果的花栗鼠，所以这是你的选择。用一块布盖住你的鼻子和嘴巴可以防止你吸入飞沫，但它需要在那里停留一段时间才能发挥作用，如果它不干净，你只是把不同的细菌带到了你的脸上。此外，加里贝解释说，尽管有总的因素，屏住呼吸可能并不是一开始就必须的:

> 另一方面，你的身体实际上对此类事情有自我防御机制——就像你的鼻毛。这些鼻毛起着第一道过滤器的作用，在细菌进入你的系统之前(希望如此)将它们截留。你的肺也有类似的防御机制。大多数情况下，如果你感染了某种疾病，它不会通过吸入飞沫传播(也有例外，但在日常生活中，你不会通过这种方式感染)。

当然，例外是你的免疫系统比大多数人的弱。在这种情况下，当你知道你会和其他病人在一起时，你可能应该戴上口罩或其他类型的保护措施。

### **使用马桶盖，用脚踢来冲水，以及“悬停”**

公共厕所是大多数人联想到最肮脏的细菌的地方之一。任何处理大量人类排泄物的地方一定是细菌的噩梦王国，对吗？一想到光着屁股和天知道有多少人坐在同一个座位上，对我们大多数人来说，够够得着薄薄的卫生纸马桶盖了。

然而，正如加里贝解释的那样，它们主要是为了内心的平静，徘徊只会让你不舒服。你身体的任何部位都不会让你生病(比如嘴、眼睛、鼻子、手等等)。)应该靠近你在马桶上休息的地方。即使马桶座圈充满了细菌，你的皮肤和免疫系统的基本防御也几乎覆盖了它。其他疾病如疱疹、肝炎和其他严重感染呢？加里贝解释道:

> 疱疹会要求目前有活跃爆发的人在整个马桶座上摩擦所述爆发区域，然后要求其他人在整个马桶座上摩擦容易“感染”疱疹的区域，即使这样，他们也更有可能在感染疱疹之前获得另一种不太持久的感染。肝炎通常是血液传播的(或者体液传播的)，所以从马桶座圈传染的唯一方法是让某人坐在马桶座上，在某人在马桶座圈上流血后马上有一块皮肤破损的区域...大多数人不会坐在血淋淋的马桶座上，尤其是如果他们皮肤破损的话。

有一种肝炎可以通过粪便传播(甲型肝炎)，但通常是由摄入污染的食物和水引起的。浴室里真正令人担心的是你手上的东西，而不是屁股上的。而且，正如加里贝指出的，你的手通常不会靠近马桶座圈，除非你在放马桶座圈盖。现在，如果你身体的任何部位接触到座椅有割伤或开放性伤口，那就完全是另一回事了。在这种情况下，你绝对应该伸手去拿马桶盖或者想办法悬停。也就是说，除此之外，你应该做些事情来包扎或覆盖你的伤口，即使你觉得裤子有点保护你。

当你冲马桶的时候，你可能会觉得需要用脚去碰把手。自动冲水器已经使这个问题变得不那么严重了，但是对于旧的、昏暗的浴室来说，这还不算太牵强。使用你的脚 [在技术上可以避免你手上的细菌，如果你正常使用冲水器](https://vitals.lifehacker.com/stop-flushing-with-your-foot-germ-avoidance-tricks-t-1690333125) ，但是你需要考虑接下来会发生什么。很多人摸着马桶把手冲水，然后*摸着把手走向马桶*。当你一碰到厕所门的把手，你就基本上碰到了厕所的把手，所以你还不如正常冲水。另外，如果你用你的脚，马桶把手上的细菌现在就在你的鞋子上，你可能会也可能不会带着它在家里的地毯上走。

总而言之，即使你碰巧碰到了马桶座圈、冲水把手或门把手，*如果你上完厕所后洗手，这些都没关系*。加里贝怎么强调这一点都不为过，她指出，她想不出你能从马桶座圈上抓到任何东西，而这些东西不会因为洗手而消失。事实上，比起厕所本身，你更害怕厕所门上的把手。洗完手后，你必须使用别人用过的门把手(颤抖)*而不是*洗他们恶心的手。如果你的本能是抓起一张纸巾，并把它作为一个障碍，好主意。任何你能立即扔掉的东西对关水龙头、开门都有好处，而且它能让你的手保持更长时间的干净。然而，手帕或任何你不扔在身上的东西都不好。你可以用手帕暂时保护你的手，但是当它进入你的口袋时，细菌仍然有机会到达你的手和脸。

### **用手背**

你的手是细菌进入危险区域的主要途径，比如眼睛、耳朵、鼻子和嘴巴。所以有些人试图用手的其他部位来避免这种危险。例如，我用手背而不是手指按人行横道按钮。这是我多年来一直在做的事情，尽管我不知道它是否有帮助。所以我问加里贝，说到用手，总是有风险的

> ...解决你用手背按人行横道按钮的问题，真的有帮助吗？是也不是。从技术上来说，这比用手的前部击打要好，但用手肘击打更好。令人担忧的是细菌会转移到你的脸上——你的手，甚至是手背，一天会多次接触你的脸，而你并没有真正意识到这一点，因为这只是一种习惯(你拂去脸上的头发，你抓痒，你摩擦眼睛或嘴巴或鼻子等)。相比之下，你的手肘通常甚至不会靠近你的脸。

这和你被告知对着手肘窝而不是手打喷嚏是一样的道理，但是反过来了。当你对着手打喷嚏时，无论何时你接触另一个表面或者和别人握手，你都会传播细菌。很少有人会碰你的手肘，包括你自己，所以这样会安全得多。 [最近的研究表明](https://vitals.lifehacker.com/skip-the-handshake-and-fist-bump-instead-to-keep-from-g-1735960839) 你传播的细菌数量与接触的表面积和接触的持续时间直接相关。基本上，一个长时间的握手会比一个快速的拳头碰撞分享更多的细菌。然而，加里贝指出，不管你选择如何问候他人，保持良好的手部卫生仍然很重要。在职业场合，撞拳头并不理想，所以你最好随身携带一些洗手液。

### **在公共场合戴手套和外科口罩**

因为你的手是避免细菌的主要关注点，你可能会认为戴手套是一个简单的解决办法。如果你乘坐公共交通工具，手套可能看起来是一个特别好的主意，因为你会抓住许多杆子、栏杆、把手等等。不幸的是，手套只有在你直接接触体液(血液、呕吐物、便便、尿液和其他有趣的东西)时才有用。

加里贝解释说，手套只能防止你短时间接触细菌。一旦你脱下它们，你的保护就会崩溃。当你拿掉它们的时候，你可能会碰到它们的外面。你还记得上一次洗冬季或驾驶手套是什么时候吗？另外，当你戴着手套的时候，你仍然可能不经意地用手套碰到你的脸。或者你会拿着你的手机，只是为了稍后用你的赤手触摸你的手机(或者当你打电话时，把你的脏手套和手机都拿到你的面前)。除非你的手很冷，否则你最好洗手。

另一方面，医用口罩有更多的优势。它们对那些免疫功能严重低下的人非常有益，可以阻止 Weiswasser 博士解释的导致呼吸道病毒(如感冒、支气管炎和流感)的飞沫。然而，对我们大多数人来说，避免直接接触病人足以保证我们的安全。当你被困在飞机上几个小时 时，你已经有足够的保护来抵御你与其他人一样呼吸相同空气而感染的细菌，甚至是 [。真正的危险是触摸他们的唾液或鼻涕，或被这些东西覆盖的物体，如托盘桌和扶手，然后触摸你的面部(如眼睛、鼻子、耳朵和嘴巴)。韦斯瓦瑟博士指出，如果你为了内心的平静而必须戴手套和口罩，你最好骑自行车穿过一次性物品。但即使这样，他解释说，这是非常极端的，没有必要，除非是一个非常精选的人群。然而，如果*生病的人是你的话，医用口罩会非常有用。在许多国家，生病时戴口罩是很常见的做法，这样可以防止传染给他人。*](https://io9.gizmodo.com/why-you-get-sick-on-planes-and-how-to-keep-it-from-ha-1661753286) 



### **细菌无法完全避免(没关系)**

底线是你不能真正避免细菌...但那完全没问题。从你出生到死亡，你无时无刻不被他们包围着。加里贝指出，并非所有的细菌都是坏细菌(许多是非致病性的)，大多数坏细菌永远不会真正有机会伤害你:

> 你在一天中接触了如此多的细菌，你甚至没有意识到。不过，如果你很健康，你的免疫系统在抵御入侵者方面相当厉害。他们来去都很低调。

你的身体和细菌的战斗只是照常进行。事实上，限制你的身体接触细菌有可能弊大于利。当你的免疫系统遇到从未见过的入侵者时，你会生病，因为你的免疫系统太弱，无法立即根除它。“”卫生假说表明，通过暴露于细菌中来适当调整和增强你的免疫系统是可能的，因此这种情况发生的频率会更低，但维斯瓦瑟博士承认这一理论有点争议。尽管如此，如果你是一个完全健康的人，还是会有“太干净”这样的事情。加里贝解释说，虽然你可能不应该*故意*让自己暴露在细菌中，但正常的暴露对于建立一个健康的、功能正常的免疫系统是必要的。就像肌肉一样，如果不使用它，它就不会变得更强壮。这对于因过度使用抗菌产品而不断涌现的新细菌尤为重要。



### **你应该真正关注的策略**

避免细菌和保持健康其实很简单。这个世界并不像它看起来那样是一个充满细菌的雷区，让疾病远离并不需要一个不断平衡的诡计和回避策略。维斯瓦瑟博士和加里贝都有一些建议，可以帮你完成大部分艰巨的工作

*   洗手:在你吃东西、做饭或触摸你的脸之前，洗手是你能做的避免致病细菌的最重要的事情。
*   **不能洗手时使用洗手液:**加里贝建议正确使用洗手液的关键是*记住实际使用它*。如果你在吃饭前或擤鼻涕后忘记把它拿出来，把它放在身边也没什么用。当你使用它时，摩擦你的双手直到它变干，至少 15-20 秒，覆盖你的手的正面和背面以及手指之间的所有表面。然而，如果你的手看起来很脏或很油，那么就需要好好洗手来清除这些脏东西。
*   避免触摸你的脸:这并不容易做到(我一直在为此挣扎)，但你越少触摸你的脸和你脸周围脆弱的开口(眼睛、耳朵、鼻子、嘴巴)，你就会越好。摸别人的脸也是一样。
*   远离生病的人:如果你和他们住在一起，这可能很难做到，但逻辑是合理的。你越少和病人在一起，你就越不可能感染他们所患的疾病。生病的时候也是一样！当你感觉不舒服时，不要去上班或去公共场所。没有必要传播一个讨厌的错误来证明你能处理得多好。
*   **关注基础知识:**维斯瓦瑟博士强调了 [充足的睡眠、良好的饮食、锻炼](http://lifehacker.com/how-to-most-effectively-battle-the-common-cold-5686387) 到 [保持你的免疫系统强壮健康](https://lifehacker.com/the-no-bs-guide-to-boosting-your-immunity-and-avoiding-5858209) 的重要性。

差不多就是这样！你身体的自然防御系统会处理剩下的事情。最后但同样重要的是，Weiswasser 博士建议你和你的家人及时了解最新的免疫接种情况，他还建议六个月以上的人每年秋天都要注射流感疫苗。
