# 如果舞台上出了问题，你该怎么办

> 原文：<https://lifehacker.com/what-you-should-do-if-something-goes-wrong-on-stage-1824075513>

在周六的“为我们的生命而游行”中，马乔里·斯通曼·道格拉斯高中学生萨曼莎·富恩特斯，枪击惨案的受伤幸存者， [登上舞台，向数千名抗议者](https://www.youtube.com/watch?v=dw6hbACrP38) 发表慷慨激昂的演讲。她的演讲进行到一半时，她蹲在讲台后面呕吐起来。



人群变得安静下来，当摄像机转向别处时，空气中弥漫着担忧的低语声。不到一分钟后，富恩特斯再次出现，微笑着喊道:“我刚刚在国际电视上吐了，感觉棒极了！”当她重新开始演讲时，观众爆发出欢呼声和掌声。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-dw6hbACrP38&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-dw6hbACrP38&start=0) 

现在，很少有什么比在数百万人面前体验无意识的身体机能更令人尴尬的了。尽管如此，十几岁的富恩特斯知道战胜可能是她紧张和创伤的结果的秘密: [你拥有它](https://lifehacker.com/how-can-i-stop-getting-so-embarrassed-when-i-do-somethi-1695888523) 。你举手说，“是的，那是我！”你说了一句俏皮话，让每个人都知道你很好，承认发生了什么是没关系的，并且一起大笑，因为你也参与其中。然后你会回到你正在做的事情上。

现在，富恩特斯的演讲非同寻常，有几个原因，我们不是说重点应该放在她生病上。但是我们确实认为她给我们上了重要的一课——逆境中的优雅——这一课我们可以用在现实生活中。

在大多数情况下，在舞台上呕吐的可能性非常小。但总有出错的可能:你的幻灯片停止播放，你不自觉地对着麦克风打了个嗝。如果你在舞台上遇到尴尬(甚至糟糕)的事情，记住富恩特斯告诉我们的:我们都是一个社区的一部分。对你来说尴尬的时刻对观看的人来说同样尴尬。他们处于一种既担心又想笑的状态。当你通过承认所发生的事情为他们提供一个出路时，他们会因此而喜欢你！更不用说，他们会对你坚持追查真相印象深刻。

信不信由你，克服舞台上的尴尬时刻对你来说实际上比它对你的伤害要大得多。也许只是一场精彩的演讲，现在也是你良好的幽默感、你的坚韧以及你在压力下如何处理自己的一个例子。所以，如果出了问题，不要担心:只要以富恩特斯为榜样，拥有它。
