# 小心泽尔骗局

> 原文：<https://lifehacker.com/watch-out-for-zelle-scams-1825526486>

到目前为止，大多数用户都意识到了像 Venmo 和 PayPal 这样的支付系统的安全弱点:你应该只向你认识的人汇款，你不应该用它来购买产品或服务，你绝对不应该用它来让你不认识的人 支付 [，他们可以在交易完成前简单地取消交易。事实证明，](https://lifehacker.com/how-to-not-get-scammed-when-selling-things-online-1825190225) [Zelle](https://lifehacker.com/zelle-is-your-new-friend-paying-service-1796061030) ，一个最近被银行用来与 Venmo 竞争的支付系统，也有一些严重的漏洞。



根据《纽约时报》[](https://www.nytimes.com/2018/04/22/business/zelle-banks-fraud.html)*的报道，Zelle 与其竞争对手相比具有吸引力的一些关键因素——包括美国银行、花旗和摩根大通在内的一些最大的银行使用它，以及它汇款的速度——使它非常容易受到欺诈。*

*“网络内的现金转账通常在几秒钟内完成——比大多数竞争对手的支付服务要快得多，”《纽约时报》报道。"这加大了银行阻止或逆转非法交易的难度."而且它并不总是在交易完成时通知客户，这意味着你可能意识不到任何欺诈行为正在发生，直到为时已晚。*

*此外，该系统需要一个电子邮件或电话号码来汇款。根据《纽约时报》的轶事，这使得骗子可以用你的电话号码注册，代替你接收现金转账。银行的安全措施各不相同，有些银行“当新的收款人与他们的 Zelle 账户关联时”不会通知客户。虽然 Zelle 表示客户不对欺诈行为负责，但它并不认为客户“故意”汇款是欺诈行为，即使预期的接收者没有收到钱。它的网站声明 Zelle 和使用它的银行都没有提供 [保护计划](https://www.zellepay.com/support/im-unsure-about-using-zelle-to-pay-for-goods-or-services-from-someone-i-dont-know) 。*

*有些骗局要复杂得多。下面是发生在宾夕法尼亚州唐宁镇富国银行客户简·巴特勒身上的事。根据第*次*的报告:*

> *骗局是精心策划的。首先，一封看似来自富国银行(Wells Fargo)的网络钓鱼邮件诱使她将自己的银行 ID 和密码输入一个欺诈网站。第二天，巴特勒女士接到一个似乎来自富国银行欺诈部门的电话。她看到手机屏幕上显示的号码与她银行卡背面的号码相符——但电话那头不是她的银行。这个叫 [的已经被冒名顶替了](https://www.nytimes.com/2011/11/23/us/whos-on-the-line-increasingly-caller-id-is-duped.html) 。*
> 
> *打电话的人骗她交出了进入 Zelle 的一次性密码，然后用这些密码从她的账户中进行了六次转账，金额从 1 便士到 999.98 美元不等。富国银行退还了巴特勒女士的损失。*

*其他的 P2P 支付服务则相当标准:一位女士以为她在买演唱会门票，但在她转账后被“卖家”鬼鬼祟祟地跟踪了(在你手里有商品之前，千万不要汇款)。在另一个事件中，骗子在网上访问了某人的银行账户，并使用 Zelle 将钱从账户中转出。*

*客户被盗资金是否能得到补偿取决于银行和具体情况。*

*《泰晤士报》报道称，问题的严重程度还不得而知，“因为 Zelle 还是一个新生事物，银行并没有报告太多关于它的数据。”*

*像 Venmo 和 PayPal 这样的其他点对点支付系统也面临着类似的挑战，事实上，PayPal(拥有 Venmo)最近 [就其安全措施与美国联邦贸易委员会](https://www.ftc.gov/news-events/blogs/business-blog/2018/02/venmo-settlement-addresses-availability-funds-privacy) 达成了一项诉讼。*

*一个安全措施可能会有助于防止有人进入你的账户:将应用程序连接到你的信用卡而不是银行账户。你得到了更好的保护，如果需要，你可以取消交易。*
