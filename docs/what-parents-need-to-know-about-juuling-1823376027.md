# 关于玖零，父母需要知道什么

> 原文：<https://lifehacker.com/what-parents-need-to-know-about-juuling-1823376027>

现在各地高中家长通讯中最大的话题之一就是朱尔。这是一个流行的电子烟系统，看起来很像 USB 闪存驱动器——你可能在你十几岁的房间里见过一个，以为它包含一篇关于*了不起的盖茨比的文章。*根据 Twitter 和 Instagram 上标签为#doit4juul 的帖子，学生们随时随地都在“胡闹”——在学校的浴室里，在图书馆里，在课堂上的课桌下。他们侥幸逃脱，因为该设备是谨慎的，“云”可以隐藏在一个袖子里，蒸汽闻起来甜而有果味，不像香烟烟雾。在青少年中，电子烟 [比传统香烟](https://www.nytimes.com/2017/12/14/health/teen-drug-smoking.html) 更受欢迎，目前朱尔品牌是王者。

以下是家长们需要了解的关于当前电子烟的信息:

### 它看起来像什么？

朱尔由一个细长的蒸发器和一次性尼古丁汁“包”组成。这种豆荚有芒果味、焦糖布丁味和黄瓜味。一些人称朱尔为“vapes 的 iPhone ”,因为它时尚简约的设计。它可以在笔记本电脑或其他 USB 端口上充电。

### 它是如何工作的？

该设备加热尼古丁汁液以产生蒸汽，该蒸汽被使用者吸入。

### 这是给谁的？

朱尔 [将自己](https://www.reddit.com/r/juul/comments/7w0mf6/official_juul_update/) 描述为“一家使命驱动的公司，寻求通过为成年吸烟者提供真正的替代选择来消除香烟。”该公司在其网站上声明，朱尔“不适合或不适合年轻人、前吸烟者或从不吸烟者。”在 JUULvapor.com 官网购买朱尔产品，你 [必须 21](https://support.juulvapor.com/home/learn/age-policy) 并经过一个 [年龄验证过程。在商店里，你必须达到你所在州购买烟草或尼古丁产品的法定年龄。但是青少年正在想方设法得到它们，比如使用假身份证，在不受监管的网站上购买，或者认识一个松懈的加油站服务员。](https://support.juulvapor.com/home/orders/age-verification/21-questions)

### 它含有多少尼古丁？

每个烟荚含有大约相当于一包香烟 或 200 口烟的尼古丁量 [。具体来说，每荚的尼古丁浓度为 59 毫克/毫升，是其他 vaping 产品的两倍以上。它给用户一个强有力的，快速的打击。](https://support.juulvapor.com/home/learn/faqs/juulpods-juice)

### 朱尔对未成年人有多危险？

对于成年人来说，vaping 通常被认为是香烟的更健康的替代品，因为它不会燃烧烟草并释放致癌物质。(根据 FDA 的规定，朱尔“不能也不会宣称其产品比香烟危害更小或更安全。”)

但是任何电子烟产品都不应该被认为是“安全的”，尤其是对未成年人来说。青少年的大脑仍在发育，所以他们特别容易上瘾。尼古丁非常容易上瘾，青少年时期接触尼古丁已被证明对大脑发育有的长期影响。它可能会影响青少年的行为、注意力、记忆力和学习能力。

对于那些吸烟的年轻人来说，还有其他的健康问题。根据发表在 [《美国呼吸和危重护理杂志](http://www.thoracic.org/about/newsroom/press-releases/resources/mcconnell-and-ecig-and-teenlungs.pdf) 上的一项研究，使用电子烟的青少年患持续咳嗽、支气管炎、充血和粘痰等呼吸系统症状的可能性是不使用电子烟的青少年的两倍。有证据表明，青少年吸烟者比不吸烟者更有可能开始吸烟。

此外，许多青少年不知道他们的 vape 里有什么。在国家药物滥用研究所赞助的一项研究 中，当高中生被问及他们认为他们吸的最后一种产品中有什么时，大多数人说“只是调味品”至于朱尔，一些青少年正在用其他物质填充豆荚，这甚至更危险。

### 你如何判断你的孩子是否在开玩笑？

普遍使用电子烟的迹象包括口渴增加(设备中的丙二醇会导致口干)，对咖啡因的新敏感性和可能的鼻出血。

无论你怀疑他们是否在使用电子烟，与他们展开一场关于电子烟的对话是非常重要的。对于朱尔，你可以用一个一般性的、非威胁性的问题开始——“嘿，我刚刚读到这个东西。你听说过什么？”—看看他们怎么说。解释电子烟的严重健康风险(记住痰、流鼻血、肺部问题、混乱的大脑和成瘾，仅举几例)。如果你的儿子或女儿有 vaping，让他们知道你不赞成，并和他们一起用积极的 取代不健康的行为 [。电子烟包含的不仅仅是有趣的水果味，当青少年使用它们时，可能会有持久的后果。](https://drugfree.org/article/reinforce-the-positive/)
