# 每个汽车制造商认证二手车项目的终极指南

> 原文:[https://jalopnik . com/the-ultimate-guide-to-every-automobile makers-certified-pre-ow-1820391875](https://jalopnik.com/the-ultimate-guide-to-every-automakers-certified-pre-ow-1820391875)

购买一辆 [认证的二手车](https://jalopnik.com/what-is-a-certified-pre-owned-car-1595837186?rev=1510585208152) 是获得二手车价值的好方法，同时也能最大限度地降低风险。认证保修范围和持续时间因品牌而异，这个方便的指南在你购买下一辆车时会非常有用。

*汽车新闻数据中心*汇编了一份 PDF 格式的 [难以置信的全面指南](http://www.autonews.com/assets/PDF/CA113029118.PDF) 介绍每一家汽车制造商的认证二手车项目，甚至包括像 SMART 和劳斯莱斯这样不知名的品牌。

我之前提到过，为一些品牌的认证车支付额外费用是不值得的 ，尤其是那些以可靠性著称的汽车。如果您仔细查看承保范围和承保时间，情况就更是如此。例如，斯巴鲁只延长了动力系统的覆盖范围，这通常是不会坏的东西，从最初的服务日期起七年和 100，000 英里。如果你想从斯巴鲁获得额外的保险杠到保险杠的覆盖范围，那你就要付出代价。

假设你正在购买一辆二手雷克萨斯，因为他们的计划提供从原始服务日期起六年的无限英里保修，这将是值得购买一辆比20142014、英里数更多的 2015 年款汽车，因为保修的剩余时间比英里数更重要。

有大量的信息需要消化，但在比较您购物清单上的汽车的认证二手车计划时，将所有信息集中在一个地方将节省您大量的时间。

该报告不仅涵盖了每个 CPO 计划的条款和条件，还指出了每个经销商认证一辆汽车的费用范围，以及如果您需要使用 CPO 保险，您将面临的任何免赔额。

你会注意到，每个项目都有不同的标准。一些要求一个干净的车辆历史报告，这意味着没有事故，而其他人会证明汽车有轻微的外观损坏。另一个有趣的趣闻是，看看哪些品牌使用第三方来认证他们的汽车，哪些品牌在内部进行认证。
