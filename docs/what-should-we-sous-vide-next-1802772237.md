上周的面包项目让我很难过，让我满嘴都是羞耻的口香糖，所以我不介意我们这周挑一个好吃的。一如既往，我有一些建议:

*   我知道我说过不要再吃面包了，但这次可能会不一样。(我希望它与众不同。)
*   **发酵泡菜:**还是酸菜？或者其他发酵的东西？
*   一道精致的海鲜:扇贝？或者虾？
