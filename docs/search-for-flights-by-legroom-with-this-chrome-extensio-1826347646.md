# 使用此 Chrome 扩展按伸腿空间搜索航班

> 原文：<https://lifehacker.com/search-for-flights-by-legroom-with-this-chrome-extensio-1826347646>

上周，在我从伦敦飞往旧金山的 11 个小时的飞机上，我坐在一对身高都超过 6 英尺的父子旁边。身高 5 英尺 6 英寸的我的膝盖轻轻拍打着面前的椅背。他们没有机会。

Watch

众所周知，飞机座位很小，除非你是乘坐头等舱(有时即使你是)，否则无论你坐在哪里，你都不会有一个宽敞的乘坐空间。

说到腿部空间，并不是所有的航空公司和飞机都是平等的。有时，当你有两个价格相似的航班时，一家航空公司会提供给你比另一家更多的房产。

如果腿部空间是你担心的问题(可能我们所有人都担心)，谷歌航班 Chrome 扩展的 [腿部空间可以帮上忙。一旦安装完毕，每当你在谷歌上搜索航班时，座位的平均伸腿空间就会出现在你所有的搜索结果中。](https://chrome.google.com/webstore/detail/legrooms-for-google-fligh/nhonfddkgankhjilponlbdccpabaaknp) 

如果你在搜索头等舱和商务舱的座位，这个扩展还会让你知道你是在寻找可以斜躺的座位还是平躺的座位。

延长不会让你找到更好的座位，但它会让你很好地了解你会让自己陷入什么。
