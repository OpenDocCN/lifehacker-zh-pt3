# 通过使用数据科学选择您的角色来赢得马里奥赛车

> 原文：<https://lifehacker.com/win-mario-kart-with-the-15-best-character-kart-tire-com-1826778879>

一天晚上，在我的教会青年小组，我认为我玩了我一生中最好的马里奥赛车游戏。不知何故，控制器完美地诠释了我的动作，比平时更好——我根本没有撞车或走错路！最后，我意识到我实际上一直在看对手的屏幕，而我从来没有意识到我实际上没有控制它。与此同时，我的角色刚刚随机倾斜到赛道的一侧。



我在马里奥赛车上从来没有进步过。但是*你*可以提高，这要归功于 [数据科学家亨利·欣内菲尔德对马里奥赛车角色选择菜单中每种可能组合的](https://medium.com/civis-analytics/the-best-mario-kart-character-according-to-data-science-7dfb65d4c18e) 的检查。

Hinnefeld 计算了角色、卡丁车车身和轮胎样式的最佳组合。虽然有 149，760 个连击，但是很多选项给了你相同的属性。一些卡丁车和轮胎在功能上是相同的，一些角色也是如此，这些角色通常可以按重量分类。只有 294 种不同的统计数据组合。其中，15 个优于所有其他的。

你可以使用除了金属马里奥和桃金娘之外的任何角色来获得一个最佳效果。你的选择取决于你想要速度还是加速度。为了获得最大加速度，选择一个婴儿角色，骑自行车，使用滚轮轮胎。为了获得最大速度，选择最重的角色等级(瓦里奥，布瑟，莫顿·库帕和米伊·重)，骑金标，使用光滑的轮胎。为了在两者之间取得平衡，从 [欣内菲尔德的图表](https://medium.com/civis-analytics/the-best-mario-kart-character-according-to-data-science-7dfb65d4c18e) 中选择其他组合之一。

Hinnefeld 在底部列出了所有的变体，因此您可以选择具有最佳“味道”的组合，同时仍然优化性能。例如，Wario 的黄金标准与光滑的轮胎完全一样，布瑟的运动跑车与赛博光滑的轮胎。

当然，这只是优化了你的潜力。您的表现还取决于您的演奏能力，以及您是否在看正确的屏幕。
