### ScreenToGif 应用程序非常适合制作 Gif

我不记得在哪里偶然发现了 [ScreenToGif](http://www.screentogif.com/) ，但是我爱这个应用。当你第一次打开它时，你会看到一个简单的菜单，展示了该应用程序的四个关键功能:录制你的屏幕并将其转换为 GIF，将你在网络摄像头中制作的傻脸转换为 GIF，启动数字白板并将你的涂鸦转换为 GIF，以及一个通用编辑器，你可以使用它将现有文件(如 MP4 视频)转换为 GIF。

对于屏幕记录，点击红色的大记录器图标，一个有趣的小窗口将会出现——一个图片框，你可以在你想要捕捉的区域周围调整大小。设置你想要的每秒帧数，点击 F7 开始录音，点击 F8 停止。

一旦你停止录制，ScreenToGif 就会启动它的编辑器，自动加载你刚刚捕捉到的内容。GIF 编辑器有点像一个简化的视频编辑器——因为 GIF 和视频都以相似(但相同)的方式 创建运动 [的外观。](https://www.reddit.com/r/explainlikeimfive/comments/210idt/eli5the_difference_between_animated_gifs_and/)

你可以一帧一帧地检查你潜在的 GIF，并在最终产品中修剪或删除你不想要的部分。您还可以添加或减少每帧之间的时间量(以缩短或延长 GIF 的速度)，四处移动帧，添加过渡和幻灯片，并为 GIF 添加机智的文本标题或其他叠加元素。

当你准备好创建你的 GIF(文件>>另存为)时，你可以使用各种选项对它进行编码(包括 [FFMPEG](https://medium.com/@colten_jackson/doing-the-gif-thing-on-debian-82b9760a8483) 和 [Gifski](https://gif.ski/) )。每一种都会给你不同的质量和文件大小的组合，所以一定要尝试几种，找出最适合你的 GIF 制作需求的。以下是我分别使用 1.0、2.0 和 Gifski 编码器创建的一些示例:
