# 你仍然可以向个人退休帐户缴纳 2017 年的税款

> 原文：<https://lifehacker.com/you-can-still-contribute-to-your-ira-for-savings-on-you-1825171977>

如果你还没有申报 2017 年的税，那么你还有时间通过向你的个人退休帐户缴款来节省一些钱。



它是这样工作的。你存入传统个人退休帐户的钱——每年 5500 美元，如果你超过 50 岁，则为 6500 美元——是免税的。如果你还没有达到 2017 年的上限，你可以在 4 月 17 日之前捐款，享受轻微的税收优惠。您不需要逐项列出才能从扣除中受益。

你会想明确你存入个人退休帐户的钱是 2017 年的，而不是 2018 年的，如果你还没有个人退休帐户，你会想尽快开一个，比如说，在这个周末之前，以确保你实际上得到扣除。(如果你是自由职业者，九月个人退休帐户是更好的选择。)重要的一点:这只适用于传统 IRA， [不适用罗斯](https://lifehacker.com/why-you-need-a-roth-ira-1823805955) 。

如果你有一个 [退休计划在工作](https://www.irs.gov/retirement-plans/are-you-covered-by-an-employers-retirement-plan)——一个 401(k)，403(b)，养老金或你的雇主贡献给你的个人退休帐户——你的 [修改 AGI](https://turbotax.intuit.com/tax-tips/irs-tax-return/what-is-the-difference-between-agi-and-magi-on-your-taxes/L7kHckNS3) 将影响你是否收到全额，部分或没有扣除。这里有一个图表，列出了 2017 年 的 [收入要求。如果你没有参加退休计划，收入限制要么不存在，要么更慷慨。您可以](https://www.irs.gov/retirement-plans/2017-ira-deduction-limits-effect-of-modified-agi-on-deduction-if-you-are-covered-by-a-retirement-plan-at-work) [在这里](https://www.irs.gov/retirement-plans/2017-ira-deduction-limits-effect-of-modified-agi-on-deduction-if-you-are-not-covered-by-a-retirement-plan-at-work) 查看 2017 年需求。

### 后门罗斯和储蓄者的信用

如果你为减税赚了太多钱，你可以做出不可扣除的贡献，然后将其转换成 Roth，以备将来可能的储蓄，这就是所谓的后门 Roth。

“如果你将传统的个人退休帐户——你已经获得了所得税减免的钱——转换成罗斯个人退休帐户，你将为你转换的全部金额交税，”RothIRA.com 写道。但如果你做出了不可扣除的贡献，“你可以把税后的传统个人退休帐户的钱转换成罗斯个人退休帐户，而不需要另一张税单，因为你已经付了税。”然后免税种植。另一个好处是:你可以想转多少钱就转多少钱(嗯，和你个人退休账户里的钱一样多)，5500 美元的年限额见鬼去吧。当你做出不可扣除的贡献时，确保你将 8606IRS 表格与你的纳税申报表一起归档。

一个警告:共和党税收法案[取消了将你的账户*转换回传统个人退休账户*的能力。](https://www.cnbc.com/2017/12/19/roth-ira-conversions-cant-be-undone-under-gop-tax-reform.html)

如果你是一个低收入的个人，还有一件事要考虑，那就是储蓄者的信用。:

> 2017 年，个人调整后总收入不超过 31，000 美元、户主调整后总收入不超过 46，500 美元或已婚夫妇调整后总收入不超过 62，000 美元的人，如果在退休账户中储蓄，并且不是全日制学生或声称依赖他人的纳税申报单，则有资格获得储蓄者信贷。该抵免额相当于你退休账户缴款的 10%、20%或 50 %,个人最高可达 2000 美元，夫妇最高可达 4000 美元，低收入人群可获得更高的抵免额。

它基本上允许你加倍扣除。你可以在这里 和 [这里](https://www.irs.gov/retirement-plans/plan-participant-employee/retirement-savings-contributions-savers-credit) 了解更多。

如果你已经申请了，但没有申请你的个人退休帐户扣除，不要担心。您可以使用本表格 中的 [填写修改后的申报表。](https://www.irs.gov/newsroom/file-form-1040x-to-amend-a-tax-return)
