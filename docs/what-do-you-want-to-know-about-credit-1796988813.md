# 关于信用，你想了解什么？

> 原文：<https://lifehacker.com/what-do-you-want-to-know-about-credit-1796988813>

不管你喜不喜欢，信用很重要。这不仅仅是申请汽车贷款或者买房子。你的房东检查你的信用。当你申请工作时，你的雇主也可以检查你的信用。账单提供商甚至被允许向信用不良的你收取额外费用。整个系统很混乱，所以告诉我们:你想了解它的什么？

信贷并不是世界上最令人兴奋的话题。事实上，这可能是世界上最不令人兴奋的话题之一。像许多个人金融话题一样，我们倾向于避免考虑信贷，因为它令人厌烦、复杂和困惑——而这正是我们被它征服的时候。因此，从财务角度来说，本着把我们的东西放在一起的精神，我们想在接下来的几周内把它分解开来。

在一系列的帖子中，我们将涉及信用如何运作的基础知识，以及如何改善你的信用。我们将讨论以下主题:

*   你的报告和你的分数之间的差异(以及为什么你应该关心)
*   如何找到你的分数并得到一份你的报告
*   FICO 如何得出你的分数
*   如何提高你的分数和信用
