# 试着在你的标签里列出一个待办事项清单

> 原文:[https://life hacker . com/try-a-to-do-list-that-lives-in-your-tabs-1823405913](https://lifehacker.com/try-a-to-do-list-that-lives-in-your-tabs-1823405913)

对于每个信奉 GTD 哲学的人来说，包括我自己在内， [任务经理](https://lifehacker.com/getting-things-done-ten-years-in-1795707084) 对于数字化管理流程至关重要。当然，只有当你真正使用它们时，它们才是有用的，正如我在寻找一个正确结合了跨平台访问、简单任务管理和一点点共享功能的任务管理器时留下的任务管理器的踪迹所证明的那样。如果它在一个我不会忘记的地方( [或避免](https://lifehacker.com/you-dont-have-a-procrastination-problem-you-have-an-im-1762132605) )，就像我的网络浏览器，那也会有所帮助。到目前为止， [Taskade](http://taskade.com) 已经设法让我使用一个不是一张纸的待办事项列表，这要归功于它几乎一直存在。当我打开一个新标签时，任务管理应用程序会将我的待办事项列表放在前面和中心，支持多种设备，并具有简单的共享功能，这使它成为你日常生活的一个很好的补充。仅仅是每天多次提醒就足以让我思考什么需要我注意，什么可以等待。

### **每个新标签页显示你的任务**

Taskade 最好的部分是它的易用性。您可以访问该网站，创建一个帐户，并开始使用 web 界面添加任务。想让你的待办事项列表触手可及？安装 Taskade 的 Chrome 或 Firefox 扩展，每当你创建一个新标签时，它就会加载你的 Taskade 页面，这是一个非常有用的功能，每个任务管理器都应该采用。

在页面上，您将看到您的任务、项目、文件夹列表，并且您可以为某些任务使用不同的模板。创建一个大纲、一个简单的清单或一个注释栏来键入您想要的任何内容，所有这些都使用 Taskade 的预置选项。如果知道要重复某种类型的任务(如聚会清单或会议大纲)，也可以构建项目模板。

### **链接共享让协作编辑变得简单**

由于有两个共享选项，在 Taskade 中与他人共享任务或家务清单非常简单。您可以邀请 Taskade 用户参与您的项目和列表，或者共享您的任务的只读列表。如果有人决定编辑你的列表中的一些项目，并添加你忘记的内容，你必须刷新才能看到这些变化，但协作选项使其优于只接受你输入的封锁任务管理器。

### 几乎在任何地方都可以买到

如果你的任务管理器在你的手机上，而不是在你的电脑上，考虑使用 Taskade 作为一个轻量级的任务捕获工具，你可以在工作时间求助于它，而不会迷失在你的应用程序的泥沼中。更好的是，如果你发现自己在网上游荡，你可以 [将 Taskade 与网站拦截器](https://lifehacker.com/how-to-block-social-sites-for-your-own-damn-good-1795484603) 配对，让你继续工作。

除了有用的 Chrome 扩展之外， [Taskade 还可以作为一个应用程序](https://www.taskade.com/downloads) 用于 Windows 和 macOS，iOS 和 Android 应用程序也在路上。它的跨平台可用性和即将推出的应用程序可能会使 Taskade 成为一个伟大的任务管理器，适合那些希望避免处理 Omnifocus 等应用程序的复杂性的人，以及那些希望使用多种设备的跨平台服务的人。

### **它快速、可爱，而且非常有用**

Taskade 还是相当新的，应该可以作为存放不太紧急的待办事项和任务的地方。它有利于和一两个朋友组织一次活动，建立一个可以和伙伴分享的购物清单，为一个项目集思广益，或者在一天结束前处理好你需要完成的所有事情。如果你忘记了你盘子里的所有东西，你所要做的就是依靠你的肌肉记忆，打开一个新标签，并收到一个急需的提醒，这是最后一次，你需要拿起一袋糙米，这样晚餐就不会完全是一场灾难。
