# 热带桌面

> 原文：<https://lifehacker.com/the-tropical-desktop-1768621465>

读者 Amy 发来的这张桌面，给我们的屏幕带来了一点夏日时光，这是我们可以一直欣赏的。最重要的是，它非常精简，不需要太多的调整就能达到最佳效果。如果你想要同样的造型，下面是艾米的做法。

Watch

如果你不知道的话，艾米正在运行 Windows，所以这是另一个雨量计设置。如果你对雨量计不熟悉，你可以在这里查看我们的初学者指南，先熟悉一下，然后试一试。

这样一来，你就需要:

*   [壁纸](http://slurpaza.deviantart.com/art/untitled-III-517535691)
*   用于 Windows 的 [雨量计](https://www.rainmeter.net/) 系统调试和监控工具
*   用于时间和日期的雨量计和其他活动小部件的 [Simplic](http://hiphopium.deviantart.com/art/Simplic-588463108) 皮肤

这周很简单，很好。即使只有一个真正的小部件在 Rainmeter 上运行，它的外观也很清晰，而且它做了 Amy 需要它做的事情。因为她直接发邮件给我们，我们不能链接到她的 Kinja 博客或任何东西，但如果你喜欢你所看到的或有问题，留下评论，希望她会停下来回答他们！
