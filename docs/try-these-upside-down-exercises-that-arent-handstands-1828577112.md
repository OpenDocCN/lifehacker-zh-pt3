# 尝试这些不是倒立的倒立练习

> 原文：<https://lifehacker.com/try-these-upside-down-exercises-that-arent-handstands-1828577112>

在我们的[](https://vitals.lifehacker.com/let-s-get-upside-down-for-the-august-fitness-challenge-1828029407)倒立挑战中，我们已经谈到了，也尝试了 [倒立瑜伽式](https://vitals.lifehacker.com/how-to-work-up-to-your-first-yoga-headstand-1828419168#_ga=2.191399850.1203445406.1534765835-1456718367.1520458611) 。今天的重点不是关于平衡，而是关于如何倒立来帮助你锻炼力量。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-5QFjmotLfW4&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-5QFjmotLfW4&start=0) 

### 谢绝俯卧撑

如果你正准备做倒立俯卧撑，这是一个很有帮助的动作，但是它们对于增强肩部力量也很有帮助。上面的视频显示了适当的形式:你的胸部垂到双手正上方，但保持身体挺直。不要让你的背部下垂，即使这看起来会让你的胸部更容易接触到你的手。



在视频的最后有一个很好的提示:抓住一对平行四边形或沉重的壶铃，把你的手从地板上抬起来。这样你可以把你的胸部放低，而不会把你的鼻子压向地面。

### 下降台式压力机

这是大多数健身房都配备的经典运动之一。这只是一个普通的卧推，你可以用一个杠铃或一对哑铃来做，但你是在一个倾斜的长凳上。你的膝盖钩在较高的一端，你的脸在底部。与常规卧推相比，这种方法对胸部肌肉的作用略有不同，因此将其纳入日常锻炼是有好处的。

### 谢绝仰卧起坐

如果你想用仰卧起坐或仰卧起坐来挑战自己，你也可以在倾斜的长椅上做。把你的膝盖钩在上面，躺下，然后试着抬起你的上半身来对抗重力的拉力。这并没有提供比常规仰卧起坐(或负重仰卧起坐)更大的优势，但做起来很有趣。
