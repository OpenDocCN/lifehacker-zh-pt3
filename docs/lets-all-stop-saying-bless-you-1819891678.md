# 让我们都停止说祝福你

> 原文：<https://lifehacker.com/lets-all-stop-saying-bless-you-1819891678>

这里有个很酷的东西。当你生病，或过敏，或有东西飞进你的鼻子，你痉挛并排出粘液时，你周围的任何人，包括完全陌生的人，都应该礼貌地引起注意。在说英语的国家，他们说“祝福你”，在欧洲大部分地区，他们说“健康”几乎在每一种文化中，礼貌的回应是“谢谢” 比如“谢谢你提醒我注意我尴尬的身体机能。”就像“谢谢你让我感谢你，而我可能还在处理我体内的东西现在在我体外的事情。”比如，“谢谢你提醒我，在接下来的三个月里，我将会和陌生人进行即兴的两行对话，因为我的身体认为花朵想要杀死它。”



我不是在装腔作势！西方文化在过去几年里达成了一个共识，也许我们都可以在公共场合少评论对方，尤其是在人们无法控制的事情上。真的，还有什么礼仪规则要求每个人在身体活动时都要抢着说些什么呢？在你对我腰部以下的身体机能冷嘲热讽之前，我只接受英国女王在镜头前对来访的政要说的话。没有，除了“祝福你。”

是什么让打喷嚏如此特别？没什么。我来告诉你为什么没什么，因为打喷嚏有一个爱尔兰双胞胎:咳嗽。它同样具有传染性，同样具有破坏性，其原因也是多种多样的。当有人咳嗽时你会怎么做？如果你有礼貌，没什么。除非他们反复咳嗽，在这种情况下，你可以谨慎地问他们是否还好。如果咳嗽非常严重，你可能会认为对方会说“对不起”，但如果他们不说，你可能会没事。你会期望他们尽最大努力抑制咳嗽，也许之后会洗手——看，我们有很多关于咳嗽的礼仪，但没有一个礼仪要求每个人都关注咳嗽。为什么会这样？礼仪就是让人感觉舒服。

你可能相信，就像我三分钟前相信的那样，“保佑你”最初是为了防止恶魔进入身体。但是有 [这个故事没有可靠的证据](https://www.snopes.com/language/phrases/blessyou.asp) ，根据第一个谷歌结果为*【祝福你的历史】*。早在公元 150 年就有文字记录表明人们说“祝福你”，当时罗马还在用基督徒喂狮子。所以如果这个短语是关于神和魔鬼的，那就是关于异教徒的。

没有可靠的证据证明“祝福你”的起源我们实际上不知道我们是如何开始的，但我们没有停止，在放弃其他古老的习俗，如祭祀或女巫审判几个世纪后。我真的想停下来。

我知道我不是一个人。或者我打赌我不是一个人。伙计们，我真的不想一个人孤军奋战。我怀疑，考虑到哪种人倾向于向公众传达“祝福你”，哪种人不会，我们大多数人会很乐意放弃这一习俗。

但是没有人(可能除了里奇·格威斯)想第一个停下来。我们不想做无礼的人。我们正在等待某个高调的人物来带头发起冲锋，就像 JFK 允许男人停止戴帽子一样。好吧，不要屏住呼吸，因为 [JFK 实际上并没有开始无帽潮流](https://www.snopes.com/history/american/jfkhat.asp) ，根据第一个谷歌结果为*肯尼迪的帽子*。

而是“祝福你”和“健康！”并不普遍。在中国、日本和韩国，人们习惯忽视打喷嚏。日本人常说“你看见死亡了！” [但他们在亿万年前就放弃了](https://thisjapaneselife.org/2012/02/15/sneezing-in-japan/) ，根据第一次谷歌搜索结果为*日本打喷嚏？？？*。数十亿人生活得非常好，没有告诉对方“我听到你打喷嚏。”我们其他人也可以。

所以现在就开始。就在这里，让我们做个约定，不再说“祝福你”此刻，你的沉默可能会冒犯到别人。如果他们反对，你可以礼貌地道歉:“我不想引起别人对你打喷嚏的注意。”你可以假装你以为是咳嗽。这仍然足以让他们产生这种想法。也许下次他们也会这么做。很快，当每个人都不再说“上帝保佑你”的时候，我们可以把这归咎于总统。
