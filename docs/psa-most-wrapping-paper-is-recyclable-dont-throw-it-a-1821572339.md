# 大多数包装纸是可回收的，所以不要扔掉

> 原文：<https://lifehacker.com/psa-most-wrapping-paper-is-recyclable-dont-throw-it-a-1821572339>

当你在今天的庆祝活动后打扫卫生的时候，记住现在大多数包装纸实际上是可回收的。



一般来说，任何简单的包装纸都可以放入回收站，而不是垃圾桶。你唯一需要扔掉的是用箔纸或闪光纸装饰的纸。 [*RecycleNow* 建议](https://www.recyclenow.com/what-to-do-with/wrapping-paper-1) 使用“揉皱测试”来确定某物是否可回收。如果它紧缩，那么你可以，如果它不紧缩，你不能。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-8PBps0ccvXc&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-8PBps0ccvXc&start=0) 

在你装满你的垃圾箱之前，你需要把所有的蝴蝶结和丝带从纸上拿掉(我建议用孩子们来做这件事)。确保你当地的回收中心会接受它。现在大多数人都这样做了，但有些人仍然没有能力处理它。

在你扔掉任何东西之前，看看你要扔掉的东西，并决定你是否可以再次使用它是值得的。几年前，我养成了收集薄纸和礼品袋以及一些较大的未损坏包装纸的习惯，在礼物被拆开后，将它们放在一个大塑料容器中。

今年我确实买了一卷纸，但我 75%的礼物都是用多年前喜欢的纸包的。我的礼物接受者一点也不知道，今年我不必再为包装礼物而烦恼，我已经有了我需要的一切。
