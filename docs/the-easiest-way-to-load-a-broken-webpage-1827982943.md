# 加载破损网页的最简单方法

> 原文：<https://lifehacker.com/the-easiest-way-to-load-a-broken-webpage-1827982943>

假设一个网页加载不正确。也许它在 Reddit 上疯传后因为太多的流量而崩溃了。也许由于 GDPR 这样的法律，它在你的国家被屏蔽了。可能是最近删的。通常谷歌会保存该页面的副本。获得保存副本的最快方法是在地址栏中键入 *cache:* 。

Watch

只需点击地址栏，将光标移动到最左侧，键入 *cache:* 并按回车键。Google 将加载页面的缓存副本，它通常看起来很像普通页面。只要谷歌仍然是你的默认搜索引擎，这个技巧在 Chrome、Safari 和 Firefox 上都有效。(Edge 尝试在微软商店加载一个应用。)

在页面顶部，谷歌给添加了一个到的链接，这是页面的纯文本版本。所以谷歌的缓存是也是一种让网页更具可读性的简单方法。

如果谷歌没有该页面的缓存版本，在网址前加上*web.archive.org/web/*就可以从互联网档案馆的 Wayback 机器上获得一份拷贝。如果这不起作用，那么对不起，以后再试。
