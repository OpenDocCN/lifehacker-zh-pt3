# 学习撬锁让我知道门锁有多糟糕

> 原文：<https://lifehacker.com/learning-to-pick-locks-taught-me-how-crappy-door-locks-1750091711>

大约一个月前，我捡到了 [一套](http://www.amazon.com/11-Piece-Unlocking-Extractor-Transparent-Practice/dp/B014PET8W2/ref=sr_1_2?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/learning-to-pick-locks-taught-me-how-crappy-door-locks-1750091711&asc_source=&ie=UTF8&keywords=lockpick set&qid=1451316534&sr=8-2&tag=kinjalifehackerlink-20) 训练开锁器。我一直很好奇撬锁是怎么工作的，到底有多难。在学习的过程中，我不仅发现它是多么容易，而且发现在保护我的家时，如何超越锁。

Watch

在你抱太大希望之前，不，这不会是如何撬锁的指南。 [在我们的](http://lifehacker.com/learn-to-pick-locks-for-fun-5672326) [【邪恶周】](http://lifehacker.com/tag/evil-week) 系列中，我们已经这样做了，我们仍然认为了解安全措施是如何被破坏的很好，所以 [它不会发生在你身上](http://lifehacker.com/how-to-break-into-a-computer-and-prevent-it-from-happe-5854079) 。然而，理解锁的工作原理并不等同于保证它们的安全。防范撬锁工具并不等同于保护你的家。

### **大多数锁都很笨，很容易被撬开**

当我第一次得到我的开锁工具时，它带有一套钥匙，在一个锁在锁上的环上。这几乎是一个挑战。“你想要这把锁的钥匙？你得先把它摘下来。”令我惊讶的是，我在五分钟内就把锁打开了。当然，这是一把专门为训练设计的透明锁。如果这是一个真正门上的普通锁，我会花费更多的时间(我第一次尝试时确实如此)，但我惊讶于它是如此简单。事实上，只要稍加练习，我就能在一个快速的动作中把拨片划过销子，并在几秒钟内解锁。在*辐射 4* 中，我仍然很难撬开锁。

很自然，我的下一个想法是“这是保护我房子的东西吗？”所以，我决定研究一下门锁和这种训练挂锁有什么不同。答案？ [不多](https://www.blackhat.com/presentations/bh-europe-08/Deviant_Ollam/Whitepaper/bh-eu-08-deviant_ollam-WP.pdf) 。有一些方法可以使锁更加防撬。例如，一些 Medeco 锁使用的插销不仅需要提升到正确的高度，而且需要在锁打开之前将 [旋转一定的方向](http://www.gaterslocksmith.com/blog/2013/12/26/what-makes-the-medeco-lock-so-hard-to-pick/) 。当然，“防撬”锁仅仅是这样。它们增加了开锁的难度，但这绝不是不可能的。只是可能要多花一点时间。更重要的是，普通门锁通常不具备这些功能。

那么，你怎么知道你家的锁能被轻易撬开呢？ANSI 的 [分级系统](http://www.schlage.com/en/home/support/understand-product-options/functions-grades.html) 是一个很好的起点。这是一个自愿的、独立的评级系统，锁业用它来自我评估一把锁在防止闯入方面有多好。3 级锁是最弱的，被评为住宅安全的最低要求(这应该让你知道你家的锁在安全图腾柱上的排名)，而 1 级锁是最强的。

出于好奇，我决定看看我家的锁有多安全。我有一把耶鲁 [双缸锁舌](http://www.yaleresidential.com/en/yale/yaleresidential-com/residential/yh-collection/deadbolts/select-keyed-both-sides/) 。这无疑是我住过的所有房子中最好的锁，而且它仍然只是 ANSI 级。对于住宅锁来说，这比平均水平要好，但仍不是你能得到的最好的。如果你想要这个街区最好的锁，一级锁是最好的。这些锁包括防撬功能、更坚韧的材料和加固的防冲击板。然而， [正如我们在](http://lifehacker.com/how-difficult-is-it-to-pick-a-deadbolt-lock-5961197) 之前讨论过的，在你的门上装一把别致的锁可能只是在向小偷宣传你家里有一些很酷的东西。如果你打算买一个 1 级锁销，至少要确保它 [看起来和普通锁销](http://www.gokeyless.com/product/489/schlage-b60-deadbolt-lock) 没有太大区别。

### **撬锁甚至不是绕过锁的最好方法**

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-131j0htYIoU&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-131j0htYIoU&start=0) 

一旦我开始学习开锁，我发现自己在一些专门教授这门艺术的论坛上。我探索得越多，我就越了解到，虽然开锁是一种有趣的爱好，但它不是绕过锁的最有效的方法。现代的 [弹子锁](https://en.wikipedia.org/wiki/Pin_tumbler_lock) 已经存在了几个世纪，人们已经找到了更好的方法来撬开它们。以下是几种方法:

*   **凹凸钥匙:**凹凸钥匙是 [就像普通钥匙](http://lifehacker.com/how-to-pick-a-lock-with-a-bump-key-399735) 一样，有足够深的插销槽，可以安装任何锁。一次快速撞击就能把锁里的每一个插销排好并打开。真是 [吓人容易](https://www.youtube.com/watch?v=131j0htYIoU) 。虽然每个凸块钥匙仍然必须匹配轮廓脊，但大多数家用锁一开始只使用几种钥匙形状。凹凸钥匙可以由你在当地五金店找到的普通空白钥匙制成，或者你可以在线购买 [现成的凹凸钥匙](http://acehackware.com/collections/bump-keys) 。见鬼，如果你愿意的话，你甚至可以 3D 打印凸起键。一个小偷在他的钥匙圈上放几个这样的东西，可以轻易地打开大多数的门锁。更糟糕的是，他甚至看起来不像是要破门而入。他看起来可能只是在摆弄一把特别顽固的锁。
*   **锁捕捉:**有些锁容易受到一种特别恶劣的攻击，称为 [锁捕捉](https://en.wikipedia.org/wiki/Lock_snapping) 。使用这种技术，以正确的方式施加少量的压力实际上会将锁分成两半，使它变得无用。在英国的某些地区，这是一个大问题，那里的许多家庭都配备了脆弱的圆筒锁，用很小的力就可以打破。这种手法如此普遍，职业窃贼 [都懒得撬锁](https://youtu.be/TCufUpr_QRY?t=134) 。大多数锁不容易受到这种类型的攻击，但当它被发现时，数百万个家庭已经安装了这些有缺陷的锁。
*   **信用卡:**如果你曾经想知道 [为什么你需要在普通锁的顶部安装一个插销](http://lifehacker.com/how-difficult-is-it-to-pick-a-deadbolt-lock-5961197) ，就是因为这个。用信用卡的方法，你可以在门和门框之间塞一块小而平的塑料。如果你要打开的锁有一个倾斜的边缘(这允许你关闭你身后一个已经锁上的门)，你可以用力把门撬开。锁定插销没有这种倾斜的边缘，并且*必须用钥匙*锁定，所以它们不能以这种方式打开。是的，这是每部电影和电视节目中使用的方法，是的 [它有时确实有效](https://www.youtube.com/watch?v=IC9n-0GAhRQ) 。

所有这些方法都只是为了绕过锁。这还不算是所有破门而入的方法。如果你是一个经验丰富的小偷，正在寻找穿过一扇锁着的门的最简单的方法，这些方法中的任何一种都将比跪在门前用几片薄金属片轻轻地移动你看不见的小针更有效和容易。撬锁，作为一种破门手段，真的是不得已而为之。

### 大多数小偷不会撬开你的锁就破门而入

一旦你知道撬锁(或绕过锁)有多容易，就很容易被阻止。毕竟，如果这么容易，每个头脑正常的小偷都会学会这种技能，不是吗？事实证明，并非如此。根据*华盛顿邮报*T5】的 [报道，最常见的入室盗窃是住在你家几英里内的男性青少年所为。换句话说，他们不是专业人士。他们是小屁孩。](http://www.washingtonpost.com/wp-srv/artsandliving/homeandgarden/features/2007/burglary-070507/graphic.html)

冲动的罪犯倾向于使用更暴力的方法进入住宅，而不是单调乏味地撬锁。小偷可能会把门踢开(这种 [加固的防冲击板是为了防止](http://www.familyhandyman.com/home-security/how-to-reinforce-doors-entry-door-and-lock-reinforcements/view-all) )，砸碎附近的窗户，或者从你忘记锁的后门进入(哎呦)。根据美国联邦调查局的统计数据，截至 2010 年，60.5%的入室盗窃涉及强行进入，包括打破门窗或威胁里面的人。

当然，这并不意味着门锁没有用。的确，如果有人非常想进去，他们就能找到进去的方法。然而，砸窗户或踢开门肯定会引起别人的注意。如果你被抓住了，直接威胁别人就更糟了，因为这会带来额外的指控。如果这是小偷唯一的选择，他们可能更愿意带着钥匙去你邻居的房子，而不是你的。

### **安全就是要比别人更聪明**

多年的电影和电子游戏告诉我，撬锁是所有小偷高手都会做的事情。太酷了，你这么做看起来很聪明。我承认，学习和练习很有趣。如果我被锁在自己的房子外面，除了坐在人行道上哭泣直到我的女朋友回家，我还有一个选择，这让我感觉好一点。

然而，学习如何撬锁并没有教会我如何阻止小偷。反而了解到了小偷真正是怎么操作的。我对如何进入上锁的房间研究得越多，我就越明白一个潜在的罪犯会如何解决进入我家的问题。有了这些知识，我发现了一些关键的要点:

*   降低房子的吸引力是最好的威慑。如果小偷想进来，你的锁阻止不了他们。所以 [确定他们不想进](http://home.howstuffworks.com/home-improvement/household-safety/security/discourage-break-ins.htm#page=1) 。不要把装你所有昂贵小玩意的大盒子放在垃圾桶旁边。拉上你的百叶窗。尽管它们可能很吸引人，但既然你知道开锁有多容易，就不要在你的门上安装那些昂贵的智能锁。上面写的是“我这里有值得偷的东西。”
*   让你很难躲在家门外。开锁并不吸引人，因为它需要时间。小偷在你家外面呆的时间越长，就越有可能被发现。通过在 外安装 [自动运动灯，让](http://www.homedepot.com/b/Lighting-Ceiling-Fans-Outdoor-Lighting-Outdoor-Security-Lighting/N-5yc1vZc7qf) [更容易发现小偷](http://lifehacker.com/how-to-cleverly-secure-your-home-against-intruders-5887264) (因此他们更有可能不会尝试)。减少前门附近的灌木数量。你的前门越隐蔽，小偷就越容易溜进来。
*   藏起你的贵重物品，即使是在你的家里。别把你家保安拦在前门。假设一个小偷仍然可以进入里面(因为他们可以)，并且 [隐藏你最重要的贵重物品](http://lifehacker.com/the-best-places-to-hide-valuables-in-your-house-5960300) 。你也可以留一点现金给小偷，让他们轻松得分，这可能会阻止他们翻箱倒柜到其他地方寻找你真正的贵重物品。

学习如何开锁是一项有趣的练习。它让我深入了解了物理安全的基本原理，但它也有助于突出真正重要的东西。说到保护你的家，你不必建造一个堡垒。你只需要比试图闯入的人更聪明。无论是通过说服他们你不是一个好的目标，还是简单地将他们闯入时造成的损害降至最低，你越能像小偷一样思考，就能更好地保护自己免受他们的攻击。
