# 停止烹饪鳄梨

> 原文:[https://lifehacker.com/stop-cooking-avocados-1822559518](https://lifehacker.com/stop-cooking-avocados-1822559518)

人们经常伤害他们所爱的人，从事食品工作的人也不例外。对于厨师、 [博主](https://www.twopeasandtheirpod.com/stovetop-avocado-mac-and-cheese/#_a5y_p=1297658) 和内容创作者反复对鳄梨进行各种基于热量的折磨，还有什么其他可能的解释？鳄梨只需要撒一点盐就非常美味，烹饪它们是该死的烹饪罪。

Watch

鳄梨不是奶酪；加热不会以任何令人愉快的方式改变它们的质地或味道。你吃到的不是奶油般浓郁但仍然清爽的一口，而是黏糊糊、黏糊糊的一口，鳄梨的美味几乎消失殆尽。虽然我肯定有些人会坚持说他们喜欢鳄梨油炸食品或鸡蛋，但我认为这些人是骗子，或者至少是被 Tasty 告诉他们的谎言蒙蔽了双眼。然而，我不是那种简单地把热门的东西放到互联网上然后就走开的人，所以让我们来看看几个热门的 acado 应用程序，以及为什么每一个都很糟糕:

*   **牛油果薯条:**首先，我们能不能不要把每一种油炸的或者裹上面包屑的棒状蔬菜都叫做“薯条”？薯条是由土豆制成的，红薯薯条应该是这个家族中最“出格”的成员。裹着面包屑、烤成楔形的鳄梨不是薯条；他们很难过。他们经常裹在面包屑涂层中，这不足以提供任何真正的质地对比，而且鳄梨很快冷却，给人一种室温糊状的感觉，即使是苹果蜂也应该羞于食用。
*   鳄梨烤鸡蛋:啊，对了，谁不喜欢一杯热腾腾的绿色糊状物里松软的蛋黄和刚刚凝固、微微发白的蛋黄呢？人们把鸡蛋放在烤面包上，因为烤面包很脆。烤面包给你一个可以依靠的结构，让你享受一个光荣的半熟蛋黄。脂肪需要对比，无论是口感还是味道。一些网站试图通过用培根 包裹这个糊状球体 [来抵消它，但结果就像一个善良的鸡蛋，如果善良的鸡蛋是撒旦自己构想并执行的话。](https://www.youtube.com/watch?v=tMfdkIY4aAA)
*   **鳄梨通心粉和奶酪:**我没有亲自把它放进我的嘴里，但是 [Food52 把这道菜描述为“麝香”，](https://food52.com/blog/20369-we-tried-the-avocado-mac-n-cheese-the-internet-can-t-get-enough-of) 众所周知，这个词肯定会让你联想到通心粉和奶酪。但老实说，即使是最迷恋玫瑰金、喝独角兽拿铁的千禧一代也不会疯狂到相信通心粉和奶酪是通过减少奶酪和增加水果来改善的。

这些是困扰我的新闻提要的头号罪犯，我相信我们可以找到更多这样的暴行的例子，但主要的收获是:鳄梨很好。当你烹饪它们的时候，你会让它们变得不那么好吃，而且在大多数情况下，你会让它们变得非常糟糕。简单地对待你的鳄梨，带着爱和尊重，如果你有一头乱发，觉得真的需要和它们做爱，那就做一杯玛格丽塔。 [牛油果玛格丽塔](https://www.bonappetit.com/recipe/avocado-margarita) 出奇的清爽，一点都不粘稠，*冰凉*。
