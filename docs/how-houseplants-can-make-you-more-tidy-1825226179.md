# 室内植物如何让你更整洁

> 原文：<https://lifehacker.com/how-houseplants-can-make-you-more-tidy-1825226179>

在家庭事务上，我并不总是最细心的人。你说你应该清洁吊扇、电灯开关和烤箱架是什么意思？谁会这么做？”所以你会认为室内植物是个糟糕的主意。



Lifehacker 非常重视春季大扫除。我们绝不会错过更新、重组和整理我们家庭生活的机会。我们也非常兴奋地按下科技使用的重置按钮，仔细查看我们的财务状况，并抛弃已经有点发霉的日常习惯。欢迎来到 [*春季大扫除周*](https://lifehacker.com/tag/spring-cleaning-week) *，在这里我们清理冬天的蜘蛛网，为未来的晴天做准备。让我们收拾一下，好吗？*

“你认为你能让他们活着吗？”当我提到我要从苗圃里挑选一些室内植物时，我丈夫怀疑地问道。

“哦，是的，当然，”我说。

(我不确定。)

但我买了一些，随着时间的推移，开始收集更多。现在有 11 种室内植物散布在我们的家中——一种蛇类植物，几个悬挂的挂钩，一种橡胶树植物，一种蜘蛛植物和一些多肉植物。

当我开始照顾植物时，一件奇怪的事情发生了:我觉得我和我的房子更有联系了。植物是有点可怜的小东西。它们需要定期关注。有时，他们需要水。其他时候，它们需要旋转以获得等量的阳光。偶尔，它们的叶子需要修剪。时不时地，它们需要被完全移植。

虽然我全心全意地让它们活着，但我也注意到植物周围需要一些关注的地方——房子里容易被忽视的缝隙和角落。我看到书架需要一点灰尘，镜子需要一点玻璃清洁剂，墙上的磨损可以用神奇的橡皮擦擦掉。对我来说，照顾植物和整理它们周围的东西已经成了一种仪式。

植物可以 [改善我们对空间](https://www.psychologytoday.com/us/blog/urban-mindfulness/200903/plants-make-you-feel-better) 的感知，让我们更加专注。他们已经为我做到了。也许我只是一个试图填补内心空虚的千禧一代。啊，好吧——至少我的空间因此变得更好了。
