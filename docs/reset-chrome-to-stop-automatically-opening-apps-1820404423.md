# 重置 Chrome 以停止自动打开应用程序

> 原文：<https://lifehacker.com/reset-chrome-to-stop-automatically-opening-apps-1820404423>

从 Chrome 自动打开应用程序应该是一种便利——但有时这很烦人。当你只想看到应用程序截图时，处理一个侵入性的 iTunes 窗口可能是令人厌烦的，特别是因为 iTunes 甚至不再处理应用程序。有时你只是想复制一个电子邮件地址，或下载种子文件，而不是在传输中运行它。当应用程序基于你点击的链接打开时，你可以感谢你的浏览器的“协议处理器”在某种程度上，它们是有帮助的，但是你可以通过遵循一些步骤(或者删除一些 cookies)来摆脱那些更烦人的。

### **什么协议处理器做什么**

一个协议处理程序允许谷歌 Chrome 打开安装在你电脑上的应用程序，并根据你选择的链接进行某些操作。从收件箱中选择 Slack 的“魔法链接”将打开实际的 Slack 应用程序并让你登录，而点击电子邮件地址可能会打开你的默认邮件应用程序，无论是 Outlook、mail 还是 Thunderbird。你可能点击了一两个未经你允许就自动打开 iTunes 的应用程序的链接。那是工作中的协议处理器。

过去，谷歌允许你从它的设置页面管理你的协议处理器。现在吗？您只能启用或禁用协议处理程序，不允许修改。至少谷歌没有。然而，有了文本编辑器和一点诀窍，您可以自己编辑协议处理程序规则，并在不抛弃您熟悉和喜爱的应用程序处理程序的情况下，将那些烦人的应用程序踢到路边。

### **如何手动清除您的协议处理程序**

如果您不介意使用您的浏览器设置，或者不想删除您的 cookies 和其他浏览信息，您可以通过编辑它们所在的文本文件来修改协议处理程序本身。

#### **Windows 10**

1.  退出谷歌 Chrome。检查你的任务栏并退出谷歌浏览器助手。
2.  在你的 Windows 10 任务栏中，输入**% local appdata % \ Google \ Chrome \ User Data**，打开资源管理器窗口。
3.  在浏览器中，选择功能区中的查看，并选中“隐藏文件”框以显示更多文件。
4.  使用文本编辑器(如记事本)或应用程序(如微软的 Visual Studio 代码)打开首选项文件。
5.  搜索短语“excluded_schemes”，它在文本文件中包含您的协议处理程序首选项列表。
6.  搜索有问题的协议(如 iTunes 的“itmss”或 slack 的“Slack”)，并将其状态从“true”更改为“false”，注意不要删除任何标点符号。
7.  保存文件，并重新打开 Chrome。

#### **苹果电脑**

1.  退出谷歌 Chrome。
2.  选择 Finder，点击 Shift-⌘-G，输入文件夹地址**~/library/application support/Google/chrome**。
3.  选择默认文件夹，并在文本编辑器(如文本编辑器)中打开首选项文件。
4.  搜索短语“excluded_schemes”，它在文本文件中包含您的协议处理程序首选项列表。
5.  搜索有问题的协议(如 iTunes 的“itmss”或 slack 的“Slack”)，并将其状态从“true”更改为“false”，注意不要删除任何标点符号。
6.  保存文件，并重新打开 Chrome。

### **轻松清除协议处理程序**

您可以完全重置您的协议处理程序，尽管这需要您清除比您预想的更多的数据。删除浏览器的历史记录将重置您的协议处理程序，并要求您决定是否希望相关应用程序再次打开它们。访问 Chrome 的设置页面，选择高级，然后选择清除浏览数据。要将您的浏览数据删除仅归入您的协议处理程序设置，请选中“Cookies 和其他站点数据”框。您将被注销网站，但它会重置您的协议处理程序数据，而无需您修改文本文件。

在 Chrome 的设置页面中防止进一步的永久性协议处理程序修改是一种永远不用担心过分热心的应用程序的方法。选择清除浏览数据选项上方的内容设置。选择处理程序，然后关闭默认的处理程序功能，禁止任何一个应用程序占用所有相关链接，并让您在选择相关链接后选择何时打开应用程序。
