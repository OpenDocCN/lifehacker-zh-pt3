# 三种配料的欢乐时光:黑莓果酱树莓

> 原文：<https://lifehacker.com/3-ingredient-happy-hour-a-blackberry-jam-bramble-1824303394>

周末快乐，欢迎回到 [*三料欢乐时光*](http://lifehacker.com/tag/3-ingredient-happy-hour) ，每周饮品专栏，提供超级简单又美味的酒水。我这周在牛津，所以我想我们应该做一个精简版的非常英国的黑莓黑莓手机。



尽管美国发明了鸡尾酒，英国发明了荆棘，世界因此变得更美好。这种饮料通常由杜松子酒、柠檬汁、简单的糖浆和黑莓利口酒制成，这很好，但成分太多了。此外，我不想再买一瓶利口酒，所以我拿了一罐果酱，完全不加糖浆。(别担心，我调了杜松子酒来弥补甜酒的不足。)要自己做一个，你需要:

*   两盎司半杜松子酒
*   1 盎司柠檬汁
*   3 汤匙非常棒的黑莓酱

将所有原料混合入加满冰块的摇壶中，摇至冷到无法保存。用细网筛滤入盛有碎冰的低球，饰以条状或柠檬皮，或黑莓(如果你身边有一个的话)。
