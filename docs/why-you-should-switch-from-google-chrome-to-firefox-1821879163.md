# 为什么你应该从谷歌浏览器转向火狐浏览器

> 原文：<https://lifehacker.com/why-you-should-switch-from-google-chrome-to-firefox-1821879163>

自首次亮相以来，Chrome 越来越受欢迎，尽管其曾经辉煌的声誉最近受到了一些打击。 [纯 Chrome 网站的例子越来越常见](https://www.theverge.com/2018/1/4/16805216/google-chrome-only-sites-internet-explorer-6-web-standards) ，让人想起微软的 ie 浏览器统治网络浏览器市场的日子。它也被证明是一个 [巨大的内存猪](https://lifehacker.com/why-chrome-uses-so-much-freaking-ram-1702537477) ，当用户创建越来越多的标签时，会降低机器的速度。如果你正在寻找 2018 年的改变，为什么不从你的浏览器开始呢？抛弃 Chrome，转而使用其长期竞争对手 Mozilla Firefox。它与 Chrome 一样快，如果不是更快的话，并且集成了一些工具来增强你的在线隐私，同时使你更容易分享和保存你在网上找到的一切。

### **和 Chrome 一样快**

至少在各种基准测试中，火狐和谷歌 Chrome 是相当的。虽然在 Mozilla 进行的测试中，Firefox 在以私人浏览模式加载网页时可能会优于 Google Chrome，或者在处理多个标签时使用更少的 RAM，但其他独立测试显示，这两种浏览器在性能方面往往旗鼓相当。在最坏的情况下，你会切换到一个同样快的浏览器。在最好的情况下，您将使用一个需要更少内存，但响应速度一样快的处理器。

### **您仍然可以跨设备同步**

就像谷歌的同步功能可以让你看到你在所有设备上的浏览活动一样，火狐自己的同步功能可以通过选择你的地址栏中的选项，轻松地将你桌面上的网页发送到你口袋中的手机上。如果你是在同一台机器上使用多个谷歌账户的人，你可以很容易地用火狐自己的多账户扩展 做同样的事情。

### **火狐阻止追踪器开箱**

需要阻止网站跟踪你吗？Firefox 自动禁止隐藏在网页代码中的跟踪器，保护你的浏览数据不被试图向你出售定向广告的第三方记录。它可以在普通和私人浏览模式下工作。删除侵入性跟踪代码也意味着 [页面加载速度比](https://blog.mozilla.org/blog/2017/11/20/firefox-private-browsing-vs-chrome-incognito/) 更快。

### **火狐的集成特性是首选**

浏览器内置了许多有用的功能，减少了添加大量第三方扩展和应用程序的需求。稍后阅读服务口袋集成到浏览器中，因此您可以轻松保存任何页面。比如截图？ [有了火狐截图](https://lifehacker.com/how-to-use-firefox-s-killer-new-screenshot-tool-1821631687) ，比以往任何时候都更容易点击一个按钮，将网页的部分或全部截图保存到硬盘或基于云的截图库中。

作为一个担心第三方服务可能包含恶意代码的人，我需要添加到浏览器中的扩展越少越好。Firefox 的集成截图工具，轻松访问保存的口袋文章，以及自动禁用不可见的浏览器活动跟踪器，使其与 Chrome 相比，具有更安全和用户友好的体验。

### 一些扩展可以跨浏览器工作

就扩展而言，谷歌可能会击败火狐，但该公司正在采用网络扩展应用编程接口，使得谷歌浏览器和微软边缘中的插件在经过开发者的一些小修改后与火狐兼容。旧的火狐扩展现在被认为是“ [”传统的](https://developer.mozilla.org/en-US/Add-ons/WebExtensions/Porting_a_legacy_Firefox_add-on) 扩展，但是你可以通过访问你的插件列表并选择“寻找替代品”来找到潜在的替代品。如果当时没有替代的扩展，你只会看到一页与新版火狐兼容的特色扩展。
