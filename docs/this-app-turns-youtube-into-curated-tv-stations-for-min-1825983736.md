# 这款应用将 YouTube 变成了精心策划的电视台，供人们盲目观看

> 原文：<https://lifehacker.com/this-app-turns-youtube-into-curated-tv-stations-for-min-1825983736>

大约一年前，我们开始在我家一次狂看一个节目。我们通常会选择至少有四到五季的节目，然后每当我们想“看什么”的时候，答案总是那个节目的下一集。是的，我们会偏离它去捕捉其他节目的新集，但是拥有“一个节目”有助于消除滚动电视节目和建议以找到一些东西的需要，而我们实际上只是在寻找一些可以在背景中播放的东西。



[千万不要认为](https://neverthink.tv/documentaries) 是一种相同的概念，但对于 YouTube 来说。使用 Android 和 iOS 应用程序(和网站)，你可以切换到一个“频道”并观看主题视频。

如果你想在消磨时间的时候得到一点教育，你可以选择“学习一些东西”频道，如果你想笑，可以选择“喜剧频道”还有科技新闻、脱口秀、美容等频道。

这个想法是，它的工作方式有点像“YouTube 的电视”,你只需选择一个频道，让它去。你不需要在 YouTube 上四处寻找什么，它会一直播放，只要你想，所以当你的第一个选择完成时，你不会面临寻找另一个选择。

这不是用来一直看的，更多的是当你在回家的火车上或者睡觉前在手机上寻找 ***什么*** 的时候。

视频都是由该应用的工作人员精心挑选的，而不是一种算法，所以尽管视频的口味可能会有所不同，但比起你在等待延误的航班起飞/火车到来时漫无目的地在 YouTube 上闲逛，你也不太可能遇到无用的东西。
