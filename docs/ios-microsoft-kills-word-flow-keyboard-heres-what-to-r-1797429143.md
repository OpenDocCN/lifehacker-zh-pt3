# iOS:微软取消了 Word Flow 键盘——以下是替代它的内容

> 原文：<https://lifehacker.com/ios-microsoft-kills-word-flow-keyboard-heres-what-to-r-1797429143>

当你在 iPhone 上安装微软的 Word Flow 键盘时，你可能会认为它是一个应用程序或扩展。原来，这是一个“实验”， [一个现在已经“完成”的实验，](https://m.windowscentral.com/microsofts-word-flow-keyboard-ios-no-more) 你需要换一个新的键盘。

Watch

你最安全的赌注是 [SwiftKey](https://swiftkey.com/en) ，微软去年初收购了它，最近它赶上了默认键盘，具有 3D 触摸光标控制和 100 多种支持的语言。多语言打字者甚至可以在飞行中切换语言，SwiftKey 会检测到这种变化。

我们还推荐谷歌的 [Gboard](https://itunes.apple.com/us/app/gboard-a-new-keyboard-from-google/id1091700242) ，它提供即时搜索和听写功能。当 SwiftKey [将你的打字数据发送到它的服务器](https://support.swiftkey.com/hc/en-us/articles/201309802-What-does-SwiftKey-do-with-my-data-) 进行定制处理时，Gboard 会将你所有的打字数据(除了搜索和听写录音)留在你的键盘上。

与 Word Flow 不同，这两种键盘都不会挤到一边来单手打字(为此你需要或[minum](https://itunes.apple.com/us/app/minuum-little-keyboard-for-big-fingers-now-smart-emoji/id914968934))。但两者都提供滑动输入，这(经过一点练习)比单手敲击要快得多。

若要启用新键盘，请在下载后前往“设置”>“通用”>“键盘”>“添加新键盘”,添加键盘，然后再次选择它并打开“允许完全访问”
