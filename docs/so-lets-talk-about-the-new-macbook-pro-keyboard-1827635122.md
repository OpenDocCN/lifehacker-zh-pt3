# 所以我们来谈谈新的 MacBook Pro 键盘

> 原文：<https://gizmodo.com/so-lets-talk-about-the-new-macbook-pro-keyboard-1827635122>

上周，苹果公司发布了一款升级版 MacBook Pro ，它采用了三个月前刚刚发布的新处理器，并对键盘进行了调整。新键盘！让我们谈谈那个。



我花了最后一天时间使用新键盘——在使用旧键盘近两年后——我立即注意到了不同之处。但我不确定我是否会说这是一种进步。我也不会说它更糟。现在，只是，嗯，不一样了。

这可能是一件好事，因为到目前为止，MacBook Pro 上的键盘一直是愤怒的主要来源。按键本身的“行程”非常短这是按键完全按下并“触底”所需的距离大多数笔记本电脑的按键大约有 1.5 到 2 毫米的行程。MacBook Pro 的厚度不到 1 毫米。这意味着它更容易被像一袋薯片的碎屑之类的东西阻塞。当面包屑卡在钥匙下面时，它不能被完全压下。它变得非常坚固，如果还能用的话，用起来很烦人。

它是如此令人讨厌，以至于超过 [32，000 人要求召回 Charge.Org](https://www.change.org/p/apple-apple-recall-macbook-pro-w-defective-keyboard-replace-with-different-working-keyboard)[至少有一个集体诉讼被提起](https://gizmodo.com/apple-slapped-with-class-action-lawsuit-over-faulty-mac-1825983089#_ga=2.96175708.1963621744.1531766196-1585549752.1531072453)[苹果悄悄启动了键盘](https://www.macrumors.com/2018/06/22/apple-macbook-pro-keyboard-repair-program/) 的维修计划。

3 月，何孙报道说，苹果公司已经获得了三项专利，可以解决按键上有一层薄膜的问题，我们有一种松了一口气的感觉。苹果拒绝透露这些专利中的技术是否用于新键盘，但这似乎至少是可能的。在上周的拆解中，iFixit 的小工具外科医生发现，新款 MacBook Pro 上的按键开关——实际的按键机制——现在被一层硅膜覆盖着。(请注意:如果你试图移除 MacBook Pro 上的按键，你很可能会弄坏它，所以请不要尝试。)

从理论上讲，这层膜应该可以阻挡碎屑和其他碎屑，从而有可能改善键盘移动距离过短带来的最大问题之一。

我在键盘上吃了一袋薯片，瞧，没有一个键卡住。但是只有一个包。这将需要更多的小吃休息，看看苹果是否真的解决了这个问题。

但是对于像我这样的书呆子来说，一个立即被注意到的变化是薄膜键盘的声音和感觉。声音更低沉。在这篇文章顶部的视频中，我试图说明两种键盘声音的差异——虽然麦克风并没有完全捕捉到差异(毕竟你并没有真正感受到键盘)，但如果你戴上耳机，你肯定可以听到它。

当你*能够*感受到它的时候，这种差异就相当明显了。新键盘缺少了我从 MacBook Pro 联想到的那种感觉。当我用我的旧型号快速打字时，按键会发出很大的声音，让我的指尖有一种震动的感觉。

当按键触底时，新键盘感觉更柔软。很温柔。我不知道我是否讨厌它——它仍然比传统的薄膜键盘感觉更快。但我肯定发现自己错过了最初的 2016 年 MacBook Pro 键盘的更硬的快照。

但重要的是要记住，我只和 2018 款 MacBook Pro 相处了几个小时。几天后，我对键盘的感觉可能会改变。当然，键盘只是等式的一部分。苹果声称新款 MacBooks 速度更快，电池寿命更长，这可能足以说服你独自升级。敬请关注。
