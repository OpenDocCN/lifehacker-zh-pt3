# 你不用开瓶器就能打开一瓶酒吗？

> 原文：<https://lifehacker.com/tip-tester-can-you-open-a-bottle-of-wine-without-a-cor-1756636043>

当你有一瓶可爱的葡萄酒，但是你的开瓶器不见了，你会怎么做？当滴酒不沾不是一个选项时，有无数的非传统方式，人们声称你可以打开一瓶酒。我们对他们进行了测试。



Tip Tester 是一个新的视频系列，我们在其中评估了普遍吹捧的“生活窍门”，今天我们参加了经典的人与科克之战。假设你把开瓶器放错了地方，你需要用手头的任何东西来应对这种情况。根据我们在网上找到的常见建议，我们试着用螺丝[回形针](http://www.wikihow.com/Open-a-Wine-Bottle-Without-a-Corkscrew) 和鞋子 打开一瓶酒 [。这些选择有可能吗？当然，但是我们不推荐它。](http://lifehacker.com/use-a-hammer-and-screw-to-open-wine-without-a-corkscrew-5044209)
