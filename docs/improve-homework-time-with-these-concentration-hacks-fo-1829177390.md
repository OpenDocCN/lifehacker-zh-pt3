# 用这些让孩子集中注意力的方法来改善家庭作业时间

> 原文：<https://lifehacker.com/improve-homework-time-with-these-concentration-hacks-fo-1829177390>

对于许多家庭来说，家庭作业时间是一种挣扎。而在 [这个数字分神的时代，](https://offspring.lifehacker.com/how-to-help-kids-concentrate-in-the-era-of-digital-dist-1823528550) 的日子只会越来越不好过。这里有一些帮助你的孩子集中注意力的窍门。如果你把它们用在自己身上，你也不会有任何评价。

### 为全家设立“工作时间”

在大学期末考试的那一周，我们凌晨 1 点钟从房间里爬出来坐在拥挤的自习室里是有原因的。同病相怜的人的存在在某种程度上激发了我们的活力，帮助我们坚持读完了经济学 19e 的最后一章。通过建立一个专门的工作时间，在家里创造一个类似的氛围，家里的每个人都坐在一起静静地工作。如果可能的话，让它在每个工作日晚上的同一时间进行(你可以设置一个闹钟或者让 Alexa 宣布它的工作时间，这样有助于减少一些谈判)。当孩子们做作业的时候，你也可以做自己的非电子类工作——整理邮件、签署文件、写感谢信、写日志、在脑海中构思小说或者为第二天做清单。

### 给你的孩子“抱怨时间”

你知道:如果你的孩子倾向于抱怨作业，告诉他们停止抱怨通常会导致更多的抱怨。也许他们需要的是一种解脱。当他们表现沮丧时，宣布他们有一分钟的“抱怨时间”然后启动计时器。在那一分钟里，他们可以向世界发泄他们的愤怒。他们可以倒在地上，挥舞着拳头，惊呼:“这太难了，为什么作业会存在，一切都很糟糕，为什么为什么为什么，啊啊啊啊啊！”仅此而已。时间到了，他们必须回去工作——不许再抱怨了。对于更长的学习时间，你可以给你的孩子两到三张“抱怨时间”票，他们可以根据需要使用。

### 改变环境

虽然有一个专门的空间做作业很好，但是如果你的孩子在学习的时候在房子里走来走去，他们可能会吸收更多的材料。学习教练 [安娜睫毛膏](https://www.youtube.com/watch?v=rzd6Obgpjso) 解释道:“假设你在厨房学习数学，然后你在图书馆学习数学，然后你在公共汽车上学习数学，大脑会像这样，‘嗯。她在许多不同的环境中使用这些数学公式。也许这些数学公式对安娜的生存至关重要。让我们巩固这些数学公式，因为嘿，她在任何地方都使用它们，所以它们一定很重要，对吗？“我知道当我写 的时候，能够在房子周围漫步帮助我获得清晰——我经常在新的环境中找到新的视角。如果你的孩子感到烦躁或卡住，鼓励她尝试一个新的位置，如她的豆袋椅或前门廊或地板上的地毯。

### 让你的孩子成为蝙蝠侠

如 [字面意思让你的孩子穿上超级英雄斗篷。](https://offspring.lifehacker.com/let-kids-dress-as-superheroes-while-doing-homework-to-i-1796821553) 一项 [研究](http://onlinelibrary.wiley.com/doi/10.1111/cdev.12695/abstract) 发现，在执行一项重复任务时，假装成蝙蝠侠等熟悉角色的四六岁儿童比那些保持原样的儿童坚持的时间要长得多。原因与所谓的自我疏远有关。报告称，当孩子表现出与自己不同的角色时，这有助于他们“超越紧急情况，并以更远的视角看待问题”。虽然皮蒂可能想在三位数加法的挑战下崩溃，但蝙蝠侠肯定不会。

### 试试铅笔游戏

这里有一个很好的给那些对于蝙蝠侠来说太老的孩子。在《失败的礼物》一书中，作者杰西卡·莱希讨论了铅笔游戏，这是儿童心理学家威廉·胡登科教给她的一种重新聚焦技巧。

> 给孩子一套两到三支的铅笔，教他在感觉到大脑分心的时候换铅笔。现在，这实际上不是关于他用哪支铅笔，而是切换铅笔给了他的大脑提示，“哦，我分心了，我需要再次集中注意力，”切换铅笔成为他大脑重新集中注意力的信号。

> 最终，他将不再需要铅笔，因为他的大脑最终将学会识别注意力分散，他将开始自动转移他的注意力，但在开始时，铅笔提供了一种方式来意识到注意力分散的微妙信号。

### **将作业时间放在剧烈游戏时间之后**

[学习](http://www.nationalacademies.org/hmd/Reports/2013/Educating-the-Student-Body-Taking-Physical-Activity-and-Physical-Education-to-School/Report-Brief052313.aspx) 后 [学习](http://portal.research.lu.se/portal/en/publications/physical-activity-during-growth-effects-on-bone-muscle-fracture-risk-and-academic-performance(2724198a-285f-4c49-81e1-23cc6224f130).html) 将锻炼与学业提高联系起来。让你的孩子充满活力地跑 30 分钟，可以向他们的大脑输送额外的血液，提供氧气和营养物质，让他们以最高效率工作。(附:这就是为什么 [我们不能切](https://www.forbes.com/sites/leeigel/2015/01/15/the-common-core-is-taking-away-kids-recess-and-that-makes-no-sense/) 。)在最后一次上课铃响和家庭作业时间之间安排一次玩耍休息。一旦所有这些摆动都出来了，你的孩子将更有准备去处理这些书。
