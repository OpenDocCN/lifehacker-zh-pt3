# 暴风雨之海桌面

> 原文:[https://lifehacker.com/the-stormy-sea-desktop-1783758834](https://lifehacker.com/the-stormy-sea-desktop-1783758834)

这个桌面足够简单，Flickr 用户 [jsmorley](https://www.flickr.com/photos/28246402@N06/) 解释说，他们想要的只是个性化一点的东西，而不是超出顶部的东西。最终结果是这个引人注目的桌面，任何人都可以轻松复制。以下是方法。

Watch

这是一个雨量计桌面，你知道这意味着什么。如果你对 [不熟悉，我们的导游可以给你基本的](http://lifehacker.com/how-to-create-an-attractive-customized-desktop-hud-wit-5828789) ，真的，这就是你所需要的。

以下是你需要的其他东西:

*   DeviantArt 的 Grivetart[壁纸](http://www.deviantart.com/art/The-Big-One-461509333)
*   用于 Windows 的 [雨量计](http://www.rainmeter.net/) 系统调试和监控工具
*   [TextClock2 皮肤为雨量计](http://jsmorley.deviantart.com/art/TextClock-2-1-609165064)T3】

这一次到此为止。只有一个皮肤，主要是因为墙纸创造了所有的戏剧。当然，如果你想要更多的显示器和其他东西，你可以随时添加，但是这种简洁的外观非常好，设置也非常简单。
