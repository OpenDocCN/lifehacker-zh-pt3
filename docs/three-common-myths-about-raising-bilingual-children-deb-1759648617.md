# 抚养双语儿童的三个常见误区

> 原文:[https://life hacker . com/three-common-myths-about-raising-双语-children-deb-1759648617](https://lifehacker.com/three-common-myths-about-raising-bilingual-children-deb-1759648617)

到两岁时，孩子通常能说几百个单词。我的儿子亚历山大几乎能听懂两种语言的所有内容——希腊语和英语——但他只会说六个单词。像许多父母一样，我们质疑我们是否做错了什么。

当我们看到更小的孩子超过他的说话能力时，我们越来越担心。(即使是专家也无法逃脱为人父母带来的恐惧和内疚)。

围绕双语现象有许多流传已久的神话，比如它会导致语言延迟和认知障碍。然而，研究表明，用双语抚养孩子不会导致语言学习困难。任何语言发展的滞后都是暂时的，所以父母不应该担心！

以下是一些被揭穿的更常见的神话:

### 用双语抚养你的孩子会导致发育迟缓

不是真的。事实上它有许多优点，比如改善了的(心理规划) [【元语言意识】](http://www.sciencedirect.com/science/article/pii/001002779090030N) (将语言视为抽象单位的能力) [心理灵活性](http://www.sciencedirect.com/science/article/pii/S1364661312000563) (适应性处理信息) [创造性思维](http://ijb.sagepub.com/content/17/4/431.short) 。

双语儿童通常会在语言发展的正常范围内遇到发展里程碑，但在某些情况下 [可能会接近尾声](http://journals.cambridge.org.ezproxy.uws.edu.au/action/displayAbstract?fromPage=online&aid=8448366&fileId=S0305000910000759) (亚历山大的情况正是如此)。

### **双语儿童落后于同龄人，不会赶上**

这是一个有争议的问题，因为双语儿童的差异很大。 [有些孩子根本不会表现出任何迟滞](http://journals.cambridge.org/action/displayAbstract?fromPage=online&aid=9370996&fileId=S0142716412000744) 。

有人提出，暂时的滞后可能源于不得不在同一个大脑中适应两种语言系统，但这些孩子将在几个月内赶上(注意，这与语言延迟不同)。

但是需要更多的研究来了解确切的机制。

### 我的孩子会混淆这两种语言

假的。尽管关于这两种语言何时分离还存在一些争议。

长期以来，人们认为两种语言最初是融合在一起的，在孩子五岁左右开始分离。最近的证据表明， [语言可能比之前认为的](https://books.google.com/books?hl=en&lr=&id=zfYYXKxPek0C&oi=fnd&pg=PA327&dq=early+bilingual+language+development+one+language+or+two&ots=f4qS-isYtr&sig=GLod1Ez_ilqwwRGIhr6WCYJWSaE#v=onepage&q&f=false) 更早分离。

例如，10-15 个月大的双语儿童根据他们与 互动的对象不同而牙牙学语 [(例如，英语牙牙学语听给母亲听，法语牙牙学语听给父亲听)。](http://talktotheslp.com/wp-content/uploads/2014/07/Myths-of-Bilinguilism-1.pdf)

这表明婴儿从很小的时候就对他们在和谁说话很敏感。这可能是语码转换的前兆(双语者在同一个话语中使用两种语言)。

### **父母双语抚养孩子的五条建议**

1.  像对待任何婴儿一样鼓励和耐心，并意识到双语儿童比只学习一种语言的儿童面临更艰巨的任务。
2.  两种语言都有实用的目的，这一点非常重要。毕竟，语言是交流的工具。如果孩子不需要使用另一种语言，他们可能会停止使用它。因此，重要的是始终如一地 [将孩子置于需要使用两种语言的情境中](http://journals.cambridge.org/action/displayAbstract?fromPage=online&aid=2747404&fileId=S0142716400009863) ，最好是 [与各种说话者](http://www.ncbi.nlm.nih.gov/pubmed/25811169) 。这样做将在每种语言中发展强大的语音类别，并确保他们学会有效地处理语音——这将有助于听和说。
3.  许多家长担心平衡的问题，也就是说，孩子对两种语言的掌握程度是否一样好。过去，人们认为要成为真正的双语者，你需要对两种语言都有同等的掌握。我对非常熟练的双语者 [进行了一系列研究，并一次又一次地观察到](http://www.sciencedirect.com/science/article/pii/S0095447010000720)[甚至流利的双语者](http://www.sciencedirect.com/science/article/pii/S0095447012000411) 也有一种 [占主导地位的语言](http://scitation.aip.org/content/asa/journal/jasa/133/4/10.1121/1.4792358) 。因此，强调一个孩子没有完全掌握每种语言是没有意义的，因为事实是几乎没有人能做到。
4.  当双语儿童混合他们的语言时，父母通常会感到担心。不要担心。这是双语发展的一个正常部分，而不是混乱的标志。甚至熟练的双语者 [混合他们的语言](http://www.sciencedirect.com/science/article/pii/S0095447011000258) 。
5.  如果你担心你的孩子的语言发展，你应该让你的孩子接受医生的评估，如果有必要的话，接受语言病理学家的评估。和其他孩子一样，双语儿童可能会出现语言延迟。如果您的孩子有语言障碍，可能需要早期干预来帮助他们学习语言。
