# 这个 Chrome 扩展给了 Trello 一个夜间模式

> 原文：<https://lifehacker.com/this-chrome-extension-gives-trello-a-night-mode-1820592289>

我喜欢尽可能使用夜间模式运行应用程序和网站。即使在白天，深色对你的眼睛也更容易，你甚至可以通过将亮白色屏幕换成柔和一点的屏幕来节省笔记本电脑或智能手机的电池电量。

Watch

现在我的清单上又多了一个夜间模式:特雷罗。

我用 Trello 来组织我生活中的几乎所有事情。网站的背景是典型的亮蓝色，这比白色好，但仍然有点多，特别是如果你像我一样整天开着它。

一个第三方开发者和 Trello 爱好者最近提出了一个 Chrome 扩展形式的解决方案。有了它，当你在黑暗的地方观看时，你可以在晚上(或永远)将你的 Trello 板变成较暗的色调，然后当你准备好了(或者，你知道，永远不会)时，将它变回明亮的蓝色。

如果你是 Trello 用户，你可以在这里 免费获得 Chrome 扩展。
