# 接下来我们应该做什么？

> 原文：<https://lifehacker.com/what-should-we-sous-vide-next-1823468977>

各位网友大家好，欢迎来到全新的 [*选题会，会吗？*](http://lifehacker.com/tag/will-it-sous-vide) ，列在那里我用我的沉浸循环器做任何你想让我做的事。



上周的菜肴绝对是一次肉多味美的成功，但现在我们必须把注意力转向新的、令人兴奋的食物。我当然欢迎你的任何想法，但我也有自己的一些想法:

*   我们可以做一大堆蛋奶沙司，把它倒入四个罐子里，给每个罐子加上不同的味道，做成一个漂亮的小样品。
*   我还没有做出令人惊叹的土豆，但我愿意继续尝试。
*   烤牛肉:它肯定不会是干的，而且我们会有很多肉汁。
*   **蛋卷:**我们可以根据你的口味定制，*和*你不必去星巴克。
*   五花肉:我不知道为什么我们还没有这样做，但我很确定它会很好吃。

像往常一样，确保在你最喜欢的上面打星号，并在评论中留下你自己独特的、美味的想法。
