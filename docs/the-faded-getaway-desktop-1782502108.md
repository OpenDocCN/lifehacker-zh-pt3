# 褪色的逃亡桌面

> 原文：<https://lifehacker.com/the-faded-getaway-desktop-1782502108>

[Ciysco](http://kinja.com/ciysco) 向我们的 [桌面展柜](http://kinja.com/tag/desktop-showcase) 提交了这个定制的桌面，我们很喜欢。不错的壁纸，角落里褪色的应用启动器，还有可视化工具。它看起来很锋利——以下是你如何让它成为你自己的。

Watch

首先，Ciysco 是用 Rainmeter 得到这个效果的，在 Windows 7 中。如果你不熟悉它，看看我们的雨量计 入门指南，看看它有什么功能。

一旦你有了基本的东西，下面是你需要的:

*   [壁纸](http://wallup.net/photography-nature-landscape-water-sea-coast-rock-5/)
*   Windows 的 [Shine 2.0 主题](http://zainadeel.deviantart.com/art/Shine-2-0-for-Windows-7-172671190)
*   用于 Windows 的 [雨量计](http://www.rainmeter.net/) 系统调试和监控工具
*   [为雨量计的木子皮](http://undefinist.deviantart.com/art/VisBubble-Round-Visualizer-for-Rainmeter-V3-0-488601501>Visbubble skin for Rainmeter</a> to create the virtualizer on the right side</li>
    	<li>The <A href=)为音乐播放器的
*   [为雨量计的彩色流皮](http://wyzzymoon.deviantart.com/art/Patrick-Muff-s-Color-Flow-for-Rainmeter-532426147) 为显示在中央的时间和日期
*   [简单回收站](http://maxvanijsselmuiden.deviantart.com/art/Simple-Recycle-Bin-201077076) 右下角自定义回收站
*   栅栏用来整理桌面物品

Ciysco 还指出，图标大多是定制的，但提供了下载:

> 这些图标大多是我自己从网上编辑的，但也有一些来自这个 [Reddit 帖子](https://www.reddit.com/r/Windows10/comments/3fjaj5/i_made_some_white_icons_for_the_taskbar/) 。我在这里提供了所有图标的下载链接(来自 Dropbox 的 zip 文件)。

总而言之，这是一个很好的外观，如果你喜欢不同的壁纸，或者想放弃围栏做其他事情，或者只是保持一个干净的桌面，这种外观很容易复制。如果你和我们一样喜欢它，请访问 Ciysco 的 Kinja 博客，链接如下，让他们知道你有多喜欢它——或者提出你的问题，让他们知道是我们邀请了你。
