# 使用 GasBuddy 追踪您的低油耗驾驶习惯

> 原文:[https://life hacker . com/now-gasbuddy-will-track-your ' s fuel-efficient-driving-h-1821368647](https://lifehacker.com/now-gasbuddy-will-track-your-fuel-inefficient-driving-h-1821368647)

不管你住在哪里，汽油都很贵。GasBuddy 是人们多年来用来寻找最便宜的加油地点的应用程序之一，现在它有了一个新功能，可以帮助你让燃料持续更长时间。

Watch

本周，该公司推出了一项新的 Trips 功能，可以精确定位你在何时何地犯了低油耗驾驶罪。这个想法是，如果你可以看到当你在特定的街区或速度或特定的高速公路上开车时，你总是刹车过度，你就可以看到这些问题发生在哪里，并调整你的驾驶习惯。

在每次驾驶结束时，你会得到一个基于三层系统的驾驶评估。你要么将你的旅行指定为“棒”、“不坏”或“不棒”

除了评级之外，您还将获得一份旅程摘要，其中包含您的旅程日期、时间和行驶距离，以及一张地图，该地图突出显示了您在旅途中遇到麻烦的确切位置以及您是如何造成麻烦的。

这不如实时知道你在做什么来烧穿你的坦克。如果你对此感兴趣，我建议试试 Dash 或 Automatic(这两者都需要一个连接到 OBD 传感器的设备来实时提供信息)。

如果你还没有准备好购买汽车监控硬件，那么这可能是一个很好的解决方案，至少可以感受一下哪里出了问题。
