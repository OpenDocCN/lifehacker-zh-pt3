# 如果一个网站最近被黑客攻击，这个 Chrome 扩展会提醒你

> 原文：<https://lifehacker.com/this-chrome-extension-alerts-you-if-a-website-was-recen-1828201736>

保持最新的安全信息是很困难的，除非你看到某个网站或服务泄露了你的用户名和密码的头条，否则你很可能不会去想网站黑客。



史蒂夫·托马斯(现已名誉扫地的 PwnedList 网站的联合创始人)创建了一项名为 HackNotice 的免费新服务，让你可以更方便地了解你在 Chrome 浏览器中浏览的网站最近是否遭到过黑客攻击。你所需要的只是一个本月早些时候发布的简单的 [Chrome 扩展](https://chrome.google.com/webstore/detail/hacknotice-extension/kbjanbmmokbpaibafdgehgkpncpdjbpf) 。

该扩展的 [前提](https://blog.hacknotice.com/2018/08/01/announcing-hacknotice-chrome-extension/) 很简单:访问 HackNotice 不断增长的被黑网站列表中的一个网站，你会在浏览器中得到一个小小的弹出警告，提醒你注意黑客攻击。点击它，你将能够跳转到一个 HackNotice 列表，它提供了关于发生了什么的更多信息以及相关新闻文章的链接。

虽然该应用程序不会警告你黑客是否直接影响了你(T1)，但你至少会得到一个警告，你可能需要调查你最喜欢的网站(或多个网站)。

我对 HackNotice 扩展的唯一问题是，第一次设置它会有点麻烦。在你的浏览器中安装扩展，然后访问 [HackNotice 的网站](https://www.hacknotice.com/) 注册账户。当你这样做的时候，你会被带到一个巨大的仪表盘。暂时忽略它，点击网站左侧边栏上的“首选项”。向下滚动直到你看到“Chrome 浏览器扩展”列表，然后点击“生成密钥”这将扩展绑定到您在网站上的帐户，并且似乎是在它可以警告您任何事情之前的必要步骤。

复制密钥，点击 Chrome 地址栏附近的 HackNotice 扩展图标，粘贴进去，然后点击“提交”你会注意到它起作用了，你可以像往常一样继续浏览网页。如果你在屏幕右下角看到一个通知，说你正在访问的网站被黑客攻击了，小心行事——也许要修改你的密码，以防万一:
