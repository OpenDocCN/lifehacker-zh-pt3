# 空荡荡的桌面

> 原文：<https://lifehacker.com/the-empty-desktop-1743764184>

抛开左下角的时间、电池电量、音乐，你可能以为这只是一张壁纸，其实不是——这是 [丹 C](http://kinja.com/dncrrnz) 的桌面，他喜欢空的。每次他坐下来，都是一个辉煌的新开始。他是这样做的。



Dan 正在开发一台 Mac 电脑，公平地说，他把 dock 隐藏起来，直到他需要它的时候——但即使这样也是定制的。当他坐下来工作时，这就是他所看到的。如果你喜欢这个造型，下面是你需要的:

*   [壁纸](http://fiftyfootshadows.net/2013/07/30/clack/) 作者 [五十岁的约翰·凯里](http://fiftyfootshadows.net/)
*   [GeekTool](http://projects.tynsoe.org/en/geektool/) 系统监视器和用于 OS X 的系统时间和电池指示器的配置实用程序
*   [cDock 2](http://w0lfschild.github.io/cdock) 定制 Dock
*   [LiteIcon](http://freemacsoft.net/liteicon/) 自定义 dock 图标
*   [锡安](http://alexlikesdesign.com/zion/) 图标包
*   [蝴蝶结](http://bowtieapp.com/) 在系统信息下以未命名的主题为“正在播放”显示
