# 列出一个“接朋友”的名单，这样你就不用支付日托的滞纳金了

> 原文：<https://lifehacker.com/put-together-a-pick-up-pals-list-to-save-you-from-day-1797453321>

你知道这种感觉，被堵在路上或拥挤的公交车上，争分夺秒地赶在日托滞纳金开始之前去接你的孩子。下一次，不要紧张，把你的“接朋友”联系人名单放在你手边。你可以避免滞纳金，你的孩子也不会因为最后一个去托儿所而感到难过。



接朋友名单背后的想法很简单:如果有人发短信说他们要迟到了，你和其他几个父母同意在日托中心接对方的孩子。你名单上的人越多，你就越有可能在同一时间有另一位家长来接他们的孩子，并帮你签出你的孩子。如果你记住以下几点，这将会更顺利:

*   **小组中的每位家长可能需要向日托中心填写一份表格**,确认你或接孩子小组中的其他人可以带走他们的孩子。
*   **团队中的每个人都需要为保护他人做出贡献**，而不仅仅是利用团队来接走他们的孩子。
*   **就通信协议**达成一致，以免群发信息过载。例如，一旦有人回应说他们可以带走你的孩子，就直接告诉他们你会在何时何地遇到他们。
*   有一个标准的时间限制另一方父母将有你的孩子。当然，如果你被堵在可怕的交通中，你可能不会遇到它，但一般来说，父母应该只让你的孩子呆一小段时间。这不能替代保姆服务，除非团队同意。

你可以根据自己的情况定制这个想法。也许你想把可靠的朋友或家人添加到你的朋友列表中。你可以列出朋友名单，从日托到过夜，到约会和生日聚会。
