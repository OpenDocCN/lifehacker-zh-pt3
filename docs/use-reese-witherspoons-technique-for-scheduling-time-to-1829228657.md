# 使用瑞茜·威瑟斯彭的技巧来安排阅读时间

> 原文:[https://life hacker . com/use-Reese-witherspoons-technique-for-scheduling-time-to-1829228657](https://lifehacker.com/use-reese-witherspoons-technique-for-scheduling-time-to-1829228657)

对于那些以阅读为乐的人来说，找时间看书可能很难适应充满无尽任务的一天:工作、锻炼、孩子的活动和家务占据了我们的大部分时间；适应一本好书可能意味着牺牲另一本。

Watch

小说提供了逃避的选择，但对许多人来说，特别是那些阅读这个网站的人，还有一种感觉，你读的东西应该有助于你成为一个更好的人——它应该给你一两个生产力提示，传授新知识或告诉你如何最大限度地利用你的时间。外面有很多好书。

你如何平衡这两者？我喜欢瑞茜·威瑟斯彭最近的新书《采访纽约时报[](https://www.nytimes.com/2018/09/20/books/review/by-the-book-reese-witherspoon.html)*中的这个建议:*

> *我喜欢在早上阅读非小说类或“思想”类的书籍——查尔斯·杜希格的《习惯的力量》，诸如此类。晚上，我读小说来让我的大脑平静下来，并扩展我对可能性的感知。我最期待的是优雅的句子，把我带入一个不同的世界。*

*当然，这并不是说你不能从小说中学到任何东西，但是如果你乘公共汽车或火车上下班，阅读非小说类书籍可以帮助你进入正确的工作状态。晚上，当你完成了一天的工作后，你可以看一本好的小说放松一下，把自己带到另一个世界，这似乎是一种很好的入睡方式。*
