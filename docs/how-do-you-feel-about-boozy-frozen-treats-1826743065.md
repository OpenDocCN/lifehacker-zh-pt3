# 你觉得冷冻烈酒怎么样？

> 原文：<https://lifehacker.com/how-do-you-feel-about-boozy-frozen-treats-1826743065>

大家好，欢迎回到 [*有什么好吃的？*](http://lifehacker.com/tag/whats-cooking) ，这是一个开放的话题，你可以在这里分享你对所有食物相关的想法、建议、食谱和观点。今天我想和你们谈谈最夏日的噱头:冷冻酒精。



在一年中的这个时候，我在想要制作和消费所有的烈酒冰棒和抨击整个流派是廉价的噱头之间摇摆不定。(“就像成年人一样吃个冰棍喝个鸡尾酒”是我不止一次嘀咕的话。)但是，就像任何事情一样，有一种正确的方式和一种错误的方式来处理冰冻甜点。我相信你们对这个问题都有很多意见，我也有很多问题:

*   **首先也是最重要的，你有最喜欢的冷冻饮酒方式吗？**我非常喜欢 granita，我认为他们被大大低估了。 [他们很容易做出](https://lifehacker.com/turn-extra-champagne-into-a-delicious-frozen-granita-1790577807) ，基本上都是成人版的雪糕筒。与冰棍和冰淇淋不同，它们非常宽容，因为你不必太担心所有东西都保持冷冻状态。
*   你觉得弗洛斯怎么样？我认为如果你添加足够的糖，这对于廉价的桃红葡萄酒来说是一种不错的用途，但是冷冻一种真正好的桃红葡萄酒只会减弱它的味道，让我生气。
*   **混合鸡尾酒怎么样？**我更喜欢加冰的玛格丽塔酒和我的代基里酒，但如果一杯冰冻混合尼格罗尼酒不是非常好和特别，我会被诅咒的。极度的寒冷缓和了苦味，我经常喝得太快以至于大脑冻结。
*   你吃过真正美味的酒精冰棒或冰淇淋吗？我几年前做的 [酸樱桃啤酒](https://skillet.lifehacker.com/how-to-turn-any-alcohol-you-like-into-tasty-frozen-pops-1782920583#_ga=2.75514389.1445189089.1528729471-426097860.1521480505) 味道很好，也很容易做，但是我还没有找到一种我喜欢其质地和味道的酒精冰淇淋。
*   你喝过可怕的冷冻酒吗？我经历了一些真正可怕的霜冻。全是酸，没有甜味。
*   烈酒奶昔被高估了吗？我倾向于认为它们是，不是因为它们味道不好，而是因为所有的脂肪减缓了我的嗡嗡声，所有的糖使我的宿醉更严重。不过，我喜欢在咖啡冰淇淋上来一杯布兰卡·门塔。

和往常一样，你可以随意回答上面的问题，也可以随意评论我们的话题。唯一的要求是，你的评论至少与某种程度上的冷冻酒有关。
