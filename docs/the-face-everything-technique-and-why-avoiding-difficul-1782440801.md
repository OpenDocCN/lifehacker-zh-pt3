# 面对一切的技巧和为什么逃避困难不起作用

> 原文：<https://lifehacker.com/the-face-everything-technique-and-why-avoiding-difficul-1782440801>

我们所有人都很擅长逃避。我们的大脑与其说是“思考机器”，不如说是“回避机器”不可思议的是，我们通常甚至没有意识到我们在回避思考一些事情。

我给你举几个例子:

*   现在你正在读这篇文章，但是可能在回避你不想去想的困难的事情。
*   我们不断地检查消息，新闻，饲料，通知...来避免做一些我们不想面对的事情。
*   当我们在生活中面临困难时，我们试图告诉自己这没关系，因为(填空)，或者忙于一些活动或麻木剂(如酒精)，这样我们就不必面对困难。
*   当问题出现时，我们的反应是想去做别的事情，把它推迟。
*   我们推迟支付账单、纳税、处理长邮件、处理杂乱，因为我们不想面对这些困难。
*   我们推迟锻炼，因为不舒服。

事实上，每天都有成千上万的例子冒出来，而我们甚至没有注意到，因为我们的大脑在想别的事情。现在就试试这个:暂停一分钟，想想你现在在避免思考什么困难。

你要么会注意到一个你不喜欢的困难，要么在一分钟结束前你的思想会很快转向做别的事情。

你所做的是我称之为“面对一切”技术的一部分，我会在我们讨论完为什么回避一切是无效策略之后解释这一点。

### 逃避不起作用

我们的大脑想要逃离我们面临的任何不适、痛苦和困难，这是暂时不用处理困难和痛苦的一个好策略。所以在当下，我们可能会感到暂时的解脱。

但是它让我们陷入了奔跑的生活。分心的生活，从不面对困扰我们的事情。我们让自己忙碌起来，却从来没有学会处理我们内心的东西，我们面前的东西。

这意味着我们受到恐惧和不适的支配。我们就像小孩子，不想做任何辛苦的工作，却想要最新闪亮好玩的东西。

这导致不去做重要的工作(或者至少把它推迟到开始变得痛苦的时候)。锻炼、健康饮食、财务、杂乱、人际关系等等也是如此。

最后，我们通常不得不处理这些事情，但它们只会变得更糟。如果早一点面对它们会更好，那时它们还不是什么大事。

### 面对一切的技巧

这种技巧是基于这样一种想法，即最好是意识到事情，并像成年人一样处理它们，而不是逃避。就算有，也没什么大不了的。

它是这样工作的:

1.  通过问“我现在在做什么？”来创造意识在一天中，设置提醒或写小纸条，提醒你问自己:“我现在在做什么？”答案可能是，“检查脸书”，或者“切换到一个新的浏览器标签”，或者“吃一些芯片”像这样简单而平凡的事情，但是问问你自己你在做什么，开始带来意识。
2.  接下来，问问你自己，“我在回避什么？”当事情变得困难或不舒服时，我们会自动切换到其他事情。我们跑。我们疯狂地躲避。你整天都在做，却没有意识到。问问你在避免什么:一些恐惧，一些困难的任务，一些困难的情绪，一些不适，或者只是停留在当下？说出你在回避什么。
3.  **现在面对它**。就停留在这恐惧、不适、困难的当下。不是你在头脑中告诉自己的关于它的故事，而是当下你身体中真实的身体感觉。有多糟糕？你会发现这没什么大不了的。再坚持一会儿。再过一会儿，挑战你自己。
4.  **采取适当的行动**。既然你已经面对了，并且发现这没什么大不了的，你可以表现得像个成年人而不是小孩子:你可以决定现在最好的行动是什么。如果你害怕做某项任务，但你已经面对过它，并看到恐惧并不是什么大不了的事情，你可以提醒自己，这项任务将有益于你和他人，并且比你的小小恐惧重要得多。如果你因为生气而避免与某人进行艰难的对话，你会发现愤怒和冒犯并不是什么大不了的事情，你可以带着同情和怜悯心，平静而恰当地与那个人交谈，并找出解决办法。

当然，并不是所有的问题都会因为使用这种方法而烟消云散，但是我可以告诉你，当你练习这种方法时，你将能够面对更多的事情。你会更好地处理不适，而不是像大多数人那样逃避。你会更擅长不拖延，不做不舒服的事情。你会更享受当下，更愿意停留在当下，而不是一直需要分心。不是一夜之间，而是通过练习。

您可能会有忽略这篇文章的冲动，以避免练习这种技术。那也是回避，我敦促你现在就面对它。
