# 用三种配料制作真正的炸弹蓝奶酪酱

> 原文：<https://lifehacker.com/make-truly-bomb-blue-cheese-dressing-with-three-ingredi-1828526803>

蓝纹奶酪酱比沙拉大。当然，它是楔形沙拉不可或缺的一部分，但它也是唯一重要的翼勺，它让一个悲伤的蔬菜拼盘变得可以忍受。商店买的很好，但是你可以只用三种原料自己制作真正的炸弹蓝奶酪酱。



第一种——也是最重要的——配料是蓝纹奶酪，你可以选择你最喜欢的*。想要超级奶油和轻度辛辣的东西吗？伸手去拿糖果。渴望一点唐？洛克福特抓住了楚。你甚至可以用一些什罗普郡干酪来制作橙蓝色奶酪——为了纪念我的母校。关键是你掌控着自己咸咸的、时髦的命运。*

你唯一需要的其他东西是一些高质量的蛋黄酱，一点雪利酒醋(最好的醋)，和一些混合设备。(我用的是我的浸入式搅拌机和附带的杯子。)加入大致等量的蓝纹奶酪和蛋黄酱(汤匙)，混合，然后加入醋，每次半茶匙，直到变稀到你想要的浓度。如果你喜欢的话，你也可以加入一点新鲜的胡椒。你现在有了一种奶油的、咸的、略带香味的、以奶酪为主的调料，可以用来倒酒、调味和蘸酱，你的生活会因此变得更好。
