# 使用这个 Anki 插件可以更快地学习中文

> 原文：<https://lifehacker.com/learn-chinese-faster-with-this-anki-add-on-1828938649>

在阅读我所涉猎的少数几种外语之一时，我最大的挫折之一是要花多长时间来查找所有我不认识的单词(诚然，这些单词很多)。这是破坏性的，我很可能在看完一篇文章、一章或一页之前就放弃了。



对于任何学习中文的人来说，现在有 [中文预习](https://ankiweb.net/shared/info/882364911) ，这是一个 Anki 插件，可以构建词汇抽认卡，以提高阅读速度和理解能力。理想情况下，如果你在写一本书，你总是提前几页学习单词(因此是“前”)，这样你的阅读进度会受到最小的干扰。

开发人员杰克·金塞拉的 [扬基方法](https://lifehacker.com/the-janki-method-shortens-the-time-itll-take-you-to-lea-5973887) 中使用的 Anki 抽认卡系统用于学习代码，本质上是使用间隔重复的抽认卡，向你显示一个单词、短语或图像你需要永久记住它的最少次数。

在你可以使用这个插件之前，你必须先下载 Anki 本身。请按照页面上针对您的操作系统的说明进行操作——请注意，中文预习仅与 Anki 2.1 兼容，因此如果您已经有 Anki 2.0，您必须升级到新版本。一旦你下载并启动 Anki，你将被要求选择并确认界面语言。这应该是你的首选语言或母语，而不是你想学的语言。

如果你是 Anki 的新手，那么 YouTube 上有一个冗长而详细的用户手册，以及 [介绍视频](https://apps.ankiweb.net/docs/manual.html#intro-videos) ，来帮助你导航和定制程序。一旦你有了基本的东西，你就可以安装中文预习了。

*   转到菜单栏中的工具>附加组件。
*   单击“获取加载项”按钮。
*   粘贴中文预习的代码，在这里找到，进入弹出框，点击确定。
*   该附加组件应出现在前一个窗口的附加组件列表中，并作为一个选项出现在工具菜单的底部。如果没有，重启 Anki。

要使用中文预习，你需要可以复制和粘贴的文本——例如，新闻报道或书籍 PDF。作为一个完全的初学者，我选择了一篇 [BBC 文章](https://www.bbc.com/zhongwen/simp/sports-45463356) 关于一个我已经熟悉的故事:小威廉姆斯的美国公开赛失利。一旦你有了你的文本:

*   使用添加功能创建一个新的卡片组来存放你的抽认卡。否则，Anki 会将牌添加到默认牌组中。这很好，尽管如果您将 Anki 用于其他用途，拥有一个专门命名的集合对学习很有帮助。
*   进入工具>中文预习。
*   将您要阅读的文本粘贴到对话框中，然后点击继续。
*   下一个窗口允许你选择你的目标词汇量。我带着推荐的 3500 个最常用词去了，因为为什么不呢？点击继续。
*   选择您之前创建的卡片组，并点击添加注释。

就是这样！要学习，点击甲板名称，并点击学习现在。点击“显示答案”后，每张卡片将显示简化字、拼音、定义和例句。按照 Anki 的方法，你会根据你对这个词的了解程度，以一定的频率再次看到这个词。

该插件目前只接受简体中文字符作为输入，所以如果你选择的文本有繁体字，你现在就不走运了。但是，它确实创建了传统的字符卡供您学习。默认情况下它们是暂停的，所以要取消暂停，进入桌面上的 Anki 主窗口，点击浏览，在搜索框中输入“卡片 :HanziTrad ”。使用 Ctrl+Alt+A(Mac 上为 Command+Option+A)选择全部，并按 Ctrl+J (Command+J)取消暂停。

有配套的 Anki 应用程序，既适用于[iOS](https://itunes.apple.com/us/app/ankimobile-flashcards/id373493387?mt=8&ign-mpt=uo%3D4)(24.99 美元)，也适用于 [Android](https://play.google.com/store/apps/details?id=com.ichi2.anki) (免费)。中文预习只能在桌面上使用，但你可以使用一种名为 [AnkiWeb](https://ankiweb.net/about) 的服务将你创建的套牌同步到你的移动应用程序。

最后，如果你已经有一些汉语经验，或者如果你把它和其他学习方法一起使用，汉语预习可能效果最好。
