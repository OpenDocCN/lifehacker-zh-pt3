### **大图模式救场**

Steam 的大图模式是你开始在本地将你的实际桌面流式传输到你的 Steam 链接所需要的。您需要将键盘和鼠标(或 Steam 控制器)与您的 Steam 链接配对，以退出大图片模式并正确导航您的 Windows 桌面。在你的主机(在我的例子中是一台 PC)上，启动 Steam。在 Steam Link 上，连接到您的主机 PC。

然后，使用右上角的电源图标，用控制器或键盘选择“最小化大图”。然后，您将能够使用 Steam 控制器和键盘来浏览窗口。有问题吗？罪魁祸首可能是你的显卡，所以一定要检查你的驱动是否是最新的。

### **只需使用记事本**

“最小化大图”可能不是在所有情况下都有效(它只在我的 Steam Link 上有效，在我的 MacBook 上无效)，但你可以尝试另一种访问桌面的方法。要想直接进入 Windows，你需要在你的游戏库中添加一个非 Steam 应用程序。您可以使用任何应用程序，但选择不需要太多处理能力的应用程序是有意义的。这很容易做到，无论是在常规的蒸汽窗口还是在大图模式。

添加一个像记事本这样的简单应用程序比加载 Chrome 这样的东西更快，Chrome 可以在你正确处理桌面后加载。在屏幕顶部的“游戏”标签中，选择“将非 Steam 游戏添加到我的资料库”，然后等待您的应用程序列表填充弹出式窗口。可以自己浏览一个类似记事本的 app(位于 C:\ Windows \ System32 \ Notepad . exe)手动添加。你可以在大图模式下通过访问设置页面，并选择“系统”部分下的“添加库快捷方式”来做同样的事情。

接下来，在您的 Steam Link 或其他设备(如您的笔记本电脑)上启动 Steam，并选择最近添加的记事本。它启动后，你可以在键盘上按 F1(Mac 用户可以按 Fn + F1)来访问 Windows 桌面。你可以在那里做任何事情，从浏览网页到写论文，再到流式传输存储在你的主机上的任何内容。这与远程桌面设置不同，在远程桌面设置中，您可以从任何地方访问您的 PC，但当您只是试图向某人展示您下载到大厅办公室 PC 上的酷视频时，这总比不得不输入并记住登录用户名和密码要好。
