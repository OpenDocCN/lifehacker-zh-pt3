# Google Home 现在可以大声朗读烹饪指南了

> 原文:[https://life hacker . com/Google-home-will-now-read-cooking-directions-out-loud-1794669057](https://lifehacker.com/google-home-will-now-read-cooking-directions-out-loud-1794669057)

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-GdVGynGWQ2Y&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-GdVGynGWQ2Y&start=0) 

当谈到家庭语音助手时，不可否认的是，它们的大部分用途是在厨房设置定时器和播放音频。谷歌的 Google Home 顺理成章地迈出了下一步，它现在可以大声读出菜谱。

Watch

它是这样工作的:

1.  使用安卓系统上的 [谷歌助手](https://assistant.google.com/) 或谷歌搜索(现在只是叫谷歌)app(在 [iOS 系统上](https://itunes.apple.com/us/app/google-search-made-just-for-mobile/id284815942?mt=8) 或 [安卓系统上](https://play.google.com/store/apps/details?id=com.google.android.googlequicksearchbox&hl=en) )选择食谱。
2.  当你找到你想做的食谱时，点击“发送到谷歌主页”按钮。
3.  当你准备开始做饭时，说:“好的谷歌，开始做饭。”
4.  当你准备好下一步时，说，“好的谷歌，第二步是什么，”或者如果你需要重复某件事，说，“好的谷歌，重复。”

这里的想法很棒，但是很难说它实际上在实时工作中有多好。同样有点令人沮丧的是，你只能用谷歌自己的应用程序发送食谱，这限制了你的选择，而且似乎没有任何方法可以真正保存和召回任何食谱。不管怎样，这仍然是朝着一个有趣的方向迈出的一步。和谷歌所有的东西一样，这将在下周左右推出，所以如果你现在不喜欢，以后再试一次。
