# 总统日在一些 AMC 影院免费看《滚出去》

如果你还没有看过这部优秀的恐怖/惊悚电影*出来*，现在是你第二次机会以它应该被看到的方式去看它。本周一，总统日，你可以在精选的 AMC 影院免费观看这部奥斯卡提名电影。

Watch

2 月 19 日的免费放映是为了庆祝这部电影的一周年纪念日，当然也意味着保持奥斯卡的嗡嗡声。所谓免费，我们的意思是完全免费。没有附加条件。也就是说，只有一场晚上 7 点的演出将在全国 55 个不同的 AMC 影院上演，所以机会有限。这是你如何得到你的票:

1.  通过此链接 ，查看您附近是否有参与 AMC 的地点 [。](http://www.getoutoneyearlater.com/) 
2.  周一早上去售票处拿一张免费票。它们只能在当天的售票处买到。确保你带着有效的身份证。每个 ID 赠送一张免费券，送完即止。

3.  那天晚上去剧院看演出，手里拿着票。如果没有指定座位，你可能想早点去抢一个好座位。

我是一个超级恐怖片迷， [你可能知道](https://lifehacker.com/classic-horror-movies-you-can-watch-on-major-streaming-1819688990) ，这部电影毫无疑问是我去年最喜欢的电影之一。《T4 滚出去》不仅是一个有着精湛表演的精彩故事，这部电影，像所有优秀的恐怖片一样，也有很多要说的。如果你本周一不能去电影院看这部电影，你也可以在 HBO Now 上找到它，或者在无数其他电影流媒体服务上租赁和购买。
