# 使用 C.O.A.C.H 加强您的在线安全

> 原文：<https://lifehacker.com/strengthen-your-online-security-with-c-o-a-c-h-1828310041>

你知道你在网上是否“安全”吗？网络安全和隐私是复杂的，风险因人而异:你可能担心被骚扰、被黑，或者你的老板发现你糟糕的旧博客文章，并以此为借口解雇你。 [Crash Override 的自动化网络安全助手](http://www.crashoverridenetwork.com/coach.html) 根据您的需求帮助您保护您的帐户，它一步一步地指导您，使您不会不知所措。



首先，告诉 C.O.A.C.H 你需要做什么:避免你的账户或网站遭到黑客攻击，隐藏你的个人信息，保护你的电脑和手机免受黑客攻击，或者清理旧账户。然后按照说明去做，这些说明太详细了，可能会有点乏味。但无聊总比不知所措好。

例如，为了避免你的在线账户遭到黑客攻击，C.O.A.C.H .会指导你在谷歌、苹果和 Dropbox 等热门账户上设置双重认证。它还可以帮助你安装一个密码管理器，这样你就可以设置更难破解的密码，并且安全地忘记它们。

假设您想要隐藏您的联系信息和地址。如果他们已经在互联网上，这是很难的，但 C.O.A.C.H .会努力。首先，你会被询问是否拥有任何域名或网站(网站的 WHOIS 信息可能会暴露你的地址和联系信息)。然后 C.O.A.C.H 会告诉你如何让它们更私密。接下来，C.O.A.C.H .会帮你把你的名字、家庭信息和家庭住址从寻人数据库 [发言人](https://www.spokeo.com/) 中删除。然后，它会显示网上至少有 272 个类似的数据经纪人，并帮助您选择最受欢迎的数据经纪人。

C.O.A.C.H .是 [崩溃超越](http://www.crashoverridenetwork.com/index.html) 的一项服务，“一个危机帮助热线，倡导团体和资源中心，为正在经历网络虐待的人服务。”你可以查看 [其他崩溃覆盖资源](http://www.crashoverridenetwork.com/resources.html) ，如 [当你面临在线虐待时与家人和警察交谈的指南](http://www.crashoverridenetwork.com/familyandpolice.html) ，以及匿名网页浏览、设备安全保护等服务的链接。

C.O.A.C.H .本身经常链接到其他资源，如 [删除我](http://backgroundchecks.org/justdeleteme/) ，它可以帮助你删除旧的在线帐户，以及 [双因素授权列表](https://twofactorauth.org/) ，它给出了在数百个网站和服务上启用双因素的说明。(C.O.A.C.H .选择的许多资源都是 Lifehacker 的最爱。)就像我们说的，这个东西很复杂。所以尝试一步，如果有帮助，再尝试一步。
