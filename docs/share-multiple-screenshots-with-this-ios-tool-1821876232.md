# 用这个 iOS 工具分享多个截图

> 原文:[https://life hacker . com/share-multiple-screens-with-this-IOs-tool-1821876232](https://lifehacker.com/share-multiple-screenshots-with-this-ios-tool-1821876232)

我的几个朋友之间经常会交换令人捧腹或尴尬的截图，尽管经常会同时收到多张图片，偶尔会出现顺序错误。无论你发送的是一条长长的短信链的截图，你与合作伙伴分享的电子邮件收据，还是你认为看起来很酷的整个网站，你都应该下载 [Tailor](http://www.trytailor.com/) ，这是一个实用的应用程序，它会自动将截图拼接成一张图片。

### **工作原理**

Tailor 的工作原理是根据你事先拍摄的截图将图像拼接在一起。它会在你的相机胶卷中寻找截图，并将相似的图片组合在一起，拼接成一张新的非常高的图片。你可以分享方向、你在文章书签服务中读到的一个故事、一组 Instagram 帖子或任何不适合单个屏幕截图的内容。虽然是免费的，但你可以花 2.99 美元去掉新图片底部的小水印。

首先，打开你想要分享的任何内容——无论是待办事项列表、电子邮件链还是路线——并捕捉你的第一个屏幕截图。您随后的截图应该有一些重叠的元素，这样 Tailor 就可以知道在哪里将两张图像缝合在一起。这意味着将最后一条短信放在第一张截图的底部，第二张截图的顶部，依此类推。一旦你得到了你需要的图片，打开 Tailor，看着它自动把你的截图缝合在一起。

根据您捕获的图像，并不是每一个缝合的最终产品都会被完美编辑。如果您的图像重叠太多，或者您添加的截图之间缺乏足够的区别(比如一个网站的两个截图只是稍微向下滚动)，您会看到图像部分重叠的迹象。虽然重叠有点烦人，但通常图像是完美拼接在一起的。裁缝的实用功能可能不会每天使用，但它确实节省了我分享图像时的时间和混乱，并保持一切美好和有序。
