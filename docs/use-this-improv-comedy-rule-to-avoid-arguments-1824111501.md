# 使用这个即兴喜剧规则来避免争论

> 原文:[https://life hacker . com/use-this-improv-comedy-rule-to-avoid-arguments-1824111501](https://lifehacker.com/use-this-improv-comedy-rule-to-avoid-arguments-1824111501)

在喜剧即兴创作中，“是且”的原则意味着首先你同意你搭档的前提，然后你再补充。没有这一基本原则，场景就无法进行。虽然将“是”和“是”应用到现实生活中有点像商业世界的陈词滥调，但我发现这是一个很好的方法，可以将潜在的争论转变为愉快的玩笑，让每个人都在同一个团队中。

在即兴表演中，如果我们走上舞台，我说“陛下，农民拿着干草叉在城堡门口！”我的意思是“你将扮演一个中世纪的国王，我是你的仆人或顾问，我们与愤怒的农民发生冲突。”这就是为什么你不回答“为什么你说话这么奇怪，弗雷迪？回到警车里去！”有一次真的很有趣，这让我们很难营造一个场景。为了得到更多的笑话，我们必须首先同意基础现实。

争论，这些卑鄙的混蛋，经常在现实生活中出现，因为我们在一些基本的现实问题上意见不一致。当我妻子说“你昨晚忘记把薯片放好了”，我知道她实际上是在说“我们担心老鼠，所以我们不要把食物放在外面，这很重要。”

如果我心情不好，我可能会争辩说这不重要，用“那又怎么样，看，没什么吃了它们！”或者也许用“我把它们漏掉了，因为我忙着报税”来为自己辩护第一个回答拒绝了我妻子的基本现实，坚持认为我们不需要担心老鼠。第二个只是无视她的现实，介绍我自己的。现在争论可以开始了。

但是如果相反，我说“哦，你喜欢我留给老鼠的小点心吗？”那我就接受她的现实。我说“我同意把食物留在外面是有后果的”，并通过取笑我的错误来让自己同意她的观点。

有趣的是，因为我把它当成了一个笑话，所以我没有感觉到我脑海中那个讨厌的小小的辩护声音，坚持认为把薯片放好是她的工作，或者老鼠甚至不吃 T2 的薯片。谁在乎呢。不管我们会不会吵架，一个小时后我可能会意识到她是对的，所以在这期间我们就好好相处吧。(这种玩笑技巧对小事很管用；不要依赖它来进行真正的大讨论。)

我敢肯定你从来没有为这么愚蠢和微不足道的事情挑起过争论。但是下一次当其他人开始争论时，试着同意他们的基本现实。如果你喜欢，可以把它当成笑话。接受你是国王，而农民正在反抗。(“我知道，它们也很难闻！”)当每个人都忙于同意对方的观点时，很难有争吵。
