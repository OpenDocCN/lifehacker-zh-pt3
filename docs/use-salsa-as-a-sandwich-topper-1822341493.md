# 用沙拉作为三明治的配料

> 原文：<https://lifehacker.com/use-salsa-as-a-sandwich-topper-1822341493>

一个好的三明治只需要肉、奶酪和面包，但加入一些番茄、洋葱或其他蔬菜可以通过增加新鲜的味道和质地使它变得真正伟大。你可以把沙拉堆在上面，而不是把每一种成分单独切片或切块，这样可以节省时间。



这个想法实际上是我的一个非常忙碌的朋友(她是一个母亲，一个酒保，一个法律系学生)向我提起的。有一天，她试图做一个三明治出门，她拿了一杯墨西哥咖啡，而不是把西红柿和洋葱分别切片。番茄、洋葱、辣椒和芫荽叶的混合物丁特别适合搭配火鸡和切达干酪三明治、烤牛肉和胡椒杰克(为了额外的热量)，甚至金枪鱼沙拉三明治。不喜欢香菜？几乎任何沙拉都可以使用，但我更喜欢这些口味组合:

*   [浆果莎莎](https://lifehacker.com/how-to-use-up-pretty-much-any-leftover-berries-by-turni-1796995979) 配火腿和布里干酪
*   玉米沙拉配鸡肉沙拉
*   芒果沙拉配烤鸡
*   豆沙黑豆鳄梨沙拉

把它舀上来(放在奶酪和肉之间，防止面包变湿)，大嚼一口，对你设法打包在三明治里的所有产品感觉非常好，而不用打破砧板。
