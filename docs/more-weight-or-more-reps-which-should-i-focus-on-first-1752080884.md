# 增加重量还是增加重复次数:我应该先关注哪一个？

> 原文：<https://lifehacker.com/more-weight-or-more-reps-which-should-i-focus-on-first-1752080884>

不管你举重的动机是什么，你的目标和我或者你旁边自拍的那个人是一样的:进步。你的第一反应可能是增加重量和举起更重的东西，但是增加重复次数是另一个选择。让我们梳理出每个选项的最佳时间。

当你需要考虑增加更多重量或者 [重复次数的时候，](https://lifehacker.com/does-it-matter-how-many-reps-you-do-when-you-work-out-1742905045) 你已经发现你的身体已经适应了健身圈里的人所说的你当前的“训练刺激”——换句话说，你已经取得了进步！不幸的是，如你所知，通往力量的旅程并不是直飞时髦小镇。

## **为什么你的训练进度停滞不前**

在解决“重量优先还是重复优先”的问题之前，我们需要面对举重房里大象大小的宿敌:缺乏进步，或者是无人之地，在那里你的锻炼感觉足够容易，但一段时间后你并没有变得更好，或感觉更强壮。用时髦的健身术语来说，你已经达到了一个平台期。

本质上，当上次让你变得更健康、更强壮的刺激不再足够好时，就会出现平台期。当然，当你增加更多的重量，更多的重复次数，或者两者都增加时，这种情况通常会消失——事实上，如果你以前从未增加过重量，这是非常容易的。但我们的对手是一个值得尊敬的对手，最终，做同样的事情，杰克蹲下…然后你再次平台期。

停滞期的发生有多种原因， [其中很多我们在](https://lifehacker.com/what-to-do-when-you-feel-like-your-fitness-progress-has-1728055487) 之前已经概述过。基本上，你可以责怪你的身体太聪明了。让我们假设持续进步的其他关键驱动因素——[睡眠、一般压力和营养](http://vitals.lifehacker.com/you-snooze-you-win-how-a-lack-of-sleep-can-hold-back-1736338596)——都处于良好的控制之中。这就留下了这些达到平稳状态的相关可能性:

*   **你训练不够**:除了努力和强度之外，你的训练量，或完成的“工作”总量，是另一个可能阻碍你的因素。大多数新手举重项目旨在慢慢建立你的基础，主要是减少练习和总次数。作为一个新手，你缺乏做更多事情的能力[*，这是来自*力量杂志&条件研究**](http://www.ncbi.nlm.nih.gov/pubmed/22158142) *的这项研究表明，帮助你获得更多力量和肌肉，而**能够充分恢复和处理工作量。优步的力量型运动员花了很多年来培养这种高工作能力。正如 Lifehacker 的定期撰稿人和力量教练 Greg Nuckols 在 [的一篇关于撕裂身体](http://rippedbody.jp/intermediate-training-greg-nuckols/) 的文章中所说:“你可以处理更多的工作，所以你可以做更多的工作，所以你可以从更多的工作中收获好处(更多的尺寸和力量)。”***
*   ***你训练得太多了:从另一方面来说，你可能做了太多让你的身体无法恢复的事情。等等……什么？格雷格解释说:“随着你变得越来越强壮，训练的新陈代谢成本就会变得越来越高。这可能会导致疲劳在训练过程中积累，导致你在锻炼中出现问题，并在锻炼后降低你的训练质量。这也直接影响了恢复——在你的状态崩溃的情况下，从多个工作环境中恢复会更加困难，你必须深入挖掘，日复一日地完成最后几个重复。”***
*   *****你的训练计划对你来说太超前了**:这个理由听起来几乎违反直觉，不是吗？一般认为锻炼越高级，进步越大。(“见鬼，如果阿诺德训练成那样，我也应该像他一样！”)但问题是，如果你没有训练很长时间，你的身体还没有熟练到能真正受益的程度。更重要的是，如果你一开始就使用先进的方法，当忍者走出高原的时候，你改变事情的选择会更加有限。***

***你可以看到持续进步并不像说“增加重量”或“增加重复次数”那么简单***

## ***那么，到底是哪个:重量优先还是重复优先？***

***现在来回答这个问题:我应该先改变重量还是重复次数？其实首先是形式。***

**不，那不是逃避。**

**事情是这样的:如果你想确保 [你的肌肉真正得到锻炼](http://www.ncbi.nlm.nih.gov/m/pubmed/26700744/) ，保持你的锻炼富有成效，并保持 [你受伤的风险最小](https://lifehacker.com/how-to-minimize-your-risk-of-injury-when-lifting-heavie-1731253133) ，保持正确的锻炼形式是最重要的。然而，当你使用你不习惯的重物时，这些很快就消失了。根据负载的不同，你的身体会塌陷，让你像银背大猩猩在冰上跳踢踏舞一样优雅地移动。毕竟，学会在较轻重量(或体重)下移动自如是一回事，但在额外重量下移动自如完全是另一回事。**

**所以，如果保持你的状态是你的第一要务，而超出你正常承受能力的体重会让它崩溃，那么你应该专注于改变重复次数，或者重复计划，首先尝试向前推进(最后，一个答案！).**

**所以*先重复，再称重*。同样的指导方针也适用于这种假设的情况，在这种情况下，你的锻炼计划要求 6-8 次深蹲。在这里，你的目标是在进入更高重量的 之前，以良好的状态 [达到代表范围的高端(8 个代表)。相反，如果你可以在中途查看 Twitter 的同时完成 8 次重复，那么，是的，你可以增加更多的重量(称为双重进展)。](https://lifehacker.com/how-to-know-when-to-move-onto-heavier-weights-in-the-gy-1739364248)**

**这一切都与增加你的训练量和做更多的工作来处理更多的工作的想法联系在一起。格雷格补充道:**

> **尽可能增加体重通常是个好主意。但是当你达到平台期时，你通常需要减轻体重，增加组数和/或重复次数。这有点包罗万象。如果你因为之前训练不够努力而停滞不前，那么这样做(增加音量)自然会提供更大的刺激。如果你因为训练要求太高而无法恢复而停滞不前，更高的运动量将提高你的工作能力(从训练中恢复的能力)，当你稍后再次增加重量时，为你取得更大的进步做好准备。这并不总是有趣的。当你增强工作能力时，你可能会感到疲惫不堪，而且由于累积的疲劳，你可能不会获得明显的力量，但当你度过另一段时间时，这是值得的。**

**这可能意味着你必须从你的运动中减去一些重量，以便能够做更多的重复或集合，这没关系，因为这就是我们的想法。**

**这两种解决方案都有助于你的身体准备好应对这些新的压力源，是的，但几个月或几年后会发生什么呢？小心不要一下子改变太多、太快。**

## ****变化小不如变化大****

**我不是一个喜欢打赌的人，但我敢打赌，你希望在晚年能够举重，享受你的力量和灵活性。如果这是真的，追逐那么高的数字是一场危险的游戏。不仅让你容易受伤，而且伤害长期的进步。**

**正如健身教练兼《身体重组笔记 [的作者莱尔·麦克唐纳在这篇针对初学者的优秀文章](http://www.bodyrecomposition.com/muscle-gain/beginning-weight-training-part-4.html/) 中所说:**

> **如果有人一开始就投入高运动量或先进的训练方法，那么当他们真正需要增加训练量时，就会遇到问题。如果交易量已经很高，进一步增加即使不是不可能，也是很困难的。如果过早地使用先进的方法，当它们出现时，就没有什么可以打破停滞状态了。**
> 
> **换句话说，所有训练的一个目标应该是用最少的训练获得最大的适应/成绩。**

**换句话说，你最好慢慢地，一点一点地改变事情，而不是用尽所有的办法，太快太快地用尽回旋的余地。也就是说，如果你真的想改变什么，保守地做或者一次只改变一件事。在本文的上下文中，这意味着你将首先关注于改变重复次数*或*的重量，并且以较小的量。明白吗？**

## ****将所有这些放在一起****

**那么，在已经过时的锻炼中，适当的调整是什么样的呢？**

**Greg 在他的文章 中概述了一些要点，并补充说，本质上你应该首先尝试改变你的销售代表。当你不再取得可衡量的进步时，尝试另一种方法——增加重量。*然后*回到 reps 继续重复循环。请注意，你也不必每次都做相同数量的重复和设置。你可以 [选择遵循一种分阶段](https://lifehacker.com/does-it-matter-how-many-reps-you-do-when-you-work-out-1742905045)[的技巧，或者在同一练习的高低重复周期](https://lifehacker.com/does-it-matter-how-many-reps-you-do-when-you-work-out-1742905045) 之间交替。**

**记住:我们的主要目标是在一段时间内(可以是几周、几个月或几年)取得进步，变得更强壮、更健康、更快。莱尔指出，首先更换代表的另一个好处是，你可以避免不知所措，做可以做到的事情，帮助你坚持回到健身房足够长的时间，让它成为一种习惯。这比什么都重要。**
