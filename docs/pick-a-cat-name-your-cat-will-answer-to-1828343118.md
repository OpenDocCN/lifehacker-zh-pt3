# 选择一个你的猫会回答的猫名

> 原文：<https://lifehacker.com/pick-a-cat-name-your-cat-will-answer-to-1828343118>

你的猫看起来是不是对你视而不见？他们不是被召唤时才来的吗？你的猫可能是个混蛋。或者他们可能只是认不出自己的名字。

Watch

一只猫的名字可能会随着时间的推移而改变。我的第一只猫名叫小猫，从那以后，情况有所改善。对小猫来说不是——她已经死了。但她活得很长，尽管她似乎并不特别依恋人类对她的称呼。现在我用 P.G .伍德豪斯的名字给我的猫命名。温哥华兽医猫大师说，事实证明这也是一个坏主意。

如果你还没有看过兽医 Uri Burstyn 博士在 YouTube 上的视频，你一定会喜欢的。他简单轻松的视频解释了你想知道的关于你的小猫 的一切，在这个视频中，他为如何引起猫的注意做了一个非常有说服力的案例:基本上，像鸟一样唧唧喳喳，像老鼠一样尖叫。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-P3N76oGdj3I&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-P3N76oGdj3I&start=0) 

以他的猫模型 Lancelot 为例，Burstyn 博士解释说，猫的耳朵已经进化到可以听到高音调的噪音。兰斯洛特并没有真正记住他的名字，因为重音并不在最后一个音节结束。而是“兰——CEY！”从非常放松的猫那里得到滑稽的反应。

这可能是因为你本能地知道你的猫会对挤压的“咿呀”声做出反应。这可能是为什么小猫最终变成了小猫，为什么伯特和乔治(我现在的模糊博伊斯)现在是伯蒂和乔治。你不能让你的猫一直注意你，但如果你尖叫得足够大声，你肯定可以吓着它看你的方向。
