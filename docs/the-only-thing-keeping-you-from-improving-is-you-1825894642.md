# 唯一阻止你进步的是你自己

本周精选来自罗马哲学家 [盖乌斯·穆索尼乌斯·鲁弗斯](https://en.wikipedia.org/wiki/Gaius_Musonius_Rufus)T3。 [在他的第九次演讲](https://sites.google.com/site/thestoiclife/the_teachers/musonius-rufus/lectures/09) 中，他谈到即使被驱逐出你的祖国也不能阻止你获取知识:

> “此外，当没有人因为流放而被阻止了解和实践必要的东西时，流放怎么会成为培养自己的东西和获得美德的障碍呢？”

### 这意味着什么

流亡如何让你无法学习，无法成为更好的人？并没有。没有人会因为这样具有挑战性的场景而被阻止去做他们需要做的工作。任何困难的情况都不能也不应该阻止你自我完善和完成任务。

### 从中吸取什么

鲁弗斯一生中三次被流放，但他仍然学习哲学，并想方设法变得更加博学。流离失所或被人看不起从未阻止他做他需要做的事情。事实上，他讨论了认为这样的事情会影响你是多么可笑:

> “因此，正如一个人住在自己的国家，但与他出生的地方不同的房子里，如果他因此哭泣和哀号，他会被认为是愚蠢的和嘲笑的对象，所以任何人认为这是一种不幸，因为他住在另一个城市，而不是他出生的城市，将被正确地认为是愚蠢的和愚蠢的。”

对他来说，在别的地方，或者在不同的情况下，并不能改变什么。你也不应该。

记住，你在哪里，或者发生了什么都不重要。你还是你，有工作要做，有抱负要实现。你可能会被强加新的限制——你自己版本的流放——但是总有办法在这些限制下工作。所以开始吧。流亡不是放弃和无所事事的理由。
