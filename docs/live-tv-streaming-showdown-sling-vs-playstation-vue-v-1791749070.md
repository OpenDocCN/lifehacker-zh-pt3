# 直播电视流媒体摊牌:Sling vs Playstation Vue vs DirecTV Now

> 原文：<https://lifehacker.com/live-tv-streaming-showdown-sling-vs-playstation-vue-v-1791749070>

像 Sling、Playstation Vue 和 DirecTV 这样的服务现在可以让你通过互联网观看直播电视，而不必从你的有线电视公司 那里获得 [一个复杂的捆绑包。我们看了一下这三种服务，看看它们是如何相互叠加的。](http://lifehacker.com/the-worst-ways-cable-companies-confuse-you-into-paying-1773089329) 



### **竞争者**

与网飞或 Hulu 等服务不同，直播电视流媒体服务为你带来有线电视体验，而不会被迫与你的互联网和电话服务捆绑在一起。你只需支付月费，就可以通过网络浏览器、手机、平板电脑或机顶盒观看电视直播。对于流媒体来说，这是观看体育直播的最佳方式之一，你甚至可以选择点播内容。以下是我们今天关注的服务:

*   [**吊带电视**](https://www.sling.com/) **:** 吊带拉开了 2015 年互联网电视直播潮流的序幕。该公司的基本计划是最便宜的，每月 20 美元起，用于选择基本频道。然后，您可以从可选套餐中进行选择，以添加额外的频道，如一组儿童主题或喜剧频道，每月 5 美元。
*   [**Playstation Vue**](https://www.playstation.com/en-us/network/vue/)**:**30 美元， [Playstation Vue 最便宜的方案](https://www.playstation.com/en-us/network/vue/channels/) 比 Sling 贵一点，但你得到一个巨大的优势:DVR。有了 Vue，你可以在你最喜欢的节目和电影播出时录制下来，以后再回来观看。
*   [**DirecTV Now**](https://directvnow.com/)**:**AT&T 拥有的 DirecTV Now 是现场最新的电视服务， [所以有点不登大雅之堂](http://gizmodo.com/directv-now-is-a-just-another-failed-attempt-at-streami-1789614922) 。像 Sling 一样，它没有 DVR，但从好的方面来说，你可以在最便宜的 35 美元/月计划的基础上，以 5 美元的价格获得 HBO 和 Cinemax，所以这很酷。

每种服务都有自己的优势和劣势，但请记住，它们仍然是直播电视节目。每个公司的套餐中的所有频道都有商业广告。如果你正在寻找现有有线电视包的替代品，你不会注意到有什么不同，但如果你习惯了网飞或 HBO 相对无广告的体验，你应该知道你会遇到什么。

我还注意到，由于一些可能远远超出该服务控制范围的因素，这三家公司的流媒体质量都有所下降。有一次，Sling 上的一集《辛普森一家》看起来乱码，而且有帧率问题，所以我在 Playstation Vue 上查看了一下。果然，它有同样的问题。这是有道理的，因为他们都在重播同一个频道。总的来说，每种服务似乎都提供了相当的视频质量，但您的体验可能会因当地电视台、网络广播能力和互联网连接而异。

### **Sling 让你建立你想要的电缆包，其他的使用电缆风格的层**

每个人都想只为他们想要的有线电视频道付费，但这种 [并不总是让观看一切](http://lifehacker.com/a-la-carte-streaming-is-here-and-it-s-just-as-expensiv-1769410396#_ga=1.93038813.1347662518.1465819317) 更便宜。Playstation Vue 和 DirecTV 现在采取了一种捆绑方式，让人想起传统的有线电视套餐。你可以从几个频道的基本套餐开始，你可以支付更高的月费来获得更多频道。

你可以 [在这里](https://www.playstation.com/en-us/network/vue/channels/) 对比 Playstation Vue 的计划，从 30 美元/月到 65 美元/月不等。基本计划包括 ESPN、迪士尼、美国广播公司和福克斯等频道，而高端计划包括 HBO 和 Showtime 等付费频道。同样，你也可以在这里浏览 [DirecTV Now 的套餐](https://directvnow.com/accounts/select-plan) (需要账号)，价格从 35 美元/月到 70 美元/月不等。与 Playstation Vue 不同，HBO 和 Cinemax 是每月 5 美元的额外附加服务。

这两种模式都有一个主要缺点:如果你想要更高层次的频道，你必须支付一大笔升级费用。DirecTV 现在可以方便地让你单独为 HBO 付费——为 [付费比你单独为 HBO Now](http://lifehacker.com/hbo-now-arrives-on-android-and-amazon-devices-1718246694) 付费少 10 美元。然而，说你想要基本包加上 [回旋镖](https://en.wikipedia.org/wiki/Boomerang_(TV_channel)) 因为你真的很喜欢史酷比和鲁尼的曲子。在 Playstation Vue 上，你必须达到每月 45 美元的价格。现在在 DirecTV 上，你只能在最高的*层获得 Boomerang，其价格是基本包的两倍。这不会让你觉得你只是在为你想要的东西付钱。*

另一方面，Sling 采取了不同的方法。你可以在 这里 [探索吊索的计划。在基本计划级别，您可以在两种套餐中选择:吊带橙色(20 美元)或吊带蓝色(25 美元)。每个频道都有不同的选择，它们之间有一些明显的重叠。如果你想要两个频道，你可以花 40 美元买橙色+蓝色。值得注意的是，橙色包只允许您一次传输到一个设备，而蓝色包允许您传输到三个设备。如果你选择组合套餐，你可以一次流到四个不同的地方。](https://www.sling.com/service) 

除了基本计划，Sling 还提供“额外服务”，即每月收取少量费用的相关频道的集合。例如，儿童附加服务包括 Nicktoons、TeenNick 和 Boomerang，每月 5 美元。喜剧演员给你 5 美元/月的 MTV，Spike 和电视土地。你也可以单独添加付费频道，如 HBO(15 美元/月)或 Cinemax(10 美元/月)。

这种模式比 Playstation Vue 或 DirecTV Now 给了 Sling 更多的灵活性。如果你想要一套基本的电视节目，外加几个体育频道和几个儿童频道，你每月只需花 30 美元就能得到。对于其他服务，你必须接受他们提供的任何等级，包括你想要的频道。不能混搭。然而，另一方面，吊索的价格可能会增加。如果你想在不花很多钱的情况下访问世界上的每个频道，DirecTV Now 或 Playstation Vue 的高级套餐可能更适合你的需求。

### 每项服务都专注于传统有线电视的不同方面

一旦你最终筛选完所有可用的计划，你将需要一个应用程序来开始观看。在这方面，Sling 和 Playstation Vue 处于领先地位。你可以在 Android，iOS，Roku，Apple TV，Android TV，Amazon Fire TV 上下载这两种服务的应用程序，并且两者都在其移动应用程序中提供 Chromecast 支持。此外，Sling 有 Xbox 应用程序，而 Playstation Vue 自然也有 Playstation 应用程序。你可以在这里找到 [链接到 Sling 的应用](https://www.sling.com/devices) 和[Playstation Vue 的大部分应用在这里](https://www.playstation.com/en-us/network/vue/) 。而 DirecTV 现在只提供针对 [Android](https://play.google.com/store/apps/details?id=com.att.tv&hl=en) 和 [iOS](https://itunes.apple.com/us/app/directv-now/id1136238277?mt=8) 的移动应用，加上 [Apple TV](https://itunes.apple.com/us/app/directv-now/id1136238277?mt=8) 和 Chromecast 支持。如果你想让它出现在你的游戏机或非苹果机顶盒上，你必须等待。我在 Android 手机应用程序和 NVIDIA SHIELD 上的 [Android 电视上试用了这三种服务，看看实际使用起来是什么感觉。](http://lifehacker.com/4k-set-top-box-showdown-roku-4-vs-shield-tv-vs-fire-1787055398)

Sling 的重点是帮助你找到你想看的节目，但它可能会有点混乱。一个名为“现在”的栏目显示了各个频道正在播放的节目，你可以点击观看你想看的节目。一个标有“指南”的独立部分可以让你浏览频道，从一个频道跳到下一个频道，而不管正在播放什么。然而，如果你想添加几个频道到你的收藏夹列表，并只是浏览这些频道，你必须去名为我的电视部分，手动添加它们。弄清楚这一点有点违反直觉，但一旦我做到了，我的电视栏目就成了最有帮助的。在这里，我可以在一个地方浏览我真正关心的频道和节目，而不用去挖掘大量不相关的内容。

DirecTV 现在更直接地帮助我找到了我要找的东西。该应用程序有节目、电影和网络按钮。如果您轻按“节目”,您可以浏览或搜索电视节目，无论它们现在正在直播还是点播。一个方便的“添加到观察列表”按钮将保存您的选择供以后使用。在网络选项卡下，您可以滚动浏览按字母顺序排列的频道列表。只需轻按一个即可开始播放。观看时，您可以左右滑动来切换频道。在所有的应用程序中，这种感觉最接近老派风格的频道冲浪。令人沮丧的是，我无法将频道添加到收藏夹列表中，也无法一次看到我的观察列表中的所有内容。

Playstation Vue 在移动应用程序上的主屏幕显示了一系列正在播放的节目和频道，但如果你添加了喜欢的频道和节目，这些将出现在顶部。一个单独的频道部分可以让您滚动浏览您订阅的每个频道。轻按一个，您可以将其添加到您的收藏夹或开始观看直播。令人沮丧的是，Android TV 应用程序完全不同。在主屏幕上，你可以看到一个频道部分，但如果你添加了任何喜欢的频道，你会看到这些。我最终发现，我可以使用指南选项卡来浏览每个正在播放的频道，但这种不一致性令人困惑。

在花了一段时间适应每个服务的应用程序后，Sling 对我来说感觉最舒服，但所有这些都需要一点努力才能实现。DirecTV 现在对那些想换频道直到找到他们喜欢的东西的沙发冲浪者来说是最友好的。Playstation Vue 最擅长不加思索地找到可以播放的节目，如果你喜欢的节目可以播放的话，它会很有帮助地把它们推到最前面。Sling 的目标似乎是帮助你找到你关心的频道和节目，并关掉其余的，尽管该应用程序并不总是清楚如何做到这一点。哪种服务让你感觉最舒服将在很大程度上取决于你看电视的方式，所以在你购买之前尝试一下是值得的。幸运的是，现在，这三个服务都提供 7 天免费试用，所以给他们一个机会，看看哪一个最适合你的需求。

### Playstation Vue 是目前唯一一款带 DVR 的游戏，但其他游戏也将很快推出

如果你正在寻找一个能够录制直播节目的电视服务，Playstation Vue 是你现在唯一的选择。当你将一个节目添加到你的观看列表时，它不仅仅是将它标记为书签以备后用。它会录下该节目的每一集。您可以将录制内容保留到原始播出日期之后的 28 天。一些节目和频道不允许 DVR，但幸运的是大多数都允许。

最重要的是，Playstation Vue 会将你的录音与点播服务进行比较。如果你错过了一些最近的剧集，或者如果它们没有 DVR，如果该剧集的点播版本可用，该应用程序将填补空白。你所有可用的剧集都按时间顺序列在节目页面上，所以你可以在一个地方浏览它们。

至于 Sling 和 DirecTV Now，都承诺 DVR 即将到来，但都还不可用。Sling 开始 [提供仅限受邀者的 beta](http://blog.sling.com/announcements/sling-tv-cloud-dvr-beta-program-starts-today/) ，但不能保证你一定能进去。Sling 还提供为期三天的“重播”功能，让你在节目播出后的三天内观看你错过的一集，但并非所有节目都提供该功能。与此同时，DirecTV Now 的 DVR [应该会在今年的某个时候推出](https://help.directvnow.com/hc/en-us/articles/217279826-Can-I-record-DIRECTV-NOW-shows-to-a-DVR-) ，但是还没有具体的发布日期。

### **结论:** **Playstation Vue 是最强大的，但 Sling 提供了定价灵活性**

电视直播订阅是一种奇怪的服务类型，旨在复制通过有线电视包观看电视的旧风格，同时为互联网时代进行更新。有时，感觉这两个目标是相互矛盾的，我们看到的三个竞争者中没有一个有明显的优势。

也就是说，Playstation Vue 因包含 DVR 支持而获得满分。仅凭这一点，它就已经脱颖而出了。如果你想以后能够录制节目并在自己的时间观看，你现在几乎必须使用 Playstation Vue。Sling 和 DirecTV 现在说他们将在今年登上 DVR 列车，但现在索尼会支持你。

另一方面，Sling 最好的资产是它的定价。如果你正在看直播电视节目，因为你想切断电源线，那么 Sling 给了你很大的灵活性，可以只获得你想要的频道，而不必为一堆你不关心的垃圾支付一大笔钱。如果你关心很多频道，它会变得很贵，但对于极简主义者来说，这是一个不需要太多钱的可靠服务。

DirecTV Now 比较难推荐，但可能会归结为你的喜好。比 Playstation Vue 稍微贵一点，而且没有 DVR。然而，很容易打开应用程序，直接跳转到我正在寻找的频道。我也很欣赏像轻触遥控器上的按钮一样切换频道的能力。DirecTV 现在是最新的产品，所以自然有一些成长空间。希望随着它的成长，它会提供更多，但现在它有点太贵了，并且缺少一些比 Playstation Vue 或 Sling 更值得推荐的关键功能。当 Hulu 在今年晚些时候发布自己的流媒体服务时，它也可能面临挑战。
