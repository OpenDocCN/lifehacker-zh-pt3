# 最好的姜糖浆不需要加热

> 原文:[https://life hacker . com/the-best-姜糖水-requires-no-heating-1828363912](https://lifehacker.com/the-best-ginger-syrup-requires-no-heating-1828363912)

鸡尾酒糖浆很容易制作，有些人会说“简单”。将糖倒入水中，加热至溶解，冷却。如果你想要一种除了“甜”以外的味道，你可以在水中煮一些植物的一部分来赋予一些特别的东西。然而，在姜糖浆的情况下，这种方法会给你一种乏味的、不那么辣的糖浆，带有一种“煮熟”的味道。

我用热水浸泡过姜汁吗？是的，它确实很好，但它无法与萨沙·帕特拉斯科的 [*关于鸡尾酒*](https://www.amazon.com/Regarding-Cocktails-Sasha-Petraske/dp/0714872814?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-best-ginger-syrup-requires-no-heating-1828363912&asc_source=&tag=kinjalifehackerlink-20) 中的免煮糖浆相比。这种糖浆结合了新鲜的姜汁和超细糖，比煮过的东西更加明亮、辛辣和细致入微。

要制作它，你只需要一大块姜，一些糖，一个榨汁机，食物处理器，或者某种带锋利刀片的搅拌设备。彻底清洗生姜，根据需要将它分开，以去除挂在生姜小腋窝里的灰尘。一旦它被分解成一两英寸的碎片，把它扔进你选择的咀嚼器里，让刀片把它变成纸浆。

将一些奶酪布塞入量杯中，将姜浆刮入布中。用干净的手挤压并拧干奶酪布，将尽可能多的果汁提取到量杯中。记下体积，将果汁倒回你的(冲洗过的)食品加工机或搅拌机中。将姜汁的体积增加两倍，在姜汁中加入等量的糖，让它发酵或混合，直到糖溶解。

将糖浆倒入一个干净的罐子里，搅拌成鸡尾酒、柠檬水，或者与苏打水和柠檬混合，制成一杯非常棒的姜汁汽水。储存在冰箱里，可以保存两周左右。
