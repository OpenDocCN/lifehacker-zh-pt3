# 解锁 iPhone 最好的方法是什么？

> 原文：<https://lifehacker.com/whats-the-best-way-to-unlock-your-iphone-1828864810>

我记得 2017 年 9 月，当苹果 [第一次宣布 Face ID](https://lifehacker.com/why-you-shouldnt-buy-the-iphone-x-at-launch-according-1819803348) 时的所有喧嚣。有无数的文章和思想批评苹果的新安全技术即将创造的可怕体验。当你把 iPhone 从裤子口袋、夹克或包里拿出来时，不仅仅是*按下 iPhone 的 Home 键(这是一项自然的任务)，你必须把你的设备拉到眼睛的高度，盯着它，然后像平常一样使用它。* 



然而，Face ID 似乎为苹果做得很好。或者，至少该公司目前坚持面部识别，因为预计它将于下周发布的 iPhones 手机都不会有任何类型的显示指纹读取器。无论这是苹果无法及时赶上今年的 iPhones 的技术限制，还是苹果最终将永远放弃触控 ID，我们只能拭目以待。

在此期间，我们来看看 iPhone 的各种解锁机制的优缺点——从密码，到手指，到 [你的大肥头](https://25yearslatersite.com/2017/12/20/mtvs-oddities-part-1-cerebral-or-simply-heady-animation-the-head/) 。

### **选择解锁技术**

首先，我的大部分截图来自运行 iOS 12 的 iPhone 8 Plus，因为~~我太便宜了~~我不需要 iPhone X。假设你没有使用五年前的 iPhone 或操作系统，步骤和设置应该与你的 iPhone 类似。

此外，苹果没有像安卓用户在智能手机上那样，在 iOS 系统中给你提供那么多解锁设备的选项。这是突破口。

首先解锁你的设备，点击设置应用程序。从那里，向下滚动一点，直到您看到触控 ID &密码(或 Face ID &密码)。轻点那个。

### 选项 1:密码

**便利:**通知*和*快速设备访问
T5】安全:😂

很有可能你已经设置了某种数字密码，可以用来登录你的设备——可能在这些时候，当你 时，它无法识别你的面部、指纹或 [:](https://support.apple.com/en-us/HT204060)

*   打开或重新启动您的设备
*   按 Home 键或者向上滑动解锁你的设备(你可以 [换这个](https://support.apple.com/en-us/HT204060#adjust) )
*   更新您的软件
*   擦除您的设备
*   查看或更改密码设置
*   安装 iOS 配置文件

若要更改您正在使用的密码的大小，请在“触控 ID 与密码”屏幕上向下滚动一点，直到您看到“更改密码”点击它，输入你当前的密码，你就可以设置一个新的了。如果你想从六位数变成四位数(反之亦然)，点击“密码选项”链接。

还可以设置**选项 2:** 一个密码。我想给它一个单独的部分，但是设置它太简单了。不要选择不同大小的数字代码，而是选择“自定字母数字代码”或“自定数字代码”选项，然后输入您想要的任何数字和文本字符串(或只是数字)。虽然杀手级密码可能不方便输入，但它比四位或六位的 PIN 安全得多(特别是如果您使用的数字具有某种意义，其他人很容易猜到)。

### 备选方案 3:触控 ID

**便利:**没有比触控 ID
更好的了**安全:**不要让你的朋友制作你的指纹模型

如果你的设备支持它——无论是 iPhone 5S 以来的任何一款带 Home 键的 iPhone，还是 iPad Air 2 以来的任何一款 iPad——触控 ID 都是解锁 iPhone 的一个很好的方式，或许是最好的方式。它又快又安全，你可能已经在你的设备上设置好了。(在你看到新 iPhone 或 iPad 的最初几个屏幕时，苹果会提示你启用它。)如果你跳过了这一部分，你可以通过我之前提到的设置应用的触控 ID 和密码部分来注册你的指纹。

你最多可以注册五个手指，所以请确保你测试了所有你通常握持或抓取手机的方式，以确保你没有在市场上留下任何小猪。完成数字注册后，滚动到触控 ID 和密码屏幕的最底部，确保选择了“抹掉数据”。这样，如果有人试图闯入你的设备，如果他们连续 10 次尝试失败，最终会清除你的 iPhone。这对你来说很烦人——如果他们做到了那一步，因为 iOS 将开始增加每次连续失败的尝试之间的时间——但总的来说安全多了。

### 备选方案 4: Face ID

**便利:**你得经常看看你的 iPhone
**安全:**你的 [邪恶双胞胎](https://www.youtube.com/watch?v=1yBCW7qHc9g) 很兴奋地看到你的新智能手机

如果你有一部支持 Face ID 的 iPhone——目前是 iPhone X，毫无疑问还有苹果本月发布的一些名字奇怪的新型号——那么你的设置屏幕将会有一个“Face ID &密码”部分，而不是“触控 ID &密码”部分，因为你的设备没有 Home 键可言。

在 iOS 12 正式发布之前，你将无法为你的 iPhone 注册多个人脸(假设你没有运行公测版)。这将是该设备的“设置备用外观”选项，这实际上是为了让你注册一个不同版本的自己，而不是一个完全不同的人。这是一个有趣的小技巧，假设苹果不收紧这一功能，只允许一张*类似于*你的脸的“替代外观”。
