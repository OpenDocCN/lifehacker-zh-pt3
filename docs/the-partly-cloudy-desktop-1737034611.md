# 部分模糊的桌面

> 原文:[https://lifehacker.com/the-partly-cloudy-desktop-1737034611](https://lifehacker.com/the-partly-cloudy-desktop-1737034611)

普什卡 的桌面总是令人愉悦，这款也不例外。我们一直在强调几个更简单、小部件更少的桌面，这是一个完美的例子。一张带我们回到夏天的壁纸，只需要几个小部件，就可以开始工作了。

Watch

这是一个 Windows 设置。以下是获得相同外观所需的条件:

*   来自 DeviantArt 的 [壁纸](http://zim2687.deviantart.com/art/Sky-354036672)
*   用于 Windows 的 [雨量计](http://rainmeter.net/) 系统调试和监控工具
*   雨量计的 [简单媒体皮肤](http://lilshizzy.deviantart.com/art/Rainmeter-Simple-Media-1-1-212678796) 用于时间、日期和温度显示
*   左下方系统监视器雨量计的 [极简系统信息皮肤](http://dashie36.deviantart.com/art/Minimalistic-System-Info-v2-333556546)
*   右下方的 [怪物猫可视化器皮肤](http://www.deviantart.com/art/Monstercat-Visualizer-for-Rainmeter-1-2-1-486330771) 为音频可视化器
*   [火箭坞](http://rocketdock.com) 以一张空白皮肤为码头左边
*   将 [乌瑞亚图标打包成](http://www.deviantart.com/art/Ourea-Icons-339972848) 来替换 dock 中的系统图标

这就是这个的全部内容。这是几个移动的部分，但放在一起看起来很棒。如果你和我们一样喜欢它，去 Pushkar 的 Kinja 博客(链接如下)让他知道你有多喜欢它。如果你对他是如何把这些放在一起有疑问，那也是问他的地方。
