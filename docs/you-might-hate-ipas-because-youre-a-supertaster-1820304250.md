# 你可能讨厌 IPAs，因为你是一个超级味觉者

> 原文：<https://lifehacker.com/you-might-hate-ipas-because-youre-a-supertaster-1820304250>

许多人喜欢在漫长的一周后痛饮几杯像印度淡色啤酒这样的啤酒，但超级味觉者却不喜欢。事实上，他们根本无法忍受啤酒的味道。如果这听起来很熟悉，你自己可能也是一个超级高手。



根据美国自然历史博物馆萨克勒比较基因组学研究所昆虫学馆长罗伯特·德萨勒博士的说法，人类分为三大类味觉者:非味觉者约占总人口的 25%；正常的味觉者大约占人口的一半；超级味觉者占了剩下的 25%。这些超级味觉者对所有味道的体验都远比其他人强烈，不同味道的影响对他们来说被放大了， [德萨勒在鹦鹉螺](http://nautil.us/issue/54/the-unspoken/how-to-tell-if-youre-a-supertaster) 上解释道。例如，如果某样东西是甜的，那么对于超级味觉者来说，它将是非常甜的。

这意味着像啤酒这样的苦味饮料，尤其是 IPAs，对于超级味觉人群来说是非常苦的。啤酒花的强烈味道和酒精的味道让人难以忍受。当谈到其他烈性饮料，如白酒时，超级味觉者也无法承受这种刺激。如果你是极少数超级味觉者中的一员(不到总人口的 1%)，即使像淡啤酒这样的啤酒也常常太苦而无法“品尝”因此，如果你不能忍受 IPAs，但能设法啜饮一些较淡的啤酒，你可能是一个超级品尝者。如果你喜欢它们，加入普通品尝者俱乐部。我们有夹克。你可以通过下面的链接了解更多关于超级味觉者的信息。
