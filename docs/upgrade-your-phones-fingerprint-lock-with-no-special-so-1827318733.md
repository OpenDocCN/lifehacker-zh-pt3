# 无需特殊软件即可升级您手机的指纹锁

> 原文：<https://lifehacker.com/upgrade-your-phones-fingerprint-lock-with-no-special-so-1827318733>

普通人只用 10%的手指力量来打开手机。这并不意味着什么，但是你*应该*教你的手机不仅仅是一个指纹。尝试这三个升级来简化您手机的指纹识别。

Watch

### 添加更多指纹

您可以将多个手指添加到 iOS。我使用食指和拇指，所以我可以用任何一只手或在平面上轻松解锁我的手机。(我让其他手指不被识别，这样我就可以用中指按 home 键，不用解锁就能看到我的锁屏。)

### 添加他人的指纹

如果你和你的另一半秘密不多，但有很多使用对方手机的理由，那么为了方便，就加上对方的指纹。将您的指纹添加到您孩子的手机，甚至是您父母的手机。这比记住别人的密码容易多了。

### 添加两次相同的指纹

如果触控 ID 拒绝你的手指太频繁，CNET 建议 [添加多份你的指纹](https://www.cnet.com/how-to/pro-tip-make-your-android-or-iphones-fingerprint-reader-work-every-time/) 。当你的手潮湿时，试着添加一个，这样你的手机在淋浴后更容易打开。

有一天，我们都将被迫转向面部识别，但在那之前，在争夺数字优势的战斗中，使用你所有的手指。
