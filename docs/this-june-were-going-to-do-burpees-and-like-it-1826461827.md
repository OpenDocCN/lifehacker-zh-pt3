# 今年六月，我们要做 Burpees，我们会喜欢它的

> 原文：<https://lifehacker.com/this-june-were-going-to-do-burpees-and-like-it-1826461827>

当然，好吧，波比很难对付，我们讨厌他们。但是它们是一种惊人的全身锻炼，试一试也不会死。也许两个。(或者十个。你知道这是怎么回事。)到月底我们都要变得这么强了！



我受到了丹妮尔·齐克勒的启发，她 [连续一个月每天做三组十个嗝](https://www.menshealth.com/fitness/a19543814/30-burpees-for-15-days/?utm_source=t.co&utm_medium=Social&utm_term=1553805883&utm_campaign=Men%27s%20Health) 。她的结果是:

> 我开始跑得更快，但没有意识到加快速度。我的肺部感觉清晰，我滑行了数英里。...以这种极端的体育锻炼开始我的一天[burpees，即不跑步]帮助我在早上感觉更清醒，而不必依赖咖啡因。...我觉得自己很强大。

我当然希望在六月底之前感觉更有力量，也希望看到我的跑步有所提高。随着温度的上升，我想我们可以在空调方面做的挑战正是门票。

以下是如何做 burpee:

*   把你的肚子放在地上。

*   像试着触摸天花板一样跳起来。

在练习中，看起来就像你在摸你的脚趾，把你的脚跳回到一块木板上，做一个俯卧撑，然后在跳起来之前把你的脚跳回到一个下蹲的位置。还有其他的打嗝方式，包括 [最初的“四数打嗝”](https://www.youtube.com/watch?v=idRaKSD5iHI) 不包括跳跃或俯卧撑。这里没有伯皮警察，所以选你最喜欢的。

(这里有个小技巧——第一天给自己拍个视频，然后月底可以做个前后对比。请继续关注，我将在我们的签到线程中分享我的进展。你也可以！)

记住，在日常挑战中，调整自己的节奏尤为重要。不要打太多的嗝，否则第二天你会感到酸痛；从感觉容易的量开始，一旦你确定你能应付，每天调整。

所以这是我们要做的。每天早上，至少打一个嗝。瞄准三组，中间稍作休息——也许每组只有一个 burpee(总共三组)。或者也许你会像丹妮尔一样跑 30 英里。你们中的一些野兽可能会瞄准 100。然后洗个澡，穿好衣服，开始你的一天，感觉有点不稳定，但也可能有点强大。
