# 我们的技术编辑用来管理混乱浏览器的 Chrome 扩展

> 原文:[https://life hacker . com/the-chrome-extensions-our-tech-editor-uses-to-manage-a-1823999980](https://lifehacker.com/the-chrome-extensions-our-tech-editor-uses-to-manage-a-1823999980)

我最不擅长整理我的浏览器——绝对最差。我会打开标签页，比如在待办事项列表上添加条目，然后在几周内忘记它们，因为更多的同龄人加入了 Chrome 地址栏上方顶层的派对。

有许多扩展可以用来管理任何组织...困难...你在铬合金。以下是我用来保持理智的一些方法。

### [伟大的吊杆](https://chrome.google.com/webstore/detail/the-great-suspender/klbibkeccnjlkjkiokjodocebajanakg?hl=en)

我们以前写过关于 [这个神奇的](https://lifehacker.com/the-great-suspender-frees-up-memory-by-suspending-brows-1702778962#_ga=2.234373527.862621771.1521471770-396842925.1520800403) (实际上是 [两次](https://lifehacker.com/the-great-suspender-suspends-memory-hungry-chrome-tabs-5982490) )，但是 [这个伟大的外挂](https://chrome.google.com/webstore/detail/the-great-suspender/klbibkeccnjlkjkiokjodocebajanakg?hl=en) 是我安装过的最好的浏览器扩展之一。尽管我有一台相当强大的游戏台式机——*16GB 内存！变得疯狂！*——我发现在我的浏览器后台打开 20 多个标签有时会让浏览器变慢，或者以某种灾难性的方式搞乱 Chrome。

有了这个伟大的吊杆，你可以设定一个时间限制——比如说，15 分钟。如果你在这段时间内没有接触过某个特定的浏览器标签，这个伟大的悬挂器就像它的名字所暗示的那样:它将标签置于某种“待机模式”,确保它们不会占用你浏览器的额外内存。更好的是，扩展将这些标签淡出一点点，这样你就不会在浩瀚的开放网站中丢失你实际使用的几个标签。

### [OneTab](https://chrome.google.com/webstore/detail/onetab/chphlpgkkbolifaimnlloiipkdnihall?hl=en)

我喜欢这个概念。我已经安装它没有问题，它的工作绝对完美。我唯一犹豫的是，是否要比现在更多地使用它，这是有节制的，因为它的前提是——将所有打开的标签浓缩到一个带有简单链接列表的标签中——让我更容易忘记它们。但那是我的事，不是一个标签的事。如果你让它为你工作，这个扩展提供了一个完美的方法来更好地控制你混乱的浏览习惯。

### [会话好友](https://chrome.google.com/webstore/detail/session-buddy/edacconmaakjimmfgnblocblbcdcpbko?hl=en)

有时候，Chrome 会崩溃。有时，它会以一种奇怪的方式崩溃，以至于其典型的“你想恢复我弄乱之前打开的标签页吗”根本不起作用。如果你有一两个正在使用的标签，这没什么大不了的；如果你有 20 个或更多，你将不得不搜寻你的历史来找出它们都是什么。

[会话哥们](https://chrome.google.com/webstore/detail/session-buddy/edacconmaakjimmfgnblocblbcdcpbko?hl=en) 救了我的培根无数次。恢复旧的浏览器会话——就像在 Chrome 变得疯狂之前工作正常的那个——只需点击几下鼠标。该扩展甚至列出了所有在其保存的会话中的网站，以帮助您验证您正在恢复正确的标签批。

### [来自谷歌地球的地球视图](https://chrome.google.com/webstore/detail/earth-view-from-google-ea/bhloflhklmhfpedakmangadcdofhnnoh)

当你在一个浏览器窗口中打开一天中的第 35 个标签时，有时有一些漂亮的东西看是很好的。[Google Earth 的地球视图](https://chrome.google.com/webstore/detail/earth-view-from-google-ea/bhloflhklmhfpedakmangadcdofhnnoh)——最简单功能的最长扩展名称——每当你打开一个新标签页时，都会显示一张新的 Google Earth 图片。就当是 Chrome 的 [禅](https://lifehacker.com/the-power-of-now-a-meditative-approach-to-living-in-th-1691668213) 时刻吧。
