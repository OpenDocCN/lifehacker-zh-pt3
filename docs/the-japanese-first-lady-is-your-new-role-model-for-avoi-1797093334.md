# 日本第一夫人是你避免尴尬对话的新榜样

> 原文：<https://lifehacker.com/the-japanese-first-lady-is-your-new-role-model-for-avoi-1797093334>

在你人生的某个时刻，你会有一次尴尬的谈话。聚会、社交活动、 [与总统的晚餐](http://theslot.jezebel.com/the-most-batshit-excerpts-from-that-trump-nyt-interview-1797086287) 你假装不会说英语的地方，它们都是雷区，潜在地散落着 [尴尬的停顿](http://lifehacker.com/how-to-avoid-the-most-common-awkward-conversation-mista-1257095824) ，令人遗憾的笑话，以及简单的 [侵入性的闲聊](http://lifehacker.com/the-small-talk-questions-you-should-never-ask-1795578480) 。你不必忍受糟糕谈话的打击。一些行之有效的方法已经存在，不管是打扰朋友还是下载应用程序:

### 请原谅

聚会上的谈话可以向任何方向发展，但是如果你和一个新认识的人聊天，或者和一个不是特别有趣的人聊天，一分钟可能会感觉像是独白。当你觉得谈话已经持续了足够长的时间，优雅的退出应该是第一选择。

你可以从一个简单的“”开始，或者放一些芥末在上面，再加上一个握手和一个微笑。自信地离开，知道你是有礼貌和直接的，然后敲击酒杯(或门)。

### 把你的手机变成朋友

屡试不爽的摆脱困境的方法很经典:紧急中断。一个朋友的短信或电话，发现哦，不，詹妮弗把自己锁在门外了，而我有备用钥匙，以及道歉的再见，这些都是你快速逃离所需要的。

当然，如果你有一秒钟的空闲时间，你可以使用一个应用程序来摆脱这种尴尬的局面。在 App Store 中搜索像切尔西·汉德勒的 Gotta Go 这样的应用程序！该应用程序允许您设置预先填充的借口，如来自联系人列表的电话和短信。

### 标记团队您的谈话

当有两个以上的人参与时，进入和离开讨论会更容易。和朋友或熟人一起参加活动？通过邀请另一个人，将与陌生人的随机一对一对话变成三人会谈。

如果你在社交活动 的 [，把你现在的对话伙伴介绍给另一个你可能认识的人。相反，你可以请他们把你介绍给他们认识的人。从那里，你可以很容易地原谅自己，溜走，而两人交谈。](http://lifehacker.com/focus-on-the-quality-of-your-relationships-while-networ-1776730927)

### 继续前进

运动中的物体保持运动状态。这个理论适用于物理和社会情况。穿过城市，为每一个游说者和街头艺人停下来，这不会帮助你到达你要去的地方，而且可能只会让你感到沮丧。

如果你觉得你需要一个有效的理由来避免为每个拿着剪贴板的人停下来，只要指向你的耳机，你的手表，阴云密布的天空，或者你方便放在皮带上的狗，然后上路。一句快速的“不，谢谢”是最完美的结尾。

### 不要误导他们

再也不想和某人交往是再正常不过的了，但是如果你用关于 [交换电话号码](http://lifehacker.com/improve-your-networking-skills-with-the-24-7-30-method-1756051452) 和拿瓶啤酒让你的逃避不那么有压力的陈词滥调来结束你的谈话，你可能会给人留下错误的印象。不要做出你不想遵守的承诺。这不仅会向你的谈话对象传达一些不真诚的信息，而且当你最终忽略他们的信息时，会让你 [看起来像个怪人](http://lifehacker.com/how-to-confront-your-flaky-friend-1796642285) 。

### 假装直到你成功

看啊。没有人知道你是谁，也不知道你为什么会出现在国际掰手腕研讨会的会后派对上，所以如果你发现自己被卷入了关于顶级翻滚训练的讨论，你宁愿没有，只要假装一下恐惧就好了。“我唯一知道如何用英语说的是‘我不会说英语’”是我的常用语，但在酒吧或聚会等拥挤的场合，假装无知或某种形式的听力损失对我来说一直很有效。另外，当他们意识到你在开玩笑的时候，你早就走了。
