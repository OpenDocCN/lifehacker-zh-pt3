# 像专业人士一样使用这种专家技巧

> 原文：<https://lifehacker.com/nose-whisky-like-a-pro-using-this-whisky-experts-trick-1819031742>

当你品尝威士忌时，嗅，把你的威士忌拿到鼻子前嗅一嗅的动作，是很多人跳过的一步，但他们不应该这么做。闻你的酒可以帮助你挑选味道和香味，这是你单独啜饮时无法察觉的。这是整个过程的重要部分。

最近，我和麦卡伦品牌教育和威望威士忌的经理查理·怀特菲尔德聊起了烈酒，以及人们应该如何品尝他们的威士忌。在我们的谈话中，他提供了一个非常独特的技巧，让我们学会用手更好地嗅出气味*。*

*“嗅可能是你最能发现威士忌不同香味和口味的地方，”怀特菲尔德说。他的技巧包括将手指放入玻璃杯中，滴几滴威士忌在手指上，然后用力在手背上摩擦手指。*

*“基本上你所做的就是擦掉酒精，”他说。“如果你闻一下手背，你会闻到橡木味、甜味和水果味，这取决于是哪种威士忌。这基本上是在解构威士忌——你擦掉了酒精，把所有的味道化合物留在手上。”* 

*他说，一旦你这样做了，你可能就能闻到威士忌中的味道，当你在玻璃杯中闻时，这些味道可能被酒精掩盖了。当你再次啜饮威士忌时，你脑海中就会有那些味道，你也能更容易地分辨出来。*

*例如，如果你在闻麦卡伦双桶 12，你可能会闻到奶油糖果、太妃糖苹果、橡木、香草奶油冻或蜜饯橙的味道。* 

*怀特菲尔德建议，当你用鼻子嗅玻璃杯时，要慢慢地嗅，而不要把脸凑过去。*

*“我总是建议慢慢地把杯子拿到你的鼻子前，直到你开始注意到那些香味和味道，”他说。“我总是建议不要像喝酒一样把鼻子直直地伸到杯底，因为酒精浓度很高。当你这样做的时候，你可能会烧掉所有的鼻毛——你可能会在一天的剩余时间里闻不到任何东西，因为你只会感觉酒精直接烧进了你的鼻子。”*
