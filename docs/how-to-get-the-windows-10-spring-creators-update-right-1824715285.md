### Spring Creators 的更新到底是什么？

在你千辛万苦地赶在传言中的下周 发布之前获得更新之前，这里有一个你会发现的一些新功能的快速回顾:

*   **任务视图的新历史**:点击 Windows 键+标签，你会看到你前几天执行的活动的历史档案，而不仅仅是你当前打开的所有应用程序的列表。你可以使用右上角的一个小搜索图标来搜索特定的活动，还可以在多个设备之间同步你的活动列表，这样当你从电脑移动到 Windows 10 平板电脑时，你就可以继续你正在做的事情。不过，不要指望在这个列表中看到你在 Windows 中所做的一切:例如，你最近访问的网站只会在你使用 Microsoft Edge 时出现(你可能没有使用)。
*   **更新设置:**你现在可以调整声音选项，包括你希望特定应用程序运行的默认音量，通过 Windows 10 的设置面板(特别是在系统选项内)。你也可以看看哪些应用程序在你的系统启动时启动；仍然可以在任务管理器中调整，但是现在也可以在设置>应用>启动中找到。一个新的字体管理系统也在设置应用中首次亮相，你可以在设置>个性化>字体中使用它。谁不爱字体？
*   **控制带宽更新使用:**如果你担心微软的 Windows 更新会消耗掉你的 ISP 允许你使用的每月数据配额(唉)，你现在可以调整 Windows 和应用程序更新的下载和上传设置。拉起设置>更新&安全>高级选项>交付优化>高级选项对你的 PC 拉下(或发送给其他人)的内容设置一些限制。
*   **附近共享:**这是微软对苹果隔空投送的回应，这项功能——只要你在 Windows 中点击一个共享按钮就可以找到——允许你通过蓝牙或 Wi-Fi 连接将内容放到附近的其他 Windows 10 电脑上。

*   **更多 Cortana 技能:**Cortana 可以帮助你完成的各种任务已经进行了一些重组，尤其是它的收藏和列表功能，你可以访问更多的服务(尤其是智能家居服务)和技能。换句话说，Windows 10 Spring Creators 更新可能会让你的台式机或笔记本电脑变得更加健谈。
*   **对焦辅助:**厌倦了在你试图工作时被朋友(或 Windows)打扰？你现在可以设置默认情况下你想收到哪种提醒和通知，甚至可以设置你想(或不想)被打扰的特殊状态:当你工作时，当你玩游戏时，等等。
*   **让应用看起来不那么模糊:**如果你在 Windows 中使用缩放，因为你使用的是高分辨率和小屏幕尺寸的显示器，Windows 10 现在将尝试在应用难以缩放时让应用不那么模糊。在设置>系统>显示>高级缩放下找到这个选项。
*   **向家庭组挥手告别:**没了。抱歉。T3】试试 one driveT5】？

### 获取 Windows 10 Spring Creators 更新:简单的方法

正如 [Neowin](https://www.neowin.net/news/heres-how-to-get-the-windows-10-spring-update-rtm-before-everyone-else) 所说，如果你选择加入 Windows Insiders 程序并选择发布预览环，Windows 10 Spring Creators 更新很容易下载和安装。(这也将确保您可以更早地获得更新，当更新发送给您时，应该已经相当稳定了。)

成为 Windows 内部人员很容易:

*   打开 Windows 10 的设置(例如，通过点击开始菜单并点击左下角的齿轮图标)。
*   单击更新和安全性。
*   单击 Windows Insider 程序。
*   点击“开始”按钮。
*   出现提示时，表明您只想接收“修复程序、应用程序和驱动程序”

完成后，值得返回到更新和安全屏幕，反复检查您是否正在获取补丁、应用程序和驱动程序——如果您想最大限度地减少早期更新搞乱系统的可能性，请将速度设置为慢。他们不应该，但它永远不会伤害到安全。幸运的话，这将允许你在其他人之前收到 Windows 10 Spring Creators 更新。如果你感到不耐烦，或者如果你想更早地测试微软的更新，你也可以切换到不同类型的内容，如“Windows 的积极发展”，或快速更新的步伐。

### **获得 Windows 10 Spring Creators 更新:更棘手的方式**

如果你只是想要 Windows 10 Spring Creators 更新，而不想成为 Windows 内部人员——如果你不想测试微软的东西，这没有什么坏处。GHacks 已经 [了一篇关于如何从微软的服务器上获取官方更新的](https://www.ghacks.net/2018/04/03/get-the-windows-10-spring-creators-update-early/) 的详尽文章。您将不得不通过一些技术障碍来为您的系统做好准备，但是您的耐心将在您的(非业内人士)朋友得到它之前得到一个完全可用的更新副本。

以下是您必须做的事情的简要概述:

*   从微软抓取 [32 位](http://fg.ds.b1.download.windowsupdate.com/c/Upgr/2018/03/17133.1.180323-1312.rs4_release_clientconsumer_ret_x86fre_en-us_b1cac23739adf65a6fd0c638e730f2a945308188.esd) 或 [64 位](http://fg.ds.b1.download.windowsupdate.com/c/Upgr/2018/03/17133.1.180323-1312.rs4_release_clientconsumer_ret_x64fre_en-us_8483d330ba2e42d8e0a2bdd377074358afb0c864.esd) ESD 文件(针对英语/美国；其他语言可用 [此处](https://pastebin.com/raw/mepnsM6D) )
*   使用 [Wimlib](http://abbodi1406.square7.ch/ESD/esd-decrypter-wimlib-37.7z) 解密 ESD 文件，您必须以管理员权限运行。
*   使用 Wimlib 将 ESD 文件转换为 ISO 文件(包括所有 Windows 版本)
*   将 ISO 刻录到 DVD 或 USB 驱动器(感谢微软的 [Windows USB/DVD 下载工具](https://www.microsoft.com/en-us/download/details.aspx?id=56485) )
*   使用您的 DVD/USB 驱动器安装 Windows 10 Spring Creators 更新

### 你没有*有*来安装大的更新

当然，等待一段时间来安装 Windows 10 Spring Creators 更新可能是值得的。正如 GHacks 的马丁·布林克曼所说:

> “我至少要等一个月才会考虑将生产机器升级到新的 Windows 10 版本。过去的版本表明，在升级过程中，事情可能会变得非常糟糕，微软正忙于解决新 Windows 版本发布后出现的问题。”

只需暂停所有更新七天，您就可以在一周内保持系统无更新，这可以在 Windows Update >高级选项中找到。对于微软来说，这可能没有足够的时间来消除 Spring Creators 更新中的所有错误，但这是有意义的。
