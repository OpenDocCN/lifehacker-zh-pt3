# 通过这个互动教程学习音乐理论的基础

> 原文：<https://lifehacker.com/learn-the-basics-of-music-theory-with-this-interactive-1796329461>

作为一个音乐迷，我想学习如何创作一首歌。然而，花时间学习音乐理论并不是我优先考虑的事情。音乐课很吓人，有点不方便，而且肯定很贵。

Watch

因此，这个来自 [阿尔伯特](https://www.ableton.com/) 的 [互动节拍制作教程](https://learningmusic.ableton.com/index.html) 非常适合我们这些崭露头角的音乐家。它有一大堆工具可以让高级概念变得非常简单，尤其是你可以自己使用它们。

课程分为章节和后续课程，从节拍制作的基础开始。这些课程用几个句子分解了你需要知道的东西，提供了一个概念的例子。每一课，从 [和弦](https://learningmusic.ableton.com/chords/chords.html) 到歌曲结构，都有一个互动的部分，所以你可以用你刚刚学到的东西进行实验。例如,“音阶”部分为您展示了一个完整的钢琴音阶，您可以随意弹奏，同时还有一个循环的钢琴轨道，您可以通过它来学习音阶。它甚至还收录了皇后乐队和碧昂斯乐队的音乐。

交互式音乐制作工具易于理解，并且一次呈现一个概念，让您了解在您的过程中每个工具是如何相互作用的。当你完成六个介绍性章节的时候，你会明白如何使用操场部分，在这个页面上你可以使用所有的工具来制作一些基本的曲子。您可以清除作品进行实验或 [导出作品](https://drive.google.com/file/d/0B63XpF0Di48Wa3QyclBta2VnWkk/view?usp=sharing) 在 Ableton Live 中进行处理。

如果你想了解更多，你可以查看 [高级部分](https://learningmusic.ableton.com/advanced-topics/advanced-topics.html) ，它提供了更多的歌曲创作工具来解释像转位和全音阶三和弦这样的概念。一个小时后，我打了一个我认为很酷的拍子，然后咔嚓一声就输出了。
