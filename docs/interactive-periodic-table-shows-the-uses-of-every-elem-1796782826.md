铼最近为你做了什么？在友好的 [元素交互周期表中查找，用图片和文字](http://elements.wlonk.com/ElementsTable.htm) 。单击任意元素查看其用法列表。你会注意到有趣的模式，比如照明中使用了多少惰性气体。
