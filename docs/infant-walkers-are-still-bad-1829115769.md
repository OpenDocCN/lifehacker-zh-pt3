# 婴儿学步车还是不好

> 原文：<https://lifehacker.com/infant-walkers-are-still-bad-1829115769>

婴儿学步车，那些给还不会走路的婴儿突然学会走路的轮子装置，是不好的。这已经被的[龙确立为](https://www.cpsc.gov/~/link/136cf994237446f5998d36f7b0c1f349.pdf)的。但是父母继续使用它们，所以儿科医生再次呼吁禁止使用。



你会问，看似无处不在的婴儿用品有什么错？难道我们不应该帮助我们可爱的一动不动的小家伙在世界上穿梭吗？ [BBC 爸爸](https://www.youtube.com/watch?time_continue=34&v=Mh4f9AYRCZY) 把他的孩子放在婴儿学步车里。你的父母可能也把你塞进去了。(我在这辆 80 年代的 [经典黄色款](https://shop.kidizen.com/items/848212) 中看到过宝贝我的老照片。)婴儿穿着它们似乎很开心！妈妈和爸爸很欣赏他们如何约束和分散他们的小摇摆器，给他们几分钟额外的时间刷牙或把一些面食扔进锅里。

至少有几个主要问题。首先，婴儿学步车会导致严重的急性伤害。根据周一发表在*儿科*上的一项 [研究](https://doi.org/10.1542/peds.2017-4332) ，并由 [NPR](https://www.npr.org/sections/health-shots/2018/09/17/646413330/as-injuries-continue-doctors-renew-call-for-ban-on-infant-walkers?utm_source=facebook.com&utm_medium=social&utm_campaign=npr&utm_term=nprnews&utm_content=20180917) 报道，“从 1990 年到 2014 年，超过 23 万名 15 个月以下的儿童因颅骨骨折、脑震荡、骨折和其他与婴儿学步车有关的伤害在美国医院急诊科接受治疗。”一项调查发现，从 2004 年到 2008 年，8 名婴儿死于婴儿学步车造成的伤害。虽然这些年来，公司已经让学步车变得更安全，增加了车轮锁等功能，但医生仍然表示，它们给父母一种错误的安全感——当兼顾多个孩子和其他责任时，他们可能不会注意到孩子蹒跚着走进不安全的区域，比如有热炉子的厨房、游泳池和楼梯。

还有一个事实是，婴儿学步车实际上并没有帮助婴儿学习如何走路——根据一些儿科治疗师的说法，他们甚至可以推迟这个里程碑。你看，婴儿的特点是他们做事情的时候会做。他们的身体需要自由探索的空间，以进行适当的粗大运动发育。将他们置于不自然的姿势，特别是长时间放置，会导致姿势不佳，阻碍他们发展平衡控制感，并使他们的体重分布不均匀。这种担忧不仅适用于步行者，也适用于健身器、健身椅和跳高运动员。

有什么合适的选择吗？ [学习表格](https://www.amazon.com/Fisher-Price-Laugh-Learn-Around-Learning/dp/B01ASVCTIG?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/infant-walkers-are-still-bad-1829115769&asc_source=&tag=kinjalifehackerlink-20) 。或者简单地在地板上放一些吸引人的非电子玩具，制造一些安全屏障，让你的婴儿试着自己支撑起来，如果他们觉得准备好了。他们会在自己的时间里学习如何走路。如果你有顾虑，去找你的儿科医生，而不是一件装备。
