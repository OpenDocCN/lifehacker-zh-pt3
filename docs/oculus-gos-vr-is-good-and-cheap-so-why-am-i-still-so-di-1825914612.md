# Oculus Go 的 VR 又好又便宜，为什么我还是很失望？

> 原文：<https://gizmodo.com/oculus-gos-vr-is-good-and-cheap-so-why-am-i-still-so-di-1825914612>

我正站在起居室的中央，但我看到的是一艘船的甲板。我可以听到我真正的狗在我推靠在墙上的沙发上拉屎，这样我就可以转动我的头来探索我的虚拟环境。他被绑在脸上的 [Oculus Go](https://www.oculus.com/go/) 遮住了。我在客厅里向前迈了一步，叹了口气，甲板上什么也没发生。我曾希望该公司会利用其第一款便携式耳机从基于智能手机的虚拟现实竞争中脱颖而出——这种竞争提供了一种类似的体验，即根据你的高端智能手机设计廉价耳机。问题是，Oculus Go 并没有真正坚持着陆。



目前，廉价的虚拟现实要么需要一块纸板，你可以在其中不稳定地设置智能手机，以实现一些基本的“沉浸式”虚拟现实，要么需要像三星 Gear VR 系列或谷歌 Daydream(价格从 65 美元到 100 美元不等的耳机)这样的耳机，这些耳机增加了一层急需的控制——和更舒适的头盔——但仅此而已。好的虚拟现实通常需要像谷歌 Pixel 2 或三星 Galaxy 最后几款旗舰设备这样的高端 Android 智能手机，但如果你通过无线运营商获得了一台，你可以以比 Oculus Go 更便宜的价格获得相应的耳机。

尽管如此，兼容虚拟现实的 Android 智能手机价格昂贵，而且不是每个人都有 700 美元买一部，外加 100 美元买一个新的虚拟现实耳机，用于娱乐目的。

售价 200 美元的 Oculus Go 是一笔不错的交易，它消除了对智能手机和耳机底座的需求。乍一看，它与 Oculus Rift 具有相同的视觉 DNA，并且比其最接近的竞争对手联想(Lenovo)400 美元的 Mirage Solo 更加舒适，这是消费者唯一的另一款无线耳机。两者都有类似实验室护目镜或潜望镜取景器的圆形矩形形状，但与 Mirage Solo 更具科技美感的设计相比，Go 有更精致的织物和塑料设计。尽管如此，Oculus Go 在外观上并不令人惊叹，当你考虑到它是该公司的第三款 VR 耳机时，这是一个令人失望的产品。

HMD 是智能手机的灵魂，具体来说，是最新一代的骁龙 821 处理器。老化的硅片偶尔会出现短暂的用户界面冻结，或者在渲染多个复杂对象时出现，但除非你积极寻找轻微的变慢，否则它的性能符合你对高端智能手机的期望。延迟很少成为问题。它的显示屏是一个明亮的 5.5 英寸 2560 x 1440 LCD，通过耳机的两个镜头可以看到。它还拥有 32GB 的不可扩展内部存储空间(64GB 版本售价 250 美元)。它通过微型 USB (lame)充电，可以持续大约两个小时，然后需要更多的电量。这些规格看起来不错——即使考虑到它有多便宜，也很棒。

在通过虚拟空间的运动方面，Go 通过坚持更具限制性的三个自由度来保持价格低廉:将你的头从一侧倾斜到另一侧，向前和向后，以及从左到右。更贵的联想 Mirage Solo 使用前置摄像头提供六个自由度，这意味着你可以在真实和虚拟空间中上下、前后、左右移动(在你走进什么东西之前会收到警告)。

还有一个内置麦克风和一对集成扬声器，可以对你在 VR 中的运动做出反应，但如果你调高音量，后者会泄漏足够的音频，打扰你旁边的人。(我敢肯定，虚拟现实中在他们旁边乱动的怪人一点也不令人讨厌。)你最好通过 Go 的 3.5 毫米插孔使用耳机。

因为你把耳机绑在脸上，你可以预期它会比你的眼镜重一点。也就是说，Oculus Go 是一种舒适的体验，重量刚刚超过一磅。如果你决定参加虚拟现实马拉松，花额外的时间调整它的三个头带，这可以减轻你鼻梁上的压力。说到眼镜，Go 带有一个橡胶衬垫，为眼镜提供额外的填充物，但你也可以为你的 GoT3】购买 [处方 VR 镜片。](https://www.framesdirect.com/virtuclear-lens-inserts-for-oculus-go.html)

Go 内置的控制器非常舒适，让我希望他们最终会像 Oculus Rift 一样，增加对两个控制器的支持。授予用户双手控制权肯定会拓宽应用程序内部可能的交互范围。这也有助于将自己与单打独斗的竞争区分开来。但是，嘿，在那个 200 美元的甜蜜点，我只要一个就够了。况且没有围棋游戏支持双手操控，反正围棋的 VR 空间也没多少事可做。

Oculus Go 及其应用生态系统的问题在于:它并没有那么好——至少目前是这样。Oculus 的应用商店中正在推广的许多热门游戏只是 Gear VR 端口，往往缺乏对附带的 Go 控制器的支持。许多游戏仍然是为使用无控制器的 Gear VR 而设计的，迫使你用你的头来指指点点，并用你的中性控制器来点击。这很糟糕，Oculus 向我展示残疾人应用程序让人感觉不真诚。

这也凸显了行动自由的局限性。我玩过的大多数游戏要么是射击僵尸和闪亮的目标，要么是拉动操纵杆修理宇航技工机器人，要么是戳一个神奇的生物，要么是在你努力控制自己的午餐时乘坐过山车。图形保真度与我在 Pixel 2 XL 驱动的谷歌 Daydream 耳机上看到的视觉效果没有太大区别。国外或博物馆的虚拟之旅很吸引人，但这种体验只能持续几分钟。老实说，我不想开着这玩意环游世界。

没有令人难以置信的引人入胜的游戏，没有杀手级应用，没有真正身临其境或引人入胜的体验。来自 WWE、网飞或 NextVR 等合作伙伴的视频内容并没有真正利用为您提供的几个自由度，并且由于视频质量差，往往会使您通过 HMD 观看的内容不太吸引人。它也缺乏像 YouTube 这样的官方谷歌应用，迫使你通过 Oculus 自己的较小的服务来观看类似的内容。但是现在有了真正的硬件来制作应用程序(并在假期刺激购买)，随着开发者在未来几个月加快速度并更新软件，这种情况可能会改变...或者几年。

Oculus Go 在虚拟现实方面的表现是否足以证明其 200 美元的价格是合理的？那取决于你在寻找什么。事实上，它是一个独立的便携式虚拟现实耳机，具有良好的视觉保真度，令人印象深刻，并将成为任何对这一概念感兴趣的人的新奇礼物。如果你有一些钱，把你的纳税申报表花在像 Go 这样的酷玩具上并不是最糟糕的决定，特别是如果你没有打算购买 Android 设备或高端电脑的话。

但是，如果你已经有了一部漂亮的安卓手机，你就可以获得相当于难以置信的相似体验，拥有可比较的应用选择，价格比 Go 便宜一点。在像[【400 美元的联想幻影 Solo](https://gizmodo.com/lenovo-made-a-souped-up-oculus-go-thats-missing-just-on-1825770797) 这样的设备上花更多的钱会给你额外的好处，比如更好的视觉效果和更多的行动自由。不过，在谷歌的 Daydream 平台上，你会有非常相似的应用选择，该平台也缺乏娱乐软件。

Oculus Go 确实让我对虚拟现实的期待清醒了一些。我花了几个多小时使用 PC 驱动的 HMD、智能手机驱动的耳机和游戏机专用的 PlayStation VR。每一种都有自己的优点和缺点，我一直是那个大谈高质量虚拟现实体验的人，即使是在移动设备上。但是，当呈现的现实如此短暂地吸引人时，什么是“高质量的虚拟现实”？作为一款设备，Go 无疑是这项技术的一大进步。但它和联想 Mirage Solo 等设备，如果没有开发者越来越多的支持，很快就会从迷人的体验变成令人遗憾的购买。
