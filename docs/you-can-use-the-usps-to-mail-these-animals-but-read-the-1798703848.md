# 你可以使用美国邮政总局来邮寄这些动物——但是首先要阅读规则

> 原文：<https://lifehacker.com/you-can-use-the-usps-to-mail-these-animals-but-read-the-1798703848>

关于 USPS 的观点可能会因你问谁(或你丢失了什么包裹)而不同，但该机构确实运送几乎任何东西。然而，知道美国邮政总局还愿意运送什么可能会让你吃惊:某些种类的活动物，而且很多。根据你要邮寄的东西，你必须遵循一些不同的规则 ，但是只要你坚持计划，你的小动物应该会平安无事地到达目的地。

### **确保你的集装箱建造坚固**

不，你不能把你的石龙子包在气泡包装里，然后放在你剩下的亚马逊 Prime 盒子里邮寄。根据美国邮政总局的集装箱要求 ，除了清楚标明里面装的是什么之外，你的运输集装箱应该“至少由 275 磅重的双层瓦楞纸耐气候纤维板(W5c)或同等材料制成”。容器必须是防逃逸的(当然)，提供足够的通风，并且应该能够承受正常的搬运而不被压碎。不，你不能使用任何美国邮政包装。

### **(大部分)鸟对船很酷**

[24 小时](https://pe.usps.com/text/pub52/pub52c5_008.htm#ep203303) 以下的家禽是可以邮寄的，只要孵化场代表注明孵化日期(包括小时)，并注明孵化场的地址。然而，并不是所有的鸟都是邮件友好型的。只有“鸡、鸭、鸸鹋、鹅、珍珠鸟、鹧鸪、野鸡(仅在四月至八月期间)、鹌鹑和火鸡”有资格邮寄。USPS 对家禽的处理略有不同，以避免在周日、国家法定假日或前一天下午邮寄。

根据美国邮政总局的指导方针，成年鸟也可以邮寄，但你必须遵守一系列法规，包括“莱西法案，濒危物种法案(ESA)，美国农业部，美国鱼类和野生动物服务局的法规，以及任何州，市或地方法规”。鸟类的重量必须在 6 盎司到 25 磅之间，并且必须使用美国邮政认可的制造商生产的容器，你可以在这里找到[。](https://postalpro.usps.com/) 

### 如果你喜欢的话，你可以邮寄 bug

需要帮助你在全国各地的朋友给他的杏仁农场授粉吗？为什么不给他送些蜜蜂呢？你可以通过邮件 [发送活蜜蜂](https://pe.usps.com/text/pub52/pub52c5_008.htm#ep183969) ，只要它们是蜜蜂。如果你是空运，你将被限制只能运送蜂王(以及最多八只随行蜜蜂)，但地面运输允许运输任何类型的蜜蜂。此外，你必须在你的包装上写上“活蜂王”，大概是为了吓唬 USPS 的员工小心处理你的昆虫。

蝎子也可以邮寄，但仅限于医学研究目的，或用于制造抗蛇毒血清。你的蝎子包必须有两个容器，最里面的一个容器可以防止食肉蜘蛛的刺穿(它们不是昆虫！).也许在你去邮局之前可以交些科学家朋友。

### 微小的冷血动物是公平的猎物

扎克的短吻鳄 粉丝们会很高兴地知道，虽然鳄鱼通过潜移默化的方式快速生长是虚构的，但在邮件中收到一条鳄鱼是完全可能的。像变色龙、小短吻鳄和蝾螈这样的动物可以邮寄，只要它们不超过 20 英寸，并且在运输过程中不需要任何食物。此外，如果它有一种“令人讨厌”的气味，不要认为你能邮寄它。

### 你也可以邮寄死去的东西！

只要你采取适当的步骤来确保它符合协议，你上周拿出的珍贵的游戏鸟是有资格装运的。只要液体容器密封良好，死亡的动物或动物部分应该用干冰或水冰冷藏。毛皮、兽皮和兽皮等物品必须适当干燥或处理。邮寄的动物不能违反原产国或目的地国的任何法律。

与所有动物运输 一样，你需要确保容器贴上正确的标签，并且你的货物可以在没有食物和水的情况下存活。此外，你很有可能会被收取一些特殊的手续费，所以预计这比邮寄一本书给你的笔友要贵一些。

### **不要邮寄你的宠物**

当然，获准动物名单意味着某些动物对于邮政运输来说是不安全的。除了前面提到的鸟类，你不能邮寄温血动物。这意味着没有狗，猫，老鼠，兔子，或任何类似的东西。像蛇、乌龟这样的爬行动物是不可消灭的，无论有毒与否。有毒的昆虫和蜘蛛也不允许邮寄。

最终你将不得不和你的动物伙伴一起旅行，所以你应该想出最好的方法来保证它们的安全和存活，无论是在飞机上 还是通过 [公共交通](http://lifehacker.com/how-to-take-your-dog-on-public-transportation-safely-1796677574) 。
