# 使用 WhoSampled 应用程序查找无意中听到的音乐样本

> 原文：<https://lifehacker.com/look-up-overheard-music-samples-with-the-whosampled-app-1827397807>

[WhoSampled](https://www.whosampled.com/) ，识别音乐样本的服务，最近 [更新了他们的 iOS/Android 应用](https://www.whosampled.com/apps/) 以包括 [Shazam](https://www.shazam.com/) 风格的音乐识别。现在，当你在酒吧或商店里听到一首歌时，你不仅可以查找这首歌，还可以查找它采样的其他音乐。



WhoSampled 返回艺术家的名字，并跟踪它是否在他们的样本数据库中，所以你可以用它作为 Shazam 或 [SoundHound](https://soundhound.com/) 的低功能替代。如果歌曲在样本数据库中，WhoSampled 将加载样本歌曲，您可以在应用程序中播放。

有两个巨大的警告。首先，WhoSampled 并不加载实际播放的歌曲的可播放版本——只是它采样或被采样的其他歌曲的版本。相反，它会链接到 iTunes 中的歌曲。如果你有苹果音乐，那实际上是理想的，但否则你可能会觉得这很烦人。

第二，音乐识别功能费用为 9.99 美元/年。这还不包括该应用在 iOS 上 3.99 美元的价格，以及免费 Android 应用上的广告。一年多收十块钱感觉很贪心，我估计 WhoSampled 的客户群也会同意。

如果你经常在新音乐上使用 WhoSampled 比方说你是一个总是在寻找新素材的 DJ——这个新功能会帮你节省一步。如果你对年度订阅不感兴趣，那就等等吧。希望被抽样的人会意识到他们要价太高，并在应用程序价格中包含该功能。在那之前， t he 非订阅版仍然是一个很棒的工具，拥有的功能比如导入 你手机的 音乐库。
