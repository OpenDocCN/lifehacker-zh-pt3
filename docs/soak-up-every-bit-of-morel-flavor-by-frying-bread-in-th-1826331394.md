# 用剩下的黄油煎面包，吸收每一点羊肚菌的味道

> 原文：<https://lifehacker.com/soak-up-every-bit-of-morel-flavor-by-frying-bread-in-th-1826331394>

羊肚菌是两样东西:好吃又贵。如果你要拿出需要的面团(或者经历寻找它们的麻烦)，你要确保你能享受到它们浓郁的肉味。



除了适当地清洗它们并驱除虫子 之外，你唯一需要做的就是用盐黄油将它们煎至金黄色，但这样做通常会让你得到满满一锅羊肚菌味的黄油。当然，你可以把它倒在蘑菇上，但是那会导致黄油留在盘子里。相反，你可以在黄油中煎一片面包，吸取最后一滴丰富的泥土脂肪。在那之后，除了享受你的奖励之外，没什么可做的了，然后尽可能多次重复这个过程，直到羊肚菌季节结束。
