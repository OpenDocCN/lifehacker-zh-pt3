# 如果没有有线电视，去哪里看《权力的游戏》

HBO 的*权力的游戏*第七季将于本周日首播，给你足够的时间来计算在 [你的死亡池](http://lifehacker.com/how-to-organize-a-game-of-thrones-death-pool-1796893792#_ga=2.242204170.903673921.1499692190-167189482.1495477587) 中哪里下注，以及如何在不需要你表哥的男朋友的妈妈的账户信息的情况下观看该剧。这很简单——只要你每月有 15 美元。

### 通过免费试用节省一些现金

注册另一项流媒体服务意味着你的信用卡上又多了一笔账单。如果你只是想看看维斯特洛有什么值得大惊小怪的，或者利用接下来的几周时间赶上最新一季，你可以通过你的订阅电视提供商或 HBO NOW 应用程序本身注册 HBO NOW 的 [30 天免费试用](https://help.hbonow.com/app/answers/detailHBO/a_id/17) 。只需在试用期结束前取消您的订阅，并避免 14.99 美元的费用。

### 如果你敢，海盗集

守财奴要小心了:试图寻找你最喜欢的奇幻剧的盗版流可能会让你陷入法律纠纷。托管《权力的游戏》剧集的 Torrent 网站过去曾收到 HBO 的撤下通知，而 torrent 用户则被指引以更合法的方式获取 HBO 的内容——即现在就注册 HBO。你可以通过一些网络搜索找到非法的流媒体网站，但很有可能它们不会持续很长时间。
