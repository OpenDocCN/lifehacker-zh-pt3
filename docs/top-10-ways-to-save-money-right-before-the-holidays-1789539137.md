# 节前省钱的十大方法

> 原文：<https://lifehacker.com/top-10-ways-to-save-money-right-before-the-holidays-1789539137>

你可能会认为为那些最后一分钟的礼物省下一两块钱已经太晚了，或者当你为别人买东西的时候，找到现金来犒劳自己也太晚了。好吧，这里有一些方法来找到一些额外的划痕，使圣诞节更加明亮。

### 10.使用可以节省开支的应用程序

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-vdOXuSI9MTg&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-vdOXuSI9MTg&start=0) 

如果实际上努力留出一定数量的钱对你来说太多了，以至于你无法应对假期中的压力和疯狂，那就让科技来为你做吧。像 Digit、Acorns 和 Tip Yourself [这样的应用程序免去了储蓄](https://lifehacker.com/these-apps-are-the-perfect-money-saving-tools-for-lazy-1778298912) 的麻烦，它们中的许多甚至可以在你外出时为你省钱，并用于你的正常开支。

为了省钱而花钱听起来可能会适得其反，但这样想想，如果你无论如何都要花钱，你不妨做一些简单的事情，比如将你购买的东西四舍五入到最近的美元，然后用 [省下找零的](https://lifehacker.com/acorns-automatically-saves-and-invests-your-spare-chang-1627545527) ，或者给自己 [小费以应对麻烦](http://twocents.lifehacker.com/tip-yourself-is-like-a-virtual-savings-jar-1764147868) 。到季末，你会有一点零花钱给自己买点好东西。

### 9.为假期设立一个自动储蓄账户

如果你不信任一个应用程序来做这件事， [大多数银行都有自动储蓄账户](https://lifehacker.com/august-s-money-challenge-save-your-spare-change-1784542075) ，在那里他们会为你做四舍五入，然后把零钱存入你也在他们那里开立的储蓄账户。例如，富国银行称之为“随用随存”，但是 [你的银行将会有自己华丽的名字](http://lifehacker.com/automatically-save-your-change-217079) 。打电话询问，通常是免费的。另外，这很容易被忘记，所以你可以在某一天登录你的银行账户，看看你已经积累了多少。

### 8.拿回现金，在你发现的交易中赚钱

说到省钱的应用程序，其他应用程序实际上会根据你买的东西给你返现。 [有些是伟大的](https://lifehacker.com/the-cash-back-apps-that-can-actually-save-you-money-w-1698209178) ！其他的，就没那么多了。但是如果你使用好的，你可以根据你必须购买的东西得到额外的一点钱，而不会有太多的麻烦。

这个列表也不是唯一的——在这个伟大的主题 中，你们有大量的建议 [来帮助你找到更多的钱，或者隐藏在你的预算中，或者以折扣、回扣和现金返还计划的形式。](https://lifehacker.com/what-s-your-favorite-app-for-saving-money-1686172246)

### 7.尽可能获得优惠券代码和折扣(并把多余的钱藏起来)

省钱的一部分——尤其是在像这样的最后一分钟——是每当你达成一笔小交易，比如 10%的折扣或免费送货，你也必须承诺把你无论如何都准备好要花的差额放在一边，这样你以后就有了它(或者可以花在特别的东西上)。如果你经常网上购物，可以考虑一下 [这款我们最喜欢的省钱浏览器扩展](https://lifehacker.com/the-best-browser-extensions-that-ll-save-you-money-and-1702736679) ，比如 [Honey](http://www.joinhoney.com/) 和 [看不见的手](http://www.getinvisiblehand.com/) 。不像上面中的一些应用，它们不会为你存下这笔钱，所以你必须记得这么做！

### 6.了解商店和广告是如何操纵你的(即使你认为他们没有)

上周我们谈到如何 [确保假日交易实际上是交易](http://lifehacker.com/top-10-ways-to-make-sure-that-black-friday-or-cyber-mon-1789238997) 时提到了这一点，但请记住，商店喜欢操纵你花钱——毕竟，这就是他们所做的，让你花钱。这很好！这是他们的特权。但也是你自己教育自己，做一个聪明的消费者。

首先，学习 [广告如何对你产生影响](https://lifehacker.com/how-advertising-manipulates-your-choices-and-spending-h-30812671) ，即使你认为它没有。然后读读 [商店如何喜欢操纵你的感官](https://lifehacker.com/how-stores-manipulate-your-senses-so-you-spend-more-mon-475987594)——听觉、视觉，甚至嗅觉——来鼓励你购买你并不想买的东西。此外，记住 [即使是网上商店也不能幸免](https://lifehacker.com/the-tricks-online-stores-use-to-make-you-spend-more-mon-1788887501) 。他们使用通知和警报来鼓励你消费，并欺骗你产生一种虚假的匮乏感，这样你就会打开钱包。

有了正确的知识武装，你将度过假期的另一个阶段，花掉你想要的，仅此而已。

### 5.不要只是买东西，也要卖东西

每个人都认为假期是一年中我们都清空银行账户的时候，因为我们必须购买礼物和包装等等——但我们只是理所当然地认为我们是花钱的人。不要——也做一些销售。对于所有进入你家的礼物和所有东西，考虑一下 [扔掉你那些没用的垃圾，从中赚些钱](https://lifehacker.com/the-complete-guide-to-selling-your-unwanted-crap-for-mo-5981335) 从中赚些钱。

毕竟，你可能会收到一些礼物。就当是“腾出空间”此外，对于每一个去商店购物的人来说，可能有更多的人在网上寻找优惠——你可以为他们提供你可以不需要的二手商品的优惠。事实上， [这是上个月的 Lifehacker 金钱挑战赛](https://lifehacker.com/november-s-money-challenge-sell-your-unwanted-stuff-1788424216) 。就算没参与，现在做也不迟。

### 4.在业余时间赚些钱

我知道，我知道，“什么业余时间？”尤其是节假日前后，时间紧缺。然而，如果你没有可以放弃的东西，但你有时间，在 T2，在 T5，你也可以做很多事情。

无论是加入零工经济，在 Airbnb 上出租你的汽车、停车位，甚至是公寓，还是通过 Fiverr 为人们跑腿，如果你有一点停机时间， [你就可以赚一点刮刮乐](https://lifehacker.com/how-to-make-money-doing-almost-nothing-1462909872) 。强调一下*的小*、——你在这里并没有变得。

### 3.使用正确的信用卡赢取奖励和现金返还

有很多[的好理由在假期使用你的信用卡](http://lifehacker.com/when-should-i-use-credit-and-when-should-i-use-debit-wh-862086499) ，只要你觉得你能负责任地使用它们。奖励，积分，返现优惠， [最佳奖励卡全部提供](https://lifehacker.com/five-best-rewards-credit-cards-1622819827) ，当你在一月份付清账单时，你会剩下一吨积分来消费，并根据你认为合适的方式返现使用。

如果你超级精明，用它们 [在你购物的商店和餐馆购买礼品卡，并获得双倍的奖金](https://lifehacker.com/get-bonus-rewards-points-for-buying-gift-cards-5980228) 。你可以 [明智地利用它们](http://lifehacker.com/whats-the-best-way-to-use-my-credit-card-rewards-and-tr-5976561) 来犒劳自己，或者兑现你对爱人许下的节日承诺。

### 2.把省钱变成游戏

也许假期并不是在你的脑海中增加一件事情的最佳时机，但对我们大多数人来说，假日购物并不是世界上最有趣的事情，如果你想增加一点趣味， [考虑在组合中加入一点游戏化](https://lifehacker.com/the-best-tools-to-productively-gamify-every-aspect-of-1531404316) 。

像我们上面提到的那些应用程序试图避免储蓄的麻烦，但是其他的像 [SmartyPig](http://www.smartypig.com/) 和 [SaveUp](https://www.saveup.com/) 让它变得更有趣一点，并帮助你设定你可以努力实现的目标。也许是节后休假，或者是新年给自己准备的特别的东西。你想怎么做就怎么做。

### 1.利用假期作为谈判时间来降低你的账单

年底是协商来年账单和费用的好时机。一些公司提供年终促销，一般来说，大多数人都很慈善——特别是当你打电话要求降低 [信用卡利率](http://lifehacker.com/negotiate-a-lower-credit-card-apr-with-this-script-1690542594) 、降低杂志或有线电视节目的月订阅费、 [减免医疗账单](http://lifehacker.com/lower-your-medical-bills-with-a-quick-payment-offer-30883326) 或任何其他事情时，如果你很友好，对他们表示出一点应有的尊重。毕竟对于某些公司来说， [你要做的就是问](https://lifehacker.com/reduce-your-bills-by-just-asking-5127638) 。

查看 [我们的降低你每月账单的完整指南](https://lifehacker.com/a-bill-by-bill-guide-to-saving-money-on-your-monthly-ex-5823762) 以获得提示，无论你正试图削减什么，如果你需要一点帮助， [试试这些脚本](http://lifehacker.com/save-a-pile-of-cash-every-year-with-these-five-phone-ca-5887659) 。
