# 没人能从你身上拿走的东西

> 原文：<https://lifehacker.com/the-one-thing-that-nobody-can-ever-take-from-you-1824988465>

欢迎回到 [**周中冥想**](http://lifehacker.com/tag/mid-week-meditations#_ga=2.170929957.440406047.1499703745-27973805.1434581949) ，Lifehacker 的每周一次的斯多葛派智慧之池探索，以及使用它的水域来反思和改善你的生活的指南。



本周的精选来自爱比克泰德的*演讲*。在 [第二册，第十章](http://classics.mit.edu/Epictetus/discourses.2.two.html) 中，他告诉我们，除了我们自己不可战胜的意志，我们没有更大的力量:

![](../Images/aef335eb562a1da6957ce25a574b4aa1.png)

“考虑一下你是谁。首先，你是一个男人；这个人除了服从意志之外，没有任何东西高于意志的能力；和能力本身，他拥有不受奴役和自由的服从。那么考虑一下你被理性从什么事物中分离出来。你已经和野兽分开了:你已经和家畜分开了。”



### 这意味着什么

停下来想想你是谁，你是什么。你是一个人，你最大的力量是你的意志，或者说你推理和做出自己选择的能力。它主宰着你生活的方方面面。这种力量是唯一不能被任何人控制或夺走的东西，也是你与世界上其他动物的区别。不要忘记你拥有这种力量，不要浪费它。

### 从中吸取什么

对我们大多数人来说，生活不过是一长串的选择:你选择做什么，去哪里，吃什么，喝什么，如何做事，等等。但更重要的是，你可以决定经历对你意味着什么，你可以选择如何应对它们。没有人能把它从你身边拿走，即使是神自己也不行，爱比克泰德在第一本书的前面说:

> 我必须死。我必须悲叹而死吗？我必须戴上镣铐。难道我也要悲叹吗？我必须流亡。那么，有谁会阻止我带着微笑、快乐和满足去呢？“告诉我你拥有的秘密。”我不会，因为这是我力所能及的。“但我会把你锁起来。”伙计，你在说什么？我戴着镣铐？你可以绑住我的腿，但我的意志连宙斯自己都无法战胜。

你的意志，你选择弹性和活泼而不是悲伤和悲伤的能力，是你最大的力量。你也许不能总是选择发生在你身上的事情，但你总是有能力选择如何应对。没有动物、人类或上帝能阻止你。
