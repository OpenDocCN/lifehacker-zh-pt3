# 如果你对“干溺死”感到恐惧，请阅读这篇文章

> 原文：<https://lifehacker.com/read-this-if-youre-freaking-out-about-dry-drowning-1796294879>

本月早些时候，一名儿童在与家人涉水时，在明显吸入水 后一周 [死亡。媒体将这种情况称为“干型溺水”，也称为“继发性”或“延迟性”溺水，即一个人在吸入水后最初似乎没事，但几小时或几天后死亡。](https://www.usatoday.com/story/news/nation-now/2017/06/08/texas-boy-dies-dry-drowning-days-after-swimming/379944001/)

这太可怕了——没有人质疑这一点。但是，随着这个故事的恐怖，有一定数量的媒体错误信息，和恐惧的传播，各种儿科医生和急诊室医生现在推回来。这不仅是为了教育公众溺水的真实风险，也是为了宣传急救和医疗干预，并确保我们在谈论溺水时使用相同的术语。

我偶然看到儿科医生凯瑟琳·汉斯莱的这篇文章 ，她在文章中提供了她所谓的“干溺水的现实检查”:

“简而言之，你的孩子不会去游泳，吞下一些水，在水中没有任何问题，然后在 4 天后突然毫无征兆地死于‘干溺水’。”我知道你读过的那些故事，它们也在我的新闻里。我知道失去孩子的想法会让你感到真正的胸痛，因为我也是这样....但是我不希望世界上的孩子们因为他们的父母被一篇研究不足的脸书文章吓到而再也不敢靠近水。<small></small>

首先，她对媒体缺乏对“吞咽”和“吸气”这两个词的区分提出异议:

> 吞咽的水进入食道，向下通过消化道。吞咽氯化池水可能会让你呕吐，但不会让你死于急性肺水肿。吸气是指水进入气管，向下进入肺部。在这种情况下，病人从水里出来后会有一段痛苦的经历。你会看到它，你会知道他们不好。

她也注意到了( [，医学博士 Seth Collins Hawkins 等人](http://journals.lww.com/em-news/blog/BreakingNews/pages/post.aspx?PostID=377) 写的这篇文章也是如此。)，医学上没有公认的“干性溺水”甚至“继发性”或“延迟性”溺水的定义。只有*溺水*——在医学界，这可能是致命的，非致命的，或者非致命的疾病/伤害。

毫无疑问，溺水是儿童死亡的主要原因，但汉斯莱博士定义的“溺水延迟死亡”(这通常是人们谈论干型溺水时的意思)的恐惧被夸大了。Hawkins(令人欣慰地)注意到，虽然溺水事件在后经历症状*相对常见，但这些“症状轻微”的病例进展到死亡是罕见的。至于关注这些症状，他说:*

> 尽管如此，它们肯定会恶化或发展，这就是为什么我们鼓励人们在出现胸痛等警告信号时立即寻求护理。溺水的警告信号是淹没或浸泡后出现呼吸困难、过度咳嗽、口中有泡沫和行为不正常。

但是说到底，术语并不是最重要的事情，很明显，最重要的是如何从一开始就避免这种情况发生。监督你的孩子不仅仅是简单地看着他们:急诊室医生敦促 [触摸监督](http://escambia.floridahealth.gov/newsroom/2014/05/2014-05-20-touch-supervision-drowning-prevention.pdf.html) ，这意味着即使有救生员在场，也要一直在孩子伸手可及的地方。应该给每个孩子分配一个成年人——人们很容易想，“哦，10 个成年人在这里，如果一个孩子开始溺水，有人会注意到的，”因为他们可能不会，事实上。

除此之外，教你的孩子在适当的年龄游泳，学习心肺复苏术。不要让媒体吓到你，让你无法享受在海滩的美好时光。
