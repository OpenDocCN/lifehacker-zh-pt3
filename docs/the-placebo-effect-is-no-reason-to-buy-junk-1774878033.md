# 安慰剂效应不是购买垃圾食品的借口

我们都相信一些我们知道可能不起作用的东西。也许是 [维生素 C](http://lifehacker.com/the-no-bs-guide-to-boosting-your-immunity-and-avoiding-5858209) 当上班的每个人都感冒的时候，或者是 [#bootea 当我们节食的时候](http://vitals.lifehacker.com/instagram-s-viral-weight-loss-teas-are-just-laxatives-1773839010) ，或者是跑步的时候压缩袜子。“就算不行，又有什么坏处？”我们告诉自己。事实是，它不是无害的，我们只是在欺骗自己。

仅仅因为那种药片或茶*不是* *有毒*并不意味着它是*无害*:你在它上面花费了金钱、时间和精力，这些精力你本可以花在寻找或使用真正有用的东西上。更糟糕的是，如果它看起来对你“有效”，你可能会相信它是真的。我们中的许多人把希望和钱包倾注在根本不起作用的伪医学治疗(或饮食或训练计划)上，只是因为安慰剂效应让我们认为它们起作用。

当你在药店买一包应急药品时，你会有多担心？那是个人决定，但我要论证的是，应该大于零。如果你要以开放的态度去尝试一些未经证实的事情，你也应该足够开放地向自己承认，没有它你可能会过得更好。

### 安慰剂效应没有你想象的那么有用

很久以前，研究人员发现，没有真正药物的药丸或注射器可以缓解症状——有时几乎和真药一样好。“ [安慰剂效应](http://www.cancer.org/treatment/treatmentsandsideeffects/treatmenttypes/placebo-effect) ”的发现改变了医生对药物的评估方式。由于患者可能会因服用假药而有所改善，药物试验可能会 [将新药与安慰剂](http://www.fda.gov/RegulatoryInformation/Guidances/ucm126501.htm) 进行比较，期望两组患者都有所改善。这种药物不仅仅要起作用；它必须比安慰剂更有效。

因此，如果安慰剂能让你感觉更好，为什么不利用这种效应，故意服用假药呢？毕竟，糖丸应该比真正的药物便宜，而且不应该引起任何严重的副作用。如果我们的思想可以治愈我们的身体(或者至少让我们感觉更好)，这难道不是一种胜利吗？

不幸的是，安慰剂不是这样“起作用的”虽然一个 [脑-体协同](https://www.psychologytoday.com/blog/the-fallible-mind/201201/the-placebo-response-not-in-your-head-in-your-brain) 可能解释了*一些*安慰剂效应，但其中很大一部分是基于根本不是由安慰剂引起的因素。1997 年，两位德国科学家分析了 1955 年的原始论文 ，该论文认为安慰剂效应在医学上有意义。他们发现大部分影响不是心理上的，而是由于即使没有虚拟药物也会发生的事情。其中许多也适用于日常生活:

*   随着时间的推移，人们自然会变得更好。例如，一项感冒治疗的试验发现，服用安慰剂的人在 6 天内有所改善。由于感冒只会持续大约 7-10 天，最糟糕的症状大约持续三天，即使没有安慰剂，这也是你所期望的。
*   人们在已经感觉好些的时候服用安慰剂。在一些研究中，当患者的病情得到控制时，只给他们服用安慰剂；如果他们有严重的症状，他们得到了真正的药物。在那次试验中，安慰剂因控制了患者的症状而受到好评。如果你用维生素或顺势疗法治疗轻微症状，你也在做同样的事情，但当事情变得严重时，就去看医生。
*   安慰剂可能不是唯一的治疗方法。试验感冒药的人被允许休息、洗热水澡和做其他能让他们感觉更好的事情。同理，如果你在节食的时候喝减肥茶，你的体重减轻可能是因为节食而不是茶。
*   **症状起伏不定。在试验中，当人们变得更好时，安慰剂得到了信任，但当症状变得更糟时，研究人员没有责怪安慰剂。例如，在一项试验中，21%的中风患者在服用安慰剂后有所改善，但 53%的患者死亡。研究人员称安慰剂有 21%的“有效”如果你只注意到一种可能的治疗对你有效的次数，而忘记了它不起作用的次数，那就是经典的 [确认偏差](http://www.webmd.com/cold-and-flu/understanding-common-cold-symptoms) 。**

当我们评估自己的经历时，我们很容易犯这样的错误。如果你的肌肉酸痛，你喝了酸樱桃汁，第二天感觉更好，你可能会认为果汁对你有用——即使有或没有果汁你都会有所改善。成为认知偏见的牺牲品并不意味着我们容易受骗或愚蠢；这意味着我们是人类。

底线是，尝试一些无用的东西，希望从安慰剂效应中获利，这不是一个好策略。即使安慰剂看起来“有效”，它可能对你没有任何好处。
