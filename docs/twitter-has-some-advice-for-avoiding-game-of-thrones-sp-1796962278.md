只需要一个名字在推特上被一遍又一遍地转发，就能毁了 GoT。Twitter 告诉赫芬顿邮报，这是 2016 年关于该节目的最多推文。错过重要的一集意味着在你知道发生了什么之前你有几个小时的时间。所以，保护自己的第一步是使用 Twitter 的静音功能。下面是基本静音的工作原理:

桌面上的

1.  **单击您的个人资料图标，然后选择设置和隐私。** 
2.  **在菜单中找到静音单词选项，然后单击它。** 
3.  **点击蓝色的“添加”按钮。** 
4.  **键入您想要静音的单词，并选择您想要它保持静音的时间。**

**您将看到一个列表，上面列出了您已经静音的内容，还有一个红色按钮，一旦您准备好开始对话，就可以取消静音。**

**对于移动设备， [根据你使用的平台不同，](https://support.twitter.com/articles/20175032) 略有不同。**
