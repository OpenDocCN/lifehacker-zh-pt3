# 这是你快乐需要的朋友数量

> 原文：<https://lifehacker.com/this-is-how-many-friends-you-need-to-be-happy-1823425885>

朋友让你快乐、健康，当大雨倾盆时，他们会在你身边。但是你需要多少呢？事实证明,《老友记》已经搞清楚了所有的科学原理。

Watch

早在 90 年代初，英国人类学家罗宾·邓巴 [博士得出了一个有趣的结论](https://www.sciencedirect.com/science/article/pii/004724849290081J?via%3Dihub) :由于我们大脑新皮层的大小，人类可能只能平均与 148 个人保持社交关系，也就是所谓的 [邓巴数](https://en.wikipedia.org/wiki/Dunbar%27s_number) 。 [更多的社会信息处理需求需要更多的认知资源](http://rspb.royalsocietypublishing.org/content/280/1765/20131151) ，而我们只有这么多的脑力。基本上，我们倾向于在生活中拥有 150 段有意义的关系，无论是家人、朋友还是泛泛之交。你的脸书可能有成百上千的“朋友”，但他们中的大部分，如果不是大多数，都不在你的脑海里。

后来， [邓巴的研究引出了“邓巴层次”的概念，](https://arxiv.org/abs/1604.02400) 其中考虑到了个体之间的情感亲近程度。这意味着你的人际关系看起来更像是一层层的，而不是 150 个人的云。最接近的一层 [有三到五人](https://www.technologyreview.com/s/601369/your-brain-limits-you-to-just-five-bffs/) ，下一层有 15 人，然后是 50 人，以此类推。这一内层构成了你的“至关重要的友谊”，或者你的亲密朋友的核心圈子。在你的生活中，这些人是你应该经常见面，谈论个人问题，并保持强烈的情感联系。在电视剧《老友记》中，每个主角——罗斯、瑞秋、乔伊、菲比、莫妮卡和钱德勒——这五个人在他们的生活中，从生物学和社会学的角度来看，这是一个相当不错的模式。如果你能以同样的方式维持三到五个亲密的友谊，你更有可能感到满足。毕竟，谁不会和永远支持你的人在一起过得更好呢？

你真的需要三到五个朋友来维持生活吗？可能没有，但是你至少需要一个。随着年龄的增长，孤独和寂寞正在杀死我们所有人 ，所以你至少要有一个最好的朋友是至关重要的。如果你没有至少一个好朋友，或者你想争取那个神奇的三五范围，是时候伸出手，把你的熟人变成朋友了。请人们喝杯咖啡，看部电影，或者吃顿午餐。加入当地的俱乐部或团体，不要害怕和那些可能已经放弃的老朋友联系。记住，要交朋友并保持友谊，你也需要支持他们。
