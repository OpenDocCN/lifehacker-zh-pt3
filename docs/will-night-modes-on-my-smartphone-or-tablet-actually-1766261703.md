# 智能手机或平板电脑上的“夜间模式”真的能帮助我获得更好的睡眠吗？

亲爱的睡前浏览器，
让我们从我们所知道的开始:眼部光照会扰乱你身体产生 [褪黑激素](https://en.wikipedia.org/wiki/Melatonin) 的自然能力，褪黑激素是由你的松果体产生的一种帮助你入睡的激素。当你打乱这个过程时，你的 [生理节奏](https://en.wikipedia.org/wiki/Circadian_rhythm) (或“内部时钟”)就会被打乱。我们应该在黑暗中睡觉，而不自然的光源让这变得更加困难。事实上，根据发表在*内分泌学杂志&新陈代谢* 上的一项研究 [，即使是视觉障碍者对漫射光也有类似的反应。](http://www.ncbi.nlm.nih.gov/pmc/articles/PMC3047226/)

智能手机和平板电脑对你的大脑有着同样的破坏性影响。光线可以提前或推迟你的生理节奏，甚至你的屏幕产生的少量光线也会提前你的生理节奏(即使在黑暗的房间里)。基本上，你的身体认为你在看阳光(即使你有意识地知道现在是晚上)，并做出相应的反应。无数的研究(就像上面提到的那个)表明，你的设备发出的明亮的蓝光 [特别具有破坏性](http://www.health.harvard.edu/staying-healthy/blue-light-has-a-dark-side) ，因为它的短波长是告诉你的大脑保持警惕，而不是休息。事实上，瑞典的一项研究， [发表在*【PLOS】One*](http://journals.plos.org/plosone/article?id=10.1371/journal.pone.0076707)*上，表明蓝光甚至可能比咖啡更能让人更加清醒 。*

### *睡前少点蓝光是好事*

*这就是苹果新宣布的夜班和 Android N 即将推出的夜间模式等功能的用武之地。它们会自动检测何时接近就寝时间，并降低亮度和调整显示器的色温，以最大限度地减少蓝光的数量。这给你的屏幕提供了一个更温暖、更深的调色板，可能看起来不那么清晰，但对眼睛来说更容易。理想情况下，你可以想看多长时间就看多长时间，因为总体来说你得到的光线越来越少，而且没有那种有问题的蓝光。*

*从表面上看，这种逻辑是合理的。通过移除令人讨厌的蓝色物质，你可以防止你的设备欺骗你的大脑认为是时候醒来并关闭褪黑激素的产生。也有证据支持。多项研究表明，晚上戴橙色防蓝光眼镜避免蓝光对睡眠模式有积极影响。例如，发表在*青少年健康杂志* 上的一项研究 [发现，十几岁的男孩在睡觉前戴上防蓝光眼镜能够睡得更好，而发表在*时间生物学国际*](http://www.jahonline.org/article/S1054-139X%2814%2900324-3/fulltext) 上的另一项研究 [对成年人也有类似的发现。在这项研究中，一组被给予紫外线阻挡眼镜，另一组被给予蓝色阻挡物。据报道，蓝光阻断组比紫外线阻断组更容易入睡(但值得注意的是，与青少年测试相比，成年参与者睡前使用设备的次数要少得多)。](http://www.ncbi.nlm.nih.gov/pubmed/20030543)*

### *夜间模式减少蓝光(但它可能不会拯救你的睡眠)*

*然而，我们不能确定的是，像夜班和夜间模式这样的功能是否足够，或者从长远来看是否具有与蓝光阻断眼镜相同的效果。目前还没有任何关于这些功能及其对人们睡眠的影响的长期研究，也没有关于像 [f.lux](https://justgetflux.com/) 或 [Twilight](https://play.google.com/store/apps/details?id=com.urbandroid.lux&hl=en) 这样的应用程序的长期研究，今天许多人使用这些应用程序做同样的事情。我们可以*假设*将你的设备屏幕的调色板转移到更温暖的温度会减少蓝光照射，从而帮助你避免褪黑激素生产的中断和扰乱你的昼夜节律，但我们还不确定*。** 

**然而，已经有研究测试了色温及其对活动、睡意和睡眠质量的影响。发表在*生理人类学和应用科学期刊* 上的一项研究 [表明，当谈到光线对你睡眠的影响时，色温比照明水平更重要。根据这项研究，低温照明(红色和橙色等暖色)使参与者更容易打瞌睡，并减少他们的活动，尽管亮度很高(这对你的夜间模式设备来说是个好消息)。发表在同一份期刊](http://www.ncbi.nlm.nih.gov/pubmed/10510514) 上的另一项研究 [表明，色温也可能影响你的睡眠质量。研究人员发现，睡眠前和睡眠中的高色温(冷色调，如蓝色和白色)对参与者在试验中的整体休息产生了负面影响。不幸的是，这两项研究都非常小(第一项研究中只有 8 名男性，第二项研究中有 7 名男性)，所以很难说当人们手中有数百万台设备时，这一切会如何。](http://www.ncbi.nlm.nih.gov/pubmed/15840951)**

**苹果的夜班和安卓的夜间模式看起来肯定是朝着正确的方向发展。也就是说，它们可能不是苹果和谷歌宣称的万能药。你的设备仍然会发出一些光，虽然这些功能可能会在生理层面上让你更容易入睡，但它们仍然会在心理层面上让你分心(特别是如果你不断收到通知的话)。如果你想在床上使用你的设备，使用它们当然没有坏处；他们显示出很大的希望。只是要控制你的期望，记住当谈到获得体面的睡眠时，还有很多其他因素。**
