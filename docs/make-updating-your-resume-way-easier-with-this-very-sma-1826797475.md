# 用这个聪明的提示让更新你的简历变得更容易

> 原文：<https://lifehacker.com/make-updating-your-resume-way-easier-with-this-very-sma-1826797475>

你正在找一份新工作，所以是时候更新你的旧简历了。但是当你试图描述你的上一个位置时，你会盯着天空想，“等等，我到底做了什么？”那么，谁比雇主本身更能描述你的角色的职责呢？

Watch

当你整天坐在办公桌前阅读这样的网站时，没有必要浪费时间斟酌最佳词汇来描述你应该在做什么。正如 Reddit 用户 emhelldpp 建议的 一样，有人已经发现了这一点。当你被雇佣时，简单地从你的雇主的工作列表中复制工作描述，并在你需要时保存它。到了简历更新的时候，你所要做的就是把详尽的、听起来令人印象深刻的工作职责粘贴到你的简历上——然后做一些必要的语言调整。

当然，你也可以在递交了辞呈或者已经离职之后这样做，因为他们可能已经为你的替代者列出了新的名单。或者你可以现在就去查看你公司的招聘信息，看看你是否能找到类似的职位。随身携带一份最新的简历总是一个好主意 。
