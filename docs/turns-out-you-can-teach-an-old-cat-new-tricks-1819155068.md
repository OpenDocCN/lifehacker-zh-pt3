# 原来你可以教一只老猫新把戏

> 原文：<https://lifehacker.com/turns-out-you-can-teach-an-old-cat-new-tricks-1819155068>

信不信由你，猫是可以训练的。事实上，一项新的研究表明，即使是看起来已经习惯了的老猫也可以被训练。



这项研究由科罗拉多州立大学 和 [的洛里·科岗博士领导，发表在期刊*动物*](http://www.mdpi.com/2076-2615/7/10/73/htm) 上，涉及 100 只收容所的猫，它们正在接受所谓的“响片训练”你可以在下面的视频 中看到这项研究的合作者之一，谢丽尔·科勒斯，DVM， [解释这个过程。无论猫的年龄、性别或害羞程度如何，大约 80%的猫容易接受最基本的训练(摸鼻子，就像视频中一样)，近 30%的猫可以学习一些更难的东西，比如按提示坐下。](https://www.youtube.com/watch?v=c5RsAXsZvI8)

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-c5RsAXsZvI8&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-c5RsAXsZvI8&start=0) 

这项研究的结果表明，几乎任何猫都可以被训练，不管它们是老猫还是超级胆小的猫。科洛斯告诉伴侣动物心理学 的扎奇·托德博士，训练课程可能有助于猫应对紧张的环境，比如收容所:

> "...即使一只猫最初看起来很害怕，但许多猫仍然是可以训练的，社交互动真的可以帮助猫积极适应收容所的环境。”

对于新家来说也是如此。所以不要认为现在训练你的猫为时已晚。所需要的只是一些奉献和练习。
