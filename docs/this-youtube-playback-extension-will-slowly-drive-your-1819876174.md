# 这个 YouTube 播放扩展会慢慢让你的敌人发疯

> 原文:[https://life hacker . com/this-YouTube-playback-extension-will-slow-drive-your-1819876174](https://lifehacker.com/this-youtube-playback-extension-will-slowly-drive-your-1819876174)

你可能很不幸，在你附近有一个令人讨厌的 [同事](https://lifehacker.com/passive-aggressive-phrases-to-get-your-coworkers-to-del-1819176196) 。你知道，就是那个说他早餐有多爱吃乳蛋饼的人。我们叫他查德吧。虽然你(很可能)不能把查德的耳机放进果冻里，来报复他把上周垒球失利归咎于你的膝盖，但你可以慢慢地让查德抓狂，伤害最深的地方是:在他宝贵的 YouTube 游手好闲时间里。

在 Lifehacker，这是 [*邪恶周*](https://lifehacker.com/c/evil-week) *，这意味着我们正在寻找不太合适的方法来完成任务。我们喜欢认为我们揭示这些策略是为了帮助你做相反的事情，但是如果你是，事实上，邪恶的，你可能会发现这一周对你很有帮助。这取决于你。*

要完成这个把戏，你需要在敌人的电脑上安装一个 Chrome 引擎扩展。由于它不是官方的 Chrome 扩展，你必须手动完成，所以确保他们出去吃午饭或抽烟休息，这样你就有足够的时间复仇。

这个邪恶的 Chrome 扩展(也称为“jerkface-chrome”)有着一个相当无害、看似有益的名字“[”Chrome 引擎](https://git.scd31.com/laptopdude90/jerkface-chrome) ”，每天你的 officemate 观看 YouTube 视频时，它会将 YouTube 视频播放速度减慢 1%，当它达到原始播放速度的 70%时就会停止。这意味着你的同事将忍受 30 天缓慢但确定的视频速度下降，并且可能想知道他们的计算机(或者如果你幸运的话，他们的大脑)发生了什么。

进入 [Chrome 引擎页面](https://git.scd31.com/laptopdude90/jerkface-chrome) ，选择“chrome-engine-packed.crx”链接。点击页面中间的“Raw”链接下载该扩展(你会在浏览器底部的下载栏中看到它)。打开一个新标签，输入“chrome://extensions”进入 chrome 的扩展页面。将下载的扩展拖到页面中，然后点击 Install。你会在地址栏旁边的扩展列表中看到一个小小的 Chrome 标志。右键点击它，选择 Chrome 菜单中的隐藏，为自己的出色表现而感到欣慰。

### 无论他们在哪里，情况都会变得更糟

如果查德决定打电话请病假，错过几天，不要担心。扩展足够聪明，可以等到它们返回，暂停减速。

如果你的同事“想家”(可能喜欢在笔记本电脑上舒适地播放种族主义的 YouTube 视频)，Chrome Engine 将与他们的谷歌账户保持同步，只要他们登录到 Chrome 浏览器。这意味着无论他们的 Chrome 浏览器在哪里，降低的播放速度都会跟着他们。

该扩展还可以补偿用户选择的回放速度调整。如果你的受害者降低播放速度，它将根据扩展的更新播放速度减慢。最终他们会发现有事发生，但如果你表现得很酷，他们永远不会知道是你做的。开始点煤气灯吧！
