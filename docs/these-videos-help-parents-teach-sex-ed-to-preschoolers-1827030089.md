# 这些视频帮助父母向学龄前儿童教授性教育

> 原文:[https://life hacker . com/these-videos-help-parents-teach-sex-ed-to-学龄前儿童-1827030089](https://lifehacker.com/these-videos-help-parents-teach-sex-ed-to-preschoolers-1827030089)

把一个男孩和一个女孩放在一个浴缸里可以吗？如果你孩子学前班的一个同学过来玩，你发现他们两个从腰部以下都在“玩医生”，你该怎么办？如果你的孩子要求检查你的私处，这让你觉得很奇怪怎么办？

有很多书籍和资源可以和孩子谈论他们的身体、性和生殖。但是它们通常面向那些孩子即将进入青春期的父母——但那已经太晚了。性健康教育家黛博拉·罗夫曼告诉我，孩子们从四岁开始就对这些话题有“正常、自然的好奇心”，如果成年人不在那里指导他们，他们最终会转向同龄人、年长的孩子和媒体获取信息。(你也不能只是等着学校来澄清事情——在 [的一个 Reddit 帖子](https://www.reddit.com/r/AskReddit/comments/6wdfi2/what_incorrect_info_did_your_sex_ed_teacher_tell/?st=jhy73ybo&sh=a741a300) 中，人们分享了他们在性教育课上学到的非常不准确的信息，比如避孕套如何增加怀孕的风险，女孩在上面不会怀孕，以及阴蒂是一个神话。)

谈话不应该仅仅是一次紧张的坐下来的谈话——相反，它应该是一次持续的讨论，比你可能想象的更早开始。这就是为什么《T2》一书的作者罗夫曼帮助为 4-9 岁孩子的父母开发了一系列的动画视频。他们制作了性教育项目 [AMAZE](https://amaze.org/) ，为我们带来了青少年视频，主题包括 [同意](https://offspring.lifehacker.com/these-videos-teach-kids-about-consent-in-an-age-appropr-1819448970) 、性别认同和性侵犯。

该系列名为，帮助父母浏览真实的、有时会让他们的孩子困惑的场景。比方说，你在车里听 NPR 的音乐，你的小孩突然问，“妈妈，什么是强奸？”(视频显示，你可以说“强奸是违法的”，这是一个完全真实的回答。)或者你正在逛玩具店，那里有“女孩通道”和“男孩通道”(抓住机会帮助孩子注意并思考性别标签。)这个视频—“演医生可以吗？”—解释了当孩子对身体和隐私区域好奇时，什么是正常和健康的。

罗夫曼说，许多父母都有一种不合理的恐惧，担心“太多太快的信息”可能对孩子有害，但事实恰恰相反。她说，受过更好教育的孩子更有可能在所有事情上做出更好的决定——包括性。
