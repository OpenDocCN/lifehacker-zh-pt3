# 黑沙桌面

> 原文：<https://lifehacker.com/the-black-sand-desktop-1789113057>

最近我们已经看到了很多水性、海滩 y 、海洋 y 壁纸，这让我们认为你们一定渴望更温暖的天气。这款来自T5 的 SebastianT7，是一款华丽、朴素的黑白桌面，只做了一点点定制。这是所有设置的方式。

Watch

Sebastian 运行的是 Windows，这意味着你需要 Rainmeter 来完成这项工作。如果你不熟悉， [看看我们的入门指南](https://lifehacker.com/how-to-create-an-attractive-customized-desktop-hud-wit-5828789) ，它会帮你的。

一旦你有了基本的东西，你将需要这些组件来使你的桌面看起来像这样:

*   来自沃尔黑文的 [壁纸](https://alpha.wallhaven.cc/wallpaper/60448)
*   用于 Windows 的 [雨量计](http://rainmeter.net/) 系统调试和监控工具
*   雨量计 [谜组曲](http://enigma.kaelri.com/)
*   雨量计的 [Omnimo 套件](http://omnimo.info) 创建黑盒背景和部分状态显示
*   [为雨量计的简易洁净皮肤](http://hiphopium.deviantart.com/art/Simple-Clean-593429217) 为时间和日期显示

这就是这个的全部内容。你必须将一些元素叠加在另一个之上，以获得想要的外观，但这是一个清晰的外观。黑色和白色，最小化，不分散注意力，但当你可以看着桌面时，仍然是有用的。如果你对它是如何制作的有疑问，或者想让你的系统看起来像但又想不出来，点击下面的 Flickr 链接，让 Sebastian 知道你有多喜欢他的作品——然后在那里找到他！
