# 三家美国航空公司将于明年 1 月开始禁止一些智能行李

> 原文：<https://lifehacker.com/three-us-airlines-will-start-banning-some-smart-luggage-1821167556>

对于喜欢从行李箱中为电子设备充电的人来说，智能行李箱是一场小小的革命。但是——也许可以归咎于三星 Galaxy Note 7 的灵感——航空公司因为担心火灾而禁止使用其中一些行李箱。

Watch

[The Verge](https://www.theverge.com/2017/12/7/16746784/smart-luggage-regulations-american-airlines-delta)报道称，从 1 月 15 日开始，达美航空、美国航空和阿拉斯加航空将禁止携带不可拆卸锂离子电池的行李。( [CNN 报道](http://money.cnn.com/2017/12/01/news/companies/american-airlines-smart-luggage-fire-rules/index.html?iid=EL) 其他航空公司可能会效仿。)你可以托运智能包，只要你把电池取出来，放在随身行李里带上飞机。你也可以随身携带装有可拆卸电池的智能包。

在你惊慌之前，检查一下你的智能包是否有可移动电池。( [Away 的](https://www.awaytravel.com/) 包包做，但是 [Bluesmart 的](http://www.bluesmart.com/) 电池内置。)

智能包电池还没有引起任何火灾，但是，考虑到货舱发生火灾可能造成的损失，美国联邦航空局采取了安全胜于遗憾的方法，将电池放在机舱中，这样任何问题都可以快速处理。
