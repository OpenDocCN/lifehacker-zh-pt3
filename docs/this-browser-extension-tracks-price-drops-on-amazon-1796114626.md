# 这个浏览器扩展跟踪亚马逊上的价格下降

> 原文：<https://lifehacker.com/this-browser-extension-tracks-price-drops-on-amazon-1796114626>

Chrome:我讨厌优惠券，但我喜欢省钱，这就是为什么我是 [Honey](https://www.joinhoney.com) 的粉丝，这是一个在结账时自动找到并应用优惠券代码 的 [浏览器扩展。更好的是，他们刚刚发布了一个更新，让你可以跟踪亚马逊上的价格下跌。](http://lifehacker.com/honey-automatically-searches-for-and-applies-coupon-cod-5978700#_ga=2.159968765.1367416964.1497481887-1040816495.1497481887)

Watch

目前，该更新仅适用于台式机和 Chrome extension 9.9.4 及以上版本。(扩展本身可以与 Safari、Firefox 和 Opera 一起工作)不过，希望他们很快会向所有人推出更新，因为它非常棒。在任何亚马逊产品页面上，都会出现一个小蜂蜜图标，询问你是否要将该商品添加到你的下拉列表中(见上图)。只需点击该按钮，Honey 就会在一段时间内为您跟踪该商品，并在价格下降时立即通知您，以便您可以在价格较低时购买。

默认的跟踪期是 30 天，但是如果你想继续观察，他们会提前给你发邮件。你必须注册一个蜂蜜账户，然后如果你想访问你的 DropList，你可以前往 [到他们的网站](https://www.joinhoney.com/insider/stores) ，登录，然后导航到 DropList。它的工作方式与其他价格跟踪工具相同(如 [骆驼骆驼骆驼](https://camelcamelcamel.com/) )，但我喜欢蜂蜜现在有双重功能:我用它来自动优惠券，所以这只是确保我获得最佳交易的又一个功能。

并不是每个人都有同样的蜂蜜运气——你们中的一些人试过它 却报告说它从来没有为你找到优惠券——但是我用它节省了很多钱。它最适用于大型零售商，这些零售商可能首先会有优惠券代码。如果你已经放弃了它，这个更新可能是再试一次的好理由(但是同样， [其他价格跟踪工具](https://lifehacker.com/five-best-price-tracking-tools-1692745053) 也可以完成这项工作)。

如果你已经使用了这个扩展，下次你浏览亚马逊的时候你会看到这个新功能。如果你不是 Honey 用户，你可以从下面的链接注册，然后为你的浏览器下载扩展。该应用可在 Chrome、Safari、Firefox 和 Opera 的桌面版本上使用。
