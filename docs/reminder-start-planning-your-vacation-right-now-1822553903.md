# 提醒:现在就开始计划你的假期吧

> 原文：<https://lifehacker.com/reminder-start-planning-your-vacation-right-now-1822553903>

今天是 [全国假日计划](https://nationaldaycalendar.com/2018/01/29/january-30-2018-national-plan-for-vacation-day-national-croissant-day/) ，所以拿出你的日历，开始翻阅那些旅游指南吧。你不会想让那些宝贵的假期白白浪费掉，只是因为你在组织一些应得的休息之前太忙了。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-lsUraofHSIA&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-lsUraofHSIA&start=0) 

旅游专家萨曼莎·布朗让我想起了今天，这是所有虚假营销噱头假期中我最喜欢的一个。在 [一段来自项目的视频:Time Off YouTube 频道](https://www.youtube.com/watch?v=lsUraofHSIA) (上图)，布朗分享了部分 [项目:Time Off 的惊人数据](https://www.projecttimeoff.com/research/power-vacation-planning) 。例如，54%的美国人在 2016 年没有使用 6.62 亿个假期。大约有 180 万年的时间没有用来啜饮玛格丽塔酒或看望祖母。我从没想到我放学后经常在电视上看的旅游节目主持人会责骂我没有休足够的假期(*和萨曼莎一起去欧洲的护照*有人吗？)，但我们在这里——她是对的。



为什么尽早计划你的假期很重要？首先，你给工作的通知越多，你实际利用分配的时间的压力就越小。研究表明，当休假本身不会给人们带来任何压力时，人们会发现他们的假期更愉快，更有活力。从长远来看，早做计划也能为你省下一大笔钱。在酒店、活动场所和航班之间，你越早把事情安排好，对你的钱包越有利。Airbnb 甚至正在大踏步地吸引那些不情愿的规划者。最近，他们已经开始 [让客人在预订](https://press.atairbnb.com/airbnb-launches-pay-less-up-front-a-new-flexible-payment-option-for-travelers/) 的时候可以少打预防针，让服务感觉更像酒店。如果客人的计划在最后一刻落空，该网站可以让客人提前预订房间，而不会有什么麻烦，并有助于防止主人失去“对现金流敏感的客人”的提前预订

所以，现在是时候了。 [选择一个目的地](https://lifehacker.com/save-tons-of-money-on-travel-costs-with-inverse-vacatio-1790999105) (可以是家) [为你的休假设定一个目标](https://lifehacker.com/the-case-for-making-a-goal-for-your-next-vacation-1710251652) (可以只是放松)，给自己充足的时间让你的休假沉淀下来( [八天左右应该就可以了](https://lifehacker.com/why-relaxing-vacations-should-be-at-least-eight-days-lo-1803816878) )，给自己额外的一天缓冲，以防万一 (最坏的情况是你有额外的一天时间放松，为你的重返工作做准备)。
