# 准备旅行时，让一个人负责每项任务

> 原文：<https://lifehacker.com/put-one-person-in-charge-of-each-task-when-getting-read-1825000215>

正如 Palazzo 先生 10 年级历史课上的那个失败的小组项目告诉我的那样，有时我们在团队中工作得不是最好的。除了课堂项目，这也是为家庭旅行做准备。通常在飞机起飞前的一两个晚上，我和丈夫会决定是“打包时间”，同时开始为我们自己和女儿把东西放进行李箱。然后，就像钟表一样，我们会意识到我们忘记了一些非常重要的东西，比如牙膏或儿童内衣。

下一次，我将使用我的小游牧民族 的旅行规则:父母中的一人应该负责每项任务。

> 旅行时不要分担任何一项职责的负担。例如包装。一个人收拾行李，知道所有东西在哪里。两个人打包，没人知道东西在哪里。酒店也一样。一个人策划，安排，预定。你有确认邮件还是我有？

当你被分配到一个特定的任务时，你就拥有了它，因此不太可能把它搞砸。没有误解或混乱。

这里有一些在合伙人之间划分的责任。

*   安排和预订酒店
*   安排航班、租车和其他交通
*   研究并预订餐厅
*   打包衣服和化妆品
*   包装电子产品和充电器
*   收集旅行证件、现金和信用卡
*   准备你的家。(安排宠物和植物护理，让邮局帮你保管邮件，停止送报纸，清空冰箱，倒垃圾。)
*   和你孩子的老师谈谈提前布置家庭作业的事。

如果你的孩子足够大，你也可以给他们布置任务，比如 [用可视化装箱单](https://offspring.lifehacker.com/get-kids-to-pack-their-own-suitcases-with-a-visual-pack-1819064093#_ga=2.202436121.948624074.1522779617-1858220506.1521654430) 给自己的行李箱装箱。然后，祝你旅途愉快！当然，计划一次带着孩子的旅行尤其令人担忧，但是这个建议适用于任何类型的旅行，无论你是两人还是一组。如果出了问题，你知道该怪谁。
