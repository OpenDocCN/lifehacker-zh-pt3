# 键盘快捷键，排序

> 原文:[https://lifehacker.com/keyboard-shortcuts-ranked-1822042994](https://lifehacker.com/keyboard-shortcuts-ranked-1822042994)

用鼠标做东西是 [酷](https://lifehacker.com/use-a-gaming-mouse-and-browse-the-web-like-a-king-1795895457) 。用键盘做事更酷。这些是最重要的键盘快捷键，从最好到最差排列。(除非特别说明，我们已经列出了 Windows 快捷键；Mac 用户用 cmd 代替 ctrl。)除了一个例外，尽管有任何缺陷，下面所有的捷径从根本上来说都是好的。

#### **1、ctrl+Z** 的缩写形式

撤销快捷方式如此重要，以至于你可能在现实生活中尝试过点击它。手机上的“摇动撤销”功能是一个噱头替代品，只有在你不需要的时候才会触发。

#### **2。ctrl+Y**

撤销你的撤销。而且用的是 z 前面的字母 Legendary。

#### **3\. ctrl+X, ctrl+C, ctrl+V**

剪切、复制、粘贴。大约一周一次，我会在手机上点击 ctrl+V，然后叹口气，试图将手指在屏幕上停留足够长的时间，以弹出粘贴菜单，这总是需要两次尝试。Cmd+X 在 Mac 上感觉有点怪异，但我用它来排列这个列表，所以我欠它一些感谢。谢谢 Cmd+X。

#### **4。ctrl+K**

在大多数文本编辑窗口中，这可以让你将选中的文本转换成链接，而不是像穴居人一样将整个 URL 粘贴到文本中。

#### **5。cmd+空格**

在 MacOS 上打开 Spotlight(或您安装来替换它的启动器)。这就像是正常人的终端命令行。真希望我能早点发现。我现在可能已经是另外一个人了。

#### **6。cmd+shift+4**

Windows 和 MacOS 提供了一系列快捷方式来捕捉整个屏幕、部分屏幕或窗口，并将其存储到剪贴板或作为文件。在 Windows 上，你甚至可以按 Windows+H 键来捕捉你的屏幕并弹出一个共享菜单。我们都有自己喜欢的。但我的是 cmd+shift+4，它让 Mac 用户选择特定区域进行截图，然后将其保存为桌面上的文件。组合键有点间隔，但即使是我的小手也能很舒服地够到。

#### 7 .**。cmd+`**

在 Mac 上，这可以在同一个应用程序的窗口之间切换。在 Windows 上，如果你安装一个叫做 [简易窗口切换器](https://neosmart.net/EasySwitch/) 的小后台 app，你也可以用 alt+`做同样的事情。多聪明的想法啊，在 tab 键的正上方加入一个未被充分利用的键。

#### 8\. ctrl+F, ctrl+G

用键盘开始搜索很酷。用键盘在结果之间切换更酷。

#### **9。ctrl+T**

新标签。有道理。很好记。t 代表 tab。在输入一个新的网址之前，你不必把手从键盘上拿开。做得很好。

#### **10\. ctrl+tab**

切换标签。经典。不要浪费你的时间扭曲你的手指到 ctrl+shift+tab 来返回一个标签。呃。不是所有东西都要优化，你这个书呆子。

#### **11。shift+cmd+I**

这是我的 [自定义 Mac 快捷键](https://lifehacker.com/how-to-remap-any-keyboard-shortcut-in-mac-os-x-5720087#_ga=2.92753053.1244687495.1515599157-1297080755.1497980211) ，用于在预览中调整图像大小。它太棒了，以至于出现在默认快捷方式列表中。奇怪！

#### 12\. ctrl+N, ctrl+O, ctrl+shift+N

新文件和打开文件是必不可少的，但很无聊。但是第三个捷径:根据上下文，ctrl+shift+N 创建一个新的文件夹，一个新的智能播放列表，或者其他一些很酷的变体，从技术上来说让你成为一个编码者。

#### 13。Windows 键+D

最小化一切。好的生活建议，好的桌面清理技巧。MAC 几乎可以做到:Cmd+option+H 隐藏除当前窗口之外的所有内容，所以如果你先在桌面上单击，并且没有打开 Finder，你可以隐藏所有窗口。

#### **14。cmd+H**

在 Mac 上，这将隐藏当前窗口。比最小化干净。要恢复窗口，只需按 cmd+tab 键即可。Windows 用户，你得 ctrl+alt+click。

#### 15。alt+箭头键

在浏览历史中后退和前进。本质上并不坏，Chrome 禁用退格键是明智的，这导致我们许多人意外关闭了我们的 Flash 游戏。但这也是鼠标上最好的两个额外按钮，所以它们并不特别。

#### 16.cmd+S

几年前，“保存文件”快捷方式会在列表的顶部。现在一切都自动保存，所以你不必强迫点击这个组合。你知道什么很酷吗？Ctrl+alt+shift+S 将 Photoshop 项目保存为图像。

#### **17。ctrl+shift+T**

重新打开关闭的标签。神奇的功能，可怕的组合键。拯救一个被遗忘的封闭标签让你感觉像夺宝奇兵 [抢回他的帽子](https://www.youtube.com/watch?v=2ui85HwTX1s) 。但是要击键，你要么伸出你的左手，要么把一只手从鼠标上拿开，这样你就可以双手击键了。我知道它是 ctrl+T 快捷方式的一个附件，用于打开一个新标签页。但是这个动作应该有一个专用的 F 键。

#### 18。alt+tab

有思想的人通过你的 Dock 或任务栏切换应用程序的替代方案。但是你想要的 app 总是比你想象的要远一点或者近一点，你不得不再次切换，现在你关注的是过程而不是继续你的工作。也许码头呼吸者是对的。

#### 19.alt+F4

这个组合键对于关闭应用程序来说太神秘了。Mac 用户用 cmd+Q，你知道吗，没问题。我们不会经常意外关闭我们的应用程序。

#### 20\. ctrl+alt+del

首先它意味着重启，然后它意味着登录。现在已经默认禁用，排名前几位的谷歌结果为 [Ctrl+Alt+Del](https://cad-comic.com/) 是 [最差网络漫画](http://nymag.com/selectall/2015/11/longest-running-miscarriage-meme-on-the-web.html) 。

还有其他重要的捷径，你将要告诉我所有的捷径，但是你已经知道它们属于这个无可争议的 20 分制。
