无论你是第一次去大学，还是在轻松的暑假后回到课堂，或者离开学校很久并对学习新东西感兴趣，现在都是时候打开它，通过一些有趣和信息丰富的课程和研讨会来提高你的技能。任何人只要有一点时间和对自我成长的热情，都可以为了自己的个人利益去旁听、阅读和“报名”这些课程。耶鲁大学、麻省理工学院、斯坦福大学、加州大学伯克利分校等学校都提供免费的在线课程，你可以坐在舒适的办公椅、沙发或计算机椅上听课并参与其中。

如果你还记得从 [我们 2015 年夏季学期](https://lifehacker.com/plan-your-free-online-education-at-lifehacker-u-summer-1704020908) 开始，其中一些课程全年都有，但许多课程只在特定的学期或学期才有，因为我们都在帮助你改善生活，所以我们在 Lifehacker 列出了今年夏天的课程清单，这些课程将激励你，挑战你，打开新事物的大门，并给你改善生活的工具。拿起你的笔和纸，确保你的电池充满电——上课了！

### **计算机科学与技术**

*   [**密歇根大学-面向所有人的编程(Python)**](https://www.coursera.org/course/pythonlearn)**-Charles Severance 教授-** 学习编程如今相当流行，但正如我们已经讨论过几次的，有一些聪明的方法可以帮助你建立实际参与现代项目所需的技能，然后...还有其他的学习方法。这门课程让你开始学习 Python，这是我们第一次学习者最喜欢的语言。最重要的是，一旦你熟悉了 Python，你就可以开始实际的基于 web 的项目，并转向其他更高级的语言。甚至这门课的教材都是免费的。
*   [**Udacity -计算机科学简介**](https://www.udacity.com/course/intro-to-computer-science--cs101)**-Dave Evans 教授** -我们强调 Udacity 课程已经有一段时间了，主要是因为如果你走自定进度的路线，只需访问课程材料，它就是免费的，但是 Udacity 已经对他们的课程的小组主导版本收取了很高的费用，但是如果你有兴趣学习将 Python 用于构建工作 web 工具，这是一个很好的入门课程。这是另一门课程，你将从头开始学习 Python，但在课程结束时，你将构建自己的小型社交网络，并学会构建一个迷你“搜索引擎”，配有一个网络爬虫来收集信息以进行索引。
*   [**【uda city】-Swift iOS 应用开发简介**](https://www.udacity.com/course/intro-to-ios-app-development-with-swift--ud585)**-Kunal Chawla 教授** -如果你对应用开发感兴趣并且 iOS 是你的首选平台，这个免费课程是一个很好的开端。你需要一点背景知识来充分利用它，即*一些*编程经验，但你不需要了解 Swift 或熟悉 iOS 应用程序开发来掌握它。但是，你需要一台 Mac 电脑(运行 OS X 10.10)和一台 iOS 设备。除此之外，带上你的热情，你就可以开始了。课程结束时，你将会开发出一个应用程序，它可以录制你和一个朋友的声音，并进行回放，然后将你的声音调整为花栗鼠或达斯·维德的声音。本课程结束时，您将掌握 Swift 的工作知识，足以让您自己的应用程序想法得以实现。
*   [**马里兰大学-为 Android 手持系统编写移动应用程序第一部分**](https://www.coursera.org/course/androidpart1) **和** [**第二部分**](https://www.coursera.org/course/androidpart2)**-Adam Porter 博士教授** -如果你喜欢的移动应用程序平台是 Android，这两门 Android 开发课程可能更适合你。这两个课程刚刚开始，虽然我们不建议你同时尝试这两个课程，但是如果你已经完成了第一部分或者你不是初学者，那么第二部分可能更适合你。在第一门课程结束时，你将掌握 Android Studio 的更多工作知识，如何为不同的屏幕尺寸和系统配置构建应用程序，以及如何根据谷歌的指导原则设计用户界面。第二部分深入探讨推送通知、通过设备网络获取数据、内置触摸和图形以及在应用中访问系统传感器等主题。这两个课程都假设你对 Java 有所了解，所以在注册之前要记住这一点。
*   [**代码学校- JavaScript 之旅第一部分**](https://www.codeschool.com/courses/javascript-road-trip-part-1)**-Jason mill house 教授** -如果你想为 web 构建，JavaScript 可能是你能学会的(如果不是最好的)语言之一。无论是 webapps、网站和工具、插件和扩展，还是任何其他支持 web 的东西，它都是您可以学习的工具，对于希望深入学习的初学者来说，这门课程是一个很好的速成课程。您将学习 JavaScript 语法的基础知识，如何使用值、文件和变量，并学习将新学到的技能融入到其他 HTML 代码中，使 web 按照您希望的方式运行。它很短，但肯定很有趣，是额外学习的一个很好的起点。
*   [**微软——开发国际软件**](https://www.edx.org/course/developing-international-software-microsoft-dis50-1x)**——比约恩·雷蒂格教授、丹尼尔·戈尔德施米特教授和纳丁·卡诺教授-** 一项经常被忽视但至关重要的技能是能够开发和设计供全球人民使用的应用程序，不仅要考虑不同的语言，还要考虑全球软件中常见的各种计算机类型、输入类型和界面。微软的这门课程旨在解决开发人员在尝试将其平台带到国外时将面临的一些常见问题，不同国家的受众对软件的期望，以及在构建平台时如何考虑不同的语言、文化等。
*   [**代码学校——试试 Git**](https://www.codeschool.com/courses/try-git)**——Gregg pol lack 和 Olivier Lacan 教授——**当你在编程时，尤其是和其他人一起开发软件时，版本控制是必不可少的。能够退出更改，以有序的方式提交对代码的更改，查看您在何时更改了什么，跟踪更改(是的，错误和失误)，这些对于开发过程都是至关重要的，现在 Git 和 GitHub 是 web 上最流行和最常用的管理软件库和开发的工具。本课程将向您展示它们是如何工作的，您如何注册并构建自己的回购协议，提交自己的软件，与他人共享并分享他人的项目，等等。
*   [**SQL 世界的深度在这个初学者课程中等待着您，它将带您从完全没有 SQL 经验到精通用于管理、构建、优化和维护 SQL 数据库的语言、工具和技术。当然，你不会是开箱即用的 DBA，但你会走得很好——如果你在某个地方工作，你肯定有资格提供帮助，你可能会得到一些实践培训。巴基的例子和教学风格也非常到位，你可以自己浏览或跟随课程。**](https://www.udemy.com/draft/12802/)
*   [**斯威本科技大学-游戏开发概念**](https://www.open2study.com/courses/concepts-in-game-development) **-教授博士克林顿伍德沃德** -学习编写和构建自己的游戏是令人兴奋的，没有时间比现在更让游戏行业听到自己的声音和做自己的事情了。本课程将帮助你打下基础，并将涵盖游戏设计的基础等概念，理解你必须从小处着手，从非常小的部分和步骤开始，当然，也包括运动、交互和人工智能等复杂的东西。准备好重新设定你的期望，但是当课程结束时，你已经准备好以自己的方式尝试游戏开发了。

### **财经**

*   [**开放大学——管理我的钱**](https://www.futurelearn.com/courses/managing-my-money)**——马丁·厄普顿教授**——非常受欢迎的“管理我的钱”课程将在本学期回归，因为它是网上最棒的个人理财课程之一(令人惊讶地少)，从头到尾为你提供各种金融主题的坚实基础。你将学习如何编制预算，并获得一些管理财务的实用建议。你还将学习如何为未来储蓄，各种类型的保险，以及什么类型的保险是你真正需要的，等等。这门课程侧重于英国，但你将获得的信息适用于任何地方。
*   [**【Udemy】-股市初学者指南**](https://www.udemy.com/the-beginners-guide-to-the-stock-market/) **-约翰·杜卡斯教授** -股市往往能看起来像个神秘的东西；一只反复无常的野兽，有时赚钱，有时完全因为一时兴起而亏损，没有任何解释。嗯，这个课程不会改变市场的行为，但它会让你对它如何运作，什么影响它，以及为什么它会以这样的方式运行有一些了解。你将获得你需要的工具来解读市场动向，用一双知情的眼睛——而不仅仅是惊恐的眼睛——阅读金融新闻。
*   [**纽约金融学院——了解美联储**](https://www.edx.org/course/understanding-federal-reserve-new-york-institute-finance-yca2015-1-2x)**——威廉·阿迪斯教授**——下一个学期回来就是这个速成班进入“美联储”，或者美联储。我们已经讨论过美联储是一个拥有大量职责的复杂组织，但是许多人在形成自己的观点之前，并不知道它是如何运作的，人们是如何被任命的，以及它的决策的广泛影响。本课程不仅介绍了美联储本身，还介绍了他们的决策如何影响国内外经济，以及为什么他们的选择如此重要。该课程考察了联邦政府和美联储之间的关系，以及他们的决定最终如何影响一切，从你获得信用卡的容易程度到你为未来储蓄的容易程度。
*   [**Udemy and DeVry 大学——财务分析:成功的秘诀**](https://www.udemy.com/financial-analysis-a-recipe-for-success/)**——Ellen Jakovich**教授——这门课程可能听起来令人生畏，但不要让它吓跑你——财务分析的艺术是你资金管理工具箱中的一个重要工具。这不仅仅是制定预算和跟踪你的财务状况，而是确保你以正确的利率储蓄正确的金额以达到你的财务目标的能力，如何为大型家庭装修或项目制定多年预算，等等。你将获得在工作场所绝对有用的技能，但你的活动将围绕日常生活中的个人和现实世界项目，所以你可以看到它们的适用性。

### **科学与医学**

*   [**罗格斯大学——分析宇宙**](https://www.coursera.org/course/analyze)**——教授特里·A·马蒂尔斯基博士**——在这门课程中探索宇宙的一些奥秘，使用公开的 NASA 图像和数据来检查和探索宇宙中一些最令人惊叹的自然和物理现象。本课程将两个大的主题放在一起，首先，你将开始学习你要回顾的图像和数据是如何获得的，它们意味着什么，以及科学家是如何解释它们的，然后你会看到你自己的信息，以一种受教育的方式学习、解释和跟随。你可以“自己做研究”，用自己的眼睛和技能探索宇宙的奥秘。
*   [**东京大学——从宇宙大爆炸到暗能量**](https://www.coursera.org/course/bigbang)**——村山仁教授**——137 亿年是一段很长很长的时间，但在这段时间里发生了很多事情，本课程将带你穿越从那时到现在的宇宙历史。当然，很多——我们指的是很多— 发生在大爆炸后的第一个万亿分之一秒，比如希格斯玻色子的出现，以及组成我们所知宇宙的其他基本粒子。在本课程中，你将使用最新的研究和信息来研究宇宙开始后的那些关键时刻，物质和能量之间的平衡是如何确定的，物质和反物质之间的平衡是如何确定的，星系、恒星和行星是如何形成的，等等，一直到今天。
*   [**阿尔伯塔大学-恐龙 101:恐龙古生物学**](https://www.coursera.org/course/dino101) **-安吉里卡·托里塞斯教授** -恐龙是令人敬畏的，但很可能你在年轻时就已经了解了许多关于它们的知识，这些年来已经发生了变化，或者你可能没有意识到的新发展。如果你是一个恐龙迷，或者你一直想成为一名古生物学家，这个免费课程刚刚开始，它是为你准备的。你将了解生活在中新生代的各种恐龙，它们是如何生活的，它们吃什么，它们如何适应环境(和彼此)，以及它们生活的地球。你还将学习我们是如何了解它们的，从骨骼如何变成化石，以及为现代科学家了解恐龙如何出生、生活和死亡留下了什么线索。

*   [**塔斯马尼亚大学——海洋与南极科学**](https://www.open2study.com/courses/marine-and-antarctic-science)**——克雷格·约翰逊教授和杰斯·墨尔本博士——托马斯**——也许是地球上最寒冷、最险恶的环境，也是了解地球所有历史的最神奇的地方之一。在这门课程中，你将了解世界各地海洋中的海底生物，水下生物的多样性，甚至地球上最寒冷地区的海洋科学是如何进行的。您将了解海洋学家做什么和他们如何工作，海洋和全球洋流如何在管理地球气候中发挥重要作用，您将了解南极科学是如何进行的，以及生态系统如何在如此寒冷的地方不仅能够生存，而且能够茁壮成长。
*   [**东京工业大学-地球深部科学介绍-第一部分**](https://www.edx.org/course/introduction-deep-earth-science-part-1-tokyotechx-geos101x#!) **-广濑圭教授** -事实上，我们所知道的关于地球和我们环境的一切都存在于地壳下翻腾的、炽热的、磁化的、高度活跃的岩石深处的一层薄薄的岩石上。它在做什么，行星是如何以这种方式形成的，地壳和我们所知道的关于地球的一切——山脉、河流、海洋、大陆和岛屿——是如何在其上运动的，以及地球在亿万年间是如何变化的？这门课是一个很好的起点，让我们了解我们脚下这个异常活跃的世界，这个星球最深处到底是由什么组成的，以及地球是如何变成现在这个样子的。
*   [**世界银行集团——降低气温:为什么必须避免 4 摄氏度的世界**](https://www.coursera.org/course/warmerworld)**——巴勃罗·贝尼特兹教授和艾伦·S·米勒教授**——气候变化显然是人类有史以来面临的最大挑战之一，我们是否有能力做些什么，或者只是被迫适应一个变暖的世界，还有待观察。本课程分为两大类——气候倡导者和政策与领导力，前者面向对该主题及其背后的科学感兴趣的人，后者面向工业、政治、商业领域的领导者，以及更多寻求与其他领导者就该主题建立联系的人。在所有情况下，本课程将带您了解全球气候即将发生的变化，随着地球变暖，世界将会变成什么样子，以及对世界每个地区的具体关注，以及这些地区可能发生的非常具体的变化(如果它们还没有发生的话)。)课程将讨论作物产量减少、病媒传播疾病的传播、生物多样性丧失、水资源短缺等主题，以及在变暖的世界中生活所需的缓解和适应措施。
*   [**斯威本科技大学——基础物理**](https://www.open2study.com/courses/basic-physics)**——韦恩·罗兰兹教授博士——**地球和物理科学这么多的基础都是物理学，没有它的基础，可能很难理解其他更复杂的东西。物理学解释了我们周围世界的几乎所有事情，从为什么轮子会滚动，为什么上升的物体一定会下降(大多数情况下)，我们如何把人送上轨道，为什么我们可以在数百万英里之外的眼冒金星，等等。本课程将从较小的规模开始，帮助你从基础物理学的角度看你周围的世界；比如牛顿运动定律以及力和能量之间的关系。不过，在你开始之前，你应该磨练你的数学技能——这门课不会回避真正解释你周围的世界是如何运转的数学。
*   [**弗林德斯大学——作为机器的人体**](https://www.open2study.com/courses/the-human-body-as-a-machine)**——科林·卡拉蒂教授**——人体是一个奇妙的仪器，由数以吨计的独立部分组成，它们在一个高度互联和相互依存的系统中一起运动。本课程讲述了身体如何协调得如此之好，介绍了构成人体的基本构件，讨论了构成人体的复杂系统，并讨论了制造另一个人所需的一切是如何在细胞水平上可用的信息——但如果没有更复杂的系统，几乎不可能重复使用。该课程甚至研究了人体如何通过创造新的人类来取代自身。

*   [**谢菲尔德大学——探索玩耍:玩耍在日常生活中的重要性**](https://www.futurelearn.com/courses/play)**——伊丽莎白·伍德教授和杰基·马什教授**——我们大多数人都明白玩耍对孩子的重要性，作为一种发现他们周围的世界、他们自己的能力和拓展他们思维的方式，但是我们很少考虑玩耍在以后生活中的重要性。本课程涉及所有这些主题，包括游戏在健康生活中的重要性，以及游戏的方面和类型往往是多种多样和必要的，而不仅仅是一种休闲活动或发泄情绪的方式。玩耍不仅是学习的一个基本部分，也是创新和创造力、解决问题和动力的一个基本部分——你将在这门课上学到如何去做。

### **额外学分:如何找到自己的在线课程**

Lifehacker U 的课程丰富而深入，但它可能不会反映你所有的兴趣或专业领域。如果你正在寻找越来越多不同的课程材料，这里有一些资源可以帮助你找到很棒的大学水平的在线课程，你可以在一天的任何时间舒适地坐在办公桌前学习。

*   [**学术地球**](http://academicearth.org/) 策划了一系列令人惊叹的视频研讨会和课程，来自世界上一些最聪明的头脑、创新者和领导者，涉及各种主题，包括科学、数学、政治、公共政策、艺术、历史等。
*   [**TED**](http://www.ted.com/) 演讲以发人深省、有趣、睿智而闻名，在许多情况下，还能鼓舞人心、增长见识。我们以前在 Lifehacker 上做过 TED 演讲，如果你在网上寻找值得一看的研讨会，TED 值得一读。
*   [**edX**](https://www.edx.org/courses) 是加州大学伯克利分校、麻省理工学院、哈佛大学等顶尖大学的免费课程集合。不多，但是提供的都是免费的，对公众开放，而且经常轮换。
*   [**Coursera**](https://www.coursera.org/) 话题从科学技术到社会科学和人文科学，都是免费的。
*   [**Udacity**](http://www.udacity.com/) 提供的课程选择更少，但提供的课程不仅通常是学分课程，而且是由讲师指导并面向特定目标的课程，技术娴熟、才华横溢的讲师会带你完成从创业到机器人汽车编程的一切。
*   [**Udemy**](https://www.udemy.com/courses/) 是一个在线学习市场和资源，包含大量免费和付费课程，主题广泛。有些由各自领域的业余爱好者和专家授课，有些则由高等教育机构提供支持，由大学教授授课。从如何掌握 Microsoft Excel 到如何学习另一种语言，你都可以在这里学到。
*   [](https://www.futurelearn.com/)
*   **[**slide rule**](https://www.mysliderule.com/)维护着一个庞大的课程数据库，可以轻松浏览和搜索这里列出的许多其他机构的课程。他们还提供自己的技能路线和课程集，旨在帮助你学习特定的东西并实现你自己的学习目标，以及他们自己主持的、导师指导的课程。**
*   **[**塞勒基金会**](http://www.saylor.org/) 提供广泛的课程和完整的课程计划，主题从经济学到政治学和职业发展。对机械工程速成班感兴趣？塞勒基金会可以帮助你。**
*   **[**Class Central**](http://www.class-central.com/)以一种易于排序和搜索的格式，聚集了开放大学和网络课程中的一些最佳课程。只需搜索你想学的东西，如果有一门课程即将开课，你就会找到它。**
*   **Education-Portal.com 的 [**有一份**](http://education-portal.com/articles/Universities_with_the_Best_Free_Online_Courses.html) 大学的名单，这些大学向学生和公众提供免费和可获得学分的在线课程。**
*   **[**creative live**](http://www.creativelive.com/)提供了许多商业、摄影和自我提升方面的互动课程，其中许多课程是免费的，可以在一天的任何时间收听。**
*   **[**开放文化的**免费在线课程清单](http://www.openculture.com/freeonlinecourses) 按主题细分，包括 YouTube、iTunes U 上的课程，以及直接从大学或学校网站上下载的课程。**
*   **[**开放课件联盟**](http://www.ocwconsortium.org/) 是一个大学和学院的集合，他们都同意使用一个类似的平台来提供研讨会和完整的课程——包括笔记、备忘录、考试和其他在网上免费提供的文档。他们还维护着一份世界各地成员学校的名单，所以你可以访问世界上任何地方的大学，参加他们提供的在线课程。**
*   **[**可汗学院**](http://www.khanacademy.org/) 提供免费的基于 YouTube 的视频课程，内容涉及数学、科学、技术、人文、备考和学习技巧。如果你想增加你的教育或者只是在业余时间参加几个视频课程，这是一个很好的起点，有很多有趣的话题。**
*   **[**Reddit 大学**](http://www.ureddit.com/) 是由拥有专业知识的 Reddit 用户众筹的一组课程和研讨会。主题从计算机科学和编程到古生物学、叙事诗和拉丁语。对授课感兴趣的个人会定期在 Reddit 大学的子主题 上发帖，以评估对未来课程的兴趣，并宣布新模块何时可用。**
*   **[**life hacker 夜校**](http://lifehacker.com/night-school) 是我们自己的一套教程和课程，帮助你解决深奥复杂的问题，比如 [成为更好的摄影师](http://lifehacker.com/basics-of-photography-the-complete-guide-5815742?tag=nightschool) 、 [建造自己的电脑](http://lifehacker.com/how-to-build-a-computer-the-complete-guide-5828747?tag=nightschool) ，或者 [了解你的网络](http://lifehacker.com/know-your-network-the-complete-guide-5833254?tag=nightschool) 等等。**
