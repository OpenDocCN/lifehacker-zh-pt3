# 氢弹和一般的原子弹有什么区别？

> 原文：<https://lifehacker.com/whats-the-difference-between-a-hydrogen-bomb-and-a-typi-1800008880>

周日，朝鲜引爆了他们声称的氢弹 和武器试验，引起了美国及其盟国的严重关注。但是是什么让氢弹比普通的原子弹更可怕呢？

其一，典型的原子弹利用 [核裂变](https://en.wikipedia.org/wiki/Nuclear_fission) ，或者分裂不稳定的 [铀](https://en.wikipedia.org/wiki/Uranium) 或 [钚](https://en.wikipedia.org/wiki/Plutonium) 原子，来增强炸弹的爆炸威力。当原子分裂时，它们的亚原子中子释放出来，分裂出更多的原子，释放出毁灭性的能量。这些类型的原子弹是在第二次世界大战中对日本使用的，专家认为朝鲜到目前为止一直在进行试验。但是在周日的测试中感觉到的震动可能属于氢弹爆炸，尽管是一次小爆炸——尽管专家们仍然持怀疑态度。

氢弹，或者说 [热核弹](https://en.wikipedia.org/wiki/Thermonuclear_weapon) ，实际上比你那些普通的原子弹要恐怖很多。它们内部有热核燃料，可以在第一阶段点燃，仍然由核裂变提供动力。但是“氢弹”还有另一个锦囊妙计。他们利用*第二*阶段的反应，利用 [核聚变](https://en.wikipedia.org/wiki/Nuclear_fusion) (为太阳提供能量的力量)，基本上放大了典型原子爆炸的破坏力。随着 [氘](https://en.wikipedia.org/wiki/Deuterium) 和 [氚](https://en.wikipedia.org/wiki/Tritium) (氢的种类)发生聚变，原子结合在一起，当它们接触到炸弹的铀层时，就会释放出中子，在连锁反应中产生更大的破坏力。如果你认为原子弹是坏的，这些东西真的很讨厌。

给你一些视角，1954 年美国在 [比基尼环礁](https://en.wikipedia.org/wiki/Bikini_Atoll) 试验的第一批高当量热核弹之一，被称为“ [城堡布拉沃](https://en.wikipedia.org/wiki/Castle_Bravo) ，“比标准原子弹威力大 1000 多倍” [小男孩](https://en.wikipedia.org/wiki/Little_Boy) ，“1945 年由 [*埃诺拉盖伊*](https://en.wikipedia.org/wiki/Enola_Gay) 投在广岛。那颗炸弹几乎瞬间杀死了 66，000 人，并使至少 69，000 人受伤和生病，所以想象一下一颗热核弹能做什么。事实上，布拉沃城堡的爆炸是如此强烈，当它爆炸时，它的设计者都感到惊讶。他们误算了爆炸的当量，相差近 1000 万吨 ，并导致该地区严重的辐射污染和当地岛民的放射病。另一种解释破坏程度差异的方式: [一颗原子弹可以杀死半个曼哈顿；一颗氢弹可以蒸发整个纽约市。](http://www.aljazeera.com/news/2017/09/hydrogen-bomb-atomic-bomb-difference-170903104649473.html)

不过，专家们对朝鲜有能力制造并引爆氢弹表示怀疑。这甚至不是他们第一次声称做这样的事情。2016 年 1 月 [他们还声称引爆了氢弹](https://www.nytimes.com/2016/01/06/world/asia/north-korea-hydrogen-bomb-test.html) ，但当时专家也持怀疑态度。氢弹试验的想法令人担忧，但是 [朝鲜也有可能试验了所谓的“助推原子弹”](https://www.sciencealert.com/what-are-the-actual-differences-between-a-hydrogen-and-an-atomic-bomb) 这些助推炸弹只是 [给炸弹的原子核](https://en.wikipedia.org/wiki/Boosted_fission_weapon) 增加了一点热核气体，增加了爆炸当量，但远不及氢弹。它们的爆炸力只有小男孩的三倍。目前，我们只能等着看美国侦察部队在未来几天能告诉我们什么。
