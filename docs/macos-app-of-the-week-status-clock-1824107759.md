### 两个钟比一个钟好

我们在 macOS 系列中推荐的第一个应用程序很简单，但如果你是一个旅行者或总是在不同时区之间转换的人，它会非常有用。也许你办公室的总部在东海岸，但是你在西海岸，你的同事总是把时间比你提前三个小时。

默认情况下，macOS 不允许你在菜单栏里放多个时钟——这很傻，我们知道。当然，你可以在你的仪表盘上放置多个时钟控件，但这几乎没有扫视屏幕顶部那么有效。

为此，我们求助于免费应用程序 [**状态时钟**](https://itunes.apple.com/us/app/status-clock/id552792489?mt=12) 。安装它，在 macOS 菜单栏中弹出一个时钟，你可以重新设置为任何时间。(您可以通过按住 Command 并拖动它来在菜单栏中移动它。)

我们对状态时钟唯一的一点小烦恼是，没有办法让它看起来和你的 macOS 时钟一样。在你的实际时间之前，总会有一个带数字标识的大“GMT ”,看起来有点蹩脚。(你必须记住时间是代表今天、明天还是其他，这取决于你所指的时区。)不过，状态时钟是一种免费、简单的方法，可以将第二个时钟转储到 macOS 中，而且它的简单性非常好。
