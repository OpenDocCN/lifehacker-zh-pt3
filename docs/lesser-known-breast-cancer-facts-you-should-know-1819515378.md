# 乳腺癌“意识”忽略了这些痛苦的现实

### 你不需要像你想象的那么多乳房 x 光片

乳房 x 光检查可以检测出原本会致命的癌症，但是它们也会产生假阳性，导致更多的扫描和活检结果为阴性。它们可以让你获得癌症患者的完整体验，了解那些生长缓慢或良性的癌症，而这些癌症其实并不需要治疗。所以乳房 x 光检查不是“越多越好”的事情。

美国预防服务特别工作组表示，你应该在 50 岁时进行首次乳房 x 光检查(除非你真的希望 T1 开始得更早)，并且应该在 74 岁之前每隔一年进行一次。

其他组织略有不同。美国癌症协会希望人们从 45 岁开始；美国放射学院和美国妇产科医生大会称年龄为 40 岁。对于你是否应该每年、每隔一年或者随着年龄的增长而变化的某种组合，每个组织也有自己的想法。

根据 USPTF，你也可以在 75 岁时停止进行乳房 x 光检查；之后，他们说没有足够的证据支持或反对筛查。其他组织推荐不同的结束日期，但一般来说，如果你可能先死于其他疾病，那么检测乳腺癌是没有意义的。美国放射学院表示，如果你知道你还能活不到 7 年，或者如果你不会对异常发现采取行动，那么是时候停止了。

### 不是每个人都认为他们的癌症是一场“战斗”

没有人想成为癌症的“受害者”，所以把注意力转移到癌症患者身上，让他们成为积极的参与者，这种感觉是积极的。也许他们在“对抗”癌症，或者他们是英雄或骑兵。如果最坏的情况发生，我们说他们“输掉了战斗”但是很多得了癌症的人不喜欢这个比喻。

如果有人在战胜癌症方面很坚强，这意味着去世或病情恶化的人不够坚强或积极。许多癌症患者对他们被认为是某人的英雄的想法感到不舒服。

英国慈善机构乳腺癌护理 [询问了一些患有乳腺癌](https://www.breastcancercare.org.uk/information-support/vita-magazine/battling-brave-or-victim-why-language-cancer-matters) 的人，他们更喜欢哪种语言。这里有一些不依赖于战争隐喻的建议:

> “我倾向于使用‘与癌症共存’和‘从癌症治疗中恢复’这样的术语，”维基说。
> 
> 苏珊说:“我只是说我接受了乳腺癌治疗。治疗现在结束了。我真的不想被贴上疾病的标签。
> 
> 卡罗琳也喜欢把事情简单化。我更喜欢说一些真实的事情，比如“我正在接受癌症治疗”或“我已经缓解了三年，度过了这个里程碑真好。”

### 癌症治疗非常昂贵

即使你很早就发现了你的癌症，并且像你应该做的那样“战斗”，你也会面临高额的医疗费用。一项针对拥有私人保险 的女性的研究发现，45 岁以下的女性在确诊后的一年里要支付 97486 美元的额外医疗费用(包括保险支付的费用)。45 岁至 64 岁的女性更有可能更早患上癌症；它们的平均成本为 75，737 美元。

先前一项针对 2010 年(患者保护与平价医疗法案之前)接受治疗的患者的研究也发现费用是天文数字。保险公司愿意支付的费用和实际治疗费用之间存在巨大差距——I/II 期癌症的平均费用为 15，000 美元，IV 期为 50，000 美元。(今天人们的自付费用应该会更低。例如，ACA 规定保险公司限制其总收益是非法的。)

但即使在今天，癌症患者仍然有大量的费用，包括共付额、免赔额和共同保险。一半的美国人连 100 美元的急诊费用都难以负担，更不用说癌症患者不得不支付的数千美元了。

癌症慈善机构 PinkFund [调查了最近或几年前患乳腺癌](https://www.pinkfund.org/about/the-financial-impact-of-breast-cancer-treatment-2017-survey-results/) 的女性，询问她们哪里来的钱支付账单。这些数字令人恶心:47%的人动用了退休储蓄，26%的人增加了信用卡债务，41%的人为了省钱而放弃了药物或治疗。与此同时，23%的人增加工作时间以获得额外收入，但 36%的人失去了工作或无法工作。

当你看到一个欢快的粉红色广告敦促你去做乳房 x 光检查时，这些费用可能不是你所期望的——但它们对许多乳腺癌患者来说是现实。当我们鼓励人们“意识到”癌症并接受检查时，我们也要记住癌症会带来严重的医疗、情感和经济负担。
