# 克罗恩病和结肠炎患者真的有神奇的浴室卡吗？

> 原文：<https://lifehacker.com/is-there-really-a-magical-bathroom-card-for-people-with-1820519049>

我可能比大多数人更经常谷歌“便便”，所以当我看到推特上到处都是从[【CrohnsandColitis.com】](http://crohnsandcolitis.com)发出的“洗手间请求卡”的定向广告时，我只是有点惊讶。这是世界上每一个卫生间的免费通行证，还是大型制药公司的营销策略？结果是两者都有一点。

Watch

[克罗恩病](https://en.wikipedia.org/wiki/Crohn%27s_disease) 和 [溃疡性结肠炎](https://en.wikipedia.org/wiki/Ulcerative_colitis) 都是可以导致你迫切需要上厕所的情况，并且经常涉及到带血的腹泻。如果有一张神奇的浴室卡，那就太有用了。

所以我点了。再说说大药厂的关系:CrohnsandColitis.com，与 [克罗恩病和结肠炎基金会](http://www.crohnscolitisfoundation.org/) 没有关系，是一个由制药公司 [艾伯维](https://www.abbvie.com/) 赞助和 [拥有](https://whois.icann.org/en/lookup?name=crohnsandcolitis.com) 的网站。该网站邀请你通过填写表格来请求“更多信息和洗手间请求卡”。该表格询问您是否患有克罗恩病或结肠炎，何时被诊断，以及一系列关于您正在服用何种药物以及您可能想要服用何种药物的问题。在表格的最后，您必须同意以下声明:“我理解我提供的信息将仅由艾伯维及其签约的第三方用于向我发送邮件、电子邮件和电话，提供关于我的病情和艾伯维治疗、产品和服务的有用信息，以及用于营销和信息目的。”

我填写了表格(在互联网上，没有人知道你是否真的患有克罗恩病或结肠炎)，并检查了我的电子邮件。有一条信息写着“我需要的信息”，这只是一个链接，是我刚刚访问的网站上的一个公开页面。没有可打印的卡片，尽管表格要求邮寄地址——所以在我们说话的时候可能已经有一张卡片在邮寄中了。

### 浴室卡是真的

我的直觉是，这是一个垃圾营销噱头。你可以得到一张卡片，上面写着你有病！万岁！但实际上这也有一定的道理。

原来 [几个州都有“盟友法”](https://www.healthline.com/health/crohns-disease/restroom-legislation) 规定企业必须向有医疗需求的人开放私人洗手间。包括克罗恩病和结肠炎。有一些警告；例如，如果一个小企业没有足够的员工来看着商店*和*为你打开浴室，他们是免税的。法律的具体细节可能因州而异。

任何地方都不需要艾伯维的卡；通常，只要告诉员工你身体不适，需要去洗手间就足够了。(老实说，基本的人类礼仪应该足够了，但该法律昵称中的“盟友”是一名年轻女子，她在被拒绝进入员工专用浴室后，最终弄脏了自己。)为了让这个请求更谨慎，看起来更正式，你可以选择带一张名片。

健康热线 [报道](https://www.healthline.com/health/crohns-disease/restroom-legislation) 拥有这种法律的州有科罗拉多州、康涅狄格州、特拉华州、伊利诺伊州、肯塔基州、缅因州、马里兰州、马萨诸塞州、密歇根州、明尼苏达州、俄亥俄州、俄勒冈州、田纳西州、德克萨斯州、华盛顿州和威斯康星州。

除了艾伯维，其他提供信用卡的组织包括胃轻瘫非营利组织[Oley](http://oley.org/?page=RestBathroomCards)，这是一家为通过管道或静脉注射进食的人提供服务的非营利组织，以及一家生产栓剂施药器的公司 [Sephure](https://sephure.com/restroom-access-card/?utm_source=hootsuite) 。 [《发炎与失控》的萨拉建议](http://www.inflamed-and-untamed.com/post/112225587319/should-you-carry-a-medical-alert-card) 如果你想得到最官方的证明，证明你患有某种疾病，让你的肠胃科医生给你写张便条，这对其他目的也很方便，比如旅行时携带医疗用品。
