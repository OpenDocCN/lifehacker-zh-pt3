# 使用 Twitter 的新书签功能保存你想稍后阅读的推文

> 原文:[https://life hacker . com/save-tweets-you-want-to-read-la ter-new-bo-1823415060](https://lifehacker.com/save-tweets-you-want-to-read-later-with-twitters-new-bo-1823415060)

我在推特上花了很多时间。至少一天一次，大多数时候，我关注的一个朋友或某人会发一些我想了解但没有时间去了解的信息。

Watch

有时候，这是一篇我想读的很棒的文章，但当我在杂货店排队时却不能读，或者是一笔交易，当我回到办公桌前时，我想仔细看看。我通常的做法是把这些用电子邮件发给自己，这样我以后就可以看到它们了。然而，本周 Twitter 发布了一项新的改变生活(对我来说)的功能，允许用户私下保存称为书签的推文。

这里的想法与任何其他书签功能几乎完全相同。当你看到一条你想稍后查看的推文时，点按底部的“共享”按钮，然后从弹出式菜单中选择“将推文添加到书签”。

目前，该功能仅在 Twitter 的应用程序中可用，老实说，这正是我想要的。

稍后，当你想访问你保存的推文时，你可以通过点击你的个人资料图标，然后在侧边栏上的个人资料、列表和时刻下方出现的“书签”标签来找到它们。

你还必须使用应用程序访问所有保存的推文，在桌面上无法访问，这有点烦人。这仍然比在飞行中把所有的东西都用电子邮件发给你自己，并且希望你的电子邮件不会在你有时间阅读它之前被埋在你的收件箱里要容易得多。
