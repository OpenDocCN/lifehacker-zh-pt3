# 使用这些平台付钱给艺术家(或获得报酬)

> 原文：<https://lifehacker.com/pay-artists-or-get-paid-using-these-platforms-1823013540>

无论你是独立艺术家的粉丝，还是想赚点小钱的创作者，现在都是时候去寻找更多可以帮助你支持你热爱的工作(或者因为做你热爱的工作而得到支持)的网站了。当然，你可以买一件乐队的衬衫，或者从你最喜欢的水彩画家那里买一件艺术品，但是如果你想尽自己的一份力量帮助你最喜欢的饥饿艺术家填满他们的金库，这样他们就可以继续创作更多他们喜欢的东西，你可以使用这些网站来提供更稳定和安全的收入流。毕竟，一点一滴都有帮助。



### **Flattr**

[Flattr](https://flattr.com/) 用略有不同的方法向支持者收钱。基于订阅的捐赠平台在你访问的支持 Flattr 的网站之间划分你选择的每月订阅费用，每月 3 美元起。你在这些 Flattr 支持的网站上花的时间越长，他们从你选择的月费中收到的钱就越多。要启用这一功能，你需要在运行 Chrome 或 Firefox 的电脑上浏览这些网站，并安装 Flattr [浏览器扩展](https://chrome.google.com/webstore/detail/flattr/imgggkfckgfikmcomnfcocnfcjmladcd?hl=en) 。

使用 Flattr 的创作者需要使用他们自己的网站。为了从 Flattr 中获益，他们会 [按照公司的指示](https://flattr.com/faq) 将一行代码粘贴到他们网站的标题页，将他们的 Flattr 账户与网站关联起来。你还可以看到每个网站的月订阅量。作为一名创作者，为了兑现你的好内容，你可以从 Flattr 中提取你的奖金(扣除 10%的处理费)。

### **点滴**

[滴滴](https://d.rip/) 是 Kickstarter 的订阅式服务，类似于 Patreon 向创作者循环捐款的商业模式。你会发现一些相当受欢迎的艺术家，如 [【雷吉·瓦](https://d.rip/reggiewatts)[彼得·伯尔](https://d.rip/peter) 和 [尚泰尔·马丁](https://d.rip/shantell-martin) ，尽管你可能还不会见到你的朋友和他的乐队。目前，滴滴只接受邀请，因此并不是每个有想法的人都能利用众筹服务。该公司确实表示将在 2018 年向公众开放，所以请密切关注。订阅者可以选择创作者定义的捐赠级别，并预计每月支付，以换取创作者的奖励，如额外的歌曲或项目更新。在费用方面，创作者可能会看到 5%的处理费，以及根据他们使用的货币而收取的其他费用。

### **乐队夏令营**

Bandcamp 除了传统的创作者确定的价格之外，你还可以选择付费，从音乐家那里购买专辑或个人歌曲。根据艺术家的选择，您可以下载不同质量水平的音频文件，除了数字文件之外，还可以购买类似唱片的实物。附带的 iOS 和 Android 应用程序允许您下载和播放您购买的音乐，并浏览新曲目。

### **帕特里翁**

类似于网飞的订阅服务， [Patreon](https://www.patreon.com/home) 是一个非常受欢迎的基于捐赠的众筹网站。如果你是一名艺术家，你可以注册创建一个帐户，并填写你的创作者个人资料页面，在那里你可以描述你的工作以及订户成为赞助人后获得的任何好处。你可以选择按月或在你创作内容时收到钱(例如，这意味着每当你发布一首新歌、一件艺术品或一集播客时，用户将被收费)。

艺术家可以与订户分享独家内容，如播客等会员专用内容的私人 RSS 链接，以及独家 Discord 访问(如果可用)。你可以每个月只捐一美元，支持你想支持的艺术家。

为了支持 Patreon 上的创建者，订户可以访问创建者的 Patreon 页面，并选择他们的订阅级别之一。每个月的第一天，或者每当创建者发布新产品作为其订阅类型的一部分时，都会向订阅者收取他们的综合订阅费。
