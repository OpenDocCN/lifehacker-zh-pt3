# 加工食品比以前含有更多的脂肪，但含糖量更低

> 原文：<https://lifehacker.com/processed-food-has-more-fat-than-it-used-to-but-less-s-1821266144>

十一月的政府报告 中有好消息也有坏消息，关于加工食品的营养成分是如何随着时间的推移而变化的。好消息是:新产品通常含有更少的糖、盐和更多的纤维。但脂肪含量的变化抵消了这一点:新的谷物、酸奶和零食往往脂肪含量更高。

Watch

这可能是因为当你从配方中提取糖时，你不得不添加一些东西，华盛顿邮报的报道称，这是 T2 的 T3，T4 的 T5。不过，也有一些警告。首先，该报告涵盖了 2008 年至 2012 年的变化，因此不包括过去几年的新产品。

此外，它没有给出许多细节，说明是什么样的产品变化导致了这些趋势。酸奶中脂肪含量较高并不是因为全脂酸奶变得更加普遍，但该报告没有详细说明，例如，2%酸奶是否更多，无脂酸奶是否更少。

底线是:如果你想关注你的食物中有什么，确保你在看所有的营养事实。不要以为一两个数字就能说明一切。
