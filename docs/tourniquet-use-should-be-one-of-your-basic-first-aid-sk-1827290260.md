# 止血带的使用应该是你的基本急救技能之一

> 原文：<https://lifehacker.com/tourniquet-use-should-be-one-of-your-basic-first-aid-sk-1827290260>

有一些你希望永远不需要使用的技能，但拥有它们可能是生死攸关的事情:欢迎来到出血控制。



BleedingControl.org，一个由美国外科医生学院发起的组织，已经成功地发起了一场全国性的止血日活动；该组织认为，如果普通公民掌握一些在紧急情况下如何止血的基本知识，许多人的生命可以得到挽救。记者兼认证 EMT Tim Mak 总结了这一天中最关键的几点，致力于提高对出血的认识，以教会更多人如何拯救生命。

止血带带来了一些危险:如果系得不恰当，时间太长，患者可能会失去一条肢体。但无法控制的出血是生死攸关的情况，因此活着的好处往往超过失去肢体的风险。像红十字会 这样的急救组织 [现在教导并推荐正确使用。如果你被教导要避免止血带，那么是时候](http://www.redcross.org/news/press-release/Red-Cross-and-Heart-Association-Announce-Updated-Guidelines) [更新你的训练](https://lifehacker.com/all-the-first-aid-stuff-thats-changed-since-you-first-l-1742121480#_ga=2.100434782.1564879289.1531147293-1456718367.1520458611) 。

根据 Mak 的说法，20%死于出血的人可以通过这种护理得到挽救，而出血致死是可预防死亡的首要原因。以下是你需要知道的。

### 要找什么

小伤口流血致死是可能的，但是大出血的人经常这样做，因为他们割破了动脉。如果血从伤口喷出，而且是鲜红色的，你就知道动脉被切断了。此外，如果血液汇集，如果受伤的人失去知觉，或者如果你看到部分或全部截肢。一个动脉被切断的人可以在 2-3 分钟内死亡。

### 如何干预

Mak 提供了一个看似简单的分步指南，但在有人大量出血的情况下，你可能会有点恐慌。简单的规则很有帮助。

首先，确保你没有踏入危险之中。然后，打 911。即使你即将成功止血，你也希望训练有素的医疗专业人员尽快赶到。然后你找到受伤的地方，施加压力来止血。

你手上可能没有止血带，但如果有，在伤口上方的血流和出口伤口之间打结。如果没有，或者如果伤口太大，止血带无法控制，拿一块干净的布或止血纱布(如果有的话)。包扎伤口，用稳定的压力压住伤口。你试着封闭动脉，直到救援到达。Mak 建议人们尽可能施加压力，因为即使伤势严重，动脉在体内也相当深。你需要使劲按才能够到它，然后关掉它。

最后，如果你有意识这样做，标记止血带或压力施加的时间。这对医疗专业人员来说是有用的信息，因为止血带放得太久是有危险的。

这些都是基本的，但是如果你想免费上一堂手把手的课，出血控制 [在全国各地都提供](https://cms.bleedingcontrol.org/class/search) 。正如 MakT5所写的那样，当需要提升时，没有什么比实践训练更好的了。
