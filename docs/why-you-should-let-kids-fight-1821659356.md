# 为什么你应该让孩子打架

> 原文：<https://lifehacker.com/why-you-should-let-kids-fight-1821659356>

孩子们有很多事情要争——最后一个恐龙块，谁更高，谁将成为 Juniper 的本周最佳朋友。在美国，父母往往会恼怒地回应这些童年的争吵，进入裁判模式，强迫孩子 [分享](https://lifehacker.com/what-to-say-to-little-kids-instead-of-share-1819753096) 或 [道歉](https://offspring.lifehacker.com/what-to-say-to-little-kids-instead-of-say-sorry-1819288365) 或在房间的对面休息一下。我们想要和平，我们现在就想要...*请问？*



但还有另一种方法:让孩子们一决雌雄。萨拉·扎斯克在《华尔街日报》的*文章《 [用德国方式养育孩子:让孩子与](https://www.wsj.com/articles/parenting-the-german-way-let-the-children-fight-1514474042?mod=e2fb) 战斗》中写道，当她住在柏林时，她发现那里的父母和老师不会干预每一场争论——相反，他们相信如果给孩子机会，他们可以自己解决大多数问题。她解释道:*

> *当然，小孩子打架是很自然的。但是我们 kita(幼儿园)的德国老师处理这些冲突的方式与美国非常不同。他们不会急于干预，除非一个孩子即将受到伤害。他们没有惩罚，发出警告，在黑板上写下淘气孩子的名字，或者把他们夹在彩虹色的厄运行为表上。*
> 
> *相反，德国教师花时间观察情况。有时他们把孩子拉到一边，和他们单独谈话；有时，他们会通过阅读与公平和善良相关的故事，直接或间接地向整个群体讲述公平和善良。有时他们什么也不做。*

*从那以后，孩子们都神奇地和谐相处了吗？嗯，不。扎斯克写道，当她女儿五岁时，她和她的两个好朋友之间出现了问题，她“被解除了朋友关系，并多次不被邀请参加遥远未来的生日聚会。”(我的美国妈妈——heart 现在想给那些孩子的父母打电话，聊一聊。)但是老师们绝不会因为这种事情责骂孩子。相反，远离冲突的热点，他们会帮助他们通过自己的行为进行思考， [强调对他人的后果](https://offspring.lifehacker.com/when-kids-break-rules-emphasize-the-consequences-for-o-1796500826) ，这已被证明可以激励孩子和成人改变他们的行为。然后他们会退后一步，让孩子们自己解决问题。*

*回报会随后到来。在日本，家长和老师对打架也采取同样宽松的态度，认为这是孩子自然的成年礼。一项的研究比较了美国和日本的四年级和五年级学生，以及他们对打架、打人和相关行为的看法。当被问及为什么他们不应该做这些事情时，92%的美国孩子说不想惹麻烦。外力影响了他们的行动。另一方面，绝大多数日本孩子没有提到惩罚，并解释说他们不应该打架或打人，因为这会伤害别人。他们获得了智慧和成熟，这只能来自生活经验。*

*那一年，扎斯克的女儿在幼儿园学到了一些重要的东西。“到她上小学的时候，她已经被认为是一个和平使者了，”扎斯克解释道，她写了一本新书，名为[*【ach tung Baby:一个美国妈妈，讲述了德国养育自立孩子的艺术*](https://www.amazon.com/Achtung-Baby-American-Self-Reliant-Children/dp/1250160170?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/why-you-should-let-kids-fight-1821659356&asc_source=&tag=kinjalifehackerlink-20) *。直到今天，她很少和一个“刻薄的女孩”发生争执，无论是作为受害者还是她自己**

*相比当场为孩子解决问题，退缩并观察情绪状况可能更困难。但他们需要的是一致的指导，一个探索他们感受的地方，一个善良的榜样。他们可能不需要一个监督每一场比赛的裁判。*
