# 见见我们新的高级技术编辑

> 原文：<https://lifehacker.com/meet-our-new-senior-tech-editor-1823529876>

嗨，生活黑客的读者们。我是大卫·墨菲，Lifehacker 的长期粉丝，现在是它的高级技术编辑。在此之前，我作为 Wirecutter 的一名作家，花了数年时间用 wi-fi 信号照射自己——还包括 wi-fi 扩展器、wi-fi 网状网络、桌面显示器、游戏鼠标和笨重的虚拟现实耳机。你们中的四个读到这篇文章的人可能还记得我，当我还是最大个人电脑公司的助理编辑时，我在一个蹩脚的纸箱里造了一台电脑。啊，回忆。

当我不下载这个，不安装那个，不考虑新的电脑升级，或者不失去*炉石*地下城运行的时候，我喜欢在旧金山湾区唱歌和表演。(我真的应该遵循我作为一名职业乞丐的真正使命。)

我在 Lifehacker 的目标是向你展示让你的 Windows 或 Mac 电脑随心所欲的所有最佳方法，包括必备软件、对你最喜爱的网络浏览器的有用扩展、让你的日常生活更加轻松的小工具，以及如何充分利用你每天已经在使用的网络应用和服务。我还会谈到其他硬件，但如果我只列出技术领域的每个产品类别，我们会在这里呆上一整天。

我有很多想法要写，但我真的很想听听你的意见:你的设备给你带来了问题，而你却没有解决它们的想法吗？你是不是刚刚发现了一个杀手级的新应用或实用程序，让你的工作效率比以前提高了十倍？你正在寻找一种方法来做一些技术性的工作，并且需要一些帮助来开始吗？你的打印机是否告诉你“PC 装载信件？”
