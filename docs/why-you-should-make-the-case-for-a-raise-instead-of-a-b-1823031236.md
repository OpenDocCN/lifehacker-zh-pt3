# 为什么你应该提出加薪而不是奖金的理由

> 原文:[https://life hacker . com/why-you-should-make-the-case for-a-raise-而不是-a-b-1823031236](https://lifehacker.com/why-you-should-make-the-case-for-a-raise-instead-of-a-b-1823031236)

包括苹果 [、美国航空和家得宝](https://www.nytimes.com/interactive/2018/business/dealbook/tax-savings-benefits.html) 在内的公司都因给一些员工发放一次性奖金而成为正面头条，他们将此归功于国会最近通过的减税措施(公司税率从 35%降至 21%)。正如《纽约时报》报道的那样，问题在于奖金取代了实际加薪。

Watch

这种趋势始于 20 世纪 90 年代，但在经济衰退期间真正得到了发展，我们还没有完全从衰退中恢复过来,*时报*报道。

> 例如，在 1991 年，用于受薪员工的临时奖励和奖金的支出，即所谓的可变薪酬，平均占总薪酬预算的 3.1%，而工资增长达 5%。
> 
> 2017 年，一次性付款占这些预算的 12.7%；加薪幅度仅为 2.9%。

他们为什么这么做？因为奖金更便宜。加薪是年金——每年，雇主支付更多，这些增长随着时间的推移而复合。奖金成本更低，更容易发放，而且效果立竿见影。在受到大规模减税的负面影响后，它们创造了企业想要的那种商誉。帕特丽夏·科恩在《纽约时报》上写道:“衰退的后遗症鼓励雇主避免增加固定成本，在人员配备和薪酬方面尽可能灵活。”。“将曾经由内部处理的工作外包以节省资金的趋势符合这个故事情节。”

重要的是，奖金的增加并没有帮助工资停滞。

科恩写道:“经通胀调整后，2016 年全职工作的男性 的收入中值低于 1973 年。”。“从那以后，他们的 [终生收入](http://papers.nber.org/tmp/96300-w23371.pdf)——包括薪水、工资、奖金和行使的股票期权——大多都下降了。”

科恩指出，奖金并不是工资没有增长的唯一原因。有很多因素，包括工会化的衰落、全球化以及 1968 年(经通胀调整的美元)比现在高的最低工资。

一些公司，如& T 公司的 [和沃尔玛](https://www.cheatsheet.com/money-career/these-companies-started-firing-employees-right-after-getting-tax-cuts-from-trump.html/?a=viewall) (拥有山姆会员店)，在解雇其他人的同时，给挑选出来的员工发奖金。

如果你的公司发放奖金，但不加薪，你显然不会拒绝。但是你还可以采取其他步骤:

*   让 [成为你加薪的理由](https://twocents.lifehacker.com/when-asking-for-a-raise-be-prepared-with-a-backup-plan-1722573423)(这只是权宜之计，并不能解决工资停滞的结构性问题，但这是你应该考虑的事情)。
*   加入和/或 [组成你公司的工会](https://www.lexisnexis.com/legalnewsroom/labor-employment/b/labor-employment-top-blogs/archive/2015/09/08/is-it-time-to-start-a-union-at-your-workplace.aspx) ，可以 [为你讨价还价](https://aflcio.org/formaunion) 。“在全职工资和薪水工人中，工会成员在 2017 年的平均周收入为 1041 美元，而非工会成员的平均周收入为 829 美元，”根据劳工统计局的。

因此，虽然奖金没什么错，但如果是非此即彼的情况，加薪对员工来说更好。
