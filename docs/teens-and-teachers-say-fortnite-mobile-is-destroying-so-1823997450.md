# 青少年和老师说堡垒之夜手机正在摧毁一些学校

> 原文：<https://kotaku.com/teens-and-teachers-say-fortnite-mobile-is-destroying-so-1823997450>

几个月来，各地的老师 [都听到学生们不停地谈论*【堡垒之夜】、*](https://kotaku.com/teachers-narc-on-students-sneaking-games-into-class-1822629697) 世界最新的痴迷、。虽然这款游戏的流行并不是什么新鲜事，但它在 iOS 上的发布显然只是让事情变得更加激烈。毕竟，堡垒之夜现在是一个巨大的分散注意力的地方，你可以把它带到任何地方。

Watch

许多学校都有禁止使用手机的规定，但从社交媒体上的帖子来看，没有什么能阻止学生们启动无情的射手。青少年说，每个人都在玩它，所以学校试图禁止它。这是一个足够大的问题，据报道，一些学校有 Wi-Fi 问题，因为青少年试图通过共享网络尽最大努力生存。人们在争夺游戏，或者他们忽略了学业。在一些学校，现在一直都是堡垒之夜:

学生并不是唯一在推特上谈论这一现象的人

 [https://kotaku.com/embed/inset/iframe?id=twitter-976513111153893376&autosize=1](https://kotaku.com/embed/inset/iframe?id=twitter-976513111153893376&autosize=1) 

“学生边玩游戏边走进教室的事件越来越多，”高中 AP 计算机科学教师尼克·古铁雷斯告诉 Kotaku 。“他们在午餐时间或上学前开始比赛，需要完成 10-15 分钟的比赛。我注意到，在我的房间里，学生们边假装工作，边试图玩耍。”

对我们许多人来说，上学期间玩游戏已经是老生常谈了:我记得在小学的时候，我偷偷把一个游戏机藏在桌子底下。许多人也有在计算器上玩游戏的记忆。让堡垒之夜与众不同的是，这是一个丰富而激烈的游戏，每个 T2 人都在玩，因为它是免费的。此外，如果你不擅长*【堡垒之夜】*，你仍然可以看着别人玩，玩得很开心。这是一项非常吸引观众的运动。玩或看它并和你的同龄人一起玩的诱惑比之前的大多数游戏都要大。

根据负责堡垒之夜移动没收箱推特的老师尼克·费希尔的说法，让 T2 福尼特在孩子们中如此流行的部分原因是它的文化与社交媒体联系在一起。玩家觉得必须与其他人谈论*堡垒之夜*，以公开他们的实力。

“当你不得不 Snapchat 你得到的每一个‘dub’(赢)或者 Snapchat 你的朋友输了的事实混合在一起时，它合并了学校里两个最大的分心...我认为 Snapchat 是一个比游戏更大的问题。但是堡垒之夜把他们两个结合成了一个怪物

手机已经成为学生们的必需品，他们可以在指定的时间使用手机，比如午休时间，但是 T2·堡垒之夜·T3 已经占据了学生们的生活，这还是令人惊讶的。“我的学校允许在午休时间打电话，除了社交，他们所做的就是玩《T4》和《堡垒之夜》，”高中老师普里西拉·卡伦说。

*堡垒之夜*并不是目前唯一占据学生头脑的射击游戏，本月早些时候 *PUBG* 也在手机上发布。但从与老师的交谈来看，许多学生似乎更喜欢其中一个。

古铁雷斯说，在他的课堂上，学生们在第一次铃响时就有一个问题需要讨论，以便为 AP 考试做准备。一天，古铁雷斯班上的学生没有钻研教材，而是讨论了一个游戏的细节。

古铁雷斯说:“学生们花了全部时间谈论 PUBG 的自动瞄准，而不是像平常一样的问题。“我开玩笑地问‘几个月后的 AP 考试，哪个更重要？这个，还是 *PUBG* ？'不出所料，坐在后排的一名学生喊道:*fort nine 的*更好。”
