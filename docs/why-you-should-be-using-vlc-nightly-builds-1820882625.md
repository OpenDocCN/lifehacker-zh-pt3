# 为什么你应该使用 VLC 每夜构建

> 原文：<https://lifehacker.com/why-you-should-be-using-vlc-nightly-builds-1820882625>

VLC 是身边最受欢迎的媒体玩家之一。它可以在大多数平台上工作，几乎可以处理任何类型的文件。这几乎是完美的，但是你可以通过使用 [实验性夜间版本](https://nightlies.videolan.org/) 而不是官方版本来从 VLC 获得更多。

Watch

### 什么是 VLC 夜间建筑？

运行 VLC 的志愿者组织 VideoLAN 提供了该软件的测试版本，名为 Nightly Builds。这些是定期生成的，适用于 Windows、macOS、iOS、tvOS、Ubuntu、Linux 和 Android。它们是在将新想法推向 VLC 的主要版本之前，用一个较小的测试小组来尝试新想法的一种方式。

缺点是 VLC 每夜构建可能不稳定或者有时完全不可用。VideoLAN 也不为这些测试版本提供任何支持。所以如果出了问题，是由你来解决还是重装正式版。然而，有一些额外的好处使它值得冒这个险。

### VLC 每夜构建有什么好的？

正如我们所说，VideoLAN 使用这些每夜构建来测试新功能。这意味着您可以在其他人之前试用 VLC 的最新更新。今年早些时候，该集团增加了 [Chromecast 支持](https://www.howtogeek.com/269272/how-to-stream-from-vlc-to-your-chromecast/) ，但这只是一长串新增功能中的最新一个。

VideoLAN 还使用每夜构建来测试新的扩展功能 。这使得向 VLC 添加特殊脚本成为可能，这些脚本可以从 IMDB 或 Last.fm 等网站获取与您正在观看的电影或正在收听的音乐相关的信息。

VLC 的安卓应用程序 的第一个版本是每夜版本。VideoLAN [也测试了每夜构建的蓝光支持](http://www.omgubuntu.co.uk/2011/12/vlc-nightly-builds-bring-tweaked-look-new-features) 。

### 如何开始

要亲自查看 VLC 的测试版本，请进入 VideoLAN 的夜生活页面 ，从选项列表中选择你的操作系统。这将把您带到一个特定于操作系统的页面，上面有一个测试版本列表。找到最近的版本(根据您选择的平台，它可能被标记为“last ”),然后单击它。

从那里你可以下载并安装最新的 VLC 每夜版本，尽管你可能需要改变你的安全设置来运行它，这取决于你的操作系统。一旦你这么做了，你应该可以开始测试最新版本的 VLC 了——假设它足够稳定可以工作。
