# 在微软的新更新之后，您仍然可以安装 MS Paint

> 原文：<https://lifehacker.com/you-can-still-install-ms-paint-after-microsofts-new-upd-1797263801>

周一，我们得知微软在 32 年后终止了标志性的微软 Paint 程序，并在即将到来的 Windows 10 更新中用 Paint 3D 取代它。这个消息对于《画皮女士》的粉丝来说是悲惨的，许多人 [公开哀悼](https://twitter.com/eshaghbeigi/status/889509353446924288?ref_src=twsrc%5Etfw&ref_url=https%3A%2F%2Fwww.washingtonpost.com%2Fnews%2Fthe-switch%2Fwp%2F2017%2F07%2F25%2Fan-internet-revolt-just-saved-microsoft-paint%2F) 的损失。

Watch

作为回应，微软决定保留这款应用程序: [在一篇博文](https://blogs.windows.com/windowsexperience/2017/07/24/ms-paint-stay/#GMDtdrxMqFyviAdh.97) 中写道:“MS Paint 将会留下来，它很快就会有一个新家，在 Windows 商店里，它将免费提供。”。

对于 MS Paint 的狂热爱好者来说，这是一个好消息，但这仍然意味着 MS Paint 将比以前更难获得。一些雇主可能会禁止员工从 Windows 商店下载应用程序，许多人可能不会特意安装该应用程序。

但是对于铁杆粉丝来说，有理由感到高兴: [复杂的 MS 绘画艺术](https://www.reddit.com/r/mspaint/) 和 [迷因](https://www.reddit.com/r/classicrage/top/) 毕竟还没有完全消亡。
