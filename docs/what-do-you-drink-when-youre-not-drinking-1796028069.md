# 你不喝酒的时候喝什么？

> 原文：<https://lifehacker.com/what-do-you-drink-when-youre-not-drinking-1796028069>

大家好，欢迎回到 [*有什么好吃的？*](http://lifehacker.com/tag/whats-cooking) ，每周一次的公开讨论，在这里你可以分享你所有关于食物的想法、建议、食谱和观点。本周我想关注一些可以啜饮的东西，特别是当你戒酒时你会啜饮的东西。

Watch

人们选择不喝酒有各种各样的原因，但这并不意味着他们不应该享受有趣和令人兴奋的饮料。不喝酒的人的选择有时看起来像是事后想起的，即使你只是想减少酒精摄入量，你可能会觉得你错过的不仅仅是酒精。(除了真正的豪饮之外，我经常发现自己怀念在漫长的一天结束时调配鸡尾酒或开一瓶好酒的例行公事。)所以我想知道的是，当你需要 ABV 中缺乏的液体治疗时，你会啜饮什么非酒精饮料。

*   你有最喜欢的牌子或口味的矿泉水吗？我目前非常喜欢橘子汁，但我更喜欢极地矿泉水，尤其是他们的黑樱桃和樱桃酸橙。
*   **你最喜欢的灌水方式有哪些？**黄瓜总是很好吃，但是 [草莓顶部做了一个令人惊讶的美味附加品](http://skillet.lifehacker.com/use-strawberry-tops-to-create-fruity-spa-water-instead-1716083424) 。
*   你最想学的是什么？我喜欢纯苏打水加新鲜柠檬汁和 [宜家的接骨木花糖浆](http://skillet.lifehacker.com/how-to-hack-the-ikea-food-market-1755817567) 。这是甜的，酸的，而且(非常)有点让人想起法国 75。
*   **你有没有遇到过不含酒精的啤酒或葡萄酒其实很值得喝？我没有。**
*   你曾经在非鸡尾酒饮料中添加过传统的鸡尾酒装饰物吗？我喜欢健怡可乐里的柠檬皮，或者姜汁汽水里的 Luxardo 樱桃。(如果我真的想犒劳一下自己，我会去喝一杯姜汁啤酒，而不仅仅是淡啤酒。)
*   什么样的饮料细节对你很重要？ [好冰](http://skillet.lifehacker.com/what-exactly-does-one-do-with-a-500-nugget-ice-machine-1794231937) 和一个好看的杯子可以提升任何饮料，不管是不是酒精饮料。用威士忌酒杯喝果汁特别好玩。

还有，不要低估童年最爱的舒适度。巧克力奶不分年龄，总觉得很特别。当我需要安慰时，我也会去喝韦尔奇的葡萄汁。
