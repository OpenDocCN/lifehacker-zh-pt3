# 你现在能找到的最好的 Youtube 频道

> 原文：<https://lifehacker.com/the-best-youtube-channels-you-can-work-out-to-right-now-1751069875>

天气很冷。 [健身房里挤满了](http://vitals.lifehacker.com/how-to-work-out-when-the-gym-is-crowded-1750419523) 。还不如呆在家里看视频...*锻炼*视频。以下是一些我们最喜欢的在 YouTube 上健身的方法。



在我们的列表中的是充满固体锻炼的频道——要么是完整的会议，要么是你可以在更长的锻炼中构建的大量内容。所有的动作(除了举重)都是在你观看的时候跟着做的。(虽然一些频道充满了快速提示、教学视频、独白、食谱或长期锻炼的预告，但我们在这里跳过了这些。)所以你可以从这个列表中选择一个视频，立即享受锻炼，不需要计划。

### 全方位健身

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-31GQrtI6ivI&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-31GQrtI6ivI&start=0) 

**频道:**[fitness blender](https://www.youtube.com/user/FitnessBlender/)
**特长:**利用体重完成的力量训练或者 [外形怪异的可调哑铃](https://www.fitnessblender.com/articles/best-dumbbells-for-home-workouts-what-type-of-dumbbells-do-fitness-blender-use-) 。
**你还会发现:**一个 [小狗锻炼](https://www.youtube.com/watch?v=QtYRj6OBAw4) 。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-m1_0VgksEDk&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-m1_0VgksEDk&start=0) 

**频道:** [杰西卡史密斯电视](https://www.youtube.com/user/jessicasmithtv/)
**特长:**有氧/力量训练(有无负重)，还有几个特色 [力量走位](https://www.youtube.com/watch?v=azbbr57_g5I) 。哦，还有一些可爱地把她的妈妈作为伴舞演员。
**你还会发现:**流言终结问答环节

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-WQXfS9rPxQs&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-WQXfS9rPxQs&start=0) 

**频道:** [索菲巴](https://www.youtube.com/channel/UCZs7ELqPGaW-KyTiWFd_htw) (“像花样滑冰运动员一样健康”)
**特长:**芭蕾舞训练中不寻常的旋转，因为索菲不是芭蕾舞演员而是花样滑冰运动员。
**你还会发现:**关于 [滑冰更好的提示](https://www.youtube.com/playlist?list=PLYkF1E40kqAziddvCulZV73BfDgHTjsnm) 。

### 举重训练

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-ukSLVLerNwk&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-ukSLVLerNwk&start=0) 

**频道:** [肌肉&力量](https://www.youtube.com/channel/UC5_i5V3xXxdqF5VKVmJWVZQ)
**特长:**健身房健身计划(观看，然后去)的讲解，适合每周例行公事，像他们的 [OT4 系统](https://www.youtube.com/playlist?list=PLp5PgSZbYOgs-L4-1e_mGcjqtT5_48x1q) 和 [快速 21 健身计划](https://www.youtube.com/playlist?list=PLp5PgSZbYOgtPbOKNr7MnSWW66Uiv4YPR) 。
**你还会发现:**食谱和采访

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-KYUuf_Y0Ias&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-KYUuf_Y0Ias&start=0) 

**频道:** [斯科特赫曼健身](https://www.youtube.com/user/ScottHermanFitness)
**专业:**重量训练(还是那句话，看着走而不是跟着走)包括一系列 [针对身体各部位的混搭训练](https://www.youtube.com/playlist?list=PLacPhVACI3MPMbxPAGt4yuF0q_swmElG1) 。
**您还会发现:**提示和视频日志

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-ThnXE3Lo2V4&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-ThnXE3Lo2V4&start=0) 

**频道:**[【Bodybuilding.com】](https://www.youtube.com/channel/UC97k3hlbE-1rVN8y56zyEEA)
**特色:**健身视频，视频前有跑步者的电影独白。
**你还会发现:**食谱、小贴士和锻炼个性的日子，比如 [Calum von Moger 在开始一天的锻炼前参观一个宠物动物园](https://www.youtube.com/watch?v=TTIii8jD8zk) 。

### 快节奏有氧运动

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-Uv8YoPMW5S4&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-Uv8YoPMW5S4&start=0) 

**频道:**[body rock](https://www.youtube.com/user/charliejames1975/featured)
**特长:**高强度间歇，包括一场 [30 天挑战](https://www.youtube.com/playlist?list=PLdLzQAslwOS7dTmPGigl_1Ju6C0I3y-6T) 每天一段视频，最长超过 45 分钟。
**你还会发现:**他们付费锻炼项目的预告；30 秒的练习演示。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-_m2PPZvo4zw&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-_m2PPZvo4zw&start=0) 

**频道:** [适合](https://www.youtube.com/user/BeFit)
**特长:**一切从 [舞蹈套路](https://www.youtube.com/playlist?list=PL1c41tQdiDhN8coW_0tCUmtxi2k8pNuDg) 到 [极限健身](https://www.youtube.com/playlist?list=PL1c41tQdiDhMe4LcQgufgL-YTO94fqi-5) 到 [初级瑜伽](https://www.youtube.com/playlist?list=PL1c41tQdiDhNbrBT0RrcbbmJihHAw5_XW) 。一定要看看 [适合去](https://www.youtube.com/playlist?list=PL1c41tQdiDhNp1NyRYN91ui4OoiVIkfVr) 系列的 [赛道](https://en.wikipedia.org/wiki/Circuit_training) 意味着可以在你的手机上观看，在任何地方完成。你还会发现:一些更短的迷你锻炼和偶尔的“健康黑客”，但仅此而已。BeFiT 是一个巨大的频道，比我们发现的任何其他频道都有更多的完整视频和更多不同类型的锻炼。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-y2gBj9VZBQ0&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-y2gBj9VZBQ0&start=0) 

**频道:** [珍妮福特](https://www.youtube.com/channel/UCmev1eHpKJ7HfQyV4Tm_qBg)
**特长:**老派踏板操
**你还会发现:**上的说明 [如何，确切地说，要踩一踩](https://www.youtube.com/playlist?list=PLKqmyEQUnlMa83HlqiXDncxgU0-66Pjnw) (以及其他几个有用的动作)。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-lQxo1lZ8H1Q&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-lQxo1lZ8H1Q&start=0) 

**频道:** [全球自行车网](https://www.youtube.com/channel/UCuTaETsuCOkJ0H_GAztWt0Q)
**专业:**虚拟动感单车课你可以自己上(在健身房使用自行车，或者在家里使用 [教练](https://en.wikipedia.org/wiki/Bicycle_trainer) (如果你有的话)。大多数只是展示其他骑自行车的人出汗的脸，但有几个播放风景，这样你就可以想象自己骑自行车上风景如画的小山。
**你还会发现:** [如何视频](https://www.youtube.com/playlist?list=PLUdAMlZtaV11U9AtszkMh0bpRqM4dtlDC) 关于骑行的一切
