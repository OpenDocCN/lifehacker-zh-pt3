# 该应用程序允许父母发送孩子们不能忽视的紧急短信

> 原文：<https://lifehacker.com/the-app-allows-parents-to-send-emergency-texts-kids-can-1798072684>

当你给你的孩子一部智能手机时，通常的想法是他或她会更容易联系到。事实上，在很多情况下，这会让你的孩子更容易联系他们的朋友，忽略你。一款名为[reply sap](https://www.facebook.com/replyasap/photos/a.1438399912892923.1073741828.1424627800936801/1438399579559623/?type=3&theater)的新应用为你解决了这个问题，让你可以发送短信，而你的孩子除了回复别无选择。



一旦你将它安装在你孩子的手机上，你作为父母通过该应用发送的消息将导致你孩子的手机上响起警报，并一直响，直到他或她看到该消息。然后警报会停止，但你也会收到通知，让你知道你的孩子看到了你的“立即回家！”消息。这意味着“我的手机静音了”或“我没看到”的借口已经过时了。

不要用这样的东西不停地给孩子发短信。

如果你发送给孩子的每条信息都是通过这个应用程序，那么我认为在你们的关系中有更大的沟通问题，一个讨厌的应用程序不会是解决方案，事实上，它可能会让事情变得更糟。如果你的孩子是青少年，在他们开车时用智能手机指路时给他们发一条刺耳的信息似乎是一种危险。

这是令人讨厌的，有潜在的危险，至少会让一个你已经很难与之交流的孩子异常疏远。另外，让我们面对现实吧，任何聪明的孩子都能够找出如何删除 tihng。

也就是说，我确实认为，当传统的电话和短信不起作用，你真的需要与你的孩子取得联系时，这是一个“最后手段”的好主意。例如，当我的祖母去世时，我错过了父母的几个电话，没有去想它。有一种方法让紧急信息通过，一个字面上尖叫“不，真的，打电话给我”可能是有用的。或者，如果你通情达理，通过传统渠道发送了一些其他信息，这可能是试图联系的最后一根稻草。
