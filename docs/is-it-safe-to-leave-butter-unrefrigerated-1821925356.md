# 不冷冻黄油安全吗？

> 原文：<https://thetakeout.com/is-it-safe-to-leave-butter-unrefrigerated-1821925356>

试图涂抹冷黄油是人生的一大悲剧。

更糟糕的是，等待着美味的新鲜法式长棍面包，或者刚刚烤好的充满峭壁和石笋的英式松饼。刮，刮，刮。感叹。



在这些情况下，室温黄油的吸引力是显而易见的，但问题依然存在:不冷藏的黄油棒安全吗？我们询问了黄油专家。

他们回答的总和是:是的，让黄油不冷藏一小段时间是可以的，但理想情况下不要超过一两天。冷藏保存了黄油的最佳风味，并延长了它的保质期，但室温黄油可能不会有太大的健康风险。

美国美国食品药品监督管理局的新闻官彼得·卡塞尔说，冰箱的温度降低了黄油上腐败微生物的生长速度。较低的温度和较少的光照也会降低氧化酸败的速度，空气中的氧气会在油脂中产生酸败的气味和味道。出于这些原因，FDA 的食物节约指南 建议黄油在室温下只能储存一两天，而冷藏可以储存长达两个月。 [加州乳品公司](http://www.californiadairypressroom.com/products/butter) 也建议限制黄油在室温下的“短期”储存

全国牛奶生产商联合会的发言人克里斯·盖伦说:“你在温暖的温度下放置的时间越长，你的食品安全问题就越多——尽管食品安全问题不多——但质量问题更多。”“在 68 度左右，黄油开始出油，这意味着油从固体中去乳化。对于关心口感或外观的人来说，这不是一个安全问题，而是一个质量问题。”

威斯康星州牛奶营销委员会 同意，对细菌或食品安全的担忧不是你应该冷藏黄油的主要原因。

“黄油的一般成分是一种高脂肪和高盐含量的乳液，这使得它成为一个非常不利于细菌生长的环境，”WMMB 技术服务总监亚当·布洛克说。“将黄油放置几个小时或几天，不会导致细菌滋生，从而引发食品安全问题。”

他补充说，经过巴氏杀菌的盐制黄油比未经巴氏杀菌的盐制黄油保存时间更长，将黄油放在室温下的盘子里会减缓氧化过程。

一句话:将一块黄油在室温下放置几天不会伤害你，但它可能会导致你的黄油更快变质，并可能从厨房的其他地方吸收不想要的气味。最佳黄油做法是在室温下留下你想要涂抹的量，并迅速用完。

话虽如此，但不要对那些坚持用不同的方式对待黄油的人评头论足，因为黄油规则就像黄油本身一样，有点黏糊糊的。

全国牛奶生产商联盟 的克里斯·盖伦说:“在这个问题上没有一个硬性的正确答案，就像生活中的许多其他事情一样。”。“这里有一些灰色地带。”
