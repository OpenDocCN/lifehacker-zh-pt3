# 如何在我的 MacBook、iPad 和 iPhone 上使用相同的密码？

*实现我认为你所指的最简单的方法是确保你将同一个苹果账户与你的每一个设备关联起来。因此，举例来说，在 MacBook 上购买应用程序需要您使用 Apple ID 登录 App Store，并且您应该在为 iPad 或 iPhone 购买应用程序时使用相同的 ID。同样，当您在所有设备上激活 iCloud 时(这应该是您最初设置它们时的提示)，请在每台设备上使用相同的 Apple ID 和密码。*

*不要跳过设置过程，也不要为每台设备创建一个单独的 Apple ID，当然，除非有不同的人将成为该设备的主要用户。(说到这个，苹果让创建家庭账户变得很容易，所有人都可以互相分享内容——并对所有购买收取单一账户的费用。)*

*无论如何，我不想太超前，只要继续前进，对所有的苹果设备使用相同的苹果 ID。当你这样做的时候，无论何时苹果要求你重新认证，你都将使用“仅仅一个密码”。*

*现在，如果你说的“只需一个密码”是指你想用一个密码将*登录到每台设备的*中，你也可以这么做。我使用触控 ID，它允许我用同一个手指登录我拥有的所有东西。就像“一个密码”，对吧？如果你没有 Touch ID、或者不想使用它，你可以一直为每台设备使用相同的密码，尽管你会以便利换取安全。*

*假设你有一个你非常喜欢的 iPhone 和 iPad 的 PIN 码，并且你想让你的 MacBook 也有同样的 PIN 码，登录你的 MacBook，点击左上角的苹果图标。在那里，点击系统首选项，然后用户和组。点击您的用户帐户，点击更改密码，并使您的 MacBook 密码与您在 iOS 设备上使用的 PIN 相同。*

*如果你想更上一层楼，你也可以用你的 iOS 设备密码作为你 Mac 的 [固件密码](https://support.apple.com/en-us/ht204455) 来代替，但那可能有点大材小用了。坚持使用普通的olMAC OS 密码，你就没事了。*

*相反，如果你的 MacBook 有一个你非常喜欢的密码，并且你想在你的 iOS 设备上也有这个密码——而且不是六位数——你也可以在你的 iPhone 或 iPad 上输入字母数字。每次当你想解锁你的设备时，都必须把它打出来，这真的很烦人，但我绝不会批评超强的安全性。*

*在您的 iOS 设备上，轻按“设置”，然后轻按“触控 ID 与密码”。输入您的当前密码，然后向下滚动一点并点击更改密码。再次输入您的密码(oof)。当要求提供新密码时，点击密码选项并选择自定义字母数字代码(如果您想使用字母和数字)或自定义数字代码(如果您只想使用数字)的选项。*

*键入您在 MacBook 上使用的密码，然后轻按“下一步”。再次输入以验证它，点击下一步，从现在开始享受输入比六位数更复杂的密码的简单，直到你厌倦这样做。*

*在这里，我还想用最简短的时间指出， [密码管理应用程序](https://lifehacker.com/the-five-best-password-managers-5529133)—如 [1Password](https://lifehacker.com/1passwords-updated-password-manager-is-a-must-have-for-1826379117) 或[LastPass](https://lifehacker.com/the-beginners-guide-to-setting-up-lastpass-1785424440)—可以很好地将记住一个强密码(解锁应用程序的密码)的便利性与针对不同网站和服务使用多个复杂密码的安全性结合起来。使用你熟悉的单一密码认证进入任何一个应用程序，然后当你需要为一个网站或服务使用它时，你可以复制并粘贴一些非常复杂的密码。你甚至可以让应用程序为你粘贴凭证。*
