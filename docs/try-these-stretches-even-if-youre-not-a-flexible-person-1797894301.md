如果你不够灵活，你可能会觉得没有必要去尝试伸展。但是没有伸展，你怎么能变得更灵活呢？像今天视频中这样的会议会有所帮助。



视频是 20 分钟，但是如果你时间不够，点击右下角的档位，将速度设置为 1.5 或两倍。毕竟，它们是静态拉伸，所以你只需将每个拉伸的持续时间缩短一半。当你的主持人 Nico 正在解释下一个伸展动作时，你可以通过继续每一个伸展动作来争取额外的时间。

伸展运动开始时很容易，然后到达可能会感到有挑战性的程度，但不像你要把肌肉从骨头上扯下来。轻轻拉伸；它仍然有效，而且你会在事后感觉很棒。
