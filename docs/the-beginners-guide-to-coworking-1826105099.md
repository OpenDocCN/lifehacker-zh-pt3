# 协同工作入门指南

> 原文：<https://lifehacker.com/the-beginners-guide-to-coworking-1826105099>

我喜欢共同工作空间的概念。尽管我已经在家工作了很多年，但有时我发现自己错过了一些东西，比如人际交往、美味的小吃、植物以及独特或超级有创意的空间，这些东西让我更容易度过朝九晚五的日子。

问题？共同工作有点像 Airbnb:有很多选择，开始时可能会感觉有点不知所措。你最不想做的事情就是每月花一百(或几百)美元在一个地方买一张办公桌，那里的便利设施和工作条件并不比你家里好多少。

### 寻找合适的合作地点

虽然你可能听说过 [WeWork](https://www.wework.com/) ，这是一家较大的合作公司，因在全国各地拥有各种独特的地点而闻名，但你可能不太熟悉你特定地点(或你要去的任何地方)周围的较小合作公司。在你开始考虑什么样的共同工作空间可能最符合你的需求(或审美偏好)之前，你必须知道那里有什么。

我们之前已经介绍过类似 [同事](https://lifehacker.com/need-an-office-on-the-quick-use-coworker-to-find-a-spa-1823618599#_ga=2.55709803.1119243524.1526323629-396842925.1520800403) 的工具，这些工具可以根据您的特定服务需求和位置偏好，轻松搜索世界各地不同的工作场所。它很好，但它没有捕捉到一切。 [呼吸器](https://lifehacker.com/breather-finds-affordable-spaces-for-events-or-to-work-1789792069#_ga=2.164142079.1119243524.1526323629-396842925.1520800403) 和 [雷格斯](https://www.regus.com) 也是值得一试的不错的工具，但是这些搜索引擎仍然让我觉得我没有看到*一切*，因为你知道，像 WeWork 这样的大公司不会出现在它们上面。

为了确保你对那里的一切有所了解，可以考虑做一个简单的 [【谷歌地图】](https://www.google.com/maps) 或 [Yelp](http://www.yelp.com/) 搜索“共同工作”，并通过评级过滤，以确保你只看到最好的，而不是一些随机的粗略的咖啡店。这是一个寻找下一个工作地点的粗略方法，但是你要确保你没有错过任何明显的亮点。

### 你的新临时工作场所需要具备什么？

虽然协同工作的搜索引擎和列表网站可以帮助你筛选出只有你想要的便利设施的地方，但即使决定什么应该出现在你的必备品清单上也是令人生畏的。如果你以前没有在一个共同工作的空间呆过，你怎么知道你需要什么？

值得考虑的一些属性和便利设施包括:

*   **高速互联网:**虽然不是每个共同工作的空间都可以有光纤线路，但你肯定希望能够处理从电子邮件到团队演示视频上传的所有事情，而不会感觉到你所在空间的互联网服务不稳定。你不仅仅需要一个大管子:还要考虑空间的无线能力。它有 [高速无线 ac](https://lifehacker.com/what-is-802-11ac-and-will-it-make-my-wi-fi-faster-5988340) 和大量的接入点吗？以太网连接？
*   24/7 大楼访问:虽然下班后还得工作很糟糕，但有时你不得不在周末晚上处理编辑工作，并与世界另一端的团队合作。一旦时钟指向五点，如果你不能呆在你美丽(昂贵)的共同工作空间里，那将是令人难过的——即使只有你一个人在开夜车。
*   当你每月花 200 美元买一张办公桌时，你最好能得到一两个免费的苹果，该死的。如果你喜欢，在炎热的夏天工作的时候，无限制地喝一杯小啤酒会更提神，但没必要太贪心。
*   **地点:**坐 [公交](https://lifehacker.com/how-to-ride-public-transportation-without-losing-your-m-5942185) 到你那里容易吗？自行车？走路？虽然你可能可以整天大吃免费的苹果和薯条，但共同工作的地方离其他美味食物的地方只有几步之遥吗？这些是你不介意一直去吃的食物吗？你能去你将要工作的任何地方吗？如果你想开车去上班，有足够的停车位吗？
*   **隐私:**如果你必须在白板上画出一些设计草图，接一个电话，或者与参观你的工作区(或在你旁边工作)的其他人进行一个快速的协作会议，有没有空间可以让你这样做？它们容易预订吗？
*   社区和活动:你不一定要和同事成为最好的朋友，但共同工作空间是一个很好的地方，可以做一些额外的工作，建立可以带来更好机会的联系，做[我在 35 岁左右还没有掌握](https://lifehacker.com/how-to-network-even-if-you-hate-networking-246199) 。此外，看看一个共同工作的空间是否提供任何形式的社区活动，无论是小型研讨会、下班后聚会、特邀演讲者——诸如此类。充实自己；毕竟，你要为此付出代价。
*   **费用:**假设你想运送或接收一个包裹，因为在你的共同工作空间比在你简陋的公寓里更容易(或更安全)完成这件事。那要额外付费吗？如果你需要一些存储空间，一个放自行车的地方，或者想要一杯来自共同工作空间的别致的小咖啡馆的咖啡，你会在已经昂贵的月费上增加费用吗？如果您需要为会议提供餐饮服务，该怎么办？演示文稿中稍微复杂一些的音频/视频设置？
*   你在哪里工作:一个特定的联合办公空间提供什么样的工作地点？你是否被困在一张长桌前，与另外八个人分享一个电涌保护器的电量？你能有自己的半私人专用办公桌吗？私人办公室？你自己的楼层？
*   **厨房:**如果你热衷于自己做午餐，拥有一个厨房是必不可少的——尽管你可能要通过杀死那个一直喝你标有清晰标签的橙汁 的 [来确立你的统治地位。如果你讨厌别人臭烘烘的午餐味道，也许找一个没有厨房的共同工作空间对你更好。](https://lifehacker.com/why-coworkers-steal-each-others-stuff-and-what-you-can-1734955344)

你如何确保你将要支付一大笔钱的共同工作场所是绝对完美的？在你花钱购买更贵的通行证或每月会员资格之前，寻找能给你一些尝试空间的合作地点。也许你可以免费使用这个空间一天，甚至可以买打折的一日通票。一定要试着全面参观这个地方，最好是在一个正常的工作日(这样你就能感受到在那里工作的真实情况)。拿出笔记本电脑 [测试连接](https://lifehacker.com/top-10-ways-to-deal-with-a-slow-internet-connection-514138634) 。参加活动或社区聚会。吃一个苹果。

### 如何度过愉快的第一天

一旦你决定了共同工作的地点，并准备好开始工作，你仍然需要做一些计划。既然你要在一个不是你家的共享空间工作，你就要确保你一整天都准备好了要处理的所有事情:

*   **买一个背包:**好吧，你没有*买*背包，但要确保你有一些好东西，可以用来将你的笔记本电脑和其他设备运送到新的工作地点。更重要的是，确保你的包里有 [你需要的所有必需品](https://lifehacker.com/the-neat-and-nerdy-backpack-1797462952) :电源适配器(如果适用，可以使用你的超长充电电源线)；您可能需要将外部存储器、显示器或以太网电缆连接到笔记本电脑的任何 [加密狗](https://lifehacker.com/rip-usb-these-are-ports-and-connectors-to-care-about-i-1796823560)；你最喜欢的 [水瓶](https://lifehacker.com/the-truth-about-bottled-water-expiration-dates-and-how-1720084038)；你宠物的照片(除非你的共同工作空间 [允许它们挂](https://www.youtube.com/watch?v=wlyH3M7zpPY))；为您的设备充电；你最喜欢的一副 [降噪耳机](https://gizmodo.com/sorry-bose-the-best-wireless-noise-cancelling-headphon-1819923798)；等等。
*   **考虑增强你的 wifi:** 如果你的笔记本电脑的无线功能不如你的工作空间提供的强大，你可能需要投资一个 [外部 wifi 适配器](https://lifehacker.com/the-10-best-ways-to-boost-your-home-wifi-5931743) 。如果你的新工作场所有高速 AC1750 接入点(3x3 wireless-ac)，但你的笔记本电脑只支持 N400 (2x2 wireless-n)，你可能需要选择一个更强的外部适配器，因为它们有时可能会很烦人，所以你不会永远等待你的工作文件传输(或你休息时的 4K YouTube 流)。
*   使用 VPN: 如果你的公司没有你可以使用的 VPN，或者你在为你的老朋友工作，考虑一下 [购买一个](https://lifehacker.com/the-beginners-guide-to-vpns-1819912770) 。虽然我们怀疑你的共同工作空间正在嗅探你的无线连接并试图窃取你的密码和数据，但你没有理由认为你的共同工作空间比你当地的咖啡店更安全。
*   锁住你的笔记本电脑:把你的笔记本电脑锁在你的办公桌上。 [不，真的是](https://lifehacker.com/how-do-i-keep-my-computer-secure-at-the-office-5848296) 。一旦你做到了这一点，确保你使用一种安全的方法登录你的笔记本电脑——比如使用你的指纹或你的脸，而不仅仅是输入你容易猜到的“cat123”密码。考虑设置一个应用程序，每当有人 [试图登录您的笔记本电脑](https://lifehacker.com/how-to-set-up-a-laptop-security-system-5044029) 时，该应用程序就会拍摄网络摄像头的照片。启用操作系统自带的任何“我的笔记本电脑丢失了，我可以使用另一台设备锁定、擦除或跟踪它”功能，或者 [安装一个允许您这样做的应用](https://lifehacker.com/can-i-track-my-laptop-or-smartphone-after-its-been-stol-5987302) 。对你的新工作空间友好，但不要相信任何人。

*   **在云端工作:**如果你一直在远程处理你的工作，现在只是转移到一个共同工作的空间，你可能已经习惯了在云端工作。如果没有，是时候开始把你的文件储存在谷歌这个、Dropbox 那个，或者 [或者你喜欢的](https://lifehacker.com/which-cloud-storage-service-should-you-use-1822761662) 的任何地方了。从一个好老头身边离开可能会有点不和谐。docx 到在线文档(或者更糟，电子表格)，但是你会习惯的。
*   **在你需要之前设置好合适的应用程序:**如果你需要拨入视频或音频会议，确保你在第一次会议之前已经下载了公司使用的任何应用程序(Zoom、Lifesize、Google Hangouts、WebEx 等等)。如果你的公司使用 Slack， [下载 Slack](https://lifehacker.com/how-to-behave-when-you-join-a-new-slack-1823462314)——在你的工作场所允许的情况下尽可能地定制。 [为你的工作和个人生活建立不同的账户](https://lifehacker.com/how-to-separate-your-work-and-personal-life-on-your-dev-1824208389) ，如果你用一台笔记本电脑同时处理两者的话。如果可以的话，在你家里的台式电脑上安装一个远程应用程序 ，以防你需要从你的新办公空间访问电脑上的某些东西。安装[GeForce Now beta](https://www.nvidia.com/en-us/geforce/products/geforce-now/mac-pc/)以防您想在午休时在功能较弱的工作笔记本电脑上玩点游戏...
