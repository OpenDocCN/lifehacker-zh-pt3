# 迈阿密桌面热线

> 原文:[https://lifehacker.com/the-hotline-miami-desktop-1729437449](https://lifehacker.com/the-hotline-miami-desktop-1729437449)

你可能记得几周前的[bionicfemme](http://kinja.com/bionicfemme)[earth bound desktop](http://lifehacker.com/the-earthbound-desktop-1721828694)——现在她带着这个令人敬畏的热线 Miami desktop 回来了，它完美地抓住了游戏的主题和感觉，我们忍不住要分享它。她是这样设计的。

Watch

再说一次，Bionicfemme 使用的是 OS X，所以如果你想让你的 Mac 也有类似的外观，下面是你需要的:

*   [热线迈阿密壁纸](http://hdwallpapersfit.com/wp-content/uploads/2015/03/hotline-miami-wallpaper.jpg) 由 [普罗斯基](http://protski.com/) (你可以在这里看到更多的 [热线迈阿密作品](http://protski.com/HOTLINE-MIAMI-POSTERS) ！)
*   [über sicht](http://tracesof.net/uebersicht/)，一款我们之前介绍过的 Mac [定制 app](http://lifehacker.com/ubersicht-customizes-your-os-x-desktop-with-widgets-and-1591137635)
*   [真实天气](http://tracesof.net/uebersicht-widgets/#authentic-weather) 控件为 bersicht，显示在左下角
*   文字中的小部件为 bersicht，在右下方
*   [绘制字体](http://www.dafont.com/painted.font) 使屏幕上的所有文本匹配得很好

这就是这个的全部内容！我们真的很喜欢 bersicht，它是一种简单、轻量级的定制 OS X 的方式，这是一个很好的例子，说明你可以用它做很多事情，而不必用它做很多事情。如果你喜欢它，或者对它有疑问，请访问 bionicfemme 的 Kinja 博客，链接如下，询问或分享你的赞美！
