# 你的互联网使用也有碳足迹

> 原文：<https://lifehacker.com/your-internet-use-has-a-carbon-footprint-too-1825827658>

我们如此习惯于使用电脑和手机，以至于它们感觉像是我们大脑的延伸——不仅仅是我，对吗？一个谷歌查询回来的速度就像一个想法一样快，我指尖的信息感觉就像大脑中的信息一样无形。

Watch

但是“云”是一个隐喻；所有这些数据都通过非常真实的物理线路传输，并在数据中心的机架间传输。所有这些机器(更不用说防止机器过热的空调 )都是用电的。这种电必须来自某个地方。

互联网是真实的，它有巨大的碳足迹。

艺术家 [乔安娜·摩尔](http://www.janavirgin.com/about.html) 设计了一个名为[defooooooooooooooooooooooorest](http://www.janavirgin.com/CO2/DEFOOOOOOOOOOOOOOOOOOOOOREST.html)的可视化图像，展示谷歌作为一家公司，随着时间的推移产生了多少二氧化碳。她计算出，该公司一秒钟的能源消耗相当于 23 棵树所能吸收的能量。石英检查了她的数字，发现关于如何计算不断搜索所有东西的碳足迹存在争议。谷歌表示，一个人一个月的搜索量与驾驶一辆汽车行驶一英里的影响相似。(他们还努力减少排放，并且 [表示，由于替代能源和购买碳补偿，公司目前实现了碳中和](https://storage.googleapis.com/gweb-environment.appspot.com/pdf/google-2017-environmental-report.pdf) 。)

不过，谷歌公司做的不仅仅是搜索。例如，你自己在搜索中的碳足迹应该包括给你的笔记本电脑或手机供电的能量。所以，对这些数字要有所保留，但要尽情享受，提醒自己互联网是真实存在的。
