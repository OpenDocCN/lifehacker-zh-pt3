# 我是歌曲爆炸装置主持人赫里什克什·赫威，这就是我的工作方式

> 原文：<https://lifehacker.com/im-song-exploder-host-hrishikesh-hirway-and-this-is-ho-1826532662>

赫里什克什·希尔威把歌曲拆开来学习它们是如何工作的。在他的播客[*Song Exploder*](http://songexploder.net/)中，Hirway 采访了索朗奇、伊基·波普、诺拉·琼斯、比约克和 Arcade Fire 等艺术家，询问他们的写作、录音和混音过程。希尔威作为乐队 [摩尔人](https://wearemoors.bandcamp.com/) 和*亚特兰大*的勒凯斯·斯坦菲尔德的一半制作自己的音乐，并为电视和电影作曲。他还与演员乔舒亚·马利纳共同主持《白宫风云》周刊的播客。我们和他讨论了这种多线程的艺术生活是如何交织在一起的。我们还得到了希尔威如何制作《T21》每一集的详细介绍。

#### 首先，告诉我们一些你的背景，以及你是如何走到今天这一步的。

我是一个搬到洛杉矶制作唱片和给电影配乐的音乐家。在这样做了几年之后，我利用我在那段时间里积累的经验、工具和关系，开始了 *Song Exploder* 播客。我非常喜欢制作播客，两年后，我和我的朋友约书亚·马利纳一起创办了*《白宫风云周刊》*播客，他是我最喜欢的电视节目之一*《白宫风云》*的演员。

#### 带我们看看最近的一个工作日。

我醒来，看了看我的邮件，回复了一些，发了一些。他们中的大多数人都在试图为歌曲《爆炸者》预定未来的嘉宾；要么我得到公关人员推荐的艺术家，要么我向公关人员推荐*歌曲《爆炸者》*。我一边吃早餐，一边听一些发给我的音乐，以供播客参考。

我发短信给乔希，告诉他我们为《白宫风云周刊》录制节目的时间表。我休息了一会儿，去了健身房，然后回家做午饭。

我走进录音室，为歌曲《爆炸者》录制了几个赞助广告。我和该节目的助理编辑克里斯蒂安·昆斯(Christian Koons)讨论了下一集的剪辑情况。傍晚，我有一节钢琴课。

在所有这些事情之间，我在看 Twitter 和 Instagram。我从卡洛斯·勒玛那里拿到了一张歌曲《爆炸装置》的插图，然后将他的插图按照我为 Instagram 页面制作的图形模板进行了格式化。

在那之后，我差不多在工作室里收拾好东西，带着我的狗去散步。我和妻子共进晚餐，在工作时看了一集《白宫群英》和娱乐时看了几集《美国人 T2》的间隙，我发送和回复了一些电子邮件。凌晨 1:30 左右，我插上手机准备睡觉。

#### 你生活中离不开哪些应用程序、小工具或工具？

用 [Pro Tools](http://www.avid.com/pro-tools) 完成*歌曲放音机*的所有录音和编辑。我也用 Pro 工具，还有 [Ableton Live](https://www.ableton.com/en/live/what-is-live/) ，做音乐。我的钥匙上有一个 [磁贴](https://www.thetileapp.com/en-us/) ，我大约每天丢一次，所以它派上了用场。我记录生活中的每一件事，并使用共享的谷歌表单工作。我有未来*歌曲爆炸*集的电子表格，正在进行的歌曲的待办事项列表，今年我看过的所有东西的列表，我的家庭预算，以及今年我在餐馆吃过的最好的一餐的列表。

#### 你的工作空间是什么样的？

我车库的一半被改造成了办公室，我把它当作工作室来制作我的音乐和播客。如果你走进去，你会看到一堆乐器:立式钢琴、电吉他和原声吉他、贝斯、大提琴、曼陀林和木琴。

有一张桌子——最初是宜家的桌子，但我用真正的木质桌面取代了层压桌面，并进行染色，以匹配房间的地板。桌子上有一台 iMac Pro，配有第二台显示器，一个带麦克风和耳机的音频接口，以及一个与之相连的 Thunderbolt hub。还有一个 [变焦录音机](https://www.zoom-na.com/) 和一个总是插入它或音频接口的麦克风。

桌子后面，靠着对面的墙，是客人来录制采访时坐的沙发。相对于房间的大小，房间里有很多宜家储物件，可以隐藏一些杂物。

#### **创作一集歌曲《爆炸者》的工作流程是怎样的？**

我记录面对面和远程的采访，这取决于艺术家是否有空。如果是远程的，我们通过电话或电脑交谈，他们在一个很好的麦克风上录制他们的采访，然后把音频文件发给我。然后 Christian 或我们的实习生 Olivia 将 [转录采访内容，并将其保存为共享的 Google Doc](https://twitter.com/SongExploder/status/905581623151812608) 。现在我不是唯一一个在*歌曲开发器*上工作的人，共享文档非常棒，尤其是“建议编辑”功能，所以我可以用笔记和提醒来注释部分。( [这是我在开始使用谷歌文档](https://www.instagram.com/p/7MpzMaSJSR/?taken-by=songexploder) 之前，自己编辑所有东西时的样子。)

 [https://lifehacker.com/embed/inset/iframe?id=instagram-7MpzMaSJSR&autosize=1](https://lifehacker.com/embed/inset/iframe?id=instagram-7MpzMaSJSR&autosize=1) 

然后，我们创建一个新的谷歌文档，其中只包含将出现在这一集里的采访部分，我试着将它们按照有意义的顺序放在一起，并添加注释，说明音乐的哪些部分应该在哪里播放。这就像在不知道最终图像应该是什么样子的情况下拼凑拼图一样。

然后我们测试论文编辑。克里斯蒂安拿着剧本，在 Pro Tools 中为这一集做了草稿。我会仔细检查并修改我们的谷歌文档，根据我听到的内容，删除部分，提前或推迟移动线条，在不同的地方开始音乐。我们这样做几次，大约五到八轮的修改。

然后，当感觉我们非常接近时，我会通过 Pro Tools 会话进行最终剪辑并进行最终混音——这是最挑剔的东西，很难与另一个人交流(在纸上或其他地方),因为我听到的是如此主观:以几分之一秒的速度改变暂停的长度，改变音乐淡出的形状，等等。

与此同时，我们正在努力让唱片公司和出版商批准音乐版权，插画师 Carlos Lerma 为第三集创作了一幅艺术家的原始肖像。

#### 为电影作曲和制作自己的音乐有什么惊人的不同吗？

我不知道这是否令人惊讶，但当我为电影/电视作曲时，我的效率要高得多。我会尽一切努力及时完成别人项目的音乐。但是当涉及到我自己的音乐时，我根本没有那种纪律。

#### 去生活帮最好的捷径是什么？

我不确定这是否足以算作生活帮奖，但我发现在我的手机上为我必须定期输入的较长短语(如我的电子邮件地址)定制键盘自动更正/文本替换快捷方式真的很有用。在我 iPhone 的键盘设置中，输入一个三个字母的术语作为替代，比如“eml”当我输入时，它会被我的真实电子邮件地址所取代。它对我的家庭地址特别有帮助:三个字母而不是键入整个街道地址、城市、州、邮政编码。

#### 带我们经历一个你在工作中的有趣、不寻常或挑剔的过程。

关于制作*宋爆炸器*的一切都很挑剔，真的。它经过了大量的编辑，目的是将我的问题从采访中删除，并使答案尽可能精简和清晰。但挑剔的部分来自于试图尽可能多地掩盖这一点，所以在听众听来，音乐家们天生就很有逻辑和口才。这意味着要花费数小时去寻找那些彼此不在一起的单词之间的最佳节奏。

#### 帮助你完成工作的人是谁，你如何依赖他们？

对于*宋*、T3、昆斯是助理编辑。他录制采访，并在后期制作方面帮助我，从编辑剧集到发布到网上。

在*《白宫风云》周刊*上，我们有一个两人小组: [玛格丽特·米勒](https://twitter.com/reginamint) 充当我们的过滤器，编辑我们的对话。然后，她将这些编辑内容传递给我们，以及我们的混音器和音频编辑器 [扎克·麦克尼](http://www.zachmcnees.com/) 。
