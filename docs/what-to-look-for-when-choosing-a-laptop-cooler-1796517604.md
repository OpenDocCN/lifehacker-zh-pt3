# 选择笔记本电脑冷却器时要注意什么

> 原文：<https://lifehacker.com/what-to-look-for-when-choosing-a-laptop-cooler-1796517604>

笔记本电脑的设计趋势总是倾向于更薄，而不是更胖，这样通风空间就更小了。这使得在照片编辑或游戏等高强度工作期间，风扇发出呜呜声成为一种不必要但常见的现象，尤其是在炎热的夏季。



如果你厌倦了处理过热的笔记本电脑，可以考虑买一个笔记本电脑散热垫，这是一个可以增加冷却能力的附件，也可以给你的笔记本电脑增加一两个端口。

### 买一个带风扇的

笔记本电脑冷却垫基本上是金属网格覆盖的矩形，内置风扇，旨在迫使冷空气进入您的笔记本电脑。增加的气流意味着您的笔记本电脑保持凉爽，不会在您决定完成一些工作的那一刻启动它自己的内部风扇。冷却垫的设计各不相同；有的有一个 [单个大风扇](http://www.coolermaster.com/cooling/notepal-series/notepal-x-slim-II/) 而有的有多个 [小风扇](http://www.coolermaster.com/cooling/notepal-series/notepal-u3-plus/) 。

风扇推动的空气量以每分钟立方英尺来衡量，较大的风扇比较小的风扇移动更多的空气。具有高 CFM 的冷却垫更适合冷却您的笔记本电脑，即使与具有多个较小风扇且每个风扇的 CFM 较低的冷却垫相比也是如此。

被动式笔记本电脑散热垫根本没有风扇。它们依赖于在笔记本电脑周围留出足够的空间，让空气更容易流动，就像笔记本电脑支架一样。被动冷却垫可能有凹槽或凸起，使您的笔记本电脑高出垫或膝盖几厘米。坦率地说，被动冷却垫是非常无用的，我建议你在考虑购买基本上是凹凸不平的笔记本电脑表面之前，找到一个带风扇的冷却垫。

### **确保它是多功能的**

获得专为您的理想环境设计的冷却垫。如果你喜欢在床上放松和游戏，你的笔记本电脑和散热垫的通风口可能会被你的床单或衣服遮住，所以寻找一个提供 [多个通风口](https://www.amazon.com/Cooling-TeckNet-Portable-Ultra-Slim-Notebook/dp/B014F4SBMK/ref=lp_2243862011_1_5?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/what-to-look-for-when-choosing-a-laptop-cooler-1796517604&asc_source=&ie=UTF8&qid=1498757933&s=pc&sr=1-5&tag=kinjalifehackerlink-20) 或足够间隙以方便气流的坞站。喜欢伏案工作吗？找一个带支腿的散热垫或者一个 [可调底座](https://www.amazon.com/dp/B01EUYRSJM?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/what-to-look-for-when-choosing-a-laptop-cooler-1796517604&asc_source=&psc=1&tag=kinjalifehackerlink-20) 。

如果你带着笔记本电脑旅行，便携性是一个考虑因素。一些散热垫具有集成的电缆管理功能，或者允许您将笔记本电脑固定在合适的位置以便于运输。

### **添加更多端口**

连接 USB 供电的笔记本电脑冷却器意味着你表面上失去了一个防止笔记本电脑过热的端口。有些确实有专用电源的可选适配器，但我讨厌无缘无故地随身携带另一个电源模块。

一些冷却垫提供直通支持，但这意味着您将把连接到笔记本电脑的任何 USB 设备插入冷却垫的 USB 端口，从而从您的端口形成一个超长的突起。找一个带额外 USB 端口的散热垫，像[ThermalTake MASSIVE](https://www.amazon.com/gp/product/B00J0NZYOK?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/what-to-look-for-when-choosing-a-laptop-cooler-1796517604&asc_source=&tag=kinjalifehackerlink-20)散热垫，这样你就不用担心取舍了。MASSIVE 还具有内置温度传感器，用于自动风扇动作。

你总是可以走高端路线，找到一个兼作基座的散热垫。一些 pad，如 [Cooler Master 的 FS-17](https://www.amazon.com/CM-Storm-SF-17-Ergonomic-Settings/dp/B00E5AEITU/ref=sr_1_1?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/what-to-look-for-when-choosing-a-laptop-cooler-1796517604&asc_source=&ie=UTF8&keywords=sf-17&qid=1498760199&sr=8-1&tag=kinjalifehackerlink-20) ，它包含四个 USB 端口、一个微型 USB 端口和一个可调风扇速度转盘。有些不仅提供传统的 USB-A 端口，还提供多个高速 USB 端口以及 USB 3.0 或微型 USB 端口。无论哪种方式，您都有足够的端口让您满意，并且笔记本电脑足够酷，可以放心使用。
