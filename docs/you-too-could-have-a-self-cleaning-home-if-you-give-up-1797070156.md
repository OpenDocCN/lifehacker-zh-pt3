# 如果你放弃你所热爱的一切，你也可以拥有一个自我清洁的家

> 原文：<https://lifehacker.com/you-too-could-have-a-self-cleaning-home-if-you-give-up-1797070156>

《纽约时报》的一篇迟来的讣告向发明家弗朗西斯·加布致敬，她设计、建造并居住在“世界上唯一的自我清洁房屋”中的 T2 在一个满是“亲爱的，我让孩子们变小了”的奇妙装置的家里，加布看起来是一个讨人喜欢的、有独创性的怪人。她的大想法是:把普通家庭变成一个巨大的洗碗机。



Gabe 用她的房子作为一项技术的原型，她希望这项技术能在全国范围内传播。这是一个混乱的过程，用现有的材料和院子里的水泥搅拌机拼凑而成，天花板上的洒水装置像雨点般洒落在防水的房间里。地板略微倾斜，涂上了清漆，床盖着天蓬，书被“防水外套”保护着。水管会爆裂，向不需要的方向喷水，正如当地新闻的视频所捕捉到的:

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-h95O5dQkoD4&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-h95O5dQkoD4&start=0) 

但这就是原型的本质。他们还没有通过设计部门；没有人挑选出消费者友好的颜色；他们没有配件来填补空白。

清洁并不容易。意思是把脏东西，或者 [【物质不在地方】](https://en.wikipedia.org/wiki/Purity_and_Danger) 放回它该在的地方。从根本上说，这是一场对抗熵的战斗，一场失败的战斗，它用肥皂、水和装满垃圾的袋子制造了自己暂时的混乱。这项最困难的工作绝大多数被分配给妇女，然后被贬低为从属角色。从事 [看门人](https://www.bls.gov/oes/current/oes372011.htm) 和 [女佣](https://www.bls.gov/oes/current/oes372012.htm) 工作的 300 万美国人平均收入远低于 3 万美元；数以百万计的全职太太为她们的伴侣给予她们的任何东西而工作；还有数百万人在全职工作之外的业余时间从事清洁工作，这是一种吃力不讨好的“第二职业”。

虽然家务技术在过去的一个世纪里取得了巨大的进步，但它仍然是高度非自动化的。现有的创新涉及权衡取舍。洗衣机和洗碗机不如手洗精确，更具攻击性。搅拌器需要重新灌装，会产生垃圾。Roomba 在复杂的地毯地形上有问题。我们考虑到这些机器:我们购买可以用洗碗机清洗的玻璃器皿和易于擦拭的台面。加布所要求的只是再多一些这样的津贴，以换取一种免于擦洗和弯腰的生活。

在某些情况下，这是一个很好的权衡。烤箱会把自己的污垢烤干；特氟隆处理过的衣服和炊具可以吸走不想要的污垢。像巴黎、波士顿和旧金山这样的城市都有自动清洗的公共厕所，它们像盖比的厨房一样自动冲洗。德累斯顿的一个例子:

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-HYx0m3TfGJA&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-HYx0m3TfGJA&start=0) 

自我清洁的公共浴室是 Gabe 的愿景，在最重要的地方实现。上面的演示视频看起来很像下面这个半开玩笑的模拟 Gabe 的房子， [由艺术家 Lily Benson](http://lilybenson.com/self-cleaning-house/) 参观实物后制作。

在这种低保真度的 CGI 处理中，滑稽不真实的水滴杂乱无章地泼在厨房家具上。这让缺点显而易见，但也表明了加布的愿景与其他宏伟的概念有多么相似，如太空电梯或 [埃隆·马斯克的 Hyperloop](http://www.businessinsider.com/hyperloop-one-how-it-works-2017-7) 。每个伟大的想法在早期阶段看起来都很荒谬。

 [https://lifehacker.com/embed/inset/iframe?id=vimeo-134357432](https://lifehacker.com/embed/inset/iframe?id=vimeo-134357432) 

在盖比的房子里，即使是最基本的概念也会显得可笑。它要求你完全重新考虑你的优先事项。为了避免拖地和除尘，你会在家里只放防水物品吗？你会经常把你所有的盘子都浸在水里以避免洗碗机卸载吗？

与我们已经同意的妥协相比，这些似乎不那么极端:你会买一袋生菜，这样你就不必洗它了吗？你会维护你朋友的数据库以便保持联系吗？你会把地球铺在人行道上以便更快地绕着它移动吗？你会用你的一生盯着灯看，以便有更多的时间盯着更多的灯看吗？

最终，公众并没有准备好做出 Gabe 提议的权衡，但像 [核能汽车](https://www.extremetech.com/extreme/186907-where-are-all-the-clean-infinite-range-nuclear-powered-cars-ships-and-planes) 、 [超音速客机](http://idlewords.com/talks/web_design_first_100_years.htm) 和 [模块化手机](https://www.theverge.com/2016/9/2/12775922/google-project-ara-modular-phone-suspended-confirm) 一样，它们仍然值得考虑。自我清洁的房子并不是作为未来住宅的模板而存在的。它的存在给我们上了一课:如果你准备降低标准，总有提高效率的空间。
