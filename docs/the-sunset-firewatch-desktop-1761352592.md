# 日落火眼桌面

> 原文：<https://lifehacker.com/the-sunset-firewatch-desktop-1761352592>

在 之前，我们已经 [推出了 Firewatch 桌面，但现在游戏推出了，我们看到了更多——谁能怪你呢，游戏的氛围是一个巨大的吸引，风景也很华丽。读者贾迪塔把这一个放在一起。](https://lifehacker.com/the-firewatch-desktop-1690335235)



这个桌面全是 windows，加上一点雨量计的魔力，也可以是你的。这是你需要的:

*   官方 Firewatch 壁纸包 中的 [日落黄壁纸](http://blog.camposanto.com/post/138965082204/firewatch-launch-wallpaper-when-we-redid-the)
*   用于 Windows 的 [雨量计](https://www.rainmeter.net/) 系统调试和监控工具
*   左下角显示时间的雨量计 的 [单线时钟控件](http://rainmeterhub.com/skins/1-line-clock/)
*   Croxray 的雨量计模拟时钟模拟时钟显示——以前是 [这里](http://customize.org/rainmeter/skins/65672515) ，但那是离线的——你可以在互联网档案馆找到 [这里](http://web.archive.org/web/20150329211159/http://customize.org/rainmeter/skins/65672515) ( 谢谢内森！)或者 Reddit 上的一些 [微调版](https://www.reddit.com/r/Rainmeter/comments/2kf5x1/fixed_croxrays_analog_clock_to_perfectly_fit_in/)
*   [行 2.0](http://rasylver.deviantart.com/art/Lines-2-0-154719861) 为系统监控小部件
*   [KantasAkra for rainmet](http://rabra.deviantart.com/art/Kantas-Akra-479898554)作为一个简单的可视化工具也在左下角

你可能会有一些困难，让模拟时钟工作，但如果你没有找到一个好的版本或完全忽略它，完成的包看起来仍然很棒。如果你喜欢它，或者对它有疑问，去 jadita 的 Kinja 博客(链接如下)分享你的想法或者提问。
