# 准妈妈们在怀孕期间可能不需要停止服用左洛复或赞安诺

> 原文:[https://life hacker . com/moms-to-be-possible-dont-need-stop-taking-zoloft-or-1818562657](https://lifehacker.com/moms-to-be-probably-dont-need-to-stop-taking-zoloft-or-1818562657)

这是一个令人痛苦的处境:你惊恐发作，但你担心在怀孕期间服用抗焦虑药物会伤害你的宝宝。关于这个主题的研究一直参差不齐，但根据一项新的研究，尽管苯二氮卓类药物(抗焦虑药物，如 Xanax)和 SSRI 抗抑郁药物(如左洛复和帕罗西汀)都有一些风险，但女性可能不需要停止服用这些药物。

金伯利·扬克斯博士的研究 [发表在本周的 *JAMA Psychiatry* 杂志](http://jamanetwork.com/journals/jamapsychiatry/article-abstract/2652968) 上，她发现苯二氮卓类药物带来了一些产妇健康风险:妊娠时间略有缩短(服用药物的妇女比未服用药物的妇女平均提前 3.6 天分娩)，剖腹产的几率更大，新生儿出生后需要更多的氧气支持。

服用抗抑郁药的女性怀孕时间也缩短了(但平均缩短不到两天)，她们的婴儿也更可能需要氧气支持。考虑统计数据很重要，但是正如扬克斯在 NPR 的采访 中所说，“我们在这里没有看到巨大的影响，这应该是令人放心的。”她的研究还发现，怀孕期间患有恐慌或焦虑症似乎不会对发育中的婴儿造成健康风险。

底线:请放心地和你的医生谈论寻找适合你的治疗方案。对怀孕期间的抑郁和焦虑不予治疗是有后果的，任何女性都不应该觉得自己必须独自承受痛苦。
