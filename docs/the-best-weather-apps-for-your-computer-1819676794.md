# 最佳桌面天气应用

> 原文：<https://lifehacker.com/the-best-weather-apps-for-your-computer-1819676794>

虽然天气预报是一门众所周知的不精确的科学，但由于语音助手、弹出通知和嗡嗡作响的智能手表，现在比以往任何时候都更容易监控当天的天气(并据此穿着)。当然，在手机上查看被删节的天气预报可能会很方便，但这也可能意味着你错过了有价值的信息，这些信息可以帮助你应对外面的高温、雨夹雪或普遍的闷热。这就是桌面天气应用的用武之地。

虽然你可能会被 Mac 应用商店或 Windows 商店中的天气应用数量所淹没，但只有少数几个既有漂亮的外观又有丰富的数据。以下是从 Windows 10 或 macOS 电脑上查看天气的最佳方法。

### **Windows: MSN 天气和天气频道**

#### **MSN 天气**

Windows 10 自带免费天气 app， [MSN 天气](https://www.microsoft.com/en-us/store/p/msn-weather/9wzdncrfj3q2?wa=wsignin1.0) ，还不错。它受益于 Windows 10 的可定制实时磁贴功能，当你查看开始菜单时，它会显示关于本地预测的更新信息。这意味着你不必打开应用程序来查看天气，你只需按下 Windows 键。

MSN Weather 显示当前的天气预报，以及未来十天的预测摘要。能见度、露点和大气压力等信息也包括在内。你可以看看你所在地区的卫星图，看看任何即将到来的风暴，或者观察最近的天气模式。

此外，MSN Weather 向您展示了大量旨在吸引偏执的业余气象学家的附加信息。你可以记下日出和日落的时间(非常适合寻找拍摄照片的黄金时间)，并且由于包含了历史降水量和温度数据，你可以知道 30 年内某一天下雨的可能性，从而知道会发生什么。

#### **天气频道**

天气频道 的免费 Windows 10 应用可能比天气爱好者更吸引过敏患者。它包括与 MSN Weather 相同的天气预报信息，并支持相同的可定制实时磁贴功能。

健康部分共享信息，如树木、草和杂草花粉的估计，以及流感活动(现在是冬天，所以流感活动图是红色的)。您还可以配置天气频道，根据您的位置向您发送花粉、降雨或恶劣天气警报，如果您忘记亲自询问您的电脑，这将非常方便。仅这一点就值得一看。然而，这并不是最漂亮的应用。它以大而醒目的方式显示今天的天气预报，但迫使你向下滚动以获取其他重要信息，并允许不相关的新闻文章占据宝贵的空间。

### **Mac:** **预报栏最好在显示**

虽然 Windows 10 有一个强大的内置天气应用程序，但 macOS 就没有这么幸运了。有一个天气应用程序可以在你的通知中心显示大致的天气预报，但它缺乏了解未来一天或一周所需的信息。如果你想更方便地获取天气信息，以及更多的天气信息，那么 [天气预报栏](http://forecastbar.com/mac) 应该是你要找的应用。我们曾经写过 [关于天气预报栏](https://lifehacker.com/forecast-bar-puts-a-ton-of-weather-info-in-your-macs-me-1733041721) 的文章，并且仍然认为它是你 Mac 上最好的天气预报应用之一。

它存在于你的菜单栏中，提供了比 Mac 默认天气服务更多的信息。您可以在 dock 或菜单栏中显示您的天气数据，并根据 Dark Sky 提供的天气数据查看动画预报。

您可以查看日出和日落时间、历史天气模式、湿度、露点等信息。该应用程序也很好看，使它成为你的 Mac 应用程序库的一个很好的补充。9.99 美元有点贵，还可以购买额外的应用程序来增加天气更新的频率(预报栏每小时更新一次)。即便如此，如果你厌倦了下雨，Forecast Bar 可能是你需要保持干燥的天气应用程序。
