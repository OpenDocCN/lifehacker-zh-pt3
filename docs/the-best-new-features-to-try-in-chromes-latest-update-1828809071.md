# Chrome 最新更新中值得尝试的最佳新功能

> 原文:[https://life hacker . com/the-best-new-features-to-try-in-chromes-latest-update-1828809071](https://lifehacker.com/the-best-new-features-to-try-in-chromes-latest-update-1828809071)

为了纪念 Chrome 浏览器 10 周年，谷歌推出了浏览器的重大更新，让它看起来更柔软、更圆润，并增加了一些新功能。Chrome 69 版本中的大多数变化都是生活质量的小调整，其中一些非常细微，你可能甚至不会注意到它是 Chrome 新设计的一部分。尽管它们可能不会改变生活，但在新的 Chrome 体验中，还是有一些值得一试的东西

### 密码生成器

Chrome 现在除了存储密码之外，还可以制作密码——这是此次更新中最重要的功能。浏览器的密码存储和自动填充功能现在包括一个 [密码管理器](https://lifehacker.com/the-five-best-password-managers-5529133) 、*a la*[LastPass](https://lastpass.com/)或 [1Password](https://1password.com/) ，这使得默认情况下使用更长、更好的密码变得更加容易。

当你注册一项新服务时，Chrome 会很乐意为你提供一个强密码供你尝试。只需点击密码框，谷歌就会为你生成一个密码，就像它会提供你已经保存的密码一样。

如果你已经在使用密码生成器，这个特性就不会觉得有什么新鲜了，但是至少将这个服务集成到浏览器中是很方便的。

### 谷歌给了搜索栏一个巨大的推动力

Chrome 现在在搜索栏中显示常见查询(如当前天气)的搜索结果，所以你甚至不用点击第二页就能找到你想要的答案。

在谷歌的 [公告博客](https://www.blog.google/products/chrome/chromes-turning-10-heres-whats-new/?_ga=2.41429028.748690221.1536081064-752918771.1534960295) 中，谷歌表示，它可以通过 Chrome 的 Omnibox 呈现各种信息，但目前似乎仅限于非常常见的特定短语，如“*超人前传 2* 有多长？”(两小时五分钟)和“天气如何？”(布鲁克林这里闷热的 91 度)。

### “Omnibox”中的选项卡搜索

我对此感到非常兴奋。你是否曾经打开了太多的标签，以至于你找不到你要找的文章或对话，迫使你系统地关闭标签，以在大海捞针中搜索你的数字生活？

你当然有。我一天至少做两次。

为了帮助我们找到我们在浏览器中保存和丢失的内容，谷歌让 Chrome 能够通过 Omnibox(地址栏)找到特定的网页。只需输入网站或页面名称，然后选择“切换到标签”即可发现您丢失了什么！我已经呼吸顺畅了。

Chrome 的美学革新还通过使每个网站的图标更加突出，使标签更容易阅读，所以你不太可能一开始就丢失标签。

### 在你的手机上，搜索栏触手可及

如果你在手机上使用 Chrome，有一个很大的用户界面变化非常突出——原谅我——拇指。

Chrome iOS 应用程序现在在屏幕底部而不是顶部有一个粘性工具栏。这有效地重新配置了单手浏览的体验，因为你不需要笨拙地将手移到手机屏幕的顶部来选择标签或拉起搜索栏。

“更多”菜单中的选项列表——工具栏右侧的省略号——也更容易阅读和浏览，这要归功于更大的文本和新的图标功能，如书签和请求页面的桌面版本。
