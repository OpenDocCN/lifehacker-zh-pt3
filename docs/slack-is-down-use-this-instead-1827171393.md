# 松弛度下降，用这个代替

> 原文：<https://lifehacker.com/slack-is-down-use-this-instead-1827171393>

今天早上，由于空闲停机，各地的远程办公室都瘫痪了。如果你的团队错过了群聊，有大量的选项可以快速替换。我个人最喜欢的是 Stride。



是由制作 HipChat 的同一批人制作的，这是我的上一个办公室(可能也是你的办公室)在他们跳到 Slack 之前使用的。

在他们的网站上注册只需几分钟，而且对无限用户免费(尽管功能有限)。

要创建一个房间，有一个(且只有一个)人大步走过去报名。注册过程大约需要 2 分钟，然后他们会被安排到一个房间，在那里他们可以通过电子邮件邀请其他人。我想从我创建一个生活黑客房间到今天早上我有同事在里面的时间大概是五分钟左右。

你 g et 免费无限用户和聊天室，以及一些内置的协作工具，文件共享和存储高达 5GB(付费版无限)，消息历史高达 25k 消息(付费版无限)。

你可以在这里 查看免费版和付费版的 [区别。](https://www.stride.com/pricing)

如果 Stride 不适合你，其他几个不错的选择包括 [Rocket Chat](https://rocket.chat/) (尽管我今天早上没能让它给我发一封确认邮件)和 [Ryver](https://ryver.com/) (以及一大堆其他的)。如果你的团队很小，通过 Gmail 中的 Google Hangouts，当然还有电子邮件，也有不错的老式聊天。
