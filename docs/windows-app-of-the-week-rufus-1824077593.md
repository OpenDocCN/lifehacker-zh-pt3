为了开始我们的 Windows 报道，我们先来看看 [Rufus](https://rufus.akeo.ie/) ，这是一个轻量级的实用程序，它使得创建可引导的 USB 闪存驱动器变得非常容易。

### 最有用的工具的最奇怪的名字

我们真的不知道名字，但没关系。Rufus 开发者 Pete Batard 可以调用这个实用程序 [Poopsmith](https://www.youtube.com/watch?v=ub0wV0TvY8Q) ，我们仍然会定期使用它。由于我们生活在一个“DVD 驱动器”或“蓝光播放器”的概念已经在台式机和笔记本电脑上过时的时代，所以将闪存驱动器用作各种“可重复使用的 DVD”非常有用。

例如，假设您想在某个闪存驱动器上保存一份 Windows 10 ISO 的新副本，以便随时重新安装。或者，就此而言，也许你只是想创建一个基本的 ISO 文件，作为一个方便的备份。

甚至在某些情况下，你可能需要一个可启动的 USB 闪存驱动器——比如说，为了更新主板的固件，或者如果你出于某种原因只想摆弄一个替代操作系统的实时版本。你甚至可能想要运行一个可启动的应用程序，比如 [MemTest86+](http://www.memtest.org/) 。

(就在前几天，我使用 Rufus 创建了一个 macOS High Sierra 的可引导 ISO 来进行试验；别告诉库比蒂诺。)

Rufus 不仅易于使用，而且速度超快。正如巴塔尔指出的:

> 哦，鲁弗斯跑得很快。例如，在从 ISO 创建 Windows 7 USB 安装盘时，它的速度大约是 UNetbootin、通用 USB 安装程序或 Windows 7 USB 下载工具的两倍。从 ISOs 创建 Linux 可引导 USB 的速度也略快。”

作为一个额外的奖励，该应用程序不需要你安装任何东西来使用它。只需下载可执行文件并双击。您可能会将大多数设置保留为默认值，但在大多数情况下，您可能会希望将 USB 驱动器的文件系统设置为 FAT32，以获得最大的兼容性。单击光盘图标加载 ISO，单击开始，享受您的新的可启动 USB 闪存驱动器。就这么简单。
