# 确定你是否太高而无法使用此应用程序驾驶

> 原文：<https://lifehacker.com/this-app-helps-you-determine-if-youre-too-high-to-drive-1821355520>

现在有一大堆不同的选择来确定你是否醉得不能开车，但是关于你是否抽了太多的大麻？

Watch

醉酒驾车可能会有危险。既然越来越多的州将娱乐性大麻合法化，他们也必须想出办法来测试司机是否不够清醒，不能上街。

在俄勒冈州，这是以现场酒精测试的形式出现的。你知道，就是你在电影里经常看到的那种，你必须走直线，把手指放在鼻子上。它的意思是判断你的反应时间。如果你失败了，那么你可能不得不做尿检，如果你在过去几周而不是几个小时内吸食过大麻，尿检会产生阳性结果，并让你因酒后驾车而被定罪。

这肯定是一个有缺陷的系统。一位大学教授认为这是一个创造更好东西的机会。输入 [德鲁伊](https://play.google.com/store/apps/details?id=com.owl.druid&hl=en) 。这是一款售价 0.99 美元的安卓应用，它会在一系列测试中判断你的反应时间，这些测试可能需要两到五分钟，然后应用会给出你是否可以开车的建议。

首先，如果你认为你有可能不适合开车，那么你就不应该开车。

然而，如果你几个小时前抽了烟，觉得自己很棒，只是想在出门前检查三遍，那么它可以是一个不错的开始。

我下载了它，在完全清醒的情况下进行了测试,(奇迹般地)收到了一个分数，说我实际上可以开车了。测试包括遵循简单的指示，比如用手指在屏幕上画一个圆圈，同时数出现的方块的数量，并在出现提示时点击屏幕上的不同点。这没什么太难的，但即使在清醒的时候，我也发现有些任务很有挑战性。

该应用程序还使用了一些穴居人字体，这不是我最喜欢的，看起来像一个瘾君子可能会做的东西，但我跑题了。

也就是说，嗑药后，我想它会派上用场的。大麻会让你慢下来，有时候(至少对我来说)我意识不到我的反应时间有多短。这绝对是一个挑战，要清醒地集中注意力并遵循指示。飘飘欲仙的时候，我想我会立刻意识到我不应该在尝试了这些任务中的一项后就离开沙发，我不需要一路走过去，让应用程序告诉我留在原地。
