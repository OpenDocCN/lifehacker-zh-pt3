# 用这种无花果和姜汁汽水让秋天变得轻松

> 原文：<https://lifehacker.com/ease-into-fall-with-this-fig-and-ginger-spritz-1829063146>

周末快乐，欢迎回到 [*三料欢乐时光*](http://lifehacker.com/tag/3-ingredient-happy-hour) ，每周饮品专栏，提供超级简单又美味的酒水。本周，我们脱去夏日的外衣，带着熟透的无花果和辛辣的生姜，进入凉爽的天气。(在鸡尾酒中加入生姜是我能想到的最接近于在鸡尾酒中加入南瓜香料的方法。)

Watch

无花果总是让人觉得是夏天最后的狂欢。他们的季节是短暂的，但他们带来的快乐是永恒的。亚得里亚海无花果太甜了，这促使可爱的杂货商将它描述为“大自然的果冻甜甜圈”，这充其量是夸张的— *面团在哪里？*——但是要表达清楚。当与强烈的波旁威士忌和强烈的姜味啤酒混合在一起时，强烈的甜味被卷入其中。它也非常漂亮。要自己做一个，你需要:

*   2 盎司波旁威士忌
*   2 个新鲜的亚得里亚海无花果
*   你最喜欢的姜啤酒是用真正的姜制成的

把无花果切碎，和波本酒一起放在摇壶里。把无花果弄乱遗忘。加入冰块，摇匀波旁威士忌，直到完全冷却，然后用细滤网滤入加冰的低杯。加入几盎司的姜汁啤酒，如果你打算在 Instagramming 上用一片漂亮的无花果片装饰(与所有的南瓜香料拿铁咖啡竞争)。
