# 设置 Twitter 的两步认证来使用第三方应用程序

> 原文：<https://lifehacker.com/set-up-twitters-two-step-authentication-to-work-with-th-1821475351>

如果你在 Twitter 账户上设置了双重认证(这是你绝对应该做的，如果你没有，你应该现在就设置)，那么现在你就不必使用短信来访问你的账户了。

现在，您可以使用第三方应用程序，如 Google Authenticator、Duo Mobile 或 Authy。如果你在另一个国家(或你朋友的办公室)这样的地方，你没有手机信号，并且想进入你的账户，这就非常方便了。

要设置为使用第三方验证器应用程序，你首先需要将你的 Twitter 设置从默认打开的短信更改为安全应用程序。为此，请在 Twitter 上点击您的个人资料图标，然后选择设置。在那里，单击帐户，然后单击安全性。在菜单中，你会看到默认情况下短信是打开的。点按“编辑”将其关闭，然后启用安全应用功能(位于其下方)。

要设置单独的第三方应用程序，请在顶部菜单中，单击您的个人资料图标，然后单击设置和隐私，然后选择帐户选项卡。单击“查看您的登录验证方法”开始。系统会提示您输入 Twitter 密码。完成后，单击设置，然后按照屏幕上的说明进行操作。

最终，你会看到一个弹出窗口显示一个二维码。您将需要扫描您的验证器应用程序中的二维码，以将其连接到您的 Twitter 帐户。然后，你的应用程序会给你一个 6 位数字的安全码，你必须输入 Twitter 才能完成连接。

一位 Lifehacker 作家也报告说，在禁用文本双因素后，他必须重新启用它，然后我才能启用基于移动应用程序的双因素。如果你在途中遇到麻烦，试试看。

整个过程肯定需要几分钟，但是完全值得。
