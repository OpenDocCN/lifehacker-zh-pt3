# 为什么你不应该马上获得亚马逊密钥

> 原文：<https://lifehacker.com/why-you-shouldn-t-get-the-amazon-key-right-now-1819924272>

本周，亚马逊发布了亚马逊钥匙，这是一种新的选择 ，当你不在家时，给司机一把特殊的数字钥匙，就可以送货上门。在兼容的智能锁和安全摄像头的帮助下，送货司机将能够打开你的前门，走进去递送你的包裹，而你可以通过智能手机观看这一切。如果你重视方便胜过安全，那么无论如何，请自便。但安全风险、隐私担忧和令人毛骨悚然的图像分析细节足以说明，至少在你看到该技术的早期版本如何发挥作用之前，推迟可能是更安全的选择。



在你安装了智能锁和云摄像头之后，包裹快递员(理论上由亚马逊事先进行背景调查)会打开你的门，把你的箱子放进你的家里。云摄像头监控整个交易过程，如果你愿意，还可以通过智能手机实时观看送货过程。

虽然亚马逊钥匙(理论上)将确保你不必担心在家就能收到你的货物，但该公司还希望客户将智能锁 [用于更广泛的用途](https://gizmodo.com/amazon-key-is-bigger-than-package-delivery-1819838149) ，例如给朋友、家人或房屋保姆临时访问权。

这一切听起来很好，对亚马逊显然认为我们都居住的看似乌托邦的社会很好，但这可能会有太多的方式出错，无论是通过实际的硬件还是被允许进入你家的人的行为。泄露的摄像机镜头，故障的服务器(it [不是第一次](https://gizmodo.com/amazon-says-one-engineers-simple-mistake-brought-the-in-1792907038) 亚马逊意外宕机)或者厚颜无耻的窃贼完全无视摄像头，强迫你 [向亚马逊](https://www.amazon.com/gp/help/customer/display.html/ref=s9_acss_bw_cg_ODSBRLS_md1_w?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/why-you-shouldn-t-get-the-amazon-key-right-now-1819924272&asc_source=&nodeId=202187520&pf_rd_i=17285120011&pf_rd_m=ATVPDKIKX0DER&pf_rd_p=c1d89432-c718-4291-ba72-5142feb88a62&pf_rd_r=HKTFMQGJF397RDEZJ2EY&pf_rd_s=merchandised-search-8&pf_rd_t=101&ref=ods_bls_hqp_lnch_2017_hpngnte&tag=kinjalifehackerlink-20) 提出索赔，这只是亚马逊密钥最终可能带来麻烦而非帮助的几种方式。

### **智能锁笨重**

智能锁虽然是一个吸引人的选择，但仍处于初级阶段。安全专家 [并不完全相信它们](https://lifehacker.com/we-asked-five-security-experts-if-smart-locks-are-ever-1797910643) ，因为安全缺陷可能会让你比使用传统锁更容易受到攻击。

事实上，今年早些时候发生了一次智能锁故障，数百名使用 Lockstate 智能锁 [的 Airbnb 客人发现他们无法进入](https://gizmodo.com/smart-locks-used-by-airbnb-get-bricked-by-software-upda-1797839523) 他们租住的房屋，原因是固件更新失败。不难想象，一个有缺陷的亚马逊密钥更新会允许任何人进入你的家，或者阻止你进入自己的家。

### **云 Cam 一直在看**

如果你喜欢在家里有一个像亚马逊 Echo 这样的语音助手， [听你的对话](http://fieldguide.gizmodo.com/how-to-lock-down-your-privacy-on-the-amazon-echo-and-go-1794697554) ，云摄像头对你来说不会那么奇怪。当它打开时，云摄像头会不断记录和擦除存储在相机上的镜头。如果它检测到运动，你家的镜头就会上传到亚马逊的服务器进行图像分析，以确定检测到的活动是否值得你注意，或者只是狗走过。通过使用云摄像头，你允许亚马逊“在云中处理和保留你的记录，以提供和改善我们的服务，”大多数人都不愿对他们的门厅视频镜头采取这些行动。

这里的底线是 Amazon Key 依赖于两项存在安全问题的技术，这应该会让消费者犹豫不决。无论是智能锁还是云连接摄像头，都无法让你安心考虑立即注册这项服务。
