# 口袋大小书写和素描袋

> 原文：<https://lifehacker.com/the-pocket-sized-write-and-sketch-bag-1766162006>

通常，你并不需要一直带着一个全尺寸的包。《每日随身携带》的读者泰勒·史密斯就是这样，他把所有需要的东西都装进了一个小小的笔记本封面。

Watch

*   [【雨中祭】点击笔](http://everydaycarry.com/go/purchase/3981/az/4)
*   [甘藏 G726M 440C](http://www.amazon.com/dp/B0146U6W3I?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-pocket-sized-write-and-sketch-bag-1766162006&asc_source=&tag=kinjalifehackerlink-20)
*   [海岸 G-19 54 流明](http://www.amazon.com/dp/B008X2QMKY/?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-pocket-sized-write-and-sketch-bag-1766162006&asc_source=&tag=kinjalifehackerlink-20)
*   [洛斯木匠铅笔](http://www.amazon.com/dp/B000GARZFW?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-pocket-sized-write-and-sketch-bag-1766162006&asc_source=&tag=kinjalifehackerlink-20)
*   [【雨中祭】4x6 笔记本](http://www.amazon.com/dp/B004QI8E6Y?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-pocket-sized-write-and-sketch-bag-1766162006&asc_source=&tag=kinjalifehackerlink-20)

*   [斑马 F301 0.07mm BP](http://www.amazon.com/dp/B001CRVUPE?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-pocket-sized-write-and-sketch-bag-1766162006&asc_source=&tag=kinjalifehackerlink-20)
