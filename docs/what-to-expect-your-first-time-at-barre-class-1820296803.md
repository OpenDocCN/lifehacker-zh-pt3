# 你第一次上芭蕾课会有什么期待

> 原文：<https://lifehacker.com/what-to-expect-your-first-time-at-barre-class-1820296803>

我最近尝试了我的第一堂 barre 课，这要感谢 [Classpass](https://classpass.com/) 的魔力，它让我很容易地进入一个新的工作室，尝试一些你从未做过的事情。我的印象是:这是一个很好的锻炼，但也有点...好吧，没有什么好的方式来表达。无聊。



以下是我认为的优点:

*   栏杆本身(房间周围的一种齐腰高的栏杆)是锻炼的好道具。
*   之后，我的肌肉肯定感觉像是经过了一次很好的锻炼。
*   如果你闭上眼睛，你可以假装你是一个漂亮的芭蕾舞演员。

缺点是:

*   这些动作往往很小，也很累人:抬起你的腿，前后摆动三百万年。
*   如果你睁开眼睛，你会被镜子包围，很明显你不是一个漂亮的芭蕾舞演员。

工作室要求我们穿袜子；许多 barre 爱好者喜欢使用底部有一些小块的粘性袜子 来提供一点抓地力。另一个可能是我工作室特有的额外待遇:他们没有更衣室，但是他们*有一个长长的柜台，上面安装了镜子。柜台上摆满了干性洗发水之类的美容产品，这样你就可以在锻炼后梳洗一番。我同时有两个反应:“想得真周到！”“来这里的其他人肯定比我时髦得多，因为我会像往常一样汗流浃背地回家。”*

巴雷课非常受欢迎，所以仅仅因为我不喜欢并不意味着 T2 你也不会喜欢。而事实上，我在 Classpass 上做的 barre 比其他任何东西都多，只是因为 barre 工作室有最方便的上课时间。他们还提供将 barre 和其他类型的运动结合起来的课程，比如跆拳道，这是一种很好的多样化运动。

所以我很好奇，对于那些做 barre 的:你爱它什么？你会对第一次尝试的人说什么？
