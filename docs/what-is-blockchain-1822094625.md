# 什么是区块链？

> 原文：<https://lifehacker.com/what-is-blockchain-1822094625>

突然间，无处不在。这项技术于 2008 年发明，一年后推出时为比特币提供动力，目前正被用于从 [版权保护](https://lifehacker.com/why-kodaks-bitcoin-scheme-is-a-scam-you-should-avoid-1821957658#_ga=2.186086191.785031894.1516112040-1313785359.1499701416) 到性同意( [是的，真的是](https://gizmodo.com/men-try-to-redefine-sexual-consent-with-blockchain-1821964907#_ga=2.186086191.785031894.1516112040-1313785359.1499701416) )的一切事情。



考虑到区块链周围每天都有大量的新闻，更不用说比特币 [和其他依赖该技术的加密货币](https://lifehacker.com/the-best-bitcoin-and-ethereum-explainers-1796654705#_ga=2.90225692.785031894.1516112040-1313785359.1499701416) 的价格飙升，你可能会想知道区块链到底是个什么地方。这实际上是一个非常简单的概念，尽管你越仔细看，事情就会变得越复杂。

考虑到这一点，这里有一些不同的方式来了解区块链，从简单的定义到深远的隐喻。

### 区块链是如何工作的？

首先，这是最简单的解释，没有隐喻或夸张。在加密货币的语言中，块是新交易的记录(这可能意味着加密货币的位置，或者医疗数据，甚至投票记录)。一旦每个块完成，它就被添加到链中，创建一个块链:一个区块链。

由于加密货币是加密的，处理任何交易都意味着解决复杂的数学问题(随着区块链的增长，这些问题会变得更加困难)。在一个被称为“挖掘”的过程中，解决这些方程的人会获得加密货币的奖励。

如果你拥有任何加密货币，你真正拥有的是它在区块链上的地址的私钥(基本上只是一个长密码)。有了这把钥匙，你可以提取现金来消费，但是如果你丢失了这把钥匙，就没有办法把钱拿回来了。每个账户都有一个公钥，让其他人可以向你的账户发送加密货币。

关于区块链的信息也是公开的。它是分散的，这意味着它不依赖于单个计算机或服务器来运行。因此任何交易对每个人都是即时可见的。这就引出了我们的第一个比喻:公共账本。

### 区块链就像一本公共账本

如果你把比特币(或其他加密货币)发给朋友，或者卖掉它，这些信息在区块链上是公开的。其他人可能不知道你的身份，但他们知道从一个人身上转移到另一个人身上的价值到底有多少。

许多人将区块链视为传统银行业的替代品。你可以使用区块链来消除中间人，而不需要银行或其他机构来验证资金的转移。

### “价值互联网”

基于公共分类账的概念，描述区块链的另一种流行方式是价值互联网。这个想法很简单:互联网使得在线自由发布数据成为可能，区块链为了钱也做了同样的事情。

互联网不再依赖报纸、电视和广播(主要由大公司控制)，而是让每个人都有发言权——不管是好是坏。区块链和加密货币使得绕过银行甚至政府等传统中间人在世界各地转移资金变得同样容易。

### 区块链就像谷歌文档

《商业区块链*的作者威廉·穆加亚尔 对区块链 [做了一个巧妙的比喻:区块链就像谷歌文档。](https://www.coindesk.com/understand-google-docs-can-understand-blockchain/)*

在谷歌文档出现之前，如果你想在网上与某人合作写一篇文章，你必须创建一个微软 Word 文档，发送给他们，然后请他们编辑。然后，您必须等待，直到他们做出这些更改，保存文档，并将其发送给您。

Google Docs 解决了这个问题，让多人可以同时查看和编辑一个文档。然而，今天的大多数数据库仍然像 Microsoft Word 一样工作:一次只有一个人可以修改，其他人都被锁在外面，直到他们完成。区块链通过即时更新任何变化让每个人都能看到来解决这个问题。

对于银行业来说，这意味着任何资金转移都要在两端同时进行验证。区块链还可以用于法律业务或架构规划——实际上是人们需要就文档进行协作的任何业务。

### **区块链就像一排保险箱**

这里有另一个来自 [在线论坛比特币对话](https://bitcointalk.org/index.php?topic=269369.0) 的有用解释。这个很好地解释了公钥和私钥是如何工作的:

> 想象一下，在某个地方的一个巨大的房间里有一堆保险箱。每个保险箱上都有一个识别号码，每个保险箱都有一个投币口，人们可以往里面投钱。保险箱都是由防弹玻璃制成的，所以任何人都可以看到任何一个保险箱里有多少钱，任何人都可以把钱放在任何一个保险箱里。当你开一个比特币账户时，你会得到一个空的保险箱和保险箱的钥匙。你记下保险箱上的号码，当有人想给你寄钱时，你告诉他们哪个保险箱是你的，他们就可以把钱放进投币口。

### 区块链就像 DNA

最后这个，从 [罗宾·肖汉上中](https://medium.com/@rchauhan/the-bitcoin-blockchain-dna-metaphor-e9500333ca9f) ，有点离谱，但是我喜欢。

区块链是交易的记录，随着越来越多的人使用加密货币，区块链在互联网上传播开来。类似地，DNA 是随着生命在地球上扩展而传播的遗传交易和突变的记录。随着时间的推移，随着我们 DNA 的进化和新的区块被添加到区块链中，两者都变得更加复杂。

每一个区块链(比特币、以太、涟漪)都像是一个独特的物种(人类、黑猩猩等)。).区块链也可以被分叉(就像比特币现金一样)来创造一种竞争性货币，就像两个不同的物种可以拥有共同的祖先一样。

当然，DNA 的变化并不容易发生——科学家认为一个基因突变需要大约 100 万年才能流行起来——建造一个区块链也不容易。进化和自然选择的过程有点像采矿，一系列复杂的步骤创造出不可思议的东西。
