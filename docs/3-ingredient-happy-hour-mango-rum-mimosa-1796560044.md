# 三种成分的欢乐时光:芒果朗姆酒含羞草

> 原文：<https://lifehacker.com/3-ingredient-happy-hour-mango-rum-mimosa-1796560044>

周末快乐，欢迎回到 [*三料欢乐时光*](http://lifehacker.com/tag/3-ingredient-happy-hour) ，每周饮品专栏，提供超级简单又美味的酒水。本周，我感受到了一种贝里尼风格的饮料，它更具热带风情，更有酒味，更有朗姆酒味。



虽然我是桃泥的忠实粉丝，但我也喜欢尽可能少地付出努力，我发现预制的冰沙——如 Odwalla 等——实际上是非常美味的鸡尾酒混合物，而不需要混合或过滤。

除了一点芒果冰沙，这款高杯饮料还含有朗姆酒(因为是夏天)和起泡酒(因为很好喝)。要做到这一点，你需要:

*   1.5 盎司瓶装芒果冰沙，如奥德瓦拉
*   1 盎司白朗姆酒
*   普罗塞克(或其他起泡酒)

冷藏你所有的原料，还有香槟酒杯。将朗姆酒和思慕雪加入长笛并搅拌。顶部放上你选择的起泡酒，小口啜饮，让心情变得更好。
