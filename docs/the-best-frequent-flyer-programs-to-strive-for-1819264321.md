# 争取最佳常旅客计划

> 原文：<https://lifehacker.com/the-best-frequent-flyer-programs-to-strive-for-1819264321>

我经常出差，但更多的时候，我不是订机票的人。这太好了，但这意味着到了年底，我几乎总是离几乎所有主要航空公司的地位相差几千英里。在去年第次发生这种情况后，我开始和帮我订票的人联系，指定我更喜欢乘坐的航空公司，这很有成效。今年，我获得了美国航空公司的精英会员资格，几周后，我还获得了达美航空公司的精英会员资格。



当你是一名普通的飞行员时，航空公司的地位并不是一件大事。当你一直坐飞机的时候，会有很大的不同。常旅客可以得到小至免费托运行李，大至免费升舱的额外待遇。你得到的额外津贴部分取决于你的地位，部分取决于你拥有那种地位的航空公司。

*积分专家* [最近看了一下](https://thepointsguy.com/guide/the-best-airline-elite-status-programs/) 所有的飞行常客计划，以确定哪一个是最划算的。

总的来说，该公司授予阿拉斯加最高荣誉，因为它拥有最好的精英计划。排在第二位的是联合航空公司，接下来是达美航空公司、美国航空公司、捷蓝航空公司和西南航空公司。

阿拉斯加之所以胜出，很大程度上是因为它如何计算自己的精英地位。大多数航空公司都采用了一种方法，你不仅要达到 25，000 英里才能达到他们的底线，你还必须花费 3000 美元。阿拉斯加州没有财政承诺，当你获得身份时，它还会提供更好的里程奖励。

虽然金钱和里程组合应该可以让你在大约同一时间达到状态，但是我的美国状态，比如说，被保持了一个月超过 120 美元。在我达到现金要求的几个月前，我已经超过了 25，000 英里的门槛。在达美航空，我早就超过了钱的要求，但仍差 3000 英里。

对于飞行 25，000 英里、每只梨花费 3000 美元的低层飞行员来说，我在多家航空公司工作，*积分专家*说阿拉斯加的 MVP 项目是最值得争取的，其次是美国航空、达美航空和联合航空的项目，捷蓝航空和西南航空不提供这个级别的项目。

*积分公司* [在其网站](https://thepointsguy.com/guide/the-best-airline-elite-status-programs/) 上有一个工具，你可以输入你的个人旅行习惯，以确定哪个常旅客计划可能适合你。一旦你弄清楚你想去哪一个，它肯定有利于你在该航空公司预订所有的航班。

该网站指出，仅仅因为一个项目基于你的里程和金钱适合你，它可能会因为你的位置而不完美。例如，阿拉斯加有最好的常旅客计划，但如果你的大部分航班都去阿拉斯加不提供服务的地点(这是很多)，那么你最好选择像联合航空或达美航空这样的公司。
