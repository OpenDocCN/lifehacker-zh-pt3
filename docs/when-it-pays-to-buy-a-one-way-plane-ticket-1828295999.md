# 当买一张单程机票值得时

> 原文：<https://lifehacker.com/when-it-pays-to-buy-a-one-way-plane-ticket-1828295999>

据《纽约时报》[](https://www.nytimes.com/2018/06/27/travel/one-way-airfares.html)*报道，单程票正成为国际旅行者越来越实惠的选择。* 

*Watch*

*航空公司过去常常抬高单程票的价格，理由是它们通常是由诺斯替派商务旅行者购买的。但是现在，根据航空公司报告公司提供给《T2 时报》的数据，单程溢价已经从 2014 年的 44%下降到 2018 年的 11%。在同一时期，购买单程票与往返票相比增加了 29%。*

*这种趋势对旅行者来说是有利的，尤其是家庭，他们可以利用更好的价格来获得更大的旅行灵活性:例如，你可以飞到一个欧洲国家，然后乘坐短途火车到附近的国家，在一个新的地方呆几天，然后以更便宜的价格飞离。如果你正在使用 Google Flights 这样的服务来预订机票，你可以将附近的机场添加到你的搜索标准中，以获得更优惠的价格。《泰晤士报》称:“如果你够灵活，找到去欧洲最便宜的入境和出境点，然后通过陆路或低成本航空公司的航班来填补空缺是值得的。”。*

*如果你要去热门的度假目的地，比如希腊或意大利，这尤其有用。*

*据《T2 时报》报道，这种变化已经发生，因为现在有更多的低成本航空公司，如挪威航空和瑞安航空，提供更具竞争力的价格。当然，这也意味着航班可能不会像更昂贵的航空公司提供的那样舒适或提供更多的便利设施(T4 时报指出，“例如，瑞安航空飞机上的食物不倾斜，登机过程可能会很混乱”)。*

*还有一件事你要记住，因为 [石英注意到](https://qz.com/quartzy/1349141/a-one-way-ticket-can-be-a-cheap-way-to-maximize-travel/) :如果你在国际旅行，国家可能会有 [规定禁止](https://www.indietraveller.co/travelling-on-one-way-tickets/) 只有单程票。“即使只是一张火车票，带一些你下一步要去哪里的文件通常也是个好主意，除非你的签证情况允许你无限期地留在这个国家，”Quartz 写道。*

*尽管如此，如果你计划一个夏末或者秋季旅行 ，寻找单程票可能是有意义的——给你机会去更多的国家或城市。*
