# 你可以用这些词来代替“非常”来加强你的写作

> 原文：<https://lifehacker.com/words-you-can-use-instead-of-very-to-punch-up-your-wr-1819968755>

学习如何写作就像学习如何演奏乐器:一旦你学会了基本规则——语法、拼写和标点——并且写出了技术上正确的句子，还有整个语法、措辞和风格的世界需要征服。这也是作家，比如音乐家，有自己的观点的地方:是写简单明了、不加修饰的散文更好，还是编织表达复杂、诗意情感的想象力更好？或者介于两者之间？对我来说，措辞很重要——我钦佩的作家在用词上很精确——我喜欢简洁明了、切中要害的句子。

这是我通过 Reddit 看到我的气窗 [的，所以很自然地引发了一些讨论:首先，谁说“非常完美”了？我们很多人都记得，你不能修饰“独特”这个词——要么是独特的，要么不是，所以跳过“非常”——除非你可以说有些东西*比其他东西*更独特。(就像，每一粒沙子都是独一无二的，但是如果你在一粒沙子中发现了米开朗基罗的大卫的微小比例模型，那将是非常独特的。)此外，有时你真的是指“非常瘦”，而不是“骨瘦如柴”，这个词显然与“瘦”有着不同的内涵最后，很多这种现象并不普遍，就像自助餐厅可能真的会震耳欲聋，但孩子们自己可能不会。](https://www.reddit.com/r/coolguides/comments/776o3r/words_to_use_instead_of_very/)

尽管如此，这仍然是一个值得考虑的有趣列表，如果只是为了提醒作者，搜索准确的单词，就像搜索吉他上准确的音符一样，是什么让写作变得有趣甚至非常漂亮。我是说，很漂亮。
