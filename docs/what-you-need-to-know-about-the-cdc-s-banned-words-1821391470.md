# 关于疾病预防控制中心的“禁用”词，你需要知道什么

> 原文：<https://lifehacker.com/what-you-need-to-know-about-the-cdc-s-banned-words-1821391470>

周五， [《华盛顿邮报》](https://www.washingtonpost.com/national/health-science/cdc-gets-list-of-forbidden-words-fetus-transgender-diversity/2017/12/15/f503837a-e1cf-11e7-89e8-edec16379010_story.html?utm_term=.49c670b5d240) 报道称，疾病预防控制中心官员告诉工作人员在预算文件中保留七个术语:胎儿、多样性、跨性别、弱势、权利、基于科学和基于证据。这些话对大多数科学家来说是就事论事，但对政治家来说，它们是政治性的。



疾控中心主任 [澄清说，并没有全面“禁止”](https://www.statnews.com/2017/12/17/cdc-chief-science-forbidden-words/) 这几个字，但并没有否认它们被排除在预算文件之外。一位匿名的卫生和人类服务官员告诉 STAT，他们选择不使用这些词，因为 CDC 认为他们的预算文件没有这些词会有更好的机会。

以下是这些词的意思，以及对它们为何现在成为政治弃儿的一些猜测:

胎儿:孕妇子宫里的那个东西在医学上被认为是 T2，T3 胎儿，T5 胎儿，或者更年轻的时候，胚胎，但是共和党议员在试图争辩堕胎应该是非法的，或者胎儿应该可以开银行账户时，经常喜欢用“未出生的孩子”这样的术语。

**多样性**:所有不同种族和其他人口统计数据的人都居住在我们的国家，CDC 的网站包含了关于 [为什么多样性在人员配备](https://www.cdc.gov/about/diversity/diversity.htm) 中很重要以及为什么我们在研究公共卫生 时必须 [关注多样性的陈述。也许谈论确保所有美国人都能获得医疗保健已经不再合适了——尽管少数群体的人健康状况更差，更难找到好的医疗保健。](https://www.cdc.gov/minorityhealth/internships/2016/whydiversityinclusionmatterspublichealth.pdf)

脆弱的:一些人群比其他人群更容易受到健康问题或自然灾害等风险的影响。疾控中心 [将紧急情况下的弱势人群](https://www.cdc.gov/diabetes/programs/vulnerable.html) 描述为那些沟通有困难，或者在没有帮助的情况下无法自行运输，或者需要持续监护的人。这可能包括老年人或残疾人。美国疾病控制和预防中心也承认那些特别容易感染传染病、糖尿病的人群。我很难理解为什么这是一个被禁止的词，除非它与医疗保健的不平等有关。今年早些时候，特朗普的政策重点经常被指责为让更多人容易受到某些疾病风险的影响。

**跨性别者:**如果你的性别身份与你出生时被指定的性别不同，你就是 [跨性别者](https://www.glaad.org/transgender/transfaq) 。跨性别者可能在获得医疗保健和寻找能够为他们提供良好护理的提供者方面面临额外的挑战(例如，认识到一个跨性别者可能仍然需要宫颈癌筛查)。疾病预防控制中心有 [资源给跨性别者和他们的看护者](https://www.cdc.gov/lgbthealth/transgender.htm) 。特朗普政府的一些政策已经干预或威胁干预跨性别医疗保健。

权利:政府项目被称为权利 [，当他们给予任何有资格](http://www.auburn.edu/~johnspm/gloss/entitlement_program) 的人援助时。目前，医疗保险、医疗补助、社会保障、许多退伍军人管理计划和福利计划(如 SNAP)都是作为权利运作的。当医疗保健和现在的税收法案正在讨论时，这些计划已经被放在砧板上。

**以科学为基础的**和**以证据为基础的**:这些术语指的是经过严格研究 和可靠数据来源的 [政策。例如，科罗拉多州提供免费宫内节育器，意外怀孕和流产率直线下降。他们还节省了数百万美元。其他以性教育为中心的基于证据的青少年怀孕项目也给出了有意义的结果。但是在政治上，一些宗教团体想要审查性教育。据报道，疾病预防控制中心告诉其分析师，不要使用这些短语，他们应该说“疾病预防控制中心的建议基于科学，考虑到社区标准和愿望。”](https://en.wikipedia.org/wiki/Evidence-based_policy)

如果科学能远离政治就好了。但是既然那是不可能的，如果政治能够远离科学，那将会非常非常好。
