# 桌面桌面

> 原文:[https://lifehacker.com/the-desktop-desktop-1786063864](https://lifehacker.com/the-desktop-desktop-1786063864)

我对那些看起来像台式机的台式机情有独钟。亚历克的桌面非常符合要求，实际上看起来像一个桌面，物品放在角落里，但仍然有足够的空间来实际工作，并进行一些定制，使其个性化。

Watch

如果您想在 Mac 上获得类似的外观，以下是您需要的组件:

*   [我的书桌模拟壁纸由 r . r .](http://techandall.com/my-desk-mock-up/)T3】
*   [über sicht](http://tracesof.net/uebersicht/)定制和 OS X 的 widget 结尾，其中 [我们在](http://lifehacker.com/ubersicht-customizes-your-os-x-desktop-with-widgets-and-1591137635) 之前已经强调过了
*   über sichtT3 的 [侧边栏小部件](http://tracesof.net/uebersicht-widgets/#Sidebar)
*   [为über sicht](http://tracesof.net/uebersicht-widgets/#morning)的晨钟小部件
*   über sichtT3 的 [励志语录小工具](http://tracesof.net/uebersicht-widgets/#inspirational-quote)
*   über sichtT3 的 [倒计时小工具](http://tracesof.net/uebersicht-widgets/#countdown)
*   über sicht 的 [YouTube 视频小工具，它实际上是在屏幕底部的 iPad 屏幕上播放](http://tracesof.net/uebersicht-widgets/#youtube-video) [这段视频](https://www.youtube.com/watch?v=28ji6G5hTRw) ，只是被屏幕底部遮住了
*   用于 Mac 的 [LiteIcon](https://freemacsoft.net/liteicon/) 实用程序，可快速更改系统图标
*   [全白图标包](http://noshery.deviantart.com/art/All-White-Icons-for-OSX-470738808) 为 OS X
*   [cDock](http://w0lfschild.github.io/app_cDock.html) dock 定制工具
*   为 Mac 定制菜单栏的实用程序

那是很多移动部件，但是这是一个特别深度定制的桌面。即便如此，这些部件还是很好地组合在一起，如果你不是从外观上了解的话，你不会想到里面放了这么多小部件。如果你对此有任何疑问，或者想知道 Alec 是如何让一个小部件如此工作的，或者它是如何工作的，请访问他的 Kinja 博客(链接如下)提出你的问题。
