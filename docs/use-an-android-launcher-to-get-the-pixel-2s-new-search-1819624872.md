# 如何在当前手机上安装 Pixel 2 的新搜索栏

> 原文：<https://lifehacker.com/use-an-android-launcher-to-get-the-pixel-2s-new-search-1819624872>

[Pixel 2](https://lifehacker.com/how-does-google-s-pixel-2-stack-up-to-its-predecessor-1819148085) 配备了 [新功能](https://lifehacker.com/the-pixel-2s-biggest-gimmick-might-actually-be-its-best-1819163006) 你不会在普通的 Android 8.0 上找到，其中很大一部分是主屏幕。谷歌重新设计了日历小工具来显示即将到来的事件，因此它将搜索栏移到了屏幕底部的 dock 下方，这样你可以更快地访问它。

Watch

如果你一直在跟进，你已经有了 [新的日历小工具](https://lifehacker.com/how-to-get-the-pixel-2s-handy-new-calendar-widget-on-yo-1819372471) 在你当前的手机上运行。现在你也可以通过使用第三方应用程序启动器来获得重新定位的搜索栏。

### 获得像素 2 搜索栏与新星发射器测试版

Nova Launcher 是最受欢迎的 Android 发射器之一，这要归功于大量的定制选项和快速的性能。版本 5.5 的 [最新测试版](https://help.teslacoilapps.com/beta) 也为旧手机增加了一些来自 Android 8.0 和 Pixel 2 的新功能。

最令人兴奋的变化是新的搜索栏，你可以将它放在 Pixel 2 的应用程序 dock 下方，或者定制它放在 dock 上方。要设置它，请转到 Nova 设置，然后停靠，然后搜索栏。你也可以通过跳回 Nova 设置，选择 Dock，然后选择搜索栏样式来调整它。

最新的 Nova Launcher 测试版还增加了 Android 8.0 的新弹出菜单——这是你长按主屏幕、应用程序或文件夹时看到的菜单。最后，当你在主屏幕上拖动应用程序时，你会看到谷歌新的图标动画。

按照这里 的指示 [让新的测试版开始运行。](https://help.teslacoilapps.com/beta)

### 动作启动器提供了更多的定制选项

如果你更喜欢 [动作发射器](https://lifehacker.com/action-launcher-pro-puts-widgets-folders-and-shortcut-5993574) 而不是 Nova，你还是很幸运的。在 Pixel 2 发布后不久，该应用程序实际上在本月早些时候添加了谷歌的新搜索栏。它还打包了大量额外的定制选项，允许你改变搜索栏的形状和添加其他快捷方式。

最新版本的 Action Launcher 还提供了一些其他新功能，包括 Google Now feed、通知点和一个新的天气窗口。你现在可以从 [Play Store](https://play.google.com/store/apps/details?id=com.actionlauncher.playstore) 抓取 29.0 版本的 app launcher，只要你运行的是 Android 5.0 或更高版本。
