# 制作一个情感转盘来帮助孩子表达他们的感受

> 原文：<https://lifehacker.com/make-an-emotion-wheel-to-help-kids-express-their-feelin-1798665324>

小孩子有很多重大的感受，这些感受会在你说“你想吃草莓酱三明治吗？”时迅速而戏剧性地发生变化。"哦，亲爱的，我们只剩下面包的两端了。"但是他们不能总是用语言表达他们的感情，所以他们经常在发脾气和流泪的时候表达。爸爸妈妈的催促— *冷静下来，我们来谈谈，说说你的感受，你不爽吗？*——只会让他们更加沮丧。



情绪轮是一个简单的沟通工具，你可以在瞬间完成。这里有一个 [的免费模板](https://kristinamarcelli.wordpress.com/2014/03/02/feelings-check-in-wheel/) ，或者你也可以轻松地绘制自己的模板。拿一些纸或卡片纸，剪出一个圆圈，然后在它周围画出不同情绪的脸。快乐，平静，紧张，愤怒，悲伤，吃了太多的糖，无论你能想到什么。然后剪下一个小箭头，用扣件系上。如果你觉得有趣，你可以拍下你的孩子表现出各种不同情绪的照片，然后把这些照片贴在方向盘上。

每当你的孩子需要表达他的感受时，他可以将箭头指向适当的情绪。(或者他可以指出多种感觉——如果我们从*里到外*学到了什么，那就是我们是复杂的生物。)在许多早晨，我都可以为自己使用情绪轮。别跟我说话。请检查一下轮子。
