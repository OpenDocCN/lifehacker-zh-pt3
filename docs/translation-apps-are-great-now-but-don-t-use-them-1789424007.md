# 翻译应用现在很棒，但是不要用

> 原文：<https://lifehacker.com/translation-apps-are-great-now-but-don-t-use-them-1789424007>

由于像神经网络学习 这样花哨的新升级 [，像谷歌翻译和微软翻译这样的翻译应用比以往任何时候都好。他们可以翻译你想要的任何东西。但是你不应该在旅行的时候依靠它们。事实上，我认为你应该尽可能少地使用它们——如果要用的话。](https://lifehacker.com/what-neural-networks-artificial-intelligence-and-mach-1789259060) 



Google Translate 和微软 Translator 都已经把 [进化成了比](http://lifehacker.com/translation-tool-showdown-google-translate-vs-microso-1787836106) 更称职的翻译工具。他们几乎可以即时翻译语音，离线阅读和翻译文本，甚至可以使用智能手机的摄像头实时翻译标志和菜单。使用这些应用中的任何一个，你都可以毫不费力地在异乡漫游。但是如果你的目标是从旅行中获得尽可能多的东西，使用它们对你自己和你的整体体验都是一个巨大的伤害。

当你去看你家大门外的世界时，语言不仅仅是一个需要克服的障碍。这是你开始理解一种文化的方式。正如作家 [丽塔·梅布朗](https://en.wikipedia.org/wiki/Rita_Mae_Brown) 所说，“语言是一种文化的路线图。它告诉你它的人民来自哪里，他们将去哪里。”因此，仅仅依靠一个翻译应用程序，你就立刻失去了理解的一个关键要素。

但是如果你做一点功课，试着掌握一些语言基础，你就可以接触到真正的人际关系。你从游客转变为旅行者，展现出无与伦比的尊重、接纳和文化欣赏，这不会被忽视。对你来说，读出新单词可能会很尴尬，但对他们来说却很温暖。

在我最近的日本之旅中，我亲身体验了这一点。当我到达的时候，我已经做好了随时掏出手机的准备。但我很快意识到，我正在通过一个过滤器来观察一种迷人的文化。第一天结束后，我把手机放在背包里，决定用走之前学的那点点日语。当我在火车上和酒店休息时，我会查阅随身携带的小常用语手册，但我非常热衷于尝试的翻译应用程序再也没有出现过，我对我的选择非常满意。

在一次偶然的机会中，我和涉谷的一位漂亮女士开始了一段对话，只是简单地说了一句我手里拿着的甜瓜面包很好吃。我阻止了令人讨厌的坚持不懈的新宿街头推销员——这是我可以说“皮条客”的最好方式——用一些关键短语来骚扰我。我得到了帮助，在秋叶原一个繁忙的七层购物中心找到了一件很难找到的东西。我无意中尝试并享受了几种食物和饮料，如果我翻译了菜单，我可能会避免这些食物和饮料。当我惊喜地对她说“晚上好”、“我很喜欢这些”和“非常感谢你”时，我看到 kombini 脸上露出了忧郁的表情。

如果我把所有的语言都留给我的智能手机，这些都是一时冲动的体验，不会发生。当然，我仍然会有一次愉快的旅行，但我会错过所有真正困扰我的小事。当你旅行时，你会看到许多壮观的地方和事物。你可以拍下这些地方和事物的照片，这样你以后就可以看到它们，但我发现一种文化中日常生活的细节要有趣得多，也更令人难忘，这些才是你真正旅行的经历——否则你可以在家看照片。总有一天我会忘记二条城 是什么样子，但我永远不会忘记我在京都与一位制刀大师关于雕刻一把新做的刀刃的对话。

此外，花时间让自己沉浸其中让我意识到实际学习一门新语言是多么的可能。发展流利的第二或第三语言看起来像是一座需要攀登的陡峭的山，但是即使是突破最小的语言障碍也会打开你的思维，让你有可能超越这些基础。当推动你跨越这些障碍时，没有什么比必要性和紧迫性更重要了。当你把智能手机放在背包里时，你就有了依靠你所学知识的时间。它迫使你努力变得更好。当你意识到你实际上有多能干时，这会给你学习更多的信心。在旅行的推动下，我决定在空闲时间学习日语的其余部分，以及它的三个字母。

如果你在玩视频游戏，使用翻译应用程序进行所有互动是最简单的设置。这听起来可能很愉快，没有压力，但你最终只会走过场，直到你意识到这一点时，你已经结束了。没有挑战，没有思考，也没有冒险——这意味着它不会那么令人难忘。

你不必像我一样把难度提高到“难”，完全避开翻译工具，但至少玩“正常”准备好翻译应用程序，以防你真的需要它们，但是帮自己一个忙，尝试让自己沉浸在文化中。当你说一个人的语言时——即使只说一点点——你邀请了原本不会有的可能性。
