# 什么会让你真的去看电影？

> 原文：<https://lifehacker.com/what-would-make-you-actually-go-to-the-movies-1821605002>

人们越来越少去看电影了。收入增加了，但上座率下降了，影院感到了压力。2017 年夏季票房是该行业十年来票房收入最差的一年， [*综艺*报道](http://variety.com/2017/film/news/summer-box-office-2017-worst-decade-1202548272/) ，销售额比 2016 年夏季下降了 14.6%。

Watch

许多因素你都可以猜到:网飞、亚马逊和付费有线电视频道正在制作高质量的电影，这些电影是“免费的”，不需要你离开你的房子。门票和爆米花价格持续上涨。在你精疲力尽之前，你只能看那么多蜘蛛侠的故事。

彭博 [调查了](https://www.bloomberg.com/news/articles/2017-12-26/material-world-movie-theaters-want-you-back-so-they-re-rushing-to-modernize) 影院为赢回顾客正在做什么，发现他们正在投资豪华座位和更多的食物和饮料选择。AMC 表示，可能会开始销售与电影相关的商品。

 [https://lifehacker.com/embed/inset/iframe?id=twitter-945725663708942336&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-945725663708942336&autosize=1) 

这一切都很好，但对我来说，最大的因素是不想在纽约市支付 17 美元的单程票(我最近得到了电影通行证 ，这很棒，尽管我不确定这笔交易可能会持续多久)，以及提供除超级英雄/漫画故事之外的其他东西。给我更多的浪漫喜剧。给我儿童免费观看(抱歉，父母们，我不想听到你们 6 岁的孩子在*最后的绝地*中问什么是原力)。

你呢？你想要更多躺椅还是更便宜的票？还有别的吗？
