# 三星 Galaxy Note 8 评测:更大，更坏，但我还是想要更多

> 原文：<https://gizmodo.com/samsung-galaxy-note-8-review-bigger-badder-but-i-sti-1799759566>

原谅我错过了把电池爆炸的笑话击垮的机会。让我们开门见山吧:三星曾经占据主导地位的旗舰平板手机在阔别两年后又回来了。新的 Galaxy Note 8 配备了令人惊叹的 18.5:9 超宽屏显示器，比以往任何时候都更大，更有吸引力，除了命运多舛的 Galaxy Note 7 之外，它还配备了比任何其他 Note 都更大的电源组。但自 2015 年以来，情况发生了很大变化——尤其是当你考虑到新款 Note 8 的起价为 930 美元时，它现在比以往任何时候都要贵。这意味着它需要回答两个问题:Galaxy Note 8 在 2017 年仍有意义吗？它可能值近 1000 美元吗？



乍一看，答案可能是否定的。Note 8 的 6.3 英寸 2960 x 1440 super AMOLED 显示屏比 S8 Plus 的 6.2 英寸屏幕大不了多少(尽管无边框设计仍然非常漂亮)，其高通骁龙 835 处理器、64GB 存储空间和内置 microSD 读卡器也没有什么不同。除了略显四四方方的边角，Note 8 甜美的玻璃和铝制机身也大同小异。你甚至可以在音量摇杆下面找到同样烦人的 Bixby 按钮，它仍然不能被重新映射到有用的东西。

一个小小的改进是三星将 Note 8 的内存从 4GB 提升到 6GB，这是对 Galaxy Note 传统超级用户粉丝群的一个很好的认可。然而，在日常生活中，这种改进并不总是那么明显(尽管基准测试表明 Note 8 比其较小的兄弟姐妹领先一点点)。这意味着这一决定实际上是对三星标志性的 S Pen 手写笔和该公司有史以来第一个双摄像头模块的对冲。

现在，我将是第一个说，经过多年对移动操作系统的改进，我们已经达到了一个点，用户界面是为基于触摸的导航而设计的。在 iOS 和 Android 中，你从来没有尖叫着想要鼠标或者更精确的东西。对我来说，这削弱了手写笔的吸引力。但在使用 Note 8 一个多星期后，S Pen 开始赢得我的心。你可以做的事情实在太多了，无论是在不解锁手机的情况下记笔记，还是使用新的整句翻译功能来弥补我蹩脚的普通话阅读理解能力。但奇怪的是，S Pen 最好的部分是它有多种方法可以让你制作自己的 gif。

选择一是拿出 S 笔，点击智能选择，在你能在屏幕上看到的任何视频上画一个矩形。嘣。gif 制作。选项二，实时消息功能，是 2017 年的新功能。Live Messages 可以让您在自己选择的背景颜色上绘制动画，甚至可以在您拍摄的图片上绘制动画。这是一个虚拟的迷因制造机器，也是一部手机的快乐附加物，这种手机经常出现在不被允许娱乐的公司高管手中。但这仍然只是回答了为什么你应该买 Note 8 而不是更便宜的 S8 的一半问题。

等式的另一部分是三星新的双后置摄像头，它将 S8 和 S8 Plus 摄像头与第二个 1200 万像素的传感器配对，焦距是第一个的两倍。这是一个我们已经在去年的 iPhone 7 Plus 中见过的相机设置。三星在组合中加入的一点特殊酱料是两个相机上都有光学图像稳定功能。这是一个受欢迎的改进，因为随着距离变得更远，相机抖动往往会成为一个更大的麻烦。不幸的是，即使有了这个新功能，Note 8 的相机性能也不是完美无缺的。

第一个问题是，就像 iPhone 7 Plus 一样，你实际上无法完全控制你使用的镜头。有时在弱光环境下，按 2x 按钮进行变焦会产生数码增强效果，而不是光学效果。这意味着你只是裁剪并得到一个较低分辨率的最终图像，而不是使用两倍焦距的镜头。

至少不是 Note 8 特有的问题。有时当 Note 8 选择数码变焦而不是光学变焦时，iPhone 没有。而其他时候，情况正好相反。真正令人沮丧的是，这两款手机都无法超越握着你手的硅保姆。

然而，即使在理想的条件下，Note 8 也并不总是成功。举个例子，在最近的一场洋基队的比赛中，当我试图给纽约最喜欢的大儿子 拍一张照片时，我得到了一些照片。起初，我非常激动，Note 8 的变焦功能让我近距离接触个人，这是许多手机相机无法做到的。但当从 Note 8 和 iPhone 7 Plus 并排看照片时，情况很接近，但 iPhone 的照片更清晰，不那么突出。

但当你付出一点努力，Note 8 仍然可以令人眼花缭乱。我被我用三星的新相机模块拍摄的纽约夜景所震撼。我用 DSLR 预算拍了这张照片，却没有得到这么好的东西。它不仅让我从变焦镜头中获得了一切，它的照片更清晰，噪音更少，而且比我从 iPhone 7 Plus 中获得的最佳照片更丰富多彩。

最重要的是，三星有一个新的实时对焦模式，可以让你调整景深，就像你在大 DSLR 或无反光镜相机上一样。这让你可以选择人像摄影师喜欢的乳脂般模糊的背景，或者根据你的需要从前到后一直保持清晰。你仍然可以获得超饱和的色彩和锐利的焦点，这是三星相机闻名的地方。

除了相机和手写笔，三星还有一些其他的小惊喜，比如新的应用程序对功能，让你可以创建组合，只需一次触摸就可以在多窗口模式下启动两个应用程序——非常适合任何想要在关注地图的同时从电子邮件或文本中找到地址的人。尽管电池容量实际上比 S8 Plus 的电池容量小(3，300 毫安时对 3，500 毫安时)，但 Note 8 在我们的耗电测试中的时间为 10 小时 21 分钟，仅比 S8 Plus 的 10:44 短 20 分钟。有了这样的东西，Galaxy Note 8 开始成为一个很好的案例，但我仍然不确定。

Galaxy Note 8 应该是一款过剩的手机。它应该拥有所有最令人发指的技术，同时推动其他公司做梦也想不到的界限。然而，它的屏幕只比 S8 Plus 大一点点，看起来也没什么不同。尽管我喜欢 S Pen 的许多新功能，如实时信息和整句翻译，但它们实际上只不过是一些方便的软件技巧。

三星新的双摄像头模块甚至不是去年 iPhone 7 Plus 的明确赢家，更不用说苹果计划下周发布的任何新手机了。 因此，尽管从屏幕到手写笔都配备了一流的组件，三星的平板手机似乎还是有点平淡无奇。如今，Note 8 唯一真正过分的是它的价格。

现在，我不是说人们不应该得到一个，这将是荒谬的！我的意思是，如果你想觉得你的钱花得值，你真的需要找到一个交易。运营商已经在兜售买一送一的促销活动，这将使购买两台 Note 8 感觉像是偷窃。三星甚至提供自己的预购奖励，你可以选择 Gear 360 或 128GB microSD 卡和无线充电器。但是支付 930 美元或更多是愚蠢的。Note 8 是一款很棒的设备(尤其是屏幕)，我认为三星已经理所当然地重新夺回了它在 Android 阵营中的位置。但是对于这么贵的手机，我还是想要更多。

### 自述文件
