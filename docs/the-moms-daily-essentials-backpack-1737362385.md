# 妈妈的日常必需品背包

> 原文：<https://lifehacker.com/the-moms-daily-essentials-backpack-1737362385>

当你是一位母亲，带着孩子乘坐公共交通工具在城里四处奔波时，你需要把各种必需品放在一个小包里。Lifehacker 读者[evebeebee](http://kinja.com/evebeebee#_ga=1.63188775.968941705.1436971740)[分享了她做那件事的袋子](http://evebeebee.kinja.com/dc-mom-s-go-bag-adults-carrying-backpacks-is-a-thing-1736763516) 。



包里是一个 [高的塞拉背包](http://shop.highsierra.com/backpacks) 。这是里面的典型内容:

*   [华硕笔记本带充电器](http://www.amazon.com/X551MA-Laptop-Intel-Celeron-500GB/dp/B00SV72ZA2/ref=sr_1_4?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-moms-daily-essentials-backpack-1737362385&asc_source=&ie=UTF8&keywords=asus laptop&qid=1445276356&s=pc&sr=1-4&tag=kinjalifehackerlink-20)
*   安卓充电器
*   一个 5.5 x 8.5 的素描本

*   我的计划器(一个用订书钉装订的小活页夹，里面装满了绘图纸)

*   [微米笔](http://www.amazon.com/MICRON-Drafting-Engineering-General-Catalog/dp/B004IMJOZ8/ref=sr_1_1?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-moms-daily-essentials-backpack-1737362385&asc_source=&ie=UTF8&keywords=micron pens&qid=1445276422&s=pc&sr=1-1&tag=kinjalifehackerlink-20)
*   铅笔、卷笔刀和橡皮

还有婴儿用品:

*   一种防水卷式换衣垫
*   尿布和湿巾
*   [苏菲长颈鹿](http://www.amazon.com/Vulli-Sophie-giraffe-Natural-Rubber/dp/B000IDSLOG/ref=sr_1_1?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-moms-daily-essentials-backpack-1737362385&asc_source=&ie=UTF8&keywords=sophie the giraffe&qid=1445276448&s=baby-products&sr=1-1&tag=kinjalifehackerlink-20)T3】
*   [勺茧自缚](http://www.babyintow.com/)
