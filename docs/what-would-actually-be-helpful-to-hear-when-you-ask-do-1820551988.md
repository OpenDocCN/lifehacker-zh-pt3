# 当你问“我看起来还好吗？”时，听到什么实际上是有帮助的

> 原文：<https://lifehacker.com/what-would-actually-be-helpful-to-hear-when-you-ask-do-1820551988>

哦，在一个盛大的夜晚到来之前，这个问题很尴尬。我们都问过或被问过“我看起来还好吗？”或者“我看起来怎么样？”在我们生命中的某个时刻。对许多人来说，这只是一个增强信心的简单呼吁。对其他人来说，这是一个实际问题。我们应该说什么？

Watch

默认的答案似乎总是某种形式的“你看起来棒极了！”或者“完美！”这肯定是最安全的回答，但是如果提问的人真的想知道他们的发型选择，这就没什么帮助了。撒谎让他们出去看起来有点傻可以吗？尽管如此，当那个人真的刚刚确认你愿意和他们一起去公共场合，不管他们穿什么，指出他们的小缺点可能是危险的。我知道当我问的时候，我很认真地想知道我大胆地选择穿一件鲜艳、多彩的衬衫配一件单调、深色的外套是否真的有用，或者我根本没有时尚感。或者我担心我的头发看起来像我想的那样糟糕。

所以让我们一起想出一个乐于助人，善良的方法。在你问“我看起来还好吗？”之后，你最想听到什么？那不是很安全吗？或者，当你想要一些善意但诚实的反馈时，有什么更好的方式来问这个问题呢？请在下面的评论中发表意见！
