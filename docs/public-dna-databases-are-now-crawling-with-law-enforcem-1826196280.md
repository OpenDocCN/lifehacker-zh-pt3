# 公共 DNA 数据库现在充斥着执法部门，我们最好习惯这一点

> 原文：<https://lifehacker.com/public-dna-databases-are-now-crawling-with-law-enforcem-1826196280>

执法人员和基因系谱学家没有浪费任何时间 [公共 DNA 数据库导致金州黑仔上个月怀疑](https://vitals.lifehacker.com/the-golden-state-killers-dna-was-in-a-publicly-availabl-1825604590) 。一家取证公司宣布了一项服务来进行这种分析*全体，*并且 DNA 数据库 [GEDmatch](https://www.gedmatch.com/) 已经改变了它的隐私政策以允许执法部门使用它。记住，即使你不在这些数据库中， [你的表亲也可能是](https://lifehacker.com/you-probably-already-have-cousins-in-a-dna-database-1825892173) 。



取证公司 [Parabon NanoLabs](https://parabon-nanolabs.com/) 告诉 Buzzfeed，他们已经 [上传了大约 100 份犯罪现场样本](https://www.buzzfeed.com/peteraldhous/parabon-genetic-genealogy-cold-cases?utm_term=.guXeVGDjN#.mhxvL8VyD) 到 GEDmatch，以寻找罪犯和身份不明的犯罪受害者。他们的服务获取执法部门提供的样本，并以类似于 23andMe 和 Ancestry 所做的 的方式 [处理 DNA。结果是他们可以上传到 GEDmatch 的数据文件。](https://vitals.lifehacker.com/how-dna-testing-services-work-1825759291)

GEDmatch 在研究家谱的人和寻找亲生父母的成年领养儿童中很受欢迎。你从一家检测服务机构上传你的数据文件，系统会帮你找到其他与你 DNA 匹配的人。这是一个很小的数据库，所以你不太可能直接找到，但有可能你至少会找到一些远亲来帮助缩小你的搜索范围。

还有其他的 DNA 数据库，可以肯定的是，执法和取证公司正在尽可能地采用类似的策略。

### 下一步是什么

一个月一百个样本不算什么。如果这种技术奏效，它似乎有可能成为全国警察部门的例行公事。GEDmatch 最近更新了其政策 ，明确允许执法部门搜索数据库，但有一些限制:

> 当您将原始数据上传到 GEDmatch 时，您同意原始数据是以下数据之一:
> 
> *   你的 DNA
> *   你是其法定监护人的人的 DNA
> *   某人的 DNA，该人已授予您将他们的 DNA 上传到 GEDmatch 的特定授权；
> *   你所知道的死者的 DNA
> *   执法部门获取并授权的 DNA 可以:(1)确定对另一个人实施暴力犯罪的罪犯；或(2)识别已故个人的遗体；
> *   人造 DNA 试剂盒(当且仅当:(1)用于研究目的；以及(2)它不用于标识 GEDmatch 数据库中的任何人)；或者
> *   从人工制品中获得的 DNA(当且仅当:(1)你有合理的理由相信原始数据是来自人工制品的前所有者或使用者的 DNA，而不是来自活着的个体；以及(2)你知道该人工制品的前拥有者或使用者已经死亡)。
> 
> “暴力犯罪”被定义为杀人或性侵犯。

但是，GEDmatch 可以随时更改策略。

一方面，杀人犯和强奸犯可以被绳之以法，家人可以结束他们失踪的亲人，这很酷。另一方面，我从未同意将我的 DNA 放入执法部门的数据库——但现在它的片段可能已经在那里了，这要感谢第三、第四或第五代表亲，他们不知道他们的祖先档案会被这样使用。
