# 如何阻止前任闯入你的账户

> 原文：<https://lifehacker.com/how-to-stop-an-ex-from-breaking-into-your-accounts-1829056513>

在本周的 [技术支持专栏](https://lifehacker.com/c/tech-911) 中，我将讨论一个令人不安的问题:如何从不友好的前任手中夺回对你账户的控制权。我希望你以前的爱人不是一个彻头彻尾的精神病患者，或者，至少，不是一个有权进入你账户的精神病患者，但这是一个再熟悉不过的故事。你和某人住在一起，你分享你的希望和你的梦想，他们想办法进入你的账户。(或者，你共享登录凭证，这也是一个非常糟糕的主意。)

Watch

在我开始之前，先提一个小问题:尽管 Lifehacker 的读者 [给我发了很多问题](https://lifehacker.com/send-us-your-tech-questions-1828824699) 让我在以后的帖子中解决——谢谢你！—前几天在我最喜欢的 subreddit，/r/legaladevice 看到了 [这个帖子](https://www.reddit.com/r/legaladvice/comments/9exm7e/ms_my_ex_continues_to_hack_into_my_phone_various/) ，一直挥之不去。作者在信中写道:

> “[我的前夫]不断侵入我的电话、电子邮件、工作档案和社交媒体。我发现之前的一部手机上有 50 多个隐藏的应用程序，我看不到也无法下载。我又买了一个，现在我知道这个上面有 3 个，我看不到也脱不下来。他从未接触过我的手机。他拿走了我已经删除的旧电子邮件账户，把它们变成了一个公司账户，并链接到他的商业账户。当我试图进入我刚用了一个月的新邮箱时，所有的东西都是克罗地亚语。我收到了混乱的 facebook 信息，经过一点调查后，我发现他不知何故做到了每次我把车开进车道时他都会收到通知。不知何故，他在我之前就收到了我所有的 facebook 信息……”

虽然解析起来有点棘手，但原海报故事的主旨是，她的前夫正在她的生活中造成各种各样的数字混乱。虽然这可能是账户和设备操作出错的一个极端例子，但还没到*那个*极端。如果你曾经给前任一个服务的密码，而你很懒，在其他服务上重复使用你的密码，事情很快变糟并不难——特别是如果你没有很多账户安全，加倍——特别是如果他们是一个可怕的，报复心强的人。

有很多方法可以解决这个问题，我将尝试从“最不讨厌”到“证人保护计划状态”的顺序进行

### 更改您的密码

显然，如果有人使用您的密码登录您的服务(或设备)，请更改密码。不要让你的新密码太显眼。不要重复使用您当前或以前使用的任何其他密码。如果你正在使用 LastPass one 1Password 等密码管理应用程序来帮助生成或跟踪你的密码，也请在*和*上更改你的主密码。

一旦你在你的主要社交媒体网站、云存储服务、电子邮件账户、银行和信用卡账户、工作登录、 [你的智能家居设备账户](https://lifehacker.com/how-to-take-back-control-of-your-smart-home-devices-fro-1827974017) 以及你的笔记本电脑或台式机上更改了密码，请确保你还检查了你的服务是否允许你查看已经登录你的账户的任何其他系统或会话。更改您的密码应该可以防止以前登录过的人重新获得访问权限(比如说，如果你们分手了，而您的前任要求控制您的共享笔记本电脑)。如果您可以撤销以前登录过您的帐户的任何设备的访问权限，请这样做。安全无妨。

此外，这是一个很好的方式来查看别人是否已经设法访问您的帐户，即使您已经更改了您的密码。如果是这样，是时候变得更狡猾了。

### 为您的设备使用更强的登录方法

如果你仍然和某人住在一起，或者你担心有人会出于某种原因从物理上访问你的设备，考虑换一个更强的认证系统。如果你的智能手机或平板电脑支持指纹或面部识别，就使用它——除非你认为有人会在你睡觉时把你的智能手机举在你面前，或者类似的奇怪事情。

您还可以切换到基于硬件的安全密钥，如 [Yubikey](http://r) ，这将阻止某人访问支持这些设备*的服务，即使*他们拥有您的登录凭据。没有钥匙，不能登录。就这么简单。

### 使用帐户警报和双因素身份验证

只要有可能，你至少应该使用两步认证——通常是以服务的形式向你的智能手机发送一条带有代码的文本消息，当你或其他人试图在无法识别的设备上登录你的帐户时，必须输入该代码。然而，如果一个狡猾的人 [利用他们可能知道的你的个人信息劫持了你的手机号码](https://www.wired.com/story/two-factor-authentication-apps-authy-google-authenticator/) ，这仍然会使你易受攻击。你最好使用认证应用，然后要求他们登录你的设备(如果你使用我们前面提到的建议锁定了它，他们应该不能这样做)。

当你这样做的时候，你应该检查你的各种服务是否在未知设备试图访问你的帐户时提供登录提醒。如果你开始一次收到一堆这样的邮件，很明显有人试图扰乱你的数字生活，这可能值得你再三检查，以确保一切尽可能安全。(例如，如果试图访问你账户的人来自同一个 IP 地址，并且这与你前任的居住地相关，你可能会有一个很好的线索，知道谁可能是这些尝试的幕后黑手。)

### **关闭位置追踪功能**

如果你一直在使用谷歌的 [位置服务](https://lifehacker.com/how-to-find-and-delete-your-google-maps-history-1828312219) ，或者你没有仔细研究过谷歌地图或苹果的“ [找到我的朋友](https://support.apple.com/en-us/HT201493) ”等应用程序，考虑一下你的主要应用程序和服务，看看是否有什么方法可以限制关于你确切位置的信息量。如果你的前任很狡猾，他们可能会在你的地图应用程序或其他应用程序中将自己添加为“朋友”，这可以让他们在任何时候看到你在任何地方。不太好。

### 重置您的设备

你可以更改世界上所有的密码，但如果有人在你的笔记本电脑、台式机或其他设备上安装了键盘记录器或远程访问工具，这将成为一场猫捉老鼠的游戏。如果您仍然注意到不是您的人正在登录您的帐户，即使在您设置了 2FA 和新密码之后，您可能需要考虑备份您的关键信息并重置您的设备的出厂设置。

这应该会清除其他人可能安装的任何烦人的跟踪或远程管理应用程序(或配置)。虽然你总是可以尝试手动搜索这些应用程序，但将它们恢复到出厂时的状态——尽管重新安装你的东西很烦人——会让你更加安心。

### 发展核能

如果你真的想让别人无法跟踪你的数字生活，那就开始新的生活吧。获得一个全新的智能手机号码。创建一个新的主要电子邮件地址，在你的数字生活结束之前，不要与任何人分享。删除您的旧电子邮件地址(在您备份并重新导入您的电子邮件后)。将您的所有服务与您的新电子邮件地址重新关联，这包括从您的汽车使用的帐户(如果您有任何花哨的服务，如 OnStar)，到您的亚马逊帐户，再到您的社交媒体帐户。不要使用你的旧电子邮件地址做任何事情。

对于您的主要服务，请考虑是否可以添加任何额外的安全措施，例如，可能需要二级 PIN 码或其他方式来验证您的无线运营商。如果没有，至少把你的安全问题和提示换成不同的，不可能让另一个人猜到的，即使是一个对你的生活很了解的人。买一个新的无线路由器，以防你的前任安装了某种第三方固件并跟踪你的一举一动。换锁。买一只巨犬。申请限制令。
