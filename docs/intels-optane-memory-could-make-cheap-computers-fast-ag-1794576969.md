不仅仅是垃圾处理器让真正便宜的电脑变慢。或者内存或显卡(或者缺少显卡)。你的廉价笔记本电脑以极慢的速度大声突突前进的主要原因是因为硬盘。廉价的笔记本电脑使用廉价的硬盘驱动器，比更好的电脑中的固态驱动器慢得多。但英特尔的新 Optane 内存改变了这一点。这个 70 美元的小芯片让廉价的硬盘和固态硬盘一样快。

### 记忆到底是什么？

内存是计算机中读写数据的任何东西。在您的处理器、RAM 以及 USB 驱动器中都有内存。“这就像一个金字塔，”格雷格·舒尔茨告诉 Gizmodo，他是一名存储分析师，也是 [StorageIO](http://storageio.com/aboutus.html) 的运营商。"沿着金字塔越往下，容量越便宜，但速度越慢."所以金字塔的顶端是 CPU 中的内存，然后是 DRAM，然后是基于 NAND 的闪存，如 SSD，然后是硬盘驱动器。最底层是最古老、最便宜的记忆磁带驱动器。

在你理解了金字塔之后，你只需要理解另外一件事:易失性和非易失性存储器。易失性存储器往往更快——它是 DRAM 或内置在 CPU 中的东西。它的速度非常快，并且以尽可能小的读/写级别写入:字节级别。然而，易失性存储器在断电时实际上什么也记不住。所以当你重启电脑的时候，易失性存储器会忘记它所学的一切。

重启后第一次在电脑上打开一个大程序，比平时花的时间长。为什么？你电脑的 DRAM 已经忘记了快速打开程序所需的所有重要信息。

非易失性存储器没有这个问题，但它也慢了很多，不能在字节级写入。那么什么是非易失性存储器呢？即使在没有通电的情况下，它也能保存所有学习到的数据。是*持久的*记忆。一旦某样东西被归类为永久记忆，它就有了另一个名字:存储。你使用的每个硬盘、软盘或拇指驱动器都是永久存储器。

英特尔 Optane 内存很酷，因为它使用了内存金字塔上的一个全新条目:3D Xpoint 内存。3D Xpoint 插槽位于 DRAM 和 NAND 之间。它和易失性的东西一样快——能够在关键的字节级别读取——但它也是持久的。真是两全其美。

### 什么是 3D Xpoint 内存？

舒尔茨说:“自从数字存储器诞生以来，只有八种主要的存储器，最近的是 3D Xpoint。”3D Xpoint 之前最后一次内存大提升？这就是 1999 年推出的 NAND。“这些事情每隔几十年才会出现一次。”

3D Xpoint 是英特尔和 Micron 早在 2015 年宣布的，距离上一次内存大飞跃已经过去了 16 年。当时，两家公司承诺 3D Xpoint 比 NAND 快一千倍，但同样持久。这意味着一种全新的超高速数据存储介质。然而，从宣布内存到实际将它放入你我都能买到的硬件，花了将近两年的时间。

### 什么是英特尔 Optane 内存？

英特尔的 Optane 内存是 3D Xpoint 用于消费级产品的第一个实例。已经有一个 [3D Xpoint 存储驱动器](http://www.tomshardware.com/reviews/intel-optane-3d-xpoint-p4800x,5030.html) 可用，利用该技术令人难以置信的速度，但该驱动器为 375GB，零售价为 1500 美元。

Optane 内存是英特尔的目标，让消费者以可承受的价格体验 3D Xpoint 的惊人速度 。它充当主存储驱动器的缓存系统，存储必要的数据，有点像 RAM，但在重新启动之间保存这些数据。

如果这个概念听起来很熟悉，那是因为制造商已经使用更快的非易失性存储器作为较慢东西的缓存系统有一段时间了。值得注意的是，苹果为 iMacs 提供了可选的 Fusion drive。这种系统结合了 SSD 和 HDD，SSD 充当缓存系统，而 HDD 处理存储。虽然它肯定比单独的固态硬盘更便宜，更大，但 [的速度还没有达到固态硬盘](http://www.anandtech.com/show/6679/a-month-with-apples-fusion-drive/6) 的水平。

希捷也推出了这种驱动器，尽管它有一个更通用的“混合”名称。带有 8GB 板载闪存的 2TB 混合驱动器售价 99 美元，但英特尔的新 Optane 设置同样便宜。一个 32GB 的 Optane 记忆棒和 2TB 的硬盘驱动器只需 140 美元。2TB 固态硬盘的成本？$550.

那么为什么要花 140 美元而不是 99 美元呢？因为英特尔 Optane 真他妈的快。

### 好吧，英特尔 Optane 内存有多好？

我对英特尔的新硬件进行了一系列基准测试，以测试它能在多大程度上提高硬盘和固态硬盘的性能。就前者而言，它大大提高了速度，以至于你永远不会再想使用无 Optane 硬盘。Adobe Premiere 和 Adobe Photoshop 就是最好的例子。这两款应用都因打开时间过长而臭名昭著。当我在普通硬盘上打开 Premiere 中的一个项目时，用了 52 秒。在 Photoshop 中打开一个巨大的项目需要 37 秒。然而，一旦我激活了 32GB 的 Optane 内存芯片，这两个过程分别需要 11 秒和 7 秒。令人震惊的是，它实际上使硬盘也比固态硬盘快，奇怪的是，在 Premiere 的情况下，带 Optane 的硬盘实际上比带 Optane 的固态硬盘快了整整两秒。

在 Adobe apps 中打开项目给 Optane 内存带来了最令人印象深刻的结果，但 Optane 显著提高了传统旋转盘片硬盘驱动器的整体性能。HDD 打开谷歌 Chrome 用了 5.5 秒，而带 Optane 的 HDD 用了不到两秒。在 Windows 的原生视频播放器中打开 4K 的视频需要 6 秒钟，但在启用 Optane 内存的情况下，只需 1.5 秒钟即可打开。

与 Optane 内存结合的固态硬盘没有看到与硬盘驱动器相同的疯狂性能提升。事实上，在上述生产率测试中，SSD 和采用 Optane 内存的 SSD 在速度上并无明显差异。

但当我们尝试加载游戏时，情况发生了变化。与打开 Photoshop 或谷歌 Chrome 等应用程序相比，打开视频游戏时，存储驱动器需要读取更多的数据，读取这些数据需要更多的时间。因此，Optane 内存的速度优势更加明显。

当打开 *Mass Effect Andromeda* 时，启用 Optane 内存的存储驱动器的速度是没有 Optane 内存的驱动器的两倍。

在负载水平的情况下，速度的提高不太明显。在这两种情况下，Optane 内存使存储驱动器的速度提高了大约 20%。装有 Optane 的 SSD 在 28 秒内加载了一个*质量效应*级别，而没有安装 Optane 的 SSD 只需 34 秒。装有 Optane 的硬盘加载关卡需要 49 秒，而没有 Optane 的硬盘则需要 61 秒。

《守望先锋》(Overwatch)这款游戏在存储设备上占用的空间比《T2》的《质量效应》(Mass Effect)少得多，它在加载时间上也有显著改善。尽管速度的提高并不一致。虽然 *Overwatch* 在有 Optane 的硬盘上加载速度是没有 Optane 的硬盘的两倍，但在有 Optane 的 SDD 上加载速度只比没有 SSD 的硬盘快一秒。

我在对计算机启动时间进行基准测试时观察到了同样的差异。带 Optane 的硬盘启动速度是不带 Optane 的硬盘的两倍，但带 Optane 的 SSD 只比不带 Optane 的 SSD 快 2 秒。

如果你所有的系统都运行在固态硬盘上，那就真的没有必要出去花钱买一个 Optane 记忆棒。虽然它有时可以提高固态硬盘的速度，但这还不足以保证在 Optane 内存上至少花费 44 美元。然而，如果你有一台电脑，所有的系统和应用程序都在硬盘上呢？Optane 内存会让你的计算生活更美好。

不幸的是，有一些相当大的警告。其中最主要的是 Optane 内存只适用于内置第七代英特尔 Kaby Lake 处理器的计算机——Kaby Lake 去年年底才开始发货。这意味着你车库里积满灰尘的又大又旧的台式机无法从 Optane 内存中受益。事实上，很少有计算机可以，因为 Optane 内存目前只适用于桌面 CPU 和主板，而 Optane 兼容的主板和 CPU 今年才开始发货*。因此，除非你是一个计算机建造者，否则 Optane 内存的好处仍然在你的掌握之外。*

但英特尔向我们保证，我们将在今年晚些时候开始从电脑制造商那里看到大量支持 Optane 内存的台式机和笔记本电脑。所以很快有一天，便宜不再意味着慢。如果你看到一台笔记本电脑正面贴着 Optane inside 标签，这是一件好事。
