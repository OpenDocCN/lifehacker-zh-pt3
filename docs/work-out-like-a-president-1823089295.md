# 像总统一样锻炼

> 原文：<https://lifehacker.com/work-out-like-a-president-1823089295>

巴拉克奥巴马 [打篮球](http://ftw.usatoday.com/2017/01/president-barack-obama-basketball-how-good-arne-duncan-photos-videos) 。比尔·克林顿和布什夫妇 [喜欢跑步](https://www.runnersworld.com/celebrity-runners/running-mates-running-with-potus) 。即使是唐纳德·特朗普 [也有时间打高尔夫球](https://www.cnn.com/2018/02/14/golf/donald-trump-golf-swing-barack-obama-george-bush-intl/index.html) 。但是如果你想尝试一些更老派的运动，这里有一些历史上总统们最喜欢的运动。

Watch

**早起游泳**。约翰·昆西·亚当斯喜欢清晨散步，如果天气好的话，他喜欢在波托马克河的一条支流里裸泳。(有一个故事说，一个记者曾经跟着他到河边，坐在他的衣服上，直到他同意对她进行采访，但是 [这可能不是真的](http://ploddingthroughthepresidents.blogspot.com/2017/02/john-quincy-adams-skinny-dipping.html) 。)

**上一堂拳击课**。西奥多·罗斯福在任总统期间，每天下午都要进行“几个小时的锻炼”。他在白宫里安装了一个摔跤垫子， [请来了格斗家](http://www.thepostgame.com/blog/throwback/201210/teddy-roosevelt-athletic-president-fitness-wrestling-boxing) 来练习拳击、摔跤和柔道。

去远足。威廉·霍华德·塔夫脱可能是我们最大的总统，体重 320 磅，据某些人说，他也是一个强壮且喜欢户外活动的人。他 [和约翰·缪尔](http://www.doctorzebra.com/prez/z_x27strong_g.htm) 一起走了四英里下山，超过了和他们一起骑马的人。(塔夫脱对马来说太重了。)他也喜欢高尔夫、网球和划船。

**打毛毛球**。赫伯特·胡佛的医生乔尔·布恩 [发明了这个游戏](https://en.wikipedia.org/wiki/Hooverball) 供总统玩。它基本上是排球，但是有一个六磅重的药球，所以当你接住它的时候，你会发出“哇”的一声。

**不要跳过臂日**。富兰克林·罗斯福早年因小儿麻痹症而双腿无力(也可能是 T2 T3，格林-巴利 T4 T5，嘘)，但他保证锻炼手臂、胸部、肩膀和背部 T8、T9。

**至少去散散步**。哈里·楚门每天步行一英里半，速度为 [确切地说是每分钟 120 步](https://www.trumanlibrary.org/lifetimes/citizen.htm) 。为什么要锻炼？他解释说:“一些上了年纪的裸露癖者试图证明他们会打网球、手球或其他任何运动...每隔一段时间就会有一个人死于心脏病发作。我说那不适合我。”
