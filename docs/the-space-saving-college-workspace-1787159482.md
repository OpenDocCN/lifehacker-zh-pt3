# 节省空间的大学工作空间

> 原文：<https://lifehacker.com/the-space-saving-college-workspace-1787159482>

大多数大学生只能靠宿舍里的书桌和家具勉强度日，但 redditor[themacu](https://www.reddit.com/user/themachu)想要一些更个性化的东西。他拿了一张宜家的桌子，自己涂上颜色，然后组装了这个看起来很锋利的长凳，上面放着一台 linux 个人电脑。



themachu 的布置之美在于，它看起来很棒，看起来他实际上使用了他桌子上的所有东西，而不是为了保持桌子干净和空着而保持干净和空着。极简主义有它的位置，但是这里有很多——也有很多有用的——但是它看起来仍然井然有序，井井有条。顶部的照片是他实际工作空间的近景，但这里有一个缩小的版本:

这台电脑是一台 Linux 个人电脑，马楚说他用这台电脑是因为他过去常常在 Windows 系统中调整他的桌面，以至于让他的系统负担过重。他有时仍会启动 Windows，但会玩一些他能在 Linux 上玩的轻度游戏(TF2，CS:Go)，然后将其余时间用于工作和学习。这是从后面拍摄的整个工作区的全景照片。

顺便说一句，那张桌子是一张 [宜家](http://www.ikea.com/us/en/catalog/products/50106773/)格顿的，污迹斑斑，就像我们提到过的，而那把椅子是一张用过的 [钢壳飞跃](https://www.steelcase.com/products/office-chairs/leap/)[之一，你最喜欢的办公椅](http://lifehacker.com/five-best-office-chairs-30776066) 。在下面链接的 Reddit 主题中有更多规格——监视器、DAC/Amp 等等，所以如果你好奇，一定要去查看更多。

如果您有自己的工作空间可以炫耀，请将其添加到我们的 [**Lifehacker 工作空间展示并告诉 Flickr pool**](http://www.flickr.com/groups/lifehacker-workspace-showandtell/pool/) 来与我们分享。确保您包含的任何照片至少为 1280x720。让他们保持在 16:9 也有帮助！包括一些文字，关于你使用的材料，你是如何想出这个设计的，以及任何其他相关的细节。如果你巧妙的组织和良好的设计感吸引了我们的眼球，你可能就是下一个特色工作区！
