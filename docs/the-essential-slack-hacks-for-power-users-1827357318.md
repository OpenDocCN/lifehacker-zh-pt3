# 超级用户必备的松弛技巧

> 原文：<https://lifehacker.com/the-essential-slack-hacks-for-power-users-1827357318>

懈怠是否让你的生活变得不守规矩？不间断的协作、交谈和过度的沟通是否让你无法真正完成任何事情？我们也是。



多亏了 Slack 亚太区增长主管阿图罗·阿勒特(Arturo Arrarte ),我们已经有了一批控制疯狂的黑客。

### **保持你的工具条整洁**

进入你的 Slack 偏好设置，点击“高级选项”，然后在“频道列表”下选择“仅我的未读频道和 DMs”。现在，您的频道列表将只显示您尚未阅读的频道。

### 使用快速信息

你冲出门去，但在离开之前，你必须提醒你的同事，你今天余下的时间都不在办公室。只需在任何消息文本框中输入 */msg @Name* 或 */dm @Name* ，就可以直接向任何人发送消息。搞定了。

### 在对话之间跳转

你是一台多任务处理机器，但你厌倦了在 Slack 上滚动、点击频道和对话。“快速切换器”是打开任何对话的最快方式:在 PC 上只需按下[Ctrl]+[K]，如果你使用的是 Mac，则按下[⌘]+[K]。

### 迅速赶上

在繁忙的日子里，当您从一个会议跑到另一个会议时，您可以通过使用[Option]+[Shift]+[↑]/[Alt]+[Shift]+[↑]移动到下一个未读对话，或者使用[Option]+[Shift]+[↓]/[Alt]+[↓]返回上一个对话来快速赶上空闲时间。

### 自动提醒

嘿项目经理们！使用提醒来提醒人们截止日期。键入 */remind @Name* ，无论您需要做什么，以及什么时候— 例如 */remind @sam 预定明天下午 4 点*的会议室。

### 保持邮件未读

提醒可能不是你的事情 — 也许你更喜欢将邮件标记为未读，而不是接收 Slackbot ping 的提醒？只需将鼠标悬停在一条消息上，选择“...”按钮，然后单击“标记为未读”以便稍后返回。

### 请勿打扰

当你需要一段不受干扰的时间来集中精力工作时，这是一个非常有用的工具。通过在任一频道中键入 */dnd* 暂停通知，并输入您希望无通知的时间长度，例如， */dnd 两小时*。那就回到状态。

### 关注重要关键词

通过在频道中使用特定词语时设置通知，避免错过您关心的话题。只需转到 Slack 菜单中的 Preferences 并在通知设置中编辑“我的关键字”来添加您的术语(不要忘记用逗号分隔)。

### 定制 Slackbot

教 Slackbot 如何回答常见或重要的问题，这样你就可以继续你的工作。通过“定制你的工作区”页面定制它在你的频道中的响应，这样当有人使用特定的单词和短语时，Slackbot 就会做出响应(请求办公室 Wi-Fi 密码是一个很好的开始)。

### **创建快速投票**

您的团队需要讨论下一个项目，但每个人似乎都忙于开会，您厌倦了预订、取消和重新预订会议室。你只需要知道什么时间最适合大多数人。添加 [简单投票](https://simplepoll.rocks/) Slack 应用，然后只需键入/poll 后跟“问题？”和“选项”来快速收集你所选主题的投票。例如: */poll“我们应该什么时候开会？”"下午 1 点" "下午 2 点" "下午 3 点"*。不再有来回。

### 电源搜索

你公司闲置的材料数量巨大。搜索“月报”会出现几十个条目，但你只是想要莎拉分享的三月份报告的版本。使用搜索修饰符直接找到您需要的内容，例如:频道中的*(查找特定频道中的消息和文件)、*到:我*(查找发送给您的直接消息)、*从:莎拉*(查找莎拉发来的消息)和*之前:日期*、*之后:日期*或*于:日期/月/年*，以将您的搜索缩小到特定的时间范围。*

### 访问 Twitter 订阅源

需要了解行业新闻和事件，但没有时间访问不同的网站？你可以通过 Twitter 订阅来帮助你保持消息灵通，这一切都不会让你懈怠。搜索 Slack 的应用目录，安装 Twitter 集成，选择@用户名和你想看的推文，瞧。

### 订阅 RSS 源

只需输入 */feed* 并粘贴你想要订阅的 feed 的地址。比如，【https://lifehacker.com/rss.】/饲料

### 明星重要的东西

找不到你的团队领导告诉每个人必须阅读的那篇文章？以后，点击星号 n 转到消息的时间戳。当您稍后点按屏幕顶部的星号图标时，您将能够访问您所有的带星号的邮件。这就像书签，但对于懈怠。

### 使用键盘快捷键

你可能是一个懒散的用电者，但是每个人都有健忘的时候。如果你需要 Slack 键盘快捷键的提醒，只需在 Windows 上点击[Ctrl+/]或在 Mac 上点击[⌘+/]即可调出完整的快捷键列表。
