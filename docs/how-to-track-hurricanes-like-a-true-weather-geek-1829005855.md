# 如何像一个真正的气象极客一样追踪飓风

> 原文：<https://gizmodo.com/how-to-track-hurricanes-like-a-true-weather-geek-1829005855>

飓风季节如火如荼，大西洋上肆虐。如果你住在海边，你有责任关注外面有什么风暴，它们的方向，以及可能产生的影响。



无论你是飓风观测的新手还是天气极客，Earther 都能帮你搞定。这些都是权威的来源和方便的工具，让你知道当热带风暴咆哮时发生了什么，以及在哪里可以找到大自然气旋恐怖的最令人瞠目结舌的图像。

### **国家飓风中心**

国家飓风中心 是特定风暴信息的黄金标准一站式商店，从它们有可能形成的时间到它们将在哪里登陆。有图，有文字，有技术讨论，有外行人讨论。有很多，但如果你在飓风或热带风暴的路径上，这些都是相关的，所以把它收藏起来。

在这个网站上你可以找到很多东西:关于哪里发布了观察和警告的信息、风速预报、降雨量预报和风暴潮预报。后两个 [特别重要](https://earther.gizmodo.com/why-hurricane-florence-could-generate-historic-rainfall-1828971346) 因为水是飓风中最致命的部分，尽管事实上用于划分飓风类别的萨菲尔·辛普森等级仅基于风。但这是另一篇文章的主题。向前！

### 实时卫星环路交互式数据探测器

科罗拉多州立大学可能地处内陆，但它提供了从太空观察飓风的最佳途径之一。实时[卫星循环交互式数据探索者(被称为 SLIDER，因为每个好的研究工具都需要一个缩写词)以接近实时的方式可视化来自美国国家海洋和大气管理局(NOAA)的 16 和 17 颗气象卫星的数据，这些卫星共同覆盖了大西洋和东太平洋。这包括大图的视图以及“飞蚊症”，气象极客们对热带风暴放大视图的说法。额外收获:它包括来自日本 Himawari-8 卫星的太平洋图像，因此你也可以追踪世界另一端的热带气旋。](http://rammb-slider.cira.colostate.edu/?sat=goes-16&sec=full_disk&x=10848&y=10848&z=0&im=12&ts=1&st=0&et=0&speed=130&motion=loop&map=1&lat=0&p%5B0%5D=16&opacity%5B0%5D=1&hidden%5B0%5D=0&pause=0&slider=-1&hide_controls=0&mouse_draw=0&follow_feature=0&follow_hide=0&s=rammb-slider) 

### 热带花絮

如果你想一窥国家飓风中心是如何做天气预报的，看看上的天气模型。这个网站是由佛罗里达州立大学气象学研究生李维·考恩维护的，它很深。你有模型，被称为整体的模型集，飓风猎人的轨迹和由 NOAA [勇敢的灵魂收集的数据，他们为了科学](https://earther.gizmodo.com/noaas-first-all-female-hurricane-hunter-crew-flew-fearl-1828257468) 飞入热带风暴。考恩上传了关于风暴的讨论。观看模特走秀的另一个好地方是 [wx.graphics](http://wx.graphics/) 。

这一切都非常有见地，但如果你不是气象学家，对模型有一句忠告:它们是有帮助的，但它们不是预测。预测是由那些了解他们在模型中看到的东西并努力寻找最佳解决方案的人创建的。因此，不要把你在某个特定模型中看到的当作真理，或者以此为基础制定计划。听听专业人士的意见。

### 国际空间站实况摄像

人造卫星很酷，但是有一个更近的空间奇迹来源。国际空间站有一台 [直播高清摄像机对准地球](https://www.nasa.gov/multimedia/nasatv/iss_ustream.html) 。它每 90 分钟绕地球一周，这意味着你很有可能在天桥上看到飓风。它的观点真实地传达了在更远的轨道上拍摄的卫星照片中可能丢失的巨大风暴。如果你错过了风暴直播，不要担心。仅仅是从太空现场观看地球本身就是一件非常令人心寒的事情，再加上宇航员经常拍摄史诗般的照片，这些照片会出现在 [站的推特账户](https://twitter.com/iss_research) 上。

### 地球风地图

如果你喜欢梵高风格的天气，那么 [地球风地图](https://earth.nullschool.net/) 就是为你准备的。它是基于被称为全球预报系统(GFS)的美国天气模型的数据。虽然它不像欧洲的**那样被大肆宣传，也不应该被认为是官方预测，但它仍然很好看。你也可以叠加不同大气层的风速和海洋温度，以更好地了解动态变化。捷克气象公司 Ventusky 提供了一份类似的迷人地图。** 

### **你当地的电视气象员**

**这听起来肯定没有风地图和来自太空的飓风酷，但我怎么强调这些人作为资源的重要性都不为过。它们是你在客厅里最接近科学家的东西。他们知道如何阅读模型，他们知道你所在的地区，他们了解飓风可能给城镇带来的具体风险，或者说，你的 [最近的工业养猪场](https://earther.gizmodo.com/hurricane-florence-could-unleash-pig-shit-coal-ash-in-1828977086) 。如果你没有电视机(举手)，你几乎可以在 Twitter 上找到它们。许多人在他们的网站上也有博客。**

**许多地方新闻网站也有天气栏目。 [首都天气组织](https://www.washingtonpost.com/news/capital-weather-gang/) 是华盛顿特区大都市地区的先驱，但许多其他地方也有自己的版本(例如，《罗诺克时报》有由 [资深气象学家凯文·麦亚特](https://www.roanoke.com/users/profile/kevin%20myatt/) 经营的天气杂志)。**

**你当地的国家气象局(NWS)办公室也是如此。这些人在暴风雨中不知疲倦地工作， [他们唯一的任务](http://www.nws.noaa.gov/mission.php) 是提供“天气、水和气候数据、预报和警报，以保护生命和财产，增强国民经济。”所以，倾听他们的心声，如果天气预报不准，不要给他们太多的悲伤，甚至可以在乌云散去、洪水退去的时候给他们送去披萨。**
