# 如何直接从研磨机中测量胡椒

> 原文：<https://lifehacker.com/how-to-measure-pepper-straight-from-the-grinder-1820994970>

直到最近，我从未测量过新鲜研磨的胡椒。如果一个食谱需要半茶匙，我会简单地在盘子上研磨浆果，直到我认为我已经“足够了”，这仅仅是基于我的鼻子和我那天感觉有多刺鼻，因为没有优雅的方法来用研磨机上的茶匙来测量它。但是，如果你花时间做一些测量，只要一次，你就能在任何食谱中分配所需的量，不需要量勺。



*这是* [***成人厨房***](http://lifehacker.com/tag/the-grown-up-kitchen#_ga=2.229571930.753484183.1503856534-949419976.1446553382) *的一部分，Skillet 系列旨在回答您最基本的烹饪问题，并填补您的家庭厨师教育中可能缺失的任何空白。*

对于最初的、有点乏味的测量过程，你需要一把秤、一把测量勺和几个模子。你的秤越精确，你未来的胡椒测量就越精确，但没必要发疯；多吃十毫克左右的胡椒并不是世界末日。

一旦你有了你的原料，在一个模子里磨出大量的胡椒。将另一个 ramekin 放在你的天平上，然后从你的供应品中舀出半茶匙胡椒，称重，并记下它的质量。你的体重会根据你的研磨机的粗度而变化，但应该在一克左右。(你也可以假设半茶匙胡椒大约是一克，跳过这一步，但我不知道你希望你的厨房有多精确。)

接下来，倒空一个模子，把它放回秤上，往里面磨更多的胡椒，数你在研磨机上转动的次数。当你达到一克(或你记下的任何质量)时，停止计数，并记下*那个*数字。然后做一个小纸条，上面写着“1/2 茶匙= X 次旋转”，其中“X”是旋转的次数，然后把它贴在香料柜的门里面或者研磨机上。(对我来说，“X”是 35。)

下一次你需要测量新鲜的胡椒时，简单地数一下研磨的次数，根据需要放大或缩小。当然，这种方法确实依赖于你操作研磨机的一致性，但它肯定胜过猜测，或者每次研磨一束胡椒并从模子里舀出来。
