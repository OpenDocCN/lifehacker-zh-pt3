# 它会腐烂吗？小龙虾酿比目鱼

欢迎来到本周的 [*它会分开吗？*](http://lifehacker.com/tag/will-it-sous-vide) ，每周专栏，我通常用我的沉浸式循环器在那里制作你想要我做的任何东西。

bOfvBY">对于有鳍的鱼，我选择了大比目鱼。我父亲在我十几岁的时候搬到了华盛顿州，但在那之前，我的海鲜饮食经历主要是南方的，有很多鲶鱼、炸牡蛎和(显然)小龙虾。我清楚地记得在鲍威尔书店呆了一天后，我的继母第一次为我们准备比目鱼。这绝对不同于我吃过的任何一种鱼，非常美味，非常西北太平洋。因此，尽管你可能不会在任何菜单上找到很多小龙虾填充的比目鱼，但鉴于我成长过程中的各种美食，它对我来说是有意义的。</p><p class="sc-77igqf-0 bOfvBY">然而，在这之前，我给了两条 8 盎司的硬鱼片和白鱼一个很好的腌制，让它们挂在外面大约半个小时，同时我让浸泡循环器温暖它们各自的浴缸。</p><p class="sc-77igqf-0 bOfvBY">然后，我在每块菲力牛排上做了相当深的垂直切口(鱼皮朝下)，并塞满了美味的鱼尾。</p><p class="sc-77igqf-0 bOfvBY">然后我把每条菲力欧鱼真空密封在它自己的小袋子里，并把它们放在两个不同温度的浴缸里。</p><p class="sc-77igqf-0 bOfvBY">为什么是两种温度？这叫试验，我的朋友们。我以前从来没有用这种方式烹饪过鱼，而真空烹饪中最受信任的两个声音——食品实验室和 chef steps——似乎在烹饪比目鱼的最佳温度上存在分歧。<span> <a class="sc-77igqf-0 bOfvBY">在他们的小浴池里洗了一个小时后，我轻轻地去掉鱼皮，把肉片放在袋子里，这样可以防止馅料掉出来。然后我用热黄油煎鱼的底部，然后把它们装盘，在上面淋上华丽的棕色黄油。<br/></p><p class="sc-77igqf-0 bOfvBY">正如你从上面的照片中看到的，这条鱼很难保持在一起，当我把它从袋子里拿出来的时候，它几乎碎成了三块。这不是最漂亮的照片，但它对味道没有负面影响。</p><p class="sc-77igqf-0 bOfvBY">所以现在你一定知道:<strong>小龙虾馅的比目鱼会腌制吗？</strong></p><p class="sc-77igqf-0 bOfvBY"><strong>答案？是的，我的朋友们。是的，确实会的。就像我用浸没式循环器烹饪的所有东西一样，对烹饪环境进行如此精确的温度控制让我可以将我的混合海鲜烹饪到我喜欢的程度，而不用担心烹饪过度，这总是一种享受。</strong></p><p class="sc-77igqf-0 bOfvBY">就温度而言，我肯定更喜欢用较高温度烹饪的鱼。大比目鱼是一种肉更多的鱼，虽然它不像旗鱼那样像牛排，但我认为它受益于几度的额外倾斜。</p><p class="sc-77igqf-0 bOfvBY">在华氏 122 度下烹饪的比目鱼绝对不差，而且它绝对不干燥，但它有点软，牙齿吱吱作响，130 度的比目鱼有更大、更坚硬、更有牙齿的薄片。至于馅料？那条鱼在砰砰作响。小龙虾已经做好了，但是当它依偎在比目鱼的肉壁里时，它变得非常温暖，所有的味道都很好地融合在一起。基本上，这是一个成功，我吃了。</p> </body> </html></iframe>
