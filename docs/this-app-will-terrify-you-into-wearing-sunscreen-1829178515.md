# 这个应用会吓得你涂防晒霜

> 原文：<https://lifehacker.com/this-app-will-terrify-you-into-wearing-sunscreen-1829178515>

防晒霜不仅仅是为了防止晒伤。随着时间的推移，特别是在浅色皮肤上，阳光是皱纹、增厚和褐色斑点的主要原因。还好， [一款好的广谱防晒霜会保护你](https://vitals.lifehacker.com/all-your-sunscreen-questions-answered-1827579794#_ga=2.224701593.262633695.1537192409-1456718367.1520458611) ，这个 app 模拟的到底是多少。



用 Sunface 自拍(iOS 上免费的或 [Android](https://play.google.com/store/apps/details?id=com.agt.sunface&hl=en_US) )，应用程序会应用皱纹、太阳黑子和所有其他与外表相关的衰老迹象，向你展示你未来不涂防晒霜的样子——五年到 25 岁。现在点击“防晒”按钮，你会看到你同样的笑脸，少了很多皱纹和色斑。(还有一个“每周晒黑”选项，它看起来很像“不防晒”选项，但具有更大的癌症风险。)

Sunface 不是在开玩笑。在它绘制出你的脸并画出皱纹后，当你思考你的未来时，你的老脸会移动和呼吸。它不时向你眨眨眼，偶尔耸耸肩，叹口气。我们明白了，sunface，好吗？

由于黑色素是一种天然的防晒霜——在某种程度上——该应用程序将滤镜更加微妙地应用于深色皮肤。所以，为了从这个应用程序中获得最佳的惊吓值，把它展示给你最苍白的朋友。
