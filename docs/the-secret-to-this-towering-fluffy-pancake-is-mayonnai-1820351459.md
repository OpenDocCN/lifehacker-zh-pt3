# 这种高耸蓬松的薄煎饼的秘密是蛋黄酱

> 原文：<https://lifehacker.com/the-secret-to-this-towering-fluffy-pancake-is-mayonnai-1820351459>

我需要先澄清一下:蛋黄酱基本上是鸡蛋和油。你会被吓到的，但是记住:鸡蛋和油。



记住这一点，我想建议你试着做煎饼——是的，甜的那种，有枫糖浆的那种——里面有蛋黄酱。

勇敢的灵魂美食 52 尝试了一种流传已久的日本煎饼配方，配料单上有蛋黄酱和苏打水。与美国薄煎饼相比，烹饪程序也是非标准的——你在薄煎饼烹饪的同一个锅里混合鸡蛋、苏打水和蛋黄酱，然后在小火上烹饪时加入面粉。

结果是一个又高又松的薄饼，带着一丝酸味。它和糖浆一起吃很好，和水果一起吃也很棒。我有没有提到在这个过程中你会弄脏零个碗？

然后做一份 [蛋黄酱炸烤奶酪](https://skillet.lifehacker.com/3-unexpected-things-you-can-do-with-mayo-that-arent-put-1798356585) 当午餐。
