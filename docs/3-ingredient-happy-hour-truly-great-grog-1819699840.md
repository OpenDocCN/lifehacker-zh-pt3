# 三种成分的欢乐时光:真正伟大的酒

> 原文：<https://lifehacker.com/3-ingredient-happy-hour-truly-great-grog-1819699840>

各位周末快乐，欢迎回到 [*三料欢乐时光*](http://lifehacker.com/tag/3-ingredient-happy-hour) ，每周饮品专栏，提供超级简单又美味的酒水。这个星期有点艰难，它让我渴望一些非常简单的东西，还有什么比海盗喝的东西更简单的呢？

Watch

现在，当谈到酒时，有许多不同的说法。可以热饮也可以冷食，可以加水也可以不加水(为什么？)，以及加不加额外香料。我——就像我之前的一样——喜欢让我的酒简单、冰镇，还有——你猜对了——三种配料。

为了制作这个简单的坏血病战斗机，你需要:

*   2 盎司深色朗姆酒
*   1/2 盎司 demerara 糖浆(由 2:1 的 turbinado 糖和水制成)
*   1/2 盎司新鲜酸橙汁(大约半个酸橙汁)

在一个装满冰(或一个大冰块)的摇壶里混合所有的东西，摇至你的饮料完全冷却。滤入含有几个小冰块或一个大冰块的双份岩石杯。不要用任何东西装饰；海盗不装饰狗屎。
