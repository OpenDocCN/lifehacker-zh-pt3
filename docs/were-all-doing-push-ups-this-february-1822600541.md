# 今年二月我们都在做俯卧撑

> 原文：<https://lifehacker.com/were-all-doing-push-ups-this-february-1822600541>

准备好迎接 Lifehacker 健身挑战的新篇章了吗？在一月份，我们在适合我们每个人的水平上做引体向上 。在二月份，我们坚持上半身练习，但是改变了脚本:每天，尝试做至少一个俯卧撑。

Watch

去年我自己做了一个类似的挑战，效果不错:月初我可以做 5 到 10 个形式不好的俯卧撑。到了月底，我又做了同样多的俯卧撑，但是它们很强壮，很直，形式也很好。这个月结束几周后，我第一次做了拍手俯卧撑。

所以这是我们这个月要做的事情:

*   **基线测试**。我可以不停地做 10 个出色的俯卧撑。以良好的姿势做任何类型的俯卧撑，即使这意味着你是用手扶着墙做的。
*   **每天至少做一个俯卧撑**，还是那种适合你的水平。你想做多少都可以，但如果你感到酸痛，只需做一个单壁俯卧撑来保持连续。
*   **最后的测试**看看你进步了多少。

为明天的第一次训练做好准备。祝你好运，月底我们再交换意见！
