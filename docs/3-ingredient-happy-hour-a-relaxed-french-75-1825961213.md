# 三要素快乐时光:轻松的法国 75

> 原文：<https://lifehacker.com/3-ingredient-happy-hour-a-relaxed-french-75-1825961213>

周末快乐，欢迎回到 [*三料欢乐时光*](http://lifehacker.com/tag/3-ingredient-happy-hour) ，每周饮品专栏，提供超级简单又美味的酒水。本周，我们正在制作一款起泡鸡尾酒，非常适合母亲节，或者任何一个真正的周日:法国 75。



从技术上讲，法国 75 是一种四种成分的鸡尾酒，包含杜松子酒、柠檬汁、简单的糖浆(或方糖)和香槟。这里我们用柠檬水来简化一下，毕竟柠檬水是柠檬汁和糖水。就气泡而言，保持干爽，但也可以随意使用任何种类的起泡酒，比如香槟、普罗塞克或卡瓦酒。你甚至可以用起泡玫瑰酒——配粉色柠檬水？？—让妈妈(或你自己)的生活变得更加美好。要制作这种饮料，你需要:

*   1.5 盎司杜松子酒
*   1 盎司柠檬水(试试粉色！)
*   您选择的起泡酒

将杜松子酒和柠檬水倒入一个玻璃杯中，稍微搅拌一下。以冰镇起泡酒为顶，享受一顿别致的早午餐。
