# 崩溃的海浪桌面

> 原文：<https://lifehacker.com/the-crashing-waves-desktop-1756059209>

塞巴斯蒂安的桌面体现了从一点混乱中建立秩序，我们完全可以看到这一点。我们也很喜欢他的壁纸，以及桌面小工具的巧妙运用。他是这样设计的。

Watch

即使你从这个设置中除了墙纸什么也得不到，你也可能在这个设置中保持良好的状态。即便如此，Sebastian 的 Windows 安装程序并不需要太多时间就能运行。这是你需要的:

*   [格拉特壁纸](http://aidendrew.deviantart.com/art/Glatt-585051406)
*   用于 Windows 的 [雨量计](http://rainmeter.net/) 系统调试和监控工具
*   [小雨皮为雨量计](http://fadeoffset.deviantart.com/art/Drizzle-v1-0-541134095) 为时间和日期
*   硬盘和天气显示的 [平坦而模糊的皮肤](http://satyajit00.deviantart.com/art/Flat-and-Blurry-v1-5-524848816)
*   [Alli 为 rainmet](http://fionnt.deviantart.com/art/Alli-Afresh-1-0-356296409)的 RSS feed 换了新皮肤(Sebastian 注意到新皮肤的工作方式和他的不太一样，如果你想要他用过的那个，让他知道，他会分享的！)
*   点击左侧展开式启动面板的 [弹出菜单皮肤](http://filipbaotic.deviantart.com/art/Pop-Menu-for-Rainmeter-422578176)
*   电池指示灯雨量计的 [Omnimo UI](https://www.omnimo.info)

这就是这个的全部内容。正如我们提到的，RSS 工具的最新版本不太像上面显示的那个，但 Sebastian 说如果你需要它，他会分享原始版本——只需前往下面链接的 Flickr 页面并让他知道。让我知道我们送你，你有多喜欢他的桌面，而你在它！
