# 我们应该吃什么终于有了共识

> 原文：<https://lifehacker.com/there-s-finally-a-consensus-sort-of-on-what-we-should-1749509246>

我们之前已经注意到，当谈到营养 时，每个人都同意的只有三件事 [。多亏了营养研究人员的会议，我们可以在这个清单上增加一些东西。](http://vitals.lifehacker.com/the-only-three-things-everybody-agrees-on-when-it-comes-1709039566#_ga=1.253474569.1055861349.1441835238)



这份共识声明的第一点来自于 11 月举行的一次会议，简单地说就是:**更多的植物、坚果、豆类、海鲜**。研究人员认可了[2015 年饮食指南咨询委员会](http://health.gov/dietaryguidelines/2015-scientific-report/) 的调查结果，该委员会在推荐这些食物的同时，建议人们避免含糖食品和饮料，以及精制谷物。

不幸的是，没有比这更多的共识。STAT 报告称，研究人员试图命名特定的食物或饮食为可接受的，但无法达成完全一致:

> 耶鲁大学预防研究中心主任大卫·卡茨博士说:“会议进行了 90 分钟，我们仍然试图就蔬菜到底是什么达成一致。“那是一个黑暗的时刻。”
> 
> …

> “我们在细节上有分歧，但我们肯定，观点非常不同的专家确实有共同点，”卡茨说，他是 old ways Finding Common Ground Conference 的联合主席。

研究人员还同意，许多不同的饮食可能是健康的；例如， [没有必要限制碳水化合物](http://vitals.lifehacker.com/carbs-arent-inherently-bad-but-your-individual-toleran-1699119195) ，尽管这可能是通向健康的许多不同途径之一。

他们还担心营养科学 有太多的混乱，新的发现在新闻中耸人听闻，即使潜在的科学没有改变。他们指出，人们需要确凿的证据、实用的工具和“食品知识”意识来理解营养及其与可持续性的联系。点击下面的链接阅读更多内容，并查看 STAT 上的文章，了解更多关于这一共识是如何形成的。
