# 这是在电梯里的正确行为方式

> 原文：<https://lifehacker.com/this-is-the-correct-way-to-behave-in-an-elevator-1823730929>

乘坐电梯可能是一次令人担忧的经历，伴随着所有关于按按钮、扶着门和闲聊的模糊礼仪，所有这些都混合着一种健康的幽闭恐惧症。



没有更多的了:一个非常特别的西雅图房东主动提出了一个在电梯里应该如何表现的大纲。西雅图 alt-weekly[*发布了通知的照片，*](https://www.thestranger.com/slog/2018/03/12/25902162/which-part-of-my-elevator-is-the-most-seattle)*最近被张贴在他们的一个作家的公寓楼里(点击进入他们的网站查看海报，非常值得完整阅读)。房东极其明智的指示包括:*

*   *“站到右边去。”既然是在人行道和自动扶梯上，那么就应该在电梯里。很简单。*
*   *“如果不麻烦的话，请扶着门。”你可能不总是想这么做，但这是礼貌的做法。*
*   ***“做按钮推手。”如果你是拥挤的电梯里离按钮最近的人，自愿按下人们需要按的任何按钮。当然了。***
*   *“移到后面去。”不言自明。*
*   *“考虑走楼梯吧。”当你身体健全，不携带任何重物，只去二楼时，为什么要为别人系上电梯？* 

*然后还有以下关于体臭的极其重要的注意事项(TL；医生:不要放屁):*

> *良好的卫生习惯应该每天坚持，尤其是如果你经常乘坐电梯的话。狭小、封闭的空间会引起对任何体味的注意。乘坐电梯时，尽量不要放屁或打嗝。如果你有，请原谅。不要带非常臭的食物上电梯。相反，把你的食物装在容器里。千万不要在电梯里吃东西。千万不要涂香水或乳液。对你来说闻起来正常的东西可能会让其他人觉得很恶心。*

*传单中还包括了一个关于礼貌闲聊的注意事项，但它是针对小型公寓建筑的，所以出于我们的目的，我们将其排除在外。闲聊的需要是严格逐案的！当有疑问时，礼貌地微笑和点头，保持沉默。*

*陌生人把这份通知捧为典型的“西雅图”，意思是西雅图人是出了名的，他们关心的是避免冒犯到荒谬的地步。那可能是真的(绝对是真的)。但是这份传单上的建议恰好也是一份无可挑剔的、准确的行为指南，适用于任何即将踏上电梯的人，挑剔的地区文化规范见鬼去吧。*

*回顾一下:友好但不要太友好；向右和向后移动，为其他电梯乘客创造空间；抓住门让其他人进来；必要时为其他乘客按下按钮；等到出去释放你的身体气体。这要求很过分吗？*
