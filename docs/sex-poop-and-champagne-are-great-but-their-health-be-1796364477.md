# 性爱、便便和香槟都很棒，但它们对健康的益处被高估了

> 原文：<https://lifehacker.com/sex-poop-and-champagne-are-great-but-their-health-be-1796364477>

欢迎回到我们的健康新闻综述，前一集*本周该担心什么*。我们把它重新命名为*现实剂量*，因为健康新闻并不总是给你担心的事情。有时候，被夸大的是利益，而不是风险。本周，我们来看一些研究，这些研究似乎是预防癌症和老年痴呆症以及成为一名更好的运动员的关键。



### 射精越多，癌症越少？

**头条** : [男性应射精以降低前列腺癌风险](https://www.attn.com/stories/17751/surprising-benefit-ejaculation) (ATTN)

**故事**:前列腺癌是男性第二大常见癌症(仅次于皮肤癌)，所以如果我们有办法预防它，那就太好了。前列腺有助于射精，也产生一些组成精液的液体，所以有理由问射精是否与前列腺健康有关。

ATTN 的故事是关于去年 的一项 [研究发现，男性一生中记得射精的次数越多，他们患前列腺癌的可能性就越小。这很有意思，而且这项研究的规模很大:近 32，000 名男性，尽管当前列腺癌](http://www.europeanurology.com/article/S0302-2838(16)00377-8/abstract/ejaculation-frequency-and-risk-of-prostate-cancer-updated-results-with-an-additional-decade-of-follow-up) [在非裔美国人](https://academic.oup.com/jnci/article/89/3/188/2526672/Why-Do-African-American-Men-Suffer-More-Prostate) 中特别常见时，93%是白人，这有点糟糕。

但是这项研究只是显示了一种联系。这并不能证明射精可以保护你远离癌症。这是假设男性诚实地报告了他们的性和手淫史。值得一提的是，之前的一项研究发现，手淫似乎不会影响患癌风险。

要点:没有太多的证据支持通过多射精来预防癌症的观点。饮食可能有所帮助，但你预防的最佳选择是年轻的 [和没有前列腺癌家族史的](http://www.cancer.net/cancer-types/prostate-cancer/risk-factors-and-prevention) 。

### 便便兴奋剂是自行车运动的未来吗？

**头条** : [动过血液兴奋剂，自行车运动员可能很快会‘便便兴奋剂’](https://www.washingtonpost.com/news/early-lead/wp/2017/06/20/move-over-blood-doping-cyclists-might-be-poop-doping-soon/?utm_term=.b7b287a930a8)(华盛顿邮报)

**故事**:我们的结肠含有大量对我们健康很重要的细菌和其他微生物——但请记住，我们仍然不太了解健康的微生物生态系统是什么样子。

一位骑自行车的人(是的，只有一位)也是微生物学家，她告诉 *Bicycling* 很多运动员都有一种叫做*普雷沃氏菌*的细菌，但是她发现自己没有。所以她给自己移植了一个骑车人的粪便。(听起来就是这样——邮报称之为“反向灌肠”。)她希望解决一些恼人的健康问题，但发现她不仅感觉更好，还开始赢得更多比赛。

听起来很棒，对吧？但是这个一个女人的实验并不意味着骑自行车的人的便便给了你超能力。请注意，当她感觉更好时，她可以更频繁地训练。如果你一周训练五天，你会比一周训练两天骑得更好。与此同时，我们不知道为什么骑自行车的人有更多的普雷沃氏菌——也许这些细菌真的很喜欢骑自行车的人倾向于消耗的凝胶和电解质饮料。

微生物学家乔纳森·艾森 [在这里挑出这个故事](https://phylogenomics.blogspot.com/2017/06/irresponsible-reporting-on-poop-doping.html) ，称该报道“荒谬地不负责任”没有证据表明*普雷沃氏菌*会让你更快，顺便说一句，DIY 粪便移植存在真正的风险。

**外卖**:不，便便兴奋剂不是自行车运动的未来。

### 好吧，但是香槟肯定能预防老年痴呆症

**头条** : [每天喝香槟“有助于预防痴呆症和老年痴呆症”](http://www.cosmopolitan.com/uk/entertainment/news/a39546/drinking-champagne-dementia-alzheimers/) (英国《四海为家》)

故事:这是一项较早的研究，本周在社交媒体上获得了推动。它上一次是在 2015 年，但这项研究本身是从 2013 年开始的。有些故事非常扣人心弦，以至于它们有时会起死回生，所以我们有这个关于香槟对老年痴呆症有好处的旧研究，同时还有这个关于酒精对老年痴呆症有害的新研究。

但是香槟听起来更好，对吗？一定喜欢香槟！简直是健康食品！那么让我们快速看一下 [研究](http://online.liebertpub.com/doi/abs/10.1089/ars.2012.5142) 。(你可以从 [科学警报](https://www.sciencealert.com/can-champagne-really-improve-your-memory-and-stave-off-dementia) 或 [NHS Choices](http://www.nhs.uk/news/2015/11November/Pages/No-hard-evidence-champagne-can-prevent-dementia.aspx) 中读到更多，他们都做了很好的拆解。)

八只老鼠每天喝香槟。六周后，他们在迷宫中的表现比八只喝苏打水的老鼠要好。就是这样。这就是这一切的基础。研究人员认为，香槟中的一种化学物质可能对你大脑的某些部分有益，但这项研究没有告诉我们这是不是真的。

**要点**:不要指望香槟能预防老年痴呆症。 [根据美国国家老龄研究所](https://www.nia.nih.gov/alzheimers/publication/dementias/risk-factors-dementia) 的说法，不吸烟、避免心血管疾病、以及...避免过量饮酒。
