# 为什么你应该使用 Facebook Messenger 而不是 SMS

> 原文：<https://lifehacker.com/why-you-should-use-facebook-messenger-instead-of-sms-1796907913>

群发短信很有趣，尤其是当你开始分享链接、gif 和其他有趣的媒体时。但是有不同操作系统的人入党总是有问题。



你正在与一群 iPhone 用户聊天，充分利用贴纸和预览链接，改变对话的名称。但是当你邀请一个拥有安卓系统的人加入群聊时，蓝色的泡泡变成了绿色，你就不能再使用任何有趣的功能了。不过，有一种方法可以解决这个问题。

有一个地方，不同颜色的泡泡无关紧要。在这里，每个人都可以使用相同的功能，看到相同的表情符号，看到相同的内容。那个地方是 Facebook Messenger。

脸书一直在为其即时通讯平台添加如此多的功能，以至于它不仅使用起来很有趣，而且非常方便易用。在某种程度上，这比普通短信要好。原因如下。

### 有 WiFi 的地方都能用

有了 Messenger，您不必担心有手机数据计划。只要你有无线网络，你就能使用它。现在的好处是几乎在任何地方都有 WiFi。尤其是在纽约这样的大城市，街上有免费 WiFi。

### 您可以自定义它

您几乎可以更改 Messenger 对话的所有内容。你可以给你的朋友起昵称，并在“编辑昵称”下随时更改您也可以将群组聊天的名称更改为任何名称，只需将光标悬停在群组聊天名称上并键入不同的内容即可。还有一个功能是在“更改表情符号”下指定特定的表情符号反应无论你选择哪个表情符号，都将取代脸书通用的竖起大拇指表情按钮。

不过，最酷的功能是改变群聊的颜色。有一个完整的调色板可以改变文本气泡的颜色。它可以是粉红色，绿色，紫色，棕色，你能想到的。

### 你可以在聊天中玩游戏

如果你在工作中感到无聊(或者其他地方，真的)，你可以邀请你的朋友踢足球或打篮球，或者简单地发送一个足球或篮球表情符号(这只在手机上有效)。点击这个小图标将会在信息窗口中启动一个小游戏，目标是尽可能多的得分并击败你的对手。

你也可以通过点击键盘条上的游戏控制器图标来玩游戏，比如和朋友玩单词游戏(这在手机和桌面上都适用)。

### 链接显示为预览

许多短信应用程序，如 iMessage，已经将链接显示为预览，但当发送给使用安卓系统的人时就不是这样了。如果你通过 Messenger 发送链接，它会以小缩略图的形式预览网站、YouTube 视频或社交媒体帖子。这很好，因为有时您不必离开 Messenger 就可以查看或阅读您的朋友发送给您的内容，并且您可以立即对其做出反应。

此外，如果您的朋友向您发送了一个脸书团体活动，由于您已经连接到脸书，因此更容易看到。该应用程序会自动将您重定向到活动页面。

### 你的老板会认为你在工作

Messenger 最大的优点是它在你的智能手机和电脑上都一样。您可以通过应用程序或浏览器登录，获得完全相同的体验。如果您正在工作，并通过 Messenger 与朋友交流，您不必在电脑和手机之间来回切换来制定晚餐计划。你所要做的就是打开一个新的浏览标签，然后输入。哦，如果你有一个严厉的老板，他或她可能会认为你超级忙，工作很专注——而不是在 Messenger 上聊天打发时间。

### 已读回执始终开启，因此没有人能够忽视您并侥幸逃脱

如果你是一个讨厌等待他人回复的人，那么 Messenger 是你的理想选择。脸书确保阅读回执始终打开，这样当人们看到你的信息而没有回复时，你可以大声呼叫他们。在每一个文本气泡下面，你会看到什么时候有人读了你的信息，而没有理睬你。如果他们读过，你会看到一个小圆圈，里面有他们的个人资料照片。如果他们没有收到，您会看到一个带有勾号的圆圈(这表示您知道他们已经收到)。

### 您可以删除消息

当你发短信时，你很容易犯错误——很多错误。但是当你通过短信发短信时，没有选项可以删除你令人尴尬的错别字或对错误的人的意外坦白。使用 Messenger，您可以删除发送的任何消息。你只需点击每个文本气泡旁边的省略号，就会出现“删除”按钮。

### 还提供语音信息和视频聊天

Messenger 不仅适用于文本对话，还适用于语音和视频聊天。该应用程序在文本输入栏旁边有一个微型麦克风和摄像头，可以让你发送快速音频记录或发起视频电话会议，就像 Skype 一样。这些功能很酷的一点是，它们非常容易使用，你的指尖已经有了所有的联系人——你不必登录任何其他网站或应用程序。

### **可以在安卓上用短信代替 Messenger】**

这个只是给安卓用户用的。但如果你有一部安卓手机，你可以让你所有的短信对话在 Messenger 中显示为紫色线程——与你的蓝色脸书聊天并排。前往“设置”，选择“短信”，然后打开“默认短信应用”，你会看到短信同步到你的 Messenger。这不会阻止你在其他应用程序中接收和阅读短信，包括标准的 Android 系统。

### **您可以加密您的信息**

越来越多的人开始要求科技公司在他们的消息应用程序中增加消息加密功能。脸书在“新信息”屏幕的右上角增加了“秘密”选项。这使得用户可以向朋友发送类似 Snapchat 的短暂消息，这些消息可以持续 5 秒到 1 天。秘密消息功能是完全可选的，但是如果使用，除了对话两端的两个人之外，没有人可以阅读这些消息。脸书、执法部门或情报机构都无法监视你。
