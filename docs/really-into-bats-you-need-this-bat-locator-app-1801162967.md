# 真的喜欢蝙蝠吗？你需要这个蝙蝠定位器应用程序

> 原文:[https://life hacker . com/really-into-bats-you-need-this-bat-locator-app-1801162967](https://lifehacker.com/really-into-bats-you-need-this-bat-locator-app-1801162967)

不幸的是，蝙蝠研究不足。也许是因为它们不可爱，或者是因为它们是夜间活动的，我们不常见到它们，或者，正如“公民科学家”丹妮尔·古斯塔夫森在彭博一份关于新蝙蝠应用的报告中所说，因为它们在流行文化中没有良好的公共关系。

但是也有一些人真的很喜欢蝙蝠，比如古斯塔夫森，他是国际蝙蝠保护组织 的董事会成员，也是我的朋友。我七岁的儿子莫名其妙地迷上了蝙蝠，古斯塔夫森给他订了一份 BCI 杂志，我称之为*蝙蝠幻想*，但实际上只是叫*蝙蝠*。

Watch

现在，蝙蝠是生态系统的重要组成部分，应该被研究，但不管是什么原因——科学家不够；一般来说，科学家都处于压力之下——蝙蝠的数据收集主要落在“公民科学家”身上，比如古斯塔夫森和她带领的业余爱好者们晚上在中央公园散步。“蝙蝠面临多种威胁——栖息地丧失、风力涡轮机，或许还有最大的威胁——误解和非理性恐惧。蝙蝠是奇妙的生物，它们需要我们的帮助，”古斯塔夫森说。

一家名为野生动物声学(Wildlife Acoustics)的公司开发了一款应用程序，帮助业余鸟类学家识别鸟鸣。该公司开发了一款新应用程序，通过改变蝙蝠回声定位的频率，帮助蝙蝠爱好者定位蝙蝠，这样就可以通过 iPhone 或 Android 听到蝙蝠的回声。公民科学家可以使用该应用程序来帮助他们定位蝙蝠，丹妮尔和蝙蝠保护国际希望很快创建一个公共数据库，用于存储和整理收集的蝙蝠种群数据。

[应用是免费的](https://itunes.apple.com/us/app/echo-meter-touch-bat-detector-recorder-analyzer/id693958125?mt=8) ，但是 [硬件](https://www.echometertouch.com/) ，179 美元，对我儿子来说有点贵。在那之前，我们将不得不依靠古斯塔夫森的蝙蝠应用程序进行蝙蝠行走——只要我的儿子被允许熬夜到那么晚。
