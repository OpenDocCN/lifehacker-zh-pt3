# 你已经具备了成功所需的基本技能

> 原文：<https://lifehacker.com/you-already-have-the-fundamental-skills-you-need-to-suc-1759311627>

我们都在寻找捷径来帮助我们度过一天，让事情变得简单一点，但你仍然需要出现并完成工作。责任的基本原则不会改变。然而，持之以恒，并应用你已经知道的东西是如何在竞争中胜出的。



*本帖最初出现在* [*的简单美元*](http://www.thesimpledollar.com/you-already-have-all-the-skills-you-need-to-succeed/) *上。*

在我大学毕业后的第一份工作中，我从事一个数据组织项目。最初，这至少是一个三人项目，但在项目开始后不久，团队的第三名成员被解雇了。

这相当令人苦恼，因为团队的第三个成员应该是我们正在组织的数据的专家。她应该理解数据的含义，我应该构建接口，而另一个人实际上保持数据库运行——换句话说，就是数据库管理员。

因此，在一年的大部分时间里，数据库管理员和我被拖到一个与团队其他成员分开的办公室。肯定有一些言外之意，这个项目遇到了麻烦，不是因为我们自己的努力，而是因为我们无法控制的事件。

不出所料，在那一年里，我开始与数据库管理员建立了非常牢固的关系。我们花了大量时间研究现有的数据，找出数据集之间的关系。我们针对相似的数据集研究了许多不同的接口，并试图借鉴这些接口的优点。

成功了。我们的项目在每一个审查里程碑上都做得非常好，我们改变了食物链上层一些决策者的观点。最终，我们得到了更多的永久性工作。

我为什么要告诉你这些？

数据库管理员是完成所有这些工作的关键人物，他被训练成一名高中教师，而不是计算机科学高中教师。他在爱荷华州的一所农村高中教农业课。那么，他是如何成为 Oracle 数据库管理员的呢？他没有计算机科学或信息服务的学位。他是怎么到那里的？

事实是，为了达到那个目标，他只是应用了一堆与计算机科学完全无关的技能。他应用了每个人都已经掌握的技能。唯一的区别是他实际上应用了它们。如果你想在事业上有所提升，或者换一个不同的职业，重返校园肯定会有所帮助。然而，你可能已经具备了实现这种转变所需的大部分技能和品质。

这里有七种你已经拥有的技能和特质，但你可能没有很好地运用它们。如果你能执行这些事情，你将比你所在领域的人有很大优势，如果你想改变职业生涯，你会发现很容易就能迈出第一步。

### 可靠

当有人需要你的帮助时，你会帮助他们吗？你是不是只有在真正生病的时候才会打电话请病假——而且这种情况非常罕见？人们会相信你在工作中做了你应该做的事情吗？

令人惊讶的是，可靠性实际上是非常罕见的，即使是在更高层次的工作。我知道很多人，当你给他们发邮件或打电话询问你需要他们做的特定任务或项目时，他们会说“当然！”然后再也不送东西了。同样，我知道许多人似乎从来不去工作——他们总是“生病”或“度假”

这些人不可靠，他们工作场所的人一定会注意到这一点，并受到负面影响。你可能是世界上最有技能的人，但是如果你对这些技能不可靠，它们就不会真正有用。

几乎世界上的每个组织都更愿意让一个可靠但相对不熟练的人坐在办公桌前，而不是一个高度熟练但不可靠的人。可靠的人也许不能创造出令人惊奇的东西，但是当你向他们要东西时，他们会创造出来；他们也会在你需要的时候出现。那个不可靠的人？他们可能会产生一些伟大的东西，但也可能什么都不会产生。

你能做些什么来提高你的可靠性？

*   **露面。成功的很大一部分就是表现出来。是的，每个人有时醒来都不想去工作。每个人都有宁愿呆在家里也不愿工作的时候。一个可靠的人无论如何都会进去，这样做在他们的工作场所变得更有价值。**
*   记录要求你做的事情，并迅速完成。 [保持一份待办事项清单](http://www.thesimpledollar.com/managing-a-never-ending-pile-of-things-to-do/) 尽可能高效地利用你的时间完成这份清单。优先考虑其他人赖以完成自己工作的事情。
*   不要浪费时间。即使在低迷时期，你也总有事情可以做。在忙碌的时候，做一些更容易应对挑战的工作。保持你的工作场所井然有序。培养技能。

### 做一个好的倾听者，尤其是在会议中

是的，许多工作场所的会议非常无聊。大多数时候，你宁愿呆在别的地方，做别的事情。当然，正是这种非常不受欢迎的会议性质使它们成为如此有价值的地方。

你所要做的就是在与他人会面时集中注意力，无论是一对一的会面还是大型的小组会议。尽可能地集中注意力，对你看到的和听到的做出自然的反应。

这里有三个非常值得采取的具体步骤。

*   **提问。**每当有人在展示某样东西，而你不清楚他们在说什么时，就去问一下。没有什么愚蠢的问题，句号。它不仅能给你所需要的信息，还能让演讲者保持警觉。
*   **记笔记。如果我应该参加一个会议，通常会有一个理由，即使对我来说并不明显。因此，我认为会议上讨论或介绍的任何内容都是有价值和重要的，即使看起来并不如此。我通常的解决办法是 [随身携带一个袖珍笔记本](http://www.thesimpledollar.com/how-to-use-a-simple-pocket-notebook-to-change-your-life/) 或硬皮速记本和一支笔，我会记下任何可能与我有丝毫关系的事情。之后，我会查看这些笔记，看看是否真的有什么需要我采取行动的。如果我能发现哪怕一两件事，那么我就领先了一两步。**
*   **跟进。**如果你在通读笔记后发现自己有问题，跟进并提出这些问题。同样，它不仅为你提供了更多的信息，还提供了一个与同事或主管建立职业关系的好机会。

### 迎接不想要的任务和挑战

总有人们不想承担的任务。很少有人喜欢当众讲话。很少有人喜欢领导小团体。很少有人喜欢负责协作报告。很少有人喜欢收拾最糟糕的烂摊子。

尽管这些任务可能并不有趣，但它们对于区分成功人士和不成功人士来说也非常有价值。站出来接受这些挑战的人会被留下来。他们是那些在晋升中被看好的人。他们是那些发现大门为他们敞开的人。他们是建立别人真正重视的新技能的人。

这里有三个步骤可以让你在自己的职业生涯中实现这一点。

*   当没有人举手时，举起你的手。是的，你可能会以一项困难且不愉快的任务结束。是的，如果没有它，你的一天可能会更轻松。另一方面，你可能会在这个过程中积累一些技能，你的上司会注意到你已经开始行动了。那会带来额外的好处。
*   不管任务有多艰难，都要全力以赴去完成它。是的，有时候你的工作会很糟糕。是的，有时候你不想做。这就是生活。成功和失败的区别在于结果。他们有没有处理好这个挑战？你准备好迎接挑战了吗？
*   采取额外的步骤来获得好的结果。简单地完成一项任务和把它做好往往有很大的区别。花额外的几分钟(或几小时，或任何需要的时间)把事情*做好*，而不是简单地做最少的事情。

### 避免消极的对话

出于各种原因，很容易陷入抨击同事的陷阱。也许他们没有达到你的期望。可能那个人性格有问题。也许那个人在群体中引起了不适当的冲突。

当然，这些都是在你内心隐藏潜在负面情绪的正当理由，但这并不意味着将这些情绪释放到野外是个好主意。在工作场合的谈话中，没有一个好的理由去抨击同事或说别人的坏话。从长远来看，不会有什么好结果。

这种消极谈话的最大负面结果之一是，说消极话的人最终被视为问题的一部分。抨击同事——即使是私下谈话——可能会以你意想不到的方式回来困扰你。

尽管如此，大多数工作场所仍然有负面的谈话和幕后的流言蜚语。人都有负面情绪，无时无刻不在发泄。以下是你如何应对这些情况，并确保自己不会被拖下水。

*   不要对工作中的某人或某事发表负面评论。没有理由提起你对某人或某事的负面感觉，*尤其是*突如其来。把这些感觉藏在你的后兜里，找些别的话题聊一聊。
*   对每个人都要积极，即使困难重重。对我来说，我发现思考我认识的人并寻找他们的优点真的很有用。地球上的每个人都是好坏特质的混合体；如果你花时间积极寻找人们身上的优点，你最终会以更积极的眼光看待他们。
*   如果你有强烈的负面情绪，找到另一种方式来发泄这些情绪。是的，我们都有需要发泄的时候。如果你需要发泄，找工作场所之外的朋友倾诉。写在你的日记里(我自己更喜欢这条路线)。去健身房打个包。任何不会在工作中引入更多消极因素的事情都是好事。
*   如果你无意中听到其他消极的对话，就避免积极参与其中。当然，如果很难做别的事情，你可以站在那里听。如果谈话变得消极，我通常会找些别的事情做。

### 一直学习和培养技能

*当你在工作时，许多工作都有停工期。许多工作需要通勤，这意味着更多的相对停工时间。很多人在家里还有额外的的空余时间。成功的最大关键之一是你如何选择利用这段休息时间，尤其是额外的工作时间。你是不是坐在那里无所事事？你会浏览网站和闲聊吗？你会去很长时间的洗手间吗？*

*当然，这些事情可能会让时间过得更快，但它们不会帮助你改善你的职业生涯或你对雇主的价值。他们只会浪费你生命中的每一分钟。试着把你的停工期和一点空闲时间花在其他事情上。*

*   *参加一些网络课程。你可以通过在线课程 获得很多认证，甚至 [一些学位，但即使你没有朝着学位迈进，你也可以通过在线课程](http://www.thesimpledollar.com/affordable-online-colleges/masters/) 学到很多东西。选择那些能培养你可能在工作中用到或者在你职业生涯的下一步中用到的技能的。*
*   *[**开博**](http://www.thesimpledollar.com/how-to-start-a-blog-a-side-business-primer/) **分享你的职业探索。当你学习新的东西时，用你自己的话重新解释，用对你来说有意义的类比，然后把这些文章贴到你自己的博客上。然后，上社交媒体，在脸书和推特上分享这些帖子，特别是和有相同兴趣的人分享。***
*   *向你的同事学习。他们做什么样的事情？他们带来了什么技能？他们在做什么？他们有什么技能，你怎么学？向你的同事提出这类问题可能是一种恭维。另外，你对他人的各种技能和任务了解得越多，你也会变得越有用。*

### *休息好，精力充沛*

*当我在那个办公室和数据库管理员一起工作时，正如我上面描述的，我们最终得到了一个小时工，他的工作是为我们手动输入数据。至少可以说，那是一次令人沮丧的经历。*

*当学生保持警觉和清醒时，他做得相当不错，尽管速度很慢。然而，有一半的时间他半醒着去上班，有时会趴在桌子上睡着。这不仅意味着当他在那里的时候，他没有生产任何东西，我们经常不得不停止我们自己的任务，唤醒这个小时工。*

*事实是，如果他只是带着一个清新的睡眠去工作，他不仅会更有效率，他也不会睡着，浪费我们的时间。这种简单的缺乏能量把对他来说可能是一个很好的职业机会——一个可以建立简历并提供一些很好的参考资料的机会——变成了一场灾难，最终他被解雇了。他需要做但没有做的一件事是带足够的能量和他一起工作。*

*以下是你应该考虑采取的三个步骤，这样你就不会陷入这种陷阱。*

*   *采取有充足睡眠和营养的日常生活方式。如果你发现自己白天很累，你要么是 [睡眠不足](http://www.thesimpledollar.com/feel-better-this-winter-get-more-sleep/) 要么是没有获得适当的营养。试着早点睡觉，如果过一会儿还是没用，打电话给你的医生。*
*   *了解你工作时的能量水平。我在早上做的第一件事是精力最充沛的，所以我试着 [承担我最具挑战性的任务](http://www.thesimpledollar.com/the-secret-of-the-launchpad/) 。我把精力和注意力较低的任务留到一天的晚些时候。这样，即使你的精力和注意力在一天中不是最强的时候，你也可以看起来为一天中的任何事情做好了准备。*
*   ***运动。**但是那*不就消耗*能量了吗？我们的身体真的很棒。事实证明，如果我们得到足够的锻炼，一旦我们习惯了这种变化，我们的身体和大脑实际上会变得更加*精力充沛。进行温和的锻炼可以提高你的自然能量水平，从而帮助你在工作中表现得更好。**

### *设定个人目标*

*五年后你想去哪里？十年后？那种生活是什么样子的？你想说多少细节都可以。现在，为了实现这一目标，你需要做些什么呢？从现在到那时，你的生活中需要改变什么，才能让这一愿景成为现实？你应该能想出一个不错的变化列表。*

*现在，你今天能做些什么来推进清单上的事情呢？今天。下周不行。不是“有一天”不是当你喜欢它的时候。*今。**

*一旦你开始考虑目标以及如何朝着目标努力，那么你生活中的许多事情就会变得井井有条。你开始看到你每天的选择不仅仅是孤立的事情，而是与你生活中的目标息息相关。*

*以下是深入研究的方法。*

*   *认真思考你想要你的生活走向何方。这是你早晚通勤的绝妙任务。你希望你的未来是什么样的？个人和职业上是什么样子的？要乐观，但不要不切实际。把事情保持在可能的范围内。*
*   *制定一个从这里到那里的计划。现在你有了这个未来的愿景，需要做些什么才能把你从现在的位置带到你想去的地方？需要采取什么步骤来实现这种转变？*
*   ***每天问自己*为了达到这个目标，你在个人和职业上都做了些什么。为了执行这个计划，你今天会做些什么？你现在*在做什么*来构建美好的未来？当你在工作中浪费时间，或者当你不去迎接挑战，或者当你懒得继续深造的时候，你是在实现美好的未来吗？****

### **最后的想法**

**这篇文章中提到的七个策略仅仅触及了你现在拥有的技能所能做的事情的表面。可以逃离低薪的工作。可以升职。你可以找到更好的工作。你只需要开始应用你所拥有的东西。**

**如果你愿意，你可以变得可靠。你可以为未来做一个计划。你可以听听。你可以更有条理。你可以建立新的技能。你可以表现得精神饱满。你可以承担艰巨的任务。**

**这些是你可以做的事情。就现在。每天都是。这些东西会让你得到一份更好的工作，一份更好的职业，更高的薪水。这些事情会带来比你想象的更多的机会。**

**你所要做的就是打开门。**
