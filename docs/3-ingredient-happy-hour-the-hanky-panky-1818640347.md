# 三种成分的快乐时光:手帕朱岳鹏

> 原文:[https://life hacker . com/3-ingredient-happy-hour-the-hanky-panky-1818640347](https://lifehacker.com/3-ingredient-happy-hour-the-hanky-panky-1818640347)

朋友们，周五快乐，欢迎回到 [*三种配料的欢乐时光*](http://lifehacker.com/tag/3-ingredient-happy-hour) ，每周饮品专栏，提供超级简单又美味的酒水。本周我们将喝我最喜欢的鸡尾酒，它来自伦敦的美国酒吧，汉克朱岳鹏。

Watch

美国酒吧——如此命名是因为鸡尾酒是美国人的发明——是一个独特的机构。说真的，这个地方什么都有*:免费赠送的鳀鱼馅橄榄，长茎双门酒杯，还有价值 5000 美元的葡萄酒。他们也是今天鸡尾酒的发源地，这是一种杜松子酒和苦艾酒，它从最微小的 Fernet 中获得刺激。*

*你可能还记得不久前 的 Fernet，当时我们用它的薄荷味，一种侵略性和振奋人心的味道来平衡 Campari 的苦味。我们今天会少用很多，但我想你会喜欢它给这款鸡尾酒带来的活力。美国酒吧实际上提供我们今天制作的饮料的复古版本，这是非常荒谬的，考虑到手帕朱岳鹏的主要成分之一是甜苦艾酒，它不太适合陈年。总之。为了保持新鲜感，您需要:*

*   *1.5 盎司杜松子酒*
*   *1.5 盎司甜苦艾酒(我喜欢多林罗索)*
*   *几个费尔内破折号*

*在加满冰块的混合杯里混合所有原料，搅拌直到所有原料都充分冷却，滤入小酒杯。一边享受经典的伦敦鸡尾酒，一边欣赏你不必穿过池塘的事实。*
