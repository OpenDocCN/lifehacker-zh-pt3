# 为这个光明节的成人 Dreidel 游戏制作大麻巧克力酱

> 原文：<https://lifehacker.com/make-cannabis-infused-chocolate-gelt-for-an-adult-game-1821264436>

Dreidel 是老姐版的赌博，或者说我从小就这么认为！ [正如历史所记载的](https://forward.com/culture/326379/the-true-history-of-the-dreidel/) ，它从爱尔兰一路穿越欧洲，在成为意第绪语和希伯来语的最爱之前。一个有四条规则的陀螺很酷，但是玩一点增强巧克力可以帮助你进入野蛮状态。制作自己的大麻巧克力硬币比听起来容易得多，并且不需要任何特殊的工具。

据我回忆，我们小时候得到的箔纸包装的巧克力并不好吃，即使在纽约市，它们也不是随处可得。我和一些朋友正计划在一个即将到来的聚餐上玩，因为它只对成年人开放，所以这是一个完美的机会，通过加入一些大麻来赌博，让事情变得更有趣一些。

当谈到巧克力食品时，无论你在巧克力中添加什么，保持体积小对于获得酥脆和真实的感觉和味道至关重要，使用油和黄油不会起作用，除非你想把黑巧克力变成牛奶巧克力。浓缩和杂碎是你保持这个过程简短、苦乐参半和有效的最佳选择。

收集物资并开始工作；从开始到结束不到一个小时。这也是一个很好的时机来提及你真的可以定制巧克力，如果你想用像脆皮爆米花这样的添加剂，像薄荷或香草这样的调味品，如果你想的话，还有白巧克力的漩涡。您将需要以下内容:

*   迷你纸杯蛋糕箔

*   各种其他纸杯蛋糕箔尺寸(可选)

*   8 盎司黑巧克力

*   克大麻浓缩物

*   玻璃量杯

*   烤盘

*   抹刀

将大约 24-30 个迷你纸杯蛋糕模具铺在烤盘上，尽可能保持它们的形状不变，以保持圆形的平底。将浓缩物在 225℉下脱羧 10 分钟，使 THC-A 转化为 THC，然后放在一边。然而，你更喜欢融化巧克力，在开始准备巧克力之前，先做好决定。一旦你有了融化装置，集中精力准备好，把巧克力切成小块，加入量杯。

在双层锅里慢慢融化，或者在微波炉里快速融化。当所有的东西都融化后，加入浓缩物，用折叠的动作搅拌均匀。现在你可以倒硬币了；每个人应该吃大约一茶匙到一茶匙的巧克力。

你倒得越浅，它们看起来就越像硬币，因为它们没有烘焙杯那么倾斜。尽量不要超过 1 英寸厚。当你全部倒好后，把烤盘在柜台上敲几下，确保没有气泡，巧克力已经一层一层地散开。

让巧克力完全凝固，当它完全变硬时，只需将纸杯蛋糕模具中多余的箔纸向内折皱，你就得到 gelt 了！
