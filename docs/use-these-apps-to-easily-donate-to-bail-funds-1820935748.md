# 使用这些应用程序可以轻松地向保释基金捐款

现金保释是司法系统中不成比例地伤害低收入人群的(许多)因素之一。这是改革和行动主义的一个热点问题，两个新的应用程序可以帮助你为这项事业做出贡献。

现金保释是为了保证被告不会在逮捕、传讯和审判之间消失。一名被告交付保释金——从几百美元到几千美元不等，取决于犯罪的严重程度和法官对逃跑风险的评估——以获得释放。付不起保释金的人不得不呆在监狱里，甚至在被定罪之前就被监禁。当然，付不起保释金的人往往是那些错过工作和审前监禁时间的其他不稳定因素伤害最大的人。

[Bail Bloc](https://bailbloc.thenewinquiry.com/) 是 New Inquiry 的一款桌面应用——它利用你电脑的一些未使用的处理能力来挖掘一种私人加密货币(称为 Monero)，当兑换成美元时，就会成为 [布朗克斯自由基金](http://www.thebronxfreedomfund.org/) 的资金，这是一个为被指控犯有轻罪的人支付保释金的组织。

不是凭空产生钱，而是克扣你的零花钱。就像硬币罐或储蓄计划，将你购买的东西四舍五入到最接近的美元，并将差额存入储蓄账户，Appolition 链接到你的借记卡，取出你甚至不会注意到的零钱，捐给 [国家纾困](https://nomoremoneybail.org/) 。

有很多方法来对抗现金保释——投票支持进步的 DAs、激进主义和对保释基金的大笔捐款——但这些应用程序会在你甚至没有注意的时候帮助你为一项良好的事业做出贡献。
