# 使用这些行为金融技巧省钱

> 原文:[https://life hacker . com/save-money-using-these-behavioral-finance-tricks-1823767864](https://lifehacker.com/save-money-using-these-behavioral-finance-tricks-1823767864)

我们都知道自动化储蓄和制定消费规则可以帮助我们省钱。但是来自杜克大学高级后见之明中心 [普通分币实验室](http://new.advanced-hindsight.com/commoncents-lab/) 的 [新报告](http://new.advanced-hindsight.com/wp-content/uploads/2018/03/Common-Cents-Lab-2017-Annual-Report.pdf) 为如何破解行为金融学以将更多的钱放进我们的口袋(或者更确切地说，银行账户)提供了新的见解。

这份报告有很多值得一读的内容，它详细介绍了该中心 2017 年的所有实验，重点关注老年人、低收入和小时工，但这里有一些关键的要点，你可以应用到自己的生活中。

### 利用年龄里程碑来实现你的目标

根据该报告，如果老年人被提醒即将到来的年龄里程碑，他们可以被鼓励改变习惯。该中心与 Silvernest 合作，试图增加使用该网站的房主数量，silver nest 是一家为老年房主和潜在租房者牵线搭桥的服务公司。他们发现，通过针对包含特定年龄里程碑信息的人群投放广告(“你 64 岁到 65 岁。你准备好退休了吗？房子合租可以有所帮助。”)而不是更通用的语言，他们能够将点击率从 2.46%提高到 5.49%。

这与过去的研究一致，过去的研究发现，当人们接近新的十年时(例如，当他们 29 岁、39 岁等时)，他们更有可能在行为/生活上做出重大改变。).你可以把这一点应用到你自己的生活中，列出你想要和需要完成的事情，在你自己的里程碑式生日之前的几个月里。通过在日历上设置提醒，告诉人们你的目标，来承诺真正完成任务。

### 限制你做某事的频率，而不是关注预算

在一项实验中，他们分析了个人金融应用 Qapital 的 3 万多笔交易，发现消费者后悔在餐馆、咖啡和快餐上的支出超过了其他类别。为了抑制这种情况，他们测试了不同的限制购买的方法。

他们的收获是:限制外出就餐次数(在这种情况下，每周外出两次)而不是花费金额的人更成功。所以在你自己的生活中，考虑设定一个支出频率，而不是 X 美元。

### 通过自动转账来减轻账单带来的痛苦

在与 Qapital 的另一项合作中，Common Cents 发现，千禧一代不太可能后悔重复交易，因为他们已经习惯于在每月的账单上看到这些交易。事实上，他们“对经常性交易的满意度比非经常性交易高 10%。”

“其中一个原因是因为人类是伟大的适应者，”报告中写道。“我们对某事的第一次体验是新奇而有趣的，但是在几次类似的体验之后，新鲜感和我们的注意力就会减弱，直到我们不再有同样的反应。撕一次创可贴就疼。多次撕掉它，我们就开始忽略痛苦。”

自动付款可以进行一致的交易，如租金、汽车付款等。，经济上不那么痛苦。“另一方面，它们让我们可能会后悔的付款(又一轮饮料或快餐)变得更加明显和新奇，因为后者更有可能用现金支付。”这样，自动化你的经常性付款是一种双赢:每月租金损失少一点，当你超支时，你会更加意识到支出(以及它给你带来的感受)。

### 拿起电话，准备等待

根据对 20 个人协商信用卡利率的观察和对 5，000 人关于账单协商的调查，Common Cents 发现降低年利率的最大障碍就是拿起电话等待。一旦消费者这样做了，他们的谈判技巧真的不会影响他们是否会得到更低的价格——他们所要做的就是问。

“最关键的一步是给信用卡公司打电话，”报告中写道。“一旦参与者拿起电话，他们继续完成任务的可能性非常高。”

所以，是的，你可能很忙，但是考虑一下:你只要问一下就可以得到更低的价格，不需要谈判。考虑到持有余额的家庭平均信用卡债务为 15，654 美元 ，从长远来看，这可以为你节省很多钱。
