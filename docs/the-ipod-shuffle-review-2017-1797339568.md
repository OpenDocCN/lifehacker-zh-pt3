# iPod Shuffle 回顾(2017)

> 原文：<https://gizmodo.com/the-ipod-shuffle-review-2017-1797339568>

苹果公司正在淘汰 iPod Shuffle，这是一个我直到昨天才想起来的小玩意。现在，我突然对一根十年来从未考虑过的白色小棍子充满了感情。我会非常想念它的。该死的苹果，别再玩弄我的心。

Watch

2006 年春末，我 14 岁的时候，在我们为两周年纪念日挑选的一家餐馆里，我的女朋友把一个盒子滑过桌子。我打开它，看到里面整齐地包装着一个运动的、石灰绿色的盒子，我高兴极了:一个 512 MB 的 iPod Shuffle。原始的 Shuffle，大约有半包口香糖那么大，有光泽的白色；这是我唯一拥有的 iPod，直到 2011 年我买了一部 iPhone，因为我羞于在写科技博客时使用功能手机。

今天，iPod shuffle 看起来像是遥远的过去的遗物；一个矮胖的 USB 2.0 拇指驱动器，上面套着一根发黄的挂绳，还有一个耳机插孔——这两项技术完全过时了，苹果也是如此。棒上有六个点击式的物理按钮(苹果也取消了这些按钮)，和一个组合随机电源开关，它可以将 120 首歌曲转换成配乐，供我拼车时播放。

那时候它几乎是完美的；一个可爱的笨拙的设备，相对来说 100 美元就能买得起。我喜欢这个该死的东西，精心照顾它，用一个丑陋的塑料果冻盒保护它，或者偶尔用一个 [改装的 Altoids tin](https://s-media-cache-ak0.pinimg.com/236x/24/5b/23/245b237bfc89e83faf0dd6d998692204--tin-hacks.jpg) ，因为我在网上看到了它的 DIY 指南。我发誓，它是完美无瑕的，直到大约 4 年后，一次短暂的奔跑中的绊倒给它留下了永远的伤痕。

昨天，当苹果公司毫不客气地把产品 从它的数码货架上拿下来时，我从抽屉里拿出我的产品，插上电源。现在挂在我脖子上的拖鞋在我拿到它几个月后就过时了。苹果在那年秋天重新设计了它的，又做了几次，直到 2010 年的最后一次更新。它在那里差不多七年没动过。对科技来说是永恒。

快速充电未能让我的 Shuffle 正常工作后，我决定让它插上电源过夜。

今天早上醒来，我听到一个充满电的洗牌和坏消息:它不会播放。它不会洗牌。一对孤独的绿色和橙色的灯闪烁着，指示着什么。在它的第一个生命周期即将结束的某个时候，我想起来我曾经把 Shuffle 当作一个真正的 USB 拇指驱动器来使用，并且把它擦干净，以便在 UPS 商店打印一些东西。今天早上，我在一台仍有 USB 接口的笔记本电脑上重新安装了 iTunes，然后点击口香糖形状的 Shuffle 图标棒来恢复它。

“欢迎使用你的新 iPod。”我的屏幕上出现了一条信息，上面有一个 iPod touch、一副有线耳塞和一个 iPod nano 的图片。我把它命名为“哈里森的最后一个 iPod”

接下来我需要歌曲。在从一台笔记本电脑到另一台笔记本电脑的多次迁移之后，我发现我的名下已经没有音乐了。从 CD 上翻录、从 Kazaa 上下载、翻录或从朋友那里下载的歌曲已经一去不复返了。相反，我登录了我的旧 iTunes 账户，在 iTunes 云端翻找旧专辑。

我曾经花 10 美元买的一些唱片从以太网上被抢购一空。本·科威勒尔的*莎莎*。仪表板忏悔的*所以不可能的 EP* 。西古尔罗斯。里洛·凯利。一首希拉里·达芙的歌。弗鲁弗鲁。我当然不会买 U2 的专辑。我点击了“自动填充”,砰的一声，44 首歌——3 小时 49 分钟的音乐——让我的生活重新回到正轨。我还有 264.7 MB 的空间。该慢跑了。

我弹出我的 iPod，扣上挂绳帽，滑动电源开关随机播放，戴上一副有线耳机，按下播放键。免费从我的手机。没有推送通知。不发微博。无法追踪我的距离或速度。只是一个令人耳目一新的简单的休息，优盘在我的胸前砰砰作响，上面有她在 t.A.T.u .说的所有事情。

我慢跑到附近的公园，路过戴着无线耳机和手机听音乐的人们——这是当今唯一实用的 iPods。在今天早上明亮的阳光下，指示灯太暗了，无法告诉我是否跳过了一首歌，也无法显示我还剩多少电池寿命。我所能做的就是倾听。

我的 shuffle 永远连不上互联网。对于现在只是从 Spotify 这样的服务上租借音乐的数百万音乐迷来说，这完全没用。当然，洗牌在今天没有任何意义，对我来说没有，对其他人来说也没有。真的，没有一个 iPod 能做到这一点——除非你想远离电网。

所以再见了，洗牌。这次我会带着音乐把你藏起来，这样有一天，在 iTunes 也死了之后，我可以拿几个 USB 加密狗和适配器，给你充电，再听你一次。
