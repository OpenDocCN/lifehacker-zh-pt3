# 你应该要求带薪病假的科学理由

> 原文：<https://lifehacker.com/the-scientific-reason-you-should-demand-paid-sick-days-1826008955>

美国公司不知道什么对它有好处——在这种情况下，带薪病假。美国是仅有的三个不要求雇主给带薪病假的工业化国家之一，而雇主经常决定不给带薪病假。截至 2011 年，只有 65%的美国员工带薪病假。



但是在过去的几年里，各州已经通过了自己的病假法。根据 2017 年的一项研究，强制带薪病假 [的城市和州的流感发病率平均下降了 5](https://voxeu.org/article/pros-and-cons-sick-pay) 个百分点。在一些流感期间，流感患者比没有强制病假的地方少 40 %!这是一个巨大的变化，尤其是去年冬天T5 导致 165 名儿童 T7】死亡，每千名美国人中就有一人住院的疾病！

根据这项研究，所有这些都是在没有显著降低工资或就业率的情况下完成的。但是，坦率地说，雇主可能并不关心这个国家的流感发病率是否降低了 40 %,他们可能会抱怨说，员工不敢滥用他们的病假。如果我们给他们病假，那些工人可能只是偶尔享受一天快乐的假期！

首先，那些雇主可以去舔柠檬了。如果他们不关心我们，我们也不必关心他们。第二，当德国在 1996 年将法定病假工资从 100%降至 80%时，上述研究的作者之一对此进行了研究。这确实减少了“不劳而获”的病假，但最终*增加了*真正的病假，以至于达到平衡。

这是个坏消息！事实是，更多生病的人留在工作岗位上，以避免 20%的减薪，反过来，让更多的同事生病。这最终增加了真正的病假天数，以至于全部扯平了。所以现在同样数量的人回家了，但是更多的人实际上生病了。德国雇主通过让人们生病来省钱——不仅仅是他们自己的员工，而是那些员工生病的任何人。在经济术语中，他们将一个的外部成本转嫁给了公众。

带薪病假让人们少生病，对雇员来说没有成本，对雇主来说只有边际成本。因此，如果你是 35%没有带薪病假的美国工人中的一员，那么就为争取一个城市或州的授权而斗争吧。去网站 [打开州](https://openstates.org/find_your_legislator/) 找到你的州议员的电话号码。当你生病时，你应该回家，你的雇主不应该因此惩罚你。

而且如果你确实有带薪病假， [*生病回家*](https://lifehacker.com/either-your-sick-co-worker-goes-home-or-you-do-1822172740) 。如果你是一个工作狂，这并不会降低你的传染性——在家工作。帮助建立一种公司文化，在这种文化中，病假是正常的，甚至是被鼓励的。因为在大多数公司，你的老板肯定不会开创这种趋势。
