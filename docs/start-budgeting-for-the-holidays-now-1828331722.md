# 现在就开始为假期做预算

> 原文：<https://lifehacker.com/start-budgeting-for-the-holidays-now-1828331722>

当你与 104 度的地铁站台 战斗并祈祷这个夏天最终结束时，你可能不会有这种感觉，但假期就要来临了。这意味着开始为礼物、航班、宴会和所有其他圣诞装饰品做预算。



全国零售联盟 估计美国人平均在感恩节和圣诞节花费 796 美元，其中 476 美元用于给家人和朋友的礼物。如果你需要在这两个假期中的一个或两个去旅行(更不用说除夕了)，那你的花费会更高。

在短时间内拿出这么多钱。所以，现在就开始计划吧。

### 制定(并保持)你的预算

根据《今日美国》 的调查，77%制定假日预算的人会超出预算，尽管通常不会超过 100 美元。尽管如此，在充满诱惑的时期，跟踪你的支出尤其重要。

《今日美国》指出:“如果你的超支意味着你不得不在假期后错过几次美味的晚餐或者在家多做几次便宜的饭菜，那么超出预算并没有造成大问题。”。“如果你的超支迫使你不支付水电费或者只用信用卡支付最低还款额，那你就有更大的问题了。”

决定你实际上能负担多少钱，并制定一个计划，从现在开始到你购物的这段时间里节省这笔钱。

你可以在你的银行、像 Ally 这样的网上银行或通过 Qapital 这样的应用程序建立一个单独的储蓄账户，为假期分离一些资金。你也可以做一个单独的旅行基金(下面有更多关于旅行的内容)。

当然，这将允许你现在就开始寻找交易，并在更长的时间内分散一些支出。当你购物时，使用像 [Digit 这样的应用程序，给自己](https://lifehacker.com/top-10-ways-to-save-money-right-before-the-holidays-1789539137) 或 [IFTTT with Qapital](https://twocents.lifehacker.com/automate-your-savings-with-ifttt-1821967872) 小费，为自己省点钱。

不要忘了给你的邮递员、收垃圾的、公寓管理员、孩子的老师等送些小礼物。

### 明智地对待旅行

机票比较网站 FareCompare 告诉《今日美国》 ,预订秋季航班的最佳时间是 8 月 27 日左右,“夏季周末和工作日的高票价都会下降，在感恩节大幅上涨之前，低票价将一直占据主导地位。”然后开始你的搜索(使用匿名浏览器)或尝试像 [Hopper](https://www.hopper.com/) 这样的应用程序来跟踪航班价格，如果可以的话，考虑用你的 [flexib le 计划](https://twocents.lifehacker.com/when-it-pays-to-buy-a-one-way-plane-ticket-1828295999#_ga=2.109117894.2105579250.1534164266-594046802.1524762060) 来获得最好的交易。

如果你开车旅行，使用像 GasBuddy 这样的应用程序来找到你所在地区最便宜的汽油。一些信用卡，我们将在下面讨论，也提供双倍现金购买汽油。

### 充分利用您的奖励

如果你要经常出差或购买昂贵的礼物，请确保你的积分累积和现金返还最大化。事实上，看一张新卡是值得的。

如果你在感恩节和圣诞节都要坐飞机(或者只是飞一趟昂贵的航班)，可以考虑一张旅行卡，比如 Capital One Venture 卡、[Chase Sapphire Reserve](https://points.theinventory.com/the-ultimate-guide-to-chase-ultimate-rewards-1825282156)或某家航空公司的卡。如果您在接下来的三个月内使用这些卡购买总额为 3000 至 4000 美元的商品，您不仅会累积奖励，而且您还可以 [赚取足够的奖励积分](https://twocents.lifehacker.com/how-to-earn-yearly-point-bonuses-on-your-credit-cards-1827997865) 来支付您回家的机票或圣诞节时送给您岳父母的费用。

其他基本的返现卡，如花旗双现金卡，提供相当不错的综合回报率。如果你打算在亚马逊上大量购物，prime rewards 信用卡 和亚马逊商店卡提供高达 5%的购物返现。 [大通、花旗和好市多](https://creditcards.usnews.com/gas) 在汽油上提供相当不错的奖励。

当然，你会想小心翼翼地把所有的东西都记到你的卡上(这就是你做预算的原因)，但是有策略地使用，你会收获假期的回报。
