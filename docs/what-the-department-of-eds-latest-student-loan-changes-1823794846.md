# 教育部最新的学生贷款变化对你意味着什么

> 原文：<https://lifehacker.com/what-the-department-of-eds-latest-student-loan-changes-1823794846>

教育部正在与各州 就他们更严格监管学生贷款服务公司的努力进行斗争，消费者权益倡导者称这可能会增加管理不善和欺骗性贷款行为的机会。

Watch

在贝琪·德沃斯的领导下，教育部已经采取了一系列措施，放松对学生贷款服务公司的监管，这些公司负责处理由教育部监管的联邦贷款账单。据《华盛顿邮报》(Washington Post)报道，包括康涅狄格州、【加利福尼亚州】和伊利诺伊州 在内的几个州[](https://www.marketwatch.com/story/states-to-devos-well-keep-cracking-down-on-student-loan-companies-2018-03-02#false)[最近已经通过或试图通过自己的立法，以更密切地监控这些公司，包括制定](https://www.washingtonpost.com/news/grade-point/wp/2018/03/09/education-dept-tells-student-loan-companies-to-ignore-state-authority/?utm_term=.014f7b6a5c4e) 一项“借款人权利法案，其中包含及时付款处理、错误纠正和沟通的最低标准”。教育部认为各州缺乏管理它们的权力。

这只是该部门在过去一年中采取的许多行动之一，消费者权益倡导者说贷款公司比学生更受青睐。教育部还表示，它将停止与消费者金融保护局的合作，并将阻止 CFPB 保护学生贷款借款人的努力，以及旨在保护学生免受掠夺性营利性学校伤害的延迟规则。去年三月， [【德沃斯】](https://www.consumerreports.org/student-loans/what-to-do-if-you-face-trouble-repaying-a-student-loan/)[取消了 60 天的宽限期](https://ifap.ed.gov/dpcletters/GEN1702.html) 以允许贷款违约的学生回到正轨，并避免他们贷款余额 16%的费用

那么，所有这些对借款人来说意味着什么？正如 [《消费者报告》](https://www.consumerreports.org/paying-for-college/what-battle-over-student-loan-servicers-means-for-borrowers/) 指出的那样，当各州和联邦政府在法庭上一决雌雄时，学生们将不得不比以往更加小心。“这种转变意味着学生借款人需要更加警惕，自己研究所有的还款选择，仔细检查他们的还款是否正确，并始终保持良好的记录，” [写道。以下是他们的建议:](https://www.consumerreports.org/student-loans/what-to-do-if-you-face-trouble-repaying-a-student-loan/)

### **做好记录**

看似很基本，但你需要知道你的服务者是谁(每月给你账单的公司)，你有哪种贷款(联邦还是私人)。例如，在 Excel 文档中记录你每个月已经付了多少钱，欠了多少钱。如果你不知道而且他们是联邦贷款，CR 建议在这个 [国家数据库](https://www.nsldsfap.ed.gov/nslds_FAP/) ( [下面是如何使用](https://studentloanhero.com/featured/how-to-use-the-nslds-to-find-your-student-loans-and-total-debt/) )。

### **了解你的选择**

根据 CR 的说法，不幸的是，如果你在偿还贷款方面遇到困难，并想改变你的还款计划，你的贷款服务机构不太可能提供超级帮助，所以使用教育部的 [还款估计器](https://studentloans.gov/myDirectLoan/mobile/repayment/repaymentEstimator.action) 或 CFPB 的 [工具](https://www.consumerfinance.gov/paying-for-college/repay-student-debt/) 来了解每种类型的还款计划。私人贷款没有太多的还款选择，但是如果需要的话，可以和你的服务商谈谈暂停还款。

### **投诉**

一些较大的服务机构有 [一个特派员](https://www.cappex.com/hq/articles-and-advice/financial-aid/student-loans/How-to-Complain-about-Student-Loan-Problems) 你可以就有关问题进行联系:

*   [AES/PHEAA](http://www.aessuccess.org/contact/office_consumer_advocacy.shtml)

*   [【ECMC 教育信贷管理公司】](https://www.ecmc.org/borrowers/account-resolution.html)
*   五大湖高等教育担保公司
*   [国家助学贷款项目](http://www.nslp.org/borrowers/ombudsman/)
*   [纳威](https://www.navient.com/about/who-we-are/leadership/advocate/default.aspx)T3】
*   [莎莉美](https://www.salliemae.com/about/who-we-are/leadership/advocate/default.aspx)T3】
*   德克萨斯担保学生贷款公司

如果不起作用， [向 CFPB](https://www.consumerfinance.gov/complaint/#student-loan)投诉(而 [阅读别人的投诉](https://www.consumerfinance.gov/data-research/consumer-complaints/) )，并且，如果你有联邦贷款，向 [教育部](https://feedback.studentaid.ed.gov/s/?language=en_US) 。
