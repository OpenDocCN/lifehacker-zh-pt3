# RIP，微软画图

> 原文：<https://lifehacker.com/rip-microsoft-paint-1797193660>

你用来编辑图像的第一个应用程序[MS Paint 可能会在 Windows 10](https://www.theguardian.com/technology/2017/jul/24/microsoft-paint-kill-off-after-32-years-graphics-editing-program) 的未来更新中被取消，取而代之的是新的应用程序 Paint 3D。微软在 Windows 10 的下一次秋季更新中将 32 岁的应用程序列在 [【弃用功能】](https://support.microsoft.com/en-us/help/4034825/features-that-are-removed-or-deprecated-in-windows-10-fall-creators-up) 下，一个小 X 标志着一个时代的结束。

Watch

这款应用无疑是一个遗迹，在那个时代，普通电脑用户无法打开 Photoshop、Skitch、Pixelmator 或数以千计的网络应用。MS Paint 无法将图像组件保存为图层或矢量；它仅用于制作平面静态图像。它不能平滑线条或猜测你的最佳意图。它做你告诉它的事情，仅此而已，忠实地表现了用电脑鼠标徒手画的不稳定运动。这是在触摸和触控板出现之前的时代。

随着更复杂的选项出现，Paint 的 janky 美学成为一种有意识的选择。“绘画”成了这种美学的代名词，即使一幅图像实际上是在另一个应用程序中创建的。 [TV Tropes 列出了定义某种外观的主要限制](http://tvtropes.org/pmwiki/pmwiki.php/UsefulNotes/MSPaint) :摇摆不定的手绘线条、笨拙的色彩处理，以及赋予 Paint 独特外观的不恰当的导出设置。

2014 年，Gawker 的 Sam Biddle 注意到了 [Paint 对阴谋论图片](http://gawker.com/chart-brut-how-the-ms-paint-graphics-of-conspiracy-too-1651851261) 的影响，称这种形式为“图表暴力”在业余侦探试图识别波士顿马拉松爆炸案的过程中，油漆的简单和参差不齐让人想起红色线条和潦草笔记的“疯狂的墙”美学，显然没有讽刺意味。

同年，互联网历史学家帕特里克·戴维森探索了 [颜料对过去十年迷因文化](http://journals.sagepub.com/doi/pdf/10.1177/1470412914544539) ，尤其是 [狂怒漫画](https://www.reddit.com/r/classicrage/top/) 的影响。局外人的艺术审美感觉适合相关的日常内容，并使艺术形式没有威胁。

当然，Paint 提供了一些功能来平滑事物，如圆形和线条工具以及“填充”工具，所有这些都在 10 年代早期的 [stoner 漫画](https://www.reddit.com/r/treecomics/top/) 中使用。重要的是，这些圆圈仍然有锯齿状的曲线。斯托纳漫画的亮色很平淡，因为 MS Paint 不支持渐变(没有 [精心破解](https://www.youtube.com/watch?v=VGQnimByXio) )。

通过自动改进艺术，Paint 3D 隐藏了图像背后的过程。颜料的粗糙可能是狂怒漫画如此受欢迎的原因。看着一幅狂怒漫画，你可以准确地说出它是如何画出来的，以及你自己可能是如何画出来的。通过准确地传递艺术家所画的东西，MS Paint 形成了一个图像，观众可以在心理上进行逆向工程和模仿。

除非你完全疯了。

Reddit 用户 Toweringhorizon [使用像](https://www.reddit.com/r/mspaint/comments/5l3lfu/to_a_little_radio/) [油漆刷](https://www.reddit.com/r/mspaint/comments/5l3lfu/to_a_little_radio/dbtiq2x/) 这样的 MS Paint 工具，煞费苦心地组装了这幅“给一个小收音机” 的画，拉伸了介质，同时保持了像素化的外观。这是一个美丽的合作艺术画廊[MS Paint subreddit](https://www.reddit.com/r/mspaint/)的顶级作品之一。浏览这幅画感觉就像翻阅高中最有艺术天赋的孩子的素描本。有一种被接受的粗糙感，一种渴望的极简主义。例如，上面这幅作品中精致的雨滴被反射在一个平坦、毫无特色的桌面上。就像晶体管收音机一样，油漆可能正在显示它的年龄，但是这个顽强的小玩意不应该被低估。

“对着一个小收音机”甚至没有接近测试油漆的极限。当我们告别这个塑造了一个时代的应用程序时，让我们来看看这个用 MS Paint 在 Windows 7 上绘制圣诞老人的 50 个小时的奇怪配乐延时。我们只能相信这是真的，因为伪造会更难。
