# 为什么完美主义和倦怠相伴而生

> 原文：<https://lifehacker.com/why-perfectionism-and-burnout-go-hand-in-hand-1763361142>

完美主义通常被视为一种美德——尤其是在工作场所——但研究表明，自我施加的压力表现完美只会导致最终的倦怠。完美不是你衡量成功的标准。



*本帖原载于* [*对话*](https://theconversation.com/perfectionism-and-burn-out-are-close-friends-best-avoid-them-44727) *。*

伏尔泰说过:“完美是美好的敌人”——他应该知道这一点。伏尔泰是存在主义完美主义的尖锐批评者，他职业生涯的大部分时间都在攻击一个充满完美神性的世界的概念。

在他最有影响力的作品《老实人》中，伏尔泰描述了庞洛斯教授，他是 [莱布尼茨乐观主义](https://en.wikipedia.org/wiki/Best_of_all_possible_worlds) 的忠实信徒，这种哲学认为上帝创造了“所有可能的世界中最好的”坎迪德是庞洛斯的糊涂学生，经常试图用理想世界的概念来调和他导师的道德缺陷，但最终绝望了。只有通过这些失败，坎迪德才痛苦地治愈了他对完美的痴迷。

快进几个世纪，坎迪德学到的经验今天仍然适用。我们生活在一个追求完美的世界里。从运动场，到教室，到办公室，以及两者之间的任何地方；完美的表现是 [巅峰](http://www.thetimes.co.uk/tto/sport/tennis/article4494515.ece)——对许多人来说，这就是成功的定义。然而，正如 Candide 所发现的，这种逻辑有一个固有的缺陷。因为完美是一个永远无法达到的标准，最终会给那些尝试的人带来所有的 [和痛苦](http://www.ncbi.nlm.nih.gov/pubmed/8561378) 。

### **常见的恐惧**

作为心理学家，我们一直对完美主义的影响着迷——完美主义是一种包含过高个人标准和苛刻自我批评的人格特质。这是一种常见的特质，你很可能会认识有这种特质的人:害怕犯错的同事，表现不佳后过分自我批评的队友，或者为错过学校最后期限而闷闷不乐的孩子。这些都是完美主义者的典型特征。

我们最近 [进行了研究](http://opus.bath.ac.uk/45449/) 发现完美主义与倦怠密切相关。倦怠是一种与慢性压力相关的综合征，表现为极度疲劳、成就感降低和最终脱离。

在我们的分析中，有趣的是，完美主义的核心自我意识思想和感觉，那些与对不完美和犯错的恐惧相关的思想和感觉，对倦怠水平有中等至大的积极影响。更有趣的是，与体育和教育相比，这种关系在工作环境中尤为明显。

我们的分析没有发现为什么完美主义在工作中特别成问题，但是有很多可能性。工作通常以绩效为中心——糟糕的绩效会带来巨大的成本，在最糟糕的情况下，会导致 [冗余](https://www.gov.uk/staff-redundant/compulsory-redundancy) 。当完美主义在这种压力下发生时，表现结果，而不是激励，很可能成为严重压力的触发因素——加剧对失败的恐惧，这是完美主义和倦怠之间的联系的核心。

### **感觉被困在工作中**

完美主义和工作倦怠之间紧密联系的另一个解释是，员工有时会感到被困住了。尤其是在经济不景气的时候，跳槽或者转行的机会有限。对于完美主义者来说，这种无法将自己从压力越来越大、不再令人愉快的工作环境中解脱出来的能力，极有可能导致精疲力竭。

重要的是要认识到，在理清完美主义和倦怠之间的关系时，我们的发现强调了一个更广泛的共识。也就是说，完美主义是一种极具破坏性的特质，其影响必须得到控制。许多完美主义者感觉到 长期的自我意识，被恐惧攫住，认为自己是骗子。

因此，组织必须清楚，完美不是成功的标准。相反，勤奋、灵活和毅力是比完美主义好得多的品质。谷歌采取了大胆的举措，奖励失败 ，试图消除抑制绩效的焦虑——这很可能被证明是极其富有成效的:不仅有利于创新，也有利于员工的福祉。

此外，由于追求完美是不理性的和有害的，所以建议设定更现实的(但仍然具有挑战性的)目标。不可避免的是，这些目标会低于完美主义者的期望——但是完美主义者需要自我保护。否则，工作狂、倦怠——甚至更糟——等着你。

作为一个社会，我们倾向于将完美主义视为美德和成就的标志。然而，坎迪德的故事是发人深省的；完美既迷人又难以捉摸。专注于完美最终会弄巧成拙，我们的研究清楚地揭示了这一事实。平衡的工作生活和减压的环境可能会在一定程度上解决完美主义的问题。但是，作为一个社会，我们也有责任挑战这种不良特征。
