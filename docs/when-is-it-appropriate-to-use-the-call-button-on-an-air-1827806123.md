# 在飞机上什么时候使用呼叫按钮比较合适？

> 原文：<https://lifehacker.com/when-is-it-appropriate-to-use-the-call-button-on-an-air-1827806123>

你几乎肯定不喜欢坐飞机:费用、不舒服、b 座那个烦人的人。只要记住，空乘人员的经历可能比你更糟糕，他们不能抱怨，因为这是他们的工作。所有这些都是说，总的来说，考虑你的空乘人员的体验是很重要的。而且似乎呼叫按钮是在运输过程中引起不良行为的一个特别常见的原因。

就在昨天，在一次从佛罗里达到德克萨斯的短途飞行中，一名乘务员不得不通过公共广播系统宣布，如果不是紧急情况，按下呼叫按钮的人应该关掉它；我们还没有到达他们可以安全走过的高度。乘务员和乘客一样受气流影响！

通过这个 [帖子](https://thepointsguy.com/guide/jetiquette-when-to-use-the-call-button/) 来自*The point Guy*关于“航空礼仪”因为它与所谓的“权利按钮”有关，下面是一些基本规则，下次你在飞机上急需另一瓶迷你酒时要记住:

### 如果是紧急情况就推

你心跳停止了吗？你刚刚分娩吗？有没有一个只有你能看到的怪物在飞机的机翼上，摆弄着发动机？好的，按那个按钮。国际空乘协会主席萨拉·尼尔森向 TPG 解释说，“首先，这确实是紧急情况下使用的。”大多数人不会在飞机座位上遇到紧急情况，除非飞机正在坠落。在这种情况下，请伸手戴上氧气面罩。他们意识到了这个问题。

### 推动沟通

有时，空乘人员需要联系您，并通过 PA 系统发布通知。也许他们在寻找需要帮助下机的客人；无人陪伴的未成年人；或者你，是的，你，特别是。尼尔森说，在这种情况下，你使用按钮是为了它的设计目的:一种简单的方式来表明你在一个装满人的长飞行管道中的位置。这样你就不用像个傻瓜一样在过道里大喊大叫了。完美。

### 如果你被困住了就推

也许你坐在靠窗的位置，而你的同座挡住了你去过道的路。他们也完全昏迷了。或者也许你抱着一个在飞机上不容易操纵的婴儿。在这种情况下，按下按钮与人交谈是有意义的，服务员可能更喜欢它，而不是让一个孩子(或成人)哭。但是如果你有孩子，不要让他们玩呼叫按钮。一点都不可爱。

### 不要为了服务而推它

作家兼礼仪顾问黛安·戈特斯曼(Diane Gottsman)提出了这个问题，让你在按门铃前问问自己:“这件事我自己能做吗，还是我需要帮助？这是试金石。”

> “虽然空乘人员是为了让你的飞行舒适，但这不是服务行业。他们不是服务员，”戈特斯曼解释道。“如果你真的需要帮助，这就是呼叫按钮的作用。如果你只是渴了，饮料车还没来，那就坐着别动，耐心点。”

这是一个有趣的观点。我绝对坐过飞机，在飞机上，他们似乎试图向你推销尽可能多的饮料，所以几杯叮铃铃可能是受欢迎的。但是如果是基本服务，你看到几排下面的推车，不要按呼叫按钮让它走得更快。不会的。乘务员意识到人们需要像水和扔垃圾这样的东西。它们会在适当的时间间隔出现。

记住，飞机的主要目的是把你从一个地方带到另一个地方。如果你想要五星级质量的食物和服务，把钱花在本质上。
