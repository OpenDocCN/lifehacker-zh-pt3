# 为什么柯达的比特币计划是一个你应该避免的骗局

> 原文：<https://lifehacker.com/why-kodaks-bitcoin-scheme-is-a-scam-you-should-avoid-1821957658>

就像你大学时的朋友不停地在脸书上发帖一样，柯达真的对 [加密货币](https://lifehacker.com/preview/beyond-bitcoin-your-guide-to-the-other-major-cryptocur-1821656104?rev=1514581574792) 着迷。在 [CES 2018](https://lifehacker.com/the-first-google-home-with-a-screen-already-offers-more-1821910986) 期间，该公司宣布了一项双管齐下的区块链战略:1)其名为 [KodakCoin](https://www.kodak.com/kodakone/default.htm) 的自有加密货币，旨在改善摄影师的图像权利；2)名为 KashMiner 的比特币开采计划，客户可以租赁设备，据称可以在两年内赚钱。



第一个主意不错。它使用区块链技术来解决一个合法的问题，柯达的成熟品牌可以帮助普及图像权利公共分类账的想法，以确保摄影师从他们的工作中获得报酬。然而，第二个声明很快被认为是一个骗局，证明了比特币泡沫即将破裂。

以下是 KashMiner 的问题，以及为什么您可能应该不惜一切代价避免它。

 [https://lifehacker.com/embed/inset/iframe?id=twitter-950861459302445056&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-950861459302445056&autosize=1) 

从表面上看，柯达的说法似乎站得住脚。本质上，该公司要求你支付 3400 美元作为购买比特币采矿平台的前期投资。Spotlite 将授权柯达品牌经营这项业务，它将支付维护费用，你们将分享利润。该公司估计，根据比特币的当前价值和当前的开采率，你每个月可以赚 375 美元(两年共 9000 美元)，但这正是整个计划的漏洞所在。

比特币挖矿的一个基本原则是，随着时间的推移，它会变得更加困难。因此，未来两年，这些喀什矿工之一的比特币生产率将大幅下降。比特币的价值有可能(甚至有可能)同时上涨，但不能保证这足以抵消投资。

据一位“比特币经济学家”称，最终利润可能只有 2457 美元。这远远低于 Spotlite 和柯达承诺的 5600 美元(9000 美元减去最初的 3400 美元投资)。

 [https://lifehacker.com/embed/inset/iframe?id=twitter-950884829435965440&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-950884829435965440&autosize=1) 

这与一开始的承诺相差甚远，考虑到比特币的波动性，你最终可能会变得更糟。唯一真正的赢家可能是柯达的股东，在本周的公告后，他们看到公司的价值上涨了 60%。
