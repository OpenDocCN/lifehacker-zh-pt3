# 在家工作可能需要许可证

> 原文：<https://lifehacker.com/you-might-need-a-permit-to-work-from-home-1825154941>

去年秋天，我搬到了一个新的城市——今年春天，我得知我需要一个家庭居住许可证才能在我的公寓里做自由职业者。

Watch

居家规则因城市而异，大多数自由职业者可能都没有意识到它们的存在；自从我开始做自由作家以来，我已经搬了不止一次家，我知道我需要查看营业执照等等，但我从未想过要获得城市允许在家工作。我的意思是，不管我们是全职自由职业者还是兼职，这不正是我们大多数人最近在做的事情吗？

住宅占用许可证最初是为了管理可能对住宅区造成潜在破坏的家庭企业类型，如日托和发廊。一些城市，如波特兰，根据给社区带来的客流量，将居家职业分为两种类型 :自由作家不需要居家许可，但自由职业家庭教师需要。其他城市要求所有的小企业主和独资经营者如果想在家里经营他们的生意，必须获得住房许可证。即使你是一个兼职骗子，你可能仍然需要申请许可，让你的家庭生意变得正式。

## 如何确定你是否需要住宅入伙纸

访问你所在城市的网站，寻找标题为“小企业”和“分区”的部分关于住房许可的信息应该在其中一个部分。如果你找不到你要找的东西，或者如果你想彻底了解，打电话给你的城市规划办公室询问。我就是这么做的，他们很有帮助！

请注意，即使你的公司地址与你的家庭地址不同，你也可能需要家庭居住许可证。如果你大部分时间在家工作，城市可能会认为这是一种居家工作。

## 如何获得房东的许可

如果你没有自己的房子，你可能需要证明你的房东已经签署了你的住房占用之前，城市会给你一个许可证。如果你拥有一套公寓或者是房主协会的成员，你可能需要物业经理和/或 HOA 的批准。在某些情况下，你甚至需要公证文件。

在我的情况下，我的房东已经知道我在家工作；当我参观这栋楼的时候，我们讨论过这个问题，我在我的租房申请表上写得很清楚。这使得住宅占用许可证的谈话相对容易:我发了一封电子邮件解释说我需要许可证，附上了许可证的副本供他审查，并请我的房东再次确认他对我在家工作感到满意。

如果你在搬进自己的公寓/公寓/HOA 的家后转变为自由职业者，你可能需要请求许可，而不是再次确认。这可能会感觉有点尴尬，但不要让它吓倒你。尽可能直截了当地谈论你的工作(“我每周花大约 40 个小时在笔记本电脑上写文章，偶尔通过电话/Skype 接受采访”)以及它对邻居的影响。

## 如果你跳过许可会发生什么？

此时你可能在想“在我的城市里有成千上万的自由职业者。他们不可能都有许可证，对吧？如果我只是...没得一个？”

好吧，当然。你可以在没有许可证的情况下继续在家工作，而且你可能不会被抓住——毕竟，你已经在家工作了一段时间，没有人试图阻止你。如果这座城市发现了你的家庭生意，会发生以下两种情况之一:

可能的情况:市政府会给你发一封信，内容大致如下:“嘿，我们看到你在家工作。我们需要您填写这份许可申请，我们可能会因为您没有提前申请许可而对您进行罚款。”

最糟糕的情况:市政府会给你发一封信，要求你立即停业。

继续之前，评估你的风险承受能力。

## 如果你申请许可证却被拒绝了，会发生什么？

如果您的住宅居住许可证未获批准，您可以提起上诉。如果所有这些都失败了，你需要停止在家工作。这很棘手，因为大多数人——不管是雇员还是自由职业者——每天都在家工作；早上的邮件检查，深夜的电子表格，所有不适合工作时间的东西。

就你的情况而言，你可能想找一个共同工作的地方，或者租一间办公室，在那里你可以开展大部分业务。是的，你最终还是会打一些商务电话或者从家里发一些电子邮件，因为*每个人都这么做*。然而，你的日常工作将在其他地方进行。

向你所在城市的小企业协会寻求建议，并再次打电话给你的分区办公室，以澄清你的计划:例如，你是否需要开设一个邮箱，以便你的企业不再与你的家庭地址相关联？

记住:你的城市希望你成功。住房占用许可证更多的是保持商业区和住宅区之间的界限，如果你能证明你的企业没有违反这些界限，你很可能会获得批准。

是的——你做自由职业来赚取额外的钱也算一笔生意。所以就这样对待它，即使这意味着申请在家工作的许可。
