# 当你厌倦了劳雷尔和燕妮的时候，听听这些听觉错觉

> 原文：<https://lifehacker.com/listen-to-these-auditory-illusions-when-you-re-bored-wi-1826081147>

单词“laurel”的声音片段在一些人(错误的人)听起来像“yanny”在一个新的，听觉上对应于 [的服装](https://sploid.gizmodo.com/some-people-see-this-dress-as-white-gold-while-others-1688330144) 。已经听过了？太好了！这里有一些更令人难以置信的音频剪辑，你应该听听。



### 麦古克效应

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-PWGeUztTkRA&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-PWGeUztTkRA&start=0) 

**你会听到:**根据说话者移动嘴巴的方式，你会听到伴随的声音，可能是“巴”、“瓦”或“打”

工作原理:这些声音都非常相似，所以我们的大脑使用视觉线索来明确我们所听到的。如果它听起来像“ba”，但这个人看起来很像在说“va”，我们就用“va”

### 相同的声音，不同的单词

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-KiuO_Z2_AD4&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-KiuO_Z2_AD4&start=0) 

**你将听到:**首先是单词“bill”，然后是“bale”、“pail”和“mayo”

工作原理:这些单词的发音都一样，或者至少在这段录音中听起来非常相似。当你看到与之相匹配的图像时，你的大脑会强调与之相匹配的声音，类似于麦克格鲁克效应。如果你仔细想想的话，bill 中的 L 和 mayo 中的 yo 并没有什么不同。

### 幻像词

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-ZY6h3pKqYI0&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-ZY6h3pKqYI0&start=0) 

你将听到的:这首歌只是一堆合成的钢琴音符，但你会发誓你能听到比吉斯的歌声。(几年前有一个 [玛丽亚·凯莉这样的片段](https://www.newscientist.com/article/dn28683-how-trippy-version-of-mariah-carey-christmas-hit-fools-the-brain/) 流传开来。)

**工作原理:**原始剪辑中的人声被转换成数码钢琴音符——不仅仅是每个音节一个音符，而是足够多的音符，听起来有点像我们在听唱歌时听到的所有不同的声音。如果你从未听过《活着》，你可能听不到所有的歌词。听听原文——上面的视频中间有一个剪辑，很有帮助——然后你会更清楚地听到幻像词。这种 [也会发生在你听乱码音频](https://gizmodo.com/this-audio-illusion-will-make-you-never-trust-your-ears-1593113324) 的时候，就像火车售票员喊停站一样——如果你知道应该停在哪里，即使你的外地朋友听不出来，你也会认出这些单词。

### 选择你自己的音节冒险

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-kB0I5x_Skzg&start=25](https://lifehacker.com/embed/inset/iframe?id=youtube-video-kB0I5x_Skzg&start=25) 

**你将会听到:**在这段视频的片段中，你将会反复听到听起来像是同一个单词的声音。那是什么词呢？

工作原理:这些音轨同时播放来自左边扬声器的一个音节和来自右边扬声器的另一个音节。音节交替，你的大脑会寻找模式，努力识别单词。(这一招没有耳机还是管用的，但是有了耳机就好多了。)这些声音非常模糊，因此这些单词有很多种可能。正如视频中所解释的，研究人员 Diana Deutsch 发现，你的感知可能取决于一些条件，比如你听音乐时的心情。

### 谢泼德音

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-BzNzgsAE4F0&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-BzNzgsAE4F0&start=0) 

你将听到的:音调升高的声音...并且持续上升...并且持续上升...当你到达轨道的终点时，你可以从头开始，它会一直走下去。

**工作原理:**就像是那些无限楼梯的听觉版。你同时听到多个音调，它们随着时间的推移而升高——想象一下，从低音到高音，一次弹奏一个音符。但每个音符其实都是低音和高音的集合。最高音变得越来越安静，随着最低音变得越来越大声而逐渐消失，所以不管怎样，中间总是有音符看起来音调升高了。还是很困惑？查看T3 中的视觉效果。
