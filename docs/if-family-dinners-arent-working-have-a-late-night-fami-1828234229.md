# 如果家庭聚餐不起作用，那就吃一顿家庭夜宵

> 原文：<https://lifehacker.com/if-family-dinners-arent-working-have-a-late-night-fami-1828234229>

关于家庭聚餐的重要性已经写了很多。每天都要留出时间，远离工作、家务和 Instagram 通知的干扰，与你的家人分享食物。在这种仪式中长大的孩子将与那些不这样做的孩子有根本的不同——科学 [证实了这一点](https://www.healthychildren.org/English/family-life/family-dynamics/Pages/Mealtime-as-Family-Time.aspx) 。然而，永恒的挑战是找到时间，尤其是当你的父母工作到很晚，或者多个孩子的时间表相互冲突的时候。现实生活看起来不像诺曼·洛克威尔的感恩节画作，而更像是一个高速公路休息站，随时都有人在进进出出。

当我们想到家庭聚餐时，我们通常会想到晚餐。但是，在繁忙的夜晚，坐一个小时对每个人来说都没有意义。如果它在你的房子里不起作用，那就是把你赶出去，要知道有很多好的选择。哈佛大学心理学教授、家庭晚餐项目 [的联合创始人安妮·k·费希尔博士写道，](https://thefamilydinnerproject.org/resources/faq/) 一周有五天晚上一起吃晚餐的家庭“受益匪浅”*但是*“没有神奇的数字，晚餐也不是天生就比其他食物更好。”

我喜欢的一个选择是深夜家庭小吃。(*称之为 [表时间](https://www.nytimes.com/guides/well/make-most-of-family-table) 。)我不是说很晚，比如午夜，而是说，晚上 8:40。工作原理:当每个人从办公室或学校的音乐排练或足球训练回家后，你们都穿上睡衣，然后围坐在桌子旁喝热巧克力或水果汽水或几碗麦片和牛奶。你可以悠闲地赶上进度，而不必考虑扎迪正在吃多少块西兰花，或者莱德还有多少数学作业要做，或者你需要往洗碗机里装多少锅。随着每个人都完成了一天的工作，你可以利用这段时间放松和拔掉插头(也许可以把你的手机放在家庭坞站)。谁知道呢？这可能会成为你最期待的传统。*
