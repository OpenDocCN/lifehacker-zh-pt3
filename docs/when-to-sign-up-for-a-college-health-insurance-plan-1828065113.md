# 何时报名参加大学健康保险计划

> 原文：<https://lifehacker.com/when-to-sign-up-for-a-college-health-insurance-plan-1828065113>

当学生们准备前往全国各地的大学校园时，他们必须做出一个与 FAFSA 或教科书无关的财务决定:如何处理他们的健康保险。



如果这是一种选择，大多数学生将能够继续他们父母的计划(根据患者保护与平价医疗法案，你可以一直呆到 26 岁)，而且这样做可能在经济上有意义。但是有几个原因让他们考虑他们大学的健康保险计划。

[《消费者报告》](https://www.consumerreports.org/health-insurance/should-you-sign-up-for-college-health-insurance-plan/) 指出，如果你要去州外的学校，你可能没有太多当地的、网络内的医疗选择。虽然你可以在学校放假期间安排体检和其他常规护理，但如果你有医疗紧急情况，你不会希望被网络机构抓住。如果你的父母没有通过雇主购买保险，你和他们疏远了，或者你只是看重你的隐私，那么你可能需要找到自己的计划。

如果这是一个考虑因素，询问你的学校保费，共付额，免赔额和其他现金支出费用是多少。你的保险费“可能与每学期的学费账单捆绑在一起，一年可能高达几千美元，”CR 报告说，所以你会想得到一个详细的细目分类。

根据的说法，你的学校可能会提供财政援助来帮助支付这项保险的费用，所以也要询问一下，你的学生贷款也可能会支付这笔费用(当然，你以后会偿还这笔费用)。

除了费用之外，还有一个缺点:它可能无法覆盖你不在学校的时间，比如夏天，这意味着你需要购买一个短期计划。

另一件需要注意的事情是:《赫芬顿邮报》报道称，一些学校自动将所有学生纳入他们的健康保险计划。《赫芬顿邮报》的安·布雷诺夫写道:“如果学生不需要或不想要该计划，他们必须证明他们的覆盖范围与学校提供的相等，然后申请学校计划的豁免。”“如果他们不在截止日期前申请豁免，他们就等于给自己买了第二份——很可能是不必要的——保单。”豁免因学校而异。

尽管如此，这些计划必须涵盖 ACA 规定的所有基本要素，包括预先存在的状况以及预防性护理、紧急护理和精神健康护理等内容。如果你不能坚持你父母的计划，并且在个人市场上找不到好的替代方案，他们可以提供你需要的保险。
