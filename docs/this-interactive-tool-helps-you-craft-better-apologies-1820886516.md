# 这个互动工具可以帮助你制作更好的道歉

> 原文：<https://lifehacker.com/this-interactive-tool-helps-you-craft-better-apologies-1820886516>

任何人都可以说出一些半吊子的道歉来挽回面子，但真正的道歉需要认真的自省、真诚，并专注于帮助受害者疗伤。这并不容易做到，但这个简单的互动工具可以帮助你。“很高兴你在这里，”当你加载它时，第一个屏幕上写着。很好。

Watch

名为 [的工具，我是问题的一部分吗？](https://elizabethsampat.itch.io/am-i-part-of-the-problem) ，出自游戏设计师兼活动家 [伊丽莎白·桑帕特](https://elizabethsampat.itch.io/) 。这是她使用 Twine 开源讲故事平台 构建的一个循序渐进的自我检查工具 [，它允许那些需要道歉的人更好地理解他们行为的意图，他们如何伤害了他人，以及他们可以做些什么来以有利于受害者的方式进行补偿。它给了你一个诚实而不被评判的机会，让你学会如何更好地道歉，以及如何成长为一个人。](https://lifehacker.com/twine-lets-you-create-choose-your-own-adventure-stori-1682868760)

例如，你会马上被问到你的行为是否有意伤害某人。如果你说是，你会看到一系列原因，比如“我生气了”、“我嫉妒了”或者“我没有安全感”然后，您会看到这样一条消息:

在这一点上，如果你说你担心自己是个坏人，这个工具会让你专注于一个好的道歉中最重要的事情，它会说:

> 事情是这样的:这是否让你成为一个坏人的问题无关紧要。我知道感觉不是那样。但是问这个问题的问题是“这个行为会让我成为一个坏人吗？”让情况和你有关，而不是和你伤害的人有关...我不能告诉你你是不是坏人。我只能告诉你，专注于被你伤害过的人，是做一个好人的重要一环。

当你继续的时候，这个工具会引导你写一个有效的，善意的道歉，并且尝试看看是否可以或者应该为你所做的做出补偿。

在某些情况下，该工具甚至会警告你暂时不要道歉，因为这可能会打断受害者的愈合过程。该工具内置了几种类型的场景，它们都以供违规者考虑的额外资源结束。桑帕特 [向边缘解释](https://www.theverge.com/2017/11/28/16706488/apology-letter-sampat-am-i-part-of-the-problem-twine) :

> “我知道，意识到生活中的某些事情需要改变，却不知道从哪里开始，这真的让人不知所措。我想确保使用这个工具的人至少有一个开始的地方。”

也就是说，桑帕特也承认她不是专家。我是问题的一部分吗？从各种各样的真实心理学和专业资源中汲取，包括大多数恢复计划中的 12 步计划，但这只是一个起点。不过，如果你最近或过去伤害过某人，试试这个工具。它会帮助你为你所做的正确的事情道歉，但它也可能让你看到一个你可以解决的更大的问题。请点击下面的链接查看。
