# 这家伙从零开始制造了一台华丽的相机

> 原文:[https://gizmodo . com/this-guy-built-a-dialogue-camera-from scratch-1826199812](https://gizmodo.com/this-guy-built-a-gorgeous-camera-from-scratch-1826199812)

你有没有发现自己处于这样的境地:你需要某样东西，但却买不起？卢卡斯·兰德斯做了 ，当他需要一个大画幅相机上课时，他找到了一个激光切割机，并用木头做了一个看起来很滑稽的相机。现在，这位路易斯安那人刚刚从零开始完成了另一台相机的制作。这是一种美。

所谓的兰德斯 AL6 马克 2 号是一个全铝的中画幅相机，看起来几乎像是一个老式的 Mamiya 原型[。四四方方的边缘暗示着它是手工制作的，而圆滑的设计暗示着精致。兰德斯说，让他的相机制造技术发展到这一步并不容易。](http://www.collection-appareils.fr/x/html/camera-11284-Mamiya_35%20II.html)

“我不会向新手推荐这样的构建，”兰德斯告诉 Gizmodo。“我花了很多年才走到这一步，只要你开始玩金属，事情就会变得危险。”

为了建造，兰德斯使用了各种铝铸件、机加工和焊接。铸造使用了珠宝商使用的同一种小窑，加工使用的工具能够达到千分之一英寸的公差。兰德斯用钨极惰性气体(TIG)焊机焊接了那部分建筑，他说这是他第一次实际焊接。除了这三个过程所需的设备之外，相机的制作还需要各种测量工具，如数字卡尺、机械直角尺和千分尺。兰德斯还在前面安装了一个 Agfa 85mm 毫米镜头，因为他的相机当然需要一个镜头。

所以这不是兰德斯一时兴起做的事。新相机实际上是他第二次尝试建造这个模型，他在自己的 YouTube 频道 上记录了整个过程 [。正如你从这些样本图像中看到的，着陆器 AL6 马克 2 号完成了它的拍照工作，而且做得非常好。](https://www.youtube.com/playlist?list=PLPtZgIRImIYS3fL59n5UhQVilczKyiJ3m)
