# 使用这个修改过的谷歌搜索网址来避免剧透

> 原文：<https://lifehacker.com/use-this-modified-google-search-url-to-avoid-spoilers-1825513671>

我们很抱歉，查姆·加滕伯格。The Verge 记者 [最近在尝试对全新*战神*游戏中的一个角色进行无害搜索时，学到了关于谷歌自动完成功能威力的严酷一课](https://www.theverge.com/2018/4/24/17277218/god-of-war-spoilers-google-autocomplete-youtube-algorithm-ps4) 。*威猛！*剧透，就在自动完成字段。他甚至没有达到(剧透)的结果。

他不是唯一一个。如果你没有看过《星球大战:最后的绝地武士》，不要在谷歌搜索中输入“卢克·天行者”。我们建议避免使用任何角色的名字，比如说，新的*权力的游戏*季的开始或者 6 月 24 日*西部世界*的结束。

事实上，对于你关心的任何媒体，谷歌的自动完成功能都是潜在的剧透雷区，即使你只是在寻找更多关于你最喜欢的节目或游戏的信息，并尽最大努力保持不被剧透。让我们解决这个问题。

### 阻止谷歌破坏你的节目的网址

如果你害怕谷歌毁掉你关心的某件事的重要时刻，或者你还没有亲身经历的某件事，那就不要在 google.com 输入你的搜索结果了。用这个网址代替:[https://www.google.com/webhp?complete=0](https://www.google.com/webhp?complete=0)T3】

这个链接应该会阻止谷歌的自动完成功能出现在它的主搜索页面上。公平的警告:如果你在搜索结果页面上，这并不总是有效的，所以要小心你的后续谷歌搜索。我已经在 Chrome 和 Firefox 的最新版本中测试过了，在运行普通的谷歌搜索时，我没有发现任何意外的剧透弹出来——或者事实上根本没有任何东西。

当然，你也可以直接在 Chrome 的地址栏中输入你的搜索。如果你试图寻找与某个关键时刻相关的东西(或人)一部电视剧、电影、游戏或一本书，你可能仍然会被谷歌的搜索结果宠坏，但至少在你完成搜索之前，自动补全功能不会弹出来破坏你的秘密。
