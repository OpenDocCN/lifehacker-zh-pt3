# 当你分享屏幕时，枪口自动禁用 OS X 通知

> 原文：<https://lifehacker.com/muzzle-automatically-disables-os-x-notifications-when-y-1797206122>

当你共享你的屏幕进行商业或学校演示时，你不希望弹出任何通知，如 sext，你的治疗师约会的日历通知，或关于你当前正在演示的问题客户的 Slack DM。你可以点击“请勿打扰”，但如果你忘记了呢？虽然 Windows 10 有一个内置选项可以在屏幕共享期间关闭通知，但 OS X 没有。免费应用 [枪口](https://muzzleapp.com/) 解决了这个问题。

Watch

当你与 Slack、Google Hangouts、join.me 和其他应用程序进行屏幕共享时，枪口会静音通知，尽管它目前在 Skype 上有问题。它静静地呆在你的菜单栏中，当你不共享屏幕时，它不占用任何内存，它的开发者布莱恩·D·K·琼斯说。

如上所述，该应用的 [下载页面](https://muzzleapp.com/) 显示了一系列你可能想要避免的羞辱通知。如果琼斯的假通知对你来说很有趣，请留意他 [说他正在开发](https://twitter.com/bdkjones/status/887859198200995840) 的恶作剧应用程序，它会将通知发送到你的实际桌面上。

顺便说一下，该应用的微小界面包括一个愚蠢的按钮，标签为“马赫超驰”往前推； [它实际上并不做任何事情](https://muzzleapp.com/help/) 。这看起来像是在反尴尬应用中开的一个愚蠢的玩笑，所以我们在这里警告你。
