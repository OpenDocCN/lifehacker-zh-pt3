# 如果你用鸡油炸鸡会怎么样？

> 原文：<https://thetakeout.com/what-if-you-fried-chicken-in-chicken-fat-1829429190>

我吃过 [乌鸡](https://thetakeout.com/how-to-make-supremely-crunchy-fried-chicken-1829017782) 。我吃过 [鸭油炒鸡](https://thetakeout.com/karaage-is-the-japanese-method-for-making-superior-frie-1798257206) 。我吃过 [叶油炒鸡](https://bigjoneschicago.com/dine/dinner/) 。我尝过用非动物脂肪炸的鸡肉。我从来没有尝过用鸡油炸的鸡肉。

这个想法受到了伊凡拉面烹饪书 的启发，该书介绍了几个使用 schmaltz 作为煎炸油的食谱——鸡肉排和红烧鸡。Ivan Orkin 的基本原理是，只使用中性味道的油，你会在餐桌上留下自由的味道。

我知道鸡油是什么味道。吃够了炸鸡皮才知道好吃。但是作为调味油呢？我不知道它会带来什么味道。

Schmaltz 很难找到来源(这不像你可以从鸡身上切掉多余的脂肪)。幸运的是，芝加哥附近有一家犹太熟食店，名字很巧，叫做 [施马尔茨的熟食店](http://schmaltzdeli.com/) ，我很幸运，它出售塑料桶的施马尔茨。我会去问问你当地的肉店，他们可能会卖给你几片鸡皮，你可以自己把它们弄干净。

带着两杯鸡油，我开始了我的实验。我去了两条鸡腿的骨头(保留了鸡皮，因为加油)，用盐和胡椒调味，然后用万能面粉把鸡块挖出来。在一个铸铁煎锅里，我将菜籽油加热到 340 华氏度。在另一个煎锅中，两杯 schmaltz 以相同的温度加热(鸡油的烟点为 375 度，高于猪油，但低于牛油)。

我煎了大约八分钟，翻转一次。装有鸡油的煎锅泛出美丽的金色光泽。这是我取出碎片时的样子。左边的鸡腿是用菜籽油烹制的，右边的是用鸡油烹制的:

两者看起来几乎一样。两人都将鸡肉烹制成了相似的质地——外表无油、脆脆的，内部却异常湿润。

鸡油炸鸡有这样一个特点:你绝对可以品尝到其中的不同。最重要的是，它用外表涂层增强了鸡肉的美味。就好像有人在鸡皮上按了 **Ctrl-B。**想象一股热气腾腾的鸡汤面；头上有些咸东西和肉汤块。*这个*就是味道传授；尝起来像炸鸡乘以 1.5。

也就是说，我故意用了一种清淡的调料，这样就不会太过了。后来，我用泰式调料腌制了一批鸡肉，用鸡油炸过，很难辨别调料添加了什么。我的意思是，它仍然很美味，但随着味道越来越浓，回报越来越少。

总体结论？好吃。但是因为 schmaltz 很难获得，您最好在另一个应用程序中使用它。

我有个主意。如果你喜欢热培根调味汁，用热鸡肉代替热培根调味汁。配菠菜沙拉？该死。
