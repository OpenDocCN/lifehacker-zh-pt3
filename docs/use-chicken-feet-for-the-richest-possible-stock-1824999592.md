# 用鸡爪做最丰富的汤料

> 原文：<https://lifehacker.com/use-chicken-feet-for-the-richest-possible-stock-1824999592>

好的鸡汤不仅仅是鸡肉味的水。虽然你应该 [目标是让你的股票变得有味道](https://skillet.lifehacker.com/how-to-make-stock-that-isnt-bland-nonsense-1820626531) ，但一只好股票也应该是丰富和令人垂涎的，有一个不会放弃的身体。朋友们，这就是为什么你应该用鸡爪来炒股。



鸡爪在肉类方面几乎没有什么，但它们在皮肤、骨骼和结缔组织方面远远超过了它，也被称为“给你的股票带来你渴望的身体的东西。”当然， [翅膀](https://skillet.lifehacker.com/instant-pot-buffalo-wings-are-fast-easy-and-very-good-1822565911) 还有 [有很多脂肪和胶原蛋白](https://skillet.lifehacker.com/chicken-wings-make-the-richest-broth-imaginable-1787814398) ，但是我宁愿 [闷那些酱](https://skillet.lifehacker.com/sous-vide-buffalo-wings-are-crispy-on-the-outside-and-t-1819914472) 。鸡爪也超级便宜；即使在我的高价杂货店，它们也要 3 美元一磅。

做鸡爪汤和做其他汤一样简单。大多数商店卖的都是预先去皮的 [以下是如何去皮的方法](https://www.newlifeonahomestead.com/how-to-peel-chicken-feet/) 如果你直接从农场订购的话，还有 [不像我们腌制](https://skillet.lifehacker.com/will-it-sous-vide-fall-off-the-bone-chicken-feet-1792466608) 的时候，不需要去掉爪子。然而，你可能会注意到表皮上挂着一些黄色的小块。别担心，轻轻一拉它就掉了。

一旦你去掉了任何小皮馅饼，简单地在你的锅里扔一磅脚(速溶的或普通的)，加入你想要的任何芳香剂，连同你的 [废料袋](https://skillet.lifehacker.com/how-to-make-seafood-stock-out-of-scraps-1823958899) 和 [溅上的醋](https://skillet.lifehacker.com/why-you-should-add-vinegar-to-your-chicken-stock-1818492266) 。为了让胃特别舒服，我会加入一整头大蒜(带皮)、一大块姜和一整根四分之一洋葱。

如果我用高压锅，我会在高压下煮两个小时，然后手动释放，然后过滤掉所有的固体。如果我走炉顶路线，我会把它煮沸，减少到煨，让它挂了至少五个小时。这比我炖大多数汤的时间要长一点，但我想确保尽可能多地提取精华。

把它倒入脂肪分离器，或者直接把它和罐子一起倒掉，等它在冰箱里凝固后把脂肪刮掉。(或者，如果你想要高脂肪的原汤，就把脂肪留在里面。别让我告诉你怎么处理你的脂肪。)用你的高汤作为任何需要烹饪液体的菜肴的基础，或者只是做一些非常好的汤。
