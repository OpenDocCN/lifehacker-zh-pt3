# 倾斜恢复工作区

> 原文：<https://lifehacker.com/the-reclining-recovery-workspace-1782183267>

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-JSUXhuimTnY&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-JSUXhuimTnY&start=0) 

读者艾伦马上就要动手术了(祝福早日康复！)并希望有一个工作空间，在他停止工作和恢复期间可以舒适地使用，但仍能有效地完成一些工作和玩一点游戏。这是他想出来的，包括一个快速的视频游览。

Watch

让我们从艾伦自己的话开始(顺便说一句，很棒的名字)，他在电子邮件中描述了他的设置，还有一张工作区的静态照片:

> 我是布鲁克林的嘻哈舞蹈老师和长期读者。Patrick 很友好地介绍了我去年的一些基础舞蹈教程 。
> 
> 我正在接受手术，这将使我几个月不能动，所以我创建了一个可以躺着锻炼的工作空间(灵感来自 [这个特色工作空间](http://lifehacker.com/the-reclining-workspace-1697771571) )。
> 
> 作为一名舞蹈演员，我必须保持身材，所以我在零重力椅的底部加了一些 SPRI 阻力带，并在附近放了一套脚踝负重！现在我可以在配对活动之间锻炼，保持完全躺着:)

首先就像我们上面说的，早日康复！第二，大名鼎鼎。第三，我们喜欢这个工作空间，如果它不清晰的话。这是另一张照片，自上而下:

艾伦说他住在布鲁克林，所以它必须是小空间友好的，这种设置肯定能实现这一点。它很节省空间，虽然它没有分散成吨的设备，但它很好地利用了可用空间，有一个足够大的漂亮架子来放置他的键盘，桌子上有一个 Xbox One 的控制器，一个鼠标，甚至还有一点额外的空间来放饮料或一碗小吃。提升他的显示器的支架也是一个很好(也很重要)的接触，因为他会被斜倚着。

触手可及的阻力带也是必不可少的，特别是因为他会在那里待很长一段时间。下次我必须从手术中恢复时，你可以打赌我会举这个例子，在不放弃我的战斗岗位的情况下，舒适地完成一些工作和游戏。
