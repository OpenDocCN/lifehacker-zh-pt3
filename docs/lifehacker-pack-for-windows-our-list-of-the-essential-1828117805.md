# 2018 年必备的 Windows 应用

> 原文：<https://lifehacker.com/lifehacker-pack-for-windows-our-list-of-the-essential-1828117805>

市面上有如此多的 Windows 应用程序，以至于为你的台式机或笔记本电脑挑选一个最好的、最必须安装的软件令人望而生畏。我们仔细研究了几页的推荐、无数的论坛帖子和大量的评论，得出了今年的 Lifehacker Pack for Windows，这是一份涵盖四个类别的软件冠军名单:生产力；互联网和通信；音乐、照片和视频；和公用事业。



***life hacker Pack**是我们最喜欢的平台上最重要的应用程序的年度快照。对于我们不断更新的所有最佳应用程序目录，请务必将我们的* [*应用程序目录*](https://lifehacker.com/c/app-directory) *加入书签，我们每周都会在这里介绍 Windows、Mac、Android 和 iOS 的惊人应用程序——还有浏览器扩展。*

## 生产力

### [Wox](http://www.wox.one/) (免费)

当然，Windows 10 自带搜索功能 Cortana 搜索，你可以通过点击键盘上的 Windows 按钮，输入应用程序、文件名或任何你想在系统上找到的东西(或了解更多信息)来访问。

我们认为开源应用程序 **Wox** 是一个很好的选择，因为它是一个快速的应用程序启动器，也非常适合查找分散在系统中的文件和文件夹。更重要的是，只需敲击键盘上的几个键，就可以轻松快速地搜索网页——无论您是想要简单的谷歌搜索，还是更愿意使用自己定制的前缀在不同的搜索引擎中加载查询。Wox 还支持第三方插件，所以你可以做任何事情，从找出你当前的 IP 地址，到搜索 YouTube 和 Twitch 流，到控制你的 Spotify 播放列表。

### [OneNote](https://www.microsoft.com/en-us/p/onenote/9wzdncrfhvjl) (免费)

由于我们之前最喜欢的 **Evernote** ，收紧了 [的免费计划](https://evernote.com/blog/changes-to-evernotes-pricing-plans/)——以及其最便宜的付费计划现在每月收费 8 美元(RIP[Evernote Plus](https://help.evernote.com/hc/en-us/articles/209005867-FAQ-about-Evernote-subscription-plans))——我们认为[**OneNote**](https://www.microsoft.com/en-us/p/onenote/9wzdncrfhvjl)是 Windows 用户最好的跨平台笔记应用。它是完全免费的，如果你想在多个设备上同步你的笔记，你可以抓取用于 [iOS](https://itunes.apple.com/us/app/microsoft-onenote/id410395246?mt=8) 和 [Android](https://play.google.com/store/apps/details?id=com.microsoft.office.onenote&hl=en_US) 的 OneNote 应用程序，并且它已经默认安装在 Windows 10 上。就这么简单。如果你有一个更好的 Windows 平板电脑，你甚至可以用手写笔(或手指)写出或画出你的笔记。

我们建议您使用 Chrome 或 Firefox 的 OneNote Web Clipper 扩展功能 [，这样您也可以保存正在浏览的有趣网页。还可以考虑为](https://www.onenote.com/clipper) [安卓](https://play.google.com/store/apps/details?id=com.microsoft.office.officelens&hl=en_US) 或 [iOS](https://itunes.apple.com/us/app/office-lens/id975925059?mt=8) 安装 Office Lens，这样你就可以将文档、白板和其他你已经“扫描”过的项目的照片直接发送到 OneNote。

### [短语表达](http://www.phraseexpress.com/) (免费)

正如 Lifehacker 校友 Patrick Allan 在 2016 年的 中所写的 [:](https://lifehacker.com/lifehacker-pack-for-windows-our-list-of-the-essential-787533613#_ga=2.72474128.2011135340.1533564000-3846207152.1521480874)

“文本扩展是你能对你的生产力做出的最大改进之一。想想你一天中所做的任何单调乏味的打字工作——地址、预设的电子邮件回复、一些代码或任何其他东西——想象一下只需击几下键就能全部输入。文本扩展就是这么做的，它可以 [*帮你节省打字的时间*](http://lifehacker.com/how-to-use-text-expansion-to-save-yourself-hours-of-typ-5611210#_ga=1.132366734.1667622160.1441735094) *。**短语表达**是[...]我们为刚接触文本扩展的人提供的帮助。*

对于一个新的、简单的、开源的选择，你也可以尝试一下[](https://beeftext.org/)**。其他一些文本扩展粉丝也对 [**Breevy**](http://www.16software.com/breevy/) 信誓旦旦，你也可以用它通过微小的文本快捷方式启动应用程序。也就是说，这款应用会花掉你 35 美元。我们将首先坚持使用**短语 Express** 。**

### **[Todoist](https://todoist.com/) 或 [微软待办事项](https://www.microsoft.com/en-us/p/microsoft-to-do/9nblggh5r558?rtc=1) (free-ish)**

**古老的战斗仍在继续。 [**Todoist**](https://todoist.com/) 是我们任务管理的首选之一，其核心功能完全免费供您使用。(你每年需要支付 36 美元/来购买额外的服务，比如自动提醒、任务备份、定制标签，以及查看给定项目中所有任务的方式。)Todoist 也是 [难以置信的跨平台](https://todoist.com/downloads) :你可以通过 Chrome 或 Firefox、macOS、iOS、Android 以及 watchOS 和 Android Wear——Gmail 和 Outlook——调出你推迟的所有任务。你现在，正式地，没有理由再推迟你的任务了。**

**[**微软待办事项**](https://todo.microsoft.com/en-us)——由开发 [**神童**](https://www.wunderlist.com/) 的团队打造——值得关注。您可以跟踪任务、添加提醒，并对各种任务列表进行颜色编码。你甚至可以得到一个有用的小“智能提醒”服务，“我的一天”，它可以鼓励你完成有用的建议，告诉你你可能想先处理哪些任务。我们现在仍然倾向于 doist，因为它可以在比微软 To-Do 更多的设备和浏览器上运行，但是后者最终可能会超过前者...只是还没有。**

**此外，如果你想看看神童，不要太执着。正如微软在 [一篇 2017 年的博文](https://www.microsoft.com/en-us/microsoft-365/blog/2017/04/19/introducing-microsoft-to-do-now-available-in-preview/) 中提到的，“一旦我们有信心将 wonderlist 的精华融入到微软的待办事项中，我们就会让 wonderlist 退休。”不能保证当 Wunderlist 消失后，你能把你的提醒和设置转移到另一个应用程序(比如微软的待办事项)，但在这个尴尬的过渡期间，你至少可以免费使用[Wunderlist 的所有专业功能](https://6wunderkinder.desk.com/customer/portal/articles/2932109-how-can-i-get-wunderlist-pro-) 。**

### **[Office Online](https://products.office.com/en-us/office-online/documents-spreadsheets-presentations-office-online) 或 [LibreOffice](https://www.libreoffice.org/) (免费)**

**如果你不想向微软支付[**Office 365**](https://products.office.com/en-us/compare-all-microsoft-office-products?tab=1)，总有[**Office Online**](https://products.office.com/en-us/office-online/documents-spreadsheets-presentations-office-online)——一种查看和创建 Word、Excel 和 PowerPoint 文件的免费方式。(和往常一样，如果你想要免费的综合文档、电子表格和演示文稿创建应用，你也应该试试谷歌的办公套件[](https://www.google.com/docs/about/)**。)****

### ****[苏门答腊 PDF](https://www.sumatrapdfreader.org) (免费)****

## ****互联网和通信****

### ****[**谷歌 Chrome**](https://www.google.com/chrome/) **或** [**火狐 Quantum**](https://www.mozilla.org/en-US/firefox/) **(免费)******

****火狐一直退居谷歌 Chrome 之后——如果你只看市场份额，现在依然如此。随着 2017 年 11 月 Firefox Quantum 的发布，Mozilla 在 Firefox 中加入了一些强大的增强功能，这两种浏览器在性能和可用性方面比以往任何时候都更接近。****

****所以，说一个比另一个更好是不对的；这一点真的是个人喜好问题。你喜欢哪个用户界面？哪个浏览器有你最常用(或最熟悉)的扩展？当你同时打开 30 个标签的时候，这感觉起来不像是一个性能猪？****

****正如我们在[life hacker Mac Pack](https://lifehacker.com/lifehacker-pack-for-mac-our-list-of-the-essential-mac-635303836)中提到的，当你试图*一次加载*大量标签时，Chrome 仍然会感觉迟钝，但它比其他浏览器使用更少的 CPU 能力和内存。这足以让天平偏向 Chrome 吗？不完全是。如果你是 Windows power 用户，你可能会在你的系统上同时安装两种浏览器，以防你出于某种原因需要切换到其中一种——或者因为你喜欢在最终决定保留哪一种之前测试两种浏览器几个月。****

****[**Opera**](https://www.opera.com/) 也是 Chrome 和 Firefox 的绝佳替代品。它同样速度很快，带有一个 [内置 VPN](https://www.opera.com/computer/features/free-vpn) ，并将 WhatsApp、Facebook Messenger 和 Telegram 直接集成到一个易于启动的侧边栏中。如果你很健谈，这可能也值得一试。****

### ****或 [弗兰茨](https://meetfranz.com/) (免费)****

****打开 Pidgin 这样的应用程序，通过各种即时通讯网络与朋友聊天的日子已经一去不复返了。如今，你可能只是发短信，发脸书消息，或者使用众多其他聊天服务中的一种，这些服务既有移动应用也有桌面应用——WhatsApp、Slack、Google Hangouts 等等。****

****如果你只是一个脸书迷，试试 grab[**Caprine**](https://github.com/sindresorhus/caprine)——一个简单优雅的独立应用程序，让你无需启动网络浏览器就能收发信息。如果想要更多的对话能力，请查看[**Franz**](https://meetfranz.com/)**，**，它可以让您在 Messenger 上聊天，还可以从一个方便的窗口使用各种其他服务。你可以用它来查看 Slack、微信、Skype、Telegram、HipChat 等等。****

### ****[Skype](https://www.skype.com/en/) (免费)****

****正如我们在[life hacker Mac Pack](https://lifehacker.com/lifehacker-pack-for-mac-our-list-of-the-essential-mac-635303836)中所写的，你可以使用大量的服务与你的朋友和同事进行视频聊天: [**WhatsApp**](https://www.whatsapp.com/) ，[**Google Hangouts**](https://hangouts.google.com/)，[**house party**](https://lifehacker.com/houseparty-makes-it-easy-to-video-chat-with-multiple-fr-1826567203)，甚至[**Facebook Messenger 毫无疑问，你的智能手机也内置了视频聊天服务。**](http://www.facebook.com/)****

****那么，为什么会有[**Skype**](https://www.skype.com/en/)T5】？微软最近更新了该应用程序，使其更加用户友好，并充满了更多有用的功能。作为一个独立的应用程序，Skype 可以完成所有功能:文件共享、一对一和群组视频聊天、文本和即时消息、屏幕共享、翻译和语音消息。(可惜它不能与脸书同步，也不能导入你现有的联系人。)****

### ****[eM 客户端](https://emclient.com/) 或 [Mailbird](https://www.getmailbird.com/) (免费)****

****当然，你可以使用直接内置在 Windows 10 中的微软邮件应用程序，但它的用户界面仍然感觉有点沉重——特别是如果你更习惯于 Mozilla 的 [**【雷鸟**](https://www.thunderbird.net/en-US/) 或(付费)[**Outlook**](https://products.office.com/en-us/outlook/email-and-calendar-software-microsoft-outlook?tab=tabs-1)等应用程序的更裸机设计。电子邮件应用程序 [**eM 客户端**](https://emclient.com/) 看起来简单多了，但在免费服务中加入了大量功能。你可以定制它的外观(包括一个黑色主题)，通过易于访问的侧边栏查看更多关于电子邮件联系人的上下文信息(包括你交换的文件或你分享的即将举行的活动)，安排在特定时间发送电子邮件，以及直接在应用程序中翻译用外语编写的电子邮件。****

****[**Mailbird**](https://www.getmailbird.com/) 就是一个很棒的替代品。虽然它对免费用户来说功能更轻，但它提供了一种简单的方法，将三个电子邮件帐户的信息融合到一个单一的、可爱的应用程序中。它的用户界面比 eM 客户端看起来更容易一些，但没有 Windows 的默认邮件应用程序那么奢侈。如果你为 Mailbird 付费——每年 18 美元或终身订阅 59 美元——你可以将各种其他账户集成到应用程序中(脸书、LinkedIn、Slack、Dropbox、谷歌等)，并利用应用程序更古怪的生产力功能，如 [快速阅读](https://www.getmailbird.com/easter-speed-read-inbox-zero/) 。****

### ****[微软新闻](https://www.microsoft.com/en-us/p/microsoft-news/9wzdncrfhvfw) 或 [Flipboard](https://www.microsoft.com/en-us/p/flipboard/9wzdncrfj32q) (免费)****

****如果你是一个老派的新闻迷，你可能会更喜欢像 [**【次世代阅读器**](http://nextmatters.com/)(6 美元)这样的严肃的 RSS 应用。但是，如果你想以更时尚的方式了解当天的事件，微软的新闻应用程序 是一个很棒的免费应用程序，它会根据你告诉应用程序的感兴趣的主题，为你提供你需要知道的可爱新闻概要。如果你只对你最喜欢的几个渠道说的话感兴趣，你也可以挑选你喜欢阅读的最喜欢的来源。****

****经典的[**Flipboard**](https://www.microsoft.com/en-us/p/flipboard/9wzdncrfj32q)提供了一种更加图形化的方式来查看世界上发生的事情，重点是类似杂志的设计，在每个“页面”上提供更大的图像和更少的故事这是很好的阅读乐趣，特别是如果你随意浏览主题，找到你可能想深入探讨的文章。微软的新闻应用程序更适合快速获取你现在需要知道的一切。****

### ****[不和](https://discordapp.com/) (免费)****

****如果你是一个游戏玩家，而你的系统上没有安装[](https://discordapp.com/)**，那你就是在给自己帮倒忙。如果你想创建一个虚拟的聚会场所，与你的朋友和公会会员聊天，或者加入一个关注你最喜欢的游戏、网站、极客爱好的神秘团体，这就是你要使用的*应用程序...任何事，真的。内置的语音聊天使计划下一次突袭、举办在线活动或与多年来一直在打字的人交谈变得非常容易。*******

## ******音乐、照片和视频******

### ******(免费)******

******在你的 Windows 系统上安装[](http://videolan.org/)****的最好理由是因为它比 Windows Media Player 好一千倍。它并不臃肿，几乎可以播放你扔给它的任何媒体文件，而且它在不太强大的系统上也能出色地制作出尽可能流畅的画面。换句话说，举例来说，如果你试图在一台不太好的笔记本电脑上观看高清视频，VLC 最有可能给你带来最高质量的体验。它不是一个奇迹创造者，但它的硬件解码能力令人生畏。**********

### ******[**Adobe Bridge CC**](https://www.adobe.com/products/bridge.html)**[**digiKam**](https://www.digikam.org/)**和** [**Google 备份和同步**](https://www.google.com/drive/download/backup-and-sync/)********

******当然，你可以通过想出一个巧妙的文件夹方案来手动整理你的照片，并确保你总是保持在你拍摄的每张照片的顶部。不过，这工作量很大。而是考虑免费的 [**土坯桥 CC**](https://www.adobe.com/products/bridge.html) app。它不是一个很棒的照片*编辑器*，但它是一个优秀的照片组织者(就其价格而言)。通过标签、评分、关键词、尺寸、创建日期来组织您的照片...基本上，通过任何你能想到的与照片相关的元数据。******

****如果你想要一个功能更丰富的应用程序(感觉像 Adobe 的 Lightroom 应用程序)，开源应用程序 [**digiKam**](https://www.digikam.org/) 值得探索。我们尤其喜欢从您庞大的离线图库导入和导出照片到数字服务(如 SmugMug(仅限导出)或脸书(两者都有))的便捷性。)，因为你不应该只是把所有的照片收集在硬盘上。****

****说到这里，一定要看看 [**谷歌备份与同步**](https://www.google.com/drive/download/backup-and-sync/) 。这不是一个照片整理应用程序，但它是一个将照片上传到 Google 相册的好方法，这是一个非常有用的免费备份工具，对大多数人的照片来说已经足够好了。(如果你拍摄的是超过 1600 万像素的照片，或者 4K 的视频，你可能需要 [向谷歌支付更多的存储空间](https://support.google.com/photos/answer/6220791?co=GENIE.Platform%3DDesktop&hl=en) 或者去别的地方看看，这样你的存档照片就会保持原来的大小，而不会受到谷歌的压缩。)如果你更愿意创建更容易与朋友分享的在线收藏，而不是建立离线档案，你也可以使用 Google 相册来整理你的图库。****

### ****[Adobe PhotoShop Express](https://www.microsoft.com/en-us/p/adobe-photoshop-express-image-editor-adjustments-filters-effects-borders/9wzdncrfj27n)，[Paint.NET](https://www.getpaint.net/)，或者 [Gimp](https://www.gimp.org/) (免费)****

****如果你只是需要做简单的照片编辑——调整一些颜色；翻转和旋转镜头；以及调整对比度、亮度和白平衡等参数——用 Adobe 的免费[**PhotoShop Express**](https://www.microsoft.com/en-us/p/adobe-photoshop-express-image-editor-adjustments-filters-effects-borders/9wzdncrfj27n)很难出错。可以编辑 RAW 照片(！)，纠正各种问题，并应用多种不同的滤镜，让您的照片更有特色。如果你需要比 Windows 默认照片应用更多的东西，这是一个很好的工具，但一想到要打开像 PhotoShop 这样的工具——完整版——就会让你害怕。****

****想要类似 PhotoShop 的功能而不付费的高级用户应该看看[](https://www.getpaint.net/)**和 [**Gimp**](https://www.gimp.org/) ，这是两个优秀的(免费)应用程序，按其复杂程度排列，比更简单的“照片校正”应用程序更能让你控制你的照片和图形。从我们记事起，这两个就一直是图像编辑领域的主要内容；它们仍然被维护和更新——后者比前者多一点——这应该给你一个它们的持久力和总体有用性的指示。(此外，在你进行付费的大冒险之前，它们是让你熟悉类似 PhotoShop 的环境的好工具。)******

### ****[](http://handbrake.fr/)****(自由)********
