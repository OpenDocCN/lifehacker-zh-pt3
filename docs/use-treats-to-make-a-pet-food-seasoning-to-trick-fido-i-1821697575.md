# 创造一种以款待为基础的宠物食品“调料”,来引诱狗狗进食

> 原文：<https://lifehacker.com/use-treats-to-make-a-pet-food-seasoning-to-trick-fido-i-1821697575>

我被祝福了(还是被诅咒了？)和一只名叫塔克的异常聪明的小猎犬生活在一起。他有大量的食物过敏，所以他不得不吃这种不含小麦或鸡肉产品的特殊狗粮，几年前我们决定给他买鲑鱼口味的。大约六个月前，当商店暂时用完了他的手提包时，悲剧降临到了我们家。天真的我反而抢了一袋同品牌的白鱼食品。鱼就是鱼，对吧？不不不是的。



第一天我给他端上新食物时，他把碗打翻了。第二天，我又做了一次，他把碗翻过来，把食物的餐垫放在上面。第三天，当我还在试图摆脱这个白鱼骗局时，他把他的食物碗推到房间另一边的垃圾桶，然后收集随机的垃圾，放在他碗里的食物上面。就这样，我终于明白了，我专程去了一家城外的商店，这样他的三文鱼就可以退货了。

对于一个喜欢吃我们早上散步时路上丢弃的玉米煎饼、薯片和三明治的动物来说，我的狗有一个特别选择性的味觉，这意味着让他吃任何新东西都有点挑战性。

他当然不是唯一的。动物和人一样，习惯了自己喜欢的口味，也可能不愿意改变。当你真的打算交换食物时(我没有)，你可以试着慢慢地将新的和旧的混合起来，这样他们就不会发现发生了什么，直到为时已晚。

当你不得不做出突然的改变时， [Reddit 用户 ucffool](https://www.reddit.com/r/LifeProTips/comments/7nj5yt/lpt_if_you_have_a_picky_cat_crush_or_blend_their/) 有一个有趣的想法:将零食碾碎，放入旧调料、瓶中，然后用它给新食物“调味”，加入你知道你的宠物已经喜欢的味道。

他们的帖子建议专门为猫使用和制作调味料，但我认为它对几乎任何动物都适用。添加一点他们喜欢的味道可能会产生与混合新旧食物相同的影响，并将使过渡到新的东西(或欺骗他们吃他们在商店里唯一的东西)变得更容易。

如果调料不起作用，总有第一种方法能打动每只狗的心:花生酱。
