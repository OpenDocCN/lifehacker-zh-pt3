# 用几个色调应用程序设置正确的情绪

> 原文：<https://lifehacker.com/set-the-right-mood-with-a-few-hue-apps-1820224917>

飞利浦自己的 Hue iOS 和 Android 应用程序是非常简单的工具，用于配置飞利浦智能灯和配件，安排在特定时间发生的操作，并将传感器与灯和预设场景相结合，这样你就再也不用碰灯开关了。但该应用程序在定制选项方面有所限制，当涉及到表达你丰富多彩的时尚感(或让你的灯光在晚上显示正确的红色色调)时，可能会令人沮丧。你可以抱怨你的智能灯只是有点智能，或者你可以花几块钱买一些色调友好的应用程序，为你的灯、传感器和开关解锁新的功能、控制和更多定制选项。



### ios

**I 连接色调(4.99 美元)**

[iConnect Hue](http://iconnecthue.com/) 是 iOS 上最受欢迎的 Hue 应用之一，理由很充分。飞利浦自己的 Hue 应用程序以牺牲更多功能为代价追求简单，而 iConnect Hue 却走向了另一个方向。它旨在帮助您定制色调照明系统的几乎每个方面(在您习惯该界面之后)。它是一个通用的应用程序，因此支持 iPhone 和 iPad 屏幕尺寸。一些应用内购买可以解锁额外的功能，使控制你的灯更容易和更方便，特别是如果你是 Apple Watch 用户。把它看作是对智能家居功能的投资，而不是一个昂贵的应用程序。

例如，您可以使用 iConnect Hue 来配置您的灯光，以响应您所在位置的日出和日落时间，或者更改夜灯模式，以使用暗红色灯光，而不是飞利浦的标准暗黄色灯光，从而在您需要在午夜去厕所时保持您的睡眠周期。您可以快速创建睡眠计时器，分组选择灯光以同步颜色、亮度或动画设置，甚至为灯光构建和自定义动画效果，并将它们分配到您的物理色调开关，以举办即时派对。

几乎每一个动作都发生在 Hue bridge 本身上，所以如果你没有带手机，你不会失去太多功能(尽管基于位置的动作确实取决于你设备的位置)。

除了定制照明设置之外，您还可以为飞利浦自己的 Hue Tap 和 Hue Dimmer 开关添加额外的功能，根据您在使用中按下(或按住)按钮的时间或方式，为单个按钮分配多个操作。

它不是最直观的应用程序，其界面功能性大于优雅性。在一个房间里管理多个灯泡可能会让人感到困惑，记住点击或长按会对什么图标做出反应需要一些时间来适应。除了界面，功能齐全的 iConnect Hue 应用程序应该是您定制智能灯以适应您的环境和活动的首选应用程序。虽然 4.99 美元的购买价格为你带来了大量的功能，但应用内购买可以解锁额外的功能，如基于位置的操作或通知小工具。尽管如此，它值这个价。它也缺乏 HomeKit 和 Siri 支持，但应用内购买解锁 Apple Watch 支持可以让你使用 Apple Watch 应用发出语音命令。

### 机器人

**Hue Pro(1.99 美元)**

Android 的 [Hue Pro](http://hueproapp.com/) 应用程序使定制灯光组和创建动画预设变得相当容易，并为音乐爱好者和派对主持人添加了一些娱乐友好的功能。售价 1.99 美元，相当便宜，与飞利浦的官方应用相比，它有更多的定制和日程安排选项。除了必要的灯光控制功能和分组选项，允许更多的灯光预设选项，Hue Pro 可以使用您的 Android 设备的麦克风，让您的灯光响应正在播放的任何音乐(您也可以自定义这些效果)。

只需稍加修改，你就可以使用 Hue Pro 来控制你的灯光，即使你不在家——这非常适合在你不在家时让它看起来像是你的地方被占用了。Hue Pro 还支持小部件功能，解决了 Android 设备上飞利浦 Hue 应用程序最令人讨厌的问题之一:糟糕的小部件。

你可以在 iOS 上使用控制中心，但 Android 的智能家居控制设置没有那么强大。Hue Pro 的小工具可以让你一次或逐个房间地控制每个房间。您甚至可以添加预设照明方案，以便在设备的主屏幕上一键轻松改变心情。Hue Pro 还具有基于位置的操作，不过你需要配套的 Hue Pro Tasker 应用程序(和 Wi-Fi 连接，以表明你已经到家或离家)。
