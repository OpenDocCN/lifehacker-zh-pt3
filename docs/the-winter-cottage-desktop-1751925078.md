# 冬季小屋桌面

> 原文：<https://lifehacker.com/the-winter-cottage-desktop-1751925078>

读者 [亚历克](http://kinja.com/stranaha) 使用了一点小魔法，让他的 mac 在坐下来工作时变得更友好，在起身离开时变得更有信息。他是这样安排一切的。



bersicht 是一个很有用的实用程序，我们在前面的 中已经强调过，它是一个轻量级的替代工具，可以替代 [等较重的工具，比如 GeekTool](http://lifehacker.com/build-an-attractive-informative-mac-desktop-with-geekt-5834676) 。它是这个设置的主干，但是如果你想自己复制它，这里有你需要的一切:

*   [美丽的圣诞树壁纸](http://archwall.xyz/beautiful-christmas-tree-by-chapel-bavaria-forest-winter-night-germany-wallpaper-wide/) 作者 [马西玛·比安奇](http://archwall.xyz/author/a_bc1m4n-shimsi/)
*   这个[über sicht](http://tracesof.net/uebersicht/)，一个我们喜爱的 OS X 定制套件，带有小部件(模糊、灵感语录、早晨时钟和天气小部件)
*   [cDock](http://w0lfschild.github.io/app_cDock.html) ，一个 [给你完全定制控制](http://lifehacker.com/cdock-customizes-the-background-and-color-of-your-mac-s-1688253035) 你的 Dock 的工具
*   [保 2](https://www.macbartender.com/) ，到 [重新排列并调整菜单栏图标](http://lifehacker.com/bartender-2-for-mac-adds-keyboard-shortcuts-search-an-1733370847)
*   约塞米蒂的 [彩色图标由 BlackVariant 打包](http://kencrandall.deviantart.com/art/Yosemite-Coloured-Folders-471755385)

除此之外，亚历克正在运行埃尔卡皮坦的黑暗主题。把这些碎片拼在一起，这就是全部了。当然，如果你在设置上有问题，或者正在寻找那些特定的 bersicht 小工具而找不到它们，请前往 Alec 的 kinja 帖子(链接如下)提出你的问题，并了解更多关于他如何设置一切的信息！
