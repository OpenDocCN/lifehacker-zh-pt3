因此，让孩子尽早养成良好的打字习惯是很重要的。今天的一代人可能永远不会知道标志性的打字老师梅维斯·贝肯(事实证明，他是一个的骗子)，但幸运的是，父母可以获得大量的在线打字工具，以帮助他们有抱负的速记员实现全面的按键命令。

以下是我们最喜欢的节目:

### 新手

新手打字员应该从手的位置、按键识别和适当的形式开始，这样有助于找到那些拥有良好的实时视觉演示的程序。

[](http://www.qwertytown.com/)**T5】**

****优势:**许多网站/应用将打字游戏化，但 QwertyTown 的突出之处在于其一对一的比赛、渐进解锁功能和社交奖励。用户设计自己的虚拟形象，并参与单人或联盟式的游戏，这更好地与 [孩子们如何使用技术](https://offspring.lifehacker.com/how-to-set-smartphone-limits-for-your-kids-1798154097) 相结合，同时为改进提供竞争性激励。**

**缺点:由于教师和学生的体验不同，QwertyTown 的软件最适合教室使用(教室中也有内置的用户社区)，这限制了它在家庭使用中的吸引力。他们提供 30 天的免费试用期，但之后，你就要付费了。**

**[**打字俱乐部**](https://www.typingclub.com/)T5】**

****优势:**打字俱乐部因其有益的视频教程(幸运的是可以跳过重复播放)、有趣的动画和可视化效果(例如一个忍者用各种物体投掷一个可恶的雪人)和俏皮的表演仿真物(我可以在阴影中打字*莫比迪克*超过 30 个小时)而获得高分...赶上来，suckas)。它也是免费的，不需要注册。**

**缺点:网站设计感觉有点过时，广告位置杂乱无章(用户可以通过注册高级配置文件来禁用广告)。虽然南瓜忍者很有趣，但在免费游戏中看到更多的变化也是很好的。此外，这是非常琐碎的，但是“打字俱乐部”这个名字不会让你的孩子觉得超级酷(好像学习打字一般会)。**

**[**跳舞毯打字**](http://www.bbc.co.uk/guides/z3c6tfr)**

**优势: BBC 不仅仅制作滑稽的电视节目。他们的跳舞毯打字程序让年轻用户从“家排”开始，通过四个充满活力的三级水平打字。“毕业生”还可以在 [KidzType](http://www.kidztype.com/browse-typing-games.html) 分支机构选择各种各样的打字游戏和测试(忽略那个令人遗憾的打字错误)。整个程序很好地融入了他们更广泛的 [主计算中心](http://www.bbc.co.uk/schools/0/computing/28972462) ，提供了更复杂的数字课程的自然延续。**

**缺点:这些游戏是基于 Flash 的，对于那些在新设备上玩的人来说很扫兴(尽管这可能只是一种干巴巴的英式讽刺)。2017 年为孩子们设计的软件没有内置社交功能，这也让人觉得毫无魅力可言。**

### **更高级**

**在你的孩子掌握了基础知识后，他们会想提高每分钟的单词量，并尽量减少错误。**

**[**rata type**](http://www.ratatype.com/)T5】**

****优点:** Ratatype 专为提高打字精度和速度而设计。用户不能简单地通过关卡，因为即使是微不足道的三个错别字也会导致重赛。如果你的孩子没有因为沮丧而砸键盘，他们肯定会发展出最灵活、最精确的手指。**

**缺点:一个更先进的打字程序应该放弃幼稚的动画和游戏多样性，这是有道理的，但 Ratatype 的课程和测试感觉有点临床，这并没有激发太多的重播。**

**[**关键英雄**](https://www.keyhero.com/)T5】**

****优势:** Key Hero 对于希望记录自己进步的用户来说非常方便，因为他们提供了强大的统计数据来跟踪个人进步，以及社区百分点排名。另一个巨大的优势是:他们在测试中使用了引人注目的文学作品中的复杂段落(例如， *V 代表仇杀，*笑的人)，这是对其他项目段落的一个全新突破，这些段落经常让人感觉是从标准化阅读测试中借鉴来的。**

**缺点: Key Hero 提供的装饰比 Ratatype 更少，文本选择中只有微弱的喜悦脉搏(当你试图快速键入它们时很难品味)，图表倾向于有利的方向。该网站的设计也相当(*叹气*)重文本。**

**[](http://play.typeracer.com/)**

******优点:**最后，一点乐趣。TypeRacer 让全球用户参加比赛，以最快的速度键入一大块文本(所有错误都必须纠正才能完成)。大众甲壳虫在屏幕上实时移动，在照片完成时鼓励超聚焦。有趣的是，文本选择不仅来自文学，还来自歌词和电视/电影引用。****

****缺点:尽管竞争的角度很有趣，静止不动的大众甲壳虫在虚线上缓慢移动并不比看进度条更令人兴奋。再次强调，对于一个中级打字游戏来说，期待一个沉浸式的体验有点过分了，但是多一点主题的活力会更好。****

****这些是我们在网络上的最爱，但也有许多其他的选择，其中一些安装软件，如果你喜欢一个程序，远离互联网的干扰。****
