# 送给已经拥有一切的人的最佳礼物

> 原文：<https://lifehacker.com/the-best-gift-ideas-for-people-who-already-have-everyth-1789706405>

在我们的生活中，我们都有这样一个人，他会自己买所有想要的东西。他们没有要求任何东西，他们说他们不需要任何东西，但是你仍然想给他们一些东西来表达你的爱、尊重或感激。你可以这么做。

### **给他们记忆而不是东西**

他们可能拥有他们内心渴望的所有物质商品，但他们可能已经忘记了他们曾经拥有的一些美好时光 。人们喜欢回忆。翻翻你的照片，找到一张你们俩被遗忘的旧照片，然后为它买一个漂亮的相框或相册。如果你所有的照片都是数码的，[Shutterfly](https://www.shutterfly.com/photo-books)的相册非常适合在一个地方收集一堆回忆。如果你愿意，你可以在上面附上一张卡片，里面有手写的记忆。

如果你能得到一些人的帮助，菲利斯·拉皮德斯告诉[一个记忆罐也是一个很棒的礼物](http://www.realsimple.com/holidays-entertaining/gifts/gift-person-everything) 。让你的每个家庭成员在一张小纸条上写下他们对收件人最喜爱的回忆，然后把它们收集在一个罐子里。当你的礼物接受者想回忆往事或需要提神时，他们可以打开罐子，拿出有趣的故事、爱的时刻和其他讲述他们如何让每个人的生活变得更好的故事。这些可能看起来有点伤感，但它们是肯定会被珍惜的礼物。毕竟，回忆是我们的全部。

### **给他们一次他们感兴趣的经历**

经历总是一份伟大的礼物，而且很有可能会让你的接受者快乐 。关键是要确保这是他们真正想要参与的事情。不是每个人都想去跳伞，所以坚持他们谈论或研究的事情。

寻找能给他们机会提高他们热爱的技能的经历。艺术课、烹饪课、体育训练课和舞蹈课是一个很好的开始。如果他们对学习电子产品感兴趣， [一个 Arduino，Raspberry Pi，或者初学者焊接套件是完美的](https://lifehacker.com/give-the-gift-of-diy-with-these-kits-and-tools-1788410161) 。或者给他们一个放松的机会。给他们买一天水疗，或者给他们一张按摩礼券。

不要担心给他们一个他们已经有过的经历。像食物和饮料一样，另一个做他们喜欢的事情的机会总是很受欢迎。另外，即使这是他们已经有过的经历，改变一个单一的变量也会使它成为一件全新的事情。这次送他们去一个不同的水疗中心，如果他们已经完成了一些初级课程，给他们买一个中级课程，或者给他们报名参加一个专注于特定类型食物的烹饪班。

### **给他们你的时间**

时间是世界上最宝贵的东西，所以 [付出一些你的](https://lifehacker.com/calculating-the-value-of-time-how-much-is-your-time-re-1749954358) 就是一种终极礼物。付出你的时间可以包括任何事情，从计划与不常见面的家人或朋友外出，到帮助你关心的人。

做所有的研究，计划，并支付一天的乐趣。主动帮助你的父母或祖父母做一些他们还没来得及做的家务。告诉你的朋友你会照看他们，这样他们就可以出去玩了。或者只是提供你的全部注意力，帮他们一整天。告诉他们你将成为他们的助手，帮助他们解决任何悬在他们头上的事情。

当然，你应该总是愿意帮助朋友和家人，所以要确保这些帮助让人觉得是很大的举动。明确表示你愿意为他们做更多的事情，或者帮助他们解决自己一直拖延的事情。主动提出倒垃圾不是礼物，但是花周末的时间帮助朋友整理他们的房子[然后为他们经营车库拍卖](http://lifehacker.com/how-to-throw-a-killer-yard-sale-5894794) 可能是。

### **给他们一个最喜欢的消耗品*和*另一个礼物**

当某人已经拥有一切的时候， [看向消耗品](https://lifehacker.com/flexible-and-frugal-gift-ideas-for-people-who-are-diffi-1789561553) 。没有吃太多喜欢的食物或饮料这样的事情。如果他们喜欢威士忌，给他们一杯上好的威士忌。如果他们喜欢奶酪，给他们一轮美食。如果他们对多力多滋和激浪有着永恒的爱，那就给他们牵线搭桥。没有人会因为免费得到他们最终会购买的东西而失望。

如果你想给某人买礼物，但你不确定他们会有多喜欢，那就用你知道他们会喜欢的消费品来增强它。当他们阅读你为他们挑选的书时，给他们喝他们最喜欢的茶。包括他们最喜欢的糖果和一些剧院风格的爆米花，还有你不确定他们会不会喜欢的电影。不管怎样，他们打开你的礼物时都会很开心。他们得到他们想要的东西，他们看到你努力为他们挑选一些东西。

### **为他们提供会员资格或服务，增强他们所拥有的东西**

有时候送一份好礼物意味着跳出盒子思考——真的。停止考虑你能给他们什么，开始考虑你能给他们什么，这将增强他们已经拥有的东西。对大多数人来说，你知道他们会使用的 [高级服务或会员资格](https://lifehacker.com/give-the-gift-of-premium-apps-and-web-services-1788463516) ，或者他们喜欢的服务的付费等级，是一份坚实的礼物，不会把他们的家弄得乱七八糟。

他们有很好的立体声系统？给他们一个 [的音乐流媒体订阅，比如 Spotify Premium](http://lifehacker.com/11-more-awesome-spotify-tips-and-tricks-youre-probably-1750897507) 或 [Pandora Plus](http://lifehacker.com/pandora-plus-the-ad-free-unlimited-skip-service-is-n-1787713933) 。他们的家庭影院是他们的骄傲和快乐吗？为他们订购网飞、Hulu 或 [不太知名的服务，如 FilmStruck](http://lifehacker.com/filmstruck-the-classic-film-streaming-service-is-now-1788460958#_ga=1.32024126.1667622160.1441735094) 。他们有没有一辆他们一直在谈论的可爱的车？给他们买一些洗车和汽车细节证明。或者，如果他们一直在哀叹自己的战车曾经风光过，可以主动帮忙支付一些汽车保养费用。 [即使是 AAA 会员资格](http://lifehacker.com/hidden-perks-of-aaa-you-may-not-know-about-1600997113) 对你关心的人来说也是一份大礼，因为他们一年到头都在路上。

如果你不确定给你的祖父母买什么，想想能让他们生活更轻松的东西。对他们来说，四处走动可能更难，所以像亚马逊 Fresh、Fresh Direct 或 Instacart 这样的杂货店送货服务，以及像蓝色围裙和电镀 这样的 [餐包送货服务可以为他们省去很多麻烦。](http://lifehacker.com/the-best-meal-kit-services-blue-apron-vs-hello-fresh-1682517921)

### **给他们一些个性化的东西**

你很难买到的收件人可能不需要任何东西，但可能有办法升级他们已经拥有的物品。你可以通过简单的雕刻或刺绣将一件有用的物品变成私人珍藏。

你想成为的美食网络明星有最喜欢的厨师刀吗，或者你爱找乐子的兄弟姐妹喜欢用长颈瓶隐藏事情？在上面刻上他们的名字。或者也许你的世界旅行朋友有一件最喜欢的行李，可以 [使用一点个性化](https://lifehacker.com/use-pet-tag-engraving-for-cheap-durable-luggage-or-key-1733511275) 。工具、手表和珠宝也是很好的雕刻选择。你还可以得到衣服、钱包，甚至是绣有他们名字或他们最喜欢的名言的杂志。
