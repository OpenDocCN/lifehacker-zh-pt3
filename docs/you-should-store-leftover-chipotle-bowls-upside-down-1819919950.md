# 你应该把剩下的墨西哥薄饼碗倒着存放

> 原文：<https://lifehacker.com/you-should-store-leftover-chipotle-bowls-upside-down-1819919950>

我不知道你们中有多少人发现自己有吃剩的 Chipotle 碗，但如果你有，你可能应该按照 Reddit 用户 williskh4n 的建议，将所有这些碗倒着存放。

Watch

正如他在帖子中解释的那样，铝面朝下储存这些碗可以防止纸板变湿，这很好，因为潮湿的纸板很糟糕。
