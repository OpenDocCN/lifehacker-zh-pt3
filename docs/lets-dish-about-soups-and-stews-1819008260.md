# 让我们来尝尝汤和炖菜吧

> 原文:[https://life hacker . com/let-dish-about-soups-and-stews-1819008260](https://lifehacker.com/lets-dish-about-soups-and-stews-1819008260)

大家好，欢迎回到 [*有什么好吃的？*](http://lifehacker.com/tag/whats-cooking) ，这是一个开放的话题，你可以在这里分享你对所有食物相关的想法、建议、食谱和观点。现在已经是秋天了，这意味着我的思想已经转向了温暖舒适的菜肴，特别是汤。

一如既往，我有一些话题让你开始:

*   让我们从股票开始:你是如何制造你的股票的？你的库存包里一般都有什么？总是有鸡的尸体参与其中还是没有肉的事情？不出所料，我的袋子里装满了各种各样的下脚料，但我最喜欢的袋子里的宝贝是大蒜皮。
*   你最喜欢的奶油汤是什么？我想不出有什么比一碗玉米浓汤更美味、更令人舒服的了。
*   你觉得在汤里使用奶酪怎么样？很明显，我喜欢用一碗法式洋葱烤奶酪，但我妈妈过去常常用天鹅绒做奶酪土豆汤。我已经好几年没用过它了，但我打赌它能撑得住。
*   在这场蛤蜊浓汤大辩论中，你的立场是什么？我是新英格兰队，但大家不都是吗？
*   哪种面汤统治你的碗？我想不出有哪一种汤面是我不喜欢的，但我现在正处于 pho 阶段。
*   你最喜欢的慢炖汤食谱是什么？事实上，我不怎么用慢炖锅做这么多汤和炖菜，但我很想听听你们的最爱。
*   你如何确保你的汤味道浓郁，而不是淡而无味？对我来说，这一切都是为了确保我有一个超级美味的肉汤，并确保我单独关注每一种成分。在蔬菜接触肉汤之前给它们上色，可以增加蔬菜的深度。
*   你做过最奇怪、最喜欢的汤或炖菜是什么？我继父以前做炖牛肉时，会在慢炖锅里放入牛肉丁、小胡萝卜、土豆丁和洋葱。然后他会把番茄酱的所有东西都盖上——我是说*盖上*——让它在低温下晾八个小时。这听起来很可怕，但实际上很不错。

像往常一样，你可以随意谈论任何你喜欢的汤类话题。如果它与汤、炖肉、浓汤或杂烩有关，我想听听。
