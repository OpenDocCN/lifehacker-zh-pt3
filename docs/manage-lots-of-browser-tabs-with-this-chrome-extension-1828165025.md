# 用这个 Chrome 扩展管理大量的浏览器标签

> 原文：<https://lifehacker.com/manage-lots-of-browser-tabs-with-this-chrome-extension-1828165025>

如果你在浏览器中完成所有的工作，你最终会在一个窗口中看到几十个标签。你可以为不同的项目打开新的窗口，把标签推来推去，或者像修道士一样训练自己停止打开标签。或者你可以像操作系统一样管理你的浏览器。Chrome 扩展 [Workona](https://workona.com/) 将你的标签组织到命名的窗口中，你可以很容易地在它们之间切换，并保存起来以备后用。这就像是 Chrome 书签和标签排序功能的高级版本。它把你从超负荷的账单中解救出来，而不会因此惩罚你。



### 尊重你的标签

我们推出了很多扩展来帮助你处理标签超载的问题，但其中很多都试图“修复”打开太多标签的习惯。Workona 的第一个前提是**使用很多标签页**很好，甚至很聪明；真正的问题是 Chrome 没有给你一个管理这些标签的界面。与类似的扩展 [Toby](https://lifehacker.com/organize-your-excessive-browser-tabs-with-toby-1826289522) 不同，Workona 将你的标签页更像是活动工作区，而不是存档。

Workona 的第二个前提是**你可以将你的标签分成若干组**。最近我打开了 18 个标签，感觉很笨拙，但也没那么疯狂:我打开了 5 个标签用于购买袜子，3 个标签用于处理保险事宜，10 个标签用于文章研究，还有一些“什锦”有了 Workona，我可以将这些标签分成不同的工作区，一切都变得更加易于管理。

Workona 的主界面是一个固定的标签，可以粘贴到所有的浏览器会话中。它在一个新的界面中显示你打开的标签，以及保存的标签集，这个界面很像 Google Drive。您可以单击不同的工作区，所有相关的选项卡都将在后台加载。(您的锁定选项卡保持锁定状态。)您可以将选项卡从一个工作区拖到另一个工作区。您可以将选项卡保存到您的工作区，这样它就可以兼作书签文件夹。

虽然 Workona 的界面和你的实际浏览器会话之间的关联一开始会让人感到困惑，但一切都很直观，你会在几分钟内学会。您可以像拖动文档标题一样拖动标签标题，将它们分类到工作区中，或者单击+按钮将它们像一组书签一样保存。因为你可以阅读每个标签的完整标题，所以更容易看到你在做什么。(根据创建者奎因·摩根的说法，背景标签被暂时冻结，因此它们不会占用太多内存。)

你可能已经在 Chrome 中使用了多个窗口和书签。但谷歌多年来没有对这些功能进行有意义的更新，它们感觉很笨重。在 Chrome 中拖动标签感觉就像把一个尖叫的孩子从派对上拉走，书签是为长期存储而建的，而不是不断的重新洗牌。Workona 以尊重您的工作流程的方式统一和简化了这两者。

### 习惯工作流程

用书签统一标签感觉怪怪的。Workona 的工作区链接到 Chrome 的窗口；i ts 搜索与您的浏览器历史记录相关。但保存的标签不会链接到 Chrome 的书签。我仍然习惯于将标签页和书签视为同一个过程的两个部分，并了解当我关闭一个工作空间时什么会消失。

移开标签可能会让你忘记它们。如果你是那种只处理立即可见的事情的人，你可能会依靠 tab 超载的压力来完成事情。但到目前为止，我发现当我需要的时候，我会回到工作区。因为免费版的 Workona 只给了我 10 个工作空间，所以我能建立的混乱程度是有限的。(即将推出的专业版包括无限工作空间，每月 6 美元。)

一旦你进入了 Workona，*你就进入了*。作为一个 Chrome 扩展，它位于你所有浏览会话的背后(数据永远不会离开你的电脑)。没有真正的“关闭”模式，除非你禁用扩展。我喜欢这样，因为每当我打开太多标签页时，Workona 就在那里，等着我整理自己。但是当我只打开了几个标签页的时候，那个多余的标签页会让我觉得有点讨厌。我也不喜欢 Workona 占据我的“新标签”页面。

如果你在多台电脑上使用相同的 Chrome 配置文件，那么 Workona 会像任何扩展一样自动在所有电脑上下载。这让你可以在所有电脑上同步你的浏览器会话，如果你经常更换设备，或者如果你有时在家工作，这很有帮助。但如果你喜欢在工作和家庭生活之间保持严格的界限，你可能会在不同的电脑上使用不同的 Chrome 配置文件，所以(像任何扩展一样)Workona 会坚持安装在任何一台电脑上。

如果你想摆脱 Workona，你可以禁用或卸载扩展，关闭你的窗口，你就自由了。它不会到处留下钩子。在卸载了给我的 Gmail 贴满特殊标签的扩展之后，这是一种解脱。

### 共享您的标签

就像谷歌文档让你共享文档一样，Workona 让你与其他 Workona 用户共享标签。您可以为每个用户共享一组存储的标签页，以便他们私下打开，或者您可以共享您实时打开的标签页，以便同步浏览会话。它比彼此之间的信息链接更活跃，但比屏幕共享更少侵入性。这对于和你的伙伴比较购物或者研究一个小组项目是很有用的。

需要注意的是，每个相关人员都需要安装 Workona。这意味着每个人都致力于这个扩展，或者在共享会话结束时禁用它。你不能把你的标签作为电子邮件发送给别人，或者把 Workona 作为一个没有扩展的网络应用程序。所以，只有像你这样的实验型极客，或者你希望长期使用它，才值得让每个人都加入进来。

Boomerang 是 Lifehacker 最喜欢的电子邮件扩展，它增加了所有应该与 Gmail 捆绑在一起的时移功能。Workona 增加了应该与 Chrome 捆绑在一起的标签管理功能。希望 Workona 能像 Boomerang 一样成长和发展。因为我已经等够了 Chrome 自己添加这些功能。
