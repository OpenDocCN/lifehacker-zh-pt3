# 了解苏格兰威士忌的产地可以帮助你知道你是否会喜欢它

> 原文：<https://lifehacker.com/knowing-where-a-scotch-was-made-can-help-you-know-if-yo-1796962809>

多年来，我一直认为自己讨厌苏格兰威士忌。后来有一天，有人劝我一起品尝大量的单麦芽威士忌，我意识到我并不讨厌苏格兰威士忌，我只是不喜欢苏格兰某个特定地区的苏格兰威士忌(因此混合酒大量使用该地区的苏格兰威士忌)。

苏格兰威士忌，或者只是苏格兰的“威士忌”(是的，没有 e)，在全国各地都有生产，但是它的产地对它的味道有很大的影响。想象一下法国葡萄酒和纳帕葡萄酒的区别，除了你看到的是同一个国家。

酿酒厂获取原料的地方、使用的水以及威士忌陈酿过程中的天气都会对最终产品的味道产生影响，并决定你是否会喜欢它。

关于这个话题，你可能会变得超级书呆子气，但由于这不是一个饮料博客，我将在一个超级基础的层面上分解几个不同的地区(阅读:威士忌鉴赏家，这篇文章可能不适合你)。取决于你问的是谁，群岛和坎贝尔镇也应该包括在这个列表中，但因为它们特别小(而且很难找到这些地区的单麦芽威士忌)，为了清楚起见，我省略了它们。

如果你刚刚开始接触苏格兰威士忌，并且知道一个你已经喜欢的品牌，你可能会想尝试来自同一地区的另一种威士忌。同样，如果你和我一样，认为自己不喜欢苏格兰威士忌，看看其他地区——你可能会惊讶地发现自己有多喜欢它们。

### **高地**

这个地区的威士忌以浓郁而闻名，带有烟味和泥炭味。当你品尝这些时，它们往往有一个甜蜜的开始，然后结束干燥。该地区受欢迎的酿酒厂包括阿伯菲尔德、迪恩斯顿、埃德拉杜尔、皇家布拉克拉和格兰莫兰治。高地威士忌往往是我的最爱。

### 斯佩塞

斯佩塞德是苏格兰威士忌产区之王，全国大多数酿酒厂都设在这里。虽然从技术上来说，斯佩塞德是苏格兰高地的一部分，但由于这里酿造的威士忌的独特味道，斯佩塞德赢得了一个属于自己的地区，这种威士忌略带甜味，带有水果味，偶尔还有一丝烟熏和泥炭味。该地区因流经该地区的斯贝河而得名。这里受欢迎的酿酒厂包括格兰威特、麦卡伦、格兰菲迪、格兰法拉斯和奥尔特莫尔。

### **艾莱**T2】

如果你喜欢吸烟，那么艾莱岛就是你的理想之地。该地区以其烟熏威士忌而闻名，这被认为部分是由于该地区的大风。来自这个地区的威士忌是干的，有泥煤味，这种味道来自他们在制作威士忌的过程中用来麦芽的泥煤。这个地区受欢迎的酿酒厂包括 Bowmore、Lagavulin、Ardbeg 和 Port Ellen。

### 低地

这个地区只有少数几家酿酒厂在运营，威士忌往往是酒体较轻，后味较干。如果你刚刚开始喝苏格兰威士忌，这可能是一个很好的开始。这个地区生产的很多威士忌都是混合的，但是受欢迎的单一麦芽威士忌包括格兰金奇、利特米尔、罗斯班克和圣玛格达琳。
