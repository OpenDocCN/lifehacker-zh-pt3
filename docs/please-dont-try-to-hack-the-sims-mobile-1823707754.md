# 请不要试图黑模拟人生手机

> 原文：<https://lifehacker.com/please-dont-try-to-hack-the-sims-mobile-1823707754>

艺电的试运行正式结束了；现在可以下载模拟人生手机版 [安卓](https://itunes.apple.com/br/app/the-sims-mobile/id1144258115?l=en&mt=8) 和 [iOS](https://itunes.apple.com/br/app/the-sims-mobile/id1144258115?l=en&mt=8) 。当你开始第一个小时，用漂亮的服装和家居饰品欺骗你的虚拟伴侣时，我恳求你:不要黑游戏。



更确切地说，不要去为 Android 搜索游戏的修改版 APK。(对不起，苹果粉丝；除非你有一个越狱设备，否则你无法在 iOS 上做太多的“黑客”操作，即使那样，这也是为你的 sim 卡采取的一个相当极端的步骤。)

### 闭嘴，我需要西蒙里昂

我并不是建议你避开黑客，因为我特别喜欢或免费的手机游戏。前者喜欢后者，后者 [可能相当阴险](https://kotaku.com/meet-the-19-year-old-who-spent-over-10-000-on-microtra-1820854953) ，尤其是对于那些直到他们对自己的银行账户做了不幸的事情才意识到自己的游戏瘾有多深的人来说。

我曾经把《辛普森一家:破产了》翻了个底朝天。事实上，这是我保留一台几乎无法使用的旧 HTC One 的唯一原因。我会用改装过的 APK 给自己无限的甜甜圈，进入游戏中的每一座建筑，给我的城镇装满东西，将游戏与艺电的服务器同步，然后在我更新的 iPad 上玩这个游戏。

尽管我采取了一些预防措施，让自己对使用经过修改的 apk 感到更安全——包括在出厂重置的 HTC One 上运行游戏，不包含任何识别信息，并且不将我的真实电子邮件地址(或我使用的任何密码)与游戏相关联——但这仍然是一项有风险的业务。

就像当你下载盗版税务软件时，你完全不知道谁对你正在使用的软件做了什么。当然，你可能会找到一个聪明的方法来获得无限的 Simoleons，但你在你的主设备上这样做是愚蠢的。黑客还可以将 [键盘记录器或恶意软件](https://www.androidauthority.com/pirated-android-apps-170550/) 扔进游戏的 APK，即使你使用像 [VirusTotal](https://www.virustotal.com) 或 [NVISO ApkScan](https://apkscan.nviso.be/) 这样的网站来检查 APK，你也永远不知道他们是否找到了*所有的东西*。

甚至你下载这些修改过的 apk 的网站也会让人觉得粗略。对于被黑客攻击的 APK 的最新更新——因为像艺电这样的公司会经常更新他们的游戏和破解黑客——你通常必须找到并注册一些“修改的 gamez”论坛才能访问最新的修改过的 apk。如果你真的不知道自己在做什么，只是“幸运地”在谷歌上搜索“模拟人生手机黑客 APK”，你很可能会从一个充满广告的垃圾网站获得虚假下载——或者更糟的是，一个对你的设备弊大于利的黑客 APK。

### 但是我仍然需要西蒙里昂

叹气。如果你真的，*真的*一定要在模拟人生手机里作弊，我有好消息和坏消息。到目前为止，似乎没有任何修改过的 apk 可以在“真实”环境中工作。换句话说，我现在还没有看到任何你可以下载的东西，下载到你的 Android 设备上，并用来无限制地访问游戏内容和/或数字货币。

有人建议使用台式机或笔记本电脑上的 [Android 仿真程序](https://boards.libre.io/threads/thesims-mobile-gameguardian-cheat-using-andriod-emulator-nox.301515/) 加载游戏，然后使用其他第三方应用 的 [直接修改模拟人生手机中的数据。我没有尝试过这种技术；我只是个信使。](https://gameguardian.net/forum/topic/19305-the-sims-mobile-currency-xp-items/)

对于这些种类的 Android 仿真程序 [安装](https://www.reddit.com/r/noxappplayer/comments/70ry6b/nox_safe/) 是否安全也有一些分歧。如果你真的想要在你的非智能手机上玩模拟人生移动版的潜在故障体验——在某种程度上击败了手机游戏的意义——至少在 [沙盒](https://lifehacker.com/how-to-safely-test-software-without-messing-up-your-sys-1680608496) 中运行你的模拟器。(并告诉我进展如何！)

我最终不再玩《辛普森一家》(The Simpsons: Tapped Out)的黑客版本，因为跟上游戏的更新——并找到实际有效的新黑客 APKs 比仅仅按照电子艺界的意图玩游戏更乏味。当然，我甚至没有那样做，因为在花了几个月的时间为自己免费建造了一个神城之后，我已经感到筋疲力尽了。事后看来，如果我没有那样做，我想我可能会玩得更开心。
