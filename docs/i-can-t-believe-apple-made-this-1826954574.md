# 真不敢相信这是苹果做的

> 原文：<https://gizmodo.com/i-can-t-believe-apple-made-this-1826954574>

除了 Mojave 中新的黑暗模式用户界面，macOS 下一版本的 [最有用的新增功能之一是 Stacks，这是一种自动将桌面上所有文件分类并排列成整齐小组的功能。](https://gizmodo.com/all-the-new-features-coming-in-macos-10-14-mojave-upda-1826531489) 



这是一个快速简单的方法来清理所有的照片、pdf 文件、随机下载文件，以及其他任何让你的电脑屏幕看起来像垃圾场的东西。但在某些方面，Stacks 让清理你的垃圾有点太容易了，这就是为什么我们如此兴奋地得知，在 Mojave，有一个隐藏的选项可以带回混乱。

 [https://gizmodo.com/embed/inset/iframe?id=twitter-1007624246103715841&autosize=1](https://gizmodo.com/embed/inset/iframe?id=twitter-1007624246103715841&autosize=1) 

它首先是由推特上的 [@ChrisMorrigOG](https://twitter.com/ChrisMorrisOrg/status/1007624246103715841) 发现的，但是我们也能够在内部复制它。首先，您需要关闭堆栈。这在 Finder 菜单栏中。进入**视图**，然后勾选**使用堆栈**选项。接下来右击桌面上的任何地方。你会看到**清理**。现在按下**选项**键，清理会变成**弄乱**。从那里，你所要做的就是点击，然后陶醉在疯狂中，因为所有你以前组织的文件会随机分散在你的屏幕上。

尽管我很难看出“一团糟”选项在日常生活中到底有多大用处，似乎每一个新的小工具或应用程序都如此不顾一切地提高速度和效率，但看到一个完全相反的功能还是令人耳目一新。恭喜你，苹果，你又让科技变得有趣了，即使只是一瞬间。

另外，在 Mojave 于秋季发布之前，你最好不要删除这个功能。
