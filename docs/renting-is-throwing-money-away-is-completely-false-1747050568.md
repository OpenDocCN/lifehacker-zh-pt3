# “租房就是扔钱”完全是假的

> 原文：<https://lifehacker.com/renting-is-throwing-money-away-is-completely-false-1747050568>

你可能听说过一句古老的谚语“租房就是浪费钱。”这似乎是常识。租的时候什么都不买，买的房子你留着留着。然而，事实远非如此。

Watch

由于个人理财博客 affordash everything 以大量令人难以置信的细节进行了分解，买房比“租房子要复杂得多，除非你能保有它。”不仅仅是你将在余生中支付的额外费用(如维修、翻新和财产税)，而且在你抵押贷款的头几年，你几乎没有获得任何资产净值！

> 你应该继续租房吗？租房比买房好吗？或者你应该买一套房子吗？购买是更好的选择吗？你的答案将取决于大量的因素，包括:
> 
> *   当地房价租金比。
> *   你会在那里住多久。
> *   你的另类投资选择。
> *   你对通货膨胀和投资收益的假设。
>     
> *   维护、修理、保险、财产税和资本支出成本。
> *   租金上涨的速度。
> *   等等等等。等。
> 
> 你明白了。我的目标是让你明白——一劳永逸地——这个“租房是在浪费钱”的神话是错误的。事实上，这很危险。它过度简化了一个改变人生的六位数的决定。这可能导致成千上万(或数百万)的人购买了他们后来后悔的房子。

整篇文章都很长，但在决定买还是租之前，应该强制阅读。虽然当你买房子的时候，你是在购买一种资产，但这种资产几乎跟不上通货膨胀的速度，你失去了进行其他投资的机会。更不用说， [租房被低估了](http://lifehacker.com/renting-is-underrated-1732159069) 。即使你认为你之前已经彻底考虑过这个话题，你也可能会发现一些你之前没有想到的信息(比如关于房价租金比的详细解释)。
