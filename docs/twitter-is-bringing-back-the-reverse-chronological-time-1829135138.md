# Twitter 正在带回倒序时间表

> 原文:[https://life hacker . com/Twitter-is-bring-back-the-reverse-chronological-time-1829135138](https://lifehacker.com/twitter-is-bringing-back-the-reverse-chronological-time-1829135138)

要么我们进入了另一个现实，Twitter 关心它的用户，要么网上确实发生了一些好的变化:这个社交网站恢复了按时间倒序查看你的时间表的选项。

Watch

Twitter 支持周一宣布，它正在研究一种方法，让用户在时间线与“最相关”的推文和最新推文之间切换。与此同时，你可以选择关闭“首先显示最好的推文”设置，这样你就可以只看到你关注的人的推文，而不是推荐或“万一你错过了”已经堵塞 TL 的推文。

在后续的推文中，Twitter 表示，它希望“当上述更容易访问的开关可用时，最终取代这一设置。”

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1041838957896450048&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1041838957896450048&autosize=1) 

这发生在一个 [帖子疯传后不久](https://twitter.com/EmmaKinema/status/1041419623386644480) 解释了如何有效地将你的时间线设置成倒序情形。用户@EmmaKinema 发现“静音 suggest_recycled_tweet_inline 和 suggest_activity_tweet”把她的时间线倒过来按时间顺序排列。(推特告诉 [The Verge](https://www.theverge.com/2018/9/17/17872276/twitter-algorithmic-timeline-settings-change-viral-tweet-response) 这些变化已经酝酿了一段时间。)

Twitter 的时间线算法是该网站经常被抱怨的功能，可能仅次于纳粹。对于那些不是活跃用户的读者来说，目前的算法意味着用户可能会看到 12 个小时前的推文与几秒钟前发布的推文混在一起，这是一种令人困惑的并列关系，特别是在突发新闻或事件中，比如超级碗或颁奖典礼，人们在网站上直播推文。用户还会看到他们关注的人喜欢的推文，而不是转发的推文，掩盖了这两种行为之间的不同功能。

如果你不相信 Twitter 会保留倒序功能，除了上面帖子中提到的黑客之外，还有一些方法可以解决这个问题:你可以使用 Tweetdeck 或 Tweetbot，并且你应该能够查看不受算法影响的时间线、推广的推文或“以防你错过”模块。
