# 在较小的非旅游城市度假的理由

> 原文:[https://life hacker . com/the-case for-vacationing-a-small-the-less-non-tourism-city-1787460242](https://lifehacker.com/the-case-for-vacationing-in-a-smaller-non-tourist-city-1787460242)

我最近 [去了趟克里夫兰](http://www.thewildwong.com/the-case-for-cleveland/) ，老实说，这座城市我从来没想过要作为旅游目的地，原因是:它不是。不过，我玩得很开心，这让我意识到，较小的非旅游城市实际上非常适合旅行。

Watch

热门城市挤满了游客是有原因的:通常有很多可看可做的事情。毕竟，克利夫兰不是东京。也就是说，仅仅因为一个城市没有出现在你最喜欢的旅游节目中，并不意味着它不值得一游。这里有几个参观一个城市的理由，尽管这个城市可能并不以旅游业闻名。

### 你可以在预算内享受美好时光

一般非旅游城市比较实惠。旅行需求减少，所以航班价格通常更便宜。此外，非旅游城市通常是较小的城市，那里的生活成本较低。例如，我来自一个大城市，当我在希尔顿市区豪华的屋顶酒吧 点了一杯酒时，我感到惊喜，我的酒钱是我在洛杉矶通常支付的一半

然而，这不仅仅是食物和饮料。找到机票通常更容易，当然这取决于你住在哪里。你可以使用 [谷歌航班](https://www.google.com/flights/) 或 [Kayak 的探索工具](https://www.kayak.com/explore) 到 [浏览世界各地的廉价航班价格](https://lifehacker.com/the-best-tips-for-finding-cheap-airfare-with-google-fli-1756974585) ，这取决于你想何时旅行。你选择一个月或特定的日期，该工具会告诉你在这段时间飞往各个城市的费用，然后你选择一个符合你预算的地方。

Budget Travel [列出了欧洲的几个中型城市](http://budgettravel.about.com/od/destinationsavings/a/midsizedeurope.htm) 在那里你通常可以找到很好的交易，Travel + Leisure 有 [他们自己的美国列表](http://www.travelandleisure.com/slideshows/best-us-cities-for-affordable-getaways) 重点是，如果你正在寻找一个新的地方探索，并且你没有像艾菲尔铁塔或巨石阵这样的特定景点，非旅游城市是一个预算友好的选择。

### 你可以在有限的时间内看到更多

然而，除了价格标签，还有更多值得爱的东西。首先，在一个游客(和普通人)较少的小城市导航要容易得多。

你不必与拥挤的人群搏斗，也不必排队一个小时才能看到这个城市的名胜。这让你有时间在你的旅程中做更多的事情。更小的城镇意味着更少的人群，这给了你更多的时间和机会去做你旅程中最酷的事情，而不用穿过人山人海和支付太多的门票。

更好的是，甚至不用为行程费心。这是我最喜欢非旅游城市的一点:你不必为你一天中的每一个小时做计划 。在克利夫兰，当我完成一些工作时，我就离开我的酒店房间，然后在我方便的时候漫步到这个城市，并大喊一些事情要做。这在旅游城市要困难得多。你永远不知道一个地方会有多拥挤，或者你是否需要门票或预订，你可能会花一个小时去那里。如果离你今天晚些时候要去的地方太远，那就算了。

不要误会我，我喜欢游览大城市，探索它们所提供的一切，但你必须在有限的时间内规划你的一天，选择一家中心酒店，基本上，计划你的 [整个旅行。在小城市，你有更多的灵活性。当然，拥有一家位于市中心的酒店还是有帮助的(我实际上住在克利夫兰市中心的希尔顿酒店，这让我很容易买到东西)。但关键是，由于人群不存在，需求也不高，你可以在闲暇时做些事情，如果你的](http://lifehacker.com/how-to-plan-an-awesome-last-minute-vacation-on-the-che-5840381) [假期目标是放松](https://lifehacker.com/the-case-for-making-a-goal-for-your-next-vacation-1710251652) 的话，这会更令人愉快。

也更容易了解一个非旅游城市。你可以去一些城市参观几次，但仍然不能完全了解它们。见鬼，我在洛杉矶已经住了六年了，我仍然跟不上它，也没有完全理解它。在一个更小的城市，你可以挤进更多的活动和景点，你不只是匆匆忙忙地参加那些活动，你有时间去探索，这使得欣赏和理解这个城市的文化变得容易得多。

### 你不必反对旅游文化

对我来说，旅行就是探索新的地方，体验新的事物。非旅游景点提供了与热门旅游目的地不同的体验。

首先，你不必担心的旅游陷阱。例如，如果你在离体育馆太近的地方吃午饭，你很可能会为蹩脚的意大利菜付出太多。你必须冒险走出人迹罕至的道路。当然，你可以在 Yelp 上搜索一些地方，但你依赖的是一大堆其他的旅游推荐，所以它仍然是断断续续的。当你在一个非旅游城市的时候，你已经离开了人迹罕至的地方！这并不是说美味的食物会落在你的腿上，但至少你不必与价格过高，高估的旅游食品抗衡。

旅游城市也有很多新奇的事物。很难判断哪些景点被高估了，哪些值得去 ，反正很多都是主观的。在非旅游城市，你不用担心这个。当然，无论你在哪里漫游，你仍然会发现一个定价过高的礼品店，你并不需要，但你也会发现许多当地人只是认为他们城市的吸引力，而不是一些旅游新奇。

一般来说，许多旅行者花费大量时间避免旅游。我们不想看到其他游客看到的被高估的风景；我们想深入挖掘，体验一个城市的心脏，像当地人一样生活。鲜为人知的目的地非常适合这个。你不必担心打击旅游业，因为你一开始就不在一个旅游胜地。当然，有很多理由去参观那些美丽、受欢迎的城市，它们都在每个人的遗愿清单上。然而，非旅游目的地提供了一种完全不同的旅行体验，绝对值得。
