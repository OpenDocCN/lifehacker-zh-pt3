# 关于参议院税收法案的最新变化，你需要知道什么

> 原文：<https://lifehacker.com/what-you-need-to-know-about-the-latest-changes-to-the-s-1820472970>

本周，参议院正忙于修改一项税收法案，该法案将对我们现有的税收法规 进行大规模修改。虽然一切都还没有确定下来(预计将在感恩节后通过参议院)，但在最近一轮的修订中，已经有了一些值得注意的变化。

Watch

也许最值得关注的变化是患者保护与平价医疗法案的个人授权。根据这一规定，大多数美国人如果跳过健康保险就必须支付罚款，新法案旨在废除这一规定。 [国会预算办公室(CBO)，](https://en.wikipedia.org/wiki/Congressional_Budget_Office) 这是一个无党派机构，当这样的法案出现时，它会处理这些数字，估计这一变化将减少联邦赤字，但它也会导致到 2027 年增加 1300 万人没有保险，并增加保险费。下面是 [他们的完整详细估算(PDF)](https://www.cbo.gov/system/files/115th-congress-2017-2018/reports/53300-individualmandate.pdf) 。

除此之外，提议的改变包括:

*   七个税级(10%、12%、22%、24%、32%、35%、38.5%)，个人税率略有下调，将于 2025 年到期。公司税的削减(从 35%到 20%的大幅度削减)将是永久性的。

*   先前提议增加到 1，600 加元的儿童税收抵免将进一步增加到 2，000 加元。但这一变化将在 2025 年再次到期。

*   该计划还将扩大对转手企业的税收优惠，但这些优惠也将于 2025 年到期。

根据纽约时报 报道，随着这些变化，大多数纳税人明年将获得减税。但根据 CBO 的数据，该法案也将在未来十年增加超过 1.4 万亿美元的联邦债务。当然，随着一些减税措施的到期，我们大多数人会看到的减税也会随着时间的推移而减少。

《纽约时报》用几幅图表展示了拟议中的变化会对你产生多大影响，这取决于你的收入。最低收入家庭(低于 25，000 美元)将在 2018 年减少 60 美元。中等收入家庭(48，600 美元至 86，100 美元)将减少 840 美元，最高收入家庭将减少约 178，000 美元。不过，与其他税级不同，高收入者的储蓄会随着时间的推移而增加。同样，虽然没有什么是一成不变的，但你可以在这些图表中看到 [提议的改变会如何影响你。](https://www.nytimes.com/interactive/2017/09/29/us/politics/trump-tax-plan-analysis.html?_r=0)T3】
