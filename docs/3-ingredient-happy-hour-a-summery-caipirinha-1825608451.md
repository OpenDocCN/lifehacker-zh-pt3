# 快乐时光三要素:总结

> 原文：<https://lifehacker.com/3-ingredient-happy-hour-a-summery-caipirinha-1825608451>

周末快乐，欢迎回到 [*三料欢乐时光*](http://lifehacker.com/tag/3-ingredient-happy-hour) ，每周饮品专栏，提供超级简单又美味的酒水。本周我们要制作一种饮料，对我来说，它充分体现了本专栏轻松愉快的精神:caipirinha。

Watch

这款鸡尾酒由 cachaç(甘蔗汁酒)制成，完全由低球制成，是人们可以组装的最简单、最明亮的饮料之一。它很酸，很浓，有点甜，而且它也是巴西的国酒。要做到这一点，你需要:

*   1/2 个酸橙，切成四份
*   1.5-2 茶匙糖，取决于你喜欢你的酒有多甜
*   2 盎司 cachaç

将酸橙加入一个低球，将糖撒在楔形物上。用搅拌棒或木勺把它搅拌均匀，直到糖溶解在酸橙汁中，然后放入冰块，加入 cacha ca，搅拌几次。尝一小口，惊叹这是多么容易，享受阳光。
