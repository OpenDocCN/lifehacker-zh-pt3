# 使用这个 Chrome 扩展与远方的朋友同步网飞流

> 原文：<https://lifehacker.com/sync-netflix-streams-with-far-away-friends-using-this-c-1824048081>

几年前(实际上是现在的 7 岁——哎呀！)我搬到了另一个国家，一个远离我的家人和大多数朋友的时区。穿越这个国家带来了许多挑战，但最大的挑战是朋友的事情。从一个相当大的朋友网络到知道几年前在一个聚会上和我一起出去的一个家伙引起了一些严重的 FOMO，当时我家乡的朋友们出去或者有一个电影之夜。

Watch

在我搬家的早期，我们尝试的一件事是让我通过视频电话参加像电影这样的集体活动。整个想法是让我们所有人在电影上同时按下播放键，然后我们可以在电影播放的时候谈论它，并一起观看。这从未按计划进行。

根据个人经验，独自同步网飞流几乎是不可能的。有人总是早到几秒钟，或者晚到几秒钟，最终，他会变得比实际情况更加麻烦。

幸运的是，现在有一些不同的工具可以让这种神奇的事情发生。

我最喜欢的选项是 [【网飞党】](https://www.netflixparty.com/) (另一个很好的类似选项是 [观众](http://showgoers.tv/) )。这是一个你和你的朋友都可以下载的 Chrome 扩展。

要使用它，你打开你想看的视频(但不要按播放！)然后点击 Chrome 工具栏上的网飞派对按钮。每个“派对”都有自己唯一的 ID，所以你可以把代码传给你的朋友，这样你们就可以同步了。

还有一个内置的群聊功能，这样你就可以实时讨论你在看什么，而不会打扰到小组中希望每个人都闭嘴并集中注意力的人。

显然，缺点是它要求你使用 Chrome 观看，如果你想在电视上和一群人一起观看，这可能会很烦人。如果你在旅行，想和你重要的另一半一起看电影，或者你只是想和远方的朋友重温电影之夜，这是一个不错的方式。
