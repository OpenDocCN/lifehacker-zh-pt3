# 这个应用程序可以让你付费下载流媒体电影

> 原文：<https://lifehacker.com/this-app-lets-you-download-your-streaming-movies-for-a-1818851598>

iOS/Android:有时你有几个小时可以消磨，在网飞的队列中有一堆节目和电影——但你在飞机上，或者你的连接很差，不可能播放流媒体。你应该做什么——独自思考？不含 [PlayOn 云](https://www.playon.tv/cloud) ，付费流媒体录像机。其独特的视频录制方法允许无罪恶感的离线或云视频存储，但它需要你做一些规划，以确保你可以顺利地获得你下载的电影。

Watch

iOS 和 Android 应用程序(也有一个略有不同的版本，用于 Windows 的 PlayOn Desktop)能够录制来自网飞、Hulu、亚马逊视频、雅虎视图、ABC、NBC、CBS、福克斯、HBO NOW、HBO GO、PBS、CW 和 YouTube 的视频。你需要订阅才能登录每项服务，所以不要认为这是盗版内容的新方法。此外，像亚马逊和网飞 [这样的服务已经提供可下载的视频](https://lifehacker.com/netflix-gets-offline-playback-on-windows-1793959923) 供离线观看，尽管网飞将只允许下载精选的内容。PlayOn Cloud 让你下载所有内容。

PlayOn Cloud 声称 [通过充当远程流媒体服务，规避了通常禁止下载流媒体视频的法律问题](https://www.playon.tv/blog/playon-legal) 。当你通过 PlayOn Cloud 下载电影时，你实际上是在录制一个正在虚拟机上的网络浏览器中播放的流。这段录音会在云端存储 30 天，你可以把它下载到你的任何设备上。

虽然这使得用户可以下载电影进行离线观看，但这也意味着下载是实时进行的。准备等片长下载完整的. mp4 文件。规划是这个游戏的名字，尤其是如果你知道你将来会没有一个稳定的互联网连接。

你需要 [购买录音积分](https://www.playon.tv/offers/cloud/) 来下载文件，每个积分可以换一次录音。在 PlayOn Cloud 应用程序中，5 次录音将花费 1.99 美元(每次录音 0.40 美元)，但你可以在该公司的网站上以折扣价购买批量配额(最多 500 个配额，每次录音 0.20 美元)。向用户收取他们已经付费的节目的下载费用很奇怪，但这总比再支付一个月的订阅费要好。
