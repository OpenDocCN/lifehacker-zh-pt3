# 如果你认为你不需要显卡，这款 80 美元的 GPU 会改变你的想法

> 原文：<https://gizmodo.com/if-you-dont-think-you-need-a-graphics-card-this-80-gp-1794460619>

近十年来，常识告诉我们，大多数人不需要在电脑中安装独立显卡。游戏玩家需要它们来制作丰富的游戏，如《古墓丽影的崛起》 hum，*一些专业人士，如视频编辑和 3D 设计师，需要它们来消除渲染大型多媒体文件的痛苦。对于其他人来说，显卡更像是一种奢侈品，而不是必要的计算机组件。但是 AMD 80 美元的镭龙 RX 550，一款比你在上面渲染的全新游戏价格略高的显卡，是如此的物有所值，以至于它可能会让你重新考虑显卡是你应该跳过的奢侈品的想法。* 

Watch

目前可从主要零售商处购买，售价 80 美元的 AMD 镭龙 RX 550 旨在升级旧台式机或作为新的超级廉价 PC 构建的核心功能。尽管 AMD 告诉 Gizmodo，他们计划在今年晚些时候推出一个移动版本，作为笔记本电脑的一个选项。

如果你很年轻，皮肤湿润，没有皱纹，并且能够连续容纳六个热狗，那么你可能不会记得每台电脑，无论多小，都需要显卡的时代。然后在 2010 年，英特尔推出了一种直接集成到 CPU 中的 GPU，它可以完成电脑的大部分工作。图形直接集成到 CPU 中意味着计算机制造商不需要在每台计算机中安装独立的图形卡，因此 RX 550 的前辈，昔日的预算友好型图形卡，基本上已经死亡。今天的市场是由为要求苛刻的应用而设计的功能强大的卡所主导的。

AMD 周一 [宣布了一系列基于其一年前的北极星微架构的新显卡](https://www.forbes.com/sites/antonyleather/2017/04/18/amd-launches-radeon-rx-580-does-it-beat-the-gtx-1060-6gb/#4958497d7581) ，这让一些人感到困惑。你看，在一月份的 CES 上，AMD 宣布了一种新的更快的架构，称为 Vega。一般来说，当新架构到来时，显卡制造商会迅速淘汰旧架构。因此，随着 [Vega 卡预计将于 6 月](http://www.pcworld.com/article/3174806/gaming/amd-radeon-infuses-bethesda-games-with-vulkan-cozies-up-to-a-geforce-now-rival.html) 在 E3 推出，AMD 决定用一些廉价的北极星卡潜在地蚕食这些销售似乎很愚蠢。除了新推出的 RX 580、RX 570 和 RX 560——取代去年 RX 400 系列的卡，AMD 还宣布了 RX 550。这是一张无与伦比的卡——虽然它的性能无法与上述任何 GPU 相媲美，但它在价格上胜过每一款 GPU。

邪恶的廉价 550 为 AMD 开辟了一个空间，这在过去七年里对任何 GPU 制造商来说都不可行。英特尔 Iris 一直非常出色，GPU 制造商从未觉得有必要争夺精打细算的消费者的心。但 AMD 显然已经改变了主意。

让我们完全清楚:RX 550 不会赢得任何比赛，但它立即使游戏比英特尔处理器内置的独立显卡更加漂亮和快速。在我们的测试中，我们将其与集成到 Kaby Lake i7-7700K 中的 Iris Intel HD 630 进行了对比。玩*文明 VI* 的时候跑的快了将近四倍。

当玩 *Overwatch* 时，在图形设置为最密集设置(Ultra)的情况下，RX 550 比集成 GPU 快三倍，尽管游戏在 RX 550 或 HD 630 上实际上都无法在该设置下玩:RX 550 每秒处理 17 帧，这是一种非常痛苦的体验，但 HD 630 甚至更差，为 5fps。在如此糟糕的帧速率下，这有点像你在玩一个定格游戏。

但当你将图形设置降至较低时，4K 的 *Overwatch* 完全可以用 RX 550 播放，它实现了每秒 70 帧的超级流畅。在相同的设置下，HD 630 无法竞争，时钟达到了无法播放的 28 fps。由于帧速率如此之低，游戏变得非常不稳定——很难实现快速、熟练的拍摄。

RX 550 甚至在渲染非游戏项目的 3D 图形时表现更好，尽管利润更小。RX 550 在 Blender 中用 9 分 40 秒渲染了一个大的 3D 文件。HD 630 多花了 5 秒钟。

尽管 RX 550 很好，但与你用 200 美元的 Nvidia 1070 或新的 200 美元的 AMD 镭龙 RX 580 这样的“真正”显卡获得的速度相比，它的性能仍然相形见绌。这些卡可以处理像*质量效应仙女座*和*古墓丽影*的游戏，这些游戏会导致 RX 550 和 HD 630 阻塞。

RX 550 尤其吸引那些从零开始制造廉价电脑的人。在英特尔的集成 GPU 上更胜一筹，搭配 AMD 优秀的 [新锐龙 CPU](http://gizmodo.com/what-you-need-to-know-about-ryzen-amds-shot-at-an-inte-1792867348) 也是再合适不过了。虽然锐龙处理器在速度上在每个价位都胜过英特尔，但它明显缺乏集成显卡。如果你加上 80 550 美元，你就可以在价格上与英特尔匹敌，在性能上全面击败英特尔。随着 AMD 计划在移动领域追赶英特尔，这意味着内置锐龙和 RX 550 的笔记本电脑可能会比类似规格的英特尔设备更快。

如果你是一个很少拥有塔式电脑的人，或者你是一个渴望创造廉价产品的建筑商，那么 RX 550 是一个不错的选择。如果你买了不喜欢它，没什么大不了的，它只比*质量效应仙女座菌株*贵 20 美元。
