# 现在您可以用 Skype 向朋友发送现金

> 原文:[https://life hacker . com/now-you-can-send-friends-cash-using-Skype-1797572254](https://lifehacker.com/now-you-can-send-friends-cash-using-skype-1797572254)

现在，当你为工作人员购买音乐会门票时，你的朋友绝对没有理由(除了没有现金)不立即还钱给你。给朋友寄钱有很多不同的选择。像 Venmo、Square Cash、Apple Pay 和脸书这样的服务可以在几秒钟内完成这项工作，现在另一个受欢迎的服务增加了这项功能:Skype。由于与 PayPal 建立了新的合作关系，Skype 用户现在可以通过其移动应用程序向使用该服务的朋友汇款。

虽然它不太可能取代 Venmo 或 Square 用于您的日常转账，但我认为当您需要向海外朋友汇款时，它会派上用场。

要使用该功能，您需要使用最新版本的 Skype 移动应用程序。付款可以发送给 22 个不同国家的朋友和家人，转账是免费的，只要你使用 PayPal 余额或连接的借记卡。信用交易费用为每笔 0.30 美元+总金额的 3.4%。如果你的朋友在另一个国家，你可以选择用他们的货币。

在应用程序中向右滑动并选择“汇款”即可完成支付您需要一个 PayPal 帐户才能使用该功能，第一次决定试用时，您需要将您的 Skype 和 PayPal 帐户连接在一起。

您的收件人不必使用最新版本的 Skype，但他们需要有一个 PayPal 帐户。如果他们没有，系统会提示他们打开一个帐户来收取您的付款。除美国之外，该功能在加拿大、英国(泽西岛、耿济岛、马恩岛)、奥地利、比利时、塞浦路斯、爱沙尼亚、芬兰、法国、德国、希腊、爱尔兰、意大利、拉脱维亚、卢森堡、马耳他、荷兰、葡萄牙、圣马力诺、斯洛伐克、斯洛文尼亚和西班牙也可用。

有趣的事实:Skype 早在 2007 年就提供了用 PayPal 支付朋友的功能，尽管当时(第一部 iPhone 发布的那一年)它只面向台式机用户。

PayPal 实际上是 Venmo 背后的所有者和权力，这可以说是我个人朋友群中最受欢迎的现金发送方式。在我们受到同行压力登上 Venmo 列车之前，Square Cash 是我首选的汇款方式。我喜欢钱马上就到了我的账户上，通常是在女服务员过来拿我的信用卡付账之前。由于 Venmo(和 PayPal，包括 Skype 集成)的工作方式有点像银行账户，所以你汇的钱会一直存在收款人的账户里，直到他或她把钱取出来。

这取决于你的朋友，退出的选择可能是一个祝福，也可能是一个诅咒。我肯定有几个朋友用他们的 Venmo 账户作为未来外出的储蓄账户。当你谈论偿还一大笔钱时，比如一大群朋友的昂贵的音乐会门票，提款要求可能意味着付款人必须在几天内给每个人一大笔贷款，即使他们在购买后几秒钟就“偿还”了。真烦人。然而，如果你已经通过 PayPal 向海外或其他地方的朋友汇款，那么这可能会使这个过程稍微容易一点。
