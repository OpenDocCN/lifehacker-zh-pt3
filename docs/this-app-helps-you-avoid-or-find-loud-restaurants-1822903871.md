# 使用 iHEARu 应用程序避免嘈杂的餐厅

> 原文：<https://lifehacker.com/this-app-helps-you-avoid-or-find-loud-restaurants-1822903871>

出去吃午餐或晚餐是和朋友叙旧的好方法，当然，除非你偶然选择了镇上最吵的餐馆进行长达一小时的聚会。

Watch

虽然 Yelp 等应用程序可以帮助你判断食物可能是什么样的，或者一个地方是否有免费 Wi-Fi，但噪音水平没有评级。现在，一家初创公司正试图以名为 [iHEARu](https://itunes.apple.com/in/app/ihearu-here/id1154107082?mt=8) 的应用程序的形式提供解决方案。

就像 Yelp 等其他众包应用一样，该应用完全依赖于其他食客报告他们吃饭时的噪音水平。去餐厅吃饭的人可以留下自己的主观反馈，也可以使用智能手机的麦克风测量实际的分贝水平， [*食物+酒*报告](http://www.foodandwine.com/news/restaurants-noise-ihearu-app) 。

在应用程序中，噪音水平以 30 分钟为增量进行分解。像其他事物一样，噪音也是可变的。这个想法是，你可以看到一家餐厅可能在周一或周二还不错，但在每晚晚餐时间开始变得喧闹，或者如果你在晚上 7:30 而不是 7:00 预订，你会好得多。

该应用程序于本周在旧金山推出，计划在未来几个月内推广到其他主要城市，目标是在全球范围内提供。
