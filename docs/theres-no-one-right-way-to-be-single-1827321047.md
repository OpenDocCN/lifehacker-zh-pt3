没有单身的实际指南(尽管粗略的谷歌搜索可能会告诉你)。而且单身生活也没有明确定义的“点”。单身的逻辑很简单:

*   你单身是因为你想单身(你正处于恋爱中，你需要自己的时间，你不在乎认真的关系，你不想被束缚，你想被不同的人束缚，等等。).

*   或者你被迫单身。

SLN，问题的一部分在于，你在寻找做事的最佳方式，而实际上只有做对你最有利的事情。没有最好的单身方式，没有关于你应该单身多久的潜规则，也没有任何关于你单身时必须做什么的普遍法则。各不相同。有些人会过得更好，独自努力；其他人应该撒播他们的野燕麦，四处约会，学习什么对他们有用；有些人知道他们需要什么，只是需要不断寻找。如果你不想安定下来，你可以永远保持单身。

我知道，你在寻找一种方法或某种公式来适应你井井有条的生活，但这不是爱情的工作方式。你谈论关系就像谈论工作一样，回顾数据点，告诉我你学到了什么，表明没有间隔年，描述你的技能，等等。我的意思是，你听起来像是在面试一个“单身男人”的职位，问我工作要求是什么，然后希望粉碎它，这样你就可以快速进入一个有福利和最大 401k 的职业生涯。但这不是职业，不是关于技能，或者你是什么，或者你如何定义自己——而是关于你的感受。寻找你感觉对的，SLN，而不是别人说的对的。

听着，你是生活黑客作家的梦想——你是。你实际上采取行动，做我们写的所有事情，让自己变得更好。这看起来很棒，但也有隐藏的危险。上周， [我指责某人没有采纳好的建议](https://lifehacker.com/how-to-make-time-to-write-when-you-have-no-time-1827141901#_ga=2.222923291.1469168944.1530552594-229306750.1526322168) 。虽然你是一个乐于助人的人，但你也太依赖别人告诉你该采取什么行动。你必须小心培养这样一种信念，即凡事总有正确的方法，生活中的每个问题都有确定的答案。没有，所以不要让它冻结你。你基本上是在让我告诉你你想要什么，而我却一点都不知道伙计！所以，我会把这个问题大炮转过来，让它瞄准正确的目标。我们开始了，点燃导火线，然后...

你想要什么?！

你想保持单身吗？你想和谁在一起吗？也许你已经准备好安定下来，但不是和这个现在表现出兴趣的人？我不知道。什么感觉对？无论是现在还是将来，什么会让你快乐。我很想试着卖给你一些“单身生活指南”——相信我，点击量很高——但我不会。以下是我的建议:

让爱发生。

你处在这样一个位置，你可以简单地探索，看看你的感觉带你去哪里。不要仅仅因为你认为还有一些单身生活的里程碑需要核对(没有)，就把一个很棒的人推开，也不要仅仅因为你认为你已经想好了你的余生，就觉得有义务和一个“看起来很好”的人安定下来。保持开放的心态，倾听你的心声。如果某个美丽、优秀的人走进了你美好的生活，并设法让它变得更好，那就随他们去吧。
