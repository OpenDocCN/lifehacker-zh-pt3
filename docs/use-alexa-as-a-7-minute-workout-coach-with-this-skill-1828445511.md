# 使用 Alexa 作为一个' 7 分钟锻炼'蔻驰与此技能

白天找时间健身有时会有点…复杂。如果你时间紧迫，但仍想锻炼一下，Alexa 可以通过 Alexa 的 7 分钟锻炼技能t .

顾名思义，这项技能会带你完成著名的 中的所有 [练习。](https://well.blogs.nytimes.com/2013/05/09/the-scientific-7-minute-workout/)

锻炼包括像跳跃和深蹲这样的事情。你很可能知道如何做大部分工作，但如果你不知道，你手机上的 Alexa 应用程序会告诉你木板实际上应该是什么样子。如果你有一个 Alexa Show，那么它会在屏幕上显示所有信息。

它不是健身房的替代品，但如果你不得不跳过一天，但仍然想偷偷进行一点锻炼，它可以是一种坚实的方式来适应很多，而不必留出太多时间来做它。
