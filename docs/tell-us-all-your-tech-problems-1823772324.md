# 告诉我们你所有的技术问题

> 原文：<https://lifehacker.com/tell-us-all-your-tech-problems-1823772324>

你好。十年前，我为 *Maximum PC、*工作，是时候全盘托出了。我是医生——我想再次成为你的医生。



该杂志每月的“问医生”专题突出了读者寄来的令人困惑的技术问题。我会像 PCs 的初级保健医生一样自己研究答案。如果一个问题非常具体，我会把它扔给另一个专家，他会考虑有用的建议并提交回复。(是的，医生并不总是只有一个人。)

### 医生转世

我仍然在考虑这个类似“问我们任何问题”的系列应该采取什么样的形式和节奏——我很乐意听取你对什么最有用的建议。我在想，我会每周坚持一个主题，这样我们就可以专注于一个主题，而不是强迫你阅读一大堆可能对你不适用的无关问题。

而且，不，我不会称自己为“博士”或任何类似的怪异。这是一个新的开始，我也不想被 BBC 起诉。

### 你应该做什么

让我们知道你在烦恼什么！我希望强调可能适用于其他人的一般问题，如“ [为什么我的电脑在晚上一直开着](https://kotaku.com/the-endless-quest-to-make-my-windows-pc-stay-asleep-ove-1823232841) ，”而不是“每当我将我那昂贵的扫描仪连接到我五年前的主板上的特定 USB 端口时，我的扫描仪都会让 Windows 输出一个非常具体的错误信息。”

我还想帮助你找到更好的方法来做你试图在你的 PC 或移动设备上做的事情，或者如果你使用昂贵的(或基于订阅的)应用程序来做，宁愿不付费。请告诉我们您正在尝试做什么，或者为什么您当前的解决方案不能满足您的需求，我们将努力提供更好的解决方案。如果我们找不到它，也许一个生活黑客爱好者会来拯救我们。

我仍然在最终确定这个系列的格式和时间表，但是现在，如果一些与技术相关的东西正在侵蚀你，请在下面的评论中提出来。也许我们会先讨论你的话题！
