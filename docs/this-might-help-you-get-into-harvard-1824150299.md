# 这可能会帮助你进入哈佛

> 原文:[https://life hacker . com/this-may-help-you-get-into-Harvard-1824150299](https://lifehacker.com/this-might-help-you-get-into-harvard-1824150299)

现在是大学录取通知书的季节，所以如果你有一个高中的孩子，你会知道全国的学生都在焦急地等待，想知道他们是被梦想大学录取了，还是不得不退而求其次。如果你有一个高三学生(或者*是*一个高三学生)，你可能会提前考虑明年秋天的申请季节——以及所有与申请哪里相关的博弈理论。

Watch

这里有一点小策略可以帮你:*你在哪里申请*当然很重要，但是*什么时候申请*也很重要。《华尔街日报》[](https://www.wsj.com/articles/the-decision-that-hurts-your-chances-of-getting-into-harvard-1522229400)*报道称,“提前决定”的申请人比后一批申请的申请人更有可能被录取。梅利莎·科恩写道，“哈佛去年录取了 14.5%的提前行动申请人和大约 3.3%的常规决定申请人。在耶鲁，这两个比例分别为 17.1%和 5%。许多院校招收的新生中，40%或更多都是早期申请者。*

*现在很明显，还有其他因素在起作用:例如，早期决定申请者包括被招募的运动员。早期决定的申请人往往是真正的明星候选人——显然，他们是那些认为自己有机会进入那所学校的孩子。但是，数字仍然令人震惊:科恩报告说，早期申请人将构成达特茅斯大学一年级的近一半，超过一半的西北大学和范德比尔特大学。*

*这让高中毕业生面临一个决定:努力成为明星，尽早申请你梦想中的学校？尽早申请你的第二选择，在那里你可能会有更好的机会，如果不完美的话，在 12 月前把整个事情解决掉？只有你自己才能知道自己对风险的承受能力，正如理财顾问所说。*

*说到经济，对于那些需要经济援助的申请者来说，谁不需要呢？—暂时搁置并同时在所有地方应用可能是有意义的，这样您就可以比较软件包。毕竟，如果你通过提前决定进入大学，但财政援助计划不是你所需要的，你就会面临另一个困境:拒绝它并希望其他人更好，或者接受它并努力让它发挥作用。*

*无论如何，有些孩子会感到兴奋，有些会感到失望:即使是帕克兰枪击事件幸存者和活动家大卫·霍格(David Hogg)， [也没有被加州大学](https://nypost.com/2018/03/28/david-hogg-having-trouble-getting-into-college-after-high-school/?utm_campaign=SocialFlow&utm_source=NYPTwitter&utm_medium=SocialFlow) 的学校录取——他的平均绩点为 4.2，并且在简历上有一些认真的社区服务。所以最好的策略可能是我从网上某个随机的人那里读到的建议:不要专注于你的“梦想学校”想办法把你正在上的大学变成你梦想中的学校。*
