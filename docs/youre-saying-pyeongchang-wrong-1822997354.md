# 你说“平昌”是错的

> 原文：<https://lifehacker.com/youre-saying-pyeongchang-wrong-1822997354>

如果你一直在观看奥运会，你可能已经听到评论员谈论了很多关于韩国主办城市平昌的事情。你可能已经听到他们把这个词搞砸了。



平昌应该读作“Pyeong-ch- *ah* -ng”，就像医生让你张开嘴说“啊”一样它不是像方那样的“平仄--ng”。亚裔美国记者协会的媒体观察小组分享了这个视频来帮助我们做好这件事。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-OiTGtAZdwq4&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-OiTGtAZdwq4&start=0) 

平昌也不是“P.F. Chang”，如果你正在 [观看美国广播公司在芝加哥的子公司 also 第 7 章](http://www.chicagotribune.com/news/chicagoinc/ct-met-pfchangs-chicago-inc-20180212-story.html) 的新闻片段。走吧。我们可以做得更好。
