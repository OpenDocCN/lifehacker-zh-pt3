# 如果你正在购买一部新的 iPhone，为苹果的退货期限设置一个日历提醒

> 原文:[https://life hacker . com/if-you-buying-a-new-iphone-set-a-calendar-reminder-f-1829019521](https://lifehacker.com/if-youre-buying-a-new-iphone-set-a-calendar-reminder-f-1829019521)

如果你打算购买一部全新的 iPhone XS、iPhone XS Max 或 iPhone XRT3，在你预订成功后，有一件大事是你想做的(如果你能在合理的运输时间内买到的话)。在你的日历上为 iPhone 提货日预约一个...并在后的两个星期*设置一个日历约会。称之为“你真的安全吗？”好的。*

Watch

原因如下。苹果公司为你在网上或零售店购买的许多商品提供 14 天的退货政策。如果你对你的新 iPhone 不满意，你有买家后悔，或者你真的可以使用你为新 iPhone 支付的 749 美元、999 美元、1099 美元或其他任何东西，你可以在购买后两周内将它退还给苹果商店，不问任何问题。(如果你愿意的话，也可以免费邮寄给苹果。)

不，举例来说，你不能把你在威瑞森购买的 iPhone 带进来，然后指望苹果处理退货。不会发生——但你知道，对不对？此外，确保你已经把所有东西都放在苹果漂亮的小 iPhone 盒子里，因为你的 iPhone 必须“重新包装，包括你收到它时包含的所有电线、适配器和文档。”我可能不需要告诉你，你的 iPhone 应该工作，而不是被刮到地狱，而不是你“不小心”让它飞出屋顶的东西。

苹果的政策对那些希望在 Craigslist 或易贝等网站上出售旧手机 的人来说很棒，因为你可以在完全投入新手机之前的 14 天里看看你的旧手机是否有任何吸引力。如果没人买，你又不想买那部 1000 美元的 iPhone，又不能卖掉旧手机来抵消成本，你就有了一点回旋的余地。

如果你碰巧从“四大”运营商之一购买了一部新 iPhone，你可以退货，而不会无意中取消你的整个账户——[【威瑞森】](https://www.verizonwireless.com/one-support/return-policy/)[T-Mobile](https://www.t-mobile.com/templates/popup.aspx?passet=ftr_ftr_returnpolicy)[AT&](https://www.att.com/shop/wireless/returnpolicy.html)和 [Sprint](https://www.sprint.com/en/legal/return-and-exchange-policy/device-refund.html) 都给你从购买 iPhone 之日起 14 天的时间来改变主意。也就是说，承运人可能会向你收取退货补货费，通常在 50 美元左右。呸。
