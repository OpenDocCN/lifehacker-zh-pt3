# 用这个技巧悄悄地关上酒店的门

> 原文：<https://lifehacker.com/use-this-trick-to-close-a-hotel-door-quietly-1828439471>

当我旅行时，我通常喜欢在外面呆到很晚，早上尽可能多睡一会儿。另一方面，我的男朋友喜欢在早上 7 点起床，开始新的一天。当我真正醒来的时候，他已经有时间吃早餐，探索酒店周围的区域，并舒适地在他的笔记本电脑上等待我最终加入生活。



从我们的酒店住宿中，我了解到一个事实:所有酒店的门都很响。

老实说，我最大的问题是通常是住在我们附近的人，他们在白天和晚上的任何时候进出房间时都让门砰然关上，但实际的房间门有时也会有点问题。旅游博主约翰尼·杰特 [最近发布了一个解决](https://www.johnnyjet.com/quietly-shut-hotel-room-door/) 问题的视频，其中涉及到在你关门之前用你的钥匙卡“解锁”门，然后慢慢关门。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-5QKVGZs3VIA&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-5QKVGZs3VIA&start=0)

<figcaption class="sc-1ptbguh-0 hxeMec caption">YouTube: Johnny Jet</figcaption> 

这种做法消除了你在关酒店门时通常会听到的响亮的咔哒声，尽管仍能听到。因此，虽然它不会让你的出口完全隐形，但如果你和一个浅睡者呆在一起，想逃离房间一会儿，它会让你的出口不那么引人注意。
