# 奥巴马医改法案已经被废除了(就目前而言)——这对于你的医疗保健意味着什么

> 原文：<https://lifehacker.com/obamacare-repeal-is-dead-for-now-heres-what-that-mean-1797336416>

昨晚，在一场惊心动魄的戏剧中，三名共和党人投票反对他们党的第三项也是最后一项医疗保健法案，该法案正式死亡，这种戏剧娱乐了那些熬夜在 Twitter 上谈论参议院投票的人。那么现在会发生什么呢？



### 真的结束了吗？

大概不会。通过废除某种平价医疗法案是过去七年的主要竞选承诺，两院的共和党人仍然希望通过一些东西。他们只是不能就什么达成一致。(多年来，他们一直抱怨奥巴马医改让医保覆盖面变得更难，并推高了保费，此后，他们提出了一个又一个法案，旨在降低医保覆盖面，并进一步提高保费。以一种让共和党人高兴的方式实现他们的承诺 [实际上可能是不可能的](http://vitals.lifehacker.com/tell-us-what-would-you-call-the-choice-that-house-repu-1793568664) 。)

即使 ACA 的废除彻底失败，你的健康保险也不一定安全。特朗普政府已经采取了几项措施，让个人保险变得更加难以获得，也更加昂贵。《纽约时报》有一篇 [很好地总结了这些举措以及它们为何重要](https://www.nytimes.com/2017/07/27/opinion/health-care-obamacare.html?_r=0) 。

更重要的是，政府一直威胁要停止成本分摊支付，这是从政府到保险公司保持低免赔额的一种方式。没有这些支付，保险公司可能不得不停止在个人市场提供计划；尽管存在不确定性，但保费已经上涨。凯泽家族基金会有一个 [分析这里的情况](http://www.kff.org/health-reform/issue-brief/the-effects-of-ending-the-affordable-care-acts-cost-sharing-reduction-payments/) 。

突然停止支付会让事情变得更糟；但也有人猜测，最终的“不”票，就像我们刚刚经历的一样， [为一些两党合作的权宜之计](https://www.vox.com/policy-and-politics/2017/6/6/15741986/senate-republicans-obamacare-repeal-vote-show-em-a-body) 打开了机会，以确保保险市场保持稳定。

### 有人暗示接下来会发生什么吗？

我们的总统 [在](https://twitter.com/realDonaldTrump/status/890820505330212864) 推特上说，政府应该“让奥巴马医保内爆，然后再处理。”

众议员马克·梅多斯表示，众议院共和党人已经开始着手另一项法案 。

一个两党众议院小组一直在开会讨论医疗保健，但 [不愿透露他们一直在讨论什么](http://www.politico.com/story/2017/07/28/house-group-obamacare-repeal-meeting-241083) 。

截至目前，米奇·麦康奈尔没有亮出任何底牌。在投票后的一次演讲中，他问道,“(民主党人的)想法是什么？看看他们对未来的建议会很有意思。"
