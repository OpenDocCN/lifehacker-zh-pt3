# 图书馆工作室工作区

> 原文:[https://life hacker . com/the-library-studio-workspace-1778011137](https://lifehacker.com/the-library-studio-workspace-1778011137)

我们喜欢极简的工作空间，但这些家庭办公室的设置看起来和感觉上更像一个书房，而不是其他任何东西，这也非常有吸引力。redditor[formula build](https://www.reddit.com/user/Formulabuild)建造了这个家庭办公室，一边是图书馆，另一边是他所有的乐器。这里还有一些照片。

Watch

Formulabuild 没有说太多关于组成工作空间的设备，但老实说，这不是最重要的一点。光是看着就让人印象深刻。他还指出，这是在清理一些杂物之前的*,但在他为所有这些东西添置了三个新的 27 英寸显示器和一个新的支架，并在他的工作空间中添加了吉他和放大器之后。至于展台，他 [向另一位评论者](https://www.reddit.com/r/battlestations/comments/4k8xvh/battlestation_home_office_library_studio/d3da7ig) 解释说，他认为是 [这张](http://ergotechgroup.com/100-series-stands.html) ，放在他的桌子上很好看。这是另一个镜头:*
