# 黑板墙工作区

> 原文：<https://lifehacker.com/the-chalkboard-wall-workspace-1782501139>

在 Flickr 上， [拉赞](https://www.flickr.com/photos/altdigitalphotography/) 发送到她的工作空间，在那里她处理日常工作，她的激情(摄影)，当然，还有她放松的地方。空间也必须是她的卧室的两倍，所以空间是一个溢价——但看看黑板墙。这里还有一些照片。



首先，我们来听听 Razan 是如何描述她的工作空间的。她说:

> 我的家庭办公室有三重功能:白天工作(IT)、兼职(摄影)和卧室。双屏是我 IT /软件分析师工作的远程工作站。在左边，戴尔多功能一体机是我的摄影编辑电脑。我是一名室内设计和建筑摄影师。我的宜家书桌下有我的佳能照片打印机。我的 DSLR 和一些设备藏在抽屉里。其余的暂时堆在角落里。背景是最近粉刷的黑板墙，我很喜欢。

除了卧室足够大，可以容纳一个漂亮的办公室，这都是对空间的良好利用。宜家 Markus 的 [办公椅](http://www.ikea.com/us/en/catalog/products/00103102/) 也很好，虽然她没有提到，但它很舒服，也不太贵(事实上， [你们很喜欢它](http://lifehacker.com/five-best-office-chairs-30776066) )。)
