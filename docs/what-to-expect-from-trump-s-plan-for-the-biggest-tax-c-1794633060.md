# 从特朗普有史以来“最大规模减税”的计划中可以期待什么

> 原文：<https://lifehacker.com/what-to-expect-from-trump-s-plan-for-the-biggest-tax-c-1794633060>

周三，特朗普总统将提出一项税收计划，其中包括“可能是我们有史以来最大的减税”以下是他想做的，以及对你的意义。

### 特朗普提议的“大规模”减税

首先，他可能会公布他的公司税率削减计划，他已经提议将公司税率从 35%削减到 T1 的 15%到 20%的 T2 T3。但是他也承诺减少个人税收。他希望减少税收等级的数量，简化制度，包括扣减和漏洞，并将最高税收等级从 39.6%降至 33%。

现在，我们的税收制度包括七个等级，而川普的税收计划将把它削减到三个等级:12%、25%和 33%。他还提议将单身纳税人的标准扣除额从 6300 美元提高到 15000 美元，将已婚夫妇共同申报的标准扣除额从 12600 美元提高到 30000 美元。为了抵消这笔费用，他提议 [取消个人免税](https://www.washingtonpost.com/news/wonk/wp/2017/04/14/trump-wants-to-give-the-rich-a-big-tax-cut-heres-what-his-supporters-want/?utm_term=.fc24d892be52) ，包括对家眷的免税，以及对户主的扣除。在竞选期间，他确实提出了一个扣除育儿费用的 [计划](http://www.cbsnews.com/news/donald-trump-unveils-affordable-childcare-proposals/) (美国进步中心 [估计](https://cdn.americanprogress.org/content/uploads/2017/04/24082736/TrumpChildcareMethodology.pdf) 它将为中产阶级家庭平均每年节省 5.55 美元)。

### 你会节省多少钱

税收政策中心处理这些数字，看看 [的变化](http://www.taxpolicycenter.org/publications/analysis-donald-trumps-revised-tax-plan/full) 的变化会给的纳税人节省多少。不出所料，减税让最富有的纳税人受益最大。中等收入家庭将获得平均约 1，010 美元的减税(税后收入的 1.8%)，而收入最高的 1%的人将获得 214，690 美元(税后收入的 13.5%)。

随着免税政策的取消，一些纳税人将会在新的计划 下实际支付更多，即那些年收入在 20，000 美元到 50，000 美元之间并且有孩子的人。你可以 [使用这个计算器](http://fortune.com/2016/10/27/how-much-tax-you-would-pay-under-trump-and-clinton-use-our-calculator/) 来看看根据川普提出的计划，你自己的税收会有什么变化。

川普还提议 [终止替代性最低税](https://lifehacker.com/what-the-alternative-minimum-tax-is-and-why-it-matter-1793283953) ，这种税收旨在抵消漏洞成本，这些漏洞让超级富人能够隐藏他们收入的很大一部分(尽管 AMT [对许多中等收入纳税人来说是复杂的税收，](https://turbotax.intuit.com/tax-tools/tax-tips/IRS-Tax-Return/Alternative-Minimum-Tax--Common-Questions/INF12072.html) 也是如此)。

### 它将如何影响经济

总的来说，他提出了一些相当大的削减，这回避了一个问题:这将如何影响预算？财政部长史蒂夫·姆努钦(Steve Mnuchin)曾表示，“税收改革将随着经济增长而获得回报。”这个想法是我们给纳税人更多的钱，他们用这些钱来刺激我们的经济[。](http://twocents.lifehacker.com/how-inflation-works-and-what-it-means-for-your-wallet-1753924925)

不过，经济学家认为事情没那么简单。税收政策中心主任马克·马祖尔说，这种影响是有限的。我们可能会看到短期的增长，但这些削减更有可能让我们陷入更深的债务，这将阻碍任何形式的增长。

正如税收基金会的经济学家艾伦·科尔在《T2》中所说的，“没有纯粹的减税可以收回成本。”
