# 如果你想变得更有魅力，就要多睡觉

> 原文：<https://lifehacker.com/if-you-want-to-be-more-attractive-get-more-sleep-1795971305>

昨晚我熬夜看《办公室》的老剧集。我现在后悔那个决定，不是因为我认为德怀特不应该用那些雪球打吉姆，而是因为根据皇家开放科学学会 5 月份的一份报告，我看起来疲惫的脸确实让其他人看起来我生病了。还好我讨厌和人说话。



由蒂娜·孙德林、马茨·莱坎德、金莫·索尔乔宁和约翰·阿克塞尔松撰写的研究报告 试图发现人们是否不太愿意与睡眠不足的人交往，以及睡眠不足是否会影响吸引力或可信度等其他品质。如果你曾经失眠过一夜，你就会知道那种试图和某人交谈，同时希望他们不会注意到你内心有多死的感觉。原来你对自己没有任何好处。

> 这项研究表明，睡眠受限会对面部外观产生负面影响，并降低他人与睡眠受限者交往的意愿。它还补充了之前对睡眠缺失后面部外观的研究，表明尽管使用了不同的量表，以及更少的实质性和更自然的睡眠缺失情况，但睡眠、吸引力和健康外观之间的关系仍然成立。

这组研究人员选取了 25 名参与者，在正常的夜间休息后和限制睡眠两天后给他们拍照。然后，122 名参与者根据他们与这 25 个人互动的可能性对这些照片进行评分。此外，睡眠不足的参与者还根据他们的健康状况、吸引力、睡意和可信度进行评级。

当你 [只是困了](http://vitals.lifehacker.com/gwyneth-paltrow-wants-you-to-buy-a-90-cure-for-a-disea-1793535195) 的时候，还有什么比被人看丑更糟糕的呢？当你只是昏昏欲睡时，被视为对每个人整体健康的危害。人们也倾向于在睡眠不足的人周围增加他们的回避行为(即人们不想和你疲惫的背后说话)。

> 结果显示，评分者不太倾向于与睡眠不足的人交往。此外，当睡眠受到限制时，参与者被认为不太有吸引力、不太健康、更困倦。

然而，对你来说有个好消息 [美丽的失眠症患者](http://lifehacker.com/how-sleep-restriction-therapy-can-help-you-deal-with-in-1792324667) 。如果你足够有吸引力，即使你错过了几个小时的睡眠，你看起来仍然很好。

> 评分者更愿意与他们认为有吸引力的人交往，而不是不具吸引力的人，有吸引力的人看起来更健康。有趣的是，结果还显示，在看起来非常困倦的人和看起来非常警觉的人之间，其他人对健康和吸引力的评级在 7 分制中增加了大约 1 . 5 级。

如果你感到睡眠不足，尤其是每天如此，解决这个问题可能比你想象的要简单。

记录你的睡眠和醒来时间，坚持相同的睡眠时间表，收集可量化的数据(通常来自健身追踪器或睡眠追踪应用程序)肯定有助于激发想要的习惯，并做出具体而持久的改变。如果你感觉特别不同步，你可以在你的夜间仪式中添加像褪黑激素 这样的睡眠辅助物。不过，不要以为你的追踪器是 [最终决定你睡得怎么样的](http://lifehacker.com/is-your-sleep-tracker-keeping-you-up-at-night-1762011236) 。睡觉时稍微动一下是正常的，所以“浅”和“深”睡眠标记并不是衡量你休息质量的最准确的方法。
