# Opera Software 的新弗罗茨瓦夫工作区

> 原文：<https://lifehacker.com/opera-softwares-new-wroclaw-workspace-1786456785>

当 Opera Software 想要在波兰弗罗茨瓦夫建立一个办公室时，他们想要一个漂亮的办公空间，同时重申他们对减少电子垃圾和环境友好的承诺。这是一个非常棒的结果，他们非常友好地给我们发了很多照片来分享。



Opera 问 [mode:lina](http://lifehacker.com) 为“世界上最伟大的办公室”，他们得到了一个相当不错的竞争者。他们的办公室还在建筑和设计中使用了 1000 多张旧软盘和其他可持续材料。顶部的照片是办公室的入口，但下面的照片让你更能感受到在他们的空间工作的真实感受。

空间明亮，光线充足，天花板很高，有足够的桌子空间。它开着，但仍然舒适温暖。离开办公桌和小办公室，这些工作空间的特点是漂亮的木材，更多的落地窗，以及开放式的设置，即使长时间的艰苦工作会有点分散注意力，但在你需要让所有人聚在一起交谈或集体解决问题的时候，这是非常好的。

下面你会看到可能是有史以来最绿色的会议室，但结合铜管和墙上的屏幕，它看起来明显是电子的，我们喜欢它。

非常感谢 mode:lina 给我们发来了 Opera 新办公室的照片，并对他们的设计和组装表示感谢。我们希望 Opera 对他们设计的空间感到满意。

如果您有自己的工作空间可以炫耀，请将其添加到我们的 [**Lifehacker 工作空间展示并告诉 Flickr pool**](http://www.flickr.com/groups/lifehacker-workspace-showandtell/pool/) 来与我们分享。确保您包含的任何照片至少为 1280x720。将它们保持在 16:9 也有帮助！包括一些文字，关于你使用的材料，你是如何想出这个设计的，以及任何其他相关的细节。如果你巧妙的组织和良好的设计感吸引了我们的眼球，你可能就是下一个特色工作区！
