# 妨碍司法公正是什么鬼？

> 原文：<https://lifehacker.com/what-the-hell-is-obstruction-of-justice-1822930252>

“妨碍司法公正”这个词最近经常被提起，尤其是当你看新闻的时候。特朗普 [总统被指控妨碍司法公正](http://thehill.com/homenews/politics-101/373216-what-is-obstruction-of-justice) 关于罗伯特·穆勒可能与俄国勾结的调查，现在众议院情报委员会主席德文·努内斯 [正把矛头指向他](https://www.nytimes.com/2018/02/12/opinion/devin-nunes-obstruction-justice.html) 为 [创建自己的另类新闻网站](https://www.politico.com/story/2018/02/11/devin-nunes-alternative-news-site-402097) 。但是妨碍司法公正是什么意思呢？



[根据美国联邦法律](https://www.law.cornell.edu/wex/obstruction_of_justice) ，妨碍司法公正是当某人”...腐败或通过威胁或武力，或通过任何恐吓信或通信，影响、妨碍或阻碍，或试图影响、妨碍或阻碍正当司法."基本上，当有人在官方司法程序中，或在国会或联邦机构的未决程序中，故意干扰或妨碍检察官或其他政府官员时，这是一种联邦犯罪，例如，对可能的串通进行调查。这项重罪指控的处罚可能包括巨额罚款，或长达五年的监禁，或两者兼而有之。

这听起来可能很简单，但最终都要证明被告的意图。根据定义，妨碍司法公正是一种极其广泛的犯罪，自 1789 年制定该法令以来，这一点没有太大变化，这往往使犯罪更难证明，因为被告必须具有“腐败意图”。说到底，要指控某人妨碍司法公正，你必须有确凿的证据证明他们有妨碍正在进行的调查或询问的特定意图。

那么，妨碍司法公正通常是什么样的？正如前联邦检察官巴拉克·科恩 [向《华盛顿邮报》](https://www.washingtonpost.com/video/politics/what-is-obstruction-of-justice/2017/06/07/17ad7a00-4bc3-11e7-987c-42ab5745db2e_video.html?utm_term=.86db6e8d15e6) 解释的那样，妨碍司法公正可以有多种形式。它可能是试图阻止证人在联邦大陪审团作证，某些类型的贿赂，阻止人们提供证词或阻止某些类型的证据出现或在国会调查中使用，或者它可能是简单的告诉某人不要说或做某些事情作为联邦调查的一部分。 [其他例子还有](https://en.wikipedia.org/wiki/Obstruction_of_justice) 在调查中，一个不是嫌疑犯的人向调查人员撒谎；有人篡改、销毁或隐藏证据；偶尔拒绝帮助警察。在更大的范围内，妨碍司法公正的指控被用于试图弹劾总统理查德·尼克松和比尔·克林顿。
