# 翻新老房子时你会遇到的最大问题

> 原文:[https://life hacker . com/the-maximum-problems-you-run-into-when-renewing-an-1772460906](https://lifehacker.com/the-biggest-problems-youll-run-into-when-renovating-an-1772460906)

老式住宅有自己的特色，通常拥有建筑窗户和实心石膏墙等特色，这在许多新建筑中是找不到的。然而，到了翻新的时候，老房子会带来昂贵的挑战，而且也不那么迷人。以下是你应该准备的最大和最常见的装修问题。

Watch

大约十年前，我丈夫和我买了一辆 1920 年的殖民车。我们对这座老房子的每一次装修都让我们害怕再做一次:这次我们会发现什么隐藏的问题呢？这要花我们多少钱？尽管翻新过程中出现的一些问题也适用于新房子，但老房子有特殊的问题。你会听到“那时的东西是不同的”，这并不总是一件好事。如果你意识到了潜在的问题，你就能为你的改造项目设定一个更现实的预算和时间表。

### 过时的管道和电气设备

电线和管道问题在老房子里很常见。20 世纪 60 年代以前，镀锌管既用于室内，也用于下水道。镀锌管的问题是，随着时间的推移，它们很容易堵塞或腐蚀。在涉及管道的翻新过程中，你可能想用聚氯乙烯或铜来代替管道。HouseLogic 表示，翻新期间更换管道的成本可能会增加 250 美元至 1000 美元——相对便宜，因为管道在工程期间会暴露在外。

旧的电线可能是一个安全隐患，需要更新以符合规范。在改造我的浴室时，我得知我需要雇一个电工来升级配电板。20 世纪 20 年代的家庭没有吹风机，即使是这么小的东西也比旧的家用电器系统能消耗更多的电力。老旧的线路也很不方便:如果你曾经在同时运行微波炉和咖啡机时没电，你就会明白我的意思。预计至少要支付[1300 到 3000 美元](http://www.fixr.com/costs/install-electrical-circuit-panel-upgrade) 升级到 200 安培的服务，200 到 750 美元把不接地的 2 脚插座换成 3 脚插座，4000 美元或更多来重新布线，以去掉会引起火灾的老式 [旋钮管式布线](https://en.wikipedia.org/wiki/Knob-and-tube_wiring) 。根据您的需求和您拥有的设备，您甚至可能需要升级现代布线以支持新的设备、计算机、娱乐系统和其他项目。拥有一个家不是很有趣吗？

### 铅和石棉等不安全的材料

如果你的房子是几十年前建造的，很有可能油漆中含有的铅，地板、管道、爆米花状天花板、屋顶和 HVAC 系统中含有石棉。如果不受干扰，这些是无害的，但是如果项目需要刮或切割这些材料，粉末或灰尘会非常危险。你可以自己 [检测含铅涂料](http://workshop.lifehacker.com/how-to-perform-a-diy-test-for-lead-paint-in-your-home-1732514709#_ga=1.153046975.744298413.1458929954) ，但对于其他任何东西，如果你怀疑它们在你的建筑中，你就需要专业人员来检测和去除这些材料。

当我让一家除污公司清除覆盖在我地下室管道上的石棉时，我付了 1000 美元。 [HomeAdvisor](http://www.homeadvisor.com/) 说石棉去除范围 [从 200 美元到 700 美元一小时](http://www.homeadvisor.com/cost/environmental-safety/remove-asbestos/) ，大多数房主花费 [在 971 美元到 2250 美元之间去除有毒铅](http://www.homeadvisor.com/cost/environmental-safety/remove-toxic-lead/) 。为了安心和家人的安全，物有所值。

### 用来建造你家的材料已经不存在了

旧房子的建筑标准和规范与现在不同，通常使用完全不同的材料。浴缸变小了，门变窄了，房间也变小了。如果你想保持你家的老特色，找到与你房子其他部分相匹配的装修材料可能会很棘手。翻新一栋旧的褐砂石建筑？你需要实木镶板门和薄橡木地板——所有这些都与今天的标准和今天房屋建筑中使用的普通材料完全不同。在建筑回收店 或 [购物，其他的再利用中心](http://lifehacker.com/shop-at-reuse-centers-to-save-a-ton-of-money-on-all-you-5990357) 会有所帮助，寻找专门改造老房子的专业人士也是如此。然而，如果你真的想得到匹配的原始材料，这些材料更多的是修复而不是翻新，请准备好支付更多的钱。

### 过时的布局，比如长长的走廊和狭小的房间

观看 HGTV 的 [房屋猎人](http://www.hgtv.com/shows/house-hunters) 的一集，你会很快了解到今天的购房者正在寻找的功能和平面图与你在许多老房子中会发现的明显不同。带步入式衣柜和附属浴室的主卧？厨房、餐厅和客厅之间的开放式平面布局？不是在老房子里。过去，房主有留宿员工为他们打扫卫生和做饭，维多利亚风格的房子为每个人都准备了一个房间。单身。目的。几十年前的普通建筑计划也包括廉价公寓，所以大型建筑被分割成小公寓。当时可能没有人认为会有人愿意生活在“开放式布局”中。

我的一些卧室甚至没有壁橱(所以对房地产经纪人来说，它们在技术上不算是卧室)，由于房子的建造方式，没有空间增加标准的壁橱。对于过时的布局，寻找空间来扩展空间或重新安排布局可能会很棘手。如果你想让你的家现代化(例如，开放空间布局)，你需要咨询建筑师或工程师，以及建筑商，以计算出真正的成本。拆除墙壁可能既昂贵又危险 [如果是承重墙](http://workshop.lifehacker.com/how-to-identify-a-load-bearing-wall-1626923331) ，如果你正在考虑打开地板，这样你就可以从客厅看到厨房，很可能你会遇到结构问题。

### 一次又一次糟糕的翻新工作

房子越老，出售或转手的次数越多，对房子的任何改动就越有可能不符合房子的原始设计。有时，就像我的情况一样，前业主的“改进”完全不顾结构的其余部分。例如，阁楼浴室的水管穿过其他房间，而不是穿过墙壁或其他隐藏的空间，它们应该这样。有时你不会发现这些问题，直到你开始一个改造项目。有一个 [能干的家庭承包商](http://lifehacker.com/how-to-hire-a-contractor-without-getting-hosed-5578353#_ga=1.154635198.744298413.1458929954) 或 [勤杂工尤为重要，你可以信任他](http://lifehacker.com/how-can-i-find-and-hire-a-good-responsible-handyman-558507563) 来撤销你发现的任何旧错误。

总的来说，如果你的房子多年来一直在更新和维护，你可能不会遇到所有这些问题。但是对于更古老的家园，尤其是那些屹立了几十年的古老而迷人的家园，你可能就不知道了。最好谨慎行事，为你的家居装修预算比你认为需要的更多的钱和时间。旧的经验法则是 [留出 15%到 20%的应急](http://www.houselogic.com/home-advice/planning-your-remodel/how-to-budget-for-home-remodel/) 用于意外的问题和变化，但是如果你怀疑你的家可能有上述问题之一，你可能想增加更多的缓冲。
