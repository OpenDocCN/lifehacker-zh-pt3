# 看新闻最好的 App 是什么？

> 原文：<https://lifehacker.com/whats-the-best-app-for-reading-the-news-1829702550>

我们在 Lifehacker 上查看了许多应用程序，最近我们写了相当大的关于 [Android](https://lifehacker.com/the-essential-android-apps-for-2018-1829424567) 和 [iOS](https://lifehacker.com/the-2018-lifehacker-pack-for-ios-the-essential-ios-app-1828788598) 的必备程序的综述。尽管如此，仍然有很多，这可能会使你很难找到最好的应用程序来做你想做的事情。



在本周的 [Tech 911](https://lifehacker.com/c/tech-911) 专栏中，我们将回答生活黑客读者**约翰**提出的一个应用主题问题:

> “我是谷歌新闻的长期用户，因为我喜欢浏览每一篇文章，并停留在我感兴趣的文章上。然而，自从那次 UI 更新把所有东西都变成白色后，我注意到它不再在后台正常同步了。就像我必须在应用程序下载的时候把它放在前台，否则它就不能正常下载所有的帖子。如果我在线时在谷歌新闻上阅读 Lifehacker，它现在似乎会加载不必要的菜单(或其他东西)，使我的手机延迟。你有没有替代建议？”

嗯。我并不真的使用很多新闻聚合应用——嗯，一个也没有——但我只是花了一些时间在谷歌新闻上，我一点也不觉得它很可怕。然而，如果你有问题，那对我来说就足够了。你没有提到你使用的是什么智能手机平台，但如果你使用的是 iOS，你可以随时切换到苹果的默认新闻应用，看看你觉得怎么样。如果你使用的是安卓系统，你可能会被预装在你的设备上的谷歌新闻(或一些蹩脚的第三方新闻应用)所困扰。

至于替代方案，我有三个主要建议，我邀请 Lifehacker 的读者在下面分享他们的一些建议。尝试安装**[**Pocket**](https://getpocket.com/)，它将允许你保存你正在使用的任何手机浏览器中的文章。这样，你就可以在你的浏览器上浏览 Lifehacker(或其他任何东西),并保存文章供以后查看——在一个可能比谷歌新闻更容易阅读的界面上。这不是一个新闻聚合应用程序，更像是一个“剪辑后阅读”应用程序，但它可能会帮助你。**

**如果你想要一个把一堆潜在文章放在前面和中间的应用，你可以试试**[**【Flipboard】**](https://flipboard.com/)。你会看到一大块你感兴趣的网站或类别，上面有你可能想看的最新(或特色)故事。Flipboard 比 Google News 更具图形化的魅力，它允许你策划自己的内容“杂志”——如果你想与他人分享你最喜欢的文章或在方便的时候阅读它们。****

****你也可以尝试一下**[**【Feedly】**](https://feedly.com/)**——类似于 Flipboard 的概念，只是没有平铺和图形。许多 Lifehacker 编辑认为这是一个很棒的内容监管应用程序，因为它给你提供了两个推荐*和*允许你定制你认为合适的新闻源。********
