# 如果你在网上玩盗版任天堂 Switch 游戏，任天堂会封杀你

> 原文:[https://life hacker . com/if-you-play-盗版-任天堂-switch-games-online-ninte-1827107838](https://lifehacker.com/if-you-play-pirated-nintendo-switch-games-online-ninte-1827107838)

是的，有一种东西叫做 [免费的午餐](https://lifehacker.com/these-100-restaurants-will-give-you-free-food-on-your-1772840294#_ga=2.217864531.47977805.1529932552-3846207152.1521480874) 。但是如果你破解你的任天堂 Switch 来免费玩盗版游戏，那就要付出的代价:你不能在线玩[](https://arstechnica.com/gaming/2018/06/inside-nintendos-perfect-method-for-detecting-online-switch-piracy/)。

值得注意的 [任天堂黑客 ScriesM 最近在 Reddit](https://www.reddit.com/r/SwitchHacks/comments/8rxg26/psa_strong_antipiracy_measures_implemented_by/) 上概述了任天堂已经实施的反盗版措施，以嗅出玩盗版游戏的游戏机，并将其从任天堂网络中永久禁止。

任天堂在每台交换机的“TrustZone”核心中存储了一个加密的客户端证书，这是该控制台独有的。当你登录时，任天堂的服务器通过那个证书识别你的主机，这意味着如果你的交换机被禁止，它将一直被禁止。正如 [Ars Technica](https://arstechnica.com/gaming/2018/06/inside-nintendos-perfect-method-for-detecting-online-switch-piracy/) 指出的那样，3DS 黑客能够伪造一个令牌来让一个被禁止的控制台重新回到网络上，但这不能与交换机一起工作。

游戏，无论是实体卡上的还是数字下载的，都有自己的加密证书来证明它们是合法的。在下载的情况下，嵌入在游戏代码中的加密票还包括关于游戏机和购买它的用户的信息。尝试玩一个在另一台游戏机上用另一个账户购买的游戏副本，然后*嘣*，你的开关被禁止。你甚至不能在网上玩合法的游戏。

所有这些证书在你的主机上线时都会被检查，所以如果你完全离线玩盗版游戏，任天堂应该检测不到。然而，这并不是百分之百确定的。游戏机的固件可能会检测到你离线玩的盗版游戏，并在你重新上线时将你的游戏机标记为禁止。

[Ars Technica](https://arstechnica.com/gaming/2018/06/inside-nintendos-perfect-method-for-detecting-online-switch-piracy/) 和 [Reddit](https://www.reddit.com/r/SwitchHacks/comments/8rxg26/psa_strong_antipiracy_measures_implemented_by/) 有更多的细节，但要点是:任天堂已经变得非常擅长捕捉盗版 Switch 游戏的玩家。如果你想在网上玩任何任天堂 Switch 游戏，要么不要黑掉你的开关，要么非常小心。
