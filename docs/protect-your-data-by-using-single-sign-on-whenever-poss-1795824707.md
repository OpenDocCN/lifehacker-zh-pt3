# 尽可能使用单点登录来保护您的数据

> 原文：<https://lifehacker.com/protect-your-data-by-using-single-sign-on-whenever-poss-1795824707>

没有一家公司，无论是初创公司还是大型企业，拥有完美的安全系统。Chipotle 的支付系统遭到黑客攻击，OneLogin 遭遇数据泄露， [甚至谷歌](http://gizmodo.com/a-huge-and-dangerously-convincing-google-docs-phishin-1794888973) 也遭遇了严重的网络钓鱼攻击，导致数千个账户从一份共享文档中泄露。换句话说，我对向没有安全记录的公司提供任何信息持谨慎态度。



这只是说所有的安全都很难，你应该养成保持登录信息安全的习惯。通常这意味着首先不要登录。这就是单点登录服务的用武之地。它们不仅比普通的用户名和密码组合更安全，而且从长远来看，你需要管理的登录信息也更少。

简而言之，单点登录(SSO)服务在受支持的网站上充当您的用户名和密码，您的主要帐户(来自谷歌、脸书、微软等)充当您的帐户。

### 一站式安检

使用脸书、谷歌、Twitter 和微软等公司提供的安全单点登录服务提供了第二层安全，这比你从收集用户登录数据并将其存储在不太理想的地方的普通网站，或使用 [较差的加密标准](http://blog.zomato.com/post/160791675411/security-notice) 获得的安全要多。

[双因素认证](http://lifehacker.com/which-form-of-two-factor-authentication-should-i-use-1784769336) 是您应该尽可能使用的附加安全层。它可以防止获得您的用户名和密码的不良行为者在没有第二种身份识别形式的情况下进一步进入您的帐户，第二种形式的身份识别通常是来自手机的短信或来自身份验证应用程序的随机生成的代码。

如果一个网站不支持双因素认证或单点登录，要小心。使用一个密码管理器(当然)来创建一个复杂的密码，如果你特别小心的话，使用一个虚假的电子邮件地址(或者 Gmail 提供的众多电子邮件地址 [中的一个)。](http://fieldguide.gizmodo.com/how-to-use-the-infinite-number-of-email-addresses-gmail-1609458192)

### 你知道他们想要什么数据

对于单点登录服务，网站需要告诉你他们正在获取的数据和原因。就像安装应用程序可能会提示您授予麦克风或联系人访问权限一样，使用单点登录服务会提示网站请求您的电子邮件或联系人列表等数据，或者代表您发布到社交网站的能力，通常会让您接受或拒绝这些数据。

请求访问私人数据的服务(如 [简讯订阅服务](http://gizmodo.com/unroll-me-ceo-so-sorry-you-found-out-his-company-sold-y-1794592557) )确实有能力获取所述数据并将其交付给第三方，可能在你不知情的情况下。是的，它们通常很长，但你应该阅读该网站的隐私政策，看看他们收集了哪些数据，以及他们具体是如何处理这些数据的。如果发生这种情况，通过单点登录服务可以更容易地切断他们的联系。

### 咔嚓一声把它们切掉

使用单点登录提供者的杀手锏是您可以轻松撤销对数据的访问，通常只需点击几下鼠标。有没有试过在一些无聊的网站上注销账户？对我来说，这通常意味着决定取消你的帐户，却被带到一个论坛的支持页面，告诉你取消你的帐户意味着给客户支持团队发电子邮件，创建一个票证，确保它被删除，而不是登录来仔细检查它是否已经消失，因为你可能会重新激活它，并希望你的数据被及时销毁(通常不会)。

当然，使用单点登录服务可以让公司访问他们通常可能不知道的信息，但从更少的位置管理您的登录信息的便利性，加上使用谷歌或脸书等安全的单点登录服务时增加的安全性，意味着您可以更容易地找到并删除违规者。

### 没有密码了

你的 [密码管理器](http://lifehacker.com/the-five-best-password-managers-5529133) 是一个储存密码、个人信息和其他你不想记在脑子里的碎片(比如你的图书证号码)的好地方。

您的密码安全性取决于其最薄弱的环节。如果你没有及时更新你的密码，或者如果你犯了在多个网站重复使用密码的错误，你就在制造安全漏洞，而这些漏洞很容易被密码管理器之类的东西填补。

但是，放弃失败并使用像谷歌或脸书这样的单一登录提供商的最大好处是什么呢？不。更多。密码！嗯，你必须知道你的谷歌和脸书密码，但除此之外，任何使用你的单点登录系统的网站都不会要求你创建另一串文本和数字来记住(或记住生成和保存)。

当然，单点登录服务也不能幸免于攻击者。除了遵循您的登录提供商概述的步骤之外，您还可以通过使用双因素身份验证来进一步保护自己免受恶意攻击，使您的二级联系信息(如您的 [备用电子邮件](https://support.google.com/accounts/answer/70206?hl=en) )保持最新，并拥有一份硬拷贝 [备份代码](https://support.google.com/accounts/answer/1187538?hl=en) 。
