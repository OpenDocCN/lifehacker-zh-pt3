# 使用妈咪-gram 通过 Alexa 给你的孩子发短信

> 原文：<https://lifehacker.com/text-your-toddler-through-alexa-using-mommy-gram-1827421212>

你可能不想给你蹒跚学步的孩子一部手机，但是在会议间隙给你在家的孩子发一条信息或者得到那个非常重要的“麦片在哪里？!"通过短信而不是十几个电话传递信息可能会很好。

Watch

在你孩子这边，他或她会告诉 Alexa“让妈妈向妈妈问好”或者其他什么信息。妈妈会收到 Alexa 发来的文字信息。她可以通过文本回复，她的消息将通过你家里的 Echo (或其他任何 Alexa 设备)读给孩子 [。](https://amzn.to/2zfUMv6) 

对话可以来回进行，也可以就此结束。

它绝对不能代替放学后的电话，但它可以是一种很好的方式，当你在约会之夜或会议之间登记，而不必与保姆玩电话追逐。它能让你的小孩以安全可控的方式发送短信，而不用交出任何人的手机。
