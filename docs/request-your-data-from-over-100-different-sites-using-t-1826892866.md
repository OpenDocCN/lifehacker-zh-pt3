# 使用此服务从 100 多个不同的站点请求您的数据

> 原文：<https://lifehacker.com/request-your-data-from-over-100-different-sites-using-t-1826892866>

几乎你在网上使用的每一项涉及用户名和密码的服务都在某种程度上收集你的信息。在某些情况下，这些信息可能只是你的电子邮件和一些识别数据点，如你的年龄或性别。在其他情况下，你会传递你喜欢和不喜欢的信息，你的朋友是谁，甚至你一天去了哪里。



网站 [My Data Request](https://mydatarequest.com/) 通读了 100 多家不同公司的隐私政策，并为您提供了一种快速简单的方法来确定正在收集哪些关于您的数据，如果您想了解特定公司的具体信息，可以从哪里以及如何下载这些数据。

网站上的一些公司包括脸书、Tinder、优步、Twitter、YouTube、Venmo、Waze、Evernote、Spotify、比特币基地、Postmates、Chase 甚至愤怒的小鸟。

对于名单上的许多人来说，你可能对他们可能拥有的关于你的信息有一个不错的想法。对其他人来说，这个答案可能有点不清楚。

要使用该网站，您可以搜索您使用的特定应用程序或网站，或者浏览它们显示的选项。当你点击其中一个时，会弹出一个窗口，里面有一个指向该公司隐私政策的快速链接，以及一个你可以点击从该公司下载你的数据的链接。

这不会阻止这些公司继续收集和存储你的信息，但这可以是一种快速简单的方式来检查你在那里发布了什么信息。
