# 看看今年你会在汽油上多花多少钱

> 原文：<https://lifehacker.com/see-how-much-more-youll-spend-on-gas-this-year-based-on-1826840142>

目前全国的天然气价格是自 2014 年 以来最高的 [，这意味着你可能要重新计算你的天然气预算。这个工具根据你的汽车型号告诉你下一年你将在汽油上花多少钱，以及你认为你将行驶多少英里。它还能告诉你与去年相比，你会多花多少钱。](https://www.eia.gov/dnav/pet/pet_pri_gnd_dcus_nus_a.htm)

计算器从时间 开始 [，使用基于](http://time.com/5306658/gas-prices-calculator/) [能源信息管理局的估计](https://www.eia.gov/outlooks/steo/) 的数字。他们的数据显示，2018 年期间，汽油平均价格将为每加仑 2.79 美元，比去年的平均价格 2.42 美元高出 37 美分。那些美分加起来很快。

比方说，如果你驾驶一辆道奇公羊皮卡，年平均行驶里程为 13，476 英里(根据美国交通部[)，你今年将为汽油支付 2，330 美元。这比去年的估计高出 412 美元。对于小型车，如丰田卡罗拉，你可以预期支付约 219 美元以上。这是少了很多，但仍然是一个相当大的变化。](https://www.fhwa.dot.gov/ohim/onh00/bar8.htm) 

当然，每个人的驾驶习惯各不相同，你可以根据自己的需要调整计算器。你可以增加或减少每年行驶的英里数，改变汽油的平均价格，甚至添加你的车型，如果它还没有在列表上的话。请记住，所有的估计都是假设你的车已经行驶了广告所说的里程数。如果你的车状况不佳，或者你开得更猛，这些数字可能会改变。如果你想亲自试用这个计算器，请点击下面的链接。
