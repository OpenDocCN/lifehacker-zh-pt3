# 大学角落工作区

> 原文：<https://lifehacker.com/the-college-corner-workspace-1754622434>

乍看之下，它可能看起来很大，因为照片中有大量的设备，但 [亚当](https://www.flickr.com/photos/132735587@N06/) 是一名大学生，这是他的房间。巧妙利用垂直空间，你就拥有了一个适合工作和娱乐的角落。

Watch

照片中有三台电脑，如果算上左边的树莓派，技术上来说是四台。他解释说，有一台戴尔 XPS 8300、一台戴尔 XPS M1330、一台 20 英寸苹果 iMac(2009 年初)和一台 Raspberry Pi 2 total，它们都有自己的工作。他没有具体说明，但我们认为 Pi 是一个媒体中心，因为它在接收器旁边，M1330 是一台笔记本电脑，所以它可能连接到顶部的大屏幕，8300 为双显示器供电，iMac 在右边。

他没有解释所有系统具体做什么，所以我们将留给你的想象力，但它肯定是一个鼓舞人心的工作空间。如果你正在浏览照片，并在他的设置中挑选出一些更有趣的组件(例如，下面的 PlayStation 3 或雅马哈接收器)，亚当确实在下面的链接中提供了照片中所有项目的完整概要。

如果您有自己的工作空间可以炫耀，请将其添加到我们的 [**Lifehacker 工作空间展示并告诉 Flickr pool**](http://www.flickr.com/groups/lifehacker-workspace-showandtell/pool/) 来与我们分享。确保您包含的任何照片至少为 1280x720。将它们保持在 16:9 也有帮助！包括一些文字，关于你使用的材料，你是如何想出这个设计的，以及任何其他相关的细节。如果你巧妙的组织和良好的设计感吸引了我们的眼球，你可能就是下一个特色工作区！
