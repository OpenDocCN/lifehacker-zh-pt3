# 黑暗星球桌面

> 原文：<https://lifehacker.com/the-dark-planet-desktop-1764428894>

读者 [tatersneggs](http://kinja.com/tatersneggs) 组装了这款基于雨量计的桌面，它足够华丽，但实际上没有使用很多小部件来获得整洁、个性化的外观。他是这样设置的。



当然，这是 Windows——如果你是雨量计的新手，你应该考虑查看我们的雨量计指南 来帮助你入门。

一旦你熟悉了它，这里是你需要的:

*   [壁纸](http://pics-about-space.com/asteroid-background?p=1#)
*   用于 Windows 的 [雨量计](http://www.rainmeter.net/) 系统调试和监控工具
*   雨量计的 [编码皮肤](http://lilshizzy.deviantart.com/art/Rainmeter-Encoded-v1-2-3-215992941) 和时间/日期显示和天气的 [优雅 2 皮肤](http://lilshizzy.deviantart.com/art/Rainmeter-Elegance-2-244373054)
*   系统监视器雨量计的 [极简系统信息皮肤](http://dashie36.deviantart.com/art/Minimalistic-System-Info-v2-333556546)

这就是全部了。Tatersneggs 甚至制作了一个教程视频，告诉你如何设置这一切，如果你想知道如何做到这一点，你可以在 YouTube 上查看 。否则，自己尝试一下，或者点击下面的链接分享你的想法或提出你的问题！
