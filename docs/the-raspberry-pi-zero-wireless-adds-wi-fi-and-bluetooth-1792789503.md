# Raspberry Pi Zero W 为 Zero 增加了 Wi-Fi 和蓝牙功能，售价 10 美元

> 原文：<https://lifehacker.com/the-raspberry-pi-zero-wireless-adds-wi-fi-and-bluetooth-1792789503>

这款 [树莓派 Zero](https://lifehacker.com/the-raspberry-pi-zero-is-a-5-computer-the-size-of-a-st-1744253282) 是一款 [神奇的微型版树莓派](http://lifehacker.com/top-10-raspberry-pi-zero-projects-that-make-use-of-its-1792403310) 它将电路板缩小到口香糖大小，但它的一个问题是缺乏无线功能。树莓派 Zero W 是一个新版本，内置蓝牙和 Wi-Fi，价格是原来 Zero 的两倍。

Watch

树莓 Pi Zero W 与标准 Pi Zero 的规格基本相同，但增加了与树莓 Pi 3 的 T3 相同的 802.11n 无线局域网和蓝牙。CPU 和 RAM 与标准 Pi Zero 相同，本质上与最早的树莓 Pi 1 机型相同。以下是树莓派 Zero W 的详细规格:

*   1GHz，单核 CPU

*   512MB 内存

*   迷你 HDMI 和 USB 移动端口

*   微型 USB 电源

*   兼容 HAT 的 40 针接头

*   复合视频和复位标题

*   CSI 摄像机连接器

*   802.11n 无线局域网

*   蓝牙 4.0

树莓派 Zero W 也恰好在第一款树莓派 的五周年之际到来，所以现在似乎是比较所有可用型号规格的好时机。此时，这包括越来越难以解析的名称，Raspberry Pi 3、Pi 2 Model B、Pi 1 model B+、Pi 1 Model A+和 Zero(单击放大):

除了 Raspberry Pi Zero W，Raspberry Pi 基金会还发布了一款官方保护套，它看起来基本上类似于全尺寸保护套 ，但被切成了一半。该保护套配有三种不同的可互换盖子，可以更方便地使用 GPIO 或摄像头模块。它将适合树莓皮零和树莓皮零 w。
