# 主板为什么要有电池？

> 原文：<https://lifehacker.com/why-do-motherboards-have-batteries-1825773436>

在本周的 Tech 911 中，一位读者写信来寻求一些明确性，而不是解决设备损坏或硬件故障的问题。这位读者可能正在摆弄他的主板，突然有了一个想法:如果我的电脑有电源，为什么我的主板有电池？

Watch

Lifehacker 读者 **SAmore** 写道:

“你能发帖解释一下我是如何使用电池的吗？电池是什么？为什么我们要在主板上安装电池？同样重要的是，更换它的正确程序是什么？”

快速、简单、容易——至少是这个问题。为了得到答案，我们将从基础开始。在某些系统中，CMOS 电池为主板上的小型超低功耗易失性内存芯片供电。这个内存芯片负责保存主板的 BIOS 设置。如果你没有电池，你的主板将总是以出厂默认的 BIOS 设置启动——类似于土拨鼠日的场景。

为 CMOS 芯片供电的电池也为主板的实时时钟(RTC)供电。它负责主板保持准确的时间，并可以帮助您确定 CMOS 电池何时即将耗尽(或没电)。如果在开机自检期间，当您打开主板的 BIOS 或操作系统时，您发现时间确实不对，您很可能需要更换 CMOS 电池。

由于在某些情况下，CMOS 芯片负责保持主板的 BIOS 设置，当遇到启动或 POST 过程中出现问题的复杂 PC 时，您可以采用的一种更简单的故障诊断技术是取出 CMOS 电池。这会将主板 BIOS 的设置重置为默认值，在某些情况下，可能会修复您遇到的任何问题。

这些问题也可以提醒你，你的 CMOS 电池快没电了。如果您发现您的主板设置一直重置为默认设置，或者您在启动操作系统时遇到问题(或者甚至找不到硬盘)，如果您已经有几年没有购买或制造电脑了，那么可能值得考虑使用新的 CMOS 电池。

对于在非易失性存储器中存储 BIOS 设置的较新的 PC，当您的系统启动时(或者时间将完全错误)，CMOS 电池有问题的唯一线索可能是一个小错误信息。您仍然可以取出电池来重置 BIOS 设置。虽然 CMOS 电池并不直接需要将这些设置保存在内存中，但您的主板可能会看到丢失的电池并说，“哦，我应该将所有设置重置为出厂默认值，就像以前一样。”您的主板也可能有一个完全不同的程序来将其 BIOS 设置重置为默认值-最好查阅手册。

CMOS 电池的寿命取决于其总容量、主板消耗的电流量以及使用系统的频率。一个很好的经验法则是，大约在三年期 左右开始考虑更换 CMOS[。(意思是“考虑一下，以防你的电脑开始变得奇怪，而你不知道为什么。”你不需要无缘无故地出去储备电池。)](https://www.intel.com/content/www/us/en/support/articles/000006488/boards-and-kits/desktop-boards.html)

你的电池也许能让它更长——五年或更长。一般电池看起来没电了我才担心。由于手表电池价格便宜，如果(或当)电脑开始出现神秘的 BIOS 相关问题，去商店购买新电池只是一个小小的不便。
