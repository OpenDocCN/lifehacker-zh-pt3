这实际上是一个比你意识到的更棘手的问题，WWR。

每段感情都是一种妥协。不管一个人有多棒，你们都不会是 100%的匹配。安定下来意味着你将不得不接受这样的事实:有些事情是你在这段关系中得不到的，有些事情是你不喜欢的。有时候，这些都是相当小的问题，你可以把它们一笔勾销，作为进入这段关系的代价，因为你的伴侣太棒了。其他时候，它们是更大的问题，你和你的伴侣必须找到一个对你们双方都有效的妥协。

有时候，你会遇到这样的情况，在那里*不能*妥协，在那里，要求*太大*无法克服，唯一的答案就是一个人或另一个人让步。 [这些问题往往会导致交易破裂，它们会毁掉一段关系。](http://www.doctornerdlove.com/2016/04/how-to-handle-dealbreakers/)

和宗教可以成为*主要的*交易者。这将是你们关系中的一个重大问题，WWR，而且我认为你还没有意识到。

当你自己不信教时，这看起来相当简单；你做你的事，她做她的事，不要大惊小怪，不要干涉。也许你稍微弯下腰，参加圣诞节和复活节的礼拜，一切都会好的。但是对于一个宗教对他们来说如此重要的人来说，从长远来看，和一个不属于 T2 教的人约会可能会很困难。对于那些*是宗教徒，他们的宗教是他们生活中重要的一部分的人来说，这不仅仅是一个周日去哪里或玩游戏“不，但耶稣不介意，因为每个人都知道上帝让人们在技术上滑冰”的问题这是他们身份、社区和生活的核心部分。你不能把它当成你可以划分或搁置的东西，因为它告诉你关于他们的一切。和一个不太适应他们生活的人保持长期、认真的关系是非常困难的。例如，有一个问题是你如何融入她的社区。如果你的价值观与他们的冲突，这意味着总会有*到*的紧张。显然，你能够顺应事物，因为你是一个基督教大学的不可知论者。但是在一个学术机构和在她的日常生活和社区中是有区别的，在学术机构中有一定数量的辩论和提问。当一个你非常不同意的话题出现时，你还能保持沉默多久？*

你能接受这种没有归属感的感觉吗？当你和她的朋友、家人、社区在一起时，你能忍受被评判的感觉吗——不管这种感觉是否会发生？还是带着转换的压力？

虽然你可以忍受任何差异，但你的家人呢？他们能和她的家人和睦相处吗？每次和她父母见面都会变成一场关于宗教的辩论吗？家庭聚会会充满令人不安的沉默吗？

然后是这给你女朋友带来的压力。这在很大程度上取决于她属于基督教的哪一个分支，但有些人冷酷无情，互相宽容，而另一些人则非常“我们或他们”。当感觉到她生命中所有重要的人都认为你对她有不良影响时，你能理解她将要承受的压力吗？很容易说，如果她爱你，她应该能处理好……直到她生命中最重要的人因为*你而避开她。*

抛开那些，你们也对你们在一起的未来有疑问。你们两个打算怎么一起抚养孩子？你将以什么样的信仰来抚养他们？星期天你会和他们一起去教堂吗？你将如何处理假期？在他们成长的过程中，你能不能把你的怀疑放在一边，这样你就不会在你、他们和你妻子之间制造矛盾了？接下来的问题是，从长远来看，她会如何看待你们两个。如果你的女朋友是一个真正的信徒——听起来确实如此——你的不可知论会让你与天堂隔绝。对于那些不认同她的信仰的人来说，相信你不会和你爱的人永远在一起是一件非常痛苦的事情。你可以尽你最大的努力去说服她，但是拯救的本质和方法是自公元 600 年尼西亚会议以来一直争论的问题；除非你是另一个托马斯·阿奎那或 C.S .刘易斯，否则你很可能不会改变她或她所在教会的神学。要记住的另一件事是:对你来说，她的宗教信仰只是她的一小部分。对她来说，这可能就是一切。当她相信你可能会回心转意是一回事。但是你们在一起的时间越长，你对她的信仰的拒绝就越像是对她作为一个人的拒绝。这种问题最终可能成为一段关系中的定时炸弹，慢慢积累，直到达到临界质量，然后爆炸，到处都是。

如果你们俩陷入僵局，那么是的，这段关系*将会*结束。这是不可能的。但是我*确实*想对呆在一起在某种程度上“浪费”你的大学时光的想法提出异议。人们常犯的一个错误是认为，除非你们中的一个或两个都死在马背上，否则这段关系就是失败的，或者是浪费时间。仅仅因为你不打算结婚并不意味着你在浪费和你女朋友在一起的时间。享受和你爱的人在一起的时光没有错，即使你知道这不会是永远。你能享受你们在一起的时光，即使知道这一切即将结束。从很多方面来说，对你来说实际上更好。有一个截止日期意味着你可以按照自己的方式结束这段关系，而不是发现你已经超越了这段关系，或者陷入痛苦和尖刻之中。相信我:如果你们的关系结束了，你们两个仍然可以回忆起快乐的时光，仍然对彼此有好感？按照我的标准，那是一段成功的关系。如果你和 T21 决定在一起，那就睁大眼睛，抱着现实的期望去做。不要把它看作是改变对方想法的最后一搏；这只会让你既沮丧又不安。珍惜你所拥有的，然后友好地分手。如果你决定结束它:快速干净地做它。彻底的决裂伤害最小，恢复最快。

我不知道我是否会给他贴上瘾君子的标签，因为这是一个相当有内涵的术语，可以随便乱用，而不用真正思考它。许多成瘾行为与行为本身关系不大，而是倾向于自我治疗的一种形式。

所以在我把这个问题称为*问题*之前，我会有很多*问题*，更不用说上瘾了。第一个是:他在主动伤害自己吗？他手淫到擦伤或损伤阴茎皮肤的程度了吗？他自慰的欲望会给他带来麻烦吗——也就是说，他会在不应该自慰的时候自慰吗？如果他*不*一天手淫几次，他会焦虑或者不舒服吗？他有没有可能翘一天两天不心疼？

另一个重要的问题是:他放弃和你做爱是为了满足自己吗？他对你们的性生活投入了吗，或者他只是想和你快速做爱，而不是更投入和相互的东西？

我想知道这些问题的答案，然后我才会放心地把这个问题贴上*问题*的标签，更别提上瘾了。不寻常吗？是的，我会这么说。不要误解我的意思，性冲动的范围从无性到色狼，但是保持一个大多数人只能在青春期早期达到的活跃水平是令人印象深刻的。但是不寻常并不意味着*不好*，尤其是如果它没有干扰他的生活或者他和你的关系。

我认为*的一个问题是*他给自己贴上离经叛道者标签的方式。我们仍然生活在一个极度否定性的文化和 [中，这种文化教导男人和女人相互冲突的“规则”,坚持认为如果我们没有以“适当”的方式做爱，我们就是坏人。特别是如果他在一个宗教家庭中长大，人们很容易将任何非标准的性表达模式(对“标准”的任意定义)视为他是一个坏的、有病的或以其他方式崩溃的人的证据。我怀疑他对自己的性取向感到羞耻……尤其是当几乎每个人都手淫的时候。](http://www.doctornerdlove.com/2013/10/what-we-get-wrong-about-sex/)

现在，如果他的手淫习惯没有对他的生活产生负面影响，那就不一定是个问题。这听起来不像是他在性方面忽视了你，尽管如果这影响了他的表现，他真的应该承诺给自己一个喘息的机会，和你在一起，而不是几个回合的“砰-砰-谢谢-龟头”。这是他自私的一面，但不一定是威胁关系的行为。如果他宁愿自己离开而不是做爱，或者如果他把你当人肉灯而不是他的伴侣，那就完全是另一回事了。

话虽如此:一些行为让我竖起了眉毛。每天六次？为了睡觉，他每天晚上都要抽搐几次？！这让我怀疑，这与其说是与任何成瘾或任何内啡肽有关，不如说是与他试图缓解的高度焦虑有关。如果是这样的话，那么他最好和治疗师谈谈，而不是每 20 分钟就敲打一次。

完全诚实地说，我认为不管怎样，和治疗师谈谈对他是有好处的——不是因为他手淫的频率，而是他对它的感觉。如果他真的感到孤立，并对自己的手淫习惯感到羞愧和内疚，那么与治疗师一起工作可以大大缓解他的整体情绪。你可以通过 [美国性教育工作者、顾问和治疗师协会推荐系统](https://www.aasect.org/referral-directory) 找到一位性积极治疗师。希望这能给他带来一些平静，并确保他的习惯是健康的而不是有害的。
