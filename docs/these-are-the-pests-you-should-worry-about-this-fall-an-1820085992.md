# 这些是你今年秋冬应该担心的害虫

> 原文:[https://life hacker . com/these-is-the-pest-you-should-you-should-you-should-on-1820085992](https://lifehacker.com/these-are-the-pests-you-should-worry-about-this-fall-an-1820085992)

像蟑螂、蚂蚁、老鼠和蚊子这样的害虫是令人讨厌的害虫，但通常只在一年中的特定月份出现。然而，由于今年的热浪和暴雨，许多通常在秋天和冬天消失的害虫仍然会在周围寻找家园——你的家。

最近， [国家害虫管理协会](https://npmapestworld.org/) (NPMA)发布了他们的 [昆虫晴雨表对即将到来的季节](https://pestworldmag.npmapestworld.org/2017/09/26/heavy-rain-heat-waves-drive-high-pest-pressure-in-fall-and-winter/) 的预测，对于那些从各种令人毛骨悚然的爬行动物那里得到紧张不安的人来说，这看起来并不好。昆虫学家和害虫专家预计这些生物会比平时承受更大的压力。这是一张来自 NPMA 的便捷地图:

由于炎热和干燥，西北太平洋地区寻找水源的啮齿动物数量将会增加。在西海岸和西南部各州，由于持续的高温，我们可能会看到更多的蟑螂、蚂蚁、蜘蛛和螫人的昆虫，而洪水会增加蚊子的数量。在中西部，温暖的春天和温暖的夏天增加了啮齿动物的数量，它们会在寒冷的天气里寻找食物和住所，更多的降雨将意味着更多的蠼螋和千足虫会在室内寻找更高的地方。在东南部，蚊子会继续恐吓任何身体里留有血液的人，除了蚂蚁之外，白蚁也会一直活跃到冬天。最后，东北部预计会看到许多臭虫和瓢虫，以及更多的昆虫和啮齿动物聚集在室内躲避寒冷。蜱也将继续成为一个问题，直到气温开始大幅下降。

然而，这不仅仅是被虫子和老鼠吓到。正如 [吉姆·弗雷德里克斯博士，NPMA 首席昆虫学家兼技术和监管事务副总裁，](https://pestworldmag.npmapestworld.org/2017/09/26/heavy-rain-heat-waves-drive-high-pest-pressure-in-fall-and-winter/) 解释说，这是一个健康问题:

> “夏季害虫活动延长，加上秋季害虫的高压，将使即将到来的季节害虫特别严重。考虑到蜱、蚊子和啮齿动物都有可能将疾病传播给人类，这也是一个额外的健康问题。”

所以，为即将到来的入侵做好准备，保持警惕。你现在能做的越多来防止虫害，你就能更好地度过冬天。
