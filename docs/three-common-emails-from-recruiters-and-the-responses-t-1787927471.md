# 三封来自招聘人员的常见电子邮件和他们实际想要的回复

> 原文：<https://lifehacker.com/three-common-emails-from-recruiters-and-the-responses-t-1787927471>

寻找新职位是你做过的最有压力的事情之一。也许是搜索背后的高风险让人们很容易过度分析它的每一部分，尤其是当你如何回复招聘人员发送的电子邮件时。我知道在我成为招聘人员之前，我花了太长时间试图对我收到的每一封电子邮件写下完美的回复。我想，他们必须是完美的，因为有一份工作悬而未决。



*本帖原载于* [*缪斯*](https://www.themuse.com/advice/3-emails-recruiters-sendand-the-responses-they-actually-want) *。*

但当我开始给候选人写电子邮件时，我很快意识到，人们给自己施加了太多的压力来“做出正确的回应”所以，在你花整个周末试图弄清楚如何签核( [)之前，最好还是真诚地](https://www.themuse.com/advice/sick-of-signing-all-your-emails-with-best-heres-when-you-can-start-dropping-the-formalities) ？)，这里有三封招聘人员发送的电子邮件，以及他们真正想从你这里得到的答案。

### 1.安排电话面试

如果你想避免在某人试图安排电话面试时冒犯他，这是有意义的。但是，根据我的招聘经验，我知道两件事:招聘人员很忙，当他们想安排电话面试时，在这个阶段，最具攻击性的电子邮件会毁掉你的候选人资格。

#### 如何应对

如果招聘人员问你什么时候有空，你可以回信说，“那个时间对我合适。在此期间，如果您还需要什么，请告诉我。”就是这样。如果他或她询问您有空发言的时间，请使用以下模板:

> *您好[招聘人员姓名]，*
> 
> 非常感谢你伸出援手。我可以在[插入您有空接受电话采访的日期和时间]发言。在此期间，如果您还需要什么，请告诉我。
> 
> 期待与您讨论该角色！
> 
> *【名称】*

这就够了。不需要赘言，甚至不需要过分强调你对这个角色的兴趣。

### 2.可以重发简历吗？

你可能在想，“等等，什么？你需要一份 [更新的简历](https://www.themuse.com/advice/a-simple-trick-to-always-keep-your-resume-updated) ？你不相信我发给你的原版吗？”如果是这样，深呼吸，放松。即使一家公司有一个复杂的系统来跟踪申请人，有时技术不配合，网站关闭，文件损坏，你的简历可能会发生各种各样的事情，让招聘人员不得不要求另一份副本。如果发生了，不要想太多。只要再发送一次，并相信你没有做错任何事情。

#### 如何应对

这个相对简单:

> *您好[招聘人员姓名]，*
> 
> 希望你一切都好。附上我的简历。请让我知道我是否可以重新发送任何其他材料。
> 
> *最佳，*
> 
> *【名称】*

说真的，这就够了。不要老想着为什么招聘人员会要求这样做。发就是了。

### 3.你今天有时间聊天吗？

这和电话面试邮件不一样。很不一样。这可能意味着很多事情。你可能会发现自己没有得到理想的工作。或者，你得到了它，是时候庆祝了。我曾经过度分析这类邮件，以至于半个小时过去了，我意识到，哦，糟糕，我可能应该回复。

不要担心你回答这个问题的时候说了什么——不管怎样，你都得到了一些消息，而且你将与招聘人员进行一次更长的谈话，在这次谈话中，你将能够使用你所有的话。

我明白了。面试很难 ，甚至更棘手，因为每一层都很紧张。毕竟，有一个职位有实实在在的薪水，实实在在的福利，可能还有非常实实在在的快乐。你想把每一部分都搞定，应该受到表扬。但是不要担心:当涉及到这类招聘邮件时，在你的回复中少说点不会让你失去你梦想中的工作。
