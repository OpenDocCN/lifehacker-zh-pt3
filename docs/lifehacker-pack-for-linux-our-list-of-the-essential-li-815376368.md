### 生产力

#### [突触](https://launchpad.net/synapse-project)T3】

我们喜欢应用程序启动器以及它们给我们的工作流程带来的速度，它们可以做 [的事情，而不仅仅是启动应用程序](https://lifehacker.com/why-you-should-be-using-an-app-launcher-and-how-to-mak-5963597) 。不幸的是，现在 GNOME Do 自 2009 年以来没有任何重大更新，Synapse——集成了时代精神的出色 GNOME Do 替代品——是我们目前的选择。如果你使用的是 Ubuntu 的 Unity 界面或者 GNOME Shell，你可以跳过这一步，因为他们有很多内置的应用启动器功能。但对于那些在其他桌面环境中的用户，我们建议至少检查一下 Synapse，以满足您的应用程序启动和其他需求。或者， [GNOME Do](http://do.cooperteam.net/) 仍然可以下载，如果你真的是一个极简主义者，你可能会喜欢 [dmenu](http://tools.suckless.org/dmenu/) 。KDE 用户也内置了方便的 [KRunner](http://userbase.kde.org/Plasma/Krunner) 。

#### [凯特](http://kate-editor.org/)T3和[吉亚尼](http://www.geany.org)T7】

当内置的 Gedit 不能满足要求时，Kate 和 Geany 将带来一些更高级的编码和开发功能。他们有类似的功能集，但 Kate 是我们最喜欢的文本编辑器，提供语法高亮、代码折叠、动态拼写检查、类似 vi 的输入模式，甚至代码自动完成。如果你需要的比内置编辑器能提供的更多，Kate 和 Geany 会让你开心。如果你想要比这些更硬核的东西，可以看看 [日蚀](http://www.eclipse.org/) 或 [崇高文本 2](http://www.sublimetext.com/2) 。如果你渴望 Notepad++的话，可以看看 [Notepadqq](http://notepadqq.altervista.org/wp/) 。

#### [车钥匙](http://code.google.com/p/autokey/) 车钥匙

文本扩展是提高工作效率的最大方法之一。想想你一天中所做的任何单调乏味的打字工作——地址、预设的电子邮件回复、一些代码或任何其他东西——想象一下只需击几下键就能全部输入。文本扩展就是这么做的，它可以 [节省你输入](https://lifehacker.com/how-to-use-text-expansion-to-save-yourself-hours-of-typ-5611210) 的时间。Linux 的文本扩展应用并不多，但是 [AutoKey 非常适合这个需求](http://lifehacker.com/the-best-text-expansion-app-for-linux-5845445) 。对于更高级的代码片段，您需要一些 Python 技能，但目前，这是我们最好的技能。

#### [图书馆](http://www.libreoffice.org/)

很有可能，LibreOffice 随您的 Linux 发行版一起提供，但为了以防万一，我们在这里添加了它。从文档到电子表格到演示文稿等等，LibreOffice 拥有您完成工作所需的工具，尤其是当您与使用 Microsoft Office 的人来回发送文件时。

### 互联网和通信

#### [铬](https://www.google.com/intl/en/chrome/browser/)

Chrome 是今年三大平台上我们最喜欢的 。它速度快，功能强大，同步一切，并有一个令人难以置信的扩展库。你的发行版附带的版本可能是它的开源表亲， [Chromium](http://www.chromium.org/) ，它拥有所有相同的优势。(一个小的区别是:如果你想要的话，你必须单独安装 flash 插件，尽管你的发行版可能有一个工具可以让这变得容易。)也就是说， [火狐](http://www.mozilla.org/en-US/firefox/new/) 也是一个很好的选择，如果你习惯的话。

#### [洋泾浜](http://pidgin.im/)

我们仍然认为洋泾浜是 Linux 上围绕 最好的 IM 客户端，尽管事实上 Ubuntu 开发者(和其他人)已经放弃了它，转而使用像 [同理心](https://help.gnome.org/users/empathy/stable/) 这样的程序。它不仅支持大量的 IM 协议和特性，而且它有一个非常可爱的扩展库，可以让你做任何你想做的事情。如果您正在使用 GNOME shell，那么 Empathy 确实有一些不错的集成选项，所以也值得一看。

#### [Skype](http://www.skype.com/) 和 [Google Hangouts](http://hangouts.google.com)

Google Hangouts [在 Linux 上轻而易举地击败了 Skype](https://lifehacker.com/the-best-video-chat-app-for-linux-5849855)，但这仅仅是因为 Skype *仍然*不适用于现代 64 位架构。不幸的是，大多数人使用 Skype 进行视频聊天，这意味着你生活中的某个人——朋友、家人或其他人——总有一天会想用它和你进行视频聊天。如果可以的话，使用 Hangouts，但是现在就解决 Skype 安装的问题，这样你就可以把它放在口袋里，以备不时之需。

### 音乐、照片和视频

#### VLC

很有可能，你选择的 Linux 发行版附带了一个相当可靠的视频播放器，比如 MPlayer。对大多数人来说，这很好，但如果你需要更多的控制，VLC 是一个好地方开始。它支持更多的视频和音频格式，几乎不需要任何工作就可以播放你的电影——尽管它确实为你的高级用户提供了一些方便的命令行工具 。

#### [digiKam](http://www.digikam.org/)T3和[shot well](http://yorba.org/shotwell/)T7】

Linux 其实有几个很扎实的照片管理工具，但是 [我们最喜欢的绝对是 digiKam](https://lifehacker.com/the-best-photo-management-app-for-linux-5877908) 。它更专业，这意味着它有更多的功能，包括大量的组织功能，支持 300 多种原始格式，能够并排比较图片，等等。不过，使用起来有点复杂——所以如果你喜欢简单一点的东西，Shotwell 可能更适合你。它可以完成大多数用户需要的基本排序、标记和编辑，此外它还内置了将照片分享到脸书、Flickr 和 Picasa 的功能(digiKam 也拥有这一功能)。

#### [GIMP](http://www.gimp.org/)

如果你正在编辑一些在 digiKam 或 Shotwell 中无法完成的东西——无论是截屏还是你只是需要一些更高级的工具 GIMP 或许可以完成。它可能不是 Photoshop，但是 [它自己可以做很多事情](https://lifehacker.com/top-10-photoshop-tricks-you-can-use-without-buying-phot-5864755) 。

#### 克莱曼婷

为这个列表挑选一个音乐播放器是很困难的。Linux 有相当多的选择，正如我们之前说过的，音乐播放器是一个令人难以置信的个人选择。最后，我们决定去克莱曼婷。它有一套很好的功能，一个易于浏览的界面，受到初级和高级用户的喜爱。如果你想要一点不同的东西，我们建议你也去看看 [女妖](http://banshee.fm/) 和 [阿玛洛克](http://amarok.kde.org/) 。

#### [Spotify](https://www.spotify.com)T3】

无论你为你的音乐播放器选择什么，我们建议你手边有一个流媒体服务，即使它不是你的主要播放器。我们喜欢 Spotify，虽然它在 Linux 上没有技术支持，但 Spotify 有一些可用的预览版本，至少可以让你启动并播放。如果你不介意放弃浏览器标签，你也可以只使用 [网络播放器](https://play.spotify.com/) 。

### 公用事业

#### [收纳箱](http://dropbox.com/)

如今，我们很多人都不止拥有一台设备。可能家里是 Linux 机器，工作是 Windows 电脑。或者可能是三台电脑、一部智能手机、一台平板电脑和一台运行 Archbang 的 [上网本。无论你的设备跨度有多大，Dropbox 对于保持你所有文件(和](http://lifehacker.com/archbang-brings-arch-linuxs-greatest-features-to-your-p-5857636) [其他东西](http://lifehacker.com/top-10-clever-uses-for-dropbox-5933884) )的同步是绝对必要的。开始时你有 2 GB 的空闲空间，但是很容易 [加载额外的空闲空间](https://lifehacker.com/the-cheapskates-guide-to-getting-free-dropbox-space-5796318) 。

当你必须下载一个大文件时，BitTorrent 几乎总是比缓慢的直接下载更好的选择。Linux 有一些不错的 BitTorrent 选择，但是我们最喜欢的客户端是大洪 T3。它使用起来很简单，功能丰富，有一个很好的插件库，所以高级用户可以根据自己的喜好调整他们的 [速度](http://lifehacker.com/how-to-boost-your-bittorrent-speed-and-privacy-286607) 和 [隐私](https://lifehacker.com/how-do-i-torrent-safely-now-that-demonoid-is-down-5936938) 的所有功能。如果你不是大洪的粉丝，试试[qBitTorrent](http://www.qbittorrent.org/)——它同样令人敬畏。

#### [坠毁计划](https://www.crashplan.com/)

每个人都需要备份。没有比硬盘崩溃、不得不从头开始更糟糕的感觉了。进入紧急计划。虽然你总是可以将 [备份到外置硬盘](http://lifehacker.com/how-to-back-up-your-computer-to-an-external-drive-5816453) ，但如果你在火灾、盗窃或其他灾难中丢失了电脑，那就救不了你了。CrashPlan 使用 CrashPlan 的云服务或朋友的电脑将你的电脑备份到云，无论如何都可以保证你的数据安全。另外，设置 真的很容易。设置它，忘记它，放松。

#### [PeaZip](http://peazip.sourceforge.net/)

Linux 有很多文件归档工具，如果你是一个命令行爱好者，只需看看终端就能搞定一切(无论是内置的 tar 命令还是令人敬畏的 [p7zip](http://p7zip.sourceforge.net/) )。但是，如果你需要一个更友好的 GUI， [PeaZip 是我们的选择](https://lifehacker.com/the-best-file-archive-utility-for-linux-5895919) 。它可能不漂亮，但它可以与超过 130 种不同的档案类型，加密档案安全保管，并与 GNOME 和 KDE 集成。另外，它还有高级用户渴望的命令行特性，在不需要 GUI 的时候使用。

#### [酒](http://www.winehq.org)

Linux 有一些很棒的应用程序，但是有时候大公司忽略了 Linux，我们被冷落了。Wine(有时)就是答案:如果你有一个你不能离开的 Windows 程序(无论是工作用的 Outlook、图像用的 Photoshop 还是娱乐用的魔兽世界)，Wine 会在你的 Linux 桌面上运行它。它并不适用于市面上的所有程序，但 Wine 的应用数据库 将帮助你找出哪些程序运行良好，因此你可以离永远离开 Windows 更近一步。

#### [](https://www.virtualbox.org/)

**当 Wine 不能满足你的需求，而你只有一两个运行 Windows 程序的时间时，VirtualBox 是你的下一个选择。VirtualBox 将在虚拟机中运行整个 Windows 安装，因此您无需离开 Linux 就可以执行所有 Windows 任务。这并不总是理想的，但是如果你在工作中坚持使用 Windows，这可能是你需要的折衷方案。**

#### **[终结者](http://www.tenshu.net/p/terminator.html)T3】**

**Linux 用户在终端上花费的时间比一般的 Windows 或 Mac 用户要多得多，这意味着你手头应该有一个非常好的终端模拟器。你的发行版自带的默认终端可能没问题，但是 [终结者会把你的命令行工作带到下一个层次](https://lifehacker.com/the-best-terminal-emulator-for-linux-5858676) 。你可以在网格中排列终端，重新排序，配置一组快捷键，保存你的布局，等等。如果你不想要或不需要《终结者》提供的一切，你可能还是想看看 [瓜克](http://guake.org/) 和 [亚瓜克](http://yakuake.kde.org/) ，你可以通过键盘快捷键访问这些令人敬畏的下拉终端。**

#### **命令行工具**

**命令行通常是完成简单任务的最快和最简单的方式，你的发行版附带的标准命令行工具已经非常强大了。不过，这些东西的来源还不止这些！这里有一些你可能没有意识到的额外工具。**

*   **[**pdf tk**](https://www.pdflabs.com/tools/pdftk-server/)**切片和切块 pdf。你是否曾经为了签署最后一页而不得不打印并扫描一份十页的合同？下一次，单独扫描页面，并运行命令:****

> ****`pdftk contract.pdf cat 1-9 output firstnine.pdf`****
> 
> ****`pdftk firstnine.pdf lastpage.pdf output signedcontract.pdf`****

*   ****[**un distract-me**](https://github.com/jml/undistract-me)**解决了当你开始一个长时间运行命令(可能是一个 [代码编译](https://xkcd.com/303/) )时，当你等待它完成时切换到你的网络浏览器，然后意识到你完全忘记回到你正在做的事情。Undistract-me 会监视耗时超过 10 秒的命令，并在命令完成时在屏幕上弹出通知。******
*   ******[](http://joe-editor.sourceforge.net/)**是你不想离开终端时的完美文本编辑器。无论你习惯使用什么样的编辑器，它都能快速加载并易于使用。那是因为它有一套变色龙般的别名。所有的 [emacs](https://www.gnu.org/s/emacs/) 快捷键都记住了吗？作为 jmacs 调用它。怀念 [微微](https://en.wikipedia.org/wiki/Pico_(text_editor)) ？称之为 jpico。如果你不知道这是什么意思，就输入 joe。(或者向上滚动到我们对凯特和吉亚尼的推荐。)********
*   ******[**Smem**](https://www.selenic.com/smem/) 测量电脑使用的内存量。当然，你*认为*你有这样的工具。但是有些内存是共享的，像 free 这样的工具并没有以最有帮助的方式来计数。Smem 给出了更有意义的数字。******
*   ****[**Powertop**](https://01.org/powertop) 是一个方便的工具，可以找出为什么你的笔记本电脑的电池消耗如此之快，或者为什么你的台式机变得如此之热。它显示了每个当前运行的程序使用了多少能量。它还可以调整你从来不知道自己拥有的低级别电源管理功能，例如，当你的 USB 控制器不使用时，它会挂起吗？现在你可以知道了。****
