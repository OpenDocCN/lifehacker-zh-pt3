# 这些国际航空公司接受宠物

> 原文:[https://life hacker . com/these-international-airlines-accept-pets-1827636252](https://lifehacker.com/these-international-airlines-accept-pets-1827636252)

几年前，我带着我的狗从加利福尼亚开车到北卡罗来纳州过圣诞节。我原本计划为我们为期一个月的访问飞行，但在 22 磅，我的狗最终有点太重了，不能合法地在我的座位下飞行，我也不愿意把他作为货物飞行。于是:#FatTucker 和我去了一次越野公路旅行。

带宠物飞行很复杂。如果你有一只不小的狗或一只超大的猫，那么你要么同意让他或她空运货物(这对你毛茸茸的朋友来说可能是危险和紧张的)，要么让他们作为服务动物飞行。(旁注:对于那些为了让宠物入伙而撒谎的人来说，地狱里有一个特殊的地方。不要这样做。)

如果你是一位宠物父母，想要带着你毛茸茸的朋友 [飞往世界各地，该组织比较了市场](https://www.comparethemarket.com/pet-insurance/content/jet-set-pets/) 最近从国际航空公司收集的宠物政策数据，列出了最佳和最差的选择。

请记住，你不能带着你的宠物跳上飞机，飞到任何你想去的国家。在你旅行的前一天，你会想了解那个国家的法律。有些会让你的毛茸茸的朋友进来，只要他有某种疫苗的证明，而其他人可能需要一个隔离期或根本不让你的动物进来。

也就是说，根据报告，这些航空公司根据成本和是否允许宠物进入机舱(出于安全考虑，您希望它们尽可能进入机舱)制定了最佳宠物政策:

*   英国航空公司

*   汉莎航空

*   TUI

*   托马斯·库克

*   土耳其航空公司

如果你正在计划一次国际旅行，并且已经知道你的宠物可以合法飞行，那么你可能想用“最佳”选项开始你的航班搜索，这样你就知道你和你毛茸茸的最好的朋友会安全快乐地到达。相比之下, [在其网站](https://www.comparethemarket.com/pet-insurance/content/jet-set-pets/) 上有更多关于每家航空公司及其要求以及您的宠物乘坐每家航空公司所需费用的详细信息。

如果你决定带着你的宠物预订一次国际旅行，确保在旅行前联系航空公司，最好是在你预订航班的时候。即使你的宠物符合所有要求，航班也往往只允许一定数量的动物上飞机，并且只允许坐在特定的区域(假设你的朋友正在机舱内飞行)。你要确保你能尽早出发，这样你就不会在假期有任何意想不到的惊喜。
