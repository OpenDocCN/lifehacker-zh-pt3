# 使用 RoboKiller 浪费电话诈骗者的时间，而不是您的时间

> 原文:[https://life hacker . com/waste-phone-scammers-time-inst-of-your-with-robo kil-1824049341](https://lifehacker.com/waste-phone-scammers-time-instead-of-yours-with-robokil-1824049341)

在我的职业中，我经常会接到很多没有存为联系人的号码打来的电话。这意味着我将在工作日接听电话，如果我不认识这个号码的话。大多数时候，电话另一端的人会打电话给我，提出与工作有关的要求，但最近每天至少有两个电话是有人试图为我并不拥有的房子再融资，或者帮我降低汽车保险的费率，因为我几乎十年没有拥有汽车了。

Watch

尽管在不要打电话的名单上，他们还是会打电话。我屏蔽了他们的号码，但第二天他们会从另一个号码打来，一个又一个。毫无疑问，这是一段糟糕的时光。现在，一个新的应用程序正在这样做，这样在电话另一端的骗子那里，一天中错过时间的人，而不是我。

这款名为 [Robokiller](https://itunes.apple.com/us/app/robokiller-stop-spam-calls/id1022831885?mt=8) 的应用，本质上只是另一个垃圾邮件拦截器。当有人从存储为垃圾邮件的 20 万个号码中的一个给你打电话时，你的电话不会响。你会收到一个通知，让你知道一个电话已被阻止。

不同之处在于打电话的人身上发生了什么。Robokiller 不只是阻止来电，而是使用一系列预先录制的信息来接听电话，这些信息旨在让来电者认为他们正在与一个真人交谈。浪费他们的时间。

不幸的是，这款应用不是免费的。订阅费每月 2.99 美元，每年 24.99 美元。也就是说，如果你是一个经常被电话轰炸的人，那么这个花费可能是值得的。

如果你对骚扰你的骚扰者不感兴趣，只是想阻止他们，有大量的应用程序可以帮助你做到这一点。美国电话电报公司和威瑞森都为他们的客户提供呼叫屏蔽应用。 [Trapcall](https://www.trapcall.com/apps) 对于智能手机来说也是一个不错的选择(有一大堆选项，只要搜索应用商店就行了)，而 [Nomorobo](http://nomorobo.com/) 将完成座机的工作。
