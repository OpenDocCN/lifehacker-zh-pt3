# 记住是一个很好的初学者密码管理器

> 原文:[https://life hacker . com/remembear-is-a-good-password-manager-for-初学者-1820888270](https://lifehacker.com/remembear-is-a-good-password-manager-for-beginners-1820888270)

你现在知道了 [你绝对需要一个密码管理器](https://lifehacker.com/how-to-create-a-strong-password-1797681069) 。但是你从来没有时间去买一个。让我们现在用[remember](https://www.remembear.com/)来解决这个问题，这是一个新的密码管理器，很容易安装和计算。我们对其进行了测试，虽然对于大多数用户来说，我们仍然更喜欢 1Password ，但我们建议初学者使用 RememBear，尤其是在其免费测试期。

### 为什么我们喜欢它

RememBear 来自于隧道熊的制作者，[life hacker 最喜欢的 VPN 之一](https://lifehacker.com/tunnelbear-mobile-is-an-elegant-easy-mobile-vpn-that-k-1635233773) 。它有着同样的熊主题的奇思妙想，有着干净、扁平的界面。就像密码管理员的无泪洗发水。

RememBear 拥有所有必要的功能:它生成新密码，测量密码强度，并支持移动和桌面上的指纹识别。每当你登录到一个网站，它提供保存登录。它与你的其他应用程序完美地集成在一起。与 1Password 一样，您可以通过 Safari 中的“共享到”菜单调用移动应用程序，从而减少移动登录的麻烦。

而且 Chrome 扩展也很不可思议。RememBear 不会让你在工具栏中寻找它的图标，甚至右键单击，而是在每个登录表单中添加一个谨慎的可点击的熊图标。

用老套的设计师术语来说，这个界面是“令人愉快的”当你输入主密码时，一只熊会观察你移动的光标。安装这个应用程序，它会奖励你“成就:放一只熊进去。”有一次可爱过头了:我第一次把 RememBear 同步到我的 iPhone 上时，应用程序*咆哮着*，尽管我处于震动模式。但是可爱并没有渗透到主 app 体验中，干净简单。

有时候，应用程序更简单，因为它的功能更少。目前的测试版只提供两个入口类别:登录和信用卡。这比 1Password 的 17 个专门类别要简单得多，其中包括社会安全号码、护照、无线路由器，甚至“户外许可证”RememBear 的两个类别包括较少的表单选项，但开放式的“注释”部分通常就是您所需要的全部。

虽然 RememBear 看起来很棒，但它仍然不适合超级用户。与目前大多数密码管理器一样，RememBear 通过云同步你的加密密码数据，但 [与 1Password](https://support.1password.com/sync-options/) 不同，你不能选择通过 iCloud、Dropbox 或私人服务器同步。当当前的测试期结束时，同步 RememBear 将需要付费订阅，就像 1Password 或 LastPass 一样。但是你现在可以免费试用，如果你一直懒得“开始”，这真的很有帮助。

### 如何安装

我把设置留到最后，因为如果你不使用密码管理器，你现在就要启动*。你要去 [点击这个直接下载链接，记住](https://www.remembear.com/download/) 并打开文件。你将要安装 [iOS](https://itunes.apple.com/us/app/remembear-password-manager/id1145554431?mt=8) 或 [Android](https://play.google.com/store/apps/details?id=com.remembear.android&hl=en) 应用。(您可以按任意顺序执行这些操作。)然后你就要下载 [Chrome 扩展](https://chrome.google.com/webstore/detail/remembear/ehpbfbahieociaeckccnklpdcmfaeegd) 。*

*第一次安装 RememBear 时，您会给它一个主密码，您应该把它写在纸上或记在脑子里。(如果你坚持把你的主密码保存在手机或电脑上，至少把它放在一个不显眼的地方，不要贴上“密码”的标签。这仍然是一个坏主意，但我知道你们中的一些人无论如何都会这样做。)*

*要在更多设备上安装 RememBear，请在应用程序中选择“添加新设备”。您可以使用设备的摄像头安装，也可以键入“新设备密钥”只要你已经安装了移动应用程序并在那里登录，这将很容易。*

*一旦你在所有设备上安装了该应用程序，它将无缝同步。如果你已经用 Chrome 的内置同步或 Apple Keychain 保存了密码，你可以将它们导入 RememBear。(然后把它们从 Chrome 和 Keychain 上删除，这两个工具的安全性不如第三方密码管理器。)*

*从现在开始，每当你登录一个新的应用程序，你可以在 RememBear 中保存你的密码。每当你创建一个新的登录名时，不要编造你自己的脆弱的密码。取而代之的是使用它的密码生成器来创建一个更长、更安全的密码，你永远也不需要去记忆甚至打字。这就是密码管理器的全部意义。*
