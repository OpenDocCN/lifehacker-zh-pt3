# Chrome Canary 中最酷的功能

> 原文：<https://lifehacker.com/the-coolest-features-to-try-in-chrome-canary-1827998753>

谷歌 Chrome 的标准版或“ [稳定版](https://www.chromestatus.com/features/schedule) ”提供了大量方便的定制和生活质量改善，但如果你想在其他人之前获得最新的实验功能，你会想看看 [Chrome 金丝雀](https://www.google.com/chrome/canary/) 。



谷歌将 Chrome Canary 描述为其浏览器的“出血边缘”版本，它包含了新的调整和工具，你可以在正式发布之前尝试一下——如果它们甚至可以成为浏览器的稳定版本。注意:过金丝雀般的生活可能会导致不稳定的性能，因为浏览器仍在积极开发中，但你可以同时在系统上保留金丝雀和稳定版本的 Chrome。

为了启用 Chrome Canary 的可选功能，首先 [下载浏览器](https://www.google.com/chrome/canary/) 并转到网址 chrome://flags/。从这里，你会看到一个巨大的功能列表——其中一些是实验性的，而其他的则在谷歌 Chrome 的标准版本中可用。以下是一些更有趣的 Chrome 金丝雀功能，你可以用它们来提升你的浏览体验:

### 重磅页面封顶

在点击一个网页之前，你不知道它会使用多少数据，但启用了大量页面封顶功能(以及 Chrome 的数据保护程序)，每当页面超过某个阈值时，你都会收到通知。启用“低”时，这可以小到 1MB，并且您可以通过单击停止所有未来的加载。你访问的大多数网页，尤其是那些带有图片或视频的网页，将远远超过这个阈值。一旦您选择停止加载未来的数据，任何尚未加载的图像将不会出现。

### 选项卡自定义

如果你发现 Chrome 中的标签过于简单，或者你不喜欢“新标签”按钮，你可以在 Chrome Canary 中解决这两个问题。启用“新标签按钮位置”以将新标签按钮移动到新位置，包括标签左侧、标签右侧或标签之后的位置。这完全取决于个人偏好，尽管我们发现把它放在标签后面更容易跟踪它们。通过打开“单标签模式”,你可以给只有一个标签的浏览器窗口一种微妙的淡入淡出效果，就好像页面上根本没有标签一样。

### 游戏手柄振动

谷歌 Chrome 可能不是你玩视频游戏的首选平台，但偶尔你会发现自己在玩一款需要控制器的基于浏览器的游戏。启用“游戏手柄振动”选项后，控制器上的隆隆声功能将在 Chrome 中工作。你在专用应用中用于游戏的控制器现在应该在玩浏览器内游戏时发出隆隆声。支持的控制器列表尚未提供，但为了测试您的控制器是否正常工作，您可以尝试 Construct 的 [游戏手柄加工具](http://gamepadplus.surge.sh/) 。

### 使用所有即将推出的用户界面功能

如果你想访问最新的用户界面变化 — 最近的功能包括圆形标签取代了硬角 — 一旦它们被添加到 Chrome Canary，启用“使用所有即将到来的用户界面功能”这将让你看到 Chrome 的 [未来版本会是什么样子，让你经历讨厌它的阶段，然后接受它，而不必永久使用它。如果图标或标签有任何变化，您将首先在这里看到它们。](https://www.ghacks.net/2018/04/23/first-look-at-google-chromes-ui-design-refresh/)

## 通过不安全的连接阻止不安全的下载

为了确保您的浏览器和系统在使用 Chrome Canary 时完全安全，您可以启用“阻止不安全连接上的不安全下载”Chrome 应该已经阻止了其他安全网站上的类似下载，但启用该功能将阻止任何在 Chrome Canary 中利用重定向链中任何一点的不安全来源的下载。如果你发现这使得下载一个必要的文件变得不可能，你总是可以暂时禁用它或者加载 Chrome 的稳定版本来完成你的下载。
