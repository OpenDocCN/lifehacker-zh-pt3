# 彩色玛格丽塔酒:现在彩虹食品也能让你喝醉

> 原文：<https://gizmodo.com/technicolor-margaritas-now-rainbow-food-can-get-you-dr-1774911681>

因为这样或那样的原因，彩虹色的食物正在成为一种趋势。就在我们的 [拿铁](http://mentalfloss.com/article/78780/rainbow-lattes-are-latest-thing-coffee-art) 里。就在我们的 [百吉饼](http://www.thebagelstoreonline.com/) 里。甚至我们的 [披萨](http://www.neatorama.com/2016/04/19/rainbow-pizza/) 也不安全。我甚至做了一些 [研究](http://gizmodo.com/why-are-we-turning-our-favorite-foods-into-rainbow-colo-1772070100#_ga=1.114092167.1204971313.1461002213) 为什么我们一直吃这些东西(颜色可能会欺骗我们的大脑去品尝不存在的味道)。现在你可以一边系上一个一边赶潮流，我们有一位专家向我们展示如何在五月五日节及时完成。

欢迎来到 Gizmodo 的欢乐时光。书呆子滥用药物。

雷尔·佩蒂特是个非常好的酒保。他在 Mulberry Project 和 Tribeca Grand Hotel 卖饮料，但主要是因为他在食用染料和模版方面的工作而出名。想要一杯切·格瓦拉酸味的 还是一杯上面浮着卡尔顿·班克狞笑的脸的 sloe 杜松子酒？他可以为你做到这一点，使用一些精心裁制的纸和这些奇怪的小喷雾瓶，在阳光下装满各种颜色——尽管你可能会用一个普通的喷雾瓶。不要担心——彩色玛格丽塔不需要佩蒂特的 X-acto 灵巧水平或任何非常罕见的成分(见下面的视频，了解制作方法)。

制成的鸡尾酒独特美丽，绝对美味——即使对我这样不太喜欢龙舌兰酒的人来说也是如此。由于蛋清的缘故，口感很浓，比普通的玛格丽塔酒更美味，后者尝起来像掺有地板清洁剂的陈年雪碧。玫瑰水为花香型食用染料图案提供了一个很好的感官搭配，墨西哥胡椒油添加了恰到好处的收尾。

根据研究，我的大脑是否根据颜色想象了更丰富、更甜美或更有趣的味道？很难说，因为我从没试过没有食用色素的食谱。不过我可以肯定的是，这种染料绝对会沾满你的口腔，并在那里停留一整夜。如果不是那么美味的话，这几乎是一种恶作剧鸡尾酒。

大多数人现在都知道，五月五日节几乎与墨西哥的独立无关，在那里也很少庆祝。大多数人也知道玛格丽塔酒的原产地可能不是墨西哥。但是在某一点上，你喊着“实际上”进入了一股不可阻挡的酒精潮流——如果你今天要喝一些严重的龙舌兰酒，你还不如喝点好的。
