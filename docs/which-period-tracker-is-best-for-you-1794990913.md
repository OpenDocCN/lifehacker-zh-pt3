# 月经追踪摊牌:哪个时期追踪器最适合你？

> 原文:[https://life hacker . com/which-period-tracker-is-best-for-you-1794990913](https://lifehacker.com/which-period-tracker-is-best-for-you-1794990913)

有这么多不同的时期跟踪应用程序，因为有这么多不同的理由使用一个程序:也许你想留意生育期 [，这样你就可以怀孕](https://lifehacker.com/the-pros-and-cons-of-using-a-fertility-app-to-try-to-ge-1785341014) ，或者也许你想每天跟踪你的情绪和症状，这样你就可以在你的周期同步时发现你是否头痛或感觉暴躁。所以我们今天不会宣布赢家，而是告诉你在我们看来，三个最好的时期追踪器的利与弊。

### 线索

**亲:肯定不是粉色。无论在哪里，女人们聚在一起讨论月经追踪器——这比你想象的要频繁得多——线索通常是欢呼雀跃的一方。它最受欢迎的特征:*它不是粉红色的*。线索小组 [包括跨性别男性](https://www.theguardian.com/careers/2017/jan/24/periods-no-pink-no-flowers-just-science-clue-ida-tin-fertility-tracking) ，所以他们很清楚，不是每个需要跟踪月经的人都愿意接触他们刻板的女性化一面。然而，性的图标是一张肌肉发达的男人的照片，所以仍然有一些异质的假设存在。** 

Pro:快速查看你在周期中的位置。轻点中央按钮，输入当天的数据(你在流血吗？你有什么想记录的症状或性别吗？)或者只是看一下圆形图像，它显示了你相对于上一次月经、下一次月经和“生育窗口”的位置，这是一个月中性行为最有可能导致怀孕的时间。如果你喜欢传统的日历视图，只需轻点鼠标。

反对:很少有选项来描述正在发生的事情。你可以把事情分成几类，从你是否疼痛到你是否在锻炼或做爱。这很方便。但是和我交谈过的 Clue 用户说，他们希望每个部分有更多的选择。例如，在“疼痛”下，您可以选择痉挛、头痛、排卵或乳房触痛。但这款应用并没有给你一种方法来区分轻度痉挛和让你整天蜷缩在沙发上的那种痉挛。

**鲜为人知的特点:**由于 Clue 根据你过去的数据预测你的下一次月经和你的生育窗口，一个异常长的周期(例如怀孕，或者只是忘记记录几个月)会打乱预测。转到分析屏幕，点击超长周期，然后点击“从 Clue 的计算中排除此周期数据”

### 史普顿

**Pro:由计划生育协会制作。**这款应用非常友好，非常有教育意义，感觉就像一个积极的拥抱。有一个方便的常见问题部分，回答关于月经、节育等问题。如果你有需要专业知识的担忧，你可以从应用程序中找到计划生育健康中心。

Pro:追踪你的避孕措施。如果你服用避孕药，SpotOn 会每天询问你是否服药。这个应用程序最聪明的特点是当你搞砸的时候会发生什么。它会告诉你如果你忘记吃药该怎么办，如果你错过了足够多的药片，你可能会再次排卵，它会警告你今天可能是一个生育日。(该应用程序还会提示你设置每日提醒，这样你就不会再次忘记。)

Pro:只显示对你有意义的生育预测。如果使用宫内节育器，就看不到生育的日子了。不管你用什么方法，只有你选择了它，你才能看到生育信息。

反对:不跟踪避孕套的使用。避孕部分(这是你个人资料的一部分)或“行动”部分(你可以在这里标记你今天发生过性行为)不提供避孕套。SpotOn 的日常症状和活动选项甚至比 Clue 更少，并且没有办法添加额外的选项。

### 周期跟踪器(又名 P 跟踪器)

专业:追踪一大堆东西。有一个巨大的症状列表可供选择(痤疮、腹胀、嗜盐等)，你可以添加自己的症状。每一种都给你三种选择，大致可以翻译成轻度、中度和重度。您还可以输入自由格式的注释，并跟踪药物和情绪。如果你想了解你的生育能力，比如追踪你的每日体温，P Tracker 也有这些内置功能。(你需要豪华版的。)

**亲:懂怀孕。**如果你怀孕了，豪华版有一个选项，可以从跟踪经期切换到跟踪你怀孕多少周。

缺点:愚蠢的视觉效果。希望你喜欢热粉色和水绿色！确切地说，这款应用的设计还不错，但它没有像 Clue 和 SpotOn 那样时尚的外观或简单的界面。主屏幕图标是一朵卡通粉色的花，尽管它自称为“P 追踪者”，目的是保持低调。
