# 花时间在家门口创造持久的邻里关系

> 原文：<https://lifehacker.com/spend-time-in-front-of-your-house-to-create-lasting-nei-1818812570>

"人们什么时候会来？"当我们凝视着外面空荡荡的街道时，我四岁的女儿问我。我们坐在外面车道上的摇椅上。我觉得很可笑。

“我不知道是否有人会来，”我告诉她。“但是我们可以在这里玩。”

今年早些时候，我读到了关于 [前院人](http://www.kristinschell.com/front-yard-people-your-tables/) 的报道，这是一场旨在让邻居回到他们家门前而不是关起门来生活的运动。它始于一位名叫克里斯汀·谢尔(Kristin Schell)的女性，她是一位全职妈妈，在奥斯汀的社区里感到有点孤立，所以她在自家前草坪上放了一张蓝绿色的野餐桌，拿出一杯咖啡和她的笔记本电脑，等着看是否有人会过来聊天。他们做到了，一个接一个的邻居。这张桌子成了一个聚会的地方(“有点像古老的村庄，”她写道)，激发她写了一本关于这一经历的书，名为 [*《绿松石桌子:在自己的前院寻找社区和联系*](https://www.amazon.com/Turquoise-Table-Finding-Community-Connection/dp/0718095588?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/spend-time-in-front-of-your-house-to-create-lasting-nei-1818812570&asc_source=&tag=kinjalifehackerlink-20) *。*

这个主意听起来不错。自从为人父母后，我变得不太爱活动，时间也少了。朋友们分散在不同的城市——我们试图在可能的时候见面，但这需要一串来回的短信，在谷歌日历上搜索三个小时的街区，并在交通中开车。我讨厌交通。我一直在想，我们需要有人在这里，在我们在的地方，在我们的生活可以以最小的努力融合在一起的地方，在我们的孩子可以感到安全和舒适的地方。但是我们的邻居很安静。我渴望那个古老的村庄。我想成为一个前院的人。

我告诉我的丈夫和女儿，有一天晚上我们要去前院逛逛，见见邻居，但是我把这个任务推迟了一会儿。我意识到在一天工作结束和哄孩子睡觉之间的几个小时里，我已经筋疲力尽了。我脱下胸罩，减压，体内没有内德·弗兰德斯的快乐。我很高兴看到朋友，但我们必须能够一起出去。认识新朋友听起来很难。谢尔承认确实如此。她说 [启动项目](http://www.mystatesman.com/lifestyles/home--garden/there-always-spot-the-turquoise-table/fxujGpeP32bMVTd9IgNE4K/) 当你焦虑地想，“有人会来参加我的派对吗？”时，你会感觉像回到了中学时代。他们会喜欢我吗？”然而，做一个“前院人”的好处在于，你不必“主持”任何事情，也不必邀请人们(喘气)*进入你的家。你只是在你家前院的桌子上(或者任何人们聚集的地方，比如公寓屋顶)做你通常会做的事情——吃晚饭，做作业，写作，喝杯酒。*

所以最后，我们做到了。昨天下班后，我让我丈夫帮我把我们的野餐桌从后院搬到前院，但结果它太重了，无法通过大门(伟大的规划，我)。所以我们搬出四把摇椅和一张边桌，把它们放在车道上。我记得谢尔建议提供一个停下来的理由——她有时会拿出一个咖啡壶，一碗给狗喝的淡水和一些游戏，人行道上的粉笔和给孩子们的拼图。我试着快速思考，然后跑进去，把一些冰棍放在冰柜里。我也擦了一些唇彩和除臭剂。

然后，我的丈夫、女儿和我坐在椅子上，吃着火鸡三明治。我们等了又等。几辆汽车从我们身边疾驰而过。戴耳机的跑步者也是如此。没有人朝我们这边看。

“那么...我们还要在这里待多久？”我老公问。

“直到洗澡时间，”我说。那是两个半小时的路程。

“我想你的冰棍要融化了，”他回答道。

我的女儿变得焦躁不安。“你给谁发短信了吗？”她问。

“嗯，我应该，”我说。我给我认识的几个有孩子的邻居发了信息，但他们回信说他们有足球训练和其他活动。我已经猜到了。

我告诉自己，好好生活吧。我拿出一些我女儿可以玩的玩具，我们一起玩。我们坐在一桶水旁，看着小海绵恐龙从胶囊中孵化出来。她唱迪士尼公主的歌。我和老公喝了酒。一个带着两条狗的男人从我们身边走过，说:“你好！这是一个美丽的夜晚！”我们挥挥手说:“就是！”不管我们是否创造了持久的邻里关系，它是。

当天渐渐黑下来，我们正准备收拾东西进屋时，我们听到一家人走过。是我们有一段时间没见的邻居。“嗨伙计们！”我有点太兴奋尖叫。我告诉他们我的小小社会实验，并给孩子们提供冰棍(有点融化，但仍然完好无损)。他们呆了一会儿，我们聊了聊。然后他们问我们是否想在街区周围散散步。我们做到了。孩子们上了滑板车，我们打开了手电筒。我们谈论工作和生活，当孩子们差点从一只死老鼠身上滚过去时，我们尖叫起来。这一天很有趣，也是一个美好的结局，如果我们不在那里，这一切都不会发生。也许这是一个开始。

我开始渴望地看着我们房子前面的一片死草地。一个永久的小桌子可能会很好。

谢尔提供了一些如何成为前院人的建议:

考虑天气和天气因素。有些日子和季节是行不通的，所以要根据天气提前做好计划。如果可以的话，把桌子放在阴凉的地方，准备好一篮子防晒霜和杀虫剂。

如果你刚刚开始，做广告。谢尔建议通过举办活动让你的朋友和邻居知道你的存在。你可以在 NextDoor.com 上张贴公告，甚至制作传单。

定期去那里逛逛。带些事情去做，但不要做那些需要全神贯注的事情。在这里，你想被打断。
