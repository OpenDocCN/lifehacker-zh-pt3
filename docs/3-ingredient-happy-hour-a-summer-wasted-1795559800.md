# 三要素快乐时光:浪费的夏天

> 原文：<https://lifehacker.com/3-ingredient-happy-hour-a-summer-wasted-1795559800>

周末快乐，欢迎回到 [*三料欢乐时光*](http://lifehacker.com/tag/3-ingredient-happy-hour) ，每周饮品专栏，提供超级简单又美味的酒水。最近，我一直在享受一种叫做“皮斯科”的南美白兰地的魅力，所以我想我们可以尝试一种夏天的，但仍然很烈的皮斯科鸡尾酒。



这种令人愉快的葡萄白兰地是从葡萄酒中蒸馏出来的，但是——正如许多类型的酒精饮料及其分类一样——对它进行进一步分类会变得有点复杂。 [食者](https://www.eater.com/2015/10/20/9532749/classic-pisco-cocktail-recipes) 的这篇文章很好地充分分解了它，如果你愿意，你应该读一读，但我想去喝酒了。我最近一直在享受[*Pisco Logia*](http://piscologia.com/)，一种秘鲁白白兰地，带有“橙花、山核桃、成熟干醋栗和香蕉的细微差别”

为了制作一种复杂(且僵硬)但仍令人耳目一新的鸡尾酒，我从纽约雷米的 [*埃尔卡皮坦*](https://www.eater.com/2015/10/20/9532749/classic-pisco-cocktail-recipes)中获得灵感，并将皮斯科与我夏天最喜欢的另一种鸡尾酒[*Cocchi Americano*](https://en.wikipedia.org/wiki/Cocchi_Americano)结合起来。如果你从未享受过这种乐趣， *Cocchi* 是一款意大利开胃酒，由于金鸡纳树皮的缘故，它具有清爽的柑橘味和克制的苦味。接下来，我加入了一些葡萄柚苦味剂——而不是 ango stura——来保持整个柑橘类食物的味道。然后我用一个美女的名字给它命名，因为我是个酒鬼，如果你整个夏天都喝很多，你会浪费掉的。

要制作这款美味的葡萄酒，你需要:

*   2 盎司皮斯科
*   1 盎司可可美式咖啡
*   2 份葡萄柚苦味酒

将所有东西倒入混合杯(或一品脱玻璃杯)，加入冰块，搅拌大约 20 秒，滤入一个小酒杯。如果你想变得非常别致——从技术上增加你的配料数量，这取决于你如何看待装饰——你可以在整个场景中挤一条柠檬皮，然后让它漂浮在你美味的饮料中。这是可选的，但不是一个坏的选择。
