# 你在每台新电脑上都安装了哪些实用程序？

> 原文:[https://life hacker . com/what-utility-apps-do-you-install-on-every-new-computer-1820316436](https://lifehacker.com/what-utility-apps-do-you-install-on-every-new-computer-1820316436)

有几个大家都知道的程序可以添加到新电脑上:Chrome、Spotify、Dropbox，以及一些工作应用程序，如 Excel 或 Photoshop。但我们中的许多人也有自己喜欢的助手应用程序和实用程序来定制我们的工作流程。

Watch

我有一整套助手应用，比如 [自由](https://freedom.to/) ，用于在我需要工作时屏蔽社交网站； [轰](http://www.globaldelight.com/boom/index.php) ，用于在笔记本电脑上调高音量；还有 [调酒师](https://www.macbartender.com/) ，一个整理我所有菜单栏 app 的菜单栏 app。在新电脑上的头几周，我最终把它们都加了进去。

但是没有 [键盘大师](https://www.keyboardmaestro.com/main/) 我活不过一天，这是一个强大的 Mac 自动机，可以将任务分配给键盘快捷键。我有很多文本工作流，我使用键盘大师填写表格，开始搜索，在 Slack 中调试我的编辑器，只需敲几下键盘就可以回复可预测的电子邮件。我选择的快捷方式就像剪切和粘贴一样彻底地印在了我的脑子里。
