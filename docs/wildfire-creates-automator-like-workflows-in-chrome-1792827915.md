# Wildfire 在 Chrome 中创建类似 Automator 的工作流程

> 原文：<https://lifehacker.com/wildfire-creates-automator-like-workflows-in-chrome-1792827915>

Automator 是在 Mac 上自动化任务的最简单的 [方式之一，记录鼠标动作的能力使任何人都可以在几秒钟内创建自己的工作流程。Wildfire 是 Chrome 的扩展，试图为 Chrome 带来类似的功能。](http://lifehacker.com/automate-just-about-anything-on-your-mac-no-coding-req-5668648) 



Wildfire 本质上允许你创建完全基于 Chrome 的小宏。点击记录按钮，Wildfire 会记录下您所做的一切，最终将其转化为工作流程。这些操作可以包括鼠标点击、改变标签、创建标签、滚动页面、输入数据等等。这有多有用完全取决于你在浏览器上花了多少时间，以及你用这些时间做了什么。无论如何，这是一个简洁的工具，对于那些做大量重复性工作的人来说可能是有用的。
