# 如何在公共厕所如厕训练中幸存下来

***关于在旅途中和公共浴室如厕训练，你都做了些什么？我是一个彻头彻尾的细菌恐惧症患者，我们现在处于这样一个阶段，当我们出去的时候，我 2 岁的女儿会要求去，我只是吓坏了。我带着一次性马桶盖、折叠马桶座、来苏湿巾、洗手液，但是伙计，这太多了(我们住在纽约，所以所有这些现在都在我的钱包里，我的背会断的)。对此能做些什么？【T2***

我要说的是，这是一个试图解决的棘手问题:这里基本上有两个问题，第一个是，“我能管理好所有这些*东西*吗？”第二个问题是，“我怎样才能让公共厕所不那么恶心？”这两个问题我都没有很好的答案！但我有答案，即使不是最好的。

让我们从材料开始:我认为这是应该有的材料。唯一可以商量的是折叠马桶座(和我一起呆在这里)，但这很大程度上取决于你的孩子，以及她对上厕所的整个过程有多害怕。从战术的角度来看，你可以让她侧身坐在座位上，这样可以防止她摔倒，但不是所有的孩子都会允许这样做，显然你不想让你可怜的孩子坐在大人的座位上，如果他们害怕这样做的话。如果您的孩子确实需要便携式座椅，有比其他型号更轻更紧凑的型号， [像这个](https://www.amazon.com/Gimars-Silicone-Portable-Reusable-Training/dp/B06ZZXQHYK?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-to-survive-potty-training-in-public-restrooms-as-a-1821387910&asc_source=&tag=kinjalifehackerlink-20) ，可能有助于减轻您背部的负担。

其余的东西——来苏湿巾、洗手液和座套——几乎没得商量，尤其是考虑到妈妈的细菌恐惧症。幸运的是，这些东西并不太重或太大，特别是如果你换成袋装的来苏湿巾，而不是罐装的。考虑到这一点，就目前而言，换回使用尿布包可能是个好主意，因为它会更宽敞，而且旨在分散其内容物的重量。

至于把公共浴室作为一个细菌恐惧症来处理，我不会试图说服你摆脱你的细菌恐惧症，但有些事情你可以试着控制你的焦虑。用来苏消毒剂擦拭马桶座圈，放下座套，用一张卫生纸或纸巾盖住冲水器、门锁和把手以及水龙头。现在，这些事情都没有必要了——上完厕所后洗手足以清除你可能接触到的任何细菌，但我也不是来告诉你如何摆脱恐惧的。

说到洗手！这是教孩子养成良好洗手习惯的好机会，也是温习自己洗手技巧的好机会。所以！作为提醒，疾病预防控制中心建议在冲洗和干燥之前，将手放在水下，用肥皂起泡，擦洗手的前后以及手指之间和指甲下面 20 秒钟。

这是一个向小组成员抛出的好话题:你有什么在旅途中训练如厕的技巧和诀窍？有哪些产品是你离不开的？你克服了细菌恐惧症吗？如果克服了，你是怎么做到的？还有，你推荐什么便携式便盆座？《后代》的编辑米歇尔·吴提到了，她说当你在一个很难找到卫生间的地方——或者特别恶心的地方——这就派上了用场。
