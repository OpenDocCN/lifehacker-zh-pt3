# 三种成分的快乐时光:朗姆酒曼哈顿

> 原文：<https://lifehacker.com/3-ingredient-happy-hour-the-rum-manhattan-1823463062>

各位周末快乐，欢迎回到 [*三料欢乐时光*](http://lifehacker.com/tag/3-ingredient-happy-hour) ，每周饮品专栏，提供超级简单又美味的酒水。本周，我们用朗姆酒代替标准的黑麦威士忌，在标志性的曼哈顿酒的基础上混合了一种变化。

Watch

很明显，我并不反对威士忌，但有时我只是想用一种更热带的精神稍微放松一下，在曼哈顿用朗姆酒让你放松一下，而不用喝“满满的代基里酒”要制作一个，你只需要:

*   2 盎司朗姆酒(最好是单桶或陈年的)
*   1 盎司甜苦艾酒(如果你想犒劳自己的话，可以选择 Carpano Antica)
*   少量的芳香苦味酒

将所有原料倒入加冰的混合杯，搅拌至完全冷却。滤入一个小酒杯，如果你敢增加你的配料数量，用一个橙子装饰，啜饮一口略带热带风味，但仍然非常严肃的酒。
