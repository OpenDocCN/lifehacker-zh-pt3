这是个别专家对一个对每个人都有不同影响的问题的一般看法——如果你想要个性化的建议，你应该去见理财规划师。

### 在短期和长期需求之间取得平衡

你把你的财务管理得井井有条，真了不起！听起来你的处境很好。

根据你提供的信息，我采访的顾问同意你可以在长期退休和短期流动性需求之间取得平衡，这一点特别重要，因为你是自由职业者，可能会有更多的困难时期。

第一:记住你可以取出供款(不是收益)到你的 [罗斯个人退休帐户](https://twocents.lifehacker.com/why-you-need-a-roth-ira-1823805955) 以备不时之需，这样就有了一些流动性。尽管你说你的应急基金足够大，但如果你担心突发事件，那它可能不够大。所以考虑在那里多存一点钱来缓解你的焦虑。如果你有 1099 收入来自与你的工会合同无关的工作，那么考虑一个 Solo 401(k)或 SEP IRA。根据你的收入，你可以在 2018 年贡献高达 55，000 美元(然而，在紧急情况下，你需要贷款)。

对你来说，一个更好的选择可能是一个应税经纪账户——仅仅因为它没有被贴上退休账户的标签，并不意味着你不能这样使用它。你可以想贡献多少就贡献多少，想什么时候取出钱就什么时候取出——只要知道每年赚的股息、利息和资本利得都是要交税的。

“这不是一个完美的解决方案，如果他能与公司和工会合作，允许向他的 401(k)计划捐款，那将是理想的，”印第安纳州国际金融理财师 Mychal Eagleson 说。“但是，如果这种选择仍然不可行，经纪账户将允许他继续为退休储蓄和投资。如果需要的话，只要他投资于流动性不受限制的基金，就能轻松获得所需的资金，而流动性受到限制是很容易避免的。

持有你投资的股票或共同基金至少一年，当你出售时，你只需支付长期资本利得税，而不是基金/股票的短期税率(长期资本利得税上限为 20%，而短期税率是基于你的普通收入)。“如果这些资金以节税的方式进行投资，并且不经常交易，每年的税收影响应该可以忽略不计，”麦智信说，他是加州的国际金融理财师人。

俄亥俄州国际金融理财师的莫尼卡·德怀尔说:“如此多的人害怕纳税，以至于他们不认为投资税后经纪账户是一种选择，但根据目前的税法，我们很可能在未来八年内处于最低的税率之一，这是我们在很长一段时间内都不会预料到的。”。你可以在 TD Ameritrade 或者 Schwab 这样的折扣券商开一个，也可以在 Vanguard 或者 Fidelity 开一个，低成本投资各种基金。把这个账户和你的支票账户关联起来，你就可以自动缴款了。

另一个想法:你的健康保险怎么样？如果你有一个高免赔额计划，增加你对健康储蓄账户的贡献将会给你三倍的税收优惠，并且是一个特别为退休后的医疗费用储蓄的好方法。
