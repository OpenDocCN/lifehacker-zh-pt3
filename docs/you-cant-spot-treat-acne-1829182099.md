# 你不能当场治疗痤疮

> 原文:[https://lifehacker.com/you-cant-spot-treat-acne-1829182099](https://lifehacker.com/you-cant-spot-treat-acne-1829182099)

非处方的现场治疗看起来简直不可思议，声称能在几个小时内消除青春痘。事实证明，事情并不是这样的——如果你满脸都是痘痘，选择性地治疗几个痘痘充其量是一个非常低效的策略。

斑点疗法似乎足够可信:长一颗青春痘，敷一整夜的斑点疗法，第二天早上醒来就没有青春痘了，准备去参加派对。当然，在现实中，事情要复杂得多。从遗传到激素，再到你使用的护肤品种类，痤疮都会受到各种因素的影响，所以我咨询了一些专业人士来帮助解决这个问题。

### 如果你是当场治疗，已经太晚了

化妆品营销人员喜欢令人印象深刻的统计数据。这一点在痤疮治疗的营销文案中表现得最为明显，几乎所有的广告都声称，在一定时间内，或两者兼而有之，以不可思议的特定百分比缩小粉刺。只有一些人不厌其烦地引用内部的“”临床研究结果作为证明。

不出所料，这些研究故意忽略了一大块难题:可见的痤疮病变已经接近其生命周期的末期。如果你的工作是“证明”一种产品治愈了一个已经基本痊愈的损伤，你可以很容易地挑选数据来支持这一说法——特别是如果你不需要像严格的实验方法或知道相关性和因果关系之间的区别这样讨厌的事情。

误导性的营销术语，相互矛盾的互联网 anec 数据，以及适当的医疗护理的高壁垒，难怪很多人对痤疮治疗的工作原理有一些落后的想法。当我问李锦梅 博士关于局部治疗的常见误解时，她是一位在加州帕萨迪纳执业的注册皮肤科医生，她说:

> 最大的误解是，斑点疗法可以有效地治疗痤疮。现场治疗就是这样..他们会处理你涂抹的地方，甚至经常不能有效地处理这些地方。斑点疗法不能防止痤疮的形成，所以[它们]不能提供有效的长期治疗。

或者，用新泽西皮肤病学家 [丽贝卡·巴 xt](https://www.cosmedical.com/doctors/dermatologists/rebecca-baxt-md-mba-faad/) 博士的话说:“到你需要斑点治疗的时候，可以说猫已经从袋子里出来了。”

所以现场治疗完全是个骗局，对吗？从技术上来说，不是。事实证明，祛斑疗法中的成分可以减少炎症并杀死痤疮细菌，但它们需要在整个受影响区域持续应用数周或数月才能发挥功效。在一个小地方呆几个小时不会有什么作用。如果你想的话，你可以用局部疗法来治疗粉刺，但是一种为日常使用而设计的产品实际上更有可能预防粉刺。不管怎样，它们可能有相同的活性成分。

### 什么是现场治疗？

大多数非处方痤疮治疗，从软膏到面霜到清洁剂，都是为了解决粉刺堵塞或细菌成分而设计的。它们的活性成分通常分为四类:

*   **去角质剂**如水杨酸、乙醇酸或壬二酸，它们通过溶解堵塞毛孔的死皮细胞块而起作用；
*   **抗菌剂，**如过氧化苯甲酰和硫磺，通过杀死痤疮细菌来治疗炎症；
*   阿达帕林和视黄醇等类视黄醇，通过增加细胞更新来加速愈合过程和疏通毛孔；
*   类固醇如氢化可的松霜，可以减少炎症。

一些成分有多重作用——硫磺和壬二酸都具有抗菌和去角质的作用——但是因为痤疮在多余的皮脂上滋生，所以它们都可能会使你的皮肤干燥。注意不良反应:根据 Baxt 博士的说法，百分之五到百分之十的人对过氧化苯甲酰过敏。轻微发红和刺激是正常的，但如果你出现肿胀，痕，皮疹或荨麻疹，立即停止使用该产品。

我采访的皮肤科医生一致认为，虽然上述所有成分在某种程度上都有效，但去角质剂和/或类视黄醇对堵塞的毛孔更有效，发炎的痘痘对过氧化苯甲酰的反应最好。此外，无论广告文案怎么说，多种活性成分并不能使配方更有效。正如李博士所说:“没有强有力的证据支持使用组合产品进行斑点治疗，[而且]组合产品可能会导致更多的刺激和干燥。”奥兰多的皮肤科医生艾利森·阿瑟 同意这一观点，但指出了一个例外:如果需要，可以在局部治疗的基础上涂上一层氢化可的松软膏或乳膏。不过，总的来说，你最好不要扮演调酒师——或者花大量的钱在混合活性物质的“双重作用”产品上。

底线:现场治疗解决痤疮的症状，而不是原因。少量过氧化苯甲酰凝胶可能会清除偶尔出现的斑点，但如果你有反复出现的大面积皮疹，皮肤科医生是你最好的选择。
