# 双显示器散景桌面

> 原文：<https://lifehacker.com/the-dual-monitor-bokeh-desktop-1778011402>

在我们的 [桌面陈列柜](http://kinja.com/tag/desktop-showcase) 中，我们几乎没有足够的双显示器桌面，所以当我们看到一个时——尤其是像这样好看的——我们必须突出它。最重要的是，它没有太多的东西，如果你有两个自己的显示器，就很容易安装。以下是方法。

Watch

读者 [工具没有一个有趣的名字](http://kinja.com/twain2) 提交了这一个，并使用雨量计来完成这项工作。如果你对雨量计不熟悉，我们这里有 [的入门指南](https://lifehacker.com/how-to-create-an-attractive-customized-desktop-hud-wit-5828789) 来帮你。

一旦你记下了这些，或者想自己尝试一下，以下是你需要的:

*   [壁纸](http://www.dualmonitorbackgrounds.com/abstract/bokeh-6.jpg.php)
*   用于 Windows 的 [雨量计](http://www.rainmeter.net/) 系统调试和监控工具
*   [为雨量计的毛毛雨](https://rainmeterdesk.blogspot.com/2015/06/drizzle-v10.html?m=1) 为时间和日期效果
*   雨量计的 [VisBubble 皮肤](http://undefinist.deviantart.com/art/VisBubble-Round-Visualizer-for-Rainmeter-V3-0-488601501) 为可视化器一直到右显示屏上
*   正在播放显示的雨量计 [明文皮肤](http://redsaph.deviantart.com/art/Cleartext-for-Rainmeter-v3-0-1-19-Apr-2016-519796161)

这就是这个的全部内容！它简洁明了，为整洁的桌面增添了一点个性，否则就可以开始工作了。如果你喜欢它的样子，去下面链接的 Kinja 博客，让没有有趣名字的 Tool 知道你的想法，以及你喜欢它的结果。
