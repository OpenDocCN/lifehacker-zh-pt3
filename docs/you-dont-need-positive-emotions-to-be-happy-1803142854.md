# 快乐不需要积极的情绪

> 原文:[https://life hacker . com/you-done-need-positive-emotions-to-be-happy-1803142854](https://lifehacker.com/you-dont-need-positive-emotions-to-be-happy-1803142854)

根据一项新的研究，快乐的秘密不是让你的脑袋里一直塞满彩虹和独角兽。它倾向于与你的价值观一致的情绪——甚至是所谓的负面情绪。如果你能找出你最想要的感觉，并在这些感觉到来时尽情享受，你会过得更好。

这项研究发表在上个月的*实验心理学杂志上:一般* 。作者写道，关于幸福的传统研究表明，我们感受到的快乐情绪越多，我们就会越快乐。但是 [亚里士多德说](http://classics.mit.edu/Aristotle/nicomachaen.2.ii.html) 我们经历我们*想要*感受的情绪越多，我们就会越快乐。

作者决定对老亚里士多德进行测试，招募了来自八个国家的 2324 名参与者:美国、巴西、中国、德国、加纳、以色列、波兰和新加坡。这些国家代表了世界上各种各样的文化区域。

耶路撒冷希伯来大学情绪&自我调节实验室主任、首席研究作者 [玛雅·塔米尔](http://mayatamir.com/members.html) 在一次电子邮件采访中写道:“一直想要快乐或喜悦是不太现实的。“永远不想感受悲伤、愤怒或恐惧是不现实的。如果我们能够接受甚至欢迎我们所拥有的情绪，无论它们是愉快的还是不愉快的，我们可能会更快乐，更满意。”

参与者在纸上或网上填写调查问卷。首先，他们对自己想要的情绪进行排序，并指出他们想要体验这些情绪的频率。然后，他们花五分钟玩情感中立的文字游戏来净化精神味觉。最后，他们报告了他们在日常生活中感受到相同情绪的频率，等级为 1-5。

有些人珍视高兴奋度的愉快状态，比如兴奋，而有些人更喜欢低兴奋度的愉快情绪，比如平静。对他们来说，决定幸福的是激情和满足的区别。

然而，有些人看重自我提升的感觉，如权力和成就。他们挖掘骄傲——但也挖掘愤怒、仇恨和蔑视。对于那些乡亲们，一个好皇帝帕尔帕庭式的“ [”让仇恨流遍你全身！”](https://www.youtube.com/watch?v=pa5cOTQreok) 确实让他们更快乐。

塔米尔说:“为我们认为有价值和有意义的情感而奋斗是有益的，即使它们并不令人愉快。”。下一次交通让你看到红灯的时候，思考的食粮——也许这是你成为最好的、最疯狂的自己的计划的一部分。
