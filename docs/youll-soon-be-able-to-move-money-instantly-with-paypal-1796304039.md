# 你很快就可以用 PayPal 即时转账了

> 原文：<https://lifehacker.com/youll-soon-be-able-to-move-money-instantly-with-paypal-1796304039>

当你想到点对点支付时，你可能会想到贝宝。但是，Square Cash 等其他服务有一个很大的优势，那就是即时兑现的能力。 [PayPal 刚刚宣布](https://www.paypal.com/stories/us/transfer-money-instantly-to-your-bank-account-with-paypal) 他们也计划提供这一功能。



在公告中，该公司的执行副总裁比尔·雷迪说:

> 今天，人们使用 PayPal 的 P2P 服务，包括 Venmo 和 Xoom，每小时向朋友和家人发送和接收超过 700 万美元...美国的 PayPal 用户很快就可以通过与他们的 PayPal 账户相关联的合格借记卡向他们的银行账户转账。我们预计资金通常会在几分钟内到达您的银行账户，尽管有些银行可能需要 30 分钟。我们的测试版现已面向部分 PayPal 用户开放，并将在未来几周和几个月内面向所有持有合格 Visa 或 Mastercard 借记卡的美国 PayPal 用户开放。

每笔即时转账费用为 0.25 美元。通常情况下，转账是免费的，但也需要几天时间，在我们这个快节奏的实时世界里，这太长了。

像 Square Cash 或即将推出的 [Zelle](https://www.zellepay.com/) 这样的服务已经提供了这种优势。 [许多大银行](https://www.zellepay.com/partners)——仅举几个例子，大通银行、联合银行和富国银行——已经与 Zelle 合作，为他们的客户提供实时点对点支付。因此，如果你是 Ally 的客户，你可以在他们的应用程序中使用这项服务，或者直接从 Zelle 应用程序中使用。据《纽约时报》报道，银行可以通过 Zelle 设定自己的费用，但“但没有一家银行宣布计划向客户收取银行间转账费用”。

Paypal 表示，除了一些较小的机构，大多数持卡人都可以使用 PayPal 和 Venmo 的新功能。你不能选择进入测试版，但如果你是它的一部分，你现在可以通过移动或网络应用程序看到这个选项。Venmo 将随后推出自己的测试版，然后美国的所有 PayPal 和 Venmo 客户将在未来几周内访问，PayPal 称 [为](https://techcrunch.com/2017/06/20/instant-bank-transfers-are-coming-to-paypal-and-venmo/)。
