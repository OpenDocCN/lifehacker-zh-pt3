# 不是每个人都适合提前退休

> 原文：<https://lifehacker.com/early-retirement-is-not-for-everyone-1792176454>

我今天和我的一个老朋友吃了一顿丰盛的午餐。我们曾经一起工作，他比我大几十岁，所以今天他实际上开始考虑短期退休。他对我作为一个靠兼职和合同为生的自由职业者的生活很感兴趣，我也分享了我对此的一些想法，但当我们开始真正的谈话时，他似乎主要是在试图弄清楚他生活的下一阶段会是什么样子。



*本帖最初出现在* [*的简单美元*](http://www.thesimpledollar.com/what-kind-of-retirement-is-right-for-you/) *上。*

他离传统的退休还有五到十年的时间。他似乎热爱他的工作。他在工作之外的主要兴趣是去看现场体育赛事，尤其是他附近的高中和大学体育赛事(在爱荷华州没有太多专业的体育赛事可供选择)。

我问他典型的一天是什么样的，大多数日子似乎要么是去上班，回家做家务，去上班，然后去看体育赛事，要么是用家务和体育赛事填满周末。

“但是...你*还做什么*？”我问他。我不是有意无礼的。相反，我是在对一个老朋友坦白。我试图帮助他弄清楚，在退休后，他可以用 T2 的大量额外时间做些什么。

问题是，他对那个问题一无所知。他谈到浏览网站，也许偶尔读一本书，看电视...仅此而已。

我的回应？我告诉他,**如果他现在真的对自己的日常生活感到满意，对他多年来赢得的工作责任和工作场所的尊重以及那些工作场所的关系感到满意，并且他观看现场体育比赛的主要爱好也符合这一点，那么他没有任何理由退休，直到他*有*可以退休。**

有趣的是。他想了大约一分钟左右，然后慢慢变得开朗起来。

对他来说，退休一点也不像是一件幸事。在他生命的最后二三十年里，他似乎在这个时空空洞里无所事事，无所事事，最终死去。他看到他非常重视的工作关系*，正在枯萎和死亡。他看到自己离开了一个他受重视并受到尊重的地方，进入了一个没有尊重的生活。*

*我不禁想起了杰克·尼科尔森(Jack Nicholson)主演的关于施密特 的电影 [*，他是一位正在为退休而挣扎的最近退休的人。他在电影的第一部分讲述了他的生活节奏，但感到非常空虚；这部电影很好地抓住了这种感觉。*](https://www.youtube.com/watch?v=G-dafG40eGU)*

*很明显，沃伦·施密特关心的是他在工作场所的尊重和价值，他在那里的关系，以及他所做的工作。这是他为人的一个非常关键的部分，也是他享受生活的原因。当然，像其他人一样，他并不热爱他工作的每一个方面，但这份工作给他的比他意识到的要多得多。*

*这并不是一件坏事。每个人都需要找到让他们感到完整，让他们感到有价值，让他们感到有参与感的地方和事物。很多人发现在工作场所。*

*另一方面，还有我。我一生中想做的事情比我一生中有时间做的事情还要多。写书。看书。徒步旅行。慈善工作。桌面游戏设计。园艺。钓鱼。政治牵连。旅行。尝试新事物。很多很多事情。*

*对我来说，虽然我很喜欢我的工作，但我也非常期待生活中的某一天，当我醒来时，我会知道我的一天将由所有这些事情组成，我将有足够的时间来有规律地做这些事情。*

*我和我的朋友从不同的地方获得个人的快乐和幸福。他的大部分快乐来自他的工作和他的一个主要爱好/兴趣。如果你拿走了这些东西中的一件，那么他的生活中就会有一个空缺没有填补。是的，他*可以*找到填补它的方法，但是*如果他已经有了自己觉得充实的生活，他为什么要去做呢？他有他重视的关系，重视他的人，做给他带来快乐和满足的事情，一个人在生活中还想要什么？**

*我的快乐来自于广泛的兴趣和激情，远远超过我目前的时间。我对我现在用时间做的事情很满意，但是如果一个元素消失了，我会很快用其他东西来填补这个空白。*

*简而言之，我有更强烈的动机为退休存钱，如果可能的话早点退休，而我的朋友也有同样的动机。我的朋友没有强烈的理由提前退休，甚至“按时”退休对他来说，尽可能长时间地工作并保留生活中他所珍视的所有方面更有意义。*

*退休储蓄和退休规划的目标不一定是退休。当你到达退休年龄时，你会有很多选择。坚持目前的职业应该只是众多选择中的一个。当这是最好的选择时，就像我的朋友一样，那么你的退休储蓄只是一个备用计划。当这只是众多选择中的一个，而其他一些选择听起来非常好，就像我一样，那么你的退休储蓄就非常非常重要。*

*你在这个天平上处于什么位置？*

*你是那种建立了伟大事业和大量良好关系的人吗？在你的职业生涯中，你是否找到了一个让你获得尊重、自我实现、身边有很多好人的地方？你想知道职业生涯结束后你会做些什么来打发日子吗？如果那是你，它仍然值得为退休储蓄，但是你可能希望工作的时间比典型的退休年龄长一点，你应该把它看作是一个备用计划。退休后你不会存很多年的钱，你有更多的时间来存钱。退休储蓄对你来说并不那么重要。*

*或者也许你是那种退休后迫不及待地尝试新事物和新想法的人。一旦你达到一个不再需要为收入而工作的点，你有一个很长很长的清单，上面列着你想要实现的事情。你退休的日子看起来绝对令人兴奋，因为你将要做的所有事情。你不一定讨厌你的工作，但你绝对是为了生活而工作，而不是为了工作而生活。我的朋友，你是那种应该尽可能积极地为退休存钱的人，这样你就可以尽可能早地退休，拥有充分享受退休生活所需的资源。提前退休并不意味着四十年的苦差事，它意味着四十年的无限可能。*

*这两种不同的态度直接指向不同的退休方式。一个是积极储蓄，最大化 401(k)和个人退休账户的贡献，寻找漏洞和角度来榨干退休储蓄的每一滴。另一种更注重把退休作为退路，更注重无债务的生活方式，以便在工作的同时提高自己的财务灵活性。*

*没有一个是绝对正确的，也没有一个是绝对错误的。相反，它们都是不同的生活方式如何促使人们对自己的财务和退休计划采取不同方式的例子。*

*这就是为什么它被称为“个人”金融。不同的人有不同的愿望、目标、抱负和态度，让每个人都遵循相同的道路是没有意义的。*

*对每个人来说，个人理财的最佳方法是从弄清楚你想从生活中得到什么开始，并做出选择，最大化你想要的东西。对一些人来说，这可能会引导你快速退休储蓄。对其他人来说，这可能是为了攒钱买房。对另外一些人来说，完全免除债务可能是最好的途径。没有永远正确的答案。我能指出的对几乎所有人都始终有效的唯一原则是*花得比赚得少*，因为未来总是不确定的，用你的收入和支出之间的差额做一些有价值的事情。还清债务，为目标存钱，不管是什么。*

*祝你好运。*
