# 使用此应用程序管理和取消订阅您的所有在线帐户

> 原文：<https://lifehacker.com/manage-and-unsubscribe-from-all-your-online-accounts-wi-1797025817>

注册一个在线账户是我们经常做的事情，以至于我们忘记了我们订阅的所有网站。你只会记得他们给你发了一封促销邮件。我知道我总是这样。



但是有很多在线工具可以帮助你在网上清理你的足迹。我最近碰到的一个新工具是 [Deseat。Me](https://www.deseat.me/) ，这是一款网络应用程序，它会扫描你的电子邮件账户，寻找你已经注册的服务，然后将它们放在一个列表中，供你删除或保留。

现在，我知道我们很多人都很关心隐私——尤其是在 [Unroll.me](http://lifehacker.com/unroll-me-the-email-unsubscription-service-has-been-c-1794593445) 收集用户的个人数据被曝光之后。但 Deseat.me 不会收集你的私人数据，也不会将其发送到云端，据其开发者称， [。](https://www.producthunt.com/posts/deseat-me)

所以，如果你想通过关闭一堆在线账户来清理你的在线生活，这似乎是一个相当安全的赌注。

### 使用 Google 或 Outlook 登录

使用 Deseat.me 很简单。你要做的就是去沙漠。我，然后用你的 Google 或 Outlook 帐户登录。如果您已经在您的浏览器上登录到任何一个，您将不必输入任何用户名或密码。您将被重定向到另一个页面，这样您就可以授予对 web 应用程序的访问权限。(如果你感到紧张，我建议你看看网站的隐私政策。)

### 选择要“保存”和“删除”的帐户

登录后，您会看到一个“等待决定”的网站列表你必须一个接一个地选择你想要保存的和你想要放入“删除队列”的。您放在删除队列中的链接将被放在一个不同的列表中，该列表包含一个指向网站取消订阅页面的链接。如果您点按该链接，您将被重定向到该网站，这样您就可以删除您的帐户。

如果您不确定是否要删除该网站，因为您不记得它是什么，您也可以选择忽略该链接，方法是按下它旁边的黄色警告图标，这表示“我不是这里的用户”

删除队列的缺点是，它不会为你注册的所有网站提供取消订阅页面的链接。只有知名的大网站才有删除链接(比如 Yelp，Apple，Digg)。

不管有没有提供删除链接，看看你在网上开了什么账户还是不错的。如果没有链接把你重定向到退订页面，你可以自己去找。

### 删除您的 Deseat.me 帐户

如果你想删除你的 deseat.me 帐户，因为你已经删除了你的帐户，或者你不喜欢它访问你的电子邮件并扫描它们，你可以点击页面底部的“删除你的帐户”。
