# 最好的 NES 视窗模拟器

> 原文：<https://lifehacker.com/the-best-nes-emulator-for-windows-1739211743>

如果你想要怀旧的老式 NES 与高清晰度电视的质量和方便，模拟器可以带你走得很远。如果你想要一个完美的 NES 体验，你会想去 RetroArch 的 Nestopia UE core。

### **特性**

*   以 rom 的形式从任天堂娱乐系统和 Famicom 磁盘系统玩游戏

*   使用几乎任何 USB 游戏手柄玩游戏，并自定义按钮布局
*   在游戏中的任何地方保存和加载你的状态
*   实时倒带游戏
*   调整各种视频设置，包括为图形添加老式效果的着色器 [或平滑的着色器](http://lifehacker.com/bring-your-retro-games-into-the-modern-age-with-these-e-1516605379)
*   录制您的游戏视频，或者将您的按键记录到 BSV 文件中
*   使用 Netplay 与朋友在线游戏

### **如何设置**

RetroArch 比“安装，文件>打开”稍微复杂一点。 [我们有一个完整的使用 RetroArch](https://lifehacker.com/how-to-set-up-an-all-in-one-retro-game-emulator-with-re-1745863517) 的指南，但这里有一个如何用 Nestopia UE 设置它的快速入门:

1.  从其 [下载页面](http://buildbot.libretro.com/stable/) 下载 RetroArch 的最新版本。它是一个 7z 文件，所以你需要安装 [7-Zip](http://www.7-zip.org/) 来解压它。
2.  打开 7z 归档文件，将文件解压到您想要的位置(我推荐 C:\Program Files\RetroArch)。
3.  双击 RetroArch exe 来启动它。您可以使用箭头键浏览界面，按 X 键选择，或按 Z 键返回。它还支持许多开箱即用的 USB 游戏手柄。
4.  要在 RetroArch 中加载模拟器，您需要安装模拟器的“核心”。前往在线更新>核心更新，并向下滚动到“NES / Famicom (Nestopia UE)”。选择它以安装 Nestopia UE 核心。
5.  返回到主菜单，并去加载内容>选择文件和检测核心。从硬盘上选择一个 ROM 文件开始播放。

你也可以 [调整众多的视频、音频和游戏手柄设置](https://lifehacker.com/how-to-set-up-an-all-in-one-retro-game-emulator-with-re-1745863517) ，但这将让你启动并运行。

### 它擅长的地方

RetroArch 最大的优势是它的设置数量。这对于一些用户来说可能有些力不从心，但是它允许您创建一个近乎完美的 NES 仿真体验，这是其他仿真器无法做到的。通过正确的设置，你可以用很酷的着色器获得像素完美的图形，以恢复旧的 CRT 外观*和*最小化输入延迟(许多仿真器的一个严重问题)。我强烈推荐你花点时间在 [中为 NES 设置回溯——这会给你一个近乎完美的体验。](http://nintendoage.com/forum/messageview.cfm?catid=31&threadid=135619)

### 不足之处

RetroArch 非常复杂，几乎没有文档可言(别担心，我们正在编写自己的指南)。如果你不熟悉 RetroArch，安装内核和调整设置真的很令人困惑，而且因为没有很多关于它的指南，你将做大量的谷歌搜索来弄清楚它，特别是如果你在不止一个仿真器上使用它。但它是我们发现的唯一一款可以消除令人讨厌的屏幕撕裂并尽可能减少输入延迟的模拟器，这意味着它是 NES 游戏的最佳选择。

### 竞争

[**Nestopia**](http://nestopia.sourceforge.net/) 和[**Nestopia UE**](http://0ldsk00l.ca/nestopia/)可以作为单独的仿真器下载，如果你不想经历设置追溯的麻烦，它们也很不错。然而，它们有一个明显的问题:你必须在极其缓慢的输入延迟(打开 vsync)和令人讨厌的屏幕撕裂(关闭 vsync)之间做出选择。RetroArch 还有其他功能可以解决这个问题，所以你可以在没有延迟的情况下使用 vsync。如果你不介意屏幕撕裂，我推荐关闭 vsync 的 Nestopia。

现在流行的**[**FCEUX**](http://www.fceux.com/web/home.html)有很多特性，大部分都是可以追溯的。FCEUX 允许你调整仿真速度，使其更快或更慢，这是一个很好的补充。它比 RetroArch 更容易使用，但默认情况下它带有一个非常不准确的调色板，并且在 vsync 打开时有音频噼啪声的问题。因此，我们更喜欢这个列表中的其他模拟器。RetroArch 有一个 **FCEUmm 内核**，其工作效果与 Nestopia UE 内核一样好，尽管颜色与 FCEUX 一样不正确。(你可以用一个. pal 文件来修改颜色，但是 Nestopia 的颜色开箱后就很准确。)**

**[**Jnes**](http://www.jabosoft.com/categories/1) 没有太多的功能，但它可能是最容易开箱即用的。它在 vsync 打开时没有可怕的输入延迟，并且它有一个准确颜色的一键选项(尽管它在默认情况下没有打开)，所以只要你不需要或不想要任何其他高级功能，你就可以很快获得不错的体验。**

**还有很多其他的 NES 模拟器(说真的，比你想象的要多得多)，但是这些是最受欢迎的，大多数人应该很容易被 Jnes 或 RetroArch 覆盖。**
