# 本周使用谷歌地图寻找瓦尔多

> 原文：<https://lifehacker.com/use-google-maps-to-find-waldo-this-week-1824221276>

从今天开始，如果你在手机、Chromebook 或 Google Home 设备上问谷歌助手“沃尔多在哪里？”你实际上会得到一个答案。

Watch

对于今年的愚人节，谷歌与瓦尔多合作，瓦尔多将在整个星期与地图用户分享他的位置。

要查看他在哪里，你需要使用手机上最新版本的谷歌地图应用程序(立即更新！)，还是谷歌地图的桌面版。

要开始搜索瓦尔多，当您看到他在屏幕一侧挥手时，请点击他的脸。当你这样做的时候，你将有机会在世界各地寻找标志性人物的位置。

他们还在分发寻找沃尔多和他的朋友的徽章(汪汪——但你只能看到他的尾巴、文达、巫师白胡子和奥德劳都在游戏中出现)。

谷歌正在慢慢推出桌面版，所以如果现在不行，几个小时后再试。
