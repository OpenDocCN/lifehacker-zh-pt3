# 和其他父母一起建立一个午餐打包俱乐部，分散劳动力

> 原文:[https://life hacker . com/start-a-lunch-packing-club-with-other-parents-to-spread-1803781129](https://lifehacker.com/start-a-lunch-packing-club-with-other-parents-to-spread-1803781129)

每天打包学校午餐可能会有点乏味，结果可能会有点单调。(对不起，我亲爱的孩子，他已经吃了五天剩下的意大利面和豌豆了。)分散劳动力，让事情变得对孩子的口味更有趣的一个方法是和其他父母一起创办一个午餐打包俱乐部。

和你孩子班上的其他几个爸爸妈妈组成一个团队，轮流负责一周的学校午餐——整个团队。这个想法是，如果你已经为一个孩子准备好了所有的食物(面包、熟食肉、葡萄等)。)，组装多餐足够简单。可以向孩子们介绍新的食物和小吃，你甚至可能开始变得有创造力(提示: [原木上的蚂蚁](https://en.wikipedia.org/wiki/Ants_on_a_log) 是午餐时间的经典之作)。当你的一周结束后，你就暂时结束了。继续再睡八分钟。

关于这个计划有几点说明。对于极度挑食的人来说大概是行不通的。此外，如果你的孩子有任何食物过敏，你会想跳过这个黑客-其他父母可能不会像你一样仔细阅读成分。你可能会像我们在 80 年代那样使用一次性纸质午餐袋——没人愿意每晚洗四个复杂的有 13 个不同隔层的便当盒。

有了这些想法，继续开始你的俱乐部吧。这里有一些关于 [打包最好的学校午餐](http://skillet.lifehacker.com/pack-the-best-school-lunch-by-thinking-in-threes-1786819696) 的建议。
