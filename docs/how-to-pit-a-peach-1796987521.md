# 如何给桃子去核

> 原文：<https://lifehacker.com/how-to-pit-a-peach-1796987521>

桃子可能有点粘人，尤其是在把它们从核里取出来的时候，但是你可以做一些事情来让这个有趣的任务变得简单一些。

首先，要知道桃子有两种不同的类型: [毛果核和粘果核](https://thepeachtruck.com/pages/freestone-vs-clingstone) 。顾名思义，一只*的肉粘在核上，而另一只则自由分离。(克林斯通季节在初夏；自由石季节从六月中旬开始。)*

很难找到 freestone 吗？别担心，你还是可以把水果和核分开的。首先，确保你的桃子尽可能成熟，因为不成熟的水果更难摘下来。然后，从茎到底部垂直地将桃子切成两半，并扭动两半直到它们分开。坑会粘在一边，用手指应该能拔出来。如果它还是有点顽固，试着用削皮刀轻轻地把它撬出来，或者把它切成片，把片从石头上拔下来。
