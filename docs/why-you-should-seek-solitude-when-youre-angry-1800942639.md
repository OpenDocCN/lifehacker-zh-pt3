# 为什么你生气的时候应该寻求独处

本周的精选来自普鲁塔克的作品《愤怒的控制》当你生气时，他有一个有用的建议:

> “因此，最好的办法是让我们自己冷静下来，或者逃跑，隐藏自己，把自己停泊在一个平静的港口，就像我们察觉到癫痫发作一样，这样我们就不会摔倒，或者更确切地说，不会摔倒在别人身上；尤其是我们最有可能落在朋友身上”。- *德·科希本达·艾拉*

### 这意味着什么

当你发脾气时，感觉愤怒就要占据你的头脑，你必须想办法让自己平静下来。普卢塔克说，最好的办法是远离让你生气的人、地方或事物，花些时间独处。这样，愤怒可以平息，你可以用清醒的头脑解决问题。

如果你没能走开，你就冒着出丑和丢面子的风险。你也可能把你的愤怒发泄在不值得你这样做的人身上，或者更糟的是，你会把它发泄在只想安慰你的朋友和家人身上。

### 从中吸取什么

普鲁塔克的建议简单明了，但很有效。下一次，当你感到怒火中烧的时候，不要大发雷霆，不要做出任何草率的决定，也不要试图从一数到十，希望事情能这么快冷却下来。只是走开。

找一个你可以独处的地方，让你的愤怒消失。找到你平静的港湾，在那里你可以起锚，从你的情绪中夺回控制权。当你平静下来后，反思一下当时的情况和你的行为。具体是什么让你生气了？你的愤怒有道理吗？你能做些什么来避免以后出现同样的情况？一旦你平息了你的愤怒，回到这个世界，用清醒的头脑来处理你一天的剩余时间。
