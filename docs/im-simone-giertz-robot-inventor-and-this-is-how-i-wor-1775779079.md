# 我是西蒙·吉尔兹，机器人发明家，这是我的工作方式

> 原文:[https://life hacker . com/im-simone-giertz-robot-inventor-and-this-is-how-I-wor-1775779079](https://lifehacker.com/im-simone-giertz-robot-inventor-and-this-is-how-i-wor-1775779079)

如果你最近看过一个机器人优雅地做傻事的视频，很可能是西蒙·吉尔兹 T3[的作品。正如一些人所说，她已经迅速成为低劣机器人的女王。](http://simonegiertz.com/) 

Watch

也许用*优雅*来形容 [一个打你脸的闹钟](https://www.youtube.com/watch?v=61FaYVGVY_I) 并不合适。西蒙妮的许多令人捧腹的发明都是令人着迷的混合了杰特森式的惊奇和扭曲的黑色幽默。它们是旨在简化生活的自动装置——几乎是！

但直到最近我才意识到，我在网上看到的所有这些产生 gif 的机器人——互联网辩论机[自动口红机](https://www.youtube.com/watch?v=WcW70-6eQcY) 等等——都是一个发明者的作品。(当然，Simone 用这些疯狂的机器人的真正目的是鼓励人们学习电子学。)现在西蒙娜已经加入了 [测试](http://tested.com) 的团队，而她的第一个项目，与 [【亚当·沙维奇】](http://lifehacker.com/im-adam-savage-and-this-is-how-i-work-5926330) 合作，是一款头戴式 [自动爆米花机](https://www.youtube.com/watch?v=U4LZbewqB-E) 。

由于通过技术改善生活通常是我们的目标，我联系了这位机器人发明家，想了解一下她是如何工作的。

* * *

**地点:**瑞典斯德哥尔摩
**目前演出:**运行我的 [Youtube 频道](https://www.youtube.com/c/simonegiertz) 与亚当·沙维奇、诺姆陈共同主持 [测试](http://www.tested.com/) 。
**一个词最能形容你的工作方式:**创意混乱。那是两个字。去他妈的系统。
**当前移动设备:**一部 iPhone 6S。
**当前电脑:**一台 2013 年末的 Macbook Pro，搭载 OS X El Capitan。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-KcxlSLNH5M0&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-KcxlSLNH5M0&start=0) 

#### 你离不开哪些应用、软件或工具？

互联网。这就像纳尼亚，但你不用进入衣柜，只需打开浏览器。我也很喜欢有双手。

#### 你的工作空间是什么样的？

我现在是我朋友哈坎办公室和工作室的寄生虫。我们是这里的三个人，制造从鼓机到超级英雄装备的所有东西。

#### 去生活帮最省时的捷径是什么？你是否自动化了一些曾经耗费时间的东西？你会把电子邮件减少到每天一小时吗？

得到一个经理是一个相当惊人的生活帮。突然间，我有时间脱离大脑的束缚，在会议间隙练习倒立。至于电子邮件，我会整天回复紧急的(和有趣的)邮件，并尝试每周两次浏览所有邮件。但有时我的收件箱会自行崩溃，变成一个黑洞，不让我离开。

#### 你最喜欢的待办事项管理器是什么？

[神童](https://www.wunderlist.com/) 是我唯一真正用过的。我喜欢。

#### 除了你的手机和电脑，还有什么小玩意是你离不开的，为什么？

我的自行车。我几乎到处都骑自行车。它也很适合自言自语。

#### 你在日常生活中的哪些方面比其他人做得更好？

谦逊。我击败了所有谦逊的人。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-PJiRijiLwbQ&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-PJiRijiLwbQ&start=0) 

#### 你工作的时候听什么？有最喜欢的播放列表吗？也许谈电台？还是更喜欢沉默？

当我周围有声音时，我会更好地集中注意力，而且我倾向于在工作时听音乐。但是最近我刚刚在后台播放了这个的精彩现场直播。我妈妈和我在过去的几个星期里一直痴迷于这些鸟，并有一个专门的聊天线程。鸟类是最好的。

#### 你目前在读什么？一本小说，漫画书，网站，杂志？

看了很多书，主要是玄幻类的。目前在读伯纳德·康维尔的《冬王】。至今不爱，但至少让我睡着了。

#### 怎么充值？

我跳林迪·霍普舞，和那种你可以不问青红皂白就搜冰箱的朋友出去玩，尽我所能远离我的手机和电脑。我基本上每个周末都会进行网络节食。

#### 你的睡眠规律是怎样的？你是夜猫子还是早起者？

我每晚睡 8 个小时左右，如果不睡，我会变得非常非常爱发牢骚，脾气暴躁。当我睡眠不足时，我姐姐叫我兽人。与普遍的看法相反，我用的是普通闹钟，而不是 [唤醒机](https://www.youtube.com/watch?v=61FaYVGVY_I) 。

#### 填空:我很想看到 _________ 回答这些相同的问题。

80 岁的西蒙。我打赌她一定很坏。

#### 你收到过的最好的建议是什么？不一定与工作或生产力相关。

不用洗甲水洗掉眼部化妆是一个很好的选择。谢谢妈妈。
