# 行为艺术家成为人类虚拟助手

> 原文：<https://lifehacker.com/performance-artist-becomes-human-virtual-assistant-1798700949>

像 Siri、Alexa、Google Now 这样的虚拟助手还处于尴尬的阶段。它们对于控制你的音响、回答偶尔的琐事问题或提醒你交通状况很有用，但它们仍然没有足够的智能或权限来真正预测你的最大需求。一点也不像有一个真正的人看着你并给你建议。也就是行为艺术家劳伦·麦卡锡承诺的，一次三天。



麦卡锡称自己为劳伦，一个“人类智能家居”如果她接受你的申请，她会拜访你的家，安装监控和智能家居设备，并花三天时间控制你的家，预测你的需求。

这是那种似乎注定要成为真正服务的艺术项目。已经有 [多种辅助服务](https://www.nirandfar.com/2015/07/the-message-is-the-medium-3-reasons-apps-as-assistants-work.html) 将电脑辅助与人工辅助结合在一起。但是它们都侧重于被动的援助，而不是主动的援助。一个专门的人类助手，其行为更像 T4 她的 T5，而不是 Siri，会贵得多。

但是随着 [财富差距的扩大](https://inequality.org/facts/wealth-inequality/) ，一个潜在的富裕客户市场正在崛起，他们可以负担得起报酬过低的全天候助理，就像中国的 [对管家](https://www.nytimes.com/2017/01/14/world/asia/rich-chinese-inspired-by-downton-fuel-demand-for-butlers.html?_r=0) 需求的上升一样。由于谷歌、亚马逊、微软和苹果 [拒绝完全整合他们的语音助手](http://lifehacker.com/alexa-and-cortana-teamed-up-but-consumer-tech-is-still-1798642894) ，人类助手可以消除这些粗糙的边缘并填补这些空白，接管管理技术的所有繁琐工作，并给出难以自动化的建议，如“你需要理发”或“我认为这音乐让你无法专心做作业。”

劳伦网站 上的推荐让她听起来很有收获，尽管有点咄咄逼人。她显然带来了某种程度的关注和洞察力，这是任何算法，甚至是一个不那么专注的远程人类都无法达到的。想象她这种服务成为常态似乎并不疯狂。
