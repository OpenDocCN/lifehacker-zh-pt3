# 用这种没有食谱的食谱，一年四季都可以得到美味的填充蔬菜

> 原文：<https://lifehacker.com/learn-this-no-recipe-recipe-for-delicious-stuffed-veget-1819942281>

作为一名家庭厨师，作为一个吃她买的蔬菜而不是让它们在保鲜盒里液化的人，没有什么比学习没有食谱的烹饪更重要的了。一旦我记住了一些常用的方法，“快速晚餐”就意味着厨房的即兴表演，而不是无缝点菜。

我 100%地将这个从 [Food52](https://food52.com/blog/20751-how-to-make-stuffed-vegetables-your-way-without-a-recipe) 的填馅蔬菜路线图添加到轮换中。我喜欢它的适应性，以及它可以做一顿完整的饭的事实——我经常烤一些美味的蔬菜，然后惊慌失措地在上面扔一大块豆腐。

要点如下:

*   将你的容器蔬菜纵向切成两半，挖出里面的东西来做船。(如果你用的是冬南瓜，清洗种子进行烘烤；如果你用的是较软的蔬菜，把里面切碎加入馅料。)

*   在 375 华氏度下烘烤容器，撒上橄榄油和盐，直到它们变软。软蔬菜需要 10-15 分钟，而硬蔬菜如南瓜需要 45 分钟。

*   当器皿烹饪(然后冷却)时，煎馅料。Food52 的配方是:“淀粉+大蒜+奶酪+坚果+干果+蔬菜+香草。”使用上述部分或全部方法。你也可以添加蛋白质，比如碎牛肉或扁豆。是的，你必须注意数量，但未使用的馅料本身也很好吃。(我会用淀粉把你的量调成合适的量。)

*   一旦做好，将蔬菜放入锅中，放回烤箱中烤 5-10 分钟，直到蔬菜完全变热并稍微变黄。

你甚至可以提前三天准备容器和馅料——只需冷藏，然后在烤箱中多放一会儿预热。

Food52 为你提供了以下建议:

*   波布拉诺胡椒配黑豆、糙米、西班牙香肠和奎索弗雷斯科香肠，浇上石灰奶油

*   甜维达利亚洋葱配法罗、凤尾鱼、羽衣甘蓝和焦糖洋葱，浇上番茄酱

*   用藜麦、帕尔马干酪、罗勒和面包屑制作的西葫芦或西葫芦

*   Portobello 蘑菇配珍珠大麦和腌制柠檬或块根蔬菜意大利调味饭

*   茄子配保加利亚干酪、碎羊肉、西红柿和许多香料
