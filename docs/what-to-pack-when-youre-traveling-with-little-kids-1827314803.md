# 当你带着小孩旅行时，该带些什么

> 原文：<https://lifehacker.com/what-to-pack-when-youre-traveling-with-little-kids-1827314803>

我刚和我丈夫的大家庭从欧洲旅行回来。这是我们第一次带着我们 5 岁的孩子出国旅行，尽管经历了几次因时间变化而导致的崩溃，我很高兴地告诉大家这是一次非常积极的经历。我还不能说我是带着孩子环游世界的专家，但我一直痴迷于从父母那里窃取一些技巧，尤其是那些掌握了包装艺术和科学的父母。在这次假期中，我很高兴我带来了一些东西，还有一些我希望我已经想到的东西。

Watch

*   一个满载的 iPad 和 kid 耳机。我可能忘了我的泳池凉鞋，但我记得把*可可*和*帕丁顿 2* 下载到 iPad 上，在飞机上享受一些安静、放松的时间，这更重要。

*   **婴儿湿巾。**至少有八次我想，“*应该带些婴儿湿巾*其中包括:当我的女儿在飞机上把一块 Kinder Egg 巧克力掉到她的衬衫上时，我后来发现她胸部上有一个奇怪的棕色皮疹，当一些不明粘性物质在乘船时落在她的头发上，以及 2018 年的大冰棒棒事件。幸运的是，小组中更有远见的父母总是在他们的包里带着包，当混乱发生时，他们友好地分享。下一次，我会在背包的前袋里放一些，方便随时取用。这让我想到...

*   **不粘腻，富含蛋白质的零食。对于有挑食孩子的父母来说，旅行可能会很艰难。有一些方法可以帮助他们适应海外美食冒险的想法(稍后会有更多的介绍)，但有时，你只需要在长途旅行中让他们的身体获得一些营养。我女儿抱怨飞机餐太辣了，她满足于在一包黄油上吃 11 个小时。我很高兴我带了一些杏仁奶油格兰诺拉麦片和麦片，这让她可以吃一些类似食物的东西。坚果、干果和健康小吃店也是不错的选择。** 
*   **一个小孩的水瓶。**我们认为这可能只是占用空间的东西，但它在整个旅程中都被使用。阿姆斯特丹到处都有公共饮水器，我们经常把瓶子装满。

*   **大号拉链袋。**我的女儿抗议她在特定日子里要穿的一些衣服(假期:就像普通生活一样——只是风景更好！)，有一次我们在她的行李箱里翻找一双特别的袜子。一些父母发誓说，如果你使用 Ziploc 袋系统，这种事情就不会发生，在这个系统中，你基本上是将孩子的衣服装在冷冻袋中(一加仑或两加仑，取决于物品的体积)。每套服装(包括配饰和袜子)都有自己的袋子，每个袋子上都贴有日期和场合的标签。你可以额外准备一个包来装备用物品，比如几件衬衫和裤子，可以搭配任何东西。这种方法的一个好处是，你可以把脏衣服放回包里，密封起来，防止它们弄脏行李箱。
    T3】
*   一件柔软的夹克。我为自己带来了我最喜欢的夏尔巴人毛绒夹克，但我的孩子可能得到了最好的利用，当她在长途公共汽车、火车和飞机上午睡时，它成了她的毯子和枕头。
    T3】
*   一本旅行日志。我嫂子为她的孩子们写了旅行日记，内容包括他们想看的所有景点的剪报、他们想尝试的食物和他们想留意的荷兰单词。书页上贴着小信封，用来装博物馆门票、餐馆名片和其他小物件。当我们走过这座城市时，孩子们会惊呼:“那在我的日记里！”开始记笔记。和你的孩子一起写日记是建立兴奋感的好方法——研究表明，在旅行开始之前，对假期的期待就能给你带来快乐。对于不爱冒险的食客来说，看到小册子上的某些菜肴可能会给他们一定的动力，让他们尝试一下 kroket 或 bitterballen。

*   **非 iPad 活动。**在我们休息的时候，纸娃娃、手指木偶和小平装书让我女儿开心又平静，这也让我开心又平静。这里有一些关于 [的建议，当你在处理](https://offspring.lifehacker.com/9-ways-to-keep-kids-entertained-during-a-long-airport-d-1820246991#_ga=2.140067891.188549136.1530613699-1791332793.1526323747) 和小孩子的长时间机场延误时，该如何打包。
