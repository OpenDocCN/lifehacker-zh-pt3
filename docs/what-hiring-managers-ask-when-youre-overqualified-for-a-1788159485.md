# 当你大材小用时，招聘经理会问什么

> 原文:[https://life hacker . com/what-recruiting-managers-ask-when-you-conditioned-for-a-1788159485](https://lifehacker.com/what-hiring-managers-ask-when-youre-overqualified-for-a-1788159485)

有多少次你看到一份工作清单会让你说，“哇，这看起来很有趣，但我太有经验了？”这种情况过去经常发生在我身上——尤其是当我开始在销售组织中晋升时。

我想全职写作，我会让自己做着白日梦，看着公司招聘初级作家。但我从未采取行动，因为我已经接受了这样一个事实:我有“太多”的职业经验，永远不会被聘用。

然而，当我成为一名招聘人员时，我意识到我对形势的看法完全错了。当雇主手上有一个好的、但技术上资历过高的候选人时，他们会讨论很多事情。通常，他们不只是问自己，“我们能对这个人说不的最卑鄙的方式是什么？”

相反，这里有一些他们问的问题。

### 1.这个人是想转行吗？

虽然你可能会看着自己的简历，认为自己资历过高，但招聘经理不一定会这么认为。为什么？比方说，你想从市场营销工作转到财务工作；很明显，在工作中你会学到很多东西。当然，你比其他求职者有更多的专业经验——但是你可能仍然拥有完全相同的行业证书。

你申请的事实表明你意识到了这个事实。你可以在求职信 的 [中陈述你的情况，让招聘经理明白你正在积极尝试转行。](https://www.themuse.com/advice/how-i-successfully-landed-a-job-in-a-totally-unrelated-field)

### 2.此人目前的头衔是否准确反映了他或她的角色？

作为一名招聘人员，我最大的惊讶之一是:许多人的头衔在纸面上看起来很高级，但并不能说明他们每天被要求做什么。事实上，我早期犯的最大错误是，假设每个“经理”角色都意味着申请人有十几个直接下属，每天都要做出高层决策。在安排了一些不够资格的候选人进行电话面试后，我得到了惨痛的教训。

那么，你应该从中吸取什么呢？实际上是两件事。第一，当你寻找工作机会时，你不应该过分强调头衔。很明显，高级副总裁和实习生是有区别的，但是不要因为你刚从大学毕业几年，就害怕申请一个头衔中有初级职位的职位。

第二件事是给那些担心自己的“高层”头衔会让招聘经理失去兴趣的人。用你的简历和求职信来解释你每天在办公室实际做什么——这些信息对招聘人员来说比你的名片上写的更有价值。

### 3.这个人会乐意向她的老板汇报吗？

当我们认为某人“资历过高”时，我们试图弄清楚的一件事是，该候选人向她的上司汇报是否合适。因为有时这意味着经理的职位比应聘者之前的职位低。其他时候，我想知道候选人是否愿意和一群头衔相同的人一起工作——而不是管理任何人。

那不一定会阻止我带人进来。但这是我们确实考虑过的事情。如果一个人明确表示他或她对团队合作不感兴趣，我们就通过了。

所以，不要老是重复这一点，用你的申请材料清楚地表明你目前的经历不会阻碍你；强调你是一个团队成员，你的自负不会妨碍你的工作。

对于你想要的工作来说，没有一门精确的科学可以知道你是否大材小用。然而，也没有硬性的招聘规则规定雇主只能雇佣符合每一项条件的人。如果你认为一家公司会通过，提交申请似乎是浪费时间。但是知道他们讨论的内容将(有希望)帮助你迈出这一步，并有希望获得一个最适合你的职位——去他妈的头衔。
