# 您可以按偏好顺序排列您的 Wi-Fi 网络

每当我带着笔记本电脑外出时，通常是在咖啡店或共同工作的地方，我都会注意到一些特殊的连接问题。我的 Mac 通常会连接到 ISP 提供的连接，而不是我手机上的热点或大楼里的 Wi-Fi，好像它更喜欢其中一个。

如果你在 ISP 供电的接入点附近，或者有一个双频带路由器，并且想要使用更快的 5GHz 连接，而不是更慢、更长范围的 2.4GHz 频带，那就糟了。尤其令人沮丧的是，当我下载大文件，想知道为什么花了这么长时间，却发现它甚至没有使用我的路由器。我花大价钱买了那个路由器。

判决结果？事实证明它确实有偏好，这取决于哪一个比另一个更重要。您在 MacOS 中的网络优先级顺序取决于其首选网络列表中的排列顺序。

当寻找 Wi-Fi 连接时，你的 Mac 会向下滚动先前使用的接入点列表，并跳到最上面的选项。当然，这并不意味着它总是你想连接的网络。但是您可以自己重新排列首选网络列表，只需拖放即可。

在 Mac 上，打开“系统偏好设置”,点击网络图标。在那里，你会在侧边栏看到你的网络连接选项。点击 Wi-Fi 选项，然后点击右下角的“高级…”。

在那里，您将看到您过去连接过的网络列表以及它们的安全偏好设置。如果您看到您知道您再也不会连接到的网络，如您设置的旧接入点或公共汽车和飞机热点，请选择它并点击列表下方的减号将其删除。

一旦你完成了春季大扫除，就开始拖吧！我喜欢把我珍视的 Wi-Fi 连接放在最上面:我工作的接入点，我家的 5GHz 网络，我咖啡馆的路由器。较小的接入点，如来自我的 ISP 的接入点，会被一路拖下来，这是我无法连接到任何其他设备时的最后手段。
