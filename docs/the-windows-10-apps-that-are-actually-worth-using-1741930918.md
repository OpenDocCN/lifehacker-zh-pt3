# 真正值得使用的 Windows 10 应用

> 原文：<https://lifehacker.com/the-windows-10-apps-that-are-actually-worth-using-1741930918>

微软对 Windows 10 的新通用应用 大做文章。虽然一些开发人员赶不上潮流，但也有一些人专门为 Windows 10 开发了新版本的应用程序，值得一看。



你能在 Windows 商店里找到的新的通用风格的应用程序仍然是相当新的，而且并不是所有的应用程序都像传统的应用程序一样强大。然而，Windows 10 应用有几个内置的关键优势:

*   他们可以通过 Windows 应用商店进行更新。【Windows 最大的问题之一是没有集中的方法来更新应用程序。有了 Windows 10 应用，你可以通过微软的商店进行更新，这不仅更容易，而且更安全。
*   它们与内置的媒体控制配合得很好。 Windows 10 改进了其 [屏幕媒体控件](http://news.softpedia.com/news/Windows-10-to-Feature-Media-Controls-for-Desktop-Apps-468846.shtml) 的工作方式。任何时候调整音量，或者使用键盘上的媒体按钮，您都会看到一个覆盖弹出式菜单，显示正在播放的内容，并为您提供音量指示器、播放/暂停、下一个和后退按钮。这自然让网飞和 Plex 这样的应用程序比基于浏览器的应用程序更有优势。
*   **界面默认支持触摸输入。**大多数 Windows 电脑都没有触摸功能，但近年来有了很多。其中有些 [还挺好看的](http://gizmodo.com/the-world-finally-admits-microsoft-surface-was-the-shit-1730622015) ，甚至。如果你在支持触控的设备上使用 Windows，你会喜欢这些应用。
*   它们看起来很棒。这可能有点肤浅，但即使 Windows 10 开始看起来比它的前辈好得多，旧的应用程序仍然使用过时和困难的设计。在新的 Windows 10 风格中，一些应用程序可能会更容易或更好用，即使他们的旧版本有更多的功能。

当然，Windows 10 应用程序的一大缺点是，微软的设计语言*仍然*并不完全适合每个应用程序。虽然有些人已经很好地利用了微软提供的东西，但仍有许多人感到被微软追求的风格所强迫或束缚。也就是说，这里是我们发现的一些在 Windows 10 上有着非常棒的体验的最佳产品。

### [](https://www.microsoft.com/en-us/store/apps/netflix/9wzdncrfj3tj)

**网飞倾向于在所有平台上都有自己的界面风格，所以它的 Windows 10 应用看起来最合适。它有很大的触摸友好按钮，在家庭影院电脑或第二个显示器上看起来也很棒。您还可以使用键盘媒体控制，以便在节目中间更容易暂停、快进或调低音量。当然，如果你更喜欢坚持使用浏览器，你也可以用 [我们的 Flix Plus 扩展](http://lifehacker.com/flix-plus-customizes-every-corner-of-netflixs-new-inter-1715305787#_ga=1.166688507.431406394.1415821409) 来调整它。**

#### **[**Plex**](https://www.microsoft.com/en-us/store/apps/plex/9wzdncrfj3q8)T5】**

**Plex 是我们最喜欢的 [流媒体](http://lifehacker.com/how-to-stream-your-media-from-home-to-your-phone-anywhe-5821512#_ga=1.166688507.431406394.1415821409) 应用之一，它非常适合作为 Windows 10 应用。大海报艺术和缩略图使它看起来像一个家庭影院电脑应用程序，或者只是浏览你的图书馆。媒体控制也与应用程序一起工作，这使它从 Plex web 界面中脱颖而出。该设计也与其他平台上的 Plex 非常一致，因此对于 Plex 老手来说应该很容易导航。**

#### **[](https://www.microsoft.com/en-us/store/apps/vlc-for-windows-store/9wzdncrfj3t0)**

****VLC 一直以能玩 [你能扔给它的任何东西](http://lifehacker.com/master-your-digital-media-with-vlc-397573) 而闻名，但不是因为它的界面。然而，它有一个为 Windows 10 设计的巧妙的应用程序，坦率地说，看起来比普通的 VLC Windows 应用程序好得多。与基本的 VLC 应用程序不同，你可以使用它来管理你的媒体库，包括专辑封面、缩略图和封面照片。它的功能不如常规应用丰富，但对于大多数基本任务来说，它真的不需要。它只需要播放视频和音频文件，这个做得很好。****

#### ****[**Adobe Photoshop Express**](https://www.microsoft.com/en-us/store/apps/adobe-photoshop-express/9wzdncrfj27n#features)****

****当谈到免费的照片编辑器时，智能手机获得了所有的乐趣。Adobe Photoshop Express 最终与桌面用户分享了这份爱。这个免费的应用程序可以在一个超级简单的界面中进行基本的修复，如裁剪、颜色校正、添加滤镜和减少噪点。它不会取代更专业的 Photoshop CS6 或任何东西，但如果你需要基本的照片编辑，这将完成工作。****

#### ****[](https://www.microsoft.com/en-us/store/apps/wunderlist-to-do-list-tasks/9wzdncrdfxzs)****

******一段时间以来，Wunderlist 一直是我们在 Windows 上最喜欢的待办事项列表应用之一。很少有开发者像 Wunderlist 一样接受微软关于通用应用的承诺。这款应用不仅外观华丽，还能根据你的窗口大小自动调整，动态优化你的可用空间。缩小窗口，你会得到一个更紧凑的界面。使用更大的窗口，你将有更多的按钮和空间来工作。虽然普通的 Windows 7 风格的应用程序工作类似，但它也缺乏一些功能，如登录谷歌，或作为通用应用程序的好处(如通过 Windows 商店更新)，这使得这个版本为 Wunderlist 用户提供了明确的选择。******

#### ******[**收纳箱**](https://www.microsoft.com/en-us/store/apps/dropbox/9wzdncrfj0pk)T5】******

****Dropbox 通常没有传统意义上的*应用*。它只是一个在后台运行的同步服务。然而，Windows 10 Dropbox 应用程序是一个完整的文件管理器和浏览器。您可以浏览您的文件夹，查看您的文件预览，并管理您的 Dropbox 设置，如相机上传和帐户安全。****

#### ******日历(股票 app)******

****内置的微软日历应用(简称日历)是目前最好的股票应用之一。您可以使用此应用程序来管理您的 Outlook、Exchange、Google 或 iCloud 日历。它看起来也很棒，这对于日历应用程序来说并不容易。它还可以与 Windows 10 的新通知中心配合使用，因此当重要会议即将召开时，你可以在桌面上收到提醒。它甚至与桌面上的Cortana 配合得很好。****

#### ****[**OneNote**](https://www.microsoft.com/en-us/store/apps/onenote/9wzdncrfhvjl)****

****OneNote 的笔记方式与我们常用的笔记工具 Evernote 不同，但 [还是很强大的](https://lifehacker.com/lifehacker-faceoff-onenote-vs-evernote-1550697885) 。OneNote Windows 10 应用程序也很好地展示了 Windows 10 应用程序的未来发展趋势。它使用了一个带状风格的菜单，但没有所有小图标拥挤在一个小空间里的混乱和困惑，这通常会破坏带状界面。如果你已经是一个 OneNote 用户，或者只是想要一个比写字板更好的东西来记东西，OneNote 是相当出色的。****

****如果你是一个 Xbox 用户(即使你不是 ，也是 [)，Windows 10 Xbox 应用程序是相当不可思议的。你可以查看你的个人资料，与朋友聊天，开始语音聊天，甚至用这个已经内置到 Windows 中的应用程序录制你的游戏。游戏录制功能并不局限于 Xbox 游戏，甚至根本不局限于游戏。因此，如果你只需要一个基本的屏幕录制应用程序，它甚至很方便。](https://lifehacker.com/windows-10s-xbox-app-is-awesome-even-if-you-dont-have-1723617404)****
