# 用这个杂志学习构建 Node.js Web 应用程序

> 原文：<https://lifehacker.com/learn-to-build-node-js-web-apps-with-this-zine-1823329167>

如果你是构建网络应用的新手，看看第一期 [你得到了这个！](https://glitch.com/you-got-this) ，来自开发者社区的友好杂志 [Glitch](https://glitch.com/) 。本期讲授 web 服务器背后的基本概念，以及 Javascript 的 npm 包管理器。它还展示了三个社区领先的 web 开发人员的职业简介，以及来自 Google、Mozilla 和 Slack 等公司的十几个开发人员的建议简介。

Watch

npm 包管理器帮助 web 开发人员构建强大的站点和应用程序，而无需从头开始制作每个组件。正如该杂志所解释的，“它使得 JavaScript 开发者重用其他开发者共享的代码变得容易……一个典型的应用程序或网站将由几十或几百个小的代码包组成。”例如，一个软件包可能会让用户使用他们的脸书帐户登录到您的网站。Npm 保持你所有的包都是最新的，所以任何错误修正或者新特性也将适用于你的站点。

每个概念性的解释都以 Glitch 项目或课程的链接结束，在那里你可以学习如何开发网站和网络应用。专业建议集中在程序员教育的早期阶段，范围很广。从表面上看，简介似乎是矛盾的:一个开发人员说，开始时只让一些东西工作，而另一个人说，这种方法减慢了她的速度。但是它们一起展示了开发人员职业生涯的不同路线，并且它们传达了一个信息:当你第一次学习开发时，采取任何能让你保持兴趣和动力的方法。
