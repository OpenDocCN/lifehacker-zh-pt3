# 使用此应用程序尝试播客

> 原文:[https://life hacker . com/try-out-podcast-with this-app-1823459525](https://lifehacker.com/try-out-podcasts-with-this-app-1823459525)

iOS:并不是每个人都像 Lifehacker 的书呆子一样有一个最喜欢的播客列表。如果你是播客的新手，现有的管理播客的应用程序可能会感到不必要的复杂——特别是如果你想要的只是你朋友提到的那个节目的最新一集。如果你不想担心存储缓存和播放列表生成之类的事情，可以看看[‘sodes](http://jaredsinclair.com/sodes/)，这是一个易于使用的播客应用程序。

Watch

Sodes 看起来非常类似于苹果自己的音乐应用程序，并且有粗体文本，清楚地描述了最喜欢的，正在进行的和未播放的部分。如果你只有几个播客，很容易进入最近的三集，或者将某些播客指定为你的最爱，这样你就可以先看到它们。

添加播客非常简单。点击搜索按钮，查看热门播客列表，或者通过名称或 URL 搜索您最喜欢的播客(您可以使用它来添加来自 Patreon 等网站的订户专属播客)。选择播客并点击您想要收听的剧集，或将整个播客添加到您的个人收藏列表中。

sodes 的简单性意味着无需管理下载数据等更复杂的参数，也没有播放列表创建器。Sodes 流式播放播客，而不是下载它们，节省您设备上的存储空间。虽然可能有其他更强大的播客应用程序可用，但 sodes 让听播客成为只想听连续剧(或升级版)的人的低摩擦活动！)而不会影响自动删除节目、通知、缓存剧集和其他选项。
