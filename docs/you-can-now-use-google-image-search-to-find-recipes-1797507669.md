# 你现在可以使用谷歌图片搜索来查找食谱

> 原文：<https://lifehacker.com/you-can-now-use-google-image-search-to-find-recipes-1797507669>

浏览食物图片一直很有趣，但你现在可以在手机上使用谷歌图像搜索，在浏览你最喜欢的食物色情片时轻松找到食谱。

Watch

这种变化似乎来自于 [新支持的图像和视频模式](http://searchengineland.com/google-image-search-now-shows-videos-recipes-278704) ，但这意味着你可以使用你的手机找到大量看起来美味(希望味道可口)的食谱。要做到这一点，只需:

1.  在谷歌图片搜索中输入他们想要的食物。
2.  点击他们希望进一步研究的小“食谱”标签。

一旦你找到了一个你喜欢的外观和声音的食谱，你就能把那个看起来美味的图像变成美味的现实，尽管你可能得先去趟杂货店。
