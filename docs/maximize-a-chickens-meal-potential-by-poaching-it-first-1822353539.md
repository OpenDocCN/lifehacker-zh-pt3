# 首先通过水煮来最大化鸡肉的潜力

> 原文：<https://lifehacker.com/maximize-a-chickens-meal-potential-by-poaching-it-first-1822353539>

如果你在网上读到烹饪，你肯定已经看过几篇吹捧一只不起眼的鸡的多种用途的文章。在这一点上，你知道一只鸡能为你做什么，所以另一个过度规定的，基于鸡肉的膳食计划似乎是不必要的。我的议程很简单:下次你买整只鸡的时候，我希望你把它煮熟。



偷猎是一种被低估的犯罪技术，鸡真的喜欢它。你不需要特殊的设备:如果你有一个搅拌碗，一个带盖子的大锅，一个滤网和肉汤的储存容器，你就可以这样做。更好的是，偷猎是万无一失的，因为烧烤不是*和*的方式，它支付更多的红利:一整只鸟的美味多汁的肉，一加仑美味的肉汤，一具可以再次炖成汤的尸体，甚至可能是一些 [变成脂肪](https://lifehacker.com/render-poultry-scraps-and-gain-excellent-cooking-fat-1821958410) 。如果你真的很想念那种酥脆的皮，只需 [将摇摆不定、怪异的水煮皮压在两个烤盘之间，然后烤它](https://skillet.lifehacker.com/save-chicken-skin-to-make-an-addictive-baked-snack-1687008258) 就能做出美味可口的脆皮。这是一个时间承诺——从开始到结束大约五个小时——但是只需要 30 分钟的积极工作，你就可以在休息日解决这个问题，并且仍然有足够的休息时间。

### 第一步:仔细计划你的饮食

用一只鸡做多餐都在计划之中。列出一周的饮食清单，从一顿以水煮鸡肉为主的大餐开始，然后相应地购物。我担保所有这些食谱:

*   [农氏考文垂](http://khaomangai.com/?portfolio=how-to-make-khao-man-gai)
*   [鸡肉面](https://smittenkitchen.com/2015/01/my-ultimate-chicken-noodle-soup/) 或 [丸子汤](https://www.bonappetit.com/recipe/matzo-ball-soup)
*   [鸡肉&饺子](http://www.geniuskitchen.com/recipe/chicken-and-dumplings-72963)T3】
*   [烧鸡](https://www.epicurious.com/recipes/food/views/red-cooked-chicken-with-shiitakes-235741)
*   Avgolemono

在丰盛的第一餐中提供黑肉或白肉，以最大限度地利用剩菜，但除此之外，这是一个你自己选择的冒险故事。你对鸡肉沙拉三明治失去理智了吗？把胸部留到最后，储备面包、蛋黄酱和松脆蔬菜。如果你想先吃白肉，把黑肉留着做墨西哥卷饼，确保你手头有玉米饼、融化的奶酪、豆子和墨西哥卷饼酱。你还会有一加仑的肉汤和高汤，所以要准备足够的汤、炖菜和/或意大利调味饭。说到原汤，别忘了块根类蔬菜和香料——我特别推荐一些 [欧洲防风草](https://skillet.lifehacker.com/how-to-make-truly-great-vegetable-stock-1822194489) 。

### 第二步:水煮鸡肉(并制作高汤)

将鸡肉从包装中取出。移除任何内脏，并修剪空腔周围多余的表皮。 [渲染修边](https://lifehacker.com/render-poultry-scraps-and-gain-excellent-cooking-fat-1821958410) 如果你喜欢；丢弃内脏或保存到另一个项目。

用保鲜膜把鸡肉包起来，让它达到室温，这个过程需要一两个小时。(在*室温*水中开始*室温*鸡肉可以确保它一直烹饪，所以不要跳过这一步！)同时，准备其他配料——我做了*烤馒头*，所以我切了些姜，切了些大蒜头，量出了盐和糖。

一旦鸡肉被加热，把它放在水槽的碗里(或漏勺里),并在上面撒一把盐。用盐擦遍全身，包括腔内，然后好好冲洗。剥离的鸡肉听起来像是一个狂热的梦想，但是盐磨砂去除了任何残留的皮肤粘糊糊——这使得肉汤更清澈，更美味。(如果你用的是犹太鸡，跳过这一步。)

将鸡肉放入锅中，使空腔朝上。首先用温水填充空腔，然后加入足够的水，覆盖鸡肉几英寸(一加仑通常对我来说就够了)。检查以确保空腔完全充满，并轻推您的鸡肉，直到其胸部朝上。

加入调味料，盖上锅盖。用中高火煮沸，然后关小火，用小火煮 25-30 分钟。关火，让鸡肉浸泡在肉汤里，盖上盖子，两到四个小时。长时间的浸泡会产生多汁的肉和味道浓郁的肉汤，而不会像在水中煮动物器官那样黏糊糊。

当冷却到你喜欢的程度时，把鸡肉从锅里拿出来。我发现最简单的方法是将一把坚固的长柄勺子放入鸡腔中，轻轻向上倾斜——这样你就可以顺着它的脖子看，而不是顺着它的屁股看——然后逐渐将鸡拿出来，如果需要的话，使用另一把勺子或钳子提供额外的支撑。这样，空腔内的液体会流回花盆。

将鸡肉放在砧板上，让它静置，同时过滤肉汤，然后用一半水煮鸡肉和所有你需要的肉汤来准备剩下的一餐。晚餐后，挑选干净的动物尸体，把所有东西都保存起来:肉留着以后吃，皮和骨头以及动物尸体留着储备。如果你厌倦了等着过滤热液体，明天再做吧。

### 第三步:利润

如果你以前从未吃过水煮鸡肉，你可能会对你盘子里的东西产生怀疑——尤其是如果你做了真正以水煮肉为特色的*烤满盖*或红烧鸡。水煮鸡怎么可能好吃，更别说放纵了？

正如我前面暗示的，秘密就在陡峭的山坡上。让鸡肉在充满逐渐冷却的水的封闭锅中完成烹饪，可以使肉汤味道最大化，这反过来为鲜嫩可口的肉提供了完美的环境。浸泡整只鸡还能确保鸡体上留下的任何东西都有足够的营养供储存。这就是为什么我在炉子上慢慢煮鸡，而不是在我的 3 夸脱速溶锅里快速煮鸡，那太小了，不能满足我对肉汤的渴望。(不过，它确实是很好的原料。)有些好事是需要时间的，水煮鸡绝对是其中之一；吃了几顿饭后，我想你会同意的。
