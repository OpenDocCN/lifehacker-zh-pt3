# 谷歌 Pixel 事件中最重要的 5 个公告

> 原文：<https://lifehacker.com/the-5-most-important-announcements-from-googles-pixel-e-1829638763>

谷歌的活动通常不会像苹果那样声势浩大，但公告对爱好者和任何喜欢闪亮新事物的人来说都一样诱人。今年，在纽约市举行的一次活动中，该公司展示了其新的 Pixel 手机，这是 Pixel 平板电脑(现在装有 Chrome OS)的翻版，一款智能显示屏，以及一系列其他设备，它甚至没有提到过(尽管我们会告诉你)。

### 有两款新的 Pixel 手机

第三代 Pixel 3 和 Pixel 3 XL 是在数月的谣言传播和未经证实的硬件泄露后宣布的。这两款设备的屏幕与机身的比例都比它们的前辈更大，此外还有玻璃前后侧和边对边显示屏。Pixel 3 配备了 5.5 英寸的有机发光二极管显示屏(比去年的更大)，而 Pixel 3 XL 配备了 6.3 英寸的有机发光二极管显示屏(尺寸有所增加)。

Pixel 3 和 Pixel 3 XL 运行在几乎相同的硬件上。这两款设备都运行在 4GB 内存的高通骁龙 845 处理器上。它们都有 64 或 128GB 的存储空间。没有 microSD 卡插槽，也没有耳机插孔，但像素现在有无线 Qi 充电，防水等级为 IP68。Pixel 3 还配备了 2915 毫安时电池(与去年相比没有太大变化)，而 Pixel 3 XL 的电池为 3430 毫安时(小于去年)。

Pixel 3 和 Pixel 3 XL 也共享相机硬件。前置摄像头是一个 800 万像素的双摄像头，具有不同的光圈，有助于拍摄情绪化的人像照片。主摄像头是一个 1220 万像素的摄像头，具有光学图像稳定功能(OIS)，能够实现一些巧妙的功能，我们将在稍后访问谷歌的机器学习公告时介绍这些功能。(如果你想向前跳，向下滚动到最后。)

### 有新的 chrome book——呃，平板电脑？

很难判断 Pixel Slate 是打算成为 Chromebook 还是平板电脑，但你需要知道的是，它能够扮演这两种设备的角色。Pixel Slate 是一款采用 Chrome 操作系统大脑的平板设备——它可以嵌入一个单独出售的 Pixel Slate 键盘([【199 美元](https://store.google.com/product/pixel_slate_keyboard) )，内置触控板，或者你可以单独携带它在 12.3 英寸的分子显示屏上观看视频，用广角前置摄像头进行视频通话，或者用单独出售的 Pixelbook 笔([【99 美元](https://store.google.com/product/google_pixelbook_pen) )在上面画画。

Pixel Slate 还拥有双前置扬声器、指纹传感器和两个 USB-C 端口，尽管后者是必要的，因为没有耳机插孔。Pixel Slate 即将上市，将采用英特尔酷睿 m3、i5 或赛扬处理器，以及 4GB、8GB 或 16GB 内存。它的存储容量高达 256GB，起价为 [、599 美元](https://store.google.com/product/pixel_slate) 。

### 和一个新的谷歌主页中心

联想和 JBL 并不是唯一使用谷歌助手生产智能显示屏的公司。该公司宣布了它的版本，谷歌 Home Hub，它被称为“最好的相框”。它的 7 英寸显示屏比一些第三方产品要小一点，而且它没有用于双人视频通话的摄像头。该设备的模型与现有的谷歌家庭设备系列相匹配。

Google Home Hub 应该放在公共区域或者你的卧室里。它具有环境均衡器模式，可以根据一天中的时间来调暗屏幕保护程序，这样它就不会在你面前发出刺耳的声音。由于它是一个中枢，它带有一个叫做 Home View 的主屏幕，为所有人提供智能家居状态控制。它的价格为 [、150 美元](https://store.google.com/config/google_home_hub) ，有四种颜色可供选择:粉笔色、炭灰色、浅绿色和沙色。

### 也有一些新的配件

在谷歌发布完 Pixel 3、Pixel Slate 和 Google Home Hub 的细节后，它更新了其在线商店，以包括所有这些，甚至更多。

有一对新的 Pixel USB-C 耳塞，这不是我们去年希望的 Pixel 耳塞的升级版，尽管它们只需 30 美元，并提供谷歌助手的按钮访问。还有谷歌 Pixel 支架，这是为 Pixel 3 制造的，现在它支持无线充电。更不用说，该公司悄悄发布了新版本的 Chromecast 加密狗，支持与其他谷歌智能设备同步的多房间音频。

### 机器学习内置于一切之中

最后——但肯定不是最不重要的，因为谷歌无疑已经在很大程度上建立了机器学习帝国——在所有关于规范和硬件能做什么的讨论中，谷歌设法对其助手进行了一些增强。由于谷歌助手横跨几个产品类别，这里有一个即将到来的快速品尝。

Pixel 3 相机有几个新的机器学习技巧，包括 **Top Shot** ，它拍摄一组突发图像，然后使用机器学习比较这一批图像，挑选出最佳镜头。还有**超分辨率变焦**、，它可以拍摄一组照片，从而可以更准确地放大照片，而不是仅仅依靠光学变焦，还有一个叫做**夜视**的功能，它的功能听起来像是:帮助照亮一个黑暗的环境。该算法帮助它计算出场景的适当曝光，然后在必要的地方填充空白。对于喜欢自拍的人来说，Pixel 3 的照片亭模式会在你微笑时拍照，不需要点击屏幕。

Google Home Hub 精确地采用了机器学习，因此它可以宣称是有史以来最好的数字相册。它引入了一个名为 **Live Albums、**的新功能，在这里你可以选择你想在幻灯片中出现的人，然后谷歌会处理剩下的事情。它使用面部识别技术来抓取它认识的人的照片，并将其放在共享相册中，你可以通过 Home Hub 传输这些照片。相册会随着从“相机胶卷”上传的每个相关文件而更新。还有一个名为**最近亮点**的功能，它会显示你的 Google 相册账户中最近最好的照片。因为在它后面有一台机器已经充分地吸收了收据的样子，你不必担心昨天的杂货账单会出现在数据流中。

甚至 Pixel 3 上的拨号器也内置了机器学习。在被称为谷歌 Duplex 的正式亮相中，这款手机将能够筛选你的来电，并告诉垃圾邮件发送者离开，而不必忍受与任何人交谈的情绪劳动。

在谷歌硬件活动的所有声明中，有一点是肯定的:谷歌活动主要不是关于它推出的硬件，而是机器学习可以用这些硬件做什么。对于许多消费者来说，将日常任务浓缩成几个按键或一个语音命令的能力物有所值。
