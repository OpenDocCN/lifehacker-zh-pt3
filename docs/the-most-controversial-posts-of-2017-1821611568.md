# 2017 年最具争议的帖子

> 原文：<https://lifehacker.com/the-most-controversial-posts-of-2017-1821611568>

在互联网上不可能让每个人都满意，但有时你会触动网上的神经，事情会变得有点激烈。以下是 2017 年让你热血沸腾的一些帖子。



### [**这种高耸、蓬松煎饼的秘密是蛋黄酱**](https://skillet.lifehacker.com/the-secret-to-this-towering-fluffy-pancake-is-mayonnai-1820351459#_ga=2.87966616.1246647813.1514324407-1667049540.1510952795)

没有一种调味品会像被称为“蛋黄酱”的乳液那样引起如此尖刻和厌恶，尤其是当你建议它被用于“正常”应用之外的时候。因此，这个煎饼黑客激怒了评论者和喜剧演员，我觉得这是不相称的。

### [**伙计们，停止“斗笠”来隐藏你们的秃顶点**](https://lifehacker.com/guys-stop-hatfishing-to-hide-your-bald-spot-1798363364#_ga=2.67021614.1246647813.1514324407-1667049540.1510952795)

这是非常丰富的，来自一个有着一头漂亮头发的男人，他还戴着许多帽子 (我可以补充说，非常好)。

### [**为什么你想成为感恩节带来沙拉的英雄**](https://skillet.lifehacker.com/why-you-want-to-be-the-hero-who-brings-salad-to-thanksg-1820194832)

### [**让我们都停止说祝福你**](https://lifehacker.com/lets-all-stop-saying-bless-you-1819891678#_ga=2.63875887.1246647813.1514324407-1667049540.1510952795)

喷嚏并不特别，尼克·道格拉斯希望我们不要这样对待它们。这让你们很难过，上帝保佑你们。

### [**如何应对工作中的抱怨**](https://lifehacker.com/how-to-deal-with-mansplaining-at-work-1803122874)
