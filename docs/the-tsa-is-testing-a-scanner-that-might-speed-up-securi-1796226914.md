# 美国运输安全管理局正在测试一种可以提高安检速度的扫描仪

> 原文:[https://life hacker . com/the-TSA-is-testing-a-scanner-that-may-speed-up-securi-1796226914](https://lifehacker.com/the-tsa-is-testing-a-scanner-that-might-speed-up-securi-1796226914)

美国运输安全管理局终于更新了它几十年的安全系统。周四，它开始在凤凰城测试手提行李的计算机断层扫描(CT)扫描仪。

Watch

由于 CT 扫描仪，乘客现在可以在通过机场安检时将他们的技术和液体留在他们的包里，这意味着排队速度将加快 50%。

除了让每个人都感觉自己有 [、美国运输安全管理局预检](http://lifehacker.com/tsa-pre-open-to-all-breeze-through-airport-security-fo-1567274913) 之外，CT 扫描通过显示手提行李中内容的 3D 图像来提高检测能力。图像可以在不打开袋子的情况下旋转和检查，比 TSA 目前的 X 射线扫描仪提供的图像更详细。

第二台机器将于本月晚些时候在波士顿进行测试。三月，DHS [宣布禁止](http://lifehacker.com/brace-yourself-there-s-about-to-be-an-electronics-trav-1795129082#_ga=2.148831797.303451680.1497830696-1887250931.1497314809) 在北非和中东 10 个国家机场的航班上使用笔记本电脑和平板电脑，此前有报道称恐怖分子正在研发伪装成笔记本电脑电池的炸弹。DHS 考虑将 [禁令扩展到欧洲](http://lifehacker.com/brace-yourself-there-s-about-to-be-an-electronics-trav-1795129082#_ga=2.204338322.303451680.1497830696-1887250931.1497314809) 但是后来 [决定反对](https://consumerist.com/2017/05/30/homeland-security-wont-expand-laptop-ban-to-flights-from-europe-for-now/) 。

据美联社 报道，CT 扫描已经被用于托运行李，但由于机器的成本和体积过大，它们不用于随身携带的行李。但是美国运输安全管理局希望更新更小的机器，像凤凰城的那种，将提高违禁品的检测，同时加快安检速度。虽然它们更小，但同样有效。

也是时候了。x 光技术已经用了十几年了，甚至都没有效果。2015 年的一份内部审查发现，美国运输安全管理局的官员无法发现 95%的卧底调查员偷偷通过安检的假武器(yikes)。

这些新机器可以测量物品的密度，这有助于安全部门更好地检测哪些物品可能是爆炸物。如果测试成功，TSA 还可能放松对液体的限制——你最终可以携带超过 3.4 盎司限制的液体和凝胶。

希望这行得通，因为我们已经厌倦了排长队等待的痛苦。
