# 如何在 Windows 10 中获得苹果的“快速查看”功能

> 原文:[https://life hacker . com/how-to-get-apple-quick-look-feature-in-windows-10-1827277007](https://lifehacker.com/how-to-get-apples-quick-look-feature-in-windows-10-1827277007)

即使你是世界上最大的微软粉丝，你也不得不承认苹果的“ [快速浏览](https://support.apple.com/kb/ph25575?locale=en_US) ”功能对 macOS 来说相当方便。如果你是一个纯粹主义者，以至于在过去十年左右的时间里，你甚至没有接触过 Mac，这里有一个简单的介绍:你点击一个文件。你按下空格键。项目的预览会弹出，如照片、PDF 的内容等。这是一个很好的快速浏览东西的方式，而不用浪费时间加载一个真正的应用程序。

Watch

不幸的是，Windows 10 的操作系统中没有内置这样的技巧。然而，从 Windows 商店快速下载一次，你就可以像你的 Mac 朋友一样预览(大多数)文件。

这款 Windows 应用被巧妙地命名为“[”quick look](https://www.microsoft.com/en-us/p/quicklook/9nv4bs3l1h4s)，”完全免费。安装它，运行它，你将能够预览“某些文件类型”——开发者没有指定——通过在文件资源管理器中高亮显示它们并按空格键打开预览。(不要按住；轻点它。)并再次点击空格键关闭预览。就这么简单。每当你启动 Windows 时，这款应用就会自动启动，这是其主要功能发挥作用的必要条件。

打开 QuickLook 后，您还可以将窗口固定在其他所有文件的顶部，单击任何其他文件都会自动弹出预览。如果你有一大堆文件想要浏览，这是非常方便的。

使用较低级的 [Windows 10 S](https://lifehacker.com/surface-laptop-owners-can-still-upgrade-windows-10-s-to-1800987251) 操作系统的人要注意了。QuickLook 不会为您做任何事情，所以不要安装它。还有， [已经更新你的操作系统](https://support.microsoft.com/en-us/help/4023898/windows-10-switching-out-of-s-mode) 。然后安装 QuickLook，享受它的有用性。
