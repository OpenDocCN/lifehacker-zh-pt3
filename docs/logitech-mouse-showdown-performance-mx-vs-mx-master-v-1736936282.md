# 罗技鼠标对决:性能 MX vs. MX Master vs. G502 Proteus Core

> 原文：<https://lifehacker.com/logitech-mouse-showdown-performance-mx-vs-mx-master-v-1736936282>

曾几何时，罗技的 MX Revolution 是围绕 最受欢迎的 [鼠标，理由很充分:它很舒适，可定制，它的“动量滚动”是救命稻草。现在，它有三个继任者，都有点不同:性能 MX，](http://lifehacker.com/five-best-desktop-mice-5931795) [新的强大的 MX 主](http://lifehacker.com/the-logitech-mx-master-a-fitting-successor-to-one-of-o-1713362467) ，以及 [专注于游戏的 G502 Proteus 核心](http://kotaku.com/logitechs-customizable-12-000-dpi-gaming-mouse-1592743267) 。以下是他们的对比。



### 竞争者

罗技有一个庞大的产品线，但当谈到他们最好的东西时，这三个拿走了蛋糕:

*   [**罗技性能鼠标 MX**](http://www.logitech.com/en-us/product/performance-mouse-mx)**([【50 美元亚马逊](http://www.amazon.com/Logitech-Wireless-Performance-Mouse-Mac/dp/B002HWRJBM/?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/logitech-mouse-showdown-performance-mx-vs-mx-master-v-1736936282&asc_source=&tag=kinjalifehackerlink-20) )是 MX 革命的第一个继承者，并保留了它的大部分最佳功能。它非常受欢迎，罗技公司已经生产了六年，没有任何变化。**
*   **[**罗技 MX Master 无线鼠标**](http://www.logitech.com/en-us/product/mx-master)**([【92 美元在亚马逊](http://www.amazon.com/Logitech-Master-Wireless-Mouse-910-004337/dp/B00TZR3WRM/?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/logitech-mouse-showdown-performance-mx-vs-mx-master-v-1736936282&asc_source=&tag=kinjalifehackerlink-20) )是 MX 系列的最新产品，也是性能 MX 的继任者。它采用了更新的设计、电池和一些新功能，是罗技希望您在性能 MX 最终消亡时使用的鼠标。****
*   ****[**罗技 G502 Proteus Core 游戏鼠标**](http://gaming.logitech.com/en-us/product/g502-proteus-core-tunable-gaming-mouse)([【76 美元，亚马逊](http://www.amazon.com/Logitech-Proteus-Tunable-Customizable-910-004074/dp/B00IRHE892?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/logitech-mouse-showdown-performance-mx-vs-mx-master-v-1736936282&asc_source=&tag=kinjalifehackerlink-20) )从技术上来说不在 MX 系列中，但它与你想象的非常相似。它拥有与 Performance MX 相同的按钮和功能，甚至在你手中的感觉也很相似，尽管有更多“游戏玩家”关注的美学和软件。****

****首先值得注意的是，不幸的是，所有这些鼠标都只适用于右撇子。罗技确实有几款 [左撇子](http://smile.amazon.com/dp/B000EXRSWG?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/logitech-mouse-showdown-performance-mx-vs-mx-master-v-1736936282&asc_source=&tag=kinjalifehackerlink-20) 和 [双手通用](http://smile.amazon.com/Logitech-Daedalus-Performance-Gaming-910-004380/dp/B00TKFD51M?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/logitech-mouse-showdown-performance-mx-vs-mx-master-v-1736936282&asc_source=&tag=kinjalifehackerlink-20) 鼠标，但这三款王者都是为右撇子打造的。****

****这些鼠标的相似之处多于不同之处，但如果你很难决定买哪个，我们会帮你解决。****

### ******观感******

****性能 MX(左)感觉就像是为你的手量身定做的。对于喜欢将整只手放在鼠标上的 [手掌抓手](http://lifehacker.com/how-to-choose-the-perfect-mouse-and-keyboard-5865351#_ga=1.173202595.127976026.1441054335) 来说再合适不过了。即使你的手比较小，这款较大的鼠标也会很适合它们。**** 

****MX Master(中)比 Performance MX 高一点，也是一个 palm gripper 的鼠标，尽管无论握在手中感觉都很好。罗技肯定是想让你以特定的方式使用这个鼠标，你一把手放上去就知道了。MX 的平滑曲线被现代的角度所取代，占用的空间更少，但使用起来仍然很舒适。它也比 MX 重一点，但它在您的桌面上占用相同的空间。****

****Proteus Core(右)的尺寸与 Performance MX 相似，但具有独特的“游戏”外观:坚硬的边缘、LED 灯和侧面的蓝色强调。它的底部还有一组重物，你可以移动或重新配置，使鼠标更重、更轻，或将重量转移到鼠标的不同侧面。这真的很好，因为它本质上让你自定义鼠标的重量和感觉到你的愿望。****

### ******按钮******

****Performance MX 在按钮方面非常简单，但拥有所有重要的按钮。它有一个左右点击，一个上下左右移动的滚轮，侧面有后退和前进按钮。它还有一个隐藏的“拇指”按钮，非常适合 Windows 中的任务视图或 OS X 中的 Mission Control/Exposé。还有一个非常小的“缩放”按钮。所有的按钮都可以通过罗技的软件进行编程。****

****MX Master 拥有 Performance MX 所拥有的所有相同按钮，但有一些改进。拇指架下的按钮现在是手势敏感的，可以做一些事情，如最大化窗口或在 windows 中向左或向右移动，或者在 OS X 中切换 Mission Control 和 exposé。MX Master 最棒的事情之一是从 MX Revolution(高性能 MX 的哥哥)中回归拇指轮。它非常适合浏览文档和滚动电子表格。后退和前进按钮就在它的后面。根据你的手，指轮和按钮很舒适，很容易够到，或者你必须将手向后拉一点，才能将拇指放在上面。对我来说，它们的位置非常好。****

****Proteus Core 在 MX 布局上增加了几个按钮，但不会太多。其实它只比性能 MX 多了三个。它有后退和前进按钮，就像 MX 鼠标一样，还有一个拇指按钮，用于在游戏中狙击(这比 MX 的拇指按钮稍难触及)。然而，与 MX 鼠标不同，它还有两个 DPI 按钮和一个未编程的“G9”按钮。所有的按钮都可以通过罗技的软件定制。****

### ******有线与无线******

****Performance MX 仅作为无线鼠标提供。从技术上讲，你可以用 USB 线把它插上，但你仍然需要插上无线接收器。如果你更喜欢无线鼠标，你会很高兴使用它，但它也可能是一个缺点——毕竟，你必须担心充电问题，我发现许多无线鼠标在以后的生活中都会遇到电源或连接问题。****

****MX Master 也是无线的，但它为混合添加了蓝牙支持。它还可以通过 USB 连接充电和运行，但你不需要长时间保持连接——罗技声称你可以在四分钟的充电时间内获得一整天的电量，我发现这是真的。同样值得注意的是，MX Master 还可以通过统一接收器或蓝牙同时“连接”三个设备。要切换活动设备，只需按下鼠标底部的一个按钮。这非常适合在家里(或在工作和家里)放一个鼠标，用来控制多台电脑。****

****另一方面，Proteus Core 只是一个有线鼠标。不过，我对此非常满意；它有一个很好的长编织电缆，感觉不像它以任何方式拖拉或阻碍鼠标。在讨厌有线鼠标几十年后，Proteus 核心改变了我。****

****请注意， [罗技 G602](http://smile.amazon.com/Logitech-Wireless-Gaming-Mouse-Battery/dp/B00E4MQODC?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/logitech-mouse-showdown-performance-mx-vs-mx-master-v-1736936282&asc_source=&tag=kinjalifehackerlink-20) 虽然不能直接替代 Proteus Core，但有点类似无线鼠标。****

### ******软件和定制******

****Performance MX 和 MX Master 使用罗技的“罗技选项”软件，该软件允许你定制指针和滚动速度、平滑滚动、滚轮模式(自由旋转或棘轮)，甚至是当你移动拇指或滚轮时应用程序滚动的默认方向(对于那些出于某种原因想要向下滚动滚轮以向上翻页的人来说)。您还可以获得常见的按钮定制选项，因此您可以将 Performance MX 和 MX Master 的每个按钮重新映射到几乎任何您喜欢的东西，甚至是按键或系统功能，如在 Mac 上打开 Launchpad 或在 Windows 上打开“开始”菜单。****

****除了按钮编程、滚动选项和指针灵敏度，仅此而已。罗技选项是准系统，虽然你可以用它来管理多个罗技鼠标，但你不能录制复杂的宏，为工作和家庭，或工作和娱乐设置不同的“配置文件”，也不能将这些配置文件保存到鼠标上——它们存储在你的 PC 上，本地。****

****Proteus Core 使用了一个略有不同的程序，名为“罗技游戏软件”，功能明显更强大。罗技的游戏软件可以让你像罗技选项一样对功能按钮和按键进行编程，还可以让你通过点击按钮来录制复杂的宏。它还可以在鼠标本身(而不是在你的电脑上)存储配置文件，“调整”鼠标到你的鼠标垫或桌面，分析你按下某些按钮的频率，并打开“角度捕捉”以获得更直的线条。所有这些特性都将 Proteus core 变成了一个非常棒的生产力工具，即使它们原本是为了游戏。****

### ******其他额外功能******

****性能 MX 和 Proteus Core 有我一直最喜欢的鼠标功能:动量滚动。滚轮像任何其他鼠标一样正常滚动，通过一系列点击滚动一定数量的行。然而，按下轮子下的按钮，它就会切换到自由旋转模式。轻弹它，它会继续以极快的速度旋转，直到你停下来，这对于滚动回大型网页或文档的顶部来说是非常棒的。没有鼠标的这个特性，我活不下去。(在:37 秒处查看上面的视频，看看它的实际效果。)****

****MX Master 通过重新引入 MX Revolution 的滚动自动检测功能，进一步完善了这一功能。是的，你仍然可以用滚轮下面的按钮手动切换动量滚动的开和关，但现在如果你足够努力地滚动并放开，MX Master 会自动切换到动量滚动模式。当滚轮慢下来时，它会点击回到手动滚动模式。完全不用改变习惯，两全其美。****

### ****三只老鼠都是冠军****

****这不是那种真正有“赢家”的摊牌——三只老鼠都很棒，任何一只都很难出错。Performance MX 和 Proteus Core 在功能方面惊人地相似，两者之间最大的决定因素是你想要的外观，你更喜欢有线还是无线，以及你的预算。(尽管 Proteus Core 的额外按钮、更强大的软件和可定制的重量确实给了它一个小优势——价格也相应增加了。)****

****另一方面，MX Master 是性能 MX 的合适升级，但价格昂贵。它的功能更加丰富，但是这些功能对你来说是否重要就是另一回事了。当您考虑 MX 时，请考虑您是否需要蓝牙或在多台电脑上使用鼠标的选项。如果 Performance MX 是您的完美鼠标，您可能不会觉得需要 MX Master 的新功能，但当您拥有它们时，它们真的很好，一旦您拥有了它们，您肯定会依赖它们。****
