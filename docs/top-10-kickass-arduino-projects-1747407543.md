# 十大顶级 Arduino 项目

> 原文：<https://lifehacker.com/top-10-kickass-arduino-projects-1747407543>

Arduino 是一种廉价的电子板， [允许你在没有大量编码经验的情况下制作自己的电子产品](https://lifehacker.com/how-to-start-making-your-own-electronics-with-arduino-a-5875365) 。我们喜欢 Arduino，但像任何电子项目一样，想出建造什么的想法是困难的。无论你只是在寻找灵感还是需要一个开始的地方，让我们来看看十个最酷的 Arduino 项目。

Watch

### 10.制作一个微型天气显示器

想要一种简单的方法来跟踪天气吗？ [这个 Arduino 项目](http://lifehacker.com/build-your-own-tiny-automatically-updating-weather-dis-1734268329) 使用一个 Adafruit HUZZAH 板(运行 Arduino)和一个小有机发光二极管图形显示器来显示当前天气、今天的高温和明天的天气。另外，它很小，每个人都喜欢微小的电子产品。

### 9.构建 MIDI 控制器

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-bM7NN-fiMXs&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-bM7NN-fiMXs&start=0) 

MIDI 控制器是使用物理硬件控制电脑上不同声音的好方法。这是一项非常古老的技术，你可以在任何音乐商店买到各种各样很酷的小 MIDI 控制器。但是如果你不想出去买一个 MIDI 控制器并和其他人一样，你可以用一个 ArduinoT3[来制作你自己的。如果你想让它看起来像这个一样时髦，你需要 3D 打印自己的外壳，但除此之外，它是由现成的组件制成的。一旦你做好了，你就可以通过 USB 控制你所有的节拍、哔哔声和哔哔声。](http://lifehacker.com/build-your-own-16-button-midi-controller-with-an-arduin-1614628540)

### 8.在你的车库门开启器上安装指纹扫描仪

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-uK8gm9a0k-g&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-uK8gm9a0k-g&start=0) 

车库开门器很无聊。如果你想增加一点趣味，把 [接入指纹扫描仪](http://lifehacker.com/make-a-diy-fingerprint-scanning-garage-door-opener-1595544228) 来触发开门。这可能并不奇怪，这是一个相当复杂的项目，但最终的结果是如此之酷，令人难以拒绝。如果这对你来说有点太多，你可以用 Arduino 和智能手机来控制那个 [车库门，而不是](http://lifehacker.com/build-a-smartphone-connected-garage-door-opener-with-an-1563008120) 。

### 7.用自动触发喷枪让你的猫远离柜台

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-bSz4FQPsir8&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-bSz4FQPsir8&start=0) 

有时候，最好的 Arduino 项目也是最可笑的。比如这个 [动作侦测水枪](http://lifehacker.com/this-auto-trigger-spray-bottle-pranks-friends-keeps-ca-1555621712) 。这个想法很简单:当它检测到运动时，喷枪就会启动。这是让猫远离柜台或家具的完美方法。

### 6.制作您自己的 Arduino

Arduino 的美妙之处在于它是开源的，这意味着你可以很容易地用现成的部件制作自己的产品。这个只需要价值约 5 美元的零件 ，当你完成建造后，你会对 Arduino 的工作原理有更好的理解。如果这个特别的项目对你来说太大了， [这个是袖珍的](http://lifehacker.com/make-your-own-pocketable-arduino-kit-5926582) 。

### 5.在床下添加运动触发夜灯

没有人喜欢在半夜起床时撞到东西，但打开床头灯也不是一个很好的体验。这个项目解决了这个问题 通过在床下挂上 LED 灯，然后用运动控制器和 Arduino 触发它们。这样起来就不用担心弄瞎自己了，但还是不会什么都碰到。

### 4.将电视上的任何短语静音

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video--SzB5OQUcOU&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video--SzB5OQUcOU&start=0) 

厌倦了金·卡戴珊、星球大战或唐纳德·特朗普？ [“已经够了”通过监控隐藏字幕检测到任何单词或短语时，会使您的电视](http://lifehacker.com/the-enough-already-mutes-your-tv-when-overexposed-celeb-5831884) 静音。只要该单词或短语不再被提及，它将在 30 秒后取消静音。你可以设置它屏蔽任何你想屏蔽的短语，让看电视变得更加愉快。

### 3.为您的液晶显示器添加一个流光溢彩传感器

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-rEVSjfAhldA&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-rEVSjfAhldA&start=0) 

给你的液晶显示器增加一点背光是让你看电影更有身临其境感的好方法。你可以用一个 Arduino 添加你自己的 [，最终结果就是一个你自己构建的杀手级电影观看系统。](https://youtu.be/rEVSjfAhldA)

### 2.制造一个机器人手臂

如果你真的想在一个项目中学习大量不同的 Arduino 相关技能， [机械臂是一个开始](http://lifehacker.com/build-a-kickass-robot-arm-the-perfect-arduino-project-1700643747) 的好地方。你将学习编程、建筑基础、实验板等等。此外，你还可以得到一个很酷的机器人手臂。

### 1.创建功能齐全的计算机控制面板

有没有想过像科幻电影里的控制面板一样使用你的电脑？Reddit 用户 smashcuts 做到了， [这就是为什么他自己建了一个](http://lifehacker.com/build-your-own-fully-functional-computer-control-panel-1725629368) 。正如你从图片中看到的，这个项目中发生了很多事情，但 Arduino 是这一切背后的大脑。除此之外，就是设置一些 USB 控制器，一些灯，并添加必要数量的开关来控制从屏幕偏好到音量的一切。
