# 太多的大麻会导致恶心和呕吐

> 原文：<https://lifehacker.com/too-much-weed-can-cause-nausea-and-vomiting-1825238436>

杂草以其缓解恶心的能力而闻名，但对于大量吸烟的人来说，它可能会产生相反的效果。这种鲜为人知的情况被称为(CHS)，包括一阵阵的极度恶心、呕吐和胃痛，可以通过热水淋浴暂时缓解。



科学家们几年前才知道这种情况，而且这种情况还很罕见。但这也很难诊断:许多医生不知道这一点，许多大麻使用者很难相信他们使用多年的东西会突然引发问题。

一项 [最近的研究](https://www.ncbi.nlm.nih.gov/pubmed/29327809) 发现，CHS 可能比我们意识到的更普遍。研究人员给那些因为其他原因在纽约市急诊室的人发了一份调查问卷，而*目前没有感到恶心。(许多疾病都会导致恶心，所以他们想排除人们患有其他疾病的可能性。如果你有先天性心脏病，它往往来来去去。)*

在 155 名填写调查并每月至少使用大麻 20 天的患者中，33%的人说他们发现热水淋浴是缓解恶心的有效方法。如果这些数字被证明代表了大麻使用者的总体情况，这意味着美国可能有几百万人经历过社区卫生服务，而他们中的大多数人可能对此一无所知。

我们不知道 CHS 的确切病因，也不知道为什么热水澡有帮助。我们的消化系统包含内源性大麻素的受体，这是大麻模仿的人体天然化学物质。与此同时，大脑中控制体温的区域下丘脑也有大麻素受体——所以该大脑区域和热水淋浴缓解之间可能有联系。

如果你认为你有先天性心脏病，去检查一下——你的恶心可能是由其他原因引起的。但是如果真的是大麻的问题，唯一已知的治疗方法就是停止使用它。你可以在这里 阅读 [更多关于大麻素呕吐综合症的内容。](https://www.saintlukeskc.org/health-library/cannabinoid-hyperemesis-syndrome)
