# 火星接近桌面

> 原文:[https://lifehacker.com/the-mars-approach-desktop-1742241710](https://lifehacker.com/the-mars-approach-desktop-1742241710)

这个拱形 Linux 桌面有一张火星在黑暗中漂浮的美丽照片，但这并不是它有趣的地方——透明的终端窗口和资源指示器完善了实际上非常高效的设置。以下是如何让它成为你自己的。

Watch

这个桌面大量使用了特定于 Linux 的小部件，所以如果你想让它在 Windows 中工作，你需要做一些调整，但是如果你运行的是 Linux(或者 OS X，就此而言)，你需要做的是:

*   [红色星球壁纸](http://www.solarsystemscope.com/nexus/wallpapers/red_planet.php)
*   [Conky](http://conky.sourceforge.net/) ，一款针对 Linux 的轻量级系统监视器
*   [Ncmpcpp](https://wiki.archlinux.org/index.php/Ncmpcpp) 用于屏幕左上角的音乐播放器和可视化工具
*   [终结者](http://gnometerminator.blogspot.com/p/introduction.html) 为桌面上的透明终端窗口
*   [Openbox](http://openbox.org/wiki/Main_Page) 作为窗口管理器
*   [XCompmgr](https://wiki.archlinux.org/index.php/Xcompmgr) 用于投影效果和窗口透明度

虽然没有看起来那么多，但最终的结果是一个漂亮的终端式桌面，坦率地说，它让我想起了我的旧 Linux 机器。如果你喜欢它，或者对如何像这样设置自己的网站有疑问，请访问下面链接的 Flickr 页面，提出你的问题或者赞美你！
