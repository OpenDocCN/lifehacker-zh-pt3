# 你应该把食物储存在透明的容器里

> 原文：<https://lifehacker.com/you-should-store-food-in-clear-containers-1828636970>

我爱我的老式冰箱到烤箱 Pyrex 菜肴。它们放在桌子上、冰箱里和照片里看起来都很棒，但它们有一个小问题:我很少会吃那些装在彩色可叠放的玻璃长方形里的剩菜。

Watch

原因其实很明显。引用美食家天才戴夫·阿诺德(去年年底他出现在我们的播客 )的话:“你看不到它。如果你看不见它，它就不存在，如果它不存在，你就不吃它。”人们先用眼睛吃饭，你的眼睛最容易被冰箱里看起来像食物的东西吸引，而不是不透明的形状。这就是为什么将剩菜储存在透明包装中——想想带塑料盖的玻璃碗、泥瓦罐和阿诺德喜欢的 [永远有用的盛汤容器](https://skillet.lifehacker.com/the-top-5-restaurant-supply-store-essentials-ranked-1797727937)——是最佳策略，如果你真的想*吃掉*你储存的食物。视觉模糊的存储系统基本上是死刑判决(对于最后一点鸡肉)。“如果你用铝箔把它包起来，扔掉它，”阿诺德解释说，“一旦它用铝箔包起来，就像被塞进冰箱，你在它上面放些别的东西，它就不见了。”
