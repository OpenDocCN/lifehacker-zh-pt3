# 用另一个啤酒瓶打开一个啤酒瓶

> 原文：<https://lifehacker.com/open-a-beer-bottle-with-another-beer-bottle-1827297354>

我们已经向您展示了如何用一把[打火机](https://lifehacker.com/open-beer-bottles-with-bic-lighters-5296623#_ga=2.216238227.448177410.1530548418-1297080755.1497980211)[订书机](https://lifehacker.com/open-a-soda-or-beer-bottle-with-a-stapler-5937811)[报纸](https://lifehacker.com/open-a-beer-bottle-with-a-newspaper-5915187#_ga=2.188320173.448177410.1530548418-1297080755.1497980211)[Mac 电源砖](https://lifehacker.com/mac-power-adapter-beer-bottle-apples-secret-church-5907389)[台面](https://lifehacker.com/how-to-open-a-beer-bottle-with-a-countertop-5624509) 、一个现在 Reddit 上的一个视频向你展示如何用啤酒瓶 打开啤酒瓶——或者，如果你聪明的话，只用一个瓶盖。



1.  把“开瓶器”瓶子倒过来。

2.  将瓶盖的盖子滑到你要撬开的盖子下面。
3.  像用开瓶器一样撬开它。
4.  当你喝完啤酒后，把瓶盖压回到空瓶上，它仍然可以作为一个开瓶器。

根据 Reddit 评论者的说法，这是德国打开啤酒的典型方式。但这需要一点学习，所以要小心，否则你会打开瓶子，把啤酒洒得到处都是。所以先清醒着试试。
