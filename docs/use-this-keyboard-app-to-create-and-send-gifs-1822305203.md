# 使用这个键盘应用程序来创建和发送 gif

> 原文：<https://lifehacker.com/use-this-keyboard-app-to-create-and-send-gifs-1822305203>

iOS/Android:你喜欢智能手机上的 [滑动友好键盘](https://lifehacker.com/ditch-apples-generic-keyboard-with-these-apps-1795587161) 吗？那么你可能听说过谷歌的 Gboard，这是一个第三方键盘，你既可以滑动来拼写，也可以搜索网页。有了最新的更新，你可以使用智能手机摄像头从键盘创建 gif，并将它们发送到任何欣赏动画图像的地方。



### **如何录制和保存 gif**

要打开 GIF 创建工具，打开你想要发布 GIF 的应用程序(或者至少是一个可以使用 Gboard 的应用程序)。打开键盘，在你输入任何东西之前，点击键盘上方的 Gboard 菜单栏中的 GIF 图标。这将启动 Gboard 的 GIF maker。你可以使用 Gboard 创建两种 gif——循环 gif 和快进 gif——如果你喜欢的话，可以保存它们以备后用。

要创建 GIF，请将手指按住快门按钮，直到完成录制。然后，您可以将动画存储到“相机胶卷”,或者将其复制并粘贴到您正在使用的任何应用程序中。

Gboard 不是唯一一个让 gif 易于访问的键盘，但它是少数几个让你不用退出应用程序就可以制作自己的 gif 的键盘之一。这是自恋的终极形式吗？创造一个你自己形象的循环图像发送给你的朋友和家人，他们无疑已经知道你在运动时的样子。也许吧。但这也很容易做到，让你离成为社会的动画反应图像库更近了一步。
