# 用韭菜给烧烤或蒸海鲜调味

> 原文：<https://lifehacker.com/use-leek-greens-to-flavor-grilled-or-steamed-seafood-1826810981>

韭菜很快成为我最喜欢的韭菜。它们不仅在煎炸 时发出奇怪的砰砰声，还可以用来给鱼和其他海鲜注入微妙、独特的味道。



*这是《与克莱尔一起吃垃圾》的一部分* [*，这是一个生活黑客系列，克莱尔·劳尔说服你将你的厨余转化为可食用的美味。*](http://lifehacker.com/tag/eating-trash-with-claire)

有两种方法可以利用葱属植物的风味传递能力。一种是在烹饪前将绿色蔬菜与你的带鳍或去壳的鱼一起密封在一个袋子里——我们正在谈论的是*——另一种是在烧烤时将它们用作保护床(一分钟后会详细介绍)。*

*首先，简单地把洗好的青菜放在一张羊皮纸或箔纸上，把鱼放在上面，像平常一样调味。我喜欢用两份芝麻油和一份酱油的混合物来刷我的头发，并涂上一些 [盐曲](https://skillet.lifehacker.com/shio-koji-helps-your-food-be-its-best-self-1826640401) 作为额外的量。柑橘轮、新鲜生姜、任何你喜欢的香草和胡椒片也是不错的选择，但是你可以随意尝试。将整个包装密封起来，在 375 华氏度的烤箱中烤半个小时，或者每面烤五到七分钟(如果你用锡纸的话)，直到用叉子将它切成薄片。* 

*烧烤时，你可以用叶子来防止食物烧完时烧焦，厨师亚伦·桑切斯在这里描述了。拿一些箔片，用它在烤架上划线，在箔片上铺一层韭菜，然后把你的食物放在蔬菜上。绿色植物会温和地给食物调味，同时防止食物烧焦。*
