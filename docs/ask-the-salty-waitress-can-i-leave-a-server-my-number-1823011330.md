嘿杰克，

你在勾引我吗？哦，你只是在假设。没错。这些天我一直都是这样。

Watch

你的问题——就像所有与心灵有关的事情一样——很复杂，取决于你具体情况的细节。

我的第一个问题是:你是兔子杰西卡小姐工作的地方的常客吗？如果是这样的话，你会比非常客失去更多。如果她通常在你在的时候工作，如果她不给你发短信，事情会变得尴尬，或者如果她发了，事情会变得更尴尬。

你可能还记得我对 [的回答:“你会和常客上床吗？”](https://thetakeout.com/ask-the-salty-waitress-would-you-ever-sleep-with-a-reg-1821033292) ，我去过，重重地撞了一下，有点后悔。我因为一场混乱的一夜情失去了一个非常好的、稳定的顾客，而年长而明智的 Salty 知道要避免这种情况。

这并不是说你应该*永远不要*试图约一个服务器出去，如果你是常客的话，只是说你需要小心行事，想想什么是危险的。如果她不打电话，你会受伤吗？你会因为这件事而不去一家非常好的餐馆吃饭吗？大多数时候，我宁愿要一份美味的鸡肉三明治，也不愿要一个男朋友，所以我个人宁愿选择餐馆。

我的第二个问题是:最初是什么促使你约她出去？你了解她吗？她对你有兴趣吗？(注意:有礼貌并不表示有性欲。)如果这对她来说只是短暂的身体吸引，杰克，天涯何处无芳草，包括那些不在你常规午餐地点工作的鱼。

最后，我建议你仔细想想如何留下你的电话号码。尽可能简单地做，以免令人毛骨悚然:只要你的名字和电话号码。没有心，没有诗，没有“我已经看着你给我做金枪鱼三明治好几周了。”

我的一个同事告诉我，她曾经给一个可爱的酒保留了电话号码，那天晚上收到了一条礼貌的短信:“嘿，很荣幸你留了电话号码，但我实际上有女朋友了。晚安！”这大概是最理想的拒绝——没有受伤的自尊，没有尴尬。但是请记住，您的服务器的响应可能并不那么友好，如果它真的出现的话。
