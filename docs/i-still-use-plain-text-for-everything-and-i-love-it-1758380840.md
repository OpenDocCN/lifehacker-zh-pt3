### 纯文本迫使我保持事情简单

我喜欢整理东西。我喜欢格式化东西。我喜欢列出大纲、关键词和清单。这通常是一个有用的性格特征，但它也经常分散注意力，在写作、设置待办事项和记笔记时，完全不用*更少用*。

给我一个带有标签、颜色选择和层次结构的待办事项应用程序，我会给你一个小时的时间来组织像购物清单这样简单的事情。给我多个笔记本记笔记，我会给你我生命中建立一个组织系统的日子。

然后，我将继续实际上不使用任何这些系统。

尽管我热爱组织，并在生活的许多方面受到它的吸引，但像基本笔记或待办事项之类的东西对我来说并不重要。重要的是简单。我想快速打开一个应用程序，写下我需要的东西，然后关闭它，而不去想它应该得到什么标签，要添加什么格式，或者其他什么。给我一张空白纸和一个闪烁的文本图标。

没有那些组织技巧，我如何找到我要找的东西？记住，这是纯文本。这是一种简单的文件格式。Control + F，在应用程序或 Mac 上的 Spotlight 中的搜索栏中键入几个词，它就会显示出我所需要的内容。当然，我也有更高层次的组织。我所有的 Lifehacker 帖子想法都在一个文件里。另一个是待办事项。另一种是虚构的想法。你明白了。它很简单，但有点混乱，这正是我喜欢它的地方。

### 我可以把我所有的创意都放在一个地方

真正让我坚持使用纯文本的一点是，我可以用它做任何我想做的事情。我不必将我的大脑空间划分为不同的应用程序或服务。纯文本就像软件所能得到的一个小的创意笔记本。

即使当你使用像 [ToDo.txt](http://todotxt.com/) 、 [Sublime Text](https://www.sublimetext.com/) 或 [Simplenote](http://simplenote.com/downloads/) 这样的应用程序来管理纯文本文件时，事情也会变得混乱。你的系统 [永远不会像 Evernote](http://lifehacker.com/ive-been-using-evernote-all-wrong-heres-why-its-actual-5989980) 这样有条理。这正是我喜欢它的原因。我今天的待办事项就在一个我永远不会写的故事片的想法旁边。我有三年前做桌子的笔记，依偎在一个充满怪异梦境的文本文件旁。我的 Lifehacker 文章创意列表就在一本小说的结构注释旁边。我所有的想法，愚蠢的，聪明的，疯狂的，还有其他的都在一个文件夹里，就像一大堆文本一样。每一天我都看到了这一切，每一天我都想了很多，即使是在我不需要的时候。

正因为如此，我的大脑不断翻腾出好的和坏的想法。废弃的想法和完成的想法生活在同一个空间。正因为如此，我可以看到什么可行，什么不可行，以及我还没想好该怎么做。这对我太好了。当我关于视频游戏的愚蠢想法出现在我的购物清单旁边时，一切都变得可行。每一个想法都值得思考，想法都值得思考多次，即使我一开始也想不出该拿它们怎么办。

当然，你也不必那样全押。大多数人可能会发现这种类型的系统让人不知所措，但是纯文本的优点在于解决这种问题是多么容易。你可以创建文件夹或使用支持标记的应用程序，就像这样，你可以很容易地组织起来。

### 纯文本永远不会需要订阅，锁定功能，或倒闭

纯文本无处不在。它可以在任何操作系统和任何移动设备上运行，不管是谁制造的。各种各样的 app 都可以看。您永远不会遇到文件兼容性错误。你可以不假思索地把你写的东西从一个应用程序带到另一个应用程序。

这很重要，因为科技行业喜欢提醒我们，没有什么是永恒的。 [我们看到应用程序一直在关闭](http://lifehacker.com/how-to-move-on-when-a-service-you-love-shuts-down-5942544) 。 [他们追加了一笔订阅费](https://evernote.com/upgrade/?var=c&tier=premium) 。 [他们把你想要的那个功能锁定在付费墙后面](http://lifehacker.com/wunderlist-unveils-an-overhauled-ui-new-premium-tools-480724448) 。这很烦人，如果你投资了一个应用程序，不管是 notes 应用程序还是待办事项应用程序，你经常被迫为一堆你不想要的功能付出代价。纯文本不会遇到这个问题，因为它可以跨平台通用，更不用说是我们所知的计算的基础了。

同样，纯文本永远不会改变。一个应用程序可能会更新新的功能和新的用户界面，纯文本几乎总是纯文本。我永远不会打开一个应用程序去寻找一个我讨厌的新设计，或者一个我必须学习的新用户体验。文本编辑器可能会改变，但总会有另一个，而且它们永远不会都只订阅。这对我真的很重要。我每天都用纯文本来完成简单的任务。我不需要任何东西来妨碍我尽快捕捉文本。

### 我已经放弃了纯文本

当然，即使有 Simplenote 这样不错的管理应用，用纯文本保存所有内容也有点疯狂。为了简化事情，我在几个地方放弃了纯文本，转而使用应用程序。

对我来说最大的改变是食谱。我用食谱管理器辣椒粉 替换了半组织文本文件 [中大量随机拼凑的配料。食谱管理是纯文本的苦差事，辣椒粉让它变得有趣。](http://lifehacker.com/the-best-recipe-manager-for-iphone-1751574246)

类似地，我写的任何公开的长篇文章都要经过类似于 [【谷歌文档】](https://www.google.com/docs/about/)[微软 Word](https://products.office.com/en-us/word) 或 [尤利西斯](http://www.ulyssesapp.com/) (实际上只是普通文本，略有增强)，这取决于需要什么。即便如此，我通常还是会从纯文本开始，将文字放到页面上，然后再发送到另一个应用程序进行格式化和编辑。像 Markdown 这样的 [格式写作对于这个](https://lifehacker.com/what-is-markdown-and-why-is-it-better-for-my-to-do-lis-5943320) 来说是很棒的。您可以以纯文本格式编写，添加一些简单的格式，然后将该纯文本文件导出为各种富文本格式，包括 HTML、PDF 或 DOC。

也有一些我曾经使用纯文本的任务，这些年来已经变得没有必要了。有了像 [Spotify](https://www.spotify.com/us/) 这样的服务，我再也不需要保存音乐推荐列表了。同样，亚马逊愿望清单让我抛弃了书籍、漫画、游戏、电影以及其他任何我曾经保存在一个纯文本文件中的推荐，转而支持组织良好的购物清单。

所以，现在，我的纯文本文件和我的待办事项列表都与创造性的努力有关。这是一个大杂烩，其他人可能会组织得更好，但我喜欢这些东西一起生活在同一个应用程序中。这让我觉得即使是我最疯狂的想法也更有可能实现。
