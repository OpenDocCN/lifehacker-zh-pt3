# 这款笔记本电脑速度如此之快，你会原谅它令人讨厌的大而吵

> 原文：<https://gizmodo.com/this-laptop-is-so-fast-you-ll-forgive-it-for-being-obn-1825204203>

微星的 GT75 只是名义上的笔记本电脑。厚度为 2.2 英寸，重量超过 10 磅，即使是一想到要拖着它到处走，我的背都会痛。哦，我有没有提到，这是在你考虑到这个 17 英寸的游戏巨头需要不是一个，而是两个独立的电源块，这加起来又增加了 4.5 磅的因素之前？

Watch

但这款电脑在便携性方面的不足得到了充分的弥补，因为作为第一款配备英特尔全新英特尔酷睿 i9 CPUT3 的系统，GT75 是我们测试过的最快的笔记本电脑。当你将该芯片与 Raid 0 中配置的大量 RAM 和 1tb 的 m.2 存储相结合时，快速的典型同义词(如轻快、炽热或甚至酷热)真的感觉不太合适。因此，尽管 GT75 庞大而令人发指的 3300 美元起价(由于一些额外的 RAM 和存储空间，审查后为 4500 美元)对少数人来说没有意义，但如果你真的非常看重速度而不是笔记本电脑中的其他一切，这款怪兽正是你需要的。

现在，在我们深入系统内部之前，让我们从 GT75 的设计开始，它实际上与去年的车型相比没有太大变化。它的车身感觉足够坚固，没有我在 GT75 的 [更小更实惠的兄弟](https://gizmodo.com/msi-s-gs65-stealth-is-a-gaming-laptop-made-for-adults-1824252259#_ga=2.7483386.91043826.1523276269-3254418773.1522168809) 上发现的任何弯曲或弯曲。外观主要由光滑的拉丝铝组成，尽管我尽了最大努力保持清洁，但它喜欢指纹的本性总是让它看起来有点油腻。

正如昂贵的游戏 PC 通常会做的那样，GT75 还在其键盘上配备了可定制的 RGB 照明，允许用户将每个键设置为不同的颜色，此外，笔记本电脑的左右鼠标按钮中还隐藏了两个可配置的灯。

沿着侧面是 USB 3.1 型端口，一个 SD 读卡器，和坦率地说不必要的 3.5 毫米音频插孔。与此同时，在后面，还有另一个支持雷电 3 的 USB-C 端口，以及 HDMI、miniDisplayport 和一个大的电源连接器，这将我们带到了 GT75 最古怪的功能之一:双充电砖。

由于高耗能的 45 瓦 CPU、大风扇和内部的英伟达 GTX 1080，微星不得不为其笔记本电脑配备两条管道，以保持系统正常供电。这是一个笨重的设备，由两个两磅多的电源砖组成，由一个 Y 形接线盒连接，在某些方面，携带起来比实际的笔记本电脑本身更烦人。如果你打算移动这个东西，把这些电线留在家里真的不是一个选择，因为 GT75 在我们的电池耗尽测试中只持续了 2 小时 53 分钟。

但当 GT75 的风扇开始旋转时，你会知道他们什么时候开始旋转，因为他们听起来像一架小飞机即将起飞，你会得到一个友好的提醒，这个系统正在做什么。(如果你想保持安静，GT75 也有特殊的生态模式。)

没有什么好犹豫的，GT75 已经超速了。在 Geekbench 4 上，该笔记本电脑的六核英特尔 i9-8950HK CPU 的多核性能达到 23，425，比其最接近的竞争对手 GS65 高出 30%，由于其第八代酷睿 i7 CPU，GS65 也不算太差，比 GT75 的酷睿 i9 低一步。结果是相似的，尽管在我们的 Photoshop 图片大小调整测试中不明显，GT75 的完成时间减少了 10%。

但是真正的快乐来自于你的游戏。这东西是为超高分辨率的图像而制造的。在全高清的 ultra 设置上，《古墓丽影的崛起》的*《T1》和《T2》《孤岛惊魂 5》的帧率都在 100 fps 以上。事实上，直到我在*的《孤岛惊魂》*中撞上 4K，GT75 甚至还在挣扎，在几乎所有铃声和哨声都打开的情况下，达到了仍然令人尊敬的 41 fps。与此同时，像 *Overwatch* 这样要求不高的游戏，即使在史诗般的设置下，在 4K 也能以超过 80 fps 的速度运行。*

所有这些数字都是在我玩 GT75 的超频功能之前。使用微星的 Dragon Center 软件，你可以调整 CPU 六个核心中每个核心的性能，并提高 GPU 的核心和 vRAM 时钟速度，在我将一切设置为最高后，将 *Far Cry 5* 的 fps 提高到 46 fps。现在，这听起来可能不是一个巨大的改进，但通过检查一些框和移动一些滑块来获得 10%的性能是非常好的。尽管在将 GT75 超频到龙中心的最高频率后弹出了一点警告，但系统很稳定，没有遇到任何问题或意外关机，当你试图将事情推进到股票速度以上时，有时会发生这种情况。

听着，GT75 是一头笨重的野兽。它又大又重，光是它的价格标签就足以吓跑大多数买家。但是考虑一下另一种选择。试图带着具有这种性能的台式塔式机加上键盘、4K 显示器和鼠标旅行，比拖着 GT75 到处跑还要困难。作为一个展示笔记本电脑能装多少马力的例子，GT75 超越了目前市场上的所有其他产品。这还是在你考虑其他小东西之前，比如系统愚蠢的扬声器，可选的 120Hz 全高清，以及漂亮的黑仔多 g 网络。

然后，你看看其他游戏品牌，如 [【外星人】](http://www.dell.com/en-us/shop/dell-laptops/alienware-17-gaming-laptop/spd/alienware-17-laptop) 和 [华硕](https://www.anandtech.com/show/12611/asus-rog-spring-gaming-system-updates/2) 也在开发 3000 美元以上的酷睿 i9 芯片笔记本电脑，但还没有准备好实际出货，GT75 的价格似乎并不荒谬。如果你对微星将桌面级部件放在一个更紧凑的盒子里收取的溢价感到犹豫，这个系统不适合你。但即使不是，你也不得不承认，这相当令人印象深刻。
