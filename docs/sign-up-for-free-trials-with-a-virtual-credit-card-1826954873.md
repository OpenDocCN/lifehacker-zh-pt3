# 用虚拟信用卡注册免费试用

> 原文：<https://lifehacker.com/sign-up-for-free-trials-with-a-virtual-credit-card-1826954873>

如果你经常忘记设置日历提醒 在免费试用自动续费之前取消免费试用，考虑使用预付卡代替你的信用卡，正如这个 [Redditor 推荐的](https://old.reddit.com/r/lifehacks/comments/8njxrx/free_trial_that_needs_a_credit_card/) 。

这有两个原因:第一，如果你在它自动更新之前忘记取消，你不会损失任何钱，第二，这是保护你的隐私和安全的一个小方法，而在线购物。

这并不适用于所有情况，尤其是如果你需要一个注册服务的地址(并且一定要检查预付卡上的月费)。在这种情况下，更好的选择是[【privacy.com】](https://privacy.com/)，它可以让您生成用于试用和定期支付的号码。你预先支付卡，然后你不需要担心黑客或骗子得到你的银行信息。

如果隐私是你主要关心的问题，花旗还提供了一个虚拟号码生成器，如果你有一张花旗品牌的信用卡(美国银行也提供这个)。您可以选择该号码的有效期，然后收到一个带有有效期和安全码的临时帐号。你可以在这里阅读更多关于 [虚拟卡](https://www.creditkarma.com/credit-cards/i/virtual-credit-card/) 。
