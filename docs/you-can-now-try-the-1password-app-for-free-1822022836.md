# 您现在可以免费试用 1Password 应用程序

> 原文：<https://lifehacker.com/you-can-now-try-the-1password-app-for-free-1822022836>

iOS:都 2018 年了，你还在所有密码栏里写“1234”啊？多亏了苹果，你现在没有借口和美国其他人共用同一个密码了。该公司的 [在其应用商店中增加了一个名为“免费试用”的新栏目](https://9to5mac.com/2018/01/11/free-trial-app-store-section/) ，可以让你在付钱之前试用一些应用。你应该尝试的第一个？T5 1 密码 T7。

iOS 应用商店中的免费试用部分(位于“应用”标签中)突出显示了向潜在客户提供免费试用的基于订阅的应用。虽然目前它只有四个应用程序，但广受欢迎的密码管理应用程序 1Password 也在名单上，这是了解密码管理的最佳时间。

访问应用程序，选择免费试用选项(每月或每年重复)，然后注册。你可以从你的 Apple ID 管理订阅，并通过以下步骤 稍后 [删除(如果你选择的话)。](https://lifehacker.com/how-to-cancel-your-useless-ios-app-subscriptions-1796017796)

考虑到 2017 年一连串的 [信用卡泄露](https://lifehacker.com/how-to-find-out-if-you-were-affected-by-the-equifax-hac-1806121695) 、 [用户名和密码转储](https://lifehacker.com/15-million-hacked-kickstarter-and-bitly-passwords-are-n-1819216049) ，以及你独特的密码实际上 [并不那么独特](https://lifehacker.com/starwars-and-the-24-other-worst-passwords-of-2017-1821338872) ，加倍你的安全是 2018 年要做出的一个重要决定。
