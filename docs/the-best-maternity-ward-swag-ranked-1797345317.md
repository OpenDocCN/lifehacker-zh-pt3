# 最佳产科病房赠品，排名

> 原文：<https://lifehacker.com/the-best-maternity-ward-swag-ranked-1797345317>

恭喜你，你刚刚有了一个宝宝！当你适应与一个无休止地吃、睡、哭、拉的小生命在一起的生活时，你需要一些增援。幸运的是，你的医院可能会为你提供保险。

产科病房的赃物，虽然谈不上迷人，但在身体恢复、母乳喂养和一般新生儿护理的早期却是天赐之物。根据我的经验，以及对朋友们的非正式调查，我整理了一份新父母从医院收到的物品的排名，从半有用到*没有这些，你到底该如何生存？*当然，每个单位的礼物都不一样(我真的很羡慕那个妈妈，她告诉我她医院给她准备了千层面，回家后可以热着吃——*什么？*)。这是人们使用、喜爱并试图走私更多的东西。

### 10.婴儿帽

无处不在的中性医院帽子在生命的最初几个小时里既柔软又舒适，是婴儿书的甜蜜纪念品(注意:你可能不会制作婴儿书)。但除此之外，它通常会被扔到一边。

### 9.痔疮甜甜圈

对于阴道分娩的妈妈来说，当你蹒跚着走出医院时，下面的情况将与你第一次蹒跚着走进医院时不一样。因此，当你坐着的时候，你不会打扰所有的裂缝和类固醇，有值得信赖的甜甜圈。如果你没有收到，不要害怕——一些妈妈说卷起来的毛巾或者你宝宝的 [Boppy 枕头](https://www.amazon.com/Boppy-Nursing-Pillow-Positioner-Naked/dp/B000KW5I6E?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-best-maternity-ward-swag-ranked-1797345317&asc_source=&tag=kinjalifehackerlink-20) 一样好用或者更好。

### 8.一次性床垫

出生后，一切都泄露了。这些一次性垫非常适合保持床和换衣桌的清洁。小贴士:如果你需要买更多，许多人证明小狗垫是 [同样的产品，而且便宜很多](https://blog.easystand.com/2016/11/medical-pads-puppy-training-pads-dog-gone-scam/) 。当你为你的人类购买犬类产品时，带上一些 [狗粪袋](https://www.amazon.com/Poop-Bag-Shop-700-Count-Dispenser/dp/B00AOAT0H0?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-best-maternity-ward-swag-ranked-1797345317&asc_source=&tag=kinjalifehackerlink-20)——无论何时你外出，你都可以把脏尿布绑在里面，然后扔进垃圾桶，而不必担心留下恶臭的痕迹。

### 7.医用袜子

好吧，虽然从技术上来说这不是一件分娩和分娩的事情，但这些袜子值得一些分数，因为它们像你奶奶几十年前给你的旧毯子一样柔软，而且它们有小抓地力，所以你不会在你一步一步走向浴室时在你的硬木地板上吃它。莉娜·杜汉姆已经将它们列入她的 [名人购物清单](http://nymag.com/strategist/article/lena-dunhams-favorite-things.html?utm_campaign=strategist&utm_source=fb&utm_medium=s1) ，称它们已经完全取代了她的拖鞋。

### 6.腹部捆绑器(又名肚兜)

对于从剖腹产中恢复的妈妈们来说，肚兜不仅可以保护切口部位，还有助于缓解她们“五脏六腑都要掉出来”的感觉。额外收获:轻轻挤压腹部可以帮助子宫 [更快恢复到正常大小](http://community.today.com/parentingteam/post/how-to-choose-the-best-postpartum-belly-wrap) 。

### 5.球形注射器(又名吸鼻涕器)

我想我第一次意识到我的生活在孩子出生后改变了多少，是在一个周五的晚上，我试图用这些设备中的一个从她的鼻子里取出鼻屎。在我看来，医院提供的灯泡注射器不如诺瑟夫里达 的肺动力 [Snotsucker 有效，但有些家长信以为真。可以在评论里自由辩论最好的吸鼻涕机制，因为那就是我们现在的生活。](https://www.amazon.com/NoseFrida-The-Snotsucker-Nasal-Aspirator/dp/B00171WXII?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-best-maternity-ward-swag-ranked-1797345317&asc_source=&tag=kinjalifehackerlink-20)

### 4.乳头垫和乳霜

当一个饥饿的小人每两个小时不停地咬你的奶头时，你会痛苦、流泪和咒骂。唯一有帮助的事情(除了更多的咒骂)是在你的乳头上擦几滴奶油。啊啊啊。好多了。很多医院都发这几管 [兰辛诺的 HPA 羊毛脂](https://www.amazon.com/Lansinoh-Breastfeeding-Salve-Lanolin-1-41/dp/B005MI648C?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-best-maternity-ward-swag-ranked-1797345317&asc_source=&tag=kinjalifehackerlink-20) ，东西很棒。一位朋友说，她买了一些兰辛诺舒缓凝胶垫，可以放在冰箱里。

### 3.所有的婴儿用品都是你因为睡眠不足而无法在商店里买到的

新生儿尿布、额外的吸奶器管、配方奶粉、橡皮奶头、耳塞、巨型大号护垫——这些都很受欢迎。

### 2.会阴冲洗瓶(又名 Peri 瓶或水喷射器，用于治疗你的阴部疼痛)

我妈妈的朋友们都不记得这个东西叫什么了，但当他们看到它的照片时，所有的感激之情再次涌上心头。阴道分娩恢复第一法则:不要擦。哦，天啊。千万不要擦。让 [peri 瓶](https://www.amazon.com/Medline-MEDDYND70125-Cleansing-Bottle-8oz/dp/B0055QO4TW/ref=pd_lpo_vtph_121_bs_t_2?_encoding=UTF8&asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-best-maternity-ward-swag-ranked-1797345317&asc_source=&psc=1&refRID=97P9NN9YCVJCBVYN3TTE&tag=kinjalifehackerlink-20) 成为你的新 BFF。当你撒尿时，温暖的水喷到你的下体，你几乎可以听到它轻轻地低语，“那里，那里。”在我的那个部位痊愈很久以后，我把瓶子给了我的女儿，让她在她的水桌上玩。有个女朋友看到了，问:“是我想的那个吗？”我眨了眨眼。她知道。

### 1.网眼内衣

[网眼内衣](https://www.amazon.com/Carer-Disposable-Fixing-Maternity-Incontinence/dp/B01N67H62K?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-best-maternity-ward-swag-ranked-1797345317&asc_source=&tag=kinjalifehackerlink-20&th=1) 是翻来覆去上来的单品。这些怪物有什么不可爱的？它们很轻。它们很通风。它们具有足够的弹性和安全性，可以放置床垫大小的大号床垫。它们是一次性的，所以你不必毁了你的“好”奶奶内裤。相当神奇，真的。(不过话说回来，你生过孩子。你的标准变了。)
