# 为什么我永远不会跑去捡垃圾

> 原文:[https://gizmodo . com/why-I-will-never-run-and-pick-up-trash-1823455055](https://gizmodo.com/why-i-will-never-run-and-pick-up-trash-1823455055)

有一种新的趋势正在席卷全国——实际上是全世界——这就是所谓的 plogging。不，不是在平台上写博客(三年前那个“plogging”[来来去去](https://www.wired.com/2015/10/long-blogging-hello-yep-going-say-plogging/) )，而是边跑边捡垃圾。Plogging 来自瑞典语短语“plocka upp”，意思是捡起来，从 [*【华盛顿邮报】*](https://www.washingtonpost.com/news/inspired-life/wp/2018/02/23/plogging-is-the-swedish-fitness-craze-for-people-who-want-to-save-the-planet-its-making-its-way-to-the-u-s/?utm_term=.84acf569f95f) 到 [*【男性健康】*](https://www.menshealth.com/fitness/a18698660/i-tried-plogging-the-swedish-fitness-craze-that-involves-literal-garbage/) 等出版物都掀起了这股热潮。就我而言，永远不会。

Watch

你可能在想，有什么不喜欢的呢？在清理环境的同时进行锻炼是双赢的，显然这对腹肌很有好处……弯腰捡垃圾时，你会收获不止一种六块腹肌。对于那些致力于应用驱动的生活的人来说， [Lifesum](https://lifesum.com/) 健身应用甚至包括作为锻炼选项的 plogging。

保持美国美丽最近与 Lifesum 合作推广这一趋势，保持美国美丽的高级副总裁迈克·罗森(Mike Rosen)， [说](https://www.kab.org/news-info/press-releases/%E2%80%98plogging%E2%80%99-scandinavian-fitness-trend-collecting-litter-while-jogging-comes)“plogging 很棒，因为它简单有趣，同时让每个人都有能力帮助创建更清洁、更环保、更美丽的社区。”

那么，我的问题是什么？我完全赞成同时改善我的健康和环境，但这有点像几年前每个人都开始喝 Soylent的那一刻，最终意识到这与真正的平衡膳食不一样。

plogging 的固有想法是，你正在清理你已经经常涉足的领域。当我在城市环境中散步或跑步时，我会选择公园或河边小路，因为它们相对安静和干净。我想很多其他潜在的笨蛋也会这么做。但是，如果我们的远足仅限于维护良好的小径或更富裕的社区，那么更多的冷僻地方——也许是收入较低或绿地较少的有色人种社区——不会有太大好处。

许多生活在边缘的人很可能没有时间或资源来沉迷于这项活动。由于各种原因，社会经济地位 [较低的人](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC3169799/) 的锻炼率往往较低。因此，尽管个人和社区的满足感可能很好，除非 plogging 的支持者齐心协力解决环境正义，否则他们的爱好不太可能帮助那些遭受污染最严重的人。

我对博客还有更基本的个人问题:在我看来，如今体验自然的一部分是学会忽略所有人类的碎屑。我不期待完美，但我期待能暂时逃离城市生活的参差不齐的角落和刺耳的声音。如果我把所有的时间都花在扑向每一个流氓垃圾上，我该如何修复我的本性？

 [https://gizmodo.com/embed/inset/iframe?id=fb-10155447204707061&autosize=1](https://gizmodo.com/embed/inset/iframe?id=fb-10155447204707061&autosize=1) 

做环保主义者，不做伪君子，基本没办法。我们都基于我们的价值观和生活在我们创造的这个臃肿的文明中的现实做出妥协，当谈到浪费时，我把它作为一种锻炼形式在路边捡起来。我相对努力地回收和堆肥，并真的努力不乱扔垃圾。我有目标。我不想再使用一次性吸管，我需要开始带不止一个可重复使用的手提袋去杂货店。

所以，虽然我可能不会这么做，但我正在努力将更多环保习惯融入到我的生活中。你也应该这样做:尽管在气候变化和海洋垃圾等重大环境问题上，个人决定似乎无关紧要，但承担个人责任有助于慢慢改变问题的轨迹。

此外，我们都可以努力让公民更多地参与进来，因为如果公民施加压力，变化就会发生。就拿塑料袋等一次性塑料禁令来说， [比如](https://www.nytimes.com/2018/03/03/opinion/sunday/plastic-bags-pollution-oceans.html) 。慢慢地，进步地区的城市开始惩罚这些不必要的浪费。最终，州政府注意到了这一点，要么将禁令中的 [推后](https://www.texastribune.org/2018/01/11/texas-plastic-bag-bans-threatened-supreme-court-challenge-laredo-ordin/) ，要么将 [尝试将](https://nypost.com/2018/03/05/cuomo-considering-statewide-ban-on-plastic-bags/) 纳入更高一级的。总有一天，整个国家都可能采用这种做法作为 [压建](http://www.latimes.com/opinion/editorials/la-ed-plastic-plan-20180220-story.html) 。

减少塑料使用是一场基于全社会环境问题和政策导向解决方案的长期斗争。博客充其量只是一种短暂的时尚。

也就是说，我确信有些人确实从这个活动中获得了道德上的满足，这对我来说没什么。埋头苦干。如果三年后你还在苦干，我会印象深刻的。
