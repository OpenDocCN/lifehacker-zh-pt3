# PC 棒对决:英特尔计算棒 vs 谷歌 Chromebit

> 原文：<https://lifehacker.com/pc-stick-showdown-intel-compute-stick-vs-google-chrome-1760226979>

想象一下，你的电脑就在你的口袋里，你可以把它带到任何有屏幕的地方，用它来完成你的工作。当然，笔记本电脑可以工作，但是 PC-on-a-stick 给了你真正的自由和灵活性，谷歌 Chromebit 和英特尔 Compute Stick 都很难实现这个梦想。我们试开了两辆车。

### 竞争者

谷歌在去年推出 Chromebit 时掀起了波澜，最终 [在几个月前开始销售](https://gizmodo.com/googles-chromebit-turns-a-tv-into-a-chrome-pc-and-is-se-1742962107) 。相比之下，英特尔计算棒已经存在了一段时间，但在同一时间获得了一些动力。它们是不同的产品(有一些很大的不同)，但它们显然在同一个领域竞争。以下是概要:

*   [**Chrome bit**](https://www.asus.com/us/Chrome-Devices/Chromebit-CS10/)**(**[**【亚马逊售价 85 美元**](http://www.amazon.com/CHROMEBIT-RockChip-3288-C-LPDDR3L-Google/dp/B0181JTPCU?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/pc-stick-showdown-intel-compute-stick-vs-google-chrome-1760226979&asc_source=&tag=kinjalifehackerlink-20)**)**:Chrome bit 由华硕制造，运行谷歌的 Chrome OS，通过 HDMI 连接到你的显示器(或电视)，通过蓝牙(4.0)连接到键盘和鼠标，通过 Wi-Fi(支持 802.11 a/b/g/n/ac)连接到互联网。)一旦你安装并运行了它，你就可以从 Chrome 网络商店安装应用程序来完成工作，每个 Chromebit 都有 100GB 的谷歌硬盘空间来存储文档、照片和其他文件，因为几乎没有板载存储空间(事实上只有 16GB。)有一个 USB 端口，用于外部存储或输入设备，以防万一。它由 Rockchip RK3288 处理器提供支持，并配有 2GB 内存。由于没有板载存储，一切都是基于网络的，你不必担心如果你丢失设备会丢失数据，或者当你想把它放在家里时把它留在工作中。你可以 [在这里](https://www.asus.com/us/Chrome-Devices/Chromebit-CS10/specifications/) 查看全部规格。
*   [**英特尔计算棒**](http://www.intel.com/content/www/us/en/compute-stick/intel-compute-stick.html)**(**[**【亚马逊 146 美元**](http://www.amazon.com/Intel-Compute-BOXSTK1AW32SCR-Windows-32-bit/dp/B01ASB0DJ8/?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/pc-stick-showdown-intel-compute-stick-vs-google-chrome-1760226979&asc_source=&tag=kinjalifehackerlink-20) **)** :英特尔的计算棒是英特尔自己做的。如果你正在考虑一个，确保你得到了 2016 年的模型，因为 2015 年的模型，虽然平易近人， [不会太经得起这种摊牌](http://gizmodo.com/intel-compute-stick-review-don-t-buy-it-1699377058) 。更新的版本是 [一个远看更好的](https://gizmodo.com/it-looks-like-intel-made-a-pc-on-a-stick-that-doesn-t-s-1751445636) 。新型号包括 32GB 的板载存储，具有用于可扩展存储的 microSD 插槽，由英特尔 Cherry Trail Atom x5-Z8300 处理器和 2GB 内存提供支持，并支持 Wi-Fi 802.11 a/b/g/n/ac(与去年的/b/g/n-only 芯片组相比)和蓝牙 4.2。有两个 USB 口(其中一个是 USB 3.0)相比去年的单个 USB 2.0 口，又是一大进步。这款电脑棒还预装了 Windows 10。就硬件而言，它的尺寸令人印象深刻。

这些是最基本的。当然，Compute Stick 和 Chromebit 并不是唯一可用的单杆电脑，但它们是最受欢迎的两种，也是最成熟的两种。有大量的 Android 棒带有古老版本的 Android 和 HDMI 端口，其中许多使用 Chromebit 使用的相同芯片，但我们对这些不感兴趣。让我们开始比较。

### Compute Stick 运行 Windows，但 Chromebit 更容易使用

设置和使用是 Chromebit 和计算棒开始分离的两个地方。当然，Compute Stick 运行的是 Windows，这使得它的可用性更高，设置和安装过程可能对许多人来说更熟悉，但 Chromebit 相当即插即用。

使用 Chromebit，您只需插入 Chromebit，将其连接到电源，切换到 HDMI 输入，并配对您的外围设备。一旦配对成功，用你的谷歌账户登录，你就可以开始了。值得注意的是，Chromebit 没有电源按钮，这意味着它一直处于开启和连接状态，直到你拔掉它。无论哪种方式，Chromebit 几乎都是插上电源、登录并开始工作。Chrome 网络商店里有你可能需要做的几乎所有事情的应用程序——特别是如果你的工作、网页浏览或社交生活让你呆在网上——甚至还有一些轻度游戏(稍后会有更多内容。)

另一方面，英特尔计算棒显然是硬件的赢家，它拥有更强大的组件和完整版本的 Windows，可以用于所有你喜欢的应用程序。这当然是一个好处，但也是一个缺点。在你的第一轮 Windows 更新花了很长时间安装之后——不是下载，只是安装——你就会明白为什么了。此外，虽然它有 32GB 的板载存储，但大部分的*将被 Windows 及其更新所消耗。结合它的更新——即使你 [使用你能使用的每一个技巧](http://lifehacker.com/reclaim-disk-space-after-windows-10-update-by-deleting-1720934718)[回收一些空间](http://lifehacker.com/run-disk-cleanup-after-the-windows-november-update-to-s-1743476619) ，你也会获得一些空间来安装一些应用程序，但不足以存储数据。你需要那个 microSD 插槽。一旦你完成了所有的设置、更新、应用安装等等，你就拥有了一台功能齐全的 Windows 电脑。*

### 两者都运行良好，但 Chromebit 获得了便携性奖

这两款电脑都很小，很容易放进口袋或随身携带。它们或多或少都是你完成工作所需要的，但如果我们必须颁发一个便携性奖，Chromebit 将会得到它。Chrome OS 的设计是在没有大量板载和连接存储的情况下运行，Chromebit 即使有 USB 端口，也不需要它。充其量，你随身携带的只是 Chromebit 本身和它的电源适配器。最坏的情况是，你需要随身携带蓝牙键盘和鼠标。更新:这篇文章的前一个版本说 Chromebit 支持 MHL——它不支持，我早该知道。我只是假设我测试 Chromebit 的电视不支持 MHL，而它一直都是 Chromebit！

另一方面，英特尔计算棒同样便携，甚至更小。它有一个电源按钮(Chromebit 没有)和两个 USB 端口，这对连接的设备来说很好。像 Chromebit 一样，Compute Stick 需要一个交流适配器来供电。然而，考虑到额外的存储空间对计算棒来说更为重要，这意味着你最多只能随身携带计算棒本身、交流适配器和 microSD 卡。在最坏的情况下，你要么添加一个组合键盘/鼠标，或蓝牙外设。计算棒的 HDMI 端有一个盖子，但你的 microSD 卡有点滑入插槽中，虽然它在大多数情况下是安全的，但我不想把它放在口袋里或任何东西里。

无论哪种方式，就便携性而言，如果你携带其中一种，你的包会比你携带最轻的超极本轻得多，所以这样比较两者可能有点吹毛求疵。Chromebit 与众不同的地方在于其操作系统如何处理便携性。谷歌将 Chrome 操作系统打造成了完全可移植的，甚至可以在设备之间移植。由于你所有的数据都存储在 Google Drive 中，而你的 Chromebit 只记得你安装的应用程序——即使那些只是 Chrome Web 应用程序——你可以将它插在任何有显示器、输入设备和互联网连接的地方，你就可以开始使用了。如果丢失了，损失不大，也不会丢失一吨的数据，这在便携设备中是巨大的。另一方面，计算棒仍然是 Windows，虽然 Windows 在许多方面都一样便携，但它不太像一个以网络为中心的连接体验，如果你丢失了 microSD 卡，你也会丢失你的应用程序和数据。硬件固然重要，但也要考虑数据的可移植性。

### 计算棒在理论上有优势，但实际上两者不相上下

如果你只是阅读规格，计算棒——尤其是最近的 2016 款——显然有优势。它配备了更强大的处理器，更大的存储空间，更好的内存(Compute Stick 和 Chromebit 都有 2GB，但 Compute Stick 有 1600Mhz DDR3 内存，而 Chromebit 有 1066Mhz DDR3 内存。电脑棒的额外 USB 端口很棒，尤其是考虑到它是 USB 3.0，可扩展存储也很棒(即使这是必要的)，但一旦你将所有这些功能投入使用，最终结果是*感觉*不像听起来那么强大。事实上，Chromebit 使用起来感觉更加流畅和快捷。

如果这听起来很疯狂，确实如此——同时使用两者感觉很奇怪。我一直在想，Chromebit 不应该像以前那样光滑，即使它受到其本质的限制。当你使用 Chromebit 时，你是在谷歌的 Chrome OS 花园中玩耍，硬件、软件和可用应用程序之间的协同作用是谷歌试图管理的事情。你最终会得到一个不太有用的体验，但效果更好，尤其是在你的需求没有超出花园围墙的情况下。然而，在计算棒上，你有英特尔的硬件和 Windows 作为软件，而这种协同作用并不存在。Windows 做的和 Windows 一样，硬件上的裂痕很快就显现出来。1600Mhz DDR3 RAM 并不能节省计算棒，因为 Windows 10 仍然只有 2GB 可用。32GB 的存储空间并不能拯救计算棒，因为 Windows 可以轻松地吃掉 20gb 以上的存储空间，一旦你安装了一些应用程序，你就会面临空间不足的警告。

也不要只相信我的话。 [PC 世界在计算棒上有一些很好的基准](http://www.pcworld.com/article/3033920/hardware/intel-compute-stick-2016-review-though-improved-it-still-lacks-mass-appeal.html) ，而 [Engadget 感受到了硬件](http://www.engadget.com/2016/01/22/intel-compute-stick-2016-review/#) 的局限性，尽管它在一个小棒上，令人印象深刻，但在 Windows 的压力下，以及像大多数用户一样使用 Windows 的情况下，他们很快就屈服了。说到性能，规格的差异让 Compute Stick 看起来物有所值。使用中的性能增量挑战了这一概念，并让你想知道是否应该拿起 Chromebit 并节省 50 美元。

### 结论:今天的 Chromebit，明天的英特尔

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-SGwfum7DYXg&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-SGwfum7DYXg&start=0) 

上面的视频来自[YouTuber Austin Evans](https://www.youtube.com/user/duncan33303)，比较了上一代英特尔计算棒和 Chromebit，得出了与我几乎相同的结论。简而言之，它们都有各自的优缺点，但 Chromebit 在比 Compute Stick 更光滑、更便宜的包装中做了几乎相同的事情，除非你需要 Windows——比如你是微软 Office 的粉丝，或者你需要运行像 PhotoShop(顺便说一下，它爬在 Compute Stick 上)这样沉重的东西，你可能可以用 Chromebit 做你需要做的一切。

如果你必须要有窗户——你知道你是谁——这个选择在 1000 年前就为你做好了。如果你因为是游戏玩家而需要 Windows，那么你可能不会考虑这些，但是如果你是，那就避开它。对于任何密集型应用，尤其是游戏或任何图形密集型应用，它们的动力都太弱了。1080p 的视频很好(4K 视频就没那么好了)，所以网飞、Hulu 和所有其他的都很好——但是更重的呢？避免它。

如果你喜欢以谷歌为中心的生活，使用 Hangouts 聊天，使用脸书和 Twitter 与朋友保持联系，甚至使用 Feedly 这样的网络应用阅读新闻，你在 Chromebit 上就可以了。当你四处走动时，你需要跟踪的东西会更少，如果有什么东西丢失了，损失也会更少。即使你正在寻找一种廉价的方式将旧显示器变成 PC，或者为你的电视找到比 Chromecast 更强大的东西，Chromebit 也是机顶盒的简单性和 HTPC 的强大功能之间的一个很好的妥协。

也就是说，令人印象深刻的是，英特尔成功地将东西塞进了那台小小的棒形 PC，加上一些他们莫名其妙地决定反对今年新推出的版本的小调整(将内置 ram 提升到 4GB 并添加 MHL 支持)，他们可能会有一台近乎完美的棒形 PC。可悲的是，他们没有做这些事情的事实意味着你可以为自己节省一些钱，并获得相同(或更好)的 Chromebit 体验。
