# 三种成分的快乐时光:两极分化的匹克莱蒂尼

> 原文：<https://lifehacker.com/3-ingredient-happy-hour-the-polarizing-pickletini-1797947090>

周末快乐，欢迎回到 [*三料欢乐时光*](http://lifehacker.com/tag/3-ingredient-happy-hour) ，每周饮品专栏，提供超级简单又美味的酒水。今天我们特别推出一款我已经啜饮了*年*的令人振奋的咸味最爱:两极分化的 Pickletini。



喝马提尼的人，包括我自己，在谈到他们最喜欢的饮料时，往往是那种看门的混蛋。说实话，谁能怪他们呢？我不只是在说整个“伏特加对杜松子酒”的事情。马提尼受到的攻击要大得多，比如苹果马提尼和其他什么东西:

 [https://lifehacker.com/embed/inset/iframe?id=twitter-898015707463643137&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-898015707463643137&autosize=1) 

纯粹主义者可能会谴责我的 Pickletini 是对杜松子酒和伏特加的犯罪，但让我们暂停一下。用橄榄卤水制成的肮脏马提尼，虽然有点笨拙，但却被公认为马提尼家族的一员。他们就像你的非常吵闹的阿姨，有点尴尬，但说实话，偶尔和他们一起聚会非常有趣。

Pickletini 基本上是马提尼的另一个肮脏的亲戚，但它比它的橄榄盐水姐妹更明亮，更少盐。如果你使用正确的盐水和杜松子酒，它也是令人惊讶的可口和美味。

你可能知道，当谈到酒和品牌时，我不喜欢过分规范，但我对 Pickletinis 有非常强烈的看法。首先，除非你自己做泡菜，否则你会想要得到一些克劳森犹太酱(任何形状都可以)。克劳森盐水是咸的、酸的、调味的适量。虽然我通常是几乎所有杜松子酒的粉丝，但亲爱的，亨德里克是来这里的最佳选择，因为那个漂亮的瓶子里的黄瓜味正是为这种鸡尾酒制作的。

好了，现在我们已经涵盖了所有的细节，让我们来制作这个辉煌的咸味杂种。您将需要:

*   两盎司半亨德里克杜松子酒
*   3/4 盎司克劳森泡菜盐水
*   半盎司干苦艾酒

将所有原料倒入加冰的混合杯(或品脱杯)中，搅拌至完全冷却并稍微稀释。通过滤网将盐水倒入小酒杯，如果你喜欢，可以用 cornichon(乔氏品牌)装饰。
