### **定向:什么是 Lifehacker U？**

无论你是第一次去大学，还是在轻松的暑假后回到课堂，或者离开学校很久并对学习新东西感兴趣，现在都是时候打开它，通过一些有趣和信息丰富的课程和研讨会来提高你的技能。任何人只要有一点时间和对自我成长的热情，都可以为了自己的个人利益去旁听、阅读和“报名”这些课程。耶鲁大学、麻省理工学院、斯坦福大学、加州大学伯克利分校等学校都提供免费的在线课程，你可以坐在舒适的办公椅、沙发或计算机椅上听课并参与其中。

如果你还记得从 [我们 2016 年夏季学期](https://lifehacker.com/plan-your-free-online-education-at-lifehacker-u-summer-1778753900) 开始，其中一些课程全年都有，但许多课程只在特定的学期或学期才有，因为我们都在帮助你改善生活，所以我们在 Lifehacker 列出了今年夏天的课程清单，这些课程将激励你，挑战你，打开新事物的大门，并给你改善生活的工具。拿起你的笔和纸，确保你的电池充满电——上课了！

### **计算机科学与技术**

*   [**密歇根大学-人人编程(Python 入门)**](https://www.coursera.org/learn/python)**-Charles Severance 教授-** 我们最喜欢的初学者友好编程课程之一又回来了。对于初次学习者来说，Python 是你最喜欢的编程语言之一。本课程将以一种可理解的方式向你介绍它，以及一般的编程。本课程从讨论编程的基础开始，然后直接让您安装使用 Python 所需的工具，并尝试使用它。本课程结束时，您将学习函数、条件代码、循环等等。
*   [**多伦多大学——学习编程:基础知识**](https://www.coursera.org/learn/learn-to-program)**——Jennifer Campbell 和 Paul Gries 教授**——这是另一门专注于编程基础知识的课程，让你开始接触 Python。通过实践练习，您将很快学会使用 Python 构建简单程序所需的工具。本课程进度稍快，但涉及的范围更广。刚开始，但还来得及进去。

*   [**centralesupe LEC-构建你的第一个 Android 应用**](https://www.coursera.org/learn/android-app)**-Virginie Galtier 和 Michel Ianotto** 教授-为 Android 开发通常被认为是一件比为 iOS 开发更无聊、更棘手的事情，但本课程将让你立即开始开发你的第一个 Android 应用，并将教你未来继续为 Android 开发所需的基本技能。从学习应用程序故事板的基础知识，到安装 Android Studio 和开发工具，再到完成应用程序，您可以从一个想法开始本课程，以一个实际可以使用的工作应用程序结束。
*   [**雷丁大学——开始编程:打造你的第一款手机游戏**](https://www.futurelearn.com/courses/begin-programming)**——Richard Mitchell、Timothy Threadgold 和 Karsten Lundqvist 教授-** 如果你的手机开发目标不仅仅是应用程序开发，而是游戏开发，那么本课程将作为入门课程。本课程在开始前还有一段时间，但无论你是一个完全的编程新手，还是带着一些帮助你入门的技能进入这个主题，它都是对移动游戏开发的全面审视。你将学习用 Java 开发，用算法解决编程问题，当然，在 Android 世界中使用 Android Studio 进行游戏开发。
*   [**约翰·霍普金斯大学——面向 web 开发人员的 HTML、CSS 和 Javascript**](https://www.coursera.org/learn/html-css-javascript-for-web-developers)**——yaa kov Chaikin 教授**——如果你对 Web 开发感兴趣，无论是小工具还是整个网站，这门课程都适合你。您将从基础开始，从头开始学习支持 web 的语言。您将从头开始构建既能在移动设备上又能在桌面上快速阅读的 web 页面，然后从这里开始构建 web 应用程序。尽管从技术上讲，这个课程是 ruby on rails 专业的一部分，但你并不需要 ruby 来学习。它对任何有兴趣学习 web 开发的人开放。
*   利兹大学-在线学习:管理你的身份 **-教授 Raphael Hallett 和 Helen Howard** -在这一点上，确保你管理自己的在线身份对于确保你控制网络上关于你的信息是至关重要的。无论你是否使用它们，你都需要成为拥有你自己的脸书页面、你自己的 LinkedIn 个人资料的人，因为我们大多数人都使用社交网络，如何拥有它们并以不损害你的线下生活的方式管理它们。这门课程是为年轻的大学生设计的，但它也为所有年龄的人提供课程，特别是那些关注我们的网络身份如何影响我们的现实世界的人——以及“网络”和“现实世界”本身是如何区分的。
*   [**普林斯顿大学——比特币和加密货币技术**](https://www.coursera.org/learn/cryptocurrency)**——Arvind Narayan an 教授——**如今，网络上关于比特币的讨论并不像一年前那样热烈，但这并不意味着我们可以了解很多关于加密货币及其技术如何工作的信息。本课程探讨了这个主题，讨论了比特币与其他货币的不同之处，决定其价格的因素，以及去中心化货币技术的未来。

*   [**加州大学伯克利分校——计算的美丽与快乐**](https://www.edx.org/course/beauty-joy-computing-apr-cs-principles-uc-berkeleyx-bjc-12x#!)**——Dan Garcia 教授——**计算机科学中的部分 AP 课程，这门课(已经开始，所以你要迟到了)教你抽象地思考，让你熟悉作为一名开发人员真正入门所需要的思维方式，并学习一门最用户友好的编程语言 Snap！(其中基于 Scratch。)您还将讨论编程的全球意义，为什么学习编码如此有力量，以及为什么帮助他人学习编码如此重要。
*   [**ITMO 大学——如何赢得编码竞赛:冠军的秘密**](https://www.edx.org/course/how-win-coding-competitions-secrets-itmox-i2cpx)**——Maxim Buzdalov 和 Pavel Krotkov 教授-** ITMO 大学是世界顶级编码杯唯一的六次冠军，这门课程教你如何增强自己的编程技能，要么与他们竞争(或并肩竞争)，要么只是如何让自己的编程努力和游戏更有效率。您将了解参与黑客马拉松和编码竞赛的好处，以及掌握快速、灵活、优雅的编程艺术所需的工具。
