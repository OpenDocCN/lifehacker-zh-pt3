# 你不需要更多的金钱建议，你只需要你能认同的建议

> 原文：<https://lifehacker.com/you-don-t-need-more-money-advice-you-just-need-advice-1793489761>

如果你正在读这篇文章，你可能至少对理顺你的财务有点兴趣。也许你已经试过了！也许 [你已经读过一堆建议](https://lifehacker.com/you-probably-dont-need-more-financial-advice-just-make-1787874086) ，但似乎没有一条对你有效。如果这听起来很熟悉，解决办法可能不是去找*更多的*建议。相反，集中精力寻找适合你的建议。



你知道，个人理财是个人的。顾名思义，但是人们似乎忘记了金钱管理与你自己独特的情况、习惯和行为有多大关系。一些金融作家专注于他们的个人债务故事。其他人，像我一样，通常专注于实用技巧。一些金融专家关注我们社会中的金钱心态。然而，从本质上讲，所有这些个人理财建议或多或少都是一样的。不同的是方法。因此，最好的理财建议是你能真正理解的建议。

### 找一个激励你的故事

当我刚开始阅读和写作关于金钱的文章时，*所以*很多人推荐拉米特·塞西的 [*我来教你变得富有*](https://www.amazon.com/Will-Teach-You-Be-Rich-ebook/dp/B004WL4BW6/ref=sr_1_1?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/you-don-t-need-more-money-advice-you-just-need-advice-1793489761&asc_source=&ie=UTF8&keywords=i will teach you to be rich&qid=1490750421&s=books&sr=1-1&tag=kinjalifehackerlink-20)*说它完全改变了他们的生活。Sethi 提供了多年来一直有效的建议，但他用一种对很多人都适用的方式包装了它。例如，他不喜欢少喝拿铁。他认为，你不应该节俭，一次只存 3 美元，而应该集中精力在重要的地方存钱——住房、食物和其他大额开支。*

*许多人觉得这种态度令人耳目一新。他们仍然可以享受小乐趣的想法，比如每天的拿铁咖啡，让他们完全想做这个钱的事情。这个建议 [似乎与传统的个人理财建议](http://twocents.lifehacker.com/money-advice-the-experts-don-t-agree-on-being-frugal-1601213528) 相矛盾，这让它很有吸引力，但当你真正深入了解它时，这个建议是标准的:减少你不在乎的东西，这样你就有更多的钱花在你在乎的东西上。然而，让塞西的建议与众不同的是他对金钱的心态。这就是一切。*

*换句话说，他有一个相关的故事:这个家伙抵制节俭，掌握控制权，做对他有利的事情。人们喜欢这样，因此，我只能想象 Sethi 有多少读者愿意开设 401(k)账户。*

*不过，你可能会找到一个不同的故事来激励你。*

*当我还在学生贷款债务的时候，我喜欢在 MSN 上阅读唐娜弗里德曼的金钱建议。她是一个作家，她很节俭。她的故事完全引起了我的共鸣，她还给出了一些实用的、严肃的省钱小技巧，这正是我当时想要的。每天阅读她的帖子激励我采取行动，并参与我自己的财务。*

*如果你已经准备好行动了，但是还没有什么对你有效，你可以考虑把注意力放在故事上，而不是建议上。如果你负债累累，这可能意味着跟随一个经历同样事情的博客作者(或者某个经历过 [和](http://twocents.lifehacker.com/the-shift-in-mindset-that-will-help-you-get-out-of-debt-1786822037) 的人)。基本上，你想要寻找与你自己一致的故事、观点和价值观。( [Rockstar 财经](http://rockstarfinance.com/) 是浏览不同类型财经博客的绝佳平台)。*

### *找到你理想的媒介*

*这也有助于找到一个方便的媒介。举个例子，我给一个好朋友买了一本*我会教你*来帮他理财，他从来没拿过，因为他就是不爱看书。当我开始给他发 [Sethi 的博客帖子](http://www.iwillteachyoutoberich.com/) 时，他注意了。在我意识到之前，他已经增加了每月的学生贷款，并存了一笔应急基金。一个更容易接近的媒介使他更容易消化建议，更重要的是，更容易采取行动。*

*如果你花很多时间通勤，播客可能是你理想的媒介( [这里是一些读者最喜欢的](https://lifehacker.com/what-s-your-favorite-personal-finance-podcast-1654210686) )。在你开车上班或开车回家的路上，很容易就能看到与金钱相关的节目。*

*不管这个建议有多相关，如果它不是你习惯使用的媒介，它只会让你的财务变得更加困难，因为你的挑战变成了双重的:让你的财务变得有序*和*腾出更多的时间买书。理清你的财务状况已经够难了，所以只需选择一个你觉得舒服的媒介，然后分别专注于 [成为一个更热心的读者](http://lifehacker.com/my-secret-to-reading-a-lot-of-books-514189426) 。*

### *让它成为一种习惯*

*一旦你找到了一个适合你的故事和一个容易消化建议的媒介，你就会想继续接受这个建议。这应该是很自然的。当你真正享受你正在阅读(或聆听)的内容时，你会真正期待新的博客文章、章节或剧集。*

*如果你发现很难养成一个习惯，那就让它成为你日常生活的一部分。每天早上醒来，喝着咖啡读一章或一篇新的博客文章。每天下班回家的路上听你选择的金钱播客。 [即使你只是承诺每天 15 分钟的金融知识](http://twocents.lifehacker.com/commit-to-fifteen-minutes-of-financial-literacy-a-day-t-1791131709#_ga=1.98326495.1392254241.1490120273) ，也会有所不同。 [习惯并不总是来得容易](http://lifehacker.com/what-it-takes-to-form-a-good-habit-5724234#_ga=1.98326495.1392254241.1490120273) ，但是当涉及到学习金钱的时候，成功的一半是找到对你来说实际上有意义的相关建议。*
