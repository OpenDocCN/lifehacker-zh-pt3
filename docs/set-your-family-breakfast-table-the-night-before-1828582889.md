# 前一天晚上摆好你的家庭早餐桌

> 原文:[https://life hacker . com/set-your-family-breakfast-table-the-night-before-1828582889](https://lifehacker.com/set-your-family-breakfast-table-the-night-before-1828582889)

在我从祖母那里学到的所有老派育儿技巧中，有一个非常突出，它是返校后的救命稻草:晚上睡觉前摆桌子。这样，当你家里的任何人醒来时，即使在错开时间，他们也会在家庭餐桌前等着他们——不需要在厨房里跑来跑去打开每个橱柜和抽屉寻找勺子、杯子或葡萄。

将这个想法向前推进一两步既快又容易，而且为每个人赢得了更多的时间

*   准备好茶壶和水壶，或者咖啡机，这样它就可以随时使用了(或者设置一个定时器，如果可以的话)
*   我们孩子的麦片或格兰诺拉麦片放入碗中，用小盘或盖子盖好，这样就不会在一夜之间变坏
*   拿出最喜欢的早餐食品，如蜂蜜和新鲜水果，整个放好，这样就不会变质。(在我们家，芒果、梨和香蕉一直是人们的最爱。)
*   用一个带盖子的水罐装满水，并提供耐储存的果汁或巧克力牛奶盒。
*   把狗粮和猫粮分好，放在柜台上准备好，从你早上要做的事情清单上再检查一件事。
*   在艰难的日子里，或者作为一个惊喜，鲜花、鼓励的字条、气球(我放在厨房的抽屉里)，或者像香蕉面包这样的特别款待，都可以在你们匆忙出门之前，带着额外的温暖和模糊的感觉开始新的一天。

对于有婴儿的家庭来说，除了时间紧迫之外，他们的手经常忙于抱着饥饿、蠕动的后代，简单地用婴儿碗、勺子、围嘴、吸管杯和玉米粥罐设置高脚椅就可以带来巨大的变化。当我花时间做这件事时，我可以一边喂我的小家伙，一边享受一杯咖啡和我自己的早餐——这真是一种享受。当我不这样做的时候，我不仅通常会错过一顿饭，我几乎无法啜饮半杯急需的咖啡因；我浪费宝贵的时间寻找突然不在它们应该在的地方的东西，或者在我需要它们的时候不干净的东西；我带着初露头角的头痛开始了我的一天。毫无疑问，这是导致机能障碍的原因。

每晚布置家庭餐桌不仅让早晨不那么紧张，更愉快；这也确保了每个人在面对忙碌的一天之前都会坐下来饱餐一顿，并且偶尔有机会和两个家人一起吃饭——还有什么能比这更好呢？
