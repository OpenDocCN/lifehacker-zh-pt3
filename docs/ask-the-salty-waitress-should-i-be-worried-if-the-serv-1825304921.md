# 问那个咸味的女服务员:如果服务员不记下我点的菜，我应该担心吗？

> 亲爱的咸味，
> 
> 难道只有我一个人不喜欢服务员在没有写下任何东西的情况下拿走餐桌上的订单吗？
> 
> 很明显，当一名服务员在没有参考一张纸的情况下，一口气说出一份精心制作的特色菜清单，然后在离开餐桌之前，完美地将我们的订单反馈给我们，这给人留下了非常深刻的印象，但现实往往是这样的:
> 
> “今晚的特色菜是...煎猪里脊肉配...[停下来喘口气/戏剧效果/回忆记忆]...”
> 
> 然后

> “那就是一份芝士通心粉，两个龙虾保湿盒，一份三文鱼和...不好意思，提醒一下你上次点的菜？”
> 
> 食客喜欢这样吗？不管我能多么完美地回忆起订单，当服务员带着我的订单离开时，我总是会感到中度到极度的不安，不管它有多简单。
> 
> 你的，
> 记事本作家

亲爱的记事本作者:

我很幸运在餐馆工作，那里鼓励或要求写下订单。我也更喜欢服务员记下订单，这是我在当服务员时做的事情。 [还有不，我们并不孤单](https://www.bonappetit.com/columns/nitpicker-columns/article/i-m-sick-of-waiters-who-don-t-write-down-my-order) 。

Watch

但是并不是所有的餐馆都给服务员配备了笔和纸。当服务员没有写下来的时候，通常是他们经理的指示。不写下订单的理由是，服务员在不写下订单的同时更好地“倾听”了顾客，他们保持了目光接触、兴趣和注意力。一些经理认为这让员工看起来更加专注和友好——我认为这让我担心他们会忘记我的牧场调料。

凭记忆背诵特色菜也有类似但不完全相同的解释:这让服务员看起来对自己的菜单知识和特色菜的熟悉程度很有信心。在一个完美的世界里，我们喜欢想象我们的服务员几乎和厨师一样了解特色菜，他们品尝过，思考过，问过问题。从便笺簿上读出来会让它看起来像机器人，就像员工不太了解他们推荐的菜肴一样。我对服务器凭记忆背诵特色菜没有太大问题；通常只有几道菜，他们可以在晚上的第三或第四桌前搞定。我确实在口袋里放了一张小抄，以防我脑子有问题。

但是回到写下订单的问题。在这一点上，我坚决支持把事情写下来。这(希望)确保了准确性，尤其是在一大群人当中，也帮助我记住了谁点了什么菜。我讨厌走到桌前，只是端着一盘菜——“谁点了帕尔马鸡肉？”——好像这是某种免费的。当我写下订单时，我可以指出谁点了哪道菜。

不过，服务员记下订单的时间不应该太长，否则会影响你的点餐流程。我曾经工作过的一家餐馆很小，我们没有计算机化的销售系统。我们把手写的订单带回厨房，钉在一起给厨房看。我们有一个速记代码:我们都知道“SF: MW，SOS”是牛排薯条，五分熟，调味汁放在旁边。我甚至每次都圈出 SOS，以确保厨师们看到特别的说明。速记可以确保订单不会永远持续下去，顾客也不必每隔几个字就停下来重新开始。我建议服务员完善他们的速记，即使他们以后不得不扩展那些快速笔记。

也许我能记住某些桌子的顺序，但为什么要冒险呢？更大、更复杂的组将是一场记忆的噩梦，尤其是如果你在组合中加入了替换，或者如果有人在我走回厨房之前改变了他们点的菜的某个方面。我想大多数人宁愿吃他们点的饭，也不愿惊叹我的记忆技巧。
