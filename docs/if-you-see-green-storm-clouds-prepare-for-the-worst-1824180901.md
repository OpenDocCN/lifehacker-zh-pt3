# 如果你看到绿色的风暴云，做好最坏的打算

> 原文:[https://life hacker . com/if-you-see-green-storm-clouds-prepare-for the worst-1824180901](https://lifehacker.com/if-you-see-green-storm-clouds-prepare-for-the-worst-1824180901)

你看到那边绿色翻滚的风暴云了吗？那些是坏的。不是因为你会被带到某个讨厌的音乐世界——不，因为那些云意味着风暴特别讨厌和危险。这甚至可能意味着龙卷风即将来临。

Watch

居住在龙卷风道的人们长期以来一直认为绿色的云是即将到来的龙卷风或冰雹的信号，这些龙卷风道是美国的一大区域，横跨得克萨斯州北部、俄克拉荷马州、堪萨斯州、内布拉斯加州以及更远的地方。 [传统民间传说认为](https://news.wisc.edu/curiosities-why-does-the-sky-turn-green-before-a-tornado-2/) 云不祥的颜色来自于被龙卷风卷到天上的所有青蛙和蚱蜢。多么可笑！(*蟋蟀唧唧*)。)但是别担心，青蛙和蚱蜢爱好者们，这些可怕的水蒸气庞然大物显然不是这么回事。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-4MGFploTFGY&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-4MGFploTFGY&start=0) 

那么为什么是绿色的呢？ [Youtuber 和风暴追逐者 Pecos Hank 解释说](https://www.youtube.com/watch?v=4MGFploTFGY) 这一切都归结于颜色光谱，以及物体吸收和反射阳光的方式:

> 一些最大的雷暴会产生看起来像瀑布一样的集中大雨和冰雹区域。风暴追逐者称之为“核心”有时风暴云会散开，让你一瞥高空的核心。如果你在风暴底部，大约在中午，核心通常是蓝色或华丽的蓝绿色。但是在下午晚些时候和晚上，当大多数风暴活动发生时，倾斜的金色和红色夕阳光可能会与蓝色降水核心混合，最终效果是绿色。黄色和蓝色组成绿色！

[俄克拉荷马大学气象学教授威廉·比斯利](http://meteorology.ou.edu/member/william-beasley/) 对此表示赞同。他向《科学美国人》解释说，这一切 [都归结于在这些午后雷雨云](https://www.scientificamerican.com/article/fact-or-fiction-if-sky-is-green-run-for-cover-tornado-is-coming/) 中发现的湿气颗粒。它们吸收红光，使散射光呈现蓝色。当它们被设置在一个重红光的环境中，比如“黄金时间”或日落时的低太阳，净效果是一种绿色。然而，事情是这样的:根据宾夕法尼亚州立大学的物理学家 Craig Bohre 的说法，绿色风暴云不一定意味着龙卷风或冰雹风暴即将来临。基本上，无论有没有绿色云 冰雹或龙卷风都可能发生 [。这就是为什么为这样的事件做好准备是很重要的，而不仅仅是依靠你所能看到的。](https://weather.com/science/weather-explainers/news/green-sky-thunderstorm-hail)

也就是说，一般来说，绿色的云是严重雷暴的一个相当不错的预报。UW-麦迪森气象卫星研究合作研究所的气象学家斯科特·巴赫迈尔 说，绿色的云通常是高云，高云是潜在冰雹风暴和龙卷风的主要因素。因此，虽然看到绿色并不意味着龙卷风即将来临，但它确实意味着龙卷风*可能*即将来临。不管怎样，你都要面对恶劣的天气，朋友。如果你想避免暴雨、大风和闪电，绿色的天空仍然是一个很好的警告标志，让你收拾行李，离开道奇。如果你在下午看到一些绿色的云向你走来，是时候离开这个区域了，或者至少到里面去，如果需要的话，你可以很容易地遵循你的应急计划。而如果你还没有应急方案， [你就做一个](https://lifehacker.com/seven-emergency-preparedness-tips-you-may-not-know-1787063633) 。
