# 使用此应用程序，通过沉浸式学习电视节目和书籍来学习外语

> 原文：<https://lifehacker.com/learn-a-foreign-language-through-immersion-with-tv-show-1828277607>

当你试图学习一门外语时，没有比沉浸其中更好的方法了。毕竟，语言沉浸是我们大多数人小时候学习第一语言的方式。当你周围的一切都是用一种特定的语言写的时候，你很可能要努力去弄清楚。



在某种程度上，这就是 [Flowlingo](https://getflowlingo.com/) 背后的想法，这是一款 iOS 和 Android 应用程序，可以帮助你通过浏览你最喜欢的网站、观看电视节目和阅读书籍来学习一门新语言。

如果你在理解上遇到困难(你会的)，这个应用程序会帮你翻译。这个想法是你通过沉浸式学习，而不是游戏或更传统的方式。

即使你已经通过另一种方法学习了，加入到你的学习循环中也是一件很好的事情。我目前正在努力提高我的法语，以便今年晚些时候去旅行，没有什么比在电视节目和报纸上听和读法语更好的了。
