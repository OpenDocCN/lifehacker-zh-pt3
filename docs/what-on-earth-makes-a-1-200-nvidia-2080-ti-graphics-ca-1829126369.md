# 究竟是什么让 1200 美元的显卡物有所值？

> 原文：<https://gizmodo.com/what-on-earth-makes-a-1-200-nvidia-2080-ti-graphics-ca-1829126369>

Nvidia 希望你花同样的钱购买其最新的顶级显卡 RTX 2080 Ti，就像你购买一台全新的电脑一样。你可以用同样 1200 美元的价格买到一台很棒的笔记本电脑或非常坚固的台式机。你可以买几个 PS4 或者 Xbox 的！你可以买一部 iPhone X 或者三星 Note 9。重点是，你可以花 1200 美元买很多不同的功能比显卡更强的小玩意——所以这到底值不值？



Nvidia 的上一代产品 10 系列大约在两年前推出，今天仍然可以使用，对于大多数游戏来说仍然足够快。我说游戏而不是东西，因为除非你在挖掘密码，否则你可能只是在市场上购买昂贵的显卡，这样你就可以在昂贵的计算机上玩游戏。你真的不需要一个昂贵的图形处理器来处理 Photoshop 或者剪辑一部电影。GPU 擅长一件事——渲染 3D 图形，我们大多数人只需要在玩游戏的时候渲染 3D 图形。

但关键是 TTX 1080 Ti 和 1080 是真正可靠的 GPU，分别可与 AMD 的镭龙 RX Vega 64 和 56 相媲美。1080 Ti 的价格为 700 美元，与新的 2080 差不多。所以为什么要多花 300 美元(这是一个新的廉价 iPad 的成本！)上了 2080 Ti？

在对售价 1200 美元的 2080 Ti 和售价 800 美元的 2080 Ti 进行了一系列标准测试，并在一台大 4K 电视和一台较小的 1440p 显示器上玩了很多游戏之后，我发现...我认为这实际上不值得——就目前而言。2080 Ti 是一款出色的硬件，毫无疑问，它是 GPU 的未来，但它太前沿了，你甚至无法在你的计算机上做它能做的最酷的东西。

2080 Ti 和它稍弱的兄弟2080，是 Nvidia 第一批基于其新图灵架构的消费级显卡。图灵架构的秘方是为人工智能投入更多的力量。

不，这个 GPU 不会驱动 Hal 9000 或任何其他谋杀机器人。相反，人工智能被用来，至少在这些面向消费者的卡片中，更好地预测游戏中元素的行为。具体来说。图灵架构允许英伟达最新的 GPU 实时执行 [光线追踪](https://gizmodo.com/why-ray-tracing-on-nvidias-new-gpus-is-so-exciting-1828964418)——游戏 GPU 的“圣杯”。如果你不知道那是什么，没关系——除了特效公司和 GPU 爱好者论坛之外，光线追踪并不是一个众所周知的事情。但是它*是*很酷。光线追踪是一种渲染，其中在计算机生成的环境中，每条单独的虚拟光线从其来源追踪到它可以反射的对象。但是光线追踪也意味着这些光线可能会被在它们前面移动的人物阻挡，或者在它们穿过半透明物体时被扭曲。这意味着在计算机生成的环境中的物体——比如电子游戏——会像在自然世界中一样更多地反射光线。

今天，当火焰在游戏中闪烁时，它可能会从周围的表面反弹，但不现实。你不会在角色的眼睛里或者附近的窗户表面看到火焰发出的光点。要做到这一点，需要大量的能力，这是 Nvidia 新的 20 系列之前的 GPU 无法实时完成的。

不幸的是，Nvidia 的 20 系列 GPU 至少在一个月内无法进行光线跟踪。虽然芯片可能有能力做到这一点(稍后会详细介绍)，但它们没有 Windows 10 的软件支持。Windows 10 中的光线跟踪需要 DirectX 光线跟踪和 Windows ML ，它们要到下个月预计的 Windows 10 2018 年 10 月更新时才会到来。因此，在接下来的几周内，这些巨大的图形处理器无法完成它们的设计任务，即使在更新之后，也只会有少数游戏支持光线追踪。Nvidia 指出目前只有 11 个支持，尽管它说更多的将会到来。

相反，2080 和 2080 Ti 只是非常强大的 GPU，可以让你当前的游戏运行得更快。尤其是在 4K。4K 本身也是一个前沿——至少在游戏方面。在图灵之前，GPU 真的很难以每秒 30 帧以上的帧速率渲染 4K 的顶级游戏。30fps 是可以接受的(这大约是电影和电视节目的帧速率)，但游戏玩家更喜欢更高的帧速率，因为在一两帧的时间跨度内可能会发生很多事情。30fps 对于休闲游戏玩家来说是不错的，但是喜欢激烈射击或战斗游戏的人会想要至少 60fps，120 或 144 也是令人垂涎的。(注意:我们手头没有 1080 Ti，但下面的基准测试让您感受到新卡与旧的 10 系列相比有多强大。)

Nvidia 的最后一个主要系列的 GPU，10 系列，就是不能削减它，但正如上面的基准测试所证明的那样，20 系列在新的视频游戏如 *Far Cry 5* 或 *Shadow of the Tomb Raider* 中每秒拉 60 多帧没有问题。它在像《古墓丽影:T4 的崛起》这样的老游戏中也表现得很好，而且它在要求不那么严格的游戏 live(T7)、文明 VI(T9)和守望先锋(T10)中给你提供了比你所知道的更多的帧数。

这两个图形处理器速度非常快，功能非常强大。

但我不知道我现在是否可以推荐 2080 或 2080 Ti。尤其是他们的价格。当然，没有一台机器，即使是 PS4 Pro 或 Xbox One X 这样的专用 4K 主机，也无法在性能上与 2080 和 2080 Ti 相媲美。这玩意熏的是它的前辈和目前 AMD 可用的设备。2080 Ti 速度很快，令人印象深刻，只是没有 1200 美元令人印象深刻。起价 800 美元的 2080 从性价比来说更令人印象深刻，如果你正在升级一台内置更新 GPU 的 PC ，那么它不会比速度慢得多的 580 美元 AMD 镭龙 RX Vega 64 或 500 美元 PS4 Pro 贵多少。

但是等待不会有什么坏处，如果我不说这个审查还没有完全完成，并且直到光线跟踪支持被添加到 Windows 并且我们可以看到每个 GPU 实际上如何处理效果时才会完成，那我就做不好我的工作。虽然 2080 和 2080 Ti 可能是非常强大的 GPU，但它们不太值得...还没有。
