# 星巴克的秘密菜单有南瓜汁，或者类似的东西

> 原文：<https://lifehacker.com/the-starbucks-secret-menu-has-pumpkin-juice-or-somethi-1819456465>

如果南瓜香料拿铁不是基本的，不管出于什么原因，我们已经决定它们是基本的，但因为它们不是秘密，会怎么样？一种星巴克南瓜饮料怎么样？这种饮料非常不简单，甚至不在菜单上。

Watch

[根据 Kitchn](http://www.thekitchn.com/hack-the-starbucks-menu-to-make-pumpkin-juice-251278) 的说法，你可以黑掉星巴克的菜单，得到一种饮料……好吧，这其实不是南瓜汁，但尝起来很像:

> 要求苹果汁加到杯子的顶部，几泵南瓜香料酱(取决于你饮料的大小)，为了好玩，撒点南瓜香料调料，剩下的加冰。

这可能有点像糖炸弹，但不管你是想在秋天有个好心情，还是想重读哈利波特，这都会有用的。
