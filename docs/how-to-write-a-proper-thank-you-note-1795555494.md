# 如何写一封合适的感谢信

> 原文:[https://life hacker . com/how-to-write-a-proper-thank-you-note-1795555494](https://lifehacker.com/how-to-write-a-proper-thank-you-note-1795555494)

几个月前，当我不在镇上时，一个朋友自愿帮我照看狗。到家后，我对她的服务表示感谢，并递给她一个信封，里面装着几张钞票。信封里缺少的东西可能就是我们和她所观察的野生动物的区别:一张感谢信。

纽约 礼仪学校的创始人 [帕特里夏·菲茨帕特里克](https://twitter.com/NewYorkManners) 告诉 Lifehacker，写一封感谢信并不像听起来那么令人畏惧。没有必要用华丽的语言或奇特的草书来绊倒自己；这是一种感恩的表现，而不是笔试。

根据 Fitzpatrick 的说法，感谢信应该在多种情况下发送。

> “感谢信是对为你做的事情或给你的礼物表示感谢和认可。任何时候有人为你做了什么特别的事，你都应该感谢他。不管是有人带你出去吃饭，请你去他们家做客，在他们家过周末，还是送你礼物。你应该总是在收到礼物后送一个。”

无论你是在面试一份新工作，还是现任总裁，在参加完一场活动后发表告别演说，有几种方法可以让你的感恩之心更有效，而且，根据 Fitzpatrick 的说法，更温暖人心。

### **做好准备工作**

感谢信是深思熟虑的信件，因为你永远不知道什么时候你可能要发送一封感谢信，所以提前做一些准备工作，给自己省点麻烦是值得的。这意味着储备文具，无论是个性化的还是其他的。没有打印纸废话。

Fitzpatrick 建议从通信卡开始:用较重的纸制作的小便条卡，配一个信封以便邮寄。至于颜色，亮白色或奶油色的卡片不会错。你可以从类似 [纸莎草](https://www.papyrusonline.com/stationery/flat-correspondence-cards) 和 [起重机&公司](http://www.crane.com/) 的文具店中抓取定制的信件和感谢卡，或者浏览 [Etsy](https://www.etsy.com/search?q=correspondence%20cards&ref=auto1) 寻找一些独特的设计。避免填空卡。“那些绝对是给小孩子看的。”

### 如果你写得很糟糕，就打你的便条

草书是我选择的写作风格，我一直喜欢自己写感谢信，但在表达感激之情时，没有必要求助于奇特的字体。事实上，Fitzpatrick 说，如果你的书写很糟糕，打字也是可以接受的。

首先也是最重要的一点，感谢信必须清晰易读，所以如果你潦草的字迹有所欠缺，不要害怕打开谷歌文档(Google Docs ),删掉一个。如果你要打印出你的感谢信，选择一些纸张大小的信纸可能是你最好的选择。它将省去您摆弄打印机设置的麻烦，以及在更小的定制卡上进行反复试验的麻烦。打感谢信似乎有点俗气，但比没有感谢信更糟糕的是无法阅读。

### **尽快发送**

如果你发现自己有资格发一封感谢信，你应该立即这样做。在 48 小时内派遣一个通常是标准的操作程序，但有些事件可能需要更方便的响应。当涉及到像工作面试这样的活动时，Fitzpatrick 建议从一封感谢邮件开始，然后是一张实体卡片。

> “有时公司决策非常迅速，因为不是每个人都发手写便条，所以发便条的人会显得与众不同。这有助于他们超越竞争对手。”

如果你不能亲自给某人一张感谢卡，可以邮寄。不要害怕问他们的邮寄地址！

### **具体一点**

谁会想要一张你签名的普通感谢卡呢？没人，就是他。个性化的感谢卡会给你的便条带来温暖感，提醒收件人他们做了什么，以及你在感谢他们时有多体贴。也不必长篇大论。一两句感谢某人的礼物或服务的开场白，接着是一点轻佻的话和关于上述礼物的结束语，你就成功了。大家都赢了！
