# 最好的脆皮土豆不含任何奶酪

> 原文：<https://lifehacker.com/the-best-potatoes-au-gratin-dont-contain-any-cheese-1825751769>

哦，你好。你注意到我了吗？你准备好发表愤怒的评论，谴责我和我最糟糕的拍摄了吗？我理解你的朋友，因为当有人告诉我脆皮土豆不需要任何奶酪时，我也有类似的反应。

老实说，那决定命运的一天真是过山车。当这位我花钱雇来教我法国烹饪方法的女士第一次告诉我我们将做脆皮土豆时，我非常高兴，但当她告诉我我们不会用任何奶酪时，我非常难过。她向我保证这将是很好的，甚至比我以前吃过的任何脆皮面包都要好。我就像“如果是真的，那就太大了”，它遭到了茫然的凝视，因为我的老师是一位优雅的法国女士，而不是一只住在互联网上的千年浣熊。

总之。从技术上来说,“脆皮”仅仅意味着“上面覆盖着一层棕色的外皮”,但是这层外皮通常是奶酪、面包屑或者面包屑和奶酪。然而，这个食谱完全省略了奶酪，只包括土豆、重奶油和盐。这看起来太简单了，但用一句话来说就是“dope”土豆中的淀粉与奶油混合在一起，形成了可以想象的最浓郁的酱料，而顶部则变成了一层神奇的深金色、冒泡的乳制品外壳，让人想起奶酪。

组装起来也不会更容易，也不会更便宜。将一些土豆切成薄片——黄褐色也可以——放入砂锅中，边缘稍微重叠。撒上盐，然后倒些奶油在上面，直到它们被淹没。重复这个过程，交替加入土豆、盐和奶油，直到填满你的盘子。如果你想的话，你当然可以在里面添加一些奶酪——我不是你的老板——但是我强烈建议你在使用它之前享受这个如此简单的迭代。在 375 度的烤箱里烘烤，直到变成棕色，奶油冒泡。让它冷却五分钟左右——这样酱汁就不会洒得到处都是——然后舀到一些非常幸运的人的非常幸运的盘子里。
