# 意见是可选的

> 原文：<https://lifehacker.com/opinions-are-optional-1821468158>

欢迎回到 [**周中冥想**](http://lifehacker.com/tag/mid-week-meditations#_ga=2.170929957.440406047.1499703745-27973805.1434581949) ，Lifehacker 的每周一次的斯多葛派智慧之池探索，以及使用它的水域来反思和改善你的生活的指南。



本周精选来自马库斯·奥勒留的个人笔记。他强调了关于观点本质的一个重要事实:

![](../Images/aef335eb562a1da6957ce25a574b4aa1.png)

你总是可以选择没有意见。没有必要为你无法控制的事情而激动或困扰你的灵魂。这些事情并不是要求由你来评判。别管他们。

*沉思*，6.52



### 这意味着什么

你不需要对任何事情有意见。不受你控制的分裂性问题将继续存在，无论你支持它们还是摧毁它们，讨论它们还是隐藏它们，爱它们还是恨它们。为什么要自寻烦恼，折磨自己，让自己痛苦，就为了让别人听到你的声音？

### 从中吸取什么

在一个观点如此容易被分享并不断被用来塑造我们个人的世界里，很难没有观点。毕竟，我们对问题、人、地方和事物的个人观点是我们网上和网下大部分对话的驱动力。有些事情——比如政治和社会事务——值得我们关注，因为通过良好的意见，它们可能会被改变。

其他事情就不那么容易被左右了——无论是风味独特的食物，经过证实的科学事实，某人根深蒂固的个性，还是一部试图满足多代影迷不可能实现的期望的大预算电影。你的意见对于那些事物的存在不是必要的，也不会以任何方式改变它们。当事情完全超出你的控制范围时，你的意见毫无意义。这只是一个工具，你用它来试图向他人定义你自己，希望有人会注意到你并认可你。

但是你应该超越你的好恶，朋友！帮你自己一个忙，放弃你需要对每件事都有看法的想法。 [首先，把自己从无用的观点](https://lifehacker.com/free-your-mind-by-having-fewer-useless-opinions-1820235528#_ga=2.104884928.238213999.1513610342-27973805.1434581949) 中解放出来，感觉负担被卸下。艺术选择、事实或不同的视角不再意味着对你自身的个人冒犯。当你遇到你一无所知的事情时，在你更好地理解它之前，不要形成观点。你可以说，“我对这个还不太了解，还不能发表意见。”也许以后你可以参加讨论，*如果你认为重要的话。最终，一大堆意见曾经存在的空白会被行动、知识、感激和平静所填补。*

一句话:观点是一种选择，而不是要求。如果你不喜欢某样东西(而且它不在你的控制之内)，那就继续前进，尊重你自己的时间和幸福。如果你非常喜欢某样东西，并且觉得有必要捍卫它，也是一样。算了吧，冠军。这些战斗不需要战士。
