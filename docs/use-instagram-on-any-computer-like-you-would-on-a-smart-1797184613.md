# 像在智能手机上一样在任何电脑上使用 Instagram

> 原文：<https://lifehacker.com/use-instagram-on-any-computer-like-you-would-on-a-smart-1797184613>

与脸书和 Twitter 不同，Instagram 缺乏桌面体验。如果用户通过电脑登录，照片共享网络会限制他们可以发布、查看和参与的内容。

在某种程度上，Instagram 不希望你从电脑上传经过专业编辑的照片是有道理的。该应用程序旨在让您分享日常生活的即时照片和视频剪辑。

如果你在浏览器上登录 Instagram，你就可以浏览你的 feed，并喜欢你所有朋友的照片。您还可以访问您的通知和浏览选项卡。但你没有上传照片或观看 Instagram 故事的选项。*直到现在。*

因为任何东西都有漏洞，所以你仍然可以通过多种方式在 Windows PC 或 Mac 上使用 Instagram，就像在智能手机上使用它一样。

### 上传照片

有了一个简单的变通办法，你就不再需要通过电子邮件、短信或空投来发布照片。

如果你使用的是 Mac，并将 Safari 作为默认浏览器，进入*偏好设置*并选择*高级。*然后你必须勾选“在菜单栏中显示开发菜单”，它位于最底部。

现在，这是有点乏味的地方。你必须打开一个私人的 Safari 窗口，进入*开发，*，然后进入*用户代理，*，选择*Safari——iOS 10——iPhone。*这将让你像在手机上一样使用 Safari:当你登录 Instagram 时，你现在会在屏幕底部看到相机按钮，你可以上传照片。

如果你在 PC 上使用 Chrome，让相机按钮出现会容易一点。登录你的 Instagram，右击页面(任何地方),出现下拉菜单。选择*检查*，点击*数位板*图标。这将自动将您的页面切换到移动视图，从而允许访问相机上传按钮。

### 查看故事

Instagram 应用程序的另一个非官方桌面功能是 Stories——这是对 Snapchat 的大胆模仿。然而，如果你在网络浏览器上安装了谷歌 Chrome 扩展，你就可以观看所有视频。

Chrome 扩展的名字简单易用，而且是免费的。你只需要 [安装 Chrome IG Story](https://chrome.google.com/webstore/detail/chrome-ig-story/bojgejgifofondahckoaahkilneffhmf/related) ，下次你通过 Chrome 登录 Instagram 账户时，你会看到你所有朋友的故事都出现在最上方。
