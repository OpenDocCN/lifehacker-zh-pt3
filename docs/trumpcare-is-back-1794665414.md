# Trumpcare 回来了

> 原文：<https://lifehacker.com/trumpcare-is-back-1794665414>

上个月，奥巴马医改法案因太不受欢迎而无法在众议院获得投票 。今天，该法案的新版本又回到了桌面上，这一次，它得到了一些最保守的众议院议员的支持。以下是新法案的内容以及它将如何影响你的医疗保健。

Watch

首先，它仍然包括旧法案中有问题的所有内容，包括:

*   对医疗补助削减 8800 亿美元。从政治上来说，这是该法案最重要的功能，因为它释放了资金用于以后的减税。(共和党人可以在没有民主党支持的情况下通过这样的“预算和解”法案，但他们必须保持收入中立)。成千上万的人依靠医疗补助来获得医疗，随着这一变化，医疗补助将更加难以获得(最终是不可能的),覆盖的医疗也将比现在更少。
*   老年人的天价保险费。与旧法案一样，新法案允许保险公司向老年人收取比目前更多的费用，并且修改了补贴，使其与计划的实际成本无关。
*   **至少有 2400 万人将失去保险**，包括那些因为再也负担不起而“选择”不买保险的人。
*   免赔额会变得更高，因为现在没有规定保费必须包含一定数量的医疗费用。所以你可以买看起来很便宜的保险，但是一旦你生病了，你就完蛋了。

换句话说，我们仍然在看上个月的法案，美国医疗保健法案，但是现在有了新的修正案。这意味着新法案有旧法案的大部分问题，即我们在 之前讨论过的 [。](https://lifehacker.com/what-s-going-to-happen-to-my-insurance-if-the-obamacare-1793258760)

一份修正案草案 [泄露给了政治](http://www.politico.com/f/?id=0000015b-a790-d120-addb-f7dc0ec90000) ，但是《华盛顿邮报》 [报道](https://www.washingtonpost.com/powerpost/house-freedom-caucus-leaders-back-new-health-care-plan/2017/04/25/3c32036e-29f9-11e7-be51-b3fc6ff7faee_story.html?utm_term=.04494679a95b) 官方文本可能不会公布，直到共和党人确信他们有足够的票数通过它。修正案草案规定，各州可以自由实施以下任何一项:

*   允许保险公司向老年人收取他们想要的任何费用。ACA 将这一比例限制在 3:1；AHCA 将其改为 5:1。修正案允许各州选择“更高的比例”,如果他们喜欢的话。
*   **取消基本健康福利**。现在，如果你购买保险，它必须涵盖任何合理的医疗保健——因此，如果你有了孩子，或者得了癌症，或者需要手术，你的保险会承担部分或全部费用。根据这项新法案，各州可以自行列出基本健康福利清单。如果他们不考虑所有昂贵的东西，我们可以回到保险计划的时代，那就是 [拿走你的钱，但几乎不覆盖](http://vitals.lifehacker.com/here-are-some-of-the-crappy-insurance-plans-that-could-1793603830) 。
*   **让已有疾病的人买不起保险**。从技术上来说，保险公司不能拒绝你的保险，但是如果他们不喜欢你过去的健康问题，他们可以在个人市场上提高你的保费——是的，你个人的。如果各州参与稳定基金，他们可以允许这样做，所以这些就是你听说过的 [高风险池](http://kff.org/health-reform/issue-brief/high-risk-pools-for-uninsurable-individuals/) 。但是该法案并不要求各州用这笔钱来支付高风险患者的保险费，所以如果你有一个预先存在的状况，你仍然会被搞砸。

国会议员和他们的助手被要求在个人市场上购买保险，这限制了他们在保险法影响到他们之前可以做出多糟糕的事情。但是新的修正案说，各州的变化都不适用于国会议员及其工作人员的保险 。所以他们不想要新法案带来的保险，但是他们可以把它强加给他们的选民。

绝对 [给你的代表](https://lifehacker.com/how-to-get-through-to-your-member-of-congress-when-thei-1792084662) 打电话，但是现在还不要惊慌:这个法案离成为法律还差得很远。即使它通过了众议院，它仍然必须通过参议院——在那里它可能会因为过于保守而失败。它的受欢迎程度是否足以获得众议院的投票？我们得看看。
