# 使用这个网站自学计算机科学

> 原文：<https://lifehacker.com/teach-yourself-computer-science-using-this-site-1824287559>

Lifehacker 收集了很多对新手特别有帮助的 [【学习编码】资源](https://lifehacker.com/tag/learn-to-code#_ga=2.60802863.346445762.1522682506-1297080755.1497980211) 。但是，如果你已经做过一点编程或者只上过几门课，并且你想在理论和实践方面得到更全面的教育，那么就从 [自学计算机科学](https://teachyourselfcs.com/) 这个网站开始吧。

Watch

TYCS 的制作者说，你可以通过边走边学来维持一份可以接受的职业。但是如果你接受了完整的计算机科学教育，你将会有更好的工作保障和从事重要的、改变世界的项目的能力。如果你想帮助建立像 WhatsApp 这样非常成功的初创公司，或者为 [开源项目](https://hackernoon.com/top-6-open-source-projects-in-2017-db34b9d034a2) 做出重大贡献，你需要有深入、广泛和结构化的计算机科学知识。TYCS 由*Bradford 计算机科学学院* [的教师编写，提供自学课程，作为大学计算机学位的替代方案。](https://bradfieldcs.com/)

该课程是围绕一系列教科书和视频建立的，其中大部分是在线免费的。有一个推荐的学习顺序，总的推荐学习时间为 900-1800 小时(23 到 45 个全职周)。课程目录解释了为什么每个主题都很重要，并列出了最好的资源。对于每一本教科书或其他资源，TYCS 都会对其风格和功能给出一些见解，并告诉你何时跳过某些章节。例如:

> 【对语言和编译器】的规范介绍性文字是 [*编译器:原理、技巧&工具*](https://smile.amazon.com/Compilers-Principles-Techniques-Tools-2nd/dp/0321486811?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/teach-yourself-computer-science-using-this-site-1824287559&asc_source=&tag=kinjalifehackerlink-20)T5】，俗称“龙之书”。不幸的是，它不是为自学而设计的，而是为教师挑选 1-2 个学期的课程主题而设计的。这几乎是必不可少的，你应该选择主题，最好是在导师的帮助下。

TYCS 采用“伟大的书籍”的方法，精选经典、高质量的资源。“你不需要另外一个‘200 多门免费在线课程’列表，”TYCS 说。你需要的是最好的资源的背景。背景是你在这里的目的:学习你实践背后的理论，学习你还没有实践的东西，这样你就为一个雄心勃勃和富有挑战性的职业做好了准备。
