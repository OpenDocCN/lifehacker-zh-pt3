# 用谷歌浏览器的“价格服务员”在线砍价

> 原文：<https://lifehacker.com/haggle-online-with-price waiter-for-Google-chrome-1827008306>

Chrome:[price waiter 扩展](https://chrome.google.com/webstore/detail/pricewaiter/iabgknpjdaencfdokkmapnbjegjbojpg)——一个自己命名价格系统，乍听起来好得令人难以置信——目前只适用于 Chrome。(抱歉，火狐用户。)当你安装扩展时，你可能会注意到...绝对没有，至少一开始没有。



如果你幸运的话——我花了一段时间才在一个合格的网站上找到一个合格的产品——你会在你的浏览器顶部看到一个大的粉色条。如果是这样的话，掰掰你的指关节，因为是时候开始购物了(和削价)。

PriceWaiter 的重点是说出你愿意为某一特定商品支付的最高价格。据推测，这个数字比你在购物网站上看到的要低得多。输入你的最低金额，点击“出价”，然后你会被要求输入更多的信息:你的电子邮件，你的邮政编码，以及第三方零售商需要知道的任何其他关键细节，以便为你提供你想要的商品。

点击“提交报价，PriceWaiter 将搜索服务网络，看看是否有商家愿意以这个价格与你见面。根据 [CNET 购物](https://www.cnet.com/how-to/pricewaiter-score-hidden-discounts-by-bidding-on-products/) 看来你可以蛮不讲理，还能找到成功:

> *比如我在* [*亚马逊*](https://www.cnet.com/tags/amazon/) *上发现了一副雷朋 Wayfarer 太阳镜。价格:143 美元。嗯，我不可能花那么多钱买一副太阳镜，所以我通过 PriceWaiter 报了一个超低价:50 美元。*

> 第二天，一家名为 Gaffos 的供应商给出了还价:85 美元，节省了 40%。然后我查看了 Gaffos 网站，假设这只是它的日常价格。没有。我通常也会在那里支付 143 美元。

你没有义务为任何向你出价的商家完成购买。你还应该知道，愿意接受你的价格的商家——或者有希望与你达成妥协的商家，可能不是你最初购物的网站。所以不要指望获得亚马逊购物保护，例如，如果你在浏览时使用 PriceWaiter 试图削弱它的一个产品列表。

虽然我发现 PriceWaiter 对合格商品的选择有点缺乏，但该扩展本身并不显眼，值得在你购物时留在你的浏览器背景中——至少几周内。幸运的话，你会省下一些钱。如果没有，你可能甚至不会记得 PriceWaiter 在那里，直到你下次修剪你的 Chrome 扩展。
