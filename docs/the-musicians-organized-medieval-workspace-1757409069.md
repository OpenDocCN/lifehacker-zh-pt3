# 音乐家井然有序的中世纪工作空间

> 原文：<https://lifehacker.com/the-musicians-organized-medieval-workspace-1757409069>

丹尼斯 热爱科技，喜欢做音乐，当然，也喜欢他放在桌子周围的中世纪道具，比如他箱子上的龙和显示器前的喇叭。他的工作空间简洁明了，井井有条，但并不夸张。



丹尼斯说他喜欢科技、音乐，当然还有游戏——但和我们所有人一样，他不得不在预算有限的情况下满足自己的爱好。我们认为他已经做得很好了。这是上面照片中的一些装备:

> *   这款台式机配备了 GTX 750Ti、8GB 内存、500GB 固态硬盘和英特尔 i5，运行定制版本的 Windows 8.1。
> *   笔记本电脑是比较老的华硕 X200CA，我在大学用的，编程用的。
> *   我的双屏是三星 24 英寸和飞利浦 22 英寸，我希望有一天能升级到匹配的屏幕。
> *   我播放和录制音乐，因此为此我有一个 Cakewalk UA-25 EX 前置放大器，连接到右边的麦克风。
> *   这些话筒分别是舒尔 SM57 动圈话筒和 McCrypt BM700 电容话筒。
> *   对于音频输出，我使用 Bose 伴侣 2.0 系统。
> *   耳机是一个无线 Kitsound Beamer，我花了几块钱从易趣上买的。
> *   鼠标和键盘是微软无线 3000 系列。
> *   最后，桌子之间的角是我在中世纪市场买到的真角。
> 
> 因为我总是在寻找便宜货，所以很多装备都是二手的，或者打折的。这个工作区的总成本不到 2000 美元，与一台较新的 macbook 相同。这是一大笔钱，但考虑到我的所得，我觉得很划算。
> 
> 正如你可能看到的，我讨厌电线，所以我的大部分设置都是无线的，或者在桌子下面清理电缆。
> 
> 我房间的其余部分有几把吉他和扩音机，更多的书和一幅维京挂毯。

诚然，这是一个小镜头，但我们喜欢我们所看到的，它组织得很好。显示器支架下有放置书籍和耳机的空间，甚至还有打印机和第二台笔记本电脑，更不用说旁边的麦克风支架和放大器了。将麦克风翻转过来，您就可以开始制作音乐或进行视频通话了，无论您实际需要做什么。干得好，丹尼斯！
