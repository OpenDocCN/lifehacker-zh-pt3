# 你应该做(吃)更多的冰箱蛋糕

> 原文：<https://lifehacker.com/you-should-be-making-and-eating-more-icebox-cakes-1797032504>

我一直喜欢丰盛的教堂晚餐。我不仅欣赏在这样的活动中提供的各种各样的选择，而且我总是对这些教堂的女士们如何用看似不起眼的原料制作美味的菜肴印象深刻。这些女人不会吹嘘一切是如何“自制”和“从零开始”，而是吹嘘她们的菜肴是如何简单、快速和便宜。

这与今天普通家庭厨师的思维方式完全不同。(没人在乎你是不是自己搅了奶油，芭芭拉。用酷鞭给自己留点时间。)在这些晚宴上经常出现的一道菜是冰盒蛋糕。这款蛋糕不仅在很大程度上体现了酷搅的特点，而且除了涂抹和堆叠之外，几乎不需要任何努力。

最简单的冰盒蛋糕只有两种成分:薄饼干和鲜奶油或冷奶油。我倾向于使用后者，既是出于怀旧的目的，也是因为——多亏了科学——你真的不必担心酷鞭崩溃。一旦你有了这两个预包装的奇迹，你只需要从一些饼干开始分层。

最具标志性的 icebox 蛋糕曲奇其实不是奥利奥，而是 [著名的巧克力威化](https://www.amazon.com/Famous-Chocolate-Wafers-Ounce-Pack/dp/B000FA38ZE?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/you-should-be-making-and-eating-more-icebox-cakes-1797032504&asc_source=&tag=kinjalifehackerlink-20) ，基本都是奥利奥，没有奶油馅。出于某种原因，我找不到任何一种，但幸运的是奥利奥瘦身霜确实存在，而且效果也很好。(我也碰巧喜欢奶油馅，所以我完全同意这种交换。)除了奥利奥，任何薄饼干都可以，包括全麦饼干、姜味薄饼干或比肖夫饼干。把第一层饼干铺成你想要的任何形状，然后在上面覆盖一层奶油。

继续交替使用饼干和奶油，直到达到你想要的蛋糕高度。最后浇上一层搅打的浇头。

你也可以在两侧涂上霜，但我喜欢让对比层暴露在外。一旦建成，整个东西要放在冰箱里至少三个小时，最好是过夜。搅打过的浇头会软化饼干，形成一个可切片的，令人惊讶的可吞食的蛋糕，它会以惊人的速度消失。

然而，在你开始切片之前，你需要某种浇头。我喜欢用更多的饼干和我身边碰巧有的任何水果为我的生日锦上添花。这不是很特别吗？

现在你已经做好了最简单的冰箱蛋糕，你可以开始吃了。一旦这样做了，你就可以升级，建造越来越精致的作品。这里有一些有趣的方法，你可以用它来玩:

*   自己制作鲜奶油，并调味:我对我的酷奶油很满意，但你可能比我更喜欢。对于基本的搅打奶油，每杯重搅打奶油加入两汤匙糖粉，搅拌直到你*刚好*得到一些硬的峰值。对于巧克力鲜奶油，加入几汤匙可可粉。对于薄荷鲜奶油，在软和硬的高峰期之间加入一茶匙薄荷提取物。(这实际上可以用任何提取物来完成，我就是喜欢薄荷。)
*   **水果类:**将你最喜欢的水果切片，放在鲜奶油层上。香蕉、草莓，随便什么——都不错。
*   **撒上一些有趣的东西:**肉桂、 [蜜桃糖](http://skillet.lifehacker.com/eating-trash-with-claire-tomato-skin-salt-and-peach-sk-1794819973) 、可可粉、柑橘皮、烤坚果、蜜饯姜，甚至是真正的糖粉都可以用来增加味道和口感。
*   用冰淇淋代替鲜奶油:这需要让你的冰淇淋变得相当软，或者你可以作弊，只需 [把一堆冰淇淋三明治一个叠一个](http://skillet.lifehacker.com/the-easiest-tastiest-ice-cream-cake-is-just-ice-cream-1786021437) 。很明显，这块蛋糕应该放在冷冻室，而不是冰箱。

做冰箱蛋糕几乎没有错误的方法，我唯一强加的规则是避开烤箱，抵制自己做奥利奥的冲动。奥利奥已经是最好的自己了，烤苍蝇面对冰盒蛋糕的整个精神。
