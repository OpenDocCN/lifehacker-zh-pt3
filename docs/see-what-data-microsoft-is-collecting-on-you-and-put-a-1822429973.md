# 看看微软在收集你的什么数据，然后阻止它(也许)

> 原文:[https://life hacker . com/see-what-data-Microsoft-is-collecting-on-you-put-a-1822429973](https://lifehacker.com/see-what-data-microsoft-is-collecting-on-you-and-put-a-1822429973)

注重隐私的用户会很高兴听到微软对 Windows 10 的最新添加，它增加了数据收集方面的透明度，并最终让用户看到哪些信息正被发送给微软进行分析。Windows 诊断数据查看器 让用户更方便地看到微软正在收集什么数据，但实际上并不能帮助用户从一开始就阻止该公司染指这些数据。要停止数据收集，你需要一个不同版本的 Windows 10，一个你在当地百思买买不到的版本。

Watch

在查看任何数据之前，您需要启用数据查看。访问您的设置、隐私设置以及诊断和反馈。启用数据查看选项，使应用程序能够访问您发送到 Microsoft 的数据。该查看器目前可供 Windows Insider 计划用户使用，但一旦更新向消费者推出，应该会有更多人使用。

### **停止数据收集很复杂**

但是，假设你反对向微软发送任何数据——如果你担心个人使用习惯的安全性，这是一个合理的意见。有一种方法可以避免数据收集问题，但是除非您愿意放弃将来的更新，否则这种变通方法可能不是最好的解决方案。你将不得不使用特定版本的 Windows 10，该版本缺乏常规的用户友好更新或功能，可能会让你重新考虑你的数据隐私是否值得一些新的功能。

虽然微软向消费者和企业销售多个版本的 Windows 10，但只有 Windows 10 企业版具有禁止向微软传输数据的选项。想要自己的企业版 Windows 10？好吧，一个合适的激活版本只有在你通过微软的批量许可计划激活它时才可用，该计划是为企业而不是书呆子消费者设计的。

但是，如果您的 PC 上有 Windows 10 Pro 许可证，您可以试驾 Windows 10 企业版版本。你所需要的只是一个 [KMS(密钥管理服务)设置密钥](https://docs.microsoft.com/en-us/previous-versions/windows/it-pro/windows-server-2012-R2-and-2012/jj612867(v=ws.11)) ，这是微软提供的，用于将客户机连接到公司运营的 KMS 主机。

如果你没有 Windows 10 的企业版，或者无法获得合法的产品密钥，你可以使用前面提到的密钥来升级你已经激活的 Windows 10 Pro 机器。注意:使用此密钥升级意味着您将使用未激活的 Windows 10 版本。

这意味着你将获得安全更新，但可能会错过增加新功能的消费者友好的更新(如秋季创造者的更新)。如果你不介意，只是想要一台不监视你的 Windows PC，用微软提供的密钥升级你的 Windows 10 版本。访问设置>更新与安全>激活，粘贴 Windows 10 企业版激活密钥。
