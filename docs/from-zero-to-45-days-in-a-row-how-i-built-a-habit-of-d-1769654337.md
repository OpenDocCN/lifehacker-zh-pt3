# 从零到连续 45 天:我如何养成每天锻炼的习惯

> 原文：<https://lifehacker.com/from-zero-to-45-days-in-a-row-how-i-built-a-habit-of-d-1769654337>

今年，我把重点放在习惯上，帮助我建立新的技能和照顾我的健康。其中一个习惯就是每天早上做俯卧撑。我从完全不做俯卧撑，变成每天早上都做。当我写这篇文章时，我已经连续 45 天了。从零到一个多月不停歇，下面是我是怎么做到的。



*本帖原载于* [*贝儿贝丝库珀的博客*](http://blog.bellebethcooper.com/pushups.html) *。*

### 从小处着手

自从发布了我的课程，[](http://habits.bellebethcooper.com/)*，我一直在关注培养更多的新习惯。我关注的技巧之一是从小处着手。我是这种方法的大力支持者，其他培养习惯和改变生活方式的专家也是，比如微小习惯 的创造者[BJ·福格，以及禅宗习惯](http://tinyhabits.com/) 的创造者 [利奥·巴巴乌塔。](http://zenhabits.net/floss/)*

*从小处着手的精髓来自两个部分:**让你的习惯变得如此容易以至于你不能说不**，以及**专注于一致性**。*

*让我们从简单开始。每天早上我做五个俯卧撑。那根本不多。这不是我所说的全面锻炼。流一点汗是不够的。不过这没关系，因为比零多了*。**

*当你开始一个新的习惯时，你会想找到尽可能少的习惯，并且仍然有所感觉。我本来可以从一个俯卧撑开始，但一个或两个后我就没什么感觉了。五个俯卧撑刚够感觉有点吃力，还不够感觉太难上手。趴在地板上做五个俯卧撑是小事一桩，这正是你刚起步时想要的:小事一桩，你不能说不。*

*从小处着手的第二部分是关注一致性。你应该以质量为目标，而不是数量。这不是要连续做 50 个俯卧撑，或者减肥，或者拥有六块腹肌。这是关于做俯卧撑，不管有多少，每天*。**

**我最近开始在俯卧撑后增加五个深蹲。但是几天后，我发现我不想每天都做深蹲。有时候我会赶着做俯卧撑，所以我会跳过深蹲。**

**这没关系。因为我只要做五个俯卧撑就能保持连胜。**

**只有当我有信心每天早上做五个俯卧撑，没有提醒，无论如何，我才会专注于做更多的俯卧撑，或者增加更多的锻炼，成为我无借口日常习惯的一部分。**

### **保持焦点对准**

**我总是专注于建立一个新习惯。再多一些，我就会忘记一些，或者我每天都努力保持记忆，因为我的注意力分散了。**

**为了帮助我专注于目前正在培养的习惯，我把它记在两个地方:我的笔记本和我的任务清单。**

**我 [用笔记本](https://exist.io/blog/strikethru/) 记下每天、每周、每月的待办事项清单。在每天清单的顶部，我写下我正在努力养成的习惯。当我完成了一天的习惯，我就把它划掉。这是一个有用的提醒，无论何时我看我的笔记本，我都需要完成我的习惯，如果我还没有完成的话。**

**我记下我习惯的另一个地方是我的任务列表。当我做我的月度回顾并选择一个习惯在下个月继续时，我为这个新习惯制定了一个任务，截止日期是这个月的最后一天。我 [确保这个任务每天都在我的主任务列表](http://blog.bellebethcooper.com/todoist.html) 中可见(你可以根据你使用的应用程序以不同的方式做到这一点——例如，使用过滤器或保存的搜索，添加一颗星，高优先级状态，或者为每月的第一天设置开始日期)。**

### **(笨拙地)跌倒**

**除了从小处着手，积累习惯可能是我发现的建立全新习惯的最有用的技巧。积累习惯意味着用一个现有的习惯(即使是你没有刻意养成的习惯，比如起床、洗澡或刷牙)来触发你的新习惯。**

**对于俯卧撑，我用打开淋浴作为触发器。每天早上，我准备好淋浴，打开水，然后在水变热的时候蹲下做五个俯卧撑。当我做更多的俯卧撑时，我必须调整这一点，这样我就不会浪费水，但现在我只是更好地利用我通常站在周围避免冷水的时间。**
