# 使用基于位置的提醒来打破坏习惯

> 原文：<https://lifehacker.com/use-location-based-reminders-to-break-bad-habits-1819292821>

你(傻瓜):“提醒是为了好事。”我(睿智):“提醒是为了坏事。”

Watch

在手机上使用基于位置的提醒来提醒自己不要被自己的坏习惯所迷惑。当你跌跌撞撞地离开你最喜欢的酒吧时，提醒你回去结账。或者当你第一次磕磕绊绊地走向它时，得到一个提醒，提醒你正在努力少喝。在前任的住处、昂贵的餐馆或者你经常去的商店设置反提醒。

这条建议来自超级泰森兄弟 播客联合主持人戴夫·J，他的电话提醒他，在每次回家的路上，*不要去蛋糕店破坏他的饮食。*
