# 一些汽车经销商终于明白了如何在网上销售二手车

> 原文：<https://jalopnik.com/some-car-dealers-have-finally-figured-out-how-to-sell-u-1820761861>

即使在网上可以买到陈年牛奶 ，你也不妨试着锁定一辆好的网络星期一汽车。虽然一些经销商已经努力使在线购买二手车变得容易，但太多的经销商仍然有不准确的 [列表或普遍不合作的](https://jalopnik.com/what-s-your-biggest-disappointment-with-shopping-for-ca-1819939035) 。

网上购买二手车本来就比购买新车型复杂。当你知道你想要哪辆新车的时候，你基本上只是在比较价格，而不是产品。但对于二手车来说，有很多因素需要考虑，比如年份、装饰、里程、保修余额和整体状况。找到二手车最佳交易的关键不是关注你从要价中节省了多少，而是关注市场上的最佳整体价值。 为了做出这样的评价，你需要掌握尽可能多的信息。

我最近在一些不同的二手车搜索中注意到，一些经销商正在利用全面的数字“包”来帮助简化流程。当我在德克萨斯州为一位客户购买二手大众高尔夫 Sportwagen 时，我遇到了一家梅赛德斯经销商，他列出的清单看起来相当不错。这个经销商使用了一个叫做 AutoiPacket 的工具来显示这辆车的信息。

现在，当谈到经销商网站时，我不容易被打动，但这使我的一天。有一个到 CarFax 报告的简单链接，这很好，因为当他们只需点击一下就可以制作报告时，谁不讨厌浪费时间向经销商索取信息。大多数地方会链接到 CarFax 或类似的历史报告，就像这个经销商所做的那样，但真正使这个特别的是在 AutoiPacket 上找到的另外两个文档。

首先是“管理系统更新项目”部分它与这辆车的原始车窗贴纸相关联——这是一个在不同品牌的经销商出售的二手车上发现的超级罕见的珍闻。

这让我可以看到所有已安装的选项和软件包。它还让我将原始的 MSRP 与当前的销售价格进行比较，以帮助确定最重要的价值分析。我不仅可以看到一个窗口标签，而且一些使用自动数据包的经销商甚至可以链接到一个“服务历史”，其中详细列出了他们可以访问的所有记录。现在，二手车买家可以看到维护是最新的，任何最近的升级，如新轮胎或刹车。

但我最喜欢这个数字包系统的一点是，一旦你的电子邮件地址出现在经销商的数据库中，当价格下降时，你会自动收到警报。我知道许多买家会发现这个功能非常节省时间，因为它消除了不断监测价格调整的需要。

一个好的销售人员不会想把某人拖进展厅，结果却发现这辆车并不是客户想要的，如果只需点击几下鼠标就能获得他们需要的信息，那么购买者宁愿省去打电话和去经销店的麻烦。

越来越多的经销商应该注意到，技术可以向潜在买家提供尽可能多的二手库存信息。隐瞒这些信息是惹恼甚至吓跑严肃顾客的好方法。
