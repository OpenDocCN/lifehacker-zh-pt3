# 制作“空中护目镜”来寻找你掉在水里的物品

> 原文：<https://lifehacker.com/make-air-goggles-to-search-for-items-youve-dropped-in-1796534055>

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-hfBoJTiVETk&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-hfBoJTiVETk&start=0) 

如果你把东西掉进了游泳池，而从水面上看不到它，这个超级简单的小技巧会有所帮助。



你知道如何在水下拉一个杯子，开口朝下，它仍然充满空气。这就是来自 youtuber[MicBergsma](https://www.youtube.com/channel/UCTs-d2DgyuJVRICivxe2Ktg)的 [这招](https://www.youtube.com/watch?v=hfBoJTiVETk) 的原理。只是用手在眼睛周围做杯子，慢慢把脸低进去。现在你已经得到了一副免费的“空气护目镜”环顾四周，寻找你的钥匙，你孩子的玩具，或者任何你可能不小心掉在那里的东西。

如果你在把脸浸入水中时没有吸入足够的空气，你可以从鼻子里放出一些气泡来补充你的全天然护目镜。当你完成后，闭上眼睛，抽出你的手，然后退出水面。
