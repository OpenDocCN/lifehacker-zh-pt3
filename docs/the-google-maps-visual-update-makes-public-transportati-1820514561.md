# 谷歌地图的视觉更新使公共交通更容易找到

> 原文：<https://lifehacker.com/the-google-maps-visual-update-makes-public-transportati-1820514561>

谷歌 [正在推出其地图应用](https://blog.google/products/maps/google-maps-gets-new-look/) 的更新，给它一个更新的视觉风格，更容易识别的标记，以及上下文感知的信息。如果你在使用公共交通工具时容易迷失方向，或者在公路旅行中对寻找加油站感到沮丧，更新可以更容易地知道你要去哪里，你周围有什么，如果你错过了你的车站，你会在哪里结束。



谷歌地图最重要的更新来自于信息呈现的形式。该应用程序将根据您的交通方式向您显示相关信息。现在，查找公共交通路线会将火车站、公交车站和地铁路线放在前面和中间，而使用谷歌地图查找驾驶方向会显示加油站等相关地标以及交通信息。如果你和我一样，你会发现相关信息的高亮显示对你错过换乘火车后如何回家更有帮助(我坐了出租车)。

谷歌地图改进的配色方案和增加的更多图标使其更容易指示特定的兴趣点，并快速确定您的下一个潜在目的地，而无需越来越接近您的位置。颜色选择本身并没有太大的区别，所以可能需要一些时间来习惯直观地理解哪个蓝色阴影表示购物中心，哪个实际上是火车站。尽管如此，更精细的分类和更清晰的图标应该更容易确定你的医生办公室离最近的潜水酒吧咖啡馆有多近。

谷歌表示，你可以期待通过软件更新看到谷歌地图应用程序的视觉更新，而依赖谷歌地图 API 的服务(如 Android Auto、谷歌搜索和谷歌助手)将在未来几周内收到更新。
