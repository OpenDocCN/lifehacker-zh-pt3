# 这些是目前网飞最恐怖的电影

> 原文：<https://lifehacker.com/these-are-netflixs-scariest-movies-right-now-1825022326>

我喜欢好看的恐怖电影，不仅仅是因为它们对你有好处 。如果你正在寻找一部关于网飞的惊悚片或恐怖片，这些是目前该网站上最恐怖的电影。你能搞定他们吗？



根据爱德曼营销公司(Edelman marketing)提供的以恐怖片为主的网飞流媒体数据，这些电影(排名不分先后)让观众在高潮到来之前就离开了:

*   [维罗尼卡](https://www.netflix.com/title/80109295)

*   [幽居病](https://www.netflix.com/title/80079473)

*   [屠杀公园](https://www.netflix.com/title/80098487)

*   [墨西哥巴尔巴罗](https://www.netflix.com/title/80079271)

*   [食人鱼](https://www.netflix.com/title/70108986)

*   [生](https://www.netflix.com/title/80115405)

*   [牙齿](https://www.netflix.com/title/70059630)

*   [《人类蜈蚣 2:全序列》](https://www.netflix.com/title/70212952)

*   [虚空](https://www.netflix.com/title/80172701)

具体来说，数据指出，大多数观众在观看了大约 70%的这些电影后倾向于放弃。不可否认，有几个原因会让人们在那个时候停止看电影。这部电影可能变得太暴力、太生动，甚至太愚蠢。但是如果这些电影烂到那种程度，人们会很早就不看了。当你有一个像网飞一样的在家自助看完的节目时，没有多少人会看一部糟糕的电影整整一个小时，希望它会变得更好。在这种类型的电影中，70%的分数通常是在废话击中粉丝之前，所以这不是一部电影太紧张而无法完成的最糟糕的指标。

我看过这些电影中的大部分，可以证明出于各种原因，其中几部相当棒。特别是，*变戏法*，*维罗尼卡*，*牙齿*，*虚空*都是我推荐的电影，如果你还没有看过的话( [也有*仪式*，最近上映，没有列出](https://www.netflix.com/title/80217312) )。*幽居病*、*原始*，其余的可能不会让你大吃一惊(尤其是*食人鱼*，这是故意的 campy)，但它们本身就令人不安，值得大惊小怪。
