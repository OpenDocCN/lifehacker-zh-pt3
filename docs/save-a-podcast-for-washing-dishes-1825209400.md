# 为洗碗保存一个播客

> 原文:[https://life hacker . com/save-a-podcast-for-洗碗-1825209400](https://lifehacker.com/save-a-podcast-for-washing-dishes-1825209400)

我喜欢洗碗。这是我为数不多的不需要任何脑力劳动的责任之一。你可能不喜欢洗碗。你可以尝试 [让他们沉浸在一种不诚实的拖延行为中](https://lifehacker.com/soaking-dishes-is-a-lie-1825105525) 。有个想法:挑一个你最喜欢的播客，只在洗碗的时候听。

Watch

至少，你应该在洗漱的时候听播客。如果你通过蓝牙扬声器或者仅仅是你的手机来听，你可以听到足够多的随意对话播客。如果你通过耳机收听(这里无线耳机有助于避免混乱)，你可以充分享受那些分秒必争的高度工程化的节目之一。这是精神麻醉剂播客。

通过*为菜肴保存*一个播客，你会做得更好。这是作为激励的播客。你只有在洗碗的时候才能听节目。(根据播客的频率和菜品的频率，按需调整。也许你只到*开始*一集就有菜了。所以，选择一个你真的很想听的节目，而不是一个听起来很乏味的节目。

这里有一个风险:即使播客开始改善你对菜肴的看法，菜肴也可能损害你对播客的看法——有点像 [用你最喜欢的歌曲作为你的闹钟](https://lifehacker.com/dont-set-your-alarm-to-your-favorite-song-1821886411) 。如果这种情况开始发生，尝试一段时间不同的播客。只有当你把播客看得比不洗碗更重要的时候，这一招才管用。

你可以在不同程度上把这个应用到各种差事上；吸尘有点难；擦拭浴室很容易。你走动或弯腰越多，你就越不需要有线耳机或手机放在你的衬衫口袋里。(我摔了十几次手机才接受这个。)

如果你正在为任务寻找一个短播客，我推荐 [歌曲爆炸器](http://songexploder.net/) ，NPR 的 [流行文化欢乐时光](https://www.npr.org/podcasts/510282/pop-culture-happy-hour) ，或者 [沙漠神谕电台](http://www.desertoracle.com/radio/) 。
