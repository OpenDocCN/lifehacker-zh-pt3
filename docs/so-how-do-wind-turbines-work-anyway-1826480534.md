# 那么风力涡轮机到底是如何工作的呢？

> 原文：<https://gizmodo.com/so-how-do-wind-turbines-work-anyway-1826480534>

我们大多数人都同意 [更清洁的能源是一件好事](https://earther.com/more-republicans-support-renewable-energy-than-fossil-f-1825968623) ，但是惊人数量的人仍然 [不希望风力涡轮机建在他们的后院](https://en.wikipedia.org/wiki/NIMBY) 。这是一种耻辱，因为驱动这些坏男孩的技术不仅是无碳无污染的——简直酷毙了。

几年前，当我参观美国第一个海上风力发电场 时，我亲身体验了这一点。当你近距离看到涡轮机时，你首先会被它的*标度*所震惊。这些不是一些破烂的小“风车”(一种 [旧技术](http://wind.jmu.edu/communityengagement/millvsturbine.html) ，我们的总统 [经常把](https://twitter.com/realdonaldtrump/status/509436043368873984?lang=en) [与](https://twitter.com/realdonaldtrump/status/273153472323543040?lang=en) [涡轮机与](https://twitter.com/realdonaldtrump/status/279654434999525376?lang=en) 混为一谈)而是摩天大楼般高的钢塔，上面有巨大的玻璃或碳纤维叶片，是为空气动力学精度而设计的。这只是你能看到的东西。

风力涡轮机背后的真正魔力发生在这个被称为机舱的结构顶部的小盒子里(好吧，是几百吨重的钢铁控制中心)。在那里，来自旋转叶片的动能被发电机转化为电能。观看上面的视频，了解更多关于普通涡轮机如何发电的细节。

额外的事实:风力涡轮机的工作通常需要很高的詹姆斯·邦德技能。例如，这位名叫 的妇女利用她高超的攀爬技巧成为了一名叶片维修工人，这份工作包括从涡轮机的鼻锥上用绳索取下来，并在数百英尺的高空进行维修工作。

所以，下次当你听到有人抱怨“风车”破坏了海景时，也许可以花点时间解释一下涡轮机是多么了不起的工程。这可能会改变他们的想法，如果我们能够改变足够多的想法，那么，我们 [可以用无碳微风为整个海岸线](http://energy.gov/eere/wind/downloads/national-offshore-wind-strategy-facilitating-development-offshore-wind-industry) 供电。见鬼，几个红州的 [已经领先冲锋](https://earther.com/red-states-are-leading-the-wind-energy-charge-1825334671) 。

***更正**:这篇帖子的早期版本称，涡轮机将动能转化为更大量的电能，这将违反热力学定律，而涡轮机尽管很酷，却不能这样做。作者为这个愚蠢的错误道歉。*
