# 离别歌曲桌面

> 原文：<https://lifehacker.com/the-departure-songs-desktop-1781920112>

我们已经看到了一些简单的桌面和一些繁忙的桌面，但这款来自 reader[Peregerine](http://kinja.com/pergerine)的桌面相当不错。它保持了中心的大部分空间，所以你有足够的空间来工作，但也在外围添加了一些有用和信息丰富的小部件和工具。



如果你想为你自己的 Windows 机器做这个，你首先需要的是一个 Rainmeter 的拷贝，以及一些充分利用它的技巧。我们这里有 [一本入门指南你可以用](https://lifehacker.com/how-to-create-an-attractive-customized-desktop-hud-wit-5828789) 。

现在，如果你想得到同样的外观，这里是 Peregerine 使用的组件:

*   [壁纸](https://www.behance.net/gallery/27933607/We-Lost-The-Sea-Departure-Songs-Complete-Artwork) ，作者 [马特·哈维](http://graphicatharsis.com/) (因为佩里格林喜欢马特的乐队， [我们失去了大海](http://welostthesea.com/) )
*   用于 Windows 的 [雨量计](http://www.rainmeter.net/) 系统调试和监控工具
*   雨量计的 [箭头发射器](http://altairseven.deviantart.com/art/Arrow-Launcher-212628381)
*   [二进制时钟皮肤](http://helkin86.deviantart.com/art/Rainmeter-Binary-Clock-v2-141831873) 为雨量计创建二进制时钟
*   雨量计的 [彩色喷泉音乐可视化器](http://alatsombath.deviantart.com/art/Fountain-of-Colors-desktop-music-visualizer-518894563)
*   [色流皮肤](http://wyzzymoon.deviantart.com/art/Patrick-Muff-s-Color-Flow-for-Rainmeter-532426147) 为雨量计创建日期显示
*   用于雨量计的 [音乐皮肤](http://mahm0udwally.deviantart.com/art/MUSIC-488094142) 在左上角创建当前播放显示
*   [令牌图标包](http://brsev.deviantart.com/art/Token-128429570) 和 [令牌启动宝珠](http://satukoro.deviantart.com/art/Token-Style-Start-Button-205471512)

大概就是这样。和通常的 Rainmeter 桌面一样，如果你不喜欢某个特定的组件，或者如果你认为你更想要一些不同的东西，你可以带走或留下特定的组件。总而言之，这是一个很好的实现好看的方法。如果你喜欢你所看到的，去 Pergerine 的 Kinja 博客，链接如下，分享你的想法。
