# 用这些工具追踪佛罗伦萨飓风

> 原文:[https://life hacker . com/track-hurricane-Florence-with-these-tools-1828938960](https://lifehacker.com/track-hurricane-florence-with-these-tools-1828938960)

预计飓风佛罗伦萨最早将于本周四在大西洋沿岸登陆。虽然沿海的弗吉尼亚州、北卡罗来纳州和南卡罗来纳州预计将受到最严重的打击，但居住在东海岸的任何人都应该密切关注风暴的进展。

Watch

截至今天早上，佛罗伦萨 [已经升级为 4 级风暴](https://twitter.com/NHC_Atlantic/status/1039182225030479882) ，预计强风和暴雨将于周三晚上或周四早上开始。 [根据国家飓风中心](https://twitter.com/NHC_Atlantic/status/1039168014837575681) ，南卡罗莱纳州到大西洋中部地区的人们应该有一个飓风计划，并准备听从当地官员的建议。如果您所在的地区可能会受到风暴的影响，请随时关注以下资源:

*   美国国家飓风中心的 [专用佛罗伦萨页](https://www.nhc.noaa.gov/graphics_at1.shtml?cone)T3】
*   Accuweather 的 [佛罗伦萨预测与跟踪页面](https://www.accuweather.com/en/hurricane/atlantic/florence-2018)
*   本 [互动图来自 CNN](https://www.cnn.com/interactive/storm-tracker/)
*   天气频道的 [互动佛罗伦萨追踪器](https://weather.com/storms/hurricane-central/AL062018)
*   NBC 新闻' [佛罗伦萨直播](https://www.nbcnews.com/video/watch-live-tracking-hurricane-florence-1316244547637?v=railb&)
*   国家飓风中心官方推特账号，加上账号[【NHC 大西洋】](https://twitter.com/NHC_Atlantic) 和 [风暴潮单位](https://twitter.com/NHC_Surge?lang=en)

一如既往，当地气象学家和新闻机构是你获得最新预报和疏散协议的最佳来源，所以在制定 [风暴准备计划](https://lifehacker.com/how-to-stay-safe-during-a-hurricane-1798391969) 时，一定要咨询他们。
