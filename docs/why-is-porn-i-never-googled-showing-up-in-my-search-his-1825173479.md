# 为什么我从未搜索过的色情内容会出现在我的搜索历史中？

对于 [Tech 911](https://lifehacker.com/c/tech-911#_ga=2.256828427.1626577895.1522540410-396842925.1520800403) 来说，没有太奇怪的问题，也没有太小的问题。本周，我们探讨了一位读者的问题，她在谷歌历史中发现了一些她声称从未搜索过的东西...从那以后只会变得更加有趣。

### 没有过度分享这回事

在来自 life hacker reader**Tiffanie**的邮件中:

*“我有一个在过去 3 年里一直存在的问题。所以，我实话实说，我和我男朋友各自用手机看黄片。我们每个人都通过谷歌查看了彼此的手机活动，当他看我的手机时，里面有一些我从未搜索过的项目，比如聊天室的实时摄像头。我从来没有在我的谷歌搜索中输入过这个，但我怀疑它们是从色情网站弹出来的。我想知道它们如何以及为什么会出现在我的搜索中。还有其他一些我应该搜索过的东西，但我知道事实上我没有。你能帮助我吗？"*

好吧。

在你浏览的过程中，很有可能会出现弹出窗口...*会话*...这些都出现在你的谷歌历史中。我刚刚(为《科学》杂志)做了一个快速的调查，这些烦人的广告可能会经常出现在一些更知名的网站上，当一个人需要一点日常娱乐时，人们可能会首先想到去这些网站。

弹出式广告可以出现在桌面和移动设备上——前者值得注意，因为如果你在智能手机和桌面(或笔记本电脑)上使用同一个谷歌账户，并在你的设备上同步其内容，你的搜索历史将包含你看过的所有内容。因此，有可能你在一个位置得到了弹出窗口，并没有真正考虑它，而是想知道为什么你的移动(或谷歌)历史充满了你不记得在智能手机上查看过的东西。

也有可能，尽管可能性较小，您登录了另一台您不知道的电脑，而使用该浏览器的人正在您的同步历史记录中添加奇怪的东西。如果是这种情况，你可能会注意到更多你不记得访问过的网站，甚至可能是一些非常奇怪的账户活动(比如你没有发送的电子邮件)。以防万一，打开谷歌的 [我的账户](https://myaccount.google.com/?pli=1) 页面，点击设备活动&安全事件，点击“最近使用的设备”部分的查看设备，点击每一个授权的设备，然后删除它。

你必须经历一个有点烦人的过程，将你经常使用的设备重新连接到谷歌，但至少你会更加安心，因为知道别人没有在你的账户上浏览。(我仍然不相信这就是问题所在，但确定一下也无妨。)

当你这么做的时候，也考虑修改你的密码，启用双重认证*，以防*除了你或你男朋友之外的其他人访问你的账户。

既然我们谈到了这个话题，现在是讨论允许任何人——甚至是你的男朋友——查看你的电子资料是否最有意义的时候了。不要太爱医生而不是技术医生，信任在一段关系中很重要，但你也应该有一点隐私。

谈论你喜欢在线观看和搜索的内容——见鬼，甚至在吃饭时分享你的最爱——但是让别人访问你的帐户和/或设备？从安全角度来说，这不是个好主意。(至少，这是我的假设；完全有可能你们只是一起浏览你的 Chrome 历史，这听起来不像是最有趣的约会，但是，*嘿*，一起泡夜店的情侣会一直在一起。或者什么的。)
