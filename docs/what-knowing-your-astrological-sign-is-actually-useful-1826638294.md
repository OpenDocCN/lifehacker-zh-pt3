# 知道你的星座到底有什么用

> 原文：<https://lifehacker.com/what-knowing-your-astrological-sign-is-actually-useful-1826638294>

我不相信占星术，但我知道我的星座。(我是传统系统中的人马座，蛇夫座的 [拘泥于天文学家的](https://io9.gizmodo.com/why-did-your-zodiac-sign-change-we-asked-the-astronome-5733004#_ga=2.157289594.11446851.1528118734-3846207152.1521480874) 。)你也可以从了解自己的经历中受益！只是 [不是奥兹博士认为的](https://gizmodo.com/dr-ozs-latest-health-advice-check-your-horoscope-1826618960#_ga=2.157289594.11446851.1528118734-3846207152.1521480874) 。



在奥兹博士的网站上，占星家丽贝卡·戈登(Rebecca Gordon)解释说，射手座的人容易出现臀部问题，白羊座的人容易出现偏头痛，而癌症不是你想象的那样，只是“水分潴留、囊肿和低能量”

让我们回顾一下，参考一些实际的科学，并考虑了解你的星座能真正为你做些什么。例如:

*   你可以享受阅读星座运势，这是一种以第二人称讲述的快闪小说形式。
*   你可以对其他人进行概括(“啊，这是狮子座应该做的事情”)，为你无论如何都会做出的抱怨或赞美增加一些额外的深度。
*   你可以买带有你的星象标志的钥匙链和其他东西(只有当你有一个看起来很酷的星象标志时才推荐)。
*   你可以 [阅读这个](https://gizmodo.com/whats-the-best-astronomical-object-in-your-zodiac-sign-1825539079#_ga=2.198184815.11446851.1528118734-3846207152.1521480874) 并立即拥有一个最喜欢的天体，以防你在聚会上被问到这种问题。

但要明确的是，你的星座并没有*而不是*告诉你 [你容易患上什么健康状况](https://gizmodo.com/dr-ozs-latest-health-advice-check-your-horoscope-1826618960) ，或者你是否以所有错误的方式寻找爱情，或者你将成为什么样的人。希望你会成为那种不相信废话的人。
