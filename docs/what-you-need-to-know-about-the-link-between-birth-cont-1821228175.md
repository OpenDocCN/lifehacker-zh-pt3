# 关于节育和癌症之间的联系，你需要知道什么

> 原文:[https://life hacker . com/what-you-need-to-know-the-link-between-birth-cont-1821228175](https://lifehacker.com/what-you-need-to-know-about-the-link-between-birth-cont-1821228175)

最近的一项研究证实了乳腺癌和激素避孕之间的联系(包括避孕药，但也包括贴剂、植入物和一些宫内节育器)。给自己两秒钟抓狂，然后深呼吸，我们来讨论。

这里的消息实际上是*而不是*这两者之间根本就有联系，但是几十年前发现的联系今天仍然适用，即使现代药丸中的激素剂量较小。为了更好地解释研究发现，我喜欢 NPR 的。尤其是关于我们真正谈论的风险有多大的这一部分:

> 美国癌症协会的流行病学家米娅·高德 说，激素避孕带来的风险类似于身体不活动、成年后体重过度增加或平均每天喝一杯或更多酒精饮料带来的额外乳腺癌风险。

因此，如果你不会过分担心缺乏健身习惯或每晚喝一杯葡萄酒，你可能也不应该对此感到惊慌。如果你*有点害怕，你可能想知道铜宫内节育器不含任何激素，对许多人来说它们是一个极好的选择。*

但问题是。尽管激素避孕会增加患乳腺癌的风险，但它也会降低患卵巢癌、子宫内膜癌以及可能的结直肠癌的风险。总的来说，激素避孕药似乎要么是中性的，要么对你的整体癌症风险有益。
