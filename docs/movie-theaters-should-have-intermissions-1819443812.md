# 电影院应该有休息时间

> 原文：<https://lifehacker.com/movie-theaters-should-have-intermissions-1819443812>

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-Mw91RJ_m_7g&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-Mw91RJ_m_7g&start=0) 

电影院是一场噩梦。它们很贵，很吵——但最重要的是，我不能开始和停止电影。在家里，我很少看完一部电影而不停下来吃点零食和上个厕所。仅此一点就让房屋租赁比电影院、 [、电影票或无](https://lifehacker.com/moviepass-now-offers-unlimited-movies-in-theaters-for-1797865400) 要好。有一件事会改变我的想法:剧院的幕间休息。

Watch

90 年代末， [平均电影时长超过 120 分钟](http://www.randalolson.com/2014/01/25/movies-arent-actually-much-longer-than-they-used-to-be/) ，此后再未跌破。电影院试图卖给你一大杯可乐，然后让你一动不动地坐两个小时或更久。到最后，你腰酸背痛，眼睛疲劳，无法欣赏演职员表，也无法安静地反思刚刚看到的，因为大家都在冲厕所线。

这个问题如此普遍，甚至有一个名为 [RunPee](http://runpee.com/) 的应用程序，它会建议你在低头撒尿时可以跳过哪些场景。不管你反对幕间休息的理由是什么，你都得承认“错过电影的一部分，这样你才不会尿湿自己”并不是一个令人愉快的现状。

在以前的电影黄金时代，当电影变长时，他们会添加幕间休息。像*宾虚*、*阿拉伯的劳伦斯*、*乱世佳人、*和*七武士*这样的史诗电影被打断了——有时在所有的放映中，有时只是在早期放映中——这样它们就可以很有分量而不会使人筋疲力尽。但是即使三小时的史诗重新流行，幕间休息却不是。在过去的二十年里，只有少数电影增加了幕间休息。昆汀·塔伦蒂诺在 [中困住了一个早就放出*的可恨的八个*](http://ftw.usatoday.com/2015/10/quentin-tarantinos-new-film-will-have-a-long-minute-intermission) ，却没有在广放。显然，像《诸神与将军》这样的四小时电影才能赢得休息时间。

我不是第一个建议恢复中场休息的人。 [Slate 的文化评论家艾莎·哈里斯](http://www.slate.com/blogs/browbeat/2014/12/08/movie_intermissions_are_necessary_in_the_age_of_long_movies_like_interstellar.html) 和 [编剧内森·哈特曼](https://theouttake.net/what-happened-to-the-movie-intermission-45afcda475ae) 都在 2014 年彻底为其辩护；《旧金山纪事报》评论家彼得·哈特劳布早在 2010 年就要求这么做了。正如所有这些专业电影观众指出的那样，幕间休息不仅对观众有好处；它们对影院也有好处，影院会获得另一轮优惠销售。剧院老板向哈特劳布证实，他们希望有中场休息的选择。但这是不允许的。只有电影公司可以决定一部电影是否有休息时间。

间歇不会破坏观众的沉浸感或故事。他们没有毁掉《音乐之声》或《T2 2001:太空漫游》。(一位评论家认为《T4》中的幕间休息是故事中不可或缺的一部分。)它们不会破坏现场剧场，在那里任何超过 100 分钟的演出都有间歇。与电视上不断的商业广告不同，电视广告将一个节目分割成疯狂的小片段，幕间休息只是将一个庞然大物分成易于管理的两半。想到看背靠背的几集*权力的游戏*或者*西部世界*；HBO 不会梦想一集接一集，没有一两分钟的喘息时间。

但是仅仅在任意点增加间歇会引起问题。剧本是在考虑幕间休息的情况下创作的；昔日的史诗也是如此。宝莱坞电影也是如此，它们仍然普遍包含“幕间休息”(这是英式英语中的叫法)。正如电影作家 Sampada Sharma 解释的那样，宝莱坞电影分为两幕，就像现场戏剧一样，而不像好莱坞典型的三幕结构。编剧和导演需要在幕间休息时计划他们的故事。这是一个巨大的要求！但要求每个人在黑暗中坐两个半小时也是一样，因为你拒绝剪掉另一个场景。这并不意味着改变整个情节；这意味着在一部电影的中间，从十几个拍子中调整一两个拍子。

想象一下额外的好处:不停发短信的人可能最终会放下他们该死的手机，如果他们知道休息即将到来的话。观众可以谈论电影的前半部分，为后半部分做准备。每个人都可以伸展他们的腿，这样电影就不会感觉像飞机飞行。当然，人们可能会中途溜进其他电影，但这并不是说他们不能这样做。人们需要保留他们的座位——当然，没问题。这在 60 年代行得通，现在也能行得通。

也许，仅仅是也许，所有这一切可以让更多的人重返影院，而不是等到一切都来到网飞。也许它能说服我买电影票，每月去一次电影院，而不是一年一次。也许它甚至会说服我买爆米花。《幕间休息》可以做一件业内人士非常渴望的事情，3D 和 IMAX 做不到的事情:让“去看电影”再次变得特别。我们都去大厅，给自己买点吃的。
