# 对于像我这样的懒人来说，这些应用程序是完美的省钱工具

> 原文：<https://lifehacker.com/these-apps-are-the-perfect-money-saving-tools-for-lazy-1778298912>

我不擅长存钱，我讨厌学习个人理财，而且在购买我想要的东西时，我倾向于“及时行乐”的心态。幸运的是，一些自动储蓄应用程序帮我做了大部分困难的事情，让我对自己的钱有了更强的掌控。

Watch

当然，市面上有很多很棒的预算软件，比如 [Mint 或者 YNAB](https://lifehacker.com/budgeting-software-showdown-mint-vs-you-need-a-budget-1764607246) ，但是我对这些软件太懒太没有耐心了。然而，在过去的一年里，我仅用这三个应用程序就净赚了超过 3000 美元的额外储蓄: [数字](https://digit.co/) 、 [橡子](https://www.acorns.com/) 和 [自我提示](https://itunes.apple.com/us/app/tip-yourself-save-money-build/id987961939?mt=8) 。对一些人来说，这可能不是很多钱，但与我之前 0 美元的记录相比，这是一大笔钱。自动化我的储蓄也帮助我养成了更节俭的心态。当应用程序为我省钱时，我意识到我可以少花钱。具有讽刺意味的是，*而不是*学习如何储蓄帮助我更好地管理我的钱。我只是需要有人为我牵缰绳，带路。

### **数字让钱远离我喜欢花钱的手**

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-9GjBcMIIqS0&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-9GjBcMIIqS0&start=0) 

当我看到我想买的东西时，没有什么能阻止我去买它。我是一个零售商的梦想，所以当务之急是在我变成一个打算在玩具店花掉零用钱的孩子之前，把我钱包里的钱拿出来存入银行。这就是 [数字出现在](https://lifehacker.com/digit-finds-extra-money-in-your-budget-and-automaticall-1681244660) 中的原因。

[Digit](https://digit.co/) 链接到您的网上银行账户，观察您的收入和支出模式，谨慎地选择一美元金额供您储蓄，并自动将它从您的银行账户转移到您的储蓄账户。这听起来像是一个黑客在不被发现的情况下偷钱的聪明方法，的确如此，但这个黑客是为你工作的。你在从自己身上“偷”钱，并把它放在一个单独的储蓄账户里。如果你的关联账户有所好转，并且没有持续的资金流入，Digit 只会在这里或那里节省几美元。但如果你资金充裕，Digit 会采取更积极的行动，开始储备更多资金。无论怎样，你都不会注意到。

现在，当我去买我想要的东西时，我知道我已经存了钱，我在支票账户上看到的金额更能代表我在不完全破坏我的储蓄计划的情况下可以花的钱。最棒的是，Digit 可以免费使用，你的账户可以随时随地通过短信访问( [iOS 也有专门的 app](https://itunes.apple.com/us/app/digit-save-money-without-thinking/id1011935076?mt=8) )。

### **Acorns 让我投资，而不强迫我学习投资**

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-vdOXuSI9MTg&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-vdOXuSI9MTg&start=0) 

我最喜欢学习投资的方式是看电影，比如《大空头》、《T2》和《华尔街之狼》。我以为我可以进入投资行业，因为我喜欢赌博的刺激，但事实证明，良好的长期投资根本不像赌博，我宁愿在骰子桌上喝免费饮料。幸运的是， [橡子](https://www.acorns.com/) 给了我一个投资的机会，而不用让我离开自己的老巢，去和经纪人交谈，或者真正了解这一切是如何运作的。最重要的是，它自动化了整个过程。

与美国银行的 [保留零钱程序](https://www.bankofamerica.com/deposits/manage/keep-the-change.go) 、 [类似，Acorns 将您的购买金额四舍五入到下一美元](https://lifehacker.com/acorns-automatically-saves-and-invests-your-spare-chang-1627545527) 并保留剩余的零钱。然而，Acorns 没有把钱存入储蓄账户，而是把钱存入交易所交易投资基金。也有选项增加更多的钱到您的帐户，并设置额外的经常性投资。你想要什么样的投资策略，你有五个选择:保守、适度保守、适度、适度进取、进取。如果你想深入研究每个投资组合，看看什么是什么， [信息是可用的](https://www.acorns.com/investments) ，但你不必。我投资股票市场已经一年多了，但我不知道我投资的是什么。我不在乎。我只关心我没有把钱花在愚蠢的事情上，而是设法用它建立一个小小的储蓄金，随着我的贡献而增长。

你将无法从你的 Acorns 投资组合中退休——有更好的方式来做这件事——但除了 401k 或 Roth IRA 之外，这是一个很好的“设置它并忘记它”的选择。对于像我这样想舒舒服服躺在床上开始投资的人来说，这尤其好。橡子每个月要花一美元，所以在它长到足以支付自己之前，它不会让你赚钱，但那仍然是放在一边的钱，投资的钱，没有被花掉的钱。另外，您的资金和指数基金可以通过 Acorns 的 [iOS](https://itunes.apple.com/us/app/acorns-invest-spare-change/id883324671?mt=8&ign-mpt=uo%3D4) 和 [安卓应用](https://play.google.com/store/apps/details?id=com.acorns.android&hl=en) 访问。

### **给自己小费可以让你更容易首先支付自己**

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-UZZa-ndlkCU&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-UZZa-ndlkCU&start=0) 

“先支付自己”是存钱的一个基本方面。在你付账单、买杂货、给汽车加油之前，你应该为你的未来留出一部分血汗钱。你也应该留出一点零花钱让自己保持理智。 [给自己小费](https://itunes.apple.com/us/app/tip-yourself-save-money-build/id987961939?mt=8) 就像一个虚拟的储蓄罐，这样你就可以以任何你喜欢的理由支付给自己任何数量的钱。你所要做的就是把它和你的网上银行账户联系起来。

每当我想到一个好主意并把它写下来时，我就会打开 Tip Yourself 应用程序，在我的“零花钱”储蓄中增加 2 美元。我做同样的事情来完成工作之外的写作任务，完成个人项目，锻炼里程碑，以及当我知道我做了一个负责任的决定的时候。举例来说，如果我决定不在一些无聊的事情上花钱，我会给自己一些小费，在未来花在有趣的事情上，而不会倾家荡产。给自己小费的行为让我有意识地想到在日常生活中省钱——这本身就是一项壮举——但它也让我变得更有效率。

Tip Yourself 还可以让你在应用程序中设定储蓄目标，并为你提供储蓄金额和储蓄频率的明细，以便你知道是否需要加快储蓄。该应用程序可以免费下载和使用，但目前只有 iOS 上的[(Android 版本正在开发中)。](https://itunes.apple.com/us/app/tip-yourself-save-money-build/id987961939?mt=8)

如果你不知道如何开始存钱，或者你真的像我一样懒，这些应用程序是一个很好的开始。说到储蓄，最重要的是你开始。手机上的几个应用程序不会让你为生活做好准备，但它们会让你明白省钱并不复杂，并让你步入正轨。
