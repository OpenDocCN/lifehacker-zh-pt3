# 这种 Ti' Spritz 是夏季的美味乐趣

> 原文：<https://lifehacker.com/this-ti-spritz-is-delicious-summer-fun-1828419769>

周末快乐，欢迎回到 [*三料欢乐时光*](http://lifehacker.com/tag/3-ingredient-happy-hour) ，每周饮品专栏，提供超级简单又美味的酒水。今天，我们要把一种已经很容易喝的饮料——ti ' punch——变成一种更容易喝的烈酒。

Watch

[我们在](https://skillet.lifehacker.com/plantation-silver-3-star-rum-will-break-your-bacardi-ha-1786688841) 之前已经谈论过 ti ' punch 这是一种美味的加勒比海用朗姆酒、酸橙和糖浆调制而成的混合物。有人会认为这再简单不过了，但最近我参加了一个聚会，我的一个朋友抱怨制作简单的糖浆很“麻烦”。(我告诉他你可以 [把它做成微波炉](https://lifehacker.com/how-to-make-just-enough-cocktail-syrup-1796553286)；他礼貌地假装他下次会那样做。)

总之。这让我开始思考如何进一步简化饮料。显然，答案是添加某种含糖饮料，其中含有糖浆，将其转化为 spritz，这是所有人在夏天都想喝的饮料。要自己做一个，你需要:

*   2 盎司你最喜欢的朗姆酒(我喜欢混合使用。)
*   四分之一酸橙
*   你最喜欢的酸橙汁汽水(斯图尔特的酸橙汁非常好)

将朗姆酒加入一个大酒杯或高脚杯中，从酸橙汁中挤出果汁，放入其中。加入一堆冰，上面放上酸橙苏打，稍微搅拌一下。如果你需要装饰的话，扔一个酸橙轮进去。
