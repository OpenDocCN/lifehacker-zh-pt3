# 更多的医疗保健并不总是更好

> 原文：<https://lifehacker.com/more-medical-care-isn-t-always-better-1777114505>

逃避必要的医疗护理通常是一个坏主意，只有那些懒惰、害怕或试图省钱的人才会这么做。但是太多的医疗保健有其自身的风险。避免某些测试或治疗并选择第二意见而不是更多的访问可能是明智的，一个好的提供商可以帮助你深思熟虑地做出决定。



我们并不是说你应该拒绝必要的照顾。当我们指出乘坐 [救护车可能非常昂贵](http://vitals.lifehacker.com/psa-ambulance-rides-aren-t-always-free-1742804149) 时，一些读者得出结论，在紧急情况下最好不要呼救。那当然不是真的:活着和 [处理医院账单](https://lifehacker.com/how-to-navigate-the-confusing-and-expensive-world-of-me-1765507579) 总比无债一身轻好。但是在很多非紧急情况下，你和你的医生有机会重新考虑你到底需要多少护理。

一个很好的起点是从美国内科医学委员会的 [明智选择](http://www.choosingwisely.org/) 网站开始。该委员会的基金会要求医学协会列出每个领域中患者和提供者应该重新考虑的前五件事。他们已经把这些建议写进了给病人 的 [说明书，但是如果你不介意费力地阅读一些医学术语，你也可以窥探给临床医生](http://www.choosingwisely.org/patient-resources/) 的 [说明书。让我们来看一些带回家的信息。](http://www.choosingwisely.org/clinician-lists/)

### 测试有风险也有好处

你可能会认为医学测试是问一个简单的问题。问题听起来没什么危害，所以我们(还有医生！)经常喜欢做个测试，找出问题所在的想法。但是没有一个测试是完美的，一个错误的测试是有后果的。

以乳房 x 光片为例。在美国，有四套不同的指导方针，关于你什么时候应该开始做，你是否应该停止，以及多久做一次。每一个都代表了对 [如何平衡这种可能被夸大的测试的风险和收益](https://lifehacker.com/breast-screening-decisions-helps-you-make-a-plan-for-1744438045) 的略微不同的判断。

请看这张代表 1000 名低至平均癌症风险女性乳房 x 光检查结果的图表。该测试每检测到两例癌症，就会漏掉一例，并标记出另外 98 名女性进行额外测试。这意味着额外的几天或几周担心你是否患有癌症，其中一些女性可能会因为实际上并未患有的癌症而接受手术、放疗或化疗。一些癌症风险低的女性更有可能受到乳房 x 光检查 的伤害，而不是帮助。这并不意味着测试是无用的，但它是一个判断电话:你和你的医生。

根据《明智选择》的建议，这里还有一些测试值得质疑。你的医生应该接受最新的指导方针，但有时旧习惯很难改变。

*   **每年进行子宫颈抹片检查以筛查宫颈癌**。这些曾经是每年推荐的，但宫颈癌发展如此缓慢，免疫系统经常自己照顾它，以至于每年的测试都是多余的。大多数女性 [应该每 3 到 5 年检测一次](http://www.uspreventiveservicestaskforce.org/Page/Document/UpdateSummaryFinal/cervical-cancer-screening) 。
*   **血液维生素 d 检测**这些只对患有某些疾病或有风险因素的人有意义。维生素 D 测试 [通常对健康人没有帮助](http://vitals.lifehacker.com/you-probably-don-t-need-a-vitamin-d-test-1720027505) 。
*   **用于前列腺癌筛查的 PSA 测试**。血液测试可以寻找前列腺特异性抗原(PSA)，但是 [的假阳性远远多于有用的结果](http://www.cancer.gov/types/prostate/psa-fact-sheet) 。
*   癌症的全身扫描。它们涉及到大量的辐射，并且经常探测到一些本来不成问题的东西。这些偶然的发现需要多年的观察，这意味着更多的扫描。好处 [只是有时候大于风险](http://www.choosingwisely.org/patient-resources/whole-body-scans-to-screen-for-cancer/) 。
*   **流产风险验血**。大多数 [通常对](http://www.choosingwisely.org/patient-resources/blood-tests-for-miscarriage-risk/) 没有帮助，会导致不必要的治疗。
*   如果你在医院，每天验血。这些通常是例行公事，但询问它们是否真的有必要也没关系。 [有时候他们不是](http://www.choosingwisely.org/patient-resources/blood-tests-when-youre-in-the-hospital/) 。
*   **急诊室 CT 扫描轻微头部损伤**。CT 扫描不能告诉你是否有脑震荡，但它可能是一个好主意，对于大脑中可能有出血的重大损伤。CT 是 [不仅没用](http://www.choosingwisely.org/patient-resources/avoid-unnecessary-treatments-in-the-er/) ，它们还很贵，并且让你(或你的孩子)暴露在辐射中。

许多其他测试，包括胆固醇测试、骨质疏松症测试和结肠癌测试，对于特定年龄或具有特定风险因素的人来说是有意义的，但它们并不适合每个人。查看关于体检 的情况说明书 [上有用的测试及其年龄范围的概要。](http://www.choosingwisely.org/patient-resources/health-checkups/)

### 治疗也有风险和成本

治疗也值得仔细检查。以抗生素为例。曾经被认为是低风险的治疗，我们现在认识到副作用包括消灭我们的有益细菌，这可能导致酵母感染，有时甚至严重腹泻。过多使用抗生素还会助长耐药细菌，如 [超级淋病](http://jezebel.com/drug-resistant-super-gonorrhea-laughs-in-the-face-of-an-5984684) 和 [MRSA](http://lifehacker.com/should-i-be-worried-about-getting-sick-when-i-go-to-the-1717461052) 。



毫不奇怪，许多明智选择指南都是关于避免使用抗生素的方法。但是也有其他值得质疑的治疗方法。记住，每种治疗都有成本，所以即使是无害的治疗也可能是浪费时间和金钱。

*   对接近生命尽头的人进行癌症治疗。停止治疗是一个艰难的决定，但治疗可能稍微延长一个人生命的机会可能不会超过许多额外的日子将在化疗地狱中度过的事实。值得拥有 的 [讨论。即使对于预后良好的患者，有时少即是多的方法(例如，用](http://www.choosingwisely.org/patient-resources/care-at-the-end-of-life-for-advanced-cancer-patients/) [某些类型的放射疗法](http://www.choosingwisely.org/patient-resources/radiation-therapy-for-cancer/) )会产生更好的结果。
*   **足底筋膜炎手术。**非手术方法 [往往效果更好](http://www.choosingwisely.org/patient-resources/treating-plantar-fasciitis/) ，手术有造成永久性损伤的风险。
*   **针对失眠(而非睡眠呼吸暂停)的睡眠研究**通常没有一个 就可以用 [来诊断和治疗失眠。睡眠研究*是否有助于诊断睡眠呼吸暂停。*](http://www.choosingwisely.org/patient-resources/tests-and-treatments-your-employees-may-not-need/)
*   过于简单的物理疗法，这种疗法通常是为老年人开的，因为他们害怕会伤害自己。但是如果练习没有挑战性， [就没有帮助](http://www.choosingwisely.org/patient-resources/physical-therapy/) 。
*   **导尿管**增加了尿路感染的风险，所以应该在必要的时候才使用。
*   用不含汞的填充物替换含汞的填充物听起来是个好主意，但是这种替换实际上会释放出比留在原处更多的汞。很多时候，最好是 [放过他们](http://www.choosingwisely.org/patient-resources/dental-fillings-that-contain-mercury/) 。

有时候，这些治疗或程序中的一种对你的特殊情况来说是个好主意，但值得问问你的医生为什么她要开这个处方，以及是否有其他选择可能更好。

### 问什么问题

对于测试来说，重要的是要问清楚基于结果的决定是什么。仅仅因为测试是例行的，或者因为你或你的医生对结果感到好奇，并不意味着测试是有用的。例如，有一些产前检查可以告诉你 [你的胎儿是否似乎有遗传异常](http://www.livescience.com/45949-prenatal-genetic-testing.html) 。因为你可能会得到假阳性或假阴性的结果，所以有必要问问自己:你会根据这些结果做一些不同的事情(比如终止妊娠)吗？如果没有，可以说你宁愿根本不做测试。

医疗专业人士都很熟悉知情同意的概念——在你同意之前，作为患者，你有权知道某项测试或治疗的风险和益处。没有医生会觉得你很奇怪，如果你问:

*   这项测试或治疗有什么风险？
*   有什么好处？
*   这种疗法有多大可能有效(或者这种测试有多大可能给出正确的结果)？
*   我们还能尝试什么呢？

如果你问了这些问题，你就不是一个专横的病人，你也不应该因为问了这些问题而感到难过。别忘了询问*，即使是你建议进行测试或治疗*。例如，如果你看到一个听起来很棒的药物广告，并询问你的医生，不要只是说“嘿，医生，我能要这个吗？”但也要询问它的风险和替代方案。同样的想法 [如果有一个测试](https://lifehacker.com/how-to-ask-your-doctor-if-a-test-is-necessary-5860040) 你听说过每个人都应该得到。

三位专家在健康事务博客上写道:“有充分的证据表明，很大一部分选择性护理给予了那些如果更好地理解他们的选择和其中的权衡，就会选择避免的患者。”。了解这些选择和权衡有助于你成为一个更聪明，也许更健康的病人。
