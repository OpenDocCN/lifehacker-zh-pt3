# 不要试图在一次参观中参观完一个博物馆

> 原文：<https://lifehacker.com/stop-trying-to-see-all-of-a-museum-in-one-visit-1821817889>

说到博物馆，我们的雄心和好奇心往往超出了我们的身体和大脑所能承受的范围。如果你真的想享受参观博物馆的乐趣，计划只看几件展品。

Watch

世界上大多数最伟大的博物馆绝对是巨大的。你走进去，决心在一天之内看完所有的东西，但是在一个又一个的玻璃橱窗里浏览了大约一个小时之后，你最终还是崩溃了。你的眼睛变得模糊了，你的脚开始疼了，你的大脑对每一个展示的注意力都减少了，其他的顾客开始让你心烦(“你能不能快点拍下你那该死的照片，我好去看看描述？!").这被非正式地称为“博物馆疲劳”，它几乎会毁了你曾经如此兴奋的一天。

对你来说幸运的是，博物馆爱好者们，有一个简单的解决办法:不要试图去参观所有的博物馆。最多选择两个你真正关心的博物馆区域，并在那里呆上一段时间。对拥挤的人群中一睹小得惊人的蒙娜丽莎不感兴趣吗？跳过它。不喜欢中世纪的挂毯吗？如果你真的喜欢印象派，那就去看看吧。之后，也许去礼品店逛逛。我的意思是花一个小时或不到一个小时，让自己错过博物馆的大部分内容。上个月我去大都会艺术博物馆时，只去了古埃及区和礼品店。太棒了。我看到了我想看的东西，花了更多的时间在我真正关心的展品上，还有时间做其他有趣的事情。我真的不觉得我错过了什么。

这对一些人来说可能是浪费，甚至对铁杆博物馆探索者来说可能是亵渎，但重要的是你要管理你的期望，接受你的人类注意力范围的限制，并尊重你的倦怠。我知道你想全部看完——我也是——但是如果你真的想看的话，也许你可以改天再来看其他的东西。很有可能你会尝试这种两部分的方法，离开博物馆的时候，你会和看完所有的展览一样高兴，如果不是更多的话。
