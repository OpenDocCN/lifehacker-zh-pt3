# 用什么代替亚马逊助手，侵入式的意愿列表替代

> 原文：<https://lifehacker.com/what-to-use-instead-of-amazons-invasive-new-assistant-1823522711>

如果你使用亚马逊的 [愿望清单](https://lifehacker.com/best-wishlist-tool-amazon-com-universal-wishlist-5416477#_ga=2.42054181.1215127283.1520195704-258269408.1513265580) 书签来记录你想要购买的所有产品或者 [收到的礼物](https://lifehacker.com/best-wishlist-tool-amazon-com-universal-wishlist-5416477) ，你可能会注意到最近的一个变化。该公司正在逐步淘汰其旧的一键式工具，转而使用更先进(也更具侵入性)的亚马逊助手。



这个问题归结为亚马逊助手和 bookmarklet 工作方式的技术差异。旧的选择基本上只是一点 JavaScript 代码，抓取你正在看的任何产品，并将其发送到你的愿望清单。相比之下，新助手是一个更复杂的浏览器插件，可以跟踪你在网上做的一切。

没错。一旦你安装了亚马逊助手，公司就能看到你在浏览器中做的一切。新的 [服务条款页面](https://www.amazon.com/gp/BIT/AmazonAssistant/TOU?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/what-to-use-instead-of-amazons-invasive-new-assistant-1823522711&asc_source=&tag=kinjalifehackerlink-20) 甚至确认了这一点，并将其标注为“自动信息”亚马逊指出，收集的数据不会与你的账户相关联，但它仍然与你的 IP 地址相关联。那不完全是 [令人欣慰的区别](https://www.techdirt.com/articles/20080121/19520029.shtml) 。

不幸的是，没有第三方按钮可以将产品信息发送回你的亚马逊愿望清单(该公司显然不会允许)，但有很多与之竞争的愿望清单服务。

最好的选择之一是 [Shoptagr](https://www.shoptagr.com/) ，它可以让你将大量不同在线商店(包括亚马逊)的产品添加到一个中央愿望清单中。您可以设置商品缺货或打折时的通知，并与他人共享您的列表以接收礼物。Shoptagr 还提供了 [按钮](https://www.shoptagr.com/get_the_button) ，就像亚马逊的旧 Bookmarklet 一样，用于快速添加新产品。

最重要的是，它 [只会在你使用按钮的时候跟踪你](https://www.shoptagr.com/legal/privacy) 。所以你不用担心 Shoptagr 在网上看你做的一切。
