# 保持草药新鲜的最好方法是什么？

> 原文：<https://thetakeout.com/what-s-the-best-way-to-keep-herbs-fresh-1827295085>

如果你足够幸运，有足够的空间和园艺技能让草本植物从春天一直长到秋天，那么恭喜你。我们其余的人忍受着在杂货店购买那些塑料外壳、4 美元一片的草药碎片的屈辱，然后祈祷它们在我们的厨房里能保持好 12 个小时以上。农贸市场的草药也是如此。那么，让这些珍贵的蔬菜保持更长时间新鲜的最好方法是什么呢？

当然，立即使用草药总是更好的，林恩·艾黎说，她是《用草药烹饪 T2 T3 T4 T5》的作者，她说它们的美味在采摘后很快就开始变质。她建议，如果你必须保存草药，把它们放在一杯水中，可以选择用塑料袋松松地盖在上面。

我从主厨杰米·辛普森那里得到了类似的建议，他是厨师花园和蔬菜烹饪学院 的行政联络员。他遵循他所谓的“Vestinos 方法”，该方法是以著名调酒师兼烈酒顾问 [彼得·Vestinos](https://www.thebarmedic.com/about)命名的。Vestinos 方法包括在香草茎的底部切一个新的切口，如罗勒、薄荷、芫荽叶、欧芹等。然后把它们放在一杯水里，水刚好盖住茎的底部。辛普森建议，把它放在柜台上；每天换水；并且每隔几天在底部进行一次新的切割。像迷迭香这样的耐寒香草可以放在湿布上，然后卷成一捆，用保鲜膜或拉链塑料袋包好，放在冰箱里。

这是对“罐中储水”方法的两个认可。为了寻求最终的意见，我求助于我手边的爱德华·c·史密斯的《蔬菜园丁圣经》，这本书——在我涉足后院园艺时——我已经开始认为它确实是一本好书。他赞同用 Vestinos 方法储存罗勒，并主要建议你干燥和储存所有其他草药以备长期使用。

Lynn Alley 告诉我，对于那些不能马上使用所有草药的人来说，干燥和储存方法似乎是唯一的选择，但处理这些草药并冷冻它们可能更有意义。她喜欢将切碎的香草和橄榄油混合，然后放在冰块托盘里冷冻。这些小方块可以放入意大利面酱、鸡肉、汤中等等。(冰块魔术也适用于经典的罗勒香蒜酱。)

“在冬天，当你没有新鲜的香草时，我最喜欢做一些意大利面，我唯一需要的就是冰箱里的香草冰块，”她说。

艾利说，制作和冷冻香草黄油也很有效。只需制作复合香草黄油——你可以按照 [这个*纽约时报*食谱](https://cooking.nytimes.com/recipes/1018532-herbed-compound-butter) ，或者只是将黄油和香草捣碎，用羊皮纸包起来(不需要食谱)——然后冷冻。艾莉喜欢把她的冻成圆柱形或圆木形:“然后你就可以像切香肠一样把它们切下来。”
