# 你应该做那个病毒肉棒吗？

> 原文：<https://lifehacker.com/should-you-make-that-viral-meat-stick-recipe-1829152738>

有好的黑客， [也有坏的黑客](https://lifehacker.com/these-coca-cola-hacks-are-as-pointless-as-they-are-ente-1827480241) ，在这个在线出版物中，我们喜欢给你好的“uns”。一个好的黑客应该让一个人的生活更容易，无论是通过以新的方式使用一个普通的对象，简化一个过程，还是解决一个你不知道自己遇到的问题。我不确定这种 DIY 挤压机*肉类*的那些标准。

Watch [https://lifehacker.com/embed/inset/iframe?id=twitter-1041122297090920449&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1041122297090920449&autosize=1) 

你可能已经看过这个非常流行的视频了。在这个视频中，一名俄罗斯男子用一个百事可乐瓶、一个可乐罐和一根弯管状的 PVC 管制作出了漂亮的肉串，我认为这就是 kofta 烤肉串。涉及到喷灯和钻头。(我们已经有了一个非常精简的、一点也不复杂的开始。)我喜欢一个好的肉管，在许多人让我注意到这个视频后，我决定制作这个肉模将是对我时间的一个很好的利用。(我的编辑同意了。)制作和使用这个新玩意儿最终成为了一个相当漫长的旅程，我邀请你一起来。

首先，我必须收集我的用品。我在街尾的奥什 商店买了 PVC，在附近的杂货店买了一瓶无糖胡椒博士，因为百事可乐是一种垃圾饮料。我有一个喷灯，但没有电钻，所以我问我的舞台工作朋友是否可以借她的，因为她很方便。她说她会把它放在厨房的桌子上，让我第二天早上自己进去拿。

在我拿起电钻后，我的朋友发短信给我说她忘记把电钻放在外面了，这让我对电钻是什么产生了怀疑。但是在我尝试制作扦孔之前，我必须把 PVC 管和胡椒瓶融合在一起。(实际上，我可能会先完成这两个步骤。)我把厨房的手电筒调到最低档，对准管子。管道立即着火了。(我没有这张照片，因为我是一个人，我的首要任务是灭火。)火被扑灭后，我又试了一次，这次把火焰举得更远。管道没有再次着火，但它确实比视频中的管道暗得多。它看起来像一个烤棉花糖，如果棉花糖的形状像肘部的话。我把胡椒博士的瓶子塞进了管子里。令我惊讶的是，它形成了一个强大的密封。

然后是时候做串烧孔了。我很确定我拿错了钻头，因为我不会钻孔。钻头刚从管子上滑下来，轻微损坏了管子。我试着用螺丝打了一个小“预孔”，但也没有成功。最后，我决定回到五金店，任由在那里工作的勤杂工摆布。我把我的烟斗瓶和我很确定是一个钻子放在一个手提袋里，然后走回奥什。当我走进商店时，这个可能是演习的东西触发了盗窃警报。

在我向服务台的女士保证我没有从外面抢劫他们之后，一个非常高兴的男人向我走来。他问我是否需要帮助，我告诉他我正在做一个“有点傻的项目”幸运的是，他喜欢愚蠢的项目。当我从包里拿出烟斗瓶时，他的脸上露出了喜色。“我完全知道你在做什么！”他说，肯定喜气洋洋。“我想我需要一个不同的钻头，”我说着，从包里拿出一个“天哪，我希望这是一个钻头”。

这位好心人没有告诉我，我拿的是*不是*一个钻子，这让我觉得自信多了。他给我看了我需要的钻头。我向他询问一些“一般钻孔技巧”，他告诉我，我可能应该使用老虎钳。“我没有老虎钳，”我喃喃自语，新发现的自信像 [水坑里的棉花糖](https://www.youtube.com/watch?v=zWAIWySclX0) 一样溶解。这个非常好的男人决定让另一个好男人在店里帮我钻孔会更容易。钻孔者没有看过视频，对我的汽水瓶和一根 PVC 管融合在一起感到困惑，但没有兴趣问太多问题。他用一个锋利的工具做了一个“预孔”，钻了一个真正的孔，然后把我的法兰克福香肠还给我。现在我准备好了一些肉的行动。

我切开瓶子，这样我就有地方放肉了。然而，我没有把它剪在最合适的地方。在我最初观看视频时，我错过了一个细节，那就是俄罗斯男子将瓶子切得比最宽点略低，这样可乐罐就可以紧贴开口，但在我喜欢称为“肉室”的地方有更多的呼吸空间。

事实证明这不是什么大事。我用串肉扦穿过串肉扦孔，用碎牛肉填满肉室，然后把肉压在一罐健怡可乐上面。肉保持不动是不可能的。

我开始怀疑肉的质地是否是问题所在，所以我查阅了一些 kofta 食谱，看看碎牛肉(或羊肉)通常是否以任何方式加工过。事实上，一些食谱确实指导人们把肉放进食品加工机。我处理了肉，重新装满枪膛，又试了一次。

我把健怡可乐罐换成了一罐意大利辣香肠，因为它更适合我，我又压了一次。这招奏效了，肉从腔里滑了出来，形成了一个完整的肉管。

挤出肉所需的努力比视频中描述的要稍微多一点，而且以挤出肉的速度移动棍子有点困难，但我成功了。肉管出来很像管子，但它们确实在棍子上滑动了一段时间。我没有煮肉管，部分是因为我没有烤架，但主要是因为我的厨房仍然有烧焦的 PVC 管的味道。我把它们放在袋子里，放在冰箱里。

我不确定我是否会称之为“好黑客”虽然这个装置确实“工作”，但它感觉不“容易”、“简单”，或者“无论如何都不值得我花时间。”作为对比，我用我的手在串肉扦周围塑造了一些肉。

虽然它可能不像 PVC 管挤出的肉管那样成型一致，但我的手工成型管肯定需要更少的五金店，而且肉不会在串肉扦上滑动。DIY 挤出机也很难清洗，或者至少看起来很难清洗。我还没有尝试清理它。我可能会把它扔掉。
