# 假装你很擅长

> 原文：<https://lifehacker.com/pretend-youre-good-at-it-1822841289>

不久前，我遇到了我的朋友珍妮·劳森，她正在镇上录制她的回忆录的有声读物版本， [*让我们假装这件事从未发生过*](https://www.amazon.com/dp/B0065S8R38/ref=dp-kindle-redirect?_encoding=UTF8&asc_campaign=InlineText&asc_refurl=https://lifehacker.com/pretend-youre-good-at-it-1822841289&asc_source=&btkr=1&tag=kinjalifehackerlink-20) 。我问她过得怎么样，她告诉我过得很糟糕——直到她惊慌失措地向她的一位作家朋友寻求建议。



这里，我让她告诉 [其余的](https://www.bookreporter.com/blog/2012/06/04/pretend-youre-good-at-it-thoughts-on-recording-lets-pretend-this-never-happened-a-mo) 。

> 我借口要去喝一杯，但我偷偷给我的朋友尼尔·盖曼发了封电子邮件，问他是如何在听起来如此冷静沉着的同时录制了这么多本书的。他的回应几乎立刻就回来了:
> 
> 假装你很擅长。
> 
> 我把他的建议潦草地写在我的胳膊上，然后溜回录音棚。我深吸一口气，然后…我假装我很擅长。我也是。我们扔掉了第一章，重新录制了一遍，我的新咒语在我脑海中回荡。

老实说，我并不觉得这是什么改变世界的建议。我当然很高兴它对我的朋友有效，但除此之外，我没有想太多。但是慢慢地，渐渐地，“假装你很擅长”对我产生了魔力。

我长期以来饱受冒名顶替综合症的折磨；在大多数努力中，我下意识地认为我会以某种方式把事情搞砸。(是的，我一直在接受治疗。*这么多疗法。*)这是一种不舒服且没有生产力的生活方式。除了解决任何根深蒂固的问题，我需要一些东西来让我迈出第一步:克服恐惧，走向世界。所以我转向“假装你很擅长”——这很有效。我假装，我的焦虑减轻了。我表现得像是擅长某件事的人，我毫不费力地完成了任务。事实上，我根本没有把事情搞砸。

从表面上看，“假装你擅长它”很像“假装它，直到你成功”，但实际上，它要好得多。原因如下:“假装直到你成功”假设你会在某个时候“成功”我可能在某个时候“成功”的想法压力太大了。我什么时候能赶上？为什么我还没成功？为什么 20 年了我还在假装？我天生就不能成功吗？此外，“假装”有一种推托的意味，暗示着某人可能在某个时候意识到你是个骗子。(这个我真的想通了。)

另一方面，“假装你很擅长”更有趣。你并没有说你会擅长某件事——你只是在摆弄你曾经擅长的事情。擅长的人会怎么做？他们会如何表现？

几个警告:“假装你很擅长”不能代替实际训练，尤其是在可能危及任何人安全的情况下。你不会希望你的飞行员假装她很擅长，而不是，比如说，完成她的执照。“假装你很擅长”也不是不必要的傲慢的借口——但是，任何对此产生共鸣的人可能都不会是那种倾向于傲慢的类型。

也就是说，我鼓励你在生活中挣扎的任何领域使用“假装你擅长它”。早上起床有困难吗？告诉自己你是最擅长早起的人。一个人去派对紧张吗？想象独自去参加聚会是你的专长。那会是什么样子？假装你很擅长，你会不可避免地惊讶于自己的实际水平。
