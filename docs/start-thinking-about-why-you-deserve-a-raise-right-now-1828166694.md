# 现在就开始思考为什么你应该得到加薪

> 原文：<https://lifehacker.com/start-thinking-about-why-you-deserve-a-raise-right-now-1828166694>

昨天是黑人女性同工同酬日——或者说是今年普通黑人女性与白人男性实现薪酬平等的象征性日子 Refinery29 发表了一篇很棒的文章，讲述了三名女性如何发现自己的薪酬过低 ，以及她们对此采取的措施。它充满了关于了解自己的价值和要求更多的坚实建议(一个重要的要点——你是老板*不能*禁止你和同事讨论你的工资，不管他们说什么)，但我尤其喜欢 Redfin 首席经济学家内拉·理查森的这个观点:

> “别说了，我还要 1 万美元。比方说，*我想要多 10，000 美元——因为这六个原因*我值得拥有它。你越能量化自己的价值，做决定的人就越容易接受这种推理。”



这意味着评估什么使你最适合一个角色，或者你给公司带来了什么别人无法带来的价值。不要等到你有了工作机会或者年度评估的前一天才开始考虑这些事情，现在就开始吧。即使你没有积极找工作。

考虑一下:你做了什么能增加价值的事情？你是如何不可替代的？把它们写下来，然后在它们发生的时候，实时添加你工作生活中的例子。那么你就有东西要带，或者你甚至会被鼓励在你打算加薪之前就要求加薪。现在就开始工作。

理查森的另一个好建议是:在接受一份工作之前，知道如何在公司取得进步。当然，起薪可能看起来很高，但是如果没有上升的空间，你如何在未来几年赚得更多？她说:

> “在面试的最后五分钟，问自己:这里的人是如何晋升的？职业阶梯是什么样子的？在这家公司的三年、四年里，我能期望自己在哪里？与同一领域的其他公司相比如何？好的雇主希望员工希望在自己的公司里发展自己的事业。我认为这些都是非常合理的问题。”

如果没有发展空间——在薪水、头衔、责任等方面。你确定你想要那份工作吗？你可能会意识到你的价值不止于此。
