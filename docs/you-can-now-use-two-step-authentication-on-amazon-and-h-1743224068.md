# 你现在可以在亚马逊上使用两步认证——下面是方法

> 原文：<https://gizmodo.com/you-can-now-use-two-step-authentication-on-amazon-and-h-1743224068>

两步认证是最简单的方法之一让你的在线账户更加安全。最后，亚马逊决定提供使用该系统的选项——下面是你如何启用它。



虽然大多数互联网巨头——苹果、谷歌、脸书等——长期以来一直提供使用两步认证的选项，但亚马逊一直坚持不懈。但是 Engadget 的理查德·劳勒已经注意到 ，它现在是在线零售商的一个选项。

当然，现在参加聚会已经很晚了，但这是一个你想参加的聚会。以下是如何在你的亚马逊账户上启用两步认证。

首先，登录亚马逊，点击右上角的“你的账户”。

接下来，向下滚动到“设置”部分，选择“更改帐户设置”

然后，选择底部的“高级安全设置”选项

在这里，您可以选择激活两步认证。点击“开始”

现在，您将被引导通过一个三步过程来设置它。您可以选择使用短信服务或专用应用程序来启用身份验证

你也可以设置一个备份方法，以防你的首选由于某种原因失败。

你应该马上打开它。然后，如果你还没有，去 [在你所有的其他账户上也做](http://gizmodo.com/its-time-to-enable-two-step-authentication-on-everythin-1646242605) 。
