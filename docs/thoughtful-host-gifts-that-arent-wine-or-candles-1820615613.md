# 体贴的主人礼物不是酒或蜡烛

> 原文：<https://lifehacker.com/thoughtful-host-gifts-that-arent-wine-or-candles-1820615613>

如果你不主持感恩节，你很可能会参加感恩节，这意味着一些非常好的人打开了他们的家，邀请了许多人。这个人应该得到礼物。



葡萄酒是这个送礼问题的(通常是最后一分钟)答案，但是除非你非常了解葡萄酒——以及你的主人喜欢的那种葡萄酒——否则这看起来有点不近人情。蜡烛是这里的另一个标准，但我发现气味是太私人的东西，不能强加给别人。(即使是昂贵的芳香蜡烛也会引发头痛。)也就是说，你还是应该带一份礼物，而且礼物不必很贵。这里有一些可爱的物品，我——一个主持事物的人——很想收到:

*   **花式盐:**华丽、片状 [马尔顿盐](https://www.amazon.com/Maldon-Sea-Salt-Flakes-ounce/dp/B00017028M?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/thoughtful-host-gifts-that-arent-wine-or-candles-1820615613&asc_source=&rawdata=[r|https://www.google.com/[t|link[p|1796338148[a|B00017028M[au|5876237249235788664[b|lifehacker&tag=kinjalifehackerlink-20) 让每一桌都觉得 [多了一点精致和成熟](https://skillet.lifehacker.com/how-to-throw-a-dinner-party-like-a-grownup-1796338148#_ga=2.243950092.884928433.1511187930-392757988.1502389723) ，很可能主持这一大餐已经耗尽了你的主人最基本的餐具。一盒上好的盐，也许再配上一个木制的，是实用和额外的相等部分。
*   **节日厨房用毛巾(或餐巾纸):**活动专用的厨房用毛巾和餐巾纸确实能让一个厨房焕然一新，但给自己购买又觉得有点太轻浮。然而，一个人永远不会有足够的厨房毛巾，一些额外的漂亮毛巾会让你的主人兴奋地攻击堆积如山的菜肴。(开个玩笑，你应该主动帮忙。)
*   **真正好的洗手液:**经常做饭的人，手洗得多，洗多了手就干了。优色林(unscented) [高级修复护手霜](https://www.eucerinus.com/products/repair/eucerin-intensive-repair-extra-enriched-hand-creme) 、 [威莱达肤食](https://www.amazon.com/Weleda-Skin-Food-2-5-Ounce/dp/B000ORV3NC?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/thoughtful-host-gifts-that-arent-wine-or-candles-1820615613&asc_source=&tag=kinjalifehackerlink-20) 、 [科颜氏极致力量护手霜](https://www.kiehls.com/ultimate-strength-hand-salve/3605970241117.html) 都为干性皮肤做了奇妙的事情。如果事情真的很糟糕，一听袋装香油可以作为一份略带幽默但能救命的礼物。
*   真正好的黄油或橄榄油:你的主人可能这两样都没有了，会欢迎重新进货。买一大块令人印象深刻的阿米什式黄油，或者一瓶上好的橄榄油，这是用来烹饪的，而不是品尝的(或者你可以两样都买)。
*   **咖啡:**你的主人要累了，累的人喜欢咖啡。从当地的烘焙店给他们买一袋好豆子， [用这个杯子](https://emilymcdowell.com/products/fucking-great-job-mug) 给他们，这个杯子会告诉你你对他们的看法以及他们的招待能力。
*   [**第二天的早餐**](https://lifehacker.com/do-your-thanksgiving-host-a-favor-and-bring-them-breakf-1789066590) **:** 你的主人做腻了，你的主人早餐(大概)不想吃火鸡。有很多方式可以作为早餐礼物——甜甜圈、自制烤饼、百吉饼——但我最喜欢的早餐礼物是 Cracker Barrel 的煎饼混合糖浆和培根。(额外收获:如果你要开车去感恩节的目的地，你可以在路上买。)

如果你真的想送酒，没有人会对酒着迷，但你可能会考虑拿一瓶更烈的酒。一瓶酒可以喝一个晚上，但是一瓶好的威士忌至少可以喝两个晚上。
