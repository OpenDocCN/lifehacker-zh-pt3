# 使用这个开源合成器应用程序学习节拍制作的基础知识

> 原文：<https://lifehacker.com/learn-the-basics-of-beatmaking-with-this-open-source-sy-1828030421>

任何科幻游戏或电影最精彩的部分之一是那些不祥的音调，暗示着一场 [对话](https://www.youtube.com/watch?v=5zWuFjlDBaE) 或 [发现](https://youtu.be/1KdEamxyaEU?t=1m54s) 即将向南——这是电影配乐的精髓部分，为体验增添了许多气氛( [和张力](https://www.youtube.com/watch?v=4ctK1aoWuqY) )。现在，由于一个开源的 iOS 应用程序，你可以制作自己的合成器声音，并自学音乐制作的基础知识。

Watch

AudioKit 的 [Synth One](https://audiokitpro.com/synth/) 应用程序完全免费下载和使用，考虑到这款虚拟合成器的功能如此全面，这已经足够引人注目了。启动它，你将有超过 300 个单独的预设可供尝试——使用应用程序的内置键盘加载和播放这些预设足够有趣。

从那里开始，天空就是众所周知的极限。我发现，了解应用程序的许多按钮、转盘和选项的最佳方式是采用试错法。不同的效果如何改变你创造的声音的性质？当你改变合成器混响的形状时会发生什么？加入失谐？拨弄序列的起音、延音和延音？改做琶音是不是更好玩？能否重现《星际穿越》的配乐？(试试预设#12: BB 断电。

虽然 Synth One 的界面起初看起来令人难以置信，但这只是在检查了您可以翻阅的许多页面中的一个选项之后，不要让它阻止您。到处玩。制造一些阴森恐怖的科幻噪音或者有趣的旋律。如果你需要更多的手来创造完美的音频循环，请观看一两个教程:

如果你想变得有趣，你甚至可以用 MIDI 键盘同步应用程序，如果你宁愿敲击物理按键而不是虚拟按键的话。您也可以录制您的合成器，并将其放入其他应用程序，如 GarageBand，这样您就可以将您的原创作品(或您调整的预置)用作更大音乐作品的组成部分。
