# 七种不同女童子军饼干的最佳啤酒搭配

> 原文：<https://lifehacker.com/the-best-beer-pairings-for-seven-different-girl-scout-c-1758847748>

你如何把已经很美味的东西变成令人垂涎三尺的体验？啤酒。如果你喜欢女童子军饼干，你也喜欢酿造，你不妨将两者结合起来，啤酒专家朱莉娅·赫兹提供了她的配对建议。

Watch

赫兹是酿酒商协会的精酿啤酒项目主管，也是 CraftBeer.com 的出版商。 在《纽约每日新闻》工作时，她建议七种 [女童军饼干](http://lifehacker.com/the-differences-among-girl-scout-cookies-based-on-where-1758741504#_ga=1.126214157.1268082208.1431441811) 搭配啤酒。以下是一些亮点:

> **THIN MINTS:** 这些受欢迎的蘸有巧克力的薄荷味饼干想要一杯咖啡或巧克力口味的烈性黑啤酒或波特酒，比如大分水岭酿酒公司的巧克力橡木陈酿皇家黑啤酒。“这就像嘴里的薄荷·派蒂，”赫兹说。
> 
> 焦糖和烤椰子圈淋上巧克力，这是比利时风格的四倍啤酒(就像俄罗斯河酿酒公司的拯救浓黑啤酒)或美国棕色啤酒，就像布鲁克林啤酒厂的棕色啤酒。
> 
> 这些饼干结合了花生酱和巧克力。“他们超级有钱，所以比利时风格的 dubbels 或四人间是完美的，”赫兹说。或者尝试用咖啡或巧克力酿造的棕色麦芽酒，比如特柳赖德酿造公司的面朝下的棕色麦芽酒，这是一个完美的搭配。
