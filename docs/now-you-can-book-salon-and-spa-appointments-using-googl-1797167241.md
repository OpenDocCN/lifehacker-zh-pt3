# 现在你可以使用谷歌地图预约沙龙和水疗

> 原文：<https://lifehacker.com/now-you-can-book-salon-and-spa-appointments-using-googl-1797167241>

我经常出差，这意味着当我在发廊的时候，我倾向于跳过安排未来理发之类的事情，因为我不确定约会的时候我是否真的会在城里。这很有道理，但这也让我在为期两周的旅行的第一天照镜子时，意识到我迫切需要修剪一下。



当这种情况发生时，我通常会喊附近的沙龙，找出离我最近的，然后打电话看看是否有预约。这有很多步骤，当一个沙龙没有任何空缺时(这总是至少是我所说的前三个)，这是一个我必须一遍又一遍重复的过程。本周，谷歌 [增加了直接从谷歌地图预订沙龙和水疗预约的功能，让事情变得简单多了](https://www.blog.google/products/maps/reserve-google-summer-bookin-happens-so-fast/) 。

此前，谷歌推出了健身课程的功能。要预订，你只需在谷歌地图上搜索你要找的企业类型(或者具体的地方，如果你已经知道你想去哪里的话)。在你通常会看到时间和评论的页面上，你会看到一个关于参与企业的大“预订”按钮，你可以点击它进行预约。

预订是通过与许多第三方预订服务(其中一些你可能已经在使用)的合作来推动的，包括 Genbook、SalonRunner、Rosy、Yocale 和 WellnessLiving ，谷歌表示更多服务正在进行中。

预订包括选择你想去的时间和日期，并提供你的姓名和电话号码等基本信息。我看过的大多数地方在预订页面上都有价格，所以你知道你要去哪里。

显然，并不是镇上的每一家沙龙和水疗中心都会注册这些服务，但很多都是如此。如果你只想看看可以通过谷歌预订的商家，你可以通过谷歌 网站使用谷歌 [预订进行搜索。](https://www.google.com/maps/reserve/) 

你可能不会用这个功能去你家乡的发廊预约(虽然你当然可以),更多的是当你在旅行时，意识到你需要理发，或者真的需要按摩。
