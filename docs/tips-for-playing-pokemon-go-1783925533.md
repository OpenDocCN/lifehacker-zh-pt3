在《精灵宝可梦 Go》中，成为最棒的人(就像从来没有人做到过一样)出奇的困难。你不能再靠自己的努力成为一个强大的神奇宝贝:你必须走出去探索。我们有一些建议可以帮助你充分利用你的旅程。

**选择登录方式**

Pokémon Go 让你用谷歌账户或作为 Pokemon 训练者登录。前者更稳定；后者受到登录问题的困扰。问题是:你必须选择一个。你不能一开始用谷歌账号，然后决定换成教练账号。这样做会创建一个新帐户。所以早点决定，坚持下去。

不要太担心你的谷歌账户的安全性，因为那里的风险被夸大了，但是如果你有兴趣保留服务条款剥夺玩家的合法权利，看看这个。T3】

**带上*pokémon Go******带着你到处跑***

*《神奇宝贝 Go》不仅会根据你的地理位置产生怪物，还会考虑你在玩游戏时走过的距离。无论何时你在一个新的或不同的地方，启动*神奇宝贝* *Go* 都是一个好主意。*

***保持*【poke mon go】*打开***

*为了让你的脚步算数， *Pokémon Go* 需要处于活动状态，并且在你手机的前台。抱歉，一心多用。*

***给手机充电***

*相关的， [*神奇宝贝 Go* 会谋杀你的电池寿命。](http://kotaku.com/pokemon-go-players-are-saying-the-game-kills-battery-li-1783187259) 我在后口袋里带着一个便携式充电器四处走动，以帮助保持《神奇宝贝 Go》*的运行，但简单地随身携带一个充电器以备紧急情况可能就足够了。至少，你应该考虑在低功耗模式(由游戏在设置中提供)或省电模式(在你的实际手机上)下玩。)**

*如果你想要更多关于如何节省电池寿命的建议，请查看 Lifehacker 的这篇文章。[也有保存数据的方法，你可以在这里读到。](http://lifehacker.com/easy-ways-to-save-data-while-playing-pokemon-go-1783372269)*

***关掉相机*** 

*看《现实世界中的神奇宝贝》一开始是新奇的，但除了可能提供有趣的截图之外，它并不能真正帮助你。在游戏界面内抓*神奇宝贝*要容易得多，所以一定要用下图的按钮关闭 AR。*

***挑选你想要的球队*** 

*老实说，没什么大不了的。我选择了我最喜欢的颜色(红色)，尽管每个团队都有自己的怪癖和污点。如果有帮助的话，你可以访问红色的[黄色的](https://www.reddit.com/r/PokemonGoYellow) 和蓝色的的 Reddit 页面，了解一下你要注册什么。或者，你可以去附近的健身房看看哪个队最受欢迎，然后从那里开始。*

***重命名你的口袋妖怪***

*尽情享受吧！请注意，您也可以使用 HTML 标签，所以尽情使用吧。*

***不断捕捉弱小*神奇宝贝*T3*** 

*在某个时候，你会厌倦你身边最常见的神奇宝贝。尽管如此，如果你有神奇球，你应该继续捕捉那些无聊的生物:它们是 XP 的素材。而且等级越高，在野外遇到的神奇宝贝*T3 就越好。 [你可以在这里阅读更多关于捕捉弱小神奇宝贝的好处。](http://kotaku.com/a-quick-pokemon-go-tip-keep-catching-weak-monsters-1783512123)**

*试着投出曲线球*

*投掷口袋球的时机一开始可能很难把握，但是一旦你掌握了，你应该试着练习更高级的投掷。如果成功，你将获得更多的经验值。只要知道像风速甚至口袋妖怪周围的环的颜色这样的事情都会影响你的捕捉尝试(绿色最容易，红色最难，但值更多的 XP。)* 

***使用浆果***

*浆果让捕捉神奇宝贝变得更容易。你可以在你的背包里找到它们。*

***计划路线***

*一边玩 *Pokémon Go* 一边闲逛有一定的魅力，但是我发现 *Pokémon Go* 在我对自己要去的地方有感觉的时候是最享受的。确保去热门的诱惑点是结识其他玩*神奇宝贝*的人的好方法，更不用说捕捉更多神奇宝贝的简单方法了。参观你不熟悉的地方和地标也很有趣，只是为了给 *Pokémon Go* 一种冒险感。如果冒险不是你的事情，如果你想用 Pokemon Go 作为锻炼的借口，规划仍然是有用的。或者， [你可以在这里找到兼做小吃店的餐馆。](http://lifehacker.com/yelp-makes-planning-pokemon-go-adventures-easier-with-a-1783754419)T15】*

***追踪特定的怪物***

*好吧，听着。 [没有人真正知道“附近”标签是如何工作的](http://kotaku.com/nobody-actually-knows-how-pokemon-gos-nearby-feature-wo-1783585598) ， [有时这个功能会被直接终结](http://kotaku.com/pokemon-gos-three-step-glitch-makes-tracking-pokemon-im-1783854268) 。尽管如此，试图找到你想要的神奇宝贝是体验的核心部分。如果你在你的追踪器上看到一个罕见的神奇宝贝，放下一切，努力找到它——是的，即使这意味着在每个主要方向徘徊，直到爪印消失。* 

*或者你想开门见山， [其实还有一个单独的非官方地图 app，可以帮你找到具体的怪物。](http://lifehacker.com/poke-radar-shows-you-where-to-find-and-catch-em-all-in-1783859653#_ga=1.160738101.946745054.1460064072)T3】*

***去旅行***

*新地方，新神奇宝贝，新旅程。谁知道你会发现什么？尝试的好地方:海滩、博物馆、公园。据说还有特定地区的神奇宝贝，所以还是那句话，随身携带 *Pokémon Go* 是个不错的主意。*

***坐在车里玩***

*也许这是作弊，但是...城市里到处都是宠物商店和神奇宝贝健身房。如果你刚好乘坐出租车(或优步、拼车等)以较慢的速度穿过城市，你可以连接到许多 Pokestops，甚至抓住许多神奇宝贝。*

*就像最初的游戏一样，如果你愿意，你可以在玩的时候骑着自行车巡游——只要确保播放上面嵌入的主题曲。Lifehacker 在如何开始骑自行车和*神奇宝贝 Go* 方面有很好的指导。*

***在一天的不同时间玩***

*有些生物我只在早上见过，有些更有可能在晚上产卵。实验！*

***和朋友一起玩***

*Pokémon Go 是一种社交体验，你周围的人越多越好——你不仅可以在走路时与朋友一起射击，当你挑战健身房时，有一个团队也会有所帮助，因为多人可以同时攻击。*

*友好地与人交谈*

*外出时，你一定会发现有人在玩《T2》的《神奇宝贝》。即使你不是外向型的人，和别人聊天也有好处:他们可能会告诉你在哪里可以找到你还没有抓到的怪物，或者你应该去找的好地方。谁知道呢，也许你会交个新朋友。*

***使用诱饵***

*从技术上来说，诱饵是用来吸引神奇宝贝的，但实际上，它们比其他任何东西都更能吸引人们去某个特定的地方。选择一个好地方，放一个诱饵，如果你愿意，开始表现得像一个聚会的主人。太爽了。*

*不要侵犯或制造麻烦*

*玩*神奇宝贝 Go* 的时候保持正念是很好的。体谅一下身边的人，尽量不要做个混蛋。绝对不要为了抓神奇宝贝而违反任何法律——正如你可能在新闻中看到的那样，在这里很容易被冲昏头脑。 [Lifehacker 写了一份关于如何在玩](http://lifehacker.com/how-to-stay-safe-while-playing-pokemon-go-1783585626) 时保持安全的有用指南。*

***感谢你的 Pokestops***

*看看你周围。看看你的周围。关于 *Pokémon Go* 最好的事情之一是它帮助你重新发现你的邻居，或者至少注意到你可能永远不会看到的更小的细节。*

***保存你的星尘***

*因为最好的神奇宝贝出现在更高的等级，所以不值得在你早期发现的东西上花费任何东西。我曾经遇到过这样的情况，我煞费苦心地试图培育一只神奇宝贝，结果却在街上发现了更强大的东西。除非神奇宝贝接近或超过 1k CP，否则不要马上给任何东西通电。* 

***孵蛋***

*鸡蛋孵化需要的公里数越多，怪物就越罕见——在这种情况下，2 公里是最常见的鸡蛋类型，10 公里是最独特的。 [这里有一个参考图，告诉你每一个可以孵化的口袋妖怪。](http://www.serebii.net/pokemongo/eggs.shtml) 注意，你需要让应用程序处于活动状态，打开，并在游戏的前台计算你走向孵化的步骤！* 

***高效***

*进入 Pokestop 后，您不必逐个点击它放下的所有东西。一旦扑克变成紫色，如果你退出菜单，你会自动拿起所有的项目。请注意，Pokestops 大约每五分钟补充一次，因此您可以多次到达同一个站点。*

***获得您想要的电子解决方案***

*通常，进化一个 Eevee 会得到一个随机的结果...除非你给你的 Eevee 起绰号。“火花”让你得到雷伊布。“雷纳”让你水伊布。“火”让你，嗯，Flareon。《神奇宝贝 Go》中更聪明的复活节彩蛋之一。*

***挑战健身房***

*由 Polygon 提供，这里有一个关于你需要了解的神奇宝贝健身房的快速解释:*

 *[https://kotaku.com/embed/inset/iframe?id=youtube-video-rfIlSN2OB4s&start=0](https://kotaku.com/embed/inset/iframe?id=youtube-video-rfIlSN2OB4s&start=0)* 

*我要补充的一点是，在挑战健身房之前，先观察一下它是值得的。里面装的是什么样的神奇宝贝？你有没有可以对抗它的口袋妖怪，无论是类型上的还是移动上的？就像主要的游戏一样，它有助于规划行动路线；否则你会被揍的。*

*如果你有信心可以打倒一个健身房领导，确保事先使用幸运蛋:XP 会更甜。*

***不要被健身房领导 CP 吓倒***

*击败那些神奇宝贝比你多几百 CP 的健身房领导是完全可能的。通过划动闪避大攻击来聪明地战斗。利用类型计数器。那 1,200 CP 的鸭嘴火兽可能看起来令人生畏，但是你的 800 CP 乘龙可能会粉碎它(或者至少减少足够的 HP，让你的另一个神奇宝贝可以完成这项工作)。*

***获得健身房后卫奖金***

*如果你设法占领了一个体育馆，你可以去商店收集一些硬币和星尘作为你的奖励。健身房防御者奖励每天可以获取一次，控制的健身房越多，奖励越多。*

***不要太担心战斗***

*很容易被其他玩家训练出来的强大神奇宝贝压垮，但是不要多心。你不需要跟上任何人。只需按照自己的闲暇和节奏捕捉神奇宝贝。*
