# 接下来我们应该做什么菜？

> 原文:[https://life hacker . com/what-should-we-砂锅-next-1822675587](https://lifehacker.com/what-should-we-casserole-next-1822675587)

大家好，欢迎回到 [*将它砂锅*](https://lifehacker.com/tag/will-it-casserole) ，在这个栏目里我们看到哪些饭菜、菜肴、菜系可以重新想象，改造成砂锅。

上周，我们做了一道三重威胁的超级碗热菜——我们把它打得落花流水——但现在我想检查一些不断被提出、但从未被执行的想法。这些砂锅伴娘值得他们的一天，该死的，所以本周我们将挑选以下三个:

*   **Carne Asada 墨西哥卷饼砂锅:**根据 [A-McGee9-3](https://kinja.com/a-mcgee9-3) ，这将包含“切丁的里贝叶、油炸品豆(在鸭油中油炸)、芫荽叶石灰饭、杰克奶酪或 queso fresco，和一份绿色萨尔萨”，上面浇上酸奶油。
*   **中国人拿出砂锅:**仰光蟹，牛肉和西兰花。
*   **美食广场砂锅:**有人在我们上次的讨论中提到了椒盐卷饼狗，我认为这需要探讨。

我通常会告诉你提出新的想法，但在添加更多精彩的选择之前，让我们从中挑选一个。说出你最喜欢的菜，告诉我“美食广场砂锅菜”对你来说意味着什么。
