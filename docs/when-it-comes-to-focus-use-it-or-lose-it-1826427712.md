# 说到专注，要么使用它，要么失去它

> 原文:[https://life hacker . com/when-it-to-focus-use-it-or-lose-it-1826427712](https://lifehacker.com/when-it-comes-to-focus-use-it-or-lose-it-1826427712)

本周的精选来自爱比克泰德和他的*话语*。 [在第四册第十二章](http://www.constitution.org/rom/epicdisc4.htm) 中，他告诉我们，如果我们不小心，我们的注意力很容易丢失:

> “当你暂时转移注意力时，不要想象这一点，当你选择时，你会恢复它；但是让这个想法呈现给你，由于今天犯下的错误，你的事情必须在更糟糕的条件下。首先，也是最容易引起麻烦的是，你养成了不参加的习惯；然后是推迟注意力的习惯。通过推迟，你不断地一次又一次地赶走生活的幸福、适当的行为、与自然相一致的存在和生活。”

### 这意味着什么

爱比克泰德说，当你让自己分心，让自己的注意力下滑时，即使只有一点点，你也不要指望能回到之前的专注状态。记住，当你现在犯了这样的错误，接下来的一切都会变得更加困难。你只是把你需要做的事情推迟到以后。对爱比克泰德来说，专注是你养成并保持的一种习惯——但它是双向的。如果你经常走神，注意力转移，你会养成坏习惯，最终影响你的工作效率。

### 从中吸取什么

你不能不停地工作，也不能永远专注于一件事——休息对身心都很重要。但是到了工作的时候，你需要全力以赴，专注于需要做的事情。正如爱比克泰德所说，如果拖延注意力是有利可图的，那么完全不集中注意力会更有利可图。我们都知道事实并非如此，那么为什么在该做事的时候，你不保持你的注意力呢？

工作时间是为了工作，玩耍时间是为了玩耍。甚至当你演奏的时候，你也应该全神贯注。他说，没有焦点的一天是虚度的一天:

> “但现在当你说，‘明天我将开始参加’，你必须告诉你说，“今天我将无耻，不管时间和地点，意味着；别人有能力给我痛苦；今天我会充满激情和嫉妒。”看看你允许自己做了多少坏事。"

尽情享受你的游戏时间，但是要专注于你正在做的事情。当心一种转瞬即逝的心态，它不断地从一件事跳到另一件事。这种分心的习惯会影响你的工作，让你的生活更加艰难。有目的地做事情，专注地做，你会发展出一个敏锐的、多产的头脑。
