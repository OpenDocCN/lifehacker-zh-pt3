### 我们如何测试

为了测试，我用每个控制器玩了几个小时不同的游戏。我花了大部分时间看《2D》游戏，看《忍者外传 3》和《超级街霸 2》。我选择*忍者外传*是因为它需要按钮和方向键的快速响应时间，而*超级街头霸王 II* 是罕见的在一个控制器上使用所有六个按钮的游戏之一*和*需要使用方向键的灵活性。至于 3D 游戏，我花得最多的是 PlayStation 1 上的 [*超级马里奥 64*](https://en.wikipedia.org/wiki/Super_Mario_64) 任天堂 64*[*疯狂出租车*](https://en.wikipedia.org/wiki/Crazy_Taxi)*dream cast*和 [*雷神之锤 II*](https://en.wikipedia.org/wiki/Quake_II) 。***

**软件方面，我在 Windows 上测试了所有带有 [RetroArch](https://www.libretro.com/index.php/retroarch-2/) 的控制器，在 Mac 上测试了 [OpenEmu](http://openemu.org/) ，在树莓派上测试了 [RetroPie](https://retropie.org.uk/) 。我测试的每一个控制器都可以在我测试的每一个操作系统上工作，尽管我在一些蓝牙控制器上有一些设置上的问题。**

**最后，经过大量的测试，我认为没有完美的全能控制器，所以我把我的选择分成两类:一个只适用于 2D 游戏的控制器，一个适用于 2D 和 3D 游戏的全能控制器。**

### **我们为 2D 游戏挑选的:水牛城经典 USB 游戏手柄**

**如果你只是想玩 NES、SNES、创世纪或 Turbografx-16 等主机代的 2D 游戏，我发现少即是多，为复古游戏设计的较小控制器比现代双模拟选项感觉好得多。** 

**在我所有的测试中，有一个控制器不仅让我感觉最好，而且来自各种渠道的推荐:布法罗经典 USB 游戏手柄 。水牛经典控制器在亚马逊上有着出色的评论，并得到了 [How-To Geek](http://www.howtogeek.com/286842/how-to-build-your-own-nes-classic-with-a-raspberry-pi-and-retropie/) 和 [*铲子骑士*开发者游艇俱乐部游戏](http://yachtclubgames.com/controllers/) 的推荐。它还经常从像[Reddit](https://www.reddit.com/r/retrogaming/comments/3104cy/buffalo_usb_snes_controller_review_feels_pretty/)和[rack boy](http://www.racketboy.com/forum/viewtopic.php?f=27&t=39614)这样的名单中脱颖而出。**

**Buffalo Classic 是原 SNES 控制器的近似精确尺寸复制品，尽管它选择了 Super Famicom 风格的面板按钮设计。这些按钮很有弹性，继续工作，就像我第一次打开盒子时一样。反馈也和原来的 SNES 控制器一样好，这意味着 d-pad 的每个动作都有令人愉快的“咔哒声”，按钮在按下后会立即反弹。既然是 USB，那就好设置了。只要插上电源，你就可以开始工作了。**

**与我们过去推荐的托米 SNES 控制器 相比，这是一个巨大的进步。托米控制器感觉制作不良，使用几个小时后，按钮开始吱吱作响，感觉糊里糊涂的。d-pad 感觉像是用泡泡糖固定在控制器上，只有当你用很大的力按下时才会有反应。**

**当然，超级任天堂的设计不是你唯一的选择。我还测试了 [Hyperkin Genesis](https://www.amazon.com/Hyperkin-Inc-Genesis-GN6-USB-Controller/dp/B00KX75UT6/ref=pd_sbs_63_t_1?_encoding=UTF8&asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-best-controllers-for-retro-gaming-1791090209&asc_source=&psc=1&refRID=TZTPAG518EJXKJ5PGRDZ&tag=kinjalifehackerlink-20) 控制器。这是我能找到的唯一一个有着不错评价的创世纪风格的控制器。六个面部按钮感觉不错，但 d-pad 比原来的 Genesis 控制器更僵硬。它也没有漂亮的滚动动作，这使 Genesis 控制器有别于 SNES，并使其非常适合像*街头霸王 II* 这样的游戏。如果你只是玩 Genesis 游戏，Hyperkin 控制器可以完成这项工作，但我发现 SNES 风格更适合一般用途，特别是当你考虑到 Genesis 游戏实际上使用了所有六个面部按钮时。** 

**如果你想要或需要无线，那么蓝牙游戏手柄的 [8BitDo](http://www.8bitdo.com/) 系列是很好的构造，尽管是奢侈的选择。虽然 SNES30 Pro 是我在这里测试最多的游戏手柄，但我也接触过其他几款 [8BitDo 型号](http://www.8bitdo.com/) ，包括[nes 30](http://www.nes30.com/)(35 美元)和[snes 30](https://www.amazon.com/Wireless-Bluetooth-Controller-Classic-Joystick/dp/B014QP2H1E/ref=sr_1_1?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-best-controllers-for-retro-gaming-1791090209&asc_source=&ie=UTF8&keywords=snes30&qid=1484081384&sr=8-1&tag=kinjalifehackerlink-20)(35 美元)，它们都是类似的构造。尽管它们的外观，控制器并不完全是原来任天堂控制器的复制品。按钮和 d-pad 比原来的更灵活，感觉更像一个现代的控制器。8BitDo 的全线控制器都是可定制的，包括固件更新， [甚至支持按键重映射](http://www.8bitdo.com/nes30pro/support.html) 。对于我们大多数人来说，这是多余的，但如果你想要大量的高级选项，那么 8BitDo 丰富的控制器系列将满足你的需求。从我的经验来看，控制器的质量延伸到他们的所有选项。**

**然而，SNES30 Pro 和 NES30 都很小，尺寸大约与原装 NES 控制器一样大(没有你记忆中的那么大)。如果你的手很大，握起来会不舒服。蓝牙也增加了不少成本，尤其是 SNES30 Pro。如果你不需要无线，这完全是大材小用，但 8BitDo 是一款优质、时尚的控制器，前提是你的手比较小，并且不介意支付额外费用。**

**我也给了任天堂 64 灵感 [Retrolink 控制器](https://www.amazon.com/gp/product/B008L3UUPS/ref=od_aui_detailpages00?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-best-controllers-for-retro-gaming-1791090209&asc_source=&ie=UTF8&psc=1&tag=kinjalifehackerlink-20) 一个普通老 NES、SNES 和创世纪游戏的机会，它真的很糟糕。控制器本身制作精良，但任天堂 64 奇怪的按钮布局意味着除了任天堂 64 游戏之外，几乎不可能用这个控制器做任何事情*。这包括笨拙放置和大小怪异的模拟操纵杆，它以令人满意但奇怪的橡皮筋效果弹回原位，就像原来的控制器一样。d-pad 就不能这么说了，最多感觉是糊状的。d-pad 上缓慢的响应时间使得在像*忍者外传*这样的游戏中很难通过一个关卡。***

*我将在下一节深入探讨这一点，但经过几个小时的测试，我发现罗技 F310、Xbox One 和 DualShock 等现代控制器太笨重，不适合专门用于 2D 游戏。这并不是说它们不工作，但它们很重，如果你不打算玩任何超越 SNES 一代的东西，额外的模拟输入是不必要的。为此，20 美元的布法罗经典 USB 游戏手柄就是你所需要的。*

### *我们为 2D 和 3D 游戏挑选的:索尼 DualShock 4 无线控制器*

*如果你正在寻找一个兼容 2D 和 3D 游戏的控制器，索尼 DualShock 4 就是你想要的。方向键非常适合 2D 游戏，而模拟摇杆也非常适合 3D 游戏。它体积太大，价格太贵，不推荐只用于 2D 游戏，但如果你想玩从 NES 到 PlayStation One 的所有游戏，DualShock 可以满足你。* 

*DualShock 4 对我来说胜在多种原因，但最主要的是 d-pad。按钮反应灵敏，每个单独的垫的海绵点击感觉比我测试的其他选项更像一个经典的控制器。当你在玩老游戏时，面部按钮的位置、大小和风格也很自然。如果你已经有了 DualShock 4，你可能会很高兴用它来玩复古游戏。它也是像 [、断线钳](http://thewirecutter.com/reviews/best-pc-gaming-controller/) 和 [电脑游戏玩家](http://www.pcgamer.com/the-best-controller-for-pc-gaming/) 这样的人的最爱。你将需要一些 [附加软件](http://ds4windows.com/) 来将 DualShock 4 与 PC 配对，尽管使用起来并不十分困难。另一方面，它在树莓派 上与 [RetroPie 配对，在 Mac](https://github.com/retropie/retropie-setup/wiki/Setting-up-a-PS4-controller) 上与 [OpenEmu 配对。如果你喜欢更便宜的选择，PS3 的 DualShock 3 和 DualShock 4 一样好，但遗憾的是，它不只是开箱即用，而且与 DualShock 4](https://github.com/OpenEmu/OpenEmu/wiki/User-guide:-DualShock-Controller-Pairing) 相比，需要 [额外的设置。](http://kotaku.com/how-to-use-a-console-controller-on-your-pc-889800353)*

*PS4 控制器的 squishier d-pad 与 XBox One 控制器的 clicky d-pad 形成了鲜明的对比，对于大多数复古游戏来说，后者感觉*可怕*。DualShock 的 d-pad 像经典的 NES 控制器一样软化点击，Xbox One 的感觉和声音更像是在键盘上按下一个键。虽然 d-pad 风格是一个偏好问题，但如果你正在寻找与旧控制器相同的感觉，Xbox One 不符合要求。除此之外，控制器中间的 d-pad 位置也很尴尬，对于长时间的游戏来说不太好用。更便宜的 Xbox 360 控制器也是如此，尽管该控制器上的 d-pad 基本上无法用于大多数复古游戏，因为它反应迟钝。* 

*前面提到的 SNES30 Pro 也和之前一样适合这一部分，但在 3D 游戏方面有所欠缺。对于大多数人来说，它太小了，不能用于任何需要双模拟棒的游戏，因为它们太小了，而且挤得太近了。同样，控制器顶部的四个按钮放置得很糟糕，所以很难使用它们。尽管如此，如果你正在寻找一款主要用于 2D 游戏的游戏手柄，也可以处理偶尔的 3D 游戏，那么 SNES30 Pro 很好，只是不要指望它会很舒服。*

*如果你只打算玩任天堂 64 游戏，那么 [RetroLink 控制器](https://www.amazon.com/gp/product/B008L3UUPS/ref=od_aui_detailpages00?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-best-controllers-for-retro-gaming-1791090209&asc_source=&ie=UTF8&psc=1&tag=kinjalifehackerlink-20) 在复制怪异的任天堂 64 控制器的感觉方面做了令人钦佩的工作，尽管单一的模拟杆使它完全无法在任何其他主机上玩游戏。除了任天堂 64 游戏，这款控制器是不可能推荐的。*

*如果你正在寻找一个便宜的选择，罗技 F310(和它的无线同行 [F710](https://www.amazon.com/Logitech-940-000117-Gamepad-F710/dp/B0041RR0TW/ref=sr_1_1?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-best-controllers-for-retro-gaming-1791090209&asc_source=&ie=UTF8&keywords=logitech f710&qid=1484155477&sr=8-1&tag=kinjalifehackerlink-20) )是大卖家，但即使是 F310 的 20 美元价位，也很难推荐。模拟拇指操纵杆感觉很紧，面部按钮有点太大，输入下降太深。这意味着按下一个按钮需要太多的力量，这给任何需要精确计时的游戏带来了问题。控制器本身的形状也很奇怪，感觉有点像 Xbox 控制器和 DualShock 的混搭。大多数评论者，包括 [多边形](http://www.polygon.com/2016/1/11/9848652/best-game-controller-custom-controller-windows-pc-2016) 和 [钢丝剪](http://thewirecutter.com/reviews/best-pc-gaming-controller/) 都指出 F310 握起来感觉怪怪的，我倾向于同意。F310 上的 d-pad 有令人满意的点击感，但这并不能转化为游戏结束时的响应能力。*

*游戏控制器是个人物品，哪一个最适合你很大程度上取决于你手的大小和你过去用过的东西。如果你对你现在所拥有的感到满意，没有理由去改变什么，但是如果不满意，有一个选项可以满足你的需求。*
