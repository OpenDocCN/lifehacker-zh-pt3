# 用这个 Facebook Messenger 机器人和颠倒的人说话

> 原文：<https://lifehacker.com/talk-to-the-upside-down-with-this-facebook-messenger-bo-1820515869>

无论你是想在不牺牲公寓押金的情况下体验*陌生事物*的精神，还是想和 SmarterChild (RIP AIM)聊聊天，都有一款聊天机器人可以满足你。



总部位于伦敦的数字创意公司 Tangent 创造了 El 机器人，这是一个脸书信使机器人，主要通过互动文本框和浅色 gif 墙进行交流。

主菜单为你提供了可点击的选项来会见角色、听音乐、对着墙说话或阅读新闻。如果你选择第一个选项，它会问你几个问题，这些问题似乎旨在确定你在这个系列中的位置和你知道多少，然后给你一系列关于你所选角色的 gif 和琐事。其他的允许你找出一个播放列表，阅读最近的新闻故事，或者用一个臭名昭著的墙的 GIF 进行一个非常简短的只有点击选项的对话。

大多数与 El 的交互需要通过按钮选项来完成，而不是文本输入；在 D&D 主要政党之外输入问题甚至人物名字只会引起混乱。她没有回应“本尼”，这似乎很不领情，如果有点合适的话。我也不知怎么的在问了太多不认识的字之后成功破解了一个插件，感觉应该会引发一个流鼻血的 GIF。

它明显缺乏态度，不像它的同名物，但它足以在你的办公桌上度过有趣的 15 分钟，我发现至少有一个有趣的小复活节彩蛋(我们不会在这里破坏它)。El bot 也只有一周大，所以她还需要学习。谁不想每天多跳一会儿舞呢。
