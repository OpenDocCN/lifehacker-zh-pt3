# 最好喝的超市品牌咖啡是什么？

> 原文：<https://thetakeout.com/what-s-the-best-tasting-supermarket-brand-coffee-1829005959>

尽管我在星巴克花了那么多钱(现在我又回到了咖啡因的行列 )，你可能会认为我会拿走其中的一部分，买一个合适的保温瓶，把周末在家喝的咖啡带到工作场所。但是，在我将要做出这样一个改变生活(也可能省钱)的决定之前，我怎么能确定我在家里，从超市买的咖啡是最好的呢？毕竟，多年来我的首选都是八点钟，但为什么福尔杰和麦斯威尔已经存在了这么久？在杂货店也很容易买到(甚至在药店，我在沃尔格林买了一大堆):星巴克、Dunkin' Donuts 和麦当劳麦咖啡。



显然，只有一个办法可以解决这个问题:在超市里做一批不同种类的咖啡，提供给我认识的一些最喜欢喝咖啡的人——我的同事——看看他们认为哪一种最好。

让我告诉你，没有什么比让人们在下午 3 点钟喝一杯不那么美味的咖啡更能让你在工作中成为朋友。我两耳之间的空间还在嗡嗡作响。我的舌头感觉麻木，好像我所有的味蕾都受够了，开始罢工。

是的，事实证明，超市咖啡(对于我们这些习惯于高辛烷值新鲜酿造的人来说)可能不太好，尤其是喝黑咖啡的时候。测试的一些品牌远不如其他品牌可口，但正如一位品尝者所说:“我宁愿不喝咖啡，也不喝这些。”我参与这个项目的办公室咖啡爱好者坚定地说，“如果我不喜欢你，我现在真的会生你的气。”很公平！

让我们从好的方面开始:人们不讨厌 T2 的邓肯甜甜圈，但也不真正喜欢它。"好浓的香气，不是很浓的味道，真的没有多少."就我个人而言，我觉得它有一点点金属味(然而， [获得了我们的最佳外带连锁咖啡](https://thetakeout.com/what-chain-has-the-best-to-go-coffee-1822106406) 盲品测试)。

更受欢迎，也可能有点令人惊讶的是 Folgers T1，由于它的熟悉度，许多人似乎都喜欢它。“它尝起来就像无底的餐车咖啡……令人欣慰。”另一个人同意，“它介于加油站咖啡和奶奶咖啡之间，我喜欢加油站咖啡。”听到这里，我们的咖啡爱好者不寒而栗，指责福尔杰咖啡尝起来像“掺水的潮湿的甜纸板”。另一位品尝者表示同意，称之为“水汪汪的豆汁”这位咖啡专家接着试图教我罗布斯塔咖啡豆和阿拉比卡咖啡豆的区别，福尔杰和麦斯威尔倾向于前者，而星巴克倾向于后者。她说，当加入奶油时，“好的咖啡豆会有明显的味道变化”，这对我来说是幸运的，因为坦白地说，我发现很难喝任何没有奶油的咖啡豆。

在所有超市咖啡中，人们最喜欢麦斯威尔咖啡。他们说它“更黑、更浓……实际上很像咖啡。”然而，这种爱并不是全面的，正如一个人所说，“它仍然没有味道”，但“它没有以不同的方式产生味道。”另一个品尝者说它尝起来像涂了黄油的爆米花，而另一个稍微有点兴奋地说，“如果你不在乎你在喝什么，这完全可以用奶油来代替。”

但是即使是微弱的赞扬也远远超过了对麦当劳麦咖啡的反应。我们这一批尝起来酸酸的，焦焦的，坦白说，一点也不好。你还会认为**星巴克**会吃得很好——它确实很突出，但只是比其他的要暗得多，尽管咖啡豆是温和的派克地方烘焙。品酒师形容它非常朴实、坚韧、更黑、更厚，但不一定比更可口的麦斯威尔好。

## 获胜者:麦克斯韦尔豪斯

我真的没想到会把这个巨大的蓝色浴缸带回家，成为我的家庭咖啡，但看起来这就是事情的发展方向。此外，我将向我的咖啡专家朋友询问烘焙我自己的咖啡豆和在家进行倒酒(她喜欢的方法)，这应该有望使我的家庭咖啡冒险更加美味。
