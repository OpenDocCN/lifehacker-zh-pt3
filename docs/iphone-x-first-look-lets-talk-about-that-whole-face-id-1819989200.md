# iPhone X 先看:先说 Face ID

> 原文：<https://gizmodo.com/iphone-x-first-look-lets-talk-about-that-whole-face-id-1819989200>

iPhone X 的[【Face ID】](https://gizmodo.com/face-id-raises-some-scary-questions-here-are-some-answe-1804313075)摄像头模块受到了比苹果预期更多的压力。面部识别安全技术是蒂姆·库克和朋友们主题演讲期间吸引所有人眼球的惊人功能——后来它又被 [反复指为长期传言的 iPhone X 短缺和延迟背后的罪魁祸首](https://gizmodo.com/rumors-say-apples-iphone-x-launch-will-be-screwed-by-de-1819413452) 。那么这是怎么回事呢？

Watch

昨天下午，我从苹果公司拿了一部 iPhone X 评测机，我做的第一件事——我需要做的第一件事——就是测试被大肆宣传的 Face ID(一旦我真的用了这该死的东西几天，就会有更完整的评测)。更好的摄像头和华丽的显示屏肯定是至关重要的功能，但没有哪个组件比 Face ID 对 iPhone X 的长期成功更重要。这是苹果在生物识别安全方面的下一次重大尝试，如果该公司搞砸了，如果 Home 键上的廉价传感器比显示屏上的精致传感器阵列更好，那么苹果只不过是建造了一个真正漂亮的 [Galaxy S8](https://gizmodo.com/samsung-galaxy-s8-review-the-prettiest-phone-wins-1794526975) 。

这是不可能的。苹果的优势不在于尖端的硬件，而在于当今手机中最友好的用户界面和体验。新的 Face ID 模块应该允许你只需扫一眼你的手机，然后滑动就可以进入你的主屏幕。但这真的比指纹传感器简单吗？

我应该指出，Face ID 也应该使手机更安全。触控 ID 绝对是整洁的，但一直以来 [相当容易受到](http://www.ccc.de/en/updates/2013/ccc-breaks-apple-touchid) 的恶搞企图。今年早些时候，纽约大学坦登工程学院主席纳西尔·梅蒙向[Gizmodo](https://gizmodo.com/apples-touch-id-is-probably-doomed-and-thats-okay-1806528982#_ga=2.229447132.1594605431.1509415742-923911060.1509115869)解释说，触控 ID 实际上会拍摄一个人指纹的多张微小照片。“它正在做的是捕捉很小很小的方块——很小的部分指纹，”他说。

据苹果公司称，这意味着陌生人有五万分之一的几率可以通过触控 ID 接入手机。有了 Face ID，苹果声称这个数字上升到百万分之一。(尽管苹果公司表示，如果你有一个同卵双胞胎或其他特征非常相似的亲戚，这个数字会大幅下降)。

但它需要看到你的整张脸才能工作，随着冬天的临近，你开始看到新技术固有的一些缺点。触控 ID 和 Face ID 都要求你摘掉手套，但是，正如我在今天下午的测试中提到的，Face ID 也要求你摘掉遮住鼻子的围巾或遮住眉毛的帽子。事实上，尽管苹果虚张声势，但在某些情况下，Face ID 开始感到不太方便。当我的手机还在口袋里时，我不能盲目地解锁手机，因为你必须向上滑动才能解锁手机，所以当我解锁 iPhone X 时，感觉有时会多一个步骤

在常规练习中，Face ID 实际上大部分时间都工作得几乎完美无缺。我想都不敢想。我把手机向我的脸倾斜，然后滑动。没有病人拿着按钮。显示屏顶部的小锁简单地切换了一下，我就进去了。让我感到有趣的是，如果我做鬼脸，它甚至会解锁，我必须承认，在我做鬼脸的时候解锁我的手机是一种享受。

更好的是，即使在你认为面部识别系统肯定会失败的情况下，它也能很好地工作。试图在漆黑的房间里打开我的手机，还是在摘下我脸上的眼镜之后？没问题！

不过，讨论眼镜是一件很重要的事情。我买 iPhone X 才几个小时，Face ID 模块花了这么长时间才理解我面前的塑料大平面。在 Face ID 的初始设置中，你盯着相机慢慢旋转你的脸，这项技术需要时间来学习你的特征，使用时间越长，它就会变得越聪明。这意味着你第一次在黑暗的房间里或戴着眼镜尝试打开手机时，可能会失败。但是在你做了几十次之后，它知道你的每一个轮廓和缝隙，而且，根据苹果公司的说法，它甚至可能在你有一把巨大的胡子或者忘记修剪你的多毛基因给你的胡子时工作。

但这只发生在我使用这款手机的第一周。当我有了更多的时间来玩这个设备，并且学到了更多的东西时，我会向大家汇报。
