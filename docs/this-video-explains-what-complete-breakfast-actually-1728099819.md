# 这段视频解释了“完整早餐”的真正含义

你可能已经有很多广告告诉你，他们的麦片是“完整早餐的一部分”，但这到底是什么意思呢？这段视频解释了为什么卡通吉祥物如此频繁地使用这个短语，以及“完整的早餐”实际上应该包含什么。



这个视频来自 YouTube 频道[(美国化学学会的一个系列)，用化学解释了你的“完整早餐”盘子里应该有什么:碳水化合物和蛋白质的组合。小妖精和兔子告诉你他们的麦片是“完整早餐的一部分”的原因是因为从技术上讲，它包含了碳水化合物部分。但是大多数这些谷物都是由简单的碳水化合物组成的，这些碳水化合物只能提供短暂的能量。对于一份真正的“完整的早餐”，你需要复杂的碳水化合物，从蔬菜和全谷物等食物中为你提供更持久、更均衡的能量来源。你还想从鸡蛋(](https://www.youtube.com/channel/UCdJ9oJ2GUF8Vmb-G63ldGWg) [适量食用没问题](https://lifehacker.com/the-us-is-finally-dropping-its-outdated-guideline-again-1686339755) )和酸奶等食物中获取一些蛋白质，以帮助你在更长时间内保持饱腹感，同时减少暴饮暴食的机会，并战胜令人讨厌的渴望。在上面的视频中了解更多这一切背后的科学。
