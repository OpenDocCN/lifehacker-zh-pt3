# 人们后悔使用的移动应用

> 原文：<https://lifehacker.com/the-mobile-apps-that-people-regret-using-1822817522>

人们自愿使用 Grindr、Two Dots 和 Reddit。根据手机应用程序和人道技术中心的一项联合调查，他们并不真的对 感到满意。

Moment 询问其用户对他们在不同应用上花费的时间有何感受。(问题不在于用户是否后悔接触过这款应用，而是他们是否希望在上面花更多或更少的时间。答案不会让你震惊，但它们更能证明，如果你直接删除一些最常用的手机应用，你会更开心。

在人们最后悔使用的 15 个应用中，没有一个是典型的工作应用。两个是约会应用，这可以被视为必要的邪恶。其中七个是社交媒体应用，许多用户都可以删减。其他六个是手机游戏，任何人都可以立即删除，而不会产生实际的不良后果。

该项目还列出了用户乐于花时间使用的前 15 个应用程序:用于冥想、阅读、音乐、播客、健身的应用程序，以及用户每天只花几分钟时间使用的天气和日历等复杂应用程序。值得注意的是，每天只使用顶空 4 分钟就足以达到 99%的满意率。

真正有趣的是这张图表，它记录了“快乐”用户和“不快乐”用户的平均每日使用时间。如果你想在不删除的情况下减少一些应用，试着用 Moment 跟踪你的应用使用情况，并将这些平均值作为你的目标。
