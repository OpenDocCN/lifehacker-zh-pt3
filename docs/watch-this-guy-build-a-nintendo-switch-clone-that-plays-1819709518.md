# 看看这个家伙建造了一个任天堂 Switch 克隆体，可以玩成千上万的经典游戏

> 原文：<https://gizmodo.com/watch-this-guy-build-a-nintendo-switch-clone-that-plays-1819709518>

仍然很难在商店里找到任天堂 Switch？或者也许你已经厌倦了等待任天堂承诺的充满复古游戏的在线商店？蒂姆·林德奎斯特(Tim Lindquist)亲自动手，从零开始创建了一个任天堂 Switch 克隆体 ，它可以模拟 50 多个经典系统的游戏。



林德奎斯特整个夏天都在设计、工程和建造他的 [任天堂 RP](https://timee1994.weebly.com/nintimdo-rp.html) (慢速拍手)游戏机，它与任天堂的 Switch 有很多共同的功能，包括一个宽大的 LCD 屏幕，两侧是(不可拆卸的)控制器。他 [在这里](https://timee1994.weebly.com/nintimdo-rp.html) 完整地记录了他的项目，但是你可以通过 Lindquist 制作的这个延时视频更好地了解制作自己的便携式游戏机需要什么。

 [https://gizmodo.com/embed/inset/iframe?id=youtube-video-pOqiqRHrAHE&start=0](https://gizmodo.com/embed/inset/iframe?id=youtube-video-pOqiqRHrAHE&start=0) 

定制的 Nintimdo RP 外壳比 Switch 的外壳厚，但这是因为它充满了现成的部件，如运行 Linux 和仿真器的 Broadcom BCM287 处理器、完整的冷却系统、巨大的 10，000 mAh 可充电电池(Switch 的电池只有 4，310 mAh)和支持两个 USB 端口和一个 HDMI 端口的主板，以便在更大的屏幕上玩游戏。

根据 [Hackaday](https://hackaday.com/2017/10/20/diy-nintendo-switch-may-be-better-than-real-thing/) 的说法，Lindquist 的创作运行了 [RetroPie 模拟器](https://retropie.org.uk/) ，让控制台可以访问一个巨大的经典视频游戏库，以及 [模拟器](http://www.emulationstation.org/) ，因此 Nintimdo RP 的主屏幕看起来像任天堂 Switch 的一样光滑。我们不知道这个家酿游戏机最终花了林德奎斯特多少钱来建造，但不管价格是多少，最终的结果似乎完全值得。
