# 如何写出完美的催眠曲

> 原文：<https://lifehacker.com/how-to-write-the-perfect-lullaby-1825931841>

我喜欢给女儿唱摇篮曲。作为一个突然拥有被俘虏的观众(一个生理上不可能离开我的人)的音乐剧极客，我怎么能不离开呢？在午睡时间，睡觉时间，真的任何时候，我都会哼唱自己熟悉的经典曲目。一对夫妇脱颖而出:“不要为我哭泣，小宝贝”(唱着“不要为我哭泣，阿根廷”)和一首歌，我刚才提到的婴儿催眠。对于那首歌，我选择了勃拉姆斯的催眠曲 的曲调，唱着“*去睡觉吧，孩子，你很困了。你越来越困了，你现在就去睡觉*”一遍又一遍。就像时钟一样，在大约 27 轮之后，这个孩子最终会昏过去。摇篮曲是我的最爱。



虽然给婴儿唱摇篮曲是一种已经传承了好几代的仪式，但它最近受到了一些关注。研究人员发现这些简短重复的歌曲有一些特别之处，小家伙们(希望如此！)联想到睡眠。根据[a*New York Times*piece](https://www.nytimes.com/2018/05/07/well/family/lullabies-parents-babies-bonding-mothers-singing.html)的说法，摇篮曲可以帮助婴儿创造平静下来的神经通路，恢复唱摇篮曲的人的幸福感(一项研究发现，给婴儿唱歌与较少的产后抑郁症症状相关 [),并为父母和婴儿提供一个联系的机会。安妮塔·柯林斯博士是即将出版的新书《催眠曲效应》的作者，她告诉《时代周刊》，“婴儿的听觉处理或听觉实际上是他们生命头两年最重要的感觉，这种感觉收集了关于他们世界的最多信息。"](https://www.nytimes.com/2018/05/02/magazine/how-to-sing-a-lullaby.html)

虽然最有意义的摇篮曲往往是你自己写的，但不是每个人都对对着一个流口水的小人儿唱编造的歌词感到完全自在。这需要一点放手。这里有一些提示可以帮助你开始。

如果你认为自己唱不准，不要担心。宝宝不在乎。

把你自己的歌词换成熟悉的歌曲，唱一些简单的个人话题。摇篮曲项目是卡耐基音乐厅威尔音乐学院的一项倡议，旨在帮助母亲们通过自己创作的摇篮曲与婴儿建立联系。该项目的创始人之一、歌手兼词曲作者艾米丽·伊根(Emily Eagen)在这个视频中分享了你如何将熟悉的歌曲(如“Frère Jaques ”)与你自己的歌词进行交换——尝试唱关于你孩子的一天、玩具或身体部位的歌曲。

保持曲调简单。斯洛文尼亚创作歌手兼人类学家 Katarina Juvancic 告诉《T2 时报》 [【T4 时报】](https://www.nytimes.com/2018/05/02/magazine/how-to-sing-a-lullaby.html) 说，民间摇篮曲的曲调很棒，因为它们提供了“久经考验的镇静效果，通常由五音符五声音阶组成。”

用你的声音“麻醉他们”。 Juvancic 还补充道，你应该保持低音量。“你基本上是通过一遍又一遍地重复同样的节奏模式给他们下药，”她说。

不要因为你的孩子长大了就停下来。你可以给任何年龄的听众唱摇篮曲——或者任何物种！宠物和老人是很好的观众。我女儿现在五岁了，我们仍然给她所有的朋友和最喜欢的毛绒动物唱“晚安之歌”。这是令人心痛的甜蜜。我会记住这些时刻。我希望她也会。
