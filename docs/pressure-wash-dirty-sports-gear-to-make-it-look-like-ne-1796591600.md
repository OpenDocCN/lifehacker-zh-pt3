# 压力清洗脏的运动装备，让它看起来像新的一样

如果你或你的孩子从事户外运动，你就会知道制服很快就会被难以去除的灰尘和草渍覆盖。这一招可以让那些制服保持最佳状态。

有时候普通的洗衣机根本不能满足你的需求——即使你用了一堆不同的洗涤剂。你真正需要的是一台压力洗衣机，然后你就可以穿着一条崭新的棒球裤溜回家了。同样的技巧可以用于运动衫、足球袜、护腿等等。

这段来自亚伯拉罕·巴特勒 T3[的 YouTube 视频和来自 redditor xian24](https://www.youtube.com/watch?v=-5DKz0LEkqg) 的这张T5】GIF 显示了这有多简单。

此外，请注意这不是某人的车库。如果你自己没有压力清洗机，下次去洗车的时候带上脏的运动装备就行了。将裤子、衬衫或其他衣物挂在地垫夹子上，然后喷洒在漂洗装置上。他们很快就会恢复到原来的亮白色。别担心，运动装备更耐用，所以，就像你在激烈的比赛中一样，它可以应对压力。
