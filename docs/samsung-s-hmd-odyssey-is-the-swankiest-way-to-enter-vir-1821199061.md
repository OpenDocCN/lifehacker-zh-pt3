# 三星的 HMD 奥德赛是进入虚拟现实的最奢华的方式

> 原文:[https://gizmodo . com/Samsung-s-HMD-odyssey-is-the-swankiest-way-to-enter-vir-1821199061](https://gizmodo.com/samsung-s-hmd-odyssey-is-the-swankiest-way-to-enter-vir-1821199061)

第一次把三星的 HMD 奥德赛捧在手中，脑子里只有一个念头:*这不是玩具。*没想到用别的 VR 头戴设备。宏碁的是 [笨重而酷炫的](https://gizmodo.com/the-new-star-wars-ar-game-is-clunky-cool-still-too-ex-1820092627#_ga=2.121587915.2126096023.1513601885-774642885.1491417459) ，PSVR 的是 [圆滑而简约的](https://gizmodo.com/skyrim-vr-is-everything-wrong-with-virtual-reality-righ-1820981227) 。但三星的 HMD 奥德赛是一个档次以上，一个完整的 1.4 磅重的改变现实的小工具挂在你的脸上，带 AKG 耳机，麦克风内置在它明显不吸引人的前置耳机中。它更类似于 HTC Vive 和 Oculus Rift 等重型、严肃的设备。三星设备有其突出的优点和缺点。我几乎立刻意识到，这是一个为深度、沉浸式游戏体验而构建的笨拙的东西。这不是一个玩具——这是一个带着台式电脑去星巴克玩英雄联盟的人的耳机。

Watch

我刚刚虚构的这个人，我们叫他迈克，是描述 HMD 奥德赛的最佳方式。迈克在星巴克，六杯浓咖啡，打破记录，扼杀电竞事业。他没有时间露面。同样，HMD 奥德赛号建造得也不漂亮。这款耳机重 1.4 磅，在长时间播放时会拖累你。这款 [比 Rift](http://www.techradar.com/news/wearables/htc-vive-vs-oculus-rift-1301375) (1.03 磅)和 Vive (1.2 磅)都重。玩*毁灭战士 VR* 的时候，我总能感觉到它在我脸上的重量，这一点变得更加明显，因为游戏不断要求你在拐角处伸长脖子并随着怪物在你身后潜行而旋转，以跟踪敌人的运动。

但是除了耳机的问题之外，毁灭战士虚拟现实被证明是麦克能力的完美展示。*毁灭战士 VR* 是一款“所见即所得”的游戏:爆头、血盘、爆炸僵尸。屠杀是不优雅的，无情的，夸张的；就像一个九岁的孩子第一次看到*变形金刚*时兴奋的叽叽喳喳。但这正是三星的高分辨率令人眼花缭乱的地方:1440×1600 像素，90Hz。这实际上比其他“一流”VR 头戴设备高了一步 HTC Vive 和 Oculus Rift 都拥有 90Hz 的 1080 x 1200 分辨率。

更准确地说，三星 HMD 公司的决心使它*物有所值*:我瞄准一个膝盖骨，错开一个怪物，然后直接将*传送到敌人*中，引发爆炸。因为有了明确的决心，我有足够的信心去瞄准弱点。弹药几乎是无限的，所以虽然我可以毫不费力地从远处砍倒怪物，但清晰的视觉效果让额外的努力物有所值。它给了一个斧头杀手外科医生的精确性，给原本可能是懒惰的伤害增加了令人惊讶的策略。作为一个在 FPS 游戏中变得非常懒惰的人，那是一个壮举。

选择性懒惰是任何新兴技术的关键。HTC Vive 是一个众所周知的难以取悦的运动训练器，要求你设置游戏区，不要被电缆绊倒。Oculus Rift 稍微简单一点，但是 [仍然需要外部摄像头](https://gizmodo.com/oculus-rift-review-this-shit-is-legit-1767483554#_ga=2.26354869.964887307.1513603287-1626665990.1512676959) 进行设置。**HMD 奥德赛是一款 Windows 混合现实耳机，只需要你把它插到一台有足够图形动力的电脑上。这不仅仅是 Windows 混合现实的优步黑，而是虚拟现实。**

**简而言之，全方位的沉浸感，更高的分辨率和内置音频——所有这些都是令人生畏的 499 美元价格的一部分——协同工作，创造出彻底的沉浸式体验。迈克不会乱来的。《毁灭战士》VR 的口吃来自于它的控制——它们感觉太轻了，以至于枪看起来太大，而且太多的动作被分配到太少的按钮上——但是《奥德赛》几乎完美地工作了。如果你要走 Windows 混合现实路线来满足你的 VR 需求，这就是你要拥有的耳机。**
