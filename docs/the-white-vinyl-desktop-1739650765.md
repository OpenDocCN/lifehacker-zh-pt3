# 白色乙烯基桌面

> 原文：<https://lifehacker.com/the-white-vinyl-desktop-1739650765>

我们已经在唱片和唱机周围看到了一些很棒的台式机 ，但是我喜欢这个，因为我有同样的壁纸。我没有与 [本 K](http://kinja.com/alivethroudeath) 相同的小部件，虽然我可能会很快更正。他是这样安排一切的。



如果你不知道的话，本正在一台 Mac 电脑上运行这个程序，用的是为壁纸的暗点和亮点定制的很好的剂量。如果你想要同样的外观或感觉，以下是你需要的:

*   [白色乙烯基壁纸](http://wallpaperstock.net/white-vinyl_wallpapers_35315_2560x1440.jpg) 来自 [本 Reddit 线程](https://www.reddit.com/r/perfectloops/comments/1nkb3s/vinyl_disc/)
*   [über sicht](http://tracesof.net/uebersicht/)，一个 OS X 定制套件 [我们之前已经精选过](http://lifehacker.com/ubersicht-customizes-your-os-x-desktop-with-widgets-and-1591137635)
*   [字时钟小部件](https://github.com/wrutkowski/ubericht-wordclock) 为中心时间
*   右上角的系统监视器的 [内存条小部件](https://github.com/cobyism/ubersicht-memory-bar) 和 [磁盘使用条](https://github.com/dinever/DiskUsage-Bar-Widget)
*   [负载](https://github.com/ttscoff/ubersicht-widgets/tree/master/load) 和 [负载图表右下角为 CPU 负载的](https://github.com/ttscoff/ubersicht-widgets/tree/master/loadchart) 小部件
*   左下角的 [播放框](https://github.com/Pe8er/uWidgets-Distribution/tree/master/Playbox) 控件显示正在播放的信息

快速和肮脏，虽然你需要做一些调整和移动部件，让一切看起来一模一样。如果你有问题，或者就像 Ben 在这里做的那样，一定要去他的 Kinja 博客(链接如下)让他知道你的想法！
