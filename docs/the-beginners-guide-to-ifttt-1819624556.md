# IFTTT 初学者指南

> 原文：<https://lifehacker.com/the-beginners-guide-to-ifttt-1819624556>

*在我们的新系列* [获得 It](https://lifehacker.com/tag/getting-it) *中，我们将为您提供您需要知道的所有信息，帮助您开始并擅长各种在线和离线技术。在这里，我们将为您提供掌握 IFTTT 世界所需的一切知识。*



如果你回想一下你高中的几何课，你可能会记得在解决数学证明时使用了“如果/那么”语句。但是，即使你不喜欢数学，你实际上每天都在使用 If/Then 语句:“如果外面在下雨，那我就带把伞。”或者，“如果我吃了这个甜甜圈，那么我会非常内疚——嗯，开心。”

IFTTT 网站将 If/Then 语句的逻辑运用到了当今高度互联的世界中。它发音与“gift”押韵，代表“如果这个，那么那个”通过 IFTTT 网站 和附带的应用程序，用户能够创建逻辑语句，帮助他们将互联网服务和设备配对，使生活变得更轻松——或更有趣。

以下是你需要知道的“获得它”的方法(注意:出于本文的目的，我们将引用基于浏览器的 IFTTT 版本，但是几乎所有的信息都可以应用于该应用程序。)

### **入门**

像几乎所有其他基于互联网的服务一样，IFTTT 要求你创建一个帐户才能开始使用它。注册非常简单，因为服务完全免费，所以不需要提供信用卡。只需访问网站，使用您的电子邮件地址或谷歌或脸书帐户注册。

### **小程序**

IFTTT 能做的一切都是通过小程序完成的。这些是你可以创建的小程序，它们使用触发器(“如果”)来执行动作(“那么就是”)。例如，一个小程序可能是:“每次纽约时报在其科学版发表一篇新文章时，给我发一封电子邮件。”

重要的是要认识到，只有与 IFTTT 连接的公司才能通过小程序访问。例如，你将能够访问控制 Nest 恒温器(以及其他几个恒温器)的小程序，但如果你有一个 Vivint 恒温器，你就要倒霉了，因为该公司没有通过 IFTTT 提供其服务。

因为 IFTTT 自 2011 年以来一直存在，有许多预制的小程序可供选择，它们已经链接了触发器和操作，所以您很有可能能够找到您需要的东西，而不必创建自己的小程序。首先我们会告诉你如何做，然后我们会告诉你如何从头开始。

### **寻找预先制作的小程序**

IFTTT 允许用户以几种方式搜索小程序。

如果你知道你要找什么，只需点击屏幕左上角的搜索文本，在搜索栏中输入你的主题，然后按回车键。例如，如果你正在寻找一个可以在日落时自动打开飞利浦 Hue 灯泡的小程序，在搜索栏中输入“Hue”。您将被带到一个页面，该页面首先列出可用于构建小程序的服务，在该页面下，您将看到预先存在的小程序的列表。在这种情况下，我们想要的显示在列表的开始，所以你所要做的就是点击它，然后点击“打开”滑块。

这将带您到另一个页面，解释需要激活哪些服务来使小程序工作。在本例中，是 Weather Underground 和飞利浦 Hue。单击 OK 按钮，您将逐步完成激活 applet 所需服务的必要步骤。对于这个例子，你需要告诉 Weather Underground 你住在哪里，然后登录你的 Philips Hue 账户。

一旦您完成了所需的步骤，您的小程序将被激活，您可以开始享受一个新的互联水平。

### **浏览**

另一种查找 IFTTT 小程序的方法是点击屏幕顶部的“发现”链接。这将为您提供一个基于您已经激活的小程序的建议列表。或者，你会注意到，当你点击屏幕顶部的“搜索”时，你会看到一个方便的网格，向你显示各种类别。点击一个类别，你会被带到一个页面，显示按流派分组的小程序。

例如，点击“联网汽车”，你会发现小程序可以做一切事情，从在谷歌电子表格中跟踪你的汽车行程的到帮助你记住你把车停在哪里的。在“天气”类别下，你可以得到一个通知 [当当地冲浪汹涌](https://ifttt.com/applets/165110p-get-a-notification-when-the-surf-s-up-at-your-local-break) 或者如果 [明天](https://ifttt.com/applets/MyGGAxEu-get-an-email-if-there-will-be-rain-in-your-area-tomorrow) 要下雨。与此同时,“社交网络”类别让你尝试一些巧妙的交叉链接，比如 [将你在脸书上被标记的照片保存到谷歌照片](https://ifttt.com/applets/c5GhgcQw-save-new-photos-you-re-tagged-in-on-facebook-to-google-photos) 。

### **从头开始创建小程序**

如果您找不到适合您需求的已有 applet，只要您想要访问的公司与 IFTTT 有协议，您就可以创建一个。首先，单击屏幕顶部的“我的小程序”，然后单击右上角的“新建小程序”按钮。

然后你会被带到一个屏幕，屏幕上有“if +this then that”字样。有点违反直觉，你需要点击文本的蓝色“+this”部分。

在这种情况下，我们将创建一个带有每日触发器的简单 applet。要设置它，点击日期和时间，你会看到一个描述服务的屏幕。选择您希望何时激活触发器(每小时、每天、每周等)，然后单击“创建触发器”

一旦您的服务被连接，您将被带回到显示大文本版本“if this then that”的屏幕，只是这一次，“+this”部分将被您刚才选择的触发器的图形所取代。

接下来，是时候选择您希望由触发器激活的动作了。要做到这一点，点击“+that”，它现在是蓝色的。

在这个简单的例子中，选择 SMS，然后点击绿色方框“给我发一条 SMS”然后，您可以自定义小程序，使其按照您在触发器中指定的时间间隔向您发送消息。单击“创建操作”,您的小程序将被创建。如果您想让 IFTTT 在您的 applet 每次运行时给您发送一条消息，您可以单击“当这个 Applet 运行时接收通知”旁边的滑块然后，点击“完成”,你就一切就绪了。

### **打开和关闭小程序**

激活一个或多个小程序后，您可以根据需要打开或关闭它们。只需点击屏幕顶部的“我的小程序”文本，点击您感兴趣的小程序，然后点击“开”按钮将其关闭。

在这些屏幕上，您还可以对您创建的任何小程序进行更改。为此，单击 applet 框右上角的齿轮图标，您将看到 applet 定制的选项。

在任何时候，点击屏幕左上角的 IFTTT 标志，即可返回主页。

你很快就会发现，你可以为任何事情创建小程序，从 [让自己摆脱尴尬的处境](https://ifttt.com/applets/DEz9XLCh-get-yourself-out-of-an-awkward-situation-international?s=era2) 到当国际空间站经过头顶 时发现 [。但是要注意:如果你在 IFTTT 网站上花了太多时间，你可能会想创建一个小程序把你踢出去，因为所有这些 if TTT 都会让人上瘾！](https://ifttt.com/applets/YfkYtQB2-get-a-notification-when-the-international-space-station-passes-over-your-house?s=era2)
