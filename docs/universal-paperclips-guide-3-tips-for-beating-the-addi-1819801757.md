# 通用回形针指南:击败令人上瘾的新点击游戏的 3 个技巧

> 原文：<https://lifehacker.com/universal-paperclips-guide-3-tips-for-beating-the-addi-1819801757>

今年最好的视频游戏可能就是 [通用回形针](http://www.decisionproblem.com/paperclips/index2.html) ，一款基于浏览器的点击器，探索未经检查的 [人工智能](https://lifehacker.com/what-makes-an-artificial-intelligence-racist-and-sexist-1796990621) 的危险。这有点像 [FarmVille](https://lifehacker.com/how-the-sunk-cost-fallacy-feeds-farmville-addiction-and-30820219) ，但是你不是在种菜，而是在做回形针——而且(**剧透提醒**)你可能最终会在这个过程中毁灭世界。

享受通用回形针的最好方式是让它冲刷你，但是如果你做错了，你可能会大大减慢游戏速度，甚至卡住。所以这里有一些小贴士可以帮助你到达终点。(**警告:前方会有更多游戏剧情剧透**)。

### 1.首先专注于提高你的记忆能力

通用回形针开始非常简单:你是一个任务是制造回形针的人工智能，你可以通过点击一个按钮来完成。但是，随着操作的增加，您可以自动执行这一过程，回形针就会开始大量出现。

制造足够多的回形针，你的人工操作者将奖励你“信任”，你可以用它来交换处理器(你建立操作的速度)或内存(你的最大操作水平)。保持两者的平衡是很诱人的，但是你真的应该专注于记忆，至少在开始的时候。

增加更多的内存是解锁项目的关键，这将加快回形针的制作过程，以及以后更复杂的计划。有了 70 的内存，你可以释放大量的催眠物质(是的，说真的)来增加对回形针的需求。然而，不要止步于此，因为你将需要更多的内存来扩展到外太空，寻找更多的材料来制作更多的回形针。

一旦你达到 250 内存，你就基本上已经饱和了。你可以继续，但你永远不会需要更多的内存。因此，在这一点上，请将您的注意力转移到处理器上。

### 2.量子计算如何工作

你会很早就解开量子计算，但这个游戏并没有很好地解释它是如何工作的。不过，这很简单，值得一试。

在通用回形针中，量子计算让你加快运算速度，甚至暂时突破你的记忆极限。如果您只是缺少一个大的新升级，并且不想等待更多的信任，这将非常有用。

为了利用该功能，请等到量子计算盒中的所有黑色和灰色盒子消失，然后一次性返回。当这种情况发生时，点击计算按钮，然后继续点击它，直到数字下降到尽可能接近零。

差不多就是这样。只是小心不要计算到负面因素。

### 3.如何获得荣誉

荣誉是游戏中引入的最后一个资源，在你专注于制作回形针已经达到荒谬的程度很久之后。你会在仪表板上看到新的提示，要求荣誉解锁某些升级。那么怎么得到呢？

只要继续扩大你的太空探索舰队，最终一个新的盒子会弹出来，你会看到黑白点碰撞和战斗。这就是你获得荣誉的方式。

你会想继续提升你的舰队，这样它就可以打败敌人。然而，通过完成其他项目，你会更快地获得更多荣誉，所以专注于此，你应该很快就能完成游戏。
