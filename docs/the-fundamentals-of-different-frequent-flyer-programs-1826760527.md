# 不同常旅客计划的基础

> 原文：<https://lifehacker.com/the-fundamentals-of-different-frequent-flyer-programs-1826760527>

跟踪不同航空公司的常旅客计划以获得最佳交易可能是一份全职工作(事实上，对少数旅游博客来说也是如此)。规则在不断变化，积分会过期，累积里程不会像以前那样多。



但是为了全面了解美国主要航空公司的计划，这份《纽约时报》 [节俭旅行者分类表](https://www.nytimes.com/2018/06/12/travel/frequent-flier-miles-airlines-tips.html) 有益地概述了主要的差异。

### 航空联盟

你可以注册任意多的飞行常客计划，因为大多数都是免费的，所以主动这样做没有坏处。

但是也要记住国际航空联盟 的存在，它会给你带来更多的回报。航空公司之间的联盟对双方都有利:航空公司可以将乘客提供给合作的航空公司，而不是竞争对手，乘客的特权——如兑换里程、坐商务舱或使用休息室——通常可以无缝转移。就你们美国的大航空公司来说，美联航属于星空联盟，美航属于一个世界，达美属于天合联盟。如果你要去这个国家或世界的不同地方旅行，这些联盟将证明是有益的。

### 美国航空公司

美国航空公司的 [高级计划](https://www.aa.com/i18n/aadvantage-program/aadvantage-program.jsp) 要求旅客飞行 25，000 有效里数并花费 3，000 有效美元才能达到其基本精英级别，即黄金级，该级别还附带免费托运行李和每花费 1 美元可获得 7 英里的额外奖励。

其最高奖励级别是行政白金卡，需要 100，000 有效里数和 12，000 有效美元，并将为您提供免费托运行李和优先登机服务。

### 三角洲

达美航空的 [天合里程计划](https://www.delta.com/content/www/en_US/skymiles.html) 给予基本会员在达美航空航班上每花费一美元可获得五英里里程，并且从那里开始增加。钻石奖章身份是最专属的，每花费 美元，您就能获得 [11 英里里程，但您需要 125，000 有效里程和 15，000 美元有效美元才能达到。](https://www.delta.com/content/www/en_US/skymiles/how-to-earn-miles/earn-with-delta.html#medal)

如果您的旅行里程达到 25，000 英里，消费金额达到 3，000 美元，您将获得银卡，这是高于普通卡的等级。您将获得一个免费托运包和偶尔的升级。

达美航空计划的一大好处是，你的天空里程永不过期，但《纽约时报》注意到，达美航空已经多次在没有警告的情况下将其积分贬值。所以这是需要记住的事情。

### 联合航空公司

美国联合航空公司 [MileagePlus](https://www.united.com/web/en-US/content/mileageplus/Default.aspx) 计划需要 25，000 合格里程和 3，000 合格美元才能达到其最低级别，即银牌，这将为您提供免费托运行李和经济舱座位。要获得最高级别的 1K 会员资格，您需要 100，000 有效里数和 12，000 有效美元。

金卡会员资格需要 50，000 有效里数和 6，000 有效美元，您将有资格获得星空联盟金卡会员资格，从而可以进入世界各地的候机室。
