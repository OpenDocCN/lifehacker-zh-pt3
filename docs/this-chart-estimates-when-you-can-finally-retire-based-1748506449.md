# 根据你的储蓄率，这张图表估计了你最终可以退休的时间

> 原文:[https://life hacker . com/this-chart-estimates-when you-can finally-retire-based-1748506449](https://lifehacker.com/this-chart-estimates-when-you-can-finally-retire-based-1748506449)

三个因素会影响你何时经济独立:你开始储蓄的年龄，你储蓄了多少收入，以及你的投资回报。这张图表说明了这些对你何时退休有多重要。

“热图”是由激进的个人理财评论员菲利普·弗雷创建的谷歌电子表格。基于保守的 4%回报率，它会告诉你是否可以在 40 岁(绿色单元格)、50 岁(黄色)、65 岁(红色)或 65 岁以上(灰色)退休。

据 [我的钱博客](http://www.mymoneyblog.com/financial-independence-heat-map.html) :

> 我相信这些假设包括(1)收入和支出数字都是税后的，(2)一旦你达到支出的 25 倍，你就达到了财务独立(4%的安全支出率)，以及(3)外部养老金和社会保障被忽略。它并不完美，我也不会从这张图表中得出硬性数字，但它仍然是一个简洁的可视化。

它还假设，自从你开始投资以来，你一直在储蓄相同比例的收入。其他在线工具可以给你更好的估计和计划你的退休，但这张图表强调了一个重要的教训: [你的储蓄率](http://lifehacker.com/how-much-you-should-save-for-retirement-based-on-139-y-1054697169) 真的是你在经济独立时唯一可以控制的东西。(为了进行不同的估计，这里有一个关于 [的经验法则，如果你想在 67 岁退休，你的退休账户](http://lifehacker.com/how-much-you-should-have-saved-in-your-retirement-accou-1663712527) 里应该有多少钱。)

这是图表。如果你愿意，可以复制电子表格，将 4%的利率改为其他利率，以获得不同的结果。
