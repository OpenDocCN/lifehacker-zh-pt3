# 使用“Airbnb 游泳池”租一天私人游泳池

> 原文：<https://lifehacker.com/rent-a-private-pool-for-the-day-with-this-airbnb-for-po-1828602385>

我喜欢游泳，但我讨厌公共游泳池。我并不真的反对与人共用一个游泳池的概念，更反对它的执行，在那里你意外地在一群年轻人玩的一场激烈的游戏中结束，或者只是有点太靠近那个穿着泳裤的老绅士而不舒服。我想我只是不喜欢拥挤的公共泳池。我喜欢我的空间。



这个周末，我偶然发现了一种方法来实现我作家预算中的私人泳池梦:。这是一种类似于 Airbnb 的泳池服务，你可以在下午租用别人的泳池。

对我来说不幸的是，在旧金山的服务上没有列出任何游泳池(人们在旧金山拥有游泳池吗？)，但其他地方有很多。例如，搜索下周五在洛杉矶的一个游泳池，会出现几个每小时 50 美元的选项。

一个游泳池可以容纳八个人和我一起游泳，还提供一个热水浴缸。另一家有一个跳水板和一个热水浴缸，并允许我带 15 个朋友一起预订。

当你考虑一个人的私人游泳时，50 美元/小时是相当昂贵的。如果你有一个四口之家或者一群朋友，那么这个价格就开始接近你去公共游泳池的价格(或者更低)，除了可能更好。
