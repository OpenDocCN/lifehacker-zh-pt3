# Lifehacker 员工非常正确的笔谈

> 原文：<https://lifehacker.com/the-lifehacker-staffs-very-correct-pen-opinions-1826497301>

在丑闻曝光后，环境保护局局长斯科特·普鲁伊特用纳税人的钱购买了价值 1560 美元的钢笔 ，这让我们思考:什么是最好的钢笔？(他也买了个性化期刊，但那是以后的话题。)



“从 DC 一家名为小首饰盒的高档珠宝店定制的银色自来水笔”肯定不是市场上最好的。不是吹牛，但我以前用过自来水笔，你猜怎么着:它们很好，不是很好。我不会花 130 美元买一个，即使它“印有环境保护署的印章”和普鲁特的签名。正如职业作家尼克·道格拉斯所说，“除非你是某种怪人，否则你最喜欢的钢笔大约一美元，你可以买一吨，永远不会吝啬或使用你不喜欢的钢笔。”

谈到钢笔，每个人都有自己的看法(或者员工乔尔·卡恩不以为然地称之为“o-pen-ion”)。不同的笔适合不同类型的人。我的字迹潦草得令人难以置信，所以我需要一支不要流太多血也不要太墨迹的笔——它只会让我糟糕的字迹看起来更糟糕。然而，其他员工表达了完全相反的情绪。

所以，这里是每个 LH 员工的首选钢笔(H/t to Vice 的 Eve Peyser，他用这款智能 [钢笔小齿轮](https://www.vice.com/en_us/article/bj3dj4/scott-pruitt-spent-1560-on-pens-and-they-werent-even-good-pens) 启发了我们):

### 帕特里克·艾伦，专职作家

我喜欢钢笔。我最喜欢的平价钢笔是 Pilot G-2s 0.7mm，我不喜欢带帽的廉价钢笔。

*帕特里克的额外想法:热爱无印良品。一些我最喜欢的记事本和自动铅笔。我也喜欢一支好的毛笔。非常喜欢。

### 尼克·道格拉斯，专职作家

我喜欢帕特里克喜欢的完全一样的飞行员 G-2 0.7 毫米。双胞胎！有时候我会穿蓝色的，因为生活是一场派对。

### 艾丽西娅·亚当奇克，专职作家

不要想多了。Paper Mate ComfortMate Ultra 1.0M 是一款非常棒的日常用笔。我也喜欢夏皮笔，但是它们会在我的日记里流血，所以我必须用更轻的东西。不过，对于厚的股票来说，夏皮笔是最好的选择。我也喜欢用蓝绿色的 [Stabilo point 88](https://lifehacker.com/create-a-not-to-do-list-1824175532) 写日志。

### 米歇尔·吴，育儿编辑

我总是试图将更多的笔记塞进小笔记本，无印良品中性笔圆珠笔非常适合写极小的字母。他们充满活力，但他们不会流血或涂抹。获得 0.5 毫米或 0.38 毫米

### 克莱尔·洛，美食编辑

我非常喜欢 Stabilo 圆珠笔，用于日常书写。

*克莱尔给你的额外建议:我留着蹩脚的钢笔给我的男朋友，他想偷我的好钢笔。

### Beth Skwarecki，健康编辑

Uni-Ball Air 是最好的钢笔，因为它可以在任何角度书写一条厚厚的光滑的墨水河，并且你可以(小心地)用 Noodler 的防弹黑色重新填充它。如果我想要一支自来水笔，拉米野生动物园是完美的；如果我想要一支可伸缩的中性笔，我会选择 PaperMate Ink Joy。

### 乔尔·卡恩，视频编辑

我喜欢一支墨水笔。感觉你在液体上写字。 [单球](https://www.boxed.com/product/929/uni-ball-vision-elite-roller-pens-10-assorted-pens-bold?pf=3&gid=929&utm_source=google&utm_medium=cpc&utm_purpose=acquisition&utm_campaign=shopping-lowbid&gclid=CjwKCAjw3cPYBRB7EiwAsrc-uWw12M6I_vq5apQu2fB_ZkABnVBOoV0PHsD3hdzD85ZePwcy732lyRoCT7wQAvD_BwE) 很棒，因为它们很结实，有非常令人满意的咔嗒声(copyright me，2018)，而且它们看起来比实际上更花哨。我也总是带着一支笔，并且一直带着。你永远不知道什么时候你会碰到一个名人并想要他们的签名，或者当收银员找不到笔给你时，还好我手边总有一支！因此，clicky 笔不起作用，因为它们可能会在你的口袋里咔哒一声打开，毁了你的裤子。

*乔尔给你的额外建议:把你不喜欢的笔借给那些不愿归还的人。那你丢了一支好笔也不心疼！

### **大卫·墨菲，高级技术编辑**

Yassss 对于单球视觉，我不会使用其他任何东西(微型，不精细)。

### 爱丽丝·布拉德利，副主编

这些是最好的:单球 Signo UM-151。
