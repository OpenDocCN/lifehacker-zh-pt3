# 我用来测试新显卡的前五款电脑游戏

> 原文:[https://kotaku . com/the-first-five-games-I-use-to-test-a-new-graphics card-1783206158](https://kotaku.com/the-first-five-games-i-use-to-test-a-new-graphics-card-1783206158)

首先，你在新显卡上花费太多。然后，你启动你最有嚼劲、最难玩的电脑游戏，让他们求饶。有那么一瞬间，你的高价购买感觉完全合理。

Watch

上周，我买了一台新的顶级 GTX 1080，我告诉自己现在就应该买，因为我还可以卖掉我最后的显卡(980Ti ),足以在一定程度上抵消这笔费用。我也只是想要一个。(迈克在这里为我们回顾了。)

每当我安装一个新的电脑组件，特别是一个新的 GPU，我都会先检查一个游戏列表。哪些游戏一直给我添麻烦？哪一个最能证明我的购买是合理的？我能让屏幕角落的小帧率计数器爬多高？

### 1.巫师 3

当然，我必须先玩《巫师 3》的游戏。这不仅是我最喜欢的游戏之一，也是我拥有的最好看的游戏之一。它在 1440p 中已经运行得很好了，但是我想让它运行得更快*呃*。的确如此。哦，是的。我会说杰洛特比以前帅了 10%。我不太确定如何对其进行基准测试。

### 2.古墓丽影的崛起

在最好的情况下， *RotTR* 和我安装的其他东西一样漂亮。这也是一个让我的上一个显卡物有所值的游戏。我打过两次(！！)这似乎比我预计的玩它的时间要长，所以我让它运行了足够长的时间，以绕着地热谷跑，并确定我确实获得了良好的帧速率。前进！

### 3.杀手

总有那么一场比赛，不是吗？这是一个运行得不好的游戏，你认为这是游戏没有优化好，但你永远不能确定。我真的很喜欢新的杀手，但它从来没有运行得很好。我已经尝试了一切(认真地说，一切)，我不能保持在 60fps 以上。

你猜怎么着？在我的 GTX 1080 上，我仍然无法保持 60fps 以上的速度。(我知道这是一个 CPU 猪，但如果一个 4.4Ghz 的 i7 跟不上，肯定有问题。)如果你对你的杀手表现感到沮丧，担心你的电脑是问题所在，不要这样。我要大胆地说，这个游戏优化得太差了。如果你足够幸运，拥有一台可以完美运行游戏的电脑，那就太好了。

### 4.孤岛危机 3

啊，*孤岛危机 3* 。一个我从来没有真正关心过的游戏，但是我一直留在这里，专门用来玩新显卡。我真的不需要玩很长时间，我只是想看看《惊魂记》的美丽，麻脸，然后去窒息一些愚蠢的细胞代理人。这个游戏在某些方面显示出它的年龄，但我的 980Ti 实际上在 1440p 时遇到了一点麻烦。看到一个新的 GPU 把它嚼碎再吐出来，很欣慰。也许我的下一张卡会用 4k 运行它。

### 5.精英:危险(VR)

这对我来说是新的。在过去，我可能会把*阿卡姆骑士*或*只是原因 3* 放在这里。但现在我实际上正在积极地玩 VR 游戏，我想确保它们运行得很好，很流畅。 *Elite: Dangerous* 在 VR 中总体来说确实不错，但我的 PC 一直难以保持 90fps。我想，直到现在。1080 终于能够以约翰·卡马克想要的 VR 帧率玩这个游戏了，它看起来确实非常好。
