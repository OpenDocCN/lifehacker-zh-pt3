# 在线备份对抗:CrashPlan 与 Backblaze

> 原文：<https://lifehacker.com/online-backup-faceoff-crashplan-vs-backblaze-1768101631>

您不想考虑备份:您希望它们在后台自动发生在您所有的计算机上。您希望从任何位置、任何设备、任何操作系统访问这些备份。为此有两个优秀的服务: [CrashPlan](http://www.code42.com/crashplan/) 和 [Backblaze](https://www.backblaze.com) 。两者都很受欢迎，我们过去都推荐过。我们来对比一下。

Watch

正如我们之前说过的 ，没有理由不备份你的电脑。有一天你的硬盘会出故障，当它出故障时，你不想丢失所有的数据。您可以将系统备份到本地驱动器，但是出于冗余的考虑，您还应该将它备份到异地位置。这就是在线备份解决方案的用武之地。 [虽然你在这个领域有很多选择](http://lifehacker.com/five-best-online-backup-services-1006345049) ，CrashPlan 和 Backblaze 是两个最有声望和最知名的服务。

### 竞争者

当你在寻找一个基于在线的存储解决方案时，要考虑的两个要点是可靠性和 [加密](https://lifehacker.com/a-beginners-guide-to-encryption-what-it-is-and-how-to-1508196946) 。异地备份本质上是文件的保险政策，你最不想做的事情就是将它们交给一些没有历史或劣质加密的初创公司。CrashPlan 和 Backblaze 都有着坚实的历史，已经存在了一段时间，并赢得了独立评论者的好评。

*   [**crash plan**](http://www.code42.com/crashplan/)归 Code42 所有，从 2007 年就有了。CrashPlan 支持 Windows、Mac 和 Linux。 [CrashPlan 早就是我们的](http://lifehacker.com/set-up-an-automated-bulletproof-file-back-up-solution-5787572) ( [和你的](http://lifehacker.com/most-popular-online-backup-service-crashplan-1039674096) ) go-to 备份方案。CrashPlan [在这里](https://support.code42.com/CrashPlan/4/Backup/How_Backup_Works) 解释了它的备份过程，但是它非常简单:它抓取一个新文件，压缩它，加密它，然后把它发送到备份目的地。你的 [数据从头到尾都是用 448 位加密](http://www.code42.com/crashplan/features/security/) 加密的，它在静止状态下是加密的——只有再次下载时才解密。CrashPlan 将您的数据存储在位于世界各地的安全数据中心。

*   [**Backblaze**](https://www.backblaze.com)只支持 Windows 和 Mac，从 2008 年就有了。虽然 Backblaze 的 128 位加密不如 CrashPlan 的 448 位加密强大，但对大多数用户来说应该足够了。如果你感兴趣，在 Reddit 上的 [问我任何问题主题中，Backblaze 回答了一些关于他们的服务器和加密过程的问题。Backblaze 在各种安全数据中心的多个驱动器上冗余存储您的数据，因此您的数据应该得到很好的保护。](http://www.reddit.com/r/IAmA/comments/rhrt4/we_are_the_team_that_runs_online_backup_service) 

因此，CrashPlan 和 Backblaze 都有良好的记录，并支持你期望从在线备份服务中获得的加密类型。下面就来深究一下是什么让他们彼此不同。

### 基本要素:存储空间和成本

首先，CrashPlan 和 Backblaze 都提供无限的数据存储，因此您可以备份任意多的数据。CrashPlan 是一人一台电脑 59.99 美元/年。Backblaze 是一人一台电脑 50 美元/年。除此之外，定价就有点奇怪了。

CrashPlan 在其在线备份服务旁边还有两个免费选项:本地备份和异地(或计算机到计算机)备份。顾名思义，本地备份允许您备份到您的个人外部硬盘。异地备份允许您将您的计算机备份到不同位置的其他计算机上，如家人或朋友的家中。如果您可以利用异地备份功能，您基本上可以跳过付费在线备份，仍然有很好的数据冗余。说到家庭成员，CrashPlan 还提供了一个 149.99 美元/年的家庭计划，可以为多达 10 台计算机提供无限的备份存储。如果你家里有很多电脑，这可以帮你省下一大笔更换 Backblaze 的费用。

Backblaze 没有免费的本地备份选项(虽然， [考虑到 PC 和 MAC 都有内置选项，](http://lifehacker.com/how-to-back-up-your-computer-to-an-external-drive-5816453) 可能没有必要)，也没有 CrashPlan 的异地备份能力。Backblaze 也没有家庭计划，所以最适合个人用户。你可以在一个帐户上用 Backblaze 备份多台电脑，但你必须为每台电脑每年支付 50 美元。

获胜者:平局。这个很大程度上取决于你想要什么，你需要什么，你拥有什么。如果你只有一台计算机，并且你想备份它，那么 Backblaze 是你最好的选择，每年 50 美元。然而，如果您需要备份多台计算机，您可以访问异地计算机，或者如果您对家庭计划感兴趣，那么 CrashPlan 每年额外的 10 美元是非常值得的。

### 易于使用和设置备份

CrashPlan 和 Backblaze 并不特别难用，但是就界面而言，Backblaze 更直观一些。当您启动 Backblaze 时，您不需要调整任何设置。安装后，它会自动开始备份您的硬盘。只要您有互联网连接，Backblaze 就会在后台持续备份文件。这并不是说 Backblaze 没有选择。你可以调整它备份的频率，调节上传速度使其不占用带宽，当你只靠电池供电时关闭备份，等等。或者不是。Backblaze 在大多数电脑上的默认设置下运行良好。

CrashPlan 有各种各样笨拙命名的设置和一个用 Java 构建而不是本机运行的应用程序。随之而来的是稍微不太直观的备份过程。当你第一次启动 CrashPlan 时，迎接你的是一大堆选项。你可以通过点击大的“立即备份”按钮开始备份过程，但是 CrashPlan 没有 Backblaze 那么明显。相反，CrashPlan 卸载了一个噩梦般的复选框、选项和菜单，它们没有清晰的功能描述。CrashPlan 的选项确实允许你调整更新发生的时间、发生的频率、CrashPlan 可以使用的 CPU 能力、备份的位置等等。这些都很有用，但是如果你只是想备份你的数据，这就有点让人不知所措了。

顺便说一下，这些设置很有必要进行调整。由于 CrashPlan 运行在 Java 上，而不是作为一个本地应用程序，它有点占用资源。CrashPlan 多年来一直承诺推出一款原生应用，但一直没有实现。最终的结果是，对于普通用户来说，这款应用太过笨重。当它运行时，你会看到你的电脑性能下降，除非你进入 [调整多少 CPU 崩溃计划可以使用](https://lifehacker.com/speed-up-crashplan-backups-and-free-up-cpu-power-with-t-1658888517) 。我第一次运行 CrashPlan 的时候，在我意识到发生了什么之前，我真的以为我的电脑彻底失败了。对于最终会找到 CPU 设置的更高级的用户来说，这不是一件大事，但是对于那些不喜欢调整设置的人来说，这有点令人不快。

**获胜者:Backblaze** 。一旦你安装了 Backblaze，你不需要做任何其他事情来启动备份过程，默认设置对大多数用户来说是好的。Backblaze 完全符合备份软件的“一劳永逸”原则，CrashPlan 需要更多的实践方法。

### 易于文件恢复和多种恢复选项

备份很重要，但这只是好的备份软件的第一考虑。恢复过程同样重要，如果不能轻松恢复，平滑备份是没有用的。虽然 CrashPlan 可能比 Backblaze 更难设置和运行备份，但在恢复时，CrashPlan 实际上更容易使用。

在 CrashPlan 应用程序中，你会发现一个“恢复”标签。点击它，你可以抓取单个文件、文件夹，或者一次恢复所有内容。超级简单。如果您正在恢复整个硬盘驱动器，CrashPlan 会为您提供调整权限和文件位置所需的设置，以便您可以继续工作。完成后，您的文件就在您离开的地方，在您期望的目录和文件夹结构中，就在您的计算机上。

Backblaze 的恢复功能有点奇怪。当你点击应用程序中的恢复按钮时，你会被踢到 web 界面，在那里你可以选择你想要的文件和文件夹或整个备份。在那里，您会收到一个可下载 ZIP 文件的链接。你也可以通过邮件获取硬盘或闪存盘上的文件，但需要额外付费。因为 Backblaze 依赖于 web 界面，所以它最终会比 CrashPlan 更难使用。

**获胜者:崩溃计划**。恢复过程发生在 CrashPlan 应用程序中，因此很容易理解，您可以找到文件。恢复快速、直观，你可以像从 Dropbox 这样的服务下载文件一样下载文件。虽然 Backblaze 的 ZIP 格式并非无法导航，但它非常耗时，而且远没有那么直观。

### 将它们区分开来的特征

CrashPlan 和 Backblaze 共享大量标准功能。它们都提供无限存储、连续备份、用于增加安全性的个人加密密钥、用于调出旧版本保存文件的无限版本、用于访问备份文件的移动应用以及带宽控制。简而言之，这两种服务都提供了普通人需要的一切。但是它们都有一些值得考虑的专有选项，尤其是如果你是一个高级用户。我们已经在上面讨论了其中的一些功能，但是让我们把它们都放在一个易于浏览的地方。

**坠毁计划**

*   家庭/团体计划选项
*   免费支持外部驱动器备份和备份到个人异地计算机
*   无限期保存文件的已删除/旧版本(Backblaze 保存四周)
*   就地文件恢复
*   备份频率为每分钟一次

**Backblaze**

*   通过邮件恢复选项(在邮件中给您发送一个物理驱动器)，用 [退款程序](https://www.backblaze.com/blog/introducing-the-restore-return-refund-program/)
*   Windows 和 Mac 的原生应用程序使应用程序不那么需要资源(相对于 CrashPlan 的 Java 应用程序)
*   默认设置应该适用于大多数用户
*   如果您的电脑被盗，请使用“查找我的电脑”功能找到它

当然，如果这些特性对你很重要，那么你的选择就很清楚了。

### 底线是:CrashPlan 最适合高级用户，Backblaze 适合其他人

CrashPlan 和 Backblaze 都很棒，都做了它们应该做的事情。但是普遍推荐一个而不是另一个是困难的。就我个人而言，我喜欢 CrashPlan，因为它可以永远保存删除的文件，并且有大量的设置可以修改，所以它会按照我想要的方式工作。所以，CrashPlan 对于喜欢修补的高级用户来说很棒。CrashPlan 的家庭计划也是任何计划备份大量计算机或想使用朋友的计算机创建自己的私人小型备份网络的人的必备计划。

但是我会向父母推荐 CrashPlan 吗？不:我推荐 Backblaze。Backblaze 开箱即用，无需修补。只需几分钟，只需很少的点击，备份过程就开始了，此后，在需要恢复之前，您再也不用考虑它了。原生应用使 Backblaze 更容易使用，并且比 CrashPlan 的 Java 应用消耗更少的资源。Backblaze 不仅更便宜，还更容易设置，这对于只需要备份单台计算机的普通用户来说是一个不错的选择。
