# 感恩节我们应该储备什么？

> 原文:[https://life hacker . com/what-should-we-sous-vide-for-thanksgivity-1819974601](https://lifehacker.com/what-should-we-sous-vide-for-thanksgiving-1819974601)

各位网友大家好，欢迎回到 [*的另一个选题环节。*](http://lifehacker.com/tag/will-it-sous-vide) ，列在那里我用我的沉浸循环器做任何你想让我做的事。

虽然十月还没有结束，但火鸡日很快就会到来，我总是发现最好早点开始准备。去年我们供应了 [一整只火鸡](https://skillet.lifehacker.com/will-it-sous-vide-a-whole-thanksgiving-turkey-1789112054) 和 [几只火鸡](https://skillet.lifehacker.com/will-it-sous-vide-glorious-thanksgiving-sides-1789074625) ，但我相信我们可以做得更多。我有一些感恩节主题的想法，包括:

*   **火鸡腿 confit:** 类似于 [这些鸭腿](https://skillet.lifehacker.com/will-it-sous-vide-rich-and-meaty-duck-confit-1794936381) 但是要大得多。
*   你知道，土豆泥，但是有很多奶酪。
*   玉米砂锅:事实上，几个月来我一直试图自己完善它，但还没有成功。
*   不是馅饼的南瓜:也许是芝士蛋糕或奶油布丁？还是两者都放在一个罐子里？
*   **地瓜派:**或者地瓜砂锅。任何有红薯的东西。

像往常一样，请在评论中提出你自己的建议。任何你认为能让你的火鸡日变得更简单、更美味或更有趣的东西都是受欢迎的。
