# 使用哪种钻头

> 原文：<https://jalopnik.com/which-drill-bit-to-use-1787295606>

你知道钻头实际上只有一个任务——钻孔。但是你在上钻孔*的方式决定了你实际需要的钻头类型。观看我最喜欢的加拿大机械师分解你曾经用过的和一些你可能从未见过的每种钻子的种类、内容和原因。*

 [https://jalopnik.com/embed/inset/iframe?id=youtube-video-I7BbrjoftJA&start=0](https://jalopnik.com/embed/inset/iframe?id=youtube-video-I7BbrjoftJA&start=0) 

Jalopnik 的老读者可能会从之前的帖子中认出 YouTuber [AvE](http://thegarage.jalopnik.com/how-to-tell-which-tools-are-cool-and-which-will-burn-yo-1786586095) 。这个家伙是一个传奇——总是一边拆解和解释我们的教育工具，一边编造有趣的脏话。



如果你曾经想知道在给定的材料上钻一个特定尺寸的孔需要哪种钻头，为什么打孔工具有用，为什么不同的钻头有不同的形状，或者钻头工作时有多快多难，这个接头有你的答案。

你的大多数 DIY 钻孔项目可能需要简单的“麻花钻”——当你在脑海中描绘钻头时，你可能会想到标准的麻花钻。要在木头、塑料或较软的金属上打小孔，使用合适的钻头主要是要用合适的材料制造质量好的钻头，并且在钻孔时不要让钻头太热。

其他钻头旨在通过“螺旋钻头”或“自动进料”等材料，这些材料往往会在柔软的材料中留下更干净的出口孔，在制作家具时会派上用场。

这个视频带我们经历了一个从软到硬的“材料层次”。在这种背景下，“最容易钻透的”在这方面，你可能已经明白“钢总是胜过塑料和木头”，但我喜欢 AvE 通过奶酪的种类来解释金属的韧性。

如果黄铜是布里干酪，铝是干酪，钢是切达干酪。显然，工具钢就像在冰箱里放了一周的干酪。

为了穿过“冷冻帕尔马干酪”大道，展出了一些非常厉害的东西，内部冷却通道只能由笨重的固定机器运行。对于你要放入手钻的较小的东西，他建议“中等压力和中等速度”如果你在制造热量而不是一个洞，你压得太紧或者转得太快。

也就是说，让你的钻头转得太慢实际上可能比超频更糟糕，因为在“200 磅的大猩猩推它”(他指的是你)的作用下，它更容易破损。

视频中的关键要点是以正确的速度旋转钻头，并选择一种比你感到厌烦的材料更硬的材料。确保你的东西不会太热。
