# 使用 Fing 应用程序轻松识别网络上的所有设备

> 原文：<https://lifehacker.com/ios-android-app-of-the-week-fing-network-scanner-1825205516>

你的网络上有什么？你总是可以打开你的路由器的网络接口来了解连接了哪些设备，但是你可能只会得到一个 MAC 地址和分配的 IP 的列表——不是很有帮助。应用程序 Fing-Network Scanner([iOS](https://itunes.apple.com/us/app/fing-network-scanner/id430921107?mt=8)， [Android](https://play.google.com/store/apps/details?id=com.overlook.android.fing&hl=en_GB) )是一个很好的免费方法，可以更好地了解路由器必须处理的所有设备。



一旦你安装了它，使用 Fing 就是一件轻而易举的事。打开应用程序。就是这样。一旦您启动它，Fing 将自动开始尝试识别您网络上的所有设备。你会看到他们所有的 IP 地址，他们的名字——或者至少是 Fing 可以识别的任何名字——甚至可能会有一个小图标，让你快速看出哪个设备是智能手机、打印机、智能家居集线器等等。

Fing 还试图找出你连接的设备的品牌和型号——如果你不想让你的设备的 IP 地址和 MAC 地址发送到应用程序的“Fingpedia”服务，你可以随意启用和禁用这种查找技术。(Fing 不要求您创建帐户或关联电子邮件地址来使用此功能。)

如果你点击列表中的任何特定设备，Fing 可以让你添加一个特定的名称，设备的描述，以及一个小纸条来提醒你把它放在了你的房子或公寓的什么地方。比如说，如果你需要记住哪个智能开关是哪个，这就很有用。Fing 还维护了一个小事件日志，记录设备在网络上出现或消失的时间，你可以直接从 Fing 的应用程序 ping 任何设备，看看它是否可以到达。
