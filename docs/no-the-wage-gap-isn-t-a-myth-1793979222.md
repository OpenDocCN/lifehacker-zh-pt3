# 不，工资差距不是一个神话

> 原文：<https://lifehacker.com/no-the-wage-gap-isn-t-a-myth-1793979222>

今天是同工同酬日，这一天致力于让人们意识到工资差距。自 60 年代以来，差距已经缩小，但仍有进步，尤其是考虑到反对者仍然认为差距不存在。



自从 1963 年,同工同酬法案签署成为法律以来，这一差距已经缩小了不少。但是女性， [尤其是有色人种](https://www.americanprogress.org/issues/women/news/2017/04/03/429810/value-equal-pay-u-s-economy/) 的女性，仍然比男性挣得少。2016 年，在控制种族和民族、教育、经验和地点后，女性的薪酬比男性低 22%。这是一个令人沮丧的问题，更令人沮丧的是，许多人认为整件事都不是真的。

不过，我试图理解怀疑论者的观点。根据我读过的破除神话的文章( [就像这篇，](http://time.com/3222543/wage-pay-gap-myth-feminism/) )，主要的脱节似乎是否认工资差距的人简化了问题，然后责怪同工同酬的支持者简化了问题。例如，这里的 [一个错误的、简单化了的](https://www.forbes.com/sites/karinagness/2016/04/12/dont-buy-into-the-gender-pay-gap-myth/) :

> 然而，出于某种原因，这些年轻女性确信，在劳动力市场上，基于性别的歧视将在未来等着她们。仅仅因为她们是女性，她们就要为每份薪水支付 22%的税，这要感谢这个偏袒男性的不公平社会。

我不知道这是一个 [稻草人论点](http://lifehacker.com/use-the-straw-man-tactic-to-ease-the-blow-when-giving-c-1725021276) 还是这个作者真的不明白这个问题。在同一篇文章中，作者承认“教育、工作年限和工作时间等因素会影响收入。”嗯，是的！我们知道工资差距不是那么简单的。然而，在许多情况下，*即使你调整了这些因素*，数字仍然显示出差距。
