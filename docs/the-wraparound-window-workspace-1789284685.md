# 环绕式窗口工作区

> 原文：<https://lifehacker.com/the-wraparound-window-workspace-1789284685>

[【Re _ on】](https://www.reddit.com/user/re_on)有一个华丽的大窗户，可以看到美丽的景色...所以他在它的正前方建立了他的工作室。它很漂亮，简单，尽管没有窗帘，但也是一个不错的工作场所。让我们看看更多的照片。



公平地说，请注意下面链接的 Reddit 帖子中的注释(此处为该评论的永久链接),虽然你可能想知道人们在查看时看到了什么，但你不必这样做:

> 晚上晚上都有窗帘，但这是暂时的情况。我刚刚安装了新的窗户和框架，所以我在考虑下一步该做什么。我有一些额外的日间窗帘，但我需要为它安装一个额外的挂杆。或者这样，或者我将应用一些模糊的床单到窗户的下部，就像我在我的旧窗户上做的那样。尽管如此，仍然需要等待画家尽快完成他的工作。
> 
> 幸运的是，没有人住在街道的另一边。我前面的建筑是一家房地产经纪公司，他们几乎不用二楼，其余的都是企业，他们只用二楼做仓库。

那就放心了！但是除了那个看起来很锋利的 NZXT H440 外壳(我正在考虑我最近的构建)，其余的规格都在下面的帖子中提到，最具体的是整个设置所依赖的 Ahrend 700 桌子，显示器上的 [Abe Kislevitz 壁纸](http://abekislevitz.com/wallpapers-of-the-month/) ，罗技 G100s 鼠标，M-Audio 扬声器，以及整个事情中心的 Realforce 87U 键盘。这是从侧面拍摄的另一张照片:

再来一张，右边有棵友好的植物。

整个设置看起来非常漂亮和简单，当然，它没有昂贵的技术或任何东西，这是一个坐下来，完成一些工作，激发灵感的好地方。如果你也喜欢它，就去下面链接的帖子，并在那里投上一票。让他知道是我们派你去的！
