# 兼职母乳喂养是一个完全有效的选择

> 原文：<https://lifehacker.com/part-time-breastfeeding-is-a-totally-valid-option-1820181813>

如果你打算母乳喂养你的孩子，但最终想要或需要给配方奶，你可能会认为你已经改变了团队:你现在是一个配方奶喂养者。但这不是一个全有或全无的选择，兼职母乳喂养有重要的好处。



根据一项新的研究 ，其中之一 [可能会降低婴儿猝死综合症(SIDS)的风险。母乳喂养两个月或更长时间的婴儿——即使他们*也*吃配方奶粉——比只吃配方奶粉的婴儿更不容易死于婴儿猝死综合症。这项研究是观察性的，我们不能确定研究人员排除了所有的混杂因素。但是研究结果表明母乳喂养可能有保护作用。](https://www.forbes.com/sites/tarahaelle/2017/10/31/any-breastfeeding-even-partial-cuts-sids-risk-in-half/#13ca0e272519)

那么，部分母乳喂养是什么样子的呢？例如，你可以把配方奶粉装在孩子的日托袋里，但是当他们在家的时候直接从水龙头里给他们喂食。或者，即使他们大部分时间都在吃配方奶粉，在睡觉前进行一些护理也没有什么不对。(如果你只是很少母乳喂养，你的身体可能会停止分泌乳汁，所以在你决定做什么的时候要记住这一点。)

显然，如果你因为医生或其他健康专家的建议而改变了孩子的饮食，你可以和他们讨论你的选择。但是我知道很多人认为他们“不得不”完全戒掉，而没有任何真正的理由这样做。如果你想继续母乳喂养，这通常是一个有效的选择。
