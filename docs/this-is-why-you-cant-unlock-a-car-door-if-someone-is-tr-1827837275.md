# 这就是为什么如果有人同时试图打开车门，你就无法打开车门

> 原文:[https://jalopnik . com/this-is-why-you-can-unlock-a-car-door-if-someone-is-tr-1827837275](https://jalopnik.com/this-is-why-you-cant-unlock-a-car-door-if-someone-is-tr-1827837275)

这是所有汽车行业中最常见的共同经历之一:你在车外，试图进入车内。在你拉把手开门的同时，车内的人正试图打开门锁。随之而来的是一场错误的喜剧，你的每一个动作都抵消了另一个动作，让你俩一遍又一遍地拉手柄或锁按钮，陷入了似乎永无止境的机械僵局。最终，有人喊着让某人放手，噩梦结束了。但是为什么一定要这样呢？这是怎么回事？

为什么你不能在开门的时候拉门把手？为什么这样一个汽车的基本机制会有这样一个令人沮丧的缺陷？答案是简单的力学。(除此之外，我们发现，门闩一点也不简单。)

最基本的解释可能是，拉动把手会接合门内的一个小机械部件，这个部件会撞到另一个机械部件，由其他人试图打开门来触发。单独的话，他们中的任何一个都可以不受阻碍的完成他们的工作。同时做这两件事，它们会互相阻碍。

这听起来很容易理解，对吗？你不知道你错得有多离谱。

门锁和插销复杂得荒谬，令人心痛。跟着我们，你会看到的。

为了理解这里发生的事情，我们需要看看车内的门锁是如何工作的。为了做到这一点，我们自己的大卫·特雷西和我从两辆不同的汽车上取下了内门板，它们代表了一系列门机械技术，从古老而简单到更现代和复杂。

简单来说，我已经撕下了我 1973 年大众甲壳虫的门卡，这基本上是 1938 年的设计，门锁系统从那时起就没有太大变化。我知道这个门系统的最后一次主要更新是在 1968 年，我的车上的设置和那个差不多。

### **为什么你不能同时解锁和打开它**

在甲壳虫上，由于支架上的手指和闩锁连杆上的螺纹接头之间的相互作用，您无法从外部拉动把手并同时解锁汽车。

在我们开始之前，让我们看看锁定/解锁机制是如何工作的。

打开这辆甲壳虫的车门需要向下移动支架，或者直接通过外部的锁芯，或者通过内部的锁柱，拉动上图所示的杆。

该杆转动锁连杆，向下推动锁连杆的左侧，从而向下推动整个支架。打开汽车的关键是，随着支架向下，闩锁释放连杆现在位于支架的右侧，因此当您从外部拉动手柄并将支架推向右侧时，闩锁释放连杆会旋转，并最终放开闩锁。

将汽车锁回原位会将支架向上移动到这样一个位置，当从外部拉动把手时，支架将不再接触锁闩释放连杆。所以车一直锁着。

至于为什么锁着的时候不能从里面打开车门:当你从里面或者外面锁车的时候，支架往下走，把锁连杆旋转到一定的位置。试图拉动内部把手会导致锁和把手连杆之间的干涉情况。

好了，现在你知道锁是如何工作的了，想象汽车如上图所示被锁上了，你决定拉把手。支架会向右移动，手指会刚好在乳头上方。解锁汽车(向下移动支架)将是不可能的，因为奶嘴和手指会干扰。

因此，您必须放开门按钮，然后再次尝试解锁。

拉动车门内把手(由右下角蓝色箭头表示)也会限制车门解锁。从上面看，你可以看到当你拉车门内把手时，把手连杆撞上锁连杆并对其施加压力。因此，当您拉动解锁杆(由红色箭头表示)时，它将无法旋转锁连杆，门将保持锁定。

所以你看:在甲壳虫上，在有人试图解锁汽车的同时打开车门是行不通的，因为乳头/手指的相互作用和锁连杆/把手连杆的相互作用。

首先，我们来看看门把手，它使上杆处于张紧状态，逆时针旋转下图所示的“连杆 A”(下图中的杆已被移除)。

链接 A 的这种旋转向上推动“链接 B”的左侧，使其围绕其枢轴点(图片中心的银色圆点)旋转，最终向下推动链接 B 的右侧。(注意:如果你用外侧把手而不是内侧把手开门，你将直接按下蓝色箭头上方那个大洞处连杆 B 的右侧)。

从那里，连杆 B 推动另一个连杆(如上图所示的连杆 C)，然后推动我标记为“连杆 D”的黑色连杆。黑色塑料连杆，连杆 D，进入张紧状态，拉下白色塑料插销释放装置，允许门打开。

### **为什么你不能同时解锁和打开它**

就像车门的开启有两个点(左上方的黄色甜甜圈是内门把手，右黄色甜甜圈正下方前方的大洞是外门把手)，你也可以从车内或车外开锁和锁车，如上图。

在任何一种情况下，锁门都会作用在连杆 D 上，将其推出并远离连杆 C(连杆 D 在下图中向您移动)。

这是它在动画中的样子。观察塑料“链节 D”从链节 C 下方移出:

由于打开门需要拉紧连杆 D，以便拉动白色插销释放装置，将连杆 D 从连杆 C 下方移出意味着拉动门把手将简单地向下移动连杆 C。这不会拉动 D，所以门将保持关闭。

那么，你不能同时打开和解锁车门的原因很简单:一旦你拉了锁着的汽车上的门把手(同样，锁着意味着连杆 D 已经从连杆 C 下面移开)，你就把连杆 C 放在了连杆 D 需要的地方，以便解锁汽车。

换句话说，拉手柄时解锁汽车只会导致连杆 D(黑色塑料片，用于拉动白色锁闩释放装置以打开车门)撞上连杆 C，而不会在连杆 C 下移动。一旦你放开手柄并将 C 向上移开，连杆 D 仍将在外侧(锁定)，直到你再次按下按钮解锁汽车。然后它会移动，连杆 C，连杆 B，最终把手将能够迫使它向下，最终释放闩锁。

所以基本上，在奥兹莫比尔上，问题是拉动把手打开车门会将连杆移动到闩锁释放机构试图移动以解锁汽车的位置。

考虑到大多数人几乎不会花太多时间考虑汽车门闩，这是非常复杂的，但我们必须弄清为什么这么多人在有人锁门时试图打开车门。只是联动干扰的问题。

至于为什么要这样设计，我们也不确定。这似乎不是完全必要的(比如，为什么不把乳头和手指从大众汽车上去掉？)，所以显得刻意。我不确定这是否意味着提供一个主要的安全益处。
