# 令人敬畏的 RSS 阅读器 Reeder 3 现在可以从苹果应用商店免费下载

> 原文：<https://lifehacker.com/psa-reeder-3-is-now-free-from-the-mac-app-store-1828739108>

如果你想买一个新的 RSS 阅读器，我们可以帮你达成交易。

Lifehacker 总部最喜欢的 Reeder 3 最近将其 Mac 应用程序的价格从 10 美元降至免费，这是带着 RSS 阅读器出去兜风的最佳时机(如果你还没有这么做)。



新闻阅读器支持 Feedbin、Feedly、Feed Wrangler、FeedHQ、NewsBlur、老阅读器、Inoreader、极简阅读器 BazQux Reader、发烧、可读性和 Instapaper。

我们喜欢它，因为该应用程序有一个真正可定制的界面，主题和链接可以根据您的需要进行调整。它还能让你同时使用其中的几个新闻服务，并并排浏览。

开发者表示，第三版 [将免费，直到今年晚些时候发布第四版](https://twitter.com/reederapp/status/1034821640864129026) ，作为该应用程序的付费升级，无需订阅。Reeder 3 于 2015 年发布。
