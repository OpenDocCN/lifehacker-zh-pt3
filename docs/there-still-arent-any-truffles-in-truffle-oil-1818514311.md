# 松露油里还是没有松露

> 原文：<https://lifehacker.com/there-still-arent-any-truffles-in-truffle-oil-1818514311>

我附近有一家相当受欢迎的当地汉堡连锁店，拒绝制作非松露品种的薯条。它们并不可怕，但它们令人难以忍受，而且它们掩盖了你碰巧正在吃的任何东西的味道。它们也不是用真正的松露做的。



松露油中没有任何松露这一事实并不算是什么新闻——Vogue 早在 2003 年就爆出过——但是粗略的松露油和其他松露风味的东西仍然会在特色食品店和时尚餐厅出现。据《纽约时报》报道，在纽约和加州提起了“四起集体诉讼，指控乔氏、乌尔巴尼松露、萨巴蒂诺和莫尼尼对其松露油产品进行“虚假、误导和欺骗性的错误品牌宣传”但是那些装着可见块菌的油瓶呢？那些有他们自己的一套问题，因为生物学就是不工作。根据《泰晤士报》的报道，只有新鲜的松露，油才会一样美味。

这是否意味着你不能享用松露薯条或松露爆米花或松露土豆泥？不伙计。你喜欢你喜欢的，但是你要知道你最可能喜欢的是合成的 2，4-二硫杂戊烷 ，一种令人愉快的对称有机硫化合物。也知道你不应该为此付出那么多；合成调味油的制作并不昂贵。
