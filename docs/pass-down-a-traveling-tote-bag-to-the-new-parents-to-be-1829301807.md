# 把这个“旅行手提袋”送给你生命中的准父母

> 原文：<https://lifehacker.com/pass-down-a-traveling-tote-bag-to-the-new-parents-to-be-1829301807>

初为人父母是一种孤立的经历。你并不是简单地给你的生活添加了一个新的拼图——相反，你感觉这个小小的人把你花了几十年拼凑起来的所有现有的拼图都打乱了，现在你必须从头再来。你累了，迷路了，闻起来有点臭。很难记得其他人也去过那里。

Watch

这就是为什么当我的朋友 Joy 分享这张手提袋的照片时，我流泪了(旁边是她可爱的女儿 Drew)。这不是普通的手提袋——这是一款新的父母旅行手提袋，里面装满了新生儿生命中美丽、残酷、荒谬的最初几个月的必需品。自 2013 年以来，这个包已经穿过了 12 个不同的家。

一个新的父母旅行手提袋的工作方式是:当它到达时，接收者添加他们孩子的名字和出生日期(或者在孩子出生后这样做)，然后开始使用里面的东西——这个包包含的物品有*What be Expect*book，一双皮革婴儿莫卡辛鞋，一件医院长袍和一个装满体贴的新妈妈笔记的笔记本。(她的包是妈妈专用的，但这个想法并不需要。)有些物品会永远留在袋子里，但当父母准备把它传给下一个家庭时，他们可以放入更多他们觉得有用的东西。“我加了一瓶 Purell，Dermabond 喷雾，韩国面膜，和感谢卡，”乔伊说。“这个包太棒了，真的能让你有被支持的感觉。”

我喜欢这个想法——它真的有助于建立新父母渴望的村庄。我自己会添加的必需品包括(这比医院免费给你的东西好多了)，一个发圈(每当我在半夜哺乳时，我总是希望我有一个柔软的发圈)，和一小叠写有我最喜欢的新父母智慧的便条卡(例如，“你不需要品味每一刻”，“小声说‘他妈的怎么回事’也没关系。”一天 32 次”和“如果你有从头到脚都有纽扣的连体衣，立刻烧掉它们”)。
