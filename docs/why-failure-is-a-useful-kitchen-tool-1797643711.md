# 为什么失败是一个有用的厨房工具

> 原文：<https://lifehacker.com/why-failure-is-a-useful-kitchen-tool-1797643711>

这几天家庭厨师的压力似乎很大。光喂你的家人吃鸡胸肉是不够的——你应该喂他们自由放养的、有机的、多汁的鸡胸肉，这些鸡胸肉是用 腌制的 [烹饪的，配有](http://lifehacker.com/tag/will-it-sous-vide) [蔬菜，这些蔬菜是你从精选的厨余](http://lifehacker.com/save-surprising-kitchen-scraps-to-grow-infinite-food-537447587#_ga=2.246524100.1987194194.1502114657-1823898594.1500911226) 中挑选出来的。这种完美主义的氛围正渗透到美食写作和家常菜文化中，每个人都努力日复一日地做出一贯完美的饭菜。

以做出好的甚至完美的作品为目标本身没有错，但是对完美的痴迷会让人害怕失败，而失败是一个极其有用的工具，尤其是在厨房里。拥抱失败实际上可以让你成为更好的家庭厨师，不仅让你在厨房里更放松，还让你更有冒险精神。老实说，对失败保持冷静是我很难做到的事情——问问我的男朋友关于 2017 年未煮熟的肉桂卷事件就知道了——但它有助于记住失败对一个人的厨房技能可能产生的积极影响。

### 失败是值得纪念的

两次烘焙失败——它们几乎总是在烘焙——在我看来特别有教育意义。第一次是在很多年前的感恩节，我试图为 11 个人做南瓜面包。我大概 10 岁，非常渴望，很快地浏览了食谱，而不是彻底阅读。结果，我在橘子面糊里加入了两汤匙的小苏打，而不是两茶匙，这样做出来的面包有肥皂味，不好吃。我很确定我哭了——因为我是一个非常骄傲的青春期前的孩子——但就在那天，我学会了仔细阅读食谱，而且不止一次。与此事相关的青春期前的痛苦尴尬已经消退，但这个教训并没有消失，为此我变得更好了。

第二次令人难忘的失败发生在我(醉醺醺地)试图猜测一批核仁巧克力饼应该放多少糖的时候。我没猜对。那些核仁巧克力饼太难吃了，我认识到也许醉酒烘焙的成功率并不高。

### 失败战胜恐惧

无论是在烹饪书上还是在网上，都有大量伟大的、经过彻底测试的食谱，完全遵循它们是很有诱惑力的，永远不要偏离好的、经过测试的词语，或者加入你自己的观点。毕竟，如果你*知道*一个特定的食谱制作出吃起来有趣的食物，你为什么要弄乱它，并潜在地制作出吃起来不那么有趣的食物呢？我来告诉你为什么，我的朋友——你把它搞成*学*。

害怕烹饪和提供平庸的食物会让你和你爱的人失望，这让你无法尝试，这让你无法成长为一名厨师。知道什么一起吃不好吃和知道什么一起吃好吃一样有价值，我已经通过把一些非常可疑的食物组合放进我的嘴里培养了非常好的味觉。

此外，试着记住你是在养活你的家人，这才是最重要的。他们可以忍受一晚清淡的鸡肉或有点太难嚼的牛排。如果真的发生了灾难性的事情，点一份比萨饼。从来没有人抱怨过即兴披萨之夜。(如果他们抱怨，他们可以给自己做个三明治。)

在厨房里的即兴创作将帮助你更好地理解为什么某些食谱如此有效，并尝试各种风味搭配，如果你每次都严格遵循食谱，你将永远不会遇到这种情况。(我唯一不会即兴发挥的烹饪领域是烘焙；那是化学。)

### 失败可以是友好的

虽然我有一个化学学位，但我最初并没有被分子美食学这个非常精确的领域吸引到烹饪中来。尽管科学无疑在厨房里有一席之地，但我最初开始烹饪是为了逃避实验室和物理化学课的紧张，它们缓慢但肯定会耗尽我的生存意志。带着可爱的奈杰尔·劳森早期的书，我一路煮到了一个更平静的地方，确信无论一堆熔岩蛋糕变成什么样，我的平均成绩都将保持不变。(此外，我会吃某种形式的巧克力蛋糕，不管它们是否设置正确，这总是令人欣慰的。)

你的厨房应该是一个温暖诱人的地方。它应该是一个安全的空间，一个你可以自由试验食谱、玩各种口味、偶尔搞砸的地方。除非你不是把你创作的照片发给戈登·拉姆齐——你为什么要这么做？没有人会知道那里发生了什么，除了你选择用来喂养你的作品的人。还有， [失败很搞笑](http://skillet.lifehacker.com/will-it-sous-vide-creamy-cheesecake-edition-1786079579) ，而且很有故事性。所以，把自己推到舒适区之外，把怪异的事情混在一起，*放松，*男人。它只是食物；反正也不会陪你那么久。
