# 看看你所在的州是否允许查看警方的人体摄像头录像

> 原文：<https://lifehacker.com/see-if-youre-allowed-to-access-police-body-cam-video-in-1822673510>

在全国范围内，警察部门让警察佩戴人体摄像机来记录他们的日常活动。这些程序旨在通过记录官员和公众与人的互动来帮助他们。在一些州，公众有可能获得这些视频。

它还显示了州立法的当前状态，每个州有不同的颜色。蓝色表示有关于公众获取人体摄像机镜头的立法提案；黄色表示此类立法已经通过；布朗的意思是还没有通过关于访问的法律，但是该州只通过了关于人体摄像机实施的法案；绿色意味着一个法庭案件已经决定了关于公众获取人体摄像机镜头的规则。使用上面的工具，我能够快速阅读我的当地警察身体摄像头政策的摘要，甚至有 [到完整政策本身](https://www.rcfp.org/bodycam_policies/CA/Pasadena_BWC_Policy.pdf) 的直接链接。

请记住，仅仅因为你当地的政策允许公众获取录像并不意味着它会很容易得到。RCFP 表示，记者和其他人已经遇到了一些困难，并解释说，每个州对公共记录(这些视频属于其中)的定义非常宽泛，各州经常对处理请求收取高额费用。他们 [引用了佛罗里达州](http://tbo.com/news/politics/police-body-cameras-could-conflict-with-floridas-public-records-law-20150315/) 的一个例子，在那里，有人要求 84 小时的警察人体摄像镜头，并被告知要花费 18000 美元。有一些方法可以避免这种昂贵的费用，比如在州法律允许的情况下尽量少要视频，或者使用费用减免，但是要意识到潜在的障碍。
