# 餐厅员工，你希望顾客对你的工作了解多少？

> 原文：<https://lifehacker.com/restaurant-workers-what-do-you-wish-patrons-knew-about-1820093935>

我相信每个人都应该在某个时候在餐馆工作，只是为了让他们知道它是什么样的，但这永远不会发生。那么，让我们来解释一下餐饮服务行业鲜为人知的来龙去脉。如果你在餐馆工作，或者曾经在餐馆工作过，你希望顾客知道哪些事情，让你的生活变得轻松一点？

Watch

作为一名前餐馆工，当我看到人们对食品服务行业的任何人采取粗鲁的行为时，我都无法忍受。我知道人们大多数时候并不打算粗鲁，但这仍然让我不理智地生气。我想，如果他们曾经在柜台后面工作过，在餐桌上擦过，或者整天侍候过混蛋，他们会知道得更多。所以，让我们告诉那些从未从事过餐饮服务的人，让他们更好地了解另一方是什么样的，以及事情为什么会这样发展。例如，让我们解释一下为什么定制订单可以如此复杂，为什么餐厅有部门以及它们是如何工作的(不，你不能坐在那里代替！)，为什么服务员会问你以前是否和他们一起吃过饭，为什么分摊账单会很困难，为什么大型聚会会自动收取小费，为什么即使有空桌子也要等一段时间才能坐下，等等。
