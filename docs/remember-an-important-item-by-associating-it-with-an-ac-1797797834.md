# 通过将一个重要的项目与一个动作相关联来记住它

> 原文：<https://lifehacker.com/remember-an-important-item-by-associating-it-with-an-ac-1797797834>

当谈到我需要从我的公寓带到另一个地方的物理物品时，我几乎忘记了一切。我至少拥有十几把伞，证明了这个事实。我知道天要下雨了，我坐在门边准备出门的时候带走，然后我到了市中心才发现我把它忘了。

最近，多伦多 Baycrest 健康服务中心的研究人员发现了一种记忆那些经常被遗忘的东西的新方法:将它们与其他东西联系起来。这群科学家 [发表论文](https://link.springer.com/article/10.3758%2Fs13421-017-0736-x) 记忆&认知

这是一个叫做“统一化”的策略研究人员希望将这项技术用于痴呆症患者。

基本上，这个想法是想一想你需要记住的事情，并把它与一个动作联系起来。这会把事情分配给你的记忆，让你在需要的时候更有可能记住它。

这项研究对 80 名年龄在 61 岁到 88 岁之间的人进行了调查。该小组(尽管人数不多)的结果发现，将一个动作与一项任务联系起来，比任何其他传统的记忆技巧都更容易让人记住。

《每日电讯报》 [本周报道了这项研究](http://www.telegraph.co.uk/science/2017/08/11/simple-memory-trick-will-stop-ever-forgetting-keys-umbrella/) ，并提供了一些尝试的建议。例如，想到你的伞被卡在门锁里，可以提醒你以后去开门的时候带着它。想想你的狗在工作时堵住了门，可以帮助你记住带狗粮回家。

当行动与事情特别相关时，你更有可能记住你需要做什么，从而让它发生。
