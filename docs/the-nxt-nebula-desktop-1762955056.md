# NXT 星云桌面

> 原文：<https://lifehacker.com/the-nxt-nebula-desktop-1762955056>

读者 [ravenclawwit](http://ravenclawwit.kinja.com/) 偶然发现了 Rainmeter 的 NXT 主题，并在这个漂亮、实用的桌面上发挥了巨大作用。左上角的那个圆圈？它实际上是一个码头，挤满了快捷方式。以下是它的工作原理和设置方法。

Watch

Ravenclawwit 运行的是 Windows 10，如果你从截图上看不出来的话，那么这里的一切都可以在最新版本的 Windows 中工作。左上角的船坞发射架最棒的地方是什么？每个快捷方式都会弹出图标，您可以从中选择以快速访问最常用的程序。但是我们已经超越了自己，以下是让这款台式机成为您的专属所需的条件:

*   [壁纸](http://tylercreatesworlds.deviantart.com/art/Excalibur-285509131) 由[TylerCreatesWorlds](http://tylercreatesworlds.deviantart.com/)在 DeviantArt
*   用于 Windows 的 [雨量计](https://www.rainmeter.net/) 系统调试和监控工具
*   雨量计 的 [NXT-OS 套件，它包括左上角的圆形底座，右上角的时间和日期显示，中间的时钟和活动音频可视化器，甚至还有一个你可以从底部中间拉起的游戏抽屉。Ravenclawwit 指出，还有其他一些功能不能立即看到，比如自定义锁屏、基于快捷方式的应用程序启动器、小工具屏幕等等。](http://nxtos.com/)
*   左上方的 [圆形图标包](http://martz90.deviantart.com/art/Circle-Icons-Pack-371172325) 定制启动器中的图标， [Metro UI 图标集](http://dakirby309.deviantart.com/art/Metro-UI-Icon-Set-725-Icons-280724102) 定制点击启动器时弹出的图标
*   雨量计的 [Cascade X 小工具](http://redsaph.deviantart.com/art/CASCADEx-for-Rainmeter-v1-3-2-17-March-2015-491014798) 用于天气和正在播放的显示，以及雨量计的 [Spotify 插件](https://forum.rainmeter.net/viewtopic.php?f=18&t=17077) 用于从 Spotify 下载歌曲
*   屏幕左侧的系统显示器和硬盘指示灯的 [雨量计的皮肤](http://satyajit00.deviantart.com/art/Flat-and-Blurry-v1-5-524848816) 平坦模糊

这就是这个的全部内容。所有的部分都很好地结合在一起，我真的很喜欢成品。你可以在下面的截图中看到这个弹出启动器的运行情况。如果你喜欢你所看到的，去 ravenclawwit 的 kinja 博客(链接如下)分享你的想法——或者问你的问题，如果你是为自己设置的。
