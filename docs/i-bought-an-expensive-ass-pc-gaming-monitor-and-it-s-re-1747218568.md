# 我买了一个昂贵的电脑游戏显示器，它真的很好

> 原文:[https://kotaku . com/I-buyed-a-friendly-ass-PC-gaming-monitor-and-it-s-re-1747218568](https://kotaku.com/i-bought-an-expensive-ass-pc-gaming-monitor-and-it-s-re-1747218568)

听着，有时候你花 709 美元买一个游戏显示器，最后你会对你的购买非常满意。我不是说每次都会这样，但这次确实发生了。

Watch

经过几个月的吞吞吐吐，我说服自己不要这么做，上周我在一台新式 G-Sync 显示器上花了一大笔钱。这是一次奢侈的购买，从许多方面来说都是荒谬的:我买的显示器的价格是我个人电脑显卡的两倍，而显卡本身的价格就相当于 PS4 或 Xbox One。不管怎样。我插上电源 30 分钟后，事情已经很清楚了:这玩意值这个钱。

作为一次升级，这款显示器比升级到 4GB 显卡产生了更大、更直接的影响；不仅仅是将我的 Windows 安装和游戏转移到固态硬盘上；肯定比升级到 Windows 10 或者超频我的 GPU 和 CPU 要多。

G-Sync 是显卡制造商 Nvidia [几年前](http://kotaku.com/nvidia-is-changing-the-way-we-display-stream-and-capt-1447952607#_ga=1.14683382.1374784816.1416946327) 推出的专有显示器技术。这个想法听起来不错:内置在显示器中的芯片允许显示器直接与 PC 的显卡对话，并动态改变其刷新率，平滑地匹配显卡输出的任何内容。

如果你看到了过去几年中关于帧速率和 60fps 的所有讨论，那也全都与刷新率有关。游戏的帧率越高(也越稳定)，看起来就越流畅。游戏的帧速率越接近屏幕的刷新率，撕裂或挂钩的可能性就越小。

从历史上看，PC 游戏玩家一直受困于两种同步帧与显示器刷新率的选择:他们可以使用垂直同步(Vsync ),人工将游戏的帧速率锁定在目标值，或者他们可以简单地以未锁定的帧速率运行游戏。这两种选择都有缺点，这两种选择都会让你觉得没有充分利用昂贵的显卡。我听到的关于 G-Sync 的一切都表明，这项技术是一种真正的、实际的、真正的同步 PC 和屏幕的方式，它使游戏运行明显更加流畅。

上周我决定，去他妈的，我要去。 [这是我买的显示器](http://us.acer.com/ac/en/US/content/model/UM.HB0AA.001) 。这是一款 27 英寸、2560x1440 的宏碁，刷新率为 144Hz，内置 G-Sync 支持。还有很多其他的 G-Sync 显示器；这款有一些 [好评](http://www.pcgamer.com/the-best-gaming-monitors/) ，所以我决定用它。 [我从亚马逊](http://www.amazon.com/gp/product/B00UPVXDA8?asc_campaign=InlineText&asc_refurl=https://kotaku.com/i-bought-an-expensive-ass-pc-gaming-monitor-and-it-s-re-1747218568&asc_source=&psc=1&redirect=true&ref_=oh_aui_detailpage_o00_s00&tag=kinjakotakulink-20) 买到的比标价低一点，但是很多商店都有卖。

**影响我决定的想法:**

1.  鉴于 1440p 显示器已经成为稳定的 PC 游戏的更合理的分辨率，它似乎是一个安全的赌注。4K 分辨率对于显示器大小的屏幕来说似乎不太实际，甚至没有必要。

2.  144Hz 是我公寓里其他屏幕 60Hz 刷新率的两倍多，但我已经看到足够多的电脑游戏玩家发誓要有更高的帧率，所以我想看看到底是怎么回事。
3.  在分辨率和刷新率之间，这款显示器似乎不会过时，至少在几年内不会过时。不管是好是坏，我已经承诺从 Nvidia 购买我最新的显卡，所以我真的不认为自己会很快转向 AMD。
4.  在波特兰，每年的这个时候，大约下午 4 点钟天就要黑了，给自己买些凉爽的东西会让我暂时忘记这些，感到快乐。

**缺点:**

1.  它只有一个 DisplayPort 输入，所以如果不购买昂贵的适配器并手动交换输入，我就无法将它兼作我的游戏机的辅助显示器。显然，这种 G-Sync 显示器总是如此，这是一个令人失望的。

2.  它的价格是 709 美元，花在游戏外设上是一笔疯狂的钱，足够买一大堆油炸圈饼和熏牛肉三明治了。
3.  我真的不喜欢英伟达做生意的一些方式。(稍后会详细介绍。)

利大于弊，所以我点了这个。几天后，它到了。我插上电源，然后你。这是一个非常好的显示器。

写这种技术很棘手，因为我不能只给你看。你必须相信我的话。所以:G-Sync 像宣传的那样工作，它明显改变了我体验电脑游戏的方式。我不再担心帧速率的波动——我只是打开一个游戏，关闭 Vsync，让它运行。我有一个 GTX 970 显卡，可以运行大多数游戏，至少 1080p，每秒 60 帧以上。由于我的显示器现在一直达到 144Hz，它有足够的空间让游戏在 60-80fps 的范围内存在，并且由于 G-Sync，它可以流畅地运行所有这些帧速率，没有撕裂。

(一些纯粹主义者可能需要他们的游戏以 144fps 的速度运行——我还没有做到这一点。我通常可以检测到游戏低于 60fps，但在游戏过程中，我真的无法区分 78fps 和 94fps 之间的区别。当我以锁定的 144fps 运行游戏时，我可以发现它异常流畅，但任何超过 60 的速度对我来说都没问题。)

直到我不再需要处理它们，我才意识到屏幕同步问题是如何彻底侵入我的电脑游戏意识的。以前，我会用一只眼睛盯着屏幕角落的 FPS 计数器，开始玩新的电脑游戏。如果我看到一个故障或一个慢点，我的双眼会迅速看向屏幕边缘，就像我试图捕捉表演中的性能下降一样。“哦，那次降到了 54fps，”我想。“那不好。”最终我将不得不关掉 FPS 计数器，这样我就不会再盯着它看，享受游戏了。

现在，每个游戏只需*运行*。 *GTA V* 在 1440p 的近超设置下能够维持 70-90fps，你应该*看*吧。看起来很完美。即使在罕见的情况下，帧速率下降到 60fps 以下，我也几乎没有注意到，因为没有挂钩或口吃。其他游戏看起来也一样好:*魔多之影*，*疯狂的麦克斯*，*巫师 3* ，*黑色行动 III* ，等等。*垂死的光*看起来像香蕉。*余波 4* 和*只是因为 3* 在以一致的帧速率运行时遇到了一些真正的问题，但即使是那些游戏的下降对 G-Sync 运行来说也不是什么大问题。

显示器有时会有点古怪:例如，每当我离开菜单时，刺客信条辛迪加的 FPS 就会降到零，尽管过一会儿又会回到 60+。*神性:原罪*有时会周期性地开始冻结和解冻，尽管这个问题通过重启游戏来解决，我实际上不能说这是否与 G-Sync 有关。无论如何，一些打嗝对我来说不会破坏整体体验。毕竟，这是电脑游戏。总会有一些*的古怪。*

然而，我还不喜欢显示器的其他一些东西，主要是它要求我对 Nvidia 的硬件生态系统做出大量的财务承诺。呃。光是输入“硬件生态系统”这一句，就让我觉得妥协了。迈出购买英伟达或 AMD 显卡的第一步是一回事。这是第一步，下次你可以随时改变主意，走另一条路。购买第二个硬件是一项更大的投资；它有效地消除了任何机会，我会切换到 AMD 的持续时间这一显示器的寿命。

这项技术是如此之好，我希望所有的显示器都有它，而且它可以与任何图形卡一起工作。所以，G-Sync 是 Nvidia 的专利，需要这样的财务承诺才能获得它，这太糟糕了。AMD 有一项名为 FreeSync 的竞争技术，听起来类似，因为它需要一台显示器配备该技术，然后您的 AMD 卡才能使用它。两者在操作方式上有一些技术差异，但一个主要的差异是，Nvidia 控制了谁可以和不可以在他们的显示器上使用 G-Sync 模块，而 AMD 则让任何可能想要支持它的公司免费使用 FreeSync。

AMD 更开放的方法无疑部分是因为他们是这个不太受欢迎的品牌，需要削弱英伟达的领先地位，但两者之间的二分法仍然加强了我对英伟达运营方式的厌恶。Nvidia 致力于在任何可能的地方注入他们的专有技术，这意味着大多数大预算游戏都带有 GameWorks 功能，这些功能可能(并且经常)阻碍非 Nvidia GPUs 的性能。我知道 PC 游戏市场的竞争有多激烈，我并不完全嫉妒英伟达成功和赚钱的尝试，但正如我们在过去几年中所看到的那样，这种残酷的操纵在帮助游戏的同时也伤害了游戏，并让那些不遵守规则的游戏玩家感到头疼。

感觉所有面向游戏的显示器和电视都能做类似 G-Sync 的事情只是时间问题。我可能不喜欢我被迫选择一个品牌，并在可预见的未来坚持使用它，但就目前而言，我最终同意使用 Nvidia。我相信他们不会去任何地方，他们的卡通常会很好地运行我想玩的游戏。暂时可以了。

我知道这篇文章可能会让你们中的一些人考虑花太多的钱在一个你并不真正需要的硬件上。我很同情！大多数电脑游戏玩家都有一件他们不拥有的设备，但是他们希望拥有。毕竟，你的个人电脑总可以更好一点；既是 PC 游戏的 [福](http://kotaku.com/the-10-best-things-about-building-a-new-gaming-pc-1605434587) 祸。你总是可以拥有稍微快一点的 CPU 或者稍微好一点的显卡；那个噼啪作响的键盘，或者那个全是按钮的鼠标。

我们所有人都有一样东西，那就是我们正在考虑得到的下一个东西。有时下一件事比它的价值更麻烦；有时候，接下来的事情是令人失望的。但是有时候，你在一件东西上花了一大笔钱，结果却物有所值。万岁，一项新技术确实实现了它的承诺。
