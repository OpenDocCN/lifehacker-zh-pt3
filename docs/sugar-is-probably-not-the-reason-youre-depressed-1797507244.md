# 糖可能不是你抑郁的原因

> 原文：<https://lifehacker.com/sugar-is-probably-not-the-reason-youre-depressed-1797507244>

最近的一项研究发现，甜食与男性抑郁症有关，但与女性无关。糖有特殊的力量能把男人赶走吗？大概不会。让我们开始吃吧。



**头条** : [吃太多糖与男性抑郁有关，穷人](http://nymag.com/scienceofus/2017/07/eating-too-much-sugar-is-linked-to-depression-for-men.html) (美国科学)

**故事**:这项研究是探索超重 的 [人群的情绪和幸福项目的一部分。过去有一些研究表明糖和抑郁之间有联系——例如，这个研究发现吃很多不健康食物的学生更容易抑郁...或者反过来。](http://www.moodfood-vu.eu/)

这项名为 [的新研究发表在科学报告](https://www.nature.com/articles/s41598-017-05649-7#Tab3) 上，它着眼于一项对英国办公室工作人员 的 [长期研究，试图排除抑郁的人是否会寻求安慰性食物。在这种情况下，抑郁症将首先出现，随后摄入含糖食物。他们没有发现这种模式。](https://academic.oup.com/ije/article-lookup/doi/10.1093/ije/dyh372#13133922)

但是当他们寻找相反的证据时，即含糖食物导致抑郁，结果是不可靠的。是的，糖摄入量最高的男性患抑郁症的风险更高。但是作为研究的一部分，研究人员问了很多问题，没有找到其中大部分的联系。例如，他们使用两种不同的问卷来判断人们是否抑郁。在任何给定的时间点——寻找抑郁同时吃大量糖的人——其中一项测试发现了联系，而另一项没有。糖摄入量最高的人也是最不容易超重的*，这与我们对糖和肥胖的了解不符。*

*这项研究还包括两倍于女性的男性，这可以解释他们确实发现的一个结果中的性别偏见。也许糖对男性和女性都有影响，但是这项研究没有足够的女性来证实这一点。或者也许在男性身上的发现是偶然的，真的没有联系。当一项研究提出许多不同的问题时，就像这项研究一样，通常会偶然发现一些在更大、更集中的研究中不成立的结果。我担心这可能就是这里发生的事情。*

***要点**:这项研究给研究人员一个暗示，糖和抑郁之间可能存在*，而且这种*在男性中可能*更强。但这绝对不能证明。**
