# 如何处理自己发来的垃圾邮件

> 原文：<https://lifehacker.com/how-to-handle-email-spam-from-yourself-1826325674>

在本周的 Tech 911 专栏中——我们的常规专栏解决您最紧迫的技术问题(您可以也应该将这些问题发送给我们 [这里](mailto:david.murphy@lifehacker.com?subject=Tech 911))——我们来看看一个有着特殊变化的经典问题:垃圾邮件。



没有人喜欢收件箱里塞满了垃圾，如今大多数服务都擅长过滤掉那些你不应该处理的明显信息，比如那些试图通过戏弄更大的支出来勒索你的钱的信息，或者那些承诺增加你胸部和/或 [威利](https://www.youtube.com/watch?v=c7JOcHKMLKQ) 尺寸的信息。

但是，如果负责发送垃圾邮件的人是...你呢。Lifehacker 的读者凯伦写道:

> *亲爱的大卫:*
> 
> 我收到了过多的垃圾邮件。大多数情况下，我会屏蔽。然而，骗子们找到了发送邮件的方法，用我自己的电子邮件地址作为回邮地址。既然我不能屏蔽我自己的电子邮件地址，我怎样才能找到它们真正的发送地址，以便屏蔽它们呢？我用 Outlook 处理我的“商务”电子邮件。

### 如何...街区...你自己？

这里发生的是垃圾邮件发送者正在欺骗您的电子邮件地址，试图绕过您所拥有的任何类型的垃圾邮件保护。不幸的是，根据你的企业的垃圾邮件防护设置，这实际上是一个相当有效的伎俩。你没有提供关于你所做的大量细节，但我认为第一步，也是最好的一步，是去窃听你的 IT 团队——如果你有团队的话——并让他们设置服务器端规则，以检查电子邮件 [是否从你的组织](http://danblee.com/anti-spoofing-for-domains-in-exchange-2016/) 之外发送，但使用你的组织的域作为发送者(假)电子邮件地址的一部分。

这个过程不是我能说的，因为我从来没有做过，但是你的 it 团队应该可以设置这个。此外，无论是谁负责您的企业基础设施，都希望确保他们已经设置了 [SPF、DKIM 和/或 DMARC](https://heimdalsecurity.com/blog/ip-email-phone-spoofing/) ，这些都是帮助您确定从您的域发送的电子邮件是真实可信的方法(并且您收到的看似来自您的域但实际上不是的电子邮件会被正确标记为垃圾邮件)。

如果您拥有自己的域名，并且您的企业是一家公司，那么值得探究一下您的 web 主机有哪些关于这三种协议的文档。你的主人也许能够以一种容易理解的方式说明如何设置它们，这可能会给你一个很好的工具来对抗你的自我垃圾邮件。

如果没有 Outlook 本身，请确保您充分利用了该程序的反垃圾邮件功能。在 [我的 Outlook](https://support.office.com/en-us/article/overview-of-the-junk-email-filter-5ae3ea8e-cf41-4fa0-b02a-3b96e21de089#ID0EAABAAA=2016,_2013) 版本中，我可以通过点击主页选项卡和微小的垃圾邮件快捷方式(在“新邮件”和“新项目”的右侧)来访问垃圾邮件过滤器。

在出现的下拉菜单中，点击“垃圾邮件选项”,首先确保它是打开的，并设置到你觉得合适的级别。至少一定要“低调”;考虑“高”，但你会想不时地检查你的垃圾邮件文件夹，以防任何合法的东西被过滤掉。你也可以设置它，只有白名单上的电子邮件地址或域才能进入你的收件箱，但这可能不会解决你的“发件人”问题，因为听起来你不一定想阻止自己的电子邮件地址。

在我考虑的时候，检查一下你的“已发送”文件夹，确保你没有向自己发送垃圾邮件。我对此表示怀疑，但如果是这样的话，这是一个全新的问题:涉及更改你的密码、禁止访问你与电子邮件账户绑定的第三方应用等。(简而言之，你被黑客攻击了，垃圾邮件发送者实际上是从你的地址发送电子邮件。)

你可以尝试安装一个第三方应用程序，比如 [Mailwasher](https://www.mailwasher.net/) 来帮助你处理垃圾邮件问题。我自己没有用过这个，但是免费版本值得研究一下，看看它是否真的能帮助你减少欺骗邮件。

如果您仍然有问题，您也可以尝试在您的公司创建一个新的电子邮件别名。将来自你“旧”电子邮件地址的邮件过滤到垃圾邮件中，只有当你想给自己发邮件时才使用这个新邮件。这是一个粗糙的解决方法，但是很简单。

最后，如果你是一个团队，你只是使用你的网站主机的电子邮件功能来接收邮件，你可能要考虑联系他们，以确保你已经为你的域启用了任何和所有的垃圾邮件防护。你也可以考虑为你的域名设置一个类似 [谷歌的 G 套件](https://zapier.com/learn/g-suite/setup-g-suite-google-apps/) 的服务。这将花费你每月 5 美元，但谷歌内置的反垃圾邮件功能非常强大，如果你愿意，你仍然可以使用 Outlook 查看你的电子邮件。
