# 很快你就可以用 Alexa 和你的电视对话了

> 原文:[https://life hacker . com/you-ll-can-to-use-Alexa-talk-to-your-TV-eventua-1795937952](https://lifehacker.com/you-ll-be-able-to-use-alexa-to-talk-to-your-tv-eventua-1795937952)

亚马逊试图用一种新的 Alexa 技能杀死你的控制器，这种技能旨在使看电视变得超级简单——但你仍然需要正确的设备。

Watch

被称为 [视频技能 API](https://developer.amazon.com/public/solutions/alexa/alexa-skills-kit/docs/understanding-the-video-skill-api?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/you-ll-be-able-to-use-alexa-to-talk-to-your-tv-eventua-1795937952&asc_source=&tag=kinjalifehackerlink-20) ，它可以让你搜索和控制视频播放以及在你支持的设备上更换频道。也支持按演员和流派搜索。

> 视频技能 API 旨在使客户无需调用特定技能即可轻松找到并使用视频内容。例如，客户可以说，“Alexa，在海边玩曼彻斯特”，而无需指定提供商或设备。通过视频技能 API，Alexa 可以感知用户拥有或订阅的视频设备和服务，并使用户能够通过语音控制这些设备和服务的体验。

如果这意味着我认为它意味着什么，很快我就可以大喊“Alexa，玩露台饭店！”看着它在我的电视上弹出。当然，想要 Alexa 兼容设备的制造商(如有线电视盒制造商)以及视频服务提供商将不得不使用新的视频技能 API 更新软件，以与 Alexa 进行交互，但语音控制电视的未来看起来会好一点。

这项技能是 [以前只有在美国有](https://www.theverge.com/2017/5/22/15674490/amazon-alexa-voice-control-dish-hopper-dvr-handsfree) 提供给有漏斗或 Wally 的 Dish 客户，国外有 Netgem 和 YouView 客户。亚马逊建议与硬件制造商或视频服务提供商相关的开发人员检查视频技能，但从技术上讲，任何人都可以使用它的视频技能 API。

当然，有了 Google Home 和 Chromecast，你就可以开始和你的电视对话，从你自己的谷歌图书馆或网飞下载内容。
