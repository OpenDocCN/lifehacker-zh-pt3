# 这个疾控中心的应用程序告诉你你需要哪些疫苗来看世界

> 原文：<https://lifehacker.com/this-cdc-app-tells-you-which-vaccines-you-need-to-see-t-1820300409>

iOS/Google:世界上不同的国家有独特的食物、风格和文化，但他们也有自己的疾病。这就是为什么在你出国旅行前四到六周接种疫苗是很重要的。如果你准备乘飞机环游世界，疾病控制中心的 TravWell 应用程序可以帮助你准备好免疫系统。



适用于 iOS 和 Android， [CDC TravWell](https://wwwnc.cdc.gov/travel/page/apps-about) 根据您要去的地方和时间，为您提供清晰的说明。下载应用程序后，您可以规划一次包含多个目的地的旅行。然后 TravWell 会给你每个地点的疫苗需求和建议，以及起飞前你应该做的其他事情的清单。

比方说，你计划去泰国旅行。一旦你输入你的旅行日期，你会看到四个标签。第一个选项卡，CDC 疫苗和药物建议，将建议您除了常规疫苗接种之外，还要接种甲型肝炎、乙型肝炎、日本脑炎、狂犬病、伤寒、黄热病、疟疾和霍乱疫苗。下一个标签，待办事项，给你一个有用的，可定制的清单，列出你旅行前和旅行中要做的事情。你会看到一些提示，比如“旅行前咨询医疗服务提供商”、“向美国国务院登记”、“将行程交给紧急联系人”以及“使用驱虫剂”。打包列表选项卡是一个可定制的药品和医疗用品清单，您应该考虑打包并在旅途中随身携带。最后，在“文档”选项卡中，您可以保存和组织重要旅行文档的照片或扫描件。

当您接受免疫接种时，TravWell 会让您保留一份记录，以及您想要跟踪的任何药物的记录。你甚至可以在应用程序中设置提醒，让你了解疫苗加强剂量的最新情况，或者在你旅行时提醒你该服用某些药物了。最重要的是，无论您是否在线，TravWell 都能正常工作。只要你能给你的智能手机供电，你就能访问你的记录和清单，阅读疾病预防控制中心的建议，并找到紧急服务的电话号码，无论你在世界的任何地方。您可以通过下面的链接下载适用于您的 iOS 或 Android 设备的 TravWell。
