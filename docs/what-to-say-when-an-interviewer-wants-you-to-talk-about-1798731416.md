# 当面试官想让你谈谈你自己时，该怎么说

> 原文：<https://lifehacker.com/what-to-say-when-an-interviewer-wants-you-to-talk-about-1798731416>

众所周知，求职面试压力很大。虽然你不可能准备好面试官会问你的每一个问题，但有几个问题你可以做好准备，这些问题可能会为接下来一个小时的面试定下基调。有人可能会问你的第一个问题是:“说说你自己吧。”



这是一个开放式的问题，你可以用从你的个人生活到你为什么离开上一份工作的所有信息来回答。然而，虽然这看起来只是一个破冰，但有“正确”和“错误”的回应方式。如果你提前做了一点计划，那么你真的可以让这份工作出类拔萃，为接下来的面试定下一个积极的基调。

美国消费者新闻与商业频道最近与几位面试专家 、琼·库尔和桑迪·古尔德谈论了如何正确回答臭名昭著的面试问题。

最好的外卖之一？根据你面试的工作量身定制你的回答。

这似乎是一个显而易见的问题，但也是很多人都会犯的错误。当你考虑如何回答这个问题时，想想你申请的职位以及面试官想要什么样的条件。

在你面试之前，准备一些关于你如何达到这些标准的合适的故事。这些不一定是具体的以工作为中心的故事。如果你在职业生涯中没有达到特定的工作要求，但通过志愿者职位达到了，也要提到这一点。不需要在谈话中提到“我看你在找一个有很好的组织能力的人……”只要说出你的故事，让面试官自己来推断就行了。

并对自己的经历充满信心。很容易陷入这样的想法:你的个人经历对某一份工作来说“不够好”。如果你在面试时把这一点说出来，那么你的面试官很可能会认为你也没有合适的经验，即使你有。相反，去参加面试时，你要确信自己是这份工作的最佳人选。如果你相信，那么你的面试官也很有可能会有同样的感觉。
