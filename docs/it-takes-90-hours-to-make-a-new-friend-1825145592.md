# 交一个新朋友需要 90 个小时

> 原文：<https://lifehacker.com/it-takes-90-hours-to-make-a-new-friend-1825145592>

你可能知道 [把人加入你的核心圈子](https://lifehacker.com/this-is-how-many-friends-you-need-to-be-happy-1823425885) 需要时间，但是从陌生人变成好友实际上需要多少时间一直是个谜——直到现在。一项新的研究表明，你需要和某人相处至少 90 个小时，他们才会认为你是真正的朋友。



发表在*社会和个人关系期刊* 上的 [报告发现，从相识到成为“普通朋友”(想想喝酒的伙伴，或者在聚会上看到的朋友的朋友)，通常需要大约 50 个小时的相处时间；大约 90 个小时成为真正的“朋友”(你们都抽出时间专门和对方出去玩)；并且超过 200 个小时来形成 BFF 类型的纽带(你感觉和这个朋友有情感上的联系)。](http://journals.sagepub.com/doi/full/10.1177/0265407518761225)

澄清一下，那是你们一起做事情的醒着的时间。可以是任何事情——喝咖啡、看电视、玩游戏等等。—只要你们俩都选择和他们在一起。根据这项研究，这意味着你和某人一起工作挣的时间不算多。同事仍然可以成为朋友，当然，但是你需要在工作之外花时间在一起。

那么，知道这些数字对你有什么好处呢？嗯，它强调了友谊的一个重要方面，我们有时都忘记了:个人投资。该报告的主要作者、传播学副教授杰弗里·霍尔解释说，交朋友其实就是花时间。 [正如他所说的](https://news.ku.edu/2018/03/06/study-reveals-number-hours-it-takes-make-friend) “不能弹指一挥间交个朋友。”不要和一个人出去玩几个小时，然后期望已经是“最好的朋友”。给你喜欢的人你的时间，希望他们也会给你他们的时间，并专注于享受美好时光。友谊随之而来。

如果你不确定你对你认为是朋友的人(或者是朋友候选人)的态度，试试霍尔的互动友谊工具 。它会问你几个问题，比如你和那个人相处了多长时间，你认识他们多久了，你对他们的感觉如何，然后将你的情况与数据集进行比较。试试看。
