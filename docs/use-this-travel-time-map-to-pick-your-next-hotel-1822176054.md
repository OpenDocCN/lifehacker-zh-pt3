# 使用这张旅行时间图来选择您的下一个酒店

> 原文：<https://lifehacker.com/use-this-travel-time-map-to-pick-your-next-hotel-1822176054>

当你去旅行时，你通常想住在中心的某个地方，方便到达几个当地目的地:一个会议中心、几家餐馆、一个有趣的社区。你不能只盯着地图看。旅行时间不仅仅取决于距离；他们依赖街道布局、高速公路和公共交通。你可以在谷歌地图上测试每个旅行时间，但有时你只想看到一个大地图，显示你在一个小时内可以去的所有地方。为此，试试 [旅行时间地图](https://app.traveltimeplatform.com/#) 。



说出一个地点，TravelTime Maps 会显示出你在 15、30、45 或 60 分钟内可以到达的所有地方。它还会显示附近的酒店、电影院或咖啡店。您甚至可以选择多个位置，并获得所有位置附近的维恩图。例如，这里有一张靠近曼哈顿中央公园和布鲁克林展望公园 的 [酒店地图。](https://app.traveltimeplatform.com/#/search/0_lat=40.67426&0_lng=-73.97078&0_color=%2339d2e2&1_lat=40.76839&1_lng=-73.98202&poi=Hotel)

TravelTime Maps 是 TravelTime 平台 的一个演示，它出售对求职和房地产网站的 API 访问。该演示在识别一些地名方面很糟糕，但如果你有困难，你可以在地图上的任何地方拖动大头针。

与谷歌地图不同，TravelTime 没有美国的交通拥堵数据，因此它的行驶时间是乐观的。但对于公共交通，它使用的数据与谷歌地图相同。(你可以告诉它你将在哪天和什么时候旅行。)这个工具对于一般的规划比较有用；通过谷歌地图和当地公交服务查询具体行程。这也是一种可爱的方式来看看什么是真正的“离家近”——或者找到一个没有人能听到你尖叫的遥远的地方。
