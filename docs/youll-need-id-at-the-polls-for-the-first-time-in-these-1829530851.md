# 在这五个州，你第一次投票时需要身份证

> 原文：<https://lifehacker.com/youll-need-id-at-the-polls-for-the-first-time-in-these-1829530851>

自 2016 年总统选举以来，各州立法机构已经实施了约 500 项与投票有关的立法， [ProPublica 报道](https://projects.propublica.org/graphics/voting-changes-2018) 。在一些州，理解这些变化可能是去投票和被拒绝投票的区别。



根据 ProPublica 的报告，阿肯色州、爱荷华州、密苏里州、北达科他州和德克萨斯州自 2016 年以来通过了新的法律，要求在投票时出示身份证。这些州的选民需要知道:

*   阿肯色州的选民身份法目前正在州最高法院接受挑战，但是选民应该计划在 11 月份实施。要求选民出示带照片的身份证；根据 [全国州议员会议](http://www.ncsl.org/research/elections-and-campaigns/voter-id.aspx#Ftn%201) 的规定，如果他们没有宣誓书，他们可以签署一份宣誓书。
*   **爱荷华州**选民 [将需要出示](https://sos.iowa.gov/elections/voterinformation/voteridfaq.html) 爱荷华州驾照、爱荷华州非运营商 ID、军人 ID、美国护照、部落 ID、退伍军人 ID 或选民证。今年，如果他们没有这些形式的身份证，他们将被允许签署身份宣誓，但从 2019 年开始，他们将只允许投临时选票。
*   密苏里州的选民需要某种形式的身份证明，尽管不需要包括他们的照片。如果你没有带照片的身份证，你会看到你还可以带什么。
*   **北达科他州**选民必须 [提供身份证](https://www.usnews.com/news/best-states/north-dakota/articles/2017-12-28/attorney-new-north-dakota-voter-law-still-unconstitutional) 如驾照或部落 ID；如果他们没有身份证，他们可以投临时票，但他们必须在选举后六天内向选举官员出示有效身份证，以便他们的选票被计算在内。
*   德克萨斯州选民必须提供政府颁发的七种身份证件中的一种，比如驾照或护照。如果他们没有身份证，他们必须签署一份宣誓书确认他们的身份。该法律是有争议的，因为它“对那些在宣誓书中做虚假陈述的人进行刑事处罚”，T3*纽约时报* 报道，“该法律的批评者认为这一举动混淆并惊吓了潜在的选民。”

总共有 34 个州要求选民在投票时出示身份证或签署宣誓书。你可以在这里看到完整的 [分解成](https://lifehacker.com/all-the-information-you-need-to-vote-in-the-2018-midter-1829328834) 。
