# 高中技术老师的包

> 原文：<https://lifehacker.com/the-high-school-tech-teachers-bag-1734704135>

学生对书包有各种特殊需求，老师也是。Lifehacker reader 和数字媒体制作教师 mattewrstover 分享了他为教学和数字媒体创作准备的包。



包里是一个 [SwissGear 1908 背包](http://www.amazon.com/SwissGear-ScanSmart-Computer-Backpack-Laptops/dp/B002OHDIWY/ref=sr_1_1?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-high-school-tech-teachers-bag-1734704135&asc_source=&ie=UTF8&keywords=SwissGear 1908 Backpack&qid=1444062424&sr=8-1&tag=kinjalifehackerlink-20) 。这是里面的内容:

*   [锚 15000mAH 便携式装载机](http://www.amazon.com/15000mAh-Aluminum-Portable-Technology-Multi-Voltage/dp/B005NGKR54/ref=sr_1_3?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-high-school-tech-teachers-bag-1734704135&asc_source=&ie=UTF8&keywords=Anker 15000mAH Portable Charger&qid=1444062435&sr=8-3&tag=kinjalifehackerlink-20)

*   充电/同步电缆 Note Pro

*   茶和维生素

*   [三星 Note Pro](http://www.amazon.com/Samsung-Galaxy-Note-12-2-Black/dp/B00HWMPSK6/ref=sr_1_1?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-high-school-tech-teachers-bag-1734704135&asc_source=&ie=UTF8&keywords=Samsung Note Pro&qid=1444062452&sr=8-1&tag=kinjalifehackerlink-20) 带蓝牙键盘

*   键盘充电线

*   药物(日/奈奎尔、Motrin 等)

*   [音频技术员 ATR-3350 领夹式话筒](http://www.amazon.com/Audio-Technica-ATR3350IS-Lavalier-Microphone-Smartphones/dp/B00HZA6EJO/ref=sr_1_1?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-high-school-tech-teachers-bag-1734704135&asc_source=&ie=UTF8&keywords=Audio-Technica ATR-3350 Lavalier Microphone&qid=1444062462&sr=8-1&tag=kinjalifehackerlink-20)
*   [三星标签 3](http://www.amazon.com/Samsung-Galaxy-Lite-7-Inch-White/dp/B00HWMPSJC/ref=sr_1_1?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-high-school-tech-teachers-bag-1734704135&asc_source=&ie=UTF8&keywords=Samsung Tab 3&qid=1444062482&s=pc&sr=1-1&tag=kinjalifehackerlink-20)

*   [缤特力 975 蓝牙耳机](http://www.amazon.com/Plantronics-Discovery-975-Bluetooth-Headset/dp/B002JIMZU4/ref=sr_1_1?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-high-school-tech-teachers-bag-1734704135&asc_source=&ie=UTF8&keywords=plantronics 975 headset&qid=1444062509&sr=8-1&tag=kinjalifehackerlink-20)

*   眼镜
*   用于 Note 4 和 Note Pro 的带 OTG 线缆的 6 插槽 USB 保护套

*   眼镜盒里装满了类似于 [索尼 MW600 蓝牙立体声耳机](http://www.amazon.com/Sony-1264-5582-MW600-Wireless-Headset/dp/B008NA985C/ref=sr_1_2?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-high-school-tech-teachers-bag-1734704135&asc_source=&ie=UTF8&keywords=Sony MW600 Bluetooth Stereo Headset&qid=1444062530&sr=8-2&tag=kinjalifehackerlink-20) 和 SD 卡适配器
    的高科技连接器
*   处方太阳镜

*   [1.5TB 东芝外置 Canvio 硬盘](http://www.amazon.com/Toshiba-Canvio-Basics-Portable-HDTB310XK3AA/dp/B00N2S6ZUQ/ref=sr_1_1?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-high-school-tech-teachers-bag-1734704135&asc_source=&ie=UTF8&keywords=1.5TB Toshiba External Canvio Hard Drive&qid=1444062547&sr=8-1&tag=kinjalifehackerlink-20)

*   [佳能 5d 马克 2 号 Dslr 配 50mm 1.8G 楞次](http://www.amazon.com/Canon-Mark-Frame-Camera-MODEL/dp/B001G5ZTLS/ref=sr_1_1?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-high-school-tech-teachers-bag-1734704135&asc_source=&ie=UTF8&keywords=Canon 5d Mark II Dslr&qid=1444062566&sr=8-1&tag=kinjalifehackerlink-20)

*   [奥特曼无线闪光触发器](http://www.amazon.com/Trigger-Altura-Photo-Wireless-Shutter/dp/B00NFYHW8Q/ref=sr_1_1?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-high-school-tech-teachers-bag-1734704135&asc_source=&ie=UTF8&keywords=Altura Wireless Flash Trigger&qid=1444062577&sr=8-1&tag=kinjalifehackerlink-20)

*   闪光用彩色凝胶

*   急救包

*   Opteka 85mm 1.8 Prime 镜头

*   [佳能 28-80mm 3.5 镜头](http://www.amazon.com/Canon-28-80mm-Standard-Discontinued-Manufacturer/dp/B00004THCX/ref=sr_1_1?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-high-school-tech-teachers-bag-1734704135&asc_source=&ie=UTF8&keywords=Canon 28-80mm 3.5 Lens&qid=1444062601&sr=8-1&tag=kinjalifehackerlink-20)

*   烟花爆竹

*   [永诺 565 马克 III 闪光](http://www.amazon.com/Yongnuo-Yn600ex-rt-Wireless-Speedlite-600ex-rt/dp/B00YA97DC0/ref=sr_1_1?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-high-school-tech-teachers-bag-1734704135&asc_source=&ie=UTF8&keywords=Yongnuo 565 Mark III Flash&qid=1444062613&sr=8-1&tag=kinjalifehackerlink-20)

*   6 节 AA 电池
