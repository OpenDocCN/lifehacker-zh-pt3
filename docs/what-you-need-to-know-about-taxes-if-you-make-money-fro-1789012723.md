# 如果你想从零工经济中赚钱，你需要了解什么税收

> 原文:[https://life hacker . com/what-you-need-know-on-tax-if-you-make-money-fro-1789012723](https://lifehacker.com/what-you-need-to-know-about-taxes-if-you-make-money-fro-1789012723)

如果你想在 之外赚取额外的 [现金，从](https://lifehacker.com/the-complete-guide-to-making-money-in-your-spare-time-1291903155) [Airbnb](https://www.airbnb.com/) 到 [Lyft](https://www.lyft.com/) 到 [Etsy](https://www.etsy.com/) 有很多选择。你可能听说过这被称为演出、共享或按需经济。不管你怎么称呼它，如果你以这种方式获得任何收入，你都必须考虑税收。以下是你需要知道的。

### 即使你挣得不多，你(可能)也要付钱

不管你做什么工作，国税局都希望你纳税。当你从事一份标准的朝九晚五的工作时，你的 [雇主会为你](http://lifehacker.com/how-do-i-avoid-owing-money-or-getting-a-huge-refund-on-5890480) 扣缴这些税。当你作为自由职业者或独立承包商赚钱时，比如说，如果你签约成为 Lyft 司机，你赚的钱还没有被征税。这取决于你确保你支付国税局。

这意味着，如果你从副业、工作或演出中赚了钱，很有可能，你需要将其报告为挣得的收入(是的，甚至是现金小费)。不过，这些规则有点棘手。

如果你一年的收入超过一定的门槛，雇主或公司应该给你一张 [1099-MISC](https://www.irs.gov/uac/about-form-1099misc) 或 [1099-K](https://www.irs.gov/uac/form-1099-k-merchant-card-and-third-party-network-payments) 表格来帮你报税。1099-MISC 通常适用于雇佣自由职业者的公司，如易贝、贝宝、优步等第三方公司。1099-MISCs 的起征点是 599 美元以上，1099-Ks 的起征点是 20，000 美元以上或纳税年度内 200 次以上的交易。

然而，仅仅因为你的收入低于这些门槛或者公司没有给你发表格，并不意味着你就不用缴税了。 [根据美国国税局](https://www.irs.gov/businesses/small-businesses-self-employed/sharing-economy-tax-center) 的规定，你可以记录自己的收入，并在报税时上报。

不过，租房子的工作方式有点不同。假设你用 Airbnb 这样的服务出租了一个房间或房子。如果你租出去少于 15 天，你实际上不需要在你的纳税申报单上申报收入。(当然，你也不能扣除与这个收入相关的支出，比如家政或清洁用品。)

当然，不是每个人都遵守规则。一些工人更喜欢现金支付，因为他们可以在纳税申报单上做手脚。不过，请记住，美国国税局希望你报告你获得的所有收入，即使是兼职收入，即使是现金收入。

### 你可以扣除业务费用

从好的方面来说，你可以扣除与你的业务相关的某些费用。

假设你是优步或 Lyft 的司机。你可以把使用你的汽车作为业务费用扣除。你有两个选择:你可以扣除使用汽车的实际费用(如汽油费、维修费、租赁费)，或者你可以采取 [IRS 标准里程扣除](https://www.irs.gov/uac/newsroom/2016-standard-mileage-rates-for-business-medical-and-moving-announced) ，即每英里 54 美分(用于商业用途。)

不过，大多数司机可能会将车用于拼车和个人用途，所以值得指出的是，你只能扣除用于商务用途的费用。在这种情况下，整理费用可能会很棘手，但好的税务准备工具 像 [TurboTax](https://turbotax.intuit.com/) 和 [TaxAct](https://www.taxact.com/) 会带你穿过它们，好的税务专业人士 [也会有所帮助](http://twocents.lifehacker.com/find-a-good-tax-professional-with-the-irss-preparer-dir-1759761514) 。TurboTax [列出了其他几个](https://turbotax.intuit.com/tax-tools/tax-tips/Self-Employment-Taxes/Tax-Tips-for-Uber--Lyft--Sidecar-and-other-Car-Sharing-Drivers/INF28820.html) 司机可以拿的扣款:

> 你向拼车公司支付的佣金是一项业务支出，你为汽车安装的技术支付的任何费用也是如此。其他税收减免包括:
> 
> *   洗车和内部清洁
>     
> *   给乘客的水、口香糖或零食
>     
> *   过路费和停车费
>     
> 
> 此外，拼车公司通常要求使用智能手机。
> 
> *   可归因于拼车工作的手机费用部分可用于减少您的自雇收入。
>     
> *   为了简单起见，为工作配备一部专用电话可能是有意义的。

如果你在 Airbnb 上租了一个房间，你可以根据 IRS 扣除任何“管理、保存和维护你的租赁物业的普通和必要费用” [。这可能包括抵押贷款利息、财产税、折旧、维修、清洁费和保险等费用。还可以扣除 Airbnb 费用(以及](https://www.irs.gov/businesses/small-businesses-self-employed/tips-on-rental-real-estate-income-deductions-and-recordkeeping) [易贝和 Paypal 费用](http://www.internetbiztaxtips.com/2007/02/can-you-deduct-your-ebay-and-paypal-fees-on-your-tax-return/) ) [](http://www.internetbiztaxtips.com/2007/02/can-you-deduct-your-ebay-and-paypal-fees-on-your-tax-return/)作为广告费或银行服务费。

然而，如果你只出租你实际居住的一个房间或房子的一部分，你将不能 100%扣除你的费用。同样，你必须把它分成商业和个人用途，一个好的税务准备者或税务软件程序会帮你完成这个过程。

如果你以任何其他身份从事自由职业，你的扣除额将因业务而异，但如果你在家工作，不要忘记扣除你的家庭办公室扣除额 。这里还有一个复杂的规则要记住:你不想双重扣除。TurboTax [解释了这是如何工作的](https://turbotax.intuit.com/tax-tools/tax-tips/Self-Employment-Taxes/Top-6-Tax-Tips-for-Sharing-Economy-Freelancers/INF29465.html) :

> 例如，你是一名拥有家庭办公室的自由平面设计师，但你也通过 Airbnb 在夏天租了两个月的公寓。在那段时间里，公寓的租金是你 Airbnb 运营的可扣除费用，而不是你作为平面设计师的活动。

否则，一定要把所有能扣的都挤进去。[99 扣款](https://99deductions.com/) 可以帮忙。只需输入你做什么样的工作或提供什么样的服务，该工具就会填充一个与你的业务相关的扣除清单。

### 你可能需要缴纳季度税

每个人全年都要交税。同样，当你有一份标准的全职工作时，你的雇主会为你支付这些费用。叫做*员工代扣*。当你从一个不预扣税的公司、企业或个人那里赚钱时，这取决于你自己。

一年当中，自由职业者和独立承包人支付估计的季度税，这和他们听起来的差不多:每年 4 月、6 月、9 月和 1 月的 15 日支付估计的税款。如果你在纳税年度结束时没有支付足够的税款，你不仅会欠下额外的税款，你还需要支付的罚款。

另一方面，如果你在全职工作的同时还兼职做自由职业者，你可能就不用担心按季度付款了。 [国税局解释](https://www.irs.gov/businesses/small-businesses-self-employed/sharing-economy-tax-center) :

> 如果你也是一名雇员，你通常可以通过从你的薪水中扣缴更多的税款来避免估计的税款。例如，如果你的共享经济活动只是一份兼职或兼职业务，这可能是一个特别有吸引力的选择。为此，填写一份新的 [表格 W-4](https://www.irs.gov/uac/about-form-w4) 并交给你的雇主。

国税局让 [在线缴纳预估季度税](https://www.irs.gov/payments/direct-pay) ， [这个计算器](http://www.kiplinger.com/tool/taxes/T055-S001-tax-withholding-calculator-kiplinger/index.php) 可以帮你算出你欠了多少钱。

最后，美国国税局有一个非常详细的网页，你可能会觉得有用，那就是 [共享经济税务中心](https://www.irs.gov/businesses/small-businesses-self-employed/sharing-economy-tax-center) 。它涵盖了这里概述的所有基础知识，以及针对特定情况的一些附加的详细信息。
