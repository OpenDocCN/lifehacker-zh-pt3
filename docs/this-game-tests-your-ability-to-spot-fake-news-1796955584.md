# 这个游戏测试你辨别假新闻的能力

> 原文：<https://lifehacker.com/this-game-tests-your-ability-to-spot-fake-news-1796955584>

现在比以往任何时候都更重要的是，当你阅读#FakeNews 时，能够分辨出来。然而，确定某样东西是真是假并不总是容易的。你认为你了解你的东西吗？给 [人为的](http://factitious.augamestudio.com/#/) 一个尝试。

Watch

网页游戏展示了已经在网络上发布的真实新闻故事，以及来源信息。你的工作是确定你所读的是真的还是假的——这并不像看起来那么容易。

我玩游戏时看到的大部分故事都是我在脸书上看到过的。我喜欢认为自己很擅长辨别一个故事是否是伪造的，但我仍然没有完全弄清楚。

这个游戏是由美国大学的一个团队制作的，旨在向人们展示如何识别假新闻。玩它的工作原理很像火绒。你一次只能看到一个故事，看完之后你必须选择是或否。一旦你做出了选择，你就找到了答案和一些提示，告诉你如何才能做出正确的选择(假设你没有)。

花几分钟时间是一种有趣的方式，你(或者你发送这封信的家人)可能会在这个过程中学到一些东西。

来自人工团队的一些建议:总是检查一篇文章的来源，想想它的目的是什么，并在网站上寻找联系信息。如果网站上没有联系信息，那么这应该是一个危险信号。你也可以使用类似于[Snopes](http://www.snopes.com)[factcheck.org](http://factcheck.org)和[Washington Post Fact Checker](https://www.washingtonpost.com/news/fact-checker/)的网站来查看某个特定的故事是否已经被证实。
