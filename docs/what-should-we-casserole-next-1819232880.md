# 接下来我们应该做什么菜？

> 原文：<https://lifehacker.com/what-should-we-casserole-next-1819232880>

大家好，欢迎回到 [*将它砂锅*](https://lifehacker.com/tag/will-it-casserole) ，新的、 [复古周](https://lifehacker.com/c/retro-week)——在这个受启发的专栏里，我们看到了什么样的饭菜、菜肴和烹饪可以被重新想象和改造成砂锅。



鉴于你上周 有多少 [伟大的砂锅点子，我认为一期过后就停止砂锅是不公平的。上次没参加的，这个栏目办的很像](https://skillet.lifehacker.com/forget-sous-vide-this-week-were-going-to-see-what-will-1818677521#_ga=2.57020330.1420464300.1505696546-949419976.1446553382) [*会不会 Sous Vide？*](https://lifehacker.com/tag/will-it-sous-vide) :你会提出你认为我应该做成砂锅的东西，我会做最受欢迎的砂锅。如果可能的话，我想远离经典的、已经被证实的砂锅菜(比如绿豆),并且用它变得奇怪。这里有一些我认为会很好的概念:

*   **基督山砂锅:**这是 D·胡珀 建议的 [，这是个绝妙的主意。](https://skillet.lifehacker.com/1818763247)
*   菠萝比萨砂锅:因为这听起来真的很不错，而且我很自私。
*   **红龙虾砂锅:**我喜欢以中级连锁餐厅为蓝本做砂锅的创意。橄榄园也会很有趣。
*   [**寿司砂锅**](https://skillet.lifehacker.com/1818731629) **:** 我知道这是一个迷因，但 [真正的](http://www.justhungry.com/handbook/cooking-courses/japanese-cooking-101-lesson-2-bonus-sushi-rice-chirashizushi) 看起来相当不错。
*   **“我现在很高”砂锅:**只是一堆瘾君子食物的分层盘。披萨卷和奇多都会涉及到。

不过，我的想法已经够多了；我想听听你的意见。你希望看到什么样的菜肴、食物或抽象概念在砂锅里被做成单一的、分层的菜肴？
