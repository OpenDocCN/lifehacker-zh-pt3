# 管家来之前怎么收拾

> 原文：<https://lifehacker.com/how-to-clean-up-for-the-house-cleaner-1825178756>

“为清洁女工打扫卫生”这个古老的比喻并不完全愚蠢:虽然你应该把真正的清洁工作留给你雇来打扫的人，但你也应该打扫卫生——甚至可能做一些抽查。如果这是你的清洁工第一次来你家，你也应该准备一些说明。这会让你们两个都更开心。



Lifehacker 非常重视春季大扫除。我们绝不会错过更新、重组和整理我们家庭生活的机会。我们也非常兴奋地按下科技使用的重置按钮，仔细查看我们的财务状况，并抛弃已经有点发霉的日常习惯。欢迎来到 [*春季大扫除周*](https://lifehacker.com/tag/spring-cleaning-week) *，在这里我们清理冬天的蜘蛛网，为未来的晴天做准备。让我们收拾一下，好吗？*

首先，清理你的杂物。如果你雇人打扫你的家，而不是让 T2 整理它，那么你欠他们一个整理得相当好的家。否则，告诉他们可以忽略哪些区域或桩。(或 [聘请专业组织者](https://lifehacker.com/what-professional-organizers-really-do-and-how-they-ca-1663496231) )。)随意把你所有的杂物放在一个房间或一个表面上，然后告诉清洁工忽略那个空间。这是一个权衡:你希望你的地方打扫得越彻底，你需要准备的就越多(或付出的就越多)。

接下来，清理一个小东西:任何特别易碎的东西，或者至少任何你不相信你的清洁工能妥善处理的东西。任何简单的东西，比如放在外面的一次性咖啡杯。(那只是尊重。)以及任何你同意清洁工不负责的东西。我喜欢自己洗碗，所以在清洁工来之前我会处理好。清洁工的来访已经成为一项家务活的最后期限——一个特征，而不是一个缺陷。

如果你不知道你的工作和清洁工的工作之间的界限在哪里，写下两个清单:一个是你希望有人为你做的清洁任务，一个是你喜欢自己做的任务。你可以把这个展示给你的清洁工，最好是在你们还在讨论他们的时间投入和费用的时候。

你的清洁工第一次来你家的时候，你最好把整个地方走一遍，面对面地把过程说出来。如果做不到这一点，写下你的指示，如果需要几次访问才能把事情做好，不要生气。在别人家周围工作，学习什么该改变，什么不该改变，是一项复杂的任务，直到你的旧的 *GQ* 不见了，或者你的化妆架全乱了，你才会知道你有多在乎某些东西。

东西会被移动。盘子可能会被放在奇怪的地方，或者毛巾可能会在你准备好之前被扔进洗衣房。这完全取决于你对你的清洁工有多少要求，以及你和他们有多少交流。不要害怕礼貌地要求你的清洁工改变一些东西。(如果你要求他们在工作上花更多的时间，就相应地付给他们报酬。)

你的清洁工可能永远也不会知道每一件小东西该放在哪里。我家里有一个小鸟雕像，每次她来，我家清洁工都小心翼翼地调不正，因为她以为是一个戴帽子的人。因为我可能几个月都见不到清洁工本人，那个放错地方的小饰品一直愉快地提醒着我，她就在我身边。还有难以置信的做工精良的床。
