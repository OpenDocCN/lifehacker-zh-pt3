# 超越截止日期的“It Follows”模型

> 原文：<https://lifehacker.com/the-it-follows-model-for-outrunning-deadlines-1821533189>

“我喜欢截止日期，”《银河系漫游指南》作者道格拉斯·亚当斯说。"我喜欢它们经过时发出的嗖嗖声。"这里有一个思考长期期限的心理技巧，基于恐怖电影《T2》的小剧透。



正如电影中的几个场景所透露的那样，之后的*的前提是，一个不可战胜的怪物正在追杀英雄。如果传到她那里，她会死的。它走得很慢，慢得像走路一样——但它从不停止。*

她每睡一分钟，怪物就逼近一分钟。如果她想休息，她需要先把车开得远远的。她可以把怪物诅咒传给其他人，但是如果怪物杀了他们，它也会回来找她。

如果有一个那样的怪物跟着你，你会怎么做？你会等到最后一秒才逃跑吗？或者，你会在世界的另一端监视一个地方，并仔细跟踪你需要再次逃离的时间，以确保那个怪物永远不会出现在你的视线范围内？

你看这是如何影响你的截止日期的。每次你得到或设定一个截止日期，你都在得到一个怪物。错过最后期限可能不会杀死你，但会伤害你，不管“伤害”是指“自我感觉不好”还是“丢掉工作”。那你为什么要等到最后一刻才逃离怪物呢？

不像*里的怪物，它跟在*后面，你可以通过完成它，把你的最后期限永远送走。你可以授权，但是你最好相信其他人来完成。因为如果他们失败了，很可能也会牵连到你。

“在截止日期前处理你的工作”并不是一个新的想法，但是对我们中的一些人来说，把截止日期想象成可怕的诅咒会有所帮助。出去在那怪物杀死你之前杀死它。
