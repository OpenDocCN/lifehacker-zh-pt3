# 亚马逊 Echo Show 是我家最好的哑智能机

> 原文:[https://gizmodo . com/the-Amazon-echo-show-is-the-best-dumb-smart-machine-in-1796380588](https://gizmodo.com/the-amazon-echo-show-is-the-best-dumb-smart-machine-in-1796380588)

亚马逊回声展在我卧室的第一个晚上，我不得不小心地把它面朝下放在我的床头柜上。显示屏在我即将到来的会议、最新消息和天气中循环播放，简直太亮了。这就像有一个超级信息的小太阳照在我的脸上。第二天晚上，就像它知道一样，回声秀的屏幕上又出现了一张幻灯片。“试着说，Alexa，请勿打扰。’”

Watch

我试了一下，屏幕立刻变暗到一个完全可以睡觉的程度。Echo Show 确切地知道我想要什么，我什么时候想要它，当这个设备做到这一点时，它就是我现在家里最喜欢的小工具之一。但是，伙计，当亚马逊的新 Echo 表现愚蠢时，这款设备就是对破灭的期望的一次练习。

但这很难责怪 Echo Show，亚马逊的 [新款 230 美元组合智能音箱和显示屏](https://www.amazon.com/dp/B01J24C0TI?asc_campaign=InlineText&asc_refurl=https://gizmodo.com/the-amazon-echo-show-is-the-best-dumb-smart-machine-in-1796380588&asc_source=&hvadid=194260425524&hvdev=c&hvdvcmdl=&hvlocint=&hvlocphy=9004058&hvnetw=g&hvpone=&hvpos=1t1&hvptwo=&hvqmt=p&hvrand=2288158661390020605&hvtargid=kwd-310736619335&ref=pd_sl_1foyxz097s_e&tag=kinjagizmodolink-20) 。这款设备是亚马逊保持领先于谷歌和苹果等竞争对手的努力。智能音箱去年是积极的，今年，所有人(目前不包括苹果)都在专注于打造一个大屏幕的数字家庭助手。谷歌正在通过使用内置于其电视和 Nvidia Shield 的 Android 版本来实现这一点，但亚马逊正在将这一切放在一个可以放在书架上的小包装中。

这个计划做得如此之好，以至于它的失败就像是附带损害。凭借一个强大的扬声器和一个明亮的、令人惊讶的高质量显示器，Echo show 非常简洁地回答了我所有的简单问题，并以如此完美的可消费格式提供数据，我一直希望它能够做得更多。为什么*回声节目不能在狗行为怪异时给我的兽医打电话，或者从一个小的德州经销商那里找到(并订购)我最喜欢的莎莎酱品牌？我积极强化的期望让我发出越来越大的挑战，这不可避免地让这个设备走向失败。* 

就像手机的智能助手一样，Echo Show 总是主动提供信息。它会自动弹出最奇怪、最有趣的新闻、天气以及关于即将召开的会议的提醒。我告诉它给我看 CNN，几乎立刻，来自 CNN 的最新消息就出现在屏幕上。然而，回声节目的一些故意元素感觉像是疏忽。它不能去网站。它不会在谷歌或必应上进行图片搜索。如果你要求它给你找色情片，它会莫名其妙地建议你看亚马逊 Prime 上的春假特辑。

Echo Show 是围绕亚马逊生态系统建立的，所以它只会显示你的主要照片库中的图像和主要媒体库中的电影。如果你想要音乐，它只会从 Prime 图书馆播放，如果你想要一本书读给你听，它需要来自你的 Kindle 图书馆。实际上，这对于我们大多数人来说都是非常棒的——尤其是如果你像我一样是亚马逊迷的话。我不知道你怎么想，但我真正使用我的旧 Echo 或我的 Google Home 的唯一时间是查看天气或时间，为狗播放音乐，设置定时器，或关闭我所有的智能灯。

对于所有这些事情，回声显示是了不起的。我总是问天气怎么样，然后在得到答案之前就走开了。Echo Show 显示未来几天的天气预报，并在屏幕上显示所有信息，直到我问它一个新问题，当我设置一个计时器时，它实际上显示计时器，所以我不必每 30 秒就问我还剩多长时间。我甚至可以轻扫一下就让计时器消失，而不是试图在闹铃声中大喊“停止”。该显示器甚至可以让我触摸控制我的音乐，提供播放、暂停和跳过按钮。

所有这些小福利都足够有用，我真的不介意在传统的回声节目上多花 50 美元。但除了作为一个小工具更有用之外，它也不那么碍眼。“哦，这比那个黑色的大管子好多了，”我的室友注意到它时说道。旧的 Echo 藏在路由器后面，但 Echo Show 像 Google Home 一样，是那些数字家庭助手之一，我不介意在客厅或卧室占据一个突出的位置——即使它的超级锐利的黑色边缘给它一种明显的 90 年代锐利的图像氛围。

而且让我感觉自己是在未来！我知道，我明白——像 Echo Show 中发现的那种 7 英寸触摸显示器比以往任何时候都更便宜，任何一个拥有旧手机、笔记本电脑或平板电脑的人，只需一点时间和编程知识，就可能制作出几乎与 Echo Show 一样有效的东西。见鬼，如果你有最新的 Android 电视或新的 Nvidia Shield，你甚至可以从你的电视上获得类似的谷歌风格的体验——尽管没有触摸控制。

Echo Show 正在做的事情并不具有开创性，但直到现在，如果你想拥有一个超级智能的触摸屏大脑来管理你的家，并提供天气和日程安排等信息，你要么自己动手，要么像马克·扎克伯格(Mark Zuckerberg)那样花费大量现金。Echo Show 是我们其他人可以拥有的第一台此类设备，在许多方面，它感觉像是迈向**杰特森*未来的第一步。*

*它甚至有一个用于视频通话的摄像头！尽管它目前只能在其他支持 Alexa 的设备上运行，包括 Echo 和手机上的 Alexa 应用程序。这是一个简洁的概念，使用起来很有趣。当我在办公室测试回声节目时，我大喊“打电话回家”，我公寓里的回声立即吓了我的狗和我的室友一跳，能够简单地说“打电话给妈妈”，让你妈妈的脸出现在屏幕上，感觉就像是《星际迷航》中的未来奇迹。但这也意味着现在亚马逊不只是在听你做的每一件事，它也在*看着*你做的每一件事，而且能感觉到...令人毛骨悚然。尤其是如果你把回声秀放在卧室里。*

*然而，Echo Show 的最大问题更多的是亚马逊 Alexa 基础设施的结果，而不是设备本身的任何特定缺陷。我对《回声报》的每一个批评都同样适用于原版《回声报》。与该设备一样，Echo Show 过度依赖技能——迷你应用程序允许 Alexa 从精心构建的生态系统中移出，进入其他网站和信息库。如果这种技能不存在，或者在你提出请求时没有具体说明，Echo Show 就会装聋作哑，导航到亚马逊 Prime videos 而不是 YouTube，或者通过亚马逊商店而不是 Fandango 查找电影时间。*

*有时它也会误解单词。我总是小心翼翼地发音，因为数字助理可能很愚蠢，但当我告诉设备“给我看乔·T·加西亚·萨尔萨中等 picante，”它听到“给我看一个笑话加西亚·萨尔萨中等 picante，”当我说“给我看 GOOP”，它听到“给我看组。”*

*不过，如果你使用 Amazon Echo，这些问题你已经很熟悉了。所以他们感觉更像是我喜欢的设备的吹毛求疵。如果你觉得有必要从你的旧 Echo 升级，或者如果你终于准备好为你的家庭体验第一个真正的消费级数字大脑，那么是的，你真的需要立即订购亚马逊 Echo 秀，它很好。但它可以变得更好，除非亚马逊改善操作系统和技能之间的交互，并使 Alexa 更加直观，否则它仍将与我准备好的未来相去甚远。*
