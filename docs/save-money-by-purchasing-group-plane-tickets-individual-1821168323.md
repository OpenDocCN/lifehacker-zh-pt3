# 通过单独购买团体机票来省钱

> 原文：<https://lifehacker.com/save-money-by-purchasing-group-plane-tickets-individual-1821168323>

每当我和一大群朋友计划一次大型旅行时，我通常会主动购买机票。信用卡积分是件严肃的事情，在我的旅程中加入几个朋友意味着他们可以利用我的航空公司身份，这取决于我们选择乘坐哪一家。这似乎一直是一个显而易见的问题，除了本周 [*惊悚片*指出](https://www.thrillist.com/travel/nation/airline-secrets-tips-and-tricks) 我可能多付了钱。

Watch

你知道，当你在寻找航班时，你看到 a “这个价格只剩两个座位了！”价格旁边的便条？如果只剩下两张 99 美元的票，你去买了五张票，你不会得到两张 99 美元的票和三张正常价格的票，你可能会以正常价格买五张票，把两张 99 美元的票留在桌子上给别人买。

原因是大多数航空公司认为你的购买有点像一张运动票，他们把五张票一起卖给你。如果他们没有五张更便宜的票，那么他们会默认卖给你五张更贵的票。

里程可能会因航空公司而异，但如果你是一个大的团体购买机票，它(字面上)看一下你可能为单个机票而不是一个大团体支付的价格。你可以多刷几次信用卡，而不是一次买很多东西，这样可以节省一些现金。
