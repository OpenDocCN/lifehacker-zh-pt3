# 这个比利时食物金字塔是健康饮食的一个方便的小抄

> 原文：<https://lifehacker.com/this-belgian-food-pyramid-is-a-handy-cheat-sheet-for-he-1819105945>

健康饮食的方式有很多，但大多数专家都认为糖果、薯条、啤酒和熏肉应该尽量少吃。比利时新的食物金字塔 将这些食物及时推出，而蔬菜和全谷物成为焦点。

其结果是一个“非常符合佛兰德饮食文化的现实模型”，但它对美国人也很适用。(我们自己的版本，[)my plate](https://www.choosemyplate.gov/)，通过不命名任何特定的食物来回避政治问题。)比利时金字塔的最大部分是纯素食者的乐园:水果和蔬菜、豆类、豆腐、全麦面包和意大利面、坚果和种子。石油也属于这一类。该网站是荷兰语的，但它可以观看 [这个视频](https://www.youtube.com/watch?time_continue=101&v=DpRuQsGr6aY) 并将字幕切换到自动翻译/英语:

下一条包括鱼、家禽、奶制品和鸡蛋。金字塔的顶端——仍然可以吃，但是比其他的“更少”——含有黄油和红肉。然后所有你已经知道对你来说不太好的东西都在另一个部分。不要认为这些是你健康饮食的一部分，图表似乎在说，但如果你真的无法抗拒，你知道在哪里可以找到它们。
