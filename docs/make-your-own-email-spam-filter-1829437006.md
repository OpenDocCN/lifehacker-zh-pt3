# 制作您自己的垃圾邮件过滤器

> 原文：<https://lifehacker.com/make-your-own-email-spam-filter-1829437006>

如果 Gmail 的垃圾邮件和时事通讯过滤器漏过了太多，或者如果你使用的私人电子邮件服务的垃圾邮件过滤器不好，那么试试这个解决方案: [隔离所有电子邮件，直到发件人确认它们不是垃圾邮件](http://blog.nawaz.org/posts/2018/Sep/solving-my-email-problem/) 。这是你自制的垃圾邮件过滤器。

Blogger Beetle B .将他们的电子邮件服务器设置为自动回复来自新发件人的每封电子邮件。回复中包含一个链接，该链接指向一个网站，发件人必须输入他们的电子邮件地址以确认他们不是垃圾邮件发送者。

这对发件人来说有点痛苦，但不是巨大的痛苦，所以这并没有阻止甲虫 b 收到任何重要的电子邮件。他们已经使用这个系统一年多了，到目前为止，它运行得很好，足以让他们 [向任何对垃圾邮件和低优先级电子邮件感到沮丧的人推荐它](http://blog.nawaz.org/posts/2018/Sep/solving-my-email-problem/) 。在接下来的中，他们列出了几个服务，可以帮助你轻松创建自己的自动回复。

如果发件人没有填写表单，他们的电子邮件就会被发送到隔离文件夹，就像典型的垃圾邮件过滤器一样。甲壳虫 b .每天检查他们的邮件，仍然发现这比老方法好得多——在老方法中，他们会被埋在不重要的邮件中，几个月都没有回复重要的邮件。

如果你担心重要邮件无法通过，你可以用白名单来缓解。当甲壳虫 b .启动他们的系统时，他们首先将所有发邮件的人列入白名单。您可以设置自己的白名单标准。甲壳虫 b .将他们的白名单做得很大，最初会让一些不需要的简讯进来，但当这些简讯进来时，他们会将它们从白名单中删除。
