# 你可以在火鸡吃完很久以后再用蔓越莓酱

> 原文：<https://lifehacker.com/you-can-use-cranberry-sauce-long-after-the-turkey-is-go-1821480716>

一罐吃了一半的果冻蔓越莓酱是一种令人悲伤的景象，但没有理由扔掉这种又酸又甜的物质，即使你吃了最后一点火鸡。

首先，蔓越莓酱可以保存相当长一段时间(长达三周)，它有各种各样的应用，既有咸的也有甜的(还有酒)。以下是我最喜欢的一些使用剩余蔓越莓酱的方法，不管是自制的还是罐装的:

*   [**精心调制鸡尾酒**](https://skillet.lifehacker.com/3-ingredient-happy-hour-the-cranberry-sauced-1820097880) **:** 一勺蔓越莓酱增加了亮度和酸度，所有这些果胶让你的饮料口感非常圆润。
*   **把它倒在你的薄煎饼和华夫饼上:**这和自制的全浆果酱一起吃效果特别好。
*   给你的肉浇上酱料:你可以把它涂在火腿、羊腿甚至烤肉上，或者你可以把它加到你最喜欢的一瓶烤肉酱里，增加额外的魅力。想在 [你的锅酱](https://skillet.lifehacker.com/how-to-make-a-freaking-pan-sauce-1795277933) 里来一阵果味？在最后扔出一大勺。
*   将其混入纯酸奶中:更甜的蔓越莓酱在这里效果最好，因为纯酸奶本身就很酸。
*   **和奶酪一起吃:**只需 [把它放在一大圈融化的、粘稠的布里干酪](https://skillet.lifehacker.com/how-to-hack-the-ikea-food-market-1755817567) 上面，或者在烧烤前把它涂在奶酪三明治的里面。它也是令人愉快的带有一点蓝色的三色点心。
*   **用作冰淇淋配料:**加入一些略带酸味的酱汁，让你最喜欢的乳制品甜点更有节日气氛。

此外，不要低估它作为简单的三明治涂抹的力量。虽然它真的与切片火鸡一起唱，但它与火腿、鸡肉或一些非常好的奶酪一起唱也不错。
