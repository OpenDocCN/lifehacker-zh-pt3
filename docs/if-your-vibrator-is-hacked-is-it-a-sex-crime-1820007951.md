# 如果你的振动器被黑了，算不算性犯罪？

> 原文：<https://gizmodo.com/if-your-vibrator-is-hacked-is-it-a-sex-crime-1820007951>

在最近一次去柏林的旅行中，亚历克斯·洛马斯的熟人向他提出了一个挑战:你能在野外找到一个支持蓝牙的插头吗？你能在没有主人帮助的情况下打开它吗？洛马斯是英国网络安全公司 [Pen Test Partners](https://www.pentestpartners.com/) 的渗透测试员，他拿出手机，咨询检测应用 LightBlue，很快确定了一个 Lovense Hush， [据称是](https://www.lovense.com/vibrating-butt-plug) “市场上最强大的振动按钮”，洛马斯说它位于一个陌生人的尾部。更重要的是，这种沉默是脆弱的，任何知道怎么做的人都可以入侵。

随着世界朝着完全应用连接的方向发展，我们的设备可以做的事情和法律可以解决的事情之间的差距越来越大，特别是在远程通信方面——或者说，你可以通过互联网远程控制的性技术。像 Lomas 确定的这种性玩具黑客情况不太可能在实验室之外发生，但将振动器连接到互联网打开了可能性，我们应该准备好讨论它。

洛马斯在 [Pen Test Partners 博客](https://www.pentestpartners.com/security-blog/screwdriving-locating-and-exploiting-smart-adult-toys/) 上发布了他的实验结果，并创造了“螺丝驾驶”这个术语，这是一种关于 wardriving(或驾车窃取他人 wi-fi)的性化游戏。在接受 Gizmodo 的 Skype 采访时，他用通俗的语言总结了这一过程:Hush 使用蓝牙低能耗，基本上是更现代的蓝牙版本，来连接智能设备。如果你在公共场合戴着烟蒂塞，一个指定的伙伴站在你的臀部 30 英尺以内，那么这个伙伴可以通过他们的手机谨慎地控制它的振动速度和模式。洛马斯说，这很好，除非那个人离开了(公认有限的)连接范围。在这种情况下，Hush“将在某种程度上自动进入发现模式，准备好让其他人发现并控制，”与插头配对——没有密码保护，或者 PIN 是很容易猜到的 0000 或 1234——不请自来地体验你的肛门体验。(在一封电子邮件中，Lovense 的代表解释说确实是这样，尽管这个玩具确实有一个功能，如果连接的设备超出范围，它会自动关闭。洛马斯指出，客户必须知道这一切都是可能的，但许多人不知道。)

洛马斯并没有与寂静同步并调高振动，但他可以做到，这就是问题所在。一个消费者可能冒险进入这个世界，打算与一个人进行秘密的性爱体验，但最终却与另一个人完全拥有了 telesex。但是这到底是什么样的犯罪——网络犯罪、性犯罪还是某种新型的混合犯罪？有人有能力处理它吗？

随着性玩具易受攻击的新闻越来越频繁，答案似乎就在附近，一点也不奇怪。白帽黑客已经暴露了许多成人公司——[【WeVibe](https://gizmodo.com/smart-sex-toy-maker-agrees-to-pay-customers-10k-each-f-1793240775)——作为他们一直在收集的令人惊讶的详细的私人用户数据的不稳定存储库，这些数据大多不为他们的客户所知。WeVibe 的数据不安全导致了 [侵犯隐私诉讼](http://fortune.com/2017/03/10/sex-toy-maker-settlement-smart-vibrator-lawsuit/) 和适度的和解，然而随机第三方可以插入 Skype 或 Chaturbate 等凸轮平台上的相互手淫会话的可能性却没有得到广泛讨论。嘘并不是唯一可攻击的玩具:几乎任何 ble 功能的玩具(或者实际上是设备，无论是助听器还是烟雾探测器)都可以被打开，接受外界的探测。连接到 [应用程序的产品，如身体聊天](http://elladodelnovato.blogspot.com.es/2017/07/understanding-and-breaking-internet-of.html) 似乎很容易受到外界干预，而配备摄像头的 Siime 眼部振动器 [很容易被任何知道内情的人劫持](http://www.bbc.com/news/technology-39490391) ，有可能向陌生人提供你生殖器的生动详细的视图。受害者当然可以声称 [侵犯隐私](http://injury.findlaw.com/torts-and-personal-injuries/invasion-of-privacy.html) ，但是这种程度的侵犯似乎更为严重。

公平地说，不受欢迎的第三方入侵性玩具的可能性非常小:正如 Lovense 在对 Lomas 实验的回应中以及在与 Gizmodo(性玩具制造商联系的物联网，Lovense 是唯一一个回应者)的电子邮件交流中解释的那样，Hush 一次只能连接一个设备，而螺丝驱动需要对 BLE 和“Lovense 协议”的复杂知识，以及大多数人不具备的“BLE 嗅探硬件”。即使有人设法扑向你的屁股插头失效的 BLE 连接，他们也需要非常近:在 30 英尺内和“清晰的视线”，所以，可能会跟着你。但购买远程蓝牙发射器和接收器是可能的，洛马斯报告说，在出版后，许多读者在他的推特上说，他们成功地通过共享墙定位了邻居的玩具。

洛马斯承认，一些嘘买家可能会成为一个陌生人的偷偷摸摸的参与，这是非常好的；在他看来，问题在于普通消费者可能不会意识到他们已经同意了一种半私密的体验——他们“本质上是带着一个巨大的屁股插头发射器四处走动”，广播他们的肛门，或者无意中提供一次阴道内的望远镜之旅。

事实上，从法律的角度来考虑远程黑客，同意应该是等式的一个重要部分:本能地，一个陌生人用生殖器振动给你惊喜会被认为是一种侵犯。从法律上讲，性侵不需要插入，仅仅是“未经接受者明确同意而发生的性接触或行为”据华盛顿特区的辩护律师、前联邦性犯罪检察官 Shanlon Wu 称，未经同意就像远程控制、被黑客攻击的性玩具所导致的结果一样，是性侵犯的信号。

吴说，“重罪型性虐待的典型定义是无意识的插入，”无论是身体部位还是物体。至于后者，如果是手或设备控制物体的运动，他看不到法律等式的改变。吴承认，一些律师可能会在虚拟犯罪方面陷入困境，并认为佩戴远程设备是对其使用的全面同意。但是同意是不可转让的，他说。

吴打了个比方:“如果我参加一场拳击比赛...很明显，我同意和我的对手竞争。如果他打我，我不能大喊，'哦，他袭击了我，他打了我！'因为我们同意打对方。但如果他的角球人，他的经纪人，在比赛中出来打我的头，他们不能争辩说，'你同意了拳击比赛，所以任何人都可以揍你。'“同样，如果你同意某人在你身上使用性玩具，这并不是邀请任何路人加入进来。

“同意就是同意，不管是亲自同意还是远程同意，我认为这是应该关注的事情，”吴说。他认为这种形式的网络窃取行为是一种直接的性侵犯起诉，但斯图尔特·贝克——律师事务所 Steptoe & Johnson 的合伙人，他的业务涵盖网络法和技术相关问题——不同意这种观点。

贝克告诉 Gizmodo “我很难将这一点准确地纳入性犯罪框架中。“如果有人闯入你的假阳具，他们要负刑事责任，”他说，但问题是怎么做。

虽然贝克同意振动器劫持扭曲了同意的概念，但他也推测，将它作为性犯罪审判可能会引发关于同意的伴侣参与的复杂问题。如果有问题的性玩具带有内置摄像头，这可能会牵连到它的主人，这种方式不会让很多人满意:贝克指出，青少年之间两厢情愿的性短信已经转化为几起[儿童色情](https://www.washingtonpost.com/news/morning-mix/wp/2015/09/21/n-c-just-prosecuted-a-teenage-couple-for-making-child-porn-of-themselves/?utm_term=.8a10a2e77569) 起诉，如果两个未成年人在 Skype 或任何其他互联网连接的视频平台上互相使用装有摄像头的振动器，他们可能会无意中陷入类似的法律伤害世界。贝克认为，最明确的前进道路是根据 1986 年的《计算机欺诈和滥用法案》( Computer Fraud and Abuse Act ),将螺丝驱动作为一种网络犯罪提起诉讼，该法案包括所有有意未经授权访问计算机以及窃取其内容的行为。虽然它没有具体处理远程控制，但 CFAA 可以说提供了一种在网络环境中表达同意的方式。

“被授权和得到同意之间的区别微乎其微，”贝克说，“所以如果你没有权力用别人的假阳具做什么，那么如果你是通过互联网远程操作，你就犯了罪，可能会变成重罪(根据 CFAA)。”

但是，谁可能不承担责任呢？贝克说，制造商，除非他们以某种方式歪曲了产品。(Gizmodo 采访的 Lovense 代表表示，他们将与首席执行官讨论在产品包装上添加澄清标签的想法。)虽然民事诉讼是由玩具制造商不安全的数据收集方法引起的，但当涉及 telesex 黑客攻击时，唯一的责任人是黑客。这意味着要求制造商和法律找出解决性玩具漏洞的方法是合理的。

对吴和贝克来说，螺丝起子案件仍然属于假设的范畴，对起诉这种罪行的一些分歧可能源于缺乏先例。一个 [CFAA 侵犯](https://www.eff.org/deeplinks/2013/03/41-months-weev-understanding-how-sentencing-guidelines-work-cfaa-cases-0) 和一个 [性侵](http://criminal.findlaw.com/criminal-charges/sexual-assault-penalties-and-sentencing.html) 都是重罪，不过，他们可能的刑期相差很大。可以说，更重要的是将性玩具劫持视为与电脑相关的犯罪，而不是针对个人的犯罪。这样做的风险是最大限度地减少最终取决于未经请求的亲密接触的犯罪行为，一名律师认为在公共场合佩戴像 Hush 这样的设备会导致未经授权的使用，这是受害者的指责。

然而，螺丝刀的法律方法可能取决于现实生活中受害者的具体情况，随着性技术越来越多地转向物联网连接——与应用程序同步，虚拟现实自慰会话，触发跨国伴侣的振动器——制造商不会停下来修补安全漏洞，预计他们会这样做似乎是合理的。虽然现在可能还不是苦恼黑客是否在你的 Skype 性爱会话中伺机而动，随时准备劫持你的振动器的时候，但也许是时候开始思考性犯罪的未来了。现在比我们到达后要好。
