# 用这些像素艺术 gif 学习游戏动画

> 原文：<https://lifehacker.com/learn-game-animation-with-these-pixel-art-gifs-1795833479>

游戏制作人(同时也是游戏工作室 [MiniBoss](http://blog.studiominiboss.com/) 的联合创始人)佩德罗·梅德罗斯懂像素艺术，他想教你。自 9 月份以来，每周二他都会发布一张动画 GIF，展示动画的一个方面。



有些专注于像素艺术，像 [清理角落](https://www.patreon.com/posts/pixel-art-part-2-11225146) 或 [模拟小于一个像素的运动](https://www.patreon.com/posts/1-pixel-movement-7652033) 。其他人解释更一般的艺术技巧像 [绘制废墟](https://www.patreon.com/posts/ruins-tutorial-8183989) 或 [制作火焰](https://www.patreon.com/posts/fire-tutorial-7293859) 。还有一些是针对游戏设计方面的，像 [关卡设计](https://www.patreon.com/posts/level-design-7190330) 和 [角色设计](https://www.patreon.com/posts/character-design-7530899) 。

每一个技巧都是如此具体，却又如此优雅和简洁，无论你是否打算亲自尝试，读起来都很有趣。这就像翻阅一本动画烹饪书。梅代罗斯的 gif 属于动画教程佳能与可汗学院的 [皮克斯在一个盒子里的](https://www.khanacademy.org/partner-content/pixar) 系列或者迪士尼的 [12 动画原理](https://digg.com/2015/12-principles-animation-ollie-johnston-frank-thomas-alan-becker) 。

在 Medeiros 的 [Patreon feed](https://www.patreon.com/saint11/posts?tag=tutorial) 上免费查看所有 gif。
