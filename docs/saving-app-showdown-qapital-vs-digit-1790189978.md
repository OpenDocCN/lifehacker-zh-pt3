# 拯救应用摊牌:资本与数字

> 原文：<https://lifehacker.com/saving-app-showdown-qapital-vs-digit-1790189978>

你可能听过一句老话，“先支付自己。”这个想法是在你有机会花钱之前把钱存起来。像 Qapital 和 Digit 这样的应用程序使 [很容易做到](https://lifehacker.com/these-apps-are-the-perfect-money-saving-tools-for-lazy-1778298912) 。他们审查你的消费，并自动存入你的储蓄账户。你设置好了，忘了它，你的积蓄会随着时间而增长。我们来对比一下这两者。



### 竞争者

无论是 [卡塔尔](https://www.qapital.com/) 还是 [数字](https://digit.co/) [借助自动化](http://twocents.lifehacker.com/qapital-boosts-your-savings-goals-with-the-power-of-aut-1775769618) 的力量提升您的储蓄。它们在 iOS 和 Android 上都是免费的。Qapital 严格来说是一个智能手机应用程序，但 Digit 允许你用他们的 web 应用程序从桌面登录。使用这两个工具，你可以通过应用程序开立一个新的银行账户，还可以链接你的当前账户。这样，应用程序可以审查您的交易，以触发这些自动储蓄。

两者都不能提供惊人的利率，但这不是重点。你不会获得丰厚的回报，但这些不是投资工具。它们只是为了帮助你拯救。把它们想象成数字化的自动存钱罐，而不是投资工具。以下是每种方法的工作原理:

*   **卡塔尔投资局**:卡塔尔投资局 [就像是 IFTTT 的金钱版。](https://lifehacker.com/qapital-boosts-your-savings-goals-with-the-power-of-aut-1775769618) 事实上，它连接到 IFTTT 以获得更大的灵活性，但我们稍后会谈到这一点。你可以为转账金额和时间以及储蓄目标制定规则，应用程序会读取你的交易，这样你就可以根据自己的消费习惯进行储蓄。例如，您可以在特定商店消费的任何时间节省一定的金额。或者存下一定比例的存款，这笔钱一落地就立刻打到你的账户上。
*   **数字**:数字简单一点。你连接了你的账户，但是 Digit 并没有自己创建规则，而是根据你能存多少钱来存。它会查看你的存款、即将到来的账单和消费习惯，计算出你每隔几天可以存多少钱。随着你收入和支出的变化，它会自动调整你的储蓄以与之匹配。

Qapital 和 Digit 都使用 SSL 加密来保护您的数据安全，它们不会将您的信息存储在它们的服务器上。总的来说，如果你对网上银行感到舒适，你应该对使用应用感到舒适。你的基金也有联邦存款保险公司的保险。

### Digit 使用算法进行“智能”存款

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-9GjBcMIIqS0&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-9GjBcMIIqS0&start=0) 

Digit 的主要吸引力在于它很聪明。它使用算法来分析你的支出和收入历史，然后根据自己的计算来决定你能存多少钱。甚至还附带了 [透支保护](https://help.digit.co/hc/en-us/articles/205360247-I-ve-received-an-overdraft-because-of-Digit-how-do-I-get-reimbursed-for-this-) 。然而，我已经用了一段时间，它从来没有接近过透支我的账户。

Digit 像魔术一样工作，因为你永远都不需要考虑它，突然你的 Digit 账户里就有了 100 美元。你甚至不必设置任何规则，尽管你可以告诉应用程序多存或少存，它会相应地调整它的算法。

当我几年前开始使用它的时候，我并不认为我能从中得到什么。我已经在我的个人退休帐户 和传统储蓄帐户中设置了 [自动储蓄，所以我认为我的预算已经相当优化了。然而，不知何故，Digit 找到了存钱的机会，自从加入以来，我已经存了 1787.47 美元。最棒的是，我从来没有错过钱，这似乎很疯狂，因为我是一个相当细致的预算者。这笔钱就这样从我的账户上消失了，所以我一直没有机会花掉它。](https://lifehacker.com/a-beginner-s-guide-to-opening-an-ira-1607498930)

Digit 与您的常规自动储蓄程序结合使用时效果最佳。这样，你可以确保达到你的储蓄目标，同时你还能找到额外的储蓄。我最喜欢的使用数字的方式是通过文本。不用下载应用程序，你只需发短信给 Digit(当你加入时，你会得到他们的号码)并输入一系列命令:暂停储蓄、取款、最近交易等等。

### 资本给你更多的灵活性

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-J2lvBfSWVHg&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-J2lvBfSWVHg&start=0) 

Qapital 稍微复杂一点，但也更有趣一点。一旦你注册了这个应用程序，你就创建了规则和目标。您的目标可能是:

*   [拨出税金](http://lifehacker.com/pay-your-estimated-taxes-online-268406)
*   [创建未雨绸缪基金](http://lifehacker.com/where-to-stash-your-emergency-fund-for-the-best-rates-a-5891971)
*   [为还债而储蓄](http://lifehacker.com/how-to-pay-off-your-debt-using-the-stack-method-576070292)

你只有一个 Qapital 账户，但该应用程序允许你在该账户内创建多个储蓄目标。你用规则为你的目标提供资金，而规则是由特定的行动触发的，比如在某个商店花钱或者得到报酬。Qapital 提出了一些现成的规则:

*   **四舍五入法则**:每次买东西自动存钱。
*   **少花钱规则:**当你花的钱少于你的预算时，省下差价(即“一周内在星巴克花少于 15 美元，省下差价。”)
*   **罪恶感快感法则**:在特定店铺消费时节省一定金额。
*   **设定&忘记规则**:每天、每周或每月存一笔特定的固定金额。
*   自由职业者 **法则**:发工资时留出一定比例的作为税。

当你建立一个规则时，你将为它指定一个目标。例如，你可以建立一个“假期”目标，然后制定一个规则，从每份薪水中留出一定的金额(或百分比)来实现这个目标。您也可以与其他qacapital用户分享目标。

今年，我用我的资本账户设定了几个目标。我想控制我的餐馆消费，所以我设立了一个“餐馆”目标，给自己一个外出就餐的预算。然后，我建立了一个规则，每次我在杂货上花费的时候，在这个基金里存下一定的金额。这样，我鼓励自己去杂货店购物，同时为外出就餐做预算。我用一个坏习惯来激发一个更好的习惯。

我还设定了一个目标，为我的蜜月存钱。作为我们八月金钱挑战 的一部分，我设定了一个目标，将我购买的所有东西四舍五入到最接近的美元，然后将零钱存进蜜月基金。

更好的是，该应用程序与 [连接，如果这样，那么与](http://twocents.lifehacker.com/the-best-ifttt-recipes-for-saving-and-managing-your-mon-1762248269) (IFTTT)连接，以获得更大的灵活性。通过 IFTTT，你可以将 Qapital 链接到任何数量的其他应用程序:Twitter、Gmail、Todoist、Wordpress。一旦你连接了这些应用，你的可能性几乎是无限的。您可以创建如下规则:

*   [下雨时自动存钱](https://ifttt.com/applets/329327p-automatically-save-a-set-amount-to-your-qapital-account-whenever-it-rains)
*   [使用某个标签](https://ifttt.com/applets/291076p-monetize-your-pet-s-cuteness-save-money-with-qapital-whenever-you-instagram-them-with-a-hashtag) 时自动省钱
*   [达到健身目标时自动存钱](https://ifttt.com/applets/291919p-reward-your-tired-feet-when-you-reach-your-distance-goal)

这为您提供了令人惊叹的定制和灵活性。这比数字操作多一点，但也让你对自己的储蓄有更多的控制权。数字在某些方面有局限性。例如，你不能告诉它存多少钱。而且你只能暂停存款 30 天。

### 结论: Qapital 有趣且可定制，Digit 简单易用

如果你不喜欢过多考虑你的预算或储蓄习惯，Digit 可能是你最好的选择。它会帮你做所有的工作，包括计算你能存多少钱，这太不可思议了。然而，如果你是一个更实际的预算者，并且你真的喜欢创建和跟踪你的金钱目标，那么 Qapital 可能是一个更好的选择。你可以创造一些有趣的、基于金钱的目标，并从中获得创造力。

我也同时使用了两者。我使用 Qapital 来帮助实现特定的目标，然后利用 Digit 从我的预算中挤出任何额外的节省。这两款应用都有独特的卖点，但它们对于一个共同的目标都非常有用:支持更好的储蓄习惯。
