# 使用马苏里拉乳清和羊乳酪盐水制作一批额外美味的米饭

> 原文：<https://lifehacker.com/use-mozzarella-whey-and-feta-brine-to-make-an-extra-tas-1819858470>

如果你喜欢新鲜的马苏里拉奶酪或羊乳酪，毫无疑问你会发现自己有一两个装着你珍贵奶酪的有点浑浊的水的容器。你可以把这种略带咸味的奶油状液体倒入下水道，或者把它作为美味的烹饪液体。

除了添加到汤和高汤中，马苏里拉乳清和羊乳酪盐水都可以(也应该)添加到你的下一批米饭中，无论你是在炉子 上烹饪还是在 [电饭煲](https://skillet.lifehacker.com/how-to-cook-a-single-cup-of-rice-in-your-instant-pot-1798322749) 中烹饪。至于乳清，用乳清代替普通的自来水，像平常一样煮米饭。盐水可能需要稀释，这取决于它有多咸。马苏里拉乳清将赋予该批次一种微妙的，轻微的奶油风味，而羊乳酪盐水将添加更多明显的臭味和盐。(如果你用的是羊乳酪盐水，那就省略掉你通常会加入锅中的盐，以免你被钠淹没。)

除了米饭，这两种与乳制品相近的液体都可以添加到面食水、面包或比萨饼面团以及汤锅中。见鬼，如果你存够了——把它放在冰箱里，直到你收集了几杯——[羊乳酪盐水是鸡肉卤汁](https://skillet.lifehacker.com/add-feta-cheese-to-your-brine-for-delicious-juicy-chic-1782694195) 的绝佳基础。
