# 马蒂·马西森的博洛尼亚碗非常令人满意

> 原文：<https://lifehacker.com/matty-mathesons-bologna-bowl-is-very-satisfying-1829668475>

我的审美——无论是关于食物还是总体而言——可以用“好色的浣熊”来形容虽然我以弄乱品尝菜单而闻名，但我对美国奶酪和博洛尼亚等加工美食的热爱是真诚的，并将继续支持具有类似倾向的厨师和美食人士。

Watch

我敢肯定没有人对此感到惊讶，但我非常喜欢多伦多厨师和 Viceland 明星 Matty Matheson(迫不及待地想得到他的新食谱)。他不仅看起来像一个热情、令人愉快的人，有很多 t 恤可供选择，他的食谱也很中肯，而且他不害怕其他更有阶级观念的人可能会避之不及的“恶心”成分。(老实说，如果你吃的是 mortadella，而不是 bologna，那你就是个骗子，应该为自己感到难过。)

总之，该死的博洛尼亚碗，马蒂从他岳母那里学来的，是最方便的早餐，也是我唯一真正有兴趣吃的早餐碗。(思慕雪碗正式结束。)我唯一的批评？他没有称之为“Bowlogna”

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-rllK0NBtk-w&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-rllK0NBtk-w&start=0) 

让 Matty 给你演示一下上面视频中的食谱，但是它非常简单。拿一个碗，放四片腊肠，扔一片美国奶酪进去，轻轻地在上面打一个鸡蛋。微波三分钟，和涂黄油的吐司士兵一起吃。这是完美的，但它也非常重复。剩下的烤土豆、蘑菇和焦糖洋葱会很好吃——只要把它们放在奶酪上面但在鸡蛋下面——我的 bff [炒韭菜](https://lifehacker.com/fried-leek-greens-are-excellent-on-scrambled-eggs-1825902069) 也会很好吃。如果你把它放在煎蛋卷里，你可以把它放在博洛尼亚碗里。
