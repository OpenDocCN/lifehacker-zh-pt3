# 为什么阴谋论如此吸引人

> 原文：<https://lifehacker.com/why-conspiracy-theories-are-so-appealing-1794604180>

戴上你的锡纸帽，用一块胶带盖住你的网络摄像头，等待即将到来的蜥蜴人，因为现在是一些阴谋论的时候了。超过一半的美国成年人相信至少一种古怪的理论，但是为什么这些荒谬而复杂的想法如此吸引人呢？



许多阴谋论诉诸于我们处理信息的基本方式。例如，我们天生就相信有意的因果关系。这意味着当你在露营时，听到灌木丛沙沙作响，你可能会认为有危险的动物潜伏在附近。你知道这可能只是风，但还是认为源头是个威胁更安全。当你不太明白事情的原因时，同样的偏执狂总是会出现。

然后是模式识别，这是我们看待世界和相互交流的基本特征之一。我们的大脑不断地寻找模式，但有时会在没有模式的地方找到它们。再加上我们无法控制局面时的恐惧，以及其他心理学原理，如确认偏差，我们的大脑实际上已经准备好寻找古怪的解释。
