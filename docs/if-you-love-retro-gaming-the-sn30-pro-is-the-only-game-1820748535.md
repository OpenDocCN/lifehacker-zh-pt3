# 如果你喜欢复古游戏，SN30 Pro 是你唯一需要的游戏手柄

> 原文：<https://gizmodo.com/if-you-love-retro-gaming-the-sn30-pro-is-the-only-game-1820748535>

任天堂承诺， [Switch 即将推出的在线服务将包括一个经典游戏](https://kotaku.com/nintendo-details-switch-online-service-will-be-20-yea-1795743981) 库，这将有助于将便携式游戏机变成一台史诗般——但并不完美——的复古游戏机。Switch 缺少的一件东西是一个真正的四向 D-pad，但这可以通过 [8Bitdo 的新 SN30 Pro 游戏手柄](https://gizmodo.com/this-snes-style-nintendo-switch-controller-is-designed-1795858424#_ga=2.10591921.487118478.1511787444-1848090429.1511787444)—[经典和现代手柄](http://www.8bitdo.com/sn30pro-sf30pro/) 的巧妙混合来轻松弥补。



今年早些时候，我评测了 8Bitdo 的优秀 NES30 Pro 控制器 ，这是你可以为任天堂 Switch 和其他兼容蓝牙控制器的游戏系统购买的最好的便携式控制器。

然而，我对 NES30 Pro 最大的抱怨是它的小尺寸，这使它非常适合旅行，但也使它在长时间的游戏过程中感到局促和不舒服，特别是如果你像我一样有一双更大的手。

新的 [SN30 Pro](http://www.8bitdo.com/sn30pro-sf30pro/) (基于最初的超级任天堂控制器)和 [SF30 Pro](http://www.8bitdo.com/sn30pro-sf30pro/) (基于最初的日本超级 Famicom 控制器)是任天堂有史以来最符合人体工程学的游戏手柄之一的近乎完美的再现。两者有什么区别？除了 SF30 Pro 上更多彩的按钮和 SN30 Pro 上稍微深一点的灰色操纵杆之外，什么都没有。SN30 和 SF30 Pro 在功能和物理上是相同的，所以从现在开始，我在这篇评论中只提到 SN30 Pro。

SN30 Pro 的设计毫无疑问是基于超级任天堂的游戏手柄，但除了增加的操纵杆之外，还有一些小的更新，让它在玩经典的 8 位和 16 位游戏时感觉非常不同。当与 8Bitdo 的 SN30(左图)并排比较时，它是一个相同的(但无线)SNES 控制器克隆，你可以看到 SN30 Pro 的方向键和 Y，X，B，A 按钮被稍微向上移动，为两个新的操纵杆腾出空间。

这是一个微小的设计变化，但它确实使用 SN30 Pro 玩 16 位游戏的感觉与你可能记得的略有不同。同样，为了适应现代游戏，你会发现原来 SNES 控制器两侧的单肩按钮在 SN30 Pro 上被分成了两个，我发现这影响了我在《真人快打》等游戏中使用的按钮捣碎技术。

但是，除非你是一个铁杆超级任天堂纯粹主义者，这些设计上的调整不会破坏交易。事实上，如果我手头没有一个原始的 SNES 控制器进行直接比较，我不确定我甚至不会意识到它的 Y，X，B，A-buttons 和 D-pad 位置的差异。

此外，这是一个为了更大利益而做出的小妥协，因为通过移动东西，8Bitdo 已经成功地为 SN30 Pro 上的额外功能按钮和一对模拟操纵杆腾出了空间，因此它也可以用来玩现代游戏。除了用运行在 MacBook Pro 和 Android 智能手机上的超级任天堂模拟器测试控制器之外，我还将它与我的任天堂 Switch 配对，玩了*马里奥赛车 8 豪华版*和*塞尔达传说:野性的呼吸*。

与我今年早些时候测试的 NES30 Pro 一样，SN30 Pro 作为无线控制器的表现完美无缺，但我发现与玩 Switch 的 Joy-Cons 或任天堂更大的 Switch Pro 控制器相比，它仍然感觉有点局促。

8Bitdo 的 SN30 Pro 上的模拟操纵杆比 Switch Pro 控制器上的更小，距离更近，这对我的大手来说，感觉不太自然。你的里程数肯定会有所不同，一些玩家可能会觉得这种布局非常舒适。但如果你认为自己是一名职业游戏玩家，SN30 Pro 可能不会是你每次都想接触的第一款游戏手柄。

我还必须为您将控制器同步到不同设备的过程编写 8Bitdo。当您使用开始按钮启动 SN30 Pro 时，您还需要按下特定的操作按钮，以将其置于 Switch、Android、Windows 或 macOS 兼容模式。我理解为什么会这样，因为每个平台与游戏控制器的通信方式都不同。虽然我很欣赏 8Bitdo 在每个游戏手柄的背面都有一个快速指南，但我认为多位置拨动开关会更好，让你在给游戏手柄加电之前轻松拨入你想要连接的设备。

难道 [SN30 Pro](http://www.8bitdo.com/sn30pro-sf30pro/) 就是统治他们所有人的那个控制器？不完全是。如果你正在寻找重温童年玩超级任天堂的最佳方式， [8Bitdo 的 SN30 游戏手柄实际上是原版](https://gizmodo.com/the-super-nintendo-retro-receiver-is-a-painless-wireles-1785066618#_ga=2.57587810.939608664.1511702889-1657940352.1511702888) 的完美复制，也是更好的选择。如果你认为自己是一个铁杆游戏玩家，并希望获得依赖两个模拟操纵杆的游戏的最佳体验，任天堂优秀的 Switch Pro 控制器是一个更好的选择。

然而，SN30 Pro 的优势在于它是休闲游戏玩家的万金油，他们花在玩经典复古游戏上的时间与玩现代游戏如*超级马里奥奥德赛*一样多。尽管有一些小的设计调整，SN30 Pro 仍然是一个 8 位和 16 位游戏的梦幻般的复古控制器。尽管一些游戏玩家可能仍然觉得它有点小，但它比 8Bitdo 的 [NES30 Pro](https://gizmodo.com/the-nes30-pro-is-the-perfect-portable-controller-for-th-1796068741) 更大，我认为这实际上使它在舒适性和兼容性方面成为更好的便携式控制器。如果你是一名 Switch 用户，一旦任天堂兑现其经典游戏“吃到饱”的承诺，SN30 Pro 无疑将是你永远想随身携带的控制器。
