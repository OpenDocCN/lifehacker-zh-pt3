# 如何让股票不平淡无奇

> 原文：<https://lifehacker.com/how-to-make-stock-that-isnt-bland-nonsense-1820626531>

股票是这么多食谱的支柱。无论是用作米饭或豆类的烹饪液体，还是作为汤料或肉汁的基础，原料的质量都会影响最终菜肴的质量。虽然这并不难做，但你可以做一些调整，确保你的汤汁丰富可口，一点也不油腻。

我这辈子从来没有用食谱来做高汤，主要是因为我只是用我存起来的蔬菜残渣和骨头来做。对我来说，这只是简单地将一加仑大小的冷冻袋价值的废料——包括动物和蔬菜——扔进一个大锅，加入足够的水以覆盖所有东西，让它们炖至少五个小时，撇去顶部的“浮渣”，并根据需要加入更多的水。或者，你可以把它们都扔进高压锅里，用“高火”煮 60 分钟。这两条路线都会给你一个非常美味的汤，炖肉和肉汁的完美储备。

*   **鸡翅:**除了自制的布法罗鸡翅，最受欢迎的鸡翅是鸡翅。虽然鸡翅可能缺少肉，但它们在脂肪和胶原蛋白方面弥补了不足，这是你做一道特别丰富的汤所需要的。鸡爪在这里也出类拔萃。
*   **变得斗志昂扬:**在你的冰箱里放两个加仑大小的冷冻袋——一个装蔬菜末、皮和果皮( [尤其是大蒜皮](https://skillet.lifehacker.com/use-papery-garlic-skins-to-give-broth-big-flavor-1797726954) )，一个装动物骨头。一旦你有足够的钱来填满一个罐子，你就可以重新进货了。
*   **烘烤增加味道:**在将蔬菜块放入汤锅之前，先在烤箱中烘烤蔬菜块——不管是碎的还是整块的——这会增加一种生吃所没有的味道，并让你的汤料颜色更深。烤骨头味道也更好，所以不要用烤鸡。
*   奶酪皮是你的朋友:，*永远不要扔帕尔马干酪皮。这些东西会带来咸味，比如哇，应该和你其他好斗的朋友一起炖。*
*   干蘑菇增加深度:虽然新鲜的真菌没有什么不好，但干蘑菇有更深、更浓的味道。
*   **添加酸:** [一点醋](https://skillet.lifehacker.com/why-you-should-add-vinegar-to-your-chicken-stock-1818492266) 可以帮助分解你所选择的骨头中的胶原蛋白，从而产生更丰富、明胶含量更高、口感更圆润的液体。
*   芳香很重要:多叶草本植物和其他植物部分增加了新鲜度和风味，并使你的食物不再单一。几根百里香、一片月桂叶和完整的胡椒粒总是有帮助的，但是不要回避柠檬草、姜根或干辣椒。

一旦你的汤炖好了——尝一尝，确保合你的口味——沥干，冷却，放进冰箱。去掉顶部凝固的脂肪，转移到一个有盖的容器中，在冰箱中储存几天。
