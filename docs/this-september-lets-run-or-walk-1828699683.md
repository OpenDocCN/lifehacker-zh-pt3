# 这个九月，让我们奔跑吧！(或者走路！)

> 原文：<https://lifehacker.com/this-september-lets-run-or-walk-1828699683>

这是开始运行 的 [最佳时间——或者回到运行状态，如果你已经休息了的话。天气变冷了，每个人都在适应他们返校的日常生活(这对学生和家长来说都是好消息)，我们刚刚花了一个月](https://lifehacker.com/late-summer-is-the-best-time-to-start-running-1785433859) [尝试倒立](https://vitals.lifehacker.com/let-s-get-upside-down-for-the-august-fitness-challenge-1828029407)——所以相比之下，一点普通的慢跑会感觉很棒。



这是一个全方位的挑战，所以我对每个读者都有一些想法，不管你是否已经知道如何跑步。如果您受伤或行动不便，无法行走或跑步，请随时加入我们，选择任何适合您的旅行方式。

### 如果你已经知道你在做什么

如果你已经定期跑步，我希望你能为这个月选择一个重点，一些你没有花足够时间去做的事情。也许你一直在做长距离跑，但从来没有做过速度训练，反之亦然。决定不了？让它 [希尔重复](https://vitals.lifehacker.com/how-to-run-hills-and-boost-your-endurance-1696792368) 。选择一座小山，明天慢跑四次，然后下周跑五次(以此类推)。到这个月底，你可能仍然讨厌山，但是你会 [非常擅长它们](https://lifehacker.com/how-to-run-hills-without-crushing-your-confidence-1783126331) 。

好的，明白了吗？没有必要阅读这篇文章的其余部分——系好鞋带，离开这里。

### 如果你以前跑步过，但是感觉不舒服

既然我们已经让那些人听不见了，是时候来点真正的谈话了。跑步是你想做的任何事情。你可以混合跑步和散步，你可以在跑步机上完成所有的跑步，你可以出去慢跑，因为 [你很慢](https://lifehacker.com/stop-apologizing-for-being-a-slow-runner-1717538541) ，*没关系*。

重要的是利用你所拥有的。如果你的努力程度让你觉得有挑战性，但不是致命的，那么不管你的确切配速是多少，你都跑对了速度。事实上，帮你自己一个忙，关掉你正在运行的应用程序中告诉你跑了多快或多远的提醒。(如果你敢， [完全放弃应用程序](https://vitals.lifehacker.com/running-apps-have-hurt-my-progress-not-helped-it-1729318194) 并通过距离*或*时间来测量你的跑步，但不能两者都用。)

你这个月的任务是:给自己制定一个时间表，并坚持下去。一周三到四次是好的，但是要慢慢增加:开始的时候混合步行和跑步，或者这周开始跑两次，然后随着时间的推移增加。即使你已经具备了进行四次强力跑步的有氧健身能力，你的骨骼和肌腱也需要一点时间来适应踩踏路面。

像更高级的人一样，你应该为这个月选择一个焦点。这可能只是一致性，或者也许你想在某项特殊技能上努力，或者尝试一种新的跑步方式——例如，在小径上而不是在公路上。基本上，如果你对跑步感到不舒服，这是你找出你喜欢做什么和留下一些快乐回忆的机会。

### 如果你完全是初学者

你必须教会你的身体什么是轻松、有趣、舒适的跑步感觉。现在，它不知道。

你现在可能有两种速度:步行和冲刺。跑得多的人，没有某种连续冲刺 30 分钟的神奇法术；他们只有 [和](https://vitals.lifehacker.com/run-smarter-not-faster-how-to-find-your-gears-for-b-1706084781#_ga=2.227863390.2102425869.1535379936-1456718367.1520458611) 之间的几个档位，所以他们可以决定跑中慢或有点快或者他们的锻炼要求的任何速度。

所以你这个月的工作是想出如何长时间徒步旅行，而不要工作得太累以至于想死。您有几个选择:

*   只是走路。如果这感觉太容易，走快一点，或者走上坡。你可以通过这种方式得到很好的锻炼，如果你是锻炼新手，这可能是你目前最理想的配速。
*   步行，但每隔几分钟增加一些短距离冲刺。这并没有真正教会你放慢速度，但是，嘿，也许你本质上是一个短跑运动员。只要你希望锻炼持续，就交替进行跑步和步行。
*   慢跑，散步休息。提醒自己慢慢慢跑，保持短暂的步行休息，这样当你再次开始时仍然会有点累。这将有助于你保持缓慢的慢跑速度，在你意识到之前，你会发现自己跳过了休息时间。

当然，你可以做到以上三点。不确定从哪里开始？明天，穿上一些舒适的鞋子——它们甚至不一定是跑鞋！—然后出去走走。

不，你 [不需要先拉伸](https://vitals.lifehacker.com/the-truth-about-stretching-when-it-helps-and-when-it-d-1718270464) 。不，你 [不需要任何特殊的衣服](http://lifehacker.com/is-there-really-a-benefit-to-special-exercise-clothing-1607198642#_ga=2.227314270.2102425869.1535379936-1456718367.1520458611) (除了一件 [运动文胸](https://vitals.lifehacker.com/large-chested-look-for-these-features-in-a-sports-bra-1714937318) ，如果适用)。你甚至不需要一个 [跑步应用](https://vitals.lifehacker.com/what-s-the-difference-between-all-these-running-apps-1779631255) 或一个“couch to 5K”程序来进行你的第一次外出，尽管你可能希望很快选择一个来帮助指导你通过最初的学习阶段。
