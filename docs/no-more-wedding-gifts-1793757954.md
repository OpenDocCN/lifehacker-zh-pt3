# 没有结婚礼物了

> 原文:[https://jezebel.com/no-more-wedding-gifts-1793757954](https://jezebel.com/no-more-wedding-gifts-1793757954)

理论上，我们在婚礼上送礼物有两个原因:

**1。**这对夫妇正从各自父母的家中搬出，是他们的家人和朋友的慷慨让他们从零开始建立了一个新家庭。

**2。**因为他们为你的鸡肉或意大利面买单。

当然，我们也可以出于爱送礼物，但是既然结婚礼物被认为是必须的，我们就把爱排除在外了。

我们不应该被期望送结婚礼物。

这不是一个普遍接受的观点。最近，我在全国各地参加了一场婚礼，花了我大约 700 美元，这对一些人来说似乎微不足道，但对我来说不是。没过多久，我向这对夫妇的一个共同的朋友承认，我没有给这对新婚夫妇买礼物，她也被邀请了，但没有去。她吓坏了。作为一个刚刚订婚的人，她认为我并没有欠他们什么礼物*是非常私人的。我没有被邀请参加她的婚礼。*

也有一些例外。可以想象的是，一些年轻或不太富裕的夫妇甚至没有一个抹刀就一起搬进了一个家，在这种情况下，登记处是合适的和预期的。对于那些需要帮助才能拥有美好一天的贫困夫妇，祝福你们。给我发个邀请，让我知道你的结婚蛋糕 GoFundMe。

然而，这个假设世界的规则并不适用于那些在布鲁克林一套 3000 美元的一居室里同居多年的 20 多岁的人。如果你能买得起自己的凯膳怡，但仍然希望在你的婚礼上有一个客人，因为他们接受了你的邀请，你可以马上滚蛋。

去参加婚礼很贵。通过邀请某人来见证你的个人舞会日，你也是在邀请他们花钱打扮自己，去旅游，住酒店，可能还会请假。如果他们参加了婚礼，他们将支付所有在中央马戏团附近出现的额外天数——单身派对、预演晚宴、新娘送礼会等。情侣们，你们猜怎么着:在和朋友庆祝重要的一天和付房租之间做出选择是很糟糕的。

办婚礼也贵。极其昂贵。但是，如果你没有从你的客人那里得到 150 美元的折扣，你就不能扔了它，那就他妈的缩小规模。一个烧烤自助餐盘子和一桌子红色天鹅绒蛋糕的成本并不等同于高脚杯。没人会资助你戴头饰或买一件定制的缎面晚礼服外套的决定。客人应该做的唯一一件事就是出现，用爱、关注和对你们合法婚姻的认可来沐浴你，而不要因为喝得太醉而让你尴尬。这比大多数人得到的都多。

也就是说，我对礼物的鄙视不仅仅适用于婚礼。生日礼物、周年纪念礼物和大多数婴儿送礼会礼物也应该放在厕所里——除了一些例外。孩子应该得到礼物，就像一些第一次生孩子的人一样，他们通常没有婴儿用品包。在这些事情上互相帮助是有意义的——尽管当人们宣布他们孤独地变老时，没有对等的支持网络。单身人士应该开始举办强制性聚会，为他们的储藏室收集食物，并建立一个家庭护理基金。那太棒了。

给 3 岁到 16 岁的孩子送生日礼物也是可以接受的，因为他们还太小，可以随意买一些不合他们口味的东西，这会把他们的家弄得乱七八糟，当他们不得不搬家时会成为负担。奇怪的是，我也不喜欢收到礼物。

给成年人最好的礼物是经历。婚礼仪式的共同经历是强大的，令人难忘的，而且通常真的很有趣。作为主人，是你在送礼物，我们不应该期待收到礼物。但是如果是这样的话，请给我一份详细的账单，我们可以公开比较费用。
