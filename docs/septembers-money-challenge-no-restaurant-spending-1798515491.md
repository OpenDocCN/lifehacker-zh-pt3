# 九月的金钱挑战:没有餐馆消费

> 原文：<https://lifehacker.com/septembers-money-challenge-no-restaurant-spending-1798515491>

信不信由你，是时候开始为假期存钱了。礼物、旅行、聚会——这些开销会很快向你袭来，所以你想尽快做好准备(也就是现在)。为了帮助你省钱，这个月我们挑战你减少一个常见的挥霍:餐馆。

Watch

我们都知道餐馆消费会越积越多(去年，这是你们大多数人认为最大的挥霍)，所以这个月加入我们，我们试着把餐馆消费削减到零。是的，这是一个雄心勃勃的目标，但不要气馁。这个想法是尽可能地接近零，所以即使你犯了错误，你仍然节省了一些现金(并且希望对你的习惯有所了解)。

月底，我们会回来检查你的表现，看看你为假期存了多少钱(或者任何目标，真的)。所以一定要把你的积蓄存起来。

你加入吗？如果是，请告诉我们您平均每月在餐厅花费多少。我们会把这个数字和你月底的总花费进行比较。
