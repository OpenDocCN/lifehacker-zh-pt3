# 高地桌面

> 原文：<https://lifehacker.com/the-highlands-desktop-1789284784>

这幅风景可能看起来像是冒着蒸汽或着火了，但它只是照片拍摄时一天中明亮、丰富的颜色——它构成了一幅美丽的壁纸。添加一点定制，Nicolas 就有了一个值得每天坐下来使用的很棒的桌面。这是所有设置的方式。

Watch

Nicolas 正在使用 bersicht， [一个 macOS 定制套件，我们在](https://lifehacker.com/ubersicht-customizes-your-os-x-desktop-with-widgets-and-1591137635) 之前已经强调过了。您将需要抓住它开始，但它几乎是所有您需要的。

一旦你安装了它，这里是你需要得到同样的外观:

*   [壁纸](https://www.reddit.com/r/EarthPorn/comments/5bh8t4/sunrise_in_the_scottish_highlands_4256x2832_oc/) ，一张由 [乔治王探险者](https://www.reddit.com/user/GeorgeTheExplorer) 拍摄的照片，摄于苏格兰的贝因河
*   macOS 的[über sicht](http://tracesof.net/uebersicht/)定制套件
*   [圆形时钟小部件](http://tracesof.net/uebersicht-widgets/#roundClock) 为 bersicht
*   [调酒师实用程序](https://www.macbartender.com/) 定制 macOS 菜单栏
*   [cDock 实用程序](http://w0lfschild.github.io/app_cDock.html) 定制 macOS Dock
*   [LiteIcon 实用程序](https://freemacsoft.net/liteicon) 来调整系统图标
*   将 [的白色图标全部换成 OS X 的](http://noshery.deviantart.com/art/All-White-Icons-for-OSX-470738808) 包来替换系统图标

所有这些都组合在一起，形成一个简单而低调的外观，它仍然可以完美地定制，以满足 Nicolas 的需求，并给普通的旧 macOS 带来新的外观。如果你喜欢它，或者对如何使它成为你自己的有疑问，点击下面的 Flickr 链接询问，让 Nicolas 知道你有多喜欢他的作品——以及我们有多喜欢他的作品。
