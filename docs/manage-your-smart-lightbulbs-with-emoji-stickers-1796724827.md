# 用表情贴纸管理你的智能灯泡

> 原文：<https://lifehacker.com/manage-your-smart-lightbulbs-with-emoji-stickers-1796724827>

我最近购买了一些智能灯泡，并计划扩大我的智能灯系列。不过，我确实注意到了安装过程中的一个小不便:如果不盯着应用程序看，很难分辨哪个是哪个灯泡。所以我用表情符号贴纸给我的灯泡添加了视觉辅助。当吊扇关闭时，看到“香蕉”灯熄灭要比试图找出哪个灯泡是“色调客厅灯泡 7”容易得多。



### 买些贴纸

当然，你需要一些贴纸。你可以在亚马逊上买到 1344 包的 [表情贴纸](https://www.amazon.com/gp/product/B01D7G5IOG/ref=oh_aui_detailpage_o00_s00?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/manage-your-smart-lightbulbs-with-emoji-stickers-1796724827&asc_source=&ie=UTF8&psc=1&tag=kinjalifehackerlink-20) 。它们很小，直径只有半英寸，所以不会对灯泡产生不利影响。如果你把灯泡放在灯罩或烛台之类的固定装置里，你甚至不会看到它们，直到你需要的时候。

由于智能灯泡是发光二极管，不会像传统灯泡那样变热，所以您不必担心贴纸融化或脱落。

### 使用类似的表情符号作为视觉辅助

你肯定在每个房间里都有很多灯，所以为了方便起见，给房间分配表情类别。我给我的客厅指定了水果表情，给我的办公室指定了笑脸表情。毫无疑问，你可以使用相应的家居相关表情符号，或者用颜色区分它们。

如果你不得不搬家的话，这会让事情变得更容易。你可以让你的灯泡设置保持原样，而不必重新分配或重新安排它们，因为你忘记了哪个灯泡放在哪里。

### 在应用程序中分配表情符号

在你挑选了一些贴纸并把它们贴到你的智能灯泡上后，你必须相应地给它们重新命名。在飞利浦 Hue 应用程序中，您可以简单地在灯泡名称前添加相应的表情符号。

即使你没有大量的智能设备，用这个表情符号速记将它们分类也可以更容易地管理你将来要添加的设备。
