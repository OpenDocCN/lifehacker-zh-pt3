### ***如何制作 GIF***

*GIF Toaster 向您显示可用于制作 GIF 的可用照片和视频列表，以及不同媒体类型的转换选项，如连拍、实时照片、延时和全景照片。如果你的头像看起来像一只会说话的狗，你也可以转换你的 Animoji 视频。它只对你的 iOS 设备上的文件有效，所以不要放入 YouTube 链接来制作剪辑(使用 [Giphy 的网络工具](https://giphy.com/create/gifmaker) 或 [Mac 应用](https://lifehacker.com/giphy-capture-for-mac-os-x-lets-you-make-gifs-of-anythi-1769716158) )。*

*选择您的照片或视频，点击下一步，并使用编辑器自定义您的 GIF。您可以剪辑视频的长度，选择帧速率，调整 GIF 速度，改变回放方向，裁剪帧，以及水平或垂直旋转视频。如果你想让你的作品更有特色，你还可以添加过滤器和文字横幅。*

*您甚至可以选择设备上已经存在的 gif，并将其转换为视频、静态照片、视频或实时照片。更重要的是，在你创建了一个 GIF 之后，当你查看 info 标签时，你可以看到所有相关的统计数据，比如帧率、长度、文件大小和总帧数。*

*如果你需要分享你的设备屏幕上的内容，GIF Toaster 可以很容易地将《愤怒的小鸟 2》中展示你技能的长录音转换为一个短而甜美的图像，与视频源材料相比，它不需要几个小时就可以发送。*

*GIF Toaster 可以免费使用，但支付 1.99 美元的应用内购买不仅可以摆脱应用底部令人讨厌的广告，还可以添加像 GIF 拼贴创作、无限分辨率的 GIF 创作和无限数量的 GIF 相册这样的功能，让你的创作井井有条。如果你想给你的 GIF 添加奇怪的贴纸和覆盖图，还有其他应用程序可以满足你的定制乐趣( [考虑 Giphy](https://giphy.com/apps/giphycam) ，它可以很容易地给你的动画创作添加滑稽的道具)，但 GIF Toaster 是迄今为止我偶然发现的最容易使用的 GIF 创作工具，它的功能足够多，可以成为我的主要 GIF 工具选择。*
