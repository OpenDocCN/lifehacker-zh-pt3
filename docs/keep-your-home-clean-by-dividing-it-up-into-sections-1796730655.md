# 把你的家分成几个部分来保持干净

> 原文：<https://lifehacker.com/keep-your-home-clean-by-dividing-it-up-into-sections-1796730655>

不管你的家有多大或多小，如果你天生不是一个爱整洁的人，那么保持它的干净会是一个挑战。我住在旧金山的一套一居室公寓里，这意味着不需要太多的杂乱就能让我的住处看起来完全失控。除了空间问题，我还有一个更大的问题，那就是我天生就是一个邋遢的人。有一段时间，我认为我注定要生活在一片混乱中——直到我开始分段清洁。

分区的想法是我在一个朋友从外地来我家住的几周前想出来的。我知道我需要把我的地方收拾得井井有条，但是我的工作日程加上我家的脏乱程度让它看起来像是一个非常不可能的任务，所以我把它分解了。在混乱的生活了一辈子之后，我觉得我的公寓从那以后一直保持着合理的清洁度。

它是这样工作的:

*   根据房间通常有多脏，把你的家分成七个部分。这里的想法是清理七个区域，这将花费你一个小时或更少的时间。所以，如果你的卧室是一个噩梦，那么也许应该把它分成两个，而你不用的客房和客用浴室可以合并成一个。想一想:一小时内我实际上能打扫什么？永远不要有比这更长的“部分”。
*   给每个部分分配一周中的某一天。例如，星期五是我的浴室日。也就是说，每周五我都会往马桶里喷一些厕所清洁剂，擦干净我的柜台，用刷子刷地板，然后冲淋浴。我第一次这样做花了很长时间。现在我每周都做，15 分钟就能做完。
*   一旦你把所有的事情都分好了，就要坚持(至少一周)你的时间表，并且把这些区域整理好。显然，这对我来说更容易一些，因为我有一个小房子。如果你有一个更大的，那么你可能需要在几周内扩大最初的清洁。让它对你和你的时间表来说是可管理的。如果不是，那么你很可能会放弃它。
*   在初次清洁后，制定一个你知道可以坚持的保养计划。例如，我会在周三晚上打扫我的生活区，这正好是我在 Hulu 上看电视节目的时间。我看电视，在广告期间整理衣服。我的厨房在工作日通常会失控，所以我会在周六处理，因为我知道我会有更多的时间。

把事情分解成更小的容易做的块听起来很简单，但是出于某种原因，这是我以前从未真正尝试过的事情。当我试图解决整个公寓的问题时，我经常失败，即使我给自己一整天的时间，因为我会太不知所措，分心，最终筋疲力尽。

现在，我自豪地拥有了一套中等清洁的公寓，几年后，它设法保持这种状态——尽管事实上我住在那里。
