# 寿司生姜的最佳标签外用途

> 原文：<https://lifehacker.com/the-best-off-label-uses-for-sushi-ginger-1828227531>

我非常喜欢腌姜。我总是点额外的，不是和寿司一起吃，而是在我吃完寿司之后。辛辣的甜味片是一餐完美的胃沉淀结束，如果你打算吃甜点，它们是很好的味觉清洁剂。



在一次送餐失败后，我没有多余的食物，我一直在买一罐罐这种东西，确保永远不会短缺。这种丰富的糖让我一整天都在嚼着糖片，并且把它放进任何我能想到的任何东西里。以下是我最喜欢的寿司生姜的一些非寿司用途:

*   将它混入商店购买的调料和酱料:用它给你可能已经厌倦的调味品增加甜味和魅力。你厌倦了普通的花生酱吗？将半杯它与 25 克左右的腌姜勾兑，帮助你再次坠入爱河。(不要用番茄酱做这个。)
*   **放在沙拉和三明治里**:毕竟是泡菜，三明治需要泡菜。腌姜配鸡肉、火鸡和(令人惊讶的)烤牛肉非常棒。对于沙拉，将切片切成丝，然后与金枪鱼沙拉、鸡肉沙拉或一大堆蔬菜一起搅拌。
*   **制作** [**这道值得在餐厅享用的生姜沙拉酱**](https://lifehacker.com/this-ginger-dressing-tastes-exactly-like-it-came-from-a-1827997044) **:** 你所需要的只是加里、酱油和烤芝麻油。
*   **用它腌制肉类:**将一罐腌制的生姜调成糊状，与少许酱油或味噌混合，T3】也许是一点廉价啤酒 ，将混合物涂在一些猪肉、鸡肉或牛肉上，以嫩化和调味。
*   向鸡尾酒里加一点盐水:你知道吗，你可以用任何你喜欢的盐水来调一杯肮脏的马提尼？可以；我在给你许可。我说的是，一杯肮脏的姜味马提尼是神圣的。

你也可以像我一样，直接吃罐子里的切片，蘸一点酱油作为咸甜小吃。或者你不能这样做；有些人觉得这令人不快。
