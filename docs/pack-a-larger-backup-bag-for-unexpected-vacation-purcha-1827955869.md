# 为意外的假期购物准备一个更大的“备用包”

> 原文:[https://life hacker . com/pack-a-large-backup-bag-for-unexpected-vacation-purcha-1827955869](https://lifehacker.com/pack-a-larger-backup-bag-for-unexpected-vacation-purcha-1827955869)

这些天来，我倾向于每个月至少有两周去旅行，这意味着，除了其他事情，我已经成为一个执着于行李箱的专家。我确切地知道我需要为一周的外出打包什么，以及如何打包，以便它们都可以舒适地放在我的随身行李中，这样我就可以避免在门口托运行李。我的武器库中最近增加的最大的东西:一个薄薄的行李袋，我在打包之前放在我的随身行李的底部。

Watch

去年年底，我在哥本哈根机场买了这个包，主要是为了用光我手头剩余的当地现金。从那以后，它就成了一件非常重要的事情。

我的包和我的手提旅行包大小差不多，但由这种仿麂皮面料制成，尽管非常非常薄，但(到目前为止)已经证明非常耐用。如果很紧的话，我随身携带的所有东西都可以放进去。它足够薄，平放时就像是我普通行李箱中的一个小小的额外内衬。它看起来不像是网上可以买到的，但它看起来像是 [这个](https://amzn.to/2K6BU1t) 的放大版。

背着包的想法很简单:我喜欢买东西，特别是当地的酒，这种酒既重又占去了一吨宝贵的行李箱。

虽然有时我可以把我买的东西和其他东西一起放进包里，但其他时候我需要更多的空间，否则在回程托运时包会超重。当这种情况发生时，我会拿出我可靠的备份包，把其他东西都装进去。行李包成了我的随身行李，我的旅行包可以处理剩下的东西。

多年来，我一直用自己购买的一个小型可折叠赫谢尔包(比如 [这个](https://amzn.to/2uYDEW7) )在理论上这样做。我买它是因为它很小，但小意味着它只能装一些东西，最终使它携带起来更麻烦，而不是有所帮助。我想(最多)我已经用它卸下了我的跑鞋和一些衣物，这真的只能在我的包里放一瓶酒。你想要一个大旅行袋。

一个更大的袋子，可以像普通行李箱的衬里一样展开，更适合这项工作，当你回家并需要想办法把其他东西带回家时，可以让打包变得更容易。

有时候你不需要它，但是当你需要它的时候，它就在手边，这是一个很棒的东西，可以省去你最后一分钟去酒店礼品店的麻烦，让你知道如何把所有东西都带上飞机。
