# 用 2-3-1 技巧写出更强的句子

> 原文:[https://life hacker . com/write-strong-sentences-with-the-2-3-1-trick-1821298844](https://lifehacker.com/write-stronger-sentences-with-the-2-3-1-trick-1821298844)

在我主修英语的六个学期中，这是我学到的最好的东西:当有疑问时，将句子中最好的部分放在最后，次好的放在开头，其余的放在中间。所以按照最好的顺序，那就是 2，然后 3，然后 1。

Watch

“最好的一点”是什么？这可能是最好听的部分。它可能会抓住你的重点。这可能是最具体或最令人惊讶的词。

在句子的其余部分，它可能只是“最好的一点”。 [在他的《如何写一句话](https://www.npr.org/2011/01/25/133214521/stanley-fish-demystifies-how-to-write-a-sentence)*》一书中，斯坦利·费什赞美了这句开场白，出自一位学生的作文:*

> *我听说箱子的事时已经在二楼了。*

*就其本身而言，“盒子”是一个平凡的短语，但这正是它在这里如此强大的原因。当你读到“当我听说”的时候，你已经为“母亲之死”或“爆炸”这样的结局做好了准备。不，是一个盒子！*箱子！那个盒子里是什么！？**

**你写的大多数句子，你都不会像这样费神去解析。2-3-1 结构在开线方面做得最好。喜欢*百年孤独*的开篇台词:**

> **多年以后，当奥雷连诺上校面对行刑队时，他会想起父亲带他去发现冰的那个遥远的下午。**

**还是*心爱的*的开场白:**

> **124 是恶意的。**

**2-3-1 规则的每一部分都是灵活的，因为它不是一条规则。你可以把最精彩的部分放在开头。你可以用最好的部分造一个完整的句子。而如果一句话已经在起作用了，就不要为了配合模式而断句。*黎明踏浪号*的开篇台词是 3-2-1:**

> **有个叫尤斯塔斯·克拉伦斯·斯克罗布的男孩，他几乎是罪有应得。**

**有时候打破规则才是重点。*钟罩*的开口线为拱形 2-1-3；**

> **那是一个奇怪而闷热的夏天，他们电死了罗森博格一家的那个夏天，我不知道自己在纽约做什么。**

**当然，到现在为止，你已经不同意这些例子中的至少一个，因为这都是主观的，挑剔的，有点假。一句话中最没意思的部分可能是介绍一个故事或一篇文章所做的最多的工作，或者可能是建立一种基调。你可能会找到一个很好的理由来改变你的句子结构。那也行！如果我可以加上最后一个拐弯抹角的警告，仔细想想你如何在句子层面上安排你的想法是很有用的。**

**但是有时候你应该从字面上理解 2-3-1 法则。每当你在组织一个句子的时候停下来，或者每当你想知道为什么你的好想法会落空的时候，试一试。然后再写一句，再试一次。**
