# Goocards 主屏幕

> 原文:[https://lifehacker.com/the-goocards-home-screen-1736809148](https://lifehacker.com/the-goocards-home-screen-1736809148)

如果你是谷歌基于卡片的界面的粉丝，并希望你的整个主屏幕都能分享这种外观，那么今天的设计就是为你设计的。它甚至还有一个黑暗的主题！

Watch

要在您的设备上获得这种外观，您需要:

*   [新星发射器](https://play.google.com/store/apps/details?id=com.teslacoilsw.launcher)T3】
*   [KLWP](https://play.google.com/store/apps/details?id=org.kustom.wallpaper.pro)T3】
*   [文件](https://play.google.com/store/apps/details?id=com.goocards)

你有没有一个很棒的、被调整到遗忘的家或者你自己的锁屏，你想分享吗？把它贴在下面的评论里，或者贴在你自己的 Kinja 博客上，标签是“主屏幕展示”(没有引号)。一定要包括你是如何制作它的描述，这样我们就可以把它作为下一个特色主屏幕。
