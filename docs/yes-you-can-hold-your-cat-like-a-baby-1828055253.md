# 是的，你可以像抱婴儿一样抱着你的猫

> 原文：<https://lifehacker.com/yes-you-can-hold-your-cat-like-a-baby-1828055253>

猫的魅力在于它们性情多变，当你伸手去抓它们时，它们可能会咬人，也可能会发出咕噜声。可能会有这样一个时刻，当你和你的猫建立了足够亲密的关系，它至少会在它要攻击的时候发出信号；既然如此，为什么不像婴儿一样随身携带呢？

Watch

兽医 Uri Burstyn 博士在 YouTube 上有一系列关于处理动物的视频，他的第二身份是 [有益的温哥华兽医](https://www.youtube.com/channel/UCtEnS6HGyK1oe71sxbJuhcw) 。即使你没有养猫，我也推荐你去看它们，因为它们令人难以置信的舒缓。他最受欢迎的一个关于处理猫的视频收到了许多人的评论，他们问是否可以像抱婴儿一样抱着他们的猫。人们想知道。

他们所说的“像婴儿一样抱着”是指在你的臂弯里翻过来。我的猫非常慷慨地允许我像这样带着它们在公寓里走来走去，甚至让我抚摸它们的肚子。这是一种荣誉和特权。但是 Burstyn 博士警告说，你不应该对一只你不信任的猫这样做。在另一只猫兰斯洛特的注视下，他和他的伙伴海盗一起表演。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video--wRdQmYXc8M&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video--wRdQmYXc8M&start=0) 

你不能随便抓任何一只猫宝宝式的原因很明显是因为你把尖尖的两端朝向你的脸，一只不喜欢这个姿势或者容易上手的猫会把你的脸当成逃跑的梯子。他还警告说，如果他们背部疼痛，他们“可能不会欣赏”这种姿势。为什么你可怜的小猫会背痛？！它不是驮畜！

从个人经验来说，你也应该对你的猫的情绪敏感。如果它们开始扭动并环顾四周，或试图坐起来，这意味着它们想被放下。尊重这些信号，在他们抓你的眼睛发出信号之前。
