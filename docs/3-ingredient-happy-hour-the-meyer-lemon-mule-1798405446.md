# 三种成分的快乐时光:梅尔柠檬骡子

> 原文：<https://lifehacker.com/3-ingredient-happy-hour-the-meyer-lemon-mule-1798405446>

周末快乐，欢迎回到 [*三料欢乐时光*](http://lifehacker.com/tag/3-ingredient-happy-hour) ，每周饮品专栏，提供超级简单又美味的酒水。今天，我们将为一部经典电影《莫斯科骡子》带来阳光明媚的一面。

我并不反对“正规的”莫斯科骡子。伏特加、鲜酸橙汁和姜汁啤酒的组合是酸和甜点心的胜利。但是有一天，当我在杂货店浏览产品时，一些看起来很开心的梅尔柠檬吸引了我的目光。(现在不是时令，但是很多专卖店全年都进口。)这些略圆、橙黄色的柠檬被认为是与柑橘或普通橙子混合在一起的，它们的果肉和果汁更甜、酸性更低。(他们也闻到了*完美的*。我真的想一直闻起来像梅尔柠檬。)

当加入骡子时，梅尔柠檬汁使稍微提神的鸡尾酒变得更柔和，更阳光。就像喝姜汁柠檬水一样，但是，你知道，有点醉。要制作这款糖醋吸管，你需要:

*   2 盎司伏特加
*   3/4 盎司新鲜梅尔柠檬汁
*   3-4 盎司辣姜汁啤酒——不是麦芽啤酒——比如 Cock 'n' Bull 或 Reed's。

将伏特加和果汁加入柯林斯杯中，稍微搅拌一下。(你也可以使用经典的铜马克杯，但是 [就不需要](http://skillet.lifehacker.com/how-to-set-up-your-home-bar-with-the-equipment-you-actu-1797905524) 。)在里面舀一些冰，上面放 3-4 盎司姜汁啤酒。再轻轻搅拌一下，喝一口液态阳光。
