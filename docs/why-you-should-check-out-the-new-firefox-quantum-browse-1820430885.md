# 为什么你应该看看新的火狐量子浏览器

> 原文：<https://lifehacker.com/why-you-should-check-out-the-new-firefox-quantum-browse-1820430885>

Mozilla 声称其更新后的网络浏览器速度是上一个的两倍，并且比竞争对手——尤其是谷歌 Chrome 使用的内存少得多。它被称为 Firefox Quantum，它可能会给网络上最流行的浏览器一个机会。



### 【Quantum 与 Chrome 相比如何

根据该公司的基准测试 将 Quantum 与 Chrome 和之前版本的 Firefox 进行比较，Firefox Quantum 的速度是之前版本 Firefox 的两倍。其内存测试显示，与 Windows 10 和 Linux 上的谷歌 Chrome 相比，内存使用量明显下降；在 macOS 10.12 上，内存使用量基本相同。

如果你是 Chrome 用户( [或 Lifehacker 阅读器](https://lifehacker.com/why-chrome-uses-so-much-freaking-ram-1702537477) )，你可能知道网络浏览器使用了多少内存。内存使用的减少意味着更快的网页浏览速度，同时打开更多标签的能力，以及在清理内存时计算机决定变慢的可能性降低。

Quantum 中也包含了一些非常漂亮的功能:比如内置的截图功能和改进的追踪器屏蔽。一个图书馆按钮可以让你访问你保存的内容，如书签，你的口袋阅读列表和截图，使这些服务的扩展变得过时。(也就是说，你仍然可以使用任何你喜欢的 Firefox 扩展，比如 Ghostery、LastPass 和其他第三方开发者提供的。)

### **如何下载和同步**

你可以从 Mozilla 的网站 下载新的浏览器。为了利用它的同步功能，确保你可以在 iOS 和 Android 应用程序上访问你的 Firefox 浏览历史和其他偏好，就像你使用谷歌帐户一样，你需要一个 Firefox 帐户。

你可以在打开 Firefox Quantum 的时候设置一个。只需输入您的电子邮件地址和密码，并指定您希望在设备之间同步的信息。您可以从 Microsoft Edge、Internet Explorer 或 Google Chrome 导入数据以保持同步，免去重复输入密码的繁琐。
