# 使用此应用程序通过抓拍封面图片来播放黑胶专辑

> 原文：<https://lifehacker.com/stream-a-vinyl-album-by-snapping-a-pic-of-its-cover-art-1825800020>

有一段时间，我有点痴迷于购买黑胶唱片。童年时，我妈妈让我拥有住在客厅里的大柜式唱机，一旦我把这个 500 磅重的东西搬进我的房子，我就决心尽可能多地听它。

Watch

许多艺术家现在用黑胶唱片发行他们的新专辑，因为这是时髦的做法。对于这些，你知道你进入了什么。对于那些随机的模糊的旧货店发现，你的结果可能不太可预测，而且大多数都没有地方让你在购买前做一个试驾记录。

一款名为唱片播放器 的新网络应用 [让这一切成为可能。有了它，你可以拍下该专辑的封面，它会在 Spotify 上找到该专辑(你需要一个 Spotify 帐户才能实现这一功能)，这样你就可以当场播放它。](http://record-player.glitch.me/auth) 

所有这些神奇的事情都要归功于谷歌的 Cloud Vision API 和 Spotify API。

是的，你可以自己在 Spotify 上查找这张专辑，不，它不会告诉你商店里你手里的唱片布满了划痕，你永远也不能听它——但它很酷。当你只是想听一些你有家的东西，但不想启动老转盘让它发生时，它也可以派上用场。

查看 [这里](http://record-player.glitch.me/auth) 。
