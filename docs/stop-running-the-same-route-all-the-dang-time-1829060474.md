# 不要总是跑同样的路线

> 原文:[https://life hacker . com/stop-running-the-same-route-all-the-dang-time-1829060474](https://lifehacker.com/stop-running-the-same-route-all-the-dang-time-1829060474)

出奇的容易陷入墨守成规。当你 [开始一个跑步或走路的习惯](https://vitals.lifehacker.com/this-september-lets-run-or-walk-1828699683#_ga=2.154142717.418006613.1536587025-1456718367.1520458611) 时，选择相同的路径两三次，突然，那条路径成为你锻炼中毫无疑问的一部分。或者你可能改变你的位置，但你总是以“哦，我的上帝，我要死了”的速度跑三英里。好吧，别说了。

Watch

是的，是的，你选择这次跑步是有原因的。但是让我们看看所有的选择。你可以跑长跑短，跑早跑晚，跑快跑慢，跑丘陵或平地，跑公路或小径。你可以以稳定的速度跑步，或者你可以通过一系列间歇来调整你的速度。你可以跑几圈短循环，或者做一个长距离的来回。你可以听音乐，听播客，或者你可以把耳机留在家里，听你自己思考。

当你选择了这些选项中的一个，你就错过了所有其他的。晚上 7 点在公园的铺好的小路上快速跑 3 英里，同时听着无线电广播，在两个喷泉前停下来，这没什么不对的。但是你应该自己去发现如果你改变一个因素会发生什么——也许跑慢一点。在这种情况下，你可能会在跑步结束后感觉更好，下次就不会那么害怕了。

如果你还在挣扎着走出家门，熟悉感可能会让你感觉舒服。如果是这样的话，把熟悉度当作一种训练工具，一旦它不再帮助你，就抛弃它。

因为最终你会发现，当你允许自己跑得快*慢*，长*短*时，你会跑得更好，有时用音乐分散注意力，有时密切关注你的周围环境和你的努力程度。

所以，你们这周的任务是改变一些东西。有什么事情是你一直在做，却忘了问的？找出当你改变它的时候会发生什么。
