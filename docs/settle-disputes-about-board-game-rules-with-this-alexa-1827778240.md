# 用这个 Alexa 技能解决关于棋盘游戏规则的争议

> 原文：<https://lifehacker.com/settle-disputes-about-board-game-rules-with-this-alexa-1827778240>

我是一个非常好胜的人，尤其是在桌游方面。这意味着我将永远是第一个宣布某人“违反规则”并拿出规则手册证明这一点的人。

Watch

也有很多时候，我们拿出一个棋盘游戏来玩，不得不花一分钟来弄清楚一个人实际上是如何玩这个游戏的，尤其是当规则书不知何故从盒子里消失了的时候。

[桌游答案](https://www.amazon.com/Brandon-Telle-Board-Games-Answers/dp/B01FNXM6SW?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/settle-disputes-about-board-game-rules-with-this-alexa-1827778240&asc_source=&tag=kinjalifehackerlink-20) 是一款免费的 Alexa 应用，让处理游戏纠纷或者只是想清楚如何在第一时间玩游戏变得更容易一些。一旦启用，你可以问 Alexa 一些问题，比如有多少人可以玩大富翁，或者谁先玩拼字游戏，然后立即得到答案，而不必翻遍规则书。

这项技能支持许多经典游戏，但如果你玩的是稍微晦涩的游戏，你可能还是得去谷歌上找答案。尽管如此，对于更常见的游戏来说，当你对规则有争议时，这是一个快速获得第二意见的好方法，而不必减慢游戏速度。
