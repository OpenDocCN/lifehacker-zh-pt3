# 接下来我们应该做什么菜？

> 原文:[https://life hacker . com/what-should-we-砂锅-next-1823840675](https://lifehacker.com/what-should-we-casserole-next-1823840675)

大家好，欢迎回到 [*将它砂锅*](https://lifehacker.com/tag/will-it-casserole) ，在这个栏目里我们看到哪些饭菜、菜肴、菜系可以重新想象，改造成砂锅。

我喜欢做砂锅，你们都喜欢告诉我做什么砂锅，但我必须说，我低调地淹没在我们仍然需要砂锅的所有美味概念中。考虑到我们还有很多旧的想法没有执行，让我们在提出新的想法之前从这三个流行的选项中选择一个。以下是竞争者:

*   **费城芝士牛排砂锅:**人们似乎一致认为美国或威士是这里使用的正确奶酪，尽管费城的一个人说 provolone 也可以，另一个人说 provolone 是唯一的选择。我觉得无论我选择什么都会被吼，但这只是在互联网上的一部分。
*   美食广场砂锅菜:我开始做噩梦了，所以也许我们应该把它扔掉。食物*推车*砂锅也是一种选择，可能类似，但可能没有那么痛苦。
*   **全英式煎砂锅:**我们还没有做过早餐砂锅，这将是一个开始的好地方。

所以让我们试着选一个。如果你爱我就选第一或第三个选项，如果你爱看我挣扎就选第二个。(我喜欢奋斗，所以不会苦恼。)
