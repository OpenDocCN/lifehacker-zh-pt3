### Windows 操作系统

**同步你的游戏和智能灯光:** [**色相同步**](https://lifehacker.com/sync-your-lights-to-your-games-and-music-with-this-app-1827437649)

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video--F-n1KgaQIQ&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video--F-n1KgaQIQ&start=0) 

*...我喜欢飞利浦的* [*色调同步*](https://www2.meethue.com/en-us/entertainment/hue-sync) *应用程序——这是一款适用于 Windows 或 Mac 的免费实用程序，可以让你将音乐、电影或游戏与灯光设置相匹配。首先，这很实际:当我在运行一天中的第五场星际争霸 2 合作比赛时，我喜欢有一些东西让我开心。让灯光与我旋转的地图的整体氛围相匹配，增加了沉浸感的乐趣。”*

**使用软件包管理器更轻松地安装和更新应用:** [**巧克力**](https://lifehacker.com/install-and-update-all-your-windows-apps-at-once-with-t-1827588626)

*“如果你想要另一只手来确保你最喜欢的应用程序保持更新，或者从用户维护的更大目录的包(应用程序)中挑选——*[*考虑巧克力*](https://chocolatey.org/docs/chocolatey-vs-ninite) *。不，不是美味的糖果，也不是基于命令行的应用程序安装程序。好吧。算是吧。*[*Chocolatey GUI*](https://chocolatey.org/packages/ChocolateyGUI)*是您想要查看的，因为它为您提供了一个可爱的图形界面来安装和管理您的应用程序，这比大量的命令行输入要容易得多。”*

**将自己喜欢的网站转化为“桌面应用:**[**web 2 desk**](https://lifehacker.com/turn-your-favorite-website-into-a-desktop-app-1827807804)

*"还是那句话，对于一次性使用，*[*web 2 desk*](https://desktop.appmaker.xyz/)*肯定是得心应手的(就算服务* [*也不全是那种独一无二的*](https://github.com/jiahaog/nativefier) *)。虽然我不会将它用于任何已经有离线应用程序的网络服务，比如 Trello、Twitter 或 Whatsapp，举三个 Web2Desk 在其网站上突出显示的例子，但如果你想让你最喜欢的闲置点击游戏在一个独立于你在网络浏览器中所做的事情的窗口中运行，它肯定是有帮助的。”*

**在你的 Windows PC 上获得苹果的快速查看预览:** [**快速查看**](https://lifehacker.com/how-to-get-apples-quick-look-feature-in-windows-10-1827277007)

*“Windows 的 app，巧妙地冠以'* [*【快看】*](https://www.microsoft.com/en-us/p/quicklook/9nv4bs3l1h4s) *，'完全免费。安装它，运行它，你将能够预览“某些文件类型”——开发者没有指定——通过在文件资源管理器中高亮显示它们并按空格键打开预览。(不要按住；轻点它。)并再次点击空格键关闭预览。就这么简单。这款应用会在你每次启动 Windows 时自动启动，这是其主要功能发挥作用的必要条件。”*
