# 洗碗时给手保湿

> 原文：<https://lifehacker.com/moisturize-your-hands-while-doing-dishes-1821750915>

我的工作 50%是写作，25%是做饭，10%是吃饭，15%是洗笨碗。我讨厌洗碗，因为它很无聊，也因为它会让我的手看起来很悲伤。为了对抗悲伤的洗碗手，我发明了悲伤洗碗手美甲，你只需要一些便宜的洗碗手套和乳液就能享受它的神奇。



乳液的话，买一些相当简单轻便的，比如 CeraVe。你不想要太重的东西，否则你的手在手套里会感觉有点太光滑。如果看着你的指甲床让你畏缩，擦上一些角质层油，或者一点椰子油。戴上手套——我喜欢有浅色衬里的手套 [，比如这些](https://www.target.com/p/clorox-comfort-choice-gloves-s-m/-/A-11145990?ref=tgt_adv_XS000000&AFID=google_pla_df&CPNG=PLA_Grocery+Essentials+Shopping&adgroup=SC_Grocery&LID=700000001170770pgs&network=g&device=c&location=9032938&gclid=EAIaIQobChMI_ImEloq92AIVENVkCh3sPQQnEAQYASABEgJvxPD_BwE&gclsrc=aw.ds)——然后在水槽里洗碗。如果你担心这会“令人讨厌”，冷静下来；你的手会感到滋润和柔软，而不是出汗和滑溜。洗碗水的热量会帮助你的药膏吸收，你的手会因此变得更好。(公子哥儿们真的应该接受这个建议；没有什么比一个有一双好手还洗碗的家伙更好的了。)
