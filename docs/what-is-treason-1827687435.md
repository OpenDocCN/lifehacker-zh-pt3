# 什么是叛国罪？

> 原文：<https://lifehacker.com/what-is-treason-1827687435>

叛国罪，即背叛自己国家的行为，是一项严重的指控。根据美国法律，它的定义如此狭窄，执行如此之少，以至于在我国历史上只有几十人面临叛国罪的指控。宪法规定了叛国罪的定义，并指出总统可以因此被弹劾，所以根据我们总统最近的言论和行动，有必要仔细看看什么算叛国罪。

### 叛国需要一个敌人

从字典的意义上来说，叛国可以指任何形式的背叛，但是如果我们谈论的是执行一项关于叛国的法律，我们需要看看它的具体法律定义。在美国, 的宪法第三条第三款中对叛国罪的定义是 [:](https://www.archives.gov/founding-docs/constitution-transcript)

> 对合众国的叛国罪，只包括对他们发动战争，或支持他们的敌人，给予他们帮助和安慰。除非有两名证人对同一公开行为作证，或在公开法庭上供认，否则不得判任何人叛国罪。

根据许多法律学者的观点，你需要一场战争才能有一个敌人。“严格来说，俄罗斯不是敌人，因为我们没有与它交战，”杜兰大学法学院的罗斯·加伯告诉《新闻周刊》的记者。加州大学戴维斯分校的卡尔顿·f·w·拉森同意这一观点，他在今年早些时候为《华盛顿邮报》撰写了关于叛国的 [神话:“与我们正式处于和平状态的国家，比如俄国，并不是敌人。(事实上，将俄罗斯列为敌人的叛国罪起诉无异于宣战。)”然而，他说，基地组织、ISIS 和朝鲜(因为朝鲜战争从未正式结束)可能在法律上被视为我们的敌人。](https://www.washingtonpost.com/opinions/five-myths-about-treason/2017/02/17/8b9eb3a8-f460-11e6-a9b0-ecee7ce475fc_story.html?noredirect=on&utm_term=.e0be8fd0ec31)

“发动战争”和“援助和安慰”也是需要明确的高标准。拉森说，宪法对叛国罪的定义旨在尽可能缩小范围，这样叛国罪的指控就不会被用作政治武器。他写道:“根据宪法，反对政府、削弱政治对手、支持有害政策，甚至将另一个国家的利益置于美国利益之上，都不是叛国行为。”

但是法院的解释才是真正重要的。其他法律学者认为，有理由认为俄罗斯是敌人，特朗普的行为可以被解释为援助。康奈尔大学法学院的 Jens David Ohlin[告诉商业内幕](http://www.businessinsider.com/did-trump-committ-treason-russia-summit-2018-7) 说，因为俄罗斯对我们的网络攻击，或者因为我们在叙利亚的武装冲突中处于对立的双方，我们可能会认为我们已经与俄罗斯处于战争状态。前白宫助理法律顾问安德鲁·赖特告诉《商业内幕》，“我不喜欢用‘叛徒’这个词...这是我希望在更多的调查过程和法律调查结果之后看到的事情。”

### 国会决定了接下来会发生什么

根据宪法，对叛国罪的惩罚取决于国会:

> 国会有权宣布对叛国罪的惩罚，但除非被褫夺公权者在世，否则不得因叛国罪而剥夺公权。

根据英国法律，犯有叛国罪的人可能会被处决，他们可能会受到 [褫夺公权](https://www.britannica.com/topic/attainder) 的惩罚，这种地位通常意味着这个人会丧失他们的财产、公民权利以及他们后代的继承权。(中断继承是“血统腐败”的部分。)

过去被判叛国罪的美国人曾被监禁，或者在某些情况下被处决。一个人可能会被指控背叛联邦政府或某个州。

根据宪法第二条第四款，犯有叛国罪的总统可能被弹劾:

> 美国总统、副总统和所有文职官员，因叛国、贿赂或其他重罪和轻罪而被弹劾和定罪时，应被免职。

“重罪和轻罪”涵盖的范围很广，因此总统可以因各种非叛国罪的事情被弹劾。

[弹劾](http://history) 需要在众议院提出指控。如果众议院大多数人投票支持弹劾，那么参议院将举行类似审判的程序，由最高法院首席大法官主持。两位总统，安德鲁·约翰逊和比尔·克林顿，都面临过弹劾审判，但都没有被定罪。
