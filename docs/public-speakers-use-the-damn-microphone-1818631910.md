# 公共扬声器，用该死的麦克风

> 原文：<https://lifehacker.com/public-speakers-use-the-damn-microphone-1818631910>

“没有这个大家能听到我吗？”登上舞台时说出最差的演说家。如果前排有三个人同意，麦克风就会关闭。然后任何重听的人(或远程聆听的人，或坐在房间后面的人)都听不到你。用那该死的麦克风，你这个混蛋。



艾丽卡·休伊特在一篇针对传教士 的文章中解释了重听人的视角 [，但也适用于任何向人群演讲的人。这个世界上有很多人听力不好，包括需要助听器但又不好意思去配助听器的人。即使 T4 人有助听器，助听器也不是魔法。它们使声音变大，包括房间里的其他声音。没有一个功能可以瞄准说“我会大声说”的人，并将他们的话传入一个人的大脑。](http://www.uua.org/worship/lab/what-youre-saying-when-you-say-i-dont-need-mic)

当你没有拿起话筒时，人们甚至不需要因为耳朵的医疗问题而痛苦。例如，根据房间的不同，你可能无法联系到后面的大多数人。如果正在进行网络直播或电话会议，任何没有对着麦克风说的话都会消失在空气中。

如果你是演讲者，你不仅应该使用该死的麦克风，你应该重复任何没有对它说的话。如果有人问了一个问题，重复给所有人听。如果另一个演讲者想补充几句，也传达一下信息(或者把麦克风递给他们)。

休伊特有更多的技巧来帮助每个人听到你，而不仅仅是前排的那三个人。以下是一些例子:

*   如果有人需要你重复某事，尤其是如果他们不得不问两遍，改变你的措辞。
*   同时尽量减少文字和音乐的层次感(例如，以钢琴音乐为背景的布道)。“助听器切换到音乐模式，”一位听力受损的人说，“你还不如用枕头蒙着脸说话。”
*   如果可能，让某人携带第二个麦克风给正在讲话的人；培养耐心等话筒。

她还指出，直接传输到助听器的听觉回路对残疾人“非常有帮助”；问问你的场地有没有安装。如果你坐在前排，耳朵很敏感，那就帮其他人一个忙。当演讲者问他们没有麦克风是否足够大声时，站起来大声喊:

使用该死的麦克风。
