# 完美的温度是 74 度

> 原文：<https://jezebel.com/the-perfect-temperature-is-74-degrees-1827572516>

一切都是愚蠢的，我们也是。欢迎来到 [*耶洗别有史以来最愚蠢的夏天*](https://jezebel.com/tag/jezebels-stupidest-summer-ever) *，这是一个长达一季的庆典，庆祝我们最糟糕、最白痴的想法和观点。*

Watch

出去吧。我刚刚做了。布鲁克林现在的气温？77 度。宜人的温度。对身体好。微风轻拂的一天。火辣辣的太阳晒在我身上。温暖。又轻又好。然后我想象它低三度。就在那里。完美的温度:74 度。是的，它是。

看看妈妈。大自然的母亲。我更喜欢她正好 74 度。一定有温暖的微风。一股太阳热微风包围着我，空气既不炎热也不寒冷，既不贫瘠也不潮湿，只是温和。我回望太阳的强光，在那里我看到了它:蔚蓝的天空没有遮盖，一点点云彩，温和夏日里的一瞥天堂。

在窗户附近，一束光线射进了猫喜欢在家里烤的地方。公园里，一个油腻的男人躺在草地上。狗狗摔跤晒太阳。洒水车浇灭干裂的草坪。黄蜂潜伏着，让房客们担惊受怕。

鸽子便便。

一个孩子奔跑，跌倒，爬起来，遍体鳞伤。粗糙。感受一下。痛苦过后是光明，阳光，光束。74 度一切正常。我们随波逐流，无意识地醒来，直到黎明。

74 度的生活很美好。
