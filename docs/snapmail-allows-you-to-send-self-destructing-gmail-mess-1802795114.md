# Snapmail 允许你发送自毁的 Gmail 邮件

> 原文：<https://lifehacker.com/snapmail-allows-you-to-send-self-destructing-gmail-mess-1802795114>

向他人发送安全且自毁的信息有很多选择。但是如果你真的只是想用 Gmail 给他们发一条信息呢？ [Snapmail](https://snapmail.co/) 是一个 Chrome 扩展，允许你像其他人一样发送消息给将在 60 秒内销毁的收件人。

当邮件到达时，它们并不完全是 Gmail 邮件。

一旦你写好了你的超级机密邮件(或者只是写下了你的网飞账户的密码)，你就可以按下发送按钮旁边的撰写窗口底部的“快照邮件”按钮。

当您这样做时，您刚刚撰写的电子邮件将被替换为一条消息，让收件人知道他们收到了一封 Snapmail，并且它将在他们访问它后的 60 秒内自毁。

邮件中的一个 URL 会将他们带到一个安全的网站，在那里他们可以阅读你写的内容(希望不是很多，因为他们只有 60 秒)。

之后，收件人将会有一个记录，表明他们收到了您的电子邮件，甚至是主题行，但他们将无法再次访问实际的邮件。

该扩展可以方便地发送诸如密码之类的东西，您希望有人能够使用一次，但也希望确保其他人无法访问。如果收件人的电子邮件遭到破坏，它还可以保护邮件内容的安全。

对于安全爱好者来说，以下是所有这些是如何处理的:

> Snapmail 使用军用级加密作为标准。您的电子邮件使用 AES 算法加密，并存储在 SnapMail 服务器上。当您按下“发送”按钮时，收件人会收到一个指向加密内容的链接。加密密钥在您的浏览器上生成，并包含在链接中。SnapMail 不存储加密密钥，因此无法读取您的邮件。当收件人点击该链接时，电子邮件内容将从 SnapMail 服务器中永久删除。因此，不可能从 Snapmail 服务器提取任何敏感数据。

你还在网上发东西。那么，这是最安全的联系方式吗？不。我不会用这个来发送核密码。但这是最简单的方法之一吗？绝对的。这比仅仅通过电子邮件发送重要密码要安全得多。
