# 我试了一个觅食应用程序，我猜还不错

> 原文：<https://lifehacker.com/i-tried-a-foraging-app-and-it-went-okay-i-guess-1796528131>

著名厨师/森林精灵雷内·雷泽皮 [发布了一款名为 VILD 疯](https://munchies.vice.com/en_us/article/bj8w4w/nomas-rene-redzepi-has-launched-a-mobile-foraging-app?utm_source=munchiesfbus)
的应用程序，帮助你用手机找到“美味的食物”。作为一个喜欢吃好草料的人(看着她的手机)，wilds 马德似乎是一个肯定能提高我生活质量的应用程序，所以我把它下载到这个手机上，然后出发去波特兰东南部的荒野。



几乎每个应用都有我喜欢的东西，也有我不喜欢的东西。我们从好的方面开始。

### 好人

*   **VILD MAD 很漂亮，也很容易驾驭:**主色调是非常讨喜的蓝绿色，感觉舒缓清新，也恰好是我公寓的主色调。食物按字母顺序和月份列出，很容易找到特定的植物部分或简单地浏览当季的食物。
*   **VILD·马德信息量大:**每种食材都至少有三张照片(包括植物本身的特写和你可能发现它的地形的照片)，以及在哪里找到它、何时找到、如何采摘、如何烹饪、味道如何、任何替代品的信息，还有一张标有“错误风险”的小纸条，警告你任何看起来相似的植物可能会毒害你。
*   VILD 疯了说起来很有趣:尤其是当你把它全大写的时候。

### **(有点)坏的**

这不是一个下载就走的交易。虽然有很多信息可以帮助你识别和收集可能潜伏在你家附近的可食用植物，但你必须在出门前熟悉这些信息。没有办法仅仅通过视觉来识别植物；你必须知道它的名字，然后才能找到它的照片。如果该应用程序包含类似于 [Pl@ntNet](http://www.collective-evolution.com/2016/03/10/the-shazam-for-plants-will-identify-any-plant-from-a-picture/) 的功能，这将是非常有用的，这基本上是“植物的 Shazam”，可以帮助你根据自己拍摄的照片识别潜在的野生食物。

还有一个事实是，VILD 疯是丹麦特有的，所以你所在地区的美食可能甚至不在应用程序中。虽然我能找到一些在《VILD 疯狂》中出现的植物，但我发现了更多的非食用植物。

### 我发现了什么

在我家附近闲逛——老实说，那里有一个社区花园式的角落，种满了各种可食用的植物——我找到了四种被列入《VILD 疯狂》的美食:覆盆子、黑莓(还没有完全成熟)、蒲公英和三叶草。我拍了照片并上传到应用程序，它授予我一个“初学者”徽章，并祝贺我如此热爱户外运动。

我还发现了香蜂草、某种非常漂亮的葡萄、未成熟的李子和迷迭香，这些都没有被列入《VILD 疯狂》。不过，我并不特别反对 Redzepi，因为他显然希望这个应用程序在丹麦使用，这是有道理的，因为他是丹麦人。

然而，似乎提供的关于每一个工厂的信息都是准确的。当我发现一些非常漂亮的紫色三叶草(照片在顶部)时，我非常兴奋，立即从花上摘了几片花瓣放在嘴里。“我的舌头感觉怪怪的，”五分钟后我心想，并开始担心三叶草被某种杀虫剂污染了。然后我咨询了 find 马德，看看三叶草是否能对我模糊的舌头感觉负责，并高兴地发现我的经验与应用程序的专业知识相符。

因此，虽然这个应用程序充满了关于寻找许多美味植物部分的有用信息，但它的有用性取决于你的居住地与丹麦有多相似。(然而，它确实有一些非常好的食谱，而且 Redzepi 批准的食谱总是受欢迎的。)
