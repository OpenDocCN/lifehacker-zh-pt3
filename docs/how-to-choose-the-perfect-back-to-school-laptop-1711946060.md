# 如何选择完美的返校笔记本电脑

> 原文：<https://lifehacker.com/how-to-choose-the-perfect-back-to-school-laptop-1711946060>

现在是买笔记本电脑的好时机。返校交易正在进行，新的模型正在上架。我们已经讨论了如何挑选最完美的笔记本电脑，但是如果你要去上大学，或者甚至回到学校，并且不确定你需要什么，你就有困难了。别担心，我们会帮你度过难关。



### 向你的校园 IT 部门咨询想法

为学校选择一台好的笔记本电脑要从了解你的学院或大学推荐给新生的笔记本电脑开始。请咨询您的校园 IT 部门。他们可能已经公布了一套推荐的笔记本电脑和台式机最低规格，以及一些他们认为你应该拥有的软件。此外，与你计划学习的部门核对一下。例如，如果你打算成为一名计算机科学专业的学生，你可能会得到与学习平面设计的人不同的建议。

请记住，部门的建议是一个基线。与你实际能买到的相比，它们通常动力不足。把它们作为一个起点，如果需要的话，就从那里开始。请记住，一旦你决定了一台笔记本电脑，你可以随时将它交给校园 it 部门的技术人员征求第二意见——当然，如果你相信他们的意见的话。当我在母校的校园 IT 部门工作时，与新生讨论规格，并指出他们可以在校内和校外获得学生折扣，这是我工作中最棒的部分之一。

### 考虑你的需求

一旦你有了一些基本的规格，和一些基于你将要做的工作的建议，考虑你自己的需求。你最喜欢的应用程序是什么？一旦你买了笔记本电脑，你想用它做什么？这里有几个问题需要考虑:

*   你想要苹果电脑还是视窗电脑？这在很大程度上取决于个人喜好，但如果你的学校或项目有推荐，那就关注它。可能会有一些软件会影响你的决定(例如，需要 Windows 的 CAD 或工程软件，或者在 OS X 更好的设计软件)。)大多数苹果笔记本电脑都有很好的构建质量，只要你购买了 Windows 许可证，就可以通过 Boot Camp 运行 Windows。你甚至可以从外部固态硬盘 [运行 Windows 来节省空间](http://lifehacker.com/use-a-usb-ssd-for-your-windows-boot-camp-and-save-space-1685242182) 。然而，如果你是一个游戏玩家，你可能更喜欢基于 Windows 的笔记本电脑，只是为了选择。购买 Windows PC 会为您提供更多选项、品牌和总体选择，以找到最适合您的规格组合。不管怎样，拥有“错误的”笔记本电脑意味着你最喜欢的应用程序不兼容，或者你不能和你的同学一起工作的日子已经一去不复返了。你仍然会遇到一点点，但不是主要的，即使这样你也可能会找到替代方案。
*   对你来说多大才算太大？便携性很重要，但同时也要兼顾处理能力和电池寿命。先说屏幕尺寸和重量。15 英寸的笔记本电脑通常具有强大的屏幕空间，但如果您打算在校园里随身携带，13 英寸的通常会轻得多。或者，你可以买一台非常小的 11 英寸便携式笔记本电脑，在你的公寓或宿舍买一台台式机。其实有时候 [买两台电脑比买一台](http://lifehacker.com/when-buying-two-computers-is-cheaper-than-buying-one-5950321) 还便宜。
*   获得尽可能大的电池。每个人都认为他们可以靠几个小时的电池续航时间勉强度日，直到他们在校园里呆了一整天，没电了。检查您的候选笔记本电脑是否有可更换的电池。当然，插座应该不会太难找到，但你可能希望在不得不插上插座之前能够上不止一两门课。更大的电池意味着更重，所以也要记住这一点。当你阅读制造商估计的电池寿命规格时，请确保在[【PCMag】](http://www.pcmag.com/)和 [CNet](http://www.cnet.com/) 等网站的评论中证实这些数字。他们通常会进行真实世界的电池测试，测试将会告诉你实际上你可以用多长时间来做一些事情，如播放电影、处理文档或检查电子邮件。此外，请记住，有时更高端的笔记本电脑实际上可能会有更差的电池寿命，因为它们必须为高分辨率触摸显示屏等功能供电。你可能会发现更普通的机器电池寿命更长。
*   **获得满足您存储需求的合适硬盘**。想想你的一些爱好和兴趣。你认为你会流很多视频或音乐，还是你是下载类型？如果是后者，也许你会想要一台有足够存储空间(或外置硬盘)的笔记本电脑。获得一台带有固态硬盘的笔记本电脑是一个很好的主意——它会很快启动——但高容量固态硬盘可能会很贵，所以要考虑到这一点。

*   您需要哪些外围设备？当你外出时，用笔记本电脑工作很方便，但回到家，在办公桌前，你可能需要一个外接显示器和键盘，以保护你的脖子和背部。即使它们很便宜，也要为外围设备做好预算，当你坐下来工作时，它们会让你感到舒适，并在家里或你的房间里给你 [一个漂亮的、符合人体工程学的工作空间](http://lifehacker.com/top-10-ergonomic-upgrades-for-your-workspace-5409915) 。这包括像笔记本支架、电脑包、充电器等物品。此外，确保你买的笔记本电脑有端口连接你需要插入的东西！
*   你玩电子游戏吗？此处的“是”答案将对您的笔记本电脑规格产生巨大影响。不过，我们并不是说你应该买一台游戏笔记本电脑。众所周知，游戏笔记本电脑又贵又大又重。然而，你可能想看看比你原本考虑的更高端的型号。考虑采用独立显卡、高速固态硬盘、 [清晰、高分辨率屏幕](http://lifehacker.com/how-to-make-windows-work-better-with-super-high-resolut-1711254465) 、更快处理器和更大内存的笔记本电脑。当你找到一个你喜欢的，记下笔记本电脑的显卡，然后前往像 [Passmark 的基准数据库](http://www.videocardbenchmark.net/) 和 [Anandtech 的 GPU 工作台](http://www.anandtech.com/bench/GPU15/1248) 这样的网站，看看这个型号在你最喜欢的游戏中表现如何。另一方面，这是另一种情况，购买两台电脑可能比试图将所有东西都塞进一台电脑要好。经济实惠的便携式笔记本电脑可能是移动办公的理想之选，而经济型游戏台式机可能更适合您的钱包和游戏体验。

这些是让你开始的基础。如果你有特殊的需求，你会有一个更长的问题列表。在每一种情况下，找到你自己的电源到便携性的最佳点。如果你不想带着你的笔记本电脑去上课，而更喜欢用平板电脑(但愿不会，记笔记)，那么便携性对你来说可能不是一个很大的因素。将笔记本电脑从房间带到图书馆，并不需要最新的轻薄硬件。然而，如果你打算每节课都带着你的笔记本电脑，你不会想要在你的包里放一块五磅重的砖头。

### 花在最重要的地方

即使你的预算允许，无论你在看什么样的笔记本电脑，把所有可用的规格都发挥到极致可能都不是一个好主意。你必须做出一些妥协来省钱，但也要避免购买超出你需要的东西。回到你要用笔记本电脑做的事情的清单上，把钱投入到能让这些事情变得更简单的功能上。如果你能找到一台 8GB 内存的笔记本电脑，价格与 4GB 的相同，甚至略高，这是一个扩展预算的好地方。如果你能找到一台配有固态硬盘的笔记本电脑，价格接近传统硬盘的价格，那也是一次值得的升级。另一方面，如果你不使用资源密集型程序或玩游戏，配备高端处理器或显卡的笔记本电脑就没有意义。

同样， [不要为了省钱而退得太远](https://lifehacker.com/what-futureproofing-your-pc-really-means-and-how-it-5808039) 。如果你在寻找配备上一代或低端处理器或低分辨率屏幕的笔记本电脑，你节省的钱将以更令人沮丧的用户体验为代价。记住，功能多一点，让你的笔记本电脑多用一两年，总比功能少而需要升级太快要好。

### 为卓越的构建质量买单

构建质量通常包括屏幕上的铰链有多坚固、键盘在上面打字的感觉如何、机箱有多坚固、其结构中使用的塑料(或金属)类型、触控板在手指下的感觉等等。当你感觉到它的时候，你知道它的脆弱。不要害怕为你喜欢的东西付钱。如果你想要一个磨砂屏幕或背光键盘，寻找那些功能，如果它们让你开心，就多花一点钱。你购买的设备可能会使用几年，所以确保你对购买的产品满意。

说到这一点，当你搜索的时候不要拘泥于“学生”或其他设计师的笔记本电脑。 [《笔记本电脑》杂志解释说](http://blog.laptopmag.com/7-essential-tips-for-picking-a-student-laptop) “商务”笔记本电脑通常具有坚固的规格、超长的电池续航时间和大量的端口，而且价格实惠。制造质量很重要，你购买的笔记本电脑的设计和手感也很重要——你不应该因为这些东西成为你购买决定的一部分而感到难过。

即便如此，你也要确保阅读你考虑的笔记本电脑的评论。你不想为了一个你可能*想要的*闪亮的功能而牺牲一些你实际*需要的*。我们这里有 [几款超极本建议](https://lifehacker.com/five-best-ultrabooks-1615183263) ，我们在 [Gizmodo](http://gizmodo.com/) 的朋友有很棒的 [深度笔记本评测](http://gizmodo.com/tag/laptops) 。线切割也有 [评论和他们最喜欢的选择](http://thewirecutter.com/reviews/the-best-windows-ultrabook/) 。更好的是，如果可以的话，去苹果商店、微软商店、百思买、微中心和 Fry's 等电子商店，亲自动手操作它们(或同一制造商的类似笔记本电脑)。在你买一台笔记本电脑之前，没有什么可以替代实际玩一玩它。



### 利用教育折扣和套餐

当你有了自己的规格和几台候选笔记本电脑要考虑时，明智地购物并节省一些钱。与你的校园 IT 部门核对一下，看看他们是否有采购计划。你也可以去你的校园书店看看他们是否对你正在考虑的系统提供特别折扣(或者任何足以改变你主意的东西)。一些大学甚至有一个电脑展示台，在那里你可以得到和直接订购一样的学生折扣，还有一个学校品牌的闪存盘，一个装你的设备的手提箱，以及一些额外的免费赠品。如果你的校园有一个，值得去看看。

此外，几乎每个校园书店和 IT 办公室都为学生提供打折或免费的软件包。在你从公开市场，甚至从亚马逊这样的网站上购买软件之前，确保它不会比直接从你的学校购买便宜。即使他们自己不卖软件，你也可以直接在制造商的网站上享受学生折扣。几乎每个主要的 PC 制造商都提供学生折扣，许多甚至提供软件或外设捆绑。

我们已经跑了 [几个地方](https://lifehacker.com/grab-student-discounts-on-these-awesome-products-this-s-5845503) 在那里 [你可以得到学生折扣](https://lifehacker.com/save-big-with-this-huge-list-of-60-student-discounts-ev-1667237925) ，包括 [软件深度折扣](https://lifehacker.com/save-big-with-this-huge-list-of-60-student-discounts-ev-1667237925) 。确保你尽可能地存钱。相信我们，你会需要钱买课本的。



### 服务计划和保修:不要忘记细节

最后，不要忘记保修或服务计划。你可能永远都不需要它，但是当你带着笔记本电脑去上大学的时候，你至少需要某种购买保护。我们过去提到过 [延长保修很少值得](https://lifehacker.com/are-extended-warranties-worth-it-5871487) ，但这种罕见的情况通常涉及笔记本电脑。考虑到现代笔记本电脑的维修和进入有多么困难，让其他人来处理屏幕更换和主板维修等棘手的事情是有意义的。此外，请记住，在标准保修结束之前，您通常可以随时购买延长保修，因此，如果您想省钱并等待一段时间再做决定，您不必在购买笔记本电脑的同时购买延长保修。



如果你讨厌延长保修期的想法，至少 [为你的购买使用好的信用卡](https://lifehacker.com/skip-the-extended-warranty-use-a-good-credit-card-inst-5697141) 。您将获得比标准保修更多的保护。如果你是动手型的，去 [iFixit](https://www.ifixit.com/) 看看你的笔记本电脑是否在列表中。你会知道打开和修复自己是多么困难。然后，留出你花在延长保修期上的钱作为“修理基金”，以防万一。最糟糕的情况是，确实发生了一些事情，你已经开始 [决定修理或更换](https://lifehacker.com/should-you-fix-that-broken-gadget-or-just-buy-a-new-one-1711409066) 你的笔记本电脑。

### 当有疑问时，耐心等待

最后，如果你已经有一台笔记本电脑，即使它不是很好，你也可能想等几个月，看看它对你的效果如何。我们知道，闪亮的新机器的诱惑很大，如果你等待，你会错过返校交易。然而，直到你真正到了大学，你才真正知道你在大学里需要什么。一旦你找到答案，你就能更好地为自己购买最好的笔记本电脑。

例如，你现在可能认为你永远不会玩视频游戏，但你的室友或室友可能会让你玩游戏作为放松的方式，然后你会发现自己希望自己买了一台显卡更好的机器。最糟糕的情况是，你用你的旧机器几个月，熟悉一下环境，然后在第一学期结束时参加假日销售。

* * *

对于大多数学生来说，您的笔记本电脑将承担工作和娱乐的双重任务。你还需要能够带着它去上课或者去图书馆。这使得对你买的东西感到真正的快乐变得更加重要。只要稍加研究，您就能确保拥有一台能够满足您在学校需求的笔记本电脑，无论您是在写论文、学习，还是用音乐、游戏或电影来放松自己。
