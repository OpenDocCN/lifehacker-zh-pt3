# 如果言语伤人，那是因为你听之任之

> 原文：<https://lifehacker.com/if-words-hurt-its-because-you-let-them-1827180381>

本周的精选来自马库斯·奥勒留和他的*冥想*，这是他写给自己的私人笔记集，从未打算出版。 [在第四部](http://classics.mit.edu/Antoninus/meditations.4.four.html) 中，奥勒留为我们提供了一个例子，展示了我们对世界的感知能力以及其中那些常常可怕、直言不讳的人们:

> "拿走你的意见，也就拿走了抱怨，"我受到了伤害。“拿走控诉，“我被伤害了”，伤害就被拿走了。不会使一个人变得比以前更糟的东西，也不会使他的生活变得更糟，也不会从外部或内部伤害他。”

或者更简单地翻译为:

> “选择不被伤害——你就不会感到被伤害。不要觉得受到了伤害——而且你没有受到伤害。”

### 这意味着什么

这基本上是“棍子和石头”或“憎恨者将会憎恨”的斯多葛派版本人们说的或做的不会伤害你，除非你允许。你选择被冒犯、伤害、尴尬、愤怒或防御。如果你认为说了什么或做了什么对你来说毫无意义，你就什么也感觉不到，因此不会改变。

### 从中吸取什么

在你的世界里，随机出现的人无法控制你的思想和情感。那些事情都在你的掌控之中，或者至少应该是。如果某个陌生人说了些粗鲁的话或做了些有意让你难堪的事，他们根本没有对你做什么。他们只是在助长自己的愤怒和对自己生活的失控。不要像他们一样。让他们的言行弹开。选择不受他人影响，你就不会受影响。

是的，人们会在身体上伤害我们，我们身边的人会在精神上虐待我们。那些事情不在你的控制范围之内，它们来了就必须处理。但是对于其他的事情——琐碎的评论、侮辱、谎言、对我们人格的攻击——如果你愿意，你会受到保护。正如奥勒留所说，“不会使一个人变得比以前更糟的东西，也不会使他的生活变得更糟……”大多数时候，别人的负面言论或行为根本没有理由会对你产生任何影响。
