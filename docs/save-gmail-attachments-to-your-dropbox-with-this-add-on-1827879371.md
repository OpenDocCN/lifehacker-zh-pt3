# 使用此附件将 Gmail 附件保存到您的 Dropbox

> 原文：<https://lifehacker.com/save-gmail-attachments-to-your-dropbox-with-this-add-on-1827879371>

所有浏览器:谷歌和 Dropbox 目前正在合作开发一款全新的“T2 邮件 Dropbox 插件”，这将使在 Gmail 中直接分享你的 Dropbox 内容变得更加容易。如果你走的是另一条路，直接把文件转储到你的 Dropbox 也容易得多，省去了你不得不打开你的下载文件夹并手动拖动文件的步骤。

Watch

Dropbox 的 [Chrome 扩展](https://chrome.google.com/webstore/detail/dropbox-for-gmail/dpdmhfocilnekecfjgimjdeckachfbec) 做了同样的事情——比这个插件现在做的更多，我稍后会描述——但安装这个插件意味着你可以从任何你想要的浏览器访问这些功能，以及现在的 [Android Gmail 应用](https://play.google.com/store/apps/details?id=com.google.android.gm&hl=en_US) 和未来某个时候的 [iOS Gmail 应用](https://itunes.apple.com/us/app/gmail-email-by-google/id422689480?mt=8)。

要安装 Dropbox 插件，请前往 [G 套件市场](https://gsuite.google.com/marketplace/app/dropbox_for_gmail/33761876029)——忽略它是以谷歌的付费服务命名的事实——并点击 Gmail 插件 Dropbox 上的巨大“安装”按钮。接受插件请求的所有权限，只需几秒钟就可以链接到您的 Gmail 帐户。

拉起 Gmail 和...看起来没什么不同。你可能希望在屏幕右侧的看到某种 Dropbox 图标，在日历、保存和任务按钮下看到——这是 Gmail 重新设计后的新增功能。我也是。正是因为缺少这个图标，才导致许多安装了附加组件的人认为有些地方出了问题:安装没有完成，或者附加组件不知何故坏了。

不完全是这样。Dropbox 在其公告 中调侃道，附加软件是存在的，只是不会做所有事情 [。现在，如果你打开一封带有附件电子邮件，你会看到一个 Dropbox 图标出现在前面提到的地方。点击它，登录你的 Dropbox 帐户(如果你还没有这样做)，你就可以点击附件，并将其保存到你当前的任何 Dropbox 文件夹中。](https://blogs.dropbox.com/dropbox/2018/07/gmail-add-on/)

你不能通过插件为文件创建新文件夹，但如果你需要做更多的组织工作，你可以点击侧边栏顶部的三点图标，跳转到基于网络的 Dropbox 版本。

仅此而已。根据 Dropbox 的说法，该插件的另一半——让你更容易与他人分享你的 Dropbox 托管文件的部分——将于“今年晚些时候”推出这是一个有用的功能，因为你可以直接链接到 Dropbox 中的文件，而不是将它们附加到邮件中，后者会占用你 Gmail 收件箱中的宝贵空间。与删除 Gmail 中的旧附件相比，你更有可能整理好 Dropbox 的 2GB 可用空间，对吗？
