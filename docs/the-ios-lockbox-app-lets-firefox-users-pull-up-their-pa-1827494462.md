# iOS 的“锁箱”应用程序让火狐用户可以快速调出他们的密码

> 原文：<https://lifehacker.com/the-ios-lockbox-app-lets-firefox-users-pull-up-their-pa-1827494462>

***3/26 更新:** Mozilla 的锁箱工具现已可用于*[*Android*](https://play.google.com/store/apps/details?id=mozilla.lockbox)*，此外还有*[*iOS*](https://itunes.apple.com/us/app/firefox-lockbox/id1314000270?ls=1&mt=8)*。*



有很多令人难以置信的密码管理器——我想到了 [1Password](https://lifehacker.com/1passwords-updated-password-manager-is-a-must-have-for-1826379117) 和[LastPass](https://lifehacker.com/you-can-now-use-lastpass-on-multiple-devices-for-free-1788458452)——但是 Mozilla 也有自己的密码管理愿望。该公司刚刚发布了两个新的移动应用程序，作为其 [测试试点](https://blog.mozilla.org/blog/2018/07/10/introducing-firefoxs-first-mobile-test-pilot-experiments-lockbox-and-notes/) 计划的一部分，其中一个是 Firefox Lockbox，这是一个非常方便的方法来获取你在 iPhone 和 iPad 上的密码。

它是这样工作的。如果你是一个 Firefox 的重度用户，并且你没有使用单独的密码管理扩展——只是在你的浏览器中保存你的密码——你是一个尝试 Lockbox 的完美候选人。将它安装在你的 [iPhone 或 iPad](https://itunes.apple.com/us/app/firefox-lockbox/id1314000270?ls=1&mt=8) 上，拉起来，用你的火狐账号登录。一旦你这样做，你会看到你所有的登录和密码保存在一个大的名单。

点击一个条目，你会看到保存的网址——根据你在应用程序设置中的设置，你可以点击它在智能手机的任何浏览器中加载该网站——以及你的用户名和密码。你可以点击其中任何一个来自动复制一个到你的剪贴板，这是方便粘贴到应用程序或网站表单。

仅此而已。

Firefox Lockbox 几乎没有——真的，没有——你会在其他密码管理应用程序中发现的花哨功能，但它的简单性有一些令人愉快的地方。此外，我很欣赏这个应用程序在一定时间后自动锁定(你也可以在它的设置菜单中调整)，它会自动要求触控 ID 或 Face ID 重新验证。

如果你可以在应用程序中编辑你的密码，并将其同步回你的 Firefox 帐户，那将是非常理想的。或者，更好的是，如果您可以在相关的应用程序中调出您的登录凭据，而不是跳出，加载密码箱，点击字段，跳回应用程序，跳回密码箱，点击...你明白了。

尽管如此，对于一个 v1 项目，锁箱完成了工作。期待在未来的某一天看到安卓 的版本。
