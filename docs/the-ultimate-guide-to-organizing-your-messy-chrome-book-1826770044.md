# 整理乱七八糟的 Chrome 书签的终极指南

> 原文：<https://lifehacker.com/the-ultimate-guide-to-organizing-your-messy-chrome-book-1826770044>

悲哀的是 Chrome 用户有一个溢出的书签栏——或者更糟，他们用一个文件夹来存储他们在网上看到的任何有趣的东西。

Watch

在 Chrome 中保持你的书签有序是一件痛苦的事情。如果你不掌握它，你很快就会发现自己有几个月的链接要浏览，却没有时间(或兴趣)去做。解决这个问题很棘手，但并非不可能。在本指南中，我将介绍一些技巧，你可以用它们来避免被书签淹没——如果你已经被书签淹没了，该怎么办。

### 试试 Chrome 的“标签”功能

谷歌 Chrome 并没有为你的书签添加标签以便于组织的内置功能。火狐做到了，这让我们很嫉妒(也很好奇为什么谷歌没有在 Chrome 中实现这个简单的调整)。虽然你可能会在 Chrome 中找到各种支持某种书签标记功能的扩展，但你并不需要它们。

### 甚至不要使用文本

如果你有几个几乎每天都要去的网站，并且你能认出它们的图标，那么你就不需要在溢出的书签栏中用名字来指代它们。编辑书签并完全删除名称，这样你就只有一个图标，这将允许你在上述存储空间中打包更多的链接。(这也适用于文件夹中的书签，当然，只是不太实用。)

如果你收藏的网站没有图标，你可以使用扩展 [来创建你自己的](https://chrome.google.com/webstore/detail/bookmark-favicon-changer/acmfnomgphggonodopogfbmkneepfgnh?hl=en-US) 。或者，如果你觉得懒，就用你最喜欢的表情符号作为网站的“名字”。在你的书签栏中，你仍然会在它的左边看到一个更难看的空白页图标，但这是值得的。

### 创建一个方案

如果你只想坚持最基本的东西——书签和文件夹——那么为了你自己和你的理智，你应该创建某种计划来让自己有条理。你不仅要坚持你选择的任何计划(只把脸书或 Twitter 这样的网站放入你的“社交媒体”文件夹，或者把你必须在截止日期前完成的东西放入你的“待办事项”文件夹)，而且你还必须保持诚实，定期浏览这些文件夹，这样它们就不会塞满多年前的链接。

考虑书签网站，比如整理你的衣橱。如果你在过去的一两年里没有穿过它(或者访问过它，就你的浏览器而言)，很可能你不会去，也许是时候说再见并删除书签了。尽管 Chrome 的书签管理器可以帮助你按名称对保存的网站进行分类，但它不能按照你添加网站的日期进行分类(不幸的是)。

### 使用扩展来帮助跟踪你的书签混乱

有时候，你只是需要一些额外的帮助来管理你的书签。并且有很多扩展可以用来给自己一些组织上的帮助。它们包括(排名不分先后):

*   [**最近的书签**](https://chrome.google.com/webstore/detail/recent-bookmarks/olndffocioplakeilhkgenfgdincjlpn?hl=en) —这是一个很棒的按最近来组织书签的扩展，可以帮助你决定查看什么，保留什么，删除什么。(您可以通过选项屏幕调整扩展列表中的书签数量，并直接从扩展列表中删除书签。)
*   [**超级搜索器**](https://lifehacker.com/supersorter-automatically-organizes-and-cleans-up-your-1636768720#_ga=2.243995974.2139771291.1528827996-1229710871.1517599951)——小心这一个，因为一旦你让超级搜索器在你鼓鼓囊囊的书签文件夹上施展魔法，就没有回头路可走了。然而，我喜欢它能自动按名字排序你的书签，并通过删除重复和清空书签文件夹来整理你的浏览器。虽然如果它也能按时间顺序排序就太好了，但如果你需要浏览器为你处理更多的组织工作，它还是很有用的。
*   [](https://chrome.google.com/webstore/detail/sprucemarks/fakeocdnmmmnokabaiflppclocckihoj/related?hl=en)**—我们开始吧。这个扩展可以自动对你的书签栏、你的“其他书签”文件夹、你的“移动书签”文件夹以及任何子文件夹中的站点进行排序，按照名称、你添加书签的日期或者它们的 URL。如果你需要比最近的书签多一点的功能，但是没有重复的书签或者其他烦人的问题需要管理，这可能是你需要的扩展。** 

### **尝试标签页**

**我们已经介绍了一些有用的扩展，您可以使用它们以有意义的方式组织书签和打开标签，包括[【OneTab】](https://lifehacker.com/onetab-unloads-all-your-tabs-into-a-shareable-list-5990088)和 [Toby](https://lifehacker.com/organize-your-excessive-browser-tabs-with-toby-1826289522) ，但是还有许多其他扩展可以用于将必读书签弹出到前端和中央(这样您就可以查看它们并删除它们，或者至少不会忘记它们的存在)。您可以尝试的其他方法包括:**

*   **[**不起眼的新标签页**](https://chrome.google.com/webstore/detail/humble-new-tab-page/mfgdmpfihlmdekaclngibpjhdebndhdj/related?hl=en) —在你打开的任何新标签页上查看你的书签栏、其他书签和最近添加的书签的好方法。**
*   **[**快速拨号 2 新标签**](https://lifehacker.com/lifehacker-faceoff-the-best-start-pages-for-your-brows-5881052)**——虽然快速拨号 2 并不直接专注于每当你打开一个新标签页时提供一个书签列表，但该扩展确实可以很容易地将最常访问的网站粘在你打开的任何新标签页上。此外，您可以使用悬停侧边栏来快速浏览(和搜索)您的书签栏、其他书签文件夹以及任何其他用于对站点进行排序的文件夹。(如果你只是想要一个书签侧边栏，仅此而已，可以查看一下 [书签侧边栏](https://chrome.google.com/webstore/detail/bookmark-sidebar/jdbnofccmhefkmjbkkdkfiicjkgofkdh/related?hl=en) 扩展。)****
