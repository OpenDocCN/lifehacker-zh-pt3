# Chrome/Firefox 本周扩展:语法上

> 原文:[https://life hacker . com/chrome-Firefox-extension-of-the-week-grammarly-1824182620](https://lifehacker.com/chrome-firefox-extension-of-the-week-grammarly-1824182620)

即使你是下一个斯蒂芬·金或乔治·r·r·马丁——如果你是后者，请试着写得更快——每个人都需要写作的帮手。这就是为什么世界上有编辑，文字编辑，语法教练，一千万本关于写作的书，最重要的是，可以帮助你提高写作水平的浏览器扩展。

Watch

让我们面对它；你可能是一个相当不错的作家，但是自我编辑你的作品可能是一个痛苦的过程。虽然 [Chrome](https://chrome.google.com/webstore/detail/grammarly-for-chrome/kbfnbcaeplbcioakkpcpgfkobkghlhen?hl=en) 和 [Firefox](https://addons.mozilla.org/en-US/firefox/addon/grammarly-1/) 扩展在语法上不能涵盖一切，但它在标记明显的语法错误和可怕的拼写错误方面做得很好。

一旦安装了扩展，Grammarly 将带您完成一个非常简单的入门过程。你被要求指出你通常写哪种类型的文章——为工作、学校或其他项目写作。你还需要自我评估你的平均写作水平:初级、中级或高级。然后 Grammarly 会建立一个关于你是哪种作家的简介，这样它就能为你的作品提供最相关和最有意义的建议。

如果您使用 Grammarly 创建了一个帐户，您的选项将在安装了该扩展的所有位置同步。就是这样！举两个例子，开始键入一封电子邮件或一篇博客文章，语法上会标记出有帮助建议或更正的实例——就像当你拼错一个单词时，浏览器中带下划线的红色小单词一样。

虽然默认是禁用的，Grammarly 也可以兼作字典和同义词库；只需点击浏览器工具栏中的图标，选择“双击显示定义和同义词”选项

如果你在那之后通过 Grammarly 弹出窗口底部的链接点击进入“我的 Grammarly”，然后点击侧边栏上的“个人资料”，你就可以将自定义单词添加到 Grammarly 的字典中。(不用担心；“Smize”已经是公认的。)你也可以改变你的语言偏好——例如，如果你写英式英语而不是美式英语，这很有用。
