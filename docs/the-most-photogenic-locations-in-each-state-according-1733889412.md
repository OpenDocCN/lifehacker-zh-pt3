# 根据 Instagram，每个州最上镜的地方

> 原文：<https://lifehacker.com/the-most-photogenic-locations-in-each-state-according-1733889412>

如果你喜欢在旅行时拍摄令人惊叹的照片，这些是美国人为了捕捉完美照片而聚集最多的地方。

你可能不太关心 Instagram，但它的用户群是巨大的。这意味着从他们的数据中可以收集到很多有趣的东西。旅游网站 [Busbud](https://www.busbud.com/) 的人正是这样做的，以找出美国最热门的地点。你一定会在蒙大拿州的冰川国家公园抓拍到一些美丽宁静的照片，毫无疑问，你会在亚利桑那州的大峡谷拍到一张令人难以置信的照片。当然，有些地方是旅游景点(如帝国大厦或迪斯尼乐园)，但你仍然可以在这些地方拍出不错的照片；尤其是现在 [Instagram 让你拍风景和人像照片](https://lifehacker.com/instagram-finally-allows-landscape-and-portrait-photos-1726966945) 。显然，每个州都有很多上镜地点，但这至少让你知道大多数人去哪里拍好照片。您也可以在下面的链接中找到加拿大的类似列表。
