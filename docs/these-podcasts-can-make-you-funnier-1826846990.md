# 这些播客可以让你更有趣

> 原文:[https://life hacker . com/these-podcast-can-make-you-funner-1826846990](https://lifehacker.com/these-podcasts-can-make-you-funnier-1826846990)

并非所有的喜剧播客都是由斯科特·奥克曼 或 [帮助马克·马龙处理他失败的 *SNL* 海选](http://www.vulture.com/2015/11/marc-maron-and-lorne-michaels-the-entire-two-decade-saga.html) 的人来做的。他们中的一些人研究喜剧和它是如何运作的，足够有意识地让你可以向他们学习。这里有四个很棒的播客可以让你变得更有趣。

Watch

## [好一个](https://headgum.com/good-one-a-podcast-about-jokes)

在每一集里，一个喜剧演员为秃鹫的高级编辑杰西·戴维·福克斯表演他们的一个笑话，然后花一个小时追溯它的起源，它的演变，以及它如何融入他们工作的主题。福克斯和每位嘉宾谈论他们的喜剧哲学以及他们如何写笑话。(大多数单口喜剧“在舞台上写作”，但正如福克斯所说，这对每个喜剧演员来说都有不同的意义。)

并不是所有被审查的笑话都来自于单口相声。 [保罗·费格](https://headgum.com/good-one-a-podcast-about-jokes/paul-feigs-spy-knife-fight) 讲述自己电影*间谍*中的一个场景； [*SNL* 编剧胡里奥·托雷斯和杰里米·贝勒](https://headgum.com/good-one-a-podcast-about-jokes/saturday-night-lives-wells-for-boys) 谈论他们的小品《男孩的井》玛丽亚·班福德谈论她的毕业典礼演讲。

在最近一集 的 [中，福克斯与卡梅隆·埃斯波西托谈论了她的新特别节目《强奸笑话》，这是一个关于性侵犯的概念，不仅仅是笑点。](https://headgum.com/good-one-a-podcast-about-jokes/cameron-espositos-rape-joke)

## [**《杰姬与劳里秀》**](https://nerdist.com/podcasts/the-jackie-and-laurie-show/)

因为没有嘉宾介绍和认识，这个节目的共同主持人(经验丰富的喜剧演员杰姬·卡希安和劳里·基尔马丁)可以深入具体。在他们最近的一集 中，他们讨论了女性漫画在忧郁中工作的压力，在这个过程中，他们调整了一个密宗性笑话的措辞，并考察了网飞对其单口相声特辑的期望。很高兴听到两位喜剧演员不仅仅是分享故事，而是真正讨论工艺和行业的实用性。

## [**如何制作现场喜剧**](https://www.podomatic.com/podcasts/howtoproducelivecomedy)

在这部迷你剧集中，主持人艾米丽·温特教授如何制作现场喜剧节目，邀请嘉宾解释如何主持、如何从你的节目中赚钱以及如何进行企业演出。冬天也出现在 [*的一集里，好的，那个爹的，和丑的*](https://www.ogpodcastnetwork.com/episode/ep-77-just-a-joke-with-emily-winter/) 谈论给你的伴侣讲笑话。

## [论喜剧写作](http://boardwalkaudio.com/oncomedywriting/)

这多是典型的像马龙或皮特·霍姆斯那样的喜剧演员访谈节目，多是涉及嘉宾的职业生涯。但在每集的结尾，主持人艾伦·约翰逊会向嘉宾抛出一个笑话或小品，并与他们一起制作。所以你可以听到来自 SNL、Cracked、T2、每日秀、纽约客和 Lifehacker 的姐妹网站 Clickhole 的喜剧演员现场表演。

我希望他们能把这一段做成完整的播客。通过注意客人问了什么问题，以及他们给了约翰逊什么选择来把他的笑话带向一个方向或另一个方向，你可以学到很多关于制作喜剧的实际情况。
