# 用 Sinemia 的新“无限”计划取代您已取消的 MoviePass 订阅

> 原文：<https://lifehacker.com/replace-your-canceled-moviepass-subscription-with-sinem-1829116759>

如果你一直在哀悼失去 MoviePass 的无限电影计划，现在你可以选择从其最大的竞争对手那里获得类似的交易。Sinemia 周一宣布，它将为 [提供一种新的“无限”机票选择](https://www.sinemia.com/new-plans/us/los-angeles) ，每月 29.99 美元。



该计划允许订户在美国、加拿大、澳大利亚和英国每天获得一张 2D 电影票。

我之前是 MoviePass 的用户(2013 年)，今年早些时候我实际上注册了 Sinemia，当时它每月免费提供 25 美元的 Restaurants.com 礼券和一部电影《T2》。

该服务与 MoviePass 有点不同，你可以提前购买电影票，这意味着你还可以选择座位(这几天对我来说是必须的)。

在它的应用程序中，你选择你想去看的电影和去哪里，然后应用程序激活一张数字信用卡，你可以用它在影院的网站上购票，就像你用自己的信用卡买票一样。当你去看电影时，你必须使用应用程序进行登记，以确认你确实是使用所购门票的人。据我所知，剧院里没人知道你用 Sinemia 买了票。

如果你不需要每天都去电影院，Sinemia 还提供个人和家庭票选项，从 4.99 美元/月的一部 2D 电影开始，到 89.99 美元/月的三天电影，每月六个人。

如果你还在为失去无限电影通行证订阅而悲伤，那么这个可能正是你所需要的来缓解痛苦。
