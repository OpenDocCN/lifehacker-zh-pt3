# 第一个带屏幕的 Google Home 已经比亚马逊的 Echo Show 提供了更多的隐私

> 原文：<https://lifehacker.com/the-first-google-home-with-a-screen-already-offers-more-1821910986>

[亚马逊](https://lifehacker.com/the-best-place-for-your-echo-dot-is-actually-on-the-cei-1820916306) 可能是第一个提供智能家居显示器的 [Echo Show](https://gizmodo.com/the-amazon-echo-show-is-the-best-dumb-smart-machine-in-1796380588) ，但是 [谷歌](https://lifehacker.com/should-you-buy-a-new-google-home-speaker-1819148716#_ga=2.27966780.1017631282.1515423304-1313785359.1499701416) 刚刚在一个关键领域击败了对手:隐私。

Watch

与联想合作，搜索巨头本周在 2018 年[CES](https://gizmodo.com/here-comes-ces-2018-the-biggest-gadget-show-in-the-wor-1821789975#_ga=2.27966780.1017631282.1515423304-1313785359.1499701416)上发布了其 [新型智能显示器](https://www3.lenovo.com/us/en/events/ces/?cid=ww:lenovosocial:bpvfzr#Shot12) 。这款设备运行谷歌助手和其他一些谷歌服务，目前还没有第三方支持。与 Echo Show 和 Echo Spot 一样，它也有一个用于进行视频通话的摄像头，但与亚马逊的设备不同，它有一个物理快门，用于在不需要使用时盖住镜头。

对于担心隐私的人来说，这是一个巨大的福利，尤其是如果你打算把设备放在某个私密的地方，比如你床边的床头柜上。盖住摄像头就像扳动设备侧面的硬件开关一样简单，所以你不需要担心谷歌(或者更糟糕的是，一些随机黑客)在你睡觉的时候监视你。

除了我们对隐私的担忧，谷歌的新产品比亚马逊的产品多了一个优势。它可以播放 YouTube 视频，而 Echo Show 不能，原因是两家公司之间正在发生争吵。你还可以使用其他谷歌应用程序，比如用于视频通话和谷歌照片的 [Duo](https://lifehacker.com/google-starts-rolling-out-duo-its-super-simple-1-to-1-1785336507) 。

然而，谈到定价，亚马逊仍然提供最好的交易。联想智能显示器 8 英寸屏幕的起价为 199 美元，10 英寸版本的起价为 249 美元(这两种型号都被列为“即将上市”，没有确定的发布日期)。 [Echo Spot](https://www.amazon.com/dp/B073SQYXTW/ref=fs_ods_rk?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-first-google-home-with-a-screen-already-offers-more-1821910986&asc_source=&tag=kinjalifehackerlink-20) 便宜很多，仅售 130 美元，而 [Echo Show](https://www.amazon.com/Amazon-Echo-Show-Alexa-Enabled-Black/dp/B01J24C0TI?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-first-google-home-with-a-screen-already-offers-more-1821910986&asc_source=&tag=kinjalifehackerlink-20) 售价 230 美元——这两个版本现在都有售。

当然，谷歌有可能在未来推出自己的更便宜的硬件，尽管不能保证这个版本会像联想一样包含关闭摄像头的开关。
