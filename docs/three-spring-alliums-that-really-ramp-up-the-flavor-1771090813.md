# 三个春天的蒜真的增加了味道

> 原文：<https://lifehacker.com/three-spring-alliums-that-really-ramp-up-the-flavor-1771090813>

春天来了，它带来了各种美味的蒜。从超级受欢迎的斜坡到甜而害羞的青蒜，这些洋葱附近的植物部分给它们接触的所有东西带来了新鲜、美味的味道。

### 扩大生产

如果你关注任何种类的美食媒体，你可能已经注意到厨师们因为一个叫做“的小东西而失去了他们热爱葱的头脑。”有时他们被称为“野生韭菜”，但这并不像“坡道”那样有趣(最有趣的事实:餐馆老板和前厨艺大师评委乔·巴斯蒂尼克有一个名为“坡道”的乐队让我们给他们听听。)

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-iQWZJztfiTA&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-iQWZJztfiTA&start=0) 

总之。这些又瘦又嫩的小家伙看起来像小韭菜，有着宽阔的绿色叶子。坡道是阴暗的主题，从黑暗，树木繁茂的地区觅食。它们的辛辣、蒜味、洋葱味和短暂的季节(通常在四月底到六月初)使它们非常受欢迎，但你应该可以在当地的农贸市场或像全食超市这样的高档杂货店找到它们。

它们的价格不菲(根据食客 的说法，每磅[20 美元)，所以要准备好游戏计划，确保你把整个东西都吃了，从球茎到叶尖。我认为坡道最好简单处理。《好胃口》推荐](http://www.eater.com/2015/4/10/8384255/everything-youve-always-wanted-to-know-about-ramps) [用少许橄榄油、盐和胡椒](http://www.bonappetit.com/recipe/grilled-ramps-with-sherry-ramp-vinaigrette) 将它们整个烤熟，然后用一些切碎的球茎、少许油和雪利酒醋、第戎酱和一点蜂蜜做一份辣酱，这样就加倍美味了。

如果你决定你非常喜欢斜坡，你只想把它们倒在你的整个生活中，你可以用简单的比例 **1:2:2:8 (1 份坚果，2 份油，2 份磨碎的奶酪，8 份叶子或香草)制作香蒜酱，加上大蒜，柠檬，盐和胡椒调味**。拌意大利面，涂在面包上，用勺子吃；你们知道怎么处理香蒜酱。

生吃它们也没什么不好，切碎后撒在涂有黄油的好面包上，或者作为任何受益于大蒜和洋葱的食物的配菜。(所以，基本上除了冰淇淋什么都有可能？)

坡道最好储存在冰箱里，卷在一个潮湿的纸巾里，放在一个未密封的塑料袋里。它们会让你的冰箱闻起来像大蒜，但这还不是冰箱闻起来最糟糕的味道。如果你担心在它们变坏之前无法通过你的斜坡供应，我们可以腌制。对于两磅斜道， [沙维尔推荐](http://www.saveur.com/article/Recipes/Pickled-Ramps) 一汤匙盐，一杯白醋，一杯糖，芥末籽、粉红胡椒粒和白胡椒粒各一茶匙；茴香、葛缕子和孜然籽各半茶匙；外加一片月桂叶。之后，只需要一点煮沸、搅拌和冷却，然后你就有了美味的泡菜，可以在冰箱里保存两周。

### 带着蒜苔逃走

每个人都知道如何处理大蒜植物的球茎，但是如何处理从硬茎大蒜 植物的球茎上长出的又直又卷的茎呢？这些是蒜苔，很好吃。花葶通常在春末夏初收获，就在它们开始开花之前。它们脆脆的，温和的，甜甜的，类似于细香葱和大葱，比球茎有一种不太强烈(但仍然存在)的蒜味。蒜薹看起来和摸起来有点像绿豆，可以生吃也可以煮着吃，只要记得去掉多纤维的顶端。

不知道该拿它们怎么办？就像使用洋葱和韭菜一样使用它们。将它们整个烤熟，作为配菜，切片，炒成薯条，或者焯一下，加入沙拉中。我最喜欢做的事情是在午餐中用绿豆代替的尼斯沙拉，这样会更刺激。和上面的斜坡一样，香蒜酱也是一个很好的场景。你可以把它们添加到基本的罗勒香蒜酱中，或者做一份只有花葶的酱，来一次非常蒜味的体验。鹰嘴豆泥中的 Scapes 也很棒，只要扔几根切碎的茎进去，然后闪电般的离开。事实上，几乎所有我们用斜坡覆盖的东西都可以用于景观:烧烤，腌制，都很好。

如果你想养肥你的 scape(为什么不呢),考虑制作 scape 复合黄油。如果你不熟悉复合黄油的神奇之处，上面的视频(以及我们的花式乳品指南 )可以带你经历这个非常简单、非常美味的过程。

Scapes 在冰箱的塑料袋中可以保存几个星期，在冰箱中可以保存一个月，但我不认为他们能保存那么久。此外，还有腌制，这是延长任何东西寿命的最美味的方法。

### 走向绿色大蒜

春天给我们的另一个礼物是青蒜，这是在鳞茎完全形成之前收获的年轻大蒜。它在早春来临，所以留意一下农贸市场，或者在你的 CSA 盒子里。它基本上是小大蒜，有一个可爱的粉红色小鳞茎，连着一根长长的葱状茎。像大多数年轻人一样，这种未成熟的葱更软、更甜，更容易顺其自然，可以煮熟或生吃。(我猜成熟的大蒜可以生吃，但是有点粗糙。虽然我不认识你，但你可能是个嚼着生大蒜的坏蛋。)

因为它含有较少的热量，所以这是一个很好的方法，可以为容易被淹没的精致菜肴添加一些大蒜风味，如扇贝，或者为容易被大蒜淹没的人，如我的父亲。虽然它比成熟的大蒜温和，但它仍然比普通的大葱更有攻击性，绿色的茎可以用来给鸡蛋、米饭或任何你会放大葱的东西增加额外的味道。

除了“像使用普通大蒜一样使用它”之外，在一些领域，绿色大蒜比它的前辈更出色。它在沙拉中非常棒，比如来自 Kitchn 的这款大麦沙拉，它结合了大麦、青蒜、烤杏仁、糖屑和薄荷，是一道超级新鲜和超级美味的菜肴。如果你想吃得多一点，但又不太重， [这些精致的小土豆蛋糕](http://www.thekitchn.com/recipe-mini-potato-pancakes-with-green-garlic-and-chivesrecipes-from-the-kitchn-168860) 利用青蒜和香葱，成为一种美味、香脆、可口的小吃。

当然，香蒜酱总是有的。事实上，我想不出任何一种蘸酱或酱料不受益于青蒜带来的甜味和咸味，所以你应该把它融入所有食物中。
