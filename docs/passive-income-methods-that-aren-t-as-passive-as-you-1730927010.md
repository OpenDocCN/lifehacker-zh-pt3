# 大多数被动收入并不像你想象的那样“被动”

> 原文：<https://lifehacker.com/passive-income-methods-that-aren-t-as-passive-as-you-1730927010>

睡觉挣钱有什么不可爱的？这就是 [被动收入](https://en.wikipedia.org/wiki/Passive_income) 背后的想法，这是你不用做太多额外工作来维持就能定期赚取的收入。你先投入一定的时间或金钱，然后当你坐下来放松时，让金钱滚滚而来。理论上听起来很牛逼，但实际上往往不是这样。

对于被动收入，想法是建立一个自动为你赚钱的系统。该系统通常需要大量的前期工作，甚至还需要一些偶尔的维护，这使得它不是真正被动的。

使用其中的一些方法，“偶尔维护”比看起来要实际得多。在 之前，我们已经 [讨论过这些技术，虽然它们理论上是被动的，但许多从中赚钱的人会告诉你有相当多的工作正在进行。这并不是说这些方法不值得追求，而是你想知道这些方法所涉及的工作。](https://lifehacker.com/the-complete-guide-to-making-money-in-your-spare-time-1291903155)

### 出租房产需要很多维护费用

无论是你家里的一个房间，还是你购买的一整栋房子，出租房产经常被认为是获得被动收入的一种流行方式。你把钱投资在房地产上，找到一个租户，然后逐月获得回报。看起来简单又被动，对吧？但是任何一个租房业主都会告诉你:没那么容易。

正如一位所有者 [在金钱树下的财经博客上解释的那样:](http://www.underthemoneytree.com/the-passive-income-myth/)

> 以下是我经常做的一些事情，来赚取所谓的被动租金收入:
> 
> *   处理周六早上不可避免的“锅炉坏了”的电话
> *   定期检查您的物业是否需要“非紧急”维修，如果不进行维修，从长远来看会让您付出巨大的代价(例如，漏水=长期潮湿)
>     
> *   优化我的
>     
> *   为迁入/迁出租户安排盘点/入住
>     
> *   筛选潜在的新租户
>     
> *   做好记录并填写纳税申报表中的财产部分
> *   执行一些详细的税务规划，以决定何时支付首付买房出租抵押贷款 【购买房产只是为了出租】
>     
> *   不断提醒自己 [的买让规则](http://www.underthemoneytree.com/the-buy-to-let-rules/)

即使你雇佣了一名经理，你也不可避免地要处理维护问题、职位空缺、意外开支、问题房客、HOA 会议等等。如果你雇佣了一家物业管理公司，那会花掉你收入的一大块。

好消息是，如果你积极维护你的财产，你也许可以把你的工作记为""据美国国税局称:

> 如果您或您的配偶积极参与了一项被动租赁房地产活动，您可以从您的非被动收入中扣除高达 25，000 美元的损失。这项特别津贴是不允许损失超过被动活动收入的一般规则的例外。类似地，在考虑了这种例外情况下允许的任何损失后，您可以从活动中抵消高达 25，000 美元的非被动收入税。

因此，虽然租金收入在技术上是被动收入，而且可能是最常见的例子，但它确实需要相当多的持续工作。以至于国税局要扣除一部分。

### 销售一本书意味着不断地推销它

另一种常见的赚取被动收入的方式是出售书籍或其他信息产品，如在线课程或网上研讨会。你写一次，上市出售，并在未来几年里从中赚取版税。当然，这可能需要大量的前期时间和努力。大多数人都为此做好了准备，但你可能会对推销你的书或产品所需的主动维护感到惊讶。正如经历过 [的作者迈克尔·埃尔斯伯格解释的](http://www.forbes.com/sites/michaelellsberg/2011/06/24/the-top-4-reasons-why-passive-income-is-a-dangerous-fantasy) :

> 当然，你可以在互联网信息产品、联盟营销或其他人们容易被“被动收入”的幻想所吸引的领域诚实地赚钱。但是，要想在可持续的长期发展中真正赚钱，你必须像对待其他业务一样对待这些业务。换句话说，你必须为有真正需求的真正客户提供真正的价值。
> 
> 你能做到的唯一方法就是积极主动。如果你一直认为你的客户想要与你互动是一件“令人头痛”的事情，或者你必须在为客户提供更多价值方面超越竞争对手并在创新方面超越竞争对手，那么你就走错了路。

除非你是一个著名的作家，你想从你的书里赚更多的钱，你就会更积极地去写它。营销你的书意味着征求评论，与其他作者建立关系网，甚至可能是更新博客和维护一个不断增长的电子邮件列表。还有与读者互动，跟上社交媒体，回复评论，等等。自出版作者 [乔安娜·佩恩甚至说](http://www.thecreativepenn.com/2012/10/12/help-my-book-isnt-selling/) 作为一名作者赚取被动收入意味着你将不得不写*多本*书来建立你的存在和信誉。

### 保持你的博客或网站的相关性

的确，你可以通过联盟营销或广告从网站或博客中赚钱。但这并不意味着没有正在进行的工作。

首先，只有当人们真的访问你的博客时，你才能赚钱。只有当你的博客与他们真正想要阅读的有用信息相关并且经常更新时，人们才会来到你的博客。也有推广的。当然，你可以雇佣其他人来为你写内容，但是就像作家兼博客写手乔纳森·菲尔兹所说的那样，你成为了一名雇主和经理。他补充说，是的，有少数博客拥有如此庞大的常青内容库，他们能够产生持续的关注和收入。然而，这些博客作者花了几年的时间来建立他们的内容和网站。

说到写博客，前期工作和持续不断的工作之间的界限很微妙，很难将写博客归类为被动收入。从博客中赚取“被动”收入通常需要多年的持续努力。

有些人认为博客是被动收入，因为你在做你喜欢做的事情，而且无论如何你都会去做，你只是碰巧从中赚钱。这太棒了，但本质上，这只是意味着这是一项有趣的工作。无论你喜不喜欢这份工作，为了获得收入，它仍然是必要的。

* * *

不可否认，被动收入的想法很棒，但它在实践中比在理论上更加主观和复杂。真正的被动收入的一个很好的例子是投资或赚取储蓄账户的利息。你实际上投资了一笔前期资产——在这种情况下是钱——然后你就可以自动地继续赚钱，而不需要做任何其他事情。

然而，几乎所有其他被引用为“被动”收入的例子都比我们被引导去相信的需要更多的主动维护。这些都是不错的赚钱选择，但是如果你打算接受它们，要做好前期和后续工作的准备。
