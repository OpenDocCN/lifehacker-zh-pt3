# 我试着用食物来化妆，但效果不佳

> 原文：<https://lifehacker.com/i-tried-using-food-as-makeup-and-it-went-poorly-1796312670>

耶女士是一个非常有创意的 YouTube 名人 ，她以不同寻常的烹饪视频而闻名，但她最近的视频显示，这位有进取心的互联网明星将食物*放在她的脸周围*，而不是里面。

首先，我只想说我喜欢这个视频。有一个有趣的故事，一只办公室鸡，*和*奥利奥耳环——这就是你在一个视频中想要的一切。的情节相当简单:在办公室工作了一整夜后，耶发现自己需要尽快赶去参加一个“网络名人节”。她戴上一张口罩(关键的一步)，然后在办公室里转来转去，寻找可以用来打造完美形象的食物和办公用品。

耶女士最终看起来非常漂亮——几乎*太*漂亮——我受到启发，想看看我是否可以用类似的产品创造类似的外观。

虽然我找不到 Yeah 用来化妆的确切成分，但我找到了一些非常相似的东西，并开始制作自己的烹饪化妆品，我心中不无疑虑。(虽然我不反对在美容应用中使用可食用成分 ，用钢笔和自来水画眼睛看起来是个糟糕的计划，但我不是一个在尝试之前就批评它的人。)看看上面的视频就知道结果了，但下面你会找到如何制作每一款美容产品的图片和说明。

### 脸

我找不到 Yeah 使用的糯米粉的确切品牌，但我不认为这个品牌会有那么大的区别。我在碗中加入一些面粉和可可粉，然后在水中搅拌，直到我的混合物达到视频中显示的食物基础的稠度。我还撒了一点杏仁油。(是啊用了芝麻，但是我出了。)

最终的产品粗糙、粘稠，看起来一点也不像粉底。就覆盖范围而言，它当然覆盖了我的脸，如果我说它没有分散我对毛孔的注意力，那我就是在撒谎，毛孔是我一直担心的问题。这当然是我用过的最去角质的粉底了，但是也许 Yeah 用的是一种不太粗糙的面粉。

我试着把纯米粉拍在脸上作为粉末，特别注意 t 区，但那也不太好。我还尝试了“用可可勾勒轮廓”的技巧，虽然说起来很有趣，但并没有给我想要的卡戴珊式的颧骨。(剧透提醒:我脸上没有颧骨。)

### 眼睛

耶用速溶咖啡、她同事的头发、钢笔、彩虹糖和——再一次——粘米粉的组合让她的眼睛闪闪发光。我跳过了头发睫毛——尽管 Beth 主动给我寄了一些她的——因为它们看起来有点太高级了，我无法完成。我尝试了所有其他的方法，在我的眉毛上涂咖啡(实际上效果不错)，用钢笔在我的眼睛上画线。

小贴士:不要用钢笔在眼睛上画线。钢笔又尖又脏，不应该放在眼睛附近。我知道会这样吗？是的，但是我是一只垃圾熊猫，为了一点点什么都愿意做，而你比那更好。

我没有像 Yeah 在她的视频中那样做一个完整的调色板，而是专注于一种单一的颜色:一种漂亮的青柠绿。我把一些青苹果彩虹糖——谢天谢地，酸橙又回来了——浸泡在一点水中，直到它们所有的光彩都消失了，然后把糖水和一些米粉混合在一起，做我的影子。它很漂亮，但是当我试着把它擦在我的眼睛上时，它就卷成小碎屑，从我的眼睑掉到我的脸上。

### 嘴唇

为了得到一双撅起的嘴唇，是的，我把火龙果汁和某种番茄酱状的物质混合在一起。我找不到火龙果，但我手头有一些草莓，所以我把它们捣碎，用番茄酱混合。然后，我把这种难闻的混合物抹在嘴唇上，尽量不让自己呕吐。即兴噘嘴涂料当然使我的嘴唇看起来又红又亮，但它没有一吨的持久力，它让它们尝起来像番茄酱，我一点也不喜欢。

在以上所有的 DIY 美容秘籍中，唯一一个有点接近真货的是速溶咖啡眉粉，但即使这样看起来也有点滑稽。也许这是我的技术。我不太擅长化妆。
