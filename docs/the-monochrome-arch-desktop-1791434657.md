# 单色拱形桌面

> 原文：<https://lifehacker.com/the-monochrome-arch-desktop-1791434657>

读者 [Linux Jess](http://kinja.com/eyecandylinux) 向我们的 [桌面展柜](http://kinja.com/tag/desktop-showcase) 提交了这款由 Arch Linux 驱动的单色桌面，我们不禁对其进行了特写。它干净、优雅、易于复制。以下是方法。



Linux Jess 对于这个设置只有几个移动的部分，对于 Linux 用户来说都不会陌生。如果你想要同样的造型，你需要做以下事情:

*   Flickr 上保罗·斯科特的 [城市阴影](https://www.flickr.com/photos/49866381@N05/31741299735/in/explore-2016-12-19/) 壁纸
*   [Conky](http://conky.sourceforge.net/) ，一款针对 Linux 系统的轻量级系统监视器
*   本 [自定义 Conky 配置](http://speedracker.deviantart.com/art/Loco-Panda-Grace-conky-650071619)
*   将 [萨迪图标](http://erikdubois.be/category/sardi-icons/) 由 [埃里克·杜布瓦](https://plus.google.com/+ErikDubois) 替换为默认图标集

大概就是这样。只有几个部分，但一个真正鲜明和现代的效果。如果你喜欢它的外观，并对如何让它成为你自己的有疑问，就去 Linux Jess 的 kinja 博客(链接如下)提出你的问题——并在桌面上赞美你！
