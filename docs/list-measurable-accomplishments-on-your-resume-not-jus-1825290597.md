# 在简历上列出可衡量的成就，而不仅仅是责任

> 原文:[https://life hacker . com/list-measurable-achievements-on-your-resume-not-jus-1825290597](https://lifehacker.com/list-measurable-accomplishments-on-your-resume-not-jus-1825290597)

简单地在简历上列出你过去的工作职责对招聘经理来说，就像说明你精通微软办公套件一样有帮助。如今，雇主们希望看到有分寸的结果。

Watch

“文件中缺乏衡量标准和结果是我最大的简历通病，” [高管简历撰稿人和职业战略家](http://www.careerimpressions.ca/) 艾德丽安·汤姆告诉 [商业内幕](http://www.businessinsider.com/biggest-resume-mistake-makes-recruiters-cringe-2018-2) 。"没有任何成功的衡量标准，这份文件缺乏技能的证明."

正如我 [之前](https://lifehacker.com/keep-track-of-your-work-accomplishments-throughout-the-1823436243) 所写的，列出成功的衡量标准，而不仅仅是你的职责，会“向你的雇主展示你一直在做的工作的结果”，这比说你以前有过工作更有价值。“例如，在简历的‘职责’部分，写‘撰写新闻稿’没问题，但是‘每周撰写 10 篇新闻稿’会给人留下更深刻的印象，”茱莉亚·盖纳为 Monster.comT5T7】写道。

那么你怎么知道要包括什么呢？Gaynor 写道:“找到量化你的成就的最简单的方法就是看看你是如何直接影响公司的金钱、时间和人员的。”以下是一些你可以包括的数字示例，从 [LiveCareer](https://www.livecareer.com/career/advice/jobs/quantifying-accomplishments) :

> *   Sales, number of goods/units sold
>     
> *   A won contract/bid
> *   Increase in profitability
>     
> *   Increase in shareholder value
>     
> *   Number of customers served
> *   Number of direct subordinates and number of people under management
>     
> *   Number of people you employ
>     
> *   The size of your team.
> *   Number of times you have been selected as a team or project leader.
> 
> *   Number of awards
>     
> *   Published quantity
>     
> *   Number of successful grant applications

如果你很难想出具体的数字，那就翻遍你的日历、笔记本、 [成就框](https://lifehacker.com/keep-track-of-your-work-accomplishments-throughout-the-1823436243) 或者任何你正在追踪你的成就的地方，找出具体的例子。

### 其他提示

一旦你有了你的要点，确保你提供了上下文。你可以说你“销售了 100 万美元”，但这意味着什么呢？在你被雇佣之前，销售是什么样的？详细说明这些信息会给你的简历增色不少。

如果你没有确切的数字，数字的范围通常是好的，只要你能在面试中证明你的成就。例如，正如《缪斯细节》 中的，“负责指导本科生研究人员”很容易变成“每年指导 7-12 名本科生研究学生，这些学生后来都去了天体物理学、物理学或数学的研究生院。”

此外，您应该添加特定的客户端和关键字。例如，你可以说，“领导耐克和其他名牌客户的营销活动”，而不仅仅是“领导名牌客户的营销活动。”
