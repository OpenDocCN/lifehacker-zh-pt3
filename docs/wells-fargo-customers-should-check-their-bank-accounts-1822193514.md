# 富国银行的客户应该检查他们的银行账户

> 原文：<https://lifehacker.com/wells-fargo-customers-should-check-their-bank-accounts-1822193514>

富国银行的在线账单支付系统周三晚上出现了“小故障”，导致一些支付被处理两次， [公司称](https://twitter.com/WellsFargo/status/953819605587415041) 。据报道，问题是 [在周四早上修复了](https://twitter.com/WellsFargo/status/953983244512489474) 。



一些 Twitter 用户表示，他们的购物和 ATM 透支费用被重复计费，导致一些人透支。其他人说，由于故障，他们无法购物，包括食品杂货。

富国银行没有立即回应置评请求，但富国银行企业沟通副总裁史蒂夫·卡尔森(Steve Carlson)告诉“任何因这一错误而产生的费用或收费都将得到处理。”

在这种情况下，你没有太多的办法去阻止它——但是如果你是富国银行的客户，一定要检查你的账户。不过，总的来说，你应该通过你的银行设置警报，这样你就知道你的钱去了哪里。

有许多不同类型的提醒，你可以选择进入帐户和卡(并通过文本，推送提醒，或电子邮件接收)。你可以决定是否只在花了一定数量的钱时、付款到期时、接近信用额度时才接收它们，或者你可以选择每天更新帐户。

其他一些警报，根据，包括:

*   可疑交易

*   无卡交易(对于欺诈保护尤其有价值，因为欺诈者在许多情况下只需要您的卡号就可以尝试使用它)
*   汽油交易(根据 creditcards.com 的说法，“加油站是小偷的另一个热点”)
*   国际交易

你也可以使用 Digit 和 Qapital 这样的预算应用程序，它也会向你发送更新信息。

如果你注册了短信提醒或推送提醒，你需要下载你的银行应用。这里是去哪里 [报名富国银行的](https://www.wellsfargo.com/online-banking/alerts/) 。
