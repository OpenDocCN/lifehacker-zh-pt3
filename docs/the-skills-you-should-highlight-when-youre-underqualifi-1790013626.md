# 当你不能胜任一份工作时，你应该强调的技能

> 原文：<https://lifehacker.com/the-skills-you-should-highlight-when-youre-underqualifi-1790013626>

你看到了一份你想申请的工作——但是有一个问题，你不能满足所有的要求。尽管如此，你知道不合格的 [和不合格的](https://www.themuse.com/advice/how-to-know-if-youre-underqualified-or-unqualified-for-a-job-theres-a-difference) 之间的区别，你自信自己属于前者而不是后者。*你可以做这份工作。*

唯一的问题是，无论你是在换领域还是有一条的曲折职业道路，你的简历都缺少一些所需的技能。如果是这样——不要放弃！你的目标是寻找任何地方来连接你以前的经验和公开的角色之间的点。

好消息是，有两种技能可以帮助你证明自己:

### 首先，关注你的可转移技能

当你发现差距时，首先考虑可转移的技能。你可能以前听说过这个术语，但是作为一个快速复习，它们就像它们听起来的那样:你在一个行业学到的东西会带到另一个行业。

例如，如果你有客户服务经验，你知道如何让不开心的人感到被倾听，并把愤怒的对话变成富有成效的对话。这将帮助你在任何需要积极倾听和解决问题的工作中取得成功。或者，也许你从未参与过正式的战略规划过程，但你帮助你的部门制定并执行了大的目标，所以你可以利用这段经历。

当涉及到在你的申请中连接这些点时，你基本上是在说:“这个角色需要 X，而我在 Y 方面的经验给了我这样做所需要的技能。”

这是关键的一步(这篇文章 告诉你如何在简历中做到这一点，而 [这一篇](https://www.themuse.com/advice/the-perfect-cover-letter-template-to-show-off-your-skills) 在求职信中)，但它没有给你机会分享那些让你引人注目但并不适合所列职位的能力。因此，虽然可转移技能确实有用，但止步于此就是止步不前。

### 这就是附加技能发挥作用的地方

就像公司列出的 [梦想资历](https://www.themuse.com/advice/3-steps-to-applying-for-a-job-when-you-dont-meet-the-requirements) 一样，这些资历并不重要，但会让候选人更有吸引力；很可能你有一些不需要的天赋，但是会增加你做好工作的能力。这些是你的附加技能。(我用“附加”这个词是因为我们的目标是关注那些会*增加你完成工作的能力的因素。)*

脱离了上面的公式，这些就转化为，“我有 Y 方面的经验，尽管这在工作描述中没有要求，但它将帮助我在这个角色中脱颖而出，因为…”

引用我以前的一篇关于 [面试一个遥不可及的角色](https://www.themuse.com/advice/the-secrets-to-interviewing-for-a-role-thats-slightly-out-of-reach) 的文章，“附加技能是你带来的独特的东西——除了所有预期的东西。想想看:如果你稍微不够资格，这是有原因的。如果你职业生涯的头两年是在不同的行业度过的，你会从那个行业获得经验。如果你比申请这个职位的其他人都年轻，那么你很有可能提交了一封非同寻常的求职信，或者拥有令人印象深刻的人脉。

你有三年而不是五年的项目管理经验吗？这是因为你花了两年时间做数据科学家，你的分析能力将帮助你简化项目。或者你在国外呆了一年，虽然不需要第二语言，但你知道你流利的语言会帮助你接触到新的人群。

### 如何利用它们成为你的优势

作为一名求职者，最重要(也是最具挑战性)的事情之一就是要令人难忘——因为正确的原因。这就是你的可转移技能和附加技能帮助你的地方。所有那些资历相当的人都会在他们的经历和角色之间画一条直线。但是它们可以和其他明显的选择融合在一起。

你不同的技能给了你优势，因为它们帮助你脱颖而出。首先，在你的求职信中，你可以同时使用可转移技能和附加技能来讲述一个两行的故事。例如，假设你是一个喜欢写作的项目经理，申请一份公司招聘人员的工作:

可转移技能:“我目前工作的一部分是鼓励人们在截止日期前完成工作，我会用我在那里学到的知识让申请人及时提交材料。”

附加技能(Additive skill):“作为一个每个月都会创建自己的社交通讯的人，我知道自己寻找合适词汇的习惯会加强我通过电子邮件招聘的努力。”

一旦你获得了面试机会，你就可以(再次)利用这些技能来展示为什么你是一个令人信服的候选人。

例如，如果你被要求解释在没有招聘背景的第一天你如何应对这个角色，你可以这样回答:

“除了(招聘信息中要求的)经验，我还带来了(可转移或附加技能)，这将帮助我(做一些其他候选人可能做不到的事情)。”

听起来是这样的:

“写有说服力的文章很大一部分是理解你的读者，我想用同样的技巧来联系那些可能对申请犹豫不决的顶级申请者。

“当你没有确切的资格时，很容易感到紧张，或者对你拥有的经验打折扣。这时，审视你所有的技能——而不仅仅是符合职位描述的技能——可以给你带来你需要的额外动力。它们很好地提醒了招聘经理，虽然你还没有发展出与职位直接相关的技能，但你已经获得了大量其他宝贵的经验，这让你成为一个不错的选择。
