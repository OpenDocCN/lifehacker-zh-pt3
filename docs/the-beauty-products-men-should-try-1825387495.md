# 男士化妆入门指南

> 原文：<https://lifehacker.com/the-beauty-products-men-should-try-1825387495>

谁说男人不能使用化妆品和其他针对女性的美容产品？在当地化妆品店的一些好心人(和我的女朋友)的帮助下，我找到了一些即使是最有男子气概的男人也能谨慎舒适地使用的东西。相信我，人们唯一会注意到的是你有多好看。

### **去角质磨砂膏**

没有什么比你脸上的一堆硬皮更糟糕的了。不，不要挑剔它。不要把皮肤浸在乳液里，希望它会消失。那层皮是死的，需要去除！去角质磨砂膏会起作用，所以你的保湿霜实际上会起作用。好的磨砂膏还能让你的皮肤焕发光彩，抵消肤色的暗淡。

我尝试了两种去角质磨砂膏，都主要由天然成分制成。对于我的脸，我用了 [鲜糖面部上光剂](https://www.amazon.com/Fresh-Sugar-Face-Polish-5/dp/B071471G16/?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-beauty-products-men-should-try-1825387495&asc_source=&tag=kinjalifehackerlink-20) 那是红糖和草莓籽的混合物(吃起来很香)。对于我的嘴唇，我用了 [咬的龙舌兰糖唇蜜](https://www.sephora.com/product/agave-sugar-lip-scrub-P415980) 。它们都非常相似，所以你可能只需要一个非常好的去角质磨砂膏就可以了。洗脸，用磨砂膏，洗掉。

### 面部润肤霜和润唇膏

伙计们，给你的脸保湿。科学证明 [保湿霜让你的皮肤看起来更好，帮助它更好地充当外界的屏障](https://vitals.lifehacker.com/lifehackers-essential-science-based-guide-to-skin-care-1824029261#_ga=2.215286103.1186395594.1523885983-542336946.1523479293) 。每天至少保湿一次，但是如果你比较活跃或者长时间在户外，每天保湿两次。我的皮肤很白皙，所以我用的保湿霜也含有广谱防晒指数 15 的防晒霜，比如 [露得清无油保湿霜加防晒霜](https://www.amazon.com/Neutrogena-Oil-Free-Moisture-Spectrum-Sunscreen/dp/B01MRO5W6U/?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-beauty-products-men-should-try-1825387495&asc_source=&tag=kinjalifehackerlink-20) 。在你使用唇部磨砂膏去除死皮后，不要忘记使用保湿润唇膏，它会确保你的嘴唇看起来很漂亮。

### **底漆或面漆**

如果你倾向于有一张闪亮的脸，或者你的润肤霜似乎增加了你的光泽，打底可以帮助你变得更亮。这种神奇的液体使你的皮肤光滑，并通过给你的皮肤一个哑光效果来减少光泽。如果毛孔太容易引起注意，它也能减少毛孔的出现。我用了 Smashbox 的超级受欢迎的美照菲尼莎基础入门 ，效果非常好。适用于你的 T 型区和其他任何需要晚上出去的地方。

半透明爽身粉类似于底漆，它给你的皮肤一个哑光的效果，隐藏毛孔，减少光泽。缺点是，如果你不小心，它可能是这里最显眼的产品。过度扑粉会让你看起来化了妆，尤其是如果你有胡须的话。我试用了蕾哈娜的[Fenty Beauty invistatte 吸粉](https://www.fentybeauty.com/invisimatte/blotting-powder/22346.html) ，它很好，很轻。几乎看不出我有抹。

### 彩色校正器

我第一次了解色彩校正器的魔力是在网飞新版的《酷儿眼》中。美容专家乔纳森·范·内斯推荐了一种叫做“绿棒”的东西来帮助改善节目中一位化妆主题的肤色。这个人的皮肤有刺目的红色底色，一点绿色校正器大大减少了红色。我的皮肤也有红色底色，所以我给了 [丝芙兰商店品牌的绿色校正器](https://www.amazon.com/SEPHORA-COLLECTION-Bright-Future-Correctors/dp/B01BJVIXU2/?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-beauty-products-men-should-try-1825387495&asc_source=&tag=kinjalifehackerlink-20) 一个尝试，它甚至帮助我的皮肤好了很多。

如果你没有红色底色，黄色校正器可以帮助隐藏棕褐色皮肤上的紫色或其他较暗的色调，橙色隐藏深色皮肤上的深色唾液，桃色隐藏浅棕褐色皮肤上的蓝色或紫色，薰衣草色隐藏黄色色调，粉红色隐藏白皙皮肤上的蓝色色调。如果你不确定自己的肤色，可以找化妆品店的人聊聊。当你选择了合适的颜色后，轻轻涂抹在任何需要均匀色调的地方，比如眼睛下方或鼻子周围，然后用手指或化妆棉混合。

### 遮瑕膏

遮瑕膏是这个列表中最容易化妆的项目。它完全隐藏了你的瑕疵，抚平了你眼睛下面的黑眼圈，并在你的肤色略有不同的地方给你一个均匀的肤色。难的是根据特定品牌的颜色选择来确定你的肤色。只有一种方法可以做到这一点，那就是走进商店，在你的手背上试一试。

我最终登陆了 [Kat Von D 的 Lock-it 遮瑕膏](https://www.sephora.com/product/lock-it-concealer-creme-P409985) 因为它不太厚，持续时间长，而且不会像油漆和折痕一样(明显看出你在用遮瑕膏)。它很好地隐藏了我的一个瑕疵。你只需涂一点点，然后用手指轻轻搅拌。
