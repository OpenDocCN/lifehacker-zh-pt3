# 用这个公式要求加薪

> 原文：<https://lifehacker.com/use-this-formula-to-ask-for-a-raise-1823616050>

当你要求加薪时，你会更成功地关注你带来的价值，而不是你自己的需求。 你可能不想脱口而出，“嘿，我有账单要付，你已经三年没扣我工资了”——即使这是真的。那么*怎么做*你说呢？

Watch

把你的要求想成一个公式会有所帮助，这个公式看起来有点像这样: **Intro + Value + Price +结论。**以下是每个因素的要点。

**简介:**首先，用简短的介绍开始你的请求。类似于，*“非常感谢你同意和我见面。我已经在[Initech]工作了[5 年]。在那段时间里，随着公司的成长，我也在帮助公司成长。”*

**价值:**然后，提醒他们你的价值。列出一些你最近的主要成就(这就是为什么，尽管听起来很老套， [保留这些](https://twocents.lifehacker.com/keep-track-of-your-work-accomplishments-throughout-the-1823436243) )会有帮助)。尽可能量化。“我已经帮助公司在短短三个月内接触了 2000 名新客户。我创建了一个演示文稿，现在已经在我们的其他五个办公室实施。像我这样有这种经验的人，标准工资是 5.5 万美元。”

价格:在你确定了自己的价值后，说出你的价格。你到底想要什么？例如，*“我希望我的薪水能与我在这家公司的成长相匹配，所以我想要求每年加薪 5000 美元。”*

**结语:**最后总结一下。高调结束，表达你的感激之情。专注于创造更多价值。*“我致力于帮助这个团队在未来更加成长，所以我感谢你的考虑。”*

你不会想照着稿子念，因为那听起来像是照本宣科。也就是说，记住这种结构是有帮助的

当你试图根据自己的成就和技能，准确地找出如何加薪的理由时。
