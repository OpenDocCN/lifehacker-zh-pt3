# 用氧化苦艾酒保存水果

> 原文：<https://lifehacker.com/preserve-fruit-in-oxidized-vermouth-1822599722>

如果你喜欢喝马提尼酒或曼哈顿酒，你的酒吧推车上可能会有几瓶苦艾酒。如果你是一个喜欢奇特开胃酒 的人，你可能会有更多。这些强化葡萄酒非常有趣，但是由于其较低的 ABV，它们不会永远持续下去。



*这是《与克莱尔一起吃垃圾》的一部分* [*，这是一个生活黑客系列，克莱尔·劳尔说服你将你的厨余转化为可食用的美味。*](http://lifehacker.com/tag/eating-trash-with-claire)

人们似乎很难就苦艾酒的保质期达成一致——有人说 [在冰箱里可以保存一个月](http://www.seriouseats.com/2014/07/best-way-to-store-vermouth-for-cocktails-fridge-vs-winesaver-rebottling.html)； [有人说是三个](https://www.drinkhacker.com/2009/01/21/how-long-does-vermouth-last/)——但是一旦你注意到你的鸡尾酒尝起来有点“变质”，是时候用一瓶新的来补充你的供应了。然而，旧的、氧化的瓶子不需要被扔掉。

首先，你可以用它来烹饪，用它来给锅脱釉或给调味汁增添魅力，就像你用来做酒一样。(“坏”的苦艾酒不会毒死你，它只是不能调制出最好的鸡尾酒。)这是一个很好的用途，但我喜欢更花哨一点，用轻微氧化但仍然可用的酒来注入一些美味的水果。

就像这些 [菠萝皮](https://skillet.lifehacker.com/make-pineapple-infused-gin-with-the-leftover-peels-like-1795233709) 和 [这些樱桃](https://skillet.lifehacker.com/will-it-sous-vide-a-boozy-chocolate-cherry-sundae-1795352092) 一样，这个过程很简单，无论你让每个人在柜台上待几天，还是使用浸泡循环器来加速这个过程。把你选择的水果切片——我选了血橙，因为它们很漂亮——把它们扔进罐子里，在上面倒上你的陈年苦艾酒。如果你没有一个真空瓶，就让你的装有酒和水果的罐子在室温下放置几天，每隔一段时间轻轻摇晃一下。如果你有循环器，把罐子放在 185 度的浴缸里一个小时，然后放在冰箱里冷藏。

没有必要过滤任何东西——它在冰箱里放的时间越长，水果就变得越多——只要你想给饮料或菜肴增加一点多汁性，就可以随时从罐子里取出浸泡过的水果。这些橙子显然是制作鸡尾酒的绝佳配菜，但当它们被挤在清淡的意大利面和海鲜菜肴上，或与酱汁和汤一起炖以获得一丝复杂的酸味时，味道会非常好。
