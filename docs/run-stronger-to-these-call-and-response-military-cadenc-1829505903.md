# 随着这些应召响应的军事节奏跑得更快

> 原文:[https://life hacker . com/run-strong-to-these-call-and-response-military-cadenc-1829505903](https://lifehacker.com/run-stronger-to-these-call-and-response-military-cadenc-1829505903)

晨跑可能会很艰难，但是通过团队合作、友情和对死亡的欢快歌唱来度过它是有帮助的。幸运的是，所有你喜欢的流媒体服务上都有军事跑步节奏。

Watch

节奏是呼唤和回应的圣歌，对于跑步来说，每分钟大约 180 步。(也有行进节奏，大约每分钟 110 拍，但这些不是我们在这里想要的。)在 Spotify、iTunes 或您选择的音乐来源中搜索专辑系列 [【以节奏跑步】](https://open.spotify.com/album/31twElbOpbA5iWXPTLRZ5c?si=97l38lElRtW95SwHcg1OvA) 或 [【以节奏跑步】](https://itunes.apple.com/us/album/workout-to-the-running-cadences-u-s-military/4960526) 。抑扬顿挫听起来像这样:

 [https://lifehacker.com/embed/inset/iframe?id=youtube-list-ET17-QJRQ9E%2FPL_fZXT29Qpn0oTS4buQBoNjoeznsU1_y0&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-list-ET17-QJRQ9E%2FPL_fZXT29Qpn0oTS4buQBoNjoeznsU1_y0&start=0) 

节奏是为日常训练打气的歌曲——正如一名跑步者描述的“团队合作，共患难”，不是为比赛或计时跑准备的。节奏是一个很好的适应，大声唱歌有助于确保你不会喘不过气来。前几天我和 [陆军空降](https://open.spotify.com/album/31twElbOpbA5iWXPTLRZ5c?si=M2EvhOH3QaKvOgxkvLY-Pw)[海军陆战队](https://open.spotify.com/album/45dsl8VzLvdwt8ztRiE59I?si=fezDbwuUQvGLlTIshBvC2g) 和 [海豹突击队](https://open.spotify.com/artist/0M1U7jlYhD3Uns4cx09UhN?si=MWzKzXnXT1edZnqzJjP5aA) 一起晨跑，我唯一的抱怨是赛道太短了——每条只有一两分钟。但是每张专辑都有很多，所以你不会用完的。

早上第一件事就是思考战争和死亡，这也是爱或恨的一面。“如果我死在战区，把我装起来，用船送我回家”有无数种不同的说法，这个主题融入了日常锻炼和军事生活的痛苦。海豹突击队员 [唱着](https://open.spotify.com/track/3kCxcTcak8Y4suxqbc6y2i?si=lnHVq-YASgqVnMeTql6bsA) “魔鬼，魔鬼，你不要害怕，因为我们生活在地狱般的地方。“陆战队员 [唱](https://open.spotify.com/track/0X4ir2DNiXtVQ621HhRdad?si=sZx0B0_LQMmkzKa0uPfRGA) “痛！/你在哪里？然后确定疼痛的位置(“在我的大腿上！”“在我胸口！”)并命令它走开。

这些曲目通常以您将听到的前一两句歌词命名，但每首曲目的内容通常都令人惊喜。这是因为抑扬顿挫并不是典型的独立歌曲，而是由当天召唤它们的人即时组合而成。有很多“左右左”和“一，二，三，四嘿。”夹杂着关于酋长如何打败超人的故事，或者实际歌曲的封面(像戈登·莱特富特的《清晨的雨》 中的 [，我发誓我在一首军队歌曲中听到了一些琼·杰特)。](https://open.spotify.com/track/0poJwC887vIjjE4gqOLdlV?si=2KAZDvJQTcWmpVsGnpLgXg)

有些曲目会让你开心，有些会激励你。尝试将 [这个](https://open.spotify.com/track/0M1dsFv4ZX5AGGAXdpP0yP?si=SVnYFzdrTK2imlAUCWjL0w) 添加到您的锻炼列表中，看看您是否不会带着微笑结束跑步:
