# 工作和娱乐两用工作空间

> 原文：<https://lifehacker.com/the-work-and-play-dual-purpose-workspace-1744405907>

今天的特色工作区一部分是战斗站，一部分是高效的家庭办公室。总之，这是一个令人印象深刻的设置，有许多小工具值得欣赏。

Watch

这个装置属于 Imgur 用户 breoidneo，他在他的工作空间的照片中详细描述了一切。桌面是由胡桃木硬木定制的，工作侧有两台 Dell U2415H 显示器，游戏侧有三台 U3415W 显示器，Varmilo 机械键盘，工作侧有一台 Dell XPS 13，以及为 battlestation 定制的 PC。点击下面的链接查看更多细节和照片。

如果你有自己的工作空间可以炫耀，请将它添加到我们的 [Lifehacker 工作空间展示和讲述](http://www.flickr.com/groups/lifehacker-workspace-showandtell/pool/) Flickr 池中，与我们分享。确保您包含的任何照片至少为 640x360。将它们保持在 16:9 也有帮助！包括一些文字，关于你使用的材料，你是如何想出这个设计的，以及任何其他相关的细节。如果你巧妙的组织和良好的设计感吸引了我们的眼球，你可能就是下一个特色工作区。
