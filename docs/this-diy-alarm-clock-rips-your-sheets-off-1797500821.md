# 这个自己动手做的闹钟会撕掉你的床单

> 原文：<https://lifehacker.com/this-diy-alarm-clock-rips-your-sheets-off-1797500821>

深度睡眠者可能需要比标准闹钟更强的东西。有些人让你把他们扔向墙壁，而其他人 [在地板上滚来滚去哀号](http://www.nandahome.com/) ，但他们没有一个人采取直接行动叫醒你。家具制造商和 [家具](http://www.instructables.com/member/1up%20Living/) 用户[1 up lifing](http://www.1upliving.co.uk/)决定让他的闹钟更加实用，增加了一个令人愉快的功能，就是在早上从他熟睡的身体上拉下床单。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-OXHLP22HkMc&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-OXHLP22HkMc&start=0) 

为了给我的噩梦做闹钟，1upLiving 在他的床下安装了一个改良的绞盘和线轴来缠绕他的羽绒被。一个 Arduino 微处理器连接到他的闹钟上，当闹钟响起时，它会向绞盘发出信号，让绞盘转动，并慢慢地将羽绒被从床上滑下来。



你总是可以建造你自己的 ，但是在建造一个精心设计的叫你起床的装置之前，你应该试着早点睡。
