# 如果你今天过得不好，试试“1-2-3 策略”

> 原文:[https://life hacker . com/if-you-have-a-tough-day-try-the-1-2-3-strategy-1827238273](https://lifehacker.com/if-youre-having-a-tough-day-try-the-1-2-3-strategy-1827238273)

我们都经历过——你的一天过得很糟糕，感觉永远不会结束。最糟糕的是，感觉你做什么都没办法让它变好。但这并不完全正确。这个策略不会解决你所有的问题，但它是一个创造性的方法，可以让你度过艰难的一天。

当事情让你沮丧时，St. Onge 说你应该试着遵循以下三个简单的步骤:

1.  **做一些你必须做的事情**:如果你 [最终从你的待办事项清单](https://lifehacker.com/how-to-get-rid-of-your-scariest-to-dos-1825597837) 中划掉一些事情，你可能会感觉好一点，尤其是如果这些事情与你的正常工作日无关的话。什么事情做完后会让你感觉更好？

2.  做一些你想做的事情:善待自己。想看书，看电影，吃点冰淇淋，或者去大自然中散散步？动手吧。 [让自己开心至少一小会儿](https://lifehacker.com/why-self-care-is-so-important-1770880812) 。

3.  **为别人做点什么**:帮助别人 [让你感觉良好](https://lifehacker.com/ten-things-you-can-do-to-be-happier-backed-by-science-1065356587)——这已经被证实了。它给你一种目标感，也给你一个机会去反思你自己可能需要帮助的问题。还有，很好看。去做好人吧。

不管你的一天有多糟糕，这个策略将确保你有所收获，你至少在练习一点点自我照顾，你在努力让世界对某人来说变得更好。你会感觉更好，或者至少能安然度过这一天的剩余时间，因为你改变了你的常规 。
