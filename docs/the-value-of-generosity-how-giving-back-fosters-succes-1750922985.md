# 慷慨的价值:回馈如何促进更多的成功

> 原文:[https://life hacker . com/the-value-of-course-how-feeding-back-fosters-success-1750922985](https://lifehacker.com/the-value-of-generosity-how-giving-back-fosters-succes-1750922985)

有时候获得成功的最好方法是先帮助别人。我们能做的帮助别人的小事——帮朋友一个忙，或者在某人运气不好时帮他买单——往往会以积极的方式回归。你自己的慷慨会促进成功。

*本帖最初出现在 Ramit Sethi 的博客上，* [*我会教你变得富有*](http://www.iwillteachyoutoberich.com/blog/most-generous-thing-ive-heard-in-a-long-time/) *。*

我最喜欢的一个关于慷慨大方的故事来自《纽约客》 的这篇文章中的酒店经营者史蒂夫·韦恩。

史蒂夫决定以 1.39 亿美元的价格出售他得到的一幅毕加索的画。(是的，一幅画卖了 1.39 亿美元，在当时是“已知的艺术品最高价格”。)那个周末他邀请了一些朋友来看这幅画，当他站在一旁向朋友们解释这幅画时，事情 [并没有完全按照计划](http://www.newyorker.com/magazine/2006/10/23/the-40-million-elbow) 进行:

> 永利患有一种眼病，即色素性视网膜炎，这种疾病影响了他的周边视觉，因此，偶尔会影响他与附近物体的互动，而且，在没有意识到的情况下，他在说话时后退了一两步。“然后我用右手做了个手势，”韦恩说，“我的右肘撞到了照片上。它刺破了画面。”有一种明显的撕裂声。永利转过身，在玛丽-泰蕾兹·沃尔特的左前臂上，在这幅画的右下象限，“一个轻微的刺伤，一个两英寸的裂缝。我们都停下来了。我说，‘我真不敢相信我刚才那么做了。哦，妈的。“哦，伙计，””

> 永利又转过身来。他把小手指放进洞里，发现一块帆布被推了回去。
> 
> 他告诉他的客人，“嗯，我很高兴是我做的，而不是你。”

这是我听过的最惊人最优雅的话之一。他没有咒骂或发火，只是说:“我很高兴是我干的，而不是你。”换句话说，他做得比你好——至少他付得起。

当我想到“富裕的生活”时，我不仅仅想到穿漂亮的衣服或拥有最好的公寓。我考虑慷慨地对待我的时间和金钱。事实上，有时一只小小的援助之手比一张 1 万美元的支票更有意义。

史蒂夫·韦恩的评论教会了我很多关于慷慨的东西。作为一个年轻的小伙子，我没有真正得到慷慨。我想过“支付我的公平份额”，但我没有像现在这样把慷慨作为我的核心价值观之一。

我记得教会我慷慨的一些小事。我记得当我还是学生的时候，我邀请商界人士和首席执行官与我共进午餐，他们总是付钱。100%的时间。尽管是我问他们的，他们还是同情一个大学生，付了钱。我永远不会忘记。每当我遇到一个年轻的大学生，我总是付钱。

我记得收到一份小礼物——只是一张简单的手写卡片——来自一个我帮助过处理一些商业事务的人。我会永远记住它。我还记得曾帮助某人在纽约找到一份工作，但几个月后才从别人那里得知。当她再次发邮件给我，要求我再帮一次忙时，我没有理睬她。

我还记得给人们送小礼物，看他们反应的喜悦。有时候，它是我在亚马逊上找到的一本书，或是地铁上一个有趣标志的照片。让人们知道你在想他们是一种恩赐。

具有讽刺意味的是，我开始变得越慷慨，我就变得越成功。有时，这种联系是显而易见的:我帮助某人，他们做介绍，然后突然我被邀请参加一个全国性的电视节目。(发生了这件事。)

其他时候，慷慨会以意想不到的方式回来。几年前，我和朋友去俱乐部，我的夹克不见了——一件价值 1000 美元的夹克。我的朋友帮我寻找它，当它不在我们桌子周围的任何地方时，我感到可怕的失望。

然后我耸耸肩说:“我比别人强。”我的朋友不敢相信。“这不打扰你吗？”当然这让我很困扰。但我知道我可以再找一个。对于几年前在经济上难以承受的事情，能够保持冷静是我非常感激的。

慷慨现在意味着我没有好的理由就送礼物给我的朋友。我喜欢买单。我在有钱之前就这样做了，现在也一直这样做。

想一想你可以变得慷慨的方法。不一定要贵。

你在街上看到过让你想起你朋友的东西吗？为什么不把照片发短信说“想你”呢？如果你看到你妈妈喜欢的书，就送给她，并在封面上写下“我爱你”。对她来说，收到这样的礼物真是一种享受。

有一次我搞不清楚怎么系鞋带，就问我爸。他突然大笑起来，然后给我看。下一次我系鞋带时，我给他发了一张照片，并向他表示感谢。想象一下从你儿子那里得到这些是什么感觉。

慷慨也来自于把人们联系在一起。我的一个朋友尼克邀请我去他家和其他有趣的人一起参加活动。当我为《纽约时报》撰稿时，另一个朋友给了我一个建议:“尽可能多地联系朋友。”传播爱。我反复说过，金钱只是富裕生活的一小部分。我是真心的。我不只是希望你考虑如何获得最大的加薪，或者开始最赚钱的生意。

我可以提供所有这些方面的建议，但接下来呢？我从一个完全专注于成就和数字的人，变成了思考我希望自己的核心价值观中更柔和的一面。慷慨是我最引以为豪的事情之一。

我没有东西卖给你。我没有行动号召。我只是想让你想想你的朋友会怎么说你。“慷慨”会出现在他们的前三个词中吗？你想这样吗？
