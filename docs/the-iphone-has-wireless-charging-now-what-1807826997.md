# iPhone 有无线充电功能。现在怎么办？

> 原文：<https://lifehacker.com/the-iphone-has-wireless-charging-now-what-1807826997>

感应充电标准被几乎所有主要制造商采用仅用了五年时间，但苹果公司最终将 Qi 感应充电(也称为无线充电)添加到其移动设备中。当然，迟到总比不到好，但人们需要关心有没有另一种方式给他们的设备充电，或者那根超长的电缆够好吗？无线充电是否物有所值，或者潜在的兼容性问题？

### **感应充电的工作原理**

[感应充电](https://en.wikipedia.org/wiki/Inductive_charging) 依靠嵌在设备或设备外壳中的线圈，称为感应线圈，以及产生电力的电磁感应，将能量从一个设备传输到另一个设备。充电板内的感应线圈与智能手机内的感应线圈结合在一起，形成一个电磁场。智能手机内部的线圈将电磁场中的能量转化为电流，并为你的设备充电。

当谈到感应充电时(至少在如此小的范围内)，邻近性很重要，这就是为什么感应充电通常需要用户将智能手机直接放在充电板上。如果你的智能手机没有无线充电功能，你或许可以找到一个内置无线充电功能的兼容保护套。

虽然如果你厌倦了插入设备，感应充电很方便，但在充电时也很慢。如果你在寻找更快的方法，你应该选择有线连接。由于你不能在平板电脑充电时移动你的设备，这意味着你必须找到其他东西让你的手摆弄。

### **齐本位无处不在**

如果你是一个没有听说过感应充电的长期 iPhone 用户，你并不孤单。根据无线电力联盟(Qi 感应充电标准的创造者)的说法，即使有超过 3 亿台 Qi 兼容设备在使用，这项技术也没有像 Lightning 电缆一样无处不在，这要归功于苹果多年来一直将该功能从其之前的智能手机中排除。幸运的是，苹果正在采用一个得到广泛支持的感应充电标准，而不是通过引入自己的标准来搅浑水。

即便如此，你仍然可以在一些非常受欢迎的地方找到它的例子。首先，Apple Watch 用户在给智能手表充电时依靠感应充电。一些星巴克的位置有感应充电站，使用竞争对手 Powermat 标准 内置在他们商店的桌子上。宜家出售各种与 [Qi 兼容的感应充电垫](http://gizmodo.com/ikea-just-made-it-crazy-easy-to-add-wireless-charging-t-1698047498#_ga=2.123859594.1275430105.1505050985-167189482.1495477587) ，以及带有集成感应充电站的家具和灯具。甚至像本田和宝马这样的汽车品牌也提供无线充电功能标准，或作为面向技术的升级包的一部分。

### **无线充电并不适合所有人**

当然，当你在漫长的一天后回到家时，你可以将手机放在充电板上，但一旦你安顿下来，你可能会在沙发上、床上或出门时将手机从充电器上取下。你准备好让你的手机无人看管，在无线垫上毫无用处，而不是把它挂在你的咖啡桌下的六英尺长的充电电缆上吗？至少你可以拿着它。

但你可能是一个很酷的未来科技的粉丝，如果你对无线充电感兴趣，你肯定可以利用它。首先，你卧室里的无线充电器(当然，你的手机不应该放在那里)可以让你不用去寻找你掉在床头柜后面的电缆，让你不要瞥一眼就盯着它看，或者让你不要错误地将连接器插入端口，在瞌睡时破坏这两者(我并不痛苦)。反正你会睡着，所以你也可以在打盹的时候用一个。

工作场所或汽车上的感应充电器也是方便的位置，尤其是当你到达办公室时容易忘记给手机充电的时候。消除手机充电时的摩擦可能是一天结束时充满电和没电的区别。
