# 你需要清理你的 Chrome 扩展

> 原文：<https://lifehacker.com/it-s-time-for-you-to-clean-up-your-chrome-extensions-1819614337>

谷歌在其 Chrome 浏览器中增加了一些安全功能，首批变化今天向 Windows 用户推出。此次更新改进了 Chrome 和谷歌的 [Chrome 清理工具](https://www.google.com/chrome/cleanup-tool/) ，该工具可以监控试图修改默认搜索引擎等用户设置的扩展，以及旨在在浏览器窗口插入额外广告的恶意软件。完美的时机，考虑到最近大量的 Chrome 扩展被发现收集用户数据，假冒更受欢迎的 Chrome 扩展，甚至未经用户同意挖掘加密货币。



有问题的扩展利用了 Chrome 的自动扩展审查过程，将恶意代码偷偷带入公司和用户的电脑。像弹出窗口拦截器 [SafeBrowse 挖掘加密货币](https://www.ghacks.net/2017/09/19/first-chrome-extension-with-javascript-crypto-miner-detected/) 这样的扩展利用了 Chrome 用户的处理能力，而游戏玩家使用的 [Steam 库存助手扩展](https://www.ghacks.net/2017/09/19/steam-inventory-helper-monitors-your-browsing-activity/) (与 Steam 无关)在一次更新后开始监控用户的浏览习惯。

谷歌最新的安全更新现在会通知用户未经他们同意所做的更改。例如，如果一个扩展改变了你浏览器的默认搜索引擎，Chrome 会弹出一个窗口，询问你是否想恢复你原来选择的搜索引擎。Chrome 清理工具将监控你在 Chrome 中的下载，扫描恶意软件或其他不需要的软件，并为你删除它们。

### **去掉无用的扩展**

您可能已经安装了比您意识到的更多的扩展。如果你想看看这些年来积累的扩展，可以去你的 Chrome 搜索栏，输入“chrome://extensions”来显示你已经安装的扩展列表。您也可以右键单击工具栏中的任何扩展，然后选择“管理扩展”来显示您的列表。在那里你会看到启用和禁用的扩展。要查看他们可以访问的数据类型，请单击每个数据类型下的详细信息链接。不是每个扩展都可以修改，但是有些扩展允许你通过细节链接旁边的选项链接来调整一些设置。

### **隐藏不常用的扩展名**

一些扩展在后台运行，只需要很少甚至不需要用户交互就能很好地完成工作。例如，在安装了 Reddit 增强套件 之后，我花了几分钟时间来调整我的首选项，并刷新了我的 Reddit 页面，以查看急需的界面调整生效。我每天都在使用它，但是我不需要把这个图标放在我的工具栏里，只是放在那里，占用空间。右击一个扩展并选择“在 Chrome 菜单中隐藏”将会从你的工具栏中删除该扩展，而不会安装它。

### **审核你要保留的那些**

没有人提倡你去掉工具栏上的每一个 Chrome 扩展，尤其是那些对你日常浏览至关重要的扩展。验证扩展的合法性需要时间，但并非不可能。当然，来自热门公司(比如 Pocket、1Password 或 Evernote)的扩展应该被认为风险很低，但大多数扩展来自较小的公司，甚至是试图解决自动播放视频或烦人的弹出窗口等问题的个人开发者。

如果您不确定某个扩展的可信度，检查该扩展的商店页面会有所帮助。衡量网络的意见也是一个可行的选择。当你访问一个特定扩展的页面时，一定要浏览它的描述，注意它需要收集什么类型的数据，以及评论的质量和数量。搜索一下这个扩展，看看人们在其他地方是怎么说或者写的。如果普遍的共识是积极的，并且审查似乎是合法的，安装了。如果没有，要么完全避免，要么寻找替代方案。
