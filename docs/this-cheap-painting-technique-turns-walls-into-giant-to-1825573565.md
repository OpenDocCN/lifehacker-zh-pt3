# 这种廉价的绘画技术将墙壁变成了巨大的触摸传感器

> 原文:[https://gizmodo . com/this-cheap-painting-technique-turns-walls-into-giant-to 1825573565](https://gizmodo.com/this-cheap-painting-technique-turns-walls-into-giant-to-1825573565)

你房子的墙壁支撑着屋顶，给你的家人一些隐私，也许还能提供一个悬挂艺术品的地方。但是迪士尼 和卡耐基梅隆大学的研究人员想出了一个让墙壁做更多事情的方法。一种廉价的油漆处理将它们变成了巨大的传感器，创造了一种通过在智能家居中走动来与智能家居互动的新方式。

与其他检测房间内人员或物体移动的方法不同，Wall++不需要引人注目的摄像机、明亮的投影仪，甚至不需要覆盖每个表面的花哨的跟踪标记。取而代之的是一种独特的油漆处理，即使是新手也可以使用。

首先使用导电涂料创建一个菱形网格，然后用另一个非常细的电线网格覆盖，像节点一样将它们连接在一起。标准乳胶房屋涂料的最后一层隐藏了下面的一切，研究人员估计处理成本约为每平方米 20 美元，但大规模生产时甚至会更便宜。

菱形和线栅本质上创造了 [一种在智能手机、平板电脑、智能手表和无数其他科技产品上发现的电容式触摸屏](https://gizmodo.com/how-do-touchscreens-actually-work-anyway-1664259772) 的非常粗糙的版本。当你导电的手指接触屏幕时，它会改变显示器特定区域的电荷，底层硬件可以利用这种变化来精确定位你手指的位置和运动。

墙上画的导电菱形网格++实验也是如此。这种方法相当粗糙，这意味着它远不如智能手机的触摸屏准确，但通过监测人靠近钻石时钻石上的电荷，墙壁和附带的定制软件可以以惊人的准确度计算出附近人的位置。

想象一下，只需轻敲墙上的任何地方就可以关掉房间里的灯，或者将手向上滑动就可以打开恒温器。不过，这种功能并不局限于人类互动。墙壁处理也可以作为一个巨大的电磁传感器，能够通过电器和电子产品产生的独特电磁噪声来检测和区分它们。

而这正是单个 Wall++单元所能做到的。大多数房间至少有三面或四面墙，当一个空间完全被传感器包围时，可能性变得更加有趣。同样的技术允许单个 Wall++检测一个人在哪里触摸它，也允许多个墙壁检测 3D 空间中的电子设备和电器。

你睡觉前记得关掉客厅的电视了吗？即使它是一个 40 年前的哑 CRT，它的设计并不适合现代智能家居，Wall++也可以让你知道它是否仍然通电。把静音智能手机放错地方的日子也可能成为过去。一条短信或一个电话就会产生电磁噪音，让你家的墙壁准确地告诉你忘记在哪里了。隔墙有耳，也许这并没有看起来那么可怕。
