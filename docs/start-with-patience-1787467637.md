# 从耐心开始

> 原文：<https://lifehacker.com/start-with-patience-1787467637>

耐心是几乎所有好习惯的基础。没有它，你就不能省钱、学习新技能或减肥。耐心也是最难培养的品质之一。

Watch

在这种情况下,“耐心”意味着耐心等待的能力。它不是在交通堵塞中失去理智。长期目标之所以得名，是因为它们不可能在一夜之间实现。 [五分钟后你才回复新信息](http://lifehacker.com/recalibrate-your-reality-5891564) 。接受健康的习惯不会立即获得回报。耐心是实现长期目标的一个因素，也是保持冷静的一个因素。

当我们不耐烦时，我们会寻找捷径。当我们走捷径时， [我们往往会变得草率](https://lifehacker.com/if-youre-looking-for-a-shortcut-youre-doing-the-wrong-1725325780) 或者做出糟糕的决定。当你不在乎工作的质量时，比如说，你只是匆匆忙忙地做了一顿晚餐，或者匆匆忙忙地发出了一张去邻居家烧烤的邀请。当你因为想要一个不费力的结果而寻找逃避工作的方法时，捷径是有问题的。

在自助运动中，不耐烦是显而易见的。自助大师们一贯宣传从 A 点到 B 点的最短路线，他们利用我们的不耐烦来推销他们的狗屁产品。他们推销不劳而获的减肥，不劳而获的财富，不流汗的锻炼，不勤奋的教育。我们接受这些糟糕的想法，因为我们没有耐心等待自我完善。灵丹妙药来得越快越好。

耐心不仅仅是自我完善的高级概念。愚蠢的日常事务也需要它。 [耐心不预购电子游戏](http://kotaku.com/stop-preordering-video-games-1713802537) 。这不是盗版电影放映员。在购买下一代 iPhone 之前，它正在等待适当的评论。没有耐心，我们就是做出愚蠢选择的消费者， [然后抱怨没人让我们为后果做好准备](http://kotaku.com/the-no-mans-sky-hype-dilemma-1785416931) 。

我注意到自己变得满足于自己的耐心。那种不耐烦已经成为我的默认状态。我会不做太多研究就盲目地从亚马逊订购一些东西，跳过学习 401(k)到底是如何工作的，或者随意地对一辆不在转弯车道上行驶的汽车按喇叭。

不耐烦是一种荣誉，因为它意味着我太忙了， [这是相对成功的标志](https://lifehacker.com/how-to-escape-the-cult-of-busy-5994072) 。事实上，急躁是导致破产、愤怒和不健康的原因。

我不认为这只是我。无论是在抱怨杂货店排队的长度时向我寻求支持的人，还是在交通中抢在我前面五秒到达红绿灯的普锐斯，还是为新职位的起薪而哭泣的朋友，对我们大多数人来说，现在都是最重要的。一想到要等待什么就会引起焦虑和恐惧。

当然，长期以来，耐心一直是人类的奋斗目标。之所以称之为美德，是因为它很难，不常见，难以实践。但也许把耐心视为一种美德是错误的做法。这种想法使人觉得你要么有耐心，要么没有。

然而在实践中，耐心更像是一种习惯，而 [习惯是人类非常善于适应](https://lifehacker.com/what-it-takes-to-form-a-good-habit-5724234) 的东西。我发现在对日复一日考验我耐心的烦恼做出反应之前，停下来对我自己最有用。最简单的方法是把我的注意力从我的观点转移到别的东西上。如果杂货店的队伍很长，我会想想收银员的一天有多难熬。如果我对交通感到恼火，我会专注于我周围的其他人如何和我一样恼火。这一点点想法，几秒钟，往往足以平息那时刻不耐烦。

长期耐心的培养也许可以从同样的习惯养成方法中发展出来。先给自己定下关于这些愚蠢事情的规则:停止预先订购媒体，拒绝在发布日购买任何东西，或者 [退出亚马逊的两天免费送货](http://lifehacker.com/opt-out-of-amazon-prime-s-free-two-day-shipping-earn-1613594987) 以获得一美元的免费电影积分。当你有闲工夫做某事时，就去做吧。花多一点时间准备晚餐，花点时间做一个兼职项目，或者阅读你保存的文章。

然后，解决需要长期耐心的大问题:401(k)计划，为职业发展学习新技能，或者攒钱买房。运气好的话，实现这些目标会容易一点。
