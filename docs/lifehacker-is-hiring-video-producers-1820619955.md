# Lifehacker 正在招聘视频制作人

> 原文：<https://lifehacker.com/lifehacker-is-hiring-video-producers-1820619955>

你对正确的做事方式有什么强烈的想法，并渴望与世界分享吗？你喜欢极客式地研究从如何修拉链到现代礼仪细节的一切细节吗？你是你朋友圈的 MacGyver 吗？你是不是也是一个聪明、精力充沛、注重细节的视频制作人，他认为拍摄加湿器和除湿机之间的皇家战争是一段美好时光？那我们想和你谈谈。



Lifehacker 有三个视频职位，分别为 [高级制作人](http://gizmodo.com/careers/job/867224?gh_jid=867224) 、 [制作人](http://gizmodo.com/careers/job/888766?gh_jid=888766) 和 [副制作人](http://gizmodo.com/careers/job/937007?gh_jid=937007) 。这些都是全职工作，有竞争力的工资和良好的福利，在 Gizmodo Media 的纽约市母舰上工作，你可以提出故事创意，编写剧本，进行冒险，为全球超过 2400 万观众拍摄和制作精彩的视频。

我们正在寻找有必要经验的制片人，他们和我们一样痴迷于从在最新的 iOS 中寻找捷径到盛大的烹饪实验(你能 [腌猪头](https://skillet.lifehacker.com/will-it-sous-vide-the-head-of-a-pig-1797683697) 吗？你应该吗？).

基本上，如果你有那种总是在一种情况下寻找“如何”和“为什么”的大脑，并且知道如何将它变成引人注目的视频，这可能是你的幸运日。

点击上面的链接申请。除了这些视频制作人的职位，我们还招聘了一名高级技术编辑，我们一直在寻找新的贡献者。如果你是一名作家，并且你有一个我们没有涉及到的杀手级技巧或快捷方式，请随时在 lifehacker dot com 上联系我们，我们会看一看。
