# 吃剩的米饭会让你生病

> 原文：<https://lifehacker.com/leftover-rice-could-make-you-very-sick-1793530699>

在从“1”到“李斯特菌”的范围内，我预计一碗米饭给我带来的胃病数量在“2”左右，但显然，如果储存不当，这种看似无害的谷物会造成很多痛苦。



北卡罗来纳州立大学 的食品安全专家本杰明·查普曼认为，煮饭不一定能杀死所有潜伏的病菌。“大米的问题是，”他通过电子邮件向我解释道，“一种病原体，蜡状芽孢杆菌，在干燥的大米中非常普遍(一些消息来源说无处不在)，可能是孢子。孢子可能在烹饪中存活。如果煮熟的米饭随后保持在室温下，孢子可以脱离它们的保护形式，发芽，营养形式繁殖。熟饭环境为生长提供了大量的水分和养分。作为生长的副产品，它们会产生一些毒素，包括一种热稳定的毒素。”

虽然还不清楚究竟有多少起食物中毒是由蜡状芽孢杆菌引起的，但一项研究估计每年有 63，623 起 T3。幸运的是，在你享用完菜肴后，将剩余的米饭放在冰箱里会保证你的安全。根据 [这篇来自*流行病学与感染*](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC2130471/) 的文章，煮熟的米饭应该“要么保持热的”(63°C/145.4°F 以上)，要么“迅速冷却，并在烹饪后 2 小时内转移到冰箱”事实上，大米最危险的温度区域是“室温到温暖”范围(15-50°C/59-122°F)。

除了冷藏任何家常饭，外出就餐时保持警惕也很有帮助。根据罗格斯大学 的食品科学家唐纳德·沙夫纳(Donald Schaffner)的说法，一些餐馆“煮了一大堆米饭，在室温下保存一整天”，然后根据需要从这一批中拿走一部分。“因为芽孢杆菌产生一种热稳定毒素，”他解释说，“这不是一种最佳做法，过去也曾导致过疫情爆发。”“热稳定”意味着毒素可以在沸腾中存活，一旦大米冷却到 59-122 华氏度的“危险区域”，细菌可以繁殖，产生更多的毒素。他指出，寿司饭应该不成问题，因为添加醋可以降低 pH 值，使其在室温下安全保存。

简而言之，让你和你爱的人免受蜡样芽孢杆菌及其肆虐的最好方法是在烹饪后两小时内冷藏任何剩菜，并与你最喜欢的米饭服务餐厅联系，了解他们如何准备和储存每一批米饭。毕竟， [大米应该帮助你的肚子](https://www.youtube.com/watch?v=bBL2thqJc0g) ，而不是让它陷入混乱。
