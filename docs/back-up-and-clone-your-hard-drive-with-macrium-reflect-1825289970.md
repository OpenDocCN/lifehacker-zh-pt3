# 使用 Macrium Reflect 备份和克隆您的硬盘

> 原文：<https://lifehacker.com/back-up-and-clone-your-hard-drive-with-macrium-reflect-1825289970>

我们的本周 Windows 应用可能不会经常使用——我们希望——但如果你正在升级系统，需要将所有东西从旧硬盘移动到新硬盘，它是一个很好的工具。



事实上，这正是我这个周末所做的，当时我把一个备用硬盘换成了一个新的(更大的)硬盘。虽然我可以使用 Windows 的文件资源管理器复制所有文件，但这意味着我不得不不时回到我的电脑前，处理当你将大量文件夹拖到新位置时往往会弹出的烦人提示。

[Macrium Reflect](https://www.macrium.com/reflectfree) (免费)帮助你绕过这一点，但当你需要备份(或克隆)整个驱动器时，它甚至更有用——特别是如果它是你正在使用的驱动器，比如你的主 Windows 分区。有了 Macrium，你可以轻松地创建一个备份映像，或者只是将驱动器的全部内容、分区等复制到一个新的位置。

如果您不习惯在自己的系统上进行大规模数据迁移，该程序的界面可能会有点复杂。然而，Macrium 很好地以图形方式呈现了这个过程(并提供了大量的提示来防止您意外地覆盖了错误的硬盘)。

Macrium 还可以帮助您制作一张救援介质(CD、DVD 或 USB 驱动器),在电脑的 Windows 操作系统无法启动的罕见情况下，您可以使用它来启动应用程序。然后，您可以恢复您最近创建的磁盘映像，甚至可以恢复您创建的操作系统的“刚安装”映像，其中已经安装了您最喜欢的调整和应用程序。

为了避免忘记备份硬盘内容这一至关重要的过程，Macrium 还可以按照设定的时间表运行其映像过程。您可以让它创建完整备份(需要一段时间),或者创建差异备份和增量备份，后者只考虑自上次完整备份或增量备份以来发生更改的文件。
