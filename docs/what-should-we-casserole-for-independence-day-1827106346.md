# 独立日我们应该吃什么？

> 原文：<https://lifehacker.com/what-should-we-casserole-for-independence-day-1827106346>

你好，欢迎回到 [*会砂锅吗？*](https://skillet.lifehacker.com/you-deserve-a-hot-and-cheesy-pierogi-casserole-1819722125?rev=1508528101066#) ，我把你的美味概念重新想象成美味的砂锅菜。我们上一个砂锅创造是一个稍微调整的经典，结果非常好，但现在是时候进入下一个任务了。



在夏天的三伏天，我并不真正渴望砂锅菜，但我不记得在砂锅圣经中读到过砂锅菜必须(合法地)趁热端上来。(我想还好我们没有把这个专栏叫做“它会是热菜吗？”。)反过来，不像人们*只有*六月到八月吃冷食；而是要找到不会粘在你肋骨上，把你压下去的东西。另外，随着 7 月 4 日的到来，我们有机会确定一个主题(我最喜欢的)。一些想法:

*   **野餐沙拉砂锅:**所有你喜欢的沙拉——土豆、通心粉、鸡蛋等。—终于在一起了。
*   **美国砂锅:**我的第一个冲动是把它装满汉堡、热狗和法式炸土豆，但是告诉我美国对你意味着什么(食物方面)。
*   **玉米砂锅:**因为玉米才是夏令的真正原因。
*   **番茄砂锅:**有点像番茄馅饼，但是是砂锅形式。我想加入切达饼干。
*   **冰淇淋沙锅:**我想象着各种冰淇淋、水果和饼干的层次，上面还有一些脆脆的甜味。

一如既往，欢迎在下面推荐你自己的不辣菜肴，越切题越好。
