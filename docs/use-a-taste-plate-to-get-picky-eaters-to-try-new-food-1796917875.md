# 使用“品尝盘”让挑食者尝试新的食物

> 原文：<https://lifehacker.com/use-a-taste-plate-to-get-picky-eaters-to-try-new-food-1796917875>

对于挑食的孩子来说，食物摆放有一个规则:你必须把它们分开。



豌豆不能碰意大利面，米饭必须离任何有酱的东西很近。甚至不要想把一些新的东西放在已经被证实的食物附近。为了给孩子的饮食引入多样性(甚至可能偏离米色食物组)，父母变得沮丧，用餐时间经常升级为一系列的恳求、贿赂和威胁。

“真正的妈妈”营养博客的作者莎莉·库泽姆查克给出了恢复平静的建议: [使用品尝盘](http://www.realmomnutrition.com/taste-plate-for-picky-eaters/) 。

“这是放在孩子主餐盘旁边的一个小碟子，”她解释道。“这是一个没有威胁的地方，你可以放一些你想让他们尝尝的食物。”

对于注册营养师 Kuzemchak 来说，挑食 [“只尝一口”的方法适得其反](http://www.realmomnutrition.com/why-i-dont-make-my-kids-take-just-one-bite/) 。这让她蹒跚学步的儿子处于守势，挑战了他的独立性，破坏了餐桌上的气氛。另一方面，品尝盘是一种无压力的邀请。Kuzemchak 把份量做得非常非常小——只有一小口那么大。品尝是被鼓励的，但不是强制性的。她的孩子可以礼貌地把食物吐到餐巾纸里，如果他们尝试后不喜欢的话。

使用品尝盘的一些其他提示:

*   挑选一个有趣的小盘子。白 Corelle 没那么令人兴奋。
*   随便你怎么称呼它！一位家长称之为“试吃碗”。我想我会叫它勇敢板。"如果你觉得自己很勇敢，试试这些你可能会感到有点紧张的食物。"
*   保持低压。不要一直催促它，说“嘿，孩子，记住你有一个品尝盘在那里！”避免大制作，如果他们做或不尝试盘子里的任何东西。有些孩子吃饭时不喜欢被人看着。
