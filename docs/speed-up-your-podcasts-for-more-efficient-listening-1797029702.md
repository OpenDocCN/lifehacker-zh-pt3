# 加快您的播客速度，提高收听效率

> 原文：<https://lifehacker.com/speed-up-your-podcasts-for-more-efficient-listening-1797029702>

有这么多好的播客(包括 Lifehacker 的 [、升级版](https://itunes.apple.com/us/podcast/lifehacker-audio-only-mp3/id508117781) )，但一天的时间就这么多。报道了一些“播客迷”，他们利用之类的应用加速播客的播放。比如，高达 5 倍的速度。当然，这可以节省时间，但也可能会让你的血压飙升，让听播客变得超级有压力。那么，在不损害健康或破坏播客的情况下，收听播客的最佳速度是多少呢？



许多播客应用程序，包括苹果的 [播客](https://itunes.apple.com/us/app/podcasts/id525463029?mt=8) ，都有加速播客剧集的功能。 [阴天](http://lifehacker.com/the-best-podcast-manager-for-iphone-5855050#_ga=2.72511126.408553697.1500300650-1887250931.1497314809) 具有智能速度，这是一种通过缩短静默来加速播客的功能。我听了升级版的 [关于尴尬的一集](http://lifehacker.com/why-your-awkwardness-is-secretly-a-social-asset-with-ty-1796810414#_ga=2.110256768.408553697.1500300650-1887250931.1497314809) 使用了阴的智能速度设置，将音频加快了两个等级。covant 在不改变声音音调或牺牲播客质量的情况下，将 53 分钟的播客变成了 34 分钟(感谢技术)。速度从 1.1 倍到 1.8 倍不等，但大多保持在 1.3 倍。随着我继续听，我能够将听力速度提高到平均 1.8 倍。我尝试了 3 倍，但对我来说感觉太快了。我仍然可以理解播客在说什么，但是它需要我所有的注意力，并且很累。

Lifehacker 的特约撰稿人尼克·道格拉斯(Nick Douglas)表示，covant 的智能速度非常适合未经编辑的脱口秀节目，但不适合制作精良的节目。听升级的时候，智能速度从正常速度(1x)到 1.1x 速度不等，节省不了多少时间。但是对于像《与克里斯塔·蒂皮特在一起》 上的 [这样的节目来说，Smart Speed 将录制速度提高了 1.2 到 1.7 倍](https://onbeing.org/)

虽然加快播客的播放速度当然可以节省时间，但它可能会影响收听体验，这取决于你的个人偏好以及你正在收听的节目类型。

显然，当时间很重要时，你不应该加快播客的速度，如冥想播客、音乐播客或喜剧播客——你不能以 2 倍的速度听《欢迎来到夜谷》,而不破坏悬念和改变故事的基调。但对于我只是想从中获取信息的播客来说，比如《纽约时报》的 [《每日》](https://www.nytimes.com/podcasts/the-daily) ，加速播客会更有效率。

就像的多任务处理一样，我在短时间内赶着做尽可能多的事情，我会继续听加速播放的播客，知道这是享受和效率之间的一点权衡。即使在不必要的时候也能更有效率，这很有吸引力——我一天中多出来的 19 分钟可以用来听另一集播客。
