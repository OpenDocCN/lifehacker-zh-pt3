# 卡诺屏幕工具包是一个有趣的建立，树莓派紧凑的显示器

> 原文：<https://lifehacker.com/the-kano-screen-kit-is-a-fun-to-build-compact-display-1750101508>

我们都被屏幕包围着，但很少有人知道它们到底是如何工作的。如果你想教你的孩子展示如何工作(或者自己学习！)并且你需要一个树莓派的显示器，Kano Screen Kit 是一个价格合理、便携的专用显示器，你可以自己制作。

Watch

### 盒子里有什么

[卡诺屏幕套件](http://us.kano.me/products/screen-kit)(149.99 美元)是一款 10 英寸的 HDMI 兼容屏幕，专为树莓派打造。与其他 [卡诺](http://us.kano.me/) 产品一样，屏幕套件装在一个精心设计的小纸箱里，里面有你需要的一切，还有有用的指南。

该套件包括 10.1 英寸液晶显示屏(1280 x 800，720p，150 PPI 显示屏)，一本教你如何构建显示屏并解释其工作原理的书，一个驱动板，一个按钮板，HDMI 电缆和一个迷你 USB Y 型电源分配器。除此之外，你还会得到一些贴纸，让孩子们可以定制显示器，一个背壳可以支撑屏幕，也可以作为 Pi 的外壳，还有一个小放大镜，让孩子们可以在你制作它的时候仔细看看一切。

### 构建屏幕

Kano 屏幕工具包的乐趣来自于实际构建屏幕，所以我不会在这里破坏任何东西。这绝对是一个非常简单的过程，当我在假期向几个更小的孩子(和非技术成人)展示时，每个人都能够在没有手册的情况下将它组装起来。这本手册确实向你介绍了很多方法和原因，所以如果你在教孩子屏幕的一般工作原理，它是值得保存的。这是一个有趣的小项目，对于任何已经习惯玩树莓派的孩子来说，构建工具包将是轻而易举的事情(即使他们不习惯)。)

### 它擅长的地方

所以，制作屏幕工具包很有趣。它也很结实。你可能会认为最终的结果是一个垃圾，错位的屏幕，一切都完美地结合在一起，你几乎看不出这是一个 DIY 套件。在花时间自己动手制作之后，孩子们肯定会有更多的自豪感和主人翁感。

即使对成年人来说，屏幕套件也有很大的吸引力。光是这个案子就让一个对组织上瘾的人的生活轻松多了。你总是有一个相对便携的 Raspberry Pi 系统随时可用，不占用你的家庭娱乐系统或电脑旁边的空间，而且一切都收拾得整整齐齐。

正如你所料，屏幕套件与 [卡诺电脑套件](http://lifehacker.com/the-kano-kit-teaches-kids-to-build-computers-and-learn-1643364469)(149.99 美元)配合使用效果最佳。屏幕套件有一个特殊的地方，电脑套件的树莓派外壳可以嵌入其中(尽管你可以很容易地在任何外壳上 velcro)，而 [卡诺套件键盘](http://us.kano.me/collections/other-products/products/keyboard)(44.99 美元)可以滑入屏幕背面，因此它超级便携。这种兼容性非常好，让它看起来像一个有凝聚力的、单一的单元，但是 Kano 的东西并不是必须的。甚至 [特殊的 Kano 操作系统](http://www2.kano.me/downloads) (旨在帮助孩子学习各种编码技能，并设置为开箱后与屏幕套件一起工作)也是免费的，即使你不购买 Kano 计算机套件。

### 不足之处

不要指望 Kano 屏幕套件与 iPad 等产品的屏幕质量相同。事情不应该是这样的。然而，就 10 英寸显示器市场(诚然很小)而言，Kano 屏幕套件在其他显示器中占有一席之地，并且可以与 Adafruit (没有机箱的好处)的类似产品相媲美。

很明显，Kano 屏幕套件真的想让你留在 Kano 生态系统中。如果你拥有他们的东西，这很好，但是如果你没有，你显然不得不笨拙地拼凑其他解决方案。不过不用担心，其他几个键盘会像 Kano 的一样完美地滑入外壳的背面。

如果你没有使用 Kano 操作系统，你还需要调整你的 Raspberry Pi 的视频设置来让屏幕工作。我发现使用 Adafruit 的 10.1 英寸屏幕 的配置设置非常好。尽管如此，它并不完全是现成的支持，所以请记住这一点。

屏幕套件的可用性也是一个问题。第一批很快售罄，第二批现在可以预订，三月份发货。幸运的是，卡诺有一个很好的准时运输记录。

### 底线:坚固的便携屏幕，老少皆宜

屏幕套件显然是为儿童打造的，但它是一个坚固的显示器，可以教会你很多东西，不管你多大。你可以从 Adafruit 以稍微便宜的价格 获得 [类似的屏幕，但我真的很喜欢 Kano 屏幕套件既是屏幕*又是存储系统*。作为一个有太多树莓派的业余爱好者，拥有一个小巧的全包设备真的很好，它很容易启动，测试，然后在我完成后储存起来。硬核 DIYer 也可以设计和 3D 打印一个案例和存储系统，但 Kano Screen Kit 对任何不想走那么远的人来说都很棒。](https://www.adafruit.com/product/1287)
