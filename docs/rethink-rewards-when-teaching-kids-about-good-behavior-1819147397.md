# 在教育孩子良好行为时，重新考虑奖励

> 原文：<https://lifehacker.com/rethink-rewards-when-teaching-kids-about-good-behavior-1819147397>

每位家长都提出了激励措施:“如果我换轮胎时你有耐心，之后我们就可以吃冰淇淋了。”或者，“如果你和你的表弟玩得很好，你可以在晚饭前使用 iPad。”老师们肯定会用行为奖励来换取时间——在我小时候，一块糖或一个小玩具来表示顺从绝对是一种安慰。



但是，琳达·弗拉纳根在为 [Mind/Shift](https://ww2.kqed.org/mindshift/2017/08/29/how-ending-behavior-rewards-helped-one-school-focus-on-student-motivation-and-character/) 撰稿时说，为行为提供激励并不一定是塑造性格和增加动力的最佳方式:“几十年来，大量社会科学 [研究](https://www.ncbi.nlm.nih.gov/pubmed/10589297) 得出结论，对某些类型的行为给予奖励不仅无益，而且有害。”

她讨论了丹尼尔·平克的书 [*驱力:关于激励我们*](https://www.amazon.com/Drive-Surprising-Truth-About-Motivates/dp/1594484805?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/rethink-rewards-when-teaching-kids-about-good-behavior-1819147397&asc_source=&tag=kinjalifehackerlink-20) *，*的惊人真相，指出他“指出了外在奖励的七个缺点:它们削弱内在动机，限制表现，压制创造力，扼杀良好行为，促进欺骗，会形成习惯，并刺激短期思维。平克写道，为例行的、无需动脑的任务颁奖可能会有一定的效果。但是为那些“天生有趣、有创造性或高尚的”任务提供奖励是一个非常危险的游戏。"

她接着描述了一所学校，这所学校过去提供“外在的”，或外部的行为动机——橡胶手镯和呼吁良好行为的公共集会——但后来改变了这一系统，转而培养一个强调尊重、责任和职业道德的社区。

现在不像是从来没有表扬过——校长说她确实会把孩子们拉到一边，告诉他们她欣赏他们的努力——但是精心奖励的表演已经一去不复返了。校长报告说，学校发生了巨大的变化，尤其是孩子们在功课上互相帮助。另一个有趣的点是:这所特殊的学校是为完全特殊教育的人群设立的，事实上，在新系统中表现最好的学生是那些有最高情感需求的孩子。

我正在攻读特殊教育专业的硕士学位(我还没有完成，所以对这一点要有所保留)，但我们研究的很多内容是如何为那些在这些技能方面挣扎的孩子树立榜样，并教给他们自我控制和适当的行为。这就是为什么我觉得对表现好的孩子奖励糖果或玩具不太好的部分原因:对一些孩子来说，不管出于什么原因，表现好只是...非常简单。对于其他人来说，不管出于什么原因，这都是一个巨大的挑战。因此，对于顺从的苏茜来说，当捣乱的提米再一次得到*啧啧*(或惩罚)时，她吃了一堆士力架，似乎适得其反:苏茜因为一开始对她来说并不难的事情得到了奖励，而提米没有得到如何成为一个高效、尊重、*内在激励的*社区成员的指导和示范。

这就是为什么我喜欢目前 ICT 课程的模式，在这种模式中，作为“行为榜样”的孩子被安排在那些适当的社会行为是一种挑战的孩子中间。合作和尊重被强调，老师给需要的孩子额外的支持。奖励让社区变得有竞争力:苏西得到一块士力架；提米没有；乔在他的哈佛申请表上写着“模范公民奖”,而玛丽得到了 bupkus。它把本应是一个社区的东西变成了一个零和游戏。

弗拉纳根的很多故事都是关于老师的，而不是父母，但显然有很多重叠。如果我们把屏幕上的时间用于参与家庭杂事或拜访亲戚，我们不是在教育我们的孩子参与家务是必要的，与琳达阿姨共度时光是有价值的——我们是在教育他们这些任务是为了获得奖励而必须跳过的铁环。教师和家长的部分工作不仅仅是协商合规性，还要教会孩子如何尽可能轻松愉快地管理他们的日常活动——即使这些活动很乏味或令人不快。我采访的几位儿童发展专家讲述了关于 [纪律](https://offspring.lifehacker.com/how-to-do-a-proper-time-out-for-a-misbehaving-kid-1818767367) 和 [挑食](https://offspring.lifehacker.com/we-read-7-of-the-most-popular-books-on-kids-and-picky-e-1795660694#_ga=2.251800138.2141952798.1507046396-2116256117.1488937670) 的故事，他们强调了一个家庭的*文化*:“在我们家，我们表现得很尊重，”(不要打人，或在餐桌上说*恶心*或诸如此类)。

这是不是意味着从来没有人吃过零食，从来没有？当然不是。我孩子学校的老师们计划举办冰淇淋聚会之类的活动，但这是为了庆祝优秀的工作，而不是奖励。和上面的校长一样，当孩子们在对他们来说很难的事情上做得很好时，他们会悄悄地发现这一点。
