# 回家过节时在家人的电脑上要做的 10 件事

> 原文：<https://lifehacker.com/top-10-things-to-do-on-your-familys-computer-when-you-g-1790003745>

假期旅行几乎总是意味着，如果你是家里懂技术、懂电脑的人，有人会让你看看他们的笔记本电脑、手机或台式机。下次你回家时，用这份建议清单阻止他们。

Watch

### 10.运行操作系统更新(到达后立即运行)

除非你认为你有足够的时间在那里闲逛和放松，在后台关闭计算机或在另一个房间下载更新和补丁，否则一旦你安顿下来，也许 [启动 Windows Update 或启动 Mac App Store 并检查更新](https://lifehacker.com/how-to-fix-your-family-s-computer-this-holiday-so-the-r-5862161) 。

首先，它会在你到达那里后给电脑一些事情做，这样你就不会立即收到技术支持请求，其次，它会解决整个家庭技术支持过程中最耗时的部分。让这些更新运行、安装、重启并重复，直到全部完成。然后你可以 [继续做其他事情](https://lifehacker.com/how-can-i-set-up-my-parents-computer-to-avoid-future-p-508998215) 在你真正需要坐下来使用的地方。

### 9.安装(或验证他们有)病毒防护和反恶意软件

当然，你在父母、兄弟姐妹、表亲或其他家庭成员的电脑上安装哪些防病毒和反恶意软件工具完全取决于你自己 ，但要确保他们有*的东西*，因为你不能总是相信“好的互联网卫生设备会保护你，而且你绝对不能总是相信某个人，如果发生不好的事情，你必须收拾他的烂摊子。

一旦你确定他们 [有一些保护](https://lifehacker.com/five-best-desktop-antivirus-applications-1607557993)——或者如果他们已经有了的话——确保它处于良好的工作状态，至少最近更新过，并运行系统扫描，只是为了更好的测量。毕竟，如果反恶意软件从来没有更新或使用过，它是没有用的。

### 8.更新他们的浏览器和扩展

如今，每个现代网络浏览器都会自我更新，通常是在有新版本下载时重启，但如果你家里有人使用的版本太旧，或者他们使用的是过时的浏览器，现在是让他们加快速度的好时机。

也许他们使用的 [浏览器扩展被证明是恶意软件](https://lifehacker.com/many-browser-extensions-have-become-adware-or-malware-1505117457) (或者 [只是一般不值得信任的](http://lifehacker.com/adblock-plus-is-now-selling-acceptable-ads-1786589122) )并且他们从来没有用值得信任的替换它们。回家过节是一个很好的时机，也可以把这些东西换出来，给他们一点互联网隐私的礼物。

### 7.卸载垃圾软件或不必要的程序

无论是 Mac 电脑还是 Windows 电脑，没有什么比 [卸载 OEM crapware](https://lifehacker.com/the-complete-guide-to-avoiding-and-removing-windows-c-1630577558) 以及任何其他在后台安静运行的你确实不需要的应用程序和服务更能给你带来更好的即时体验了。扔掉它们，给你的家人一台速度更快的电脑。

### 6.尽你所能实现自动化

回到到他们最喜欢的应用、设置和工具， [尽可能自动化一切](https://lifehacker.com/what-kind-of-maintenance-do-i-need-to-do-on-my-windows-5815256) 。系统更新？将它们设定为在夜间或电脑不使用的“安静时间”自动安装。您的备份也是如此。将您的家庭成员的所有喜爱的应用程序配置为自动下载更新和安全补丁，然后在下次应用程序打开时安装它们。

### 5.现在进行一些速度测试，这样你以后就有弹药了

这可能看起来很傻，但是当一切正常的时候，点击并运行带宽测试。当你在用的时候，对你家的电脑做一下 [轻微的基准测试](https://lifehacker.com/the-differences-between-synthetic-real-world-and-hybr-1663435313) 。这看起来像是不必要的矫枉过正，但是 [它给了你一个参照系](https://lifehacker.com/pc-wizard-analyzes-and-benchmarks-your-pc-398656) 从中你可以解读任何真正的问题。

如果你了解他们的宽带速度在正常情况下是什么样的，你就会知道当他们抱怨速度慢时，是他们的电脑还是家庭网络对他们的服务提供商。如果你知道他们的 CPU 实际运行有多热，你就能知道它是否过热。现在做一点笔记会让你免于猜测——即使他们是受过教育的人——当他们要求你快速解决他们遇到的任何问题时。

### 3.备份他们的数据，并建立定期备份

每个硬盘都会出故障，这只是时间问题——这就是为什么每个人都需要良好、强大的备份 来确保他们不会丢失珍贵、不可替代的数据。当我们说“强大”时，我们指的是遵循旧的 [3-2-1 备份规则](http://www.hanselman.com/blog/TheComputerBackupRuleOfThree.aspx) : 3 个备份(如果重要，两个不够)，2 种不同的格式(如 dropbox 和外部硬盘，或外部硬盘和 DVD)，1 个异地备份(如果你的房子被烧毁或被盗，你丢失了你的电脑和那个闪亮的外部硬盘，你将如何恢复？)

[轻松设置防弹备份系统](https://lifehacker.com/set-up-an-automated-bulletproof-file-back-up-solution-5787572) 只需几分钟的功夫。如果像我们最喜欢的 [Crashplan](http://www.code42.com/crashplan/) 这样的东西太复杂， [考虑另一个更简单、更精简的工具](https://lifehacker.com/online-backup-faceoff-crashplan-vs-backblaze-1768101631) 来完成工作。你甚至可以使用内置工具，比如 Windows 10 中的 Windows Backup，或者 macOS 中的 Time Machine。不管是什么，把它完成，这样就不会有人惊慌失措地打电话给你，说他们丢失了所有的照片、音乐或一份超级重要的文档或文件。哦，在你回家之前，一定要测试恢复。
