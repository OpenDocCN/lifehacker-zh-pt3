# 扭曲的桌面

> 原文：<https://lifehacker.com/the-twisted-desktop-1784530692>

距离我们看到 [JonRedcorn2](https://www.flickr.com/photos/134554762@N08/) 的定制桌面已经有一段时间了，但这款桌面如此引人注目，我们无法将目光移开——我们必须突出它。以下是如何让您的电脑拥有相同外观的方法。



像通常的 Windows 桌面一样，这款使用 Rainmeter 来获得美丽的定制，所以如果你想要相同的外观，你需要在进一步了解它的基础知识之前。

如果您已经和我们在一起，只是想知道使用哪些组件，下面是您需要的内容:

*   [壁纸](http://www.pixelstalk.net/abstract-minimalist-wallpapers/)
*   用于 Windows 的 [雨量计](http://www.rainmeter.net/) 系统调试和监控工具
*   雨量计 [谜组曲](http://kaelri.deviantart.com/art/Enigma-103823591)
*   [雨量计的最小排版皮肤](http://lysy1993lbn.deviantart.com/art/Minimal-Typography-updated-1-0-7-280486134)系统监视器的
*   用于天气显示的 [MinimalOne 天气皮肤](http://mistrjosh.deviantart.com/art/MinimalOne-Weather-1-0-510234680)(jonred corn 2 注意到已经对其进行了大量修改以适应)
*   右上角的 [为雨量计的辐射遮挡皮肤](http://pvoijen.deviantart.com/art/RadialClock-1-02-539127755) 为时钟
*   左边 [雨量计](http://bradd9.deviantart.com/art/Soul-323678927)灵魂皮肤快速链接(感谢厚爱！)
*   雨量计的 [弧度可视化器](http://freezingclouds.deviantart.com/art/Radian-Linear-Visualizer-1-9-3-535341307) 音频播放器和可视化器在中间

许多移动部件，但它们组合在一起创造了一个漂亮好看的包，作为第二屏幕或就在你可能已经打开的窗户下面是很棒的。如果你对这些修改感到好奇，或者想知道他是如何做到这一切的，请访问下面的 Flickr 链接——让他知道我们给你发了邮件，并且喜欢他的作品！
