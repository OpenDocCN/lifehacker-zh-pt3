# 这个 DIY 活动罐可以让孩子们在没有 iPad 的情况下保持忙碌

> 原文：<https://lifehacker.com/this-diy-activity-tin-keeps-kids-occupied-without-an-ip-1797533629>

很难与 iPads 和手机的便利性竞争，让孩子们在汽车、飞机、餐馆、等候室和任何其他地方忙碌，“我很无聊！”经常被听到。但是只要稍微计划一下，你就可以为这些情况准备一个活动罐。

Watch

足智多谋的妈妈埃莉诺·PJ 为她的儿子制作了上面的游戏集，并带到了营地，因为电子设备在旅途中是被禁止的。她用了一个空的铅笔盒，里面放了各种玩具和游戏，以获得一些老式的触觉乐趣:乐高积木、彩色笔和便利贴。她用记号笔在盖子的内侧画了游戏格子，然后将宝石粘在磁铁纸上作为游戏棋子。埃莉诺用一张白板纸盖住了罐子的正面，并把一支带磁性盖子的擦干净的笔附在上面。

“我儿子喜欢和与他共用一个帐篷的朋友一起玩游戏，”埃莉诺说。“他认为这很棒。”
