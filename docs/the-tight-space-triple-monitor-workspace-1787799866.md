# 紧凑的三显示器工作空间

> 原文：<https://lifehacker.com/the-tight-space-triple-monitor-workspace-1787799866>

我们以前在小空间里展示过工作空间，但是 reader Edrei 的空间非常小，但是同时里面有很多很棒的设备。三个显示器，一个可调节的鼠标垫和书写表面，甚至一些偏置和键盘照明——不知何故都在那里。这里还有一些照片。

Watch

上面的照片是整个工作区的一个镜头，我们仍然惊讶于他设法将三个显示器放入其中。他指出:

> 距离我上次展示我的桌面已经一年多了，我想我会再次展示它，因为我对它做了一些修改。
> 
> 1.  在桌子旁边增加了一个玻璃搁板，这样我就可以放我的键盘和 HOTAS。
>     
> 2.  我没有带照明的机械键盘(我的是在它风靡一时之前买的，机械键盘可以使用一年半)，所以我在桌子边缘的下面贴了一个 LED 灯。问题解决了。
>     
> 3.  在游戏键盘的左侧建造了一个可移动的桌臂。这是一个带木板的壁挂式显示器臂，连接在桌腿上。耳机通过挂钩螺丝挂在这张桌子上。
>     
> 4.  第二个更长的可移动桌臂位于鼠标和书写的右侧，因为有了键盘，书写变得更加笨拙。可移动的桌子带来了很大的不同。

非常喜欢左边的可定制键盘和右边他为飞行棒留出的空间，但真正令人印象深刻的是那张可调节的桌子。这里有一张他把它当作写字台做笔记的照片:

这里，当他不需要在上面写字时，他把它作为鼠标垫的完美位置，摆动到一边并翻转，这样更容易把手臂放在上面:

这是一个非常好的定制设置，虽然桌子扶手是这场秀的明星，但他甚至为一些键盘照明腾出了空间，只是为了保持美观:

这是一个很棒的外观，也是一个将所有外围设备和所有东西放入一个小空间的好方法——这看起来像是一个壁橱，或者是隐藏在一个大得多的房间中的另一个角落。如果你和我们一样喜欢这个造型，去他的 Flickr 页面(链接如下)为你点赞吧。
