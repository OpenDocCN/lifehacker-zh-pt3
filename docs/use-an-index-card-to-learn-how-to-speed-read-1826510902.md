# 使用索引卡来学习如何快速阅读

> 原文：<https://lifehacker.com/use-an-index-card-to-learn-how-to-speed-read-1826510902>

快速阅读是一项很难掌握的技能。有大量的书籍、讲座和文章致力于教人们学习技能的“正确”方法。本周， *Fast Company* [采访了一位写了一本关于主题的书的专家](https://www.fastcompany.com/40574769/how-to-train-yourself-to-become-a-speed-reader) ，以获得一些关于如何学习的快速而肮脏的提示。她的建议之一是:使用索引卡。

Watch

这背后的想法非常简单。每当你阅读书籍或报纸等实体媒体时，在你阅读的句子上方放一张白色索引卡。把卡片放在你正在阅读的东西下面，会屏蔽掉它下面的所有东西，帮助你把注意力集中在你现在正试图处理的句子上。

虽然许多尝试这种技巧的人把卡片放在他们正在阅读的内容下面，但专家认为，把你已经读过的内容放在外面会鼓励你回头再读一遍。把这张卡片放在上面会把你已经做过的事情挡在外面，让你专注于你要去的地方，而不是你已经去过的地方。它还会把你的视线推向页面下方。

对于在线文本，你可以做类似的事情，总是从浏览器窗口的最顶端阅读文档。一边走，一边滚动，这样你读过的那一页就看不见了。

诀窍也不一定是索引卡。我对书签做了类似的操作，得到了类似的结果。它还能让我专注于我正在阅读的地方，所以当不可避免的干扰来临时，我可以从我停下来的地方继续阅读。
