# 这可能就是你在高速公路上看到这么多鞋子的原因

> 原文：<https://lifehacker.com/this-is-probably-why-you-see-so-many-shoes-on-the-highw-1818805900>

有没有在路上看到一双鞋子，想知道它们是怎么到那里的？不，这不是什么可怕事故的残留物。实际上有一些非常简单的解释。



别担心，好奇的朋友们。Redditor 86Quiche 最近肯定也遇到了“路上随意丢鞋”的现象， [昨天问了同样的问题](https://www.reddit.com/r/AskReddit/comments/72ebdg/people_whove_lost_a_shoe_on_the_highway_howd_that/):*有人在高速公路上丢了一只鞋，这是怎么回事？其他评论者很快做出了回应——一些人也要求知道真相，而另一些人很快给出了自己的解释。* 

最常见也是最可信的原因是，人是健忘的动物。几个 redditors 讲述了他们如何将运动鞋/礼服鞋/童鞋放在车顶上，然后分心，然后钻进汽车，带着车顶上的鞋离开。最终，这种踢腿经不起你的车的高速，它们飞到马路上，希望成为过往司机的一个有趣的谜。

人们也会出于某种原因扔鞋，尤其是孩子。在整个 Reddit 帖子中，父母们哀叹他们的孩子开车时想把东西扔出窗外的愿望——包括他们不需要的又小又贵的耐克。其他人讲述了配偶沮丧地扔鞋的故事(为什么？)，还有一些只是简单解释了一下自己的鞋子是怎么在开门的时候掉出车外，等自己意识到的时候已经太晚了。如果你想消磨时间的话，读读 会很有趣。

因此，虽然人们的鞋子在被车辆撞击时确实会飞掉，但当你发现一只运动鞋孤零零地停在公路路肩上时，你可能不会看到这种情况。它只是一只被某人遗忘、丢弃或丢失的鞋子。
