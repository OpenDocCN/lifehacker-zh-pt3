# 这些贴纸会告诉你你的孩子是否发烧，这样你就不用叫醒他们了

> 原文：<https://lifehacker.com/these-stickers-tell-you-if-your-kid-has-a-fever-so-you-1797946802>

有好几次，当我四岁的女儿睡着时，我和丈夫在她上空盘旋，困惑地摸着她的额头。“她觉得热吗？我觉得她觉得热。”“不，她可能只是因为所有的毯子而感到温暖。”“不，摸摸她——她真的很温暖。要不要叫醒她，给她量量体温？”“不，就让她一个人呆着吧。”"好吧，我就站在这里看看。"(是的，她是第一个也是唯一的孩子。你怎么知道的？)

耳朵和前额的温度计很棒，但是你不会一直带在身边，尤其是在旅行的时候。CNN 主持人凌志慧在脸书上发布了她的解决方案。休假时，她用 [发烧-Bugz 贴发烧指标](https://www.amazon.com/Physio-Logic-Fever-Bugz-Stick--Indicator/dp/B005JXC7EU/ref=sr_1_8?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/these-stickers-tell-you-if-your-kid-has-a-fever-so-you-1797946802&asc_source=&ie=UTF8&qid=1314975975&sr=8-8&tag=kinjalifehackerlink-20) 监测女儿的体温。

这些标签可以监测孩子的体温长达 48 小时，甚至在洗澡的时候。它们上面的小数字会亮起来指示读数，精确到 1 华氏度以内。家长们可以安心了，孩子们的额头上也有一只很酷的大虫子(或蝴蝶或蠕虫)。
