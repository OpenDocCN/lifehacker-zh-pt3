# 不，核桃不是减肥的关键

> 原文：<https://lifehacker.com/no-walnuts-aren-t-the-key-to-weight-loss-1798302763>

核桃已经有了健康食品的形象，但是加州核桃委员会想知道更多。一项新的研究询问，核桃——已经与降低患和 [【二型糖尿病】](http://jn.nutrition.org/content/143/4/512.full.pdf+html) 心脏病的风险相关——是否可能通过改变我们大脑对食物的反应来带来一些健康益处。



**头条:** [核桃减肥？](https://www.nytimes.com/2017/08/18/well/eat/walnuts-for-weight-loss.html)[(](https://www.nytimes.com/2017/08/18/well/eat/walnuts-for-weight-loss.html)N[e](https://www.nytimes.com/2017/08/18/well/eat/walnuts-for-weight-loss.html)wY[o](https://www.nytimes.com/2017/08/18/well/eat/walnuts-for-weight-loss.html)r[k](https://www.nytimes.com/2017/08/18/well/eat/walnuts-for-weight-loss.html)[T](https://www.nytimes.com/2017/08/18/well/eat/walnuts-for-weight-loss.html)I[m](https://www.nytimes.com/2017/08/18/well/eat/walnuts-for-weight-loss.html)e[s](https://www.nytimes.com/2017/08/18/well/eat/walnuts-for-weight-loss.html))

**故事:**首先，这项研究绝对没有测试任何关于减肥的内容。同样值得注意的是，这项研究部分是由加州核桃委员会资助的，但是 CWC 与这项研究的设计、执行或出版没有任何关系；研究人员自己提出了这项研究，并向 CWC 申请资助。

如果这不是减肥，那是什么？

这是一项小规模但严密的研究——双盲和安慰剂对照。九名肥胖参与者被给予核桃奶昔或安慰剂(核桃味的，热量和宏量营养素分解相同)，作为他们五天体重维持饮食的一部分。这一过程重复了两次，间隔一个月，第一轮服用安慰剂的人在第二轮服用核桃，反之亦然。在五天结束时，受试者被问及他们的饥饿程度，他们大脑对食物图像的反应被记录在核磁共振成像 中 [。](http://fmri.ucsd.edu/Research/whatisfmri.html)

在调查中，喝核桃的人报告说比安慰剂组感觉不那么饿(与该研究实验室 之前的发现相吻合)。大脑扫描结果是新的部分，揭示了当看到高回报食物的图像时(与蔬菜等低回报食物或岩石等非食物相比)，喝核桃的人在大脑中被称为脑岛的区域表现出更多的活动，这被认为与食欲和冲动控制有关。这种功能磁共振成像分析，或称 fMRI，为研究人员提供了对大脑活动的令人兴奋的见解，但在其复杂的方法学 中发现了 [缺陷，对 fMRI 发现的解释是](http://ideas.ted.com/much-of-what-we-know-about-the-brain-may-be-wrong-the-problem-with-fmri/) [仍有争议的](https://www.theguardian.com/science/neurophilosophy/2015/apr/09/bold-assumptions-fmri) 。

那个活动是不是意味着面对垃圾食品的时候核桃给了你更多的意志力？不一定。除了肥胖受试者，这些结果还适用于瘦型患者吗？不知道。核桃很特别还是其他坚果也可以？Idk。

好吧，但是核桃对这些参与者的大脑有什么影响呢？我很想知道，这项研究背后的研究人员也想知道。这是在弄清楚人们所吃的许许多多食物中的一种对健康的明显益处背后的机制和程度的长征中迈出的一小步。

要点:核桃*可能*会让你感觉更满足，而核桃*可能*会刺激你大脑的一部分，而*可能*会增强你对冲动的控制。关于饱腹感和冲动控制如何影响我们少吃的能力，我们可以得出自己的结论，但这项研究绝对不是关于减肥的。
