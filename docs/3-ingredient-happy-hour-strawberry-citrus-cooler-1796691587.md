# 三种成分的欢乐时光:草莓柑橘冷却器

> 原文：<https://lifehacker.com/3-ingredient-happy-hour-strawberry-citrus-cooler-1796691587>

周末快乐，欢迎回到 [*三料欢乐时光*](http://lifehacker.com/tag/3-ingredient-happy-hour) ，每周饮品专栏，提供超级简单又美味的酒水。本周，我们将夏季草莓与我最喜欢的柑橘利口酒搭配在一起: [皮埃尔·费朗干橙库拉索酒](http://pierreferrandcuracao.com/en/section/view/210) 。



不像它的暴力蓝，更甜的表亲，干库拉索岛是平衡和多汁的坚果温暖和一个很好的苦乐参半的味道。它足够美味，可以单独享用，但我喜欢在将它倒在碎冰上之前，将它与搅拌好的草莓和新鲜柠檬汁搭配，以获得几乎像冰一样的体验。它不仅是夏日啜饮者的缩影，也是任何没有做成馅饼的浆果的绝佳用途。

要制作它，你需要:

*   草莓(4-5 个小的或 2 个大的)
*   2 盎司干柑桂酒(*不要为这种饮料准备任何其他柑桂酒* )

*   3/4 盎司新鲜柠檬汁

在摇壶里捣碎草莓，加入所有其他原料，和一些冰。用力摇晃至少 30 秒，滤入满满 3/4 碎冰的低杯。用柠檬汁中的柠檬皮装饰，在将柠檬皮放入玻璃杯之前，将柠檬汁挤在成品饮料上。
