# 让你相信世界正在崩溃的认知偏见

> 原文：<https://lifehacker.com/the-cognitive-biases-that-convince-you-the-world-is-fal-1822620516>

这个世界现在看起来很艰难。如果你打开电视或打开网络浏览器，你几乎肯定会受到坏消息的轰炸。但是这个世界真的注定要毁灭吗？还是你的动物大脑在作弄你？振作起来——事情并不总是像看上去的那样糟糕。

### “衰落主义”正在抬头

首先，对大多数人来说，我们国家的未来看起来相当严峻。根据美国心理协会 的数据，63%的美国人对美国的未来感到紧张，59%的人认为“这是他们记忆中美国历史上的最低点”这种类型的思维很可能是由于“ [衰落主义](https://en.wikipedia.org/wiki/Declinism) ”，这是一种偏见导致的信念，即一个社会或机构总是随着时间的推移变得越来越差——无论事实如何。基本上，任何时候你认为“事情永远不会像过去那样好”，那就是衰落主义。我觉得我经常听到类似“2017 年很糟糕吧？”或者“事情不能再糟了……”不对！事情肯定会变得更糟，事实也的确如此。时光倒流得越远，事情就变得越糟。

当然，有很多问题需要解决，但事情实际上正在逐渐好转。如果你看看事实，人们的总体福祉比以往任何时候都要好。你只需要看看认知心理学家(和终极乐观主义者)的工作就够了。他的作品，像*的《我们本性中更好的天使》和他即将出版的书*现在的启蒙:理性、科学、人文主义和进步的案例*，展示了自从有史以来，很多“坏东西”实际上是如何逐渐减少的。那为什么现在一切看起来如此糟糕？*

### 我们带着玫瑰色的眼镜回望过去

我们的衰落主义态度有两个主要的促成因素:被称为“”和“”的偏见第一，正如临床精神病学家戴尔·阿彻解释的那样，成年人倾向于对青春期和成年期发生的事情有更生动、更愉快的记忆和怀旧感。基本上，当我们回顾我们年轻时的美好时光时，这个世界似乎更光明，更美好，但现实是我们对工作的压力更小，可能不必担心钱，也不太关注新闻。

另一方面，积极效应是指随着年龄的增长，我们倾向于记住更多积极的事情而不是消极的事情。在 2005 年的一项研究 中，研究人员发现了这种大脑把戏在起作用的证据，这表明老年人通过自然保留更高比例的积极记忆来更加关注情绪调节。本质上，我们越老，我们越想只想到过去的好东西，甚至当我们试图以批判的眼光回顾过去，改变我们在特定时间点对世界的看法。

阿彻 说: [积极效应与回忆碰撞完美结合:](https://www.psychologytoday.com/blog/reading-between-the-headlines/201702/declinism-why-you-think-america-is-in-crisis)

> 随着年龄的增长，大脑不仅更倾向于记住积极的刺激而不是消极的刺激，而且更倾向于强调积极的，尽量减少消极的记忆。因此，如果我们热衷于回忆过去，并强调这些回忆中的积极因素，那么今天似乎比昨天糟糕得多就不足为奇了。

我们不能不主观地思考过去、现在和未来。但阿彻指出，这不是我们性格的缺陷，这只是我们大脑的工作方式。你无法阻止自己经历这些偏见，但是了解这些偏见可以帮助你保持客观。

### 我们被现在的消极所吸引

我们有时会忘记，我们强大的人类，这个星球的征服者，只是关心自己生存的愚蠢动物。这种生存的进化驱动力导致我们的大脑总是在寻找威胁，高估危险，并且不会长时间停留在积极的事情上以避免变得自满(这在过去意味着挨饿或成为别人的午餐)。这种本能就是我们现在所说的“ [”消极偏见](https://en.wikipedia.org/wiki/Negativity_bias) ，”这基本上意味着我们把更多的注意力放在坏事上，而不是做好事。

所说的偏见对我们的基本生存需求来说可能是好的，但它让我们从娘胎里就对现代世界持相当悲观的看法。我们不仅会带着积极的情绪回顾过去，而且注定会带着消极的情绪看待现在的一切。多谢了，布莱恩。更糟糕的是，人们知道我们容易消极，所以营销人员、政治家和媒体利用这一点。有没有想过为什么新闻总是一件接一件可怕的事情？这是因为他们需要收视率，而我们喜欢呆呆地看着这个世界的暴行，不管我们愿不愿意承认。如果你患有抑郁症，[2011 年](https://link.springer.com/article/10.1007%2Fs10608-011-9397-4) 的研究表明，消极偏见会对你产生更大的影响。这是一个消极的恶性循环，不断地自我反馈，但这就是我们被设定的方式。

### 我们被我们的信仰所束缚，这是个问题

我们都相信我们对我们的国家，对世界，对生活本身有正确的看法。那很好，坚持己见是很自然的。但是当你把类似于“”的确认偏差扔进混合物中时，我们经常会在我们的信念中如此根深蒂固，以至于我们错过了打破无休止的消极链的机会。如果你不熟悉，确认偏见是我们倾向于寻找只支持我们个人世界观的事实，拒绝挑战它的事实。本质上，确认偏差是选择性的事实保留，或者说是一种“一厢情愿的想法”我们都这样做，这就是为什么意识到这一点很重要。

如果不加以控制，确认偏见可能是危险的，但这也是为什么我们很难摆脱我们对世界预设的愤世嫉俗的观点的原因。因为消极偏见，我们认为现在的事情很糟糕；因为回忆和积极效应，我们认为过去的事情会更好；因为衰落主义，我们认为未来的事情会变得更糟。确认偏见是我们糟糕的大脑圣代上的樱桃。天哪，我打赌你现在在想，“不，事情*糟透了。让我去找一些事实给你看。”但这就是问题所在！我们需要停止只寻找坏的方面，而*尝试*寻找好的方面。外面的世界比你想象的要多。*

这并不是说世界上没有问题，但是了解这些认知偏见可能会帮助你从不同的角度看待世界——或者至少理解为什么你认为事情看起来如此可怕。永远不要忽视需要解决的问题，但也不要让你大脑对世界的悲观看法让你陷入绝望。
