# 怀孕不会让你成为更好的运动员——它可能会让锻炼更加困难

> 原文：<https://lifehacker.com/pregnancy-doesn-t-make-you-a-better-athlete-but-it-doe-1794633994>

塞雷娜·威廉姆斯在赢得澳网冠军时可能已经怀孕，所以《新科学家》问 [她的怀孕是否可能对](https://www.newscientist.com/article/2128439-how-pregnancy-could-affect-an-elite-athlete-like-serena-williams/) 有所帮助——好像她之前的 [38 个主要头衔](https://en.wikipedia.org/wiki/Serena_Williams) 都是侥幸。但这是一个已经流传了一段时间的神话，所以让我们来谈谈怀孕对运动员身体的真正影响。



女人的身体 [在怀孕期间产生更多的血液](https://www.ncbi.nlm.nih.gov/pubmed/4075604) 。运动员在训练时也会发生类似的变化。有了更多的血液，你就能给你的肌肉提供更多的氧气。这是否意味着怀孕能提高成绩？

可惜没有证据支持这个想法。怀孕运动员的经历往往会走向相反的方向。我在为一场马拉松 训练的时候怀孕了 [，我发现即使我坚持训练，我的体能还是下降了。通常你越训练越快；我变得越来越慢。我的长跑速度从每英里 10:30 减慢到 12 分钟，并且我一直是最后一个完成团体训练跑的人。怀孕五个月的时候，我觉得自己很幸运，能在赛道开放的六个小时内完成马拉松。平均配速:13:09。](https://lifehacker.com/the-truth-about-exercising-while-pregnant-its-okay-to-1682602976)

为什么即使我们有更多的血容量，我们还是会慢下来？可能是因为孕妇额外的血液和氧气不能给肌肉提供能量；*交给宝宝*。似乎人们对血容量和摄氧量的增加感到兴奋，却忘记了这个重要的事实。这使得小威廉姆斯最近的头衔更加令人印象深刻:如果她怀孕八周，她可能一直在处理*耐力下降的问题。但她还是赢了。*
