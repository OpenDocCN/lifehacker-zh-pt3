# 使用这些 Mac 快捷键来代替 Esc 键

> 原文：<https://lifehacker.com/use-these-mac-shortcuts-instead-of-the-esc-key-1828236210>

这里有一条愚蠢的捷径，总有一天会派上用场。在一些应用程序中， **⌘-.**与退出键的作用相同。所以如果那个键坏了，或者你的 [愚蠢的触摸条](https://lifehacker.com/fix-your-touch-bar-with-this-app-1823463782) 藏着你愚蠢的虚拟退出键，或者你的手够不到你键盘的一角，或者你只是更喜欢人体工程学，那就去争取吧。



感谢黑暗模式戴夫的提示:

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1023551777260613633&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1023551777260613633&autosize=1) 

它在一些应用程序(包括 Chrome)中不起作用，所以你必须反复试验。正如卡尔·布朗 [指出](https://twitter.com/CarlBrwn/status/1023592202566594560)**Ctrl-【**是另一个选项，这个选项在 Chrome 中确实有效。至少有时候是这样。

为了更可靠的替代，OS X 日报 [建议](https://twitter.com/osxdaily/status/1023596964334559232) 手动映射大写锁定以逃避，这有一个额外的好处，就是每当你要大喊大叫时就让你关机。

键盘快捷方式的使用是一个基本的标志某人的“擅长电脑”这就是为什么电影里的黑客都没碰过鼠标。如果你真的很认真，你 [自己做快捷键](https://lifehacker.com/turn-any-action-into-a-keyboard-shortcut-a-beginners-g-316589) ，但最起码你需要背下 [几个必不可少的标准连击](https://lifehacker.com/keyboard-shortcuts-ranked-1822042994) 。
