# Lifehacker 正在招聘一名高级科技作家

> 原文：<https://lifehacker.com/lifehacker-is-hiring-a-senior-tech-writer-1820893406>



你是否有大量的技术技巧、诀窍、观点和见解，但却没有人可以传授给你？你是一个机智、简洁、深刻的作家，能把复杂的过程提炼成引人入胜、易于理解的指令吗？你是否一直在尖叫着杀手级的 Chrome 扩展、免费的 VPN、秘密的 Roku 频道和省时的 IFTTT 食谱，但你听到的只是你自己的声音在黑暗中回响？我们谈谈吧。

Lifehacker 正在招聘一名 [高级科技作家/编辑](http://gizmodo.com/careers/job/932734?gh_jid=932734#_ga=2.201317677.2030091814.1512064874-672228682.1489692420) 。这是一个全职职位，有竞争力的工资和良好的福利，在 Gizmodo Media 的纽约市母舰上工作。你将推销和撰写故事，帮助塑造 Lifehacker 上技术报道的愿景，完全以最快、最聪明、最简单和最好的方式完成工作。这不是一份报道科技新闻或评论小工具的工作——这是一份适合希望接触科技的 MacGyver 类型的工作，旨在弄清楚我们全球超过 2400 万的观众如何利用科技让他们的生活更快乐、更有效率、更有趣。

听起来很吸引人？我们迫不及待地想听到你的消息。
