当然，你需要完成下周的 PowerPoint 演示文稿，但快速检查一下脸书，看看马克是否上传了上周末聚会的照片，这不会有什么坏处。然后从亚马逊快速下单。你真的应该喜欢画廊里的一些照片...很容易在网上浪费几分钟(或几小时)而没有意识到。如果你是一个受虐狂，并且想知道这种无所事事的浏览到底花费了你多少时间和金钱，一个名为“懒惰值得 T2 T3”的火狐和 Chrome 扩展可以帮上忙。它跟踪你的网上冲浪，让你知道什么时候“快速浏览”开始失控。

在某种程度上，这款应用的功能依赖于你有一份不为互联网写作的工作。本周，我和尼克·道格拉斯都带着 Sloth Worth 出去兜了一圈，并很快得出了相同的结论:它不是生活黑客作家的延伸。

这个扩展似乎认为很多网站都浪费了时间，包括这个网站，所以如果你的工作就是让 Lifehacker 一整天都开着，你会很快浪费掉大量的时间。然而，如果你真的应该整天处理 PowerPoint 演示文稿和 Excel 电子表格，那么它可能会派上用场。

有了这个扩展，你输入你的时薪，然后它会记录你一天中在网上“浪费”了多少时间，用一个等同于你时薪的金额来量化。所以，如果你每小时挣 30 美元，在脸书玩了 3 个小时，你基本上“浪费”了 90 美元(或者说你在脸书得到了 90 美元，这取决于你在哪里冲浪)。

在 Chrome 的扩展图标中，以及每个违规页面的右上角，都有一个运行记录。虽然这是对你浪费时间的所有网站的连续记录，而不是一个网站一个网站的细分。

不幸的是，你也不能调整哪些网站被认为是浪费时间，你必须依靠懒惰的价值。这意味着它可能(并且很可能会，根据我的经验)错误地计算你需要工作的一些网站，就像它对脸书一样。

虽然运行计数对几乎所有人来说都是毫无意义的，但我认为，网站角落上的时钟非常有用。对我来说，阅读故事或浏览脸书邮报很容易掉进兔子洞。在我意识到之前，一个小时(或两个小时)已经过去了，我已经大大落后于当天需要完成的事情。记住，边上的时钟是时间的一个很好的数字表示，也是我需要的鼓励(或内疚)来关闭几页，继续下一件事。

鉴于我的职业，我认为懒惰对我来说有点太烦人了，我不能总是装着离开。也就是说，当我需要在一个大项目中集中精力时，或者当我试图主动减少对某个特定网站的使用时，我肯定能看到它派上用场。
