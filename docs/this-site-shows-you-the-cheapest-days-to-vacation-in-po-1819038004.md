# 这个网站告诉你在热门城市度假最便宜的日子

> 原文：<https://lifehacker.com/this-site-shows-you-the-cheapest-days-to-vacation-in-po-1819038004>

在过去的两年里，我的两个密友去意大利度过了三次一至两周的假期。乍一看，他们的多次欧洲度假让他们看起来像是突然获得了大量现金。事实证明，关键在于他们旅行的时间:在淡季。



虽然去佛罗伦萨和罗马的旅行可能会达到数千人，但他们已经开始在 Expedia 上预订套餐旅行，这些旅行的价格仅为数百元。我被深深打动了，我和我的男朋友最终在三月份加入了他们在佛罗伦萨的聚会。太棒了。

根据你旅行的地点，人们去那里旅行有“淡季”和“旺季”之分。旺季的天气通常更好，旅游也更容易。然而，在淡季旅游也有它的好处。在我们的佛罗伦萨之旅中，我们的旅行团只有几个人，街道很容易导航，所以我们可以真正享受它们，我们可以随便逛逛博物馆和教堂，然后直接走进去。相比之下，几个月后去了这座城市的朋友报告说，他们花了几个小时排着对我们来说根本不存在的队，不得不在每个地标、火车站和餐馆与一群人打交道。是的，我们在那里的时候有点冷，但是我想我们赢了。

如果你想在淡季去某个地方旅行，一个名为 [Offpeak.io](http://offpeak.io) 的新网站可以帮上忙。该应用程序查看一年中酒店价格的中位数，以确定该地区的“淡季”可能是什么。然后你可以利用这些信息来计划一次旅行。

此外，它还考虑到了镇上正在发生的可能导致成本上升或使场所拥挤的事件，并让您知道这些事件是什么。例如，下个月在旧金山，Salesforce 将举行年度 Dreamforce 会议。你可以通过下图中那个巨大的峰值看到它在哪里。

如果我正在计划一次去城市的休闲家庭度假，那么我可能会利用这些知识来计划一周(甚至几天)后的旅行，那时酒店更容易找到，而且价格只有原来的四分之一。

可以按酒店的星级来排序信息，最多提前 11 个月看。你可能还是想在预订前多做一些调查(波士顿的酒店之所以这么便宜，是因为这个城市通常会在那个星期被大雪覆盖吗？)，这样你就知道自己进入了什么状态。该网站还列出了每天的典型高/低温度，因此您可以使用这些图表来找到天气和价格的完美组合。
