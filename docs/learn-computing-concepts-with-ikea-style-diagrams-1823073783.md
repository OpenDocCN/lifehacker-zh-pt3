# 通过宜家风格的图表学习计算概念

> 原文：<https://lifehacker.com/learn-computing-concepts-with-ikea-style-diagrams-1823073783>

如果你想学习计算机算法，或者向其他人解释它们，三位计算机科学家按照宜家指南 的风格制作了一些 [有用的视觉辅助工具。](https://idea-instructions.com/)

Watch

桑多·p·费科特、塞巴斯蒂安·莫尔、塞巴斯蒂安·斯蒂勒至今已经创作了六幅图，讲解了 [公钥密码学](https://idea-instructions.com/public-key/)[二分搜索法](https://idea-instructions.com/binary-search/)[归并排序](https://idea-instructions.com/merge-sort/)[平衡树](https://idea-instructions.com/avl-tree/)[图扫描](https://idea-instructions.com/graph-scan/)T21【Fleury】一笔画图的算法 。有些解释比其他的更清楚，所有的解释都和一些更传统的解释结合在一起是最有用的。作为一个门外汉，我发现公共键和合并排序图最有帮助，部分原因是它们最具体。

也可以看看 [一个类似的打包算法图](https://morr.cc/split-packing/) ，这是莫尔为他的硕士论文创作的，可以将圆变成其他形状。
