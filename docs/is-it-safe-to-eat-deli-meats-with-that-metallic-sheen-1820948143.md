# 食用带有金属光泽的熟食肉安全吗？

> 原文:[https://the take out . com/is-it-safe-to-eat-deli-meats with the metallic-sheen-1820948143](https://thetakeout.com/is-it-safe-to-eat-deli-meats-with-that-metallic-sheen-1820948143)

你有没有翻过一片烤牛肉，抓住那一丝丝？一丝彩虹般的彩虹色，对于一个普通的冷切来说有点超凡脱俗？即使是新鲜的，一些肉片也会散发出金属光泽，让你犹豫不决。但是你看到的是普通的干涉光学现象，而不是腐臭腐败的标志。除非你的肉开始发臭，或者彩虹般的颜色伴随着一些生长中的模糊微生物，否则你的午餐肉是可以安全食用的。

每当一个表面反射的光波遇到附近另一个表面反射的光波时，就会发生光学干涉。当这些波重叠时，我们只能看到合成的和。如果两个波同相——也就是说，如果波峰与波峰对齐，波谷与波谷对齐——那么总和就是一个更大的波，称为*相长干涉*。如果两个波是异相的——波峰与波谷排成一行——那么总和就完全抵消了，称为*相消干涉。*

就午餐肉而言，有两种干扰效应是可能的。

第一种叫做*薄膜干涉*。一些熟食，尤其是腌肉，富含脂肪和油。如果这些脂肪渗出，它可以在表面形成一层，并将情况从简单的单一反射变为双重反射——一个离开空气与脂肪层相遇的*前*表面，另一个离开脂肪层与肉相遇的*后*表面。由于脂肪层很薄，这种微小的距离差异意味着两个反射波是异相的；与没有移动的波峰相比，必须移动的波峰的位置被移动了。当波异相时，它们相互抵消，光就消失了。*然而，*虽然这对于大多数穿过薄层的光来说是正确的，但是可能有一些选定波长的光穿过薄膜并返回的距离*恰好*保持前后反射同相。那些选择的颜色不会被取消，所以它们成为我们唯一看到的颜色。

这种类型的干涉赋予了浮油紫色和绿色的色调，也赋予了气泡彩虹般的闪烁。这些薄膜，就像一片火腿上的脂肪层，厚度只有几微米(与光的波长相当，比人头发的宽度小 50 倍),这意味着我们通常只有几十个分子厚的层。从一个点拿走一些脂肪滴，把它们堆积在其他地方，厚度变化足以改变波长，使之适合该点的相长干涉条件。你看到的“彩虹”是随着厚度的变化，从一种主色逐渐转变为另一种主色。

第二种干涉效应叫做*衍射*。每当光穿过等间距狭缝的重复网格或从等间距反射器的表面反射时，就会发生衍射。这种被称为*衍射光栅的结构*会产生许多反射点，这意味着我们不能只考虑两个反射波如何干涉，而是必须考虑大量反射波的总和。在这种情况下，几乎所有波长的强度不仅减弱，而且完全消失；建设性干涉的颜色变得*非常*清晰。如果你看过 CD 或 DVD 上的凹槽发出的光，你就会看到这种效果。与浮油的逐渐波动不同，当你倾斜圆盘时，你会看到一个绚丽的万花筒在光谱的所有颜色中旋转。

这和午餐肉有什么关系？肌肉是由蛋白质组成的，这些蛋白质结合成股，而这些股又聚集成长纤维，每条纤维的直径约为 1 或 2 微米。这些构建模块在单块肌肉中不断重复，身体在运动时可以协调收缩部分或全部这些纤维。当一块肌肉被烹饪并最终被切割时，肌肉纤维的这种重复结构被暴露出来，形成可见光的。入射到肉片上的任何混合光都会遇到这些有序的纤维，并且在特定的角度只反射出特定的颜色。事实上，科学家测量不同颜色的光从不同类型的肌肉纤维衍射的角度，以研究这些肌肉群的结构和大小。

然而，并不是所有的午餐肉都闪闪发光。彩虹效应最容易在对比度更大的深色肉类上看到，如烤牛肉或火腿。它在从整块肌肉上切下来的肉上也最明显——而且只有逆着纹理切的时候——这保持了肌肉纤维的自然周期性。“ [重组](https://www.youtube.com/watch?v=RrmcO1QBEo8) ”或者像熟食火鸡或大腊肠这样的加工肉类是通过将单独的小块肉捆绑在一起并重新塑形而制成的。在该过程中，所有肌肉纤维的顺序都丧失了，此外，粘合剂倾向于堵塞存留下来的纤维之间的任何间隙，进一步降低了格栅的周期性和结构。

这种效果在生肉中也很少见。尽管原始肌肉中的纤维轮廓分明，但它们更小更松散，这意味着它们 [无法保持衍射所需的刚性形状](http://www.slate.com/blogs/browbeat/2013/09/09/iridescent_deli_meat_why_some_sliced_ham_and_beef_shine_with_rainbow_colors.html) 。一旦肉被烹饪或腌制后，纤维变紧，增强了磨碎效果。此外，在烹饪过程中吸收盐的纤维会保留更多的水分和膨胀，保持必要的重复结构。

虽然彩虹肉看起来可疑，但并没有什么问题。这种金属光泽只是一种光学把戏，它会影响你的午餐肉的外观，但不会影响它的味道或安全性。再堆上一层咸牛肉，吃光。
