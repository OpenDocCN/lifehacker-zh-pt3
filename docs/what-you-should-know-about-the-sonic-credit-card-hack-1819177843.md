# 你应该知道的声波信用卡黑客

> 原文:[https://life hacker . com/what-you-should-known-the-sonic-credit-card-hack-1819177843](https://lifehacker.com/what-you-should-know-about-the-sonic-credit-card-hack-1819177843)

昨天，快餐连锁店 [Sonic 发布了一份声明](http://www.businesswire.com/news/home/20171004006270/en/SONIC-Drive-In-Notice-Payment-Card-Breach) 承认，在今年的一次恶意软件攻击后，客户的信用卡和借记卡信息可能被盗。我们现在知道该怎么做了，对吧？

安全专家和调查员 Brian Krebs 早在 4 月份就发现了潜在的黑客行为，当时他在一个被盗数据论坛 上发现了一批被盗的卡出售。克雷布斯报[报](https://krebsonsecurity.com/2017/09/breach-at-sonic-drive-in-may-have-impacted-millions-of-credit-debit-cards/)报:

> 显然是从索尼克那里偷来的账户是一批卡的一部分，小丑的仓库称之为“Firetigerrr”，它们是由城市、州和邮政编码索引的。这种地理上的特殊性允许潜在的购买者只购买从住在他们附近的 Sonic 客户那里偷来的卡，从而避免了一种常见的反欺诈防御，在这种防御中，金融机构可能会阻止来自已知受损卡的州外交易...Firetigerr 批次中宣传的卡的价格比其他违规事件中被盗的卡要高一些，可能是因为这批卡非常新，不太可能被发卡银行注销。

黑客闯入销售点系统，使用恶意软件复制卡磁条上的账户数据。Krebs 解释说，有了这些数据，窃贼就可以制作一张伪造的卡，从“电子商店和大盒子零售商那里”购买昂贵的东西。

根据他们的声明，Sonic 将为今年在其任何地点使用信用卡的任何客户提供来自 Experian 的免费信用监测。正如Consumerist 指出的那样，然而，“Experian 监控服务 [在其服务条款](https://www.experianidworks.com/terms/) 中包含一项强制仲裁条款，这意味着如果你对这项服务有法律纠纷，你将无法起诉 Experian。”(同样值得指出的是，像 Equifax 一样，Experian 的信用监控 URL 与 Experian.com 位于不同的域，这可能使其成为网络钓鱼的容易目标。)

此外，正如我们之前告诉你的，有许多其他方式来免费监控你的信用 ，所以这不像 Sonic 提供你自己已经无法获得的东西。像 Lifelock 这样的付费服务提供了更多的监控(例如，其中一些服务说他们会在黑暗的网络上寻找你的信息)，但像 [WalletHub](https://wallethub.com/) 或 [CreditKarma](https://www.creditkarma.com/) 这样的免费服务会向你显示你的信用报告的定期更新，甚至在出现问题时提醒你。

如果你还没有，你也可以在这三个局中的每一个局对你的信用档案进行 [冻结或](https://twocents.lifehacker.com/everything-you-need-to-know-about-a-credit-freeze-1803139061#_ga=2.175944490.945973407.1507219650-1268082208.1431441811) 欺诈警告。当然，这是另一个提醒，定期检查你的信用卡和银行账户。大多数银行还允许你为超过一定限额的交易设置提醒。

那么为什么这种情况会一直发生呢？克雷布斯指出，黑客窃取信息如此容易的部分原因是，许多金融机构仍在使用磁条卡，而 [尚未改用更安全的芯片卡](https://twocents.lifehacker.com/how-the-newest-generation-of-credit-cards-protects-your-1732724619#_ga=2.175944490.945973407.1507219650-1268082208.1431441811) 。他补充说，在其他餐馆使用的卡也有可能被泄露，但具体位置仍未得到证实。
