# 请教专家:良好的跑步状态和避免受伤

> 原文：<https://lifehacker.com/ask-an-expert-all-about-good-running-form-and-avoiding-1786806434>

作为一个没有经验的跑步者，我不认为我需要知道太多；跑步就像走路，但是更快，对吗？但是，当然，花足够的时间在人行道上，你会很快了解到，除了耐力，还有更多可以避免跑步受伤的方法。你需要注意*如何*跑。



丽贝卡·舒尔茨博士将为您提供一些关于正确跑步方式的指导。舒尔茨博士是斯坦福大学的临床生物力学专家，也是研发下一代健身追踪器的初创公司[【Lumo body tech】](http://www.lumobodytech.com/)的研究员和设计师。她研究了跑步的字面人体力学，可以提供一些建议，告诉你如何改善你的形式，以提高效率，降低受伤的风险。如果你正在为马拉松进行训练，或者只是想改善你的日常慢跑，舒尔茨博士将在接下来的一个小时里提供建议。下面留个问题！
