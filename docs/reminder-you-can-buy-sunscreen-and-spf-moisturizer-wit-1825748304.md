# 提醒:你可以用你的 FSA/HSA 的钱买防晒霜和 SPF 润肤霜

纽约的气温高达 90 度，这意味着人们终于脱下了他们的皮大衣和单调的黑色冬装，换上了不那么单调的黑色夏装 g。随着层层叠叠的去除，一年一度的提醒也随之而来，出门前要涂上厚厚的防晒霜。

防晒霜是健康护肤的重要组成部分。正如我们的健康编辑贝丝·斯克瓦莱茨基所写的那样，你应该“每天”都戴着它，以保护自己免受太阳无情的射线的伤害，即使不是 90 度。

它也很贵，介于各种不同的覆盖范围和花哨的含 SPF 的精华液和保湿霜之间。(令人不安的是， [《消费者报告》](https://www.consumerreports.org/cro/sunscreens/buying-guide/index.htm) 发现，在过去的五年里，他们测试的防晒霜中有大约一半没有达到瓶子上标明的防晒指数。所以你会想在购买前 [查看他们的评分](https://www.consumerreports.org/products/sunscreen/ratings-overview/) 。幸运的是，如果你有 FSA 或 HSA，你可以报销购买防晒霜的费用。

只要防晒霜是 [SPF 30 或者更高的](https://www.cigna.com/qualified-health-care-expenses) ，你应该可以用你的 [FSA 或者 HSA 的钱](https://member.aetna.com/appConfig/memberSecure/assets/html/HSA_Allowable/HSA_Allowable.html) 来支付。你会想与你的保险公司仔细核对，看看有哪些符合条件，你也可以在 [FSA 商店](https://fsastore.com/Sun-Care-C253.aspx) 和你的保险公司网站上找到符合条件的商品。

更多好消息？SPF 值高的面部润肤霜也符合条件。你可以在 FSA 商店的网站上看到一些，包括来自品牌[【CeraVe】](https://fsastore.com/CeraVe-Sunscreen-for-Face-Lotion-SPF-50-25-fl-oz-P22823.aspx)[理肤泉](https://fsastore.com/La-Roche-Posay-Anthelios-50-Daily-Tone-Correcting-Primer-P25040.aspx) 和 [Supergoop](https://fsastore.com/Supergoop-Day-To-Day-Value-Set-P25107.aspx) 的选项。( [唇部护理](https://fsastore.com/Lip-Care-C51.aspx) 也算！)

如果你想超越防晒霜，阻挡 UVA 和 UVB 射线的处方太阳镜也有资格报销，如果你最终被烧伤， [芦荟霜和软膏](https://fsastore.com/FSA-Eligibility-List/A/Aloe-Vera--E797.aspx) 如果你有处方也可能有资格。同样，你应该在匆忙出门买东西之前咨询你的计划提供者。

最后提醒一句(因为我今年一直在大声吹响防晒号角):正如贝丝所写的，仅仅防晒霜不会保护你免受太阳的暴怒。如果你长时间呆在户外，衣服、帽子和防紫外线太阳镜也是必不可少的——遮荫也是如此。还有别忘了 [保护你的眼皮](https://vitals.lifehacker.com/you-re-probably-forgetting-to-put-sunscreen-on-the-most-1821122791#_ga=2.196736044.107940156.1525096857-594046802.1524762060) 。
