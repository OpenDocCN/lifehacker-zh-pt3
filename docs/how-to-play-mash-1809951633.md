# 怎么玩 MASH

> 原文:[https://lifehacker.com/how-to-play-mash-1809951633](https://lifehacker.com/how-to-play-mash-1809951633)

算命游戏 MASH 被小学生保留了几十年，只需要铅笔、纸和一个朋友。下面是如何玩这个小游戏。

Watch

*拿出那些捕兽器，削尖你的 2 号铅笔——这是* [*返校*](http://lifehacker.com/tag/back-to-school) *周！在教室之外，我们将带给你一些天才的技巧和想法，告诉你如何开始日常活动，重温旧技能或在这个秋天学习新东西。*

1.  写一份列表，列出你的类别:生活伴侣、孩子数量、工作、薪水、汽车和居住地。

2.  为每个类别列出四五个选项，让你的朋友为每个类别选择一个糟糕的最后选项。
3.  从 3 到 10 中选择一个数字；数出你的选择，划掉一个，然后重复。
4.  当一个类别中除了一个选项都被划掉时，圈出剩下的一个。
5.  读出你的全部财富。

有几十种变化。一些玩家把他们的类别放在一个巨大的井字格子里；有些人通过画点或记号来选择他们的随机数。其他类别包括宠物、蜜月计划、大学专业、伴侣的工作以及你能活多久。

习惯上，你的朋友会为你写下你的财富，而你会告诉他们你的选择。礼节要求在他们完成你的财富后，你要回报他们。
