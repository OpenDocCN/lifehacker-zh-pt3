# 你的浴室里需要手巾

> 原文：<https://lifehacker.com/you-need-hand-towels-in-your-bathroom-1797974287>

浴室里没有擦手巾的人:你认为你的客人用什么擦手？



你希望我们用你的浴巾吗？你认为我们知道你用毛巾的哪一部分洗脸，还是用自己的手？或者你知道我们可能会抓住你擦遍屁股的那部分毛巾？

你想让我们在空中疯狂地挥舞双手，把水洒满你的浴室吗？你想让我们征用你的浴帘或者卫生纸或者我们自己的裤子吗？你以为我们就是不洗手吗？

你意识到你引起的混乱了吗？羞耻、困惑和背叛感？每次你举办派对时浴室里都会发生大规模恐慌？

如果你不想要手巾，那是你的特权。你只是在自己的身体上来回交换你的虱子(和死皮片)。

但是强迫你的客人使用你的浴巾，天知道最后一次是什么时候洗的，传达了一种近乎敌意的粗心，而这绝对不是你的本意。这应该是电影反派建立统治地位的一个保留动作，就像 [强迫一个懦弱的角色剃掉他们的](https://www.youtube.com/watch?v=ikUz-s8lBVI) 。

也是不愉快的亲密。裸体时用浴巾；它适用于你的私密部位。一个客人在上面搓手，是在摸索你刚洗过澡的早晨的自己。如果毛巾还是湿的，我们会立刻意识到这种潮湿不是来自你自己的手，我们在问候或告别时会高兴地抓住你的手，而是来自你身上我们从未见过的部分，更不用说爱抚了。你不妨让我们用你的牙刷。

最后， [共用毛巾是健康隐患](http://www.businessinsider.com/microbes-bacteria-bathroom-laundry-wash-your-bath-towels-2015-9) 。你传给自己的细菌比你传给别人的细菌危害小，因为别人的身体并不总是具有相同的免疫力。当然，这仍然是手巾的一个风险，但不那么大了，因为这些手巾在软绵绵地挂在你家最潮湿的房间里“晾干”之前，没有摩擦过你生殖器孔的奇异培养皿。

你的手巾可能很便宜。他们可能很丑。没人在乎，只要它们能被清楚地识别为手巾——不是浴巾，也不是地板巾。为此，它们必须比其他毛巾小。除此之外，狂野一点。

请不时洗一下你的手巾。理想的情况是每周一次，但至少要和其他衣物一样频繁。少了什么，你的客人可能会用你的嘴擦干他们的手。
