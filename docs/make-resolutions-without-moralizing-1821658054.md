# 下定决心，不要说教

> 原文：<https://lifehacker.com/make-resolutions-without-moralizing-1821658054>

“自我提升”是一个棘手的决议框架。我们认为这句话是理所当然的，但它真正在说什么呢？改变生活习惯会改善你自己？这意味着你的选择具有道德价值，给一些习惯贴上“好”的标签，给另一些贴上“坏”的标签。最终，你的生活方式选择会影响你作为一个人的内在价值。老实说，那很糟糕。



我们都有很多不理想的习惯。我们把盘子放在水槽里，我们不吃蔬菜，我们在手机上浪费时间。但是这些习惯是不好的，因为它们有不好的影响——厨房里有虫子，维生素缺乏，极度疲劳。同样，好习惯也有好结果:保持家里整洁可能会让你感到平静，或者锻炼有助于减轻你的背痛或焦虑。

但是这些习惯不具备道德价值。它们不能决定你作为一个人的价值。

因此，当我们进入新的一年，决心如火如荼的时候，这里有一个挑战:你能在不说教的情况下制定一个决心吗？想要改变自己而不相信改变前的自己是不好的，这可能吗？(如果你接受了它不会让你成为一个更好的人的事实，你还会坚持你的决心吗？)

不要把一个改变贴上从根本上来说是好的标签，而是要找出实际的好处。首先，这会激励你坚持改变，即使这很困难。但当你几乎不可避免地犯错时，它也会减轻打击。吃奇多的人在道德上并不比吃胡萝卜条的人差，他们只是过着不同的生活。

如果你想不出你想做的决定的实际好处，那就是一个信号，退一步想想你的决定对你是否真的有价值。在这里，减肥是一个常见的罪魁祸首——我们被训练得认为我们需要变瘦，以至于我们经常不去想*为什么*。(这意味着我们经常觉得自己应该更瘦，但……也许这并不重要。)而当我们对应该是中性的个人生活方式选择进行道德说教时，我们就把自我价值放在了砧板上。

坏习惯已经让你因为它们的负面影响而感觉糟糕——垃圾食品引起的胃痛，吸烟者的咳嗽，透支焦虑，不管是什么。你不必同时让他们对你的个人价值进行公投。
