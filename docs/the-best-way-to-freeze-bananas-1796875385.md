# 冷冻香蕉的最好方法

> 原文：<https://lifehacker.com/the-best-way-to-freeze-bananas-1796875385>

没有什么比冰沙中的冷冻香蕉更梦幻的了，对于准备做香蕉面包或松饼的面包师来说，没有什么比手边有冷冻香蕉更容易的了。但是互联网上充斥着关于如何冷冻这些可爱水果的糟糕建议。

Watch

坏建议分为两大阵营:

### 方法 1:将整个香蕉带皮冷冻。

**这个问题:**

1.你不容易剥一只冷冻的香蕉。为了剥开它，你需要或者用传统的方法解冻它(通过时间)；将其放入微波炉中，这增加了第二个步骤；或者 [把它从皮里面切片](http://skillet.lifehacker.com/its-not-impossible-to-peel-a-frozen-banana-1792145825) ，这个过程我觉得有点繁琐。

**2。没有什么比解冻的、没有皮的香蕉更恶心的了。解冻未去皮的香蕉会产生一种黏糊糊的米黄色液体，你必须想办法去除它，这又增加了一个步骤。(事实上我不是 [哈罗德·麦吉](http://curiouscook.typepad.com/site/about-harold-mcgee.html) 但我猜想这是因为果皮不像水果本身含有大量水分。)**

所以这种方法很糟糕。

### 方法 2:去皮，切成块，冷冻在饼干纸上。

这些说明通常是这样的:“剥香蕉，把它们切成块，把这些块——不要接触——放在羊皮纸覆盖的饼干纸上，然后把饼干纸放进冰箱。然后，当它们冷冻后，把大块放入冰箱的保鲜袋或保鲜盒中。

**这个问题:**

**1。**这是很多步骤。

**2。这需要大量的饼干纸和冷冻空间，尤其是如果你打算冷冻一整串香蕉。**

**3。**如果你要冷冻香蕉用于烘焙食品(而不是思慕雪),食谱通常需要许多整根香蕉，这意味着你必须将一串长度可能接近整根香蕉的大块排列起来，这又增加了一个步骤。好吧，这是一个小的诡辩。

**4。**根据我的经验，没有去皮的香蕉在冰箱里会粘在一起，很难分离，这意味着在你经历了在预冷冻的薄片上分离每一大块的麻烦后，当它们一起扔进 Ziploc 时，它们可能会凝结成一个固体块，你会试图用刀子或其他锋利的工具提取大块，这是一个可怕的想法。

### 解决方案

**1。**剥香蕉皮(如果你 [从“错误的”/无茎端](http://lifehacker.com/open-a-banana-like-a-monkey-5311002) 剥，你会得到干净得多的皮，带着更多的“线”和末端那讨厌的棕色部分)。

**2。**分别用保鲜膜、蜡纸或铝箔包好。

**3。**将包裹好的水果放入冰箱的保鲜袋或塑料储存容器中。

**4。**当你准备好喝奶昔时，你的香蕉也准备好了。

如果需要(比如说，你的搅拌机不是最强的)，你可以很容易地把香蕉掰成两半。对于烘焙食品，你仍然需要解冻香蕉，但是你不需要处理任何恶心的软泥。

注意:一些商店对难看的、过期的香蕉提供很大的折扣。抓住他们！这些普通的吸盘非常适合冷冻(一定要尽快把果皮拿出屋外——它们是果蝇的主要目标)。
