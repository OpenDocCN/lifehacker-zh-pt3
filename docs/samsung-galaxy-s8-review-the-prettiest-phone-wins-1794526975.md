# 三星 Galaxy S8 点评:最漂亮的手机胜出

> 原文：<https://gizmodo.com/samsung-galaxy-s8-review-the-prettiest-phone-wins-1794526975>

随着 Note 7 电池爆炸的惨败，三星无意中做了一件现在越来越难的事情:它让智能手机在一瞬间变得有趣。事实上非常有趣。除了 Note 7 引人入胜的悲伤传奇——其令人尴尬的召回让该公司损失了数十亿美元，三星还为 Galaxy S8 设定了一个戏剧性的发布叙事。这不仅仅是另一部智能手机——这是一部决定成败的设备，它肩负着拯救一家陷入生存危机的公司的重任。一部在虚空中尖叫的智能手机——呀！



由于赌注很高，三星推出了一款与众不同的设备。Galaxy S8 是一款足够好的手机，三星可能会将 Note 7 的戏剧抛在身后。(虽然，你知道，我们仍然在等待看到什么不幸的灾难将降临到 S8 现在。)最终，让 S8 成为赢家的并不是制造商在过去几年里一直大力推动的任何智能辅助功能——事实上，三星的参赛者 Bixby 还不够。S8 凭借行业领先的硬件设计赢得了胜利，挑战了 iPhone 的美学优势。

如果你购买 Galaxy S8，你是在购买它，因为它是目前最漂亮的手机。虽然它没有达到完全无边框，但也可能是这样。尽管 LG 在今年早些时候发布了一款类似的无边框设备，但它遭遇了的过度简化。三星远远领先于你实际考虑的大多数竞争对手，如 [谷歌像素](http://gizmodo.com/the-google-pixel-is-too-dumb-and-ugly-to-replace-your-i-1787877249) 和 [iPhone 7](http://gizmodo.com/iphone-7-review-ready-or-not-this-is-the-future-1786898943) ，它们都被包裹在难看的边框监狱中。

这一设计是三星的一次巨大飞跃，多年来，三星一直在凭借其 edge 显示设备朝着这个方向前进。这个东西实际上是所有显示器的正面，三星甚至摆脱了它的物理 home 键。这款手机柔和的圆形边缘和曲线创造了一种显示器的效果，它只是漂浮在空中或悬停在你的手上方。最小的设计没有不和谐的干扰。这感觉就像你在看一部来自未来的手机，而不是过去一个成功设计的老调重弹——不出所料，据报道，苹果公司正在考虑为其即将推出的 iPhone 设计类似的外观。

S8 有两种显示屏尺寸——5.8 英寸的 Galaxy S8(750 美元)和 6.2 英寸的 S8+(850 美元)。我测试了 S8+，但这两款设备的唯一区别是尺寸和价格。这些显示器的尺寸很难与其他设备相比。两款 S8 都拥有 18.5:9 的[长宽比](http://fieldguide.gizmodo.com/heres-why-the-displays-in-new-phones-are-so-weird-and-w-1793555119) ，与更传统的 16:9 外形形成对比。因此，S8+比 iPhone 7 Plus 略厚、略高，也略窄。撇开吹毛求疵不谈，它们的大小基本相同:

对于新的 18.5:9 宽高比的优点有一些争论(LG 今年也做了类似的事情)。你在网站、网飞、Hulu 等网站上看到的大部分内容都是 16:9 的，所以它占据了像 iPhone 这样的所有设备。同时，在 S8 上观看同样的内容，画面会被黑条包围。这感觉很浪费，但是，正如你在上图中看到的，这并不是世界末日。

与这些天你可能考虑购买的其他手机相比，S8 的有机发光二极管明显更明亮、更生动。您的照片将在此显示器上弹出。

除了硬件设计，S8 的另一个新功能是 Bixby，三星新收购的数字助理技术，旨在与谷歌助手、苹果的 Siri 和亚马逊的 Alexa 竞争。

在发布时，Bixby 将不支持语音命令，尽管三星承诺该功能将在今年春天登陆。我向自己保证，我不会因为三星发布未完成的产品而指责它(尽管上次三星这么做时发生了什么)。最终我不需要。大多数语音助手都笨得要命，包括也内置在 S8 中的谷歌助手，但 Bixby 的糟糕程度足以使添加语音支持甚至不会有什么不同。

Bixby 的许多功能乍一看似乎很神奇。就像三星内置在手机侧面的硬件按钮。它通过触摸而不是大喊来唤醒 Bixby。我最讨厌点名求助助手了。好吧，谷歌——告诉我我听起来像个混蛋！按下按钮，Bixby 的类似 Google Now 的仪表盘显示屏就会弹出，显示你的约会、天气以及来自三星 Health 和 CNN 等支持应用的信息。

实际上，额外的硬件按钮只会造成混乱。我经常按它而不是音量降低按钮，拉起助手而不是拍照或降低音乐的音量。更糟糕的是，一天早上，S8 的闹钟很早就把我吵醒了，我笨手笨脚地摸索着我的手机，我没有唤醒手机来关闭闹钟，而是按下了 Bixby 按钮，上面显示了我当天计划的所有会议，好像 Bixby 的唯一目的是提醒我，我醒着的每一个小时都将是辛苦和痛苦的。

该助手可能最令人印象深刻的功能是 Bixby Vision，它允许你使用手机的摄像头进行图像搜索。这种技术已经存在多年了——亚马逊在三年前 推出了一个令人印象深刻的功能，叫做 Flow [。所以这个功能差到令人难以置信。](https://www.bloomberg.com/news/articles/2016-09-18/samsung-crisis-began-in-rush-to-capitalize-on-uninspiring-iphone)

当你点击 S8 相机应用程序中的小 Bixby Vision 按钮时，该工具会花一点时间来处理你正在看的东西，然后为你提供使用图像进行一般搜索、购物搜索的选项，或者在它识别葡萄酒或文本的情况下，相应地处理视觉信息。

唯一能在任何一致性水平下工作的 Bixby 视觉工具是购物工具，它识别出一种怪物能量饮料(无糖！)和一本尼尔·德格拉斯·泰森的书没错。当我指向非常昂贵的耳挂式耳机时，它试图向我出售价格适中的耳塞，这很公平，我不需要花 900 美元买耳机。

更一般的图像搜索功能是一个笑话，感觉像一个失败。它唯一能够正确识别的图像是德国纪实摄影师奥古斯特·桑德的一张老照片。当然，这只是一个 Pinterest 搜索，但我不明白这到底是什么意思。

至于葡萄酒和文本工具——它们需要认真的工作。Bixby 唯一一次辨认酒瓶，我没有用它指着酒。文本工具的工作很辛苦，除了最干净的 Helvetica 字体，它往往会把一切都解释成乱码。

除了令人沮丧的助手，三星的软件实际上非常精致。TouchWiz 是该公司在 Android 上的专有产品，不再是多年来毫无吸引力的恐怖产品——主要是因为三星越来越远离谷歌。三星的一些软件非常有用，比如 Apps Edge 抽屉，从显示屏的右边缘轻扫一下，就可以滑出你最喜欢的应用程序的可定制集合

。

三星的相机对你的拍照来说足够好了。虽然没有你在其他设备上看到的双摄像头，但至少它内置了一些类似 Snapchat 镜头的滤镜——酷吗？

不太酷的是，三星将指纹扫描仪移到手机背面相机的侧面是一个错误。这并不是放在背面，这可能是必要的，使美丽的无边框的前面。这是因为传感器偏离了中心，所以它缺乏指纹传感器的直观感受，就像谷歌 Pixel 和 LG G6 一样。更重要的是，三星已经把这个奇怪的摄像头放在了你的大脑想要传感器的地方。Instagram 你的炸鸡晚餐风险自担。

最终，Galaxy S8 轻松击败 Android 竞争对手，成为你能买到的顶级手机。它比 Pixel 更漂亮，更光滑，尽管 Bixby 暂时感觉有点未完成，但你仍然可以使用谷歌助手。S8 也与 iPhone 7 相当，无论你选择哪一个，都可能归结为一个简单的问题，即你是想要三星的前瞻性想法还是苹果的精致简单。或者你可能想要一个耳机插孔——S8 就有一个！

至于你是否应该升级，那完全是另一个问题。你现在的手机有多烂？它坏了，你需要一个新的吗？S8 是一个很好的选择。你的手机用了不到两年，还能正常工作吗？等到明年吧。

S8 真正有趣的是，仅仅凭借其出色的设计，一部手机就能如此引人注目。Bixby 基本上是垃圾，然而手机是赢家。或许这并不令人惊讶:如今，配置强劲的顶级旗舰手机将会表现出色，不会出现故障。它们都有很好的摄像头和一整天的电池寿命，以及可以处理大型手机游戏的处理器。让一款设备脱颖而出是如此具有挑战性，以至于制造商们正在使用各种各样的噱头，比如语音助手和模块化配件，来看看他们是否能让他们的设备脱颖而出。这些技术中没有一项能像一项迎合我们最原始的肤浅的设计那样改善我们的生活；我整天盯着我的手机——我想看些漂亮的东西。
