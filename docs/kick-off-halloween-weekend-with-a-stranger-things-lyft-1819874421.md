# 今年万圣节，你可以乘坐 Lyft 去颠倒一下

> 原文：<https://lifehacker.com/kick-off-halloween-weekend-with-a-stranger-things-lyft-1819874421>

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-1vycaHQgYOs&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-1vycaHQgYOs&start=0) 

如果你在本周五和周六在洛杉矶或费城，你可以通过一个*陌生事物*主题的 Lyft 开始你的万圣节周末。



上面的视频告诉你会发生什么，如果你是一个*陌生事物*的粉丝(或者只是想恶作剧你的朋友)，你可以在“陌生模式”下请求你的 Lyft 这些游乐设施在下午 4-9 点开放，尽管从 [Lyft 的网站](https://blog.lyft.com/posts/lyft-x-stranger-things) 还不清楚你是否能保证得到一辆*陌生的东西* Lyft，或者是否只有一些提出要求的骑手能得到一辆。

如果你已经计划在这个促销期间在洛杉矶或费城乘坐 Lyft，那么尝试一下奇怪的模式可能是值得的，以融入万圣节的精神，并增强你对新一季的周末狂欢观看， [明天将在网飞](https://www.avclub.com/stranger-things-dodges-the-sophomore-slump-by-cribbing-1819771382) 上映。他们甚至在主页上安装了倒计时钟，以免你忘记你的下一个预约:

如果你不在洛杉矶或费城，你至少应该能够将你的 Lyft 应用程序切换到以*陌生事物*为主题，汽车显示为包裹在圣诞灯中，以及展览中的其他参考。
