# 为拉通磨刀器辩护

> 原文：<https://lifehacker.com/in-defense-of-pull-through-knife-sharpeners-1797860631>

一把锋利的刀可以打造一个安全的厨房，但是外面有太多相互冲突的思想流派，学习如何磨自己的刀会让人不知所措。我的哲学是:我是厨师，不是铁匠。我需要一种快速、可靠的磨刀方法，因为我在五金店磨刀时没有备用刀可用，而且我不在乎花几十年时间来掌握磨刀的艺术。因为这个原因，我很喜欢拉式磨刀器。如果你也是，我在这里验证你。

Watch

一般来说，手持磨刀器的工作原理是使用两块呈“V”字形的碳化钨来磨掉刀片两侧的金属，形成新的刀刃。根据你的观点，这要么是一个伟大的方式来恢复迟钝的刀片，要么是一个绝对令人厌恶的。"但是这个*角*！"刀子势利者哭泣，珍珠咯咯地。“还有那些手持磨刀器去掉*那么*多的金属！你会缩短你的刀的寿命！”嗯，你通过去除一些金属来创造一个优势，不是吗？手持磨刀器只是比石头更快更不精确。至于“缩短你的刀的寿命”这种论调，我真的不知道该怎么说。几乎不可能真正毁掉一把刀，因为刀是由金属制成的。偶尔磨一磨不会让金属变得更脆，也不会让刀刃磨损成一个可悲的、无用的小块，即使你非常非常强壮。

不过，安格斯的观点有一定的道理。V 形磨刀器只能按照自己的形象重新打造刀刃，因此它们在以类似角度倾斜的刀上效果最好。我用的 [AccuSharp](https://accusharp.com/) 一个 20 度左右的角度，挺标准的；因为我用的是相当标准的刀——T4[奇异果](http://skillet.lifehacker.com/the-top-5-restaurant-supply-store-essentials-ranked-1797727937)T7】和威斯霍夫厨师刀，外加一把挑战者面包刀——对我来说效果很好，我觉得很舒服。当然，它不能像专业的罐子一样在我的手腕上涂上漂亮的镜面抛光漆，但它能在几分钟内让刀刃变得非常锋利。

如果你是一个拥有一千美元碳钢刀的时髦小伙子，我显然不建议你使用 AccuSharp。你为这种优势付了很多钱，这意味着你可以继续花很多钱让别人为你维护它。但是对于我们这些只需要锋利东西的人来说，手持磨刀器可以又好又快地完成工作。记住茱莉亚·切尔德的不朽名言:“让你的刀保持锋利，最重要的是，玩得开心。“无论哪种磨刀方法能让你在厨房里玩得开心，我都鼓励你坚持下去，不要有负罪感。
