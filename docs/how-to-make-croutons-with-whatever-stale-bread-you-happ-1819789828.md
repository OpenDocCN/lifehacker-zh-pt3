# 如何用不新鲜的面包制作油炸面包丁

一份好的沙拉就是要达到味道和质地的完美平衡，没有什么比油炸面包丁更能增加口感了。你可以很容易买到预制的面包块，但是如果你想要一个真正的金色、酥脆、耐嚼的面包，你应该自己做。它们也非常容易制作。

除了给你一个更美味的浇头，自己制作油炸面包丁也是一个很好的方式来用完你在可怕的老化之前没有消耗的最后一点华丽的硬皮面包。整个过程非常简单，你甚至不需要食谱。简单地拿起你的面包——在我的例子中，是一条切达哈巴内罗面包——把它切成或撕成面包丁大小的块或方块。

立方体的所有表面都将更加松脆，手撕的油炸面包丁将有各种各样可爱的质地。两者都会很好地为你服务。

一旦你选好形状，把面包切成小块，把它们放在碗里，淋上橄榄油。面包不应该被油浸泡，但它应该明显地被油弄湿。我通常会淋一点进去，把面包扔来扔去，然后再淋一点，重复这个动作，直到每块面包都有油一样的光泽。

然后就是调味的时候了。盐和胡椒是显而易见的选择，但你可以加入大蒜粉、洋葱粉、辣椒粉、帕尔马干酪、印度香料、营养酵母和新鲜香草。只要洒上它，把面包扔来扔去，品尝一个“生的”面包丁，并根据需要调整你的香料。

在烤盘上撒上一层面包丁，在 375 度的烤箱中烤至金黄酥脆，大约需要 15 分钟。让它们冷却，在密封的容器中保存一周。

用你的脆皮作品给汤或沙拉增加质感，或者简单地把它们作为松脆的碳水化合物点心单独享用。
