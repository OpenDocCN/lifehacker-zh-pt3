# 用一种配料改善任何一道菜

> 原文：<https://lifehacker.com/improve-any-dish-with-one-ingredient-1795937477>

美食作家兼食客的自由编辑海伦·罗斯纳有整个烹饪界最简单的秘方提示

Watch [https://lifehacker.com/embed/inset/iframe?id=twitter-869169573538529283&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-869169573538529283&autosize=1) 

用一种新的配料改造一道菜感觉很神奇，就像你是一只*炖菜*老鼠 [大嚼一大口奶酪和草莓](https://www.youtube.com/watch?v=Yyah49_Oz78) 。它激励一些人随身携带 [辣酱](https://www.washingtonpost.com/lifestyle/food/how-did-hot-sauce-get-in-so-many-african-americans-bags-anyway/2016/04/21/63bebca0-07d2-11e6-b283-e79d81c63c1b_story.html) 或一种 [专有盐拌](http://observer.com/2013/08/werner-herzogs-kickstarter-video-is-the-best-thing-youll-watch-all-day/) 。对于我们这些在中西部平淡无味的环境中长大的人来说，这尤其具有启发性。

其他通用附加组件包括:

*   咸味菜:辣酱或辣椒粉加热； [鱼露](http://www.seriouseats.com/recipes/2013/03/steak-salad-with-cucumber-peppers-and-spicy-f.html) 为咸味和鲜味；味精(如果你同意 [它的安全](https://fivethirtyeight.com/features/how-msg-got-a-bad-rap-flawed-science-and-xenophobia/) )为鲜味
*   酱料: [撮糖](http://www.epicurious.com/ingredients/sugar-tomato-sauce-secret-article) 以平衡酸味
*   米饭菜肴:芝麻油增加味道
*   墨西哥/南美菜肴:酸橙(作为柠檬的气候匹配替代品)
*   素食菜肴: [营养酵母](http://blog.fatfreevegan.com/2011/10/what-the-heck-is-nutritional-yeast.html) 为营养和鲜味；烟熏辣椒粉带来培根般的烟熏味

但最重要的是，盐和柠檬。在烹饪书*盐脂肪酸热量*中，Samin Nosrat 写道，“盐比其他任何成分对味道的影响都大。”她谈到一位厨师在诺斯拉特的玉米粥里加了满满三手掌的盐。她吓坏了，但她尝了尝:“玉米不知何故更甜了，黄油更丰富了。所有的味道都更加明显...无论我怎么尝试，*咸*这个词都不适用。”

诺斯拉特偏爱的酸是醋；她有一个类似的故事，在胡萝卜汤里加醋，担心它会变成“令人厌恶的糖醋”，但发现它带出“胡萝卜中的黄油和油、洋葱和高汤，甚至糖和矿物质”的味道。尽管如此，她还是为鲜榨柠檬、酸橙或橙子中的“最后一秒钟的酸”腾出了空间。“挥发性芳香分子会随着时间的推移而消散，”这就是为什么餐馆主菜中的柠檬会放在盘子里让你自己榨。

罗斯纳推荐在 [中加入新鲜柠檬，她出色的汤食谱叫做“罗伯托”](https://tinyletter.com/hels/letters/issue-1-hello-roberto) ，这是为没有经验的厨师准备的，其中有这样的描述:“[香肠]应该看起来有黑点，像豹子或可爱的狗。”
