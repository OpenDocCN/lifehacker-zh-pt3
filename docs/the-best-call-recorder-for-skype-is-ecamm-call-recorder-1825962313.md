# 最适合 Skype 的通话记录器是 Ecamm 通话记录器

> 原文:[https://life hacker . com/the-best-call-recorder-for-Skype-is-ecamm-call-recorder-1825962313](https://lifehacker.com/the-best-call-recorder-for-skype-is-ecamm-call-recorder-1825962313)

我是一名记者。(算是吧。)我得经常录电话，以补充我的笔记。当我通过 Skype 打电话时，我使用的是 [Ecamm 通话记录器](http://www.ecamm.com/mac/callrecorder/) ，这是一个简单但功能齐全的 Mac 专用 Skype 扩展。

Watch

当 Skype 打开时，Call Recorder(相当于“乐队”的应用程序名称)会自动打开，这很重要，因为否则我会忘记开始录音。更好的是，你可以告诉电话录音开始记录每次你打电话。(为了避免文件混乱，应用程序可以丢弃短呼叫。)

它对于录制重要的对话、采访或播客非常有用。它的价格高达 39.95 美元，但如果你经常记录 Skype 对话，它是值得的。该许可证可在多种设备上使用，并包括所有未来的应用程序更新。通话记录器也可用于 [FaceTime](http://www.ecamm.com/mac/callrecorderft/) ，同样售价 39.95 美元。

通话记录不会提醒任何人它的存在，所以你可以记录而不被其他人注意到。但是让他们知道是礼貌和道德的。如果你计划秘密地记录电话，查一下当地的法律，这样你就知道你在做什么。

录制是原始应用程序应该具备的基本功能之一；也许 Skype 只是不想让人们担心自己被录音。好像现在还有人期待隐私似的。
