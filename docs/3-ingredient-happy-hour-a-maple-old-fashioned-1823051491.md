# 三种成分的快乐时光:老式枫树

> 原文：<https://lifehacker.com/3-ingredient-happy-hour-a-maple-old-fashioned-1823051491>

各位周末快乐，欢迎回到 [*三料欢乐时光*](http://lifehacker.com/tag/3-ingredient-happy-hour) ，每周饮品专栏，提供超级简单又美味的酒水。这个星期因为各种原因有点忙乱，所以我觉得需要一些疯狂精简的东西，即使以我们的标准来看。我想倒，搅拌，啜饮，枫糖浆会帮我达到目的。

Watch

除非你在威斯康星州(那里对鸡尾酒有非常独特的看法)，否则传统鸡尾酒通常由方糖、少量苦味酒和几盎司你最喜欢的波旁威士忌(或黑麦威士忌)制成。我们将使用一小块枫糖浆，而不是白糖块——反正我也没有。如果你仔细想想，枫糖浆是一种现成的鸡尾酒甜味剂，完全不需要你做任何准备工作或煮沸，比蔗糖做的东西更有深度。

基本上，你不用任何额外的成分或努力就能得到更多的味道，这正是我目前在酒精饮料中需要的。要做到这一点，你需要:

*   2 盎司波旁威士忌
*   1 茶匙枫糖浆(我喜欢宣称它有“强烈的味道”——[原名 B 级](https://skillet.lifehacker.com/a-guide-to-usda-maple-syrup-grades-1802726151) )
*   3-4 滴安古斯图拉苦味酒

将所有材料加入低杯，搅拌直到糖浆完全融入威士忌。加入一大块冰块，继续搅拌，直到你的饮料完全冷却。喝了它，放松到周末。
