# Aincrad 桌面

> 原文：<https://lifehacker.com/the-aincrad-desktop-1732684767>

看，你可以爱或恨骚，但这个桌面很好看，而且来自频繁的 Jalop 评论者， [jester74](http://kinja.com/jester74) 。这是一个好看的 Linux 桌面，而且设置简单。



Jester74 不久前在我们的 [**桌面展示**](http://kinja.com/tag/desktop-showcase/) 中留下了这个，但我们现在才开始。看起来还是很棒。下面是在您的 Linux 系统上设置它所需的内容:

*   克里斯·莱伊曼创作的 [安克拉德壁纸](http://www.brokentone.net/wall/158-aincrad/)
*   [Conky](http://conky.sourceforge.net/) ，一款针对 Linux 的轻量级系统监视器
*   [哈玛特](http://zagortenay333.deviantart.com/art/Conky-Harmattan-426662366) ，Conky 的定制工具和主题，在桌面上透明
*   [板](https://launchpad.net/plank) 作为应用程序启动器停靠(屏幕底部)
*   [numix 圆形图标包](https://numixproject.org/) 替换系统图标并自定义 dock

大概就是这样。简单，好看，一切都很好地配合在一起。听着，我对骚也有看法，但是如果我能超越他们去欣赏桌面，你也能。如果你正在定制你自己的，有问题，需要帮助让一切看起来正确，或者只是想告诉 jester74 你有多喜欢他的作品，请在他的 Kinja 帖子上留下评论，链接如下！
