# 2018 年最好的转售名单基本上都是卡车和拉力赛车

> 原文：<https://jalopnik.com/the-best-resale-list-for-2018-is-basically-all-trucks-a-1822584771>

谈到汽车的总成本，售价只是其中的一部分。你需要考虑转售，以及你的车会损失多少价值。有些下降速度比其他人慢，根据 [*凯利蓝皮书*](https://www.kbb.com/new-cars/best-resale-value-awards/best-resale-top-10-cars/) *，*这些 2018 款卡车、SUV 和全轮驱动轿车将是你最大限度减少折旧的最佳选择。

这份名单的大部分不会令人惊讶，因为丰田的 4x4s 排在第三位。任何试图在二手市场购买 Tacoma、Tundra 或 4Runner 的人都会告诉你一些二手价格有多疯狂。

令人好奇的是，为什么 Tundra 的排名高于 4Runner，尽管它们有相同的 60 个月剩余，但 4Runner 保留了其转售的 65.8%，而 Tundra 略低于 64.7%。我们联系了凯利蓝皮书来澄清这些排名。

同样有趣的是，Yotas 甚至比传说中狂热的牧马人转售还要持久。这不仅仅是吉普车的事。

至于其他选择，雪佛兰和福特的大卡车一直是二手车买家的需求。相比之下，清单中遗漏了 Ram 的半吨产品。与雪佛兰/GMC 1500 系列和 F-150 车型相比，您会发现 Ram 卡车的二手车价格存在相当大的差距。

本田山脊线是一个很好的惊喜，尽管不是一个“真正的卡车”的一些人。由于本田汽车在耐用性方面的声誉，它可能会保持价值。当然，我们有 WRX，这是唯一一款进入榜单的车，但它坚持高质量的 AWD/四轮驱动汽车如何保持价值的总主题。
