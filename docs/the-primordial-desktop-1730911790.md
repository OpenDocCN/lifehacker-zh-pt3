# 原始桌面

> 原文：<https://lifehacker.com/the-primordial-desktop-1730911790>

麦肯齐 的桌面是一个美丽的树木繁茂的休养地，只要看着它就能降低你的压力水平——这使它成为工作桌面或第二屏幕的完美选择。下面是如何设置它。



这是一个 Windows 桌面(从外观上看并不是非常明显)，但幸运的是，你不需要太多就能让整个系统工作。如果您愿意，您可以单独使用壁纸，也可以添加一层雨量计表皮，让整体效果更佳，一切由您决定。

*   [4K 森林壁纸](http://wallpics4k.com/new-4k-wallpapers-wallpics4k-nature/)
*   用于 Windows 的 [雨量计](http://rainmeter.net/) 系统调试和监控工具
*   雨量计的 [Wisp 主题](http://fiizzion.deviantart.com/art/Wisp-v2-4-2-for-Rainmeter-385708923)用于天气预报和时间/日期显示

这一次就这么多了。显示或隐藏任务栏(它隐藏在这里)，无论你喜欢做什么。简单，赏心悦目，令人放松。有时候，正如他们所说，少即是多。

如果你喜欢你看到的，去麦肯齐的 flickr 页面(链接如下)让他们知道你喜欢成品的样子，或者问你关于它是如何设置的问题。
