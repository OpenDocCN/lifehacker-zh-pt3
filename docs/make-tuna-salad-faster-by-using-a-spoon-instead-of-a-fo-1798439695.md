# 用勺子代替叉子做金枪鱼沙拉会更快

> 原文:[https://life hacker . com/make-tuna-salad-fast-by-use-a-spoon-to-a-fo-1798439695](https://lifehacker.com/make-tuna-salad-faster-by-using-a-spoon-instead-of-a-fo-1798439695)

我对金枪鱼沙拉的偏好非常简单:大黄蜂水中的纯白长鳍金枪鱼(没有“大块光”，没有油，没有星芒)，每罐一汤匙蛋黄酱(Hellmann's)，仅此而已。我偶尔会疯狂地添加刺山柑，或者一点儿 [凤尾鱼酱来提鲜味](http://lifehacker.com/make-better-tuna-salad-with-these-two-umami-boosting-in-1782377745) ，但绝对不要洋葱或芹菜。我确信你自己的金枪鱼口味与我的大相径庭，这是理所应当的——当你处理像鱼罐头和蛋黄酱这样有争议的食物时，你会遇到很多强烈的意见。但是大多数金枪鱼沙拉爱好者都同意的一件事是你捣碎配料的工具——你用叉子，对吗？我想建议一个更快的选择。

让我们明确一点:首先，做金枪鱼沙拉并不是一件特别耗时的事情。把金枪鱼沥干，用蛋黄酱捣碎，不要弄乱。金枪鱼三明治、三明治和砂锅菜之所以如此受欢迎，正是因为它们方便快捷。但作为所有经验丰富的金枪鱼制造商(调谐器？突尼斯人？)知道，你必须花一点时间和努力来确保金枪鱼被充分打碎，金枪鱼和蛋黄酱被充分混合。

大多数人用叉子捣碎金枪鱼。我过去常常用叉子捣碎金枪鱼。然而，根据罐头中长鳍金枪鱼块的大小和稠度，较大的块可能不易粉碎。你认为你想用叉子，因为叉子似乎比勺子更具破坏性。但是汤匙的背面比叉子尖的背面有更大的表面积，根据我的经验，用汤匙切碎金枪鱼可以节省 60 秒的金枪鱼制作时间——你的努力只会得到更多的粉碎。如果你用和舀蛋黄酱时一样的勺子捣碎，那就少了一个脏器皿。

如果有人想弄脏两个餐具，理论上可以用叉子作为附属品。一旦罐装新鲜金枪鱼块大部分被勺子分解，用叉子耙片可能会很有趣。然而，人们总是冒着金枪鱼被叉子尖夹住的风险，把它拿掉可能会让你节省时间的努力前功尽弃。

**注:**有人用搅拌机或食品加工机做金枪鱼。这肯定会创造出非常快速的糊状金枪鱼产品，但当你考虑到清理以及意外将金枪鱼过度搅拌成鱼奶昔的风险时，勺子会胜出。
