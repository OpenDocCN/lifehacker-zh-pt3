# 在 PC 或 PS4 上设置虚拟现实需要什么

> 原文:[https://life hacker . com/what-you-need-to-set-up-virtual-reality-on-a-PC-or-ps4-1821648526](https://lifehacker.com/what-you-need-to-set-up-virtual-reality-on-a-pc-or-ps4-1821648526)

得到一个虚拟现实耳机可能会成为一份令人印象深刻的圣诞礼物，但除非你有足够的计算能力来使用它，否则就把它当成一个非常昂贵的镇纸。根据为耳机供电的系统，无论是 PC 还是 PS4，以下是如何为虚拟现实做准备的方法。

### **弄了个 PS4？购买 PlayStation VR 套装**

每个 PlayStation 4 都支持虚拟现实，PlayStation 4 Pro 由于其更强大的内部组件，提供了略好的视觉体验。当然，他们只通过索尼自己的 PlayStation VR 头显支持虚拟现实。然而，你需要购买正确的版本，否则你会错过一个非常关键的部分。

如果你有任何 PlayStation 4 游戏机，无论是最初的 PlayStation 4，其更新，更薄的版本，还是更强大的 PlayStation 4 Pro，你都可以使用 PlayStation VR 耳机。你还需要一个额外的硬件组件，PlayStation 摄像头，来跟踪耳机和你的动作。

如果你想一次获得所需的一切，请购买 PlayStation VR 套装，其中包括 PlayStation 摄像头以及其余的 VR 耳机组件。某些捆绑包还包括 PlayStation Move 运动控制器，可以取代传统的 DualShock 4 控制器。

### 有电脑吗？你可能需要新的硬件

与 PlayStation VR 不同，在 PC 上使用虚拟现实可能需要的不仅仅是购买一套耳机。你的 PC 应该满足你的虚拟现实耳机的最低规格，无论是一个 [Oculus Rift](https://gizmodo.com/palmer-luckey-millionaire-founder-of-oculus-rift-love-1786994094) 还是 HTC Vive。你可以 [在你的电脑](https://lifehacker.com/see-if-your-pc-is-vr-ready-with-the-oculus-rift-compati-1751429543) 上运行基准测试，看看你的电脑是否为虚拟现实做好了准备。如果您不想进行任何升级，下面列出的每款耳机的推荐规格可能会超出您的预期:

[**Oculus Rift 推荐要求**](https://support.oculus.com/170128916778795/)T5 

**显卡**:英伟达 GTX 1060 / AMD 镭龙 RX 480 以上

**替代显卡**:英伟达 GTX 970 / AMD 镭龙 R9 290 以上

**CPU** :英特尔 i5-4590 / AMD 锐龙 5 1500X 以上

**内存** : 8GB+ RAM

**视频输出**:兼容 HDMI 1.3 视频输出

**USB 端口**:3 个 USB 3.0 端口，外加 1 个 USB 2.0 端口

**操作系统** : Windows 7 SP1 64 位或更新版本

[**HTC Vive 推荐要求**](https://www.vive.com/us/ready/)

**显卡**:NVIDIA GeForce GTX 1060 或 AMD 镭龙 RX 480，同等或更好

**CPU** :英特尔酷睿 i5-4590 或 AMD FX 8350，同等或更好

**内存** : 4 GB 以上内存

**视频输出** : 1x HDMI 1.4 端口，或 DisplayPort 1.2 或更新版本

**USB 端口**:1 个 USB 2.0 端口或更新版本

**OS** : Windows 7 SP1 或更高版本

这意味着你可能不得不从你的电脑中更换一些部件， [无论是它的内存还是一个新的显卡](https://lifehacker.com/how-to-speed-up-your-old-pc-1821481732) 。只要 [你能舒服地从你的电脑中取出零件](https://lifehacker.com/how-to-build-a-computer-lesson-3-building-the-compute-5827491) ，就去做吧。如果没有，你可以咬紧牙关，用专门为 VR 头戴设备出售的支持 VR 的机器来替换你的旧 PC。你可以从 Oculus 自己的商店购买标有“ [Oculus Ready](https://www.oculus.com/oculus-ready-pcs/#pc-offers) 的电脑，或者在百思买、亚马逊或 [上寻找“VR ready”电脑，无论你在哪里购物](https://www.vive.com/us/ready/) 。你可以预计支付 700 美元左右的低端电脑可以处理虚拟现实，所以也许可以考虑先升级自己的电脑。

一旦你有了合适的硬件，你还需要一样东西:实际空间。遵循 [HTC Vive](https://www.youtube.com/watch?time_continue=1&v=rv6nVPPDmEI) 或 [Oculus Rift](https://www.youtube.com/watch?v=EiY2xYHzcus&list=PLL2xVXGs1SP5Bdvdz7qYAXlveqGteoSiX) 附带的跟踪相机的定位说明，确保你不会将手臂伸向任何危险或尖锐的物体，并尽情享受。
