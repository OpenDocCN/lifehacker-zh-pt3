# 这个机器人会自动喜欢你朋友的 Instagram 帖子

> 原文：<https://lifehacker.com/this-bot-auto-likes-your-friends-instagram-posts-1821848732>

我白天会发一些微博，我的男朋友几乎总是第一个给我发赞，有时在我发消息的第二秒，赞的通知就出现了。



他很快，在他快速点赞的第一个月左右后，我记得我问他是不是在用一个机器人盲目地喜欢我的帖子。原来，他实际上已经设置了一个通知，所以当我发布新消息时，他会在桌面上弹出一个窗口。他把我的推文当作一个短暂休息的机会，点击通知，看看我写了什么，在大多数情况下喜欢它。

虽然他没有使用机器人，但我本周发现，Instagram 上确实有一个机器人。

有了它，你可以设置自动喜欢某个用户的帖子。当类似的东西发出后，你会在 Slack 中收到一条消息，让你知道发生了什么，这样你就可以自己查看帖子了。而且你绝对应该尽快看那个帖子，而不是让别人认为你看到了(并且喜欢上了)你没有看到的东西。这里的想法是在你非常忙的时候避免“你为什么不喜欢 X”的谈话，而不是完全把你自己从等式中删除。

这个机器人可以在 Github 上免费下载。它的创造者 gulzar 1996 创造了它，作为一种自动喜欢他女朋友的帖子的方式。目前，它只适用于 Instagram，但他计划在未来添加 Twitter 支持和一层面部识别功能，因此你可能只会喜欢包括你朋友(或其他重要人物)面部的图片，而不是他们时间线中的所有内容。

另一个不错的选择是:当你的朋友发帖时，设置通知。你可以点击 Instagram 上他或她的个人资料，点击页面右上角的三个点。在那里，选择“打开帖子通知”，每次有新帖子进来时，Instagram 应用程序都会发送推送通知。那你就真的可以喜欢了。
