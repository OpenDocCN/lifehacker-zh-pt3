# 使用这些搜索技巧来控制您的 Gmail 收件箱

Gmail 是我生活中必不可少的东西之一，但这也让我的生活变得更加烦人。几年前，我放弃了 [收件箱零](https://lifehacker.com/how-i-went-from-1-000-emails-to-inbox-zero-and-stayed-5984417) (此时我对收件箱 1000 封感到满意)，每天我都会收到几十封无用的垃圾邮件，这让我更难找到任何不到一周的重要邮件。

如果你像我一样，但你想更好地管理你的 Gmail 账户，有一种方法可以做到这一点，而不必承担彻底清理收件箱的艰巨任务。相反，你可以使用一些聪明的 Gmail 搜索技巧(由 [科普](https://www.popsci.com/advanced-gmail-searches) 提供)来浏览垃圾邮件，甚至可能清理一些空闲空间。

### 查找旧邮件的技巧

如果你无法找到一封旧邮件，有一些搜索技巧供你使用。这完全取决于你在寻找什么样的信息。

需要找到一封您从未真正打开过的重要邮件？将“是:重要的是:未读”添加到您的搜索中，根据发件人和邮件内容找出谷歌认为对您重要的未读邮件。

在找很古老的东西吗？使用搜索词“之前:2016/12/31”(或者任何你想要的日期)。如果你没有具体的截止日期，你也可以搜索“之前:2017”或另一年。

最后，如果你正在寻找通过 Hangouts(又名 Gchat)发送的消息，只需在搜索中添加“in:chats”。这将缩小结果范围，并排除任何相关的电子邮件。

### 节省存储空间的技巧

如果你遇到了 Gmail 慷慨的存储分配，你也可以使用一些聪明的搜索来释放更多的空间。最快的解决方法是搜索“有:附件大于 10mb”这将调出任何带有大附件的电子邮件，这样你就可以删除那些你不需要的。

你可以通过搜索“抄送:我”或“密件抄送:我”(或者两者都用“抄送:我”或“密件抄送:我”)，然后删除你不想要的邮件，从而删除那些不重要的邮件。你也可以搜索一个特定的人/公司，用“发件人:*电子邮件地址*”堵住你的收件箱或者尝试使用“文件名:jpg”或“文件名:gif”或“文件名:pdf”等查询来搜索特定的文件类型，然后从收件箱中删除这些文件。

如果你想避免未来存储空间不足，你可以使用 Gmail 搜索一次取消订阅一堆邮件列表。只要搜索“label:^unsub”就能找到所有带有退订链接的邮件，然后发疯吧。

### 保持条理的诀窍

这里还有一个 Gmail 搜索技巧，可以让你更好地控制收件箱。如果你是那种喜欢用标签来组织收件箱的人，你可以搜索“has:nouserlabels”来找到任何逃过你系统的邮件。从那里，删除它们或者根据需要进行分类
