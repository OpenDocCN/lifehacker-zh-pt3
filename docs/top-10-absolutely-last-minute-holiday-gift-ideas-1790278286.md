# 十大绝对紧急的节日礼物创意

> 原文：<https://lifehacker.com/top-10-absolutely-last-minute-holiday-gift-ideas-1790278286>

剩下的时间不多了——事实上是几个小时。也许你忘记了什么，或者你的清单上有一个惊喜礼物是你几天前不知道的。不管怎样，以下是如何在今年从你的帽子里变出一只兔子，送给他们一些好东西。

Watch

### 10.给予恩惠而不是物品

当你还是个孩子的时候，你可能会记得这个——如果你没有礼物可以送，你总是可以送一个“优惠券”，比如洗碗、倒垃圾或打扫房间——这些事情你可能无论如何都应该做，对吧？嗯，这些东西也是很好的最后一分钟礼物，甚至对那些众所周知很难送给 的人有用。

无论是帮助他们取下所有节日装饰品的礼物，还是在新的一年里帮助他们整理房子或在他们需要时帮助他们搬家的礼物，一个帮助都可以走得很远，有时甚至更远，这是其他礼品卡或手机充电器所不能做到的。

### 9.给他们一次经历，或者计划一次

说到无形的礼物，不要忽视经历和记忆的礼物。最终，它们比物质更重要，虽然如果你现金不足，它们很有用，但在最后一刻，当所有的商店都关门了，这是平安夜，它们也很有用。

一张手工制作的卡片，承诺你可以愉快地教授一些课程，或者承诺一起做一些很棒的事情，比如参加烹饪班，进行一次很棒的公路旅行，音乐会门票或歌剧之夜，或者其他一些对你们俩都很重要的活动或经历，都是一份不可思议的礼物。如果你想出点子有困难，可以考虑之前提到的之类的服务，它们可以帮你想出好点子。

### 8.去杂货店买包装好的礼物(或者自己组装篮子)

或者，如果所有的商店都没有关门，有一个地方你可以去，那里充满了最后一刻的礼物创意——如果你知道如何包装它们的话。 [杂货店](https://lifehacker.com/four-halfway-decent-last-minute-gifts-you-can-grab-at-t-1749488549) 。

从为邻居或同事准备的简单水果篮到美味的包装糖果和其他好吃的东西，杂货店有很多食物可供选择，可能只需要稍微打扮一下，贴上新鲜的礼物标签，你就可以走了——此外，如果你发现一些美味的东西，你知道它会被欣赏和享受，即使它是一个快速的礼物。当然，礼品卡在杂货店也很容易买到，所以这也值得考虑(但我们稍后会谈到这些——以及它们的数字选项)。

### 7.赠送精美的食物或咖啡

当你在杂货店的时候，你可以买一些美食和新奇的东西，比如货架上最奇特的咖啡，或者一些超级限量版的糖果——这些都是很好的选择。你也可以做得一样好，甚至在最后一刻赠送咖啡订阅服务，就像前面提到的 [MistoBox](http://www.mistobox.com/) 或 [Boxo Coffee](https://boxo.coffee/) 提供的服务一样，我已经试过了，很喜欢。

如果订阅一杯咖啡不是你的风格， [有适合各种人、喜欢和口味的订阅礼物](https://lifehacker.com/the-best-subscription-gifts-you-can-give-1469248862) 。你有很多选择，从有趣的小主题盒子到奶酪、葡萄酒或肉类的常规运输。如果你还不确定， [Cratejoy](https://www.cratejoy.com/) 是一项服务，旨在 [帮助你找到合适的](https://lifehacker.com/cratejoy-finds-the-perfect-subscription-box-for-your-in-1754737677) 。然后以你朋友的名义注册并支付订阅费，把打印件或解释订阅内容的卡片交给他们。然后他们会一整年都喜欢你的礼物，看着你——你在送礼物的前一天晚上在网上买的。

### 6.赠送博物馆会员资格

如果你的礼物不是吃喝型的，而是文化型的，周末外出型的，考虑送博物馆会员的礼物。即使在博物馆基本免费的地方(比如我的老镇华盛顿 DC，) [订阅可以让你享受到一系列的好处](https://lifehacker.com/the-best-benefits-of-museums-you-might-not-have-known-a-1585311185) ，包括特别活动、放映、签名售书、深夜巡演、社交活动和其他其他人要么从未听说过，要么不得不为获得门票而竞争的精彩活动。

此外，考虑到博物馆会员资格通常意味着免费或低价入场、特殊礼物和博物馆商店折扣，你的礼物不仅会支持你购买它的博物馆，还会全年不断地送给你的朋友或心爱的人。只要确保他们使用它！

### 5.赠送人人喜爱的优质网络应用和服务

由于数字礼物肯定是最好的最后礼物，也许你应该考虑赠送他们已经使用的服务的高级或付费版本，而不是选择稍微超出范围的礼物。也许你的名单上有一个 Pandora 或 Spotify 的粉丝，但他们从未想过要支付订阅费。也许你的名单上有一个网飞迷，他真的希望他们能看更多的老电视节目，而 Hulu 正好符合他们的胃口。无论哪种方式， [我们在这里](https://lifehacker.com/give-the-gift-of-premium-apps-and-web-services-1788463516) 都有一个很棒的优质网络应用和订阅服务列表，它们中的任何一个或所有都可以成为很好的礼物——最棒的是，你可以立即在网上购买它们，并让你的礼物订阅以电子方式交付。甚至不需要买卡。

### 4.以他们的名义进行慈善捐赠

这可能不适合每个人，但你可以给一个很好的礼物——特别是给那些拥有一切的人——以他们的名义进行慈善捐赠，最好是给他们热爱和支持的组织，或者无论如何都会支持的组织。毕竟，现在有很多非常值得的慈善机构和团体需要帮助 ，以你朋友或爱人的名义送礼物真的意义重大。

当然，这也意味着传单、地址标签和更多的邮件，但只要你在捐赠前做好慈善的功课，你就可以很容易地确保你的捐款将真正用于该组织的使命，并帮助它打算帮助的人。如果你需要更多的建议，我们这里有 [一些专门以节日为主题的](https://lifehacker.com/the-best-charities-to-donate-to-this-holiday-season-1668416606) 。

### 3.赠送电子书和有声读物

电子书和有声读物总是一个很好的快速礼物，很容易随时在网上购买，很容易送给朋友或爱人(你只需要他们的电子邮件地址)，几乎任何人都会喜欢它们。最棒的是，在亚马逊，你可以通过电子方式发送邮件，甚至可以指定邮件发送给收件人的日期——这样他们就可以在圣诞节早上醒来，看到收件箱里有一些好东西，而不仅仅是常见的垃圾邮件。

如果你需要一些建议，试试我们整理的这份改善生活的书籍清单 ，当然，也可以看看这些有声读物服务的一些建议——它们都有礼品订阅和促销活动，你可以利用。

### 2.礼品卡总是有用的，不管是实物还是虚拟的

当然，如果没有礼品卡，这个清单就不完整。当然，你可以跑去杂货店买一些我们上面列出的东西，但是你也可以很容易地在网上购买礼品卡，通过电子递送，这样你就不用离开家了。最重要的是，这种方式可以发送几乎任何你想要的礼品卡——无论是像亚马逊这样的大人物还是只有你的收件人知道和喜欢的专业商店。

只要确保你拿到的 [礼品卡是值得拥有的](https://twocents.lifehacker.com/the-best-and-worst-gift-cards-to-buy-this-year-based-o-1789558986) (还有 [没有疯狂的费用或随着时间的推移慢慢消耗价值](https://lifehacker.com/everything-you-need-to-know-about-giving-and-receiving-5867167) )和 [包括一个个人说明](https://lifehacker.com/an-easy-way-to-give-cash-or-gift-cards-and-keep-the-gif-1789840517) 无论是在卡上还是在礼物邮件中，只是为了让它更真诚一点。

### 1.给现金就行，但是要漂亮点

最后，现金总是有的。打开你的钱包或点击自动取款机，拿一张空白卡，然后发疯。确保附上一张私人便条 ，只是为了让礼物更加个性化，或者甚至是如何消费的建议，比如一瓶应得的葡萄酒或豪华大餐，或者是他们一直想去的公路旅行的油钱。它可以让简单的现金礼物变得更加个性化。

如果你是狡猾的类型，嗯， [你也可以在那里变得异想天开](http://lifehacker.com/fold-a-money-shirt-for-a-fun-last-minute-gift-1784486370) 。节日快乐！
