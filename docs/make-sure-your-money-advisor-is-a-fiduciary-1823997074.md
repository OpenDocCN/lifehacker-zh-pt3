# 确保你的理财顾问是“受托人”

> 原文：<https://lifehacker.com/make-sure-your-money-advisor-is-a-fiduciary-1823997074>

如果你在乎你的钱，你应该记住“受托人”这个词如果你雇人帮你理财，这很重要。随着信托规则在砧板上，这取决于你来确保你的财务顾问的行为符合你的最大利益。



一个叫国际金融理财师的人曾经让我在一篇报道中确认她就是这样的人。“注册商标和一切，”她说。我问她为什么这如此重要，她告诉我 CFP 宣誓要以客户的最大利益行事。当你的 CFP 是受托人时，你知道她是来帮忙的，而不仅仅是卖给你劣质的金融产品。

事情是这样的:非信托金融和退休规划者可能会推荐他们从中拿回扣的投资或其他金融产品，而这些产品的表现并不太好。 [正如我们之前告诉你的，](https://lifehacker.com/the-death-of-the-fiduciary-rule-is-bad-news-for-your-re-1823837417) 白宫经济顾问委员会表示，非受托人每年要花费退休投资者170 亿美元。不过，首席财务官并不是唯一进行信托宣誓的金融服务专业人士。例如，注册投资顾问也是受托人。如果你不确定，就去问。

底线:如果你雇佣任何人来看管你的钱，问他们一个简单的问题:*你是受托人吗？如果答案不是一个明确的是，那么是时候继续前进了。虽然他们可能会提供一些合理的理财建议，但很可能他们的主要目的是向你推销某种投资产品，即使这对你的钱包没有好处。*

此外，请记住:一些银行、信用合作社和投资服务机构也可能为你提供免费的金融帮助(例如，在 Redditor 的 [案例中)。要点是一样的——询问他们是否是受托人。](https://www.reddit.com/r/personalfinance/comments/7xukby/my_credit_union_offered_me_an_appointment_with_a/)
