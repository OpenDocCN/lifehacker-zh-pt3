# 成人午餐的情况

> 原文：<https://lifehacker.com/the-case-for-the-adult-lunchable-1797651881>

上周，当所有的作家聚集在纽约参加一个盛大的会议时，我发现自己和 [【贝丝】](https://kinja.com/bethskw)[【帕特里克】](https://kinja.com/patrickallan) 在一个杂货铺里，寻找瓶装水和营养零食。我抓了一份午餐，然后一边往嘴里塞一堆太圆的火鸡片和加工过的奶酪，一边想“为什么我不经常吃这些呢？”



老实说，我不经常吃它们，因为——除非你在某个地方，喝酒——我年轻时的午餐真的不好吃，但这并不意味着它们不好，从概念上来说。毕竟 [熟食板只是成年人的便当](https://www.reddit.com/r/Showerthoughts/comments/3xmbnv/charcuterie_boards_are_lunchables_for_adults/) ，熟食板才是*的好*。但是，如果说为真正的孩子准备的午餐教会了我什么的话，那就是你不必坚持经典的肉类奶酪饼干的配置。事实上，构建午餐桌的唯一规则是:

1.  里面的东西必须不需要烹饪，而且必须能够直接食用，不需要进一步的准备，最好不需要器具。
2.  里面的各种食物必须能够以令人愉快的方式与午餐中包含的所有其他食物相结合，唯一的例外是可选的甜点，它应该最后食用。(最好是奥利奥饼干或有趣的士力架。)
3.  里面的食物必须被整齐地分类，并且以一种几乎对消费者耳语的方式“嘘，你是安全的”。这个世界*到底有*有*的秩序*

我最喜欢的一些成熟的、DIY 的午餐组合包括以下内容:

*   意大利腊肠+几轮硬皮面包+奶酪片+葡萄
*   烤鸡块+迷你皮塔饼+鹰嘴豆泥+ *切片*葡萄番茄
*   花式饼干+无花果片+切达奶酪片
*   Mortadella(有态度的花式腊肠)+美国奶酪(吼我)+里兹饼干(耶，再吼一些)
*   米饼+煮熟的冷虾(或者那种假的 Krab 肉)+黄瓜片+芥末酱+一包酱油

一旦你选定了一个方便美味的组合，简单地把它装在一个 [便当盒](http://lifehacker.com/the-best-bags-containers-and-boxes-to-upgrade-your-br-915843552)——或者一个用纸杯蛋糕衬垫或其他东西隔开的可重复密封的容器——然后放进冰箱，直到你想吃的时候。成人便当是很棒的，很容易预先准备好的便当，但是我建议下次你准备晚上喝酒的时候为自己做一份。这不仅能确保你在酒精浸泡的睡眠中入睡前吃点东西，还能让你远离炉子，这非常重要。
