# 接下来我们应该做什么菜？

> 原文：<https://lifehacker.com/what-should-we-casserole-next-1820751557>

大家好，欢迎回到 [*将它砂锅*](https://lifehacker.com/tag/will-it-casserole) ，在这个栏目里我们看到哪些饭菜、菜肴、菜系可以重新想象，改造成砂锅。

Watch

我们上一个感恩节灵感的庞然大物相当令人愉快，但我准备把这个特殊的节日抛在脑后，继续前进。我有一些想法:

*   砂锅馅饼:许多不同口味的馅饼，层层叠叠，上面还有一些松脆的配料。
*   意大利砂锅菜:我想象着许多酥脆的腌肉，一些意大利干酪，美味的面包，还有一点点醋。
*   汉堡砂锅:我说的不是汉堡助手——我说的是牛肉、奶酪、培根、洋葱和烤芝麻小圆面包的组合。你可以跟*打赌，你的*小圆面包上面会有泡菜丁。
*   美食广场砂锅菜:这里会涉及到美食广场的所有经典美食——蹩脚的披萨、某种糖醋鸡，或许还有牛排三明治。

当然，请随意留下您的精彩想法。我对砂锅菜持开放态度，只要它不涉及更多的火鸡。
