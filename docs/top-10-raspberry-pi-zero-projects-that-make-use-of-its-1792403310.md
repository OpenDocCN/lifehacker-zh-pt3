 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-x3Zb-4zI_rE&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-x3Zb-4zI_rE&start=0) 

微小的显示屏很棒，所以树莓派零度被塞进其中是完全合理的。 [这些小东西组成了完美的办公桌配件](https://lifehacker.com/use-a-raspberry-pi-zero-to-make-your-own-miniature-info-1748578164) 。

我们已经看到了这个想法的一些不同的迭代， [像这个](https://lifehacker.com/use-a-raspberry-pi-zero-to-make-your-own-miniature-info-1748578164) 显示你的网络信息或者这个 [显示像素艺术](https://lifehacker.com/use-a-raspberry-pi-zero-to-power-a-pixel-art-frame-1788868060) 。见鬼，即使是这个 [迷你动画 GIF 显示](http://lifehacker.com/make-an-animated-gif-photo-frame-with-a-raspberry-pi-1658839211) 也可以很容易地修改为与 Zero 一起工作。

### 9.只需按一下按钮，就可以在任何地方播放 NPR 的音乐

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-8FhpGqXJzso&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-8FhpGqXJzso&start=0) 

如果你是 NPR 的忠实粉丝，那么你可能会喜欢 NPR One 应用，它随机播放一系列精选的 NPR 播客。你可能会感到惊讶，从命令行可用的流，这意味着你可以在一个树莓 Pi 零访问。

这里的一个项目的自然想法是用树莓做一个小小的 [NPR 一号收音机零](https://lifehacker.com/build-a-npr-one-radio-with-a-raspberry-pi-zero-1779639819)T3。你有跳过一个故事和暂停播放的按钮，而整个东西被塞进一个别致的 DIY 小盒子里。

### 8.给电动滑板供电

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-2WLEur3M8Yk&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-2WLEur3M8Yk&start=0) 

如果你使用滑板出行，你可能早就梦想着为滑板提供动力以获得更好的机动性。很自然的， [有人决定用一个树莓来给一个](https://lifehacker.com/this-diy-electric-skateboard-is-powered-by-a-raspberry-1788887593) 供电。

这里的想法是把圆周率 0 藏在滑板下面。然后 Pi 驱动马达，你可以用遥控器来控制它。这很荒谬，但这并没有减少使用它的乐趣。

### 7.建造一台电源大小的微型计算机

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-dfg4g3YuvnY&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-dfg4g3YuvnY&start=0) 

圆周率零点的最大优势是它的大小。一个项目利用这一点，将 [树莓 Pi 零点直接干扰到电源](https://lifehacker.com/wire-a-raspberry-pi-zero-directly-into-a-power-supply-f-1791290336) 上。

这里的想法很简单。你用 SSH 设置了 [Raspberry Pi Zero 或者 VNC](https://lifehacker.com/how-to-control-your-raspberry-pi-from-any-computer-usin-1788592777) ，然后用 Sugru 把它直接挂载到电源上。现在，你可以把它插在墙上，远程控制它。这使得它几乎看不见。

### 6.建立一个移动图书馆

你见过小区里那些挤满免费书籍的可爱迷你图书馆吗？你可以用 Raspberry Pi Zero 以数字方式做到这一点，从 Wi-Fi 热点共享你自己的整个图书馆。T3】

本质上，你只要把你的 Raspberry Pi Zero 变成一个 Wi-Fi 接入点，然后分享你自己的无 DRM 的书籍、杂志、pdf 或任何你有的东西。显然，你应该保持这种合法，但这是一种分享你自己的东西的有趣方式。

### 5.随机播放任何一集《辛普森一家》(或任何电视节目)

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-E_SgUrJ3l9E&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-E_SgUrJ3l9E&start=0) 

如果你抛弃了有线电视，转而通过媒体 实现了 [全流媒体，你可能仍然会怀念在电视上随意看一集你最喜欢的节目的日子。树莓派零度可以把它带回来。](http://lifehacker.com/ditch-cable-tv-this-weekend-and-keep-watching-your-sho-5834909)

[在这个项目](https://lifehacker.com/build-a-raspberry-pi-powered-media-player-that-plays-a-1776524528) 中，他们使用了存储在 SD 卡上的几集*《辛普森一家》*。通过自定义脚本，你可以点击一个按钮，它会随机播放一集。你可以在这里添加任何你想要的媒体，如果你真的想要的话，包括多个节目，但是*辛普森一家*是一个很棒的选择。

### 4.制作一个动作感应相机

树莓派长期以来一直是一款 [出色的动作感应相机](https://lifehacker.com/build-a-motion-sensing-security-camera-with-a-raspberry-1768562499) ，但 Pi Zero 的低调让它更加出色。

有了 Raspberry Pi Zero，你的运动感应相机的尺寸更小，这意味着它可以在更小的空间里工作。例如， [这款用一对吸盘](https://lifehacker.com/the-raspberry-pi-zero-makes-a-great-motion-sensing-secu-1791167441) 安装在你的窗户上，大概是它能得到的最轻便的东西了。

### 3.打造世界上最小的(可能是)MAME 橱柜

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-0O0DeRL7P-w&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-0O0DeRL7P-w&start=0) 

MAME 橱柜很棒，但是它们太大了。对于完全相反的方法，您可以使用一个 Raspberry Pi Zero 来制作一个和 Pi 本身一样小的。

你需要一些材料来让这个吸盘工作，包括 3D 打印的盒子，但最终橱柜是完全实用的，这意味着你将拥有办公室里最好的桌子配件。

### 2.制作一台微型加密狗电脑

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-qbWYUFgLhSU&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-qbWYUFgLhSU&start=0) 

虽然 Pi Zero 本身是一台漂亮的小计算机，但它是一台更好的加密狗计算机。这样，你就可以把它连接到你的任何一台电脑上，然后它会自己直接系上，这样你就不需要添加 USB 或任何网络了。

这个项目最好的部分是你不再需要另外一个鼠标、屏幕或键盘。只需插入普通的台式电脑，你就可以开始工作了。

### 1.把一个树莓派零塞进一个旧的学校控制器里

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-j5dXx0NhLxg&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-j5dXx0NhLxg&start=0) 

我们都知道树莓派制造了一款 [出色的 DIY 视频游戏机](https://lifehacker.com/how-to-turn-your-raspberry-pi-into-a-retro-game-console-498561192) 。设置过程只需要几分钟，树莓 Pi Zero 完全能够处理 SNES 时代和更早的旧游戏。当整个系统都在控制器内*时，整个项目会更好。*

举个例子(咳咳)，这个项目用的是一个 [SNES 手柄](http://lifehacker.com/cram-a-raspberry-pi-retro-console-inside-a-snes-control-1787227404) ，这个 [一个铲球 NES 手柄](https://lifehacker.com/stuff-a-raspberry-pi-zero-into-a-nintendo-controller-fo-1753013570) ，这个用的是那个 [好旧的原装 Xbox 手柄](http://lifehacker.com/stuff-a-raspberry-pi-zero-into-an-xbox-controller-for-o-1745525351) 。无论你决定怎么做，你都会得到一个非常漂亮的小型 DIY 复古游戏机，你可以带着它去任何地方。
