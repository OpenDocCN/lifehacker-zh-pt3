### 从实际到过那里的人看来，太空实际上是什么样的

我从小就想成为一名宇航员，我敢打赌，你们中的大多数人至少在某个时候想知道在太空中到底是什么样的——不仅仅是整个微重力的事情，而是它实际上是什么样的，就像在宇宙飞船中闻起来是什么样的，每天的感觉是什么样的，在太空中吃、喝、睡和生活是什么样的。嗯，新书， [*太空是什么样子的？由 Ariel Waldman*](https://www.amazon.com/dp/1452144761?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/what-space-is-really-like-using-algorithms-to-improve-1780396194&asc_source=&linkCode=ogi&psc=1&smid=A1KY5HRJRH6YF1&tag=kinjalifehackerlink-20&th=1) 创作，采访了一些真正去过的人，记录了他们的经历。

在一天结束时，这个播客值得一听和考虑，即使是以这种或那种方式挑战你自己的确认偏见，并了解更多关于什么是一个更具争议性的行业，甚至更多的光被导向它。[ *via* [*魔鬼经济学*](http://freakonomics.com/podcast/payday-loans/) ]
