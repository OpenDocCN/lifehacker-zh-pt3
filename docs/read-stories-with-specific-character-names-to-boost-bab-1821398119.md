# 阅读有特定人物名字的故事来促进婴儿的大脑发育

> 原文：<https://lifehacker.com/read-stories-with-specific-character-names-to-boost-bab-1821398119>

我们大多数人都知道，给婴儿朗读是一件非常好的事情——它与 [语言和认知发展](https://doi.org/10.1111/j.1467-8624.2006.00911.x) 联系在一起，帮助 [加强亲子关系](http://apps.who.int/iris/bitstream/10665/42878/1/924159134X.pdf) ，并在我们试图进入我们的 [推荐的每天 3 万词](https://www.education.com/magazine/article/30000_words/) 时给我们一个受欢迎的脚本，而不必再次说唱*新鲜的贝莱尔王子*主题曲。但是为了获得最佳利益，仅仅拿起任何一本纸板书或泰国菜外卖菜单并开始背诵单词可能是不够的。根据一项新的研究，你读的书的类型可能会有很大的不同。

佛罗里达大学的大脑、认知和发展实验室发现，阅读给不同的人物和物体起名字的书 [可能比以通用方式提及人物和物体的书](http://onlinelibrary.wiley.com/doi/10.1111/cdev.13004/abstract) 产生更大的大脑发育益处(“看看这些动物！”).实验室主任 Lisa S. Scott 的团队记录了婴儿在阅读不同书籍时大脑的自然电流，她在 [对话](https://theconversation.com/for-babys-brain-to-benefit-read-the-right-books-at-the-right-time-83076) 中解释了结果:

> 在我们的研究中，我和我的同事在婴儿出生后的六个月里跟踪观察他们。我们发现，当父母给婴儿看带有分别命名的面孔的 [书籍时，他们会学到更多，将他们学到的东西归纳到新的情况中，而](https://doi.org/10.1111/j.1467-9280.2009.02348.x) [会显示出更特殊的大脑反应](https://doi.org/10.1016/j.neuropsychologia.2010.02.008) 。这与没有标签的书或在书中每个图像下具有相同通用标签的书形成对比。婴儿期的早期学习也与四年后儿童期的益处有关。

一个有单独名字的故事和一个泛指一个物体的故事的例子。

幸运的是，有大量的儿童书籍都有人物名字，包括所有的经典故事(例如杰克和吉尔，小缪费小姐，彼得兔)。如果你宝宝最喜欢的书上没有这些，也没关系。正如司各特所写，“我自己的女儿喜欢那些 [*拍兔子*](https://www.penguinrandomhouse.com/books/241481/pat-the-bunny-first-books-for-baby-pat-the-bunny-by-dorothy-kunhardt-and-edith-kunhardt/) 的书，以及关于动物的故事，像 [*【亲爱的动物园】*](https://www.panmacmillan.com/authors/rod-campbell/dear-zoo) 。如果名字不在书里，我们就随便编。”
