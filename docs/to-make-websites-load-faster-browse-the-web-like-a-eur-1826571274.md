# 要让网站加载更快，像欧洲人一样浏览网页

> 原文:[https://life hacker . com/to-make-websites-load-fast-browse-the-web-like-a-EUR-1826571274](https://lifehacker.com/to-make-websites-load-faster-browse-the-web-like-a-eur-1826571274)

我们现在生活在一个 GDPR 世界。如果你不知道，GDPR 代表欧盟的“一般数据保护条例”隐私法，该法于 5 月 25 日生效，导致你过去一个月使用的每项服务都挤满了你的收件箱。

Watch

尽管这些隐私邮件令人讨厌，但法律已经迫使许多大公司对收集和存储你的数据的方式做出了相当大的改变，包括让你有更多机会了解他们对你的了解，并删除你不想让他们知道的数据。

奇怪的是，GDPR 可能还会提高你的浏览速度——至少，根据一些在多个网站进行过实际测试的消息来源。正如营销潜水的埃里卡·斯威尼 [描述的](https://www.marketingdive.com/news/study-many-publishers-eu-sites-are-faster-and-ad-free-under-gdpr/524844/) :

> *“例如，在 GDPR 实施后，今日美国网站的美国版网页平均加载时间为 9.86 秒。相比之下，英国版网站的加载时间为 0.42 秒，法国为 0.75 秒，德国为 0.51 秒。更快的加载时间可以归因于大多数外部第三方功能的删除，如广告服务器、谷歌服务和分析、社交媒体插件等。”*

网络开发人员 Marcel Freinbichler 发现了类似的结果——不仅《今日美国》的网站加载速度更快，而且网站的“欧盟”版本也将请求的数据量从 5.2MB 减少到 500KB，减少了大约 90%。

虽然你们当中精通网络的人可能会认为，你可以通过在浏览器中安装广告拦截器来复制这种“欧盟体验”，但 Freinbichler 指出，你可能体验到的更快的速度仍然不如一个网站的精简版“欧洲版”，如果它提供的话。

虽然我强烈建议计划一个可爱的欧洲假期来享受更快的浏览速度(和可爱的乡村)，但仍然有一些技巧可以让你在非欧盟国家的家中浏览“欧洲风格”的网页。不过，有一点需要注意；一些公司 [完全阻止来自欧盟](https://econsultancy.com/blog/70065-gdpr-which-websites-are-blocking-visitors-from-the-eu-2/) 的访问者，要么是因为他们需要更多的时间来使他们的网站符合 GDPR，要么是因为他们完全没有这样做的打算。

对于你可以访问的网站，这里有一些像欧洲人一样浏览的方法:

*   **使用虚拟专用网:**有了虚拟专用网，你基本上是连接到其他地方的一台服务器，而那台服务器正在路由你的请求，并使*看起来*像 [你实际上位于虚拟专用网服务器所在的任何地方](https://lifehacker.com/the-beginners-guide-to-vpns-1819912770) 。如果你有一个快速的虚拟专用网(VPN)并能很好地连接到一个欧洲服务器，这可能是一个很好的方式来“假装”你不是来自某个地方，来浏览公司配置的特殊(更快)网站，以保持与 GDPR 兼容。抵制使用免费虚拟专用网或网络代理的冲动，因为你永远不知道他们会在你的浏览会话中用他们可能捕获的任何数据做什么。(我想付费 VPN 也是如此，但是免费 VPN 更有动力想方设法从你身上获利，因为你没有付钱给他们。)
*   **使用 Opera:** 尽管我刚刚警告了免费 VPN，但值得注意的是， [Opera 网络浏览器](https://www.opera.com/) 带有内置的免费 VPN，如果你只是想更快地快速浏览你最喜欢的新闻网站，它可以让你非常容易地切换到欧洲服务器。要激活它，点击 Opera 的菜单(“O”图标在左上角)，设置，隐私&安全，并选择“启用 VPN。”当你去加载一个网页，看到浏览器地址栏左边的小 VPN 图标，点击它，选择“欧洲”作为你的虚拟位置。
*   **让你的浏览器充满隐私扩展:**嗯，真的，你可能只需要一个组合和一个扩展，比如 [NoScript](https://noscript.net/) 、 [uMatrix](https://github.com/gorhill/uMatrix) 、 [ScriptSafe](https://chrome.google.com/webstore/detail/scriptsafe/oiigbmnaadbkfbmpbfijlflahbdbdgdf?hl=en-US) ，或者 JavaScript Switcher([Firefox](https://addons.mozilla.org/en-US/firefox/addon/quick-js-switcher/)、 [Chrome](https://chrome.google.com/webstore/detail/quick-javascript-switcher/geddoclleiomckbhadiaipdggiiccfje?hl=en) 你的目标是阻止尽可能多的垃圾加载到你访问的网站上，包括 Javascript，除非它对网站的运行是绝对必要的。如上所述，虽然这可能仍然不能让你获得像网站专门为特定受众创建精简版本一样的体验，但这是你能做到的最好的，除非你走 VPN 路线——或者，如果你能找到一个，你最喜欢的网站的基于文本的版本。啊，甜蜜的简单。
