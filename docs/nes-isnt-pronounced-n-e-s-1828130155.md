# NES 不发音为 N.E.S

> 原文：<https://lifehacker.com/nes-isnt-pronounced-n-e-s-1828130155>

人们怎么发音，单词就怎么发音，坚持不发音就是打一场失败的仗。这就是为什么[“giff”和“jiff”同样有效](https://lifehacker.com/lets-end-the-gif-jif-pronunciation-debate-right-now-1796419121#_ga=2.24661562.1421609258.1533563148-1297080755.1497980211) 。但对于那些关心的人来说，根据任天堂 3DS 新游戏，NES(任天堂经典娱乐系统)发音为“ness”



在 Twitter 上，游戏玩家凯尔·麦克莱恩(Kyle McLain)展示了日本版 3DS WarioWare Gold 的一个屏幕，其中将发音为“ne”和“su”的字符放在 NES 之上。因此，“nesu”是日本汉字中最接近“ness”的一个词欧洲玩家 [确认](https://www.eurogamer.net/articles/2018-08-03-nintendo-finally-confirms-how-you-pronounce-nes) 。

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1025315977754365952&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1025315977754365952&autosize=1) 

用户 LeoTheCollector 回复了一则任天堂的美国广告，广告的结尾是“仅在 N.E.S .播放”

 [https://lifehacker.com/embed/inset/iframe?id=twitter-1025479430188269575&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-1025479430188269575&autosize=1) 

因此，如果 80 年代游戏机的发音是你喜欢争论的事情，你仍然有任何一方的证据。如果比起美国广告商，你更信任任天堂当前游戏的日本文案，那就选“ness”如果你认为因为“NES”代表英语单词，它的发音不应该由音译回日语来决定，那么 WarioWare Gold 不会改变什么。

我只想指出一点:如果你遵守规则，“ness”意味着 SNES 读作“sness”，3DS 读作“three deess”，N64 读作“nuh-64”

在他的文章《政治和英语》中 奥威尔列出了改进句子的五条规则。但他补充了第六条:“打破这些规则比说出任何赤裸裸的野蛮要快。”
