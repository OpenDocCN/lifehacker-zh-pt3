# 十月是硬拉节

> 原文：<https://lifehacker.com/october-is-for-deadlifts-1829438935>

欢迎回来，朋友们！在我们的 [健身挑战](https://lifehacker.com/c/lifehacker-fitness-challenge) 中，我们已经进行了大量的体重练习，上个月我们开始了一些 [跑步或步行](https://vitals.lifehacker.com/preview/how-to-measure-your-progress-in-running-1829391989) 。这个月我们将去健身房做一些举重运动——但是不要担心，即使你没有健身房会员，我们也会为你提供选择。



这个月我们将攻克 [硬拉](https://vitals.lifehacker.com/why-deadlifts-are-worth-the-hype-and-how-to-do-them-sa-1756526319#_ga=2.118749449.1639559872.1538407826-1456718367.1520458611) 。你在地板上有一个很重的东西，你抓住它，然后站起来。一旦你开始练习硬拉，你会发现这种方式比其他方式能举起更多的重量。即使身材矮小的人也能举起三位数的重量。对于初学者到中级举重运动员来说，一个里程碑是硬举一个与你身体重量相同的杠铃。因此，我们对此寄予厚望。

但是这些都是不可能的，除非你有良好的状态，并且你知道如何安全地硬拉。就像 [举任何东西](https://vitals.lifehacker.com/what-lift-with-your-legs-really-means-1829166958) 一样，你要保持脊柱中立——不要弄圆或挖空——并将重量保持在身体附近。幸运的是，杠铃的形状旨在使这变得容易，互联网上到处都是视频，展示在硬拉过程中你的身体应该如何移动。(我们在 [中链接了一些最佳硬拉指南](https://vitals.lifehacker.com/why-deadlifts-are-worth-the-hype-and-how-to-do-them-sa-1756526319#_ga=2.254988043.202922725.1538410833-1723114163.1524514905) 。)

### 这个月做什么

如果你从未做过硬拉:专注于学习硬拉，但不要担心重量。在家里抓一个扫帚柄，或者在健身房抓一个很轻的杠铃或者 [身体棒](https://www.gymsource.com/equipment/body-bar) 。每次锻炼花几分钟练习这个动作，只有当你准备好了才增加重量。与此同时，试试这些只消耗身体重量的动作 ，它们锻炼的肌肉与硬拉一样。

如果你准备好尝试，但没有经验:从上面的初学者动作开始，然后增加重量——但要轻。请记住，你可以用哑铃或壶铃，或轻负荷的杠铃进行硬拉。考虑使用足够轻的重量，你可以做三组 8-12 个良好的形式代表。

如果你知道自己在做什么:决定一个时间表，计划好这个月的重点硬拉。你可以一周举一次重物，再花一两次时间进行较轻的硬拉或辅助锻炼来加强你的背部和腿部。

试一试，并保持安全！我们将在每周五带着额外的提示和签到线索回来。
