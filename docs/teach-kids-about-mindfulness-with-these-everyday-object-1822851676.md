# 用这些日常物品教孩子关于正念

> 原文：<https://lifehacker.com/teach-kids-about-mindfulness-with-these-everyday-object-1822851676>

正念对儿童和成人一样强大——它可以帮助他们调节情绪，在生活变得紧张时更加冷静地应对。而是简单地告诉你的孩子“清除你的想法！”或者“活在当下！”可能只会让他们更加困惑(因此*更加*有压力)。



我们已经写了一个填充动物玩具形式的 [【呼吸伙伴】](https://lifehacker.com/use-breathing-buddies-to-teach-your-kid-self-control-1819400969) 如何让孩子们注意他们的呼吸。其他物品对他们的练习也有帮助——他们可以看到、感觉到甚至品尝到的东西。我喜欢苏珊·凯瑟·格林兰的这些正念活动，她是《T4》[*【正念的孩子】*](https://www.amazon.com/Mindful-Child-Manage-Happier-Compassionate/dp/1416583009?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/teach-kids-about-mindfulness-with-these-everyday-object-1822851676&asc_source=&tag=kinjalifehackerlink-20) 的作者，也是在线资源 [【内心的孩子】](https://www.susankaisergreenland.com/inner-kids-program/) 的创始人。它们需要你家里可能已经有的简单物品。

对象:

### 雪球或闪光水球

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-8dMiv5B7I0Y&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-8dMiv5B7I0Y&start=0) 

在 [这段视频](https://www.youtube.com/watch?v=8dMiv5B7I0Y) 中，格林兰演示了一项她称之为“看清楚”游戏的活动。你可以告诉你的孩子，我们生来都有清晰、冷静的视角，就像闪光球里的水一样。“如果我把球放在我面前，我仍然可以看到你，”她说。“你能看见我吗？”你知道这是怎么回事吗？

然后让你的孩子摇摇球，以显示压力遮蔽了我们的视角。当这种情况发生时，我们看不到也不理解对方。所以感受我们的呼吸和休息是很重要的。正如格林兰解释的那样，当我们这样做时，所有遮蔽我们视野的事物都“沉淀到底部”正念觉知并不能消除我们生活中的不好的事情，但它能帮助我们重新看得清楚。

### 一个艾斯·库伯

啊，手握冰块挑战。谁知道这个痛苦的游戏可以用来教孩子们正念呢？一位内心童趣的老师 [演示了](https://www.youtube.com/watch?v=tjyZ_bdbbNQ) 如何引导活动。在冰块砸到你孩子的手之前，问问他身体感觉如何(也许是紧张和紧张？).然后当立方体在他手里的时候，看看他是否能保持冷静，即使有点痛。这表明我们可以把我们的反应从我们的身体感觉中分离出来。事后，你可以问问你的孩子他做了什么来保持冷静。深呼吸。闭上他的眼睛？想点别的？这些工具可以在任何情况下使用。

### 一个苹果

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-j2uooeprDkE&start=52](https://lifehacker.com/embed/inset/iframe?id=youtube-video-j2uooeprDkE&start=52) 

这更多的是一个比喻，但它真的很可爱。格陵兰用苹果证明，虽然我们的过去是我们的一部分，但我们不应该被它分散注意力。同样，我们也不应该全神贯注于未来会发生什么。相反，我们可以享受这一刻。

### 好时之吻

你可以教孩子用慢动作吃饭，以便品尝和欣赏他们的食物。(我的孩子吃东西非常慢，但那是因为她心不在焉，而不是因为她在品味什么。)让你的孩子注意他们的食物看起来、摸起来和闻起来如何。让他们注意咀嚼和吞咽时的感觉。在他们的 [意识游戏活动卡](https://www.amazon.com/Mindful-Games-Activity-Cards-Mindfulness/dp/1611804094?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/teach-kids-about-mindfulness-with-these-everyday-object-1822851676&asc_source=&tag=kinjalifehackerlink-20) 中，格林兰和安娜卡·哈里斯写道，好时之吻是一种很好的食物，可以作为特殊的款待。“孩子们可以试着把它含在嘴里，直到巧克力完全融化，同时注意所有的五种感官:看到亲吻，听到铝箔被拿掉的声音，品尝、闻和感觉嘴里的巧克力。”
