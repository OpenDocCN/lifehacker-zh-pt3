# 怀疑脸书的假新闻？使用新的上下文按钮

> 原文：<https://lifehacker.com/suspicious-of-fake-news-on-facebook-use-the-new-contex-1819217472>

脸书有一个 [假新闻问题](https://gizmodo.com/facebooks-fake-news-solution-isnt-going-to-solve-the-pr-1809062985) 。它最近的热门新闻失误导致匿名留言板 4chan 发布了关于拉斯维加斯枪击案的错误信息。为了打击假新闻的传播(以及对该公司日益增长的反弹)，这家社交网络正在测试一项新功能，让用户可以通过点击“更多信息”按钮，自己区分来自值得信赖的出版商的文章和比平时更可疑的文章。



该公司新的 [上下文信息按钮](https://newsroom.fb.com/news/2017/10/news-feed-fyi-new-test-to-provide-context-about-articles/) 将相关信息添加到你的新闻订阅源中共享的任何链接文章中，例如关于出版商的信息(像维基百科页面)、关于该主题的其他文章，以及该文章如何在脸书上被共享。如果一篇文章缺少像维基百科页面那样的上下文信息，或者缺少到其他出版商类似故事的链接，脸书会提醒用户缺少上下文信息。根据脸书的说法，这个新工具是为了帮助用户“对阅读、分享和信任哪些故事做出明智的决定”在脸书解散其 [人性化趋势新闻团队](https://gizmodo.com/facebook-removes-human-curators-from-trending-topic-mod-1785818153) 后，脸书自身分享的假新闻的存在已经成为一个越来越成问题的问题。

然而，脸书之前为用户识别假新闻的努力——在有问题的文章上贴上“有争议”的标签——[——并没有多少改变脸书年轻用户的习惯](http://www.niemanlab.org/2017/09/adding-a-disputed-label-to-fake-news-seems-to-work-a-little-but-for-some-groups-it-actually-backfires) 。我们联系了脸书，询问这种新的假新闻识别工具如何比以前更有效，并将相应地更新报道。
