# 消除网络烦恼的 10 大浏览器扩展

> 原文：<https://lifehacker.com/top-10-browser-extensions-that-eliminate-web-annoyances-1787371353>

网络上充斥着烦恼。当你访问一个你不得不关闭的页面时，弹出广告，甚至在背景标签中自动播放音频的网站，重新加载并带走你输入的所有文本的页面，它们都很糟糕。以下是一些浏览器插件，它们让网络成为每个人更好的去处。

### 10.覆盖层后面(Chrome/Firefox)

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-0Fc5q-Ba8S8&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-0Fc5q-Ba8S8&start=0) 

你知道你访问的那些网站，开始阅读一篇文章，然后突然你得到一个弹出窗口，要求你注册一个时事通讯或什么的，唯一的解决方法是点击“不，谢谢，(在此插入贬低的东西)，”是啊，我也讨厌那些，Chrome 的 和 Firefox 的的 [覆盖层后面，永远驱逐它们。](https://chrome.google.com/webstore/detail/behindtheoverlay/ljipkdpcjbmhkdjjmbbaggebcednbbme)

[我们不久前报道过](https://lifehacker.com/behindtheoverlay-instantly-removes-annoying-overlays-fr-1588091959) ，它仍然很有魅力，不管是那种令人讨厌的“是的，太棒了/不，我讨厌好东西”广告，还是其他任何弹出广告，比如那种恳求你注册或登录的广告。

### 9.第一页(Chrome/Safari)和重新分页(Firefox)

令人欣慰的是，如今网络上的多页幻灯片远不如过去常见，但许多网站仍然依赖它们将一篇文章扩展成十几个或更多的实际点击和页面浏览量。读起来又烦又慢，但是 [你不用受它的折磨](https://lifehacker.com/how-to-fix-annoying-multi-page-articles-all-over-the-we-1164869508) 。Chrome 和 Safari 用户可以安装 [第一页](http://globalmoxie.com/blog/page-one-safari-chrome-extension.shtml) ，它将所有的幻灯片片段放在一起，并显示在同一个页面上，Firefox 用户可以使用 [重新分页](https://addons.mozilla.org/en-US/firefox/addon/repagination/) ，右键单击也可以做同样的事情。

如果你根本不想安装任何东西，你可以随时使用页面的“打印机友好”版本或浏览器的“阅读模式”来获取所有内容，而不必点击十几次，或者将其发送到 Pocket 或其他稍后阅读的服务，这些服务通常会删除 cruft。

### 8.拉扎勒斯(Chrome/Firefox)

Lazarus 已经存在 [很长很长时间了](http://lifehacker.com/lazarus-form-recovery-saves-web-page-form-data-5097334) ，但是它为我保存的大量信息保证了它在这个列表中的位置，如果你已经使用它几乎同样长的时间，你知道它可能也为你节省了一吨头痛，可能是因为你填写的表格，试图提交，以及一个页面超时，或者清空你所有的文本框，因为你没有以某种方式格式化一个必填字段。

这些烦恼是 Lazarus 的主要目标，永远结束它们是 Chrome 网上商店 的快速安装 [，或者，如果你是 Firefox 用户，Mozilla 插件](https://chrome.google.com/webstore/detail/lazarus-form-recovery/loljledaigphbcpfhfmgopdkppkifgno?hl=en) 的。它不止一次救了我的命，既然我已经看到一些抱怨说这个插件在 Chrome 中不再好用了，如果你需要一个替代方案，可以考虑 [TIRE(文本输入恢复扩展)](https://chrome.google.com/webstore/detail/text-input-recover-extens/gjhalkgpmhdpbfjdmeghbdbmbannipjf?hl=en-US) 。

### 7.魔幻动作(Chrome/Firefox/Opera)

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-pwHX9JT2APQ&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-pwHX9JT2APQ&start=0) 

YouTube 上充满了娱乐视频，但它无疑是最令人讨厌的网站之一。从糟糕的评论到恼人的(但容易被禁用的)叠加和随机分辨率，让一切都按照你想要的方式进行可能是一种痛苦——然后一个改变可能会撤销你在另一个系统上的偏好。 [魔法动作](https://www.chromeactions.com/) 修复了这一切。

除了确保任何可以高清播放的视频(或者更好的是，4K)都可以播放，Magic Actions 还抑制广告( [，你可能想也可能不想做](http://lifehacker.com/i-paid-to-get-rid-of-ads-on-youtube-and-now-my-life-is-1742985089) )，取消 YouTube 评论部分，让你用鼠标滚轮控制音量，并添加了一个电影模式，可以削减 cruft ，让你专注于你正在观看的视频。总而言之，它给你的 YouTube 体验更像媒体播放器，而不是视频门户。另外，它是我们最喜欢的 Chrome 扩展之一，尽管它也适用于 Firefox 和 Opera。

### 6 .维基百科(Chrome/Firefox/Safari)

总的来说，维基百科可能是网络上最有用的网站之一，但它也有自己的烦恼。页面加载速度很快，但往往难以导航，有大量的部分，可以使用更新鲜，更干净的外观。维基百科过去曾尝试过重新设计，但作为 Chrome、Firefox 和 Safari(也可用于移动设备)的附加软件，[wiki wand](http://www.wikiwand.com/)[完全将维基百科的每个页面](https://lifehacker.com/wikiwand-overhauls-wikipedia-makes-it-easier-to-browse-1618271448) 转变为更愉快的阅读体验。

该插件将目录移动到边栏，您可以在阅读时快速找到它，使字体更容易阅读，在链接和图像上添加悬停预览，等等。如果你喜欢这种外观，但想要一些不那么激烈的东西，可以试试前面提到的，它给你 [更多一点的控制](http://lifehacker.com/wikitweaks-adds-pop-up-summaries-removes-clutter-on-wi-1652215201) 。

### 5.无声网站声音拦截器(Chrome)

我们 [刚刚提到了这个 Chrome 插件](https://lifehacker.com/silent-site-sound-blocker-only-lets-whitelisted-tabs-pl-1786969068) ，但是 [无声网站声音拦截器](https://chrome.google.com/webstore/detail/silent-site-sound-blocker/hkbdhiphllimobjnfeeekaogfibmdgfe) 解决了我对许多网站最大的不满之一——当你加载它们时，它们试图通过在它们网站的某个小角落自动播放随机视频来扩大视频浏览量并达到目的。这很烦人，尤其是当你打开了多个标签页，却不知道视频实际上是在页面的哪个位置播放的时候。

无声网站声音拦截器会自动关闭非您关注的标签，即使这样，它也会使用白名单系统来确保只有您批准的标签才能播放音频。你甚至可以允许一个站点播放一次音频，然后再次询问，或者屏蔽一次，然后在下次访问时再次询问。当然，你也可以将网站列入黑名单，这样它们就永远不会播放音频了。如果你想要一个替代方案，请查看前面提到的 [智能标签静音](http://lifehacker.com/smart-tab-mute-makes-sure-sound-plays-in-one-tab-at-a-t-1783679224) ，可用于 [Chrome](https://chrome.google.com/webstore/detail/smart-tab-mute/dnfbgicfhchdpogmafjifjgbcjdaikgn) 和 [Firefox](https://addons.mozilla.org/en-US/firefox/addon/smart-tab-mute/) 。

### 4. Imagus (Chrome/Firefox)

微小的缩略图无法打开大图——或者更糟，只能打开同一页面中的大图或者打开链接而不是更大的视图——令人讨厌，前面提到的[Imagus](https://lifehacker.com/imagus-is-a-hover-zoom-replacement-to-enlarge-images-on-1490681966)(Chrome 和 [Firefox](https://addons.mozilla.org/en-US/firefox/addon/imagus/) 可用的)修复了它们。取而代之的是，只需将你的鼠标悬停在你想以更大的视角观看的图像上，它就会弹出来，又大又好看，这样你就可以查看它了。

我们曾经推荐 Hover Zoom 做这样的事情，但是自从 [一吨有用的 Chrome 扩展去了黑暗面](http://lifehacker.com/many-browser-extensions-have-become-adware-or-malware-1505117457) ，我们再也不能了，而 Imagus 是一个合适的替代品。

### 3.口袋(Chrome/Firefox/Safari/Opera/Edge)

[袖珍](https://getpocket.com/) 对很多事情都很有用，虽然严格来说 [是一项“以后再看”的服务，它可以让你保存文章](http://lifehacker.com/read-it-later-showdown-pocket-vs-instapaper-1756021479) 以便在你有时间的时候离线阅读。这很棒，有很多专业的技巧可以让你从中获得最大的收益，但这也是消除传统网络体验中几乎所有烦恼的好方法。此外，它适用于几乎任何网络浏览器。

例如，你可能已经知道，通过将一篇文章保存到 pocket，你可以将分页的、幻灯片式的文章拉进一个长的、易读的页面 。你还可以去掉广告和其他会让你的设备变慢的膨胀的东西，或者让你很难阅读它们周围的内容。使用合适的袖珍工具，你甚至可以估计阅读一篇文章需要多长时间。 dr 用要点概括你的文章，这样你可以先略读，然后再深入研究，等等。所有这些都不需要忍受自动播放视频、评论、广告或其他烦恼。

### 1.uBlock Origin (Chrome/Firefox)

这个网站由于广告而保持引擎运行的事实并不意味着我们不知道广告是网络中最令人讨厌的部分。也许它们会占据整个页面，或者当你试图阅读或点击时，它们会干扰你的浏览器。也许它们会自动播放或阻止您正在阅读的页面，直到您可以关闭它们。也许他们只是减慢你的浏览器。我们觉得痛苦、和 [uBlock Origin](https://github.com/gorhill/uBlock) 是我们最喜欢的广告拦截器对于 Chrome 和 [Firefox](https://addons.mozilla.org/addon/ublock-origin/) 和[Opera](https://addons.opera.com/en-gb/extensions/details/ublock/)——不仅仅是因为它足够轻松地完成工作，而是因为 [难以置信地强大和可定制](https://lifehacker.com/the-best-browser-extensions-that-protect-your-privacy-479408034) 。

uBlock Origin 可以让你控制加载哪些元素，不加载哪些元素，你可以看到哪些广告，你可以禁止哪些广告。你可以随心所欲地定制它，它的 [比 Adblock Plus 或 Ghostery 的](http://lifehacker.com/ublock-is-a-fast-and-lightweight-alternative-to-adblock-1625246461) 更快、更轻便，这两家都有自己的问题与 [销售广告](http://lifehacker.com/adblock-plus-is-now-selling-acceptable-ads-1786589122) 或 [与广告公司合作](http://lifehacker.com/ad-blocking-extension-ghostery-actually-sells-data-to-a-514417864) 。如果你想要一个替代方案，可以考虑 [断开](https://disconnect.me/) ，这是一个浏览器扩展，它不会阻止广告，但会通过阻止跟踪来保护你的隐私——它主动阻止的唯一广告是恶意广告， [，你无论如何都要阻止这些广告](https://lifehacker.com/how-to-protect-yourself-from-malvertising-on-the-web-1745588094) 。
