### 任天堂 64 模拟的状态:它是复杂的

任天堂 64 模拟器已经存在了十多年，但是发展变得缓慢和分散。没有一个仿真器是完美的。事实上，它们中的大部分都相当不准确，大量的游戏仍然无法玩。

因此，没有一个真正的“最好的”任天堂 64 模拟器。某些游戏在一个模拟器上可能比在另一个模拟器上运行得更好，所以你可能不得不保留几个。此外，任天堂 64 模拟器还允许管理视频、声音和控制器输入的不同插件，它们都有自己的优势和怪癖。一些游戏可能最好使用 Project64 附带的“Jabo 的 Direct3D8”视频插件。其他人可能用 [Glide64](https://code.google.com/p/glidehqplusglitch64/) 视频插件更好，你必须单独下载并安装。有些插件，比如 Glide64，还支持额外的功能比如 [高分辨率纹理包](https://lifehacker.com/bring-your-retro-games-into-the-modern-age-with-these-e-1516605379) ，非常酷。

安装可能因插件和模拟器而异，你可能不得不在多个模拟器和插件之间切换，以获得任何给定游戏的最佳体验。面对所有这些问题，我们不可能对任天堂 64 模拟器做一个全面的指导。但是每个人都必须从某个地方开始，Project64 的兼容性、功能和易用性使它成为我们的“起点”选择。

要查看任何给定游戏的最佳模拟器和插件配置，请查看 [这个(现已解散，但已存档)数据库](https://web.archive.org/web/20131103161616/http://bmgcl.atspace.cc/n64mgcl/N64ConfigList.htm) 。你也可以在各种仿真论坛上找到很多关于如何安装和配置不同插件的信息。谷歌是你的朋友！

### 特征

*   以 rom 的形式玩任天堂 64 的许多游戏
*   使用几乎任何 USB 游戏手柄玩游戏，并自定义按钮布局
*   在游戏中的任何地方保存和加载你的状态
*   支持视频、音频和控制器输入插件，以获得额外的功能和稳定性
*   从多种分辨率和宽高比中进行选择，以适应任何电视或显示器
*   调整各种图形设置，在电视或显示器上获得最佳画面(有些依赖于插件)
*   将高分辨率纹理添加到游戏中，以获得更好的高清体验(取决于插件)
*   使用内置的作弊软件和 GameShark 代码解锁额外功能

### 它擅长的地方

虽然“最好的”模拟器取决于你想玩什么游戏，但我们发现 Project64 是大多数人的最佳起点。与其他模拟器相比，它具有良好的兼容性，并且(相对)易于设置和使用。这可能仍然需要大量的配置，但至少你有 [一个充满信息的论坛](http://forum.pj64-emu.com/) 和一个布局良好的 GUI 来探索它的所有不同设置。

### 不足之处

Project64 没有接近完美的模拟，尽管其他任天堂 64 模拟器也没有。这意味着一些游戏可能有小问题，大问题，或者根本无法玩。有些游戏可能需要某些配置才能正常运行。

Project64 中最臭名昭著的是声音模拟，除非为该游戏正确设置了声音插件，否则可能会出现噼啪声或其他问题。其他模拟器没有这个问题，尽管 Project64 的其他优势(主要是易用性)超过了这个缺点。

还有:Project64 的最新版本是 2.2，但有些人可能会觉得 1.6 更稳定(两者各有利弊)。我们建议从 1.6 开始，如果您遇到问题，请升级到 2.2。开发仍在进行中(尽管很慢)，所以未来的版本有望全面改进。还要注意，verison 2.2 有一些捆绑广告软件，所以在安装过程中**一定要取消选中它**。

### 竞争

[**mupen 64 plus**](http://www.mupen64plus.org/)是 Project64 最接近的竞争对手。一些游戏可能在 Mupen64Plus 上运行得更好，而另一些在 Project64 上运行得更好。Mupen64Plus 也没有 Project64 偶尔出现的声音问题。然而，Mupen64Plus 有一个很大的缺点:设置起来要复杂得多。它只有命令行，所以你必须通过配置文件做所有的设置调整，你需要非常精通技术才能让一切正常工作。然而，Mupen64Plus 上的开发似乎更快一些，所以它有可能在未来领先。Mupen64Plus 在[**RetroArch**](http://www.libretro.com/)中也是可用的，如果你已经在为其他仿真器使用 retro arch，这很好。

[**1964**](https://code.google.com/p/emu-1964/) 是一款老旧且大多废弃的任天堂 64 模拟器。但是，它仍然比 Mupen64Plus 更容易使用和设置，并且它的运行速度一般更快，所以旧硬件的用户可能会发现它对一些游戏很有用。

对于任天堂 64 模拟器来说，有很多好的东西即将出现。[**EN64**](http://www.cen64.com/)还没有完全可用，但它的目标是成为一个周期精确的任天堂 64 模拟器，这将是一个巨大的游戏改变者。目前正在开发的 [**GlideN64** 插件](http://gliden64.blogspot.com/) 也将为 Project64 等其他模拟器的用户带来一些改变。但是开发是缓慢的，所以在这些事情发生之前，我们只能有一些不完美的(但是非常有用的)选择。
