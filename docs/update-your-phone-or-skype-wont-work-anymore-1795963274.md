# 更新您的手机或 Skype 将不再工作

> 原文:[https://life hacker . com/update-your-phone-or-Skype-wont-work-more-1795963274](https://lifehacker.com/update-your-phone-or-skype-wont-work-anymore-1795963274)

为了纪念 Skype 的最新更新，视频聊天和消息应用程序将停止对各种平台的支持。

Watch

据 [Techcrunch](https://techcrunch.com/2017/06/08/skype-retires-older-apps-for-windows-and-more-as-it-pushes-new-features/) 报道，十几个 Skype 应用程序和平台(包括 SkypeKit 驱动的手机)将于 7 月 1 日失去 Skype 支持，这意味着你将无法在某些设备上使用该服务。真扫兴。

我从来不支持放弃对旧操作系统的支持，尤其是如果维护它只需要很少的工作。另一方面，失去支持的操作系统都是多年前的产品或者没有得到充分利用，这意味着 Skype 完全停止对它们的支持可能更有经济意义。失去支持的操作系统包括 iOS 7 和更早版本、Android 4.0.2 和更早版本、黑莓 OS 7.1 和更早版本以及电视上的 Skype。

支持 SkypeKit 的设备也正在失去支持，尽管它的消亡是以 2013 年公司宣布停止支持这项服务为开端的。Linux 用户也失去了对旧版本 Skype 的支持，尽管如果他们想继续使用 Skype for Linux beta，他们应该使用它。

然而，消息并不都是坏的。Skype 的最新版本 [具有更新功能，使其更符合其竞争对手，如 Facebook Messenger、WhatsApp 和 Snapchat 等消息应用程序。它甚至有一个故事风格的短消息功能，叫做亮点。](https://www.skype.com/en/download-skype/skype-for-mobile/)
