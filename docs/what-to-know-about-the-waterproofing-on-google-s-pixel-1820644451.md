# 关于谷歌的 Pixel 2 和 Pixel 2 XL 上的防水应该知道什么

你的新智能手机可能缺少一个耳机插孔，但它可能有一个很酷的新功能来弥补它:防水。谷歌的 Pixel 2 和 Pixel 2 XL 智能手机是搜索引擎巨头的最新智能手机，它放弃了 3.5 毫米的插孔(提高了防水工作的效率)，转而支持防尘和防水。但是，防尘防水设备的分类方法不止一种。以下是如何在不使用一桶沙子和一根橡胶软管进行测试的情况下，确定你的新智能手机的耐用性。

### 什么是 IP 等级？

谷歌 Pixel 2，以及苹果的新 iphone[三部曲](https://lifehacker.com/what-do-you-like-or-hate-about-iphone-x-1820179520) 和三星的 Galaxy Note8，都是防水防尘的。它们都有工业保护(也称为 IP)排名，表明异物(如灰尘、沙子和小工具)和水分(如暴露在雨中或一小时的海洋潜水)会或不会进入设备。越来越多的高端智能手机在其功能列表中增加了防水和防尘功能。

Pixel 2、iPhone X 和 iPhone 8 的防护等级为 IP67，而三星 Galaxy Note8 的防护等级为 IP68。IP 等级中包含的可选信息可以表明设备是否还能抵抗跌落和撞击等机械冲击，或其他不利条件(如高电压)。

### **这些评级的实际含义——以及如何解读它们**

#### **第一位:**

IP 等级中的第一个数字表示其固体颗粒防护等级。对于智能手机，固体颗粒防护等级为 6 意味着您的设备是防尘的，并经过吸尘过程以防止入侵。把你的手机带到沙滩上，给你的羽毛除尘技术拍些特写，或者在沙丘上自拍。等级为 5 表示您的设备仅具有一定程度的防尘性，如果灰尘、沙子或其他异物进入设备内部，设备仍应能正常工作。任何低于 5 的数字都意味着设备没有通过防尘认证，容易受到从蚂蚁、电线到手指等任何东西的侵入。

如果您的 IP 等级有一个 X，而数字应该是，这表明该设备没有经过灰尘或液体侵入保护测试，不应被视为适合这些条件。避开沙丘。

#### **第二位:**

IP 等级中的第二个数字表示其液体防护等级。如果你曾经拿着手机去维修，却发现由于潮湿的淋浴造成了一些水渍，你可以感谢你的设备无法阻止水进入。谷歌 Pixel 2 的液体防护等级为 7，这意味着它可以在早上淋浴后存活，并在一米深的水中全力浸泡 30 分钟，让你有足够的时间脱下裤子，在手机后跳入水中。

仅仅因为你的手机可以在游泳池里浸泡，并不意味着你应该立即开始拍摄 Aquaman cosplay 照片。水可能会影响触摸屏的操作方式，或者使其在浸没状态下无法操作。三星的 Galaxy Note8 的液体防护等级为 8，这意味着它至少可以在 1.5 米深的水中生存 30 分钟。制造商应指明 IPX8 设备适用的特定条件。无论液体侵入防护等级如何，如果您的智能手机落入一池盐水中，您应该用淡水冲洗它以防止腐蚀。

#### **附加字母:**

您可能会在设备的 IP 等级旁边看到额外的字母。字母“f”表示设备的耐油性，而字母“H”表示设备可以承受高电压。你可能不会在智能手机上看到这些，但其他更坚固的设备可能会有额外的字母来确定它们的耐用性。
