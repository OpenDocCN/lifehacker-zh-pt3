# 让我们为八月的健身挑战来个倒立

> 原文：<https://lifehacker.com/let-s-get-upside-down-for-the-august-fitness-challenge-1828029407>

我不会倒立。或者倒立。我甚至有点害怕把我的腿放在墙上。你和我在一起吗？让我们走出舒适区。如果你已经会倒立了，那么，我也会给你一些额外的挑战。



入门的一个方法就是试试 [墙头漫步](https://lifehacker.com/work-up-to-handstands-with-wall-walks-1792863049) 。你从双手和膝盖开始，双脚接触墙壁。把你的脚放在墙上，一次一只，看看你能爬多高。不骗你，我只爬到一半就退缩了。很难弄清楚我的困难有多少是精神上的，而不是肩部力量的缺乏。我希望在这个月的过程中，我会弄清楚。

另一个初学者的动作是把你的脚放在一个盒子或椅子上，倾斜你的躯干直到它垂直(头向下，屁股在空中，脚仍然放在盒子上)。你可以在 [这篇倒立俯卧撑](https://vitals.lifehacker.com/work-your-way-to-your-first-handstand-push-up-1781827660) 的介绍中看到这个技巧，但是对于我们这个月的目的来说，俯卧撑是可选的。

倒立的方法有很多种，所以选择一种作为你这个月的目标，每天练习。一些建议:

*   用上面的方法练习倒立
*   努力练习瑜伽头倒立
*   做其他的瑜伽倒置动作，比如肩站或车轮式，或者甚至是下犬式，如果那是你感觉舒适的地方
*   用手行走，如果只是站在那里对你来说太容易了

使用这些工具时，请确保安全！如果你对自己不确定，找一个教练或有经验的朋友来帮你，待在柔软的表面上，阅读其他与你选择的方法相关的安全指南。
