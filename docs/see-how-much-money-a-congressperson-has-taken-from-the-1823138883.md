# 你们的国会议员从全国步枪协会拿了多少钱？

> 原文：<https://lifehacker.com/see-how-much-money-a-congressperson-has-taken-from-the-1823138883>

我们正处于这样一个时刻，你读到的几乎所有东西都可能需要一两个事实检验，尤其是涉及到政客的时候。



当涉及到枪支时，这个问题似乎变得复杂了一些。

Chrome 扩展“ [Follow the Money](https://chrome.google.com/webstore/detail/follow-the-money/icakhelpmhobgbphgmjedeopoafhcmdk?authuser=1&ref=producthunt) ”取代了获得最多 NRA 资助的 20 名国会议员的名字，以包括每当你看到他们被提及时他们从枪支组织获得的钱数。

例如，只要提到他的名字，你就会看到“约翰(7，740，521 美元，来自全国步枪协会)”而不是“约翰·麦凯恩”。

这个想法是，无论你对枪支有什么看法，当你阅读一篇关于其中一个人的文章时，在网上提供这些信息可能有助于平衡你对这个故事的看法。

如果你只想知道你的国会议员从全国步枪协会得到了多少钱，但不想一直看到所有人，你也可以在这里 找到这些信息 [。(注意:链接文章中的信息是 NRA 当年的贡献，而插件会跟踪政治家一生的贡献)。](https://www.washingtonpost.com/graphics/national/nra-donations/?utm_term=.68dc0057395d)
