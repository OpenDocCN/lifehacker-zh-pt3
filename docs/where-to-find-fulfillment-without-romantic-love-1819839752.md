# 没有浪漫的爱情，哪里能找到满足感

> 原文：<https://lifehacker.com/where-to-find-fulfillment-without-romantic-love-1819839752>

“最近我对寻找爱情感到有点绝望，”Ask MetaFilter 用户 seraph9 在一篇名为 [“爱情的替代品”的帖子中写道](http://ask.metafilter.com/314703/Alternatives-to-Love) “你最有收获、最充实、最惊艳的非浪漫经历或事业是什么？”受访者列出了丰富多彩的活动，说明生活中除了工作和浪漫还有很多。



志愿服务是最受欢迎的建议。“为自己之外更伟大的事情做贡献是令人难以置信的回报，”用户雷诺上尉说。寻找能让你与其他人建立深厚关系的角色。许多人缺乏可靠的亲人的支持网络，所以你的情感可用性正是他们所需要的。

用户 fairlynearlyready 建议通过 [生命线给孤独的](http://solitarywatch.com/about-lifelines/) 的囚犯写信。用户 Wilder 建议在临终关怀医院或其他临终关怀机构工作。“许多人独自一人，或与家人疏远，没有人帮助他们有尊严地去世。”更多想法，用户 janey47 推荐《T4》[*我能帮上什么忙？*](https://www.amazon.com/gp/product/0394729471/?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/where-to-find-fulfillment-without-romantic-love-1819839752&asc_source=&tag=kinjalifehackerlink-20) 作者拉姆·达斯和保罗·戈尔曼。

友谊可能看起来显而易见，但是需要有意识的努力来区分它们，就像人们优先考虑寻找“那个人”一样随着年龄的增长，建立一个亲密的朋友群体变得越来越困难，尤其是当其他朋友开始成双成对，然后有了孩子。但是就像有更多的朋友一样。许多夫妇喜欢有单身朋友——他们是与其他夫妇的一个很好的休息——成为“家庭的朋友”可能是一个终身的充实角色。但是要明确告诉他们你是否想和约会对象或其他朋友“约会”。如果没有指导，夫妇们将不可避免地试图同化单身人士。

此外，和长期单身的人交朋友。你需要有像你一样的朋友，他们能跳出感情的圈子思考问题。总有一天，你会因为一个浪漫的伴侣而“失去他们”,但是他们也在和你一起承担这个风险。尽量组建一些大于两个人的密友群；这是你相对于长期夫妻的一个巨大优势，因为长期夫妻必须把他们的生活集中在另一个人身上。

把尽可能多的朋友活动带到你家或他们家。恋爱的另一个乐趣是在私人空间创造时间，但这不仅仅是一个浪漫的活动。走出酒吧和咖啡馆，在自己的空间里感受与朋友交谈的亲密。

集体活动，比如舞蹈班、乐队、唱诗班、教堂、读书俱乐部，甚至是玩龙与地下城&，让你更容易结识朋友，因为他们给共同的活动带来了直接的目的。当你去参加读书、喜剧表演或音乐会等活动时，看看这个场所或组织提供了哪些更长期的选择。

找一个定期聚会的小组。一段长期关系的部分乐趣，尤其是一旦你们住在一起，在于它是默认“开启”的。这消除了日程安排和不确定性的压力。常设团体承诺的工作方式类似；如果你们都已经把友谊作为一个固定的优先事项，你就不必担心找不到时间交朋友。

多年来，我的每周 D&D 小组像巡航控制一样为我们的友谊工作。用户 wwax 特别推荐以地下城主的身份主导一个游戏:“和一群其他人一起讲故事的感觉太神奇了。”

用户 Quisp Lover 说,冥想为让生活更加充实打下了基础。它可以颠倒对非浪漫满足的追求:“精神上的快乐不是升华的性；性是被压抑的精神愉悦。”

拥有一只宠物不能代替人类的爱，但它有很多好处。一个动物会给你无条件的爱，这是大多数人类做不到的。fairlynearlyready 说:“在我有了小猫咪之前，我从未感受过这种爱，是她给了我这种爱。”宠物完全依赖你，但它的需求没有孩子或伴侣复杂。

“下午晚些时候的梦幻酒店”的用户说，不要取消在前后的**约会。只要你诚实、开放，通过约会生活满足更多短期需求(你知道，比如性)，同时在其他地方找到你的长期需求并没有什么本质上的错误。“爱情可以排在第二位，把它的组成部分分门别类，而不是先寻找最终目标，先满足这些需求。”**
