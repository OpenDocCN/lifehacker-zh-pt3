# Windows 的最佳 Twitter 客户端

> 原文：<https://lifehacker.com/the-best-twitter-client-for-windows-5809448>

多年来，Windows 的 Twitter 客户端领域已经被淘汰，但仍有少数开发者在努力开发一些杀手级应用。Aeries 是一个令人印象深刻的 Twitter 客户端，能够通过通用的 Windows 应用程序满足专业需求。



更新:几个月前，我们之前最喜欢的 Windows 版 Twitter 客户端 TweetDeck 停止使用。我们已使用新的 Windows 应用程序更新了下面的选择。然而，如果你是 TweetDeck 的忠实用户，请查看下面的竞争部分，了解该应用的更多信息。

### ****特性****

*   **从多个帐户管理仪表板和推文**
*   **一次查看多个栏，包括时间线、提及、个人资料和列表**
*   **将列表组合成一列，以便在多个列表之间轻松切换**
*   **根据您的喜好自定义图像大小、字体大小和配色方案**
*   **过滤某些 Twitter 客户端发送的人、关键词、标签和推文**
*   **保存列中的读取位置**
*   **自定义触摸和鼠标手势**
*   **为以后的推文添加书签**
*   **保存自定义搜索，将搜索锁定到主屏幕**
*   **通用 Windows 应用程序适用于所有 Windows 平台。**

### ****擅长的地方****

**首先，Aeries 仅仅因为是一个 Windows 应用程序而获得了一些分数。正如我们在下面的竞争部分所讨论的，我们以前的 pick TweetDeck 仍然作为一个网络应用程序存在，它有自己的优势和劣势。如果你仍然对 TweetDeck 作为一个网络应用感到满意，你可以跳过这一步。然而，本地 Windows 应用程序提供了一些好处:它可以在你离线时下载推文阅读，它可以与 Windows 的通知中心集成，当 Windows 商店的 Xbox 集成到来时，你也可以在你的主机上使用它。如果这些事情对你很重要，Aeries 是一个显而易见的选择。**

**就像 TweetDeck 一样，你可以在 Aeries 中一次查看多个栏目，让你可以在一个视图中查看你的时间线、你的提及和你的列表。与 TweetDeck 不同，列分组更有意义。例如，不是每个列表都有一整列，而是有一个列表列。**

**在“列表”栏的顶部，您会看到一个列表转盘，允许您在同一栏中的列表之间快速跳转。它为可以大量使用它的社交网络节省了大量空间。同样，你可以在同一个空间中轻松地在你的提及、消息和转发之间循环，而不必为每一项添加单独的栏。还有一个侧边栏，可以让你快速跳转到不同的列表或视图。**

**Aeries 还允许大量的定制。您可以独立调整字体和图像大小，以充分利用屏幕的实际空间。你也可以根据自己的喜好调整手势。比如你可以设置单次鼠标点击查看推文详情，打开快速回复框，或者喜欢一条推文。这种定制对于高级用户来说很方便。搜索功能还允许您保存某些搜索参数，以便以后查找。您甚至可以将某些搜索固定在主屏幕上，以便轻松查看它们。**

### ****不足之处****

**这可能属于个人偏好，但 Aeries 的过滤选项适用于整个应用程序，而不是允许用户只过滤某些列或列表。如果您想在一个地方对每一列应用过滤器，那么应该在上面的部分中。然而，如果你像我一样，喜欢灵活地选择性过滤你的列表，这是一种拖累。从好的方面来说，Aeries 允许你设置过滤器的截止日期，这对于避免剧透非常方便，而不必记住你过滤了一个主题。**

**客观上不好的一面是，Aeries 缺乏安排推文的能力。对于管理多个 Twitter 账户的人来说——尤其是为了工作——这是一个相当大的不利因素。从多个账户发推文也需要切换仪表盘，而不是像 TweetDeck 那样简单地点击不同的账户缩略图。这是一个小问题，但是当你每天都使用它时，一个小烦恼可以变成一件大事。**

**Aeries 的价格点也是一个症结，但由于 Twitter 的令牌限制怪异，我们愿意容忍它。简单地说，如果太多人使用 Twitter 客户端，新用户就会被拒之门外。除了支付开发人员的辛勤工作之外，门口收取 3 美元的费用有助于确保注册并获得这些令牌的人至少是愿意付费使用它的人。**

### ****比赛****

**只要该公司继续坚持其令牌限制废话[，就很难为 Twitter 客户找到竞争对手。Aeries 是一款真正的 Windows 应用程序，并且拥有一些竞争对手没有的优秀功能。话虽如此，Twitter 拥有的](http://lifehacker.com/bypass-falcon-pros-twitter-token-limit-with-this-chea-754679503)[**TweetDeck**](https://tweetdeck.twitter.com/)**(免费)**在 Chrome 中仍然很好用。另外，你知道，这是免费的。TweetDeck 和 Aeries 在功能上非常相似，它们之间只有一些微小的差异。主要是，你可以通过更少的点击来安排和使用多个账户。**

**最大的缺点(对某些人来说)是 TweetDeck [现在只能在网络上运行，或者作为 Chrome 应用](http://lifehacker.com/tweetdeck-for-windows-is-officially-dead-1765530136) 。Chrome 可以很容易地将标签页作为独立的窗口运行，但对于所有其他浏览器来说，从一个专用应用程序转移到一个单独的 TweetDeck 标签页可能会很烦人。**

**如果你想要一个专用的 Windows 应用程序， [**官方 Twitter 应用程序**](https://www.microsoft.com/en-us/store/apps/twitter/9wzdncrfj140) **(免费)**也非常好用。它也比 TweetDeck 或 Aeries 简单得多。即使在桌面上，也只有一个分栏视图。你可以切换到查看你的通知或信息，并撰写你自己的推文，就像你在手机上做的一样。如果你不需要 Aeries 或 TweetDeck 的所有专业级功能，Twitter 的官方 Windows 应用程序令人惊讶地胜任。**
