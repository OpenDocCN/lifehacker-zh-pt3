# 为一周中的每一天设置不同的闹钟

> 原文：<https://lifehacker.com/set-a-different-alarm-for-every-day-of-the-week-1827493988>

每周三， [这个 YouTube 系列](https://www.youtube.com/watch?v=9K4-jllrPrE&list=PLy3-VH7qrUZ5IVq_lISnoccVIYZCMvi-8) 都会增加一个新的音乐视频，视频中一只尖叫的青蛙在说“今天是周三，我的兄弟们。”难道这些周三歌曲中的一首不会比“Radar (Default)”更能让你在周三早上醒来吗？你能做到的，伙计们。



## 最好的闹钟是你喜欢但从未听过的音乐

你应该 [永远不要用你最喜欢的音乐作为闹钟](https://lifehacker.com/dont-set-your-alarm-to-your-favorite-song-1821886411) ，因为一旦一首歌成为你的闹钟，它会在你听到它的任何时候让你紧张。就我个人而言，我喜欢使用新奇的音乐，比如《T4》、[、](https://www.youtube.com/watch?v=eT4shwU4Yc4)、的《这不是月亮》，或者、 [vaporwave Seinfeld](https://soundcloud.com/jetaimeabelard/seinwave) 、:不会出现在 shuffle 或咖啡馆里的音乐。

这种音乐的关键在于它有趣、朗朗上口，但它的*已经有点烦人了*，而且它甚至经常在 Spotify 或 iTunes 上找不到。它通常不会出现在你的慢跑播放列表中。这东西能发出很棒的铃声，也能发出很棒的警报。

或者，你可以使用你知道的“经典”歌曲，但不会真的放入你自己的音乐库。比如老歌，或者电影或电子游戏中的音乐，或者你通常跳过的深度剪辑。他们只需要打败“马林巴”或“海边”或你令人遗憾的 99 美分下载的“小黄人说香蕉”

## 每天都有不同的闹钟

想想这个:如果你的闹钟告诉你今天是什么日子，你醒来的第二秒钟就领先了。新奇的音乐和经典的老歌都能很好地告诉你今天发生了什么。考虑这个阵容:

*   星期一:手镯， [【狂躁星期一】](https://www.youtube.com/watch?v=SsmVgoXDq2w)
*   星期二:伊洛维马孔宁和德雷克， [《星期二》](https://www.youtube.com/watch?v=avFq9errZCk)
*   星期三:周日， [【周三】](https://www.youtube.com/watch?v=Oct2xKMGOno)
*   周四: [【周四】](https://youtu.be/hAFau7OGS1U?t=22)
*   周五: [真的要问吗？](https://www.youtube.com/watch?v=kfVsfOSbJY0)T3】
*   星期六:芝熙， [《公园里的星期六》](https://www.youtube.com/watch?v=PLiMy4NaSKc)
*   周日:U2， [《周日血腥星期日》](https://youtu.be/Yv5U0A10hrI?t=35)T3】

并非所有这些歌曲都可以在 iTunes 或任何你购买音乐的地方找到。也许你实际上并不拥有任何音乐，而是通过 Apple Music 或 Spotify 或(哈哈哈)Tidal 来播放。没关系，你仍然可以使用下面的应用程序将任何歌曲转换为音频文件。

## **如何将任何音乐变成闹钟**

如果你想用流行音乐作为你的闹钟，大多数平台都很容易做到。Android 和 iOS 允许你使用你的音乐库中的任何歌曲作为闹钟。*(澄清:这包括 iOS 上的 Apple Music。)*同样如此的还有 [Google Home 音箱](https://lifehacker.com/how-to-set-a-music-alarm-with-your-google-home-speaker-1822673391) 和 [亚马逊 Echo](https://gizmodo.com/amazons-echo-spot-is-a-good-as-hell-alarm-clock-1821375611) 。

如果你的歌曲被埋没在 YouTube 视频或 Soundcloud 流中，那么用网站 [Peggo](https://peggo.tv/) 或免费 app[4K YouTube to mp3](https://www.4kdownload.com/products/product-youtubetomp3)提取该音频的 MP3。这些是我们从 YouTube 上获取音频的两种最喜欢的方式。

如果你的歌曲在其他流或视频中，如 Spotify 或 AVI 文件，你仍然可以通过从任何应用程序捕捉音频来录制它。只需使用音频编辑器[Audacity](https://sourceforge.net/projects/audacity/)(Windows/Mac/Linux 免费)，或者免费试用 [Piezo](https://rogueamoeba.com/piezo/) (仅限 Mac)，这是一个简单的应用程序，专门用于从任何应用程序中捕捉音频。

然后把你的新唱片放进你选择的移动音乐库，你就大功告成了。或者你也可以每天早上醒来，对着一个向下尖叫的系统 [“醒来！”](https://www.youtube.com/watch?v=HrQsGeKN6qk)T3】
