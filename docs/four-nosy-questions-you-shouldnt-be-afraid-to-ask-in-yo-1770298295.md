# 求职面试中你不应该害怕问的四个八卦问题

> 原文：<https://lifehacker.com/four-nosy-questions-you-shouldnt-be-afraid-to-ask-in-yo-1770298295>

如果你像我认为的那样为面试做了充分的准备，你可能会有一长串 [问题要问](https://www.themuse.com/advice/51-interview-questions-you-should-be-asking) 。但是你可能也会担心哪些是可以提出来的，哪些会让你丢掉工作。



*本帖原载于* [*缪斯*](https://www.themuse.com/advice/4-nosy-interview-questions-youre-allowed-to-ask-because-the-answers-matter) *。*

是的，当我是一名招聘人员时，人们肯定会谈论令人不快的话题，但我也发现有些人会回避完全正常的询问，因为害怕 [看起来粗鲁](https://www.themuse.com/advice/4-ways-youre-being-rude-in-your-interview-without-even-knowing-it) 。

所以，为了让你得到你需要的答案，这里有几个问题，大声说出来是很好的。

### **1。这是一个新职位，还是你想填补这个职位？**

事情是这样的——不仅询问完全没问题，大多数招聘经理都会乐于分享细节，即使这会让他们不舒服。你为什么要提出来？如果这是一个新角色，这是公司处于成长期的好迹象(这很好)。如果这个职位是因为某人的离开而变得可用，问一些后续的问题也是可以的，比如为什么那个人已经不在这个职位上了。如果你对答案不满意(或者得知最近五个人在一年内都辞职了)，如果有机会的话，不去也没关系。

### **2。对这一角色的期望是什么——员工多久接受一次评估？**

你可能已经在问前半部分了，这很好。但是，你在问后半句时很可能会犹豫，因为你不想超越自己。但你应该这么做，因为最糟糕的感觉之一是不知道你的经理是否满意你的表现(不幸的是，这在工作场所仍然相对普遍)。

所以，除了获得 [内部消息](https://www.themuse.com/advice/22-interview-questions-thatll-get-you-the-real-inside-scoop-on-company-culture) 关于对该角色的期望，借此机会了解公司是否有定期的评估期来讨论绩效和薪酬。一些公司将这一过程正式化，而另一些公司则优先考虑全年的定期反馈。不管是什么情况，不要羞于提出来。



### **3。员工有哪些职业成长的机会？**

当你还没有工作的时候，询问职业发展可能看起来很可怕。当然，关注你的未来是很重要的。然而，如果你询问公司的其他员工如何 [提高他们的技能](https://www.themuse.com/advice/5-mostly-free-ways-you-can-advance-your-career-this-weekend) 甚至在组织内转移到其他职位，大多数人不会被冒犯。

事实上，这个问题向他们表明你有兴趣在那里呆很长时间，这在任何想要做出重大招聘决定的人眼里都是一个优势。虽然你应该避免谈论太多你将来想做的事情的细节(不需要讨论你经营自己公司的最终计划)，但是不要害怕询问职业发展机会。

### **4。是什么让你对加入公司感到兴奋？**

当一家公司想要雇佣你的时候，招聘经理会试着卖掉这个职位。*硬*。然而，如果你想知道更多关于是什么让这个组织成为一个很好的工作场所的信息，可以随意问你的面试官是什么让他从以前的工作跳到现在的工作。

当然，这个回答可能有点牵强，但它会为你提供更多的背景，让你了解他对这家公司的感觉。了解某人跳槽的思维过程不仅会帮助你做出决定(“哦，我希望和他走在同一条路上！”)，还要看他在那里工作开心不开心。如果他的回答看起来太做作，那对你来说是有价值的信息。

这个过程很艰难，如果你觉得任何失误都可能让你失去一份理想的工作，这是可以理解的。但是，你有这种疑问也是合乎逻辑的。虽然你当然不应该在这个过程的早期就提出任何关于薪酬或假期的问题，但你应该对提出上述例子感到满意。
