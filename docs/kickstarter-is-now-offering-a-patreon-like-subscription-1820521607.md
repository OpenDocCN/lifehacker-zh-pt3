# Kickstarter 现在提供类似 Patreon 的订阅服务

> 原文：<https://lifehacker.com/kickstarter-is-now-offering-a-patreon-like-subscription-1820521607>

众筹网站 Kickstarter 基于订阅的 Drip 可以为艺术家、边缘人和任何有创意的人提供另一种赚钱的方式。



滴滴的 最初是一项众筹服务，让音乐粉丝按月订阅艺术家和唱片公司，以换取提前获得音乐等额外待遇。Kickstarter 在 2016 年收购了该公司，使其免于关闭。

这次重新启动保持了这种订阅模式，但将其扩展到包括各种其他创意项目。有资格通过 Drip 获得支持的作品分为以下几类:视觉和表演艺术、电影和视频、出版、设计和技术、音乐、漫画、食品和工艺以及游戏。每一类都相当宽泛，很难想象*和*有什么不同。



像 Patreon 一样，Drip 允许创建者分配订阅层级，并向每一层级的用户奖励额外津贴。如果你曾经向 Kickstarter 项目捐款，你将能够使用相同的帐户信息来支付 Drip 订阅。创作者的点滴页面的早期订户将被视为“创始成员”创始人能得到什么由创造者决定，但 Kickstarter 建议创造者为早期订户提供“提前加入的特殊奖励或地位”如果你正在寻找一个新的地方来为你的两个白人交谈喜剧播客获得一些资金，滴滴可能是一个好地方，但目前它只接受邀请:你必须注册该网站的时事通讯才能被考虑。
