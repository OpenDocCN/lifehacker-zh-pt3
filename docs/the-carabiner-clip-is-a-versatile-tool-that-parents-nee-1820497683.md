# 登山扣夹是父母需要的多功能工具

> 原文:[https://life hacker . com/the-carabine-clip-is-a-versatile-tool-than-parents-nee-1820497683](https://lifehacker.com/the-carabiner-clip-is-a-versatile-tool-that-parents-nee-1820497683)

攀岩者一直以来都很喜欢这款深受喜爱的登山扣，这是一款多功能性无与伦比的工具。但你知不知道这也是婴儿登记的主要内容？大约 10 年前，一些营销天才将无处不在的金属夹重新命名为一个可怕的名字: [【妈咪钩](http://www.themommyhook.com) 。它是什么:呃，仍然只是一个大的登山扣夹。它能为父母做什么:实际上就这么多！如果你有这些东西中的一个——不管你是否选择用妈妈化的名字来称呼它——这里有一些方法来使用它，让你的生活变得稍微轻松一点:

*   **婴儿车或购物车机库:**打开夹子，在你的购物袋和钱包上滑动，然后把它钩在提手上。(只要保证重量不会让婴儿车向后倾斜就行。)在拥挤的地方，当挂在婴儿车时，它还可以作为小孩的把手，这样他们就可以呆在附近。
*   **高脚椅挂钩:**当你想在餐厅把包从地板上拿下来时。
*   **卫生间地摊帮手:**如果你有尿不湿包、外套等物品，可以把它们都附在夹子上，挂在门钩上，这样你的所有物都不会碰到讨厌的地板。
*   **购物袋提篮:**它可以让你 [舒适地拿着几个袋子](https://lifehacker.com/comfortably-carry-multiple-plastic-shopping-bags-with-a-5949530) 而不用把手伸进你的手里。当你一只胳膊抱着一个婴儿，而你宁愿用另一只手拎着 25 磅的东西，也不愿意多次旅行时，这是一个很好的选择。
*   壁橱收纳器:在壁橱挂几袋玩具。

Watch

把一个(或几个)这样的东西放在你的包里，以便随时携带——或者，如果你想看起来很酷，把它们挂在你的腰带环上。你可能没有抱石，但你在父母的落基山脉，这仍然是一种坏蛋。
