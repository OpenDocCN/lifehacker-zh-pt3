# 微软 Surface 书评:太好了，我可能会换回 Windows

> 原文：<https://gizmodo.com/microsoft-surface-book-review-so-good-i-might-switch-1737680767>

这里有一个疯狂的。在从原始池中爬出来的一系列小工具中，微软雄心勃勃的 Surface Book 感觉像一个标点符号。这是一个出乎意料的转变，至少对我来说，这是一台如此诱人、习惯正在改变的电脑，以至于我在想——嘿，Windows 怎么样？



### 这是什么？

微软的第一台笔记本电脑。它的设计很巧妙，这使得它的 13.5 英寸，3000 x 2000 显示器(267 dpi)显示器可以从刚性键盘底座上拆下来，这样你就可以像使用 mondo 平板电脑一样使用显示器。基本款搭载最新的英特尔 Skylake Core i5 处理器，售价 1500 美元——它基本上是一款 [Surface Pro 4](http://gizmodo.com/microsoft-surface-pro-4-review-i-love-it-but-not-for-1737565503) ，采用了更加花哨的工业设计。你可以将它升级到配备独立 Nvidia GeForce 图形处理器的 Core i5，起价为 1700 美元，然后再升级到配备 Nvidia GPU 和 1TB 存储的 i7，价格为 3200 美元。

这篇评论的大部分内容是指 1500 美元的基本配置，但是，我们也对配备 Nvidia 显卡的 2100 美元的 Core i7 型号进行了一些测试。

### 为什么重要？

Surface Book 的发布给消费电子领域带来了紧张的惊喜，但它并不是凭空而来的。多年来，微软一直试图扭转局面 。Surface Book 的高价和精心设计似乎是对优秀硬件和软件同等重要的彻底认可。

你可以将这款新笔记本电脑追溯到三年前推出的第一款 [Surface 平板电脑-笔记本电脑混血儿](http://gizmodo.com/microsoft-surface-pro-review-too-much-future-5981224) 。自从那场灾难和与 Windows 8 同时出现的以来，MSFT 已经走了很长一段路，制造出人们真正想用的设备和软件。微软最近加倍努力，通过 [收购诺基亚](http://gizmodo.com/breaking-microsoft-is-buying-nokias-phones-business-1243005211) ，这几年来制造了引人注目的 Windows Phone 手机，却被微软极简的移动操作系统所拖累。更重要的是，微软刚刚推出了 Windows 10， [这是一款经过大幅改进的操作系统，旨在适用于所有小工具外形](http://reviews.gizmodo.com/windows-10-the-gizmodo-review-1720872266) 。这是一个单一的手机操作系统——新的微软 Lumias 将于 11 月推出——电脑、平板电脑，甚至是该公司的下一代 Xbox One 游戏机。这是一个引人注目的操作系统，在 Surface Book 发布时，它已经安装在 1.1 亿台设备上。

那么，这给 Surface Book 这个古怪的东西留下了什么呢？天啊，该死，Surface Book 太棒了，与众不同。它功能强大，价格昂贵。它的设计让你思考计算机能为你做什么——而不是仅仅满足于你需要计算机做什么。您的笔记本电脑应该是一个巨大的平板电脑，配有行业领先的显示屏，并且您应该能够使用完美的手写笔在上面书写。不妥协。这是 Surface Book 认为你应该对一款设备提出的要求。

这是一个令人向往的概念，对用户和公司本身都是如此。坦率地说，1500 美元的笔记本电脑永远不会到达微软的大部分客户手中，他们中的大多数人可能会使用第三方制造商生产的低于 1000 美元的 Windows 10 电脑。但它是一个图腾，代表着任何人都可以接受的想法。作为微软雄心和未来的一种表达，Surface Book 揭示了一种近乎乔布斯式的对细节的专注。如果 Surface Book 作为一种可行的概念替代物成功取代苹果已经证明的陈旧现状，那么，未来将会看起来大不相同。

当然，这样一个雄心勃勃的设计有很多潜在的陷阱。有很多活动部件。一次失败就能毁掉整个竞选。

### 设计

Surface Book 如果不是 Windows 10 的一个过度设计的容器，那它什么都不是。

从一开始，它看起来就像一个闪亮的大镁翻盖，但正如我们所知，这款笔记本电脑是一个可转换的，整个事情都是因为一个完美的铰链设计。这一切都与铰链有关，它由燕尾榫连接在一起的窄条组成。这种机械装置开合时没有一点嘎吱声。它的设计令人感动。

使用键盘右上角的一个小释放按钮，屏幕就会分离，成为独立的平板电脑。值得注意的是微软是如何实现这种可转换格式的。显然，为了使显示器可拆卸，它必须包含设备运行所需的所有元件，包括处理器、闪存、电池等。

*释放按钮位于键盘的右上角。按住它一段时间，你会听到显示屏发出“自由”的滴答声。*

然而，键盘作为一个填充内脏的地方不会浪费。在基本型号中，键盘装有额外的电池。在更昂贵的图形版本中，键盘装有独立的 Nvidia 图形处理器。

Surface Book 整体上最好的对比是 MacBook Pro，但没有键盘，它与苹果的 iPad Pro 非常相似。Surface Book 的显示屏比苹果的 12.9 英寸巨型平板电脑略大且略重。显示屏很大——13.5 英寸令人满意的金属和玻璃，600 万像素。3000 x 2000 的分辨率相当于每英寸 267 像素，而苹果的分辨率为 265 像素。这可能是我在电脑上见过的最漂亮的显示。从技术角度来看，它的额定对比度是屏幕可以表现的最宽明暗范围的指标，让所有其他竞争对手望尘莫及。

从严格意义上的独立屏幕开始，你也可以将显示屏*向后*重新连接到键盘上，这样它就可以折叠起来，显示屏朝外。微软称之为“剪贴板”位置。当你像这样把它抱在怀里时，我有点想象自己在商店里忙着盘点存货，或者在医院里巡视。

这就是重新设计的 Surface Pen 发挥作用的地方。银色触控笔在美学和可用性方面已经从最初的版本走了很长的路。也许这次最重要的增加是它的磁性，所以它可以卡在显示器的侧面。

### 使用它

Surface Book 的前瞻性设计面临的挑战是试图同时满足所有需求。那么，在这些不同的可能用例中，它是如何成立的呢？

#### **作为笔记本电脑用于工作……**

尽管设计应该使 Surface Book 成为一个多功能的发电站，但我主要是将这款设备用作笔记本电脑，在这种传统模式下，它基本上是一个完整的梦想，只有几个警告。

首先，3:2 的宽高比确实为你提供了比 MacBook Pro 的 16 x 10 显示屏更多的观看空间。此外，i5 配置可以流畅地执行基本的互联网浏览、网飞观看和轻微的 photoshopping 类型的行为。电池寿命非常长——我经常在不充电的情况下工作 6-7 个小时。 [感谢 Skylake！](http://gizmodo.com/intels-new-skylake-processors-what-you-need-to-know-1728020921)T3】

键盘咔哒咔哒地快速移动。这里没有抱怨，但我是一个相当马虎的打字员，所以如果它阻碍了我，我可能甚至不会知道。我会说，当你的手指在琴键上跳舞时，我完全被琴键发出的清脆的低音迷住了。

此外还有负面影响。第一个是不一致的触控板，跳动不定，经常过于敏感或不够敏感。在使用 Surface Book 近两周之后，我已经习惯了触控板的不稳定行为。但这不是你想要描述一个小工具的方式，尤其是一个理论上是万能的超级小工具。

有几次，我最终连接了一个外部鼠标，因为我对它非常失望。好消息是，微软已经承认了这个问题，并表示将在 11 月初推出固件补丁。性能太差了，所以我建议在你确定固件实现了它的性能承诺之前，不要做任何购买决定。

另一个困难，尽管是一个小问题，是灵活的设计实际上使笔记本电脑有时有点棘手。当你把笔记本电脑放在桌子上，或者你坐着的时候，电脑可以完全平放在你的膝盖上，使用起来是一种乐趣。不幸的是，大多数时候人们并不是这样使用笔记本电脑的。如果你蜷缩在机场的角落里，躺在床上，或者摊开在沙发上，由于 Surface Book 的屏幕重量分布，你会遇到问题。MacBook Pro 将所有重量都放在键盘底座上，而 Surface Book 则放在顶部。

#### **作为一个大喇叭平板电脑…**

当从键盘上拆下来时，这个漂亮的显示屏看起来更加引人注目。只要准备好用两只手套拿着它，因为 13.5 英寸的平板电脑是一个大家伙。取决于你到底想做什么，这可能有点麻烦。当你躺在床上时，iPad 等较小的平板电脑可以长时间支撑，而 Surface Book 的重量为 1.6 磅(不含键盘)，听起来并不算多，但如果你试图阅读长篇杂志文章或观看电视节目，你就会真正开始感觉到这一点。

简而言之，这款笔记本电脑/平板电脑的混血儿绝对更适合膝盖。

在 i5 配置上，您不会损失任何性能，因为处理器完全包含在计算机的上半部分。在采用独立显卡的配置中，当您断开显示器时，您显然无法获得留在键盘底座中的处理器的优势，因此不要期望在旅途中也能享受台式机级别的游戏。

#### **作为笔记设备…**

诚然，使用手写笔在显示器上记笔记远不是我的自然冲动，但 Surface Book 和新的 Surface Pen 是一种使用乐趣。微软将玻璃的厚度削得如此之薄，以至于对你的眼睛来说，表面终于开始感觉你实际上是在显示器上写或画像素了。

其实很直观。上周末，我的一个有艺术倾向的朋友拿起手写笔，在几秒钟内就在微软的 OneNote 应用程序中画了一幅小图。感觉完全自然。也许微软在这里需要克服的最大障碍是像我这样的人不得不在玻璃上拖动金属的本能厌恶。感情上感觉不太对，但是使用手写笔的物理体验已经赶上现实了。换句话说，我*可以*使用它，但我还不确定我是否真的想这么做。

#### 图形呢？

为了更好地衡量，我尝试了 Surface Book 的独立显卡配置，看看它能做什么。

显然，这不是一款游戏笔记本电脑，但如果你想玩游戏，这款 2100 美元的笔记本电脑能跟上吗？我玩了几个图形密集型游戏，如《古墓丽影》，发现它们在高帧率下运行良好，图形设置为中等质量。然而，在高质量下，性能变得相当不稳定。所以你*可以*玩游戏，但它不能处理最大 11 的图形。

独立显卡更有可能在设计环境中使用。我们让 Gawker 动画师 Devin Clark 将他的巨型 Wacom Cintiq 显示器连接到 Surface Book 上，看看它是如何工作的。Surface Book 自己为巨大的触摸屏供电没有问题，然而，当电脑屏幕和 Cintiq 都工作时，事情变得相当滞后。所以它有一些力量，但微软并没有把奇迹硬塞进那个纤薄的设计中。

### 喜欢

漂亮的展示。难以置信的电池寿命。创新的多功能设计。

### 不喜欢

垃圾触控板。

### 该不该买？

Surface Book 并不完美，从严格的性能角度来看，它是一台昂贵的电脑。考虑到其基本配置，Surface Book 只是一台 900 美元的 Surface Pro 4，配有刚性键盘和增强的显示屏。在 Windows 10 平台上，它在价格或设计上没有竞争对手，这一点让它感觉几乎出奇地过分。尽管如此，数百万人多年来一直在为苹果电脑支付溢价。你想要高级视窗？微软现在有你正在寻找的东西。

当然，由于其可爱的显示屏和令人印象深刻的设计，使笔记本电脑可以迅速转变为哥斯拉平板电脑，这款设备的雄心承诺比他们在实用性方面提供的更多。而且也有缺点，最明显的是双层床触控板。然而，如果微软像它承诺的那样解决了这个问题，你将会得到一台你不会后悔的昂贵的机器。

至于是选择基本款还是考虑 1700 美元(对于初学者来说！)采用独立 Nvidia 显卡的型号，答案是...也许吧？再说一次，对于一台性能不佳的电脑来说，这是一大笔钱。你将支付部分设计费用。你有没有想过你会这样说微软的一个小工具？哇，微软*设计了*这个东西。

我喜欢它。
