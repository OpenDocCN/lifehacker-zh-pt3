# 你应该担心天坑吗？

> 原文：<https://lifehacker.com/should-you-be-worried-about-sinkholes-1797655549>

佛罗里达州目前有一个饥饿的天坑，在过去的一个月里吞噬了七户人家——这是过去 30 年来最大的一次。这是一个令人不安的景象，但是你到底应该担心多少呢？

### 什么是天坑？

美国地质勘探局水科学学院 解释的天坑 [，是一个没有自然外表面排水的地面区域。随着降雨，水积聚在天坑中，然后排入地下，侵蚀石灰岩、碳酸盐岩、盐层和其他微溶基岩，形成下方的空间和洞穴。最终，地下没有足够的支撑力支撑地表，地面就会坍塌。](https://water.usgs.gov/edu/sinkholes.html)

天坑的大小从几英尺宽到数百英亩不等，通常超过 100 英尺深，这在有房屋和道路的地区具有难以置信的破坏性。一些天坑很好地保留了水，成为自然形成的池塘。

### 它们多久发生一次？在哪里？

我们不知道它们发生的频率，因为没有足够的关于自然现象的数据。许多发生在农业地区或没有任何人跟踪的地方，也没有国家天坑数据库可供检查。

尽管如此，我们知道它们最常发生在哪里。在地表以下的岩石由石灰石、碳酸盐岩和盐层组成的地区，天坑是最常见的。佛罗里达州、德克萨斯州、阿拉巴马州、密苏里州、肯塔基州、田纳西州、宾夕法尼亚州、纽约州东部、印第安纳州、明尼苏达州东南部以及南达科他州、怀俄明州、新墨西哥州和俄克拉荷马州的部分地区。

### **天坑的预警信号有哪些？**

天坑形成非常缓慢，因此很难预测，可能根本没有任何预警信号。也就是说， [留意你家和其他建筑](http://news.nationalgeographic.com/news/2013/08/130812-florida-sinkhole-disney-world-explainer-urban-science/) 地基上新出现的裂缝，并留意让你的门无法正常关闭的歪斜门框。其他迹象包括你院子里的一棵树突然开始向一边倾斜，外面的地面出现裂缝。如果你看到这些东西，请避开该区域，不要靠近地面上的任何裂缝或印痕。联系可以安全调查该区域的人。

### 保险包括损害赔偿吗？

不幸的是，自然出现的天坑被认为是“地面运动”，这将它们与地震归为一类。这意味着你的标准房屋保险计划 很少覆盖天坑。它可能是你的提供商提供的一个可选的附加服务，但如果不是，你将不得不购买一个单独的天坑特定的保险政策。

然而，佛罗里达州法律规定，房屋保险公司必须为居民提供“灾难性地面覆盖物倒塌”的保护所以，如果你是一个拥有房屋保险的佛罗里达居民，你可能已经被保险了。但为了安全起见，最好仔细检查一下。如果你住在一个以前有煤矿的地区，你也可以得到煤矿塌陷保险。检查你所在州的保险部门，看看你能得到什么样的保险。

### 你应该担心天坑吗？

不一定，但要看你住在哪里。大多数天坑不会导致死亡，但这是有可能的——特别是如果塌陷在没有任何警告的情况下就发生在你的下面。如果你的家是上述任何一个地区，确保你至少有天坑保险。

即使你在这些地区开车，你也应该时刻注意警告信号。许多伤亡发生在人们 [不小心驾驶进入道路](http://news.nationalgeographic.com/news/2013/08/130812-florida-sinkhole-disney-world-explainer-urban-science/) 的天坑中。不过，在大多数情况下，保持警惕，确保你的 [时刻准备着应对紧急情况](https://lifehacker.com/9-kits-you-should-have-in-your-home-to-prepare-you-for-5941374) ，不要太担心地球会把你吞掉。
