# 问那个咸咸的女服务员:厨师们讨厌我的全熟汉堡要求吗？

> 原文:[https://the take out . com/do-chefs-hate-my-good-done-burger-request-1827030289](https://thetakeout.com/do-chefs-hate-my-well-done-burger-request-1827030289)

> 嘿咸，我住在一个美食友好的城市，允许我爱汉堡的心(哲学上，而不是身体上)几乎每周都尝试一个新的汉堡店。我不是特别挑剔，喜欢尝试特色汉堡。如果有一件东西我不是特别喜欢，我通常会尝试一下，看看他们是否能改变我的想法。也就是说，我确实有一个一贯的要求，我想知道我会因此受到多少评价，或者这是否会被认为是粗鲁的。
> 
> 我需要汉堡做得熟透。真的。我没有任何医学诊断需要这个，但任何一种粉红色都让我恶心，我真的无法忍受。当我要求做得很好时，一些厨师会做得五分熟，希望能说服我(我想)，这让我羞愧地吃了汉堡的外环，离开了粉红色的中心。我知道这很幼稚，对我来说很糟糕，也是对烹饪的轻视。我只是不能忍受它。
> 
> 我的问题是，厨师对这样的要求有多不满？我不会坚持，也不会把东西退回去(除非是非常罕见的情况)，通常我甚至会为自己的选择道歉。有些地方甚至在他们的菜单上放了一个免责声明，声明如果一道菜被要求做得很熟，厨师不对味道负责，即使这样，我还是胆怯地要求我的菜也要做得很熟。这是在侮辱厨师吗？如果我吃了一个半生不熟的汉堡，那这是故意的而不是真正的错误的可能性有多大？我是怪物吗？
> 
> 真诚地，烧焦了

亲爱的烧焦的，

为了节省评论者的时间，我首先要说的是:“如果汉堡做得很好，你怎么会喜欢它们呢？!"

Watch

但我不是来羞辱你的。你做你的，亲爱的。出去吃饭就是这个意思吧？我的意思是，除非你坐在某个铺着白色桌布的地方的主厨桌前，否则你就是在餐馆里吃你喜欢的东西。所以如果你喜欢做得很好的木炭牛肉煤球，那就是你喜欢的。

只是不要指望厨房会对此感到兴奋。我怀疑，当你的汉堡没有达到你的要求时，并不是因为厨师是邪恶的，他们想和你捣乱。这可能是因为他们习惯于做五分熟的汉堡，所以即使是五分熟对他们来说也太“老”了。也许一个人做得好就是另一个人的中等程度...这些不是一成不变的。厨师希望你的汉堡味道好，对他们来说，这可能意味着偏粉色。

所以，是的，也许他们对你有点看法。但是，除非餐馆有某种政策反对做得很好的汉堡，否则你应该按自己喜欢的方式点汉堡。你不是怪物——我把这个词留给了我的前岳母。在最坏的情况下，你对肉类温度的品味有问题，这算不上犯罪。如果你的汉堡被退回来，里面有粉红色，你有权利退回去。毕竟，将一个汉堡煮至熟透比将一个熟透的汉堡变回三分熟要容易得多。

但我真的希望有什么方法能让你相信粉色汉堡的乐趣。我会念一串念珠，希望你有一天能看到光明。
