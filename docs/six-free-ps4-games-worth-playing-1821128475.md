# 六款值得一玩的免费 PS4 游戏

> 原文:[https://kotaku . com/six-free-ps4-games-value-playing-1821128475](https://kotaku.com/six-free-ps4-games-worth-playing-1821128475)

多亏了 PSN 商店里的一些免费游戏，你不用花一分钱就可以在 PS4 上玩得很开心。这些免费产品中的一些可以媲美你对一款成熟的 60 美元游戏的期望，如果不是超过它们的话。以下是我们挑选的最值得玩的 F2P PS4 游戏。

### **堡垒之夜**

在陷入开发地狱之后，*堡垒之夜*已经成为今年最热门的游戏之一——这是有充分理由的。虽然主要的僵尸塔防模式需要花钱，但免费的皇家战役模式值得下载*。*

*许多人对明显的*战场*灵感持怀疑态度，但这款游戏已经发展成为一款出色的单机体验。是的，你出生在一个岛上，在那里你拼命寻找武器和资源，希望能在与其他 99 名玩家的战斗中幸存下来，但是*堡垒之夜*已经为这个公式添加了自己的曲折。你可以建造避难所或者可笑的装置。你可以用发射台跳到空中，到达新的区域，或者攻击毫无防备的敌人。你可以为自己制作一个藏身的灌木丛，这可能会导致各种各样荒谬的绑架。而 [现在又有了狂野的新 50 vs 50 模式。](https://kotaku.com/fortnite-battle-royale-s-new-50-vs-50-mode-is-awesome-1821128043) 在所有这些新增内容中，Epic Games 也不断表明，他们在考虑平衡和 bug 修复时会倾听玩家的意见，同时会就这些变化进行沟通。堡垒之夜的无忧无虑的氛围，加上健康的、不断增长的玩家基础，让 BR 值得一玩。*

### ***Gwent***

*炉石在数字卡牌游戏中发起了一场复兴，但是很少有人真正脱离了启发它的*魔法:聚集*模式。然而, *Gwent* 有。在《巫师 3》迷你游戏转变成的独立卡牌游戏中，每场比赛都要经过三轮和六排战场。在一场比赛中，你与放置牌的对手交换回合，看谁能获得更多的分数。一些卡片，像名义上的巫师自己，提高你的分数，而其他的则产生天气效果或释放伤害你对手的法术。数百种不同的卡和五个派别可供选择，游戏促进了许多不同的游戏风格，所有这些都通过华丽的艺术和全语音的俏皮话赋予了生命。虽然官方的游戏仍在测试阶段，但实际上它已经足够平衡和完美了。如果你想的话，你可以花钱买新的卡牌，用游戏给你的卡牌开始，然后从那里磨出更多的卡牌，这是一个完全可行和令人满意的方法来建立一个收藏库* 

*-伊森·加奇*

***布拉柴维尔***

*平台战士*战斗大厅*不是*超级粉碎兄弟*。，却搔到了同样的痒处。玩家在平台上与原创战士对决，比如赛博朋克枪手巴拉扎或亡灵战士尼克斯。任何有《T4》超级粉碎兄弟肌肉记忆的人都可以毫不费力地学会*争吵大厅*，但这个游戏也足够欢迎新手加入。不是每个角色都有自己的动作，战士们用斧头或镰刀之类的武器战斗。没有过于复杂的地图。此外，扣球对手是超级容易的，因为球员可以跳三次，很多战斗发生在舞台下——这种技术通常是为经验丰富的*扣球*球员保留的。只需 0 美元，任何人都可以在本地或网上与六个角色轮流竞争。* 

*-塞西莉亚·达娜斯塔西奥*

### ***让它死去***

*系好安全带，因为这是一个 Goichi Suda 游戏。*任其自生自灭*是一只充满个性、热血、滑板骷髅的奇怪地牢爬虫。但是不要让流行美学欺骗了你。*让它死去*是一个残酷的游戏，有着类似罗格的元素，你要穿越一个怪物出没的地牢。每一个生命开始时，你或多或少都是赤裸和脆弱的，但随着你的进步，杀死敌人和找到随机的战利品，你会稳步成为一个更有能力的战士。诀窍在于，你可能会在一瞬间失去一切，然后不得不重新开始，再次爬上队伍。如果你设法走得足够远，你将不得不面对自己疯狂的尸体——也就是说，假设其他玩家的尸体没有先杀死你。*

### ***帕拉丁***

*思考*圣骑士*的最好方式是，它是一个类似于*看守*的射手，但是更像 MOBA。你可以期待类似的角色类型，除了在这里，一旦你选择了一个冠军，你就不能改变它。每个冠军都有一张影响能力的卡，当你通过一场比赛时，你可以获得货币来进一步升级你的技能。你仍然会获得积分和移动有效载荷，但是*圣骑士*比*看守*等游戏对你的游戏方式提供了更多的控制。* 

### ***战架***

*几年前作为一个不起眼的免费游戏射击游戏开始，已经成功地成为最引人入胜的多人游戏盛会之一。Warframe 的中心战役跨越了数以吨计的不同星球和任务，所有这些结合在一起构成了一个不稳定的太空漫游，这是不可抗拒的怪异，就像特瑞·吉列姆导演了一部*命运*游戏。有一大群半机械人克隆体、大型企业、受感染的变种人和机械外星人，所有这些人你都可以用空间魔法射杀、刺伤或击昏。每个任务(你将会玩上百个)最终都是为了收集材料和战利品来制造更厉害的武器。幸运的是，游戏的跑酷元素和流畅的战斗让它不会变得太单调。今年， *Warframe* 还增加了 Eidolon 平原，这是一个很大的开放世界区域，除了所有常见的“杀死那个家伙以获得奖励”之外，你还可以在那里钓鱼、采矿和与其他玩家一起打猎。如果你曾经被困在一个荒岛上，这将是你的游戏。*

*-伊森·加奇*
