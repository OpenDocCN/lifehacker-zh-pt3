# 笨重的机器人桌面

> 原文:[https://lifehacker.com/the-hulking-robot-desktop-1782903347](https://lifehacker.com/the-hulking-robot-desktop-1782903347)

贾科莫的 Windows 10 桌面没有太多的皮肤或移动部件，但结合壁纸，它非常引人注目。这里是他如何设置这一切，你也可以。

Watch

尽管是 Windows 10，但 Rainmeter 在上面仍然工作得很漂亮。如果你不熟悉 Rainmeter，这里有一个方便的入门指南(虽然有点过时)，可以帮助你制作你的第一个定制桌面。

从那里，这里是你需要的:

*   [壁纸](https://www.desktoppr.co/wallpapers/902)
*   用于 Windows 的 [雨量计](http://www.rainmeter.net/) 系统调试和监控工具
*   雨量计的 [明文皮肤音乐播放器的](http://redsaph.deviantart.com/art/Cleartext-for-Rainmeter-v3-2-28-Jun-2016-519796161)
*   [风华 2 皮为雨量计](http://lilshizzy.deviantart.com/art/Rainmeter-Elegance-2-244373054)为时间和日期显示
*   [线 2 皮为雨量计](http://rasylver.deviantart.com/art/Lines-2-0-154719861) 为系统监视器
*   [夜帘时钟皮为雨量计](http://murasaki55.deviantart.com/art/Night-Shade-Clock-150267470) 为模拟时钟

大概就是这样。没有太多的皮肤，当然，如果你的桌面不需要三个时钟，你可以省略其中的一两个来创建你自己的外观。如果你喜欢这个造型，去贾科莫的 Flickr 页面(链接如下)或者 [去他的个人博客](http://lifehacker.com/thenerdystudent.blogspot.com) 让他知道你喜欢他的作品！
