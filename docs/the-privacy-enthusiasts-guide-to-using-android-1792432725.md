# 隐私爱好者使用 Android 指南

> 原文：<https://lifehacker.com/the-privacy-enthusiasts-guide-to-using-android-1792432725>

从当地骗子到政府机构，每个人都试图获取你的数据，这是加强你的隐私游戏的最佳时机。幸运的是，Android 上有大量的选项可以保护你的信息、文件和手机的安全。



在我们开始之前，我们应该指出这一点:使用智能手机总是有风险的。尤其是运行谷歌服务的公司。你可以使用这些技巧和应用程序来保护你的一些通信，但只要你使用 Android 手机，你就永远不会完全脱离网络。不过，这并不意味着你必须让攻击者轻松应对。

### **更改这些系统设置以保护您的隐私**

刚拿到手机的时候，是开始保护自己的好时机。在设置过程中，请确保禁用任何要求跟踪数据的选项。在那之后(或者如果你已经设置了你的手机)，你可以也应该采取一些预防措施。我们推荐以下列表中的所有产品，但它们都有一些便利性方面的牺牲，所以请自行决定您需要哪些产品:

*   **设置强字母数字密码。** Android 给你提供了使用图案或 PIN 来锁定手机的选项，但为了安全起见，你应该使用 [一个强字母数字密码](https://lifehacker.com/this-is-how-you-should-secure-your-iphone-5914602) 。打开设置应用，前往安全>屏幕锁定。设置包含数字和字母的密码。
*   不要使用你的指纹登录。指纹传感器很方便，但周围的法律很复杂。虽然仍在法庭上讨论，但目前 [警方可以强制你使用指纹解锁手机](https://www.theatlantic.com/technology/archive/2016/05/iphone-fingerprint-search-warrant/480861/) 。最好不要使用它。在 Nexus 和 Pixel 设备上，进入设置>安全>像素印记，删除你保存的任何指纹。
*   给你的手机加密(如果还没有加密的话)。有些厂商默认不加密你的手机。如果你必须在手机启动前输入密码，它可能已经被加密了。为了确保安全，请前往设置>安全。在加密下，你应该看到“加密手机。”如果下面写着“加密”，那就没问题。否则，点击它并按照说明加密您的手机。这可能需要一段时间，可能会降低一些旧手机的速度，但为了保护您的数据，这是值得的。
*   **在锁屏时隐藏通知信息。** Android 即使在手机锁定的情况下也会显示通知，但如果不想让窥探的眼睛看到，可以隐藏敏感信息。前往设置>通知，然后点击顶部的齿轮图标。最后，轻按“在锁定屏幕上”您可以选择“隐藏敏感通知内容”来隐藏信息和电子邮件内容，或者选择“完全不显示通知”来确保没有人看到任何内容。
*   **禁用谷歌的追踪活动。**谷歌是周围最大的数据饕餮者，所以禁用他们的跟踪几乎是一个的永无止境的任务，但你至少可以尽可能多地关闭。点开 [这个链接](https://myactivity.google.com/myactivity) ，点击顶部的菜单按钮，选择“活动控件”在这里，您可以禁用位置跟踪、搜索跟踪、语音跟踪，甚至您的 YouTube 历史记录。请注意，谷歌可能仍然保留一些关于你的匿名信息，但这可以尽量减少他们有多少。
*   **关闭谷歌备份。**谷歌备份了你设备的大量信息，包括通话记录、应用程序，甚至你连接的无线网络。如果你不希望谷歌有这些信息，前往设置>备份&重置>备份。您可以完全禁用备份(并且 [制作您自己的](http://lifehacker.com/how-to-set-up-a-fully-automated-app-and-settings-backup-5784857) )或者选择性地禁用您不想存储的数据。
*   **关闭任何不必要的应用权限。**从 Android 6.0(棉花糖)开始，谷歌终于可以让你选择给应用赋予哪些权限了。首先，这意味着如果你担心自己的隐私，你可能应该买一部运行棉花糖的安卓手机。然后，进入设置>应用，点击顶部的齿轮图标。然后轻按“应用程序权限”在这里，您可以查看日历、联系人、位置和麦克风等内容的权限。点击每个应用并禁用任何你不信任的应用。请记住，如果您不确定为什么应用程序需要该权限，这可能会破坏一些应用程序。如果你真的不确定是否可以信任某个应用，你最好卸载它。
*   **开启双因素认证。**除非您启用了双因素认证，否则您的帐户只和您的密码( [不太安全](http://lifehacker.com/your-clever-password-tricks-arent-protecting-you-from-t-5937303) )一样安全。你可以在这里 为你的谷歌账户开启 [，然后为](http://lifehacker.com/your-clever-password-tricks-arent-protecting-you-from-t-5937303) [你在这个列表](http://lifehacker.com/heres-everywhere-you-should-enable-two-factor-authentic-5938565) 上的任何其他账户开启。使用像 Authy 这样的应用来管理你的认证令牌也是一个好主意，因为它允许你用 PIN 锁定应用。这可以在有人偷了你的手机并解锁时保护你的代币。
*   **启用 Android 设备管理器。ADM 可以远程找到你的手机，所以从隐私的角度来看，这可能有点难。一方面，这意味着谷歌将掌握你的位置信息。不过，你也可以用它来定位或远程擦除你的手机。如果你想在丢失设备的情况下使用核武器，这可能是一个很好的工具。**

这应该可以处理 Android 手机带来的大量数据和漏洞(尽管你仍然应该假设你的账户有某种数据被跟踪)。然而，这只是等式的一部分。接下来，你需要看看你每天使用的应用程序。

### **保护您隐私的生产力应用**

大多数生产力应用程序都是为了帮助你尽可能方便地完成工作。然而，保护你的隐私很少是最方便的。因此，你可能需要使用旨在保护尽可能多的数据的应用程序，即使它们不得不放弃一些有用的功能，如云同步或复杂的计算机分析。有些应用，比如 VPN，你可能并不总是需要，但你几乎应该一直使用像 [、密码管理器](https://lifehacker.com/lifehacker-faceoff-the-best-password-managers-compare-1682443320) 这样的应用。

还要记住，如果你通过 Android at Work 程序 从你的公司 [收到了你的手机，即使你使用这些应用程序，他们仍然可以看到你的数据。如果你想让你的信息尽可能保密，就用你自己的手机，并且尽可能把它锁起来。](https://www.android.com/work/)

#### **网络浏览器:勇者**

虽然许多浏览器声称保护你的隐私，但我们发现 [【勇敢的](https://play.google.com/store/apps/details?id=com.brave.browser) 是最好的浏览器之一。它使用 [HTTPS Everywhere](https://www.eff.org/Https-everywhere) 来保持你的流量加密，它阻止脚本，cookies，网络钓鱼，弹出窗口和广告。如果你发现一个网站不能使用 Brave，你可以有选择地重新启用这些功能，找出是什么破坏了这个网站，如果你认为值得的话，甚至可以将其列入白名单。

在桌面上，Brave 有一个奇怪的系统，让用户选择 [付费给出版商，而不是看广告](https://brave.com/publishers.html) 。然而，这在移动版本中似乎不存在，所以您可以放心地忽略它。它并不完美，但与 Chrome 相比，它给了你更多的灵活性来阻止第三方数据跟踪。

#### **电子邮件:质子邮件或 Gmail**

对于最彻底的电子邮件隐私，proton mail 是最好的选择。默认情况下，它会加密您的所有邮件。你可以向其他 ProtonMail 用户发送电子邮件，他们可以像平常一样阅读。如果你把它发送给其他人，他们会得到一个链接，需要输入密码才能阅读你的信息。这是一种非常不方便的发送邮件的方式，但是如果你需要保证除了收件人之外没有人阅读你的邮件，这是一个好办法。

如果你不那么多疑，Gmail T2 T3 仍然是一个很好的方法来保护你的电子邮件隐私，除了谷歌。每封电子邮件都是通过 SSL 发送的，谷歌会对从发件人到收件人的每条信息进行加密。不幸的是，谷歌本身仍然能够 [查看和扫描你的电子邮件](https://support.google.com/mail/answer/6603?hl=en) ，如果他们收到法律请求 ，它甚至可能不得不 [将一些数据移交给政府。然而，Gmail 至少会保护你的邮件免受一些随机窥探者或找到你手机的人的攻击(只要你有一个强密码和双重认证，就像我们前面提到的那样)。](https://www.google.com/transparencyreport/userdatarequests/legalprocess/)

一般来说，电子邮件很难保护，因为它总是依赖第三方服务器来发送消息。如果你有一个戴锡纸帽子的好理由，你可以随时 [试着滚动你自己的电子邮件服务器](http://lifehacker.com/the-best-self-hosted-alternatives-to-popular-services-1579866571) ，但是记住这真的很难。

#### **信息:信号或 WhatsApp**

如果你真的需要与某人安全地交流， [信号](https://play.google.com/store/apps/details?id=org.thoughtcrime.securesms) 和 [WhatsApp](https://play.google.com/store/apps/details?id=com.whatsapp) 将比电子邮件更能保护你的隐私。这两款应用都具有端到端的加密功能，它们不会在你的信息发出后存储它们，而且它们都可以在短信的基础上进行语音通话。不方便的是，双方都需要安装该应用程序才能使用，但建立一个帐户相当容易。

然而，对于那些想要绝对最大可能的隐私的人来说，Signal 提供了一点优势。它不会存储你在和谁通话的元数据，但 WhatsApp 会。这两款应用都不知道你在说什么，但 WhatsApp 知道你在什么时候和谁在聊天。如果 WhatsApp 获得合法授权，它可以交出这些数据。WhatsApp 也可以将你的信息备份到 Google Drive，尽管它们是加密的，所以应该没什么大不了的。即使执法部门要求谷歌提供，他们也无法阅读。尽管如此，重要的是要知道备份就在那里。

#### **密码管理器:LastPass 或 1Password**

保护强密码的最好方法是不知道它是什么。密码管理器可以为您生成长而复杂的密码，并自动将其输入您访问的网站。我们最喜欢的密码管理器是 [LastPass](https://play.google.com/store/apps/details?id=com.lastpass.lpandroid) 和 [1Password](https://play.google.com/store/apps/details?id=com.agilebits.onepassword) 。

LastPass 是免费的， [可以让你跨平台](https://blog.lastpass.com/2016/11/get-lastpass-everywhere-multi-device-access-is-now-free.html/) 同步你的密码库，但缺点是它使用自己的服务器来做这件事。虽然你的数据在 LastPass 的服务器上是加密的，但如果有人针对该公司 [(曾经发生过](http://lifehacker.com/lastpass-hacked-time-to-change-your-master-password-1711463571) )进行攻击，它仍然有可能被黑客攻击。他们的加密足够强大，可以防止攻击者获得用户存储的密码，这是令人鼓舞的，但如果你需要超级小心，你可能不想冒这个险。

1 另一方面，Password 提供两种同步方式。您可以 [每月支付 3 美元，通过 1 密码服务器](https://1password.com/sign-up/) 来同步您的帐户，或者您可以只使用自己的 Dropbox 帐户。您也可以完全跳过同步，将所有密码储存在本地保管库中，并在需要时手动将它们从一个设备复制到另一个设备。这可以确保没有人能够访问您的存储库，即使他们攻击第三方。

#### **VPN:英宗、隧道熊或北 VPN**

使用 VPN 是保护你所有网络流量最基本的方法。一旦你连接到一个 VPN，你的流量被加密，所以没有人窥探可以看到你在看什么。当您在公共网络上时，这尤其有用，因为您可能无法控制您的互联网连接。

在这方面，我们喜欢[nord VPN](https://play.google.com/store/apps/details?id=com.nordvpn.android)[TunnelBear](https://play.google.com/store/apps/details?id=com.tunnelbear.android)。每项服务都需要按月付费，但你每月可以免费获得一小部分数据。你不会想用它来看几个小时的《网飞》，但当你在机场或酒店时，它可以帮助覆盖你的交通。

#### **音符:音符锁定**

Google Keep 是 [给笔记记录者](http://lifehacker.com/not-just-another-notes-app-why-you-should-use-google-k-509256637) 的一个惊喜，但是你不能锁定你的笔记，它们都存储在 Google 的服务器上。 [笔记锁](https://play.google.com/store/apps/details?id=net.newsoftwares.noteslock) 另一方面，将你所有的笔记保存在你的设备上，并通过密码、PIN 码或模式锁来保护它们(尽管为了最好的安全性，你可能应该使用密码)。您可以使用备忘录锁定功能以各种颜色和字体写下备忘录或创建待办事项列表。即使作为一个通用的 notes 应用程序，它也非常健壮，这使得安全功能锦上添花。

如果您想要同步笔记，您可以选择将笔记保管库存储到 Dropbox，以便在设备之间共享。与 1Password 一样，这让您可以更好地控制如何存储您的数据，而不是将其交给像谷歌这样的公司。当然，这意味着有人可以在 Dropbox 中找到你的 notes vault，但它仍然是加密的，所以他们无论如何都不能读取它。

#### **云存储:SpiderOak**

Dropbox 非常擅长保护你的数据，但如果你需要更进一步， [SpiderOak](https://play.google.com/store/apps/details?id=com.spideroak.android) 是在云中存储数据并保持其安全的最佳方式。该公司采用“零知识”政策，使用本地加密，所以它不能读取你的文件中的内容，甚至在你上传它们之前。SpiderOak 不提供任何免费存储空间，但它提供每月 100GB 的存储空间，价格为 5 美元，与 Dropbox 和 Google Drive 等服务具有竞争力。

这应该是保护你的典型 Android 使用免受窥探的良好开端。这里(或任何地方，真的)没有什么是 100%防弹的，但你会领先一步，这通常足以让你摆脱日常攻击。
