# 你不能相信网上的负面评论

> 原文：<https://lifehacker.com/you-cant-trust-negative-online-reviews-1826837616>

最近，在研究我正在考虑预订的度假套餐时，我注意到，尽管所有关于目的地、导游和旅程安排的客户评论都很热情，但有几个人给出的评级很低，原因似乎不像是旅游公司的错:他们的航班被取消了，所以他们错过了一天；他们食物中毒了，所以不能享受旅行；或者他们没有购买旅游保险，因此当他们无法继续旅行时，他们无法获得退款。

Watch

在我看来，这些经历并不一定要在旅行套餐本身的评级中考虑。我主要是在评论中寻找安全提示，所以少数反对者没有动摇我的决定。但是对这个包的低评分肯定会让我重新考虑。

这种对产品进行负面评价的倾向在亚马逊这样的网站上更加明显，在那里人们会对他们没有读过的书或者因为包装在运输过程中被破坏而留下一星评级。而今天发表在《纽约时报》 [*上的一篇文章*](https://www.nytimes.com/2018/06/13/smarter-living/trust-negative-product-reviews.html) 证实了我的怀疑:基本上，你不能相信网上的负面评论。至少，你应该吃的时候放一两勺盐。

根据*乘以*，因为与正面评价相比，负面评价相对较少，所以我们给予负面评价更多的权重，尽管我们可能不应该这样做。阅读负面评论会让我们觉得在花血汗钱之前，我们已经了解了事情的全貌，尤其是最坏的情况。

另一个有趣的地方是:《市场研究杂志》发表的一项 2014 年的研究 发现，网上评论者“比普通消费者更有可能购买不寻常尺寸的东西，获得回报，结婚，生更多孩子，更年轻，更不富裕，并拥有研究生学位。”据麻省理工学院斯隆管理学院的市场学教授、该研究的作者邓肯·西姆斯特 称，只有大约 1.5%的人会留下评论。我们真的应该相信他们的话吗？

文章还指出，当谈到旅游评论时，评论者与谁一起度假在他们如何评价旅行中起着很大的作用。

> 去年秋天发表在《电子商务研究与应用》杂志上的一项研究查看了 125076 条在线评论，发现与重要他人一起旅行的人写的评论最积极，其次是与朋友或家人一起旅行的人。独自旅行或商务旅行的评论者是最消极的。我们的经历会随着我们的期望、旅游专业知识和同行的人而改变。

幸运的是，我和一个好朋友一起旅行。一旦我们回来，我一定会给你我的评级。
