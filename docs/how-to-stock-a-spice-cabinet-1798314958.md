# 如何储备香料柜

> 原文：<https://lifehacker.com/how-to-stock-a-spice-cabinet-1798314958>

在香料柜里储存香料可能有点让人不知所措，原因很简单，那里有太多的香料。储存你能看到的每一种令人兴奋的调料可能很诱人，但是这些东西确实有保质期，你可能最终会有一柜子无味、昂贵的粉末需要更换。



*这是*[***The Grown Kitchen***](http://lifehacker.com/tag/the-grown-up-kitchen)*的一部分，Skillet 系列旨在回答您最基本的烹饪问题，并填补您的家庭厨师教育中可能缺失的任何空白。*

但是没有人喜欢清淡的食物，我推荐每个人手边都有一些调料。一旦你有了这些基础，你就可以用你最喜欢的食谱中经常出现的其他香料来补充。

*   **盐:**盐让其他味道流行起来。准备一盒烹饪用的犹太盐 [和一盒整理用的马尔东](http://skillet.lifehacker.com/what-it-means-to-finish-a-dish-1797691879#_ga=2.158321402.1948575403.1503283631-949419976.1446553382) 。
*   **胡椒:**跳过预先研磨的材料，用研磨机研磨全黑或彩虹胡椒粒。(额外奖励:当你需要增加恐惧 时，再获得一个带有 [白胡椒粒的。)](http://skillet.lifehacker.com/white-pepper-is-the-secret-ingredient-to-kfc-s-original-1786020292)
*   肉桂:你需要整根的肉桂用于泡制和泡茶，磨碎烘焙，撒在美味的菜肴上。这听起来可能很奇怪，但是在我的炖牛肉里加一小撮肉桂改变了我的生活。
*   整个肉豆蔻:新鲜磨碎的肉豆蔻是我众多食谱中的“秘密配料”。是的，它非常适合烘烤食品，但也非常适合任何奶酪菜肴，与丰盛的肉类和淀粉类蔬菜搭配也很好。我也喜欢把它直接磨碎放在朗姆酒鸡尾酒里。
*   **磨碎的辣椒:**辣椒是一种多功能的热量载体，可用于各种烹饪中，只要捏一下就能给一大桶豆子增添美味，或者让你的香料更上一层楼。
*   **辣椒粉:**辣椒粉是一种鲜红色的粉末，你可能会在鸡蛋上看到。辣椒粉可以是温和的、甜的或辣的，但它应该永远是匈牙利的。除了炒鸡蛋，你可以用它来完成任何需要一点辣味的菜肴，或者用它来调味、腌泡汁和调料。
*   孜然:我迷恋孜然，它的甜，土，温暖的味道。它让多肉的菜肴尝起来“完整”，绝对适合墨西哥菜和咖喱菜。当我品尝一道菜并认为它需要“额外的东西”时，那东西通常是孜然。
*   干百里香:新鲜的香草在你可以得到的时候是很棒的，但是你应该一年四季都享受百里香的木头味。这是地中海和意大利烹饪中的一种主食，为一系列美味菜肴增添了微妙的风味。
*   牛至干:像百里香一样，牛至一年四季都有应用，如果没有它，我的意大利面酱就不一样了。
*   **月桂叶:** [月桂叶就像完美的伴唱歌手](http://skillet.lifehacker.com/do-bay-leaves-even-do-anything-1793275685#_ga=2.188048488.1948575403.1503283631-949419976.1446553382) ，提供和谐而不分散注意力。它们微妙的、美味的和隐约的药用味道使米饭、汤和炖菜味道更好，而不会让它们尝起来有太多味道。
*   **碎红辣椒片:**由干辣椒和碎辣椒混合制成，这些辛辣的小家伙可以为你洒在上面的任何东西增加一点水果香味，比如披萨、比萨饼或比萨饼。(我也在腌蘑菇中使用它们，但是拜托，我们都知道碾碎的红辣椒是用来做披萨的。)

当然，香料柜因人而异，取决于烹饪的菜肴类型，这是一件美好的事情。哪些调料是你绝对离不开的，你发现了哪些可以跳过的？
