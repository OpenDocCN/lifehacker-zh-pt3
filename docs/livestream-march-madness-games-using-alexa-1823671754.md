# 使用 Alexa 的直播疯狂三月游戏

> 原文：<https://lifehacker.com/livestream-march-madness-games-using-alexa-1823671754>

我在北卡罗莱纳州长大，在那里，我们对大学篮球都非常认真(T2)。当你住在离北卡罗来纳大学和杜克大学、威克森林附近的几英里范围内时，我们只能说你的三月疯狂括号不仅仅是你和朋友一起做的一件有趣的事情——你在和中画出了潜在的友谊终结线。



好吧，也许没那么严重。但这绝对是一件大事。我记得有同事打电话来请病假，这样他们就可以去看他们的团队比赛，或者重新安排大型会议，这样他们就可以挤在电视机前。这似乎是每个人都喜欢做的事情，比我现在在加州的家更喜欢。

如果三月疯狂对你来说仍然是一件严肃的事情(对我来说当然是)，那么 Alexa 今年可以成为跟踪大事件的一个不错的资源。

### 收听直播

今年，亚马逊与 Westwood One 合作，创建了一个新的直播和赛后直播技能。要使用这个你需要 [启用西林一个运动技能](https://www.amazon.com/Westwood-One-Sports/dp/B079HBYTLS/ref=sr_1_1?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/livestream-march-madness-games-using-alexa-1823671754&asc_source=&ie=UTF8&keywords=westwood one&qid=1520297911&s=digital-skills&sr=8-1&tag=kinjalifehackerlink-20) 。当你这样做的时候，如果你问一个类似“谁会赢得卡罗莱纳州的比赛？”从 3 月 13 日开始，你将获得比分，还可以选择收听比赛的音频直播。

如果你被困在一个会议中，完全错过了一场比赛，Alexa 可以给你最后的分数，并在你问谁赢了时播放一段可选的游戏音频。

如果你启用了[NCAA 疯狂进行曲技能](https://www.amazon.com/NCAA-Digital-March-Madness/dp/B01N7VLHEG/ref=sr_1_1?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/livestream-march-madness-games-using-alexa-1823671754&asc_source=&ie=UTF8&keywords=march madness alexa&qid=1520707319&sr=8-1&tag=kinjalifehackerlink-20) ，你还可以向 Alexa 索要实时赛事比分、赛程信息和比赛直播。像“Alexa，问问 March Madness 现在在玩什么游戏？或者“Alexa，请疯狂三月玩俄勒冈州对爱奥那岛的游戏”将与这个一起工作。

### 保持消息灵通

如果你不是真的想看直播，但仍然想了解最新消息，Alexa 在这个疯狂的三月也有几个选择。具体来说，你可以问她这样的问题:

" Alexa，密歇根州立大学下一场比赛是什么时候？"

“Alexa，精英八人组里有谁？”

“Alexa，三月疯狂还剩下哪些队伍？”

" Alexa，杜克队在锦标赛中的下一个对手是谁？"

你也可以问她对今年谁会赢的预测，以及像过去几年谁赢了比赛这样的历史信息。

是的，如果你喜欢爸爸的笑话，让 Alexa“给我讲一个篮球笑话”也会给你带来一个。
