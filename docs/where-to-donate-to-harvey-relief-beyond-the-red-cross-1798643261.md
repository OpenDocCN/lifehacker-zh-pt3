# 向 Harvey Relief 捐款的地方——红十字会以外

> 原文：<https://lifehacker.com/where-to-donate-to-harvey-relief-beyond-the-red-cross-1798643261>

红十字会可能是这个国家最知名的紧急救援组织——但是他们不一定是最好的[。对](https://www.washingtonpost.com/news/post-nation/wp/2017/08/28/people-are-urging-donations-for-harvey-relief-efforts-just-not-to-the-red-cross/?utm_term=.e7e204680ff0) [管理不善](http://www.slate.com/articles/business/metropolis/2017/08/don_t_give_money_to_the_red_cross_we_need_a_new_way.html) 的指责已经 [困扰着他们对国家和国际灾难的反应](https://twitter.com/ProPublica/status/902978775368990720) ，他们的知名度(以及简易的短信捐赠系统)可能会让当地和基层组织相形见绌。



当你决定如何支持救灾工作时，不仅要考虑你在哪里捐赠，还要考虑捐赠的内容。当然，供应是非常需要的，但即使一个组织有亚马逊的愿望清单，直接的金钱捐赠也可以让他们以更快的速度和更大的灵活性做出回应。参见 [这个*连线*片](https://www.wired.com/story/harvey-donations-amazon-wish-lists-cash/) 了解更多原因。(并且请不要向任何组织发送*未经请求的*捐赠物资。他们很可能无法使用它。)不要忘记，大多数非营利组织，即使他们组织了志愿者，也仍然依赖员工来运作——金钱捐赠有助于确保那些关键员工得到报酬。

大多数组织会允许你将捐款专门用于哈维救济，但考虑进行一般性捐款——如果哈维需要资金，那就是你的钱的去处，但无限制的捐款也允许组织对下一场灾难做出快速灵活的反应。如果这对你来说很重要，在你捐款前检查一下，看看你的捐款是否可以免税。GuideStar 有一个强大的指南来决定在哪里捐赠 [这里](https://learn.guidestar.org/give-to-charity/donor-resources/giving-to-disaster-relief-and-recovery) 。(当然，如果你是本地人，你也可以向这些组织寻求志愿服务。)

用罗杰斯先生妈妈的不朽名言来说， [“寻找帮手。”](http://www.fredrogers.org/parents/special-challenges/tragic-events.php) 嗯，这里有一堆:

### 国家组织

[AmeriCares](https://secure.americares.org/site/Donation2?df_id=22188&mfc_pref=T&22188.donation=form1&_ga=2.27797372.1946876975.1503855566-771809081.1503855566) 提供紧急物资和医疗救护。

【ICNARelief】又名穆斯林博爱组织，从事广泛的慈善工作，包括救灾。

心连心国际 协调救援物资的分发并提供医疗支持。

南方浸信会救灾组织已经存在了五十多年，提供食物、水，并帮助重建工作。

团队 Rubicon “将退伍军人的技能和经验与第一反应者结合起来，快速部署应急响应团队。”

[大休斯顿联合之路](https://www.unitedwayhouston.org/flood/flood-donation/) 解决食物和住所等紧急需求以及长期恢复。

### 地方组织

[无家可归者时期项目](http://www.homelessperiodproject.org/) 给无家可归者、流离失所者和受风暴影响的人发放月经用品。(另外两个地方组织——[aquíEstamos](http://www.aquiestamosrgv.org/)和[Neta](http://netargv.com/home/)——也在为这些努力募集资金。)

[飓风哈维救济基金会](https://ghcf.org/hurricane-relief/) 由休斯顿市长西尔维斯特·特纳组织，隶属于大休斯顿社区基金会。

[S.H.A.P.E .社区中心](http://www.shape.org/hurricane-harvey) 是休斯顿的一个社区组织，致力于“改善非洲裔人(所有人)的生活质量”

[德克萨斯尿布银行](https://texasdiaperbank.networkforgood.com/) 分发给婴儿、老人和残疾人。

得克萨斯工人救济基金会 是得克萨斯州劳联-产联的一个慈善分支机构，“成立的目的是在工人及其家庭有需要时提供帮助。”

### 关注弱势群体

休斯顿/哈里斯郡无家可归者联盟 是一个致力于保护无家可归者和结束无家可归现象的社区组织。

活的希望轮椅协会接受捐款和旧轮椅的捐赠。

Portlight 专注于帮助残疾人，包括在灾难救援中。

[南德克萨斯人权中心](https://southtexashumanrights.org/make-a-donation/) 致力于保护移民家庭。

休斯顿 美国跨性别基金会正在帮助受风暴影响的易受伤害的跨性别者、阴阳人和性别酷儿

回家之路致力于结束休斯顿的无家可归现象——他们与该地区的几家收容所合作。

### 帮助动物

德克萨斯动物保护联盟 接受捐款和物资——在圣安东尼奥亲自捐赠或通过亚马逊愿望清单捐赠。

奥斯宠物存活已经收留了数百只走失或流离失所的宠物。他们正在接受金钱和物质的捐赠。

得克萨斯州的 SPCA 正在帮助协调动物救援工作。

休斯顿人道协会 正在接受捐款帮助受风暴影响的当地动物。

### 食物银行

[“喂养德克萨斯”](https://donatenow.networkforgood.org/feeding-texas) 是一个全州范围的非营利组织，负责协调和整合州和联邦的救援工作。[【NPR】还收集了一份遭受风暴袭击地区的德州食物银行名单:](http://www.npr.org/sections/thetwo-way/2017/08/28/546745827/looking-to-help-those-affected-by-harvey-here-s-a-list) [休斯顿食物银行](http://www.houstonfoodbank.org/)[加尔食物银行](http://www.galvestoncountyfoodbank.org/)[金新月食物银行](http://www.victoriafoodbank.org/) (维多利亚) [科珀斯克里斯蒂食物银行](http://www.foodbankcc.com/)[东南德州食物银行](http://www.setxfoodbank.org/) (博蒙特)
