# 这个网络应用描绘了北美的土著历史

> 原文：<https://lifehacker.com/this-web-app-maps-out-north-americas-indigenous-history-1796488592>

让任何人说出一个美国土著部落的名字，你可能会得到同样多的名字(或者是茫然的凝视)。但是可视化工具 [故土](http://native-land.ca/) 可以帮你识别数百个部落，甚至是你当前所在城镇的部落。这是一个丰富多彩、信息丰富的视觉化图像，也是通向你可能不知道的历史的大门。

Watch

Native Lands 是一个 web 应用程序，在输入您的位置并选择您感兴趣的信息(选项包括领土、语言和条约)后，它会向您显示彩色编码的区域。

将鼠标悬停在您想要查看外部网站链接的任何区域，这些链接会提供更多信息。包含的链接很简洁，因为它们通常指向政府主办的解释部落历史的网站，或者是部落自己主办的网站，而不是维基百科页面。

我自己对美洲原住民历史的了解相当粗略——我在学校里唯一记得的是易洛魁联盟的“六个民族”的绰号。但是《故土》向我展示了美国东北部地区的几十个其他部落，包括被易洛魁人征服的较小的部落。
