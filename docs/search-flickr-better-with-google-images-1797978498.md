# 用谷歌图片更好地搜索 Flickr

> 原文：<https://lifehacker.com/search-flickr-better-with-google-images-1797978498>

在像 Lifehacker 这样的营利性编辑机构，当我们需要一张图片来发布我们的帖子时，我们不能只进行谷歌图片搜索，就能得到第一个结果。我们必须使用经过适当许可的照片。有时我们使用自己的原创照片，有时使用付费的 Getty images，有时使用通过知识共享许可免费使用的数百万张[Flickr](https://www.flickr.com/creativecommons/)照片。

但从 2014 年开始，谷歌图片还允许用户 [通过许可](https://www.cnet.com/how-to/how-to-filter-google-image-searches-by-usage-rights/) 过滤照片。与 Flickr 不同，Google Images 使用了全球最复杂的搜索算法。所以它能挖掘出一些连 Flickr 都挖掘不出来的 Flickr 照片。

在最近的一篇文章中，我需要一张手巾的英雄照。 [在 Flickr 上搜索“手巾”](https://www.flickr.com/search/?text=hand+towels&license=2%2C3%2C4%2C5%2C6%2C9) 返回了很多手*附近和周围*的手巾，再加上平时的第二人生截图和 NSFW 艺术，Flickr 就知道我要的是关于手和手巾的内容。

但是谷歌猜测我的措辞很重要， [找到了更多实际的手巾](https://www.google.com/search?q=hand+towels&tbm=isch&source=lnt&tbs=sur:fc&sa=X&ved=0ahUKEwjR5-m0iuHVAhXh44MKHU5qB0wQpwUIHg&biw=1740&bih=1320&dpr=1) ，就像 [我的帖子](http://lifehacker.com/you-need-hand-towels-in-your-bathroom-1797974287) 中那个漂亮的红黄相间的数字。Flickr 知道那是一张毛巾的图片，但是它不能识别“手”的部分，因为这个词在图片的主页上没有出现。

谷歌的人工智能算法可能已经从字面上识别出这是一条手巾，或者谷歌只是知道其他网站已经链接或嵌入了相关短语的图像。不管怎样，它挖掘出了一个没有明确标记的图像，但正是我需要的。

因此，对于任何棘手的图片搜索，你可能还是想在尝试一个专门的引擎之前，深入谷歌的“工具”菜单。Google Images 将根据大小、颜色、新旧程度、许可证来过滤图像，甚至挑选出照片、面孔、线条画、动画或剪贴画。

如果你是通过许可证搜索，你仍然需要从谷歌搜索结果点击到图片的原始页面，以确认许可证并遵循任何限制，如给出归属。谷歌很强大，但它并不完美，当有人指责你不遵守规则使用他们的照片时，你不能责怪它。
