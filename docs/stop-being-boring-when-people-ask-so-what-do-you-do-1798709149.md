# 当人们问“那你是做什么的”时，不要再无聊了

> 原文：<https://lifehacker.com/stop-being-boring-when-people-ask-so-what-do-you-do-1798709149>

闲聊永远不会不尴尬，因为要了解一个人，你必须问一些令人担忧的问题。有些问题 [不应该问](http://lifehacker.com/the-small-talk-questions-you-should-never-ask-1795578480) ，但是问“你是做什么工作的？”不过，回答这个问题可能会很麻烦。



也许你最不想谈的就是你的烂工作。也许你的工作很复杂，你不容易解释清楚。也许，像我一样，你进入治疗模式，开始卸下你职业生涯中所有的情感障碍。为了避免这些陷阱，试试[*快速公司关于吸引人地描述你做什么*](https://www.fastcompany.com/40461483/how-to-answer-the-question-what-do-you-do-without-boring-anyone)*的建议:*

1.  *找一些相关的东西:用普通人可能接触到的方式来解释你的工作。“你知道麦片盒子的盖子有简单的不会裂开的槽吗？我的公司发明了这些。”“如果你遭遇了致命的车祸，是汽车公司的错，我决定 [是否发布召回](https://www.youtube.com/watch?v=l5EY8oXamoM) 。”*
2.  *不要列出你的简历或者抛弃你所有的职业忧虑。这对自由职业者或企业家来说尤其重要，他们可能会觉得需要对自己所有的收入来源进行说明。其他人不想要时间表，他们想要一个愉快谈话的起点。选择你感觉最好的东西，不管它是否赚钱，然后谈论它。*
3.  *展示而不是讲述:如果你能举一个你所做事情的例子，那就去做吧。但是除非你是在一个社交活动上，否则不要把这变成一个推销。*

*这些建议击中了生活黑客团队的要害。写作生涯是非线性的，涉及很多志向和让步，所以作家往往会过度解释或自我贬低。这对另一个人来说可能会很尴尬，他可能会觉得有义务支持你的自尊或告诉你如何度过个人危机。在任何工作中，自信地谈论你所做的事情。没人想用这个问题来问你。*

*但是一定要根据你的听众量身定制你的答案。我的 Lifehacker 同事选择“博客作者”或“记者”的头衔，取决于他们准备好接受多少关于 lamestream 媒体的质询。总有办法让你的工作变得更好或更差，而不需要过度吹嘘或自责。*

*我最喜欢的回答是“你是做什么的？”来自儿时伙伴的父亲:“我提供产品和服务。”他不会透露更多。这比那家伙可能做过的任何办公室工作都更令人难忘。*
