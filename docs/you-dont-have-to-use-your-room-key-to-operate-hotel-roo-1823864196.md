# 你不必用你的房间钥匙来操作酒店房间的灯

> 原文：<https://lifehacker.com/you-dont-have-to-use-your-room-key-to-operate-hotel-roo-1823864196>

我最近入住的酒店房间中，至少有一半要求你把房间钥匙插在门边的一个槽里，以便做开灯或使用电源插座之类的事情。这是一个我在旅行中至少会忘记一次的细节，最终我会花几分钟摸索着去确定为什么我似乎不知道如何打开床头灯。



本周，推特上流传着一个提示 这是一个游戏改变者:你不需要在那个槽里使用你的酒店钥匙。实际上，你可以把几乎任何一张卡放进去作为替换。例如，你钱包里的一把旧的酒店钥匙或者当地杂货店的积分卡。

说实话，当你在房间里时，这个小槽可以用来保管你的钥匙，但很高兴知道你不必使用它。

这意味着理论上你可以放一张卡在房间里，并在整个住宿期间都留着它，或者在必要时使用另一张卡，如果你想出去一会儿，并在你出去时将笔记本电脑或相机等物品留在房间里充电。这个技巧已经在 Twitter 上被一些人测试过了，但是并不是在所有地方都有效。

另一个解决办法是:索要比你需要的更多的房间钥匙，并打算留一把在卡槽里以备入住。
