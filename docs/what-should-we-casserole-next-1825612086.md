# 接下来我们应该做什么菜？

> 原文：<https://lifehacker.com/what-should-we-casserole-next-1825612086>

大家好，欢迎回到 [*将它砂锅*](https://lifehacker.com/tag/will-it-casserole) ，在这个栏目里我们看到哪些饭菜、菜肴、菜系可以重新想象，改造成砂锅。



我们最后做了芝士牛排砂锅，尽管我找不到真正的高手，我还是对它的结果很满意。但是现在我们必须继续新的美味的，多层次的冒险。我们可以走的一些路线:

*   **全英式煎砂锅:**我是 [英式早餐及其众多组成部分](https://www.instagram.com/p/Bhv-POlBFqq/?taken-by=clairelizlower) 的忠实粉丝，我很想一口就把它们都体验一遍。
*   老实说，我的意思是，我们应该就此结束吗？
*   某人在评论中提到了这个，但是我不确定除了肉酱以外还会有什么。如果你有一个 scrapple 砂锅愿景，请提出建议。
*   **咖喱砂锅:**我理解“咖喱”是一个非常宽泛的范畴，有很多不同的口味和配料。请帮我缩小可能性的范围。

当然，你也可以随意提出自己的原创想法。你最初的想法通常很好。
