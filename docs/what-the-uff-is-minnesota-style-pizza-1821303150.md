# 明尼苏达披萨是什么？

> 原文：<https://thetakeout.com/what-the-uff-is-minnesota-style-pizza-1821303150>

我认为自己是披萨艺术的权威学者，但直到 30 分钟前，我还从未听说过明尼苏达风格的披萨。圣保罗先锋出版社 最近在 [*的一篇报道中提到，双子城一家名为*](http://www.twincities.com/2017/12/12/reds-savoy-declares-itself-minnesota-style-pizza/)*[Red ' s Savoy Pizza](http://savoypizza.com/)**的知名连锁店声称拥有一种国家认可的披萨风格——这种披萨风格必须用拉长的上中西部方言合法发音为:“sota-style”。* 



先锋出版社的报道是这样描述明尼苏达风格的:

> 薄薄的外壳，方形的切口，大量的浇头奶酪和辛辣的“被动-主动”酱。

 [https://thetakeout.com/embed/inset/iframe?id=twitter-751419264867848192&autosize=1](https://thetakeout.com/embed/inset/iframe?id=twitter-751419264867848192&autosize=1) 

我对胡说八道的严重过敏迫使我去调查，我说的调查是指在办公室里粗略地问一下，给明尼苏达州的美食作家朋友发几条短信。以下是我半小时研究的结果:

*   大家真的很喜欢 Red 的 Savoy！它得到了三位明尼苏达裔朋友的赞许。

*   2006 年，这家连锁店的停车场臭名昭著，因为前参议员诺姆·科尔曼 81 岁的父亲在这里被抓到与妓女 T2 在一起。
*   至于真正的明尼苏达比萨饼的合法性？一位在那里生活多年的同事表示:“那只是他们的营销，实际上并不是一件事。明尼苏达州没有人会认为真的有这样的事情，或者使用这个短语。”
*   布莱德·安德森是《纽奥良时报》的餐馆评论家，也是明尼苏达州人，他告诉我他所知道的明尼苏达州风格的比萨饼是一种切成方块的馅饼。当我告诉他这样的馅饼存在于中西部偏上的地区之外，比如芝加哥和圣路易斯(这种纵横交错的结构被称为“派对切”)，安德森回应道:“伊利诺伊州和圣路易斯认为这是他们风格的人可以亲我的瑞典屁股了。”
*   安德森接着说，他不知道明尼苏达州特有的特定成分，但他注意到“这里讨论的明尼苏达州馅饼中有很多奶酪。”
*   我从他位于明尼苏达州圣路易斯公园的办公室打电话给旅游频道*奇异食物*的主持人。以下是我们对话的逐字记录:外卖:安德鲁，有明尼苏达风味的披萨吗？
    **安德鲁·齐默恩:**操不
