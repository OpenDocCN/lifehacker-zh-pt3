# 把最后一点腌沙拉变成鸡尾酒

> 原文:[https://life hacker . com/turn-the last-bit-of-salad-into-a-cocktail-1829278964](https://lifehacker.com/turn-that-last-bit-of-marinated-salad-into-a-cocktail-1829278964)

美味的鸡尾酒正在流行，但你在杂志上看到的花式鸡尾酒似乎总是包含三种定制的注入和一种手工制作的装饰。如果这不是你的风格，检查一下你的冰箱——你可能已经有了你需要的复杂、美味的成人饮料。

无论你认为它们是腌泡汁、调味汁还是盐水，从浸软的沙拉中剩下的味道浓郁的液体在技术上是浸剂——它们就在鸡尾酒里。你可以使用任何听起来不错的东西，但是坚持选择主要是酸、盐和糖，很少或没有脂肪的食物。(酸菜卤水可能不错；奶油凉拌卷心菜，没那么多。)为了获得最佳效果，把盐水当成苦酒或者 [喝醋](https://skillet.lifehacker.com/you-should-add-vinegar-to-your-cocktails-1823772944#_ga=2.175429537.243210930.1537805984-144977662.1506607947) :一点点就够了，你可能需要一种非酒精成分来保持平衡。考虑到这一点，这些组合中的任何一个都将是一流的:

*   [**西红柿和伏特加**](https://skillet.lifehacker.com/this-is-the-easiest-freshest-way-to-make-a-bloody-mary-1826671554) **:** 浸泡过的西红柿沙拉的汁液用在血腥玛丽或 Michelada 中当然很有意义，但我认为它作为肮脏马提尼的即兴重复很有潜力。
*   黄瓜和烧酒:切成薄片的黄瓜浸在调味的米醋和额外的盐中，对我来说是一种荒岛沙拉，我忍不住想把残渣和烧酒混合在一起。
*   西瓜和龙舌兰酒或墨西哥酒:一份好的西瓜和羊乳酪沙拉是新鲜的、水果味的、扑鼻的和时髦的。剩余的腌泡汁将会是玛格丽塔酒的一个美味的、意想不到的附加物——或者是一个伟大的追逐者。
*   甜菜和杜松子酒:等等，回来！腌制的甜菜盐水有一种泥土的味道，和松木味的杜松子酒非常相配。我保证我没有疯:卡奇卡烹饪书推荐用甜菜浸泡的杜松子酒，费城的 [亚伯·费希尔](http://abefisherphilly.com/#home) 让掺有甜菜的尼格罗尼情形*不可思议。*

不准备把杜松子酒和腌甜菜汁混合？我明白。开始尝试听起来不错的口味组合，并在沿途的惊喜中寻找灵感。无论你做什么，不要把自制的泡菜汁倒进水池里；这可能是你最喜欢的鸡尾酒的开始。
