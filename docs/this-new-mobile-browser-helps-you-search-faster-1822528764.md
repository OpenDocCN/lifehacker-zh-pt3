# 使用这款新的移动浏览器加快搜索速度

> 原文:[https://life hacker . com/this-new-mobile-browser-helps-you-search-faster-1822528764](https://lifehacker.com/this-new-mobile-browser-helps-you-search-faster-1822528764)

iOS/Android:我讨厌手机浏览。当我打开 Safari 时，我通常会尝试快速搜索一些东西，然后回到我的其他应用程序。最近我一直在尝试移动浏览器，它今天为 iOS 和 Android 公开发布，它使搜索速度快得多。

Watch

Cake 是围绕搜索和触觉浏览构建的。它的主屏幕顶部有一个巨大的搜索栏，包括五个颜色鲜艳的搜索按钮，分别用于视频、图像、网络、新闻和购物结果。(你也可以像普通地址栏一样只输入一个 URL。)

在大多数移动浏览器上，搜索会将你带到一个搜索结果列表，在那里你必须再次点击才能得到实际的结果。但是蛋糕会直接把你带到第一个结果。你可以向右滑动进入下一页，或者向左滑动进入通常的结果页。Cake 在后台加载它们，并用一个可选的广告拦截器清理它们。但是如果谷歌真的有一个很棒的搜索结果页面，Cake 会注意到并首先把你带到通常的结果页面。

如果你使用图片搜索，你会得到谷歌的典型结果页面，但你可以滑动搜索来自 Giphy、Bing、Flickr、Imgur 和一些图片网站的结果。视频搜索将你带到 YouTube，或者滑动搜索谷歌视频结果、Vimeo、Bing、雅虎和 Yandex。新闻搜索的工作方式和普通搜索一样，只是针对新闻结果；购物使用谷歌购物、亚马逊、沃尔玛、易贝和其他商店。您可以自定义所有这些选项卡，隐藏或重新排序它们，但不能添加新的选项卡。

如果你像我一样经常搜索，这些变化足以让你把蛋糕放在你的主屏幕上。Cake 并没有试图为应用内链接替换你默认的浏览器；只是针对你有意打开浏览器 app 的时候。你可以把它看作一个特性或者一个缺陷。

有一些烦恼。如果你启用了通知，Cake 会发送消息*要求*你把它放在你的主屏幕上，即使你已经有了。而且该浏览器还没有在页面中查找的功能。对于一个面向搜索的应用程序来说，这是一个奇怪的差距，但除非这是一个杀手锏，否则 Cake 是最好的搜索移动浏览器。
