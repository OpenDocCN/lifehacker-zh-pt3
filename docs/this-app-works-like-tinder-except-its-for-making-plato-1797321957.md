# 这个应用程序的工作原理类似于 Tinder，只是它是用来交柏拉图式的朋友的

> 原文:[https://life hacker . com/this-app-works-like-tinder-except-its-for-making-Plato-1797321957](https://lifehacker.com/this-app-works-like-tinder-except-its-for-making-plato-1797321957)

当你在高中和大学时，你几乎每天都会交几个朋友。然而，一旦你毕业了，培养新的友谊，甚至认识你可能想成为朋友的人，会变得有点困难。现在有了一个应用程序。这款名为 [Patook](https://patook.com/) 的应用程序设计得很像 Tinder，只不过你在市场上遇到的每个人都是柏拉图式的朋友——甚至不允许在平台上调情。它在过去的一年中一直处于测试阶段，并于本周向公众开放。

乍一看，这个想法似乎有点疯狂——除了它是我几年前实际看到自己使用的东西。六年前，我从北卡罗来纳州搬到了旧金山。我三十了，干着朝九晚五的工作，在城里正好认识一个人:我哥从中学起的前女友。这可不是美好社交生活的必备条件。

我很幸运地在一个行业工作，在那里我被邀请参加大量的社交活动，所以我最终设法结识了一些人并交了一些朋友，但这是一条艰难的道路，充满了尴尬的聚会团体和独自去酒吧。成年后你是如何开始交朋友的？作为一个在家乡有一大群朋友的人，我甚至不知道如何开始。

使用 Patook，您可以先连接您的脸书帐户，或者使用您的电子邮件地址注册该服务。

我通过连接我的脸书账户注册，这是最简单的方式。在那里，你被要求选择一些你在脸书的照片出现在你的个人资料中，并回答几个类似于你在约会资料中填写的问题。例如，它会问你以什么为生，你最喜欢做什么，你希望在应用程序上遇到什么样的人。所有问题都是可选的，所以你不必分享你不想分享的细节。

之后就跟火绒一样了。如果你对见某人不感兴趣，你向左滑动，如果你感兴趣，向右滑动。有人可能是一个男人，一个女人，甚至是一对夫妇。在我使用这款应用的短暂时间里，我遇到了不少刚搬到旧金山的已婚夫妇，他们和我一样，在几年前希望结识一些新朋友。当匹配成功时，你将被连接并可以在应用程序中开始聊天。如果你们很合得来，那么就像 Tinder 一样，你可以安排一个时间在网下见面。

该应用的算法将检测调情信息，甚至不会发送它们(发送者可能会被禁止)，因此理论上你在使用它时不会被人搭讪，尽管如果互联网教会了我一件事，我相信有人会知道如何做到这一点。也就是说，如果你刚刚搬到一个新的地方，或者只是想脱离你的核心朋友群体，那么这可能是一个寻找新朋友的好方法。
