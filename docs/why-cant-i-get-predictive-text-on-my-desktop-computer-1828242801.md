# 为什么我的台式计算机上无法获得预测文本？

> 原文：<https://lifehacker.com/why-cant-i-get-predictive-text-on-my-desktop-computer-1828242801>

没有什么比预测文本更快地消除错别字了。想想你一天中有多频繁，不停地发短信，使用智能手机的内置功能来预测你试图在完成之前输入的重要单词。如果你很优秀，你可以不用写一封真正的信就写出一个像样的想法。



那么，为什么你不能在你的电脑上也这样做呢？

*Lifehacker 读者 **1wheeldrifter** [写道](https://lifehacker.com/1828168393) :*

> *“我很想为 Windows 找到一个自动完成的工具，像 SwiftKey 和其他类似的功能一样工作，我们在手机上认为这些功能是理所当然的。他们内置了词典等。有什么小技巧吗？”*

*你可能在这里谈论的有两种技术:预测文本，你开始拼写一个单词，并从你的设备提供的选项中选择一个；和自动更正功能，确保你总是——*总是*——发短信告诉你的朋友你“无法回避”的事情*

*在这种情况下，Windows 已经内置了两种特性。如果你在平板电脑模式下使用触摸屏键盘，Windows 会在你开始输入字母的第二秒钟提示单词。至于自动更正，你可以通过访问设置应用程序>设备>输入来打开和关闭它。查找“自动更正拼写错误的单词”选项。*

*如果你在物理键盘上打字，就像在典型的台式机 或笔记本电脑上一样，你可以在同一个设置窗口同时开启这两个功能。只需向下滚动一点，在硬件键盘部分下查找“键入时显示文本建议”和“自动更正我键入的拼写错误的单词”。*

*关于硬件键盘位有一个警告:文本建议和自动更正功能不是通用的。例如，当我在 Lifehacker 的内容管理系统中输入这篇文章时，Wi ndows 正在建议并修复 zilch。然而，当我打开 Microsoft Word 时，我得到了预测和修正:*

*一般来说，这些功能适用于 Windows 10 系统应用、大型微软应用和一些通用应用——记事本、微软 Word、People 等——但通常不适用于你下载的常规第三方应用，如 Chrome、Steam 或网飞(仅举几例)。*

*最终，当你在 Windows 10 的平板电脑模式下敲击字母时，你将能够使用你在问题中提到的应用程序——swift key 本身。(如果你有 [Windows 10 Insider 预览版 Build 17692](https://blogs.windows.com/windowsexperience/2018/06/14/announcing-windows-10-insider-preview-build-17692/#FXEO4r6qulO2hO4q.97) ，你现在就可以尝试一下。)至于当你在物理键盘上打字时，为你使用的每一个应用程序获得预测文本或自动更正，我还没有找到一个伟大的、通用的解决方案。我不确定是否存在。*
