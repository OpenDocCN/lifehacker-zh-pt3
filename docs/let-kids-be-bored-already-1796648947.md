# 已经让孩子们感到无聊了

> 原文:[https://lifehacker.com/let-kids-be-bored-already-1796648947](https://lifehacker.com/let-kids-be-bored-already-1796648947)

每个父母都不喜欢从他们的孩子那里听到可怕的“我很无聊”。一种自然的本能可能是填满他们的时间表，但是无聊有很多好处。《幸福轨迹*[*的作者艾玛·塞普帕拉写了关于*](http://amzn.to/2t2LWgh) *[父母给孩子的坏建议](https://qz.com/1021749/a-leading-happiness-researcher-says-were-giving-our-kids-bad-advice-about-how-to-succeed-in-life/)* *，比如“保持忙碌”她建议鼓励你的孩子什么也不做，尽情享受。**

*在西方国家，人们重视忙碌和兴奋，但平静这样的低强度情绪也同样重要。东亚和西方都重视积极的情绪 ，但东亚人更重视低强度的情绪，他们从散步等更平凡的活动中感受到这些情绪。 [研究发现](https://bingschool.stanford.edu/news/distinguished-lecture-how-does-culture-shape-our-feelings) 香港华裔成年人总体上比欧美成年人更满意自己的生活，这种心态可能是原因之一。*

*[兴奋会让孩子精疲力竭](http://www.cnn.com/2006/EDUCATION/08/30/overscheduled.kids/index.html?_s=PM:EDUCATION) ，自由时间对于培养孩子的创造力很重要。 [研究](http://groups.psych.northwestern.edu/mbeeman/documents/CurrentDirxns_Kounios-Beeman_2009.pdf) 表明，当我们冷静且注意力不集中时，我们更有可能想出绝妙的主意。*

*此外，平衡所有这些活动会导致更多的压力。孩子 [从 20 世纪 50 年代开始变得越来越焦虑](http://www.apa.org/news/press/releases/2000/12/anxiety.aspx) 。通过给他们更多的自由时间，这有助于他们更好地管理压力和放松。音乐课和体育运动可以丰富孩子的生活，但给孩子时间什么也不做也很重要。让他们花一点时间盯着云，做白日梦。*
