# Pokémon Go 的新更新将不再适用于旧的 iOS 设备

> 原文：<https://lifehacker.com/pokemon-go-s-new-update-will-no-longer-work-on-older-io-1821999597>

iOS:如果在你生活中扮演怪物牧马人的孩子迷上了捕捉奇怪形状的虚构动物(或者如果你自己沉迷于游戏)，你最好希望他们最近有一部手机，可以玩他们最喜欢的游戏。Pokémon Go 开发商 Niantic 宣布了一项即将到来的更新，该更新将纳入苹果的 ARKit，带来更加完美的增强现实体验，但禁止无法运行 iOS 11 的设备玩游戏。抱歉，运动鞋。



Niantic [宣布变更](https://pokemongolive.com/en/post/iphone5discontinuedsupport) 将在二月底进行。“在那个日期之后，”Niantic 说，“使用受影响设备的训练者可能再也不能从那个设备访问他们的 Pokémon GO 帐户，或者使用他们袋子里的 poké硬币或其他物品。他们将需要切换到支持的设备才能继续玩 Pokémon GO。”不用说，这很糟糕。这也使得人们不愿意为了继续向不存在的动画怪物投掷数字球而购买新手机，从而无法玩这个游戏。

以下是截至 2 月 28 日将不再支持 Pokémon Go 的设备列表:

*   iPhone 5c
*   iPhone 5
*   iPad(第四代)
*   iPad(第三代)
*   iPad mini(第一代)
*   iPad 2

如果你是一个成年人，很有可能你已经换了一部更新的手机。然而，Pokémon Go 的表面目标受众是儿童，他们中的大多数人并没有每月付费的智能手机升级计划。

从好的方面来看，有事业心的年轻人总是可以利用这一点，说服容易上当受骗的父母用过时的手机 换一款更新的型号。毕竟，这款游戏仍然可以在较旧的 iPhone 5s 和 iPhone 6 智能手机上运行。得一网打尽，对吧？
