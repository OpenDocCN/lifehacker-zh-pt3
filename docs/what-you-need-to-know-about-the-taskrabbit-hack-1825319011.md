# 关于 TaskRabbit 黑客你需要知道什么

> 原文：<https://lifehacker.com/what-you-need-to-know-about-the-taskrabbit-hack-1825319011>

现在 TaskRabbit 由于某种黑客问题而离线，你将无法使用该服务召唤某人来帮助你完成日常任务。但与房间里隐约可见的大象相比，这只是一件小事:你的账户信息可能已经被泄露了。

Watch

TaskRabbit [昨天下午](https://twitter.com/taskrabbit/status/986015274422812672) 在推特上表示，它将暂时关闭其服务，但该公司没有说明这可能会对其用户的账户安全产生什么影响。然而，它确实建议，如果每个人在其他网站或服务上使用相同的登录和密码组合，都应该更改他们的密码。这可不好。

它的网站目前分享相同的信息以及一个简短的常见问题。不要费心点击它的问题，因为所有相关的信息都是你想知道的，比如“什么信息被泄露了？”——只是返回样板答案:“我们正在与外部网络安全公司和执法机构合作，以确定事件的细节，我们会尽可能提供更多信息。”

### 为潜在的安全问题做准备

如上所述，在回应 TaskRabbit 的问题时，你最起码应该做的是想想你还在哪里使用你的在线 TaskRabbit 密码——如果你将密码存储在第三方服务中，比如[【LastPass】](https://lifehacker.com/the-beginners-guide-to-setting-up-lastpass-1785424440)或 [1Password](https://lifehacker.com/the-beginners-guide-to-1password-1794464866) ，这很容易判断。

虽然你应该为每个网站和服务使用一个唯一的密码 ，密码管理员可以帮你处理，但说实话:很多人在做任何事情时都使用一个(或多个)相似的密码。所以，当类似这样的问题出现时，你要做更多的跑腿工作来维护你的在线安全。

除此之外，我们还不知道 TaskRabbit 黑客是否影响了信用卡号码——无论是你在支付任务费用时提交的号码，还是你为了更方便支付而存储的任何信息。TaskRabbit 使用 [Braintree](https://support.taskrabbit.com/hc/en-us/articles/204411560-How-Do-I-Pay-My-Tasker-) 来处理支付，所以 TaskRabbit 遇到的任何问题都不会影响到你的财务状况。(我们已经向 Braintree 发送了一封电子邮件，看看他们对支付安全有什么看法，我们会向他们反馈我们发现的任何情况。)

不过，现在将是一个很好的时间来检查和查看你的信用卡发行商是否提供任何类型的交易警报，你可能想在下周左右打开(如果这不会太烦人的话)。否则，做一个日历提醒，在这个月底(也许下个月也是)检查你的信用卡账单，只是为了确保没有任何奇怪的费用，你的信用卡发卡行出于任何原因没有自动标记。

如果你是一名黑客，那么更多的信息——包括你的银行账号和银行代码 、你的家庭住址和出生日期——可能会因为这次黑客攻击而被泄露。同样的建议也适用于此:在 TaskRabbit 确认此次黑客攻击泄露了哪些财务数据(如果有的话)之前，只要在接下来的一两个月里密切关注你连接到该服务的任何账户。如果你看到任何差异，立即通知你的银行或信用卡公司。

你也可以采取 [一些更大的步骤](https://lifehacker.com/five-steps-to-take-immediately-if-youre-the-victim-of-1507265334) 来开始保护你的生活免受潜在的身份盗窃，比如设置欺诈警报。虽然我建议等到 TaskRabbit 分享更多的信息，这样你就不会经历这些不必要的安全动作，但熟悉一下你的选择绝对没有坏处。糟糕的是，你不能雇佣别人来帮你监控你的财务和网络安全——至少不能通过 TaskRabbit。
