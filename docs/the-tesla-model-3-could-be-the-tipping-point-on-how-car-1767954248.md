# 特斯拉 Model 3 可能会成为汽车在美国销售的转折点

> 原文：<https://jalopnik.com/the-tesla-model-3-could-be-the-tipping-point-on-how-car-1767954248>

你最后一次看到有人 [提前三天排队](http://jalopnik.com/someone-is-already-in-line-for-the-tesla-model-3-1767947093) 订车是什么时候？这是你在 iPhone 发布会上看到的东西，而不是汽车。特斯拉汽车公司为购车注入了一剂硅谷的强心剂，而 [特斯拉 Model 3](http://jalopnik.com/tag/tesla-model-3) 可能是购车革命的开端。

Watch

凭借 Model S，特斯拉创造了一款前所未有的电动车。首先是豪华车，其次是“绿色汽车”。有一些买家买了一辆，因为他们想减少对环境的影响，但埃隆·马斯克知道环保市场是有限的。

现实是，大多数买家在买车时并没有把“低碳足迹”放在优先考虑的首位。这听起来可能很肤浅，人们买车是因为他们认为它很酷。事实证明，S 型车非常酷。

当然，不同的买家对“酷”的定义各种各样，但只要看看购买了多少 SUV 而不是小型货车就知道了。因为一辆小型货车， [尽管是更好的工作工具](http://jalopnik.com/if-youre-a-real-grown-up-minivans-are-cooler-than-cros-1764226230) ，却并不“酷”从不会在车厢里看到灰尘的皮卡车到车主甚至不会接近汽车性能极限的跑车。人们买车不一定是因为他们能做什么，而是因为这辆车让车主感觉如何。

不管你喜不喜欢，Model S 是一个技术展示平台，其性能可以与法拉利和兰博基尼等传奇制造商的超级跑车相媲美。不幸的是，这些车对大多数人来说是遥不可及的——直到现在。

鉴于围绕 Model S 和 Model X 的大肆宣传，无论是否应得，对于普通消费者来说，特斯拉已经成为一个令人向往的品牌。不仅如此，特斯拉绕过过时的经销商特许经营系统的商业模式对 21 世纪的购车者有着巨大的吸引力。普通美国人不想听那些关于特许经营如何保护消费者的废话；他们所知道的是，他们几乎可以在网上购买家中的任何东西，他们也想为他们的汽车做同样的事情。

经销商特许经营及其保护主义立法，加上肮脏和欺骗消费者的历史，使特斯拉成为一代厌倦了以传统方式购买汽车的消费者的典型代表。

在美国销售的新车平均交易价格约为 33，000 美元。Model 3 的 [在激励措施](http://jalopnik.com/elon-musk-says-model-3-will-cost-35-000-before-incent-1679351127) 之前的目标价格为 35，000 美元，这将使它在普通买家的承受范围之内。当然，最大的问题是，当规模扩大到更大规模时，特斯拉的商业模式能否持续。

如果该品牌能够实现这一目标，我们可能会看到一代消费者要求从其他品牌获得“特斯拉风格”的购买体验。

老牌豪华汽车制造商已经看到了不祥之兆，他们正在调整营销和产品开发策略，以应对 Model 3。奥迪的新 A4 轿车将在与特斯拉相似的价位上竞争，它有一个侧重于汽车技术的广告活动。大多数新车也是这样。主流买家希望他们的汽车采用科技，而特斯拉击败了他们中的大多数人。

豪华车经销商已经变成了拥有“产品天才”和交互式售货亭的苹果商店。然而，阻碍他们的是他们仍然被经销商特许经营系统所束缚。

雷克萨斯一直在精选经销商中尝试的“不讨价还价”模式，但这只有在每个经销商都认可的情况下才行得通。为了让“不讨价还价”起作用，你必须普遍执行，否则买家仍将不得不谈判。

截至目前，其他欧洲品牌对特斯拉与经销商的斗争相当矛盾。然而，入门级豪华轿车市场竞争激烈，非常拥挤。如果大玩家开始看到由于 Model 3 导致的市场份额下降，他们可能会改变立场，希望有自己的直销模式。

经销商游说是一个强大的力量，但如果更多的品牌支持特斯拉，并且更多的人对他们不能像购买其他东西一样购买汽车表示失望，那么汽车购买格局将不得不调整。

许多奢侈品买家表达了购买特斯拉的愿望，只是因为他们不必在令人沮丧的经销商系统中导航。这是一个购买的过程，即使对熟练的人来说，也是一个耗时且令人沮丧的过程。

Model 3 的成功可能只是一个爆发点，让美国消费者意识到确实有更好的买车方式。这并不是说经销商在不久的将来会消失，而是他们的经营方式会有很大的不同。
