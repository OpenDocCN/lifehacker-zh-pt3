# 父母最好的 Chrome 扩展

> 原文：<https://lifehacker.com/the-best-chrome-extensions-for-parents-1819142533>

为人父母就像是唱红脸唱白脸，没完没了，对于今天这一代养育孩子的人来说，这包括在数字前沿制定法律。

Watch

幸运的是，有一些产品可以帮助你管理孩子的数字生活——无论他们是在线学习，无休止地浏览 YouTube，还是在两者之间来回切换——因此，我们根据你正在寻找的帮助，为父母编制了一份我们最喜欢的谷歌 Chrome 扩展列表:

### 提高他们的学习时间:

[**保姆**](https://chrome.google.com/webstore/detail/nanny-for-google-chrome-t/cljcgchbnolheggdgaeclffeagnnmhno/details)T5】

要成为一名高效的监工，你不必盯着秒表。Nanny 允许您对孩子最喜欢的娱乐网站设置时间限制，以帮助他们更好地管理工作负荷并保持专注。你也可以限制对某些网址的访问。

[**衬垫**](https://chrome.google.com/webstore/detail/liner-private-webpdf-high/bmhcbmnbenmcecpmpepghooflbehcack?hl=en)T5】

一旦你在令人作呕的互联网绞架上做好了准备，剩下的就可以成为你孩子教育和灵感的源泉。Liner 是一个非常好的学习工具，它允许孩子们在网页和 pdf 文件上高亮显示(和注释)文本，甚至在移动设备上。更好的是，它将高亮显示的段落编辑成册，以备日后回顾，这是写文章的必备条件。

[**语法上**](https://chrome.google.com/webstore/detail/grammarly-for-chrome/kbfnbcaeplbcioakkpcpgfkobkghlhen)T5】

当我们谈到单词的时候，如果你是一个严格的语法学家，Grammarly 是鼓励孩子在网上写作时养成良好拼写和语法习惯的最佳延伸。如果你想放弃对 Twitter 的语法监督，你可以关闭对特定网站的监控。

[](https://chrome.google.com/webstore/detail/imtranslator-translator-d/noaijdpnepcgjemiklgfkcfbkokogabh?hl=en)

**使用谷歌翻译的开放 API，ImTranslator 通过弹出气泡或直接与文本对齐来翻译外语段落。它还拥有一个定义新词的词典。**

### **保护他们的隐私，阻止不想要的内容**

**[**web filter Pro**](https://chrome.google.com/webstore/detail/webfilter-pro-the-best-fi/ejgfoklefkbjadjcgjmnhfbdfjolojnn)T5】**

**与 Nanny 相比，它更多地是为时间管理和特定网站屏蔽而设计的，WebFilter Pro 允许父母使用基于关键字的类别来清除整个令人反感的网站，这使你不必在令人不安的疯狂恋物癖色情网站中寻找要禁止的 URL。**

**[**F 关**](https://chrome.google.com/webstore/detail/f-off/cgncneknljcfojiedmehojdlenlajkbh?hl=en)T5】**

**尽管它的名字听起来很刺耳，但 F Off 是一个阻止网页上带有星号的污言秽语的扩展。你也可以自定义可以接受的词，如果你把某些骂人的话看作是一种逐渐的必然性，这是很有帮助的。**

**[](https://www.ghostery.com/)**

****广告商会不择手段地追踪行为模式，这也包括你孩子的行为模式。Ghostery 检测并阻止常见的跟踪设备，如像素、错误和信标，让您放心，并提高页面加载时间。****

## ****休息时的娱乐:****

****[**电视**](https://chrome.google.com/webstore/detail/tv/bppbpeijolfcampacpljolaegibfhjph)**** 

****当学校周结束时，你不能指望孩子们在那个古老的等离子屏幕前看周六早上的卡通片。他们需要分散注意力！通过在线电视扩展，他们可以在舒适的浏览器上收看和保存世界各地的电视频道。****

****[](https://chrome.google.com/webstore/detail/honey/bmnlcjabgnpnenekpadlanbbkooimhnj?hl=en)****

******如果你觉得足够慷慨，愿意交出你的信用卡(好像你的孩子还没有记住号码)来进行网上购物狂欢，那么先安装 Honey 吧。它从网上搜索优惠券代码，并自动应用它们，在结账时为你省钱，不管是不是你在购物。******

******[](https://chrome.google.com/webstore/detail/soapbox-%E2%80%94-screen-webcam-r/lmepjnndgdhcgphilomlfekmgnnmngbi?hl=en)******

******Soapbox 是一个简化的视频录制和编辑工具，对于那些对视频日志感兴趣或为大学录取申请准备视频文章的孩子来说非常有用。它可以同时记录网络摄像头和屏幕活动，并在分屏中编辑它们，这使得它很容易炫耀击败了一个超级困难的视频游戏老板，或早熟的编码掌握。******

******[**音乐加**](https://chrome.google.com/webstore/detail/music-plus-for-google-pla/ipfnecmlncaiipncipkgijboddcdmego?hl=en)T5】******

****幸运的是，你的孩子没有生活在一个他们不得不冒着感染你的电脑的风险从 KaZaA 下载最新热门歌曲的世界里。如果他们使用 Google Play 音乐进行流媒体播放，这个扩展为他们提供了一个迷你播放器，可以让他们在不中断浏览的情况下浏览播放列表。它还具有歌词的快速链接(当我们需要一个新的 AIM Away 消息时，它在哪里？)****

****[**坐立不安**](https://chrome.google.com/webstore/detail/fidget-spinner/npobhgmanmlkdlognjfkbnejafkcgaal/related?hl=en)T5】****

****即使在网上，也无处可逃。投降吧，好警察。****
