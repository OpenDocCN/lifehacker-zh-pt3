# 本周我们读到的最好的东西

> 原文：<https://lifehacker.com/the-best-things-we-read-this-week-1824001093>

当我们不写关于或 [为什么不应该退出脸书](https://lifehacker.com/dont-delete-facebook-just-be-smarter-on-facebook-1823922407) 的文章时，Lifehacker 的员工通常会翻阅他们的 [口袋清单](https://lifehacker.com/read-it-later-showdown-pocket-vs-instapaper-1756021479)[搜索 Reddit](https://lifehacker.com/if-meditation-stresses-you-out-try-this-1823969908) 或了解最新的畅销书。幸运的是，阅读是工作的一大部分。



有很多好的文章和报道，但是这些文章和书籍在本周给我们留下了深刻的印象。如果你感兴趣的话，可以收藏一些作为周末休闲阅读的书签，并让我们知道本周什么引起了你的注意。

### 主编梅丽莎·基尔希

我看推荐“ [”复苏扼杀伟大的写作吗？](https://www.nytimes.com/2018/03/13/magazine/does-recovery-kill-great-writing.html) ”莱斯利·贾米森在*纽约时报杂志*。她着眼于著名的醉酒作家康复后的作品，寻找饮酒对于创造力不是必不可少的证据。像贾米森的许多最好的散文一样，它一半是个人的，一半是报道的，这让我为她今年的新作品集感到兴奋。

### 执行主编弗吉尼亚·k·史密斯

游戏进行得很晚，但我终于读到了这篇关于枪支游说者 的简介，他是佛罗里达州极其宽松的法律的幕后推手(这些法律后来被其他州效仿)。令人不安，但写得很好，报道得很好，对了解这个国家的政策实际上是如何制定的很有帮助。

### 高级技术编辑大卫·墨菲

前几天我读了这篇 [*《纽约时报》*](https://www.nytimes.com/2018/03/21/technology/facebook-zucktown-willow-village.html)的报道，它真的让我开始思考这些地方(旧金山、纽约市等)的未来住房。)，以及这个想法有多危险，tbh。当你被解雇时会发生什么？被解雇了？当你也失去了你的房子/公寓/什么的时候，生活会变得更加灾难性。

### 健康编辑 Beth **Skwarecki**

我在看 [颠覆性的文案编辑](http://www.subversivecopyeditor.com/) 。这不是关于如何编辑，而是关于在编辑和许多其他工作中发生的所有人际关系。比如:如何给予和接受批评，如何与难相处的人保持良好的工作关系，如何在人们指望你的时候保持清醒，以及如何打破规则并逃脱惩罚(当这是正确的事情时)。

### 专职作家尼克·道格拉斯

这个星期我读了詹姆斯国王的书。他把它写成苏格拉底式对话，解释了为什么巫术和魔法是真实的和坏的，读起来很像丹尼尔·马洛里·奥特伯格的“ [两个和尚发明西方艺术](https://imgur.com/gallery/brwgR) ”系列(可惜现在离线了)。在此之前，我已经连续放弃了三本书，但我在几天内就看完了这本薄薄的电子书。

### 个人理财作家艾丽西娅·亚当奇克

这本来自马乔里·斯通曼·道格拉斯高中学生的日记集是如此的感人肺腑，让我不下四次热泪盈眶。我也是作家兼播客的忠实粉丝， [乔斯林·k·格雷](http://jkglei.com) ，我真的很欣赏 [这次采访](https://ownyourcontent.wordpress.com/2018/03/19/jocelyn-glei/) 她给了我很多很好的建议和对创作的见解。
