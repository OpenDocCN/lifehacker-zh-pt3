# 任何人都可以学会的提高收入潜力的六个简单技巧

> 原文：<https://lifehacker.com/six-simple-skills-anyone-can-learn-to-improve-earning-p-1787666044>

人们为了赚钱而做的各种各样的事情看起来彼此完全不同，就像彩虹中不同的颜色一样，但事实是他们之间有很多相似之处。



*本帖最初出现在* [*的简单美元*](http://www.thesimpledollar.com/six-simple-skills-everyone-can-learn-to-improve-their-earnings-potential-at-work/) *上。*

事实上，几乎每个人都可以用六种技能来提高他们在工作中的收入潜力。不管你的工作是什么，如果你在工作中运用了这六项技能，你就为自己将来获得更高的薪水做好了准备。

### **谈判技巧**

一个具有良好谈判技巧的人能够直接去找他们的老板(或客户)，以积极的方式展示他们的工作，并有效地要求升职或加薪(或其他福利)。

很多人不这样做，原因有很多。第一，他们害怕谈话。以这种方式和他们的老板交谈似乎令人生畏，他们会想象不切实际的负面结果。没有老板会因为你要求加薪而解雇你；他们可能会说“不”，但没有人会因为提出自己的理由而被解雇或受到谴责。第二，如果他们真的有勇气这么做，他们没有很好地陈述他们的情况。他们只是没有提供任何加薪的好理由，这意味着他们有一些与随机招聘的新员工不同的属性。第三，他们很容易接受“不”这个答案，也不会谈判。通常，“不”只是一个起点。

培养这种技能将帮助你为现在的工作获得更多的报酬，并为你的下一份工作获得更好的起薪。你可以通过阅读几本关于谈判的关键书籍来开始构建它——我推荐罗伯特·恰尔迪尼的[](https://www.amazon.com/Pre-Suasion-Revolutionary-Way-Influence-Persuade/dp/1501109790?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/six-simple-skills-anyone-can-learn-to-improve-earning-p-1787666044&asc_source=&tag=kinjalifehackerlink-20) ***以及罗杰·费希尔和威廉·尤里的 中的 [*变得对:谈判协议而不让步。*](https://www.amazon.com/Getting-Yes-Negotiating-Agreement-Without/dp/0143118757?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/six-simple-skills-anyone-can-learn-to-improve-earning-p-1787666044&asc_source=&tag=kinjalifehackerlink-20)***

***然后，在不太重要的情况下练习使用这些技能。在你的生活中寻找任何和所有你正在和某人谈判或试图说服某人做某事的情况，然后使用本书中的技巧。当你有点经验的时候，就在谈判桌上用它们来提高你的薪水。***

### *****敢于直言的勇气*****

***在很多很多工作场合，员工普遍的反应是对工作场所的低效和挑战只字不提。只管埋头做事，不要谈论问题，最重要的是，不要自愿做任何事情。***

***信不信由你，这实际上是确保你再也赚不到一分钱、再也得不到晋升的最佳方式。这能确保你不被注意，而不被注意的人也不会得到加薪。他们得不到晋升。当谈到工作时间时，他们经常会吃亏。***

***即使在最初级的工作岗位上，敢于直言的勇气也是至关重要的，而且随着职位的上升，这种勇气会变得更加重要。***

***当然，要知道直言不讳和不断抱怨是有区别的。这里有一个简单的方法来区分。抱怨的重点是让你受益——它是对一种情况的抱怨，这种情况会让你的情况变得更困难，但可能对其他人有益。畅所欲言的重点是让企业受益——这是对可能耗费企业资金或可能提高企业效率的情况的观察。只有在*极端*的时候才抱怨；只有在明显有用的时候才大声说出来。***

***说出来需要勇气，尤其是当这可能意味着你在短期内会有更多的工作时，但正是这种说出来的勇气极大地增加了你作为一名员工的价值。***

### *****时间管理*****

***无论你的工作是什么，都涉及到时间管理的方方面面。不管你的工作是什么，知道如何聪明地利用你的时间，在分配给你的时间内完成所有要求的任务是很有用的。***

***时间管理让你不用额外工作(通常是无偿工作)就能完成所有的事情。时间管理允许你做一些额外的事情来脱颖而出(或者和同事建立良好的关系)。良好的时间管理能直接缓解工作压力，让你更好地集中注意力，同时感到更加冷静和可控。***

***有很多简单的时间管理系统。其中大多数都是标准待办事项清单的变体，你只需简单地写下需要完成的任务，当它们出现在你的脑海中或交给你时，添加新的任务，然后专注于清单上最重要的任务，尽可能高效地完成它。对于有很多预定会议的人来说，日历和待办事项列表的组合效果最好，待办事项列表上的项目填补了日历事件之间的空白。对于大多数入门级工作来说，一个袖珍笔记本和一支笔可以很好地处理这一点。***

***随着你的工作要求越来越高，任务种类越来越多，一个与电脑同步的好的手机待办事项应用程序会非常有用。我真的很喜欢这款“物有所值”的待办事项应用(我用[【Omnifocus】](http://www.omnigroup.com/omnifocus)，但我认为价格差异不值得——Omnifocus 更好，但不是*那款*好得多)，我绝对喜欢 [谷歌日历](http://www.google.com/calendar) 。***

***在时间管理方面值得一读的书，我非常推荐大卫·艾伦的[](http://www.thesimpledollar.com/getting-things-done-five-key-things/)*(其中)以及加州纽波特的 [*深度工作*](https://www.amazon.com/Deep-Work-Focused-Success-Distracted/dp/B0189PVAWY?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/six-simple-skills-anyone-can-learn-to-improve-earning-p-1787666044&asc_source=&tag=kinjalifehackerlink-20) 。****

### ******职业道德******

****即使你拥有世界上最好的时间管理技能，你仍然需要将它与实际完成任务的能力结合起来。这意味着你必须能够专注于手头的任务并处理好它，然后一遍又一遍地这样做。这就是职业道德，也可能是你在工作场所能拥有的最有价值的东西。这会让你*成为一名非常有价值的员工(让你更容易协商加薪或升职)，这也是你能够接受更具挑战性的机会所需要的。*****

****你如何建立职业道德？我认为，对许多人来说，与职业道德相关的一个关键因素是专注(我认为大多数有更深层次职业道德问题的人可能不会阅读一本名为《简单的美元》的书来改善他们的财务或职业)。大多数人都愿意工作，但是许多人很难保持专注于他们的任务。他们会被任何事情分心，从他们的电话到他们的同事，到一次谈话，到他们的白日梦，到此刻外面发生的任何事情。****

****那么，如何提高注意力呢？最有效的策略是尽量减少分心。关上你的门，有时挂一个“请勿打扰”的牌子。关掉手机，关闭网络浏览器和电子邮件程序。如果可能的话，关上百叶窗。让它成为你的领域中除了你和你的任务几乎没有别的东西。****

****另一件我发现非常有用的事情是专注冥想。这基本上相当于去健身房，除了你在锻炼你集中注意力时使用的精神肌肉。这真的很容易做到，只需要五分钟，尽管只有每天(或一天多次)这样做，好处才会真正显现出来。只是坐在一个舒适的地方，闭上眼睛，专注于你的呼吸。吸气，呼气。吸气，呼气。当你这样做的时候，你的思想会完全走神，这很好。只要*注意到*你的思想在走神，然后让它回到你的呼吸中。每次你注意到它并将注意力带回你的呼吸，你基本上是在弯曲你的精神聚焦“肌肉”，而*它就像冠军*一样工作。它帮助我保持冷静，帮助我专注于手头的任务，帮助我处理不知所措的感觉。我不能建议这样做的高度不够。****

****职业道德的另一个重要方面是对工作的简单承诺。始终要记住的关键一点是，有人为你的工作支付报酬，如果你希望为这项工作获得更多报酬，你必须比其他人生产得更多。你必须比街上的人生产更多的东西。因为，不管你喜不喜欢，这就是你的雇主在某种程度上对你的看法。你是某种产品的金钱交换，如果其他人可以用更少的钱生产同样多的产品，或者用同样的钱生产更多的产品，那就很难证明你是对的。设身处地为他们想想——你是愿意花 100 美元雇佣一天能生产三样东西的人，还是愿意雇佣一天能生产四样东西的人？如果你想加薪，你必须努力工作。工作中不要懈怠。让你的思想集中在手头的任务上。言归正传。****

### ******积极联网******

****我所说的积极的人际关系网，只是指与其他员工建立纯粹积极的关系，也就是说，他们不会围绕着与他人建立消极的感觉或消极的关系。****

****事实是，无论你做什么，你都不会被所有人喜欢，你也不会被所有人喜欢。这就是生活的现实。问题不在于你的感受或别人对你的感受，而在于你如何表达这些感受。****

****事实是，很少有人想和那些一直表现消极的人交往。在一些工作场所，你可能会发现一两个人在他们的肩膀上有一个大芯片。你甚至会发现，那些人身边有一个亲密的圈子，一些赞同和支持消极者的谄媚者。然而在现实中呢？你不想让他们在你身边。其他人不希望他们在身边。通常，除非他们有某种特殊的权力要求，否则他们的日子屈指可数。****

****相反，大多数人想与之共事的人是对每个人都友好积极的人。这并不意味着过度快乐；这意味着承认他人，倾听他人，在被询问时提供有用的想法和反馈，参与对话，并且除非是私下提出，否则绝不提出负面的批评。****

****我给你举个例子。我曾经和一个人一起工作，他永远不会在别人面前说你的坏话。你永远不会听到他说另一个人的坏话，不管那个人是否在场。他唯一一次说出批评的话是在一对一的情况下或通过电子邮件，而且通常是与积极的事情一起发表，显然是为了让你过得更好。****

****那个人看起来很普通。他并不总是说得好。他有点超重，有一些严重的书呆子式的兴趣。但是每个人都喜欢他。每个人都重视他的建议。他基本上有一份终身工作，并且经常得到加薪的奖励*，没有人会介意。*****

****这是因为他非常擅长积极的人际交往。他努力与办公室里的每个人建立积极的互动关系，他只是避免批评。如果有人来找他，对其他人提出批评，他通常会什么都不说，或者温和地指出这个人的优点，或者将投诉转给真正的主管。如果你想随便聊点什么，他几乎总是乐意的。如果你要求对某件事的反馈，如果他觉得有必要批评，他会挖掘一些积极的东西来配合批评。****

****我偶尔会和工作中有这种行为的人一起工作，每次和他们一起工作都是一种真正的快乐。每一次，这些人都得到了持续的雇佣、加薪和晋升。****

****成为那个人。永远不要在工作场所消极。如果你不得不批评某人，私下里批评，把它放在他们做得好的地方。努力与每个人建立积极的关系。如果别人在谈话中表现消极，不要参与其中。****

### ******领导力******

****这是你可以在工作场所练习的最后一项技能，它将帮助你提高收入，而且可能是所有技能中最重要的一项。领导力简单来说就是当一群人需要某样东西时挺身而出的那个人。它意味着成为一个愿意代表一组人提出计划或采取行动的人。****

****这并不意味着成为经理。事实上，通常情况下，最好的领导方式并不来自于经理。****

****想想你的工作场所。想想当你遇到问题或挑战，或者需要帮助制定计划时，你会去找的那个人。想想当有工作挑战或大项目时，每个人都会看的人。那个人就是领导。为什么那个人是领导？那个人就是想出计划的人。那个人会四处走动，找出共识，然后说出来。****

****简而言之，成为一名领导者基本上是这里提到的其他技能的结合。一个好的领导是一个积极的人际关系网，与每个人都有良好的关系。一个好的领导不怕直言不讳。一个好的领导者善于管理他们的时间。一个好的领导有良好的职业道德。好的领导会谈判。如果你把所有这些特质结合到一个人身上，你就会成为一个天生的领导者。****

****如果你想开始尝试用你的双手和你的头脑来思考领导力，我建议从西蒙·西内克的书 [*开始，从为什么*](https://www.amazon.com/Start-Why-Leaders-Inspire-Everyone-ebook/dp/B002Q6XUE4?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/six-simple-skills-anyone-can-learn-to-improve-earning-p-1787666044&asc_source=&tag=kinjalifehackerlink-20) 开始，这本书本身是基于西蒙·西内克的 [精彩的 ted 演讲，基于相同的主题](https://www.youtube.com/watch?v=sioZd3AxmnE) ，你可以免费观看。****

### ******最终想法******

****正如我在领导力一节中所说的，这些因素都是相互关联的。所有这六种行为往往会相互加强。时间管理强化了良好的职业道德，反之亦然。积极的人际关系网有助于提升领导力。谈判技巧倾向于直接帮助你鼓起勇气说出来，因为你正在燃烧你对听到“不”的恐惧。这些技巧都是互相帮助的。****

****然而，更重要的是，它们共同提升了你的 T2。无论你在哪里工作，他们都会让你成为更受欢迎、更有价值的员工。它们让你成为一个不仅个人更有效率，而且还能提高他人效率的人。它们让你成为一个倾向于挑战而不是逃避挑战的人。****

****这些是世界上每家公司都在寻找的特质，并且他们会为此付出代价。练习这些技能并建立它们。让它们成为你在工作场所自然表现的一部分。无论你的工作是什么，回报*都会随之而来。*****
