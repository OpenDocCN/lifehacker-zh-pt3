# 使用 Cast Rewinder 从头开始订阅播客

> 原文:[https://life hacker . com/subscribe-to-a-podcast-from-beginning-with-cast-rew-1827697387](https://lifehacker.com/subscribe-to-a-podcast-from-the-beginning-with-cast-rew-1827697387)

我有一个坏的播客习惯。我不断发现伟大的，长期运行的播客，并下载他们的整个积压。现在我已经排好了 32 集 128 集的队。也许我应该使用 [Cast Rewinder](https://rewind.website/) ，它一次给你发送一个旧的播客片段。

Watch

只需输入 RSS 源或 iTunes 页面(即使你用的是安卓系统)，告诉它你是想要每天、每周还是两者之间的新剧集。Cast Rewinder(目前处于测试阶段)为你的延时剧集提供 RSS 提要。把它粘贴到你的播客播放器中( [下面是](https://medium.com/@joshmuccio/how-to-manually-add-a-rss-feed-to-your-podcast-app-on-desktop-ios-android-478d197a3770) )。现在你会得到旧的情节，就像它们是新的一样。如果你赶上了，你可以切换到“真正的”播客。

这是一个很酷的工具，但不是魔法。*这个美国生活*只让你下载最新四集，Cast Rewinder 改变不了这个。因此，如果你担心剧集会在你赶上之前消失，那么你应该以正常方式下载。但是如果你想赶完 [130 集*欢迎来到夜谷*](https://itunes.apple.com/us/podcast/welcome-to-night-vale/id536258179#) ，一个晚上一个晚上，然后把它插到 Cast Rewinder 里。

Cast Rewinder 是一个发布到 GitHub 的 [开源项目，所以如果你是一个开发者，你可以安装你自己的版本，并根据你的喜好进行调整。](https://github.com/joachimesque/Cast-Rewinder)
