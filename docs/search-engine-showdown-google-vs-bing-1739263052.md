# 搜索引擎对决:谷歌 vs 必应

> 原文：<https://lifehacker.com/search-engine-showdown-google-vs-bing-1739263052>

### 竞争者

你可能已经对这两个搜索引擎很熟悉了，但是再复习一下，下面是我们正在谈论的内容:

*   谷歌:迄今为止世界上使用最多的搜索引擎，谷歌搜索自 1997 年出现以来，一直在不断改进，增加了新的智能结果、高级功能以及与其他谷歌产品的集成。截至今年 2 月，谷歌目前拥有 [64.5%的美国搜索市场份额](http://www.comscore.com/Insights/Market-Rankings/comScore-Releases-February-2015-US-Desktop-Search-Engine-Rankings) 。
*   Bing 是微软搜索引擎的最新名称，之前叫做 Windows Live Search 和 MSN Search。Bing 将自己标榜为“决策引擎”，旨在提供更真实的结果，而不仅仅是在页面上找到文本。必应还为雅虎的搜索引擎提供支持，该搜索引擎与必应、 [一起，为 32.6%](http://www.comscore.com/Insights/Market-Rankings/comScore-Releases-February-2015-US-Desktop-Search-Engine-Rankings) 的美国互联网搜索用户提供服务。

### 基本布局和搜索功能

当涉及到基本的搜索结果时，这两个网站的外观和感觉非常相似。事实上，除了页面顶部的字体和徽标，很容易将两者混淆。以下是我在仔细研究了每一个之后注意到的主要区别:

*   必应的视频搜索明显优于谷歌。这是两者最大的区别(也是为什么必应有点“色情搜索引擎”的名声)。它不是给你一个小缩略图的垂直视频列表，而是给你一个大缩略图的网格，你可以在不离开 Bing 的情况下点击播放。对于一些视频，如果你悬停在上面，它甚至会给你一个预览。
*   在大多数情况下，Bing 给出的自动完成建议比 Google 多。谷歌只给四个，必应给八个。如果你正在 [使用自动完成功能查找替代产品](http://lifehacker.com/discover-popular-alternative-products-with-google-autoc-5944579) 或 [获取通配符建议](http://lifehacker.com/get-wildcard-suggestions-with-a-google-autocomplete-tri-5583436) ，这将特别有用。
*   谷歌的购物建议比必应的出现得更频繁，而且通常也更好。因此，如果你试图找出哪些商店出售某种产品，或者在网上哪里可以找到最好的价格，谷歌会比必应更好。
*   谷歌的图片搜索界面在你使用时感觉更流畅一些，尽管必应有一两个更高级的选项，比如“布局”(可以让你搜索纵向或横向图片)。必应的图片搜索还可以让你一键删除搜索词的某些部分，这很酷(我希望它能对所有搜索都这样做)。
*   Bing 把相关搜索和相关图片搜索放在你搜索结果的右边，而 google 把它们放在底部附近。这其实不是一件好事或坏事；只是区别而已。

必应还采用了谷歌的许多“智能搜索” ，比如电影放映时间、单位转换、当地天气、名人信息等等。这些东西大部分都非常相似，尽管谷歌有一些必应没有的东西，比如健康信息以及电影和视频游戏的发布日期。如果你依赖智能搜索，就用谷歌吧。(有一个例外:如果你在搜索航班，必应有一个很好的功能，那就是 [预测机票价格是上涨还是下跌](https://lifehacker.com/the-best-things-bing-does-better-than-google-1483983630) 。)

### 高级功能和无聊的东西

令人惊讶的是，对于高级操作员来说，这两个引擎非常相似。两者之间的语法可能略有不同( [这里列出了谷歌的高级运算符](http://www.googleguide.com/advanced_operators_reference.html) ， [这里列出了必应的](https://msdn.microsoft.com/en-us/library/ff795661.aspx) )，但有很多重叠之处。不过，谷歌有更多，而且在高级搜索方面会更好*，除了只有 Bing 能做的两个搜索:*

*   `contains:`，允许您搜索包含特定文件类型(如 PDF)的页面。在谷歌上，这只会将你链接到 PDF 文件本身，但在必应上，它会给你包含 PDF 文件链接的页面，这可能是有用的。
*   `linkfromdomain:`，它显示了从某个网站链接到的排名最好的页面(例如，Lifehacker 在某个主题上链接到的最好的网站)。
*   `feed:`，它可以让你搜索特定主题的 RSS 提要。

谷歌的搜索还内置了一些额外的细节，如反向图像搜索( [，这是令人难以置信的](https://lifehacker.com/clever-uses-for-reverse-image-search-473032092) )、即时搜索(在你键入时显示结果)、语音搜索(让你用麦克风搜索)，当然，还有与 Gmail、Google Now 和 Google Contacts 等谷歌服务的集成。如果你用了很多谷歌的东西，就很难想离开它的搜索。

另外，Bing Rewards 也值得一提:如果你注册了这项简洁的服务，你可以为你在 Bing 上的每一次搜索积累积分，最终你可以用积分兑换亚马逊、星巴克、GameStop 的礼品卡，甚至捐赠给你最喜欢的慈善机构。

最后，这是一个小问题，但也是一个有趣的问题:如果你在谷歌 上搜索某个网站，它不会显示任何知识图谱信息。冰会。

### 搜索结果的质量

现在我们来看真正最重要的:实际的搜索结果。如果一个网站不能给你你想要的，那么以上的功能都没有用。

当涉及到基本搜索时，我发现两个引擎，即使它们显示不同的结果，一般都能给我想要的。一个搜索在 Bing(“`raspberry pi`”)上可能会好一点，而另一个在 Google(“`avengers age of ultron`”)上可能会好一点，但总的来说都没有产生糟糕透顶或大相径庭的结果。我很惊讶地发现，必应在某些方面做得更好，但也只是稍微好一点。

当搜索一些真正具体的东西时，真正的差异就显现出来了，比如技术问题(“`project64 mario kart too fast`”)。如果你的问题的答案隐藏在某个大型论坛或博客中，谷歌可能会做得更好，找到它*和*组织结果。而且问题越具体，谷歌赢的就越多。

### 赢家:谷歌(但比你想象的要少)

客观地说，谷歌是更好的搜索引擎，我认为没有人对此感到惊讶。它存在的时间更长，迄今为止最受欢迎，并且在不断创新。我对这两个搜索引擎在功能上如此相似感到惊讶。 [必应甚至还做了几件更好的事情](https://lifehacker.com/the-best-things-bing-does-better-than-google-1483983630) ，比如视频。但是，虽然我偶尔更喜欢 Bing 的搜索结果，而不是谷歌的搜索结果，但谷歌从互联网最黑暗的角落挖掘东西的能力实在太有价值了。这比我预期的要难，但是谷歌仍然是最好的搜索引擎。
