# 为什么每个车载信息娱乐系统看起来都那么蹩脚？

> 原文：<https://lifehacker.com/why-does-every-car-infotainment-system-look-so-crappy-1794534701>

用户界面的设计很难，但是这些年来我们在这方面做得越来越好，甚至连恒温器都很容易使用。然而，汽车制造商似乎把他们的头埋在沙子里，他们的信息娱乐控制台的设计灵感来自旧的 Winamp 皮肤，而不是任何类型的现代界面。



我最近一直在购买汽车，这意味着我坐在各种新老车型中，一名销售人员试图证明为什么他们的劣质信息娱乐系统——通常至少控制收音机和电话选项的小仪表板——比竞争对手更好。再加上多年来的一些租赁，我看到了周围的每一个汽车信息娱乐系统。除了几个非常罕见的例外，它们普遍都很糟糕，让人觉得它们是由一个自 2000 年以来就没用过电脑的人设计的。

大多数汽车制造商都有自己品牌的信息娱乐系统，他们糟糕的设计不仅仅是表面现象。例如，奥迪有 [MMI](https://www.audiusa.com/technology/intelligence/mmi) ，这是一个*疯狂的*系统，使用拨号盘而不是触摸屏进行菜单导航，除非你花钱升级它。奥迪也不是唯一一个回避现代创新的公司， [宝马的 iDrive](http://www.bmw.com/com/en/insights/technology/technology_guide/articles/idrive.html) 和 [奔驰的 Comand](https://www.mbusa.com/mercedes/owners/videos/videoId-aa1de8650706e310VgnVCM2000007d184335RCRD) 也都依赖于表盘进行输入。多年来，雷克萨斯一直依靠基本上是鼠标的 [来导航](https://en.wikipedia.org/wiki/Remote_Touch) ，这听起来非常愚蠢，在现实生活中使用起来很有挑战性。

我甚至不会触及这些东西的导航系统有多糟糕，但请放心，它们就像你记忆中的一样糟糕。不管出于什么原因，老爷车 GPS 系统已经落后于智能手机选项很长时间了，即使在这些东西中输入地址不需要 35 分钟，但很有可能你在屏幕上看到的只是一张几乎无法阅读的像素化放大的混乱地图。

当然，也不全是坏消息。特斯拉至少看起来是在正确的轨道上，尽管他们汽车中的 [17 英寸触摸屏](https://www.tesla.com/software) 如此之大，我想这是一个巨大的干扰。我不知道，因为我连特斯拉都坐不起，更别说开了。沃尔沃的 [Sensus](http://www.volvocars.com/us/about/our-innovations/sensus) 似乎也在正确的轨道上，出于某种古怪的原因，这是一个垂直的屏幕而不是水平的，但在其他方面，这是一个四吨移动车辆中你想要的最小化设计的界面。它很容易使用，你可以在很小的范围内定制它，并且它通常是无害的。它也是全新的，以前的沃尔沃系统看起来不怎么样。

然后是 [苹果 CarPlay](https://www.apple.com/ios/carplay/) 和 [安卓 Auto](https://www.android.com/intl/en_ca/auto/) ，这感觉就像是汽车制造商放弃自己的系统，让别人来做。这两个选项都利用了您已经用于音乐和导航的手机的功能，也为汽车的信息娱乐选项提供了动力。

这有它自己的利弊。Android Auto 和 CarPlay 是对汽车制造商糟糕的第一方信息娱乐选项的普遍改进，但它们仍然存在一些问题。也就是说，应用程序必须实际支持这两种服务，这意味着你不能总是访问每个应用程序。例如，使用 CarPlay，你会被锁定在大多数第一方的苹果应用上，比如苹果地图。对于 iOS 用户来说，这是一个巨大的不利因素，因为即使苹果地图这些年来有所改进，但它仍然不如 [谷歌地图或 Waze](https://lifehacker.com/turn-by-turn-navigation-showdown-google-maps-vs-waze-1761550298) 。谢天谢地，你至少没有被苹果音乐或苹果播客锁定，因为[Spotify](https://itunes.apple.com/us/app/spotify-music/id324684580?mt=8)[Pandora](https://itunes.apple.com/us/app/pandora-music-radio/id284035177?mt=8)[Pocket Casts](https://itunes.apple.com/us/app/pocket-casts/id414834813?mt=8)[阴天](https://itunes.apple.com/us/app/overcast-podcast-player/id888422857?mt=8) ，还有一些支持 CarPlay。 [Android Auto 稍微好一点](https://play.google.com/store/apps/collection/promotion_3001303_android_auto_all?hl=en) ，但是在你拥有的第三方选项上还是蛮有限的。无论选择哪一种，很有可能至少有一个你用来播放音频的应用程序不能使用它。

两者都有一个小问题，那就是你仍然需要偶尔使用汽车的劣质信息娱乐系统，因为 CarPlay 和 Android Auto 都不能处理任何汽车范围内的设置更改。尽管如此，这两者都是一种进步，如果没有别的，至少表明汽车制造商已经完全放弃试图强迫我们接受他们设计糟糕的系统。

但是，并不是每个人都有 iPhone 或 Android 设备，也不是每个人都想首先将他们的手机连接到他们的汽车音响上。无论如何，这些汽车信息娱乐系统和它们愚蠢的名字将成为未来几年生活的一部分，所以如果它们至少看起来像是过去十年设计的就好了。

我们这里不需要太多！只是不要将菜单嵌套在其他子菜单中，将你的图标设计交给真正的设计师，而不是从 Shutterstock 那里获得一些图标，使用驾驶时易于点击的大按钮，并额外花费几美元给整个东西带来现代光泽，这样它就不会看起来像 1998 年的东西。

听着，我知道这很难，也很贵。我知道汽车通常只有古老的硬件，除了一个文本界面之外，不能运行太多。我知道这几乎是汽车优先列表的底部，但也许是时候把它向上移动一点了。

见鬼，你甚至可以切断对任何设置的访问，如果我调整错误，这些设置可能会让我丧命，然后打开 Linux 风格的娱乐部分，这样我就可以在那里安装我想安装的任何东西。如果没有别的，至少有操作系统升级选项就好了，这样我至少可以有改变的希望。
