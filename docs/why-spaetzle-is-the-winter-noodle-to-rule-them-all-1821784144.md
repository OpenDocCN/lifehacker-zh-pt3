# 为什么面条是统治所有人的冬季面条

> 原文：<https://thetakeout.com/why-spaetzle-is-the-winter-noodle-to-rule-them-all-1821784144>

面条是我最想吃的食物，尤其是那种又辣又粘的蒜味炒面。但是一旦温度降到冰点以下，就全靠面条了，宝贝。将这些德国鸡蛋面换成意大利面是如此简单的冬季厨房小窍门，以至于我觉得有必要传播他们的福音。

为什么你应该立即用所有的意大利面/米饭/碳水化合物换成面条？我的案例:

1.  你可以自己做沙拉！我的 Oma 的食谱(如下)只需要四种原料:面粉、鸡蛋、水和盐。我认识的人中很少有人有时间或精力在工作日的晚上自制意大利面，但 spaetzle 不到 10 分钟。
2.  Spaetzle 非常好吃。制作得当，它具有耐嚼、有弹性的质地，仍然具有吸水性，可以吸收所有深层果汁、肉汤或炖肉，这些都是冬季菜肴的最佳组成部分。
3.  Spaetzle 也可以酥脆！如果你的菜需要咬一口，只需在煮沸后用黄油煎炸你的面条。它们的边缘长出漂亮的褐色外表，比米饭或意大利面含有更多的美拉德营养。
4.  它们超级能填饱肚子。我不确定是鸡蛋的原因还是 spaetzle 用了一大堆面粉，但这些小块的快乐以冬天最好的方式粘在我的肋骨上。

我已经成功地在各种食谱中用面条代替淀粉。显然，它们非常适合用来浸泡炖肉，比如炖牛肉，但也很适合用来代替鸡肉面汤中的面条。我妈妈总是把它们加到火腿和豌豆汤里，这样会让碗更丰盛。你甚至可以将它们与奶酪和焦糖洋葱一起烘烤，来一份德国的 MAC ' n ' cheese(*kaesespaetzle*)。你相信吗？请继续阅读。

**关于批量的注意事项:一般的经验法则是每半杯面粉加一个鸡蛋，所以扩大这个食谱很容易。为你服务的每个人准备一个鸡蛋。下面的食谱足够四个人吃，但是即使你只做一两个人吃，也要做双份。它冷冻得很好，没有人会对剩饭剩菜生气。

## My Oma’s Spaetzle

*   4 个鸡蛋

*   2 杯万能面粉

*   杯子温水

*   盐的味道

Optional:

*   黄油

*   切碎的欧芹

1.  烧一大锅水。
2.  将面粉放入一个中等大小的碗中，最好是重型搅拌碗(碗的重量有助于你将面团中的废物打出来)。撒些盐。
3.  加入鸡蛋。用一把木勺，把鸡蛋揉进面团里。技巧是这里的关键:不要搅拌面团，而是用弧形运动用力将鸡蛋打入面粉。你试图在面团中制造气泡，这样面条会变得又轻又蓬松。如果面团看起来太干，加入温水。当面团变得有弹性时，当你举起它的时候，让它粘在木勺上，形成气穴。如果面团变得又粘又亮，你就做对了。
4.  一旦面团粘在碗和勺子的边上，用压勺或漏勺将面团从孔中压入沸水中。你也可以用削皮刀把皮刮下来做成更大的饺子。
5.  当面条浮到顶部时(这根本不需要太多时间)，用漏勺捞出。用毛巾风干。

或者，你现在可以在煎锅里加热一些黄油和香草，如果你想让面条变脆，就用平底锅煎。
