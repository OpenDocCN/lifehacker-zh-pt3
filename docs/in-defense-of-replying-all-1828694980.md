# 在答辩所有

> 原文：<https://lifehacker.com/in-defense-of-replying-all-1828694980>

这些天来，我们的收件箱里塞满了垃圾邮件，远远超过了实际指定的“垃圾邮件”文件夹。有我们发誓已经退订的时事通讯，来自政客的令人毛骨悚然的个人信息，来自脸书的绝望的关注请求，现在，由于没有人要求的最新 Gmail 更新，像“嘿，凯特，已经五天了！为什么你没有回复那个公关代表，他仍然不知道你五个月前被解雇了？”

Watch

但我认为，在这个垃圾堆中，可以抢救的部分是你的朋友和聚会伙伴的杂乱无章、永无止境的团队线索。回复-一切都应该珍惜，而不是憎恶。线程数越多越好。如果你因为不能“相信”他们不会“辱骂”回复所有人而密件抄送给你所谓的朋友去烧烤，考虑一下你实际上讨厌这些人的可能性，你应该交新朋友。

当然，在过去，我们看不到宾客名单——在更正式的场合，如婚礼或毕业派对，通过蜗牛邮件发送的请帖也是如此。多么古怪。但是当你给我发密件的时候，那种场合就像，周五晚上的卡拉 ok，那会让你看起来很可疑。你在密件抄送名单上藏了什么，或者是谁？

“密件抄送在数字上相当于在聚会上把每个人分别拉到一边，告诉他们同样的事情，”我的朋友萨宾说，她明白这一点。“只要告诉小组成员，让他们互相交流。不要做 BCC 社会独裁者。”

我听到有人说，人们使用密件来保护群体电子邮件地址的隐私。但是，你在保护谁呢？如果你担心你的一个朋友会给名单上的某个人发不恰当的私人信息，也许问问你自己，让不值得信任的人做朋友是否值得。至少，把他们从任何未来的群发信件中驱逐出去，单独向他们发出邀请。

随着你的“无毛骨悚然”政策的确立，有很多很好的理由让 CCs 开放，让你的朋友们也参与进来:

### **电子邮件线程充当一种预备方**

萨宾说，抄送不仅是“我们都应得的民主电子邮件系统”，而且“无论如何，任何计划中最好的部分都来自抄送邮件线程，”她说。表面上，你邀请这群人是因为你喜欢他们，认为他们很有趣，擅长聚会，那么为什么不创建一个论坛，让他们可以发表评论并提出建议，甚至可能改善聚会呢？

我的朋友戴夫热衷于群发抄送邮件，也是回复邮件之王，他同意这一切都是为了娱乐。“我认为回复所有人给了彼此是朋友的人一个四处闲逛和做迷因的机会，也让原本正常的电子邮件去了陌生的地方，”他说。多年来，我一直是这些电子邮件的接收端，虽然我可以证明，当然，一些回复比其他人更机智，一些“奇怪的地方”也许最好不要探究，我不是如此势利，一个黯淡的 GIF 会成为交易的破坏者，导致我拒绝了一个否则很有希望的邀请，再也不和一群人说话。我宁愿有那些有点邋遢或令人讨厌的朋友，不管是在网上还是网上，也不要那些因为我给他们发了一个糟糕的双关语而监视我的朋友。

### 你会让更多的人来

如果你能克服对回复所有邮件的非理性恐惧(仔细想想，这本质上是害怕收到朋友的电子邮件，这没有任何意义？)出于几个原因，抄送一份聚会邀请是很实际的。这里有一个:几乎不可能记住邀请你想邀请的每个人。通常，发件人会附上一个免责声明，比如“把这个转发给我忘记的任何人！”但是如果你用密件抄送这封邮件，正如一位明智的朋友指出的，“如果人们不知道你邀请了谁，他们怎么知道你错过了谁？”废话。

在收件人方面，收到抄送的邀请可以让潜在的社交活动更具吸引力。“就我个人而言，如果我知道哪些人/哪些朋友会去，我更倾向于去参加聚会，”另一位匿名朋友说。"密件抄送的邮件也让人感觉不那么私人，因此更容易被忽略."也许当你看到你的心上人受到邀请时，你会变得兴奋(花时间做好心理准备/为此困扰几天/计划好你的服装，无论你如何处理这种痛苦)，或者是一个你很久没见的朋友。也许你会看到你的邻居受到邀请，并选择与他们拼车，或者在聚会前喝一杯饮料。

### 你节省了自己的时间

从逻辑上来说，以密件抄送的方式发送一封计划邮件可能会很麻烦。如果您需要发送后续邮件或提醒客人计划的变化，您必须将每个电子邮件地址复制并粘贴回“密件抄送”字段，而回复所有邮件只需一次点击即可完成。

### 你把你的朋友从潜在的不愉快争吵中解救出来

如果你仔细想想，让客人名单可见是对你朋友的一种体谅。如果你的死对头或前任会出席，提前打个招呼会有帮助。在社交场合对那些你表面上很在乎并邀请参加你的聚会的人视而不见是不好的。把他们当成成年人来尊重，给他们所需要的信息，让他们做出明智的决定，如何和谁一起度过他们的时间。此外，如果你使用密件抄送来避免群发邮件时友敌共存的混乱局面，那么当他们在同一个房间时，你会有什么感觉？

知道谁被邀请了也让每个人避免了向朋友提起这个活动却发现他们没有被邀请的尴尬。

对于所有“密件抄送”的支持者——或者我喜欢称他们为“披露的敌人”——当人们回复时，他们很快就会翻白眼，滔滔不绝地说这是多么糟糕的事情——我要问:到底是谁伤害了你？当然，也许你的朋友是垃圾，但是不要把他们当成垃圾邮件。如果堆积如山的回复真的让你厌烦，有一个简单的解决方法:把线程静音。不要拒绝其他人他们渴望并应得的社交论坛。朋友不让朋友互相密件。
