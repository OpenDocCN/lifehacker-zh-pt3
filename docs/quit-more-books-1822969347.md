# 停止阅读你并不真正喜欢的书

> 原文:[https://lifehacker.com/quit-more-books-1822969347](https://lifehacker.com/quit-more-books-1822969347)

有些人一旦不再喜欢一本书，就知道如何戒掉它。但是我们中的许多人都感到某种完成主义的压力，要坚持读完我们开始读的每一本书，即使是为了娱乐而阅读。我们纠结于我们实际上不喜欢的东西，所以我们不太可能拿起书，而更可能拿起电话。我们开始减少阅读。

Watch

如果你希望你能读更多的书，试着戒掉你正在读的书。如果你不在的时候，它不是每分钟都在召唤你，也许你应该放下它，去找一本能召唤你的书。事实上，每当一本书连续两页(或五页，或十页)让你厌烦时，就放弃它。向前看。如果你想知道接下来发生了什么，你随时可以回来。

问题是，当你退出一本书的时候，*你必须开始读另一本书*。最理想的是就在那一分钟。你必须继续阅读，但你可以阅读任何你想看的*。*

*如果第二本书很无聊，你也可以放弃。你可以连续退出的次数没有限制。你永远不会没有书。仅你当地的图书馆拥有的免费书籍就比你有生之年能读到的还要多。*

*我最近读了卡夫卡的短篇小说集——嗯，大部分都是。这些故事是按时间顺序排列的，所以一开始我只是涉水阅读一些即使是文学专业的学生也不会被要求阅读的小故事。当一个朋友问我在读什么时，我开始一连串地焦虑，觉得自己对卡夫卡“不够聪明”。他温和地问了一个显而易见的问题:为什么不直接跳到好的方面呢？*

*我照做了，这太棒了，我的阅读速度加快了，因为我只阅读那些吸引我的故事。我没有读完每一个卡夫卡的故事，但我读了所有重要的故事，而且我很快就读到了《地海巫师 T2》。好书！*
