# 问咸女服务员:救命！我的服务员说话像个千禧年的傻瓜

> 原文：<https://thetakeout.com/help-my-server-speaks-like-a-millennial-nitwit-1829339247>

> 亲爱的 Salty:在过去的几年里，我注意到很多服务员(大多在时尚餐厅)用我怎么说呢，互联网蠢驴千年语言跟顾客说话。没有更多的礼仪，如:“食物怎么样？”相反，我们会问:“我们对烤章鱼有兴趣吗？黑仔，对吗？”或者“伙计，你打败了尼格罗尼”上周，一个服务员称我妻子为“伙计”
> 
> 我 40 岁了。那不算老。当我说在主菜要 35 美元的地方被这样称呼是荒谬的时候，我不觉得自己像个爷爷。我能做些什么来表明我更愿意像一个顾客而不是某人的 Twitter 追随者一样被交谈？
> 
> 谢谢，不是你的兄弟，兄弟

亲爱的非兄弟，

我不能删除互联网，我不能抓住所有这些自以为是的人的耳朵，我不能破译我十几岁的侄女给我发的一半短信。所以，你知道，不要指望我在这件事上移山倒海。

Watch

但我能做的是张开我那张美宝莲粉色的大嘴，向所有时尚餐厅的服务员发出一份公益广告:顾客不是你的“闺蜜”有一种方法可以做到友好而不成为*朋友*，亲切但不太随意。当然，你们可以告诉我，我听起来像一个大老 b。但这是你的小费，不是我的，你们这些小讨厌鬼，这封电子邮件不是我第一次听到这样的抱怨。

顾客在你的餐厅能感受到的最糟糕的事情之一就是不被尊重或不舒服。不管你喜不喜欢，我们与顾客交谈时使用的语言都是尊重的信号——或者缺乏尊重。听着，我知道你在餐馆工作很努力。你想被认真对待，对吗？有时这意味着练习你选择的词语，让顾客感到舒服。这是好客的一部分。经理们，如果你觉得“花花公子”病毒在员工中传播，你可以在员工会议上讨论这个问题。

对服务器来说，一个很好的经验法则是:倾听你的客户是如何说话的，然后照着说。(这是我在零售业短暂工作期间学到的一个小技巧。)如果他们很安静，就不要逼他们。如果他们好奇，就给他们详细的讲解。如果他们对你漫不经心，那你也要漫不经心。但是如果他们不是，试着跟上他们设定的基调。

现在，回到你，写信人。监管他人的语言是很难的，因为语言与文化、我们来自哪里以及我们是谁息息相关。我知道这种千禧一代的网络垃圾让你很烦，但是我们不想因为别人和我们不一样就在别人说话的时候到处拉屎。所以，虽然这让你很生气，但你可能不得不放手。我是说，还有什么选择？向经理投诉？我拿出了咸 PSA，所以希望餐馆注意到并淡化口头表情符号。

哦，如果你想减轻听到你妻子被称为“伙计”时的震惊，也许可以把它当成对她年轻的赞美。
