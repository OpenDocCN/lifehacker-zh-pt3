# 每个厨房都需要的三把铲子

> 原文：<https://lifehacker.com/the-three-spatulas-every-kitchen-needs-1800013242>

虽然我想在我的厨房用具瓦罐里放上现有的每一把星球大战主题的锅铲，但我没有足够的空间——或者说没有必要——放那么多的锅铲。真的，一般的家庭厨师只需要三把铲子:



### 鱼铲

[正如我们之前讨论过的](http://skillet.lifehacker.com/kitchen-tool-school-the-flippin-awesome-fish-spatula-1683871427#_ga=2.239900163.1498726717.1504461867-949419976.1446553382) 一样，鱼铲不仅仅可以用来翻鱼。它长而结实，但超级薄，非常适合翻转和转移各种精致的食物。(老实说，如果你能把一整条鱼翻过来而不把它弄干，那你有什么*不能*翻过来的呢？)它也是一个了不起的刮刀，它的狭槽使它非常适合从油中捞出油炸食品。(而且，如果你倾向于在不粘的情况下做很多烹饪，OXO 制造了一种 [硅胶涂层的](http://skillet.lifehacker.com/kitchen-tool-school-the-flippin-awesome-fish-spatula-1683871427) 那种， [根据 Epicurious](http://www.epicurious.com/expert-advice/the-only-four-spatulas-you-need-article) 的说法，耐热高达 600℉。)

### 橡胶或硅胶抹刀

橡胶或硅胶抹刀是可怕的鳍状肢，但它们擅长在炒鸡蛋和折叠蓬松的蛋清中滑行。它们也是 ace 碗刮刀，非常适合确保您将每一点布朗尼面糊都放进锅里。很多人更喜欢用单一材料制成的——我偏爱光滑的硅胶——这意味着头部不会在不合时宜的时候从身体上掉下来，但我从未对 Le Crueset 的 [这个木制手柄的数字有过异议。](https://www.lecreuset.com/revolution-spatula-spoon) (我也爱 [这个来自 Tovolo](http://www.tovolo.com/product/silicone-scoop-and-spread-2/) 的小家伙，它的另一端有一个尼龙扩张器。)

### 木车工

你可能只需要以上两个就可以了，但我喜欢身边有一个 [多用途木铲](https://www.amazon.com/Eddingtons-Italian-Olive-Spatula-12-Inch/dp/B001APPMKM/ref=sr_1_8?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-three-spatulas-every-kitchen-needs-1800013242&asc_source=&ie=UTF8&keywords=wooden spatula&qid=1504635582&s=kitchen&sr=1-8&tag=kinjalifehackerlink-20) ，以防我的鱼铲或硅胶铲(咳咳，脏)。这东西可以刮，搅拌，折叠和翻转，只要你好好照顾它，它永远不会死。

这三个将完成所有的工作，在你的器皿缸中留下空间给你令人印象深刻的木勺收藏，或者，如果你必须这样做的话， [这个死星铲](https://www.amazon.com/dp/B01DE9YKW2/ref=asc_df_B01DE9YKW25155825/?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-three-spatulas-every-kitchen-needs-1800013242&asc_source=&creative=395033&creativeASIN=B01DE9YKW2&hvadid=167134584690&hvdev=c&hvdvcmdl=&hvlocint=&hvlocphy=9032938&hvnetw=g&hvpone=&hvpos=1o5&hvptwo=&hvqmt=&hvrand=10229478149875285183&hvtargid=pla-305175582889&linkCode=df0&tag=kinjalifehackerlink-20) 。
