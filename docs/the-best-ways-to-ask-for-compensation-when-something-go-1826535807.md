# 当您的航班出现问题时，要求赔偿的最佳方式

> 原文：<https://lifehacker.com/the-best-ways-to-ask-for-compensation-when-something-go-1826535807>

虽然大多数航班顺利起飞，但恼人的事情还是会发生。延误、改道、坏掉的托盘桌、故障的娱乐控制台以及不稳定的无线网络都会让一次普通的飞行变成一次不愉快的经历。当不好的事情发生时，考虑像这样要求赔偿。



在我们进入细节之前，请记住法律只要求 [航空公司赔偿你](https://www.transportation.gov/airconsumer/fly-rights) 的一些东西。即非自愿的颠簸，停机坪上的不合理延误，行李丢失。其他一切由航空公司决定。一般来说，如果他们在合理的时间内安全地把你带到你需要去的地方， [你就没有多少立足之地了](https://lifehacker.com/the-myth-of-flyer-s-rights-1795169133) 。尽管如此，出了问题要求赔偿也无妨。法律可能不要求他们帮助你，但他们有兴趣留住客户。

然而，在你提出要求之前，你需要知道什么值得补偿，什么不值得。例如， [某人在你前面斜倚着他们的座位](https://lifehacker.com/the-person-reclining-their-airplane-seat-is-not-your-en-1823808027) ，令人讨厌的颠簸，被迫坐在人类胀气的旁边，他们用完了你的膳食选择，这些都不是你应该期待补偿的事情。抱歉，朋友，这些都是因为运气不好。但是，折断的扶手、失灵的耳机插孔、计划外的航班改道，以及当你明确提到自己过敏时，空乘人员会分发花生，这些都可能值得询问。如果你在航空公司有任何精英身份，这一点尤其正确。他们更有可能补偿忠诚计划的成员。

当出现问题时，确保尽可能多地记录问题，这样你就可以清楚地解释出了什么问题，以及为什么它对你来说是个大问题。写下发生的事情，并在登机牌上写上你的名字，拍照， [一定要写上重要的航班信息](https://www.forbes.com/sites/grantmartin/2014/10/24/how-to-complain-to-airlines-on-twitter-and-what-youre-doing-wrong/#1e458878afe4) ，比如你的出发日期、航班号、出发和到达机场以及行李标签号。

当谈到像美国航空公司、达美航空公司和联合航空公司这样的大型传统航空公司时， [读者在积分盖伊建议](https://thepointsguy.com/guide/what-to-ask-for-flight-issues/) 以下是一些索要赔偿的最佳方式:

*   询问空乘人员:空乘人员可能会当场为你提供某种补偿。这可能是免费的一餐或一杯饮料，甚至是将航空里程计入你的账户。例如，美国航空公司(American Airlines)的空乘人员开始使用一款名为 iSolve 的平板电脑程序，该程序允许他们轻松提供飞行补偿。wifi 坏了吗？有一个破托盘桌？有没有一个讨厌的乘客坐在你旁边？说点什么！

*   在他们的客户服务账户上发微博:虽然一封老式的信或电子邮件也可以，但发微博肯定能得到某种回应。航空公司不希望关于他们的未解决的投诉在公共空间传播，所以他们通常会试图快速处理这些问题。伸手向 [美国](https://twitter.com/AmericanAir) 、 [德尔塔](https://twitter.com/Delta) 、 [美联航](https://twitter.com/united) 等人在推特上用快速、冷静的方式描述问题。尽量不要太热。你可能会生气，但没有人愿意帮助一个不理智的人。

如果事情没有进展，一定要跟进你的要求。记住，除非你提出要求，否则你通常不会得到任何补偿。航空公司不想分发免费的积分和积分，所以当你认为你应该得到某些东西时，要积极主动。尽管试一试无妨，但当你要求赔偿时，不要期望太多，除非有什么东西侵犯了美国运输部的飞行权 。
