# 你应该能回答的钱的问题

> 原文：<https://lifehacker.com/the-money-questions-you-should-be-able-to-answer-1823456583>

美国人真的是 [强调](https://www.thebalance.com/survey-says-even-the-rich-are-stressed-about-money-1289584) 出来讲钱。这是可以理解的: [储蓄低](https://www.valuepenguin.com/banking/average-savings-account-balance) ，开销高，我们背负着 [越来越多的债务](https://www.usatoday.com/story/money/personalfinance/2017/04/28/average-student-loan-debt-every-state/100893668/) 有时我们似乎永远也赚不到足够的钱来退休或去度假，这是我们梦寐以求的。



缓解你资金紧张的一个方法是了解你财务状况的基本情况。这似乎是显而易见的，但是问问你自己——你最后一次检查你的 401(k)费用是什么时候？你知道你的信用卡利率吗？

那么你对自己的财务状况到底了解多少呢？如果你和大多数人一样，可能不是很多。这里有一些你应该能够回答的问题。

### 你有多少债务？

根据[*The Atlantic*](https://www.theatlantic.com/business/archive/2015/09/americans-and-debt/403081/)报道，2015 年 [的一项研究](https://www.newyorkfed.org/medialibrary/media/research/epr/2015/EPR_2015_comparisons_brown.pdf) 发现，美国人“估计他们的欠款比他们的贷款人说的信用卡欠款少 40% ”,并且“列出他们的学生贷款总额比信用报告上显示的少大约 25%。这可不好。如果你要还清债务，第 0 步是知道你有多少钱。

一个额外的问题:你要多久才能还清债务？如果你没有一个好的想法，那就意味着你没有一个还清的计划。你应该这么做。你的计划应该详细说明你需要多长时间(以月或年为单位)来还清债务，以及你计划何时增加还款。如果你已经结婚或处于长期关系中，你需要知道你的伴侣有多少债务，因为这也影响到你。

有些债务(比如学生贷款)需要你一段时间才能还清，这没关系。但是一个灵活的计划会使它更容易管理。

### 你的储蓄账户里有多少钱？

这似乎是一个很容易回答的问题，但是你上一次查看你的储蓄账户余额是什么时候？知道你存了多少钱会让你知道你的消费习惯，并让你制定一个计划(是的，另一个)来建立你的现金缓冲。

如果你有超过几个月的开销(甚至几千美元，取决于你是谁)，你可以考虑将其中一部分转入个人退休帐户、经纪账户或其他目标。再说一次，如果你们是共享财务的夫妻，但你让你的伴侣处理一切，这是加倍重要的。你需要知道你有多少。这是走向财务自由的第一步。

### 你的信用卡余额是多少？你的信用卡利率是多少？

你可能不再用账单来支付日常开销了。刷( [或挥动](https://twocents.lifehacker.com/what-to-know-about-contactless-payment-limits-1823355784) )塑料卡或通过应用转移资金比分发现金更容易。这意味着很容易超支。

但是每月控制你的信用卡余额是很重要的。你通常可以得到发短信给你的交易和余额信息，或者养成每天查看银行 app 的习惯。为了最大化你的信用评分，尽量不要让 [超过你限额](https://www.creditkarma.com/credit-cards/i/credit-card-utilization-and-your-credit-score/) 的 30%，并在整个月付清。

如果你不能每月付清信用卡账单——嘿，生活就是这样——你需要知道你将被收取多少利息。这将有助于你区分优先偿还哪些债务(最高的年利率)，在紧急情况下使用哪张卡(最低的年利率)，它可以鼓励你寻找一张整体利率更好的卡。

### 你的 401(k)费用是多少？

这不是你必须要记住的，但是当你注册 401(k)并增加资金时，你应该看看费用。最近 [TD Ameritrade 调查](https://s1.q4cdn.com/959385532/files/doc_downloads/research/FY2018/Investor-Sentiment-Infographic-401k-fees.pdf) 发现，60%的投资者“要么不知道他们的计划是否收费，要么错误地认为他们没有被收取任何费用，”Real Deal Retirement 的 [Walter Updegrave](http://time.com/money/5137127/retire-richer-401k-mutual-fund-fees/) 写道。

那是一个代价高昂的错误。即使是看似很小的费用差异(比如 0.25%对 1%)，在 30 年的时间里也会产生很大的影响。不仅仅是因为费用消耗了你越来越多的储蓄，还因为这些损失是复合的。

另一个原因？晨星公司的研究 显示“成本较低的基金往往比成本较高的同行做得更好。”

你的费用应该是多少？根据美国最佳 401k 计划的首席战略官乔希·罗宾斯 的说法，“介于 0.5%和高端的 0.75%之间。”“全押”不仅指你的计划提供商(比如 Vanguard 或 Fidelity)收取的费用，还指你投资的共同基金本身收取的费用。

所以检查一下你的费用，大多数公司都在他们的网站上提供。如果你的雇主不能提供足够的低费用选择，考虑开设个人退休账户。

### 你想赚多少钱？

我想澄清的是，金钱不是一切，有无数我们无法控制的因素影响着我们的收入。更简单地工作以赚更多的钱不是生活的方式。也就是说，我提出这个问题是因为我相信 [设定目标](https://twocents.lifehacker.com/howd-you-do-on-your-month-long-financial-detox-1823370993) 并且在生活的不同领域有所向往和努力。在你的财务生活中，心中有一个数字可以帮助你做出职业决策，为自己创造更好的生活。

问问你自己:你在为做什么？如果是财务独立，那么，你需要赚一定的钱，以便能够存够钱提前退休或自己创业。如果是为了你的家人，他们有需求和愿望，这些都需要花钱。只有你知道这个数字或范围是什么。它应该是灵活的，如果你没有在 X 日期前实现它，你不应该责备自己(同样，你不能控制一切)。

当然，并不是所有的事情都是或者应该是关于钱的。但是因为我们 99%的人需要工作来谋生，所以设定目标来确保我们从中获得最大收益并没有错。

如果你不能回答这些问题，在这个周末或者在你的下一个 [个人盘点日](https://twocents.lifehacker.com/schedule-a-personal-inventory-day-each-month-1821807681) 花些时间来解决它们。
