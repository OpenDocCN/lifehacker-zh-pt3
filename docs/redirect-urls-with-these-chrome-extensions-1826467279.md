# 使用这些 Chrome 扩展重定向 URL

> 原文：<https://lifehacker.com/redirect-urls-with-these-chrome-extensions-1826467279>

如果你是 Chrome 超级用户，你最终会想要设置一个自动 URL 重定向。也许你想 [在一个极简主义的网站](https://lifehacker.com/automatically-open-youtube-links-in-an-uncluttered-view-1825389630) 上看你所有的 YouTube 视频；也许你喜欢——或者讨厌——去一个网站的移动版本。也许你只是每次输入某个网址都犯同样的错别字。有许多用于重定向 URL 的 Chrome 扩展，每一个都有它的长处。我们找到了两个最好的选择:简单的和强大的。



## 新手

最简单的选择就是 [Switcheroo](https://chrome.google.com/webstore/detail/switcheroo-redirector/cnmciclhnghalnpfhhleggldniplelbg) 。很好理解，如果不是很厉害的话。它允许您编辑现有的重定向，但除此之外它没有太多的附加功能。它比更强大但不可靠的扩展更容易使用。

要重定向一个 URL，只需从工具栏打开扩展，输入输入 URL(就像每个 YouTube 视频 URL 的标准部分)和输出 URL(就像 YouTube 的一个 [极简替代)。](https://lifehacker.com/avoid-unexpected-youtube-bing-watching-sessions-with-tu-1825275041)

输入有点不稳定:如果你在点击“添加”之前离开，Switcheroo 会忘记你输入的所有内容。所以如果需要粘贴两个字符串，在第一个后点击“添加”，然后复制第二个字符串，点击“编辑”添加即可。这很蠢，但不是交易杀手。

## 先进的

如果你需要更高级的特性，比如通配符，请尝试扩展。这需要更多的工作来理解，但除了 URL 重定向，它还允许您阻止某些 URL、插入脚本或模拟不同的设备。(我用它来清除我的亚马逊链接中的搜索词。)对于开发者或者超级用户来说是一个很棒的选择。

Requestly 也有一个更复杂的界面，你可能会发现比 Switcheroo 的界面更好或更差。要添加新的重定向，从工具栏打开扩展，单击右上角的+按钮，并从整页菜单中选择*替换主机*(蓝色)。

至于安全性，Switcheroo 开发者 Ranjdar Abass 告诉 Lifehacker，他的扩展只在你的电脑上存储数据，不在任何服务器上。(可以在 GitHub 上阅读 [Switcheroo 的源代码](https://github.com/ranjez/Switcheroo) 。) [Requestly 的隐私政策](https://www.requestly.in/privacy/) 称，该扩展不会存储个人数据或私人 URL 重定向规则，但会在其服务器上存储公共规则(如果你选择使用谷歌登录，还会存储你的电子邮件地址)。两者都是安全的，除非你在你的 URL 重定向中存储了国家机密或高犯罪。
