# 当你室友的男朋友不受欢迎的时候

有些人遇到问题，需要合格的专业人士给予微妙的建议。其他人只需要在网上随便找一个人，用他们直率的诚实狠狠揍他们一顿。我是后者。欢迎回到 [彪悍的爱情](http://lifehacker.com/tag/tough-love#_ga=2.269169905.1295021153.1516057126-819016074.1513967708) 。



本周我们有一位房主，他觉得他们室友的男朋友可能欠他们一些房租，因为他经常住在那里。

> 你好帕特里克，
> 
> 我有一套房子，有两个朋友和我一起住，做室友。我给他们很好的租金，因为他们赚不了多少钱，而且我喜欢帮助我的朋友(在我看来，不必忍受完全陌生的室友是一个巨大的优势)。

> 最近，我的一个室友，让我们称她为 J，有了一个男朋友，虽然他是一个很好的人，但他已经连续一两周几乎每天晚上都在我家过夜，我开始怀疑我是否应该收取租金——特别是现在有些早上，当所有人都去工作后，他实际上是唯一一个留在家里的人。事情是这样的，正如我们共同的朋友所指出的那样，他没有(据我所知)吃我们的食物，他通常呆在她的房间里，没有真正闯入公共空间或任何东西，而且，据我所知，他没有在我们的地方洗澡或任何东西。
> 
> 另一方面，我给了 J 一个合理的租金，而且我不收我任何一个室友的水电费。我想最终我觉得她是在利用这种情况。最重要的是，是我的另一个室友先向我抱怨的，值得称赞的是，他的女朋友通常一周只呆两三次，这在我看来更合理。我是不是太敏感了，还是我有理由这么做？我试着和 J 讨论这个话题，但是当我被问到为什么我在乎时，我不知道如何回答，所以我只是暂时放弃。
> 
> 真诚的，乐于助人的房主

**嘿，乐于助人的房主** :

你需要小心不要对你的房客，呃，我是说，朋友太好了。给朋友提供租金优惠以避免和陌生人住在一起是一回事，但是支付他们的水电费(为什么？)而让他们在你身上走来走去就太过分了。你不希望你的朋友，那些应该关心你和你的感受的人，因为你不愿意说不而利用你。如果你觉得 J 的男朋友呆得太久了，就是这样，这就是你的感受。你不需要用图表来解释为什么它会困扰你。“你为什么在乎？”j 问。因为你知道。

如果这个家伙不吃你的食物，躲在你的视线之外，而且从来不在那里洗澡，这都没关系(值得怀疑)。他就在那里——连续几天使用你的庇护所，吸收你付费购买的热空气或冷空气，磨损地毯，在厕所里拉屎撒尿，到处掉死皮，可能还在那里存放个人物品。当然，它蜷缩在视线之外，但它基本上是在你的房子里免费筑巢。你知道那还有什么作用吗？老鼠。他们可能只会吃旧的、腐烂的垃圾，一直藏在墙里，从不使用你的淋浴，但这并不意味着你想让他们呆在你的私人空间里。更糟糕的是，J 和这个人都没有问你对他一直在你家过夜有什么感觉，这很不好。当你提出这个问题时，她会变得很有防卫性并质疑你的动机，这是非常不酷的。再加上你的另一个室友觉得这种行为很讨厌，毫无疑问需要做点什么。

现在，听起来你并没有在你的室友搬进来的时候给他们很多限制，这是一个很大的错误，即使他们是朋友(尤其是如果他们是朋友)。不管你喜不喜欢，你是他们的房东，偶尔不得不这样做。大多数住宅都有关于非房客可以连续住宿多长时间的政策，所以这应该是你尝试实施的第一件事。告诉住在那里的每个人，客人一次最多只能住三个晚上，除非有特殊情况，或者类似的情况。如果他们问为什么，告诉他们这是你的房子，事情就是这样，他们很幸运你为他们提供了这样一个负担得起的居住环境。如果他们不想遵循你的非常简单和荒谬的合理规则，他们可以住在其他地方，那里的租金更贵，他们必须支付自己的水电费。或者，你可以随时要求 J 的男朋友正式搬进来交房租。见鬼，如果你的墙壁里有一只老鼠，你也可以从中得到一些东西，对吗？

过度敏感，HH？你不够敏感！别说什么超级支持的朋友废话了，好好做生意吧。我是说，你的要求并不高。j 要么申请一个新地方，要么闭嘴。这是你的房子，不是他们的。不要忘记这一点。
