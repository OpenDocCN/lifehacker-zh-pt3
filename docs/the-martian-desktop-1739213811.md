# 火星桌面

> 原文:[https://lifehacker.com/the-martian-desktop-1739213811](https://lifehacker.com/the-martian-desktop-1739213811)

[安库什](https://www.flickr.com/photos/ankushmenat/) 倾心于*火星人*，我们真的不能怪他。他没有提到他是受这本书的启发还是受电影的启发制作了这个桌面，但不管怎样，我们都喜欢它。下面是如何设置它。

Watch

Ankush 在这里运行 Windows，屏幕底部的可视化工具会随着他正在听的音乐及时移动，所以当你设置好之后就可以期待了。这样一来，下面是在您的系统上重新创建它所需的内容:

*   [火星上的日落](http://mushstone.deviantart.com/art/The-Martian-Sunset-on-mars-565114998) 壁纸
*   用于 Windows 的 [雨量计](http://rainmeter.net/) 系统调试和监控工具
*   将 [编码皮肤](http://lilshizzy.deviantart.com/art/Rainmeter-Encoded-v1-2-3-215992941) 为时间和日期显示在桌面上
*   [兰诺可视化器](http://marcopixel.deviantart.com/art/Lano-Visualizer-Highly-configurable-visualizer-545915953) 为雨量计屏幕下方创建的可视化器
*   雨量计 的 [Spotify 插件，将 Spotify 的当前曲目拉入雨量计](https://forum.rainmeter.net/viewtopic.php?f=18&t=17077)
*   雨量计的 [明文](http://redsaph.deviantart.com/art/Cleartext-for-Rainmeter-v2-6-1-17-Sep-2015-519796161) 插件，显示桌面上正在播放的 Spotify

大概就是这样。你可能需要争论一下，才能让 Spotify 中正在播放的内容正常工作，但这并不困难。如果你有问题，或者只是想分享你有多喜欢你所看到的，去 Ankush 的 Flickr 页面(链接如下)让他知道你的想法！
