# 开立一个子账户，为这个盛大的假期存钱

> 原文：<https://lifehacker.com/open-a-subaccount-to-save-for-that-epic-vacation-1818807153>

一些银行允许你在你的储蓄账户里开一个储蓄账户。你给这些账户贴上特定的标签，比如*季度税*或者 *2018 意大利之旅*或者*权力的游戏纹身*——无论你存钱是为了什么。把它从你的主账户中分离出来，这样在你存钱的时候更容易看到你的进度。



### 子帐户如何工作

子账户的设置非常简单。你在你的主储蓄账户的保护伞下开一个单独的账户，给它起一个昵称或者一个标签，然后你转账的时候可以选择在这个账户里存。

WellsFargo、CapitalOne360 和 SmartyPig 都有子账户的选项，但如果你的银行没有，你可以尝试使用类似于 [Qapital](https://www.qapital.com/) 或 [Tip Yourself](https://www.tipyourself.com/) 的工具来为特定目标存钱。使用 Qapital ，您只需链接您的银行账户，然后 [在 Qapital 应用](https://lifehacker.com/qapital-boosts-your-savings-goals-with-the-power-of-aut-1775769618) 中创建目标。你甚至可以为特定目标的储蓄创建自动规则。而 [Tip Yourself 就像你自己的虚拟储蓄罐](https://twocents.lifehacker.com/tip-yourself-is-like-a-virtual-savings-jar-1764147868) 。你建立一个账户，把它和你的银行联系起来，你就可以随时在那个账户里给自己“小费”。

### 子账户使你的目标成为现实。

给你的储蓄目标起个名字，让它变得具体，这很重要，而且符合美国储蓄互助协会的目标标准 ，它说你的目标应该是:

*   具体

*   可测量的

*   可实现的

*   相关

*   有时间限制的

如果你的目标仅仅是“存更多的钱”，你就在用一些非常模糊的术语。那个目标对你没有任何作用。然而，当你带着目标 前进时，更容易看到隧道尽头的亮光，实际上坚持目标——你确切地知道你会从中得到什么。

### 感觉被授权可以让你存更多的钱

不过，这也是关于 [感觉被授权](http://www.thewildwong.com/money-is-a-tool/) 。《消费者研究》杂志上发表的一篇 [2014 年论文](https://www.gsb.stanford.edu/faculty-research/publications/money-bank-feeling-powerful-increases-saving) 表明，感觉强大可以让你存更多钱。

研究人员将受试者置于各种情境中，让他们感觉自己有更多的控制权，然后问他们愿意存多少钱。科学日报解释道:

> 在一项研究中，一些参与者感觉自己很有力量，并被要求坐在一把高椅子上。其他参与者感到无力，并被要求坐在一个矮脚凳上。所有参与者都被要求回答一些问题，然后让他们选择以现金形式领取研究补偿，或者将补偿存入实验室储蓄账户。结果显示，坐在高椅子上的人比坐在矮脚凳上的人节省更多的钱。

简而言之，研究发现有权有势的人储蓄率明显更高。对他们的研究有一些警告，但总的来说，论文总结道:“在五项研究中，这项研究揭示了感觉强大会增加储蓄。”

您可能想知道:太好了，这与命名您的帐户有什么关系？

像“停止过度消费”或“多存钱”这样模糊的目标不仅会挫伤积极性，还会微妙地暗示你正在做错事，并抑制你的消费冲动。这可能有一些道理——好吧，可能有很多道理——但是将一个模糊的目标与一个具体的目标相比较，比如“为大学存钱”或“为更好的公寓存钱”，含义就会发生变化。具体的、有意义的目标会让你处于主导地位，因为*你会从整个省钱的过程中得到一些东西。*

这也是真的——你的金钱习惯为你服务 ，无论是支付账单还是存钱找乐子。然而，当你用具体的、可衡量的、有意义的术语来描述你的习惯时，你更有可能感觉在控制之中，这让你更有动力去追求你的目标，去度一个史诗般的假期。
