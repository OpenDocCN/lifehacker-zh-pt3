# 使用 Wine 3.0 在 Android 上运行这些 Windows 应用程序

> 原文：<https://lifehacker.com/you-can-run-these-windows-apps-on-android-with-wine-3-0-1822342695>

有数以百万计的 Android 应用程序可供选择，但如果你想超越 [Google Play](https://lifehacker.com/watch-out-for-this-fake-whatsapp-app-in-the-google-play-1820222637) 所能提供的，那么你很幸运。由于最近的一次更新，你可以使用一个名为 [Wine 3.0](https://www.winehq.org/announce/3.0) 的程序在 Android 上运行 [Windows 7](https://lifehacker.com/lifehackers-complete-guide-to-windows-7-5386953) 应用程序。



Wine 实际上从 1993 年就出现了，使得在苹果硬件和 Linux 驱动的电脑上运行 Windows 软件成为可能。随着它的最新更新，同样的功能正在扩展到 Android，尽管它的功能有一些限制。

[安卓警察](http://www.androidpolice.com/2018/01/22/wine-popular-windows-compatibility-layer-can-now-installed-android/) 注意葡萄酒 3.0 有点不稳定。它似乎在新设备上运行得最好，但可能会使旧手机或平板电脑崩溃。它还可以全屏运行所有程序，所以不要以为你会像在实际的电脑上一样同时运行两个程序。

此外，Wine 3.0 在 x86 Android 设备上运行得更好(这意味着处理器基于英特尔技术)。这意味着如果你有一个基于 ARM 的设备，你将被限制使用与 [Windows RT](https://lifehacker.com/run-full-windows-applications-on-your-windows-rt-tablet-5985193) 一起工作的程序。即使有这些限制，Wine 3.0 仍然开启了一些有趣的可能性。

如果你喜欢游戏，你可以使用 Wine 3.0 在任何 Android 设备上运行经典的第一人称射击游戏雷神之锤及其续集 [雷神之锤 II](http://forum.xda-developers.com/attachment.php?attachmentid=1640830&d=1358070370) 。如果你运行的是 x86 芯片，你也可以尝试运行完整版的 Steam(而不是受限的移动应用)来运行更多的视频游戏。

不是游戏玩家？试试 [Greenshot](http://forum.xda-developers.com/showthread.php?t=2114273) ，这是一个高级的 Windows 截屏程序，让你可以立即给你的图像添加注释、打印或者在电子邮件中分享。或者获得运行在 Android 上的流行(免费)数字音频编辑器 [Audacity](http://forum.xda-developers.com/showthread.php?t=2103779) 。你甚至可以用 [Min vMac 仿真器](http://forum.xda-developers.com/attachment.php?attachmentid=1662947&d=1358737954) 把你的手机变成早期的麦金塔电脑。

这只是可用资源的一个快速示例。你可以查看 XDA 开发者 的完整列表，以获得更多可以在搭载 Wine 3.0 的安卓系统上运行的 Windows 应用。
