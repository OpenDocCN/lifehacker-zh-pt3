# 你的登机牌上有大量的个人信息:粉碎它

> 原文：<https://lifehacker.com/your-boarding-pass-has-a-ton-of-personal-info-on-it-sh-1735143777>

当你坐飞机回家时，你可能会忍不住扔掉你的登机牌。然而，这份文件包含了大量关于你的个人信息，其中一些并不明显。把它撕碎。

Watch

正如信息安全博客 Krebs on Security 指出的那样，你的登机牌是一个信息宝库。在纯文本中，您通常会将您的姓名、航班号等信息打印在通票上。然而，即使你不担心这一点，这些条形码还包含更多的信息:

> “除了他的名字、常旅客号码和其他(个人身份信息)，我还能获得他的记录定位器(也就是他当天乘坐的汉莎航空航班的“记录钥匙”)，科里说。“然后我进入汉莎航空公司的网站，使用他的姓氏(编码在条形码中),记录定位器能够进入他的整个账户。我不仅可以看到这一次飞行，还可以看到从星空联盟预订到他的常旅客号码的任何未来航班。”

当然，每家航空公司都有不同的登机牌信息存储方法。即使没有编码信息，你的登机牌上清晰可见的数据也应该值得粉碎。如果你认为你不担心你能看到的信息，无论如何，对你看不到的信息采取预防措施是值得的。
