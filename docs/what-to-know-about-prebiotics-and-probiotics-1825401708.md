# 关于益生元和益生菌你需要知道什么

> 原文：<https://lifehacker.com/what-to-know-about-prebiotics-and-probiotics-1825401708>

即使在我们最干净的时候，我们也被细菌覆盖，我们充满了细菌。没事的！这完全正常。这些微生物对我们的健康至关重要。你可能不需要弄乱它们，但如果你想尝试，你需要知道的术语是*益生元*和*益生菌*。



### 益生菌是“好”细菌

我们身体中最大的有益菌群生活在大肠中。食物在胃和小肠中被消化和吸收后，我们剩下的食物浆被喷入大肠，也称为结肠。当我们的身体重新吸收水分时，它可以在那里呆上几个小时。在那段时间里，我们的食物变成了生活在那里的生物的食物。

我们的结肠里有数万亿个微小的朋友。当它们消化我们的废物时，它们为我们做了大量重要的事情。例如，它们制造了我们大部分的维生素 K。它们保护我们免受有害微生物的感染，并有助于保持我们的免疫系统正常运行。我们的肠道微生物有许多种类，健康的肠道是一个多样化的地方，就像雨林一样。如果你的消化系统不太健康，你可以考虑添加一些有益的细菌。这些是益生菌。

但是，尽管我们的肠道中自然生活着成吨的物种，但作为益生菌出售的东西只包括几种类型的细菌。其中一些很容易在乳制品中生长，它们是将牛奶转化为酸奶的生物。所以获取益生菌的一个简单方法就是吃含有“活性细菌”的酸奶(标签上的这些文字只是意味着酸奶没有经过杀菌处理，所以它们还活着。)也可以买益生菌药丸或者粉剂。

一个重要的提示:益生菌不会在你的肠道中占据一席之地。看起来他们应该去，但是研究表明他们是游客。他们来了，他们可能会改变一些事情，然后他们继续前进。与此同时，它们仍然会对你的健康产生影响，但它们很少会导致任何永久性的改变。

一些研究发现，益生菌可以帮助腹泻或其他健康问题，但科学是不确定的。益生菌可能有一些好的效果，但它们对我们的健康不是必不可少的(大多数人没有它们也很好)，除了少数特定情况下的，它们可能根本没什么作用。

### 益生元是有益细菌的食物

我们的肠道细菌吃我们吃的东西，但它们只吃剩菜。在我们的食物到达它们之前，我们已经吸收了我们可以消化和利用的糖、蛋白质和脂肪。这意味着他们没有太多其他东西:只有纤维和其他我们身体认为不可消化的东西。

其中一些难以消化的东西是我们微生物的绝佳食物，食用它们可以促进某些类型的微生物生长和繁殖。例如，婴儿的肠道微生物群在生命早期由母乳中称为 [人乳低聚糖](https://academic.oup.com/glycob/article/22/9/1147/1988076) (HMOs)的益生元形成。这些 HMO 不仅喂养某些有益微生物，还防止病原体引起感染。

*益生元*我们吃的东西可能会通过影响肠道微生物来影响我们的健康。大多数已知的益生元是糖样分子的短链:低聚果糖(FOS)、低聚半乳糖(GOS)和菊粉，仅举几例。一些类型的膳食纤维也被认为是益生元，这取决于你问谁。如果你将益生元和益生菌包装在一起，你可以将这种药丸称为*合生素*。

我们成年后也吃益生元，不管我们是否意识到。例如，小麦和洋葱 [提供了我们大部分的菊粉和低聚果糖](https://academic.oup.com/jn/article/129/7/1407S/4722578) 。你也可以购买含有更高浓度益生元的益生元补充剂，但这样做并没有被证明对健康有益。

### 可以尝试，但不要期待奇迹

没有研究证明益生元或益生菌补充剂对健康是必要的。所以，如果你有消化问题，想尝试一下——嘿，请随意尝试。(注意，随着你的身体习惯新的饮食，扰乱你的肠道细菌会导致轻微的消化问题，比如胀气。)

但是我还没有遇到一个对补充剂深信不疑的微生物组研究人员。更多的时候，我听到他们说他们在饮食中加入了健康的食物:酸奶和发酵食品用于活性培养；各种植物性食物，如蔬菜和全谷物，可以提供益生元。
