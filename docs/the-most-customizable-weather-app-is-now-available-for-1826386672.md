# 最具可定制性的天气应用程序现在可以在 Android 上使用了

> 原文：<https://lifehacker.com/the-most-customizable-weather-app-is-now-available-for-1826386672>

在 iOS 和 Mac 上以多嘴著称的天气应用程序胡萝卜天气(Carrot Weather)现在在安卓(Android)上处于测试阶段。该应用以讽刺的方式报告天气，对来自*门户*的滥用人工智能表现平平。谢天谢地，你可以切换到“专业”模式，不受干扰地享受一个复杂的，可定制的天气应用程序。

Watch

这是一款天气应用，适合那些对天气应用从不满意的人，他们清楚地知道我们想要什么，不想要什么。例如，我想知道下雨的几率和“感觉”温度，我一点也不在乎气压。在夏天，我想知道紫外线指数，这样我就知道是否要戴上我那顶令人尴尬但有保护作用的宽边帽。胡萝卜天气让我得到这个信息。

你可以定制胡萝卜天气的“今天”总结，每小时的时间线，和十天的预报的每个数据点。每个槽可以显示几个数据点之一:实际温度，“感觉”温度，降水机会和数量，风速，气压，湿度，紫外线指数，云量，能见度，露点，日出/日落，月相和最近的气象站。你可以同时显示摄氏和华氏温度，就像你的手机是一个汇丰银行的路标。你甚至可以选择两个数据源，地下天气和黑暗天空。

要移除广告，并启用可定制的主屏幕小工具(尚未发布)，你需要每年支付 4 美元。收取天气应用程序的订阅费有点厚脸皮，所以肮脏的临时工可以坚持使用像[1 天气](https://lifehacker.com/the-best-weather-app-for-android-5982791#_ga=2.231589279.1884048105.1527600543-1297080755.1497980211) 这样的一次性付费应用程序。但是如果你想要一个超具体的预报，并且可以自由地按季节改变它，那就付钱吧。然后选择你想要的讽刺程度(有几个*)并进行定制。*
