大家好，欢迎回到另一个为 [*挑选话题的环节。*](http://lifehacker.com/tag/will-it-sous-vide) ，列在那里我用我的沉浸循环器做任何你想让我做的事。

虽然我喜欢我们的肉类，火鸡，但我完全厌倦了感恩节食物，想换点完全不同的东西，不管是甜的，美味的，还是其他的。一些想法:

*   某种注入或调味的奶油布丁:我知道奶油蛋卷这样做很好吃，但让我们看看我们能给面糊带来多少味道。
*   **蟹蘸:** [你们其中一个人建议这个](https://skillet.lifehacker.com/1819622780) ，我用它来做小丑。
*   剥了吃的虾:贝类腌制得很好，我打赌我们会从那些贝壳中提取各种味道。
*   鸡肝慕斯:用混合物或芳香剂轻轻烹制它们，可以做出非常美味的酱料。
*   密西西比泥饼:我很想吃巧克力。
*   [](https://skillet.lifehacker.com/1819627777)**可以说是“半熟”的花生。**
