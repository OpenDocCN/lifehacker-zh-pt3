# 三种成分的欢乐时光:强化颜培珊

> 原文:[https://life hacker . com/3-ingredient-happy-hour-a-defensed-shandy-1796170353](https://lifehacker.com/3-ingredient-happy-hour-a-fortified-shandy-1796170353)

周末快乐，欢迎回到 [*三料欢乐时光*](http://lifehacker.com/tag/3-ingredient-happy-hour) ，每周饮品专栏，提供超级简单又美味的酒水。本周我们正在制作一种饮料，非常适合在水体中懒洋洋地漂浮时啜饮:香迪。

Watch

事实上，这不仅仅是一个 shandy，因为它有一个额外的成分来帮助它。虽然我不反对混合等量啤酒和柑橘汽水(或柠檬水)的“传统”配方，但我发现它们在 ABV 部门有点甜，有点淡。为了解决这个非常严重的问题，我加了一点伏特加。我喜欢原味的 ol '，但你也可以用任何一种 [深涡](https://www.deepeddyvodka.com/) 柑橘(或蔓越莓)味的伏特加，味道出奇的好。

因为这种饮料没有冰，所以重要的是你所有的原料都要预先冷藏，所以如果你要在户外混合它们，把所有的东西都放在冰箱里。要制作强化夏吸管，你需要:

*   1 盎司伏特加
*   3 盎司廉价啤酒(你要加苏打，所以买些相当无味的。)
*   3 盎司柑橘苏打，任何你想要的口味

将所有东西倒入一品脱玻璃杯(或红色 Solo 杯，或隔热平底玻璃杯)中，轻轻搅动。在顺流而下的时候啜饮，或者，如果不行，至少把你的脚悬在水池里。
