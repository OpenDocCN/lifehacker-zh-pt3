# SNES 经典已经被黑客攻击增加了更多的游戏

> 原文：<https://lifehacker.com/the-snes-classic-has-already-been-hacked-to-add-more-ga-1819297251>

任天堂怀旧风格的《超级 NES 经典》发布还不到两周，但已经被黑客攻击增加了更多游戏、自定义背景等等。



通常情况下，SNES 经典赛有 21 场比赛的默认设置，包括之前未发布的 *Star Fox 2* ，有固定的菜单和背景。该黑客使用了一个名为[hacki 2](https://github.com/ClusterM/hakchi2/releases)的 Windows 漏洞程序，允许你向系统添加更多游戏(或 rom)，应用自定义背景图像来填充整个 16:9 的配给屏幕，并通过一些调整启用隐藏的“扫描线”过滤器。

该黑客是由一位名叫“ClusterM”的俄罗斯黑客设计的，他也知道如何利用之前发布的 NES 经典。现在 hakchi2 可以在两个系统中工作。要使用它，你只需要使用微型 USB 电缆将你的系统插入 Windows 计算机，然后运行 hakchi2 并按照屏幕上的说明操作。但是，请记住，这样做可能会使您的 SNES 经典车的保修失效。并不是所有的 SNES 光盘都能在这款设备上很好地运行。
