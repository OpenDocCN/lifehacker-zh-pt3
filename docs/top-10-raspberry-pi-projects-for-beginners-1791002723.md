### 10.气象站

当然，当然，你总是可以看看窗外的天气，但是当你可以得到精确的温度、气压、风速和天气预报时，这有什么意思呢？ [看不到比你的树莓 Pi](https://lifehacker.com/collect-and-display-all-kinds-of-weather-data-with-this-1788867279) ，还有 [几个多余的组件](http://lifehacker.com/build-your-own-raspberry-pi-powered-weather-station-1697061082) 。

当然，如果你想要更灵活一点的东西，或者更有趣一点的东西，可以考虑建造 [这种变色灯，它会根据你的 Pi 的颜色和光图案来显示天气预报](http://lifehacker.com/build-a-lamp-that-changes-color-based-on-the-weather-fo-1787933143) ，或者 [这种稍微简单一点的版本](https://lifehacker.com/build-a-raspberry-pi-powered-weather-forecast-display-1783350558) ，如果我自己说的话，在书架上看起来相当不错。

### 9.照相亭

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-kMGjroIRzj4&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-kMGjroIRzj4&start=0) 

当你们一起出去玩得很开心的时候，还有什么比和朋友一起拍很多很棒的照片更有趣的呢？把你坐了很久的 Pi[变成一个照相亭](https://lifehacker.com/make-your-own-easy-to-use-touchscreen-photo-booth-with-1782907957) ，专门用来捕捉那些精彩的瞬间，然后与你的朋友分享。

你也有选择——上面的那个有一个触摸屏和一个小的倒计时定时器，但如果你不想花那么多精力， [试试这个更简单的版本](https://lifehacker.com/make-your-own-photo-booth-with-a-raspberry-pi-1594011522) (显然你不需要定制的盒子)，它会自动将 GIF 动画形式的照片上传到你为活动或你的项目创建的 Tumblr 帐户。

### 8.家庭监控系统

树莓派的美妙之处在于它是一台袖珍电脑。这意味着它几乎可以去任何地方，比如车库，前门附近，或者任何你需要一双眼睛和一个电源插座的地方。这个初级项目将立刻把你的 Pi 变成一个家庭安全系统 。

当你准备好拍摄月球时，如果你想让整个房子或院子都有眼睛，你可以用覆盆子 建立一个多相机系统。 [这个项目使用 Windows IoT 对你的 Pi 做同样的事情](http://lifehacker.com/build-a-motion-sensing-security-camera-with-a-raspberry-1768562499) 如果你正在寻找一个可能看起来更熟悉的替代方案。

### 7.DIY 街机棒

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-bYHXUECdQAg&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-bYHXUECdQAg&start=0) 

到目前为止，最受欢迎的 Pi 项目之一是街机，我们将谈到这一点。我们有点本末倒置，建议你把你的复古游戏控制台*放在街机游戏杆*里，这样你就可以在完美的控制器 上用 [同时玩你最喜欢的老式学校游戏。](https://lifehacker.com/this-raspberry-pi-powered-arcade-stick-is-pre-loaded-wi-1481961115)

上面的一个使用了经典的街机设计，用美丽的丙烯酸完成，但如果你想要一个对初学者来说稍微容易一点的东西——完成一步一步的指导视频，告诉你如何制作外壳和控制台本身， [这个对你来说是完美的](https://lifehacker.com/build-your-own-portable-arcade-stick-with-a-raspberry-p-1785215175) 。

### 6.一个完全便携的数字电子书图书馆

如果你喜欢电子书，并且随身携带大量的电子书，甚至与他人分享，这是你的难题， [这个移动数字图书馆](https://lifehacker.com/build-your-own-mobile-digital-library-to-share-books-fr-1790196470) 非常适合保存、分享和与朋友交换书籍。最棒的是，它完全由树莓派提供动力。它是便携式的，把你的 Pi 变成一个其他人可以连接的 Wi-Fi 热点，找到一些东西来阅读，并抓住它带走。分享不是很美好吗？

### 5.一台随处可佩戴的相机

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-KXVkN8eS9z4&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-KXVkN8eS9z4&start=0) 

也许你想记录你的生活，或者做一个“生活中的一天”的实验视频，让某人可以穿着你的鞋走一英里。也许你只是计划一次很棒的徒步旅行或骑自行车旅行，并想拍下整个旅程。好吧，如果你有一个 Pi，你就有一个完美的项目来捕捉所有的乐趣。 [这款可穿戴相机由一个 Pi 和一个电池](https://lifehacker.com/build-a-wearable-camera-with-a-raspberry-pi-1782899159)供电，小到可以塞在衬衫上或挂在脖子上的挂绳上。

如果你想更进一步，你可以建立一个 [互联网连接模型，实时向 YouTube](https://lifehacker.com/set-up-a-raspberry-pi-as-a-live-streaming-camera-that-b-1790317509) 传输视频，但是你可能想把它放在你的 Pi 有信号或 Wi-Fi 的地方。

### 4.全网广告拦截器或呼叫总部 VPN

如果你更倾向于技术，考虑一下 [这款全网广告拦截器](https://lifehacker.com/create-a-network-wide-ad-blocker-with-a-raspberry-pi-1727295925) ，它将保护你网络上的所有设备和系统免受广告、恶意广告和其他烦人的悬停和弹出广告的影响，即使是在你的移动设备上。这是一个小工作，但它是值得的。

为了更加安全和隐私，考虑把你的 Pi 变成一个 VPN 和代理 T3，当你不在家的时候也可以使用。毕竟， [当你在旅途中时，VPN 是保护你的数据免受窥探的理想选择](http://lifehacker.com/why-you-should-be-using-a-vpn-and-how-to-choose-one-5940565) ，连接回你的家庭网络可以让你无论走到哪里都保持流量和数据的安全。另外，这不是一个超级难完成的项目。想要更多的隐私吗？ [添加 Tor 到 mix](http://lifehacker.com/use-a-raspberry-pi-as-a-tor-vpn-router-for-anonymous-br-1682296948) 。

### 3.流媒体网络电台

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-2aca18JURvg&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-2aca18JURvg&start=0) 

使用 Raspberry Pi 来流式传输 Spotify、Pandora、Google Music 和其他流式网络广播和播客是另一个非常受欢迎的项目。为什么不呢？这很容易，取决于你想在这个项目上投入多少精力，你可以有 [一个触摸屏功能的点唱机，任何想控制播放列表的人都可以使用](https://lifehacker.com/how-to-build-a-raspberry-pi-jukebox-any-non-geek-can-us-1712107103) ，或者你可以有 [一些漂亮的东西，放在架子上](http://lifehacker.com/transform-a-vintage-radio-into-a-raspberry-pi-powered-w-474682370)[看起来像一个老式的高保真音响](http://lifehacker.com/this-diy-raspberry-pi-powered-vintage-radio-pumps-out-g-511571375) 。

此外，这个项目还有大量的变化，像这个 [看起来现代而锐利](http://lifehacker.com/build-a-touchscreen-music-player-with-a-raspberry-pi-1700721764) ，这个 [像一个多房间的 Sonos 克隆](http://lifehacker.com/turn-a-raspberry-pi-into-a-multi-room-wireless-stereo-1480556758) ，但是 [它们都从这个基础教程](https://lifehacker.com/how-to-turn-a-raspberry-pi-into-a-private-streaming-mus-1583221462) 开始，它会启动你选择的任何项目。

### 2.一个自己动手做的亚马逊回声

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-4yfK90OJrek&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-4yfK90OJrek&start=0) 

亚马逊实际上想让你把你的树莓尿变成回声点。说真的，他们发布了官方指令(当然是在人们开始 [自己做](http://lifehacker.com/build-a-raspberry-pi-powered-diy-amazon-echo-1762678229) 之后)，然后不久前他们解锁了他们最初保留的唯一东西——[用唤醒词](http://lifehacker.com/you-can-now-trigger-your-raspberry-pi-powered-diy-alexa-1787489420) 触发它。

当然，你不需要为了自己做一个而经历所有这些。我们在 [中为您介绍了这份完整的、从头到尾的指南，在这里](https://lifehacker.com/how-to-build-your-own-amazon-echo-with-a-raspberry-pi-1787726931)——一旦您建立了一个，您就可以使用它来做任何您通常使用 Echo 的事情，从琐事到网络搜索等等。

我们曾经取笑过它，并且在过去讨论过，但是到目前为止，我们对初学者的首选建议是 [构建你自己的树莓 Pi 驱动的](https://lifehacker.com/how-to-turn-your-raspberry-pi-into-a-retro-game-console-498561192) 复古游戏控制台，玩你最喜欢的旧游戏。
