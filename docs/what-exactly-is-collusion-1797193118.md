# 到底什么是串通？

> 原文:[https://lifehacker.com/what-exactly-is-collusion-1797193118](https://lifehacker.com/what-exactly-is-collusion-1797193118)

如果你最近浏览过新闻，你可能会看到或听到“勾结”一词，指的是特朗普总统的高级职员被指控与俄罗斯有见不得人的交易。但是什么是串通呢？这真的是犯罪吗？

今天，特朗普总统的高级顾问兼女婿贾里德·库什纳 [站在参议院调查人员面前，否认在 2016 年总统竞选前后与外国特工](https://www.washingtonpost.com/world/national-security/kushner-arrives-at-senate-for-closed-door-questioning-on-russia/2017/07/24/f5be2b26-7073-11e7-8f39-eeb7d3a2d304_story.html?hpid=hp_hp-top-table-main_kushner620am%3Ahomepage%2Fstory&utm_term=.49db3b8d5051) 有任何勾结。当然，他的声明指的是一则新闻，即一名声称拥有希拉里·克林顿(Hillary Clinton)和特朗普核心圈子的破坏性材料的俄罗斯国民实际上在 2016 年 6 月举行了一次会议。从那以后,“共谋”一词就成了一个政治术语，但它在很大程度上被用作一个笼统的说法，在美国法律中并不像你想象的那么重要。

这个词的 [字面定义](https://www.merriam-webster.com/dictionary/collusion) 是“特别是为了非法或欺骗目的的秘密协议或合作”谈到竞争和反垄断法，在美国这里*是*非法的。然而，除此之外，串通本身是*而不是*一种特定的联邦罪行。从技术上讲，你可以在任何时候与外国政府“勾结”，因为没有这样的法规这样说。该术语含义模糊，被不恰当地用作各种可能犯罪的简称。

尽管如此，仅仅因为联邦法律没有明确地将共谋定为犯罪，并不意味着其他犯罪没有发生。正如国土安全部负责政策的前副助理部长保罗·罗森茨韦格在政治 所解释的那样，川普的工作人员可能会被指控阴谋 [(18 U.S. Code 371)](https://www.law.cornell.edu/uscode/text/18/371) 违反美国选举法。在这种情况下，美国法律禁止外国公民向竞选活动提供任何“有价值的东西”——希拉里·克林顿的丑闻可能属于这种宽松的术语。 [根据理查德·尼克松的前白宫法律顾问约翰·w·迪恩](http://www.politico.com/magazine/story/2017/07/12/what-is-collusion-215366) 的说法，其他可能的违法行为包括“”(《美国法典》第 18 编第 2 条)、“与计算机有关的欺诈和相关活动”(《美国法典》第 18 编第 1030 )、“电信欺诈”(《美国法典》第 18 编第 1343 条 )

谈到“勾结”， [凯莉·科德罗](https://www.law.georgetown.edu/faculty/cordero-carrie-f.cfm) ，律师，乔治敦大学法学院兼职教授，国家安全助理司法部长的前法律顾问， [指出](http://www.politico.com/magazine/story/2017/07/12/what-is-collusion-215366) “区分政治和法律讨论”很重要这个术语本身将继续被使用，但是知道它没有任何特定的法律含义是件好事。没有法律说特朗普的高级职员不能和俄罗斯人共事。要产生影响，必须有证据表明他们一起犯下了特定的罪行，互相帮助犯下了特定的罪行，或互相帮助掩盖特定的罪行。现在，我们必须让调查顺其自然，看看会有什么结果。
