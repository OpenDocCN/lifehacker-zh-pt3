# 什么是最好的剩菜？

> 原文：<https://lifehacker.com/what-makes-the-best-leftovers-1828463317>

在学校的时候，我并没有从单餐的角度去考虑餐厅的食物。我总是试着点一些我能在以后的饭菜中吃的菜，因为像大多数大学生一样，我有点穷。任何一道菜成为未来美食的能力都有很多因素。

Watch

当然，有份量的问题，但是你也必须考虑上菜的温度。有些食物加热后味道很好，有些冷吃时味道很好，而有些在第一次食用后味道就不太好了(寿司)。我(显然)对此想了很多；我最喜欢的剩菜包括:

*   **披萨:**披萨是剩菜之王。重新加热很容易——只要在煎锅里放一点橄榄油[——但是你不需要这样做。冷披萨是完美的早餐，我不会接受任何批评。](https://lifehacker.com/reheat-your-pizza-on-the-stove-top-for-fast-pizza-with-1792124374)
*   **意大利面:**把你的宽面条或意大利面放入微波炉中，加入一点水 就可以让剩下的意大利面重新焕发活力，但是，再说一次，我不讨厌冷的意大利面。
*   牛排:我几乎从不加热牛排。相反，我把它切成薄片，在上面放一些红洋葱和一种古怪的奶酪，再淋上一点超级酸性的调味汁，就成了美味的凉拌沙拉。

*   **炒饭:**这是大学里的一道菜。你不仅可以得到很多食物，而且你可以用冰箱里的任何东西来填充剩菜，或者添加一个煎蛋来获得额外的蛋白质。
*   炸鸡:没有什么能让炸鸡——或者真正油炸过的任何东西——恢复到它以前那种松脆的荣耀，但是如果它不能做成最好的鸡肉三明治，我会觉得很危险。凌晨时分也很棒，被冰箱的灯光吞噬。

显然，炸薯条不应该出现在这个名单上。虽然我已经看到了一些关于将它们切碎并做成家庭炸薯条的争论，但这需要大量的油，而且它们仍然有独特的剩余炸薯条味道。三明治，除非被压，第二天也不会做得这么好。

在你看来，什么是最好的剩菜？最糟糕的是什么？你喜欢重新加热什么，在冰箱里放了一夜后什么味道最好？还有，你发现吃剩的薯条的用途了吗，吃剩的薯条甚至是会发生的事情吗？
