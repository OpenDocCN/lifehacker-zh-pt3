# 七个语言学家支持的制作强力抗议标语的技巧

> 原文：<https://lifehacker.com/seven-linguist-backed-tips-for-making-powerful-protest-1794551494>

今天是科学 的 [游行，全国人民走上街头抗议一切反科学的议程和政策。如果你打算表达你的支持，还有时间用这些简单的文字游戏提示做一些令人难忘的标志。](https://www.marchforscience.com/)

正如语言学家兼播客主持人 [丹尼尔·米奇利](http://danielmidgley.com/) [在 Quartz](https://qz.com/888554/a-linguist-explains-how-to-write-protest-signs-that-everyone-will-remember/) 上解释的那样，制作巧妙的抗议标语是很困难的，因为你的空间有限，而且你的信息必须被迅速吸收(因为你会四处走动)。无论你要抗议什么，米奇利提供了七种方法，可以帮助你的标志真正被看到:

1.  **排比**:使用结构相似性作为工具，让你的信息更容易记住，比如“我的身体/我的选择”

2.  押韵:押韵让任何抗议口号都更加吸引人，比如“我们的土地上没有石油”，或者“不要仇恨，教育。”

3.  个人属性:如果你在抗议某个人，就像许多人最近抗议唐纳德·特朗普总统一样，你可以说“我们会战胜一切的”

4.  怀疑:幽默的对话语气可以给一种情况带来轻浮和注意力，比如“我和他们在一起”和指向你周围人的箭头，或者“[需要引用]。”只是保持幽默有所指。一个写着“我的胳膊累了”的标志很有趣，但不完全是支持。

5.  模仿别人说过的一句口号或话，并根据你的意愿来改变它，比如“美国依靠科学”，而不是 Dunkin' Donuts 最初的口号。

6.  积极:保持乐观和直截了当，比如“我们支持科学！”或者“科学拯救生命”

7.  重复:做一个标志来表达你已经听过或看过的东西没有错，比如“科学没有议程”抗议不是创造力竞赛。

最后，不要太担心你的招牌有多好看。一张空白的海报板，上面写着一些有力的文字就足够了。现在出去吧，注意安全，支持你认为正确的事情。
