# 一个月来每天做俯卧撑我学到了什么

> 原文：<https://lifehacker.com/what-i-learned-doing-push-ups-every-day-for-a-month-1791921298>

一月初，我偶然看到一个你可能看过的视频:一个年轻女子 [做俯卧撑 100 天](https://www.youtube.com/watch?v=gu_1L6XJM-c) 。她开始“骨瘦如柴”，但变得更强。我想这么做。我做到了，你也可以。

Watch

就在这之前，我只是偶尔锻炼一下——生活因为三个孩子而变得忙碌——所以一开始我觉得自己很瘦。 [俯卧撑一开始很难](https://lifehacker.com/how-to-do-the-perfect-push-up-5992941) ，但是后来变得容易了。我确实变得更强了。以下是我一路走来学到的东西。

### 疼痛不一定会阻止你

一开始我 [可以做全俯卧撑](https://lifehacker.com/how-to-work-up-to-full-push-ups-without-starting-from-y-1718034181) ，但是他们很难。我倾向于在日常锻炼中避免它们，因为如果我做的超过一组或两组，第二天我会非常酸痛。所以如果我打算每天做俯卧撑，我必须小心翼翼。

幸运的是，我知道如何克服疼痛。你不会跳过锻炼；你只需要尽可能的简单和温和。第二天我一次只做了几组，第三天也一样。花了一个多星期才完全摆脱疼痛，但我保持了较轻的工作量，只专注于坚持锻炼*。我知道如果我提前跳过一天，我将很难说服自己继续下去。* 

### *一致性是有回报的*

*虽然很诱人，但我从未错过一天。我还把我的目标缩减为最重要的一件事:每天至少做一个俯卧撑(并拍下视频)。*

*由于我已经很难找到时间去健身房 ，我知道我没有太大的希望每天进行长时间的锻炼。我犹豫着是否要签署任何具体的计划，甚至是[100 个俯卧撑的套路](http://lifehacker.com/one-hundred-push-ups-takes-you-from-zero-to-a-hundred-i-398581) ，因为无论如何我都必须在开始时修改它，因为疼痛。*

*有些日子我做俯卧撑 [作为体重锻炼](http://vitals.lifehacker.com/everything-you-need-to-build-your-own-bodyweight-workou-1770226539) 的一部分，但大多数时候我只是在工作日做，只要我觉得我可以休息一下。我 [设置了 Nagbot](http://lifehacker.com/nagbot-sends-mean-texts-to-help-you-stick-to-your-resol-1791188092) 每天给我发短信，这样我就不会忘记。*

### *你进步得比你想象的要快*

*我想我的第一个里程碑将是一次做更多的练习。然而，事实并非如此。我看到的第一个成就:我的状态改善了。*

*一开始，我给《生命体征》的作者斯蒂芬妮·李展示了一些我的俯卧撑视频，并征求了他的形式评论。她指出我的肘部向外张开，手的位置也很宽，并预言随着我变得更强壮，我可能会自动修复这个问题。*

*事情就是这样。甚至一个星期后，当我还在忍受疼痛的时候，我注意到我可以把我的手直接放在我的肩膀下面。我也可以一直走到地面，而不是像以前那样中途作弊。*

*我没有做更多的 T1 代表，我做了更好的 T3 代表 T2。所以为了保持这种势头，我开始尝试新事物。例如，我把脚放在椅子上做俯卧撑，最近又把脚放在桌子的高度。我还让我的孩子坐在我的背上，如果我用尽全力，我可以把他举起来。*

*当我开始的时候，我不知道我会提高这么多。我很高兴能继续保持这种势头，看看下个月底，甚至整整 100 天后我会在哪里。*
