# 有哪些怪异的厨房电器是你离不开的？

> 原文：<https://lifehacker.com/what-weird-kitchen-appliance-can-you-not-live-without-1797434642>

大家好，欢迎回到 [*有什么好吃的？*](http://lifehacker.com/tag/whats-cooking) ，这是一个开放的话题，你可以在这里分享你所有关于食物的想法、建议、食谱和观点。今天我想了解一下你最喜欢的古怪却又必不可少的厨房工具，即使它们是——gasp—[一次性](http://lifehacker.com/watch-alton-brown-demonstrate-why-unitaskers-have-no-1749470145) 。

Watch

在厨房电器的世界里，有一些显而易见的“苦力”——那些我不能(也不应该被期望)没有的东西。其中包括我的 [棒搅拌器](http://skillet.lifehacker.com/kitchen-tool-school-the-portable-powerful-immersion-b-1782522838) ，我的巨型木勺，我的铸铁锅，还有(令人震惊的)我的 [浸没循环器](http://lifehacker.com/tag/will-it-sous-vide) (全部三个)。但也有令人惊讶的必备品，我从未见过的真正的英雄。然后就是我的 [Tovolo 铲子&地摊多功能地摊](https://www.amazon.com/Tovolo-Multi-purpose-Spreader-spreader-included/dp/B007I82PEQ/ref=sr_1_10?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/what-weird-kitchen-appliance-can-you-not-live-without-1797434642&asc_source=&ie=UTF8&keywords=tovolo&qid=1501603753&s=home-garden&sr=1-10&tag=kinjalifehackerlink-20) 。

虽然它看起来像一个不起眼的小铲，这东西是如此之多。在一端，你有一个坚硬的尼龙刮平器，你猜对了，它非常擅长刮平和展开。但它的另一端，硅胶勺端，是绝对必要的。它不仅非常适合煮鸡蛋，还能从任何罐子、瓶子、平底锅或碗中取出最后一滴。这也是一个非常有用的电镀工具，因为没有东西会粘在上面。我说的是，这是我最喜欢的事情之一。

所以现在我想听听你的意见！哪个怪异的厨房工具出其不意地偷走了你的心？你有拒绝放弃的一次性电器吗？*有没有比大木勺更有用的工具？外面还有人收集木勺吗？我们可以成立一个木勺俱乐部吗？*
