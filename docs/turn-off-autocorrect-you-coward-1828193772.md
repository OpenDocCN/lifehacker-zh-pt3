# 关掉自动更正，你这个胆小鬼

> 原文:[https://life hacker . com/turn-off-auto correct-you-coward-1828193772](https://lifehacker.com/turn-off-autocorrect-you-coward-1828193772)

“自动更正”上次让您的生活变得更轻松是什么时候？回想一下，试着想象一下“自动更正”确实找到它的时候。我知道我们只在技术不起作用的时候才注意到它，但如果你想不出一个，那就该接受这个事实了:[就是](https://lifehacker.com/avoid-autocorrect-mistakes-by-typing-an-extra-letter-on-30782448) [坏](https://lifehacker.com/create-fake-contacts-to-fix-ducking-autocorrect-1787538703) 。

需要澄清的是:我所说的“自动更正”是指软件自动用它认为你一定要键入的内容替换你已经键入的内容。我对预测文本和拼写检查没有意见，这对于有阅读障碍和书写困难的人，或者任何用手机发送重要邮件的人来说都非常有帮助。如果没有预测文本算法，许多辅助功能，如滑动输入和自定义键盘快捷键，就不会存在。自动更正背后的技术不是问题，但执行起来有点糟糕。

我禁用自动更正功能已经很多年了，有两个主要原因让我再也不回去了。首先，没有它，我的观点更容易被理解。几个换位的字母会让一条信息更难解析，但是一个自动替换的单词可以改变它的整个意思——尤其是当你的手机一直把这个单词塞进后续的信息中的时候。第二，我喜欢我写的东西听起来像是我真的写的。我们都有独特的用语言表达自己的方式；“自动更正”如此执着于让*更正*我们，以至于我们所有的信息读起来都一样。我在职业电子邮件中的声音与我在足球拉屎群聊中的声音非常不同，我的无自动更正功能手机一直保持着这种状态。

如果您开始接受，禁用自动更正非常容易。iOs 用户可以从键盘本身 [访问键盘设置](https://support.apple.com/en-us/HT202178)；Android 键盘的右上角有一个齿轮图标，可以弹出一个菜单，上面有“智能打字”和“恢复默认”选项。前者包含预测文本、拼写检查、自动替换和自定义键盘快捷键的菜单，后者允许您清除缓存的词典数据。(苹果，出于我确信完全合法的原因， [在“通用”设置菜单](https://www.iphonelife.com/blog/5/tip-day-how-fix-unwanted-quicktype-suggestions) 中隐藏了字典重置功能，而不是其他键盘设置。)在设置里瞎折腾，看看有什么吸引你的。

当然，这个建议并不适用于所有人。也许您和许多人一样，认为自动更正功能完美地预测了他们的每一次击键。然而，对我们其余的人来说，禁用自动更正就像把自行车的辅助轮拆下来一样——如果这个想法让你不舒服，那就更有理由把它做完。
