# 做这个黄瓜杜松子酒

> 原文：<https://lifehacker.com/make-this-smashed-cucumber-gin-tonic-1826148638>

周末快乐，欢迎回到 [*三料欢乐时光*](http://lifehacker.com/tag/3-ingredient-happy-hour) ，每周饮品专栏，提供超级简单又美味的酒水。本周，我们将带来已经令人耳目一新的杜松子酒&补品，让它变得更加清新，如果你能想象这样的事情的话。



本周早些时候，我去了一个新书发布会。我通常不参加图书活动，但这本书是关于杜松子酒 (和滋补品)的，我被告知会有免费的杜松子酒(和滋补品)。这本书什么都有——疟疾、羊驼推销员、从飞机上扔出的猫——我学到了很多，但主要是它让我想喝很多杜松子酒(和滋补品)。

很多年来，亨德里克的杜松子酒在我心中一直占据着特殊的位置，因为我喜欢那种新鲜的黄瓜味道，但在我所在的地区价格昂贵，而且奎宁水往往会掩盖那些更微妙的花香。所以，对于 G&T 的，我开始搅拌真正的黄瓜——很便宜！—加入不太贵的杜松子酒(Boodle 的伦敦干杜松子酒是冰镇的)，在第*杯上加奎宁水。如果你是奎宁苏打的忠实粉丝，请随意在下面的食谱中添加额外的一盎司奎宁，但我觉得这有点甜。要制作这款好喝的饮料，你需要:* 

*   3 片黄瓜
*   2 盎司不太花哨的杜松子酒
*   2 盎司的奎宁水(你可以在这里得到真正的幻想，但我不会超过 Shweppe 的)

将黄瓜片放入低碗中，捣碎。加入杜松子酒，再给黄瓜一次重击。在杯子里装满冰块，并加入奎宁水。呷一口，品味微风习习、阳光普照的日子的味道。
