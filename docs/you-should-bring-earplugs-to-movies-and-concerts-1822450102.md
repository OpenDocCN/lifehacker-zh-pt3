# 你应该带耳塞去看电影和音乐会

> 原文：<https://lifehacker.com/you-should-bring-earplugs-to-movies-and-concerts-1822450102>

不仅仅是你——表演和电影可能真的很吵。在你的钱包或口袋里带着耳塞听起来像是一个脾气暴躁的老人可能会做的事情，但如果你想在年老时能够抱怨像剧院噪音这样的事情，最好现在就保护你的听力。



声音越大，你越应该限制你的暴露。70 dB 以下的声音想听多久就听多久， [根据美国演讲-语言-听力协会](http://www.asha.org/public/hearing/Loud-Noise-Dangers/)；70 dB 大致是一个大声的群体对话的水平。但是一天八小时工作的安全极限是 85 分贝，而一部两小时的电影的安全极限是 91 分贝。

电影，尤其是动作片， [可以达到那个范围](https://www.today.com/health/loud-movies-some-theaters-could-damage-your-hearing-experts-say-t104666) ，虽然如果电影的其余部分更安静，偶尔 100 dB 的爆炸不会损害你的听力。音乐会各不相同，但很多音乐会的音量会导致听力损伤。

但是如果你知道你对周围的声音感到不舒服，那么确切的音量并不重要。你完全有权利带耳塞去任何你喜欢的音乐会、电影或活动，而且你可能不是唯一一个。我丈夫认为自己对噪音很敏感，有一次我们排队去听音乐会时，他抱怨说他把耳塞落在家里了。所以我就在售票处要了一些，他们很高兴地以一美元的价格卖给我们一双。*这是人做的事*。你不是唯一一个。挺好的。

你可以在药店或家装店花几分钱就能买到一次性耳塞，或者听听我们的资深视频制作人乔尔·卡恩对这些 20 美元的降噪耳塞 的推荐。照顾好你的耳朵，你就能在老年时抱怨电影院的音量了。
