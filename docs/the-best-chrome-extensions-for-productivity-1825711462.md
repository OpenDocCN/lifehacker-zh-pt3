# 提高生产力的最佳 Chrome 扩展

> 原文：<https://lifehacker.com/the-best-chrome-extensions-for-productivity-1825711462>

毫不奇怪，作为一个致力于提高工作效率的网站的编辑，我痴迷于细化我的科技生活细节，以确保每一个元素都在帮助我完成工作。虽然我有我所信赖的软件([Deckset](https://lifehacker.com/deckset-creates-great-looking-presentations-from-any-ma-1563410885#_ga=2.152466619.367633666.1525095724-3846207152.1521480874)[Evernote](https://lifehacker.com/ive-been-using-evernote-all-wrong-heres-why-its-actual-5989980#_ga=2.152466619.367633666.1525095724-3846207152.1521480874))，但我更喜欢我最喜欢的浏览器扩展。它们很轻便，易于安装，而且通常是免费的，但是它们对我的工作效率有着深远的影响。这些是我最热爱和最经常推荐的 Chrome 扩展。

Watch

作为一名虔诚的 [收件箱零](https://lifehacker.com/how-i-went-from-1-000-emails-to-inbox-zero-and-stayed-5984417) 从业者，我认为隔夜邮件是一种战争行为，所以感谢上帝为 Gmail 的电子邮件安排提供了回旋镖，让我可以早起回复邮件，而不会打扰我的同事。我可以在早上 5:30 写一封电子邮件，但是要确保它在早上 10:30 之前不会被投递。我可以想出一些我想在他们度假时告诉他们的事情，并在他们回来几天后安排一封电子邮件，这样我就不会打断他们的休假。

### [Checker Plus for Gmail](https://chrome.google.com/webstore/detail/checker-plus-for-gmail/oeopbcgkkoapgobdbedcemjljbihmemj?hl=en)

Checker Plus for Gmail 在您的 Chrome 工具栏中放置了一个小信封图标，它会在您有新邮件时通知您，并允许您在那里阅读和回复邮件，而无需打开 Gmail 标签。无论你是否打开 Gmail 标签，它都可以工作，这意味着你可以关闭 Gmail 桌面通知，如果曾经有过的话，这是一个生产力吮吸。

### [Checker Plus for Google Drive](https://chrome.google.com/webstore/detail/checker-plus-for-google-d/pppfmbnpgflleackdcojndfgpiboghga?hl=en)

同样有用的是 Checker Plus for Google Drive，它允许您直接从工具栏浏览和打开 Google Drive 中的文档或表格或演示文稿。任何能让我远离 G 套房的东西都是天赐之物。

### [就念](https://chrome.google.com/webstore/detail/just-read/dgmanlpmmkibanfdgjocnabmcaclkmod?hl=en)

从网页中删除格式、广告、注释和干扰，并以清晰易读的风格显示文本。您可以根据自己的意愿定制扩展，但我很喜欢默认的 CSS。

### [再次制造美洲小猫](https://chrome.google.com/webstore/detail/make-america-kittens-agai/klchnmggepghlcolikgaekpibclpmgcm?hl=en)

它被设计成用小猫代替唐纳德·川普的图像，但你也可以输入你自己想要阻止的人的列表——阻止哈维·韦恩斯坦！屏蔽前任！把他们都变成小猫！

### [脸书新闻提要根除者](https://chrome.google.com/webstore/detail/news-feed-eradicator-for/fjcldmjmjhkklehbacihaiopjklihlgg?hl=en)T3】

你还在等什么？这个扩展解决你的 FOMO 仅仅几秒钟。没有更多的新闻了！离你一直威胁的脸书分手又近了一步。

### [OneTab](https://chrome.google.com/webstore/detail/onetab/chphlpgkkbolifaimnlloiipkdnihall?hl=en)

把你打开的十万亿个标签页转换成一个列表。你可以节省内存，并获得关闭所有标签页的满足感。使用方便的 URL 与同事、朋友和您自己共享该列表。如果您想从另一台计算机访问 OneTab，请确保将它加入书签，因为它没有云同步功能。

### [保存到口袋](https://chrome.google.com/webstore/detail/save-to-pocket/niloccemoadcdkdjlinkgdfekeahmflj?hl=en)

Pocket 是我选择的“稍后阅读”应用程序，我使用 Save to Pocket Chrome 扩展作为另一个工具来关闭所有这些标签。一天之内，我会打开至少五个我没有时间阅读的长篇故事，但保存到口袋里意味着它们可以在周末安全可靠地阅读。(如果你正在寻找好的真实犯罪文章，当我完成时，我尝试 [推荐它们](http://getpocket.com/@dh9T2A01g55c4p9686d49d4debp0g085469cebi978K4eDTeW7dtsa37w8aEYcpr) 。)

### [天气](https://chrome.google.com/webstore/detail/weather/iolcbmjhmpdheggkocibajddahbeiglb?hl=en)

工具栏中的温度和天气预报。我设置了一个 IFTTT 食谱，每天下午 4 点通知我们团队 Slack 频道第二天的天气，这满足了我的一些天气需求，但全天持续的温度，全天强劲是一个梦想，特别是当纽约的天气在一天中从 50 到 80 度变化时。

### [【维基墙】](https://chrome.google.com/webstore/detail/wikiwand-wikipedia-modern/emffkefkbkpkgpdeeooapgaicgmcbolj?hl=en-US)

它让浏览维基百科变得更漂亮、更经典、更好。你在维基百科上花的时间可能比你想象的要多，Wikiwand 的左侧导航和其他设计调整使体验更快、更愉快。

这些并不是我所依赖的所有扩展——它们只是对我的工作效率影响最大的扩展。是的，我是一个忠实的 Chrome 用户，尽管它的用户界面很糟糕，而且不必要的臃肿，但是我知道许多这些，或者可接受的类似物，可以用于其他浏览器——如果你有一套杀手级的 Firefox 或 Opera 扩展，请在评论中告诉我们。
