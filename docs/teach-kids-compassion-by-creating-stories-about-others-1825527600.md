# 教孩子同情心的一个窍门

> 原文：<https://lifehacker.com/teach-kids-compassion-by-creating-stories-about-others-1825527600>

很容易得出结论，人普遍很烂。不是吗？有让你插队的司机，有在你耐心排队时不知从哪里冒出来偷走托盘上最后一份好市多样品的女士，有连续三年忘记你生日的“朋友”。我明白为什么我们认为其他人没有努力。

但是，这当然是一种有害的观点。它切断了我们与外界的联系，让我们变得暴躁和痛苦。作为一名家长，我想教我的女儿用同情而不是判断来看待他人——这是一项很难学会的技能，但它将每天为她服务。Sabina Nawaz 为《哈佛商业评论》 撰写文章，她分享了一项我非常喜欢的活动。她和她的孩子们玩一种他们称之为“多重意义”的游戏，这是一种简单的观察人的游戏，可以促进共鸣。它是这样工作的:

> 我们轮流根据在往返学校途中观察到的人和事来创作故事。例如，如果我们看到一个人在人行道上快速行走，手臂上有纹身，穿着无袖背心，我们可能会编造一个故事，说他上班迟到是因为他的车坏了，所以他走得很快去寻求帮助。也许他在桥对面开了一家纹身店，是他生意的活广告。或者他可能要去公园见某个人，而且要迟到了。当我们的孩子对家里或学校的事情感到不安时，他们就会使用这种技巧。这在儿子们吵架来找我调解的时候特别有帮助。为了减少冲突中的热度，我问道:“关于你哥哥为什么借你的乐高飞机，你还能做出什么别的解释？”目标是能够让自己冷静下来，更有同理心，所以他们带着好奇心而不是判断去接近别人。

我们经常教育孩子们少管闲事。但是如果我们没有呢？如果我们教他们对人感到好奇，甚至是那些可能伤害他们的人，会怎么样？如果我们提醒他们每个人都在打一场硬仗呢？如果威尔促使他们挑战自己的假设，并假定他人无罪——或者更好的是，询问他们的生活，会怎么样？在布琳·布朗的书《强势崛起》中，她问丈夫是否相信人们已经尽了最大努力。他的回答是这样的:“我不知道。我真的不知道。我所知道的是，当我假设人们尽了最大努力时，我的生活会更好。它让我远离判断，让我专注于现状，而不是应该或可能的情况。”正是如此。

和你的孩子一起，帮助他们利用他们对故事的天生热爱，为他们正在杂货店里发脾气的小孩、在银行里变得不安的男人或他们阅读的书中的恶霸想出他们自己的故事。最终，他们改变的故事将是他们自己的。
