# 如何检查优步或 Lyft 中隐藏的摄像头

> 原文：<https://lifehacker.com/start-checking-for-cameras-when-using-uber-or-lyft-1827831769>

根据圣路易斯邮报 最近的一篇报道，至少有一名为优步和 Lyft 工作的司机在未经乘客同意的情况下直播乘客。Twitch 上的观众一直在对女性乘客进行评级，猜测收入，并谈论拼车服务不知情用户的婚姻和个人生活。

根据 Post-Dispatch 的报道，已经有至少两个其他已知的司机将乘客变成直播对象的案例，有时经过同意，有时未经同意，包括加利福尼亚州的一些案例和威斯康星州的一个案例。

虽然我们都可以深呼吸一下，承认这些是拼车公司提供的大量通常无害的日常出行中的几个严重不道德的案例。尽管如此，这严重侵犯了隐私，也给骑车人带来了潜在的安全问题。

根据 Lyft 和优步两家公司向《邮报》发表的声明，圣路易斯的司机已经被切断了联系，所以你不会遇到他，但你还能做些什么来避免在搭便车时侵犯隐私呢？

首先，意识到风险是一个好的开始。寻找相机，并询问有关任何提及录音设备的贴纸的问题。这辆车后面的标签上写着:“注意:为安全起见，本车装有音频和视频记录设备。进入车辆表示同意。”接受采访的乘客说他们没有注意到这一点。意识到像这样的潜在通知有助于确保你知道如何避免它们——或者可以要求司机在你乘车时关闭摄像头。

也就是说，这个故事中令人不安的一个方面是，当司机被特别问及仪表板上安装的摄像头时，他撒谎了，并说它们是“为了安全”的记录设备。虽然传统的出租车公司通常会安装摄像头，表面上是为了司机的安全(并有标志通知你它们的存在)，但你应该怀疑 Lyft 或优步汽车上任何对准你的摄像头。

如果有一个标签提到录音或监视，而你对被拍摄感到不舒服，那就选择退出。安装摄像头不是公司的政策，如果你没有禁用录像/流媒体的选项，请订购另一辆车。这也是一个很好的做法，向公司报告，并在你的评级中提到这一点。如果对公司没有影响，至少可以让其他车友知道。

不幸的是，摄像头是可以隐藏的，去那个方向的司机不太可能会在车外贴一个友好的警告标签。所以这是“骑行风险自担……”并且可能要穿上你所有的衣服。
