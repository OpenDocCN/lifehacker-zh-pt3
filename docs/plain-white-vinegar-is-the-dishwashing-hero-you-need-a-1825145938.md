# 普通的白醋是你需要的洗碗英雄

> 原文:[https://life hacker . com/plain-white-醋是洗碗英雄-你需要-1825145938](https://lifehacker.com/plain-white-vinegar-is-the-dishwashing-hero-you-need-a-1825145938)

脂肪让食物变得美味，但油腻的食物在用餐结束后很久还会留在空气中——以及盘子上。如果你厌倦了在你发誓很干净的盘子上发现浮油，你应该在喷雾瓶里装满醋，并把它放在厨房水槽附近。

醋不能做整体健康博主声称它能做的一半事情，但是它的脱脂和除臭能力使它成为我洗碗库中不可或缺的工具。我用塑料汤容器来储存从炖菜到鸡尾酒糖浆的所有东西，所以为了避免味道污染，我需要它们完全干净。给曾经盛过椰子咖喱的塑料汤容器——或者特别是橄榄油味的抹布——大量喷洒醋，可以确保它从洗碗机中干净地出来，随时可以重复使用。(如果你没有洗碗机，用醋预处理油腻的盘子就更重要了——尤其是如果 [你的自来水是硬的](https://lifehacker.com/get-rid-of-all-kinds-of-hard-water-stains-with-vinegar-489265084) 。)由于油确实附着在气味上，醋冲洗也可以防止和/或减轻烹饪气味在塑料和 [硅树脂](https://lifehacker.com/how-to-clean-your-instant-pot-ring-1822165776) 等多孔材料中的积累；我用醋喷洒我的硅胶抹刀，每次使用前用热水冲洗它们，以尽量减少将旧食物(或肥皂)气味转移到我正在制作的任何东西上的可能性。醋分解干面包面团的速度也比我用过的任何其他东西都快，这让不少海绵免于一场肮脏、胶粘的死亡。

和任何清洁剂一样，醋也有其局限性:它既不是表面活性剂，也不是一种广谱消毒剂，醋酸的重量百分比约为 5%。对于粗糙、烧焦的油脂，你最好使用 [像酒保的朋友](https://skillet.lifehacker.com/how-to-clean-burned-on-gunk-off-your-instant-pot-insert-1823565725#_ga=2.122548683.1156816996.1523368713-144977662.1506607947) 和 [之类的东西，光用醋可能无法杀死生猪肉或鸡肉中更讨厌的微生物](https://www.ncbi.nlm.nih.gov/pubmed/9713753)；此外，它的酸性会腐蚀由铝、铸铁、无衬里铜和碳钢等活性材料制成的炊具。但是对于一种高效的轻型脱脂剂和除臭剂来说——这也恰好是完全食品安全的——醋是很难打败的。
