# 现在你可以和 Alexa 谈论松弛

> 原文：<https://lifehacker.com/now-you-can-talk-to-alexa-over-slack-1797164787>

Alexa 旨在与你的声音一起工作，但有时对虚拟助理大喊大叫并不完全理想。现在你可以和 Alexa 交谈，并在空闲时使用她的大部分能力。



它叫[SilentEcho](https://silentecho.bepoken.io/)T3。由 Bespoken 开发的这个机器人( [，目前正处于测试阶段](https://bespoken.tools/blog/2017/07/13/silent-echo-for-alexa) )使用文本到语音转换技术将你输入的单词转换成语音。然后，它使用亚马逊的 API 将语音信息发送给 Alexa 来处理你的任务，即使背景中有尖叫的孩子、手提钻或大声的音乐。 

这个想法是为开发 Alexa 应用程序的开发人员创造一种更容易与虚拟助手互动的方式，但它也可能对非开发人员有用。

他们现在正在我的大楼里施工，这让我在白天舒适地躲在我的降噪耳机里。不幸的是，屏蔽掉电锯意味着我也屏蔽掉了我的回声。现在我可以做大部分我白天会用回声做的事情，而不用对着建筑设备大喊大叫。

无声回声不是万能的。出于安全原因，该应用不允许你做任何与支付账户相关的事情，所以你不能从亚马逊订购多米诺骨牌或卫生纸。然而，你可以做一些事情，比如打开家里的灯，向播放列表添加一些内容，或者询问天气更新。一个普通的静默 Echo 帐户(绑定到一个拥有超级有限权限的 Amazon 帐户)也可以在群聊频道中使用。

您需要成为您的 Slack 帐户的管理员才能启用该机器人。如果你是一个小团队的应用程序开发人员，这应该不会太难，但对于那些为大公司工作的人来说，说服老板签约可能有点挑战。

这当然不是你会一直使用的东西(有时甚至网络搜索会比这样做更容易)，但它有时肯定会派上用场。

Twitter 版本也即将推出，可能会更加有用。
