# 六月书友开放线程:死后还有生命吗？

> 原文：<https://lifehacker.com/june-book-club-open-thread-is-there-life-after-death-1827284492>

欢迎您的 [六月书友](https://lifehacker.com/our-june-book-club-pick-is-1826549082) 开帖。欢迎你对玛丽·罗奇的《T4 幽灵:科学解决来世》有任何想法。请记住，这完全是为了好玩——本书涉及的许多话题(宗教、死亡、伪科学)都有争议，但是，用福克斯新闻频道评论家和民主党领导人的话说，让我们 [保持文明](https://www.cnn.com/2018/07/01/politics/civility-in-the-civil-rights-era/index.html) 。

Watch

我的观点:罗奇写这本书绝非易事——毕竟，人死后会发生什么是个永恒的问题。尽管如此，我发现*幽灵*的结论令人满意(一分钟后会有更多)。尽管它是非虚构的，但它有一定的虚构倾向，可能是因为所有早期的自我反省“科学”和对灵媒和 EVP 的关注，这使它成为一种容易和迷人的阅读。

我发现最引人注目的部分是关于灵媒、灵媒和灵光的两章，可能是因为，正如罗奇概述的那样，人们直到今天仍然相信它们。(如果我说我自己没有读过一两篇文章，那我是在撒谎。)

从 177 页开始:

> 这个周末我学到了一些东西，尽管这不是我来这里的目的。我认识到我对灵媒的看法是错误的。我不再认为他们是故意杜平他们的客户。我相信，他们相信，诚实和确信，他们从超自然的来源获得信息。只是对一组事实的不同解读。

我认为这是关键:我们永远不会知道，所以我们都有自己不同的解释。一些人相信有天堂，另一些人相信我们的灵魂会以黑洞的形式出现。谁能说什么是真实的。

至于死后的生活，你可以把我算作怀疑者之一。罗奇在整本书中指出，我们会相信我们想要的，听到我们说服自己的，并附和媒体所说的，因为我们需要令人愉快。我们需要有答案，确切地知道当我们死去时会发生什么，而且不会只是一片虚无——因为如果是的话，那么，这一切是为了什么？

为此，我同意罗奇在第 294 页的结论:

> 我在证据收集者中的一年是否让我相信了一年前我不相信的事情？的确如此。它让我相信了布鲁斯·格雷森的信念。我问他是否相信濒死体验提供了死后生命的证据。他回答说，他所相信的只是它们证明了一些我们目前无法解释的事情。我想我相信，并不是我们人类在生活中遇到的每一件事都可以整齐而令人信服地藏在有序的科学橱柜里。

考虑到一些科学家认为灵魂呈现绿色，它包含在精子中，以及星质不久前是真实的，我倾向于相信有无限的事情我们永远无法确定。
