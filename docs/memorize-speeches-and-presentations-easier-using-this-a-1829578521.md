# 使用这个应用程序更容易记住演讲和演示文稿

> 原文：<https://lifehacker.com/memorize-speeches-and-presentations-easier-using-this-a-1829578521>

记忆东西，尤其是像演讲和演示这样的长东西，可能很难做到。像所有的事情一样，熟能生巧。然而，除非你身边有一个朋友来帮助你练习，否则你很容易在练习时出错，错过一些部分，或者一开始就没有把整个记忆过程做好。

Watch

[熟记于心](https://itunes.apple.com/us/app/memorize-by-heart/id1436935583?) 是一款试图提供帮助的 iOS 应用。使用 0.99 美元的应用，你可以将文本复制并粘贴到应用中，然后通过让应用选择性地从组合中删除字母或单词来测试你记住文本的能力，有效地让你填充空白。

你也可以像在房间里有一个朋友一样使用应用程序，并背诵课文。它会让你知道你在哪里搞砸了，并给出你在这个过程中所处位置的百分比分数。例如，它可能会让你知道你只记住了 50%的演讲，或者它会确认你已经记下并准备好了。

如果你有一些东西需要记忆，这个应用程序可以处理“无限记忆”，尽管我不完全确定你的大脑可以。尽管如此，如果你正在为班或工作会议准备完美的演示文稿，它可能是你的武器库中的一个很好的工具。如果你正在寻找其他的解决方案， [使用这种字体也可能有助于](https://vitals.lifehacker.com/remember-what-youve-read-with-this-font-1829554064) 。
