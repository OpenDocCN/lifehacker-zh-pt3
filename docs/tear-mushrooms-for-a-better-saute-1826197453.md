# 撕蘑菇以获得更好的嫩煎

> 原文:[https://life hacker . com/tear-mon 菇-for-a-better-saute-1826197453](https://lifehacker.com/tear-mushrooms-for-a-better-saute-1826197453)

如果你想要一个完美的烤焦，蘑菇是一个巨大的痛苦。不管你是否冲洗它们，仅仅切片和切碎就能释放出足够的天然水分，即使是之前干透的蘑菇在出锅前也会变湿——这根本不行。

Watch

我们都知道水分是脆脆的金褐色美味的敌人，所以尽可能保持蘑菇干燥是成功煎炒的关键。我用了两个准备技巧来减少水分，这两个技巧都是我从这个关于烤蘑菇的 Bon Appétit 视频 中学到的。首先，我用毛巾擦掉它们，而不是冲洗。漂洗过的蘑菇实际上不会吸收那么多水分，但*内部*的水分不是问题:即使是最快的漂洗也会使表面变得黏糊糊、潮湿，而且不适合毛巾擦干。第二，也是更重要的，我用手指去梗和撕蘑菇，而不是用刀切碎。(不要把那些梗扔掉；他们做 [极好的蔬菜高汤](https://skillet.lifehacker.com/how-to-make-truly-great-vegetable-stock-1822194489) 。)这将蘑菇破碎成更小的块，而不会破坏保持在水中的纤维结构，这确保了当它们与热脂肪接触时，暴露的表面是漂亮和干燥的。它也需要大约一半的时间。

撕裂的方法不会对所有的准备工作都起作用——它可能不是 [*杜塞尔斯*](https://skillet.lifehacker.com/will-it-sous-vide-a-most-glorious-beef-wellington-1790825718#_ga=2.84553049.1369191435.1526907270-1723114163.1524514905)T5】的伟大选择——但如果你想展示新鲜蘑菇的天然质地和风味，它是无法击败的。
