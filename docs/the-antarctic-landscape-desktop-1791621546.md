# 南极风景桌面

> 原文：<https://lifehacker.com/the-antarctic-landscape-desktop-1791621546>

读者赛法利过去曾向我们的 [桌面展示区](http://kinja.com/tag/desktop-showcase) 提交过桌面，但这是一个全新的外观，我们喜欢它。如果你也喜欢它——或者只是喜欢其中的南极风景——以下是如何让你的电脑也有同样的外观。



Saifali 正在使用 rainmet 来获得这种外观，我们刚刚更新了我们的指南 [使用 rainmet](https://lifehacker.com/how-to-create-an-attractive-customized-desktop-hud-wit-5828789)定制您的 Windows PC，如果您有兴趣亲自尝试的话。这是一本很好的入门书，可以帮助你了解这个应用的来龙去脉，并开始构建一个可以与我们分享的桌面！

一旦你记下了这些，下面是你需要的:

*   [南极洲谷歌地球查看](https://earthview.withgoogle.com/antarctica-1159) 壁纸
*   用于 Windows 的 [雨量计](https://www.rainmeter.net/) 系统调试和监控工具
*   雨量计的 [英格玛](http://kaelri.deviantart.com/art/Enigma-103823591) 套件，用于当前播放小部件、日历小部件和天气小部件
*   [Aperture Labs 信息标志皮肤](http://wistfulwriter.deviantart.com/art/The-Portal-2-Desktop-Theme-for-Rainmeter-245722598) 中央为时钟和日期显示

这就是它的全部——没有太多的活动部件，但无论如何看起来都很不错。如果你喜欢它，或者对它有疑问，可以去 Saifali 的 Kinja 博客(链接如下)询问。趁你还在唱赞美诗！

你有自己的好看又实用的台式机可以炫耀吗？分享给我们吧！使用标签 [***将它发布到您的个人 Kinja 博客***](http://kinja.com/tag/desktop-showcase) *或将其添加到我们的*[***life hacker 桌面展示和告诉 Flickr 池***](https://www.flickr.com/groups/lifehacker-desktop-showandtell/pool/) *。截图必须至少为 1280x720，并请包括您使用的信息，链接到您的壁纸，皮肤和主题，以及任何其他相关的细节。如果你的超棒的桌面吸引了我们的眼球，你可能会成为我们的特色！*
