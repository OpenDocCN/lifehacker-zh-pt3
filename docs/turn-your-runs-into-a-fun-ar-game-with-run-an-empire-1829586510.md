# 通过“经营一个帝国”，将您的跑步变成一个有趣的 AR 游戏

> 原文：<https://lifehacker.com/turn-your-runs-into-a-fun-ar-game-with-run-an-empire-1829586510>

跑步一直是我需要某种动力的事情之一。为了我的健康，周六出去跑步？我不去了。周六出去跑步，这样我就可以在 FitBit 挑战赛中击败我所有的朋友？现在我们在谈话。



本周，我发现了一款新游戏，它已经让我家附近的几次长途步行变得热闹起来: [经营帝国](http://www.runanempire.com/) 。

这是口袋妖怪 Go 和模拟城市的结合。你通过在你的邻居中进行真实世界的散步或跑步来玩这个游戏。当你穿越一个地区时，你可以以你的帝国的名义占领它。你移动得越多，你就有越多的货币来建立这个帝国。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-vwq9u1JjX0c&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-vwq9u1JjX0c&start=0) 

与其他跑步游戏不同，这款游戏可以让你在回家后保存所有的游戏内容。

我对它还很陌生(它在 2 号发布)，但是到目前为止它很有趣，并且是(真的)更进一步的巨大动力。
