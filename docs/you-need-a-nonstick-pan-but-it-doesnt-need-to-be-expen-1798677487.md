# 你需要一个不粘锅，但不需要很贵

> 原文：<https://lifehacker.com/you-need-a-nonstick-pan-but-it-doesnt-need-to-be-expen-1798677487>

一般来说，我不使用不粘锅，但也有一些情况——比如 [烹饪完美的两面煎鸡蛋](http://skillet.lifehacker.com/how-to-make-a-perfect-over-easy-egg-1798540985) 、煎蛋卷和薄煎饼——其他东西都不行。然而，我不建议在一个上面花很多钱。



现在，在我们进一步讨论之前，让我们来解决每个人最喜欢的不粘怪物:特氟隆，以及围绕它的安全问题。 [根据美国癌症协会](https://www.cancer.org/cancer/cancer-causes/teflon-and-perfluorooctanoic-acid-pfoa.html) 的说法，你可能会因为吸入过热的特氟隆(其中 [不应该加热到 500 华氏度](https://www.chemours.com/Teflon/en_US/products/safety/key_questions.html) )排出的气体而出现类似流感的症状，所以不要这样做。除此之外，还有一些关于 PFOA 的小问题，它在特氟隆的制造过程中使用，但不存在于最终产品中(或者只存在非常少量)。

特氟隆会随着时间降解，当它降解时，你应该扔掉它。这不是因为潜在的健康危险，而是因为，一旦你的特氟隆开始失效，你的食物就会开始粘粘的，那么有一个粘粘的不粘锅又有什么意义呢？有一些不含特氟隆的陶瓷选择——我也有一个——但我发现它们比特氟隆更快失去不粘的魔力，而且它们往往更贵。是的，铸铁是一种不粘的选择，但即使是最老练的锅也不会像特氟隆一样光滑。 [只是科学](https://lifehacker.com/why-nothing-sticks-to-teflon-pans-and-cookware-1790527904) 。(如果你能让它工作，那就给你更多的力量。)

因此，我强烈主张买一个便宜的特氟隆涂层不粘锅，使用直到它开始抓住食物，然后扔掉它，换一个新的。因为涂层不可避免地会被破坏，而且你家里的某个人也很可能会在某个时候用金属刮刀来刮，所以花大价钱是没有意义的。( [)认真吃同意我这个](http://www.seriouseats.com/2017/02/why-cheap-nonstick-skillets-are-best.html) 。)

你可以在餐馆供应店找到一个便宜的，但是，如果你有一个感应燃烧器的情况，你需要一个不锈钢制成的，因为 [铝不能与感应燃烧器](https://en.wikipedia.org/wiki/Induction_cooking) 一起工作。但是对于其他人来说，铝是可以的，只要它不太薄。[Farberware](https://www.amazon.com/Farberware-Restaurant-Aluminum-Nonstick-Skillet/dp/B000658GT6?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/you-need-a-nonstick-pan-but-it-doesnt-need-to-be-expen-1798677487&asc_source=&tag=kinjalifehackerlink-20)(认真吃也是一个粉丝)的这个是一个二十美元以下的好选择，很难击败。
