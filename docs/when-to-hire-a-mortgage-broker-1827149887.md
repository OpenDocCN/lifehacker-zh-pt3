# 何时雇用抵押贷款经纪人

> 原文：<https://lifehacker.com/when-to-hire-a-mortgage-broker-1827149887>

假设你自己正在做一些抵押贷款研究——你已经和一位财务规划师谈过了，并准备在购房过程中迈出下一步。乔恩想知道:

> 我的计划是在今年夏天/秋天买些东西，并且一直在努力把我的事情和首付安排好。我通过我的员工援助计划(EAP)与一名财务规划师就一些事情进行了交谈，并讨论了预批准流程。几个月前，我联系了我的银行，想知道他们会预先批准我做什么。在与财务规划师和一些朋友交谈时，他们提到我应该寻找至少一家传统银行和两家抵押贷款公司/经纪人。
> 
> 关于我应该在抵押贷款经纪人中寻找什么，有什么建议吗？它是一家全国性的、地区性的还是地方性的公司有关系吗？



如果你想买房子，雇佣一个抵押贷款经纪人会让你更容易比较利率，找到一个好的交易。但是有必要吗？

以下是你需要知道的。抵押贷款经纪人本质上是你和贷款人之间的中间人。他或她努力为你找到一个能给你提供最优惠利率的贷款人。Redfin Mortgage的负责人杰森·贝特曼说，抵押贷款经纪人可以帮你省去确定所有贷款选项的跑腿工作，他们可能有一个贷款人网络来帮助你及时获得几个报价。这些贷款人可能不会独自与个人合作。

但是专家们对你是否需要雇佣一个持不同意见。

一方面，“与抵押贷款经纪人合作并不能保证你会得到最好的交易，他们也不能保证他们的估价，”贝特曼说。"通常，你自己做工作会产生同样的结果，而不需要支付费用."

与此同时，联合批发抵押贷款公司(United Wholesale Mortgage)的总裁兼首席执行官马特·伊什比亚(Mat Ishbia)指出，虽然买房是我们将要做出的最重要的财务决定之一，但人们通常在不得不扣动扳机之前不会有太多的经验。经纪人可以提供帮助，而且可能值得这笔费用，通常是贷款金额的百分之一，并且可以帮助你节省一些其他的购房费用，如申请费或评估费。

“你不能独自办理抵押贷款，这不像买机票，”伊什比亚说。

不管怎样，在你见经纪人之前，你会想做一些调查。多引用几句也无妨。使用在线抵押贷款计算器来了解利率，并自己从贷款人那里获得一些报价。Trulia 抵押贷款副总裁兼总经理艾琳·兰茨说:“也可以向朋友和家人寻求建议。“抵押贷款经纪人是你想信任并感到舒适的人，就像房地产经纪人一样。”当你寻找经纪人时，本地的更好，因为他们会非常了解市场和你所有的选择。
