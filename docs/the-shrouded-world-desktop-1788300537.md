# 笼罩的世界桌面

> 原文：<https://lifehacker.com/the-shrouded-world-desktop-1788300537>

读者 [Alec](http://kinja.com/stranaha2) 向我们的 [桌面展柜](http://kinja.com/tag/desktop-showcase) 提交了这款桌面，除了一张惊艳的壁纸，就是一台定制的整整齐齐的 Mac。他是这样安排一切的。



Alec 正在使用 bersicht， [一个 macOS 定制套件，我们在](https://lifehacker.com/ubersicht-customizes-your-os-x-desktop-with-widgets-and-1591137635) 之前已经强调过了。您将需要抓住它开始，但它几乎是所有您需要的。

安装完成后，下面是其他部分:

*   来自 DeviantArt 的 [Tierra 壁纸](http://krysis08.deviantart.com/art/Earth-Tierra-Wallpaper-HD-383143504)
*   macOS 的[über sicht](http://tracesof.net/uebersicht/)定制套件
*   用于可定制日历的 [侧边栏小部件](http://tracesof.net/uebersicht-widgets/#Sidebar) 和隐藏在屏幕一侧的实用面板
*   桌面时钟的小部件
*   [励志语录](http://tracesof.net/uebersicht-widgets/#inspirational-quote) 小工具
*   [全白图标](http://noshery.deviantart.com/art/All-White-Icons-for-OSX-470738808) 图标包为 OS X
*   [cDock 定制实用程序](http://w0lfschild.github.io/app_cDock.html) 调整 Dock 布局
*   调整系统图标的 [LiteIcon](https://freemacsoft.net/liteicon) 实用程序(带有白色图标包)
*   [调酒师](https://www.macbartender.com/) 实用程序来调整菜单栏
