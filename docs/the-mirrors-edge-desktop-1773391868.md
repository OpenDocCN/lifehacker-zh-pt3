# 镜子的边缘桌面

> 原文：<https://lifehacker.com/the-mirrors-edge-desktop-1773391868>

我对《镜的边缘:催化剂》感到非常兴奋，而这个由读者 [0BUX](http://kinja.com/sahajkoka) 提交给我们的 [桌面展示](http://kinja.com/tag/desktop-showcase) 的桌面肯定没有帮助。看起来很神奇。以下是如何让它成为你的。

Watch

这是一个 Windows 10 桌面，但尽管它的屏幕尺寸和分辨率很奇怪，但你仍然可以在你的 PC 上获得相同的效果。它使用 rainmet，所以如果你不熟悉如何用它定制你的桌面， [在开始之前，在这里](https://lifehacker.com/how-to-create-an-attractive-customized-desktop-hud-wit-5828789) 查看我们的 rainmet 入门指南。

现在，这是你需要的:

*   [镜边壁纸](http://xboxone.gaming-universe.org/screens/e3_mirrorsedge.jpg)
*   用于 Windows 的 [雨量计](https://www.rainmeter.net/) 系统调试和监控工具
*   [桌面皮肤为雨量计](http://aidendrew.deviantart.com/art/DeskTop-505302516) 左上角为日期显示
*   [风华二皮](http://lilshizzy.deviantart.com/art/Rainmeter-Elegance-2-244373054)为雨量计为时钟左上角
*   左下方为系统监视器的 [雨量计的简约系统信息皮肤](http://dashie36.deviantart.com/art/Minimalistic-System-Info-v2-333556546)
*   用于雨量计的 [Monstercat 可视化工具](http://marcopixel.deviantart.com/art/Monstercat-Visualizer-for-Rainmeter-1-3-486330771) 用于随音乐移动的音频可视化工具(穿过底部)
*   [经典外壳](http://www.classicshell.net/) 用于 Windows，使任务栏透明——0BUX 注意到该应用程序的所有其他功能都被禁用

这是这一个的所有移动部分。如果你喜欢的话，你可以用更少的部件很容易地得到同样的效果，或者你可以只拿走那张壁纸，然后就到此为止。我们不会评判。不管怎样，如果你喜欢这里的作品，去 0BUX 的 Kinja 博客(链接如下)让他们知道你喜欢这个成品！
