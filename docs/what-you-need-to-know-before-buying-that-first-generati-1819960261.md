# 在购买第一代智能手机之前，你需要知道什么

> 原文：<https://lifehacker.com/what-you-need-to-know-before-buying-that-first-generati-1819960261>

对于第一代智能手机来说，2017 年将是非常非凡的一年。Android 创始人安迪·鲁宾 [首次展示了他新公司的基本手机](https://gizmodo.com/a-normal-persons-guide-to-the-essential-phone-the-most-1797961321) ，该手机使用嵌入式磁铁连接到 360 度摄像头和充电底座等设备。游戏配件公司 Razer 宣布了其以娱乐为中心的 [Razer 手机](https://gizmodo.com/the-razer-phone-is-here-but-is-it-for-real-1819873239) ，这是另一款第一代智能手机的精神继承者，云连接的 Nextbit Robin。你可能会对试用这些全新智能手机的想法感兴趣。但在你跳入未经测试的水域之前，你应该考虑购买本质上是独立智能手机的东西。配有模块化配件或高性能显示屏的手机绝对值得一看，但你应该做好应对潜在障碍的准备。



### **首先，要知道你要做什么**

当你从一家新公司购买第一代智能手机时，要知道这一点:你会在其中度过一段美好时光，而不是很长时间。

像 Essential Company 和 [Nextbit](https://gizmodo.com/nextbit-robin-review-a-great-smartphone-with-its-head-1759734910) 这样的初创公司的手机充满了创新的想法和新的交互方式，你可能不会在更主流的设备中看到。例如，游戏友好型 Razer 手机的显示屏刷新率为 120Hz，非常适合快节奏的游戏，这在智能手机中是闻所未闻的。

不幸的是，由于各种原因，它们比更成熟的智能手机制造商的产品更容易失去支持。将其归咎于销售不佳(亚马逊 Fire Phone)、管理不善(惠普 Pre 3)或从其他公司收购(Nextbit Robin)。

每一部第一代智能手机都有可能成为一家公司的最后一部手机。以 Nextbit 的 Robin 为例，这是一个 Kickstarter 项目，在只做了一款手机后就被 Razer 收购了。或者是 Canonical 的 Ubuntu Edge，它在达到 3200 万美元众筹目标的 40%后就再也没有出现过。惠普(通过收购 Palm)推出了其首款 WebOS 智能手机 [惠普 Pre 3](https://gizmodo.com/goodbye-webos-the-greatest-phone-youll-never-use-5832332) ，仅在第二天就停止了整个 WebOS 设备系列。如果你不是赌博型的，你应该坚持你所知道的。

### **阅读保修书**

如果你买的是第一代智能手机，肯定会有一些问题没有解决。如果你的设备出现任何问题，你的保修就是你的朋友。虽然像苹果这样的公司有可靠的维修服务，如事故友好型 Applecare+，但缺乏相同基础设施的新公司意味着你将邮寄你的智能手机并等待几天，而不是将其带到零售店进行快速更换。

而且自己修理比用更受欢迎的手机要困难得多。你可以在 iFixit 等网站上购买流行的 iOS 和 Android 设备的维修零件，但新手机的零件几乎不可能买到。如果可以的话，从手机制造商或无线运营商那里购买延长保修或保险。这项投资将使你不用担心如何更换破裂的屏幕。

### **拿去试运行**

虽然获得第一代智能手机是一场赌博，但如果你觉得它不适合你，你不必坚持使用它。利用你的零售店或无线运营商的退货政策。如果你是从制造商那里购买，请阅读退货政策细则。大多数公司提供 14 天退货政策，少数公司和无线运营商收取补货费。你可能会损失一些钱，但至少你不会被一部只因为与众不同而吸引你的手机所困扰。

### **第三方支持并不是一个真正的选择**

除了手机制造商，全新的手机并不能为任何人赚钱。这意味着你可能不会从流行的配件制造商那里找到很多兼容的配件。不要指望走进百思买(Best Buy)就能得到一个额外的保护套或屏幕保护器。你最好的选择是直接从制造商那里购买配件，这是唯一一家关心配件是否畅销的公司。

### **您支持创新(直到支持结束)**

虽然这可能会让你觉得签约是为了被欺骗，但还是要从你支持创新的事实中得到安慰。将它视为竞争中的一项投资，是扭转这种局面的一种方式，无论是对你自己，还是对问你为什么不买 iPhone 的朋友。毕竟，如果没有人购买 Nextbit Robin，Razer 新的专注于游戏的智能手机可能仍然是首席执行官陈民亮眼中的微光。

购买第一代智能手机对于激发竞争精神是必要的。竞争者之间的多样性是提高发布产品质量的众多方法之一。毕竟，如果你是一家年复一年都处于领先地位的公司，那么除非有人超过你，否则为什么还要去挑战极限呢？
