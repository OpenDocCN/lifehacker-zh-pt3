# 如果你负担不起治疗师的费用，试试这些心理健康资源

> 原文：<https://lifehacker.com/if-you-cant-afford-a-therapist-try-these-mental-health-1818506208>

如果你有心理健康问题，但负担不起(或抽不出时间来)一对一的治疗，有许多替代选择仍然可以让你接触到有执照的治疗师。还有丰富的自我保健资源可以补充治疗。作家蒂芙尼·霍华德 [在这篇推特帖子](https://twitter.com/OnlineWithTiff1/status/908851180884959238) — [中列出了他们的名单，下面是重新格式化后的一份易于阅读的名单](https://tttthreads.com/t/908851180884959238) 。



资源包括 [开放路径心理治疗集体](https://openpathcollective.org/) ，其中列出了低成本的治疗师(30-50 美元一次)，以及世俗自助组织 [康复国际](https://www.recoveryinternational.org/) 。霍华德还链接了一些心理健康播客和在线治疗形式，如 [更好地帮助](https://www.betterhelp.com/) 的在线治疗师目录。

还要翻翻 [Greatist 的 81 个平价精神卫生资源](https://greatist.com/grow/resources-when-you-can-not-afford-therapy) 的清单，我们 [在 2015](http://vitals.lifehacker.com/find-mental-health-help-for-almost-anything-with-these-1728037154) 覆盖了，还有 [我们自己的平价资源清单](http://lifehacker.com/top-10-free-and-affordable-mental-health-and-counseling-1788814933#_ga=2.166729463.873860214.1505741708-1297080755.1497980211) 。
