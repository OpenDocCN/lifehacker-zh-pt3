# 你的胸罩尺寸是一个神话

> 原文:[https://lifehacker.com/your-bra-size-is-a-myth-1791827401](https://lifehacker.com/your-bra-size-is-a-myth-1791827401)

在一个虚荣的尺寸和神秘的名称如“中号”的世界里，你通常必须试穿衣服才能知道它们是否合适。胸罩尺寸看起来会有所不同，因为它涉及到数字和数学，但我在这里有一个坏消息:没有一个真正的胸罩尺寸。

### 甚至没有人同意如何衡量自己

这听起来很科学:你只需测量你的胸腔和胸部，然后使用一个简单的公式将这些测量值转换为由一个数字和一个字母组成的胸罩尺寸(例如，“34B”)。这是典型的公式，但正如我们将看到的，它会有所不同:

*   你的胸围尺寸(数字)来自对你胸部的测量，不包括你的胸部。有些说明书让你在乳房下测量；有的让你量上面，基本上就在腋下。
*   你的罩杯尺寸(字母)来自于胸围尺寸和胸围尺寸之差。在一些版本的公式中，你只需减去。在其他情况下，在比较这两个数字之前，你会在你的表带尺寸上增加几英寸(通常是 4 英寸)。

如果公式中的变化能给你相同的尺寸，那就没问题了，但事实并非如此。我做了一个小实验来说明这一点。

我测量了胸围上方、上方和下方的尺寸，并把它们输入到各种计算器中。大多数说明要求你穿已经很合身的胸罩，事实上我有这样一个胸罩:华纳的 品牌的 [中的 34D。让我们看看各种计算器是如何组合的:](http://warners.com/)

*   [【Calculator.net】](http://www.calculator.net/bra-size-calculator.html?bustsize=37&bustsizeunit=inch&framesize=30&framesizeunit=inch&x=75&y=12):34F。
*   [**母性母性**](http://www.motherhood.com/SizeChart/SizeChartBraCalculator.asp)T5】:34C。(他们卖孕妇文胸，但是告诉你买孕前的尺码。)
*   [**维多利亚的秘密**:](https://www.victoriassecret.com/bras/find-your-perfect-fit#step/fit-guide-bra-size)T5】32DD。
*   [**HerRoom**](http://www.herroom.com/bra-fitting-advice,901,30.html)**:**32DD
*   [**老练一对**](http://www.sophisticatedpair.com/bra-size-calculator/) : 30G。
*   [**琳达文胸女士**](http://www.lindasonline.com/bra-fitting-calculator.html) : 30G。
*   [**适合的文胸**](http://www.brasizecalculator.tk/) : 30DDD 或者 30F(他们说我是中号。)

华纳没有计算器，但他们有一个教学视频。我按照说明，并提出了 33D。由于表带尺寸只有偶数，这听起来像是对我穿他们品牌的 34D 的一个很好的认可。

这里的底线是:遵循一个品牌的说明可能会帮助你在那个品牌找到合适尺寸的胸罩*，但是不要指望那个尺寸会帮助你在另一个品牌找到合适的胸罩。*

### 罩杯大小(几乎)没有任何意义

人们很容易认为杯子的大小就是杯子的大小。例如，如果你是 D 罩杯，你可能会认为任何 D 罩杯的胸罩都应该适合你的胸部。但是事实证明，举例来说，32D 和 40D 在杯子里没有任何接近相同体积的空间。缝制胸罩的人需要为每个罩杯购买合适尺寸的胸罩钢丝，这张比较胸罩钢丝尺寸和胸罩尺寸的图表 揭示了两者之间的关系。32D(反正是这个牌子的)用的线和 38A 的线一样大。

这种怪癖就是所谓的 [姐妹尺码](https://lifehacker.com/find-your-perfect-bra-quickly-by-knowing-your-sister-br-1384527738) 背后的原因:罩杯尺寸相同，但肩带尺寸不同。举例来说，如果你穿 36C，34D 是同样的罩杯，但带子较短。迷惑？请看本指南中的图表:

在将字母大小映射到尺寸上还有一个更大的困惑:一旦你超过了 D，每个人似乎对下一个字母有不同的想法。一些品牌直接到 E，F，G，等等，而另一些品牌是双重的:DD，然后是 DDD，然后是另一个字母。HerRoom 使用一种他们称之为的标准来调和品牌之间的差异，因此如果你想尝试一个新品牌并想知道从哪里开始，这可以提供帮助。但是，如果你想要的品牌不在他们的清单上，或者你在一个没有转换图表的实体店，该怎么办呢？

### 你最好的办法是:试穿一下

仅仅通过研究图表和计算器，你永远不会得到一个关于胸罩尺寸的完整答案，希望你能走进一家商店，拿起一件 X 号的胸罩，带回家，知道它会很舒适。在某些时候，你必须实际试穿胸罩，并评估它是否合身。简而言之，你在寻找这样的东西:

*   带子水平放置(不是在你的背部上下拉动)。
*   你的乳房填满了罩杯，没有留下空隙或溢出。
*   肩带不会从你的肩膀上滑落，也不会陷进去。
*   电线(如果有的话)不戳你；这可能意味着杯子太小了。

回到我们最初的实验，从一件合适的胸罩开始:仅仅看标签不是欺骗。 [如果符合，就符合](https://jezebel.com/bra-sizing-makes-no-goddamn-sense-we-have-to-game-the-716381756) 。

除了胸罩的尺寸，合身也取决于形状。例如，一些胸罩是为了更圆的乳房而裁剪的，而另一些是为了更像泪珠形的乳房。说到底，胸罩就像其他衣服一样:标签暗示了它的合身程度，但只有试穿才能确定答案。
