# 如何应对生活中在财务上不负责任的人

> 原文：<https://lifehacker.com/how-to-deal-with-the-financially-irresponsible-people-i-1788293059>

对于那些试图过上或保持财务责任生活的人来说，最大的挑战之一就是在他们的生活中存在财务上不负责任的人。有些人利用社会压力来说服你做出错误的消费选择或养成不良的财务习惯。有人利用他们与你的关系来说服你给他们钱。有些人会问你借钱，却从来不还。你甚至可能有人会直接动用你的资金，并把它们用于不必要的用途。

不管你对自己的良好财务选择有多努力，这些事情都会严重扰乱你的财务进展。这是因为，在每一种情况下，财务上不负责任的人会利用你生活中财务以外的方面来鼓励你做出糟糕的财务选择。他们可以利用你生活中的家庭、浪漫、社交甚至职业领域，潜移默化地(或不那么潜移默化地)将你推向不良的金钱行为。

许多人在这种情况下的两难处境是，他们觉得好像必须在金钱和人之间做出选择，而选择金钱是错误的(T2)。事情是这样的:你拥有的钱几乎总是你个人努力工作和艰难选择的结果。你做了很多牺牲才赚到那笔钱。如果你愚蠢地花掉它，你就不会有那笔钱。通过愚蠢地使用它或者把它送给会愚蠢地花掉它的人，你不是在浪费你的钱，你是在浪费你的生命*。你牺牲了所有改善财务状况的艰难选择和努力工作。*

我应该知道——我已经犯了很多这样的错误。我花钱和朋友“保持联系”。我把钱给了朋友和家人，知道这些钱永远不会还回来(有时希望会还回来，结果却失望了)。我让我的配偶花的钱比我预期的要多(老实说，我肯定过去我也这样对我的配偶)。有人利用我和他们的私人关系向我要钱，或者邀请我的妻子(总是我的妻子)参加一个“聚会”，在那里，社会压力被用来说服她购买价格过高的商品。

每一件事都是个错误。每一件这样的事情的发生都是因为让经济上不负责任的人在我的生活中拥有太多的利益。

所以，我开始限制赌注。总的来说，我采取了两种方法中的一种:我要么找到方法最小化那些在财务上不负责任的人影响我财务的能力，要么我温和地最小化他们在我生活中的角色。以下是我曾经使用过的或者我推荐给处于这种情况下的人的一些具体策略。

### **经济上不负责任的配偶**

许多经济上有责任感的人被经济上不负责任的配偶所困。这可能以几种方式发生，但最常见的途径包括一个人在婚后有了财务顿悟，而配偶却没有，或者有人结婚时认为他或她可以改变他们的配偶。两者都有问题，都需要困难的解决方案。

首先也是最重要的，在婚姻中任何艰难险阻的两个关键因素是沟通和妥协。你必须愿意并且能够谈论这个话题，并且不带愤怒或人身攻击。如果你不能在不诉诸尖叫和人身攻击的情况下就婚姻中的粗糙边缘进行文明的讨论，你需要寻求一位婚姻顾问，他可以帮助你达到一个点，在那里你可以用健康的婚姻所需的交流类型进行文明的对话。

一旦你能够坐下来以健康的方式讨论这个问题，你就要意识到这不是一个“对”或“错”的问题，而是不同价值观的问题。一个有经济头脑的人只是以不同的方式看待事物，但这并不是说两个人都是天生错误的。解决办法是**找到一个对你们双方都好的折中方案**。一个好的解决办法是建立一个预算，允许每个伴侣都有钱，他们可以自由地花在个人物品、礼物、爱好上——无论他或她希望什么——但钱有每月上限，这样仍然可以取得积极的财务进展。

可能破坏这一点的主要问题是**信任**。如果一方或另一方故意并反复违反你们两人达成的协议，那么这段关系中就有很深的信任问题，这可能是一些更深层次关系问题的迹象。再次，我建议在采取任何进一步措施之前先和婚姻顾问谈谈，但是伴侣之间缺乏信任是需要尽快解决的问题，以免它彻底侵蚀你们的关系。

### **经济上不负责任的父母**

有些人不幸发现自己处于父母在经济上不负责任的境地。在某些情况下，父母直接要求子女提供经济援助；在许多其他情况下，父母会超支，只是有一个心照不宣的假设，即如果最坏的情况发生，他们的孩子会照顾他们。

许多孩子这样做是出于对养育他们并(希望)保护他们度过童年的父母的感激之情。然而，这给孩子们带来了真正的经济负担，因为他们不得不处理父母的经济需求，同时还要维持自己的经济状况。

在这里，沟通是绝对重要的。如果你不沟通，双方将继续带着未言明的假设运作，这些假设最终会成为现实，导致一场*非常*令人讨厌的冲突，很容易破坏关系。如果你的父母当面告诉你，他们不指望以任何方式依赖你，那就坚持到底。

然而，即使有这种类型的交流，如果父母在经济上挣扎，许多孩子也会面临强烈的负罪感。在这种情况下，你会怎么做，他们的奋斗不仅仅是想象的未来，而是今天的现实？

首先，**寻找非财务方式的帮助。**帮他们做家务。帮助他们做预算。帮他们跑腿，购物。你甚至可以更进一步，通过同居来帮助他们。这些是你可以帮助解决问题的方法，而不是简单地砸钱。

如果你决定要帮忙，就要为此做好预算。仔细算出你能给他们多少钱，然后*为之做计划*。不要一时冲动打开钱包，除非这笔钱来自你预算中的弹性支出部分。如果你打算持续提供帮助，你需要从现在*开始*就做好计划。

我也**强烈反对贷款**，这一点在本文中还会多次出现。永远不要借钱给家人或朋友。如果你要打开钱包，把钱交出来，就当礼物，而不是贷款。在你的生活中扮演贷款人的角色会让你的关系变成贷款人和借款人的关系，没有人会对他们的银行家有好感。

要记住的最重要的事情是**你没有*有*帮忙。**有时候，人际关系会变得苛刻、控制欲强、消极，这些是你*在生活中永远不需要的东西，即使是你的父母。不要害怕走出消极的局面。*

### **经济上不负责任的成年子女**

如果是你的孩子在财务上不负责任怎么办？也许他们不断向你要钱，甚至定期向你要津贴。也许他们甚至住在家里，没有为你的家庭经济做出足够的贡献。你怎么处理这件事？

首先，重要的是要记住**他们还年轻，还有很多年的生活要过**。他们精力充沛，有很大的收入潜力。他们也有能力接受一份低工资的工作——他们不必一直坚持某种完美的工作。(我当然没有——我最初的工作之一就是铲土。)

换句话说，你可以把它们剪掉。年轻人有精力找到一种方法让他们生活中的事情运转起来。他们可以找到一两个入门级的工作。他们可以为自己找一套公寓。他们可以平衡自己的预算。如果需要，他们可以找到资源来帮助自己收支平衡。事实上，他们需要做这些事情，因为这是学习如何生活的一部分。

我建议**给你的孩子一个截止日期**。让他们知道财务变化即将到来，他们需要采取行动来应对这些变化。他们需要找份工作。他们需要找一间公寓。他们需要调整他们的预算，以适应没有存款的生活。

同时，尽可能多地提供非财务支持。问他们是否需要帮助，如果他们需要，就投入进去。如果他们需要帮助，就帮他们找个公寓。如果他们需要帮助，就帮他们找工作。帮他们搬出去。如果他们问你，尽可能多的提供建议，给他们一个敞开的大门。有些孩子会想要这个；别人不会。确保你有空就行了。

通常，孩子们需要最后的推动才能最终走出巢穴，找到自己承担财务责任的道路。

### **经济上不负责任的大家庭成员**

你生活中的叔叔、堂兄弟、成年兄弟姐妹和其他可能对你产生经济影响的人呢？当你的兄弟或侄女敲你的门，要求贷款或其他帮助时，你会怎么做？

首先，**不要借钱给家人。**不要。永远不会。避免它。你确实*不*希望与大家庭成员建立借贷关系。首先，你很有可能无法让他们还钱给你。另一方面，缺乏回报将导致家庭不和，这将在未来许多年引发问题。这是一个一遍又一遍发生的故事，永远不值得。*不要借钱给大家庭成员。*不要。

相反，公开提供非经济上的帮助。让那个人搭车去上班。给那个人一些建议。帮助那个人找到工作。邀请他们过来吃饭。给他们一个倾诉的耳朵和一个哭泣的肩膀。

如果大家庭成员做了鼓励超支的事情，比如保持赠送昂贵礼物的传统，或者建议一起进行昂贵的旅行，该怎么办？通常情况下，交流，但要在那些昂贵的场合之外进行。

例如，如果你的家庭有昂贵的寒假送礼传统，谈论这件事的正确时间是在春天或夏天，而不是在深秋或冬天。如果你在事件的边缘做了正确的事情，你可能会引起不好的感觉，因为人们已经开始为它做计划了。相反，远离任何这样的计划。同样，如果昂贵的旅行发生在夏天，那就在冬天谈。

当你谈论这样的事情时，建议一个合理的妥协。不要给每个人都买昂贵的礼物，而是画一张礼物图，或者给礼物的价格设定一个上限。与其昂贵的旅行，不如一起做一次更适度的旅行(例如，我是我们国家公园的超级粉丝，所以这是一个适度的假期，我*想*去)。不要只是说你不想继续这个传统，因为这看起来好像你在拒绝*他们*而不是拒绝昂贵的例行公事。

### **经济上不负责任的浪漫兴趣**

你正在和某人约会，你发现他们在消费上比你要宽松得多，或者在最近的过去是这样的。那个人花钱简直轻松得令人恐惧，尤其是当那个人的收入似乎无法支撑的时候。你注意到在他们的公寓里有许多来自大通银行或美国银行的信封。这些都是你会注意到的，当你越来越接近的时候。但是，除了经济上的担忧，这场比赛似乎很棒。你是做什么的？

我诚实的建议是要非常小心这段关系。我并不是说你应该马上分手，而是说你应该用批判的眼光看待整个关系。当你把你的财务生活结合在一起时(无论你是否保持账户独立)，对支出的截然不同的观点可能会成为婚姻中的一个主要问题，而且截然不同的财务责任水平会导致一些问题。

这也是一个好机会，让你开始学习如何就此类问题进行沟通。良好婚姻的关键是良好的沟通，很少有问题比金钱问题更依赖良好的沟通。如果你能就你们的支出差异进行健康的金钱讨论，并能提出一个对双方都有一些妥协的好策略，那么这对你们的长期关系来说是一个好兆头。如果这个对话很难或不可能，那么这是另一个强烈的负面信号。

一旦你做出妥协，你的伴侣会坚持吗？这是一个信任问题，因为你相信你的浪漫伴侣能够信守承诺。同样，如果你能谈出妥协，而你的伴侣却不坚持，那就有信任问题了。(如果你不坚持下去，还会有一个信任问题。)

### **经济上不负责任的朋友**

让我在这里直截了当地说:世界上有很多很多经济上有责任感的人可以和我交朋友，所以我真的不想和那些鼓励我超支的人维持友谊。句号。我只是不会努力去和那些维持友谊代价高昂的人维持友谊。

这并不意味着我没有品味昂贵的朋友。这只是意味着当我和那些朋友一起做事情时，没有任何花钱的期望，我们一起做的事情通常成本很低。

我的绝大多数密友只是简单地邀请对方过来参加社交活动。我们有晚餐聚会、游戏之夜、电影之夜和狂欢观看马拉松。当我们做其他事情时，我们通常会讨论一下，让两个最好的讨价还价者(我和团队中的另一个人)搜索折扣和优惠券，并计划出最便宜的方式。

如果你的朋友似乎有昂贵的品味，你会怎么做？首先，至少在某个时候建议一些不太贵的选择。很少有人会反对有时做一些不需要太多花费的事情。如果是这样，那么你和那个人之间就有很深的价值断层。偶尔和朋友做点贵的事情没问题，但不应该是常态。

你也不应该因为你的低级趣味而接受“朋友”的否定和批评。如果一个“朋友”嘲笑你的车，而你买这辆车是出于有意省钱的策略，那么你不仅看到了价值观的差异，还看到了朋友间善意的放弃。这是一种完全可以放弃的友谊。

另一个策略是为自己选择低成本的社交活动，并尝试在那里结识朋友。过去几年里，我的许多密友都是在社区游戏之夜和志愿者活动中认识的。几乎所有这些朋友都是非常节俭的人，我们的社交活动通常都很便宜。如果你遵循这一策略，你会发现你的社交日历会变得越来越充满廉价的活动。

### **财务上不负责任的同事**

在工作场所，你有时会发现社交压力，比如出去吃昂贵的午餐或晚餐，或者购买昂贵的东西，比如手表或小玩意。这种“融入”工作和建立牢固关系的压力会让你花掉很多原本不会花的钱。

不过，事实是。最持久的职场关系是建立在其他东西之上的，比如可靠、善良和健康的坦诚。他们不是因为花 50 美元吃一顿午餐而建立起来的。它们是通过成为一名优秀的同事、处理好你承诺要处理的事情、勇于面对挑战、不在背后捅刀子、积极参与工作场所的对话而建立起来的。

另一个策略是有意识地将你的午餐分散给许多用餐伙伴。过几天自己带午饭，和留在办公室吃午饭吃剩饭的人一起吃。去了解他们。出去吃饭，有时和昂贵的人一起，但有时和便宜的午餐人一起吃。这不仅能减少你的午餐开支，还能让你和很多人交流，或许还能结识你不认识的人。

简单地和昂贵的人群出去不会对保证你在工作中的地位有多大帮助。成为一个好同事将比其他任何事情都更能确保这一点。

### **最终想法**

这条线索中的策略都可以归结为几个关键原则。不要以个人名义借钱给人。寻找低成本的社交活动，珍惜与那些和你分享这些活动的人的关系。如果你想要更低成本的义务，要清楚地沟通(不要在这种情况下这样做)。不要害怕更新你的社交圈。和你爱的人交流，交流，沟通。

只有通过这些策略，你才能与生活中一些经济上不太负责的人保持健康的关系，而不会走上一条经济上不负责的道路。

祝你好运！
