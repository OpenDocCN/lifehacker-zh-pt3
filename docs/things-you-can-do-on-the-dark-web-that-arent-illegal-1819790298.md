# 你可以在黑暗网络上做的不违法的事情

> 原文：<https://lifehacker.com/things-you-can-do-on-the-dark-web-that-arent-illegal-1819790298>

[暗网](http://fieldguide.gizmodo.com/how-to-search-the-deep-web-safely-1770828854) 不仅仅是为了 [购买毒品和雇佣刺客](https://www.businessinsider.com.au/tor-silk-road-deep-web-2013-3#the-hidden-wiki-catalogs-several-tor-sites-that-would-otherwise-be-impossible-to-find-it-shows-you-a-number-of-sites-claiming-to-offer-things-for-sale-both-legal-and-illegal-1) 。这是一个存在于主流互联网文化之外的庞大的网站和社区网络，在不违反任何法律的情况下，黑暗网络上有很多事情可以做——从读书俱乐部到危机准备。



以下是一些最怪异、最具颠覆性的黑暗网站，它们不会让一队联邦特工破门而入。

### 什么是暗网，它与深网有什么不同？

在我们开始之前，有必要澄清一下。“深度网络”和“黑暗网络”这两个术语经常被混淆，但是它们之间的区别非常简单。

deep web 指的是你在搜索引擎中无法访问的任何东西，要么是因为它受密码保护，要么是因为它深埋在一个普通网站中。暗网是深度网的一个分支，你只能用一个像 Tor 这样的 [特殊浏览器来屏蔽你的 IP 地址。它包括像臭名昭著的丝绸之路这样的非法市场，以及许多其他不那么令人反感的网站。](https://lifehacker.com/what-is-tor-and-should-i-use-it-1527891029)

下面是黑暗网络的一些更健康的产品——尽管你需要下载 Tor 来实际访问它们。

### 加入读书俱乐部

阅读社区多年来一直是黑暗网络的一部分。甚至《丝绸之路》的创作者罗斯·乌布里希特 [也有一个隐藏的网页](https://www.dailydot.com/news/silk-road-dread-pirate-roberts-book-club/) ，人们在那里讨论文学经典，如拉尔夫·埃利森的*《隐形人》*以及更有争议的材料，如 [*无政府主义食谱*](https://lifehacker.com/top-10-diy-hacks-that-could-poke-your-eye-out-5752664) *。*

乌尔布里切特的在线读书俱乐部已经停止运营，但是黑暗网络上还有很多其他的选择，包括像 [Jotunbane 的读书俱乐部](http://c3jemx2ube5v5zpg.onion/) 和 [特兰托尔的帝国图书馆](https://xfmro77i3lixucja.onion.to) 这样的网站。这些网站很大一部分是下载非法的流行书籍，但你也可以找到活跃的讨论和一些有用的无版权的阅读材料。

如果你正在寻找一些不那么知性的东西， [主板](https://motherboard.vice.com/en_us/article/vvbbdb/the-legal-side-of-the-deep-web-is-wonderfully-bizarre) 也推荐“ [关于故事的博客](http://7hk64iz2vn2ewi7h.onion) ”，一个人们分享真实和虚构的打屁股故事的黑暗网站。

### 为世界末日做准备

如果你在黑暗的网络上寻找更具体的信息，查看一下 [战略情报网](http://4iahqcjrtmxwofr6.onion/) 。它包含了如何应对地球上任何地方的任何危机的信息，从自然灾害到暴乱或全面战争。

如果你还不满意，去看看 [隐藏答案](http://answerstedhctbek.onion/) ，根据 [MakeUseOf](http://www.makeuseof.com/tag/little-known-corners-deep-web-might-actually-like/) 的说法，基本上黑暗网络版本的雅虎答案的主题包括政府、法律和金融服务，以及毒品和色情。黑暗网络甚至有自己的新闻渠道，手电筒，专注于比特币、在线隐私和其他类似的主题——你可以在这里 实际查看手电筒，而无需专门的浏览器 [。](https://kxojy6ygju4h6lwn.onion.link/) 

### 社交化

黑暗网络很像我们熟悉和喜爱的常规互联网——只是无政府主义者的比例更高。它有自己的电子邮件服务、社交网络甚至在线游戏。

如果你正在寻找一个超级安全的电子邮件提供商，看看[anonybox](http://ncikv3i4qfzwy2qy.onion)，它每年收取一小部分比特币的费用，并承诺完全保密。据 [黑暗网络新闻](https://darkwebnews.com/news/the-deep-web-and-its-weird-but-legal-hangouts/) 报道，还有一个拥有数十万用户的社交网络“创世纪行动”。最后，前往 [选择](http://theches3nacocgsc.onion/) 在经典棋盘游戏中与来自世界各地的玩家对决。

在黑暗的网络上还有很多事情可以做，但是这些网站应该可以帮助你领略互联网黑暗面的光明面——而不会在这个过程中冒坐牢的风险。
