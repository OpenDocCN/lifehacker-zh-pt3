# 挑战生日锻炼的乐趣

> 原文：<https://lifehacker.com/the-joy-of-doing-challenging-birthday-workouts-1806114449>

上周，我以最快的速度做了 31 个引体向上、31 个空中深蹲、31 个俯卧撑，以及每条腿 31 个反弓步(几天后还骑了 31 英里的自行车)。我手臂上的灼烧感让我严肃地质疑自己的人生决定，但我仍然热爱其中的每一分钟。

在过去的几年里，我每年生日那天都做一些不同寻常的 [锻炼](http://thefyslife.com/article/birthday-workout-challenge/) 。去年，我在单杠上做了一组 30 个相当于我体重的深蹲。这是最可怕的方式。前年我只游了 29 圈。我把今年可怕的 20 分钟记录在了下面这个滑稽的延时视频中。

这些训练是我纪念又一年过去的的方式，同时帮助我摆脱不可避免地会发现自己陷入的巡航控制。他们提醒我，当其他一切都很轻松的时候，感到身体不舒服是件好事(这可能很有趣)...可能)。

尽管过去的锻炼很艰难，但我的生日锻炼在海蒂·多罗 40 岁生日锻炼时什么都没有。作为一名纽约的慈善家，多罗有一个疯狂的想法:在连续 40 个小时的时间里，她每小时都要跑过轮胎，用手和脚爬行，用绳子拉重物，爬上爬下梯子，并穿过她在纽约市贾德森教堂内建造的室内两英里障碍跑道。

她成功了，尽管她没能全身而退。但是不仅仅是为了纪念她的生日。现年 52 岁的多罗告诉我，她一直着迷于超耐力运动员在极端条件下的表现。(其中一些“超级马拉松”如可以长达 135 英里。)

所以对她来说，这种锻炼类似于痛苦。但也许最重要的是，她想确保随着年龄的增长，她仍然感觉和看起来很强壮。“年龄滋生自满和对更多舒适的渴望。多洛说:“我们把体育锻炼作为对抗衰老界限的一种方式。

### **开始你自己的挑战训练**

你也可以在生日那天进行一项富有挑战性的锻炼，但是不一定要在你生日的那天进行。

而“挑战”对每个人来说看起来都很不一样。例如,《大西洋月刊》曾报道过人们如何愉快地花钱去参加危险、、[艰苦的军事式障碍赛、](https://www.theatlantic.com/health/archive/2017/03/breaking-the-ennui-of-office-work-through-painful-exercise/520746/)、以逃避办公室的无聊。成千上万的人报名参加世界各地的长跑比赛。仅在 2016 年纽约市马拉松比赛中，就有超过 50，000 名跑步者参加。我也参加过马拉松。一次。然后是乔尔·鲁尼恩(Joel run yon)，他已经在七大洲完成了七次超级马拉松，他说:“我喜欢超级马拉松的部分原因是他们超越了大多数人认为可能的事情。”

重点是？做点什么，尤其是当你发现自己墨守成规的时候。

你的挑战不必如此疯狂，你当然也不需要跑超级马拉松( [，也许从 5 公里开始！](http://vitals.lifehacker.com/how-to-prepare-for-your-first-race-whether-it-s-a-5k-o-1735174408#_ga=2.127243853.1798624850.1505429005-1294460130.1504745335) )。如果你想做一些更短、不需要太多准备时间的事情，这里有一些想法(其中 X 可能是你的年龄):

*   冲刺 100 码 X 次
*   在 X 分钟内完成传统的长跑或锻炼
*   X 秒/分钟内 X 次体重下蹲
*   x burpee 的数量
*   试图在旋转课上保持疯狂的旋转速度 X 秒或分钟
*   用绳子爬 X 英尺
*   扛 X 磅走 X 米 X 次
*   做连续 X 分钟的弓箭步

推动自己完成任何持续时间的挑战可能同时是可怕的和令人敬畏的:实际的锻炼或活动本身*suuuucks*，但之后的感觉让你忘记了地狱般的分分秒秒(事后来看)。Runyon 还提出了一个很好的观点，在处理了像超级马拉松或另一种超级挑战性锻炼的斗争和痛苦之后，你以前看起来如此令人生畏的日常问题似乎变得更容易管理了...甚至可能微不足道。

然而，无论你做什么，都要确保达到那个健康水平。你的锻炼应该具有足够的挑战性，让你感到不安，但不要太极端，远离你的能力，以至于最终伤害到你自己。但是，嘿，我不是你妈妈。
