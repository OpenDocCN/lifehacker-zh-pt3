# 本影桌面

> 原文：<https://lifehacker.com/the-umbral-desktop-1744516553>

佐戈登 的 Windows 10 桌面超级干净简洁，但无论他没有打开任何东西，还是只打开了几个窗口，都有足够的有用信息。他是这样设置的。

Watch

一个隐藏的任务栏，一点清理，几个简单的小部件和一个漂亮的，细致入微的壁纸，这就是让这个桌面成为你的桌面所需要的一切。如果你喜欢这个造型，下面是让它成为你自己的所需:

*   [色调](http://slurpaza.deviantart.com/art/Tones-370950717) 壁纸由[Slurpasa at DeivantArt](http://slurpaza.deviantart.com/)
*   用于 Windows 的 [雨量计](http://rainmeter.net/) 系统调试和监控工具
*   [雨量计平坦模糊的皮肤](http://satyajit00.deviantart.com/art/Flat-and-Blurry-v1-5-524848816) 在屏幕右侧创建系统监控、天气、电池、当前播放等实用小工具
*   [材质垃圾桶皮肤为雨量计](http://raion1337.deviantart.com/art/Material-Trash-Bin-568530258) 为自定义垃圾桶
*   一个 [自定义雨量计皮肤由佐戈东](https://drive.google.com/folderview?id=0B1XeFHC1KzhoSk8yLUdjbzZaMW8&usp=sharing)的为睡眠按钮
*   鼠标光标的 [晶莹剔透的光标皮肤](http://theblueguy07.deviantart.com/art/Crystal-Clear-v3-1-298678459) (截图中未显示)

差不多就是这样。好看又简单，整个包装看起来很棒。你可以用上面的墙纸来做这件事，或者用你自己的——或者你可以只拿墙纸和你喜欢的任何一种，剩下的就不管了，由你决定。如果你喜欢整件事的外观，一定要去 Zorgodon 的 Flickr 页面(链接如下)为你唱赞歌。
