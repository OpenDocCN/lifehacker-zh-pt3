# 了解从传送带上的玩具野马中牵引的危险

> 原文:[https://jalopnik . com/learn-on-1787401437](https://jalopnik.com/learn-about-the-dangers-of-towing-from-a-toy-mustang-on-1787401437)

如果你正在考虑做一些拖车，你也认为所有关于拖车重量分布的关注都是重量分布游说团体制造的一堆废话，那么我真的建议你看看这个小视频。因为玩具有趣的地方在现实中是可怕的。

 [https://jalopnik.com/embed/inset/iframe?id=youtube-video-4jk9H5AB4lM&start=0](https://jalopnik.com/embed/inset/iframe?id=youtube-video-4jk9H5AB4lM&start=0) 

这个看似来自 [安大略省警察商用车辆委员会](https://twitter.com/opcvc) 的演示，是如此奇妙的简单有效。小野马(大多数辨别塔选择的拖车)拉着一辆有两套重物的拖车:一套在前面，一套在后面。

Watch

当更多的重量在前面时，事情保持相当稳定。即使上帝之手在拖车后面推了一下，也不能真正让拖车停下来。

但是一旦更多的重量被放在后面，这个小小的侧向推力就会引发各种钟摆式的摆动和麻烦；令人惊讶的是，这个系统变得不可控制的速度是如此之快。

这甚至可能比全面的真实世界情况更糟糕，因为传送带以稳定、恒定的速度前进，汽车的前轮保持笔直。在现实中，一旦摇摆开始，野马司机可能会在恐慌中交替转动方向盘，同时踩刹车，在这两个动作之间花一些时间来浪费他们的裤子。

所以，从传送带上的一些玩具来看:小心你如何分配你所牵引的东西的重量！

此外，它可能是好的，没有一堆乐高迷你无花果在传送带上，或者玩具野马会试图犁下来。
