# 是时候永久埋葬葬礼自拍了

> 原文：<https://lifehacker.com/its-time-to-bury-funeral-selfies-for-good-1822542273>

葬礼是一个记住所爱的人并衷心道别的时刻，而不是吸引别人的注意力。然而，有些人确实是这样做的。仅仅因为你盛装打扮并不意味着一张[#葬礼自拍](https://www.instagram.com/explore/tags/funeralselfie/) 是正当的。



来自魁北克的著名葬礼导演丹尼斯·德斯罗彻斯在加拿大广播电台的 *Gravel le Matin* 上谈论了葬礼自拍的问题。德斯罗彻斯解释说，他和他的业内同事 [很难教育人们正确的葬礼行为](http://www.telegraph.co.uk/news/2017/02/08/stop-taking-selfies-funerals-say-funeral-directors/) ，并提到葬礼自拍是糟糕礼仪的主要例子之一。他主要担心的是那些家庭，他们通常不想在社交媒体上看到这种类型的照片，因为它们不断提醒着他们的损失。他们也有点不尊重人，因为这种类型的照片通常品味很差。如果你不相信我，可以看看现在已经废弃的 [葬礼自拍 Tumblr](http://selfiesatfunerals.tumblr.com/) 。

当 CNN 的 Melonyce McAfee 与艾米丽波斯特研究所的礼仪专家 Lizzie Post 聊起这个问题时，她惊讶地发现人们把哀悼的时间变成了拍照的时间 。波斯特解释说，大多数人都不想在那些日子里拍照，不管是在集体场合还是其他场合，自拍肯定是不合适的:

> “我觉得你做出可爱的表情和鸭子的嘴唇并把它标为#葬礼是不合适的。”在去葬礼的路上自拍“这是我穿的，我看起来很棒吧？“你知道这不是关于你的，这是关于死者的，所以你能暂时放弃自拍吗？”

波斯特认为，在这些情况下，许多犯罪者都是年轻人——有些人以前从未参加过葬礼——这也许可以解释为什么它会发生在一开始。但由于相机技术变得越来越移动，社交媒体继续主导许多人的生活，这个问题应该得到解决。每个人，无论老少，都需要知道拍一张“#悲伤”的葬礼自拍是彻头彻尾的自私。

正如 [马克·陶伯特](https://twitter.com/drmarktaubert) 博士，一位专门研究悲伤和社交媒体的姑息治疗医生， [向科技内幕](http://www.businessinsider.com/are-funeral-selfies-bad-2016-5) 解释的那样，葬礼自拍给自己带来关注，并把它从死者身上带走。葬礼的意义在于，它关乎死者的生活，而不是像陶伯特所说的那样，关乎你的“数字更新”。在我们这个后网络时代，葬礼自拍可能只是现代悲伤过程的一部分，但陶伯特建议有一种更有品位的方式来数字化悲伤。不要分享你在葬礼上皱眉头的自拍照，而是发送一条表达你对死者的爱或安慰他人的短信。
