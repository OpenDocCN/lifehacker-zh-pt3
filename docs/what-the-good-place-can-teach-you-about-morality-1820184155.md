# 《好地方》能教会你什么是道德

> 原文：<https://lifehacker.com/what-the-good-place-can-teach-you-about-morality-1820184155>

尽管它有古怪的幽默和过度照明的场景，但《好地方》通过教授一些非常真实的哲学，设法解决了生活必须提供的一些最大的道德困境。看着看着，我不禁觉得这个节目是一个优秀的，即使是基础的，道德哲学导论课。这里有几个你将从这个节目中学到的重要概念的例子。

### **成为“好”是你必须学习和实践的东西**

“好”是什么意思？是不是有些人天生就是坏种子？我要破坏第一集的前提，不过没关系，网飞的描述也是一样。基本上，埃莉诺(基丝汀·贝尔饰)死后会在来世醒来。她被告知，她过着道德上健全的生活，她是为了“好地方”而努力的。但有一个问题:埃莉诺被误认为是别人，不属于那里。她在生活中是一个可怕的人，属于所谓的“坏地方”她试图逃避，但很快意识到她需要想出一个计划，否则每个人都会知道她是一个骗子。

幸运的是，她在指定的灵魂伴侣希迪那里找到了帮助，他是生活中的道德伦理教授。通过一些私人辅导，埃莉诺开始了解每个人都应该知道的事情:成为“好”是你做出的选择。正如亚里士多德所说，你的性格是自愿的，所以没有人生来就是“好”或“坏”你必须决定你想成为哪一个。希迪解释说，不仅有可能改变一个人自私和不为他人着想的本性，还有可能改变一个人的生活方式，把他人放在自己的前面。当然，改变不可能在一夜之间发生，但是学会变得“好”是可能的——只要你练习。像其他习惯一样，善待他人是一种习惯，而实践有助于你追求完美(尽管你永远也不会达到完美)。

### **做“好事”并不一定会让你成为一个好人**

在*好地方*中，你被送到的来世版本是基于一个复杂的积分系统。做“好”的事情会让你获得一定数量的积极积分，做“坏”的事情会减去这些积分。你死时的总点数决定了你的去向。似乎很公平，对吧？

尽管《好地方》让生活感觉像一个基于分数的电子游戏，但我们很快发现道德并不像正面和负面那样非黑即白。有一次，埃莉诺试图通过为人们开门来增加分数；一次 3 分的动作。客观地说，她的分数是-4008，她需要达到 1222821 的平均分。她要花很长时间才能到达那里，但这是一种方法。至少，如果成功的话会是这样。过了一会儿，她很快意识到她没有得到任何分数，因为她实际上并没有试图对人友好。她唯一的目标是积累分数，这样她就可以留在好的地方，这是一个天生自私的原因。这种情况提出了一个有效的问题:出于自私的原因而做的“好”事情还是“好”事情吗？

我不想破坏太多，但随着系列的进行，我们看到这个问题被一次又一次地问及其每个角色。希迪可能一生都在研究道德伦理，但知道追求“善”意味着你就是“善”吗？塔哈尼一生都是慈善慈善家，但她所做的一切都是为了最终超越她近乎完美的姐姐这一令人质疑的追求。她做了很多好事，但是她“好”吗这是你每天都要考虑的事情。试着做“好”的事情，但是偶尔问问自己这些“好”的事情到底是为了谁。

### **道德是暧昧的**

现在，你可能已经注意到，我一直把“好”和“坏”放在引号中(这可能会让你发疯)。那是因为“好”和“坏”是主观的，使得它们几乎不可能被定义。许多你可能认为“好”的东西，对其他有不同观点的人来说，实际上可能被认为是“坏”的。*好地方*总是处理那条线，这是一个需要解决的重要问题，尤其是对于那些从小被告知对错有明确版本的人来说。当你看的时候，你慢慢开始理解道德是一个集体的社会决定，它可以根据你所处的社交圈而变化。

正如希迪经常解释的那样，大多数伦理道德都陷入了“灰色地带”，在那里，意图、场景和其他变量可以改变人们对事物的感受。有一次，希迪被迫尝试著名的“电车思维实验”，他必须决定是否杀死一个人来救五个人。常见的“更大的善”答案是改变轨道，让只有一个人死，五个人活。但是现实世界总是有变数的。如果那个人是个孩子呢？那五个人打扮成纳粹怎么办？如果你认识其中一个人呢？或者，如果你坐以待毙呢？做旁观者“好”吗？一旦实验成为现实，希迪，一个花了一生研究道德伦理的人，崩溃了。思想实验没有道德上合理的答案。事实上，现实世界中很少有道德上合理的答案。

我肯定你们中的一些人在想，“不，有些事情就是不对。杀人是不对的！”但如果你是，你就没抓住重点。像《好地方》这样讨论道德并不是像我们在法律和宗教中看到的那样重新定义基本的道德倾向——而是学习如何从外部看待它们。通过巧妙的笑话和愚蠢的情节转折，该节目让你更好地理解道德的真正模糊性，同时也巧妙地教你看看*为什么*事情是这样的。
