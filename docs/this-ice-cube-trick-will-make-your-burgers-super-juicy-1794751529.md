# 这个艾斯·库伯的小把戏会让你的汉堡超级多汁

> 原文：<https://lifehacker.com/this-ice-cube-trick-will-make-your-burgers-super-juicy-1794751529>

入口即化的多汁汉堡是生活中最大的乐趣之一。如果你的馅饼经常变得又硬又干，这个聪明的方法可能会解决这个问题。



据主厨*评委 [格雷厄姆·艾略特](https://www.grahamelliot.com/) 称，超级多汁汉堡的秘诀就是一个普通的冰块。何 [向解释了一下为什么前阵子回](http://www.foxnews.com/food-drink/2015/06/05/ice-cube-for-better-burger-graham-elliots-bizarre-grilling-trick.html) :*

> *“做你的馅饼，然后把你的小冰块放在里面，然后当你烤它的时候，它会保持湿润，防止它变干。”*

*只要把它折叠到每个肉饼的中心，就可以吃了。烹饪时，冰块融化，将所需的水分分散到整个馅饼中。如果我是鲍勃·贝尔彻，我会叫它“冰到肉的汉堡”*

*埃利奥特也提醒你汉堡肉的过度调味。他建议跳过胡椒、洋葱粉等等，让多汁的肉成为表演的明星。只要确保你得到一些像样的碎牛肉，这样味道才会真正流行。你可以在 这里 [找到他的个人汉堡食谱。](https://www.tastingtable.com/entry_detail/chefs_recipes/19301/topic_coke/Graham_Elliots_update_on_the_classic_American_burger.htm)*
