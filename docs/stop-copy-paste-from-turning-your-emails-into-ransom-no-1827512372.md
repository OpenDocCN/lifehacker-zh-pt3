# 不要把你的邮件复制粘贴成勒索信

> 原文：<https://lifehacker.com/stop-copy-paste-from-turning-your-emails-into-ransom-no-1827512372>

你正在键入一封电子邮件，你从 Word 文档中粘贴了一些文本，突然你的电子邮件的一半与文档的字体相同。或者你把一个标题粘贴到谷歌文档中，它会以 48 磅的巨大文本显示出来。这种恼人的粘贴行为是大多数 Windows、MacOS 和 web 应用程序的默认设置。在发出的文件和电子邮件中，这会让你看起来很邋遢。以下是修复方法。



## 方法 1:添加一个班次

 [https://lifehacker.com/embed/inset/iframe?id=twitter-580603279878537216&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-580603279878537216&autosize=1) 

在某些应用程序上，当你按下 Ctrl-V (Win)/Cmd-V (Mac)时，你也可以按下 Shift。这将激活“粘贴并匹配样式”，将文本转换为与目的文稿相同的样式。这通常是你想要的。

在 Mac 电脑上，有些应用程序让你按下 Option 键以及 Shift 和 Cmd 键。猜测每个 app 用的是哪个，很烦。但是你可以用我们的下一个方法解决这个问题。

## 方法 2:制作键盘快捷键

在 **MacOS** 上，你可以轻松的 [给任何菜单项](https://lifehacker.com/how-to-remap-any-keyboard-shortcut-in-mac-os-x-5720087) 分配键盘快捷键。你可以将 Cmd-Shift-V 指定为“粘贴和匹配样式”,这样它在每个应用程序中都可以正常工作。

或者您可以颠倒默认行为:将 Cmd-V 分配给*粘贴和匹配样式*，将 Cmd-Shift-V 分配给*粘贴*。开发人员 Craig Hockenberry [为这种特殊情况](https://furbo.org/2015/03/30/paste-without-style/) 写了一份指南，包括一些细节以确保一切完全正确。

要在**窗口**、、[上分配键盘快捷键，你需要一个第三方应用](https://fieldguide.gizmodo.com/create-your-own-keyboard-shortcuts-to-do-anything-on-wi-1821529700)、。Gizmodo 推荐免费应用 [WinHotKey](https://directedge.us/content/winhotkey) 和 [AutoHotkey](https://www.autohotkey.com/download/) 。

Windows 上又多了一个选择:下载 [PureText](https://lifehacker.com/puretext-strips-formatting-from-your-clipboard-when-you-5043566) ，这是一个免费的去除格式的应用程序。粘贴专用 app 看似很多，但不比下载键盘快捷键 app 差。

## 但是请记住

有些时候，您确实想带着格式粘贴。最常见的是在粘贴链接文本时。假设你想将前一段粘贴到一封电子邮件中，而链接保持不变。在一些应用程序中，使用*粘贴和匹配样式*会去除文本。

你有两个选择，都很糟糕。您可以“粘贴和匹配样式”，但您需要手动添加链接。或者你可以粘贴，你需要手动改变字体和大小的文字。

在 Gmail 上，选择所有粘贴的文本，打开*发送*按钮旁边的*格式选项*菜单。选择与邮件其余部分相匹配的字体和字号。(如果你不确定，在选择字体和大小之前，先选择整封邮件。)如果适用，也更改文本颜色和粗体/斜体/下划线选项。

下载应用程序并分配键盘快捷键后，您已经成功地将一行文本从电脑的一部分移到了另一部分。恭喜你！电脑让生活变得简单！
