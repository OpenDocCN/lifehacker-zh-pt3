# 使用“环形理论”来避免在危机中说错话

> 原文：<https://lifehacker.com/use-the-ring-theory-to-keep-from-saying-the-wrong-thi-1798540880>

飓风哈维继续袭击德克萨斯州，摧毁房屋，摧毁生命。不管你是不是德州人，你可能都知道有人受到了影响，向你的朋友和熟人伸出援手的冲动是强烈的。你应该这么做！

然而，在你这样做之前，花一分钟想想你想说什么。你想告诉他们这对你来说有多难吗？你想告诉他们你对他们的家园被毁有多难过吗？或者这是如何唤起几年前影响你的飓风的记忆？换句话说，你要让他们的创伤都与你有关吗？

你需要学习环理论。

戒指理论是由临床心理学家苏珊·西尔克提出的，她(和巴里·戈德曼一起)在 2013 年为《洛杉矶时报》 撰写了相关文章。这个想法是，在任何危机中，都有一个核心圈子的人直接受到影响；同心圆从人们的中心向外延伸，每个代表另一个受危机影响较小的群体。苏珊和巴里将对此进行更详细的解释:

> 画一个圆。这是中心环。在里面，把那个人的名字放在当前创伤的中心[...]在那个环上写下最接近创伤的人的名字[...]根据需要多次重复这个过程。在每个较大的环中放入下一个最亲近的人。父母和孩子在更远的亲戚之前。小戒指里是亲密的朋友，大戒指里是不太亲密的朋友。当你完成后，你有一个 Kvetching 订单。

现在你有了自己的圈子，这里有最重要的事情要记住:规则是“舒适进来，倾倒出去。”你可以尽情抱怨这场危机——但你只能向你之外的人抱怨。对比你更接近中心的人，你应该只提供安慰。就是这样。

你为什么要遵循指环理论？很明显，你可以做你想做的事情，但是考虑一下:当有人经历危机时，他们不需要处理你的感受。你的感觉没有错，你应该觉得你可以和别人分享。但是有了指环理论，你就可以让那些已经承受了太多悲伤的人解脱出来。折磨不那么痛苦的人。

这是否意味着站在擂台中央的人有权嚎啕大哭，咬牙切齿，向任何愿意倾听的人诉说自己的一切？为什么是的，是的。因为*是*关于他们的一切。这才是重点。
