# 这些瑜伽动作实际上会缓解背痛

> 原文：<https://lifehacker.com/these-yoga-moves-will-actually-relieve-back-pain-1796335626>

如果你背痛，瑜伽可能是你最不想做的事情。但是在 12 周温和的初级瑜伽课程后，在最近的一项研究中，人们的疼痛缓解程度与那些进行物理治疗的人一样。这两种锻炼方式都比什么都不做要好。



如果你想亲自尝试这些动作， [这是他们发给参与者的指南](https://static-content.springer.com/esm/art%3A10.1186%2F1745-6215-15-67/MediaObjects/13063_2013_1959_MOESM1_ESM.pdf) 。这些体式从非常基本的开始，不需要任何特别的灵活性或力量。在许多情况下，你靠在墙上保持稳定，或者做一个简化的“婴儿”版本的 lululemon 模型所做的事情。参与者也不是有经验的瑜伽练习者:他们是居住在波士顿地区的 320 名背痛患者，种族多样，大部分是低收入人群。他们没有背部受伤，只是有一种医生无法解释的疼痛， [可能与缺乏力量和灵活性](http://www.physio-pedia.com/Non_Specific_Low_Back_Pain) 有关。

他们每周去上一个小时的瑜伽课，每天没课的时候被安排自己练习 30 分钟。指南上说明了如何在家练习，但如果你想了解更多细节，可以查阅一下 [教师手册](https://static-content.springer.com/esm/art%3A10.1186%2F1745-6215-15-67/MediaObjects/13063_2013_1959_MOESM2_ESM.pdf) 。在瑜伽项目之前，70%的参与者使用止痛药。后来，只有 50%的人。(没有进行瑜伽或物理治疗项目的人，他们的止痛药没有变化。)这个瑜伽项目不会创造奇迹，但看起来确实有帮助。

我尝试了这套动作(我直接跳到更高级的动作，因为我以前做过瑜伽)，它们简单、温和，完全没有威胁。如果你背痛，就去检查一下，但是知道瑜伽可能会有所帮助还是很好的。
