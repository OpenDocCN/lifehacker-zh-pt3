# 邮购基因测试可能会出现很多假阳性

> 原文：<https://lifehacker.com/mail-order-genetic-tests-may-turn-up-a-lot-of-false-pos-1824126192>

如果你在开玩笑的时候做了一个基因测试，它告诉你你有一个变种，使你有患癌症的风险，你应该直接去找专业人士做一个更昂贵的测试来确认。但是在最近的一项研究中，那个最初的、只是为了好玩的测试有 40%的时间被证明是错误的。



这项研究规模很小，只有 49 个人，所以我们不知道直接面向消费者的测试是否总是有 40%的假阳性率；总的来说，可能会更好，也可能会更差。同样值得注意的是，这项研究中的人是那些结果令人担忧的人，他们寻找专业人士进行临床测试。

这项研究是在做临床测试的遗传学实验室进行的。从某种意义上说，这使得他们成为他们所批评的测试的竞争对手。但是像 23andme 这样的公司已经在免责声明中指出，他们的测试不应该是你医疗建议的唯一来源，所以临床测试通常是使用这些服务的必要后续措施。

### 如何偷看你的原始数据

当你从 23andme 或 Ancestry 这样的公司订购基因测试时，该公司通常不会告诉你它找到的所有信息。Ancestry 根本不报告健康结果，23andme 只报告 [某些](https://lifehacker.com/23andme-now-provides-risk-information-for-10-genetic-di-1794085470) FDA 授权的 [检测](https://lifehacker.com/what-you-can-learn-about-your-breast-cancer-risk-from-2-1823547638) 的严重健康状况。

但是如果你很好奇，你可以下载一个“原始数据”文件，其中包含该公司测试的所有SNP。这些是你的 DNA 中个人倾向于彼此不同的地方。其中一些变异与癌症易感性或其他健康状况有关。你可以将原始数据文件发送到几十个第三方分析网站中的任何一个，它们会告诉你那里有什么。

但是这些结果——尤其是来自第三方的结果——不像临床遗传学测试那样严格。如果你得到的结果可能会改变你的医疗保健或刺激你采取激烈的行动，如手术，你应该通过更彻底的测试来确认。

Ambry 是进行这些更严格测试的公司之一，它检查了他们的记录，发现有 49 人要求进行临床测试，因为一项直接面向消费者的测试告诉他们，他们有一种有害的变异(如对癌症的易感性)。根据他们发表在*【医学遗传学】* 的 [论文，他们只能证实 60%的初步发现。](https://www.nature.com/articles/gim201838)

### DTC 和临床试验的区别

直接面向消费者(DTC)测试包括任何你可以将唾液发送到公司并获得结果的测试，而无需与医生或其他医疗专业人员交谈。另一方面，临床测试是医生指定的，测试实验室遵循更严格的协议。

“没有一种测试每次都是准确的，”遗传顾问兼全国遗传顾问协会主席埃里卡·拉莫斯说。她为 Illumina 工作，这是一家为临床测试提供测序技术的公司，但它也部分拥有 DTC 测试服务 Helix。不同类型的测试以不同的方式处理那些潜在的错误。

大多数 DTC 服务使用的策略是测试你的大量 DNA，非常便宜:多达一百万个不同的点。23andme 表示，如果它在报告中正式描述了一种情况，那就是已经过“质量控制检查和验证”的结果，并且为了获得 FDA 对他们一些测试的授权，他们必须证明他们的技术“超过 99%”准确。也就是说，即使很小的错误率也可能导致几个 SNP 出现错误。

但是 23andme 不能保证，尤其是关于你的原始数据中的其他结果。事实上，当你下载数据时，它们包括一个免责声明，声明它的准确性未经检查，并且“仅适用于研究、教育和信息用途，不适用于医疗、诊断或其他用途。”

(最近的研究没有点名任何公司，所以我们不能确定任何原始数据都来自 23andme。然而，对于这种类型的测试，它们是非常受欢迎的服务。)

临床测试更彻底。实验室通常会读取你询问的单个基因或几个基因的完整序列，而不是查看基因组中的 SNP。他们还做额外的测试，以确定你的 DNA 的相关部分是否被删除或复制。如果最初的结果看起来不确定，实验室会再次检查他们的工作。

尽管付出了额外的努力，这些测试可能会很昂贵。Ambry 表示，他们的价目表上，现金支付者的价格在 200 美元到 1500 美元之间，但有保险的人的平均费用“不到 100 美元”

### 解释事项

数据错误只能解释橱柜研究中的一些假阳性。另一些来自不同的解释:如果你在 DNA 的某个位置有某种变异，这对你的健康意味着什么？

这是一个棘手的问题，因为对一些 SNP 可能存在分歧。研究中的一些假阳性发生是因为 DTC 服务将结果列为增加一个人患癌症或疾病的风险，但 Ambry 认为这种变异是无害的。你可以在这里 看到这些变体的表格，以及它们如何被不同的公共数据库分类。

如果你在家里玩，考虑对你的 DTC 或第三方报告中的变体做一些研究。该分析的第一作者朱舒·康纳说 [ClinVar](https://www.ncbi.nlm.nih.gov/clinvar/) 是检验原始数据中的变体的好地方。它列出了来自不同实验室的解释，即使他们不同意。(实验室自己使用这个数据库来查看其他人是如何解释一个给定的变体的，并试图找出他们真正应该告诉患者的内容。)她还建议[LOVD](http://www.lovd.nl/3.0/home)。

### 当邮寄来的基因测试结果令人害怕时该怎么办

第一，不要慌。无论结果是阳性还是阴性，邮寄测试都不会告诉你遗传的全部情况，更不会告诉你疾病风险的全貌。记住，你的家族史和其他风险因素也应该是讨论的一部分。

如果 DTC 测试或第三方分析显示您患癌症等疾病的风险增加，您可以向您的医生提出这些担忧。也就是说，并不是每个医生都明白 DTC 和临床测试之间的区别。(康纳说他们应该这样做，但她遇到过一些不这样做的人。)

另一个获得建议的途径是直接与遗传顾问交谈。你可以搜索寻找你附近的人，或者可以通过电话联系到的人。选择“家庭/直接面向消费者的 DNA 测试”,寻找解释和处理这类结果的专家。他们可以帮助你确定你是否需要更多的测试，以及如何得到它。
