# 如果你模仿凯莉安·康威，却被别人听到说你同事的坏话，该怎么办

所以上周四，总统顾问凯莉安·康威被人无意中听到嘲笑她在川普白宫的同事 。“不许泄密，你们这些家伙！”她模仿白宫办公厅主任雷恩斯·普利巴斯(显然完全无效)试图阻止本届政府成员向媒体不断泄露信息的做法说道。她的模仿被另一个参加聚会的人无意中听到，他把这个故事发布到推特账户[@ kellyanneliaks](https://twitter.com/KellyanneLeaks)。

她在乎吗？大概不会。肖恩·斯派塞说她实际上是在取笑媒体，说他们关于白宫存在紧张局势的说法是完全错误的。

所以这里真的没有损害控制可做，因为显然它没有发生。但是让我们假设*你是一个普通的上班族，可能决定开个玩笑，说会计部的玛丽有多讨厌来发泄一下，也许你甚至会模仿玛丽的长相、声音、穿着等等。你被偷听了。玛丽被激怒和羞辱；你希望你喝了两杯就停下来了，或者有人在你发飙之前把你的头塞进了潘趣酒碗里。你*做什么*？*

总部位于华盛顿的公共关系和公共事务公司的副总裁安德鲁·里奇说，有几个非常明确的第一步，而第一步是最重要的。

### 掌控局面

“如果你(和你的同事)消除误会，就会消除很多紧张情绪，”Ricci 说，“解决问题的速度也会更快。也许有挫折。从一开始就解决这个问题。这会带来一种更好的尊重和共事文化，从长远来看，会让你更好地与他人共事。”

### 要耐心

在职场中，建立良好的声誉需要几年，甚至几十年的时间，而在一个热麦克风上的一句愚蠢的评论就可能毁掉它。“要意识到，仅仅因为你今天表现得很好，并不是每个人明天都会喜欢你，”瑞奇说。当我问一个人可以做些什么来启动原谅和继续前进的阶段，就像一个面临公关噩梦的名人向孤儿院等机构捐了一大笔款，他说“向外的善意姿态不会伤害你，但它们不是缝线，而是创可贴。他们可以开始愈合过程，但如果你的性格受到质疑，严重的康复需要很长时间。”

### 从现在开始做一个正直的人

这既是出于玩世不恭的原因——你想成为被认为是直截了当的人——也是出于真诚的个人提升原因:你想让 T2 真正成为直截了当的人。利玛窦说，好的领导者“建立并保持一种共同的使命感”，让你的同事们不信任你会危及这种共事合议。如果你已经完成了第一步——“去找那个人，告诉他这个评论是基于沮丧或愤怒——这就为长期的康复努力做好了准备。做到这一点需要勇气、个人诚实和脆弱。”

现在，一个有效的，相互尊重的工作环境也需要团队其他成员尊重的强有力的领导，很有可能 Kellyanne Conway 不喜欢在她的办公室里。“[白宫]和我所经历过的任何办公环境都不一样...从高级职员开始，一整天都在工作，这似乎是完全的功能障碍。”

那么你怎么知道是时候退出了呢？我个人的晴雨表是当我无法阻止自己抱怨同事的时候。利玛窦是一个更乐观的人，他说他通常能够忘记过去，解决问题。“但如果你认为没有挽回的机会(这源于犬儒主义)，那么可能是时候继续前进了。但是你冒着名誉扫地的风险。”
