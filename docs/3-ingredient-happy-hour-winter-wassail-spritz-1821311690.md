# 三种成分的快乐时光:冬天的汽酒

> 原文：<https://lifehacker.com/3-ingredient-happy-hour-winter-wassail-spritz-1821311690>

各位周末快乐，欢迎回到 [*三料欢乐时光*](http://lifehacker.com/tag/3-ingredient-happy-hour) ，每周饮品专栏，提供超级简单又美味的酒水。继续我的趋势，把冬天的温暖变成寒冷的啜饮，我想向你介绍我的朋友，冬天的 Wassail Spritz。

Watch

Wassail 是一种加香料的苹果酒，通常是温热的，最初是作为一种仪式的一部分，旨在确保苹果丰收。你可以在很多商店找到预先加了香料的葡萄酒瓶，一旦买到，你就可以在里面添加酒精。我喜欢 Trader Joe 的产品，里面有醋栗汁，给它一种令人愉悦的粉红色。要让这款中世纪产品焕然一新，你需要:

*   1 盎司杜松子酒
*   1 盎司苹果酒(或任何其他加香料的苹果酒)
*   起泡酒到顶

在香槟杯中加入杜松子酒和冰镇的瓦塞尔酒，再加入冰镇的起泡酒。喝了它，唱了许多颂歌。
