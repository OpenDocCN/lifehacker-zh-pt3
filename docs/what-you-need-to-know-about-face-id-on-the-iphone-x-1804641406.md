# 关于 iPhone X 上的 Face ID，你需要知道什么

iPhone X 的全屏显示可能是你注意到这款新设备的第一件事，但苹果升级的前置摄像头也值得你关注。也许更是如此。

苹果公司利用 iPhone X 推出了指纹扫描标准，该标准是它与 [【触控 ID】](http://lifehacker.com/fix-your-iphone-s-touch-id-by-retraining-it-1680003851)一起引入的，取而代之的是面部扫描技术。Face ID 更加安全(至少苹果是这样认为的)，它为 999 美元的智能手机 [带来了一些很酷的新功能，以及你可能会想到的面部扫描技术带来的隐私问题](http://gizmodo.com/face-id-raises-some-scary-questions-here-are-some-answe-1804313075) 。以下是你需要知道的。

### 它是如何工作的

Face ID 是由一个新的原深感摄像头系统驱动的，这个系统在屏幕顶部的小条中集成了大量的技术。这包括一个常规的前置摄像头，一个红外摄像头，一个点投影仪，一个泛光照明器，一个接近传感器和一个环境光传感器。

每当你看你的手机，泛光照明检测你的脸。然后，红外摄像机拍摄红外图像，点投影仪使用 30，000 多个红外点来创建点图案。两组数据都被发送到 iPhone 的新 A11 仿生芯片，该芯片使用经过 10 亿多张图像训练的神经网络，将其与已经存储在设备上的面部数学模型进行比较。

根据苹果公司的说法，这一切都应该快速无缝地发生。整个过程对人眼来说是不可见的，它在黑暗中工作。

不过，值得注意的是，Face ID 一开始可能会有点问题，在周二的活动中，苹果实际上 [很难使用它](http://www.businessinsider.com/apple-iphone-x-face-id-fails-in-first-public-test-2017-9) 。公平地说，随着时间的推移，触控 ID 有所进步，Face ID 可能也会如此。但是，如果你今年做出改变，在 iPhone X 上市后第一时间购买它，你可能会冒一点风险。

### 比触控 ID 更安全

使用 Face ID 的最大原因可能是增加您的安全性。触控 ID [还没有完全被证明是防黑客的](http://www.iphonehacks.com/2016/02/iphone-touch-id-hacked-with-play-doh.html) ，苹果公司称其面部扫描技术是一大进步。据苹果公司称，只有一百万分之一的人能够用 Face ID 来欺骗你的设备，而在触控 ID，这个数字是五万分之一。

与触控 ID 一样，您所有的 Face ID 数据都存储在 A11 处理器的安全区域。这意味着它不会被转移到苹果的服务器上，在那里它可能成为黑客的攻击目标。

苹果还指出，你不能用一张照片或一个面具来愚弄 Face ID。为了确保万无一失，该公司甚至用一些怪异逼真的面具来训练它的新软件。

### 增强现实和其他功能

为 Face ID 提供动力的相同传感器也将支持一些有趣的新功能，特别是在增强现实方面。Snapchat 已经在开发使用 TrueDepth 系统的新滤镜，比如这种可以与你的面部动作完美同步的面部彩绘面具。

苹果还展示了 Animoji，这是一个新的信息功能，可以让你记录视频信息，作为 12 种不同表情符号中的一种(是的，便便表情符号也包括在内)。这是一个有趣的新功能，也是一个很好的例子，说明了 Face ID 带来的可能性。

目前，这些新功能可能感觉更像是噱头，但如果苹果向外部开发者开放其 Face ID 技术，可能性将是无限的。

首先，该公司只需要证明它的面部扫描系统足够好，可以取代我们都依赖的指纹扫描仪。
