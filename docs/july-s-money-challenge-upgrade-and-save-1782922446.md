# 七月的金钱挑战:升级和节省

> 原文：<https://lifehacker.com/july-s-money-challenge-upgrade-and-save-1782922446>

上个月 [我们挑战你们讨价还价](http://twocents.lifehacker.com/how-d-you-do-during-june-s-money-challenge-1782817623) ，你们中的一些人节省了数百。我们印象深刻。这个月，我们有一位客人。 [钱包黑客](https://wallethacks.com/) 的吉姆王加入我们，我们挑战你使用他的 [“升级并保存”策略。](http://wallethacks.com/upgrade-and-save-strategy/)T11】



在[之前我们已经给大家讲过这个策略了，不过我们还是让王来解释一下它是怎么运作的:](http://twocents.lifehacker.com/cut-back-on-expenses-with-the-upgrade-and-save-strategy-1742037788#_ga=1.134605041.1268082208.1431441811)

> 升级和保存非常简单。有时候，你通过升级某些东西来节省更多的钱，而不是把它削减到财务的最低限度。我们被训练成认为，如果我们想省钱，我们应该削减，节省，或者用更少的钱来凑合。有时，花更多的钱购买更高质量的产品是值得的，因此你可以减少更换产品的频率，或者更方便地做出正确的决定，从而避免糟糕的财务选择。

一个经典的例子:内部调制解调器租赁。许多 ISP 向你收取月租调制解调器的费用，正如王在他的原始帖子中指出的，调制解调器并不好。你可以买自己的和，这样随着时间的推移，你可以为自己节省一些现金。王在他原来的博文 中又列举了几个[的例子，但思路是一样的。不要寻找可以削减的东西(我们已经做过的)，试着看看你可以“投资”的开支，这样随着时间的推移就能省钱。](http://wallethacks.com/upgrade-and-save-strategy/)

虽然这里有一个很大的警告。有些人可能会用这种方法作为花钱的借口。所以请记住:这里的目标仍然是储蓄。升级只是锦上添花。

您可以从查看您的预算和费用开始，看看您可以通过升级节省哪些费用。你加入吗？如果是这样，请告诉我们您计划如何升级，以及您预计今年剩下的时间可以节省多少钱。我们会在月底回来检查。
