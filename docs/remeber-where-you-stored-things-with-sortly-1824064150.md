# 还记得你用 Sortly 储物的地方吗

> 原文：<https://lifehacker.com/remeber-where-you-stored-things-with-sortly-1824064150>

我和我的男朋友住在一个两居室的公寓里，只有两个特别小的壁橱。最接近的大到足以容纳我们的衣服，但当谈到其他东西，人们可能会放在壁橱里，这些东西藏在我们的房子里。

Watch

我每周至少有一次“X 去哪儿了？”一会儿，我们花了一点时间在我们的存储点搜寻，最终找到了楼下的车库(老实说，我们的很多东西都在那里)，直到我们设法找到它。 [Sortly](https://play.google.com/store/apps/details?id=com.sortly.mythings) 是一款旨在让搜索变得更简单的应用。

有了这个应用程序，你可以拍下你存储的内容和位置，然后你可以在以后搜索它。它不是保持事物有序的替代品，只是一种简化记忆的方式。

很明显，你不会用它来整理你的整个房子，但对于某些事情来说，它会非常有用。例如，如果你有一个不错的威士忌收藏，但只想把几瓶放在户外，你可以把其他的放在壁橱里，但仍然可以很容易地知道你有什么，在哪里。或者，如果你需要跟踪不同季节的衣服，或者你只需要在圣诞节前后拿出来的厨房用具……想想你肯定想存放的东西，但也可以快速取用。

该应用程序免费提供多达 100 种商品。如果你需要跟踪更多信息，订阅这项服务的费用是每月 5 美元。
