# 现在你可以使用 LinkedIn 找到一个导师

> 原文：<https://lifehacker.com/now-you-can-find-a-mentor-using-linkedin-1797553251>

一个好的导师可能是一件很好的事情，但很难找到。现在，LinkedIn 正试图通过一项新的辅导功能，将那些寻求导师建议的人与愿意分享建议的专业人士联系起来。本周，它开始在几个选定的地点 推广，并计划很快推广到全国。

Watch

目前，所有可用的导师都是由 LinkedIn 亲自挑选的(尽管它最终将对每个人开放)，他们可以设定他们愿意指导的人和主题的范围。例如，一位导师可能会说，他们愿意帮助毕业于同一所大学的学生，或者那些在网站上与他们的专业网络中的其他人有几度关系的学生。

学员也有机会筛选他们在导师身上寻找的东西，当有匹配时，这两个人将在网站上联系起来，这样他们就可以相互传递信息。这是一个类似火绒的概念，因为除非你首先匹配，否则你将无法与对方聊天。你也可以随时结束谈话，所以如果你在第一次谈话中觉得不合适，双方都没有义务继续谈下去。

这是一个有趣的想法。我知道，在我写作生涯的早期，我从与曾经经历过我的人交谈中受益匪浅，我不时会收到母校学生的电子邮件和电话，他们试图进入正在寻求建议的领域。我总是乐意帮忙，但是如果没有我以前的指导老师来促成这些联系，我不完全确定现在的学生会如何找到我。建立这些联系对双方来说都很困难，而这可能会稍微简化这个过程。

而且肯定有很多职业咨询服务已经在为钱做这件事了。然而，当你在寻找新的工作时，免费往往是一个更容易的提议。尽管缺乏资金可能会让你很难留住真正有用的导师。目前，在该功能可用的地区(不幸的是，目前只有旧金山和澳大利亚)的用户将在 LinkedIn 应用程序的仪表板上的职业建议选项卡下看到成为导师或学员的选项。

你怎么想呢?你会用 LinkedIn 找导师还是自己在平台上做导师？
