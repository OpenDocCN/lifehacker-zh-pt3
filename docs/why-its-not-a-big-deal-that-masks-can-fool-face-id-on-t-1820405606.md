# 为什么口罩能在 iPhone X 上骗过 Face ID 没什么大不了的

> 原文：<https://lifehacker.com/why-its-not-a-big-deal-that-masks-can-fool-face-id-on-t-1820405606>

当苹果首次宣布 iPhone X 的 [Face ID 时，它声称这一新功能比触控 ID 安全得多，甚至连最逼真的面具都骗不了。但事实可能并非如此。](https://lifehacker.com/what-you-need-to-know-about-face-id-on-the-iphone-x-1804641406)

越南网络安全公司 Bkav 在你抓狂之前，下面是这个消息对你的智能手机安全的实际意义。

### Bkav 如何击败 Face ID

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-i4YQRLQVixM&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-i4YQRLQVixM&start=0) 

根据 Bkav 的说法，要制作一个能欺骗 Face ID 的面具，你只需要扫描这个人的面部和价值 150 美元的材料。

第二部分很简单。这个面具是用一些雕刻硅、印刷塑料、化妆品和剪纸制作的。关键部位，比如眼睛和嘴巴，实际上是用 2D 的照片粘贴到 3D 表面上重建的。这个设计是基于这样一个发现，即 Face ID 只扫描你大约一半的脸，而且据称不需要眼球运动就能工作，这使得它非常容易被欺骗。

不过，第一部分有点复杂。Bkav 使用的手持扫描仪需要五分钟才能工作。所以扫描某人面部的唯一方法是在同一个房间里，有他们的参与(无论是选择还是强迫)。

### 为什么你不应该担心

有人需要与你共处一室这一事实意味着 Face ID 黑客对大多数人来说并不构成太大威胁。Bkav 指出，世界领导人和首席执行官可能面临有针对性的攻击，但对我们其他人来说，这不值得担心。

最终，用智能手机摄像头快速扫描某人的面部，甚至使用照片，你也有可能得到同样的结果。但同样，你现在不需要担心这个。

我们也仍然不知道 Bkav 的说法到底有多合理。该公司可能故意在设置 Face ID 方面做得很差，以便更容易作弊，这会破坏结果。Bkav 拒绝回答《连线》的一系列问题，但表示将在本周的新闻发布会上透露更多信息，因此我们应该很快就会了解更多信息。

在那之前，没有理由停止使用 Face ID，除非你觉得用起来很烦。
