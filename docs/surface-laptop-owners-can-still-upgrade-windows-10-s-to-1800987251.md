# Surface 笔记本电脑用户仍然可以免费将 Windows 10 S 升级到 Windows 10 Pro

> 原文：<https://lifehacker.com/surface-laptop-owners-can-still-upgrade-windows-10-s-to-1800987251>

微软 6 月份发布的 Surface 笔记本电脑的用户有更多的时间从附带的 Windows 10 S 升级到全功能的 Windows 10 Pro 操作系统。

在本周的一篇 [博客文章](https://blogs.windows.com/devices/2017/09/05/surface-laptop-colors-arriving-additional-countries/#mmRcMKAuu15KwXg1.97) 中，微软宣布将其 Windows 10 S 升级计划延长三个月，最初定于 2017 年底完成。新的升级截止日期是 2018 年 3 月 31 日，这让这款出奇沉稳的笔记本电脑的所有者有更多的时间来决定他们想要运行哪个版本的 Windows 10，是对教育工作者友好的 Windows 10 S 还是更强大的 Windows 10 Pro。

如果你目前的犹豫不决导致你错过了延长的截止日期，你可以支付 49 美元购买升级许可证，但你可能应该在此之前做出决定。升级足够容易，如果你没有找到你喜欢的 Pro 版本，你可以 [降级回 Windows 10 S](https://www.microsoft.com/surface/en-us/support/warranty-service-and-recovery/downloadablerecoveryimage?ranMID=24542&ranEAID=TnL5HPStwNw&ranSiteID=TnL5HPStwNw-pRiHcuYTR6.1aAKGyJ22dA&tduid=(1ce3be92cff5a794b792fe8035fca88a)(256380)(2459594)(TnL5HPStwNw-pRiHcuYTR6.1aAKGyJ22dA)()) 。

Windows 10 S 本质上是传统 Windows 10 Pro 操作系统的精简版，专为低端设备设计。它的设计考虑到了效率，增强了安全性，适合在学校和办公室使用。它还允许其他制造商(如惠普的 ProBook x360 )制造更便宜的 Windows 10 设备，以与学校使用的越来越多的 Chromebooks 竞争。不幸的是，即使它可以访问微软的应用程序商店，也有很大的限制。

Windows 10 S 强迫你使用微软的 Edge 浏览器和 Bing 搜索引擎，两者都不能改变(当然，如果你足够绝望，你可以只让谷歌成为你的主页)。由于它不能运行传统的 Windows 应用程序，只能运行 Windows 商店中的应用程序，因此您的应用程序选择受到严重限制。像 Adobe 的 Creative Suite 应用程序这样的全功能程序，或者像 Google Chrome 这样的第三方网络浏览器，基本上都是行不通的。

升级到 Windows 10 Pro 使用户能够像普通电脑一样运行传统的 Windows 应用程序，但由于 Windows 10 S 中的应用程序沙箱功能，您将错过针对恶意软件和其他病毒的增强保护。

Windows 10 Pro 也比更精简的 Windows 10 S 使用更多的功能。您可以访问微软的 BitLocker 文件加密功能，以及使用远程桌面连接功能的能力，让您从另一个位置控制 Windows 10 Pro 计算机。简而言之，升级会给你带来很多好处，但代价是电池寿命和启动时间会变慢。

微软的 Surface 笔记本电脑售价 1000 美元，在 Windows 10 S 笔记本电脑中属于高端产品，类似于谷歌的 Chromebook Pixel，售价 1299 美元。如果你花了一千美元买了一台笔记本电脑，你也许应该帮自己一个忙，升级你的操作系统，这样你就不会发现自己陷入困境。
