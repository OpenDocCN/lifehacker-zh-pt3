# 用 Mozilla 的“备忘录”在火狐浏览器和安卓设备上同步你的待办事项列表

> 原文：<https://lifehacker.com/sync-your-to-do-list-across-firefox-browsers-and-androi-1827513237>

正如我今天早上早些时候提到的，Mozilla 前几天推出了两个新的应用程序，作为其测试试点计划的一部分:一个用于在你的 Firefox 浏览器和你的 iOS 设备(不久将是 Android)之间同步密码，另一个应用程序(和扩展)用于在你的 Firefox 浏览器和 Android 设备之间同步笔记。



后者被火狐 简称为“ [”笔记，“是一种记录你想法的简单方法，而不必处理超级复杂的扩展——或者，取决于你的操作系统，一个专门做笔记的独立应用程序。由于 Firefox 浏览器是跨平台的，你可以在 Mac 和 Windows 系统上查看你写的笔记。安卓也是，如果你安装了火狐 app](https://testpilot.firefox.com/experiments/notes/) 单独的 [便签。](https://play.google.com/store/apps/details?id=org.mozilla.testpilot.notes)

整个过程是这样的。你首先要安装 Test Pilot，并在浏览器中启用 Notes 扩展——只要在 Mozilla 的 Firefox 测试网站的 [上寻找一个巨大的按钮就可以了。同意扩展的权限，浏览器的左侧会弹出一个侧边栏，上面有一条来自 Mozilla 的可爱的欢迎消息。](https://testpilot.firefox.com/experiments/notes/)

正如你将看到的，Notes 使用 Markdown 格式:因此，你需要记住**blahblahblah**是你加粗文本的方式，例如，而不只是键入“blahblahblah”，高亮显示它，然后按 Command(或 Control) + B。你可以在 Firefox notes 中做所有的基本操作——项目符号列表、编号列表，甚至是代码的特殊文本格式——但你不能插入超链接。我的意思是，你可以输入一个网站的网址，当然，但没有办法把它转换成一个可点击的超链接(目前)。

但是，您可以将笔记导出为 HTML 文件。这仍然不能神奇地在 URL 之外创建可点击的超链接，但这至少是保存您已经处理过的一些大便笺的好方法。

举个例子，一旦你在 Firefox 扩展中写下了购物清单，在 Firefox 的 Android 应用程序中调出清单就很容易了。安装这个应用程序，用你的 Firefox 帐户登录，你之前在电脑上输入的所有内容都会出现在你的智能手机或平板电脑上。相反，你在 Notes 应用中输入的任何内容都会出现在 Firefox 的 Notes 中。

笔记比 [熊](http://www.bear-writer.com/) 之类的 app 好用吗？不。你不能剪辑文章，然后轻松地放入火狐笔记中，上面提到的超链接可能会让你发疯，你甚至不能调整文本颜色(或者笔记本身的背景颜色，如果你想发疯的话)。不过，这是一种在任何火狐浏览器和你的 Android 设备之间同步笔记的简单方法。如果这就是你所需要的，Mozilla 的实验性扩展和应用程序可以满足你。
