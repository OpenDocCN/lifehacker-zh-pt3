# 如果你厌倦了糟糕的桌面电子邮件应用，试试 Mailspring

> 原文：<https://lifehacker.com/try-mailspring-if-youre-tired-of-terrible-desktop-email-1826216206>

除非你是微软 Outlook 的超级粉丝，否则你很可能不会在家里使用桌面电子邮件客户端。你可能使用基于网络的电子邮件服务，你的日常工作可能包括启动浏览器删除、移动或忽略你的邮件。像 Mail 这样的第三方应用(用于 Windows *或* Mac)被诅咒了。

Watch

事实上，这正是 Ars Technica 的员工 [最近在](https://arstechnica.com/information-technology/2018/05/inbox-zero-and-the-search-for-the-perfect-email-client/) 内部争论的问题:还有什么好的桌面电子邮件客户端吗？这个问题曾经有过屈指可数的很棒的答案:雷鸟、麻雀、邮箱，不一而足。以前还没有从数字地球上消失的体面的电子邮件应用程序不再更新了，这不是收件箱安全的最佳做法。

除非你坚持使用你的操作系统内置的邮件应用程序，或者给微软 Office 一个昂贵的尝试，否则现在唯一一个似乎并不可怕的免费桌面电子邮件软件是一个叫做 Mailspring 的开源应用程序。诚然，我自己并没有怎么使用它，因为我只在雇主需要的时候才使用桌面电子邮件应用程序，但一些 Ars Technica 的评论员对此深信不疑，该网站的 it 编辑兼国家安全编辑肖恩·加拉格尔也是如此:

> “我现在平均每周收到近 7000 封电子邮件。我开始使用 MailSpring，这是一个开源的邮件客户端，我在 Linux 上使用它(出于多种原因，这是我首选的桌面环境)来处理多个网络邮件帐户，并且我有积极的垃圾邮件过滤功能。我倾向于把电子邮件放在我的收件箱或者存档，这样我就可以在里面搜索我在特定话题上的联系人。我已经学会不再担心，爱我的“未读”号码。”

[Mailspring](https://getmailspring.com/) 适用于 Windows、Mac 和 Linux。(在 Windows 上)设置它是一个非常简单的过程。该应用程序会要求你创建一个特殊的 Mailspring ID 来使用该应用程序，这是它与其他“只需导入你的电子邮件帐户并运行”应用程序相比的少数缺陷之一。

从那里，你只需从一个流行的提供商列表中选择你的电子邮件帐户提供商。

由于我只有 Gmail 账户*到*连接到 Mailspring，我很高兴地看到应用程序实际上会将你跳转到 web 来完成配置，而不是要求你直接在应用程序中键入用户名和密码。(由于我的 Gmail 密码巨大，复制粘贴是一件痛苦的事。)如果你已经在网上登录了你的电子邮件账户，你必须授权 Mailspring 访问它的内容。

我也很高兴看到 Mailspring 实际上提供了不同的查看选项，可以尝试模拟您基于网络的体验(桌面应用程序所能做到的最好的)。

同样值得注意的是——我很高兴应用程序将这一点放在前面——免费版的 Mailspring 允许你最多连接四个电子邮件帐户。你必须每月支付 8 美元才能获得更多，还可以使用 Mailspring 的一些更好的功能:阅读回执、信息打盹、有用的“稍后发送”命令和后续提醒。

尽管如此，Mailspring 的基本版本是一个非常实用的电子邮件客户端，感觉使用起来很快——现在我实际上已经试用了一天左右——并且组织得非常好。我不确定是否有足够的东西来吸引我离开 Gmail，尤其是如果你是优先或“集中”收件箱概念的忠实粉丝，这是 Mailspring 所缺乏的，但我可以肯定地看到 Mailspring 在那些只想要一个未读邮件的简单列表而不是铃铛、口哨或花哨的用户界面的人的桌面上找到了一个幸福的家。
