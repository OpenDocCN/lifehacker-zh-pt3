# 通过“行李检查”停止过度包装旅行

> 原文：<https://lifehacker.com/stop-overpacking-for-trips-with-a-luggage-audit-1795863848>

我们大多数人在去 旅行的时候都倾向于过度包装。你还不如扔掉那些多余的衬衫、充电器和相机配件，因为你永远不知道你可能需要什么，对吗？当然，你带来的东西有一半都没用完。如果这听起来很熟悉，下次旅行时考虑一下“行李检查”。

Watch

旅行结束后，很容易就把所有东西都卸下来，忘得一干二净，然后在下一次度假时带上同样没用过的垃圾。为了克服这一点，试试这个:当你打开行李时，花一分钟想想你打开的每一件物品。问自己几个问题:*你多久使用一次？值得带去吗？如果需要节省空间的话下次出行可以跳过吗？*

从那里，创建一个“跳过它”的列表。记下你后悔打包的每一件物品。把它们写下来，在你的行李里扔一张纸条。这样，下一次你打包旅行时，如果你想节省空间，你会确切地知道你可以和应该跳过什么。

我最近在一次欧洲之行后这样做了。虽然我带了一些轻便的东西，但我意识到我还是带了一些在那一周中从未用过的东西:一双高跟鞋(那顿别致的晚餐从未有过结果)、几双多的袜子(我以为会更冷)、一个笔记本(我高估了自己开始写旅行日记的愿望)。为了节省下次旅行的空间，我回家后迅速检查了一下行李。当我打开行李时，我记下了在下一次旅途中跳过这些不用的东西。

当然，这并不总是可能的——也许我的下一次旅程会很冷，我会*需要*那些额外的袜子——但它可以很好地提醒我们 [如何削减你的行李箱](http://lifehacker.com/how-to-fit-two-weeks-worth-of-luggage-under-the-airplan-5990659) 并停止打包那些你从未用过的 [一样的东西。](https://lifehacker.com/what-do-you-always-regret-packing-when-you-travel-1793430154)T9】
