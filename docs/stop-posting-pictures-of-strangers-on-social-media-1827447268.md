# 停止在社交媒体上发布陌生人的照片

> 原文：<https://lifehacker.com/stop-posting-pictures-of-strangers-on-social-media-1827447268>

未经某人同意就拍下他的照片并把它发布到网上是一件很糟糕的事情。这是侵犯性的，不恰当的，甚至会将他人置于危险之中。在一个有意义的世界里，这不需要更多的解释。这将是社会契约的一个普遍理解的部分。

相反，仅上周就有关于令人难以忍受的#PlaneBae 传奇、、以及最近记忆中一个更令人苦恼的、[亲爱的普鲁迪问题](https://slate.com/human-interest/2018/07/dear-prudence-im-not-the-father-of-my-cousins-baby.html)、的故事，让互联网的大门变得漆黑一片。

就前者而言，Twitter 用户 Rosey Blair 花了几个小时在 Twitter 上直播飞机上坐在她前面的两个陌生人的调情，并配有图片，获得了成千上万的转发(她后来渴望利用她的 [病毒般的名气在 Buzzfeed](https://twitter.com/roseybeeme/status/1014667854594048000) 工作，这应该会告诉你你需要知道的一切)。在后一个案例中， [一位亲爱的普鲁迪写信人在被抓到未经授权拍摄一位超重同事的照片并“分享到一个在线社区，在那里我们讨论我们生活中的肥胖人士”后，看起来被免除了责任](https://slate.com/human-interest/2018/07/dear-prudence-im-not-the-father-of-my-cousins-baby.html)

这两个事件背后的意图——“让我从这个发生在陌生人之间的可爱爱情故事中获得一些甜蜜的 RTs！”“让我在网上和我的一群混蛋朋友一起烤某人”——显然是非常不同的。但在这两种情况下，张贴照片是一种粗暴的、近乎虐待的对待他人的方式。

同样，这不需要解释。人们不会心照不宣地签署一份弃权书，同意仅仅因为他们敢出门，就把他们的形象、行为和实际位置传遍互联网。模糊他们的脸(或其他识别细节)稍微减轻了伤害，但即使这样，也不是真的！布莱尔在她的 Twitter 帖子中模糊了这对夫妇的脸，故事中的女人——她理所当然地不想参与其中——[拒绝公开她的名字，此后不得不删除她的社交媒体资料，以保护自己的隐私。](http://www.businessinsider.com/plane-bae-viral-social-media-creepy-2018-7) 仅仅因为你不是出于残忍的目的分享一张照片或视频，并不意味着照片或视频的主体想要成为任何类型的公众评论的中心，或者结果可能不会对他们个人造成毁灭性的影响。

这条规则有明显的例外。滥用权力的警察、纳粹和 [在街头游行的白人至上主义者](https://gizmodo.com/doxed-white-supremacist-caught-red-handed-with-torch-at-1797800347) ，任何试图伤害他人的人——这些人不再值得尊重隐私，应该积极记录在案，以更好地确保他们为自己的行为承担后果。

但是陌生人做他们自己的事情就不应该被打扰。不是每个人都想被变成内容。并不是一切都需要如此。
