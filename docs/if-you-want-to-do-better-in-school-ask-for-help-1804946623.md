# 如果你想在学校做得更好，寻求帮助

> 原文：<https://lifehacker.com/if-you-want-to-do-better-in-school-ask-for-help-1804946623>

学习可能感觉像一项孤独的活动——回顾我的学生时代，我记得独自坐在书桌前，只有我的课本、荧光笔、可乐罐和不断扩大的自我怀疑。如果我不理解一个概念，或者找不到答案，我会拼命地在书页里搜索，想着也许有我错过的线索。如果看起来仍然没有希望，我会睡觉，合理化，也许不知何故，启示会出现在我的梦里。(通常不会。)

科学表明，有一种更有效的策略是学生们没有充分利用的，那就是**求助**。(老鼠，我本来希望是买更多的荧光笔。)

一篇 [*《华尔街日报》*](https://www.wsj.com/articles/the-smarter-ways-to-study-1502810531) 一篇集锦[2016 年，圣路易斯大学的研究人员对 414 名参加科学导论课的大学生进行了研究](http://www.lifescied.org/content/16/2/ar30.short) 。那些在上班时间向老师求助的学生更有可能得到 A，但只有不到五分之一的学生这样做了。这并不是说办公时间一定是取得好成绩的关键，但重要的是这些学生采取了《华尔街日报》所说的“积极的方法”这意味着他们认为自己有责任实现自己的目标，所以如果他们有什么不明白的地方，他们会问老师或同学，或者寻求其他资源的指导。

教授们希望学生在进入大学之前掌握这种技能，然而许多人并没有。文化和性别可能是因素(男性倾向于 [更少寻求帮助](https://www.facultyfocus.com/articles/edtech-news-and-trends/why-students-dont-ask-for-help-and-what-you-can-do-about-it/) )，以及担心“打扰”忙碌的老师或被视为毫无准备。密歇根大学心理学家的一项研究发现，随着孩子年龄的增长，他们越来越少向同学寻求帮助来理解概念，而更有可能只是想抄他们的作业。但是专家说，寻求帮助可以最大化学习。简单地阅读和重读材料不会将其锁定在记忆中。

化学老师詹姆斯·肯尼迪 [甚至在自己的教室里用图表记录了](https://jameskennedymonash.wordpress.com/2015/06/01/high-achieving-students-ask-for-more-help/) 的观察结果。

肯尼迪写道:

> 我在收件箱里搜索了我所教的每个学生的姓氏，然后数了数他们自本学年开始以来集体问我的数百个问题。我将学生分成四分位数，并绘制出过去几个月中被问问题的平均数量与他们当前学术表现的对比图。
> 
> 结果令人震惊:不仅成绩较高的学生比成绩较低的学生通过电子邮件问我更多的问题，而且相关性惊人地强(R = 71%)。这就引出了一个问题:成绩优异的学生得到更高的分数是因为他们寻求更多的帮助吗？从老师那里得到更多的帮助和取得更高的成绩之间有因果关系吗？常识表明是有的。

能够自信地寻求学校作业的帮助是需要练习的。这里有一些给学生的建议，告诉他们如何与他人接触以获得更大的成就:

*   使用 5 分钟规则。肯尼迪写道，如果你在一个问题上卡住超过五分钟，却毫无进展，你应该向老师或导师寻求帮助。
*   找到或组成一个学习伙伴或小组。一项 [2015 年的研究](http://advan.physiology.org/content/39/4/335) 发现，与使用其他学习技巧的学生相比，每周聚集在一起并相互测验的学生取得了更高的成绩。

*   如果你对亲自向老师寻求帮助感到紧张，给他们发电子邮件。记住你的要求要具体，要知道通过询问，你表明了你想尽力而为。
