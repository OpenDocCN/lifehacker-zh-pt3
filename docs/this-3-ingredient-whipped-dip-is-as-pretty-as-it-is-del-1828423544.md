# 这种三种成分的搅打蘸酱既美味又漂亮

> 原文:[https://life hacker . com/this-3-ingredient-whipped-dip-is-as-pretty-as-it-is-del-1828423544](https://lifehacker.com/this-3-ingredient-whipped-dip-is-as-pretty-as-it-is-del-1828423544)

奶油咸蘸酱吸引了所有人的注意，但将一种食物蘸上另一种食物不应该仅限于美味小吃。甜点蘸酱让吃水果变得更有趣，而且不知何故，它们甚至比洋葱蘸酱更容易做。你所需要的只是一个奶油奶酪和某种电动搅拌器。

在其最简单的形式中，奶油甜点蘸料可以由单一成分组成。拿一块柔软的、有味道的山羊奶酪——像 Trader Joe's honey chèvre 或其他有蔓越莓的东西——搅打它，直到它足够软，可以让草莓穿过。如果你只有纯奶油奶酪，那也很好；加入一汤匙糖粉、一茶匙杏仁或香草精或一些可可粉搅拌。(或以上任意组合。)

这些都很好吃，但是如果你想做一种甜腻的蘸酱，让你的眼睛和舌头都感到愉悦，那就吃一些冻干水果吧。当在食品加工机中粉碎时——扔一把进去，让它旋转，直到你找到粉末——只要一汤匙冷冻干燥的浆果就能给你搅拌的任何东西增加酸味和明亮的颜色。要制作类似芝士蛋糕的甜品酱，你需要:

*   8 盎司全脂室温奶油奶酪
*   1 汤匙糖粉
*   1 汤匙粉碎的冻干水果(蓝莓和草莓是我的最爱。)

将所有材料放入碗中，用立式搅拌机或手动搅拌机搅拌，直到材料变软、易消化、颜色均匀。搭配水果、椒盐卷饼、饼干，面带微笑(或怒容；永远不要让任何人告诉你微笑)。
