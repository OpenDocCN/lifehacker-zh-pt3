# 如果有人以你的名义申请学生贷款，该怎么办

> 原文：<https://lifehacker.com/what-to-do-if-someone-takes-out-student-loans-in-your-n-1825882484>

身份盗窃有多种形式，其中一种可能包括有人以你的名义申请学生贷款。那岂不是一个很好的惊喜！



由于 2017 届学生的平均贷款余额为 39，400 美元，我们假设你会想要解决这个问题。下面是要做的事情。

### 在你的信用档案上贴上欺诈警告

在你的信用档案中放置一个欺诈警告意味着债权人将不得不采取额外的步骤来验证你的身份，然后才能开立或扩大新的信用额度。你可以申请一个持续 90 天的初始欺诈警报，或者一个有效期为 7 年的延长警报，并要求债权人亲自联系你。如果你的身份已经被盗，你会选择延长警报。

为此，请联系三家主要的信贷机构:

*   **Equifax** : [在线](https://www.alerts.equifax.com/AutoFraud_Online/jsp/fraudAlert.jsp) 或电话 888-766-0008
*   **Experian** : [在线](https://www.experian.com/fraud/center.html) 或电话，888-397-3742
*   **环球联盟** : [在线](https://www.transunion.com/fraud-victim-resource/place-fraud-alert) 或电话，888-909-8872

通知一个局 [需要](https://www.consumerfinance.gov/ask-cfpb/what-do-i-do-if-i-think-i-have-been-a-victim-of-identity-theft-en-31/) 他们通知另外两个局，但是为了安全起见，你应该自己给三个局都打电话。

另一件要知道的事情是:当你对你的信用发出初始警报时，除了你每年收到的典型的免费报告之外，你还有权获得三份信用报告中每一份的免费副本。根据 [消费者金融保护局](https://www.consumerfinance.gov/ask-cfpb/what-do-i-do-if-i-think-i-have-been-a-victim-of-identity-theft-en-31/) 的规定，当您发出延长警报时，您有权在 12 个月内免费获得两份。

或者，你可以对你的信用报告进行安全冻结，这意味着债权人将无法以你的名义开立新账户。如果你决定走这条路，你需要单独通知每个机构。

### 向警方报案

为了还清债务，你需要一份警方报告给债权人看。要向警方报案，请携带一份概述欺诈债务的信用报告副本，前往当地警察局。

你也可以 [向美国联邦贸易委员会](https://www.identitytheft.gov/) 申报，也可以联系 [社会保障局](https://www.ssa.gov/pubs/EN-05-10064.pdf) 。

### 对错误提出质疑

带着你的三份信用报告，和中介机构争论债务问题。然后，联系贷款人——如果是公共贷款，要么是联邦政府，要么是银行或其他私人金融机构。

如果是联邦贷款，有了适当的证明 就有可能得到 [解除。你将不得不与私人贷款人和他们的过程，你应该给他们一份你已经提交的警方报告的副本。](https://studentaid.ed.gov/sa/repay-loans/forgiveness-cancellation#false-certification)

正如 [身份盗窃资源中心](https://www.idtheftcenter.org/Identity-Theft/fafsa-and-student-loan-identity-theft.html) 指出的那样，梳理你的信用报告，寻找骗子可能已经开设的其他欺诈账户，因为它们可能不止于学生贷款。

### 联系学校

联系骗子贷款的学校，告诉他们贷款实际上不属于你。请他们结束贷款，并在结束后给你发一封信。

### 向教育部报告这个问题

最后，你可以通过在线 或拨打 800-647-8733 向能源部监察长办公室 [投诉。对于所有这些步骤，确保你写下了你联系的人和时间，并保留你收发信件的副本。](https://www2.ed.gov/about/offices/list/oig/hotline.html)

这里有一些来自联邦贸易委员会的 [其他步骤](https://www.identitytheft.gov/Steps) 。

你不能 100%地防止欺诈的发生，但是你可以采取措施降低它发生的可能性或者立即发现它。例如，确保你利用了免费的年度信用报告，每四个月向三家机构中的一家索取一份。

像 Credit Karma 和其他网站这样的应用程序也可以让你全年访问信用报告，并进行监控。如果拥有您个人信息的公司出现数据泄露，欺诈警报也是您可以采取的措施。
