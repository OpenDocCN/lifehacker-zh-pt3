# 你应该使用借记卡吗？

> 原文：<https://lifehacker.com/should-you-use-a-debit-rewards-card-1826434652>

借记卡奖励卡不像以前那么普遍了，但对于那些对使用信用卡持谨慎态度的人来说，它们仍然是一种选择。



许多网上银行提供奖励支票账户，与传统银行相比更具竞争力。例如， [Discover Bank](https://www.discover.com/online-banking/checking/) 为每月购买 3000 美元的借记卡提供 1%的现金返还( [网上银行](https://www.bankofinternet.com/checking/rewards-checking) 也有奖励支票)。美国银行也提供 [奖励](https://www.bankofamerica.com/deposits/checking/debit-card/) 。

一个潜在的好处是，你可能倾向于比用信用卡花更少的钱。也就是说，它们通常没有信用卡项目慷慨。

“顾客通常从借记卡购物中获得 1%的现金返还，”注 [资助](https://www.bankrate.com/banking/checking/think-twice-about-debit-card-reward-programs/) 。“使用一些最佳奖励信用卡，持卡人可能会获得高达 6%的现金返还。”你可以很容易地找到一张卡，每次购物都能给你双倍积分或里程，而在借记卡程序中，“你可能不得不花几美元才能获得一个积分或里程”。许多信用卡支付数百美元仅仅是为了注册和满足日常购物的基本消费需求。

另一个重要的考虑因素是:你的借记卡会以一种信用卡不会的方式决定你在哪里存款，因为它与你的支票账户相连。更重要的因素是费用、最低余额要求和其他产品的质量。想想看:如果你犹豫要不要一张年费为 95 美元的积分卡，但选择了一个附加在每月维护费为 12 美元的支票账户上的积分卡，那你就没有出头之日。

最后，你可能已经有了扣款奖励积分，却不知道。据Credit.com 报道，许多消费者正受益于 Cardlytics，这是一个于二月份上线的程序。该公司与美国银行和 PNC 等主要银行合作，为您的日常购物提供奖励。登录您的银行应用程序，查看您是否累积了奖励。

但总的来说，信用卡有更好的回报交易。
