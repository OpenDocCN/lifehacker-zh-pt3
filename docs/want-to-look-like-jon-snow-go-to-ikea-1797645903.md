# 如何用宜家地毯制作琼恩·雪诺斗篷

> 原文:[https://life hacker . com/want-to-look-like-Jon-snow-go-IKEA-1797645903](https://lifehacker.com/want-to-look-like-jon-snow-go-to-ikea-1797645903)

我敢打赌，你和我一样，看了一眼《权力的游戏》中守夜人的斗篷，心想“该死，如果它们不是跳蚤猖獗，沾满死人的血，我会毫不犹豫地穿上它。我上哪儿去弄一件那样的斗篷？”你可以像时装秀的服装设计师一样，用该死的宜家的 [SKOLD](http://www.ikea.com/ca/en/catalog/products/80271380/) 和 [LUDDE](http://www.ikea.com/aa/en/catalog/products/90166468/) 羊皮地毯来制作你自己的斗篷。

《权力的游戏》服装设计师 [米歇尔·克拉普顿](http://www.micheleclapton.com/) 在洛杉矶盖蒂博物馆关于中世纪服装 的演讲中解释了地毯斗篷的起源。

简而言之，它们没有任何花哨之处。守夜人成员身上破旧的斗篷不是由猛犸象毛皮制成，而是由宜家的羊皮地毯制成，这些地毯经过染色、粗糙处理和修剪，看起来像是去过第一批人的拳头，然后又回来了。

> “这些帽子实际上是宜家地毯。这有点小技巧。我们能拿什么就拿什么。我们把它们剪短、剃光，加上结实的皮带，然后把它们拆开...我想让观众几乎闻到服装的味道。”

地毯被进一步处理，并彻底打蜡和磨砂，让它们看起来像是住了太久了。

自从得知这个消息后，人们显然成群结队地去宜家购买羊皮地毯，大概是为了把自己裹在里面，为夜王的到来做准备。

### 如何制作你自己的

感觉有创意？克拉普顿可能是一名职业服装师，但这并不意味着你必须在寒冷中颤抖时讨好时装秀中的斗篷。你可以用一些材料和一些缝纫技巧来制作自己的衣服。至于尺寸， [一定要看看这本方便的指南](http://imgur.com/r/LARP/1FhbUVr) 来算出你需要多少布料。

### 收集你的材料

*   地毯和布料:当然，如果你想穿得像守夜人一样，你需要和守夜人一样的布料。这意味着宜家之旅即将开始。拿起你的 [SKOLD](http://www.ikea.com/ca/en/catalog/products/80271380/) 和 [LUDDE](http://www.ikea.com/gb/en/products/textiles-rugs/rugs/ludde-sheepskin-white-art-83235110/) 地毯(提前打电话给商店，确保他们还有存货)然后离开。实际上，在你离开瑞典的沃尔玛之前，你应该为斗篷的其余部分准备一些黑色的布料。他们按码卖，很不错。
*   我想你可以看出，刺眼的白色宜家地毯还不太适合搭配你的守夜人斗篷。 [这里就是服装染色](https://www.amazon.com/Dylon-Permanent-Fabric-Ounce-Black/dp/B004BQ5SAQ?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/want-to-look-like-jon-snow-go-to-ikea-1797645903&asc_source=&tag=kinjalifehackerlink-20) 进场的地方。你可能染过你最喜欢的牛仔裤，让它们回到你的服装搭配中，你也可以用同样的方法把白色的地毯染成黑色。
*   皮带和皮带扣:粗略搜索一下皮带和皮带扣，会在网上找到数百个结果。你的最佳选择？计算出表带的长度，选择一个你信任的供应商，也许是一个熟悉皮革制品的供应商。至于皮带扣，你总是可以搜索易贝，那里的供应商倾向于提供像皮带扣这样的杂七杂八的商品，价格比亚马逊便宜。
*   缝纫材料:守夜人没有缝纫机，但你可以打赌克拉普顿在做这些斗篷时有。当然，你的消费级缝纫机可能无法穿过皮革，所以你可能不得不用针、线、剪刀和打孔机自己缝制那个部分。

### **测量两次，切割一次**

当然，在你去寻找地毯、针线包和布料来缝制一件属于你自己的斗篷之前，确保你知道你需要多少材料。当然，更长的斗篷需要更多的布料，但如果太长(或太短)，你会被自己绊倒或诅咒你冰冷的小腿。

有些布料可能不够宽，无法单独剪成半圆形。在这种情况下，你可能不得不将多片缝合在一起，要么是楔形(想想比萨饼切片)，要么是在半圆的外缘填充相同布料的更小的弯曲片。

### 苦恼的要死

我没在守夜人成员身上看到一件干净的斗篷。是什么让你如此特别？是时候让这件新斗篷经受考验了。拿把剪刀，扔进一个装满石头的桶里，让你的狗在里面滚几个~~天~~分钟，瞧！你自己的斗篷，准备好抵御从哈德姆到高庭的风暴。
