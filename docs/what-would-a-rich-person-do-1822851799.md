# 一个有钱人会怎么做？

> 原文：<https://lifehacker.com/what-would-a-rich-person-do-1822851799>

我的朋友曾经为一个非常忙碌和成功的人做助理，他正在国外旅行。接近他的旅行日期时，他意识到他的护照过期了，并要求她办理更新。经过大量的研究，她报告说:他将不得不花一天时间在政府办公室等待更新。这就是规则。他对着她叹了口气，问道:“一个有钱人会怎么做？”

Watch

可悲的事实是，如果你有足够的钱，“规则”并不总是适用。这家伙受够了。所以他们把它扔向问题。她了解到，虽然护照办理员 [不能比](https://lifehacker.com/why-you-shouldn-t-use-a-courier-or-expeditor-service-1788722221) 更快地办理护照，但他们*可以做所有的跑腿工作，让你省去一趟办公室。对这个人来说，这是值得的。*

你时不时会遇到这样的问题，如果你愿意，哪怕就这一次，花很多钱，这个问题也是可以解决的。这时你会选择当天送货、长途出租车、100 美元的理发店或 40 美元的客房服务。或者你找到一些幕后的解决方案，让你对金钱可以买到的东西感到厌恶。但这不会伤害任何人，所以你去做。

你问自己“一个富人会怎么做？”因为你不是一个富有的人，所以你这样做只是因为你意识到这种一次性服务的异常高的价值。你这样问是为了不让它成为一种习惯。(毕竟如果你 [习惯了便宜的方式](https://lifehacker.com/bad-coffee-will-make-you-a-happier-person-1822250111) 你会更开心。)

你会解决问题，你会继续前进，你下次会准备得更好。或者你会意识到你生活中的一个领域比其他领域花费更多的钱。资本主义下没有道德消费，不过，嘿，既然我们已经得到了，就让我们好好利用吧。
