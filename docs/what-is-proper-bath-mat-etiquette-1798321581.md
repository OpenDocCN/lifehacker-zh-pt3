# 什么是适当的浴垫礼仪？

> 原文:[https://life hacker . com/what-is-proper-bath-mat-礼仪-1798321581](https://lifehacker.com/what-is-proper-bath-mat-etiquette-1798321581)

好了，来说一个很严肃的话题:浴垫。使用它们有对错之分吗？

Watch

这个世界上有三种人:

1.  那些在淋浴或浴缸中稍微擦干，然后踩在浴室垫子上完成擦干的人。

2.  那些等到洗完澡或洗完澡站在浴垫上才擦干的人。

3.  那些根本不用浴垫的。

一方面， [踩在浴垫上之前先烘干掉](http://ask.metafilter.com/311758/please-explain-to-me-the-purpose-of-bath-mats) 种遗迹的那种浴垫。另一方面，等到你踏上浴室防滑垫后再擦干意味着你会有一张湿漉漉的浴室防滑垫。如果你不用，你就是个怪人。那么，你是哪种类型的人，为什么？让我们彻底搞清楚这个浴室防滑垫的问题。
