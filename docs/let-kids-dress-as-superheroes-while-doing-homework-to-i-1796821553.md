# 让孩子们在做作业时打扮成超级英雄，以增加毅力

> 原文:[https://life hacker . com/let-kids-dress-as-super heroes-while-do-I-1796821553](https://lifehacker.com/let-kids-dress-as-superheroes-while-doing-homework-to-i-1796821553)

如果你的孩子在做作业的时候睡着了，也许他应该穿上斗篷。

一项的新研究发现，在执行一项重复任务时，假装成蝙蝠侠等熟悉角色的四六岁儿童比那些保持原样的儿童坚持的时间要长得多。研究人员会问孩子，“蝙蝠侠工作努力吗？”蝙蝠侠的确是。

这一结果与心理学家所说的自我疏远有关。当我们处于困难之中时，很容易在压力下崩溃，陷入痴迷的分析中。这项发表在《儿童发展》上的研究解释说，精神分离有助于人们“超越紧急情况，以更远的视角看待问题。”对于小孩子来说，认同一个角色的积极品质可以给他们更大的成功动力。

成功着装的把戏也适用于成年人。在 [的另一项研究](http://www.nytimes.com/2012/04/03/science/clothes-and-self-perception.html) 中，西北大学的研究人员让一组受试者穿上白大褂。那些被告知穿着医生外套的人比那些被告知穿着画家外套的人更加专注。

所以，让你的小学生带着神奇女侠的盾和剑去她的下一次辅导课吧。每个人在处理乘法时都可以使用一些额外的超能力。
