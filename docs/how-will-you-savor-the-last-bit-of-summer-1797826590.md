# 你将如何品味夏天的最后一点时光？

> 原文：<https://lifehacker.com/how-will-you-savor-the-last-bit-of-summer-1797826590>

大家好，欢迎回到 [*有什么好吃的？*](http://lifehacker.com/tag/whats-cooking) ，这是一个开放的话题，你可以在这里分享你对所有食物相关的想法、建议、食谱和观点。夏天(很遗憾)即将结束，所以我想借此机会谈谈你最喜欢的夏季美食时刻，以及你打算如何品尝这个季节的最后一点时光。

Watch

拿我来说，我会吃掉我能拿到的所有桃子，但还有时间吃更多。一如既往，我想知道你的食物主题计划，并有一些问题让你开始:

*   这个夏天你吃过最好吃的烧烤是什么？虽然我对烤肉情有独钟，但今年夏天我最喜欢的烧烤经历是烧烤披萨派对，它的特色是一个有新鲜山羊奶酪、樱桃和火腿的馅饼。
*   你摘了一些时令水果了吗？ **那是什么？你是怎么吃的？**我从来没有去过一次真正的浆果采摘之旅，但我确实从我家附近的灌木丛中抓了一把黑莓，在回家的路上吃了起来。那很好。
*   今年你找到新的夏季食谱了吗？这不是真正的食谱，但我确实知道如何 [腌制冰淇淋](http://skillet.lifehacker.com/will-it-sous-vide-a-boozy-chocolate-cherry-sundae-1795352092) ，这很酷。
*   你吃了多少番茄三明治？我已经记不清了。
*   哪种水果被做成馅饼，你用了哪种馅饼皮配方？我还有一个宏伟的计划，那就是做一个桃子派，尝试一下 [斯特拉·帕克斯的老式馅饼皮配方](http://www.seriouseats.com/recipes/2016/06/old-fashioned-flaky-pie-dough-recipe.html) 。
*   你吃了多少玉米？我吃了很多玉米，大部分是以 [elote](http://skillet.lifehacker.com/savor-the-summer-with-mexican-street-corn-1796746553#_ga=2.249693199.888717557.1502647015-949419976.1446553382) 的形式，但我也在没有任何帮助的情况下，吃掉了一整批我的 [酸浆玉米](http://skillet.lifehacker.com/will-it-sous-vide-super-sweet-and-creamy-creamed-corn-1796902814) 。
*   你还需要执行哪些最后一刻的夏季食物计划？除了整个桃子派的事情，我真的很想在外面还热的时候多吃点海鲜。几周前，我在海边吃了一些螃蟹和对虾，这是一次近乎完美的经历。
*   你的“夏日饮品”是什么？我最终喝了比平时多得多的垃圾啤酒，主要是雷尼尔啤酒。我也喝了很多金巴利酒和苏打水，但是不管什么季节我都会喝。

很明显，你可以随意离开我的调查问卷，告诉我你曾经或者仍然计划要做的任何或者所有的夏季食物冒险。还有时间，我的朋友们。还有时间。
