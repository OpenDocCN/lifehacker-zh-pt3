# 把每一天都过得像父亲节一样

> 原文：<https://lifehacker.com/live-every-day-like-its-fathers-day-1826862488>

或者母亲节。或者你的生日。

我爸爸不喜欢礼物。每个父亲节，我都告诉他我给他买了同样的东西:知道我没有在他不需要的东西上花一分钱。他的回答是:“这是我一直想要的！”



他声称我们不需要一个虚构的节日来显示我们有多爱我们的家人。同样的道理也适用于母亲节，或者某人自己的生日:你应该每天都感到自己是特别的、被欣赏的。

这条规则对于避免六月第三个星期天拥挤的早午餐也很有效。为什么不在你喜欢的时候带爸爸出去吃早午餐呢？(是的，所有这些也适用于妈妈。)

另外，你见过商店买的父亲节贺卡吗？他们认为所有的父亲都喜欢钓鱼、高尔夫或木工。这个假期不是为抚养我长大的人准备的。他喜欢在阅读道路地图的时候在后院睡着，所以我需要自己做一张卡片(谢谢，纸莎草纸)。想象一下在一个随机的日子给某人一张手写的卡片的喜悦。父母会喜欢的。你会很快成为最受欢迎的孩子，你的姐姐不会有机会。
