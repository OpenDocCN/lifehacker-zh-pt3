# 如果你这个周末不能离开，听听国家公园的溪流声

> 原文：<https://lifehacker.com/stream-sounds-from-a-national-park-if-you-cant-get-away-1828702554>

如果这个劳动节你被困在家里，但希望你能去国家公园徒步旅行，国家公园服务会为你提供帮助。为了庆祝它的 102 岁生日，基金会已经开始播放国家公园的声音。



由国家公园管理局自然声音和夜空部门制作的名为 PARKTRAKS 的新 [12 分钟赛道](https://findyourpark.com/news/parktracks) 。

此外，该部门还拥有针对落基山 [和德纳里国家公园](https://www.nps.gov/romo/learn/photosmultimedia/soundlibrary.htm) 的鸟类声景的录音，以及生活在公园中的不同动物的录音，如 [考区的斯巴德蟾蜍](https://www.nps.gov/subjects/sound/cspadefoottoad.htm) 和 [松鸡(那是一种鸟)](https://www.nps.gov/subjects/sound/sounds-ptarmigan.htm) 。

当然，它不能代替亲自去公园，但它可能是在家小沙发露营的完美配乐，或者如果你只是需要从办公室的喧嚣中短暂休息一下，并想假装你正在外面放松。
