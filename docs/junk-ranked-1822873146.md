# 垃圾，排名

> 原文：<https://lifehacker.com/junk-ranked-1822873146>

清理你的房子是一项巨大的工作，无论是身体上还是精神上。每一个扔东西的决定都是对你生活方式的反思。即使当你决定扔掉一顶帽子、一张 DVD 或一个花瓶，你也必须决定是送人、卖掉还是真的扔掉它。今天让我们只关注简单的决定:那些你可以扔掉、回收或者(对你有好处的)的东西。)堆肥。



下面，我们列出了一些你可以不假思索就扔掉的东西。我们尽量不包括你可以捐赠或出售的东西，或者带有情感依恋的东西。这个列表还假设你有更好的选择，或者可以从你的手机上以低于 100 美元的价格购买。如果你扔不起垃圾，那就不要扔垃圾。

把这个列表留在你的日历上，过一个相对平淡无奇的周末。放上播客，快速浏览列表。我们将这些从最简单到最不简单进行了排序，所以当列表不再有用时，你可以退出。如果任何一项让你犹豫，不要处理它！趁你还有冲劲，赶快做一个简单的！走吧。

**肮脏的海绵、旧牙刷、**任何变得粗糙需要更换的东西，因为 [它们实际上影响你的健康](https://lifehacker.com/how-to-keep-track-of-your-gross-household-sponges-1822662815#_ga=2.25451005.193698699.1517863099-1297080755.1497980211) 。

塑料袋:它们超级有用！他们带着狗屎，他们排着卫生间的垃圾。扔掉一半。

纸箱:除非你要搬家，否则扔掉所有有灰尘的空盒子。请把它平摊开。

**坏笔:**你有一种喜欢的一次性笔，买得起二十支，其他的都可以扔掉。

**纸张:**所有已经数字化的纸张，或者已经失去效用的纸张。收据、卡片、文件、手册、月结单。这里有一个关于 [你需要把你的论文](https://lifehacker.com/how-long-should-i-keep-old-documents-5994409) 保存多久的指南。现在不要担心扫描你所有的文件，或者无纸化。下周再做吧。



糟糕的食物:看看你冰箱里的每样东西。如果它看起来不好吃，就把它扔掉。脆皮辣酱，你从来没有打开过也永远不会打开的食物，你阿姨强加给你的食物，因为她没有勇气自己扔掉它。让对你浪费的内疚提醒你更加谨慎地购物。

不新鲜的食物:不要在新的面包还没被使用的时候就用完了旧的不新鲜的面包。把不新鲜的面包放在冰箱里(抱歉，我们很浪费，但不是*那样*浪费)然后开始新的面包。堆肥你枯萎的绿色，你的可疑牛奶，你的粉状水果。或者全部吃掉，现在，在一顿大餐里，*现在*。

**Cords:** 如果你不能立即识别它连接到什么，就扔掉它。如果一年没用过，就扔了。如果它只在你的旧手机上有效，扔掉它。最好每三年羞怯地更换一次扔掉的绳子，然后一直用它们塞满抽屉*。* 

***破衣烂衫:**不用担心自己很少穿的衣服。扔掉那些被弄脏、撕破或拉伸的衣服。除非你身体上没有足够的衣服穿，否则扔掉那些破烂的，不管它们以前有多好。对袜子和内衣要特别积极。现在可以在网上订阅内衣，不要穿快散架的。* 

***孤儿对的东西:**袜子，耳环，手套。或者至少把所有的孤儿都藏起来，这样当你穿衣服的时候，他们就不会在精神上拖你的后腿。把它们放在一个鞋盒里，我们让你扔的时候你没有扔。* 

*同样适用于**劣质毛巾、毛巾或亚麻布**。便宜的毛巾比脏毛巾好。它诚实、端庄、谦逊。* 

*品牌垃圾:现在我们进入了“精神混乱”的领域:工作正常但让你的家变得不那么舒适的东西。冰箱贴、咖啡杯、t 恤衫和其他带有你不关心的公司标志的东西。我最近收到了一封邮寄的品牌创可贴。多么难以置信的无用！如果你有足够多的非品牌替代品，扔掉品牌的。如果你没有，扔掉那些有品牌的，买新的。* 

***衣架:**如果你不在乎你的衣架是塑料的还是金属的还是木头的，那就都留着吧！如果你暗地里讨厌你的一半衣架，那就换掉它们。要求你做[](https://www.youtube.com/watch?v=tUkE9qaVgmo)*场景。把它放大。探索太空。** 

**杂志:这会很痛，但感觉很好。我们曾经扔掉了四年的卷了角的《T2》《纽约客》,我们在房子里飞来飞去，像神一样自由。现在就做，否则你会后悔不不不做*不*用有希望的封面打开问题*不！*** 

****去年的小玩意配件:**不适合你新机型的外壳和镜头、支架。我们正步入“捐赠吧”的领地，但即使是拥有旧手机的人也想要新手机。不要用你用过的东西侮辱他们。** 

****默认耳机:**保留一副备用耳机。扔掉剩下的。** 

**有缺口的盘子:有些盘子是成套的，你保留那些有裂纹和缺口的，因为你不是在白宫管理瓷器。但是如果你有大量的咖啡杯，扔掉那些已经散架的。人生苦短，不能用有缺口的杯子喝水。** 

**如果这很容易，现在扔掉所有坏掉的东西。如果你现在不愿意修复它*，你将永远无法修复它。*** 

***任何你有一百个的东西:所有那些你一辈子都用不完的东西，它们有办法埋葬你所有的东西。回形针，火柴盒，橡皮筋。你可能五年后就用完了，你会买更多，你会过得很好，与此同时，你的垃圾抽屉里会有更多的空间。这看起来很浪费，但和旧电线一样，这实际上是为你的真正物品腾出了存储空间。*** 

***你父母从沃尔格林超市给你的奇怪的随机礼物:没人想要。善意都不要。商店甚至不想要它，这就是为什么你的父母清仓时得到它。*** 

***现在，如果你还有精力(哇，看看你！)，你可以尝试真正难的东西，比如你永远不会读的书，不合身的衣服，还有你孩子做的东西。或者看看 [这张长长的 PopSugar 列表](https://www.popsugar.com/smart-living/List-Things-Dispose-24575006) 中你可以扔掉的其他东西。或者去处理你的收件箱。***

***或者只是放松！你做得太好了！我们骄傲，甚至嫉妒！邀请某人过来，展示你的地方，他们会留下深刻的印象，他们会说他们永远不会做得这么好。你会不同意，就在昨天你还是一个像他们一样迷失的灵魂，但是他们可能会像你一样转变！“现在去做同样的事，”你会说，他们会，祝福会倾注在你们两个身上。***
