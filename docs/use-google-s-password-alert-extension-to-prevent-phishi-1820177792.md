# 使用谷歌的密码提醒扩展来防止网络钓鱼

> 原文：<https://lifehacker.com/use-google-s-password-alert-extension-to-prevent-phishi-1820177792>

网络钓鱼攻击变得越来越难以识别，在这种攻击中，骗子伪装成合法公司，诱骗您交出用户名、密码或信用卡信息等敏感信息。甚至谷歌也容易受到更复杂的攻击，从电子邮件附件到共享的谷歌文档，这些攻击无处不在。



如果你是谷歌 Chrome 用户，你可以通过谷歌自己的一个简单扩展更安全地保护自己免受的网络钓鱼攻击。该公司为谷歌 Chrome 浏览器 开发的 [密码提醒扩展旨在防止网络钓鱼诈骗在你察觉之前危及你的账户。如果你有哪怕是最轻微的恐惧，有人可能会试图闯入你的帐户(](https://chrome.google.com/webstore/detail/password-alert/noondiphcddnnabmjcihcjfbhfklnnep) [你应该](https://lifehacker.com/search-for-your-email-address-to-see-if-your-password-h-1819780168) )，这个工具应该是你的武器库的一部分。

密码警报的工作原理是在浏览器中临时存储一个修改过的、部分随机化的密码版本。每次你输入一个密码，扩展就会将你的密码与其保存的版本进行比较，并检查该网站是否是一个合法的 Google 登录页面。根据谷歌的说法，没有键盘记录，这意味着你不必担心当你输入密码时，分机会监视你的输入。密码提醒也适用于同一谷歌 Chrome 浏览器上的多个配置文件，因此您可以保护您的工作和个人账户。

如果密码提示检测到一个潜在的钓鱼网站试图窥探您的个人 Google 帐户信息，您将收到一个提示，要求您重置密码以保护自己，或者选择忽略该提示。

密码提醒是一个非常需要的额外安全级别，特别是如果你还没有将你的帐户与谷歌认证器或 Authy 等双因素认证应用程序配对。当然，短信验证是一个开始，但总有你的号码被劫持的风险，如果发生这种情况，你就很不走运了。
