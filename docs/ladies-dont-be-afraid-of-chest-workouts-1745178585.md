# 女士们:不要害怕胸部锻炼

> 原文：<https://lifehacker.com/ladies-dont-be-afraid-of-chest-workouts-1745178585>

包括专门的胸部例行锻炼将改善你的上身力量，姿势和信心。不幸的是，许多女士基于被误导的观念，对恐惧给胸部带来的好处不以为然。以下是你需要知道的关于胸部锻炼的知识，以及它们是如何影响你的——咳咳——胸部的。

首先，快速解剖课:你的胸部由两块主要肌肉组成，称为胸大肌和胸小肌(我们简称它们为胸肌)。那些赤膊的男性健身模特有着巨大的金刚胸，他们的胸肌非常发达，是两个人中肌肉比较大的一个。是的，我们女人也有同样的胸肌！

正如你在镜子里可以清楚看到的，虽然，他们没有那么明显。它们也被我们可爱的女性肿块所覆盖，这些肿块实际上由脂肪和腺组织、导管和淋巴结网络、神经末梢和血管组成。它们的主要功能当然是分泌乳汁喂养婴儿。在健身方面，他们做的不多( [除了有时候硬起来](http://vitals.lifehacker.com/why-your-nipples-get-hard-during-a-workout-and-what-to-1732449052) )。

## **别担心，不是胸功让你的胸部缩水**

许多女性回避胸部特定的运动，因为她们认为她们的乳房会变小。你们中的一些人窃笑，但这种恐惧听起来很像我们最初的 [担心在力量训练](https://lifehacker.com/skip-the-tiny-weights-weight-lifting-isnt-that-differe-1724484355) 时变成巨大、肌肉发达的坦克；换句话说，他们似乎是基于一些关于力量训练的错误观念。

现在，跟我重复:单独做一堆胸部运动不会(或者说，不能)缩小你的胸部。

他们不能因为这样的事情会暗示有可能 [瞄准并消灭你身体特定区域的脂肪](http://www.yalescientific.org/2011/04/targeted-fat-loss-myth-or-reality/) ，或者“斑点减少” [，我们现在知道这些都是假的。相反，完全不同的事情正在发生:缩小的乳房是你全身脂肪细胞都在缩小的标志。](http://vitals.lifehacker.com/crunches-alone-wont-give-you-a-six-pack-the-myth-of-sp-1681379949)

回想一下，你的乳房主要是两块产奶脂肪组织，其大小在很大程度上取决于遗传、激素等诸如此类的因素。如果你以有利于减肥的方式训练和饮食，你会减掉全身的脂肪，这不可避免地会导致更小的腹部、手臂，甚至是胸部。真的，你无法控制脂肪从身体的哪个部位吸收。此外，如果你一开始就有较高的体脂，你可以预期总体上减少更多的脂肪， [可能一开始速度更快，](http://vitals.lifehacker.com/unsexy-realistic-expectations-for-healthy-weight-loss-1723469623) 也是如此。

## **胸部锻炼的好处**

所有这些关于锻炼和缩胸可能性的讨论可能会让你跑步——呃，走路——爬山。但是在光谱的另一端，你可能听说过胸部运动也有其额外的好处——从字面上看，胸部运动可以让女士们“更有精神”

这在某种程度上是对的，但这并不意味着你的乳房已经神奇地开始反抗地心引力了。根据的说法，真正发生的是下面的一个一二三组合拳:

1.  常规的胸部训练有助于锻炼你的胸肌，但是因为你的胸部和周围的脂肪组织掩盖了它们，所以你看不到那些肌肉。
2.  随着体重和胸部脂肪的减少，你实际上暴露了更多的胸大肌，让你的胸部更加突出。
3.  随着肌肉变得越来越大，它们似乎抬起了乳房，所以它们在你的胸部上坐得更高一些。

这些因素的结合让乳房看起来更“自信”。换句话说，胸部锻炼有助于锻炼你的胸肌，并能间接改变你胸部的外观。当然，这种影响可能或多或少，取决于它们的大小。

## **强健的胸部不仅仅是为了胸部隆起**

除了美学，胸部锻炼还能改善投掷或攀爬等运动，甚至是许多你不知道的涉及胸肌的日常运动。你的胸肌负责帮助施加“水平推力”,比如推婴儿车或推沉重的门。除了推东西，发达的胸部:

*   改善你的姿势。只要想一想人们常说的“挺胸自豪”这句话，你就能想象一个强健的胸部，以及同样强健的肩膀和背部， [如何改善你的体态](http://robbwolf.com/2015/09/09/better-posture-why-shoulders-down-and-back-isnt-the-best/) 。力量和体能教练兼运动专家凯特·加里特补充道:“你会有一个更加‘堆叠和排列’的骨架，自然会站得更高，举得更高。”她说，这也有助于改变你的胸部外观。
*   保持整体“平衡”的身体。类似于上一点:如果你相当平等地发展你的主要肌肉群，你会看起来更“对称”。你会有一个看起来更丰满的胸部(因为胸肌伸展到你的肩膀前面，一直到你的锁骨),以补充你的手臂和肩膀。此外，你还会有很多业内人士所说的“肌肉平衡”，这基本上意味着你没有一个主导肌肉群来过度补偿较弱或不活跃的肌肉群。这可以让你更好地整体移动，并安全有效地进行其他 [练习](http://vitals.lifehacker.com/how-to-minimize-your-risk-of-injury-when-lifting-heavie-1731253133) 。
*   让你像放克一样强壮。我还没有一个单身女性朋友告诉我，“上身力量？哼，不用了，谢谢。”坦率地说，无论你的目标是做什么，一个更强壮的胸部会帮助你做得更好——无论是做无膝俯卧撑；或者 [甚至是举着某些烂瑜伽姿势的](http://vitals.lifehacker.com/how-to-turn-yoga-into-badass-exercise-with-advanced-pos-1732317425) 。

但是，假设你完全不训练你的胸部，而仍然训练其他主要肌肉群，比如你的背部、腿部和肩部——最坏会发生什么呢？嗯，我自己也很好奇，问了詹妮弗·布雷克、力量机和私人教练。她说:

> 如果一个女人完全忽视胸部训练，这意味着她没有做任何上半身水平推动，而最糟糕的是她错过了许多有趣的练习，这些练习将增加她的训练和上半身的形状。当涉及到加强上身时，胸肌起着非常重要的作用。事实上，你不能在不针对胸肌的情况下增加你的水平推进力度(想想俯卧撑，卧推，单臂哑铃卧推)。

这并不是说你需要打破卧推世界纪录，但事实是:如果你不加强你的胸部，你就阻碍了你获得相当大的上身力量。
