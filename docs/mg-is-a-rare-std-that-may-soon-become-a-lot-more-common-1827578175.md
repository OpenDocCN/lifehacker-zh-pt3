# MG 是一种罕见的性病，可能很快会变得更加常见

> 原文：<https://lifehacker.com/mg-is-a-rare-std-that-may-soon-become-a-lot-more-common-1827578175>

好像担心的性传播疾病还不够多似的，*生殖支原体*现在正受到政府官员更严密的审查。一个英国性健康组织 [发布了新的指导方针](https://www.bashh.org/news/news/bashh-launches-new-nice-accredited-guidelines-to-help-prevent-mycoplasma-genitalium-becoming-the-next-superbug-but-funding-cuts-may-hinder-implementation/) 来阻止它成为下一个超级细菌，我们至少可以意识到它的存在。

这种细菌——简称 MG——在英国感染了大约 1%的人，2007 年的一项研究 在美国的年轻人中发现了类似的比率。这听起来可能很罕见，但它比你肯定听说过的淋病的发病率高。

MG 一直不为人所知，因为它的症状很模糊，许多人不知道他们有这种病。它会导致阴道刺激、骨盆疼痛、排尿疼痛以及分泌物或出血。这些症状都不会让你的提供者说“哦，是的，你有 MG”，因为它们都有其他原因，包括衣原体。而有些人根本没有任何症状。但是，如果不进行治疗，它 [可能会引起盆腔炎](https://www.self.com/story/mycoplasma-genitalium) ，从而导致不孕。

直到最近，甚至还没有一种商业化的测试方法。英国性健康和艾滋病毒协会表示，所有这些因素结合在一起，使这种隐性感染经常被滥用，增加了它对抗生素产生耐药性的机会。换句话说，除非提供者更加意识到这一点，检测变得更加可行，否则它可能变得更容易被发现，更难治疗。

现在，你我所能做的就是意识到这种性病已经存在。如果您有与 MG 症状相符的不明原因的疼痛，请向您的提供者提出这种可能性，并询问是否适合做 MG 测试。
