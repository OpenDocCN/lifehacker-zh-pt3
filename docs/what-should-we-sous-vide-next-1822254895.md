# 接下来我们应该做什么？

> 原文：<https://lifehacker.com/what-should-we-sous-vide-next-1822254895>

各位苏维的朋友、粉丝、家人大家好，欢迎回到另一个为 [*选话题的环节，它会苏维吗？*](http://lifehacker.com/tag/will-it-sous-vide) ，列在那里我用我的沉浸循环器做任何你想让我做的事。



上周的块根蔬菜和果汁实验相当成功，但现在我很想吃点海鲜。我们已经研究了虾，但我认为是时候研究其他甲壳类动物了。一些想法:

*   扇贝:这是出了名的容易煮过头，是理想的选择。
*   龙虾:我真不敢相信我们还没有这样做过。如果你们对户外龙虾或者卷尾有什么想法，我很想听听。
*   小龙虾:我们可以做小龙虾，但我认为我们必须用不同的温度来做土豆和玉米。
*   海鲜茶碗木石:我们已经做了以甜蛋羹为主的菜肴，为什么不试试我们的 [美味的日本蛋羹](https://en.wikipedia.org/wiki/Chawanmushi) ？

和往常一样，请在评论中自由自在地写下你的想法，并提出你脑海中可能出现的任何视频实验。
