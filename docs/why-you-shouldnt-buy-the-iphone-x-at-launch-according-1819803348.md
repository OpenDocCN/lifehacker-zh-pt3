# 苹果公司的一位创始人说，为什么你不应该在 iPhone X 发布时购买它

> 原文：<https://lifehacker.com/why-you-shouldnt-buy-the-iphone-x-at-launch-according-1819803348>

[iPhone X](https://lifehacker.com/what-you-need-to-know-about-face-id-on-the-iphone-x-1804641406) 即将上市，但苹果最大的支持者之一对这款新手机并不兴奋。 [史蒂夫·沃兹尼亚克](https://lifehacker.com/how-apple-co-founder-steve-wozniak-gets-things-done-5222989) ，这位自 1985 年离开公司以来一直在场外欢呼的苹果联合创始人，在接受采访时表示，他不会在第一天拿起 iPhone X。

如果你对苹果的新手机一直持观望态度，沃兹尼亚克的决定可能足以让你崩溃。以下是他对拒绝苹果最新产品的解释:

### 为什么沃兹尼亚克不急于购买 iPhone X

这位沃兹(他的朋友和粉丝都这么称呼他)没有提供他对 iPhone X 缺乏兴趣的具体原因。他指出，他已经有了一部 iPhone 8，他喜欢这部手机，尽管他承认这部手机与之前的 iPhone 7 和 iPhone 6 非常相似。所以有可能他只是觉得今年不值得再升级一次。

“出于某种原因，iPhone X 将是我第一天没有升级到的第一部 iPhone，”沃兹尼亚克告诉美国消费者新闻与商业频道，“但我的妻子会，所以我会离得很近，可以看到它。”

这位苹果联合创始人还担心，Face ID——取代 iPhone X 上触控 ID 指纹扫描仪的新面部扫描技术——在发布时可能不会完美运行。鉴于苹果公司推出故障新功能的记录，这是一个合理的担忧，尽管如果 Face ID 没有完全做好市场准备，该公司似乎不太可能冒这样一个巨大的错误。

沃兹尼亚克过去也改变了对苹果产品的看法。 [去年](http://www.techradar.com/news/phone-and-communications/mobile-phones/steve-wozniak-apple-was-right-to-drop-iphone-7-headphone-jack-1330152) 他警告苹果不要移除 iPhone 7 上的耳机插孔，但在手机正式公布后又出来支持这一决定。因此，只要 Face ID 不完全是一场灾难，我们很快就有可能看到沃兹拿起 iPhone X。

与此同时，你可能想接受沃兹尼亚克的建议，转而购买 iPhone 8。
