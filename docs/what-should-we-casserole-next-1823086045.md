# 接下来我们应该做什么菜？

> 原文：<https://lifehacker.com/what-should-we-casserole-next-1823086045>

大家好，欢迎回到 [*将它砂锅*](https://lifehacker.com/tag/will-it-casserole) ，在这个栏目里我们看到哪些饭菜、菜肴、菜系可以重新想象，改造成砂锅。

Watch

上周的墨西哥卷饼启发，分层喜悦终于得到了它的一天，这是辉煌的，但现在我们需要转移到一个新的热菜。在上次讨论中，你们都提出了一些很好的想法，其中一些是我最喜欢的:

*   **美食广场砂锅:** [我一直在斗胆](https://skillet.lifehacker.com/what-should-we-casserole-next-1822675587) 做一道菜，灵感来自斯巴罗、熊猫快餐、查理芝士牛排、Cinnabon、玉米热狗、软椒盐卷饼和焦糖爆米花。这看起来有点过分，但是我想我可以以一种连贯的方式完成五个左右的作品。
*   芝士牛排砂锅:我只是需要费城的人告诉我用哪种芝士，因为我的冲动是用 provolone *和* Cheese Wiz。
*   **韩国牧羊人馅饼:**基于 [galbi-jjim](https://en.wikipedia.org/wiki/Galbi-jjim) ，这将包括红烧短排骨、胡萝卜、香菇、洋葱、栗子和大枣，并在上面浇上松软的土豆泥。
*   奶酪盘砂锅:基本上是大量的烤布里干酪，一些山核桃，一点蜂蜜，也许还有一些烤苹果梨。
*   **宜家砂锅菜:**这包括肉丸、土豆泥和我能在他们的食品市场找到的任何其他东西。

一如既往，请随意向我抛出你自己的绝妙想法，不管你认为它们有多疯狂。你们中的几个人上周提到了“热狗”，但如果你们能在这一点上有所扩展，那将会很酷。
