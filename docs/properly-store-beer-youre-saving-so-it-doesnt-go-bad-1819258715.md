# 正确储存你保存的啤酒，这样它就不会变坏

> 原文：<https://lifehacker.com/properly-store-beer-youre-saving-so-it-doesnt-go-bad-1819258715>

啤酒，就像大多数其他食品和饮料产品一样，有保质期，以及正确和不正确的储存方法。当你最终打开酒瓶时，你选择在哪里和如何储存你的啤酒会对它的味道产生巨大的影响。

对这个问题的快速回答是，你要么把啤酒储存在冰箱里，要么储存在阴凉、黑暗的地方，比如壁橱或地下室的角落。啤酒的三个敌人是氧气、热量和光线。一般来说，只要有可能，你应该让你的啤酒远离这三样东西。

以下是原因的简要概述:

### 热

有一种误解认为所有的啤酒都需要放在冰箱里。虽然是的，如果你想喝，那么你可能希望它是冷的，如果今天不喝，你就不必把它放在冰箱里。

你可以把啤酒在室温下保存一两周，不会对它造成任何损害。虽然这可能会延长到一个月，但一旦你的啤酒开始长时间搁置，味道就会开始以你不喜欢的方式发生变化。

理想情况下，你应该把啤酒储存在华氏 35-45 度之间。你的储藏室越暖和，里面的味道就会越浓。你永远也不想在室温以上储存啤酒。炎热的一个下午会让你的啤酒味道发生快速(而不是美味)的变化。

这意味着夏天去商场购物时，不要把啤酒放在行李箱里，或者把它放在你家里阳光充足的地方。说起孙……

### 光

光也是如此。啤酒暴露在阳光下(和荧光灯下)，即使是很短的时间，也会显著改变它的味道，而不是变得更好。啤酒包装有助于这一点(这就是为什么你的啤酒装在黑色的瓶子或罐子里)，但这并不意味着你不应该尽可能地把它放在阳光下。

棕色玻璃瓶能够阻挡 98%来自太阳的有害波长。绿色瓶子只能阻挡 20%的有害射线，并且可以在 5 分钟或更短的时间内变暗，而透明瓶子(显然)阻挡 0%的光线，并且可以在阳光下几分钟内变暗。

### 氧

我们可能都在小学学过，如果你把一罐汽水开一夜，它会失去一些碳酸化作用。除了开着罐头，氧气还会进入瓶装或罐装的啤酒中。是的，酿酒厂使用的机器有助于消除大量氧气，但氧气仍然会存在。

啤酒中的氧气会有很多负面影响。也就是说，它会在你的啤酒中产生一些“异味”，这是啤酒酿造者不想要的，也是你不想要的。与陈啤酒相关的味道可能包括蜡底或口感、纸板或纸的味道，以及暗淡的啤酒花味道(对于啤酒花来说)。

大多数啤酒厂会在瓶子上贴上“最佳食用”日期，让你知道啤酒的味道什么时候会变得不适合饮用。如果你的啤酒没有日期，假设它在装瓶后的三个月内仍然有效。在绝大多数情况下，你绝对不会想喝超过 6 个月的啤酒。

说到储存啤酒，没有什么比确保啤酒不会在冰箱后面丢失和过期更重要的了。此外，确保你留着以后喝的啤酒上没有“最佳”日期。这些年来，我有几十个朋友把啤酒保存了好几年，这些啤酒本来是要立即饮用的。不要成为那样的人，因为等了太久才喝，而错过了很棒的啤酒。
