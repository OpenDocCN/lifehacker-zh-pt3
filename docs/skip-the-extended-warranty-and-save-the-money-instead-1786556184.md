# 跳过延长保修，节省资金

> 原文：<https://lifehacker.com/skip-the-extended-warranty-and-save-the-money-instead-1786556184>

如果你从一家信誉良好的公司购买产品，可以随意跳过零售商的“延长保修”——你可能已经在制造商的保修范围内，潜在的维修费用可能比你想象的要低。

### **不要为您不需要的保险付费**

根据消费者报告，零售商保留这些合同收费的 50%或更多，这是一个非常高的利润——远远超过他们自己销售产品的利润。

美国消费者联合会的执行董事杰克·吉利斯 说:“延长保修期通常是一笔糟糕的交易。”。“首先，它们不是保证书，人们不会得到与保证书相同的法律保护。它们是服务合同，或者更确切地说，是定价过高的保险计划。”

话说回来，如果你真的想要额外的保护，那也没问题——只要确保*实际上*提供超出你可能已经拥有的额外保障是一笔好交易。

### **制造商的保修已经涵盖了更换**

许多购买的产品已经包含制造商的保修，如果你的产品有问题，将会修理或更换。零售商延期保修的问题是，它经常在相同的时间内重复相同的覆盖范围。更糟糕的是，延期保修有时只在后的*生效，你已经试图先通过制造商的保修找到解脱。阅读细则，确保延长保修实际上超过了你已经支付的保修范围(像苹果这样的公司有慷慨的两年保修，而其他公司可能只提供 30 或 90 天)。*

另外，如果你的产品坏了或不能用了，即使你没有任何形式的保修，要求更换也不会有什么坏处。如果你有收据，如果产品在不合理的短时间内损坏，许多公司会更换产品。

### **廉价产品可能不需要保险**

例如，像廉价电器这样的低成本项目的维修费用可能不会比服务计划高多少。根据 HomeAdvisor 的数据， [平均家电维修费用总计 171 美元](https://www.homeadvisor.com/cost/kitchens/repair-an-appliance/) ，如果延长保修期的费用比这还高，就不值得了，因为他们通常会做多年计划。

### **您的信用卡可能会为您提供保险**

如果可以的话，用你的信用卡支付产品，因为许多 [卡会自动延长制造商的保修期](https://www.nerdwallet.com/blog/the-official-nerdwallet-guide-your-credit-cards-extended-warranty-policy/) 长达一年。在这种情况下，延长保修是免费的。

### **从信任的公司购买产品**

如果你从一家声誉好的公司购买产品，你通常会买比廉价仿制品更耐用的东西。此外，这些品牌往往有更慷慨的制造商保修。也在网上寻找产品的评论——如果它经常出问题，你肯定会读到一些关于它的东西。

对于更多关于延期保修，FTC 已经提供了额外的指导， [这里](https://www.consumer.ftc.gov/articles/0240-extended-warranties-and-service-contracts) 。
