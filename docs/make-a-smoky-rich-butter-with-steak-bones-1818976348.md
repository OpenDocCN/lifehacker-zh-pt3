# 用牛排骨头做一道烟熏味浓郁的黄油

> 原文：<https://lifehacker.com/make-a-smoky-rich-butter-with-steak-bones-1818976348>

牛排一直是我最喜欢的蛋白质，我更喜欢带骨的那种。但是，一旦肉吃完了，马丁尼酒喝完了，扔掉骨头似乎总是有点令人难过，因为骨头太小了，做不出可观的肉汤。

我的意思是，你可以*攒下一大堆牛排骨头，最终做成肉汤——或者你可以把一根放入你通常的鸡汤中——但是，只要你没有啃过它，你就可以用那根孤零零的骨头给黄油注入肉味和烟熏味。*

除了骨头，你还可以用锅里的油滴来调味你最喜欢的奶酱。只需倒掉并保留烹饪油脂——在我的例子中，通常是鸭油——然后用一点葡萄酒给锅上光。(红色或者白色都可以，但是红色会大大影响颜色或者你的黄油，很明显。)

用木勺刮一下锅，全部倒进碗里。接下来，你必须决定如何提取所有的肉味。你可以在炉子上用一个简单的锅，一个慢炖锅，或者——我最喜欢的——[你可以用](https://lifehacker.com/c/will-it-sous-vide) 腌制。

无论你用什么方法，只需在你选择的烹饪器皿中混合锅里的水滴和美味的脱釉的碎片、骨头和一根黄油。如果你用的是塑料袋，那就把它封起来。

然后你会想用低火慢煮。对于 sous vide 来说，这意味着让所有东西在 185 度的水浴中浸泡两个小时，但你也可以很容易地将其放入慢炖锅中同样长的时间，或者将其全部放入调味汁锅中，盖上盖子，用小火煮几个小时。

经过几个小时的注入动作后，将热黄油倒入一个细滤网中，让多肉黄油在冰箱中变硬。正如你从上面的照片中看到的，这种烟熏的，富含鲜味的黄油非常适合放在饼干上，但它会被搅成土豆泥，拌上烤蔬菜，或者涂上厚厚的一层玉米穗。我敢打赌，和热意大利面一起搅拌会非常棒，特别是如果你在里面放一些烤蘑菇，并在上面撒上新鲜的胡椒粉。事实上，我想那可能是晚餐。
