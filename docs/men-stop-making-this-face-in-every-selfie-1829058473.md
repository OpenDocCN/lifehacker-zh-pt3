# 男人们，不要每次自拍都做这个表情

> 原文：<https://lifehacker.com/men-stop-making-this-face-in-every-selfie-1829058473>

为什么那么多男青年在所有的自拍里都是一个模糊的脸？为什么他们都扬起眉毛，紧闭双唇，就像上班族在星期二互相问候一样？他们在隐藏什么？他们感觉如何？ [Vox 问了其中一小撮](https://www.vox.com/the-goods/2018/9/14/17827822/selfie-trends-men-male-facial-expression) ，得到了一些极好的自嘲答案。例如，自拍爱好者马克斯·莱维兹克说:

> 扬起的眉毛表示，“哦，哇，你让我措手不及！”哈！“哦，我的照片？”这实际上是主体证明他们正在自拍的一种方式。假笑就像是，‘我在学校太酷了，但我还是会拍这张自拍，因为嘿，我是个有趣的人。’。



所有的答案在一个主题上有所不同:男人在自拍时会感到尴尬，他们会选择最大限度减少这种尴尬的面孔。但是如果你曾经和一个试图隐藏自己缺点的人相处过，你会发现这只是吸引了你的注意力。一个试图在自拍中显得“受人尊敬”的男人看起来比一个只是为了自拍的人更尴尬。

公平地说，男人确实会因为做某些鬼脸而受到惩罚。这与女性因其表情而受到的惩罚相去甚远——大多数男性从未被陌生人要求“多笑笑”，或被指责“休息的婊子脸”，或因为在 Tinder 上拒绝某人的追求而被称为丑陋的婊子——但事实是，男性经常被嘲笑为 [敢于在图片](https://knowyourmeme.com/memes/nu-male-smile) 中张开嘴。右翼批评者称之为“男性化的微笑”，并声称这是一个没有男人味的“大豆男孩”的标志

再说一次，这跟女人面对的事情没什么两样，所以伙计们，坚强起来。但也要记住，那些称你为“大豆男孩”的人是不知道快乐的纳粹崇拜者。他们的英雄是乔丹·皮特森，一个被遗忘的偏执的教授，他认为 [假扮](https://merionwest.com/2018/06/08/part-two-the-argument-against-jordan-peterson/)[皱眉](https://www.theatlantic.com/ideas/archive/2018/08/why-the-left-is-so-afraid-of-jordan-peterson/567110/) 将 [使](https://nationalpost.com/news/canada/could-jordan-peterson-become-the-best-selling-canadian-author-of-all-time) 他看起来 [聪明](https://www.washingtonpost.com/blogs/post-partisan/wp/2018/05/08/jordan-peterson-comes-to-town-a-conversation-between-skeptics/?utm_term=.a02c4293a13f) 。

我不是说你应该咧开嘴笑，尽管嘿，至少这是一件事。尝试一个正常的露齿微笑，或真的假笑，或看起来非常怀疑，或充满希望，或幸福。试着用一种表达方式，不是为自己被拍到而道歉。

尝试和放松。拿一个“傻的。”做鬼脸。这些图片是免费的。你在 Snapchat 或 Instagram 或 Twitter 或 Tinder 上展示的面孔越多，任何一张照片就越不会破坏别人对你的印象。就像，一张令人毛骨悚然的 [诺曼·贝茨的脸](https://tvtropes.org/pmwiki/pmwiki.php/Main/KubrickStare) 当它被你傻傻的、深思熟虑的、严肃的、英俊的脸包围时，就不那么令人毛骨悚然了。这些表达可能仍然是玩笑和讽刺性的，但它们有了*的一些*含义，而不是“对不起，我在自拍”而且拍的越多，越能习惯自拍，直到不再难受。

自拍不丢人。每个人都喜欢面孔。你的朋友想看你的。给他们玩，而不是给想象中的(或真实的)无聊到在网上取笑你的脸的陌生人玩。

或者干脆不发自拍，那也行！或者继续做那个表情，如果那是你觉得舒服的，但是要知道你错过了什么。我想我要说的是，嘿，伙计们...你笑起来更漂亮。
