# 试试这个军事冥想程序，快速入睡

> 原文：<https://lifehacker.com/try-this-military-meditation-routine-to-fall-asleep-fas-1828661790>

需要快速入睡？美国海军飞行前学校开发的一种中世纪放松技术，据说在几分钟内就能奏效。这可能不是魔术，但这似乎是放松身心的好方法。



在 [*放松并赢得*](https://www.amazon.com/gp/product/0498025411/ref=as_li_tl?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/try-this-military-meditation-routine-to-fall-asleep-fas-1828661790&asc_source=&camp=1789&creative=390957&creativeASIN=0498025411&ie=UTF8&linkCode=as2&linkId=FSXA3GZUT5SFDAZ4&tag=kinjalifehackerlink-20) 和 [中描述的基础知识，在 Medium](https://medium.com/s/story/combat-tested-training-unwind-and-sleep-anywhere-in-120-seconds-27d5307b7606) 上总结，是一部分一部分地放松你的身体，然后尝试在十秒钟内完全清理你的头脑。这里有一个脚本:

*   放松面部肌肉，包括舌头、下巴和眼睛周围的肌肉。
*   尽可能放低你的肩膀。然后放松一侧的上臂和下臂，然后放松另一侧。
*   呼气，放松你的胸部。
*   最后放松双腿，先大腿再小腿。

花大约一分半钟的时间来完成这个清单，完全放松身体的每一个部位，然后试着用十秒钟来理清你的思绪。正如《男子气概的艺术》所报道的，巴德·温特(该项目的设计者)建议你从这些心理图像中选择一个:

*   躺在平静湖面上的独木舟里，头顶上只有蓝天。
*   在漆黑的房间里依偎在黑色天鹅绒吊床上
*   一遍又一遍地说“不要想，不要想，不要想”十秒钟。

如果这听起来很熟悉，那是因为类似的顺序——一部分一部分地放松你的身体——是几乎所有 [睡眠冥想](https://vitals.lifehacker.com/how-to-meditate-in-bed-without-using-your-phone-1828094298) 轨迹的基础。那么，这有用吗？几乎可以肯定是的。

但这一招真的是最好或最快的入睡方法吗？现在的空军网站 [有一堆睡眠建议](https://www.airforcemedicine.af.mil/News/Article/688911/sleep-is-serious-catch-your-zzzs/) ，没有提到具体的、强制性的两分钟脚本。与此同时， [军队建议](https://www.army.mil/article/186260/healthy_sleep_habits_are_you_practicing_them) “获得宁静睡眠的底线是做对你有益的事情。除了听从自己的身体，没有什么神奇的公式。”
