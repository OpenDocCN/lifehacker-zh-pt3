# “想要是你头脑提供的一个选项，而不是你必须遵循的一个命令”

> 原文：<https://lifehacker.com/wanting-is-an-option-your-mind-provides-not-an-order-1735293027>

“想要”和“需要”之间有着本质的区别，这都归结于选择。尽管击退你的欲望看起来很困难，但提醒自己你才是发号施令的人是有帮助的。

Watch

在他的博客上， [James Clear](http://jamesclear.com/) 建议，采取积极措施减少你的“无疑问消费”,同时找到一种方法减少消费者的欲望，这会让你的生活变得更好。要做到这一点，你必须正确看待事物，认识到什么是真正的想要:

> 放下想要的东西。永远不会有一个水平让你不再想要什么。总有可以升级的地方。买辆新本田？你可以升级到奔驰。买辆新奔驰？你可以升级到宾利。买辆新宾利？你可以升级到法拉利。买辆新法拉利？你想过买一架私人飞机吗？意识到想要只是你头脑提供的一个选项，而不是你必须遵循的命令。

底线是你选择想要的东西。你不会选择需要什么。需求*是*的命令。当你开始把想要当成一种选择，而不是你无法控制的原始欲望时，你就能防止自己做出错误的选择。渴望并不总是坏事，但它只会让你表现出对它的控制。
