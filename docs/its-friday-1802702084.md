# 今天是星期五

> 原文：<https://lifehacker.com/its-friday-1802702084>

在我们提供的最简单的生活帮中，我们自豪地通知您，今天是星期五。正如我们博客科学指出的， [昨天是周四](https://www.thecut.com/2017/09/why-a-short-week-makes-it-so-hard-to-remember-what-day-it-is.html) 。更有帮助的是，他们解释了为什么在短时间的工作周内很难记住每一天，就像我们正在结束的这一周。

Watch

社会心理学家安妮·威尔逊博士在接受《美国科学》采访时表示:人对时间的感知是有可塑性的，并会受到五天工作制等因素的严重影响。对你的大脑来说，“星期一”或“星期三”的概念不仅仅是五次中的一次，而是一系列相关的情绪和想法，比如“中度疲劳”或“我通常想吃泰国菜的那一天”当你有不寻常的一天时，它会打乱那些关联，暂时打乱你大脑内部的日历。

所以忘记星期几是正常的，很酷，这并不意味着你是这个*太空幽灵海岸到海岸*剪辑中的胡言乱语的白痴。
