# 自己制作礼品卡，确保它们能被使用

> 原文：<https://lifehacker.com/make-sure-gift-cards-get-used-by-making-them-yourself-1820945598>

恐怖故事时间:你给某人一张贴心的礼券，也许是为了在他们心仪的温泉浴场享受一次按摩，或者是为了参加他们一直想参加的绘画班。你拿出现金，把卡给他们。他们很激动！他们感谢你。然后，他们不使用它。



我经历过这场恐怖秀的两方面，双方都感觉很糟糕。送礼者感到沮丧，因为他们花的钱没有被使用；接受礼物的人会因为不够热情、不够有组织、不够勇敢或其他什么而感到内疚。当然，赠送礼物的目的是*赠送*，但是能不能少一点浪费？

是啊！从由来已久的手绘优惠券传统中汲取灵感，但不是“一个免费的拥抱”，而是为你已经购买了礼券的东西写一张优惠券。“适合一堂绘画课”或“适合一次按摩(由专业人士进行，我会付费)。”

这对亲密的朋友和爱人最有效——你的丈夫比你的同事更有可能说“嘿，我找到了一个我想预订的烹饪班”——但不管怎样，这些人通常会让你非常害怕没有使用你周到、慷慨的礼物。这样的话，礼物还是很贴心的，但是不会浪费。
