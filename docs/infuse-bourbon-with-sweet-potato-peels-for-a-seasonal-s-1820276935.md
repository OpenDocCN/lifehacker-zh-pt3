# 在波旁威士忌中加入红薯皮，作为季节性饮品

> 原文：<https://lifehacker.com/infuse-bourbon-with-sweet-potato-peels-for-a-seasonal-s-1820276935>

“普通”白土豆的皮可以相对容易地制成美味、香脆的小吃，但无论你怎么处理，红薯皮都有点纤维状，难以下咽。然而，它们确实有一点红薯，可以用来泡一瓶波旁威士忌。



*这是《与克莱尔一起吃垃圾》的一部分* [*，这是一个生活黑客系列，克莱尔·劳尔说服你将你的厨余转化为可食用的美味。*](http://lifehacker.com/tag/eating-trash-with-claire)

[就像之前的菠萝皮](https://skillet.lifehacker.com/make-pineapple-infused-gin-with-the-leftover-peels-like-1795233709) ，用红薯皮来注入一种精神是再简单不过了。你只需将(洗过的)果皮放入一个玻璃罐中，在上面放入足够的酒——在这种情况下是波旁酒——盖住，让它们在室温下静置几天，每天摇晃几次罐子。你不需要用你的波旁威士忌——野生火鸡就可以——你甚至不需要用波旁威士忌。如果你问我，我觉得红薯朗姆酒听起来还不错。

如果你有一个真空系统，这个过程会更快。只需将罐子放在 150 度的浴缸中几个小时，让它冷却，然后通过衬有粗棉布( [或手帕](https://skillet.lifehacker.com/you-can-use-a-handkerchief-as-cheesecloth-1819621778) )的细网筛将秋天的浸液倒入另一个罐子或瓶子中。用它来做一杯传统的节日酒，或者用一大块岩石做成的玻璃杯啜饮。
