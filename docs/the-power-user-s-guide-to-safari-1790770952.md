# Safari 高级用户指南

> 原文:[https://life hacker . com/the-power-user-s-guide-to-safari-1790770952](https://lifehacker.com/the-power-user-s-guide-to-safari-1790770952)

苹果的 Safari 网络浏览器往往会受到不公正的指责，因为它非常无聊，而且每台 Mac 都有，但多年来，苹果已经悄悄地让它变得非常有用。另外，Safari 比你想象的更受欢迎。如果你现在正在 Safari 中阅读这篇文章，下面是如何充分利用你的经历。

谈到速度，这些天来浏览器大战一直很平淡。即使发生了变化，一款浏览器推出了必备的新功能，另一款也会更新得如此之快，以至于很难长期保持领先地位。所以，这一切都是关于功能，而 Chrome、 [Firefox、](https://www.mozilla.org/en-US/firefox/products/) 或 [Vivaldi](https://vivaldi.com/?lang=en_US) 强调定制，Safari 强调简单。不过，它确实有一些锦囊妙计，比如画中画模式和 Safari 阅读器。由于它是为你的 Mac 和操作系统设计的，它也可能会给你更长的电池寿命。就这样，让我们开始吧。

### 学习最省时的捷径

在大多数情况下，Safari 与其他所有浏览器共享 [键盘快捷键。](http://lifehacker.com/back-to-basics-learn-to-use-keyboard-shortcuts-like-a-5970089) ，但是如果你从来没有掌握使用键盘快捷键的窍门，现在是时候了。以下是我们在 Safari 中的一些最爱，其中大部分可以在任何其他浏览器中使用:

*   **Option+箭头键或空格键**:点击空格键或 Option+向上或向下箭头，以增量方式向下滚动网页，非常类似于 Windows 上的 Page Up 或 Page Down。
*   **Shift+Command+\** :在一个漂亮的网格视图中显示所有打开的标签。
*   **Shift+Command+Click** :在新标签页中打开一个链接，并使其成为活动标签页。
*   **Ctrl+Tab 和 Ctrl+Shift+Tab** :在所有打开的标签页间循环。
*   **Command+1 到 Command+9** :选择特定的选项卡，1 为第一个，2 为第二个，以此类推。
*   **Shift+Command+T** :重新打开你关闭的最后一个标签页或窗口。
*   **Esc** :如果你在 URL 字段/搜索框中输入，Esc 恢复当前网页地址。
*   Shift+Command+R: 以阅读器模式呈现网站，去掉广告和其他干扰。

当然，这只是一小部分快捷键，所以一定要看看这个 [的完整列表，里面列出了每一个 Safari 键盘快捷键](https://support.apple.com/kb/PH21483?locale=en_US) 。如果你每天都用鼠标做一些事情，很有可能有一个键盘快捷键可以让你做得更快。

### 使用画中画模式进行多任务视频观看

自从苹果开始每年免费发布其操作系统(和 Safari)的更新后，有用的新功能往往会在混乱中丢失。苹果在 macOS Sierra 中添加到 Safari 的最好的东西之一是画中画模式，可以缩小视频并在屏幕的一角播放。无论您是在笔记本电脑上工作还是在 27 英寸的 iMac 上工作，对于那些倾向于在后台处理视频的人来说，这都是一个非常棒的功能。

您可以通过两种方式之一调出画中画模式。如果一个网站支持画中画，比如 [Vimeo](https://vimeo.com) ，你会在视频的角落看到一个小的画中画图标。点击这个按钮，视频就会出现在屏幕的一角。如果你*没有*看到那个图标，就像在 [YouTube](https://www.youtube.com/) ， [右击视频两次](http://lifehacker.com/use-sierras-picture-in-picture-mode-on-unsupported-vide-1786898604) ，选择进入画中画。如果您不喜欢视频捕捉到屏幕角落的方式， [只需按住 Command 键](https://lifehacker.com/move-a-picture-in-picture-window-on-your-mac-anywhere-b-1790321561) 并点击它，将其移动到您想要的位置。

### 
管理标签中的声音

如果你在 Safari 中听音乐或看很多视频，你就会知道寻找播放声音的标签来静音是多么尴尬。Safari 有几个不同的工具来管理这个:

*   如果标签正在播放声音，您会在地址栏和标签上看到扬声器图标。点按地址栏中的扬声器图标，使每个标签中的声音静音。
*   按住 Alt/Option 并单击当前选项卡上的扬声器图标，使所有其他*选项卡上的声音静音。*
*   按住 Ctrl 并单击扬声器图标，以获得当前播放音频的所有选项卡的列表。

你再也不需要通过标签来寻找声音的来源了。

### 使用阅读器模式消除干扰，获得清晰的阅读体验

阅读器模式可以清除你正在阅读的文章中的所有杂物，这样你就可以专注于文本，而不用看广告或其他页面上的杂物。这与类似于 [Instapaper](https://www.instapaper.com) 或 [Pocket](https://getpocket.com/) 的服务类似，但不需要转到另一个网站或使用第三方服务。

如果您在阅读文章时在 URL 栏中看到一个四行图标，请点按它(或按下 Shift+Command+R)来触发阅读器模式。

你可以稍微定制一下它的外观。当你处于阅读器模式时，点击地址栏中的“aA”图标。在这里，您可以更改字体、字体大小和页面颜色，以更好地满足您的需求。

### 锁定你经常使用的网站，将其余的添加到你的阅读列表中

如果你每天访问一两个网站，比如你的电子邮件、脸书或 Twitter，那么你可能想把它们钉在你的标签栏上。为此，只需右键单击选项卡并选择“锁定选项卡”这将选项卡缩小到一个 favicon，并将其推到窗口的左侧。

至于你想保存下来以后阅读的单篇文章，Safari 有一个叫阅读列表的功能。阅读列表就像 Instapaper 或 Pocket，但在 Safari 内部，不需要外部服务或扩展。

要将页面添加到阅读列表，请点按书签菜单并选择“添加到阅读列表”，或者按下 Shift+Command+D。要查看阅读列表，请点按“显示”菜单并选择“显示阅读列表”，或者按下 Control+Command+2。默认情况下，您的阅读列表会在 Mac 上的 Safari 和 iPhone 或 iPad 上的 Safari 之间同步。

### 自定义工具栏

工具栏是 Safari 窗口的整个顶部。默认情况下，工具栏包括一堆你可能不想要的东西。你可以自定义它，但你认为合适。

右键单击工具栏上的任何空白区域，然后选择自定义工具栏。这将弹出定制面板。单击任何元素，然后将其拖放到工具栏中您希望它出现的位置。如果您想要移除某个项目，请点按它并将其拖离工具栏，然后放开。您也可以使用“灵活空间”项添加空白空间。

就我个人而言，我复制了 Chrome 的工具栏，去掉了侧边栏、分享和显示标签按钮，然后把我的扩展移到了地址栏的右边。

### 添加最佳扩展

Safari 没有 Chrome 或 Firefox 那么多扩展，但它拥有大多数流行的扩展。如果你只是想浏览，苹果的扩展库 有大量官方批准的扩展。这里有一些我们最喜欢的:

*   [**LastPass**](https://safari-extensions.apple.com/details/?id=com.lastpass.lpsafariextension-N24REP3BMN)**/**[**dash lane**](https://safari-extensions.apple.com/details/?id=com.dashlane.dashlanesafari-5P72E3GC48)**/**[**1 password**](https://safari-extensions.apple.com/details/?id=com.agilebits.onepassword4-safari-2BUA8C4S2C):你用哪个密码管理器都没关系， [只要用一个密码管理器](https://lifehacker.com/lifehacker-faceoff-the-best-password-managers-compare-1682443320) 。Safari 包括一项名为 iCloud Keychain 的服务，这是苹果在密码管理器方面的尝试。 [别用这个](http://lifehacker.com/stop-using-icloud-1788441810) 。iCloud Keychain 在其他操作系统上是不可用的，这意味着苹果将你和你所有的密码锁定在他们的生态系统中。没有办法导出它们。没有办法分享。请改用第三方密码管理器。继续操作 [将 iCloud 钥匙串完全关闭](https://support.apple.com/en-us/HT204085) 。
*   [**Pocket**](https://safari-extensions.apple.com/details/?id=com.ideashower.pocket.safari-ET279A6R5N)**/**[**Instapaper**](https://safari-extensions.apple.com/details/?id=com.instapaper.extension-CAM49M58WK):无论是 [Pocket 还是 Instapaper 都是很好的书签服务](http://lifehacker.com/read-it-later-showdown-pocket-vs-instapaper-1756021479) ，比 Safari 内置的阅读列表要健壮的多。有了 Pocket 或 Instapaper，你实际上可以在 Safari 之外组织和访问列表，这非常有用。
*   [**ublock**](https://www.ublock.org)/[**Disconnect**](https://disconnect.me/disconnect):如果你厌倦了广告、弹出窗口、脚本、追踪器或其他 [令人讨厌的互联网隐私侵犯部分](http://lifehacker.com/the-best-browser-extensions-that-protect-your-privacy-479408034) ，那么 Disconnect 或 Adblock Plus 值得安装。Disconnect 保护您免受跟踪、恶意软件和恶意广告的攻击，而 uBlock 也能做到这一点，而且它还能阻止广告，这一点 [比 AdBlock Plus](http://lifehacker.com/ublock-is-a-fast-and-lightweight-alternative-to-adblock-1625246461) 好得多。
*   [**nomore iTunes**](https://nomoreitunes.einserver.de)**:**每当你登陆一个应用程序的 iTunes 页面，Safari 就会令人讨厌地打开 iTunes 或 Mac App Store。没有官方方法可以关闭或切换这种行为。NoMoreiTunes 是一个第三方扩展，它关闭了这个可怕的功能。

许多其他流行的服务也有扩展，包括: [Evernote](https://safari-extensions.apple.com/details/?id=com.evernote.safari.clipper-Q79WDW8YH9) 、 [OneNote](https://safari-extensions.apple.com/details/?id=com.microsoft.onenote.clipper-UBF8T346G9) 、 [Bear](https://safari-extensions.apple.com/details/?id=net.shinyfrog.bearExtension-9K33E3U3T4) 、 [Todoist](https://safari-extensions.apple.com/details/?id=com.todoist.todoist-safari-S3DD273774) 、 [Wunderlist](https://support.wunderlist.com/customer/portal/articles/1767713) 、 [Honey](https://safari-extensions.apple.com/details/?id=forge.safari.honey985c8d42048c4cbeb0bed522bd235ee7-4NREU46PA6) 和 [Pinterest](http://www.pinterest.com/) 。

你也可以安装苹果官方页面 上没有出现的 [扩展，但是你需要手动双击扩展，然后点击“信任”按钮来安装，这可能会让一些用户感到不快。](https://support.apple.com/en-us/HT203051)

### 启用调试菜单

苹果喜欢把隐藏的调试菜单 [塞在终端命令](https://lifehacker.com/how-to-enable-the-debug-menu-in-a-bunch-of-apples-mac-a-1788596192) 后面，Safari 也不例外。打开终端(应用程序>实用程序)，然后输入并按回车键:

```
defaults write com.apple.Safari IncludeInternalDebugMenu 1

```

完成后，重启 Safari，你会发现一个调试菜单选项。以下是该菜单中一些值得研究的功能:

*   **在 20** 禁用每选项卡 Web 进程/限制每点击 Web 进程:打开这两个选项中的任何一个都会阻止选项卡在后台重新加载。虽然每个标签的设置对我们大多数人来说可能有点极端，但是如果你倾向于打开很多标签，20 的上限应该是非常有用的。
*   **视频/音频需要用户动作:**不是自动播放视频的粉丝？此开关试图完全停止它们。只需进入调试菜单，选择媒体标志来打开它。这将阻止 YouTube 等网站上的视频在你加载页面的第二秒自动播放。
*   **重置热门站点/重新计算热门站点:**如果你喜欢 Safari 中的热门站点功能，该功能会打开一个新标签，显示你最常访问的站点，但发现它不是特别准确，那么你可以强制 Safari 重新计算这些站点，或者只是清除历史记录并重新启动。

这些是我觉得有用的选项，但你可能会发现其他设置，使您的浏览更好。继续尝试不同的选择，看看它们能做什么，看看它们是否适合你。

### 修复 Safari 最大的烦恼

Safari 会做一些非常烦人的事情，并试图让您使用 iCloud Keychain。虽然每个人的具体烦恼都有所不同，但以下是我为了让 Safari 对我更好而做的事情。

*   **把你的默认搜索引擎改成谷歌:**Safari 里默认的搜索引擎是雅虎，这无论你在什么年代都很可笑。若要更改它，请打开 Safari 菜单，选取“偏好设置”，然后搜索。选择你想要的搜索引擎。当您在此菜单中时，如果您不喜欢 Safari 自动填充搜索结果或记录您最常访问的站点，您可能想要将“包括 Safari 建议”、“启用快速网站搜索”和“显示个人收藏”切换到关闭状态。
*   **关闭书签侧边栏，打开一个类似 Chrome 的收藏夹栏:**有些人可能喜欢书签侧边栏，但即使给了它一个机会，我也不得不放弃它。点击查看>隐藏侧边栏或查看>隐藏书签侧边栏来去掉侧边栏。如果你喜欢你的收藏夹出现在地址栏下方，就像在 Chrome 中一样，点击查看>显示收藏夹栏。
*   **停用自动填充:**由于我们已经建议您不要使用 iCloud 钥匙串，请继续停用自动填充(因为您应该将所有信息保存在密码管理器中。)前往 Safari >偏好设置>自动填充并取消选中所有复选框。
*   **更改新标签页和窗口选项:**默认情况下，当您打开新标签页或窗口时，Safari 会打开您的 Top Sites。除非你总是去相同的五六个网站，并且不介意任何人在你身后看着你知道那些是什么网站，否则你可能想要改变这种行为。前往野生动物园>偏好>常规。在这里，您会看到“Safari 打开方式”、“新窗口打开方式”和“新标签打开方式”的下拉列表把这些换成你想要的。就我个人而言，我对它们都使用空白页面，这样我就不会分心，忘记为什么要打开一个新标签页。

Safari 永远不会是一个超级强大或可定制的选择，但它肯定是最受 Mac 用户欢迎的选择之一。这不是一件坏事:Safari 很稳定，更安全，而且很难出错。它也比你想象的强大得多，只要你花时间根据自己的喜好定制它。
