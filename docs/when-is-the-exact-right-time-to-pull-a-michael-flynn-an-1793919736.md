# 什么时候才是迈克尔·弗林背叛你同事的最佳时机？

> 原文：<https://lifehacker.com/when-is-the-exact-right-time-to-pull-a-michael-flynn-an-1793919736>

如果你一直在关注迈克尔·弗林事件的发展，其中 [弗林提出为](http://theslot.jezebel.com/mike-flynn-has-reportedly-offered-to-testify-in-trump-r-1793877556) 特朗普政府与俄罗斯的关系作证，你可能会很好奇。当然，我很好奇弗林要说什么(难道我们不想知道吗！)但也只是好奇他的*战略*:为什么是现在？为什么不是一个月前或者一个月后？也许你问过自己，“如果我是某个邪恶行动的一部分，不管是有组织的犯罪，还是仅仅为了外国势力的利益而合谋调整美国政策，我怎么会知道*确切地说*在“囚徒困境”中拉下拉索并提出作证以换取豁免权的正确时机？”

我的意思是，如果你是偶然或故意在一个可疑的企业中的一个齿轮，你觉得联邦调查局在你的门口嗅来嗅去，你能虚张声势多久，更重要的是，保持你的神经？

这是一个棘手的问题，前联邦公设辩护人、华盛顿特区凯泽迪隆律师事务所合伙人马特·凯泽说，答案取决于你知道什么，你认为控方知道什么，你对你的同谋者了解多少(以及他们对*你】*翻脸的可能性)，大环境，以及一点博弈论。

不过，凯泽说，一般来说，“如果你处在一个严重犯罪正在发生的情况下，联邦调查局盯上了你，而且开始变得真实起来，**你想翻盘的时刻正是他们很明显要前进的时刻——但在任何人*或其他人*翻盘**之前。”

好吧，如果你过早行动，或者拖延太久，会发生什么？凯泽说:“不时会发生一件尴尬的事情，那就是有人过早翻脸认罪。然后，法院或陪审团会认定所发生的事情并不是真正的犯罪。”因此，对于一个实际上并没有被证明是犯罪的行为，你无论如何都要进监狱，或者你的律师匆忙撤销认罪。

如果你等得太久呢？“很多人都太迟了。如果你是第五个人，那就太晚了。你得不到任何荣誉。”他们不再需要你了。

凯泽说，提供证词以换取豁免权的整个过程有点像跳舞，或者扑克游戏。这就是这些谈判中真正有趣的部分:我以“你应该给我的人一张完整的‘免于牢狱之灾’卡开始，我不能告诉你为什么。“其他人说，‘你是个不错的家伙，但是，嗯，*不*’然后你给他们更多的信息。他们给你一些信息。就像约会一样。如果你不小心，你会得到一个坏名声。”

哦，如果你是一个女人，而你卷入了一个见不得人的企业？熟读 [囚徒困境](http://www.econlib.org/library/Enc/PrisonersDilemma.html) 。当凯泽还是联邦公设辩护律师时，他一半以上的客户是女性。“这其中的性别部分令人心碎。这些可怜的女人，仅仅是妻子和女朋友，被一名检察官逼迫着说出一些关于男人的事情。*的家伙们*会一直互相攻击。女朋友和妻子经常站在她们的男人一边，并因此受到了打击。

那么凯泽是怎么想的呢——弗林翻盘是太早了还是太晚了？他笑了。“我不知道。我们不知道他知道些什么。如果我们有，那就没用了。”
