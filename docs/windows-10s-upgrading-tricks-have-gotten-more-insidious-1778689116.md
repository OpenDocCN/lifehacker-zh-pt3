# Windows 10 的升级伎俩变得更加阴险

> 原文：<https://kotaku.com/windows-10s-upgrading-tricks-have-gotten-more-insidious-1778689116>

未来几年，微软希望 [Windows 10 在 10 亿台设备上运行。](http://techcrunch.com/2015/04/29/microsoft-expects-1-billion-windows-10-devices-in-2-3-years/) 确保这种情况发生的一种方式是通过欺骗用户以不正当的方式升级。但真正的问题是:Windows 10 是一个很好的操作系统，而微软的策略给了它一个坏名声。

Watch

那些关于人们意外升级到 Windows 10 的故事并不是虚构的； [几个月前发生在我身上的](http://kotaku.com/my-pc-upgraded-to-windows-10-without-asking-then-immed-1764756440)T3！有一段时间，当升级框弹出时，我会简单地将其安排在以后——冲洗并重复。然后不管什么原因，它决定一夜之间自发升级。由于一个小故障，Windows 10 随后锁定了我的文件，并由于驱动程序问题中断了我的互联网连接。

直到我在 *PC World* 上读到 Brad Chacos 的这篇精彩文章，我才明白发生了什么，他在文章中指出了微软做出的改变。

几个月来，由于微软加大了安装力度，旧版本 Windows 提示的升级框已经改变。

它过去看起来是这样的:

即使它随着时间的推移而发展，如果您还没有准备好进行安装，Microsoft 也总是包含一些选项来安排您以后的安装。但是这些选项缓慢但肯定地隐藏在界面中。

 [https://kotaku.com/embed/inset/iframe?id=twitter-675122217940860928&autosize=1](https://kotaku.com/embed/inset/iframe?id=twitter-675122217940860928&autosize=1) 

在另一项发展中，它取消了告诉 Windows 你不想升级的方式。你唯一的选择就是故意按下红色的“x”

从这里开始，事情变得更加阴险，这似乎解释了我是如何设法被踢到 Windows 10 的。

该弹出窗口的最新版本如下所示:

你可能会认为日程安排的细节有助于清楚地阐明 Windows 试图做的事情，即使你不想要它。但是在这一点上，Windows 已经训练你认为红色的 X 表示升级被取消。这是 X 的前提:取消/退出。

然而，看看微软自己的支持网站所说的:

> 如果您单击 OK 或红色的“X ”,您就为升级做好了一切准备，没有什么要做的了。

那是*屁话，*并且充分解释了我是如何在 Windows 10 上结束的。我打了 x。

当时，我感到愤怒和背叛，并讨论重新安装 Windows 7。这个操作系统运行得很好，尽管 Windows 10 是崭新的，但微软公开承诺在 2020 年前支持 Windows 7 的安全更新。我没有升级是因为一个原因。我知道电脑是如何工作的，我不需要被人牵着手。然而，我却被当成了孩子。

鉴于我是为 Kotaku 报道电脑游戏的，我把这次事故当成了前进的机会。无论如何，我计划在微软开始收费的最后期限之前升级。(7 月 29 日！)你猜怎么着？一帆风顺。虽然我不是 Windows power 用户，但这是我每天使用最多的操作系统。我编辑视频、流式游戏、为 *Kotaku* 写作——这一切都发生在我的 Windows 电脑上。如果你是一个临时用户，你想要的只是操作系统不要搞砸，除了坎坷的升级过程，Windows 10 没有搞砸！

开发一个新的操作系统并不性感。如果一家公司做得很好，就不会有大量的头条新闻称赞他们有能力。但对于 Windows 10，每篇报道都是对微软如何处理这一过程的另一个烦恼。( [就在昨天，冲击了一个离开酒店的游戏开发者](http://kotaku.com/spontaneous-windows-10-update-prevents-game-dev-from-ch-1778519338) ！)在升级过程中，他们每拖一个人，就好像又惹怒了 10 个人。这不是一个好的比例。

Windows 10 是个好产品，微软。也许让它自己说吧。
