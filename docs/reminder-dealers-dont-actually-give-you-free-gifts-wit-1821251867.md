# 提醒:经销商实际上不会在您购车时赠送“免费”礼品

> 原文：<https://jalopnik.com/reminder-dealers-dont-actually-give-you-free-gifts-wit-1821251867>

对于那些在这个假期外出购车的人来说，你可能会看到一些经销商 [为你的购车宣传一些“免费”赠品](https://jalopnik.com/watch-out-for-dealerships-that-give-things-away-for-fre-1746181580) 。这些事情可能包括一台电视，一些油的变化，甚至是一个免费的假期。我不想告诉你这个，但是这些东西没有一样是“免费的”



最近，一个朋友告诉我他想租一辆车。当他看合同时，有一个奇怪的行项目，上面写着“免费 50 英寸电视。”在合同起草之前，没有讨论过这一点，也没有向他透露过。

这辆车的价格似乎也比他预期的要高。这时，这位精明的购车者问销售人员:“那么，没有电视的租赁价格是多少？”他说，销售人员的反应是“哦，妈的，这家伙知道发生了什么。”他没有和那家商店成交。

每年的这个时候，一些经销商为了达到销售目标而进行最后冲刺，他们会想尽一切办法让人们进入展厅。这可能包括在购买或租赁新车时免费赠送广告。就像意志薄弱的人容易受到力量的影响一样，对汽车购物如何运作不太敏感的人通常会相信，他们实际上是通过把某种免费赠品带回家而得到了一笔“交易”。此外，这些礼物可能是秘密的——在计入购买价格之前，不会被宣传，甚至不会被提及。

当然，不需要数学天才也能算出客户还在为这个所谓的礼物买单。他们只是间接地这样做。几乎在每一个例子中，这些“礼物”都被添加到交易的总价格中。通常这些交易都有一些附加条款，一旦你算了一下，这笔交易并不像你想象的那么好。

就拿这个三菱的广告来说吧，每辆 2017 款欧蓝德都提供一台免费电视。

哇，在一辆剩余的 2017 款三菱欧蓝德上节省 4000 美元！你知道人们仍然购买三菱汽车吗？如果这让人感到意外，那么你可能也意识到，卖不出去的剩余机型降价 4000 美元并不是一个很大的折扣。不需要做太多的工作就可以得到一个大幅度的更好的价格，因为有很多广告宣传的欧蓝德汽车都有超过 6000 美元的折扣。【T2

现在小字是事情真正变得有趣的地方“购买 336/月，84 个月期限，包括所有税费，4.5%的年利率。”

这么说很悲哀，但大概只有绝望或者不明智的人才会为一辆三菱融资 7 年。即使有人这样做，这笔贷款的本金余额是 24，172 美元，其中包括当地的销售税和 DMV 费用。然而，在贷款结束时，你将总共支付 28，224 美元，或者比那辆车的标价多支付 2，289 美元。

这甚至没有考虑到这样一个事实，即以 4.5%的年利率零首付购买一辆新三菱汽车，并为其融资 84 个月，这是一种极其危险的方式，会让你发现自己处于水下贷款的境地。当然，那笔交易会给你带来一台电视机。但是它真的值得额外的汽车成本吗？当然不是。

这实际上可以归结为这样一个事实，即大多数使用这种免费赠送技术来销售汽车的经销商正在寻找一种特定类型的客户。一个不知情的人，一个容易上当受骗的人。

如果你碰巧在今年冬天想买一辆新车，看到这种广告，你最好联系他们的竞争对手，像往常一样，提前书面报价，这样你就可以在到达之前发现任何花招。仔细分析你要报名参加的项目。
