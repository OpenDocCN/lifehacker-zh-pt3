# 所有权贷款是一个骗局，你需要避免

> 原文：<https://jalopnik.com/title-loans-are-a-scam-you-need-to-avoid-1746849109>

急需现金却找不到愿意借给你的人？有一辆有明确标题的车？也许你听说过，你可以用那个头衔借钱，获得“头衔贷款”不要。



理论很简单:你借钱，把你的汽车所有权作为抵押。这些网站经常承诺你可以在“几分钟内”拿到现金，这听起来比银行贷款给你的时间要快得多。但是你会注意到网站上少了一些东西。比如贷款的利率是多少。或者贷款人是否有执照在你的州经营。

这些贷款已经以各种形式存在了一段时间，但最近越来越受欢迎，因为经营者发现了有利可图的所有权贷款。搜索短语“产权贷款”,看看有多少光滑的网站弹出。大多数都贴有迷人女性的照片，试图通过你的显示器递给你现金。

许多产权贷款机构收取过高的利率——是的，事实证明利率是由法律规定的，贷款机构不能对个人贷款收取 500%的年利率。至少，不是那些遵守法律的人。

但是有大量的故事表明，产权贷款机构收取疯狂的利率，几乎没有任何文件，只是收取他们想要的任何费用。有时，只要他们愿意，他们就会“收回”抵押品。依赖产权贷款的人通常没有能力做太多。

在密歇根州(以及许多其他州)，这些贷款人在法律之外运作。一些州甚至禁止产权贷款人根据这些贷款收回车辆。或者告诉贷方他们不能通过法院来强制执行贷款。但是执行情况充其量是参差不齐的。许多贷款人在遥远的管辖区运作，比如南太平洋的岛屿。他们从那些不抱怨的人身上赚了足够多的钱，他们可以承受失去那些抱怨的人的利润。

而像密执安州这样的一些州似乎被整个过程弄糊涂了，什么也没做。监管消费者保护的部门认为这应该由监管银行业的部门来监管。猜猜银行部门认为谁应该监管它？

我给你的建议是:避开头衔贷款人。如果你被一个律师缠上了，咨询一下当地的律师。也许有办法让你摆脱这种状况，同时还能保住你的车。需要快速借钱？避免产权贷款人。他们只不过是换了名字的高利贷者。
