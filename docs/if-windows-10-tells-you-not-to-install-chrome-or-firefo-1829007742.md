# 如果 Windows 10 告诉你不要安装 Chrome 或 Firefox，忽略它

> 原文：<https://lifehacker.com/if-windows-10-tells-you-not-to-install-chrome-or-firefo-1829007742>

据报道，Windows 10 将在安装非边缘浏览器时开始警告用户。测试版测试人员目前正在提前了解微软即将于 2018 年 10 月发布的 Windows Update，一些人已经注意到了一个相当令人讨厌的新内容。



当第一次安装新的非微软 Edge 网络浏览器时，比如 Chrome、Firefox、Opera 或 Vivaldi，一些测试版用户会收到一条系统消息，似乎在“警告”不要运行这些应用程序。

从技术上来说,这不是一个“警告”,也不会阻止你执行安装或运行应用程序，但这个消息暗示 Windows 试图保护你不犯一个潜在的危及系统的错误。

不用担心:谷歌 Chrome 和 Firefox 等非 edge 浏览器并没有成为 Windows 用户的主要风险 — 微软只是*真的*希望你使用 Edge。

我们希望这不会出现在十月的更新中。值得庆幸的是，如果出现这种情况，你应该只在第一次安装新浏览器时才会看到这条消息，所以在最坏的情况下，这将是一条奇怪的(尽管对某些人来说可能是一个警告)一次性消息。您可以简单地忽略这个“警告”并继续安装。

根据测试人员在 Twitter 上发布的警告 [，](https://twitter.com/SeanKHoffman/status/1039573136168169475/photo/1?ref_src=twsrc%5Etfw%7Ctwcamp%5Etweetembed%7Ctwterm%5E1039573136168169475&ref_url=https%3A%2F%2F9to5google.com%2F2018%2F09%2F12%2Fwindows-10-chrome-warning-prompt%2F) 的图像，似乎微软似乎已经有可能首先禁用这些弹出窗口。由于我们没有对新的 Windows 10 更新进行 beta 测试，我们不太确定如何禁用警告，但我们希望它至少类似于当前的 [禁用 Windows 10 的方法(坦率地说，绝望地)试图让用户使用 Edge](https://lifehacker.com/disable-those-annoying-browser-notifications-once-and-1822154599#_ga=2.233427926.979447446.1536778661-1094059819.1536778661) 。

如果微软在 Windows 10 中包含这样的“警告”看起来不真诚，那是因为它确实如此。从技术上来说，Edge 确实比其他一些浏览器更快更安全，你会在论坛和评论区发现很多关于什么浏览器最好的争论。然而，现实是，每一个可用的网络浏览器本身都是一个可行的选择，充满了自己的安全措施和独特功能。

我们对微软试图将用户从竞争中转移到边缘并不感到惊讶。在 Windows 10 的整个生命周期中，该公司曾试图引导用户使用其专有软件或混淆替代应用程序。如果你是 Windows 10 用户，你可能已经看到并忽略了 Cortana、OneDrive 和其他 Edge 广告，这些广告过去都自称为“警报”。这些警告只是一个奇怪策略的最新体现，该策略尚未对微软有利。

好消息是，我们距离 2018 年 10 月的更新还有几周时间。由于测试版测试人员对不必要的弹出窗口感到不满，不确定“警告”消息是否会出现在最终版本中。该公司在过去一直接受测试者的反馈，现在越来越多的人知道了，微软有希望在向公众推出之前完全放弃这个想法。我们会耐心等待，直到那时，如果警告信息碰巧进来，你可以肯定我们会告诉你如何关闭它们。
