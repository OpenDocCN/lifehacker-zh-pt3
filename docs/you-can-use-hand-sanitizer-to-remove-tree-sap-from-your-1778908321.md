# 你可以用洗手液清除汽车上的树液

> 原文：<https://jalopnik.com/you-can-use-hand-sanitizer-to-remove-tree-sap-from-your-1778908321>

除了粗略的机械结构、速度限制和有故障的氧气传感器，树液是你的汽车最大的噩梦。一旦它破坏了你原本干净的油漆工作，这些东西就不会脱落。好消息是，实际上有一种简单的方法可以清除植物排泄物。

市面上有成千上万的产品承诺去除汽车油漆中讨厌的树液。他们中有些人 [工作](http://www.amazon.com/Turtle-Wax-T-520A-Remover-Trigger/dp/B0009JKGII/ref=sr_1_1?asc_campaign=InlineText&asc_refurl=https://jalopnik.com/you-can-use-hand-sanitizer-to-remove-tree-sap-from-your-1778908321&asc_source=&ie=UTF8&keywords=tree sap remover for car&qid=1464273517&sr=8-1&tag=kinjajalopniklink-20) ，有些人完全是垃圾。但是，当你可以从 7-11 便利店买些洗手液，然后收工的时候，为什么要浪费时间在这些东西上呢？

我的切诺基吉普车一直停在一棵大松树下 [，因为几个月前我炸了它的引擎](http://truckyeah.jalopnik.com/how-blowing-up-my-engine-led-to-a-wild-night-of-offroad-1742787968#_ga=1.13276343.1273901456.1445270859) ，所以现在它沾满了成吨的树液。

在试着用湿抹布擦掉这些粘粘的东西后，我知道我需要一些更有效的东西。于是我抓起一小瓶洗手液，挤了一滴到污渍上，用手指擦了擦。它非常有效！你应该这么做

#### 1.在汁液上挤一点洗手液

清洁汁液不需要太多洗手液，所以从一小滴开始。

#### 2.用手指轻轻摩擦树液

不要疯狂地摩擦，只要轻轻移动你的手指在汁液点上，观察消毒剂溶解粘液。

#### 3.用布/纸巾把它擦干

一旦树液斑点消失，就把它弄干。可能还会有一些残留物。

#### 4.用湿布/纸巾清洗污渍并擦干

要去除残留物，只需在抹布上倒一点水，然后擦拭该区域。然后用抹布擦干，你就可以走了。结果是一个无味的油漆工作:

不过，值得注意的是，这种洗手液可以去除蜡，一些论坛成员仍然担心洗手液对 clear coat 的影响(他们担心其 70%的酒精含量)。

因此，在这样做之前，我建议在一个不可见的油漆点(可能在引擎盖下)上测试这种方法，以确保你的油漆和透明涂层与消毒剂很好地协调。

在多次这样做之后，我觉得推荐它很舒服。*驾驶雅阁*论坛用户 [蔻驰史蒂夫](http://www.driveaccord.net/forums/18-maintenance-detailing/348825-safe-use-hand-sanitizer-paint.html) 也是如此，他说:

> 首先，我还没有使用过一种专门配制和销售的树液去除剂，这种产品能像 [消毒剂](http://www.amazon.com/dp/B000H0TTD2?asc_campaign=InlineText&asc_refurl=https://jalopnik.com/you-can-use-hand-sanitizer-to-remove-tree-sap-from-your-1778908321&asc_source=&tag=kinjajalopniklink-20) 一样快速去除树液。在这种情况下，当你考虑到溶解和清除汽车上的树液所需的量很少，并且考虑到酒精蒸发的速度，你很难对你的漆面造成任何明显的损害。

他继续说:

> 我特意在这个问题上做了广泛的、非科学的测试来证明我的假设，而且还没有造成任何伤害。句号。

这样你就不用再做关于你车上有树液的噩梦了。如果一棵树决定滴在你的宝贝车上，就去街角的商店买终极防树液武器吧。
