# 最适合您电脑的硬件附件

> 原文：<https://lifehacker.com/the-best-hardware-add-ons-for-your-pc-1823073598>

个人电脑是伟大的，但是他们经常感觉他们是为除了人类之外的另一个物种而制造的。每多插一根电线，你就多了一根新电线需要管理和整理。它们是可定制的，但打开一个并开始取出它的内脏是令人生畏的。你桌子上的那个大显示器占了很多柜台空间。幸运的是，您可以回收桌面上的空间，用一站式连接商店取代杂乱的电缆，将所有东西放回原处，用这些工作场所的必备附件来提高您的生活质量(和脖子上的痛)。



### **电缆延伸**

不要用你的手和膝盖去插你的 USB 驱动的杯子加热器。拿一根 [USB 公母延长线](https://www.amazon.com/USB-3-0-Extension-Cable-6-5ft/dp/B072M721QD/ref=sr_1_10?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-best-hardware-add-ons-for-your-pc-1823073598&asc_source=&ie=UTF8&keywords=usb extension cable 6ft&qid=1518815945&sr=8-10&tag=kinjalifehackerlink-20) ，把它插到你的电脑上，把母头插到你想连接的任何东西上，过你的生活，知道你再也不用去找另一个 USB 端口了。它非常适合插入 USB 闪存驱动器或充电设备，如内置 USB 插头的电池。

### **耳机支架**

把耳机放在你的桌子上(或者放在家里)，基本上就是自找麻烦。当你的狗咬着焊接在你最喜欢的一对罐子上的不可更换的 3.5 毫米电缆时，你有没有忍住愤怒的尖叫？是啊，我也是。总之。

一个耳机支架，无论是放在桌面上的[还是贴在下面的](https://www.amazon.com/Anchor-ORIGINAL-Under-Desk-Headphone-Stand/dp/B00P31BMHG/ref=sr_1_10?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-best-hardware-add-ons-for-your-pc-1823073598&asc_source=&ie=UTF8&keywords=headphone stand&qid=1518816047&sr=8-10&tag=kinjalifehackerlink-20) ，不仅可以保护你的耳机免受意外伤害，还可以建立一个专属位置，这样你就再也不用寻找它们了(因为所有东西都需要它的位置)。如果你真的想提升你的生活，买一个带有一些 [供电 USB 端口](https://www.amazon.com/Satechi-Aluminum-Headphone-Stand-Holder/dp/B019PI9QD4/ref=sr_1_4?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-best-hardware-add-ons-for-your-pc-1823073598&asc_source=&ie=UTF8&keywords=headphone stand usb&qid=1518816068&sr=8-4&tag=kinjalifehackerlink-20) 的耳机支架，以便为你的蓝牙耳塞、便携式 DAC 或智能手机充电。

### **USB-C(还有 USB-A！)适配器**

如果你买了一个依靠通用端口的设备(或者你的朋友带着一个坏了的智能手机过来)，为你的电脑配备至少一个 [USB-C 转 USB-A 适配器](https://www.amazon.com/Anker-Adapter-Converts-Technology-Compatible/dp/B01COOQIKU/ref=sr_1_2?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-best-hardware-add-ons-for-your-pc-1823073598&asc_source=&ie=UTF8&keywords=usb-c to usb-a&qid=1518816195&sr=8-2&tag=kinjalifehackerlink-20) 可以解决很多问题。它让您的工作空间经得起未来考验，无需更换 PC 内部的 USB 端口或购买全新的机器来连接最近生产的设备。它们很便宜，所以买两个吧。除了 C-to-A 适配器之外，您还应该获得一个 [USB-A 转 USB-C](https://www.amazon.com/Adapter-Syntech-Laptops-Chargers-Interface/dp/B0749Q9Y4N/ref=sr_1_16?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-best-hardware-add-ons-for-your-pc-1823073598&asc_source=&ie=UTF8&keywords=usb-a to usb-c cable&qid=1518816210&sr=8-16&tag=kinjalifehackerlink-20) 适配器，以覆盖您所有的底座，以防*您*购买一个 USB-C 设备来连接您配备 USB-A 的 PC。

### **带无线充电器的台灯**

台灯非常适合用来照亮你的桌子。办公室里有一盏以上的灯可以减轻眼睛疲劳，根据灯的不同，用暖黄色的灯(非常适合晚上工作)或冷蓝色的灯照亮你的环境，以便白天工作。

如果你真的想挥霍，你应该给自己买一个内置无线充电板的台灯，或者一些 [USB 充电端口](https://www.amazon.com/TaoTronics-Eye-caring-Dimmable-Charging-Licensing/dp/B00VUTAFR8/ref=sr_1_4?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-best-hardware-add-ons-for-your-pc-1823073598&asc_source=&ie=UTF8&keywords=desk lamp usb led&qid=1518816559&sr=8-4&tag=kinjalifehackerlink-20) 。然后，你可以把你的手机放在桌子上，远离你的床，当它的闹钟开始响的时候，强迫自己直立着开始新的一天。

### **监视器臂**

如果你的桌子上有一台显示器，把它从桌子上拿走可能是你做过的最好的决定。除了为其他项目腾出空间(比如你所爱的人的照片，或者你最喜欢的流行乐队的某种神龛)，一个显示器臂实际上给你的屏幕增加了一个长颈鹿一样的脖子。改进的人体工程学将改善您的姿势，并可以缓解下背部疼痛等问题，因为您不会弓着身子坐在办公桌前盯着屏幕及其校准不当的高度。

您可以向几乎任何方向移动显示器，根据您的视角向上或向下指，并且当您需要更多垂直空间时，可以轻松地将其旋转到纵向模式。某些显示器臂还包括电缆管理功能或不同的安装选项，因此您可以将笔记本电脑而不是屏幕放在上面。

### **供电 USB 集线器**

如果你的电脑在桌子下面，在显示器后面，或者有一个一体化的外形，两者被融合成一个邪恶的机器，插上东西会变得很烦人。如果你不把自己扭到桌子底下就够不到港口，那就更糟了。

[一个带电源的 USB 集线器](https://www.amazon.com/Anker-10-Port-Adapter-Including-Charging/dp/B005NGQWL2/ref=sr_1_2_sspa?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-best-hardware-add-ons-for-your-pc-1823073598&asc_source=&ie=UTF8&keywords=anker usb hub&psc=1&qid=1518816607&sr=8-2-spons&tag=kinjalifehackerlink-20) 不仅可以让你方便地接入端口，快速为你的设备充电，你还可以使用所有这些端口将你的智能手机或 DSLR 等设备连接到你的电脑上。搭配一些短电缆，USB 集线器既实用又美观。
