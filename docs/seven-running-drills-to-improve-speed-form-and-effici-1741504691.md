训练很快成为我跑步生涯中的一个主要部分，在大学的每一次艰苦训练之前，我们都会作为一个团队进行训练。但是如果你从来没有在一个团队中跑步，就很难知道如何进行跑步训练——以及*什么时候*做这些训练。你可能会有很多问题:

今天我想分享我最喜欢的七种跑步训练，它们可以提高速度、形态和效率，你可以将它们融入到你的训练中。但是首先，让我们深入了解一些关于训练的背景知识。

### 谁应该进行跑步训练？

所有人！对于初学者来说，这是一个很好的方式来提高他们的运动能力，加强适当的机制，并改善他们的状态。出于同样的原因，它们也有益于更高级的跑步者。

就像跑步和 [力量练习](http://strengthrunning.com/2015/07/best-strength-exercises-for-runners/) 是几乎所有人都推荐的锻炼形式一样，形式练习也是几乎所有跑步者都推荐的。

唯一需要谨慎进行形式训练的跑步者是那些从严重受伤中恢复的人。演习增加的冲击力(特别是如果你在沥青或混凝土上进行)可能会使你再次受伤。

### 什么时候是进行训练的最佳时间？

训练是一种*技能训练，对跑步者来说*也被称为*技术活*。因此，它们通常应该在热身之后，大部分训练之前进行。这种逻辑顺序可确保您从锻炼中获得最大收益(受伤风险最小)。

以下是大多数锻炼的建议顺序:

1.  [动态热身](http://strengthrunning.com/2015/06/mattock-dynamic-warm-up-exercises/)
2.  平滑运转
3.  技能工作(如训练)
4.  [大踏步](http://strengthrunning.com/2012/10/what-are-strides/)
5.  锻炼(山峰、节奏、重复等。)
6.  平滑运转
7.  力量锻炼(像 [健身球锻炼](http://strengthrunning.com/2014/04/tomahawk-medicine-ball-workout/) )

该序列显示良好的编程。请记住，并非每个课程都会包含所有这些培训内容。

就频率而言，大多数跑步者可以每周做两次训练以获得最佳效果。它们最好在更快的训练、比赛和其他挑战性训练之前进行。它们也可以作为电路的一部分来完成。你可以在力量跑的 [高级循环训练](http://strengthrunning.com/circuit-workout-download/) 中看到这一点。

### 哪里可以做形体训练？

你几乎可以在任何有空间的地方进行这些训练。你通常需要大约 50 米的无障碍空间，如道路、人行道或田野。但是有几个地方特别推荐:

*   艰苦锻炼或比赛前的道路(坚硬的路面可以帮助增加——从而提高你的表现)
*   在 [室外跑道](http://strengthrunning.com/2015/04/how-to-run-track-workouts-on-an-outdoor-track-infographic/)——更柔软的地面，没有被什么东西绊倒的危险，在那里做这些无聊的练习更容易被接受！
*   在草地上或人造草坪上——最适合容易受伤的跑步者或那些想要赤足训练以增加脚和小腿力量的人

就在几天前，我看到一个跑步者在我当地星巴克附近的小巷里做形体训练。因此，您几乎可以在任何地方真正做到这一点！

### 我应该做多少次跑步训练？

没有必要每次都做这七种形式的练习。那需要另外 20 多分钟——对大多数跑步者来说是宝贵的时间！

相反，选择三到四次训练，并遵循以下指导原则:

*   大多数钻孔应完成 30-50 米(carioca 可能会占用更多空间)
*   在开始下一个练习之前，走回你开始的地方
*   在开始下一个练习之前，每组做两到三组

许多跑步者可能想知道，如果他们不应该做所有的训练，应该选择什么训练。虽然做所有这七个练习都很好，但时间限制是我们生活中的现实。

最好选择有助于特定训练课程的目标以及你的具体需求的训练。

例如，如果你正在与 [IT Band 综合征](http://strengthrunning.com/2014/01/it-band-syndrome-treatment/) 作斗争，我推荐 A-skip、carioca 和直腿跳跃，因为它们可以增加节奏，改善臀部灵活性，激活臀肌——所有这些都有助于治疗和预防 ITBS。

以下是根据难度(1 号是最容易的，7 号是最难的)以及每个练习的帮助程度列出的上述练习:

1.  **踢腿**(视频中的第二个变化):提高股四头肌和臀部屈肌的灵活性，同时加强高节奏
2.  **高膝盖**:加强中足落地、高节奏和腿筋灵活性
3.  **A-Skip** :加强中足落地，高节奏，提高协调性
4.  **B-Skip** :提高协调性和腿筋灵活性
5.  **踢腿**(第一种变化):增加臀部屈肌力量，加强中足落地
6.  **卡里奥卡**:改善协调性，增加臀部灵活性
7.  **直腿跳跃**:激活臀肌，提高协调性

当然还有其他的训练，但是我选择了这些，因为它们提供了最大的好处，同时也是最容易学习的。
