# 任天堂 Switch 网络适配器确实有助于在线游戏

> 原文：<https://kotaku.com/the-nintendo-switch-network-adapter-really-helps-with-o-1797435922>

如果你喜欢多人游戏，你可能已经听说过如何使用有线连接进行游戏。这款交换机和之前的 Wii U 一样，甚至没有以太网端口——但你可以解决这个问题。



一周半前，我在亚马逊上购买了 Hori 的任天堂授权局域网适配器 。售价 29.99 英镑，官方的东西不是一个便宜的附加功能，考虑到 PS4 和 Xbox One 都内置了这项功能，不得不支付任何费用尤其令人恼火。尽管如此，我还是想测试一下这个适配器，因为我最近玩了很多《T4》卡通 2 。

在大多数情况下，我的 Wi-Fi 连接在*splayon*中似乎还可以。大约每 10 场比赛，一场地盘之战就会因为连接问题而短暂中断，或者我会神秘地退出比赛，结果却收到这样的警告:

这个游戏让我听起来像是恶意退出比赛，但事实并非如此。我猜？我的连接打了个嗝，因为众所周知，Wi-Fi 不如有线连接稳定。所以我试着修好它。

对我来说，这意味着使用一根非常长的以太网电缆，并通过网络适配器将其连接到我的交换机上。适配器插入交换机上的 USB 插槽，这意味着您只能在对接时使用它。对于一些人来说，这不是一个选项——例如，可能你的路由器太远了。但是如果你有钱也有能力去做，我会全心全意地推荐它。

十几个小时后，我还没有从一场*splanton*比赛中退出，也没有看到任何奇怪的连接问题。一切都像黄油一样光滑；我的照片会照我想的去，而且从来没有什么瞬间移动的小孩。可笑的是，我现在遇到了与相反的*问题:我呆在其他玩家断开连接的比赛中，这在像 Salmon Run 这样的模式中尤其令人痛苦。我敢打赌那些人也都有无线网络。*

除了游戏内的好处，当你下载游戏时，有线连接非常有用。以前可能需要一两个小时的事情现在可以在 10 或 15 分钟内完成，这使我可以更快地投入到行动中。有交换机网卡的生活肯定更好，如果你能摆动它的话。
