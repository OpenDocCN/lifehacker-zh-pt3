# 这款笔记本电脑是人们希望苹果制造的 MacBook Pro

> 原文:[https://gizmodo . com/this-laptop-is-the-macbook-pro-people-wish-apple-would-1824138523](https://gizmodo.com/this-laptop-is-the-macbook-pro-people-wish-apple-would-1824138523)

因此，我在离家 6000 英里的外国(我不会说他们的语言)独自报道一个贸易展览，我意识到我的 XPS 13 上的电源插头坏了。这是一场灾难。我的笔记本电脑是我的生命线。我需要它来写故事、编辑照片以及与家乡的团队合作。没有它，我真的完蛋了。更糟糕的是，我的工作 XPS 是最后一代不支持 USB-C 充电的产品，因为西班牙根本不在乎戴尔，所以没有替代产品(相信我，我试过)。

但当我紧张地看着笔记本电脑的电池电量从 68%下降到 67%时，我想起了一个解决办法。这是因为在我的包里，我带着华为最新笔记本电脑 MateBook X Pro 的评测单元。好了，是时候来一场真正的火炼了。

虽然华为最出名的是智能手机，但过去几年，它一直在慢慢进军笔记本电脑和平板电脑市场。它的第一次重大成功来自最初的 [MateBook X](https://consumer.huawei.com/us/tablets/matebook-x/) ，它采用(或借用)了苹果 12 英寸 MacBook 的模板，并修复了一些问题。但对于 MateBook X Pro，该公司将目光投向了更大的东西:13 英寸的 MacBook Pro。你知道吗？华为完全成功了，因为对于苹果顶级笔记本电脑的每一个缺点，华为似乎都有答案。

像苹果一样，华为从实心铝制机身开始，然后在两侧添加了惊人的强大扬声器，下面是一个大的一体式触控板，左上角是智能电源按钮/指纹识别器组合——所有这些都是顶级的。然后就是 Matebook X Pro 的背光键盘。虽然它有点浅，但键盘相对较高的驱动重量和更深的按键行程感觉比现代 MacBooks 上的垃圾要好得多。华为还智能地包括两个 USB-C 端口(其中一个支持雷电 3)和一个 USB 3 Type-A 端口，因此您可以过上无加密狗的生活。如果华为在 X Pro 上找到了 SD 卡插槽的空间，我会非常喜欢，尽管对于美国人来说，华为包含一个免费的 [MateDock 2](https://www.giztop.com/official-huawei-matedock-2-for-huawei-mate-10-matebook-x-e.html) (它附加了一个额外的 USB-A、USB-C、HDMI 和 VGA 端口)可能有助于消除对连接性的任何担忧。

一旦你看完了 X Pro 的设计，下一个吸引你目光的是它的出色屏幕。它不仅超级亮，最高亮度达到 485 尼特，而且是目前所有笔记本电脑中最薄的。我说的不仅仅是顶部和侧面的微薄利润，X Pro 的下巴也同样薄，导致屏幕与身体的比例为 91%。这比像 [新 XPS 13](https://gizmodo.com/the-new-dell-xps-13-is-so-good-you-can-almost-forgive-1823323549#_ga=2.225513884.727823878.1522152698-3254418773.1522168809)(80.7%)和[Galaxy S9](https://gizmodo.com/samsung-galaxy-s9-review-the-best-android-phone-dont-n-1823575944)(83.6%)这样的设备好得多。在每天使用它三个多星期后，我真的不想回到普通的笔记本电脑屏幕上，看着别扭的杰·雷诺比例。

但这还不是全部，因为通过减少边框，华为有了更大屏幕的空间，所以你可以得到一个高分辨率的 13.9 英寸 3000 x 2000 触摸屏，而大多数竞争系统的上限是 13.3 英寸。在这种分辨率下，再加上屏幕出色的色彩还原能力，您正在观看的图片、电影和其他任何内容都非常棒。

华为甚至为 X Pro 配备了一个隐藏网络摄像头的巧妙技巧。在屏幕上方或下方没有大边框的情况下，华为将网络摄像头藏在一个按键中，正好位于功能行的中间。当键盘的背光打开时，它非常明显，因为它是唯一不亮的键。但除此之外，你通常会忘记它的存在，这正是网络摄像头在你不使用它时应该做的。网络摄像头位置的另一个额外效果是，当它被按下并指向系统内部时，它就没有机会被黑客攻击并用来在不知不觉中监视你。

现在，对于所有的苹果铁杆粉丝来说，X Pro 不运行 macOS 可能是一个障碍。但这真的是苹果的错，因为它不让其他公司许可其操作系统，因为我怀疑华为可能也想效仿。但我并不真的感到困扰，因为华为在与微软合作在 Matebook X Pro 上安装 Windows 10 Pro 签名版时也做了同样好的事情。没有膨胀或添加任何废话。

至于规格，X Pro 有足够的动力，这要归功于第八代英特尔酷睿 i5 或 i7 CPU，高达 16GB 的内存，512GB 的 SSD 存储，如果你想要的话，甚至还有 Nvidia MX 150 GPU。与最近改进的 [XPS 13](https://gizmodo.com/the-new-dell-xps-13-is-so-good-you-can-almost-forgive-1823323549#_ga=2.103071488.1490696918.1522072599-99892821.1503319351) 和 [LG Gram 15](https://gizmodo.com/this-is-the-big-laptop-to-buy-when-all-you-care-about-i-1823728253#_ga=2.98868382.1490696918.1522072599-99892821.1503319351) 等竞争系统相比，我们满载的 X Pro 评测单元在我们的每一项标准基准测试中都获得了最高分，其中包括 WebXPRT 2015 中的浏览器性能、Photoshop 中的图片大小调整测试和 Geekbench 4。你不能说 X Pro 的图形性能也是如此，因为它的 MX 150 比标准的集成显卡更上一层楼，而不是为游戏而生的东西。

即使是 X Pro 的电池寿命也相当出色，在我们的标准消耗测试中，它持续了 11 小时 7 分钟。这比戴尔的 [XPS 13](https://gizmodo.com/the-new-dell-xps-13-is-so-good-you-can-almost-forgive-1823323549#_ga=2.68989457.1490696918.1522072599-99892821.1503319351) 和更大的系统如 [15 英寸 LG Gram](https://gizmodo.com/this-is-the-big-laptop-to-buy-when-all-you-care-about-i-1823728253#_ga=2.57561642.1490696918.1522072599-99892821.1503319351) 都要好，分别持续了 9 分 28 秒和 10 分 39 秒。

我不太明白的一个小怪癖是，默认情况下，X Pro 的固态硬盘被格式化为两个分区:一个较小的 80GB 分区和一个较大的 380 GB 分区。这很奇怪，因为只有一个物理驱动器，所以系统只是在没有真正原因的情况下将其存储分成两部分。也就是说，这是一个非常小的问题。

真的，试图挑出 MateBook X Pro 的缺陷并不是一件容易的事情。不管你认为华为多么无耻地抄袭了苹果的公式，它绝对在许多非常重要的方面改进了那个模板。

X Pro 的起价为 1200 美元，实际上比类似配置的 12 英寸 MacBook 或 [13 英寸 MacBook Pro](https://www.apple.com/shop/buy-mac/macbook-pro) 便宜 100 美元，同时提供更多端口、更新的第八代 CPU、更大、分辨率更高的屏幕，当然还有漂亮的弹出式网络摄像头。在高端方面，X Pro 看起来更好，因为 1500 美元的价格，你可以获得两倍于同等价格的 MBP 13 的 RAM 和 SSD 存储，以及 Nvidia 150MX GPU 的独立显卡。与此同时，配有真正 GPU 的最便宜的苹果笔记本电脑是 2400 美元的 MBP 15，它确实提供了更好的图形性能，但对大多数人来说过于昂贵。

在每天使用这个系统近一个月之后，我实际上不确定还有没有另一个我更愿意使用的日常驱动程序。这么说感觉有点怪怪的:但是你怎么称呼一台看起来像 MacBook Pro，但实际上好很多的笔记本电脑呢？我猜答案是一本 Matebook。
