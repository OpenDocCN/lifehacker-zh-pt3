# 穿上工装裤以防小偷

> 原文：<https://lifehacker.com/wear-overalls-to-keep-yourself-safe-from-thieves-1825501946>

我刚从为期两周的欧洲度假回来，为此我变得更聪明了。我不仅在牛津大学呆了一段时间，学会了如何在茱莉亚·切尔德自己的家里把 [做成高耸的](https://www.instagram.com/p/Bhgvtt8hLDm/?taken-by=clairelizlower) 蛋奶酥，还学会了工作服是防止扒窃的关键。

Watch

你可能知道也可能不知道， [我是背带裤](https://lifehacker.com/i-bought-a-pair-of-overalls-and-now-i-will-never-wear-a-1795306121) 的忠实粉丝，几乎一直都在穿。然而，在我从波特兰飞往巴黎的航班上，我认为我穿运动裤会更舒服，所以我决定把钱包放在一个斜挎包里，我把它戴在身上，有点像项链。这太愚蠢了。在我已经旅行了很长时间的恍惚状态中，包在我身后轻轻摇摆，几分钟后，我就少了一叠现金。

幸运的是，拿走现金的人掉了我的钱包，但如果我穿上围兜，把东西藏在胸前的口袋里，这一切都可以避免。虽然我很少意识到我身后发生了什么，但我总是会注意到有人朝我的胸部移动，尤其是在他们拉开拉链的时候。我的意思是，从我的错误中吸取教训，把重要的东西藏在围兜里。
