# 使用 FDA 的新应用程序搜索可靠的药物信息

> 原文：<https://lifehacker.com/search-for-reliable-drug-info-using-the-fdas-new-app-1824053792>

说到在网上搜索医疗信息，互联网可能是一个…大杂烩。本周，美国联邦药品管理局(Federal Drug Administration)发布了一款应用程序，可以让你的医生订户(或你在电视上看到的另一种药物)更容易找到新药的信息，而且更加可靠。

Watch

该应用名为 [Drugs@FDA Express](https://play.google.com/store/apps/details?id=gov.fda.daf&hl=en) ，本质上是 FDA 的 Drugs@FED 网站的移动版。通过它，您可以搜索有关 FDA 批准的品牌和通用处方以及非处方人类药物和生物治疗产品的信息。该应用程序包括已获该机构批准的所有类型药物产品的信息，以及患者信息、药物标签、批准函、评论和其他信息。

使用该应用程序，您还可以根据产品名称、活性成分或申请号搜索药物信息，并查看最新的药物批准。

说实话，它对开药的医生来说要比个人服药有用得多，但是如果你是一个每天服用大量药物的人，它可能是你手边的一个极好的资源。
