# 富国银行发行了一种体面的奖励信用卡

> 原文：<https://lifehacker.com/wells-fargo-has-launched-a-decent-rewards-credit-card-1827105186>

看起来富国银行正试图通过推出新的奖励信用卡来赎罪(或至少获得一些新客户)。



使用 [Propel 美国运通卡](https://newsroom.wf.com/press-release/innovation-and-technology/wells-fargo-and-american-express-introduce-new-propel-card) 在旅行、交通、外出就餐和网飞、Spotify、Apple Music 和 Hulu 等流媒体服务上花费的每一美元可以获得三倍积分。最后一个类别比任何实质性的东西都更具噱头，但酒店、加油站、出租车、通行费、公共交通、餐馆和高级餐厅的三倍积分是不容忽视的。

您还可以在所有其他购买中获得一个积分，并且积分永不过期。I t 没有年费，这使其与其他顶级餐饮和现金奖励卡具有竞争力，i 如果您在开卡后的前三个月消费 3000 美元，您将获得 30000 奖励积分，相当于 300 美元。有一年的介绍性零年利率，然后是 14.24%到 26.74%的可变年利率。

一个潜在的缺点是:无论美国运通在哪里都能被接受，这比 Visa 或 Mastercard 更受限制。

它旨在与 [Chase Sapphire Reserve 卡](https://creditcards.chase.com/a1/sapphire/DualA?CELL=6RRW&jp_cmp=cc/Sapphire+Reserve_Brand_BMM_Sapphire+Reserve_SEM_US_No+xLOB_Standard_NA/sea/p25780930708/Chase+Sapphire+Reserve&gclid=EAIaIQobChMIv7vHypDl2wIVEcRkCh1deQuXEAAYASAAEgK_GvD_BwE) 直接竞争，后者在相同类别上提供三倍积分，但第一年后需要支付 450 美元的年费。尽管如此，正如 [商业内幕](http://www.businessinsider.com/wells-fargo-launches-new-propel-card-2018-6) 解释的那样，大通的信用卡确实有优势，即使费用很高。“这张卡上的积分可兑换 1 英镑现金，1.5 英镑可兑换通过 Chase 购买的机票，或者在转让给常旅客合作伙伴时可兑换 1.5 英镑以上的积分。”无论如何，Propel 卡的积分值 1 英镑返现。

这一切听起来都不错，富国银行也需要如此。这家银行不仅没有任何其他有竞争力的奖励卡选择(富国银行的现金明智 Visa 卡 提供 1.5%的退款)，而且在欺骗了客户 之后，它需要重新赢得他们的好感。这张免费卡可能会有所帮助。
