# 像对待肉一样对待蘑菇，让它们更有味道

> 原文：<https://lifehacker.com/treat-mushrooms-like-meat-to-give-them-more-flavor-1829304093>

蘑菇是肉质和味道最丰富的蔬菜——这就是为什么蘑菇非常适合腌泡汁的原因。而且，与肉类不同，这种海绵状的“蘑菇”能快速吸收 flfavor*，这对没有耐心的人来说总是个好消息。*

*但是布朗宁呢？将蘑菇暴露在超出清洗绝对必要的液体中被认为是一大禁忌，但我发现，只要你用纸巾将它们拍干，浸泡过的蘑菇就会变得又黄又脆。事实上，取决于你的腌泡汁里有什么，它们可能会变得更快。(糖有这样的倾向。)你可以用你最喜欢的肉腌泡汁腌蘑菇，但我发现油基混合物和盐水效果最好。*

*如果你想走油基路线，你可以加入你最喜欢的醋油沙司(或 [姜汁调料](https://skillet.lifehacker.com/this-ginger-dressing-tastes-exactly-like-it-came-from-a-1827997044) )，或者将两份油、一份醋和一份糖(或蜂蜜、或糖浆)混合，再加上你身边的香草、香料、辣椒或任何其他风味增强剂。像泡菜汁这样的醋盐水也很好用，你冰箱里挂着的任何种类的发酵液也一样。(泡菜蘑菇，有人吗？)如果你正在处理较大的“蘑菇”，那么 [将它们撕成容易处理的碎片](https://lifehacker.com/tear-mushrooms-for-a-better-saute-1826197453) ，如果你正在处理 portobello，那么先刮去 gil ls。用你的腌料盖住肉块，让它们晾半小时到一小时，用纸巾拍干，然后像往常一样嫩煎或烘烤。*
