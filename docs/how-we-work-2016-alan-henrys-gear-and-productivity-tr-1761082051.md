每周，我们都会分享我们最喜欢的专家和名人的快捷方式、工作空间和生产力提示。这个星期，我们谈论如何工作，今天，轮到我了。不知何故，我现在坐在大椅子上，所以这里是我如何保持这艘船的航向。

显然我对这个系列并不陌生。2015 年，我对 [的工作空间做了一些非常巧妙的改变](http://lifehacker.com/how-we-work-2015-alan-henrys-gear-and-productivity-tr-1686103861) (我还做了三显示器！)2014 年，我 [把自己打造成一个跨平台的挺家伙](http://lifehacker.com/how-we-work-2014-alan-henrys-gear-and-productivity-t-1519560726) 。2013 年，我被 [用爱诅咒服务注定关闭](http://lifehacker.com/how-we-work-alan-henrys-favorite-gear-and-productivity-5983834) 。在*甚至是*之前，我是 [新来的小子](http://lifehacker.com/what-we-use-alan-henry-s-favorite-gear-and-productivit-5884841) 。这是一条漫长的道路，但让我们专注于新的东西——除此之外，至少是有趣的东西。

#### 你离不开哪些应用、软件或工具？

事实上，我办公桌上的所有东西都是通过 [谷歌文档和](http://drive.google.com/) 驱动的。我仍然在苹果电脑上用 [航空邮件](http://airmailapp.com/) 和 [邮箱](http://www.postbox-inc.com/) 管理电子邮件，尽管我已经看到一些有希望的测试版可能会取代这两个。好的桌面电子邮件客户端很难找到。与 Lifehacker 的其他人不同，大多数人都是 LastPass 爱好者，Dashlane 仍然是我的首选密码管理器。它有助于我仍然有一个旧的祖父级免费帐户，满足我的所有需求。

对于每个人来说，我仍然在智能手机中游泳。当然，我上面列出的那些都在这里，但这不算数 [几个月前我测试的 Nexus 5X](https://lifehacker.com/slim-pocket-showdown-the-google-nexus-5x-vs-the-oneplu-1748600419) 和[Nexus 6P](https://lifehacker.com/flagship-battle-the-google-nexus-6p-vs-the-oneplus-2-1747623790)，或者我仍在试图送回一加的 OnePlus X 和 OnePlus 2(伙计们认真地把这些手机拿回去吧，我不想要它们了。)

我的 [Nexus 5](http://www.google.com/nexus/5/) 是我的纯 Android 测试平台，我的 [OnePlus One](https://oneplus.net/one) 或多或少是当我的 Moto X 需要充电而我手头没有它时我使用的手机(我也用它向我的 Chromecast 进行了许多流媒体播放。)不过，我一直打算换掉我的 Moto X——它为我服务了很长时间，我在想 Nexus 6P 可能会是我的下一部手机，可能在下个月左右，假设今年春天的一轮发布会 [没有什么惊天动地的消息。我们走着瞧。](https://lifehacker.com/how-to-pick-your-next-android-phone-2016-edition-1760793478) 

#### 你的工作空间是什么样的？

老实说，从去年的这个时候到现在，变化并不大。当你发现一个可行的设置时，坚持下去。但不利的一面是，在短短的几个月内，我将搬家，这意味着我将不得不把一切都颠倒过来。祝我好运。

在那之前，我仍然在我的三显示器设置的中心摇摆一个 [30”单轴 IPS 显示器](http://www.monoprice.com/Product?p_id=10734) ，两侧是 [两个(现已停产)27”单轴 IPS 显示器](https://lifehacker.com/three-cheap-but-awesome-alternatives-to-the-apple-cinem-1500692411) 。Ganymede 是我值得信赖的(但已经老化)2010 Macbook Pro，它位于显示屏的右侧，当我需要随身携带工作时，我可以将它滑出并放入包中。我定制的 Windows PC deneb 在左边。我甚至还有 [空格键管理器](http://www.amazon.com/Quirky-Spacebar-Monitor-6-Port-PSPBP-WH01/dp/B00KJGYDAI?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-we-work-2016-alan-henrys-gear-and-productivity-tr-1761082051&asc_source=&tag=kinjalifehackerlink-20) ，它可以将 27 英寸的显示器保持在一个比 30 英寸更合适的高度，它们可以作为两台电脑的 USB 集线器。不久前，我终于将所有的显示器连接器都转移到了 DisplayPort，让我告诉你，如果你仍然在使用 DVI-D，你不必这样做，请改用 DisplayPort。线缆管理变得简单多了。

在 Deneb 上，除了升级到 Windows 10，没有太多变化。它既是我的游戏电脑，也是我的 Windows 测试平台。这里有人需要每天使用 Windows，这个人可能就是我。我一直在考虑建造一台新电脑，每次我更新我们的电脑建造指南 时，我都会心痒难耐，但我想我会在此之前挑战超频极限。

我仍然(我知道你闭嘴已经两年了)为我的灯加摄像头支架感到自豪:它兼具光源和摄像头支架的双重功能。我可信赖的 [Loigtech HD C920](http://www.logitech.com/en-us/webcam-communications/webcams/hd-pro-webcam-c920) 使用 [Joby Gorillapod](http://joby.com/gorillapod/find-my-gorillapod) 安装在那里，因此当我使用谷歌视频聊天工具或 Skype 通话时，相机靠近我的脸，并且通常在我进行播客采访或其他视频通话时处于一个很好的角度。

这就是我谈论外设的地方。我不幸的毒瘾。我仍然在为我的 Windows PC 摇动我的 [Corsair K70 RGB](http://gaming.corsair.com/en-us/corsair-gaming-k70-rgb-mechanical-gaming-keyboard-cherry-mx-red) ，如果你们有任何关于令人敬畏的照明配置文件的线索，我很乐意看到它们。在我的 Mac 上，我实际上一直在使用 [Das 键盘的 Division Zero X40](http://www.divisionzerogaming.com/x40/) ，这是他们不久前发给我的。我的意思是，看，这是一个游戏键盘，但我绝对喜欢他们定制的写作开关。他们感觉很像樱桃 MX 红，但他们是两倍的安静，这使得他们写的乐趣。甚至当我完成了我的评论，把我的 Corsair 换回我的 Windows PC 时，我还在继续写这个。我去年提到的那位更酷的大师诺瓦蒂埃·TKL(nova touch )仍然在四处游荡，就在我桌子下面的那个储物柜里，和我所有的其他键盘一起，等待着我想改变一下的时候。

但是鼠标的情况已经完全改变了。我从一个性能 MX 升级到了 [罗技的 MX 主](http://www.logitech.com/en-us/product/mx-master) ，比一个合适的继任者 多了。我从 Windows 上的漂亮的无线 [罗技 G602](http://gaming.logitech.com/en-us/product/g602-wireless-gaming-mouse) 换成了一台 [罗技 G502 Proteus Core](http://gaming.logitech.com/en-us/product/g502-proteus-core-tunable-gaming-mouse) ，因为我喜欢这种感觉，胜过我喜欢桌子上没有电缆的感觉。我知道，去年我说过“需要一些诱惑才能让我再在办公桌上挂一根线。”嗯， [那 G502 就是刚刚那个好的](http://lifehacker.com/logitech-mouse-showdown-performance-mx-vs-mx-master-v-1736936282) 。至于我的性能 MX 和 G602？别担心——它们和我养的其他十几只老鼠一起在我文件柜的抽屉里，因为很明显我有问题。 

#### 去生活帮，你最节省时间的捷径是什么？

这些年来，我告诉你要 [提前安排你的一天](http://lifehacker.com/start-work-fresh-tomorrow-by-slaying-your-most-unpleasa-5504649)[你的一周](http://lifehacker.com/front-load-your-work-week-to-reduce-stress-604463686) ，我还告诉你要坚决地处理你的待办事项——尽可能地坚决。但是今年我想强调的是: [积极捍卫你的界限](http://lifehacker.com/how-people-pleasers-can-learn-to-say-no-more-often-1524324151) 、你的个人空间和你的工作量。保卫自己的休息时间， [休假](http://lifehacker.com/stop-waiting-for-a-good-time-to-take-vacation-and-jus-513992056) ，不用工作就不工作。花时间和你爱的人一起做你喜欢的事情。

如果说我在进入 Lifehacker 后学到了什么的话，那就是灵活和被动是多么重要，但积极捍卫自己的界限也是如此。过去做项目经理帮了我很多，我仍然认为 [懒惰是强大的生产力武器](https://lifehacker.com/how-to-turn-your-laziness-into-efficiency-1648450688) 。它甚至适用于这里(现在就把事情做完，这样你以后就可以偷懒了)，但这仍然是一个微妙的平衡。说到生产力，这是我今年最常挂在嘴边的一句话:

> 生产力不仅仅是完成更多的工作。而是把你必须完成的事情做完，这样你就可以花更多的时间去做你想做的事情。
> 
> 你的时间很宝贵。花时间去做那些充实、激励、挑战和赋予你力量的事情。不要退而求其次。对你的时间和精力无所畏惧。这是我们仅有的了。

是啊，我们就这样吧。

#### 你最喜欢的待办事项管理器是什么？

Asana 仍然是我亲切地称之为“想法银行”的地方，它有不同的板块，用来存放文章和特色想法、宠物项目，以及其他一些我仍然保密的东西。只是作为一个玩笑， [周日摊牌](http://lifehacker.com/tag/sunday-showdown) 是那些创意银行项目中的一个，它作为一个创意的火花进入那里，并成熟为一些令人敬畏的东西。

Todoist 仍然是我的日常事务管理器，主要是因为我喜欢它处理重复事件的方式，它超级容易使用，并且几乎为你能想到的所有操作系统、移动操作系统、电子邮件应用程序或浏览器都提供了应用程序和插件。它无处不在，所以它一直在我身边，这正是我从一个待办 app 中需要的。

对于其他事情，比如购物清单、打包清单、笔记(比如我在健身房做了多少次什么重量的运动)，以及其他快速笔记和对自己的提醒，我都是关于 [谷歌保持](https://drive.google.com/keep) ，而 [仍然被难以置信地低估了](http://lifehacker.com/not-just-another-notes-app-why-you-should-use-google-k-509256637) 。

#### 你在哪些日常事务上比其他人做得更好？

即使过了这么多年，我还是不太喜欢这个问题，主要是因为我不认为我在这些事情上比其他人更好。我只是喜欢认为我擅长这些。去年我提到了适应性(和我的 BS 检测器)，但我不知道这个概念有多有先见之明，也不知道灵活和适应性在接下来的一年里会变得有多重要——以及它现在有多重要。我可能是我所知道的适应性最强的人之一，我每天都面临着挑战，要找到我应该愿意调整的东西和我应该划定的界限之间的界限。这是一场在关注我的界限 [和推进我的舒适区](http://lifehacker.com/the-science-of-breaking-out-of-your-comfort-zone-and-w-656426705) 之间的持续斗争。

这就是重点，不是吗？愿意弯曲而不折断，但保留你是谁，是什么让你，嗯，你。这是一种技能，而不是天赋，这是需要实践的东西——所以在某种程度上，我很感激拥有了这么多。

#### **你收到过的最好的建议是什么？**

**准备好回答了吗？你可以在这一点上谷歌一下。这里:**

> **早在 2006 年，我就在一种双重生活中挣扎。一方面，我在技术部门工作，正在读研究生，试图决定是否要在一个有企业技术的办公室工作谋生。另一方面，我已经开始自己写作了，并且真的想把它变成我的职业。我联系了 [Xeni Jardin](http://xeni.net/) ， [Boing Boing](http://www.boingboing.net/) (仍然是我最喜欢的网站之一)的创始合伙人和联合编辑，作为一个初出茅庐的作家，我突发奇想地向更有经验的人寻求建议。她回答道:**
> 
> **寻找那些不为人知的故事，那些让你着迷的事情，尽你所能诚实地告诉他们。如今这个世界上几乎没有真理，任何一点点真理都是珍贵的东西，会被欣赏，并找到一个感恩的观众。** 
> 
> **不要寻找“大”新闻，只寻找对你来说重要的新闻。你做得越多，你的作品就会越好，就会有越多的人看到它。** 
> 
> **我还在努力，但每天我都很感激她的话。我也想分享 8 世纪禅宗之父石头希谦的一句话:“浮云遮不住天。”我努力记住的其他事情，尤其是在这一行。**
