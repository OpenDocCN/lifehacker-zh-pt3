# 这个网站为父母匹配弹性工作制

> 原文:[https://life hacker . com/this-site-matches-parents-with-flexible-work-1828145783](https://lifehacker.com/this-site-matches-parents-with-flexible-work-1828145783)

这里有一个大胆的统计:43%的高素质女性 [在有了孩子](https://hbr.org/2005/03/off-ramps-and-on-ramps-keeping-talented-women-on-the-road-to-success) 后离开了职场。对于这个阵营中的许多人来说，这并不是因为他们失去了动力，或者他们无法兼顾吸管杯和销售电话，或者他们突然觉得注定要花每一分钟盯着他们的婴儿，就像他们被困在帮宝适广告中一样。更多的时候，是因为我们的传统结构把他们拒之门外。我不断读到这样的报道，有些女性在生理上或心理上还没有准备好就被抛回每周工作 50 小时的岗位，而一旦她们回来，就会遭遇*华盛顿邮报*专栏作家 Amy Nelson 所说的 [“母亲墙】、](https://www.washingtonpost.com/news/posteverything/wp/2018/01/29/i-started-my-own-business-i-still-couldnt-escape-stereotypes-about-working-moms/?utm_term=.7b70c4877e4d) 一种由对职业母亲的歧视而建立的障碍。最终，他们觉得有必要“选择退出”——这种描述从根本上来说是有缺陷的，因为其他选择留在哪里呢？

一家试图重新定义标准职业轨迹的公司是[Mom 项目](https://themomproject.com/) 。被描述为[【Airbnb for talent】](http://www.chicagotribune.com/business/ct-mom-project-0420-biz-20160419-story.html)这是一个职业平台，它将妈妈们(是的，爸爸们也是)与灵活的工作联系在一起——自由职业者或合同项目、远程职位、 [再入项目](https://www.workingmother.com/on-ramping-programs-for-working-moms) 和“产假”(填补父母休假造成的空缺的机会)。这些也不是你所熟悉的全职妈妈的工作——快速浏览后，我发现了一份基于合同的项目经理、一家广告公司的校对和一名远程 C++工程师的招聘信息。父母只是创建一个免费的档案，并与满足他们需求的机会相匹配。

有了孩子后，我的职业道路崎岖不平，充满了不充分的感觉，因为我的同龄人似乎天衣无缝地做着这一切。(事实并非如此，社交媒体是一个谎言。)我不能完全像生孩子前那样工作，但我可以工作，而且我可以做得很好。我最终意识到，呆在朝八晚五、坐在办公椅上的要求范围内，公司错过了像我这样的员工能带给他们的东西。(相信我——没有人比一个整天重复听《五只小鸭子》的妈妈更愿意接受智力挑战了。)Mom 项目和类似网站的目的——[après](https://apresgroup.com/)[第二班](https://www.thesecondshift.com/) 和兵团队的 [Mom 兵团](https://www.corpsteam.com/corps-team-home)——并不是简单地帮妈妈们一个忙。在工作场所找到更多的灵活性有助于家庭*和企业*的成功。并不是所有的父母都想在有了孩子后重返职场，但对于那些想重返职场的人来说，找出实现这一目标的方法符合每个人的最佳利益。
