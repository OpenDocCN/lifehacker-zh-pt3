# 你骗不了 YouTube 的版权机器人

> 原文：<https://lifehacker.com/you-cant-fool-youtubes-copyright-bots-1822174263>

如果你去 YouTube 看了一集电视节目的非官方上传，甚至是你最喜欢的动漫中的一个场景，你可能见过上传者为了阻止 YouTube 删除他们的视频而做的奇怪事情。你的节目可能会加快一点，声音变小，视频水平翻转或被数字降雪覆盖。也许你经历过，认识到这种降级的质量是避免 YouTube 的版权机器人的必要牺牲。坏消息是，这可能完全没有必要。

Watch

文化评论家迈克·鲁格内塔说，所有这些视频剪辑只是迷信。YouTube 依赖于与拥有电视节目、电影和音乐的公司的良好关系。因此，该网站花费了大量的时间和金钱来开发一个内容识别系统，它可以识别几乎任何人可以识别的东西。(大部分视频都贴上了正确的节目和剧集标题，这很有帮助。)他告诉 Lifehacker，“人们不遗余力地规避版权措施，认为他们打败了 Content ID，而事实上他们只是上传了 Content ID 并不寻找的内容。”

这些上传者可能是错的，但他们没有疯。几年前，欺骗 YouTube 的算法要容易得多。“我记得有人建立博客和整个一次性 YouTube 频道，只是为了测试这个系统，”Rugnetta 说。由于 YouTube 会关闭一个有太多版权问题的账户，频繁上传者的风险很高。

但随着 YouTube 的发展，它的资源也在增长，它需要让商业伙伴满意(并避免诉讼)。现在 YouTube [允许任何人用内容 ID](https://support.google.com/youtube/answer/2797370?hl=en) 注册他们的内容，并追捕侵权者。该算法在误报方面出错:该网站甚至 [删除了计算机生成的静态](https://gizmodo.com/man-s-youtube-video-of-white-noise-hit-with-five-copyri-1821804093) ，因为它听起来像一个现有的静态视频。Rugnetta 在他的播客《合理的声音》的一集 [【自动复制错误】](http://reasonablysound.com/2018/01/15/automated-copywrongs/) 中探讨了合理使用的寒蝉效应。

尽管如此，上传者尝试了各种各样的编辑技巧。Rugnetta 制作了一个动画上传的播放列表，扭曲了视频，试图逃避内容 ID。该名单已经发布两周了，八分之五的视频已经被屏蔽。

更多的上传者添加了像 [【无版权意图】](https://waxy.org/2011/12/no_copyright_intended/) 这样的免责声明，这些声明甚至更没用，因为没有人做出这些判断——只有无情、无情的算法。如果内容“超过”内容 ID，那是因为内容所有者没有寻找它——或者告诉 YouTube 让侵权上传保持不变。内容所有者可以选择在视频上放一些广告并从中获利。

好消息是，许多内容所有者选择了更宽松的选项。有些内容甚至没有广告。Rugnetta 指的是 TheFlamingShark，他上传流行动漫的完整片段，嵌入他实时观看和反应的视频。

这种类型被称为“盲目反应”，它处于合理使用的灰色地带。这可能意味着没有版权所有者曾经挑战过视频共享，或者他成功地证明了他的视频是评论性的，因此是合法的合理使用。

Rugnetta 认为发生了三件事之一:

> 1.这家伙得到了 DMCA，反注意，并赢得了
> 
> 2.这家伙不把他的视频货币化，不管是谁发布的，牛仔-Bebop 制定的规则说“如果他们不在我们的作品上做广告，随他们去吧，没关系”
> 
> 3.星际牛仔不在指纹数据库里

“我认为不太可能的是，”他说，“一个人看了这个视频，用他们的大脑决定，‘这是合理使用，随它去吧。’“至于算法，这些解说视频肯定骗不了它。要么是版权所有者没有关闭未经授权的上传，要么是他们关闭了，而黑客不得不努力让他们恢复。(TheFlamingShark 尚未回复置评请求。)

所以当你上传一些属于别人的内容时，不要放慢速度或者翻动它。你可以试着添加你自己的视频评论，但是你可能要和一个删除通知作斗争。如果你输了，你可能会失去你的频道。

话又说回来，你总是可以把 [上传到 Bittorrent，而不是](https://lifehacker.com/how-to-share-your-own-files-using-bittorrent-5534190) 。
