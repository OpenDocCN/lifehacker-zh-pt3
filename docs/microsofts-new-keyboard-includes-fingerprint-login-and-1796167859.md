# 微软的新键盘包括指纹登录——和糟糕的人体工程学

> 原文：<https://lifehacker.com/microsofts-new-keyboard-includes-fingerprint-login-and-1796167859>

微软巧妙地将指纹识别器集成到其最新的键盘配件中，但删除了一些使其键盘系列易于使用的重要方面。它非常适合快速登录，看起来肯定很酷，但是在上面打字可能不是最舒服的体验。



### 专为一键式登录设计

新款 [售价 129 美元的现代键盘](https://www.microsoft.com/accessories/en-us/products/keyboards/microsoft-modern-keyboard-with-fingerprint-id) 在右手 Alt 键的右边有一个巧妙伪装的指纹传感器。它的指纹传感器兼容 [Windows Hello](https://www.microsoft.com/en-us/windows/windows-hello) ，这是微软的生物识别登录功能，可以让你用你的面部或指纹进入你的 Windows 设备(不过你需要兼容 Windows Hello 的硬件)。一旦您注册了指纹，您就可以使用 Windows Hello 指纹验证登录，而无需使用您的密码，从而节省一些时间，并避免与偶尔的记忆失误相关的问题。

它还适用于 Windows Hello 支持的应用程序，因此您只需按一个键就可以批准购买应用程序。通过有线和蓝牙连接，这似乎是一个智能更新，可以让更多人使用 Windows Hello，而无需购买新的笔记本电脑或昂贵的网络摄像头来添加到您的 PC 上。

根据其规格，它完全兼容 MacOS 10.10.5、10.11.1 和 10.11.4。它的 Android 和 iOS 兼容性选择很奇怪，因为它只能与 iOS 8 和 9、Android 4.2 设备、Nexus 5 上的 Android 4.4.4 和 Nexus 9 上的 Android 5.0 兼容。

### 人体工程学的噩梦

看起来不错，但很有可能会感觉不好。微软的键盘遭遇了困扰苹果桌面键盘(以及几乎所有其他键盘)的同样问题:人体工程学。

这很奇怪，因为微软通过发布两种不同的，看起来很奇怪的键盘解决了键盘的人体工程学问题。 [Surface 人体工程学键盘](https://www.microsoft.com/en-us/surface/accessories/surface-ergonomic-keyboard) 有一个 [掌托](http://lifehacker.com/wrist-rests-are-for-your-palms-not-your-wrists-1672756283#_ga=2.10959348.1996763788.1497012067-1203953178.1494430188) 覆盖着 alcantara 面料，这是微软在其 Surface Pro 键盘盖上添加的一种耐用面料，以及一种弯曲的结构，让你的手腕保持在更中性的位置。甚至空格键也是弯曲的，切成两半以适应你的每个拇指。

微软新的现代键盘借鉴了苹果的键盘设计。他们抛弃了 alcantara 面料的——那是相当丑陋的。但他们也摆脱了掌托的其余部分——这是一个更令人困惑的设计选择。据我所知，掌托在防止你的桌子边缘卡住你的手方面非常有用。
