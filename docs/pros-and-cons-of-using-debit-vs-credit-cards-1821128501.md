# 信用卡和借记卡的利弊

> 原文:[https://life hacker . com/使用借记卡与信用卡的利弊-1821128501](https://lifehacker.com/pros-and-cons-of-using-debit-vs-credit-cards-1821128501)

根据 Chime 2015 年的一份报告，近 70%的千禧一代 [更喜欢借记卡而不是信用卡](https://www.chimebank.com/2015/10/01/2015-millennial-money-mindset-report/) ，而 Transunion [报告称](https://www.creditcards.com/credit-card-news/transunion-millennials-gen-xers-card-study.php) 他们比老一代人拥有更少的银行发行的和私人标签卡。

可以理解他们可能会很谨慎——信用卡债务在美国是一个严重的问题，一旦你陷入其中，就很难自拔。信贷行业难以置信的不透明和掠夺性。但是，信用卡也是一个重要的工具，当负责任地使用时，可以帮助你改善一生的财务状况。

这里有一些关于借记卡和信用卡的区别要记住。

### 借记卡

优点:

*   钱是自动取出来的，所以可以让日常开销更加透明。
*   你可以设置自动转账到你的储蓄账户。
*   你可以从自动提款机或其他自动售货机提取现金。
*   消费不影响你的信用评分。

缺点:

*   它不如信用卡安全，尤其是在网上购物时。发卡机构对欺诈性信用购物的退款速度要比借记购物快得多。
*   如果您的 [卡](https://www.consumer.ftc.gov/articles/0213-lost-or-stolen-credit-atm-and-debit-cards) 被盗，并且您“在得知丢失或被盗后超过 2 个工作日，但在您的对账单发送给您后不到 60 个日历日”才报告失窃，您可能需要承担高达 500 美元的欺诈费用。
*   你可以透支，但必须支付相应的费用(平均每笔资金的 [为 33.38 美元](http://www.bankrate.com/banking/checking/checking-account-survey/) )。
*   消费并不能提高你的信用评分。

### 信用卡

优点:

*   积分、奖励和返现增加了你已经花费的钱的价值，以及所有不同的类型——旅行、返现等。—让您优先考虑对您重要的事情。
*   比借记卡更安全，尤其是在网上购物时。由于《公平信用账单法》(FCBA) 【T3)，在你报失后，你最多需要支付 50 美元，而一些公司会免除这一费用。

*   如果你的卡被盗，更容易得到退款。
*   可以用来建立信用评分，以帮助借款——你的支付历史是你的 FICO 评分最重要的组成部分(占 35%)，但你的信用历史的长度(即你拥有信用卡的时间)也是考虑的因素。当你在一生中为房子或汽车贷款时，这可以为你节省数千美元的低利率。

缺点:

*   如果你过度消费，很容易陷入债务:根据 NerdWallet 的数据，有信用卡债务的普通家庭的余额为 [、16883](https://www.nerdwallet.com/blog/average-credit-card-debt-household/)。
*   很难比较和对比每一种卡，以及它如何帮助(或伤害)你。
*   如果你花钱不小心，会损害你的信用评分。
*   许多人不知道如何最大化他们的分数，偿还他们的债务，甚至不知道他们的利率是多少或如何运作。
