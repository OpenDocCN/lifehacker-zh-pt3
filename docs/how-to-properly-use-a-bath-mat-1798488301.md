# 如何正确使用浴室防滑垫

> 原文：<https://lifehacker.com/how-to-properly-use-a-bath-mat-1798488301>

哦，伟大的浴垫，使用你的适当礼仪是什么？我们问了你一些很棒的人 ，这是你告诉我们的。

Watch

首先，每个人都有一些很好的建议，告诉我们为什么要使用浴室防滑垫。是的，它们吸收水分，但它们也 [防止我们滑倒并撞破我们的头穿过马桶](http://lifehacker.com/1798346703) ， [作为我们脚趾在热水淋浴和冰冷地板](http://lifehacker.com/1798349252) 之间的温度缓冲。哎呀，浴室垫子真漂亮！

说到用法，普遍的共识是这是错误的做法:

*   结束淋浴

*   走到浴室垫子上

*   抓起毛巾

*   然后擦干

不管谁在你之后洗澡，它都会让浴室垫变得湿漉漉的。它还会让你在干燥过程中感觉更冷。事实上，大多数提到这种方法的人都在抱怨:
