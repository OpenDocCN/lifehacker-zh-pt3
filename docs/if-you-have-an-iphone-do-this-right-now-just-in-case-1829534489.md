# 如果你有 iPhone，现在就这么做——以防你被警察拦下

> 原文：<https://www.theroot.com/if-you-have-an-iphone-do-this-right-now-just-in-case-1829534489>

iPhone 用户已经创建了一些快捷方式，允许苹果的旗舰产品自动录制视频，向紧急联系人发送位置短信，甚至阻止警察进入手机——以防 iPhone 用户与执法人员进行互动。

Watch

苹果最近推出了 iOS 12 的“快捷方式”，这款应用允许 iPhone 用户为自己的手机设计自动命令。尽管这个概念并不新鲜(作者使用了一个先驱，如果这是那么那就是)，但它作为默认应用程序的存在促使用户设计了一系列有用的语音激活工作流，包括日历提醒、电子邮件通知，甚至计算小费的方法。

[据 Mic](https://mic.com/articles/191674/pulled-over-by-police-siri-can-quietly-record-video-and-send-your-location-to-an-emergency-contact#.d613RbSrz) 报道，Reddit 用户 Robert Peterson 使用虚拟助手 Siri 创造了一个小技巧，降低手机亮度，打开勿扰模式，将 iPhone 主人的位置发短信给紧急联系人，让他们知道你被警察拦下了。该快捷方式还会自动开始录制视频，完成后，手机会将视频发送给联系人或保存到云服务。

[快捷方式在这里](https://www.icloud.com/shortcuts/c3445ba6ecf54adbb393f5c5989e5182) 可用，而另一个用户创建了一个 [工作流，自动重启手机](https://www.icloud.com/shortcuts/97bb4f2b68f6417bad0e6ac43df97658) ，使得指纹或面部识别功能无用，直到有人输入密码。《华盛顿邮报》报道称，尽管警方可以强迫手机用户使用指纹识别或面部扫描，但他们不能合法地强迫嫌疑人放弃密码。

“我在新闻报道和电视报道中注意到，在许多情况下，警察说发生了一件事，而靠边停车的公民说的是另一件事，”彼得森告诉 Mic。“有时警察有人体摄像机，有时没有。当他们这样做时，视频并不总是及时发布。我想让被拦下的人有一个自己的记录。”

我已经下载了快捷方式，但我一直无法找到一个阻止子弹或定罪的警察。
