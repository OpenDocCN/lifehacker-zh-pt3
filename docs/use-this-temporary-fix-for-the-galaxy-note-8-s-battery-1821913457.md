# 对 Galaxy Note 8 的电池问题进行临时修复

> 原文：<https://lifehacker.com/use-this-temporary-fix-for-the-galaxy-note-8-s-battery-1821913457>

12 月下旬， [Galaxy Note 8](https://lifehacker.com/how-to-install-the-android-8-0-oreo-beta-on-your-galaxy-1821265769) 车主开始报告 [手机电池](https://lifehacker.com/top-10-ways-to-improve-the-battery-life-on-your-phone-a-1725843518) 出现离奇问题。如果电量下降得太低，它就无法充电，一旦电池耗尽，就根本无法使用设备。



几天后，三星 [证实了](http://bgr.com/2018/01/01/galaxy-note-8-battery-drain-issue/) 的漏洞，声称它只影响一小部分设备。尽管如此，自 2017 年 8 月推出以来，该公司已经售出数百万部 Note 8s，该问题似乎也困扰着一些 [Galaxy S8 和 S8 Plus 设备](https://www.xda-developers.com/samsung-galaxy-note-8-deep-discharge-protection/) 。因此，即使很小的比例也可能意味着成千上万的人处于危险之中。

三星尚未发布官方修复方案，但与此同时，有一个第三方解决方案可以防止手机电池电量下降太低。 [深度放电保护工具](https://forum.xda-developers.com/galaxy-note-8/themes/g95-deep-discharge-protection-t3731103) 承诺做到这一点，当你的电池开始下降时警告你，并在你越过不可逆转点之前当电池电量达到 12%时完全关闭你的手机。

一个小警告是，你的手机需要 [根](https://lifehacker.com/everything-you-need-to-know-about-rooting-your-android-5789397) 自动关机功能才能工作。当电池电量开始下降时，你仍然会收到一连串的两声嘈杂的警报，开发人员说，如果你正在睡觉，它们的声音应该足以吵醒你。他们还在考虑让整个程序在非根三星手机上运行。

你现在就可以从 XDA 开发者论坛 的 [下载并安装该应用。希望三星很快发布一个官方修复程序(可能在它的](https://forum.xda-developers.com/galaxy-note-8/themes/g95-deep-discharge-protection-t3731103) [Android 8.0 更新](https://lifehacker.com/how-to-get-the-galaxy-s8-android-8-0-oreo-update-ahead-1821878689) )，但在那之前这是你最好的选择。
