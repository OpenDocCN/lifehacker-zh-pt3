# 停止为轻度二手车支付过高的价格

> 原文：<https://jalopnik.com/stop-overpaying-for-lightly-used-cars-1828576980>

我有一条新闻要告诉你们中的一些人:经销商喜欢那些坚持二手车总是比新车更好买的人。大多数情况下可能是这样，但并不总是这样，事实上，一些买家在寻找一些只有几年历史的低里程数的东西时支付了太多。



我已经在各种帖子中讲述了无数的购车神话， [其中一些似乎不会消失](https://jalopnik.com/these-common-car-buying-myths-refuse-to-die-1828248263) ，但仍然真正流行的一个是，所有二手车都比新车更划算。这种建议的逻辑通常出现在某人说的话中，“好吧，你一开出停车场，汽车就损失了 40%的价值，所以最好让别人来承担折旧。”

虽然这个论点是合理的，但缺失的部分是“平均而言，汽车会损失 40%的价值”，因为不是所有的汽车都以同样的速度贬值。豪华车比非豪华车的加速折旧要大得多，这就是为什么买一辆经过认证的二手车宝马、奥迪、雷克萨斯或其他什么车能提供很大的价值。

但对于一些寻找主流品牌的人来说，他们过于关注二手车的节省，他们不会购买新车，有时会错过更好的交易。

这里有两个案例可以说明这一点。

我在西雅图有一个客户，他刚刚搬家，正在寻找一辆二手斯巴鲁。他喜欢高级装饰，想要价格在 25，000 美元以下、行驶里程合理的东西。对该市场汽车的快速横截面显示，2016 年款的平均价格约为 24，000 美元，看起来很容易适应。

然而，随着 2019 款 Outbacks 即将上市，经销商几乎是以几乎前所未有的折扣赠送 2018 款汽车。 我在一个零售价为 29，854 美元的内地网站上买到了售价为 24，776 美元的商品。

这比标价低了 5000 美元，而新车只比两年前的二手车平均价格高出 776 美元。当你考虑到销售价格可以与低年利率融资相结合时，购买二手似乎是愚蠢的。

现在斯巴鲁的情况并不完全普遍，因为事实上我在新车上得到了很大的折扣，但即使你只能在新车型上得到适度的折扣，它也足以弥补差距，证明为新车型多付一点钱是合理的。

我有另一个客户在洛杉矶市场寻找二手 GTI。虽然他可以轻松地以不到 2 万美元的价格获得 30，000 英镑或更多的 2015 款，但他不想要里程数很多的东西，而是希望拥有苹果 CarPlay，这将他限制在 2016 款和更高的型号。在 200 英里内的搜索没有发现一吨的汽车，它们都是 2017 年的 trim 车型，平均价格在 2.3 万美元左右。

我在新车型上看到了一些非常激进的广告价格，所以我想看看 2018 款车型可能会有什么样的价格。我拿到的最好价格是 24799 美元。

虽然这比二手车更贵，但当你考虑到 2018 年的汽车有一些升级的技术，造型和更长的保修期时，1800 美元的溢价似乎是值得的。

所有这些都意味着:下次你在购买一辆不像豪华品牌那样大幅贬值的二手车时，考虑向新车型开放网络，因为差距可能比你想象的更近。
