### 为什么有人会关心 Animoji？

在我们开始之前，你可能想知道苹果最新的噱头功能有什么大不了的。Animoji 是一种动画表情符号，它使用 iPhone X 的 [Face ID 摄像头](https://lifehacker.com/what-you-need-to-know-about-face-id-on-the-iphone-x-1804641406) 模仿你的面部动作。这听起来没什么大不了的，但它正迅速成为一个不可避免的迷因。

通过 [Twitter](https://twitter.com/petegarber/status/926523150078226432) 或 [YouTube](https://www.youtube.com/watch?v=-G5XWVFoWO0) 搜索，你会发现无数的 Animoji 视频。它甚至引发了一种叫做 [Animoji 卡拉 ok](http://mashable.com/2017/11/06/animoji-karaoke-roundup/)的新趋势，在这种趋势中，人们录制冗长的唇同步视频——通常带有和声和多个表情符号。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-5ob2LRPMsjI&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-5ob2LRPMsjI&start=0) 

结果很有趣，但做出结果并不容易。你必须进入 Messages 应用程序，使用 iOS 11 中的屏幕录制功能来捕捉更长的视频。然后你需要用一个单独的程序把它们编辑在一起。

进入 SBSAnimoji，一个由开发者 Simon B Stø vring 开发的非官方应用程序，它应该会让事情变得简单一些。

### 如何下载 SBSAnimoji

苹果公司尚未向第三方开发者开放 Animoji，因此斯特沃林的应用程序通过访问该公司的私有 API 来工作。这意味着你肯定不会很快在应用商店里找到 SBSAnimoji。

相反，你必须 [从 GitHub](https://github.com/simonbs/SBSAnimoji) 下载应用程序，然后 [将其](https://lifehacker.com/how-to-install-unapproved-apps-on-an-iphone-without-jai-1749519150) 下载到你的 iPhone X 上。你还需要创建一个免费的开发者帐户( [9to5Mac](https://9to5mac.com/2016/03/27/how-to-create-free-apple-developer-account-sideload-apps/) 如果你需要帮助，会有一步一步的指导)。

正如我们所说，这不是最简单的过程，但当你用 20 秒长的 Animoji 视频打动你的朋友时，这是值得的。
