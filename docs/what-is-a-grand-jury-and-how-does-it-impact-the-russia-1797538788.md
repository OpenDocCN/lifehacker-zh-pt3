# 什么是大陪审团，它对俄罗斯调查有何影响？

> 原文：<https://lifehacker.com/what-is-a-grand-jury-and-how-does-it-impact-the-russia-1797538788>

据 [【华尔街日报】](https://www.wsj.com/articles/special-counsel-mueller-impanels-washington-grand-jury-in-russia-probe-1501788287)报道，司法部特别法律顾问罗伯特·穆勒因俄罗斯干预 2016 年总统选举召集大陪审团。这个陪审团将让穆勒寻求传票——迫使人们在法庭上作证，并为调查提供文件。

大陪审团已经在华盛顿特区工作了几个星期，并就小唐纳德·川普去年与一名俄罗斯律师的会面发出了传票 ，报道 [路透社](https://www.reuters.com/article/us-usa-trump-russia-idUSKBN1AJ1SW) 。

### 什么是大陪审团，他们与其他类型的陪审团相比如何？

大陪审团由随机挑选的公民组成，他们查看可能的犯罪行为的证据，并决定是否对被告提出刑事指控。所有联邦重罪指控都需要大陪审团，但州和县检察官也可能使用大陪审团。

大陪审团可以发出传票，强迫证人在法庭上宣誓作证。联邦大陪审团通常有 16 至 23 人，而审判陪审团通常有 12 人。大陪审团也持续更长时间(至少持续几周，但通常持续几个月，有时甚至几年)，并且不公开。

与其他陪审团不同，大陪审团不听取辩方的意见。相反，陪审员和联邦检察官可以向证人提问，证人在提问时不允许有律师在场。利用这些信息，大陪审团决定某人是否有可能犯罪，而不是(如在审判陪审团中)某人是否有罪。如果大陪审团没有起诉被调查的人，检察官仍然可以指控那个人犯罪，如果他们说服法官那个人可能是有罪的。

### 关于俄罗斯的调查，这说明了什么？

根据 [希尔德](http://thehill.com/homenews/administration/345271-muellers-grand-jury-what-it-means) ，穆勒召集大陪审团并不奇怪。法律专家表示，事实上，当特别调查人员受命调查白宫可能存在的不当行为时，这是意料之中的。正如一名华盛顿辩护律师告诉 [美联社](https://www.nytimes.com/aponline/2017/08/04/us/politics/ap-us-trump-russia-probe.html) “大陪审团的使用既不会升级，也不会为调查设立时间表。”

但调查小特朗普与俄罗斯会面的大陪审团并不是第一个调查俄罗斯与总统关系的陪审团。在穆勒接手调查之前，联邦检察官在弗吉尼亚州亚历山大召集了另一个大陪审团，调查特朗普前国家安全顾问迈克尔·弗林的商业交易和竞选联系。

新的大陪审团是调查正在升级的一个迹象——这表明俄罗斯的调查已经从弗林扩大到特朗普的几个家庭成员以及现任和前任助手，并且可能会继续多几个月。这也意味着刑事指控是可能的(而且穆勒有可能犯罪活动的证据)。目前还不清楚调查的目标是谁。

一个大陪审团在 [发出传票，调查希拉里的电子邮件服务器](http://www.politico.com/blogs/under-the-radar/2017/04/27/hillary-clinton-emails-subpoenas-fbi-237712) ，但从未指控她有任何不当行为。也有大陪审团调查水门事件中的理查德·尼克松和白水事件中的比尔·克林顿。

### 特朗普是如何回应的？

作为对调查的回应，川普称这是一个“完全虚构的俄罗斯故事”，并表示民主党人正在利用它作为“美国政治史上最大损失的借口”，周四在西弗吉尼亚州的集会上。

与此同时，特朗普的律师一直在调查穆勒团队可能存在的利益冲突，并警告说，调查特朗普的财务状况与调查无关。特朗普的律师杰伊·塞库洛(Jay Sekulow)表示，大陪审团参与调查没有什么可担心的，称这是“当你遇到这种情况时的标准操作程序”。
