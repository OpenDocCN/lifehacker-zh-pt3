# Mac 用户:你应该把你的 Dock 放在哪里？

> 原文：<https://lifehacker.com/mac-users-where-should-you-put-your-dock-1797093527>

Mac 用户，你好！让我们来讨论一下你的屏幕的哪一边最适合你的 dock。(PC 用户请忽略这一块，不要取笑我们。)



普遍的共识是，只有肮脏的临时工离开底部的码头，在那里它吃掉了太多宝贵的垂直房地产。在一个 [Reddit 的帖子里](https://www.reddit.com/r/apple/comments/dwqpc/how_many_of_you_badasses_put_your_dock_on_the_left/) 关于 dock 的安置，上面的评论者，下面的 [随便讨饶](https://www.reddit.com/r/apple/comments/dwqpc/how_many_of_you_badasses_put_your_dock_on_the_left/c13iehd/) 。Bottom dock 用户指出，有了自动隐藏，空间就不是问题了。当你可以用 Spotlight 启动应用程序时，谁还需要 dock 呢？

苹果历史学家斯蒂芬·哈克特 [更喜欢右边的](https://512pixels.net/2012/08/on-that-dock-placement-thing/) ，紧挨着 OS X 自动新增文件的那点桌面。他称底坞“浪费宝贵的垂直空间”，并发现自动隐藏令人迷失方向。他将 dock 向下推到右下角，以获得更多空间。

iDownloadBlog 的作者杰夫·本杰明制作了一个六分钟的视频为左派辩护。按照意义递减的顺序，它可以归结为三点:

1.垂直地产弥足珍贵，雅哒雅哒。

2.大多数人都是右撇子，所以他们的手倾向于触控板的右侧，所以左侧的 dock 不会碍事。

3.说英语的人从左向右阅读，所以把 dock 放在左边就像把它放在“开始”一样

AppAdvice 博主 Daniel Celeste [为底部 dock](https://appadvice.com/appnn/2015/02/to-the-side-dock-people-why-my-macs-dock-stays-on-the-bottom) 辩护，指出如果你在桌面上平铺多个窗口，水平空间会更加珍贵。如果你在多个屏幕上玩杂耍，只有一个底部的 dock 会出现在所有的屏幕上。

其中一位博主是对的。其中两个是错的。选择权在你。
