# 成功家庭游戏之夜的最佳棋盘游戏

> 原文：<https://lifehacker.com/the-best-board-games-for-a-successful-family-game-night-1819399430>

在一个以屏幕为中心的世界里，桌游对一些人来说似乎已经过时了，尤其是对孩子们。但在我们家，我们发现放下平板电脑和手机，玩玩桌游(和一大碗爆米花)会让家庭之夜很成功。我们的游戏从 2 岁到 11 岁不等，复杂程度各不相同。小贴士:让大孩子和小孩子一起合作！



这里有一些我家人最喜欢的游戏，包括经典游戏和新游戏，按年龄组分类。

### 学龄前儿童

[***坎迪兰***](https://www.amazon.com/Candy-World-Sweets-Amazon-Exclusive/dp/B00000DMF5?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-best-board-games-for-a-successful-family-game-night-1819399430&asc_source=&tag=kinjalifehackerlink-20)

因为不需要阅读,《糖果乐园》是学龄前儿童的完美游戏。所需要的只是跟随指示的能力以及一些基本的计数技巧。目标是找到糖果王国失落的国王康提国王，每个玩家轮流抽一张牌，这张牌可以让他们进入下一个指定颜色的空间。糖果主题的地点和角色出现在整个棋盘游戏中，我的孩子们认为奶奶纳特很可笑，因为谁会想住在花生糖做的房子里呢？他们喜欢一遍又一遍地演奏这首曲子。

[***完美***](https://www.amazon.com/Hasbro-C0432-Perfection-Game/dp/B01NALEP04/ref=dp_ob_title_toy?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-best-board-games-for-a-successful-family-game-night-1819399430&asc_source=&dpID=61oC%2BXWF2FL&dpSrc=detail&preST=_SY300_QL70_&tag=kinjalifehackerlink-20)

这个游戏上计时器的滴答声让每个人都紧张兴奋地想玩。完美是一个配对游戏，玩家有 60 秒的时间将所有 25 个棋子放入他们配对形状的空间，然后向下推。如果你把所有的棋子都放好了，你可以关掉定时器，但是如果定时器到时你还在工作，你放好的棋子会突然不在位置上，到处都是。不管我们玩多少次这个游戏，计时器到时散落的碎片总是让我的孩子吃惊，他们喜欢这个游戏。

[***序为娃儿***](https://www.amazon.com/Jax-8001-Sequence-for-Kids/dp/B00005UM9P?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-best-board-games-for-a-successful-family-game-night-1819399430&asc_source=&tag=kinjalifehackerlink-20)

与糖果乐园不同，糖果乐园只关注简单地遵循方向并向前移动，儿童序列鼓励孩子们预测其他玩家在轮到他们时可能会做什么。每个玩家都有三张牌和一堆游戏筹码。轮到你的时候，你把你的一个彩色筹码放在棋盘上与你的牌相匹配的地方:一个小动物的图像。你每使用一张牌，弃掉一张新的。为了获胜，您必须连续获得四个空格。当你用你的筹码阻止对手连续四次的企图时，这个策略就出现了。哥哥姐姐可以帮助弟弟妹妹掌握这些更复杂的技能。

### **初小学**

[***垄断少年***](https://www.amazon.com/Hasbro-A6984-Monopoly-Junior-Board/dp/B00EDBY7X8?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-best-board-games-for-a-successful-family-game-night-1819399430&asc_source=&tag=kinjalifehackerlink-20)

正规大富翁的游戏可以持续很久，很快失去年轻玩家的注意力，所以我们喜欢大富翁少年较小的棋盘。少年版不仅仅是像“大富翁”这样的街头场所，还拥有滑板公园和玩具店等属性，使其更具关联性。初级版本的钱数要低得多，这让年轻玩家更容易计算。财产仍然可以买卖，游戏结束时，只有一个玩家必须破产，而不是所有剩余的玩家。赢家是第一次破产发生时钱最多的玩家。大富翁初级版是孩子们练习预算技巧的好方法，但是不要告诉孩子们这就是我喜欢它的原因。

[***赫德班兹***](https://www.amazon.com/HedBanz-Game-Edition-may-vary/dp/B003AIM52A?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-best-board-games-for-a-successful-family-game-night-1819399430&asc_source=&tag=kinjalifehackerlink-20)

这个游戏会很吵，所以当你把宝宝放下来小睡一会儿的时候，不建议玩这个游戏，但是这个游戏很有趣。每个玩家都有一个塑料头带，并发给一张面朝下的卡片，然后他们将卡片放在头带中，这样其他人就可以看到卡片上的图像。设置好计时器后，玩家试着问其他玩家一个问题，让他们猜猜卡片上的图像是什么。唯一被禁止的问题是，“我是什么？”合作和交流是 Hedbanz 成功的必备条件。大部分乐趣可以在玩家错误的假设中找到，甚至爸爸妈妈也设法做出一些可笑的猜测。

[***对不起！***](https://www.amazon.com/Hasbro-390095-Sorry-Amazon-Exclusive/dp/B00000IWD0?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-best-board-games-for-a-successful-family-game-night-1819399430&asc_source=&tag=kinjalifehackerlink-20)

虽然对这个游戏进行了一些更新，但我们家仍然玩经典版的抱歉！对于孩子们来说，这是一个很棒的游戏，因为他们越来越习惯于不总是赢，甚至被淘汰出局。每个玩家有四个棋子开始，为了开始，必须抽一张或两张牌。在那之后，玩家移动他们的棋子到棋盘上。然而，如果另一个玩家落在同一个地方，他们会把你的兵撞回起点。棋盘上还有滑梯可以迫使你的棋子后退几格。总的来说，这个游戏可能会变得激烈，但这是一个练习良好的体育精神以及优雅输球的好方法。

### **小学到中学后期**

[](https://www.amazon.com/Rummikub-Original-Rummy-Tile-Game/dp/B00000IZJB?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-best-board-games-for-a-successful-family-game-night-1819399430&asc_source=&tag=kinjalifehackerlink-20)

**我的孩子喜欢这个游戏。年幼的孩子开始和他们的哥哥姐姐一起组队学习，而年长的孩子从小时候和大人一起组队时就开始玩了。每个玩家从牌堆中随机挑选 14 张牌，并把它们排列在他们的棋盘上。为了“出柜”，玩家的总值必须达到 30。从那时起，玩家必须有一串相同颜色的连续号码，或者不同颜色的相同号码。第一回合后，你可以在另一个玩家的回合或回合中加入一两个牌，以便继续游戏。当第一个玩家用完牌时，游戏结束，剩下牌的玩家必须为他们剩下的所有牌扣分。**

**[***金字塔***](https://www.amazon.com/Pyramix-Three-Sided-Strategy-Game/dp/B00JRGG3ZE?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-best-board-games-for-a-successful-family-game-night-1819399430&asc_source=&tag=kinjalifehackerlink-20)**

**这是一个相对较新的除了我们的棋盘游戏收藏，但迄今为止，我们很喜欢它。棋盘以金字塔形状堆叠的棋子开始，每个玩家轮流移除一个立方体。每个立方体颜色都有一个点值，只要它不接触 cobra 立方体，并且可以看到立方体的两个或三个面，您就可以删除它。当你取出立方体时，其他立方体越来越靠近底部，直到只剩下一层，这就是游戏结束的时候。然后每个玩家把他们的颜色加起来，如果你有最高数量的颜色，你可以收集底部剩下的相同颜色作为你的个人收藏。把你的分数加起来，找出获胜者！**

**[***线索***](https://www.amazon.com/Winning-Moves-Games-1137-Classic/dp/B00349MPQQ?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-best-board-games-for-a-successful-family-game-night-1819399430&asc_source=&tag=kinjalifehackerlink-20)T7】**

**对于家庭游戏之夜来说，一款谋杀悬疑游戏看起来确实有点恐怖，但愚蠢的名字甚至武器都设法淡化了一个严肃的主题。线索的目标是找出谁是凶手，使用了什么武器。每个玩家在棋盘上走一个回合，最后出现在都铎王朝的九个房间中的一个。做笔记是在 Clue 上取得成功的一个重要部分，我的孩子们通过玩变得更擅长演绎推理，即使他们大多只是注意到诸如 Plum 教授和上校 Mustard 这样的愚蠢名字。**
