# 徒步旅行者步道礼仪指南

> 原文：<https://lifehacker.com/the-hikers-guide-to-trail-etiquette-1827102953>

可能感觉在户外没有任何规则，但肯定有你应该遵循的指导方针。无论你是徒步旅行的新手，还是一直对这条路线的该做和不该做感到困惑，这本指南都会让你明白。

### 上山的徒步旅行者有先行权

大多数时候，爬山比下山更难。当人们爬山时，他们往往视野较小，因为他们通常向下看自己的脚。这就是为什么让徒步旅行者在上坡时有优先通行权是很重要的。如果你走在一条狭窄的小道上，看到徒步旅行者走过来，站到一边，让他们继续走，不要打断他们的步伐。也就是说，这是爬山者的决定。如果他们选择停下来向你挥手，那就去做吧。

### 注意山地车手和马匹

一般来说，山地自行车手应该向所有徒步旅行者屈服(他们知道永远不要指望你屈服)。也就是说，通常最好的办法是观察它们，当你看到它们的时候，靠边站，尤其是当它们正在沿着小路飞行或者努力攀爬陡坡的时候。

如果小道上有马，每个人都应该服从它们。给他们让路，给他们尽可能多的空间，避免他们经过时做出任何突然的动作或大声的噪音。你不想让一匹受惊的马踢你的牙齿。

### 在一群人里排成一列徒步旅行

集体徒步旅行是一项有趣的活动，无论是与朋友、家人还是某种组织一起，但你不想堵塞小路，破坏其他人的乐趣。如果一条小路很窄(大多数都很窄)，那就排成一列纵队走。然而，如果一条小径很宽，你可以留出空间让人们通过，挨着走也没关系。

### 最慢的徒步旅行者应该领路

当你作为一个团队徒步旅行时，最好让每个人都在一起。如果你让最快的徒步旅行者带路，他们会把所有人都甩在后面。如果你有最慢的徒步旅行者在后面，他们可能会迷路或受伤，没有人会知道。为了安全起见，让最慢或最没有经验的徒步旅行者领路。这样，如果他们必须停下来，整个团队可以一起停下来。想一想——最慢的徒步旅行者可能比其他人更需要休息。但是如果他们在后面，他们只有时间在团队停下来的时候赶上进度，而且他们永远不会真正休息。有一句老话值得我们牢记在心:“我们走得和最慢的人一样快。”

然而，应该指出的是，这种集体徒步旅行的方法还有待商榷。对于一些团体来说，这是不必要的，对于有经验的徒步旅行者来说，被困在一个慢吞吞的人后面是令人沮丧的。有时候最好把一个团队分成几个小组，这样走得快的人可以按自己的速度走，走得慢的人可以慢慢来。问问你的团队他们的感受，看看什么适合你。

### 注意你的音量

人们去远足是为了锻炼身体，但也是为了接触大自然和许多宁静的声音。没人想听到你大喊大叫制造回声，互相大声说黄色笑话，或者在手机上播放音乐。当你徒步旅行时，从背包里的蓝牙扬声器播放音乐是非常不礼貌的。带上耳机，兄弟。没人关心你最新的果酱是什么——我们想听鸟叫、水声和穿过树林的风声。

### 继续追踪

当你徒步旅行时，你会不断地被风景点和看起来像其他小径的东西所诱惑，但重要的是你要留在主要的小径上。当你离开小路时，你会践踏植被，侵蚀泥土和岩石，这可能会破坏当地的生态系统，使任何人都不太可能看到野生动物或有趣的植物群。如果每个人都留在小径上，这个地区会保持尽可能美丽的样子。但如果每个人都偏离了轨道，它就会逐渐被摧毁。

### 站到一边休息一下

虽然你不应该离开主路太远，但你也不应该在路中间停下来休息。如果你需要停下来休息，拍照，或者从你的包里拿点零食，站到一边去，这样别人就可以轻松地超过你。这意味着你不应该停在不可能的地方。确保有足够的空间。

### 遵循七个“不留痕迹”的原则

户外伦理中心列出了一份很棒的清单，列出了每个人在任何时候外出都应该遵守的七条规则。有很多要复习的——主要是关于正确处理废物，大小便规则，哪里应该和不应该露营，不要从路上拿东西，等等。因此，请点击链接查看。当有疑问时，记住这句口头禅:只拍照片，只留下脚印。

### 打招呼

徒步旅行是一种社交活动，不管你愿不愿意。向路上遇到的任何人打招呼都是礼貌的。你不必停下来谈论你的希望和梦想，但是互相承认也很好。这也是一个安全问题。你想让人们记得见过你，你想记住你见过的人。如果你或其他人失踪了，有人可以提供帮助的线索。

这也是找出谁应该靠边站，或者让别人知道你正在超越他们的最简单的方法。一个简单的问候让他们知道你在那里，你承认他们，你准备好了任何需要的策略。

### 查看路线指南

无论如何，请务必阅读 trailhead 指南。大多数小径都有相同的规则，但有些小径可能有特定的规则。也许狗是不允许的，或者必须一直拴着，也许这个地区有大量的熊，人们应该保持警惕，或者也许扁虱最近特别成问题。只需要几分钟就能读懂这些标志。
