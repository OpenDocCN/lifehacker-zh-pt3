# 用你厨房里已经有的东西制作便宜的起泡酒鸡尾酒

> 原文：<https://lifehacker.com/make-cheap-sparkling-wine-cocktails-with-stuff-you-alre-1821057378>

香槟鸡尾酒可能是最具庆祝性的鸡尾酒，但将真正的大写 c 香槟与掩盖味道的添加剂混合在一起有点浪费。然而，这并不意味着你不应该享受气泡饮料。事实上——加上一瓶价格合理的气泡酒——你可以用你厨房里已经有的东西做出美味、廉价和喜庆的酒。

起泡酒非常容易鸡尾酒，所以这里的变化几乎是无穷无尽的。(你可以随意添加任何你认为合适的内容。)下面你会发现一些我最喜欢的泡泡插件，几乎所有的都已经存在我的冰箱或餐具室里了:

*   **柠檬+方糖:**将方糖(或一茶匙糖)溶解在半盎司柠檬汁中，加入四盎司起泡酒。用一盎司的香槟酒换一盎司的杜松子酒，你会得到一杯法国 75。
*   果酱:取一茶匙你最喜欢的水果蜜饯，舀进香槟酒杯的底部。顶着泡泡，给饮料一个*非常*的温柔搅拌，尽情享受。
*   新鲜香草+鸡尾酒樱桃糖浆:几枝美味的百里香抵消了糖浆令人作呕的性质。一点点就够了，所以在你的笛子里加入 1/2 茶匙，加入葡萄酒，并在杯子里加入一些新鲜的百里香。(将药草拍在手掌上，释放香气。)
*   桃子罐头:我*喜欢*桃子罐头，它们包装的果汁(不是糖浆)是很棒的人造贝里尼(每支笛子一盎司这种东西就足够了)。另外，罐子里已经有装饰了。
*   啤酒:两份起泡酒和一份黑啤混合在一起，就成了一种奶油味十足、果香浓郁的鸡尾酒。把黑啤酒倒在泡泡上面就行了，让两人在玻璃杯里互相认识。

当然，喝不含“令人分心的”添加物的起泡酒没什么不好。我甚至不确定长笛是否有必要。
