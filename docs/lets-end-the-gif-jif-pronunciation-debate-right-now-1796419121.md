# 让我们现在就结束 GIF/JIF 发音的争论

> 原文：<https://lifehacker.com/lets-end-the-gif-jif-pronunciation-debate-right-now-1796419121>

在过去的八年多里，每周我都会无意中听到这样一句话:“是 GIF，不是 JIF。”"实际上，它是正式的 JIF . "如果论述者在这个问题上受过教育，他们会喋喋不休地支持他们的论点:它是 JIF，因为它的发明者这么说，它就像“长颈鹿；它是 GIF，因为它代表“图形”，就像“礼物”



显然，每个人都有一点乐趣，争论如何读一个缩写只是一种有趣的打发时间的方式。只是它不再那么可爱了。辩论已经彻底结束，没有什么要补充的了。

在[【howtoreallypronouncegif.com】](http://howtoreallypronouncegif.com/)上有 GIF 的七分防御，在 [上有 JIF 的五分防御(还有定期更新的博客)](http://www.olsenhome.com/gif/) 。甚至还有一个由程序员兼博客写手安迪·拜奥提出的古怪的伟大妥协建议。GIF 是 30 年前的。争论它的发音就像“坚持住”小猫海报一样新鲜。

像 [牛津逗号辩论](https://en.wikipedia.org/wiki/Serial_comma) ， [制表符/空格辩论](http://www.businessinsider.com/tabs-vs-spaces-from-silicon-valley-2016-5) ，句号辩论 后的 [空格，GIF/JIF 发音最好作为一个风格问题来对待。随便挑一个，始终如一，处理好自己选择的后果。正如](http://www.slate.com/articles/technology/technology/2011/01/space_invaders.html) [语言学家史蒂芬·平克告诉](https://www.theatlantic.com/entertainment/archive/2014/12/steven-pinker-interview/384092/)*的:*

> *这并不是说优秀的作家选择了藐视规则；那就是规则首先就不是规则...什么使规则成为规则？谁来决定？它从哪里来？他们写得好像有什么法庭或规则委员会来制定英语规则，但实际上并不存在，或者这是一个逻辑或客观现实的问题，但事实并非如此。*

*每一场辩论都涉及到有效的论点，与 GIF/JIF 不同，其中一些论点有其后果。牛津逗号可以防止误解，这就是为什么它在法律事务中至关重要；一个缺失的逗号让一家乳品公司损失了 1000 万美元的工资。*

*但是逗号不是解决所有短语歧义的一招。正如平克指出的，语言太模糊了:你可以把“四位教授组成的性小组”重新安排成不那么淫秽的“四位教授组成的性小组”，但如果这个小组“吸毒”，你又会有麻烦。这就是为什么《美联社文体手册》(反逗号)和《芝加哥文体手册》(前逗号)都提到了可能的例外。正如《芝加哥手册》所说，“最好保持灵活。”*

*类似地，关于制表符/空格的争论也有实际的后果——制表符会减小文件的大小；空格更精确。在某些情况下，这些都很重要！只是不足以抹杀另一个选择。程序员兼 Stack Exchange 创始人杰夫·阿特伍德说，最重要的是团队一致性:*

> *选择标签，选择空间，选择任何对你和你的团队有意义的布局惯例。你选择哪种编码风格并不重要。重要的是你和你团队中的其他人**坚持这些惯例，并始终如一地使用它们**。*

*答案不是一个 [抒情的](https://www.youtube.com/watch?v=P_i1xk07o4g) “谁他妈的在乎一个牛津逗号？”而是一个理性的“我们该给谁？”进行一场合理的辩论，决定你将如何决定任何其他温和的决定，然后遵守那个决定。* 

*无关紧要的小辩论很诱人，因为它们是讨论更大问题的安全空间。当你为牛津逗号而战时，你实际上是在为精确的含义而战；当你称它为 JIF 时，你是在保护发明者的精神权利；当你 [改正句子后双倍行距](http://www.heracliteanriver.com/?p=324) 的记录时，你就是在拥护历史修正主义。*

*这就是为什么这些琐碎的争论变得如此激烈，也是为什么保持一些观点是重要的。每个有意义的团队都有一些关于形式和风格的分歧，但每个有效的团队都会解决这些分歧。我愉快地为支持和反对逗号的编辑写文章；我生活在一个不同信仰的 GIF/JIF 家庭。我们把分歧放在一边，这样我们就能和真正的敌人战斗:那些不在乎的人。*
