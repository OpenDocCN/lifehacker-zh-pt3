# 确保您不会访问 Equifax 网络钓鱼网站

> 原文：<https://lifehacker.com/make-sure-youre-not-going-to-an-equifax-phishing-site-1818623746>

如果说 Equifax 数据泄露事件还不够糟糕的话，许多消费者已经在抱怨该公司处理善后事宜的草率方式。例如，有 Equifax 的网站专门针对此事:。

正如 Gizmodo 的 Dell Cameron 指出的, ,这个看起来非官方的网址很容易被欺骗，这正是开发者 Nick Sweeting 在创建 securityequifax2017.com 时所做的。Sweeting 制作了这个假网站，以证明复制 Equifax 的网站进行网络钓鱼欺诈是多么容易，然后欺骗消费者访问该网站。看看这个:

为了证明 Sweeting 的观点，Equifax [实际上发送推文](https://web.archive.org/web/20170920155011/https:/twitter.com/Equifax/status/910265181976104960) 引导客户到这个假网站。因此，如果您要访问 Equifax 的网站获取更新，这是一个很好的提醒，以确保您不会进入一个虚假的钓鱼网站。

作为保护您可能被盗的数据的一部分， [我们已经建议在每个局冻结您的信用报告](https://twocents.lifehacker.com/everything-you-need-to-know-about-a-credit-freeze-1803139061#_ga=2.67170841.981042000.1506009513-1268082208.1431441811) 。这里是你可以在三大信用局中的每一个注册冻结的地方:

你也可以给他们打电话:

*   Equifax: 1-800-349-9960

*   益百利:1-888-397-3742

*   工会:1-888-909-8872

然而，许多读者报告说，他们无法注册，正如我们提到的，冻结的问题是，如果黑客闯入正确的金库(这就是整个惨败的开始)，你的 PIN，即你需要打开和关闭冻结的号码，可能会被盗。事实上，Brian Krebs 指出，甚至不一定要很难得到你的 PIN 码:

> 一位机警的读者最近让我注意到了由三大信用机构益百利提供的免费在线服务，该服务允许任何人申请个人身份号码(PIN)来解锁之前在益百利被冻结的消费者信用文件。
> 
> 立即泄露任何人的冻结 PIN 码的第一个障碍是提供该人的姓名、地址、出生日期和社会安全号码(所有这些数据都已经被破坏了 100 多次——包括最近的 Equifax breac h 中的】和在网络犯罪地下组织中广泛出售的数据)。

所有这些问题使得我们很难认真对待这些安全措施，但这并不是说我们应该忘记它们。然而，这是一个很好的提醒，没有人比你更关心你的数据。确保监控您的信用、 [信用卡账户](https://twocents.lifehacker.com/your-credit-cards-may-also-have-been-compromised-in-the-1818522666#_ga=2.67170841.981042000.1506009513-1268082208.1431441811) 和社会保障信息。
