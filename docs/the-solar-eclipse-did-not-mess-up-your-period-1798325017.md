# 日食没有打乱你的月经

> 原文：<https://lifehacker.com/the-solar-eclipse-did-not-mess-up-your-period-1798325017>

最近的日食是一次强有力的经历。无论你是跋涉数百英里去看日全食，还是在午休时出去和朋友分享日全食眼镜，你都经历了一个美妙的天文和社交时刻。你现在可能会感觉到与太阳和月亮在智力上或精神上有很强的联系。但是这种联系并没有到达你的子宫。



不过，你不会从铺天盖地的头条新闻中得知这一点。在*嘉人* : [女性已经注意到今天日食](http://www.marieclaire.com/health-fitness/features/a28913/solar-eclipse-menstrual-cycle/) 的一个奇怪副作用。在*青少年时尚* : [日食如何影响月经周期](http://www.teenvogue.com/story/solar-eclipse-period) 。*喧嚣的* take 有一个类似的标题——[日食可能如何影响你的月经，而你甚至没有意识到](https://www.bustle.com/p/how-the-solar-eclipse-may-affect-your-period-without-you-even-realizing-77984)——但这篇文章很快揭穿了这种联系。

这些文章依靠轶事——少数女性说她们的月经在过去几周很奇怪——并援引占星术和神秘能量。至少《青少年时尚》的文章被归入了精神类。因为，从生理上来说， [这里没有什么事](https://drjengunter.wordpress.com/2017/08/21/no-the-eclipse-didnt-mess-with-your-menstrual-cycle/) 。

当然，月亮会影响潮汐。是的，月亮 28 天的轨道可能与一些人的月经周期相匹配。但这并不意味着子宫内膜能感受到月亮的引力。并且*即使它是*——它不是——太阳和月亮的精确视觉对准与标准的 [【春】潮](http://sciencing.com/tide-corresponds-solar-eclipse-3392.html) 没有明显的不同，在标准的[【春】潮中，月亮和太阳在地球的同一侧。](http://sciencing.com/tide-corresponds-solar-eclipse-3392.html)

日食是一个可怕的事件，将数百万人团结在一个天文现象周围。那里有大量的实际科学。如果这周你的月经真的很奇怪，也许去看医生。
