# 即使你不是小孩，网飞的互动故事也很有趣

> 原文：<https://lifehacker.com/netflixs-interactive-stories-are-entertaining-even-if-y-1796299663>

[周二](https://media.netflix.com/en/company-blog/interactive-storytelling-on-netflix-choose-what-happens-next) ，网飞发布了它的第一个互动故事。就像我们童年时代选择自己的冒险书籍一样，网飞的电视节目让观众选择故事的发展方向，这是有史以来最好的事情。



目前，只有一个互动故事:“书中的猫:被困在史诗故事中。”幸运的是，它很有娱乐性，写得很好，对成人和孩子都足够了。

在这一集里，观众可以跟随普斯一起经历流行童话的混音。他们遇到了一本书，这本书提示他们在两个选项之间进行选择，而讲述者和猫对这两个选项都进行了评论。屏幕底部没有进度条告诉你看视频到了什么程度，而是有互动的书籍页面，让你跳转到你可以做出的决定。如果观众没有及时做出决定，网飞会自动为他们做出选择。

也有来自*的角色升级扮演新角色，比如扮演海盗的小猪托比。有三只小熊，它们要么善良，要么不善良，这取决于观众的选择。还有我最喜欢的角色，一个邪恶的女王进入了故事，对着她的镜子说:“我只是说，也许我们可以偶尔谈论一些事情，而不是谁是最漂亮的。比如，我不知道，艺术，音乐？”*

可惜的是，这一集只在智能电视、游戏机、iOS 设备、Roku 设备上互动， [按消费主义](https://consumerist.com/2017/06/20/netflixs-new-interactive-tv-shows-let-you-choose-what-happens-next/) 。在网络、苹果电视、Chromecast 和 Androids 设备上，剧集的播放就好像是网飞在为你做选择。

显然创造互动故事已经是一个想法了， [花了两年时间执行](https://www.nytimes.com/2017/06/20/business/media/netflix-interactive-television-puss-in-boots.html?_r=0) 。网飞将于 7 月 14 日发布“好友惊雷”系列的另一集互动节目。2018 年将会有一集互动的《拉伸阿姆斯特朗》。

因为它是互动的，所以不容易一口气看完一大堆剧集。还有其他好处，比如让孩子们思考创造故事的不同途径以及选择的潜在后果(猫最终会受伤很多)。

Hulu 也在修补自己选择的冒险。他们正在开发一个名为“1 号门”的虚拟现实应用系列，但观众是成年人。
