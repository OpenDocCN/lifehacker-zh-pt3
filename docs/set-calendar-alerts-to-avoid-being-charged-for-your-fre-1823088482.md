# 设置日历提醒，避免因“免费”试用而被收费

> 原文：<https://lifehacker.com/set-calendar-alerts-to-avoid-being-charged-for-your-fre-1823088482>

如果在这个残酷的资本主义世界里有一个可靠的真理，那就是:每个人都喜欢免费的东西。享受更好(更自由)的东西是快乐的一个天真来源，但不幸的是，营销人员盯上了我们所有人。什么，你认为这是一个巧合吗？每个产品现在都试图将自己包装在“订阅服务”模式下，并且还试图用“免费试用”来吸引你，只是碰巧仍然需要你分发你的信用卡信息？

Watch

正如他们所说，“他们就是这样得到你的。”

几乎每个人都在某个时候上当了，不小心让免费试用变成了付费订阅，只有在为时已晚的时候才意识到，费用正沾沾自喜地坐在我们的信用卡账单上，嘲笑我们的天真。

但是有一个非常简单的方法可以避免这个陷阱，吃掉所有经过你面前的免费赠品，而不用支付任何费用。每当你注册一个新的订阅或服务的免费试用，在试用期快结束时立即设置一个提醒，在你被收费之前提醒你是时候退订了。

有很多方法可以做到这一点，所以选择你的毒药:谷歌日历事件或提醒，iPhone 提醒，Slackbot 提醒，你的“欢迎”电子邮件的回旋镖，无论你注册了什么服务，等等。媒介并不重要，只要你在注册的第二秒就设置了提醒，最好是在付款开始前给自己 48 小时的缓冲时间。(值得一提的是，如果你是那种经常通过新卡 进行“信用冲浪”以获得最大介绍奖励的人，这个系统也可以提醒你在年费开始之前取消卡。)

当然，这是一个非常简单的技巧，但是如果我们所有人都这么做，这种商业模式早就崩溃了，公司也不会像发糖果一样发放免费试用。这意味着，我们所有人都有足够的空间来更有效地利用这个系统。有时候你只是需要一个提醒。
