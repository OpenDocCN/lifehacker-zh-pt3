# 汉语普通话现在可以在语言学习应用 Duolingo 上使用

> 原文：<https://lifehacker.com/mandarin-chinese-is-now-available-on-the-language-learn-1820486948>

iOS、Android 和网络上的免费语言学习应用 Duolingo 终于推出了针对英语使用者的中文课程。现在，你可以免费尝试英语母语者最难学的语言之一。



除了今年早些时候推出的日语、之外，中文是多林哥最受欢迎的语言课程，这使得此次推出相当重要。中文是地球上最常用的语言(大约有 12 亿人以中文为母语)，而 [是商业中最有用的学习语言之一](https://lifehacker.com/the-most-useful-language-for-english-speakers-to-learn-1793742182) ，所以人们一直渴望这次更新。

[多林哥中文课程](https://www.duolingo.com/course/zh-CN/en/Learn-Chinese-Online) ，我在推出之前试了一下，教 [现代标准普通话](https://en.wikipedia.org/wiki/Standard_Chinese) (也就是普通话普通话是中国和台湾的官方语言，也是新加坡、香港和澳门的官方语言。随着您的学习，本课程将涵盖汉语的四个主要声调，并涵盖 1000 个最常用的汉字。你还将学习超过 1000 个词汇，涵盖问候、食物、文化、职业、旅游、商务，甚至网络俚语等主题。

如果你完成了广泛的课程，多林哥研究中心的科学家建议你能够通过官方汉语水平考试的一至四级。或者换句话说，你可能处于描述语言能力的国际标准[【CEFR】](https://en.wikipedia.org/wiki/Common_European_Framework_of_Reference_for_Languages)的 A2 水平(高级初学者)。

所以，如果你能完成这门课程，你就不会流利了——甚至差得远——但对于智能手机上的免费应用程序(带广告)来说，这仍然是不错的！要开始使用 [，你可以点击网页](https://www.duolingo.com/course/zh-CN/en/Learn-Chinese-Online) ，或者在你的 iOS 或安卓设备 上更新应用 [。](https://itunes.apple.com/app/duolingo-learn-spanish-french-and-more/id570060128?mt=8)
