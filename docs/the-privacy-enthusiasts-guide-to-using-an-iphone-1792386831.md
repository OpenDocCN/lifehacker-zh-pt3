# 隐私爱好者使用 iPhone 指南

> 原文:[https://life hacker . com/the-privacy-发烧友-iphone 使用指南-1792386831](https://lifehacker.com/the-privacy-enthusiasts-guide-to-using-an-iphone-1792386831)

你的隐私很重要，现在比以往任何时候都更重要的是，似乎每个人都在试图查看你的个人数据。这可能包括广告商、政府或你生活中的某些古怪的偷窥狂。好消息是，你可以对你的 iPhone 做一些事情，让它更加安全和隐私友好，而不会破坏体验。

Watch

首先:这是一部苹果设备，也是一部智能手机，所以你永远不会完全隐藏自己，但你可以做一些事情来填补漏洞，确保你不会让别人轻易收集你的私人信息。我们不想在这里给你一种无法穿透的隐私的错觉，但下面的提示和各种应用程序至少会在不干扰你日常活动的情况下尽可能多地锁定信息。

### 您要为隐私更改的系统设置

首先，你需要检查你的系统设置，并做一些改变。以下是我们的建议，但请挑选对您最重要的功能:

*   **设置一个强有力的字母数字密码**:进入设置>触控 ID &密码，确保你有一个密码。 [包含数字和字母的字母数字密码](http://lifehacker.com/this-is-how-you-should-secure-your-iphone-5914602) 通常被认为比数字密码更安全。
*   不要用触控 ID:触控 ID 很方便，但在隐私方面却很糟糕。法律还不清楚这一点，但现在，警察 [可以强迫你使用你的指纹](https://www.theatlantic.com/technology/archive/2016/05/iphone-fingerprint-search-warrant/480861/) 解锁你的手机，但他们不能让你吐出密码。要关闭触控 ID，前往设置>触控 ID &密码，并禁用 iPhone 解锁开关。
*   **删除任何显示个人信息的小工具** : [iOS 10 推出锁屏小工具](http://lifehacker.com/how-to-make-sense-of-ios-10s-new-lock-screen-1786541529) ，这些小工具很棒，但它们也可能会显示各种你可能不想轻易获取的信息。在锁定屏幕上向右滑动，然后轻按“编辑”来移除您安装的任何显示您不想让陌生人看到的私人数据的 widgets。
*   **禁用某些主屏幕功能**:前往设置>触控 ID &密码，寻找“锁屏访问”删除任何会让他人访问您的个人信息的内容，如今日视图、Siri 和钱包。你可能也想在这里禁用回复信息，因为有人可以在不解锁你的手机的情况下回复收到的信息。
*   **禁用跟踪**:进入设置>隐私>位置服务>系统服务，关闭常用位置。这是一个地图功能，在改善搜索的幌子下跟踪你经常去的地方。
*   **在不需要的应用中关闭联系人、照片、电子邮件或日历、位置访问**:进入设置>隐私。在这里，您将看到一系列不同系统服务的列表，包括位置、联系人等。这些是您可以授予应用程序访问权限的 iPhone 服务。这里可能有一些你不记得授权过的应用程序，或者你不想要了。轻按一个服务，然后浏览并停用任何您不想访问该服务的应用程序。
*   **移除通知预览**:你可能不想完全禁用通知，但你可能想隐藏那些通知在锁屏上显示的内容。前往设置>通知，然后禁用邮件和信息预览。
*   **启用双因素认证**:双因素认证是锁定您帐户的最佳方式，即使陌生人知道您的密码，也无法访问。您可以在这里 为您的 Apple ID 设置 [。你应该对](https://support.apple.com/en-us/HT204915) [以及](https://lifehacker.com/heres-everywhere-you-should-enable-two-factor-authentic-5938565) 所有其他账户使用双因素认证。
*   **启用查找我的 iPhone:** [“查找我的 iPhone”](https://support.apple.com/explore/find-my-iphone-ipad-mac-watch)从隐私的角度来看有点令人困惑，但大多数人使用它会比不使用它受益更多。启用“查找我的 iPhone”后，您可以使用 iCloud 来跟踪丢失的手机，并且可以远程擦除您的手机。苹果将获得相同的信息，所以归根结底是你想让数据不落入苹果的手中(在这种情况下，你根本不应该使用 iPhone)还是不落入小偷的手中。
*   **关闭选定应用程序的 iCloud 备份** : iCloud 备份非常有用，虽然极端隐私主义者会禁用 iCloud 备份，以便将数据从苹果的服务器上移除，但更简单的解决方案是关闭某些应用程序。如果你前往设置> iCloud >存储>管理存储>备份，你可以选择哪些应用程序备份到 iCloud，哪些不备份。禁用任何保存敏感数据的应用程序。

调整 iPhone 上的一些设置是这个过程的一部分。你选择使用的应用程序也很重要。

### 保护您隐私的生产力应用程序

为了方便起见，大多数生产力应用程序完全无视你的隐私。这不是一件坏事，因为云同步和智能组织功能正是你*希望*从生产力应用中获得的功能。不过，你可能不希望你的所有数据以某种方式公开，这正是这些关注安全的应用程序派上用场的地方。一些工具，比如网络浏览器或密码管理器，一直都很有用，而另一些工具，比如加密的 notes 应用程序或 VPN，只对某些事情有用。

如果你有一部设置了 [移动设备管理](http://www.apple.com/iphone/business/it/) 的公司管理的 iPhone，这些应用程序都不会保护你的数据隐私。如果是这样的话，买一个单独的电话，除了工作之外，不要用你的工作电话。

#### 网络浏览器:勇敢还是火狐焦点

你会在 App Store 中找到大量声称保护你隐私的不同网络浏览器，但我们最喜欢的两个是 [【勇敢】](https://itunes.apple.com/us/app/brave-web-browser-fast-built/id1052879175?mt=8) 和 [火狐焦点](https://itunes.apple.com/us/app/firefox-focus-privacy-browser/id1055677337?mt=8) 。

默认情况下，Brave 到处使用[HTTPS](https://www.eff.org/Https-everywhere)。它还阻止脚本、cookies、网络钓鱼、弹出窗口和广告。您可以在每个站点的基础上关闭和重新打开任何这些功能，这使得对任何问题或白名单站点进行故障排除变得容易。Brave 的桌面应用有一个 [奇怪的支付系统](https://brave.com/publishers.html) 来支付出版商，即使它阻止了广告，但它似乎没有在移动设备上到位。Brave 可以替代 Safari。Brave 拥有你在浏览器中需要的功能，比如书签、历史和密码管理器支持。

Firefox Focus 与 Brave 非常相似，但更进了一步。它阻止广告、追踪器、社交媒体、cookies 等等。Firefox Focus 还可以让你轻而易举地清除浏览器历史记录，删除所有密码和 cookies。Firefox Focus 没有选项卡式浏览功能，所以当你想让你的浏览不被记录时，Firefox Focus 是 Safari 的最佳补充。

#### 电子邮件:质子邮件或 Gmail

从隐私的角度来看，电子邮件有点难处理，你在这里做什么取决于你为什么关注隐私。如果你不想让任何人看到你发送的电子邮件，那么你可以用 [ProtonMail](https://protonmail.com/) 设置一个免费的电子邮件地址。没有 ProtonMail 账户，你无法使用 [ProtonMail iOS 应用](https://itunes.apple.com/us/app/protonmail-encrypted-email/id979659905?mt=8) ，但它是最安全和最隐私的应用。

ProtonMail 会加密你发送的每一封邮件，这意味着该公司甚至无法阅读你的电子邮件。如果您在两个 ProtonMail 帐户之间发送邮件，这将自动发生。如果你给不在 ProtonMail 上的人发邮件，他们会得到一个需要密码才能阅读的邮件链接。如果你需要保持电子邮件的私密性和安全性，ProtonMail 是你想要使用的应用程序。

对于其他任何事情，请继续使用您一直使用的 Gmail 帐户。尽管有谷歌的数据挖掘，Gmail 应用程序 在防止你的私人数据落入他人之手方面做得很好。每封电子邮件都通过 SSL，Gmail 从发件人到收件人对电子邮件进行加密， [双因素认证](https://itunes.apple.com/us/app/google-authenticator/id388497605?mt=8) 可以在您丢失设备的情况下保护您的电子邮件。当然，谷歌在那里有它的眼睛，但如果你担心一些随机的人找到你的手机，那么 Gmail 是一个可靠的赌注。

除了像 ProtonMail 这样的服务或者运行你自己的服务器，没有什么好的方法来保持电子邮件的私密性，至少在你的手机上。在这种情况下，Gmail 至少是一个安全的选择。

#### 短信:Signal 还是 WhatsApp

当谈到安全和私人消息时，你有两个流行的选择:信号和 [WhatsApp](https://itunes.apple.com/us/app/whatsapp-messenger/id310633997?mt=8) 。两者都有端到端的加密功能，都不会在信息发送后将其存储在服务器上，都有语音通话和信息功能，而且都非常容易使用。哪个 app 更适合你取决于你的联系人在哪里，因为一个聊天的双方需要使用同一个 app。

Signal 确实比 WhatsApp 多了一些安全功能。Signal 不存储元数据，但 WhatsApp 会。这意味着，虽然 WhatsApp 不知道信息的内容，但它知道信息是在谁之间传递的，并记录了数据，然后可以通过搜查令将数据移交给执法部门。Signal 不存储任何这些内容。如果你使用 iCloud 备份你的 iPhone，Signal 不会将你的信息存储在备份中，但 WhatsApp 会。您的 iCloud 备份是加密的，因此这不会有什么影响，但您通常会希望尽可能将这些数据保存在本地。对我们大多数人来说，这些都无关紧要，但仍然值得一提。

#### 密码管理器:LastPass 或 1Password

不让某人进入你的各种在线账户的最好方法是拥有一个好的、强有力的密码。最好的密码对我们大多数人来说太复杂了，记不住，这就是为什么你 [应该使用密码管理器](https://lifehacker.com/lifehacker-faceoff-the-best-password-managers-compare-1682443320) 。我们喜欢 [LastPass](https://itunes.apple.com/us/app/lastpass-password-manager/id324613447?mt=8) 和[1 pass。](https://itunes.apple.com/us/app/1password-password-manager/id568903335?mt=8)

你的密码管理器为你所有的网站生成随机密码，这使得任何人都很难进入你的账户窥探。这有安全和隐私的双重好处，因为你不知道你账户的密码，让其他人更难得到它们。LastPass 是免费的，可以跨平台同步数据。[1 如果你想同步](https://1password.com/sign-up/) ，Password 每月 3 美元，但如果你不想在服务器上存储数据，也可以在本地存储你的密码库。

#### VPN: Hideman、metro bear 或 nordvpn

VPN 是一种保护和加密你基本网络使用的简单方法。当您连接到 VPN 时，您的所有流量都是安全的，这在您使用公共网络时最为有用。当你在星巴克的 Wi-Fi 上时，你可以连接到你的 VPN，然后在公共网络上保护你的流量。这样，没有人可以窥探你的流量。

你有一大堆好的 VPN 选项，但说到基本的可用性，我们喜欢 iOS 上的[nord VPN](https://itunes.apple.com/us/app/vpn-by-nordvpn-easy-unlimited/id905953485?mt=8)和 [Tunnelbear](https://www.tunnelbear.com/) 。如果你使用大量数据，这三者都需要每月付费，但它们也给你一些免费数据，这对于偶尔在咖啡店、酒店或机场浏览公共网页是足够的。

#### 备注:保持

这里要明确的是，如果你有你想要永远保密的敏感数据，你不应该把它保存在你的手机上。你也不应该把它存储在像 Dropbox 或 Evernote 这样的云服务上，因为没有一个云存储是完全安全的，除非你提前检查并加密你的所有数据。

如果你有一些愚蠢的照片或笔记，你只是想远离窥探的目光，那么 [Keeply](https://itunes.apple.com/us/app/keeply-hide-photos-lock-notes/id757172898?mt=8) 是一个很好的应用程序来保护你的数据。Keeply 让你可以选择将它与 Dropbox 连接起来，或者将数据保存在手机上，并让你设置一个特殊的 PIN 码来锁定应用程序。Keeply 存储笔记、照片、密码和信用卡。我个人坚持只使用它来记录笔记和照片，但是除了 PIN 保护之外，它还可以加密所有数据，所以它应该是相当安全的。这里重要的是 Keeply 中的数据永远不会离开你的手机(除非你打开 Dropbox sync)，所以它没有机会最终在线。

#### 云存储:蜘蛛橡树

安全地私下进行云存储有点困难，但是 [你最好的选择](https://lifehacker.com/the-best-cloud-storage-services-that-protect-your-priva-729639300) 是 [蜘蛛树](https://spideroak.com/about/price-list) 。SpiderOak 的“零知识”隐私政策使得加密在本地进行，所以他们不知道你在他们那里存储了什么。该服务没有像 Dropbox 或 Google Drive 那样的免费层，但该服务的价格具有竞争力，每月 5 美元可以获得 100GB 的存储空间。

iPhone 的 [应用](https://itunes.apple.com/us/app/spideroakone/id360584371?mt=8) 不如 Dropbox 那样强大，但它确实保持了 SpiderOak 的安全级别。如果你需要一个安全的在线存储选项来同步文件，并使它们可以从你的手机上访问，SpiderOak 是你最好的选择。

这样的话，你对 iPhone 的正常使用应该是私密的，不会被在线用户和接触你手机的人发现。通常有很多方法可以绕过这一切，但至少你不会让人们很容易跟踪你。
