# 这个聊天机器人浪费骗子的时间，很光荣

> 原文：<https://lifehacker.com/this-chatbot-wastes-scammers-time-and-its-glorious-1820334967>

在某个时候，我们都曾被一个尼日利亚王子、失散多年的叔叔或某个无法获得银行账户的家伙联系过，并承诺如果我们只进行一次小额电汇，明早我们的账户就会有数百万。



不可否认的是，我回复了一些只是为了好玩，看看对话会发展到什么程度。正如 [*The Verge* 报道](https://www.theverge.com/2017/11/10/16632724/scam-chatbot-ai-email-rescam-netsafe) 所言，现在一家安全公司创造了一个聊天机器人，它可以让你回复所有那些烦人的诈骗短信，并像一开始发送电子邮件一样浪费他们的时间。

要使用它，你只需将第一条违规信息转发给。当你这样做时，一个代理电子邮件地址将开始为你回复骗子的电子邮件。回应看起来很真实。

Rescam [在他们的网站上列出了几个](https://www.rescam.org/) 。例如，这是聊天机器人对电汇请求的第一次响应。

它很容易使用，而且说的足够让对话继续下去，而不会让人觉得你实际上是在用机器人说话。
