# 用这种想象克服对湍流的恐惧

> 原文：<https://lifehacker.com/overcome-fear-of-turbulence-with-this-visualization-1827105024>

我讨厌飞机旅行。我知道，我知道，这是最安全的旅行方式，汽车要危险得多，blar de blar，但是当我系好安全带，我们起飞的时候——进入*天空*，那不是人类应该去的地方——我能想到的就是我被困在一个 [银色闪闪发光的死亡机器](https://www.goodreads.com/quotes/273826-the-plane-had-lost-power-in-all-three-engines-dropped) 里。

Watch

然而，在深呼吸和少量苯二氮卓类药物的帮助下，一旦我们完全升空，我通常会没事。除非，湍流开始于。这时候我的手掌开始出汗，我抓住扶手，每一次倾斜和颠簸都气喘吁吁；更糟糕的是，如果我和别人在一起，或者任何人同情地看了我一眼，我就会哭。因为我们都要死了，这不是很悲哀吗？难道我们没有一个美好的生活，一个直到那一刻我们才珍惜的生活吗？

我知道颠簸 [不会导致飞机坠毁](http://www.askthepilot.com/questionanswers/turbulence/) 。这个我知道！看在上帝的份上，我是一个该死的*生活黑客*。当小小的推挤分散了我看《我，冬妮娅》的注意力时，我应该完全平静下来然而。

然后，几个星期前，我在飞机上想起了这个生活黑客的提示 :

> 与其紧张，不如集中精力放松你的肌肉，这样你就能随着湍流而动，而不是逆着它。这可能很难做到，但它也给了你其他的东西去思考。

我还记得在什么地方听到过这句话:就像你不应该担心路上的颠簸，你也不应该担心飞机上的颠簸。所以我把这两个智慧结合到下面的技巧中:我想象我在开车。我想象我的手握着方向盘。(我没有举手，因为我没有疯。)我感觉自己(在心里)踩了油门。我在颠簸中前行，就像我们在土路上一样，享受着一次冒险，而不是在天空中飞驰。

就这样，湍流…不再困扰我了。我发现自己甚至有点喜欢它？它不再是一个会杀了我的东西，而开始感觉只是旅程的另一部分。也许很烦人，但并不恐怖。我在颠簸和倾斜中放松下来，专注于前方(想象中的)道路。

我仍然惊讶于它的效果，我期待着再次尝试。
