# 使用“我切，你挑”的策略在孩子们之间分配东西

> 原文：<https://lifehacker.com/use-the-i-cut-you-pick-strategy-for-dividing-stuff-b-1797207063>

“她还有！”“他的杯子装得更高了！”她的切片更大！" "*这不公平！*



哦，作为一个面对生活中巨大不公的孩子。在分配麦片、果汁、乐高积木和蛋糕时，从来没有真正的平等。听到孩子抱怨的成年人通常认为他们只有几个选择:1)让他们打一架，抱最好的希望，2)把东西从每个人身边拿走，同时跺着脚大喊，“就是这样！谁也拿不到！”或者 3)拿出测量工具，精确地分份，感觉像个吸盘。

有一个更简单的方法。在 [*《游戏理论家育儿指南》*](http://www.amazon.com/dp/0374160015/?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/use-the-i-cut-you-pick-strategy-for-dividing-stuff-b-1797207063&asc_source=&tag=kinjalifehackerlink-20) 中，作者保罗·雷伯恩和凯文·佐尔曼提出了“我切，你挑”的策略，允许父母创造 [“无嫉妒”的分工](http://www.slate.com/articles/life/family/2016/05/game_theory_can_help_parents_win_every_argument_with_their_kids.html) 。

你只需要让一个孩子把想要的东西分成两堆(或者把一杯果汁分成两个杯子，或者诸如此类)，另一个孩子可以挑选。

下一次他们被要求分享时，孩子们会确保平均分配。根据雷伯恩的说法，这种策略效果很好，只要你轮流选择谁裁谁选。
