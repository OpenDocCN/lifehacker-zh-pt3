# 1Password 的更新密码管理器是 Mac 的必备软件

> 原文:[https://life hacker . com/1 passwords-updated-password-manager-is-a-must-have-for-1826379117](https://lifehacker.com/1passwords-updated-password-manager-is-a-must-have-for-1826379117)

macOS(还有我们写完这个之后， [Windows](https://blog.agilebits.com/2018/05/29/1password-7-for-windows-the-best-ever/) ！):几年前，AgileBits 发布了其杀手级密码管理应用程序 [1Password](https://1password.com/downloads/) 的重大更新，这就是为什么大家对 1Password 7 上周的首次亮相如此兴奋。在这一点上，这个应用程序应该是一个家喻户晓的名字，因为它是我们推荐的创建和存储超级安全的 [密码和密码短语](https://lifehacker.com/how-to-create-secure-passwords-that-arent-impossible-to-1825048324) 的主要密码管理器之一。

如果您没有使用 1Password，您可能会使用 LastPass 等应用程序，该应用程序允许您在您的设备之间存储和同步密码，而无需支付月费。尽管如此，每月 3 美元对于 1Password 提供的便利性和安全性来说并不算多，而且您可以在它的最新版本中找到所有有趣的新功能。

我不打算讨论 1Password 7 中所有以修饰和生产力为主题的变化——包括一种新字体“Courier Prime Bits”——我将重点关注每个人都应该使用的一个大的变化:1Password Watchtower。剧透提醒那些还需要看太空堡垒卡拉狄加的人，但是在这种情况下，了望塔不会把你变成赛昂人。

相反，应用程序的这一部分(可通过 1Password 7 侧栏访问)为您提供了大量关于密码强度的信息。我不是说连续使用 31 个不同的疯狂字符来确保你的密码实际上是不可破解的。**瞭望塔会查看您存储在 1Password 7 中的登录信息，并让您知道这些信息何时被泄露**由于似乎每周都会发生一次重大数据泄露事件。

如果你启用了这个功能，瞭望塔甚至可以在的时候通过数据库检查你的密码。如果你的密码出现了，而且相当独特，那么很有可能，是的，它就在网上某个地方，还附带着你的登录信息。换句话说，停止使用那个密码，并在你*用过的地方*修改它。

我也很喜欢**瞭望塔告诉你什么时候为不同的服务重复使用密码**。对大多数人来说，重复密码的清单似乎很长，因为我们都很懒，我们喜欢对每个在线账户使用同一个密码。这是一种可怕的安全做法，一旦你的“通常”登录被某种黑客泄露，几乎可以保证你的其他账户会被侵入。这是一个很难打破的习惯，但希望 1Password 新的“重复使用密码”警告将是您需要的所有激励。

**当您为使用不太安全的 http://而不是 https://的网站保存登录信息时，以及当您为使用二元身份验证但尚未在 1Password 的应用程序中设置该过程的网站保存密码时，瞭望塔还可以让您知道您的密码** [**何时不是很强**](https://support.1password.com/change-website-password/)——例如，很容易被猜到或被强行破解。

虽然 1Password 7 有许多其他有趣的功能可供使用，包括更易于管理的密码库、与 macOS 的 Spotlight 的集成以及对 Secure Notes 的降价支持，等等，但该应用程序的新 Watchtower 为其比咖啡还便宜的月费增加了许多理由。如果您只关心创建优秀的密码，以易于访问的方式存储密码，并有一种简单的方法来查看何时应该更改密码，1Password 7 是适合您的密码管理应用程序。
