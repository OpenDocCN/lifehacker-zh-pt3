# 最好的比特币和以太坊解释者

> 原文：<https://lifehacker.com/the-best-bitcoin-and-ethereum-explainers-1796654705>

比特币，这种由白人 主导的分散式数字货币 [，在每一次丑闻、崩溃或黑客攻击后，似乎都处于消失的边缘。但是八年后，它不仅在增长，而且还在加速，从一月份到T5，总价值翻了三倍，超过 450 亿美元](http://www.coindesk.com/new-coindesk-report-reveals-who-really-uses-bitcoin/) 。而且它不是唯一的加密货币； [竞争对手有莱特币、Dogecoin、以太币](https://coinmarketcap.com/) (一种支持“智能合约”的货币)。



按照传统的货币标准，比特币的高波动性让它变得可怕，作为一种高风险的投资工具，它是奇妙的，作为一种文化实验，它是迷人的。这是对货币如何运作的戏剧性反思，对普通人来说非常复杂和陌生。多年来，数百家新闻媒体试图解释它，这是幸运的，因为它往往需要几个解释才能把握住。这里有一些最好的，从三个最重要的指南开始。

*   [比特币如何工作，五分钟之内](https://www.youtube.com/watch?v=l9jOJk30eQs) :越来越技术性的简单指南。当骑得太快时就跳下来。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-l9jOJk30eQs&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-l9jOJk30eQs&start=0) 

*   [比特币的历史](http://historyofbitcoin.org/) :比特币发明、成长、重大黑客攻击的时间线。

*   以太坊入门指南:描述了一个较新的区块链，具有数字货币之外的应用，包括自动执行的合同、身份管理、计算机使用共享、DRM 等。
*   [Lifehacker 的比特币指南](https://lifehacker.com/what-is-bitcoin-and-what-can-i-do-with-it-5991523) :挖掘、购买、使用比特币的实用指南
*   [区块链如何让数字交易像实物交易一样运作](http://www.coindesk.com/bitcoin-explained-five-year-old/) :比特币等加密货币如何追踪区块链交易的无数学指南。解释如何将数字对象“给予”某人，而不仅仅是复制它。
*   [区块链如何记录交易](https://thenextweb.com/evergreen/2017/07/04/ultimate-3500-word-plain-english-guide-blockchain/) :对加密货币用户如何在没有中央授权的情况下处理交易的稍微更技术性的解释。
*   [智能合约如何工作](https://medium.com/@nik5ter/smart-contracts-for-dummies-a1ba1e0b9575) :将加密货币比作数字自动售货机，其中交易的“规则”内置于机器本身。对区块链在货币以外的用途的一个很好的解释。
*   [【程序员的智能合约】](https://solidity.readthedocs.io/en/develop/introduction-to-smart-contracts.html) :以太坊更技术性的解释。
*   [为什么比特币即使价格波动也有用](https://www.vox.com/cards/bitcoin/is-volatility-going-to-be-a-fatal-problem-for-bitcoin) :如何使用比特币作为传统货币的在线支付方式。
*   [人们如何盗取比特币](https://www.theverge.com/2013/12/19/5183356/how-to-steal-bitcoin-in-three-easy-steps) :解释黑客如何一次又一次盗取价值数百万美元的比特币、。
*   [如何保证比特币的安全](https://en.bitcoin.it/wiki/Cold_storage) :保证你的加密货币安全(通常涉及物理存储)的最佳方法的利弊。
*   [比特币如何向记录交易的人支付](https://www.bloomberg.com/view/articles/2014-01-02/bitcoin-is-an-expensive-way-to-pay-for-stuff) :解释转移比特币的“隐性交易费”；也看看回复帖子， [“美元是一种可怕的支付方式。”](https://guan.dk/payments)
