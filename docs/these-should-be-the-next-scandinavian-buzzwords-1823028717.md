一旦你掌握了丹麦人对舒适的概念[](https://lifehacker.com/how-to-get-through-a-miserable-winter-with-the-danish-c-1791001000)*，还有更多北欧的想法可以探索。瑞典语 [*lagom*](https://lifehacker.com/embrace-the-swedish-lagom-lifestyle-to-reduce-stress-1819730614) 可以帮你平衡生活； [*尼克森*](https://lifehacker.com/learn-how-to-do-nothing-with-the-dutch-concept-of-nikse-1822310051) 是正念的荷兰语亲戚。但是这些国家有一些更能改变生活的想法，也许它们应该成为美国的潮流。*

*在谷歌翻译的帮助下，我提出以下建议:*

*   *sundhedsvsnet，丹麦语，意为医疗保健。丹麦有很好的医疗保健，包括比美国低得多的婴儿死亡率，并向所有公民提供医疗保健。当然，税金为其买单，但每人的花费仍然是 [这里](https://www.denverpost.com/2009/09/03/health-care-in-denmark/) 的一半。我并不是说我们需要完全复制丹麦的医疗系统，但是我们可以用*sundhedsvsnet*来表示“更好的医疗保健，也许每个人都可以拥有一些。”*
*   *frldraleidighet 在瑞典语中是育儿假的意思。夫妻获得 [16 个月，在双方伴侣](https://sv.wikipedia.org/wiki/F%C3%B6r%C3%A4ldraf%C3%B6rs%C3%A4kringen_i_Sverige) 之间平分。已支付。实行育儿假政策的国家往往在儿童健康方面有更好的结果。2011 年的一项研究发现，即使我们在美国有少量的育儿假——12 周，不带薪，对那些有能力休假的人来说——[也导致婴儿死亡率略有下降](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC3698961/) 。*
*   *免费的挪威学校对美国人来说似乎有点不切实际，但是你能相信在其他国家中，挪威的学院和大学 [学费基本上是免费的](https://en.wikipedia.org/wiki/Higher_education_in_Norway) 吗？我知道这很疯狂，但不要忘记，在美国，我们对小学也做同样的事情，并把它看作是对我们孩子未来的一种负担得起的公共投资。值得考虑！*

*将一个斯堪的纳维亚单词作为一个概念来欣赏的好处在于，我们不必采用，比如说，丹麦的确切的医疗保健系统，这有其利弊。我们可以用这些词来表示值得引进的思想本身。就像“热茶”和“羊毛毯子”一样，这些词也可以用在日常对话中。当你在工作的时候，你接到电话说你的宝宝生病了，你必须回家，继续说“啊，我真的需要一些*的食物。”**
