# 无人机的白天外出包

> 原文：<https://lifehacker.com/the-drones-day-out-bag-1761977014>

玩无人机很有趣，但仅仅因为它有趣并不意味着你不应该做好准备。读者 Uthor 分享了他们在玩四轴飞行器时随身携带的包。



这个包是 Timbuk2 斜挎包。这是里面的内容:

*   [战神气质 QX 130 四轴飞行器](http://www.amazon.com/ETHOS-QX130-ULTRA-MICRO-QUADCOPTER/dp/B00J9JFW0Q/ref=sr_1_1?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-drones-day-out-bag-1761977014&asc_source=&ie=UTF8&keywords=Ares Ethos QX 130 Quadcopter&qid=1456772629&sr=8-1&tag=kinjalifehackerlink-20)

*   包含的变送器

*   定制皮革工艺品 1100 聚酯袋

*   备用充电 AA 电池

*   备用脂肪电池

*   LiPo 电池充电器

*   [案例逻辑 TBC-403 相机包](http://www.amazon.com/Case-Logic-Medium-Camera-TBC-403/dp/B007Y4NWQE/ref=sr_1_1?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-drones-day-out-bag-1761977014&asc_source=&ie=UTF8&keywords=Case Logic TBC-403 camera bag&qid=1456772673&sr=8-1&tag=kinjalifehackerlink-20)
