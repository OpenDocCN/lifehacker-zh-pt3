# 永远不要吃吃豆人咖喱

> 原文：<https://kotaku.com/never-ever-eat-the-pac-man-curry-1818534498>

昨天在万代纳木错的 [VR 区新宿](https://kotaku.com/i-played-mario-kart-vr-and-it-was-pretty-okay-1818534478) 试用*马里奥赛车 VR* 等等时，我参观了虚拟现实街机内的咖啡馆。我正要犯一个错误。



这家咖啡馆叫 Glampers。就像露营，就像“迷人的露营”他们提供一些看起来开胃的食物，如覆盖着融化的奶酪酱的汉堡和看起来不错的炸鸡。(考虑到你可以在许多日本便利店买到令人惊叹的炸鸡，它可能相当不错。)

我可能要去买汉堡。但后来我看到了:“巴古柏吃豆人的咖喱。”菜单上说，姜黄大米被用来制作黄色的吃豆人球，红色甜菜被用在咖喱酱中来制作人们熟悉的 Blinky 形状。最后，马苏里拉奶酪提供了眼球和圆点。(那应该是游戏的黑色背景，在右边。)

*这个*，我以为*，是 Kotaku 的博文。*

而且，是咖喱。我喜欢日本咖喱！我想，我得在这道非官方的国菜上试试这种旋转。再见，汉堡和你流淌的切达酱；你好，Pakupaku 吃豆人的咖喱。

太可怕了。

首先，形成吃豆人嘴巴的黑色小东西甚至不是食物。这是一片塑料，当我试着用手指把它拿下来开始吃的时候，我发现它非常热。这被证明是不可能的，所以我不得不围绕它吃饭。米饭很好。是米饭。

但是后来我就入戏了。日本咖喱有如此多的口味和质地；这个会是什么样的？事实证明，它的味道和最糟糕的美国预包装肉酱一模一样。就像，意大利番茄酱和碎牛肉，但是你可以在学校自助餐厅或者罐头里买到的那种。

所以我不得不坐在那里，用勺子吃一整盘肉酱。整个盘子在制作后似乎都在热灯下加热过，这就解释了燃烧的热塑料和吃豆人下面的酱是石头一样冷的事实，这是一个令人兴奋的恶心的新时刻，我甚至不相信这种食物可以实现。

无论如何，去虚拟现实区玩*马里奥赛车虚拟现实*，或者进入福音战士，或者乘坐高达。但是到了吃午饭的时候，不要被吃豆人咖喱所诱惑。吃豆人连这个都不吃，什么都吃。
