# 口袋跑池是 iOS 上最好的台球游戏

> 原文：<https://lifehacker.com/pocket-run-pool-is-the-best-billiards-game-on-ios-1826593163>

iOS:对于苹果爱好者来说，游戏开发者扎克·盖奇几乎是一个家喻户晓的名字。苹果不仅喜欢在 App Store 上展示这位独立开发者和他的游戏，而且他还赢得了苹果的金棕榈奖(Palme d'Or of apps)，这是一个令人垂涎的设计奖，以及许多其他行业对他有趣创作的认可。你可能知道 [几个名字](http://stfj.net/apps/index.html) :荒谬的捕鱼，圣人纸牌，魔法塔，和令人沮丧的真正糟糕的象棋。或者你可能已经看过他的最新作品: [袖珍泳池](https://itunes.apple.com/us/app/pocket-run-pool/id949982914?mt=8) 。



盖奇的秘方是采取常规的游戏和活动，并赋予它们足够的创造性，使它们变得有趣和平易近人，而不会感到过于古怪。口袋里运行池感觉就像它的到来，几乎没有大张旗鼓。我甚至都不记得是怎么偶然发现的，但我很高兴我做到了。尽管这款游戏的物理特性需要一点学习曲线——感觉不太像一款完全真实的台球游戏——但考虑到这款游戏的有趣之处，这是完全可以原谅的。

你的目标是在口袋里运行池下沉球。你犯“错误”的机会是有限的，或者在一个回合中什么都没有，这稍微增加了赌注。该游戏的独特之处在于，每个球洞都被分配了一个特定的得分乘数，每当你沉球时，这些乘数就会旋转(或随机化)。因此，为了获得高分，你需要确保你的目标是更高的乘数，这使得游戏的台球策略更加疯狂。

如果你只是独自玩标准的免费版本的游戏，得分方面并不那么重要——它只是增强你的成就感，并给你一些东西截图给你的朋友。一个“高赌注池”模式让你在游戏中下注货币来参加挑战(并制造假钱来购买新的化妆品)。

支付 4 美元解锁游戏的完整版本，你将能够在“insta-锦标赛”中与世界各地的其他玩家竞争，以及游戏的“每周休息”模式，该模式给每个人一个相同的设置，让他们竞争，看谁能获得最高分。

即使你只是坚持游戏的标准版本，当你意识到你没有给予足够的*活力*来沉下你瞄准的球时，这也是一个非常有趣的台球，只有一丝沮丧——就像真正的台球一样。谢天谢地，游戏的忧郁音乐应该足以让你不至于压力过大。嘿，还有下一场比赛。
