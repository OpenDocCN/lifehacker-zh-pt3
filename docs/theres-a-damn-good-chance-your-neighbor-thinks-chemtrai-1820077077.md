# 你的邻居很有可能认为化学痕迹是真的

> 原文：<https://gizmodo.com/theres-a-damn-good-chance-your-neighbor-thinks-chemtrai-1820077077>

化学轨迹是不真实的，但这并没有阻止越来越多的美国人认为它们是真实的。

Watch

哈佛科学家的研究于周二发表在《帕尔格雷夫通讯》 上，研究显示，有相当多的人相信政府在天空秘密喷洒化学物质来控制天气或我们的思想或其他东西的狗屁理论。

科学家们在美国大选前对 1000 人进行了调查。他们发现，10%的受访者认为 chemtrail 阴谋论“完全正确”，20-30%的人认为它“有些正确”(这个范围是因为他们要求人们做一个“最佳猜测”作为跟进)。这相当于 1.2 亿美国人站在“化学轨迹是真实的”阵营中，尽管事实上他们绝对不是。

这项研究是为了让科学家了解公众对地球工程的态度，这是一项实际的技术——尽管有风险和争议——科学家正在考虑为我们快速过热的星球降温。相反，他们发现很大一部分人相信一个疯狂的人对天空中跟随喷气式飞机的条纹的解释。

这些条纹存在的真正原因是燃烧喷气燃料产生的水蒸气变得过冷，附着在小颗粒上形成冰晶。哦，它们被称为轨迹。

多年来，气象学家一直试图用 [来消除](https://gizmodo.com/geostorms-weather-control-tech-is-exceptionally-bogus-1819783644) 的阴谋论。但相信这种毫无根据的想法的人的比例已经从 2010 年一项类似调查中的 16.6%上升到目前的高水平。

在我们的新媒体生态系统中，化学预告片有更多的途径来传播他们的信息，有更多的方式将自己密封到社区中， [相信这是真的](https://twitter.com/search?q=chemtrails) 。以 Twitter 为例，它是一个错误信息的粪坑，有 3 亿人莫名其妙地在那里消磨时间。

在那里，想入非非的理论主导了围绕地球工程的讨论，一个我们 [需要在气候变化的背景下理性讨论](https://gizmodo.com/the-white-house-wants-scientists-to-explore-geoengineer-1791064961) 的话题。分析发现，在过去十年中，59%关于地球工程的推文只是关于化学轨迹的阴谋。关于实际科学的中立报道仅占讨论的 3%。

像亚历克斯·琼斯这样的右翼媒体生态系统(根据调查结果，他是 chemtrail Twitter 上讨论的常见来源)也煽动了阴谋的火焰。但是在一个高度极化的时代，化学追踪似乎也是一件团结政党的事情。

分析显示，chemtrail 的信徒聚集在一起，希望川普的当选将最终停止喷洒(对他们来说，不幸的是， [他们仍在等待](https://theoutline.com/post/1568/chemtrail-truthers-are-still-waiting-for-trump-to-do-something) )。分析还显示，化学预告片讨论了来自不同政治领域的阴谋，比如希拉里在班加西做的事情，以及孟山都公司创造变异植物。令人费解的是，兴趣列表还包括“室内设计”，嘿，我想更新我的客厅装饰，所以找我吧，chemtrails 的人。

“化学轨迹角度的出现主要有两个原因:即使不看数据，这也是一个值得注意的现象，死亡威胁等等，”哈佛大学的经济学家 Gernot Wagner 告诉 Earther。“一旦我们分析了数据，尤其是社交媒体的数据，它就跳了出来。

“尽管我们对网络回音室(和)机器人了如指掌，但这些阴谋数字高得惊人。”

瓦格纳承担了这项研究，作为他在哈佛大学从事的 项目的一部分，该项目侧重于从技术、伦理、政治和经济角度来看地球工程实际上有多可行。该项目背后的基本想法是将粒子注入平流层，以阻挡一些射入的阳光，帮助冷却我们快速过热的星球。

现在，这是一个反乌托邦式的解决气候变化的科幻小说。但是，随着地球持续变暖，天气变得怪异，并导致许多其他连锁影响，社会需要掌握我们的选择。这包括找出如何减少碳排放——包括飞机——如果地球工程是安全的，但不用担心虚假的政府精神控制计划。

虽然瓦格纳说他希望这次选举是 chemtrail 热情的顶峰，但我并不那么有信心。从披萨饼到俄国调查，美国正进一步分裂成 [无法就一系列共享事实](https://www.vox.com/policy-and-politics/2017/11/2/16588964/america-epistemic-crisis) 达成一致的社区。我想不出化学轨迹会有什么不同的原因。
