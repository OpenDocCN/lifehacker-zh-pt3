# 问咸的女服务员:如果只点水，我是不是最差？

> 原文：<https://thetakeout.com/ask-the-salty-waitress-am-i-the-worst-if-i-only-order-1822308390>

* * *

> 亲爱的咸味，
> 
> 我和老公在外面吃饭的时候，如果没有鸡尾酒的话，一般都是喝水。我不喝汽水了；冰茶不适合我的胃；热饮也没有了。水对我来说是最好的选择——这不仅仅是因为我很便宜。我的问题是，我是否从我的服务器得到一些内部侧目，我选择了一个不增加账单的选项？我总是从 20%开始给小费，但多年来我一直对此感到疑惑。
> 
> 已签名，
> 
> H2Oer

亲爱的 H2Oer，

我们都应该每天摄入 72 盎司的水，但如果健怡可乐不算的话，我喝了大约一盎司的水，那是我刷牙时喝的。所以是的，对你有好处。



你的问题让我想起了我收到的来自 [布兰迪·麦克博林菲【](https://thetakeout.com/ask-the-salty-waitress-is-it-weird-that-i-always-order-1821708542)】T3】的邮件，这个家伙总是点同样的东西，并担心他的服务员会对他评头论足。你的“怪癖”都没有那么奇怪或讨厌。

很多人只是点了水，有一百万个理由:他们不喜欢苏打水；他们正在戒酒。他们是指定的司机；也许他们只是渴得要命。高级餐厅会让你选择无气泡或气泡、瓶装或自来水或反渗透或 [生水](https://thetakeout.com/twitter-reacts-to-raw-water-movement-with-open-minded-1821713797) ，但如果你喜欢自来水，你不应该为点昂贵的东西而感到内疚。

你还加上了一句“如果我没有鸡尾酒”，意思是你有时会点一杯鸡尾酒。但是，即使你没有偶尔点一些酒，*选择水是好的*。这是一个自由的国家，女士，你不应该再考虑你的水合真理。你可能有惊人的皮肤。

如果你想要进一步的保证，给你。在一家昂贵的餐馆，一杯冰茶最多要 5 美元？如果你给 20%的小费，如果你只点了水，我就少了一美元。那对我来说不是成功就是失败。更重要的是你有一顿舒适的晚餐，包括享用你选择的饮料。另外，你需要保持水分来消化这些盐和糖。
