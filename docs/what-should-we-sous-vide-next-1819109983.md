各位网友大家好，欢迎回到另一个激动人心的话题——为 [*挑选话题。*](http://lifehacker.com/tag/will-it-sous-vide) ，列在那里我用我的沉浸循环器做任何你想让我做的事。

现在我已经把从我的系统中剔除，我准备好回到我的初恋。下面是我不介意保存的一些东西:

*   一个 terrine: 或者某种 paté。他们两个我都爱。我也很接受鸡肝慕斯。
*   某种利口酒:也许是柠檬酒？或者我们应该做些更草本的？
*   **肉丸:** [上次有人建议这个](https://skillet.lifehacker.com/1803140419) 我很失望。
*   **章鱼:**因为没有什么比煮过头的章鱼更可悲，但也没有什么比做对的章鱼更光荣。

一如既往，我欢迎你所有的创意和/或疯狂的建议。只是 [没有更多的烤货](https://skillet.lifehacker.com/dont-sous-vide-bread-unless-you-like-the-taste-of-gummy-1802761145) 。我不想再次心碎。
