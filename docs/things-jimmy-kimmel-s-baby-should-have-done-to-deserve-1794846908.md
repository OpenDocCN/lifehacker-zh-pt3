# 根据共和党人的说法，吉米·基梅尔的孩子应该做的事情应该得到医疗保健

> 原文：<https://lifehacker.com/things-jimmy-kimmel-s-baby-should-have-done-to-deserve-1794846908>

昨晚，深夜节目主持人吉米·基梅尔发表了一段感人肺腑的独白，讲述了他刚出生的儿子出生时有心脏缺陷，仅三天大就接受了救命手术。今天，共和党人正试图获得通过医疗保健法 的投票，这将使像 Kimmel 的孩子这样的人几乎不可能得到他们负担得起的医疗保健。



“任何父母都不应该决定他们是否有能力挽救孩子的生命，”Kimmel 说。但是，计划投票支持失败的[【AHCA】](http://vitals.lifehacker.com/what-s-going-to-happen-to-my-insurance-if-the-obamacare-1793258760)、现在随着 [麦克阿瑟修正案](http://docs.house.gov/billsthisweek/20170424/MacArthur%20Amendment.pdf) 复活的立法者，认为这是一个很好的场景，可以纳入一个国家的医疗保健版图。该修正案将允许保险公司销售不包括产妇护理、新生儿护理、手术或住院的保单。保险公司也可以对已有疾病的人收取天文数字的费用(比如小比利，当他长大后)。他们还可以继续销售没有自付费用上限的保单——因此，在几千美元之后，保险公司可以耸耸肩，说他们已经支付完了。

没关系，计划投票支持僵尸特朗普医改的共和党人(抱歉是党派之争，但他们都是共和党人)说。他们说，保险计划可以排除必要的护理，这没什么，因为低劣的保险计划 比好的保险计划便宜。不管怎样，你真的需要保险吗？吉米·基梅尔的宝宝可以尝试这些方法中的任何一种:

### 过“美好的生活”

阿拉巴马州众议员莫·布鲁克斯(Mo Brooks)喜欢新的特朗普医保条款，该条款允许保险公司向有先存疾病的人收取任何费用。那些家伙 [的保费估计是骇人听闻的](https://twitter.com/ASlavitt/status/857034899575975937) 。这意味着像吉米·基梅尔这样的家庭(包括那些有同样健康问题但钱少得多的家庭)将不得不每年多支付数千美元——或者根本没有保险。

他在接受美国有线电视新闻网 采访时说:“(向以前有健康问题的人收取更多费用)有助于抵消所有这些成本，从而减少那些过着美好生活的人的成本，他们很健康，他们做了一些事情来保持身体健康。”。听到了吗，比利？你应该做些事情来保持身体健康。就像，你知道，长了一颗更好的心。

众议员布鲁克斯确实明白，过着“美好生活”的人无论如何都会生病。他只是不在乎:“公平地说，这些人中有很多人已经有了疾病，他们有这些疾病并不是因为他们自己的过错...我认为在这种情况下，我们的社会需要帮助。然而，挑战在于，这是一个艰难的平衡行为，在这些强制措施的较高成本之间，人们因为负担不起健康保险政策而被拒绝保险...并且有足够的覆盖面来帮助那些真正需要帮助的人。”

所以我们需要帮助，但不是要花任何人的钱。也许是众议员布鲁克斯需要长个更好的心脏？

### 选择不生病

“当然，这是一种商业模式，”宾夕法尼亚州代表迈克·凯利 [告诉 CNN](http://www.padems.com/2017/05/congressman-kelly-pennsylvanians-pre-existing-conditions-losing-insurance-course-business-model/) 。“(你给保险公司带来的)风险越大，(你的)保费就越高。”

他说这对年轻和健康的人来说是公平的，他们现在可以“挑选”他们的保险，尽管还不清楚人们如何“挑选”他们以前有什么健康问题。

### 在国会找份工作

既然还没出生就很难过上好日子，或者没有时光机和魔杖就很难改变健康史，那就多了一个选择:你可以当国会议员，也可以为国会工作。

麦克阿瑟修正案 [的条款不适用于国会议员或他们的工作人员](https://www.vox.com/2017/4/25/15429982/gop-exemption-ahca-amendment) ，所以如果比利长大后成为一名国会议员，他不会因为他先前存在的状况而被收取额外费用，他会知道他的家庭政策必须涵盖基本的健康福利，如产妇护理和救生手术。AHCA 的其余部分仍然适用，但其和[负担最重的是老年人和低收入人群](http://vitals.lifehacker.com/what-s-going-to-happen-to-my-insurance-if-the-obamacare-1793258760)和，所以如果他能设法成为一个富有的年轻人，他应该会做得很好。

如果其他方法都失败了，他可以少买几部苹果手机 。
