# 只有当你准备好失去比特币时，才买它

> 原文：<https://lifehacker.com/only-buy-bitcoin-if-youre-ready-to-lose-it-1820006087>

如果你想参与区块链的货币交易，首先读一下科技作家马克·弗劳恩菲尔德关于失去 3 万美元比特币 T3 的故事。问问你自己，如果你知道每次你猜错了，你的钱就会被锁起来几个小时，然后几天，然后几年，你是否能承受试图猜出一个七位数密码的压力。问问你自己，如果你的投资得到了十倍的回报，却葬身火海，你会怎么做。



弗劳恩菲尔德为了安全起见陷入了困境。2016 年 1 月，他购买了 3000 美元的比特币，然后将其锁在一个名为 [Trezor](https://trezor.io/) 的专用 USB 设备中。他这样做是为了让没人能通过黑他的电脑，甚至是偷设备来黑他的比特币。该设备由一串 21 个单词和一个 7 位密码锁定。弗劳恩菲尔德把这些话写在一张纸上，他的清洁服务把这张纸扔了出去。他记住了他的密码，但后来又忘记了。收回他的钱压力太大了，光是阅读就让我脸上的血都流干了。

当然，弗劳恩菲尔德犯了一个很大的错误，他的密码太不安全了，根本没有保存他的密码。你可能认为你不会犯这样的错误！但是你可以把你的 PIN 码和密码保存在某个*也*容易拿到的地方。这是假设你甚至会买一个花哨的安全 USB 设备，而不仅仅是一个典型的闪存驱动器。它还假设你永远不会丢失设备。或者把比特币留在你的电脑上，然后拿到被黑掉的*。*

或者你根本不会把比特币放在电脑上，而是放在网上交易所。我在 2011 年就是这么做的，当时 400 美元的比特币现在价值超过 30 万美元。除了我在交易网站 Mt. Gox 上买了我的比特币，并把它放在那里三年，直到 [该网站关闭交易，并声称所有的钱都被盗](https://www.cyberscoop.com/bitcoin-mt-gox-chainalysis-elliptic/) 。计算一下刚才的 30 万美元...不好玩。意识到自己丢了一堆改变人生的钱，很痛苦。也许我不应该把我的加密货币托付给一个始于卡片交易论坛的网站——mt . Gox 最初代表魔术:在线交易聚会。但我用我的安全换取了便利，因为我想不用做太多工作就能进入比特币。

因此，我失去了我的钱，俄罗斯交易所 BTC-e 的用户也是如此，该交易所因洗钱指控关闭了。当 [窃贼闯入他的 Gmail 和 T-Mobile 账户](http://fortune.com/2017/08/22/bitcoin-coinbase-hack/) 从他的比特币基地账户中偷走他的比特币——以及他的以太坊，另一种流行的加密货币——时，人工智能高管肖恩·埃弗雷特也是如此。许多人买了一些比特币，然后把它交给别人，或者放在不安全的电脑上，或者放在不知何故消失的 USB 驱动器上。管理你的比特币需要大量的研究和工作。即便如此，也没有万无一失的方法来保住你的比特币。每种存储方式都会带来一些安全风险。

所有储存传统货币的方式也是如此！只是我们已经建立了精心设计的系统来保护这些钱，比如银行、联邦存款保险以及信用评级。所有这些有时也会失灵，这也是替代货币吸引力的一部分。全球金融危机、财政悬崖和银行黑客攻击都表明政府发行的货币基础不稳固。

但是围绕加密货币的安全措施，仍然是新的和陌生的，不断地瓦解。弗劳恩菲尔德故事的一个剧透:他只能通过一个辅助黑客来找回他的钱，这在数字上相当于雇一个锁匠闯入他自己的家。这种攻击也可能被真正的攻击者利用，绕过 Frauenfelder 的 USB 驱动器附带的所有昂贵的安全措施。他之所以能拿回他的钱，是因为它最终还是不安全的。

如果你准备好冒这种风险，那就去做吧！为了让加密货币流行起来，变得更加安全和方便，首先人们需要犯所有这些错误。但是现在，不要放入你还没准备好失去的东西。投资加密货币既有趣又好玩。但是，尽管加密货币的系统还很新，人们对它的了解还很少，但它是一种高风险的货币。如果你真的不需要花这笔钱，那就不要把它当成一场赌博或者一种爱好——一种你乐意投入金钱的东西。投资更传统的渠道——如果你失去了所有的钱，至少每个人都不会说，“嗯，废话。”
