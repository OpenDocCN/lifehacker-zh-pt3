# 城市安全跑步初学者指南

> 原文：<https://lifehacker.com/the-beginners-guide-to-safe-urban-running-1727699007>

在大都市的中心地带跑步与在安静的街区跑步或者在小道上跑会有很大的不同。风景本身就是一个障碍，每个角落都潜伏着潜在的危险。幸运的是，如果你知道如何驾驭它们，在城市中跑步是保持身材的好方法。

### 第一:经常练习基础知识

归根结底，在市中心安全跑步并没有什么不同。毕竟，这仍然是同样的锻炼形式；你只是在一个不太有利于它的领域里做。所以，不要忘记那些让你处处安全的小事:

*   保持警惕，抬起眼睛，这样你就能发现你周围的任何危险
*   过马路前先看看两边
*   如果有指定的慢跑路径或人行道，使用它
*   尽可能与路边或马路保持距离
*   穿鲜艳的颜色，甚至反光板(尤其是在晚上)

此外，不要忘记其他一些让你顺利起步的要素。买一双好的跑鞋，这样你就不会伤到你的脚，绑紧鞋带以适应你的脚型，慢慢开始，这样你就不会拉伤了。这份指南将关注许多城市特有的提示，但不要忘记适用于各种跑步者的重要内容。

### **绘制最安全的路线**

这座城市是一个活生生的地方，对跑步者来说充满了潜在的危险。因此，在出发前计划好跑步的各个方面，而不是心血来潮，这一点尤为重要。

达拉斯警察局的社区关系官员希拉·卡瓦纳解释说，最重要的一步是 [注意你所在城市区域内的“危险点”](http://www.runnersworld.com/advice/how-to-run-more-safely) 。“危险点”是能见度低的无人居住区域，在那里有人很容易攻击你。使用谷歌地图或运行中的地图工具 来绘制一条你知道会让你远离危险地点的路线。如果你不确定，把它画出来，然后和朋友一起开车或散步去寻找。

位置并不是唯一重要的事情:考虑你要跑步的时间，尽量避开行人和汽车流量大的时候。当周围行人少的时候，你在人行道上躲避的障碍物就少了。当周围行驶的汽车较少时，就更容易看清道路(你也不太可能被撞)。此外，纽约市特殊外科医院的医学博士詹妮弗·所罗门建议，通过避开高度拥堵的街道，当你在第三英里上喘气时，你会吸入更少的汽车尾气 。所以当你计划跑步的时候，考虑在清晨，或者晚高峰时间之后，路上车和人少的时候跑步。只要确保你仍然停留在视野开阔的地方。

### **带上合适的齿轮**

一旦你画出了路线，你会想要收集一些物品随身携带。这听起来可能不太舒服，你可能需要在如何做这件事上发挥创造力，但是有一些基本的东西你应该在城市里一直带着:

*   某种个人身份证明(身份证、驾驶执照等)。)
*   相关医疗信息(医疗状况、过敏等。)
*   几美元用于应急用水、食物或乘坐公共汽车(如果有必要的话，可以塞在鞋子里)

如果你认为携带所有这些听起来很痛苦，你可能会想要考虑获得一个 [Road ID](http://www.roadid.com/) 。Road ID 是专为跑步者准备的物品，你可以戴在手腕上、脖子上，甚至系在鞋子的鞋带上，它在一个地方列出了所有最重要的信息。例如，它可能会列出你的名字，你来自哪里，一些紧急联系电话，并说你没有已知的过敏症。你还可以将道路标识 [附加到你的健身追踪器](https://www.roadid.com/Common/Catalog.aspx?C=ForFitbit) 上，这样你就不必再戴一个额外的手镯了。

带上你的手机也不是一个坏主意。你可能已经这样做了，这样你就可以听音乐或者用应用程序 跟踪你的跑步，但是当你需要打紧急电话时，它可以是一个救命稻草。如果你觉得有人跟踪你，你可以报警；如果你受伤了，你可以叫救护车；如果你把自己逼得太紧了，你甚至可以叫朋友来接你。如果你不想带你闪亮昂贵的新手机，记住大多数旧手机仍然可以拨打紧急电话，即使它们没有激活。

### **考虑不带耳机跑步(或者保持低音量)**

对于大多数人(包括我自己)来说，这可能是一场斗争，但不带任何耳机跑步是城市中最安全的跑步方式。这有几个原因:音乐和播客在你耳边响起，让你很难听到周围发生的事情；你不能专注于你的环境，因为你专注于你正在听的东西；最后但并非最不重要的一点是，许多人把音量调得太高，慢慢地损害了他们的听力。

音乐对于保持你的能量和步伐是惊人的，但你可能会惊讶于在没有任何音乐或小工具 T3 的帮助下跑 [是多么令人耳目一新。你可以专注于跑步的纯粹乐趣，更多地关注你的身体发出的反馈，你有机会在跑步时真正探索你的想法。当你锻炼的时候，把它当成一个冥想的机会。](https://lifehacker.com/easy-ways-to-refresh-your-run-and-make-it-more-fun-1603531251)

如果你不相信，还有一些其他的选择。您可以将音量调得尽可能低，这样您仍然可以听到您的音乐，但仍然可以听到您周围发生的事情。我的诀窍是在播放音乐时，看看我是否能听到别人在跟我说话。与室友、家人交谈，或者打开电视，确保你至少能听到有人警告你有危险。

你可能还想考虑购买一副开耳耳机，比如用于有线连接的 [松下 RP-HGS10-W](http://www.amazon.com/Panasonic-RP-HGS10-W-Conduction-Headphones-Reflective/dp/B00QN43YFU/ref=sr_1_6?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-beginners-guide-to-safe-urban-running-1727699007&asc_source=&ie=UTF8&keywords=open ear headphones&qid=1441031777&s=electronics&sr=1-6&tag=kinjalifehackerlink-20) 和用于蓝牙连接的[after shokz as 330 f Bluez](http://www.amazon.com/Aftershokz-AS330F-Bluetooth-Wireless-Headset/dp/B00E92LJNM/ref=sr_1_7?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-beginners-guide-to-safe-urban-running-1727699007&asc_source=&ie=UTF8&keywords=open ear headphones&qid=1441031842&sr=8-7&tag=kinjalifehackerlink-20)。开耳耳机使用骨传导将声音发送到大脑，因此您仍然可以听到周围发生的事情，两全其美。你也可以选择一种耳机设计，它位于你的耳朵中，但不“插入”你的耳朵，就像索尼 MDR-J10 耳机一样。这些可以让你从 50 美元到几百美元不等，所以弄清楚你想要什么，在你扔钱之前货比三家。最后但同样重要的是，如果你有一部 iPhone，你可以 [下载感知应用](http://lifehacker.com/awareness-keeps-you-mindful-of-your-surroundings-even-5896401) ，它可以在你听音乐的时候过滤你周围环境中的声音。

### **考虑和一个伙伴一起跑步(尤其是在晚上)**

你知道在电视和电影里人们总是在晚上被抢劫或袭击吗？是啊，那是因为 [更多的抢劫发生在晚上](https://books.google.com/books?id=-O0zZygYdNUC&pg=PA380&lpg=PA380&dq=Most+street+robberies+occur+at+night&source=bl&ots=ehCMRLD7rY&sig=zxJ866uF3DvBNEIi35I1MHNYf-0&hl=en&sa=X&ved=0CD0Q6AEwBGoVChMIlZ7C68LTxwIVg20-Ch0eSgCw#v=onepage&q=Most%20street%20robberies%20occur%20at%20night&f=false) 。如果你要在晚上跑步，或者其他天黑的时候，你最好和一个伙伴一起跑。 [劳伦·哈格雷夫主动解释道](http://www.active.com/running/articles/6-running-safety-tips) :

> 两个人比一个人更难控制，所以攻击者不太可能攻击，如果他们攻击，您的生存机会就增加了一倍。如果你没有可以一起跑的人，那就养一只狗。或者借一只狗。这不仅会让你成为一个不那么有吸引力的目标，狗有时会比我们更早感觉到危险。

很明显，一只小小的小狗并不适合，但是你明白了。攻击者希望事情变得简单。身边有另一具尸体会增加很多麻烦，而且可能会吓退很多罪犯。

但是如果你认识攻击者呢？或者说如果攻击者不是随机去追人呢？跟踪者和熟人变成坏人是一个不幸的现实，尤其是对女性而言。有一些小技巧可以用来避免或处理这些威胁，特别是如果你喜欢在天黑的时候跑步:

*   定期改变你的路线，这样坏人就不知道你会在哪里
*   不要在社交媒体上分享太多关于你在哪里跑步的信息
*   考虑上 [防身术课](https://lifehacker.com/basic-self-defense-moves-anyone-can-do-and-everyone-sh-5825528)([Krav Maga](https://en.wikipedia.org/wiki/Krav_Maga)有人吗？)
*   携带狼牙棒或胡椒喷雾

最后，也是最重要的一点，明尼阿波利斯跑步中心的内森·弗里伯格推荐 你总是告诉别人你要去哪里。即使只是给你在网上信任的人发一条短信或信息。你也可以给你的室友或家人留个便条，告诉他们你跑步的大致情况。

### **在道路的“正确”一侧行驶**

如果你是一个经常跑步的人，你可能已经知道你应该逆着车流跑，这对城市跑步者来说更重要。逆着车流跑可以让你看到向你驶来的车，这样你就可以做出反应，在一辆车失去控制时躲开。然而，有时这个规则是无效的，甚至是不安全的。国家安全委员会项目开发主任詹姆斯·所罗门、《傻瓜跑马拉松》一书的作者杰森·卡普和美国路跑者俱乐部的执行董事让·克纳克建议, 你可以在道路的任何一边跑步:

*   为您提供最佳视野(前方和后方)
*   给迎面而来的汽车提供最佳的视野
*   有专用的运行路径
*   至少有一条人行道或路肩
*   有一个出口(万一一辆车向你驶来，你可以跳水或奔跑的地方)

不要总是假设逆着车流跑是最好的选择。你希望司机看到你的次数和你看到他们的次数一样多，甚至更多。如果可能的话，远离马路，随时准备逃离某个地方。

### **小心接近十字路口和街道**

穿越繁忙的十字路口是城市跑步者面临的最大危险。如果跑步者不注意，他们可能会被车撞。如果司机注意力不集中，跑步者可能会被车撞。看看跑步者无论如何都会输掉这场战斗？除了朝两边看，在每个十字路口还有几件事你应该做。

[主动建议](http://www.active.com/running/articles/tips-for-safe-night-runs) 尽可能跟在车辆后面跑(尤其是转弯车道上的车辆)。仅仅因为他们停下来并不意味着他们看到你，知道你要跑过去。Raia 还建议你在进入十字路口时，试着与司机进行眼神交流，确保他们认出你。我喜欢挥手或举起手来表示“嘿，我在这里”，以示确认。我几乎被三辆车撞过(其中一辆居然给了我一个爱心水龙头)，但自从我开始挥手，还没有这么近过。

试图闯人行横道灯也不是个好主意。你在赌你会及时赶到，而且有点太依赖你的身体了。你可能会绊倒，绊倒，或有其他“哦”的时刻，当灯变了的时候，你可能会倒在地上。另外，你正从大多数司机的视野之外冲出，所以当他们踩下油门时，他们可能不知道你在那里。在红绿灯处等下一个白人告诉你没事。如果你想保持心率，原地慢跑。或者更好的是，在你等待的时候做其他运动，比如俯卧撑和深蹲。

也不要对司机做出假设。不要假设他们看到了你，或者他们真的会在停车标志前停下来。小心接近每一个十字路口，这样你就能活着迎接新的一天。

### 在人行道上与人擦肩而过时要有礼貌

礼貌总是有好处的，尤其是当你不得不与一大群过着各自生活的行人共享人行道时。当你在你的城市跑步时，遵循这些基本的礼貌规则，以避免事故或不必要的对抗:

*   如果你和别人一起跑步，那就排成一列，或者至少在接近别人的时候排成一列
*   坚持右边，从左边通过，就像你是路上的一辆车
*   美国路跑者俱乐部推荐 当你经过行人时，用简单的“在你左边”或“对不起”来提醒他们

最后一点尤其重要。以我个人的经验来看，我曾经“悄悄靠近”那些没有意识到我在那里的人，几乎引发了一种自卫行为。最后，如果你认为有人没有注意，他们可能没有。当你接近他们时，安全比抱歉好，减速以确保没有碰撞。在人行道上狂奔的孩子也是如此。给他们足够的空间，放慢速度。

### **相信你的直觉(大多数时候)**

你内心深处已经知道这个城市可能很危险，你可能知道什么时候感觉不对劲。感受，不要思考，相信你的直觉。 [詹·马茨在散步慢跑时解释道](http://blog.walkjogrun.net/2014/09/22/10-ways-to-stay-safe-on-early-morning-runs/) :

> 如果一条街看起来太暗，不要沿着它跑。如果一个人向你走来，让你感到紧张，转过身去。如果你觉得有人在跟踪你，就报警。安全总是比后悔好，所以相信你的直觉。

仅仅为了锻炼而冒生命危险是没有意义的。也就是说，有些本能你需要更加警惕去相信。有时你的直觉可能会告诉你，你比实际能力更强，这可能很危险。《跑步者世界》的凯蒂·麦克唐纳·内兹将 解释为许多跑步者都受其影响的“无敌情结”。你感觉很强壮，因为你经常跑步，所以你可能认为你可以随时随地跑步，没有人能追上你。另外， [你很敬业](https://lifehacker.com/a-road-map-for-taking-your-running-to-the-next-level-1719785814) ，所以可能会把你的训练放在个人安全之前。你可能觉得你可以做任何事情，但是保持冷静，听从你的安全本能。
