# 今天是全国奶酪汉堡日

> 原文：<https://lifehacker.com/today-is-national-cheeseburger-day-1829121511>

今天是全国芝士汉堡日，虽然我不太相信这些人为的美食假日——如果你爱自己，每天都是芝士汉堡日——但我确实喜欢一两个。这里有几个地方你可以买到便宜的(或者免费的)汉堡——还有煎饼！)今天:

Watch

*   [IHOP](https://markets.businessinsider.com/news/stocks/celebrate-national-cheeseburger-day-at-ihop-restaurants-with-a-free-side-of-pancakes-1027539928) :购买任意一款终极牛排汉堡，可免费获得两张煎饼。(因为谁不想要南瓜香料煎饼配汉堡呢？)详见 [他们的脸书事件](https://www.facebook.com/events/481389735676961/) 。

*   [Ruby Tuesday](http://rubytuesday.com/CheeseburgerDay2018) :“如此联系”的会员可以在 9/18 星期二或 9/19 星期三购买主菜，获得一个免费汉堡。
*   [温蒂的](https://lifehacker.com/how-to-get-a-free-wendys-cheeseburger-every-day-this-mo-1829027843#_ga=2.213517972.1643967429.1537201724-426097860.1521480505) :使用 [温蒂的 app](https://www.wendys.com/wendys-app) 在本月每天购买任意其他商品，即可获得一个免费的芝士汉堡。
*   :整个月使用他们的应用程序，1 美元就能买到(几乎)任何种类的三明治。
*   [索尼克](https://www.sonicdrivein.com/deals) :索尼克其实每周二从下午 5 点到打烊都有 1/2 价位的汉堡*，本周二也不例外。他们也将在本周三提供一美元的热狗。我知道热狗不是芝士汉堡，但它仍然很好吃，我想你应该知道。* 
*   [强尼火箭](https://www.johnnyrockets.com/eclub/) :如果你注册了火箭电子俱乐部，你可以用购买另一个项目获得一个免费的汉堡。
*   [【白色城堡】](https://www.whitecastle.com) :打印 [此券](https://www.whitecastle.com/cheese-slider-day-coupon) 凭此券购买任何其他商品可免费获得一个奶酪滑块。
*   [汉堡 Fi](https://burgerfi.com) :买一个芝士汉堡，一美元买一个。
*   [【红罗宾】](https://www.redrobin.com/national-cheeseburger-day.html) :购买任意饮料，五块钱就可以获得任意美食汉堡和无限牛排薯条。
*   [【Wayback Burger】](https://waybackburgers.com):用应用程序购买另一个芝士汉堡，即可获得免费芝士汉堡。

如果你不想出去，你还是应该庆祝一下。我可能会做这些 [薄博伊斯](https://skillet.lifehacker.com/thin-burgers-are-better-than-thick-ones-1827807997) 之一，但如果你从未经历过 [金智敏露西](https://skillet.lifehacker.com/how-to-make-a-juicy-lucy-with-any-cheese-1827842827) ，今天可能就是这一天。
