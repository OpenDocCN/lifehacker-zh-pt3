# iPhone 浏览器对决:Chrome vs. Safari

> 原文:[https://life hacker . com/iphone-浏览器-摊牌-chrome-vs-safari-1793167099](https://lifehacker.com/iphone-browser-showdown-chrome-vs-safari-1793167099)

Safari 长期以来一直是 iPhone 上的首选浏览器，但在苹果最终在 iOS 8 中向其他浏览器开放了 Safari 的秘密速度增强功能后，现在有可能完全放弃 Safari，转而使用另一种浏览器。Chrome 是最明显的选择。但是值得吗？

### 竞争者

你在 iOS 上有许多浏览器选项，但其中大多数，如注重隐私的 [【勇敢的](https://itunes.apple.com/us/app/brave-web-browser-fast-with-built-in-adblock/id1052879175?mt=8) 、基于手势的 [海豚](https://itunes.apple.com/gb/app/dolphin-web-browser-fast-internet/id452204407?mt=8) ，或以速度为中心的 [Opera Mini](https://itunes.apple.com/gb/app/opera-mini-web-browser/id363729560?mt=8) ，都填补了一个空白，而不是试图成为你的日常司机。在这方面，Chrome 和 Safari 领先一步。下面是它们的功能集的简要总结:

*   Safari 浏览器 : Safari 浏览器是我们多年来熟知的默认浏览器。它做了浏览器需要做的一切，包括提供私人浏览模式、书签，以及从 iOS 9 开始， [广告屏蔽](https://lifehacker.com/the-best-ad-blockers-for-ios-9-1731433501) 。Safari 与 iOS 深度整合，这意味着你可以在 Spotlight 中搜索某样东西，然后在 Safari 中打开它，而且默认情况下大多数应用都会打开 Safari 中的任何链接。
*   [**Chrome**](https://itunes.apple.com/us/app/chrome/id535886823):Chrome 最初登陆 iOS 时，速度很慢，但仍能与谷歌的其他产品配合得如此之好，以至于 [人们还是使用了它](https://lifehacker.com/most-popular-ios-web-browser-chrome-5928529) 。现在，两者速度一样快。除了你在任何浏览器中预期的功能，iOS 上的 Chrome 还有一些谷歌特有的功能，如内置的 QR 扫描仪、Google Now 支持、翻译等。Chrome 还倾向于隐藏大量界面，放弃 Safari 中普遍存在的底部导航栏，而是将所有内容都放在顶部。这是一件美学上的事情，但设计足够重要，你可能更喜欢其中一个的工作方式。

哪种浏览器最适合你在一定程度上是个人偏好的问题，但也与你使用的其他应用程序有关。让我们深入探讨 Chrome 比 Safari 更有意义的地方，反之亦然。

### Chrome 的语音搜索非常好用

如果你想用声音在网上搜索东西，你应该使用 Chrome。谷歌在 iOS 的 Chrome 中大力推广其语音助手，效果好得令人傻眼。只要用力按下 Chrome 图标，你甚至可以在打开应用程序之前就进行搜索，一旦你进入 Chrome，你就会发现语音搜索图标无处不在。

更重要的是，语音搜索是准确的，完美地调出大多数结果。这听起来像是一个小功能，但如果你不能或不擅长在手机的小键盘上打字，拥有本质上的听写功能就方便了。

就其本身而言，Safari 有内置于 iOS 的常规老式听写按钮，但它不是语音搜索专用的，也不太好用。当然，你也可以使用 Siri 进行搜索，但它不像 Chrome 中的语音搜索那样流畅或有用。

### Chrome 可以和谷歌的所有应用程序兼容，Safari 可以和任何东西兼容

不久前，苹果推出了一项“深度链接”功能，本质上是让应用程序相互交流。虽然它不能取代 Android 上的“默认”应用程序，但它确实让开发者可以在 Chrome 而不是 Safari 上打开 Gmail 的 URL 链接。如你所料， [谷歌充分利用了这个](https://lifehacker.com/how-to-get-a-perfect-all-google-experience-on-the-iphon-1740638647) 。

如果你是许多服务的谷歌用户，从 Drive 到 Gmail 到地图，你可以让所有这些应用程序都与 Chrome 通信。例如，当你在 Chrome 中点击一个地图链接，它会打开谷歌地图，或者在 Chrome 中点击一个电子邮件链接，它会打开 Gmail。如果你对谷歌有很深的投资，这非常有用，让你可以完全忽略 Safari。

如果你不是一个重度谷歌用户，事情就没那么有趣了。Safari 仍然是默认浏览器，这意味着你在短信、Apple Mail 中获得的 URL，或者你在 Spotlight 中搜索的链接，都是通过 Safari 路由思想的。虽然一些第三方应用程序，如或 [航空](http://airmailapp.com/ios) ，有一个“打开”选项，让你选择你的浏览器，但并非所有应用程序都是如此，所以你肯定会不时地使用 Safari。除非你非常喜欢谷歌，否则 Safari 可能是你最好的选择，这样你就不必总是在两者之间徘徊。

### 两者都与其桌面应用程序同步

你选择一个浏览器而不是另一个的最后一个重要原因是。Safari 和 Chrome 都可以与桌面同步。这意味着你的标签页、历史记录和书签都会从手机同步到桌面。如果这对你很重要，那么你会想用你电脑上的任何一个浏览器。

就我个人而言，虽然同步似乎总是一个很酷的想法，但我从来没有使用过，所以这对我来说没有我最初认为的那么重要。如果你在同一条船上，并没有真正利用这一点，不要让你的想法影响你决定使用哪个浏览器。

### Safari 的阅读列表很棒，但 Chrome 也在路上

Safari 中一个经常被遗忘的功能是它的阅读列表。 在这里，你会得到一个类似 Instapaper 的文章列表，这些文章是你保存下来以后阅读的。如果你在手机上阅读很多内容，并且对 Pocket 或 Instapaper 等更高级的选项没有兴趣，那就太好了。

Chrome 现在根本不提供这种功能，但据传很快就会推出。在我们知道它的实际工作原理之前，Safari 是你更好的选择。当然，就目前而言，Chrome 用户仍然可以使用类似于 [Pocket](https://getpocket.com/a/queue/list/) 或 [Instapaper](https://www.instapaper.com/) 的第三方 app。

### Safari 可以屏蔽广告和追踪器

如果你是一个使用广告拦截器 拦截广告或第三方追踪器 [的人，那么你的选择非常明确:使用 Safari。](https://lifehacker.com/the-best-ad-blockers-for-ios-9-1731433501) 

iOS 9 推出的“内容拦截器”只支持 Safari，Chrome 没有任何类似类型的插件。这些内容拦截器与广告无关，它们还会拦截 cookies、奇怪的页面元素、特定的 URL、评论等等。所以，如果你想尝试这些东西，你应该坚持使用 Safari。

### 结论:如果你深入谷歌的生态系统，Chrome 很棒，Safari 最适合其他人

哪种浏览器最适合你取决于很多因素，包括对其工作方式的总体偏好。有些人喜欢 Chrome 更简约的美学，以至于功能实际上并不重要。其他人喜欢 Safari 无需任何调整就能自动集成的事实。在任何情况下，如果你喜欢其中一个的“感觉”,要知道它们速度一样快，功能一样全，所以没有理由刻意回避任何一个。

如果你是一个狂热的谷歌产品用户，一个明显的区别就来了。Chrome 在与谷歌其他应用程序的连接方面做得非常好，足以让你感觉你甚至不在 iPhone 上。那样的话，坚持 Chrome，你就开心了。如果你找到其他与 Chrome 有深度链接的应用，你可以假装 Safari 不存在。

对于其他人，坚持使用 Safari。这些年来，它已经取得了很大的进步，并且运行得非常好。是不是有点无聊？当然可以。但是你希望你的浏览器有多刺激呢？
