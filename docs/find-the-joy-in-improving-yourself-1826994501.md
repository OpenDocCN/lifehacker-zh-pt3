# 在提升自己中找到快乐

> 原文：<https://lifehacker.com/find-the-joy-in-improving-yourself-1826994501>

欢迎回到 [**周中冥想**](https://lifehacker.com/c/mid-week-meditations) ，Lifehacker 的每周一次的斯多葛派智慧之池探索，以及使用它的水域来反思和改善你的生活的指南。



本周的选集来自苏格拉底，通过斯多葛派哲学家爱比克泰德。在爱比克泰德的*演讲* (3.5)中，他分享了一句关于自我提升重要性的永恒名言:

> “但是苏格拉底怎么说呢？就像一个人喜欢改善他的农场，另一个人喜欢改善他的马一样，我也喜欢日复一日地关注自己的进步。"

### 这意味着什么

这个非常简单，但这也是它如此伟大的原因。苏格拉底在说，当别人每天都在享受改善他们所拥有的东西时，他却在享受改善 T2 自己。值得指出的是，他并不是说提高自己“很重要”——这是显而易见的——而是说他在这个过程中感到“快乐”。

### 从中吸取什么

事实是，我们喜欢改进事物。这是我们的天性。但是我们优先考虑和选择花费时间去改善的东西经常是扭曲的。人们投入大量的时间、精力和资源来改善他们的房子、汽车、财产或生意。尽管毫无疑问，一个漂亮的家、一辆体面的车和成功的事业肯定能让生活更舒适，但给自己足够的时间、精力和资源来成长为你想成为的人仍然很重要。

接下来是我们应该警惕的虚假改善。我们许多人选择改善不真实的东西——它们是虚拟的。想想人们花了多少时间来练级虚构的角色，照料虚拟农场，或者组织完美的幻想团队。当然，适度的这些事情是有空间的，一点点乐趣也没什么不好，但我们有时会允许它们在我们的生活中占据优先地位。我们做虚假改进的时间越多，做真正改进的时间就越少。

为了找到自我提升的乐趣，你必须把它放在你头脑的最前面。我的建议是，每天，在你做其他事情之前，花一点时间制定一个简短的每日自我提升计划。这并不复杂:

*   选择一个锻炼的时间。

*   想想你的膳食与你的饮食的关系。

*   安排一些与朋友或独自的快乐时光。

*   挤时间学点东西。

*   确保你得到充足的休息。

这样做，你很快就会发现和苏格拉底所说的一样的“关注自我进步的快乐”。你会*希望*先提高自己，然后再关注其他的事情。

朋友们，照顾好自己的健康，保持稳固的关系，解渴求知，永远不要停止努力提升自己。总有一天，你会失去时间，当那一天到来时，你会后悔你没有完成的事情，而不是你犯的错误 。你也可以尽你所能让自己变得伟大。毕竟，你的身体和思想是你注定要坚持的两样东西。
