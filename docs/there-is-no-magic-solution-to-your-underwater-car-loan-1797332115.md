# 没有神奇的方法可以解决你的水下汽车贷款

> 原文:[https://jalopnik . com/there ' s-no-magic-solution to-your ' s under-water-car-loan-1797332115](https://jalopnik.com/there-is-no-magic-solution-to-your-underwater-car-loan-1797332115)

不幸的是，有些人负债累累。我经常收到那些汽车贷款陷入困境的人发来的电子邮件，大多数人都希望找到一些违背数学定律的解决方案。真正的答案是:没有真正的答案。

我一直以惊人的频率收到这样的邮件:

> “我有一个问题。我的车大约欠了 24，000 美元，我最多只能得到 16，300 美元的折价。我想上一辆卡车，但我希望我的支付保持大致不变，那就是 431.07。”

数百万人开着价值远低于他们贷款余额的汽车到处跑。有多种方法可以让他们潜入水下，现在不是评判和指责的时候。

然而，是时候接受冷酷的事实了。而且不需要数学高等学位也能算出来，只要一个计算器就行。

如果贷款余额是 24，000 美元，而你在交易中得到的最高报价是 16，300 美元，那就是高达 7，700 美元的负资产。不久前，我写道，你可以通过强有力的激励措施租赁一些便宜的东西，打破水下贷款循环。 但通常情况下，只有当你只有几千英镑的存款时，银行才会让你这么做。

一旦你进入 7，000 美元或更高的负资产范围，贷款人就不太可能承担这种风险。将负资产转入租赁的第二个警告是，你必须有良好的信用。很多时候，处于这些水下贷款场景中的人都远低于 700 FICO 标准。

由于租赁不是一种选择，你可以买些东西，然后将余额转入贷款。这是不建议这样的赤字，再次，如果你的信用不好，银行可能不会批准它。

当然，有贷方愿意在这类贷款上冒险，让人们背上沉重的债务，但是我们已经知道通常会是什么结果。【T2

假设说，如果你想滚动 7700 美元，支付 431.07 美元，第一步是启动一个方便的贷款计算器，将 7700 美元分为五年的贷款期限，并假设利率为 5%。总共是每月 145 美元。

然后我们需要从你的目标 431 美元中减去 145 美元，我们得到 286 美元。所以，如果你想保持同样的付款，并滚动所有的负资产，你需要找到一辆卡车，每月给你 286 美元的付款。

在贷款计算器上做一点逆向计算，同样的五年期和百分之五的利率揭示出你需要购买一辆价值 15，000 美元的卡车。现在你完全进入了二手车领域，这使得贷方更不可能将 7700 美元转入你的下一笔贷款。

那么你应该怎么做呢？只有两种选择可以让你保持良好的财务状况。第一种选择是拿出大量现金，以消化部分或全部负资产。第二个选择是继续使用你现有的资金，直到你的贷款余额与你的余额达到平衡，或者更好的是等到你有一些资产来进行你的交易。

抱歉粉碎了你的梦想，但不要怪我。归咎于数学。
