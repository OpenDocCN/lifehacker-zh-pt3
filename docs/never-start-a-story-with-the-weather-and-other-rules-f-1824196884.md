# 永远不要以天气和其他更好讲故事的规则开始故事

> 原文：<https://lifehacker.com/never-start-a-story-with-the-weather-and-other-rules-f-1824196884>

成为一个善于讲故事的人可以提高你在工作中的表达能力，提升你的社交技能，让你总体上更讨人喜欢。但这不是每个人天生就有的能力。如果你不确定如何讲述吸引观众的故事，这些简单的注意事项会给你一个好的开始。



最近，纽约市漫画 [杰夫·西蒙](http://www.jeffsimmermon.com/) 分享了一些他在成为专业单口喜剧演员和讲故事者的过程中学会的讲述更好故事的 [一般经验法则。他甚至在课堂上教授这些技巧，它们似乎对人们很有帮助。以下是一些亮点:](https://medium.com/@jeff.simmermon/this-is-a-boring-shark-attack-8-rules-for-fascinating-storytelling-3b9d2bab6dca)

1.  不要以天气开始故事:没人关心天气、一天中的时间、气味等等。除非这些东西直接影响了故事中实际发生的事情。

2.  从故事的开头开始，到结尾结束:放轻松，塔伦蒂诺。不要试图中途开始让你的观众困惑。给观众一个清晰的开头，中间，结尾。

3.  不要用除了“说”之外的动词来进行对话:不要说“惊叫”、“惨叫”或“咕哝”，自己表演出来！让人们感觉他们也在你的故事中。

4.  永远不要在口头语境中使用网络语言:不要说“标签”不要在你的故事中这样做，也不要在你的日常对话中这样做。使用肢体语言、语调和面部表情来传达意思。

5.  永远不要使用“突然”或“一切都乱套了”这样的字眼:这些都是陈词滥调。打断自己，转而表达突然发生的事情。升级对混乱的描述，而不是说“一切都失控了”把他们带进故事里！

正如你所看到的，这些规则都不难遵守，但是对于听你讲故事的人来说，它们会有很大的不同。你的故事将从单调乏味的打盹变成至少有趣的轶事。嘿，赢了！这一切都是为了讲述一个清晰的故事，将细节限制在必要的范围内，描述事件而不是依赖陈词滥调，并做一点点表演来使角色变得生动。你可以通过下面的链接了解更多关于 Simmermon 讲故事的规则，并查看其余内容。
