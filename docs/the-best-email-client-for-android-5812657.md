# Android 的最佳电子邮件客户端

> 原文：<https://lifehacker.com/the-best-email-client-for-android-5812657>

Gmail 应用程序不再仅仅是 Gmail 的了。在添加了对 IMAP、POP 和 Exchange 的支持，以及大量管理和控制您的电子邮件的功能后，我们建议 Gmail 管理您在 Android 上的所有电子邮件帐户，即使是非 Google-y 帐户。

### ****特性****

*   **支持 IMAP、POP、Exchange/Outlook、雅虎邮件**
*   **统一收件箱**
*   **存档、标记和标记邮件**
*   **阻止和取消订阅电子邮件地址**
*   **静音线程**
*   **存档、删除或回复通知中的电子邮件**
*   **按重要性对邮件排序**
*   **设置假期响应者**
*   **仅允许通过 Wi-Fi 下载附件以保存移动数据**

### ****擅长的地方****

**在 Android 的早期，谷歌有一个单独的电子邮件应用程序来处理你所有的非 Gmail 邮件。然而，现在 Gmail 应用程序支持 IMAP、POP 和 Exchange，所以你可以将几乎所有的电子邮件帐户添加到你的设备中。你甚至可以在一个地方查看所有的收件箱。**

**当然，这是谷歌，所以它也附带了一些方便的奖励工具。如果您受到某人的骚扰，您可以阻止发件人；如果您受到公司的骚扰，您可以取消订阅邮件列表。这两个按钮都是常规的菜单按钮，所以它们比创建一个过滤器或在邮件底部的微小文本中挖掘一个取消订阅按钮更容易访问。**

**你也可以将线程静音，这样你仍然可以收到电子邮件，但不必每次都收到通知。Gmail 还允许您回复或存档/删除(您可以在应用程序的设置中选择)通知。**

### ****不足之处****

**Gmail 流行是有原因的，但它仍然不完美。您仍然不能像使用 Boomerang(不幸的是，它不支持 POP 或 IMAP)一样，在以后的邮件中提醒自己。也缺乏与日历或待办事项列表应用的集成。虽然 Gmail 可以做一些*智能的事情，比如当你收到酒店或旅行信息时，在日历上添加事件，但这似乎仅限于 Gmail 账户。***

### *****比赛*****

***哦伙计。在电子邮件客户端领域有竞争吗？你如何处理电子邮件将在很大程度上取决于你的偏好。 [**回旋镖**](https://play.google.com/store/apps/details?id=com.baydin.boomerang) 是一个顶级的替代选择，如果你可以只使用 Gmail 和 Exchange 账户的话。它可以让你快速设置提醒，在某个时间或地点再次查看邮件。您还可以自定义滑动手势，以便更快地查看您的收件箱。***

***拳手[](https://play.google.com/store/apps/details?id=com.boxer.email)**势均力敌，争夺冠军宝座。它有一个内置的待办事项列表，支持最常见的电子邮件类型，加上比谷歌更多的通知动作。不幸的是，Exchange 支持不是免费的，你需要为一些基本功能支付额外费用，如自定义快速回复或签名。*****

*****Dropbox 的 [**邮箱**](https://play.google.com/store/apps/details?id=com.mailboxapp) 是另一个有吸引力的选择。你可以将电子邮件暂停到稍后，将对话静音，还有一个内置的待办事项列表。它甚至可以随着时间的推移学习你从某些发件人那里采取了什么行动，所以你可以随时滑动来暂停你老板的电子邮件，但存档来自零售商的电子邮件。还有一个缺点是，它只适用于 Gmail 和 iCloud 账户。POP、IMAP 和 Exchange 不可用。*****

*****微软也凭借[**Outlook**](https://play.google.com/store/apps/details?id=com.microsoft.office.outlook)T5】登上了舞台。该应用程序支持大多数 Outlook、Exchange、雅虎、iCloud、Gmail 和 IMAP，另外你可以集成 OneDrive、Dropbox 和 box 的存储，这令人印象深刻。你可以安排电子邮件稍后回复，并与你的日历整合。当然，前提是你的日历是 Outlook。如果你更喜欢微软的服务，Outlook 可能是最好的选择。*****

*****最后一个选项是*技术上*不属于这里，是 GmailT5 的T3 收件箱。收件箱只支持 Gmail 账户，所以它根本不是一个真正的电子邮件客户端。然而，它鼓励 [一种完全不同的电子邮件方式](http://lifehacker.com/how-inbox-changed-how-i-use-email-1719517132#_ga=1.152286265.127976026.1441054335) ，把你的电子邮件变成一个待办事项列表，而不是一个信息系统。它还可以显示重要信息，如跟踪号码、预约、日历约会，以及许多其他有用的功能。*****
