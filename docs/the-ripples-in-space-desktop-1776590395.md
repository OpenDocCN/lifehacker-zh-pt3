# 太空桌面上的涟漪

> 原文：<https://lifehacker.com/the-ripples-in-space-desktop-1776590395>

的桌面结合了一个小小的雨量计和一张漂亮的壁纸，创造出一个完美契合的外观。布满系统显示器的星空和一个很棒的正在播放的小工具，也是一个很好的坐下或第二屏幕显示器，有足够的空间来完成工作。



如果你不熟悉雨量计， [我们的入门指南](http://lifehacker.com/how-to-create-an-attractive-customized-desktop-hud-wit-5828789) 会让你快速上手。一旦你到了那里，你需要做的就是把这个设置变成你自己的:

*   [壁纸](https://wallpapers.wallhaven.cc/wallpapers/full/wallhaven-330726.png)
*   用于 Windows 的 [雨量计](https://www.rainmeter.net/) 系统调试和监控工具
*   [最小字体皮肤](http://lysy1993lbn.deviantart.com/art/Minimal-Typography-updated-1-0-7-280486134) 为雨量计
*   雨量计的 [明文皮肤](http://redsaph.deviantart.com/art/Cleartext-for-Rainmeter-v3-0-1-19-Apr-2016-519796161)

这就是这个的全部，没有太多的活动部件，这很好。nVIsionary 设法获得了一个伟大的外观，而没有添加大量的自定义皮肤和主题的软件包，我们肯定会欣赏。如果你喜欢这里的作品，去 nVisionary 的 Kinja 博客(链接如下)让他们知道你喜欢这个成品！
