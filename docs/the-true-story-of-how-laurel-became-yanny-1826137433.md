# 劳雷尔变成燕妮的真实故事

开头，[月桂夹](https://gizmodo.com/is-this-voice-saying-yanny-or-laurel-heres-the-only-re-1826053448) 表示月桂除了月桂什么也不是。 [这里是片段](https://soundcloud.com/vocabularycom/laurel-original-recording) 的原始版本，由一位为 Vocabulary.com 工作的歌剧演唱家录制。很有可能，你也会听到劳雷尔。让我们将其与病毒片段进行比较，我们会看到到底发生了什么变化，以及为什么我们中的一半人听到的病毒片段是 yanny。

最初的录音来自于词汇网 2007 年的努力，包括该网站最常查找的单词的发音。这是为单词制作的录音。这是由 [杰伊·奥布里·琼斯](http://time.com/5281287/jay-aubrey-jones-laurel-yanny-voice/) 说的，他是该公司委托的八位歌手之一，使用提供的笔记本电脑、麦克风和便携式音箱在家里朗读歌词。(歌剧演员被训练读 [音标](https://en.wiktionary.org/wiki/Wiktionary:IPA_pronunciation_key) ，字典也用的发音代码。)琼斯亲自为该网站阅读了约 36000 字。

Vocabulary.com 的联合创始人兼首席技术官马克·廷克勒说，要处理的声音剪辑太多了，软件自动完成了所有工作:修剪每个文件的开头和结尾，根据需要应用降噪滤波器，并转换为 mp3 格式，从而节省空间和带宽。

mp3 格式的美妙之处在于，它可以将声音文件压缩到一个足够小的空间内，以便于在互联网上共享。mp3 格式的缺点是，它通过从录音中去除声音并引入一些毛刺、微小的噪声来做到这一点。如果您可以接受一个只是轻微压缩的文件，您可以保留大部分原始声音信息。但是，廷克勒说，“这是在 2007 年或 2008 年，所以我们在下采样过程中相当积极。”

我问廷克勒，他是否有可能拿到原始文件进行比较。他在储藏室的 DVD 上找到了它，并上传到了 Soundcloud 上让所有人都能听到。从技术上讲，这并不是真正的原版，因为 Soundcloud 提供的是 mp3 格式，但它最初是以更高质量的格式录制的。但是它比你和你的朋友们争论的版本更有创意。很明显，我们现在可以确切地知道文件被压缩时发生了什么。

### 什么丢了

压缩月桂夹有了重大伤亡:第二个 [语音共振峰](https://en.wikipedia.org/wiki/Formant) 。没有声谱的这一部分，L 可能听起来像“ee ”, R 可能听起来有点像 n。

原因如下。声波都有频率。频率越高，我们听到的音调就越高。但是讲话的声音同时包括许多频率。我们的 [耳朵](http://www.cochlea.eu/en/cochlea/function) 和 [大脑](https://en.wikipedia.org/wiki/Auditory_cortex) 从这混乱的声音中挑出最强的频率。语音科学家称这些共振峰为*。两个最低音调的共振峰给了我们足够的信息来区分一个元音和另一个元音。*

*如果你在分析程序中打开一个声音文件，你可以看到这些共振峰。这正是我的父亲，一个在加州大学宾夕法尼亚分校 做 [演讲的科学家，在我给他看病毒剪辑时所做的。他在一个名为](https://www.calu.edu/inside/faculty-staff/profiles/robert-skwarecki.aspx) [Wavesurfer](https://en.wikipedia.org/wiki/WaveSurfer) 的程序中打开了 mp3，并在厨房桌子上的笔记本电脑上向我展示了我是如何错误地听到 laurel 的，因为共振峰与发音“ee”、“a”相匹配，这是一个模糊的词，可能是“n”，也可能是最后的“ee”燕妮。*

*但是，他承认，如果你真的想听的话，你就能听到劳雷尔的声音。(他一开始怀疑文件是精心制作的暧昧，一个故意的 [音频错觉](https://vitals.lifehacker.com/listen-to-these-auditory-illusions-when-you-re-bored-wi-1826081147) 。)声明一下，从这份档案里除了劳蕾尔，我什么都没听过。*

*我的父亲可以从黑白光谱图中读出共振峰，但 Wavesurfer 可以很好地检测到它们并对它们进行颜色编码。Wavesurfer 同意我爸爸的观点，并把这个文件解读为 yanny。但如果你给它原始文件，它会突出显示一组不同的共振峰:非常清楚地说，劳雷尔。*

*要了解发生了什么，看一下红色和绿色的线。这是前两个共振峰，我们称之为 F1(红色)和 F2(绿色)。它们都徘徊在声谱图的底部。第三个共振峰 F3 以蓝色高高浮动在他们上方，在 laurel 的 R 音中非常接近 F2。*

*但是再看看处理过的片段，你和你朋友分享的那个。(点按上面幻灯片中的箭头。)由于 mp3 压缩，整体噪音更大。这种噪音模糊了 F1 和 F2 之间的差异，让 Wavesurfer 和我们的大脑都认为下面可能只有一个共振峰，即 F1。这意味着较高的线，中间向下倾斜的线，看起来一定是 F2。*

*在剪辑的开始，F1 和 F2 彼此相邻，听起来很低，但如果 F1 在下面，F2 远高于 2000 赫兹，那就是“ee”或 Y 音。消除第二个共振峰改变了我们解读声音的方式。*

### *但是有足够的空间来反对*

*经过病毒处理的剪辑并没有完全消除第二个共振峰，只是让它更难从噪音中分辨出来。如果你希望听到 laurel，如果你的耳朵和大脑可以在低频找到两个共振峰，你仍然可以听到 laurel。但是对于不同的人来说，yanny 的声音可能会更突出， [正如 Gizmodo 本周早些时候报道的](https://gizmodo.com/is-this-voice-saying-yanny-or-laurel-heres-the-only-re-1826053448) 。*

*那么，那些视频和滑块工具显示你可以通过只听高频或低频来改变你所听到的东西呢？事实证明，这种现象也可以归结为语音共振峰。*

*如果你去掉较高的频率，你将只剩下频谱的较低部分，也就是原来的 F1 和 F2。你的大脑将会更加努力地工作来区分这两个共振峰，而不是锁定它认为的 F2 更高的共振峰。相比之下，如果你只听较高频率的片段，你的大脑会把真实的 F3 作为一个可能的 F2，并假设在频谱的较低部分一定有一个 F1。*

*所以这个技巧改变了你的感知，不是因为 yanny“在”频谱的一部分，而 laurel“在”另一部分，而是因为不管怎样，你的大脑只听到了一些共振峰，并猜测其他的可能是什么。*
