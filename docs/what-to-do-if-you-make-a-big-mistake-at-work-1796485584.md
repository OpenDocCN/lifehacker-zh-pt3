# 如果你在工作中犯了大错该怎么办

> 原文：<https://lifehacker.com/what-to-do-if-you-make-a-big-mistake-at-work-1796485584>

犯一个大到足以让你被解雇的错误是没有人想遇到的情况，但是知道如何反应可以让你免于失业。如果你发现自己处在这样一个糟糕的境地，你可以这么做。

### **立即做什么**

一旦你意识到你犯了一个错误，就让你的老板注意到它。这可能是一次可怕的谈话，但是如果你等着，而你的老板从别人那里听说了这件事，你的处境会更糟。你的经理也会希望你解释清楚这个错误是如何发生的，所以也要准备好解释。求职策略师、的创始人汉娜·摩根建议，你可以带着一些建议的解决方案来参加这次对话，既能解决已经发生的问题，又能防止它再次发生。你的老板可能也有自己的解决方案，他们想付诸行动。

如果你没有所有这些细节(比如错误是如何发生的)，承认你的行为，并承诺找出发生了什么，并尽快将解决方案带给你的老板。你的老板会和你谈论错误的影响、发生了什么以及接下来应该发生什么，所以积极主动地向他们表明你了解情况的严重性，并且你正在努力纠正错误，这样你就能更好地避免被解雇。

### **如何修复自己的名誉**

坚持到底是修复你的职业声誉和你与经理关系的关键。推进你老板已经批准的解决方案，让他们了解进展和结果。摩根解释说，与老板开诚布公的交流，即使感觉尴尬或痛苦，也能向他们表明你致力于改善，这让他们更容易在你从错误中前进时支持你。

你还需要在混乱之后的至少六个月内保持警惕。注意不要再犯更大的错误，努力表现出一流的水平来证明你是可以被信任的。这可能意味着投入额外的时间来仔细检查你的工作，确保你已经完成了所有的基础工作，但是如果你的声誉得到恢复，你的经理重新建立了对你的信心，这是值得的。

### **如果你被解雇了怎么办**

有时候，即使人们尽了最大努力，一个大错误仍然会让他们被解雇。如果你发现自己失业了，摩根建议你试着抛开自己的情绪去追求进步:

> 现在是时候反思你为什么会犯这个错误，以及你本可以做些什么不同的事情。你在过去犯过类似的错误吗？你被警告过吗？或者这个错误预示着其他潜在的问题？不要责备。掌控你能控制的东西。这也可能需要参加培训或辅导来强化你的弱点。我们都有可以改进的地方，认识到这一点并采取措施改进并没有错。

当然，对于大多数人来说，被解雇是一件情绪化的事情，所以花些时间来处理这种损失，并为你现在的处境做好准备。你需要找到可靠的证明人，因为你的前任经理可能无法在你找工作时充当证明人。想想你将如何回答未来工作面试中发生的问题。

犯大错，尤其是在工作中，可能很难面对，但你越早向老板说明发生了什么，并承诺扭转局面，你的情况就会越好。
