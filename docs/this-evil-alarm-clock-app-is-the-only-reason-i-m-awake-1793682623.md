# 这个邪恶的闹钟程序是我现在醒着的唯一原因

> 原文:[https://life hacker . com/this-evil-alarm-clock-app-is-the-only-reason-I-m-awake-1793682623](https://lifehacker.com/this-evil-alarm-clock-app-is-the-only-reason-i-m-awake-1793682623)

我相信我的梦想。说我在梦到了机场，或者找到了我没有复习的期末考试的房间。当闹钟响起时，我会暂停它:我正在执行一个非常重要的任务，不能被打扰。这就是为什么我需要这个邪恶的，可怕的闹钟应用程序。

Watch

叫 [我唤醒不了](http://kogcreations.com/) ，有 [安卓](https://play.google.com/store/apps/details?id=com.kog.alarmclock&hl=en) 或 [iOS](https://itunes.apple.com/us/app/i-cant-wake-up!-alarm-clock/id733590762?mt=8&ign-mpt=uo%3D4) 可用。取决于你如何设置，它可以是任何东西，从你现在的闹钟的稍微更有效的版本到把你从床上拖起来、向你身上泼一桶冷水、把你锁在卧室外面的数字等价物。

当警报响起时，你会得到一个谜题。只要你正在拼图，闹钟的音乐就会轻柔播放。你一松懈，声音又变大了。因为我通常有一个婴儿睡在附近，所以我有额外的动力继续工作，所以闹钟保持安静。我的闹钟通常是这样的:

该应用程序有八个可能的任务。我在日常闹钟中使用的六个是不需要太多脑力就能解决的谜题，但它们确实需要我的注意力。当我完成这六项任务时，很有可能我已经足够清醒，能够记住现实生活比我的梦想任务更重要。

但是如果我知道我真的必须起床，我会设置一个包括大人物在内的闹钟。

### 如何让它变得更糟

条形码任务是这个应用程序最好和最差的功能。它不会让你休息，直到你扫描了它要求的条形码。你可以输入自己的条形码，这样你就可以挑选床头柜上的书，或者你浴室里的洗发水瓶子，或者你厨房里的麦片盒子。你也可以从任何标签上剪下条形码，贴在任何你想贴的地方——也许是你汽车的仪表板。

你也可以让谜题变得更难。每一个都有一个难度设置，所以不要在 4x4 的网格上玩记忆，你可以用 5x6 的网格来代替。或者，你可以让你的闹钟让你做长除法和平方根，而不是你在视频中看到的我做的数学题。

如果你在闹钟响后选择打盹，你可以让应用程序播放你最喜欢(或最不喜欢)的播放列表中的音乐。如果你试图通过简单地关掉闹钟来逃避这一点，清醒测试可以救你一命。这意味着应用程序会在你醒来五分钟后检查你。如果你不拖动滑块到“是的，我醒了”，它会再次启动闹钟。

你还可以让应用程序完全锁定你的手机，直到你在高级设置中完成闹钟。你也可以让它在开机时检查未完成的报警，以防你认为重启手机是回到床上的一种方式。

简而言之，这个警报会像你希望的那样邪恶。我曾经在三个小时的车程结束时，在我比平时早几个小时起床开始公路旅行后，让它响了。我的早期闹钟工作正常，我喝了很多咖啡愉快地开车，但在我通常的 7:30 起床时间，我的手机停止了导航，而是让我做拼图。(我把手机塞在夹克下面，找了个地方靠边停车，这样我就可以重启手机，继续我正在做的事情。)如果你和我一样是个窥探者，但负责任地享受最邪恶的设定，这个应用就是个奇迹。
