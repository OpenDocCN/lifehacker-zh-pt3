# 如何选择燃烧清洁、持久的最佳柴火

> 原文:[https://life hacker . com/how-to-pick-the-best-chicken for-clean-burning-long-l-1733720859](https://lifehacker.com/how-to-pick-the-best-firewood-for-clean-burning-long-l-1733720859)

正式入秋了，也就是说是时候 [清理壁炉](http://lifehacker.com/clean-your-brick-fireplace-with-cream-of-tartar-1696693826#_ga=1.106149763.127976026.1441054335)[检查烟囱](http://workshop.lifehacker.com/sweep-your-chimney-without-getting-on-the-roof-1679228929#_ga=1.106149763.127976026.1441054335) 了。这也是建立柴火堆的理想时间。以下是如何挑选和准备最好的柴火。

### 越干燥越好

木柴的经验法则:越干燥越好。木材由小管组成，这些小管将水从根部输送到树干和树枝中。木材被砍伐后，这些管子可以保持数月的水分。潮湿和含水的木材不能正常燃烧。来自火的能量必须首先干燥木材，这是浪费，并且来自绿色(新砍伐的木材)的烟雾是更强的污染物。

风干的柴火是干的，切成小块，便于携带和生火。

### 如何选择木柴

如果你从零售店购买柴火，遵循以下建议，确保你挑选的是最好的木材:

*   确保储存木柴的地方干燥且通风良好。它可以在户外，只要它被覆盖，而不是在一个可能被淹没的地区。
*   寻找深色到灰色的木材，这将意味着更古老和更干燥的木材。
*   把几根木头敲在一起。如果它们听起来空洞，那是件好事。
*   木材应该是轻的，这意味着很少或没有水。
*   寻找末端开裂的木材，这意味着它是干的。
*   挑选树皮缺失或脱落的部分。树皮需要水分才能附着在木头上。
*   不要从最上面选择最新的木材。挖一点，拔出你能找到的看起来最古老的木头。

最好的柴火不仅干而且密。像橡树和山核桃树这样的坚果树是硬木，是非常理想的柴火，因为它们燃烧的更干净，时间更长。松木和白杨等较软的木材会烧得更快。

分销商出售的柴火是按体积出售的，这种体积被称为“绳索”，是 128 立方英尺(8 英尺长，4 英尺高，4 英尺深)的木材。这取决于你在冬天使用多少柴火，这可以让大多数人度过一个季节。如果你需要更少的木头，你可以买一个 8 英尺长，4 英尺高，和木头一样深的“面绳”或“瑞克”。

根据你所在地区的供求情况，一根风干的柴火可以卖到 200-350 美元。绿线也卖的比较低。在春季和夏季购买也可以打折。

### 如何寻找柴火

寻找自己的柴火可能是一次有趣的远足。在春天或夏天开始这个过程，这样木材在冬天使用之前有足够的时间变干。在进入森林之前，也要查阅当地和州的法律。

最好的木材来自低垂的树枝，这些树枝很容易折断，这意味着树枝已经死了。在地面上发现的枯木可能是一个很大的分数，但也可能充满了水。

确定你所在地区的硬木和坚果树，并关注这些树的木材。抓几块软木，因为它们更容易点燃，也是最好的火种。

如果你打算砍倒下的树(圆木)和 [砍自己的柴火](https://lifehacker.com/use-a-bungee-cord-to-chop-wood-more-efficiently-5404331) ，你应该非常熟悉如何使用电锯、楔子、大锤和劈槌。 [本文](http://www.naturalhandyman.com/iip/infwoodburning/infcuttingfirewood.html) 出自《自然杂工》将带你了解切割你的木材的基本步骤。

### 如何堆放和储存木柴

一旦你得到了你能找到的最好的木材，你需要用正确的方式堆放和储存它。

木材最好储存在不受天气影响的棚子里。如果放在室外，确保木头被防水布覆盖，并高出地面一点。

柴火应该交叉堆放在一堆的两端，然后纵向交叉放置(见上图)。这就形成了一个稳定且通风良好的木堆，你可以在冬天轻松取出。你可以在这里 找到 [其他一些有效的技巧。](http://www.woodheat.org/stacking-firewood.html)
