bOfvBY">如果你在重新油炸你自己的豆子——你应该这样做，因为整个鸭油的事情——在一个小的平底锅中用中火加热脂肪，然后将整个去皮的蒜瓣加入食用油中。把它们煮到两面都变成棕色，然后用木勺把它们捣成糊状。加入豆子，搅拌，煮到豆子开始有点散开。使用木勺、土豆捣碎器或浸泡搅拌机，把豆子搅成你想要的浓度，然后加入调味料。将豆子和大米混合在一起，放在你的砂锅里，在上面放一杯杰克干酪和切达干酪。</p><div
