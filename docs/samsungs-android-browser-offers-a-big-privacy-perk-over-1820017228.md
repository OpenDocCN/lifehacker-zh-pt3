# 与 Chrome 相比，三星的 Android 浏览器提供了一个很大的隐私保护

> 原文：<https://lifehacker.com/samsungs-android-browser-offers-a-big-privacy-perk-over-1820017228>

[三星](https://medium.com/samsung-internet-dev/samsung-internet-v6-2-now-stable-ab7f95ed8b4b) 正在将其 [智能手机浏览器](https://lifehacker.com/the-best-web-browser-for-android-5832026) 带到所有安卓手机上(假设你运行的是安卓 5.0 或更高版本)，它比谷歌自己的 [Chrome](https://lifehacker.com/lifehacker-pack-for-chrome-our-list-of-essential-chrom-880863393) 浏览器有一大优势。三星互联网应用程序具有内置广告跟踪拦截器的功能，可防止其他网站跟踪您在互联网上的活动。

这就是为什么如果你关心隐私，你可能会考虑将三星互联网作为你的主要 Android 浏览器。

### 跟踪拦截器的作用及其重要性

你经常被大公司和小网站在网上跟踪。这些信息中的大部分被用来制作有针对性的广告，而有些可能更具恶意。无论哪种方式，你可能不希望任何人悄悄跟踪你的活动，这就是这些拦截器的用处。追踪拦截器可以防止隐形追踪者看到您的行为，从而保护您的个人信息。差不多就是这样。

即便如此，在某些圈子里，它们已经成为一个有争议的话题。当苹果更新其 Safari 浏览器以限制在线追踪时，引发了来自数字广告和营销公司的反弹。在给 [The Verge](https://www.theverge.com/2017/9/14/16308138/apple-safari-11-advertiser-groups-cookie-tracking-letter) 的一份声明中，苹果反驳说广告追踪已经变得太普遍了，称之为对用户隐私的威胁。

在广告跟踪方面，谷歌也有不稳定的历史，可能是因为该公司从在线广告中赚了很多钱。在谷歌的 Chrome 浏览器中， [追踪无法完全关闭](https://www.techwalla.com/articles/how-to-turn-off-google-chrome-tracking) ，除非切换到 [微服模式](https://lifehacker.com/enable-your-extensions-in-incognito-mode-for-safer-mor-1792622065) 。最近的一项研究还发现，广告追踪器可以相当容易地绕过 Chrome 的扩展插件。

使用 Samsung Internet，默认情况下会在秘密模式下启用跟踪器阻止功能。如果你想让追踪者永远看不到你的活动，你也可以在应用程序的设置中打开它进行常规浏览。

### 三星互联网还能提供什么？

除了跟踪拦截，三星互联网支持各种广告拦截器，并包括一个特殊的菜单，在一个地方管理所有这些扩展。它还具有一个黑暗主题和减少光线的夜间模式，用于在床上浏览互联网，以及一个黑色背景和亮白色文本的高对比度模式，更容易阅读。

如果你在台式电脑上使用 Chrome，你可以将书签与三星互联网应用同步。最后，三星包括一些后端改进，以保持事情顺利运行。

这些都是相当次要的功能，但三星的跟踪拦截器可能足以赢得你的芳心——至少除非谷歌在未来的 Chrome 更新中推出类似的东西。
