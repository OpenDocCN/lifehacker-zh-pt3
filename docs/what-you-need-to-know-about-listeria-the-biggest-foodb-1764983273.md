# 你需要了解李斯特菌，最大的食源性妊娠威胁

李斯特菌现在出现在星巴克的三明治里。但这还不是全部:仅在过去的 60 天里，由威胁胎儿的微生物引起的召回还包括三文鱼、哈密瓜和两种品牌的奶酪。你如何避免在如此随机的地方出现的东西？

### **李斯特菌很罕见，但它可以杀死你的宝宝**

虽然食物中毒从来都不好玩，但这种由单核细胞增多性李斯特菌[](https://en.wikipedia.org/wiki/Listeria_monocytogenes)*引起的特殊类型的食物中毒，如果你怀孕了，就特别可怕。大约四分之一的孕妇感染李斯特菌 [导致流产、死产或新生儿死亡](http://www.innocua.net/web/download-2767/riog001004-0179.pdf) 。*

*这里唯一的亮点是李斯特菌感染很罕见。它导致美国每年 1600 例疾病和 260 例死亡，这并不算多。但是如果你真的被感染了，并且没有马上寻求帮助，你的机会就不大了。每当李斯特菌爆发时，27%受影响的人是孕妇。其他人通常是免疫系统脆弱的人:要么是老年人，要么是患有其他疾病，如艾滋病或癌症，这意味着他们的身体无法抵抗病菌。*

*[感染](http://www.foodsafety.gov/poisoning/causes/bacteriaviruses/listeria/) 的症状，称为，表现为发烧、呕吐，以及(如果感染了大脑)颈部僵硬和意识模糊。如果你在怀孕期间经历了这些，即使你认为可能是不太严重的疾病，如流感， ***立即寻求医疗护理*** 进行 [测试和抗生素治疗](http://www.acog.org/Resources-And-Publications/Committee-Opinions/Committee-on-Obstetric-Practice/Management-of-Pregnant-Women-With-Presumptive-Exposure-to-Listeria-monocytogenes) 。*

**李斯特菌*是 [广泛存在于我们周围的世界](http://www.fsis.usda.gov/OPPDE/NIS/Outreach/Docs/Listeria.htm) :它存在于水、泥土中，自然也存在于人和动物的肠道中。食物中的李斯特菌的棘手之处在于，与其他种类的细菌不同，它可以在冰箱温度下生长。例如，大肠杆菌不会在你的火鸡片放在冰箱里的时候长满它们。但是李斯特菌会。*

*也就是说，尽管李斯特菌是冰箱中的女王，但它很容易被高温杀死。如果你将食物烹饪到足以杀死*大肠杆菌*或*沙门氏菌*(肉类和家禽中更常见的细菌)的温度，任何*李斯特菌*都会消失。*

### *李斯特菌几乎可以在任何地方出现*

*如果我们只需要注意午餐肉和生牛奶奶酪，这将是一个可控的问题。但是最近李斯特菌出现在更多的地方。*

*以下是过去五年里李斯特菌大爆发 的来源:*

*   *袋装沙拉(2016 年)*
*   *软奶酪(2015)*
*   *冰淇淋(2015)*
*   *焦糖苹果(2014)*
*   *豆芽和豆腐(2014)*
*   *两个品牌的奶酪(2014 年)*
*   *更多奶酪(2013 年)*
*   *乳清干酪(2012 年)*
*   *哈密瓜(2011)*

*北卡罗来纳州立大学的食品安全专家本·查普曼 说*李斯特菌*不一定比过去出现在更多的地方，只是我们越来越善于寻找它。例如，星巴克的三明治，工厂在一些表面发现了李斯特菌，并宣布召回。没有人生病。尽管如此，还是有很多真正的疾病爆发，包括上面列表中的疾病。*

### ***熟食肉和软奶酪并没有自动被禁止，尽管***

*给孕妇 的标准建议是避免以下任何一种情况，除非她们非常火辣:*

*   *用生牛奶制成的软奶酪*
*   *熟食肉和热狗*
*   *冷藏馅饼*
*   *冷藏烟熏海鲜，比如熏鲑鱼*

*人们还被告知在怀孕期间要避免其他事情，但这些是与李斯特菌特别相关的规则。*

*但是这里我们有一点好消息:仍然有可能享受这些食物。例如，熟食肉可以被重新加热以消灭李斯特菌。查普曼说，当他的妻子怀孕时，他会为她点一份烤火腿和奶酪三明治，然后在家里用微波炉加热，并用温度计在三明治的多个位置检查温度。*

*即使你没有那么痴迷，你仍然可以用微波炉加热熟食直到它蒸熟。与几十年前相比，现在的午餐肉也不太可能携带李斯特菌:制造商现在将乳酸混合在 T2 和 T3 中，以减缓细菌的生长。*

*如果软奶酪是用巴氏杀菌奶制成的，也是可以的，根据法律 ，它们 [几乎都是。所以你根本不用放弃布里干酪。陈年(硬)奶酪不需要使用巴氏奶，但它们](http://www.realrawmilkfacts.com/raw-milk-regulations) [可能会滋生*李斯特菌*和](http://www.ncbi.nlm.nih.gov/pubmed/18724749) ，所以如果你格外小心，即使是硬奶酪，也要寻找成分标签上的巴氏奶。*

*不过，当心自制或小批量的软奶酪。李斯特菌感染的几次爆发都可以追溯到这些原因，足以说明拉丁美洲妇女面临更高的风险，仅仅因为她们更有可能吃新鲜的、手工制作的生牛奶。查普曼建议询问小规模食品生产商他们的李斯特菌管理计划。如果他们不知道你在说什么，不要吃他们的奶酪。*

### *除了做饭，你没什么可做的*

*“有人能做什么？不是很多，”查普曼说。我真的希望他能给我一套新的规则，让我更容易避免李斯特菌。没有这样的运气。他说，关于午餐肉和奶酪的标准建议仍然适用，如上所述，此外，你应该避免你怀疑不是以最严格和最清洁的方式制作的食物。*

*然而，在他们召回之前，我们可能会认为像蓝铃冰淇淋和沙布拉鹰嘴豆泥这样的大公司生产的食品是安全的。企业在召回事件后会进行清理，并试图赢回你的信任 ，但如果你仍然心存疑虑，我们可以理解。*

*虽然你不能解决工厂车间里发生的事情，但你可以采取一些措施，让你的厨房成为一个不那么欢迎李斯特菌的地方。这些都不是完整的解决方案，但每一点都有帮助:*

*   *用温度计确保你的 [冰箱尽可能的冷](http://lifehacker.com/what-temperature-should-you-keep-your-refrigerator-set-533534221)——大约 32 华氏度(*李斯特菌*越冷生长越慢)*
*   *不要把食物放在冰箱里太久。 [四天是它](http://lifehacker.com/follow-the-2-4-rule-to-store-and-enjoy-leftovers-safely-5896689) 吃得最多的时候。 [基于最近的研究](http://www.ncbi.nlm.nih.gov/pubmed/24290625) ，查普曼不建议将切开的哈密瓜保存超过一天。*
*   *根据推荐温度，烹饪你能做的任何食物(所以，不是冰淇淋)。对大多数物品来说是华氏 165 度。高温也会杀死 T4 李斯特菌和其他细菌。*

*大概就是这样。无论你做什么，李斯特菌的风险都会存在。幸运的是，这是一个很小的风险:在美国大约 400 万人中，每年大约有 500 名孕妇患李斯特菌病。你应该担心多少？这真的取决于你。*

*我上次怀孕的时候，有一次*李斯特菌*爆发 [和我最喜欢的鹰嘴豆泥](http://www.fda.gov/Safety/Recalls/ucm441863.htm) 品牌挂钩。他们说只是某些批次，但我整整九个月没吃鹰嘴豆泥。我也对袋装沙拉(和沙拉吧)和熟食肉说不，除非是在滚烫的三明治上。这是除了放弃酒精和接触运动。谢谢你，*李斯特菌*，让怀孕变得更加痛苦。*

*最终，你必须制定自己的规则。你无法预测李斯特菌下一次会在哪里出现，我也不会假装我有万全之策来避免下一次爆发。你最好的办法是尽你所能消除风险(比如粗糙的奶酪)，尽量不要对你力所不及的事情大惊小怪。*
