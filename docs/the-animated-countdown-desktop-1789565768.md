# 动画倒计时桌面

> 原文：<https://lifehacker.com/the-animated-countdown-desktop-1789565768>

[ravenclawwit](http://kinja.com/ravenclawwit) 的桌面太漂亮了，不能忽略。中心的时钟是一个令人敬畏的赛博朋克和*继续说话，没有人爆炸*挑战的结合——但它只是时间。这是所有设置的方式。

Watch

如果你想要一个类似的外观，首先你需要 Rainmeter， [，我们在](https://lifehacker.com/how-to-create-an-attractive-customized-desktop-hud-wit-5828789) 之前已经强调过了，这是一个给 Windows 桌面带来一点个性化和定制的简单方法。

一旦你熟悉了它，这里有你需要复制外观的其他部分:

*   [壁纸](http://i.imgur.com/ftBXlMD.jpg)
*   用于 Windows 的 [雨量计](https://www.rainmeter.net/) 系统调试和监控工具
*   [迪森皮](http://hiphopium.deviantart.com/art/Disen-610135382) 为时钟和温度( ravenclawwit 稍微调整了一下， fyi )
*   雨量计的 [英格玛套件](http://enigma.kaelri.com/) 用于系统监视器和 UI 元素
*   经典 Shell 实用程序，使任务栏透明并改变开始图标
*   正在播放信息的雨量计的 [明文](http://redsaph.deviantart.com/art/Cleartext-for-Rainmeter-v3-2-3-16-Sep-2016-519796161) 皮肤
*   [彩色喷泉](http://alatsombath.deviantart.com/art/Fountain-of-Colors-desktop-music-visualizer-518894563) 雨量计为可视化器的皮肤
*   [系统代码字体](https://www.behance.net/gallery/40835365/System-Code-Free-Font) 使所有文本匹配起来

没有太多的活动部件，但是当它们都组合在一起时，它们就有了一个真正戏剧性的、真正与众不同的外观。如果你喜欢它，或者如果你对如何制作这种或类似的外观有疑问，点击下面的链接查看桌面上的 ravenclawwit 的 Kinja 帖子，并询问关于设置的问题！
