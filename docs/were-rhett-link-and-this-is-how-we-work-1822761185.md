# 我们是瑞特&林克，这就是我们的工作方式

> 原文：<https://lifehacker.com/were-rhett-link-and-this-is-how-we-work-1822761185>

瑞德和林克，节目主持人和病毒视频的创作者如 [“不！查克·泰斯塔，"](https://www.youtube.com/watch?v=LJP1DphOWPs) 是 YouTube 上最著名的两个循规蹈矩的成年人。他们多年来一直在制作有趣的视频，现在他们运营着多个 YouTube 频道，每个频道都有数百万订户。我们问他们是如何工作的，他们是如何发展出如此令人眼花缭乱的职业生涯的。

#### 首先，告诉我们一些你的背景，你的第一次“大突破”，以及你是如何走到今天这一步的。

我们从一年级就认识了，从那以后我们一直在寻找关注的焦点。YouTube 为我们提供了一种接触全球观众的方式，超越了我们长期以来被迫忍受恶作剧的朋友/亲戚。我们已经做了十多年了，还没有真正的大突破，而是很多小突破，这些小突破让我们能够经营一家生产我们所做的一切的公司。

#### 带我们看看最近的一个工作日。

对我们来说，典型的一天是拍摄一集《美好的神话般的早晨》所需的一切(通常不止一集，以适应一周中的其他事情)。这要花去半天时间。我们会午休，但最近这种“休息”已经变成了我们边吃边讨论即将到来的部分或片段的会议。GMM 的拍摄结束后，我们直接去参加编剧会议，计划更多的演出。

在一天结束时，我们的助理会给我们发一封电子邮件，告诉我们所有未完成的任务/问题，这些任务/问题是我们无法在工作日内完成的。我们通常在哄孩子睡觉和和妻子一起看电视后，分而治之。是的，这是我们两个人的时间表(我们基本上做着同样的事情，这看起来很奇怪——的确如此——但这是我们唯一能发挥作用的方式)。

#### 你生活中离不开哪些应用程序、小工具或工具？

我们公司非常依赖谷歌文档。我们在任何事情上都与我们的作者合作，聊天和实时编辑(通常在我们拍摄节目时)的能力已经变得绝对不可或缺。因此，我们的手机和笔记本电脑是我们生活中不可或缺的一部分。我想这让我们变得不可思议的正常。

#### 你的工作空间是什么样的？

我们有立式办公桌，但我们通常坐在那里。我们唯一一次使用站立功能是当有人来到我们的办公室，我们想向他们展示桌子是如何工作的。我们办公室里有一个会议室，里面有舒适的椅子和沙发。我们也许应该邀请一位治疗师来，并为此使用沙发，但相反，我们只是在那里开会。我们在办公室的一个秘密阁楼里放了躺椅(这不是开玩笑)，但我们在两年内已经小睡过两次了。

#### 去生活帮，你最节省时间的捷径是什么？

几年前，我们有了一个好主意，我们应该在我们两个人之间分配我们的工作，而不是必须一起完成每一项任务。这看起来简单明了，但对我们来说是一个真正的突破。我们认为我们所花的时间比不这样做的时候少一半，因为如果一个人把东西从地面上拿下来，而另一个人对它进行微调和加工，创造过程实际上会变得更有效率。

#### 帮助你完成工作的人是谁，你如何依赖他们？

考虑到我们的工程背景，我们对拥有能最大化我们产量的系统非常感兴趣。我们团队中的每个人，从制片人到编剧到助理，都帮助建立和维护这些系统，没有他们的帮助，我们不可能制作出如此大量的内容。我们还曾经付给新奥尔良的一个女巫 50 美元，她把一只鸡爪放进汤里，用克里奥尔语说了些什么。她可能也帮了忙。

#### **怎么充值？当你想忘记工作的时候你会怎么做？**

我们充电不够，但我们确实依赖偶尔的按摩、感觉剥夺池之旅和家庭度假。
