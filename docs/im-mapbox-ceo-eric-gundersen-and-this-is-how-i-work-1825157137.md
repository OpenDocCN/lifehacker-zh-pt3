# 我是地图盒子首席执行官埃里克·贡德森，这是我的工作方式

> 原文：<https://lifehacker.com/im-mapbox-ceo-eric-gundersen-and-this-is-how-i-work-1825157137>

你可能不认识这个名字，但你已经从 CNN、Snapchat、Instacart 和天气频道看到了该公司的服务。(你也可以用它来 [做一个漂亮的锁屏](https://lifehacker.com/make-a-map-for-your-phones-lock-screen-1823832028) 。)埃里克·贡德森在为无国界医生组织等国际发展机构开发开源地图工具后，创建了 Mapbox。我们和他谈了他的移动工作空间——他最近用 iPad 取代了他的笔记本电脑——以及把他的整个团队放在 GitHub 上。



* * *

**姓名:**埃里克·贡德森
**地点:**加州三藩市
**当前职位:** CEO，Mapbox
**一个最能描述你工作方式的词:**移动
**当前移动设备:** iPhone X
**当前电脑:** iPad Pro，带备份 Macbook

#### 你的工作流程是怎样的？

我推动人们解决根本问题。我喜欢团队高度协作。最好的想法来自不同的团队，他们并肩工作来解决真正复杂的问题。

#### 带我们看看最近的一个工作日。

我在办公室开始审查我们物流和导航产品的数据更新，下午在一个软件会议上做了炉边谈话主题，晚上在办公室招待了 DC 副市长，就多样性和包容性进行了讨论。那是忙碌的一天。

#### 你生活中离不开哪些应用、小工具或工具？

[收纳盒纸](https://www.dropbox.com/paper) 改变了我的思考方式。在飞往明斯克的航班上，它在 iPad 上几乎是完美的，我和我的团队用它来实时指导我的一天，从会议到跟进和提醒。

#### 你的工作空间是什么样的？

我的桌子是充电的坞站。我大部分时间在办公室工作，在 sprint 中与产品团队一起工作，与客户或新员工开会，或者飞往 DC 或我们在世界各地的办公室做同样的事情。

#### 去生活帮最好的捷径是什么？

预订远回程的往返机票通常比单程机票便宜。

#### 带我们经历一个有趣的、不寻常的或挑剔的工作流程。

我们在 [GitHub](https://github.com/) 上运行我们的通信。不只是我们的工程师，而是所有人。所有这些都是，但是松散耦合的开放系统有一些特殊之处，它允许我们快速扩展。

#### 你如何跟踪你必须做的事情？

我有一个 Dropbox 纸质文档，我和我的团队在里面列出了我每天需要关注的最重要的事情——我把事情划掉——他们把它填满。

#### 你是如何充电或者休息的？

去年我旅行了 300，000 英里，但只错过了一个和孩子们在家的周末。我喜欢我们可以在旧金山的一月里度过一个海滩日。

#### 你还有什么想补充的，读者和粉丝可能会感兴趣的吗？

我们开始为世界银行、联合国、无国界医生组织等非政府组织提供咨询——我在阿富汗花了两个周期帮助美国国务院监督选举结果。在我们绘制地图的地方，地图是空白的，所以我们必须擅长实时绘制世界地图和可视化复杂数据。作为一家公司，这些经验是我们的 DNA，也是我们产品之所以伟大的核心。
