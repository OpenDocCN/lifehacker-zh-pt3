# 通过与数字伙伴配对，提高在家工作的效率

> 原文：<https://lifehacker.com/stay-focused-on-a-project-by-working-with-a-virtual-cow-1826182870>

在家工作很辛苦。不可避免的是，每当你坐下来做一个大项目时，你会想起你真的需要洗一堆衣服，或者打扫浴室，或者做任何你实际上不需要做的事情。

Watch

对付这种情况的一个简单方法是找一个你能与之共坐的负责任的伙伴。当其他人知道你现在应该做某事时，你更有可能去做，因为有人会知道你是否会吃零食而不是完成报告。

当你在办公室工作时，这很容易。当你在家工作时，就没那么多了。最近，一位在家工作的朋友向我推荐了一个解决这个问题的网络解决方案:。

它的工作方式非常简单:你注册这项服务，然后选择一个你想工作的时间段。特别是以 50 分钟为增量。因此，我可能会选择上午 8-8:50 或下午 6-6:50。

当您注册时，您将与另一个在同一时间段注册的人配对。到了工作时间，你通过网站和那个人开始视频聊天，告诉他们你在做什么，然后开始工作。

现在有人知道你应该在工作，并且知道如果你离开去做别的事情，你可能会强迫自己比那个人不在的时候更专注一点。

诚然，这听起来像是你的视频伴侣搞恶作剧的机会，但这项服务保持得相当干净。

除了让你在工作时保持注意力集中，它也是让你早上起床的一个不错的动力。如果有人在等着，你更有可能在早上 8 点起床工作
