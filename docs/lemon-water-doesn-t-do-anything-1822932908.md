# 柠檬水没有任何作用

> 原文：<https://lifehacker.com/lemon-water-doesn-t-do-anything-1822932908>

水:提神。柠檬:也很提神。将两者混合，你会得到一种味道清新的可饮用液体。*这就是你得到的全部*。不是解毒药剂，不是新陈代谢促进剂。让我重复一遍:当你把柠檬加入水中，你得到的只是柠檬水。

(如果你也加糖，你会得到柠檬水。)

如果你喜欢用一杯热柠檬水开始你的一天，我当然不会阻止你。但是让我们来看看这个仪式的好处。

**不是神奇的代谢补品**。《读者文摘》在 [中列举了](https://www.rd.com/health/diet-weight-loss/lemon-water-health-benefits/) 喝柠檬水的“12 个疯狂的好处”，引用了一项研究来支持柠檬中的一些化合物“防止体重增加”的观点。这项研究是在年轻的老鼠身上进行的(它们仍在生长，因此研究的是体重的增加),而不是早上喝一杯柠檬水，而是给它们一种多酚混合物，这是从柠檬中提取的化学物质，占它们饮食的 0.5%。如果你记得这些人不是人，他们没有喝柠檬水，他们也没有减肥，那么结果是有希望的。

**它是由液体**制成的。柠檬水的另一个减肥好处是，它可以填饱你的肚子，这样你就不会吃太多早餐了。这话没错，但是 [也适用于常规水](https://lifehacker.com/drink-a-glass-of-water-shortly-before-each-meal-to-eat-1751321390) 。

**很低热量**。《读者文摘》再次指出，加入一片柠檬的水比你在同样大小的玻璃杯中加入许多橙子的糖汁所含的热量要少。是的，确实如此。柠檬水几乎完全由水组成。

**它不会“排毒”你**。因为那不是东西。

**没有充满电解质**。“我没有直接喝，但是半杯柠檬汁含有大约 125 毫克的钾，相比之下，一根香蕉含有大约 211 毫克的钾，”迪娜·加奇曼 [在预防](https://www.prevention.com/food/2-cups-of-lemon-water-every-day) 杂志上写道。好吧，但是如果你不喝半杯的话，数学对我们有什么好处？这里有 [一个柠檬片中果汁量的数字](http://nutritiondata.self.com/facts/fruits-and-fruit-juices/1938/2) 。看看这些零！

它不会帮助你消化任何东西。胃酸是酸性的，柠檬也是酸性的，所以柠檬肯定能帮助你消化食物，对吗？首先，胃酸只涉及消化的一个小方面，所以增加胃酸对你的身体没有多大好处。但事实证明 [你并不能真正改变你胃里的酸度](https://courses.washington.edu/conj/bess/acid/acidreg.html) 。如果你吃碱性的东西，你的胃细胞会喷出更多的酸；如果你喝柠檬汁，他们会忍住。

**它不提供果胶或任何其他纤维**。除非你正在*咀嚼和吃*柠檬片，在这种情况下，你会得到一点点。

**也不要对维生素 C 过于兴奋**。一个楔子能提供你每天所需维生素 c 的 4%。如果你喜欢柠檬水，喝一整个柠檬的话，你就能达到 36 %,这是维生素 c 的一个很好的来源。但是你也从其他食物中获取维生素，不是吗？我相信，如果你有患坏血病的危险，你会从海盗船上跳下来，在去看医生的路上买一个多汁的橙子(你每天所需量的 138%)。如果你只是希望预防感冒， [维生素 C 不能做到这一点，](https://lifehacker.com/immune-boosting-supplements-won-t-protect-you-from-ba-1804490868) ，但如果你服用大剂量，它*可能*缩短感冒。比一杯柠檬水里的多得多。

总之，柠檬水是由液体制成的，含有很少的热量，可以说味道很好。这就是它的优势。每天早上一杯柠檬水可能会帮助你戒掉喝咖啡的习惯(尽管是为什么),或者帮助你感觉你的一天有了一个健康的开始，因为你凝视着窗外，想象着你稍后将尝试的瑜伽动作。好吧。随便啦。但它能做的也就这些了。
