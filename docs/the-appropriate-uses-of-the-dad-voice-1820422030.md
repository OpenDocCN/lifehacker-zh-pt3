# 何时使用“爸爸的声音”

> 原文：<https://lifehacker.com/the-appropriate-uses-of-the-dad-voice-1820422030>

为人父母的悖论是，你可以执行所有的功能，但仍然觉得自己是个冒名顶替者。换尿布、玩躲猫猫、骑在别人背上——这些看起来都像是演戏。当你成为一个真正的父亲的时候到了。有一天，你会和你的孩子在一起，张开你的嘴，发出爸爸的声音。像守护神一样，它会在空中疾驰，将目标定格在它蹒跚学步的小脚印上。



爸爸的声音很大，但没有喊叫，是指导性的，但不是威吓性的。使用爸爸声音时，你不是一个 [尖叫狂人](https://youtu.be/aoMmbUmKN0E?t=142) 。你不是 [精神错乱](https://www.adforum.com/creative-work/ad/player/6067/please-be-quiet-tax-day-20/hr-block) 。相反，你坚持不懈，不屈不挠——也许设定了寻找 [各种房子](https://www.youtube.com/watch?v=rRuYrOKYf3c) 的期望。

用你的振膜提高分贝，像扩音器一样广播。删掉你的音节。不要变得罗嗦或机智——关键不在于你说什么，而在于你怎么说。此外，现在不是胡闹的时候。

什么时候使用爸爸的声音？这里有几个场景。

### **即兴的蹒跚学步的摩西坑**

当太多的孩子聚集在太小的空间里，可怕的事情就会发生。信息素或发育不全的额叶或有光泽的甜甜圈会产生一个能量反馈环。有尖叫，尖叫和嬉戏。偶尔，那匹马会冲出畜栏，冲向它所经过的一切。

一群群扭动着的蹒跚学步的孩子是无法解释的。甜言蜜语地说要好好打球是无法让他们平静下来的。他们只知道以眼泪结束的狂暴的毁灭。只有一种方法可以产生一个足够长的定格画面来改变他们的能量:父亲的声音。

### **存在易碎物品时**

有多少次我被一个“要做的事情”日历欺骗了？太多次了。为了打发醒着的周末时光，我绝望地求助于互联网，它向我保证，当地艺术博物馆的活动有很多“儿童活动”。我们来到一个场景，破碎的蜡笔和影印的彩色书籍页面，不稳定地靠近百万美元的花瓶。突然间我努力挣来的每一块钱都岌岌可危。大学基金、退休基金、保险免赔额基金——所有这些都可以在一屋子无价之宝中被一次热情的大发脾气而化为乌有。

这就需要先发制人的爸爸声音。在任何鲁莽行为出现之前很久，父亲的声音可能会以一种修改过的方式被用来描述最佳行为的重要性以及任何鲁莽行为发生时惩罚的严厉程度。先发制人的爸爸的声音是致命的安静，并伴随着一眨不眨的凝视。如果你的孩子像梦游一样缓慢地走过精致的物品，就能很好地听到它，比如斯德哥尔摩综合症俘虏。

### **行人和自行车安全**

一旦你的孩子学会走路，他们就会像角马穿越鳄鱼出没的河流一样被吸引到街上。对孩子们来说，汽车是闪亮、轻便和神奇的。它们沿着宽阔光滑的轨道向下延伸，非常适合玩耍。所以，当你的学龄前儿童冲向路边时，不是友好提醒走路的时候。是时候大声喊“停！”用父亲的声音。

在司机身上用爸爸的声音也没问题。去年春天，我和女儿骑自行车去上学。突然，一辆奥迪车中的某个混蛋飞快地拐了个弯，以至于车的尾部旋转了起来。当他沿着街区向我们开枪时，我张开双臂，爸爸喊道:“嘿！慢点！”原来我在做别人爸爸的工作。司机是一个面容姣好、头发稀疏的少年。该死的孩子。

### **与危险动物互动**

一天下午，三只比格犬大小的浣熊走进了我的前院。我大喊了一声，“嘿，浣熊！滚出去！”他们只是看着我。我走下门廊，重复了一遍。当他们慢吞吞地穿过街道时，其中一个人回头看了我一眼，脸上带着厌恶的表情，就像一个开着奥迪的少年。

无论是一只在附近游荡的陌生狗，还是一只在远足小径上的郊狼，你和你的孩子总有一天会遇到动物。是的，你在办公室工作，在空闲时间玩电子游戏，但是一只癞皮狗不知道这些。对他来说，你是顶级掠食者。表现得像点！即使当你面对更危险的事情。我从没在树林里见过黑熊，但是 [他们说](http://www.bearsmart.com/play/bear-encounters/) 你应该让自己看起来很大，然后对着它吼。不要这样对待灰熊！不像狮子座，你会死，拿不到奥斯卡。不确定哪个是哪个？ [教化自己](https://www.nps.gov/articles/bear-identification.htm) 。

### **在医疗紧急情况下**

如果你猜错了熊，你将再次使用你爸爸的声音！

严肃地说，当你接受 CPR 训练时，他们告诉你的一件事是 [大声清晰地说话](http://www.redcross.org/take-a-class/first-aid/perfoming-first-aid/first-aid-steps) 。你应该指着一个特定的人说，“打 911！”或者“给我拿个急救箱来！”或者“十毫克肾上腺素，马上！”

当有人在商场癫痫发作或在餐馆心脏病发作时，旁观者的大脑会像瘾君子一样做出反应，说“哇？”如果你知道如何帮助处于困境中的人，你也必须告诉旁观者他们可以如何帮助你。这需要用父亲的声音发出简单的命令。别担心。没人会认为你是个傲慢的混蛋。他们会松一口气，因为爸爸似乎是负责人。
