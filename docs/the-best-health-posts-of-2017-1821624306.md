# 2017 年最佳健康岗位

> 原文：<https://lifehacker.com/the-best-health-posts-of-2017-1821624306>

今年我们经历了很多，从政治对我们精神健康的影响，到健康供应商试图向我们出售我们不需要的东西。以下是我们 2017 年最佳健康指南、解释者和流言终结者。



### [卖‘安心’的婴儿可穿戴设备只是劣质医疗器械](https://vitals.lifehacker.com/baby-wearables-selling-peace-of-mind-are-just-bad-med-1791379503#_ga=2.255763208.1482156858.1514299010-1382985742.1510266239)

本月早些时候，我们在消费电子展 上看到了一批与婴儿相关的智能新设备首次亮相 [。虽然有些可能会被证明是有用的——我会为像](http://lifehacker.com/the-tech-trends-from-ces-2017-that-will-actually-matter-1790903186) [威洛](https://www.willowpump.com/) 这样的吸奶器而杀人——但其他人更有可能耗尽父母的钱包，而不会提供任何真正有价值的东西。

### [如何知道自己是否准备好开始马拉松训练](https://vitals.lifehacker.com/how-to-know-if-you-re-ready-to-start-training-for-a-mar-1791124012#_ga=2.255763208.1482156858.1514299010-1382985742.1510266239)

没有什么比一个雄心勃勃的目标更能集中你的训练了，跑马拉松绝对符合这个要求。许多普通人已经完成了 26.2 英里的比赛，但这需要时间、计划，当然还有适当的健康水平。以下是如何知道马拉松对你来说是否是一个现实的目标。

### [如何熬过高额免赔额的健康保险计划](https://vitals.lifehacker.com/how-to-survive-a-high-deductible-health-insurance-plan-1792436777#_ga=2.108964866.1482156858.1514299010-1382985742.1510266239)

如果你陷入了高额免赔额的健康保险计划，你并不孤单:[39%的美国人](https://www.cdc.gov/nchs/data/nhis/earlyrelease/insur201702.pdf)(65 岁以下)有一个至少 1300 美元免赔额的计划。多年来，我有一个家庭免赔额高达 8000 美元(T4)的计划。不过，你还没完全完蛋。如果你知道一些诀窍，你仍然可以在不倾家荡产的情况下得到照顾。

### [一个月每天做俯卧撑学到的东西](https://vitals.lifehacker.com/what-i-learned-doing-push-ups-every-day-for-a-month-1791921298#_ga=2.219110106.251250443.1514321847-1313785359.1499701416)

### [每年吓唬人的防晒霜评级都是扯淡](https://vitals.lifehacker.com/the-sunscreen-ratings-that-scare-people-every-year-are-1795824253#_ga=2.217016470.1482156858.1514299010-1382985742.1510266239)

### [这里是对付面筋的](https://vitals.lifehacker.com/here-s-the-deal-with-gluten-1796820761#_ga=2.219110106.251250443.1514321847-1313785359.1499701416)

### [我在格温妮丝的 Goop 峰会](https://vitals.lifehacker.com/i-went-on-a-quest-for-legit-health-tips-at-gwyneth-s-go-1796136053#_ga=2.176475045.1482156858.1514299010-1382985742.1510266239) 上寻求合法的健康建议

### [关于大麻在怀孕期间的安全性，我们做了什么和不知道什么](https://vitals.lifehacker.com/what-we-do-and-don-t-know-about-the-safety-of-marijuana-1796656907#_ga=2.176475045.1482156858.1514299010-1382985742.1510266239)

大麻比许多其他药物(或者，取决于你问的是谁，酒精)更健康，它可以缓解怀孕期间容易出现的恶心等症状。但是我们积累的证据表明，如果你怀孕了，大麻可能不是一个好主意。特别是，THC 成分可能会影响婴儿的大脑，直到他们长大后你才会注意到这一点。
