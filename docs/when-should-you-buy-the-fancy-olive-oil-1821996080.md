# 你应该什么时候购买花式橄榄油？

> 原文：<https://lifehacker.com/when-should-you-buy-the-fancy-olive-oil-1821996080>

当谈到令人困惑的标签和价位时，我发现橄榄油仅次于葡萄酒。一旦你掌握了整个贞操的事情，还有地区的问题要处理，很难知道你“应该”和哪个人一起做饭。



最简单的形式，“橄榄油”是从橄榄树的果实中提取的油，可以是“精炼的”或“未精炼的”“精炼”听起来更花哨，根据 [和](https://www.thekitchn.com/whats-the-difference-between-olive-oil-and-extra-virgin-olive-oil-word-of-mouth-218767) 的说法，尽管它确实能去除油中的“瑕疵”，但它也能去除颜色和味道。“未精炼的”橄榄油被进一步分为两类——“初榨”和“特级初榨”不过，通常在购买橄榄油时，你最有可能看到这三种:

*   **“普通”橄榄油:**瓶子上只会写着“橄榄油”或“纯橄榄油”，试图超越它的地位。这通常是初榨橄榄油和精炼橄榄油的混合物，这意味着其中至少有一部分经过加热和/或化学处理。它有一种非常中性的味道，可以用于各种烹饪。

*   **特级初榨橄榄油:**“EVOO”，如果你是来自 aughts 的美食博主。这是好的，未经提炼的东西，有最多的颜色，味道和抗氧化剂。你可以用它来烹饪——只要知道它有相当低的发烟点(325 到 375 华氏度),而且 [高温烹饪会抹杀油的味道](http://www.nytimes.com/2010/11/17/dining/17curious.html) ，所以把这些花哨的东西留给和(可能)一些轻炒。

*   **“轻”油:**这种油不缺热量，缺的是风味和色泽。它是精制的，并且具有更高的 465°F 的烟点，所以可以放心地用它来煎炸。

不幸的是，有些橄榄油是 [瓶欺](https://lifehacker.com/the-most-and-least-fake-extra-virgin-olive-oil-brands-1460894373#_ga=2.202965459.318781610.1515430986-1667049540.1510952795) 。这很有道理，因为奢侈让我们所有人都成了骗子。(我总觉得自己在那些花里胡哨的橄榄油店里被人主动骗了。幸运的是，我们有一些方法来确保你在购买昂贵的东西时不会被骗，包括 [检查批准印章](https://lifehacker.com/check-for-a-seal-of-approval-before-buying-expensive-ol-1352095992#_ga=2.122948041.318781610.1515430986-1667049540.1510952795) 和 [寻找品种](https://lifehacker.com/look-for-the-cultivar-when-shopping-for-olive-oil-849999531) 。

一旦你有了一瓶好油，把它留着品尝，把煎炸留给那些不太好的瓶子。特级初榨橄榄油的味道因地区而异，可以是青草味、胡椒味或甜味，我不希望你的嘴错过这段旅程。
