# 再生木材工作空间

> 原文：<https://lifehacker.com/the-reclaimed-wood-workspace-1729341434>

今天的特色工作区的墙壁排列着回收的木材，Flickr 用户 getthef.ckup 收集这些木材，干燥并打磨，油漆，涂油，并再次打磨。这些墙本身就像艺术一样。



除了用回收的木材进行装饰，他还重新利用了一盏以前在风暴中毁坏的旧街灯，并用它将显示器和扬声器的电线穿过桌子。这是一个没有线缆、整洁、引人注目的工作空间。

如果你有自己的工作空间可以炫耀，请将它添加到我们的 [Lifehacker 工作空间展示和讲述](http://www.flickr.com/groups/lifehacker-workspace-showandtell/pool/) Flickr 池中，与我们分享。确保您包含的任何照片至少为 800x450。将它们保持在 16:9 也有帮助！包括一些文字，关于你使用的材料，你是如何想出这个设计的，以及任何其他相关的细节。如果你巧妙的组织和良好的设计感吸引了我们的眼球，你可能就是下一个特色工作区。
