# 使用这款 Chrome 扩展，找出您的密码是否在数据泄露中泄露

> 原文:[https://life hacker . com/know-if-your-password-has-leaved-in-a-data-breach-1826373137](https://lifehacker.com/know-if-your-password-has-been-leaked-in-a-data-breach-1826373137)

在这一点上，似乎每周都有新的数据泄露，用户的用户名、密码和其他个人信息暴露给黑客。跟踪他们可能是一项艰巨的任务。

Watch

Passprotect 是一个 Chrome 扩展，它使用来自网站的数据，让你知道你正在使用的密码是否在任何数据泄露中被发现。如果有，那么是时候改变它了。

要获得它， [点击谷歌 Chrome 商店](https://chrome.google.com/webstore/detail/passprotect/cpimldclklpfifolmdnicjnfbjdepjnf) 的链接，然后在弹出的页面右上角点击“添加到 Chrome”。

首先避免此问题的一个好方法是使用密码管理器(如 1Password)来创建和存储安全密码。密码是随机生成的，很难破解，这使得它们最终比你自己想出来的任何东西都要安全一些。1Password 已经将其密码与 Pwned 数据库进行了比较。

总的来说，时不时地在上检查一下自己是个好主意。该网站只是要求您输入您的电子邮件地址，然后会让您知道该地址涉及哪些数据泄露以及黑客窃取了哪些信息。在我的案例中，我参与了 7 起，其中大部分我知道，但有几起我不知道。像大多数事情一样，安全总比后悔好。
