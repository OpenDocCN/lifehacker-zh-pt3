# 用 20 世纪 80 年代的经典广告替换脸书的广告

> 原文：<https://lifehacker.com/replace-facebook-ads-with-vintage-ones-from-the-1980s-1796642059>

我和你一样厌倦了看脸书的烂广告。但是我没有运行标准的广告拦截器，而是用更有趣的东西来代替广告。特别是 80 年代的复古广告。激进。



[社交广告，复古版](https://vintage-ads-are.cool/) 是谷歌 Chrome 浏览器的一个扩展，它用怀旧的照片取代了脸书的广告，这些照片展示了你所喜爱的一切，现在都很复古。想想《忍者神龟》、卡西欧合成器和 Coleco 街机橱柜——所有这些都取代了那些传统的、无聊的广告。

当浏览我的脸书 feed 时，我宁愿看到任天堂的 Power Glove 的广告，这是视频游戏配件的圣杯，而不是我曾经谷歌过的某个 MP3 播放器的折扣。当然，这并不是说我真的能买下拳王手套，但这就是重点。我再也不用避开那些攻击我钱包的广告了。
