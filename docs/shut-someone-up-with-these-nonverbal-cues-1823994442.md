# 用这些非语言暗示让某人闭嘴

> 原文：<https://lifehacker.com/shut-someone-up-with-these-nonverbal-cues-1823994442>

有些人喜欢听自己滔滔不绝地讲话，以至于你根本插不上嘴。嗯，部分问题在于你在等他们把海螺递给你——而他们不会这么做。不过，你不必直截了当地打断他们；你可以简单地使用你的肢体语言。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-g7kaTvYnjIQ&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-g7kaTvYnjIQ&start=0) 

通过最基本的操作，你可以很容易地发出你想插入你的想法的信息。凡妮莎·范·爱德华兹是《T0】[](https://www.amazon.com/gp/product/0399564489/?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/shut-someone-up-with-these-nonverbal-cues-1823994442&asc_source=&tag=kinjalifehackerlink-20)*[人才学](https://www.scienceofpeople.com/) 的创始人，她在上面的视频中给出了五个最佳动作。它们如下:*

1.  *鱼:你张大嘴巴(像鱼一样)表示你现在准备好说话了。* 
2.  *书签:你伸出手滑入他们所说的内容，并添加一些东西。与鱼很相配。* 
3.  *学生:你举手了，就像在教室里一样。它也微妙地看起来像“停止”的通用标志* 
4.  *触摸:你轻轻地触摸他们的肩膀或手臂。这通常会阻止他们，并给你一个小机会来利用你的优势。这可能是为了增加一些东西，或者仅仅是说再见，逃避谈话。(“和你聊天真开心！”)* 
5.  *老师:你举起你的手指，就像老师要求教室里保持安静一样。它看起来类似于“嘘”的手势。这是最具侵略性的一招，只有在其他方法都失败的时候才应该使用。*

*从鱼开始，然后一步步往下，直到它们闭嘴。如果另一个人有任何社会意识，他们应该学会其中至少一个。或者你可以试试我最喜欢的(也是相当粗鲁的)策略——问他们的节目是否有中场休息。如果他们不停下来，很明显他们不尊重你或你的时间，你应该停止和他们说话。去找一个想和你交谈的人，而不是一个只会找听众的多嘴多舌的人。*
