# 问那个咸咸的女服务员:我超级讨厌的服务员正在毁掉一家我喜欢的餐厅

> 原文：<https://thetakeout.com/my-super-annoying-server-is-ruining-a-restaurant-i-like-1828060514>

> 亲爱的 Salty:我们小区有一家比较新的 brewpub，我和妻子去过几次。这是一个闲逛的好地方，他们有有趣又好喝的啤酒，风景也很棒。我们遇到的问题是，有一个服务器，我们总是与谁非常不愉快结束。
> 
> 我认为我们是好人。我们都在食品服务行业工作过，我们都是 25%的小费支付者——不管是什么小费。我们很友好，很好，没有任何不合理的地方，但是这个女人很糟糕。我们在谈论:取笑我们的选择，奇怪地咄咄逼人，然后消失，张着嘴咀嚼，靠在你的耳朵上，感觉像永远一样，让桌子上的每个人都不舒服——但我不能指着某样东西说“请不要这样。”
> 
> 每次我们去酿酒厂，被她缠住时，我们整个桌子的人都会在晚上结束时离开，然后说“哇，这个人真可怕。”
> 
> 我们已经开始坐在不同的地方，但最后一次她注意到并跟踪我们(这意味着我们不能坐在我们最喜欢的地方，那是可以看到美丽天际线的顶层露台)。我们能做些什么吗？她毁了(我不认为我疯了，她没那么坏)在一个很酷的地方的美好时光，这太糟糕了。我们有什么选择？
> 
> 说真的，她是最糟糕的

亲爱的说真的，

我能知道这位女士的号码吗？她需要一次严厉严厉的谈话。她怎么敢给我们这些努力工作，拼命干活的服务员一个坏名声？她为什么一边工作一边嚼东西？她在嚼什么？！最好是口香糖，高尔，但还是不行。我会告诉她我的想法，我肯定会的。



现在看看我，把我的羽毛弄得乱七八糟。

但是至于你能做什么，这是个棘手的问题。我将提供四个选项，按照我尝试的顺序:

1.  从酒吧点菜。我不确定这是否是你所说的特别的 brewpub 的一个选项(虽然哪个 brewpub *没有*酒吧？)但有些会让你向调酒师点食物和饮料。当然，这意味着你必须站起来，但这可以让你省去与嚼嚼麦考姆珀森互动的麻烦。
2.  对她说点什么。你提到你无法指出她做的许多令人讨厌的具体行为……然后你列出了其中的四个。嘲笑你的订单尤其出格，所以让我们以此为例。如果你再次被她缠住，而她嘲笑你的汉堡和麦芽啤酒，闭嘴。“这是我点的菜，如果你不介意的话，我真的可以不加任何评论。”同样的，这种冲动也是如此。“我想我们都很熟悉菜单，所以没必要解释。谢谢。”
3.  把它带给经理。除了拥有像鞋子里的石头一样的个性之外，这位优秀的员工似乎也不擅长自己的工作。她让你不舒服，擅离职守，以至于——你的话不是我的，甜心——*毁了你的晚餐。你会因为这个服务器而减少去 brewpub 的可能性吗？那么经理需要知道。具体描述她的行为，希望经理要么和她一起改正，要么解雇她。经理的工作是确保客人有一个愉快的体验，我敢打赌你不是唯一一个注意到这种不良行为的人。*
4.  找一家不同的餐馆。让我们颠倒一下情况:您拥有世界上最好的服务器。她迷人又风趣，让每个人都感到受欢迎。但是食物很难吃。就像，皮鞋坏了。你会回到餐馆吗？

我知道你可能害怕说出你的抱怨。但是这个服务器的行为太过分了，表明她没有有效地完成自己的工作。她的工作是让客人感到舒适，给他们一个有趣的体验，并确保他们喜欢他们的订单。她什么都没做，你不必为此感到难过。
