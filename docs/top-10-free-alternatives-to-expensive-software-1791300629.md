# 昂贵软件的 10 大免费替代品

> 原文:[https://life hacker . com/top-10-免费替代昂贵软件-1791300629](https://lifehacker.com/top-10-free-alternatives-to-expensive-software-1791300629)

除非你有一家公司为你购买许可证，否则像 Photoshop、Pro Tools、Maya 甚至 Windows 这样的工具可能会严重侵蚀你的钱包。这里有一些比通常昂贵的应用程序更实惠的选择，可以帮助你停止烦恼，完成更多工作。

### 10. [GIMP](https://www.gimp.org/) (免费，用于 Windows、macOS 和 Linux)

第一个有点明显。如果你没有钱订阅 Adobe Creative Cloud，或者从未为自己购买过 Photoshop 的副本，那么 [GNU 图像处理程序](https://www.gimp.org/) (简称 GIMP)就是为你准备的。它功能丰富，功能强大( [，使用 GIMPShop 插件](http://lifehacker.com/get-the-feel-of-photoshop-in-gimp-with-gimpshop-272600) 更是如此)，而且只需一点点工作， [就可以在你需要的时候像 Photoshop](https://lifehacker.com/how-to-make-the-gimp-work-more-like-photoshop-1551318983) 一样工作。

如果你在开始使用它时有一点困难，那也没关系——它可能看起来有点令人生畏，但是 [这个充满快捷方式和提示的备忘单](https://lifehacker.com/learn-all-the-gimp-keyboard-shortcuts-with-this-cheat-s-1446463913) 可以帮助你导航它并快速上手。毕竟是 [你最喜欢的 PhotoShop 备选](http://lifehacker.com/most-popular-photoshop-alternative-gimp-1484836993) 是有原因的。如果你想要更多的选择，Windows 用户可以尝试(免费)，Mac 用户可以查看 [备受好评的](http://lifehacker.com/jumping-ship-from-photoshop-to-pixelmator-a-switchers-1689515606)[pixel mator](http://www.pixelmator.com/)(30 美元。)

### 9. [Libre Office](http://www.libreoffice.org/) (免费，适用于 Windows、macOS 和 Linux)

微软的办公套件是全球办公室的生产力标准，但当你在家里使用电脑时，如果你不想花钱购买许可证，或者无法通过其他方式使用它，你仍然需要一些东西来处理——特别是如果你必须处理别人发给你的 Word 文档、Excel 电子表格或 PowerPoint 演示文稿。进入 [Libre Office](http://www.libreoffice.org/) ，我们的 [长期喜爱的替代方案](https://lifehacker.com/battle-of-the-office-suites-microsoft-office-and-libre-1147940828) 如果你需要一个本地的，非基于网络的办公套件。

Libre Office 近年来也取得了长足的进步。过去看起来非常像刻板的“由委员会设计”的实用工具套件 [现在已经成为自己的](https://lifehacker.com/libreoffice-updates-doesnt-look-like-crap-anymore-1682558952) ，作为 Office 的强大而优雅的竞争对手，它仍然可以打开 Office 文档，编辑它们，并保存它们供 Office 用户无缝使用。当然，它是完全免费的。

### 8. [Inkscape](https://inkscape.org/en/) (免费，适用于 Windows、macOS 和 Linux)

如果你使用矢量图形、构建徽标，甚至为印刷或其他出版物设置文档格式，Adobe Illustrator 是一个必备工具，但同样，如果你不打算购买 Creative Cloud 订阅服务，[【Inkscape】](https://inkscape.org/en/)一直是 [长期以来最受欢迎的](https://lifehacker.com/manipulate-vector-images-with-open-source-inkscape-5115354) ，因为，你可以在 Illustrator 中完成几乎所有的事情，而没有大多数人实际上懒得使用的花哨功能。另外，它是免费的、开源的、跨平台的。

### 7.(免费，适用于 Windows、macOS 和 Linux)

如果你对 3D 建模、3D 打印或 3D 动画感兴趣，那么 [Blender](https://www.blender.org/) 是你的首选工具，至少是在这个领域入门的时候。当然，Maya 和其他类似的工具被用于动画和建模工作室，但如果你想开始，甚至是 [为 3D 打印](https://lifehacker.com/the-best-free-resources-for-getting-started-with-3d-pri-1774721138) 建模你自己的对象，Blender 是一条路要走。

不过，一开始理解起来可能有点棘手，我们的 3D 建模夜校 可以帮助你入门。如果这对你来说还不够， [这本来自 Wikibooks](http://en.wikibooks.org/wiki/Blender_3D:_Noob_to_Pro#Beginner_Tutorials) 的免费电子书是另一个帮助你学习 Blender 的好工具。

### 6. [Avira](http://www.avira.com/) (免费，针对 Windows)和 [Sophos](https://www.sophos.com/en-us/products/free-tools/sophos-antivirus-for-mac-home-edition.aspx) (免费，针对 macOS)

毫无疑问，当你点击什么，下载什么，访问什么网站时，良好的浏览卫生和聪明的感觉是无可替代的。然而，在你的电脑上同时安装 [一个好的反病毒和/或组合反恶意软件工具](https://lifehacker.com/the-difference-between-antivirus-and-anti-malware-and-1176942277) 也无妨，尤其是在那些可能不像你一样有网络智慧的人的电脑上。

出于这个原因， [Avira](http://www.avira.com/) 是 [我们为 Windows](http://lifehacker.com/the-best-antivirus-app-for-windows-5865356) 挑选的最好的，而 [Sophos](https://www.sophos.com/en-us/products/free-tools/sophos-antivirus-for-mac-home-edition.aspx) 是 [我们为 macOS 挑选的最好的，当谈到防病毒时，](http://lifehacker.com/the-best-antivirus-app-for-mac-488021445) [MalwareBytes](https://www.malwarebytes.com/) 。如果你喜欢别的东西？ [我们有更多的选择](https://lifehacker.com/five-best-desktop-antivirus-applications-1607557993) 。总之，绝对没有理由支付昂贵的反病毒或反恶意软件订阅费，免费工具——尽管最近出现了一些肮脏的做法——做同样的工作，而没有你可能在任何地方的办公室计算机上认识的大牌带来的臃肿和低劣。

前身为 Linux 多媒体工作室的[【LMMS】](https://lmms.io/)，是苹果 Garageband，甚至更贵的 Pro Tools 等行业标准应用的强大免费替代品。如果你正在寻找专业的工作，它可能不会取代它，但是如果你是一个在家的业余爱好者， [这个免费的跨平台工具值得一试](https://lifehacker.com/lmms-turns-your-desktop-into-a-music-sequencing-monster-5216103) 。

### 4. [VirtualBox](https://www.virtualbox.org/) (免费，用于 Windows、macOS 和 Linux)

虚拟化不仅仅是极客们玩的东西——它是测试新工具和软件的好方法，并且 [确保它不会对你的计算机](https://lifehacker.com/how-to-safely-test-software-without-messing-up-your-sys-1680608496) 造成严重破坏。 [Virtualbox](https://www.virtualbox.org/) ，免费且跨平台，严格来说可能不是最好或最有效的工具，但它的功能和成本效益比是无与伦比的。它超级强大，而且完全免费。

如果你曾经想过“伙计，我希望我有一台计算机来测试这台不是我的计算机”，我们的[Virtualbox](https://lifehacker.com/the-beginners-guide-to-creating-virtual-machines-with-v-5204434)入门指南可以帮助你了解基本知识，如果这些对你来说都是老掉牙的，我们的 [高级用户指南](http://lifehacker.com/the-power-users-guide-to-better-virtual-machines-in-vir-1569943402) 可以帮助你。如果你想知道在 Fusion 或 Parallels 等工具上使用 Virtualbox 会错过什么，请在这里查看 [我们的虚拟化对决](http://lifehacker.com/run-windows-on-your-mac-virtualbox-vs-vmware-fusion-vs-1750474360) 。

很难找到一个既好又免费的视频编辑工具，尤其是与 iMovie 等常见的家用工具或 Adobe Premiere 等专业级工具相比。然而，如果你正在寻找一个替代方案， [达芬奇解决](https://www.blackmagicdesign.com/products/davinciresolve) ，它最初是一个复杂的(和昂贵的)颜色分级工具， [几年前获得了视频编辑功能](https://lifehacker.com/build-your-own-adobe-creative-suite-with-free-and-cheap-5976725) 。从那以后，它为自己赢得了一个全面、可靠的替代视频编辑器的名字，虽然对初学者来说有点难，但一旦你掌握了它的窍门，它就足够强大，可以取代其他更昂贵的工具。如果你想付钱的话，有一个付费版本，这里有免费和付费版本的比较。

当然，Linux 理应在榜单上占有一席之地。当然，你使用哪个发行版 [完全取决于你](http://lifehacker.com/five-best-linux-distributions-5904069)—[你选择哪个桌面环境](http://lifehacker.com/five-best-linux-desktop-environments-1648022755)——但是如果你从 macOS 或 Windows 转移到 Linux，或者为了简单起见，或者因为你不想支付新的许可证，或者你只想让一台旧电脑起死回生， [Ubuntu 和 Linux Mint 是你掌握窍门的两个最佳选择](https://lifehacker.com/ubuntu-vs-mint-which-linux-distro-is-better-for-begin-5993297) 。

如果您想深入了解，Arch Linux 是这些部分的最爱，但它不是超级新手友好的。尽管如此， [我们有一个很好的指南来设置](http://lifehacker.com/build-a-killer-customized-arch-linux-installation-and-5680453)——并且在这个过程中学习了很多关于 linux 的知识。此外，请务必查看我们的[Linux 版 life hacker Pack](https://lifehacker.com/lifehacker-pack-for-linux-our-list-of-the-essential-li-815376368)，它是提高工作效率和充分利用新安装的 Linux 的必备工具。
