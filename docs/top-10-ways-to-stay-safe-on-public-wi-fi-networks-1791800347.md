# 保持公共 Wi-Fi 网络安全的 10 大方法

> 原文:[https://life hacker . com/top-10-ways-to-stay-safe-on-public-wi-fi-networks-1791800347](https://lifehacker.com/top-10-ways-to-stay-safe-on-public-wi-fi-networks-1791800347)

免费 Wi-Fi 是一笔意外之财，特别是如果你在图书馆或机场工作，或者如果你只想在手机或笔记本电脑上保存数据。尽管如此，当你出门在外时，你还是得关心安全问题 。以下是如何在任何设备上安全冲浪。

### 10.养成良好的互联网卫生习惯

除了软件和承诺保护你隐私的工具之外，也许我们能给你的第一个也是最大的建议是实践良好的互联网卫生。当您使用不安全的公共 Wi-Fi 时，避免处理(至少是在线)敏感数据。这可能是查看新闻或阅读你最喜欢的博客的好时机，但这可能不是做网上银行的最佳时机，如果你明白我的意思的话。

### 9.使用正确的网络(避开不好的网络)

并非每个公共 Wi-Fi 网络都是平等创建的。比如那种潜伏在后台的“免费机场 Wi-Fi”网络，无疑比机场 *其中一家咖啡店、商店或者零售商 [提供的任何 Wi-Fi 网络都要差。请选择这些。您将体会到增加的安全性(尽管实际上只是通过模糊性实现的安全性，这是脆弱的)以及可能改进的性能。](https://lifehacker.com/all-the-national-chains-that-offer-free-wi-fi-1646148637)*

如果你很好奇，你可以使用 WiFox 和 [等工具及其移动应用](http://lifehacker.com/wifox-puts-thousands-of-airport-wi-fi-networks-and-thei-1787159421) 来绘制你的机场网络，并选择使用哪些网络以及何时使用。哦，万一需要说的话，如果有像“这里有免费 Wi-Fi！”或者“绝对免费的互联网！” [你大概应该避开他们](https://lifehacker.com/holiday-travel-tip-avoid-airport-honeypot-rogue-wifi-n-223491) 。

### 8.使用半开放的无线网络

当谈到您使用什么网络时，您可能并不总是有选择，但如果您有选择，请考虑“半开放”Wi-Fi 而不是完全开放的网络，考虑为机场休息室、附近的咖啡店提供隐藏 SSIDs 或将其密码写在收据上而不是免费提供的网络，等等。有时候 [工具，比如我们上面提到的 Wi-Fox](https://lifehacker.com/find-free-airport-wi-fi-with-this-interactive-map-1787550652) ，他们的数据库里就有那些。你也可以求助于 Google Places、Yelp 或者 [甚至老掉牙的 FourSquare](http://lifehacker.com/find-free-wi-fi-passwords-for-local-spots-on-foursquare-509034398) 来找到那些密码。

或者，在机场问讯处、图书馆服务台或咖啡店柜台询问 也没什么不对，但如果你想偷偷摸摸，我们已经介绍了 [比](http://lifehacker.com/get-free-airport-wi-fi-with-a-simple-url-hack-5040118) [更多的一些技巧](http://lifehacker.com/get-free-airport-wi-fi-using-chrome-or-firefoxs-develo-1607111327) [来获得高级](http://lifehacker.com/circumvent-wi-fi-time-limits-at-coffee-shops-by-spoofin-5916339) 、专属 wi-fi，并在使用公共网络的人群中隐藏自己。

### 7.关闭文件共享和隔空投送选项

您可能无法控制谁在使用什么网络，但是您可以控制您的电脑。无论您使用的是 Windows PC 还是 Mac，您的电脑可能都有一些文件共享选项，这些选项假定您与其他受信任的电脑在受信任的网络上。在 Windows 和 macOS 中关闭文件共享，启用系统的内置防火墙，并将与互联网连接的应用程序和服务保持在最低水平。Mac 用户，采取额外的步骤，将隔空投送的 [设置为仅限联系人的](https://lifehacker.com/set-airdrop-to-contacts-only-to-protect-your-privacy-1661694999) 。无论如何你都应该这么做，但是现在是个好时机。

然后， [自动执行这些设置](https://lifehacker.com/how-to-stay-safe-on-public-wi-fi-networks-5576927) ，这样当你在家或在受信任的网络上时，你的机器是打开的，当你不在家时，它会自动切换到更安全的设置。以下是方法。

### 6.不使用时请关闭 Wi-Fi

安全的基本规则之一是，如果你不需要连接到网络的东西，就不要连接它。当你完成在线工作后，关闭你的笔记本电脑、平板电脑或智能手机上的 Wi-Fi。当你使用不可信的网络时，这是一个很好的安全习惯(如果你不得不使用它们的话)，它也可以节省你的电池。在 macOS 中很容易做到 [，在 Windows 中你甚至可以将笔记本电脑设置为](https://lifehacker.com/disconnect-from-a-wi-fi-network-in-os-x-without-turning-1714736241) [在短时间离线后自动开启 Wi-Fi](https://lifehacker.com/windows-10-can-automatically-turn-wi-fi-back-on-after-a-1788015284)。

你可以更进一步，下载你的电子邮件和文件的本地副本，以便在没有连接的时候使用，或者在没有互联网接入的情况下使用 Google Drive offline(或类似的服务)工作，或者在根本没有连接的时候只需抓取电影或播放列表来听。

### 5.保持您的防病毒和防恶意软件处于最新状态

这个列表没有特定的顺序，但是我们敢打赌，你们中的一些人可能已经期望很快看到这个技巧。如果你*曾经*使用公共的、不可信的网络，确保你的电脑正在运行某种 [反恶意软件实用程序和补充反病毒实用程序](https://lifehacker.com/the-difference-between-antivirus-and-anti-malware-and-1176942277) 。

对于前者， [Malwarebytes](http://www.malwarebytes.org/) 是我们最喜欢的，定期 [出现在测试](http://lifehacker.com/10-malware-removal-apps-tested-malwarebytes-comes-out-1614046598) 之上。对于后者， [你有选项](http://lifehacker.com/five-best-desktop-antivirus-applications-1607557993) 。我们喜欢[Avira for Windows](http://lifehacker.com/the-best-antivirus-app-for-windows-5865356)和[Sophos for MAC OS](http://lifehacker.com/the-best-antivirus-app-for-mac-488021445)(也适用于 Windows)。无论你选择使用什么，都要保持更新，并保持运行——尤其是当你外出时。众所周知，公共无线网络 [会在你浏览](https://arstechnica.com/tech-policy/2014/09/why-comcasts-javascript-ad-injections-threaten-security-net-neutrality/) 的时候植入广告，我们都知道 [广告植入会有多糟糕](https://lifehacker.com/how-to-protect-yourself-from-malvertising-on-the-web-1745588094) 。

### 4.安装保护隐私的浏览器扩展

反恶意软件是伟大的，但它只真正保护你从你下载和执行的东西，禁止恶意广告或恶意软件缠绕在你的系统上，而不是你自己的过错。下一步是 [用专为保护你的隐私而设计的工具来加固你的浏览器](https://lifehacker.com/the-best-browser-extensions-that-protect-your-privacy-479408034) 。

你可能已经使用了广告拦截器，但是一个好的、可定制的广告拦截器，像可以让你在需要的时候控制什么被拦截，什么不被拦截，并且可以在你需要的时候锁定任何东西。 [断开](http://disconnect.me/) 是另一个很好的选择，保护你免受同网攻击 [如会话劫持和点击劫持](http://lifehacker.com/privacy-protecting-add-on-disconnect-now-keeps-your-bro-5961724)——这两种 [仍然是真正的威胁](https://lifehacker.com/how-to-protect-yourself-from-apps-that-make-wi-fi-hacki-955023487) ，可以让人们访问你的亚马逊账户或脸书账户，即使你正在安全地浏览。

### 3.尽可能使用 HTTPS

在 HTTPS 工作的任何地方，使用它——电子前沿基金会 的 [HTTPS 无处不在——可以确保你做到。当然，并不是每个网站都支持它，但如果它可用，插件会试图把你带到网站的安全版本，如果它不在那里，你会回滚到普通 HTTP 版本。](https://lifehacker.com/https-everywhere-updates-to-keep-you-secure-on-thousand-1695541941)

这并不是绝对完美的安全，但是确保你到一个给定站点的连接是安全的意味着至少 [发送给你的信息(以及你发送到该站点的信息)是加密的](https://lifehacker.com/why-should-i-care-about-https-on-facebook-or-other-web-5745086) ，这大大有助于确保任何窥探你通信的人不会得到任何敏感的东西。所谓“窥探”，我们实际上是指某人只是从网络上获取数据包，寻找信用卡号、密码或会话 cookies 来访问登录的帐户——不是众所周知的强大对手，对此几乎没有追索权(老实说，不是我们在这里谈论的人。)

### 2.使用虚拟专用网(或者自己动手)

当你不得不使用一个不可信的网络时，我认为最好的保护是直接加密访问一个可信的网络。 [这意味着当你外出时使用 VPN](https://lifehacker.com/why-you-should-be-using-a-vpn-and-how-to-choose-one-5940565) 。无论你是使用第三方 VPN 服务提供商还是 [你在家里部署自己的 VPN](http://lifehacker.com/roll-your-own-vpn-with-a-raspberry-pi-and-openvpn-1563401069)并在外出时连接到它，使用一个可以确保你和服务提供商之间的所有数据都是加密的，将与你在同一网络上的任何人都锁在外面，因为他们可能会四处窥探。

当然，如果你在家里安装了自己的网络并连接到它，我们所说的可信网络就是你的家庭网络和你的家庭 ISP。它可能会慢一些，但至少你知道同等的体验是你在网上冲浪或在家工作。如果你使用 VPN 提供商，你需要 [确保他们值得信任](https://lifehacker.com/how-do-i-know-if-my-vpn-is-trustworthy-508866499)[不要泄露你的 IP 地址](http://lifehacker.com/how-to-see-if-your-vpn-is-leaking-your-ip-address-and-1685180082) ，或者 [泄露你的任何其他信息](http://lifehacker.com/how-to-make-sure-your-vpn-is-honest-about-its-internati-1760243121) 。移动虚拟专用网 也是如此，当你使用它们的时候，它们可能会被恶意软件入侵并窃取你的私人信息。

### 1.请带上你自己的无线网络

对不可信网络的真正保护是根本不使用它。当然，这不是让公共 Wi-Fi 变得更安全的真正方法，但是如果可以的话，考虑完全抛弃公共 Wi-Fi，带上你自己的 Wi-Fi。无论你是 [使用 MiFi 或 Karma](https://lifehacker.com/five-best-mobile-hotspots-1523254435) 之类的移动热点，还是你 [只是系上你的智能手机](https://lifehacker.com/why-should-i-buy-a-hotspot-when-i-can-just-tether-to-my-1576789744) 并使用你的无线运营商的数据，这两种方法都可以让你在机场等待转机时摆脱粗略的公共 Wi-Fi，转而使用蜂窝数据。

当然，价格可能会有所不同，这并不在每个人的预算中，但有些人有按使用付费的选项， [像我们最喜欢的，](http://lifehacker.com/karma-kills-the-neverstop-unlimited-plan-introduces-pu-1759893383) 。如果你不打算购买和携带其他设备，你可以只使用你的智能手机——只需随身携带一个外部电池组，并在工作时查看你的数据上限。
