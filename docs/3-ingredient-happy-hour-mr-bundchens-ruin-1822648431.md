# 三要素快乐时光:邦辰先生的毁灭

> 原文：<https://lifehacker.com/3-ingredient-happy-hour-mr-bundchens-ruin-1822648431>

祝大家超级碗周末快乐，欢迎回到 [*三种配料的欢乐时光*](http://lifehacker.com/tag/3-ingredient-happy-hour) ，每周饮品专栏提供超级简单又美味的酒水。本周我们制作鸡尾酒向美国最具争议的四分卫致敬:汤姆·邦辰先生。



我承认今年我没有太关注足球，所以我没有掌握最新的数据和事实，但我知道关于老汤姆的这些事情是正确的:

1.  他是一名优秀的四分卫。
2.  他的这本 200 美元的食谱毫无欢乐可言(还有西红柿)。
3.  他的妻子做了 [这个](https://www.youtube.com/watch?v=oZ_DqsheyI0) 。
4.  [他从未吃过草莓](https://www.thecut.com/2016/09/tom-brady-spends-his-suspension-napping-and-repping-uggs.html) 。
5.  我真的不想让他再赢一次超级碗。

基于这些事实，制作绿色鸡尾酒(支持费城，不管多么吝啬)是有意义的，而且这种绿色鸡尾酒应该是草莓味的。起初，这看起来像是一个傻瓜的差事，因为草莓显然不是绿色的，但后来我想起了哈里波草莓小熊软糖，它是——莫名其妙地——绿色的。

受牛顿 T3 的启发，我把 91 克草莓小熊软糖放在一个装有 200 克伏特加的罐子里，然后把罐子放在 140 度的浴缸里三个小时。然后，我用一些咖啡过滤器过滤利口酒，并惊叹于滤液的美丽。

一旦你做到了这一点，你有两个选择:你可以让伏特加达到室温，然后制作下面的鸡尾酒，或者你可以把它倒入小杯子里，制作一些非常简单、非常强烈的双成分果冻。

我讨厌果冻，所以我选择了鸡尾酒。鉴于我们已经使用了两种原料来制作甜酒，我们只需要再添加一种。要毁掉邦辰先生，你需要:

*   2 盎司绿色草莓利口酒
*   苏打水(或者姜汁汽水，如果你需要甜一点的)

抓起一个加冰的小玻璃杯，倒入你的利口酒，加入苏打调味，搅拌混合，加入冰。举起你的酒杯到费城，吃一些小吃，重复直到新英格兰被征服。
