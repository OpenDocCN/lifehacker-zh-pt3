#### 首先，告诉我们一些你的背景，以及你是如何走到今天这一步的。

我来自南方腹地——确切地说是艾尔的塞尔玛——我 18 岁时搬到亚特兰大，在那里上了莫尔豪斯学院(T2)的 T1 和 T3。2003 年，我在那里获得了数学本科学位，然后我在这座城市的几家公司工作过(在& T，WebMD 等等)。)2008 年获得电信管理硕士学位之前。那年晚些时候，我开始吃午饭，然后在 2013 年开始修订路径，在 2017 年底，我加入了 Fog Creek 的团队。

#### 你的工作空间是什么样的？

因为我远程工作，我可以在任何有网络连接的地方设置。有时这是一家当地的咖啡店，也可能是一个共同工作的空间。大多数时候我在家工作，我的家用电脑有一个双显示器设置，必要时我会把我的笔记本电脑连接到其中一个屏幕上。我在我的家用电脑上有一个 [Nighthawk Pro X 机械键盘](http://www.maxkeyboard.com/max-keyboard-nighthawk-pro-x-cherry-mx-rgb-multicolor-backlit-mechanical-keyboard.html) ，我使用一个 [罗技 MX Ergo 无线轨迹球鼠标](https://www.logitech.com/en-us/product/mx-ergo-wireless-trackball-mouse) 和罗技 FLOW 软件，所以我可以像多显示器设置一样在两台计算机上使用它。我的桌子上还有一个 [Rode PSA1 吊杆](http://www.rode.com/accessories/psa1) 和一个 [Audio-Technica AT2020 录音室麦克风](https://www.audio-technica.com/cms/wired_mics/a0933a662b5ed0e2/) 用于轻松录制视频或播客。

当我在纽约办公室时，我有一个稀疏的隔间空间。但是因为我们公司大部分是由远程员工组成的，我们为直接在总部工作的人提供了很好的办公空间，包括许多基于不同工作风格的不同工作区。

#### 关于有效设计，你从播客嘉宾那里学到了什么？

问正确的问题非常重要。我认为如果服务于正确的目标，所有的设计都是有效的，但我发现有时客户和企业并不清楚目标是什么。作为一名设计师，你应该尽可能多的问一些问题来澄清任何模糊之处，这样你就能抓住问题的核心。我认为这对每个人来说都是一个更好的设计过程。

#### 你在生活帮的最佳捷径是什么(不管有多小或多小众)？

调度和时移；这是我能掌控一切的唯一方法。我提前几周安排播客节目。我按季度安排社交媒体。我甚至定时发送邮件，这样如果我熬夜回复邮件，我就可以定时发送邮件，在我睡觉的时候发送。时间是一种建构，所以去破解它，让它为你所用。

另外，我的“现在”页面还有。 [为德里克·西弗斯的这个绝妙想法大声欢呼。人们总是问我在做什么，我只是指给他们看。为我节省了大量回复随机邮件的时间。](https://nownownow.com/about)

#### 你如何记录你必须做的事情？

谷歌日历和雷鸟。如果它不在我的日历或收件箱里，那么它对我来说就不存在。我的记性很好，但我用一个修改过的谷歌电子表格作为我联系过的每个人的客户关系管理系统。

#### **如何充电或休息？**

音乐和写作。整天，每天。

#### 你最喜欢的副业是什么？

现在，我正在构思一个图形小说的概念，这个概念我已经思考了好几年了。这是我一直想做的事情，但当其他工作出现时，我总是把它放在一边。现在这些天我有了更多的时间，我终于把这个想法从我的脑袋里取出来，放进笔记本里，这样我就可以开始写了。重温、修改和充实这些角色和故事真的很有趣。

#### 你目前在读什么，或者有什么推荐？

我总是在读一些东西，不管是我口袋队列里的一篇文章，还是我书架上的一些东西。我在工作的时候也会听很多有声读物和播客，所以推荐对我来说很难。我刚刚看完大卫·塞达里斯的新书 [*卡里普索*](https://www.amazon.com/Calypso/dp/B07BHVQS7R/?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/im-revision-path-host-maurice-cherry-and-this-is-how-i-1826916236&asc_source=&tag=kinjalifehackerlink-20) 我正在看 [*火中无灰:美国黑人自由时代*](https://www.amazon.com/No-Ashes-Fire-Coming-America-ebook/dp/B075CSDR9J/?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/im-revision-path-host-maurice-cherry-and-this-is-how-i-1826916236&asc_source=&tag=kinjalifehackerlink-20) 来自达内尔·l·摩尔。
