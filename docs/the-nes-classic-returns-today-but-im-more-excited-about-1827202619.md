# NES 经典今天回归，但我对这个无线控制器更感兴趣

> 原文：<https://gizmodo.com/the-nes-classic-returns-today-but-im-more-excited-about-1827202619>

当 NES 经典版首次发布时，我们 [认为 40 美元的 8Bitdo NES30 是微型游戏机的最佳无线控制器替代品](https://gizmodo.com/the-nes-classic-is-so-much-better-with-a-wireless-contr-1791570302#_ga=2.1756849.1469666760.1530188579-1791892859.1530188579) 。但现在任天堂从今天开始再次销售 NES 经典游戏，我对 8Bitdo 的新游戏手柄更感兴趣，这是一个更好的无线替代品，价格更便宜，还有一个专用的“home”按钮。



像 NES30 一样，N30 借鉴了它的设计，缺乏人体工程学，来自 80 年代第一代 NES 附带的四四方方的矩形游戏手柄。它可能不舒服，但它肯定会满足你怀旧游戏的渴望。

然而，与 NES30 不同的是，N30 仅适用于 NES 经典版。它不使用蓝牙，而是使用专用的 2.4GHz 无线连接来连接内置的加密狗，因此一旦开机，游戏手柄就会立即自动配对，并准备好开始游戏。动作按钮被重新排列，以匹配原来 NES 游戏手柄上的布局，NES30 的肩部按钮被移除。N30 增加了一个“home”按钮，用于访问 NES Classic Edition 的主菜单，这也是 NES30 的功能，但专用的一键式按钮肯定是更好的方法。

不过，8Bitdo 新推出的 N30 游戏手柄最大的特点无疑是其 25 美元的价格。这比 [便宜 15 美元，NES30](https://gizmodo.com/the-nes-classic-is-so-much-better-with-a-wireless-contr-1791570302#_ga=2.93424285.1469666760.1530188579-1791892859.1530188579) 便宜 40 美元，这使得购买付费的 N30 来替换 NES 经典版附带的两个控制器更加经济实惠。让你的决定更容易的是，在几周前 E3 对其产品线进行重大更新后，8Bitdo [实际上不再销售 NES30】。因此，当 NES 经典版明天再次发售时，这绝对是你想要与之搭配的无线控制器。](https://gizmodo.com/the-nes-classic-is-so-much-better-with-a-wireless-contr-1791570302#_ga=2.1756849.1469666760.1530188579-1791892859.1530188579)
