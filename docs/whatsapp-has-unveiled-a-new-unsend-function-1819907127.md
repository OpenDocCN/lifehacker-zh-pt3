# 如何在 WhatsApp 上取消发送信息

> 原文:[https://life hacker . com/whatsapp-has-published-a-new-un send-function-1819907127](https://lifehacker.com/whatsapp-has-unveiled-a-new-unsend-function-1819907127)

手指移动速度超过大脑的 WhatsApp 用户会很高兴知道该公司推出了一项新功能，让你收回最后一条消息，并将其从对话中删除。目前该功能正在向某些用户推出， [根据下一个 Web](https://thenextweb.com/apps/2017/10/27/whatsapp-now-unsend-feature-lets-delete-drunk-texts/) 。

Watch

根据 WhatsApp 的 FAQ 页面 显示，预期删除功能允许你从记录中删除自动更正失败或发送给错误的人的信息。每个人，无论是与一个人聊天还是与一个群组聊天，都会看到“此消息已被删除”，而不是您之前的消息。然而，你只有 7 分钟的时间来摆脱谈话。在那之后，你就不走运了。

为了确保你是可以删除消息的幸运用户之一，你必须确保你在 iOS、Android 或(哈！)Windows Phone(又来了，哈！)设备。您的聊天接收者也需要将 WhatsApp 更新到最新版本。更新后，您可以通过选择七分钟内的消息来检查您是否拥有该功能。iOS 用户将按住该消息，选择删除选项，然后为所有人删除(如果可用的话)。Android 和 Windows Phone 用户可以选择一条或多条消息，选择删除选项，然后为每个人选择删除。

在您依赖基于聊天的撤销按钮之前，确实需要做一些小的改进。在您删除邮件之前，收件人可以看到您的邮件，并且没有办法验证邮件是否已从每个人的对话中成功删除。不过，如果你喝了几杯酒，想要收回发给你的裁缝而不是你的约会对象的浪漫预付款，撤销按钮可能是一个救命稻草。
