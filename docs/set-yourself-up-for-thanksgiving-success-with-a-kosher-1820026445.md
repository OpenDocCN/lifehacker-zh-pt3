# 用一只犹太火鸡为感恩节的成功做好准备

> 原文：<https://lifehacker.com/set-yourself-up-for-thanksgiving-success-with-a-kosher-1820026445>

感恩节是一个奇怪的节日，原因有很多，而整只火鸡可能是最奇怪的。成千上万的食谱敦促你以改善适口性的名义腌制、摩擦、炖煮、烟熏，甚至油炸这种仪式用的鸟，这证明了火鸡是一种需要大量哄骗的蛋白质。最简单的解决办法是:买一只犹太火鸡，继续你的生活。

什么使火鸡符合犹太教规？有多种解释(有些解释比其他解释更长)，但我在这里将坚持简短的解释:犹太火鸡被迅速而人道地宰杀，仅在冷水中清洗，并彻底腌制和冲洗以沥干最后一丝残留的血液。在清洗和加工过程中，冷水可以保持肉类的超新鲜，但腌制是关键步骤。任何称职的厨师都知道(我很抱歉)，增加肉中的钠含量有助于在烹饪过程中保持水分——特别是如果它会被冷冻的话，几乎所有的火鸡都会被冷冻。另外，显而易见的是，盐让食物变得美味。

天然火鸡的倡导者坚持认为犹太火鸡太咸，但老实说，我不买它。浸泡和冲洗过程是专门设计来去除几乎所有的盐，所以你可以盐水，摩擦，烟熏，或油炸一只犹太鸟，让你的小心脏满意。但除非你真的想做，否则你不必做所有这些，因为只要用橄榄油(或黄油，如果你不符合犹太教规)和盐，一只符合犹太教规的火鸡就会烤得多汁美味。无论你习惯于大规模生产的黄油球还是自由放养的传统鸟类，今年都可以考虑买一只犹太火鸡；我想你会惊喜的。
