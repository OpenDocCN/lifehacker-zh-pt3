# 接下来我们应该做什么？

> 原文:[https://life hacker . com/what-should-we-sous-vide-next-1798504555](https://lifehacker.com/what-should-we-sous-vide-next-1798504555)

各位网友大家好，欢迎回到另一个激动人心的话题选择环节 [*会不会？*](http://lifehacker.com/tag/will-it-sous-vide) ，列在那里我用我的沉浸循环器做任何你想让我做的事。

Watch

上周的香蕉布丁聚会非常有趣，主要是因为我吃了很多香蕉布丁，但现在我们必须继续另一个真空实验。几周以来一直流传的一些想法:

*   **冷冻千层面:**像 [热口袋](http://skillet.lifehacker.com/will-it-sous-vide-you-picked-hot-pockets-1786400775) 在它之前，真空冷冻千层面可能是有史以来煮得最均匀的冷冻千层面。我也可以使用我的手电筒，这总是一个好时机。
*   **面包:**人们一直在建议面包，我也不确定为什么。浸没循环器不会变得足够热以形成任何类型的外壳，但我想我们可以研究一下 [低温面包](http://lifehacker.com/bake-delicious-bread-low-and-slow-even-overnight-1761355095) 的情况。(我确实认为我们可以用这个东西做一些很好的校对。)
*   **另一份全餐:** Fin Flang Flareon 建议一份 [袋装餐情况](http://skillet.lifehacker.com/will-it-sous-vide-three-whole-meals-in-a-bag-1787768799) 有烤猪肉、黑豆和米饭、车前草和丝兰根，我不反对。
*   羊颈肉:我只是有一段时间没吃羊颈肉了，我觉得它放在我们烹饪用的热水浴缸里会很好吃。

像往常一样，在评论中随意添加你自己未列出的想法，不要害羞。这周我有心情做一些非常奇怪的事情，所以去疯狂吧。
