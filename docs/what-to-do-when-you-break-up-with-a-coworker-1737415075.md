# 当你和同事分手时该怎么办

> 原文：<https://lifehacker.com/what-to-do-when-you-break-up-with-a-coworker-1737415075>

你拥有完美的——直到你没有。现在你们分手了。你把他或她的所有东西都从你的公寓里搬走了，还把你们可爱的情侣脸书的照片都取消了标签。但是当你在办公室见到你的前任时，事情就复杂了。

正常的分手很难，但办公室分手就完全不同了。不幸的是，事情是以双方一致同意还是激烈争吵而告终并不总是那么重要。和你以前的另一半一起工作真的很尴尬*。*

但是，这种不舒服的氛围并不意味着你需要跑出去找一份全新的工作。以下是度过办公室恋情破裂的五个小贴士——即使你的前任就在隔壁。

### **1。保持专业**

希望你和你以前的另一半从一开始就同意将你们的关系排除在工作场所之外。不管怎样——现在不是让你的个人事务悄悄影响你的工作效率和职业声誉的时候,。

尽管看起来不可能，但你需要像对待其他同事一样对待你的前任。保持你所有的沟通礼貌和严格的专业(比其他任何人都要多，当休息还新鲜的时候)，避免小声嘀咕讽刺的话。是的，咬你的舌头会很难。但是，这很重要——因为办公室里的一场尖叫比赛不会让你的情况有任何好转。

### **2。避免流言蜚语**

当一段感情破裂时，向任何愿意倾听的人说你前任的坏话是很有诱惑力的。毕竟基本上是分手手册的第一页。但是，这是一次办公室分手。所以，游戏规则有点不一样。

远离 [办公室八卦](https://www.themuse.com/advice/8-noncondescending-ways-to-respond-to-a-coworker-who-wont-stop-gossiping) ，避免和同事聊你们分手的细节(不管他们让你透露多少细节)。虽然你脑海中那个复仇的声音可能会鼓励你彻底毁掉他或她的名声，但这只会让*你*看起来很糟糕。

没必要给办公室剧火上浇油。而且，你不希望你们的分手成为一个夸张的办公室传奇。(“有一次，营销部门的一对夫妇分手了，嗯，我们从来没有见过有这么多秘密提及作弊的演示。”)

### **3。限制你的互动**

好吧，所以你接受了这个事实，你需要放下你的骄傲，放下你的负面情绪，继续和这个人一起工作。但是，这并不意味着你们需要经常参与对方的日常生活。

所以，当你的情绪还未成熟时，你可能想练习古老的回避分手策略。不，你不能为了和前任保持距离而逃避那些重要的团队会议或大会。但是，你可以改变你的时间表中的一些小部分，这有助于缓解你的紧张状况。

早一点到达工作地点，避免一起走进办公室时令人不舒服的沉默。在你前任的午餐时间，远离休息室，或者叫一个同事和你一起吃饭。

是的，这可能看起来像一个完全的高中策略，但你不需要幼稚，甚至不需要大声说出你在做什么。而且，如果这有助于从你的动力中消除一点点焦虑，这是完全值得的。

### **4。将你的精力投入到工作中**

我们可能都曾在某个时候对分手有点痴迷。这是人之常情，真的。但是，与其浪费你的时间纠结于你的前任已经在谈论这个即将到来的周末的约会，不如将所有被压抑的能量和激情投入到你的工作中。

这并不总是容易的，但是试着在办公室门口控制你的情绪，专注于你的工作。提醒自己，你只需要熬过接下来的九个小时。这不仅会把你的注意力从与前任的关系上转移开，还会向你的 [老板](https://www.themuse.com/advice/4-ways-to-impress-your-tough-new-boss) 展示你是一名多么有效率和令人惊叹的员工。这足以让你感觉好一点了。

### **5。再次参与前暂停一下**

你花很多时间在工作上。所以，浪漫会在那里绽放是完全可以理解的(甚至是意料之中的)。

但是，当你从这种特别紧张和尴尬的情况中恢复过来的时候，花些时间评估一下环境。办公室恋情值得如此格外小心谨慎吗？也许是，也许不是。无论你做出什么决定，花时间认真思考这个问题是很重要的——尤其是在开始另一段浪漫的工作关系之前。

所有的分手都至少会有一点点不舒服、痛苦和分心。但是，办公室分手给本已脆弱的局面带来了全新的尴尬元素。

不，你不需要辞掉工作，搬到全国各地。简单地把这些建议运用到工作中，让你有尊严地度过这一切。哦，你仍然可以在下班回家的路上买一份高热量的零食。你应得的。
