# 最懒惰，最便宜的方式来规避你的窥探 ISP

> 原文：<https://lifehacker.com/the-laziest-cheapest-way-to-circumvent-your-snooping-i-1793789594>

[国会决定](https://lifehacker.com/senate-votes-to-let-internet-providers-sell-your-web-hi-1793574677) 你的 ISP 应该被允许出售你的私人浏览数据，但是解决这个问题的方法是 [有点复杂，成本高，或者只是一个痛苦](https://lifehacker.com/why-is-everyone-talking-about-vpns-1793768312) 。一个懒于上网的人会做什么？使用 Opera。

Watch

[去年](https://lifehacker.com/operas-free-vpn-built-right-into-the-browser-rolls-ou-1786935648) ，Opera，这个大家似乎都忘了的小浏览器，推出了免费的 VPN。 [虽然它因泄露 IP 地址而立即遇到了安全问题](https://lifehacker.com/stop-opera-s-new-vpn-from-leaking-your-ip-address-1773359437) ， [它现在已经被修补好了](https://lifehacker.com/stop-opera-s-new-vpn-from-leaking-your-ip-address-1773359437) ，并且是最简单、最便宜、更合理的私人方式来访问 VPN，这会绕过你的 ISP。不过，它也有一系列的警告。

如果你感兴趣，这里是 Opera 的隐私声明 ，但是让我们浏览一下要点。 [在接受 PCWorld 采访时，](http://www.pcworld.com/article/3120493/browsers/hands-on-operas-free-unlimited-browser-vpn-is-ready-for-secure-surfing.html)Opera 发言人表示，VPN 是一种无日志服务，这很好，但还有更多。首先，虽然 Opera 是一家挪威公司，因此根据挪威法律行事，但提供 VPN 服务的 SurfEasy 公司是一家加拿大公司，加拿大以 [移交情报数据](https://en.wikipedia.org/wiki/Five_Eyes) 而闻名。不管怎样，使用 VPN 意味着你同意 SurfEasy 的隐私政策。Opera 去年也被一个中国财团收购，所以 Opera 收集的任何数据都有可能在某个时候被 T21 获得。

还需要记住的是，不像传统的 VPN 将你设备上的所有流量作为一个整体进行加密，只有你在 Opera 中进行的网页浏览才会通过他们的 VPN。因此，它更像是一个代理，而不是 VPN，但它仍然适用于我们这里的目的。你还可以在 [安卓](https://play.google.com/store/apps/details?id=com.opera.vpn) 和 [iOS](https://www.bestvpn.com/goto/operaapps/vpn/iphone) 上抢到 Opera VPN，在移动设备上加密你的流量。移动应用程序像普通的 VPN 一样工作。

对于普通人来说，这些都不重要，简出于懒惰绕过 ISP，希望给康卡斯特/频谱/美国电话电报公司/任何人一个大大的 f-you，但出于隐私目的，这仍然是一个重要的信息。毕竟，物有所值。

好吧，尽管有这些问题，Opera 仍然是一种让你的数据不落入 ISP 之手的方法。下面是它的使用方法:

1.  为你的操作系统下载 [Opera](https://www.opera.com/download) 。
2.  单击设置(在 Mac 上单击 Opera >偏好设置)。
3.  单击隐私与安全部分。
4.  单击启用 VPN。当你在这个屏幕上时，你也可以点击不要跟踪和阻止广告的复选框，以帮助避开其他类型的跟踪。

启用后，你会在地址栏看到一个 VPN 按钮。我还会下载并安装 [HTTPS 无处不在](https://addons.opera.com/en/extensions/details/https-everywhere/) 以及 [隐私獾](https://www.eff.org/privacybadger) 或 [断开](https://addons.opera.com/en/extensions/details/disconnect/?display=en) 来增加一点安全性和隐私性。
