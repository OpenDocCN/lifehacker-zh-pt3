# PSA:二手电子烟蒸汽对每个人都不健康，尤其是孩子

> 原文：<https://lifehacker.com/psa-secondhand-e-cigarette-vapor-is-unhealthy-for-ever-1795762043>

当你使用电子烟或 vape 时，你呼出的蒸汽不会神奇地消失。二手蒸汽是一个非常重要的东西，它们对你周围的任何人呼吸都是不健康的，尤其是小孩子。

虽然吸 vaping 被认为比吸普通香烟 更健康——因为你不会让你的肺被致癌的焦油覆盖——但无论如何它仍然不是一种“健康”的做法。 [正如医务总监解释的](https://e-cigarettes.surgeongeneral.gov/knowtherisks.html#aerosol-exposure) ，当你使用电子烟时吸入的烟雾中含有有害化学物质和超细颗粒，你可以将它们深深吸入肺部。二乙酰、苯、镍、锡和铅只是在电子烟蒸汽中发现的一些东西。科学家们仍在试图确定电子烟液体中其他潜在有害化学物质对健康的影响，以及它们在高温下变成气溶胶时的反应。此外，电子烟含有尼古丁，无论你如何摄入，尼古丁仍然很容易上瘾。

现在，如果你想用电子烟，那完全取决于你。这是你的健康，你可以随意使用或滥用它。有问题的是电子烟和 vape 使用者似乎没有意识到这些蒸汽在呼出后仍然可以呼吸。就像二手烟一样，二手烟是使用者周围每个人都关心的问题，即使这种烟雾比传统香烟烟雾更不明显。美国疾病控制和预防中心最近发布的一份报告显示，三分之一使用电子烟的成年人不认为二手烟有害，另有 40%的人表示接触电子烟只会对儿童造成“很小”或“一些”伤害。但是不要搞错了，仅仅因为这不是真的香烟，仍然存在真正的危险。

除了已经提到的有害化学物质，暴露在这些二手烟雾中的孩子的风险包括意外的尼古丁成瘾、情绪障碍和冲动控制能力下降。尼古丁还可以通过改变年轻人大脑中突触的形成方式来影响他们的大脑发育。美国疾病预防控制中心吸烟与健康办公室研究翻译副主任布莱恩·金博士表示，缺乏监管、产品的“新颖性”以及行业将电子烟宣传为“更健康的替代品”的方式 [导致了混乱](http://www.npr.org/sections/health-shots/2017/05/25/530026958/many-adults-dont-think-exposure-to-vaping-is-bad-for-kids) 。但不要上当；电子烟和 vapes 与普通香烟没有太大区别，所以在使用时要采取预防措施。另外， [它们会，呃，爆炸](https://lifehacker.com/why-e-cigarettes-explode-and-how-you-can-vape-safely-1782001156) ，所以也要小心。
