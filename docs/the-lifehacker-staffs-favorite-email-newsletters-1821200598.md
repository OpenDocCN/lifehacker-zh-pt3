# 2017 年 Lifehacker 员工最喜欢的电子邮件简讯

> 原文：<https://lifehacker.com/the-lifehacker-staffs-favorite-email-newsletters-1821200598>

Twitter、脸书和新闻提醒都证明了这一点:一对多沟通的最佳形式是电子邮件简讯。Lifehacker 的员工会关注时事通讯，了解早间简报、对技术的独特见解或个人故事。这些是我们的最爱。



### 专职作家帕特里克·奥斯汀的精选

*   [**NextDraft**](https://nextdraft.com/):戴夫·佩尔的“互联网执行主编”头衔是对 next draft 整体的一个相当准确的描述。佩尔对你应该关心的事情(除了比当天其他新闻更有趣的故事之外)的快速机智和出色品味，使它成为我允许访问收件箱的两份简讯之一。
*   [**轨迹变化**](https://trackchanges.postlight.com/tagged/newsletter) :来自纽约数码产品工作室 Postlight 的简讯。如果你是一个书呆子，或者对网络如何工作有短暂的兴趣，从开发到设计，再到操作，都不如你的 Instagram 页面那样光彩夺目，订阅吧。你将每周收到最有趣的设计、工程故事、工具和服务。

### 育儿编辑米歇尔·吴的推荐

*   [**Quartzy**](https://qz.com/newsletters/quartzy/) :虽然主要是最近 Quartz 生活方式故事的精彩卷轴，但它读起来像一封来自一位聪明朋友的私人信件。时事通讯策展人 Jenni Avins 将生活观察与值得我花时间的链接融合在一起——这是一个受欢迎的从标准的头条新闻综述中走出来的弯路。

*   [**Pocket hits**](https://getpocket.com/explore/pocket-hits):Pocket 的《每日文摘》是一份标准的标题综述，但它令人兴奋的地方(是的，我是少数几个真正对电子邮件简讯感到兴奋的人之一)在于，它可以展示任何地方的内容，从《纽约时报》到个人博客——无论是什么最有趣的内容。编辑们看看口袋里存了些什么，并挑选出他们认为每个人都应该阅读的内容。*【订阅，注册 Pocket，进入你的* [*通知设置*](https://getpocket.com/notifications) *，在“Pocket Hits”下选择一个频率】*

### 主编梅丽莎·基尔希的选择

*   [**下一稿**](http://nextdraft./) :它每天下午三点到来，就像闹钟告诉我，如果一天还没有结束，那就快结束了，我会接受。时事通讯以一天中的十大新闻为特色，无论它们是最大的新闻还是人们在 Twitter 上谈论的主要事情。不知何故，戴夫·佩尔确切地知道我感兴趣的是什么，他用一种温和的观点来表达，这不会让我讨厌他。这是我订阅的唯一一份不矛盾的每日时事通讯。

### 个人理财作家艾丽西娅·亚当奇克的建议

*   她不常发，但我一直期待 Buzzfeed 文化作家 Bim Adewunmi 的 tinyletter， [**...这他妈是什么？**](https://tinyletter.com/bim) ，里面写满了优美的文字和沉思。
*   卡拉·卡特鲁祖拉的 [**黄铜戒指日报**](https://us14.campaign-archive.com/home/?u=b1819370f078f7aaea854586f&id=985443e86d) 帮助我更有效率。
*   我每天至少打开三个来自 [**公寓疗法的简讯**](https://www.apartmenttherapy.com/mailing_list) 的链接。

### 专职作家尼克·道格拉斯的精选

*   [**海伦:一篮子**](https://tinyletter.com/hels) :美食作家海伦·罗斯纳在她偶尔的简讯中向我介绍了 [罗伯托](https://tinyletter.com/hels/letters/issue-1-hello-roberto) ，这是我现在最喜欢的汤。一个月后，她批评这部 1954 年的电影“叙事世界建设的根本失败”(*布里格登*讲述的是一个小镇在每个世纪只有一天醒来，罗斯纳指出这不是和平而是可怕的。)即使你和我一样，几乎没有听说过布里格登，烤肉也是非常有趣的。

*   [**沙特纳·沙特纳**](https://www.shatnerchatner.com/) :马洛里·奥特伯格(Slate 的《亲爱的普鲁登斯》专栏作家和 [The Toast](http://the-toast.net/) )最近开始为她的电子邮件时事通讯沙特纳·沙特纳收费，你知道吗，这是值得的。奥特伯格专门研究历史讽刺，她的最新一期以摄政时代为基础:“奇怪的是，在这个时代，每个人似乎都有枪，但我们仍然在做剑的事情？我有这个权利吗？”
*   [**像这样**](https://tinyletter.com/meaghano) :当梅根·奥康奈尔即将出版的《母亲回忆录》 [*现在我们拥有一切*](https://www.amazon.com/Now-We-Have-Everything-Motherhood-ebook/dp/B0756C7L6X/?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-lifehacker-staffs-favorite-email-newsletters-1821200598&asc_source=&tag=kinjalifehackerlink-20) 让你哭笑不得，生孩子的时候，你会想回去看看她的简讯档案。你会一遍又一遍地小声说:“是的，就是这样。”
*   [**不能抱怨**](https://tinyletter.com/Emily_Gould/) :在艾米莉·古尔德回忆录式的时事通讯中，你也会说“是的，事情就是这样”，但你会说得更大声，也许会拍桌子。试试那期简短的 [“妈咪是个怪物。”](https://tinyletter.com/Emily_Gould/letters/mommy-is-a-monster)T9】
