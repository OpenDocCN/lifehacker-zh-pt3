# 三星 Dex 令人心碎地接近将手机变成一台成熟的电脑

> 原文：<https://gizmodo.com/samsung-dex-comes-heartbreakingly-close-to-turning-a-ph-1795028661>

三星 Dex 是一款小巧的新智能手机坞站，它小心翼翼地走向一个智能手机成为我们主要电脑的世界。它大约有一个冰球那么大，当连接上 [Galaxy S8](http://gizmodo.com/samsung-galaxy-s8-review-the-prettiest-phone-wins-1794526975) ，一个显示器，一个蓝牙键盘和鼠标时，Dex 将手机变成了一个功能惊人的桌面。这个想法是，你可以把你的手机放在家里、办公室等等的一个坞站里，而不是带着笔记本到处走。

对接智能手机并将其转变为全功能 PC 的概念是一个雄心勃勃的梦想，但在过去已经失败了很多次。 [Palm Foleo](http://gizmodo.com/palm-foleo-a-laptop-for-your-smartphone-264550) (从未发布) [摩托罗拉 Atrix](http://gizmodo.com/motorolas-atrix-android-phone-twin-cpu-cores-and-a-lap-5725505) (大规模翻牌)，以及 [微软 Continuum](https://www.microsoft.com/en-us/windows/continuum) (从未被广泛采用)只是实现神奇对接智能手机的几个可悲尝试。即使手机有足够的计算能力，实际体验也从未达到真正功能电脑的承诺。但考虑到 Galaxy S8 有多好，我有理由期待三星可能最终破解了这个难题。

Dex 的售价为 150 美元——这当然不包括 750 美元的手机价格。该设备的背面包括一系列端口，如用于视频的 HDMI out、用于电源的 USB-C、用于稳定互联网的以太网，以及用于连接有线鼠标和键盘或闪存驱动器的两个 USB 2.0 端口。

设置 Dex 相对容易，只需要两根电缆。第一次设置设备时，需要几分钟时间，这取决于您是否需要移动一些家具来布线。之后的每一次，就像把手机插到 Dex 扩展坞一样简单。几秒钟内，三星标志就出现在屏幕上，几秒钟后，你看到的是一个熟悉的桌面环境。

Dex 用户界面像其他流行的操作系统一样工作。在屏幕的左下角是一个应用程序启动器，类似于你在 Chrome 或 Windows 中看到的。在右下角，你会找到你所有的设置。屏幕底部还有一个任务栏，显示所有正在运行的程序。

使用 Dex 最令人惊讶的一点是它运行不同程序的速度。我可以轻松运行十几个浏览器标签，同时在后台播放音乐，没有丝毫延迟。

虽然这些应用程序的性能不错，但软件体验却不够完美。我几乎立刻就注意到，Dex 无法运行我在 Galaxy S8 上下载的所有标准应用程序。特别是，我错过了 Slack 和 Spotify，这是我工作中必不可少的——使用这些应用程序的劣质浏览器版本就不一样了。

小小的不便迅速加剧。因为我不会用 Spotify，所以我转到了 YouTube。但之后，每当我在 Twitter 或脸书上浏览自动播放的 GIF 或视频时，我的音乐就会完全停止播放。

当我发现我不能调整所有应用程序的大小时，我遇到了另一个主要障碍。一些被固定在电话的标准 16:9 比例。我很惊讶，即使是像脸书这样的著名应用程序也不会调整大小。

尽管存在这些问题，我还是决定坚持使用它，因为我认为使用 Dex 的主要好处是可移植性。在去芝加哥的一次短途旅行中，我带着写了一半的评论，在不同的显示器上设置了停靠栏。旅行时的设置过程有点让人头疼，最终，我后悔没有坚持使用我那可靠的旧笔记本电脑。Dex 并没有让旅行变得更容易，我想如果我在我工作的各个办公室(家里、办公室、父母家)有几个 Dex 坞站，也不会有什么不同，因为这种体验无法与桌面操作系统相比。

在最好的情况下，使用 Dex 就像使用一个非常蹩脚的 Chrome OS 版本。这些应用程序并不总是能正常工作，而且除了在 Word 文档中打字之外，它并没有让任何事情变得更容易。最可悲的是，我真的不需要码头。我可以一直使用连接到手机的蓝牙键盘。

最后，我发现 Dex 虽然有点前途，但令人沮丧。Galaxy S8 的强大功能足以让我相信，有一天我们可能会将智能手机作为我们的主要电脑。Dex 没有实现这个目标，尽管已经非常接近了。

## **自述**

*   德克斯速度超快。我从未体验过任何延迟，即使在启动了 10 多个应用程序之后。
