## 专家 1:汤姆·麦克帕兰不会打鼓，但会玩旋转棍子

丹，我想我们都同意鼓手是一个特殊的品种。我自己一直都想打鼓，虽然我最接近打鼓的方式是在我摇着工具或拉什的时候敲出一些空气节拍。(没错，我是个呆瓜。)我甚至有一个大学朋友，他是一个非常优秀的鼓手，我同意教他如何划船，如果他教我一些节拍的话。但他搬到了佛罗里达，所以从未成功过。

别再说我了，你需要一辆车。你最好的选择是一个带轮子的大箱子，这就是为什么你想要一个本田元素。这个元素是本田对跨界车更有趣的想法之一。可悲的是，大多数人不喜欢巨型烤面包机的外观，而是购买了 CR-V。这个元素的特别之处在于内部空间。它很大，后座可以很容易地拆除或折叠起来，以获得更多的空间。宽敞的空间加上平坦的地板和用于装载的巧妙尾门，使它成为需要搬运大量装备但不想要商务车的人的完美之选。那是你，丹！

由于元素是基于 CR-V 使用本田无处不在的 2.4 升四缸发动机，这是一个令人难以置信的可靠的汽车，恢复了可观的燃油经济性。你附近的许多质量选项都有手动变速器，这可能会使它变得更糟，但如果你坚持要自动 [这里有一个在预算内](https://washingtondc.craigslist.org/nva/ctd/d/2003-honda-element-ex-sport/6471082591.html) 附近的一个可能有点问题的经销商。

## 专家 2:迈克尔·巴拉班是你今天该死的仙女教母

男孩哦男孩，丹。丹。今天是你的幸运日。你将得到一辆尼桑 S-Cargo。你只是还不知道而已。

日产 S-Cargo 是日产的“派克汽车”之一，很像我们尊敬的杰森·托尔钦斯基的 [日产 Pao](https://jalopnik.com/everything-thats-delightful-about-my-imported-1990-niss-1822341396) 。但 Pao 是一辆复古的城市车，S-Cargo 是一辆很棒的货车，自然看起来像一只蜗牛。显然，它是完美的。

它里面有大量的货物室，没有庞大和笨重的外观。它有各种各样的风格，从友好的外部车身到你会在车流中敲击的单辐雪铁龙式方向盘。

现在我知道，相信我，*我知道*，你不想要太“怪异”或“不可靠”的东西。但这货是辆尼桑，伙计。这没什么奇怪的。它的引擎和美国卖的一堆尼桑一样，所以如果它坏了，你可以在任何地方买到零件，尽管它不应该坏。

但如果你担心日产 S-Cargo 本身过于平淡，请放心，我的朋友。 [这里有一辆](http://www.duncanimports.com/used/Nissan/1989-Nissan-S-Cargo-e1c700b70a0e08f7472bc0953d69d898.htm) 待售，在弗吉尼亚州 [Gary Duncan 进口](https://jalopnik.com/have-your-mind-blown-by-the-most-amazing-secret-car-col-1792485778) 只有 44000 英里，配有拉力灯，蓝色轮毂盖，还有一顶帽子。要价有点超出你的预算，但加里是个讲道理的人。如果你不喜欢那一个，他有另外五个出售，其中几个甚至更便宜。

这是你的时刻，丹。抓住它。
