# 这个 USB-C iPhone 传闻太多了

> 原文：<https://gizmodo.com/this-usb-c-iphone-rumor-is-too-much-1826768618>

由DigiTimes[发布的一篇帖子引用了未透露姓名的“模拟 IC 供应商的消息来源”,他们说苹果可能会在 2019 年将 iPhone 和 iPad 转移到 USB-C。这一举动将意味着苹果专有的 Lightning 端口的逐渐消亡，该端口于 2012 年推出，自那以来一直是几乎所有 iProduct 的连接器。](https://mashable.com/2018/06/12/iphone-2019-usb-c-rumor/#hyjFPnRcvmqy)



那真是太酷了！想象一下，使用同一个插头给手机、平板电脑和笔记本电脑充电。想象一下，使用与笔记本电脑相同的 HDMI 转 USB-C 或 SD 卡转 USB-C 适配器，在手机上获取视频或照片。早在 2014 年宣布该标准时，我们就得到承诺，一个全 USB-C 世界，最大的手机和平板电脑制造商将它放入手机和平板电脑将是该标准变得无处不在所需的最后一次推动。

但是让我们现实一点...USB-C 可能不会很快出现在 iPhone 上。

苹果引领了笔记本电脑领域的 USB-C 充电。它是首批在 2015 年新款 MacBook 中加入这一功能的笔记本电脑制造商之一。它在 2016 年底掀起了波澜，新款 MacBook Pro 扼杀了几乎所有其他支持 USB-C 的端口。看看这一点，人们可以推断苹果接下来将在手机和平板电脑中采用该连接器，跟随当今几乎所有顶级安卓手机制造商的脚步。

首先，考虑到按照苹果的标准，闪电还是一个婴儿！FireWire 在苹果产品中存在了十二年。你抽屉里的那个 iPod 里的 30 针适配器已经用了 11 年了。闪电只有六岁左右。

但更重要的是，Lightning 是苹果的 [MFi 许可计划](https://developer.apple.com/programs/mfi/) 的关键部分。MFi 是为 iPod/iPhone/iPad 制造的缩写。为了获得出现在授权产品上的很酷的小标志，包括值得信赖的闪电电缆和其他配件，被授权者必须向苹果支付一些钱。这笔钱通常相当于产品建议零售价的 1.5%到 8%。但是在 2014 年， [根据这个 Quora 帖子](https://www.quora.com/Apple-company/What-is-the-royalty-Apple-charges-for-devices-that-use-its-30-pin-connector-for-iPhone-iPod-or-iPad) ，价格下降到每个连接器 4 美元。

这一固定费用尚未确定。为了加入 MFi 计划，潜在的被许可方必须签署一份禁止披露价格的 NDA。如果他们不签署，或者选择完全退出许可计划，制造商仍然可以生产 Lightning 产品，但它将缺乏认证，而 [正如我们在过去看到的](https://gizmodo.com/inside-the-chip-apple-uses-to-stop-you-buying-cheap-cab-5952093) ， [未能获得认证](https://www.macworld.co.uk/feature/apple/why-you-shouldnt-buy-cheap-iphone-cables-3661782/) 会导致一些 [讨厌的错误](https://www.cultofmac.com/246236/ios-7-killed-off-unlicensed-lightning-cables-but-heres-a-fix-that-might-work-for-you/) 出现在 iProducts 本身上。

因此，大多数著名的电缆和外围设备制造商都加入了 MFi。这意味着亚马逊每卖出 8 美元的 USB-A 转 Lightning 线 ，苹果就能从中分一杯羹。

还有一个事实是 [USB-C](https://gizmodo.com/usb-type-c-ive-never-been-so-excited-about-a-dumb-litt-1690839061) 还在继续...被 [种的一塌糊涂](https://gizmodo.com/cheap-usb-c-cables-could-kill-your-phone-or-laptop-1757115350) 。USB-C 产品没有认证程序。通常，如果不进行测试，甚至没有办法知道一根电缆可以处理多少数据或能量(这可能会损害这些电缆插入的任何 USB-C 端口)。这就是为什么许多手机和笔记本电脑制造商不得不在其产品上过度构建 USB-C 端口。他们正在防范一个堂兄，他在加油站买了他所有的电缆，然后奇怪为什么一个星期后它们就不工作了。

那么，苹果为什么要放弃它漂亮、安全、赚钱的端口，转而拥抱 USB-C 呢？特别是照明目前支持与 USB 3.0 相同的数据速率——USB-C 也支持这种数据速率。到目前为止，该公司没有真正的理由在其最受欢迎的产品上走出安全区。

当然，DigiTimes 可能是对的。MacBook 和 MacBook Pro 可能预示着苹果全 USB-C 的未来。但 2019 年感觉遥不可及。
