# 如何写大学申请的个人陈述

> 原文：<https://lifehacker.com/how-to-write-the-personal-statement-for-college-applica-1823014335>

如果你第一次看到你的高中生通过大学申请过程，你可能会发现这个过程的几个方面让他们像写个人陈述一样恐惧。

这是有道理的:大多数四年制大学要求的个人陈述询问了许多青少年，要求他们用 650 字或更少的话来证明是什么让他们与众不同，是什么因素将他们塑造成今天的样子。他们必须反思自己的生活，并写一部清晰、简洁的回忆录，这对大多数成年人来说都是一项挑战，更不用说 17 或 18 岁的孩子了。

但也有好的一面:个人陈述是大学申请中学生可以发挥创造力的地方。这也是他们完全可以控制的事情，不像他们的 GPA 或 SAT 分数。一份令人难忘的个人陈述可以让成绩不太好的学生在竞争中获得优势。以下是制作一个的方法。

### **头脑风暴**

作为一名大学顾问，我已经工作了几年，我发现对学生来说，最困难的部分是产生想法。“我没有什么可写的。“我从未遇到过有趣的事情，”这是一句常见的套话。

这可能不是事实，但是很难清楚地看到自己的生活，知道别人会发现什么是令人信服的。(在与一名声称生活无聊的学生交谈时，我发现她是在西藏难民营长大的。我告诉她，我认为这可能会引起大学招生人员的兴趣。)

但是，虽然损失和剧变显然是回忆录的素材，但一些最好的个人陈述集中在更小、更安静的经历上；诀窍是用风格和悬念来渲染它们。

头脑风暴是一个很好的开始。当学生感到困惑时，我让他们列出清单:我告诉他们写下五到十件他们生活中的事件，如果有人为他们写传记，这些事件必须包括在内。然后我让他们列出他们崇拜的人，以及他们经历失败和成功的时间。

同样有帮助的是请青少年描述他们自己，谈论他们的爱好和兴趣。(一名学生很快就认定自己是女权主义者；她还提到她是一名击剑运动员，曾不止一次在比赛中击败过自己的父亲。通过结合这两种激情，我们有了一篇关于学习击剑作为一种女权主义行为的文章，它如何赋予她力量和技巧来对抗父权制。)

通常，在谈论他们列出的事件、激情和目标时，学生们会偶然发现一篇文章的种子。当他们这样做时，看看 [常见的申请短文提示](http://www.commonapp.org/whats-appening/application-updates/2018-2019-common-application-essay-prompts) 会有帮助，并考虑他们的故事可能适合哪一个。幸运的是，这些提示是开放式的，即使学生申请的学校不采用通用应用程序，他们感兴趣的大学也可能会有非常相似的提示。(提示:避开最后一个提示，它让申请者选择提交一篇关于“任何主题”的论文我听说招生官员认为这是一种逃避。)

### **什么是好的个人陈述话题？**

谈到话题时，我尽量不做硬性规定，但有两个例外:如果学生对谈论这个话题感到不舒服，我会鼓励他们选择另一个话题；他们没有义务向一个不知名的管理员吐露心声，尤其是关于他们还没有处理的困难经历。

另一个要避免的话题是慈善旅行或志愿旅游经历，通过这些经历，学生发现了贫穷，并学会了不要把他们所拥有的视为理所当然。这是让大学知道他们不了解自己的经济特权(坦白地说，也不了解泡沫之外的世界)的最快方式。

她的校园 增加了任何政治争议话题、成就和催人泪下的故事的禁忌列表。作者所说的“催人泪下的故事”指的是围绕发生在作者*和*身上的悲剧事件的文章，而不是他们积极克服的困难。当一个作家把自己描绘成一个具有能动性和适应力的人时，它确实会成为一篇引人注目的文章，尽管我认为任何主题都可以成为一种个人陈述，如果它是有重点的，并精心制作成一个有开头、中间和结尾的故事。

变革的经历有助于引人入胜的阅读；我的一个学生写道，在他母亲从手术中恢复过来的时候，他承担了母亲的责任，为家人做(和烧)晚餐，在家庭作业之间跑去自助洗衣店，他加深了对为人父母的辛苦的欣赏。发现一种激情也可以是迷人的:另一个学生讲述了她去印度看望家人的经历，在那里她得知滴滴涕仍在使用；她对亲属健康的担忧激励她追求环境正义。另一个人写到了在中学被排斥的经历，这种经历并不少见，但她如此具体地描述了被欺凌的时刻，以至于我感受到了她的痛苦和她坚持下来的力量。

### **在页面上获取**

对于任何创造性的非小说类项目，一个好的起点是“大脑垃圾场”有一个强有力的想法并不意味着作者知道她将如何组织她的文章；通过把她的想法写在纸上，而没有写一些有条理和质朴的东西的压力，这种结构可能会开始出现。

另一个很好的指导原则是将小说的技巧应用到个人文章中:高中生不要概括地讲述他们的故事(“当我母亲生病时，我学到了关于责任的重要课程”)，而是应该写下他们在场景中的经历，通过包括详细的描述、对话、他们当时的想法和感受来展示。(例如，对于一篇关于生病的父母的文章，学生可以从父母告诉他生病的那一刻开始，描述他自己对此的想法和感受。)

个人陈述应该起到一种叙事的作用，让读者站在作者的立场上。我鼓励作家以一个钩子开始，一个象征他们故事主题的特定时刻。例如，写被欺负的学生以对话开头，描述了她无意中听到同学谈论她的一个瞬间。通常，一旦作家有了开头，文章的其余部分就从那里开始了。

因为个人陈述相对较短——通常只有几页——关键是作者要精挑细选，只包括服务于他们故事的细节和背景。文章不必是线性的——许多学生以一个场景开始，然后闪回以提供他们如何到达那里的背景——但它必须全部保持在一起。

结尾和开头一样重要，这也是学生最容易使用陈词滥调来描述他们是如何学习和成长的地方。但是最后的句子会让读者流连忘返，所以需要仔细选择。

精通这种写作形式的最好方法是阅读；幸运的是，如今个人文章比比皆是，所以鼓励你的孩子去寻找一些例子。一些学校还提供了他们喜欢的个人陈述的例子——从约翰霍普金斯 的这些例子开始。

### **修改**

和任何其他形式的写作一样，编辑是写作过程中至关重要的一部分。当我和学生们一起做这个的时候，我们会从宏观和微观两个层面来看他们的草稿，考虑组织、节奏和清晰度，以及单词选择和句子结构。

在修改阶段要考虑的问题是，这是否讲述了一个故事？它是否在一开始就抓住了我，并一直保持着我的兴趣？它以一种有意义且有影响力的方式结束了吗？它是否回答了其中一个提示？它是否向我展示了这个人作为个体的一些有趣之处？

注意重复和罗嗦。以复杂的语言为目标是好的，但简洁也很重要，当一个高中生使用他们不太熟悉的单词时，这一点通常很明显。归根结底，个人陈述应该是作者自己的声音，因为这是大学感兴趣听到的。

这可能需要一段时间——我看到一些学生修改了多达 20 次——这就是为什么在高三开始写个人陈述是个好主意，在高中学生被秋季的 SAT 考试和大学申请淹没之前。

岚娜·舒巴赫是一名自由作家、教师和大学顾问，他通过阳光大学准备项目和哥谭作家工作室指导高中生撰写个人陈述。
