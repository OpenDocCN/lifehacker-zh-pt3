# 这个感恩节计算器会告诉你每个人需要多少食物

> 原文：<https://lifehacker.com/this-thanksgiving-calculator-tells-you-exactly-how-much-1743784784>

如果你在本周四举办以火鸡为中心的大餐，你可能会对提供多少食物感到焦虑。这个计算器可以帮助解决这个问题。

Watch

只需输入您期望的客人数量，它就会给出您需要提供的相应数量的食物(和酒)。

[Open *kinja-labs.com*](http://kinja-labs.com/txgiving_planner/)

计算器的比率来自于 [美食网的](http://www.foodnetwork.com/thanksgiving/thanksgiving-entertaining/thanksgiving-dinner-portion-planner0.html) 部分规划文章，以及我的一些个人经验。

请注意，火鸡的数量与整只鸟、骨头和所有东西的重量有关，以便为每个人提供足够的肉。此外，我们对所有以分数结束的项目进行了四舍五入——多总比少好！毕竟谁不爱感恩节剩菜呢？
