# 你需要一个高科技 Dopp 套件

> 原文:[https://lifehacker.com/you-need-a-tech-dopp-kit-1821023553](https://lifehacker.com/you-need-a-tech-dopp-kit-1821023553)

你传统的 dopp 包( [或洗漱包](https://lifehacker.com/establish-a-permanent-toiletries-bag-for-easy-and-speed-5409595) )是一个有用的旅行伴侣。这是你放置牙刷、梳子、除臭剂或其他修饰物品的地方，这些物品是让你看起来不邋遢的必备物品。

你还需要为你的电子设备准备一套技术装备。tech dopp 套件集中了您在旅途中不可避免需要的所有加密狗、电缆和适配器。它让旅行变得更容易，因为你不必担心在酒店房间或机场的安全线打包和拆包时会丢失任何东西。这些都在工具箱里。

### **在你的科技工具包里放些什么**

在你把一件东西放进你的技术装备之前，你应该对你拥有的设备做一个评估，以及你可能需要它们做什么。如果你不打算在假期做任何演示，请随意将迷你 DisplayPort-to-HDMI 适配器留在家里。如果你在旅行中同时带着 iPhone 和 iPad，你可能需要包括一对与你的电池组和电源适配器兼容的 Lightning 电缆。

### **使用化妆包**

您的 tech dopp 套件不需要内置线缆管理器、多个小袋或松紧带。普通的洗漱包就可以了。你应该选择防水的洗漱包，以防万一你像我一样，会把水溅到背包里的每个电子设备上。

### **长(短)USB 电缆**

你的技术工具包应该放在你放 USB 线的地方。然而，不要带那种小小的三英尺长的，要带一些更长的六英尺长的电缆。当你在旅行的时候，你不可避免地会使用不在最方便位置的插座。当最近的插座在房间的另一边时，多出来的几英尺真的会很有用。你还应该包括 [一些非常短的 USB 电缆](https://www.amazon.com/AmazonBasics-Lightning-USB-Cable-Centimeters/dp/B010U3XA0I/ref=sr_1_3?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/you-need-a-tech-dopp-kit-1821023553&asc_source=&ie=UTF8&keywords=short lightning cable&qid=1512505834&sr=8-3&tag=kinjalifehackerlink-20) ，用于将设备连接到笔记本电脑之类的东西。你可以找到短至 6 英寸的电缆。

### **别忘了 Velcro 领带**

将线缆放入工具包时，需要整理它们吗？附上并切割成合适的尺寸 [一些 velcro 领带](https://www.amazon.com/VELCRO-Brand-Black-2-Inch-91140/dp/B001E1Y5O6/ref=sr_1_1_sspa?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/you-need-a-tech-dopp-kit-1821023553&asc_source=&ie=UTF8&keywords=velcro ties&psc=1&qid=1512505811&sr=8-1-spons&tag=kinjalifehackerlink-20) 你可以用它们来保持杂乱的线缆整齐有序。

### **USB-C(和 USB-A)适配器**

在您的技术套件中包括 USB-C 和 [USB-A 适配器](https://www.amazon.com/Adapter-Syntech-Laptops-Chargers-Interface/dp/B0749Q9Y4N/ref=sr_1_13?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/you-need-a-tech-dopp-kit-1821023553&asc_source=&ie=UTF8&keywords=usb-a to usb-c&qid=1512505445&sr=8-13&tag=kinjalifehackerlink-20) 意味着遇到全新的硬件不会阻止您使用 USB-A 电缆给它充电或连接。它们很小，很便宜( [Anker 的两包](https://www.amazon.com/AUKEY-Adapter-Samsung-MacBook-Google/dp/B01AUKU1OO/ref=sr_1_17?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/you-need-a-tech-dopp-kit-1821023553&asc_source=&ie=UTF8&keywords=usb-c adapter&qid=1512505205&sr=8-17&tag=kinjalifehackerlink-20) 售价约为 7 美元)，并且将延长你已经拥有的电缆的寿命，因为你不需要更换它们。有些型号可以直接插入您的设备，看起来像带有母 USB 端口的闪存驱动器。

如果你选择购买小型的无线适配器，你应该把它们放在另一个包里，比如零钱包。没有人喜欢到处寻找零钱大小的东西。

### **电池组**

当然，你应该在你的技术装备中包括一个电池组(尤其是如果你没有那些装有电池的奇特行李箱的话)。您的套件中包含哪种电池组取决于您要充电的内容。有些包括 [内置的 Lightning 或微型 USB 线](https://www.amazon.com/dp/B01A6L85CC/?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/you-need-a-tech-dopp-kit-1821023553&asc_source=&linkCode=xm2&tag=kinjalifehackerlink-20) 这样当你需要给手机充电时，你就不需要考虑把线放在哪里了。如果你的技术更新，你也可以得到配备 USB-C 的电池组。

### **蓝牙跟踪器**

您可能不想丢失装有保持连接和在线所需的加密狗和电缆的袋子。如果你的包里还有空间，可以放入一个像 [磁贴跟踪器](http://gear.lifehacker.com/the-waterproof-tile-pro-series-solves-almost-all-of-the-1797755839) 这样的装置来照看你的包。该公司最新的追踪器 Tile Pro 使用蓝牙 le 在 200 英尺的范围内连接到你的手机。丢了你的工具箱？报告您的图块丢失，该区域的其他图块用户将自动找到并向您报告您的套件位置。

### **双端口电源适配器**

建议使用双端口电源适配器，这样您就可以充电，而不必选择哪个设备应该充电。只需为您的主要设备(如智能手机)和便携式电池组充电。每个人都赢了。
