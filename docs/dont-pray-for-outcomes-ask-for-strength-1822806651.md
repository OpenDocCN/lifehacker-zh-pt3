# 不要祈求结果，要祈求力量

> 原文：<https://lifehacker.com/dont-pray-for-outcomes-ask-for-strength-1822806651>

欢迎回到 [**周中冥想**](http://lifehacker.com/tag/mid-week-meditations#_ga=2.170929957.440406047.1499703745-27973805.1434581949) ，Lifehacker 的每周一次的斯多葛派智慧之池探索，以及使用它的水域来反思和改善你的生活的指南。



本周的精选来自马库斯·奥勒留。在 9.40 的*冥想*中，他建议我们停止祈祷我们希望发生的事情，从不同的角度解决我们的问题:



一个男人祈祷道:“我怎么能和那个女人睡觉？”你的祈祷是:‘我怎么能失去和她睡觉的欲望？’另一个人祈祷:“我怎样才能摆脱那个人？”你祈祷:‘我怎样才能停止想要摆脱他？’另一个:“我怎么才能救我的小孩？”你:“我怎样才能学会不害怕失去他？”诸如此类。把你所有的祈祷都给我，然后观察会发生什么。



下面是另一个翻译:



试着用不同的方式祈祷，看看会发生什么:不要问“和她睡觉的方法”，试着问“停止和她睡觉的方法”。不要问“摆脱他的方法”，而要问“不渴望他死亡的方法”。不要问“一种不失去孩子的方法”，而要问“一种消除我对孩子的恐惧的方法”。



### 这意味着什么

不要祈祷你生活中的某些结果，而是祈祷从欲望、愤怒、嫉妒和恐惧中解脱出来，这样你就可以处理任何出现在你面前的事情。生活中的许多麻烦都超出了我们的控制范围，所以不要再抱有希望，要掌控自己的生活。

### 从中吸取什么

即使你不祈求上帝，我们都会以这样或那样的方式祈求结果。我们都轻声说“请…”在需要的时候，希望幸运会眷顾我们。但不管是不是神，奥勒留相信，我们最好掌握主动权，要求我们能控制的事情。他说:

> 但是你可能会说:‘神已经把这些东西放在我自己的力量里了。’那么，在自由中运用你自己的力量，而不是对你无法控制的事情表现出一种卑躬屈膝、无所事事的关心，不是更好吗？

事实上，当你祈求力量、韧性和驾驭生活风暴的能力时，你在召唤自己，就像你在召唤一个看不见的神一样(“把你所有的祈祷都给这一次，然后观察会发生什么”)。你的要求从被动的等待转变为主动的。你开始想办法帮助你的祷告。所以，不要去要求更好的生活，要去要求让生活变得更好的意志。
