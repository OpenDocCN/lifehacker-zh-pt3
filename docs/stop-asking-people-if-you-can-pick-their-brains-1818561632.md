# 不要再问别人你是否能“集思广益”

> 原文：<https://lifehacker.com/stop-asking-people-if-you-can-pick-their-brains-1818561632>

现在和名人联系非常容易，不是吗？在社交媒体或公司简历上点击几下，你就可以向你所在领域的最新摇滚明星发送一条信息，这个人正在创造、破坏和重新发明从公共汽车到酒店的一切。你希望见到这位摇滚明星本人——也许真诚地收集对你自己的职业轨迹有用的信息，或者也许只是把你的脸放在他们面前，希望有一些有意义的职业联系——但有机地发生的可能性很小。所以你把自己定位成一个求知欲强的学习者，一个恳求者，并问“能一起喝杯咖啡吗，我可以向你请教一下？”

Basecamp 的联合创始人兼首席执行官杰森·弗里德在 LinkedIn 的 [上说，不要这样做。“集思广益”这几个字让人很难说*是的*。](https://www.linkedin.com/pulse/few-words-advice-brain-pickers-jason-fried/)

他指出，首先，智囊团从中得到了什么？这仅仅是一个呼吁，而不是同事之间互利的会议。他建议考虑一下你能提供什么，把会议设计成更多的交换:“我想问你几个关于 X、Y、Z 的问题，同时分享一下我对 A、B 和 c 的看法。”

我还要补充一点，如果你的职业生涯刚刚开始，而对方已经表示愿意承担导师的角色，那么要求一次信息性面试是可以的，但如果他们说他们只是没有时间，也不要沮丧。

弗里德还补充道，潜在的脑力劳动者应该提前告知所需的时间——没有 15 分钟的咖啡这种东西，所以不要假装这是少于 20-30 分钟的承诺。并主动提出在他们方便时候去找他们。

那么，一个渴望社交的人应该做些什么呢？有趣的问题，沃顿商学院管理和心理学教授 Adam Grant 在《纽约时报》[](https://www.nytimes.com/2017/08/24/opinion/sunday/networking-connections-business.html?_r=0)*中提出了一个问题。他指出，社交——在派对上向不认识的人发飙，发送冷冰冰的电子邮件，诸如此类——作为一种策略，完全被高估了。有什么*作用？做一些值得注意的事情，即使是小规模的，并通过成为他们领域中合法的新兴玩家来吸引摇滚明星的注意。**

*在某种程度上，所有这些都是一种安慰:你不一定要成为一个擅长社交、积极交际的人才能获得成功——你需要做好工作。做得足够多，很快人们就会要求挑选你的大脑。*
