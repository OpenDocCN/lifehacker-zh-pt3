# 这个应用程序会让你知道你是否赢得了强力球头奖

> 原文：<https://lifehacker.com/this-app-will-let-you-know-if-youve-won-the-powerball-j-1797783214>

我通常不玩强力球，但每当累积奖金超高时，我(像地球上的其他人一样)就会去商店买一两张票。今晚的强力球头奖是 3.56 亿美元，这意味着如果你是一个赌徒(或者即使你不是)，也没有像现在这样的时间去抢票。

Watch

如果你对前往当地加油站加油不感兴趣，现在有一款名为 [AutoLotto](http://autolotto.com/) 的应用可以在你的手机上处理整个过程。它还会让你知道你什么时候用你买的那些实物票赢了。

有了它，你可以购买一张彩票，并在你的手机上知道你是否中奖。要购买，你需要在应用程序中注册，然后允许应用程序对你的手机进行地理定位(确定你在哪个州购买彩票)。目前，在该应用内购买仅限于新罕布什尔州，这里是彩票的故乡，也是 AutoLotto 的办公地点。预计很快会有更多的州加入进来。

只有一个州的原因是彩票法律要求该应用程序在其出售彩票的任何地方都有一个办公室。

如果你是大多数人，不能在你所在州的应用程序中直接购买门票，你仍然可以扫描你购买的门票，如果你赢了，会收到通知。

如果你买了几十张票，一旦号码公布后，你就可以更容易地浏览它们了；尽管有 3 . 56 亿美元悬而未决，我想我还是会在扔掉之前仔细检查我所有的彩票。
