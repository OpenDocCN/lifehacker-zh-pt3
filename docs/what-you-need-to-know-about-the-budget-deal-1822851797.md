# 关于预算交易，你需要知道什么

> 原文：<https://lifehacker.com/what-you-need-to-know-about-the-budget-deal-1822851797>

肯塔基州参议员兰德·保罗为您带来了政府短暂关闭后，国会在周五早上设法通过了一项预算协议——这是三周内的第二次关闭。特朗普总统 [签署法案](https://www.wsj.com/articles/house-passes-two-year-budget-deal-1518172770?mod=e2tw) 后不久。



据《纽约时报》报道，为期两年的协议将国内和军事支出上限提高了 3000 亿美元， [协议将提高联邦债务上限](https://www.nytimes.com/2018/02/08/us/politics/congress-budget-deal-vote.html?hp&action=click&pgtype=Homepage&clickSource=story-heading&module=span-ab-top-region&region=top-news&WT.nav=top-news) 至 2019 年 3 月。它让联邦政府获得了六周的资金，所以我们可以一起回来，在不到两个月的时间里再做一次。

以下是你需要知道的。

### 兰德·保罗为什么关闭政府？

保罗抗议增加数千亿美元的联邦赤字，并希望就维持预算上限的修正案进行投票，但遭到了领导人的拒绝。在周四参议院的一次广泛的演讲中，保罗抨击了政府的浪费，特别是国家科学基金会和阿富汗的开支。目前还不清楚为什么当保罗投票支持最近的税收法案 时，这个增加赤字的法案如此激怒他，该法案预计将在十年内增加大约 1.4 万亿美元的赤字。

### 账单里有什么？

预算更像是一个粗略的大纲，分为大约 1650 亿美元用于军事，1310 亿美元用于国内事务:为唯禁欲教育提供资金，60 亿美元用于对抗阿片类药物危机的研究和预防项目，以及 800 多亿美元用于佛罗里达州、波多黎各和德克萨斯州的救灾。

也许是因为立法者意识到他们不应该将儿童健康保险作为一种的讨价还价工具，至少在一段时间内，它还将儿童健康保险计划的资金额外延长了四年(这意味着在国会于 1 月底 同意延长六年后，CHIP 将获得十年的资金)。立法者将制定更多的确切支出细节(换句话说，实际上是起草立法)，并将在 3 月底再次投票。

该协议没有解决移民问题，这激怒了一些民主党人，包括众议院少数党领袖南希·佩洛西，她周三在众议院就童年抵美移民项目的延期行动发表了八个多小时的讲话，并投票反对该法案。民主党人希望立法“确保保护被称为梦想家的年轻非法移民，作为任何预算措施的一部分，但共和党人拒绝了，” [路透社报道](https://www.reuters.com/article/us-usa-congress-shutdown/congress-to-vote-on-long-term-budget-deal-averting-shutdown-idUSKBN1FS232) 。"取而代之的是，他们承诺本月在参议院举行一次单独的移民辩论."

财政保守派反对支出水平。“随着这个支出方案的通过，我担心共和党人已经放弃了我们领导我们的国家远离最终国家破产的道德权威。得克萨斯州众议员杰布·亨萨林 说。

幸运的是，这项交易为赛马所有者和汽车运动娱乐公司保留了一项特殊的税收优惠 。因此，尽管年轻移民可能需要继续担心他们在美国的未来，但所有中产阶级赛马主今晚可以高枕无忧了。
