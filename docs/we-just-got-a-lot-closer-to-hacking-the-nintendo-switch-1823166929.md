# 我们离把任天堂 Switch 黑进一台能用的平板电脑又近了一步

> 原文：<https://lifehacker.com/we-just-got-a-lot-closer-to-hacking-the-nintendo-switch-1823166929>

[【任天堂 Switch】](https://lifehacker.com/if-you-hate-joy-cons-use-an-old-gamecube-controller-on-1819836689)是一款很棒的游戏设备，但(除了流媒体 [Hulu](https://gizmodo.com/its-about-time-the-nintendo-switch-got-a-real-streaming-1820292731) )它没什么别的功能。谢天谢地，如果你愿意黑掉你 300 美元的平板电脑，这种情况可能不会持续太久。



黑客集体 fail0verflow 本周末发布了一段视频，展示了一个运行 [Linux 操作系统](https://lifehacker.com/how-to-get-started-with-the-linux-operating-system-1819644874) 的改良版任天堂 Switch。长达一分钟的演示展示了一个工作桌面、一个功能浏览器和触摸屏支持。这是它实际运行的样子:

 [https://lifehacker.com/embed/inset/iframe?id=twitter-964954316892119040&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-964954316892119040&autosize=1) 

在另一条推特上，黑客解释说，他们正在利用的漏洞无法用软件更新来修补。这意味着世界上已经存在的任何交换机硬件都可以被破解来运行 Linux。你也不需要一个 [modchip](https://kotaku.com/working-playstation-3-modchip-is-not-a-hoax-australian-5616952) (一个额外的电子设备，通常用来改变游戏控制台)来入侵交换机。

不幸的是，fail0verflow 实际上并没有透露黑客是如何工作的，除非他们透露(或其他黑客发现)，否则你只能按照任天堂的意图使用开关。一旦这些信息被泄露，你就可以把它变成一个普通的平板电脑，用它做任何事情，从查看 Twitter 到玩任天堂官方不支持的游戏(你也可以下载偷来的游戏，但这取决于你和你的良心)。

潜在的不利因素是，你可能最终会弄坏你的开关。一旦设备遭到黑客攻击，任天堂很可能不会履行保修义务，所以在你等待 fail0verflow 发布关于如何将 Switch 变成 Linux 平板电脑的官方指令时，这是值得考虑的事情。
