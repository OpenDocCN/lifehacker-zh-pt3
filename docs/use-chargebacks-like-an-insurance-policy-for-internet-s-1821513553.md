# 像网上购物的保险单一样使用退款

> 原文：<https://lifehacker.com/use-chargebacks-like-an-insurance-policy-for-internet-s-1821513553>

于是你从点了东西。或者阿里巴巴。或者一些你从未听说过的粗略网站，因为它有你想要的运动鞋，而其他地方没有。你知道这可能是一个好得难以置信的信号…但你真的很想要这双运动鞋，所以你还是愚蠢地输入了你的签证号码。

Watch

当箱子到达时，运动鞋的尺寸或颜色都不对，或者完全是山寨货。当你试图联系该公司时，它要么 A)没有回应，B)拒绝退款，或者 C)说你负责支付双向运费。按照我们大多数妈妈的说法，那就是你在网上买东西得到的东西。

但是正如一位 Reddit 用户最近指出的，你的信用卡公司可以帮你把钱拿回来。怎么会？以拒付的形式，当你与信用卡商家本身对账单上的费用有争议时发生。

例如，Reddit 用户在网上购买了一件有缺陷的商品。在试图退货时，商家说他们必须支付退货运费和 20%的进货费。当用户联系花旗银行启动退款时，信用卡发行商一再表示这是不可能的。

幸运的是，用户知道他们的权利，并最终坚持下来。以下是他们的 TL/DR 情况:

“如果您购买了一件商品，而卖家发送的商品与描述明显不同或有缺陷，您有权拒绝该商品，并要求卖家取回该商品，费用由他们承担——无论卖家的退货政策如何规定。”

如果你和商家之间没有任何进展，你的信用卡公司就需要代表你来施加影响。

基本上，如果您收到有缺陷的、损坏的、与描述不符的商品或服务，或者您根本没有收到它们，您可以请求退款。(点击此处查看按发卡行组织的 [退款原因代码](https://www.merchantconnect.com/CWRWeb/pdf/CBRReasonCodeListUS0908.pdf) 列表。)

请注意，商家有权对你的索赔提出异议，之后由信用卡发行商决定谁是对的。如果它站在你一边，商家必须偿还费用，外加手续费；如果它站在商家一边，费用会回到你的账单上。

你最近犯的网络错误符合条件吗？然后按照的建议联系你的信用卡公司:

“理想情况下，所有信用卡退款应在您收到有问题的账单后的 60 天内进行，这样您就可以获得《公平信用计费法案》的全面保护。只能通过电子信息(通过银行的在线支持平台)或邮件要求退款。”

尽管如此，这一政策应该只是在商家拒绝满足你的(合理的)要求的情况下的最后手段。因为，正如 Redditors 警告的那样，请求退款可能会让你被商家禁止——如果出于某种原因，你还想在那里购物的话——如果你真的不需要它，这是完全不酷的。反复要求退款也可能会损害你与你的信用卡公司 的关系。

明智地使用你的力量，朋友们。
