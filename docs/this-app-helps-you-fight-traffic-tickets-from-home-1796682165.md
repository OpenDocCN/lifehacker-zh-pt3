# 这个应用程序可以帮助你在家与交通罚单作斗争

> 原文：<https://lifehacker.com/this-app-helps-you-fight-traffic-tickets-from-home-1796682165>

你超速行驶了五英里，这时你注意到后面有蓝色的闪光灯。一名警察让你靠边停车，给了你一张超速罚单。你可以像大多数人一样，忍气吞声，付罚单，或者去法院提出异议。幸运的是，有一个应用程序可以让你足不出户就能买到票。

Watch

[场外记录](https://itunes.apple.com/us/app/off-the-record-fight-your-traffic-ticket/id1032930471?mt=8) 是一款 [iPhone](https://itunes.apple.com/us/app/off-the-record-fight-your-traffic-ticket/id1032930471?mt=8) 和 [web](https://offtherecord.com) app 平均成功率 97%，据 [GeekWire](https://www.geekwire.com/2016/off-record-speeding-ticket-app/) 。要使用这款应用，用户需要给自己的车票拍张照片，回答几个简短的问题，然后付费。费用从 59 美元到 599 美元不等，取决于你所在的州和县。然后，该应用将案件分配给附近的律师，为用户进行辩护。这些律师是根据他们在法庭上的成功率被审查的。用户可以通过该应用程序发送消息并接收律师的更新，而不必亲自与律师见面或出庭。

目前，该应用程序只处理小的违规行为，如稍微超速和闯停车标志。这些罚款大多在 150 美元左右，但应用程序的开发者表示，支付这笔费用是值得的，以避免多年来保险费的增加。但是，如果这张票确实留在你的保险记录上，这个应用程序会退还你费用(除了新泽西以外的每个州)。

但重要的是要注意，大多数轻微超速罚单不会提高你的保险费率 ，所以聘请律师来争论费用可能并不总是值得的。另外，你可以随时 [不请律师](https://thegarage.jalopnik.com/how-to-fight-a-traffic-ticket-without-an-attorney-1722205649#_ga=2.176104103.480883058.1499263406-1887250931.1497314809) 打交通罚单。如果你想知道一张罚单会如何影响你的保险费率，你应该打电话给你的汽车保险公司询问。

根据该应用程序，81%的情况下，律师会放弃罚单，16%的情况下，罚单会减少到非移动违规，这意味着你的保险费率不会增加，但你仍然需要支付罚款。

这项服务只在 18 个州可用，但是当它到达你的城市时，你可以 [注册电子邮件更新](https://offtherecord.com) 。也有其他应用程序可以让你轻松聘请律师处理交通罚单纠纷，如 [Legal Tap](http://app.legaltap.co/faq/) (进行 15 分钟的咨询)、[get dissed](https://itunes.apple.com/us/app/getdismissed/id955145724?ls=1&mt=8)(针对加州司机)，以及 [WinIt](http://www.appwinit.com/faq/parking-tickets) (针对纽约市的司机)。但是，如果你发现同样的服务有一个更好的价格，非记录将匹配它。
