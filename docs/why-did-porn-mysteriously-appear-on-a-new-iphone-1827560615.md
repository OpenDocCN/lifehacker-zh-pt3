*啊，鬼片。可怕的东西。但是严肃地说，这有点奇怪——我相信你没有无意中点击任何东西。(我这么说不是为了油嘴滑舌；我只是想建立基本规则。)*

*我的第一反应是，某个成人网站一定是在另一台设备的浏览器上打开的，这台设备与你的新 iPhone 链接到同一个 Apple ID。但是即使你在 iOS 上打开 Safari，你也不得不 [挖掘一下](https://support.apple.com/en-us/ht202530) 才能从其他系统访问打开的标签。谷歌 Chrome 这样的第三方浏览器也是如此。你在其他系统上打开的标签页不会在你第一次加载浏览器时神奇地出现。所以这可能是不可能的。*

*我怀疑你的 iPhone 已经感染了任何类型的恶意软件，因为这并不是大多数 iPhone 用户必须处理的事情——对于刚刚擦干净的 iPhone 7 来说，这也不是一个问题。*

*我唯一的其他想法，除了明显的解释，也许是你访问的某个网站 [归咎于](https://support.apple.com/en-us/ht203987) ，是 [你的路由器的 DNS 设置](https://www.avg.com/en/signal/what-is-dns-hijacking) 可能以某种方式改变了，这导致了不想要的弹出窗口(或重定向到你不打算去的成人网站)。我承认，这有点冒险，但值得一试——至少换一个比你的 ISP 更快的 DNS 提供商。*

*首先打开路由器的配置页面。如果你不知道怎么做，你可以随时查阅你的路由器手册。或者只需遵循以下快速建议(对于非网状网路由器，您可能只需使用智能手机应用程序访问):*

*   *在 Windows 系统上，打开命令提示符并输入`ipconfig`。被列为默认网关的 IP 地址可能是您路由器的 IP 地址。*
*   *如果你用的是 Mac，打开系统偏好设置>网络，点击右下角的高级。单击下一个窗口顶部的 TCP/IP 选项，并查找您路由器的 IP 地址。* 
*   *如果你的 iPhone 上有幽灵色情，点击设置，然后 Wi-Fi，点击你连接的 wifi 网络旁边的“I”图标。您的路由器的 IP 地址应该就在那里。*

*一旦你有了路由器的 IP 地址，在你的浏览器中打开一个新的标签，在地址栏中输入 IP 地址，然后点击回车。登录您的路由器——如果您不知道如何登录，它的默认用户名和密码可能印在它的手册上或路由器本身上，可能印在它的背面或下面。*

*一旦你在你的路由器的设置，你有一些事情要注意。首先，更新路由器的固件。在路由器配置屏幕的菜单中可能会有一个选项；它可能被埋在某个地方。您可能还需要访问路由器制造商的网站，找到正确的支持页面，下载最新的固件以便安装。这可能是一个恼人的过程，但并不费力。*

*一旦你做到了这一点，去改变你用来访问你的路由器的登录凭证——特别是如果他们仍然是“管理/管理”或“管理/密码”的默认设置。不，不，不。选择一个唯一的用户名和密码。*

*接下来，查看路由器的设置，看看在哪里可以编辑它的 DNS 配置。这可能被设置为某种“自动获取”设置，但如果它是一个随机出现的 IP 地址，那(可能)就不好了。无论哪种方式，你都应该将其主要和辅助 DNS 切换到一个更好的选项，如 [谷歌 DNS](https://developers.google.com/speed/public-dns/) 、 [Cloudflare](https://1.1.1.1/) 或[OpenDNS](https://www.opendns.com/setupguide/#familyshield)——指令(以及你想使用的 IP 地址)都是超链接的。*

*最后，查看路由器的设置，寻找任何“远程管理”或“远程管理”选项。如果你找到一个， [轻弹它](https://lifehacker.com/the-most-important-security-settings-to-change-on-your-1573958554)——在这个短语的两个意思中。你永远不会太确定。*
