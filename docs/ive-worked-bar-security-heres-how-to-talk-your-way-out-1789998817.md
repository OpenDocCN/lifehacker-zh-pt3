# 我做过酒吧保安，下面教你如何在酒吧打架中脱身

> 原文：<https://lifehacker.com/ive-worked-bar-security-heres-how-to-talk-your-way-out-1789998817>

孙子曾经说过“不战而屈人之兵”当饮料流动时，这个数字会翻倍。人们喝醉时会有点头脑发热，在某些情况下，还会动用身体。根据我作为一名保安的经验，这些圆滑的谈话技巧将帮助你平静一个紧张的局面，并逃脱的潜在打击。



### **发展你的仇恨“蜘蛛感应”**

从争吵中脱身的最好方法是从避免可能性开始。这意味着 [发展你的情境意识](https://lifehacker.com/develop-your-situational-awareness-with-the-ooda-loop-1685050030) 并开动脑筋。正如劳伦斯·凯恩和克里斯·怀尔德在他们的书《如何赢得一场战斗:避免和幸存暴力指南》中解释的那样，酒吧斗殴往往发生在晚上某个时间之后

> 重要的不是时钟上的时间，而是人群的情绪。大多数人玩得很开心，早在屎开始前就离开了。几乎所有想勾搭的人都已经勾搭上了，开始享受生活。随着人群开始变得稀薄，剩下的就是那些除了制造麻烦之外无事可做的人。带着沮丧和愤怒的荷尔蒙，那些坚持到深夜的人是那些陷入暴力的人。

环顾酒吧，感受一下气氛。酒吧里有没有人在沉思？有没有一个女孩大声愤怒地向她的朋友发泄？你能听到有人试图和别人挑起事端吗？如果是这样，也许是时候找另一家酒吧或者回家了。在我为音乐会场地、酒吧和俱乐部做保安的经历中，夜晚总是会有一个转折点，从有趣变成混乱。学会感受那个转折点，在事情变得糟糕之前离开。

### **保持冷静，伸出双手，表明你的意图**

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-_w4K5aB3e38&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-_w4K5aB3e38&start=0) 

如果你不能在激怒某人之前发泄出来，而他们又在你面前出现，抵制任何与他们的能量相匹配的冲动。如果他们侮辱你，不要回敬他们。如果他们推你，不要推回去。如果他们盯着你看，不要瞪着他们。一旦你回报同样的能量，事情很可能升级。这种情况与自尊无关。你的目标是继续你的夜晚，安全回家，所以抛弃自我。

相反，按照这个顺序做这三件事:深呼吸，把你的手放在你面前，掌心对着它们，并明确表示你对打架没有任何兴趣。深呼吸会帮助你保持冷静，避免与他们的强度水平相匹配。正如杰伊·费拉里在《男性健康》杂志上解释的那样，举起手来表明你不准备接受他们的争吵，如果需要的话，则表示你准备好了为自己辩护。自卫专家称之为“栅栏”——在上面的视频中演示过——这个过程也是在你和教唆者之间创造空间的绝佳时间。

一旦你竖起了篱笆，你必须告诉他们你不想惹麻烦。你越早说越好。在 [*武术揭示:好处、问题和解决方案*](https://books.google.com/books?id=bvf5dzjHaHEC&pg=PA14&lpg=PA14&dq=talk+your+way+out+of+a+fight&source=bl&ots=lWvabYKL_J&sig=49u2UJwlrOrnrM-I9xmaHV1Vmao&hl=en&sa=X&ved=0ahUKEwjuge_HgOPQAhUoAcAKHY8sBDM4ChDoAQhKMAk#v=onepage&q=talk%20your%20way%20out%20of%20a%20fight&f=false) 中，作者杰米·西布鲁克解释了原因:

> 被激怒时让几分钟过去了会招致灾难。这是因为潜在的攻击者可能会认为你在“玩弄”他/她，如果你真的对战斗不感兴趣，你就会说出来。此外，当别人骂你并怂恿你时，你更容易感到不安和沮丧。当这种情况发生时，自然的反应是报复。

Seabrook 还指出，在这种情况下，你说的话相当重要。说一些像“退后，失败者”这样的话是没有用的，所以要明智地选择你的措辞。平静而清晰地说话，这样他们就不会把你说的话理解为任何形式的侵犯。准备好被叫出去并被怂恿。不要上钩，享受你剩下的夜晚。

### **道歉并介绍自己**

如果说你不想打架还不足以让他们冷静下来，那么是时候开始唤起他们的人性了。他们可能是一个充满酒精和愤怒的球，但在内心的某个地方仍然有一个理性的人。以道歉开始，即使在相遇之前发生的一切都是他们的错。他们觉得你在某种程度上让他们受了委屈，所以你只要让步，说声对不起，他们就不会来烦你了。 [你不需要为每一件小事道歉](https://lifehacker.com/how-to-stop-apologizing-for-everything-you-do-1746770762) ，但当有人处于战争状态时，他们是不会动摇的。你必须做出选择:你是想安全地做错事，还是做正确的事。

在你道歉后，他们可能会劈头盖脸地侮辱你，让你自生自灭。如果没有，扔给他们一个弧线球，像一个文明人一样介绍自己。这做了两件事:它提醒他们你是一个真实的人，而不是一个温暖的出气筒，它让他们措手不及。你扔给他们的扳手改变了他们的心态，让他们不断升级的脾气嘎然而止。他们认为你会生气并和他们打架，但是你却对他们表示尊重并和他们谈判。他们可能对真正了解你没有任何兴趣，但你会“破坏气氛”，他们可能会转移到更令人兴奋的事情(或人)上。

### 将他们的攻击转变为同情或困惑

同情是暗示的有力工具，即使你在撒谎。告诉煽动者你正在度过一个艰难的夜晚，你刚刚和你的男朋友或女朋友分手，你的姑婆萨莉刚刚去世，你的狗失踪了，你刚刚被解雇，你患有癌症，或者你能想到的任何其他事情来吸引他们的同情。如果你能卖出哪怕一点点，他们很可能会放过你。没人想用一只丢失的狗来痛打一个失业的癌症患者。

困惑也是一个强大的工具。在他的博客 上，空手道教练杰西·恩坎普 [分享了他从魔术师达伦·布朗](http://www.karatebyjesse.com/one-little-verbal-trick-to-save-your-a-in-self-defense/) 那里听到的一个技巧，涉及到对潜在的攻击者说一些奇怪和不合适的话。布朗仅仅通过说“我房子外面的墙没有四英尺高”就避免了一场打斗。袭击他的人困惑地停下来，问了一句“什么？”当布朗开始谈论西班牙的其他城墙时，他最终放弃了:

> 这个人必须弄明白我在说什么，而在这样做的过程中，他变得非常困惑。通过向他提供更多同样的东西(谈论西班牙)，他觉得自己可能会从困惑中得到一些缓解，但这样的澄清并没有出现。他方寸大乱，迷茫，不再掌控。他经历了“肾上腺素分泌过剩”，这让他变得一无是处。在我告诉他西班牙城墙的情况后，我补充道，“但是在这里，它们太小了！看看这些！”我指着我们旁边花园周围的一堵三砖高的小墙。他看着墙，那个动作告诉我，我现在占了上风。他回头看着我，此时他自己也有些萎靡不振，发出一声长长的"哦，呜-呜-呜-呜……"然后瘫倒在地，绝望地垂下了头。

这一招和自我介绍一脉相承。你用一些他们意想不到的东西给他们惊喜，他们别无选择，只能让他们的大脑去思考你到底在说什么。这可能看起来有点傻，但最终，一旦肾上腺素消失，战斗的欲望也就消失了。

### **提醒侵犯者为什么你们两个都在那里*也许*会请他们喝一杯**

与一个喝醉的、愤怒的人讲道理很少奏效，除非你把它变得简单，并专注于享受美好时光。在你为引起问题的原因道歉之后，提醒煽动者你们两个都是来找乐子的——大概是和和你一起来的人——你们两个都应该回去享受美好时光。向他们解释不管谁赢，你们都会毁了你们的夜晚，因为你们都会被踢出去，或者受到伤害，或者被逮捕，或者以上所有的事情。把一切都告诉他们，并建议你们两个都不要打扰对方。

当我还是一名保安的时候，这一招几乎总是奏效，我每天晚上都要面对人们的争吵。如果你真的想达成交易，请他们喝一杯或者和你一起喝一杯。当然，如果你的“嘿，别这样，我们不都是来找乐子的吗？”问题用“不，我是来战斗的”来回答，你只有 [一个另外的选项](https://lifehacker.com/basic-self-defense-moves-anyone-can-do-and-everyone-sh-5825528) 。

### **sInform Security 或另寻酒吧**

当所有的话对煽动者都不起作用时，就用在员工身上。告诉保安、保镖或酒保有人威胁你，煽动者会很快被赶走。酒吧员工不希望在他们的场所打架，他们会毫不犹豫地把他们扔出去。

这可能不太顺利，但做一个喜欢打小报告的人，总比做一个被揍得屁滚尿流的想要成为硬汉的人要好。如果他们没被赶出去，就去找另一家酒吧。最好是关心顾客安全的。
