# 本周 Windows 应用程序:WinDirStat

> 原文：<https://lifehacker.com/windows-app-of-the-week-windirstat-1825098404>

这几乎是一种逃避，因为现在所有的 Windows 用户*都应该*知道 WinDirStat。这款应用已经问世近 15 年了，它是你可以用来清理硬盘的最有用的工具之一。

Lifehacker 非常重视春季大扫除。我们绝不会错过更新、重组和整理我们家庭生活的机会。我们也非常兴奋地按下科技使用的重置按钮，仔细查看我们的财务状况，并抛弃已经有点发霉的日常习惯。欢迎来到春季大扫除周，在这里我们清理冬天的蜘蛛网，为未来的晴天做好准备。让我们收拾一下，好吗？

[WinDirStat](https://windirstat.net/) 完全免费；它不应该是，考虑到它有多棒，但它确实是。要让它发挥它的魔力，你只需下载一个文件——就这样。虽然我们希望有一个 WinDirStat 的便携版本，但安装过程并不需要很长时间(几乎没有任何空间)。当您关闭安装例程时，默认情况下会加载 WinDirStat。

当应用程序打开时，您选择一个您希望 WinDirStat 扫描的位置。这可能是您的整个主硬盘驱动器，您使用的辅助硬盘驱动器被神秘地填满了存储空间，或者只是您的 Windows 用户文件夹的内容(或图片，或下载等。)然后，这个程序开始施展它的魔法，在你选择的区域内的每个文件夹、子文件夹和文件中搜寻。

WinDirStat 在应用程序的底部以一个大而不规则的图形显示它所发现的内容。它还按大小升序显示您的文件夹，并在您点击它们时继续显示，以深入了解您的驱动器内容。这使得找到占用千兆字节空间的神秘文件夹变得非常容易(或者更糟)，这很可能是一部大型电影的意外复制和粘贴的结果，或者是 Windows 中的临时文件发生了一些疯狂的事情，或者是你忘记了的一大堆~~色情~~ Linux 发行版。

无论你的文件是什么毒药，WinDirStat 都是硬盘空间不足的解药——我敢打赌，十有八九是用户错误，另外三个是 Windows 的错。虽然你可以使用 WinDirStat 甚至创建特殊的例行清理您的驱动器和文件夹，如删除每。BAK 或者。TMP 文件，您最好使用 WinDirStat 的有用信息作为通过文件资源管理器进行大规模删除的指路明灯。回收你宝贵的空间，Windows 用户。没有比这更容易的了。
