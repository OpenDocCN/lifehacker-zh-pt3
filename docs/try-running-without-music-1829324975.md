# 试着不听音乐跑步

> 原文:[https://lifehacker.com/try-running-without-music-1829324975](https://lifehacker.com/try-running-without-music-1829324975)

跑步有两个主要的好处:它是一种伟大而有效的锻炼形式，也是一种迷失在大自然中的方式，可以花 45 分钟到一个小时(或者不管你跑步有多长)几乎不受干扰地思考。

Watch

当然，一个预先制作的播放列表可以为你的跑步打气，或者帮助你保持速度。但是如果你想真正解放你的思想，马丁·弗里茨·胡伯在《T4》中建议，试着不听音乐跑步。

胡贝尔的观点是，户外跑步是人们，尤其是城市居民，能够沉浸在自然世界中的少数几次机会之一，没有屏幕，短信或通知遮挡你的视线。

> 当你为跑步预选一首配乐时，你是在试图心照不宣地控制你的体验。在许多情况下，这正是要点:例如，让我振作起来的音乐帮助你度过痛苦的最后几英里。但是，正如这个例子所说明的，我们的很多音乐都带有先入为主的心理联想；我们已经知道某首歌会带给我们怎样的感受。跑步的一大好处是，它可以提供一个机会来远离这一切，哪怕只是一个小时左右。

他称之为大脑的“微观冒险”:虽然你可以蒙着眼睛跑 3 英里，但你永远不知道你的思绪会飘向哪里。

这也是一种让你的跑步严格属于你自己和你身心健康的方式。你不是听播客来试图在跑步中学习一些东西，或者放音乐来加快速度，而是为了跑步而让自己沉浸在跑步中。更充分地欣赏沿途的风景，倾听你周围发生的事情，让你的大脑休息一会儿。

“我不认为我们总是需要在一些无形的生活改善清单上打勾——尤其是在从事一项本身富有成效的活动时，”他写道。我会去的。
