# 用新的“休息一下”功能让自己远离 YouTube

> 原文：<https://lifehacker.com/keep-yourself-from-binging-youtube-with-the-new-take-a-1826009385>

YouTube 视频:你永远不可能只看一个[。这不是你的错。YouTube 通过不断提供量身定制的建议(和一个积极的自动播放系统)来保持你的参与度。](https://lifehacker.com/watch-youtube-videos-without-ads-for-free-with-this-and-1824262552)



现在，由于一个新的更新，YouTube 终于承认太多的好事可能是坏事。以下是如何使用该服务新的“提醒我休息一下”功能来抑制你对 YouTube 的沉迷。

你需要做的第一件事是确保你有最新版本的 YouTube，适用于 [iOS](https://itunes.apple.com/us/app/youtube-watch-listen-stream/id544007664?mt=8) 或 [Android](https://play.google.com/store/apps/details?id=com.google.android.youtube&hl=en_US) 。在你被 [最新的漫威粉丝理论](https://www.google.com/search?q=who+is+the+avengers+infinity+wars+secret+survivor&oq=who+is+the+avengers+infinity+wars+secret+survivor&aqs=chrome..69i57.8311j0j4&sourceid=chrome&ie=UTF-8) (或者 YouTube 的算法认为你感兴趣的任何东西)分散注意力之前，点击右上角的你的个人资料照片。点击设置，你会看到一个名为“提醒我休息一下”的新选项

点击这个选项，你就可以设置 YouTube 在设定的时间后(选项范围从 15 分钟到两小时)向你发送一个友好的提醒，让你停止盯着手机看。你现在可以回去看的视频(或者烹饪指南，或者任何你可能感兴趣的东西)，知道 YouTube 会支持你——除非你因为太忙于看视频而忽略了它的通知。
