# 退订垃圾邮件只会让情况变得更糟

> 原文：<https://lifehacker.com/unsubscribing-from-spam-only-makes-it-worse-1796170464>

上次我查看垃圾邮件文件夹时，我注意到一些邮件包含退订链接。我想，那很好。也许垃圾邮件发送者意识到有些人永远不会回复，所以他们想精简他们的列表以提高效率。我点击了“取消订阅”那是个错误。



迈克菲实验室博客 称，虽然“合法公司”接受退订请求，但“可疑公司”只是使用退订按钮来确认您的地址，并向您发送更多垃圾邮件。 [Sophos 博客作者艾伦·蔡奇克说](https://nakedsecurity.sophos.com/2014/09/04/5-things-you-should-know-about-email-unsubscribe-links-before-clicking/) 点击取消订阅会告诉垃圾邮件发送者你打开了他们的邮件，可能是因为你对它感兴趣或者怀疑它是真的。通过访问垃圾邮件发送者的虚假退订页面，您将向他们提供您的浏览器信息和 IP 地址，甚至将自己暴露在恶意软件的攻击之下。

如果一封邮件看起来真的像可疑的垃圾邮件(而不仅仅是你讨厌阅读的时事通讯)，不要点击任何链接。只需将其标记为垃圾邮件，然后继续前进。
