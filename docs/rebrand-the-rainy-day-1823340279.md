# 重塑雨天品牌

> 原文：<https://lifehacker.com/rebrand-the-rainy-day-1823340279>

虽然这是老生常谈，但孩子真的就像小海绵，对父母来说，这是最酷也是最可怕的事情。你应对各种情况的方式，包括你选择的词语，可以帮助塑造他们对生活的世界观。没有压力！



即使看起来很小的互动也可能非常有意义。以关于天气的对话为例。在《巨头的工具:亿万富翁、偶像和世界级表演者的战术、套路和习惯》一书中，作者、前国际象棋神童 Tim Ferriss 向父母提出了这样的建议:在雨天改变你的语言。

他解释道:

> “在(我儿子)杰克出生后的第一年，我观察到的最大错误之一是，父母在天气好坏的问题上使用了没有意义的语言。每当下雨的时候，你会听到妈妈、保姆、爸爸说，“天气不好。”。“我们不能出去”或者“如果不能出去，天气很好”。我们可以出去了。这意味着，不知何故，为了能够出去玩得开心，我们在外部依赖于完美的条件。因此，杰克和我从来没有错过一场暴风雨，无论是下雨还是下雪，我们都出去在暴风雨中嬉戏。也许他生病的时候我们漏掉了一个。我们围绕这种语言的美丽发展了它。现在，每当下雨天，杰克就说，'看，爸爸，这是一个多么美丽的雨天，'然后我们就出去玩。**我希望他有这种内在的控制点——不依赖于外在条件*就像*一样。**

作为一个住在南加州的人，每当气温下降到 72 度以下时，他都会说公园“太冷了”，这让我很吃惊。正如挪威人所说，“没有坏天气，只有坏衣服。”好吧，好吧。但是这个建议也适用于当天天气预报之外的情况。很容易根据我们看到的情况得出结论，就此打住。"我在考试中得了一个很差的分数，因为老师对材料的解释很差."“我不成功是因为我所在的行业竞争太激烈了。”"我不快乐，因为我有糟糕的父母。"如果我们把注意力转移到我们能控制的事情上，并帮助我们的孩子也这样做，会怎么样？

你可以尝试给你的孩子更多的选择，让他们看到他们选择的后果，帮助他们找到他们有能力改变的事情，并支持 [一种成长心态](https://lifehacker.com/dont-be-a-know-it-all-be-a-learn-it-all-1794974291) 。下雨的时候，带上你的雨衣和靴子就可以走了。记住，孩子们，没有坏天气——只有像我这样有点讨厌寒冷的妈妈。
