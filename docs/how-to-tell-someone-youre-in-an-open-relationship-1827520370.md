# 如何告诉别人你处于一段开放的关系中

> 原文：<https://lifehacker.com/how-to-tell-someone-youre-in-an-open-relationship-1827520370>

开放的关系，伦理上的非一夫一妻制，一夫多妻制，一夫一妻制:有太多的词语可以用来告诉某人，当事情发展到这一步时，你正在和别人约会。以下是如何以尊重他们的时间和精力的方式做到这一点。



有很多 [如何打开关系的指南](https://lifehacker.com/how-to-have-an-open-relationship-1795829694) 。管理嫉妒，设定界限，处理你的感觉。多角恋社区非常擅长分享信息和相互支持，甚至发展他们自己的语言来描述非等级关系，这种感觉与嫉妒相反(这被称为补偿)。

对他们来说是好事。你很少听说的是，一个单身的人登录 Tinder，和一个可爱的人配对，在他们的简历中看到他们也很棒，然后在底部读到，“伦理上非一夫一妻制。”我很抱歉，但世界上没有足够多真正有道德的人来解释在约会应用程序上声称实行道德非一夫一妻制的人的数量，伙计们。

作为一个经历过无数次与公开关系中的人约会的女士，我有一些想法，关于你如何以尊重他们的经历的方式告诉新的约会对象你的关系状态。他们也是人。处理一下。

### 把它写进你的简历

泡在道德饥渴的人身上很有趣，但是当有人直截了当地说他们已经对某人有了承诺的时候是最好的。关于你在简历中的情况，你能传达(或想传达)的就这么多，但最好能有一些暗示。然后我，一个单身的人，可以决定我是否觉得喜欢处理它。

就在昨晚，一个男人在我们的约会中向我透露他处于一种开放的关系中。他说，他通常会在见人之前告诉他们，但通常在他提到这件事之后，他们会拒绝他。我问他为什么不把它写在他的简历里，他说当他写的时候，他根本没有找到匹配的。呜呜呜。不管怎样，结果都是一样的，但是如果你等待，你可能会浪费别人的时间，也浪费你自己的时间。有些人对此并不介意。允许他们睁大眼睛选择你。

如果你在现实生活中遇到某人，并且你正在调情，只要确保在他们正式约会或者你们接吻之前告诉他们，如果事情进展很快。那只是出于礼貌。

### 不要偷梁换柱

我和男人、女人、非二元人约会。用表情符号的话说，这让我成了一只独角兽。应用程序上的许多夫妇都在寻找第三个。再说一次，这很好。如果我有兴趣，我就有兴趣。但这种情况已经发生了很多次，我和一个本该单身的人约会，却收到了加入三人行的消息。大多数情况下，是一个女人，但我确实在一个男人说“我的女朋友等不及要见你了”之前，安排好了和他见面的时间和地点

这不仅是骗人的，而且可能会毁掉你的机会。没有人能用谎言套索抓住这只独角兽。

### 聊点别的

也许你刚刚开始走上这条非一夫一妻制的道路，你希望所有人都立即阅读你的[](https://www.amazon.com/Opening-Up-Creating-Sustaining-Relationships/dp/157344295X?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-to-tell-someone-youre-in-an-open-relationship-1827520370&asc_source=&tag=kinjalifehackerlink-20)*副本。开始一次新的冒险是令人兴奋的，尤其是如果事情已经变得陈旧或者你突然有了很多有趣的性爱。请记住，这不是你的婚外恋对象必须做的事情。如果有人真的同意和你出去，而你所做的只是用你特有的开放的语言，喋喋不休地谈论一夫多妻制，他们将不会感到被补偿。他们会感到无聊。*

*我个人更喜欢有人把他们的情况说出来，然后允许我问我需要回答的问题。这些问题因人而异。然后，把它当成一次普通的约会，而不是像你的论文答辩那样“人类并不倾向于一夫一妻制”*

### *清楚自己的极限*

*在早期约会的通常阶段，你在试探事情会如何发展。这可能会带来更坚定、更支持和更长期的东西。谁知道呢？可能性是存在的。*

*当你和处于开放关系中的人约会时，你的期望会有所不同。有些人在他们的主要关系之外有着同等重要和关心的关系；但更多时候，人们只是自由地发生性关系或享受随意约会。然而，我们是人，浪漫的感情可以在规则之外发展。如果这不是你的安排中的一个选项，不要沉迷于有趣的新恋情，玩弄那些界限，然后当你的放纵开始要求更多的时候反弹。*

*你对你的伴侣有义务，但这并不意味着你对你所见的每一个人都有更少的诚实和清楚的义务。说，“你一直都知道我在谈恋爱！”不是为所有糟糕行为开脱的灵丹妙药。没有承诺不等于没有同情和尊重。*

*丹·萨维吉创造了 [营地规则](https://www.urbandictionary.com/define.php?term=Campsite%20rule) 的概念，指的是有年龄差距的关系:年长的人应该让年轻的人“比他们发现他们时更好”这也应该适用于你在恋爱关系之外约会的人。毕竟，今晚结束时，你要回家了。*
