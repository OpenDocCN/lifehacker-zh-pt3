# 真正重要的睡眠技术

> 原文：<https://lifehacker.com/the-sleep-tech-that-actually-matters-1822558367>

我们正处于一个可以说是不必要的健康设备的黄金时代，包括几个旨在优化我们睡眠的新市场，或者至少给我们提供了大量描述睡眠的数据。我们询问了几位睡眠专家，哪些有用，哪些没用。以下是他们喜欢的功能:

Watch

### 任何能提醒你睡觉的东西都很重要

约翰·霍普金斯睡眠中心的萨拉·本杰明 博士说:“(睡眠科技)做的最好的事情是让人们注意到睡眠的重要性，你应该意识到你的睡眠。”。

她和我们采访的另一位睡眠专家，夏洛茨维尔神经病学和睡眠医学的克里斯·温特博士喜欢免费的 [Shleep](http://www.shleepbetter.com/) 应用程序背后的想法，该应用程序为你提供如何获得更好睡眠的小技巧和课程。(两人都没有时间检查应用程序中的每个建议，但两人都表示，他们看到的看起来像是合理的建议。)

温特博士说:“从表面上看，一切与睡眠有关的事情都很简单，”因为我们大多数人需要听到的信息只是按时上床睡觉。“但有时你必须以不同的形式看到它，或以不同的方式听到它，才能让它留下来。”

我点击了 Shleep 关于睡眠习惯的测试，对我周末和工作日晚上的睡眠时间做了最好的猜测，并承认我有时会在床上使用手机。随后，这款应用让我在一只卡通绵羊的指导下开始了为期十天的课程，内容是如何让我的手机在深夜不分散我的注意力。

### 对你睡眠时间的现实检查

我们的两位专家都不太相信睡眠追踪器对你夜间睡眠状况的分析。“一个健身追踪器可能会过度概括浅睡眠，”本杰明博士说，导致她的病人认为他们睡得不好，而实际上他们晚上大部分时间都在[【N2】](http://healthysleep.med.harvard.edu/healthy/science/what/sleep-patterns-rem-nrem)睡眠，这是完全正常的。

她指出，药物和衰老也会影响你的睡眠阶段，所以没有一个完美的睡眠结构是每个人都可以追求的。

但是任何睡眠追踪小工具——无论是床下垫、头带还是坐在床边听你呼吸的麦克风——都应该能够让你很好地感觉到你何时睡着。温特博士说:“(睡眠追踪器)的好处在于它能说真话。”。你可能认为你睡眠充足，但如果应用程序告诉你本周每天晚上都熬夜，你知道你需要做什么。

如果你决定寻求帮助，睡眠追踪器的数据也可以帮助你开始与专业人士的对话，但你的总睡眠时间可能是更相关的数字之一。本杰明博士说:“如果你患有严重的睡眠呼吸暂停症，并且你睡了 10 个小时，但没有恢复，你需要治疗呼吸暂停症。”

### 智能警报

我怀疑智能闹钟——那些能在你睡眠周期的正确时间叫醒你的闹钟——是否真的有用，但我们的两位专家都说它们有道理。温特博士说:“如果人们在快速眼动睡眠期间醒来，他们通常会感到有点恶心。”。在这个阶段，你的身体处于麻痹状态，所以在它唤醒你之前，一个小装置可以等你稍微活动一下。

如果你的闹钟能叫醒你而不打扰睡在你旁边的人，也有助于你伴侣的睡眠。

### 但是也许少用一些科技产品你会睡得更好

如果你不知道如何处理数据，那么获得一堆数据是没有帮助的。温特博士曾为睡眠科技公司提供咨询，他乐观地认为，睡眠设备可能很快就能对你的睡眠问题提供更多解释，并标记出可能是你应该与医生讨论的问题。但他说，如果我们只是获得了大量关于睡眠的数据，却没有任何关于正在发生什么的解释，那么所有的应用程序或小工具都在帮助我们“正视我们的问题”

两位专家都不想完全否定我们谈论的睡眠小工具，所以如果一个的 400 美元的头带帮助你入睡，使用它可能没什么问题。然而，没有一个新的设备有足够的数据来支持它们的有效性，所以你真的要靠自己来弄清楚你是否真的会从中获得价值 400 美元的好睡眠。

然而，有时候你需要的根本不是更多的技术。就像 Shleep 应用告诉我的那样，晚上摆弄屏幕或闪光灯不是一项有利于睡眠的活动。温特博士说，沉迷于承诺改善你睡眠的“更华丽、更性感”的产品是有危险的，而不是你忽略的简单行动——比如早上多晒晒太阳或者给自己买一个更舒适的枕头。
