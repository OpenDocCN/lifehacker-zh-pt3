UniChar 是一个第三方的 iOS 键盘，可以打开奇怪的 Unicode 字符的广阔世界。服务标志、版权标志、数学符号、技术符号，如苹果的“splat”图标或放射性指示器等，都可以在独立的应用程序和键盘插件中找到，只要你需要键入一些东西，你就可以使用它们。

你可以使用键盘将这些特殊字符添加到你的推文和文本中，但附带的独立应用程序提供了更多的功能。如果您发现自己在 web 的幕后工作，并且需要为特殊字符复制一些代码，您可以长按一个字符来复制符号、它的 Unicode 代码或在 web 上正确显示它所需的 HTML 代码。您还可以使用“按图形搜索”功能来查找您可能不记得其名称或类别的图标。它并不总是完美的，但它可能会唤起你足够的记忆，给你一个粗略的想法，你在寻找什么。
