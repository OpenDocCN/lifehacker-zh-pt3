# Sierra 10 台式机

> 原文：<https://lifehacker.com/the-sierra-10-desktop-1790830981>

读者 [亚历克](http://kinja.com/stranaha2) 提交了这个桌面，它给 macOS 带来了一点 Windows 10，但只是好的部分。Mission Control 有一个可爱的统一外观，底部有一个应用程序启动器，功能像一个任务栏。这是所有设置的方式。



如果你不是很了解的话，你会认为 Alec 的桌面实际上是一台 Windows PC，但仔细看看上面的 GIF，你会发现不——它实际上是一台 Mac，运行 Sierra。它只是稍微定制了一下，引入了 Windows 的一些熟悉的方面，作为一个跨平台工作的人，我可以理解这一点。Alec 在这里选择的工具是[über sicht](http://tracesof.net/uebersicht/)，一个我们在 之前已经强调过的定制工具 [。](https://lifehacker.com/ubersicht-customizes-your-os-x-desktop-with-widgets-and-1591137635)

一旦您熟悉了它，以下是您需要的组件:

*   [冬季壁纸布伊恩猿](http://cdn26.us1.fansshare.com/photo/macwallpapers/winter-lan-ape-wallpaper-retina-hd-download-mac-lan-ape-wallpapers-1483511063.jpg)T3】
*   macOS 的[über sicht](http://tracesof.net/uebersicht/)定制套件
*   bersicht 的 [晨钟](http://tracesof.net/uebersicht-widgets/#morning) 小部件
*   bersicht 的 [WeatherNow](https://github.com/briandconnelly/weathernow-widget) 小部件
*   [电池充电控件](http://tracesof.net/uebersicht-widgets/#battery-charge) 为 bersicht
*   [模糊控件](http://tracesof.net/uebersicht-widgets/#blur-widget) 使 dock 背景半透明
*   [cDock 实用程序](http://w0lfschild.github.io/app_cDock.html) 定制 macOS Dock
*   [调酒师实用程序](https://www.macbartender.com/) 定制 macOS 菜单栏
*   [LiteIcon 实用程序](https://freemacsoft.net/liteicon) 来调整系统图标
*   [地铁 UI 图标包](http://dakirby309.deviantart.com/art/Metro-UI-Icon-Set-725-Icons-280724102) 自定义系统图标

亚历克解释道:

> 自动隐藏菜单栏的 MacOS Sierra。主题码头使用 cdock 和 liteicon 给它一个窗口的感觉。

> 通过使用 cDock 隐藏 MacOS Dock 背景，并使用模糊小部件作为“人造”dock 背景，允许 ubersicht 小部件覆盖 dock。这允许电池小部件显示在右下角。

总而言之，我认为成品看起来很好，每次我看到这些 bersicht 定制的台式机之一，都会让我想在工作中对我的 Mac 进行一些有趣的调整。如果你也喜欢它，或者对 Alec 如何设置有疑问，请点击下面的 Kinja 博客链接，在那里留下评论并提出你的问题。
