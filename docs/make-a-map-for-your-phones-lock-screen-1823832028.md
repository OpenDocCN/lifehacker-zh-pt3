# 制作手机锁屏地图

> 原文：<https://lifehacker.com/make-a-map-for-your-phones-lock-screen-1823832028>

用你重要的另一半的照片作为手机壁纸的问题是，把它换掉会很尴尬。幸运的是，我妻子刚刚把她的锁屏从我的照片换成了我们喜欢的宝宝的照片，所以我把我的换成了阿尔瓦·卡托提供的一个很酷的地图。

Watch

阿尔瓦·卡图是出售 [时尚地图海报](https://alvarcarto.com/) 的商店之一，它建立了一个简单的移动友好网站，可以弹出你输入的任何地点的极简地图，分辨率适合大多数主要手机。

如果你想要更多的选择，在 [Mapbox](https://www.mapbox.com/signup/) (一个为开发者建立的更具技术性的服务)和 [做一个免费账户，去他们的静态 API 游乐场](https://www.mapbox.com/help/static-api-playground/) 定制 17 种不同风格的地图。记得检查“视网膜”框的最大分辨率为 2560x2560，疯狂吧:
