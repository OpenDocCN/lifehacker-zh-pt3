# 你在漫长的周末做了什么吃了什么？

> 原文：<https://lifehacker.com/what-did-you-cook-and-eat-over-the-long-weekend-1798708577>

大家好，欢迎回到 [*有什么好吃的？*](http://lifehacker.com/tag/whats-cooking) ，这是一个开放的话题，你可以在这里分享你对所有食物相关的想法、建议、食谱和观点。今天我想谈谈我们刚刚度过的愉快的三天周末，特别是你在这期间吃了什么。

Watch

长周末总是开始新的和令人兴奋的烹饪项目的完美时间，或者只是闲逛和烧烤，同时喝很多啤酒。不管你最后做了什么吃了什么，我都想知道。让你开始的几个问题:

*   你烤肉了吗？我没有真正接触烤架的机会，所以我只能通过你来替代。
*   你有没有着手任何令人兴奋的新项目？结果如何？在被太多伟大的食谱麻痹后，我终于根据 Stella Parks 的惊人新食谱， [*BraveTart:标志性的美国甜点*](https://www.amazon.com/BraveTart-American-Desserts-Stella-Parks/dp/0393239861?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/what-did-you-cook-and-eat-over-the-long-weekend-1798708577&asc_source=&tag=kinjalifehackerlink-20) *做出了布朗尼。我是盒装混合布朗尼的长期粉丝，但这些油腻的杰作让我全心全意地改变了信仰。*
*   本周末的饮料情况如何？你尝试过新的鸡尾酒吗？这并不是什么新鲜事，但我确实重燃了对辛纳的爱，整个周末我都在啜饮 [和](http://skillet.lifehacker.com/3-ingredient-happy-hour-the-pleasantly-bitter-cynar-fi-1798679527) 这种烈酒。
*   你参加过有趣的野餐或家常便饭吗？谁带来了最好的菜？我没有被邀请参加任何这样的活动，但是我参加了一个生日聚会。我带来了前面提到的布朗尼，他们非常喜欢。
*   如果你去户外探险，你会带什么样的野餐？我去 Washougal 河游泳，但除了薯片和健怡可乐，我什么也没带。不过，上周我去了河边，带了鸡肉沙拉三明治和几罐香蕉布丁，那真是太棒了。

此外，如果你只是呆在家里，吃外卖，我也想听听。我以一个巨大的香肠蘑菇披萨开始了周末，这是一个不错的选择。
