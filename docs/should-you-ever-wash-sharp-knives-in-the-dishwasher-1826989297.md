# 你应该用洗碗机清洗锋利的刀具吗？

> 原文：<https://thetakeout.com/should-you-ever-wash-sharp-knives-in-the-dishwasher-1826989297>

我曾经有一个室友，他把所有的东西都放在洗碗机里，这导致了我的两个花式酒杯的破碎。我被弄糊涂了——谁会拿起一个轻如鸿毛、茎干精致的酒杯，然后想“是的，这应该在金属盒周围叮当作响一阵子”？

不管怎样，我已经原谅她了。但让我感到惊讶的是，有些人把所有东西——包括锋利的刀子——都扔进了洗碗机。最近在几个朋友家目睹了这种做法后，我开始怀疑我是否没有错。也许技术已经发展了，旧的不把刀放进洗碗机的事情只是一个过时的神话。我咨询了刀具和洗碗机方面的专家。

“洗碗机绝对能让锋利的刀刃变钝，”肯莫尔 电器公司的产品经理杰弗里·贝克告诉我。“随着时间的推移，强大的喷雾作用会使刀片变钝，使你购买刀具时更难切割食物。”

所以洗碗工是站在我这边的。刀客呢？

“我们建议你不要把你的刀子放进洗碗机里，”沃斯霍夫的网站 在清洁和护理部分写道。它继续说，从技术上来说，它的刀具“适合在洗碗机中清洗”，但该公司建议手洗刀具，以防止它们被损坏或腐蚀。

Kenmore 和 Wüsthof 都警告说，将刀具放入洗碗机对正在卸载洗衣机的人来说也是危险的，所以如果你要将锋利的物体放入洗碗机，一定要将刀片放在显眼的地方。更好的办法是手洗，因为刀具的锋利边缘可能会撞击洗碗机的篮子或架子，破坏涂层，导致生锈和腐蚀。

我是两个对两个！

沉醉于新发现的洗碗机知识，我想知道我还应该远离这个电器。以下是制造商 Kenmore 和 [惠而浦](https://www.whirlpool.com/) 告诉我不要放在洗碗机里的其他一些厨房用品:

*   铸铁:你会毁掉“调料”/涂层，让它容易生锈。
*   水晶眼镜:看我上面的警示故事。
*   **银质餐具:**将不锈钢和银质餐具混合在一起会产生化学反应，导致餐具损坏。
*   **铜质马克杯:**高温和清洁剂会使铜质表面褪色并毁坏。
*   **木制砧板/器具:**高温会导致翘曲和开裂。
*   **隔热旅行杯:**高温会损坏真空密封。

所以你有它。我证明了我洗手的刀，砧板，铸铁，和花式玻璃器皿。如果你像我一样花钱买了好刀，那么你也会想花钱买橡胶手套。
