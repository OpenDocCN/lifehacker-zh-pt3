# 用一氧化二氮搅拌定制的利口酒

> 原文：<https://lifehacker.com/whip-up-a-bespoke-liqueur-with-nitrous-oxide-1825862121>

赠送自制的利口酒作为礼物是我最喜欢的聚会技巧。定制它们很有趣，制作成本也不高，而且——如果你有吸管的话——非常容易在最后一刻组装起来。



除了高档餐厅和布卢门撒尔粉丝的厨房，充氮奶油充电器最出名的是提供一个短暂的“派对高潮”你不应该用它们来做这个，但是你应该用它们在不到 20 分钟的时间里做出美味的、定制的啜饮酒。

制作利口酒是一个简单的过程。你只需取一种 ABV 至少为 40%的基酒，用美味的根、水果或皮( [使用 Gizmodo](https://gizmodo.com/how-to-rapidly-flavor-any-booze-using-whip-its-5955339#_ga=2.28546618.763968073.1525709521-426097860.1521480505) 的这种方法)，然后用等量的糖浆稀释它，使它进入利口酒的 ABV 范围——大约 20%。如果你没有虹吸管，输液步骤需要等待很长时间，但 iSi 罐及其充电器使这成为一个悠闲的下午项目。下面是怎么做的:

1.  向你的虹吸管中加入一杯酒。伏特加是不错的中性选择，但你也可以用朗姆酒、龙舌兰酒或任何你喜欢的东西。你也可以使用谷物酒精，你只需要把它稀释多一点。
2.  将你的调味料组合起来。对于你应该添加多少没有硬性规定，但我通常会用至少一杯切碎或捣碎的水果，或者每杯酒用一个完整柑橘的皮。对于草本植物来说，一两个小枝通常就可以了，但是要记住，柔软的草本植物会比坚硬的木本植物注入得更快。对于我的蓝莓梅尔柠檬利口酒(如上图)，我用了一杯大致捣碎的蓝莓和一个梅尔柠檬的皮。对于我的生姜蜂蜜酒，我用了一片两英寸的生姜，切碎。
3.  关闭虹吸管，用一根鞭子给它充电。等一分钟。取下充电器，装上一个新的。轻轻摇动虹吸管一分钟。轻轻按下 iSi 的触发器，释放罐中的压力。打开盖子，让剩余的氮气逸出，尝一尝。
4.  如果味道不够好，再放些东西进去，重复第二步和第三步。一旦你有了一杯有效的饮料，用筛子过滤出固体，然后用一些奶酪布过滤出较细的颗粒。
5.  现在你需要用糖浆稀释。根据你的口味，这种糖浆可以用普通蔗糖、turbinado 糖、蜂蜜或龙舌兰制成。对于颗粒糖，用等量的糖和水制成糖浆；对于已经是糖浆状的糖，使用两份蜂蜜或龙舌兰对一份水的比例。因为你要将 ABV 降低到 20%(我们初始 ABV 的一半)，你需要和注入酒精一样多的糖浆。(所以，如果你有一杯泡制的伏特加或其他什么，你需要一杯糖浆。)将糖浆和酒混合，倒入一个漂亮的瓶子里，要么作为一份非常贴心的礼物送人，要么自己留着。

这些注入的快速性质意味着你可以在很短的时间内玩各种口味，创造你自己独特的利口酒，并调整它们，直到它们完全符合你的口味。就糖而言，除了味道，还要考虑外观。蜂蜜可以增加深度和调和辣味，这就是为什么我在我的生姜利口酒中使用它，但它会使你的最终产品变暗。一些利口酒——比如蓝莓柠檬——非常好喝，如果用深色糖浆掩盖起来就太可惜了，所以更有活力的批次还是坚持用纯白糖吧。一旦你冰镇了你的利口酒，把它用在鸡尾酒里，或者把它和一些苏打水混合，就成了一种简单的、奇特的成人饮料。
