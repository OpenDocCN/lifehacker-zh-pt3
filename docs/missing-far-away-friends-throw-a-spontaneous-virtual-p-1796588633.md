# 想念远方的朋友？用这个应用程序举办一个自发的虚拟聚会

> 原文：<https://lifehacker.com/missing-far-away-friends-throw-a-spontaneous-virtual-p-1796588633>

如果你可以在任何时候举办一个虚拟派对，让你来自世界各地的朋友都可以参加，那会怎么样？这就是 Houseparty 背后的想法，一款为自发对话而设计的 [iOS](https://itunes.apple.com/us/app/houseparty-group-video-chat/id1065781769) 和 [Android](https://play.google.com/store/apps/details?id=com.herzick.houseparty) (点击链接下载)视频聊天应用。

Watch

Houseparty 在青少年中有大量的追随者，但我直到大约 6 个月前才听说它。

它的工作方式非常简单:当你启动应用程序时，你创建了一个聊天室，然后你的朋友也可以使用这个应用程序。你的朋友可以进行设置，以便在你启动应用程序时通知他们(如果他们肯定不想错过你的话)，并且你可以邀请朋友加入自己无法加入的聊天室。

六年前，我从东海岸搬到了旧金山，我发现这个应用程序是一种与不在身边的一些好朋友保持联系的神奇方式。如今，与人视频聊天有无数种方式，我们也在使用这些方式，但能够说“哦，凯拉在线”并顺便聊聊也是一件好事。

对于其他视频聊天选项，如 Skype、Facetime 和 Google Hangouts，我总是觉得我需要在打电话之前问一下，或者提前和朋友约个时间。在我拨号之前，如果没有某种计划好的聊天设置，视频聊天就不是我喜欢随便做的事情。有了 Houseparty，我可以一个人在家喝啤酒，决定我想要有人陪伴，并立即邀请所有拥有该应用程序的朋友加入我的行列。

当然，有时这意味着我还是一个人喝啤酒。然而，更多的时候，我的一些朋友也在某个地方，他们可以视频聊天，一个虚拟的聚会出现了。

例如，最近的一个晚上，我和来自纽约、芝加哥、北卡罗来纳和洛杉矶的朋友聊了一个小时。我们在大学里都是朋友，但已经十多年没有在同一个房间里了。尽管聚会是在我小小的手机屏幕上进行的，但我们所有人同时出现在同一个地方还是相当令人惊讶的，我们根本没有计划过。

房间最多可以容纳八个人，您可以同时打开多个房间。这个想法是你可以在派对上不同的“房间”之间漂浮，和不同的朋友聊天。

如果你想私下聊天，你可以锁上房间。否则，他们会对已经在房间里的人的朋友开放。例如，如果我和我在纽约的朋友保罗聊天，那么他的朋友莎莉，我不认识，也可以加入我们的房间。事实上，我已经通过这种方式认识了一些人，后来我亲自和他们一起出去玩。

使用该应用程序需要您的朋友也安装该应用程序。然而，有趣的是，一旦你有了几个朋友，它会很快在你的朋友群中传播开来。当我开始的时候，我大概有三个朋友在使用这个应用程序，现在我已经超过三十个了，增加了我找到那个深夜喝酒伙伴的几率。
