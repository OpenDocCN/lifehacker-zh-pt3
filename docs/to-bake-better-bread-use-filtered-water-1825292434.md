# 要烘焙更好的面包，使用过滤水

> 原文：<https://lifehacker.com/to-bake-better-bread-use-filtered-water-1825292434>

烘焙漂亮的面包需要熟练地操作三个大的、混乱的变量:技术、设备和配料。拙劣的技术是大多数不合格结果的原因，但厨房条件或原料供应的突然变化甚至会让经验丰富的面包师陷入困境。如果你的面包莫名其妙地开始发臭，你的水可能就是罪魁祸首。



自来水有两种性质会阻碍发酵面包的情况:氯含量和硬度。含氯化合物，特别是 [一氯胺](https://en.wikipedia.org/wiki/Chloramine) 被用于水处理，因为它们在极低的可饮用浓度下就能杀死危险的微生物——但这意味着它们也能杀死酵母。至于矿物质含量，过硬的水(其 [溶解矿物质](https://lifehacker.com/get-rid-of-all-kinds-of-hard-water-stains-with-vinegar-489265084) 水平升高)由于其碱度增加而抑制了良好的上升。我们的小酵母朋友在几乎没有酸性的条件下茁壮成长，所以强迫他们在强碱性的环境中发育充其量是适得其反。

绝大多数饮用自来水不含有足够的氯或矿物质来干扰烘焙；一般来说，酵母发酵的面包在除了最极端的水分条件下都能做得很好。然而，根据我的经验，自然发酵的面团——尤其是酸面团发酵剂——对水分变化非常敏感。我怀疑这归结于发酵剂之间的代谢差异。速溶(或活性干)酵母非常活跃，可以大量添加到面团中，这两者都有助于在各种面团条件下快速发酵。酸面团中的野生酵母和乳酸杆菌工作速度要慢得多，这需要更长时间接触溶解在自来水中的各种化学化合物——并增加了不良反应的可能性。

家庭烘焙论坛充斥着关于水质的帖子；有些人甚至会选择性地再矿化 [蒸馏水](https://lifehacker.com/here-are-all-the-ways-you-should-be-using-distilled-wat-1821080543) 以精确控制他们的发酵剂的化学成分。坦率地说，这是一些下一级的控制怪胎狗屎:所有其他事情都是平等的，如果你的自来水是可饮用的，它可能是好的。然而，如果你的水高度氯化，非常硬，或者非常含硫，试着在加入面包面团之前过滤一下。费城的自来水(不好意思，“伍德尔”)非常含氯，改用过滤水完全改变了游戏规则:我们的首发不再提前制作烈酒，酸面团面包充分发酵，不会在发酵过程中莫名其妙地坍塌，酵母面团高高升起。

过滤水不会神奇地修复那些 [揉面不足](https://www.youtube.com/watch?v=hC8LcLnqMl8) 或 [过度醒发](https://www.youtube.com/watch?v=6oAfl1u0fIw) 的面包，所以在指责你的水导致糟糕的烘焙之前，确保你的技术是可靠的。但是，如果你的自来水喝起来不舒服——或者硬到需要定期除垢——过滤器可能正是你开始自己制作美味面包所需要的。
