尽管放弃了“OS X”而选择了“MAC OS ”, Sierra 是我们最近看到的最不重要的升级之一。这不是一件坏事，在这种情况下，这意味着升级的压力远没有过去那么大。不过，这并不是说问题不会出现。我们来看看细节。

### 谁应该立即升级

如果你的 Mac 还不到几年，并且你对苹果的生态系统相当投入，那么现在升级到 Sierra 是显而易见的。升级过程很顺利，变化很小，不会影响任何人的工作流程，总的来说，新功能对每个人都有好处。

如果你喜欢 iPhone 上的 Siri，你也会喜欢 Mac 上的 Siri。只需一个快捷的键盘快捷键或点击鼠标，你就可以调用 Siri，问它任何你需要的问题，最终实现与电脑对话的梦想。同样，如果你使用苹果照片，你会很高兴地看到一些急需的改进，包括照片搜索和人脸识别，终于在应用程序中实现了。如果你是一个狂热的照片用户，这些都值得升级。

类似地，如果你对苹果的生态系统非常投入，因为你有一个更新的 Mac，一个更新的 iPhone 和一个 Apple Watch，你将从升级中收获最多的回报。所以，你还是尽快做吧。例如，如果你有一个苹果手表，用它解锁你的 Mac 听起来非常酷，所以升级当然是有意义的。

### 谁应该推迟一段时间

我们没有遇到任何软件问题，我们在 Sierra 中看到的大部分错误报告都很平淡，但这并不意味着问题不会出现。

如果你在工作中使用 Mac，或者如果你每天都需要某些软件，通常最好推迟一点，这样你的第三方软件的开发者就有足够的时间来推出升级。Sierra 可能看起来不像是一个大的更新，它确实不是，但这并不意味着它不会潜在地打破你每天依赖的六年之久的时间管理软件。所以，在你做任何事情之前，先看看咆哮应用的兼容软件列表 。

如果说我们看到的关于 Sierra 的任何主要抱怨，那就是对网守设置的更改。网关守护设备是系统范围的设置，用于防止未经批准的应用程序安装在您的 Mac 上。在以前版本的 macOS 中，您可以更改 Gatekeeper 的默认设置，以允许任何地方的任何应用程序运行。现在，你只有两个默认选项，Mac App Store 下载的应用和苹果批准的应用。这对于一直在打开未被批准的应用的开发者来说是很烦人的。一个解决办法是 [使用右键点击一个应用](http://lifehacker.com/quickly-override-gatekeeper-with-an-right-click-1761975597#_ga=1.265038983.968941705.1436971740) ，但如果你经常这样做，还是有点讨厌。

如果您想将旧的“任何地方”选项添加回 Gatekeeper，您需要深入了解终端。在终端中键入以下命令:

```
sudo spctl --master-disable

```

现在，您可以像往常一样更改网守设置。这是对你的 [Mac 的安全文件](https://developer.apple.com/legacy/library/documentation/Darwin/Reference/ManPages/man8/spctl.8.html) 的编辑，所以只有当你对你的 Mac 进行系统级的修改时才这样做。

### 谁不应该升级

Sierra 是一段时间以来第一个放弃对旧 Mac 支持的 Mac 操作系统。因此，如果你在 Mavericks、Yosemite 和 El Capitan 的过程中升级了你的旧 MacBook，你会想再看看新的系统要求。以下是使用 Sierra 需要的 Mac 型号:

*   MAC Pro(2010 年年中或更高版本)
*   iMac(2009 年末或更高版本)
*   MAC mini(2010 年年中或新款机型)
*   MacBook Pro(2010 年年中或新款机型)
*   MacBook(2009 年末或更高版本)
*   MacBook Air(2010 年末或新款机型)

然而，这并不是故事的结尾。Sierra 的一系列新功能，包括整洁的通用剪贴板和 Apple Watch 解锁，只能在装有蓝牙 4.0 的 Mac 上工作，这意味着他们需要一台更新的 Mac 才能使用新功能:

*   MAC Pro(2012 年末或更高版本)
*   iMac(2012 年末或更高版本)
*   MAC mini(2012 年末或新款机型)
*   MacBook Pro(2012 年年中或新款机型)
*   MacBook(2015 年初或更高版本)
*   MacBook Air(2012 年年中或新款机型)

这并不是说你不应该升级，因为你不会得到新的蓝牙功能，但这是一个对旧 MAC 来说不太有趣的更新，因为你唯一能得到的主要新功能是 Siri。我们在一台旧的 2009 款 iMac 上运行了 Sierra，从性能角度来看，它运行得非常好，但作为 El Capitan 的更新，它几乎不引人注意，因为大多数功能都不兼容。

### 底线:不要着急，但对大多数人来说这是一个可靠的更新

就功能而言，Sierra 是一个非常温和的更新，但这意味着它对大多数用户来说也是一个非常安全的升级。目前，公开测试版已经发布了几个月了，这意味着大多数主要问题都应该解决了。如果你担心性能， [全新安装](http://lifehacker.com/in-defense-of-the-clean-install-1726140915) 可能有助于清除你系统中的垃圾，但除此之外，每个人都将从免费更新中受益。

综上所述，没有理由停止你正在做的更新。随着成千上万的人争相下载 Sierra，苹果的服务器可能会受到冲击，这意味着下载可能需要一段时间。Sierra 没有什么突破性的东西，所以请慢慢升级，等到下午有空的时候再运行更新。
