# 什么知道罗斯个人退休帐户转换

> 原文：<https://lifehacker.com/what-to-know-about-roth-ira-conversions-1825654230>

就在减税和就业法案通过后，我写了一篇关于罗斯个人退休帐户如何比以往任何时候都更好看的文章 多亏了较低的税率。然而，一个缺点是，罗斯个人退休帐户有收入限制:如果你在 2018 年修改后的调整后总收入为 120，000 美元，你可以缴纳的金额开始逐步减少，如果你的总收入为 135，000 美元，你就完全没有资格(对于单一报税人)。总的来说，这不是一个坏问题——和大多数税收规则一样，高收入人群仍然有办法向 Roth 捐款。

你可以通过所谓的后门罗斯(Backdoor Roth)来做到这一点，资金从传统的个人退休帐户转换成罗斯个人退休帐户。转换没有收入限制。它可以通过以下几种方式实现:你可以向传统的个人退休帐户缴纳不可扣减的费用(换句话说，你已经缴纳了税款)，然后将该帐户转换为罗斯帐户，或者你可以在转换时缴纳税款。

你可以兑换多少钱也没有限制。因此，如果你已经为你的个人退休帐户贡献了最大值(目前，50 岁以下的人每年 5500 美元，50 岁及以上的人每年 6500 美元)，你可以一次性将它滚动。不过，你要小心:你兑换的钱将被计为收入，这意味着你可能会在兑换的那一年冒险将自己提升到更高的税级。(正如 [基普林格](https://www.kiplinger.com/article/retirement/T046-C032-S014-roth-ira-utopia-is-now-the-right-time-to-convert.html) 所说，“额外的收入也会影响你的社会保障税和你为医疗保险支付的费用”，如果你已经退休的话。)在这种情况下，你可以考虑将你滚存的金额分散在几年内，以保持你的税单易于管理。

有什么好处？你的钱和任何收益都免税。因此，当你达到退休年龄时，你可以取出资金，而不用担心税收问题。(记住，你需要将兑换的钱放在罗斯银行 [至少五年](https://www.nerdwallet.com/blog/investing/roth-ira-5-year-rules/) ，否则你会欠收益税。)Roths 也没有像传统 IRA 那样要求最低分配额，这使它们成为向你的继承人传递财富的便利工具，假设你不需要分配额来生活(另一方面，传统 IRA 的 rmd 从 70 岁开始)。

一般来说，如果你在与传统个人退休帐户相同的金融机构开立你的罗斯帐户，那么转存资金就不成问题。然而，如果你选择 60 天的展期，你将有足够的时间来兑换这笔钱，或者如果你不到 59 岁，你将面临 10%的提前提款罚款。

还有一点需要注意的是:正如我 [在](https://twocents.lifehacker.com/roth-iras-just-became-an-even-better-deal-for-retiremen-1822134829) 之前写的，税单去掉了你的“重新定性”选项，意思是一旦你把钱兑换了，就不能再兑换回来了。因此，这使得转换时机变得更加重要——当转换会让你进入更高的税级时，你不会想这么做。

当然，这是否适合你要视情况而定——有很多税务方面的考虑要考虑，如果你正在考虑这么做，你应该向税务专家寻求个性化的指导。
