# 当你有债务要还的时候，在一些无聊的事情上挥霍可以吗？

> 原文：<https://lifehacker.com/is-it-ok-to-splurge-on-something-frivolous-when-you-hav-1827966983>

你应该把所有可自由支配的钱都用来偿还债务，还是可以挥霍在你一直关注的东西上？这就是我们本周讨论的内容。



每周一，我们会通过向一些理财专家寻求建议来解决你的一个紧迫的个人理财问题。**如果你有一个一般性的问题或金钱问题，或者只是想谈谈 PeFi 相关的事情，请在评论中留下，或者发电子邮件给我，地址是 alicia.adamczyk@lifehacker.com。**T3】

本周的问题来自凯撒:

> 我最近买了一双200 美元的鞋子，在一个艰难而忙碌的日子里。对于一段时间的幸福来说，这是不是太多了，尤其是当你有债务和学生贷款的时候？我按时或提前支付所有款项。我确实感到内疚，因为我想到这些钱本可以用在别的什么地方，但我又想起这只是钱，反正都要花掉，所以为什么不花在我喜欢的东西上呢？

这是个别专家对一个对每个人都有不同影响的问题的一般看法——如果你想要个性化的建议，你应该去见理财规划师。

### 这是你的钱和你的财务未来

很明显，把额外的 200 美元放在你的债务上，而不是另一双运动鞋，将是财务上负责任的事情，这是你清楚了解并正在努力解决的事实。但是在这种情况下你可以让自己休息一下。

即使是我们当中最节俭的人也会偶尔犯错，虽然你不应该养成每次心情不好就买昂贵的鞋子的习惯，但陷入内疚只会让你感觉更糟。以我的经验来看，感觉自己像个废物并不是一个很好的动力。

这是美国生活中一个有趣的(阅读:破坏性的)现实，我们以各种形式促进消费主义，但当我们有债务要付或过着收支平衡的生活时，却因为购买我们不“需要”的东西而感到羞耻。在某种程度上，我们将自己的价值和道德等同于我们拥有顶级信用评分和零债务的能力，每年最大限度地提高我们的退休供款，并拥有充足的应急基金。这是同样的心态，如果你想要医疗保健，你应该 [停止购买 iphone](https://www.washingtonpost.com/news/wonk/wp/2017/03/07/if-jason-chaffetz-wants-to-compare-healthcare-to-iphones-lets-do-it-the-right-way/?utm_term=.88f1a312047b)或者 [不给服务员小费](https://www.eater.com/2018/7/27/17577272/bad-tipping-in-america-restaurants) 将在某种程度上导致餐饮业更好、更公平的工资环境。但是，有债务并不意味着你不能享受琐碎的事情，或者你应该为你没有花在“必需品”上的每一分钱而感到内疚。

所以你有债务。你在付款——有时很早——这很重要。是的，任何额外的一点点都可以帮助你更快地被挖掘出来，但是对待我们自己也有一些事情要说。我们都应该时不时地得到一束鲜花或一双新鞋，享受它们的时候不要感到内疚或羞愧。我们都应该拥有能带给我们快乐的东西。

### 抑制我们的消费冲动

但是，如果冲动消费真的困扰着你——也许这就是你一开始就有一些债务的原因——有几个策略你可以用来避免这样做:

*   一个很明显的: [只用现金](https://lifehacker.com/get-out-of-debt-by-switching-to-a-cash-only-policy-5796030)

*   或者，尝试使用一张 [预付借记卡](https://lifehacker.com/use-a-prepaid-debit-card-to-curb-impulse-sending-1588277919) 几天/几周

*   遵循克里斯汀·王的 [10/10 法则](https://twocents.lifehacker.com/how-to-be-frugal-without-wasting-your-time-1759480175)

*   问自己这个问题: [购买会如何改善你的生活](https://twocents.lifehacker.com/control-impulse-spending-by-asking-how-a-purchase-impro-1783420951) ？
*   在线删除您 [保存的付款信息](https://lifehacker.com/delete-saved-payment-info-to-reduce-online-impulse-buyi-1567518390)
*   [关注机会成本](https://twocents.lifehacker.com/think-of-impulsive-shopping-as-the-enemy-of-your-financ-1762992612)

理解你的触发器。你说你今天过得不好，所以你买了这双鞋。那是经常发生的事情吗？当情况变得艰难时，除了网上购物，你还有别的出路吗？下一次，不要细读你的购物清单，试着散散步或在日记中写抱怨(我更喜欢的应对方法)来清理你的头脑。任何时候我们觉得买东西从来没有这么容易过，但是问问你自己你是否真的需要它，特别是如果这样做会让你事后感到内疚。

接下来，考虑你的其他财务目标。也许这 200 美元不需要用于偿还债务，但是当你打开你最喜欢的商店的邮件时，有没有其他对你来说很重要的事情(比如，为度假存钱或有一天拥有一栋房子)是你不会考虑的？下次你冲动地去买东西时，考虑一下这些目标。

最重要的是，确保你有一个偿还债务的 [计划](https://lifehacker.com/a-step-by-step-guide-to-getting-out-of-debt-1475515477) ( [滚雪球或雪崩](https://twocents.lifehacker.com/more-research-shows-that-the-snowball-method-is-the-bes-1790533681) ，或你自己创造的另一个)。如果你对你的债务偿还计划感到满意，你投资了一点点，你没有让自己陷得更深，那么买一双昂贵的鞋子不会给你的财务未来带来灾难。

好好享受你的新鞋。你穿上会很好看的。
