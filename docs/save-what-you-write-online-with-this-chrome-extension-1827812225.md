# 用这个 Chrome 扩展保存你在线写的东西

> 原文：<https://lifehacker.com/save-what-you-write-online-with-this-chrome-extension-1827812225>

[不小心丢了所有工作](https://www.youtube.com/watch?v=_GxC4kKD9qA) 比以前难了。应用程序带有自动保存功能，Chrome 会在你关闭一个有未保存工作的标签页之前试图警告你。但是输入太快，或者崩溃，你仍然会丢失很多内容。它可能发生在脸书更新、申请表或博客文章上。但是 Chrome 扩展 [类型表单恢复](https://chrome.google.com/webstore/detail/typio-form-recovery/djkbihbnjhkjahbhjaadbepppbpoedaa?hl=en) 可以保存你写的内容并马上粘贴回去。

Watch

一旦你安装了 Typio，你会在每个网页表格的角落看到一个蓝色的小按钮。您还会在页面顶部看到一条绿色条纹，每当 Typio 保存新文本时，它就会发光。在设置中禁用条纹。要恢复丢失的文本，您可以单击蓝色按钮，键入可自定义的键盘快捷键，或者在右键菜单中找到它。

如果这些选项中的任何一个变得令人讨厌，你可以关闭它们中的任何一个或全部。例如，我只能通过右键点击来访问 Typio。我讨厌蓝色按钮，它出现在每个网络表单上，包括推文、登录和搜索。(默认情况下，Typio 不保存密码或信用卡号码，但你可以更改。)

你可以告诉 Typio 保存条目的天数。当你打开 Typio 时，首先它会显示一个你最后几个条目的极简菜单。点击“浏览所有条目”查看更长的菜单。所有数据都存储在本地；没有任何内容发送到 Typio。(可以 [查看源代码](https://bitbucket.org/nicklassandell/chrome-form-recovery/) 。)而且你可以在任何网站上禁用这个扩展，比如 Twitter 或 Google。

如果你在博客、社交媒体或求职申请等表格上写长文章，这尤其有用。例如，我只是故意不保存就关闭了这篇博文，然后用 Typio 点击了几下就把它恢复了。

Typio 取代了 [Lazarus](https://lifehacker.com/lazarus-form-recovery-saves-web-page-form-data-5097334) ，一个类似的扩展(有一个更好的名字)最终被放弃， [被踢出了 Chrome 网络商店](https://www.reddit.com/r/chrome/comments/7zzyr0/lazarus_form_recovery_is_gone_from_chrome_store/) 。文·拉撒路不得不死去，但他的精神永存。
