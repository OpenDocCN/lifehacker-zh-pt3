# 小孩的包

> 原文：<https://lifehacker.com/the-little-kids-bag-1790763574>

小孩子也会随身携带包，这就是为什么 Flickr 用户包配置分享他们儿子的包，里面装满了小孩子需要的所有东西。



包里是一个 [幼崽背包](http://cubcubs.com/collections/new/products/cub-ruck?variant=19774364422) 。这是里面的内容:

*   补丁。通常是我设计的或者来自 [幼兽选择的](http://bit.ly/TrackCubBadges) (精选的是幼兽幼兽 [游走](http://cubcubs.com/collections/kids/products/adventure-ho-badges?variant=29924817868) 和 [蜂助](http://cubcubs.com/collections/kids/products/bee-helpful-badge) 徽章)。

*   [婴儿湿巾](http://amzn.to/2h2oWa1)

*   [仿制登山扣](http://amzn.to/2h2j94c)

*   [书写器](http://amzn.to/2hLOdE3)

*   [组织](http://amzn.to/2h2mQa2)

*   [PDW 指南针](http://prometheusdesignwerx.com/collections/compass/products/expedition-watchband-compass-orange?variant=17911941575)

*   [玩具](http://amzn.to/2h2oZTf)

*   [小吃](http://amzn.to/2i6TB3I)

*   衣服。帽子、 [手套和备用 t 恤](http://cubcubs.com/collections/kids/products/updated-camp-kids-tee-midnight-blue)
