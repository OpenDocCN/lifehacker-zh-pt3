# “一个文件盒，一个文件夹”的返校购物规则

> 原文：<https://lifehacker.com/one-file-box-per-kid-one-folder-per-year-is-the-back-1797317059>

在学年开始的时候，你的孩子带回家的每一份写作作业、证书和艺术品都是一份甜蜜的惊喜，是一份值得珍藏多年的纪念品。但是，很快，你的房子就堆满了文件，现在才 11 月，该死。



返校购物季正在如火如荼地进行，但是在你把购物车交给你的孩子并让他们在目标通道里自由奔跑之前，你需要一个计划来 [保持混乱到最低](https://lifehacker.com/8-tips-for-decluttering-your-kids-massive-amount-of-stu-1796713214) 。 [快乐专家格雷琴·鲁宾](http://lifehacker.com/how-to-hack-happiness-with-gretchen-rubin-1794998732) 有一个简单明了的解决方案 [对于特殊的学年纪念品:给每个孩子用一个文件盒，每个文件夹代表一年。她告诉《华盛顿邮报》*,“只包括捕捉一个典型年份所必需的东西，比如孩子的生日聚会邀请、学校照片和一些艺术项目。查看系统运行的一些示例:此处*](https://www.washingtonpost.com/news/parenting/wp/2017/06/20/gretchen-rubin-shares-her-tips-for-a-happier-parenting-existence/?utm_term=.9edd5848fb92)*[此处](http://www.iheartorganizing.com/2015/08/uheart-organizing-creating-school.html)* *和 [此处](http://thesimpleproof.com/kids-art-school-work/) 。*

这是一个简单的设置，可以让你保留你最喜欢的作品(比如你的孩子写的关于她的父母有多棒的诗)。对于所有其他的东西，你可以扔了它或者用一个应用程序把它数字化，比如 [ArtKive](https://www.artkiveapp.com/#home) 。
