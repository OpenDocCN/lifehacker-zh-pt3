# 你只需要关注当下

本周的精选来自马库斯·奥勒留和他的*冥想*。他建议你不要只关注过去或未来，而是现在:

> “不要让你生活的全景压迫你，不要沉湎于过去或将来可能发生的各种麻烦。只要在当下的每一个实例中问自己:‘这份工作中有什么是我不能忍受或支持的？’你会羞于做这样的忏悔。然后提醒自己，让你感到沉重的既不是未来，也不是过去，而是现在——这样，现在的负担就会减轻..."

### 这意味着什么

不要让你生活的广阔范围阻碍了你。不要专注于你过去的艰难，也不要担心未来可能会出什么问题。现在集中注意力。有什么关于这个时刻或者这个特殊任务的事情是你不能处理的吗？大概不会，因为你知道你能挺过去。记住，你应该背负的不是未来的重量，也不是过去的重量。让那些负担消失，当下的重量会变得轻很多。

### 从中吸取什么

我们倾向于关注我们已经去过的地方和我们将要去的地方，但是我们没有对现在投入足够的思考。但真正重要的是现在，因为这就是一切。过去只是记忆、文字或记录，未来只是基于这些东西的推测。你从来不在过去或未来，所以为什么要在那里浪费这么多时间和精力呢？

当生活变得艰难，或者你在艰难的工作中挣扎，记得停下来，放下无用的负担，比如过去(它是不可改变的)，和未来(它是不可预见的)。不要总想着你的整个人生，或者某段时间，想着你将如何度过。看着此刻，就在此刻。你能熬过这一刻吗？是的。好吧，这个怎么样？当然，和上一次也差不了多少。这个呢？再说一遍，是的。看，你唯一需要承载的重量就是现在，而你肯定有足够的力量举起它。你已经做了一辈子了。

人生是一次漫长的旅程，一场经历的马拉松。你是怎么走这么远的距离的？一步一步来。专注于你现在的一步——而不是你已经犯下的错误，或者你可能必须采取的步骤——你会到达你需要去的地方。
