# 律师什么时候会需要 16 部手机？

> 原文:[https://life hacker . com/when-a-lawyer-ever-need-16-phones-1825712656](https://lifehacker.com/when-would-a-lawyer-ever-need-16-cell-phones-1825712656)

上周，联邦检察官从唐纳德·特朗普的律师迈克尔·科恩那里缴获了 16 部手机 ，这让我们许多人不禁好奇，什么样的律师会需要 16 部手机，而且都可能与工作有关？所以我们问了一些律师。

Watch

就业律师 Yosef Peretz 认为 16 部手机听起来很粗略:

> 如果手机都是活跃的，那么对于我们这一行的人来说，这是非常不合理的。但科恩似乎是一个狂热的记录者，因为他声称自己录下了很多对话。即便如此，对于一个不贩毒的律师来说，16 部手机已经很高了，非常高了(我甚至没有说科恩在贩毒)。

民权律师希拉里·本汉姆-贝克对此表示赞同:

> 我觉得超级诡异。我只有一部手机。我认识几个律师，他们有两个——一个是私人的，一个是工作的。我从没见过比这更多的律师。

一些律师告诉我们，老实说，手机可以叠加。科恩的一些手机是黑莓手机，这意味着这批货可能包括许多科恩从未处理掉的旧手机。很多人家里都有一两部旧手机。如果你同时玩四部手机，并且身边有三代老手机，你最终会有 16 部手机。

但是一开始就摆弄四部手机？商业律师 Roman Fichman 看到了许多光明正大的目的:

> 律师需要多部手机有其合理的业务和实践原因:
> 
> *   随着座机的使用减少，多部手机正在取代多线系统。
>     
> *   在不同业务领域执业的律师通常使用不同的电话号码来适当地转接电话。
>     
> *   本地号码通常更可取，因此在不同州或在一个州内执业的律师更喜欢该州/市/县的本地号码。
>     
> *   许多律师保留旧手机是因为手机中已有的数据，他们可能不知道如何移动或不希望移动。
>     
> *   出差的律师可能更喜欢在某些城市比在其他城市更好的某些运营商。
>     
> *   最后，跨国旅行的律师通常有他们要去的国家的专用电话(例如，如果去中国，出于安全原因，律师可能希望有一个空白电话，以防电话被搜查或手机数据被窃听)。

因此，如果你发现你的律师囤积了十几部手机，也许他们正处于最好打电话给索尔的情况，也许他们只是有很多分散的客户，在多个州或国家做生意。像 [在七个州的秘密房地产交易](https://splinternews.com/sean-hannity-reportedly-owns-a-real-estate-empire-1825459754#_ga=2.50762667.367633666.1525095724-3846207152.1521480874) ，或者 [秘密布拉格之旅](https://splinternews.com/surprise-michael-cohen-apparently-lied-about-a-trip-to-1825264277#_ga=2.50762667.367633666.1525095724-3846207152.1521480874) 。
