# 你的幸福是建立在小事上的

"幸福是由一小步一小步实现的，但绝非小事."

杰出哲学家的生活 7.1.26



### 这意味着什么

幸福，或者说一种普遍的满足感，是一个崇高的目标，但并不总是容易实现的。它需要你采取“小步骤”，日积月累，逐渐塑造你的性格和困境。没有捷径、诀窍或一夜之间的蜕变。它是你一生中在自己内心发展的东西，而不是你在偶然的时刻发现的东西，就像人行道上的零钱。

### 从中吸取什么

你满足的顶峰就像一个用砖砌成的家，这个结构中的每一块砖都是幸福的微小象征，一件让你微笑、让你感到安全和温暖的小事。一块砖可以是任何东西——你的家人、你的朋友、在办公室完成一项任务、选择呆在家里而不是出去、寒冷早晨的一杯热咖啡、一个让你发笑的笑话、你在漫长的一天工作后脱掉鞋子时的感觉、阳光照射在特定叶子上的方式...任何事。一些砖块是通过感激和欣赏找到的，而另一些则必须通过工作、纪律和热情获得。找到你的完美幸福状态需要你以自己选择的方式收集和构建它们。

但是在你建造的时候不要感到被践踏。记住，建立你的满足感需要时间，这“真的不是一件小事”每天醒来，准备好向你的幸福迈出“一小步”,并认识到这将是一项持续的工作。采取好的行动，不管多小；做出有益的选择，无论多么简单；对你所拥有的表示感激，不管多么基本。假以时日，这些小事情会累积成真正伟大的事情。
