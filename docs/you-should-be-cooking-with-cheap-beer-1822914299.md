# 你应该用便宜的啤酒做饭

> 原文:[https://life hacker . com/you-should-to-cooking with the 廉价啤酒-1822914299](https://lifehacker.com/you-should-be-cooking-with-cheap-beer-1822914299)

便宜的啤酒名声不好，但是这些天我越来越倾向于选择它而不是高档的。与浓烈的 IPAs 不同，普通的 lagers 不会与食物竞争，也不会让我宿醉，而且用它们做饭绝对是一种享受。

葡萄酒垄断了用酒做菜的市场；除了啤酒罐鸡肉或吉尼斯蛋糕，你很少听说以啤酒为中心的食谱。对于葡萄酒，经验法则是只用你喜欢喝的东西烹饪，这对那些有机会获得美味、廉价葡萄酒的人来说是一个很好的建议，但很难在“足够好喝”和“足够便宜炖”之间找到平衡。这就是为什么我如此喜欢用啤酒做菜，也是为什么我认为更多的人应该尝试一下，尤其是那些用劣质酒毁了一道难做的菜又害怕再尝试的人。

显然，啤酒不能复制你从葡萄酒中得到的味道，但它提供了用酒烹饪的所有好处，而且风险很小。下面是一罐你最喜欢的啤酒比葡萄酒的味道:

*   **温和风味和低酒精:**酒精比水更容易溶解脂溶性风味化合物，这有助于形成丰富、深刻的风味。由于啤酒的味道和酒精含量比葡萄酒轻，你可以把它倒入而不会压倒其他成分。
*   得益于淀粉和糖的混合，啤酒在稀释后增加了一种完全甘美的感觉。
*   **碳酸化:**气泡以物理方式提升一道菜——想想啤酒烤鱼或啤酒速食面包——并使肉变嫩。
*   一次性容器: Gross wine 有一种方式可以提醒你的选择。无论你喝瓶装的还是瓶装的，啤酒都不会像这样伤害你。
*   **访问:**在宾夕法尼亚州，25 美元可以买到一箱米勒淡啤或者*或者*两瓶葡萄酒，这是一个显而易见的理财选择。此外，根据你当地的法律，你可以在杂货店买啤酒和你的其他配料，而不必在酒类商店停下来买一些辣葡萄汁。

最重要的是，啤酒的数量是已知的。你可以毫不费力地想象出你最喜欢的啤酒的味道，但买一瓶不熟悉的廉价葡萄酒就像玩品尝记录轮盘赌。当然，也有便宜的葡萄酒不是涂料稀释剂和糖水的邪恶组合——但是一个不幸的猜测可能会毁了晚餐。

说到油漆稀释剂和糖水，用啤酒煮的时候跳过麦芽酒；IPAs，浓麦芽酒，酸奶和比利时酵母通常也不喝。任何便宜的淡啤酒都很棒——我喜欢米勒淡啤——就像黑啤一样。基本上，用你最喜欢的便宜的 [喝啤酒](https://www.youtube.com/watch?v=XtMy5IBmX7E) ，不管是什么。

用像啤酒这样的中性酒精烹饪的美妙之处在于可能性是无穷无尽的；除非你在寻找一种特定的葡萄酒风味，比如葡萄酒炖鸡或马萨拉鸡肉，否则只要你想用葡萄酒，就用啤酒。这些是我个人的最爱:

*   牛排法士达:我把厚牛排上的贝杰斯肉捣出来，切洋葱，然后把它们一起浸泡在啤酒、酸橙汁、油、大量盐和香料的混合物中过夜。
*   蘑菇炒面:啤酒特别适合搭配任何朴实的食物。我几乎把洋葱和大蒜焦糖化，用啤酒脱釉，然后加入蘑菇。
*   下一次你做你选择的蔬菜时，加入一些啤酒，用大量的大蒜做一个长时间的慢炖菜。
*   **任何有白菜的东西:**啤酒和白菜T3】为对方做了 T5】。我对掺啤酒的罗宋汤和 [schi](https://www.thespruce.com/traditional-russian-cabbage-soup-shchi-recipe-1135534) 的热爱是强烈的； [红烧白菜](https://www.epicurious.com/recipes/food/views/braised-red-cabbage-231603) 配黑啤是神物。
*   **香肠和辣椒:**红烧香肠和啤酒炒辣椒和洋葱；完了，完了。
*   法式洋葱汤:最后用一点黑啤来代替高汤，或者除了高汤之外再加点。
*   **辣椒:**无论你是在做墨西哥辣酱、辛辛那提式辣椒，还是纯素食辣椒配南瓜，一罐啤酒(也许还有一些可可粉)就是你所需要的。

如果你也用啤酒做菜，我很想听听你最喜欢的食谱——尤其是如果你用它来做披萨面团的话。很多人都信以为真，但我还没有找到我喜欢的食谱，我真的想开始做。
