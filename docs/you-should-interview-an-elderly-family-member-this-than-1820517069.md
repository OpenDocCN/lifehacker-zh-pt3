# 这个感恩节你应该采访一位年长的家庭成员

> 原文：<https://lifehacker.com/you-should-interview-an-elderly-family-member-this-than-1820517069>

两年前的感恩节，在吃完我祖母安妮住的养老院为我们准备的火鸡午餐后，我拿出我的 iPhone，点了“录音”我的奶奶 88 岁了，她的健康每况愈下，但是她有一个很好的日子，所有的家人都来看她。于是我把椅子挪近她，问了我事先计划好的问题。我想听听她的过去，那些烙在她记忆中的故事，那些塑造了她的时刻。

Watch

奶奶一直是一个天生的讲故事的人，她过去曾和我妹妹分享过她在战时利物浦的童年的点点滴滴。但是这次采访很特别。她谈到自己七岁时，站在父母自助洗衣店的橙色板条箱上，为顾客熨衬衫。她告诉我们她是如何逃离一场迫在眉睫的包办婚姻，与她一生的挚爱——我的祖父——在一起的。她描述了他们如何搬到美国，住在她叔叔餐馆的储藏室里。

那次谈话后仅仅两个月，我奶奶就去世了。我剪辑了一些采访片段，在她的追悼会上播放。如果你想听的话，就在这里:

正如 StoryCorps 的创始人戴夫·伊赛所说，“灵魂包含在人的声音中。”你爱的人的录音有一种神奇的东西，不同于视频。在采访我的祖母时，我使用了 StoryCorps 的[The Great Thanksgiving Listen](https://storycorps.org/participate/the-great-thanksgiving-listen/)提供的工具，这是一个鼓励人们通过录制对一位长者的采访来创造美国口述历史的运动。

想试试吗？选择一个你想采访的人，然后去采访。

一些录音和面试技巧:

*   提前准备一些问题，但在谈话中要留有余地，以防意外。 StoryCorps 有 [一个庞大的问题列表](https://storycorps.org/participate/great-questions/) 来帮助你的受访者展开话题。一些好的通用问题:谁是你生命中最重要的人？你一生中最快乐的时刻是什么？最难过？谁对你的生活影响最大？你一生中对你最好的人是谁？你在生活中学到的最重要的经验是什么？你最早的记忆是什么？你对我最美好的记忆是什么？

*   准备好你的录音设备。那可能只是你的手机。有一个免费的 [应用程序](https://storycorps.org/participate/storycorps-app) 用于伟大的感恩节监听，它可以轻松地记录你的对话，并将其存档在国会图书馆和 StoryCorps 网站上，但你也可以使用另一个应用程序，如语音备忘录，或实际的录音机来获得更好的音频质量。

*   找一个安静舒适的地方，开始录音。然后提出你的问题，并仔细倾听你所爱的人讲述他们的故事。
