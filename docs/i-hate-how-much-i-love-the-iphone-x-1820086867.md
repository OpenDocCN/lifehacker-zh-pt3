# 我讨厌我有多爱 iPhone X

> 原文：<https://gizmodo.com/i-hate-how-much-i-love-the-iphone-x-1820086867>

我是一个电子产品迷，五年前，一部新的智能手机让我对自己点头称是，“是的，我需要它胜过需要空气。”但围绕 iPhone X 的传言让我比往常更加兴奋。不仅仅是因为 iPhone 最终放弃了边框，采用了有机发光二极管显示屏——三星的 Galaxy S8 在 3 月份失去了边框——还因为 iPhone X 是该系列自天知道什么时候以来的第一次重大检修。我应该知道不要被宣传所迷惑，但在花了近一周的时间使用这款设备后，我实际上说服了自己，花 1000 美元买一部手机似乎是个好主意。如果你恨我这么说，没关系，我也恨我自己。



显示屏显然是你注意到的第一件事，如果不是因为著名的 notch，它只会像三星 Galaxy S8 或 Note 8 一样令人印象深刻。色彩没有你在三星手机上看到的那么鲜艳——三星手机往往会使色彩过于饱和——但作为一个喜欢准确性而不是出血流行的人，我真的很喜欢 iPhone 渲染色彩的自然方式。

但我只想说:我喜欢独特的缺口。有些人认为它很碍眼，但对我来说，这个缺口以一种非常令人满意的方式突显了手机的炫耀。它给了整个 5.8 英寸显示屏一种空间感，这在竞争对手的手机上根本找不到。事实上，它的显示屏实际上比物理尺寸更大的 iPhone 8 Plus 上的 5.5 英寸屏幕更大。

当屏幕围绕前置摄像头模块弯曲时，感觉有点像苹果以某种方式欺骗并奖励了手机更多的视觉空间，至少当你在主屏幕上或在专门更新的应用程序中使用 iPhone X 的独特显示屏时。在其他应用程序中，iPhone X 感觉有点小，显示屏顶部和底部的黑带限制了曾经的膨胀感。

大显示屏不会对电池寿命产生太大影响。我从来没有发现自己在一天中想要更多的能量，我甚至在第二天还经常使用。在我们的电池电量耗尽测试中，我们通过 wi-fi 播放 YouTube 视频，直到手机电量耗尽，iPhone X 的电池续航时间略低于 10 小时，与 Gizmodo 今年测试的所有手机相比，这完全是平均电池续航时间，由于它能够快速无线充电，重新充电从来都不是问题。

这款手机最让我惊讶的是，我并不想念触控 ID 或 home 键。自从第一次有传言说触控 ID 可能撑不到苹果的十周年 iPhone，我就一直痴迷于指纹技术的转变。他们怎么可能做这件事！这成为了现实，和许多人一样，我对这项技术将永远被 Face ID 取代感到震惊。你现在不用用手指按 home 键，而是扫一眼手机，用拇指快速向上滑动。

在过去的几天里， [我花了几个小时摆弄](https://gizmodo.com/watch-me-try-to-break-face-id-a-bunch-of-different-ways-1820020355#_ga=2.92707357.1298675679.1509582084-923911060.1509115869) 的前置“真实深度”设置，使 Face ID 魔术成为可能。这是迄今为止 [手机最具技术创新的方面](https://gizmodo.com/iphone-x-first-look-lets-talk-about-that-whole-face-id-1819989200) 。它使用相机、红外激光器和红外扫描仪的组合来创建面部的 3D 地图。相机可以让你 [创建 animoji](https://gizmodo.com/5-things-you-can-do-with-animoji-1820002741) ，而 [让 Snapchat 这样的应用程序访问你的面部地图](https://gizmodo.com/whats-really-up-with-apple-giving-face-data-to-app-deve-1820085175#_ga=2.160315709.1298675679.1509582084-923911060.1509115869) ，这样他们就可以在你的脸上扔出看起来很逼真的面具。

但杀手级特征显然是 Face ID。我看着手机，它立刻打开了。从那以后，它就像一个更加华丽的触控 ID 版本。当我登录 Twitter 时，它认出了我的脸，让我登录了。当我使用 Apple Pay 无缝订购午夜汉堡时，它会扫描我的面部，然后让我按下电源按钮确认购买。

虽然一开始有一些打嗝，但自从我周一从苹果公司拿起电话以来，原深感摄像头已经学会了我的脸。现在，当我试图在黑暗中使用它的时候，或者在化浓妆的时候，或者早上穿着低低的连帽衫挤在狗公园的长椅上的时候，它都不会犹豫。这个相机让我着迷，我甚至不用把它直接举在面前。只要相机对着我的脸，我看着它，手机就会打开。

iPhone X 上的后置摄像头做不出那么酷的东西。它们仍然非常好，但如果你试图在 iPhone 8 或 8 Plus 或 iPhone X 之间做出决定，那么后置摄像头不会让你动摇。

你购买 iPhone X 的原因是因为它过于奢华的设计，大胆的享乐主义显示器，以及超级酷的深度感应自拍相机。这款手机并没有像最初的 iPhone，甚至是 iPhone 4 那样，以一种宏大的方式重塑我们对手机的看法。虽然 Face ID 可能是手机生物识别技术的未来，但这种未来不会像从密码到生物识别那样戏剧性地改变你的世界观。iPhone X 并不是向手机设计新世界的飞跃，它只是一个展示真正昂贵的豪华手机应该是什么样子的橱窗。我不需要 1000 美元的 iPhone X，但在用了几天后，我真想去买一部。
