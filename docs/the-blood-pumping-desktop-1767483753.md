# 令人热血沸腾的桌面

> 原文：<https://lifehacker.com/the-blood-pumping-desktop-1767483753>

读者 [麦克塔库斯](http://kinja.com/PaladinJ) 将这个跳动的心脏桌面提交给了我们的 [桌面展柜](http://kinja.com/tag/desktop-showcase) ，并以动画的形式，情不自禁地展示给世人。以下是他是如何做到的，你也可以这样做。

Watch

这里 有很多 [的雨量计，但都很好，看起来都很棒。](https://lifehacker.com/how-to-create-an-attractive-customized-desktop-hud-wit-5828789)

如果你想复制这种造型，你需要以下这些:

*   制作动画作品的壁纸和自定义剪切都在麦克塔库斯的收存箱中
*   用于 Windows 的 [雨量计](https://www.rainmeter.net/) 系统调试和监控工具
*   [Monstercat 可视化器](http://marcopixel.deviantart.com/art/Monstercat-Visualizer-for-Rainmeter-1-3-486330771) 为音频可视化器的雨量计皮肤
*   [迂回两个](http://flyinghyrax.deviantart.com/art/CircuitousTwo-376652231) 皮为雨量计为音量、CPU 和 RAM 系统监视器
*   我需要一件夹克吗？左下方皮肤为雨量计为天气
*   雨量计的 [简单媒体皮肤](http://lilshizzy.deviantart.com/art/Rainmeter-Simple-Media-1-1-212678796) 用于时间和日期
*   雨量计的 [Vark](http://toastbrotpascal.deviantart.com/art/Vark-for-Rainmeter-171756034) 皮肤用于文本快捷方式，标签与 [Basic5](http://www.deviantart.com/art/BASIC5-TEXTUAL-102918120) 皮肤和世纪教科书字体将它们联系在一起。

这里有许多移动部件，但它们结合起来创造了真正独特的东西——说真的，我们以前从未见过这样的东西，这是一个很好的例子，说明有人将他们的桌面真正变成了自己的，而不是只是拿走给他们的东西。这是最完美的个性化。如果你也喜欢它，去下面链接的 McTarkus 的 kinja 博客让他知道你有多喜欢它！
