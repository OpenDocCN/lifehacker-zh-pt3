# 在家工作时如何对抗孤立

> 原文：<https://lifehacker.com/how-to-fight-isolation-when-you-work-from-home-1823220279>

尽管你对你的同事不断嘟囔着“那是她说的”翻了翻白眼，尽管每次你在厨房遇到她时都要听 HR 的《猫的故事》中的颂歌很尴尬，但当你开始在家工作时，你开始怀念在办公室里的日常人际交往。周末回顾。午餐搭档。当你穿着和某人相似的服装出现并一起拍照的时候，标签是“办公室双胞胎”。

在家里，有...没人。禁闭太多天，你会忘记如何与人相处。它会变得孤独——甚至令人沮丧——如果你不改变你的日常生活，它会影响你的一生。

远程员工和自由职业者告诉我，他们在家工作时必须积极对抗孤立。这里有一些方法可以做到这一点(并不是所有的方法都需要穿上真正的裤子)。

在咖啡馆工作。即使每周这样做一次也能给你带来动力。我很感激我家附近的咖啡店没有挤满在笔记本电脑上打字的超级认真的专业人士(我猜除了我之外)。人们在那里交谈，噪音让我充满活力。

在酒吧工作。 [这就是为什么](https://lifehacker.com/why-i-work-at-bars-instead-of-coffee-shops-1823164289) Lifehacker 食品&饮料编辑 Claire Lower 对它深信不疑。

和其他在家工作的人一起吃午餐。不知道为什么，第一次做的时候感觉好奢侈。此外，如果你们在相似的领域工作，你们可以谈谈工作，互相交流想法。

**看看在** [**租一张桌子一个联合办公空间**](https://lifehacker.com/join-a-coworking-space-to-network-and-find-mentors-1785524556) **。如果你只是偶尔想换换风景，许多餐馆会提供一日游。**

去健身房。我的朋友 Soo 说，虽然她过去喜欢跑步和游泳等个人活动，但自从她开始独自工作后，她更喜欢瑜伽和拳击等社交活动。“在家工作会因为缺乏结构而产生决策疲劳——所以我需要有人花一个小时告诉我到底该做什么，何时以及如何呼吸，以及我的手和脚应该放在哪里。”

做志愿者或找一份兼职。如果你是一名自由职业者，有一些空闲时间，一份灵活的兼职工作或志愿者项目可以迫使你与你圈子之外的人联系。我听说过创意产业中的自由职业者，他们得到了咖啡师、步行导游、电影引座员和教师的工作。他们之间的互动很有趣，有助于激发他们的创造力。

寻找要加入的当地团体或要参加的课程。查看 Meetup、脸书或 Nextdoor 寻找创意。

关注工作之外的社交生活。Lifehacker 的特约撰稿人 Patrick Allan 告诉我，对他帮助最大的是确保他每周至少有三天与朋友有约。“可以是欢乐时光、游戏之夜、一起健身，或者只是看一场电影。提前计划好事情，这样你就可以一直有社交活动可以期待。期待对*有所帮助，所以*很大

在网上建立联系。虽然 Lifehacker 的健康编辑 Beth Skwarecki 每天都有各种面对面的接触，但她告诉我，“老实说，我大部分的人际交往都是在网上进行的。我在推特和脸书上与人们联系，我们的工作很轻松。Slack 之前的日子对我来说很孤独！”

养一只宠物。他们真的可以成为最好的办公室伙伴。作家凯瑟琳说:“我跟我的狗说话，就像她是我的朋友和同事一样。很奇怪吗？我不在乎。当我在下午 3 点穿着睡衣决定我是否可以不洗澡就去接我的孩子时，她不会评判我。”

如果你是一名远程员工，向你的经理提出建立联系的策略。有时候，还有其他远程员工在你所在的城市工作。看看你的公司是否会赞助季度聚会来帮助建立一种更强的工作文化。

拥抱孤独。在问人们如何对抗孤立时，一位朋友埃迪回答道:“对抗孤立？我邀请它！工作就是这样完成的。如果说有什么不同的话，那就是我在与分心和互动做斗争。”这很容易让人产生绿草如茵的心态，但是记住在家工作的乐趣是很重要的。你自己一个人！拥抱光荣的孤独。如果*变得太*安静， [这里有一个播放列表](https://lifehacker.com/the-work-from-home-week-playlist-1823087515) 陪你。
