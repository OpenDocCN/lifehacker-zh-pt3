# 事实证明，益生菌可能会伤害你

> 原文:[https://life hacker . com/probiotics-can-potentially-beat-you-it-turn-out-1828803797](https://lifehacker.com/probiotics-can-potentially-hurt-you-it-turns-out-1828803797)

药店里充斥着不一定能治愈我们的东西——维生素 C、顺势疗法滴剂、益生菌——但我们耸耸肩，还是买了它们，因为，嘿，它们 [伤害不了](https://lifehacker.com/the-placebo-effect-is-no-reason-to-buy-junk-1774878033) 。但是现在我们有一些具体的证据表明，如果益生菌不完全适合你试图治疗的健康状况，益生菌*可能会造成伤害。* 

Watch

[新的研究](https://aem.asm.org/content/aem/early/2018/08/27/AEM.01408-18.full.pdf?ijkey=oR5/yBXItwBkw&keytype=ref&siteid=asmjournals) 是在老鼠身上进行的，但它对我们人类来说是一个警告。研究人员正在寻找一种治疗 [*隐孢子虫*](https://www.cdc.gov/parasites/crypto/index.html) 的方法，这种寄生虫通常会导致幼儿腹泻感染。没有任何有效的药物或疫苗来对抗这种情况，所以研究人员想知道益生菌是否有帮助。他们用 Crypto 感染老鼠，并给其中一些老鼠服用市售的益生菌。(他们没有命名益生菌。)但事实证明，服用益生菌的老鼠比没有服用益生菌的老鼠感染更严重。

我们称益生菌为“好细菌”，但它们不是对抗感染的超级英雄；它们只是无害地和我们生活在一起的细菌。科学家们仍然在困惑我们的肠道内到底发生了什么，以及每种微生物扮演了什么角色。在某些情况下，益生菌有帮助，但这取决于哪种益生菌，你试图治疗的是什么感染或健康状况，也许还取决于你的肠道已经被什么微生物称为家。所以我们不应该假设益生菌总是准备好击退“坏人”的“好人”

益生菌有着良好的记录，在某些时候。一些特定的品牌已被证明在治疗某些特定的条件或症状有用。(其中 [列举的就是这里的](https://vitals.lifehacker.com/this-guide-to-probiotics-tells-you-which-brands-actuall-1779854845#_ga=2.193566573.1177246067.1536068232-1723114163.1524514905) 为例。)益生菌的研究还处于起步阶段。我已经说了很多年了，因为这是真的:我们不知道益生菌在每种情况下会做什么。当你决定尝试益生菌 时，你总是在做一些自我实验 [。也许你会变得更好，也许它什么也不会做，也许你会感觉更糟。](https://vitals.lifehacker.com/what-to-know-about-prebiotics-and-probiotics-1825401708)

在这种情况下，它就像任何其他补充剂或有问题的药物。例如，顺势疗法药物的 [基本上应该是水](https://vitals.lifehacker.com/what-homeopathy-is-and-why-its-useless-1795694290) ，但也有案例表明它们的 [含有毒](https://www.forbes.com/sites/brucelee/2017/01/28/fda-toxic-belladonna-in-homeopathic-teething-products/#77ae1ba640dd) 或者，最近的 [污染了水](https://www.fda.gov/NewsEvents/Newsroom/PressAnnouncements/ucm618691.htm) 。感冒药是真药，但往往 [根本不太管用](https://vitals.lifehacker.com/how-to-find-the-cold-medicines-that-actually-work-1822637530) 。维生素是健康所必需的，但是如果你吃得太多，它们会让你生病。现在，我们可以把益生菌添加到可能对你有好处的食物清单中，但是 [也可能适得其反](https://vitals.lifehacker.com/if-it-has-no-side-effects-it-probably-doesn-t-work-1818771976) 。
