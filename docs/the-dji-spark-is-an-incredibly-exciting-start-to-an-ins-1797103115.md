# DJI 火花是一个令人难以置信的令人兴奋的疯狂未来的开始

> 原文:[https://gizmodo . com/the-DJI-spark-is-a-exceible-excited-start-to-an-ins-1797103115](https://gizmodo.com/the-dji-spark-is-an-incredibly-exciting-start-to-an-ins-1797103115)

小型无人机并不新鲜。玩具大小的四轴飞行器已经上市多年，以相对合理的价格和不太专业的知识帮助孩子(和爸爸)开始飞行。然而，小型无人机几乎可以做大型无人机能做的任何事情？那是新的。这就是为什么 DJI 火花如此激动人心。

关于火花，你会注意到的第一件事，也是最重要的事，是它的大小。它很小。它太小了，让非常小的 Mavic Pro 看起来像一个肥胖的巨人。如果 Mavic Pro 有意大利三明治那么大，那么 Spark 就有美味的奶油甜卷那么大。它重 300 克，也和一个煎饼卷一样重。自从 [最近的一项法院裁决](http://money.cnn.com/2017/05/19/technology/drone-registration-faa/index.html) 发现美国联邦航空管理局(FAA)不能要求业余爱好者注册他们的无人机后，你就可以把火花从盒子里拿出来，开始飞行取乐了。如果您将它用于商业目的，这些规则仍然适用。

这也是为什么需要两个人来检查这个小小的四轴飞行器的部分原因。迈克尔是一名有执照的商用无人驾驶飞机驾驶员，所以他负责操纵控制。亚当是一名休闲飞行员，所以他担任观察员(兼摄影师)。作为一个真正的父亲，你应该经常和朋友一起做无人机飞行。至少，当你试图盯着飞机时，两套眼睛会派上用场。既然火花如此之小，你将需要所有你能得到的帮助。

### 这是什么微型无人机？

概括地说，Spark 拥有与更大的折叠式 Mavic Pro 相同的所有功能，但一切都被简化了。Spark 的最大速度为 31 英里/小时，没有 Mavic Pro 的 40 英里/小时快。Spark 的最大传输距离为 1.2 英里，不能飞得像 Mavic Pro 那样远，Mavic Pro 的射程为 4.3 英里。由于电池只有 Mavic Pro 的一半大，Spark 不能飞那么久。一次火花充电能让你飞行 16 分钟。Mavic Pro 的电池续航时间为 27 分钟。Spark 上的 1080p 摄像头和双轴万向节远不如 Mavic Pro 上的 4K 摄像头和三轴万向节。

然而，与 Mavic Pro 和 Phantom 4 一样，DJI Spark 采用了避障技术和额外的传感器，可以实现智能飞行模式，如轻击飞行、主动跟踪和手势。最后一个是火花真正突出的地方。由于飞机前面有一个红外传感器，你实际上可以用手掌控制火花。很多人把这叫做“绝地模式”，当它起作用的时候，非常酷。

这尤其令人兴奋，因为 Spark [只需要 500 美元](http://store.dji.com/product/spark) 。这使得它不仅是 DJI 有史以来最小的无人机，也是最便宜的。但有一个陷阱，500 美元的价格标签不包括控制器的成本，你肯定会想带着控制器飞火花。

要得到一个控制器，你必须花 700 美元购买的 Spark Fly More Combo。组合配备了许多你肯定想要的其他东西，如螺旋桨护罩、额外的螺旋桨和额外的电池。然而——这是一个大写的“H”——让我们提醒你，你不需要控制器来飞火花。你可以用手来飞，也可以用智能手机或平板电脑。对于生活中不需要另一个操纵杆的新手来说，这很好，但对于经验丰富的无人机飞行员来说，这种体验可能并不理想，因为他们喜欢控制器的触感。

### 这架微型无人机在空中表现如何？

把 Spark 想象成一架个人无人机。它的一切都是为了让你感到安全和可控——特别是如果你跳起来的时候有螺旋桨警卫。技术上可以用手放飞火花，用手指做个相框自拍。把火花扔进背包，去加州度假。它可以从你的手掌上起飞，跟踪你和你的朋友，当你在一棵红杉旁摆好姿势，拍张照片，然后降落在你的手掌上。除了加州度假的事，我们做了这个。成功了。

但是手势控制远非完美。你真的需要学习不同的手势，并稍微训练一下自己，让它们完全正确。即便如此，你能做的也非常有限。基本上，火花会给你拍照，并在你手掌几英尺内飞行。这充其量是一种室内把戏。别想在风中尝试。火花在微风中来回跳动，这似乎让红外传感器无休止地困惑。

但这项技术仍然感觉像是令人兴奋的新一波无人机的第一代，它们工作起来不费吹灰之力，只需要训练有素的人来驾驶它们。又或许，在未来，这些无人机会有感知能力，接管世界。我们还不知道，这就是为什么它如此令人兴奋！

### 它除了自拍还做什么？

问题是，你根本不需要手势控制。对于一架已经很牛逼的无人机来说，这是一个有趣的奖励。这就像是无人驾驶飞机的马自达·米亚塔。当然，这不是你能买到的最大或最强大的东西。但是非常有趣。

我们真的可以看到火花对想要成为无人机赛车手的人来说是额外的乐趣。虽然 31 英里/小时不是 DJI 无人机的最快速度，但当你在运动模式下飞行 Spark 时，感觉很快。因为 Spark 的大小与你在 ESPN 无人机竞赛联盟中看到的无人机大致相同，你会开始觉得你可以掌握这项爱好的诀窍。最令人失望的是，Spark 目前无法与该公司的第一人称视角(FPV)耳机 DJI 护目镜兼容。

与此同时，这款相机非常适合拍摄简单的东西，如自拍或拍摄城市天际线的鸟瞰图。然而，Spark 相机确实做得不好的一件事是向上或向下倾斜相机镜头。准系统双轴框架移动不顺畅；它本质上是从一个位置跳到下一个位置，这将使火花对希望平滑平移的初露头角的电影摄影师无用。

如果你发现自己对一些小缺点感到失望，比如抖动的万向节或缺少 FPV 护目镜，那么火花可能不适合你。你可能已经拥有一辆幻影或者 Mavic Pro，或者，谁知道呢，一辆 3300 美元的 Inspire。不过，你可以考虑给你的孩子买一个火花。见鬼，给你喜欢找乐子的妈妈或者你说服了的好朋友买一个。这是一种昂贵的无人机入门方式，但对于合适的人来说是值得的。

### 你应该买火花吗？

但是在你花钱之前，考虑一下你的使命。你是一个初学者，希望得到第一架可以可靠地用于大多数目的的无人机吗？火花是个不错的选择。你是一个长期的幻影车主，正在寻找更便携的东西吗？Spark 是一个很好的选择，但 300 多美元的 Mavic Pro 更好。你是一个希望在好莱坞电影中拍摄一些镜头的空中摄影师吗？你现在甚至不应该读这篇文章，因为你应该为 [攒下 5000 美元的 DJI 矩阵](http://store.dji.com/product/matrice-600-pro) 。

这是另一种说法，随着 Spark 的加入，DJI 确实为每个专业水平出售无人机。令人印象深刻的是，500 美元的 Spark 对大多数人来说刚刚够用。不管你的飞行员水平有多高，火花飞行确实很有趣。

随着年龄的增长，它可能会变得更好，这要归功于潜在的固件更新和手势控制的改进。否则，这将是一个令人兴奋的无人机未来的神奇一瞥，这种飞机比我们想象的可能更小，可以做比以前更多的事情。
