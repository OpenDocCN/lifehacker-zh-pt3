# 关于信用合作社的四个神话被揭穿

> 原文：<https://lifehacker.com/four-myths-about-credit-unions-debunked-1787164603>

在 2008 年银行业危机和最近富国银行的惨败之间，人们很容易对银行持怀疑态度。如果你不喜欢大银行，还有一个选择:信用合作社。信用合作社已经变得越来越受欢迎，但是对于他们如何工作和他们提供什么仍然有很多误解。



### 神话:信用合作社不赚钱

银行是盈利机构，像其他企业一样，从客户身上赚钱。信用合作社 [的工作方式稍有不同](https://lifehacker.com/the-differences-between-banks-and-credit-unions-1748879210) 。像银行一样，他们仍然保留你的钱并提供贷款——但他们不以盈利为目的，拥有“会员”而不是客户。当你在信用社开了一个账户，你就成为了会员和部分所有者，这基本上意味着你可以在董事会投票。

传统上， [许多信用社](http://lifehacker.com/five-best-credit-unions-1656438536)[也旨在支持他们的社区](http://www.cdcu.coop/about-us/what-is-a-cdcu/) ，推广金融知识，并提供 [金融咨询](https://www.americafirst.com/services/other-services/financial-counseling.cfm) 。然而，正如一位读者指出的，这并不是说信用社对赚钱不感兴趣:
