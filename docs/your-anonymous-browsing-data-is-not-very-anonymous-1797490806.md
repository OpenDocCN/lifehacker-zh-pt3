# 你的“匿名”浏览数据不是很匿名

> 原文：<https://lifehacker.com/your-anonymous-browsing-data-is-not-very-anonymous-1797490806>

即使你通过打开一个新的匿名窗口来掩盖你的踪迹，你的网页浏览历史可能并不像你想象的那样隐私。关于你在网上做了什么的信息，甚至每个网址，都可能被任何想要的人在网上买到。虽然在大多数情况下，人们出于营销原因进行购买，但他们也可能选择恶意使用他们新发现的知识。



记者斯维娅·埃克特与一名数据科学家合作，研究他们到底能买到什么样的个人互联网用户数据。事实证明，答案是很多。

两人创建了一家虚假的营销公司，拥有一个虚假的网站和职业页面，然后开始购买信息，特别是原始浏览历史，伪装成他们需要它来训练一个将彻底改变营销的人工智能平台。他们在拉斯维加斯的黑客会议 DefCon 上展示了这个项目， [*【卫报】*报道](https://www.theguardian.com/technology/2017/aug/01/data-browsing-habits-brokers) 。

他们买的是三百万德国人的匿名浏览历史。虽然他们中的一些人只列出了几个网址，但其他人有他们的整个浏览历史，色情和所有的，在那里可以看到。该团队实际上甚至不需要购买它——一个数据经纪人免费给了他们。虽然它应该是匿名的，但他们很容易很快就能找出一些用户是谁。

那么这是怎么做到的呢？

一旦他们解释清楚，答案就很明显了。比如，看你的 Twitter 分析页面，这是一个很大的提示，表明你就是那个人，因为只有你能看到。这一招同样适用于一家名为 Xing 的德国网站。

对其他人来说，这有点困难，但仍然很容易。想想有多少人在你的公司工作，并正在查看类似内部人力资源页面的内容，然后通过可能在你隔壁组的人再次过滤。人不多吧？有了组中正确的 URL，缩小数据所有者的范围并不那么复杂。

就德国人的数据而言，大部分数据是从一个浏览器扩展中收集的，该扩展在其使用条款中声明向其出售数据。

要点:如果你想保护你的浏览数据的隐私，就不要让任何人拥有它。即使在发送给他们的时候是匿名的，也很有可能与你有关。如果你需要帮助锁定东西， [这里有一些关于保护你和你的数据安全的提示](http://lifehacker.com/everyones-trying-to-track-what-you-do-on-the-web-heres-5887140) 。
