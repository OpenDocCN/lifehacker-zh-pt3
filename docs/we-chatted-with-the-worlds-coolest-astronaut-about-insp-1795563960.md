# 我们和世界上最酷的宇航员聊起了激励孩子和把狗送上太空的事情

> 原文：<https://gizmodo.com/we-chatted-with-the-worlds-coolest-astronaut-about-insp-1795563960>

利兰·梅尔文是人类的独角兽。他被国家橄榄球联盟选中，在最后的边疆飞行(两次)，现在， [写了一本书](http://www.barnesandnoble.com/w/chasing-space-leland-melvin/1124484021) 叫做*追逐太空*。在互联网上，这位前美国宇航局宇航员最出名的是这张令人难忘的照片 [和他的两只狗](http://gizmodo.com/the-story-behind-this-astronauts-viral-photo-is-even-cu-1791888042) ，杰克和斯科特登上了他新回忆录的封面。Gizmodo 与 Melvin 坐下来谈论蒸汽领域的多样性、教育，当然还有太空中的好狗。

* * *

雷伊·保莱塔:为什么你的生活如此酷？

**利兰·梅尔文:**我的生活“酷”？你在说什么？

RP:嗯，你曾经是一名足球运动员，一名宇航员——这是每个孩子的梦想，也许除了《侏罗纪公园》和《T2》中的恐龙牧人。

我从来没有计划过这些东西，它只是蜿蜒到这些空间。很多人支持我，说我可以尝试不同的东西。我从未想象过真正的太空之旅。

Ryan F. Mandelbaum:我仍然想知道一个人是如何踢足球和去太空的。这是怎么发生的？

LM: 有很多工作，很多纪律。我从五年级开始踢足球。所以这是多年的准备和重复的快乐。后来，我在一场高中比赛中，在达阵区触地得分，我以为我的教练会揪住我的面罩把我拉出去。他说，“回去接球，”我回去接球，这让我获得了里士满大学的足球奖学金。

这让我明白，如果你不放弃——凭借勇气、毅力和决心——你可以做大事。这就是我的旅程，就像那个小引擎，你知道吗？

RFM:你什么时候知道你想成为一名宇航员的？

1996 年宇航员班的“沙丁鱼”——他们有 45 个人，他们像沙丁鱼一样聚在一起。当我儿子进去后，他飞回了美国国家航空航天局兰利总部，在那里我和约翰·杨一起工作——约翰·杨在月球上漫步。有一天，我和约翰·杨聊天，他说，“利兰，你应该申请！”我想，“约翰·杨说我应该申请，所以我想我应该申请，”所以我申请了，我被录取了。我想这是 1997 年。

显然，成为一名宇航员是有史以来最酷的工作，但它非常困难。你面临的最大挑战是什么？

**LM:** 我就在这次训练事故中。有一天，我穿上白色的米其林男式套装，来到这个 600 万加仑的水池中进行太空行走训练，他们忘记把我的垫子放在那里了——这是一个瓦尔萨尔瓦垫子，一个可以压着鼻子清理耳朵的垫子。我的不在里面。所以我不能清理我的耳朵，我失去了我的听力，有血从我的耳朵里流出来，而且不得不失去一种感觉——我的大脑不得不重新连接自己才能再次听到。我不知道我是否还会飞，或者根本不会飞。然后，事情开始发生，我变得更好，我得到了飞翔的机会。

RFM:你在 STEAM 中倡导多样性，做了很多了不起的工作。我们如何让更多的女性和有色人种进入太空？

LM: 首先，你要做好你现在正在做的事情。在这个摄像机里，你展示的是一个穿着飞行服去过太空的兄弟。所以也许有一个长得像我的小孩会说，“嘿，他做到了——我也能做到。”这是一个媒体的事情，得到更多的信息。

当我第一次去美国国家航空航天局的时候，有一个非洲裔美国妇女抓住我的胳膊说，“利兰，你要来为美国国家航空航天局工作。”

我说，“不，我不会，我会为杜邦或多德工作。”她说，“不，你要去国家航空航天局。”她有权当场雇用我，因为他们试图变得更加多样化，那是在 1989 年。

研究表明，最有创意和最优雅的解决方案来自最多样化的团队，不同种族，不同性别，我的意思是，你能想到的多样化的每一种方式，当你有团队在一起时，你就会得到最好的解决方案。我认为让人们感到受欢迎——不管他们带着什么来参加会议——首先要有一个欣赏你的才华和价值的老板。

**RP:所以要枢，书！你刚刚写了一个，叫** [***追空***](https://www.harpercollins.com/9780062496720/chasing-space) **。当你反思时，有哪些事情让你印象深刻？**

我认为最重要的事情之一是我的父母。当我从国家航空航天局退休后，我搬回家和父亲在一起。我在国家航空航天局工作了 24 年...我和我父亲谈了一次，第二天他就去世了。当整件事情发生时，我作为一名宇航员，一名公务员的身份被侵蚀了。然后我开始从邻居那里了解所有这些故事——他们告诉我“你爸爸让我免受牢狱之灾”，或者“我从戒毒所出来时，你爸爸给我买了辆车。”所以如果我当时不在那里，我就不会听到我写在书里的这些故事。它是关于人们帮助人们做得更好。现在我觉得我的工作是走出去，鼓励孩子们走出去，做他们想做的任何事情，我从父亲那里学到了很多。

**RP:说到孩子和教育，如果这项新的财政预算获得批准，NASA 内部的** [**教育部门将遭受重大打击**](http://gizmodo.com/destroying-nasas-office-of-education-fails-future-scien-1793345393) **。不管政治气候如何，我们怎样才能继续激励孩子们？**

我认为你必须让孩子们参与到做事情中来，不仅仅是为了做这件事，而是为了拯救我们的星球。例如，有人如何能帮助你或你的猫没有毛球？不管是什么，他们应该有帮助他人的观点。作为一个从太空回到地球的人，有了这个轨道视角，我看到了我们是如何联系在一起的，我想把这传递给孩子们。

**RP:很明显，*你的新书封面上那张*你的狗的照片，*追逐太空，*已经无处不在....**

**LM:** 给你什么感觉？

RP:它让我感觉像阳光一样？就像彩虹和独角兽在我脑中爆炸。但是你偷了我的问题——你看到那张照片有什么感觉？

有趣的是，自从那张照片出现在这本书的封面上后，我每天都能看到它。我一开始没有意识到的是...这是我们的手:两只爪子和我的手都连在一起。就像我们在击掌，感受所有的爱。这就是我感觉非常好的地方——我们所有人的手。

RFM:只是三个好孩子。

三个好孩子在踢球。

RFM:我们如何把这些好孩子送上太空？我想要一只太空狗。

嗯，在商业领域——埃隆·马斯克和 SpaceX——你也许能说服他让你和你的几个手下一起上去。

RP:所以你是在告诉我火星猫群不是不可能的？

**LM:** 猫？我爱所有的动物。不要憎恨猫科动物。但是没有什么比得上一只狗。

RFM:我的狗会在太空呕吐。

就像我一样？

RFM:我只知道那是他会做的第一件事。我也会在太空中呕吐。

**LM:** 我在太空中并没有感到恶心，所以感觉怪怪的。太突然了。我吃了一个变质的火腿三明治，我想这就是我的死因。

RP: **所以现在我们要问你一些愚蠢的问题。你最喜欢的皮特保罗歌曲是什么？**

我现在没有最喜欢的，但我会去探索它*(编者注:在这次采访过程中，不止有一次皮特保罗舞曲。)*

RFM:你最喜欢哪种奶酪？

**LM:** 豪达。

RFM:冥王星应该是一颗行星吗？

我想是的，因为在我们的太阳系里还有另一颗行星...可以用这些规则来定义的东西[ [来自 IAU](https://www.iau.org/news/pressreleases/detail/iau0603) ]，还有 [我们会在我们的太阳系中找到更多的行星](http://gizmodo.com/the-word-planet-is-so-important-and-such-bullshit-1793480282#_ga=2.235853313.1070992436.1495804061-1520736541.1475842057) ，我们必须弄清楚如何处理它们。所以冥王星应该是一颗行星。

RFM: **无论是运动员还是书呆子，仍然有很多污名，很明显，你已经完全摧毁了这一点。你怎样才能成为一个热爱太空和运动的人呢？**

从历史上看，总是书呆子帮运动员做作业。他们不想承认，但是运动员需要书呆子。有很多人认为他们只能成为运动员、音乐家或数学家，但仔细想想，数学和音乐是唯一真正同时使用左右脑的领域。我们需要打破你只能做一件事的障碍。我们必须向孩子们灌输成长思维，因为数学和创造力是如此紧密地交织在一起。

RP:做一个书呆子很酷吗？

书呆子是炸弹。我的意思是，你有知识，你了解世界上的事情是如何运作的，你可以用你所拥有的知识去影响他人。如果我们是一个集体，一起努力以更好的方式影响世界，我们就能一起变酷。
