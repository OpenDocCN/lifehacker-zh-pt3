# 告诉我们你所有需要解决的夏季问题

> 原文：<https://lifehacker.com/tell-us-all-your-summer-problems-that-need-solving-1827207739>

上周至日，热浪将于下周袭击美国大部分地区，夏天正式来临。随之而来的是一大堆令人发痒、出汗、甚至可能发臭的问题。



我们想知道*你的:*你还在寻找一种方法来阻止你的头发被晒伤吗？你想治好被虫子咬过的腿，这样你就不会在这个夏天剩下的时间里看起来残缺不全了？似乎总是摧毁你试图烧烤的一切？需要想办法让 [野生动物远离你的泳池](https://www.youtube.com/watch?v=ZcSo122Vunk) ，让 [沙子远离你的沙滩小吃](https://skillet.lifehacker.com/for-beach-picnics-bring-foods-you-can-eat-with-toothpi-1797344881) ？不知道该怎么做才能让你的孩子在接下来的两个月里不发疯？

请在评论中告诉我们，我们会尽最大努力找到答案——甚至是黑客！—针对你最讨厌的季节性压力来源。
