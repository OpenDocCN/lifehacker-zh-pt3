# 使用 2017 年全国拼字比赛冠军的这些拼写技巧

> 原文：<https://lifehacker.com/use-these-spelling-tricks-from-the-2017-national-spelli-1826295046>

“关于拼写，重要的是要知道，这不仅仅是死记硬背，”2017 年斯克里普斯全国拼写大赛的冠军 Ananya Vinay 说，他将在下周举行今年的大赛。虽然 Vinay 使用抽认卡来学习特定的单词，但她说真正的诀窍是了解不同单词的来源。

英语拼写大多模糊不清，因为它借用了许多其他语言的词汇。每种语言都有更可靠的拼写规则。所以要拼一个单词，你要知道它的来源。(在蜜蜂中，参赛者可以 [要求](http://spellingbee.com/sites/default/files/inline-files/Contest_Rules_of_the_2018_Scripps_National_Spelling_Bee.pdf) 一个词的起源，以及它的定义、词性和在句子中的用法。)

大多数时候起源于拉丁语或希腊语。Vinay 说，这两种语言占英语的 60%。当你知道一个单词是拉丁化的，你就知道要注意单词开头附近的双重辅音。最接近拉丁语的现代语言意大利语也使用双字母；Vinay 喜欢这个词*卡布奇诺*。

法语中有令人惊讶的无声字母，比如*中的 *t* 蜗牛*，或者 *assoilzie* 中的 *z* ，这个词的意思是“免除或宣告无罪”，从法语到苏格兰再到英语，听起来仍然像一个法语单词:“ah-soil-ee。”

德语有自己的怪癖:德语单词中的一个 *sh* 音可能会被拼成 *sch* ，德语是英语令人恼火的“I 先于 E”规则的背后:在德语中， *ie* 和 *ei* 总是意味着两个不同的音。一个有用的记忆术(我们没有从 13 岁的 Vinay 那里得到)是 [*wein*](https://www.fluentu.com/blog/german/german-spelling/) 与 *wine* 押韵，而 *bier* 就像 *beer* 。

源自亚洲的英语单词和西班牙语单词一样，幸运的是都是语音的。一些英语化的元音可能会让你出错，但是你不太可能会碰到不发音的字母。

Vinay 使用学习应用 [Quizlet](https://quizlet.com/) (其代表让我们与 Vinay 取得联系)来检查她的抽认卡。她在网站上发布了两套学习集 [给初学者](https://quizlet.com/288108438/beginnerintermediate-spelling-list-flash-cards/) 和 [给高级拼写者](https://quizlet.com/292044807/advanced-spelling-list-flash-cards/) 。(之前的斯克里普斯冠军不能再参赛，所以现在 Vinay 执教她学校的球队。)她喜欢把自己的单词按照具体的语言来源分类。该应用程序跟踪她的错误，这样她就可以回到有问题的单词。“如果我想的话，我可以在一小时内完成 600 个单词，”她告诉我们。

当你的电脑可以用红色虚线帮你拼写时，为什么还要学习拼写呢？因为学习单词和语言意味着学习历史和神话:Vinay 兴奋地向 Lifehacker 讲述了像“自恋者”这样的拉丁词，这些词是以一位仙女和河神的悲剧儿子的名字命名的，他爱上了自己的倒影并淹死了。

Vinay 说拼写在其他研究和学术竞赛中帮助了她，比如辩论和历史课。在科学碗中，她可以通过识别词根来识别矿物、化学物质或物种的名称。她能看到概念之间的联系，以及这些概念是如何传播和进化的。拼写不是琐事——它是人类历史的记录。
