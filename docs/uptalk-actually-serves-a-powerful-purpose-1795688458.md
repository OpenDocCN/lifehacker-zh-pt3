# 升调实际上有一个强大的目的

> 原文：<https://lifehacker.com/uptalk-actually-serves-a-powerful-purpose-1795688458>

升调过去似乎是一件坏事。“山谷女孩”的演讲模式，即演讲者的陈述以向上的音调结束，使它们听起来像问题，在 20 世纪 70 年代由语言学家 首次发现，并长期被嘲笑为不严肃的标志。1993 年， [*《纽约时报》*作家詹姆斯·戈尔曼](http://www.nytimes.com/1993/08/15/magazine/on-language-like-uptalk.html) 承认它可能有一些用途，但他担心它会被像飞机驾驶员这样的权威人士使用。



这是一个糟糕的比较；典型的飞行员对讲机独白听起来像一个连续的句子，在每个短语的结尾都轻快活泼。升调越来越普遍，也越来越辩护，在 [石板](http://www.slate.com/blogs/lexicon_valley/2014/12/16/uptalk_is_okay_young_women_shouldn_t_have_to_talk_like_men_to_be_taken_seriously.html)[*观察者*](http://observer.com/2015/03/girl-talk-when-it-comes-to-language-women-are-in-charge/)[BBC](http://www.bbc.com/news/magazine-28708526)[*时代*](https://well.blogs.nytimes.com/2013/12/23/overturning-the-myth-of-valley-girl-speak/)。一位加拿大人力资源顾问建议，歧视像升调 [这样的语调可能构成歧视](http://www.canadianbusiness.com/innovation/get-like-over-it/) 。升调是如此普遍，以至于在研究关于它的老研究时，我几乎认不出这些例子有什么不寻常。

也许是因为男人也大声说话——《纽约时报》甚至在上面喊出了乔治·w·布什的名字——这削弱了性别歧视的文化污名。也许是因为千禧一代，这一代(有时是由向上的谈话者抚养长大的)已经到了有权掌权的年龄。也许是因为在这个电子干扰的时代，提高语速对于保持某人的注意力至关重要。

记者 S.I. Rosenbaum [最近在 Twitter 帖子](https://twitter.com/sirosenbaum/status/869603618894356480) 中指出，升调有特定的会话目的。

 [https://lifehacker.com/embed/inset/iframe?id=twitter-869603618894356480&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-869603618894356480&autosize=1) 

她将这种用法与美国手语中的类似技巧联系起来，美国手语中的 [使用同样的扬眉表情](http://www.lifeprint.com/asl101/topics/rhetorical-questions.htm) 来表示是/否问题，并让对话伙伴注意。《T4 大西洋月刊》提供了一个口译员莉迪亚·卡莉斯的例子，她生动地诠释了纽约市市长迈克尔·彭博的飓风桑迪演说。

语言学家 Arika Okrent 在《大西洋月刊》中解释道:

> 在这个例子中，彭博警告人们最糟糕的风暴即将来临。最糟糕的事情即将发生。她眉毛一扬表示最坏的情况即将发生，然后又一扬表示可能发生。这种眉毛上扬表示话题化，是许多语言使用的一种常见结构。在话题化中，一个句子的成分被放在前面，然后被评论。对她的判决的粗略估计是“你知道最糟糕的是什么吗？很快？会发生的。”

根据辛西娅·麦克勒莫尔 和 [阿曼达·里查特和阿马莉亚·阿凡尼蒂](http://idiom.ucsd.edu/~aritchar/RitchartArvaniti_SpeechProsody2014.pdf) 的研究，升调也有同样的作用:通过把一个声明变成一个问题，它邀请听者积极倾听，点头或确认，就像加上“你知道吗？”或者“对吗？”敬一句话。它还有一个更基本的功能，即“坚守底线”，通过暗示还有更多的事情要发生来防止被打断；它把句号变成分号。

罗森鲍姆将这种联系从语言学延伸到社会学:就像说美国手语的人一样，年轻女性经常被忽视或被谈论。所以他们发展出类似的防御是有道理的。随着面对面的对话越来越受到数字干扰的威胁，我们其他人采用它是有道理的。

虽然有很多关于 [避免升调](https://www.wework.com/creator/start-your-business/ux-user-experience/say-it-like-you-mean-it-breaking-the-uptalk-habit/) (这会给某些观众带来不确定性)的建议，但培养它可能同样有用。语言学家马克·利伯曼告诉《T4 时报》说，在对话中，更有权威的人经常使用升调，比如医生对病人，老板对雇员，他们用升调来表示自信和友好。随着升调失去了它的名声，它在任何你需要吸引听众注意力的情况下都会更有用:当他们无法将视线从手机上移开时，或者当他们似乎根本不重视你的观点时。通过让他们回应和参与，uptalk 可以显示出你值得倾听。
