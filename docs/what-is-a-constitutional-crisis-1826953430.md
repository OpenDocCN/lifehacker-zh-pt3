# 什么是宪法危机？

> 原文：<https://lifehacker.com/what-is-a-constitutional-crisis-1826953430>

在过去的两年里，“宪法危机”这个词已经出现在很多人的脑海里，但是它意味着什么呢？这会导致什么？以下是你需要知道的。

实际上，“宪法危机”这个词有点模糊，没有确切、固定的定义，就像“倦怠”或“神经崩溃”一样，尽管大多数政治学家和律师都知道这个词的意思。一个 [宪法危机](https://en.wikipedia.org/wiki/Constitutional_crisis) 是指一个国家的宪法无法明确解决手头的问题，无论是冲突还是法律问题。宪法危机的原因 [分为几大类](https://fivethirtyeight.com/features/constitutional-crisis/) :

*   宪法对某个特定的问题没有任何规定。

*   宪法的措辞不明确，可以有多种解释。

*   宪法规定了该做什么，但这是不可行的或者不再相关。

这里有一个简单的例子:假设你和一些朋友在玩一个棋盘游戏，其中发生了一些规则手册没有涉及的奇怪事情，或者使用模糊的术语来解释。这就是宪法危机中发生的事情。桌上的每个人都有自己的观点，认为游戏应该如何进行，这往往会导致激烈的辩论。但这也可能导致玩家通过行使自己对规则的解释来利用他人——至少在所有玩家同意用自己的内部规则修改规则手册以弥补规则差距并使其对所有人都公平之前是如此。

你可能知道，当一方拒绝妥协并寻求某种形式的解决方案时，这种危机的最大问题就会出现。他们要么继续利用规则(因为“这是他们的游戏”)，他们退出游戏并破坏游戏，或者最糟糕的是，他们在愤怒中翻动棋盘和/或桌子。宪法危机也是如此，只不过当适用于整个国家时，事情显然要重要得多，规模也大得多。美国内战是这方面的一个主要例子。美国南部的几个州想要脱离联邦，美国宪法没有明确规定一个州是否可以合法脱离联邦。但是联邦政府无论如何都告诉他们“不”，所以这些州可以说翻转了桌子。

然而，内战并不是宪法危机的唯一终点。其他可能的路径导致行政瘫痪，政府陷入功能失调的僵局，直到最终崩溃；或者政治合法性的逐渐丧失，以及来自该国公民的信任和信念，有人称之为“”的宪政腐败此外，值得注意的是，宪法危机不同于叛乱、革命或政变，在叛乱、革命或政变中，政府以外的政治派别直接挑战政府的主权——通常以相当突然的方式。宪法危机是渐进的，甚至可能不为公众所察觉。一些人认为美国宪法现在处于危机之中，而且已经有一段时间了，列举了一些事件，如 2016 年总统大选、穆斯林旅行禁令、詹姆斯·科米被解雇、儿童被迫与父母分离的移民营地，以及传言中的特别顾问罗伯特·穆勒可能被解雇。
