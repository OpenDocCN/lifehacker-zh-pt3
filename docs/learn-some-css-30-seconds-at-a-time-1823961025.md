# 学习一些 CSS，每次 30 秒

> 原文：<https://lifehacker.com/learn-some-css-30-seconds-at-a-time-1823961025>

如果你正在学习 CSS，或者你想友好地了解它的一些术语和概念，尝试一下 30 秒的 CSS[。这个网站上的每个条目都显示了不同的代码，演示了结果，并解释了代码的每个部分是如何工作的。这里有一些条目，即使你不是设计师或开发者，你也能理解:](https://atomiks.github.io/30-seconds-of-css/) 



*   [弹跳载入器](https://atomiks.github.io/30-seconds-of-css/#bouncing-loader) 不使用任何图像文件创建动画。
*   [禁用选择](https://atomiks.github.io/30-seconds-of-css/#disable-selection) 使用户无法选择文本。(恶！)
*   [系统字体堆栈](https://atomiks.github.io/30-seconds-of-css/#system-font-stack) 让开发者在其他字体不可用的情况下选择要显示的字体。

*   当用户悬停在某个东西上时，其他文本变灰，比如菜单栏。

该网站解释了 33 个 CSS 片段，并包含一些有经验的设计师 [从未见过](https://www.designernews.co/stories/92184-30-seconds-of-css-) 的片段。对于超过 1500 个片段(没有详细的解释)，去 LittleSnippets.net。
