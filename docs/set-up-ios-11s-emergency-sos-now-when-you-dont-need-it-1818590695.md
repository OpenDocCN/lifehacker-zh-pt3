# 不需要的时候现在就设置 iOS 11 的紧急 SOS

> 原文:[https://life hacker . com/set-up-IOs-11s-emergency-SOS-now-when-you-dont-need-it-1818590695](https://lifehacker.com/set-up-ios-11s-emergency-sos-now-when-you-dont-need-it-1818590695)

iOS 11 中的紧急求救是你可能永远不会使用的功能之一，但现在设置它是一个好主意，以防你需要它。

Watch

**在两种情况下，你会想使用紧急呼救:**当你遇到真正威胁生命的紧急情况，需要尽快呼救时，以及当你遇到某种情况(例如，抢劫或遭遇警察)并想禁用手机上的 TouchID 时。

允许你禁用 TouchID 背后的想法是，执法部门或你的攻击者将无法强迫你解锁你的手机。可以说，如果你被抢劫了，小偷拿走了你的手机，你可能会认为不值得保留密码，但在 TSA 代理或商场警察等人员试图访问你的手机的情况下，禁用 TouchID 将迫使他们通过正确的法律渠道，而不是允许他们违背你的意愿用你的手指解锁你的手机。

连续五次按下设备上的电源按钮即可激活紧急求救信号。

### 建立

根据您希望如何使用该特性，设置会有所不同。你可以在设置中找到紧急求救菜单。

如果你对紧急呼救感兴趣的是快速呼救的能力，从紧急呼救菜单中打开自动呼救。这将使你的手机自动拨打 911(或任何正确的紧急号码是你在哪里)，当你激活紧急呼救信号。

在紧急 SOS 菜单中，您还可以设置紧急联系人。无论如何你都要填写这些信息——如果你发生了事故，医院可以使用这些信息。

当你激活紧急求救信号时，你的手机还会给这些人发送一条信息，让他们知道你已经联系了紧急服务机构，以及你目前的位置。在我成长的过程中，我妈妈经常生病，去了无数次医院。能够知道她打电话寻求帮助，以及从哪里打电话，这将是非常惊人的，因为她有能力通过点击电话的一侧来打电话，而不是解锁它，启动拨号器，并键入号码。

当你晚上一个人走回家，意外地发现自己处于危险境地时，你可以打电话求助，让人们知道你在哪里，而不用从口袋里拿出手机，这也很好。

如果你笨手笨脚，认为你可能会意外启动紧急呼救，那么你也可以选择在紧急呼救启动时启动倒计时声音。在呼叫急救服务之前，你有三秒钟的时间取消求救信号。它还会发出*非常*响亮的警报声。如果你被抢劫或袭击，仅仅这一点就足以让袭击者离开。也就是说，我打算关闭倒计时。如果我真的需要使用它，很可能是在我想谨慎行事的情况下。

**为了安全**:不幸的是，我们生活在这样一个时代，你可能会发现自己处于这样一种境地，某个处于权威地位的人希望你交出你的 iPhone，让他们检查上面的信息。

当你激活紧急呼救时，触控 ID 会自动关闭，你必须输入密码才能解锁手机。

即使您在 SOS 呼叫执法部门之前取消了 SOS，TouchID 仍将被禁用。更好的是，没有办法知道 TouchID 被故意禁用，所以无论是谁试图访问你的手机都不会知道你很快限制了访问。

当然，这种情况发生的几率很低，但你被车撞的几率也很低。设置紧急求救信号只需几分钟时间——安全胜于遗憾。
