# 仍然没有怀孕期间饮酒的“安全”量

> 原文：<https://lifehacker.com/there-is-still-no-safe-amount-of-alcohol-you-can-drin-1806113968>

本周关于怀孕期间饮酒的研究得到了两种说法:根据一些新闻报道，少量饮酒是完全安全的，但在另一些新闻报道中，这是绝对不允许的。抱歉，但这个很扫兴。



**头条** : [提醒:孕期喝(一点)没关系](http://www.refinery29.com/2017/09/171870/drinking-alcohol-while-pregnant) (Refinery29)

**故事**:是*而不是*怀孕期间突然可以喝一点，即使 Refinery29 这样告诉你，说出来真让我心痛。酒精很好喝。我在上次怀孕的最后一个月研究了当季最好的南瓜啤酒和 [酿造我自己的泡制酒](http://skillet.lifehacker.com/how-to-make-flavored-spirits-that-are-way-tastier-than-1723723712) 等宝宝一出来就喝。可悲的是，这项新研究并没有给怀孕的人提前打开那些啤酒的许可。

是这样的。甚至在这项研究之前，我们就知道酒精会给发育中的婴儿带来问题。影响 [包括](http://americanpregnancy.org/pregnancy-complications/fetal-alcohol-syndrome/) 精神和行为残疾、主要器官系统畸形。

大量的酒精会导致严重的问题，当你看到越来越少的酒精量时，你会看到越来越不严重的影响。有道理。所以我们自然希望有一个下限，你不会造成任何伤害。一天一杯太多了，但也许一周一杯？整个孕期一次？

问题是我们没有足够的数据来判断两种情况中哪一种适用:

*   也许少量饮酒对婴儿无害。
*   也许少量饮酒对他们有一点伤害，但是我们还没有做足够敏感的研究来发现这些影响。

毕竟，当你看着你的孩子在你喝了几杯酒后“变得很好”时，你无法知道如果你弃权了，孩子会不会变得更聪明或更健康。即使在大型研究中，我们也没有确凿的证据。而且很少有研究跟踪接触酒精的孩子直到学龄，所以即使我们有证据也不能说明全部情况。

那么这个 [新研究](http://press.psprings.co.uk/Open/september/bmjopen015410.pdf) 在哪里呢？本质上，它证实了零饮酒量和绝对有害健康的饮酒量之间的差距非常小，如果真的存在的话。作者回顾了 24 项研究，发现如果母亲每周饮酒超过 32 克，婴儿更有可能早产或小于胎龄儿。那比两杯酒多一点点。*每周。*

作者得出结论，孕妇每周饮酒不应超过 32 克，我们不知道饮酒少于 32 克会有什么危害。

**要点**:这项研究*没有*扩大“安全”饮酒的水平。我们现在知道，每周两杯酒的水平可能是有害的，低于这个水平仍然是一场赌博。零仍然是你最好的选择。(再次抱歉。)
