# 是的，对于季节性过敏来说，今年是特别可怕的一年

> 原文：<https://lifehacker.com/yes-this-is-a-particularly-horrendous-year-for-seasona-1795018482>

这个过敏季节是可怕的，不是吗？在过去的几周里，我愚蠢地打了个喷嚏，至少有趣的是，在纽约，我认识的一半人都挖出了自己的眼睛，用网兜淹死了自己。但是*今年是不是*比往年糟糕，还是我只是个大宝宝？我采访了纽约大学过敏症和免疫学助理教授苏扬·帕特尔(Sujan Patel)，听听他对这个季节是否像看上去那么糟糕的看法。并找出解决的办法。

### 证实:太可怕了。

“他们今年很糟糕，”帕特尔说。“过敏季节越来越糟糕的原因之一是全球变暖，以及包括二氧化碳在内的温室气体的积累，这使得植物更加肥沃。因为它们更容易繁殖，所以在这个季节的早期它们会释放更多的花粉。温和的冬天，加上降水...然后我们在整个春季经历了异常温暖的温度爆发，这让植物生长起来。”他停顿了一下。"所有这些都是让病人痛苦的原因之一."

那么，我们能做些什么让自己感觉更好呢？帕特尔为过敏症患者列出了他的典型防线。

### 1.找出你对什么过敏。

“首先要知道你对什么过敏，这很有用。尘螨或蟑螂会加剧春季过敏，”他说。如果你很痛苦，去找过敏症专科医生谈谈。

### 2.关上你的窗户。

我知道，现在是春天，天气很好，你想要微风。但是要关上窗户，尤其是在晚上。“实际上是在黎明时分，树木开始掉落花粉，花粉进入了你的家。所以我建议使用空调或风扇，因为它们有过滤装置，可以防止花粉进入，”帕特尔说。

### 3.睡觉前洗个澡。

“洗洗你的头发，”帕特尔说。“这样你就不会把所有的花粉都拖进你的枕头里，然后整晚都在里面摩擦你的脸。”

### 4.两个月前开始服用任何药物。

如果你在症状出现之前就开始使用过敏药物，并且在整个季节坚持使用，那么所有的过敏药物都会发挥最佳效果。在三月或四月左右开始关注在线花粉跟踪器，并在你严重发炎之前开始任何药物疗法。帕特尔说:“当树木盛开时，你不会受到太大的打击。”。

### 5.但是什么药物？

“第一道防线是新型抗组胺药，”帕特尔说。(他不再推荐像苯那君这样的老的、使人昏昏欲睡的药物。)在新的 OTC 抗组胺药中，你基本上有四种选择:西替利嗪(Zyrtec)、左西替利嗪(Xyzal)、非索非那定(Allegra)和氯雷他定(Claritin)。帕特尔说，如果你有鼻塞，你会想要一种旁边有字母 *D* 的药物，而且“一些患者需要使用片剂和鼻腔类固醇，如 Flonase、Nasacort 或 Rhinocort。”

### 6.不要抓眼睛。

用冷敷来代替。抓挠会释放更多的组胺，并将花粉驱赶到组织中。

### 7.每年更换两次空调过滤器。

“空气净化器之类的东西没有任何作用，所以我不推荐给患者。重要的是更换炉子或空调上的空气过滤器。”

### 8.护目镜。

我的晨跑现在更像是一次晨跑，因为我抓着我流下来的眼睛，我问帕特尔关于戴大眼镜或护目镜以防止花粉进入的问题。他说，他有一些患者戴上了眼罩，运气不错。“这看起来有点滑稽，但戴上护目镜可以防止花粉进入你的眼睛。过敏眼药水(如酮替芬)[也会有帮助]。”

### 9.镜头。

注射一个疗程是治疗季节性过敏的唯一方法。过敏症状永远消失需要三到五年的定期注射，但患者在两到六个月后开始感觉症状缓解。

那么，这意味着什么，继续前进？一百年后，人类会在一个喷嚏中灭绝吗？帕特尔医生认为情况正在好转吗？他回答说:“老实说，我不认为情况会好转。我认为情况会变得更糟，因为许多环境控制法规正在被取消。”

污染使得季节性过敏更加严重:“被污染的城市实际上有‘超级过敏体质’。”像柴油机废气颗粒这样的东西实际上会使过敏原更强。“所以同样数量的花粉在纽约比在奥马哈让人感觉更糟糕。幸运的内布拉斯加人，悲伤的纽约人。当地球温度升高 20 度，变成一个有茂盛植物和弱小打喷嚏的人类的星球时，我希望他们能为我们这些城市居民腾出空间。
