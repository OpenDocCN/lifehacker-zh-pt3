# 雄辩的主屏幕

> 原文：<https://lifehacker.com/the-eloquent-home-screen-1740855817>

大多数主屏幕设计都用信息和颜色来吸引你。这种主屏幕设计选择了简洁、视觉上有趣的极简设计。

Watch

这个主屏幕是为主题设计的。要在您的设备上获得这种外观，请下载。压缩下面的源代码链接，并按照以下步骤操作:

1.  移动。zip to SD card 0 > my color screen > Themer > Exported > zip(您的初始位置可能会有所不同。)
2.  打开主题并浏览主题。
3.  在“类别”下，选择“导出”
4.  选择“雄辩. zip”
