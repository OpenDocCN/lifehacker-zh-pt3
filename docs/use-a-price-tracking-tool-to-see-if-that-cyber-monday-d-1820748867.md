# 使用价格跟踪工具，看看网络星期一的交易是否值得购买

> 原文:[https://life hacker . com/use-a-price-tracking-tool-to-see-if-that-cyber-Monday-d-1820748867](https://lifehacker.com/use-a-price-tracking-tool-to-see-if-that-cyber-monday-d-1820748867)

网络星期一可能是抢购节日礼物的好时机。这也可能是一个很好的时机，认为你得到了一笔交易，并意外地为一些东西支付了比你必须支付的更多的钱。

零售商在网络星期一提供大量不同的折扣。虽然其中一些可能是千载难逢的大折扣，但其他一些可能只是看起来很便宜，但实际上并不比其他地方的价格高。

那么，你怎么知道你是否得到了一笔交易呢？

有很多工具不仅能让你感受到你在别处能买到什么样的特定物品，还能感受到该物品的历史价格。

我们最喜欢的是 [骆驼骆驼骆驼](https://camelcamelcamel.com/) ，它专门跟踪亚马逊上的价格，并帮助你绘制趋势图，找到在该网站上购买商品的最佳时间。

如果你没有在亚马逊上购物(或者即使你有)，另一个很好的工具是。它会查看你正在购买的东西，并让你知道你是否能在其他地方买到更便宜的东西。如果可以的话，它会提供一个更便宜网站的链接。它还会显示商品的价格，即使零售商隐藏了价格，这可以省去你在网络星期一上把一堆东西放入购物车以确定它们实际价格的麻烦。

如果你不喜欢使用某个工具，在你决定购买大件商品之前，一个不错的老式谷歌搜索也是一个不错的尝试，它通常可以在其他地方找到更好的交易。
