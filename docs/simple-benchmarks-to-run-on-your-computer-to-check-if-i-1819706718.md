# 在您的计算机上运行简单的基准测试来检查它是否运行良好

> 原文：<https://gizmodo.com/simple-benchmarks-to-run-on-your-computer-to-check-if-i-1819706718>

乍一看，您桌上的电脑可能与去年的型号或您同事拥有的型号很相似，但外观可能具有欺骗性-如果您真的想对您的笔记本电脑进行评级，并在功耗和性能方面找到自己的位置，这就是基准测试的用武之地。



长期以来，作为一种客观判断装备原始能力的方式，玩家们推崇基准测试，它是衡量你的装备性能与你朋友的装备性能的最佳方式。在测试您的计算机时，有两种类型的基准需要考虑:真实世界和合成世界。真实世界的基准测试更简单，比如计时 Photoshop 打开或 Blender 渲染文件的速度。合成基准是运行精心定制的一系列测试的软件包，有些旨在复制“真实世界”的情况(如您实际玩的游戏)，有些旨在尽可能地推动计算机组件。

一旦基准测试完成，您将获得一系列数字，这些数字本身并没有多大意义，但是将它们与其他系统进行比较，或者随着时间的推移对它们进行跟踪，基准测试对游戏玩家和非游戏玩家都变得更加有用。

### **为什么要基准？**

运行基准测试的主要原因一直是评估计算机的性能，但这不仅仅是为了让游戏玩家尽可能提高每秒帧数——您可以使用相同的测试来检查笔记本电脑的性能，或者在考虑升级时与新型号进行比较。

它们有助于检查您是否得到了您应该得到的东西，例如互联网速度测试，或者在问题发生之前发现问题，这正是硬盘基准可以派上用场的地方。

毫无疑问，在没有添加常规基准程序的情况下，您已经有很多分心的事情了，但是养成运行它们的习惯，您就可以在何时支付新笔记本电脑方面做出更明智的决定，在潜在的硬件问题变得严重之前发现它们，在软件和组件升级方面做出更好的选择，获得您正在运行比别人更好的系统的证据，等等。

有许多不同的变量会影响基准分数，我们建议至少运行三次这些测试，然后取一个平均值。大多数基准软件程序都有一个附带的网站，你可以在那里将你的分数与其他系统进行比较。

### **1)性能指标评测**

通用性能指标评测贯穿从 CPU 速度到 GPU 能力的方方面面，使用它们来衡量您的笔记本电脑有多先进(或过时)，以及您可以运行的顶级游戏类型。

[Novabench](https://novabench.com/) 是一款不错的免费 Windows 基准测试工具，你可以用它来完成工作，而 [Geekbench](http://www.geekbench.com/) 更专业、更全面，可以在 Windows、macOS、iOS 和 Android 上使用——但一旦你用完免费试用版，它会让你花费 9.99 美元。

如果你对图形性能特别感兴趣，那么 [Cinebench](https://www.maxon.net/en/products/cinebench/) 可以免费使用，并且可以跨 Windows 和 macOS 工作，而 [3DMark 和 PCMark](https://www.3dmark.com/) 是免费基础版的不错的 GPU 测试工具(更高级的软件包每个要花 29.99 美元)。

定期运行这些性能指标评测，您可以了解性能如何随着时间的推移而保持不变，检查系统硬件和软件设置的升级或更改的影响，并(通过网上发布的全球性能指标评测分数)计算出购买新笔记本电脑的最佳时间。

每个基准都是不同的，所以请查看每个基准的指导方针 Geekbench 的基准是 [这里是](http://support.primatelabs.com/kb/geekbench/interpreting-geekbench-4-scores) 。一台英特尔酷睿 i7-6600U Surface Book 被用作基线，单核得分为 4000，所以如果你在图表 中远远低于 [，也不要惊慌，只要你的机器仍然在做你需要它做的事情。另一方面，如果你的笔记本电脑价格低至几千(](https://browser.geekbench.com/v4/cpu) [中端智能手机水平](https://browser.geekbench.com/v4/cpu/4568363) )那么它可能值得考虑升级。

同样，对于游戏，在网上查看 3DMark 分数 ，看看你的系统如何达标，但如果你不是最好的，也不要绝望(Fire Strike 测试中的 10，000 及以上是衡量一台像样的游戏机的好标准)。Futuremark 网站让您确定特定组件得分的方式有助于计算特定升级可能为您带来的更多性能。

### **2)浏览器性能指标评测**

如今，我们都在浏览器上花了很多时间，但是你现在使用的浏览器真的是最好的吗？你的浏览器是不是吸干了内存和磁盘空间，拖慢了你的整个系统？是时候转换了吗？

当然，你可以简单地安装几个相互竞争的浏览器，自己试用，看看哪个感觉更快，权衡一下你最喜欢的网络应用在每个浏览器中的表现。为了更科学一点，使用一个基准。

有很多工具可以测试各种不同的 web 技术和浏览器体验的不同方面。我们在 Gizmodo 测试设备时使用 [WebXPRT 2015](http://www.principledtechnologies.com/benchmarkxprt/webxprt/2015/v19982/) (我们总是使用系统的原生浏览器进行测试)，因为它在网络浏览器中测试各种场景，包括制作电子表格、编辑照片和加密文件。然而，还有其他很好的选项可以让您全面了解浏览器的性能。

为此，我们还喜欢 [Basemark](https://web.basemark.com/) (针对较新的 web 技术，包括交互元素和 WebGL 图形)，较老的 [Peacekeeper](http://peacekeeper.futuremark.com/) (针对一般页面渲染性能)，以及 [JetStream](http://browserbench.org/JetStream/) (专门用于测试 JavaScript 性能)。你选择哪一个并不重要，只要你手头有一个选择，并坚持下去。

根据你得到的数据，以及在网上公开发布的数据，你应该能够计算出你选择的浏览器中哪一个性能最好。检查每个浏览器的不同版本的表现也很方便。如果一个浏览器低于标准， [有方法](http://fieldguide.gizmodo.com/4-easy-tricks-to-make-google-chrome-run-faster-1782564390)[加速它](http://fieldguide.gizmodo.com/4-easy-tricks-to-make-firefox-run-faster-1782835740) 。

对于浏览器来说，更多的是不同的程序在你自己的系统上如何比较，而不是它们如何与其他用户的分数相比较(尽管你仍然可以查看 [公开分数](https://powerboard.basemark.com/home) 作为参考)。解释结果也取决于浏览器中对你来说最重要的是什么。

Basemark、Peacekeeper 和其他检查浏览器性能各方面的工具的分数很有帮助，因为广泛的指导方针和专业基准测试(如基于 Javascript 的 Jetstream [)会给你更多信息](http://browserbench.org/JetStream/in-depth.html)——测试 web 应用程序启动的速度和运行的流畅程度。

### **3)互联网速度基准**

说到互联网的使用，你的下载和上传速度怎么样？测试这些速度是有用的，原因有很多:让您的互联网服务提供商对承诺负责，解决问题，并随着时间的推移检查性能。

这方面的首选是 [Speedtest](http://beta.speedtest.net/) ，但这不是唯一的选择——事实上，如果你确实在带宽和速度方面有第二或第三种意见，那是最好的。如果你只是搜索“网速测试”，谷歌会测试你的连接速度，而专门检查你与网飞连接的也值得一试。

Speedtest 将测试你的电脑连接到互联网时的响应速度(ping 测试)，以及你的下载和上传速度，然后告诉你你的分数与其他用户相比如何。在 ping 测试的情况下， [30 毫秒或更少的](http://testinternetspeed.org/blog/what-is-a-ping-test/) 对于宽带连接来说是很好的成绩。

如果您没有获得 ISP 向您承诺的下载和上传速度，您可能需要与该公司联系以了解原因。不过，在你接触之前，先进行 ISP 自己的基准测试(现在大部分都提供了)——这可能不是最公正的选择，但有统计数据在手还是不错的。

你还可以使用互联网速度测试基准来计算出你可以将笔记本电脑放置多远才能获得良好的连接，或者你的家庭办公室需要放置在哪里，或者六个 [亚马逊](https://gizmodo.com/after-almost-3-years-theres-a-new-echo-1818846014) 对你的速度有多大影响，等等。运行测试，对您的网络布局进行必要的更改，然后再次运行测试，看看有什么变化。

### **4)系统组件基准**

除了我们上面提到的图形测试之外，您还可以获得专注于笔记本电脑内部特定组件的基准测试。例如，在排除机器故障或判断升级时间时，这些信息会很有用。

听到硬盘发出一些不健康的声音？运行基准测试。 [CrystalDiskMark](https://crystalmark.info/software/CrystalDiskMark/index-e.html) 对于 Windows 来说是一个有用的免费选项，而[black magic Disk Speed Test](https://itunes.apple.com/us/app/blackmagic-disk-speed-test/id425264550?mt=12)将在 macOS 上做几乎相同的工作，并且也不会花费你任何成本(这两个选项都应该适用于固态硬盘以及旧的机械驱动器)。

如果你想给你的 CPU 施加一些压力，那么 [Prime95](https://www.mersenne.org/download/) 是基于 Windows 的机器的一个受欢迎的免费选项。特定于 CPU 的 macOS 选项很少，但是像 [Geekbench](http://www.geekbench.com/) 这样的通用基准测试工具有特定于 CPU 的部分。

所有这些工具对于诊断现有问题和防止将来可能突然出现的潜在问题都很有用，但是您需要定期运行这些基准测试，并随着时间的推移记录性能，以标记任何出错的地方。例如，如果你注意到你的笔记本电脑的硬盘性能开始下降，可能是时候 [备份你的东西](http://fieldguide.gizmodo.com/all-the-best-ways-to-back-up-your-data-1796291120) 并购买一个替换硬盘了。
