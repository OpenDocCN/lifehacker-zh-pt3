你可以使用许多网站来创建的一次性电子邮件地址，你甚至可以在 Gmail 等服务中尝试 ol' " [添加加号或句号](https://lifehacker.com/use-the-same-gmail-account-for-multiple-online-accounts-5341896) "的技巧来创建一个“假”地址，它总是会转发到你的真实地址。第一种方法的问题是，这种电子邮件地址不会永远存在——例如，如果你想要一个品牌的时事通讯，但不想在注册时说出你的真实电子邮件地址，这就不是很有用。

第二种方法是有帮助的，但也仅此而已。如果第三方出售你修改过的电子邮件地址，你的收件箱中仍然会收到垃圾邮件，而且你仍然需要创建过滤器来管理像“david..murphy@lifehacker.com "而不是" david.murphy@lifehacker.com。"

这就是为什么我很高兴地偶然发现了 [Chrome](https://chrome.google.com/webstore/detail/burner-emails-easy-fast-d/dnnmaoecogobjdiieopljefemjdkkaja) 和 [Firefox](https://addons.mozilla.org/en-US/firefox/addon/burner-emails/) 扩展燃烧器电子邮件，这使得创建(和管理)变得异常容易！)转发到你真实地址的虚拟电子邮件账户。我几乎不需要解释这个扩展是如何工作的，因为它使用起来非常简单，但是我告诉你:安装这个扩展，用你的真实电子邮件地址注册一个账户。这是大部分的工作。

现在，如果你去一个要求你输入电子邮件地址来创建账户的网站，你应该会在文本输入栏看到一个带火焰的黑色小图标。点击它，电子邮件地址字段将自动填充一个全新的，唯一的地址。注册网站时使用它，网站发送给它的任何电子邮件都会自动转发到你的真实电子邮件地址。

要管理你的众多电子邮件地址——如果你愿意，这个扩展可以为每个服务创建一个新的电子邮件地址——只需在浏览器主窗口点击它的图标。如果您收到太多邮件，并且不信任发件人会将您从他们的列表中删除，您可以暂时关闭或完全删除电子邮件地址。你的选择。

虽然你不能重命名你的各种电子邮件地址以使它们更容易被找到，但是 Burner Emails 以你的名义创建的任何地址都应该包含该网站的名称。如果你想要更多定制的电子邮件地址，你可以随时点击一次性电子邮件 [网络工具](https://burnermail.io/) 来创建你自己的电子邮件地址。只是这样做比使用超级方便的浏览器扩展要慢一点。
