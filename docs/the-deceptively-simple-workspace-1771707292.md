# 看似简单的工作空间

> 原文：<https://lifehacker.com/the-deceptively-simple-workspace-1771707292>

Redditor [mrrej89](https://www.reddit.com/user/mrrej89) 将这个漂亮的工作区发布到了 Battlestations subreddit 上。乍一看，这很简单，几乎有点无聊，但随后你会更仔细地看细节——从灯到键盘到显示器支架到书架，很简单，但都做得很好，建造得很好。

Watch

在 Imgur gallery(链接如下)中有大量的整个空间的照片，但线索(也链接如下)是他解释他桌子上大多数设备来自哪里的地方。他添加到桌子上的皮革垫与木头非常相配(他说他这样做是因为他不需要垫子)，他还摇着一副耳机，这是我个人最喜欢的耳机[AKG K553 Pros](http://www.amazon.com/AKG-K553-Pro-K-553/dp/B00X3MT7F6?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-deceptively-simple-workspace-1771707292&asc_source=&tag=kinjalifehackerlink-20)。这是另一张带有搁板的工作区照片:

就连显示器支架，一个[【54 美元】银石铝一体式支架](http://www.amazon.com/SilverStone-Technology-Aluminum-Unibody-MR01S/dp/B00R45T66S?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-deceptively-simple-workspace-1771707292&asc_source=&psc=1&tag=kinjalifehackerlink-20&th=1) 都是为空间精心考虑的。鼠标显然是 Steelseries，但键盘是另一个我可以担保的产品，[Cooler Master nova touch TKL](http://www.amazon.com/Storm-NovaTouch-TKL-Exclusive-Capacitive/dp/B00MY3S288?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-deceptively-simple-workspace-1771707292&asc_source=&tag=kinjalifehackerlink-20)，我已经使用过并喜欢它(机械，但由于开关周围的定制静音，它很好，很安静。)他甚至还有 [自定义键帽](http://www.amazon.co.uk/dp/B013KP92AK) 。

他还分享了一张更大的照片，展示了这些书架的真正美丽:

此外，如果你喜欢他工作区的艺术，你可以在他的网站 上抓取一个打印并支持艺术家的 [。总的来说，这是一个非常犀利的造型。看似极简和简单，但每一件和每一个部分都经过了深思熟虑，所以它完美地结合在一起——非常适合 mrrej89。更多的照片，点击下面的链接，查看完整的图库。](https://www.hiddenmoves.com/collections/all/standard-a3-prints)

如果您有自己的工作空间可以炫耀，请将其添加到我们的 [**Lifehacker 工作空间展示并告诉 Flickr pool**](http://www.flickr.com/groups/lifehacker-workspace-showandtell/pool/) 来与我们分享。确保您包含的任何照片至少为 1280x720。将它们保持在 16:9 也有帮助！包括一些文字，关于你使用的材料，你是如何想出这个设计的，以及任何其他相关的细节。如果你巧妙的组织和良好的设计感吸引了我们的眼球，你可能就是下一个特色工作区！
