你可能会认为 Edge 1.86%的市场份额——根据我们撰写本文时 Statcounter 的数据——会高一点，因为微软将浏览器植入了 Windows 10，这是你在一个全新的、未受影响的操作系统版本上的唯一选择。但是你走了。没有人真的在使用 Edge，但这并没有阻止微软在 11 月正式推出适用于安卓和安卓的浏览器版本，并在本周推出对 iOS[和安卓平板](https://blogs.windows.com/windowsexperience/2017/11/30/microsoft-edge-now-available-for-ios-and-android/) 的支持。

### 为什么要给 Edge 一个尝试？

如果你已经迷上了 Chrome 或 Firefox，说服你转用另一款浏览器——也就是微软的浏览器——可能是徒劳的，但我们至少会尝试给出一个合理的理由来解释为什么 Edge 如此有趣。

当你启动 Edge 时，微软会要求你将应用程序与现有的微软帐户同步，这是一种标准的浏览器做法，允许你在个人电脑、智能手机和平板电脑之间共享你的收藏夹、历史记录、密码、浏览器数据和设置。这没什么新鲜的。

浏览器本身有一个漂亮的流线型设计，底部有导航按钮——对 Safari 和 Firefox 用户来说是一个受欢迎的景象，对 Chrome 粉丝来说是一个轻微的调整。

说起来，当我们在两种浏览器的 iOS 版本上运行浏览器的 JetStream 测试时，Edge 和 Chrome 一样快，实际上它在浏览器的速度表 测试中以大约 20%的增幅击败了 Chrome。对于日常任务的使用，比如添加或修改书签，拉起(或切换)标签，它的反应令人难以置信。

### Edge 的最大特点:向你的 Windows 10 电脑添加标签

微软的“在 PC 上继续”功能运行完美。假设你已经将 Edge 和你的其他台式机或笔记本电脑连接到你的微软帐户，你可以将你在智能手机或平板电脑上看到的任何内容发送到你的一台电脑上。该页面会立即在 Windows 10 的 Edge 浏览器中弹出，你坐在办公桌前继续阅读(或观看)。

这是一个单行道(至少目前如此)，所以你不能将你在 Windows 10 上的 Edge 中查看的标签发送到你的智能手机或平板电脑上。我们只能假设微软会在未来的更新中解决这个问题。

总的来说，Edge 给人的感觉是流线型和极简的。如果你想配置的话，你可以配置的选项并不多——远远少于你在 Chrome、Firefox 或 Safari 上找到的选项。如果说有什么不同的话，Edge 有点不成熟，但如果你已经购买了微软的生态系统，它会非常适合你。除此之外，它是一个快速、简单的浏览器...如果你能被说服做出改变。
