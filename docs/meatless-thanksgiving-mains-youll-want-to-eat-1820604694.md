# 你想吃的感恩节无肉主食

> 原文:[https://life hacker . com/meat less-thanksgiving-mains-youll-want-to-eat-1820604694](https://lifehacker.com/meatless-thanksgiving-mains-youll-want-to-eat-1820604694)

感恩节就是让人们聚在一起，但对于素食者和纯素食者来说，节日大餐可能会以肉类为中心，到了疏远的地步。没有人——即使是你最坏的表亲——应该在其他人想吃什么就吃什么的时候，去拨弄吃剩的原油、酸果沙司和*也许*一些填料。

许多厨师在需要一道令人叫绝的主菜时都会选择肉，因为除了火鸡，很容易就能做出美味的肉。不过，正如你将会看到的，素食主菜并不一定是一块上面有法国洋葱的普通豆腐，或者当有人说“素食感恩节”时你想到的任何东西无论你是一个终身素食主义者，还是一个今年要喂养一些非肉食者的杂食者，我希望你能在这个列表中找到一些启发你的东西。

### 一种以植物为基础的“火鸡”烤肉

为我最热的感恩节大餐做好准备:Tofurky 类的规则。它有着完美的人造肉质地——不是超级糊状，也不是超级有弹性——尝起来像咸馅。坦白地说，我每次都会选择它而不是干燥多筋的火鸡胸肉。

以小麦和大豆为基础的假肉在最近几年已经走了很长的路；不管你喜不喜欢，素食主义者从来没有像现在这样被宠坏了。(如果你不喜欢，也许花点时间想想为什么快乐的素食者会让你如此不安，兰德尔。)但是，正如我从这篇文章中读到的素食肉类评论中了解到的那样，所有的素食火鸡都是生来平等的，所以不要只是抓住你看到的第一样东西就收工。像和 [野外烧烤](http://fieldroast.com/product/celebration-roast/) 等常年供应强劲的品牌，往往是季节性选择的安全赌注； [素食加烤全火鸡](http://vegetarian-plus.com/product/all-natural-vegan-whole-turkey/) 不出所料，*非常*两极分化。如果你能找到它，圣杯品牌似乎是[garde in](https://gardein.com/products/holiday-roast/)——这也恰好是全食素食熟食店的选择品牌，根据某些素食火鸡阴谋论者的说法，它是 [乔氏非常受欢迎的自制品牌假烤火鸡](https://www.traderjoes.com/fearless-flyer/article/1951) 的制造商。但老实说，如果你要为感恩节的特定人群挑选一份纯素烤肉，提前给他们发短信，问他们是否有任何品牌偏好。

### 一个大的干酪砂锅

奶酪砂锅是素食主菜的简单模式，但有时没有必要在正确和简单之间做出选择。通心粉和奶酪已经是一些人的感恩节主食，我认为你可以做得比一大盘浸泡奶酪的意大利面更糟糕。我最喜欢的食谱来自安吉拉“Kitchenista”戴维斯 ，她应该是你做任何与舒适食品相关的事情的第一站(如果她还没有的话)。

如果你觉得通心粉和奶酪已经过时了，我不太理解，但我仍然会帮助你。这不是传统的感恩节食物，但茄子干酪是我在这个愚蠢的漂浮垃圾岩石上最喜欢的食物，而来自 Food52 的这个“天才”食谱无疑是我做过的最好的版本。克莱尔上个月做的 [皮罗吉砂锅](https://lifehacker.com/you-deserve-a-hot-and-cheesy-pierogi-casserole-1819722125) 同样美味，配上蘑菇、额外的焦糖洋葱，甚至一些代替培根的泡菜；蔬菜穆萨卡 也是一个非常好的选择。

### 惠灵顿蔬菜

正如我在 [这篇文章的肉版本](http://skillet.lifehacker.com/break-free-from-turkey-tyranny-with-these-alternative-t-1820251130#_ga=2.155419640.2141885543.1511138820-949419976.1446553382) 中提到的，威灵顿就是戴着主菜帽的馅饼——如果有哪个节日需要这道菜，那就是感恩节。如果你想吃传统的威灵顿素食，这份梅丽莎·克拉克的食谱 会非常不错。但是传统可能非常无聊；为什么不把一大堆 [萨莫萨馅](https://www.bbcgoodfood.com/recipes/1127668/vegetable-samosas) 包在酥皮里当主菜呢？我会非常兴奋地和我的土豆泥和填料一起吃一两块萨莫萨-威灵顿三明治；蔓越莓酱会把香料衬托得很美，你不觉得吗？

### 斯潘纳科皮塔

我妈妈把我养大成素食主义者，她在特殊场合吃的菜是 spanakopita——确切地说，是莫丽·卡曾在莫斯伍德烹饪书中的版本。它很美味，大部分是健康的，可以在晚餐时间前很好地组装；换句话说，工作中的共同父母需要从美味佳肴中得到的一切。出于同样的原因，Spanakopita 也是一种很好的感恩节素食主菜，但不要对经典的菠菜和羊乳酪馅料感恩戴德，除非它是你的真爱。也许是焖韭菜和茴香、香草烤南瓜、蘑菇*炖菜*，甚至是绿豆砂锅菜 [烤？](https://lifehacker.com/grilled-green-bean-casserole-is-your-new-fall-side-dish-1819483711)——会为节日的叶饼做美味的馅料。如果你以前从未与 phyllo 合作过，它一点也不坏——只要你买的是冷冻的，而不是像某种初露头角的连环杀手一样自己制作。只要有耐心，使用大量的黄油(或“黄油；大多数叶子是纯素的！)每层之间。你能行的。

### 饼干和(素食)肉汤

我知道我们正在进入一个激进的领域，但我真的认为我在这个问题上有所进展。我的逻辑如下:

1.  饼干在各方面都比帕克屋卷饼好，因此，它应该是标准的感恩节面包；
2.  有两种或更多肉汁的感恩节菜单天生就比只有一种肉汁的要好；
3.  每个人，不管吃多少肉，都喜欢饼干和肉汁。

对于素食客人来说，饼干和肉汁是我迄今为止所建议的最简单的方法:为所有人制作饼干而不是面包卷，加上一些蘑菇或蔬菜香肠肉汁。很简单。 [纯素饼干和肉汁](https://minimalistbaker.com/vegan-biscuits-and-gravy/) 工作更多，但不多。无论你选择哪一种，我建议 [在你的饼干](https://www.chowhound.com/recipes/sweet-potato-biscuits-10886) 中加入一些甘薯泥，让它们更甜更有节日气氛。

### 几乎任何阿曼达·科恩或约坦·奥托林吉的食谱

有时候，做出一道完美菜肴的最好方法就是模仿专业人士。阿曼达·科恩(Amanda Cohen)是 Dirt Candy， [的主厨，这是一家位于纽约州](http://www.dirtcandynyc.com/amanda-cohen) 的只提供品尝菜单的“蔬菜餐厅”，她相信全蔬菜烹饪就像乏味的兄弟们相信从头到尾吃一样。 [奥托林吉先生也是一名厨师](https://www.ottolenghi.co.uk/) ，但我是从他那本华丽的、以素食为主的食谱中认识他的，那本书以我一千年都想不到的方式组合了各种食材。这篇文章的灵感来自《卫报》最近发布的奥托林吉冬季蔬菜食谱的帖子，但你也应该买一本 [《泥巴糖果食谱](https://www.amazon.com/Dirt-Candy-Flavor-Forward-Vegetarian-Restaurant/dp/0307952177?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/meatless-thanksgiving-mains-youll-want-to-eat-1820604694&asc_source=&tag=kinjalifehackerlink-20)——阿曼达·科恩的食谱绝对精彩。

我按照《卫报》文章中奥托林吉的花椰菜奶酪配方做了一些小改动——数量减少了一半，用羊乳酪代替了斯蒂尔顿奶酪，因为我就是这么做的——味道棒极了。不过，当我再做一次的时候，我会把整个花椰菜煮熟，或者把它切成大块，然后按照食谱来做。一整头褐色的、覆盖着奶酪的花椰菜构成了一个美丽的展示，但我的没有在建议的时间内完全煮熟，我发现自己渴望有更多的表面积让奶酪挂在上面。尽管如此，这仍然是一个令人惊叹的食谱，将会有奶酪爱好者为了刮盘子的权利而相互争斗——但即使它不太叫你的名字，我保证[否则](https://www.splendidtable.org/recipes/portobello-mousse) [他们已经](https://www.splendidtable.org/story/the-key-3-amanda-cohen-of-dirt-candy) [做出了](https://food52.com/recipes/36374-amanda-cohen-s-secret-weapon-stir-fry-sauce-for-vegetable-fried-rice-more) 的意愿。事实上，我可能会用我买的备用花椰菜头来做 [泥巴糖熏制花椰菜和](https://www.pannacooking.com/recipes/smoked-cauliflower-waffles-horseradish-cream-amanda-cohen/) 华夫饼，作为感恩节后的早午餐。
