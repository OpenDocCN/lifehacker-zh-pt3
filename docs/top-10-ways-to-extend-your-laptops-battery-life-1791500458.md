# 延长笔记本电脑电池寿命的 10 大方法

> 原文：<https://lifehacker.com/top-10-ways-to-extend-your-laptops-battery-life-1791500458>

当你试图把事情、做完，却找不到插头给电池充电时，没电可不是什么好事。如果你要在旅途中工作，这里有一些简单的技巧来延长你的笔记本电脑的电池寿命，无论是在紧要关头还是在你离开家之前。



### 10.调整你内置的省电选项

首先，简单的解决办法:熟悉你的笔记本电脑的电源管理功能。在 Windows 中，它是“电源选项”，你可以通过点击开始并输入这个选项来打开它。在 macOS 中，它在“节能器”下的“系统偏好设置”中。不要忽视这些，也不要认为默认设置是为了节省能源。如果你知道你将带着你的笔记本电脑外出，告诉你的笔记本电脑在离开前尽可能轻地耗电，这样你就不必在外出后缩减你正在做的事情。

更好的是，您可以通过热键 (至少在 Windows 中为)和 [定制您的笔记本电脑何时以及如何警告您电池电量](https://lifehacker.com/how-to-customize-your-computers-battery-warnings-and-a-5923359) ，让您永远不会措手不及。

### 9.使用笔记本电脑制造商的电池维护工具

根据笔记本电脑制造商的不同， [他们可能会提供电池维护工具](https://lifehacker.com/prolong-laptop-battery-life-with-your-systems-battery-c-5966576) ，这些工具可以告诉你很多关于电池健康状况的信息，以及他们延长电池最长寿命的建议。例如，索尼和联想都根据你使用的 Windows 版本调整了电池维护面板或自己的实用程序。

在这里，你会对一些热门话题有一个最终的结论，比如电池校准，你应该多久让你的电池完全放电一次(或者相反，你应该多久完全充电一次)。)从源头听取建议总是更好。

### 8.获取电池监控器，深入了解电量、循环次数和剩余寿命

如果你的制造商没有专门的电池维护工具——或者你在上一次 [全新安装或 PC de-](http://lifehacker.com/the-complete-guide-to-avoiding-and-removing-windows-c-1630577558)中清除了它，一切都还没完。有许多免费的实用程序可以告诉你任何你想知道的关于笔记本电池的信息，不管它是谁制造的(包括你，MacBook 的拥有者)。)



macOS 用户可以试用，Windows 用户可以试用 [BatteryCare](http://batterycare.net/en/download.php) ，这两个 [我们之前都在 times](http://lifehacker.com/batterycare-displays-detailed-battery-information-5334722) 提到过 [。您还可以查看用于 Windows 的](http://lifehacker.com/batterybar-adds-a-power-meter-to-the-taskbar-5145063) [BatteryBar](http://batterybarpro.com/) 、用于 macOS 的 [【超级监视器】](http://web.archive.org/web/20130814044014/http://superbarmonitor.de) [，或者用于 macOS](http://lifehacker.com/superbarmonitor-puts-system-monitoring-right-on-your-wi-5488717) 的[battery diag](https://itunes.apple.com/us/app/battery-diag/id836505650)[等等。最佳选项不仅会取代内置的电池统计数据，还会向您显示精确的信息，如电池在其寿命中有多少充电周期，它在充电周期中的精确点，并提供一些关于如何从中榨取更多电量的提示，以及一些有用的工具。](http://lifehacker.com/battery-diag-is-a-pretty-informative-battery-monitor-f-1567889220)

### 7.出门在外时，请调低亮度

如果你已经出门在外，看着在一块可能无法到达下一个插座的电池，你可以做的第一件事就是切断笔记本电脑最大的耗电设备:屏幕的电源。显然当你的笔记本电脑不使用时，它会处于睡眠状态，但当它在使用时，你可以通过 [保持屏幕亮度尽可能低来节省一些电量](https://lifehacker.com/how-to-maximize-the-battery-life-of-your-windows-laptop-5566020) (当然，仍可使用。)如果你有 Windows 机，你甚至可以考虑 [禁用自适应亮度](http://lifehacker.com/disable-windows-annoying-adaptive-brightness-feature-on-1567886994) 自己控制。

这在像这样的指南中出现了很多，虽然它肯定是有帮助的，但它不是万能的。让你的屏幕暗到你几乎看不见可能会延长几分钟的可用电池，但 [不会给你额外的一小时，你需要](https://lifehacker.com/how-much-brightness-actually-affects-your-battery-life-5994135) 在你回到办公室之前，或者弥补一夜忘记充电。就当是应急措施吧。

### 6.保持冷静

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-vTs_4eeCdRM&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-vTs_4eeCdRM&start=0) 

热量是电池杀手，它不仅会缩短电池的短期寿命，还会缩短电池的长期健康状况。我们已经在 [之前的几个场合](http://lifehacker.com/why-is-my-laptop-so-hot-and-what-can-i-do-about-it-5885376) 中 [提到过这一点，但它有助于让你的笔记本电脑尽可能保持凉爽，或者至少在你外出使用时避免限制气流。拿一个笔记本电脑桌，如果风扇在底部，保持背部抬高一点，避免两边或背部拥挤。](https://lifehacker.com/how-to-maximize-the-battery-life-of-your-windows-laptop-5566020)

我们已经向你展示了如何制作笔记本电脑冷却器，从 [到鸡蛋盒](https://lifehacker.com/keep-your-laptop-cool-with-an-egg-carton-1587234841) 到 [水瓶](http://lifehacker.com/prop-and-cool-your-laptop-with-a-water-bottle-385742) 到 [烘焙架](http://lifehacker.com/use-a-baking-rack-to-cool-nearly-any-piece-of-overheati-5915458) 到 [外观时尚的 DIY 笔记本电脑托盘](https://lifehacker.com/diy-laptop-tray-keeps-your-computer-cool-legs-burn-fre-5427081) 。选择你的武器，让它足够便携，你的电池会感谢你的努力。

### 5.避免完全放电

我们不打算 [卷入或解决](http://lifehacker.com/top-10-pervasive-tech-myths-that-are-only-wasting-your-5911623) 关于电池是否健康的争论，让 [电池完全放电](https://lifehacker.com/how-often-should-i-charge-my-gadgets-battery-to-prolong-5875162) 然后再充电——那是另一天的事了。然而，有一件事是肯定的——大多数电池都有有限的充电周期数 [，之后它们的效率就会急剧下降](http://www.digitaltrends.com/computing/how-to-care-for-your-laptops-battery/) 。

除此之外，让电池完全耗尽，然后必须充电到你的设备甚至可以再次工作的程度，这种麻烦已经足够了，无论如何，这可能不值得。如果你的电池快没电了，那就关掉笔记本电脑。不要让电池完全没电。

### 4.杀死耗电的应用程序和进程

除了最小化笔记本电脑显示屏的亮度，另一个既能在你电量不足之前又能在你电量不足时节省能量的方法是，在你工作时关掉所有在后台运行的耗电应用或工具。 [这是另一个老技巧](https://lifehacker.com/how-to-maximize-the-battery-life-of-your-windows-laptop-5566020) ，回到你可能有大量后台进程消耗能量的时代——比今天更多，真的——但它仍然值得关注。

这条规则在智能手机和平板电脑等移动设备上比 Windows 和 macOS 更重要，但当你在 45 分钟的电池续航时间内完成一小时的工作时，你最不希望的事情就是发现你的 torrent 应用程序一直在后台播种，或者你一直在安静地运行 Photoshop，因为你在完成后没有退出。

### 3.也杀死耗电的硬件

除了耗电的应用程序，如果你想在旅途中尽可能延长设备的电池寿命，关闭任何不必要的硬件或功能也会有所帮助。现在是禁用 Wi-Fi 的好时机，除非你需要它、T1 或蓝牙。不要为了多争取一点时间而疯狂地禁用你所有的小硬件功能，但是如果你出门在外，不需要那些耗电的功能，那就一定要把它们关掉。

### 2.尽可能使用休眠，而不仅仅是睡眠

每次不使用笔记本电脑时，你可能会试图关闭屏幕，以尽可能多地节省电池，如果你稍后再打开它，这是一个好主意，但如果可以，请使用休眠。区别很简单——在休眠状态下， [你的笔记本电脑根本不耗电](https://lifehacker.com/hibernate-vs-standby-33356274) ，而在睡眠状态下，它仍在消耗电能，只不过消耗少了很多。

这是一个古老的区别，由于 Windows 8 和 10 中的现代电源管理，这个区别变得不那么重要了(macOS 的并没有真正做出区别)。)这在短期内不会有什么影响，但如果你的笔记本电脑会在工作间隙关闭一段时间，使用休眠代替睡眠 [可以为你节省一些电量](https://lifehacker.com/how-much-battery-life-does-sleep-mode-really-drain-5526542) ，甚至 Windows 平板电脑上的。哦，如果你不确定 Windows 8 或 10 中的、休眠或混合睡眠选项在哪里，、[这本来自 How-To Geek 的伟大指南解释了所有的](http://www.howtogeek.com/102897/whats-the-difference-between-sleep-and-hibernate-in-windows/)、，以及在哪里启用它们(这也是上面的截图。)

### 1.取消这些标签，或者尝试不同的浏览器

对于许多人来说，这可能是一个失败的例子，但你的网络浏览器可能是你使用的最耗电的应用程序或工具，而且很多浏览器开发者都知道这一点。首先，如果你是那种总是打开十亿个标签的人， [熟悉如何轻松管理它们，](https://lifehacker.com/master-your-browsers-tabs-with-these-tricks-and-extensi-5883299) ，或者试试 [一个像大吊杆](http://lifehacker.com/the-great-suspender-frees-up-memory-by-suspending-brows-1702778962) 这样的附加程序，它可以在不使用时从内存中卸载它们——这也节省了你的电池。

我们这里不是开玩笑——打开浏览器标签，尤其是很多标签， [突突倒果汁](https://lifehacker.com/how-open-browser-tabs-affect-your-battery-life-866134721) 。把它们放在记忆之外或者关闭会给你额外的能量去做你需要做的其他事情。

最坏的情况是，你可以考虑使用不同的网络浏览器。我们知道，每个人都喜欢 Chrome 或 Firefox，甚至 Safari——但电池寿命最好的浏览器是 Windows 中的 [实际上是 Edge](https://lifehacker.com/web-browser-battery-usage-compared-internet-explorer-s-1511149914)([说真的](http://lifehacker.com/remains-of-the-day-microsoft-claims-edge-is-the-best-b-1782310915) ！)以及 Windows 和 macOS 中的 Opera。两者都更好地管理内存，并且 [Opera 甚至有一个低功耗模式](https://lifehacker.com/operas-power-saving-mode-cuts-battery-drain-for-longer-1776276324) 用于延长笔记本电脑的电池寿命。可怜的 Chrome [只是一个杀手](http://lifehacker.com/google-chrome-kills-battery-on-windows-faster-than-ie-o-1605816299) ，尽管我是用它来写这篇文章的。不管怎样，考虑一下你的浏览器——它可能会产生最大的不同。
