# 当你的电脑无法运行新游戏时要做的前 10 件事

> 原文:[https://kotaku . com/the-first-10-things-to-do-when-your-PC-can-t-run-a-new-1789489551](https://kotaku.com/the-first-10-things-to-do-when-your-pc-can-t-run-a-new-1789489551)

你刚刚得到了一个新的电脑游戏！伙计，你很兴奋。你期待这一天已经很多年了。你把它装上，然后…哦，见鬼。

Watch

有点不对劲。你的帧速率在剧烈波动。鼠标输入全错。每当有人讲话时，介绍性的序列会暂停一两秒钟。当你移动摄像机时，会有某种可怕的口吃。

***此片最初出现于 11/30/16。**T3】*

你进入选项菜单，进行通常的调整。关掉 MSAA。关闭阴影质量。关闭倒影。关闭任何具有奇怪名称和 Nvidia 品牌的功能。这些都没有多大帮助。这个游戏在你的电脑上运行不太好。

这是你接下来要做的。

### **1。看看“自动检测设置”是多么的错误。**

嘿，这个游戏有一个自动检测最佳设置的选项！也许这能让它体面地运行。你会自动检测你的设置。

打开 MSAA 时，你无法获得稳定的帧速率，但现在它被设置为 MSAA x8…阴影被提升到一种叫做“英伟达 GCSS 真实世界生活”的东西，你甚至从来没有听说过。分辨率现在超级采样到 4K，然后回到显示器的原始分辨率。这款游戏开启了一种名为“远程四采样”的功能，它将你的显示信号发送到阿拉米达的一个服务器农场，在那里进行重新处理，然后发送回你的显示器。

你的游戏现在以大约 11fps 的速度运行。干得好，自动检测。

### **2。访问游戏论坛，痛苦地阅读那些没有问题的人。**

每一款大型新游戏都会在 NeoGAF 和 Reddit 等论坛上获得 PC 性能线程。如果你的游戏运行得不好，访问这些线程并浏览它们，直到你找到声称他们没有问题的人。这些帖子很容易找到，因为它们总是以下内容的变体:

> 我不明白这里的每个人都在抱怨什么。在 GTX 770 上以最高设置运行游戏，每秒 60 帧。不蘸。

盯着那些帖子。拥抱你心中的恨。那些人不知道他们在说什么。他们一定在撒谎。他们显然是坏人。或者他们的眼睛有问题。也许你应该同情他们。

### **3。沉迷于基准测试。**

一批 PC 游戏出版物将会运行游戏的基准。尽可能多地找到这些。每一个都有一个 FPS 结果的集合，像绿色糖块一样堆叠在一起。在他们中间的某个地方有一个条，上面有你的图形卡的名字。

无论你的电脑与某个出版物的基准测试中列出的规格有多接近，你都无法像基准测试所建议的那样运行游戏。TechTimesPro 的这个人用你的卡平均 80fps！MegaBitz 的这个人每秒 75 帧。你最多只能得到 32fps。显然这是你的错。

至少花一个小时仔细研究这些基准，想知道你的不足是什么。也许你还有一些需要调整的地方？或者也许…哦，上帝。可能这个游戏是 CPU 绑定的。

### **4。尝试确定游戏是否“受 CPU 限制”**

当你访问游戏论坛时，你会发现许多人在讨论游戏的性能是否受到 CPU 的限制。你已经几年没有升级你的 CPU 了，因为你认为你的显卡更重要。看了几个关于 CPU 使用的论坛帖子，你就不再那么肯定了。

有些人会坚持认为，在这个新游戏的性能等式中，CPU 实际上是最重要的东西。其他人会说事实并非如此。你会读到这些相互矛盾的观点，并开始对你的 CPU 感到隐约的悲哀。也许你应该买一个更好的 CPU？

### **5。考虑升级您的整个电脑。**

或者如果你的电脑只是坏了怎么办？从根本上说。你有一个非常昂贵的显卡和一个快速的 CPU。但如果是更一般的，无所不包的坏呢？堵塞的数据管道和插头的某种模糊组合，使你的计算机很难以应有的速度处理数据？

如果你把主板扯下来重新来过呢？或者问题出在你的电脑外壳上？如果你买了一个全新的箱子，重新开始，会怎么样？如果你搬到一个新公寓会怎么样？在一个新的状态？如果你改了名字，然后用那个名字重新安装 Windows 会怎么样？

### **6。躺在床上盯着天花板，想着超频。**

显卡超频了怎么办？或者再超频*多*呢？如果你超频了你的 CPU 会怎么样？如果你买了一个新的 CPU 和一个新的显卡，并且你超频了它们会怎么样？你的新公寓可能会有更好的通风和更好的电源插座。当然，这将有助于超频。

### **7。不要想你已经在这上面花了多少时间。**

在游戏发布后的三天里，你已经花了整整六个小时排除电脑性能故障。你千万不要纠结于这个事实。

例如，不要考虑你可以用其他方式度过这段时间。你可能已经和失去联系的朋友重新联系上了。你可能正在尝试一个令人兴奋的晚餐食谱。你可以玩任何一款已经在你的电脑上运行良好的游戏。你可能在打盹。

什么都不要想。让这个游戏运行起来是你生命中最重要的事情。你会找到解决办法的，这一切都是值得的。

### **8。简单考虑一下获得控制台版本。**

当您打开丽娃调谐器统计服务器第五次重新调整 FPS 上限时，请暂停并考虑控制台版本。所以游戏在主机上以 30fps 的速度运行。也许还是更好的体验？至少，你不用担心调整更多的图形设置。

让自己在这件事上紧张足够长的时间，以至于你开始真正考虑在 Steam 上退款，转而在游戏机上购买游戏。考虑一下你的游戏机下载游戏需要多长时间，以及玩游戏机版本会让你不断怀疑你是否太早跳槽了。意识到放弃 PC 版将是多么的失败。才过了几天！如果他们修好了呢？你当然可以再等一会儿。

抛弃一切控制台版本的想法。带着新的决心回到你的电脑前。如果这个游戏要了你的命，你也要让它成功。

### **9。真诚地努力放松，享受这该死的游戏。**

深呼吸。你为了玩这个游戏已经等了几个月了。当然，它有一些性能问题，但那又怎么样呢？看看那些说他们玩得很开心的人！

关闭屏幕一角的 FPS 计数器。不要不停地转动相机寻找挂钩。也许一旦你读完了第一章，事情就扯平了。也许过一段时间你就不会注意到这些问题了。也许你的整个体验不会被这种痛苦的感觉所污染，即你正在通过玩一个不完整版本的游戏来毁掉自己的东西，你等待多年来体验的不可思议的惊喜和史诗般的设定将被恼人的技术问题不可挽回地破坏。

可能…妈的。

### 10。放弃等待补丁。

别玩了。查看发行商的 Twitter feed，看看他们是否提到了某个补丁。检查游戏的子编辑，看看是否有人在谈论补丁。谷歌“[游戏名]补丁”，看看有没有关于某个补丁的消息。不会有，但是继续找。最终你会发现论坛上满是抱怨补丁没有任何消息的人。从阅读中找到些许安慰。

最终，你会玩这个游戏。总有一天。
