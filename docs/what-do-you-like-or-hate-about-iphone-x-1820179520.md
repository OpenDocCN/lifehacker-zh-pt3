# 你喜欢(或讨厌)iPhone X 的什么？

> 原文：<https://lifehacker.com/what-do-you-like-or-hate-about-iphone-x-1820179520>

最近发布的 iPhone X 从拿起智能手机 的那一刻起，在 iOS 用户如何与智能手机 [互动方面带来了相当彻底的转变。它有一个相当大的屏幕，面部扫描技术，和一对相机，旨在使你的主题流行而不用携带一个花哨的相机。](https://lifehacker.com/iphone-x-tips-and-tricks-you-need-to-know-right-now-1820086402)

Watch

但并不是 iPhone X 的一切都是朝着正确方向迈出的一步，这取决于你问谁。例如， [不再有 Home 键](https://lifehacker.com/use-this-hidden-feature-when-your-iphones-buttons-are-b-1819959579)——取而代之的是，X 使用新的 [Face ID 技术](https://gizmodo.com/iphone-x-first-look-lets-talk-about-that-whole-face-id-1819989200) 作为登录方式。那个生动的 5.8 英寸有机发光二极管屏幕顶部有一个切口，这意味着某些应用程序以及你在设备上观看的每个视频都会在它的侧面有一个黑色条(除非你手动调整视频的大小)。此外，在维修成本方面，玻璃覆盖的智能手机将会让你花一大笔钱。

所以，iPhone X 的机主们:你们喜欢你们 iPhone X 的什么？你讨厌什么？虽然未来的面部扫描、散景友好设备听起来很棒，但当它与正常人的现实发生摩擦时，似乎会遇到一些障碍。你对新智能手机最大的不满是什么？现在千元手机是你的了，你离不开什么？
