# 当你吃饭的时候，尝试这个仪式来更好的自我控制

> 原文：<https://lifehacker.com/try-this-ritual-for-more-self-control-when-you-eat-1829132300>

仪式可以帮助我们更好地掌控自己的生活，在会困扰我们的情况下更舒服。一组研究人员现在表示，他们发现一种奇怪的、无意义的饮食习惯可能有助于我们摄入更少的卡路里。

Watch

在仪式中(在*个性与社会心理学杂志* 的付费墙后面发表，但这里也有)，你每次吃饭都要做三件事。以下是研究人员向研究参与者展示的说明:

*   首先，在吃之前把你的食物切成块。

*   第二，重新排列这些碎片，使它们在你的盘子上完全对称。也就是说，让你盘子的右半部分看起来和左半部分完全一样。

*   最后，把你的食具压在食物上面三次。

研究人员在校园健身房招募了正在努力减肥的女大学生，如果她们在进行这项活动的小组中，她们每天摄入的热量会更少。对照组只是被要求“用心”进食，但没有任何指示。

研究人员说，这些步骤都没有什么特别之处；他们设计的仪式是任意的，但简单且可重复。它也很严格:“为了在研究中，你必须在每次吃饭时做这个仪式的三个步骤。”

### 遵循这个仪式是什么感觉

我尝试了几次这种仪式。感觉*傻*。昨天午餐，我吃了一些剩下的泰国咖喱饭。我用叉子的侧面切了几块较大的鸡肉，然后尽最大努力对称地安排用餐。我感觉这种仪式并不适用于所有食物；你会做什么汤？(也就是说，它远没有那些 [用橙子擦嘴唇](https://www.youtube.com/watch?v=7CCr5w3ox_4) 的用心饮食练习傻。)

晚餐是披萨。我切好，整理好，用我的食具(呃，我的手指)在食物上按了三次。第二天的早餐，我考虑是要烤面包还是燕麦片，然后决定不去打扰了。我不是唯一一个对这种仪式感到厌倦的人:研究人员指出，“在研究结束时，我们的参与者说他们认为这种仪式不是很有帮助，并报告说他们不太可能继续下去。”

但我确实喜欢在每餐前增加一点仪式，让我对自己吃的食物多一点关注。过于刻板的仪式 [可能是饮食失调的迹象](https://www.eatingdisorderhope.com/information/eating-disorder/ritualistic-eating) ，但是在我们的生活中也有健康仪式的空间。例如，也许你喜欢每天早上用同样的方式煮咖啡。我们也在社交场合参与食物仪式:在我们端上生日蛋糕之前唱歌，在我们喝酒之前碰杯。

所以，如果你想看看这个仪式对你是否有效，那就试试吧。或者，更好的是，自己编一个。
