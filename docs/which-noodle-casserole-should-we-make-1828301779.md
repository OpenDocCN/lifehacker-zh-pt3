# 我们应该做哪个面条砂锅？

> 原文:[https://life hacker . com/which-noodle-砂锅-should-we-make-1828301779](https://lifehacker.com/which-noodle-casserole-should-we-make-1828301779)

大家好，欢迎回到 [*将它砂锅*](https://lifehacker.com/tag/will-it-casserole) ，在这个栏目里我们看到哪些饭菜、菜肴、菜系可以重新想象，改造成砂锅。

上周的乳糖热梦想是一种享受，但我想解决一个更经典的砂锅类:砂锅面条。砂锅面流派纷呈，然而当我在 [*会砂锅档案*](https://lifehacker.com/c/will-it-casserole) 中翻找的时候，一个也找不到。这显然必须被纠正，但是我需要你的帮助来挑选做哪个砂锅(我经常这么做)。以下是一些想法:

*   **金枪鱼砂锅面:**这一流派的标志。我第一次吃的时候，是用卡夫通心粉、金枪鱼罐头和冷冻豌豆做的。我喜欢它，但我觉得我们可以把它修饰一下。
*   砂锅鸡肉意大利面:这道米色、甜椒和甜椒点缀的奶油面条是我继母最喜欢做的一道菜，我很想为你做这道菜。(稍微调整一下，因为我必须调整。)
*   **火鸡 tetrazzini:** 我没吃过这个，但是看起来还不错(而且简单)。
*   砂锅鸡肉饺子:饺子不是面条，但我认为我们可以在这里做一些风格上的改变，做出一些非常特别的东西。

这是我们可以想象的四个非常基本的选择，但我很想听听你最喜欢的砂锅面条是什么，因为老实说，我从小到大没吃过多少。我想 kugel 也适合这种类型，所以如果你有 kugel 的建议，我也愿意接受。
