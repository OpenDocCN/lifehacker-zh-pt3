# 提交工作申请后你可能会犯的三个错误

> 原文：<https://lifehacker.com/three-job-application-mistakes-you-might-make-after-alr-1789837315>

在某些方面，点击“提交”按钮申请工作是一种宣泄。你已经付出了很多努力来美化你的简历和求职信，坦率地说，你对整件事已经没什么感觉了。问题是，对许多人来说，仅仅过了几分钟，他们就开始思考他们可能做错的所有事情。

如果我告诉你我在申请一些工作后的所有担忧，这篇文章至少会是现在的五倍长。但事实证明，你担心的很多事情(我把招聘经理的名字拼对了吗？我应该寄一份奖金写作样本吗？)现在已经不受你的控制了。所以不要过度紧张，集中精力不要犯这三个错误。

### 1.你忽略了工作列表上的后续说明

如果我每在网上发布一份描述中有“不打电话”字样的工作就能得到一美元，那我就有几十美元了。但是，如果每次有候选人无视这些指示，我都能得到同样的一美元，我明天就可以退休了。

这应该是你申请职位时首先要考虑的事情之一，尤其是在你提交了材料之后。当然，传统观点可能会说这是抓住招聘人员注意力的好方法。技术上来说，是的。

但是，如果一份工作清单强调公司不想听到求职者的声音，在这种情况下“积极主动”对你的事业没有帮助。事实上，如果你在电话中过于坚持，你可能会失去回复的机会。

相反，如果你非常想知道你的状态，电子邮件是一个更好(也不那么烦人)的选择。在一份申请上写一封跟进邮件可能很难，但是每当你陷入灵感时， [使用这个方便的模板](https://www.themuse.com/advice/how-to-follow-up-on-a-job-application-an-email-template) 。

### 2.你没有仔细检查你是否在公司有关系

想在你积极申请的公司中保持领先地位吗？冰冷的 [跟进邮件](https://www.themuse.com/advice/how-to-follow-up-on-a-job-application-an-email-template) 不一定管用，但快速浏览一下雇主的 LinkedIn 页面，看看你是否认识目前在那里工作的人，或许会管用。

通常我会说你应该在申请前做这件事，但我明白了。有时你看到一个听起来很棒的演出，你想尽快提交你的材料。我已经这样做了很多次，只是后悔立刻跳过了这一步。在大多数情况下，我只是简单地把它归结为一个已经不存在的错过的机会。

事实是，虽然等到你提交了申请之后再联系你梦想中的公司并不总是理想的，但一些雇主实际上更喜欢这样。即使公司没有强烈的偏好，在朋友和前同事最初提交申请材料后，我已经为他们写了很多推荐信。所以，如果你还没有找到时间去了解你认识的人，不要害怕在事后去发现(并伸出手)。

### 3.你没有在找其他工作

有时候，你发现一份工作看起来如此不可思议，以至于你只想无所事事地等着雇主的电话。既然这份工作适合你，那就没有必要去考虑外面还有什么，对吗？不对。太，太不对了。

我知道编辑和修改简历和求职信有时会很麻烦，尤其是当你只想尽快找到下一份工作的时候。但现实情况是，这是你表达对梦想工作的兴趣的方式——而这只是漫长过程的第一步。

当然，你可能会从你所指望的一个雇主那里得到回复，一切都可能如你所愿。如果你是这种情况，我邀请你在推特上对我说，“你错了！我等到了我的梦想公司，也只有我的梦想公司，一切都很神奇。”

但是，如果你像我们这些普通人一样，在过去因为令人惊叹的工作而被拒绝，那就帮自己一个忙，找到继续寻找的能量。毕竟，尽管这个职位听起来很棒，但可能还有一个你还没见过的更好的职位。

事实是，无论何时你申请一份工作，你都把很多控制权交给了另一边审核候选人的人。这是一个可怕的提议，尤其是考虑到一旦你点击了“发送”，你就无法改变你的应用程序然而，这并不意味着你应该恐慌和犯错，或者你应该只是坐以待毙。虽然你不再能控制这个职位的进程，但你仍然有一些——拥抱它。
