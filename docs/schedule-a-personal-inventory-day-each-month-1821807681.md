# 每月安排一个个人盘点日

> 原文:[https://life hacker . com/schedule-a-personal-inventory-day-month-1821807681](https://lifehacker.com/schedule-a-personal-inventory-day-each-month-1821807681)

与制定新年决心同样重要的是，弄清楚我们如何实现它们，不要陷入工作中而忘记照顾自己。在记者安·弗莱德曼和科技女性黑手党的联合创始人阿米纳图·索乌主持的播客 [给你的女朋友](http://www.callyourgirlfriend.com/episodes#/level-up-2018/) 打电话上，人权技术专家 [萨布丽娜·赫尔西·伊萨](http://beingbrina.com/) 建议每个月安排一个个人盘点日，定期盘点你的时间都花在了哪里。

赫尔西·伊萨是数码机构 Be Bold Media 的首席执行官，也是 Jump Canon 的风险投资合伙人，Jump Canon 是一家为“非常规、代表性不足和引人注目的创始人”服务的风险投资公司，因此她对完成工作略有所知(Sow 称她为“世界上有史以来最有能力的人”)。在播客中，她强调了留出时间来考虑我们的时间去了哪里以及我们希望它去哪里的重要性——不仅仅是在年初或我们的生日，而是每个月。个人盘点日就是有意识地盘点你想要怎样过你的生活，你的优先事项是什么，以及你如何实现它。

“无论我觉得我需要做什么来保持我的生活运转紧密，我都在这一天执行，”她说。“评估一下你自己想要什么，对自己真正诚实，*对自己，*对别人。毫无歉意地关注你的需求，并把它写下来。写下你的目标，制定一个计划，然后把它放在一个你可以很容易找到的地方，以便将来参考。”

以下是她的建议:

*   每个月在你的日历上划出一天(她选择了自己的生日，16 号)，用一个重复出现的事件，叫做个人盘点日。
*   利用这一天完成你需要完成的任务。这可能意味着检查你的信用评分，查看你的储蓄目标和预算，安排医生的预约，或者跟踪你的其他决定。
*   问问自己:这一个月我学到了什么？什么让我吃惊，为什么？我感激什么？我感激谁？(赫尔西·伊萨会给她感激的人发送简短的感谢信。)
*   每个月把你想在 PID 上处理的任务添加到 Evernote 或 Google Doc(或老式的纸笔清单)上，这样你就不会忘记任何重要的事情。
*   在年底，或者在你生日的时候，回顾一下你从你的 PID 中收集的数据。你为自己做过什么？你感激什么，感激谁？你在学什么，你对什么感到好奇？“看看你现在的位置和你想要的位置之间的差距在哪里，并用它来衡量你需要努力的方向，”她说。

最终，PID 是关于确保你在你想走的道路上，并且有意识地安排你的时间。

赫尔西·伊萨说这提醒了我“我设定自己的优先事项，我掌控自己的生活。如果你的梦想对你很重要，不要不尊重它们，把它们塞进任意的新年决心或生日截止日期。”

这也降低了她的压力水平，释放了过去被担忧和怀疑占据的大脑空间。

“你掌控自己的生活，你设定自己的优先事项，然后你开始茁壮成长，”她说。
