# 你不能学习你认为你已经知道的东西

> 原文：<https://lifehacker.com/you-cannot-learn-what-you-think-you-already-know-1796095391>

本周的斯多葛派引语来自希腊哲学家 [爱比克泰德](https://en.wikipedia.org/wiki/Epictetus) ，他生来就是奴隶，他的一个学生把他的教义写了下来。这些作品最终会成为他的*演讲*和*演讲*。

> "一个人不可能学到他认为已经知道的东西。"——爱比克泰德，*论述*，第二册，ch。17

另一个翻译:

> “一个实践哲学的人的首要任务是什么？摆脱自负。因为任何人都不可能开始学习他认为他已经知道的东西。”——爱比克泰德，*论述*，第二册，ch。17

### 这意味着什么

无论是一项技能、一门手艺，还是普通知识，提高自己就是学习。但是如果你认为已经没有什么可以学的了，你就学不会了。如果你知道的只是很多可能性中的一种，而且有些比其他的更好呢？如果你所知道的可以被其他人所知道的进一步加强会怎么样？如果你知道的事实上是错的呢？

如果你不表现出谦逊，把所有的事情都当成一个空的、自愿的容器，你就不会成长。永远不要假设你的头脑是满满的，或者你所知道的压倒了别人所知道的。毕竟，如果每个人都认为他们一学会*某件事情*就知道*所有事情*，那么在任何领域都不可能取得进步？这是傲慢和愚蠢的，即使是对那些在他们领域的顶尖人物来说。

### 从中吸取什么

无论你在生活中追求什么，都要保持对学习的渴望。选择永远有一颗初心，不断寻找另一个角度。接受错误，并在错误发生时乐在其中。只要你把它作为一个教训，错了也没关系。不要害怕提问，让自己看起来像个业余爱好者。如果你愿意，开放的心灵会是你最大的财富。

我们是固执的生物，喜欢假设我们比别人更聪明或更有能力，但是希望你更优秀会让你更自卑。抛弃自我，接受你作为一个永远的学生的位置。再说， [没人喜欢一个无所不知的](http://lifehacker.com/dont-be-a-know-it-all-be-a-learn-it-all-1794974291#_ga=2.95879967.24664540.1497201492-27973805.1434581949) ，真正的“专家”是那些不懂事也会承认的人。为什么？因为他们想学。
