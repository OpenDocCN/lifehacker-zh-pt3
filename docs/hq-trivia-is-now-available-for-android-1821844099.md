# HQ 琐事现在可以在 Android 上使用

> 原文：<https://lifehacker.com/hq-trivia-is-now-available-for-android-1821844099>

如果你一直嫉妒你的 iPhone 朋友玩 HQ 琐事的能力，你很幸运:这款游戏本周正式登陆 Android 设备。

Watch

如果你一直生活在岩石下，不知道什么是 HQ 琐事:基于应用程序的琐事游戏主持现场琐事展示，每个人都可以用应用程序回答问题。

现场主持人负责提问，所有答对的人都将获得一份真正的现金奖励。现金奖励因游戏而异(如果没有人赢，则滚动到下一个游戏)，并在所有赢家之间分配。例如，如果一个游戏的奖金是 1000 美元，1000 人“赢”了，那么每个人将得到 1 美元。

我们 [上周写了一个在 Android 上提前获得游戏的变通办法](https://lifehacker.com/how-to-get-hq-trivia-for-android-now-1821602002) 。现在， [在 Google Play](https://play.google.com/store/apps/details?id=com.intermedia.hq) (简单多了！)，而是作为测试版演示。这意味着你可能会遇到一些打嗝，但在大多数情况下，应该会很顺利。一个完全合法的版本应该很快就会出现。
