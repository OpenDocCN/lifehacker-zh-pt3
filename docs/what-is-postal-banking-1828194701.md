# 什么是邮储银行？

> 原文：<https://lifehacker.com/what-is-postal-banking-1828194701>

大多数美国人可以依靠直接存款、借记卡和在线账单支付在日常生活中无缝工作。当然，可能会有费用，而且我们大多数人都可以在我们的账户上多花点钱，但是金融系统的机制运作得如此之好，以至于我们甚至想都不用想它们。



但对于美国约 900 万家庭来说，情况并非如此，他们没有银行账户或塑料卡可以刷，也不能在银行系统的正常范围内工作。一些人不信任银行，而另一些人负担不起大多数国家银行收取的费用和账户最低限额。银行没有向这些客户提供金融产品的压力，因为他们不会从中获利(毕竟，你的支票账户需要付费是有原因的)。

仍然有更多的人依靠天文利率的发薪日贷款来度过下一周，因为他们没有其他选择。 [琼斯母亲](https://www.motherjones.com/politics/2018/08/kirsten-gillibrand-has-an-ambitious-plan-to-take-on-payday-lenders-the-post-office/)注意到，“没有银行账户的家庭平均每年花费 2412 美元 [或者他们平均年收入的 10%左右](https://harvardlawreview.org/2014/02/its-time-for-postal-banking/#_ftnref11) ，用于支付大多数银行客户免费获得的金融服务的利息。”正如[*时间*](http://time.com/5351706/postal-banking-kirsten-gillibrand/) 所说，贵在穷。

纽约州参议员柯尔斯顿·吉利布兰德(Kirsten Gillibrand)公布了一项立法，为这个问题提供了一个解决方案:要求美国邮政局 向客户提供基本的金融服务，包括支票账户、有息储蓄账户和短期贷款。

吉利布兰德在一份新闻稿中说:“对于数百万无法获得或只能有限获得传统银行服务的家庭来说，兑现薪水支票或获得小额贷款以修理汽车或支付汽油账单的简单行为最终可能会花费数千美元的利息和费用，几乎不可能还清。”

提供支票账户足以让许多美国无银行账户的人进入金融系统，但吉利布兰德的计划远不止于此，据报道:它针对的是掠夺性的发薪日贷款行业:

> 该法案规定，邮政银行可以以与月国库券收益率一致的利率一次发放高达 500 美元的贷款，目前月国库券收益率为 1.65%。考虑到这些贷款中有多少可能违约，这个利率可能太低了。邮政服务监察长的报告描绘了接近 25%的利率，与年利率通常为 400%左右的发薪日贷款相比，这仍将为客户节省数百美元。

这对吉利布兰德来说并不容易——发薪日贷款人在华盛顿很有势力(问问消费者金融保护局局长米克·马尔瓦尼 )就知道了。但这是一个值得考虑的严重金融问题的解决方案。
