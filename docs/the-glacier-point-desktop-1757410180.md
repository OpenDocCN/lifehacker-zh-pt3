# 冰川点桌面

> 原文：<https://lifehacker.com/the-glacier-point-desktop-1757410180>

Stranha 没有留下很多关于他如何制作这个漂亮的桌面的信息，中心有半个圆顶，但我们可以弄清楚。如果你想让你的 Mac 看起来和摸起来都像这样，你需要做以下事情。



这里发生了一些实实在在的定制，但总的来说，这只是少数几件事。这是你需要的:

*   壁纸包含在 OS X“埃尔卡皮坦”Beta 6 中，所以你可能已经在你的电脑上安装了。如果没有，或者你运行的是 Windows 或者 Linux， [在这里抓取](http://osxdaily.com/2015/08/04/get-a-stunning-new-el-capitan-wallpaper-of-glacier-point-from-beta-6/)
*   [über sicht](http://tracesof.net/uebersicht/)，一个我们喜爱的 OS X 定制套件，带有小部件(电池显示、时钟和天气小部件)
*   [cDock](http://w0lfschild.github.io/app_cDock.html) ，一个给你 [完全定制控制](http://lifehacker.com/cdock-customizes-the-background-and-color-of-your-mac-s-1688253035) 你的 Dock 的工具，用于 Dock 间隔和透明度的调整
*   [极简白色图标包](http://noshery.deviantart.com/art/Minimalist-White-Icons-Mac-OS-X-470738808) 自定义任务栏图标

这里发生的事情不多，但发生的事情很好，放置得很好，所以你可以自由地完成一些工作，只在需要时才看到小部件——图标和 dock 设置为与 stranha 一起工作，而不是让他为他们工作。如果你喜欢这个成品，或者对它是如何完成的有疑问，点击下面的 Flickr 链接。
