# 让我们储存一些足球零食吧

上周的甜辣利口酒制作和饮用都很有趣，但本周我想让我们回到可咀嚼的话题。足球是目前正在发生的事情，无论你是否观看，你都必须承认观看足球比赛的食物是一种享受。我很想看看我们是否能提供一些尾门(或从你的沙发上看)费用，例如:

*   翅膀:我怀疑真空鸡翅的问题是如何将肉留在骨头上，这并不是世界上最糟糕的问题。
*   **肉丸:**更好——*迷你*肉丸。
*   好吧，这不是标准的后挡板食物，但我认为在一个以足球为中心的聚会上，肉酱会很合适。
*   热狗:在烧烤季节，我们不知何故错过了这个。
*   奶酪塞辣椒:他们吃完后，我们可以给他们一个快速火炬。

我知道上面的清单有点重肉，所以如果你有一些素食者友好的足球观赛小吃，我也很想听听。如果你有一个非常好的建议，我甚至愿意听一些与足球完全无关的建议。
