# iPhone X:六个月后

> 原文：<https://gizmodo.com/iphone-x-six-months-later-1825722111>

iPhone X 发布已经六个月了，有一点是成立的:它仍然很贵。然而，尽管我恨我自己 [买了一部 1000 美元的手机](https://gizmodo.com/im-going-to-buy-the-iphone-x-and-i-hate-myself-for-it-1804418016) ，尽管我恨我自己 [喜欢它](https://gizmodo.com/i-hate-how-much-i-love-the-iphone-x-1820086867) ，与 iPhone X 相处半年揭示了一些新的真相。第一，当 Jony Ive [说 Face ID 比看起来更重要时，他是对的。第二，苹果对卓越质量控制的把握正在下滑。第三，一旦你爱上了 iPhone X，就没有回头路了。](https://9to5mac.com/2017/10/11/jony-ive-interview-iphone-x/)

iPhone X 就是这么容易爱。但即使是最可爱的东西也有令人厌恶的缺陷。开箱即用，iPhone X 是一款设备的怪异变种。它看起来(有点)像 iPhone，你可以像 iPhone 一样与它互动。但这款 iPhone 与之前几代设备之间的一些主要差异是显而易见的。屏幕延伸至整个设备。Home 键没了。以奇怪的方式滑动是导航设备的新方式。这意味着，首先，简单的苹果式的无废话互动似乎并不存在。这些基本事实证实了 iPhone X 的与众不同，但我对自己学习新行为的速度感到惊讶。在使用手机的几个星期里，不同的感觉开始变好。

Face ID 首先让我大吃一惊。苹果并不是第一个将面部识别技术引入设备的公司，当苹果宣布这项技术时，Face ID 确实感觉像是一个噱头。但是这项技术非常好用，以至于它改变了我使用 iPhone 的方式。一百次中有九十九次，当我看着我的手机时，它不到一秒钟就解锁了。我戴着帽子看手机，手机解锁。我在黑暗中戴着帽子看手机，手机解锁。它对其他任何人都不解锁— [目前来看，至少](https://gizmodo.com/watch-me-try-to-break-face-id-a-bunch-of-different-ways-1820020355) 。当我做一些疯狂的事情时，比如把手放在面前，它就不会解锁。手机解锁如此容易，以至于这个功能似乎消失了。我只是不再考虑解锁我的手机。

这个魔术并不是对每个人都有效。在与朋友和同事交谈时，Face ID 可能会前后矛盾。我的一些朋友说，它有一半的时间是有效的，而另一半时间他们必须使用密码。也就是说，Face ID 仍然是你可以在 iPhone X 上得到的东西，你在其他型号上得不到，对我来说，这是一个变革性的功能。作为一个额外的奖励，Face ID 让我可以无缝地看到通知，然后只需扫一眼就可以解锁它们的内容。所有这些加起来就是一部简单易用的手机。

在日常使用方面，我也被这款设备的计算速度所震惊。据报道，iPhone X 的 A11 仿生芯片使该设备比 MacBook Pro 更强大。这很棒，但我没有在手机上做很多视频编辑，所以我肯定不理解芯片的真正威力。我经常玩《模拟城市》,我很高兴地告诉大家，这款游戏的加载速度可能是我的旧 iPhone 7 的两倍。这意味着，iPhone X 是一个强大设备的真正标志是，我从来没有真正感觉到我在等待这款设备发挥作用。

有一件事确实需要时间，那就是学习一些新的手势，但是说实话，这比我想象的要简单。我最喜欢的两个是两个最简单的:触摸唤醒和唤醒。现在查手机这么容易！与此同时，在没有主页按钮的情况下，有许多滑动可以让你导航设备。向上滑动以带回家可能是最自然的一种，而且它实际上很有趣，以至于我发现自己在使用旧款 iPhones 时也在尝试这样做。(没用。)滑动屏幕底部来切换应用程序也很流畅，尽管直到我拥有手机几个月后，我才真正养成了这样做的习惯。并不是所有的新手势都很棒。强制戒烟尤其糟糕，因为它不仅不直观，而且很麻烦。

奇怪的是，我对 iPhone X 的其他抱怨都围绕着硬件。这款价值 1000 美元的设备开箱后看起来光彩照人，但通过日常使用，在没有外壳的情况下， [它的损坏速度相当快，](https://gizmodo.com/iphone-x-damage-report-two-months-later-1822644663) 。该显示器被证明特别容易出现划痕，这个问题 [已经在其他地方](https://www.reddit.com/r/apple/comments/7p9wqc/iphone_x_screen_scratches_easily/) 报道过。有些划痕非常严重，我可以在显示器打开时看到它们，所以我要说这不仅仅是一个外观问题。显示屏玻璃上的划痕特别奇怪，因为尽管我总是把手机放下，屏幕朝上，但背面玻璃在六个月后几乎没有瑕疵。突出的相机镜头肯定有助于保持背面玻璃的安全，但他们磨损程度的差异真的让苹果看起来好像在正面和背面使用了不同类型的玻璃。

我对这款昂贵设备的另一个问题是纯粹的装饰。在我的太空灰型号上，不锈钢边缘在镀铬层上有一层烟熏的涂层。然而，我最近注意到，漆面正在碎裂或剥落，有时甚至没有出现损坏。这让我想起了我的 iPhone 5 的蹩脚表面 的 [问题，我也不是唯一一个经历这个问题的](https://gizmodo.com/the-iphone-5-damage-report-two-months-later-5963445) [。再说一次，这个问题完全是装饰性的，但是花很多钱在一个几个月后感觉脆弱或有缺陷的小玩意上是令人沮丧的。](https://www.reddit.com/r/iPhoneX/comments/80uiee/how_common_is_it_for_the_edge_of_the_space_grey/)

我知道，我知道。“放个箱子上去，”你在想。“把所有漂亮的工业设计藏在皮革或硅胶下面。把钢化玻璃放在显示器上，这样每次触摸时都感觉很恐怖。”我从来不喜欢案例，原因很简单，现在也肯定不会开始。为什么我会，当 iPhone X 没有一个是如此美丽？至少一开始是这样。

即使有瑕疵，我的 iPhone X 体验相当于积极体验的拼贴。我的手机有漂亮的大显示屏，可以很方便地放在口袋里。当我的狗做一些可爱的事情时，我可以抓起电话，用光学变焦快速地拍下她的小狗脸，这样我就不会错过这个时刻。我可以看《教父》,不会因为画面太小而心烦意乱。(我已经在我的 iPhone X 上看完了整个三部曲。)我可以在我的日历、我的短信和一个航空公司应用程序之间切换，并在旅途中计划一次旅行，而不想失去理智。这就好像 iPhone X 缓解了许多关于经常使用智能手机的痛点。这款设备并不完美，但同样，使用起来也很愉快。

在一天结束的时候，iPhone X 几乎感觉像是一部永远的手机，我永远都不需要升级。易剥落的边缘和粗糙的屏幕令人讨厌，但我可以随时使用我的 AppleCare 并更换它们。(我已经试图让苹果更换容易划伤的屏幕，但 Genius Bar 告诉我划痕是我的错，并要求 30 美元的屏幕更换费。)然而，就勇气而言，我无法想象苹果会很快推出有意义的新硬件——Face ID 就是一个有意义的新技术的例子。我还 [相信传言](https://gizmodo.com/it-looks-like-apple-may-be-making-a-bunch-of-cheap-new-1825351219) 苹果将为低价机型带来边到边显示和 notch，iPhone X 外形将成为新标准。

如果我的 iPhone X 也能坚持几个产品周期，那就太好了。太贵了！我放在口袋里的 256GB 型号价格超过 1200 美元。如果时光倒流到 iPhone X 发布的 11 月 3 日，我会毫不犹豫地再次购买。如果在另一个世界，我还在使用我的 iPhone 7，我会三思而后行，现在就买一部新的 iPhone。苹果通常会在 9 月份发布新手机，这与日历上的时间很接近，我可能会等着看接下来会发生什么。但请相信我，苹果的下一款手机几乎肯定会与 iPhone X 非常相似。它 [可能不会被称为 iPhone X](https://gizmodo.com/dont-be-surprised-if-the-iphone-x-goes-away-1825565223#_ga=2.211363925.1123332413.1525113942-1282447779.1518012133) ，但这无关紧要。如果它和 iPhone X 一样好用，你会想要它的。
