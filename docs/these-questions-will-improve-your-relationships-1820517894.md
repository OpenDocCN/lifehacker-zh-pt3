# 这些问题会改善你的人际关系

> 原文:[https://life hacker . com/these-questions-will-improve-your-relationships-1820517894](https://lifehacker.com/these-questions-will-improve-your-relationships-1820517894)

你和陌生人有过心灵相通的瞬间吗？我说的不是浪漫或性关系(虽然这些也很好)，而是更多的是你在街上经过时的一个快速微笑，或者在杂货店排队等候时分享的一个一次性笑话，或者其他一些简短的、共同的经历，让你觉得那个陌生人实际上很特别，在其他情况下，可以成为朋友？我喜欢那些为数不多的时刻，因为它们让我觉得潜在朋友的世界比我想象的要大。我一直想知道为什么那些时刻会发生——为什么它们发生在一个人身上而不是另一个人身上，或者发生在某个时间而不是另一个人身上。

我最近偶然看到凯瑟琳·谢弗勒的帖子 [如何在一秒钟内改变你的生活](https://www.thriveglobal.com/stories/16020-4-questions-we-unconsciously-ask-near-constantly) ，她在其中讨论了她的信念，即我们总是问我们身边的每个人四个问题——我们与之有关系的每个人，从泛泛之交到我们的浪漫伴侣。她说这些问题来自玛娅·安杰洛(尽管她没有具体指出是哪里)，是对大大小小的承认和肯定的不断要求。

这四个问题是:

1.  你看见我了吗？
2.  你在乎我在这里吗？
3.  我对你来说够了吗，还是你需要我在某方面变得更好？
4.  我能从你看我的方式看出我对你来说很特别吗？

任何曾经和一个慢慢离开的人有过浪漫关系的人都会明白她问这些问题的意思:我曾经和一些人交往过，当浪漫接近尾声时，他们似乎不在乎我是否在房间里，或者不在乎我们是否一起参加聚会，或者只和我说话说些批评的话。我对我慢慢疏远的合作伙伴也做过同样的事情。

Schafler 注意到，她诊所的病人进来抱怨说，他们走进卧室，他们的伴侣几乎没有抬头看电话。或者他们让他们的孩子穿好衣服准备上学，却没有真正地认识他们或进行眼神交流。花点时间让别人知道你看到了他们，你在乎他们在那里，他们就足够了，你认为他们很特别，这是让一段好关系变好的本质。

我有一个朋友，她是让我自我感觉良好的大师——她问我问题，似乎对答案很感兴趣，并且把我当成一个特殊的人，作为朋友，我可以给她独特的东西。我一直认为她有着异常出色的社交能力，但现在我意识到，她总是(无意识地)回答我(无意识地)问的这四个问题。

Schafler 并不是第一个注意到这种要求是稳固关系的基础的心理学家。心理治疗师约翰·戈特曼(John Gottman)的研究可以用来预测哪些夫妇会继续在一起，哪些会离婚，他将这些请求称为*投标*:[一方对另一方的任何关注、肯定、喜爱或任何其他积极联系的尝试](https://www.gottman.com/blog/turn-toward-instead-of-away/) 根据他的研究，并不是所有的出价都会得到回应，但重要的是，你在 85%的情况下会肯定地回应。(有 33%的夫妇可能会离婚。)

这适用于所有类型的关系，即使是短暂的关系。不管出于什么原因，你们中的一个人出价了，而另一个人在一两分钟内全神贯注。我喜欢这个策略，因为它帮助你考虑你给予了什么，而不是你得到了什么，并帮助你控制改善你的关系。它可能不会在一秒钟内改变你的生活，但它可能会让在杂货店排队等候的过程变得更有趣。
