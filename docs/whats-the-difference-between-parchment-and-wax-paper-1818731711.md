# 羊皮纸和蜡纸有什么区别？

> 原文：<https://lifehacker.com/whats-the-difference-between-parchment-and-wax-paper-1818731711>

众所周知， [我是羊皮纸](https://lifehacker.com/sunday-showdown-parchment-paper-vs-silicone-baking-ma-1797765791) 的忠实粉丝，但它的不太贵，但仍然不粘的朋友蜡纸呢？为什么会存在？你*需要*它吗？



*这是* [***成人厨房***](http://lifehacker.com/tag/the-grown-up-kitchen#_ga=2.262024973.1420464300.1505696546-949419976.1446553382) *的一部分，Skillet 系列旨在回答您最基本的烹饪问题，并填补您的家庭厨师教育中可能缺失的任何空白。*

虽然这两种纸都可以用来包装食物，将烘焙食品分层储存，并为糖果提供不粘表面，但这两者之间有一个很大的非常重要的区别。你看，蜡纸是易燃的，这意味着你不能，不应该，也不会把它放进烤箱。

但不能因为它受不了热，就说它应该滚出厨房。以下是几种使用涂蜡纤维素的方法:

*   在着手处理杂乱的项目之前，把它放在你的柜台和桌面上，以便于清理。
*   防止冷冻食品( [如培根](https://lifehacker.com/freeze-bacon-slices-individually-with-wax-paper-to-avoi-1578895104) )粘在一起。
*   [用它们做临时的 Swiffer sheets](https://lifehacker.com/use-wax-paper-as-a-replacement-for-swiffer-sheets-5843050#_ga=2.162427196.141285593.1506347835-392757988.1502389723) 。
*   清洁并润滑你的开罐器 。
*   [包一个三明治](https://lifehacker.com/the-best-way-to-wrap-a-sandwich-for-on-the-go-eating-1770367702) 。
*   让糖果和点心在上面凝固。(我知道我已经提到过了，但是糖果制作季节就要到了。)
*   将它折成圆锥形 [做成滚边袋](https://www.youtube.com/watch?time_continue=4&v=XRkwFJv0olY) 。

基本上，只要不涉及热源，你就是好的。你能用羊皮纸做这些吗？是的，但是蜡纸更便宜，有一些蜡纸可以让你把更贵的羊皮纸留着烤精致的马卡龙或者做美味的饭菜[*en papillote*](https://lifehacker.com/get-excited-about-parchment-paper-one-of-the-most-usef-1738145434)*，*这很好，因为用完羊皮纸是一件可怕的事情。
