# 三种成分的快乐时光:令人愉快的苦味西纳汽水

祝(长)周末愉快，欢迎回到 [*三种配料的欢乐时光*](http://lifehacker.com/tag/3-ingredient-happy-hour) ，每周饮品专栏，提供超级简单又美味的酒水。正如孩子们所说，本周我坚定地回到了我的废话上，喝一种起泡的、苦涩的、以意大利利口酒为燃料的饮料:西纳汽水。

如果你像我一样，是标志性的金巴利葡萄酒的粉丝，但正在寻求拓展业务，那么辛纳是你应该购买的下一瓶葡萄酒。你可能会注意到标签上有一个朝鲜蓟，那是因为这瓶酒是用朝鲜蓟制成的。但是，这种酒的味道不像蒸蔬菜，而是有一种木头的味道。结果是有点苦，但愉快的草本液体，可以像使用金巴利一样使用。

对我来说，这是一款完美的夏秋过渡利口酒，因为它具有金巴利酒的所有苦味，但它的味道和外观更像秋天。我把它保持得很简单，只加一点点柠檬和苏打水。要制作这个精灵，你需要:

*   2 盎司辛纳
*   1 盎司新鲜柠檬汁
*   2 盎司苏打水

拿一个低球，扔一个大冰块进去。加入洋葱和柠檬，搅拌，加入苏打。啜饮并享受。
