# 如果它没有副作用，它很可能不起作用

> 原文:[https://life hacker . com/if-it-has-no-side-effects-it-possible-t-work-1818771976](https://lifehacker.com/if-it-has-no-side-effects-it-probably-doesn-t-work-1818771976)

如果一种药物或补充剂或治疗方法真的有效，它既有好处也有风险。例如，这就是为什么 FDA 批准的药物会在说明书中列出它们的副作用。但是粗略的健康治疗很少有这样的事情。

根据经验，声称有好处而没有任何负面影响或风险的东西是:

*   说谎的
*   或者它什么也不做。

大多数顺势疗法药物，包括 [几乎所有针对幼儿的感冒“药”](https://lifehacker.com/don-t-waste-your-money-on-homeopathic-drugs-1773105997)[都只是普通的老水](https://vitals.lifehacker.com/what-homeopathy-is-and-why-its-useless-1795694290) 。他们的支持者很高兴地解释说，顺势疗法没有副作用，所以它们是完全安全的。

但实际上，一些顺势疗法产品被证明含有大量的活性成分 ，足以杀死婴儿。增肌补充剂也有类似的悖论:许多补充剂什么也不做，但少数含有未公开的兴奋剂或激素或促进新陈代谢的药物。

在谷歌峰会 的 [上，一名销售人员提出了一个关于保健品益处的理论。(Goop 代言的产品是](https://lifehacker.com/i-went-on-a-quest-for-legit-health-tips-at-gwyneth-s-go-1796136053) [被过度宣传并且经常无用或有害的](https://www.truthinadvertising.org/tina-takes-goop-claims-to-regulators/) )。)我问她副作用的问题，她用手指在类似 [【性尘】](https://moonjuice.com/products/sex-dust?utm_medium=cpc&utm_source=googlepla&variant=42949164803) 和 [太阳药水](https://www.sunpotion.com/products/organic-reishi-mushroom-powder) 的产品的成分和功效清单上找了找，告诉我它们的功效。她解释道:你的身体会带走它需要的东西，然后留下其他的。如果有些东西不适用于你，你就不会得到那种好处。她拒绝了补充剂可能有害的想法。这不符合 Goop 的哲学，即尝试一切，看看什么可能对你有用。

另一方面，每次我的孩子在体检时接种疫苗，我都会收到一份带回家的信息表，上面详细描述了所有可能发生的可怕但非常罕见的事情。即使没有床单，我也知道打针会让我大一点的孩子手臂酸痛，可能会让孩子在接下来的一天里变得暴躁甚至发烧。我给他们打针是因为好处大于风险。我不只是把我发现的所有东西注入我孩子的血液里，然后看看会发生什么。

当然，真正的药物和治疗不一定是危险的。有时副作用很小，比如手臂酸痛。其他时候，有一种风险是罕见的，或者只影响患有某些其他疾病的人。比如像 [泰诺](https://www.drugs.com/acetaminophen.html)[布洛芬](https://www.drugs.com/ibuprofen.html) 这样的非处方药，风险较小。我们仍然服用它们。但是你可以在包装插页上读到关于它们的所有内容。

尽管这些药物有副作用，但大多数时候对大多数人来说还是值得的。只有少数真正的治疗方法没有副作用:也许是维生素 C，只要你服用正常剂量。(大量服用会引起 [恶心，更有](http://www.mayoclinic.org/drugs-supplements/vitamin-c/safety/hrb-20060322) 。)如果有人试图卖给你零风险、零副作用的东西，他们要么是在撒谎，要么是在卖没用的东西。
