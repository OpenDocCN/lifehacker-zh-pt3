# 问专家:普通人的健康问题

> 原文：<https://lifehacker.com/ask-an-expert-all-about-fitness-for-the-average-person-1754942165>

向史蒂夫·卡姆布问好，他自 2009 年以来一直在帮助普通人和普通人在 [书呆子健身](http://www.nerdfitness.com/) 过上更健康的生活。书呆子健身社区致力于指导我们这些不是真正运动大师的人“提升我们的生活水平”

Watch

对一些人来说，这意味着将你计划目标的方式游戏化，而对另一些人来说，即使你不认为自己是一个“健康”的人，这也只是一个开始的问题。史蒂夫有一本新书，书名很恰当，叫做 [提升你的生活](https://levelupyourlife.nerdfitness.com/) ，书中他阐述了如何过上更健康的生活方式的框架。想问史蒂夫一个关于美国桌面职业玩家和游戏玩家如何开始锻炼的问题？下一个小时他会在这里，所以问吧！
