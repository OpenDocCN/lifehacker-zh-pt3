# 唤醒模式说服我的孩子去睡觉

> 原文：<https://lifehacker.com/wake-mode-convinced-my-child-to-go-to-bed-1793341957>

iOS/Android:如果昨晚没睡好，今天会很累。对吧。 [唤醒模式](http://wakemode.com/) 显示你的疲劳程度，作为你身体的电池电量。对你我来说，这是个可爱的花招。对我七岁的孩子来说，这是一个让*真的去睡觉*的令人信服的理由。

Watch

毕竟，他知道当他看 YouTube 并收到低电量警告时会出现的恐慌。你必须马上插上电源，否则！所以当他看到自己身体的能量水平处于红色时，他突然对上床睡觉产生了浓厚的兴趣。

昨晚睡觉前我帮他安装了唤醒模式。该应用程序会询问你前一天晚上几点睡觉，你实际上花了多长时间入睡，以及你什么时候醒来。从这些数字中，可以很好地推测出你睡眠不足的程度。

我和我的孩子都在 10%左右。我们为他的手机打开了手动睡眠追踪功能(唤醒模式也可以从可穿戴的健身追踪器中读取数据，如果你有一个的话)，他就去睡觉了。我晚睡了一会儿，然后凌晨 2 点左右宝宝把我吵醒了。早上我还睡眼惺忪，这时我的孩子跑了进来，喊着“看，妈妈！我百分之百确定！”

唤醒模式鼓励你在白天通过咖啡因或小睡来“提升你的能量”;一杯健怡可乐给了我额外的百分之二。晚上，该应用程序会询问你一天过得如何，以更好地校准你未来的能量水平。它还有一个很好的仪表板，可以查看你的睡眠跟踪数据，如果你喜欢监控这类事情的话。像唤醒模式这样的应用程序并不能真正告诉你任何你的身体还不知道的事情。毕竟不睡觉会累。但是它可以方便地看到电池电量的下降——甚至可能是你最终需要关掉手机和 [进入睡眠](http://lifehacker.com/get-better-sleep-by-sticking-to-your-bedtime-1640793775) 的轻微推动。
