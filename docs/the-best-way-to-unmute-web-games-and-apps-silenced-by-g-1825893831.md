# 解除被谷歌浏览器最新更新屏蔽的网页游戏和应用的最佳方式

> 原文：<https://lifehacker.com/the-best-way-to-unmute-web-games-and-apps-silenced-by-g-1825893831>

谷歌对 Chrome 的最新更新对浏览器如何处理自动播放视频进行了重大改变，但它现在导致了各种网络应用和游戏的大量音频问题。

Watch

对于大多数人来说，谷歌的改变是很好的修复——视频仍然可以播放，但 Chrome 默认情况下会静音。随着 [Chrome 学习你的习惯](https://lifehacker.com/mute-most-autoplay-videos-with-the-latest-chrome-update-1825354497#_ga=2.6151792.2130151020.1525698930-3846207152.1521480874) ，当你访问你积极参与的网站时，它让视频自动播放声音，这个问题 [如果你讨厌被噪音惊到，你仍然可以手动解决](https://lifehacker.com/how-to-stop-google-chromes-new-update-from-autoplaying-1825741782#_ga=2.6151792.2130151020.1525698930-3846207152.1521480874) 。

然而，Chrome 的新“静音”功能也通过完全静音来破坏一些网页游戏和应用程序。在开发者调整他们的应用程序(或者谷歌调整 Chrome 的自动静音功能)之前，你将不得不使用一些变通方法来恢复你的声音。

### 如何恢复某些网页游戏和应用程序的声音

在我们开始解决之前，这里有一个 Chrome 问题的快速例子。在最新版本的 Chrome 中调出 [这个游戏](https://ponywolf.itch.io/darkestcastle)——这是一个有趣的游戏，相信我。在其他浏览器中，游戏听起来是这样的:

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-n3p_mtWUQGg&start=1](https://lifehacker.com/embed/inset/iframe?id=youtube-video-n3p_mtWUQGg&start=1) 

然而，在 Chrome 中，你会注意到最黑暗的城堡 [听起来像沉默](https://youtu.be/fP8nfFGtLcY?t=49s) 。Chrome 中没有简单的设置可以用来启动游戏的音频，甚至不能通过在浏览器中点击网站地址左侧的锁图标来启动。如果你想享受游戏的完整体验，你必须换一个浏览器——或者尝试一两个聪明的变通方法。

### 选择一:改变 Chrome 的自动播放政策

正如 Reddit 用户 [dirtynj](https://www.reddit.com/r/chrome/comments/8hunsl/google_chromes_autoplay_blocker_has_an_unintended/dyny42q/) 指出的，触发音频在你最喜欢的(静音)网页游戏和应用上播放的一种方式是调整 Chrome 在实验性功能中的自动播放策略。在你的 Chrome 浏览器中调出这个地址:`chrome://flags/#autoplay-policy`

点击“自动播放策略”的下拉菜单，将其更改为“无需用户手势”使用右下角的按钮重新启动你的浏览器，神奇的是，你的 should 应该会重新出现在你经常使用的大多数(如果不是全部)网络应用和游戏中。

### 选项二:安装一个变通扩展

如果实验功能选项消失或停止工作，您也可以尝试使用一个简单的扩展来启用音频，您必须手动安装。

前往 Github 页面查看取消静音。一旦你下载了扩展的内容(绿色按钮)并按照指示安装它——这很容易——你会看到一个新的披萨图标出现在 Chrome 的 omnibar 旁边。(别问了。)

现在，当你访问 Chrome 已经静音的网页游戏或应用程序时，披萨图标上可能会有一个数字。如果是这样，这对你来说是个好消息。点击图标，网站的声音就会神奇地重现。在某些情况下，你甚至可以在 Chrome 浏览器的右下角看到一个弹出窗口，你也可以点击它来启用 web 应用程序或游戏的音频。

### 选择三:给你的应用或游戏的开发者(或主机)安装窃听器

在某些情况下，托管你最喜欢的网络应用或游戏的网站可能会推出一个通用的解决方案，允许声音在 Chrome 中播放——不需要浏览器调整或扩展。例如，在 [itch.io](https://itch.io/) ，你现在可以启用一个新的“点击播放”选项，恢复网站上每个游戏的音频。

要切换该功能，请点击 itch.io 网站右上角用户名旁边的箭头，调出你的账户设置。在那里，向下滚动，直到你看到“需要点击运行 HTML5 游戏嵌入”的选项。启用它，保存您的设置，您将在整个站点范围内恢复您的音频功能。

其他开发人员，如 Soundslice 的工作人员——一种将音频(或视频)与歌曲的实际乐谱 同步的有趣方式——也一直在幕后工作，为 Chrome 的自动静音怪癖部署解决方案。然而，对于一些开发者来说，回到旧的应用程序或游戏中去破解他们的游戏应用程序是一件令人头疼的事情
