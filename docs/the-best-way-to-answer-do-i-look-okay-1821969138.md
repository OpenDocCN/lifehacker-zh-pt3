# 回答“我看起来还好吗？”的最佳方式

> 原文：<https://lifehacker.com/the-best-way-to-answer-do-i-look-okay-1821969138>

当你正准备在镇上度过一个重要的夜晚时，你的同伴转向你，问:“我看起来还好吗？”*大口*。你会说“是的！”不看吗？你告诉他们他们总是看起来非常非常好吗？还是给他们一些诚实的反馈？ [我们问你你会说什么](https://lifehacker.com/what-would-actually-be-helpful-to-hear-when-you-ask-do-1820551988#_ga=2.262720911.2003097520.1515359740-819016074.1513967708) ，这是你告诉我们的。



我应该指出，有一个普遍的共识，当人们问这样一个模糊的问题时，他们通常只是想确认他们看起来不像某种怪物，并且你愿意被看到和他们在一起。很公平！但是你们中的一些人分享了一些很棒的建议，我想每个人都应该知道。对于新手来说，总是用一些好的东西来打开话题，然后问一个你自己的问题来帮助他们变得更具体(以防他们有什么担心和害怕直说的事情)。

[评论者**rhyader**推荐](https://lifehacker.com/1820557707#_ga=2.119817928.1164671194.1515682973-392757988.1502389723) 说些类似于，“*耶，你看起来棒极了！为什么？有什么感觉不舒服吗？他们的回答将有助于确定他们真正感到难为情的是什么，让你更容易给出一个有帮助的答案，或者平息他们的恐惧。同理， [**PieBrain** 建议](https://lifehacker.com/1820557308) 你设身处地为他们着想。假设他们对自己的总体风格很满意，只是简单地问了一下他们服装中有什么不正常的地方。*

如果你确实想说实话，但是你担心你的回答会冒犯他们，那么 [评论者**沙拉**建议](https://lifehacker.com/1820565303#_ga=2.32172221.2003097520.1515359740-819016074.1513967708) 你默认一句增强自信的评论——比如“你看起来很棒！”——接着是艺术学校式的评论。首先，想想他们正在参加的活动。然后，说一些你喜欢他们服装的地方，接着用类似“我喜欢你穿的那件背心，但是某某人可能更适合(活动)”这样的话来进行引导这样，他们知道这不是关于他们的个人外表或他们的时尚感，而是关于什么对正在讨论的事件是最好的。

只要记住，时机就是一切， [表示](https://lifehacker.com/1820559300#_ga=2.97642908.2003097520.1515359740-819016074.1513967708)**和 [**Lymis**](https://lifehacker.com/1820557641) 。如果他们不能当场改变他们的服装或外表——他们没有别的衣服可穿，他们不在家，等等。—只是给他们一个信心助推器。在这种情况下，这才是他们真正想要的，也是你所能提供的。如果他们不能解决问题，没有必要用直率诚实的方式把他们拖下水，尤其是如果事情发生在他们参加活动之前。**
