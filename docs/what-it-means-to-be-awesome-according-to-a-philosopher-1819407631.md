# 根据一位哲学家的说法，令人敬畏意味着什么

> 原文：<https://lifehacker.com/what-it-means-to-be-awesome-according-to-a-philosopher-1819407631>

在你生命中的某个时刻，你可能说过某人要么很棒，要么很烂。我知道我有。但是从哲学的角度来看，这意味着什么呢？一位哲学家认为这完全是关于社交场合中的思想开放。

Watch

一个人如何把主观的东西定义为很棒或者很烂？尼克·里格尔(Nick Riggle)是一名前职业滑冰运动员，拥有 NYU 大学的哲学博士学位，他认为他在自己的新书《变得棒极了:如何不吸 的统一理论》中找到了答案。他向《科学美国人》的加雷斯·库克解释他的理论:

> ...“棒极了”通常在一种独特的社会意义上被用来谈论以某种方式将人们聚集在一起的人和行为...令人敬畏是善于创造“社交机会”——当人们打破常规和惯例，通过表达自己的个性，让别人表达自己的个性时，人们之间相互欣赏的时刻。

基本上，当你自在地做自己，并且让别人也自在地做自己时，你就很棒。这就是让人们成为他们想成为的人，而不是在社交场合小题大做。有道理，但是“吸”是什么意思:

> 当有人无缘无故拒绝一次社交聚会时，他很糟糕。

里格尔指出，吮吸等同于吹牛、吹牛，或者最好的说法是扫兴。本质上，吮吸是认为你在学校里太酷了。你关闭了自己的个性世界，同时也关闭了他人的个性表达。

里格尔举了一个职业篮球比赛的例子，播放一首歌，人们出现在大屏幕上。当轮到你出现在屏幕上时，表现得很棒就是参与并跟着唱——甚至可能是围绕着 [跳舞，或者重演音乐视频](https://www.youtube.com/watch?v=IqCNP65Bt4s) 。吮吸是坐在那里拒绝参与，因为你“不认为这很酷”或者你不想看起来很傻。所以大家一起唱。你可以选择牛逼。
