# 三种配料的欢乐时光:冷的热葡萄酒

> 原文：<https://lifehacker.com/3-ingredient-happy-hour-chilly-mulled-port-1820903766>

各位周末快乐，欢迎回到 [*三料欢乐时光*](http://lifehacker.com/tag/3-ingredient-happy-hour) ，每周饮品专栏，提供超级简单又美味的酒水。本周，我们将一款经典的冬季保暖产品——热葡萄酒进行冷加工。

“加热”仅仅意味着饮料被加糖、调味和加热，我们将简化这一过程。我们将从甜红葡萄酒开始，而不是甜干葡萄酒。我今天也在反复无常地定义“成分”;我没有把每一根肉桂棒和香料豆荚单独算作一种成分，而是拿了一包混合香料，把混合香料算作一种成分。

你可以喝温的或冷的饮料，但我喜欢冰镇的酒。为了让这种舒适的寒冷天气成为经典，你需要:

*   两杯红宝石葡萄酒
*   1 个大橘子(或一些橘子附近的柑橘类水果)
*   1 汤匙混合香料，加上一包肉桂棒

将波特酒倒入一个小平底锅中，放在炉子上。用削皮器将橘子去皮，在将橘子放入锅中之前，将每条橘子皮上的油榨出来。将橘子榨汁并倒入，然后加入混合香料。将所有东西煮沸，转小火慢炖，让它冒泡五分钟，然后从火上移开，盖上盖子，让它达到室温，然后将整锅放入冰箱冷藏。(如果你想享受温热的波尔图葡萄酒，只需在倒入马克杯前再炖十分钟。)一旦所有东西都冷却下来，就可以放入碎冰或一个大冰块中食用。
