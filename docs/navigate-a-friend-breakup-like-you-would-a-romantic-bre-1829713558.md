# 像对待浪漫分手一样对待朋友分手

> 原文:[https://life hacker . com/navigate-a-friend-break-like-you-want-a-romantic-bre-1829713558](https://lifehacker.com/navigate-a-friend-breakup-like-you-would-a-romantic-bre-1829713558)

像浪漫的关系一样，友谊也会结束。我们倾向于把失去一个朋友仅仅看作是一个短暂的瞬间，但是这种分裂通常也同样令人痛苦。让我们开始给予分手友谊应有的尊重。

在某些方面，这比失去一个浪漫的伴侣更难接受，因为我们几乎没有关于如何对待友谊破裂的文化例子。卡莉·布雷特 [在*时间*就此话题采访了](http://time.com/5402304/friendship-breakups-worse-romantic/) 一位名叫米丽娅姆·克尔迈尔的临床治疗师和一位心理治疗师玛妮·弗曼。

Kirmayer 专门研究年轻人和成年人的友谊，并认为友谊的结束如此艰难的部分原因是我们“期望友谊对成年人来说应该是容易的。”他们不是！但是有很多很好的理由，你可以做一些事情来减轻痛苦——甚至可能在文化层面上改变我们对待友谊的方式。

### 进行“谈话”

如果你感觉到一个朋友正在离开，你可能需要和一个浪漫的伴侣进行一次类似分手的谈话。“我们倾向于认为友谊的结束是因为一些巨大的背叛,”Kirmayer 说,“但事实上，你们可能只是作为人越来越疏远了。”。这使得很难确定什么时候该讨论你们的关系状态，或者你是否需要讨论。有时候，你只是让它死去。还是很糟糕。

Kirmayer 说:“这可能会导致我们最终感觉受到伤害。”。“不管是处理不当，还是仅仅因为意外，我们真的不知道(友谊破裂)应该是什么样子。”

与你疏远的人发生这种冲突似乎不值得，但这种对话提供了浪漫分手所能提供的重要东西:结束。如果你讨论正在发生的事情，即使很尴尬，你可能会发现它远没有那么令人困惑和开放。

例如，你可以指出你们之间越来越大的距离，说你觉得你需要暂停制定计划，或者甚至说你的关系已经变得不健康，如果是这样的话。这可能很难，但至少你们都知道自己的立场。

### 为未来设定参数

分手谈话有时也包括决定你们将来如何相处。你们可能再也见不到对方了，但让我们面对现实吧。世界真小。更好的办法是知道如果你在外面遇到其他人，你会如何互动。

“关于友谊破裂意味着什么，人们有很多困惑，”弗曼解释道。“你还打算在特定的语境下交流吗？如果你们有共同的朋友，你们愿意在集体环境中见面吗？”

> 相反，一个觉得自己的需求没有得到满足的朋友可能会保持沉默。据 Feuerman 说，那个人可能会意识到友谊没有结果，更倾向于让它自然结束。这种缺乏沟通会伤害到其他朋友，就像他们不知道自己做错了什么一样。

与其和某人签订友谊合同，不如试着提醒自己，如果有人让你不快或冒犯了你，你需要尊重地沟通——当他们抱怨你的行为时，倾听他们。这可能足以从一开始就阻止未来的分手。

### 放下羞耻

因为我们不怎么谈论这些事情，人们在友谊结束时会感到羞愧和孤立。这意味着你们中的一个是坏人，或者做错了什么，对吗？不。有时友谊就是不起作用，这并不能反映出任何有关当事人的情况。他们只是不适合对方，这没关系。如果你设法放下那种感觉，那么就公开谈论它。对你身边的每个人都有帮助。

“人们觉得他们应该明白这一点，并认为其他人都明白这一点，”Kirmayer 说。"他们觉得经历友谊破裂是做错了什么."

当浪漫关系结束时，我们并不感到惊讶。不应该有压力去假设所有的朋友都是最好的朋友。

### 感受你的悲伤

当我们不谈论朋友分手时，我们不会因为与我们生命中有意义的人分离而感到悲伤。即使最终是一段糟糕的关系，那个人对你来说也很重要，失去他们会很痛苦。

“你实际上会经历一点悲伤的过程，这没关系，”弗曼说。“如果你觉得你无法改变有毒的友谊状况，哀悼它没关系，继续前进，找到更令人满意的关系。”

你不必假装你已经释怀了，或者其他朋友会马上来代替他们。诚实地说出你的感受并处理它。想念某人是很正常的，即使他们在你的生活中已经没有位置了。
