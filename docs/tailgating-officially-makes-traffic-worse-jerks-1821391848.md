# 尾随官方使交通更糟，混蛋

> 原文：<https://lifehacker.com/tailgating-officially-makes-traffic-worse-jerks-1821391848>

如果你曾经遇到过“幽灵般的交通堵塞”，或者没有明显原因的交通减速，很可能是一些货车尾随某人。麻省理工学院的研究人员发现，尾随不仅不能让你更快到达任何地方，它实际上造成了本不应该存在的交通堵塞。



尾随 ，或者说跟在某人后面开得如此之近，以至于你可以在他们后窗的泥土上写下“给我洗洗”，这种行为一直都是粗鲁的(更不用说不安全了)。在最好的情况下，你前面的车会因为害怕被公路暴怒谋杀而加速一点，但更有可能的是，你看起来就像一个不耐烦的混蛋。麻省理工学院教授贝特霍尔德·霍恩和博士后王曦梁 说 [，更糟糕的是，你给自己和车道上的其他司机带来了不便。为了让交通保持顺畅，霍恩说，汽车需要在道路上均匀分布:](https://www.csail.mit.edu/news/improving-traffic-tailgating-less)

> “我们的研究表明，如果司机都保持他们两边(前后)的车之间的距离相等，这种‘干扰’会在他们行驶在一条交通线上时消失，而不是扩大到造成交通堵塞……”

基本上，当你追尾时，如果前面的车减速，你就失去了逐渐减速的能力。因为你没有足够的时间做出反应，避免追尾你前面的车，你用力踩刹车，瞬间带走了你车辆的所有动力。当你再次加速时，你后面的所有车也必须迅速减速。它会引起连锁反应，最终导致交通拥堵，或者引发事故，而没有什么好的理由。

当然，Horn 和 Wang 非常清楚，人们不会停止尾随或更加关注自己与其他车辆之间的空间，所以他们希望未来的车辆增加一些功能可以解决这个问题。通过在自适应巡航控制系统中实施他们所谓的“双边控制”,虚幻的交通堵塞可能会成为过去。他们估计司机甚至可以以两倍的速度到达某个地方。在那之前，尽可能在你和前面的车之间留有足够的空间。这对路上的每个人都有好处。
