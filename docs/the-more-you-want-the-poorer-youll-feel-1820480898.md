# 你想要的越多，你会觉得越穷

本周的精选来自罗马哲学家 [小塞内加](https://en.wikipedia.org/wiki/Seneca_the_Younger) 。他认为贫穷是一种精神状态。一个你可以控制的:

<section class="q35npn-0 MZtuu">![](../Images/aef335eb562a1da6957ce25a574b4aa1.png)

贫穷的不是拥有太少的人，而是渴望更多的人。

信二:关于阅读中的话语，第 6 行。

</section>

### 这意味着什么

基本上，“穷”不是你拥有什么，而是你想要什么，尽管你已经拥有了什么。你想要的越多，你会觉得越穷，不管你已经拥有什么。塞内卡的这段话实际上是对伊壁鸠鲁的一句话的重复，他说“知足的贫穷是一种光荣的财产。”塞内加解释说，“的确，如果它是满足的，它根本不是贫穷。”他接着说，如果你一直在诅咒你没有的东西，那么你拥有多少并不重要，尤其是因为这让你对未来的收益不抱希望。

### 从中吸取什么

我们不要混淆塞内卡在这里所说的。他并不是说贫穷不是真实存在的。自从有了文明以来，人们一直生活在真正的贫困之中，极度需要某种东西——无论是食物、水还是住所。塞内卡在这里谈论的是贫困的一般概念，即把我们自己定义为“穷人”他说是想要的行为，贪婪，让你处于那种精神状态。你想要的越多，你就越觉得自己缺少了什么，你自己也就越贫穷。

我喜欢把生活想象成一个水杯，把水想象成你生活中想要和需要的东西。幸福和满足——塞内卡称之为“足够”——是当你的杯子装满水的时候。然而，你想要的越多，你的杯子就必须越大，这意味着你需要更多的水来装满你的杯子并获得快乐。此外，你会一辈子痛恨自己的杯子没满，而你本可以花精力去想办法把它装满。另一方面，如果你有一个小玻璃杯，只需要很少的水就可以装满它，这就更容易达到满足的状态。

至少，重新评估你的需求。现实地削减它们，区分它们的优先次序，定义你认为的“足够了”，那些你真正需要快乐的事情。停止沉湎于生活中的命运，专注于*如何*获得这些东西。更好的是，不要再感到如此贫穷，不要再想要太多。毕竟，如果你什么都没有，也什么都不想要，你就拥有了你所需要的一切。
