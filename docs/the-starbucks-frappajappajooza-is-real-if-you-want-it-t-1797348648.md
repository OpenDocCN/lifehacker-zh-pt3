# 如果你想的话，星巴克是真实的

> 原文:[https://life hacker . com/the-Starbucks-frappajappajooza-is-real-if-you-want-t-1797348648](https://lifehacker.com/the-starbucks-frappajappajooza-is-real-if-you-want-it-t-1797348648)

没有一种秘密的星巴克饮料叫做。这只是推特用户 wellmanlaylah 把蓝色佳得乐放进杯子里。但是从另一个角度来说，这里的*是*一种 Frappajappajooza，它由你想要的任何东西制成，只要你选择现有的星巴克原料。

Watch [https://lifehacker.com/embed/inset/iframe?id=twitter-890244409165565955&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-890244409165565955&autosize=1) 

秘密菜单就像你做的一样真实。像星巴克和麦当劳这样的连锁店的员工通常非常乐意为你做一件特别的东西 ，如果你礼貌地询问，解释这是什么，并等待他们与经理计算价格。而 In-N-Out 早在 [官方就承认了其客户自创的秘制菜单](http://www.in-n-out.com/menu/not-so-secret-menu.aspx) 。

请记住，大多数时候，员工没有接受过制作菜单外食谱的培训。所以，不要走进墨西哥卷饼店，点了 [墨西哥卷饼](http://hackthemenu.com/chipotle/secret-menu/burritodilla/) 却不解释那到底是什么。也不要在高峰时间点精心制作的特餐，让队伍慢下来。但是，嘿，官方菜单上的星冰乐是 [已经让咖啡师](http://nymag.com/thejob/2017/04/the-starbucks-barista-who-says-the-secret-menu-is-fake.html) 头疼了，所以你还不如要一杯紫色糖浆。

小说家托尼·图拉西穆特说，当他在麦当劳工作时，“绝对有一份秘密菜单”，因为顾客经常点同样的菜单外的菜，但他没有接受过这些特殊食谱的培训。一位在星巴克工作的朋友对此表示赞同，“我完全满足于任何人点任何他们想要的奇怪的垃圾自制饮料。”

所以全力以赴。从里到外点你的 Panera 三明治，吃你的超值套餐，发明你自己版本的 Frappajappajooza。哦，一定要多给小费。
