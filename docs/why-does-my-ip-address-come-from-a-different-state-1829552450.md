# 为什么我的 IP 地址来自不同的州？

> 原文:[https://life hacker . com/why-my-IP-address-come-from-a-different-state-1829552450](https://lifehacker.com/why-does-my-ip-address-come-from-a-different-state-1829552450)

人际关系网可能有点令人困惑，尤其是当你做了(或没做)的事情没有任何逻辑意义的时候。您安装了新的电缆调制解调器和无线路由器，但速度比使用旧设备时更慢。你用千兆位以太网连接你的房子，却无缘无故地看着你的连接在千兆位和快速以太网之间切换。

奇怪的网络问题还在继续。这是我们本周从《生活黑客》读者那里收到的一个简单的问题:

> “我的公开 IP 地址显示的是一个完全不同的城市和州。这是为什么呢？IP 地址也会改变。请指教。我能联系谁？”

首先，我不会担心。真的。比起我的外部 IP 地址(你可以在这里查找)，我更担心我的 ISP 的速度。我现在就这么做了，网站告诉我 IP 地址来自加州的一个城市，离我住的地方很近(大约 20-30 分钟的路程)，但不是我住的地方。它不会影响我和我的室友以正常速度连接到互联网的能力。一切都好。

至于你的 IP 地址来自完全不同的州，那也是完全正常的。正如 WhatismyIP.com 指出的:

*“使用我们的 IP 查找工具可以找到您的 IP 地址。由于许多不同的因素，没有 IP 查找工具是 100%准确的。这些因素包括知识产权所有者在哪里注册、控制知识产权的机构在哪里、代理、蜂窝 IP 等。如果你在美国，而 IP 的控制机构位于加拿大，那么 IP 地址查询结果很可能会显示为加拿大。在威瑞森网络的移动用户中，在美国北部显示加拿大的 IP 地址是很常见的。”*

如果一个网站或服务试图根据 IP 地址查找向你提供信息，比如说，你最喜欢的在线地图应用程序，这可能很烦人，但你必须处理这个小麻烦。我也能感觉到你。我曾经不得不为纽约市外的一家公司使用 VPN，默认情况下，每当我打开网站时，谷歌地图总是加载非加州的内容。我已经习惯了。

如果你的问题真的困扰你，你可以试着打电话给你的 ISP，看看他们是否有任何解决方案，但老实说，如果客服人员真的一开始就理解你的问题，我怀疑他们能做什么。你的 IP 地址可能与康卡斯特的调制解调器 MAC 地址绑定在一起，他们对此可能无能为力。

你也可以向你的 ISP 支付一个静态 IP 地址的特权，这可以更好地反映你实际居住的地方，但是我不会在这上面浪费钱，除非这个问题严重妨碍了你的生活。就像我说的，这是一个你必须习惯的烦恼。嘿，情况可能会更糟，正如我们在 Splinter 的朋友在 2016 年写的:

> 任何地理迷都知道，美国的精确中心在堪萨斯州北部，靠近内布拉斯加州边界。从技术上讲，中心点的经纬度坐标是北纬 39° 50 '西经 98° 35 '在数字地图上，这个数字很难看:39.8333333，-98.585522。所以回到 2002 年，当 MaxMind 第一次在它的数字地图上选择美国中心的默认点时，它决定清理测量结果，并采用一个更简单的附近经纬度:38 N 97 W 或 38.0000，-97.0000。
> 
> 因此，在过去的 14 年里，每当 MaxMind 的数据库被查询到它无法识别的美国 IP 地址的位置时，它都会吐出一个距离该国地理中心两小时路程的默认位置。这种情况经常发生:5000 家公司依赖 MaxMind 的 IP 映射信息，总的来说，现在有超过 6 亿个 IP 地址与该默认坐标相关联。如果这些 IP 地址中的任何一个被骗子、电脑窃贼或有自杀倾向的人使用，并联系了求助热线，MaxMind 的数据库会将它们放在同一个位置:38.0000，-97.0000。
> 
> 恰好在乔伊斯·泰勒家的前院。"
