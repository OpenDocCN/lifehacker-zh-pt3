# 这个扩展让你在谷歌日历的重新设计中隐藏浪费的时间

> 原文：<https://lifehacker.com/this-extension-brings-back-working-hours-to-google-cale-1827815781>

谷歌更新版的 [日历](http://calendar.google.com)——你再也不能选择退出——看起来很可爱，比以前的版本更容易操作。然而，日历的新“材料设计”版本遗漏了一个主要功能，似乎是 [惹恼了一些用户](https://productforums.google.com/forum/#!topic/calendar/o56fJ_OCR9Q) (真的包括你的用户):封锁你睡觉时间的能力。



在日历的最后一次重大迭代中，谷歌在其设置的实验室部分有一个“隐藏早上和晚上”功能(这也是日历的当前版本所缺少的)。如果您启用了它，那么您可以返回到您的主日历视图，拖动顶部和底部的一个小灰色条来压缩您的日历视图。

换句话说，你可以把一天中你熟睡的时间从日历中删掉，因为这只是浪费空间。(我怀疑很多朋友或同事会在凌晨 3 点给你发邀请，但你永远不会知道。)

不幸的是，在当前版本的谷歌日历中没有这个选项。如果你注册了谷歌的 G 套件，或者将其作为工作的一部分，你可以设置“工作时间”，这至少可以防止人们在你不打算工作的时候与你预约会议。然而，这仍然不会压缩你的日历，从午夜到你开始一天工作的任何时候，你的日历顶部仍然会有一大块空白。你没有太多的选择来解决这个问题，但你确实有一些。

### 用灰色标出你未使用的时间

这不会让你的日历上的一大块时间消失，但你至少可以创建一个新的日历，为你的典型睡眠时间设置一个定期的日常会议——比如， [午夜到早上七点](https://vitals.lifehacker.com/how-little-sleep-can-you-get-away-with-1788533845) 选择石墨作为日历的颜色，然后让会议每天重复。至少你不会再看到一大块空白空间，你的眼睛可以更好地专注于醒着的时候发生的彩色日历项目。

### 试试谷歌日历扩展

这个扩展可用于 [Chrome](https://chrome.google.com/webstore/detail/google-calendar-plus/mjelhipeelammmhpghkpigkdonihkakj) 和 [Firefox](https://addons.mozilla.org/en-GB/firefox/addon/gcalplus/) ，允许你设置日历开始的时间。从午夜开始调高小时计数器，日历上相应的行就会消失。理论上，该扩展还应该允许你自下而上地缩小日历——从午夜开始回拨，如果你睡得早一点，就隐藏晚一点的时间——但这显然在谷歌日历中要复杂得多，因为调整这一设置在该扩展中还没有任何作用。希望它的作者能尽快完成这项工作。

### 让谷歌知道你想要更多的日历定制

虽然很多人已经告诉谷歌，他们希望能够调整他们日历上显示的时间，但谷歌可能需要更多的反馈才能实现这一点(以及实验室支持！)到未来的日历更新中。让他们通过 [这个方便的反馈表](https://www.google.com/tools/feedback/intl/en/) 得到它——并在你的谷歌日历上设置提醒，这样你就不会忘记做这件事。
