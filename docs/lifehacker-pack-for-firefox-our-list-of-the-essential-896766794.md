# Firefox 的 Lifehacker Pack:我们的基本扩展列表

> 原文：<https://lifehacker.com/lifehacker-pack-for-firefox-our-list-of-the-essential-896766794>

Firefox 仍然是一个强大的浏览器，随着 Chrome 变得比以往任何时候都更重，许多人都把他们的老朋友 Firefox 作为替代选择。另外，它仍然拥有最好的、经过严格审查的扩展库。这里有一些必要的 Firefox 扩展，你可以根据自己的意愿来改变网络。

***life hacker Pack**是我们最喜爱的平台上我们最喜爱的必备应用程序的年度快照。这个列表是我们为 Firefox 做的第一个包。*

### 生产力

#### [Instapaper](http://www.instapaper.com/) ， [Pocket](http://getpocket.com/) ，或者 [可读性](http://www.readability.com/)

很有可能，你在一天之内会遇到如此多的链接、博客文章和新闻故事，以至于你当时不可能全部读完——尤其是当你试图完成工作的时候。进入书签和随后阅读服务: [Instapaper 和 Pocket](https://lifehacker.com/read-it-later-showdown-pocket-vs-instapaper-1756021479) 是你在这个领域的两大选择，可读性也是，它仍然有效，拥有忠实的追随者，但已经多年没有更新了。它们都将这些链接发送到各自的服务中，去掉广告和格式，给你一个易读的版本，以后你可以在任何设备上使用。Pocket [内置于 Firefox](https://support.mozilla.org/en-US/kb/save-web-pages-later-pocket-firefox) 中，我们确实喜欢 Pocket，所以你可能不需要另一个扩展来完成这项工作。

#### [添加到神童列表](https://www.wunderlist.com/)

Wunderlist 仍然是我们最喜欢的待办事项管理器 之一，Firefox 的扩展让它变得更好。只需点击一下，你就可以将任何网页内容直接保存到你的待办事项列表中，非常适合 [让这些待办事项列表变得详细可行](https://lifehacker.com/back-to-basics-how-to-simplify-your-to-do-list-and-mak-5954123) 。它快速、免费、灵活，即使去年 [Wunderlist 被微软](http://lifehacker.com/wunderlist-one-of-our-favorite-to-do-apps-just-announ-1708546711) 收购，它仍然照常营业，Firefox 扩展甚至在 2016 年 1 月进行了更新。根据记录，我个人最喜欢的是 [Todoist](https://todoist.com/) ，其中 [还有一个火狐扩展](https://todoist.com/firefox) ，如果你正在寻找一个替代品的话。

#### [拉扎勒斯](http://getlazarus.com)T3】

如果你安装任何扩展，安装这个。你有没有在浏览器窗口中写东西，却发现它崩溃了，丢失了你所有的工作？拉扎勒斯是你的救命恩人。当你在表单中输入时，Lazarus 会持续保存你正在做的事情。如果你的浏览器崩溃了，或者你不小心关闭了标签页，你只需要在那个页面上的空文本框中点击鼠标右键，就可以找回那个文本。你可能认为你现在不需要它，但当那决定性的一天到来时，你会很高兴你安装了它。

#### [Evernote Web Clipper](http://evernote.com/webclipper/)或[OneNote Web Clipper](https://www.onenote.com/clipper)

如果你是 Evernote 用户，还没有 Evernote Web Clipper，你一定要安装它。如果你正在使用 Evernote，但没有使用 Web Clipper，你就错过了 Evernote 最好的用途之一[。然而，如果你是 Evernote 用户](http://lifehacker.com/ive-been-using-evernote-all-wrong-heres-why-its-actual-5989980) [，对](http://lifehacker.com/how-to-jump-ship-from-evernote-and-take-your-data-with-1782841075) [最近的价格上涨和免费账户限制](http://lifehacker.com/evernote-limits-device-sharing-for-free-users-bumps-up-1782744350) 颇有微词，微软 OneNote [的功能很有竞争力，自带的](https://lifehacker.com/note-taking-showdown-evernote-vs-onenote-2016-editio-1765707423)web clipper 非常值得使用。

不管是哪种情况，Web Clipper 都可以让你将整篇文章、一页或一页的一部分保存到你的笔记中，这样你就不必浪费时间，而不只是在笔记本上草草记下一些东西。然后，您可以在这篇文章上记笔记，将它与项目的其他相关笔记一起保存，等等。

### 安全性

#### [LastPass](https://lastpass.com/) 或 [Dashlane](https://www.dashlane.com/)

至少每周都有新的黑客入侵。不幸的是， [你聪明的密码把戏不再起作用了](http://lifehacker.com/your-clever-password-tricks-arent-protecting-you-from-t-5937303) ， [唯一安全的密码是你不记得的](http://lifehacker.com/the-only-secure-password-is-the-one-you-can-t-remember-5785420)——这就是为什么你需要一个密码管理器来保持它们都是正确的。LastPass 是我们最喜欢的，但 Dashlane 是另一个优秀的，易于设置和 [功能相当的替代品](https://lifehacker.com/lifehacker-faceoff-the-best-password-managers-compare-1682443320) 。将你所有的密码放入密码管理器并不像听起来那么困难:它会为你创建安全的密码，让你自动登录，如果你需要自己手动输入密码，你可以非常容易地访问它们。

[LastPass](http://lifehacker.com/lastpass-can-now-automatically-change-your-passwords-1668860126)和 Dashlane 甚至有工具 [只需点击一下](http://lifehacker.com/dashlane-password-changer-can-change-hacked-passwords-w-1668174260) 就能为你更改泄露的密码——甚至不用你亲自去网站。查看 [我们的 LastPass](http://lifehacker.com/the-intermediate-guide-to-mastering-passwords-with-last-5645162) 指南开始使用，如果 LastPass 不是你的速度，查看 [替代方案](https://lifehacker.com/the-five-best-password-managers-5529133) 。当然， [LastPass 本身并不是无懈可击的](http://lifehacker.com/is-lastpass-secure-what-happens-if-it-gets-hacked-1555511389) ，( [远非如此，事实上](http://lifehacker.com/lastpass-hacked-time-to-change-your-master-password-1711463571) )而且它是 LogMeIn 最近收购的 [(并不是说这改变了什么)，但这比试图记住多个网站的多个密码要好得多，甚至更糟，为多个服务使用同一个密码。](http://lifehacker.com/logmein-makers-of-one-of-our-favorite-remote-managemen-1735630523)

#### [到处都是 HTTPS](https://www.eff.org/https-everywhere)

使用 HTTPS 是保护你的网络浏览 和防止你的凭证被盗的最简单的方法之一。 [HTTPS 无处不在很简单](http://lifehacker.com/https-everywhere-3-0-keeps-your-data-safe-on-1-500-more-5950311) :它强制你的浏览器在任何支持它的网站上使用 HTTPS，所以你永远不用考虑它(尽管如果 HTTPS 无处不在导致问题——这可能会不时发生——你可以将某些网站列入白名单)。HTTPS 无处不在已经存在了一段时间，但它一直在定期进行规则集和功能更新。你可以在 EFF 阅读更多关于它的 [。](https://www.eff.org/https-everywhere)

#### [子块原点](https://addons.mozilla.org/en-us/firefox/addon/ublock-origin/) 或 [**AdBlock 加**](https://adblockplus.org/)

听着，我们在 Lifehacker 靠广告谋生，但这并不意味着广告总是一件好事。有些网站有过分热情的广告，当你悬停在文本上时会妨碍你，而其他网站有彻头彻尾的欺骗广告，使你很难找到你实际上正在寻找的“立即下载”按钮。如果你厌倦了广告挡道，uBlock Origin(或 AdBlock Plus)是解决方案。我们更喜欢 uBlock Origin，因为它是 [更精简的替代方案，可以节省宝贵的内存](https://lifehacker.com/ublock-is-a-fast-and-lightweight-alternative-to-adblock-1625246461) 并仍然可以阻止广告(以及网络上大量其他脚本和其他恼人的、消耗内存的元素)，而且它可以使用已经存在更长时间的 AdBlock Plus 可以使用的所有列表和规则。最重要的是，它给了你更多的控制来白名单，过滤和动态过滤你认为合适的元素。

这两种方式都会屏蔽你遇到的任何页面上的广告(尽管我们希望你将你想要支持的网站列入白名单)。它还会 [保护你在网上的隐私](https://lifehacker.com/the-best-browser-extensions-that-protect-your-privacy-479408034) ，屏蔽你想隐藏的其他部分(如 [YouTube 评论](http://lifehacker.com/block-youtube-comments-with-adblock-plus-5830390) )，甚至 [屏蔽恶意软件](http://lifehacker.com/why-you-should-use-ad-block-extensions-even-if-you-don-5649025) ( [查看这篇文章](http://lifehacker.com/everything-you-can-do-with-adblock-plus-that-isn-t-jus-1469861264) 了解它能做的一切)。你也可以用 [NoScript](https://addons.mozilla.org/en-us/firefox/addon/noscript/) 阻止脚本在任何页面上运行——尽管我们通常发现这样做不值得，因为它会破坏你使用的许多网站。

#### [断开](http://disconnect.me/)

[每个人都试图在网上跟踪你](https://lifehacker.com/everyones-trying-to-track-what-you-do-on-the-web-heres-5887140) ，断开连接就是修复。断开连接会在广告追踪器、社交小工具和其他追踪元素加载之前停止它们， [加速你的网络浏览](http://lifehacker.com/disconnect-2-speeds-up-the-web-protects-you-from-third-472942968) 并阻止你的网络活动。完整的桌面应用程序更进一步，包括 [一个阻止恶意软件和广告软件的 VPN](http://lifehacker.com/disconnect-offers-a-malware-adware-blocking-vpn-to-pro-1710493422) ，但浏览器扩展对大多数人来说已经足够强大，而且它很轻量级。你们中的一些人可能正在使用类似的 [Ghostery](https://www.ghostery.com/en/) 扩展，但是 Disconnect 没有 [那么多围绕它的争议](http://lifehacker.com/ad-blocking-extension-ghostery-actually-sells-data-to-a-514417864) ，所以我们推荐它。

无论您使用 Gmail、Outlook 还是其他网络邮件应用程序，学习加密您的敏感邮件在当今的网络上是必须的，Mailvelope 是一个易于使用、简单的扩展，可以让您做到这一点。它建立在广泛使用和广受好评的 [PGP 标准](https://en.wikipedia.org/wiki/Pretty_Good_Privacy) 和 [之上，我们有完整的指南来用它加密你的电子邮件](https://lifehacker.com/how-to-encrypt-your-email-and-keep-your-conversations-p-1133495744) ，包括如何设置它的逐步说明。

### 公用事业

#### [时尚](http://userstyles.org/)

有时候，你不需要一个完整的扩展来做你想做的事情——有时候，你所需要的只是一个放置得很好的脚本。它们易于安装，易于学习编写，并且让您完全控制自己的网络体验。通常，这些都是以 Greasemonkey 脚本和用户风格的形式出现的，无论是 [修复 YouTube 的烦恼](https://lifehacker.com/how-to-fix-all-of-your-biggest-youtube-annoyances-hide-5906775)[改进 Gmail 的最新迭代](http://lifehacker.com/fix-gmails-newest-annoyances-with-these-userstyles-and-5857983) ，还是 [组织你的网飞队列](http://lifehacker.com/the-netflix-queue-sorter-userscript-organizes-your-mess-476414743) 。Userscripts 需要 Greasemonkey 扩展，Userstyles 需要 style，所以我们建议手头上有这两者。不要担心，它们不会运行或消耗系统资源，直到你真正需要一个你已经安装的脚本。

#### [YouTube 高清](https://addons.mozilla.org/en-us/firefox/addon/youtube-high-definition/)T3】

有很多方法可以解决你在 YouTube 上的烦恼，其中一些我们已经提到过了，但是 YouTube 高清版给了你一大堆超越这些工具的控制。这个插件可以让你选择所有视频的加载分辨率(例如，我把它设置为总是以最高可用分辨率加载视频)，自动播放它们或者直到你按下播放键才播放它们，自动静音等等。它将高清视频支持扩展到 Reddit、脸书和 Twitter 等网站，因此您不必手动更改分辨率，让您调整视频大小以适应您的浏览器，可以永久禁用注释，等等。最重要的是，即使其中一些功能在 YouTube 上可以切换，当你在其他地方观看嵌入的视频时，视频是否会服从你的设置还是个问题。YouTube 的高清晰度确保他们在网络上随时随地都能做到这一点。

#### [**【tab mix plus】**](https://addons.mozilla.org/en-US/firefox/addon/tab-mix-plus/)

Tab Mix Plus 是其中一个非常棒的扩展，它是唯一让一些人继续使用 Firefox 的东西——这是有充分理由的。它 [让你完全控制和定制火狐的标签](https://lifehacker.com/master-your-browsers-tabs-with-these-tricks-and-extensi-5883299) ，从复制标签，到控制标签焦点，决定当你点击标签时会发生什么，等等。不信你看看它海量的设置页面就知道了。它可能会对性能产生负面影响，但一旦你开始使用它，你会想没有它你怎么生活。

#### [DownThemAll](http://www.downthemall.net/) 和 [视频下载助手](http://www.downloadhelper.net/)

DownThemAll 可能在石器时代就已经存在了，但它仍然是 Firefox 最重要的扩展之一(而且它只在 Firefox 上可用)。它完全改变了你下载文件的方式:你可以下载网页上的所有链接、图像或其他对象，只需点击一下就可以下载符合特定标准的文件，自动重命名文件，甚至提高下载速度。当你在看的时候，我们也建议你下载视频下载助手——它会从 YouTube 和其他流媒体网站下载视频到你的电脑上。

#### [Xmarks](http://www.xmarks.com)

Firefox 已经可以在你的电脑之间同步你的书签和打开标签，但 Xmarks 是一个功能更全面的同步解决方案——另外，它还可以在 Chrome，Safari 和 Internet Explorer 之间同步，如果 Firefox 不是你唯一的浏览器，这很重要(我们愿意打赌它不是)。)您可以创建配置文件(如“工作”和“播放”)，决定将哪些文件同步到哪些电脑，从地址栏获取有关网站的信息，等等。如果你想要的比火狐的同步服务所能提供的更多，它值得一看。

#### [蜜糖](http://joinhoney.com/)

当你在网上购物时，只要有正确的优惠券或代码，你几乎总能得到一点折扣。Honey 通过查找您所在网站的代码并自动将其应用到您的订单中来自动化这一过程，因此您无需做任何事情。这是自动折扣的最佳方式，这样你总能得到最好的价格。如果你喜欢有更多控制的东西，结账时的 [优惠券](https://addons.mozilla.org/en-us/firefox/addon/coupons-at-checkout/) 是一个很好的选择。

#### [【隐形人】](http://www.getinvisiblehand.com/)

InvisibleHand 是 [另一个重要的购物扩展](https://lifehacker.com/the-best-browser-extensions-that-ll-save-you-money-and-1702736679) ，它做好了一件事:当你购物时，它会自动搜索网络的其余部分，找到你正在看的产品的更低价格。如果它在另一个网站上发现了更低的价格，它会让你知道，所以你总是得到尽可能低的价格。

#### [**盘旋猎犬**](https://addons.mozilla.org/en-US/firefox/addon/newzon/)T5】

电脑硬件很贵，所以你总是想确保你买的是最好的价格。悬停猎犬扩展 [只需将鼠标光标悬停在新蛋与 NCIX 和亚马逊](http://lifehacker.com/hover-hound-compares-newegg-prices-with-tiger-direct-an-1609738668) 的硬件价格上。您还可以查看价格历史，了解每个卖家的最高价和最低价。对于经常购买计算机硬件的人来说，Hover Hound 是必不可少的，它会定期更新以启动新功能。例如，它最近的更新让亚马逊客户能够按照符合 Prime 条件的组件进行分类，还有一个兼容性检查器和价格历史跟踪器，这样你就可以以最佳价格获得零件。

如果你用脸书，你可能以前抱怨过脸书。社会修复者是你需要的一个扩展，通过摆脱[讨厌的政治职位，追踪与你解除好友关系的人](http://lifehacker.com/how-to-find-out-when-someone-unfollows-you-on-facebook-5965816)[和更多的](http://lifehacker.com/how-to-get-rid-of-facebook-timeline-bring-back-a-simpl-5885858) 。如果你想要一个替代方案， [看看 FB 纯度](http://www.fbpurity.com/install.htm) ，但是 Social Fixer 背后的开发者指出 [一个相当大的更新即将到来](https://www.facebook.com/socialfixer/posts/10154066899214342) 这是我们所期待的。
