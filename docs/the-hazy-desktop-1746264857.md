# 模糊的桌面

> 原文：<https://lifehacker.com/the-hazy-desktop-1746264857>

晴朗的天空是最理想的，但是乔格尔梅斯特 成功地将这个朦胧的城市景观变成了一个非凡的桌面。如果你喜欢这个造型，这里有一些你需要的东西，让它成为你自己的风格。

Watch

你可能无法立即分辨，但这是 Windows 10，只是带有一点常见的雨量计魔法。下面是设置它所需的内容:

*   [高雄市](https://www.flickr.com/photos/feng1002/20541881003/sizes/l) 壁纸由 [和平投](https://www.flickr.com/photos/feng1002/) (修改)
*   用于 Windows 的 [雨量计](http://rainmeter.net/) 系统调试和监控工具
*   [色流皮为雨量计](http://joergermeister.deviantart.com/art/Patrick-Muff-s-Color-Flow-for-Rainmeter-532426147) 为时钟和日期
*   用于天气显示的 [Amana 发射器和雨量计的 water skin](http://www.deviantart.com/art/Amana-Launchers-and-Weather-268651139)
*   右下角的 [为雨量计的 Kantas 皮肤](http://joergermeister.deviantart.com/art/Kantas-470626119) 为音频可视化器
*   将 [仪表盘图标设置为](http://www.smashingmagazine.com/2015/01/freebie-dashel-icon-set-svg-psd-png/) 来替换默认的系统图标
*   空气动力学玻璃实用程序 将空气动力学透明度带回窗口
*   用于任务栏透明的 [wTrans 实用程序](http://suprvillain.deviantart.com/art/wTrans-140462809)

这部电影中肯定有一些移动的部分，但灰色的外观很好，即使你肯定不想呼吸那个镜头中的空气。此外，它是定制和个性化的，但在打开几个窗口进入正题时，一点也不会分散注意力。如果你喜欢你看到的，让 joergermeister 在他的 Flickr 页面上知道，链接如下。
