# 摩天大楼桌面

> 原文：<https://lifehacker.com/the-skyscraper-desktop-1784902866>

我们的朋友 joergermeister 在这台台式机上运行的是 Windows 10，但一如既往地按照他想要的方式定制了它的外观和工作方式，最终结果看起来很棒。如果你喜欢这个造型——或者只是其中的一部分——以下是你自己如何得到它的方法。



首先，由于这是一个 Windows 桌面，Rainmeter 是当天的订单， [本入门指南将帮助您](https://lifehacker.com/how-to-create-an-attractive-customized-desktop-hud-wit-5828789) 完成基本操作。这有点过时，但足以让您开始使用下面的皮肤和工具。

一旦你有了基本的东西，下面是你需要的:

*   壁纸，马修·维贝在 T2 国际摄影展拍摄的多伦多
*   用于 Windows 的 [雨量计](http://www.rainmeter.net/) 系统调试和监控工具
*   [为雨量计简单的清洁皮肤](http://hiphopium.deviantart.com/art/Simple-Clean-593429217) 为时钟和音乐显示
*   雨量计的 [MinimalOne 天气皮肤](http://mistrjosh.deviantart.com/art/MinimalOne-Weather-1-0-510234680)用于天气显示
*   [Ecqlipse 2 图标包](http://chrfb.deviantart.com/art/ecqlipse-2-ico-66339090) 替换任务栏系统图标
*   [Glass2K](http://glass2k.softonic.de/) 实用程序来启用任务栏和开始菜单的透明性
*   由[neio](http://neiio.deviantart.com/)视觉主题为 Windows

总的来说，这是一幅美丽的图画。我特别喜欢它的视觉风格，任务栏和开始菜单的透明性，即使我没有把桌面的其他部分定制到同样的程度，这也是我可能会抓住的东西，但它们在一起仍然很棒。如果你对这一切是如何完成的有疑问，或者想知道皮肤的样子，点击下面的 Flickr 链接，进入 joergermeister 的页面询问。
