# 你可以通过加上 Absolute 把任何物体变成一种侮辱

> 原文：<https://lifehacker.com/you-can-turn-any-object-into-an-insult-by-adding-absol-1825755510>

“如果你把‘你绝对’放在前面，你几乎可以把任何物体变成一个很好的侮辱，”Tumblr 热门帖子 说，这已经成为一个关于特定语言现象的帖子。Tumblr 的绝对轿车发现*说出*也能把话变成侮辱，还有更多:



*   **绝对 x:**醉了
*   **彻底 x:**震惊
*   完全固定:筋疲力尽/不知所措

在跟帖中，博客 [的歌词和语言](https://s-peak-in-tongue-s.tumblr.com/post/171292495592/cardboardfacewoman-rooksandravens) 提供了这样的解释:

> 这种简洁的特征被称为**搭配替代**，当某些单词与某些上下文和/或短语紧密联系时，就会出现这种情况。当你读到/听到一对通常不会放在一起的单词时，你的大脑会在上下文中填入通常推断出的内容，给出最初的短语配对。

搭配替代并不完全是一个常见的语言学术语——元过滤评论者 [沃布菲特](https://www.metafilter.com/173717/you-absolute-coat-hanger#7382165) 暗示了心理学术语“聚合替代”——但是这个短语扫描。一个 [搭配](https://www.thoughtco.com/collocation-examples-1210325) 是一个你可能都想不到的短语:整理床铺，自由自在，完成交易。一点文字游戏可能会涉及改变其中的一个词，就像每天早上“做棺材”的吸血鬼。那是搭配替换。

所以在上面的短语中，隐含的词是最常与形容词搭配的词。有几个常见的同义词听起来像家常便饭，这很有帮助:*绝对喝醉了* , *绝对喝醉了*。

通过电子邮件，语言学家约翰·麦克沃特 告诉 Lifehacker 另外两个侮辱性的搭配: *you little X* 和 *X-ass* 。*屁股*的用法如*瘸腿*、*孤腿*等。可能从字面上的*大屁股*开始，现在*大屁股*增加了一种轻蔑或古怪的味道。

绝对也有另一面，它在 Twitter 上因讽刺性的英国恭维 [*绝对疯子*](http://knowyourmeme.com/memes/the-absolute-madman) 而广受欢迎，这条推文称赞了一个绝对单位*:*

*感觉上绝对的在这里更像是一种赞美，因为它更像是一种侮辱。语境很重要！你只能把你的替代物拉伸到它断裂之前的程度。所以当你想出你的*绝对*侮辱时，确保你没有意外地称赞某人是绝对的喷气式战斗机。*
