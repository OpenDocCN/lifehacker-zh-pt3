# 一月的健身挑战:抓住引体向上吧

> 原文：<https://lifehacker.com/january-s-fitness-challenge-grab-onto-that-pull-up-bar-1821717656>

引体向上很难，但只要我们一起努力，就能让它变得更有可能。欢迎参加 Lifehacker 健身挑战赛，我们每个月都会为所有级别的锻炼者设定一个新目标。



就像在 [Lifehacker 终极金钱挑战赛](https://lifehacker.com/tag/lifehacker-ultimate-money-challenge) 中一样，我们会在每个月初解释我们的健身挑战，给你一些工具来帮助你到达那里，然后我们会在结束时签到。我们也会让尽可能多的人参与这些挑战。这个月，如果你以前从未做过引体向上，我们会让你更接近第一个。如果你已经有几个了，是时候增加你的次数或者增加一些重量了。

这是你的挑战:

1.  今天或者尽快，把你自己吊在一个横杆上，如果可以的话，手臂弯曲。如果你已经能做多个引体向上，数一数有多少个。
2.  每周两到三次，做引体向上的集中锻炼。下面我们有一些选择。
3.  月底，你会再次给自己计时(或者测试你的引体向上次数)，并惊叹你变得如此强壮。

如果你有一个合适的门道，你可以在家里安装一个拉杠:要么是带有旋入式支架 的 [，要么如果你的门道顶部有装饰，一个](https://www.amazon.com/gp/product/B0016BNDXI/?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/january-s-fitness-challenge-grab-onto-that-pull-up-bar-1821717656&asc_source=&tag=kinjalifehackerlink-20) [门挂杠](https://www.amazon.com/ProSource-Multi-Grip-Chin-Up-Pull-Up-Doorway/dp/B002YQUP7Q/ref=sr_1_4?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/january-s-fitness-challenge-grab-onto-that-pull-up-bar-1821717656&asc_source=&ie=UTF8&keywords=doorway pull up bar&qid=1514926400&s=sporting-goods&sr=1-4&tag=kinjalifehackerlink-20) 也可以，不需要任何特殊安装。在游乐场也很容易找到酒吧。几乎每个健身房都有它们，但你可能需要仔细观察:有时它们会藏在另一件设备的顶部，如电缆机。当你准备好开始锻炼时，这里有几样东西可以试试:

*   **初级:**如果你刚开始梦想引体向上，我们推荐和下面中级一样的锻炼方式(别担心，你能行！)但是如果你想要一个更简单的入门，去健身房做任何你喜欢的运动，包括用你的手臂拉。一个 [lat 下拉机](https://www.bodybuilding.com/exercises/wide-grip-lat-pulldown) 就很完美了，你可以把它调到你喜欢的轻，或者试试一个 [辅助引体向上机](https://www.youtube.com/watch?v=NMF20RR_XXw) 、 [索排](https://www.youtube.com/watch?v=IzoCF_b3cIY) 、 [哑铃排](https://www.bodybuilding.com/exercises/one-arm-dumbbell-row) ，或者 [倒排](https://www.nerdfitness.com/blog/inverted-row-are-you-missing-out-on-this-great-exercise/) 。如果你是超级新手，考虑预订一次健身之旅或与教练进行一次会谈。

*   **中级:**试试 [这种锻炼](https://vitals.lifehacker.com/this-strength-workout-can-get-you-closer-to-your-first-1793796371) 包括杠铃、哑铃、阻力带和脚滑块。只有几招 [真的管用](https://lifehacker.com/two-weeks-of-this-workout-brought-me-within-inches-of-a-1794336550) 。去年，我从 8 秒的屈臂悬垂到 20 秒，最后我可以做一个完整的引体向上，几乎可以做一个完整的引体向上。不幸的是，从那以后我就退步了，所以我要和你们一样重新开始。
*   **高级:**如果你想从一个或两个引体向上增加到更多个，上面的锻炼仍然很棒。在任何级别，也可以考虑 [改变你的握法](https://vitals.lifehacker.com/work-your-way-to-a-pull-up-with-these-intermediary-exe-1685251314) ，这样你就可以获得更多的重复次数，固定你的姿势(你有没有犯这些常见的错误 ？)或者 [学习做基普引体向上](https://vitals.lifehacker.com/this-video-shows-you-the-proper-way-to-do-kipping-pul-1787871232) ，这些*不是作弊*，只是一种完全不同的练习，学起来可能会很有趣。你也可以试着增加你做的引体向上的次数，或者 [用加重的背心](http://relativestrengthadvantage.com/7-ways-to-add-resistance-to-pull-ups-chin-ups-and-dips/) 、背包 [或者稍小的孩子](http://passionisbornunwantedlikeacancer.tumblr.com/post/124706642160/jessie-graff-is-a-precious-cinnamon-roll-and-you) 。
