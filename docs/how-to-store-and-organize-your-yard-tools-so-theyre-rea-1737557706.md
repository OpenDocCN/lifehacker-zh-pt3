# 如何存放和整理你的庭院工具，以便随时使用

> 原文：<https://lifehacker.com/how-to-store-and-organize-your-yard-tools-so-theyre-rea-1737557706>

如果你只是把你的庭院工具扔到外面的栅栏上，它们不会持续超过几个季节。以下是如何整理和保养你的庭院工具，让它们在未来的几年里照顾你。

Watch

所有类型的庭院工具应储存在阴凉干燥的地方。湿气是工具生锈和腐烂的主要原因，但阳光和高温也会导致手柄弯曲和开裂。

户外电源设备对自然环境也很敏感，绝不应暴露在阳光下或雨中。塑料电缆和橡胶索环在夏天可能会损坏，暴露在外的金属和敏感的发动机部件可能会生锈，使其难以启动。

### 如何清洁和保养您的工具

在收拾任何户外工具之前，你应该清除它们上面所有的泥土和灰尘，这些东西会保持水分。如果需要的话，清洗你的工具，用 [钢丝刷](http://www.amazon.com/Forney-70505-Scratch-4-Inch----014-Inch/dp/B000BQSBUE/ref=sr_1_12?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-to-store-and-organize-your-yard-tools-so-theyre-rea-1737557706&asc_source=&ie=UTF8&keywords=shop brush&qid=1445429123&s=power-hand-tools&sr=1-12&tag=kinjalifehackerlink-20) 擦洗并去除多余的污垢。你可以用松节油去除工具上的汁液，用醋去除铁锈。

定期涂抹亚麻籽油可以延长木柄的寿命。这将防止他们开裂和分裂，也保护他们免受水和水分。

你也可以在金属工具上涂亚麻籽油或 WD-40 来保护它们不生锈。在剪切机和剪切机的枢轴点涂抹三合一润滑油。

### 如何在车库存放你的工具

车库储存的东西比我们的汽车多得多，已经成为事实上的 [车间](http://workshop.lifehacker.com/how-to-transform-your-garage-into-the-ultimate-diy-work-1719480723) 和储藏室，储存了所有不适合放在室内的东西。院子里的设备和户外工具都属于这一类，但是你需要一个合适的方法来存放它们，这样它们就不会被损坏或者引起绊倒的危险。

最好让庭院工具远离地面，这意味着你需要利用你所有的墙壁空间。大多数庭院工具在手柄的末端都有一个洞，这样它就可以滑过螺柱上的钉子。如果 [没有洞，就钻一个](http://100things2do.ca/organization-garden-tools/) 。

你也可以购买夹子和支架，将工具固定在墙上。这个 [园林工具挂钩](http://www.amazon.com/Suncast-4-Foot-Handle-Garden-V748/dp/B0000DI83G/ref=sr_1_1?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-to-store-and-organize-your-yard-tools-so-theyre-rea-1737557706&asc_source=&ie=UTF8&keywords=yard tool storage&qid=1445430189&s=power-hand-tools&sr=1-1&tag=kinjalifehackerlink-20) 只有四英尺长，每个挂钩上可以存放多种工具。像 [角斗士车库](http://smile.amazon.com/Gladiator-Garage-Works/pages/2592639011?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-to-store-and-organize-your-yard-tools-so-theyre-rea-1737557706&asc_source=&tag=kinjalifehackerlink-20) (如上图所示)这样的模块化系统使用安装在壁挂式轨道上的支架，在悬挂工具时给你更多选择。他们还有吊篮，非常适合存放小工具，如泥铲、 [耕耘机](http://www.amazon.com/Fiskars-7877-7977-Composite-Cultivator/dp/B0002YVE54/ref=sr_1_7?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-to-store-and-organize-your-yard-tools-so-theyre-rea-1737557706&asc_source=&ie=UTF8&keywords=cultivator&qid=1445430649&s=lawn-garden&sr=1-7&tag=kinjalifehackerlink-20) 和修枝剪。

### 车库里没地方了？试试棚子

后院的棚子是存放庭院工具的好地方。存放在车库里的耙子和铲子可能会受到撞击，掉落到你或你的车上。当安全地存放在一个棚子里时，院子里的工具不会碍事，但仍然容易拿取。

网上有很多棚子的 DIY 计划，包括这些来自 [家庭杂工](http://www.familyhandyman.com/sheds) 和 [大众机械师](http://www.popularmechanics.com/home/outdoor-projects/how-to/g112/build-your-own-simple-shed-from-scratch/) 。YouTube 也是搭建棚子的一个很好的资源，包括这个来自 [房屋改善](https://www.youtube.com/watch?v=Xh_QMJALct4) 的系列。

[家得宝](http://www.homedepot.com/b/Storage-Organization-Sheds-Garages-Outdoor-Storage-Sheds/N-5yc1vZbtz2) 和 [劳氏](http://www.lowes.com/Outdoors/Sheds-Outdoor-Storage/Sheds/_/N-1z0wg8d/pl) 提供可以在一个周末安装的木制、钢制或塑料棚子套件。它们有各种尺寸和价位。

如果你方便的话，你可以在 Craigslist 上找到二手棚子，但是你经常需要自己拆卸和移除它们。

一旦你搭好了棚子，用同样的悬挂工具方法让你的院子里的工具离开地面。把你的棚子放在不会被淹没的地方，考虑把它放在一个 [凸起的平台](http://workshop.lifehacker.com/build-a-square-and-level-platform-for-a-shed-or-tiny-ho-1691636428) 上。
