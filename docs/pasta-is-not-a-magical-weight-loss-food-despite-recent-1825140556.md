# 尽管最近有研究表明，意大利面并不是一种神奇的减肥食品

> 原文:[https://life hacker . com/pasta-is-not-a-magic-weight-loss-food-Justin-1825140556](https://lifehacker.com/pasta-is-not-a-magical-weight-loss-food-despite-recent-1825140556)

如果有一项研究从好的方面展示了意大利面，这是很难忽视的。人人都爱意大利面，对吧？我知道。很可能你也是。 [巴里拉](https://www.barilla.com/en-us) ，意大利面的制造者，*肯定*会。

**头条** : [吃意大利面有助于减肥，分析发现](https://www.usatoday.com/story/news/health/2018/04/05/eating-pasta-can-help-you-lose-weight-analysis-finds/488728002/) (今日美国) [吃意大利面与减肥联系在一起的新研究](http://www.newsweek.com/pasta-weight-loss-869178) (新闻周刊)

**故事**:“我认为上下文有点迷失了，”约翰·西文派珀说，他是 [新研究](http://bmjopen.bmj.com/content/8/3/e019438) 的作者之一，当我问他对上面这些标题的看法时。该研究实际上发现，在低血糖指数饮食的背景下，少量意大利面与体重增加无关。“我们认为我们可以推广到其他健康的饮食模式，但这并不是说‘你可以在任何饮食模式中想吃多少就吃多少意大利面，并期望看到这种好处，’”他说。

这项研究不是一项新的实验，而是对以前发表的研究的分析。作者找不到单独测试意大利面减肥效果的研究，但是有包括意大利面的低血糖指数饮食的研究。

(血糖生成指数衡量的是食用特定的富含碳水化合物的食物会导致多大程度的血糖升高。玉米片的 GI 值是 81，速食土豆泥是 87。意大利面比你想象的要少:49，根据 [这个列表](https://www.health.harvard.edu/diseases-and-conditions/glycemic-index-and-glycemic-load-for-100-foods) 。)

意大利面，一周食用一小部分(半杯，三次)，不会单独破坏低 GI 饮食。这就是新的分析发现，这个消息不应该让任何人感到惊讶。我问 Sievenpiper，这种对任何食物项目的研究是否得出过这样的结论:有一种食物可以抵消健康饮食的影响。他想不出任何例子。

### 为什么会有关于一项研究的头条新闻，而这并不是真正的新闻？

公平地说，任何时候一项研究发表在科学杂志上，都可能是新闻事件。这项研究是加拿大分析特定食物在健康饮食中的作用的一部分，Sievenpiper 说他的团队对坚果和豆类(豆类和扁豆)做了类似的研究。尤其是意大利面名声不好，所以值得研究一下它是否对减肥有什么特别的作用。好吧。

但是每年都有成千上万的研究发生，却没有人听说过。(还记得我们都为 [这个分析豆](https://www.ncbi.nlm.nih.gov/pmc/articles/PMC4016088/) 激动的时候吗？是啊，我也是。)在学习和媒体狂热之间还有几个步骤，其中最大的一步是新闻发布。

在这种情况下，大部分记者看到的新闻稿是 [圣迈克尔医院](http://stmichaelshospital.com/media/detail.php?source=hospital_news/2018/0403) 发布的这个。这项研究的几位作者在这家医院工作。当研究人员发表一项可能有新闻价值的研究时，机构(如医院和大学)通知媒体是新闻周期的标准部分。新闻稿是一个预先写好的故事， [一些媒体可能会完整运行](https://www.sciencedaily.com/releases/2018/04/180403085034.htm) ，而其他媒体可能会以此为起点 [做自己的报道](https://www.popsci.com/pasta-glycemic-weight-loss-health) 。

但这还不是全部。据公关机构 [埃德尔曼](https://www.edelman.com/) 的一名代表称，意大利面制造商 Barilla 也为这项研究提供了“支持沟通”。在这项研究发表前几天，她给我发了一封电子邮件，问我是否愿意和营养师谈谈低 GI 饮食和意大利面。在最初的那封电子邮件中，Barilla 的名字没有出现在任何地方，这是我最近注意到的一种模式，当时公司希望公布一项有利于其品牌的研究。

Barilla 并没有资助这项新的研究，尽管它确实为分析所基于的一些试验提供了支持——主要是免费面食。Sievenpiper 告诉我，研究人员和公司的人经常保持联系，他们向 Barilla 的人提到他们有一项新的面食研究即将问世。

外卖:少量的意大利面可以融入健康饮食，但这不是新闻。
