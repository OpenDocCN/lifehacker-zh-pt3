# 如果你有食品加工机，你可以做这个西瓜冰冻果子露

> 原文：<https://lifehacker.com/if-you-have-a-food-processor-you-can-make-this-waterme-1828624903>

我唯一后悔买的厨房用具是冰淇淋机。它是那种有一个可冷冻的旋转容器的，据说可以把你的作品搅拌到完美，我花了 50 美元买了它。当然，从技术上来说，它“起作用了”，但有一次我把碗放在冰箱里整整一周*，当我摇晃它的时候，仍然能听到液体四处飞溅的声音。*

*Watch*

*台面上的冰淇淋机可能体积庞大，不方便，但自制冰淇淋如此耗费时间并不是他们的错。热力学是一个残酷而严格的女主人，普通的家用冰箱无法产生成功的冷冻甜点所需的剧烈温度变化。我租的，这意味着我在过去的三个公寓里都有一模一样的冰箱——你不会知道，这是你能在任何出售电器的地方买到的最便宜的冰箱。电器价格与电力输出密切相关，因此被廉价租赁电器困住的有抱负的自制冰淇淋师傅处于严重劣势。*

*即使我有一个更好的冰柜——或者最终腾出空间给 [一个卧式冰柜](https://skillet.lifehacker.com/your-apartment-needs-a-chest-freezer-1826933762#_ga=2.139534775.584420699.1535377527-144977662.1506607947)——我仍然不会去做冰淇淋，因为商店买的东西太他妈好了。我每天都想念 [Tillamook 冰淇淋](https://www.tillamook.com/products/ice-cream.html) ，但我还是被选择宠坏了。我家附近的超市里有各种各样的冰淇淋、冰糕和冰淇淋，如果我想吃些更好的，我家附近的全食超市有一个很棒的冰淇淋区。费城也喜欢它的独家商店，在 15 分钟的自行车骑行路程内有不少于六家，其中许多按品脱出售他们的手工美食。如果我想吃冰淇淋，我有很多选择，但没有一个选择是把一大碗冰淇淋塞到我的冰箱里一个星期，然后祈祷它这次真的会结冰。*

*所有这些都是说我对自制冷冻甜点一点也不热情，所以当我看到我实际上想做的“ [”双成分西瓜果子露](https://www.thekitchn.com/recipe-2-ingredient-watermelon-blender-sherbet-247123) ”的食谱时，我非常震惊。食谱很简单，但看似合理:将冷冻的大块甜瓜与加糖炼乳和盐混合，冷冻，然后食用。炼乳是一种强有力的乳化剂，并且含有太多的糖以至于不能凝固，所以这看起来是可行的。因为我是可预测的，所以我放了些盐，还加了些柠檬来加强西瓜的味道。它*统治着。**

*我不能说这个冰冻果子露改变了我对自制冷冻甜点的看法，但我可以说它非常美味，而且几乎是自己做的。我喜欢它需要很少的事先计划——甚至更少的实际工作——但尝起来不像你走了任何捷径。最重要的是，它完美地抓住了旺季西瓜的精髓，使它成为夏末炎热天气的理想选择。*

### *非常简单的西瓜果子露*

**(勉强改编自*[*The Kitchn*](https://www.thekitchn.com/recipe-2-ingredient-watermelon-blender-sherbet-247123)*)**

*这里你不需要冰淇淋机，但你需要一个食物处理器或搅拌机，以及足够的冷冻空间来容纳一个平底锅。除此之外，这是非常简单的:冷冻西瓜块，用甜炼乳和一点柠檬和盐将它们捣碎，冷冻，舀起来，狼吞虎咽。纯素食者可以用 [甜浓缩椰奶、](https://skillet.lifehacker.com/make-this-vegan-one-ingredient-caramel-sauce-in-your-i-1823718112) 来代替，说实话，它们可能味道更好。*

*如果你不喜欢西瓜，我想它可以和你喜欢的任何冷冻水果搭配，只要它有一点酸味来平衡炼乳:黑莓、覆盆子、菠萝、桃子或芒果都是不错的选择。最后，如果你想要 Instagram 上的独家新闻， [加一两汤匙伏特加](https://www.bonappetit.com/story/creamy-frozen-desserts-booze) 。(我没有喝酒，质地也没有问题。)*

***配料:***

*   *2-2 磅切丁的西瓜(半个 5 磅重的就可以了)*
*   *1 杯(约 300 克)甜炼乳*
*   *茶匙食盐*
*   *一个柠檬或酸橙的皮和汁*
*   *1 汤匙伏特加或其他烈酒(可选；确保光滑、可舀的质地)*

***指令:***

*首先，在你的冰箱里腾出一个足够大的空间来放一个平底锅。将切好的西瓜放在铺有羊皮纸的烤盘上，成单层；如果一层西瓜太多，在上面再铺一层羊皮纸，重复上述步骤。转移到冰箱里 [冷冻至固体](https://theconcourse.deadspin.com/rainbow-cake-recipe-inspires-comment-apocalypse-1592575661) :一个强大的冰箱可以在三四个小时内做到这一点，但如果你的像我一样虚弱，最好把它们放在那里过夜。*

*当西瓜完全冷冻后，将西瓜块转移到你的食品加工机或搅拌机中。如果很紧，先把甜瓜切成小块，然后再加入其余的原料。我不耐烦了，把所有的东西都扔了，去了镇上；它工作得很好，但我不得不停止处理器几次，将一些顽固的大块推向刀片。*

*一旦你加入了炼乳、盐、柠檬皮和果汁，继续搅拌直到你得到一个非常光滑的淡粉色糊状物。我的食品加工机在大约三分钟内完成了这项工作，但是如果你的需要更长的时间，不要担心——这是一个宽容的食谱。此外，你的果酱越光滑，你的冰冻果子露就越柔软，越容易舀。*

*品尝并根据需要调整调料，然后将冰冻果子露转移到冷冻安全的容器中，将羊皮纸压在表面上，冷冻至固体。像第一步一样，这将需要两个小时到一个晚上，取决于你的冰箱。*

*如果冰冻果子露完全硬了，在舀之前让它在柜台上变软几分钟。照原样端上桌，或者，如果你喜欢的话，撒上柠檬皮或芝麻酱。*
