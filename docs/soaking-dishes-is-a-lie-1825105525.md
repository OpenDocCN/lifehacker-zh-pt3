# 你不必浸泡你的盘子

> 原文：<https://lifehacker.com/soaking-dishes-is-a-lie-1825105525>

你早上醒来，摇摇晃晃地去厨房煮咖啡，这时你闻到了咖啡的味道。它强劲、可怜，而且来自你的水槽——脏盘子。前一天晚上，你在一些有硬壳的碗和锅里装满了水，假装以后会清理它们，但是你忘记了，现在你后悔了。是时候停止欺骗自己了——是时候停止浸泡了。



Lifehacker 非常重视春季大扫除。我们绝不会错过更新、重组和整理我们家庭生活的机会。我们也非常兴奋地按下科技使用的重置按钮，仔细查看我们的财务状况，并抛弃已经有点发霉的日常习惯。欢迎来到 [*春季大扫除周*](https://lifehacker.com/c/spring-cleaning-week) *，在这里我们清理冬天的蜘蛛网，为未来的晴天做准备。让我们收拾一下，好吗？*

有哪些菜其实*反正需要*泡？很多时候，你只需要泡那些你做的不够快的菜。如果你掌握了所有的东西，并且很早就接触到食物残渣，你可能永远都不需要浸泡任何东西。我的意思是，有人会浸泡他们的盘子还有其他潜在的原因，但主要原因是“我现在不想洗碗……”所以你把它们扔进水槽，然后继续做你的事情，以为你以后会让事情变得更容易。但是在你离开那个装满浸泡过的盘子的水槽之前，问问你自己:它们真的需要浸泡吗，还是你在偷懒？

即使你最终被食物烧焦或因为它是一种很快变干的物质而粘上了一些污垢，你仍然可以在洗碗的时候做一些事情来处理异常坚硬的污垢，而不是被动地浸泡掉你的问题。如果你有一个锅，上面烧着东西，把锅放回炉子上，加入一些水，然后把火调大，直到水沸腾。不会花很长时间的，那些烧焦的东西会马上掉下来。或者你可以用醋给你的锅快速脱釉。瞧啊。

如果你的盘子、杯子或马克杯上有东西结块，加入一点水，在上面放另一个盘子，然后在微波炉中加热 30 秒。吊杆，蒸汽清洗。此外，如果你有洗碗机，真的没有必要过度清洗碗碟。在你把盘子扔进洗衣机之前，它们不需要一尘不染。事实上，大多数现代餐具洗涤剂使用酶来消化你的盘子和其他厨房用具上的蛋白质和淀粉。如果里面没有食物来激活它们，清洁剂就没什么用了。

听着，我承认，有些东西确实需要浸泡。烤盘、搅拌机罐和其他不适合微波炉或不能在炉子上烧水的奇怪物品，需要一两分钟，用一些温水和肥皂。此外，我们只是人，我们有时都会忘记事情，好好泡一泡可能是唯一的选择，可以把放在你桌子上两天的杯子里的燕麦片拿出来。浸泡掉。

也就是说，如果你要浸泡，在洗其他盘子的时候也要浸泡。开始浸泡，清洗其他不需要浸泡的物品，然后回来浸泡物品。它们不需要很长时间就可以准备好擦洗。做我喜欢称之为“完成循环”的事情在完成手头的任务之前，不要转移到其他事情上。你已经开始了，你已经用橡胶手套、刷子和浓缩洗碗皂武装了自己，所以完成你已经开始的战斗吧，战士。向这些油脂战士举起你的武器，毫不留情。
