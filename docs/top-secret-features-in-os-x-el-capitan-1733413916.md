# OS X El Capitan 中的绝密功能

> 原文：<https://lifehacker.com/top-secret-features-in-os-x-el-capitan-1733413916>

OS X El Capitan 没有很多标志性的功能，这意味着大多数很酷的东西都在引擎盖下。让我们来看看一些鲜为人知的特性。



### Mail 现在支持标签

你是那种喜欢一次写多封邮件的人吗？恭喜你，El Capitan 的新邮件版本让标签变得简单多了。你现在可以用 Command+N 为电子邮件打开新标签，就像你在网络浏览器上一样，尽管它只能在全屏模式下工作。

### 轻扫以删除 Mail 中的电子邮件

在 iOS 上滑动删除电子邮件是当今几乎每个电子邮件应用程序都使用的伟大的小功能之一。在 El Capitan 中，你可以用触控板做同样的手势。

### 从 Mail 快速将事件添加到您的日历中

与 iOS 9 一样，El Capitan 现在可以扫描你的电子邮件以获取数据指标，只需点击一下，就可以将事件和联系人从邮件直接添加到日历中。当它看到你的电子邮件中列出的日期时，你会收到一个小通知，如果你愿意，可以将它添加到你的日历中。

### 追踪 OS X 任何地方的航班

就像在 iOS 里一样(注意到这里有个主题？)，El Capitan 现在检测航班号。如果它在邮件、笔记、信息或其他任何地方看到它认为是航班的东西，当你把鼠标放在它上面时，它会显示一个框。点击它，你就会得到那次航班的最新信息。

### 从上下文菜单中重命名文件

如果你不喜欢“缓慢点击”一个文件名来重命名它，你现在可以通过右击一个文件，然后选择“重命名文件”

### 更改 App Store 密码设置

如果你的电脑有一个非常安全的设置，你现在可以更改设置，设置你需要输入密码从 Mac App Store 下载东西的频率和时间，就像你在 iOS 上一样。

### 右键单击复制文件的路径

现在，您可以从上下文菜单中复制文件的完整路径名。只需右键单击文件，轻按 Option 键，您就会看到“拷贝路径名”选项如果你经常使用终端，这个非常方便。

### 显示和隐藏菜单栏

就像 Dock 一样，您现在可以轻松地显示和隐藏菜单栏。打开“系统偏好设置”>“通用”,选中“自动隐藏和显示菜单栏”复选框

### 文件拷贝恢复从你停止的地方开始拷贝

在以前的 OS X 版本中，当文件复制因连接中断或电脑进入睡眠状态而中断时，你必须重新开始。现在，当您备份并运行时，它应该会自动恢复复制过程。

### AirPlay 视频而不共享全屏幕

不管出于什么原因，在 El Capitan 之前，在 Safari 之外播放网络视频的唯一方式是共享整个浏览器屏幕。现在，您可以在 Safari 中轻按视频上的 AirPlay 图标，然后选择您的设备，而不会在桌面上显示其他任何内容。
