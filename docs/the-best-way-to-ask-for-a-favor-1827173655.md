# 请求帮助的最好方式

> 原文：<https://lifehacker.com/the-best-way-to-ask-for-a-favor-1827173655>

所以，你需要一些帮助。没关系，我们有时都需要帮手。问题是，我们很多人不知道如何去问——所以我们没有问。根据一位社会心理学家的说法，这里有一个最简单、最有效的请求帮助的方法。

[*援军:如何让人们帮助你*](https://hbr.org/product/reinforcements-how-to-get-people-to-help-you/10079-HBK-ENG)*[告诉 The Verge](https://www.theverge.com/2018/6/22/17475134/heidi-grant-reinforcements-help-social-psychology) 要牢记的最重要的一点是，求助总是没问题的。我们倾向于认为人们会拒绝我们的请求，或者瞧不起我们，所以我们没有像应该的那样经常寻求帮助。然而，根据研究，当我们寻求帮助时，人们更喜欢我们。但是，你如何着手让他们真正贯彻到底呢？*

### *挑选一个人来问*

*格兰特说最好问一个人，而不是一群人。询问一个团体会导致“责任分散”现象，没有人会觉得自己必须帮忙，因为他们认为别人会帮忙。选择一个你认为对你最有帮助的人，或者至少同时向几个人发送个人请求，而不是在群聊中掉线。*

### *明确表示你需要帮助，并且要具体*

*人们害怕因不必要的帮助而冒犯他人，所以要清楚地表明你是在寻求帮助，而不仅仅是发泄。此外，确保你的要求非常具体。不要含糊你需要什么，也不要用“你能帮我个忙吗？”这样的话来打开话题。格兰特说，这是一种操控，因为它迫使某人在你告诉他们你需要什么之前做出承诺。一句简单的“你能帮我做一件具体的事情吗？”会的。*

### *给他们一个时间表，和一个逃脱的机会*

*好吧，这个提示是我的，不是格兰特的，但它仍然很重要。当你寻求帮助时，给这个人一些时间表或者软期限。像“只要你有时间”或“如果你有时间”这样的不确定的短语把责任推给了已经在帮你忙的人。你可能认为让他们选择是礼貌的，但实际上这有点粗鲁，因为你给了他们更大的压力。你在说，“嘿，我需要你的帮助，但我也希望你知道我什么时候需要它，”这让压力更大了。现在，他们又有一个抽象的任务要完成。问他们什么时候有空没有错，但是这有助于他们知道你什么时候真正需要帮助。例如:“今天下午某个时候你能帮我做[具体的事情]吗？我希望今天能结束这件事。”*

*如果你知道他们很忙，给他们一条出路也很好。这可以确保你的求助感觉不像是一种要求。保持优雅。*

### *不要道歉*

*当你寻求帮助时，你可能会为占用了他们的时间和精力而道歉。不要。根据格兰特的说法，这是一个坏主意，因为贬低自己会让别人在帮助你时感觉不到快乐。我们很多人 [反正道歉太多](https://lifehacker.com/how-to-stop-apologizing-for-everything-you-do-1746770762#_ga=2.84567066.2009130109.1529935949-229306750.1526322168) 。*

### *不要犹豫绕回人*

*格兰特希望人们了解一种尚未开发的帮助资源:那些过去拒绝帮助我们的人。以前有人不帮你，不代表现在不会帮你。事实上，研究表明他们比 T2 更有可能提供帮助。为什么？因为他们想对之前拒绝你的行为感觉好些，并“修复关系”所以找一个以前不会或者不能帮你的人，再问一次。*
