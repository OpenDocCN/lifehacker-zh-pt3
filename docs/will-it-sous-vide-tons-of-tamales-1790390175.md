# 它会腐烂吗？成吨的玉米粉蒸肉

各位厨师和食物消费者好，欢迎回到另一集充满乐趣的 [*它会分开吗？*](http://lifehacker.com/tag/will-it-sous-vide) ，每周专栏我用我的沉浸循环器在那里制作任何你想让我做的东西。

本周我们来到了玉米粉蒸肉，这令人兴奋有两个原因:

1.  玉米粉蒸肉非常好吃。
2.  我从来没有做过玉米粉蒸肉，我喜欢一种新的烹饪学习体验(这就是我想要的)。

cULvCC"><div class="bxm4mm-4 bxm4mm-5 cbuuRJ"/><div class="bxm4mm-4 bxm4mm-6 jhbXha"/></div></div></div><p class="sc-77igqf-0 bOfvBY">首先，我们将一些谷壳在清水中浸泡 20 分钟，然后彻底晾干。接下来，我们融化了一杯起酥油(你也可以用猪油),用我的立式搅拌器搅拌，直到它变得浑浊。</p><p class="sc-77igqf-0 bOfvBY">在另一个碗里，我们混合了三杯速溶豆粉、一茶匙大蒜粉、一小撮辣椒粉、一小撮孜然粉和一茶匙发酵粉。然后将该混合物逐渐加入熔化的起酥油中，同时用桨式附件搅拌。</p><div id="mgid-container" bOfvBY">接下来，我们逐渐舀入热汤，仍然用立式搅拌机搅拌，直到我们有一个光滑的混合物，我们可以用勺子画一个图案。然后我们进行了“漂浮测试”,看看我们的湿润粉糊是否足够蓬松。</p><p class="sc-77igqf-0 bOfvBY">不够蓬松，所以我们多加了一点克里斯科。</p><p class="sc-77igqf-0 bOfvBY">接下来，我们组装填充物。虽然我很想做一道美味的猪肉馅，但我认为这会一下子引入太多的变数，所以我想把重点放在湿润粉糊上。(此外，我讨厌花几个小时来完善肉馅，却在最后全部散架。)因此，我决定搭配一个简单的辣椒馅(十二个烤阿纳海姆辣椒和一个干哈奇辣椒)，奶酪(每种 10 盎司奎索弗雷斯科奶酪、科蒂加奶酪和切达奶酪，全部切碎)，和一个大番茄。</p><p class="sc-77igqf-0 bOfvBY">然后我们把湿润粉糊撒到谷壳里，大约三分之二。Amanda 特别为此准备了一个小的湿润粉糊刮刀，但是你也可以用勺子。</p><p class="sc-77igqf-0 bOfvBY">接下来，我们加入辣椒混合物。</p><p class="sc-77igqf-0 bOfvBY">然后是奶酪。</p><p class="sc-77igqf-0 bOfvBY">然后我们把它们包得很漂亮。</p><p class="sc-77igqf-0 bOfvBY">这些婴儿被真空密封，然后浸入 185 华氏度的水浴中。</p><p class="sc-77igqf-0 bOfvBY">玉米粉蒸肉通常要蒸几个小时，但我是一只好奇的猫，决定在一个小时后拿出一只来。它摸起来像浆糊一样，我甚至不用打开外壳就知道它仍然是液态的，所以我重新密封袋子，再放进去一个小时。</p><p class="sc-77igqf-0 bOfvBY">经过两个小时的腌制，我小心翼翼地打开一个外壳，找到了这堆热乎乎的东西(坦白说:我吃了它，味道非常好):</p><p class="sc-77igqf-0 bOfvBY">显然两个小时是不够的，所以剩下的两个玉米粉蒸肉回到浴室，我去了酒吧。一个小时后，喝了三杯鸡尾酒——我是一名专业人士——我回到了我的公寓，兴奋地吃着(希望)硬玉米粉蒸肉。</p><p class="sc-77igqf-0 bOfvBY">原来，三小时的腌玉米粉蒸肉看起来就像两小时的腌玉米粉蒸肉。(如果你已经忘记了那是什么样子，请看上面的图片)。这对喝醉的克莱尔来说是令人失望的，但还不至于失望到她没有吃到一份墨西哥玉米卷，外加一份<span> <a class="sc-1out364-0 hMndXN sc-145m8ut-0 fBlGIv js_link" data-ga="[[&quot;Embedded Url&quot;,&quot;External link&quot;,&quot;https://twitter.com/clairelizzie/status/811129053629382656&quot;,{&quot;metric25&quot;:1}]]" href="https://twitter.com/clairelizzie/status/811129053629382656" target="_blank" rel="noopener noreferrer">冰冻的 3 只麝香猫棒</a> </span>。喝醉的克莱尔接着就睡着了。</p><p class="sc-77igqf-0 bOfvBY">当我第二天早上醒来时，我决定再试一次，并把另一袋真空密封的玉米粉蒸肉扔进了水浴中。我几乎没抱什么希望，但是在第四个小时的时候，我遇到了这位小女士:</p><p class="sc-77igqf-0 bOfvBY">仍然有一些问题，但是事情正在好转。我吃了差不多的玉米粉蒸肉，重新密封袋子，把玉米皮美人又泡了一个小时。然后，<em>终于</em>，在五个小时后，我打开了这个:</p><p class="sc-77igqf-0 bOfvBY">我在上面放了调味汁。</p><p class="sc-77igqf-0 bOfvBY">我吃了。它又干酪又辣又嫩，非常好吃。我很开心。我打开了第二个。</p><p class="sc-77igqf-0 bOfvBY">这是毁灭性的。我打开第三个，它很好。我打开第四个，它并不好。所以成功的几率是一半一半。当然，这为我们提出了一个很好的“问题”:<strong>玉米粉蒸肉会分开吗？</strong></p><p class="sc-77igqf-0 bOfvBY"><strong>答案？</strong>嗯，算是吧？虽然这一堆玉米粉蒸肉中没有一个味道不好的，但只有一半的玉米粉蒸肉在五个小时的烹饪后保持了原有的形状。这是一个相当令人沮丧的收益率。虽然真空烹饪提供了一个舒适、潮湿的烹饪环境，但我认为它可能有点<em>太</em>潮湿了。尽管玉米粉蒸肉通常是蒸的，但仍然有一些油和水分从湿润粉糊中逸出，这与真空密封袋的情况不同。当然，还有其他的变量，比如湿润粉糊层的厚度和使用的脂肪和馅料的类型，我不反对将来重新讨论这个话题，尤其是如果你们有建议的话。我愿意吃很多玉米粉蒸肉，甚至是那些热腾腾的各种各样的；我就是那么无私。</p><p
