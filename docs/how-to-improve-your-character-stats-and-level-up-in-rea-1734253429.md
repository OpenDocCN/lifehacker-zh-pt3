# 如何在现实生活中提高你的角色属性和等级

> 原文：<https://lifehacker.com/how-to-improve-your-character-stats-and-level-up-in-rea-1734253429>

在《龙与地下城》中，每个角色都有一定的长处和短处，这取决于他们的能力分数:力量、敏捷、体质、智力、智慧和魅力。你现实生活中的自己也是由这些能力组成的，你可以像在游戏中一样提升它们。

Watch

### 强度(STR)

力量是不言自明的:它描述了你的肌肉和体力水平。虽然在日常生活中，你可能无法用你的千真万确之剑打倒兽人，但力量仍然可以派上用场——更不用说改善你的外表了。无论你是正在努力减肥的还是在瘦骨嶙峋的骨架上练肌肉的，举重都是增强力量的好方法。查看 [我们的](http://vitals.lifehacker.com/how-to-finally-start-working-out-even-if-you-hate-it-1729593891#_ga=1.172530851.127976026.1441054335) 入门指南，其中包括找到一个好的初学者计划——比如 [起始力量](http://startingstrength.com/) 或 [最低可行体能](http://www.mvf.io/mvf-part-three-highly-effective-workouts/)——对你有用。

### 灵巧度

是关于其他类型的运动能力。在龙与地下城的世界里，这意味着拥有闪避来袭的敏捷，以及射出弓箭的手眼协调能力。虽然你当然可以练习武术或射箭来增强这些技能，但成年后你可以开始大量有趣的运动，从攀岩到足球到轮滑。你也可以只用一个网球 来增强你的运动能力和手眼协调能力...但这一点也不令人兴奋。除非你假装那是一个半精灵巫师射来的火球。

### 宪法(续)

[体质](http://www.dandwiki.com/wiki/SRD:Constitution) 是一个稍微抽象一点的概念，但是它代表了你的角色的健康和耐力。它决定了你的角色在死前有多少“生命值”，决定了他们对毒药的抗性，并在他们的肩膀上嵌有连枷时给予他们继续前进的能量。在现实世界中，当你试图 [忍受身体疼痛](https://lifehacker.com/how-to-manage-your-tolerance-for-physical-pain-1689263947) 或 [增强你的免疫系统](http://lifehacker.com/the-no-bs-guide-to-boosting-your-immunity-and-avoiding-5858209) 时，你更有可能遇到体质问题。见鬼，甚至 [吃得更好](http://vitals.lifehacker.com/the-only-three-things-everybody-agrees-on-when-it-comes-1709039566) 也能增加你的体质...因为，你知道，这会降低你死于心脏病的可能性。

### 智能(INT)

在灰鹰之地，智力对于巫师学习不同的法术很重要。但是在现实世界中，智力对我们所有人都是有价值的。人们普遍认为你的智商不会随着时间而改变，但这个 [不一定是真的](http://lifehacker.com/nine-stubborn-brain-myths-that-just-wont-die-debunked-5867049#_ga=1.114390151.127976026.1441054335) 。所以 [通过学习问正确的问题来磨砺那些批判性思维技能](http://lifehacker.com/how-to-train-your-mind-to-think-critically-and-form-you-1516998286) ，通过 [抛弃那些丑陋的谬误来提升你的逻辑能力](http://lifehacker.com/the-definitive-guide-to-winning-an-argument-1693076653#_ga=1.152090425.127976026.1441054335) ，当有疑问的时候， [假装它，直到你做到](http://lifehacker.com/how-to-sound-like-you-know-what-youre-talking-about-ev-5853250) 。你不会很快施展魔法导弹，但至少你能在争论中坚持自己的立场，做出更明智的决定。

### 智慧

不要与智力混淆，有点令人困惑的 [智慧](http://www.dandwiki.com/wiki/SRD:Wisdom) 技能决定了意志力、常识和感知等东西。它在 D & D 世界和现实世界中都有类似的功能，比如 [检测某人何时说谎](http://lifehacker.com/how-to-detect-when-someones-lying-and-get-them-to-tell-5959952) 或 [行使自我控制](http://lifehacker.com/how-self-control-works-and-how-to-boost-your-willpower-5802572) 。事实上，当你试图建立新的健康习惯时，意志力是非常有用的技能。虽然增加你的意志力并不容易，但你可以确保有效利用你的意志力。再用一个 RPG 的比喻: [把它想成魔分](https://lifehacker.com/think-of-your-willpower-as-magic-points-to-maximize-men-1718102063) 。把它们“花”在最重要的地方，因为当你用完的时候，你就完了。

### 魅力(茶)

[魅力](http://www.dandwiki.com/wiki/SRD:Charisma) 正是它听起来的样子，它在现实世界里和在幻想世界里一样有用。与普遍的看法相反，魅力不是你与生俱来的: [这是一种你可以通过用你的身体和声音表达来培养的技能](https://lifehacker.com/how-to-develop-your-charisma-and-become-more-likable-1673988208) 。良好的魅力还意味着与他人交往，知道何时倾听而不是谈论。 [信心起很大作用这里](http://lifehacker.com/how-to-build-your-confidence-and-why-it-matters-1442414831) 也是如此。魅力的好处不仅仅是“被人喜欢”。就像在龙与地下城&中一样，拥有魅力意味着 [更有说服力](http://lifehacker.com/learn-the-simple-tricks-to-the-art-of-persuasion-5911188) 和 [成为更好的领导者](http://lifehacker.com/classic-leadership-skills-you-should-apply-to-your-soci-1425624672) ，这在工作和家庭中都是无价的。
