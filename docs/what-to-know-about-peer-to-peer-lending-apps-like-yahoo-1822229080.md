# 如何了解雅虎的“Tanda”等 P2P 贷款应用

> 原文：<https://lifehacker.com/what-to-know-about-peer-to-peer-lending-apps-like-yahoo-1822229080>

雅虎金融正寻求将最古老的无息贷款形式之一货币化。



该公司上周五推出的 Tanda 应用程序可在安卓上使用，很快也将在 iOS 上使用，正如其名称所暗示的那样。它可以让你加入一群人，一前一后地为储蓄目标而努力:每个用户向一个罐子里支付商定的金额，选择他们何时收到钱。最早需要的人支付费用，等待时间最长的人获得 2%的奖金。根据 [Android Police](http://www.androidpolice.com/2018/01/20/yahoo-finance-launches-tanda-app-help-save-money-wisely-others/) 的说法，雅虎金融抽取第一笔支出的 8%和第二笔支出的 7%。

基本上就是数字化的借贷圈。例如，如果五个人中的每个人在五个月内每月支付 100 美元，每个人在其中一个月会得到 500 美元。如果你是第一批收到钱的人之一，这就像一笔低息贷款。如果你是最后一批人中的一员，这是一种省钱的方式(通过雅虎的应用程序，还可以获得奖金)。这些类型的信用系统一直存在——非正式地说，它们存在于家人、朋友或邻居之间，根据在墨西哥被称为 *tandas* 或 *cundinas* ，在巴西被称为 *pandeiros* ，在西非和加勒比海被称为 *susus* ，在亚洲被称为 *hui* 。

当在朋友之间进行时，该系统作为一种向需要贷款的人提供无息贷款的方式，同时作为一种储蓄工具，供后来收到付款的人使用。你不会带着比以前更多的钱离开(除非你得到了储蓄奖金)，但是你在帮助朋友，为你自己的储蓄目标而努力。

雅虎财经并不是第一个想到将这一系统的更正式版本货币化的人——网站自 2013 年起向公众开放，服务超过[【300 万美元](https://www.emoneypool.com/about) ，应用 [KyePot](https://play.google.com/store/apps/details?id=com.kyepot.india.chitfund&hl=en) 和 [Cashare](https://play.google.com/store/apps/details?id=com.cashareinc.cashare&hl=en) 也有类似的目的。在 Tanda 上，用户会得到一个信任分数，分数越高，用户就可以获得更大的资金池，最高可达 2000 美元。

因此，即使在较高的水平上，这也是一种相对较小金额的储蓄工具。但是雅虎指出，来自国家经济研究局 T2 T3 的研究表明，当人们集体行动时，他们会更好地储蓄。例如，在一项实验中，研究人员发现，有储蓄任务并可以选择公开宣布进展的人，储蓄几乎是没有选择的人的两倍。(为此，该应用还提供了一个类似 Venmo 的社交 feed，供用户评论和添加更新。)

如果有人退出，坦达会承担他们的部分。到目前为止， [评论者](https://play.google.com/store/apps/details?id=com.yahoo.tanda) 指出了使用该应用程序的一些挫折，比如当他们从零开始时，很难提高他们的“信任分数”。但作为一种让自己坚持储蓄目标的方式，测试一下可能是值得的。
