# 关于贝尔氏麻痹，一种使你半边脸瘫痪的疾病，你该知道些什么

> 原文：<https://lifehacker.com/what-to-know-about-bells-palsy-a-condition-that-paraly-1797313261>

安吉丽娜·朱莉 [在名利场采访](http://www.vanityfair.com/hollywood/2017/07/angelina-jolie-cover-story) 中透露，她去年患上了贝尔氏麻痹，导致她一侧面部瘫痪。这种情况可能会突然出现，但往往会在几周或几个月内自行消失。(朱莉后来康复了。)



贝尔麻痹的原因有时被描述为未知，但它可能是由一种 [病毒感染引起的，这种病毒感染突然发作并刺激了连接大脑和面部肌肉的神经](http://www.medicalnewstoday.com/articles/158863.php) 。男性和女性都可能得这种病，但是如果你是孕妇或糖尿病患者，你的风险更高。朱莉患有的高血压也可能使你更容易患上贝尔氏麻痹。

对于轻度病例，贝尔麻痹通常会随着时间的推移而好转，但寻求治疗仍然是一个好主意。抗炎和抗病毒药物在某些情况下有帮助。物理治疗可以帮助面部肌肉恢复，如果你的眼睛不能正常闭合，你可能需要人工泪液或其他治疗来保护你的视力。

对她来说，朱莉说针灸帮助她康复，一篇关于针灸治疗贝尔麻痹 的综述发现，针灸似乎在某些情况下有所帮助——但现有的研究质量很低，所以我持怀疑态度。

中风也会导致半瘫痪的脸，所以不要忽视这个症状。如果你真的中风了，时间就是大脑:你接受治疗越快，你遭受长期损害的可能性就越小。急诊室医生 [可以区分贝尔麻痹和中风瘫痪](http://www.jems.com/articles/print/volume-39/issue-5/features/differentiating-facial-weakness-caused-b.html) ，所以一定要去检查一下。提醒一下，你对中风症状的记忆是:

*   面部麻木或无力，尤其是一侧
*   **A:** 手臂麻木或无力，特别是一侧(像僵尸一样举起双臂；它们是水平的吗？)
*   口齿不清或说话困难
*   打 911 的时间到了
