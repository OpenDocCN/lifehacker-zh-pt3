# 如何观看《星际迷航:随处发现》

> 原文：<https://lifehacker.com/how-to-watch-star-trek-discovery-anywhere-1818763050>

CBS 的《新科幻冒险》， [*星际迷航:发现*](https://io9.gizmodo.com/everything-you-need-to-know-about-star-trek-discovery-1818597046) ，终于来了。第一季分为两章，前八集将于 9 月 24 日周日至 11 月 5 日周日播出，第二章将于 2018 年 1 月播出。然而，如果你认为你可以打开电视去没人去过的地方，那你就错了。该节目是一个流媒体独家节目，这意味着你必须注册 CBS All Access，该公司的点播和直播流媒体服务，并在你不断增长的每月义务清单上再增加一笔订阅费。



### **太好了，又一个流媒体服务**

两个小时的首映式的第一个小时在哥伦比亚广播公司播出，后半部分，以及该系列的其余部分，只能通过 [CBS All Access](https://www.cbs.com/all-access/) 流媒体服务观看。这意味着无论你的有线电视计划如何，你都需要支付另一个订阅服务(呃)来获得你的*星际迷航:发现*修复。你可以通过 [CBS 的网站](http://www.cbs.com/) 或其流媒体应用程序访问 CBS All Access 服务，该服务几乎可以在你拥有的所有设备上使用，包括你的 iOS 和 Android 设备、Apple TV、Roku player、游戏机(抱歉，这里没有 Switch 或 PS3 支持)或亚马逊 Fire TV。

有两种 CBS All Access 计划可供选择。你可以以每月 5.99 美元的价格注册有限的商业计划，或者以 9.99 美元的价格注册免费商业计划。这两个选项都提供了一个你可以提前购买的年度计划，与每月选项相比，你可以节省一些钱。

当然，你可能只是想看最新的科幻冒险节目，但值得注意的是，CBS All Access 给你的不仅仅是*发现*。这项服务提供“超过 9000 集的点播节目”——你可以观看从黄金时间的节目，旧赛季的精选节目，以及基于当地 CBS 电视台的直播电视节目。

### **在美国境外观看**

即使你不在美国，你也可以通过其他途径了解星际探索。国外的网飞用户可以和美国观众同时观看《探索》节目。加拿大人可以在贝尔传媒的科幻频道或者通过贝尔传媒的点播视频流媒体服务 [CraveTV](http://www.cravetv.ca/) 欣赏该节目。

如果你不愿意注册流媒体服务来观看你可能不喜欢的单一节目，我理解这种担忧。但是你也可以在 CBS All Access 上观看你所知道和喜爱的所有《星际迷航》老剧集，这要感谢它的所有以前的《星际迷航》系列的完整目录，从最初的系列到*企业*。
