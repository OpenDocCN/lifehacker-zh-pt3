# 设置 Dropbox 帐户的最佳方式

> 原文：<https://lifehacker.com/the-best-way-to-set-up-your-dropbox-account-1828630098>

有很多服务可以用来在云中托管你的文件。Dropbox 是一个可靠的选择，尽管如果你不支付月租费，它不会给你太多的存储空间。不过，这是一种非常方便的方式，可以在您的计算机和设备之间访问共享的文件池。即使它已经成为“家喻户晓”的软件，这里也有一个指南，让您充分利用这个伟大的存储解决方案。

### 如何安装和配置 Dropbox

在你的系统上安装 Dropbox 非常简单。你可以获得所有主要平台的应用程序——甚至是 Linux，如果你觉得有趣，你可以通过命令行把它放到你的服务器上:

*   [Windows / Mac](https://www.dropbox.com/install)

*   [iOS](https://itunes.apple.com/us/app/dropbox/id327630330?mt=8)T3 
*   [安卓](https://play.google.com/store/apps/details?id=com.dropbox.android&hl=en_US)

出于本指南的目的，我们将浏览 Mac 安装程序，尽管如果你的首选平台不是苹果的，Dropbox 应该很容易设置。至少在 Mac 上，你真的不需要考虑太多事情，直到应用程序安装完毕，Dropbox 的登录屏幕出现在你面前。如果您已经有帐户，请输入您的凭据。简单。

但是，如果你还没有账户，就不要急着去创建一个。你应该看看你的任何使用 Dropbox 的朋友是否愿意邀请你使用 [、推荐代码](https://www.dropbox.com/help/space/earn-space-referring-friends) 的服务。如果是这样，当你注册时，他们(和你)将获得 500MB 的免费存储空间——是你的免费存储空间配额(2GB)的四分之一。如果你没有朋友使用 Dropbox，在网上找到一个推荐链接 [应该不会很难](https://www.reddit.com/r/dropbox/comments/7i6ezu/referrals_please_post_referral_requests_in_this/) (确保你使用相同的社区/平台来吸引其他人使用*你的*推荐链接。犒劳自己。)

你还应该检查一下 Dropbox 的网络设置。默认情况下，该应用程序不会限制其下载速度，但如果你的连接速度较慢，并且发现 Dropbox 使你正常的网络浏览、下载或玩游戏速度变慢，你可能需要设置一些限制。上传也是如此，Dropbox 会自动限制上传。如果你使用的是更快的互联网计划(或者光纤连接)，你可能不需要 Dropbox 来拖你的后腿。让你的上传飞起来。

### 在 Dropbox 上获得更多空间

如果你希望备份电脑上的大量数据，Dropbox 可能 [不是一个很好的选择](https://blogs.office.com/en-us/2018/02/06/make-the-switch-to-onedrive-for-improved-productivity-and-cost-savings/)——至少，如果你坚持使用免费版本的 2GB 存储空间，而不是 Dropbox 的 [付费计划](https://www.dropbox.com/help/billing/cost) 的话。不过，你仍然可以做一些事情，给自己一些额外的免费千兆字节。

首先，尽可能多地使用 Dropbox 的推荐程序。不要成为那种向你能想到的每一个网络论坛发送邀请的人，但是对于你邀请谁和如何邀请要有策略。甚至不要试图做一些愚蠢的把戏，比如在你的 Gmail 地址中插入一个句号，来创建一个你可以邀请的“新朋友”。当然，这适用于邀请部分，但如果你试图在一台已经使用 DropboxT3[的电脑上登录一个新账户，Dropbox 会注意到的。](https://www.dropboxforum.com/t5/Sharing-and-collaboration/Can-t-resent-the-referal-invite/td-p/256565)

不过，你仍然可以获得一点点创意。如果你有一个工作电子邮件地址和一台从未接触过 Dropbox 的工作电脑，请随意邀请自己。使用您的工作电子邮件创建一个新帐户，照常登录，在退出并切换到与您的个人电子邮件帐户相关联的 Dropbox 之前， [检查您的免费 500GB](https://www.dropbox.com/account/plan) 。

寻求尽可能多的免费空间的狡猾的 Dropbox 用户——违反了 Dropbox 的 [服务条款](https://www.dropbox.com/privacy#acceptable_use)——也会将虚拟电子邮件帐户与虚拟机结合起来。通过改变虚拟系统的配置、MAC 地址和 IP 地址等参数(例如，使用 VPN)，他们会欺骗 Dropbox，让它认为受邀用户是一台独特计算机上的真实人物。重复此操作总共 32 次，您将能够获得 16GB 的额外存储空间。

不要忘了浏览 Dropbox 的“入门”指南，当你完成 时，它会为你免费提供 500MB [。你也可以参加 Dropbox 的在线论坛，为自己赢得免费的空间来进行深思熟虑的回复，但不能保证你的时间和努力会为你赢得任何东西。不过，这是一个很好的姿态。同样，如果你在 Twitter 上关注 Dropbox 或告诉 Dropbox 你为什么喜欢它，你可以为每个活动](https://www.dropbox.com/gs) 额外赚取 250MB—125MB 。

以前有许多其他活动和挑战 你可以做，包括安装其他 Dropbox 应用程序，如 Mailbox 或 Carousel，以积累免费的千兆字节。今天，Dropbox 的推荐计划仍然是获得免费存储的最大和最好的方式。

### 借助第三方工具让 Dropbox 更有用

您可以在许多不同的应用程序和服务中连接到您的 Dropbox，这使得下载和上传内容更加容易。浏览器扩展允许你将你的 Dropbox 与你的 Gmail 账户同步——或者你可以通过谷歌的 [G 套件市场](https://lifehacker.com/save-gmail-attachments-to-your-dropbox-with-this-add-on-1827879371) 直接整合。你可以打开 Dropbox，在通讯应用程序中共享文件，如 [Slack](https://slack.com/apps/A0F82E5R8-dropbox) 或 [Hootsuite](http://apps.hootsuite.com/92/dropbox) ，生产力应用程序，如 Readdle 的牛逼 [PDF 阅读器](https://readdle.com/) 或 [DocuSign 的应用程序](https://www.docusign.com/) ，连接其他服务的服务，如 [IFTTT](https://ifttt.com/dropbox) 或 [Zapier](https://zapier.com/apps/dropbox/integrations)

你可能会知道你是否可以将你的 Dropbox 链接到各种应用程序，但如果你想要一点帮助，只需仔细阅读 Dropbox 的 [可搜索图库](https://www.dropbox.com/business/app-integrations) 。

当然，还有很多其他服务允许你以的新颖有趣的方式使用 Dropbox。您可以创建 web 表单并 [直接在您的 Dropbox](https://www.jotform.com/dropbox/) 中收集答案，自动 [对您的 Dropbox](https://www.sortmybox.com/) 进行排序，自动 [将您需要的文件*复制到*您的 Dropbox](https://spotdox.com/) 中，并同步您想要的任何文件和文件夹，而不是将它们放入 Dropbox 文件夹中(在 [Mac](https://www.sebthedev.com/macdropany) 或 [Windows](http://satyadeepk.in/dropbox-folder-sync/)

在你疯狂使用第三方插件之前，请注意哪些应用程序可以完全访问你的 Dropbox 帐户。定期检查您的 [已连接的应用](https://www.dropbox.com/account/connected_apps) 并解除那些您不再使用的应用，并且不要授权看起来可疑的应用。同样，不要将包含你的社会安全号码、信用卡号码和其他重要信息的文本文件放入你的 Dropbox 进行安全保管——至少，如果你让许多其他服务访问你的数字保险箱，就不要这样做。
