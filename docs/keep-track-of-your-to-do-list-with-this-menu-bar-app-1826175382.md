# 使用这个菜单栏应用程序跟踪你的待办事项列表

> 原文：<https://lifehacker.com/keep-track-of-your-to-do-list-with-this-menu-bar-app-1826175382>

我通常在开始每一天的工作时，都会列出当天需要完成的事情。我曾经在 Notes 应用程序中做过的事情，最终变成了只使用笔记本中的一页。随着新东西的出现，我会添加到列表中，最终以一个混乱的列表结束，一些东西被划掉，另一些写在空白处，不可避免地，有一样东西不小心漏掉了。



本周，我偶然发现了一个菜单栏应用程序，它与我笔记本旁边的笔记本功能相同，但在处理工作方面要好得多。被称为 [思维列车](https://www.thoughttrain.cc/?ref=producthunt) ，它是一个让你记录你想做的事情的地方，这样当你完成其他任务时就不会忘记。

因为它使用你的菜单栏，所以无论你在做什么，它都很容易访问。要使用它，你只需点击页面顶部的梯形图标，然后输入。

您可以添加多个想法，存档和删除想法，锁定想法以备后用，只需点击一个复选标记即可删除已完成的内容。

当应用程序被最小化时，你需要做的任务也会在你的菜单栏顶部滚动，所以它们永远不会遥不可及。已删除的项目将被归档，您可以在以后查看。

这是一个简单的界面，但是如果你经常需要添加东西到待办事项列表中，这是一个非常有用的界面。
