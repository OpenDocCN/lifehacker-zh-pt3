# 三种配料的快乐时光:一个梨白兰地

> 原文：<https://lifehacker.com/3-ingredient-happy-hour-a-pear-brandytini-1820317491>

快乐的星期五，我最好的宝贝们，欢迎回到 [*三种配料的快乐时光*](http://lifehacker.com/tag/3-ingredient-happy-hour) ，每周饮品专栏，提供超级简单又美味的酒水。本周我们拥抱秋天的水果，同时用梨白兰地做的马提尼保持优雅。

Watch

从技术上来说，这种饮料只有两种成分，但我算上一条必要的柠檬皮，把它增加到三种。老实说，梨白兰地非常好喝——很明显，它有强烈的梨味——只需要一点点 Cocchi Americano 就可以把它变成一种复杂的鸡尾酒。作为火鸡开胃酒，或在餐后快速搅拌几杯帮助你放松。要做到这一点，你需要:

*   两盎司半梨子白兰地
*   3/4 盎司美国椰子
*   1 条柠檬皮

将两种烈酒倒入搅拌杯中，加满冰块。搅拌，直到完全冷却，滤入一个 coupe 玻璃杯，从杯中挤出果皮中的油。将皮放入杯中，尽情享用。
