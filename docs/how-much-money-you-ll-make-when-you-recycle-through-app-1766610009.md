*   **iPhone 5:**16GB、32GB 和 64GB 型号的 100 美元苹果礼品卡
*   **iPhone 5c:** $100 的苹果礼品卡，适用于 8GB、16GB 和 32GB 型号
*   **iPhone 5s:**16GB、32GB 和 64GB 型号的 150 美元苹果礼品卡
*   **iPhone 6:**16GB、64GB 和 128GB 型号的 250 美元苹果礼品卡
*   **iPhone 6 Plus:**16GB、64GB 和 128GB 型号的 300 美元苹果礼品卡

需要注意的是，如果你的设备不能正常通电，屏幕严重损坏，金属外壳弯曲或破损，被水损坏，或者任何按钮不起作用，你的退货率将会更低。此外，该网站没有提到任何关于雕刻模型的事情，所以这可能会影响你的设备的价值。如果你的设备真的很糟糕，他们可能根本不会给你提供任何东西。然而，他们确实免费提供环境安全处置。

苹果的 Renew 计划将你的旧 iPhones 和 iPads 视为里面材料的总和。它是什么颜色或者它有多少内存并不重要。对于购买基本型号的人来说，这是一个公平的交易，但对于那些为了更多内存而额外支付数百美元的人来说，这就不太好了。如果你试图出售 (并获得现金而不是苹果礼品卡)，你的设备可能会得到更多一点，但如果你想在苹果产品线中升级，这总体上不是一个坏交易。苹果还会 [拿走你的旧 iPods 和其他小玩意](http://oem.srsapp.com/applepoweredbysims/) ，但是没有礼品卡奖励。如果你想安全地处理你的 iPod，他们会免费，但你得到的只是妥善处理旧电子产品的满足感(这对一些人来说仍然很有价值)。然而，如果你把你的旧的非 Shuffle iPod 拿到苹果商店，你可以得到 10%的折扣。

### **苹果电脑和 Windows 电脑的优势**

你可能认为苹果会回收旧的 IMAC、Mac Minis 和 Macbooks，但你也可以回收 Windows 笔记本电脑和台式机。只要您的 Mac 或 Windows PC 是 2007 年之后制造的，它就有资格获得 Apple Renew。

市面上有很多不同型号和组合的苹果电脑，所以你最好去找你自己的专用电脑，比如 T2、T3、T4 和 T5。然而，给你一个大概的估计，你可以期待苹果礼品卡从 35 美元的旧 MacBook Core 2 Duo (2.0GHz)到 500 美元的 MacBook Air Core i7 (2.2GHz)到 720 美元的 13 英寸 MacBook Pro Core i7 (3.1 GHz)。对于台式机，你可以从 230 美元的 Mac Mini Core i7 (3.0 GHz)礼品卡到 800 美元的 27 英寸 iMac Core i7 (4.0 GHz)和 5K Retina 显示屏到 1500 美元的 Mac Pro (3.7 GHz) Ivy Bridge 四核。如果你是一个忠实的 Mac 用户，并且你不打算卖掉你的旧电脑，与苹果公司一起回收你的电脑似乎是一个很好的方式来支付你的升级费用。

然而，对于 [Windows PC 回收](http://reuserecycle.poweron.com/) 来说，你不会得到多少，即使机器工作得很好。例如，一台搭载英特尔酷睿 i7 (3.0-3.49GHz)处理器的 Windows 台式电脑只能为你赢得一张 75 美元的礼品卡，如果你包括电源线、工作键盘和鼠标、*和*的话，如果它有固态硬盘的话。光是处理器就价值近三倍(假设它还处于良好状态)。只有当你的 Windows 机器坏了，你不想尝试出售它的零件，并且你只是在寻找一种环保的方式来处理它时，与苹果一起回收你的 Windows 机器才是值得的。
