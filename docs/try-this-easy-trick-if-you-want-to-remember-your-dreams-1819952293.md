# 通过开始你的意图来记住你的梦想

> 原文:[https://life hacker . com/try-this-easy-trick-if-you-want-to-remember-you-dreams-1819952293](https://lifehacker.com/try-this-easy-trick-if-you-want-to-remember-your-dreams-1819952293)

在梦发生后记得你的梦，或者在做梦的时候意识到自己在梦里，这可能很有趣，但并不容易做到。

Watch

[*科学警报*](https://www.sciencealert.com/researchers-suggest-three-tips-for-lucid-dreams) 报道称，只有大约一半的人能够意识到自己正在做梦，只有四分之一的人声称自己经常做清醒梦。然而，如果你想记住你的梦，研究人员提出了一种可能对你有用的技术。他们的工作最近发表在 [*做梦*](http://psycnet.apa.org/doiLanding?doi=10.1037%2Fdrm0000059) 上。

研究人员观察了几种被认为有助于清醒梦的不同过程。该小组的领先者是一种叫做“温和”的技术(清醒梦的记忆诱导)。

这项技术背后的想法是，当你醒着的时候，重复这句话“下次我做梦的时候，我会记得我在做梦”。虽然 17%的参与者报告说当他们使用其他技术时会做清醒梦，但那些使用轻度的人报告说成功率高达 46%。

仍然有一个很好的机会，它不会工作，但在那里的选择，这是最好的赌注。

在你进入梦乡之前重复这句话，你就在你的头脑中创造了这样的意图，事实上，你会记住你的梦。研究人员说，这可能会导致清醒梦。

更好的是，那些使用这种技术的人报告说第二天没有感到更累，不像其他一些测试的技术，比如让自己在梦中醒来，会让参与者第二天感到累。

46%的成功率当然不能保证它对你有效，但是如果你在寻求清醒梦，它值得一试。
