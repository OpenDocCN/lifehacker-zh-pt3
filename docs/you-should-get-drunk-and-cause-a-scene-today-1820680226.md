# 你今天应该喝醉，然后大闹一场

> 原文：<https://lifehacker.com/you-should-get-drunk-and-cause-a-scene-today-1820680226>

你现在是在逃避你的家人吗？不顾一切地想要逃离它们，以至于你在感恩节坐在浴室里看生产力博客？害怕不可避免的关于福克斯新闻频道的争吵，或者你缺少一个伴侣，或者哪个孩子霸占了大学基金？这是你的解决办法:喝醉，然后大吵大闹。

无论如何，一场战斗将会发生。现在就看你的了。你可以选择话题，先发制人地用你会赢的战斗来预防你会输的战斗。或者你可以避开政治或宗教，转向一些个人的争论，为团队做出一个选择，以避免让每个人陷入存在主义的恐惧。

如果你先发制人，你也可以定下基调和策略。让事情变得大声，或者咬着牙小声说话，不时地把红酒泼在硬木地板上。(如果你是一个滴酒不沾的家庭中唯一的饮酒者，还有加分。)

如果你来自那种不吵架，而是把所有事情都藏起来好几年的家庭，也许是时候打开安全阀了。开始一场美好安全的战斗，让每个人都在某种受控的环境中表达自己的感受。

事情会不会失控？大概吧！但不管怎样，这都是会发生的，现在你给了这个家庭一些支持的理由:责备你。所以，趁你在圣诞节前还有时间和好，现在就消除误会吧。
