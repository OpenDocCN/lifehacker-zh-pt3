# 用这款 Chrome 扩展对抗假新闻

> 原文：<https://lifehacker.com/fight-fake-news-with-this-chrome-extension-1826810600>

**Chrome:** 同一个开源软件公司，想 [把蹩脚的广告挡在你的浏览器之外](https://lifehacker.com/how-to-stop-sites-from-harvesting-cryptocurrency-from-y-1819712943#_ga=2.150738550.1759341111.1528682319-396842925.1520800403) 也想让“假新闻”丰富你的生活。或者，至少，Eyeo 想让你知道你最喜欢的新闻网站是否充满了 FUD 和偏见。如果那是你喜欢的茶，那就喝一大口；如果没有，你可以继续前进，知道你没有被令人发指的声明和彻底的伪造所蒙骗。



Eyeo 的全新 [可信新闻扩展](https://chrome.google.com/webstore/detail/trusted-news-for-google-c/koejmcafidkcjlncgkpjfbijkhkpchei) ，目前是 Chrome 的测试版，它根据四个不同来源的评论评估你访问的新闻网站:

*   [元认证](https://metacert.com/)

*   [梅丽莎](https://docs.google.com/document/d/10eA5-mCZLSS4MQY5QGb5ewC3VAL6pLkT53V_81ZyitM/preview)

当你访问一个大多数可信新闻来源认为 [是合法新闻提供者](http://www.cc.com/video-clips/w9dj97/the-colbert-report-5-x-five---colbert-moments--stephen-cracks-up) 的网站时，你会看到一个绿色的小勾号。坚持读下去；你安全了。如果网站是 [公然讽刺](https://entertainment.theonion.com/jurassic-world-2-to-feature-more-scientifically-accur-1826798188) ，你会看到一个蓝色的笑脸。笑啊笑啊笑，但是不要把它的内容发布在脸书上，作为对你的朋友和爱人的真正警告。

根据可信新闻的分析，如果该网站“故意发布虚假和/或误导性信息”，它会得到一个红色的感叹号。不好。宣传带有政治偏见内容的网站会收到一个橙色的刻度图标，公然点击诱饵的网站会得到一个红色的靶心。充满恶意软件的网站，如果这是你喜欢访问的，得到一个红色的大“x”

当然，一个人的讽刺是另一个人的周日报纸，Eyeo 通信总监 Ben Williams 最近告诉 [TechCrunch](https://techcrunch.com/2018/06/13/adblock-plus-wants-to-use-blockchain-to-call-out-fake-news/) 该公司也希望将用户评级纳入分析:

> 所以最初几周我们要做的是加入一些用户可以通过扩展提供反馈的东西。他们可以争论一些事情。他们可以说“嘿，我觉得这个网站不应该被列为有偏见的网站，因为无论如何”。我们将利用这些反馈来改进产品。”
> 
> 下一步是将它与任何服务器和任何第三方分离，直接交给区块链。这样，反馈就可以在那个地方独立存在，好的反馈可以在用户中得到重视和奖励，而提供坏反馈的人则不会。所以这是下一步。”

是的，区块链。你没听说吗？每个人都想拥有自己的特殊货币，如果这种货币升值，他们就会成为百万富翁。而且，很明显，如果你提供了关于你喜欢的新闻网站的合法性的良好数据，你可能会赢得一些“”的元币，有朝一日这些元币可能会比 [一枚斯坦利镍币](https://www.youtube.com/watch?v=3IYQXUfezxY) 更值钱。

那么，可信新闻表现如何？CNN 获得绿色支票——对不起，川普总统——布莱巴特因“偏见”获得橙色等级 NPR 带着绿色支票得分很大，因为你还在做什么 [早上醒来](https://www.cartalk.com/) ，而福克斯新闻频道也带着绿色走开了。至于像“ [【愤怒的爱国者运动](http://www.angrypatriotmovement.com/) ”这样的网站，“你可能不需要分机告诉你分数，但你的朋友或亲人可能会。
