# 使用您的密码管理器获得安全答案

> 原文:[https://life hacker . com/use-your-password-manager-for-security-answers-too-1829498257](https://lifehacker.com/use-your-password-manager-for-security-answers-too-1829498257)

我已经有一段时间没有在新服务上创建账户时输入一些愚蠢的回答了。你知道我在说什么:忘记你的密码，输入你第一只宠物的名字(Mrglglrm 先生)，你最喜欢的运动队的名字(萨斯卡通塞壬)，或者你长大的街道(第三街)，你就可以重新进入你的账户。

Watch

如果你还没听说过，这种问答对安全性来说是可怕的，因为对于某人来说，找出这些答案比暴力破解复杂的密码要容易得多。

这个简单问题的显而易见的解决方案是，每当你被迫回答这样的问题时，就创造虚假的答案，但是有一个第二十二条军规:编造一个彻头彻尾的谎言，或者一些字母和数字的疯狂组合，当你最需要的时候，你可能会忘记你的虚假答案。最好的情况是，你必须与公司取得联系，请求重新获得你的账户；最坏的情况是，你没有办法验证这个账户是属于你的，那你就倒霉了。

以下是解决这个问题的几种方法，按有效性排序:

### 撒谎，但只是一点点

当一个服务要求你输入你的第一部音乐剧的名字作为账户安全问题时，你不必说实话。如果你小时候第一次看《歌剧魅影》，你总会说它是*汉密尔顿*。或者*希斯。*或者干脆不选音乐剧。跟*圣诞节前的噩梦*(其实应该是音乐剧，不过我跑题了)。

只要你能记住你的善意小谎言，别人就很难通过发现你在网上发布的会泄露手头问题实际答案的东西来侵入你的账户。

### 将您的问答视为密码提示

如果你想变得更疯狂一点，你总是可以用更有创意的方式来模糊你的答案。就拿凯特·科切特科娃的做法来说，从 [卡巴斯基博客](https://www.kaspersky.com/blog/security-questions-are-insecure/13004/) :

> 如果你愿意，你甚至可以把最糟糕的安全问题的答案改成没人能猜到的——你母亲的娘家姓是什么？XCU *(T2)s 1042！—但是，当然，你也需要小心不要把自己搞糊涂了。
> 
> 作为一个更好的选择，你可能会把娘家姓伍德豪斯去掉，改成辅音字母:wdhs。均匀穿插出生日期 04.08.80 得到 04wd08hs80。不算高明的把戏，但比原来的好多了。"

你现在甚至比以前更安全，因为你使用了一些模糊的数字和字母的组合，而不是字典可猜的名字。这并不能阻止 [强大的暴力攻击](https://arstechnica.com/information-technology/2013/05/how-crackers-make-minced-meat-out-of-your-passwords/) ，但它至少可以击败任何只是随意输入城市、昵称或其他任何答案*可能是*的人。

坏处呢？类似“J2uS*SD12(#..sfa！”很难记住。你最不应该做的就是把它写在某个地方——可以是你显示器上的便利贴，也可以是你桌面上的文本文件——除非你已经把你的答案列表放在了一个安全的地方。开始解决第三个问题！

### 使用密码管理器存储您的问答

是的，你的密码管理器不仅仅是密码。假设您的 LastPass 或 1Password 帐户使用强密码本身、双因素身份验证和任何其他技巧 [LastPass](https://blog.lastpass.com/2014/10/7-ways-to-make-your-lastpass-account-even-more-secure.html/) 或 [1Password](https://support.1password.com/account-security/) 进行保护，您也可以在其中存储帐户问题的答案。(没错，LastPass 和 1Password 之外还有 [很多其他选项](https://lifehacker.com/the-five-best-password-managers-5529133)；这些只是我们的最爱。)

如果你在[1 密码](https://1password.com/) 上，这个过程同样简单。将您的答案放入 [安全便笺](https://blog.agilebits.com/2015/11/20/secure-all-the-things-in-secure-notes/) ，或者只为任何网站条目创建 [自定义字段](https://support.1password.com/custom-fields/) ，并将您的帐户恢复 Q &留在那里。看起来会像这样:

使用密码管理器存储帐户安全问答的最大好处是，你甚至可以让这些应用程序为你创建答案。(一个“答案”毕竟只是另一个密码。)如果你这样做了，你可能需要冷静下来——例如，没有符号——如果你使用的网站或服务不允许你说你的第一辆车是“H0n $ @ @ $ $ 0RD”
