# 标准电子书是一个古腾堡项目，你将实际使用

> 原文：<https://lifehacker.com/standard-ebooks-is-a-gutenberg-project-you-ll-actually-1796273390>

当我为我破旧的 Kindle 或 iPad 买书时，我总是试图节省一些钱。我不是一个电子书迷，但我确实用它来翻阅我应该已经读过的经典书籍——那些在我早晚通勤时携带起来太笨重的书。幸运的是，像古登堡计划这样的地方有成千上万的免费书籍。只有一个问题:很多看起来很糟糕。

Watch

你可能和我一样，想要格式合适的电子书，或者封面不是蓝底白字的书。因此，志愿者推动的项目 [标准电子书](https://standardebooks.org/) 正在重新发布公共领域的电子书，以符合当今的电子书读者和标准。每本书的页面都有字数统计、阅读分数、书名提要，以及书本身的变更日志。

> 标准电子书从古登堡计划(Project Gutenberg)等来源获取电子书，使用精心设计的专业级风格指南对其进行格式化和排版，略微现代化，全面校对和纠正，然后构建它们以利用最先进的电子阅读器和浏览器技术。

古登堡计划中的普通书籍和标准电子书之间的差异是相当大的。首先，所有标准电子书都有美观的封面，而不是稀疏的文字封面。

像弯引号和长破折号这样的印刷怪癖得到了解决，所以电子书看起来像书，而不是文本文档。断字，脚注和适当格式的章节标记也出现了，所有这些都有助于使公共领域的书籍更容易进入现代。

也很容易将它们下载到您的移动设备上。每本书都有 epub、Kindle、Kobo 和新的“epub3”格式可供下载。在我的 iPad 上，在 Chrome 中打开一个 epub 文件会迫使我在打开之前下载该文件，而 Safari 则让我选择直接在 iBooks 中阅读。
