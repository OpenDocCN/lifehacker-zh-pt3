# 停止网飞的预告片，用《网飞》充实你的未完待续内容

> 原文：<https://lifehacker.com/stop-netflix-trailers-and-bump-up-your-to-be-continued-1827533105>

在你的笔记本电脑或台式电脑上观看《网飞》最烦人的一点是该服务喜欢自动为你播放预告片。我不反对我可能关心的电影或电视节目的有趣预告片。每个人都因为电影中他们不感兴趣的预告片而受苦——这只是游戏的名字而已。



我不喜欢预告片自动播放的时候，尤其是当我刚刚调好系统的音量，期待着 [播放我最喜欢的节目](https://www.netflix.com/title/80063224) 的时候。当我点击播放按钮时，我喜欢知道声音之墙正在到来；不管我什么时候打开网飞的主页，都不会被它击中，还会有一个分散注意力的视频。

值得庆幸的是， [Chrome](https://chrome.google.com/webstore/detail/netflix-tweaked/piocfidbkeehbojkamgamfhflpkoaifh/related) 和 [Firefox](https://addons.mozilla.org/en-US/firefox/addon/netflix-tweaked/) 扩展网飞调整阻止服务的预告片在你的浏览器中自动播放。这还不是全部。作为一个额外的奖励，这个扩展还会把你的“继续观看”和“我的列表”放在顶部服务特色内容下面的第一和第二个位置。你不必滚动浏览任何网飞的促销活动，就能调出你正在看或想看的节目。

现在，如果有人能找到在我的电视上工作的方法就好了...
