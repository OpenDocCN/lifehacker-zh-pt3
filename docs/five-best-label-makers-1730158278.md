那些提名 PT-1230PC 的人称赞它像手持式标签制作机一样易于使用和携带，而且还让您可以选择将其连接到 PC，并使用 Windows 中可用的字体、语言、标点和符号，而不会限制您使用标签制作机已经知道的任何东西。你提到你用它来标记电缆、电脑、储物箱等等，它就在你的电脑显示器旁边，就在 USB 端口旁边。您还指出，您经常使用您的产品打印条形码，用于资产跟踪或组织，这是我们的综述中没有任何其他产品支持的一个好处。更多 [可以在](http://lifehacker.com/vote-brother-p-touch-pt-1230pc-why-cheap-reliable-a-1729791616) 的提名帖中阅读。
