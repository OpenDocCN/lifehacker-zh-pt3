# 如何不使用 Chrome 商店继续下载 Chrome 扩展

> 原文：<https://lifehacker.com/how-you-can-still-download-chrome-extensions-without-us-1826796797>

我不记得上一次我没有安装谷歌 Chrome 网络商店的扩展是什么时候了。然而，到目前为止，开发者已经被允许以内嵌下载的方式提供他们的扩展。换句话说，他们可以在网站上放下一个下载按钮，你点击它，就会看到一个典型的安装确认对话框(就好像你正在从 Chrome 网络商店安装扩展一样)，在你知道之前，你已经+1 到扩展了。



### 再见，在线安装

众所周知，这种 [直列安装功能即将消失](https://developer.chrome.com/webstore/inline_installation) ，因为我们无法拥有美好的事物。使用内嵌安装，开发者可以在他们的网站上以任何方式谈论一个扩展。想必，你不会太关注你从 Chrome 网上商店获得的扩展的实际描述。这可能会导致一点混乱，在最好的情况下，或恶意，在最坏的情况下，如果一个开发人员想拉 ol' swippy-swappy 和欺骗你安装一些可疑的东西。正如谷歌的[所写的](https://blog.chromium.org/2018/06/improving-extension-transparency-for.html) :

> *...我们继续收到大量来自用户的投诉，关于不必要的扩展导致他们的 Chrome 体验意外改变——这些投诉的大部分归因于网站上混淆或欺骗性的内联安装使用。在过去几年里，我们一直试图解决这个问题，我们了解到，Chrome 网络商店中显示在扩展旁边的信息在确保用户能够做出是否安装扩展的明智决定方面发挥着关键作用。与通过内嵌安装安装的扩展相比，通过 Chrome 网络商店安装时，扩展被卸载或引起用户投诉的可能性要小得多。”*

从今天起，开发者将不被允许为新发布的扩展提供内嵌安装。现有的扩展将从 9 月 12 日开始禁用内联安装，谷歌将在 2018 年 12 月初 Chrome 71 上市时取消整个内联安装 API。

### **谷歌的改变对你意味着什么**

如果你访问一个 chrome 扩展开发者的网站，发现你无法点击一个下载按钮并安装一个扩展，谷歌的调整很可能是罪魁祸首。在你放弃这个扩展之前，尝试手动搜索 Chrome 网络商店，看看这个扩展是否存在——应该是，除非开发者已经明确表示，这是一个你应该下载到你的浏览器中的扩展。(你真的不应该这样做，除非没有其他方法来获得特定的扩展，你必须有*和*验证该扩展不是伪装的恶意软件。)

### 什么是侧装？

例如，如果谷歌从 Chrome 网上商店下载了你最喜欢的扩展，而你并不关心安装一个“失控”的扩展所涉及的安全问题，那么侧装这个扩展可以让你绕过谷歌的安装限制。

首先，在地址栏中输入`chrome://extensions/`,然后点击右上角的“开发者模式”按钮。

拿分机的。你先前下载的 CRX 文件——假设你有一个——并使用类似于 [CRX 提取器](http://crxextractor.com/) 的服务将其内容转换成一个简单的 ZIP 文件。在你的台式机或笔记本电脑上解压这个文件，然后点击`chrome://extensions/`窗口中的“Load Unpacked ”,找到解压文件的文件夹。除非出现任何错误，否则扩展应该会直接进入 Chrome。

如果你不知道去哪里找？CRX 文件为扩展名，你可以查看曾经有用的 [Chrome 扩展名档案](https://crx.dam.io/) 。从这里下载一个. ZIP 文件，将其重命名为。CRX，并通过上述过程手动将其放入您的浏览器。是的，你仍然需要使用 CRX 提取器创建另一个 ZIP 文件，并将该文件夹的内容导入到你的浏览器中。你会没事的。

(不过，实际上，你应该尽可能使用 Chrome 网络商店进行扩展。恶意软件 [可以溜进](https://blog.trendmicro.com/trendlabs-security-intelligence/malicious-chrome-extensions-found-chrome-web-store-form-droidclub-botnet/) ，但至少谷歌对检查并移除它保持警惕。)
