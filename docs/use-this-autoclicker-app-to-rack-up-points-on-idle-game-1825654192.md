### 让应用程序帮你点击

当你不坐在电脑前时，自动操作鼠标的工具已经存在了几十年。当我还是个孩子的时候，我用它来欺骗那些“在你的系统上安装这个永久的横幅，当你使用你的电脑时，我们会付钱给你”的公司。谢谢你免费的 100 美元。

我真的没有发现通过一个无聊的点击游戏作弊有什么不对，因为你只是真正影响了你自己的体验。如果你想花几周时间而不是几个月(或一年)来尝试解开游戏中的每一项成就——这是一项只有你知道的壮举——那么尽一切办法作弊吧。最棘手的部分是在“我正在自动化一个闲置的游戏，使它变得更容易，但仍然有趣”和“我刚刚在十分钟内点击了一百万次，现在我没有其他事情可做”之间找到正确的平衡

当你思考这个问题的时候，去抓取免费的 [自动舔食者](https://sourceforge.net/projects/orphamielautoclicker/) 工具。这是一个单一的 Windows 可执行文件，无需安装。把它放到一个文件夹中，双击它，你就可以设置点击间隔，从毫秒到小时不等。该应用程序可以自动触发鼠标左键、中键或右键的点击，你甚至可以疯狂地让每次点击都变成双击(如果你真的想浏览你的游戏)。

该应用程序默认使用 F6 热键来启动和停止自动点击，但你可以随意更改。您也可以让应用程序一直点击，直到您停止它，或者您可以让它在设定的点击次数后自动停止。

这是一个非常简单的任务的大量定制:无意识的快速点击，这样你就不必浪费时间在一个主要设计为在后台运行的游戏中。尽管如此，我还是无法想象没有它玩这些游戏。
