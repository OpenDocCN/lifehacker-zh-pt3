# 本周 Windows 应用:Steam Mover

> 原文：<https://lifehacker.com/windows-app-of-the-week-steam-mover-1824285685>

如果你已经在 Valve 的蒸汽服务上玩了很长时间，你可能听说过 [蒸汽推进器](http://www.traynier.com/software/steammover) 。这是一个很好的工具，可以将你的几千兆字节的 Steam 游戏转移到你系统上的不同硬盘上，以防你的主硬盘空间不足(或爆满)。



我们喜欢 Steam Mover 的原因是，它的用途不仅限于 Steam 游戏，上次我们谈到这款应用时， [还简单提到了它](https://lifehacker.com/how-to-move-a-pc-game-to-another-hard-drive-without-re-1714706774) 。哦，不。该应用程序可以在你的系统上为你想要的任何文件夹创建 [符号链接](http://lifehacker.com/how-to-use-symlinks-in-windows-5496652#_ga=1.44292192.515970378.1434646157)；你不仅仅局限于“steamapps”。

### 符号链接，用简单的英语

符号链接有点像路标。它告诉 Windows 文件夹 X 的内容实际上在文件夹 Y 中，尽管我们出于一些原因要保留文件夹 X。

例如，假设您在 Windows 的标准“图片”文件夹中有一个 RAW 照片的存档文件夹。RAW 图片占用了大量空间，您想将它们移到其他地方。然而，你不希望总是在你选择的照片编辑工具的硬盘之间来回切换。当然，你可以用捷径来解决后者。符号链接允许您将原始照片移动到辅助硬盘，但您仍然可以通过导航到“图片”中的原始归档文件夹来访问它们这些文件并不真的在那里，但是它们看起来好像在那里。

显然，如果您需要移动应用程序，或者移动在应用程序的主安装文件夹中占用大量空间的子文件夹，但希望保持原始应用程序目录结构不变，符号链接非常有用。

你可以使用其他的窗口工具来自动创建这些符号链接，但是 Steam Mover 非常有用，因为它的图形界面向你展示了一个文件夹真正所在的位置——并且给你一个方便的“点击箭头”设置来来回移动它到它的原始(或新的)位置。
