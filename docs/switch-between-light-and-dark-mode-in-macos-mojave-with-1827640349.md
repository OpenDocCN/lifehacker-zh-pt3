# 使用这个方便的应用程序，在 macOS Mojave 的亮暗模式之间切换

> 原文：<https://lifehacker.com/switch-between-light-and-dark-mode-in-macos-mojave-with-1827640349>

Mac: 尽管你可能不应该在你的主要设备上安装一个操作系统的公开测试版——在这种情况下是你的笔记本电脑——但还是继续吧。 [给马科斯莫哈韦试试](https://lifehacker.com/how-to-get-apples-mac-iphone-watch-and-tv-betas-witho-1826578952) 。有很多有趣的功能可以用 来玩，包括操作系统新的黑暗模式(接近我的列表顶部)。



但是黑暗模式并不适合所有人。也许你总是更喜欢叛军而不是帝国；阴对阳；闪亮的 [苹果银](https://www.apple.com/shop/product/MQ052LL/A/magic-keyboard-with-numeric-keypad-us-english-silver?fnode=56) 到稍贵的 [太空灰](https://www.apple.com/shop/product/MQ052LL/A/magic-keyboard-with-numeric-keypad-us-english-silver?fnode=56) 。你甚至可能在不同的时间喜欢黑暗和光明两种模式:一种是当你在深夜躲在你的书呆子窝里的时候，另一种是当你试图在白天完成工作的时候。

苹果已经使 macOS Mojave 中的亮暗模式之间的切换变得相对容易——只需点击左上角的苹果图标，点击系统偏好设置，点击通用，然后切换到你最喜欢的模式。

不过，这工作量很大。

相反，我喜欢小菜单栏实用程序。将它安装在你的系统上，每当你启动 macOS 时运行它，你会得到一个小图标——当然是一只猫头鹰——你可以点击它在亮暗模式之间切换。你甚至可以设置它，让在你切换的时候发出可爱的小猫头鹰的声音。

这是目前该工具所做的全部工作，但是 NightOwl 的开发者笔记表明未来的版本可能会触发日出和日落的不同模式。
