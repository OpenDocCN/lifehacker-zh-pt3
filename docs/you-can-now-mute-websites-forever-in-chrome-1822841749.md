# 你现在可以在 Chrome 中永远屏蔽网站

> 原文：<https://lifehacker.com/you-can-now-mute-websites-forever-in-chrome-1822841749>

休息吧，我的孩子。CNN 的自动播放视频不能再伤害你了。在最新的 Chrome 公开版中，你可以右击任何标签，选择“屏蔽网站”

Watch

即使网站目前没有任何噪音，你也可以这样做。所以，去吧，先发制人地屏蔽掉所有你想屏蔽的网站。 [Lifehacker 撰稿人](https://kinja.com/timdonz) 蒂姆·唐纳利演示:

 [https://lifehacker.com/embed/inset/iframe?id=twitter-961651391851843587&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-961651391851843587&autosize=1) 

Chrome 过去允许用户屏蔽标签，但它不会记住用户一直想屏蔽的网站。正如我们 [在 12 月](https://lifehacker.com/now-google-chrome-can-automatically-silence-annoying-au-1821369846) 报道的那样，Chrome 在 Chrome 64 的测试版中引入了自动播放静音功能。但现在这一改变将扩展到我们所有使用 Chrome 正常公开发行版的普通人。(如果您仍想使选项卡静音，您可以重新打开该功能；将向下滚动一半[这篇文章就变成了](https://www.makeuseof.com/tag/mute-entire-websites-chrome/)的。)

CNN 知道，在内心深处，你只是想阅读一篇新闻文章，而不是观看概括文章的视频。但是，和其他网站一样，CNN 从视频广告中赚的钱要比从一页文字上的静态广告中赚的多得多。所以他们只是立即播放视频，不需要你的同意，希望广告会在你最终阻止它之前播放。

网站会做些傻事来强迫你看视频广告。他们甚至可能在一篇关于如何静音的文章中间嵌入一段视频！真是个世界！
