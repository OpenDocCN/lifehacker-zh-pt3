# 这些是让你无法获得面试机会的求职信错误

> 原文：<https://lifehacker.com/these-are-the-cover-letter-mistakes-that-keep-you-from-1820646903>

我们都有在网上申请工作的经历——你把简历和求职信提交到某个在线招聘门户网站的黑暗深渊，除了自动生成的“感谢申请！”电子邮件，你再也听不到雇主的任何声音。

“他们收到了上百万份申请，别往心里去，”你的朋友告诉你。是的，这是真的，他们可能确实收到了一百万份申请，这就是为什么有那么多关于如何 [写](http://lifehacker.com/how-to-write-a-cover-letter-that-employers-will-actuall-5880545) [封面](http://lifehacker.com/write-your-cover-letters-like-your-job-depends-on-it-b-479770619) [字母](http://lifehacker.com/write-your-cover-letter-like-an-acceptance-speech-1587406177) 和工艺[视觉上吸引人的](http://lifehacker.com/this-tool-helps-make-your-resume-visually-appealing-1709672920)[行动导向的](http://lifehacker.com/highlight-your-impact-not-your-tasks-for-a-better-res-1780559134) [简历](http://lifehacker.com/improve-your-resume-with-this-simple-formula-recommende-1566107526) 让你脱颖而出的文章。

但是，你在求职信中犯的一些很小的、很容易改正的错误往往会让你在别人看到你的证书之前就被拒绝。我们采访了科技、媒体、广告和学术界等行业的数十位高管和招聘经理，了解他们在求职信中最大的烦恼。

### 你有打字错误或语法错误(尤其是当工作涉及写作时)。

拼写检查。语法检查。大声朗读这封信。你会惊讶地发现有多少错误是这样的。“对于一些职位来说，一个小小的打字错误就足以让你被拒绝，”哈尼施基金会主席露丝·安·哈尼施说。

### 你使用了过时的或性别歧视的问候语(例如，“亲爱的先生”)。

现在是 2017 年。如果你倾向于写“亲爱的先生”，甚至“亲爱的先生或女士”，我们建议你重新考虑。你信的另一端的人不是“先生”的可能性很大；他们既不是先生也不是女士，或者他们反对被这样称呼，这也是很有可能的。

“致相关人员”作品；“致[公司名称]的招聘团队”很好。如果你能找到人力资源代表或招聘经理的名字，那就更好了——“亲爱的 Kirsch 女士和 Lifehacker 的招聘人员”表明你做了功课。

### 你打招呼太随便了(例如，“嗨”，“你好”，“嘿！”).

就像你不想太正式一样，你也不想以穿着脏运动服参加面试的方式打开你的信。这不是一个容易引起争议的信息，这是一份你表面上想要的工作的求职信。

### 你一点问候都没有。

有人在读这些信(直到机器人抢走了我们所有的工作)，所以抓住每一个机会和这些人说话。一封没有问候的信看起来像是普通应用程序上的个人声明——也就是说，一个无聊的、形式上的练习，没有人想看。

### 你没有提到你申请的公司的名字。

这个出现了很多次。个性化你的求职信！是的，这需要额外的一点时间来显示你确实看了一下你申请的公司和职位，但是这是必要的。“我写这封信是为了申请[公司]的[职位]。然后具体说一下为什么你非常适合这个职位，为什么你渴望为这个公司工作。

Racked 的招聘编辑妮莎·奇塔尔表示:“笼统提及“职位”且从不提及公司名称、看起来像是被复制粘贴并发送给 1000 家随机公司的求职信”是不可取的。"告诉我你为什么关心我的公司和这个职位."

经理在招聘时考虑的因素之一是你有多想要这份工作。你不需要乞求，但是看起来对这个职位完全不感兴趣绝对没有任何好处。

### 很明显，你只是把你申请的公司的名字换成了另一家公司的名字，但除此之外，这封信是没有人情味的。

见上文。他们识破了你的诡计。

### 你似乎并不热衷于为这家公司工作。

当招聘经理评估你的候选资格时，你的求职信是他们首先看到的。不要欲擒故纵！不要表现出不在乎得到这份工作。你想要！可以这么说！不像，比如说，邀请某人参加冬季舞会，表现出想要这样做的热情并不是一件可怕的事情。

### 你没有把你目前的经验转化到你申请的工作中。

“如果你在申请一个编辑职位，但你过去的所有经验都是在市场营销方面，而你没有在求职信中解释为什么你有资格转型，这是一个非常明显的拒绝，”缪斯的自由编辑阿德里安·格兰塞拉·拉森说。如果你简历上的经历与职位没有任何关系，求职信就是解释这一点的地方。

### 你对公司做了一些不真实的假设(例如，“我知道你的公司经历了艰难的一年，正在进行重组…”)。

表明你已经做了调查，但是要小心鹦鹉学舌，不要重复你在交易中读到的关于该公司的事情，这些事情可能是真的，也可能不是真的——尤其是如果它不是特别讨人喜欢的话。

### 你说你现任或前任雇主的坏话(这也适用于面试)。

即使你非常想摆脱目前的工作，也不要抱怨。新公司不在乎你以前的工作困境；他们想要一个态度积极的人加入他们的团队。

### 你把公司名称弄错了，或者拼错了。

是的，你申请的每一份工作确实都符合你的要求，但如果你说“我认为我会是福特汽车公司的一笔巨大财富”，并且你申请在特斯拉工作，特斯拉的人力资源人员已经在寻找下一个候选人了。

### 你拼错了招聘经理的名字。

最好用一个通用的“致相关人员”来代替。

### 你说这份工作会是你的“敲门砖”。

仅仅因为它是真实的并不意味着你这样说。任何传达出“我不会长期干下去”、“我对你来说太好了”或“这份工作远远低于我，我只是在申请福利”的信息都是一个坏主意。
