# 每种变速器的最差驾驶习惯，解释如下

> 原文：<https://jalopnik.com/the-worst-driving-habits-with-each-type-of-transmission-1785113367>

这里有一个有趣的分类，说明坏习惯是如何损坏你的汽车的，以及它们是如何根据你驾驶的是手动、自动、CVT 还是双离合变速器而变化的。



杰森·芬斯克在他的 YouTube 频道 [*上做了一项伟大的工作，他将汽车的各个部件如何工作概念化，工程解释*](https://www.youtube.com/channel/UClqhvGmHcvWL9w3R48t9QXQ) *。*在这一年中，他发布了具体的视频，分解了不同类型的变速箱不要做什么(任何为什么)。现在他发布了一个关于 CVT 的，我们有完整的一套！

剧透警告，要点通常是:不要在空档滑行，不要不停地改变方向，不要半挂档——不管你开的是哪种车。

但是更好的地方是更细微的，对为什么应该避免一些坏习惯的解释总是很有趣。

### **手动**

 [https://jalopnik.com/embed/inset/iframe?id=youtube-video-_cbZlhduYJY&start=0](https://jalopnik.com/embed/inset/iframe?id=youtube-video-_cbZlhduYJY&start=0) 

芬斯克关于手册的主要声明是“不要把你的手放在换挡杆上。”他认为，你可能会导致啮合齿轮的小拨叉过早磨损。

他还建议在红灯时将汽车置于空挡，而不是紧握离合器，以延长分离轴承的寿命。

### **自动**

 [https://jalopnik.com/embed/inset/iframe?id=youtube-video-6zzEtxJkC7Y&start=0](https://jalopnik.com/embed/inset/iframe?id=youtube-video-6zzEtxJkC7Y&start=0) 

几乎只有在被拖走的时候，自动档才应该挂空挡。在运动时将它从档位上拿掉会释放对汽车的显著控制，额外的换档会造成不必要的磨损，没有任何好处。

你可能已经知道在你开车的时候不应该把自动档放在驻车档，但是芬斯克给了我们一个白板图表来解释为什么不应该这样做。

### **双离合**

 [https://jalopnik.com/embed/inset/iframe?id=youtube-video-W42zsFYjZvM&start=0](https://jalopnik.com/embed/inset/iframe?id=youtube-video-W42zsFYjZvM&start=0) 

像大众 GTI、福特福克斯和一系列高性能汽车上的双离合变速箱与传统的自动变速箱有所不同。他们也需要区别对待。

芬斯克说，不要让汽车在斜坡上“盘旋”而不踩刹车，因为与“常规”自动变速器不同，双离合器的离合器组件会旋转并接触不必要的热量。

显然双离合器也不喜欢“缓慢前进”，尤其是在牵引或爬陡坡时。由于离合器在低速时只是部分接合，所以磨损会更快。

### 无级变速器

 [https://jalopnik.com/embed/inset/iframe?id=youtube-video-HSxg97m8lLA&start=0](https://jalopnik.com/embed/inset/iframe?id=youtube-video-HSxg97m8lLA&start=0) 

基本上有两种类型的 CVT——“传统型”通过一个滑轮系统将动力从发动机转移到车轮，而“E-CVT”则是混合动力汽车中的一种，在混合动力汽车中增加了一个电动机。

自动挡的大部分驱动逻辑都可以应用到 CVT 上。不要在空挡时将车加速并挂入前进档，不要在不必要的时候挂空挡。

希望你在看完这些之后没有意识到你一直都在错误地驾驶，但是我喜欢芬斯克如何变得足够技术性来解释汽车发动机和车轮之间发生的事情。
