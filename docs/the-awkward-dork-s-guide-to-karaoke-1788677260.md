# 笨拙的呆子卡拉 ok 指南

> 原文：<https://lifehacker.com/the-awkward-dork-s-guide-to-karaoke-1788677260>

如果你从未唱过卡拉 ok，这可能看起来像一场噩梦。你站在一群随机的人面前，一个人唱着歌，祈祷带着歌词的显示器不要熄灭。但是你的很多恐惧都是没有根据的，你可以做很多事情来让自己放松到聚光灯下。

### **第一次和朋友租卡拉 ok 包厢**

卡拉 ok 最好和朋友一起在提供包房的场所进行——有时被称为“ [卡拉 ok 包厢](https://en.wikipedia.org/wiki/Karaoke_box)”——特别是如果你以前从未做过或者你只是为朋友加油。赌注较低，因为唯一能听到你的人是朋友，较小的房间鼓励集体唱歌。最好的情况是:你表演了一场精彩绝伦的“神奇墙”，你的朋友们与你击掌庆祝。最糟糕的情况是:当你开始看起来不舒服时，你的朋友会为你加油，并帮助你。卡拉 ok 的意义在于娱乐，而不是看谁是最好的歌手。

如果你所在的地区没有卡拉 ok 包厢，你可能还有其他选择。查看当地酒吧的预定活动，并尽量去工作日有卡拉 ok 之夜的较小地方。很有可能最后只有你和你的船员。然而，无论你做什么，第一次都要和朋友一起去。同事可能会邀请你出去，但是你的第一次真正的外出应该是和你觉得放松的人一起。

### **不走第一，但不走最后**

除非你是卡拉 ok 老手，否则让别人来开球吧。看着别人走了，没有被人拿着火把和干草叉追出去，你会感觉舒服一点。此外，你将有时间喝一两杯，放松一下。但给你个忠告，唱歌前不要喝醉了。太多的液体勇气可能会帮助你在那里玩得开心，但相信我，你将是唯一一个开心的人。

不过，一定要早点选好歌曲。第一个走可能很可怕，但最后一个走可能更可怕。你不想给自己太多时间思考。你的神经会接管，你会陷入几个小时浏览书籍寻找完美的歌曲，在你知道它之前，夜晚会结束。瞄准中间。

### 坚持听你已经烂熟于心的歌曲

你可能犯的最严重的错误是选择一首你喜欢的歌，而不是一首你知道的歌。很多唱卡拉 ok 紧张的人害怕搞砸。因此，如果这听起来像你，严格挑选歌词你知道至少 90%没有任何帮助的歌曲。如果是说唱歌曲或者其他快节奏的曲子，就更是如此。是的，歌词就在你面前，但它们仍然可以让你措手不及。

当你选择一首歌时，总是问自己“我知道这首歌吗？还是我只知道*合唱*？”如果你不确定，在你的智能手机上调出歌词来检查一下。如果你相当确定你知道这首歌，问问自己“我能在合理的范围内唱这首歌吗？”你不需要知道一堆音乐或者唱歌的术语，只要回想一下，试着回忆一下你以前有没有唱过。如果你不费力就做到了，那就去做吧。请记住，这与技巧和你能给人留下多么深刻的印象无关，而是让你自己对麦克风感到舒服。

如果你有几首喜欢的歌，你知道你可以不需要任何帮助就能唱，制作一个卡拉 ok 播放列表来听和练习。您可以在淋浴或汽车上创作您最喜欢的歌曲，并且您可以在找到更多歌曲时将它们添加到播放列表中。最终，下次你出门时，你会有一大堆曲目可以选择。此外，当你喝了几杯酒，想决定下一首唱什么的时候，这个播放列表是一个很好的参考。

### **以一首团体歌曲开始，然后接着是一首取悦大众的歌曲**

没有什么比和一个伙伴一起唱歌更不尴尬的了。找一首多声部的歌，问几个朋友是否想和你一起唱。或者选择一首你和你的朋友喜欢的歌曲，并共享麦克风。只是要小心像奎恩的《波西米亚狂想曲》这样的歌曲这是一首组合歌曲，但是很难唱，而且很长，所以在你开始唱之前要确保每个人都在。

现在，酒已经开始了，你已经用一首团体歌曲消除了你的恐惧，为你的第一首独唱选择一个乐观的观众。不管怎样，人们会和你一起唱，你甚至会得到一些伴舞。它会让你感觉很棒，即使这首歌本身是所有的重担。如果你不喜欢乐观的流行音乐，那就尽可能找最接近的。你可能想唱那首缓慢、忧郁的民谣，但它会让情绪低落下来，把注意力放在你自己和你的声音上。

最好避免那些很长的歌曲，歌词重复的歌曲，大部分是音乐中断的歌曲，或者没人知道的歌曲。我的经验法则是:如果你对小众曲调竟然出现在卡拉 ok 书中感到惊讶，那就把它留到晚上大家都有机会去的时候吧。

### **成为一名出色的观众**

如果你为你的朋友欢呼鼓掌，他们会更愿意为你做同样的事情。你的朋友可能正在唱迈克尔·杰克逊的歌，但是听着，跟着节拍拍手，跳舞，微笑，当他们唱完的时候给他们一点掌声。

不要嘘或开玩笑，即使你是好意。他们可能知道自己并不完美——和你一样——但是卡拉 ok 就是要站起来，去做，因为它让你快乐。所以，欢呼吧，笑吧，保持这种积极的氛围。当轮到你唱歌的时候，你的朋友也会同样礼貌地对待你。当有疑问的时候，一笑而过，尽情享受。
