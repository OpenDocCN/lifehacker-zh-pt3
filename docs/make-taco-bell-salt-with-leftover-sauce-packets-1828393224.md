# 用剩下的酱包做塔可钟盐

> 原文：<https://lifehacker.com/make-taco-bell-salt-with-leftover-sauce-packets-1828393224>

我有一点可耻的快餐习惯，这不仅让我保持脚踏实地和谦逊，而且让我的包抽屉里只存放最好的单一服务调味品。麦当劳对他们的糖醋酱很吝啬，但塔可钟对他们的辣酱非常慷慨，我有很多。



*这是《与克莱尔一起吃垃圾》的一部分* [*，这是一个生活黑客系列，克莱尔·劳尔说服你将你的厨余转化为可食用的美味。*](http://lifehacker.com/tag/eating-trash-with-claire)

火过盛！酱汁并不完全是一个“问题”，但调味品确实有一种特殊的味道，最适合——如果不是塔可钟的话——垃圾食品谱中接近“塔可钟”的食物，如薯条和爆米花。因为在薯条和爆米花上浇酱会使它们变湿，自然的解决方法是脱水并做成塔可钟盐。

像任何用湿调味剂制成的调味盐一样，这个过程非常简单。简单地将酱汁和盐混合，在烤箱中脱水，然后粉碎。要做塔可钟盐，你需要:

*   6-10 包酱，取决于你喜欢盐的味道有多浓。六威尔给了你一种带有塔可钟香味的调料，但是你需要更多的调料来加热。
*   1/4 杯普通食盐

在食品加工机中混合调味汁和盐，直到你得到颜色均匀的沙状混合物。将盐摊在铺有羊皮纸的烤盘上，放入 200 度的烤箱中至少两个小时，直到盐完全干透。把它放回食品加工机，粉碎，然后把你最喜欢的垃圾食品的精华撒在松脆的零食上。
