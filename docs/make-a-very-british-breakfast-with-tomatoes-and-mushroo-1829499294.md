# 用西红柿和蘑菇做一顿非常英式的早餐

> 原文：<https://lifehacker.com/make-a-very-british-breakfast-with-tomatoes-and-mushroo-1829499294>

美国人非常擅长早餐和早午餐，但地道的英国人却无人可比。超越 [炸面包](https://skillet.lifehacker.com/fried-bread-is-better-than-toast-1827629232)——这比烤面包好得多——加入富含鲜味的蘑菇和甜的，顺便对你有益的西红柿，使它成为一顿不可忽视的饭。

我喜欢以沾沾自喜的满足感开始我的一天，蘑菇和西红柿是一种很好的方式来放松和(健康地)多样化经典的培根和鸡蛋系列。另外，这里有很多不同的方法可以让你做好准备。

### 在炉子上

在你完成熏肉和/或香肠的烹饪后(和在油锅里炸你的面包 )，你可以在同一个锅里加入一点黄油，然后炸蘑菇直到它们变成金黄色并且边缘酥脆。(任何蘑菇都是很好的选择，从普通的白蘑菇到羽毛状的鸡油菌。)然后，拿一个切成两半的番茄，用盐和胡椒调味，然后把它切面朝下放入锅中。煮几分钟，翻转两半，再煮几分钟，直到番茄变软，轻微焦糖化，但仍保持形状。

### 在烤箱里

烤蔬菜可能看起来不像典型的早晨活动，但它应该是，尤其是在寒冷、悠闲的早晨。只需扔几把小( [或撕碎的](https://skillet.lifehacker.com/tear-mushrooms-for-a-better-saute-1826197453) )蘑菇和等量的切半的樱桃番茄，加入橄榄油、盐和胡椒，然后在单层平底锅上以 375℉的温度烤半小时。用少许麦芽醋调味，与鸡蛋和烤面包一起享用。如果你想在繁忙的工作日享受这种舒适的一面，在周末烤一大堆，需要时在烤箱里重新加热。

### 在沙拉里

有点令人惊讶的是，我是早餐沙拉的超级冠军，腌制的松脆蘑菇在一碗安特子午线绿叶蔬菜中非常美味。只需 [将它们浸泡在美味的调味汁或盐水中大约半个小时](https://skillet.lifehacker.com/treat-mushrooms-like-meat-to-give-them-more-flavor-1829304093) ，然后油炸至它们非常脆。放在你的沙拉上面，还有一个荷包蛋和油炸面包丁。至于西红柿，就在这里生着吃吧。生的樱桃番茄一年四季都很好吃，而且它们几乎不需要任何准备工作。(除非你决定走 [去皮路线](https://skillet.lifehacker.com/how-to-make-cheap-side-dishes-that-seem-fancy-1825900642) ，我不会因此而责怪你。)
