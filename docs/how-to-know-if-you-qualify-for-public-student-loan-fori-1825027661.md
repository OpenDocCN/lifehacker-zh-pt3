# 如何确定你是否有资格获得公共服务学生贷款减免

本周的问题来自布莱恩的电子邮件:

> 我正在还款，我写信是想看看有什么法律可以保护借款人？这些年来，我的贷款被转让给了几个服务商，而我目前的服务商只给了我三年(公共服务贷款豁免)中的六个月的贷款，并花了一年时间来“重新调查”我的支付历史。我计划向[消费者金融保护局]和我的州检察长投诉，但不知道我能指出什么，从法律上来说，以加强我的主张。

这是个别专家对一个对每个人都有不同影响的问题的一般看法——如果你想要个性化的建议，你应该去见理财规划师。

### 确保你确实在正轨上

这是非常令人沮丧的，但不幸的是，太常见的事情。事实上，你的许多电子邮件和评论只是询问如何确保学生贷款服务机构正确跟踪你的信息。

明天我们将深入了解公共服务贷款减免计划的细节。今天，回答你的问题，不幸的是，就联邦法律而言，对消费者的保护并不多。

虽然你可以先提出投诉(稍后再详细说明):但你真的需要反复检查你是否真的在 PSLF 注册并符合严格的条件:

*   你有联邦直贷(如果你不知道你在什么还款计划，登录 [我的联邦助学](https://studentaid.ed.gov/sa/?login=true) )。民间借贷不可原谅。如果你有联邦贷款，你可以在 [国家学生贷款数据系统的网站](https://www.nslds.ed.gov/nslds/nslds_SA/) 上看到它们的列表以及你的服务者是谁。
*   符合条件的还款计划包括 [基于收益的还款计划](https://studentaid.ed.gov/sa/repay-loans/understand/plans/income-driven) 。
*   你已经在直接贷款计划下支付了 120 次合格的月付款(不需要连续支付)。您可以在这里 看到您已经 [支付了多少次符合条件的付款。](https://myfedloan.org/)
*   你必须为所有 120 笔付款全职工作( [意为](https://studentaid.ed.gov/sa/repay-loans/forgiveness-cancellation/public-service#qualifying-employment) “你符合雇主对全职的定义，或者每周至少工作 30 小时，以较大者为准”)。
*   你需要为联邦、州、地方或部落政府组织或 501(c)(3)非营利组织工作(美国律师协会目前正在就此事的 501(c)(3)方面 [起诉](http://www.abajournal.com/news/article/education_department_seeks_to_toss_aba_suit_says_loan_forgiveness_calls_are) )。
*   你应该每年填写并提交 [就业证明表](https://studentaid.ed.gov/sa/sites/default/files/public-service-employment-certification-form.pdf) *以确保你走上正轨(并且每次你换雇主时确保你仍有资格)。*
*   在您支付了 120 英镑后，您需要提交此表格，以便实际上 [获得宽恕](https://studentaid.ed.gov/sa/sites/default/files/public-service-application-for-forgiveness.pdf) 。

我采访过的专家说，告诉你的服务商你想加入基于收入的还款计划是很重要的，因为他们没有动力让你加入这个计划(这不会让他们赚很多钱)。马克·坎特罗威茨是一名 [学生贷款和财政援助专家](http://www.kantrowitz.com/kantrowitz/mark.html) ，他说:“如果服务人员看起来不确定，就挂断电话，然后再打回去和其他人谈谈。”。你需要每年主动检查一次或两次，以确保你仍在正轨上——没有人会替你做这件事。

哦，对了，别忘了阅读你的邮件或你的学生贷款服务商发来的垃圾邮件。里面可能有至关重要的信息。

### 提出投诉

如果结果是你的服务人员欺骗了你，那就没什么可求助的了。但是你可以这样做:

Kantrowitz 说:“收集显示你何时支付每笔贷款的记录，以及关于你何时提交年度重新认证文件的记录，以便向服务人员提供你的收入信息。”

文书工作是这里的关键组成部分。律师兼学生贷款专家莱斯利·h·泰恩 说，如果你已经偿还贷款有一段时间了，你可能没有对这些年来所有的文件进行编目，这就很难证明任何事情。“当你有学生贷款时，你必须自己记录，不要依赖另一方。记录每一笔付款和每一封信件。保持真正好的文件，”泰恩说。“他们失去了付款，你需要证明这是如何发生的。”

除了向 CFPB 的 [在线投诉系统](https://help.consumerfinance.gov/app/studentloan/ask) 投诉外，坎特罗威茨还表示要向美国教育部的 [反馈系统](https://feedback.studentaid.ed.gov/s/) 投诉，因为 CFPB 和美国教育部之间不会共享投诉。也可以向 [FSA 调查官](https://studentaid.ed.gov/sa/repay-loans/disputes/prepare) 提出争议。

但如果你一直在关注，你就会知道 CFPB 和教育部门现在并没有关注消费者保护——事实上，他们似乎在积极反对选民的最大利益。所以你需要寻求国家保护。当然，这些会有所不同，所以你住在哪里会在你的选择中起到很大的作用。

如果实在不行，你可以聘请律师起诉，在这种情况下，你最好确保你有所有的文件和付款记录。这甚至可能不意味着诉讼，而是有一个中间人来解决你的条款。

### 游说你的代表

抵押贷款服务商受《房地产结算程序法》监管，信用卡公司受《信用卡责任和披露法》监管，这两部法律都对服务提出了严格的要求。但是学生贷款是没有可比性的，学生贷款主要受《高等教育法》管辖。

学生贷款借款人援助项目的负责人、国家消费者法律中心的一名律师 Persis Yu 说:“就正确处理付款以及当付款丢失时借款人有什么样的追索权而言，这些都是缺失的，这是一个大问题。”。

因此，如果你真的想改善问责制，那就游说你的代表。每天都有数不清的行业游说者与你们的国会议员交谈，所以你们也需要发出自己的声音。

首先，参议院健康、教育、劳工和养老金委员会已经要求借款人就学生贷款和财政援助发表声明。你可以发邮件到这里:【HigherEdStories2018@help.senate.gov】。

但是你需要更进一步。学生贷款计划者 的创始人特拉维斯·洪斯比说:“有很多法案没有成功，但都是由选民的关注发起的。”。“这不是一个超党派问题，所以他们可能更愿意倾听。”
