# 三种成分的快乐时光:朗姆酒边车

> 原文：<https://lifehacker.com/3-ingredient-happy-hour-the-rum-sidecar-1822426172>

各位周末快乐，欢迎回到 [*三料欢乐时光*](http://lifehacker.com/tag/3-ingredient-happy-hour) ，每周饮品专栏，提供超级简单又美味的酒水。本周，我们将对 Sidecar 鸡尾酒进行改良，这是一款经典的三种配料鸡尾酒。



边车通常由干邑白兰地、橘子利口酒和柠檬汁制成，但我发现很少有人会在家里的酒吧里定期储备干邑白兰地。因此，我决定使用一种深色、醇厚的混合朗姆酒，不仅因为它更普遍，而且因为我更喜欢这种味道。我对朗姆酒的品牌并不十分挑剔——尽管我会远离马里布、加香料的朗姆酒以及其他诸如此类的垃圾——但我对橙色利口酒非常挑剔。虽然普通的三秒钟不会毁了你的经典小跑车——装满了酒，但我更喜欢皮埃尔·费朗 T3 推出的更复杂、不那么令人作呕的库拉索干葡萄酒。要制作这款柑橘味浓郁、令人惊讶的饮料，你需要:

*   1.5 盎司深色朗姆酒
*   1 盎司干库拉索酒(如果你使用另一种橙色利口酒，减少到 3/4 盎司)
*   3/4 盎司新鲜柠檬汁(保留一条皮作为装饰)

将所有材料加入到装满冰块(或一大块冰块)的鸡尾酒摇壶中，摇至完全冷却。滤入一个小酒杯，在饮料上挤一条柠檬皮(黄色面朝下),然后放入杯中。
