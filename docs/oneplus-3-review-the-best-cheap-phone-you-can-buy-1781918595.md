# OnePlus 3 评论:你能买到的最好的廉价手机

> 原文：<https://gizmodo.com/oneplus-3-review-the-best-cheap-phone-you-can-buy-1781918595>

2014 年，一加突然冒出来一个强有力的想法:如果一部手机既好看、功能强大，又不贵，会怎么样？OnePlus One 是一个麦克风下降，OnePlus 2 是一个成功的大二后续产品，但新的 3 消除了仇恨——[OnePlus 3](https://oneplus.net/3?utm_source=androidcentral&utm_medium=media-us&utm_campaign=op3-launch)是一款“旗舰黑仔”，售价 400 美元，有了 [不再有愚蠢的邀请系统](http://gizmodo.com/oneplus-is-finally-ditching-its-annoying-invite-system-1779820957) ，它不仅是有史以来最好的一加手机，也是最好的廉价手机之一。



这款手机的任何部分都没有其外部设计有更大的进化飞跃。尽管 400 美元以下的手机令人钦佩，但一加的设计水平从未与其更昂贵的竞争对手相匹敌。一加联合创始人 Carl Pei 告诉 Gizmodo，最初的一加使用了其著名的砂岩饰面，因为全金属太贵了。对于 3，一加 ups 的价格有点去完全终结者。

OnePlus 3 由一块阳极氧化铝制成，并大量借鉴了竞争对手流行的设计怪癖，最终看起来像是它一直想成为的“高端”手机。3 的摄像头位置和天线设计看起来令人难以置信 [受 HTC 启发的](http://gizmodo.com/htc-10-review-a-last-grasp-at-greatness-1770943344) ，而 5.5 英寸的 AMOLED 1080p 显示屏从边框中冒出来，很像 iPhone。

但它不仅仅是一个没有灵魂的模仿者。

像 OnePlus 2 一样，3 也运行一加自己的 Android mod，名为 Oxygen OS。对于智能手机爱好者来说，这意味着一加比标准的 Android 有更多的定制选项——深色主题，图标大小，更多的多任务选项。Oxygen OS 基本上是 Android Marshmallow，Android 的当前版本，但对用户界面进行了一些小的调整。为了向你展示我的意思，这里有运行 Nexus 5X 的 Android 和运行 OnePlus 3 的 Oxygen OS。如果你能猜出哪个是哪个，奖励积分:

3 有一个类似的规格清单，像最近最好的手机休息。USB Type-C？检查。最新的骁龙处理器？是的先生。指纹传感器，高像素摄像头，快速充电？是的，是的，当然。事实上，你唯一可以尝试放在负号栏中的是 1080p AMOLED 显示屏，而不是大多数其他手机(除了 iPhone)上的像素密集型 2K 屏幕。但除非你打算把 VR 绑在脸上，否则你可能会对 OnePlus 3 一样满意。

但一个主要的不同是 OnePlus 3 的内存容量——6gb。六个！这相当于一台笔记本电脑——一台劣质的——但仍然如此！这真的让氧气 OS 眼前一亮。像苹果和 iOS 一样，一加优化了其内部软件，以便在 OnePlus 3 上运行得特别好，因此它可以以其他 Android 竞争对手无法做到的方式利用所有内存。

在并排测试中，OnePlus 3 轻松跟上 iPhone 6s Plus。我们在两款手机上打开了相同的巨型 PDF，OnePlus 3 只是勉强击败了 iPhone。当你考虑到苹果缓慢得令人痛苦的动画时，你甚至会觉得它快了一点。

当我们试图在后台运行十几个其他应用程序的情况下打开相机时，三星 Galaxy S7 Edge 只用了 2.6 秒。但 OnePlus 3 在不到一秒的时间内就成功了。

一旦相机弹出，OnePlus 3 显然在软件和硬件方面取得了一些令人印象深刻的进步。首先，你可以使用 1600 万像素的前置摄像头进行 RAW 拍摄——这对于在 post— *和*中保存曝光不良的照片非常有用。更新后的相机应用程序具有手动控制功能，因此你可以随意调整所有高级相机设置。

可悲的是，修补的能力不能挽救普通的光学器件。这里有一个与 OnePlus 3 及其三个竞争对手 Galaxy S7、LG G5 和 iPhone 6s Plus——的比较。

[Open *tools.kinja-labs.com*](http://tools.kinja-labs.com/iframe/galleries/130)

尽管这张照片即使在光线不足的情况下也能捕捉到整体图像，但与 Galaxy S7 和 6s Plus 相比，史蒂夫·乔布斯(不是库彻)的面部细节并不清晰。我们没有谈论灾难性的差异，但可能足以阻止人们寻找最好的移动相机。

但并不是略逊一筹的摄像头是 OnePlus 3 的最大缺陷。是续航。在我和 3 号在一起的一周里，它至少有两次在深夜死去。有一次，它把我锁在公寓外面，无法联系到我的室友。我不得不走到一个朋友家，用力敲门，求他让我睡在他的沙发上。正是在这些时刻，可以持续到第二天早上的手机(三星 S7 的电池寿命 [真的是难以置信的](http://gizmodo.com/samsung-galaxy-s7-review-inching-toward-perfection-1763438896) )比不能持续的手机有用得多。

3*配备了自己的混合快速充电功能，称为 Dash Charge，该公司称其提供“半小时一天的电量”在手机关机并电量耗尽的情况下，我能够在 40 分钟内用 3 的 3000 毫安时电池充电 80%。很好，但不惊人。一加认为人们会在一天中定期给手机充值，但这不符合我的习惯。*

*除此之外，很难不看看 OnePlus 3，惊叹这家不知名的中国公司如何如此迅速和自信地以如此低廉的价格制造出一款旗舰手机。无论从哪个角度来看，这都是一款顶级设备。以前的一加手机充满妥协，而 OnePlus 3 几乎没有妥协。就价格而言，OnePlus 3 绝对是你能买到的最好的手机。*
