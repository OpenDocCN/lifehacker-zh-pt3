# 谷歌新推出的 Pixelbook 笔记本电脑值这个高价吗？

> 原文:[https://life hacker . com/is-Google-new-pixel book-laptop-worth-the-high-price-ta-1819154127](https://lifehacker.com/is-googles-new-pixelbook-laptop-worth-the-high-price-ta-1819154127)

大多数[Chrome book](https://lifehacker.com/this-interactive-chromebook-comparison-tool-helps-you-f-1783758254)便宜、简单、可靠，但谷歌多年来一直在发布运行其 Chrome 操作系统的高端笔记本电脑。该系列的最新产品是 [、1000 美元的 Pixelbook](https://gizmodo.com/google-wants-to-sell-you-a-1000-chromebook-1819126689#_ga=2.91830489.233542325.1507127109-472331419.1499788931) ，它看起来准备与其他笔记本电脑和平板电脑的混合体竞争，如 [微软的 Surface Pro 和 iPad Pro](https://lifehacker.com/pro-tablet-showdown-ipad-pro-vs-surface-pro-4-1753293972) 。

谷歌的 Pixelbook 有机会吗，它值得高价入场吗？让我们找出答案。

### 不完全是平板电脑

与 iPad Pro(全尺寸版本 799 美元及以上)或[Surface Pro](https://www.microsoft.com/en-us/surface/devices/surface-pro/overview?&SEMID=1&WT.srch=1&ocid=OCSurfL_SEM_Google_PI_AWC_en-US_ipad%20pro&wt.mc_id=OCSurfL_SEM_Google_PI_AWC_en-US_ipad%20pro&wt.srch=1&wt.mc_id=AID620867_SEM_GY9U0CRu)(799 美元)不同，这些基本上都是带有可连接键盘和可选手写笔的平板电脑，Pixelbook 是一款触摸屏笔记本电脑，谷歌称它也可以完美地用作平板电脑。

与其他 [可转换笔记本电脑](https://gizmodo.com/the-best-transforming-windows-laptops-under-500-1710553853) 类似，Pixelbook 屏幕在平板电脑模式下可以翻转整整 360 度。如果你想休息和看视频，你也可以把它调整到一个角度，这样键盘就可以兼作支架。这是对谷歌之前的笔记本电脑 Chromebook Pixel 2 的一次重大升级，chrome book Pixel 2 有触摸屏，但不能变成平板电脑。

因此，如果你想要一台笔记本电脑，同时也作为一个有点笨重的平板电脑，Pixelbook 是一个可靠的选择。但如果你正在寻找一款真正的平板电脑，在必要时可以兼作笔记本电脑，你最好选择苹果和微软提供的产品。

### Chrome 操作系统越来越好

Chrome OS 最初只是一个浏览器，但在过去几年里，谷歌扩展了其操作系统的功能。最大的变化之一是增加了对 Android 应用程序的支持 Chromebook Pixel 2 的 [也能支持](https://www.engadget.com/2016/09/27/pixel-2-joins-the-few-chromebooks-that-can-run-android-apps/) 。作为额外的奖励，Pixelbook 将获得专为其更大屏幕设计的 Snapchat 特别版本(谷歌没有说这是否会出现在任何其他型号上)。

iPad Pro 运行 iOS 应用程序，而不是 Android，但在其他方面，它的功能与 Chrome OS 非常相似。苹果还在 iOS 11 中对其 iPad 系列进行了一些相当大的改进，增加了新的应用 dock 和多任务功能，使其感觉更像 macOS。

但如果你想要一台功能齐全的平板电脑，Surface Pro 是不二之选。它运行 Windows 10，这是一个实际的桌面操作系统，也是为触摸屏控制而设计的。

### 手写笔额外收费

谷歌首次为其新 Chromebook 提供数字手写笔。Pixelbook 笔会让你多花 99 美元。这是一个非常标准的手写笔，有一个很大的额外功能:按住内置按钮可以立即访问谷歌助手。

苹果和微软也提供他们自己的版本，价格也是 99 美元。Apple Pencil 非常注重创造性工作，内置传感器，因此你可以更用力地向下推，以绘制更粗的线条和其他艺术效果。Surface Pen 介于两者之间，为草图提供压力敏感性，加上一个按钮，可以快速激活微软自己的人工智能助手 Cortana。

### 那么该不该买呢？

如果你正在寻找一台带有一些非常智能的软件的时尚可转换笔记本电脑，Pixelbook 是一个伟大的(但昂贵的)选择。如果你正在寻找一款专业级的平板电脑，苹果和微软已经满足了你的需求，至少在谷歌将 Chrome 操作系统放在一款成熟的平板电脑上之前是如此。
