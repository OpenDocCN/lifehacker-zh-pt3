# 在任何使用该应用程序的 Android 手机上试用三星的 Galaxy S9

> 原文:[https://life hacker . com/try-out-samsungs-galaxy-S9-on-any-Android-phone-using-t-1823595266](https://lifehacker.com/try-out-samsungs-galaxy-s9-on-any-android-phone-using-t-1823595266)

如果你一直在考虑购买 Galaxy S9，但不确定你是否准备好咬紧牙关，三星为你创造了一种方法，让你使用当前的设备试驾这款手机。

Watch

Galaxy S9/S9+ 的 [体验应用可以安装在任何运行 Android 6.0 或更高版本的 Android 手机上，让你感受一下“S9 体验”可能是什么样的。](https://play.google.com/store/apps/details?id=com.samsung.retailexperience.retailstar.star.google&rdid=com.samsung.retailexperience.retailstar.star.google)

实际上，决定你是否喜欢使用手机的最好方法是亲自试用手机。如果你正在考虑购买 S9，我建议你去当地的移动商店(甚至不一定是你的运营商)购买，并在下周发布时带它出去兜风。说真的。不要花 700 多美元买你从未见过的东西，而是计划每天都用。

也就是说，在你费力地去商店之前，有必要多了解一些设备。

该应用程序将带你了解手机的新相机功能，以及 AR 表情符号和超级慢动作的工作原理。它还解释了 S9 中新的面部识别功能，强调了手机指纹传感器的新位置，并展示了新的杜比 Atmos 扬声器的工作原理。

值得注意的是，这些功能中的大部分都需要 S9 中的硬件才能运行，所以你实际上只是获得了这个应用程序的功能概要，而不是使用 S9 的真实模拟体验。大多数演示都是视频，向您展示如果您手中有一台 S9 而不是另一台设备，某些东西可能会如何工作。

也就是说，如果你有兴趣了解更多关于这款手机的知识，这是一个不错的开始，如果你通过演示，在手机到达你家门口之前，你就已经是所有这些新功能的专家了。
