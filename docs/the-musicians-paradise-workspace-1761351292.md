# 音乐家的天堂工作区

> 原文：<https://lifehacker.com/the-musicians-paradise-workspace-1761351292>

读者 [杰森·伯恩斯](https://www.flickr.com/photos/140318854@N06/) 是一名音乐家，在他的家庭工作室做很多工作。我们已经看到了许多工作室，但这个工作室充分利用了空间，包含了大量的设备，很明显杰森生活和呼吸着他的音乐。



文章顶部的照片是他的录音棚，他在那里写歌，录音和编辑都在电脑上进行。他指出，他不想让他的电脑放在桌子上占用空间，所以 Mac Pro 和他用于存储的 RAID 阵列都被安装在他的桌子下面，你可以在这里看到 。下面，你可以看到他更随意的工作空间，配有一张 [Evo 坐/立办公桌](http://www.evodesk.com/) ，他可以根据自己的工作感觉进行调整。Macbook Pro 在左边，外部显示器在中间。左边的书架上有许多玩具、游戏和其他灵感，我们可以在你工作和生活的任何地方欣赏它们。

这是他坐/站工作空间的另一个视图，在正下方，你可以看到桌子被抬高了。

最后，整个空间的一个镜头，全部放在一起。如果空间的使用还不明显，它不仅仅是一个工作区或办公室——杰森也住在这里，这是显而易见的。他指出，他花了几个月的时间才把一切弄好，但我可以想象，光是把吉他挂在墙上和搭建娱乐中心就花了几个月的时间。总的来说，这是一个伟大的期待。

如果您有自己的工作空间可以炫耀，请将其添加到我们的 [**Lifehacker 工作空间展示并告诉 Flickr pool**](http://www.flickr.com/groups/lifehacker-workspace-showandtell/pool/) 来与我们分享。确保您包含的任何照片至少为 1280x720。将它们保持在 16:9 也有帮助！包括一些文字，关于你使用的材料，你是如何想出这个设计的，以及任何其他相关的细节。如果你巧妙的组织和良好的设计感吸引了我们的眼球，你可能就是下一个特色工作区！
