# 你可以在 Android 的开发者选项中解锁最酷的功能

> 原文：<https://lifehacker.com/the-coolest-features-you-can-unlock-in-androids-develop-1789517222>

Android 的开发者选项菜单是一个秘密的调整宝库，喜欢冒险的用户可以尝试。如果你想在你的 SD 卡上安装*任何*应用程序，加速窗口动画，甚至节省一些电池寿命，下面是如何解锁和使用这个隐藏菜单。

### **如何启用开发者选项**

启用 Android 的开发者选项很容易，但它是隐藏的，所以你不会意外地发现它。请记住，隐藏这些选项是有原因的。它们旨在为开发人员提供测试所需的工具，在不知道它们做什么的情况下更改它们可能会导致一些应用程序无法正常工作。如果某些东西停止工作，在向开发人员抱怨之前，尝试恢复您所做的更改。也就是说，其中一些调整真的很有用，而且效果很好。要打开开发人员选项，请按照下列步骤操作:

1.  打开应用程序抽屉，轻按“设置”以打开“设置”应用程序。
2.  向下滚动并点击关于电话。查找设置“内部版本号”
3.  连续轻按“构件号”七次。
4.  应该会出现一个提示通知，告诉你现在是开发人员了。就是这样！你被录取了。

一旦你启用了开发者模式，你应该会在你的设置应用中看到一个新的“开发者选项”菜单，里面有各种各样的好东西。这些是你可以用它做的一些很酷的事情。

### **将任何应用程序安装到 SD 卡上**

如果您尝试将应用程序安装到您的 SD 卡，但它不起作用，您可以在开发者选项中启用“强制允许外部应用程序”。这将迫使应用程序让你把它存储在你的卡上，而不是占用你的内存。当然，开发人员阻止这个选项可能有一个很好的原因。一些应用程序在特定的地方寻找文件，如果应用程序安装在它不期望的地方，就不会工作。因此，如果应用程序在安装到您的卡上时不工作，请尝试卸载它并将其放在内部存储上。然而，其他开发人员只是为了简单起见而这样做，即使该应用程序可以工作。如果你下载了一个巨大的应用程序，并想利用你幸运拥有的 SD 卡，打开它，看看你能腾出多少空间。

### **强制任何应用程序在分屏模式下工作**

谷歌 [在安卓牛轧糖](http://lifehacker.com/all-the-new-features-coming-in-android-n-1763790001) 中引入了分屏模式，非常方便，但有些应用在分屏中无法工作。不过，无论如何你都可以强制它们分屏打开， [这样你就可以一边玩 Pokémon Go，一边查收邮件](https://lifehacker.com/use-android-nougats-split-screen-on-apps-like-pokemon-g-1786839888) 之类的。同样，这通常是因为分屏模式可能会破坏应用程序，但有时这只是因为他们还没有为此进行优化。如果您仍想尝试，请在开发者选项中启用“强制活动可调整大小”并重新启动您的设备。现在，所有应用程序都将在分屏模式下工作。只是有可能其中一些看起来不太好，或者它们可能会一起崩溃。

### **手机充电时保持屏幕打开**

一些手机允许你在 30 分钟内防止屏幕变暗，或者只要屏幕上有活动就一直开着，但大多数手机最终会关闭显示屏以节省电池。如果你正在充电，这不是问题，特别是如果你喜欢在充电时或手机放在桌子上时关注你的通知。在开发者选项下，启用“保持清醒”开关，只要你的手机还在充电，它的显示屏就会一直开着。

### **加速窗口动画**

当你切换应用程序时，从主屏幕到应用程序或屏幕之间的转换等动画对你的手机感觉有多快起着很大的作用。快速的动画让你的手机感觉轻快，而较慢的动画可以掩盖你第一次启动应用程序时的滞后。开发者选项给你三种不同的动画，你可以改变:*窗口动画比例*，*过渡动画比例*，和*动画持续时间比例*。无论您选择什么，将所有这些设置为相同(您可以选择. 5x、1x、2x 等)可能是一个好主意。

如果你想让你的手机感觉更快，你可以将这些设置为 0.5 倍，这将使速度翻倍。然而，请记住，开发人员通常用这一秒钟左右来加载他们的应用程序或相关数据。这可能看起来不多，但它很重要，如果动画太短，你会被困在不同的屏幕上。如果你的手机已经很难加载应用程序，你可以让动画时间更长。你仍然会等待你的应用程序加载，但不会*感觉*那么久，因为你给了你的应用程序更多的时间来为你做准备。

### **切换到黑白模式以节省电池**

如果你的手机有 AMOLED 显示屏，你可以通过使用更暗的主题 来节省 [大量的电池寿命。如果你想延长电池寿命，试试单色模式。滚动到“模拟色彩空间”并选择“单色”当然，这不是灵丹妙药。事实上，如果你不做其他任何不同的事情，你可能根本不会节省很多。然而，如果你使用黑暗的主题，你的应用程序的黑暗模式，并依靠单色模式的高对比度而不是鲜艳的颜色来使事情可读，你可以将你的设备调到](https://lifehacker.com/pixel-off-turns-off-pixels-to-save-battery-on-amoled-di-1696157699) [节省大量的电力](http://www.greenbot.com/article/2834583/how-much-power-does-a-black-interface-really-save-on-amoled-displays.html)[，这在紧要关头特别有用，或者当你知道你暂时无法充电](http://www.greenbot.com/article/2834583/how-much-power-does-a-black-interface-really-save-on-amoled-displays.html) 。

如果你碰巧有某种形式的色盲，Android 也提供了一些颜色空间来帮助你。您可以尝试 Deuteranomaly 或 Protanomaly 来帮助治疗红绿色盲，或 Tritanomaly 来帮助治疗蓝黄色盲。

这个隐藏菜单中的功能是不断变化的，所以你可能会发现更多的好东西，这取决于谁制造了你的手机。虽然其中一些可能超出了普通用户的理解范围，但是有许多方便的小调整可以尝试。
