# 早餐吃非早餐三明治的理由

> 原文：<https://lifehacker.com/the-case-for-eating-non-breakfast-sandwiches-for-breakf-1828496043>

我早上有两件事:生气和恶心。我一直都是这样，甚至在年轻的时候也是这样，这使得在早上 7 点开始物理化学变得更加困难。早餐煎一个鸡蛋根本不是一个选择，但我会在讲座期间非常饿，以至于我会把午餐(通常是火腿三明治)作为早餐。

然后我突然想到:为什么我不做两个火腿三明治呢？早餐三明治上通常有火腿和奶酪。唯一真正的区别是没有鸡蛋，我又一次完全没有能力在课前准备。就像我的午餐三明治，我可以在前一天晚上做好，然后在出门的时候拿起来，在上课的时候吃，而不会抢走自己的午餐。

当然，并不是所有的午餐三明治都是很棒的早餐三明治。中午前的金枪鱼沙拉是一种彻头彻尾的战争行为，更强烈、更辛辣的腌制或发酵成分可能是第一件事，特别是如果你像我一样想吐。而且，尽管熏鱼是一种完美的早餐选择，但把这种感官体验强加给你的同学有点不体贴。以下是一些我最喜欢的“非早餐”三明治，它们实际上是相当不错的早餐:

*   法棍上的布里干酪、黄油和火腿(或普通火腿)
*   小麦上的火鸡、鳄梨和切达干酪(调味品就交给你了。)
*   培根和蛋黄酱抹在白面包上(老实说，为什么要把农产品复杂化呢？)
*   鳄梨、黄瓜和鹰嘴豆泥放在种子面包上
*   奶油奶酪和果酱在白色背景。
*   花生酱(或任何坚果黄油)和果冻
*   切达干酪、苹果和火鸡卷

你会注意到缺少西红柿和其他湿产品——这是为了防止三明治在冰箱里过夜变湿。如果你一定要吃这些东西，在前一天晚上把它们切片，放在一个单独的袋子里，在吃之前把它们加到你的早餐里。
