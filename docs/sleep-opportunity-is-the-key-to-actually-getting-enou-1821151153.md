# “睡眠机会”是获得充足睡眠的关键

> 原文：<https://lifehacker.com/sleep-opportunity-is-the-key-to-actually-getting-enou-1821151153>

你知道 [睡眠黑客](https://lifehacker.com/top-10-tricks-for-getting-better-sleep-5876083) 。你在下午停止摄入咖啡因，避免蓝光照射，也许在睡觉前洗个热水澡，伸个懒腰。然而，即使你在闹钟响之前的几个小时就上床睡觉了，还是不够。



[The Cut](https://www.thecut.com/2017/11/how-to-get-a-tiny-bit-more-sleep.html) 写道，睡眠科学家马修·沃克在他的书《为什么我们要睡觉》 中提出了“睡眠机会”的想法，作为你每天不会筋疲力尽的策略中可能缺失的一块。睡眠机会意味着关注的不是你需要多少时间的睡眠，而是你需要多少时间的睡眠来获得足够的睡眠。

我们中很少有人是睡眠专家，一上床就睡着了，甚至包括仰卧的 Instagram-scrolling。沃克建议专注于给自己需要的睡眠机会，为入睡准备一个垫子，或者，比方说，你的狗在闹钟响前 20 分钟哀叫着出去。(消除那种焦虑，天哪，狗在哀叫，现在我睡不着了，也有帮助。)

沃克坚持严格的八小时睡眠时间，知道即使睡得少，也会睡得饱。你理想的睡眠时间会有所不同——每个人都需要不同的睡眠时间——但这将帮助你给自己一个真正获得睡眠的机会。
