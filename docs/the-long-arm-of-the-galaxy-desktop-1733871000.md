# Galaxy 台式机的长臂

> 原文：<https://lifehacker.com/the-long-arm-of-the-galaxy-desktop-1733871000>

就像我们之前说过的，有时候少定制就是多定制，这款来自 reader [Mackenzie](https://www.flickr.com/photos/136087398@N08/) 的漂亮桌面是另一个很好的例子，说明了仅仅几个皮肤——不要太多——就能产生很大的影响。她是这样设置的。



这是一个 Windows 系统，Mackenzie 注意到它实际上只是一个雨量计皮肤——甚至这个皮肤也删除了一些元素，而不是添加。如果你想要一个相似的外观，这里是你需要的:

*   [壁纸](http://i.imgur.com/y8PRWgs.jpg)
*   用于 Windows 的 [雨量计](http://rainmeter.net/) 系统调试和监控工具
*   [只是雨量计的新星](http://rainmeter.deviantart.com/art/Simply-Nova-267726929) 主题，用于时间/日期显示、天气和系统监视器

就是这样。没有冗长的小部件列表，也没有大量需要定制的脚本——结果看起来非常棒。如果你喜欢你看到的，去麦肯齐的 flickr 页面(链接如下)让他们知道你喜欢成品的样子，或者问你关于它是如何设置的问题。
