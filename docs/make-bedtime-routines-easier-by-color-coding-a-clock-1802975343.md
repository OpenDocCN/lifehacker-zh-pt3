# 通过对时钟进行颜色编码，让放学后的日常工作变得更容易

> 原文：<https://lifehacker.com/make-bedtime-routines-easier-by-color-coding-a-clock-1802975343>

作为家长，我常常觉得自己就像一个会走的秒表。“再看三分钟电视！”“离刷牙时间还有八分钟！”“离睡觉时间还有 24 分钟！”



*拿出那些捕兽器，削尖你的 2 号铅笔——这是* [*返校*](http://lifehacker.com/tag/back-to-school) *周！在教室之外，Lifehacker 将为您带来天才的技巧和想法，告诉您如何开始日常工作，重温旧技能或在今年秋天学习新东西。*

对我四岁的女儿来说，她对时间只有一个模糊的概念，所有这些似乎都是可以商量的。"再玩两分钟的乐高朋友，好吗，妈妈？"有乞讨。还有撅嘴。这是妈妈的声音。

如果你像我一样，需要一些帮助来推进你的日常工作，尤其是在放学后，当生活变成晚上 8:30 的疲惫倒计时时，用颜色标记一个时钟来让你的孩子继续工作。 [孩子活动博客](https://kidsactivitiesblog.com/91715/school-routine-clock) 分享了一个 DIY——只要打开一个挂钟，用记号笔给时间段标上颜色，然后在一张纸上，列出每种颜色代表的任务。看，这钟不讨价还价。对于还不会看时间的孩子来说，这是一个特别好的工具。
