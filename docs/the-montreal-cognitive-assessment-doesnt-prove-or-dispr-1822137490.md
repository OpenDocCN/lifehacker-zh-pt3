# 蒙特利尔认知评估并不能证明或反驳特朗普是否疯了

> 原文：<https://lifehacker.com/the-montreal-cognitive-assessment-doesnt-prove-or-dispr-1822137490>

你是一个正在检查老年病人的医生吗？你想确定你会注意到你的病人是否患有轻度痴呆症吗？然后你可以问一些简单的问题，比如病人是否能认出一幅狮子的图画，是否能记住五个单词。然而，如果你想知道一位总统是否适合任职，你可能需要在这个问题上花费超过十分钟的时间。

我们的总统在周五的体检中以优异的成绩通过了蒙特利尔认知评估 ，满分 30 分。唷，他*知道狮子长什么样吗？*

这个结果并不能证明他处于精神健康的巅峰，也不能证明他有能力接受审判。这个测试真正的意思是:如果你有轻度认知障碍，比如早期阿尔茨海默氏症，你有大约 90%的机会在测试中得分很低。如果你没有，你有 87%的机会通过。

这就是它告诉我们的一切。心理健康有很多很多方面是这样的测试检测不出来的。如果你想知道你会如何得分，找个朋友来看看吧:测试在这里 和说明在这里 。它应该由医生管理，所以不要对结果有太多的解读(除非你的朋友是医生)。

哦，还有一个有趣的事实:开发这种测试的齐亚德·纳斯雷丁 ，十几岁时从黎巴嫩移民到加拿大。他在接受加拿大《国家邮报》采访时发表了一篇非常低调的恶心文章，其中写道，“他希望总统从[测试]中吸取教训:这表明移民和阿拉伯人如何能够为美国社会做出宝贵贡献。”
