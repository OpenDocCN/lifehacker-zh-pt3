# 证明自己的价值

> 原文:[https://life hacker . com/the-value-of-proof-yourself-1761315650](https://lifehacker.com/the-value-of-proving-yourself-1761315650)

你还记得你的第一份工作吗？因为你没有任何实际工作经验，所以你基本上是凭信念被雇用的。你在阶梯的底部，需要证明为什么你值得这份工作。无论你在职业生涯的哪个阶段都是如此:你仍然需要证明自己的价值。

*本帖最初出现在* [*的简单美元*](http://www.thesimpledollar.com/the-value-of-proving-yourself/) *上。*

几天前，我发现自己在读一篇来自《大西洋月刊》的老文章，作者是约瑟夫·威廉姆斯，题为《我的零售生活:肮脏、野蛮、贫穷、T6、T7》。在文章中，威廉姆斯概述了他自己的故事，他的职业生涯突然出乎意料地崩溃，他发现自己从事的是入门级的零售工作。这是一篇引人入胜的文章，我鼓励每个人花几分钟读一读。

从这篇文章中你可以学到很多东西。有些人可能会认为威廉姆斯有资格并认为自己“凌驾于”这份工作之上。其他人可能会认为，所描述的零售工作是对工人的极大贬低，这整个故事是美国企业如何对待入门级员工的一些深层问题的象征。还有一些人可能会看到威廉姆斯在工作场所处理自己的一些缺陷，即使是考虑到这篇文章的片面性(毕竟这是威廉姆斯的观点)。

相反，我一直站在威廉姆斯的角度考虑问题。我在他的情况下会怎么做？

现在，我有一份能有效“证明”自己的工作。我可以在家工作，不需要任何日常管理。我已经“证明”我可以在几乎没有任何管理的情况下坚持写作并按时完成任务，他们知道我可以做到。我已经做了很多年了。然而，我花了*许多年*才达到这样的程度，我已经向我的各种客户“证明”了自己。

大约十年前，当我第一次开始写《简单的一元》 的时候，我并没有因为写作的一致性或相对质量而获得任何声誉。我必须证明这一点。我不得不每天都写，一遍又一遍。我坚持不懈地做了这件事，人们开始期待了。几年后，我已经“证明”我可以长期坚持这样做。

如果我突然不得不在零售领域重新开始，就像威廉姆斯不得不做的那样，我可能会觉得我已经“证明”了自己。但那不是事实。

事实是，我将在一个新阶梯的最底层重新开始。我还没有爬上哪怕一个梯级。简而言之，如果我在一份新的职业中重新开始，我还没有在那份职业中“证明”自己。

如果我的职业生涯突然崩溃，我发现自己在一个入门级的零售职位上工作，我会预计我将不得不经历威廉姆斯经历的许多事情。当我来上班和离开的时候，我希望我的包能被托运。我希望被搜身。

为什么？因为我还没有证明自己。

你必须证明自己。没有人认为你是特别的或特殊的。

你看，像这样的入门级工作的一个方面是，几乎任何人都可以得到这样的工作，而没有任何证据表明他们的性格或职业道德。你通常不需要在简历上写什么就能得到这样的工作。当然，随之而来的事实是，你的工资不会很高，你的雇主不会完全信任你(一开始)，你会被视为(尤其是一开始)一个完全可以替代的人。你在那里花时间做相对简单的任务，如果你做不了，他们会找别人来做。是的，这些任务可能看起来很费力，有时，你可能会觉得有很多事情要做。

你可能会觉得不被尊重。一天下来，你肯定会感到筋疲力尽。但那不是重要的事情。

真正重要的是你如何应对所有这些。你会以此为借口在工作中不努力吗？当你还没有证明的时候，你会怨恨你的老板没有把你当作一个特殊的人吗？

还是你会认真思考，想出如何尽你所能做好自己的工作，学习你的老板是怎么做的，并准备在机会到来时接手这个职位？

这里值得注意的另一件重要事情是:做一天、一周或一个月的优秀员工也是不够的。你可以培养的最有价值的东西之一是可靠性和一致性，这是我在最近一篇关于 [你已经拥有的成功技能](http://www.thesimpledollar.com/you-already-have-all-the-skills-you-need-to-succeed/) 的文章中偶然发现的。

所以，这就是所有的一切。

你想在一个全新的职业中重新开始吗？你必须证明自己。你必须从梯子的底部开始，忍受考验来证明你自己，其他人都必须忍受。如果你失败了——如果你决定离开而不认真对待这件事——你就不会往上爬。句号。

你想在你现在的职业生涯中提升吗？你必须证明自己。你必须*同时*涵盖你目前工作的所有基础，并弄清楚你需要什么，以便能够获得下一份工作。拉不下来？你不会往上爬的。

你想从 [开始自己的副业](http://www.thesimpledollar.com/50-side-businesses-you-can-start-on-your-own/) 吗？你必须证明自己。你必须自己创造一些对客户和顾客有吸引力的东西。你必须弄清楚如何让这些客户和顾客意识到你有什么是有价值的，以及他们为什么需要它。你觉得你做不到吗？那你的副业就不会大获成功。

每一次你想更上一层楼，每一次你想开始新的轨迹，你都要证明自己。

这里我喜欢用军队作为例子。军队的整个等级是围绕证明你自己而建立的。达到某一等级意味着你已经向他人展示了你达到了该等级所必需的技能。整个系统是建立在尊重那些获得更高职位的人的基础上的。

你猜怎么着？一周的努力工作不足以在军队中获得高军衔。一年是不够的。十年是一个好的开始。

但是，当你终于忍耐了足够长的时间，当你已经学会了完成本级任务和下一级任务所必需的所有技能，你终于准备好提升时，会发生什么呢？

你已经证明了自己。有了这个证据，就有了回报。它们也不仅仅是金钱上的回报，尽管这些回报可能很甜蜜。回报有其他形式——比如，别人知道你已经证明了自己，所以你不必拐弯抹角。你也不必再继续一些较低层次的任务。

那么，如果我处在约瑟夫·威廉姆斯的位置，我会怎么做呢？如果我在体育用品店的第一天发现自己突然重新开始，我该怎么办？

我要做的第一件事就是意识到我没有向与我一起工作的人证明过任何关于我自己的事情——而且，除此之外，几乎没有人具备这份工作的资格。起初，我会意识到自己实际上是一个齿轮，一个处于阶梯最底层的人。然后，我会决定我是否只是去做最低限度的收集一张支票，或者我是否要向上爬。了解我的话，我会顺着梯子往上看。

从我踏进那扇门的那一刻起，我就开始吸收我能吸收的每一点知识。每天，我都会问自己是否证明了自己一点点。我今天做得足够证明我自己了吗？我做了我应该做的高质量的事情了吗？我是否也展望了梯子的下一步，并准备好迈出那一步？

每当你开始进入一个新的领域，你都会有这样的机会。你什么都没证明，或者证明得很少。你没什么特别的。这取决于你来证明这一点，通过你的一致性和努力工作，愿意学习和渴望和野心向上移动。通过你的行动，每一天，每一小时，每一分钟，你都在参与这项工作。

如果你坚持下去，如果你继续努力，日复一日、周复一月、年复一年地证明自己，机会就会到来。只要愿意证明自己，确保自己以可靠的方式做好工作，并了解阶梯上的下一步是什么，你就会让自己与周围的大多数人不同，这一点会被看到。

你会证明你有价值，而这种价值会得到回报。不过，这取决于你。没有别人。

所有这些在任何工作或任何创业企业中都是真实的，无论是快餐店的入门级工作，还是高级技术工作或管理职位，或者是你试图建立的新公司。大多数时候，你根本没有证明自己，而成功通常会降临到那些证明了自己的人身上。

你会证明你应该得到你认为应该属于你的成功吗？表现出来。
