# 你想看什么午休锻炼？

> 原文：<https://lifehacker.com/what-lunch-break-workouts-would-you-like-to-see-1798417833>

嘿，工人们！自从我们在今年夏天早些时候推出 [午休锻炼](http://lifehacker.com/tag/lunch-break-workout) 以来，我们一起做了很多: [拉伸](http://vitals.lifehacker.com/try-these-stretches-even-if-youre-not-a-flexible-person-1797894301)[跳舞](http://vitals.lifehacker.com/bust-a-move-on-your-lunch-break-with-this-90s-inspired-1797664457#_ga=2.151886078.631411099.1503325890-255842443.1487643310)[坐姿锻炼](http://vitals.lifehacker.com/get-a-workout-without-leaving-your-chair-1796823587#_ga=2.240622020.631411099.1503325890-255842443.1487643310) ，甚至 [尝试倒立](http://vitals.lifehacker.com/get-a-workout-without-leaving-your-chair-1796823587#_ga=2.151886078.631411099.1503325890-255842443.1487643310) 。现在我们接受提问:接下来你想看什么？

Watch

一些需要考虑的事情:

*   我们应该坚持无器械锻炼，还是你想时不时地去健身房锻炼一下？
*   更多瑜伽？更多的有氧运动？更多的力量训练？你最喜欢的锻炼方式是什么？
*   你是想深思熟虑地朝着一个目标努力(比如变得更加灵活或者做更多的俯卧撑)，还是只是关掉你的大脑，做一些让你动起来的事情？
*   坦白:你真的在午餐时间做这些吗？(不要也完全没关系！)

我们将利用您的反馈来塑造午休锻炼的未来，所以请在评论中留下您的想法！
