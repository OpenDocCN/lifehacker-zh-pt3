# 安全消息应用摊牌:WhatsApp vs. Signal

> 原文:[https://life hacker . com/secure-messaging-app-摊牌-whatsapp-vs-signal-1794684943](https://lifehacker.com/secure-messaging-app-showdown-whatsapp-vs-signal-1794684943)

所以，你对安全、加密的聊天应用感兴趣。你有几个不同的选择，但与任何聊天应用程序一样，你所有的朋友都在使用什么很重要。为此，Signal 和 WhatsApp 无疑是最受欢迎的。以下是他们的对比。

### 竞争者

随着我们开始意识到公司每天收集多少关于我们的数据，安全消息应用程序越来越受欢迎。与电话不同，更容易想象一些大型企业实体或政府获得了你的聊天记录，并将其用于邪恶的手段。虽然有许多应用程序做到了这一点，包括苹果的 iMessage、WhatsApp 和 Signal 在受欢迎程度、平台可用性和功能方面都处于领先地位。除了消息传递，这两款应用还包括语音和视频通话，尽管我们在这里将集中讨论两者的文本消息传递功能。让我们来分析一下两者的工作原理:

*   [**Signal**](https://whispersystems.org/) :虽然 WhatsApp 在这两者之间拥有最多的用户，但每当人们谈论加密时，你可能更经常在新闻中听到 Signal。从斯诺登到俄罗斯，你读到的几乎所有关于安全的文章都提到了信号。这是因为通过 Signal 发送的每条消息都支持端到端加密(我们将在下面详细讨论这意味着什么)。这种安全措施意味着，如果有人截获了你的信息，或者在某个地方的服务器上发现了它们，他们看到的将是胡言乱语，而不是真正的对话文本。Signal 也是开源的，经过同行评审，并定期审计，这意味着从安全角度来看，它几乎总是最新的。
*   [**WhatsApp**](https://www.whatsapp.com/):WhatsApp 拥有 [超过 10 亿用户](https://blog.whatsapp.com/616/One-billion) ，这是一个疯狂的数字，综合考虑。同样疯狂的是，WhatsApp 与 Signal 背后的公司 Open Whisper Systems 合作，集成了与 Signal 相同的端到端加密聊天协议。与标榜自己为安全应用的 Signal 不同，WhatsApp 首先标榜自己是一个消息应用，这意味着它有各种各样的愚蠢的聊天内容，如贴纸和 gif。虽然 WhatsApp 用于加密的信号协议是开源的，但该应用的其他部分不是，所以我们不知道幕后发生的一切。

当实际比较这两个应用程序的工作方式时，粗略的描述并不意味着什么，所以让我们深入研究一下，从最重要的部分开始，加密。

### 两者都支持端到端加密，但是 Signal 不会保存任何不必要的东西

端到端加密是我们最近经常听到的一个术语，但它是如何工作的以及它为什么对安全性如此重要并不总是很清楚。很久以前，我们以纯文本方式发送信息，这意味着任何人都可以看到信息的内容，如果他们在信息从发送者到接收者的过程中捕捉到它的话。如今，许多消息应用程序使用端到端加密。如果操作正确，端到端加密可以防止第三方查看对话。它通过向接收者发送密钥来解锁消息，这样只有他们可以查看它。为了更详细地了解它是如何工作的， [查看去年的技术审计](https://eprint.iacr.org/2016/1013.pdf) (PDF)。

这也意味着，即使是提供商，在这种情况下，WhatsApp 和 Signal，也无法看到消息的内容。所以，如果第三方，比如政府，试图访问这些信息，他们看不到信息的内容；他们会看到一堆乱码。WhatsApp 和 Signal 都使用相同的加密协议。这意味着无论您使用哪种服务，您都可以假定您的邮件是安全的。不过，它们存储个人信息和消息元数据的方式有所不同。

元数据是这里的重要部分。元数据可以显示你在何时向谁发送信息。你可能还记得斯诺登泄密事件中的术语， [，因为中情局正在收集电话通话的元数据](https://gizmodo.com/this-government-phone-tapping-thing-just-got-as-bad-as-511762830) 。虽然 WhatsApp 不会在试图传递消息的过程之外保留你的消息(如果收件人离线，它会留在 WhatsApp 的服务器上，直到消息通过)，但它确实会收集你的许多其他信息。根据他们的 [隐私政策](https://www.whatsapp.com/legal/#privacy-policy) ，这包括使用和日志信息、设备信息、联系信息、cookies、状态更新(比如你上次在线的时间)，以及你的位置(如果你选择分享的话)。他们也可以使用其他人的信息将元数据放在一起。例如，如果你*没有*共享你的联系人列表，但是你的一个朋友共享了，而你也在其中，那么他们可以将这两条信息放在一起。还值得记住的是，脸书拥有 WhatsApp，这意味着它为广告定位分享数据。 [你可以选择退出这个](https://lifehacker.com/whatsapp-and-facebook-now-share-data-for-ad-targeting-1785743624) ，但这是一个值得注意的功能，因为两者之间的关系会让一些人不舒服。无论从哪个角度来看，这些都不是坏事，但仍然值得一提。

安全研究人员还倾向于担心这样一个事实，即默认情况下，WhatsApp 会将你的数据，包括聊天记录，备份到你手机的谷歌或 iCloud 账户中。这意味着人们可能会不小心将一些数据交给第三方而没有意识到。当然，您可以禁用这些备份，但是许多用户可能没有意识到这一点。对 WhatsApp 安全性的最后一个挑剔来自这样一个事实，即参与对话的每个人都需要使用支持加密的最新版本的应用程序才能工作。对我们大多数人来说，这不是什么大事，但如果你有一个朋友从不更新他们的应用程序，加密可能不会在对话中出现。

信号依次是我见过的最短的隐私策略。它们只存储服务运行所需的内容，如您的电话号码、随机密钥和您的个人资料信息。然后，它会根据发送信息的需要保留您的 IP 地址。像任何服务一样，Signal 可以并将向合法请求的政府实体提供他们拥有的信息，但他们一开始就没有多少数据。

这两个选项都是安全的，但 Signal 更加隐私友好，部分原因是它缺少 WhatsApp 拥有的功能。

### WhatsApp 的用户和功能更多

WhatsApp 最初是一款消息应用，后来增加了安全功能。信号最重要的是安全。正因为如此，WhatsApp 现在有了更多传统的聊天功能。Signal 和 WhatsApp 还包括语音通话和视频通话，以完善即时通讯应用。

WhatsApp 允许你发送已读回执、查看打字提示、静音对话、屏蔽联系人、设置通知以及自定义下载内容。您可以发送您的位置、您正在收听的内容、语音备忘录、添加 gif 等内容。如果你在 iOS 上，WhatsApp 与 Siri 集成，这样你就可以在旅途中发送消息或听取消息。WhatsApp 也有一些自己的技巧，比如广播列表，这是一种看起来像是直接发送给收件人的群发消息，几乎就像一种老式的短信群发垃圾聊天。你甚至可以为每个对话主题设置壁纸背景。WhatsApp 几乎做了 iMessage、Messenger 或 Hangouts 之类的所有事情。

Signal 也有很多功能，尽管它们更低调，也更实用。Signal 包括现代风格的群聊功能，支持包括 gif 在内的多媒体消息，可以发送您的位置，并且您可以设置消息在设定的时间后自动删除。总的来说，它更简单，更新时间更长，但这是故意的。越简单，就越容易保证数据的私密性和安全性。

### 也没有优秀的桌面应用程序

尽管 WhatsApp 拥有我们在即时通讯应用中所熟悉的所有新奇功能，但奇怪的是，它在桌面应用方面落后，尽管 Signal 只是稍微好一点。

你可以在电脑上接入信号，但它是一个 [的 Chrome 扩展](https://chrome.google.com/webstore/detail/signal-private-messenger/bikioccmkafdpakkkcpdbppfkghcmihk?hl=en) 。尽管如此，它工作得足够好。反过来，WhatsApp 的 [有 web app](https://web.whatsapp.com/) 和 [桌面客户端](https://www.whatsapp.com/download/) ，但它们使用你的手机访问 WhatsApp，这意味着你的手机需要在附近。如果您的手机没有连接到互联网，桌面客户端将无法工作。这是一种清洗，都不是伟大的，但至少他们有选择。

### 结论:WhatsApp 的安全性很好，Signal 的隐私性最好

好消息是 Signal 和 WhatsApp 都很棒，都做得很好。你可能会去你朋友所在的地方，但是如果你要在这两个选项中选择，那么选择适合你需要的。

哪个最适合你取决于*为什么*你在使用一个信息应用。如果你想要安全，比如，你不希望某个随机的人或政府阅读你的信息，那么 WhatsApp 对你来说足够了，尽管在安全漏洞的情况下，WhatsApp 会在存储中有更多关于你的信息。如果你更喜欢一个成熟的，全方位的，“没有人可以看到你以任何方式做的任何事情，无论是什么形式”的解决方案，那么 Signal 就是你的应用程序。
