# 通过这个红十字会应用程序学习基本的宠物急救

> 原文:[https://life hacker . com/learn-basic-pet-急救与此红十字会-app-1829249906](https://lifehacker.com/learn-basic-pet-first-aid-with-this-red-cross-app-1829249906)

当我第一次得到我的狗时，我花了很多时间给兽医办公室打电话。我没有养过狗，像任何新妈妈一样，我有无数个“这正常吗？”问题。我的毛球也喜欢街边食物，街边食物是指其他人丢弃在街上的食物。

Watch

他现在 15 岁了，但是我们早期的日子是…充满挑战的。现在，红十字会发布了一个应用程序，可以让其他新宠物父母的生活稍微轻松一点。这款名为 [宠物急救](https://play.google.com/store/apps/details?id=com.cube.arc.pfa&hl=en_US) 的应用程序提供了一些建议，告诉你的宠物什么是正常行为，以及当你的宠物遇到麻烦时该怎么做。例如，如果他们似乎对某些东西过敏，遭遇车祸，或者需要 CPR。

显然，如果你的狗溺水了，你不会启动一个应用程序来找出该做什么，但该应用程序可以作为一种指南，你可以事先阅读，教育自己如何处理这些情况。宠物急救好让大家都知道。就像人一样，身边有一个会对宠物实施 CPR 的人真的是救命恩人。

对于应用程序无法帮助你解决的问题，它可以帮助你找到最近的兽医医院。您还可以使用该应用程序查找宠物友好型酒店。

手机上有一个应用程序不能代替真正的兽医，就像 WebMD 不能代替医生一样，但通读一些信息可以让你对如何护理你的宠物有一个像样的了解，当紧急情况真的到来时，这可能是无价的。
