# 学习变戏法最简单的方法

> 原文：<https://lifehacker.com/the-easiest-way-to-learn-how-to-juggle-1797315089>

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-OiVSDqT7nNw&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-OiVSDqT7nNw&start=0) 

如果你一直想学习如何变戏法，你会很高兴知道它其实很容易学。你需要的只是一点耐心——和几个塑料袋。

Watch

在上面这段来自 YouTube 频道 的视频中，最长时间杂耍的世界纪录保持者大卫·斯里克展示了大多数杂耍者是如何开始的:用围巾。没错，如果你想学变戏法，你可以扔漂浮的围巾，而不是球、环、警棍、别针或火。

你练习的方式很简单。坐在地上。从两条围巾开始——一条在你左边的地上，一条在你的左手。现在把左手的围巾扔到右边，捡起地上的那条。在另一边重复，你就有了基本的技巧。现在添加第三条围巾。拿两条围巾，一条放在你的左边。左手扔围巾，左手捡围巾，右手扔围巾，右手捡围巾。扔，捡，扔，捡，等等。最终，当它们悬在空中时，你会足够快地扔出并接住它们。

最重要的是，你不需要去买专门的杂耍围巾。你可以使用塑料购物袋，因为它们很容易被拿走。随着时间的推移，你将能够使用同样的技巧继续玩球或豆袋。当你准备站着做的时候，或者你在尝试更重的东西的时候，试着站在床边，让事情变得更简单。坚持下去，你很快就会成为世界级的魔术师。
