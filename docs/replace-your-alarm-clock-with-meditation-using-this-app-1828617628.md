# 使用这个应用程序，用冥想代替你的闹钟

> 原文:[https://life hacker . com/replace-your-alarm-clock with-mostation-using-this-app-1828617628](https://lifehacker.com/replace-your-alarm-clock-with-meditation-using-this-app-1828617628)

我们都知道被闹钟叫醒是最糟糕的起床方式，尽管可以说是最有效的。

Watch

[iOS 和 Android 应用程序 bed itiations](https://highlymeditated.com/beditations/)有点改变了我们如何入睡和如何醒来的想法，不仅提供引导冥想让你在晚上入睡，还提供早上冥想让你醒来，取代传统的闹钟。

晚上，你选择一个舒缓的背景声景来帮助你入睡，选择一个醒来的时间和一个你想开始新一天的冥想。

到了早晨，你选择的冥想会“轻轻地把你从睡眠中唤醒”

该应用程序提供了一些免费选项，如果你决定这是为你准备的，并想改变一些事情，也有一些付费选择。

这是一个有趣的想法。虽然在冥想中醒来远没有刺耳的闹铃刺耳，但在有人引导你冥想的声音中醒来还是有点奇怪。我确实发现最初的几个有点不和谐，但后来真的很欣赏缓慢的醒来，而不是被吓醒。

如果你真的决定尝试一下，我建议你在周末不需要在某个特定的时间起床的时候尝试几次。如果你心中有一个明确的起床时间，第二个闹钟在最初几天也不是最糟糕的主意
