# 为什么我喝“蹩脚”啤酒

> 原文：<https://lifehacker.com/why-i-drink-crappy-beer-1819228643>

我喜欢各种各样的啤酒，但如果你打开我的冰箱，你会发现大多数冷激活罐的库尔斯轻。为什么？因为现在*正好* *晴*。没错，是我说的。



我不确定这是什么时候发生的，但在某个时候，拿一罐六瓶装的大众市场啤酒成了某种社交失礼。“你为什么要喝那个？”当我举办家庭聚会时，或者当我坐在酒吧里喝着几乎透明的啤酒时，有人会问我。显然，不点牛奶黑啤、IPA 或一些用南瓜香料或其他东西酿造的精酿啤酒是一种犯罪。老实说，我有点讨厌这种发展起来的潮人啤酒文化。就像你没尝过某某啤酒厂的新玩意儿就不酷一样。感觉就像人们把他们的“啤酒信誉”戴在袖子上，好像他们凌驾于我们这些吃银弹的平民之上。我喜欢小型手工酿酒厂的存在，真的，但是当你喝啤酒的时候，你并不比其他人更好。你还在喝该死的啤酒。

我明白了——人们基本上想知道，当我可以得到一个厚厚的牛排汉堡，上面有菠萝和红烧酱以及一堆其他装饰时，我为什么要麦当劳的芝士汉堡。因为简单，实惠，坦白说，舒服。这是 [和我喝盒装酒](https://lifehacker.com/why-you-should-be-drinking-boxed-wine-1794001928) 的原因一样——这也很好，非常感谢。有时候，当我和几个朋友点一杯饮料，或者啜饮一些“既振奋人心又富有挑战性”的东西时，我不想扩大自己的味觉。有时候我只想喝点淡啤酒，喝点小酒，这样我就可以忘记这个充斥着可怕的人的可怕地方，开怀大笑。

我的意思是，我很清楚我不是什么大学兄弟会的成员，我买得起“好”啤酒，但事实是我几乎不想要它。很多推荐给我的“好”啤酒味道很好，但是它们感觉很重，很难喝。蹩脚的东西很便宜，味道也够好。如果像百威啤酒、库尔斯啤酒和米勒啤酒这样的大众市场比尔森啤酒味道不好，它们就不会存在这么久。我还喜欢调制啤酒鸡尾酒 ，比如米其林达斯或红啤酒，劣质啤酒是一个很好的基准，所以我仍然可以品尝那些饮料中的其他好东西。此外，我经常用啤酒做饭，这种蹩脚的东西对我也有好处。

最后，我可以喝很多酒而不会感觉不好，无论是以一种内疚的“我在酒上花了太多钱”的方式还是以一种“我实际上醒来时没有宿醉”的方式。关于库尔斯淡啤酒的笑话是，它们被掺水了，或者大部分是水，是的！我喝了几杯，保持了相当多的水分，我可以更轻松地调整自己的节奏，它们的卡路里也少得多，所以我不会在每次想放松的时候破坏我的饮食。可以说，我也可以喝啤酒。所以，想笑就笑我吧，但我认为每个人都应该有一杯自己最喜欢的蹩脚啤酒，可以一直依赖。你呢？
