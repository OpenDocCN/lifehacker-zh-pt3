# 飞行自由桌面

> 原文:[https://life hacker . com/the-flying-freedom-desktop-1787800629](https://lifehacker.com/the-flying-freedom-desktop-1787800629)

读者[snovcrash](http://kinja.com/snovvcrash)在我们的 [桌面展柜](http://kinja.com/tag/desktop-showcase) 分享了这个定制的桌面，它也让我们感觉像在飞。以下是如何在你的电脑上获得相同的外观。

Watch

snovcrash正在使用雨量计，如果你不熟悉， [我们的入门指南](http://lifehacker.com/how-to-create-an-attractive-customized-desktop-hud-wit-5828789) 可以帮你解决。

一旦你有了基本的下来，这里是你需要的:

*   [壁纸](http://wallpapersonthe.net/18196-birds-nature-forest/)
*   用于 Windows 的 [雨量计](https://www.rainmeter.net/) 系统调试和监控工具
*   雨量计 [谜组曲](http://kaelri.deviantart.com/art/Enigma-103823591)
*   时钟和日期雨量计的优雅 2 皮肤
*   雨量计的 [SimplyNova 皮肤](http://novasev.deviantart.com/art/Simply-Nova-267726929)侧边栏和系统监视器的
*   右上角的 [为雨量计的笔记本](http://hammer-and-nail86.deviantart.com/art/Notebook-215230161) 为记事本
*   [SimpleRecycleBin 皮肤](http://maxvanijsselmuiden.deviantart.com/art/Simple-Recycle-Bin-201077076) 为雨量计定制回收站，很明显

你也可以在下面的链接中看到另一个桌面，所以其中一些正在第二个显示器上使用，但这个真正引起了我们的注意。如果你喜欢的外观，或者对它有疑问，点击下面的链接到snovcrash 的 kinja 博客来问你的问题！
