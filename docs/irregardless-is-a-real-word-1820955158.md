# “不管”是一个真实的词

> 原文：<https://lifehacker.com/irregardless-is-a-real-word-1820955158>

没有一个词像“不管”这样受到如此多的词汇鄙视——我只是打了一个寒噤。但与它经常被归入其中的自创术语不同，包括“大概”和“谢伯特”，从技术上讲,*是一个真实的词。韦氏词典词典是这么说的。* 



Kory Stamper 是韦氏词典的词典编纂者，也是《逐字逐句:词典 的秘密生活》一书的作者，他在这段视频中解释说，这个词在 20 世纪初首次用于美国的方言中，在英语口语中仍然很常见。它的意思是“不管”，由两个词混合而成:“不管”和“不管”

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-5_fUMcTb1jI&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-5_fUMcTb1jI&start=0) 

在它来自的方言中，“不管”是一个强调的“不管”因此，如果你的母语是使用这个词的特定方言，斯坦普告诉 [商业内幕](http://www.businessinsider.com/irregardless-real-word-regardless-kory-stamper-education-dictionary-mean-girls-lexicon-merriam-webster-2017-6) 你可能会有这样的对话:

> 我可能会说，“爸爸，让我借一下车。我是一个真正的好司机。”他会说，“不管怎样，我不舒服。”我会说，“哦，但是来吧。我会把它弄得更详细，我会在里面加油。”他会说，“不管，不。”这个“不管”的意思是停止对话。

然而，在标准英语中...不，它很笨重，而且很容易假设前缀“ir”会使它“T0”而不是“T1”...啊？斯坦普指出，“仅仅因为一个词是一个词，并不一定意味着你应该使用它”，并鼓励人们使用“不管”来代替。重复强调:用“不管”代替。

如果读这篇文章让你生气，请放心，这个词可能不会卷土重来。根据一项的语法民意调查，74%的投票者说“不管”不是一个真实的词，尽管我们都知道大众投票有多重要。
