# RIP USB:这些是 2017 年要关心的端口和连接器

> 原文：<https://lifehacker.com/rip-usb-these-are-ports-and-connectors-to-care-about-i-1796823560>

我们不想打断你，但过去 20 年来你用来连接相机、控制器和其他小发明的传统 USB 端口对这个世界来说并不长久。整个队伍被一些很好的选择所取代，尽管他们都有各自的优缺点。如果你想在今年升级你的设备，这里有你需要熟悉的端口，以跟上当今的技术。

### 接口

USB-C 是传统 USB 端口的一种更小的可逆替代产品。它的连接器是对称的，所以你再也不用猜哪边朝上了。把它插进去。

与苹果 iOS 设备和 Mac 配件专用的 Lightning 端口不同，USB-C 既可以在 Android 智能手机和笔记本电脑等移动设备上找到，也可以在台式机、显示器和外部驱动器等主机设备上找到。简而言之，它将成为你连接一切的新方式。

在智能手机中，USB-C 正在取代设备底部的微型 USB 连接器。一些 USB-C 加密狗与配备 USB-C 的智能手机兼容，这意味着你可以将 USB-C 加密狗插入谷歌 Pixel，并连接外部驱动器或以太网电缆等附件。

目前，USB-C 是相当新的，但并不完全普遍。当然，它存在于你的新 MacBook 和最新的 Android 手机中，但最近微软决定放弃其最新笔记本电脑的未来连接，声称其端口选择满足了大多数用户的需求。

像微软这样的公司一直回避不可避免的未来，选择在他们最新的笔记本电脑上留下端口，因为害怕在没有适当生态系统的情况下给用户提供新的端口标准。

#### 缺点

像微软这样的诋毁者说得有道理。许多 USB-C 设备的质量都有问题，制造商制造的 [加密狗和电缆](http://gizmodo.com/your-guide-to-usb-c-dongle-hell-1788344714) 承诺过多，交付不足，如果你试图节省几块钱，甚至会烧毁你的设备。在寻找优质电缆或适配器时，从 Belkin、Apple 或 Anker 等知名公司购买是值得的，在寻找 USB-C 集线器时，你应该更加彻底。

USB-C 相当新，因此购买 USB-C 设备可能会带来与您当前设置的兼容性问题。你需要适配器来插入传统的 USB-A 设备，如你的手机充电线，或者你必须购买一个昂贵的基座，以确保你的旧电子设备仍然可以与你的新机器通话。

### 雷电 3

把雷电 3 想象成服用了类固醇的 USB-C。Thunderbolt 2 使用了 Mini DisplayPort 连接，但后来改用了可逆的 USB-C 连接器。除了支持 USB-C 协议，所以任何雷电 3 电缆都可以作为 USB-C 电缆，它可以传输更多的数据，并支持其他各种传输协议。你会在各种设备上发现雷电 3 端口，如苹果最新的 MacBook Pro 笔记本电脑、华硕 Predator 等 Windows 笔记本电脑、专业级显示器和高性能存储驱动器。

除了支持 USB-C 之外，雷电 3 还支持 PCI Express 协议，用于使用外部图形卡，DisplayPort 用于驱动多个 4K 显示器或单个 5K 显示器，Thunderbolt 用于菊花链设备，并提供高达 20Gbps 的传输速度，是 USB-C 的两倍

雷电 3 可以同时提供从电源、快速文件传输、互联网连接到外部显示器的所有功能，使一个端口成为多功能端口。其菊花链支持意味着您可以使用雷电 3 电缆将多个设备连接在一起，并使用单个端口将它们插入您的计算机。例如，一个雷电 3 端口可以处理连接到两个外部显示器的外部硬盘。

#### 缺点

虽然从 DisplayPort 连接器切换到 USB-C 是一个很好的举动，但你需要一个适配器来使用较新标准的旧 Thunderbolt 设备。

因为它可以处理除 USB 之外的各种协议，所以雷电 3 设备比其他设备都贵。甚至它的连接器也比竞争对手的贵。在亚马逊上，一根 3 英尺长的电缆要 20 多美元，而同样的 USB-C 专用电缆要 16 美元左右。

如果你想充分享受雷电 3 连接的好处，你需要合适的电缆。USB-C 电缆在雷电 3 端口工作，但它们不能利用其他传输协议，并将雷电 3 端口变成普通的 USB-C 端口。

### 闪电

苹果的可逆闪电连接器是其 30 针连接器的续篇，用于从 iPod 到 iPhone 4s 和第四代 iPad 的移动设备。

毫无疑问，你已经在 iPhones 和 iPads 上看到过它，但它也出现在更多的苹果设备上。苹果自己的无线键盘和鼠标使用该端口来充电和连接到电脑。Apple Pencil 也有一个用于充电的插头 Lightning 端口，需要苹果提供的 Lightning-to-Lightning 适配器。您可以购买带有 USB-A 或 USB-C 端的 Lightning 电缆，这样您就可以使用您喜欢的任何端口为您的 iOS 设备充电。Lightning 有一些很酷的技巧，比如内置的数字音频转换器，可以为耳机提供更好的音质，以及为配件充电的能力。

#### 缺点

由于 Lightning 是苹果的专有连接器，所以你不会在其他设备上找到它(除了苹果制造的配件)。甚至像 iPhone 电池盒这样的一些设备也需要不同的连接器来充电。未来，当 USB-C 无处不在时，苹果不愿意为其移动设备采用该标准将意味着你仍然需要携带一根单独的电缆来为你的 iOS 设备充电，而不是你可以在任何地方使用的双端 USB-C 电缆。

带有 Lightning 连接器的设备，如苹果的耳机，只能与配备 Lightning 的 iOS 设备配合使用。当我们的执行编辑试图将她的标准 Lightning 耳机连接到她的 MacBook Pro 时，我们发现没有适配器是不可能的，而适配器目前并不存在。

### **我的 Micro-USB/Mini-HDMI/USB-B/等呢？港口？**

毫无疑问，你已经有了一些没有 USB-C 或 Thunderbolt 端口的设备，在不久的将来，你肯定会获得更多。像数码相机这样的设备仍然有 Micro-USB 端口，打印机 rock fat USB-B 端口，我仍然有一台普通的电视，后面有一些 HDMI 端口。

你不能忽视他们，但你可以绕过他们。USB-C 电缆的末端有像 Micro-USB 或 HDMI 这样的连接器，所以你的旧技术可以在交换电缆后与你的新设备一起工作。Belkin 做了一个 [USB-C 转 USB-B 线](https://www.amazon.com/Belkin-USB-IF-Certified-Printer-Meters/dp/B00WJSPW6S?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/rip-usb-these-are-ports-and-connectors-to-care-about-i-1796823560&asc_source=&tag=kinjalifehackerlink-20) 用于连接新设备和旧打印机，苹果自己的 USB-C hub 支持 USB-A，HDMI，和另一个 USB-C 线充电。你也可以购买 [小型 USB-C 转 USB-A 加密狗](https://www.amazon.com/AUKEY-Adapter-2-Pack-MacBook-Samsung/dp/B01AUKU1OO/ref=sr_1_1?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/rip-usb-these-are-ports-and-connectors-to-care-about-i-1796823560&asc_source=&ie=UTF8&keywords=usb c adapter&psc=1&qid=1499960136&sr=8-1-spons&tag=kinjalifehackerlink-20) ，这也可以给你的传统连接器一些额外的寿命，直到你决定永远抛弃它们。简而言之，您的旧设备将继续与您的新设备一起工作，但要为所有端口合二为一的那一天做好准备。
