# 如果我突然单身了，我会在金钱上有所改变

> 原文：<https://lifehacker.com/the-money-changes-id-make-if-i-were-suddenly-single-1734692446>

一位读者最近问我，对于已经走上经济独立之路的单身人士来说，什么样的生活策略是有用的。我很幸运结婚成家，但这让我思考:如果我突然又单身了，没有妻子和孩子，有我现在的经济状况，我会怎么做？



*本帖最初出现在* [*的简单美元*](http://www.thesimpledollar.com/if-i-were-single/) *上。*

马克写了一个有趣的问题。

> 我真的很喜欢你的文章，并且在过去的一周里阅读了许多档案中的文章。你的故事和我的很像，但在某些方面有所不同。大学毕业后，我也开始了一份薪水不错的工作，但也让自己背上了债务。大约五年前，我创办了一个 YouTube 频道，大约三年前，这个频道成了我的全职工作。我现在或多或少没有债务了。
> 
> 最大的不同是我是单身。我不约会，也没有任何兴趣这样做。我没有孩子，也不打算要孩子。

> 如果你处在我的位置，你会怎么做？

我是这样理解马克的问题的:**如果我突然又单身了，没有妻子和孩子，有我现在的经济状况，我会怎么做？**

或者，换句话说，对于一个正在走向经济独立的单身人士来说，什么样的生活策略是有用的？

我想，这是一个值得思考的有趣问题。我真的不敢想象现在没有莎拉和孩子们的生活。它们真的是我生命中最闪亮的部分。

然而，如果我退后一步，想象我的生活*如果他们突然不存在*，我肯定会比现在的开销更低，因为我会从五个人的生活费变成一个人的生活费。另一方面，我会失去萨拉的收入，这意味着靠一份收入生活的时代将会结束。

我的支出仍然远远低于我的收入，但我每年的储蓄会更少。然而，我的年度开支会减少，所以我在银行的存款实际上会是我年度生活开支的很大倍数。我会有更多的自由时间，而不需要成为三个孩子的父亲或妻子的丈夫。

简而言之，我将很快走向经济独立……但我将独自前往。

那会改变什么？我的生活会是什么样子？

我将不再害怕事业上的飞跃。现在，我对我目前的写作职责很满意，其中大部分出现在 [简单的美元](http://www.thesimpledollar.com) 上。这让我可以灵活地在早上我的孩子上公交车的时候在那里，在我的孩子完成学业的时候在家，如果他们生病了，可以很容易地去接他们，可以参加学校活动，等等。当我年轻的时候，这种灵活性对我的父亲来说是非常困难的，我很高兴能够为我自己的孩子做到这一点。

不过，如果我是单身，这种灵活性就没那么重要了。它仍然会很美好，但不会是我生活中的头等大事。我的主要关注点是，我是否觉得我正在做的工作令人兴奋和满足，如果我觉得自己对某个特定工作的兴趣正在减弱，我可以准备转向其他工作。

现在，鉴于我的生活限制和我想要做出的生活选择，我有一份非常完美的工作，但如果我是单身，这些限制和选择自然会改变。

我会更积极地投资，但不会鲁莽行事。我的投资计划将减少债券和现金，增加房地产和股票。如果这里只有*我的*金融未来面临风险，我会更有可能挑战我投资组合的波动性和风险。

这是否意味着我会选择像 [比特币](http://www.thesimpledollar.com/decrypting-bitcoin/) 或 [贵金属](http://www.thesimpledollar.com/eight-money-truths-to-tell-your-annoying-facebook-friends/) 这样的东西？没有。这些事情甚至超出了我个人的风险承受能力。我只是让我的投资变得更加激进，而不是激进。

我会在晚上参加许多社区活动。这些天来，我的许多夜晚都充满了各种事情，比如准备饭菜、确保我的孩子参加跆拳道练习以及处理就寝时间。如果没有那种持续的晚间承诺，我的非工作时间将会非常不同。

我将怎么办?我几乎每天晚上都参加社区活动。简单看一下就提供了一个起点。在我家相当小的半径范围内，有 100 多个各种不同主题的小组。实际上，我有兴趣至少尝试其中的 30 组。

我可以很容易地在每个工作日的晚上加入我所在地区的不同小组，而且我很乐意这样做。我有很多兴趣，这将是一个很好的社交渠道。

我可能会比现在过得更便宜。我认为*我的*个人支出会下降。例如，看看食物:这些天我唯一一次出去吃饭是在 [当我们一家人外出的时候](http://www.thesimpledollar.com/15-strategies-for-saving-money-at-a-restaurant/) ，打包一顿五个人的饭相当困难。我可以很容易地为自己打包一顿饭，但为五个人打包一顿饭就相当困难了。

我仍然会 [做批量餐](http://www.thesimpledollar.com/how-making-meals-in-advance-saves-time-and-money-make-ahead-lasagna/) ，但那是因为我不介意连续三四个晚上晚餐吃同样的东西。我会 [摆脱有线电视](http://www.thesimpledollar.com/the-ultimate-guide-to-cutting-the-cable-cord/) 并且可能干脆彻底摆脱我的电视。

费用降幅最大的？住房。具体说说那个吧。

我会住在不同的家，不同的地方，但不是完全不同的地方。我会住在小一点的房子里，或是公寓或联排别墅里。如果我是单身，我就不需要这么大的房子了。我可以很容易地去掉一两个卧室、多个浴室和一些生活空间。那会大大减少面积。

我真的很喜欢住在爱荷华州中部，所以我可能会留在这个地区，但我会使用一个相当广泛的半径，同时寻找一个新的居住地。我可能会弄清楚哪里有就业和创业机会，我在哪里建立社交圈和社区存在，然后搬到方便的地方。

我可能会回到学校。当我在大学的时候，我研究的领域是我觉得进入职业生涯后会有不错收入的领域。虽然我非常喜欢我所学的专业，但我意识到在决定选择哪个专业时，我使用了一个很大的“金钱过滤器”。这导致我跳过了其他似乎没有明显赚钱途径的领域，比如英语、哲学、政治学和历史。

因为我已经走上了经济独立的道路，而且我的生活方式不需要大量的经济投入，所以回到学校学习这些领域的想法听起来很有吸引力。我不太担心毕业后找工作，原因有二。

第一，我不需要非常高的收入，所以很多工作对我开放。第二，我善于寻找途径 [创造自己的个体经营](http://www.thesimpledollar.com/50-side-businesses-you-can-start-on-your-own/) 。我想，很多走上财务独立之路的人都是在同一条船上。

我会去更多的地方旅行，但要非常低调。所谓“低调”，我指的是那种背着背包四处闲逛的旅行，尽量避开游客多的地方，住在便宜的住处。例如，我喜欢在我们的每个国家公园呆一两个星期，在一个简单的帐篷里露营，在小路上徒步旅行，当我累了的时候读一本书放松一下。对我来说，这听起来像是一次天堂之旅，但对小孩子来说，这是一次非常困难的旅行。我甚至可能会做一些事情，比如走阿巴拉契亚小径。

这种观点与我最近写的一篇文章一致，这篇文章讲述了 [金钱“买不到时间带来的](http://www.thesimpledollar.com/money-cannot-buy-what-time-delivers/) ”我真正渴望的旅行体验不是来自昂贵的旅行，而是来自时间。

最终，我生命中最大的变化将纯粹以时间的形式出现。我根本不会改变我在财务上的做法。如果有的话，我会加倍节俭。相反，我会利用充裕的空闲时间做一些我真正重视的事情——继续我的终身学习使命，加入一些与我的热情相关的社会团体，低调地旅行，过一种简单而快乐的生活。

我不想再单身了，但如果我单身了，看起来会像那样。
