# 完美酒吧推车的剖析

> 原文：<https://lifehacker.com/the-anatomy-of-a-perfect-bar-cart-1829343409>

酒吧推车是家中非常私人的一部分，但是你仍然应该保留一些基本的东西来制作你最喜欢的经典鸡尾酒。在这里，我们向你展示你需要建立一个功能，成年人，适当储存的酒推车的基本要素。一旦你掌握了这些基本的构建模块，你就可以自由地玩了。 [想潜入 tiki](https://skillet.lifehacker.com/how-to-craft-your-very-own-tiki-drink-1818650367) ？扩大你的朗姆酒选择。需要一杯玛格丽特吗？加入龙舌兰酒。如果你和我一样，你最终会得到一辆主要是阿玛罗和利口酒的车，这没关系。
