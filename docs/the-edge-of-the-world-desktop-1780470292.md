# 世界桌面的边缘

> 原文：<https://lifehacker.com/the-edge-of-the-world-desktop-1780470292>

这款 OS X 桌面简洁优雅，将一些简单的小工具和华丽的壁纸完美结合。如果你喜欢你所看到的，以下是你如何设置它，自定义你自己的菜单栏，并给你的 Mac 相同的外观。

Watch

这个桌面来自 DN Carr，他把它提交给了我们 Flickr 的桌面展示和展示。要让它成为你自己的，你需要使用 GeekTool，我们这里有一个很好的入门指南 。

一旦您安装了它并掌握了基本知识，您将需要以下内容:

*   [壁纸](http://www.wallpaperup.com/73184/Japan_the_arch_night_sunset_horizon_sea_ocean_calm_sky_blue_gate.html)
*   用于 OS X 创建时间和日期显示的 [GeekTool](http://projects.tynsoe.org/en/geektool/) 系统监视器和配置实用程序
*   用于正在播放显示的 [蝴蝶结](http://bowtieapp.com/) 实用程序
*   这个 [吉尔桑斯鲍尔特](http://dreadedkilla.deviantart.com/art/Gill-Sans-Bowlet-165699073) 具体来说，有一些修改(他提供了 [他在这里使用的具体文件](http://drive.google.com/open?id=0B2rHM-p4Gw38V1VlVFVXUTJNc3c) )
*   用于重新排列 Mac 菜单栏中项目的工具
*   [cDock 2](http://w0lfschild.github.io/app_cDock.html) 实用程序和 [Liteicon](http://freemacsoft.net/liteicon/) 自定义 Dock
*   [锡安图标包](http://alexlikesdesign.com/zion/) 改变系统图标
*   [香槟和](http://www.dafont.com/champagne-limousines.font) 系统字体

总而言之，这里进行了大量的调整和定制，以使它成为 DN Carr 喜欢的方式，但效果很小，很好看。像往常一样，你可以为自己的系统选择或保留你喜欢的组件——例如，也许你不需要桌面上的时间和日期，但你确实喜欢图标包和定制的 dock。不管怎样，点击下面的链接看看他是如何制作的，让他知道你有多喜欢他的作品。
