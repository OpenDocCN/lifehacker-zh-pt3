# 网购时用最浅的颜色预览衣服

> 原文：<https://lifehacker.com/preview-clothes-in-the-lightest-color-when-online-shopp-1797189344>

你在网上订购了一些东西，但当它到达时看起来完全不同。首先，你没有注意到衣领的怪异形状，也没有注意到背面用小字 写着 [【星期一很糟糕】。](https://www.reddit.com/r/malefashionadvice/comments/5k4dm8/is_there_an_actual_reason_why_stores_like_forever/)

Watch

来自 Reddit 帖子 的避免这个问题的建议:先看最浅颜色的衣服，再看其他颜色的。较浅的颜色可以让你捕捉到原本会错过的细节，而且你可以更近距离地看到它们，而不会让它们融入阴影中。

例如，下面是上面这件上衣的对比图:

在黑色上衣上，细节融入了上衣的其余部分，但像衬衫袖口上方的褶边和衬衫领带这样的东西在裸粉色版本中显得更加清晰。

你还可以注意到一些你通常不会在较轻的版本中看到的东西，比如衣服的透明度。

另一方面，黑色服装 [有助于注意镜子上的污渍](http://lifehacker.com/wear-black-when-cleaning-mirrors-to-see-smudges-and-str-5834628) 。
