# 长途飞行需要打包的基本物品

> 原文：<https://lifehacker.com/the-essential-items-to-pack-for-a-long-flight-1755742741>

当你开始一次将让你在空中停留几个小时的长途飞行时，带上一个储备充足的包包可以让你从航空旅行的折磨中解脱出来。

Watch

你可能想要某种形式的娱乐，但是跳过沉重的书籍，选择杂志、平板电脑和较小的书籍(抱歉，*威廉莎士比亚全集*！).你还需要为你的电子产品充电，所以考虑一个 [便携式电池组](http://lifehacker.com/five-best-external-battery-packs-509802431) 。扔进一些零食， [降噪耳机](http://lifehacker.com/how-do-i-choose-the-best-noise-cancelling-headphones-511026873#_ga=1.44550884.1720946729.1390842781) 或耳塞，一个空水瓶，一旦你通过安检就可以装满，你会很好地度过你的旅程，完好无损。
