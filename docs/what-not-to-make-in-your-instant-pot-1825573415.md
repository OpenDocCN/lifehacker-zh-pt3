# 不要在你的速溶壶里做什么

> 原文:[https://life hacker . com/what-not-to-make-in-your-instant-pot-1825573415](https://lifehacker.com/what-not-to-make-in-your-instant-pot-1825573415)

我真的，真的很喜欢我的速溶壶，根据许多博客、食谱和 [脸书集团](https://skillet.lifehacker.com/these-facebook-groups-will-help-you-master-the-instant-1821718330) ，世界其他地方也是如此。但是，就像任何被大肆宣传的设备一样，事情变得有点失控。知道什么时候*不*使用速溶壶和知道什么时候*应该*使用它一样有价值。

Watch

速溶壶的核心是一个容易接近的高压锅。我可能热力学差点不及格，但连我都知道:压力越高，煮的时间越短。这对于富含纤维的蔬菜、 [含有大量结缔组织的肉块、](https://skillet.lifehacker.com/5-succulent-meats-you-should-make-in-your-instant-pot-1822311136) 、 [丰盛的谷物](https://skillet.lifehacker.com/just-how-multi-functional-is-the-instant-pots-multigrai-1823336340) 和干豆来说是很棒的。

当然也有例外——速溶锅做出了 [美味的煮鸡蛋](https://lifehacker.com/tag/instant-pot?startIndex=20) ，还有 [出奇好吃的芝士蛋糕](https://skillet.lifehacker.com/make-cheesecake-for-two-in-record-time-with-your-instan-1821065475) ，但是也有食物在无处不在的多功能锅里不好吃。事实上，如果你从这些开始，你可能最终会讨厌你的 IP，那就太糟糕了。

### 精致的蔬菜

虽然即时锅将 [改变你的红薯游戏](https://skillet.lifehacker.com/cook-multiple-things-in-one-instant-pot-through-the-cle-1819410423) ，并向你输送 [一些漂亮的王牌捣碎器](https://skillet.lifehacker.com/you-dont-have-to-drain-these-instant-pot-mashed-potatoe-1823748665) ，但它对娇嫩的青菜来说并不太好。基本上，如果 [一种蔬菜非常适合热烫](https://skillet.lifehacker.com/when-and-how-to-blanch-vegetables-1819598711)——想想嫩芦笋或鲜绿色的西兰花——就没有真正的理由把你的高压锅拿出来。我在我的速食锅里煮过一次西兰花，我把它设定为 0 分钟。当 IP 受到压力时，西兰花已经做好了，按照某些人的标准，有些部分已经煮过头了。在炉子上热烫或蒸也很简单，这两种方法可以让你在烹饪时看到蔬菜，这意味着你不会得到糊状的产品。

### 易碎的贝类

虾、 [、贻贝](https://lifehacker.com/how-to-make-a-whole-mess-of-fancy-mussels-1823939510) 等煮得很快，如果煮过头会变得像橡胶一样不好吃。尽管即食锅对于 [海鲜原料](https://skillet.lifehacker.com/how-to-make-seafood-stock-out-of-scraps-1823958899) 来说是一个很好的选择，但是没有理由用巨大的压力来残忍地对待新鲜的贝类。因为速溶壶并不真的是即时的——它需要几分钟的时间来真正达到压力——它并不总是一个省时的方法。事实上，在贻贝的情况下，炉灶面是更快的路线。如果我不能直观地监控我的贝类，并在完成后立即将其从高温下移开，我也会感到紧张——这在速溶锅里是完全不可能的任务。

### 薄煎饼

它们被称为“煎饼”是有原因的，因为你应该用该死的锅来做。 [即食锅烙](https://lifehacker.com/dont-make-pancakes-in-your-instant-pot-1824084132)——或称 ipan cake——又粘又稠，外皮又厚又脆。它不好，我不喜欢它。

### 几乎所有烘焙食品

[即食罐芝士蛋糕](https://skillet.lifehacker.com/make-a-four-flavor-cheesecake-sampler-in-your-instant-p-1822396556) 都是好东西；由于速溶锅的潮湿环境，这是一种不错的烹饪方式，可以制作蛋挞或 [布丁样的甜点](https://skillet.lifehacker.com/you-should-make-molten-chocolate-cakes-in-your-instant-1822998333) 。但 IP 是一个可怕的地方烤面包，或任何蛋糕，你希望有一个微妙的面包屑。烤箱放在你的厨房里是有原因的，使用它们是没问题的。在密封、潮湿的环境中，你永远也不会得到好的面包皮，所以对即时罐烤说不吧。(不过， [可以用它来打样小面包](https://livingsweetmoments.com/crusty-instant-pot-sourdough-bread/) ，但是我们 Skillet 在尝试大规模 IP 打样时并不满意。)

### 有什么你想修改的吗

如果你喜欢在烹饪时品尝和修补食物，速溶锅可能会让你感到难过。一旦你把它封上，就没办法打开来检查了。这在很大程度上是好的，但是如果你的调料不太合适，你就会被结果所困。(你可以在最后添加一些东西，但那不太一样，你肯定不能取消季节。最好的解决方法是要么使用你喜欢的可靠的食谱，要么接受学习曲线，不要尝试昂贵的食物。

速溶壶受欢迎是有原因的。它使压力烹饪成为一种诱人的烹饪方法，这很有价值。但是，仅仅因为某种东西能做出美味的排骨，并不意味着它能做出蛋奶酥，了解速溶锅的弱点会让你专注于它的优点，让它在厨房里更有价值。
