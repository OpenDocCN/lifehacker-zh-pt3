# 关于游戏从 iOS 应用商店消失，你需要知道什么

> 原文：<https://kotaku.com/what-you-need-to-know-about-games-vanishing-from-the-io-1795863206>

苹果即将推出的 iOS 11 将结束对 32 位游戏的支持，今年秋天将推出。这意味着开发人员正急于用 64 位基础设施更新他们的游戏，可悲的是，几个不幸的游戏将从 iOS 应用商店中消失。

Watch

iOS 11 目前处于测试阶段，但即使是现在，一些 32 位游戏也没有出现在 App Store 中。你仍然可以在 [Touch Arcade](http://toucharcade.com/) 上搜索它们，正如 *Gizmodo* 的瑞德琼斯 [指出](http://gizmodo.com/32-bit-apps-arent-showing-up-in-the-app-store-but-dont-1795799607) ，“如果你使用直接链接，这个应用仍然可以下载。这意味着你仍然可以获得你打算购买或想要再次拥有的旧应用程序。”不过，这种方法只能维持一段时间；未来几个月，iOS 11 将完全放弃对 32 位游戏的支持。

苹果放弃 32 位基础设施迫使移动应用程序开发者在更新他们的旧游戏或让它们死去之间做出选择。这是一个已经到来很久的事情，一个已知的必然性 [自从今年早些时候](https://arstechnica.com/apple/2017/01/future-ios-release-will-soon-end-support-for-unmaintained-32-bit-apps/) 以来。本文采访的大多数开发者都选择投入工作来拯救他们的应用。幻想飞行游戏公司已经发布了几个手机版的桌游，正在更新旧游戏，如*《嘿，那是我的鱼》*、*长者标志:预兆*和*战斗传说:命令*。“幸运的是，我们已经采取措施升级所有 3 个应用程序以支持 64 位架构，”奇幻飞行媒体互动副总裁基思·赫尔利告诉 Kotaku。他们的三款 32 位游戏都将及时更新到 iOS 11。

Rodeo Games 的技术总监理查德·布鲁克斯表示，他的团队已经更新了他们的游戏名称*猎人 2* 和*战锤 4 万:死亡观察*，目前正在修复在 64 位 iOS 兼容性到来之前发布的*战锤任务*。甚至*可笑的捕鱼*，一个非常受欢迎的关于用电锯捕鱼的游戏， [在自上次更新以来的三年多时间里，正在进行](https://twitter.com/Vlambeer/status/871444707947282432) 的 64 位改造。

很难确定有多少游戏会从 App Store 消失。F5 Games 的创始人布兰登·波莱开发了包括 *IncrediBlox* 在内的几款游戏，他告诉 *Kotaku* 他的“大部分”游戏，比如*口袋英雄*，都不会入选。波莱告诉 Kotaku “我们目前有 5 款游戏将在苹果杀死 32 bit 时从商店中移除，”他补充说，“基本上这是我们的全部目录。有可能我们可以更新其中的两个，但其他的需要大规模重写。”他澄清说，那不一定是因为苹果；F5 使用软件 Corona SDK 构建他们的游戏，自游戏发布以来，Corona SDK 改变了他们的图形技术。

“为了发布 64 位版本，我们必须重写游戏的图形层，”波莱说。

希望 32 位 iOS 游戏会在某个地方存档。但由于用于播放它们的技术正在消亡，许多可能会成为遗迹。
