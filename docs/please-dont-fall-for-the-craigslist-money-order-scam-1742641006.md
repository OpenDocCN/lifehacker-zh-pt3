# 请不要陷入 Craigslist 汇票骗局

> 原文:[https://jalopnik . com/please-don-fall-for-the-craigslist-money-order-scam-1742641006](https://jalopnik.com/please-dont-fall-for-the-craigslist-money-order-scam-1742641006)

有相当多的[网络诈骗](http://thegarage.jalopnik.com/how-to-avoid-craigslists-rampant-sight-unseen-purchase-1707158966) 在那里，我已经谈论过其中的一些。但是我一直听说有人被“汇票”骗局骗了。

Watch

你在 Craigslist 上发布广告出售你的二手车。你列出一个合理的价格。一个人从很远的地方打电话给你，说他们一直在找这辆车。以至于他们甚至不需要看到它-他们会立即把钱寄给你，甚至支付汽车运输费。你需要做的就是坐下来兑现汇票。

这是骗局可能向几个不同方向发展的地方。我最近听说的一个案例涉及到向卖家发送的支付汽车和运输费用的汇票。卖方能否打电话给船运公司，安排提货，并用汇票上的钱付款？船运公司的号码是假的，由“买家”操作，他只会把钱装进口袋。

或者，汇票金额过高，但买家欠住在附近的朋友的钱，他会去取车。当司机取车时，你能把多余的钱给他吗？

一个非常复杂的转折甚至把 [的目标律师](http://www.abajournal.com/news/article/six_indicted_in_32m_internet_collection_scam_that_snagged_80_lawyers/) s .假客户寻求雇用我们，当我们说我们会这样做时，一个拟议的“被告”滚过来，立即给我们一张巨额支票或汇票——律师可以保留其中的三分之一，然后将另外三分之二汇给“客户”

这些骗局的核心是支票或汇票是伪造的。任何有扫描仪和彩色打印机的人都可以伪造看起来像真的支票和汇票。当你把它拿到银行时，他们通常会把它当成真的来处理。当它在系统中运行时，该工具被踢回上游的某个地方，并被送回你的银行。然后你的银行逆转整个过程-除了你现在没有任何钱你交给托运人，司机或国外的客户。

这种骗局之所以奏效，是因为它利用了卖家(或律师)想要快速达成一笔有利可图的交易的欲望。当有人过来向你扔钱时，你凭什么怀疑？

当有人寄给你一张汇票或其他据称是现金的工具时，仔细检查一下， [我已经得到了这些](http://oppositelock.kinja.com/messing-with-internet-scammers-for-a-reason-and-fun-1679193117) (我经常和那些试图这样对我的人打交道)拼写错误的银行名称。但是如果票据看起来合法，问问你的银行，在你确定它已经通过了发行银行的审核之前，还要多久。在该日期过去之前，不要释放任何东西(资金或你正在出售的物品)。你可能还想打电话给开证行，看看账户是否存在，票据是否合法等等。相信我:他们会告诉你的。

过去，我接触过的所有人都声称自己在亚洲，需要密歇根的法律服务。然后他们从加拿大给我快递了伪造的支票。这一切都没有意义，但他们希望我会跳过我没有得到的巨额费用。无所事事获得 216，000 美元可能类似于如果有人出价购买你的 PT Cruiser(只需要“一点点工作”就可以运行)的感觉。
