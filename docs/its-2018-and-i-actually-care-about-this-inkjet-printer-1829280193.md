# 现在是 2018 年，我实际上很关心这台喷墨打印机

> 原文：<https://gizmodo.com/its-2018-and-i-actually-care-about-this-inkjet-printer-1829280193>

惠普 Tango X，一款小巧的家用打印机，实际上非常酷。不，现在不是 1999 年，我也不会在我妈妈厨房角落里的 Gateway 台式机上写这篇文章。这真的是一个令人兴奋的喷墨打印机。说实话。我发誓！



但是有一些警告，主要是(很明显)你必须是那种仍然需要在家里打印东西的人。如果你不介意使用办公室打印机打印税务文件或最新的租约，那么这台打印机可能对你来说毫无意义。但是对于很多人来说，在家打印仍然是偶尔的必需品，即使这不是你每天都要做的事情。Tango X 是一款吸引人的单线打印机，不用时可以塞在壁橱里或藏在橱柜里，感觉像是在办公室打印机上打印个人垃圾或跟踪 Kinko 或仍有旧喷墨打印机的老年家庭成员的一种有吸引力的替代选择。它大约有几本精装小说那么大，由于有一个硬布封面，可以兼作纸盘，它不需要太多空间来完成它的任务。

当惠普让我坐下来谈论 Tango X 时，公司代表急于指出所有旨在吸引年轻人的酷功能。他们指出，只要你的手机上有这个应用程序，你就可以在任何地方打印，而且它可以连接到 Alexa 设备，所以我想，你可以对着它大喊大叫，打印更多。这有点像惠普市场部的某个人抓起一把千禧一代的流行语扔向墙壁。

实际上，打印机迄今为止表现良好，这并不是因为连接功能允许我在火车上从我的手机上打印一些东西，或者通过 IFTTT 连接到我的智能家居，并在每次有人新登录我的 wifi 或其他东西时打印。这些都是喷墨打印机通常不具备的简洁功能。但更整洁的是，这台打印机合上后放在我的办公区时看起来很有吸引力，因为它大约有 90%的寿命。

售价 200 美元的 Tango X 配有一个布套，可以兼作纸盘，惠普承诺将提供其他纸盘，包括一个迷人的软木纸盘。基本款探戈的零售价将降低 50 美元，不包括托盘，而是用一种更傻的塑料制成。

是的，我知道，我称之为基本的打印机塑料，并暗示 50 美元的封面使打印机更酷。但是听着，我现在已经是个成年人了。我有抱枕和书柜，我希望我家里的技术很明显，但不突兀。我喜欢东西看起来像是融入了空间，而不是大声喊着它们是技术。

为此，这个愚蠢的小打印机让我兴奋，但我还不能发誓。惠普表示，打印机中的墨水应该可以使用大约 7 到 9 个月，然后才需要更换——并暗示，随着墨盒寿命自 1999 年以来显著提高，它甚至可能更长。然而，这仍然是每 7 个月 40 美元，高于 Tango X 的 200 美元的价格，这对于我和我认识的大多数人偶尔打印来说可能太多了。所以我打算在接下来的一年里像往常一样使用这台打印机，看看墨水是否能保持住或者模糊不清。如果你不在乎墨水的潜在寿命，那么你可以从今天晚些时候开始购买 HP Tango X。
