# 用这个工具快速找到任何主要城市的时髦社区

> 原文：<https://lifehacker.com/quickly-find-the-hipster-neighborhoods-in-any-city-with-1818681545>

当你第一次参观一个城镇时，了解那里的情况会很困难。尤其是如果你要搬到一个新的地方，很难判断一个社区是否比另一个更适合你，或者你是偶然(或有意)租到了镇上最时髦的社区。现在，有一个应用程序。或者网站，实际上。

Watch

这个名为 [Hoodmaps](https://hoodmaps.com/) 的网站拥有主要城市的众包地图，充满了特定地区是否吸引潮人、游客或富人以及你可能在那里找到什么的信息。

例如，每个旧金山居民都会告诉你，码头是你通常会发现“兄弟会”的地方，在卡斯特罗的一个角落，在温暖的日子里，你可能会看到一个裸体的家伙在那里走来走去(我在镇上的第一个周末坐公交车时发现了这一点)。但是这并不完全是你在一本旅游指南中能找到的信息，或者至少不是同一本旅游指南中的所有信息。

创始人 Pieter Levels 根据自己的旅行产生了这个想法。

“每次我去一个新的地方旅行，都很难弄清楚该去这个城市的哪些地方。我经常在游客中心结束。我来自阿姆斯特丹，我知道 90%的游客永远不会了解“真正的”阿姆斯特丹，因为他们只是呆在游客中心。这是一个与荷兰文化毫无关系的虚假领域，”Levels 在发布这款网络应用的博客文章 中说道。“我想快速了解一下城市是什么样的。有哪些很酷的「臀」区？较富裕的地区在哪里？哪些地区更郊区(也许对我来说很无聊？)."

他实际上从头到尾公开建造了整个工程，记录了每一步。如果你对应用程序开发感兴趣， [他的博客文章](https://levels.io/hoodmaps/) 可能会很有意思。目前在全球 2000 个主要城市都可以买到。

帽衫地图用你可能在附近找到的人的类型进行颜色编码。选项有潮人、富人、游客、西装、大学和“正常人”你可以在地图上切换一些东西:标签、咖啡馆、房租和房价。

标签分享类似裸男洞察力或旧金山里士满区将充满雾的事情，可以通过所有标签或仅新标签进行排序。

正如 [*城市实验室*](https://www.citylab.com/life/2017/08/hoodmap-stereotype-your-city-hipsters/538218/)所言，这个想法并非没有问题。这种用户贡献的方面很容易使负面的刻板印象永久化，这些刻板印象可能是真的，也可能不是真的。每个街区的标签也都是以白人为中心的，所以你肯定看不到全貌。“正常人”到底是什么意思？

也就是说，如果你要去一个新的城市，这可能是一个快速快速了解你要去哪里的快捷方式，并制定一个稍微更有教育意义的计划，告诉你去哪里订酒店房间。
