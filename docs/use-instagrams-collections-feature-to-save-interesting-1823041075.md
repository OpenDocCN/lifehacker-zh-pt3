# 使用 Instagram 的收藏功能来保存有趣的帖子

> 原文：<https://lifehacker.com/use-instagrams-collections-feature-to-save-interesting-1823041075>

去年 [Instagram 推出了收藏](https://lifehacker.com/instagram-adds-collections-to-help-organize-bookmarked-1794385586) ，这是一种将应用程序中的图片组织到不同的策划组中的方式。这个功能已经存在了大约一年，但是大多数人都没有意识到它的存在。

Watch

如果你经常使用 Instagram，这个功能对于保存你认为有趣的帖子非常有用，这样你就可以在以后找到它们，而不用无休止地滚动你的订阅源。

要使用它，你只需点击图片右下角的书签图片，然后当它出现在图片上时，选择“保存到收藏”。在那里，系统会提示您创建一个新的收藏或将该图片添加到您已经拥有的收藏中。

要做到这一点，关键是在保存图片时要有条理，而不是把它们都放在一个文件夹里。

虽然一个普通的文件夹本质上对任何人都没有好处(并且你回去查看它的机会几乎为零)，但是当你保存图片时，将它们组织到不同的桶中实际上是一个非常有用的做法，并且当你以后想找到那些很棒的 Instagram 帖子时，可以节省你的时间。以下是一些利用该特性的方法:

### 可以尝试的餐厅

如果你的朋友正在享用龙虾大餐，而你希望它就在你的面前，把它加入到你的餐馆尝试文件夹中。稍后，当你想知道去哪里吃饭或者晚餐做什么的时候，浏览一下你收集的食物照片，寻找一点灵感。

事实上，我更倾向于做我想尝试和烹饪的食物，而不是我想亲自去吃的地方。例如，几周前，我的一个朋友上传了这个来自纽约某个高档意大利餐馆的看起来美味的汤团。因为我住在旧金山，所以晚餐的通勤会很麻烦，但在看到照片并在网上搜索了一些食谱后，我正在去一个美味的晚餐的路上，如果我没有看到照片，我可能不会自己想出这个主意。

### 旅行的地方

没有什么比去度过一个美好的假期更能激励人们在 Instagram 上发布照片了——一个你希望自己正在度假的美好假期。将沙滩和风景如画的山脉的照片保存到“旅行之地”文件夹中。更好的是，如果你的朋友正在你想去的特定国家旅行(或者已经计划去)，那么就为每个目的地创建特定地点的收藏。

例如，我男朋友和我计划今年秋天去意大利和法国旅行。这是我们从去年就开始计划的旅行，我们的计划开始时只考虑了基本的国家。在过去的一年里，随着朋友们去那些地方旅行，我保存了一些看起来有趣的东西。现在，我不用说“哦，我在杰克的 Instagram 上看到了这个看起来很酷的东西”或试图记住某个东西是由杰克或约翰或朱莉发布的，我只需打开我方便的意大利文件夹，立即找到我正在寻找的帖子。

### 宠物/婴儿照片

如果你有一只宠物或一个婴儿，或者你的兄弟姐妹刚刚得到一只宠物或有了一个婴儿，那么人们会希望你有照片。从生日聚会/假日聚会中收集所有最好的照片，然后当人们问起时展示给他们看，而不是在你手机的相机胶卷中挖掘绝望的深渊。
