# 为什么你总是在同一个地方长痘痘

青春痘不公平。我们没有要求它们，我们当然不应该让它们进入成年，当你终于让一个巨大的粉刺消失，但几天后它又回来了，这是特别侮辱人的。

今天的热点问题很简单。读者[puddlv](http://vitals.lifehacker.com/1820124520)问道:

> 为什么我会在完全相同的地方长痘痘？

事实证明，面部某个部位比其他部位更容易长痘痘可能有几个不同的原因。囊肿性痤疮是臭名昭著的露营在你的皮肤下，并导致痤疮，需要永远停止再现。“这通常是毛孔被皮脂和皮肤细胞堵塞、发炎和扩张后产生的，”温哥华皮肤科医生凯蒂·贝莱兹奈 博士说。“同一个粉刺会变大或变小，这取决于多种因素，包括你的皮肤分泌了多少油脂。”

但是你不需要一个隐藏的囊肿来策划一片粉刺。波士顿的皮肤科医生 Susan Huang 博士指出，粉刺自然出现在皮肤出油最多的部位，如你所料，包括面部、胸部和背部。女性的下巴和下颚经常长粉刺，因为那里的皮肤对荷尔蒙波动更敏感，荷尔蒙波动会产生额外的油脂。

你也应该考虑你可能做的事情，把污垢和油擦进皮肤。“例如，你的自行车头盔可能会堵塞你额头的毛孔，从而[导致]该区域出现粉刺，”黄博士告诉我们。

两种皮肤都推荐水杨酸和过氧化苯甲酰产品，你可以在任何药店找到。但是如果你的痤疮很严重，或者有很深的囊肿，你可能需要医生的帮助。( [这里有一个很好的纲要](https://www.aad.org/public/diseases/acne-and-rosacea/severe-acne) 当你寻求治疗严重痤疮时，你会看到什么。)
