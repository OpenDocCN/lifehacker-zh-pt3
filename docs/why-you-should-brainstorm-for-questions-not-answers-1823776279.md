# 为什么你应该头脑风暴的问题，而不是答案

> 原文：<https://lifehacker.com/why-you-should-brainstorm-for-questions-not-answers-1823776279>

头脑风暴的目标是找到问题的可能解决方案，但是这个过程经常成为直言不讳的人的平台，他们一次又一次地提供相同的观点。取而代之的是:让每个人提出更多关于这个问题的问题，这样你就能更好地理解这个问题到底是什么。麻省理工学院领导中心的执行主任哈尔·格雷格森提出的这种违反直觉的方法，让每个人都思考并参与进来，可以让一场平淡无奇的头脑风暴变得更加有效。

根据格雷格森的说法，集思广益寻找答案可能会表现出一种破坏性的群体动力，扼杀创造力和参与。毕竟，想出答案是很难的，没有人想显得愚蠢，所以人们最终会把自己的想法藏在心里。但是当你集思广益寻找问题时，你可以更深入地看待问题，并给你的伙伴们机会，让他们对问题提出新的观点。格格森是这样做的:

> 1.**设置舞台**:选择一个问题，邀请一个小组从不同角度帮助你考虑，用粗线条的方式描述，这样你就不会限制小组的思维。
> 
> 2.**只提出关于挑战的问题**:重新引导任何提出解决方案的人，并解释不允许有冗长的开场白或理由。追求问题的数量和速度。这将为理解或解决问题开辟新的途径。
> 
> 3.追求至少其中一条途径:设计一个近期行动计划来保持动力。

提出问题比试图得出答案容易得多，所以会有更多的人参与进来。随着会议的进行，人们的问题可能会激发某人的天才，创造出前所未有的答案。在一次普通的头脑风暴会议中，你在等待闪电的到来。为什么不制造更多的风暴云，从而为闪电创造更多的机会呢？在你的下一次会议上尝试一下，看看它会带你去哪里，如果你想了解更多关于这个过程，请查看下面的链接。
