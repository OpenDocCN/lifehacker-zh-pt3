# 立即使用 Amazon Prime 来帮助那些需要帮助的人

> 原文：<https://lifehacker.com/use-amazon-prime-now-to-help-those-in-need-1821332895>

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-8oKl_qDJHt4&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-8oKl_qDJHt4&start=0) 

你可能经常看到这样的场景:有人独自坐在寒冷的户外，举着一个牌子，上面潦草地写着求助。好吧，即使你不再携带现金，或者有时间跑到附近的商店，你仍然可以通过这个聪明的善举来帮助他们。

Watch

[Amazon Prime Now](https://primenow.amazon.com/?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/use-amazon-prime-now-to-help-those-in-need-1821332895&asc_source=&tag=kinjalifehackerlink-20) 让某些地区的 Amazon Prime 订户在两小时内订购并收到商品。我曾经用该服务订购了一个键盘，45 分钟就拿到了。快速得到你需要的东西是非常不可思议的，但是 YouTuberT5】Rob BlissT7】想出了一个更加无私的方法来使用它。布利斯不再为自己买东西，而是开始为他在街上遇到的无家可归的人订购东西。它快速、简单，而且它实际上为人们提供了他们需要继续保持的东西。你可以在上面的视频中看到这一切是如何进行的。

如果你想尝试一下这种善举，你首先需要注意几件事:

*   这个人的名字。

*   离他们最近的地址。

*   这个人需要什么。

一旦你有了这些信息，你就可以在继续一天的工作时快速下单。只要确保你让那个人知道有东西在路上，这样他们就不会离开，并给快递员一张纸条，说明把包裹交给谁。布利斯为他的实验写了一条信息，大概是这样的:

哦，别忘了给你的快递员小费！请记住，为了让这种事情发生，你需要成为 Amazon Prime 的订户，并且在你所在的地区有 Amazon Prime。此外，这可能只适用于在某一天有更多快递员外出的人口密集的大都市地区，这是毫无价值的。不过，下次你看到有需要的人时，可以考虑尝试一下。你可能会让某人的一天无限美好。
