# iPhone 的最佳系统显示器

> 原文:[https://life hacker . com/the-best-system-monitor-for-iphone-5846917](https://lifehacker.com/the-best-system-monitor-for-iphone-5846917)

乍一看，系统显示器在你的 iPhone 上可能不像在台式电脑上那么有用，但它们可以包含大量有用的数据。这包括详细的电池寿命故障、存储空间、数据速度等。对于普通用户来说，我们最喜欢的 iPhone 系统显示器是 Omnistat。

### [Omnistat](https://itunes.apple.com/us/app/omnistat-system-status-activity/id898245825?mt=8)

**平台** : iPhone(和 iPad)
**价格**:1.99 美元
[**下载页面**](https://itunes.apple.com/us/app/omnistat-system-status-activity/id898245825?mt=8)

### 特征

*   可定制的通知中心小部件让您决定显示什么数据以及在哪里显示
*   适用于 iPad 和 iPhone 的通用应用。还包括 Apple Watch 支持
*   显示活动和统计数据:设备名称、设备型号、当前操作系统版本、当前操作系统版本号、设备正常运行时间、Wi-Fi 详细信息、移动运营商数据使用情况、下载和上传速度、存储信息、CPU 使用情况、电池详细信息等
*   选择您的数据套餐何时重置，以便您始终能够准确跟踪移动数据的使用情况
*   估计剩余电池时间

### 它擅长的地方

Omnistat 最大的优势是通知中心小部件。系统监视器是您想要快速访问的东西，通知中心小部件是实现这一点的一个聪明方法。有了 Omnistat，您可以定制哪些统计数据作为小部件出现，任何时候您想浏览它们，只需下拉通知中心。Omnistat 为每个活动提供了自己的小部件，因此您可以轻松地定制通知中心的布局。

除此之外，Omnistat 提供了大多数人想要的细节。这包括电池寿命，包括对剩余通话、文本和数据时间的估计。您还可以轻松跟踪 Wi-Fi 和手机数据的使用情况。对于数据使用，Omnistat 支持为单元数据创建一个自动重置日期，以便它始终与您的数据计划保持一致。如果你在 16GB 的 iPhone 上运行，或者你总是靠在墙上，还有剩余空间，那么存储小工具对于检查你的剩余存储空间非常有帮助。Omnistat 有大量其他小部件，从网络细节到设备 CPU 使用情况，因此它应该有您最需要访问的数据。

### 不足之处

Omnistat 之所以出色，是因为包含了通知中心小部件。然而，Omnistat 并不是最广泛的系统监视器。虽然它确实跟踪了普通用户想要的大多数活动，但对于任何寻求更细粒度方法的人来说，它缺少了很多数据。同样，Omnistat 提供了大量的概览数据，但你不能专注于更具体的信息，如你倾向于在什么时间使用更多的数据，Wi-Fi 网络的历史记录，或类似的任何其他信息。

### 竞争

对于一般人来说，Omnistat 非常适合浏览一些宽泛的信息，但如果你想深入挖掘数据，这不是你想要的应用程序。幸运的是，系统监视器空间充满了可靠的应用程序。

对于爱海量系统细节的人来说， [**系统监控终极**](https://itunes.apple.com/us/app/system-monitor-ultimate/id740765454?mt=8) (免费)值得一看。系统监视器终极版显示大量关于你的 CPU、GPU、网络、活动连接等的数据。System Monitor Ultimate 并不是最好看的系统显示器，也没有太多功能，但它是免费的，可以显示你在 iPhone 上可以跟踪的几乎所有数据。没有通知中心小部件支持，但如果你不喜欢小部件，System Monitor Ultimate 是你想要的应用程序。

如果你正在寻找与具有更多交互功能的系统显示器旗舰版相同的数据量，那么 [**系统状态**](https://itunes.apple.com/us/app/system-status-activity-monitor/id401457165?mt=8)**(2.99 美元)符合要求。除了监控大量数据点、网络信息、电池和内存之外，系统状态还会显示文件统计数据、详细的页面统计数据、跟踪三分钟的后台活动，并允许您通过电子邮件导出所有这些图表。如果你喜欢看*和*保存活动监视器数据，但不关心小部件，系统状态完成这项工作。**

**最后，Omnistat 并不是唯一带有小部件的系统监视器， [**用法小部件**](https://itunes.apple.com/us/app/usage-widget/id922914146?mt=8) (Free/99)和[**snap stats**](https://itunes.apple.com/us/app/snapstats-device-stats-info/id939343881?mt=8)(Free)都包括通知中心小部件以及基本的系统监视器。与 Omnistat 不同，这两个应用程序都在一个小部件中显示所有的统计数据，因此您不能随意移动它们或对它们进行定制。不过这是个人喜好的问题，所以如果你不介意把所有数据都放在一个地方，这两款应用都值得一看。**
