### 哪些台式机和笔记本电脑兼容 macOS Mojave？

据苹果称，这些是你可以与 macOS Mojave 配合使用的各种电脑配置。如果你有一台旧的 Mac 台式机或笔记本电脑，你就不走运了(除非 [你有创意](http://dosdude1.com/mojave/) ):

*   [MacBook](https://support.apple.com/en-us/HT201608)2015 年初或之后推出的机型

*   [MacBook Air](https://support.apple.com/en-us/ht201862)2012 年年中或之后推出的机型

*   [MacBook Pro](https://support.apple.com/en-us/HT201300)2012 年年中或之后推出的机型

*   [Mac mini](https://support.apple.com/en-us/HT201894)2012 年末或之后推出的机型

*   [iMac](https://support.apple.com/en-us/HT201634)2012 年末或之后推出的机型

*   iMac Pro(所有型号)

*   [2013 年末推出的 Mac Pro](https://support.apple.com/en-us/HT202888) 机型，以及 2010 年中期或 2012 年中期推出的配备推荐金属功能图形处理器的机型，包括微星游戏镭龙 RX 560 和蓝宝石镭龙 PULSE RX 580

### macOS Mojave 会毁了我的系统吗？

尽管苹果的升级过程相当顺利，但升级到新版本的操作系统总有不顺利的可能。我自己还没有遇到过这种情况，但这并不是说这是不可能的。根据电脑的年龄和规格，升级到新版本的 macOS 可能会导致速度变慢或其他意外的性能问题。

如果您发现您的系统在安装了 macOS Mojave 后比没有安装时更糟糕，您有几个选项可以在事后解决这个问题。在你开始升级到 macOS Mojave 之前，确保你做了一个 [Time Machine 备份](https://support.apple.com/mac-backup) 你系统的当前状态。(为此，您需要连接的驱动器或 Time Machine 支持的网络存储设备。)然后，如果 macOS Mojave 出现任何问题，您可以随时从备份中恢复您的系统，继续您的生活。

正如 [Macworld](https://www.macworld.co.uk/how-to/mac-software/downgrade-macos-mojave-3581872/) 所指出的，移回早于 High Sierra 的 macOS 版本可能是一个问题，因为你必须为 HFS+文件系统而不是 High Sierra 和 Mojave' APFS 重新配置你的驱动器。

如果您不能备份 Time Machine，您也可以(很可能)恢复到 Mac 附带的操作系统。这将会抹掉您的数据和设置，直到您将 Mac 同步回 iCloud，并且您必须重新安装您的应用程序，但这是一个选项。开机后立即按住 Mac 的正确组合键 (通常是 Command+R)并选择重新安装 macOS 选项，即可启动恢复。

### 如何安装 macOS Mojave

这个很简单。使用 Mac 上的任何浏览器点击 [此链接](https://itunes.apple.com/us/app/macos-mojave/id1398502828?mt=12) 。下载并安装 Mojave。*哒哒*。

### 我安装了 macOS Mojave。现在怎么办？

首先，确保你在 macOS Mojave 中选择是使用 [亮模式还是暗模式](https://lifehacker.com/switch-between-light-and-dark-mode-in-macos-mojave-with-1827640349https://lifehacker.com/switch-between-light-and-dark-mode-in-macos-mojave-with-1827640349) 。拥有一个漂亮的桌面很重要，因为这是你一天中大部分时间都会盯着看的东西。(如果你犹豫不决， [试试这个工具](https://lifehacker.com/switch-between-light-and-dark-mode-in-macos-mojave-with-1827640349) )。)

从那里，你可以在马科斯莫哈韦 玩一大堆有趣的东西。我的建议包括使用 Mojave 的新 Stacks 功能整理你的桌面，给自己一张可爱的动态桌面壁纸，通过苹果的更新应用阅读一些新闻，让你的所有智能家居设备与苹果 home 应用的桌面版本兼容，查看新的应用商店(任何运行 iOS 11 的人都应该很熟悉)，以及在全新的 Safari 12 中打开收藏夹图标。

如果你对苹果的 Group FaceTime 功能还没有准备好在黄金时间推出感到失望，你有 [几个其他选项，你可以使用](https://lifehacker.com/what-to-use-for-video-calls-instead-of-facetime-1828527644) 进行所有大型视频聊天。苹果的文件和屏幕共享工具“回到我的 Mac”也不见了，但是你还有 [和](https://lifehacker.com/what-are-the-best-back-to-my-mac-alternatives-1828565951) 的一些替代品。别忘了在 Mojave 中 [调整你的摄像头和麦克风权限](https://lifehacker.com/how-to-stop-apps-from-accessing-your-camera-and-microph-1828158934) ，以防某些应用程序对你生活的访问超出你的适应范围。

对于那些在过去几个月里通过 macOS Mojave 公共测试版过着狂野生活的人来说，你可能会考虑 [退出该计划](https://beta.apple.com/sp/betaprogram/unenroll) ，这样未来的增量测试版——可能不会像主操作系统更新那样稳定——不会给你带来任何痛苦。

最后，如果你怀念你为了进入 Mojave 而放弃的操作系统，我们理解。至少你还可以在浏览器里用 [重访你的老操作系统好友](https://lifehacker.com/how-to-revisit-every-version-of-mac-os-x-from-your-brow-1828537876) 。
