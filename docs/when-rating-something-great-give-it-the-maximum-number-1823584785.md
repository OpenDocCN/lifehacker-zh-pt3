# 当评价某样东西很棒时，给它最高的星数

> 原文:[https://life hacker . com/when-rating-something-great-give-it-the-maximum-number-1823584785](https://lifehacker.com/when-rating-something-great-give-it-the-maximum-number-1823584785)

“这家优步/Airbnb/餐厅真的很棒，”你坐下来写评论时心里想。你输入你喜欢的关于旅程/公寓/食物的一切，然后你将光标悬停在星级上。“嗯，”你想，“总有改进的余地！”或者“这很棒，但不是我吃过的最好的一次旅行/住宿/用餐”，然后你给 4/5 或 9/10 颗星。

不要那样做！

这可能看起来违反直觉，但评级服务不是传授你总是有改进空间的心态的时候。对于 Airbnb 的房东、优步的司机和许多其他人来说，星级评定是他们业务的一个重要部分，任何低于完美评级的都是不合格的。

我与 Airbnb 和 VRBO 的一位主持人交谈过，他对四星评级的热情洋溢、100%正面的评论感到沮丧。这两项服务都为收视率高的主持人提供额外津贴，这些津贴反过来帮助她吸引更多的业务。在 Airbnb 上，不仅仅是高平均评级，还有高比例的五星评级，都有助于获得“超级房东”的地位，这是吸引租房者选择特定房源的重要因素。她说，“我真的不认为人们意识到这 5 颗星对东道主以及优步和 Lyft 车手有多重要。”

对于优步和 Lyft 的司机来说， [的评分仅仅比完美的 5.0](https://www.buzzfeed.com/carolineodonovan/the-fault-in-five-stars?utm_term=.ydLZ0bDdE#.mvA13eWo4) 低十分之几就可能意味着缓刑甚至解雇。对于夹在中间的司机来说，乘客和拼车公司对星级的理解不一致可能是危险的。

零工经济评级不是对抗分数膨胀的地方。如果事情很好——所有的满意度都达到了，没有错，非常好——那就给自己一个完美的星级。如果您有反馈或小意见，您可以与您的主人或司机分享，或者在您的评论正文中留下合格信息。 [完美在实践](https://lifehacker.com/tag/perfection) 中是高不可攀的，但在点评和评分中应该是常见的。通常，一些人的生活依赖于此。
