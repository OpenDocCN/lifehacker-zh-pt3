# 这个网站帮助你找到你附近的日食事件

> 原文：<https://lifehacker.com/this-website-helps-you-find-eclipse-events-near-you-1797508236>

你可以在 8 月 21 日现场直播日食 ，但这比不上在现实生活中看到它。一个 [网站](http://🌚🌞.ws) (其网址上有月亮和太阳表情符号**🌚🌞**)通过告诉你附近的官方日食事件和日食条件，如天气，多久可以看到日食，以及日食每个阶段的时间，来帮助你计划日食日。要开始，只需在网站首页 [上插入你所在的城市](https://🌚🌞.ws) 。
