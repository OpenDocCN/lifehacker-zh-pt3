# 水不会变坏

> 原文：<https://lifehacker.com/water-doesnt-go-bad-1826120883>

好吧，看，这对我来说是个新闻，尽管我因为刚刚了解到这一点而被我的编辑们批评，但我打赌这对你们中的一些人来说也是个新闻:你可以把一杯水放在外面几个月，只要它被适当地清洗，它就不会让你生病。



我并不认为水会腐烂，好吗？我只是认为，有足够多的细菌漂浮在家里，或在自来水中，或当你抿一口时在你的嘴唇上，如果在玻璃杯中单独一个月，它可能会生长，然后让你生病。但是，正如食品安全专家本杰明·查普曼博士在一个有点尴尬的电话中告诉我的，不会的。

但肯定有办法的，对吧？查普曼博士说，是的，如果你没有正确清洗杯子，留下了像果汁或其他含糖残留物这样的营养物质。更有趣的是，他补充道:“重要的是，比如说，有人把便便粘在手指上，然后把它塞进去。”谢谢你，本。但是他说，即使你用玻璃杯喝水，把嘴放在上面，留下唇印，然后又把杯子放在外面——即使这样，你也不会让自己的口腔细菌毒害自己。

现在，即使是干净的水在几个小时后也会变坏。化学家苏珊·理查森T3 告诉连线 T5，这主要是因为人类在温水中比在冷水中能品尝到更多的东西。所以当水达到室温时，你可以品尝到更多的化学物质。自来水的化学成分在露天存放时会发生变化，像氯这样的化学物质会“释放”到空气中。加上一些实际的水蒸发，加强了水中其他物质的浓度。所以水变得“更浓”，它的味道与你习惯的不同，所以你不喜欢它。

显然，如果水源被污染，一切都完了。如果它离开水龙头时是有毒的，那么放在外面也是有毒的。但是很明显，只要一开始是好的，即使是超级恶心的陈年水也是健康的，我是一个无知的憎水者。好吧。但我并不孤单。我很好奇，因为烹饪博客 [The Kitchn](https://www.thekitchn.com/day-old-water-stale-safe-to-drink-257595) 问了同样的问题——或者也许他们是在为像我这样的乡巴佬陈述显而易见的事情。
