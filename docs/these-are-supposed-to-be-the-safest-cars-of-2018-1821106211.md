# 这些应该是 2018 年最安全的汽车

> 原文：<https://jalopnik.com/these-are-supposed-to-be-the-safest-cars-of-2018-1821106211>

许多 2018 年款汽车是安全的，但 62 辆汽车被评为特别擅长保护乘客。其中，15 种被认为是最安全的。

Watch

高速公路安全保险协会 [称自己](http://www.iihs.org/iihs/about-us) 是一家独立的非营利性研究机构，该机构根据碰撞生存能力和避免技术对汽车的安全性进行评级。

自 1992 年以来，该公司一直在弗吉尼亚州 T3 的一处设施中测试这种技术。今年，该组织公布了 62 辆 2018 年款汽车，根据视频中解释的标准，这些汽车比其他汽车更安全。

 [https://jalopnik.com/embed/inset/iframe?id=youtube-video-0fNAbIFG7WY&start=0](https://jalopnik.com/embed/inset/iframe?id=youtube-video-0fNAbIFG7WY&start=0) 

汽车通过 [进行评级，它们在“高速正面和侧面碰撞测试”中的表现](http://www.iihs.org/iihs/topics/t/crash-testing-and-crashworthiness/topicoverview) 包括乘客侧面碰撞和小重叠正面碰撞，以及对车顶强度、翻车弹性、座椅和头枕以及前灯帮助你看清的程度的评估。

你可以在 IIHS 网站 的 [上挖掘任何一辆特定的汽车在这些挑战赛中表现如何，或者你也可以看看这里的最佳表现者名单。](http://www.iihs.org/iihs/news/desktopnews/higher-standards-62-models-meet-tougher-criteria-to-earn-iihs-awards)
