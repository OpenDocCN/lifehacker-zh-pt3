# 哪些厨房工具和小玩意是你离不开的？

> 原文:[https://life hacker . com/which-kitchen-tools-and-gadgets-can-you-not-live-without-1823733813](https://lifehacker.com/which-kitchen-tools-and-gadgets-can-you-not-live-withou-1823733813)

大家好，欢迎回到 [*有什么好吃的？*](http://lifehacker.com/tag/whats-cooking) ，这是一个开放的话题，你可以在这里分享你对所有食物相关的想法、建议、食谱和观点。本周我想谈谈你最喜欢的厨房工具、器皿和小玩意。

Watch

在速溶锅、真空循环器、花式搅拌机和空气油炸锅之间，有许多超级炒作的小玩意可以玩。我明显偏爱前两个，但我也是个好木勺的吸盘。想听听有哪些厨具工具让你的生活更美好，哪些你觉得不值得炒作。

*   你最喜欢的锅是什么？我爱上了 [我的苹果形小屋荷兰烤箱](https://shop.lodgemfg.com/dutch-ovens-and-casseroles/3-5-quart-red-apple-pot.asp) 。除了做汤和沙司，它的形状和大小非常适合制作不揉面包。
*   你最喜欢的锅是什么？我爱我的铸铁，但我最喜欢的锅是我的 [全包不锈钢炒锅](https://skillet.lifehacker.com/stainless-steel-doesnt-get-enough-love-1818582513) 。它有很好的硬度，能迅速适应温度的变化，而且比铸铁更容易护理。
*   哪件厨房用品不见了会让你抓狂？我真的很依赖我的巨型 Pyrex 量杯。那个壶嘴真是太有用了。
*   哪种餐具你用得比你想象的多？我使用我的 [浸泡搅拌机](https://skillet.lifehacker.com/kitchen-tool-school-the-portable-powerful-immersion-b-1782522838) 的方式比我的食物处理器多。我用它来做奶油汤，乳化蛋黄酱，做奶昔和冰沙，以及搅打奶油。它还非常容易清洗，这对于没有洗碗机的家庭来说非常重要。
*   有哪些你不常使用的所谓“改变生活”的器具或用具？除非我在做大量的 T2 烘焙，否则我很少用我的凯膳怡立式搅拌机。我用木勺搅拌大部分面糊。
*   什么“改变生活”的电器真正改变了你的生活？坦白:我母亲比我早整整一年迷上了速溶咖啡，因为我很快就把它当成了一种“时尚”我并不真的使用许多预编程的功能，但有一个易于使用的，“设置它，然后忘记它”的高压锅确实使烹饪晚餐更快更美味。( [还我还在痴迷这些芝士蛋糕](https://skillet.lifehacker.com/make-cheesecake-for-two-in-record-time-with-your-instan-1821065475) )。)我也非常喜欢我的 Nespresso 咖啡机。我讨厌成为那种人，但我是。
*   如果你只能拥有一把锅铲或锅铲，你会选择哪一把？我爱我的木铲胜过我拥有的任何一把锅铲。
*   你们中有谁有真正别致的搅拌机吗？值得吗？老实说，我看不出你有食物加工机和搅拌器有什么意义。
*   对于那些有速溶咖啡的人来说:它是否如你所想的那样改善了你的生活？我是你的超级粉丝。
*   你觉得真空烹饪怎么样？我是说，T3】你知道我 T5】的感受。
*   **最被高估的热门厨房电器是什么？**我认为花哨的搅拌机没有必要，但没有什么比现在停产的 [Juicero](https://skillet.lifehacker.com/how-to-obtain-juice-without-a-juicero-1794714370) 更没必要了。 [我也讨厌这个盐罐](https://skillet.lifehacker.com/is-this-bluetooth-salt-shaker-the-juicero-of-seasoning-1795131493) 。
