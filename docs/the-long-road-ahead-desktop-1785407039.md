# 前方漫漫长路桌面

> 原文：<https://lifehacker.com/the-long-road-ahead-desktop-1785407039>

读者 [Alec](http://kinja.com/stranaha) 向我们的 [桌面展示](https://kinja.com/tag/desktop-showcase) 提交了这个简单、极简的 Mac 桌面，它简单、低调，看起来很棒。以下是你也可以设置它的方法。



这是一款看起来简单而优雅的台式机，预先没有太多东西，但经过巧妙的定制，变得更加精简和简单。如果你自己也想拥有这样的外表，你需要做以下事情:

*   [壁纸](http://all4desktop.com/4171336-way-to-heaven.html)
*   针对 OS X 的定制框架，我们已经在 前 [高亮显示](http://lifehacker.com/ubersicht-customizes-your-os-x-desktop-with-widgets-and-1591137635)
*   über sichtT3 的 [侧边栏小部件](http://tracesof.net/uebersicht-widgets/#Sidebar)
*   用于 Mac 的 [Liteicon](https://freemacsoft.net/liteicon/) 实用程序，可快速更改系统图标
*   [全白图标包](http://noshery.deviantart.com/art/All-White-Icons-for-OSX-470738808) 为 OS X
*   [cDock](http://w0lfschild.github.io/app_cDock.html) dock 定制工具
*   为 Mac 定制菜单栏的实用程序

大概就是这样。当然，如果您不喜欢右下角的小部件，您可以完全省略 bersicht，使用其他工具，如 cDock 和 Bartender，这两个工具本身就很有用。如果你喜欢这个造型，或者对如何让它成为你自己的有疑问，请访问 Alec 的 Kinja 博客(链接如下),提出你的问题或者赞美你。
