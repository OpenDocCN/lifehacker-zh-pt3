# 今天是电子邮件债务免除日

> 原文：<https://lifehacker.com/today-is-email-debt-forgiveness-day-1825652598>

当你浏览下午的邮件时，看看收件箱的其他部分。也许你有一些旧邮件在里面。2017 年的邮件。2016 年的邮件！你回复的时间越长，你就越害怕，现在你把它们放在收件箱里是为了什么，让自己内疚吗？毁了你的早晨？今天是你解决这件事的日子。



今天是第四个一年一度的 [电子邮件债务免除日](https://emaildebtforgiveness.me/) ，在这一天，每个人都被鼓励发送逾期已久的电子邮件，甚至不必提及已经过了多久。哪怕是一句简单的“抱歉耽搁了！”是可选的。相反，你可以直接插入上面的链接，它解释了这个概念。

[的主持人回复所有](https://www.gimletmedia.com/reply-all) ，一个关于互联网的播客，在 2015 年发明了 EDF 日，并从 开始每年都有 [庆祝，鼓励人们尽可能简单和无罪恶感地回复旧邮件，因为完美是完成的敌人。](https://www.usatoday.com/story/life/entertainthis/2017/04/30/email-debt-forgiveness-day-podcast/101007140/)

今天也是归档你永远不会回复的邮件的好日子。如果某件事已经在你的收件箱里呆了一年多，现在是时候让它离开了！存档！删除！放下你的负担！在电子邮件债务免除之前，有 [电子邮件破产](https://lifehacker.com/declare-inbox-bankruptcy-when-necessary-save-yourself-5512791) 。

这种方法最适用于其他人从未跟进的电子邮件，你不知道他们是否真的关心这一点，或者你真的讨厌和其他人交流。省去了他们形成自己的回应的麻烦，点击存档，让整个事件无声无息地漂进过去。

我们一直非常支持 [放弃旧目标](https://lifehacker.com/kill-your-old-ideas-so-you-can-be-more-creative-1796074082) 作为生产力健康的一部分。虽然有些人太容易放弃*了，但雄心勃勃、有成就的人往往会在另一个方向上犯错，对每一个好的意图都抓得太紧，以至于他们做得更少。重要的是要现实一点，清理掉不太重要的任务。*

但这也意味着，有时候，做半吊子的工作是可以的。再说一次，总比什么都没有好，今天你有了来自回复所有人和 Lifehacker 的“官方”许可。不要对满屏的回复感到压力，满屏的道歉和问候。只需回复两行— [以下是](https://lifehacker.com/literally-email-like-a-boss-1820931425#_ga=2.20903800.1970518754.1524514905-1723114163.1524514905)—仿佛才过了几分钟。看看这会把你带向何方。
