# 这些简单的煎锅让剩菜更有食欲

> 原文：<https://lifehacker.com/these-easy-skillet-arepas-are-the-perfect-vehicle-for-l-1819730510>

Arepas 是由玉米湿润粉糊(面团)制成的金黄色圆盘，通常会被填满或覆盖，是许多拉丁美洲人的“日常面包”(尤其是如果你来自我的祖国哥伦比亚或邻国委内瑞拉的话)。)虽然从技术上来说不是面包，但它们可以在一天中的任何时候食用，就像发酵的东西一样。它们和一碗热气腾腾的油炸豆子一起端上来，早餐涂上黄油，上面放一个煎鸡蛋或炒鸡蛋，或者塞满各种美食，如鸡肉沙拉或腌烤牛排条。选择是无穷无尽的。



当我还是个孩子的时候，我的父亲设计了一种 arepa“烤架”,他用两个螺钉将一个圆形金属片固定在一起，顶部是更薄的金属烤架片——这样 arepa 就可以整齐地放置——还有一个用于从炉子上取下它的长柄。这个装置位于我们电炉元件的顶部。难道你不知道吗，那该死的东西起作用了。

对我来说，这似乎是很大的努力，当我搬出去的时候，我妈妈还留着这个 arepa maker。如果你和我一样，缺乏这样的烤架(或者没有勇气做一个)，你可以烤 arepas，如果你觉得有趣，可以用帕尼尼压榨机，或者你可以走我最喜欢的捷径，用一个可靠的热铸铁煎锅来烹制这些金色的圆盘。只需制作 arepa 面团(见下文),并将其塑造成你想要的任何尺寸。(我喜欢它们有英国小松饼那么大。)加热一些油(比如牛油果油)，烤至古铜色漂亮。

随时准备这些小吃是非常容易的。你需要一杯在任何拉丁杂货店都能买到的 arepa harina(如 Harina Pan ),一小撮盐，一点点油和一杯温水来开始这个派对。混合湿润粉糊，静置几分钟，让所有的玉米粉重新水合。你的面团应该光滑柔软。掐掉一小块，把它卷成一个球，然后压平成大约四分之一英寸厚的圆盘。如果阿雷帕的边缘裂开了，你需要再加点水。一次加入大约一茶匙，直到你的面团变得柔软、有水分。

向热煎锅中加入一点鳄梨或植物油，然后将光盘放在平底锅上，确保它们不拥挤。在这些婴儿开始变色并长出“硬皮”之前，不要乱动或摆弄他们。然后翻面，另一面煮。就像牛排一样，当他们松开锅表面的“抓手”时，你就会知道每一面都熟了。如果你不等的话，你会把那块美味的面包皮留在锅里，而不是留在你的肚子里！

现在有趣的部分来了。无论你是在做一个解酒助手，还是去参加一个深夜派对后聚会，或者只是在你的下一次聚会上提供小吃，浇上或填满这些美食是一个“任何事情都可以”的提议。有剩余的鳄梨酱吗？一些商店买的烤鸡怎么样？将鸡肉切碎，或者加入鳄梨色拉酱和一点莱姆汁，或者用鳄梨色拉酱作为基料，上面放上鸡肉。用一些芫荽叶装饰，保证你会吃得很开心。吃剩的黑豆汤？烤牛排？几乎任何你内心渴望的东西都很管用。是的，甚至是火腿和奶酪。

如果你觉得有创意，你可以在烹饪前拿一些你最喜欢的融化奶酪夹在你的未煮过的湿润粉糊中间。这就像拉丁美洲的“烤奶酪”你不知道你需要在你的生活中。
