# 10 项有用的 Windows 10 功能，您可能从未使用过，但应该使用

> 原文：<https://gizmodo.com/10-useful-windows-10-features-you-probably-never-use-bu-1820980875>

Windows 已经以这样或那样的形式存在了整整 32 年，在这段时间里，它积累了许多功能——不仅仅是最新的功能，还有那些你可能已经忘记或者根本没有发现的长期存在的功能。这里有 10 个真正有用的技巧，Windows 10 能够做到，你可能不知道，但绝对应该知道。



* * *

### **1)虚拟桌面**

Windows 10 终于迎来了操作系统的虚拟桌面，给你单独的空间来安排你的程序窗口和工作。如果你点击**任务视图**按钮(就在任务栏搜索框的右边)，你可以调出所有打开的窗口，以及下面的虚拟桌面——点击**新桌面**创建一个新空间。

任务栏会发生变化，以显示您在该桌面上打开的任何应用程序，尽管固定的应用程序快捷方式(在任务栏上)在所有桌面上保持不变。如果您需要一种方法来管理网页浏览、电子表格制作和图像编辑的不同区域，这是理想的选择。在任务视图中，单击任何虚拟桌面缩略图旁边的十字图标将其关闭。

* * *

### **2)视频和媒体流**

你很可能会使用类似 Plex 的 [来将视频和音乐从 Windows 下载到另一个设备，比如你的大屏幕电视，但微软的操作系统自带内置的视频和媒体流协议，其形式为 DLNA(在](http://fieldguide.gizmodo.com/6-reasons-to-use-plex-to-create-your-own-personal-netfl-1705440916) [数字生活网络联盟](https://en.wikipedia.org/wiki/Digital_Living_Network_Alliance) 之后)。你所需要的只是一个兼容 DLNA 的设备，比如 Xbox One 的或 Roku 的。

要启用它，从任务栏中搜索“媒体流”，单击出现的选项，然后选择**打开媒体流**。我们不会在这里做一个完整的 DLNA 指南( [见这里](https://support.roku.com/article/208754908-how-to-use-roku-media-player-to-play-your-videos-music-and-photos) 获取 Roku 指令)，但假设你的接收设备已经正确配置，流媒体内容就像右击一个文件并选择 **Cast to Device** 一样简单。

* * *

### **3)调度任务**

任务调度程序是一个经验丰富的 Windows 实用程序，自 Windows 95 时代就开始为用户服务，远在 IFTTTT3[出现之前。顾名思义，它可以让你在 Windows 中安排任务，无论你是想在晚上的特定时间关闭机器，还是让系统维护应用程序定期运行。](http://fieldguide.gizmodo.com/15-new-ifttt-triggers-to-make-life-easier-1788943325)

在搜索框中键入“任务调度程序”来找到实用程序并打开界面:起初它可能看起来有点令人生畏，但如果你点击周围并试验所提供的内容，你很快就会找到窍门。点击**创建基本任务**或**创建任务**开始，然后你需要为每个任务提供一个触发器(一个时间)和一个结果动作。

* * *

### **4)恶意软件移除**

微软在 Windows 中加入了安全软件，但就目前而言，它是这样的:如果你不想为防病毒保护付费，那么 Windows Defender 将在让令人讨厌的恶意软件远离你的系统方面做得足够好，尽管你肯定可以通过选择第三方替代品来获得更全面的安全保护。

另一个你可能不知道的内置 Windows 安全工具是 [恶意软件移除工具](https://support.microsoft.com/en-us/help/890830/remove-specific-prevalent-malware-with-windows-malicious-software-remo) ，它在后台静默运行，以确保没有任何东西在你的系统上扎根。如果你已经将 Windows 10 配置为自动更新(现在很难不这样做)，那么你也将自动拥有最新版本的 MSRT。

* * *

### **5)回收丢失的磁盘空间**

磁盘清理是另一个老派的 Windows 实用程序，仍然很强大，你可以通过在任务栏搜索框中键入它的名称并选择第一个结果来启动它。它解决了最长期的问题，窗口膨胀，通过检测和删除所有那些多余的文件占用你的系统空间，你真的不需要保留。

单击列表中的任何条目，查看文件的描述，以及通过删除这些文件可以在硬盘上回收多少空间(回收站中的文件将是列出的选项之一)。在您知道不需要的文件类型旁边的框中打勾，然后选择**清理系统文件**开始清除。

* * *

### **6)远程协助**

如果一个朋友或家庭成员正在努力解决计算机问题，那么你可能会尝试使用 Skype 或 TeamViewer 之类的软件，但 Windows 实际上允许你在没有任何额外软件的情况下运行托管远程协助会话。在任务栏的搜索框中键入“远程协助”，然后选择顶部的选项开始。

开始的闪屏让您选择是想要获得帮助还是给予帮助，然后您将被引导完成发送或接受邀请的过程。您可以完全控制对方的电脑，包括鼠标和键盘。如果你们两个都是 Windows 10，那就找 Quick Assist 实用程序吧。

* * *

### **7)在家里共享文件**

如果您的家人需要一起处理文档、共享音乐或每个人都需要访问的家庭电影，那么这些文件可以非常简单地与每个人共享，不需要使用共享链接或电子邮件附件或其他任何东西(他们需要 Windows)。作为额外的奖励，您还可以在整个家庭中共享打印机。

要开始使用，从设置中进入**网络&互联网**，点击**家庭组**——如果该服务尚未设置，点击**创建家庭组**，并按照说明进行操作。在显示其他人需要加入家庭组的密码之前，Windows 将询问您要在本地网络中共享哪些类型的文件和设备。

* * *

### **8)检查丢失的文件和磁盘错误**

到命令提示符！几个长期存在的文本命令在诊断和修复计算机问题时会很有用:首先，通过在任务栏搜索框中键入“cmd”来启动命令提示符，右键单击**命令提示符**链接，并从出现的弹出菜单中选择**以管理员身份运行**。

输入“sfc /scannow”运行系统文件检查器，它会在硬盘上搜索操作系统正常工作所依赖的任何丢失的 Windows 文件，并替换任何需要替换的文件( [参见此处](https://answers.microsoft.com/en-us/windows/forum/windows_10-update/system-file-check-sfc-scan-and-repair-system-files/bc609315-da1f-4775-812c-695b60477a93) 了解更多信息)。您也可以尝试“chkdsk /f”来检查本地硬盘驱动器的问题，如果可能的话，在此过程中修复这些错误(更多信息 [可在此处获得](https://technet.microsoft.com/en-us/library/ee872425.aspx?f=255&MSPPError=-2147217396) )。

### **9)打印成 PDF 格式**

不要让你的朋友和家人看格式糟糕的专有文件类型，这些文件类型在每个系统上看起来都不一样——而是给他们发送一个漂亮干净的 PDF 文件。这将在所有设备上保持其外观。谢天谢地，Windows 10 增加了对 PDF 导出的原生支持，所以你不需要安装任何其他第三方工具来完成这项工作。

在您尝试输出文件的任何程序中,“打印到 PDF”选项都会出现在您的打印机列表中，因此，无论该程序在哪里，都要找到它的打印选项，并从打印机列表中选择 Microsoft 打印到 PDF 条目。在导出选项方面，你实际上没有太多的选择，但是你可以通过**打印机属性**选择输出文档的大小。

* * *

### **10)记录屏幕活动**

我们之前已经指出了这一点，但它值得重复:如果你曾经有过任何类型的屏幕录制，那么 Windows 10 自带了一个内置的应用程序来完成这项工作。名为 Game DVR——它旨在录制游戏中的镜头以供在线分享，但你可以轻松地将其用于录制教程或提示或 Windows 10 上大多数应用程序中的其他内容。

首先打开你想要录制的应用程序，然后如果你点击 **Win+G** 键盘快捷键，控制台就会打开——你可能需要勾选一个框来确认你正在运行一个游戏，即使你没有。您可以使用底部的复选框来打开或关闭音频录制，然后单击红色的大录制按钮开始录制镜头。
