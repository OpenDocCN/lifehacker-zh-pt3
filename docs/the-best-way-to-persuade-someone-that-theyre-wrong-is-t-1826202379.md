# 说服某人他们错了的最好方法是向他们展示他们是如何正确的

> 原文：<https://lifehacker.com/the-best-way-to-persuade-someone-that-theyre-wrong-is-t-1826202379>

当你和别人讨论一个话题时，避免直截了当地告诉别人他们错了，这对你最有利。它所做的只是让其他人产生防御心理，让他们进一步坚定自己的信念。相反，告诉他们所有他们是对的，然后引导他们自己认识到他们是错的。

这种说服性辩论的方法可以追溯到 17 世纪的哲学家。他发现改变某人想法的最好方法是从他们的角度看问题，然后让他们改变自己的想法。帕斯卡尔在《第九节[](http://www.ntslibrary.com/PDF%20Books/Blaise%20Pascal%20Pensees.pdf)*中写道:*

> *“当我们希望以优势纠正，并向另一个人展示他的错误，我们必须注意到他从哪一方面看待这件事，因为在这方面通常是真实的，并向他承认真理，但向他揭示它是错误的。他对此很满意，因为他看到他没有错，他只是没有看到所有方面...人们通常更容易被自己发现的原因说服，而不是被别人想到的原因说服。”*

*简而言之，如果你想让某人意识到他们错了，首先向他们展示他们是如何正确的——然后向他们展示他们可能没有意识到的事情。给他们提供信息，引导他们走向自我启蒙，这样你就可以避免激烈的争论。正如帕斯卡解释的那样，没有人会因为看不到事物的方方面面而生气。我们只是人类，对吗？但是当你告诉人们他们错了的时候，他们会被冒犯，因为这感觉像是对他们性格和智力的人身攻击。一旦你这么做了，你合作的机会就没了。*

*亚瑟·马克曼是德克萨斯大学奥斯汀分校的心理学教授，他告诉 QuartzT3 的奥利维亚·戈德希尔，这种哲学上的修辞在现实世界中非常有效。正如马克曼所言，你给了某人放松警惕的机会，并允许他们改变主意，而不用担心这会让他们难堪。合作回到了桌面上，由于某种不言而喻的社会交易，智力讨论可以蓬勃发展，比如“我同意你对*这个*是对的，并给你一些我的信息，然后你可以自由地承认你对*那个*是错的，而不会受到任何不必要的惩罚。”有时候改变某人的想法就是给他们机会。*

*一位哲学家用了 350 年的老把戏让人们改变主意，现在得到了心理学家 |石英*
