# 你永远不应该问的闲聊问题

> 原文:[https://life hacker . com/the-small-talk-questions-you-should-never-ask-1795578480](https://lifehacker.com/the-small-talk-questions-you-should-never-ask-1795578480)

结婚最大的好处就是爱管闲事的熟人不再问你:“那你们俩什么时候结婚？”最糟糕的是，他们开始问你，“那么你们两个什么时候开始要孩子？”

现代社会不知何故让某种粗鲁的问题从缝隙中溜走了:友好地询问某人达到传统生活里程碑的进展。这些潜在的令人担忧的问题经常被同事、远亲和熟人当作轻松的话题。我们已经探索了 [如何回答这些问题](https://lifehacker.com/how-to-answer-the-awkward-private-questions-your-famil-1747202575#_ga=2.196426922.769583846.1496180771-1891331057.1496166663) ，无论是礼貌地转移话题还是解决潜在的情感问题。现在，让我们深入了解一下什么是首先不要问的。

不恰当的提问几乎是不可避免的。闲聊是为了认识人，不可能预知每个人的个人界限。有意参与更有意义的对话会令人满意。重要的是要意识到看似老生常谈的问题如何触及痛处。

纽约州博蒙特礼仪 学校的创始人麦卡·梅尔说:“良好的礼仪就是让别人感到舒适和受尊重。”。因此，在你问问题之前，想想对方的感受。任何可能让某人感到压力、不舒服、害羞或受到评判的问题都是要避开的。”

## 关系和家庭

“你们两个最终什么时候结婚？”这可能看起来是一个无伤大雅的玩笑问题。但是你期待什么样的回答呢？你不是在要求提醒。当有人订婚时，他们肯定会告诉你。感觉更像是你在询问他们对婚姻制度的看法，或者暗示他们的关系看起来已经过时了。两者都不是合适的话题引子。

如果要问关系，就从比较单纯的开始吧。如果你以一句含糊的“你的另一半好吗？”你可以让对方决定如何亲密地回答。或者敞开你自己的生活，但不要要求他们上升到满足你的亲密程度。

“你们两个什么时候要孩子？”更惨。不生孩子的原因是无数的，个人的，有时是不快乐的。诚实的回答可能是“上周流产后事情变得很艰难”或“事实证明收养机构有点种族歧视”，但除非你是一个亲密的朋友，否则你问的人将不得不编造一个更无辜的答案，在你不知情的情况下与你保持距离。“相反，”梅尔建议道，“你可以问别人‘跟我说说你的家庭’，即使他们没有孩子，他们也可以告诉你他们来自哪里，或者关于他们的亲戚。”

当和有孩子的人交谈时，避免判断性的电话，比如“你什么时候会有更多孩子？”或**“你想要一个女孩吗？”**只需询问那些确实存在的孩子:他们多大了，长什么样？不要问孩子们达到了什么样的里程碑；任何父母最不需要的就是另一个人来测试他们的孩子。

## 怀孕

很明显为什么会问“你怀孕了吗？”不飞。你真的不能太谨慎。我和一个八个月大的朋友聊天，仍然等着她先提出来。还是比假阳性好。

同理，如果你知道某人怀孕了，**“你怀的是双胞胎吗？”**越界。如果是，他们肯定会提的。当你听到预产期的时候，相信它。非常努力地避免暗示孕妇肥胖的哪怕一点点暗示——或者纠结于她们有多瘦。避免暗示你在评估他们的身体。

如果他们告诉你婴儿的性别，做出积极的反应，但不要将其视为“胜利”(除非父母已经公开这样做)。我一个同事怀了男孩，几个人问**“你老公幸福吗？”甚至有人祝贺她“做了她的工作”，就好像她在为亨利八世生一个男性继承人。**

不要开始谈论抚养孩子的计划。**“你要母乳喂养吗？”**或**“你是顺产吗？”**都是有分量的决定，你必须给出的任何建议肯定会被十几本育儿书更好地解释，并被十几本其他书反驳。唯一合适的建议完全是可有可无的，不受价值判断的影响，专注于让准父母感觉更好、更自由。人们有各种各样的方式生儿育女，其中大多数都很好，你不是那个能解开婴儿猝死综合症秘密的大胆思考者。

## 工作

“你喜欢你的新工作吗？”那不是很粗鲁，但也可能很尴尬。如果他们不喜欢自己的工作，他们可能不会告诉所有人，所以你会迫使他们撒谎。只是用一种不需要价值判断的方式来重新表述。“你的工作是什么样的？”或者“你在做什么？”如果没有对职业成就感进行深刻的自我反省，这个问题会更容易回答。

检查一个长期项目是另一个灰色区域。“你的书/播客/副业进展如何？”感觉你在请求进度报告。相反，询问你学到的任何细节，或者跟进他们在社交媒体上发布的关于该项目的任何信息。这样，你就接近了他们已经选择向世界展示的部分，这可能比他们已经完成的工作的百分比更有趣。

当第一次与某人见面时，即使是最普通的开场白也可能有风险。迈耶建议不要用那句老台词开场:“那你是做什么的？”听起来像是投机取巧。如果你的目标是更好地了解某人，那么这个问题只有在他们碰巧喜欢自己的工作并愿意在业余时间谈论这份工作时才有用。(更糟糕的是我在一次技术会议上得到的加载版本:“你制造什么？”)如果你开始感到寒冷，“你做什么娱乐？”要有用得多。根据定义，答案是对方想要谈论的事情。

如果你知道有人在找工作，善意的跟进听起来会像是唠叨。正如 [缪斯的粗鲁求职问题清单](https://www.themuse.com/advice/the-polite-persons-guide-to-answering-all-the-rude-questions-people-ask-when-youre-unemployed) 所示，这是一个雷区。努力找工作的人会觉得整个世界都在说他们没有价值，这不会给你带来任何好处。避免提出类似**“你想过回学校吗？”**你不会成为说服他们改变路线或和解的人。哪怕是像**这样简单的问题“你申请了哪里？”缪斯说:**建立一种期待，你会在晚些时候查看进度报告。如果你真的想帮忙，问问他们想要什么。如果你提出任何线索，要小心谨慎；给他们留下足够的空间来礼貌地拒绝你。

## 运用你的判断力

上面的大部分问题在某些情况下是合适的:在伴侣、密友或直系亲属之间。这是他们不适合闲聊的一个原因:他们暗示亲密，要求脆弱。通过询问某人的生活进度报告，你也在要求他们评估你和他们的关系。

随着时间的推移，我们自然希望增加一些关系的亲密度，所以我们无法避免测试边界。但是，通过更密切地关注你的问题的潜在含义，你可以将它们与关系的背景相匹配。正如迈耶建议的那样，“在你问一个问题之前，先衡量一下它的敏感程度。”如果这个话题比你和你的熟人通常谈论的话题敏感得多，那就留到下次吧。
