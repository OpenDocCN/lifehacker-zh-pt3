# 微软的按需文件非常适合超极本所有者、存储吝啬者

> 原文：<https://lifehacker.com/microsoft-s-on-demand-files-are-perfect-for-ultrabook-o-1796092583>

微软的文件同步功能进一步模糊了本地存储和云存储之间的区别，让你可以像对待离线文件一样对待在线文件。

Watch

按需文件功能让您可以像访问本地文件一样访问云存储，而不必无限期地存储它们。该功能非常适合使用少量存储空间的超极本的 Windows 用户。

对微软来说，按需访问云文件是新鲜事，但这种存储友好的特性并非闻所未闻。像 Google Drive 这样的标准云存储服务会迫使你将想要访问的云文件存储在本地存储设备上。去年，Dropbox 推出了 [智能同步](https://www.dropbox.com/business/smartsync?_tk=blog_&oqa=mindpf08bbbody) ，这是一项专注于业务的按需文件存储功能，可以像对待本地文件一样对待在线文件。

> 文件点播还允许您使用 Windows 文件选择器从桌面或 Windows 应用商店应用程序中打开在线文件。只需在文件选择器中选择想要打开的文件，文件就会自动下载并在你的应用中打开。此外，由于它可以与您的个人和工作 OneDrive 以及您的 SharePoint Online 团队网站配合使用，因此您的家庭和职业生活都会受到保护。

该功能将在 [Windows 10 秋季创作者更新](https://www.microsoft.com/en-us/windows/upcoming-features) 中向所有 Windows 10 用户推出，尽管你可以稍微早一点尝试。运行 build 16215 的 Windows Insiders 成员可以使用它，还有不耐烦的 Windows 10 用户可以 [手动下载更新](https://go.microsoft.com/fwlink/?linkid=851311) 。
