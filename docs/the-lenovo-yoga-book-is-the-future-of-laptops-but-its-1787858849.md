# 联想 Yoga Book 是笔记本电脑的未来，但它缺少一个操作系统

> 原文：<https://gizmodo.com/the-lenovo-yoga-book-is-the-future-of-laptops-but-its-1787858849>

联想的新 Yoga Book 吸引了当今其他平板电脑或笔记本电脑无法比拟的眼球。人们并不总是确定我在看什么漫画或者记什么笔记，但是他们知道这很有趣，很小，很酷。凭借其电容式触摸键盘，按下按钮即可变成绘图板，Yoga Book 绝对是笔记本电脑和平板电脑以及二者之间模糊空间的未来。但联想有史以来最薄的笔记本电脑也有一个主要问题，即它只有 Windows 和 Android 版本。这两款操作系统都没有为联想的未来做好准备。



为了这篇评论，我花了一周时间使用联想的 Android 版本的光滑的新后空翻笔记本电脑。从本质上来说，它和 Windows 10 的变种是一样的。它们都配备了英特尔凌动 x5-Z8550 处理器、4GB 内存、64GB 的设备存储和 1920 x 1200 分辨率的显示器。安卓版起价 500 美元，Windows 版起价 550 美元。

因为 Windows 10 不能很好地与英特尔凌动(Intel Atom)等低电压处理器配合使用，所以该版本的电池寿命将比我测试的版本更差，而且它也没有安卓版本那么“爽快”。这意味着它将拥有更好的应用范围，比如完整版的 Photoshop 和 Microsoft Office。与此同时，Android 对于 Atom 来说是完美的。在三天的时间里，我经常使用将近 12 个小时。

但是 Android 对于笔记本电脑来说并不完美。联想试图将 [比尔](http://shop.lenovo.com/us/en/tablets/lenovo/yoga-book/yoga-book-windows/)Yoga Book 作为 [一款平板电脑](http://shop.lenovo.com/us/en/tablets/lenovo/yoga-book/yoga-book-android/) ，但我很难称之为仅仅因为它使用了 Atom 处理器和 Android 变种。在设计方面，这款设备可爱的 360 度表带式铰链类似于联想的 Yoga 系列笔记本电脑。因此，就像它更大的笔记本电脑 cousins 一样，Yoga Book 可以像笔记本电脑一样使用，或者旋转成帐篷模式观看电影，或者旋转回到平板电脑配置。

在常规使用中，我发现自己大部分时间都是把瑜伽书当平板使用。联想知道我更喜欢平板模式吗？这就是它称之为平板电脑的原因吗？这是不可能的，因为联想指望人们喜欢这本新书和旧 Yoga 系列的主要区别:“光环”键盘。该设备的下半部分是电容式触摸设备。按下一个按钮，键盘就亮了。再次按下它，键盘消失了，它变成了一个 Wacom 图形平板电脑的近似物——这是一个概念上很酷的功能，但当我在 Android 上使用它时，实际上就没那么酷了。尽管它与 225 美元的 Intuos Pro 等高端 Wacom 平板电脑一样具有 2048 级的灵敏度，但 Android 缺乏 Windows 设备的所有优秀应用程序，因此你无法真正充分利用这一功能。

也就是说，向朋友炫耀它仍然非常整洁，当你打开键盘时，它就像《星际迷航》中的东西一样亮了起来，这是一种明确的发自内心的*我正在触摸未来的感觉。*不幸的是，光环键盘的布局很糟糕。甚至像加入 Slack(或者你正在使用的任何办公室间信息系统)这样简单的事情都让我沮丧不已。

在很大程度上，键盘令人沮丧，因为尽管联想考虑周到的工程，它仍然感觉不对劲。Halo 键盘有适度的触觉反馈，但不足以模仿在真正的咔哒咔哒键上打字。反馈也有微小的延迟，这会导致额外的误差。我发现自己疯狂地一遍又一遍地敲着同一封信，等待着指尖快速的嗡嗡声。

从布局到反馈的一切都可以通过刷新来修复——这是非常需要的。键盘的另一个主要问题是，需要进行工程改造。键盘和触控板在 Windows 版本上工作得相当好，但在 Android 上，很明显该操作系统还不能用于笔记本电脑。在类似笔记本电脑的设置上使用安卓系统就像去年一样令人恼火，因为去年的 [大多是出色的像素 C](http://gizmodo.com/google-pixel-c-review-androids-not-ready-for-a-tablet-1747368432) 。

Android 现在允许你在显示器上同时打开和运行多个应用程序，但它将应用程序缩小到智能手机版本，这意味着你经常不得不俯身向屏幕，眯着眼睛阅读内容。Android 也不擅长在很多应用程序上切换横向和纵向模式。我经常发现应用程序被拉长并且有问题，直到我把它们返回到纵向模式。Android 上的键盘快捷键也很少，所以没有 Windows、Mac 甚至 iOS 的快速效率。

因此，瑜伽书给人的感觉不像是一台很酷的微型笔记本电脑(或生产力平板电脑)，而更像是一个时髦的分散注意力的小工具。Yoga Book 真正需要的是一个在笔记本电脑和平板电脑场景中运行良好的操作系统(如 Windows 10)，并且足够轻便，在运行 Atom 处理器时不会很慢(如 Android)。Chrome 操作系统拥有运行 Android 应用程序的新功能，可能是一个不错的选择，但它不是触摸友好的操作系统。

真正的解决方案还不存在，但据传谷歌正在努力。代号为“仙女座”的新操作系统据称将是 Android 和 Chrome OS 的独特和更好的融合。它将拥有 Chrome 的所有笔记本电脑功能和 Android 的所有应用程序以及基于触摸的功能，而且它不会成为一个资源猪。不幸的是，它最早要到 2017 年才能上市，而且没有任何关于它将在何种设备上运行的文档。

但它会是瑜伽书的完美素材。这种超薄设备感觉像是移动计算的未来，它离好是如此之近，以至于坏的时候让人抓狂。通过对键盘和操作系统进行一些调整，Yoga Book 可能会成为你甚至不知道自己想要的笔记本电脑和平板电脑的混合体。但现在这感觉更像是一个很酷的派对把戏。如果你想用一款新设备让你的朋友们惊叹不已，那就放下你的 500 美元，但不要指望能完成任何工作。
