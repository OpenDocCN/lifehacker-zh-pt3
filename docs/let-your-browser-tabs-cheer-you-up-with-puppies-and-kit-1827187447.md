# 让你的浏览器标签为你带来小狗小猫的快乐

> 原文：<https://lifehacker.com/let-your-browser-tabs-cheer-you-up-with-puppies-and-kit-1827187447>

Chrome，Firefox: 对我们许多人来说，这是艰难的一周，很可能你现在有点压力过大。



那么，一个人该怎么做呢？这里有一个简单的方法:让你的浏览器用一些可爱的动物让你开心起来。可爱的动物不关心政治，只关心游戏时间；对阻挠议事者的友好；午睡新闻( [假](https://lifehacker.com/figure-out-which-facebook-friends-share-fake-news-1825016350#_ga=2.102062275.315288407.1529932351-1723114163.1524514905) 或其他)。每次你打开一个新标签，你都会看到一些让你惊喜的东西——一个可爱的动物来填补日益增长的绝望和怀疑的空虚。

**如果你在 Chrome 上，**我强烈建议你去看看开发者 mystart.com 的“ [【我的猫】](https://chrome.google.com/webstore/detail/my-cats-adorable-cat-kitt/gamgcdlfhmmigjmbffodgkpglbnejkjm) 扩展。虽然表面上看起来很简单——一个新的随机的猫图像填充了你打开的新标签——但是这个扩展也有一些小技巧，使得它不仅对*和*有用。

不仅每个新标签都有一个内置的网络搜索(你可以点击右上角的汉堡图标，将其设置为谷歌而不是默认的雅虎)，而且你可以轻松地调出你最喜欢的 Chrome 应用程序、书签和你最常访问的网页列表。你可以将待办事项列表放到你的猫标签上，查看时间和六天天气预报，快速召唤一些“放松”音乐，访问你最喜欢的网站的快捷方式，并基本上创建你自己定制的标签页——中间有一只大的老猫。

还有 [一个版本的延伸，对于狗迷](https://chrome.google.com/webstore/detail/my-dogs-hd-wallpaper-cute/bhmpnibiagopmobamhlgaghkojlcjnfn?hl=en-US) 也是如此。机会均等。

如果你在火狐上，你会得到更简单的待遇。如果你*只是*关心猫，而且只是猫，我强烈推荐看看可爱的 [虎斑猫](https://addons.mozilla.org/en-US/firefox/addon/tabby-cat-friend/?src=search) 扩展版。每次你加载一个新的标签，你都会看到一个新的卡通形象，有不同的姿势，不同的颜色，坐在不同的玩具旁边(或与不同的玩具互动)。

你打开的标签越多，你得到的玩具就越多，有些玩具只吸引特定的猫。..你现在知道该怎么做了。开心地花更多的时间盯着你的空空如也的标签，而不是你经常访问的网站。

抱歉，我找不到狗的杀手扩展，所以你可能只能用稍微更通用的 [小动物画廊新标签扩展](https://addons.mozilla.org/en-US/firefox/addon/baby-animals-gallery-new-tab/) 。你可能从它的名字就能猜出它是做什么的，但是如果没有，这里有一个简单的解释:打开一个新的标签，在每个标签上得到一个新的小动物的图像。高兴地尖叫。重复一遍。

就像我提到的 Chrome 扩展一样，小动物画廊的新标签也有一个小搜索栏、天气更新和访问其他网站的快捷菜单。但你不是为此而来的。你可以在这里看到可爱的动物，这个扩展声称提供了一个持续旋转的“高分辨率的小动物图像”你还需要什么？
