# 山顶桌面

> 原文：<https://lifehacker.com/the-mountaintop-desktop-1788086032>

经常使用桌面的作者 JonRedcorn2 通常使用 Windows，但这次他有一台 Mac——这并没有阻止他按照自己喜欢的方式定制。这是他如何设置这个漂亮干净的桌面。

Watch

JonRedcorn2 使用的是 bersicht， [一个我们在](https://lifehacker.com/ubersicht-customizes-your-os-x-desktop-with-widgets-and-1591137635) 之前已经强调过的 macOS 定制套件。您将需要抓住它开始，但它几乎是所有您需要的。

以下是所有的片段:

*   来自 DeviantArt 的 [壁纸](http://slurpaza.deviantart.com/art/Blanc-581166976)
*   macOS 的[über sicht](http://tracesof.net/uebersicht/)定制套件
*   正在播放显示的 [播放框控件](http://tracesof.net/uebersicht-widgets/#Playbox)
*   不透明度稍微降低的简单时钟部件
*   [天气现在 widget](http://tracesof.net/uebersicht-widgets/#weathernow)
*   用 [圆点](http://dradis75.deviantart.com/art/Dots-462010667) 图标包来替换系统图标

他还建议，如果你运行的是 Mavericks 或更低版本，可以使用 [黑曜石菜单栏](http://www.obsidianmenubar.com/) 和 [黑色 Dock](http://www.macupdate.com/app/mac/51149/cdock) 工具(它们可以在 El Capitan 运行，这是他正在运行的，但它们可能不会安装，这是一个令人失望的东西)。无论如何，如果你喜欢这个外观，去他的 Flickr 页面(链接如下)让他知道你喜欢这个设计！
