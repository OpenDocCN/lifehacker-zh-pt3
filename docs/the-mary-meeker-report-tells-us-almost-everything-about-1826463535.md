# 玛丽·米克尔报告告诉了我们关于科技世界的几乎一切

> 原文：<https://lifehacker.com/the-mary-meeker-report-tells-us-almost-everything-about-1826463535>

如果有人让你描述整个全球技术格局，你可能只会摇摇头，宣布这项任务不可能完成。但凯鹏华盈(Kleiner Perkins Caufield & Byers)的玛丽·米克尔每年都会这么做。她的报告已经发布，详细介绍了从全球智能手机出货量，到我们上网的时间，再到中国作为全球主要科技中心的发展。



米克报告中的大部分数据可以通过 LinkedIn 的 SlideShare 服务上发布的294 幻灯片获得。

虽然去年智能手机出货量首次没有增长，但我们花在网上的时间却增加了。无论是从原始数据还是从占全球人口的比例来看，互联网普及率都在持续增长。不出所料，WiFi 网络正以指数速度扩张。

去年，数字货币的使用范围扩大了，超过 60%的日常交易都是通过数字方式进行的。

随着我们逐渐脱离电视和电信网络等传统服务交付模式，消息服务、Twitch 流媒体时长和移动视频使用都在增长。

幻灯片中有很多内容需要消化。虽然数据很重要，但要理解这些数据的含义还需要一些时间。我想知道今天这些数据对你有什么真正的价值。2018 年已经过去了将近一半，米克的报告虽然有趣，却是六个月前的世界快照。
