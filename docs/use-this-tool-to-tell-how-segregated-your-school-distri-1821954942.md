# 使用这个工具来判断你的学区实际上有多隔离

> 原文：<https://lifehacker.com/use-this-tool-to-tell-how-segregated-your-school-distri-1821954942>

人们可能会认为几十年的社会进步会消除教育中的种族隔离，但经济政策研究所 2013 年的一项研究发现，现在美国学校的种族隔离比 1973 年时更加严重 。这在很大程度上是由于几十年的歧视性住房政策，这使得白人和有色人种住在不同的街区，但这也是由于学区的界限。一般来说，孩子们，尤其是小学生，会去他们的邻居学校，这意味着白人孩子和他们的白人邻居一起上学，黑人孩子和他们的黑人邻居一起上学。



但是根据阿尔文·张在 Vox 发表的一篇详尽而有趣的新报道，学区*没有*来加强潜在的住房隔离。它们可以用来改善种族隔离——也可以用来加剧种族隔离。正如 Chang 所写的，“上课区经常被不公平地划分，把白人学生安排在比他们所居住的社区还要白的教室里。”这在教育上相当于不公正的选区划分。

现在，这似乎是一个无法克服的问题——例如，解决几十年来种族主义住房政策的持续影响是一个持续的挑战。但常指出，这类政策是由学校董事会在地方一级制定的，其连锁效应会持续几代人。这意味着，如果你致力于减少你所在地区的种族隔离，参加当地的学校董事会会议。你自己去竞选吧。(为了说明这一点，常指出“”的研究表明，拥有注册民主党成员的学校董事会倾向于划定界限，以减少种族隔离。”)

与此同时，这个故事提供了一个工具，可以看出你所在的学区是否反映了住房歧视，寻求解决它，或者使它变得更糟。但是没有简单的解决办法:在重划了边界的地区，父母更有可能搬家或者把他们的孩子送到私立学校。在我家附近的地区，富裕学校(主要是白人学校)的入学人数过多，最近几年，一些孩子被分配到附近一所不太富裕也不太白人的学校。许多许多家长的反应是把他们的孩子转到私立学校，直到他们能在他们喜欢的地区获得一席之地。

但最终重新划定学区界限可能是唯一的解决方案，因为学校取消种族隔离会导致住房取消种族隔离，反之亦然:正如记者妮可·汉娜·琼斯在推特上写道，“住房隔离会减少...因为人们不再能通过购买白人社区的房子来保证进入白人学校。”
