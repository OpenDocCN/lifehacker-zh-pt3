# 比起自制的食物，你更喜欢哪种商店买的食物？

> 原文：<https://lifehacker.com/which-store-bought-foods-do-you-prefer-over-homemade-1828004307>

大家好，欢迎回到 [*有什么好吃的？*](http://lifehacker.com/tag/whats-cooking) ，这是一个开放的话题，你可以在这里分享你对所有食物相关的想法、建议、食谱和观点。大多数食物自制时味道更好，但有几样我更喜欢购买，无论是因为我的能力不足、懒惰，还是因为一个人根本无法像菲多利罐头公司的科学家那样制作油炸玉米饼。

Watch

毫无疑问，自制食物通常对你更好，但有些东西只有由别人(或某物)制作时味道才会更好。我对这个问题有很多想法，而且一如既往，我对你的偏好有疑问。

*   在你看来，哪种食物是最好的自制食物？除非你是糕点专家，否则你可能做不出比在面包店买的更好的羊角面包。除非你在纳贝斯克工作，否则你可能做不出比在塔吉特百货买到的薄荷奥利奥更好的薄饼。
*   你最喜欢的预包装饼干是什么？薄荷奥利奥薄饼。还有女童子军萨摩亚饼干。

*   **薯片呢？**多力多滋美味可口，我永远不会尝试复制它们，但没有什么能比得上金箔莳萝泡菜味薯片的薄脆完美。
*   有没有一种瓶装沙拉酱是你无法在家复制的？好吧，直到今天，本来是 [这个姜汁调料](https://skillet.lifehacker.com/this-ginger-dressing-tastes-exactly-like-it-came-from-a-1827997044) ，但是我终于想通了一个。
*   你对自制薯条有什么看法？不要做！(除非你有一个像样的油炸设置。)除了大量的油炸油，你什么也得不到，而且它们不会像麦当劳或其他有油炸锅的餐馆的薯条那样好吃。
*   子问题:你最喜欢的冷冻薯条是什么？矿-伊达皱切。
*   什么烘焙食品最好留给专业人士来做？前面提到的羊角面包和长棍面包。(除非复杂的烘焙项目对你来说是舒缓的。)
*   自制布朗尼真的比盒装布朗尼好吃吗？很少。[brave tart 食谱中的食谱](https://www.amazon.com/BraveTart-American-Desserts-Stella-Parks/dp/0393239861?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/which-store-bought-foods-do-you-prefer-over-homemade-1828004307&asc_source=&tag=kinjalifehackerlink-20) 是我唯一弄乱的，但你无法击败易味比或盒装组合。很少有自制食谱能达到完美的皱皮和超级耐嚼的边缘。
*   什么样的鸡尾酒是你在家不太会的？我做的马提尼很一般，但是 [佩佩·勒·可比](http://pepelemokopdx.com) 做的马提尼很完美，我不明白为什么他们的马提尼如此完美无瑕。
*   **有哪些快餐或批量生产的零食经过了科学优化？**我喜欢塔可钟 Doritos Locos Taco Supreme 的垃圾完美，塔可钟的“薯条”尝起来像美味的化学战。我对鸡蛋松饼的爱也很强烈。

*   在家制作的调味品中，哪一种(哪些)更糟糕？ [番茄酱。](https://skillet.lifehacker.com/nobody-wants-your-homemade-ketchup-1827140540#_ga=2.265869256.653606342.1532967447-426097860.1521480505)
*   你最喜欢的冷早餐麦片是什么？最近我真的很喜欢嘎吱船长浆果，我不可能做出任何接近它们的东西。

你可以随意回答上面的几个或几个问题，或者简单地漫谈你最喜欢的从商店买的食物或饮料。如果你在破解商店购买的最爱的代码方面取得了巨大的成功，也分享一下吧！
