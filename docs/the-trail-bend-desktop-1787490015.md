# Trail Bend 桌面

> 原文:[https://lifehacker.com/the-trail-bend-desktop-1787490015](https://lifehacker.com/the-trail-bend-desktop-1787490015)

读者 [斯潘塞](https://www.flickr.com/photos/144082128@N08/) 在他的 Mac 电脑上组装了这个定制的桌面，上面只有他想要显示的小部件和工具。壁纸是他自己的一张照片，但如果你想要类似的外观，其余的照片很容易在你自己身上找到。

Watch

斯潘塞使用的是 bersicht，这是一个定制套件，我们在 T5 之前已经强调过的T3。您将需要抓住它开始，但它几乎是所有您需要的。

一旦你有了它，这就是你想要的:

*   壁纸是斯潘塞自己的一张照片(所以你可以通过下面的 Flickr 链接向他要！)
*   用于 macOS 的[über sicht](http://tracesof.net/uebersicht/)定制套件
*   [为über sicht](http://tracesof.net/uebersicht-widgets/#cpu-bar)的 CPU 栏小部件
*   [记忆栏小部件为über sicht](http://tracesof.net/uebersicht-widgets/#memory-bar)T3】
*   über sichtT3 的 [roundClock 小部件](http://tracesof.net/uebersicht-widgets/#roundClock)
*   [现在播放的 widget 为übersicht](http://tracesof.net/uebersicht-widgets/#now-playing)T3】

这里有几个小部件，但是它们都很容易集成到 bersicht 中，所以您实际上只有一个工具在运行。如果你喜欢这种外观，但不确定如何才能获得同样的效果，请查看下面链接的斯潘塞的 Flickr 页面，让他知道你有多喜欢他的作品——或者问问他是如何让一切变得如此完美的。
