# 均衡器上每个滑块的作用

> 原文：<https://lifehacker.com/what-each-slider-on-an-equalizer-does-1818854612>

当你通过廉价的耳机或扬声器听音乐时，均衡器将有助于均衡声音。大多数均衡器都带有流派预置，但您也可以手动调整它们。它有助于了解每种频率影响哪种乐器。



博主 Ziyad Nazem [列出了 iTunes](http://www.ziyadnazem.com/post/956431457/the-perfect-eq-settings-unmasking-the-eq) 上每个频率之间的差异，从许多扬声器系统甚至无法播放的 32 赫兹低音炮声音，到影响人声“鼻音”的中音 2KHz 频率，直到可以让歌曲听起来“嘶嘶”的 16KHz(即使您自己的软件具有不同的参考频率，该列表仍然有效。)

为了避免音乐削波(达到最大音量并降低质量)，请将滑块向下拉，而不是向上拉。然后相应提高整体音量。

正如 Nazem 所说，没有“完美”的均衡器设置；关键是根据你特定耳机的怪癖和选择的曲调，把音乐调整到你喜欢的程度。没有均衡器设置会完全“修复”廉价耳机。你可以通过购买高品质耳机或扬声器并禁用均衡器来获得理想的体验。
