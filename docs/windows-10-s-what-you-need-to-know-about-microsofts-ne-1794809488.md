# Windows 10 S:关于微软新的轻量级操作系统你需要知道什么

> 原文：<https://gizmodo.com/windows-10-s-what-you-need-to-know-about-microsofts-ne-1794809488>

谷歌 [Chrome 已经变得如此之好，以至于说到负担得起的高质量笔记本电脑](http://gizmodo.com/samsungs-new-chromebook-is-a-nearly-perfect-budget-comp-1792183698#_ga=2.119983311.672853170.1493736297-62593606.1484261959) ，它比廉价的 Windows 设备更划算。因此，微软需要一个轻量级的操作系统来竞争，它真的希望专注于教育的 Windows 10 S 将成为那个操作系统。



### 这是什么？

Windows 10 S 是微软新推出的轻量级操作系统。这意味着它比 Windows 10 Home 或 Pro 更便宜，对处理器的要求更低，Windows 10 Home 或 Pro 是当今大多数笔记本电脑和台式机上的操作系统版本。

### 是什么让它变得如此轻巧？

后台运行的程序要少得多。根据微软的说法，运行 Windows 10 S 的笔记本电脑加载用户配置文件(包括应用程序和偏好设置)的速度比运行 Windows 10 Pro 的相同笔记本电脑快 15 秒。

### 我为什么要在乎？

如果您在学校或在学校工作，这是一件大事，因为它将使整个校园的计算机使用标准化变得容易(今天的大部分公告演示文稿都集中在学校的教师和 it 工作者)。如果你不在教育行业，Windows 10 S 可能会是一笔糟糕的交易。但在我们开始看到内置 Windows 10 S 的电脑之前，我们不会知道。如果硬件足够好，足够便宜，它可能是 Chrome OS 的竞争对手，至少它可以运行完整的微软 Office 套件[。](http://gizmodo.com/why-are-you-still-using-microsoft-word-1794219158)

### 微软以前不是做过轻量级的 OS 吗？

是啊！Windows 10 S 并不是微软第一次尝试轻量级操作系统。在 2011 年的 CES 上，微软推出了 Windows RT，这是 Windows 8 的一个专注于移动的版本，旨在降低成本，但让消费者继续使用 Windows。不幸的是， [RT 在做](http://gizmodo.com/what-windows-rt-cant-do-5952934) 的事情上超级受限。最大的问题是应用程序。你不能使用旧的应用程序或任何酷的东西，除非它已经针对 Windows 应用商店进行了更新。商店也很糟糕——经常对在非 RT 电脑上免费的应用程序收费。 [RT 还占用了笔记本电脑的大量空间](http://gizmodo.com/windows-rt-takes-up-half-of-the-32gb-microsoft-surfaces-5957824) ，这让你不得不为存储支付更多费用，或者忍受 32GB 存储驱动器上只有 16GB 空间的痛苦。

2012 年微软停止更新 RT 后，将精力集中在一款新产品上，即带有 Bing 的 Windows 8.1。这是一个比 RT 功能更全的“轻量级”操作系统，但它仍然需要你从 Windows 商店获得应用程序，并且它由不太出色的必应搜索服务提供补贴。这意味着你不得不在你的桌面上看广告，更糟糕的是，你不得不经常使用必应 。带有 Bing 的 Windows 8.1 只能由制造商安装在电脑上。不幸的是，原始设备制造商显然对补贴操作系统不感兴趣。

### 这与以前的轻量级 Windows 版本有何不同？

Windows 10 S 似乎从其轻量级、低成本的前辈那里借鉴了一些想法。这意味着应用程序只能来自 Windows 应用商店。(尽管数字趋势公司表示，未来的更新将允许你安装任何你喜欢的应用程序 )。这令人失望，但至少自 2014 年以来，商店已经有了很大的改善——上次微软要求我们只能使用它。

微软声称此举是出于安全考虑。通过限制仅从商店安装产品，简单的应用程序安装不太可能使计算机感染病毒或严重破坏操作系统。如果 Windows 10 S 机器在一个满是不知道如何安全浏览的孩子的教室里，这是很有用的——这也是微软选择将 Windows 10s 作为教育工作者解决方案的原因之一。

对于我们其他人来说，这仍然是一个挫折。虽然该商店现在有一些有用的应用程序，如微软 Office 和 Adobe Photoshop Express，但它缺乏许多其他大型应用程序，如 Chrome。

这意味着如果你想在 Windows 10 S 设备上浏览互联网，你需要准备好接受微软 Edge。

### 它将在哪些系统上可用？

包括戴尔、惠普和宏碁在内的主要电脑制造商推出的大量廉价笔记本电脑。最便宜的笔记本电脑起价仅为 189 美元。

### 多少钱？

它不需要任何成本，因为它只能预装在设备上或通过学校技术代理使用的供应商获得。

但如果你使用的是 Windows 10 S 设备，并且厌倦了被 Windows 商店束缚，如果你是学生，你可以免费升级到 Windows 10 Pro，如果不是学生，则需要 49 美元。

### 什么时候有？

Windows 10 S 预计将于今年夏天推出。
