# 支持治疗这些(其他)致命癌症的研究

> 原文：<https://lifehacker.com/support-research-to-cure-these-other-deadly-cancers-1829533045>

乳腺癌是常见的，也可能是毁灭性的，但它比许多其他癌症得到了更多的媒体关注。当你掏出钱包去买一些粉红色的首饰时，这里有几个需要一点爱的理由。



以下数字来自 [美国国家癌症研究所的常见癌症列表](https://www.cancer.gov/types/common-cancers) ，加上一些来自美国临床肿瘤学会 的皮肤癌 [信息。我们按照它们每年导致多少人死亡的顺序对它们进行了排名，并推荐了一些专注于研究的相关慈善机构。如果你不能决定，还有美国癌症协会的。](https://www.cancer.net/cancer-types/skin-cancer-non-melanoma/statistics)

*   肺癌:154050 人。试试美国肺癌基金会的。
*   结肠直肠癌:50，630 人。试试大肠癌联盟。
*   胰腺癌:44330 例。试试 [胰腺癌行动网](https://www.pancan.org/) 。
*   乳腺癌:每年 40，920 名女性，480 名男性。代替科曼，考虑支持 [乳腺癌研究基金会](https://www.bcrf.org/) 。
*   肝癌和肝内胆管癌:30，200 人。试试 [美国肝脏基金会](https://liverfoundation.org/medical-professionals/alf-research-awards-program/) 。
*   前列腺癌:29430 人。试试 [前列腺癌基金会](https://www.pcf.org/research/) 。
*   白血病:24370 例。 [试试白血病研究基金会](https://www.allbloodcancers.org/) 。
*   非霍奇金淋巴瘤:19910 人。试试 [淋巴瘤研究基金会](https://www.lymphoma.org/) 。
*   膀胱癌:17240 人。试试 [膀胱癌研究网](https://www.bcan.org/bladder-cancer-research-network/) 。
*   肾癌:14970 例。试试肾癌协会的。
*   子宫内膜癌:11350 例。试试 [女性癌症基金会](https://www.foundationforwomenscancer.org/types-of-gynecologic-cancers/uterine/) 。
*   黑色素瘤:9320。试试 [黑色素瘤研究基金会](https://www.melanoma.org/) 。
*   非黑色素瘤皮肤癌:5860 例。试试 [皮肤癌基金会](https://www.skincancer.org/giving/supporting-the-foundation) 。
*   甲状腺癌:2060 人。试试美国甲状腺协会的。

有很多其他癌症相关的慈善机构可能需要你的支持，所以把这个列表作为一个起点。也有一些罕见的癌症不在这个名单上，但仍然是致命的，可以使用研究经费，所以也考虑支持这些事业。一些罕见癌症- 的相关组织被列入罕见癌症研究基金会。
