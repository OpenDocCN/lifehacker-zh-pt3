# MoviePass 的订阅费现在是 7.95 美元，并附带免费的 Fandor 订阅

> 原文：<https://lifehacker.com/moviepass-subscriptions-are-now-7-95-and-come-with-fan-1822898410>

MoviePass 正试图以更低的新价格吸引更多的客户，免费访问流媒体服务 Fandor。如果你一直对参军持观望态度，现在是时候了。

Watch

现在，新注册使用 MoviePass 服务的用户将每月支付 7.95 美元，而不是每月 9.95 美元。Fandor 的订阅也伴随着这项协议，它让用户能够访问各种各样的独立电影。

MoviePass 首席执行官米奇·劳 [向*【好莱坞报道】*](https://www.hollywoodreporter.com/news/moviepass-cuts-price-again-adds-streaming-1083507) 描述了作为一项实验的促销协议，所以如果你想利用它，你应该尽快这样做。该公司还没有宣布低价何时结束，因此可能只会持续几天。

像任何好交易一样，这一笔也有很多陷阱。

首先，注册需要 19.95 美元的手续费，这将使你一年的总费用达到 115.35 美元。你必须一次签约一年，所以你们都必须有足够的可支配收入，并准备好承诺一年的电影通行证。

上个月晚些时候，MoviePass 还宣布了一些值得一看的新条款和条件。

由于与该品牌的纠纷，MoviePass 最近也选择停止允许客户在 AMC 的一些最大的影院使用该服务。看起来 AMC 和 MoviePass 都没有在这场战斗中退缩，所以你可能会发现更多的影院被取消这项服务。

最后，一些客户最近站出来抱怨他们的订阅被 MoviePass 取消了，因为他们“看了太多电影”

MoviePass 声称，这些订阅被取消了，因为人们使用这项服务来抵消观看 Fathom Events 或 IMAX 电影的费用(movie pass 只能用于常规的 2D 电影)，或者他们购买了门票，然后在几分钟后兑换成礼品卡。

这绝对是一个他说，她说，但值得你注意的事情。
