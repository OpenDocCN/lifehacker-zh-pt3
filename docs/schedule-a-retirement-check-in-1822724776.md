# 安排退休登记

> 原文:[https://life hacker . com/schedule-a-retirement-check-in-1822724776](https://lifehacker.com/schedule-a-retirement-check-in-1822724776)

你知道你应该为退休存钱，但是你真正存了多少呢？你最后一次检查你的储蓄是什么时候？

Watch

这不是新闻，你需要知道你在哪里，以确保你在正确的轨道上。所以，每隔几个月安排一次退休检查(在你下一个个人盘点日期间进行是个好任务)，看看你的情况如何。

俄亥俄州特拉华州的注册理财规划师、[forest view Financial Partners](http://www.forestviewfp.com/)LLC 的总裁马修·斯图尔特(Matthew Stewart)说:“知道自己与退休目标的关系是应对市场波动的最佳应对工具之一。”。“这有助于你对事物保持长远的眼光...它只是有助于将事情保持在你面前。”

以下是你在办理退休手续时要做的事情:

### 第 0 步:开一个退休账户

如果你还没有为退休存钱，你还在等什么？也许你没有赚很多钱，或者你有其他的目标，比如偿还你的学生贷款，这些目标看起来更重要。但是如果你没有，你应该现在就开始存钱。

你开始得越早，你每个月为达到目标需要存的钱就越少。隐安·奥伦和哈罗德·波拉克在索引卡中建议:“如果你早点把钱存起来，你就可以把你的投资利润进行再投资。”。又名你受益于复利。

如果你是全职员工，你可能会选择开一个 401(k)账户。如果你是自由职业者或为小公司工作，你需要开一个个人退休帐户(如果你有 401(k)计划，你也可以开一个个人退休帐户，我们将在下面讨论)。

以下是我们的一些入门指南:

如果你的雇主提供 401(k)和缴费匹配，缴费达到这个数额，然后考虑开设个人退休帐户或罗斯个人退休帐户。使贡献自动化。

### 第一步:知道你的目标

如果你很年轻，你可能没有退休目标，你只知道你需要存钱。在你职业生涯的开始，这很好，但随着你的进步，你应该决定你的目标是什么。你打算继续住在你的房子里吗？搬到税率较低的州？环游世界？

也许你心中有一个具体的数字，一个你想退休的年龄，或者你只是想尽可能多地存钱。你可能会决定退休后继续做兼职，或者依靠你的投资生活。

有 [退休计算器](https://www.nerdwallet.com/investing/retirement-calculator) 你可以用它来了解你需要存多少钱，或者你可以去见一个理财规划师，他可以给你个性化的策略。下面是一些T5 其他选项 T7】。

### 第二步:检查你的余额

登录你的退休账户，看看你存了多少钱——标准建议是把你税前收入的 10%到 15%存起来。这不是每个人都能做到的，但是你存得越多，哪怕是每月多存几美元，你未来的日子就会越好。

以下是 2018 年退休账户的缴费限额:

*   401(k)::50 岁以下的 18500 美元，50 岁以上的 24500 美元
*   50 岁以下的人 5500 美元，50 岁以上的人 6500 美元
*   罗斯个人退休帐户:50 岁以下 5500 美元，50 岁以上 6500 美元
*   **SEP IRA:** 不得超过员工薪酬的 25%或 55，000 美元，以较低者为准
*   **简单 IRA:**[【12500 美元](https://www.irs.gov/retirement-plans/cola-increases-for-dollar-limitations-on-benefits-and-contributions)

如果你有 401(k ),至少为雇主匹配贡献*(例如，你的雇主可能匹配你工资的 4%)。当你接受一份工作时，401(k)匹配是你薪酬的一部分——所以如果你没有支付全部匹配金额，就像你告诉你的雇主他们可以支付你低于你同意的金额。你会告诉你的老板可以从一两个月的薪水中扣除一百美元左右吗？当然不是。所以不要用你的退休账户。*索引卡*上写道:“简而言之，*没有比合理使用你的固定缴款退休账户*更好、更有效或更有利可图的方式来开始建立和增加你的长期储蓄了。”。*

税务律师、注册财务规划师丽贝卡·瓦尔泽建议年轻人先缴纳雇主匹配金，然后再缴纳更多的罗斯个人退休账户金，她说这对于现在工作的人来说有更好的税收意义(有了罗斯个人退休账户，你存钱时就要交税，这意味着它可以无限期免税)。瓦尔泽说:“对于正在积累退休财富的千禧一代甚至 x 世代来说，在 401(k)计划中积累是在用更少的薪水积累。”"现在付税，把它存入一个罗斯."

### **第三步:将你的贡献增加 1%**

你能忍受每个月不去餐馆吃一两顿饭吗？然后考虑将你的贡献增加 1 %,即使你注册了自动升级。如果您还没有，并且您的计划允许，请选择每年自动增加一个百分点供款的选项。

### 第四步:重新平衡

虽然如果市场下跌，你不应该恐慌和重新平衡，但每隔一段时间，你就需要 [重新平衡你的资产配置](http://time.com/money/4964526/do-this-one-thing-each-year-if-you-want-a-better-retirement/) (你账户中的钱如何投资)，并以你觉得舒服的方式改变你的股票债券组合。

“关掉市场新闻，尽量不要太关注它。尤其是在市场持续走低的情况下，”Forestview Financial Partners 的斯图尔特表示。“如果你的投资按照长期计划得到了合理配置，那么你就不会在乎市场在更短的时间内，比如几天、几周甚至几个月内会发生什么。你担心的是它的多年回报率，比如 3 年、5 年、10 年的平均回报率。”

有些人会用目标日期基金来自动重新平衡，但要知道，这类基金可能会收取与主动管理型基金类似的高额费用。

康涅狄格财富管理公司的董事 Jarrett Solomon 告诉,，你可能只需要每隔几年做一次。

> “只要你的资产组合偏离目标 10%到 15%，你就可能做得最好，”所罗门说。因此，如果你的目标是将 60%的投资组合放在股票上，那么当这个比例上升到 66%到 69%时，你应该卖出一些股票，如果市场暴跌将你的敞口降至 51%到 54%之间，你应该买入更多股票。

对于大多数 401(k)提供商来说，这很简单——你可以在网站上做，或者直接打电话给基金经理，让他们在你在线的时候做。

此外，检查与您的计划提供商提供的不同共同基金相关的费用。看似很小的费用却有所不同，正如 [【今日美国】](https://www.usatoday.com/story/money/personalfinance/2017/01/03/401k-fees-could-eating-away-your-retirement-savings-right-now/93819526/) 所指出的，

> 这些费用通常以百分比的形式表示为“费用比率”虽然像 1.1%这样名义上很小的年费乍一看可能听起来不错，但它在一个重要的储蓄账户上累积起来就是一大笔钱——例如，在 50 万美元的投资组合中，每年高达 5500 美元。

这里有一个 [计算器](http://401kfee.com/) 可以帮你搞清楚有多少费用侵蚀了你的余额。

还有一点要注意: [重新平衡一个应税账户](http://time.com/money/4517972/retirement-portfolio-rebalancing-advice-401k/) 会让你付出代价，重新平衡一般会有税务影响。所以这绝对不是一件可以养成习惯的事情。

### **第五步:滚转旧的 401(k)账户**

如果你有多份工作，你可能有不止一个 401(k)计划。考虑把它们转到你现在的 401(k)计划中，让事情变得更简单，这样你就不会忘记这笔钱了。检查基金选项和相关费用，看看它是否有意义。你可以在上研究各种方案的价值。并且知道你的公司不允许延期。

这伴随着一个重要的警告:如果你不检查细则，特别是如果你想延长个人退休帐户，这可能会让你付出代价。索引卡上写着*展期个人退休账户通常伴随着高昂的费用，随着时间的推移，这些费用会拖累你的储蓄。事实上，美国证券交易委员会和金融行业监管机构几年前都在调查展期个人退休账户营销，以了解它如何影响退休储蓄者。*

“从 401(k)转移到 IRA 的工人正在离开高度监管的计划，在这些计划中，雇主必须为参与者的最大利益行事，而在监管较少的市场中，经纪人通常不需要将投资者的利益放在第一位，”KiplingerT3 报道说。

所以检查小字，然后再检查。政府问责局发现，30 家个人退休帐户提供商中有 7 家声称提供“免费”个人退休帐户，但没有充分解释投资者仍然支付投资费用。

这可能看起来工作量很大，但是请记住，你每年只安排一次或两次——潜在的回报是为你的无压力退休做准备。
