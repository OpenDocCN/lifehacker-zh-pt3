# 瓷砖是最糟糕的科技产品

> 原文:[https://gizmodo . com/tile-is-the worst-tech-stocking-stuffer-1821046107](https://gizmodo.com/tile-is-the-worst-tech-stocking-stuffer-1821046107)

我丢东西。很多很多东西——我的钥匙(在沙发上)，我的夹克(在出租车里)。有一次，在占领华尔街抗议期间，我把我的 MacBook 忘在了一家地铁餐厅里。我很幸运；第二天早上，当工作人员供应早餐时，我发现它一动不动地放在一把椅子下面。我承认:当要记录我的东西时，我需要一些帮助。但是最亲爱的朋友、家人和同事们，不管你们在想什么，不要给我买瓷砖。

Watch

“可是！但是！瓷砖是很棒的礼物！!"你，一个错误的人，说。“显然你可以用一个。”嘘！我代表世界各地健忘的人说:我们意识到了这个问题，当我们找到它时，我们会解决它。如果你想要一块瓷砖，当然，去给自己买一块吧。但是瓷砖完全是蹩脚的礼物——廉价的令人沮丧的塑料块。

也许你在百思买，浏览一些网站，或漫步经过机场自动售货机时，你看到了展出的这些蓝牙跟踪器之一。这个小小的[【25 美元](https://www.amazon.com/Tile-Mate-Finder-Anything-Finder/dp/B01L3VEC08/ref=sr_1_3?asc_campaign=InlineText&asc_refurl=https://gizmodo.com/tile-is-the-worst-tech-stocking-stuffer-1821046107&asc_source=&ie=UTF8&keywords=tile&qid=1512656754&sr=8-3&tag=kinjagizmodolink-20) 加密狗被设计用来连接你担心可能会丢失的东西，所以你可以用一个应用程序跟踪你的东西在哪里。也许在最激动的时候，当你的假日购物压力达到顶峰时，你一看到它就松了一口气。"一份*简单的*礼物？"你以为。"一个整洁的圣诞礼物？"不，别这么做。瓷砖是一种礼物，用闪亮的白色塑料包裹着，既怪异又没有人情味，还带有侮辱性。是叫你不负责任的礼物。挂在钥匙链、相机或背包上的瓷砖发出尖叫声:“我把它放在这里是因为我的脑子一团糟。我丢了东西，每个人都必须知道。”瓷砖也是一个很小的设置负担，更大的使用负担，并且在概念上令人沮丧，因为它们不能提供你所期望的内心平静。

瓷砖迫使你的大脑担心跟踪，这是因为瓷砖并不像它应该的那样被动。

为了最大限度地利用磁贴，您需要始终打开手机的蓝牙设置。默认情况下，无论何时关闭蓝牙，你的磁贴都会不断提醒你这个事实。如果你有 iPhone，你也需要让 Tile companion 应用程序一直开着，以便跟踪你可能在蓝牙信号范围之外丢失的东西。不要担心，无论何时关闭，磁贴应用程序都会告诉您这一点。即使你把所有这些东西都关掉，如果你选择进入该公司的失物招领网络，该应用程序还会让你知道其他瓷砖何时靠近你丢失的东西，即使你的东西实际上并没有丢失。

所有这些并不是说瓷砖毫无用处。在许多方面，它们完成了工作——只要你不意外打破它们，记得在电池没电时更换它们，如果你不介意付出努力的话。但是磁贴通常是令人沮丧的小东西，除非它们更被动地工作(就像苹果的查找我的 iPhone 功能)，否则对大多数人来说，它们会比它们的价值更麻烦。对其他人来说，你可以给自己买一个。如果你真的不同意我的观点——如果你认为瓷砖真的很棒——那么我敢打赌你已经这么认为了。
