# 当青少年的大脑准备好学习时，学校应该推迟开学

> 原文:[https://life hacker . com/schools-should-start-la ter-when-teen-brains-ready-t-1800671670](https://lifehacker.com/schools-should-start-later-when-teen-brains-are-ready-t-1800671670)

青少年在课堂上打瞌睡是有原因的，而不仅仅是关于多项式的最后一堂课。他们困了。他们犯困的一个原因是学校太早开学了。

多年来，人们一直在推动初中和高中学生推迟上学时间，这些学生每天早上 7 点就拖着身体坐到座位上。 [美国儿科学会](http://pediatrics.aappublications.org/content/early/2014/08/19/peds.2014-1697)[美国心理协会](http://www.apa.org/pi/families/resources/school-start-times.pdf)[国家睡眠基金会](http://www.sleepfoundation.org/article/hot-topics/backgrounder-later-school-start-times)[疾病控制和预防中心](https://www.cdc.gov/media/releases/2015/p0806-school-sleep.html) (CDC)，这里仅列举几个团体。然而，正如大多数学区所代表的那样，学校的时间安排与青少年生物学并不同步。

虽然大多数小孩天生爱早起，但在青春期，睡眠-觉醒周期会发生变化，大多数青少年在晚上 11 点至早上 8 点之间获得最佳睡眠 T3。因此，当一个学生不得不在早上 6 点起床(或者更早去赶公交车)时，那就是每天损失两个或更多小时不可恢复的睡眠。正如 [一项研究所说的那样，](http://www.tandfonline.com/doi/full/10.1080/17439884.2014.942666) “对于年龄较大的青少年来说，早上 7 点的闹钟相当于一个 50 多岁的老师早上 4:30 就开始工作。”

当然，这可能会导致与低效睡眠 相关的 [问题，包括抑郁、肥胖、偏头痛和免疫系统减弱。睡眠不足的青少年也更有可能卷入车祸，并从事诸如吸烟、饮酒和滥用药物等危险行为。](https://sleepfoundation.org/sleep-news/insufficient-sleep-among-high-school-students-associated-variety-health-risk-behaviors)

好消息是事情终于有所改变。45 个州的学校正在推迟开学时间，以更好地符合健康指南。在加利福尼亚州， [州议会正在辩论的一项法案](http://laschoolreport.com/later-school-start-time-could-boost-californias-economy-study-shows/) 将禁止初中和高中早于上午 8:30 开始上课。其好处将超越个人层面——[兰德公司最近的一项研究](https://www.rand.org/news/press/2017/08/30.html) 发现，在十年内，全国范围内将上课时间改为上午 8:30 可以为美国经济贡献 830 亿美元。学生可以在学术上取得更大的成就，并且更加健康和安全。

有人反对——有些人担心推迟开学时间会给课外活动留下很少的时间，或者这种转变会让父母上班迟到，或者没有足够的空间让所有孩子同时开始上学，因此交错的开学时间肯定存在。但是倡导团体“晚开学”对你能想到的任何问题都有的回应。

父母也应该尽自己的一份力量，通过强制执行屏幕时间限制和帮助他们的孩子得到更好的睡眠来做到这一点 。如果你的学校还没有加入，以下是你如何 [开始请愿](http://www.startschoollater.net/sign-or-start-a-petition.html)
