# 何时以及如何与你的搭档写一份财务协议

本周的问题来自一位希望匿名的读者的电子邮件:*

> 我刚买了一套房子，将在明年左右结婚(我们还没有确定日期)。我和我的未婚夫将把这栋房子作为我们的主要住所。在申请抵押贷款时，只考虑我的工资和资产，我只用我的信用购买了房子。还没有付款(除了首付，都是我付的)，但我们有一个口头协议，每人支付 50%的抵押贷款。我知道每个婚姻都不成功。那么，我们是否应该寻求某种合法的财产协议，明确规定每个人在结婚前拥有的房产占总房产的比例？我们没有任何额外的资产，所以婚前协议似乎没有必要。我的未婚夫可以接受灰色地带，但我不行，我喜欢事先明确说明事情的安全感。

这是个别专家对一个对每个人都有不同影响的问题的一般看法——如果你想要个性化的建议，你应该去见理财规划师。

### 想想什么让你最舒服

回答你这个问题的关键在最后一句。你的未婚夫可能会适应灰色地带，但你不会——所以写点东西是有意义的。很可能，你已经知道了。

金钱是人际关系中的头号压力源。一份书面协议不会解决所有潜在的未来问题，但如果你和你的未婚夫分手了，它会让你更容易处理这些问题，不管怎样，它都会让你安心。事情发生了，你需要保护自己，尤其是因为听起来你已经承担了所有的责任。

不浪漫，但很现实。所以，调查一下同居协议，它类似于婚前协议，将使你所做的任何口头协议正式化。

“如果你打算让你的伴侣成为财产的所有者，你也可以这样规定，”詹妮弗·麦克德莫特 、 [、finder.com](http://finder.com/)的消费者权益倡导者在谈到这些协议时说。“这可能感觉非常正式，但为了避免未来出现任何问题，投资是非常值得的。”

最好你能在签署文件之前就这样做，但是现在还不算太晚。房地产律师或家庭律师可以为你起草一份同居协议，它应该包括房屋产权如何分割，如果你们分手会发生什么，谁负责其他账单(水电费、有线电视费)和修理费，如果你们中的一个不能支付自己的一半会发生什么，如果你们中的一个有一天想卖掉房子，或者如果你们中的一个去世了，等等。这对你来说尤其重要，因为你是支付首付的人，而且虚线上很可能是你的名字。

### 这也符合你伴侣的最佳利益

你的伴侣可能不介意灰色地带，但你应该解释起草一份协议也符合他们的最大利益。如果你的伴侣没有所有权，即使他们支付了一半的抵押贷款，他们也没有投资权益。

另一件要记住的事情是:你需要为丧失能力或死亡做准备，不管这看起来有多可怕。国际金融理财师的黛布拉·内曼告诉 [Bankrate](https://www.bankrate.com/finance/real-estate/homebuying-tips-unmarried-couples.aspx) “如果你在没有遗嘱的情况下去世，你希望你的财产归你的共同所有人所有，除非在遗嘱中有明确规定，这是不会发生的。”。“它可能会去找你的父母。或许你的父母不支持你们的关系。”

麦克德莫特补充道:“如果你希望你的另一半在你生病时负责与你的护理有关的决定，或者在你去世时从你的退休计划、保险单或资产中获得任何好处，你需要将这一点写下来。”“更新他们作为受益人的任何政策，并要求律师起草一份财务委托书。”

以下是一些需要记住的其他事情:

*   实现财务透明。麦克德莫特说:“在做出同居这样的重大决定之前，坦白任何不为人知的财务丑闻是很重要的。“在这次讨论中，你应该涵盖的要点包括你对储蓄和消费的态度、你的信用记录以及任何债务的细节。”
*   **设立一个联名银行账户**，抵押贷款的每一半可以直接存入该账户。
*   为家庭开销做一个预算。“建立一个新家可能会很贵，而且有些东西对一方来说比另一方更重要。麦克德莫特说:“通过就谁在购买和‘拥有’沙发或大屏幕电视等大件商品达成一致来避免争论。”"保留购买记录，以避免分割时所有权上的任何分歧。"
*   讨论如何分担家务以及谁将负责修理或雇人修理。例如，如果你需要一个新的屋顶，会发生什么？你将支付它，还是它将是共同的费用？
*   考虑一下如果你不能支付抵押贷款会发生什么。你说你用信用卡买了房子。这意味着如果你和/或你的未婚夫没有付款，你的信用就会受到影响。
*   如果你有了孩子会怎么样？这是你在起草协议前需要进行的另一次讨论。

你想做让你最舒服的事情，同时尽量减少争论和潜在的法律纠纷。不要留下任何“如果”吊着。
