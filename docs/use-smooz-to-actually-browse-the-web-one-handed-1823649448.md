# 使用 Smooz 实际上可以单手浏览网页

> 原文：<https://lifehacker.com/use-smooz-to-actually-browse-the-web-one-handed-1823649448>

我不太喜欢在手机上浏览网页，主要是因为我的浏览器糟糕的控制方案。一切都太分散了。键盘在垂直方向显示屏的底部，重要的控件藏在难以触及的角落，我不想冒险用单手操作新标签来摔手机。然而，使用 [iOS/Android 浏览器 Smooz](http://smoozapp.com/) ，你可以避免依赖一个不可访问的界面，而使用一个更适合所有滑动、点击和浏览的界面，这要归功于该应用对手势控制的强调。下面是它的工作原理，以及如何充分利用它:

### **简单的同步，更简单的标签管理**

使用谷歌 Chrome 这样的浏览器时，最困难的部分是到达角落浏览其他标签，输入新的网址，或访问我需要更改的设置。Smooz 首先依靠手势控制，让你可以在标签之间滑动，输入新的 URL，并管理当前的标签集。还有一个标签管理器，你可以用它来锁定、重新排列和恢复经常出现的和最近关闭的标签。当您将脸书或 Twitter 帐户与应用程序关联时，Smooz 会在您的设备之间同步您的浏览历史记录、书签和标签。它的基本手势与其他浏览器不相上下，但真正有用的功能在于它的高级手势控制。更多信息请见下文。

### **自己定制手势**

单手魔法以可选手势的形式出现，你可以在 Smooz 的设置中启用。有五种预先设定的手势，你可以用一个动作对每一种进行编程。现在，我可以通过向上向右滑动来打开一个新标签，向下向左滑动来关闭它，三次点击来关闭所有标签。有十几个操作可供选择，因此您可以打开书签文件夹，恢复关闭的标签，或进入私人模式，而无需过度伸展您的拇指，非常适合小手智能手机用户。

### **浏览免费，广告拦截额外收费**

Smooz 是免费的，但提供了一个高级版本，提供了广告拦截和无限的文本识别等功能每月 3.49 美元，你最好真的不喜欢广告。可定制的手势控制和简单的标签管理使 Smooz 成为一个非常强大的浏览器。其他简洁的功能，如减少数据使用的图像屏蔽，以及让你看到其他 Smooz 用户对特定网站的评论的浏览器内评论功能(你可以禁用它)，使它成为 Safari 或谷歌 Chrome 等其他可用选项的强大替代选择，我相信我会继续使用一段时间。
