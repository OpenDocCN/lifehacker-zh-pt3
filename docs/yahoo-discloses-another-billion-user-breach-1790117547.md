# 雅虎披露另一个十亿用户入侵

> 原文：<https://lifehacker.com/yahoo-discloses-another-billion-user-breach-1790117547>

雅虎今天下午宣布，黑客在 2013 年的一次攻击中窃取了超过 10 亿用户账户的数据。据信，这次攻击与他们在九月 宣布的 [*和其他* 2013 年的漏洞是分开的。](https://lifehacker.com/500-million-yahoo-accounts-hacked-change-your-password-1786958537) 



攻击者被认为是第一次攻击中的“同一个国家支持的演员”，他们带着大量的用户数据逃走了。这包括姓名、电子邮件地址、电话号码、出生日期、散列密码(使用 MD5)，甚至加密和未加密的安全问题和答案。然而，调查人员**并不**认为未加密的密码、信用卡或借记卡号码或银行账户信息被盗。

雅虎正在通知任何潜在受影响的用户，但最好是积极主动，开始自己保护您的信息。尽快更改您的密码， [使用强密码](http://lifehacker.com/four-methods-to-create-a-secure-password-youll-actually-1601854240) 或 [一个密码管理器](http://lifehacker.com/lifehacker-faceoff-the-best-password-managers-compare-1682443320) ，千万不要使用与您在另一个网站使用的相同密码。如果你还没有这样做，请尽快为你的雅虎账户 启用双因素认证。你可以在下面的链接中阅读更多关于这个漏洞的信息。
