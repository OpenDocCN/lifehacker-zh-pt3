# 拥抱“支付的痛苦”来省钱

> 原文：<https://lifehacker.com/save-money-by-embracing-the-pain-of-paying-1828933622>

如果你想让你的支出井然有序，一个更有效的方法是拥抱“支付的痛苦”，正如乔·平斯克在 [*【大西洋】*](https://www.theatlantic.com/family/archive/2018/09/spending-personal-finance-pain-of-paying/569575/) 中所写的。举例来说，这意味着用现金而不是信用卡支付。因为信用卡允许我们现在购买，以后支付，我们倾向于对我们的购买和超支感觉更好；用现金支付是一种精神上更痛苦的购物方式，因为你当场就把钱花掉了。



平斯克写道，通过利用支付的痛苦，你也许可以在自由支配的支出上节省一些钱，他说他在完成每一笔信用卡交易后都会在手机上记录下来。“我买了东西后，会在手机上记录交易，记录价格和我买了什么，”他写道。“这个想法是通过强迫自己记下自己的支出来增加支付的痛苦，特别是用信用卡支付。”

在制定规则后，平斯克估计他的可自由支配支出下降了 10%至 15%，并补充说，虽然这在短期内增加了痛苦——这是放弃冲动消费的好方法——但从长期来看，它减轻了不知道你的信用卡账单会有多少的痛苦。双赢。

行为经济学家丹·艾瑞里 建议，有一种方法可以改进平斯克的系统:在你购物前*把它们记录下来。这更有可能抑制不必要的支出。*

Ariely 的另一个建议是，考虑用你将要花的钱还能买些什么。例如，你可以制定一个规则，每周花一定数量的钱，周末剩下的钱可以花在晚餐上。

像 [Mint](https://twocents.lifehacker.com/how-do-you-customize-mint-1828887234) 这样的预算追踪器提供了这样做的方法，但在 Notes 应用程序(或类似 Bear 或 OneNote 的应用程序)中记录交易的想法很简单，比按类别分类更有吸引力，也更省时。

平斯克指出，该系统不会解决你所有的钱的问题。他写道:“相比不喝一杯咖啡，重温一个人的汽车或房屋保险政策，或一个人的电话账单，通常会节省更多的钱。”做这么多微小的决定会增加你与金钱相关的压力。在一些鸡毛蒜皮的事情上花费 [就好了，毕竟](https://twocents.lifehacker.com/its-ok-to-buy-the-damn-coffee-1825722875) 。但如果你正试图削减或更好地控制你的可自由支配支出，这是一个很好的系统。
