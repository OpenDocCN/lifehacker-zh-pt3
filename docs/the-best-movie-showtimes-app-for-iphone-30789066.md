# iPhone 的最佳电影放映时间应用程序

> 原文：<https://lifehacker.com/the-best-movie-showtimes-app-for-iphone-30789066>

在一个有多个应用程序帮助你在大屏幕上观看电影的世界里，有一个应用程序脱颖而出。它是免费的，很容易使用，是在旅途中购买电影票的最佳方式。来到你身边的一部 iPhone:Fandango。

### **特性**

*   按电影、剧院或时间段查找放映时间。
*   “现在就去”功能为您找到附近影院的下一场电影放映时间。
*   将特定的剧院添加到收藏夹列表中。
*   将电影添加到您的收藏夹，以创建您想要观看的即将上映的电影或您已经看过并想要评价/评论的电影的观看列表。
*   当您的收藏夹列表中即将上映的电影的门票开始销售时，您会收到通知。
*   当您已购买门票的电影即将开始时，会收到提醒通知。
*   观看即将上映电影的预告片。
*   研究演员、工作人员和电影简介信息。
*   通过 iPad 上的“脉搏”实时查看哪些电影票正在热卖

### **擅长的地方**

Fandango 应用程序拥有所有基本功能，如通过电影、影院、GPS 位置或时间范围搜索电影放映时间，但它也是用手机购买电影票的最佳方式。Fandango 的电影票购买系统(包括无纸、可扫描的移动电影票 )被其他几个电影放映时间应用程序和服务使用，所以为什么不跳过中间人应用程序，直接进入源头呢？你可以在应用程序中保存你的信用卡信息，只需轻按几下，就可以一次性为全家购买电影票。

“现在就去”功能让你把一个无聊的晚上变成一个自发的电影之夜。你会看到你附近的每一家影院，以及即将上映的电影，这样你就不必再做任何研究了。只需选择电影，在应用程序中购买门票，然后出发。你也可以得到去电影院的路线，Fandango 让你选择使用谷歌地图还是苹果地图。如果您启用了通知，您会收到一个提醒，提醒您电影将在半小时后开始。这样，如果你外出走动，你就不会分心去做别的事情，也不会错过在演出开始前买零食的机会。

Fandango 的收藏夹系统也简单明了。你可以将某些影院指定为你的首选目的地，还可以关注尚未上映的电影。只需在应用程序中选择一家影院或电影，然后点击右上角的心脏。当你寻找放映时间时，你最喜欢的任何一家影院都会首先出现在列表中，如果你大部分时间都去同一家影院，这很好。当您喜欢某部电影时，该电影会被添加到您的“我的电影”列表中。到了那里，你会收到最新消息、独家优惠和门票发售时的通知，这样你就不会错过首映周末。

### **不足之处**

然而，Fandango 并不完美。有两件事让 Fandango 远离了那个头衔:广告和评论分数。无论你在 Fandango 应用的哪个标签页，通常在屏幕的顶部或底部都会有一个小横幅广告。偶尔，当你换标签时，你也会看到全屏广告。没有“额外”费用或会员或应用程序版本来摆脱它们，所以它们可能会让一些人感到沮丧。

当谈到 Fandango 的评论分数时，它们不像 Flixster 和 IMDb 应用程序等竞争应用程序中的分数那样有用。Fandango 的用户评论分数是基于五星系统的，与像 [【烂番茄】](http://www.rottentomatoes.com/)这样基于百分比的汇总系统相比，更难从中收集任何信息。此外，根据 FiveThirthyEight 最近的一份 [分析，Fandango 的粉丝评论分数值得怀疑，因为电影似乎从来没有少于三星。幸运的是，Fandango 确实显示了专业电影评论家的评论分数(以及基于所有链接的评论家分数的 Metascore)，以及每个完整评论的链接供您阅读。](http://fivethirtyeight.com/features/fandango-movies-ratings/)

### **比赛**

Fandango [刚刚收购了其最大的竞争对手之一](http://variety.com/2016/digital/news/fandango-rotten-tomatoes-flixster-1201708444/) 、Flixster(我们之前的选择)和电影评论聚合网站烂番茄。Fandango 和 Flixster 目前仍将作为独立的实体运营，但 Flixster 的最佳功能可能很快就会出现在 Fandango 上，反之亦然。对 Fandango 来说，一个更好的评论评分系统可能正在路上(希望如此)，Flixster，唯一一个没有使用 Fandango 购票系统的电影放映时间应用程序，可能会开始使用它。

[](https://itunes.apple.com/us/app/imdb-movies-tv/id342792525?mt=8)**(免费)绝对是一个可靠的选择( [以及我们对安卓](https://lifehacker.com/the-best-movie-showtimes-app-for-android-5889566) 的挑剔)，如果你是一个想要更多幕后信息的严肃电影迷。除了所有 IMDb 著名的东西之外，你可以按影院或在特定时间范围内查找任何电影的放映时间，并观看即将上映的电影的预告片。你甚至可以通过应用程序购买门票，但这项服务是由 Fandango 提供的。**

**[**movie fone**](https://itunes.apple.com/us/app/movies-by-moviefone-theater/id307840047?mt=8)(免费)的电影是另一种选择，但该应用程序的功能很少，购票系统由 Fandango 提供支持。然而，Moviefone 没有任何广告，并使用 Metacritic 作为其电影评论评分，因此对一些人来说可能是值得的。同样地，[**show times**](https://itunes.apple.com/us/app/showtimes-local-movie-times/id285018181?mt=8)(免费)在功能方面也是相当基本的。该应用程序有横幅广告，门票销售由 Fandango 提供，Showtimes 的电影评论分数只是 Flixster 应用程序中的平均分数。**
