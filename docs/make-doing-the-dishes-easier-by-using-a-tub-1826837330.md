# 使用浴盆让洗碗变得更容易

> 原文：<https://lifehacker.com/make-doing-the-dishes-easier-by-using-a-tub-1826837330>

洗碗——英国人称之为“洗碗”——是在家做饭和自己吃饭最糟糕的部分。尽管我已经开发出了激励自己的小技巧，但很少有什么比简陋的洗碗盆更容易在一个单独的、不可分割的水槽里清洗炊具了。



这个洗碗盆以前被我称为“我的英国朋友伊恩放在他水槽里的奇怪的特百惠塑料制品”，现在已经成为我洗碗过程中不可或缺的一部分。不像其他， [略显多余的英国东西](https://skillet.lifehacker.com/since-americans-dont-use-egg-cups-heres-how-to-eat-a-s-1796854693) ，你的厨房里需要一个(除非你有洗碗机，在这种情况下，就沾沾自喜地坐在那里)。把一个盆放在另一个盆里似乎是多余的，但是，相信我，不是的。恰恰相反，尤其是如果你没有分体式水槽的话。洗碗盆减少了水的浪费*和*释放了你的水槽，这意味着你有*空间*来擦洗和冲洗。如果您仍然很难准确理解它是如何工作的，让我来告诉您这个过程:

1.  从一个干净、空的— [可能亮过的](https://skillet.lifehacker.com/shining-your-sink-will-make-your-life-better-1826391845)—水槽开始。
2.  一旦你知道你要制作餐具，把浴盆放在水槽里，装满热肥皂水。
3.  将产生的脏盘子和器皿放入桶中。
4.  准备好洗碗后，将浴缸从水槽中取出，放在台面上。
5.  漫威:你需要冲洗和擦洗的空间很大，而且你节省的水量惊人，没有填满你的整个水槽。
6.  把盘子从桶里拿出来，冲洗干净，擦洗那些需要的地方。(大多数人在用热肥皂水浸泡后冲洗一下就没事了。)
7.  一旦洗碗盆空了，用洗碗水给一些植物浇水，如果你有的话。
8.  清洗浴缸，并彻底擦干。把它放回水槽。

哪里可以找到洗碗盆？我在一家五金店花了大约 6 美元找到了我的——它的标签上实际上写着“洗碗盆”——但是任何塑料垃圾桶都可以。定期清洗，偶尔用稀释的漂白剂消毒，你的洗碗盆会让你以后的生活更轻松。
