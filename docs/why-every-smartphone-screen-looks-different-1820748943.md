# 为什么每个智能手机屏幕看起来都不一样

> 原文：<https://gizmodo.com/why-every-smartphone-screen-looks-different-1820748943>

推出的 [OnePlus 5T](https://gizmodo.com/the-oneplus-5t-is-very-a-damn-good-phone-for-a-reasonab-1820596179) 有五种颜色校准选项可供选择；Google 已经推至 [给](https://www.xda-developers.com/google-pixel-2-xl-warranty-color-mode/) [像素 2 XL](https://gizmodo.com/pixel-2-review-google-sticks-to-what-it-does-best-1819557944) 增加一个额外的颜色模式；而 [DisplayMate 称 iPhone X](http://www.displaymate.com/iPhoneX_ShootOut_1a.htm) 对色彩的管理比其他任何手机都好。那么，为什么智能手机显示屏上的颜色突然成了一个问题？无论你是在改装自己的手机还是打算买一部新手机，以下是你需要知道的。



我们谈论色彩管理比以往任何时候都多的部分原因是因为智能手机屏幕逐渐从(液晶显示器)转向 [【有机发光二极管】](https://en.wikipedia.org/wiki/OLED) (有机发光二极管)显示器:两者都已经存在了一段时间，但有机发光二极管正变得更加主导，苹果和 LG 今年都做出了转变。

我们不会在这里再次讨论 LCD 与有机发光二极管的利弊— [我们在](http://fieldguide.gizmodo.com/everything-you-need-to-know-about-phone-and-tv-display-1796619322) 之前已经深入讨论过了—但是就色彩管理而言，你需要知道有机发光二极管通常提供更宽的色域(或范围)，这反过来通常意味着在调色板的极端处有更饱和的色彩。

这导致了我们大多数人喜欢智能手机的明亮、生动的颜色需求与实际代表现实世界的颜色需求之间的冲突。我们大多数人都喜欢屏幕弹出，但如果你用手机浏览照片或在网上购买特定颜色的毛衣，自然色通常更受欢迎。

这就是为什么越来越多的手机引入了多种颜色模式，类似于你在电视上处理的模式，但在你了解哪种模式最适合任何给定的场景之前，你需要了解色域和相关的(但技术上非常不同的)色彩空间。

### **色彩空间和色域**

手机、显示器和其他设备上的颜色由两个非常密切相关的概念控制，色彩空间和 [色域](https://en.wikipedia.org/wiki/Gamut) 。事实上，它们是如此的密切相关，以至于经常被认为是同一个东西。

但他们不是！

色彩空间是组织颜色的一种特定方式。你肯定听说过一些，像 sRGB，Adobe RGB，rec。709 和 P3 总督察。色彩空间可以用数学方法开发(如前面提到的那些),也可以更随意，如 Pantone 色彩空间。每个色彩空间都有一个色域，即空间中的实际颜色范围，这就是为什么它们联系如此紧密。

因此，正如一盒蜡笔 将宇宙中的所有颜色减少到某一选择，颜色空间及其色域对图像和显示器也是如此(在许多其他领域也是如此)。更宽的色域意味着更饱和的颜色，因为它的盒子里包括一些更极端颜色的蜡笔。

然而，仅仅因为显示器声称它可以实现特定的色彩空间，并不意味着它可以产生完整的色域。液晶显示器和有机发光二极管显示器可能都能够进行 DCI-P3，但是有机发光二极管，因为它可以产生更多的颜色，有更多的蜡笔供其使用，因此可以更准确地再现色彩空间的颜色。

过去，大多数手机显示器都只有一个固定的工厂设置的色彩空间 [和色域](https://www.noteloop.com/kit/display/color-space/dci-p3/) ，但现在越来越多的手机显示器增加了额外的模式，特别是随着有机发光二极管显示器的趋势。应用程序和移动操作系统同时变得越来越了解色彩空间。

### **手机上的颜色**

那么这对你的手机有什么影响呢？各种因素控制着红色、蓝色和绿色在手机显示屏上的外观，从亮度设置到你所处的日光颜色，再到手机上设置的色彩空间(和色域)。

[Galaxy Note 8](https://gizmodo.com/samsung-galaxy-note-8-review-bigger-badder-but-i-sti-1799759566) 是一款具有多种色彩管理选项的手机——自适应显示、AMOLED 影院、AMOLED 照片和基本——用户可以设置这些选项。如果你选择的手机有这样的选项，你应该在显示设置里找到它们；如果没有，它们可能对您不可用。

这些选项中的每一个都会改变显示器使用的色彩空间和色域。在 Note 8 的情况下，Basic 与无处不在且历史悠久的色彩空间相匹配，最初由微软和惠普开发，目前用于创建大多数消费者内容。与此同时，AMOLED 影院与更新的 [DCI-P3](https://en.wikipedia.org/wiki/DCI-P3) 色彩空间相匹配，该空间现在用于 4K 内容，其色域(即色彩范围)比 sRGB 大 26%。

其他手机也在效仿 OnePlus 5T 让你在五种模式中选择，包括 sRGB 和 DCI-P3，而 iPhone X 也支持 sRGB 和 DCI-P3。苹果就是苹果，切换是在 iOS 中自动完成的，基于 [应用](https://developer.apple.com/ios/human-interface-guidelines/visual-design/color/) 和你正在使用的 iPhone 或 iPad 的需求。

Android Oreo 是第一个支持 sRGB 之外的色彩空间的 Android 版本，但仅适用于明确要求它的应用程序。这可能是你第一次得到 Pixel 2 屏幕 [时觉得它看起来有点柔和](http://www.androidpolice.com/2017/10/25/oreos-color-profiles-work-pixel-2-2-xl/) 的原因之一。它可能有更大的 DCI-P3 色彩空间，但它坚持使用更常见的，如果不太活跃，sRGB。

谷歌正在推出一项更新，让用户为每个应用程序强制使用更宽的 DCI-P3 色域，但警告说这种 [可能会使颜色不太准确](https://productforums.google.com/forum/#!topic/phone-by-google/y1FWSMFcDgA) 。换句话说，一些颜色可能会延伸到一个它们没有配置的颜色空间中(所以当你在手机上买毛衣时要感到疲倦)。与此同时，还有 [奥利奥上色器](https://play.google.com/store/apps/details?id=org.shortfuse.colorizer)——它迫使你的 Android 8.0 设备使用色域更宽的色彩空间，从而在整体上 [更饱和的色彩](https://9to5google.com/2017/10/25/google-pixel-2-app-vibrant-colors/) 。

### **控制显示颜色**

还有一个需要考虑的因素:。这是手机及其显示屏显示颜色的方式，它涉及几个不同的因素，其中只有一个是我们刚才解释的色彩空间设置。

通读 [一篇对智能手机](https://www.anandtech.com/show/10753/the-google-pixel-xl-review/3) 的真正深入的评论，你会看到对其色彩准确性的参考，通常还会提到支持的色彩空间和色域——这基本上是制造商在试图准确地表现其选择的色彩空间中的颜色时是否正确。

因此，当一部白色手机的 [和下一部](https://twitter.com/vladsavov/status/920697103613128704) 看起来不一样时，不仅仅是颜色空间的问题:显示器是如何制作的，以及制造商的选择也是问题。DisplayMate 喜欢全力以赴进行色彩准确度测量，如果你感兴趣，它有 [一些你能想到的最详细的测试](http://www.displaymate.com/iPhoneX_ShootOut_1a.htm) 。

制造商已经引入了各种专有的调整来提高颜色精度，尽管很多时候你不会听说它们或在规格列表中看到它们(当出现问题时，你更有可能注意到)。

正如 DisplayMate 专家指出的 ，真正的责任在于手机制造商创造显示屏，无论屏幕上有什么，色域都可以始终显示真实的颜色，就像 iPhone X 一样。如果它需要在 sRGB 或 DCI-P3 色彩空间中显示某些东西——由内容或应用程序设置——那么它会进行校准，以便能够准确地再现它。

也就是说，正如几年前关于 [一件衣服的颜色](https://sploid.gizmodo.com/some-people-see-this-dress-as-white-gold-while-others-1688330144) 的长期争论所证明的那样，颜色也是一种个人选择:随着时间的推移，我们大多数人只是习惯于我们正在看的任何展示，无论准确与否。色彩空间、色域和校准是值得在你下次购买智能手机之前检查的额外规格——你将会花很多时间看着屏幕。
