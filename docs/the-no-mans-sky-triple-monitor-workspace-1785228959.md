# 无人区三显示器工作区

> 原文:[https://life hacker . com/the-no-mans-sky-triple-monitor-workspace-1785228959](https://lifehacker.com/the-no-mans-sky-triple-monitor-workspace-1785228959)

[洛克](https://www.flickr.com/photos/142453651@N02/) 的工作空间不是设计师的，也不是超级花哨的，它看起来就像一个我们任何人都乐意称之为自己的家庭办公室。但他确实有三个显示器，上面贴着无人区壁纸，一把舒适的椅子，一个小冰箱，还有弹吉他和发挥创造力的空间。这就是我们喜欢它的原因。

Watch

上面的照片是整个设置的广角镜头，你可以看到右下角的迷你葡萄酒冰箱(我不确定它是用来存放葡萄酒还是其他饮料的)，一把额外的椅子和一些桌子空间，供他旁边的客人使用，甚至在他椅子左边的桌子上有更多的座位。我特别喜欢他的显示器后面的背光，以及它如何延伸到他的桌子下面(这是一个经常光线很差的地方)，这样他就可以看到下面的吉他踏板。你可以在他的 Flickr 页面上查看照明和电缆管理的照片，链接如下。

这里有一张桌子表面的近照，以及那个非常光滑(但现已停产)的 Steelseries Merc 隐形键盘:

我还可以想象，你们中的许多人要么喜欢要么讨厌显示器的安装方式(或者墙纸没有整齐地覆盖所有的显示器，我可以理解这一点。)此外，作为一个曾经因为我工作的办公室太热而在桌子上放一台风扇的人，我很欣赏 Vornado Flippi 的存在。除此之外，我还喜欢那里的鼠标线固定器，它可以防止鼠标线与其他东西纠缠在一起，或者在桌面上太松弛。
