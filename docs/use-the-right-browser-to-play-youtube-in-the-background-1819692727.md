# 如何在手机上后台播放 YouTube

> 原文：<https://lifehacker.com/use-the-right-browser-to-play-youtube-in-the-background-1819692727>

我依靠一首歌作为面试前的热身音乐:[2011 年电子游戏*【天际】*](https://www.youtube.com/watch?v=AVy7YPNP_zI) 的主题曲。我唯一找到这首歌正确版本的地方是在 YouTube 上 midi 和管弦乐编曲不适合它。但每当我处于这种高风险的时刻，我不可避免地会避开 YouTube，或者只是锁定我的手机，以及激动人心的*dovahkin 的圣歌！*硬生生切出。



YouTube 知道，我们希望能够在不观看视频的情况下收听视频——无论我们是在进行多任务处理，还是视频只是一首歌曲的载体——这就是为什么 YouTube 中的背景播放仅限于 YouTube Red 订户。但是一位无畏的作家在的边缘，弗拉德·萨沃夫，已经找到了安卓和 iOS 的变通办法。

在这两种情况下，技巧都是使用浏览器而不是 YouTube 应用程序来播放视频。在 Android 上，这在 Chrome 上也能工作。在浏览器中打开 youtube.com，然后在右上角的三点菜单中，选中“请求桌面站点”现在，当你播放视频时，当你离开(或打开另一个 Chrome 窗口)时，它会暂停，但你可以通过通知菜单继续播放。

对于 iOS，你需要下载另一个浏览器: [海豚](https://itunes.apple.com/gb/app/dolphin-web-browser/id452204407?mt=8) 。但是路径是相似的:去海豚的 youtube.com，让你的视频播放。当您退出 Dolphin 时，使用 iOS 控制中心再次开始视频播放。

然后自信地走向你的工作面试，或者屠龙，或者其他什么。
