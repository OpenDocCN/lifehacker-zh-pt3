### ****如何设置****

**RetroArch 比“安装，文件>打开”稍微复杂一点。 [我们有完整的使用回溯](https://lifehacker.com/how-to-set-up-an-all-in-one-retro-game-emulator-with-re-1745863517) 的指南，但这里有一个如何用 bsnes 设置它的快速入门:**

1.  **从其 [下载页面](http://buildbot.libretro.com/stable/) 下载 RetroArch 的最新版本。它是一个 7z 文件，所以你需要安装 [7-Zip](http://www.7-zip.org/) 来解压它。**
2.  **打开 7z 归档文件，将文件解压到您想要的位置(我推荐 C:\Program Files\RetroArch)。**
3.  **双击 RetroArch exe 来启动它。您可以使用箭头键浏览界面，按 X 键选择，或按 Z 键返回。它还支持许多开箱即用的 USB 游戏手柄。**
4.  **要在 RetroArch 中加载模拟器，您需要安装模拟器的“核心”。前往在线更新>核心更新，向下滚动，直到你看到 bs nes-水星核心。如果你有一台非常强大的计算机(CPU 频率高于-3GHz)，试试 bsnes-mercury-accuracy。如果你的电脑功率更低，用 bsnes-mercury-balanced 或 bsnes-mercury-performance 代替。**
5.  **返回到主菜单，并去加载内容>选择文件和检测核心。从硬盘上选择一个 ROM 文件开始播放。**

**你也可以 [调整众多的视频、音频和游戏手柄设置](https://lifehacker.com/how-to-set-up-an-all-in-one-retro-game-emulator-with-re-1745863517) ，但这将让你启动并运行。**

### ****擅长的地方****

**RetroArch 最大的优势是它的设置数量。这对于一些用户来说可能是压倒性的，但它允许您通过启用 GPU 同步来降低输入延迟，或者为经典的 CRT 外观添加着色器，从而创建尽可能好的仿真体验。**

**bsnes 核心是最准确的 snes 模拟器，这意味着在任何游戏中应该很少或没有错误或故障。如果你有运行它的资源，它应该是近乎完美的。**

**此外，虽然 RetroArch 可能很复杂，但它的设置要比桌面版的 bsnes 模拟器 Higan 简单一些。**

### ****不足之处****

**正如我们上面提到的，回溯可能相当复杂。如果您不熟悉 RetroArch，安装内核和调整设置确实会令人困惑，而且由于没有很多关于它的指南，您将进行大量的 googling 搜索来找出它，尤其是如果您将它用于多个仿真器的话。但是这比尝试设置 Higan 要简单得多(特别是如果您已经熟悉了来自其他模拟器的 RetroArch)。**

**其次，bsnes 内核需要一台相当强大的计算机来运行——[精确的代价](http://arstechnica.com/gaming/2011/08/accuracy-takes-power-one-mans-3ghz-quest-to-build-a-perfect-snes-emulator/) ，不幸的是——所以如果你在一台特别弱的机器上，它甚至可能无法很好地运行 bsnes-mercury-performance。在这种情况下，你可能需要满足于 Snes9x。**

### ****比赛****

**[**snes 9x**](http://www.snes9x.com/)**可以说是最流行的 Snes 模拟器，而且理由充分。它很坚固，功能丰富，非常易于使用。它的界面和设置比 RetroArch 简单得多，尽管不如 bsnes core RetroArch 提供的那么精确。如果你有一台不能处理 bsnes 的电脑，或者如果你只是想现在就开始玩*而不是为了达到完美的准确性，Snes9x 是一个很好的选择。(如果您想将所有仿真器放在一个地方，RetroArch 也有 Snes9x 核心可用。)*****

*****[**ZSNES**](http://www.zsnes.com/)**是 SNES 模拟器的另一大亮点，也是从前的热门产品。然而现在，它被认为是旧的、过时的、不准确的——尽管 [也被认为是](http://filthypants.blogspot.com/2015/06/latency-testing.html) 中输入延迟最低的。不幸的是，这带来了噼啪声问题，以及一些游戏中的重大缺陷。[**ZMZ**](http://www.smwcentral.net/?p=section&a=details&id=5681)**是一个基于 ZSNES 接口的仿真器，可以使用 RetroArch 的 libreto 内核。有了它，你可以玩比 ZSNES 更高精度的游戏，但比其他模拟器的输入延迟更低。然而，它与 ZSNES 一样存在声音爆裂的问题，因此大多数人会更好地使用 Snes9x。*********
