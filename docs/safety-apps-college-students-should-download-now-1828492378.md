# 大学生:现在就下载这些安全应用

> 原文：<https://lifehacker.com/safety-apps-college-students-should-download-now-1828492378>

我们不是要告诉你无论去哪里都要带一个朋友，每天晚上回家都要给妈妈打电话，但是如果你是新来的，上夜班或者计划在高峰周外出参加派对，采取一些预防措施不会有什么坏处。

这里有一些免费的应用程序，可以向你的朋友发送位置更新，如果你在校园里走动感到不安全，还可以发出警报。

### bSafe

该应用程序附带了大量功能来帮助您保持安全:

*   你可以让一个朋友在你走路/回家的时候看地图。
*   如果你有危险，你可以向联系人(称为监护人)发送信号，他们可以实时查看你周围发生的事情。
*   如果你找不到你的手机，它有一个声控报警器。
*   你可以把你的准确位置发送给某人。

还有 [更有](https://itunes.apple.com/us/app/bsafe-personal-safety-app/id459709106?mt=8&ign-mpt=uo%3D4) 。

### 六人圈

[这款应用](https://www.circleof6app.com/) 是最受欢迎的安全应用之一，只需轻点两下，你就可以与最多六个人分享你的位置。它的大头针图标让你的朋友知道他们应该来你的位置，而电话图标表明你需要他们中的一个给你打电话。在 [iOS](https://itunes.apple.com/us/app/circle-of-6/id507735256?mt=8) 上都有。

### 一声尖叫

这个应用程序 是声控的，如果它检测到附近有尖叫声，就会请求帮助确定你的确切位置。它说，它不会简单地检测“巨大的噪音和顽皮的尖叫”，而是“表示真正痛苦的紧急喉音尖叫。”

如果你有安卓手机，试试这个版本的 。

### 风筝制作

[Kitestring 是一款短信服务](https://www.kitestring.io/) (这样你就不用下载应用程序了)，当你告诉它时，它会检查你，当你没有回应时，它会提醒你的朋友。

### 您学校的应用程序

许多学校都有列出当地紧急电话号码、公共汽车和班车时刻表(这样你就不会在学习到深夜后独自在外面呆一个小时)等更多信息的应用程序，以帮助你安全回家。如果你的校园提供自己的安全服务，它值得放在你的手机上，并可能补充其他应用程序。
