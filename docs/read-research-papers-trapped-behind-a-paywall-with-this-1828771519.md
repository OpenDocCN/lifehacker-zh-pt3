# 使用 Chrome 扩展阅读被困在付费墙后的研究论文

> 原文：<https://lifehacker.com/read-research-papers-trapped-behind-a-paywall-with-this-1828771519>

大约一半的时间，我去看我在新闻报道中读到的一项研究，实际的研究被困在付费墙后面。

Watch

我明白，大学也有账单，但是花 30-40 美元来获得更多阅读我真正感兴趣但没有实际用途的科学研究的机会，不太现实，至少不是经常。

是一个 [Chrome 扩展](https://chrome.google.com/webstore/detail/unpaywall/iplffkdpngmdjhlpjmppncnlhomiipha?hl=en) 旨在使这个过程变得更简单。

每当你遇到一篇需要付费的论文，这个扩展就会搜索这篇文章，看看作者是否也在任何地方发布了免费版本。

重要的是要注意:这不是让你绕过付费墙或窃取内容。相反，它所做的是看看是否有另一个合法的在线版本，你可以阅读。强调那句话中的“合法”。如果网上有另一份拷贝，你会在屏幕上看到一个绿色标签让你知道。

该扩展将只显示“经出版商自己完全明确授权”出版的文档。据该扩展的创建者称,“大多数学术期刊允许作者在大学和政府的网络服务器上‘自我存档’他们的论文。Unpaywall 帮助用户找到的正是这些合法的自我存档文件。任何未经出版商许可发表的论文都不包括在我们的索引中。”

尤其是现在新学年开始了，这可能是研究论文写作时间的一个有价值的工具。
