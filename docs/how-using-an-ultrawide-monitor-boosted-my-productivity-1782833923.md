# 使用超宽显示器如何提高我的工作效率(也能提高你的工作效率)

> 原文：<https://lifehacker.com/how-using-an-ultrawide-monitor-boosted-my-productivity-1782833923>

几周前我搬家了，所以我不得不停下来重新布置我的工作空间。当我这样做时，我用一个新的 34 英寸超宽 LG 曲面显示器取代了我的双显示器设置。它有它的怪癖，但现在我不确定我是否还能回去。这就是为什么，为什么它对我如此有效。

Watch

[上次我们讨论超宽显示器](https://lifehacker.com/ultrawide-vs-dual-monitors-which-are-better-for-produc-1695967682) 时，桌面上的一个显示器提供两个甚至三个显示器的好处的承诺很有吸引力。您可以获得巨大的水平空间，清理杂乱的电缆，以及足够的空间来保持多个窗口和应用程序打开。理想情况下，这会帮助你更有效率。

虽然单个宽显示屏的吸引力符合我的工作风格(许多窗口和应用程序一个挨着一个打开，相互隐藏，从来没有一个东西是完全全屏的)，但如果你不是这样，或者你一直保持一个窗口最大化，这样的显示器可能不适合你(但这种特殊的显示器有一些技巧，你可能会喜欢，我们稍后会谈到。)不过，如果你更像我，你就有得吃了。

### 从多显示器到超宽显示器是一个简单的转变

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-00PBeWTJbb0&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-00PBeWTJbb0&start=0) 

在我切换之前，我有 [三个显示器用于两台电脑](http://lifehacker.com/how-we-work-2016-alan-henrys-gear-and-productivity-tr-1761082051) 。中间是一个 30 英寸(2560 x 1600)的显示屏，两侧是两个 27 英寸(2560 x 1440)的显示屏。其中一个 27 英寸是我的 Macbook Pro 专用的，但 30 英寸和另一个 27 英寸是连接到我的 Windows PC 上的，我用它来做轻松的工作，做一些设计，玩很多游戏。当 LG 给我提供了一个产品线，让我试用他们的全新 [LG 34UC98 34 英寸 21:9 曲面 IPS LED 显示屏](http://www.lg.com/us/monitors/lg-34UC98-W-ultrawide-monitor) (仅次于 [【亚马逊](https://www.amazon.com/dp/B019O78DPS?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-using-an-ultrawide-monitor-boosted-my-productivity-1782833923&asc_source=&linkCode=ogi&psc=1&smid=A11EJNOC51XDV2&tag=kinjalifehackerlink-20&th=1) 的 950 美元)，我认为这将是一个简单的过渡。

尽管如此，我还是要说——我确实喜欢这个面板的*外观*,我桌子上更有条理的轮廓也是我所欣赏的。从一堆电源、USB 和视频电缆到一个电源和一个视频是一个很好的好处。

至于实际的过渡，我真的很惊讶从多个显示器回到单个显示器是多么容易，但我认为水平工作空间——3440 x 1440 native——是造成这种情况的原因。回去工作很容易，我错过的唯一真正的怪癖或功能是在显示器之间移动或捕捉内部挡板的能力，或者在玩游戏时从我的主设备看向我的辅助设备以阅读网页或查看电子邮件，或者在我玩游戏时聊天。不过，这与其说是一个超大范围的问题，不如说是一个单一显示器的问题。

### 你的工作方式将决定超宽是否值得

当谈到工作时，显示器本身实际上很好用，但这主要是因为我的工作方式。我会在我正在写的那个窗口后面的一个窗口中打开研究，一个视频或音乐播放器跑到我能看到它的一侧，一个聊天窗口在其他地方打开，或者 Twitter 或 Slack 躲在后台。当你拥有像这样的显示器给你的巨大水平空间时，你可以让它们一个接一个地打开，同时关注所有的东西。让谷歌文档出现在一个窗口中，一个参考视频出现在它右边的一个小窗口中，Tweetdeck 停靠在一边，*和* Slack 或者一篇文章在角落里打开并不难。当我不得不浏览一个巨大的电子表格时，有这么大的水平空间是很漂亮的。

虽然我并不是一个真正的“不受干扰，任何时候都是单一窗口”的用户，但这样的显示对这些人来说并不理想，除非你使用的应用程序可以利用所有的空间。例如，如果你是一个有创造力的专业人士或工程师，你可以用终端窗口和文本编辑器填充同样的空间，尽管更多的垂直空间可能真的对你有好处。如果你是一个视频编辑或摄影师，你可以很容易地用一个预览窗口，多个视频和音频编辑轨道，或者 Photoshop 窗口，动作和工具栏来填充所有的水平空间。

另一方面，我看到人们抱怨显示器每年都变得越来越宽，而大多数网站仍然将所有有用的内容都放在一列中，两边留有空白——这在 3440 x 1440 的全屏下看起来很傻。对于这些人来说，这种特殊的显示器能够模拟两个甚至三个更小的显示器，无论是垂直还是水平布局，因此您可以在一个大而宽的面板上获得多显示器的感觉。我尝试了几次，当我回到自己管理整个显示器时，我可以看到这些配置如何简化从两个显示器到一个大显示器的过渡，或者给那些“我宁愿有更多的垂直空间来阅读”的人一些东西。如果更多的超宽制造商在其显示器中包含 LG 的功能，那将是一件好事。

不过，归根结底，像这样的超宽屏幕是否会改变你的工作方式，很大程度上取决于你是否能利用屏幕空间。如果这对你来说太多了，或者你是一个单一任务者，使用的工具会浪费所有的空间，那就不值得了。不过，对于其他人来说，这是值得考虑的——增加的房地产对我来说是一个巨大的好处，帮助我不再瞎折腾，工作速度也快了很多。

### 游戏和电影绝对精彩，但也有一些怪癖

当您投资购买超宽显示器时，不可避免的事实是，目前您的大多数游戏和视频可能是 16:9，可能不完全支持 21:9 显示器。这意味着，如果你全屏观看 YouTube 视频或下载的电影，或者启动你最喜欢的视频游戏，你会在屏幕两侧看到黑色的大条(称为“边框”)，限制你在 1440 像素高的情况下获得最大支持的 16:9 分辨率。

当然，肯定有支持 21:9 分辨率的华丽游戏和视频，当你找到它们时，你会激动不已。过一段时间你也会习惯于这种邮筒，但是当你看不到它的时候，你会觉得很神奇。如果你像我一样喜欢视频游戏，访问 [宽屏游戏论坛](http://www.wsgf.org/) 和 [查看他们的主游戏列表](http://www.wsgf.org/mgl) 看看你最喜欢的游戏支持哪种 21:9 显示。您还可以查看哪些游戏允许您自定义他们的视野(FOV)设置，以便在您玩游戏时为您提供宽广的电影般的外观，当然，哪些游戏可以手动调整，即使超宽游戏不受官方支持。那里的社区很棒，充满了非常聪明的人。

LG 为这款特定机型配备了通过 DisplayPort 的 [AMD FreeSync](http://www.amd.com/en-us/innovations/software-technologies/technologies-gaming/freesync) ，它承诺通过支持动态帧率来平滑帧率并消除游戏中的屏幕撕裂。如果所有这些都超出了你的理解范围，那么它旨在让你的游戏——尤其是在大型显示器上——[运行更流畅，看起来更好](https://en.wikipedia.org/wiki/AMD_FreeSync) 。当然，你需要有一个支持这一点的 AMD 显卡(我正好有一个)，你必须确保你的游戏配置符合 FreeSync 的帧率要求，但一旦符合，这是一个很好的体验，特别是如果你的游戏真的可以利用整个面板。显然，不要为了这个跑出去买一个新的 GPU，但这是一个不错的额外津贴。

### 我可能会回到多显示器...只是现在不行

当我在 LG 测试的时间结束时，我可能会回到我的旧设置，但我正在认真考虑加入 21:9 的行列。我不太愿意直截了当地说是*让我*更有效率，因为我不认为硬件真的能*让*你做什么。然而，当你找到一个适合你的工作(和娱乐)风格的时候，它可能会让你自己的能力大放异彩。对我来说，那当然是真的。

虽然我的 30 英寸(2560 x 1600)和 27 英寸(2560 x 1440)在技术上比这个 34 英寸(3440 x 1440)多了*个像素，但它*感觉*更大了，好像我可以用我拥有的空间做更多事情。一部分可能是缩放，一部分可能是简单性对您工作的影响。从工作的角度来看，这个实验取得了巨大的成功，但是你的经历可能会因你所做的工作而有所不同。从游戏和媒体的角度来看，这也是一个成功，很大程度上是因为当我遇到 pillarboxing 时，我可以越过它，当我没有遇到它时，这种体验——特别是在游戏时——是不可思议的。*

所以，是的，我可能会回到多重离散显示。仅仅...不是马上。我再等一会儿。别问我多久。
