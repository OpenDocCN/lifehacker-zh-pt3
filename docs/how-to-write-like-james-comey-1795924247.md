# 如何像詹姆斯·科米一样写作

> 原文:[https://life hacker . com/how-to-write-like-James-comey-1795924247](https://lifehacker.com/how-to-write-like-james-comey-1795924247)

前联邦调查局局长詹姆斯·科米的证词于昨天在他今天的听证会前以书面形式发布。这是对他与总统的几次谈话的实事求是的叙述，所以你不会指望这是一个有趣的阅读。但制作精良:细节恰到好处，场景设置恰到好处。需要讲故事吗？像詹姆斯·科米一样使用这些提示来告诉它。

### 保持你的介绍简短

科米可能花了数百个字来描述他为什么作证，以及他如何准备他的陈述。没人在乎。所以他保持简短和甜蜜:

告诉你你需要知道的一切，对吗？当人们在读(或听)你的介绍时，他们渴望得到好的东西，所以这不是解释背景信息的好时机。科米*有背景吗——例如，他什么时候和为什么开始记录对话——但他在讲述时会把它放在故事的相关点上。*

### *场景化写作*

*对话不仅仅是对话。讲得好，是一个场景。编辑兼写作教练 [杰克·哈特](http://www.press.uchicago.edu/books/hart/About_Jack_Hart.html) 在 *Storycraft* 中写道:“当你开始构思一个非虚构的故事时，把自己想象成一个剧作家也无妨。”。“毕竟，你必须创造一个舞台...你可以把它和人物联系起来。然后，打个响指，这些角色就可以呼吸、移动、行动了。”*

*科米描述了五次对话，并为每一次对话设定了场景。我们知道它们发生在哪里，何时发生，以及房间里的气氛。以下是他如何为 1 月 27 日的晚宴设置场景:*

*重点不仅仅是娱乐——细节可能最终对将这个故事与其他事件和人物联系起来至关重要。特朗普有单独用晚餐给人惊喜的习惯吗？海军乘务员有他们自己的故事要讲吗？在小说中，这些细节可能会与故事的其他部分联系起来。现实生活更混乱，但我从阅读科米的文件中感觉到，他正在为未来的场景奠定基础，这些场景可能会出现，也可能不会出现。请看椭圆形办公室会议的细节:* 

*落地钟旁边的门有什么意义？也许是为了唤起其他人的回忆:*哦，是的，我记得在那扇门外的人群中等待*。也许他加入这个细节还有其他原因，我们还不知道。我只知道，《落地钟旁的门》听起来像是一本我绝对会读的书。* 

### *扩展和收缩时间*

*科米描述的事件持续了几个月，其中包括一定是冗长的会议、简报和晚宴。科米跳过了令人生厌和/或机密的部分，但放慢了速度，仔细地一点一点播放重要的内容。*

*在那次绿色房间晚餐中，科米描述了谈话的要点，而不是一行一行地渲染；内容很重要，但他不需要告诉我们每一个字。随着问题变得越来越不恰当——科米说他感到“不安”——讲述的速度变慢了。然后我们到了这个时刻:*

*他不说是，也不说不是，甚至不点头也不做鬼脸。这很重要，因为委员会想知道他是如何回答这个问题的。但它也传达了当时的紧张气氛。它表明，唐纳德·特朗普，一个如此渴望交谈以至于不断打断自己的人，让对话停止了。* 

### *写作要有目的*

*罗伊·彼得·克拉克在他的 [快速 50 种写作工具](http://www.poynter.org/2006/fifty-writing-tools-quick-list/76067/) 中建议:“围绕一个关键问题展开你的工作”。在这种情况下，悬而未决的问题是*总统是否参与了妨碍司法公正？*这不是科米要回答的问题；法律上这是 [让别人决定](http://www.cnn.com/2017/06/06/politics/comey-testimony-refute-trump-russian-investigation/index.html) 。但他似乎选择了细节，设置了场景，为回答这个问题提供了必要的信息。*

*在一部小说或一篇文章中，我们可能期望作者帮助我们理解关键问题的答案。克拉克写道:“帮助读者闭合意义圈”。这在这里不会发生，在任何文学形式中也不合适。举例来说，有时候一首诗的目的是给读者一些思考的东西。所以考虑一下你的故事的结局是什么，是否会出现在页面上。任何时候你认为读者的兴趣可能会减退，试着用“俄罗斯的妓女”这个短语。*
