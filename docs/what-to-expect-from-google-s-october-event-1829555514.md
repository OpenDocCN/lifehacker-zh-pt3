10 月 9 日星期二，谷歌将在纽约市举行年度硬件活动。在这些活动中，我们听到了这家科技巨头正在准备的所有重大设备和服务新发布。

## 期待什么

虽然我们不能确定谷歌将展示什么，但过去的展示涵盖了从新的智能手机和平板电脑到智能家电的所有内容。然而，根据传闻和最近的声明，我们对周二的新闻发布会有一个大致的了解。

**官方 Pixel 3 和 Pixel 3 XL 公布**

几乎显而易见的是，谷歌已经为即将到来的活动最终确定了其下一代旗舰智能手机 Pixel 3 和 Pixel 3 XL 的展示。已经有许多泄漏，似乎没有什么关于所谓的新手机留下来透露，但我们希望在周二的活动中获得真正的交易公告。

如果泄露的规格是可信的，Pixel 3 和 Pixel 3 XL 都将运行骁龙 845 芯片组和 Android 的修改版本。

在物理设计方面，这两款设备都有望保留过去 Pixel 手机的单摄像头设计。Pixel XL 据说有 5.3 英寸的显示屏和更小的边框，而 3 XL 将有 6.2 英寸的边对边屏幕，尽管据报道它在顶部和底部边框上也有一个“凹口”和大“下巴”，这让一些人懊恼。

最后，一些传言甚至暗示升级版 Pixel Bud 无线耳机将与 Pixel 3 和 3 XL 一起发布

我们怀疑这将是该活动的重大宣布之一，所以期待获得所有信息，包括价格和发布日期，以及确认它将由新推出的谷歌维修中心覆盖。

**Pixel Slate 2 合 1 和 Pixelbook 2**

Pixelbook 是谷歌对苹果的 iPad 和 Macbook 以及微软的 Surface 设备的回应。最初的 Pixelbook 在去年 10 月的活动中推出，这使得更新硬件的可能性很大。预计 Pixelbook 2 的潜在升级包括 4K 显示屏、新处理器和更薄边框的更新设计。

另一个传闻中的平板电脑是谷歌的 2 合 1 平板电脑 Pixel Slate。虽然 Pixelbook 可以兼具笔记本电脑和平板电脑的双重功能，但它的键盘是不可拆卸的，它主要专注于替代笔记本电脑。另一方面，传闻中的 Pixel Slate 可能会配备可拆卸键盘，以实现更传统的平板电脑功能。然而，关于 Pixel Slate 最酷的传言是另一个二合一功能:一些泄密者建议 Slate 可以在同一台机器上运行 Chrome OS 和 Windows 10。

**Chromecast 3**

看起来新的 Chromecast 也即将推出。这可能有点令人惊讶，但据报道至少有一个人已经在百思买购买了一个，所以除非 Reddit 用户 LitheBeep 真的擅长 Photoshop，否则它似乎是一个真实的东西。

谷歌最近更新了 Chromecast，它有 4K 支持的 Chromecast Ultra 和纯音频的 Chromecast Audio，所以很难准确指出 Chromecast 3 可能包括哪些新功能或规格。也就是说，我们知道 Chromecast 3 至少对 Chromecast 的外观做了一些改变，有更厚的机身和更新的标志。无论发生什么变化，都可能在周二被讨论。

**项目流支持？**

我们很可能会听到更多关于谷歌即将测试的流媒体服务 Project Stream 的消息，特别是如果这项服务将出现在任何新发布的设备上。虽然 Project Stream 目前的重点是通过 Chrome 浏览器进行基于云的游戏，但谷歌仍有可能将这项服务用于游戏以外的领域。

**Pixel Stand & Google Home hub 智能显示屏**

一系列潜在的新 Pixel 设备可能会主导谣言周期，但如果没有对其智能家居和物联网(OIT)计划、Google Home 和相关设备的关注，这就不是谷歌的活动。有传言称，一种新的 Pixel 手机配件 Pixel Stand 将把你的手机集成到你的智能家庭网络中。据称，该设备是一个无线充电坞站，可以让你在对接时使用手机上的谷歌助手，即使屏幕被锁定。

同样，谷歌 Home Hub 智能显示器应该，嗯，正如它听起来的那样:一个集中的中枢，将通过谷歌助手对连接的谷歌 Home 设备进行完全控制。据报道，该屏幕显示天气、突发新闻和设备状态等重要信息，并作为控制其他设备的触摸界面。
