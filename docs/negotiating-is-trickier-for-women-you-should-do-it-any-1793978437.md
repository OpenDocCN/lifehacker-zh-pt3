# 对女性来说，谈判更棘手。无论如何你应该做它。

> 原文:[https://life hacker . com/negotiating-is-trickier-for-women-you-should-do-it-any-1793978437](https://lifehacker.com/negotiating-is-trickier-for-women-you-should-do-it-any-1793978437)

这是我第一次与新的潜在客户会面，他们抛出了一个数字。这是一个不错的数字，但我最近发誓要谈判更多。所以我抛出了一个更高的数字，然后在尴尬的沉默中屏住呼吸。我讨厌充满焦虑、令人紧张的谈判过程，但这就是为什么我已经学会接受它，尽管事实上。

### **为什么谈判对女性来说更棘手**

虽然自 1963 年以来，工资差距已经缩小了不少，那一年,同酬法案被签署成为法律，但女性、 [尤其是有色人种女性](https://www.americanprogress.org/issues/women/news/2017/04/03/429810/value-equal-pay-u-s-economy/) 的收入仍然低于男性。自 60 年代以来，我们已经走过了漫长的道路，但仍有许多工作要做，特别是考虑到有些人 [实际上认为差距是一个神话](https://www.vice.com/en_us/article/calling-bullshit-on-the-men-who-think-the-pay-gap-is-a-myth) 。

同酬问题很复杂，工资差距存在的原因也很复杂。例如，一些人认为女性工资低是因为她们不谈判。她们说，女人就是不问。有些统计 [显示那是真的](http://www.payscale.com/career-news/2015/05/millennials-and-women-dont-negotiate-salary-heres-why-thats-important)[其他统计显示那是假的](http://www2.warwick.ac.uk/fac/soc/economics/news/2016/9/new_study_suggests_women_do_ask_for_pay_rises_but_dont_get_them/) 。问题是，女人是否谈判的问题并不像我们谈判时会发生什么的问题那样重要。

[据《哈佛商业评论》](https://hbr.org/2014/06/why-women-dont-negotiate-their-job-offers) 报道，情况是这样的:

> 在反复的研究中，人们发现，女性为争取更高的工资而付出的社会代价要高于男性。

另一件事情发生了，根据卡斯商学院、华威大学和威斯康星大学的一项研究:

> ...女性要求加薪的频率和男性一样高，但男性提出加薪要求的可能性要高出 25%。

首先，女性在谈判时不仅被认为比男性更消极——男性被认为是自信和强大的，而女性被认为是难以相处的——她们得到自己想要的东西的次数也比男性少。

突然间,“神话”似乎并没有被终结。

### 薪酬过低的女性的特征

我一直对工资差距问题有着深远的理解，但直到我读了芭芭拉·斯坦尼的《六位数女性的秘密 》一书，才真正在个人层面上理解了这个问题。我最初对这个书名持怀疑态度，但这本书提供了一些关于女性如何理财的重要见解。

斯坦尼为她的书采访了 150 名女性，发现那些收入较低的女性有一些共同的特点，其中最主要的是她们避免谈判。以下是长期收入过低的女性的一些特征 :

*   他们对低工资有很高的容忍度:他们知道他们可能会赚更多的钱，但他们接受更低的工资。他们甚至可能试图证明它是正当的。
*   他们低估了自己的价值:他们认为自己永远不会得到高薪。例如，我总是告诉自己，“作家不应该赚很多钱，所以这种垃圾工资没问题。我不是为了钱。”
*   他们愿意免费工作:免费工作 有一个 [的理由，但初学者不会对此多想。](http://lifehacker.com/when-and-if-you-should-ever-work-for-free-509683212)
*   他们讨厌谈判:他们不愿意要更多的钱，他们害怕如果要了会发生什么。例如，我总是害怕如果我要求更多，另一方会被冒犯并完全取消报价。
*   他们认为贫穷是高尚的:他们认为贫穷能让你获得某种道德感，金钱是邪恶的。这深深地触动了我。我在贫困中长大，一直认为富人都很贪婪。我甚至写了一篇关于我如何“浪漫化贫穷”的文章

当然，这些特征显然也适用于男性，但斯坦尼特意采访了女性来编制这份清单。这也很容易理解为什么女性在职业上低估了自己。你所要做的就是接受那些打破神话的论点，其中最糟糕的是诉诸于这样的论点，即女人就是不如男人聪明或有资格。

撇开统计数据不谈，如果你想赚更多，谈判是这个过程中必不可少的一部分。就像我说的，我讨厌谈判。我没有对抗或主张。我一直讨厌难相处的想法。

但在阅读了关于这个问题的文章，尤其是 Stanny 的研究后，我开始把谈判不仅仅视为增加自己收入潜力的一种方式，还视为抗议一个更大的系统性问题的微观方式。换句话说，谈判对女性来说更棘手；我开始觉得我应该进行更多的谈判，因为这是事实，而不仅仅是不顾事实。

### 我是如何适应谈判的

读完 Stanny 的书，我发誓要开始要求更多。最初的谈判实验变成了信心和授权的练习。我协商了更高的自由职业者工资。我通过谈判摆脱了预算削减。我甚至走出了一家汽车经销店，结果那天晚些时候他们不停地给我打电话，要求接受我的提议。

如果你了解我，你会惊讶的。我一直是那个说话温和、讨人喜欢、讨厌对抗的女孩。但这是问题的一部分。克服我对谈判的恐惧意味着我必须 [克服我对发言的恐惧](https://lifehacker.com/how-i-got-over-my-fear-of-confrontation-and-learned-to-1699617728) 。这里有一些我牢记在心的 [要点和提示](http://twocents.lifehacker.com/how-to-get-over-your-fear-of-negotiating-and-ask-for-mo-1760907136) 来克服我的恐惧。

你最大的恐惧可能是被拒绝，这也没那么糟糕

在我的谈判过程中，最有力量的时刻之一就是被拒绝了。我向一个大客户要更多的钱，他们说，“我很抱歉，但这不在我们的预算之内。希望我们能在未来实现它。”拒绝一直是我谈判中最大的恐惧，也没那么糟糕。另外，即使我没有得到我想要的，我仍然成功地拓展了我的收入潜力，这感觉很好。

**在赌注低的时候练习**

你谈判的次数越多，你就越能适应。这就是为什么在低压环境下练习会有帮助。例如，我会为我的手机或有线电视账单讨价还价: *AT & T 有更好的促销价格，你还能提供什么？*或者，我会在家具店讨价还价:*你会考虑把两张凳子都换成 175 美元吗？我常常是成功的。这是习惯提问的一个简单方法，对我来说这是最难的部分。另外，它给了我信心，让我作为谈判者被认真对待。*

**把谈判想成** [**【联合解决问题】**](https://lifehacker.com/make-negotiating-easier-by-approaching-it-as-joint-pro-1697401167)

我不再认为谈判是一种对抗，而开始认为它是一种相互达成的协议。 [正如《哈佛商业评论》指出的](https://lifehacker.com/make-negotiating-easier-by-approaching-it-as-joint-pro-1697401167) ，将谈判视为对抗实际上确保了它将是对抗的。相反，他们建议将它视为共同解决问题:“手头的关键问题是什么，我的利益和他们的利益是什么，以及满足这些不同利益的一些不同的可能选项是什么？”

这不仅帮助我克服了“我讨厌对抗”的问题，还增强了我的自信。我没有关注我能从这种情况中得到什么，而是考虑我能带来什么样的价值。谈判与贪婪无关。这是关于努力发挥我的潜力。

**研究就是一切**

无论是有线电视账单、你的薪水，还是一辆新车，研究对于任何谈判都是至关重要的。

当你协商加薪的时候，在像 [Glassdoor](https://www.glassdoor.com/) 这样的网站上研究薪资信息。拜访经销商时，在 [上研究车辆价值。当你讨价还价的时候，研究竞争对手的促销价格。这些只是开始，但是任何有谈判经验的人都会告诉你:研究就是一切。你不仅会在谈判中准备得更充分，还会更有信心知道自己在谈什么。](https://www.kbb.com/)

我对鼓励女性以不同方式谈判以避免雇主偏见的建议感到非常困惑。我的朋友艾琳·洛瑞(Erin Lowry)也是一位财经作家(也是即将出版的新书《打破千年的 T2》的作者)，她告诉我:

> 实际上，我曾经和一位律师交谈过，他向我承认，他没有给一位女性下属像她的一些同事那样大幅度的加薪，因为他知道她的丈夫收入如此之高。

发生这种事很糟糕，但它确实发生了。出于这个原因，洛瑞说，女性可能会考虑保守个人信息，如订婚或育儿计划。许多其他成功女性都同意，女性必须以不同于男性的方式思考谈判问题 。

然而，除了偏见之外，我相信 90%的成功谈判只不过是 [鼓起勇气问](http://www.thewildwong.com/how-to-negotiate/) 。你可以从书中学到所有的技巧和诀窍，但是大部分的工作(和回报)都发生在你开口说“你能不能一年挣 55000 美元？”

在我和那个客户的谈话结束时，他们同意了更高的价格，我敢肯定不是什么谈判专家。我很紧张，有点口吃。也许我问错了，但我问了。谁知道那个客户现在对我的看法会不会不同？也许他们真的认为我很贪婪。也许他们*做*觉得我很难。事实是，我问。不管成功与否，谈判变得更加轻松的感觉很好。对“难相处”感到更舒服的感觉更好。
