# 通过预订连通房而不是套房，节省大量酒店费用

> 原文：<https://lifehacker.com/save-big-at-hotels-by-booking-connecting-rooms-instead-1826294783>

一个多房间的酒店套房为你的聚会提供了一个公共区域，但也允许每个人都有自己房间的隐私。它们非常适合家庭聚会、婚礼和单身派对。问题是，这些类型的套房通常贵得惊人。以下是如何以更低的价格获得套房般的体验。



诀窍是预订酒店业所说的“连通套房”或“连通房间”，而不是传统的套房。这些是酒店房间配置，其中多个卧室通过公共区域、小走廊或它们之间的门相互连接。相连套房的每个房间都有一个通向主走廊的门，这样客人可以进出而不会打扰到他们相连的房间。此外，连通客房保证每个房间都有自己的私人浴室，而有些套房可能只有一个。

在分析了专门预订酒店套房的网站的数据后，Priceonomics 发现 [连通房的价格通常是传统两居室套房](https://priceonomics.com/how-to-hack-the-price-of-a-hotel-suite/) 的一半。例如，在美国一个主要城市的两居室套房中度过一个周末的平均费用约为 2，665 美元。另一方面，一间连通房的平均价格只有 1484 美元左右——同样的住宿价格要少 1181 美元。在一些地区，比如纽约市，通过预订连通房而不是套房，你可以节省高达 5409 美元。当然，如果你愿意花大约 1300 美元(不包括平台和清洁费)，你也可以去一个有两个卧室的 Airbnb 过周末。但是这样你就错过了酒店提供的一些便利设施，尤其是如果你住在拉斯维加斯这样的地方，所有的活动都集中在酒店里。

那么如何预订这些神奇又省钱的连通房呢？打电话到酒店问问。几乎每个酒店都有，但可能没有列在第三方预订网站上，所以直接向酒店前台查询。也就是说，一定要提前预定一个连通的房间。出于显而易见的原因，这些类型的房间很受欢迎，所以如果你想省钱，请不要犹豫，尽早预订。
