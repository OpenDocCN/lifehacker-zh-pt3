# 用药错误正在增加——以下是如何做到安全

> 原文：<https://lifehacker.com/medication-errors-are-on-the-rise-heres-how-to-be-safe-1796927310>

服用错误的药片，服用过多，混淆处方——一项新的研究表明，人们在药物治疗方面犯的错误比以往任何时候都多。



[根据美国食品和药物管理局](https://www.fda.gov/Drugs/DrugSafety/MedicationErrors/ucm080629.htm) 的数据，美国每年约有 130 万人因用药错误而受伤，但其中一些病例远比其他病例严重。这项名为 [的研究发表在期刊*临床毒理学*](http://www.tandfonline.com/doi/full/10.1080/15563650.2017.1337908) 上，研究人员查看了美国各地毒物控制中心收集的数据，然后统计了所有发生在医疗机构以外的严重用药错误。研究人员发现，2000 年至 2012 年间，威胁生命的病例数量翻了一番，仅 2012 年就有 6855 例，其中约三分之一的病例导致住院。更糟糕的是，其中很大一部分是很容易预防的。

请记住，这个数据子集仅来自毒物控制中心，因此它不能很好地说明这一趋势的严重性。美国毒物控制中心协会前主席杰伊·绍本[向 NPR](http://www.npr.org/sections/health-shots/2017/07/12/536519077/took-the-wrong-medicine-by-mistake-study-finds-such-errors-are-on-the-rise) 解释说，当出现用药错误时，并不是每个人都会打电话给毒物控制中心。该研究的第一作者蒋曦儿·霍奇斯说，这些数字可能低估了真实的数字，在家里发生的危险的用药错误可能被低估了。

这些错误看起来像什么？可能是有人服用了错误剂量的药物。或者有人不小心吃错了药。其他错误包括没有等待足够长的时间来服用第二剂，药剂师以错误的浓度配药，以及孩子服用他人的药物(6 至 12 岁的孩子最有可能服用错误的药物)。所有简单的人为错误都可能让某人付出生命代价。

霍奇斯是俄亥俄州哥伦布市[全国儿童医院](http://www.tandfonline.com/doi/full/10.1080/15563650.2017.1337908) [的研究科学家，他建议在你的药盒上安装儿童锁，并把它们放在够不到、看不见的地方。如果一个孩子应该服用一些药物，总是帮助他们，这样你就知道他们没有吃错药。你也可以写下服药或给药的时间和剂量，避免使用普通的厨房勺子来测量药物。](http://www.nationwidechildrens.org/)
