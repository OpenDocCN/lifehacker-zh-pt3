# 当一只陌生的狗靠近他们时，教孩子们“成为一棵树”

> 原文：<https://lifehacker.com/teach-kids-to-be-a-tree-when-an-unknown-dog-comes-near-1820253886>

只要孩子 [恰当地这样做(规则 1:首先接近主人)](https://lifehacker.com/how-to-introduce-yourself-to-a-dog-1794744432#_ga=2.173546603.919070749.1509938319-21963309.1506530761) ，他们与新狗接触是很好也很甜蜜的，但有时狗可能会太喜欢甚至侵犯他们。 [Doggone Safe](https://doggonesafe.com/) ，一个专注于预防狗咬伤的组织，教导孩子们如果有陌生狗靠近他们，就“做一棵树”。为什么？树对狗来说很无聊。



它是这样工作的。当狗靠近时，孩子应该:

1.  停下来。
2.  把他们的“树枝”(手)折在身前。
3.  看他们的“根”生长(看他们的脚)。
4.  在他们的头上数他们的呼吸，直到救援到来，或者狗走开。

当教孩子们如何成为一棵树时，让他们 [在一个低压力的环境](https://clickertraining.com/node/3535) 中练习，用一个填充动物玩具或他们真正的宠物，这很重要。ASPCA 还有其他减少被狗咬伤风险的 [建议，强调理解狗的肢体语言是关键。](https://www.aspca.org/pet-care/dog-care/dog-bite-prevention)
