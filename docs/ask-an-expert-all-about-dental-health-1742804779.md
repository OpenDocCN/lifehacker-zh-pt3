# 请教专家:关于牙齿健康的一切

> 原文：<https://lifehacker.com/ask-an-expert-all-about-dental-health-1742804779>

来跟马克·伯亨博士打个招呼吧。B 博士从一个更广阔的视角来看待口腔健康，而不仅仅是牙齿护理——毕竟你的口腔是你身体的一部分，但它经常被当作一个独立的生态系统来讨论。



当然，我们都试图保护我们的牙齿和口腔健康，但我们中的许多人都有点懒——我真的需要每天用牙线剔牙吗？我们能做些什么来保持牙齿的最佳状态呢？不注意口腔卫生会对你身体的其他部分产生什么影响？

B 医生有 30 多年的牙医经验，在他的网站上写了关于口腔健康的文章， [【向牙医提问】](http://askthedentist.com/) ，以及他最近的书，[《8 小时睡眠悖论》](http://www.amazon.com/gp/product/B015DIQBZ6/?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/ask-an-expert-all-about-dental-health-1742804779&asc_source=&tag=kinjalifehackerlink-20) 。接下来的一个小时他会在这里，所以在下面的评论里问一个尖锐的问题吧！
