大约 20 多年后，Windows 95 感觉并不十分酷——与微软的视频可能让你相信的相反——但由于一个聪明的小黑客，你可以享受像 MS-DOS 提示符(现在的命令提示符)、电话拨号器(现在的 Skype)和 Freecell(现在的 [一个奇怪的应用](https://www.microsoft.com/en-us/p/freecell-collection-free/9wzdncrdn9kt) 这样的应用程序的怀旧之情。

Slack 工程师费利克斯·里瑟伯格最近使用 [电子](https://lifehacker.com/turn-your-favorite-website-into-a-desktop-app-1827807804) 框架创建了一个 Windows 95 操作系统的桌面应用程序。一旦你从他的 GitHub 下载了上述应用 [，双击就可以启动一个版本的 Windows 95。](https://github.com/felixrieseberg/windows95)

我说“一个版本”，因为它不是完整的桌面体验。当然，这是 Windows 95，但有些组件不工作:Internet Explorer 什么都不做，任何其他需要任何网络访问的应用程序也是如此。没有声音，所以你不会听到。不要想点击 MS-DOS 提示应用程序，除非你想让你的系统崩溃:

尽管如此，把 Windows 95 作为一个应用程序来玩还是很有趣的。画一幅画。扫一些地雷。还记得 Windows 95(以及你最喜欢的应用程序或游戏)曾经耗尽你系统的所有资源吗？现在，Windows 95 对于现代计算机来说，就像一个带有几个打开的标签的网络浏览器一样费力。时代变了。
