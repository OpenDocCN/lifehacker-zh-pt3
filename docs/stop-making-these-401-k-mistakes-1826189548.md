# 停止犯这些 401(k)错误

> 原文:[https://life hacker . com/stop-making-these-401-k-errors-1826189548](https://lifehacker.com/stop-making-these-401-k-mistakes-1826189548)

如果你已经承诺为你的 401(k)计划做出贡献——太好了！在为健康退休做准备方面，你走在了很多人的前面。

Watch

但是设置你的自动缴费是比较容易的部分。正如米奇·塔奇曼在 [市场观察](https://www.marketwatch.com/story/all-the-ways-you-can-mess-up-your-401k-even-if-you-max-out-your-contributions-2018-05-18) 中所写的，有很多方式会损害你的退休储蓄，而这些方式是你想要避免的。有些是显而易见的(收费！)，其他的就不那么了:

### 不投资

好，那么你已经开始捐款了。太好了！现在你需要真正挑选你的投资，否则你的现金只是，嗯，现金。有很多选择，但低成本的指数基金是必由之路。

### 没有得到雇主的完全匹配

许多公司提供相应的激励计划，让你向自己的退休账户缴款。如果你的贡献达不到那个数额——比如说，你工资的 6%——那么你就是在让你的雇主为你的工作支付更少的工资。

### 前期投入

如果你能为自己的 401(k)计划贡献最大金额——18500 美元，或者 24500 美元(如果你 50 岁或以上)——那么你应该尽早这样做，以最大化你的收益。但这意味着你可能会错过雇主的贡献，因为一些公司将他们的支付分散在全年。

这不是每个雇主的问题。按 [芬拉](https://www.finra.org/investors/too-fast-match-401k-misstep-could-hurt-retirement-savings) :

> 一些 401(k)计划包括一个功能，允许工人获得最大的雇主匹配，即使他们已经在年初完成了他们的 401(k)计划。这项名为 true up 的功能允许员工获得雇主匹配，否则他们会因为 401(k)前期加载或因为他们在一年中不均匀地分散 401(k)缴款而错过。

正如 FINRA 指出的，如果你打算在年底前离开你的公司，不管你是否会错过完整的比赛，前期投资可能是有意义的。

### 支付高额费用

你知道我们会来的。我们已经涵盖了 [这一大堆](https://twocents.lifehacker.com/401-k-fees-you-need-to-know-1825819800) ，但是检查你的费用。随着时间的推移，即使看似无关紧要的金额也会导致你储蓄金额的巨大差异。

### 忘记你的账户

我们 [也涵盖了这一个](https://twocents.lifehacker.com/keep-tabs-on-old-retirement-accounts-1825712142) 。你可能想要一个“ [套而忘了](https://twocents.lifehacker.com/how-to-build-an-easy-beginner-set-and-forget-investm-1686878594) ”的投资组合，但你不想忘记自己的实际账户。密切关注你的退休账户，这样你就不会成为 2500 万人的一员，错过数千亿美元的养老金福利。

这里有更多关于 [如何跟踪](https://twocents.lifehacker.com/keep-tabs-on-old-retirement-accounts-1825712142) 的内容。

如果你没有 401(k)和 [，许多工人没有](http://www.pewtrusts.org/en/research-and-analysis/issue-briefs/2017/02/retirement-plan-access-and-participation-across-generations) ，或者不喜欢你的计划选项，考虑一个 [IRA](https://twocents.lifehacker.com/a-beginner-s-guide-to-opening-an-ira-1607498930) 或 [Roth](https://twocents.lifehacker.com/why-you-need-a-roth-ira-1823805955) 。许多相同的规则也适用。
