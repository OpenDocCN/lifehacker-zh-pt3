# 为什么总有关于巧克力的新研究

> 原文:[https://life hacker . com/why-there-s-always-a-new-study-about-chocolate-1819679562](https://lifehacker.com/why-there-s-always-a-new-study-about-chocolate-1819679562)

你对巧克力，也许是黑巧克力，是否有一个模糊的概念，认为它对你有好处？这对你的心脏健康还是有好处的。在某种程度上，你知道糖果实际上不是健康食品，但似乎总有标题说它是。

Vox 的一些 [精彩报道解释了这种印象的来源:由火星资助的数十年研究。(](https://www.vox.com/science-and-health/2017/10/18/15995478/chocolate-health-benefits-heart-disease) [雀巢](http://www.research.nestle.com/) 和 [美国巧克力制造商协会](http://www.nca-cma.org/american-cocoa-research-institute-acri/) 也尽了自己的一份力量。)

即使科学是完全合理的，研究资助的模式最终也会误导我们。以下是发生这种情况的几种方式:

*   如果巧克力研究比其他领域的研究有更多的研究资金，就会有更多的科学家对巧克力进行研究。
*   即使巧克力公司不编辑结果，他们仍然可以选择向那些之前的工作使巧克力看起来不错的科学家提供资助。
*   他们可以为他们想得到答案的问题提供资金。例如，他们会资助研究巧克力的益处，但很少甚至没有资助研究巧克力的危害。
*   当这项研究发表时，他们肯定会发出新闻稿。

这不是巧克力独有的问题。例如，如果你看到一项吹捧蔓越莓有益健康的研究，它几乎总是由海洋喷雾资助的。如果是关于彩蛋的，大概就是 [美蛋板](http://www.aeb.org/) 。你明白了。与此同时，谁在资助研究这种单一食物的关注对于公众理解营养是多么可怕？几乎没有人。

大多数研究都不是开创性的，甚至都不实用；“巧克力”研究通常包括给老鼠或人喂食可可豆制成的补充剂，你肯定不会为了好玩而吃。Vox 报告指出，一剂“有益心脏健康”的黄烷醇需要你吃 2.5 *磅*牛奶巧克力。黑巧克力的浓度更高，但是你仍然需要 750 卡路里的热量。这可不是你每天都能健康饮食的。
