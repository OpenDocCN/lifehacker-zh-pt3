# 这是最好的转换图像的简单应用程序

> 原文:[https://life hacker . com/this-is-the-best-simple-app-for-converting-images-1796238125](https://lifehacker.com/this-is-the-best-simple-app-for-converting-images-1796238125)

如果你曾经需要上传一两张图片，或者处理一份扫描文件，你应该知道伴随转换而来的所有令人头疼的问题，以及对不同文件类型的网站有多挑剔。

将扫描图像转换成 PDF 格式要么需要一些图像编辑知识，要么需要访问一个在线转换网站，在那里你的数据被上传到天知道的地方并为你转换。这并不容易，也绝对不安全。

为了防止将您的数据发送给未知的第三方，请使用 Mac 应用程序。它为您处理转换，将多种图像文件类型转换为 pdf、jpg 和 png。

它存在于你的菜单栏中，当你点击它时，它会打开一个拖放窗口。你也可以直接从键盘上点击⌘+Shift+8.来访问它在那里，你可以放入不同文件类型的多张图片，并选择是将它们转换成 JPG 文件还是 PNG 文件，或者将照片合并成一个 PDF 文件。

它目前只处理几种文件类型，尽管随着 Alchemy 将来增加更多支持的文件类型，你将能够将多张图像转换成一张 GIF。它是一款开源应用，所以你也可以在它的 [Github 页面](https://github.com/dawnlabs/alchemy) 上监控它的发展。
