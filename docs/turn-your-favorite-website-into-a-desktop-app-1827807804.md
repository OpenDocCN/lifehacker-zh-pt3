# 将您最喜爱的网站变成桌面应用程序

> 原文：<https://lifehacker.com/turn-your-favorite-website-into-a-desktop-app-1827807804>

**Windows，Mac:** 你可能有几个你一直在使用的网站——也许是你工作需要的一个特殊的 CMS，一个你用来为客户跟踪和计费时间的时间跟踪网站，或者一个网页游戏。如果你厌倦了每次需要访问浏览器时都打开它，你有一个选择:把它变成一个应用程序。

Watch

我这么说有一个警告，作为网络服务[web 2d esk](https://desktop.appmaker.xyz)*真的没有*做大量繁重的工作来将你喜欢的网络应用转换成桌面应用。它为网站创造了一个 [电子](https://electronjs.org/) 包装器。如果你把你喜欢的每个网站都“转换”成一个“应用”并同时运行其中的几个，这些包装器会占用你硬盘上相当大的空间，并且会占用大量系统内存，所以要谨慎选择。

尽管如此，对于一次性使用，Web2Desk 肯定是方便的(即使服务 [并不完全是唯一的](https://github.com/jiahaog/nativefier) )。虽然我不会将它用于任何已经有离线应用程序的网络服务——比如 Trello、Twitter 或 Whatsapp，举三个 Web2Desk 在其网站上突出显示的例子——但如果你想让你最喜欢的闲置点击游戏在一个独立于你在网络浏览器中所做的任何事情的窗口中运行，它肯定是有帮助的。

当一个网络应用程序只有你不用的操作系统的桌面版本时，Web2Desk 也是有用的。如果你只是不想等待你的浏览器加载那些你发誓有一天会通过的 40 多个标签，将你的关键网站“转换”成“应用程序”可以节省你一点时间(并且可能比你膨胀的浏览器消耗更少的系统资源)。

要开始，只需点击 Web2Desk 的网站。如果它还没有你喜欢的服务的应用程序，只需输入网站的地址，给你的应用程序命名，选择一个图标(或创建你自己的-仅 PNG 文件)，输入你的电子邮件，然后等待 web2desk 创建一个应用程序供你下载。完成后，您会在电子邮件中收到一个链接。

如上所述，我不会为你使用的每一个网络应用程序都这样做。一个在线游戏 Cookie Clicker 的包装在我的机器上吃掉了超过 100MB，对我来说这似乎是一个很大的数目。尽管如此，Web2Desk 仍然是一个很好的工具，可以将有用的书签保存在浏览器的文件夹中，以防万一，你需要一种更简单的方法来访问主浏览器之外的一些喜欢的 Web 服务。
