这是另一个翻译:

“如果有人犯了错误，请友好地纠正他们，并指出他们遗漏了什么。但如果你做不到，那就怪你自己——或者不怪任何人。”

### 这意味着什么

人都会犯错。当他们确实犯了错误，你觉得有必要指出来时，至少善意地告诉他们下次如何避免。但如果你指手画脚，拒绝教导，责任就在你身上。那是你的失败，不是他们的，因为你选择了忽略做好事的机会。当然，如果你不能帮助他们学习，谁也没有责任。他们犯的错误很可能是你犯的。

### 从中吸取什么

责备对任何人都没有好处。它让被指责者感到沮丧，不愿意再次尝试，它让指责者感到自以为是，没有过错，它培养了一种为他人推卸责任的文化。这是一种自我保护的自私行为，旨在降低人们对其他人的评价，这样你可能在比较中看起来更好。在一个学习往往伴随着尝试和错误的环境中，这是大多数工作环境，失败的感觉往往是足够的惩罚。没必要指指点点，放大。再说了，团队工作，一个人的失败就是所有人的失败。

指出别人的错误还是可以的。毕竟，许多人甚至没有意识到他们正在犯错误，直到有人这样做，他们需要被告知，以便他们可以变得更好。但是如果你要强调某人的错误，你应该愿意告诉他们如何在将来避免这些错误。指导有助于他们学习，并鼓励他们再次尝试，通过教学，你进一步加深了自己对主题的理解，并巩固了你作为团队中积极有用的一员的地位。所以不要怪——教。如果你不能教，因为你也在学习，知道你的位置，只担心你自己。
