# 离开沙发，开始使用 C25K 应用程序跑步

> 原文:[https://life hacker . com/get-off the-couch-and-start-running with-the-c25k-app-1825351919](https://lifehacker.com/get-off-the-couch-and-start-running-with-the-c25k-app-1825351919)

iOS/Android:我真的很讨厌跑步。我一直讨厌跑步。每次我决定(再次)尝试跑步时，我都会想我有多愿意骑自行车、举重物或做其他任何事情(除了深蹲)。然而，当我去跑步时，我会使用免费的 C25K 应用程序来尝试并指导我跑得更远更好。

Watch

你可能从它的名字就能猜到，C25K ( [iOS](https://itunes.apple.com/us/app/c25k-5k-trainer/id485971733?mt=8) ， [Android](https://play.google.com/store/apps/details?id=com.c25k&hl=en_US) )旨在让你在八周内从沙发上(身材没那么好)跑完 5 公里(身材还不错)。如果你不能保证每周跑步三次，每次半小时左右，这个应用程序就没有任何用处。然而，由于它试图让你达到 5 公里，你会逐渐了解跑越来越长的距离的概念——部分是为了你的健康，部分是为了你不会在第一天意识到不间断地跑 30 分钟有多难时退出。

我最喜欢 C25K 的一点是，它的免费版本非常适合你跑步时的基本动作。你的智能手机中央会有一个计时器，为你的锻炼设定倒计时，你会收到音频更新——理想情况下，通过你慢跑时使用的耳机或耳塞——告诉你何时在跑步和步行之间交替。记住，你正在学习如何跑 5000 米:你首先要混合轻度和常规锻炼，以建立完整的无休息跑。

你可以花 5 美元(在 iOS 上)或 3 美元(在 Android 上)升级该应用程序，以解锁其“专业”功能，包括距离跟踪、卡路里跟踪和消除应用程序中的广告。如果你发现自己被运行中的 bug 咬了一口，这是值得购买的。也就是说，当我开始的时候，我并不太在乎距离或卡路里；我只是想确保我的耳朵里有一个方便的向导，告诉我什么时候走，什么时候跑，C25k 非常适合这一点。
