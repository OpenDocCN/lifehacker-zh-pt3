# 天空中的金字塔桌面

> 原文：<https://lifehacker.com/the-pyramids-in-the-sky-desktop-1740853829>

[MistakeNot](http://kinja.com/mistakenottwo) 说他们从来没有玩过命运，但这个来自游戏的概念艺术仍然制作了一个漂亮的壁纸。结合当前正在播放的小部件、工作空间和位于中心的可视化工具，您就拥有了一个出色的软件包。



如果你喜欢这种外观，这里是你设置它所需要的:

*   [金字塔战舰](https://www.artstation.com/artwork/rlod2) 壁纸、观念艺术 [多杰贝尔布鲁克](https://www.artstation.com/artist/dorje)
*   用于 Windows 的 [雨量计](http://rainmeter.net/) 系统调试和监控工具
*   [为雨量计编码皮肤](http://lilshizzy.deviantart.com/art/Rainmeter-Encoded-v1-2-3-215992941) 为日期和时间
*   [圆形音乐进度条](https://www.reddit.com/r/Rainmeter/comments/2rakrr/any_good_circular_music_progress_bars/) 为内圈，显示歌曲进度
*   雨量计的 [Visbubble 皮肤](http://undefinist.deviantart.com/art/VisBubble-for-Rainmeter-488601501)随音乐移动的外圈可视化器的
*   [生活是雨量计](http://fabricioply.deviantart.com/art/Life-Is-Strange-1-10-543366965) 的奇怪皮肤，不用时隐藏，悬停时显示文本快捷方式
*   雨量计的 [、歌曲信息的](http://marcopixel.deviantart.com/art/Lano-Visualizer-Highly-configurable-visualizer-545915953) 就在正在播放的小工具下
*   左上角的 [雨量计的谜皮肤](http://enigma.kaelri.com/downloads/) 为正在播放的 widget

MistakeNot 在他下面的 Kinja 帖子中有更多关于每个小部件如何配置的细节，所以如果你对如何设置一切有问题，或者只是想让他们知道桌面看起来有多棒，就去那里吧。
