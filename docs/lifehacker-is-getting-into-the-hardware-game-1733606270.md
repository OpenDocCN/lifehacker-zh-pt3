# Lifehacker 正在涉足硬件游戏

> 原文:[https://life hacker . com/life hacker-is-get-into-the-hardware-game-1733606270](https://lifehacker.com/lifehacker-is-getting-into-the-hardware-game-1733606270)

不久前，Lifehacker 回避了硬件审查。我们可以在这里或那里谈论一些我们喜欢的事情，但总的来说，我们避免了许多小工具的评论，因为担心这不是我们的专长。我们结束了。从现在开始，我们将推出几个新系列，将我们的“这是否改善了您的生活”评论风格推广到更多的小工具、外设、音频设备、PC 组件等。为此，我们引入了三件大事:

*   周日对决:这是一个新的每周系列节目，周日对决将根据我们自己的测试和研究，在现实世界中让两个(或更多)特定的产品、应用或服务相互竞争。Kodi 和 Plex 哪个更适合你的家庭影院电脑？廉价的剩余库存背包真的比昂贵的日常携带包好吗？性能 MX 是不是比专注于游戏的 Proteus Core 更好的全能鼠标？我们会知道的！因为它将在周日运行，它将取代 Hive Five。在过去的至少七年里，《蜂巢五号》一直是一个惊人的每周专题节目。我们会不时更新旧的，我们会不定期地定期更新新的，但这不再是每周一次的事情了，我们更愿意利用这个空间来动手做我们正在写的项目。
*   **我们喜欢的东西的回归**:我们一直有一个 [我们喜欢的东西](http://lifehacker.com/tag/stuff-we-like) 专栏，但是很少，也很少那么详细。但是，好吧...我们喜欢很多东西。仅仅因为某样东西不是最受欢迎的，或者不是班上最好的，并不意味着它不好。我们希望花更多的时间来突出我们真正喜欢的设备和小工具，并且认为你也会这样做——无论是耳机、枕头、PC 组件还是防水笔记本(好吧，我们已经做过了)。不要把它想象成一个正式的回顾系列，而更像是我们工作方式的一个持续版本。
*   **Lifehacker 评论任何东西**:当然，我们可以整天评论我们喜欢的东西，但归根结底，我们是为你们而来的——所以我们愿意评论 [任何你想让我们评论](http://lifehacker.com/the-lifehacker-staff-will-review-anything-you-want-1733408447) 的东西。那里不缺 unitaskers 乞求你的厨房柜台空间，或承诺帮助你减肥的疯狂运动工具。也许你想听听我们的意见，那个香蕉切片器或带盖子盛蘸酱的杯子是不是真的是生活帮。也许你只是想让我们在你投资之前，先试试新的高压锅或 Kickstarter 支持的电池组。来吧，我们要带走所有的人。
