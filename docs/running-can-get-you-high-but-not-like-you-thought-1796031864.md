# 跑步可以让你兴奋——但不像你想的那样

> 原文：<https://lifehacker.com/running-can-get-you-high-but-not-like-you-thought-1796031864>

与你(或你的教练)可能相信的相反，内啡肽对你在长跑后感到的头晕目眩的精力充沛没有责任。与这种受欢迎的感觉相关联的是让你真正兴奋的东西:大麻。

Watch

具体来说，我们谈论的是内源性大麻素家族中的一种化学物质。与大麻中发现的传统大麻素(如 THC 和 CBD)有些类似，内源性大麻素是在体内产生的。

YouTube 频道 SciShow 的主持人迈克尔·阿兰达(Michael Aranda)描述了“跑步者的亢奋”可能来自于你在锻炼过程中产生的内源性大麻素。

> 内源性大麻素与你大脑中的相同系统相互作用，就像大麻中的四氢大麻酚一样，但它们是你的身体自然产生的。它们与缓解焦虑和降低疼痛敏感度有关。

长期以来认为内啡肽是跑步者兴奋的原因是错误的——内啡肽不能穿过大脑的半透膜到达你的头部。内啡肽是有效的止痛药，但只有当你锻炼后感到身体和肌肉疼痛时才有效。

阿兰达引用了 2015 年的一项研究，研究人员将老鼠放在轮子上，发现它们产生更多的内源性大麻素，称为“anandamide”。然后他们把这些老鼠放在一个热板上，观察它们对压力的反应。

> 跑的老鼠血液中有更多的 anandamide，并且在热板上需要更长的时间变得明显激动。

阻断小鼠的内啡肽受体不会影响在热板上度过的时间，尽管阻断 anandamide 受体会使小鼠在热板上更加焦虑，类似于没有进行 3 英里快步跑的小鼠。

当然，你不会在跑步后像风筝一样兴奋，但是如果你想找一点自然的快感，内源性大麻素可能会帮助你克服这个困难，让你真正享受你的锻炼。你是否想在慢跑前小睡一会儿完全取决于你自己 。
