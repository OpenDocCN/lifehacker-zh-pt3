# 最常见的硬拉错误及其纠正方法

> 原文:[https://life hacker . com/the-most-common-hard lift-errors-and-how-to-fix-them-1787477967](https://lifehacker.com/the-most-common-deadlift-mistakes-and-how-to-fix-them-1787477967)

你可能想知道弯腰捡起重物有多复杂，但是如果做错了，你的背部会有力地告诉你，这非常复杂。硬拉是一种 [不可思议且实用的全身锻炼](https://lifehacker.com/why-deadlifts-are-worth-the-hype-and-how-to-do-them-sa-1756526319) ，但是很多事情，从你握重的方式到你如何“站立”，都有可能出错。

### **你的臀部不够弯曲**

首先要注意的是，蹲下来接杠，然后再背起来，这是一个很好的方式来显示你的身体你讨厌它。硬拉的主要动作应该是“前后 ”正如 [我们在硬拉入门](https://lifehacker.com/why-deadlifts-are-worth-the-hype-and-how-to-do-them-sa-1756526319) 中解释的，这种最大的臀部弯曲但最小的膝盖弯曲运动被 [称为臀部铰链](https://lifehacker.com/why-deadlifts-are-worth-the-hype-and-how-to-do-them-sa-1756526319) 。

做臀部铰链看起来就像你在向后推你的臀部，但弯曲你的臀部，而不是简单地拱起你的背部，好像你试图撞你身后的人。脊柱保持中立，整个背部和颈部保持一条直线，然后将骨盆向前推，重新站起，完成硬拉。髋关节是这项运动的关键，很多人都搞砸了。

### **你的首发位置烂透了**

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-WFUOtnI1jwk&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-WFUOtnI1jwk&start=0) 

一次好的硬拉实际上在你拿起重量之前就已经开始了。新手硬拉运动员通常倾向于 [在杠上过于前倾](https://www.youtube.com/watch?v=WA2CCK9TLk4) ，当他们真正举起重量时，会对下背部施加很大压力。上面的视频，由 at lean-XT9 制作，讲述了如何设置一个合适的硬拉。你的很多设置，像你的站姿，你的脚的宽度，你离吧台有多近，等等，都取决于你的身体结构和一点个人喜好。

在你拿起重物之前， [深呼吸](http://vitals.lifehacker.com/the-right-way-to-breathe-for-more-powerful-weightliftin-1765146303) 并保持它，迫使空气和紧张进入你的腹部。挤压你的背阔肌(保持背部挺直和稳定的主要肌肉)，就像用[腋窝挤果汁一样，缓解杠铃](https://www.youtube.com/watch?v=iboVuB-Ig3k) 带来的一些紧张，并准备好通过将你的重量压入脚后跟来尽可能用力地拉。当然，所有这些都需要流畅而迅速地发生。它会随着练习而来。

### **你的背太圆了**

也许重量太重，你没有通过保持中性脊柱和收紧腹肌来做好足够的身体准备来支撑重量，或者你的背部或臀部肌肉太弱。当你拿起重物时，有很多原因会使你的背部弯曲。

如果你没有长时间硬拉，很难保持背部中立是正常的。你可能只是不习惯保持这种最初尴尬的姿势。不要担心，随着更多的练习和身体变得更强壮，你会变得更好。但是如果你不弯腰就够不到地板上的杠铃，可以把杠铃放在两三个重物或深蹲架上的销钉上，以此来提高杠铃的高度；并练习部分硬拉，以练习脊椎中立位。

### **你的握力太大**

当你抓杠铃时，如果抓得太宽，你就不能很好地利用你的侧肌，而侧肌是硬拉的重要组成部分。可能感觉也有点尴尬。

波士顿 Cressey Performance 的表演教练 Eric Cressey 建议 [将双手尽量靠近臀部](https://lifehacker.com/a-simple-grip-adjustment-can-instantly-improve-your-dea-1782626971) 。从这里开始，你应该仍然可以伸手抓住杠铃上的环，找到最适合你的宽度。此外，通过更紧密的握杆，你可以让杠铃更靠近你的身体，以便在你的举起过程中更好地控制。

### **你的手臂做了太多的“工作”**

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-c8_bfhKnyzc&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-c8_bfhKnyzc&start=0) 

你的手臂在硬拉中的主要作用是保持杠铃稳定，并在整个过程中保持杠铃紧贴身体。从开始到结束，你的手臂应该保持伸直并完全伸展。你的手臂永远不应该拉或猛拉杠铃。保持肘部锁定并靠近身体。

奥马尔·伊素福 的这段视频详细解释了保持双臂伸直的重要性。他解释说，伸展手臂有助于保持你的背阔肌运动。(开始看到这里的格局了？)

### **酒吧离你渐行渐远**

在一次好的硬拉中，想象一下杠铃沿着一条垂直的直线上下移动。如果它偏离了这条路径，通常意味着你开始向前倾斜太多，你没有足够地收缩你的 lats，或者横杠离你开始的姿势太远。除了收紧鞋带，试着在开始时调整小腿与横杠的距离。如果你的小腿正对着吧台，试着向后移动几英寸。如果你离得太远，那就靠近一点。

### **你在电梯顶部超伸背部**

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-PUDZ4K5Om-A&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-PUDZ4K5Om-A&start=0) 

当你用臀部向前推动到达硬拉的顶部时，称为“锁定”，许多人在顶部拱起他们的下背部，在他们不应该的时候伸出他们的骨盆。要完全锁定硬拉，你的臀部和膝盖需要“锁定”到位。用你的背部代替你的臀肌，向前移动你的膝盖，防止你的臀部和膝盖锁定。相反，当杠铃经过你的膝盖时，想想“收拢你的尾巴”并用力挤压你的屁股。为了加强这个想法，看看上面由 [柴油强度&调节的视频。](https://www.youtube.com/channel/UCH8YnzTWYsxR-SP1uzKjjOw)T3】

### **你没有承诺拉动**

这比我最近第一手学到的任何东西都更有精神。前几天，我不确定自己能否拉 250 磅，但我还是试了试。我第一次尝试甚至连移动重量都失败了。幸运的是，我的一个一直在观察的朋友告诉我，我需要做出承诺。“趴下来，尽可能用力拉，”他告诉我。当我这样做的时候，体重增加了。

当然，我为自己新的个人最好成绩而欢欣鼓舞，但这也让我充分体会到了 [“全力以赴”的真理](https://www.youtube.com/watch?v=VVF30XJzeVk) 即使重量令人生畏，你也必须坚定地将重量从地面上卸下来。没有“测体重”你要么捡起来，要么不捡。任何犹豫的迹象都会毁掉你的计划，让你精神上一团糟。
