# 你应该用动物脂肪制作非肉类食物

> 原文：<https://lifehacker.com/non-meat-foods-you-should-make-with-animal-fat-1822531180>

如果你在这些地方呆过一段时间，你就会知道我的生活基本上是由鸭肉 赞助的 [。无论我是](https://skillet.lifehacker.com/turn-even-the-driest-turkey-into-a-delicious-spread-1820483164) [腌制火鸡腿](https://skillet.lifehacker.com/these-confit-turkey-legs-taste-like-the-renaissance-fai-1820339857) 还是 [制作蛋黄酱](https://skillet.lifehacker.com/why-duck-fat-mayo-is-the-mayo-you-should-be-making-1821972304) ，我几乎都是为了餐桌上的美味而活。

但是鸭子并不是唯一一种提供优质美味脂肪的动物。Shmaltz(鸡油)、鹅油和培根油在厨房中都有其美味的用途，尤其是在准备蔬菜或其他非肉类美食时。以下是我最喜欢的几个:

### 绿色和含淀粉的植物部分

如果你不处理这些动物脂肪，你应该煮一些土豆。用上述任何一种方法给一大块红褐色土豆涂上油脂，然后在烘烤前涂上大量的盐，会给你带来一生中最好的烤土豆，培根油烤胡萝卜或布鲁塞尔几乎不需要任何额外的调料。

我也非常喜欢蘑菇炒培根油和鸭子的脂肪，因为蘑菇基本上是小海绵，如果它们要吸收任何东西，那也可能是培根油。所有这些动物脂肪在炒饭时也是极好的，特别是如果这是一份蔬菜为主的食谱，将受益于一点肉类风味。

### 糕点、面包和饼干

传统的全黄油面包皮没什么不好，但是 [这种带有百里香斑点的鸭肉馅饼](http://www.adrunkenduck.com/savory-duck-fat-pie-crust-pate-brisee/) 想成为你的新朋友。虽然把它保存起来用于美味的应用是有意义的，但我认为它更适合用来衬托苹果派的甜味。

但是鸭子并不是唯一一种比面包产品更好的禽类。 [用施马尔茨制作奶油蛋卷](http://www.alwaysorderdessert.com/2011/07/schmaltz-brioche.html) 会给你一个面包屑非常嫩、外皮非常细腻的面包，我很久以来都是用熏肉油制作 [饼干](http://theunmanlychef.com/2015/12/02/bacon-fat-biscuits/) 和 [玉米面包](https://food52.com/recipes/24670-buttermilk-bacon-grease-cornbread) 的粉丝。

### 超级丰富的蛋黄酱

自制蛋黄酱很棒，但自制 [鸭脂蛋黄酱](https://skillet.lifehacker.com/why-duck-fat-mayo-is-the-mayo-you-should-be-making-1821972304) 却低调的改变人生。通过将一些中性油换成动物脂肪，你可以得到味道和质地更丰富的酱料。我做了鸭肉、鹅肉和培根蛋黄酱，所有这些都大大提高了我三明治的质量。

### 最好的煎蛋

我是吃着我爸爸所谓的“脏鸡蛋”长大的，这意味着在脆猪肉条被移走后，它们被直接打碎到仍然非常油腻的培根锅中。这些鸡蛋不仅有最脆的边缘，而且有烧焦的培根小斑点，不需要额外的盐。老实说，在我 7 岁搬到洛杉矶之前，我不知道人们会用培根油以外的东西煎鸡蛋，巧合的是，这也是我知道可以从橄榄中榨油的时间。

### 罪恶的沙拉酱

在生菜上浇上热乎乎的、多肉的油脂可能看起来有点反常，但是一份热培根调味汁 (或 schmaltz 调味汁，或鸭油调味汁)会是一份非常美味的沙拉。你也可以将这些脂肪混合在你最喜欢的调味汁中，形成肉味十足的主干，但你必须在瓶子中保留一些植物油(至少一半体积)以保持流动。
