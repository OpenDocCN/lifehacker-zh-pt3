# 如果你喜欢旅行，你需要这些应用程序

> 原文:[https://life hacker . com/if-you-love-to-travel-you-need-these-apps-1825478443](https://lifehacker.com/if-you-love-to-travel-you-need-these-apps-1825478443)

除了打包的压力，确保所有电子设备都安全存放在某个地方的压力，记住旅途中为设备充电所需的每根电缆的压力，以及确保宠物喂食和植物浇水的压力，还有应用程序。

在你开始一次盛大的欧洲之旅之前，你应该下载什么应用程序来确保你最大限度地利用假期时间，参观最好的地方，并且不会迷路？

积分达人最近和几个旅游爱好者聊了聊，想弄清楚他们在环游世界时会用什么应用。虽然每个人在智能手机和平板电脑上安装的软件方面都有自己的偏好和需求，但以下是那些热衷于旅行的人往往会提到的应用:

### WhatsApp ( [iOS](https://itunes.apple.com/us/app/whatsapp-messenger/id310633997?mt=8) 、T5】Android)

这个方便的安全消息应用程序是必备的，这并不奇怪。所有人都在调查。(其日活跃用户超过 10 亿。)旅行时，通过支付国际数据速率而不是短信，这是一个省钱的好方法。你在上面发送的信息是端到端加密的，当你在外国咖啡店使用 wifi 时，你肯定希望拥有这种加密功能。WhatsApp 的 [功能丰富](https://lifehacker.com/secure-messaging-app-showdown-whatsapp-vs-signal-1794684943)——和你智能手机的默认消息应用一样多，如果不是更多的话。

### 谷歌地图([iOS](https://itunes.apple.com/us/app/google-maps-gps-navigation/id585027354?mt=8)[Android](https://play.google.com/store/apps/details?id=com.google.android.apps.maps&hl=en_US))

再见，苹果地图。挥手再见，瓦兹。到目前为止，谷歌地图是度假的首选地图应用。原因显而易见:离线地图为你提供所需的所有信息，同时让你将智能手机保持在飞行模式(以免产生昂贵的国际费用)，或者如果你在一个偏远的地方，根本没有服务； [自定义地图](https://lifehacker.com/how-to-plan-a-trip-itinerary-using-custom-google-maps-1440584988) 让你在踏上飞机前，设定好所有必去地点；你可以 [保存你偶然发现的新地点](https://lifehacker.com/how-to-make-the-most-of-google-maps-1793442153) 以便有空的时候重游；你可以 [ping 你的旅行团](https://lifehacker.com/google-maps-can-now-share-your-location-with-friends-in-1793520380) 让他们知道你在哪里；你还可以 [与想要预订类似旅行的朋友分享你最喜欢的度假地点](https://lifehacker.com/google-maps-now-lets-you-organize-your-favorite-places-1792293019) 。

当然，如果你不介意谷歌知道你去的每一个地方，你也可以用谷歌地图的 [便捷的时间轴](https://lifehacker.com/google-maps-now-shows-your-location-history-in-a-timeli-1719543525) 重温你的旅行。

### 优步( [iOS](https://itunes.apple.com/us/app/uber/id368677368?mt=8) ， [Android](https://play.google.com/store/apps/details?id=com.ubercab&hl=en_US) )

说说你对该公司之前的商业实践的看法——包括对数据收集和隐私的所有重要变化几年前就应该做出，此外还有对乘客安全的急需改善。

假设你不会遭遇价格飙升，优步可能是当地出租车或公共交通的绝佳选择。根据地区 和你司机的质量，你可能会得到更好的服务，更便宜的票价，或者 [关于你要去的地区或旅游景点的知识源泉](https://lifehacker.com/why-you-need-to-start-talking-to-your-lyft-driver-1793022552#_ga=2.31047612.2095465166.1524498080-542336946.1523479293) (还有小吃！ [和](https://youtu.be/s7NKqzUJOJY?t=6m10s) ！)

当然，不要忘记使用像 [RideGuru](https://ride.guru/) 这样的服务，以确保你的旅行得到尽可能好的价格。

### 猫途鹰( [iOS](https://itunes.apple.com/us/app/tripadvisor-hotels-restaurants/id284876795?mt=8) ， [Android](https://play.google.com/store/apps/details?id=com.tripadvisor.tripadvisor&hl=en_US) )

这个问题让我有点惊讶，因为我以为大多数人都会向 Yelp 寻求建议，比如去哪里吃饭，看什么，去哪里度假，但是我错了。比 Yelp 更值得推荐的是，猫途鹰有大量的信息、评论和用户评论，涵盖了你的度假目的地所能提供的一切。

我喜欢利用猫途鹰，不是直接计划我的旅行，而是确认我想去参观(或停留)的地方并不都是围绕着不完美体验的美丽画面。换句话说，如果一家酒店看起来很棒，但有臭虫、糟糕的服务或糟糕的无线网络，很可能已经有人投诉过了。通读足够多的评论，你会对你所谓的神奇目的地的真实情况有一个大致的了解。

虽然你最好单独预订酒店、游览和餐饮，而不是通过猫途鹰的应用程序，只是为了确保你得到最好的交易，但一定要使用其他猫途鹰会员创建的旅行指南 [来激励你的计划过程](https://lifehacker.com/the-best-travel-destinations-according-to-tripadvisor-1825240108#_ga=2.225017625.2095465166.1524498080-542336946.1523479293) 。用户，而不是应用本身，是猫途鹰的秘密。

### PackPoint ( [iOS](https://itunes.apple.com/us/app/packpoint-travel-packing-list/id896337401?mt=8) 、T5】Android)

只有一个人说他们使用 PackPoint，所以我在这个条目上有点作弊，但我检查了这个应用程序，如果你是那种讨厌健忘的人，它看起来非常有用。我明白了。当我打包行李时，我总是感到紧张，因为我总是觉得我数错了一件重要的物品，忘记了一根电缆，或者带了所有的相机配件，却把相机放在我的架子上。诸如此类的事情。

有了 PackPoint，你可以输入目的地，输入旅行日期，并从度假时可能想做的常见活动列表中进行选择:比如游泳、海滩、吃大餐等等。然后，PackPoint 会根据你想做的事情和你要做的时间，以及旅行的潜在天气，创建一个简单的清单，列出你可能想放在行李或手提行李中的物品。

你可以刷走任何你知道你不需要的东西，一旦你把剩下的东西扔进你的行李中，就可以点击它们。当然，你也可以添加自己的物品，以防应用程序唤起你对一些你可能不想忘记的重要事情的记忆。
