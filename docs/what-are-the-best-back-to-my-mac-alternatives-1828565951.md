*如果你以前从未使用过“回到我的 Mac ”,它可以让你完成两件事:从一个远程系统拷贝文件(你也已经用你的 Apple ID 设置并回到我的 Mac ),并远程控制那个系统。幸运的是，对于许多其他应用程序来说，这并不难做到，尤其是那些与你在“回到我的 Mac”中找到的应用程序一样有用(如果不是更有用的话)的应用程序。*

### *远程控制 Mac 系统*

*我将首先从第二个特性开始，稍后您将会看到原因。总的来说，我是 [**谷歌的 Chrome 远程桌面应用**](https://chrome.google.com/webstore/detail/chrome-remote-desktop/gbchcmhmhahfdphkhkmpfmihenigjmpp?hl=en) 的忠实粉丝。即使你是一个苹果纯粹主义者，无法忍受在你的系统上使用谷歌的任何东西——如果你不得不整天使用 iCloud 网络界面，你肯定会很痛苦——谷歌的应用程序让远程访问系统变得非常容易。每当我不在家的时候(或者懒得起床做一些设置)，我就用它来控制我的 Mac 笔记本电脑上的 Windows 桌面。*

*Chrome Remote Desktop 最大的好处是，即使对电脑新手来说，它也非常容易安装。如果你是家人或朋友圈的技术支持者，这甚至会让你更加理想。你不需要煞费苦心地解释技术过程，你只需要带着人们完成 Chrome 远程桌面的设置。然后，他们可以使用其远程协助功能来生成一个访问代码，您可以在您的 Chrome Remote Desktop 版本上输入该代码来远程控制他们的系统。比起你试图向他们解释，你也许能更快地解决他们的问题，但是你可以随时解释，至少帮助他们理解你在做什么。*

*您可以使用的第二个选项是[**TeamViewer**](https://www.teamviewer.us/)T5】。我喜欢这个应用的地方在于，它允许你向一个睡眠中的系统发送 [Wake on LAN](https://community.teamviewer.com/t5/Knowledge-Base/How-does-Wake-on-LAN-with-TeamViewer-work/ta-p/33) 命令，而 Chrome Remote Desktop 没有这个功能。这样，你可以两全其美:当你不在家时可以节省一点能源，但可以远程拨号，而不必让你的电脑一直运行。TeamViewer 还使得在两个相连的系统 之间共享文件变得非常容易，这也可以解决“回到我的 Mac”中缺少的问题。*

### ***远程访问您的 Mac 文件***

*远程访问文件的最简单方法是在所有系统上为您的帐户启用 iCloud drive(通过“系统偏好设置”>“iCloud”)。这样，当你下一次打开其他系统的电源时，你放在桌面或文档文件夹中的任何文件都会自动与它们同步。你甚至可以在 iCloud 中获得(巨大的)5gb 的免费空间。发疯吧。*

*有很多其他云服务可以让你做同样的事情，我们已经涵盖了其中的大部分( [和它们的成本](https://lifehacker.com/is-google-one-cloud-storage-a-good-deal-1826049257) )。唯一的缺点是，如果你需要一个不在你的同步文件夹中的特定文件，你必须使用上述应用程序之一远程连接到你的系统并拖动它，然后等待它同步到你的远程系统。*

*如果你更疯狂一点，你可以省去中间人，在家里建立自己的 VPN 服务器——通常是在 [高端路由器](https://restoreprivacy.com/vpn-routers/) 或 [网络附加存储盒](https://nordvpn.com/tutorials/synology-nas/openvpn/) 上的一个选项，或者作为 [现有网络基础设施的一个独立附件](https://www.amplifi.com/teleport/) 。使用类似 [Tunnelblick](https://tunnelblick.net/) 的应用程序拨号，您的远程系统将能够看到您家庭网络中的所有其他系统。然后，您可以访问您之前共享的驱动器并以这种方式获取您的文件。*

*那都说了，也许最好的选择就是干脆 [尽可能地住在云中](https://lifehacker.com/how-do-you-back-up-your-data-in-the-cloud-1828396275) 。虽然你可能会对将文件交给第三方服务感到有点“呃”(特别是如果你的公司禁止你对工作文件这样做的话)，但当你不是对其福祉负责的人时，访问你的东西会容易得多。*
