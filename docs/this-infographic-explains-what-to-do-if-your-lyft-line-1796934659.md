# 这张信息图解释了如果你的 Lyft 线路发生事故该怎么办

> 原文：<https://lifehacker.com/this-infographic-explains-what-to-do-if-your-lyft-line-1796934659>

拼车服务让我们许多时间紧张的生活变得更加轻松，但伴随这些便利而来的是一系列其他问题:司机和 [乘客安全问题](https://www.theatlantic.com/technology/archive/2015/03/are-taxis-safer-than-uber/386207/)[公司丑闻](http://gizmodo.com/travis-kalanick-has-resigned-his-role-as-uber-ceo-1796287392) ，当然，还有潜在的交通事故。不管你的车是发生了轻微事故还是追尾事故，你都需要知道这些。



最大的问题也许是:保险到底涵盖了什么？毫不奇怪，这是一个合法的沼泽地，对各个州和城市的政策很敏感。大多数司机应该有自己的个人保险，但是他们也可以从他们的拼车合同中得到补充。 [谁在载你](http://www.whosdrivingyou.org/faq) ，一个倡导保护司机和乘客的网站写道:

> 优步、Lyft 和 Sidecar 都声称拥有 100 万美元的“责任”保险。这听起来让人印象深刻，让人放心，但专家们一致认为“拼车”保险在覆盖范围上存在危险的缺口。例如，TNC 司机打开应用程序并等待 pinged 的这段时间现在通常被称为“第一阶段”。

时段 1，根据下面由 [Now Sourcing](http://nowsourcing.com/) 提供的信息图，是该服务的应用程序打开的时间，还没有接受旅行，并且没有乘客在车上。这意味着拼车公司可以说，从技术上来说，你还没有 [【为他们工作】](https://www.wired.com/2015/07/california-forces-uber-rivals-bolster-insurance/) 。至少在加州，最近的一部 [法律](https://leginfo.legislature.ca.gov/faces/billNavClient.xhtml?bill_id=201320140AB2293) 已经强制要求运输网络公司(TNCs)提供主要保险，以涵盖第 1 期内第三方的伤害、死亡和财产损失的费用。如果你是一名司机，保护自己:钻你的保险公司，并研究城市的法律。

如果你是拼车事故中的乘客，一定要记下司机的所有信息，包括司机的联系方式、保险信息和拼车公司。更多信息，请查看下面的信息图。
