# 这个工具可以让你一次检查大量的亚马逊产品评论

> 原文：<https://lifehacker.com/this-tool-lets-you-examine-tons-of-amazon-product-revie-1821018875>

想花一大笔钱买那个闪亮的新玩意儿吗？好吧，你可能想先看看一些亚马逊的评论，这样你就知道你要做什么了。如果挖掘数百篇评论听起来像是一件苦差事，这个免费工具会让它变得简单。



[评论指数](https://thereviewindex.com/us) 是一个基于神经网络的工具，它扫描亚马逊用户对电子产品和小工具的评论，寻找评论中的重复模式，然后以易于阅读的图表和图形向您显示它们。你所要做的就是在工具中粘贴一个亚马逊产品页面的 URL(它现在只对电子产品和电器有效)，然后点击 summarize，然后你会看到有多少关于该产品的评论，其中有多少是旧的或未经验证的，以及这些评论是否充斥着虚假的垃圾评论。审阅索引通过检查每个审阅者的审阅计数、查找未验证的审阅条纹、短时间内的大量审阅以及其他因素来检查垃圾邮件。

该工具还向您展示了该小工具的几个关键品质，然后向您展示了在这方面有多少正面或负面的评论。例如，一个键盘的摘要可能会列出关键质量，如控制、舒适度、构建质量、游戏和产品可靠性。每个关键质量的绿色条越多越好。这是一个非常简单的方法，可以在很短的时间内全面了解产品。然而，如果你想了解更多信息，你可以打开每一个列出的质量，并阅读该产品的正面或负面评论。

审查指数是相当新的(和有限的)，但它有一个坚实的开端。工具 [一度是产品搜索](https://www.facebook.com/thereviewindex/) 上的头号热门人工智能产品，他们甚至在组合 中添加了 [的 Chrome 扩展。这仍然只是一个测试版，他们希望在此基础上](https://chrome.google.com/webstore/detail/amazon-reviews-summarizer/fciomokgnajoifiiaglcjkonligobneo) [在亚马逊上扩展到更多的产品类型，以及在未来覆盖不同的市场](https://thereviewindex.com/blog/about) 。你可以点击下面的链接亲自尝试一下。
