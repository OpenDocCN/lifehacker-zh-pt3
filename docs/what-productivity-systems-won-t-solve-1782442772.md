# 生产力系统解决不了的问题

> 原文：<https://lifehacker.com/what-productivity-systems-won-t-solve-1782442772>

如果你像我一样，你总是在 [寻找](http://unexecutive.com/perfect-system/) 完美的生产力系统。可惜不存在。问题不在于生产力系统——问题在于任何系统都无法解决的问题:拖延、对不确定性的恐惧和对不适的恐惧。



*本帖原载于* [*禅习*](http://zenhabits.net/unsolved/) *。*

例如，在我今天的任务清单上排在第一位的是“写禅宗习惯贴”和“写去杂乱课程简介”…很简单，对吧？两个相当简单的任务。但是当我开始写这篇文章时，我去查看和回复邮件，给我的妻子发信息，检查我的财务状况，稍微打扫一下厨房。当我推迟大任务的时候，我在小事情上非常有效率。

无论生产力系统有多好，当我开始拖延时，它就会崩溃。

当一项任务令人不舒服时，我会拖延，就像大多数人一样。当我面对一项任务的不确定性时，我会拖延。像我们大多数人一样。通常，我们会致力于一个新的生产力系统，尝试一个新的待办事项应用程序，开始组织，或者开始将我们的电子邮件收件箱清空，或者开始阅读一种新的生产力方式(就像你现在一样)……*所有这些都是为了拖延不确定或不舒服的任务*。

沿着分心的事物和小任务走下去要比面对一个重要但可怕的大任务容易得多。去你的银行网站查看你的余额，查看你最喜欢的博客和新闻网站，打开你最喜欢的社交媒体网络等等都很容易。这些没有准入门槛，总会给你一些奖励，而困难任务给你负面反馈，有入门障碍。不公平的比赛，是吧？

那么我们该如何应对呢？几个重要的方法:

1.  **退一步**。你必须远离分心的事物和琐碎的任务。离开电脑，出去走走，洗个澡，冥想，或者只是坐着什么也不做一分钟。这就够了。退后一步，想想你现在应该做什么，而不是最容易和最有诱惑力的事情。
2.  **记住你这样做是为了谁**。当你退后一步，想想谁会从你做的这项重要任务中受益。是同事、客户、顾客还是爱人？是你吗？这将如何让他们的生活变得更好？也许这会让他们觉得事情更简单，但也许会解决一个真正的痛点或满足一个重要的需求。我发现，当我专注于自己的舒适时，我会拖延，但当我专注于帮助别人时，我会克服拖延*。*
3.  **让自己潜入不适**。清除杂念，只要有你需要做的这一项任务，并允许自己不舒服几分钟。是的，这很糟糕，但是一旦你开始，它实际上并没有那么糟糕。这就像跳入冷水中——只管潜进去，不要多想。一旦你进去了，放松下来，面对不舒服的感觉，看看它并没有你担心的那么糟糕。
4.  **拥抱不确定**。很多时候，是对不确定性的恐惧阻碍了我们。我们不知道如何做一些事情，或者事情会如何发展，这让我们有点害怕。我们不想做这样的事情，甚至想都不想。我们回避、拖延和合理化。相反，将不确定性视为生活中不可或缺的一部分(谁想要一种你知道一切会如何发展的生活？超级无聊)，并把它视为生活中令人兴奋的一部分，一个充满惊奇、好奇、学习、探索和冒险的部分。不要把不确定性视为一件坏事，拥抱它，看看你是否能善于探索它。比如不舒服，一头扎进去放松一下。
5.  一点一点地做。是的，不确定性和不适可能是困难的。所以不要为了掌握它们而累死自己，只要少量地做就行了。花几分钟完成任务，然后四处走走，休息一下。然后坐下来，再次集中注意力，放松进入不适和不确定状态。重复一遍。

通过这种正念练习，你正在训练自己变得擅长不舒服和不确定。这是一个很好的技能！

最后，你不仅会面临生产力最困难的方面，你还会在这个过程中为他人做些好事。
