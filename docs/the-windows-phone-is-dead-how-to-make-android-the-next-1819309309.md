# Windows Phone 已死——如何让 Android 成为下一个最好的东西

> 原文：<https://gizmodo.com/the-windows-phone-is-dead-how-to-make-android-the-next-1819309309>

Windows Phone [已死](https://gizmodo.com/microsoft-confirms-windows-phone-isnt-getting-resurrect-1819278084#_ga=2.266555150.1287841135.1507495270-637574394.1506543594) ，但是微软并没有退出移动领域——事实上它正在加倍努力。随着大量 iOS 和 Android 应用程序的推出和快速改进，你可以在现有的手机上重现 Windows Phone 的许多感觉，只要你可以没有这些旋转的磁贴。以下是如何在移动领域全力以赴与微软合作的方法。



我们在这里将重点关注 Android，因为它比苹果的移动平台更具可定制性和可定制性，而微软正在充分利用这一优势。你可以在 iOS 上获得一套类似的 Redmond 应用程序，但你的体验仍然主要是苹果的。给自己拿一部安卓手机，我们就开始吧。

### **安装微软启动器**

对于外行人来说，当你点击 Home 键时，Android 启动器就会出现——它们控制图标、壁纸、主屏幕布局、应用抽屉等等。这跟安装 [一个全新版本的安卓](https://fieldguide.gizmodo.com/the-best-alternative-oses-you-need-to-install-already-1792312246) 不太一样，但感觉可以。

自 2015 年以来，微软一直在制作自己的发射器，名为 Arrow Launcher，但最近它被改造并更名为微软发射器。在撰写本文时，新版本尚未公开发布，但你可以通过点击 [这里](https://www.arrowlauncher.com/) 的下载链接，在你的 Android 设备上安装测试版。

启动启动器，你可以选择界面的某些方面，还可以选择让必应每日壁纸显示在你的手机上。如果你想轻松地将你所有的微软内容同步到你新修改的安卓手机上，你必须给这个应用一些权限，这样它才能控制你手机上的一切，并使用你的微软账户登录。

和谷歌的股票启动器一样，你可以在主屏幕上滑动鼠标，获得天气、约会和新闻的信息。小工具显示你经常使用的应用程序、最近的消息和照片以及最喜欢的联系人，所有这些都可以定制。你还可以使用便捷的小工具在你的设备和 OneDrive 上记笔记和查看文档。

你不必全身心投入微软的服务——比如，你可以显示谷歌日历而不是 Outlook 日历——但是正如你所料，一切都与微软自己的服务配合得最好。任何小工具都可以固定在你的主屏幕上，以获得更好的外观。

在其他地方，你的主屏幕图标和应用抽屉得到了视觉上的彻底检查，屏幕顶部持久的谷歌搜索框被持久的必应搜索框取代——如果你喜欢，你可以实际上将它切换到谷歌或雅虎，或者完全禁用它。点击主屏幕上的启动器设置按钮，几乎可以定制启动器的每个方面，从应用程序滚动的方向，到整个系统支持的手势。

### **安装微软的应用**

因此，你已经改变了你的 Android 设备的外观和感觉，并赋予它独特的微软色彩——然后你需要安装微软为 Android 提供的所有移动应用程序，这些应用程序的运行比你想象的要多。

这里有[Outlook](https://play.google.com/store/apps/details?id=com.microsoft.office.outlook)[Skype](https://play.google.com/store/apps/details?id=com.skype.raider)[Word](https://play.google.com/store/apps/details?id=com.microsoft.office.word)[Excel](https://play.google.com/store/apps/details?id=com.microsoft.office.excel)[PowerPoint](https://play.google.com/store/apps/details?id=com.microsoft.office.powerpoint)[OneNote](https://play.google.com/store/apps/details?id=com.microsoft.office.onenote)，以及 [Xbox](https://play.google.com/store/apps/details?id=com.microsoft.xboxone.smartglass) 为例，还有一个基本的 [到](https://play.google.com/store/apps/details?id=com.microsoft.todos)

如果你想让你的手机 Windows 化，有些应用程序比其他的更重要。 [Cortana](https://play.google.com/store/apps/details?id=com.microsoft.cortana) 可以取代 Google Now、Google Assistant、Samsung Bixby 或任何你已经拥有的智能 AI buddy，如今实际上可以通过长按 Home 键启动——按照应用程序内的说明进行设置。

[OneDrive](https://play.google.com/store/apps/details?id=com.microsoft.skydrive) 与此同时，把你微软云存储柜里的所有东西都带到你的安卓手机上。如果你愿意，它还会自动将你的照片和视频上传到云端，不过如果你用完了，你需要 [支付更多的空间](https://onedrive.live.com/about/en-US/plans/) 。同样，这可以在你安装应用程序的时候设置。

终于有了微软 Android Edge 的——在我们写这篇文章的时候，这款浏览器只有来自微软 的 [邀请才可用，但它应该很快就会面向所有人推出。不管 Edge 与 Chrome 之争的各种来龙去脉，对微软忠实者来说，最大的好处是你可以同步桌面版 Edge 的一切，包括书签和浏览历史。](https://www.microsoft.com/en-us/windows/microsoft-edge-mobile)

随着微软 Launcher 和微软应用程序的全部安装，Cortana 准备好按下按钮或喊出语音命令，你的 Android 手机突然看起来更像 Windows Phone。眯着眼睛，在背面贴一张诺基亚贴纸，它几乎可以是真的了。

### **将 Android 连接到 Windows 10**

自从发布以来， [Windows 10](http://fieldguide.gizmodo.com/how-to-make-your-windows-10-computer-work-like-new-agai-1787706643) 已经使连接到 Android 变得非常简单，这部分要归功于 Android 的多功能性，部分要归功于微软使其大牌应用程序更加专注于云，并随时可以在移动设备上使用。无论你的手机是什么牌子或型号，在 Windows PC 上使用 Android 已经很简单了。

在任何版本的 Windows 10 上，安装[Windows Phone Companion](https://www.microsoft.com/en-us/store/p/microsoft-phone-companion/9wzdncrfj3pm)，你就可以在你的电脑和手机之间交换文件、照片、音乐等——大部分情况下，这是通过安装我们上面提到的应用程序来完成的，但该伴侣让这个过程变得更加简单。

随着秋季 Creators 的更新，Windows 10 对 Android 手机(和 iPhones)变得更加友好，允许你在桌面上继续进行移动网络浏览，并(最终)实现跨两个平台无缝编辑文档等技巧。秋季 Creators 更新将于 10 月 17 日发布，但如果你是 Windows Insider 计划的一部分，现在就可以测试该功能。

首先在你的电脑上打开设置，进入电话区，选择**添加电话**:输入你的手机号码，你会收到一条短信发送到你的手机上。接下来在手机上打开 Cortana，选择尝试在 PC 上继续功能(它应该作为一个选项出现在即将到来的面板上)。完成这些步骤，你不仅可以继续在你的电脑上浏览，你还可以在你的台式机或笔记本电脑上看到未接来电和短信。

它仍处于早期阶段，但现在有一些东西可以尝试:在 Cortana 应用程序的即将到来的面板中点击任何新闻故事，然后点击右下角的“在 PC 上继续”按钮。选择**现在继续**，就像变魔术一样，相同的浏览器标签应该出现在你的 Windows 10 机器上的 Microsoft Edge 中。

随着时间的推移，所有这些都有望变得更加顺畅。在你的 Android 设备上，这可能不是一个完整的 Windows Phone 体验，但你可以如此接近它，就不难理解为什么微软决定停止开发自己的整个移动操作系统，而它可以建立在谷歌和苹果正在做的事情上。
