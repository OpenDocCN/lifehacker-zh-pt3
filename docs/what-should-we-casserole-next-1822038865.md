# 接下来我们应该做什么菜？

> 原文：<https://lifehacker.com/what-should-we-casserole-next-1822038865>

大家好，欢迎回到 [*将它砂锅*](https://lifehacker.com/tag/will-it-casserole) ，在这个栏目里我们看到哪些饭菜、菜肴、菜系可以重新想象，改造成砂锅。



又到了我们选择另一个砂锅概念的时候了。我们在三明治的砂锅菜制作方面取得了巨大的成功，所以我认为我们应该拓展一些更抽象和概念性的东西。我有一些想法:

*   **美食广场砂锅:**这个我提过几次。可能会有一些甜的炸鸡块。还有披萨。
*   **百吉饼砂锅:**也许百吉饼和熏鲑鱼？(尽管热熏鲑鱼听起来很恶心。)
*   **红龙虾砂锅:**你可以买切达湾饼干混合料，这样就不用担心浇头了。
*   墨西哥胡椒砂锅:或者辣椒砂锅？某种填充胡椒砂锅是为了。
*   奶酪盘砂锅:这是一个奇怪的想法，但你知道当你吃到完美的奶酪盘时，有一块饼干，你最喜欢的奶酪，一点蜂蜜和一些干果？我觉得我们可以抓住这个机会，把它做成某种大尺寸的烤布里干酪。但也许我是一个疯狂的梦想家。

这些是我的想法。显然，如果你有任何我没有提到的，请在评论中留下。你经常有很好的想法。
