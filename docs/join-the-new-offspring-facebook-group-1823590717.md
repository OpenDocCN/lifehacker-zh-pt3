# 加入新的后代脸书集团

> 原文：<https://lifehacker.com/join-the-new-offspring-facebook-group-1823590717>

作为一名家长，你可以使用一些备份——聪明的想法，节省时间的技巧，以及来自与你并肩作战的父母们的完全非判断性的支持。这就是为什么我们刚刚在脸书推出了 [后代:一个生活黑客育儿小组](https://www.facebook.com/groups/2018785615043946/) 。



这是一个面向所有人的团体(是的，甚至是你，勇敢的父母，他刚刚告诉你的孩子，当冰淇淋用完时，冰淇淋车会播放音乐——*特别是*你)。从挑食到吸毒，从社交媒体礼仪到丹尼尔·泰格(Daniel Tiger)为何不穿白天穿的裤子，这是一个讨论一切的地方。问问题，得到答案，为你刚刚尝试的奇怪的育儿方法唱赞歌，不知何故，奇迹般地，奏效了。

这个小组将由我和其他 Lifehacker 的编辑主持，他们将帮助创建内容和引导对话。我们很高兴能更好地了解你，并从中获得一些乐趣。

但是等等！如果你真的加入，你将有资格赢得为期 12 个月的 [KiwiCo](https://www.kiwico.com/) 订阅服务，每月为你的孩子发送一箱 STEM 和艺术项目。它价值 205 美元，有 0-16 岁以上儿童的选择。**你所要做的就是在脸书小组的第一个帖子上发表评论，告诉我们你最好的育儿技巧。**我们将选出我们最喜欢的，并向获胜者发送一张 KiwiCo 礼品卡。比赛将于美国东部时间 3 月 18 日周日晚上 11:59 结束。

[*完整大赛规则此处*](https://legal.kinja.com/contest-guidelines-1750917548) *。本次竞赛不是由脸书赞助、管理、支持或关联的。*

我们希望能在群里见到你！
