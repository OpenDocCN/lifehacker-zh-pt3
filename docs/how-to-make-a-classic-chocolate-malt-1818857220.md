# 如何制作经典的巧克力麦芽

> 原文：<https://lifehacker.com/how-to-make-a-classic-chocolate-malt-1818857220>

做奶昔似乎已经成为 [一项极限运动](https://www.google.com/search?q=black+tap+milkshakes&tbm=isch&tbo=u&source=univ&sa=X&ved=0ahUKEwjo9f6iisbWAhVP3GMKHQKVDMQQsAQIMQ&biw=1366&bih=637) 。有烈酒奶昔，糖果奶昔，还有整片蛋糕的奶昔。然而，有一种经典的、标志性的奶昔从这个浸满乳制品的狂热梦想中消失了:巧克力麦芽。



*欢迎来到* [***复古周***](https://lifehacker.com/tag/retro-week) ***，**在这里，我们将启动通量电容器，为您带来 20 世纪 50 年代的各种知识，从砂锅制作到防辐射防空洞建设，再到让孩子们放松和玩垃圾的乐趣。*

如果你从未喝过巧克力麦芽酒，我为你感到难过，但这不是你的错。经典的冷饮店并不在每条主要街道的拐角处，而且——如果你想自己制作——麦芽奶粉必须从万维网 订购。

然而，这是值得的麻烦。巧克力麦芽是巧克力味的，没错，但它也是麦芽味的，这意味着它有一种烘烤的烘烤味，这是普通巧克力奶昔所没有的。(或者，更简洁地说:它的味道像华勃或麦芽汁，后者要好得多。)

无论如何，今天美国缺少麦芽酒是一种讽刺。当一杯麦芽奶昔如此难找时，像 [这种满是猫眼的怪物](https://www.google.com/search?q=black+tap+milkshakes&tbm=isch&tbo=u&source=univ&sa=X&ved=0ahUKEwjo9f6iisbWAhVP3GMKHQKVDMQQsAQIMQ&biw=1366&bih=637#imgrc=2fv17eptVlSRLM:) 怎么能被允许存活呢？幸运的是，所有的都没有丢失，因为在家里做一个很容易。要成为你想在世界上看到的苏打怪人，你需要:

*   几大勺非常好的香草冰淇淋(我用了半品脱。)
*   你最喜欢的巧克力糖浆
*   全脂奶
*   2 或 3 大汤匙麦芽奶粉
*   生奶油
*   一种鲜红的黑樱桃
*   某种混合装置(我用了我的 [浸入式搅拌机](https://skillet.lifehacker.com/kitchen-tool-school-the-portable-powerful-immersion-b-1782522838) ，它允许我在我喝麦芽酒的杯子里制作麦芽酒。)

这种甜蜜舞蹈的步骤非常简单。从把你的冰淇淋舀进一个大杯子开始。

加入适量的巧克力糖浆。

用全脂牛奶包围你的冰淇淋，但不要淹没它。(说真的，如果我抓到你用脱脂或 2%我们会有话说；现在不是吝啬的时候。)

加入麦芽奶粉。

你现在准备好混合。为了获得完美的浓稠但仍可入口的稠度，你需要在搅拌和用勺子搅拌之间交替进行。(你可以在普通搅拌机里这样做，只是注意不要过度搅拌。)如果你的麦芽有点稀，简单的多加点冰淇淋。

在你的作品上放一些鲜奶油和一颗樱桃，并在里面插一根吸管。

啜饮一口，让自己回到一个更简单的时代，一个对抗重力、棉花糖包裹的冰淇淋饮料出现之前的时代。如果你想更像一个(苏打)怪人，你可以“一路点一个”，这是“用巧克力糖浆和巧克力冰淇淋制成的巧克力麦芽”的俚语
