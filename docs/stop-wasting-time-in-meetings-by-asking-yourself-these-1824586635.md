# 通过提前问自己这些问题，停止在会议中浪费时间

> 原文：<https://lifehacker.com/stop-wasting-time-in-meetings-by-asking-yourself-these-1824586635>

除非你非常幸运(或者只是非常年轻)，否则你很可能已经放弃了生命中宝贵的时间去参加会议，这些会议要么开得太久，要么根本就不需要召开。



不一定要这样。这些年来，我们已经花了 [大量的时间](https://lifehacker.com/how-to-not-suck-at-meetings-5976593)[来讨论如何磨练](https://lifehacker.com/how-to-have-a-meeting-that-isnt-a-complete-waste-of-tim-1570834406) 你的会议主持技巧，而且很多都归结于一套特定的基础知识。也就是说:总是设定一个议程，愿意引导人们回到正题，为每个话题留出一定的时间以避免事情拖得太长，等等。

然而。不是世界上的每个人都明白这些关键的真理，我们仍然经常发现自己在会议中缺乏重点，时间太长，或者没有完成他们设定的目标。因此，我们很欣赏作家兼演说家 Seth Godin 最近的博客文章 ，它提供了一个简洁的问题清单，你应该能够提前回答这些问题，以使会议有价值。其中包括:

*   有具体负责会议的人吗？
*   会议的预期结果是否已经让每个参与者都清楚了？
*   除了开会之外，还有什么更好的方法来完成手头的任务吗？
*   每个人都提前得到了他们需要的所有信息吗？

清单上的其他项目包括确保为讨论中的会议分配适当的时间(不要太多或太少)，以及为了使对话富有成效而需要出席的每个人都将出席。

根据个人经验，其他几个关键问题可以添加到清单中:

*   会议中有没有不需要出席的人(谁的时间最好用在其他地方)？
*   会后谁将负责跟进行动项目？
*   充分讨论议程上的每个项目需要多少时间？

当然，如果有人离题或说得太长，世界上计划得最好的会议也可能偏离轨道，所以主持会议的人需要愿意(礼貌地)打断人们，并将焦点转移到需要的地方(“我们需要继续前进”是英语中最甜蜜的词)。如果你有任何避免浪费时间的会议的万全之策，请在评论中留下它们。这是一个很好的观点，一旦我们完成了今天议程的其余部分，我们将有时间回答问题和提出建议。
