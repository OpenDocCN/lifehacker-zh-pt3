# 如果客人到来之前你只有时间打扫一个房间，那就把它变成浴室

> 原文：<https://lifehacker.com/if-you-only-have-time-to-clean-one-room-before-guests-a-1821191162>

如果你有客人来，而你只有时间打扫一个房间，那就把它变成浴室。



它发生了。也许你的工作会议开得太晚了，或者那份本该花 20 分钟的烤鸡秘方却花了 80 分钟。

忘了卧室吧——关上并锁上那个厄运之洞，希望你最好的朋友的新女友不会要求参观。跳过厨房水槽——它很快就会堆满更多的脏盘子。

经常被忽视的客房浴室是你应该集中注意力的地方。Rico Gagliano 是“ [晚宴下载](https://www.dinnerpartydownload.org/) ”的共同主持人，也是新书[*Brunch Is Hell:How to Save The World by throw a Dinner*](https://www.amazon.com/Brunch-Hell-World-Throwing-Dinner/dp/0316338958?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/if-you-only-have-time-to-clean-one-room-before-guests-a-1821191162&asc_source=&tag=kinjalifehackerlink-20)的共同作者，他在 WBUR 的 Hear & Now 播客上分享了 [这个建议](http://www.wbur.org/hereandnow/2017/12/04/brunch-is-hell) ，解释说如果你做这件事，“至少做了你的大部分尽职调查”他是对的。

远离喋喋不休和熟食店的干扰，这是房子里唯一一个让你的客人安静的地方，留下来观察，留下来默默判断。是的，他们*会*看到马桶下的那张蜘蛛网，那张你过去七个月踩过的蜘蛛网。是的，他们会查看水槽塞子周围的霉点。他们会知道你那条皱巴巴的手巾有段时间没换过了。他们总是知道。

但是你可以解决这个问题。客房浴室通常是一个小空间，你可以快速梳洗。如果你有 10 分钟，你可以在镜子上使用玻璃清洁剂，擦干净你的水龙头，把几乎空了的卷纸换成新的饱满的，用吸尘器吸掉瓷砖上的毛发，把你的手巾换成干净的。如果你有 30 分钟，天哪，你有时间把你的浴室变成一个豪华的绿洲。你有没有带香味的蜡烛，名字像“火炉边的舒适”或“冬日仙境”的那种？把那个坏男孩赶出去。一个放餐巾纸的小托盘怎么样，这样你的客人就不用去碰其他参加晚宴的人的虱子了？带上它。哦，还有松散的贝壳。这是一个众所周知的事实:一个浴室里放着松散贝壳的人是一个生活在一起的人。

这是你大放异彩的机会。不要浪费时间在入口处整理鞋子。
