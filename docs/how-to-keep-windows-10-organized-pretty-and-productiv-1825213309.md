# 如何让 Windows 10 保持有序、美观和高效

> 原文：<https://lifehacker.com/how-to-keep-windows-10-organized-pretty-and-productiv-1825213309>

微软视窗可能会变得混乱。这不(总是)是操作系统的错。你下载大量的应用程序和文件，并创建自己的新内容，直到你的“下载”目录看起来像旧内容的垃圾填埋场。你的桌面上满是图标，你看不到你漂亮的壁纸。你的开始菜单看起来像一个应用自助餐。简而言之，你的操作系统是一团糟，但它不是不可修复的。

有几个免费的应用程序可以用来为你的 Windows 世界添加一些急需的组织。以下是我们最喜欢的几个:

### [drop it](http://www.dropitproject.com/)T3】

这个 app [我们很早以前就覆盖过](https://lifehacker.com/dropit-sorts-all-your-desktop-clutter-with-just-a-drag-5806828#_ga=2.104885507.1869468182.1523479293-542336946.1523479293) ，但是很值得复活。 [DropIt](http://www.dropitproject.com/) 是一个很棒的实用工具，如果你是那种把你下载(或复制到你电脑上)的所有东西都放在一个文件夹里的人，它可以帮助你保持有条理——一个巨大的、庞大的枢纽，许多文件进入其中，但很少离开。

DropIt 允许你设置一大堆不同的规则，只要你把文件拖到这个工具的小图标上，这些规则就会启动。例如，您可以将应用程序设置为总是将图像文件移动到您的主照片文件夹中，将视频文件移动到您的视频文件夹中，并将 Word 文档移动到您的文档文件夹中。

这只是开始。如果你想变得更高级，DropIt 可以自动扫描文件夹(如你的下载文件夹)，并对它找到的任何东西应用更高级的过滤器，如自动解压缩档案，根据你的参数重命名文件，或压缩大量文件，否则会占用比你想要的更多的空间。

自动化是帮助你在 Windows 中保持有序的一个好方法，DropIt 实际上给了你一个触手可及的虚拟助手。

### [digiKam](https://www.digikam.org/)

如果你庞大的照片库需要一些严肃的组织，但你不想为 Adobe Lightroom 这样的东西付费，开源应用 digiKam 是一个很好的选择。

使用此应用程序对您的照片进行排序，并创建(或编辑)元数据，以便您可以在一个易于访问的库中准确找到您想要的内容。如果你也有点照片完美主义，你可以使用 [digiKam](https://www.digikam.org/) 编辑你的常规和原始照片，使它们成为完美的照片。

这个应用程序是一个更好的解决方案来组织镜头，而不仅仅是将它们倾倒到任意的 Windows 文件夹中。你杂乱无章的硬盘会感谢你，这样你就不太可能丢失(或忘记)图像。

### [launch box](https://www.launchbox-app.com/)T3】

我们不会问你为什么在你的系统上安装了一堆模拟器，我们会假设所有分布在你硬盘的“游戏”部分的文件夹结构中的 rom 都是完全合法的。正确不管怎样，如果你只是花了最后一天下载了成千上万个不同的复古游戏存档，在你的现代电脑上玩，让这些游戏处于控制之下将会感觉势不可挡。

我们建议使用 [LaunchBox](https://www.launchbox-app.com/) ，这是一个很棒的“游戏组织者”工具，可以让你快速找到并播放你巨大库中的游戏。你可以利用该应用的众包数据库，在你的游戏中添加有用的信息，如发布日期、类型、发行商和图像，还可以将某些游戏标记为收藏夹，以便在你有一点时间消磨时更容易找到主题。

LaunchBox 还可以(在某种程度上)方便地从你喜欢的发行服务中导入游戏，比如[【Steam】](https://www.youtube.com/watch?v=-ujUXrggFQo)[【Battle.net】](https://forums.launchbox-app.com/topic/40100-guide-how-to-import-blizzard-games-into-launchbox/)[GoG](https://forums.launchbox-app.com/topic/39474-launching-gog-games/)(仅举几例)。如果你是世界上最大的游戏玩家，玩你能下载的所有游戏，并且总是从所有主要服务中抓取新游戏来尝试，LaunchBox 是在一个数字屋顶下组织你的游戏的一个很好的方式。

### [aqua snap](https://www.nurgo-software.com/products/aquasnap)T3】

大家都知道 Windows 的 Aero Snap 快捷键吧？按 Windows 键+键盘上的一个箭头键，让您的活动窗口在屏幕上飞来飞去:最小化、打开、缩小以填充显示器的四分之一或一半，以及完全弹出您的主显示器(如果您有多显示器设置)。

[AquaSnap](https://www.nurgo-software.com/products/aquasnap) 把这个概念拿来增压。你可以将你的窗口与你的显示器的不同部分对齐，就像 Aero Snap 一样，但是你还可以做更多的事情。

例如，如果你有三个窗口以可爱的平铺方式打开，你可以通过拖动鼠标来调整所有三个窗口的大小——每个窗口都可以根据需要放大和缩小。你的窗口现在可以彼此对齐，而不仅仅是显示屏的角落，你可以将连接的应用程序作为一个大块*移动。*

双击一个窗口的边缘可以让它向那个方向扩展，填满整个屏幕，这是一个很方便的技巧。抓住一个应用程序的窗口，用你的鼠标摇动它——是的，摇动它——将窗口变成透明的，并将其设置为“总是在你打开的任何其他窗口的顶部”。如果你需要使用计算器之类的东西，但又不想在不使用的时候太碍事，这是一个很棒的小功能。

如果你想变得疯狂，你可以使用 AquaSnap 的键盘热键来管理你的 Windows 魔法。你的同事和朋友会对你疯狂的移窗技巧感到敬畏。

### [【tile icon】](https://github.com/Jonno12345/TileIconifier)

如果你是那种用颜色 [排列智能手机应用](https://lifehacker.com/organize-your-home-screen-app-icons-by-color-1823427479#_ga=2.93487645.1869468182.1523479293-542336946.1523479293) 的人，你会爱上 [TileIconifier](https://github.com/Jonno12345/TileIconifier) 。虽然如果你对开始菜单的外观很感兴趣的话，这可能会花费你一些时间，但是这个工具可以让你自定义应用程序的磁贴。

例如，如果你喜欢一种特定的颜色(绿色)，并且对你最喜欢的应用程序的背景颜色与你的色度偏好不一致感到不安，你可以解决这个问题。你可以上传自己的自定义图像，用作中等或小尺寸的磁贴——抱歉，不支持大尺寸或宽尺寸的磁贴——或者你可以简单地使用应用程序的现有图标(缩放至任何尺寸)，并带有任何你想要的背景颜色。如果你想切换窗口主题的话，你也可以制作浅色和深色版本的图标 。

当然，大多数图标的背景应该只是切换到你在 Windows 10 中选择的任何颜色(设置>个性化>颜色)。TileIconifier 是一个很好的方法，可以让更顽固的图标屈服于你的意志，或者，更糟糕的是，用一个可爱动物的不同图像替换你瓷砖上的每个官方图标。(如果你走这条路，一定要给我们发一张截图。)
