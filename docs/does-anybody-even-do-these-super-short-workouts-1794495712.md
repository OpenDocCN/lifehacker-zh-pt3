# 有人做过这些超级短的锻炼吗？

> 原文：<https://lifehacker.com/does-anybody-even-do-these-super-short-workouts-1794495712>

纽约时报有一个新的超级快速锻炼程序，你可以在任何地方做，即使你今天不能去健身房。我试过了，但是...嘿，等等，难道只有我在做这些吗？



理论上，短时间的高强度锻炼似乎是个好主意。如果你还没有开始锻炼，可能是因为你觉得没有时间去健身房。但是有人做这些短时间的锻炼吗？当然，我有九分钟和一些地板空间，但我想让我的工作服出汗吗？如果我要去换衣服，为什么我不出去跑一跑，或者打开 [Sworkit 或者耐克训练俱乐部](http://lifehacker.com/this-7-minute-research-based-workout-exercises-your-wh-498676327) 做些更接近完全锻炼的事情呢？

毕竟， [让《纽约时报》在他们的](https://well.blogs.nytimes.com/2013/05/09/the-scientific-7-minute-workout/?_r=0) [超级短锻炼](https://www.nytimes.com/well/guides/really-really-short-workouts) 剧目上开始 的七分钟锻炼有着古怪的历史。这是发表在科学杂志 上的 [，但这并不意味着任何科学家都测试过它是否容易坚持，或者它是否给出了承诺的结果。相反，它是利用先前研究的线索设计的，旨在以足够快的速度给人们一个迷你力量锻炼，以算作有氧运动。设计师是一家](http://journals.lww.com/acsm-healthfitness/Fulltext/2013/05000/HIGH_INTENSITY_CIRCUIT_TRAINING_USING_BODY_WEIGHT_.5.aspx) [企业健康学院](https://www.jjhpi.com/) 的教练。换句话说，你老板付钱让你锻炼的人。

我们喜欢谈论这些短时间的锻炼，但我没有看到任何证据表明人们愿意进行几分钟的力量或循环训练。新的九分钟锻炼在本月早些时候取消了，但我搜索了社交媒体，发现几乎没有人热衷于或抱怨进行这种锻炼的感觉——只是一篇接一篇地分享它存在的事实。

### 那么，九分钟锻炼怎么样？

首先，九分钟的锻炼实际上需要 11 分钟。我只是想让你知道。不过，有九个练习，每个练习一分钟。结构如下:

*   一分钟深蹲，一分钟俯卧撑，一分钟爬山。
*   休息一分钟。
*   一分钟前臂板，一分钟跳跃劈腿深蹲，一分钟单腿胯桥。
*   休息一分钟。
*   一分钟打嗝*加俯卧撑*，天啊，硬核。一分钟的单脚脚趾接触。抬腿一分钟。

这似乎是一个不知道自己想成为什么样的锻炼计划。NYT 将它标榜为一个力量项目，但随后表示，它是根据“高强度间歇训练原则——即 H.I.I.T。”设计的

它最终成为了一个两者都没有完成的混合体。如果你想通过做分腿深蹲让你的腿 疲劳来增强力量，你会在一分钟内完成。但是如果你试图把练习延长到一分钟，你将不得不以低于 HIIT 要求的速度进行。

这就是这次训练的过程。一分钟对某些锻炼来说太长了，对另一些来说又太短了，总的来说感觉不像是力量训练或者间歇锻炼。不过，就它本来的样子来说，还是不错的！一些挑战你身体的小事情，比坐在办公桌前或沙发上同样长的时间要好得多。不过，如果这是你的目标，你可以找到比这个尴尬的 [更好的锻炼计划](https://lifehacker.com/the-best-workouts-for-your-first-day-back-after-a-break-1790629478) 。
