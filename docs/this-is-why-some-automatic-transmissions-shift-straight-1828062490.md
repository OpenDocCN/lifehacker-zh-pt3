# 这就是为什么有些自动变速器是直的，而有些是曲折的

> 原文：<https://jalopnik.com/this-is-why-some-automatic-transmissions-shift-straight-1828062490>

如果你曾经带着鄙视或真诚的感激看着安装在地板上的自动变速器换挡杆，你可能会注意到它们往往分为两大类:一类是通过 PRNDL 系列的简单线性路径，另一类是通过齿轮使用复杂的迷宫般的路径。为什么在这些设计上有两种截然不同的思想流派？有机械上的区别吗？一个比另一个好吗？这是怎么回事？

我认为简单的线性换挡模式是更受欢迎的选择，但迷宫般的自动变速器——有时被称为“门控换挡器”——也并不罕见，已经被梅赛德斯-奔驰、丰田、斯巴鲁、捷豹(使用了特别疯狂的 J 形门)、通用汽车等各种汽车制造商采用。

那么，门控自动换挡器有什么优势呢？*有优势吗？我曾经想知道是否有一些成本节约与之相关，比如迫使司机以更复杂的方式移动操纵杆可能会节省变速器内的一些机械零件，但我没有找到这方面的证据。*

*此外，第一辆车，我认为使用门控自动似乎是 1963 年奔驰 SL 级。在这一点上，我可能大错特错，但至少它确实是早期采用者。事实上，这可能是由梅赛德斯推出的，这似乎表明，削减成本可能不是门控自动背后的主要动机。*

*因此，为了尝试弄清楚这一切，我联系了专家 Franco Cattani，他是 FCA 的自动变速器工程师。他很友好地给了我一个非常详细的回答(强调我的):*

> **棘爪*弹簧的位置和客户偏好是两个因素。一个例子是以简单线性门致动器为例。它们中的一些，但不是全部，在换档器和缆线通向的任何地方，或者就在换档器组件内部，包括*棘爪*弹簧。在一些原始设备制造商的电缆连接换档到公园系统(我知道福特这样做)。例如，切诺基的换挡杆带有自动驾驶模式。当你驾驶这款车时，无论你是否在移动，你都可以将你的旋钮(不需要按住换档旋钮上的按钮)推到空档，然后回到驾驶，甚至到自动驾驶，你想来回多少次都可以。*
> 
> *通过移动旋钮，你可以移动棘爪弹簧的位置，这将向 ECU 发出信号，并让汽车知道你处于什么模式(空档、驱动、自动驾驶)。很多自动变速器都是这样工作的，不管是什么模式。这就是为什么自动挡汽车有“换挡感”，因为你实际上是在移动卡簧。当你想停车、倒车或开车时，移动旋钮的物理动作实际上是移动零件。*
> 
> ***一些原始设备制造商可能会有一个问题，即在移动时将汽车从驱动位置移动到空档太容易了，这可能会决定他们选择让您移动旋钮来实现车辆换档的模式。**它们可以让你翻上翻下，防止意外将车推到空档。我知道捷豹有换挡器，因为它们的左侧允许您将车辆锁定在 3 档、2 档或 1 档，为了防止意外触发，它们会让您四处移动，这是它们的原始设计(所以再次回到客户偏好)。*
> 
> *这是原始设备制造商正在争论的一个大问题，因为两者都可以被认为是“正确的答案”。所以我的简短回答是，这取决于车辆是否是线控驻车/换档，以及客户的偏好。"*

*好吧，这是很多信息，但是，从根本上说，我们可以去掉这个:没有真正的技术原因来解释为什么自动机器人必须使用这种或那种方法。使用直线齿轮模式或更复杂的门模式的决定与 Franco 所说的“客户偏好”有关，或者，重要的是，OEM 可能决定换档器太容易移动到不需要的档位的粗体部分。*

*这似乎是门控自动挡存在的根本原因:具有之字形、迷宫式换档模式的自动挡在每个档位之间设置了硬停止，迫使操纵杆在每个档位停止时暂停，这防止了意外地将换档器置于错误的档位。*

*我们都有过这样的经历:驾驶自动档汽车，注意到它的转速似乎比正常情况下高了一点，却发现我们把操纵杆卡在了 2 档，而不是 D 档，不是吗？我确实有。门控自动挡虽然有时使用起来很痛苦，但不允许出现这种错误。*

*此外，门控自动系统似乎也能在一定程度上节省成本。在大多数线性自动变速器中，需要在杆上有一个按钮/弹簧/棘爪型装置来锁定倒档，这样就不会被意外选择。门控自动车，由于其设计更加简单，不需要这个。*

*用于换档器的冲压金属或塑料蛇形门比换档器本身中的整个按钮组件更便宜且更容易制造。*

*所以，让我们在这里总结一下:门控自动变速器存在的原因是为了帮助司机绝对确定他们选择什么齿轮。这是一个相对便宜和容易的故障保险，以防止选择错误的齿轮，这是，让我们诚实地说，并不困难，与传统的线性换档。*

*在我们的电传时代，其他类型的变速器重新引起了人们的兴趣，如按钮和旋钮，虽然大多数爱好者都不喜欢，但似乎也有助于解决这些问题。*

*虽然我不得不承认，我发现这些门控自动装置有时使用起来很麻烦，但我确实喜欢它们奇怪的、几乎像生物或地质一样的模式。也许这就足够让他们留下来了。*
