# 介绍后代

> 原文：<https://lifehacker.com/introducing-offspring-1795705380>

欢迎来到《后代》:这个网站是为父母、准父母、父母的朋友、或者只是喜欢阅读婴儿食品混合技术或时间表技巧(这里不做评判)的孩子般的灵魂而设的。后代是 Lifehacker 的智慧、节省时间、好主意和天才技巧的宝库，它将使您的孩子——无论他们是无助的孩子、小胡子青少年还是介于两者之间的孩子——的照顾和喂养不仅更容易，而且实际上是一种享受。

“为什么是育儿技巧？”你可能会问。除了父母，谁更需要一些实用的建议，让生活变得更简单，让事情做得更快，让你的孩子在不诉诸威胁的情况下刷牙的简单方法？

今天做父母的会被如此多善意的建议、相互矛盾的智慧和制造恐惧的声明所淹没，以至于导致瘫痪。从头到尾，你都要服从每个人的意见:*现在是 60 岁以下——你的宝宝应该戴手套！你应该给那个孩子喂奶！一个月后护理只是炫耀！你什么意思，你的幼儿园还没开始读书？等等，你的孩子*不是*三语吗？应该有人告诉你的青少年剃掉那可笑的小胡子，而那个人就是我！*无论你走到哪里，你都会发现有人认为“肚子痛时间”是自由媒体制造的一个情节，或者疫苗导致了过分的无礼。很多时候，你和那个人有关系。

### **你会在后代身上发现什么**

关于如何让你的生活变得更轻松的聪明、实用的想法。你会找到最好的方法来抱着、打嗝和包裹婴儿，这是有科学依据的。乐高玩具、安抚奶嘴和暖湿巾的非品牌用途。和你的孩子谈论大事的有效方法(为什么是贾里德·库什纳？)和小(这就是我们用除臭剂的原因)。基本的，没有废话的建议，关于如何结束挑食，战胜操场恶霸，并成功地执行谈话。

### **你在后代身上找不到的东西**

我们保证不刊登名人的“肚腩”照片——我们发誓，我们甚至永远不会使用“肚腩”这个词不会有关于我们应该如何享受每一次拥抱的文章，不会有对为人父母的奇迹的赞歌，也不会有对为人父母的恐怖的咆哮。这并不是说这些东西没有它们的位置，但是后代是 Lifehacker 的一部分，也就是说它与服务有很大关系，关于寻找聪明的解决方案，使你的生活更容易，更有效率，更好。
