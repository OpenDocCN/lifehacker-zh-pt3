# 将此内容告诉 Siri，以获得机智的回应

> 原文：<https://lifehacker.com/tell-siri-this-for-a-witty-response-1797172897>

自推出以来，Siri 的虚拟助手袖子里藏了一大堆笑话。如今，一个新的说法正在网络上流传:告诉 Siri“我看到一个男人的小剪影。”去吧，我会在这里等你。



这是一个有趣的把戏，可以在今天下午喝酒时或明天早上在饮水机旁展示给你的朋友们，尤其是女王的粉丝们。

如果你喜欢这个，这里有一些其他的俏皮话(他们的官方称呼)值得一试:

你会说唱吗？

你会 beatbox 吗？

零除以零是多少？

狐狸说了什么？

给我读一首俳句。
