# 关于你孩子学校的正念项目，你应该知道些什么

> 原文：<https://lifehacker.com/what-to-know-about-mindfulness-programs-in-your-kid-s-s-1819713137>

越来越多的学校将正念加入到学生的日常生活中，通常作为快速冥想休息。它们似乎有助于一些孩子缓解压力，但它们是否有效背后的科学是复杂的。

[Vox 报道](https://www.vox.com/science-and-health/2017/5/22/13768406/mindfulness-meditation-good-for-kids-evidence) 好处可能只是让孩子们从忙碌的日程中休息一下，但即使是这样，也不妨让孩子们休息一下。也就是说，对于一些学生来说，正念可能弊大于利:也就是说，那些患有精神分裂症或最近有创伤经历的学生。

据 Vox 报道，如果你孩子的学校或班级正在考虑一个正念项目，这里有一些你可能想问的问题:

> 正念指导是否正在取代另一种潜在的更有价值的活动或资源？例如，我们知道身体健康对身心健康至关重要。正念程序占用了健身的时间吗？还是数学指导？还是读书？在学校的日子里只有这么多时间。
> *   How much does this project cost? Where does the budget come from? Is there a strong evidence base for the mindfulness practice used by teachers, such as mindfulness decompression (MBSR)?
>     *   Is it used to control behavior or reduce anxiety?
