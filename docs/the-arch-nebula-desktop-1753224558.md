# 拱形星云桌面

> 原文：<https://lifehacker.com/the-arch-nebula-desktop-1753224558>

新年是我们最喜欢的桌面创造者之一 [dobbie03](http://dobbie03.kinja.com/) 的好时机。在这里，他运行的是 Arch Linux，虽然看起来没什么事情，但让我们看看他工作时是什么样子，以及他是如何设置这一切的。



Dobbie03 对他的设置是这样说的:

> 我有一个默认 Gnome-Shell 主题的编辑版本。我所改变的只是旧金山显示字体的默认字体，我真的很喜欢那个字体。带有 Musiq 主题的 Covergloobus 是我桌面上唯一增加的东西。我喜欢空空的桌面。

我们也是。如果你想看看小工具启动并准备好工作是什么样子，看看下面这张照片:

如果你想把它变成你自己的，你需要做的是:

*   [壁纸](http://hdwpics.com/space-patterns-hdw2059811?tegs=space+pattern&pref=&next=floral-patternin-space-hdw499887)
*   [Deepin GTK 主题](https://aur.archlinux.org/packages/deepin-gtk-theme/) 自定义窗口和布局
*   用 [顶点图标](https://aur.archlinux.org/packages/vertex-icons-git/) 代替系统图标，他结合 [陆丹蓝图标](https://github.com/moka-project/moka-icon-theme) 使用
*   桌面上的 [Covergloobus](https://launchpad.net/covergloobus) 实用程序，用于控制当前播放和音乐
*   [Musiq 主题为 Covergloobus](http://iacoporosso.deviantart.com/art/Musiq-covergloobus-328107466) 定制的播放器

当你坐下来工作时，最终的结果看起来干净而空旷——精神振奋，这很好。然后，到了该做事情的时候，它就活跃起来了。Dobbie03 解释说，当没有最大化窗口打开时，他甚至让标题栏透明，当有最大化窗口打开时，则切换到纯深灰色以更加突出。好东西。如果你喜欢你所看到的，一定要去他的 Kinja 博客(链接如下)分享你的爱。
