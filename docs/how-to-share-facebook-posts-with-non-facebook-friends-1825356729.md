# 如何与非脸书朋友分享脸书帖子

> 原文：<https://lifehacker.com/how-to-share-facebook-posts-with-non-facebook-friends-1825356729>

尽管脸书很普遍，但并不是每个人都使用社会服务。也许他们讨厌社交网络，或者他们对脸书持续的隐私感到失望，或者他们不了解技术。在本周的《科技 911》中，一位读者询问他们如何在脸书(多少)有围墙的花园之外分享内容。

Watch

在一封电子邮件中，Lifehacker 的读者 Amy 写道:

*“我是*[*【kidpost.net】*](https://www.kidpost.net/)*的订户，这是一项非常棒的服务，可以与不使用脸书的朋友和家人分享我在脸书的帖子。(它也适用于 Instagram、Twitter 和 Flickr。)他们月底就要停工了，我找不到替代的。那正是我向你寻求帮助的时候，但是我在 Lifehacker 也没有看到任何东西。我认为，在#删除 facebook 的这个时刻，这项服务可能会变得更加有用。我不会那样做，但我的许多朋友会这样做。”*

最显而易见的解决方案是在脸书上公开分享你的内容，并给你的朋友和爱人一个你个人资料的链接。然而，这可能不是你(或大多数人)愿意去寻求的解决方案，因为你可能不想与世界分享你生活中更多的个人细节。

既然你提到了 kidpost.net，我想你也在寻求与不在脸书的其他大家庭成员和朋友分享关于你孩子的信息和照片。我冒昧地猜测，大多数父母都不想在社交媒体上曝光他们孩子的生活，让任何人看到。

即使你使用的服务即将消失，你仍然可以将你发布到脸书的信息分享给不在脸书的人。我能想到的最简单的方法是设置一个 IFTTT 触发器，它会自动向任何人发送一封电子邮件，其中包含您发布的新帖子的内容。

### 如何使用 IFTTT 通过电子邮件发送你的脸书帖子

点击 [IFTTT](https://ifttt.com/) 并注册一个账户(如果你还没有的话)。进入网站后，点击搜索按钮，输入“脸书”，然后将你的账户连接到 IFTTT。对“电子邮件”也这样做，尽管我也建议使用“Gmail”服务将你的 Gmail 帐户连接到 if TTT——Gmail 帐户会使这个过程容易得多。

完成后，点击“我的小程序”，然后选择“新建小程序”点击“this”链接并选择脸书服务。现在，您将看到许多可以用来(最终)发送电子邮件的触发器。例如，如果你想确保你在脸书上更新的每一条纯文本状态都被发送到一个电子邮件地址，选择那个。如果你想在发布照片时只发送电子邮件，选择它。你甚至可以设置触发器，只有当你在你的文本或照片帖子中放置特定的标签时才会激活:比如#forgrandmaandgrandpa(或者比它更短的东西)。

无论您的偏好是什么，选择它，配置您需要设置的任何选项(如标签)，然后在最终出现的“如果那么那个”屏幕中点击新的蓝色“那个”链接。

如果你没有 Gmail 账户，你可以使用“电子邮件”服务在脸书触发时给自己发邮件，然后你可以在你选择的电子邮件服务中设置过滤器，自动将这些邮件转发给一群人，比如你不在社交媒体上的年长的亲人。

如果你有一个 Gmail 账户(并按照建议将其连接到 IFTTT ),你可以使用 Gmail 服务同时向许多人发送你的脸书帖子。选择它，选择“发送电子邮件”，然后键入您想要发送的地址。

我建议在“收件人”栏中填写你自己的电子邮件地址，并将其他人都添加为密件抄送人，这样你就不会陷入一场回复风暴。你可以调整正文的参数，为你要发送的内容添加一个非正式的介绍，或者让正文保持原样(只要你已经告诉你要发送邮件的人你在忙什么，这样他们就不会突然收到随机的脸书帖子)。

单击“创建操作”，单击“完成”，您就完成了！现在，无论你设置了什么动作作为脸书触发器，比如发布一张照片，它都会自动以电子邮件的形式将你的帖子内容发送给任何人。抵制用 800 张你的猫的照片来轰炸你家人的冲动。
