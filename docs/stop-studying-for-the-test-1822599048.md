# 停止为考试而学习

本周精选来自塞内卡。在他的*道德书信(*[*88.7*](https://en.wikisource.org/wiki/Moral_letters_to_Lucilius/Letter_88)*)*中，他描述了从荷马的*奥德赛*中吸取的真正教训:

你会提出这样一个问题，“尤利西斯流浪到了哪些地区？”而不是时刻努力防止自己误入歧途？我们没有闲暇听讲座的问题是他是否是意大利和西西里之间的海上，还是在我们已知的世界之外(事实上，这么长时间的流浪不可能发生在狭窄的范围内)；我们自己也遭遇精神风暴，它每天都在折腾我们，我们的堕落驱使我们陷入困扰尤利西斯的所有疾病。对我们来说，从来不缺少诱惑我们眼睛的美，或者攻击我们的敌人；这边是以人血为乐的野蛮怪物，那边是耳朵的奸诈诱惑，那边是海难和各种各样的不幸。相反，以尤利西斯为榜样，告诉我如何去爱我的国家，我的妻子，我的父亲，即使遭遇海难，我也要朝着这些目标扬帆远航，尽管它们是可敬的。

### 这意味着什么

从古至今，学者和学生们都在努力找出并记住荷马史诗中的所有名字、日期和地点，但所有这些细节都无关紧要。仔细看看这条线:

> 相反，以尤利西斯为榜样，告诉我如何去爱我的国家，我的妻子，我的父亲，即使遭遇海难，我也要朝着这些目标扬帆远航，尽管它们是可敬的。

对塞内卡来说，这个故事的真正教训是道德教训。我知道，令人惊讶吧？*奥德赛*不是关于独眼巨人怪物，特洛伊木马，或乘风破浪；这是关于爱，荣誉，诱惑，傲慢和毅力。

### 从中吸取什么

回到学校，老师可能会就一本书、一次讲座、一次旅行或一部电影的细节对你进行测验，看看你是否真正注意到了这些材料。这是一个有时有效的系统，但是它在我们的头脑中灌输了错误的教育方法。我们不再关注一个故事的真正教训——为什么会这样——而是学会了记下谁、什么、何时、何地。我们学会了“为考试而学习”如果你还没有，是时候停止了。

我们大多数人现在都离开了学校，不再有考试让我们学习。你该学会只为自己而学习了。当你读书、看电影、看戏剧时，想办法把角色学到的东西应用到你自己的生活中。除非你打算成为一个智力竞赛节目的获胜者，否则没有必要记住那些不会让你成为一个更好的人的小细节。
