# 让驾驶更安全、更轻松、更有趣的最佳安卓应用

> 原文：<https://lifehacker.com/the-best-android-apps-to-make-driving-safer-easier-an-1755443280>

你最不应该花大量时间打电话的地方是在汽车方向盘后面。幸运的是，许多应用程序旨在帮助你安全地查找信息和处理路上的干扰。

Watch

### **在回家的路上找到加油站、餐馆和杂货店**

最近的一次 [**谷歌地图**](https://play.google.com/store/apps/details?id=com.google.android.apps.maps) 更新 [增加了甜美的新驾驶功能](http://lifehacker.com/unlock-google-maps-new-driving-mode-in-the-latest-updat-1752679889#_ga=1.169317220.431406394.1415821409) 。这项功能在你开车时显示周围地区的地图，就像大多数 GPS 装置在不处于导航模式时一样。你也可以用它来搜索附近的加油站、餐馆、杂货店以及谷歌能为你找到的任何东西。

作为奖励，当你处于驾驶模式时，你可以说“好的，谷歌”来激活语音搜索，这样你就可以不用手就找到你要找的地方。当你找到你想去的地方时，谷歌会读回你的结果，并提示你更多的命令，如“方向”,以激活导航模式。

### **获得恶劣驾驶条件的提醒，或提交您自己的提醒**

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-y_7yoEUrVhw&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-y_7yoEUrVhw&start=0) 

交通糟透了。由此可见，避开交通是最好的。谷歌地图有一些交通警报，让你知道什么时候交通不好，但如果你真的想要终极导航应用，试试 [**瓦兹**](https://play.google.com/store/apps/details?id=com.waze) 。Waze 是一个社交驾驶工具，允许你和其他用户报告交通警报。该应用程序跟踪你的驾驶，并向你显示使你的通勤变得更糟的拥堵、事故和施工。如果你想变得更主动，你也可以让应用程序知道你什么时候看到会减慢流量的东西，这样其他人就会被通知。预警之外，Waze 是一款自带 的 [蛮杀手导航 app。](http://lifehacker.com/how-to-turn-waze-into-the-ultimate-navigation-app-1740833171#_ga=1.130914634.431406394.1415821409)

### **获得驾驶主页屏幕，其中包含基本任务的简单快捷方式**

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-Mbuj2J8klJo&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-Mbuj2J8klJo&start=0) 

汽车仪表盘应用程序多如牛毛，但 [**AutoMate**](https://play.google.com/store/apps/details?id=com.bitspice.automate) 脱颖而出，很大程度上是靠 [把 Android Auto 风格的界面带给用户](http://lifehacker.com/automate-leaves-beta-brings-android-auto-style-dashboa-1750077703#_ga=1.92222303.431406394.1415821409) 这才没有为 [一个昂贵的头单元](http://www.amazon.com/Kenwood-eXcelon-DDX9902S-Multimedia-Receiver/dp/B00XK5GC2M/ref=sr_1_2?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-best-android-apps-to-make-driving-safer-easier-an-1755443280&asc_source=&ie=UTF8&keywords="android auto"&qid=1453910075&s=car&sr=1-2&tag=kinjalifehackerlink-20) 。AutoMate 可以显示您即将到来的日历事件，阅读和回复收到的信息，并管理您的音乐控制。如果你将手机用作音响和导航应用，并在开车时接收信息，这是一个方便的主屏幕，它甚至可以与谷歌地图很好地匹配。

当然，AutoMate 的缺点是你需要点击屏幕来获得一些功能。如果你想尽量少看屏幕，[**Drivemode**](https://play.google.com/store/apps/details?id=com.drivemode.android)为你正在使用的应用程序提供了一个免提覆盖。你可以用 wave 屏蔽或接听电话，盲目滑动以启动音乐应用程序并控制播放，以及为给你发短信的任何人创建自动回复信息。

### **找到停车的地方，记住你把车停在哪里**

对于城市司机来说，找到一个停车的地方不是一件容易的事情。 [**最佳停车场**](https://play.google.com/store/apps/details?id=com.bestparking&feature=search_result) 能不能 [帮你找到附近价格最优惠的停车场](http://lifehacker.com/best-parking-helps-you-find-nearby-parking-options-1674033206) 这样你就不会开着车兜圈子，希望能买到比你每月保险还便宜的停车场。当你准备回家时，Google Now 已经做得很好了，它会记住你把车停在哪里，并帮你找到回家的路。

### **大声朗读收到的通知**

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-b8bt7vPr4AY&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-b8bt7vPr4AY&start=0) 

上面的几个应用程序已经可以在你的短信进来的时候阅读了，这很好。如果你需要更强大的功能，[**ReadItToMe**](https://play.google.com/store/apps/details?id=robj.readit.tomefree)会读取你收到的每一个通知。你可以选择朗读哪些应用程序(因为你并不真的需要它在 Spotify 每次更改歌曲时朗读)，并过滤掉某些你不想听到的联系人。对于付费用户，还可以用语音回复消息。

### **将手机变成仪表盘摄像头**

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-Cj9i5h8GFcw&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-Cj9i5h8GFcw&start=0) 

大多数时候你可能不需要仪表板摄像头，但下次你遇到事故或 [流星飞过](https://www.youtube.com/watch?v=dpmXyJrs7iU) 时，你会很高兴你有一个。如果你想在没有整个 dashcam 单元的情况下尝试一下，[**camon road**](https://play.google.com/store/apps/details?id=com.camonroad.app)可以用手机的摄像头记录你的驾驶。它可以将视频保存到手机的本地存储中，但该公司还向注册用户免费提供 2GB(或约 3 小时的视频)的云存储。

### **插入你的车载诊断系统，显示你的汽车的大量信息**

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-EPAl9qdz9XA&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-EPAl9qdz9XA&start=0) 

我们已经讨论了有用信息的 [宝库](http://lifehacker.com/how-to-unlock-a-treasure-trove-of-useful-data-from-your-1563366446#_ga=1.123163214.431406394.1415821409) 你可以通过插入你的汽车的车载诊断端口来获得。通过一个简单、廉价的 OBD-II 适配器，你的手机可以读取你的汽车行驶速度、你使用的汽油量，甚至可以读取仪表板警报。我们最喜欢的应用程序之一 [Dash](https://play.google.com/store/apps/details?id=com.dashlabs.dash.android) 使用这些信息来跟踪你的路线，给你如何节省汽油的建议，甚至存储你的汽车的有用信息，如你的车牌号码和 VIN。如果你真的想了解汽车内部数据的本质， [**扭矩**](https://play.google.com/store/apps/details?id=org.prowl.torquefree) 为你提供了大量的诊断工具。
