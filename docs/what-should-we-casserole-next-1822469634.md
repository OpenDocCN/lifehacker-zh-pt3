# 接下来我们应该做什么菜？

> 原文:[https://life hacker . com/what-should-we-砂锅-next-1822469634](https://lifehacker.com/what-should-we-casserole-next-1822469634)

大家好，欢迎回到 [*将它砂锅*](https://lifehacker.com/tag/will-it-casserole) ，在这个栏目里我们看到哪些饭菜、菜肴、菜系可以重新想象，改造成砂锅。

Watch

我们上一个陀螺灵感的创作完全是一个醉酒发烧梦，但现在是时候继续前进，寻找更多的砂锅灵感。如果说上周证明了什么的话，那就是没有我的帮助，你们完全有能力想出好主意，但我有几个想法可以让你们开始:

*   美食广场砂锅菜:我会一直提到这件事，直到你们对此感到兴奋为止。告诉我你最喜欢的美食广场，这样我们就可以一起做这个怪物了。
*   橄榄园砂锅:我们可以用一道菜来完成“意大利之旅”。我也仍然很喜欢红龙虾砂锅的主意，因为我只是想找个借口去买切达湾饼干。
*   **芝士盘砂锅:**正如我上次解释的，这是一种用水果和坚果烘烤的大型布里干酪。
*   超级碗砂锅菜:我想象着许多蘸酱和许多布法罗酱，但我愿意听听“超级碗砂锅菜”对你意味着什么。
