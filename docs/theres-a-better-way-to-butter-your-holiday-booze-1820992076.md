# 有一种更好的方法给你的节日酒涂上黄油

> 原文：<https://lifehacker.com/theres-a-better-way-to-butter-your-holiday-booze-1820992076>

热黄油朗姆酒在理论上是如此之好，但在实践中，它总是令人失望。啧啧有声地从你的上嘴唇上抹去一层黄油有点过分，即使对我们这些像吃香蕉一样吃一条黄油的人来说也是如此。



谢天谢地，著名的鸡尾酒制造商大卫·勒博维茨有这个诀窍。将你选择的酒用黄油浸泡 3 天，然后取出(保存！)酒黄油筏，用一个衬有咖啡过滤器的细网筛过滤乙醇。像所有最好的注入，这留给你两个伟大的成分:棕色黄油注入酒和注入酒的棕色黄油。

大卫让你做了一种老式的棕色黄油波旁威士忌——听起来很棒——但这种技术也适用于黑麦威士忌或优质的深色朗姆酒。就我个人而言，我会坚持喝威士忌，要么做一杯曼哈顿，要么喝一杯加冰的蛋奶酒，因为我是一个喜欢甜食的温柔、细腻的孩子。至于那醉醺醺的黄油排，我会把它放在一边，直到我有一只肋眼需要涂上黄油。
