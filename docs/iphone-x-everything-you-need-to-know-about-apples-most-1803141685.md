# iPhone X:关于苹果多年来最有趣的手机，你需要知道的一切

> 原文：<https://gizmodo.com/iphone-x-everything-you-need-to-know-about-apples-most-1803141685>

这是正式的。苹果十周年纪念版 iPhone 终于来了，它被称为 iPhone X。这款手机的边框基本上被取消了，它的屏幕更大了。回归多年前的设计元素，新手机有一个玻璃背面，这是自 iPhone 4s 以来我们从未见过的。



iPhone X 将有太空灰和银色可选，和去年的型号一样，它既防尘又防水。

对于那些不常到库比提诺围墙花园外面看看的人来说，iPhone X 不仅比以前的 iPhone 有所改进，而且由于无线充电、面部识别和令人惊叹的有机发光二极管显示屏等新功能，它还进一步缩小了苹果手机和旗舰 Android 设备之间的差距。

### 设计:超大屏幕，回归玻璃

iPhone X 的 Super Retina 显示屏配备了 OLED 面板，而不是之前苹果手机上的液晶显示屏，不仅比以前更加丰富多彩，对比度也更高，分辨率为 2436 x 1125，高于之前的任何 iPhone。然而，最引人注目的是 iPhone X 的超薄无边框设计，这使得苹果可以将高分辨率的 5.8 英寸显示屏装入比 5.5 英寸的 iPhone 8 更小更紧凑的机身中。

在苹果无法消除的微小边框的屏幕外部，是微小的黑色边框，似乎有助于模糊屏幕结束和手机其余部分开始的地方。一些 iPhone 忠实用户对屏幕顶部弯曲在 X 听筒周围的小切口表示异议，但我认为这是一个令人耳目一新的新设计方向，只是让 iPhone 最大限度地利用其显示屏。

### 规格和特点:更好的性能，但我的主页按钮在哪里？

在性能方面，苹果新的 A11“仿生”芯片具有新的六核架构，其中两个核心用于提升重量，另外四个核心用于提高能效。话又说回来，iPhones 在速度方面从来没有懈怠过，所以我们需要一些深入的实践时间来看看 A11 的性能提升是否真的显而易见。

无线充电的加入是一件大事。这不仅是苹果第一次将它包含在 iPhone 中，也是对未来的认可和对日常可用性的帮助。随着苹果 AirPods 的成功，不难想象 iPhone X 可以在整个生命周期中不插任何东西。同样不言而喻的是，苹果没有把耳机接口带回来。

至此，你们中的一些人可能想知道苹果传统的 home 键在哪里。答案是它消失了，部分原因是 X 的无边框设计，这使得屏幕下方没有触控 ID 传感器的空间。但是不要太难过，因为尽管是第一款没有 home 键的 iPhone，苹果已经加入了新的 3D 面部识别技术，名为 Face ID，可以通过看到你的脸来解锁手机。当你想真正回家时，你必须从屏幕底部向上滑动，而不是点击一个按钮。

与其他手机上的类似设置不同，苹果的面部扫描技术使用多点红外传感器，以确保简单的面部照片不会让可疑的人进入。事实上，苹果公司表示，其 Face ID 传感器被其他人的杯子愚弄的可能性只有百万分之一，该系统非常智能，它甚至可以判断出你何时直视它，或者只是看向它的方向。

### 摄像头:苹果的双摄像头甚至更好

与 iPhone 8 Plus 一样，iPhone X 也将配备 1200 万像素的双后置摄像头，这两个摄像头都带有光学图像稳定功能。这将允许真正的 2 倍光学变焦以及苹果新的 beta 人像照明照片模式，该模式创建一个深度图来将主体从背景中分离出来，然后调整前景和中地之间的照明以获得最佳效果，这一切都是实时的。

### 这些是什么鬼东西

显然，老派的、不动的表情符号已经不够好了，所以苹果公司正在升级游戏，推出由你的脸驱动的新表情符号，或 Animjois，这将真正使你的信息变得栩栩如生。
