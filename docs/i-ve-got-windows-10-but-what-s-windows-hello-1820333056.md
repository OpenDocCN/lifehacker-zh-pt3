# 如何使用 Windows Hello 的面部识别来保护您的电脑

希望你利用了微软的免费升级服务，该服务允许消费者将运行 Windows 7 或 8 的电脑升级到 Windows 10(如果你没有， [)还不算太晚！](https://lifehacker.com/get-your-free-windows-10-upgrade-before-the-end-of-the-1820215794) )。如果你有，你应该利用 Windows 10 中最方便和最令人愉快的功能之一:只需看着你的电脑，使用 Windows Hello 就可以登录。这很容易设置，但根据您目前拥有的电脑和附件，可能会花费您一些钱。

### **什么是 Windows Hello？**

微软对其人体扫描登录方法的统称叫做 Windows Hello。它允许您通过使用生物特征(如指纹扫描、虹膜扫描和面部识别)来授权您访问您的帐户，从而避免输入密码。(请注意，要使用安全功能，您很可能需要为您的计算机配备一些附加附件，如兼容的网络摄像头或指纹扫描仪。)

Windows Hello 也可以在某些 Windows 10 应用程序中工作，让你无需密码即可登录你的 Dropbox 或 OneDrive 帐户。不幸的是，Windows Hello 并不像人们希望的那样无处不在，而且很少有应用程序支持该功能。

用你的脸作为密码并不新鲜，但这项技术在过去还不足以作为一种安全措施。照片、视频、面具，甚至彩绘的面孔 [都能够骗过面部扫描技术](https://gizmodo.com/man-its-still-so-easy-to-fool-facial-recognition-secur-1692220368) ，这要归功于它对一个基本摄像头的依赖。Surface Pro 或 iPhone X 等设备中发现的改进扫描技术现在将传统相机与集成的红外投影仪(有利于在弱光条件下识别)和深度传感器相结合，以便正确识别哪个人脸是真的，哪个是假的。

我一直在使用 Windows Hello 登录 Surface Pro 笔记本电脑(从微软借来的)，对结果印象深刻。作为一个戴眼镜的大胡子黑人，我敏锐地意识到，嗯，面部识别与 [不白的人](https://gizmodo.com/how-apple-says-it-prevented-face-id-from-being-racist-1819557448) 有着千丝万缕的联系。我很高兴地说 Windows Hello 很少让人失望。在极少数失败的情况下，Windows Hello 会尝试再次扫描您的面部，并提供如何调整您的位置的建议。如果再次失败，它会要求您输入您在设置时选择的安全 PIN，并让您关闭锁屏并继续工作。

### **如何启用 Windows Hello**

打开你的设置应用，选择登录选项(或在任务栏的搜索框中输入“登录”)。在那里，选择面部识别下的设置。你的网络摄像头会启动，你会看到，嗯，你的脸。将你的头放在蓝色方框的中央，直到扫描完成。

扫描后，您需要输入一个 PIN 码，以防 Windows Hello 使您的登录尝试失败。如果您发现自己经常遇到登录问题，您可以选择“改进识别”来再次扫描您的面部。这是一个非常简单的过程。

### **你可能需要一个新的网络摄像头**

没有拐弯抹角:如果你想使用 Windows Hello 的面部识别功能，你可能需要购买一些新硬件。Windows Hello 需要一个“RealSense”摄像头，这是英特尔对其面部和手势识别技术的术语。不幸的是，它们并不便宜。罗技唯一的 Windows Hello 友好型网络摄像头 [售价近 200 美元](https://www.amazon.com/Logitech-BRIO-Conferencing-Recording-Streaming/dp/B01N5UOYC4/ref=sr_1_7?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/i-ve-got-windows-10-but-what-s-windows-hello-1820333056&asc_source=&ie=UTF8&keywords=realsense&qid=1510324914&sr=8-7&tag=kinjalifehackerlink-20) 。还有 [更便宜的可选](https://www.amazon.com/LilBit-Recognition-Camera-Streaming-Conference/dp/B06Y2WLNFN/ref=sr_1_1?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/i-ve-got-windows-10-but-what-s-windows-hello-1820333056&asc_source=&ie=UTF8&keywords=windows hello&qid=1510324937&sr=8-1&tag=kinjalifehackerlink-20) ，但不多。无论如何，这是升级老旧网络摄像头的一个理由。

如果你使用的是配备网络摄像头的笔记本电脑，不支持 Windows Hello，这可能不是最方便的登录方式(我的意思是，你不会随身携带 USB 网络摄像头吧？).

### **你也可以使用你的指纹**

虽然 Windows Hello 在其品牌中使用了笑脸，但面部识别并不是它支持的唯一技巧。有了兼容的指纹扫描仪，或者微软的 [看起来不舒服的内置指纹读取器的键盘](https://lifehacker.com/microsofts-new-keyboard-includes-fingerprint-login-and-1796167859) ，无论你涂了多少 facepaint，你都可以保护你的设备。你需要一个兼容 Windows Hello 的指纹识别器，它比兼容的网络摄像头更便宜(从 25 美元到 50 美元不等)，也更有效。

您可以设置 Windows Hello 进行指纹扫描，就像启用面部识别一样。您将被要求将手指放在扫描仪上几次，然后输入 PIN。
