# Instagram 模特可以教你一两件关于照片中性感的事情

我们大多数人看起来不像 Instagram 模特(我们都有那些半出名的人，他们有源源不断的完美无瑕的 Instagram 姿势)，但这并不意味着我们在照片中不能像他们一样好。你只需要 [准备一些锦囊妙计](https://lifehacker.com/how-to-look-better-in-photos-based-on-your-body-type-1640221096) 当你的朋友拿出电话说:“我正在写这个！"

Watch

无论你是拍集体照还是单人照，都有一些指导原则要记住，这样你才能在照片中杀死它。

*   走出你的舒适区。拍照时，你可能会有一个默认的姿势，比如伸出舌头或把手放在口袋里。跳出你的舒适区，改变你的姿势，或者至少远离你现在最喜欢的无聊姿势。
*   **去做假动作**。看起来你在做一些事情会比你只是站在那里或者每次都摆同样的姿势更有趣，更吸引人(我在看你们这些女生联谊会的擅自占地者)。但是，你不想以和你实际正在做的事情一样的速度移动(比如走路，跳上你的朋友，调整你的手表)，因为那会导致模糊的照片。这一开始会觉得很尴尬，但是你不会看起来很尴尬。

 [https://lifehacker.com/embed/inset/iframe?id=instagram-BOxMRRmjTEx&autosize=1](https://lifehacker.com/embed/inset/iframe?id=instagram-BOxMRRmjTEx&autosize=1) 

*   [**如果它弯曲，就弯曲它**](https://lifehacker.com/if-it-bends-bend-it-to-look-better-in-photos-1558996391) 。如果你要拍一张静止的照片，弯曲你的胳膊和膝盖，看起来更自然。尽量不要僵硬，这样你看起来就放松了。
*   **自然微笑**。真正的微笑会让你在照片中看起来很棒。如果你或照片中的其他人很难挤出真诚的微笑，我发现会发出放屁的声音(不要真的放屁！)效果不错。太幼稚了...但它完成了任务。

 [https://lifehacker.com/embed/inset/iframe?id=instagram-BHcWX3XgNyR&autosize=1](https://lifehacker.com/embed/inset/iframe?id=instagram-BHcWX3XgNyR&autosize=1) 

*   **请摄影师从中线拍摄**。拍照时，我们倾向于将相机放在面部高度，但这可能不是最佳角度。如果摄影师这样做，请他们蹲下或稍微弯腰，以获得更接近小组腰部的角度。从更高的角度拍摄会让你的身体看起来不相称。

除了这些指导方针，你还可以使用最佳实践来拍摄出色的照片，比如良好的照明( [必要时使用你或别人的智能手机](https://lifehacker.com/use-your-tablet-or-smartphone-as-a-photography-light-so-1719976433) )和 [调整相机的设置](https://lifehacker.com/how-to-take-better-pictures-with-your-smartphones-camer-5662812) 。如果你需要一些姿势灵感，看看你喜欢的音乐家、名人和艺术家的 Instagrams，因为他们的风格和审美会吸引你。
