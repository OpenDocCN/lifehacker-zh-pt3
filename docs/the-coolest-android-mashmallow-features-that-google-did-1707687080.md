# 谷歌没有宣布的最酷的 Android Mashmallow 功能

> 原文：<https://lifehacker.com/the-coolest-android-mashmallow-features-that-google-did-1707687080>

安卓棉花糖终于来了。在谷歌 I/O 上首次公布 [，该公司已经调侃过](http://lifehacker.com/all-the-important-stuff-google-announced-at-i-o-2015-1707454800#_ga=1.96417369.431406394.1415821409) [一堆炫酷功能](http://lifehacker.com/all-the-new-features-of-android-m-1707454646#_ga=1.155023997.431406394.1415821409) 。以下是谷歌没有公布的一些可爱功能。



目前，Android Marshmallow 仅适用于某些 Nexus 设备。如果你还没有收到更新，下面介绍如何 [跳过一行，手动更新](http://lifehacker.com/how-to-skip-the-line-and-manually-update-your-nexus-dev-1726640490) 。对于其他人来说，一定要看看 [我们的设备列表，预计在未来的某个时候会得到棉花糖](http://lifehacker.com/every-phone-thats-getting-updated-to-android-marshmallo-1734676576) 。

### 如果开发者使用，谷歌有一个强大的新应用备份工具

谷歌一直依靠云来永久备份你的大部分数据。然而，一个关键的例外是应用程序数据。如果你卸载了一个应用或移动到一个新的设备，你必须重新改变你所有的设置。虽然该公司已经允许开发人员有选择地将数据备份到 Google Drive 一段时间，但该公司最终还是建立了一个强大的新系统。

对你来说，这意味着你不应该需要像钛备份 这样的 [应用程序来让你的主屏幕恢复正常。当然，它也有一些限制。首先，只备份 25MB 的数据。这应该包括你的设置，但可能不包括你下载的 5GB 的音乐。开发人员也可以选择不包括某些文件，或者完全禁用备份。这是一件好事，因为有](http://lifehacker.com/how-to-set-up-a-fully-automated-app-and-settings-backup-5784857) [某些类型的应用程序和数据](http://lifehacker.com/how-to-practice-safe-sexting-without-resorting-to-digi-698798261) 你*不想*默认备份。然而，这确实意味着仍然有一些东西你必须手动转移。

问题是开发者仍然可以选择退出，而且到目前为止大多数人都这样做了。为了使用新系统，开发人员必须专门针对 API 级别 23(阅读:Android 6.0，外行人)。因此，任何没有更新应用程序的开发者都不会使用备份。此外，如果备份系统对他们不起作用，开发人员可以选择退出。虽然备份系统很棒，但不要指望它能在短期内发挥作用。话说回来，大多数人都需要一段时间才能得到棉花糖，所以下次你的手机升级时，你可能会看到它。

### **您可以手动添加或重新排列快速设置磁贴**

在 Android Lollipop 中，谷歌修改了快速设置阴影，现在，不是包括一堆你不需要的选项，阴影会智能地向你显示你最常用的设置。这是一个很好的特性，但是它也让你时不时地要去寻找一些东西。在安卓棉花糖中，终于可以 [自定义快速设置](http://www.androidpolice.com/2015/05/28/android-m-feature-spotlight-system-ui-tuner-allows-you-to-customize-the-quick-settings-tiles/) 中出现哪些磁贴了。

这是新的隐藏系统 UI 调谐器的一个方便的特性。为了访问它，你需要拉下你的设置阴影，然后长按你的个人资料图片旁边的小齿轮图标。然后你会看到一个提示通知，说系统用户界面调谐器已经添加到你的设置应用程序。打开它，你会看到一个叫做快速设置的选项。使用这个来重新组织你的快速设置阴影。

### **显示您的电池电量百分比并隐藏状态栏中的图标**

另一个很酷的系统用户界面调谐器的功能，你现在可以添加一个数字电池百分比到你的指标在状态栏中。在你的设置应用中打开调谐器，并启用“显示嵌入式电池百分比”切换选项。

您也可以使用系统 UI 调谐器来隐藏系统栏中的图标。您可以隐藏几个图标，包括蜂窝数据、Wi-Fi、飞行模式、闹钟、蓝牙、请勿打扰和热点。其中一些可能比其他的更有用，但是如果你空间不够了，你可以清理掉一些垃圾。

### **在锁定屏幕上向左滑动打开语音搜索，而不是拨号器**

在 [Android Lollipop](http://lifehacker.com/the-coolest-new-features-of-android-l-google-didnt-anno-1596944149) 中，谷歌增加了在主屏幕上向左滑动访问拨号器的功能，提醒所有人手机的存在。显然，没有人真正关心手机，尽管谷歌已经改变了这种行为。现在，当你向左滑动时，你可以 [执行语音搜索](http://www.androidpolice.com/2015/05/28/android-m-feature-spotlight-a-swipe-from-the-left-on-the-lock-screen-now-launches-google-voice-search-instead-of-the-dialer/) 。这可能比打开手机应用程序要方便得多，而且会减少拨号次数。然而，屁股搜索将会显著增加。

### **你的 SD 卡会(终于！)被视为内部存储**

谷歌多年来一直试图忽视 SD 卡的存在，甚至从 Nexus S 开始就没有 Nexus 设备，该公司声称，这种情况的部分原因是因为当你有多个存储卷时，文件管理变得更加困难。通过 Android Marshmallow，谷歌通过允许用户 [将 SD 卡视为更大的内部存储](http://www.androidpolice.com/2015/05/28/android-m-feature-spotlight-external-storage-can-be-adopted-as-true-internal-storage-or-accessed-normally-with-no-additional-apps/) 来解决这一问题。

这一点尤其重要，因为谷歌最近一直在让开发者使用 SD 卡存储 变得更加困难。现在，你可以让 Android“采用”你的 SD 卡，它将受到适用于你的内部存储的相同规则的对待，这应该会给你的应用程序在如何使用你的额外空间方面提供更多的灵活性。换句话说，拥有 SD 卡不应该再有任何奇怪的惩罚。
