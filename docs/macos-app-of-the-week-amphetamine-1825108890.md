# macOS 本周应用:安非他命

> 原文:[https://life hacker . com/MAC OS-app-of-the-week-安非他明-1825108890](https://lifehacker.com/macos-app-of-the-week-amphetamine-1825108890)

去吧，让你的坏笑话从你的系统中消失。我们会等的。一旦你完成，我们建议尽快安装安非他明。这个应用程序是一个超级小的下载，就像给你的 macOS 机器 [咖啡因药丸](https://www.youtube.com/watch?v=bflYjF90t7c) 。一旦激活，你的系统将停止进入睡眠状态。

Watch

“但是等等，”你问。“为什么我不想让我的 MacBook 进入睡眠状态？这样可以节省电池寿命。”

是啊！绝对的。一台令人昏昏欲睡的 Mac 通常是一件很棒的事情。你应该尽你所能确保你的系统在几个小时无所事事的时候不会无谓地消耗资源。如果您使用电池供电，那么当您不坐在电脑前打字时，您绝对希望确保您的系统使用尽可能少的能量。

也就是说，有很多时候你不希望你的系统进入睡眠状态——尤其是当你的屏幕保护程序或者“关闭显示”设置设置得很低的时候。例如，假设您拖着笔记本电脑去开会。如果你的“需要密码”设置对睡眠中的 MacBook 来说很严格，那么每次你想查看笔记时都必须验证你的系统，这将会很烦人。即使他们不是，处理睡眠屏幕的不断开关也可能会令人讨厌。

或者，假设你想在吃午饭的时候打开公司的 Slack，这样你就可以在吃饭的时候看到正在发生的一切。有了安非他命，你只需点击 macOS 菜单栏上的一个图标，就能让你的系统永远保持清醒。

你也可以设置应用程序，这样点击它的图标——我喜欢满的/空的咖啡杯——只会在设定的时间内触发安非他明的能量。

如果你倾向于参加持续一个小时( [或更少](https://lifehacker.com/waste-less-of-your-time-in-meetings-with-the-10-30-50-9-1715014085) )的会议，安非他明会在一段时间后自动失效。你也可以设置它，让安非他明只在它发现特定应用程序正在运行时，当你连接到特定的无线网络时，在设定的时间段内(你最喜欢的午餐会议)，或者每当你系统的电池寿命超过某个阈值(或充电)时才启动。

你可以在热键上使用安非他明，你可以设置它让你的屏幕保护程序启动而不使你的系统进入睡眠状态，你甚至可以在不睡觉的时候定期收到通知。该应用程序为您提供了大量以各种疯狂的方式定制其功能的选项，我们欣赏它的彻底性。我几乎每天都在使用安非他命，对它上瘾是我最大的快乐。

(**奖励:**如果你还没有准备好一路跳到安非他明，并且你的系统更喜欢一种更柔软、功能更少的药物，那么看看[keeping you awake](https://github.com/newmarcel/KeepingYouAwake)——安非他明的前身咖啡因的开源替代品。)
