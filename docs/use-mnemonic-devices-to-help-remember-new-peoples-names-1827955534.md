# 如何记住别人的名字

> 原文：<https://lifehacker.com/use-mnemonic-devices-to-help-remember-new-peoples-names-1827955534>

当你遇到人时，尤其是在某个地方同时遇到很多人，比如一次社交活动，一开始很难记住他们的名字。

加州大学戴维斯分校记忆与可塑性项目主任查兰·兰加纳思认为，对于为什么会遗忘，最简单的解释就是你并不那么感兴趣。在本周 发表的一篇文章中，他向 *Time* 谈到了这个问题 [。你不知道你刚刚遇到的人可能是你新的最好的朋友，相反，现在他们是你在这次互动后可能再也不会遇到的人。为什么要浪费宝贵的大脑空间去记住他们的名字呢？](http://time.com/5348486/why-do-you-forget-names/) 

如果你在记住某人的名字方面有困难，Ranganath 建议尝试一种记忆方法来帮助你记忆。所以，你可能会记得沃尔特是“作家沃尔特”或者“垮掉的拳击手鲍勃”

当你能把某人的名字和其他东西联系在一起时，你以后就更容易回忆起那个名字，你会欺骗你的大脑。他还建议在对话开始几秒钟和几分钟的时候用这个名字来测试自己，并尝试在对话中多次使用这个人的名字来帮助坚持。

如果所有这些都失败了，你还可以继续问那个人的名字。
