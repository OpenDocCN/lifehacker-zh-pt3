# 从此 Mac 壁纸学习外语

> 原文:[https://life hacker . com/this-MAC-wallpaper-teaches-you-a-foreign-language-1822909258](https://lifehacker.com/this-mac-wallpaper-teaches-you-a-foreign-language-1822909258)

有许多在线服务可以帮助你通过日常练习尝试并掌握一门新语言。我目前正在 Duolingo 网站上努力学习法语，试图找回我在学校学了八年，显然已经忘记的法语。

Watch

根据你想学的语言，新的桌面壁纸可以帮助你加快学习速度。

这款壁纸名为 [Backwords](https://backwords.fyi/) ，目前只适用于 Mac 电脑，并且只面向想要学习荷兰语、中文(普通话)、韩语或日语(汉字)的人。

每天，壁纸会从你选择的语言中显示一个新单词。你可以跳过任何你已经知道(或不在乎)的单词，每种语言都有超过 15000 个单词，所以你应该坚持每天一个单词。如果你讨厌疯狂的彩色背景图片，也不用担心，你可以用自己的背景图片替换掉它。

很明显，即使你通过了所有的 15000 题，你也不会流利地掌握其中的任何一个。如果你想全面学习一门语言，我建议你从 Duolingo 开始。

也就是说，我们每天都要盯着自己的电脑看很长时间。每天只学一个单词可以很快积累起来，壁纸可以成为你正在参加的任何其他语言学习课程(在线或离线)的坚实补充。
