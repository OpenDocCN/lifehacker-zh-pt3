# 我们如何工作，2016 年:安迪·奥林的装备和生产力技巧

> 原文:[https://life hacker . com/how-we-work-2016-Andy-orins-gear-and-productivity-tip-1758948893](https://lifehacker.com/how-we-work-2016-andy-orins-gear-and-productivity-tip-1758948893)

当你玩黑客游戏时，时间过得真快，不是吗？又到了一年中的特殊时刻，Lifehacker 的工作人员会告诉你我们的工作习惯和我们用来完成工作的最喜欢的工具。我是安迪，这是我的工作方式。

Watch

去年 我告诉过你我的秘密:我是一个杂乱无章的人。我夸张了。但我确实保持着一种相当即兴的工作风格，因为我不喜欢制定计划，我宁愿只是对正在发生的事情做出反应。我使用的大部分硬件和软件都和去年一样，因为我尽量把东西保持在最低限度。顺便说一句，你们中的一些人一直在问我们正常的 [我是如何工作的](http://lifehacker.com/howiwork) 节目是否会回归。他们会回来的！我只是花了些时间做研发。

**地点:**纽约
**当前位置:**life hacker 的编辑助手
**最能描述你工作方式的一个词**:反应式
**当前移动设备:** iPhone 5s。理想情况下，我试图让它持续到 iPhone 7，但电池开始褪色。总的来说，我并不热衷于手机的具体化。
**当前电脑:** [联想 Yoga Pro 2](http://gizmodo.com/lenovo-yoga-2-pro-review-flipping-fantastic-1471231347) (英特尔酷睿 i7-4500U，8GB 内存，配有非常棒的 QHD+触摸屏)在家运行 Windows 10，在工作时运行 MacBook Air(也是 i7，8GB 内存)。我对空气很着迷，不是因为它的移动性，而是因为过去几年我一直在使用 2011 款 iMac 相比之下，这是一辆破旧的租来的车。

#### 你离不开哪些应用、软件或工具？

很多工作都在 Gmail 中完成。除了回复电子邮件，我还在草稿中做笔记，列出一天要做的事情，诸如此类。我不使用任何插件或任何花哨的“实验室”东西——除了撤销发送！我保留了 20 秒的时间来取消发送信息，我一直都在使用它。

我们在内部使用谷歌文档。好吧。每当我自己写东西(相对于编辑别人的作品)时，我经常从 Evernote 开始。它感觉不像谷歌文档的空白页那么吓人，更像是思想的沙箱。

虽然我通常让 [Adium](https://adium.im/) 或 [Pidgin](https://www.pidgin.im/) 在 IMs 的后台运行，但很多都被 [Slack](http://slack.com) 所取代。我们在闲暇时讨论故事和标题，当然，白天也只是闲聊。Gawker Media 的所有网站都使用 Slack，每个网站都有自己的房间，正如你可能想象的那样，在一个聊天平台上拥有一家媒体公司所有神经质和机智的个性会引发一些愚蠢的对话。“傻”可能是一个慷慨的委婉说法。

[TweetDeck](http://lifehacker.com/how-to-take-your-timeline-back-from-all-of-twitter-s-we-1758279646) 是我整天盯着看的必要附属品。我有点沉迷于信息(因为这是我的工作，所以很方便)，我有科技新闻、记者和同事的专栏。一连串的科技新闻对工作很有帮助，其他的专栏培养了一种无处不在的意识。其中一些有助于引导网站上科技以外的新闻相关性，但我也喜欢在 Twitter 上找乐子。我发推特的次数比以前少了一点——对我来说，这项服务一直主要是一种消费方式——但你仍然可以每天在那里找到我。

我也花大量时间和 [Instapaper](http://lifehacker.com/read-it-later-showdown-pocket-vs-instapaper-1756021479) 一起阅读。我不知道你是否了解纽约，但住在这里意味着你一天 90%的时间不是在等地铁就是在地铁上，感觉就像。地铁时间是读书时间。

#### 你的工作空间是什么样的？

我喜欢让事情变得非常简单。只是在家里有一个便宜的外接显示器的联想，和在办公室有空气的几乎相同的设置。我用的是薄铝苹果键盘。我在工作中习惯了它，所以我也买了一个供个人使用。(在 Windows 电脑上使用苹果键盘会有一些小问题，但不会影响交易。)我不是那种使用机械键盘的人——我更喜欢在笔记本电脑式的小键盘上打字。还有一只 [罗技 M705 鼠标](http://www.amazon.com/Logitech-Wireless-Marathon-3-Year-Battery/dp/B0034XRDUA?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-we-work-2016-andy-orins-gear-and-productivity-tip-1758948893&asc_source=&tag=kinjalifehackerlink-20) 。

#### 去生活帮最省时的捷径是什么？

我尽量不去想。关于现实的大部分！如果你考虑任何事情，几乎所有的生活和工作都会产生压力，因为你真的无法控制任何事情。我尽量不去想它。泛泛而谈，你知道的。我不擅长不思考，但我会尝试。

(显然，我有点开玩笑，但另一种说法是，我试图让后台处理处理大部分工作量。)

#### 你最喜欢的待办事项管理器是什么？

就 Gmail！看到很多未发的草稿让我焦虑，所以这就是我的动力，真的。对抗焦虑可以描述我大部分的生活。所以我尽量不去想！

#### 除了你的手机和电脑，还有什么小玩意是你离不开的，为什么？

伙计，我得告诉你，除了我的电脑和手机，我真的不用任何小玩意。我之前提到过 [气动压力机](http://lifehacker.com/dropping-the-drip-how-to-get-started-making-better-cof-5778831) ，我现在还在日常使用。我喝相当多的咖啡。顺便说一下，我并不特别喜欢高端的单一产地咖啡。品尝豆子精心种植的土壤的微妙味道？不了，谢谢，伙计！

就耳机而言，我非常喜欢 [Koss Porta Pros](http://lifehacker.com/the-best-headphones-you-can-buy-under-20-1480550409) 。我非常吝啬，所以 40 美元的耳机是我愿意花的最高价，但它们真的很棒。我也喜欢随身听的美感。

哦，你知道吗，我最近很喜欢黑翼帕洛米诺铅笔 。如果你不是一个超级铅笔迷，一个真正的铅笔迷，我真的不建议你买一堆铅笔，但是我喜欢它们。它们主要是为了好玩；我有时画画:

#### 你在日常生活中的哪些方面比其他人做得更好？

我想我有独特的幽默感。你看到我们拍的一些视频了吗？ 我有土豆般的屏幕魅力，但我认为他们很有趣。(主要感谢 Devin Clark 和视频团队，是他们让这一切成为现实，而我和 Madeleine 只是在镜头前闲逛。那场演出发生在剪辑阶段。)

也许我很擅长谦虚，如果你认为这是一种技能的话。

#### 你工作的时候听什么？

最近我一遍又一遍地听约翰·科尔特兰的《巨大的台阶》。几周前，我从未真正听过柯川的歌，但我无法释怀。对我来说，一个人能如此擅长一件事简直是疯了。它帮助我不假思索地打字。当倒计时开始时，你会情不自禁地敲击键盘，就好像你在努力跟上他疯狂的号角。

#### 你目前在读什么？

我没读过多少书，如果你想让我对你说实话的话，伙计，我想你想要真正的东西。我最近读了克雷格·汤普森 的漫画小说 [毯子，这是一个可爱、感人的故事，配有精彩的插图。](http://www.amazon.com/Blankets-Craig-Thompson/dp/1603090967?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-we-work-2016-andy-orins-gear-and-productivity-tip-1758948893&asc_source=&tag=kinjalifehackerlink-20)

#### 怎么充值？

天哪，我看了很多电视。我看所有大型 HBO 节目和所有大型网飞节目。全部都是。去年电视上还有什么节目？最好叫索尔，广告狂人的终结，机器人先生。波尔登。我喜欢流言终结者(RIP)和愚蠢的 Top Gear (RIP)。幸存者仍然是一个很好的节目！我喜欢最近首播的由拉什达·琼斯主演的《警察小分队》式的节目——我在一个周末的时间里看了整整一季。老实说。我从未离开过我的公寓。

一边闲荡一边听爵士乐是我的另一个主要爱好。

#### 你的睡眠规律是怎样的？

我经常晚睡，因为我真的讨厌睡眠不足。但我是史蒂芬·科拜尔的超级粉丝，我经常看一些午夜过后才睡觉的深夜节目。

#### 填空:我很想看到 _________ 回答这些相同的问题。

科尔伯特。我也爱柯南。我对深夜脱口秀很着迷。我看约翰尼·卡森的 DVD。这是一个很好的生活帮:观看四十年前的话题幽默，你也可以有老人的幽默感。

#### 你收到过的最好的建议是什么？

我反复 [引用柯南的名言](https://www.youtube.com/watch?v=jr_X4w8FdGk) 说你要努力，要善良，那真的是你唯一需要遵循的口头禅。我仍然在思考这个问题，尤其是犬儒主义是如何有害的。当在一个永远在线的媒体周期中工作时，很容易变得厌倦和愤世嫉俗，我试图限制我的眼球转动。一个人如何避免愤世嫉俗？不确定，但是看皮克斯电影有帮助。

#### 你还有什么想补充的，读者和粉丝可能会感兴趣的吗？
