# Squarespace 全新的西村工作空间

> 原文：<https://lifehacker.com/squarespaces-brand-new-west-village-workspace-1787489474>

Squarespace 让任何人都可以很容易地建立一个网站，但在这项服务的背后，实际上是一个由富有创造力和才华的人组成的大团队。该公司刚刚搬进纽约市的新公寓，它们看起来棒极了。这是他们新工作区的一些照片。



Squarespace 的新住所将美丽的美学与舒适、放松的会议和谈话公共空间结合在一起。实际的工作空间不会吸引任何实际上不喜欢开放式办公室布局的人(因为这是一个非常 T2 的开放式布局)，但它看起来确实是一个每天工作的舒适地方。上面的照片是他们的一个混合空间的视图，就在桌子旁边，人们可以在这里聊天，远离开放工作区的骚动，或者溜进一个较小的房间开会。这里有一个混合工作/会议空间，有一面白板墙供头脑风暴使用:

说到那些非常开放的平面图，这张他们的一些办公空间的照片要么会让你想在那里工作，要么会让你想到就不寒而栗:

一位 Squarespace 代表解释了他们空间背后的一些设计动机:

> 虽然大多数科技公司都试图将他们的工作场所与成人游乐场结合起来，但我们采取了一种略有不同的方法。我们的目标是创造一个创造性、舒适性和合作性可以共存的环境。此外，作为一家以专注于设计而闻名的公司，我们希望确保我们的新办公室反映出我们的身份。我们已经将我们的大厅——曾经是一个旧的装货区——改造成了一个入口通道，用来展示当今一些最有才华的艺术家的作品。(Squarespace 的客户埃米尔·卢卡斯(Emil Lukas)是我们空间中的艺术家之一。)
