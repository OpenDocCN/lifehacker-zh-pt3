# TD Ameritrade 现在允许你通过 Tweet 交易

> 原文:[https://life hacker . com/TD-ameritrade-now-let-you-trade-via-tweet-1822833665](https://lifehacker.com/td-ameritrade-now-lets-you-trade-via-tweet-1822833665)

当你想买卖一些股票时，没有必要浪费时间登录一个应用程序或打开你的桌面——现在，如果你有一个 [TD Ameritrade](https://www.amtd.com/newsroom/press-releases/press-release-details/2018/TD-Ameritrade-Launches-Market-Access-and-Investor-Education-Through-Twitter/default.aspx) 账户，你就可以从你的 Twitter DMs 上方便地做到这一点。

Watch

从今天开始(哦， [今天是推出](https://twitter.com/byHeatherLong/status/961707477996843008) 的好日子)，投资者可以在 Twitter 上直接发送消息 [@tdameritrade](https://twitter.com/TDAmeritrade) 或 [@thinkorswim](https://twitter.com/thinkorswim) ，获得市场更新、股票报价和研究，甚至可以查看余额和进行交易(据一位女发言人说，你可以在平台上登录你的 TD 账户)。不是客户的人也可以通过发送 DM 来获取更新和研究。

它是这样工作的。你给其中一个账户发 DM，它会给你市场新闻或类似“401(k)”的信息

你也可以 DM 一个公司名称(如果它被列出)或它的踢球者符号。它会向你发送该公司的报价和其他信息，如关于该公司的推文量，以及聊天主要是正面还是负面的。

如果你有 TD Ameritrade 的账户，你可以买卖股票和交易所交易基金。该公司新兴技术和创新总监 Sunayna Tuteja 表示，你的信息不会与 Twitter 共享，它只是为你提供服务的一个不同的“容器”。下面是该公司提供的一个交易示例:

Tuteja 表示，其目的是为投资者提供信息，让交易更加方便用户。“我们试图帮助你优化投资体验，”她说。“我们希望将这些信息联系起来，让它们与你的生活相关且及时，这样你就知道该怎么做了。”如果你遇到麻烦，机器人可能会找一个人类代表来帮你。

TD 在 Facebook Messenger 和亚马逊的 Alexa 上提供类似的服务。这项服务是免费的，除非你做了一笔交易，在这种情况下，TD 的 [标准佣金是每笔交易收取](https://www.tdameritrade.com/pricing.page)6.95 美元。

这是一种了解你感兴趣投资的公司的有趣方式——机器人提供实时信息和 30 秒钟的视频来通知投资者。就个人而言，我更愿意坚持使用应用程序或访问网站，但图特哈指出，许多投资者使用 Twitter 来了解最新的市场新闻并相互聊天，这使其成为在他们熟悉的媒介中进行买卖的便捷方式。我会小心这太容易了。你不想因为太方便而支付大笔费用。
