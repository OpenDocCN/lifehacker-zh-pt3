# 这家意大利航空公司现在提供免费的罗马中途停留

> 原文：<https://lifehacker.com/this-italian-airline-is-now-offering-free-stopovers-in-1823076442>

计划去欧洲旅行吗？为什么不在永恒之城罗马短暂停留一下呢？意大利航空公司是意大利的旗舰航空公司，目前为乘客提供长达三晚的免费中途停留。



如果你想知道什么是中途停留，请允许我解释。这基本上是一次有意的长时间停留，这样你就可以在转机飞往下一个目的地之前在一个城市呆上一段时间。不过，问题在于:飞往罗马的航班、中途停留以及你飞往下一个目的地的航班，从技术上来说都是一次航班，一张机票。你不是在支付飞往罗马的机票，然后是飞往比如说巴黎的机票——你只是在支付飞往巴黎的机票，而该航班恰好在罗马停留几天。它是这样工作的:

*   你必须从意大利航空公司提供服务的目的地出发。在美国，这是洛杉矶国际机场、JFK 机场和迈阿密机场。( [他们的网站上只说印度、南非、肯亚](https://www.alitalia.com/en_en/offers/stopover.html) ，但是 [Frommer's 已经和意航确认航班可以从他们服务的任何地方始发](https://twitter.com/Frommers/status/961678491241668608) 。)

*   您的整个旅程(所有航班)都必须通过意大利航空公司预订。

*   最终目的地不能在意大利或北美。这意味着如果你住在纽约，你不能购买一张去迈阿密的单程票，在罗马免费停留或类似的事情。 [欧洲、非洲、亚洲、中东的大部分主要城市都有资格](http://flights.alitalia.com/en/) 。

*   如果您在罗马期间想轻装旅行，意大利航空公司将在机场免费寄存您的行李。

*   您可以享受罗马参与住宿的特价优惠(一旦您预订，他们会帮您联系)。

中途停留是为你的旅行增加一个新目的地的好方法，就像假期中的假期，但它们通常会让你额外付费。为了促进罗马的旅游业(和收入)，意大利航空公司将从现在起到 2018 年底提供免费服务。如果你有兴趣， [前往意大利航空公司网站了解更多](https://www.alitalia.com/en_en/offers/stopover.html) 。当你准备好预订的时候，最好是 [拿起电话给他们打个电话](https://www.alitalia.com/en_en/support/contact-assistance/alitalia-worldwide-contacts.html) 把一切都安排好。
