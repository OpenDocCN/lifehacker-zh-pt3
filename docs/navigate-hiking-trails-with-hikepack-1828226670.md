# 使用徒步旅行包导航徒步旅行路线

> 原文：<https://lifehacker.com/navigate-hiking-trails-with-hikepack-1828226670>

iOS:沿着道路导航很容易——只需启动谷歌地图或 Waze——但在林中小径上寻找道路往往需要拼凑 pdf、纸质地图和仅适用于一个公园或系统的应用程序。 [徒步旅行包](https://hikepack.earth/) 旨在改变这种情况。

Watch

Hikepack 使用来自 [OpenStreetMap](https://en.wikipedia.org/wiki/OpenStreetMap) 的数据，其中包括小径和通常的其他特征，如浴室和露营地。在该应用程序中，您将首先选择一个公园，然后您将能够看到该公园提供的内容。该应用程序知道每条路线的总距离和海拔变化，并会估计你徒步旅行会燃烧多少卡路里。

用免费版的 app，可以查看地图；使用“专业通行证”(前七天免费，之后每年 9.99 美元)，您可以跟踪您的位置并获得一些高级用户功能:

*   下载地图以便离线使用
*   计划你自己的路线，看看距离和燃烧的卡路里
*   点击某个功能(如露营地或水站)，然后点击“带我去那里”它会自动规划从您的位置到该功能的路线，并在您接近时倒数距离。

昨天带着 app 进行了一次 [的试跑](https://lifehacker.com/how-to-take-your-running-to-the-trails-for-a-more-sati-1692428030) 。我知道我当地公园里的大多数小径，但要找到一条新的短程小径可能很困难，因为我通常不知道确切的距离，而且在我通常使用的地图上很难也不可能测量。(你打算跑两英里还是五英里？将比例尺与这条曲线进行比较，找出答案。)但有了 Hikepack，我很快就能找到一条我从未尝试过的小路，我知道它有 1.7 英里长。(我同时用一个正在运行的应用程序跟踪了它，没错，完全正确。)

将 GPS 与精确轨迹地图应用程序放在同一个应用程序中是一个游戏改变者。(真正迷路是很可怕的，知道自己在哪里，也知道路在哪里，但没有办法准确地将两者联系起来。)在昨天的跑步中，有一次小路分叉了，我在两边都找不到任何的踪迹，但我能够从应用程序中判断出我走错了路。

也就是说，仅仅依靠数字工具在荒野中导航从来都不是一个好主意。启用 GPS 的应用程序往往会消耗电量，如果电池没电了会发生什么？(体贴的触摸:当你的电池电量低于 40%时，Hikepack 会通知你，并建议节省电量的方法。)在一次严重的徒步旅行或背包郊游中，你应该总是携带纸质地图和指南针，即使只是作为备份。对我自己来说，去当地公园慢跑一个小时，我对这个应用程序感觉很舒服，因为我对公园非常了解，我相信没有它我也能找到回家的路。

到目前为止，该应用程序只有几百张地图，但他们正在快速添加地图并接受请求。当我告诉一名徒步旅行包代表我想试用这款应用程序，但在我附近找不到任何地图时，他们很快添加了我最喜欢的公园，以及我所在州最受欢迎的公园。你可以直接从应用程序中请求新的地图，他们会在两周内完成。到目前为止，他们已经有了美国、英国、瑞士、芬兰和罗马尼亚的公园地图，并计划增加意大利和斯洛文尼亚。
