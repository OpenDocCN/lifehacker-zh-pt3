# 你可能在 DNA 数据库中已经有了表亲

> 原文：<https://lifehacker.com/you-probably-already-have-cousins-in-a-dna-database-1825892173>

一个 DNA 数据库 [通过他的远房表亲上传的数据引导警方找到了金州黑仔嫌疑人](https://lifehacker.com/the-golden-state-killers-dna-was-in-a-publicly-availabl-1825604590) 。现在，群体遗传学研究人员已经计算出*你的*亲属将他们的遗传信息提供给类似数据库的概率。



根据他们的计算 ，在一个 100 万人的数据库中，我们大多数人都有少数几个第三代表亲，如果数据库包含 500 万人，大约有 100 个，而在一个 1000 万人的数据库中有 200 多个。在任何一种情况下，数据库中至少有一个人是你的第四代表亲或更远，这种可能性接近 100%。

执法部门在金州黑仔案件中使用的数据库 GEDmatch 目前拥有大约 65 万条记录。AncestryDNA 大约有 500 万，23andMe 大约有 200 万。( [)这些数字](http://thednageek.com/genealogical-database-sizes-august-update/) 是系谱学家利亚·拉金去年收集的。)

较大的数据库目前没有被执法机构用于 DNA 搜索，因为它们不接受数据文件，只接受样本。但是对家谱的兴趣只会随着时间的推移而增长，所以关注这些数字是值得的。

将这些计算放在一起，很有理由期待我们大多数人在 GEDmatch 上有远房表亲，可能在 AncestryDNA 和 23andMe 上有更近的表亲。

这些数字有很大的局限性。他们假设这些数据库中的人口是总体人口的随机样本，但事实可能并非如此。(我敢打赌，这是白人、富人和摩门教徒。)他们还假设没有近亲繁殖，人们选择伴侣完全是随机的。最后，它们是平均值。有可能只是碰巧你没有任何亲戚对家谱感兴趣，或者相反，你的妈妈和姐姐正在研究家谱，并说服了你所有的近亲参加。

但这些科学家说，底线是，执法部门在公共数据库中找到一个嫌疑人家庭的 DNA 可能根本不是一个幸运的发现，而是一个完全意料之中的发现。
