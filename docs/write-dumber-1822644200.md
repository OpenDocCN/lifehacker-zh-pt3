# 写得更蠢

> 原文：<https://lifehacker.com/write-dumber-1822644200>

有没有注意到微软 Word 向你展示的“阅读水平”统计数据？他们让高年级水平的写作看起来是好的，低年级水平的写作是愚蠢的。但事实恰恰相反。内容营销网站[的创始人沙恩·斯诺心满意足地比较了几本畅销书的阅读水平](https://www.linkedin.com/pulse/how-much-does-reading-level-matter-shane-snow/) 。迈克尔·克莱顿的作品达到了八年级的阅读水平。七年级的托马斯·品钦，第五名的简·奥斯汀，第四名的欧内斯特·海明威，*晚安，月亮*第三名。

Watch

这里可以得出两个教训:

1.阅读水平有点扯淡。七年级学生能比品钦更好地理解迈克尔·克莱顿，初学走路的孩子能理解晚安月亮。

2.无论你在写什么，你都应该试着达到一个较低的阅读水平。

看，阅读水平分数只衡量你的单词、句子和段落有多复杂。它们与你的想法有多复杂，或者你的写作有多好没有关系。好的写作对于一个三年级学生来说很容易记住，或者对于一个大学生来说很难理解。在斯诺的图表中，每个区间都有好的和坏的写法。

除了这一点:如果你为了看起来聪明而试图让你的写作更复杂，你的写作将会更糟糕。如果你让它更简单，它会更好。你不必像海明威那样闲下来；你只需要检查任何复杂的部分都是有原因的。不要使用单词，除非你理解它们。除非你理解分号，否则不要使用分号。如果你想听起来更好，写“更笨”
