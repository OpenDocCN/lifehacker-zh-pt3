# 使用增强现实成为一个更好的艺术家

> 原文：<https://lifehacker.com/use-augmented-reality-to-become-a-better-artist-with-th-1827789651>

画画是我一直希望自己已经掌握的技能之一，但却从未真正掌握。我在艺术方面一直是班上的佼佼者，直到我上了中学，但不幸的是，那是我的巅峰，从那以后就一直走下坡路。

Watch

如果你的徒手技巧没有你希望的那么好，应用程序 [SketchAR](https://sketchar.tech/) 可以帮你完成剩下的部分。

该应用程序使用增强现实来显示你想在表面上绘制的东西，然后你可以通过手机或平板电脑来“追踪”图像。

虽然你当然可以用它在纸上画些东西，但当你试图在孩子的墙上画壁画或商店的标志时，这特别有用(即使你的绘画技巧不差)。基本上任何徒手画可能会有问题的地方。

该应用程序内置了大量模板，对于那些想要真正学习如何画画而不是伪造的人，该应用程序中有一个部分可以帮助你开始画画。

这不会让你一夜之间成为世界著名的艺术家，但它可以帮助你瞬间提升你的水平。
