# 使用这份 2017 年假日运输指南，确保你的礼物准时到达

> 原文：<https://lifehacker.com/this-2017-holiday-shipping-guide-shows-when-to-mail-gif-1820904317>

每个人都知道，当谈到包裹递送时，假期是巨大的。网上购物让你很容易为名单上的每个人准备礼物，直接送到你家门口，当你需要给全国各地的朋友和家人邮寄礼物时，所有这些亚马逊盒子肯定会再次派上用场。

Watch

如果你要邮寄那些跨国包裹，那么你不仅要记住假期本身的日期，还要记住你需要邮寄礼物的最后期限，以确保礼物及时到达。假日运输也特别棘手，时间和天数都有限。如果你等待的时间太长，你可能最终会为运送礼物支付比你当初支付的更多的钱。

同样重要的是要记住，为了假日运输的便利性和可靠性，不是所有的承运人都是平等的。

例如，你可能认为你可以在 21 日两天内通过联邦快递运送圣诞礼物，但从 22 日开始，联邦快递的送货时间有限，24 日或 25 日根本不会送货。你的包裹可能到不了那里。在这种情况下，你应该用美国邮政的优先邮件快递，这样它会在 25 号到达。
