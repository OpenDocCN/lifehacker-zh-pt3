# 创办一个短篇小说俱乐部，而不是读书俱乐部

> 原文：<https://lifehacker.com/start-a-short-story-club-instead-of-a-book-club-1827514968>

大学一年级，朋友创办了一个读书会。我们选了尤利西斯。我们见过一次，我还没读过《T2》和《尤利西斯》。书友辛苦了！他们太难了，以至于博客列出了当 [书友会中没有人读这本书](https://www.bustle.com/p/12-tips-for-hosting-a-successful-book-club-meeting-when-no-one-read-the-book-9607947) 时可以谈论的事情。(“大声朗读对这本书的评论，并谈论它们。”)如果你的读书会真的只是一个闲逛的借口，那也行！但是如果你真的想阅读和讨论一些文学作品，试试短篇小说俱乐部。



### 指望每个人都能完成

阅读一本典型的书友会书籍需要 [小时](https://www.personalcreations.com/blog/how-long-does-it-take-to-read-popular-books) 。对大多数人来说，这需要多次阅读。如果读者在俱乐部聚会的早上还没有开始阅读，他们就完了。有了小故事，他们一个午休就能看完。见鬼，他们可以读两遍。

### 冒更大的风险

因为花的时间太长，读一本烂书很累人。看烂短篇小说就像好吧，谁在乎，下一个。所以你的社团可以尝试更有挑战性的，前卫的，或者有争议的故事。你可以“试用”一个新的作者，而不必承诺写一整本书。

如果一个故事对大多数读者来说并不成功，至少没有人会觉得他们浪费了生命。尤其是当你加倍下注的时候。

### 对折起来

每次聚会你可以读两到三个小故事，但最终你的作业还是比一本书少。从一个作者、一个流派、一个时期或同一选集里挑选多个故事。或者有意挑选对比鲜明的故事。

如果人们只读了一两个故事，他们仍然可以有意义地参与，他们不必担心破坏他们才读了一半的书。

### 免费阅读

大多数当代书籍都要花钱，但许多短篇小说在网上免费出版。这也意味着你可以在手机上阅读它们。以下是我最喜欢的一些:

*   利·亚历山大(Leigh Alexander)如果陀思妥耶夫斯基沉迷于 YouTube 上的“奇怪的满足感”视频，他会这样写《T4》的替身。

*   [《性爱机器人》](http://www.vol1brooklyn.com/2018/02/25/sunday-stories-sex-robots/) 露西·布里茨:其实是关于专业吃披萨的人。布里奇斯像詹姆斯·瑟伯或米兰达·裘丽一样写作。布里奇斯的大部分故事都集中在孤独者和古怪的天才身上，他们有各种各样支持或不支持的男朋友。
*   [【我们科学家】](https://catapult.co/stories/we-men-of-science) 拉斐尔·鲍勃·瓦克斯伯格:当你在谷歌上搜索了露西·布里奇斯的所有小说后，你正在等待她的下一个故事，这会很好地帮你渡过难关。
*   [《小卡尔，和那里发生的事》](https://www.eater.com/2016/11/2/13499170/carls-jr-brother-chris-onstad) 克里斯 Onstad:网络漫画 [*阿切伍德*](http://achewood.com/) 创作者关于兄弟情谊的非虚构作品。
*   柯蒂斯·西顿菲尔德(Curtis Sittenfeld)的《草原妻子》(The Prairie Wife):一个关于我们对童年好友的怨恨，以及互联网如何帮助我们平息这些怨恨的故事。
*   [【夜行者】](http://coloradoreview.colostate.edu/features/the-night-runner/) 克里斯汀·鲁佩尼安:行为不端的女学生在这个故事中恐吓了一名和平队志愿者，这个故事的作者是 [名副其实的](https://www.nytimes.com/2017/12/11/books/cat-person-new-yorker.html) *纽约人*的故事 [【猫人】。](https://www.newyorker.com/magazine/2017/12/11/cat-person)
*   [【unwinder】](http://correlatedcontents.com/?p=1593)作者:迈克尔·卢茨(Michael Lutz):一个以论坛线程呈现的万圣节故事，这是一个非常聪明的反向讲述故事的方式。
*   [【道场和妈妈】](https://www.mcsweeneys.net/articles/the-ashram-and-mom) 杰西·艾森伯格:*社交网络*演员也写小说和幽默，他的系列 [“鲷鱼让我打嗝:一个特权九岁孩子的餐馆评论”](https://www.mcsweeneys.net/columns/bream-gives-me-hiccups-restaurant-reviews-from-a-privileged-nine-year-old) 是经典的麦克斯威尼的。

*   [《魔女之地的记述》](https://theoffingmag.com/fiction/account-land-witches/)Sofia Samatar 著:对于 A·S·拜厄特迷来说，一个关于神话和学术界的席卷性、复调的故事。

*   [【爆头】](https://motherboard.vice.com/en_us/article/ae3agk/headshot) 朱利安·莫蒂默·史密斯(Julian Mortimer Smith):我不是加里·施泰因加特(Gary Shteyngart)的未来主义讽刺作品 [*超级悲伤的真实爱情故事*](https://www.amazon.com/Super-Sad-True-Love-Story/dp/0812977866?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/start-a-short-story-club-instead-of-a-book-club-1827514968&asc_source=&tag=kinjalifehackerlink-20) 的粉丝，但我是乔恩·博伊斯(Jon Bois)的[*17776*](https://www.sbnation.com/a/17776-football)的粉丝，这个关于士兵作为抽动飘带的故事让我想起了这两者。
*   [“继续？肯德拉·福特迈尔(Kendra fort Meyer)y/N "](http://the-toast.net/2015/05/04/continue-yn-a-short-story/):电子游戏中的卖花女拒绝继续做非玩家角色。这就像如果*黑镜*是好的。

### 挑选一个精选系列

短篇小说集。以上几个来自正在进行的在线系列。你可以追随一个你整个团队都喜欢的人。《纽约客》 [每期](https://www.newyorker.com/magazine/fiction) (外加一期年度小说)都会出一篇短篇小说，其档案可以追溯到几十年前。在你达到付费上限之前，你可以在网上免费获得这些故事。

或者订阅 [*一个故事*](https://www.one-story.com/) ，这是一本文学杂志，每月只送你一篇短篇小说。(每三个月你还会得到一个青少年的故事。)

你也可以将一本出版的选集分散到几个讨论中。如果你正在恶补经典名著，试试《诺顿短篇小说选集》[](https://www.amazon.com/Norton-Anthology-Fiction-Shorter-Eighth/dp/0393937763/?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/start-a-short-story-club-instead-of-a-book-club-1827514968&asc_source=&tag=kinjalifehackerlink-20)*。一个多世纪以来，一年一度的*最佳美国短篇小说*选集收集了本年度的最佳小说。 [今年的版本](https://www.amazon.com/Best-American-Short-Stories-2018-ebook/dp/B078FGZ95D?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/start-a-short-story-club-instead-of-a-book-club-1827514968&asc_source=&tag=kinjalifehackerlink-20) 由作者罗珊妮·盖伊编辑。*

*你也可以 [创办一个文章俱乐部](https://cupofjo.com/2014/10/an-articles-club/) ，就像作家乔安娜·戈达德那样。这可以产生活跃的讨论，但它不会像讨论小说那样逃避现实。*

*除非俱乐部里的每个人都喜欢同一个作者，否则你可能要避免浏览一个作者的短篇小说集。但如果你这样做了，花一整年的时间讨论博尔赫斯、伍尔夫或罗莉·摩尔，同时仍有足够的时间进行其他阅读，这可能是一种有趣的方式。*

*或者你可以在下面的评论中给你的整个阵容提出建议。读者们，你们最喜欢和别人谈论的短篇故事是什么？*
