# 我们并不像自己认为的那样思想开放

> 原文：<https://lifehacker.com/were-not-as-open-minded-as-we-think-you-are-1759787196>

我们都喜欢认为自己思想开放，但选举季节暴露了我们中最优秀的一面也暴露了最糟糕的一面。即使是我们中最“开放”和“知情”的人也比我们想象的要封闭得多，他们也会公开承认。让我们看看为什么会出现这种情况，以及我们真正可以做些什么。



传统上，思想开放意味着承认一个想法是真还是假的可能性。我们都会对很多事情封闭自己的思想，这本身并不是坏事。 [一旦一个想法被证明是错误的，我们*应该*抛弃它](http://vitals.lifehacker.com/the-hpv-vaccine-doesn-t-deserve-its-reputation-get-it-1748529758#_ga=1.104677384.968941705.1436971740) 。我们中的大多数人不需要浪费时间去接受事实上错误的东西，对一切事物保持开放的心态意味着你永远无法对任何非事实或虚构的事物形成实际的观点。令人欣慰的是，你可以思想开放，但仍然保留批判性思维，但这需要努力。

### 挑战我们的信念会让我们的先入之见更加强烈

没人想听到他们错了。当有人挑战你的信念时，你应该接受新的信息，重新评估你的感受。然而，反过来通常是正确的。当我们面对新信息的挑战时，它只会让我们原有的信念更加坚定。这就是所谓的 [逆火效应](https://lifehacker.com/why-you-cant-win-that-argument-on-the-internet-5811255) 。

有趣的是，今年在希拉里和伯尼·桑德斯支持者的争论中，我最常看到适得其反的效果。在桑德斯一方，我见过的最流行的观点是克林顿太务实，太受王朝驱动，不值得信任。当有人提出不同的观点时，比如在脸书问题上，桑德斯的支持者会蜂拥而至，加入支持他们观点的链接。克林顿支持做同样的事情，结果是人们无意识地来回争论，只提供观点来加强他们已经形成的观点。我大胆猜测，除了标题之外，没有人会费心去阅读任何一个链接。

我们的先入之见挑战了我们接受新信息的能力。这使得我们很难对新思想保持开放的心态。上面的例子很简单，是关于我认识的一些随机的人的轶事，但它不像这样存在于脸书供稿的真空中。 [你没那么聪明](http://youarenotsosmart.com/) [抛出了几个这种效果的例子](http://youarenotsosmart.com/2011/06/10/the-backfire-effect/) :

> 当科学家们让人们观看 1996 年鲍勃·多尔与比尔·克林顿的辩论时，他们发现辩论前的支持者倾向于相信他们喜欢的候选人赢了。2000 年，当心理学家在莱温斯基丑闻期间研究克林顿的爱好者和憎恨者时，他们发现克林顿的爱好者倾向于将莱温斯基视为不可信任的家庭破坏者，并发现很难相信克林顿在宣誓后撒谎。当然，憎恨者的感觉完全相反。快进到 2011 年，你会看到福克斯新闻频道和 MSNBC 争夺有线新闻领域，两人都承诺永远不会挑战某一部分观众的信仰。保证有偏见的同化。

简而言之很简单:挑战我的信念，我不会批判性地思考你的挑战，而是坐在那里思考我有多正确，你有多愚蠢。当我们看到一些破坏我们信念的东西时，我们会感到困惑，甚至愤怒，但我们很少改变自己的想法。这与文章的基调无关。人们可能会争辩说，如果一种信念以更中立的方式受到挑战，它会导致更好的话语，但事实从来不是这样。一个论点越中立，就越容易被驳回(或者只是翻转到你这边)。

可悲的是，我们在这里实际上并不能做很多事情来解决这个问题。你能做的最好的事情是接受你错了的可能性，等待一场激烈的辩论，或者一篇引起愤怒的文章，然后在你冷静下来后重新评估。

### 我们关注的是对我们直接有益的东西

这可能不足为奇，但我们更关注直接影响我们的问题，而忽略那些不直接影响我们的问题。正因为如此，我们可以说对一个不直接影响我们的问题更加小心翼翼。这发生在很多问题上，尤其是阶级、种族、性别和性取向，但从教育的角度来看，钱是最简单的例子之一。

学生贷款债务是最简单的例子。每一位候选人 [都对过度的学生贷款债务](http://www.nerdwallet.com/blog/loans/student-loans/presidential-candidates-student-loans/) 有自己的立场，以及某种处理这一问题的表面计划。如果你现在有很多学生贷款，你可能会考虑哪个候选人对你最有利。这也可能是你支持某个候选人胜过另一个候选人的重要理由之一。也许你同意 [川普削减教育部资金并确保联邦政府不会从学生贷款中获利的做法。或者也许你更热衷于克林顿以较低利率提供再融资的计划。如果你刚从大学毕业，学生贷款改革可能是你最密切关注的平台，提供一个让你直接受益的项目的候选人将是你支持的人，其他问题暂且不论。](http://thehill.com/policy/finance/248913-trump-why-is-federal-government-making-money-on-student-loans)

当然，另一方面是另一个没有学生贷款的选民，要么是因为他们没有上过大学，他们已经还清了贷款，要么是他们有足够的钱自掏腰包支付大学费用。在这种情况下，学生贷款可能根本不在他们的考虑范围内。事实上，这可能是一个无关紧要的问题，以至于他们不明白为什么这是一个主要的话题。更糟糕的是，学生贷款改革的资金必须来自某个地方，这可能会在某个时候影响他们，所以他们可能有兴趣投票反对你的利益。

同时看到两种观点真的很难。如果你已经努力还清了贷款，或者自己掏钱上了大学，你会觉得被骗了，或者想知道为什么其他人不能这样做(尽管显然有原因)。另一方面，如果你因为一半的薪水用于学生贷款而难以支付房租，你无法想象为什么政府还没有解决这个问题。如果你债务的利息高于贷款本身，那么提供再融资是有意义的。

当然，这是一个难以置信的复杂问题，有很多变量，但我们大多数人都无法看到我们自己可能会带走的好处。

事实是，当我们谈论直接影响我们的问题时，我们经常会有一种自私的观点。这并不是说一个观点比另一个更有效，但这确实意味着当政策直接影响到你时，你很难保持开放的态度。很难把自己从等式中抽离出来，去考虑更大的利益。设身处地为他人着想是一个愚蠢的建议，它总是伴随着思想的开放，但这可能是唯一能帮助你改变观点的方法。

### 此外，实际上没有人会阅读反对意见

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-MZ1cRbkoPus&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-MZ1cRbkoPus&start=0) 

我们之前说过，如果你想打开你的思维， [阅读来自各种渠道](http://lifehacker.com/how-to-avoid-sounding-like-an-idiot-when-discussing-pol-5990295) 的对立观点，围坐在一起反复思考每一个论点，然后形成你自己的观点。让我们诚实一会儿:我们大多数人实际上从来没有这样做过。

当然，我们很多人可能会花时间去看对方的辩论。我们甚至可能让自己相信，我们实际上是在关注对立的观点，而不是在寻找漏洞。但我大胆猜测，共和党辩论的受欢迎程度与其说与开明有关，不如说与壮观有关。人们喜欢看川普做川普，因为它很有娱乐性。如果你同意他，你就会团结在他的身后。如果不是，那就是对着电视大吼大叫的绝佳借口。听着，我同意，对着电视大喊很有趣。生气并对一些政治意识形态大声咆哮是很有趣的。但是，拜托，大多数人没有阅读对立的观点，没有戴上批判性思维的帽子，没有深入思考手头的问题。

适得其反的效果倾向于这样，即使我们花时间去阅读相反的观点，它最终只会强化我们现有的信念。很少有人会反驳说，“看，我带着开放的心态读了那篇专栏文章，这里是我不同意的地方，这里是激发我一些新想法的地方，这里是我想了解更多的地方。”我们大多数人不会带着可能是真或假的想法去阅读一个对立的观点。我们认为它是错误的，或者我们通过阅读别人的观点给自己带来一些自虐的好处，我们用它作为燃料来点燃我们先前存在的观点之火。

解决这个问题的最好办法是闭上嘴，听听反对的意见。不要反应，不要戳孔。你可以想看什么就看什么，但是如果你在谈话，如果你闭嘴听，你就有更好的机会听到对方在说什么。在你用你的观点反击之前，花些时间考虑一下这个新想法。

事实是，以开放的心态参与政治对话是非常困难的。见鬼，在简单的层面上很难。我很久以前就决定我厌倦了超级英雄电影，尽管每个人都好评如潮，我拒绝承认我可能会喜欢死侍。*可笑又没有道理*。但是当我想到自己在这种愚蠢的情况下的封闭思维，它至少帮助我在宏观层面上认识到这些偏见。我只能希望这就够了。
