# 躺在床上如何解锁 iPhone X

> 原文：<https://lifehacker.com/to-unlock-the-iphone-x-in-bed-hold-it-farther-away-fro-1820335701>

[【Face ID】](https://lifehacker.com/what-you-need-to-know-about-face-id-on-the-iphone-x-1804641406)， [iPhone X](https://lifehacker.com/iphone-x-tips-and-tricks-you-need-to-know-right-now-1820086402) 中推出的新型人脸扫描安全摄像头，在大部分时间里工作得相当完美。但是有一个地方苹果的最新技术可能会给你带来麻烦:床上。



最近 Reddit 上的一条(由 [BGR](http://bgr.com/2017/11/10/iphone-x-features-tips-get-face-id-to-work-in-bed/) 发现)显示，许多人在躺在床上时解锁新智能手机时遇到了麻烦。一名 iPhone X 用户认为问题可能是枕头遮住了他们的脸，但事实证明解决方案甚至更简单。

### 如何在床上解锁 iPhone X

问题不在于你脸的位置，而在于你拿 iPhone 的方式。那是因为如果相机离你太近，Face ID 就不起作用。根据 [苹果](https://www.youtube.com/watch?v=OvVKnC6gGtg&feature=youtu.be&t=6m47s) 的说法，它需要离你的脸 10-20 英寸才能得到准确的扫描。

通常这不是一个问题，因为大多数人都把手机放在面前大约这个距离，但是在床上使用 iPhone 就是另一回事了。您可能会将设备握得更近，因为您通常戴眼镜或隐形眼镜，但还没有戴上，或者您可能刚刚醒来，视力有点模糊。无论哪种方式，你的本能可能是把屏幕举到你的面前，但对 Face ID 来说，这不起作用。

习惯这一点可能有点烦人，但这可能是最好的。根据的一项研究，把智能手机拿得离脸太近会导致眼睛疲劳，甚至头疼。所以在这种情况下，可能苹果确实最了解。
