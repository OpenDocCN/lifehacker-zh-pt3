# 为什么你应该学习这个设计原则

> 原文：<https://lifehacker.com/why-you-should-learn-this-design-principle-1829280545>

由于苹果和其他公司对华而不实的产品或演示的狂热，谈论将“设计原则”应用于其他学科变得时髦起来。但是根据 [设计师 Rie nrregaard](https://www.sypartners.com/leader/rie-norregaard/)的说法，我们都应该学习的关键设计课程是许多设计师仍在学习的课程:为“默认”用户以外的人设计。

Watch

诺勒加德的播客 [*为人类而设计*](https://soundcloud.com/designing-for-humanity) (点击)探索了包容性作为设计的一个基本部分。以人为中心的设计，她最近写了 ，意思是问，**“我们如何一次性为一系列不同的人解决【给定的】问题？”**

这意味着要考虑不同性别、种族、能力的人，以及处于不同临时环境中的人。随着设计师学会问和回答这个问题，许多其他职业可以向他们学习:开发人员，企业家，创意人员，任何其工作将被一个以上预先确定的、严格定义的人群使用的人。

学者兼作家 Sinéad Burke 多年来一直倡导包容性设计，但作为《T2》《T3》节目的嘉宾，她承认，直到有人为她的 ted 演讲 联系她 [，她才认为这是一个设计问题。她说，身高 3 英尺 5 英寸的她，在设计这个世界的时候并没有考虑到她。她说:“作为一个社会，我们已经说过，让某些东西变得可访问会破坏它的美感，因为我们把可访问性视为丑陋的东西。”。她每天都生活在这种后果中。她说，最实用的凳子在设计时并没有考虑到成年人的尊严。因此，在家里，她使用一个粉红色和蓝色的卡嗒声凳子，“这不是我在公共场所使用的东西，因为我的尊严和产品的情感从来没有被考虑过。相反，她说，“我实际上只是没有，挣扎或寻求帮助。”](https://www.ted.com/talks/sinead_burke_why_design_should_include_everyone)

这是设计选择的人类结果。诺尔加德告诉 Lifehacker，设计选择不仅仅是由设计师做出的。她强调了程序员越来越大的责任，以及“无意识的意图”的危险，这种危险经常出现在面部识别(对于深色肤色的[)或数字助理(通常](https://gizmodo.com/can-we-make-non-racist-face-recognition-1827639249) [默认为女声](https://gizmodo.com/its-so-stupid-that-digital-assistants-are-women-1821779672) )等技术解决方案中。当设计师不能质疑和解决他们的假设，他们的“默认”时，问题就出现了(正如资深 iPhone 工程师 Ken Kocienda [最近告诉 Lifehacker](https://lifehacker.com/stop-misusing-autocorrect-1828972260) 的那样，这就是为什么科技公司需要雇佣和提升更加多样化的员工队伍:当决策者几乎都是身体健全的白人男性时，他们往往无法预见其他所有人的需求。)

这些决策也出现在各种非技术角色中。诺勒加德告诉《生活黑客》，“设计在塑造事物的过程中占据了一席之地，而这在十年前还是没有的。”。 [嘉宾劳拉·麦克拜恩](http://famous),[斯坦福 d.school K12 实验室](https://dschool.stanford.edu/programs/k12-lab-network) 的联合主管，来自教育规划的背景，她正在利用自己的角色寻找每个层次的设计解决方案，寻找“小颗粒实验”来提高每个学生的表现，小到像重新安排课桌这样的选择。她承认，有时她仍然不好意思称自己为设计师；她听到同事们称自己为“只是一个老师”和“只是一个校长”，她认为设计的语言是这些教育者拥有自己决定的影响力的一种方式，并意识到他们所拥有的权力，而不仅仅是遵循通过更大的系统传递下来的规则。当系统存在缺陷，或者只为“默认”学生设计，而没有意识到教师和校长所面对的现实时，这一点是至关重要的。

包容性设计不是设计师从高处赐予我们这些非设计师的东西。它的原则和经验来自其他领域。 [嘉宾塔克·维迈斯特](https://soundcloud.com/designing-for-humanity/tucker-viemeister) ，一位工业设计师，帮助设计了OXO“好握柄”厨房工具，他的方法不仅归功于他的工业设计师父亲，也归功于他的社会工作者母亲。“设计是我们对待彼此的方式，”他在播客中说。设计的核心不是让某样东西看起来好或者感觉有创新，而是考虑最终接受者的体验。

“我们现在生活的世界主要是为平均水平而设计的，这意味着它并不真正适合许多人，”rregaard 告诉 Lifehacker。"它也没有在很多时候考虑到我们中的很多人."这就是我们需要改变的。不仅仅是工作头衔中带有“设计”的人，而是我们所有人，他们的决定会影响到其他人。

这甚至不限于那些对大型系统拥有明显权力的人，而是任何不得不弯曲或破坏系统以满足其需求的人，尤其是当这些需求被边缘化时。“生活在‘边缘’的人总是在寻找解决方案。我们有时都会这样，”诺勒加德说。对于 Lifehacker 的读者来说，这是一个好消息:无论何时你修改某个东西以使它更好地适应你的情况，你都是在进行设计。
