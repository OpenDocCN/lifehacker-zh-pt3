# 如何处理你没有订购的亚马逊包裹

> 原文：<https://lifehacker.com/what-to-do-with-amazon-packages-you-didnt-order-1825205083>

你打开门，发现在你的迎宾垫上，有一个明显标记的来自亚马逊的包裹。问题是，你没有从他们那里订购任何东西，经过仔细检查，很明显这次送货不是给你的。你该怎么办？



简单的交货错误时有发生。如果只是地址错误/街道正确的简单情况，并且你知道包裹是给谁的(比如邻居)，你可以步行到他们那里。如果足够小， [放在他们的](https://lifehacker.com/what-do-i-do-when-i-receive-someone-elses-mail-1821882594#_ga=2.7439922.1869468182.1523479293-542336946.1523479293) 邮箱里。如果没有，敲敲他们的门，把它交给他们。否则，就把它放在他们的门前，让运输标签朝上，这样就能清楚地看到。

如果你不知道收件人，那么，亚马逊给了你一份礼物。根据美国联邦贸易委员会，任何邮寄给您但您没有订购的商品都可以作为免费赠品合法保存。这适用于亚马逊 [或任何其他发货的零售商](https://lifehacker.com/you-re-allowed-to-keep-products-retailers-accidentally-1789443602) ，包括邮寄到你地址的任何不正确的物品，即使包裹正确地寄给了你。更有可能的是，如果你联系客服，亚马逊会告诉你保留该商品。希望这是件好事！

好吧，但是说这个东西不是你想要的，或者你肩膀上会说话的卡通蟋蟀不会让你留着它。在这种情况下，你可以联系亚马逊，提出退货。只要确保亚马逊支付退货运费的所有费用即可——他们不这样做是违法的。这意味着你给他们一段“合理的时间(比如 30 天)”来取货和退货，而你不应该为此支付一分钱。如果他们没有及时捡起来，你可以留着或者扔掉。

如果这种情况发生在你身上很多次，并且所有包裹上的名字都是一样的，这可能是以前的租户使用他们旧的默认亚马逊送货地址的情况。随时联系当地的美国邮政检查员，让他们知道那个人已经不住在那里了。你别无选择。所以，要么享受免费的礼物，要么准备好和亚马逊客服聊很多。
