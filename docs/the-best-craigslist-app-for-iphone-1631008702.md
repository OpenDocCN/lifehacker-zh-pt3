# iPhone 上最好的 Craigslist 应用

> 原文：<https://lifehacker.com/the-best-craigslist-app-for-iphone-1631008702>

虽然过去有几十种选择，但 Craigslist 应用程序远没有过去那么流行了。尽管如此，你还是有一些好的选择。如果你经常使用这项服务，我们选择 CPlus。



### [CPU lus](https://itunes.apple.com/us/app/cplus-for-craigslist-app-mobile/id955286870?mt=8)

**平台:** iPhone
**价格:**免费
[**下载页面**](https://itunes.apple.com/us/app/cplus-for-craigslist-app-mobile/id955286870?mt=8)

### 特征

*   发布、编辑和浏览列表
*   多种颜色主题选项
*   一次搜索多个城市
*   在网格、相册、地图或列表中显示搜索结果的各种选项
*   保存您经常查找的搜索，并为新列表设置通知
*   地图搜索附近的列表
*   灰色显示你已经看过的物品

### 它擅长的地方

CPlus 并不华丽，但它工作可靠，有你需要的基本功能，让你访问 Craigslist 的重要部分，而不用让你处理糟糕的 Craigslist 移动网站。

你可以用 CPlus 做你在普通的 Craigslist 上能做的一切事情，这意味着发布、编辑和浏览。它还增加了一些新功能，比如用关键字通知新的列表，方便以后保存列表，以及为保存的列表添加注释的能力，当你比较不同的项目时，这很方便。

### 不足之处

CPlus 是可服务的，它非常好，但是它并没有真正增加 Craigslist 的体验。如果你正在寻找一个应用程序来管理 Craigslist 的大量使用，超出了基本的 web 应用程序所能做的，你不会在这里找到。当你看到一个实际的列表时，它基本上只是 Craigslist 移动网站的一个重新制作的版本，这使得你很难通过应用程序本身联系到某个人。

### 竞争

[**Qwilo**](https://itunes.apple.com/us/app/qwilo-craigslist-mobile-app/id942337227?mt=8)**(免费/99)是对 CPlus 最直接的竞争。就功能而言，Qwilo 和 CPlus 非常相似，但是 Qwilo 在设计上更现代化一些。有一个方便的夜间模式，每个列表上都有一个大的联系按钮，Qwilo 针对 iPad 和 iPhone 都进行了优化。然而，Qwilo 将保存的搜索和通知锁定在 1 美元的付费墙后面。这并不多，但是事实上 CPlus 包括了这两个免费的东西，这就使它领先了。除此之外，这两者之间差异主要是审美上的，所以如果你不喜欢 CPU 的话，这是值得一看的。**

**你唯一可用的其他选择是[](https://itunes.apple.com/us/app/csmart-for-craigslist-free/id797077334?mt=8)**(免费/$2.99)，这也是非常过时和难看的。不过在功能方面，它可以与 Qwilo 和 CPlus 相媲美。如果出于某种原因，你不喜欢 Qwilo 或 CPlus 的设计，那么也许 CSmart 值得一看。****
