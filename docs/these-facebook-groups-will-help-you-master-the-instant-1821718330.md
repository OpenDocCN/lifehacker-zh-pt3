# 这些脸书小组将帮助您掌握速溶壶

速溶壶是这个假日季节赠送和获得的厨房玩具，但它们的黑盒氛围一开始可能有点吓人。如果你对所有这些按钮和功能有点不知所措，只要深呼吸，拥抱一个速溶壶脸书集团的“数字安全”方法。

像任何有狂热追随者的设备一样，有大量的在线群组，从模糊的、膨胀的到极其具体的。这些群体非常庞大——希望能多读读“老公”这个词——他们是一般建议、美味小贴士和(当然)食谱的绝佳资源。(你必须申请会员资格才能加入他们中的大多数，但是，如果他们接受我这样的人，你进去应该没有问题。)以下是我们最喜欢的几个:

*   :这个小组非常适合众包食谱，掌握所有这些按钮，缓解你的新手紧张情绪。
*   :每个人都应该多吃蔬菜，你不一定要成为素食主义者才能享受这组提供的所有创造性的植物性食谱。
*   速食锅越南菜食谱:快速、美味的 pho 是你需要加入这个团体的唯一原因，但一定不要太沉迷于 Bo Kho、糯米和奶茶。

*   [印度菜速食锅](https://www.facebook.com/groups/1728640847356719/) :速食锅能快速加工谷物、豆类和味道浓郁的酱料，非常适合做印度菜。作为一个希望戒掉外卖的人，我发现这个团体非常令人兴奋。
