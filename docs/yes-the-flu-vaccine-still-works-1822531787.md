# 是的，流感疫苗仍然有效

> 原文：<https://lifehacker.com/yes-the-flu-vaccine-still-works-1822531787>

今年的流感季节是一个特别糟糕的季节，也许当你听说有些人已经接种了流感疫苗，但仍然感染了流感时，你会感到沮丧。(这种情况每年都会发生，但当它发生在你或你认识的人身上时，似乎更真实。)但是流感疫苗*仍然有效*。以下是你需要知道的。

### 这个季节很糟糕

今年的流感病例、住院和死亡人数 [有望至少与近年来的任何其他季节](https://www.nytimes.com/2018/01/26/health/flu-rates-deaths.html) 一样多。(看起来我们不可能战胜 2009-2010 年的 H1N1 流感，但只有时间能证明一切。)

### 这种疫苗可能有超过 10%的有效性

一些新闻文章大肆宣扬 10%的数字作为流感疫苗只是勉强起作用的证据。但这实际上是来自一份报告，该疫苗对 T4 的一种流感病毒(H3N2 型)和澳大利亚的流感病毒(T7)只有 10%的有效性。如果你想知道这个镜头在美国的效果如何，你需要更多的信息。

[疾控中心报告](https://www.cdc.gov/flu/about/season/flu-season-2017-2018.htm#effectiveness) 去年的美国疫苗含有相同的 H3N2 成分，今年在野外流通的 H3N2 似乎与去年的 H3N2 相似。这意味着我们可以通过考虑这些数字来很好地猜测今年的疫苗有效性。结果显示，去年，该疫苗对 H3N2 的有效率为 32%。

今年我们有很多 H3N2 在流通。它由上面图表中的红条表示。但它不是唯一的流感类型，无论如何，其他类型往往是疫苗的更好匹配(H3N2 是一种特别狡猾的小病毒)。在过去的十年里，流感疫苗的有效性从 19%到 60%不等。今年可能会更低，但现在说还为时过早。

### 但是但是但是…

你现在应该知道这些，但是:

*   不，流感疫苗不会让你患上流感，就像吞下西瓜籽不会让你的肚子里长出西瓜藤一样。只是的[和](https://vitals.lifehacker.com/don-t-let-these-myths-scare-you-away-from-a-flu-shot-1787136173)的不一样。
*   如果你注射了流感疫苗，然后又得了流感，那么恭喜你，你是一个统计数据。但是很多人注射了流感疫苗，而*没有*感染流感，也没有将流感传播给婴儿、老人或癌症患者。
*   注射流感疫苗有助于你对流感做出更好的反应，所以即使你生病了，注射疫苗也比不注射要好。
*   接受 sho t 几乎没有负面影响，这就是为什么它被推荐给从六个月大的婴儿开始的几乎*每个人*。点击 阅读CDC 的推荐。婴儿需要打两针，老人需要高剂量，鼻喷剂没有上市，因为过去几年这种鼻喷剂无效。
*   现在还不算太晚。你当地的药店可能没有流感疫苗，所以去 vaccinefinder.org 找一个还有库存的地方。但是从免疫学角度来说，如果你以前没有注射过疫苗，现在仍然值得注射一针。(如果你*在本赛季早些时候得到了一个，请坐。你完了。)*
