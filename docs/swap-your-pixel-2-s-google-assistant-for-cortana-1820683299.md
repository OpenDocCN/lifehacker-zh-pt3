# 把你 Pixel 2 的谷歌助手换成 Cortana

> 原文:[https://life hacker . com/swap-your-pixel-2-s-Google-assistant-for-cortana-1820683299](https://lifehacker.com/swap-your-pixel-2-s-google-assistant-for-cortana-1820683299)

你的谷歌 Pixel 2 有一个非常能干的内置管家，它的形式是 [谷歌助手](https://lifehacker.com/how-to-change-google-assistant-to-a-male-voice-1819293856) 。然而，并不是所有拥有安卓手机的人都全力投入谷歌的生态系统。事实上，有些用户可能是——等等——微软用户。这意味着 Cortana 是他们的虚拟助手。多亏了微软，你可以抛弃谷歌的语音助手，换上一个更适合你的。

### **如何在手机上安装 Cortana**

要将你的语音助手从谷歌的默认助手 [换成微软的 Cortana](https://lifehacker.com/alexa-and-cortana-teamed-up-but-consumer-tech-is-still-1798642894) ，你需要合适的应用。你可以从谷歌 Play 商店下载 Cortana 。下载完成后，系统会要求您使用 Microsoft 帐户登录(如果您还没有，请创建一个)。

### **从谷歌换到 Cortana**

登录后，你可以按照 Cortana 的屏幕提示，将其设置为你的默认助手。点击“将 Cortana 添加为默认助手”将被重定向到您的设置页面，特别是助手和语音输入页面。在这里，您可以更改您的助手应用程序，编辑您的 Active Edge 设置，并更改您的语音识别首选项。若要手动访问，请访问“设置”>“应用与通知”>“高级”>“默认应用”。

你需要把你的助手从增强的谷歌服务换成 Cortana。请记住，您将授权它访问您智能手机中的应用程序信息，以及任何个人数据，如联系人和日历。

### **使用语音命令**

要唤醒 Cortana，只需按住 Home 键，Cortana 就会开始监听，同时它会向你快速展示你一天的剩余时间。如果你在计划即将到来的会议时有问题，可以在 Cortana 应用程序中打开语音识别。在它的设置页面中，启用“嘿 Cortana ”,这样你和你的新助手就可以不用手，在应用程序内或在设备的锁屏上工作。一个缺点是:使用 Cortana 作为你的主要语音助手会禁用 Pixel 2 的 Active Edge 功能。挤压侧面不会激活 Cortana。
