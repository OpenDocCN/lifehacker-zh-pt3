# 七月书友会公开线程:如何快乐 10%

> 原文：<https://lifehacker.com/july-book-club-open-thread-how-to-be-10-happier-1828028377>

“正念”的好处和获得它的方法早就被宣传了，但致力于这一实践的自助书籍、论坛和应用程序的数量在过去几年里绝对是爆炸式增长。随着科技继续侵入我们生活的各个方面，我们发现很难摆脱这一切，正念向我们承诺了一种逃避，一种通过意向性和接受我们到底是谁而变得更快乐的方式。



这听起来很棒，直到你问自己它到底是什么，为什么它如此难以实现。

那就是，在我看来，成了丹·哈里斯畅销书的症结所在，[](https://www.amazon.com/10-Happier-Self-Help-Actually-Works/dp/0062265431/ref=sr_1_1?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/july-book-club-open-thread-how-to-be-10-happier-1828028377&asc_source=&tag=kinjalifehackerlink-20)**七月书友精选*。哈里斯是一名广播记者和前战地记者——这是一种充满压力的生活，正如他在书的前几章中解释的那样，他内心的批评使事情变得更糟。作为一个住在纽约、碰巧也在媒体工作的人(他一直担心我和我的同事及朋友的工作安全)，我可以理解他的不足感和对改进的渴望，同时又想拒绝同样的感觉和动机。或者，正如哈里斯所写，“我们所有人都努力在我们呈现给世界的形象和我们内心世界的现实之间取得平衡。”***

**对哈里斯来说，正念，佛教意义上的正念，而不是西方版本的正念，是克服所有这些并压制内心批评的关键。正如他所定义的，“正念是一种识别你头脑中正在发生的事情的能力——愤怒、嫉妒、悲伤、脚趾受伤的疼痛，等等——而不会被它冲昏头脑。”这是“被动生活的另一种选择。”听起来不错。**

**在结构层面上，我觉得这本书读起来很有趣(也很快)——我从不感到无聊，哈里斯显然是一个有天赋的故事讲述者。我也欣赏他进入他生活中更混乱的方面，不仅包括他的吸毒，还包括许多小的，看似无关紧要的事情，比如他的发际线。我们都有这样的事情困扰着我们，即使我们知道它们是愚蠢的。**

**在讨论他更认真对待信仰和宗教的旅程时，我也欣赏他的诚实。信仰对于无数人的生活和身份来说至关重要，更多的记者和作家(以及整个人类)尝试理解这一点将会受益匪浅。**

**最重要的是，哈里斯不只是遵从当今大师告诉他的任何事情——他以批判的眼光，在通往正念幸福的道路上不断尝试和犯错。这对我们所有努力保持平衡的人来说都是一个很好的教训。**
