### **特性**

*   基本文件管理功能:复制、粘贴、剪切、创建、删除、重命名、共享和发送存储在 SD 卡或内部存储器上的文件
*   一次选择多个文件
*   在一个位置浏览照片、音乐和视频的集合
*   管理、安装和卸载应用程序，并探索应用程序中的文件结构
*   打开，阅读，提取和解密压缩，7ZIP，RAR 和焦油文件
*   管理 Dropbox、box、OneDrive、Google Drive、Sugarsync、Copy、Mediafire、Owncloud 和 Yandex 上的云文件
*   通过插件为亚魔卓、Mega 等添加更多存储选项
*   使用密码锁定对网络位置的访问
*   使用超级用户访问权限执行仅限超级用户的操作
*   使用正则表达式或变量一次批量重命名大量文件
*   使用 FTP、SFTP、WebDav 和 SMB/CIFS 客户端远程访问文件
*   为文件夹添加书签，以便以后快速访问
*   使用内置媒体播放器观看图像和聆听音频
*   向您的 Chromecast 播放媒体

*   在浅色、深色和黑色背景主题之间进行选择
*   材料设计界面

### **擅长的地方**

固体资源管理器几乎完美地涵盖了你所期望的文件管理器的所有功能。收藏允许您在一个位置查看所有照片、音乐和视频。你可以像 Dropbox 一样连接你的云存储帐户来远程管理你的文件。它还支持远程访问协议，如 FTP，你甚至可以使用它作为一个根文件管理器。即使对于高级用户来说，它也是一记重拳。

除此之外，它还*华丽*。虽然大多数文件管理器的设计停留在 2009 年和石器时代之间的某个地方，但 Solid Explorer 优先遵循谷歌的 [材料设计规范](http://lifehacker.com/updating-all-the-new-stuff-in-androids-l-release-prev-1595928268) 。你也可以自定义你的主题和颜色，甚至可以选择亮色和暗色的主题，因为文件管理器没有理由蒙蔽你。

### **不足之处**

虽然我们更喜欢尽可能选择免费应用，但 Solid Explorer 不属于这一阵营。你可以免费试用两个星期，但之后，你将不得不支付 2 美元才能继续使用。除此之外，该公司还对 Mega 等一些插件收费，甚至提供额外的图标包以获得更多的钱。这很烦人，但大多数附加组件要么是免费的，要么是可选的。好处是你至少知道 Solid Explorer 的资金来源。由于我们的最后一次选择是因为添加了粗略的广告软件，我们将称之为喜忧参半，而不是彻底的负面影响。

### **比赛**

ES 文件浏览器的免费版可能已经失去了我们的推荐，但是如果你不想说再见，你至少应该去看看 [**ES 文件浏览器 Pro**](https://play.google.com/store/apps/details?id=com.estrongs.android.pop.pro) 。只要 2.99 美元，所有的嵌入式应用程序“建议”和垃圾软件都被删除，而且没有广告。它仍然拥有我们过去喜欢的所有功能，包括远程文件访问、ZIP 支持和应用程序管理器。当然，Solid Explorer 中的大部分相同功能只需少花 1 美元。

[**FX 文件浏览器**](https://play.google.com/store/apps/details?id=nextapp.fx) 对于喜欢 ES 文件浏览器设计但不想要垃圾的用户来说也是一个很棒的选择。免费版本涵盖了本地文件管理的基础知识，而你可以支付 2.99 美元解锁高级功能，如云或网络存储和应用程序管理器。对于基础用户来说，免费版本比 Solid Explorer 更有优势，但高级用户最终会为大版本支付略高的费用。不过，在撰写本文的时候，FX 文件浏览器正在进行一次促销，将价格降至 1.99 美元，专门针对 ES 文件浏览器的用户。

同样值得指出的是，如果你对你的文件管理器没有太多需求，Android N 的 [和](http://lifehacker.com/the-coolest-features-of-android-n-google-didnt-announce-1764114846) 也会覆盖你。谷歌已经开始在系统中内置一个基本的文件管理器。您可以复制和移动文件，重命名文件和文件夹，以及创建新文件夹，所有这些都在系统本身中进行。它与列表中其他应用程序的高级功能相去甚远，Android N 甚至还没有发布，但如果你是少数运行 N Preview 的 [之一，或者将来从 Android N 设备上阅读这些内容，你可能不需要完整的文件管理器应用程序。](http://lifehacker.com/how-to-install-the-android-n-developer-preview-on-your-1763813851)
