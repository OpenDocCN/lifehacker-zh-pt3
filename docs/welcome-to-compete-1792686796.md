# 欢迎参加比赛

> 原文：<https://kotaku.com/welcome-to-compete-1792686796>

欢迎来到 *Compete* ，这是一家由 *Deadspin* 和 *Kotaku* 组成的合资公司，致力于报道电子竞技和竞技游戏中最好的、最差的、最愚蠢的、最激动人心的、最有趣的，以及总的来说最愚蠢的——在这个词的更好和更坏的意义上。我们在这里刚刚起步，但希望这里成为你在互联网或其他地方永远不会读到的那种报道的目的地。



为什么 *Kotaku* 和 *Deadspin* 会在这里合作是一个显而易见的问题；我们认为答案同样显而易见。电子竞技是传统体育和游戏的混合体——精彩的亮点和荒谬的暗光来自激烈的竞争，其中物理元素以像素和多边形的形式出现；团队、联盟、出版商和广播公司延续骗局和诈骗；关于规则和公平的辩论；一种新型名人运动员的出现，他们的运动在网上比在电视上播出的多——因此需要混合报道，这种报道是体育网站和游戏网站通过合作可以提供的，它们具有不同的优势和共同的感受力。我们要做的是，越过天花乱坠的宣传，回到真实的故事——不管是愚蠢的、惊人的，还是仅仅是引人注目的。这是一个值得拥有的领域。

这里的团队建立在 Ethan Gach 在周末提供的精彩电子竞技报道的基础上，将由广义上报道竞技游戏的 *Deadspin* 和 *Kotaku* 作家和编辑组成。埃里克·范艾伦和麦蒂·迈尔斯——我们的全职作家，同时为两个网站工作(网站托管在 Kotaku 上，与网络托管技术有关；它在任何意义上都是一家合资企业)——了解你能想到的每一款主要竞技游戏，从 *Dota* 和*英雄联盟*到*街头霸王*和 *Smash* ，从*炉石*和 *CSGO* 到...*凯瑟琳*？当然可以。那也是。我们的专职作家和自由职业者将对他们进行补充，因为他们会挖掘任何涉及游戏和试图获胜的人的东西，从棋盘游戏到赌场等等。

埃里克将在奥斯汀工作，在那里，他作为自由职业者为诸如 *ESPN* 、 *Vice* 、 *Paste* 和 *IGN* 等网站工作，这让他名声大噪。他也有电视背景，这很好，因为竞争计划的一部分是用大量伟大的视觉元素来报道新闻和亮点，涵盖该领域发生的最杰出、最愚蠢、最有趣和最令人惊讶的事情。

麦蒂将在波士顿工作，她最近在波士顿的玛丽苏担任编辑。Maddy 有自己的电子竞技背景，所以她不仅带来了观察者的专业知识，还带来了参与者的专业知识。她将致力于每日报道，同时也在准备更深入的特写。

他们想做什么很大程度上取决于他们对什么感兴趣，但我们主要希望只问正确的问题，从而进入你在其他地方不会读到的那种故事。脑药真的能让你打得更好吗？NCAA 在这个不断发展的领域中扮演什么角色？与顶级网球运动员相比，《英雄联盟》的顶级运动员有多灵巧——灵巧对谁更重要？希望我们能在这里得到一些答案，并找出一些值得发现的东西，无论你是一个对竞争感兴趣的游戏玩家，一个对游戏感兴趣的体育迷，还是一个只对有趣的事情感兴趣的人，我们认为都会有适合你的东西。

两个音符。一个是，虽然这个项目最初是由吉列赞助的，虽然我们很高兴他们这样做，但这对我们的编辑团队来说是完全抽象的，他们已经并将不会与他们有任何联系。竞争和其他 *Deadspin* 和 *Kotaku* 做的事情一样独立，它的报道是它的员工和读者感兴趣的功能。

另一个是我们想要你的建议！看到很酷的东西了吗？知道一些扭曲的腐败吗？只是大致知道一些我们应该知道的事情？你可以发电子邮件给我们在 eric.vanallen@deadspin.com 或 maddy.myers@deadspin.com 的撰稿人，或者我们在 tips@kotaku.com 或 tips@deadspin.com 的全体员工。

前进。
