# 归档网页的最佳工具

> 原文：<https://lifehacker.com/the-best-tools-to-archive-web-pages-1794802605>

最近似乎很多网页从互联网上消失了。如果你想自己承担存档的责任，有各种各样的工具可以做到这一点。



### 将页面提交到回程机器

互联网档案馆是一个非营利性的数字图书馆，试图收集尽可能多的数字知识，包括大量的网页收藏。互联网档案馆的 [回溯机](https://archive.org/web/) 是*查找*旧版本网站最明显的选择，但你可以通过手动保存特定网页来帮助这个过程，包括该页面上的所有资产，这也是毫无价值的。如果你愿意，你也可以下载 PDF 格式的网页。

要保存一个网站， [前往返程机](https://archive.org/web/) ，然后将一个 URL 粘贴到显示“立即保存页面”的框中。你还可以使用 Wayback 机器的浏览器扩展([Chrome](https://chrome.google.com/webstore/detail/wayback-machine/fpnmgdkabkmnadcjpehmlllkndpkmiak?hl=en-US)/[Firefox](https://addons.mozilla.org/en-US/firefox/addon/wayback-machine_new/))来保存你当前正在查看的任何页面，只需点击几次。

### 自己手动下载页面

Wayback 机器对公共利益来说是很棒的，但是如果你想要你自己的个人备份，你会想要考虑一些不同的选择。最明显的方法就是自己下载一个页面或网站。

存档可能是最简单的方法，还可以在线保存网站。将一个 URL 放入 Archive.is，当它完成页面渲染时，您会看到一个下载 Zip 选项，可以下载整个内容以供您安全保存。Archive.is 也将托管该备份，方式与 Internet Archive 非常相似。否则，像 [SiteSucker](http://ricks-apps.com/) 这样的桌面工具也能轻松完成这项工作。

当然，如果你习惯使用命令行， [Wget 是批量下载和镜像站点](https://lifehacker.com/geek-to-live-mastering-wget-161202) 最简单的方法。

### 屏幕截图页面

如果没有必要下载一个站点的原始 HTML 文件，那么屏幕截图可能会有用。当页面的布局或视觉风格很重要时，这也很有用，因为归档网站经常会把事情搞得一团糟。

世界上有大量的屏幕捕捉工具，但是有几个比其他的好。如果你是 Google Drive 用户，那么 [保存到 Google Drive](https://chrome.google.com/webstore/detail/save-to-google-drive/gmbmikajjgmnabiglmofipeabaddhgne) 扩展可以立即将页面转换为截图，然后保存到 Drive。如果你不使用驱动器， [全页截屏](https://chrome.google.com/webstore/detail/full-page-screen-capture/fdpohaocaechififmbbbbbknoalclacl?hl=en-US) 做同样的事情，但是保存到你的本地驱动器。否则，像 Windows 上的 [PicPick](http://ngwin.com/picpick) 或 Mac 上的 [Skitch](https://evernote.com/skitch/) 这样的桌面工具就行了。大多数网络浏览器也有导出为 PDF 的选项，如果你喜欢这样保存网页的话。

### 使用 Notes 应用程序的 Web Clipper 保存页面

如果你使用现代笔记应用程序，如[【Evernote】](https://evernote.com/webclipper/)、 [OneNote](https://www.onenote.com/clipper) 、 [Google Keep](https://chrome.google.com/webstore/detail/google-keep-chrome-extens/lpcaedmchfhocbbapmcbpinfpgnhiddi?hl=en) 或 [Bear](http://www.bear-writer.com/faq/Extensions/Browser%20extensions/) ，那么你就可以使用一个 web clipper 将一页的全部内容保存到你的笔记中。一旦你为你的 notes 应用程序安装了 web clipper 扩展，你就可以保存一个站点，它的内容，可选地包括图片，将被保存。
