# 开始作业'咨询时间'

> 原文：<https://lifehacker.com/start-homework-consulting-hours-1823000261>

作业大战似乎是学龄儿童工作日晚上的一个标志，而父母总是被打败的一方。当我的女儿今年去幼儿园时，我害怕这场争吵。因为*将会有*场战斗...正确会有她不想做作业的时候，我们必须想办法让她做作业，一切都会很困难。我一直在振作精神。



但一本新书让我想到，或许有可能在战争开始前就放下它。在 [*的《自我驱动的孩子:让你的孩子更好地掌控自己生活的科学和意识》*](https://www.amazon.com/Self-Driven-Child-Science-Giving-Control/dp/0735222517?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/start-homework-consulting-hours-1823000261&asc_source=&tag=kinjalifehackerlink-20) *，*中，威廉·斯蒂克斯鲁德和内德·约翰逊认为，与你的孩子为家庭作业争吵不仅给家庭带来巨大压力，从长远来看，这实际上并没有帮助他们。当你，家长，表现得好像确保你的孩子做作业(或练习钢琴或记住学校戏剧的台词)是你的工作时，你在发出信号，他们之外的人要对他们的行为负责。然后，他们失去了做决定的经验，失去了面对这些决定的后果的经验，失去了检查自己以评估当前道路的经验。同样重要的是，你正在阻止你的家成为它真正需要的东西:一个“安全的基地”约翰逊告诉《科学美国人》 ,那里应该是“他们可以去寻求暂时休息的地方，在那里他们感到安全和无条件的爱，在那里他们可以完全放松，这样他们就可以聚集能量重新出去。”

作者建议做一个大的转变，对我来说，这是一个既令人兴奋又令人恐惧的转变。这就是:不要把自己视为孩子的经理，而是像顾问一样为人父母。这意味着当涉及到像家庭作业或钢琴练习这样的问题时，你可以问自己，“顾问会怎么做？”你可以激励孩子，提供建议，回答问题，并在那里，但你必须退缩，给你的孩子导航自己生活的力量。Stixrud 和 Johnson 认为，这不是一种自由放任的育儿方式，你应该设定限制，告诉孩子你担心什么，并“提供一个救生筏”但是“你不是在掌舵，”他们写道。

正如斯蒂克斯鲁德 [告诉 NPR](https://www.npr.org/sections/ed/2018/02/14/584275859/the-key-to-raising-a-happy-child) 的，建立这种家长-顾问模式的一种方法是建立家庭作业“咨询时间”

> 我对父母说的是，如果你决定不再为此争吵，你应该说，“我能帮什么忙？”你认为自己是一名顾问，并尊重地承认这是孩子的作业。你不能让你的孩子做这件事。你能做的就是提供帮助。

> 你可以在晚上 6:30 到 7:30 之间设置我所说的咨询时间，然后直接说:“我不会和你吵架的。我只是太爱你了。我不想要这些摩擦。这是你的工作，我尊重你能解决这个问题，我会帮助你。”一户人家刚刚告诉我，他们家的温度下降了 20 度。

在书中，Stixrud 和 Johnson 写道，你的咨询时间应该明确——你的孩子可以利用它们，也可以不利用。如果她在晚上 9:30 来找你，说她需要帮助，你可以告诉她，“作业时间已经过了。明天 6:30 到 7:30 我都有空。”当她了解时间表时，她可能会变得更加专注。你可以根据需要破例——例如，如果她在做一个大项目或者材料特别有挑战性。

真实的对话:我内心的老虎父母对这个建议感到极度兴奋，尽管我相信它是有价值的。我想让我的孩子掌握自己的生活，但我也想让她在老师要求时给所有的行星涂上颜色。如果她不做呢？如果她从不这样做呢？如果老师试图让我*让*她做呢？我父母让我做了一些事情，我很高兴他们这么做了。Stixrud 和 Johnson 在他们的书中对这些问题和担忧做出了回应，但主要归结为一个事实，即我们不能强迫我们的孩子做我们想让他们做的事情，或者成为我们想让他们成为的人。当然，为人父母是复杂的——你不能只是给一个被你擦过屁股的孩子一张新的名片，上面写着“妈妈是顾问”，但你可以从你的孩子有一个聪明的大脑，并且是一个有能力的人这个前提开始。

我仍然在脑子里翻腾着这些。我的一部分仍然持怀疑态度。但是作者说，一旦我们能够把握现实，我们就可以放下对孩子进行微观管理的责任，而只是享受他们。我们的孩子将开始发展选择自己道路的技能。
