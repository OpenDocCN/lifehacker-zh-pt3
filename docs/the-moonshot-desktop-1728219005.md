# Moonshot 桌面

> 原文：<https://lifehacker.com/the-moonshot-desktop-1728219005>

这款 Windows 10 桌面超级简单，但一切都刚刚好。部分原因是因为 [维克多](https://www.flickr.com/photos/vzhang/) 正在一台 60 英寸的电视上使用它，但它缺少小部件和开关并没有降低它的实用性和美观性。这里是你需要一个相似的外观。

Watch

幸运的是，你不需要 Windows 10 来实现这一点，但我们可以看出 Victor 正在运行 Windows 10。毕竟，这里的整个东西真的只是一个好看的壁纸和一些替换的任务栏图标。他们在这里:

*   [探月壁纸](http://i.imgur.com/qcJucbO.jpg) 根据插画师 [周汉林](http://www.flyingmouse365.com/) 的 [这幅原作改编。](https://www.flickr.com/photos/65414366@N00/5059435888/in/dateposted/)
*   [黑色图标从 Iconsdb](http://www.iconsdb.com/black-icons/) 中打包来替换任务栏图标

这就是这个的全部内容。简单的调整，很容易在任何地方复制。如果你喜欢这种外观，很容易重新创作，周汉林的壁纸真的将这一切联系在一起，所以请确保你查看 [他的网站](http://www.flyingmouse365.com/) 或 [他的 Flickr 页面](https://www.flickr.com/photos/65414366@N00/) 以了解更多他的艺术
