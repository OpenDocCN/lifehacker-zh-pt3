# 当你烦恼的时候，用这个五点指南找到耐心

我很容易生气，而且一直如此。当我小的时候，我常常求我妈妈让我妹妹不要在车里唱歌。“但这是一种快乐的声音，”她会说。(她是对的。我是个混蛋。烦恼是小问题，不是严重的问题，如果我们能学会不再被它们困扰，我们所有人和我们周围的人都会更快乐。

心理学家艾伦·亨德里克森博士写道，耐心是一种自我控制，她说，这是“调节你的情绪和行为的能力，即使你的冲动在尖叫。”因此，耐心是一种可以训练和加强的精神状态——它不是圣人和超级英雄的专利。烦恼，正如任何一个被它困扰的人都会意识到的，是愤怒的一个子集。

亨德里克森利用关于自我控制和愤怒的研究，起草了这个五点指南，来摆脱烦恼，找到耐心:

> 1.要知道你的目标还是会实现的。
> 2。给自己想象中需要的东西。
> 3。改变你的结论。
> 4。假装你被监视了。
> 5。把这个故事留到以后再说。

1.不耐烦和烦恼经常出现，不是因为我们没有得到我们想要的，而是因为我们没有足够快地或以我们理想的方式得到它。队伍太长，航班延误，音乐糟糕，荧光灯闪烁。不管是什么，记住我们的目标仍然会实现是有帮助的——我们仍然会去购物或吃饭或做其他事情。专注于此。

2.我们很多人在脑海中有一个理想化的自己——耐心、冷静、与世无争。问问自己她有什么是你没有的。现在，想象你拥有它。无论是内心的平静还是一个暂停时间的遥控器，这样你就可以跳到队伍的前面，只要想象你有那种神奇的东西可以抚慰人心。对于像内心平静这样的无形的东西，它实际上至少能让你在某种程度上达到目的。

3.我们总是认为我们知道发生了什么。队伍移动得很慢，因为收银员很糟糕。你的配偶在寻求你的帮助，因为他们懒得解决自己的问题。陌生人对你无礼是因为他们是混蛋。惹恼我们的并不是慢吞吞的排队或求助，而是我们因此而做出的假设。因此，尽可能慷慨地改变这些假设。想象一下你的方式——就像你可能得出的任何其他结论一样真实。

4.无论你想象你正被某个你想给人留下深刻印象的人或某个你想为其树立好榜样的人注视着，你都有冷静和快乐的行为——这可能是这样做的动机。

5.在令人沮丧的情况下，还有什么比开始写你的*天啊，你不会相信那天我发了*牢骚更好的方式呢。好吧，有很多更好的方法，因为编造那个沮丧的故事只会让你感觉——你猜对了——更加沮丧！相反，想想你在车管所漫长而愉快的下午会发生什么。你看到一个戴着酷帽子的女人了吗？一个孩子古怪又可爱吗？或者你是耐心的化身？准备好你要讲述的故事。
