# 虚幻的关联:导致错误思维的常见心理错误

> 原文：<https://lifehacker.com/the-illusory-correlation-a-common-mental-error-that-le-1755442156>

我们都使用愚蠢的逻辑来帮助我们合理化一个混乱的世界。以满月为例。几个世纪以来，人们一直将与满月同时发生的令人费解的行为归咎于满月。但这是一种虚幻的关联——我们欺骗自己相信一些基于我们记忆中最突出的东西。

几个世纪以来，人类一直将奇怪的行为归咎于满月。例如，在中世纪，人们声称满月可以将人类变成狼人。在 18 世纪，人们普遍认为满月会导致癫痫或发烧。我们甚至改变了我们的语言来匹配我们的信仰。单词 *lunatic* 来自拉丁语词根 *luna* ，意思是月亮。

今天，我们(大部分)恢复了理智。虽然我们不再将疾病归咎于月相，但你会听到人们用它来解释疯狂的行为。例如，医学界有一个常见的故事，在医院一个混乱的晚上，一个护士经常会说，“今晚一定是满月。”几乎没有证据表明满月会真正影响我们的行为。对 30 多项同行评议研究的完整分析发现，满月与住院、赌场支出、自杀、交通事故、犯罪率和许多其他常见事件之间没有关联。

但有意思的是:尽管研究表明并非如此，但 2005 年的一项研究显示，70%的护士仍然认为“满月那天晚上会导致更多的混乱和病人。”

这是怎么回事？

那些发誓说满月会导致奇怪行为的护士并不愚蠢。他们只是一个困扰我们所有人的常见心理错误的受害者。心理学家把这个小小的大脑错误称为“虚幻的关联”

它是这样工作的。

### 我们如何愚弄自己却没有意识到

当我们错误地过分强调一个结果而忽略其他结果时，就会产生一种虚幻的关联。例如，让我们说你参观纽约市，当你正在登上地铁时，有人打断了你。然后，你去一家餐馆，服务员对你很粗鲁。最后，你在街上向某人问路，他们会拒绝你。

当你回想起你的纽约之旅时，很容易想起这些经历，并得出“纽约人很粗鲁”或“大城市的人很粗鲁”的结论

然而，当服务员表现得非常正常时，你忘记了你吃过的所有饭菜，或者你在地铁站台上经过的数百人没有打断你。这些实际上是无关紧要的事情，因为没有什么值得注意的事情发生。因此，比起你愉快地吃饭或平静地坐地铁的时候，更容易记起别人对你无礼的时候。

这就是脑科学发挥作用的地方:

数百项心理学研究已经证明，我们倾向于高估我们容易回忆起的事件的重要性，低估我们难以回忆起的事件的重要性。越容易记住，我们就越有可能在两个弱相关或根本不相关的事物之间建立强有力的关系。

### 如何发现虚幻的关联

有一个简单的策略，你可以用它来发现你隐藏的假设，并防止你自己做出一个虚幻的关联。这叫做列联表，它迫使你认识到日常生活中容易忽略的非事件。

让我们来分析一下满月和疯狂入院的可能性。

*   **细胞 A:** 满月，忙碌的一夜。这是一个非常令人难忘的组合，在我们的记忆中被过分强调，因为它很容易回忆起来。
*   **细胞 B:** 满月，却无动静。这是一个无关紧要的事件，在我们的记忆中没有被强调，因为什么都没有发生。很难记住*没有*发生的事情，我们往往会忽略这个细胞。
*   **细胞 C:** 没有满月，却是忙碌的一夜。这很容易被视为“疯狂的一天”
*   **细胞 D:** 没有满月，一个普通的夜晚。两端都没有发生值得纪念的事情，所以这些事件也很容易被忽略。

这张应急表有助于揭示满月时护士们内心正在发生的事情。护士们很快就记起了有一次满月时医院人满为患的情况，但却忘记了有很多次满月时病人数量正常的情况。因为他们可以很容易地恢复关于满月和疯狂之夜的记忆，所以他们错误地认为这两件事是相关的。

我第一次知道这个列联表策略是在阅读 [50 个流行心理学的伟大神话](http://jamesclear.com/book/50-great-myths-of-popular-psychology) 的时候，我发现这个简单的表可以适用于许多不同的情况。理想情况下，您可以在每个单元格中插入一个数字，这样您就可以比较每个事件的实际频率，这通常与您容易记住的每个事件的频率有很大的不同。

### 如何修正你被误导的想法

我们在生活的许多领域都产生了虚幻的关联:

*   你听说比尔·盖茨或马克·扎克伯格从大学辍学开始了十亿美元的生意，你高估了自己头脑中的故事。与此同时，你永远也不会听说所有的大学辍学生都没能创办一家成功的公司。你只听到命中的消息，却听不到未命中的消息，尽管未命中的数量远远超过了命中的数量。
*   你看到某个特定民族或种族背景的人被捕，所以你认为所有有这种背景的人都更有可能参与犯罪。你永远不会听说 99%的人没有被逮捕，因为这不是一个事件。
*   你在新闻中听说了鲨鱼袭击事件，并拒绝在下次海滩度假时下海。自从你上次下海以来，鲨鱼袭击的几率并没有增加，但是你从来没有听说过每天有数百万人安然无恙。新闻永远不会刊登题为“每天有数百万游客漂浮在海上”的报道。你过分强调你在新闻中听到的故事，并做出一个虚幻的关联。

我们大多数人都没有意识到我们对事件的选择性记忆是如何影响我们日常生活中的信念的。我们很难记住没有发生的事情。如果我们看不到它，我们假设它没有影响或很少发生。

如果你理解这些思维错误是如何发生的，并使用如上所述的列联表测试等策略，你就可以揭示你甚至不知道自己拥有的隐藏假设，并纠正困扰我们日常生活的误导性思维。
