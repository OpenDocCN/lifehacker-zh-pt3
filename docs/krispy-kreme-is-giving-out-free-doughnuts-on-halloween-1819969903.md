# Krispy Kreme 在万圣节发放免费甜甜圈

> 原文:[https://life hacker . com/krispy-kreme-is-giving-out-of-free-doughnuts-on-Halloween-1819969903](https://lifehacker.com/krispy-kreme-is-giving-out-free-doughnuts-on-halloween-1819969903)

上周我们告诉了你在万圣节 墨西哥卷饼店分发 3 美元的墨西哥卷饼。现在，你可以在墨西哥卷饼午餐后吃一份免费的甜甜圈甜点。(谁还需要糖果来吹你明天的热量预算！)

[*《财富》*报道](http://fortune.com/2017/10/30/free-krispy-kreme-donut-halloween/) 称，该零售商将在万圣节向任何身着奇装异服出现在美国和加拿大的 Krispy Kreme 门店的人免费发放一个甜甜圈。通过促销活动，你可以免费获得任何你想要的甜甜圈(甚至是一个别致的万圣节主题的奶油馅的数字)。这样你就不用排队等着吃原味蜜饯了(尽管它们很好吃)。

优惠从开始到结束都有效，如果你住的地方有很多 Krispy Kreme 店，如果你想的话，你可以点击每一家，并多次利用赠品。我们有什么资格评判？

[Krispy Kreme 在脸书的促销活动](https://www.facebook.com/events/180635675834459/) 称，该交易不包括杂货店和便利店，也不适用于 Mobile 或 Foley AL、新泽西州泽西城、衣阿华州、路易斯安那州或内布拉斯加州的 Krispy Kreme 商店。
