# 翻译工具对决:谷歌翻译 vs 微软翻译

> 原文:[https://life hacker . com/translation-tool-摊牌-Google-translate-vs-microso-1787836106](https://lifehacker.com/translation-tool-showdown-google-translate-vs-microso-1787836106)

无论你是环球旅行者、语言学生还是企业主，像 Google Translate 和 Microsoft Translator 这样的工具可以让世界上的每个人更容易相互理解。尽管两者都可以翻译文本、语音和图像，但它们仍然各有各的怪癖。所以请登上决战列车！下一站:翻译站。

### **竞争者**

在我们深入研究并开始比较功能之前，让我们快速浏览一下我们的翻译巨头:

*   [**谷歌翻译**](https://translate.google.com/) **:** 谷歌的一款免费翻译工具，可以通过 [浏览器](https://translate.google.com/?hl=en) 、 [手机浏览器](https://translate.google.com/?hl=en) 、 [安卓 app](https://play.google.com/store/apps/details?hl=en&id=com.google.android.apps.translate) 或 [iOS app](https://itunes.apple.com//app/google-search/id414706506?mt=8) 使用。浏览器和移动浏览器版本都可以翻译文本和网页，非移动浏览器也可以翻译一些文档。Android 和 iOS 应用程序可以翻译文本、实时语音、图像、网页，甚至某些语言的实时视频。还有 [一个 API](https://cloud.google.com/translate/?hl=en) 开发者可以用来搭建自己的翻译工具。
*   [**微软翻译器**](https://www.microsoft.com/en-us/translator) **:** 微软的一款免费翻译工具，可以在你的浏览器或手机浏览器中使用(通过 [Bing 翻译器](http://www.bing.com/translator) )，并拥有 [Windows](https://www.microsoft.com/en-us/store/p/translator/9wzdncrfj3pg) 、 [Windows Phone](https://www.microsoft.com/en-us/store/p/translator/9wzdncrfj3pg) 、 [iOS](https://itunes.apple.com/us/app/microsoft-translator/id1018949559) 、 [Android](https://play.google.com/store/apps/details?id=com.microsoft.translator&hl=en) 、等应用此外，Microsoft Translator 可以与其他 Microsoft 应用程序集成，如使用 Microsoft Translator API 的 [Microsoft Office、Skype 和 Visual Studio](https://www.microsoft.com/en-us/translator/apps.aspx) (商业应用程序不免费)。浏览器版本只能翻译文本和网页，但微软翻译应用程序可以处理文本、实时语音和图像。

谈到翻译工具，谷歌翻译 [一直是最受欢迎的](https://lifehacker.com/most-popular-language-translation-tool-google-translat-1636028816) ，但是 [由于一些重大更新](http://lifehacker.com/microsoft-translator-can-translate-with-voice-text-an-1722432376) ，微软翻译(也称为必应翻译) [在过去几年里一直在追赶](http://lifehacker.com/microsoft-translator-adds-offline-translation-support-f-1759839604) 。现在，当谈到功能时，它们是相当相似的。

### **谷歌拥有几乎两倍的语言，但它们并没有得到完全的支持**

一个翻译工具不应该仅仅用它能翻译的语言数量来衡量，但它确实是一个重要的组成部分。语言越多越好。谷歌翻译的 [103 种语言](http://translate.google.com/about/intl/en_ALL/languages.html) 胜过微软翻译的 [对 54 种语言](https://www.microsoft.com/en-us/translator/languages.aspx) 的支持。但是在我们为谷歌翻译打分之前，重要的是要注意到，他们目录中的每一种语言都有相同水平的支持。

谷歌翻译可以处理 103 种语言，但是 [并不是每种语言都有每种功能](http://translate.google.com/about/intl/en_ALL/languages/) 。例如，可以使用 Translate 的所有六个功能来翻译法语:键入、书写、交谈、捕捉、查看和离线(稍后将详细介绍)。除了抓拍照片，阿拉伯语什么都能用。而尼日利亚主要使用的 [豪萨语](https://en.wikipedia.org/wiki/Hausa_language) ，只能通过文本翻译。因此，谷歌翻译的范围如此广泛是很酷的，但一旦你超越了更广泛使用的语言，这个工具本质上就变成了一个词汇词典。

当谈到微软翻译器的 54 种语言时， [是一个类似的故事，但规模小得多](https://www.microsoft.com/en-us/translator/languages.aspx) 。例如，您只能对阿拉伯语、汉语普通话、英语、法语、德语、意大利语、葡萄牙语、俄语和西班牙语使用实时语音翻译。但是微软翻译器的 54 种语言中有 44 种可以离线使用，而谷歌翻译只支持其中一半的离线使用。谷歌总共有更多的语言，还有一些可供离线使用，但在对相同的流行语言提供全面翻译支持方面，它们都差不多。

### **谷歌让你翻译实时视频，微软让对话更简单**

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-06olHmcJjS0&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-06olHmcJjS0&start=0) 

如前所述，谷歌翻译有六个主要功能:

1.  输入:你在浏览器或手机上输入一些文本。
2.  **书写**:你用手指在触摸屏上涂写一个单词或短语。
3.  说话:你对着手机的麦克风说话，它会实时翻译你说的话。
4.  抓拍:你给文本拍一张照片，它会翻译它。
5.  **见**:你将手机摄像头对准一些文本，它会实时翻译。
6.  **离线**:您可以在没有互联网连接的情况下使用上述功能。

谷歌翻译的 [实时视频翻译功能](http://lifehacker.com/google-translate-adds-word-lens-real-time-text-translat-1679431740) 是它真正从微软产品中脱颖而出的原因。它的工作原理 [很像应用程序 Word Lens](http://lifehacker.com/word-lens-the-real-time-translation-app-is-now-free-1579220195) ，它实际上改变了你眼前看到的文本。它可以使阅读标志和导航一个不熟悉的领域变得快速和容易。

微软翻译器有五个主要功能:

1.  文本:你在浏览器或手机上输入一些文本。
2.  演讲:你对着手机的麦克风说话，它会实时翻译你说的话。
3.  照片:你给文本拍一张照片，它会翻译它。
4.  **对话模式**:你和另一个人对着你手机的麦克风说话，它会被实时翻译。
5.  **离线**:您可以在没有互联网连接的情况下使用上述功能。

Translator 的实时对话模式才是真正出彩的地方。屏幕分成两半，您可以为每一边选择一种语言。比方说，我想和一个只会说西班牙语的人聊天。我会在我这边选择英语，在他们那边选择西班牙语。然后，我们只需点击屏幕两侧的麦克风按钮进行通话，剩下的工作由应用程序来完成。使用 Google Translate 的 talk 功能进行实时对话是可能的，但它非常笨重，不利于实际的来回对话。

一旦你找到一个翻译，这两个应用程序都会让你听一个音频文件，这样你就知道正确的发音。它不是每种语言都可用，但当它可用时，这是一种很好的方式。如果你想给某人看一个像提示卡一样的短语，你也可以将翻译扩展到整个屏幕。这两个应用程序都记录了你过去的翻译，所以你可以快速查阅你已经查找过的内容。谷歌翻译让你最喜欢的翻译带星号，这样它们会被永久保存，微软翻译也做同样的事情，但用大头针。

不过，微软翻译器还有一个优势:它有一个内置的短语手册，可以快速查找旅游指南、住宿、健康等短语。拥有这样的时刻很好，但如果你真的想学习你正在翻译的语言，这也很有帮助。

### 谷歌提供了更多的语言，但微软选择质量胜于数量

谷歌翻译的实时视频翻译是神奇的，这是毫无疑问的。谷歌神经机器翻译 最近增加的 [将整句翻译的准确率提高了 60%，适用于中文到英文这样的艰难工作流程。此外，谷歌翻译总体上提供了更多的语言，可以](https://research.googleblog.com/2016/09/a-neural-network-for-machine.html) [翻译 Android](https://www.youtube.com/watch?v=4xtEp55MKRE) 上其他应用程序中的文本，很明显为什么它是目前最受欢迎的选择。

也就是说，我不觉得谷歌翻译的功能让它站在竞争对手之上。实时视频翻译很棒，但我不知道我有多经常用它来做同样的事情，而不是仅仅抓拍一张照片(这两个应用程序都可以做得很好)。我不确定手写功能是为谁准备的，但也许它对不容易适应标准键盘的语言有用。说话无论哪种方式都快很多，感觉有点没必要。

Microsoft Translator 通过分屏模式专注于对话，Apple Watch 和 Android Wear 应用程序——谷歌出于某种原因没有——让你只需在手腕上轻轻一点就可以快速翻译短语。如果你有智能手表，微软翻译器是显而易见的选择。当然，它提供的语言不多，但它涵盖了所有的大语言，并对大多数语言提供了全面的翻译支持。我不能说微软的整句翻译比得上谷歌闪亮的新神经机器翻译系统，但对大多数用户来说，无论如何都很难看出区别。

这两种翻译工具都不会错。他们都完成了任务，微软也认真努力与谷歌更受欢迎的工具竞争。事实上，我在进行这种比较时期望谷歌翻译占主导地位，但微软翻译给我留下了深刻的印象。以至于在我即将到来的日本之旅中，我可能会用它来代替谷歌翻译。
