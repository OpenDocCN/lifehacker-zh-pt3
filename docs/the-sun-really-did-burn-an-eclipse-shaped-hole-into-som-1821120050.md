# 太阳确实在某人的视网膜上烧了一个日蚀形状的洞

> 原文:[https://life hacker . com/the-sun-really-did-burn-a-eclipse-shaped-hole-into-som-1821120050](https://lifehacker.com/the-sun-really-did-burn-an-eclipse-shaped-hole-into-som-1821120050)

还记得那些看日食时使用护具的警告吗？《美国医学会眼科学杂志》 的一项新研究显示，如果你不这样做，会发生什么。

Watch

今年八月的月食后，一位名叫妮娅·佩恩的妇女出现在西奈山的纽约眼耳医院，她说她的视力有一个空白点。医生让她画出斑点的形状；这是日食时太阳的形状(圆形，被咬了一口)。

当太阳被遮住 70%的时候，她在没有保护眼睛的情况下看了几秒钟。然后她借了一副日蚀眼镜，又看了 15 到 20 秒。

这些眼镜很可能是赝品；佩恩 [说](http://www.cnn.com/2017/12/07/health/eclipse-eye-damage-case-study/index.html) 当她透过它们看时，太阳似乎真的很亮。西奈山的医生使用最先进的设备对她的视网膜伤口进行拍照，发现它与她的盲点形状完全相同。华盛顿邮报 [报道称，这是医生第一次能够如此精确地看到日食带来的损害。](https://www.washingtonpost.com/news/morning-mix/wp/2017/12/08/the-solar-eclipse-burned-a-crescent-wound-on-a-womans-retina-she-wasnt-wearing-proper-glasses/?hpid=hp_hp-morning-mix_mm-eclipse%3Ahomepage%2Fstory&utm_term=.02d8f7d34ddf)

下次你有机会观看日食，使用合适的眼镜——或者用 [做一个针孔观察器](https://lifehacker.com/forget-special-glasses-the-best-views-of-the-eclipse-j-1797949429) 代替。
