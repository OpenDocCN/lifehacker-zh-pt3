# 在雨中穿人字拖

> 原文：<https://lifehacker.com/wear-flip-flops-in-the-rain-1788090026>

现在是北美的秋天，这意味着你住的地方可能要下雨了，或者刚刚下完雨，或者你正在外面雨中读这篇博客，而你的手机已经湿透了。没有人喜欢全身湿透！那是给鱼的。避免潮湿是人类发明雨衣、雨伞和鳄鱼的原因。然而，今天我真的要对你说:在雨中穿人字拖。



在雨中穿人字拖解决了一个非常具体的问题:水侵入你的靴子，并占据了那里。一旦你的大便湿了，它不会干，至少在任何时间线上不会。我已经花了很多个下午，我的鞋子浸满了水，我本来受保护的脚泡在一个愚蠢的沼泽监狱里。你也可能无意中以同样的方式拥有了自己。潮湿天气鞋的中间地带不是居住的地方。你可以全力以赴，将你的脚包裹在坦克一样的装置中，以保护他们免受每一滴水的伤害，或者你可以拥抱自由。请允许我为后一种选择辩护。

很明显，如果你在雨中穿人字拖，你的脚会被弄湿，但是露趾雨鞋的潜在好处是它们会很快变干。想象一下，如果你愿意，湿着脚到达你的目的地，但是仅仅五分钟后就可以走了，就好像从来没有下过雨一样。当然，你的脚可能一直都是干的，但这意味着你要多带几磅重的鞋底和复杂的防水皮革。还有，那是懦夫的方式。

显然，这并不适用于所有人。也许你住在纽约市，那里的垃圾在街上液化，每次暴雨都会使垃圾在所有东西上起泡。除非你喜欢脚上的垃圾，否则不要这么做。我是加州人渣，我很乐意承认这种技术没有在更冷的天气里测试过。

此外，如果你在实验室、木材店或其他一些需要穿露趾鞋的地方工作，那就别穿了。也许你只有劣质的拖鞋，不适合外面的世界。挺好的！接受湿漉漉的鞋子的拥抱。

但是如果你想要更好的东西，如果你不怕为了没有阻碍的干燥的脚而走过一点水，那就试试吧。如果你不喜欢，你随时可以回到沼泽生活。
