# Moto X 纯版评测:这款手机做安卓比谷歌好

> 原文:[https://gizmodo . com/moto-x-pure-edition-review-this-phone-does-Android-bet-1728270123](https://gizmodo.com/moto-x-pure-edition-review-this-phone-does-android-bet-1728270123)

当谈到以谷歌的方式体验 Android 时，你的选择少得惊人。两个，真的。谷歌自己的 Nexus 智能手机——由一系列不同的硬件合作伙伴制造——和摩托罗拉。新的 Moto X 纯净版是新的未受污染的 Android 冠军。它可以比谷歌做得更好。

### **什么事？**

Moto X Pure Edition 是来自摩托罗拉的新款 5.7 英寸 Android 旗舰智能手机，~~谷歌公司~~ [联想公司](http://gizmodo.com/lenovo-now-officially-owns-motorola-1652690190) 。它配备了最新的 Android Lollipop 软件和一些精选的 Moto 应用程序。这也是你可以买到的定制化程度最高的手机，这要归功于 Moto Maker，这是一个网络应用程序，让你在购买前改变手机的材料、纹理和颜色。根据你的口味，Moto X 可以是任何东西，从引人注目到低调，通过摩托罗拉的通用 SIM 卡，你可以只花 400 美元购买一个解锁版本，可以在美国任何主要的手机网络上播放。

我们经常称它的前辈为“ [”安卓手机，而 Moto X 更是保留了这一区别。](http://gizmodo.com/moto-x-2014-review-still-the-android-phone-for-every-1633601557)

### **这有什么关系？**

Moto X Pure Edition 不仅非常棒，而且实际上比摩托罗拉制造的任何旗舰手机都要便宜。当然，400 美元听起来可能不便宜，但现在美国的手机补贴正在消失，尖端智能手机可能要花 750 多美元，顶级安卓手机 400 美元是一笔惊人的交易。

或许最重要的是，**终于**有了一个出色的摄像头，这是一部摩托罗拉手机。大部分是。我们会谈到这一点。

### **设计**

Moto X 的整个设计特点一直是能够决定手机的外观。挑选各种组件的颜色，个性化设计，让它成为你自己的。

既然如此，实际上很难衡量新 Moto X 的整体眼球吸引力。就我个人而言，我喜欢我们收到的深色木材和青铜色灰色版本。木头有点滑，但远没有最近三星设备上的大猩猩玻璃那么糟糕，倾斜的背部(从一开始就是 Moto X 的商标)让 X 握起来很舒服。

凭借 5.7 英寸的屏幕，这绝对是一款大手机，但由于这种曲线，它实际上感觉易于管理。我在使用它的时候从来没有感觉到“哇，这是一个巨大的手机”，我以前从来没有说过大于 5.5 英寸的手机。

<small>*被去年的 6 英寸 Nexus 6(左)和 5.2 英寸 Moto X(右)夹击的 Moto X 纯版*</small>

不过，我确实有一些抱怨——首先是铝制相机外壳和智能手机背面之间的接缝。这是很小的问题，但是有几次灰尘和其他碎屑会不知何故地进入裂缝并被卡住。我必须找到一些薄的东西，比如我钥匙链上的 SIM 工具，才能把它挖出来。现在，我认识到，我对保持我的小工具干净几乎有强迫症的水平——我是那个经常用我的衬衫擦拭我的智能手机的人——但这是一个非常烦人的小细节。

另一个设计难题是前置 LED 闪光灯，这是智能手机的新趋势，也是所有 Moto 手机的第一个趋势，旨在让你的手机成为完美的自拍伴侣。但在这里它是一个眼中钉，我讨厌它。看看就知道了。看啊。在。它。

我明白我们生活在一个自拍的世界，前置闪光灯非常有意义，但它给 Moto X 带来的丑陋，特别是黑色边框，对我来说是不合理的。如果你是一个自拍迷，你可能不会介意——你甚至会高兴——但对我们其他人来说，这有点糟糕。

在其他所有地方，摩托罗拉在高端装修领域与三星和苹果并驾齐驱。您可以在边缘找到铝，这是您在背面选择的奢华材料。摩托罗拉取消了去年 Moto X 的“环形”闪光灯，无论如何都不太好用，并采用了更传统的布局，LED 位于摄像头传感器下方。

好吧，最后一个设计抱怨:后方的酒窝小得多，这实际上使它不太有用。在 Moto X 上，尤其是 Nexus 6，这个凹痕有助于你抓住更大的手机，提供更好的平衡。现在，摩托罗拉似乎保留了纯美学的设计选择，而不是功能，因为它没有舒适握持的位置，除非我像个白痴一样握着手机(我不认为我是。)

但总的来说，Moto X Pure 看起来很棒。因为你几乎可以用几十种不同的方式让它看起来，毫无疑问，其中一种会让你喜欢。

### 软件

摩托罗拉一直致力于 Android 智能手机的简单性，Moto X Pure 也秉承了这一理念。这款手机只是*避开*，让 Android 做它最擅长的事情。Android Lollipop 所有最好的部分，包括材质设计和超流畅的动画都在这里，Moto 所有的附加应用其实感觉都是不错的创意。让我们从最好到最差逐一介绍其中的几个:

**Moto Display:** 轻松实现 Moto X 的最佳附加功能。Moto Display 可以让你在锁屏上预览收到的通知，无需解锁手机。你只需点击图标，就可以预览通知，无论是电子邮件、短信还是群签到。真的很方便，我也很喜欢。如果你想保持隐私，你也可以选择阻止某些应用程序显示在 Moto Display 上。

很好，如果你不介意在公共场合对着手机大喊大叫并引来询问的目光的话。你设置一个短语，Moto Voice 一直在监听，然后即使它睡着了，你也可以发出语音命令。因为我一直认为红蜘蛛是我的精神变形者，所以我说“好吧，威震天。”

这么说，不管我的手机是关机了还是在房间的另一边，它都会活过来。你可以用它来设置闹钟，获得免提的路线指引，或者 Google Now 可以为你做的任何事情。

Moto Assist: 这个功能在理论上很酷，如果实施得当，它可能会是*真正的*酷，但感觉有点空洞和半生不熟。Moto Assist 的工作方式是，你指定家庭和工作等“地点”或开车、睡觉或开会等“事件”, Moto Assist 会智能地让你的手机静音，在你开车时大声朗读短信，甚至自动回复电话。

它基本上工作得很好，但它把我的许多谷歌日历事件解释为“会议”，而实际上它们只是提醒我需要写的文章或其他非会议的事情。所以我就彻底关机了。

来自 Moto Assist 的唯一伟大的事情是，我听到一个人工智能私人助理从我的一个*地下城&龙在线*公会成员那里大声说出这句话:

> 刚刚在精英史诗上杀了我一条巨大的红龙...她是 CR54

我的回答是“太棒了。”

### 扬声器

智能手机扬声器是大多数智能手机上一个长期被忽视的功能。像苹果和三星这样的大男孩似乎是事后才想到的，有些人(我看着你 LG)把它们放在背面，根本没有任何好处。

但 Moto X Pure 显示屏的两侧是两个前置扬声器，虽然它们不像你在一些 HTC 手机上看到的绝对美妙的 Boomsound 扬声器那样低沉和丰富，但它们有一件事做得很好——它们让你实际上听到了一些东西。有一次，我试图给一个朋友 [看一段视频](https://www.youtube.com/watch?v=z9dg8PI2sEg) ，记录了流行视频游戏包装盒上“下巴朝下，眼睛朝上”的奇怪现象。我加载了视频，出于本能，我把手放在扬声器周围，这样声音就会直接传到我们的脸上。只是这一次，我不需要。即使出租车呼啸而过，喝醉的纽约人尖叫着猥亵，我也能听到视频。这是一件简单的事，但却是一件伟大的事。

### 喜欢

描述 Moto X Pure 显示屏的最佳方式是*电影化*。凭借更大的 5.7 英寸 QHD 分辨率显示屏和两个前置扬声器，这家伙是一个伟大的小型多媒体冠军。此外，库存 Android 在尽可能多的像素上看起来最好。

关于个人品味的标志性短语是“各有所好”，而 Moto X 正是给了你这一点。我个人非常喜欢我的深色智能手机——一部青铜色的智能手机，因为它符合我灵魂的内容，我的灵魂主要是深色和黑色。但也许你讨厌它！幸运的是，有一大堆不同的选项可供选择，所以你可以尽情地讨厌我的设计选择(你可能是错的，但无论如何)。

这个软件太棒了。它重量轻，动画精美，而且简单。我不需要翻阅一百万份菜单。它感觉像一部 Android 手机，这很有意义，因为这就是你要买的东西。

价格绝对让它。在 700 美元以上(这是你通常为顶级 Android 支付的价格)，我会说有太多的硬件缺失。没有无线充电，没有指纹传感器。但是 300 美元的大幅折扣，我想说这个价格实在太高了。只是很划算。

### 不喜欢

这似乎并不完全是面向未来的。我们知道谷歌下一个发布的 Android，Marshmallow，将正式为 Android 带来指纹传感器支持。随着 Android Pay 也将在几天或几周内推出，Moto X Pure 上没有它可能会很烦人。Moto 的代表告诉我，这项服务肯定仍然可以在手机上使用，你只需要用 pin 码进行认证。由于移动支付的整个承诺是快速和无痛，没有指纹传感器是一个明确的缺点。

而在白天，Moto X Pure 简直令人惊叹。它不仅击败了 S6，让我们完全惊叹，它甚至击败了 G4。但是伙计，在弱光下相机就像像素为*像素为*的亚当·桑德勒，也就是说非常糟糕。如果你发现自己更喜欢白天工作，这款相机会创造奇迹。但是如果你是一只夜鹰，准备好悲伤吧。

这款电池本身并不令人讨厌，但也不出色。只是还可以。早上 8 点取下充电器，晚上 10 点插上，我通常只有 5%的电量，这是一个相当普通的使用习惯。那不可怕，但也不伟大。如果您熬夜或倾向于大量使用(例如，大量的流媒体、网页浏览、视频录制等。)，Moto X 要死在你身上了。幸运的是，它有快速充电功能，所以如果你发现你急需一些果汁，你可以很快得到一些。

虽然价格很相似，但 400 美元的价格是针对 16GB 存储空间的。对于一部拍摄 4K 的手机来说，这是不够的。够了。与。16GB。手机。

### 该不该买？

如果你正在寻找绝对物超所值的安卓智能手机:是的。是的，你应该。你心中唯一应该感到犹豫的是，谷歌最有可能在本月底发布两款 Nexus 智能手机。一部谷歌手机意味着两款与 Moto X 非常相似的设备，肯定会首先获得即将到来的棉花糖更新，甚至可能成为谷歌新的 [项目 Fi 无线服务](http://gizmodo.com/project-fi-googles-plan-to-fix-your-wireless-service-i-1699499531) 的一部分。

但 Nexus 最有可能不会有专门为你量身定制的外观和你想要使用的合理有用的 Moto 应用程序。扣动扳机或者等待——这是一个双赢的局面。
