# 看看你的推特粉丝到底有多假

> 原文：<https://lifehacker.com/see-exactly-how-fake-your-twitter-followers-are-1829600902>

我妻子和我的推特粉丝数量差不多——分别是 16k 和 14k 但她的推特数量一直超过我。事实上，根据 Twitter 营销服务 SparkToro 的 SparkScore 工具 的数据，她的一条推文平均获得三倍的转发量，*获得十倍的收藏量。* 

Watch

怎么会这样？她的推文 客观上能比我的好吗？如果我的自我拒绝接受这个怎么办？如果是我的追随者的错呢？SparkToro 还有另一个工具， [虚假关注者审计](https://sparktoro.com/tools/fake-followers-audit) ，来检查任何 Twitter 用户的受众中有多少是真实的。

这似乎解释了这一点:我有一个糟糕的机器人感染，废弃的帐户，和各种其他的死重量。

另一方面，我的妻子有非常真实的追随者。

所以我坐在谎言的宝座上。然而，我有足够多的真正的追随者，我应该得到更多的喜爱和转发。我一定是一个低劣的高音，或者在我的时代被误解了。

为了让自己好受些，我查了一下雪儿，看看她的 350 万粉丝中有多少是机器人。这是个糟糕的主意。

雪儿有 22%的虚假粉丝，而我有 24%，但 SparkToro 给了她更多的信任，因为像她这样的大账户往往有很高的平均假货率。她已经不在 SparkScore 的名单上了。

SparkToro 并不能帮你摆脱你的伪追随者； [那就要花钱做](https://lifehacker.com/how-to-block-your-fake-followers-1822739611) 。这是一个很好的地方，你可以拿起屏幕帽，嘲笑那些你不喜欢的人。但首先要确保他们不会把矛头指向你。*不要在雪儿身上用这个。*
