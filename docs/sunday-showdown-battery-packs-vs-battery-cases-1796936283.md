# 便携式电源摊牌:电池组与电池盒

> 原文：<https://lifehacker.com/sunday-showdown-battery-packs-vs-battery-cases-1796936283>

无论你如何划分，在你的身上保持一些额外的电池能量可以创造奇迹，影响你如何在世界上移动。如果你想随身携带一些集中电力以备不时之需，你有两种选择:

### 竞争者

*   **电池盒**:毫无疑问，你以前见过电池盒。它本质上是一个智能手机外壳，内置电池和连接器，旨在全天候保持在您的手机上。许多通过微型 USB 端口充电，并有可视指示器，让你知道里面还有多少电池电量。

*   **外接电池组**:电池组是外接电池，可以比你的智能手机充电更多。配备一个或多个 USB 端口，只要你有合适的电缆，你就可以为任何移动设备充电。外部电池组往往比电池盒携带更多的电力，不仅可以为手机充电，还可以为你携带的任何无线耳机或其他小型电子设备充电。

两者都给你的设备充电，而且都需要自己充电。但其中一个为了方便而牺牲功能，另一个放弃了一刀切的方法，转而支持为智能手机定制电池。

### 外部电池组:功能性、友好性和多样性

我随身带着一个很短的电池组，只是因为我带的东西比我的智能手机多。我的蓝牙耳机和数码相机总是放在我的包里，但当电池没电时，它们并不总是被拿走。这就是外部电池组发挥作用的地方。存在几乎无限的种类，从可以放入你钱包的包到可以在需要找到插座之前给你的设备充电多次的巨大块。我的可以放在我的手掌里，至少可以给我的手机充电两次。

电池组通常通过微型 USB 电缆充电，并具有一到两个 USB-A 端口。根据您的需要，您应该选择一个具有适当额定电流的电池组来为较大的电子设备充电。2.1 安培的电池组非常适合为 iPads 和平板电脑充电，并且比 1 安培的充电器更快地为智能手机充电。

如果你身边的朋友手机没电了，外置电池组也能把你变成救世主。我曾在不太理想的情况下为朋友和家人的设备充电，只是因为我把电池组放在背包里。

外部电池组的功能是有代价的。简而言之，你必须记得给它充电，并带上正确的电缆。不像你的手机电池盒，你很可能每天晚上在你的床边充电，外部电池组就在你的包里，耐心地等待你记住它的存在。通常这发生在它奄奄一息的时候，就在它变成一大块无用的锂离子电池之前。没有比在绝望的情况下拿出你的电池组，却发现它彻底报废更糟糕的感觉了。

### 电池盒:功能胜于形式

为了确保无论你在包里放什么都有电，你应该检查一下电池盒。电池盒更有可能保持充电状态，因为你需要插上电池盒给手机充电。当你每天晚上插上电源时，你的电池盒和智能手机都会充电，给你留下两个充满电的设备。仅仅是这种便利就值得考虑购买一个外壳，而不是一个外部电池组。

有一个缺点:由于它总是连接到你的手机上，你将携带一个明显更大的设备。电池盒并不是最时尚的配件。甚至苹果自己的 iPhone 电池外壳看起来也很奇怪，给你的 iPhone 后部增加了一个矩形的不均匀隆起。

案子越大，进出口袋越难。当我用一个电池盒做了几个月的实验后，我发现自己偶尔会把电池盒从我的设备上拉下来，因为我的口袋很小。摩尔菲等公司正试图通过智能手机外壳来打破这种差异，当智能手机外壳磁性附着在外部电池组上时，可以无线充电。但这意味着你仍然需要携带外部电池。

### 结论:轻装旅行，携带电池盒

那么你应该随身携带什么呢？这取决于你有多少设备。如果你只是随身携带手机，并且希望无论外出多长时间都能够查看社交信息和与朋友交流，那么你应该考虑电池盒。它不仅能让你的智能手机运行时间延长一倍，还能保护你的手机在跌落时不会摔坏。老实说，我想我可能又要带着这个了。

如果你携带多种设备，如电话、相机和一副无线耳机，外置电池组可能更有效。你必须随身携带一些电缆，但如果你已经把电子产品放在背包或钱包里，为什么不放入电池组，以防万一。你只要记得充电就行了。
