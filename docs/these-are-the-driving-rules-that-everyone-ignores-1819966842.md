# 这些都是大家忽略的驾驶规则

> 原文：<https://jalopnik.com/these-are-the-driving-rules-that-everyone-ignores-1819966842>

我喜欢开车。我的车是我快乐的地方。我喜欢带着它到处走。不过，我经常发现自己坐在方向盘后面，对着另一个司机拉的一些白痴垃圾大喊大叫。因为和别人一起开车会毁了驾驶。因为其他人是地狱。

Watch

上周，我向你们询问了你们希望人们更多遵守的 [驾驶规则](https://jalopnik.com/what-driving-etiquette-or-rules-do-you-wish-people-foll-1819799931) 。老实说，我和你一样困惑，为什么更多的人不跟随他们。我记得大多数都是在驾驶课上学的。其余的只是常识。最重要的是，不要做个混蛋。

这一次，我在评论中感受到了很多愤怒。但这是一种很好的愤怒！这种愤怒有望引发变革。

许多优点都被集体挑选出来，比如使用转向灯，除超车外不要走左车道，以及注意远光灯。我收集了一些更微妙的。
