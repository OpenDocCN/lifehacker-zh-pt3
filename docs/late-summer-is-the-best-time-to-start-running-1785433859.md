# 夏末是开始跑步的最佳时间

> 原文:[https://life hacker . com/late-summer-is-best-time-to-start-running-1785433859](https://lifehacker.com/late-summer-is-the-best-time-to-start-running-1785433859)

随着夏天进入秋天，我们站在开始或重新开始跑步的黄金时间。我们期待着两个月左右的好天气，早上和晚上有充足的光线，一旦你坚持到 11 月，几乎可以保证一场方便的目标比赛。系好鞋带，我们开始吧。

### 天气很好，而且只会越来越好

因为现在是夏天，你可能已经在户外享受了很多时间。也许你会多走走，或者带你的孩子去海滩或游乐场。如果你像我一样，你是 [负责大量地面狩猎神奇宝贝](https://lifehacker.com/how-to-hatch-pokemon-eggs-and-have-a-great-run-at-the-s-1784272881) 。比较一下，如果是在一月份，你会有什么样的感觉，你不得不解开你的睡衣，从沙发上爬起来。趁你还可以的时候，好好利用你阳光、积极的倾向。

但是你说，外面很热。这实际上不是什么大问题。假设今天有 90 度的高温。你可能会在早上的某个时候找到一个 70 度的好天气，或者在日落时稍微暖和一点的时间。然后，随着我们进入九月，天气会变凉，你的跑步会变得更加愉快。

就算避不了热，也没事。在炎热中跑步会让你慢下来，但是从长远来看 [会让你更强壮](https://lifehacker.com/how-to-get-used-to-exercising-in-the-heat-and-why-you-1716804690) 。如果你能在天气闷热的时候站起来锻炼，当天气凉爽的时候，你的身材会更好。就像给自己一个超能力。

不管怎样，在冬天来临之前，你会有几个月的好天气。如果你生活在寒冷的气候中，你需要 [购买一些额外的装备，并学会应对冬季天气](https://lifehacker.com/how-to-run-outdoors-in-the-winter-1745499007) 。但是现在是夏天啊！那些日子还很遥远。当寒冷最终降临时，你可以一次几度地增加你的衣柜和勇气。

### 现在是开始新生活的好时机

如果你不得不按照学校的时间表工作，新学年的开始是一个很好的时间来安排你的日常事务。当你已经养成了适合你的习惯时，坚持锻炼是很容易的，所以考虑一下你的时间表吧。

我喜欢在把孩子送到日托所后马上锻炼，这样我就可以在开始一天的工作前结束跑步。暑假已经毁了那个时间表，所以我期待着学年重新开始的时候，我可以恢复那个习惯。

如果你自己是一名学生，把你的锻炼想象成你一天中必须安排的又一堂课。选择一个你会坚持的时间段。如果你不报名参加早上 8 点的课程，你可能不会喜欢早上 8 点的跑步。

当你在夏末开始跑步时，还有一个时间安排上的优势:清晨和傍晚仍然有充足的光线。这意味着你几乎可以在任何时候安排一次户外跑步，而不必担心携带手电筒和躲避看不见的坑洞。随着白天变短，你必须调整你的时间表(或者买一个手电筒)，但至少你可以在阳光持续的时候享受阳光。

### 有一个刻有你名字的土耳其小跑

如果没有明确的训练目标，我很难坚持锻炼计划。这就是为什么我总是试着在我的日历上安排一场比赛，即使是几个月后。

即使你根本不跑，你也可以在两个月内增加到跑 5 公里(大约 3 英里)。 [经典的 couch-to-5K 计划](http://lifehacker.com/run-your-first-5k-36525) 需要九周时间，而运行网站和应用程序的培训计划往往需要六周或八周时间。

这意味着你在感恩节前有足够的时间训练，因为几乎每个美国城市都会举办一场 5 公里赛跑，并将其命名为 [火鸡赛跑](https://en.wikipedia.org/wiki/Turkey_trot) 。把你的位置输入到[Runner ' s World ' s race finder](http://www.runnersworld.com/race-finder)，我几乎可以保证你附近会有一个。

不要为此感到紧张。像火鸡赛跑这样的比赛(实际上，你作为一个普通的跑步者会遇到的大多数比赛)很容易就能以跑步为乐。你支付你的费用，确定你的数字，加入各种水平的跑步者，这基本上是一个大的团体训练。大多数 5 公里比赛包括跑步者和竞走者，所以无论你跑得多慢，你都不会是最后一个冲过终点线的人。

到今天为止，离感恩节还有三个多月。你可以做一个九周的项目，有足够的时间。见鬼，你可以做两次为期六周的项目。准备好了吗？你能行的。

### 如果你错过了这个完美的窗口，你还有其他选择

如果你在八月或九月读到这篇文章，请跳过这一节。现在就开始跑步，不要怀疑自己。

还在吗？好吧，肯定是别的季节。有一些第二好的开始时间，所以你不必太失望:

*   春天几乎和夏末一样好。你避开了夏天的炎热和冬天的寒冷。唯一的缺点是你必须马上应对寒冷的天气，而不是慢慢适应。
*   如果你生活在温暖的气候中，冬天是很棒的。你可以忽略我所说的关于冰雪的一切，享受一些不太热的天气去跑步。
*   一月适合找公司。每个人都在跳上新年决心列车，所以很容易找到一个伙伴加入你。
*   5 公里比赛前两个月，任何 5 公里比赛。如果你所在的城镇有一场你想参加的比赛(或者如果你找到了一场你想去参加的比赛)，你有一个内置的训练计划。两个月的时间足够了。

事实上，开始跑步的最佳时间是现在*，不管现在*是什么时候*。等待一个更好的时机是没有意义的。*

*但我真的认为夏末有一些特别之处。天气很好，反正你在户外，在黑暗和寒冷到来之前，日历上有很多美好的日子。当你可以的时候好好利用。这是开始的最佳时机。*
