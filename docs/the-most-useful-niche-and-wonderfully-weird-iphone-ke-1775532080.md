### Slash Keyboard 将大量其他应用程序集成到您的键盘中

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-5hUNeRcZ-WQ&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-5hUNeRcZ-WQ&start=0) 

[斜线键盘](https://itunes.apple.com/us/app/slash-keyboard-gifs-stickers/id1020956821?ls=1&mt=8) (免费)是一个有趣的工具，让你可以通过键盘轻松搜索 YouTube、Spotify 和谷歌地图等各种服务。有了 Slash，你不必离开当前正在输入的应用去做基础研究。

轻按键盘上的斜线图标，选择您想要搜索的服务，然后键入您的搜索。当你在键盘上找到你要找的东西时，轻按它来发送它。最明显的是，你可以搜索谷歌或者在 YouTube 上寻找视频。也许更小众一点，你也可以找到《纽约时报》的一篇文章，从 Spotify 上下载一首歌，在 Foursquare 上查看附近的位置，等等。

如果这个想法听起来很熟悉，那么之前提到的 [ReBoard](https://itunes.apple.com/us/app/reboard-keyboard-multitasking/id984982881?mt=8) 就是一个类似的 app，但是 Slash 做得更好。如果你倾向于向朋友发送大量链接，Slash 是最有用的，但是这里有足够广泛的工具集，人们应该能够找到各种各样的用法。

### iTranslate 直接从键盘翻译文本

如果你在和一个母语和你不同的人发短信，那么你应该知道在你的短信应用和翻译应用之间切换很糟糕。it translate(免费)通过在其基本的翻译应用程序中包含一个自定义键盘，使这一点变得更容易。

若要翻译文本块，请键入一个短语，选取您想要翻译的语言，然后轻按“翻译”按钮。iTranslate 完成翻译后，您可以将该短语粘贴到文本框中。遗憾的是，翻译你收到的任何东西都是一个有点复杂的过程(你必须复制、粘贴和反向翻译)，但是对于你正在输入的文本的即时翻译，iTranslate 非常有效。与大多数 iOS 键盘不同的是，除了键盘之外，还有一个功能齐全的应用程序，这也很好。这样如果想做更复杂的翻译，或者语音翻译，就不需要单独下载 app 了。

### Giphy Keys 让每一张 GIF 都触手可及

gif 是最伟大的现代交流和表达形式之一。在尝试了许多选项后， [Giphy 键](https://itunes.apple.com/us/app/giphy-keys.-the-gif-keyboard/id1010794981?mt=8) (免费)是我最喜欢的搜索 gif 以添加到短信中的键盘。

Giphy Keys 的魅力在于其简单性和整体可用性。轻按搜索栏，键入您要查找的 GIF 类型，滚动找到您要使用的 GIF，然后使用快速复制和粘贴命令发送它。

您还会发现 Giphy Keys 中包含一些很酷的高级功能。例如，您可以将经常使用的 gif 添加为书签，或者动态创建自定义的动画 gif 文本。Giphy Keys 非常坚固，因为它只是一个 gif 键盘。如果你经常在不同的应用程序之间切换，将 gif 复制并粘贴到对话中，那么 Giphy Keys 就是你想要的应用程序。

### Microsoft Word Flow 使单手打字变得容易

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-RlcE1334_ew&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-RlcE1334_ew&start=0) 

微软的 Word Flow (免费)在 Windows Phone 上似乎很受欢迎，iOS 键盘在将这种体验带到 iPhone 上方面做得很好。乍一看，Word Flow 的滑动式字符选择似乎与 Swype 或 SwiftKey 等键盘没有什么不同。不过，字符选择并不是真正让 Word Flow 与众不同的地方。是单手模式。

当启用单手模式时，单词流会在屏幕底部划过键盘。这使得用一只手打字变得相当容易，尤其是在像 iPhone 6s Plus 这样的大设备上。对于在更大的手机上单手打字的问题，这是一个绝妙的解决方案。如果你发现自己经常用一只手同时处理多项任务，那么 Word Flow 值得一看。

### Hemingboard 让你唾手可得

喜欢双关语？那么 [鹤鸣板](https://itunes.apple.com/app/id1019312381?mt=8&ign-mpt=uo%3D8)(3.99 美元)就是为你打造的键盘。Hemingboard 是一个键盘，它包含了同义词词典、押韵词典，以及最重要的双关语词典。没错。双关语。只要输入一个单词，你就可以搜索相似的单词来表达。

同义词和押韵很有用，原因很明显，尤其是如果你倾向于发送大量需要增加趣味的电子邮件，或者你在手机上录制歌词。尽管如此，毫无疑问，双关语是这里的主要内容。双关语的妙处在于，它们仍然需要你的一点思考。这不是等着房间里的任何人礼貌地笑着讲爸爸的笑话。Hemingboard 只显示包含您的搜索术语的单词，如“catastrophe”或“catalog”中的“cat”这仍然取决于你是否能说出让一屋子人翻白眼的双关语。

这些键盘中的大多数不会在日常生活中取代 iPhone 的库存键盘，但这并不是真正的重点。它们是很好的补充，让发短信变得更有趣，有时甚至更简单。
