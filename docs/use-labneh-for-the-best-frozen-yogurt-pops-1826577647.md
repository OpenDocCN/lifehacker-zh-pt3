# 使用 Labneh 制作最好的冷冻酸奶汽水

> 原文：<https://lifehacker.com/use-labneh-for-the-best-frozen-yogurt-pops-1826577647>

在冷冻甜点的等级体系中，冷冻酸奶“冰棍”很少进入我的前 10 名。发明这种方法是为了欺骗孩子们，让他们以为他们在吃“甜点”，这些冰冻的水果棒和酸奶很少达到被认为“放纵”所需的质地和味道。然而，通过将“普通”酸奶换成更稠、更令人满意的酸奶，这一切都解决了。



Labneh 是一种过滤酸奶奶酪，是所有乳制品中最好的。如果你要画一个维恩图，上面有三个圆圈，分别标着“酸奶油”、“奶油奶酪”和“浓烈的酸奶”，那么 labneh 就在正中央。

它本身就有如此多的味道，没有必要添加太多其他的。将它与一些非常好的水果蜜饯混合并冷冻，你就有了一个带有芝士蛋糕倾向的酸奶汽水。将它与芝麻酱和蜂蜜混合(最好撒上一些芝麻)，你就有了一份精致的、略带咸味的甜点，你不会想和孩子们分享的。要做四大杯酸奶汽水，你需要:

*   2 杯 labneh(商店购买的或者 [自制的](https://lifehacker.com/how-to-make-your-own-labneh-and-how-to-eat-it-1796785416) )
*   1/4 杯美味的混合食品(芝麻酱、水果蜜饯、坚果黄油、浸渍水果)
*   2 汤匙蜂蜜(如果你用的是水果蜜饯之类的甜味混合食品，可以减少或省略)

将所有材料加入搅拌碗中，搅拌混合。尝一尝，如果你喜欢，可以加入更多的蜂蜜。将混合物转移到冰棒模具中，如果需要的话，使用漏斗和筷子，冷冻过夜。在热水下运行模具，以帮助释放 pops(如果需要的话),享受你尝过的最奶油、最辛辣的酸奶甜点。
