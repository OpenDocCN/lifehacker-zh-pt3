# 人们喜欢争夺的技术决策

> 原文：<https://lifehacker.com/the-tech-decisions-that-people-love-to-fight-over-1829142215>

技术的身份政治是真实的。我们变得痴迷于为我们的生活提供动力的设备，它们可以驱使我们与任何喜欢竞争产品、平台或服务的人进行激烈的争论——无论多么荒谬。



通过购买科技产品来过滤你的生活，把对你使用的设备或服务(或制造它的品牌)的批评误认为是对你生活选择的不必要的批评，这是非常容易的。别往心里去。这种激烈的竞争从特斯拉和爱迪生的时代就开始了，如果不是更早的话。

下面是一些比较著名的技术争论的入门书，你可能已经插上了你的旗帜，不管你是不是有意的。

### Windows 与 Mac

这是最古老的科技忠诚之一，苹果将它变成了经典的广告活动[。你是 PC 人还是 Mac 人？](https://www.youtube.com/watch?v=0eEG5LVXdKo) 

像 Android 和 iOS 一样， [Mac 与 PC 的争论](https://lifehacker.com/which-is-better-mac-or-windows-1792800437) 归结为“易用性”与“定制化”Windows 10 有一个可访问的顶层，但用户可以更多地访问底层的计算机核心系统(默认情况下)。MAC 的设计使系统的每个方面都直观、易用、平易近人。

为了确保这种体验，macOS 限制用户访问电脑的核心系统。有人甚至会说，在 Windows 上安装程序这个简单的动作要比在 Mac 上复杂得多——卸载也是如此，因为在苹果上，你只需直接从 Finder 中删除应用程序...好了，我们不再往火上浇汽油了。

一般来说，有某些类型的人倾向于被苹果电脑所吸引。创意专业人士。学生。重视形式胜于功能的人(或者形式*和*功能，取决于你的忠诚)。PC 也有一定的受众，包括程序员、游戏玩家和系统构建者。至少，我们想不到有多少人想从零开始打造新的 Mac 桌面。

随着智能手机取代个人电脑成为大多数人生活中的主要电脑，Windows 与 Mac 的辩论似乎已经成熟了几年。可以肯定的是，仍然有铁杆 Mac 和 Windows 粉丝，但似乎每个人都在很大程度上接受了这两个系统各有优缺点。当然，你可能永远也不会真正得到一个 Windows 版本的 of Boot Camp，但即使是这两个平台也能更好地协同工作。苹果似乎更愿意强调其设备能做什么，而不是如今的[Windows 设备不能做什么](https://www.youtube.com/watch?v=IN2COsYqTW8) 。

在我看来,“不和”的最大因素是 Windows 和 Mac 系统的差异足以让它们无法完全互换。当人们习惯了一个操作系统的细微差别时，另一个操作系统可能会感到无所适从——也许足以让他们讨厌它。如果是这样的话，你很可能会跑回你的舒适区，继续更激烈地捍卫你选择的电脑。

### iOS vs Android

对我们大多数人来说，智能手机是我们拥有的最重要的技术。它是我们交流的主要工具，是我们想要与世界互动时首先要做的事情(即使我们的好朋友就坐在我们旁边)。想想看，人们在辩论两大智能手机平台的优点时会变得非常愤怒:苹果的 iOS 和谷歌的 Android，前者为 iPhone 和 iPad 提供支持，后者几乎可以在其他任何平台上运行。

就像 PC 对 Mac 的辩论一样，iOS 和 Android 用户可能会在定制和易用性上产生分歧。与 iOS 相比，Android 提供了更多的透明度，并允许用户更多地调整他们的设备，而 iPhone 则以“工作正常”而闻名。另一方面，由于缺乏更好的方式来描述，这使得用户的体验“更简单”，阻止了他们在自己的设备上做各种有趣的实验。对于许多(大多数)人来说，这不是一个问题，但是那些更喜欢——或者坚持——拥有一条走出“围墙花园”的道路的人可能会因为 iOS 的局限性而感到窒息。

Android 用户也(正确地)指出，苹果在采用全行业硬件改进方面非常缓慢。以有机发光二极管显示器为例，或者我们一直看到的传说中的显示器内指纹识别器。最近，苹果在 [iPhone Xs](https://gizmodo.com/apple-iphone-xs-the-complete-rundown-1828979967#_ga=2.19954751.647277737.1537282231-396842925.1520800403) 、XS Max 和 Xr 上增加了双 SIM 卡插槽，这是 2010 年 的一个新颖想法。这是一个特别令人震惊的例子，但事实仍然是，大多数前沿技术都不会出现在 iPhone 上，直到苹果公司可以确定它可以工作，并大规模生产它。

另一方面，iPhone(以及延伸而来的 iOS)通常被称赞为的高级安全性。同样，iOS 应用商店虽然受到更严格的监管，但没有受到非法克隆和模仿的拖累，其中一些可能包含恶意软件。(当然，苹果的应用审查团队 [并不完美](https://gizmodo.com/top-apple-mac-app-secretly-sends-your-browser-history-t-1828881137) 。)

### 苹果 vs 三星

在 iPhone 与 Android 的战争中，一个具体且相当大的战线是 iPhone 和三星 Galaxy 系列智能手机之间的竞争。在过去的五年里，Galaxy 已经成为最受认可的 Android 手机，该公司迅速宣传 Galaxy S 是 iPhone 的一个很好的替代品

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-R59TevgzN3k&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-R59TevgzN3k&start=0) 

作为最受欢迎的两家智能手机制造商，苹果和三星多年来一直相互追逐，通过调整彼此的硬件功能来实现某种平等。当苹果在 2013 年将触控 ID 指纹传感器添加到 iPhone 5S 时，三星在第二年的 Galaxy S5 中加入了指纹传感器。巧合的是，三星也在 S5 中推出了集成无线充电功能，比苹果将其应用于其首款智能手机 iPhone 8 早了三年。

这些微小的差异是粉丝们争执的巨大燃料。Galaxy 用户对 iPhone 用户嘲笑苹果添加新功能的缓慢——或者在耳机插孔 的 [案例中，该公司对*删除*重要功能的快速反应。相反，苹果粉丝对批评者对 iPhone 最大成就的屏息赞美得意洋洋。](https://lifehacker.com/the-5-announcements-from-apples-iphone-event-that-actua-1829008155)

尽管存在军备竞赛和由此引发的争吵，但与更大的 iOS 与 Android 之争相比，特定设备的对决总是处于次要地位。让 Galaxy 和 iPhone 与众不同的许多功能往往是 iOS 和 Android 的复杂之处。Galaxy 已经成为转向 Android 的门户设备，就像 iPhone 曾经是门户智能手机一样。

### 索尼对微软

众所周知，游戏玩家是这个星球上最微不足道的科技爱好者。我是带着爱这么说的，但自从世嘉(Sega)吹嘘创世纪(Genesis)做了“任天堂没有做的事情”以来，专用游戏平台粉丝之间的“主机战争”(console wars)一直在持续

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-65E53rNC1io&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-65E53rNC1io&start=0) 

通过选择一个控制台而不是另一个，你表明了一种控制器偏好，优先考虑一组独占游戏，并可能限制你的社交圈，因为大多数多人游戏不允许不同控制台之间的“交叉游戏”。从历史上来看，玩家们对这些选择都非常谨慎。

今天，这场战斗基本上是双方面的。你有索尼 PlayStation 4 和微软 Xbox One 的粉丝。任天堂还生产一款游戏机，即广受欢迎的任天堂 Switch，但该公司目前基本上没有受到关注。(鉴于其较低的硬件规格、不合规的硬件外形和恼人的友好代码，将该交换机与其他控制台进行比较是一个特别愚蠢的主张)。由于大多数游戏都在这两个平台上出现，这种区别主要归结为少数游戏。

不过，下一代游戏机即将问世。微软 Xbox 部门的负责人菲尔·斯潘塞说，下一代 Xbox 正在进行中。据推测，索尼将在同一时间推出新的主机。你可以期待很快看到大量的“PS5 对 Xbox 4”的讽刺文章。

### **AMD vs 英伟达**

在电脑游戏玩家中，永无止境的平台之争集中在你的显卡上。它无疑是系统构建者最常讨论的 PC 组件，因为它对游戏的外观和感觉至关重要。

AMD 和 Nvidia 是两大显卡厂商。尽管从这两家公司中选择一款显卡不会阻止你与朋友一起玩游戏或诸如此类的无聊之事，但在希望尽可能构建最好(或最快)系统的 PC 制造商之间存在着固有的竞争，这导致粉丝们选择支持哪一方。

也就是说，这种竞争近年来有所减弱。虽然 AMD 为许多设备提供显卡——包括 [、PS4](https://www.sony.co.uk/electronics/playstation-systems/playstation-4#product_details_default) 和[Xbox One](https://www.amd.com/en/xbox-one-case-study)——但 Nvidia 的显卡已经占领了市场。人们普遍认为 Nvidia 的显卡更强大，而该公司 [很乐意围绕这一事实收取溢价](https://gizmodo.com/what-on-earth-makes-a-1-200-nvidia-2080-ti-graphics-ca-1829126369) 。虽然许多出版物可能 [认为](https://www.pcgamesn.com/amd-vs-nvidia) 中低端 AMD 卡比 Nvidia 卡更有价值，但希望在最新游戏中获得每秒最多帧数的游戏玩家更有可能选择 Nvidia 而不是 AMD。

然而，现状可能很快就会改变，因为 Nvidia 的下一代 20 系列显卡一如既往地昂贵——其顶级显卡高达 [令人瞠目结舌的 1200 美元](https://gizmodo.com/nvidias-beastly-new-20-series-rtx-gpus-claims-up-to-6x-1828464202) 。这种定价可能会促使更多考虑跳槽到 Nvidia 卡的玩家重新考虑他们的购物清单(或者考虑交叉火力设置而不是全新的 Nvidia 卡)。

由于电脑游戏玩家对加密货币矿工的仇恨增加，也出现了暂时停火。在过去的 2-3 年里，由于日益雄心勃勃的加密货币矿商的贪得无厌的需求，这两家公司的高端 GPU 变得非常难以获得，他们一直在购买高端显卡以扩大他们的采矿业务。值得庆幸的是，即使是 [Nvidia 自己](https://www.fool.com/investing/2018/08/22/the-cryptocurrency-bubble-is-over-for-nvidia.aspx) 也认为加密推动的 GPU 淘金热正在结束，因此 PC 游戏玩家将很快能够再次将他们的怒火集中到彼此身上。

### Chrome vs. Firefox

虽然这种竞争很大程度上被归入论坛和互联网的书呆子角落，但有些人对你应该使用什么样的浏览器有非常强烈的感觉。你有几个选择，但鼓通常围绕两个主要阵营:谷歌 Chrome 和 Mozilla 最近全面改革的 Firefox Quantum 浏览器。

不像这些争斗，它们实际上可以渗透到随意的谈话中和/或对你的日常生活产生某种影响，网络浏览器的争论始于性能(以及你是否信任谷歌)。

Chrome 和 Firefox 的外观和感觉非常相似，但 Chrome 有一个致命弱点——它是一个巨大的内存猪。Mozilla(和许多用户)声称 Firefox 通常比 Chrome 快，但在 Chrome 不知何故占用了你所有内存的情况下，这种差距是明显的。(也就是说，Chrome 加载网站和网络应用的速度更快，所以...)

许多用户容忍 Chrome 的内存问题，因为他们对浏览器感到舒适，他们喜欢它与谷歌其他网站和服务的集成方式，或者他们实际上将自己的浏览内容传输到谷歌 Chromecast 等设备。然而，这种集成对于一些注重安全性的用户来说是一个严重的问题。在谷歌搜索、Chrome、Gmail、Android 和谷歌助手之间，他们通常不喜欢谷歌在你与互联网交互的每一个方面收集多少关于你的数据。另一方面，Mozilla 的 [显得更不利于在线追踪](https://blog.mozilla.org/futurereleases/2018/08/30/changing-our-approach-to-anti-tracking/) 。
