# 如何在大学课本上省钱

> 原文：<https://lifehacker.com/how-to-save-money-on-college-textbooks-1728445882>

大学课本购买对每个学生来说都是一项昂贵的例行公事。但是富有同情心的教授和容易获取信息的工具可以让人们很容易找到削减成本的方法。利用这些工具，用它们来为你的课本开支节省一些钱。



***这个帖子最初出现在*** [***这个简单的美元***](http://www.thesimpledollar.com/how-to-save-money-on-college-textbooks/) ***。***

在过去的几天里，我收到了一些大学生的来信，他们正面临大学秋季学期的开始，正在寻找削减教科书费用的明智策略。下面是来自 Jen 的一条示例消息:

我刚刚给我秋季学期的书定价，在校园书店总共是 950 美元。即使在亚马逊上，它们也真的很贵。一定有更好的方法来做这件事。

你说得对，珍。有一种更好的方法可以做到这一点。在过去的几年里，我有很多年轻的朋友和家人向我寻求帮助，为他们寻找便宜的教科书，这里是我们使用的策略，真的很有效。

### 先检查一下你的教学大纲

人们在购买大学教科书时通常会去大学书店——或者，现在更常见的是，去他们大学的网上书店——搜索他们的每一门课。对于每一本书，可能会列出一两本或三本书，所以你只需将它们添加到购物车并付款。

那是一个错误。你应该看的第一个地方是你的课程大纲，上面列出了这门课的课本。

为什么要找大纲？教学大纲通常会指出哪些书是必修的，哪些是选修的。它通常还会指出是否需要最新版本的书，或者旧版本是否可以接受。

仅仅通过查看教学大纲，你就可以减少需要购买的书籍数量，同时，增加可以满足你需求的版本数量，这样更容易找到二手书。

### 询问教授旧版本是否可以接受

如果一位教授有一份教学大纲，列出了教科书的具体版本，给你的教授发一封简短的电子邮件(或者，如果你第一天上课，请举手)。简单地问一下以前版本的教材是否可以接受，如果可以，是哪个版本。

在我的许多课上，在某种程度上，以前版本的教科书是完全可以接受的。通常，一本书的每个版本都有一些变化，但是在许多版本中，这些变化积累起来，并显著地改变了教科书。

大多数教授都不介意学生使用这本书的前一版本，甚至更早的版本。你只需要确定哪个版本真正适用于这个类。

### 检查大学和社区图书馆

一旦你确定你知道这门课需要什么书，就开始查大学和社区图书馆。

对于很多课程来说，这不会有什么帮助，但是对于一些课程，比如文学课或者历史课，图书馆实际上可能有你要找的书。

如果你能从图书馆找到你需要的书，这本书就免费了，至少在一段时间内是免费的。我的一个朋友从社区图书馆为他的英语文学课程借了一本书，并续借了两次，这意味着他可以在课程需要时一直保留这本书。

### 用“占领书店”来比较价格

所以，你终于意识到你不得不为你的一些书付钱。如果这是你的情况，我强烈推荐使用 [占领书店](http://occupythebookstore.com/) ，这是一个免费的谷歌 Chrome 插件，将使整个过程变得非常非常容易。

只需通过上面的链接将扩展添加到你的浏览器中，然后进入你的大学书店的网站。查找你上课需要的书。当你访问你需要的每一本书的页面时，扩展会弹出，给你一长串其他地方购买这本书的选项——默认情况下，这个列表是按价格排序的。

我通常建议人们做的是使用 [占领书店](http://occupythebookstore.com/) ，找到他们想要的书，然后浏览列表找到他们信任的第一家商店(比如说亚马逊)，并从那里购买。

### 租，不要买，你大部分的书

许多在线教科书资源，如 [Chegg](http://www.chegg.com/) ，提供*租赁*而非购买。这是一个很简单的过程，真的——你支付一个比平时更低的价格，他们把书寄给你，你必须在学期结束时把书送回来。然后，他们可能会在下学期开始时把这本书再次“出租”给别人。如果你意识到你确实需要这本书超过一个学期，大多数书店通常会有“购买”选项。

对于你的大多数课程来说，你实际上不需要或不想要这本书超过一个学期，所以这种服务不仅可以为你节省大约 60%的课本费用，还可以在学期结束时为你处理掉课本，这样你就不必处理转售的问题。

实际上，在我大学生涯快结束的时候，我用一个非常早期的教科书租赁服务租了几本教科书，发现它非常有用，而且非常省钱。

### 如果你必须买，尽量买二手的

如果出于某种原因，你不得不买一本教科书，而不是简单地去租一本，试着找一本用过的版本。包括亚马逊在内的大多数在线教科书销售商以折扣价提供二手版本的教科书。

当然，有时你会得到一本有边注和亮点的书，但这些东西很容易被忽略，尤其是因为你真的只是在看这本书的文字和图像。

买一本二手版本的教科书会给你省下一大笔钱。我在大学时基本上是二手教科书的购买者，我经常发现我通过购买二手书籍节省了高达 60%的钱。(当然，那时候，我大多不得不依靠钉在校园公告板上的笔记，而不是互联网。)

### 如果你必须买新的，试试电子书

有时候，最糟糕的情况会发生。一个班级需要一本全新版本的教科书。用过的没有了。教授不允许这本书的旧版本，并且有要求新版本的作业。

它发生了。令人沮丧。不管你做什么，它都会让你的钱包鼓起来。

这里的解决方案也很简单。如果你发现自己处于那种情况，就买电子版的教材。电子版通常比印刷版便宜得多，而且在你学习的时候搜索特定的术语也容易得多。

我是个守旧派——当我学习的时候，我仍然喜欢印刷品。然而，我也研究过使用电子资源，老实说，如果你处于学习的心态，知道如何关闭干扰，它们工作得很好。

### 与朋友分享书籍

另一个值得考虑的方法是和一个和你上同一门课的朋友共用一本教科书。如果你们两个都选了那门课，你可以只买一本教材，然后分享，分摊费用。

这种方法最大的问题是，这学期可能会有一两个关键点，你们都想同时使用那本教科书。例如，在考试的前一天晚上，你们可能都想用那本书。

这里的解决方案是一起学习，所以如果你考虑遵循这种计划，确保和你一起购买课本的人是一个你可以一起以有益的方式学习的人。

### 在学期结束时卖掉你不需要的书

在学期末，如果你没有买电子书或租课本，你将面临一个决定。我是否需要保留这本教科书？对我未来的课程或事业有帮助吗？

如果你决定留下它，很好。在我早期的职业生涯中，我发现几本教科书是有用的参考资料，很高兴我把它们留在身边。

如果你决定不保留它们，在学期之间卖掉它们。卖书的途径有很多，但去人多的地方，在像 [亚马逊](http://www.amazon.com/?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/how-to-save-money-on-college-textbooks-1728445882&asc_source=&tag=kinjalifehackerlink-20) 或 [Craigslist](http://www.craigslist.org/) 这样的人流量大的地方卖书是有意义的。

你不会收回所有的投资，但你会收回一部分，这些钱会及时用于购买下学期的书籍。

### 最后的想法

大学课本购买对每个学生来说都是一项昂贵的例行公事。教科书制造商收取的价格高得离谱，这确实会增加中学后教育的经济负担。

然而，我们生活在一个(大部分)富有同情心的教授和容易获得信息工具的世界，这些工具可以很容易地找到和购买二手教科书。利用这些工具，用它们来为你的课本开支节省一些钱。

如果这样做可以帮助你减少学生贷款负担，即使每学期只减少几百美元，在大学期间会增加几千美元，在随后的几年里会节省几千美元的利息。

祝你好运！
