# 简单的工具条桌面

> 原文：<https://lifehacker.com/the-simple-sidebar-desktop-1778987386>

这个 chill，cityside beach 桌面是由读者 [Yuki](https://www.flickr.com/photos/127192490@N06/) 提交给我们的桌面展示和讲述池的，尽管它不是精心制作的，但它看起来仍然很棒。以下是你也可以设置它的方法。



Yuki 正在 Windows 上使用 rainmet 来获得这种效果，所以如果你不熟悉它，你可以在这里 查看我们的 rainmet 入门指南。

在此基础上，以下是让您的电脑拥有相同外观所需的组件:

*   [壁纸](https://unsplash.com/photos/o90ghyPAbLw)
*   用于 Windows 的 [雨量计](http://www.rainmeter.net/) 系统调试和监控工具
*   [谜](http://enigma.kaelri.com/) 日历和左侧栏的雨量计皮肤
*   Gmail 插件的 [雨量计](http://poiru.deviantart.com/art/Gnometer-1-3-0-3-182181512) 雨量计皮肤
*   [毛毛雨](https://rainmeterdesk.blogspot.com/2015/06/drizzle-v10.html?m=1) 为雨量计的时间和日期效果皮肤

这就是这个的全部内容。虽然没有太多的移动部件，但效果相当不错，尤其是如果你是那种怀念早期 Windows 版本的侧边栏，并且想在你的桌面上恢复它的人。如果你喜欢它的外观，请访问下面的 Flickr 链接，让 Yuki 知道你喜欢这个桌面——是我们让你来的！
