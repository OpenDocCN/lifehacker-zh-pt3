首先，一个很大很大的免责声明:未经他人允许监控他人的电脑不仅仅是错误的，它可能是联邦犯罪。只有几个*非常* *特定的条件*允许你监控别人使用的电脑。例如，雇主可能被允许监控某些东西，但只能监控公司拥有的硬件。父母可以追踪孩子的活动，因为他们对孩子负有法律责任。然而，在这种情况之外，没有太多回旋的余地，也不应该有。

即使在可能合法的情况下，如果你不小心的话，你仍然可能面临隐私诉讼。你可能最终会通过日志软件记录员工的脸书密码，但如果你随后用它来访问他们的私人信息，那就越界了。

最后，在别人不知情的情况下跟踪他们(即使这样做合法)几乎肯定会侵犯他们的信任。如果你是有一个十几岁孩子的家长，监控他们的网络活动可能是可以接受的，但你可能会疏远他们。法律和道德上的理由并不能修复破裂的信任。在继续之前，也要考虑跟踪的社会成本。

### **首先，确定你想跟踪什么(以及是否跟踪)**

发现别人在做什么并不像说“计算机！看好他们，向我汇报！”你需要真正知道你在寻找什么。如果你是家长，你可能想给你的孩子隐私，同时确保他们不会访问互联网上的不良内容。如果你是一个需要监控员工的不道德行为的雇主，你可能想要记录所有的事情。一般来说，监控分为三类:

*   注意某项活动。如果你通常不在乎某人使用设备做什么，只要他们遵守规则，这是非常理想的。例如，您可能想要查看员工何时使用脸书，或者孩子何时观看色情内容。有时这可以通过过滤来完成，而不是监视，所以尽量考虑所有选项。
*   **监控某些应用程序中的所有活动。**雇主可能想监控某些应用，但不想窥探他们的脸书。你可能只关心他们如何处理公司的电子邮件、聊天应用或专有软件。在这种情况下，更多的选择性监测可能会更好。
*   监控每个应用程序中的所有活动。在这种情况下，你可能希望看到某人正在做的每一件事。这可能包括监控孩子的聊天记录，如果你担心他们可能会做一些见不得人的事情，或者监视你认为可能从公司偷窃的员工。

非常明确地决定要跟踪什么非常重要，因为跟踪太多会导致法律问题。如果你只需要让员工远离某些网站，不要使用键盘记录器。如果你想让孩子远离色情，你可能不需要监控每一条脸书的信息。保持你的跟踪最小化，以避免任何复杂的事情发生。

### **选择合适的软件**

一旦你决定了如何以及为什么要监控别人的电脑，你就需要找到合适的软件。虽然有大量的监控工具，但我们有一些来自知名组织的建议。记住，你在和监控软件打交道。你也许能从一个随机的开发者的网站上找到一个免费的键盘记录器，但是你不能总是知道那个工具是否会偷偷打电话回家。在这种情况下，最好还是选择那些你知道可以信任的老牌公司。

#### **监控孩子的活动:家长控制工具(Windows/OS X)**

Windows 和 OS X 都有内置工具，允许成年人监视孩子的活动。您可以使用这些来监控和过滤您孩子的活动。有几个关键的不同，但是两个平台之间有一些共同的特性。以下是这些功能的分解情况:

*   阻止或监控网站和应用程序的使用(两者)
*   设定电脑使用的时间限制(两者)
*   按年龄限制筛选应用程序和游戏(两者)
*   在字典中隐藏亵渎(OS X)
*   阻止某些硬件，如照相机(OS X)
*   阻止通过消息和电子邮件等特定应用程序与某些人交流(OS X)

要使用这些功能，这两个平台都要求您设置一个儿童帐户。您可以在 OS X 上点击 或在 Windows 10 上点击 找到如何创建新的孩子账户 [的信息。您还可以在这些链接中找到有关如何管理每个平台设置的信息。](https://support.apple.com/kb/PH18571?locale=en_US&viewlocale=en_US)

### **跟踪员工的生产力:**[**activ trak**](http://activtrak.com/)**(Windows/OS X)**

如果你是一个想要更详细了解你的员工在做什么的雇主，ActivTrak 是一个不错的解决方案。它允许你同时监控几个员工的电脑，并提供用户活动的详细报告。以下是一些关键特性:

*   记录应用程序使用的时间和频率的统计数据。
*   定期截屏用户活动。
*   生成图表，直观显示在各种应用程序中花费的时间。
*   当用户开始特定的有害活动时，发送警报消息。
*   如果公司使用 ActiveDirectory，可以远程安装(仅限付费版本)。

作为管理员，您可以登录到基于 web 的控制面板，在设置好办公室的计算机后几分钟内查看活动使用统计数据。你可以根据办公室里有效率的和无效率的东西来设置自定义类别。例如，你的社交媒体实习生使用脸书可能是“高效的”，但案例经理可能不是。您还可以使用“报告”选项卡来查看您的一名或所有员工最常用的应用程序和网站。

该服务的免费层允许您用一个管理员和有限的存储量来监控三个用户(称为“代理”)。您可以付费添加更多代理或新功能，如远程安装程序。付费等级从 5 个代理每年 336 美元开始，到 50 个代理每年高达 1920 美元。显然，除非您知道您的业务将需要它，否则您不会想要更高的层。

### **调查一名员工的潜在不法行为:**[**Spector**](http://www.spectorsoft.com/)

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-QRm0_Tylc8U&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-QRm0_Tylc8U&start=0) 

如果你非常怀疑某个员工可能在做违法或违反公司政策的事情，SpectorSoft 有一套庞大的软件，你可以用它来进行短期和长期的调查。基本 Spector CNE 调查器专为短期调查而设计，配有一系列工具:

*   每 30 秒截图一次。
*   使用关键字或打开特定应用后触发截图。
*   跟踪网站使用情况、在网站上花费的时间以及上次访问的时间。
*   存储和归档公司电子邮件。
*   记录几乎所有消息平台的即时消息聊天日志。
*   跟踪下载或上传到计算机的文档的名称、文件类型和时间。
*   记录用户机器上的所有击键活动。或者，您可以选择不记录密码。

SpectorSoft 还提供 Spector 360，其中包括分析用户活动的高级报告，在所有平台中查找特定可疑单词或短语的全局关键字搜索，以及网页过滤。该公司表示，Spector 360 是为更长期的调查和监控而设计的，因此大多数雇主可能不会介意 CNE 调查员。

所有 SpectorSoft 程序都是付费应用程序，你只能通过打电话询问报价来获得价格。自然，这意味着它可能相当昂贵，这不是你的普通人想要的那种东西。然而，如果你正在经营一家 [不断泄露预告片](http://io9.com/marvel-subpoenas-google-over-the-avengers-age-of-ultro-1655853984) 的大型工作室，这可能是你感兴趣的事情。
