# 这张图表显示了在预防癌症中真正重要的东西

> 原文：<https://lifehacker.com/this-graphic-shows-what-really-matters-in-preventing-ca-1826609707>

健康的生活方式有助于预防癌症。(然而，没有癌症是完全可以预防的；遗传和运气不好也是因素之一。)但是哪些生活方式因素真的很重要呢？世界癌症研究基金会有一个 [得心应手的图形](https://www.wcrf.org/dietandcancer/interactive-cancer-risk-matrix) 来指出它们。



唯一令人信服地降低癌症风险的是体育活动(降低结直肠癌风险)和步行(减少体重增加，一些癌症与体重增加有关)。图表上的下一行圆圈是与癌症风险密切相关的事情，包括吃蔬菜、母乳喂养，以及一些与健康饮食和锻炼有关的显而易见的事情。(他们 [在这里解释他们如何判断证据](https://www.wcrf.org/dietandcancer/judging-evidence) 。)每个圆圈的大小代表它可能影响多少癌症病例，因此较大的圆圈可能对您的风险产生最大影响。

在图表的较低层次，情况变得更加模糊。维生素 D 属于“有限提示”类别，这意味着有一些证据表明它可以帮助预防结肠直肠癌，但还不足以真正确定。接下来，图表会涉及到可能会增加患癌症风险的事情。令人信服地增加风险的因素包括:酒精饮料、成年后身体肥胖、饮用水中的砷和高剂量β-胡萝卜素补充剂。

一些因素会增加一种癌症的风险，但会降低其他癌症的风险。例如，成年人身体肥胖可能会降低绝经前乳腺癌的风险，但会增加患其他癌症的风险。(你可以在这里 查看最强证据的 [汇总图。)](https://www.wcrf.org/sites/default/files/CUP-Strong-Evidence-Matrix.pdf)

没有办法完全战胜癌症，但这个工具可以帮助你了解哪些因素最有证据表明它们与癌症风险有关。如果你想要一个更简单的清单来说明你应该做些什么来降低你的风险，世界癌症研究基金会在这里有一个建议清单。以下是主要项目:

*   保持健康的体重。
*   锻炼身体。
*   吃蔬菜。
*   限制快餐。
*   限制红肉和加工肉类。
*   限制含糖饮料。
*   限制饮酒。
*   不要依赖补品。
*   如果可以的话，母乳喂养你的宝宝。
