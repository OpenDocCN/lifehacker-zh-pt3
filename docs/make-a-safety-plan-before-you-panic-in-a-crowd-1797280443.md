# 在人群中惊慌失措之前，制定一个安全计划

> 原文：<https://lifehacker.com/make-a-safety-plan-before-you-panic-in-a-crowd-1797280443>

如果你被人群吓到了，你可能都不知道。你可能会紧张或生气。(在我注意到这种模式之前，我已经在农贸市场烦躁了好几年。)而某些拥挤的地方，比如时代广场或空荡荡的体育场，会让任何人感到害怕。正如丽兹·拉斯基博士在《纽约时间》、、[中所说，有一个安全计划](https://www.timeout.com/newyork/blog/we-tested-out-tips-for-calming-down-in-the-middle-of-times-square-072517)、是有帮助的。



穿过人群的一个基本方法是盯着你的目的地。这向别人指明了你的道路，帮助你专注于目标而不是障碍。

结合拉斯基博士的建议和我们自己的一些建议，这里有更多的方法可以让你不惊慌地穿过人群:

*   吃过东西后到达:你的水分、食物和着装越好，你就能更好地控制天气。带上一些水或者零食。
*   设定见面地点:如果你和别人一起旅行，商定好你的目的地，如果需要分开，在哪里见面(人群之外)。
*   **寻找休息站:**寻找可以躲避人群的地方，喘口气。
*   **关掉它:**按照拉斯基博士的建议，打开一些音乐，让你的大脑活动起来。试着随着音乐移动；把你穿过人群的动作当成一种微妙的舞蹈。这感觉很傻，但它能帮助你放松。
*   **背诵一段口头禅:** *托尼*作家杰米·卢茨走过时代广场重复着“这很好”，就像来自迷因 的卡通狗一样，发现效果不错。事情并不顺利，你可能会觉得自己快被活活融化了，但你不一定要开心；你只需要挺过去。
*   奖励自己:一旦你走出人群，深呼吸，吃点点心，祝贺自己。在你的脑海中真实地想象一下，计划、维持和走出压力情境的另一端的感觉有多好。这会帮助你下次记得做同样的事情，即使你措手不及。
