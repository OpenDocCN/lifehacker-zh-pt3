# 当 Windows 不能删除文件时，Lockhunter 允许您删除文件

> 原文:[https://life hacker . com/lock hunter-let-you-delete-files-when-windows-wont-1825467437](https://lifehacker.com/lockhunter-lets-you-delete-files-when-windows-wont-1825467437)

自从我们上次使用 Lockhunter 以来，已经过去了将近十年，这是一个非常有用的 Windows 应用程序，它可以确保您始终能够删除文件资源管理器拒绝删除的文件和文件夹。开发者 Crystal Rich Ltd 继续用有用的功能更新该应用程序，但只是零星地更新:从该应用程序的 3.1 版本到 3.2 版本花了大约 3.5 年的时间。

Watch

尽管如此， [Lockhunter](http://www.lockhunter.com/index.htm) 还是值得保留在你的 Windows 桌面或笔记本电脑上，以备那些罕见的时刻——或者不太罕见，如果你运气不好的话——当 Windows 因为某个文件夹或文件被其他进程锁定而拒绝让你操作它的时候。有时，您可能需要关闭一个有问题的程序来重新获得对文件的访问。其他时候，你可能需要采取更激烈的措施。

*(Hat tip to life hacker reader**GH Dave**for*[*提醒我们这个有用的 app 存在*](https://lifehacker.com/1825415406) *。)*

你只需要使用一个安装程序来安装 Lockhunter，因为该应用程序现在可以检测你运行的是 32 位还是 64 位版本的 Windows，并安装正确的应用程序版本。之后，如果您遇到一个顽固的文件或文件夹，您可以在文件资源管理器中右键单击它，并选择新的选项，“什么是锁定这个文件？”

当 LockHunter 加载时，您将得到一个快速的故障分析，为什么您可能无法操作有问题的文件或文件夹。(您还可以单击新添加的“更多详细信息”选项来查看有关进程的更多具体信息，然后您可以在命令提示符中使用“ [Taskkill](https://lifehacker.com/create-a-shortcut-that-quits-every-running-windows-prog-5670054) ”命令来终止这些信息。)

LockHunter 可以尝试代表你解锁或删除文件，如果这些选项失败，你也可以让应用程序在下次 Windows 重新启动时尝试删除。

虽然 Lockhunter 执行起来很简单，但这个设计良好(尽管很少更新)的应用程序值得放在你的数字工具箱里。只是不要用它的“检查更新”功能——到 Lockhunter 3.3 推出的时候，世界很可能会变得非常非常不同。我们会通知您何时寻找新版本的应用程序。
