# 三星最新的平板电脑试图超越 iPad

> 原文：<https://gizmodo.com/samsung-s-latest-tablet-is-trying-to-out-pro-the-ipad-1828021569>

如今，平板电脑处境艰难。你是廉价推出本质上是大手机的产品，还是提高价格，寻求额外的功能，如手写笔和键盘支持？在过去的几年里，三星试图通过制造更接近可转换笔记本电脑而不是标准平板电脑的平板电脑来充分利用 Android。对于 2018 年，三星将通过即将推出的 Galaxy Tab S4 变得更大。



由于采用了新的 10.5 英寸 AMOLED 显示屏，Tab S4 不仅在物理上比其前身更大，其电池也增加了 20%，达到 7300 毫安时，因此其寿命也应该会延长。然而，Galaxy Tab S4 最大的变化是加入了 Samsug 的 Dex 软件，无论你在哪里，都可以提供接近真实桌面的工作体验。

以前， [Dex](https://gizmodo.com/samsung-dex-comes-heartbreakingly-close-to-turning-a-ph-1795028661#_ga=2.262396748.1957076736.1533062852-75259643.1519645932) 的设计是为了让像 [Galaxy S9](https://gizmodo.com/samsung-galaxy-s9-review-the-best-android-phone-dont-n-1823575944) 和 [Galaxy Note 8](https://gizmodo.com/samsung-galaxy-note-8-review-bigger-badder-but-i-sti-1799759566) 这样的手机能够通过适当的外部坞站连接鼠标、键盘和显示器来执行传统的笔记本电脑任务。但现在，在 Galaxy Tab S4 上，Dex 已经演变成一个更简单的解决方案，使你能够在运行纯 Android 的标准平板模式和 Dex 支持的桌面模式之间切换，这种模式的功能就像 Chrome OS 和 Windows 10 之间的融合。

除了完全支持所有你最喜欢的 Android 应用程序之外，在 Dex 模式下，你还可以在底部获得一个任务栏和通知抽屉。你甚至可以在桌面上看到常用程序的图标。正如你对普通笔记本电脑的期望一样，你可以在后台运行多个程序(最多 20 个)，可以轻松调整窗口大小，可以随意拖放文件。当你想切换回平板电脑模式时，只需点击 S4 通知面板上的一个按钮，或者将它设置为在断开键盘连接时恢复原状。

盒子里有 Tab S4 的手写笔，三星说它从屏幕上吸取能量，不需要充电。就像我们在去年的 Tab S3 和之前的 Galaxy Note 手机上看到的那样，S4 的手写笔可以处理许多技巧，例如屏幕关闭备忘录——让你无需解锁设备就可以记下笔记——Air Command 可以满足你所有的截图和即时 gif 制作需求，甚至可以进行实时翻译。

不幸的是，Tab S4 的键盘没有捆绑销售。相反，这是一个额外的花费 150 美元。这真的很遗憾，因为可选键盘实际上感觉很好。它的按键行程和弹跳比你从苹果的 iPad Pro 智能键盘上获得的还要多，尽管三星也犯了一个错误，没有内置触摸板。当然，你可以随时将自己的键盘融入其中，但要真正最大限度地发挥 Tab S4 的移动生产力，键盘似乎是必不可少的。对于那些真正想要获得完整桌面般体验的人来说，你可以使用 USB-C 转 HDMI 转换器(另一个可选配件)将 Tab S4 连接到外部显示器，以便在前面获得大屏幕视图，而 Tab S4 的显示器则充当辅助屏幕。

至于 Tab S4 的其他规格，它的组合有点令人印象深刻:一个更老的 Snapdragon 835 芯片，4GB 内存，64GB 存储，1300 万像素后置摄像头和微可扩展性。这与去年 Galaxy Note 8 上的组件基本相同，但少了 2GB 内存。也没有内置的指纹识别器，这意味着你必须依赖虹膜识别器或坚持使用老式的 PIN 或密码。

但三星为什么不干脆用 Chrome OS 之类的东西呢？你仍然可以获得对 Android 应用程序的支持，以及为提高工作效率而设计的用户界面，但是没有那么复杂的灵活性。三星的一名代表告诉我，作为一款平板电脑，Android 对 Tab S4 更有意义，三星“致力于”Android。至于我，我不太相信。在这一点上，很明显，在苹果之外，平板电脑正在转向 Chrome OS。

我主要关心的是 S4 的价格。不带键盘盖和显示器加密狗的起价为 650 美元，Tab S4 比 Surface Go 贵得多，与 9.7 英寸的 iPad Pro 价格差不多。在我使用 Tab S4 的短暂时间里，它感觉是一款完全胜任的设备，加上 Dex 使执行某些注重生产力的任务变得更加流畅，它绝对是一款比 Tab S3 更全面的设备。但我仍然觉得三星可以做得更好。无论哪种方式，敬请期待更深入的印象。该设备将于 8 月 10 日上市。
