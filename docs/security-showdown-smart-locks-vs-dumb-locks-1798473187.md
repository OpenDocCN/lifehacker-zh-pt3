# 安全摊牌:智能锁与“哑”锁

> 原文：<https://lifehacker.com/security-showdown-smart-locks-vs-dumb-locks-1798473187>

一把合适的锁是家庭安全的重要组成部分，可以阻止不必要的入侵者，从四条腿的害虫到潜在的窃贼。虽然我们肯定不能敲经典的死栓，但有些锁比其他锁有更多的技巧。新上市的智能锁有足够酷的功能来吸引每个人，从 [Airbnb 业主](http://gizmodo.com/smart-locks-used-by-airbnb-get-bricked-by-software-upda-1797839523) 到亲密的家庭。然而，与传统的锁相比，它可能会给你带来更多的麻烦，而你不愿意以牺牲便利为代价来处理这些麻烦。



### **比赛**

锁，不管是智能的还是智能的，都是为了防止入侵者和其他不受欢迎的人进来，同时让合适的人进来而不会有太多麻烦。不管是实体的还是数字的，你仍然需要某种钥匙来进入。

#### **传统锁**

无论你想称它们为愚蠢的、传统的还是模拟的，普通的门锁只做一件事，而且只做好一件事:防止东西进入。虽然传统锁的内部结构因制造商、安全等级或锁定机制而异，但它们通常是用钥匙打开的。

#### **智能锁**

智能锁利用了你一直随身携带的东西:你的智能手机。无论您是通过蓝牙连接，使用地理定位识别您何时在家，还是通过支持 Wi-Fi 的应用程序控制锁，您都可以使用您的 [智能锁和智能手机](http://gizmodo.com/august-smart-lock-and-connect-review-ill-use-keys-tha-1685319060) 配合使用，无需钥匙即可开门。现有的智能锁品种相对较少，部分原因是它的新颖性和相对市场新人的地位。

### **智能锁是我们还没有完全准备好的未来**

说到方便，智能锁胜过传统锁。准备好花 175 到 230 美元去偷一个吧。一些智能锁可以通过相应的应用程序开门，让你授权人们在几英里外进入。这种便利，加上其他酷功能，如“临时”钥匙和基于地理位置的自动锁定，使它成为当今物联网社会的完美锁。不幸的是，它们就像我们使用的其他智能家居技术一样不安全。

智能锁不仅容易受到恶意方的攻击，还可能被公司自己禁用，这取决于所涉及的软件。最近，智能锁公司 Lockstate 通过一次拙劣的软件更新，意外地将数百把自己的智能锁 砌砖。这些锁是 Airbnb 推荐给房东使用的，让租房者被锁在他们的临时住所之外，几乎没有追索权。当我们问一群安全专家他们自己是否会使用智能锁时，我们得到了不同的回答。不是什么好兆头。

### **只要你有一把额外的钥匙，传统的锁就可以用**

只要你有自己的钥匙，传统锁提供的简单进入是很方便的。购买传统锁时，可供您选择的数量几乎是无限的，您可以根据自己的安全需求很容易地找到一种。传统锁的价格从 20 美元到 100 美元不等。 [破解传统锁](http://lifehacker.com/use-an-electric-toothbrush-to-pick-locks-1706031244) 也比黑智能锁更难。首先，你需要在锁旁边，而不是在几英里外的电脑上。

然而，传统锁失败的地方正是智能锁擅长的地方。如果你的朋友想在你外出时进来，他们必须有自己的钥匙来打开你的哑锁。如果你不能见到他们，你需要把它放在一个不显眼的地方，以免别人发现(请不要把它放在垫子下面)。这种程度的不安全感可能足以让人们放弃传统的锁，但是一点点计划(和家里一两把额外的钥匙)往往可以很容易地解决这个问题。

### **结论:智能锁很有用，但还没准备好迎接黄金时代**

我最近用一种更好的传统插销锁替换了我前门的薄锁。虽然我确实考虑过智能锁，但我不想应对可能无法进入自己家的情况，这要感谢一些在线黑客，一家公司推出了 [有问题的软件更新](http://gizmodo.com/smart-locks-used-by-airbnb-get-bricked-by-software-upda-1797839523) 。此外，向我的房东解释智能家居技术会是另一件麻烦事，尽管他性情随和。

虽然给像 [灯泡](http://lifehacker.com/manage-your-smart-lightbulbs-with-emoji-stickers-1796724827) 、手表甚至安全摄像头这样的设备添加智能是有道理的，但相信一个新生而昂贵的安全系统可以进入你的家是你应该避免的，至少现在是这样。如果你对整个“未来之家”的生意是认真的，那么考虑一个值得信赖的锁品牌的智能锁，而不是一个新成立的公司。
