# 你可以用 Android 的隐藏系统 UI 调谐器做的最酷的事情

> 原文：<https://lifehacker.com/the-coolest-things-you-can-do-with-androids-hidden-syst-1773364942>

谷歌在 Android 棉花糖中引入了一个甜蜜的隐藏菜单，称为系统 UI 调谐器。它包含了一大堆巧妙的小调整，比如隐藏状态栏图标或者显示你的电池电量。下面是如何打开它，以及你可以用它做什么。



系统 UI 调谐器包含了大量的小选项，可以让你的手机更好用。要使用它，你需要安装 Android Marshmallow 或更高版本。要打开它，请按照下列步骤操作:

1.  拉下手机上的通知按钮，然后打开快速设置菜单。
2.  点击并按住屏幕顶部的齿轮图标大约五秒钟。然后，您将看到一条消息，说明系统 UI Tuner 已添加到设置中。
3.  打开手机上的设置应用，滚动到底部。在那里你会发现系统用户界面调谐器。

如果你使用的是 Android 的皮肤版本，这个过程可能会有所不同。一旦菜单被激活，这里有一些你可以调整的设置。

#### **在状态栏中显示您的电池电量百分比**

对于我们这些经常对电池进行微观管理的人来说，这是天赐之物。此设置将在状态栏的电池图标上方显示您的电池百分比。您可以选择仅在充电时显示百分比，也可以一直显示。

#### 从状态栏中隐藏不必要的图标

你的状态栏塞满了图标，但你可能不需要看到所有的图标。比如我不需要一直看蓝牙是否开启。如果我需要检查，我可以向下滚动通知阴影。此菜单允许您从状态栏中永久隐藏蓝牙、Wi-Fi、蜂窝数据、飞行模式或勿扰模式等图标。

#### **添加、删除或重新排列您的快速设置图标**

Android 中的快速设置阴影试图智能地猜测你需要经常调整的设置，并自动添加它们。不幸的是，它有时会猜错。在 Android Marshmallow 中，你可以进入系统 UI 调谐器菜单，手动添加快捷方式或重新排列现有的快捷方式。注意:在 Android N 预览版中，这个功能已经被嵌入到常规操作系统中，所以这个隐藏的菜单选项是不必要的。

#### **启用夜间模式调节色温并切换到深色主题**

谷歌在整个夜间模式的事情上有点优柔寡断，但在 Android NT3 中 [又回来了。如果你正在运行](http://lifehacker.com/the-coolest-features-of-android-n-google-didnt-announce-1764114846) [开发者预览版](http://lifehacker.com/how-to-install-the-android-n-developer-preview-on-your-1763813851) ，你可以使用系统 UI 调谐器菜单来激活夜间模式。当激活时，您的手机将自动检测一天中的时间，并相应地调整您的显示器的色温。它还会在应用程序中使用深色主题，比如设置。

#### 调节音量时显示请勿打扰模式

免打扰模式在你需要手机关机一分钟的时候超级方便。大多数手机会激活它，如果你按下音量按钮时，你的手机已经在沉默，但系统用户界面调谐器让你添加一个更明显的切换时，你调整音量。如果您不想意外激活正常音量按钮快捷方式，您也可以禁用它。像新的夜间模式功能一样，这一调整是 Android N 预览版独有的。

* * *

随着谷歌添加或删除功能，系统 UI 调谐器菜单会随着时间的推移而变化，所以每当你获得 Android 的更新时，都值得再次查看。
