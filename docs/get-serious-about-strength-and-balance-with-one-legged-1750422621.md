# 认真对待单腿深蹲的力量和平衡

> 原文：<https://lifehacker.com/get-serious-about-strength-and-balance-with-one-legged-1750422621>

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-Ecxt4TtVlFQ&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-Ecxt4TtVlFQ&start=0) 

手枪式深蹲是一种单腿运动，它提供常规深蹲 的所有 [肌肉锻炼益处，同时挑战你的平衡。这是超级艰难的，但伴随着吹牛的权利。](http://lifehacker.com/if-you-only-do-one-type-of-weight-training-do-squats-5921984)



如果你连一个手枪式深蹲都做不到，你并不孤单。幸运的是，一针肾上腺素有一个全面的指导来建立你需要的肌肉和技能来尝试这个动作。首先是有规律的、无重量的蹲起；然后保持半蹲姿势，重心转移到一条腿上。

从那里开始，你可以做部分重复动作(坐在椅子上),然后进行下蹲，在那里你可以使用门框或高架表面来辅助。点击下面的链接，查看完整的进度，以及将其付诸实践的训练计划。
