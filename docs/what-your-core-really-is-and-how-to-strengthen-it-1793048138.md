# 你的核心是什么，如何加强它

> 原文：<https://lifehacker.com/what-your-core-really-is-and-how-to-strengthen-it-1793048138>

仰卧起坐和木板不是一切。如果你想强化你的核心，你需要想得更大。你的核心包括躯干的所有肌肉，从肩膀到臀部。它的作用不仅仅是帮助你坐起来。



### 为什么薄弱的核心是一种负担

这个周末我上了一堂的 TRX 课。大多数动作都需要你保持核心稳定，同时用手抓住 [悬浮训练器](http://vitals.lifehacker.com/amp-up-your-bodyweight-workout-with-these-suspension-ex-1751397400) 的带子，做某种推或拉的动作。这是一项令人惊叹的全身锻炼。但是在我那排的最后有一个人是第一次到那里，他要么不能保持他的核心稳定，要么不知道如何保持。当他应该做俯卧撑时，他的背会下垂，或者他会把争吵变成嘎吱声。结果，他没有从练习中得到最大的收获。

强壮的核心对跑步者和游泳者来说也很重要。你的身体随着每一个脚步或每一次划水而扭曲，所以你需要保持躯干稳定和强壮。事实上，由于你的核心与你的手臂和腿相连，控制核心有助于拉你的手臂通过水或稳定你的腿在地面上。没有一个强大的核心，你就不能游泳或跑步。

当我玩轮滑的时候，一个强壮的核心对于打出强有力的一击是必不可少的。用你的肩膀推某人，你的身体需要从地板到撞击点都是坚实的——所以包括你的腿和躯干。如果你去做臀部检查，你可能会认为你不必担心臀部以上的任何东西。但是如果没有强壮的核心肌肉群，你的上半身会在撞击时左右摆动，让你失去平衡。

即使你不是出于爱好而检查身体，一个强壮的核心对日常生活也很重要。有了它，你可以移动沙发和铲雪，而不会伤害你的背部。你可以轻而易举地推开最重的门。你可以把一个愤怒的孩子带出博物馆礼品店，而他们却像一条巨大的垂死的鱼一样扑腾在你的怀里。(问我怎么知道的。)

另一方面，没有一个强大的核心，你在健身房的时间将不会得到回报。你会做蹩脚的俯卧撑。你会做蹩脚的硬拉。你会有不好的姿势。随着年龄的增长，你起床或从厕所起来会更加困难。你会 [更容易摔倒](https://www.ncbi.nlm.nih.gov/pubmed/23568373) 。你现在越强大，你的一生就会越好。

### 你的核心到底是什么

我们很多人把我们的“核心”和腹部前部的六块肌肉混为一谈，称为*腹直肌*。是的，当你做仰卧起坐时，你能感觉到肌肉收紧。但是这如何训练我们适应现实生活呢？向前弯曲并不需要很大的力量。但是，当腹直肌与所有其他核心肌肉合作时，它有助于在各种姿势下保持身体稳定。

这就是为什么厚木板比仰卧起坐更好:如果做得好，厚木板可以同时锻炼你所有的核心肌肉(另外还可以锻炼你的胳膊和腿)。每块肌肉都在收缩，为另一块肌肉提供阻力。最终结果是一个坚如磐石的核心。这与我们上面的许多例子中的情况相同，你需要保持稳定来抵抗某种外部压力，无论是臀部检查还是挥舞的孩子或沉重的门。

但是有更多的方法来锻炼你的核心，而不仅仅是简单的原地踏步。让我们来看看我们真正要针对的肌肉是什么:

*   **腹直肌** [使你的腰部](https://en.wikipedia.org/wiki/Rectus_abdominis_muscle) 向前弯曲。
*   竖脊肌或背伸肌沿着你的脊柱纵向延伸。他们 [把你的腰向后弯](https://en.wikipedia.org/wiki/Erector_spinae_muscles) 。
*   **内外斜肌**是环绕腰部的两层肌肉。他们的 [肌肉纤维是对角的](http://antranik.org/muscles-of-the-abdominal-wall/) ，所以参与扭转运动。
*   **横腹肌**是帮助你收腹的肌肉。把它想象成一条天然的举重带:它 [有助于稳定你的脊椎](https://en.wikipedia.org/wiki/Transverse_abdominal_muscle) 。
*   **多功能**是一系列肌肉，它们 [将你脊椎的骨头相互连接](https://en.wikipedia.org/wiki/Multifidus_muscle) 。
*   **腰方肌**住在你的腹部深处，T3】连接你的骨盆和脊椎 T5。

一些教练会认为你的“核心”也包括附近的肌肉，比如你的臀部和上背部。例如，你的 [髂腰肌](https://en.wikipedia.org/wiki/Iliopsoas) 将你的脊柱连接到你的腿，并帮助你将大腿带到腹部。例如，当你做仰卧起坐时，你会用到它们。

### 如何全方位锻炼你的核心

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-XZV9IwluPjw&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-XZV9IwluPjw&start=0) 

有这么多肌肉参与，你可以看到为什么坚持仰卧起坐不会削减它。好消息是，如果你做得好，几乎任何全身运动都会在某种程度上锻炼你的核心。那包括 [俯卧撑](http://lifehacker.com/how-to-do-the-perfect-push-up-5992941)[引体向上](http://vitals.lifehacker.com/work-your-way-to-a-pull-up-with-these-intermediary-exe-1685251314)[用大锤敲打拖拉机轮胎](http://vitals.lifehacker.com/what-are-those-giant-tires-in-gyms-1785814265) 。以下是如何识别一些最适合你的核心动作:

*   你必须保持你的核心稳定。这包括木板，以及形态良好的俯卧撑(你的身体一直在一块完美的木板上)。它还包括倒置的行，就像上面的视频一样。
*   你用一只胳膊或一条腿，这样你的核心就能适应不平衡的力量。 [死狗](http://vitals.lifehacker.com/deadbugs-are-an-underrated-core-exercise-you-should-be-1785370634) 和 [鸟狗](http://vitals.lifehacker.com/the-bird-dog-is-an-abdominal-exercise-that-helps-reduce-1789963476) 就是完美的例子。
*   **你将重物放在身体的一侧**，以增强单臂或单腿挑战。例如，如果你一次用一只手臂做哑铃划船或举重，你的核心必须参与进来以防止你摔倒。举个更极端的例子，看看 [的土耳其装扮](http://vitals.lifehacker.com/you-should-really-try-working-out-with-kettlebells-1790152387) ，或者做一边倒的 [农民的散步](http://vitals.lifehacker.com/the-farmer-s-walk-is-a-simple-powerful-total-body-exer-1788952098) 。
*   你扭动你的身体来完成这个动作。这里的一个经典例子是一个 [木工](https://www.youtube.com/watch?v=tH8ZeVgupYY) ，其中一个哑铃或缆索机器模仿了一把斧头的重量。你把它斜着拉过你的身体，就像你在劈柴一样。当然，你可以砍一些真正的木头来做类似的锻炼。或者尝试一个 [俄罗斯转体](https://www.youtube.com/watch?v=GefmYOxdY9c) (有或者没有健身球)来代替仰卧起坐。
*   **你用的是不稳定的平台**。不要将前臂放在地板上，而是放在健身球、健身球或上。

有了这些动作，你就不必花一天或一部分时间进行核心训练。当你做大的功能性运动时，你的核心和身体的其他部分一起工作。如果你更喜欢隔离特定身体部位的锻炼，那很好——但要确保花时间以各种方式锻炼你的核心。
