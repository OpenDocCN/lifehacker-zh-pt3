# 如果你真的想玩游戏，买新的任天堂 2DS XL 吧

> 原文：<https://gizmodo.com/if-you-actually-want-to-play-games-get-the-new-nintendo-1797050089>

[【任天堂 Switch】](http://gizmodo.com/nintendo-switch-review-plays-zelda-great-doesnt-do-mu-1792852009)的存在，是一个神奇的游戏系统，必要时你可以在酒吧、汽车或火车上玩。手机也存在，手机上的游戏比以往任何时候都好。所以你凭什么拥有其他东西？因为游戏。Switch 的库仍然很小，智能手机仍然缺乏那些可以让你沉迷几天的游戏。因此，如果你想要一个可以去任何地方并玩一些有史以来最好的游戏的移动系统，你需要任天堂 3DS 系列的东西，尽管它已经有 7 年的历史，但没有任何迹象表明它将很快结束生命。



在过去的七年里，已经有六款 3DS 游戏机，每一款都有一些缺陷，使其无法成为技术和价格的完美结合——但是 [新任天堂 2DS XL](https://www.nintendo.com/2ds/features/compare/) 改变了这一点。

当我写这篇评论时，你会注意到我在谈论新的任天堂 2DS XL 时用了大写的“新”。那不是我的做作。这是任天堂超级混乱的命名方案的一部分。新的 2DS、和T3】2015 年推出的新 3DS，在技术上比原来的 3DS 系统更上一层楼。它不仅仅是一个新的机箱— [这些系统可以玩原来的 2DS 和 3DS 不能玩的游戏](http://www.neogaf.com/forum/showthread.php?t=1177455) ，像 *Xenosaga Chronicles 3D* ，或者包括*大金刚之国**街霸 Alpha 2* 和 *F-Zero* 在内的各种 SNES 游戏。虽然没有那么多独家游戏，但有足够多的好游戏，如果你已经拥有一个旧的 3DS 系统，你可能会希望你有一个新的。

新的 2DS 和新的 3DS 一样，也有更快的处理器，一个用于控制台存储的 microSD 插槽，用于所有用户的 NFC 功能，设备肩部的两个新按钮，以及一个 C-Stick 模拟操纵杆。从各方面来看，这都是对你可能拥有的任何非新的 2DS、3DS 或 3DS XL 的升级。

新的任天堂 2DS XL 不仅仅是丑陋的旧 2DS [的更新，它看起来更像是廉价的山寨](http://kotaku.com/the-internet-reacts-to-the-2ds-reveal-1217271879) 而不是来自创造移动游戏的公司的系统。随着新设备的推出，任天堂抛弃了 2DS 丑陋的平板设计，模仿其他 3DS 的折叠设计。售价 150 美元，比原来的 2DS 贵了 70 美元，比新的任天堂 3DS XL 便宜了整整 60 美元(3DS XL 不包括充电器，所以我把这个价格加到了总价中)。这是现在的中档移动控制台，除非你真的，*真的*喜欢 3D，否则它应该是你买的那个。

除了比新的 3DS XL 便宜之外，它也轻了很多。它重 9.2 盎司——与最初的 2DS 相同。3DS XL 重 11.6 盎司。你可能不认为这是一个主要的区别，但当我把 3DS 扔进我的钱包时，我肯定会注意到这一点，我经常忘记我有 2d 在里面。

它也比 3DS 系列中目前可用的其他两个控制台更小，折叠起来。平板状 2DS 是 5 英寸乘 5.66 英寸。新款 2DS XL 可折叠至 3.4 英寸乘 6.3 英寸。它不仅比它的前辈更易于携带，还比 3DS XL 略小，3DS XL 的折叠尺寸为 3.7 英寸乘以 6.3 英寸。

与最初的 2d 不同，这个系统不会因为人们讨厌 3D 或者想省钱而惩罚他们。这是目前最好的 3DS 系统，只是尺寸小了一点，而且去掉了一个功能。它甚至在新的 3DS XL 上有所改进！这个系统有两个大问题:游戏盒总是可见的，而且总是很丑，你需要一把螺丝刀和一点耐心才能更换 microSD 卡。新的 2DS XL 通过将两个插槽隐藏在设备底部的一个小门后解决了这两个问题。

但是 2DS XL 也不是没有一些小问题。扬声器已经从设备的上半部分移到了底部的小槽中，你的手掌经常会盖住这些小槽。所以当你的手掌盖住扬声器时，声音会变得低沉，当你的手掌打开时，扬声器的声音会变得更小。滑块和铰链感觉也更便宜，铰链有一点“给”它(如左图所示)。新的 3DS XL 感觉非常豪华。这是雷克萨斯，这家伙的丰田凯美瑞很结实。

作为一个经常开凯美瑞的人，我完全可以接受！任天堂 2DS XL 对我来说绰绰有余。随着系统上梦幻般的 DS、3DS 和新 3DS 游戏的巨大选择，以及一个月后全新的 Metroid 游戏，我不认为这个小工具会很快离开我的包。如果你已经拥有一台新的 3DS XL，但从未使用过 3D，那么这可能值得一看，如果你正在从最初的 3DS 升级或只是尝试合法的移动游戏，那么这是 7 月 28 日发售时可以购买的主机。
