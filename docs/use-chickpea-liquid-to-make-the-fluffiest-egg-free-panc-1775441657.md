# 使用鹰嘴豆液体制作有史以来第一个无蛋煎饼

> 原文：<https://lifehacker.com/use-chickpea-liquid-to-make-the-fluffiest-egg-free-panc-1775441657>

不吃鸡蛋的人，你们的早餐祈祷被回应了。Serious Eats 开发了一种不含鸡蛋(实际上，完全素食)的厚而松软的薄煎饼配方，秘诀就在一罐鹰嘴豆中。

Watch

正如 J. Kenji Lopez-Alt 在 [食品实验室](http://www.seriouseats.com/columns/the-food-lab/index.html) 解释的那样，鹰嘴豆液体，也称为“aquafaba”，有能力蓬松和稳定各种无蛋食品，从蛋白酥饼到松饼，但这不是简单地倒入一些鹰嘴豆液体就可以结束的事情。在典型的食物实验室风格中，Kenji 摆弄着他的基本煎饼配方，直到他找到了一个完全纯素食的配方，制作出蓬松、温暖、美味的煎饼 。

点击下面的链接阅读他的整个过程(和食谱)，但在制作这些和其他无蛋的奇迹时，有几个关键的事情要记住:

*   添加一点额外的糖会增加鹰嘴豆液体的粘度，给它更多的结构持久力。
*   不要把 aquafaba 搅拌到你的其他湿配料中，而是把它搅拌成蓬松的硬峰，然后*轻轻地*拌入你的面糊中，小心不要破坏泡沫的通风质量。
*   因为 aquafaba 缺乏真正鸡蛋的蛋白质凝固特性，所以这种面糊需要更多的时间来凝固。为了确保它们有足够的时间变干和烹饪，使用比“普通”煎饼更低的温度。

请记住，上面的建议可以(也应该)应用于你用 aquafaba 做的任何烘焙，所以在你的下一个无蛋蛋糕、饼干或松饼上试试吧。
