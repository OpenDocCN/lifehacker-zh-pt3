# 接下来我们应该做什么？

> 原文：<https://lifehacker.com/what-should-we-sous-vide-next-1824031758>

各位网友大家好，欢迎来到全新的 [*选题会，会吗？*](http://lifehacker.com/tag/will-it-sous-vide) ，列在那里我用我的沉浸循环器做任何你想让我做的事。



上周的鸡蛋啃咬 是一种天鹅绒般的享受，但我想我已经准备好把我的注意力转向更有肉的东西了。复活节离我们并不太遥远，所以我在想，下面这些主菜中的一个会很完美:

*   火腿:我知道我们只是在加热它，因为火腿大多是预先烹饪好的，但是真空烹饪实际上是在不干燥的情况下重新加热火腿的最佳方式。另外，我打赌我们真的可以给猪肉注入很多很多的味道。
*   **羊肉:**要么是排骨，要么是一整条腿；我觉得两个都不错。
*   **上等肋排:**在我们的烹饪热水浴缸中长时间、低而慢的烹饪，最有可能产生一种烹饪完美、完全嫩滑的上等肋排，具有真正一流的 jus。

当然，不要觉得被我的想法限制了；如果你有一个绝妙的想法想让我测试一下，请在下面的评论中留下，别忘了给你最喜欢的打星号。
