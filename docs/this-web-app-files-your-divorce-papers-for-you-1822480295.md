# 这个网络应用程序会帮你整理离婚文件

> 原文：<https://lifehacker.com/this-web-app-files-your-divorce-papers-for-you-1822480295>

无论多么友好，离婚都是艰难的——也是昂贵的。决定离婚是一个巨大的决定，随之而来的是另一个大问题:你怎么做？



离婚通常会涉及律师、无数次谈话和大量你不想花的现金。一款名为“ [这太简单了](https://www.itsovereasy.com/) ”的新网络应用试图通过将整个体验放到网上来让这变得更简单。诚然，这种解决方案只适用于最友好的离婚，但是，如果你打算离婚，但仍与你的配偶保持良好的关系，那么这也是一种廉价而简单的方式。

这款应用是由名人离婚律师劳拉·瓦瑟开发的。她曾为布兰妮和斯蒂维·旺德等人办理过离婚。当不是名人的朋友向她寻求他们或他们朋友离婚的帮助时，她意识到她无处可发，于是这款应用诞生了。

从本质上来说，是填写和提交离婚文件的一种简单方式。该应用程序会问你一些问题，你的回答会被填入办公室的文件中，你可以在法院存档(或者让应用程序为你存档)。

只要你和你的配偶已经在这方面达成某种协议，这款应用程序就可以处理儿童监护和抚养以及财产分割的文书工作。

同样，这不是那些需要解决分歧的人使用的东西，而是更适合那些已经经历了所有这些，只是需要帮助使其合法的人。

Wasser 解释表格的视频包含在复杂的部分中，每个包都包含一些与律师面对面的真实时间，可以帮助你明确细节或回答任何问题。

基本计划运行 750 美元，而顶级的保费计划需要 2500 美元以上的处理和州费用。

是的，在网上离婚并不是最浪漫的选择，但如果你更愿意在自己家里处理一切，这可能是一个体面的解决方案。
