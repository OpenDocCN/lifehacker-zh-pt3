# 为什么澄清黄油如此美味(以及如何自己制作)

> 原文：<https://thetakeout.com/why-clarified-butter-is-so-tasty-and-how-to-make-your-1821887554>

酥油和澄清黄油是一种秘密成分。它们是黄油，但更好。它们是更丰富、更浓烈、更浓缩的黄油。它们就像黄油的平方。



酥油和澄清黄油非常相似，但酥油的制作方式通常会产生更丰富的味道。两者都是通过烹饪黄油，去除牛奶固体和水，只留下纯黄油制成的。酥油更进了一步，轻煮纯黄油以获得更深的坚果风味。

酥油是印度烹饪和烘焙的主食，也出现在一些中东食物中(还有更有趣的黄油同志，发酵的 [smen](https://www.npr.org/sections/thesalt/2014/10/09/353510171/smen-is-moroccos-funky-fermented-butter-that-lasts-for-years) )。如果你曾经在印度餐馆吃过 [咖喱或炖菜](https://thetakeout.com/a-cheat-sheet-to-indian-restaurant-menus-1798256742)——比如 biryani——很有可能是酥油。它被刷在烙饼和查帕蒂面包上。

不过，澄清黄油和酥油不应该仅限于家乡的印度炒面。它们比典型的黄油更美味，可以而且应该用在任何使用黄油的地方:在咖喱中调和香料和洋葱，是的，也可以用在黄油酱中，煎鸡蛋，炒蔬菜，做烤奶酪三明治。(说到奶酪:在澄清的黄油中煎炸一些北欧面包奶酪，以达到最高的享乐主义。)

通过去除牛奶固体，酥油/澄清黄油比普通黄油有更高的烟点，这意味着你可以用它在普通黄油燃烧的更高温度下烹饪东西。如果你正在做一个需要椰子油的食谱，你可以用澄清黄油来代替。牛排在煎锅里用澄清的黄油烹制也很美味。

你可以在印度的杂货店里找到酥油，在美国的超市里也越来越多地找到酥油和澄清黄油(Trader Joe's 和 Whole Foods 都有卖)。)但是在家里自己制作要便宜得多，而且相当容易。

## 如何制作澄清黄油

1.  将未加盐的黄油放入厚底锅中，用小火加热。让它静置，不要搅拌。当乳脂产生泡沫时，它应该液化并慢慢起泡。这一步可能需要一段时间——预计要花大约半个小时慢慢融化黄油，让泡沫升到顶部。你还会看到牛奶脂肪沉入底部。
2.  用一把大金属勺，小心地去除漂浮在罐子顶部的泡沫。
3.  一旦所有的泡沫都被去除，把锅从火上拿开。
4.  大约五分钟后，当黄油仍然是液体并且温热但稍微冷却时，小心地将锅内金黄色的黄油倒出。锅底会有更多浅色的乳脂；不要倒出来。
5.  可选:使用粗棉布(或咖啡过滤器)来去除混入的牛奶固体。用两层粗棉布衬在滤网上，将黄油倒入。(我从来不这样做，但如果你是一个纯粹主义者，想要你的黄油 100%澄清，这是一条出路。)

## 酥油怎么做

1.  像澄清黄油一样，在一个坚固的锅里用小火融化无盐黄油。
2.  黄油液化后，把火调到中火。
3.  撇去顶部的泡沫，如在澄清的黄油中。
4.  把温度调高一点，到中高火。会形成更多的泡沫，锅底的牛奶固体会变成褐色。
5.  撇去顶部的二次泡沫，将锅从火上移开。
6.  剩下的和澄清黄油一样:当黄油还是液态温热但稍凉时，小心地把锅里金黄色的黄油倒掉。锅底会有更多棕色的乳脂；不要倒出来。
7.  用粗棉布(或咖啡过滤器)过滤掉混入的牛奶固体。用两层粗棉布衬在滤网上，将黄油倒入。

## 有用的提示

1.  你可以在烘焙中使用酥油和澄清的黄油，但是在需要普通黄油的食谱中用它来代替是有风险的。大量专门使用酥油的烘焙食谱可以在网上找到。
2.  在澄清过程中，你会损失大约四分之一杯，所以请记住，从一杯黄油开始，会产生大约四分之三杯澄清黄油。
3.  使用无盐黄油。与黄油风味一样，澄清黄油中的盐含量也会更高。
4.  带有锋利边缘的大金属勺比木勺更容易撇去顶部的泡沫。厚而圆的边缘更具挑战性。
5.  使用薄底锅会导致黄油烧焦。很明显你不想发生这种事。如果你注意到底部变褐，就把火关小。
6.  澄清黄油和酥油比普通黄油保存时间更长，因为你基本上去除了变坏的部分。它们不需要像普通黄油一样放在冰箱里，只要放在密封的容器里就可以了。值得一提的是，和我一起工作的厨师，他教我如何制作澄清黄油，确实把他的容器在冰箱里放了一夜。它变得非常坚固；为了把它加热回液态，他把它放在温暖的炉灶上，放了一整天，随时可以用来煎炒食物。
