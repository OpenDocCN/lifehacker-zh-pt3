# 尤利西斯是我一直想要的纯文本、Evernote 风格的写作应用

> 原文：<https://lifehacker.com/ulysses-is-the-plain-text-evernote-style-writing-app-i-1776133398>

长期以来，我一直是从笔记到一般写作的所有内容 的纯文本粉丝，但直到最近，我才找到了我的圣杯:一个可以与多种设备同步并处理多种输出类型的应用程序。T5【尤里西斯】T6。这是我一直想要的 Evernote 的纯文本版本。

我想先提一下尤利西斯的几个问题:首先，它只适用于苹果设备(Mac/iOS)。第二，它很贵: [Mac 应用](https://itunes.apple.com/us/app/ulysses/id623795237?mt=12&ign-mpt=uo%3D4)44.99 美元(尽管你可以免费试用一个 [试用版](http://www.ulyssesapp.com/) )，而 [iPhone/iPad 应用](https://itunes.apple.com/us/app/ulysses-mobile/id950335311?mt=8&ign-mpt=uo%3D4)24.99 美元。iOS 和 Mac 应用程序具有相同的功能集，因此您可以在两者之间轻松切换。

考虑到像 Evernote 这样的应用是免费的，价格标签肯定是*要价很高，但它也是同类应用中唯一让我满意的。 [Scrivener 接近](https://lifehacker.com/how-scrivener-helped-me-organize-all-my-writing-1599446028) ，但其丑陋的设计和缺乏移动选项使其无法成为我唯一的工具。尤利西斯没有那个问题，这让它值这个价。既然尤利西斯已经融入了我的工作流程，我想我应该分享一些使用它的技巧。*

### 尤利西斯不仅仅是一个文本编辑器，它还是一个写作环境

尤利西斯不完全是一个纯文本编辑器。很长一段时间，《尤利西斯》的定位是“纯文本增强”，虽然我讨厌屈服于广告标语，但这确实很好地描述了《尤利西斯》。这是一个更智能的纯文本版本，可以只作为纯文本使用，或者如果你愿意，可以处理更多的内容。不仅如此，《尤利西斯》是一个完整的写作环境。要想很好地使用尤利西斯，你需要把所有的东西，从你的笔记到你的书稿，都放到尤利西斯里。

如果你使用过 Scrivener 这样的应用程序，你就会知道尤利西斯会带来什么。尤利西斯中的主要文本编辑器就是这样，一个文本编辑器。你可以用一种很好的、不受干扰的模式打字。如果你愿意，可以添加格式，就这样。我喜欢的一个小功能是，当你想打字时，尤利西斯会自动隐藏界面，这让你可以非常轻松地开始工作。

至于写作本身，你可以写纯文本，一个特殊版本的 Markdown， [经典 Markdown](https://lifehacker.com/what-is-markdown-and-why-is-it-better-for-my-to-do-lis-5943320) ，HTML 标记引擎 [Textile](https://en.wikipedia.org/wiki/Textile_(markup_language)) ，或者你可以创建你自己的标记语言(这对于你使用出版商的特定标记风格的书籍来说很方便)。您可以在任何需要的地方添加注释，创建群组来整理所有内容，甚至添加图像。尤利西斯基本上就是我想象中 Evernote 的样子，如果它是专门为作家制作的话。

就像我开始说的，尤利西斯不仅仅是一个文本编辑器，它是一个写作环境。这意味着大量的组织工具。

### 为每个项目分组，不要害怕创建大量的工作表

《尤利西斯》中的组织特征是它最大的优势。它们是《尤利西斯》与极简写作应用程序的区别，比如《作家 T2》中的。虽然组织功能很棒，但要习惯它的工作方式可能需要一点精神上的努力，因为尤利西斯的系统与其他应用程序有点不同。

Ulysses 是一个独立的系统，在这个系统中你可以创建独立的小项目。代替笔记本，尤利西斯将项目分类成组。每个组可以有无限数量的子组。每个组或子组可以有无限数量的图纸。工作表只是文本文档的一个花哨的词。如果这种语言让你感到困惑，不要担心，我也花了一点时间来适应它，但是一旦我开始实际使用组和子组，它让我的生活变得容易多了。

假设我正在写一本书。首先，我在侧边栏中创建一个组，给它一个名称，然后选择一个图标来代表它。这是整个项目的总体组。接下来，我将为第一章创建一个子组。当这些都设置好后，我将创建一个新的工作表。根据手头的项目，我通常会为大纲创建一张纸，为写作本身创建另一张纸。虽然 Ulysses 有一个内置的注释功能，你可以在任何表格上添加注释、图像和 URL，但我通常也会添加一个工作引用表，以跟踪我引用的任何内容。最后，我在这个分组中添加了一个“想法转储”表。这是我在旅途中或打字时突然想到的想法的地方。然后，我开始打字。当我完成时，我选择我想要导出的页面，尤利西斯将它们拼接成一个大文档。

如果不明显，我喜欢在写作前整理东西，尤利西斯让我完全控制这样做。我意识到不是每个人都这样，如果是这样的话，尤利西斯对你来说可能是多余的。然而，如果你喜欢在一个地方进行分类、概述、追踪资源和编辑笔记，Ulysses 做得很好。

### 搜索是强大的，使用它

除了有一个坚实的手动组织写作的系统，Ulysses 还有一个非常强大的小搜索程序。轻按 Command+O 并键入您的搜索。结果将包括带有该搜索词的任何文本。找到您想要的表单，按下 Return 键，Ulysses 会直接带您进入该文档的文本编辑器。这听起来并不多，但它让你可以轻松地在不同的工作表之间来回移动，而无需触摸鼠标，这在你只想写作时非常有用。

### 使用标签和智能过滤器来自动整理您的写作

虽然尤利西斯中的搜索可以以令人难以置信的速度调出页面，但你可能仍然希望使用标签来组织你的写作。这样做有两个原因:第一，标记一个工作表有明显的好处，使它更容易在搜索中找到。我通常这样做是为了收集笔记，或者如果我正在处理一个可能与更广泛的目标相联系的想法，添加一些主题标签。例如，如果我在写*权力的游戏*，我会在每一页上标注每章的主要人物，这样我就可以很容易地找到他们。

Ulysses 还支持自动更新滤镜组，其工作方式与 Finder 中的 [智能文件夹完全相同。创建新过滤器时，您可以选择标签、文本本身的关键字或修改日期。然后，Ulysses 会自动用任何匹配的内容填充该组。这使得你可以创建一个群组来收集一个特定的关键字，这样就很容易再次找到。在*的《权力的游戏*》中，我可能会添加一个“死亡”关键词，然后创建一个群组来跟踪包含死亡的每一章，这样我就可以很容易地再次找到它。](https://support.apple.com/kb/PH19078?locale=en_US&viewlocale=en_GB)

### 不要害怕根据你的喜好定制尤利西斯

如果你喜欢修补，那么你会感激你能改变尤利西斯多少。您可以改变书写环境的外观，创建自己的文档导出选项，甚至自定义标记语言。

尤利西斯用主题来改变编辑器。你可以从尤利西斯网站 下载一些 [受欢迎的主题，如果没有你喜欢的，你也可以自己创作。使用主题，您可以更改编辑器的颜色、外观和感觉。我是](http://styles.ulyssesapp.com/themes/?page=1&sort=popular)[80 年代新浪潮主题](http://styles.ulyssesapp.com/bundle/New+Wave+80s/5407d108ce6703c6350c1977) 的忠实粉丝，因为它在我写作时给一切添加了一种可笑的马戏团外观，这让我每隔几分钟就咯咯笑起来。如果你正在写一些同人小说，上图中的 [早期电脑主题](http://styles.ulyssesapp.com/bundle/Early+computers/536f7ab2bc6a24f607dae106) 是你想要用来进入正确情绪的。不管您的偏好如何，美妙之处在于主题对导出文档的方式没有影响，因此您可以选择您最喜欢的方式，而不用担心以后的格式问题。

Ulysses 使用样式将您的作品导出为其他格式。完成写作后，您可以将其导出为 PDF、ePUB、富文本或 HTML 格式。当您这样做时，您的文档会按照您选择的样式进行格式化。像主题一样，你可以在尤利西斯网站 上查看许多用户创建的 [风格。如果你使用多种媒介或者你计划以任何方式自助出版，样式是非常有用的。基本上，您可以用简单形式的纯文本编写，然后用更好的方式导出纯文本。例如，如果您正在编写一个电影剧本，您可以使用](http://styles.ulyssesapp.com/?ref=u3_stylePrefs)[Filmscript PDF style](http://styles.ulyssesapp.com/bundle/Filmscript/538502dc081b0faa4d320f29)将您的文本导出为工作室友好的剧本格式，而不必在编写时过多担心格式问题。

如果可下载的主题或样式选项都不符合您的需要，您可以在首选项面板中创建自己的主题或样式。这需要一些工作来做好，所以我建议先浏览一下可下载的选项。如果你决定自己制作，你会在尤利西斯网站 上找到关于制作的完整指南。

最后，Ulysses 支持几个值得一提的写作专用模式。从菜单栏中，您可以切换到黑暗模式以便在晚上更容易地书写，切换到全屏模式以获得不受干扰的体验，在文本编辑器中更改字体，或者启用打字机滚动选项以在您键入时向上推送文本。就我个人而言，我最常用黑暗模式和全屏模式，但如果我决定使用它们，知道还有其他选项总是很好的。

我想喜欢尤利西斯已经很久了，但总是被缺乏移动选项所阻碍。随着 [、iPhone 和 iPad 应用](https://itunes.apple.com/us/app/ulysses-mobile/id950335311?mt=8) 的发布，它终于适应了自己，做了我需要的一切。我可以在电脑上输入又大又长的文本，但仍然可以在手机上即时阅读。我可以把所有东西都放进去，而不用担心会丢失任何东西。它还做了每个写作应用程序都需要做的一件重要事情:当我只想写作时，它会走开。
