# 如何有效地头脑风暴

> 原文:[https://life hacker . com/the-best-brainstorm-is-divergent-1821356300](https://lifehacker.com/the-best-brainstorming-is-divergent-1821356300)

你有问题。你召集了一群聪明、有创造力的人，然后说，*让我们头脑风暴一下*。你们一起思考一堆想法，不断琢磨，直到找到答案:解决方案。

Watch

好吧，如果你是这么做的，那你就有另一个问题了:你头脑风暴的方法不对。

利用头脑风暴来集中和提炼想法是*收敛*——利用你们集体的创造力来锁定正确的答案。相反，尝试*发散*头脑风暴，目标是提出尽可能多的不同想法，偏离看似好的想法，探索可能行得通的遥远边缘。

以下是来自 [石英](https://work.qz.com/1155586/weve-been-structuring-brainstorm-sessions-all-wrong/) 的关于如何为发散性头脑风暴搭建舞台的一些指导方针:

**扩大你的圈子。**邀请人们参加头脑风暴，不要局限于你习惯共事的团队，也不要局限于习惯思考这个问题的团队。

**让团队多样化。你想要尽可能多的视角，所以要寻找年龄、背景、性别、经历和个性的多样性。**

**阐明问题。**确保每个人都明白这个问题。此外，尽可能多的从不同的角度看待它——挑战你对其原因和影响的假设。

追求数量胜于质量。“没有坏主意”是头脑风暴的口头禅，但是如果没有质量的替代品，你可能会失去努力的目标。这就是数量的来源。产生尽可能多的想法——你以后会担心什么是好的。

不要试图同意。一旦一个好主意出现，你可能会感到兴奋或轻松。省省吧。保存这个想法——你不会失去它——并不断产生更多的想法。如果你过早达成共识，你可能会错过那些真正古怪的、开创性的想法，如果你再多花一点时间，这些想法就会实现。

拥抱嬉戏。你真正追求的是创造力，你知道谁擅长无拘无束的创造力吗？孩子们。无论你能在多大程度上放松你众所周知的职业关系，去做吧。回到你小时候那个奇怪的，自由的大脑。这就是疯狂的——优秀的——想法所在。
