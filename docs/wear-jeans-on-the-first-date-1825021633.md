# 第一次约会穿牛仔裤

> 原文：<https://lifehacker.com/wear-jeans-on-the-first-date-1825021633>

在第一次约会中脱颖而出的压力很大，这种压力不可避免地会影响到你的着装选择。但是在你第一次去见某人之前，没有必要换一套愚蠢的服装。舒适、休闲的日常用品，比如一条好的牛仔裤，就是你所需要的。

Watch

假设牛仔裤没有破洞或被撕成碎片，在大多数社交场合，它总是一个安全的选择。每个人都有一双，没有人真正去评判穿这双鞋的人。在 Mulberry's Cleaners 进行的一项调查中，绝大多数受访者认为牛仔裤是头号约会对象的最佳选择。对于男生来说，大多数男性和女性都认为牛仔裤是第一次约会的最佳选择。对于女孩来说，大约一半的男性和女性认为牛仔裤也很时髦。基本上，牛仔裤是意料之中的，这是一件好事。 [重点应该是对方](https://lifehacker.com/the-best-ways-to-break-the-ice-and-get-to-know-someone-1736039897) ，而不是你的裤子。

此外,“第一次约会”无论如何也不是真正的约会——更像是一次面试。大多数“第一次约会”包括在咖啡店、酒吧或咖啡馆见面，以感受彼此的感受。你试图看看是否有化学反应，并确保对方不是一个想吃你的小牛的怪人；没有唤起超级浪漫的气氛。穿暴露的衣服或者穿西装打领带会给他们错误的印象， [，甚至可能吓跑](https://lifehacker.com/the-worst-first-date-mistakes-and-how-to-bounce-back-1768885084) 。这是一个随意的环境，所以保持轻松是很好的。如果你担心，可以稍微打扮一下，但不要太过火。

现在，当我说“牛仔裤”时，我不一定指真正的牛仔裤。我不喜欢大牛仔布。也许你讨厌牛仔裤，或者没有牛仔裤——我不知道你的衣柜里有什么！我说的“牛仔裤”指的是你感觉舒适的典型服装。如果你喜欢裤子，或者裙子，或者其他什么，穿上它们。只要它是半休闲的，穿着它不会让你看起来像得了流感，你就可以去。最重要的是你在这个地方感到舒适，穿着得体。如果天气冷，穿件夹克。如果你认为你可能要走一些路，穿不伤脚的鞋子。当第一次面对面交流时，*不要担心你看起来会怎样，而要关注你的感受*。舒适意味着你可以更有风度和自信，这将使你比穿着完美的有纽扣衬衫或花哨的裙子更有吸引力。所以，当有疑问时，穿牛仔裤——不管这对你意味着什么。
