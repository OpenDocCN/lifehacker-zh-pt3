# 骗子瞄准了丢失宠物的人

> 原文：<https://lifehacker.com/scammers-are-targeting-people-with-lost-pets-1809941864>

失去宠物感觉很糟糕。你很伤心，害怕，绝望——你会做任何事情让你的毛茸茸的朋友回来。这让你成为骗子的完美目标。



[据商业改善局](https://www.bbb.org/council/news-events/bbb-scam-alerts/2017/07/lost-pet-watch-out-for-this-text-message-scam/) 报道，最近骗子们一直试图利用丢失宠物的主人。它是这样工作的:

1.  你在公共场合张贴你丢失的宠物的信息——或者在网上，或者在海报上，或者两者兼而有之——并列出你的联系方式，以防有人找到它们。

2.  很快，你收到一条短信，声称有人找到了你的宠物。但当你要求他们描述它或给你发一张照片时，他们说他们不能，借口是不在城里或没有智能手机。

3.  然后，他们向你要钱、礼品卡或某种奖励，以便归还你的宠物。

不要上当！你可能会不顾一切地尝试任何能确保你宠物安全的事情，但是他们没有你的猫或狗。如果你给他们钱，你就再也听不到他们的消息了。

BBB 提供了一些有用的建议，你可以遵循，以避免这些骗局。首先，当你发布关于你丢失的宠物的信息时，限制你分享的信息量。例如，不要分享你的宠物独特的身体特征的信息。这样，当有人说他们有你的宠物，你可以问他们，并验证他们确实有它。第二，向打电话的人/发短信的人要电话号码，这样可以给他们回电话。他们可能使用了伪造的电话号码。最后，尽可能让他们给你发一张照片。如果他们能给你发一张你宠物的照片，看起来不像你贴的那张，你就知道那是真的。
