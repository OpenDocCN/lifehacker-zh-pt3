# 用 Sugru 为你的孩子做一个防摔相机

> 原文：<https://lifehacker.com/make-a-drop-proof-camera-for-your-kid-using-sugru-1797449398>

通过你孩子的眼睛看世界的快照是很有趣的。看着他们跑来跑去，手忙脚乱地拿着你的相机或手机可不是什么好玩的事儿(*不要掉，不要掉，请不要掉*)。

Watch

花 40 美元，你可以买一台这些年来越来越好的儿童相机，但是如果你有一台旧数码相机放在某个地方，你可以用 Sugru 简单地防止它掉落。根据相机的不同，你可能会获得更高质量的照片，而且你会抛弃儿童机型标配的愚蠢的自拍滤镜和游戏，保持老派风格。(你的初露头角的摄影师可能不需要她的脸出现在一只弹竖琴的卡通蜗牛上。)

查看[Sugru.com 教程](https://sugru.com/tech-gadget/how-to-make-an-awesome-bouncy-kids-camera) 如何用“墙”盖住你的相机，让它经受住不可避免的撞击和跌落。然后把它交给你的孩子，说，“茄子！”
