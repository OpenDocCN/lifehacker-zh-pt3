# 使用这些键盘而不是 Swype

> 原文:[https://life hacker . com/use-these-keyboards-instead-of-swype-1823169350](https://lifehacker.com/use-these-keyboards-instead-of-swype-1823169350)

Swype，我们 [最喜欢的安卓](https://lifehacker.com/five-best-android-keyboards-5922522) 和 iOS 的 [键盘之一，是](https://lifehacker.com/the-best-third-party-keyboards-for-ios-8-1636566071) [死](https://www.xda-developers.com/swype-keyboard-android-dragon-dictation-discontinued/) 。快公司称之为 [一个时代的终结](https://www.fastcompany.com/40533206/swype-keyboards-demise-feels-like-the-end-of-an-era?partner=feedburner&utm_source=feedly&utm_medium=feed&utm_campaign=feedburner+fastcompany&utm_content=feedburner) 。但是我们已经为仍然想用手势打字的 Swype 爱好者找到了一些替代品。

Watch

在 Android 上，你最好的选择是 [SwiftKey](https://swiftkey.com/en/) ，它有滑动输入和从你的习惯中学习的预测文本。或者试试不断改进的 [默认谷歌键盘](https://play.google.com/store/apps/details?id=com.google.android.inputmethod.latin)；它已经用了好几年了。

在 iOS 上， [SwiftKey](https://swiftkey.com/en) 很好用，但我最喜欢的还是谷歌的 [Gboard](https://itunes.apple.com/us/app/gboard-a-new-keyboard-from-google/id1091700242) 。两者都提供滑动打字和预测文本学习。Gboard 还能让你通过名字找到表情符号，或者在键盘上运行谷歌搜索。
