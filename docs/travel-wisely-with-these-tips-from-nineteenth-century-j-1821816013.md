# 根据这些来自 19 世纪日本的建议，明智地旅行

> 原文：<https://lifehacker.com/travel-wisely-with-these-tips-from-nineteenth-century-j-1821816013>

在游览日本这个美丽的国家时，围绕着优图·罗根·保罗和他幼稚的滑稽动作 有很多争议。好吧，如果你没有意识到，当你去不熟悉的地方时，有一种你应该和不应该做的事情。日本人几百年前就知道这些规则了。

Watch

早在 1810 年，Yasumi Roan 为日本游客写了一本终极指南:*魉呼游记*。它充满了多年来从其他旅行者和过去的各种著作中收集的提示、技巧和建议。这本书大受欢迎，至今仍在出版，其中包括一个英文版本，名为[*，由威廉·斯科特·威尔森*](https://www.amazon.com/Afoot-Japan-Nineteenth-Century-Walking/dp/1535503297?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/travel-wisely-with-these-tips-from-nineteenth-century-j-1821816013&asc_source=&tag=kinjalifehackerlink-20)*翻译。罗恩知道交通规则，也知道当一个人在离家很远的地方做客时应该怎么做。*

<section class="q35npn-0 MZtuu">*![](../Images/aef335eb562a1da6957ce25a574b4aa1.png)*

*“你不应该无所事事地伸手去拿路边的房子或花园里生长的水果，如日本梨、柿子、香橼和柑橘，不管它们有多成熟和丰富。当然，你不应该错误地踩在村庄或花园里正在晾干的谷物上。如果人们抱怨你在不属于你的领域的行为，无论你是对是错，你都不会站在争论的最有利的一方。”*

</section>

*很明显，你不应该拿走或破坏别人的食物，但这里的教训并不是关于水果和谷物。注意最后一行。你必须格外小心，以免在旅行时冒犯他人。无论你认为自己是对是错，如果当地人不喜欢你的行为，你都不会好看。所以也许 [不要跑来跑去，把死鱼和章鱼的部分粘在人们的脸上寻求刺激和观赏](https://kotaku.com/japan-isnt-happy-with-logan-pauls-tokyo-video-either-1821766445) 。* 

<section class="q35npn-0 MZtuu">*![](../Images/aef335eb562a1da6957ce25a574b4aa1.png)*

*“当你遇到年轻女士、女性割草者或一群女性在山里或穿过田野的路上经过你时，最好提供简单的问候，但不要再继续任何无用的谈话。此外，你不应该不假思索地嘲笑你(在路上)可能遇到的人的土气表情。要知道，麻烦可能从小事开始。”*

</section>

*再说一次，真正的教训不是关于你应该如何对待女性(非常不同的时代和文化)，而是关于知道界限在哪里。小心你所说的和你所嘲笑的，尤其是因为某些事情不同了。不要待得太久，问粗鲁的问题，发表批判性的言论，或者指着和你不同的人(或做不同的事情)嘲笑他们。*

<section class="q35npn-0 MZtuu">*![](../Images/aef335eb562a1da6957ce25a574b4aa1.png)*

*“当任何人去一个他不熟悉的地区时，[他会发现]各种说话方式和习俗都会不同。由于这些话与他生活的地方不同，他将习惯于听到它们，而不习惯于看到他周围的东西。尽管他会觉得这些事情很奇怪，但可以肯定的是,(这个陌生地区的)人们会对他有同样的想法。意识不到这一点，嘲笑另一个地方的习俗和语言，是一种错误。嘲笑和蔑视他人的话语或措辞可能会引发争吵。”*

</section>

*你可能认为一些不熟悉的食物或习俗很奇怪，或者认为你所去的国家的人很奇怪，但他们肯定对你有同样的想法。嘲笑或瞧不起异乡的风俗和人是给自己找麻烦的捷径。当你引起这样的麻烦时，你让其他人都像你一样看起来很糟糕。记住，当你旅行时，你是你祖国的非正式使者。*

<section class="q35npn-0 MZtuu">*![](../Images/aef335eb562a1da6957ce25a574b4aa1.png)*

*“在旅途中你不应该停下来看的事情:打架、争吵、赌博、围棋比赛、T2 商学比赛、乡村舞蹈、乡村相扑比赛、意外死亡的人，或者有人被杀的地方。总的来说，你不应该停下来盯着很多人聚集的地方。”*

</section>

*我不相信人们需要被告知这些(看着你洛根)，但是呆呆地看着(或记录)事故、打架、争吵或*死尸*无论你在哪里都是粗鲁和不尊重的，尤其是当你是客人的时候。不管发生了什么吸引人群的事都与你无关。你不仅没有卷入这起事件，你甚至不是一个有权关心此事的公民。如有必要，通知当局，否则就离开。现在，Roan 还将游戏、舞蹈和相扑比赛列为你不应该盯着看的东西，但由于这些类型的东西在很大程度上被认为是旅游景点，你可以继续看下去。动动脑筋。如果你在看一些事情展开时感到尴尬或不确定，不要逗留，出于同情，不要拍摄它。*
