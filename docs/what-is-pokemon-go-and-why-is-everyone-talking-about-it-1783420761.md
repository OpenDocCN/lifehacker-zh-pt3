# Pokémon Go 是什么，为什么大家都在谈论它？

> 原文：<https://lifehacker.com/what-is-pokemon-go-and-why-is-everyone-talking-about-it-1783420761>

首先，我的朋友在超市买了一个小火龙。然后洪水开始了:方向盘上的硬币。酒吧的 Rhyhorn。煎锅上的鲤鱼王。它们都是要抓的神奇宝贝，是一款名为 [*神奇宝贝 Go*](http://www.pokemon.com/us/pokemon-video-games/pokemon-go/)T5】的新游戏的一部分。你可能听说过它——但是什么是*它，为什么每个人都对它着迷？* 



神奇宝贝 是“口袋妖怪”的综合，也是一个有着同样悠久历史的疯狂流行的系列。在《神奇宝贝》中，怪物在陆地上游荡，你的任务是找到、捕获并训练它们。然后你让他们和其他玩家战斗。随着年龄的增长，我在任天堂游戏机上玩了很多原版的神奇宝贝游戏，并在几代掌上游戏机上跟随这些冒险经历。我收集了交易卡，痴迷于(仍在播放的)电视剧。幸运的是，*你*不需要任何以前的神奇宝贝经验就可以享受*神奇宝贝 Go* 。

## 怎么玩*口袋妖怪 go* ？

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-sW5_wjSoLAA&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-sW5_wjSoLAA&start=0) 

*Pokémon Go* 是一款免费的手机 app，你可以为 [iOS](https://itunes.apple.com/us/app/pokemon-go/id1094591345?mt=8) 或 [Android](https://play.google.com/store/apps/details?id=com.nianticlabs.pokemongo&hl=en) 下载。下载并开始玩是免费的，但是你可以选择使用真钱购买名为 [PokéCoins](http://www.ign.com/wikis/pokemon-go/Shop) 的游戏内货币。(介于 100 枚神奇硬币 0.99 美元和 14500 枚高达 99.99 美元之间。)这些神奇宝贝硬币用于购买神奇宝贝球，这是你需要能够抓住神奇宝贝的游戏内物品。现在你不需要花费真金白银，但这仅仅意味着你需要付出你的时间和精力(这就是它的乐趣所在！).

这款游戏通过使用手机的 GPS 来定位你的真实世界，并通过增强现实在你的屏幕上显示那些看起来很酷的神奇宝贝，覆盖在你面前的东西上。而你——数字版的*you*——可以定制服装、派系(或你可以加入的玩家“团队”)和其他选项，你可以在游戏中升级。

每当我们在街上走几码远，绕过一个拐角，或者进入一个新的地方，寻找新的神奇宝贝时，我都会看到我的朋友们兴奋地拿出他们的手机。如果时机合适，野生神奇宝贝会向你扑来，让你有机会用神奇球抓住它们。当你捕捉一只神奇宝贝时，它会被添加到你的 Pokédex，一种神奇宝贝数据库，你可以在以后对它们进行个性化设置。然后是有趣的部分:你可以去 [你当地的“健身房”](http://www.ign.com/wikis/pokemon-go/How_to_Become_a_Gym_Leader) 和你的神奇宝贝与其他训练者(也是真人)战斗。

另一方面，通常是预先确定的地标，你可以与之互动并从中获取物品。这些项目中的一些将进一步提高你作为训练者的“能力”，或者只是吸引大量其他兴奋的 *Pokémon Go* 玩家到你的位置。总而言之， *Pokémon Go* 给了你很多可以做的事情，但是最大的吸引力之一还是它的社交方面。

## 为什么好像每个人都在玩？

这些天，如果你没有看到关于《神奇宝贝 Go》的头条新闻，你就无法阅读新闻。如果你浏览推特或脸书，你会看到一些看起来很普通的照片——杂货店、某人的冰箱，甚至是 [一位母亲生下](http://www.telegraph.co.uk/technology/2016/07/08/man-catches-a-pidgey-on-pokmon-go-as-his-wife-gives-birth/)——你可能会发现里面有一只口袋妖怪。甚至我和我的朋友们的群聊也经常会聊起附近有哪些健身房和赌场。

每个人都喜欢《神奇宝贝 Go》,尽管玩它会消耗电池和 [蜂窝数据](https://lifehacker.com/easy-ways-to-save-data-while-playing-pokemon-go-1783372269) 。出于好奇，我在脸书和推特上提出了这个问题，得到了一堆答案。它从显而易见的事情开始:

 [https://lifehacker.com/embed/inset/iframe?id=twitter-751988991772073984&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-751988991772073984&autosize=1) 

在脸书，斯蒂芬妮承认这不仅仅是神奇宝贝本身的吸引力:

> “把他们全抓起来！*咄。*“但是要回答你的问题:怀旧。其他人都在谈论这件事，所以有点同龄人的压力。

安德鲁喜欢在现实生活中捕捉神奇宝贝的想法。

> 这是我们拥有的最接近真实生活的神奇宝贝体验的东西。很多人玩游戏是为了怀旧，但神奇宝贝一直是个很酷的主意。

另一方面，安吉洛和这些推特用户从玩《神奇宝贝 Go》中发现了一些巨大的现实生活价值:

> 完美主义者“必须抓住所有人”对我来说绝对是一种诱惑。与现实生活中的地点相结合给了我一个理由，让我站起来，步行/跑步到附近我从未见过的地方。我过去两天做的有氧运动绝对比过去三个月加起来还多。此外，社交方面:我在现实世界中遇到了一群其他神奇宝贝训练师，这是一种与陌生人搭讪的简单方式。在现实世界中全神贯注于寻找神奇宝贝(并四处走动)的人数之多令人吃惊。

这里的共识是，游戏不仅有趣，特别是对于那些以某种方式与特许经营权相关联的人来说，而且它是人们走出去、探索他们从未去过的地方和结识新朋友的一个很好的方式。因为你遇到的人也在玩 *Pokémon Go* ，这个游戏立即为你提供了共同点(只是试图抓住那只皮卡丘),让你可以轻松地与其他训练者建立联系，或者建立难以置信的新友谊。甚至不玩这个游戏的人也受益匪浅:“我有朋友带他们的孩子出去抓神奇宝贝，”一个朋友告诉我。

当然，这不是唯一一个这样的游戏。精灵宝可梦 Go 背后的公司 Niantic 几年前制作了另一个受欢迎的 ARG，名为 Ingress，它有许多相同的好处，并提供相同的社交方面。它显然没有背后像神奇宝贝这样的品牌受欢迎，但我们之前已经提到过 [这些社交功能如何在现实世界的游戏](https://lifehacker.com/how-ingress-googles-real-world-smartphone-game-got-me-1710320867) 中发挥作用。

## 入门有多容易？

《精灵宝可梦 Go》的部分魅力在于它很容易玩。你只需要一部 iPhone 或 Android 手机，并有一个强大的 Wi-Fi 连接或蜂窝数据可用。如果你没有无限的数据计划，你绝对应该考虑只在有 Wi-Fi 的地区玩或者 [做大量数据使用。据报道，这款游戏每小时会占用](https://lifehacker.com/easy-ways-to-save-data-while-playing-pokemon-go-1783372269)[2 兆到 8 兆](http://www.pokemongodb.net/2016/04/10-best-ways-to-reduce-data-usage.html) 的数据，这取决于你在做什么。否则，有办法 [减少和保存尽可能多的数据](http://lifehacker.com/how-can-i-avoid-using-too-much-smartphone-data-1479097074) 用于 *Pokémon Go* 。

另外， *Pokémon Go* 真的可以 [吸你手机电池](https://lifehacker.com/how-to-save-your-phone-s-battery-while-playing-pokemon-1783445442) 。虽然游戏有自己的“省电”模式，但你应该把其他的 [省电措施掌握在自己手中](http://lifehacker.com/top-10-ways-to-improve-the-battery-life-on-your-phone-a-1725843518) 。首先，你可以降低手机屏幕的亮度。如果您打开了蓝牙，请将其关闭。如果你知道你无论如何都会耗尽你的电池电量，那么考虑 [配备一个外部电池组](http://lifehacker.com/five-best-external-battery-packs-509802431) ，就像 [这款 Anker 的 power core+(9.99 美元)](https://www.amazon.com/Anker-PowerCore-Lipstick-Sized-Generation-High-Quality/dp/B005X1Y7I2/ref=sr_1_4?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/what-is-pokemon-go-and-why-is-everyone-talking-about-it-1783420761&asc_source=&ie=UTF8&keywords=portable phone charger&qid=1468175221&s=wireless&sr=1-4&tag=kinjalifehackerlink-20) 。

如果你刚刚开始玩 *Pokémon Go* 并且不确定该做什么，请查看我们 Kotaku 的朋友提供的 [这份深度指南](http://kotaku.com/how-to-play-pokemon-go-1783354400) 。

Pokémon Go 是一个游戏，也是一个有趣的游戏，但它的本质是将陌生人聚集在他们不会去的地方。我的意思是，我的朋友可以聚集在街上的垃圾桶周围，专注地盯着他们的手机，这让我感到非常惊讶，并吸引了更多的路人，他们也立即掏出手机检查。
