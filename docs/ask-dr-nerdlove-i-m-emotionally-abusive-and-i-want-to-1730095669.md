好吧，AAA，这将是一个两部分。第一部分是给你的；关于你的行为，我有几件事要说，但在此之前，让我们直截了当地说:你需要和心理医生谈谈。NerdLove 医生不是真正的医生，当你谈论自杀意念时，你需要和医学专家交谈，而不是一个有专栏的大嘴巴约会教练。

这是显而易见的:是的，你在虐待你的女朋友，AAA。坦白地说，你不应该和任何人谈恋爱，除非你把自己的事情处理好。如果你真诚地对待你的问题，那么你需要去看心理医生并坚持下去。如果你对我直言不讳，那么你的问题只能通过和精神病医生一起工作*才能解决。焦虑、抑郁、惊恐发作和自杀想法是需要立即关注的事情，通常需要像抗抑郁药这样的医疗解决方案来让你到达其他形式的治疗可以帮助你的地方*。** 

*说完这些，我将把下一部分介绍给读者。*

*所以，完全公开，我在这个特定的话题上没有任何形式的目标。我以前有过被虐待的经历，我对这个话题特别敏感，不愿意给虐待者带来好处。坦率地说，这封信中的一些内容让我的蜘蛛侠感觉到刺痛。但是我们一会儿会谈到所有这些。我选择这封信的原因之一是因为它说明了一些很多人不一定了解的东西:虐待并不完全是暴力，甚至不是大喊大叫或让某人的伴侣感到威胁——许多虐待者利用了这一点。有很多*方式可以让施虐的一方对某人造成情感或身体上的伤害，而不一定要举手或发声；事实上，没有身体上的伤害或危险通常是施虐者让受害者相信没有虐待发生的方法之一。T9】**

*虐待关系——无论虐待是身体上的还是精神上的— [几乎总是与控制](http://www.doctornerdlove.com/2012/10/relationship-toxic/all/1/) 有关。施虐者正在控制他们的受害者，不管是通过暴力、恐吓还是恐惧。有时，这种恐惧是对施虐者伤害受害者的恐惧——直接(通过身体虐待)或间接(破坏受害者的财产，伤害或杀死他们的宠物，通过社会破坏等)。其他时候，这种恐惧是害怕施虐者伤害自己*。*

自残威胁其实是一种很常见的控制一个人伴侣的手段。事实上，我从一开始就已经 [守备](http://www.doctornerdlove.com/2013/05/dr-nerdlove-hostage-situations/) [书信](http://www.doctornerdlove.com/2013/05/dr-nerdlove-hostage-situations/) 了。不同的是，通常我听到的是被虐待者的声音，而不是施虐者的声音。实际上，这种情感虐待者正在制造人质危机。唯一不同的是，他们把自己作为人质，一边说“你敢离开我，否则人质就死定了”，一边用枪指着他们的头。*

*通过威胁自残，他们将受害者的负罪感和责任感武器化，将施虐者的行为归咎于受害者:“这都是你的错！如果你没有让我心烦意乱，这永远不会发生。”很多时候， [受害者将责任内化，开始相信这*是*他们的错](http://www.doctornerdlove.com/2012/06/post-mortem-used-abused/) 。因此，现在他们觉得他们必须保持高度警惕，不断监控自己的行为，以免引发另一次爆发。所以施虐者保持对受害者的控制，甚至不用动手。*

*然而，威胁并不是人们试图在一段关系中保持控制的唯一方式。当某人犯了一个错误(或者他们因为做了错事而被叫出来)时，大声地过度道歉也是*也是*一种控制行为；通过制作这样一部超越负罪感的作品，他们扭曲了故事情节，颠倒了角色。现在，不再是“某人 X 对某人 Y 做了错事”，而是“某人 Y 必须向某人 X 保证，他们不是世界上最糟糕、最糟糕的人”。它改变了焦点；人 Y 不再有权利生气，因为人 X 是*，所以*后悔继续难过是残忍的，等等。当那个人 X 把自残加到组合中时，通过打自己、用拳头或头撞墙等方式“惩罚”自己。-然后 Y *人不得不*放下他们将要说的任何话，因为害怕对方最终会严重伤害自己。*

*在这两种情况下，另一方都要承担责任，负责安抚他们的伴侣。在这两种情况下，他们不得不把自己的欲望、需求和不满放在一边，照顾他们的伴侣，而控制、虐待他们的伴侣则被免除责任。*

*回到我们的写信人，AAA:你告诉我，你从小就默认自残来回应批评；你有如此强烈的羞耻感和负罪感，以至于你会自然而然地受到这些不适应行为的攻击。好吧，好吧……除了这符合*确切的*同样的操纵、控制行为，这种行为要么免除你的责任(我应该说“这是可以理解的，这不是你的错”)，要么作为一种潜在的威胁(“对我温柔点，否则谁知道我会做什么？”).*

*公平地说，也许我理解错了。也许是因为英语不是你的第一语言，这是一种沟通上的错误。也许是你漏掉了一些可能会增加上下文的东西。但是你信中的某些东西没有通过我的嗅觉测试。*

*有没有可能你不是有意识地虐待你的女朋友？嗯，完全有可能你不认为这是控制，你小时候就知道这种行为会让别人不再烦你。作为一个物种，人类不擅长表演；如果我们以一种特定的方式行为足够长的时间，我们的大脑开始合理化这种行为，并为我们这样做的*为什么*想出理由——即使我们有意识地知道这是一种行为。所以，是的，从理论上来说，这是可能的，通过在孩童时期采取这种行为，你已经将其内在化并合理化，你不是在控制，你这样做是因为你出于内疚和羞耻。*

*但即使是这样， ***无意的虐待依然是虐待*** 。这是无意的*这一事实并不能成为你不这么做的借口。如果你真的想变得更好，那么你需要负起责任，而第一步就是接受治疗。靠你自己。**

*你看过《追艾米》吧，LitS？因为现在你基本上是被 Holden。你也不想完全被套牢。

所以有些直谈性取向。人们的性行为比我们想象的要复杂和多变。双性恋或泛性者可能对多个性别有性吸引力，但只对其中一个性别感兴趣。一些异性恋或同性恋者很可能决定他们想尝试走在野外，和那些不符合他们偏好性别的人鬼混。有些人更喜欢一种性别，但他们会闭上眼睛，假装这是离开的必要条件。*

*见鬼，有时候只有一个人是他们性取向的例外。一个完全的直男可能会意识到他们喜欢的是一个特定的人。同样，女同性恋可能意识到她对某个特定的男人有意思。这并不意味着他们不再分别是异性恋或同性恋，这只是意味着性是他妈的复杂，吸引力通常不会在乎你如何给自己贴标签。*

*这是怎么回事？会不会是她对你有意思却忘了是什么时候遇到别的女人的？当然，理论上是可能的。也有可能她更接近金赛量表中的双性恋部分，觉得这与她的女同性恋身份相冲突。可能她是双性恋，但也是同性恋；也就是说，她在性方面对男人和女人感兴趣，但只对女人感兴趣。或者也可能是她喜欢光亲热，但她对你还没有到想要约会的程度，她认为把这放在她的性行为上比告诉你更友好。

不管怎样，问题的核心不在于她是同性恋、双性恋、异性恋还是其他什么。只是她不像你希望的那样喜欢你。她可能很爱你，但这并不意味着她以你想要的方式爱你。太糟糕了，我完全理解。但是，留在她身边，希望她会忽略整个“阴茎”的事情只会以眼泪告终，并可能最终伤害你们的友谊。*

*你已经完成了你的计划。她拒绝了你。如果你想保持友谊，你需要成为她的朋友，这意味着要学会忘记她。朋友不会带着希望改变他们朋友想法的议程到处闲逛。那是 [老好人的领地](http://www.doctornerdlove.com/2012/12/problem-nice-guys/) 。*

*祝你好运。*
