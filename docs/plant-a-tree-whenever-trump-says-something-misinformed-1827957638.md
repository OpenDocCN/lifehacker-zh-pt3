# 每当特朗普用 Treespond 说一些关于环境的错误信息时，就种一棵树

> 原文：<https://lifehacker.com/plant-a-tree-whenever-trump-says-something-misinformed-1827957638>

如果你对川普总统在气候变化和其他环境问题上的错误想法感到沮丧，现在你可以在他每次使用网站 [Treespond](https://treespond.com/) 发表错误想法时做一些有助于环境的事情。



这个想法是，你用一棵树来回应总统可能不正确的推文或声明，这实际上有助于改善情况，而不是一条空洞的推文或脸书邮报，这不会真正移动指针。

该网站收集了特朗普最令人沮丧的一些言论。你从集合中选择一个，将你的树响应归于。

 [https://lifehacker.com/embed/inset/iframe?id=youtube-video-YDSbufFxZok&start=0](https://lifehacker.com/embed/inset/iframe?id=youtube-video-YDSbufFxZok&start=0) 

情感上的满足是有代价的，特别是每棵树 9.99 美元。你通过网站购买的每棵树都有自己的 ID 号。你会得到一张你的树的照片，以及它在世界上的地理位置，所以你可以在某个时候去拜访它，如果心情不好的话。现在所有的树都被种植在加利福尼亚，那里的野火摧毁了大片林地。

诚然，一棵树(或 100 棵树)不会产生巨大的影响，但这是一种快速而简单的方法，可以将一点点愤怒和沮丧转化为一点点好事，并且可能是一种很好的做法除了定期 [联系你的政府代表](https://lifehacker.com/the-best-ways-to-contact-your-congress-people-from-a-f-1788990839) 。
