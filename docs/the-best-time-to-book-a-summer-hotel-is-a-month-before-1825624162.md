# 旅行前一个月预订，享受夏季酒店的最优惠价格

> 原文:[https://life hacker . com/1825624162](https://lifehacker.com/the-best-time-to-book-a-summer-hotel-is-a-month-before-1825624162)一个月前是预订夏季酒店的最佳时间

说到预订酒店，有很多关于什么时候是预订的“最佳”时间的理论。我认识一些人，他们发誓你应该在开始计划旅行时就预订，还有一些人认为在你出发前几天是达成交易的最佳时机。事实上，答案介于两者之间。

Watch

猫途鹰 [分析历史酒店价格数据](https://tripadvisor.mediaroom.com/2018-04-25-TripAdvisor-Announces-the-Best-Time-to-Book-Hotels-for-Maximum-Summer-Savings) 以确定在世界各地预订酒店的最佳时间。它的决定是，夏季最佳购买时间是在你旅行前一个月。

根据其研究，夏季酒店的平均费用将为每晚 236 美元。选择在出行前一个月预订的旅行者可以在高峰时段享受 15%的折扣。

在美国，波士顿、芝加哥、奥兰多、圣地亚哥、拉斯维加斯、迈阿密、纽约市、费城、旧金山和华盛顿特区是等待预订客房的最省钱的地方。

在欧洲，情况有所不同。尽早预订通常能节省最多的费用；然而，8 月份去伦敦或巴黎的人应该考虑在行程的五周内预订房间，猫途鹰认为在这段时间内你可以以 25%的折扣预订房间。
