# 这张图片显示了每一个飓风等级之间不断增强的强度

> 原文：<https://lifehacker.com/this-visual-shows-the-increasing-strength-between-each-1800826226>

飓风哈维于 8 月 25 日袭击了得克萨斯州海岸，成为 4 级飓风[。](http://www.weather.gov/crp/hurricane_harvey) 

本周早些时候，厄玛以 5 级飓风袭击了小安的列斯群岛北部，并继续向西和向北移动，引起了加勒比海和南佛罗里达的高度警惕。

Watch

虽然 [我们已经讨论了这些不同类别类型的实际含义，](http://lifehacker.com/hurricane-categories-and-what-they-mean-1800012360#_ga=2.227897693.1152811464.1504616873-1313785359.1499701416) 但很难想象第 3 类和第 5 类之间的真正区别，因为它们只是夜间新闻中流传的不祥术语。

为了帮助这种可怕的心理练习，这里有一个关于飓风等级不同强度的动画，由大学大气研究公司(UCAR)和国家大气研究中心(NCAR)的 [AtmosNews](https://www2.ucar.edu/atmosnews) 开发，科学作家 [Brian Kahn](https://twitter.com/blkahn) 分享:

 [https://lifehacker.com/embed/inset/iframe?id=twitter-905141924939649024&autosize=1](https://lifehacker.com/embed/inset/iframe?id=twitter-905141924939649024&autosize=1) 

不管是好是坏，这种视觉在某种程度上使得“毁灭性破坏”(第 3 类)和导致“不适宜居住”区域的“灾难性破坏”(第 4 类)之间的区别看起来更加具体。
