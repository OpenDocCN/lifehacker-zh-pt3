# 平面灰色主屏幕

> 原文：<https://lifehacker.com/the-flat-gray-home-screen-1729909756>

干净的主屏幕是快乐的主屏幕。这种设计让你的手机有条理的大而清晰的小工具，包含你需要的所有基本信息。



这个主屏幕是为 Nova 和 Apex 等定制启动器设计的。这是你需要的:

*   [新星](https://play.google.com/store/apps/details?id=com.teslacoilsw.launcher.prime) ，，[顶点](https://play.google.com/store/apps/details?id=com.anddoes.launcher) ，或者其他自定义发射器
*   [终极定制 Widget](https://play.google.com/store/apps/details?id=in.vineetsirohi.customwidget)
*   [平面灰色主题](https://play.google.com/store/apps/details?id=com.mobixscreens.darkgray) 或 [棕色变体此处](https://play.google.com/store/apps/details?id=com.mobixscreens.brownflat)

你可以在平面灰色主题的 Play Store 列表中找到如何安装这种设计的说明。

你有没有一个很棒的、被调整到遗忘的家或者你自己的锁屏，你想分享吗？把它贴在下面的评论里，或者贴在你自己的 Kinja 博客上，标签是“主屏幕展示”(没有引号)。一定要包括你是如何制作它的描述，这样我们就可以把它作为下一个特色主屏幕。
