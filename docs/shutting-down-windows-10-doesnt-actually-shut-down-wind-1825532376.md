# 关闭“快速启动”来强制 Windows 10 真正关闭

> 原文:[https://life hacker . com/shutting-down-windows-10-sints-actually-shut-down-wind-1825532376](https://lifehacker.com/shutting-down-windows-10-doesnt-actually-shut-down-wind-1825532376)

这听起来很奇怪，但当你点击开始菜单上的那个电源按钮来关闭 Windows 10 时，你并没有关闭 Windows 10。当然，Windows 10 会经历关机的过程。你的电脑看起来好像关机了。有点像，但是没有。

Watch

这是一个较老的 Windows 10 怪癖，但值得不时强调一下，以防你忘记“关机”并不是真的“关机”。当你关闭电脑时，你实际上在做的是将它置于混合状态，即 Windows 10 的设置中默认启用的“”功能。正如 [Windows Central](https://www.windowscentral.com/how-disable-windows-10-fast-startup) 所描述的:

> *“在启用快速启动的情况下，选择关闭您的电脑可能看起来像是完全关机，但实际上，您的电脑正在进入关机和休眠的混合状态。确实使用了休眠文件，尽管它比通常的要小。为什么？您在文件创建之前就注销了，这意味着您的会话没有被记录。速度提升来自于 Windows 内核保存在硬盘上，并在启动时加载。”*

### 为什么要关闭你的电脑？

Windows 10 的快速启动功能没有任何问题。根据您的系统配置，当您“启动”您的 PC 时，它可以很好地让您快速回到操作系统。

然而，有时候你可能想要真正地关闭你的系统——例如，如果你的 Windows 有一个奇怪的问题。正如 [How-To Geek](https://www.howtogeek.com/349114/shutting-down-doesnt-fully-shut-down-windows-10-but-restarting-it-does/) 所指出的，一次标准的关机可能不足以让 Windows 在这些有趣的时刻停止崩溃:

> “我们自己也亲身经历过这个问题。当遇到可能由错误的驱动程序或其他低级软件问题引起的奇怪系统问题时，在关闭我们的电脑并重新启动后，问题仍然存在。”

你可以尝试的一个快速技巧是重启你的系统，这将丢弃 Windows 内核，而不是创建一个快照来加速启动。但是如果你真的想关掉你的系统，你有几个选择:

### 考虑禁用快速启动

像平常一样关闭系统，在智能手机上打开你最喜欢的秒表应用程序，计时当你重新启动电脑时，它需要多长时间才能启动到 Windows。写下这个数字——或者只是记住它。

现在，点击开始按钮，输入“电源”，选择“电源和睡眠设置”选项，然后点击“其他电源设置”在出现的窗口中，点按左侧边栏中的“选取电源按钮的功能”，并取消选择“打开快速启动(推荐)”您可能必须首先单击“更改当前不可用的设置”来取消选中快速启动。

再次关闭你的系统。开始备份时，记录计算机启动到 Windows 所需的时间。如果这个数字和上一个数字之间的差异非常大，那么重新打开快速启动。如果只是几秒钟，或者你觉得在日常使用中不会注意到的时间，你可以不选中快速启动，以确保你总是(真的)关闭你的电脑。

快速启动可能也没什么区别。当我计算时间时，从按下按钮到锁定屏幕并启用“快速启动”用了 29 秒...关了 27 秒。\_(ツ)_/

### 管理混合关机的混合方法

如果你想使用 Windows 的快速启动，但仍然希望在你认为可能需要的时候执行完全关机，有一些技巧可以尝试。

最简单的方法就是按住 shift 键，然后单击电源图标，在 Windows 的开始菜单、Ctrl+Alt+Del 屏幕或锁定屏幕上选择“关机”。这将迫使你的系统实际上关闭你的电脑，而不是混合关闭你的电脑。

如果你想变得更有趣，你也可以在命令提示符下关闭你的系统(真的):只需输入“shutdown /s /f ”,然后回车。/s 标志表示您想要关闭您的系统，而/f 表示您想要强制关闭任何打开的应用程序(如果您添加/t 和一个数字，您也可以在系统关闭之前给自己这么多秒的延迟。)

您甚至可以将这个小命令设置为桌面快捷方式，以便于访问。右键点击你的 Windows 10 桌面，选择新建，然后快捷方式，并输入这个快捷方式的位置:`shutdown /s /f `(感谢 life hacker reader**cmdr keene**关于较小快捷方式的说明！)

之后，点击下一步，给你的快捷方式起一个有趣的名字(比如“回收站”)，然后点击完成。下次你想关闭你的电脑，只需双击你的关机快捷方式。更好的是，把它拖到你的工具栏上以便于访问(以及当你想点击文件浏览器时意外点击“关闭”所带来的乐趣)。
