### **本周讨论:Windows 10 对 macOS**

苹果的软件长期以来一直被誉为设计更好、更易于使用的系统，Windows 是一个不那么光鲜但适应性更强的选择。这种过于简单化的说法还有道理吗？嗯，这两个系统现在是如此的功能一致，以至于旧的竞争不再是真的了。这两个系统都提供了你所期望的现代操作系统的所有主要特性，对于日常使用来说，大多数人都可以接受。

不过，他们的不同之处在于细节。如果你是一个游戏玩家，你可能会特别想坚持使用 Windows，因为你可以调整你自己的硬件来满足你的需求。这可能意味着从头开始构建一个混乱的系统，或者只是升级你的显卡。

不利的一面是，Windows 更容易成为病毒和恶意软件的目标，原因很简单，因为各种版本的 Windows 占有如此大的市场份额，以至于它很容易成为目标。这并不意味着 macOS 对恶意软件免疫；只是不太常见的目标。

macOS 和 Windows 10 都有语音助手，Windows 上有 Cortana，Mac 上有 Siri。虽然当你只想查看天气或在线搜索时，它们会很有帮助，但我们发现我们并不经常使用它们。这感觉比在电话上使用语音助手不太自然，笔记本电脑的麦克风也不总是那么好用。

这两者之间最大的区别是实际上你可能使用的硬件。为了探究这两者的区别，Gizmodo 的 Christina Warren 和 Alex Cranz 加入了我的节目。多年来，苹果的笔记本电脑一直是设计精良、坚固耐用、日常使用的电脑的相对黄金标准，你可以买到它，而不必担心它的规格。有很多更便宜的选择，但 MacBook Pro 就是好用。然而，最近更新的 MacBook Pro 系列包括独特的 Touch Bar，如果你想要高端规格，它们会变得特别昂贵。不过，如果你想使用 macOS，你基本上只能依赖苹果的硬件。(或者你可以尝试走‘Hackintosh’路线——也就是 [在非苹果硬件上安装 MAC OS](https://lifehacker.com/the-always-up-to-date-guide-to-building-a-hackintosh-o-5841604)。)

相比之下，Windows 被设计用于各种硬件，从笔记本电脑到台式机和触摸屏平板电脑。这意味着你只需花几百美元就能买到一台低端 Windows 笔记本电脑，但微软也推出了新的高端硬件，如 Surface Book 。也有很多中庸之道，戴尔 XPS 13 的 [通常被推荐为物超所值。当然，如果你想为游戏或其他处理器密集型应用构建你自己的装备，那么 Windows 是最好的选择。](http://gizmodo.com/dell-xps-13-review-2015-the-windows-laptop-to-beat-1681976712)
