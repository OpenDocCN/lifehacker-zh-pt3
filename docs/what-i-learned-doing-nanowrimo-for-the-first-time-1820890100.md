# 我第一次做 NaNoWriMo 学到的是

> 原文：<https://lifehacker.com/what-i-learned-doing-nanowrimo-for-the-first-time-1820890100>

过去的一个月，我参加了全国小说写作月(也称为“NaNoWriMo”)。这很有趣，令人筋疲力尽，充满压力，有时还很痛苦。但这也是我在时间管理和创造力方面学到的最重要的一课。

Watch

NaNoWriMo 的目标是在一个月内写完 50000 字的小说。这可以分解为每天 1667 个单词(作为参考，这篇文章还不到这个数字的一半)。你的小说不一定要写完——我的不是——但是为了“赢”,你必须至少写下这么多单词。这是令人兴奋的努力！想象一下一个月后你的生活，你有了一本小说，或者至少是一本小说的伟大开端，你就会明白了。尽管如此，当你有一份全职工作、一段私人生活、一个家庭和其他正在进行的兼职项目时，要找到时间把你的话说完是非常困难的。

但这一切都是值得的。NaNoWriMo 教会了我很多东西。我一劳永逸地知道了白天什么时间最适合我写作；我明白了在写作时，听环境声音和听音乐一样好，甚至更好；我认识到最好把有趣的部分留到以后写，这样当你开始下一次会议时，你就可以立即投入工作；我发现，当我努力做一个大项目时，我会为其他项目产生更多的想法。这就像一个创造力反馈循环，你做得越多，你就越觉得自己越有能力。

然而，最重要的是，我认识到总有时间去创造事物。无论我有多忙——全职工作、其他写作项目、多次周末去圣地亚哥旅行、去纽约出差、在多日会议上主持会议——总会有一点时间坐下来，为我的故事说几句话。我甚至设法在我生日的时候写信。伤心欲绝吧，斯蒂芬·金。NaNoWriMo 给我上了一堂关于时间管理的严肃课，比我从事的任何其他创造性努力都要多。我知道“找”时间是行不通的。你不找时间， [你“制造”时间](https://lifehacker.com/you-don-t-find-time-you-make-it-1785505571) 。但是这个挑战对我来说十倍地证明了这个概念，特别是随着时间的推移，保持正轨变得越来越难。每天落后一点点，就意味着快到终点的时候落后很多。如果有一天我没有写够，我知道我以后必须补上，这在下半月变得很明显。毫无疑问，在我看来，如果我在 10 天后就放弃了，这一课就不会对我造成影响。你需要整整一个月。

这个挑战也迫使我重新审视我在开始做 NaNo 之前的空闲时间。尽管我写的东西比以往任何时候都多，但在 11 月份，我仍有充足的时间去看朋友、看电视、看电影、旅游，甚至玩电子游戏。不可否认的是，我做这些事情的次数没有以前多了，但令人高兴的是，我并没有那么想念它。这有点像吃几口甜点，而不是狼吞虎咽地吃一整盘。我仍然得到了做那些事情而不过度的满足感。你可以鱼与熊掌兼得——只是不能全部吃掉。

我不知道这种来之不易的心态的影响会持续多久。到目前为止，我几乎不能像以前那样坐下来享受视频游戏或看电视，因为我觉得我没有效率。就像有一个穿着西装的小肩魔不断用成功的干草叉戳我。有动力的感觉很好，但也很累。尽管如此，在我尝试之前，我会回顾我的生活，想知道我到底做了些什么。我相信我已经开始形成一个新的、非常有益的习惯，而且我不打算在短期内甩掉这个魔鬼。
