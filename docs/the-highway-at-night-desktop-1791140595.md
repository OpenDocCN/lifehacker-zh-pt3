# 夜晚的高速公路桌面

> 原文:[https://life hacker . com/the-highway-at-night-desktop-1791140595](https://lifehacker.com/the-highway-at-night-desktop-1791140595)

读者 [塞巴斯蒂安](http://kinja.com/sebastianerz) 向我们的 [桌面展柜](http://kinja.com/tag/desktop-showcase) 提交了这个漂亮的桌面，壁纸和几个有品味的皮肤的结合是一个坚实，锐利的外观。他是这样设计的。

Watch

首先，Sebastian 使用 Rainmeter 得到这个外观，而 [我们刚刚更新了我们的 Rainmeter](https://lifehacker.com/how-to-create-an-attractive-customized-desktop-hud-wit-5828789) 指南，所以如果你想开始调整它，你可以检查一下。

一旦你下载了它，并且熟悉了它的工作原理，你需要做的就是把它变成你自己的样子:

*   [城市高速公路夜景壁纸](http://www.wallpapersxl.com/wallpapers/2560x1440/city-highway-night/222083/city-highway-night-photography-landscape-nature-hd-222083.jpg)
*   用于 Windows 的 [雨量计](https://www.rainmeter.net/) 系统调试和监控工具
*   音乐文本的雨量计的 [明文](http://redsaph.deviantart.com/art/Cleartext-for-Rainmeter-v3-3-0-21-Dec-2016-519796161) 小部件
*   “正在播放”作品的雨量计的 [封面](http://rainmeter.deviantart.com/art/CoveR-481351911) 小部件
*   [考恩时钟](http://marcarnal.deviantart.com/art/Cowon-Clock-for-rainmeter-191393706) 皮为雨量计右上方为时钟
*   左下方的 [圆形发射器](http://lybrica.deviantart.com/art/Circle-Launcher-2-11-03-Aug-2016-547475678) 为办公发射器图标的雨量计皮肤
*   他平板电脑上用于天气显示的 [Simplic](http://hiphopium.deviantart.com/art/Simplic-588463108) 皮肤，由于 Duet Display， [我们最喜欢的将 iPad 变成第二个屏幕的应用](http://lifehacker.com/the-best-second-screen-app-for-ipad-1706351639) 。

你可以通过下面的链接把 iPad 作为第二个屏幕，但除此之外，它是一个简单、整洁的定制外观，不需要大量的移动部件就可以实现。这也是我们如此喜欢它的部分原因。如果你对自己的桌面有疑问，或者如何让你的桌面看起来像这样，点击下面的链接，在 Sebastian 的 Kinja 博客上查看。一定要让他知道你和我们有多喜欢这一切的结果！
