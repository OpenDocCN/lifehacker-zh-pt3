# 蒸玉米最好最简单的方法

> 原文：<https://lifehacker.com/steam-corn-using-either-of-these-two-easy-very-differe-1826779004>

我对夏玉米很着迷。你把奥普拉对面包的感觉乘以三，这就是我对玉米的感觉。那些脆甜的日照玉米粒是夏季农产品的所有优点，而且几乎不需要准备。



如果我不烤它——由于我没有院子和烤架，这是常有的事——我就把玉米棒子放在壳里蒸。蒸会让玉米粒散发出新鲜的玉米味，*和*它会让丝很容易去除，这真的是你对玉米烹饪方法的全部要求。在大锅里蒸并不难，但这是一个你必须注意的锅，还有另外两种更不干涉你的方法。

### 如果你只是为几个人煮玉米，使用速溶锅

即食锅蒸玉米很快。一旦炊具达到压力，你只需三分钟就可以大嚼玉米棒了。只需在插入物中加入两杯水，并在里面放置一个三脚架或蒸笼(你只是想让玉米不在底部)。修剪玉米棒的末端，使它们适合锅里(留下外壳)，尽可能多的塞进去。对于较长的棒子，水平堆放可能是你最好的选择。

关闭速溶锅(或任何具有蒸汽功能的高压锅)，确保释放阀翻转到“密封”，并按下“蒸汽”按钮。使用“+”和“-”按钮将时间调整为三分钟，然后走开，Renee。一旦你听到速溶壶的悠扬音调“我吃完了！”尖叫，手动释放压力，开锅，领取你的黄金奖励。通过修剪玉米棒的尾部，并从底部剥去外壳来去除外壳和丝。丝绸会很容易脱落。(如果你需要/想要一次融化一大堆黄油——你当然会这样做——只需将水从插入物中倒掉，然后在里面扔一根加盐的块状黄油。)

### 对于一群人的玉米，使用冷却器

“”是在人们使用“食物黑客”这个术语之前的一个食物黑客，它之所以成为经典是有原因的。你所要做的就是把一大堆玉米扔进一个大冷却器里，加入两壶开水，关上盖子，然后让它自己呆上半个小时。这是一个稍长的过程，但它是 99%的非活动时间，你不必用一个巨大的沸水锅挤满你的炉子。至于“化学物质渗入你的玉米”，这是一个常见的抱怨，每当提到较冷的玉米时，就找一个不含 BPA 的冷却器，或者使用一个巨大的 Cambro(这是用来储存食物的)或一个带盖的大锅。你只需要一个能装 20 只左右耳朵的密封容器，外加两壶水。(然而，考虑到毒药的剂量，我并不担心吃两个玉米穗，它们可能会或可能不会短暂地停留在提取冷却器的稀释溶液中，但你会。)
