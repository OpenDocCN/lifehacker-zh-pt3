# 任天堂 Labo:Kotaku 评论

> 原文：<https://kotaku.com/nintendo-labo-the-kotaku-review-1825412625>

你可能对 Labo 的第一个问题是:一盒纸板和一些迷你游戏有什么特别的？关于 Labo，首先要了解的是，盒子里的东西只是吸引力的一部分。



更有价值的是这个小小的墨盒及其附带的纸板能够实现的功能:放学后的下午和周末一起躺在地板上，周围是笔和胶带，制作东西并弄清楚它们是如何工作的。虽然这些模型的独创性和工程会给任何成年人留下深刻印象，但 Labo 在家庭环境中最有意义。当你将纸板和迷你游戏与孩子们随心所欲的想象力结合在一起时，它们会变得更加丰富多彩。

为了回顾 Labo，我查看了综艺套装，这是任天堂今天开始销售的两个 Labo 套装之一。另一个是机器人工具包，可以让你制作一个纸板背包，连接到你可以用全身运动控制的游戏。

开箱即用，Labo 多种套件让您随心打造。一个简短的实践教程产生了一个小小的 Joy-Con 支架，然后是盒子里的五个模型:一个简单的 RC 汽车，在平坦的表面上抖动，由 Joy-Con 振动控制；有按钮和曲柄的小房子；钓鱼竿；简约风格的摩托车；还有一架纸板钢琴，当它与游戏机和控制台结合在一起时，就变成了一架功能齐全的钢琴。要制作这些模型，你需要遵循 Switch 屏幕上类似乐高积木的说明，这些说明清晰、全面，并带有俏皮的个性。然后，你可以玩为每个作品设计的小游戏，并进入充满可访问教程的发现菜单，告诉你它们是如何工作的。

每个模型需要半个小时到两个小时来建造，但是一旦说明完成，你不必停下来。用胶带、笔、贴纸和你手头的任何其他工艺用品定制模型是独立于软件之外的一大乐趣。我把一辆遥控汽车变成了一只长着清洁管道的粉红色卷尾巴的猪，另一辆变成了一张戴着眼镜、一头漂金色头发的自画像。我停下来，差一点给我的摩托车涂上一层完整的三力主题油漆。

Labo 模型本身就是低调的奇迹。标签完全适合插槽，直观的图案有助于引导什么去哪里，平板纸板折叠成管和盒子，一起滑动，Joy-Cons 舒适地躺在纸板支架上。任何有模型制作经验的人都会欣赏这些工具的构思有多好；比宜家还要好。只有当你制作这些模型时，你才能完全欣赏它们:纸板钓鱼竿看起来很酷，但当你自己组装它，组装伸缩杆，在卷轴上缠绕橡皮筋，插入纸板标签，这样它就会发出令人满意的噼啪声，你就会完全欣赏它的工作原理。交换机的部件很好地集成到构建中。例如，开关屏幕可以兼作你放鱼竿的水体。该系统的右侧 Joy-Con 嵌入钓竿的手柄，利用其传感器来检测钓竿的倾斜度；左边跟踪卷轴的旋转。

这是任天堂作为疯狂的发明家，作为圣诞老人的精灵，为玩具提出疯狂的想法，并制作原型，直到它们完美。

这第一批套件中最突出的型号是钢琴:虽然你被限制在一个八度，但不仅琴键可以工作，而且你可以插入不同的旋钮来添加混响和效果。猫叫效应是最受欢迎的。录音室模式甚至可以让你组合简单的作品:你可以像指挥一样挥动 Joy-Con 来设定节奏，在琴键上演奏一首曲子，甚至可以使用穿孔卡绘制出一个鼓的节奏，并将其插入钢琴的顶部。穿孔卡片的鼓声震撼了我的心灵，并展示了一些意想不到的东西:每当你认为你已经达到了一个模型及其软件的极限，Labo 就会稍微更进一步。

遥控汽车在一个平坦的表面上嗡嗡作响，由欢乐一族的振动控制。您可以试验频率来平衡控制或调整速度。由于右侧 Joy-Con 上的红外摄像头，遥控汽车可以意外地看到他们面前的东西:你可以让他们跟随周围的物体或为他们发明游戏，如在黑暗的房间里设置障碍课程。将武器或挖掘机附件或又大又尖的牙齿安装到汽车上，就像剪下一些纸板并粘贴上去一样简单，从而实现机器人战争式的战斗。

钓鱼竿和摩托车模型更依赖于它们所打包的软件。摩托车解锁了非常简单的赛道比赛，你可以以不同的速度行驶，钓鱼游戏简单但有趣，你的线沉得越深，难度就越大。这所房子变成了一个疯狂的玩偶之家，不同的按钮和转盘附在它的两侧，成为开关屏幕上的水龙头、微波炉或电灯开关。

没有一款任天堂 Labo 游戏有如此长的寿命。如果你想去兔子洞，你可以进入玩具反斗城的车库，开始发明你自己的东西，摆弄玩具反斗城的功能。我想大多数玩家会非常满意制作和玩盒子里的东西，但任天堂制作的视频展示了可以实现的诱人一瞥:纸板吉他和鼓，简单的枪支游戏。几个月后，我毫不怀疑我们会看到来自创客社区的一些不可思议的作品。

Labo 并没有试图隐藏这些模型的魔力。探索版块对一切事物都有不强烈但对儿童友好的解释，从欢乐合唱团如何工作到加速度计做什么到基本编程:一个好奇的孩子可以根据他们的好奇心深入其中。(“来到这里，你不需要变得聪明，”其中一个教程写道:“是你在这里学到的东西让你变得聪明。”)信息通过三个专门从事编程、纸板定制和玩耍的卡通人物呈现，反映了 Labo 的“玩耍、创造、发现”元素。还有一些 Switch 和任天堂的琐事藏在那里:我了解到 Joy-Con 中的快乐有双重含义:“联合控制器”，用于加入事物。

一岁时，我自己的儿子太小，不理解电子游戏或艺术和手工艺的吸引力，所以我借了一些大孩子一个下午，看看他们对 Labo 的反应。自然地，他们比成年人需要更多的时间来建造模型(尤其是一个有着令人尴尬的复杂的乐高建筑经验的人)，但是他们后来变得更加专注于定制模型。花了一个小时来完善遥控汽车创作，丹和鲍勃 mk 3。

孩子和成人从 Labo 中得到的东西是不同的，这不仅取决于年龄，还取决于个性。更复杂的模型可能会失去年幼孩子的注意力，但年长的孩子(和父母)将会发现它们是如何工作的。这在很大程度上是一项家庭活动，因为无论每个人最感兴趣的是玩游戏、制作模型、提出新的发明或定制，还是钻研编程原则，每个人都有所贡献。

儿童娱乐的一大谬误是，孩子们需要简单的东西。孩子们喜欢深入复杂的事物，积累他们周围的成年人无法接触到的知识。任天堂一直明白这一点，但 Labo 是这种理解的独特表达，是超越虚拟世界的东西。与其说它是一个游戏，不如说它是一个疯狂的工程工具包，它滋养并奖励了创造力和好奇心。作为一个成年人，你只是随波逐流，但如果你的生活中有富有想象力的孩子，看看他们对这个神奇的玩具工厂的反应将是一大乐趣。
