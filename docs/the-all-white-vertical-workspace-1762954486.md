# 全白色的垂直工作空间

> 原文：<https://lifehacker.com/the-all-white-vertical-workspace-1762954486>

我们已经看到了令人敬畏的单色工作空间，但如果那太麻烦，你仍然可以在像这样的空间中欣赏简单。读者看看下面更多的照片。



这篇文章顶部的照片光线很好。pljohns 实际上有一些更柔和、更蓝的灯光，用于那些需要你工作到晚上的夜晚。你可以看看下面——照片说明说 pljohns 正在使用一台 [飞利浦色相绽放](http://www.amazon.com/Philips-259945-Frustration-Personal-Wireless/dp/B00I12YFP0?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-all-white-vertical-workspace-1762954486&asc_source=&tag=kinjalifehackerlink-20) 来获得蓝色效果，你实际上可以看到光线就在工作区的中心。

所以这一切都很好，但他如何让电缆和底部的一切看起来不像一场噩梦？好吧，看看下面这张照片，是开着闪光灯拍的。桌面正下方有一个小抽屉，用于存放所有技术和设备，包括路由器、USB 集线器、外部硬盘驱动器、电源板、Mac Mini 和其他各种电缆和连接器。

这还没完， [这张照片展示了挂在右上角](https://www.flickr.com/photos/50187068@N04/24447867674/in/pool-lifehacker-workspace-showandtell/)[的甜咖啡海报，这张照片让我们一瞥那个签名足球](https://www.flickr.com/photos/50187068@N04/25078525745/in/pool-lifehacker-workspace-showandtell/) 和它生活的书架。总而言之，它组织得很好，布置得很好，尤其是作为一个 DIY 空间。

如果您有自己的工作空间可以炫耀，请将其添加到我们的 [**Lifehacker 工作空间展示并告诉 Flickr pool**](http://www.flickr.com/groups/lifehacker-workspace-showandtell/pool/) 来与我们分享。确保您包含的任何照片至少为 1280x720。将它们保持在 16:9 也有帮助！包括一些文字，关于你使用的材料，你是如何想出这个设计的，以及任何其他相关的细节。如果你巧妙的组织和良好的设计感吸引了我们的眼球，你可能就是下一个特色工作区！
