# Hue 的最新更新让这款应用终于值得使用了

> 原文：<https://lifehacker.com/hues-latest-update-makes-the-app-finally-worth-using-1826411596>

iOS / Android:飞利浦发布了其 Hue 应用程序的重大更新，实际上使其值得使用。我这么说并没有夸大其词。在今天之前，Hue 应用程序一团糟。感觉好像要花很长时间来加载；说得好听点，用户界面很有挑战性；这让我开始寻找最好的第三方应用程序，我可以用它来控制我花哨的色调灯，而没有官方色调应用程序的头痛和压力。

Watch

随着 Hue 3.0 版本的发布，飞利浦似乎终于愿意围绕智能灯打造一个像样的用户体验了。虽然这个应用程序还有一段路要走，但它现在实际上是可用的，我会给该公司所有应得的荣誉。

### 更快的加载时间

自从我买了第一个智能灯泡以来，我就一直在我的 iPhone 上保留着飞利浦的 Hue 应用程序，不时查看该应用程序，看看飞利浦是否对其功能进行了任何改进。从一开始，我就喜欢这个最新版本的 Hue 加载速度很快。我总觉得旧版本的应用程序在弹出之前有一点延迟，这很烦人，每当我想把灯切换到不同的颜色时，我总是想求助于小工具或智能助手。

### (稍微)更智能的界面

像以前一样，你可以从应用程序的主屏幕上打开和关闭房间的灯，或者只是调节灯的亮度。虽然应用程序中没有指出这一点，但你现在可以点击并按住房间的图标来弹出快速选择窗口。在这里，你可以从你最近使用的四个场景中选择一个，或者你可以稍微灵活一点，选择一个全新的色温(白色)或完全不同的颜色。这个小技巧，加上应用程序更快的加载时间，让你瞬间改变灯光变得非常容易。我喜欢它。

### 更多场景可供选择

点击一个房间，你会在一个更小、更容易导航的网格中看到你创建的所有场景(相对于该应用以前提供的巨大、笨重的圆圈，这是一个很大的改进)。与之前相同的色轮控件也出现了，为您提供了另一种方法来将房间的颜色转换为您想要的任何颜色(通过将所有灯光设置为一种颜色，或者在颜色或温暖轮的不同点放置不同的灯光)。每当你分离或组合灯光时，该应用程序都会振动，让你知道你已经做了一些事情——这是一种很好的触摸。

点击以创建一个新场景，然后点击底部的相机图标，您将能够从许多新的预设场景(总共 30 个)中进行选择。它们很可爱，虽然我希望它们生活在应用程序的一部分，在那里你可以点击你创建的场景，因为这将节省你寻找它们的一点挖掘。像 Hue 应用程序的以前版本一样，如果你想让你的房间看起来像一只巨大的 Pusheen 猫[，你也可以让应用程序用你拍摄的任何照片的颜色创建一个场景。](https://www.instagram.com/p/Biu6gPRDyeQ/)

### Hue 的点击-保持技巧也适用于单个灯

如果你回到主“房间视图”——因为没有更好的方式来描述它——你可以点击看起来像项目符号列表的东西来调出你为那个房间设置的所有单独的灯。然后，你可以点击并按住每盏灯，调出和以前一样的快速选择菜单，这使得你可以更快地以任何方式定制你的灯。

最重要的是，我对这个和前面提到的色轮/暖轮选择印象最深。无论你怎么做，在 Hue 应用程序中设置自定义照明配置都不会花费你任何时间，这将使你更容易为下一场派对或 [小型桌面音乐会](https://www.npr.org/2016/02/16/466945874/ben-folds-tiny-desk-concert) 获得情绪照明。

### 下一步是什么？

我认为飞利浦在进入程序、探索或设置标签之前就用完了颜料，因为这些标签与之前的版本没有变化。我很好奇 Hue 是否会解决这些问题，以及该应用程序有些繁琐的“你登录了吗？你已经登录了。所以，现在你没有登录”的方法。(当我试图远程管理我的灯或安装它的一个“实验室实验”时，Hue 有时会让我感到悲伤。)

我也期待着有一天 Hue 应用程序也允许我为灯光设置*动画*。我想让我的房间看起来像一个闪烁的壁炉，一场雷雨，或者一个(发光的) [无生命的碳棒](https://www.youtube.com/watch?v=T8lUZEenFfk) 。是时候了，飞利浦。我已经掌握了静态场景；让我设计一个灯光秀。

除了我个人的愿望清单，如果你是一个定期的光特纳或特纳提供者，你绝对没有理由不抓住更新的色调应用程序。如果飞利浦过去对 Hue 的不满已经让你永远远离了这个应用，我建议伸出橄榄枝，看看这个最新版本。希望飞利浦不要花几年时间来为 4.0 版本开发一些新功能。
