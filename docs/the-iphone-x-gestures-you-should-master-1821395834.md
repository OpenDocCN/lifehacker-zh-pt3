# 你应该掌握的 iPhone X 手势

> 原文：<https://lifehacker.com/the-iphone-x-gestures-you-should-master-1821395834>

对一些人来说，苹果新 iPhone X 最大的变化不是更大的屏幕、奇怪的凹槽或身份验证 Face ID 理工大学。这是 Home 键的消失，自 iPhone 2007 年首次亮相以来，这一标志性的硬件出现在每个 iOS 设备上。随着按钮的消失，苹果正在利用额外的空间来改变你与 iPhone 的交互方式，改变你已经熟悉的手势。有些变化是更好的，而有些似乎执行得很差。以下是如何通过正确的滑动、点击和手势来导航您的新设备。



### 你的分开展示

把你的 iPhone X 的 5.8 英寸显示屏想象成分成四个部分。底部是你的四个固定应用程序；中间部分，您可以搜索和访问小部件；左上角部分，您可以在其中找到您的通知；右上方是控制中心。

### 你的新主页按钮

iPhone X 的 home 键已经被屏幕底部的黑色 home 栏取代。该条可以根据显示器的方向或您所在的应用程序移动。在横向模式下观看 YouTube 视频或玩迷你地铁之类的游戏会将您的家庭酒吧移到横向显示器的底部。若要解锁您的 iPhone X，或从您所在的任何应用程序返回主屏幕，请从黑色条向上滑动一两英寸。

从主工具栏向上滑动并暂停在显示屏中间，将显示您最近打开的应用程序。将手指放在一个应用上进入编辑模式，你可以向上滑动或点击“减号”来强制关闭应用。

如果你已经刷走了一个应用程序，但需要仔细检查一些东西，或者想重新访问你以前使用的应用程序，你不需要再次搜索图标。只需在屏幕底部从右向左滑动，即可重新访问之前打开的应用程序(从左向右滑动会将您带到最近的应用程序，但不会将您带回主屏幕)。

### 控制中心更难使用

最初，控制中心位于 iPhone 和 iPad 屏幕的底部。然而，要访问 iPhone x 上的控制中心，你需要从 iPhone X 烦人的切口右上方向下滑动。

### 轻松检查通知

若要查看您错过的任何通知，如提醒和未查看的文本，请从屏幕顶部向下滑动(最左侧中间的任何位置)。如果你在手机的锁定屏幕上，你可以从屏幕中间向上滑动来查看最近的通知(只要你的设置允许你在 iPhone X 解锁时看到通知)。

你可以强制触摸你的通知旁边的小 X，一次清除所有的通知，这样你就不用一个一个地处理它们了。

### 搜索你的手机

您可以使用内建的搜索功能来搜索 iPhone 的联系人、应用程序、文本和其他信息列表。搜索功能不仅可以让你在手机或网络上轻松找到你想要的东西，还可以用来整理你的主屏幕，消除干扰，同时迫使你更有意识地搜索你需要的应用程序，而不是你心不在焉地浏览的应用程序。

从显示屏中间向下滑动以激活搜索功能。它会向您显示推荐应用程序的列表，并让您在应用程序中搜索提醒、附件和信息。你甚至可以在 Omnifocus 这样的任务管理应用中使用它来查看你的日程安排。

### 组织应用程序

按住应用程序(不激活 Force Touch 选项)将允许你移动应用程序，并根据你的喜好重新排列它们。如果您有一组想要从屏幕的最后一页移到第一页的应用程序，或者有一组想要添加到文件夹的相关应用程序，您可以一次选择并移动多个应用程序，省去了逐个拖移它们的麻烦。

### 能达到性

iPhone X 的显示屏为 5.8 英寸。它很漂亮，但很高，如果你有作家般纤细的手指，接触访问控制中心和其他通知可能会很麻烦。

你可以启用小手友好的可达性模式，以便更容易地获得那些最顶端的应用程序和服务。要启用，请点击设置>通用>辅助功能。向下滚动到交互部分并打开可达性。要真正使用该功能，请从屏幕底部向下滑动(仅在主工具栏上方半英寸处)。它会把你的显示器放在你手指够得着的地方。你可以通过从右侧最上面的应用程序的右上方向下滑动来访问控制中心，或者，如果你在应用程序中，向下滑动电池和 Wi-Fi 图标所在的位置。

它当然不是完美的，重新安排这些交互发生的地方将是一个受欢迎的改进——但现在，主页按钮正在消失，手势将会留下来。
