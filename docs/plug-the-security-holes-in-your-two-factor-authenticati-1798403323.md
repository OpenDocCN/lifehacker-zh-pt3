# 堵住双因素身份认证的安全漏洞

> 原文：<https://lifehacker.com/plug-the-security-holes-in-your-two-factor-authenticati-1798403323>

周二，Techcrunch 作家约翰·比格斯 [的电话号码被一名黑客](https://techcrunch.com/2017/08/23/i-was-hacked/?ncid=rss) 窃取，该黑客控制了比格斯的 T-Mobile SIM 卡，授权他使用比格斯的电话号码来验证他的身份。Biggs 在他的帐户上正确地使用了基于 SMS 的双因素身份验证，但是忘记了为他的无线运营商帐户添加额外的安全层。攻击他的人继续将他锁定在账户之外，并试图以比特币索要赎金。



比格斯最终能够恢复他的账户，重新控制他的电话号码，但你可以把他晚上的头痛和密码重置变成对你自己的教育，并学习如何阻止类似的事件发生在你身上。

### **将 PIN 码放入您的手机帐户**

确保没有人能控制你手机的无线帐户的最简单的方法是添加安全 PIN 或密码。这就像打电话给你的手机运营商并要求启用 PIN 保护一样简单(这是免费的)，或者登录你的在线帐户并访问你的安全设置。

这不是你可能用来解锁智能手机的 PIN 码，而是你在与运营商打交道时需要输入或说出的数字或密码。如果您正在与客户支持代表通话，如果不提供 PIN 或密码，您将无法对您的帐户进行任何更改。您可以通过拨打运营商的电话或使用有效身份证明访问零售店来设置您的 PIN。

不记得密码了？像[AT&](https://www.att.com/esupport/article.html#!/wireless/KM1051385)和 [T-Mobile](https://support.t-mobile.com/docs/DOC-5448) 这样的运营商会让你通过电话或在线重置 PIN。每个主要运营商都会让你带着有效的身份证明走进零售店，并以这种方式更新你的 PIN。当涉及到 PIN 码本身时，一定要避免简单的 PIN 码，如“1234”或与你的生日有关的 PIN 码，因为这些可能会被黑客在你的社交媒体档案中窥探可识别信息而猜到。

### **使用更好的双因素认证服务**

基于短信的身份认证，通过向你发送进入你账户所需的随机密码来验证你的身份，是更安全的数字生活的良好开端，但如果你想确保没有安全漏洞，你必须再提高一个档次 。通常，只有在没有其他双因素身份认证过程可用时，才应该使用它。

请记住，您的手机可能不是接收认证信息的唯一设备，尤其是当您的信息在多个设备(如平板电脑或电脑)之间同步时。它们可以被发送到其他在线信息服务，如谷歌语音或 Skype，这些服务可以从智能手机以外的地方访问。布里格斯发现，如果没有适当的安全协议，它也容易受到基于运营商的 SIM 卡传输的影响。

像 Authy 或 Google Authenticator 这样的双因素认证应用程序要安全得多，并且不涉及电子邮件地址或短信，给攻击者提供了更少的切入点。设置比输入发送到手机的数字要复杂一些，需要你在输入周期性随机数字串的同时准备好身份验证设备，无论是智能手机还是平板电脑。

### **使用密码管理器**

不要认为增加更多的安全层意味着你必须记住每一个新的 PIN、密码或其他密码。当你设置额外的安全检查点时，在 [中输入你选择的密码管理器](http://lifehacker.com/the-five-best-password-managers-5529133) 中的信息。你可以用它来存储备份代码，客户支持号码，或运营商专用的电子邮件地址，确保它远离黑客的窥探，只有你可以访问。

### **将一次性密码放在手边**

设置像 Google Authenticator 这样的双因素认证应用程序通常需要保存一个备份密码，以防手机丢失或被盗。谷歌建议你把它们打印出来，存放在一个安全的地方。你可以把它们放在一个文件夹里，藏在家里的某个地方，或者放在你的密码管理器里，以便于访问。无论如何，有一个备份计划以防你原来的备份计划失败，这是一个很好的方法来保护你自己和你的身份免受恶意个人的攻击。
