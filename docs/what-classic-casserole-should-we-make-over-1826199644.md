# 我们应该做什么经典的砂锅菜？

> 原文:[https://life hacker . com/what-classic-sarbang-should-we-make-over-1826199644](https://lifehacker.com/what-classic-casserole-should-we-make-over-1826199644)

大家好，欢迎回到 [*将它砂锅*](https://lifehacker.com/tag/will-it-casserole) ，在这个栏目里我们看到哪些饭菜、菜肴、菜系可以重新想象，改造成砂锅。

在这个系列中，我们通常会采用一些不是砂锅的东西——像上周的全英文——并重新排列其成分，使其成为一种多层的热菜，大于其各部分的总和。这很有趣，但我想尝试一些不同的东西。我想用现有的砂锅菜图标，给它们一点改头换面，而不是用*不是*砂锅菜的东西。下面我有几个候选人，但显然我想听听你对我们应该更新哪些砂锅以及如何更新的想法。以下是我的一些想法:

*   **汉堡砂锅:** [我祖母的烹饪书](https://www.instagram.com/p/BZkTSc9AyB-/?taken-by=clairelizlower)——其中也有一种砂锅叫做“多利托派”——有这样的记载，里面有碎牛肉、米饭、各种蔬菜(洋葱、芹菜、甜椒)和奶油蘑菇汤。就升级而言，汤是一个很好的起点，我想在里面放一些不同的蔬菜，还有大蒜和任何香料。
*   花椰菜和奶酪砂锅:再说一遍，大多数食谱都含有浓缩汤，但我认为我们最好吃蘑菇沙锅。我还想在菜里放一些韭葱或青葱，如果我能在这个季节这么早找到的话，再放一些花椰菜。
*   金枪鱼砂锅面:这道菜有点难。我不认为使用新鲜的金枪鱼会使它变得更好，但我打赌使用真正保存良好的金枪鱼会。我还认为调味是关键。相对于一些油腻和过于奶油化的东西，一种以橄榄油为基础的，带有柑橘味的情况可能对我们更好。

这些都只是跳跃点。如果你有任何经典的砂锅菜或热菜，你希望看到它们被重新制作，或者有任何关于如何改进上述任何一个方面的好主意，请在下面的评论中留下它们。
