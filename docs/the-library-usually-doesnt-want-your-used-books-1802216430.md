# 图书馆(通常)不要你用过的书

> 原文：<https://lifehacker.com/the-library-usually-doesnt-want-your-used-books-1802216430>

你以为图书馆会想要你的旧书！但是除了偶尔的旧书出售，图书馆通常只从特定的卖主那里买书。把你的旧书扔给他们，你实际上是在给志愿者创造工作，让事情变得更糟。

Watch

我们采访了图书管理员斯蒂芬妮·安德森(Stephanie Anderson)，她是纽约公共图书馆和布鲁克林公共图书馆共享技术服务合作组织 [BookOps](https://sites.google.com/a/nypl.org/bookops/home) 的助理馆长。她解释了把你的书扔在图书馆有什么不好，以及为什么他们不想要它们。

### 先问

“几乎所有的图书馆都有明确的捐赠政策，”安德森说。所以，在你把书推过来之前，先查一下。“很可能在他们的网站上，如果没有，你可以在图书馆的任何一个柜台询问，或者打电话给他们。”问他们谁想要你的旧书，他们通常会有一个现成的答案。

如果图书馆想要你的书，你可以不看这篇文章！恭喜你！

### 大多数图书馆不能使用你的书

安德森知道潜在的捐赠者都是出于好意。但是，她说，“当你坚持把书捐赠给一个不接受它们的图书馆，或者坚持捐赠一个图书馆不能使用的书，这是对你试图支持的机构的不尊重。”

安德森说:“你不会相信有多少人在得知我们的捐赠政策后感到愤怒，并与图书管理员发生争执。”。想象一下那种令人尴尬的人，因为他们没有做调查而大发雷霆。千万不要成为那样的人！

### 为什么他们不想要你的书

安德森说，简而言之，书并不都是一样的，你要扔掉的那些书并不是图书馆需要的。甚至新的也没有:

> 在最好的情况下，我们可以对捐赠的书籍做两件事之一:将其添加到收藏中，或者将其放在图书销售中。许多新手捐书者希望我们做前者，但我们几乎从不这样做，因为在捐书的时候，我们需要这本书的可能性极小。为了明智地使用图书预算，我们做了很多规划，而这些规划不能依赖于图书捐赠。

相反，图书馆从供应商那里购买。这是唯一可靠地匹配供求关系的方法，也是获得绝对没有臭虫或其他隐藏缺陷的新书的唯一方法。购买实体书并不是唯一需要花钱的部分。

### 即使他们用了你的书，也不是免费的

当图书馆增加一本新书到他们的收藏中时，他们不仅仅是打开盒子，把它放到书架上。他们必须添加一个条形码，脊柱标签，并层压它或添加一个清晰的聚酯薄膜覆盖，使其持续更长时间。他们必须分配条形码并将图书添加到目录中。“这些都要花费图书馆的钱，处理这本书的工作人员的时间也是如此，”安德森说。"图书馆从来没有足够的工作人员时间。"

图书馆也需要空间来摆放每本新书。因为他们已经在做他们的工作，购买他们需要的所有书籍，你不需要的书只是在占用空间，直到工作人员花更多的时间回收它。

### 当你可以捐书的时候

当然，这并不适用于图书销售，图书馆利用图书销售来筹集资金和处置自己多余的图书。(他们不会总是需要 23 份去年的《约翰·格里森姆》。)不过，*查看捐赠政策*，该政策可能会禁止没有人会购买的教科书、杂志和其他材料。如果你“捐赠”这些，同样，你是在为图书馆创作作品。

如上所述，还有其他例外，这是你在谷歌上搜索图书馆的捐赠政策时了解到的！例如，布鲁克林公共图书馆正在其中心位置运行一个试点项目，接受 [新的和不常用的书籍](https://www.bklynlibrary.org/support/where-donate-books) 。

### 改捐什么

你还能帮忙！大多数图书馆会很乐意接受你的(免税)捐款。“我们中的许多人依靠这些捐赠来保持我们的收藏处于良好状态，”安德森说，“每一美元都有所帮助。”

你也可以做志愿者，这将帮助你了解图书馆是如何工作的，并欣赏他们在幕后做的许多事情(以及他们在书籍之外提供的许多服务)。记住，大多数图书馆人手不足，你可以让受过高等教育的图书馆员从事更多的技术工作，这样他们会感谢你的帮助。

### 这些书该怎么处理呢

嗯，图书馆可能会告诉你去哪里捐赠。你可以通过把它们卖给二手书店来赚钱(或存储信用)。或者，这可能会让你震惊，*你被允许扔掉它们*。“是书圈生活的一部分！”安德森说。

可能感觉怪怪的！书是有魔力的。但是它们也被大量生产，现在它们也被数字化了。如果你的书很稀有，那么它在亚马逊和 [AbeBooks](https://www.abebooks.com/) 上卖得很好，这是你应该买的地方。否则不要担心，在某个地方有足够的副本给任何想看的人。如果你的图书馆没有这本书，建议他们买一本。老实说，他们宁愿从供应商那里买一个新的。

安德森指出，在这个国家的几乎所有地方，书籍都是可回收的。因此，你的旧书将获得第二次生命，成为你新书的亚马逊盒子。
