# 如何在 Waze 上设置免提控制和其他新功能

> 原文:[https://life hacker . com/waze-adds-hands-free-controls-motorcycle-directions-an-1820846212](https://lifehacker.com/waze-adds-hands-free-controls-motorcycle-directions-an-1820846212)

周三，宣布了一个 [巨大的新更新](https://blog.waze.com/2017/11/tis-season-to-ride-your-own-waze.html) ，为 Android 和 iOS 应用程序带来了三个最受欢迎的功能。从现在开始，你可以用你的声音控制 [Waze](https://lifehacker.com/turn-by-turn-navigation-showdown-google-maps-vs-waze-1761550298) ，获得摩托车特定的方向，并将拼车车道纳入你的通勤。

以下是如何设置 Waze 的所有新功能，以便您可以尽快开始使用它们。

### 免提控制

要使用 Waze 设置语音激活，请先打开应用程序并进入设置。选择“声音与语音”，然后选择“与 Waze 交谈”最后，打开“收听 OK Waze”下一次你说“OK Waze”时，应用程序会响应，这样你就再也不用把手从方向盘上拿开了。

目前，这项功能仅限于美国、英国、加拿大、澳大利亚、南非、爱尔兰和新西兰的英语使用者。更多的语言选项应该“很快”就会出现

### 摩托车导航

汽车的最佳路线并不总是摩托车的最佳选择。要利用这一功能，进入设置，选择“车辆类型”，并选择摩托车(其他选项是私人或出租车)。如果你以后想换回汽车，只要做同样的事情，但选择私人代替。

### 在您的路线中添加拼车车道

最后，如果你通常开车带着一个以上的乘客，Waze 现在可以帮助你找到高占用率的车辆(又名拼车)车道，以节省通勤时间。若要打开此功能，请前往“设置”，然后前往“导航”，然后选择“添加收费/HOV”您将看到您所在区域的拼车车道列表，您可以选择想要使用的车道。

这项功能首先在加拿大和美国的部分地区提供，以后还会有更多。以下是您现在可以向 Waze 添加拼车车道的地区列表:
