# Windows 10 的 11 月大更新有什么新内容

> 原文：<https://lifehacker.com/whats-new-in-windows-10s-big-november-update-1742108755>

Windows 10 今天获得了第一次重大更新，提供了许多你想要的功能——比如彩色标题栏、开始菜单的修复，以及(终于！)激活 Windows 10 许可证的更好方法。

Watch

### 你终于可以用你的 Windows 7 和 8 键激活了

现在，当您安装 Windows 10 时，您可以使用 Windows 7、8 或 8.1 产品密钥激活它。没有这种“ [安装 Windows 7，然后更新，然后从头重新安装](http://lifehacker.com/how-to-do-a-clean-install-of-windows-10-1720775893) ”的生意只是为了做一个 Windows 10 的干净安装。现在一切都应该更流畅了，虽然这不会影响你已经安装了 Windows 10 的电脑(因为它们将由硬件自动激活)，但你尚未升级的任何电脑都应该更简单。

### 开始菜单有一些新的改进

开始菜单，虽然不会取悦任何 Windows 7 的粉丝，但在这次更新中得到了一些改进。您现在可以将列拉伸到四个平铺窗口的宽度(设置>个性化>开始>显示更多平铺窗口)，开始菜单作为一个整体可以包含超过 512 个平铺窗口，准确地说是 2048 个。上下文菜单也有了更一致的外观(尽管它们仍然和桌面上的不一致)。

Windows 10 的“开始”菜单一直都有“推荐应用”，它会不时地推荐你安装。幸运的是，这次更新带来了一个简单的设置来关闭它:只需进入设置>个性化>开始，然后将“偶尔在开始中显示建议”切换到关闭状态。

### **彩色标题栏又回来了**

有点傻的是，Windows 10 允许你自定义开始菜单和任务栏的颜色，但不能自定义窗口的标题栏——无论如何标题栏都是白色的。现在，如果你进入“设置”>“个性化”>“颜色”并选中“在开始、任务栏、操作中心和标题栏显示颜色”选项，你的颜色自定义也会显示在你所有的桌面窗口上。

### Windows 有一个内置的“查找我的设备”

我们一直建议 [安装一个像 Prey](http://lifehacker.com/how-to-track-and-potentially-recover-your-stolen-lapt-5643460) 这样的应用程序来找到并跟踪你被盗的笔记本电脑，虽然它仍然是一个很棒的功能丰富的选项，但你不再需要这样做了！Windows 内置了一个新的“查找我的设备”功能，如果你的笔记本电脑或平板电脑丢失或被盗，它将跟踪你的笔记本电脑或平板电脑。只需进入设置>更新&安全>找到我的设备并点击“定期保存我的设备的位置”即可启用。

### Cortana 学会了一些新技巧

Cortana 粉丝会喜欢这个更新，因为个人助理甚至更好地知道你想要什么。Cortana(现已在加拿大、日本、澳大利亚和印度推出)现在可以理解手写笔记，从你的电子邮件中跟踪电影和其他门票，提醒你在 Edge 中访问的网站的优惠券，并将你的消息和通话记录与 Windows phone 同步，并在你的桌面上通知你未接来电(你可以在 Cortana 的设置中找到这些)。Cortana 甚至可以在知道您不在办公室时让您的电脑进入睡眠状态(设置>系统>电源和睡眠)。

而且，最重要的是，Cortana 最终可以与本地帐户一起工作，所以你不需要一个微软帐户来获得好处。

### 其他小改进

这些都是很大的变化，但你也会发现一些不错的新变化:

*   平板电脑模式现在的工作方式更像 Windows 8:在任务视图中，你可以通过将应用程序的缩略图拖到屏幕底部来最小化应用程序，当你在一个屏幕上拆分应用程序时，多任务处理现在可以同时调整两个应用程序的大小。
*   在“设置”>“设备”>“打印机和扫描仪”中，您现在可以选择让 Windows 管理您的默认打印机。启用此选项后，您的默认打印机将是您最后使用的打印机。
*   你可以在锁屏时关闭默认的 Windows 背景，只使用素色。转到设置>个性化>锁屏更改此设置。
*   Microsoft Edge 现在可以在 Windows 10 设备之间同步您的收藏夹和阅读列表，并在您将鼠标悬停在标签上时为您提供预览。
*   Skype 现在内置在 Windows 10 中，有三个应用程序:Skype 视频、消息和电话。不过，您仍然可以下载普通的 Skype 应用程序。
*   平板电脑和其他低存储设备可以让你在任何地方安装新的应用程序，比如 SD 卡。您可以在设置>系统>存储中找到此设置。

这次更新有很多东西在进行，但这些是大多数人关心的主要和次要的变化。在启动时间和其他任务方面也有很多性能改进，所以您应该会注意到一些更快的速度。

要下载更新，请转至“设置”>“更新与安全”,然后检查更新。然而，请注意，更新似乎推出得很慢——我的台式机立即获得了更新，但我的笔记本电脑仍显示为“最新”,因此您可能需要耐心。你可以在下面的链接中读到更多关于更新的内容。
