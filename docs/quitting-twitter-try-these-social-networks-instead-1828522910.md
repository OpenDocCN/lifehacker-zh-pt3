# 退出 Twitter？试试这些社交网络吧

> 原文：<https://lifehacker.com/quitting-twitter-try-these-social-networks-instead-1828522910>

现在推特上最热门的话题之一似乎是要不要离开推特。滥用在“微博”平台上猖獗，每天我都看到有人抱怨其版主没有控制住骚扰和不良行为。作为一个多年来主要写视频游戏的人，我看到我的许多同龄人被误导的游戏玩家的暴民超越，几乎没有保护自己的手段。我也看到(或者至少读到)保守的用户相信 [他们的帖子被平台](https://www.washingtonpost.com/news/politics/wp/2018/07/26/trump-shadow-ban-tweet-a-f-a-q/?utm_term=.227629064d9e) 压制。



该公司的不作为在上周达到了顶点，它因发布仇恨言论和煽动暴力 而将散布错误信息的 Infowars 及其创始人亚历克斯·琼斯 [停职七天，这是由用户的强烈抗议引发的，他们要求该服务加入脸书、YouTube 和其他最近禁止 Infowars 进入其网络角落的数字媒体平台。Twitter 区分了完全禁止和什么都不做之间的区别，这导致了对该平台的更多绝望。没人开心。一些另类右翼和新纳粹类型的人已经通过迁移到 Gab，](https://gizmodo.com/twitter-suspends-alex-jones-for-7-days-leaves-infowars-1828351734) [这是一种迎合持有令人憎恶观点的用户的新服务，](https://www.theverge.com/2018/8/9/17671188/microsoft-gab-hate-speech-hosting-ban-deplatform) 我们不会在这里推荐它，原因很明显。

如果你正考虑离开 Twitter，去寻找一些不那么愤怒或不那么有害的东西，我们已经整理了一个列表，在那里你可能会以某种方式、形状或形式找到类似的经历。没有任何替代物能把 Twitter 做得这么好:一些模仿了 Twitter 的所有特征——严格的字符限制、可滚动的 feed、“收藏”等等。—但不要提供同样的社交范围。其他人有很深的，培养良好的社区，但不要真的让你读和写你的想法一样。没有什么可以完全替代，但是如果你准备跳槽，想要一些新的东西来填补空缺，以下是你的最佳选择:

### [](https://joinmastodon.org/)

****字符限制:** 250** 

**如果你喜欢使用 Twitter，但只是不喜欢参与社区的某一部分，对大多数人来说，乳齿象实际上是最好的/唯一真正的 Twitter 替代品。乳齿象是一个去中心化、开源的 Twitter 风格的平台。用户可以创建具有不同隐私级别的独立托管和管理的社区。根据服务器的不同，你可以发出“嘟嘟声”和/或跨服务器跟踪用户，也可以完全隔离。它可以像 Twitter 一样，也可以像一个私人聊天室一样。**

**乳齿象需要比传统社交网络多一点悟性。你得加入一个服务器，也就是说除非你有专门拉你去的人，否则你得挑一个信息不多的。也没有官方的乳齿象手机应用程序，所以你也必须从中选择一个。这些决定并不难，但没有容易的入口，所以可能需要时间来找到适合你的社区和设置。**

**作为一个主要使用 Twitter 作为联系读者和同事的专业工具的人，我的问题是，“如果你只打算与你认识的人交流，那么使用 Twitter 有什么意义？”乳齿象的标志性品质，即保持私密或至少适度的能力，限制了它的效用。除了势不可挡的迁移，在乳齿象上发布一条“病毒式”消息永远不会像一条聪明、愚蠢或有影响力的推文那样有影响力。**

**但是对于那些希望将整个社区移出公众视线，或者希望在一个意识形态不那么激进的空间结交新的在线朋友的群体来说，乳齿象拥有所有必要的工具来过你的 Twitter 生活，但是没有巨魔。**

### **[](https://www.tumblr.com/)**

******字符限制:**不适用**** 

****在其他成熟的社交网络中，Tumblr 似乎最适合扮演 Twitter 为许多人扮演的角色——在这个社区中，人们可以与同样感兴趣的人分享他们对自己感兴趣的话题的想法。****

****Tumblr 的存在并不是为了成为“Twitter 的替代品”，所以与这个列表中的其他条目不同，它看起来并不完全一样，也不符合相同的约束。文本帖子没有字符限制，其社区的更大一部分仅仅是为了管理和“转发”网站上已经存在的其他内容。也就是说，没有人阻止你用浅显易懂的语言发表你的想法，不一定少于 280 个字符。****

****Tumblr 提供的东西，是你从一个试图直接模仿 Twitter 的服务中得不到的，是任何主题的深厚、成熟社区的源泉。搜索任何话题——从“政治”这样的广泛话题，到关于你最喜欢的星球的粉丝艺术这样的具体话题，你都会找到一个兔子洞。(抱歉)。无论你喜欢谈论什么，Tumblr 上都有人写东西，分享 gif，或者就你感兴趣的任何话题互相交流。此外，与 Twitter 不同，这些社区都是共存的，但非常孤立，因此虽然你不一定能免受网络钓鱼者的攻击，但你不太可能会有交集。****

### ****[](https://aminoapps.com/)****

********字符限制:**不适用****** 

****如果你主要查看 Twitter 来阅读和撰写关于你的一个或几个非常具体的主题，Amino 是一个由深度粉丝社区组成的移动专用网络，让你加入、发布和与人聊天，谈论你的激情。Amino 感觉很像 Tumblr，形式化了创建离散社区的过程。不是简单地创建一个博客，写些什么，而是引导你加入 Aminos，围绕特定主题的社区，在那里你可以写博客文章，并在聊天室与粉丝们交谈。****

****在这种情况下，Amino 是非常面向目标的。你必须知道你想谈论什么，并对那件事充满热情。Amino 绝对不是一个了解世界上正在发生什么的地方，但它可能是一个很好的地方，可以让你说出关于昨晚《河谷镇》的一切。****

### ****[**Plurk**](https://www.plurk.com/)T5】****

******字符限制:** 360**** 

****Twitter 自 2006 年就已经存在，所以大多数试图竞争和/或复制它的创业公司已经来了又走了。Plurk 是 Twitter 的早期竞争对手之一，也是仅存的长期竞争对手之一。****

****坦白说，普鲁克是一个鬼城。该平台自 2008 年推出以来，感觉就像没人碰过一样。在游戏化的“karma”系统(你可以通过经常发帖获得积分)和对柔软、圆润的帖子气泡的强调之间，我在 Plurk 周围漫游，不禁回忆起早期的 Twitter 和 Foursquare(我真的对 Foursquare 着迷了一分钟。请不要评价我)。****

****虽然我感觉这种情况并没有得到很好的维持——该网站提供付费移除广告的选项，但我没有找到一个——有 iOS 和 Android 应用程序，至少有几个主题似乎经常出现在关于迷因、游戏和动漫的对话中(尽管其中很多似乎是日本的)。****

****很难看到使用 Plurk 作为 Twitter 的合法替代品——你肯定不能用它来获得新闻或评论，但这就是 Twitter 上滋生大量有毒流量的原因，所以也许独自思考并不是那么糟糕？****
