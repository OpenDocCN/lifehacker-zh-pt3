# 内向者培养外向者指南

> 原文：<https://lifehacker.com/the-introvert-s-guide-to-raising-an-extrovert-1826489237>

作为一个与内向者结婚的内向者，我认为我们的儿子当然会是一个内向者。



作为一个蹒跚学步的孩子，瑞安会在操场边上站一个多小时，然后才会加入进来。作为一个学龄前儿童，他会在放学时间哭泣。我丈夫和我会看着对方，眼里闪烁着会心的光芒，我们之间有句话在说:“这就是我们的小内向者！”

然而，随着瑞安年龄的增长，他的个性真正开始发光，我们意识到了一些事情。他不害羞——他很谨慎。他不担心其他孩子或群体环境；他只是在过渡中挣扎。其他人并没有像我在一次晚宴或假日聚会后发现自己一样耗尽他的精力。相反，一旦他在某种情况下变得舒适，他似乎就能从他人的能量中茁壮成长。

现在，将近 8 岁的他——也是我们唯一的孩子——渴望有其他孩子的陪伴。我对他交朋友的轻松和他对维持友谊的重视感到敬畏。他最开心的周末是和他的同学、他的足球队友或有时会去巷子里的房子玩的孩子们在一起。

问题？我。作为他的母亲(或者像我喜欢说的，他的“巡航导演”)，他依靠我来安排这些事情。“打电话给马修的妈妈，”他会说。“告诉她，他可以过来。他可以带上他所有的军人，他的枪和任何他想要的东西。嘿，也许大流士也应该来！”

给马修的妈妈打电话？两年来几乎每天都见到马修的妈妈，但我几乎不认识她了。我只见过 Darius 的妈妈两次，基本上和没见过是一回事。我能说什么呢？为什么我心跳加速？

我永远不会成为那种喋喋不休的家长，在学校接机时设法与所有酷妈妈交朋友，并随意安排即兴郊游。我必须学会如何让 Ryan 保持活跃，并与其他孩子保持接触，而不是让自己太过于突出，以至于我想萎缩和死亡。

我是这样做的:

文件夹中的便条技巧。当我需要与一位从未谋面(或者，咳咳，只见过两次面)的家长进行初次接触时，我会在 Ryan 的学校文件夹中发送一张纸条，上面写着我的电话号码。他把它交给朋友，让他传给父母，压力就消失了。通常是有效的，如果无效，我可以正式说我试过了。

发短信。是内向者最好的朋友。这种方式简短而甜蜜，比打电话不那么尴尬(专业提示:生日派对请柬上的名字和电话号码几乎总是属于母亲的。省省吧！一旦你通过短信回复了一个聚会，你就为将来的游戏约会建立了一个完美的沟通渠道。)

**3)托管。没有人真的想举办一个游戏日，对吧？这也不是我一直以来最喜欢的度过周六的方式，但它很有效。我邀请家长简单地放下孩子(最少闲聊！)，当孩子们玩耍的时候，我会洗衣服或者看书。每个人都赢了。**

团队运动。很多很多的团队运动。瑞安可以同时和一大群孩子互动，我可以坐在一旁和我的丈夫和*交谈，没有其他人*也不会显得粗鲁。

夏令营。很多很多夏令营。这些甚至比团队运动*还要好*，因为我可以把他放下，然后去附近的咖啡店找个安静的角落做点事情。

当所有这些都失败时，我只是祈祷，希望一个外向的人会同情我，和我建立友谊，这样我就不用这么做了。(这至少在一个场合奏效了。)
