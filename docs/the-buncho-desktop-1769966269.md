# 本科桌面

> 原文：<https://lifehacker.com/the-buncho-desktop-1769966269>

我们的朋友和经常的桌面贡献者 Dobbie03 已经从 Windows 切换回 Linux，并且兴致勃勃地——他的新 Arch Linux 桌面看起来很锐利。这是很好的定制，以适应他的工作流程，我们喜欢 Java 麻雀(又名 bunchos。)以下是他如何设置这一切的。

Watch

像往常一样，我们的 Linux 特色桌面需要更多的移动部件，并且不使用像 GeekTool 或 Rainmeter 这样的集中式套件，但您仍然可以通过少量工作获得很好的结果。这是你需要的:

*   卡迪尔的 [壁纸](https://500px.com/photo/15634689/birds-by-kadir-i-)
*   [Covergloobus](https://github.com/aur-archive) (来自 AUR 档案馆，因为它已不在 AUR)和 [这个 Covergloobus 主题](http://dobbie03.deviantart.com/art/Jane-Austen-Covergloobus-526591678) 来自 Dobbie03
*   [Conky](http://conky.sourceforge.net/) ，一款针对 Linux 系统的轻量级系统监视器
*   GTK 主题 中的 [(根据 Dobbie03，在 AUR 也有)](https://www.archlinux.org/packages/community/any/deepin-gtk-theme/)
*   侏儒外壳
*   [Noto Sans 常规](https://www.google.com/get/noto/) 字体
*   [顶点图标](https://github.com/horst3180/vertex-icons) (再次，在 AUR)
*   [Neofetch](https://github.com/dylanaraps/neofetch) 定制终端

有一些移动的部分，你必须调整每一个来看看你是否喜欢它，但这是 Linux 的魅力所在——你可以选择或离开任何你喜欢的元素来看看它是否适合你，然后用你喜欢的任何东西来工作。这是 Dobbie03 桌面的另一张照片，窗口是开着的:
