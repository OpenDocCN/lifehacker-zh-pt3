### 在浏览器中下载并安装 LastPass

LastPass 作为一个扩展存在于你的浏览器中。它几乎适用于所有主流浏览器，包括[Chrome](https://chrome.google.com/webstore/detail/lastpass-free-password-ma/hdokiejnpimakedhajhdlcegeplioahd)[Firefox](https://lastpass.com/lastpassffx/xpi.php)[Safari](https://lastpass.com/download/cdn/lastpass.safariextz)和 [Opera](https://lastpass.com/lpopera.php) 。

1.  前往 [LastPass 下载页面](https://lastpass.com/misc_download2.php) 并为您的首选浏览器安装扩展。
2.  安装完成后，您将在工具栏中看到一个新图标。点按该图标并选择“立即创建帐户”
3.  键入您的电子邮件地址，并创建一个主密码。 [将此密码设置为强密码](http://lifehacker.com/four-methods-to-create-a-secure-password-youll-actually-1601854240) ，这是您将始终用来访问 LastPass 的密码，以及您用它创建和存储的所有密码。别忘了这个——[做了](http://lifehacker.com/what-to-do-if-you-lose-the-master-password-to-your-pass-1686432765) 你就没有太多选择了。

还有一个 [LastPass Mac 应用](https://lastpass.com/misc_download2.php) ，但大多数人会发现浏览器扩展是他们所需要的。如果你打算从你的移动设备访问 LastPass，那就下载适用于[iOS](https://itunes.apple.com/us/app/lastpass-free-password-manager/id324613447?mt=8)或 [Windows Phone](http://social.zune.net/redirect?type=phoneApp&id=9b86eadc-16e8-df11-9264-00237de2db9e) 的应用程序，但如果你想在多个设备之间同步密码，也要准备好注册 [高级订阅](https://lastpass.com/features_joinpremium4.php?a=1) 。

### 保存您访问的网站的登录信息

现在是时候将您的密码保存在 LastPass vault 中了。有几种方法可以做到这一点，但最简单的是像平常一样使用互联网，并在登录每个新网站时保存密码。这样，LastPass 会为您工作。

1.  当你进入一个有登录页面的网站时，输入你的用户名和密码，但不要点击登录按钮。
2.  点按密码栏内的 LastPass 图标，然后点按“存储此站点的凭据”

当你这样做的时候，你的 LastPass 帐户将逐渐用你所有的密码和登录信息填满你的保险箱。现在，当你再次访问这些网站时，LastPass 可以自动为你输入用户名和密码。

如果您过去使用过浏览器的内置密码管理器，或者您使用过其他受支持的密码管理器，如 [1Password](https://1password.com/) ，您可以将所有登录信息直接导入 LastPass。这一过程因你之前使用的管理器而异，但你可以在 [LastPass 的导入密码指南](https://helpdesk.lastpass.com/importing-from-other-password-managers/) 中找到你需要的所有信息。

### 通过安全挑战修复您的弱密码

接下来，让我们修复你一直在使用的那些无用的、容易被破解的密码。随着你在 LastPass 中输入越来越多的密码，你会想进去 [审核这些密码](http://lifehacker.com/how-to-audit-and-update-your-passwords-after-a-service-5712907) 并创建更好的密码。有几种方法可以做到这一点，但在开始时，最简单的方法是使用 LastPass 的内置安全挑战。

1.  在浏览器中，单击“LastPass”图标>我的保管库。
2.  单击安全挑战选项卡。
3.  单击显示我的分数。
4.  出现提示时，输入您的 LastPass 密码。
5.  等待 LastPass 分析你所有的密码。

LastPass 将向您显示所有密码的报告，分为四个不言自明的部分:更改泄露的密码、更改弱密码、更改重复使用的密码和更改旧密码。

点击每个部分以展开它，并查看 LastPass 建议您更改哪些密码。对于许多流行的网站来说， [LastPass 可以自动更改你的密码](https://lifehacker.com/lastpass-can-now-automatically-change-your-passwords-1668860126) ，无需你付出任何努力。只需点击自动更改按钮，LastPass 将在后台自动为该网站创建一个新密码并保存，以便您下次访问时使用。

如果网站不支持自动更改，您需要手动更新您的密码。LastPass 试图尽可能地减少痛苦，但这仍然是一件小事:

1.  点击“Launch Site ”, LastPass 将在新标签页中打开该网站。
2.  使用您的用户名和密码登录，并在该站点的帐户详细信息中找到更改密码部分。
3.  在“新密码”栏中，点按“最后一次通过”图标，然后选择“生成新密码”LastPass 将为网站创建一个新密码。
4.  出现提示时，选择“保存网站”保存您的新密码信息。

取决于你在这里修复多少不同的网站，这可能是一个漫长而繁琐的过程，所以在网飞上放一部电影，并留出一点时间来处理它们。

### 添加表格填写信息，以便在新网站上快速结帐和创建新帐户

除了处理密码，LastPass 还可以安全地存储您的信用卡信息和地址，以帮助您在新商店、零售商或任何网站注册帐户时加快创建新帐户的过程。设置很简单:

1.  单击浏览器中的 LastPass 图标。
2.  选择表格填充>添加表格填充。
3.  在此输入您想要存储的所有信息。

LastPass 有选项卡可以输入各种信息，包括您的姓名、地址和信用卡信息。将这些信息添加到 LastPass 中，这样当您在新网站上创建帐户时，LastPass 可以自动为您填写数据，这样您就不必键入数据。

这是完全可选的，但它是 LastPass 内置的一个有用功能，你的信息会和你的密码及其他数据一起被加密。如果你不舒服或者不想在这里存放信用卡之类的东西，添加你的姓名和地址仍然是值得的，这样你就不必到处重新输入。

### 向您的 LastPass 帐户添加多因素身份验证

正如你所期望的那样，LastPass 是一款旨在让你的数字生活更加安全的工具，它支持。一旦启用，您的主密码不足以登录您的 LastPass 帐户并访问您的保管库。你还需要一个从认证者 处获得的密码 [。这个额外的安全层可以保证你的 LastPass 账户的安全，这也有助于保证你所有密码的安全。设置只需点击几下:](http://lifehacker.com/which-form-of-two-factor-authentication-should-i-use-1784769336)

1.  单击浏览器中的 LastPass 图标，然后选择我的保管库。
2.  单击帐户设置。
3.  单击多因素身份验证选项卡。
4.  单击您要使用的验证器旁边的钢笔图标，并将“启用”下拉菜单设置为“是”。然后按照屏幕上的指示。

这里的过程是不同的，取决于您想要使用哪个验证器。通常，你会将 LastPass 添加到你选择的应用程序中，然后应用程序会给你一个代码，让你在浏览器中进入 LastPass。LastPass 支持 [多个常见的多因子认证 app](https://helpdesk.lastpass.com/multifactor-authentication-options/)，包括[Google Authenticator](https://support.google.com/accounts/answer/1066447?hl=en)。 [LastPass 也有自己的认证器 app](https://lifehacker.com/lastpass-authenticator-simplifies-two-factor-authentica-1765243508) ，里面包含了一个俏皮的 [一键审批系统](http://lifehacker.com/lastpass-authenticator-now-has-a-one-button-approval-op-1785138823) ，简化了登录。如果你还没有使用任何其他支持的认证应用程序，LastPass 是一个很好的开始(即使它只适用于 LastPass。)

就是这样。设置 LastPass 很容易，唯一令人头疼的是可能很长的密码审计过程。值得庆幸的是，这是一次性的事情，一旦你所有的密码都是强有力的，你只需要担心如果一个网站被黑客攻击，你会改变一个。

最后，虽然我们许多人不需要它的高级功能，但值得指出的是，LastPass 还可以与其他人[共享密码，授权爱人](https://helpdesk.lastpass.com/emergency-access/)[紧急访问您的帐户，甚至可以安全地存储您可能不想让其他人看到的笔记](https://helpdesk.lastpass.com/secure-notes/) 。不过现在，继续前进，拍拍自己的背，因为在推迟了这么多年之后，终于设置了一个密码管理器。
