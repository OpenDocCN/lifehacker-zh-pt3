# 为什么你应该在做家务时设置计时器

> 原文：<https://lifehacker.com/why-you-should-set-a-timer-when-you-do-chores-1820577275>

那些在需要做的时候平静而愉快地做家务的人，你现在可以不读了。其他人:啊，我知道，对吧？

Watch

家务很糟糕，但是纠结于这个事实会让它们看起来比实际情况更令人畏惧。令人畏惧的家务永远也做不完——实在是太难鼓起勇气去做它们了！但是 [公寓疗法](http://www.apartmenttherapy.com/the-stopwatch-timer-trick-for-getting-chores-done-252359) 有一个让你摆脱心理障碍的妙招:设置一个计时器。

下次你要做你最讨厌的家务时，先设置一个计时器。你不是在和时间赛跑——这纯粹是观察所得。一旦你看到它实际上只花了很少的时间，你就会意识到你最害怕的家务实际上并不是字面上的折磨。

(这类似于记住有些事情基本上不需要时间去做。对我来说，在这方面最大的启示是当我走进公寓时挂好我的外套和围巾——当我感到懒惰时，我会把它们扔在椅子上，但我意识到这基本上没有节省我的时间。如果你实际上没有从中获得任何东西，放纵又有什么好处呢！)

是的，有些杂务不是五分钟或者十分钟就能完成的。但是设置一个计时器也有帮助。这一智慧来自于 Tumblr 的《非常寒冷的清洁建议》[](https://www.amazon.com/Unf-Your-Habitat-Youre-Better/dp/1250102952?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/why-you-should-set-a-timer-when-you-do-chores-1820577275&asc_source=&rawdata=[r|https://www.google.com/[t|link[p|1791338553[a|1250102952[au|5730718882459761393[b|lifehacker&tag=kinjalifehackerlink-20)*。作家雷切尔·霍夫曼反对过度清洁，这种做法太累人了，以至于下次会让你不想清洁。相反，霍夫曼建议用 20 分钟的时间来清洁，中间间隔 10 分钟。(就像悠闲的 [番茄](https://lifehacker.com/productivity-101-a-primer-to-the-pomodoro-technique-1598992730) 。)频繁的休息可以确保你不会筋疲力尽，虽然你可能无法在 20 分钟内打扫完整个家，但你会看到在这段时间内可以取得什么进展。*

*当你看到可以在 20 分钟内完成的事情时，清洁不再是一个要么全有要么全无的命题。你可以利用你所拥有的时间来打扫你能打扫的地方，你的家会比你开始打扫之前更干净。*

*你也可以使用这种方法来帮助自己克服不愿开始的情绪:我会设置一个十分钟的计时器，告诉自己我只需要打扫——通常是把要洗的衣服收起来——那么长时间。杂务很讨厌，但我能坚持十分钟！大多数时候，我甚至在计时器响之前就完成了令人畏惧的家务。*
