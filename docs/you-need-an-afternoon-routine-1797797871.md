# 你需要一个下午的惯例

> 原文：<https://lifehacker.com/you-need-an-afternoon-routine-1797797871>

人们总是告诉你如何最大限度地利用早晨的，但是你的早晨惯例——不管是什么——都是*好的*。你真正需要的是下午的例行公事。

Watch

不要误解我，以正确的方式开始你的一天是很好的。对我来说，这意味着喝一杯水，遛狗，快速做一顿富含蛋白质的早餐，然后喝一杯浓浓的黑咖啡。之后，我会坐下来浏览重要的电子邮件，检查空闲时间，放一些符合我当天心情的音乐，然后开始写作。这几年对我来说变化不大。

但是到了下午，我的精力下降了。午餐后我仍然很饱(食物昏迷时间)，早上的写作课让我筋疲力尽，我的注意力开始消退，所以我开始漫不经心地浏览互联网，我的视频游戏和网飞队列在我耳边低语着甜言蜜语。你可能知道这种感觉。下午是分心的力量最大的时候——你疲劳、易怒、 [并且更容易冲动](http://www.psychologicalscience.org/news/minds-business/is-there-an-ideal-time-of-day-for-decision-making.html) 。因此，我必须想出一个让我在下午保持高效的例行程序，但仍然承认我的自然工作流程。以下是我的推荐。

### 站起来，动起来

首先，从你的办公桌前站起来，移动你的身体。如果你只是坐在那里，你的不适只会变得更糟。我把每天的锻炼时间从早上挪到了下午，现在我感觉更有精神和活力了。不过，你不必跑步或举重来获得好处。下午 3 点左右简单的散步——理想的是在户外靠近大自然呼吸一些新鲜空气——会让你摆脱恐惧。即使你不能去外面， [反正在办公室里走一走](http://lifehacker.com/why-walking-throughout-the-day-is-just-as-important-as-5990300) 。每天在同一时间移动你的身体，这样你的大脑就知道你还在做事情。

### 做简单的事情

进入“简单列表”每份工作的某些方面并没有那么难。你知道，家务，组织，重复性的任务，和一般的通信。把所有这些东西留到下午，你可以打开自动驾驶仪，听你最喜欢的音乐。对我来说，这意味着回复电子邮件，寻找故事来推销，为其他故事做研究，编辑照片，根据需要组织文件。除非那天有什么紧急的事情，否则我会在午饭后才去处理它。只有我，我的耳机，一些欢快的音乐，一杯苏打水，还有一个在我必须做的乏味事情上的精神恍惚的冲刺。你会惊讶地发现，仅仅这个改变就能为你的一天带来多大的改变。

### 只花五分钟在那件“大事”上

如果，也只有如果，你需要完成一个更大的项目， [告诉自己，你只会为此工作五分钟](https://lifehacker.com/work-for-just-five-minutes-to-break-procrastination-hab-1586070938) 。实际上，在你的手机上设置一个计时器或者其他什么东西，在这五分钟内做你能做的任何事情。你可能会进入一个良好的工作流程，并在这五分钟内战胜下午的低迷。如果你这样做了，一定要奖励自己并强化这种行为。如果没有坚持下来，没关系，不要惩罚自己。暂时回到简单的事情，稍后再试。

### 杀了你的宝贝

不管你做什么类型的工作，我们都有“小宝贝”，或者个人最喜欢的元素，我们认为对我们的工作是必要的，但实际上不是。在写作中，它通常是指引起眼球转动的单词或短语，但这个概念可以存在于任何类型的作品中。例如，你的 PowerPoint 幻灯片中那些特别花哨的图形，或者那些实际上并没有改善用户体验的花哨的代码行。基本上就是你个人热爱但没有批判性思考过的东西。

嗯，在一天结束的时候，你基本上已经释怀了，对吗？那是出击的时候了。经典的说法是，“写醉了，编辑清醒了”，但 [我更喜欢“写醉了，编辑宿醉了”的做法](https://lifehacker.com/why-you-should-edit-your-writing-when-youre-hungover-1742296090) (不，我不是天天写醉)。当你宿醉——或者在这种情况下，在一天结束时筋疲力尽——你不会关心所有那些你在当天早些时候认为非常聪明的小事。利用你的易怒情绪，精简你所做的事情。

### 计划好明天

最后但同样重要的是，花一天的最后一个小时到半个小时左右为明天做一个计划。当你早上醒着并且精力充沛的时候，决定哪些事情应该优先处理，哪些事情需要完成。其他事情都可以移到你下午的“简单清单”上我通常也用这段时间给自己写备忘录。这样，当我第二天早上上班的时候，我会看到到处都是便利贴提醒，告诉我需要做什么，这样我就不会浪费精力去记住每件事。
