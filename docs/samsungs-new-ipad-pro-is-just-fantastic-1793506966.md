# 三星的新 iPad Pro 简直棒极了

> 原文：<https://gizmodo.com/samsungs-new-ipad-pro-is-just-fantastic-1793506966>

在苹果第一款 9.7 英寸 iPad Pro 发布一年后，三星的新版本感觉很大胆。虽然它有着同样光滑的线条，一样轻，并且一侧有磁性连接，便于键盘盖的连接，但三星 2017 年的 iPad Pro 被莫名其妙地称为 Galaxy Tab S3，与以前的 iPad 不同，这款 iPad 运行安卓系统。

Watch

从技术上来说，如果你想“准确”地说，这不是 iPad Pro，而是三星一年多来的第一款高端 Android 平板电脑。2015 年，安卓在与 iOS 的平板电脑大战中输了。虽然谷歌的移动操作系统由于廉价的 Kindle Fire 而统治了预算市场 [，但没有人真正表达过对运行绿色机器人小大脑的 600 美元媒体消费机的渴望。见鬼，在这一点上T5 人们甚至不再买 iPads 了](http://gizmodo.com/amazons-50-fire-tablet-sucks-but-its-all-the-tablet-i-1731494027)T7】。想要平板电脑的人已经有了。

三星希望通过一款闪亮的新设备改变人们的想法，这款设备恰好与当前最畅销的高端平板电脑 iPad 有着惊人的相似之处。它只比同类 iPad 轻百分之一磅，短不到十分之一英寸，两款设备的厚度都只有 0.24 英寸。32GB 版本的售价也是 600 美元，它们可以在脸书上零延迟地处理从绘图到快速书写的所有事情。也许三星认为其新产品与去年的 iPad Pro 之间的显著相似之处是对苹果的一种赞美，而不是赤裸裸的争夺日益萎缩的市场份额。虽然苹果、 [和法院、](http://gizmodo.com/samsung-vs-apple-verdict-apple-wins-huge-5937762) ，往往意见不一。

这两款平板电脑有些不同。除了运行 Android Nougat 而不是 iOS 10，Tab S3 还配备了高通 820 处理器(理论上比据报道计划用于 Galaxy S8 手机的 835 处理器慢)，只有 32GB 版本，并包括一个漂亮的小笔，用于在公认充满活力的 AMOLED 显示屏上绘图。由于苹果铅笔要多花 100 美元，三星 Galaxy Tab S3 感觉很划算。

这两款设备都不包括键盘盖。苹果公司售价 150 美元，它的外壳和键盘看起来就像是你用泡沫塑料在帆布上打字。三星的键盘和带有 chiclet 风格按键的外壳售价 130 美元，在上面打字更令人满意。虽然每个键都足够小，位置也够奇怪，以至于我在使用它的时候不得不进行两次三次四次拼写检查。

我有点惊讶于 Tab S3 作为一个生产力设备是如此的方便。Android 作为一个操作系统真的不能在平板电脑上运行。只有一小部分应用真正针对我们使用平板电脑时的风景模式进行了优化。长期以来，操作系统首先面向手机，其次才是平板电脑。因此，能够在平板电脑上写下这篇评论或修改照片是一个惊喜。然而，如果你想从 Android [获得真正的笔记本电脑般的体验，你最好选择 Chromebook 而不是](http://gizmodo.com/samsungs-new-chromebook-is-a-nearly-perfect-budget-comp-1792183698)——因为这些产品至少是以生产力为重点，而不是作为一个方便的奖励。

Galaxy Tabs S3 的一个大问题是，它应该和 iPad Pro 存在于同一个奇怪的地方。虽然它可以帮助你制作电子邮件(或博客)，或画一些不错的漫画，但它主要是为了消费媒体，而不是创造媒体。这就是为什么 2，048 x 1，536 的显示器支持[【HDR】](http://gizmodo.com/what-the-hell-is-hdr-1790991673)的原因，也是为什么我每天晚上躺在沙发上休息时都要使用它，但我还是成功地充了一周电。

除了类似 iPad 的装饰，Galaxy Tab S3 的核心是一款辅助计算设备，我认为三星和苹果都不太了解这个群体。这不是商业用途，也不是学生的主要设备，更不是艺术家的必需品。这是纯粹的奢侈品，三星和苹果坚持认为我们需要，即使我们已经有了手机和笔记本电脑，可以做 S3 做的一切。这是你买的东西，因为你厌倦了看电视时腿上放着一台电脑，或者你想在下次出差时带上轻便的东西。

三星 Galaxy Tab S3 非常擅长作为补充设备。如果你弄坏了你的 iPad，或者最终攒够了钱来购买你的第一台高端平板电脑，那么 Tab S3 是一个 600 美元的不错选择。它比 iPad Pro 和铅笔便宜 100 美元，唯一真正的缺点是 Android 偶尔会对平板电脑不友好。在我看来这是一个很小的缺点。随着 iPad 山寨版的出现，Galaxy Tab S3 占据了至高无上的地位。
