# “稍后收听”就像 Instapaper 上的播客节目

> 原文：<https://lifehacker.com/listen-later-is-like-instapaper-for-podcast-episodes-1824150938>

大多数播客应用程序都是围绕查找和订阅节目而构建的，而不是剧集。但是听听笔记， [Lifehacker 最喜欢的播客搜索引擎](https://lifehacker.com/the-best-podcast-search-engine-1818560337) ，现在可以让你建立各种播客剧集的定制播放列表。

Watch

使用 [稍后收听，](https://www.listennotes.com/listen/?s=m) 你可以建立一个剧集列表，你可以在任何播客播放器中订阅，如苹果播客或 Stitcher。然后在收听笔记中搜索剧集，将它们添加到您的稍后收听列表中，它们会自动显示在您的播放器中。

例如，我找了以《地下城世界》为特色的播客，这是一个类似于《龙与地下城》的游戏，但是……不同。我将它们添加到我的稍后收听播放列表 中，它们会在我最喜欢的播客应用程序中弹出。我还可以通过 [收听提醒](https://www.listennotes.com/alerts/?s=side) 获得新的潜在剧集通知，这就像谷歌播客提醒一样。

如果你有一个朋友或亲戚“想加入播客”,但没有时间或头脑自己订阅节目，那么“稍后收听”是一个给他们发送节目的好方法。做上面所有的事情，但是在他们的手机上。然后，您可以选择播客剧集添加到他们的稍后收听播放列表中。这样你就可以给你爸爸播放历史播客了。
