既然我们已经解决了这个问题，你应该停止使用 iTunes——也就是你从苹果下载的版本。你没有完全退出应用程序。你应该把它换成苹果刚刚在 Windows 商店推出的版本。

不，Windows Store iTunes 看起来和你从苹果下载的 iTunes 没什么不同。它不包含任何额外的功能。你不会看到 Clippy 弹出来问你想把什么音乐同步到你的设备上，尽管这可能会让我们很高兴。

### 为什么要用 Windows Store 版本的 iTunes？

如果你有[Windows 10s](https://gizmodo.com/microsoft-is-going-to-stop-cramming-windows-10s-down-pe-1823578944)——我为你感到难过，如果是这样——你没有选择。由于你不能从网上下载和安装任何你想要的应用程序，iTunes 的 Windows Store 版本是你能够使用 iTunes 的唯一方式。很抱歉，你花了这么长时间才能够将音乐同步到 Windows 上的 iPhone。

对于大多数 Windows 用户来说，将 iTunes 用作 Windows Store 应用程序，而不是从苹果公司独立下载的唯一令人信服的原因是，你不会再被更新所困扰。就是这样。如果这对你来说不是一个倒胃口的东西，那么，无论如何，如果你想要的话，坚持使用你从苹果公司获得的版本。

然而，我无法忍受苹果公司在 Windows 上的软件更新工具偶尔弹出的窗口。每次苹果有更新都要手动安装 iTunes，很烦。第一千次，我不想安装 iCloud for Windows。我从来不想安装 iCloud for Windows。我宁愿去买一台 Mac，也不愿安装 I cloud for Windows——这就是苹果的实用程序实现得有多差。

### 如何轻松切换到 iTunes 的 Windows Store 版本

如果你使用的是从苹果下载的 iTunes，现在就停止你正在做的事情。你不必卸载应用程序，把你的音乐库移到一个临时文件夹，或者，嗯，做任何事情。什么都不做。

取而代之的是，启动微软商店应用程序——或者只需点击 [这个便捷的链接](https://www.microsoft.com/store/productId/9PB2MZ1ZMB1S)——然后安装 iTunes。作为安装过程的一部分，你在苹果下载的 iTunes 中所做的一切都会迁移到你从 Windows Store 下载的版本中。这再简单不过了:

虽然 Windows Store 版 iTunes 的一些用户报告说，让应用程序识别文件或将文件同步到他们的 iPhones 或 iPads 有困难，但我没有遇到任何问题。Windows Store iTunes 的外观、感觉和操作都和你从苹果直接下载的版本一样。不过，如果你有问题，你可以卸载 Windows Store 版本，从苹果网站 下载 iTunes 安装程序 [。](https://www.apple.com/itunes/download/)

(不，回到苹果下载版本并不像转移到 Windows 商店版本那么容易。您必须完全卸载 iTunes 的 Windows Store 版本，才能成功运行您从 Apple 下载的 iTunes 安装程序)

现在判断通过苹果软件更新工具发布的 iTunes 更新和 Windows Store 版本应用程序的 iTunes 更新之间是否会有任何延迟还为时过早。我只能假设两个版本会在大致相同的时间收到更新，但如果我注意到 iTunes 通过 Windows Store 更新要花很长时间，我会报告回来。
