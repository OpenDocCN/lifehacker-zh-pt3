# 全明星工作区

> 原文：<https://lifehacker.com/the-all-star-wars-workspace-1756058678>

卡罗琳娜真的很爱《星球大战》，我们也不能怪她。我们也不能因为这个令人惊叹的黑白工作区而责怪她，这里挤满了主题合适的人物、玩具和一些其他惊喜，以及充满了坚实的技术。



Carolina 在她的 Flickr 帖子(链接如下)中没有提供太多组成她的工作空间的组件。然而，你只需看一眼就能知道很多——戴尔宽屏(我们打赌那是一款 [27 英寸戴尔 U2711 显示器](http://smile.amazon.com/Dell-UltraSharp-U2711-27-inch-Widescreen/dp/B0039648BO/?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-all-star-wars-workspace-1756058678&asc_source=&tag=kinjalifehackerlink-20) ，*感谢 vray44！* [~~30 英寸戴尔 U3014 显示器~~](http://www.amazon.com/Dell-UltraSharp-30-Inch-PremierColor-Monitor/dp/B00C2RPW8O?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-all-star-wars-workspace-1756058678&asc_source=&tag=kinjalifehackerlink-20) )上面贴着《星球大战》的壁纸，雅马哈书架上的扬声器放在自己的支架上，iPad(或者可能是 iPad Mini)放在左边那里，一个海盗船游戏耳机和 [弯刀 MMO/MOBA 游戏鼠标](http://www.corsair.com/en-us/scimitar-rgb-optical-moba-mmo-gaming-mouse) ，那个 [网件夜鹰路由器](http://www.amazon.com/NETGEAR-Nighthawk-AC1900-Gigabit-Router/dp/B00F0DD0I6/?asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-all-star-wars-workspace-1756058678&asc_source=&tag=kinjalifehackerlink-20) 等等，更不用说《星球大战》的动作人偶、风暴了)

如果你能指出那是哪个电脑机箱，我们很想知道。除此之外，请尽情欣赏一位粉丝的工作空间，以及卡罗莱纳州在将整件事放在一起时使用的大量对比。

如果您有自己的工作空间可以炫耀，请将其添加到我们的 [**Lifehacker 工作空间展示并告诉 Flickr pool**](http://www.flickr.com/groups/lifehacker-workspace-showandtell/pool/) 来与我们分享。确保您包含的任何照片至少为 1280x720。将它们保持在 16:9 也有帮助！包括一些文字，关于你使用的材料，你是如何想出这个设计的，以及任何其他相关的细节。如果你巧妙的组织和良好的设计感吸引了我们的眼球，你可能就是下一个特色工作区！
