# 制造商正在努力制造苹果的 Face ID 相机，这意味着 iPhone Xs 将越来越少

> 原文:[https://life hacker . com/manufacturers-is-difficult-to-make-apple-face-id-cam-1819851796](https://lifehacker.com/manufacturers-are-struggling-to-make-apples-face-id-cam-1819851796)

iPhone X 即将上市，但一份新报告称苹果正在努力制造足够的复制品。据 [彭博](https://www.bloomberg.com/news/articles/2017-10-25/inside-apple-s-struggle-to-get-the-iphone-x-to-market-on-time?utm_content=tech&utm_campaign=socialflow-organic&utm_source=twitter&utm_medium=social&cmpid%3D=socialflow-twitter-tech) 报道，该公司甚至可能降低了其 [新款面部识别 Face ID 相机](https://lifehacker.com/what-you-need-to-know-about-face-id-on-the-iphone-x-1804641406) 的精度要求，以帮助其供应商按时满足需求。

那么 iPhone Xs 会足够多吗？Face ID 会兑现苹果的宣传吗？以下是你需要知道的。

### 苹果的 Face ID 相机是怎么回事？

当苹果最初在 9 月份发布 iPhone X 时，它声称 Face ID 比它要取代的指纹扫描仪安全得多。该公司表示，一个随机的人拿起你的 iPhone X 并用他们的脸解锁的概率是百万分之一，而触控 ID 设备的概率是五万分之一。

尽管彭博的报告指出，Face ID 仍然比触控 ID“好得多”，但如果他们的消息来源是正确的，iPhone X 可能不会像承诺的那样安全。

在一份官方声明中，苹果公司否认了这一传言，并重申了其“百万分之一”的说法，但在 iPhone X 发布和黑客对其进行测试之前，我们无法确定。

### iPhone X 的库存会足够吗？

即使苹果确实降低了 Face ID 的精度规格，这可能也不足以让生产走上正轨。根据经常被引用的苹果分析师 Ming-Chi Kuo 的说法，苹果在发布时手头可能只有 200-300 万台，在今年剩余时间里可能有 2500-3000 万台。

相比之下，据报道，苹果在第一个周末就卖出了 550 万部 iPhone 8 和 8 Plus。之前型号的销量甚至更高，在发布会上有近 1000 万人购买了 iPhone 6 或 6 Plus。

基于这些数字，看起来对 iPhone X 的需求可能会轻松超过发布时的供应量。尽管如果这篇关于 Face ID 相机的报道是正确的，但无论如何，在购买新的智能手机之前再等一会儿可能是有意义的。
