# 腌制一些东西

> 原文：<https://deadspin.com/pickle-some-stuff-1819911663>

从来没有人——我查过了；这实际上从未发生过——已经过去了，“该死，要是我没有腌制一些食物就好了。”这是否意味着腌制一些食物对你来说是一个好主意，现在，无论你在哪里，即使你现在正在一个灰色的会议室里参加一个重要会议，周围都是你的工作上级？是的，实际上，它确实意味着。腌制一些东西！



但是腌制，你在抱怨，绞着你的手，急不可耐——*那不是酒吧食品吗？我不想成为一个悲伤的酒鬼，吃着罐子里的腌鸡蛋。但是在这里你所做的一切都是在映射你自己的无知！腌制不仅仅是酒吧苍蝇和他们悲伤的酒吧食物的专利。腌制是为任何人准备的——不要说了，是为*每个人*——喜欢好东西的人准备的。老实说，我甚至不知道你怎么能声称自己是一个喜欢好东西的人，如果你不马上腌制一些植物性食物的话。我认为...我觉得你不能。*

现在你会说*呸，我不需要腌制植物性食物，因为我不是严寒的北方居民，冬天必须将一些蔬菜“储存”在罐子里，否则就会死于营养不良。在这里的文明中，我可以去超市，即使是在一月，只要我他妈的高兴，就可以买一些新鲜的水果和蔬菜。听着，我不是让你腌制东西来保存。我是在告诉你腌制食物是为了享受。明亮的味道能活跃你的味觉，把你不太愉快的食物变成，呃，更愉快的食物。为了极度的快乐。如果你认为你不能在你的生活中使用更多的乐趣，你可以去的地方是:地狱。*

听着。今天是星期五，我没时间听你胡扯。让我们跳过你剩下的微弱的抵抗，直接看你被真理的力量吓到、打垮、打败，屈服于腌制食物的命令。作为胜利者，这是我的特权！

很简单。你不需要温度计和无菌罐以及所有那些愚蠢的垃圾。你有一加仑的冷冻袋吗？好吧。你可以用它来腌制一些东西。或者你可以把它包在你该死的头上，如果你仍然决心不腌制一些东西！

黄瓜适合腌制。也许你注意到了，超市里有一整排。我喜欢快速腌制一些切片黄瓜，这是一种非常半吊子但却很好的模仿当你出去吃韩国烤肉时，你会在一个小盘子里得到黄瓜泡菜。我快速腌制的黄瓜和黄瓜泡菜一样好吃吗？不，他们不是。但是它们很好吃，我可以在家里买到，因为是我自己做的。这就是该死的想法。

我是这样腌制黄瓜的:首先，我拿一根在杂货店单独包装出售的巨大无籽黄瓜，我打开包装，把它切成薄片。曼陀林切片机很好，但还有什么是好的:一把刀。切黄瓜大概是 75%的工作。然后，我将黄瓜片倒入一加仑的冷冻袋中，倒入大约四分之一杯的米醋、盐、鱼露、糖或红糖或蜂蜜以及许多许多红辣椒丁的临时混合物。然后，我把袋子密封起来，把它扔几分钟，让里面的液体和小辣椒分散开。然后我把它放在冰箱里，暂时忘掉它。后来，在接下来的两周内的某个时候，我吃了很多该死的辣腌黄瓜！他们很棒！

我打赌你有过这样的想法:“我想要一些墨西哥玉米卷。”你知道玉米卷上什么好吃吗？该死的腌沙拉，是什么！我用非常非常薄的红色卷心菜丝、切细的墨西哥豆瓣菜、芫荽叶和切碎的墨西哥胡椒做了一份腌沙拉，放在一个一加仑的冷冻袋里，里面有少量的酸橙汁、我手边的任何非香醋、盐，可能还有一点孜然粉，还有一点食品室里的任何糖。你几乎不需要这些液体。从半颗卷心菜和一个棒球大小的豆豉中得到多少沙拉，只要几汤匙就够了。我把所有的东西都扔在那里，然后把袋子放进冰箱。然后，下一次我有这个想法，“我想要一些墨西哥玉米卷”(大约每 48 小时发生一次)，我可以吃墨西哥玉米卷...上面有美味可口的粉红色沙拉。

你现在明白了。少量某种适度加盐、适度增甜的醋混合物，一些切碎或切片或以其他方式变小的植物——在我的高度推荐下，包括一些辣椒来使它变热——以及在寒冷的地方密封袋中的一些时间。那是腌制。如果你愿意，你可以用厌氧发酵和数学或科学或其他方法使它变得更复杂，但现在，只要把一些(比喻)大便粘在一个装有醋液的袋子里，过一会儿再吃，当它有一段时间变得有风味、辛辣、酸味和美味时。你能做到的。连*你*都能做到！

腌制东西很有趣。一旦你开始腌制东西，你就会开始腌制东西。如果你在做越南主题的米粉沙拉，你会腌制一些胡萝卜和黄瓜，也许还会放一些豆芽。如果你吃汉堡，你会在汉堡上面放一些红洋葱。如果有猫在你家附近游荡，你会好奇地看着它们，想着罐子的大小。最终你的冰箱里会装满一袋袋腌制的东西！你甚至不会记得它们都是什么。你会成为一个泡菜狂人。这是一个建议。

腌制一些东西。还是不要！我一点也不在乎。
