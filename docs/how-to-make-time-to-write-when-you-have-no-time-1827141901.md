# 当你“没有时间”时，如何腾出时间写作

> 亲爱的帕特里克:
> 
> 我做律师已经两年了。我去了法学院，因为我想学习如何思考、写作和发表一些可信的关于公民/人权的法律。在解决了法学院的巨额债务后，我接受了一家律师事务所的高薪工作，这样我就可以在财务上保持稳定和独立。如果事情变糟，我的父母没有能力支持我。

> 但是我的工作没有灵魂，有时候我挣扎着起床去工作。从法学院毕业后，我真的没有写过什么新东西(与工作无关的)。我想为自己写更多，但一天下来，我的大脑一片混乱，我的意志力完全耗尽了。我没什么可给的了。当我下班回家时，我特别虚弱，因为我情绪低落，我想要即时的满足，所以我通常屈服于看电视或其他逃避现实的活动，直到我不得不上床睡觉，第二天又开始整个循环。我一直在尽我所能利用一切空闲时间来写作，但我似乎做不到。世界上所有关于时间管理的生产力文章等等。没有成功。我现在觉得唯一的出路就是辞职。
> 
> 换一份诉讼律师的工作并不是一个真正的解决方案，因为工作要求是相当普遍的。唯一的区别是，在其他雇主那里，挑战性/枯燥性较低的工作薪酬较低。我担心，如果我接受另一份工作，即使是法律之外的工作，我也会讨厌这份工作，我会放弃一份至少稳定且收入丰厚的工作。如果我彻底停止工作，我担心每天没有地方可去，没有截止日期，或者没有我要负责的人，会让我变得不守纪律，那时我实际上也不会写任何东西。此外，我没有钱，在经济上照顾自己的压力太大，无法专注于写作。
> 
> 我的一部分认为我应该留在这里，也许有一天我会慢慢喜欢和欣赏我目前的工作。另一方面，我认为我呆的时间越长，我就越无法实现我的梦想。我不想难过，不想有遗憾。我该怎么办？

嘿，想成为作家的人:

在成为职业作家之前，我也经历过类似的过程。我想出版，但我已经在做一份要求很高的案头工作，每天等五点钟。下班后，我会回家放松一下，因为我完全筋疲力尽了。我总是找不到时间坐下来敲击那些琴键，尤其是在精神焕发的时候。我靠在绳索上，确信自己会永远被困在原地。你目前正处于一个类似的十字路口，WW。现在，你的问题不是“我如何成为一名作家？”那很简单——你写。你应该问自己的问题是“我真的想成为一名作家吗？”这是一个简单的“是”或“不是”的问题，比你想象的要难回答，因为没有回旋的余地。

当我站在自己的十字路口时，我学到了职业生涯中最有价值的一课:你找不到时间去做你热爱的事情，你只能挤出时间。你牺牲爱好，你改变睡眠时间表，你排除干扰，你抓住自己的耳朵，把你的懒屁股拖到电脑前，关掉你的无线网络。

你说你从法学院毕业后就没写过东西，因为你太忙太累了？

那是马粪。

几乎每个作家 [都是在从事另一份工作](https://lifehacker.com/your-day-job-isnt-an-excuse-to-not-pursue-your-passion-1622380773) 时开始的，包括我自己。我早起，下班前写作；我用我的周末写作(你有周末，不是吗？);我边吃晚饭边写；我甚至在办公室工作不忙的时候偷偷写信。WW，你要么不愿意承诺，要么不想成为一个作家，简单明了。如果你真的考虑过这个问题，并决定你不想成为一名作家，那也没关系。你只是在经历你的四分之一生活危机，感觉一切都是机械的和不确定的——就像你被困在一条没有成就感的道路上，导致生活缺乏激情。会过去的；不要离开你的工作。做一些志愿者工作，加入当地的俱乐部，多锻炼，或者做任何你能改变你日常生活的事情。你会感觉更好。

然而，如果你真的想成为一名作家，你仍然不应该辞职(这肯定不是唯一的出路)，但你需要向自己证明，边工作边写作是可能的(这肯定是可能的)。所以这是你的药方——每当你觉得自己太忙或者太懒的时候，在精神上想想:

*   开始把自己当成一个作家。我不在乎你是否认为这听起来很愚蠢——做吧。如果你不允许自己相信自己有能力，你永远也不会说服自己写更多。你是个作家。
*   提醒自己写的东西不一定要好；它只需要存在。不要让恐惧或完美主义阻碍你。写作反正是 90%的修改。记下一些奇怪的词。
*   仔细看看你的日程表，寻找你有空的时间。明明就在那里，不要骗自己。即使是十分钟的街区也足够开始了。你可能会惊讶于你每天实际上有多少时间。
*   创建一个写作计划。这很重要，因为你需要养成写作的习惯。没有习惯，没有写作，没有改变你的生活。如果你知道晚上写作是不可行的，因为你工作太累了，那就早点睡觉，然后早点醒来，在早上工作前写作。也许甚至可以问问你的老板，你是否可以每周在家工作几天，并在工作时间挤出一些时间写作。如果你想要改变，那就去改变。通过反复试验，你最终会发现什么适合你。
*   [坚持你的写作进度](https://lifehacker.com/how-to-stick-to-a-writing-schedule-1653234644) 。 [普利策奖获得者詹妮弗·伊根说](https://lifehacker.com/three-writing-tips-from-pulitzer-prize-winning-author-j-1786507353) “锻炼是写作的一个很好的比喻。如果你不习惯锻炼，你会想永远避免锻炼。如果你习惯了，不习惯会感觉不舒服和奇怪。无论你在写作生涯的哪个阶段，写作也是如此。即使每天 15 分钟也会让你保持习惯。”做你的练习，WW。
*   到了该写的时候，又不知道写什么，就改做这个练习: [写自己的传记](https://lifehacker.com/recalibrate-your-career-goals-by-writing-your-own-biogr-1708295019) ，用第三人称。它会让你写点东西，并迫使你以一种更具分析性的方式从外部视角看待你的生活(这可能会帮助你理清你的职业困惑)。

根据需要重复剂量。如果症状持续，重新评估你是否真的想成为一名作家。请记住，如果你不选择按照给出的建议采取行动，关于生产力和时间管理的文章一文不值。现在是时候采取行动了，WW。去，写吧。这是区分崇拜者和真正作家的唯一标准。
