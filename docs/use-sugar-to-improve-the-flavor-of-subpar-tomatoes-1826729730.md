# 在不合格的西红柿上撒糖来改善它们的味道

> 原文：<https://lifehacker.com/use-sugar-to-improve-the-flavor-of-subpar-tomatoes-1826729730>

除了圣女果——它们几乎全年都很好——我尽量避免至少在 7 月前购买西红柿，因为过早购买通常会导致失望。但我并不完美，前几天我不小心买了几个。正如人们所预料的那样，它们在调味部门有点乏善可陈。

Watch

与其扔掉它们，我决定摆弄一下，看看是否能让它们尝起来更接近真正的夏季番茄。我把一片切成薄片，撒上一些盐，然后在薄片上放一些健康的蔗糖。然后我把它们放在铁丝网上，抽出一些水，大约 10 分钟，然后撒上一点盐和糖。

这一过程并没有神奇地将番茄变成八月番茄，但是它有了很大的改进。抽出水分有助于浓缩风味，糖的加入增加了一些急需的甜味，使它尝起来更成熟、更有果味。我把切片放在涂有奶油奶酪的百吉饼上，非常开心地吃了起来。(注意:这一招有助于番茄味道更好，但对粉性没有帮助；粉状的西红柿应该被 [混合并过滤成血腥玛丽](https://lifehacker.com/this-is-the-easiest-freshest-way-to-make-a-bloody-mary-1826671554) 。)
