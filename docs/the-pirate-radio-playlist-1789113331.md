# 盗版电台播放列表

> 原文：<https://lifehacker.com/the-pirate-radio-playlist-1789113331>

音乐和盗版有着错综复杂的关系。只要我们一直在录制歌曲，那些法律之外的人就一直在向任何愿意听的人传播信号。这个播放列表颂扬了处于法律灰色地带的最早的音乐形式之一:海盗电台。



回到无线电的全盛时期，海盗电台广播未经许可的信号，在某些情况下甚至来自国际水域。虽然盗版电台今天仍然以某种形式存在，但音乐创作者和粉丝的反叛本性仍然在这个行业的表面下冒泡。

来自 Spotify 用户 [Mattchu Newitt](https://open.spotify.com/user/1216852495) 的这个播放列表吸引了任何曾经反叛过任何事物的人。从滚石乐队到野兽男孩，从莫特利·克鲁到弗兰克·辛纳屈。如果你曾经不得不打破一些规则来保持广播继续，你会在这个播放列表中找到一些喜欢的东西。
