# 若要运行更好的 D&D 游戏，请启用此网飞功能

> 原文：<https://lifehacker.com/to-run-a-better-d-d-game-turn-on-this-netflix-feature-1828313836>

运行一个龙与地下城游戏，或者任何桌面角色扮演游戏，需要告诉你的玩家他们看到了什么。玩家依靠你来给他们一种气氛和氛围的感觉，并且指出任何有趣的或者与他们的任务相关的事情。但是他们也需要你给他们留有提问和探索的空间。一个好的游戏大师学会如何足够详细地描述一个场景，但不要太详细。redditor non _ player on r/RPG 说，学习这种技能的一种方法是在观看电影和电视节目时打开音频描述。

Watch

音频描述是为视力受损的人提供的功能。旁白描述屏幕上发生的事情，小心避免与对话重叠。打开音频描述，你可以听完整部电影，甚至不用看屏幕，就知道发生了什么。而且非常接近一个游戏高手对一个场景的描述。(它比广播剧或电影剧本要具体得多，两者都没有如此详细、即时的叙述。)

音频描述可用于 DVD、下载或流式播放的许多(但不是全部)电影。这里的 [是带音频描述的网飞书目列表](http://www.acb.org/adp/netflixad.html) ，而 [是带音频描述的亚马逊书目列表](https://acb.org/adp/amazonad.html) 。大多数网飞原版都有这个功能。

要在大多数服务中打开音频描述，开始观看一个标题，然后进入音频和字幕设置(在这里你可以更改语言)。网飞有的[指令给所有的设备](https://help.netflix.com/en/node/372)和。在苹果设备上，你可以在设置> 通用>辅助功能中[默认启用音频描述](https://help.netflix.com/en/node/57028?ba=SwiftypeResultClick&q=audio%20descriptions)。

u/non_player 说:“我已经默认打开了它，现在我观看的所有东西都在可用时使用它，”这使他们成为更好的游戏大师:

> 它真的扩展了我描述游戏中场景和事件的能力，我强烈鼓励任何想升级游戏的人尝试一下。解说员描述事物的方式非常有趣，甚至扩展了我的常规词汇。

u/Non_player 第一次听到这个技巧是在一集幕后的 [骰子换大脑](http://diceforbrains.com/d4bp/) 节目中，这是一个“实际播放”播客，主持人通过麦克风播放《星球大战》RPG(有时还有《哈利·波特》RPG)，然后添加音效来增强故事情节。听 [实际播放播客](https://www.reddit.com/r/rpg/comments/6y8ifc/best_actual_play_podcasts/) 是学习 GM 技能的另一个好方法。
