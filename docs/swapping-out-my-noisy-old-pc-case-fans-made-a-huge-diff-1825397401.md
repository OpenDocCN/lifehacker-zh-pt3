# 换下我的嘈杂的旧电脑机箱风扇产生了巨大的差异

> 原文：<https://kotaku.com/swapping-out-my-noisy-old-pc-case-fans-made-a-huge-diff-1825397401>

通常当我听到“升级你的电脑”的时候，我会想到买一个新的 CPU 或显卡。但是有很多更小、更便宜的东西你可以升级，但仍然有很大的不同。比如:粉丝。



上周末，我终于全面升级了我的游戏电脑的机箱风扇，它让每天使用我的电脑变得多么愉快。我现在在这里建议你至少考虑做同样的事情。

我的电脑放在现在的机箱里已经有四五年了，在这段时间里，我慢慢地做了改进，换上了新的部件。随着今年春天波特兰的天气变暖，我注意到我的设备一直在大声运行，特别是当我在玩 CPU 密集型游戏时，如《孤岛惊魂 5》或《T2》刺客信条起源。我会玩游戏，然后意识到我可以听到风扇的嗡嗡声，更令人心烦意乱的是，每当 CPU 达到一定温度，风扇进入高速运转时，我的电脑机箱就会发出低沉的嗡嗡声。通过我的游戏耳机可以听到它，一旦我听到它，我就不能不听到它。

当然，我做的第一件事是清理我的电脑。这些天来，我保持我的风扇和机箱相当干净，所以这并不难做到，但当确定为什么你的电脑似乎比你记忆中的声音大时，这总是一个很好的第一步。几年前，当我玩我的第一台自建电脑时，我记得我注意到它发出可怕的噪音，甚至因为过热而开始关机。我打开箱子，结果拉出了大量的灰尘，之后它运行得更加安静和冷静。这一次没有发生如此戏剧性的事情，但清理一下仍然不失为一个好主意。

接下来，我把注意力转向了我的粉丝。我有一个更冷的 Master Trooper 机箱，我将它留在一个配置中，在机箱前面运行两个 120 毫米的机箱风扇，以便空气在主板前面从左侧到右侧通过。我用海盗船 H100i v2 一体式液体冷却器来冷却 i7 7700k CPU。当我安装冷却系统时，我买了几个 Noctua 风扇，所以我有一个 140 毫米的 Noctua 机箱风扇在机箱后面通风，一个 120 毫米的 Noctua 风扇在 CPU 冷却器的散热器上向上吹。散热器实际上可以带两个风扇，但是我订购的时候亚马逊上只有一个 Noctua 库存，所以我有一个库存的 Corsair 风扇作为散热器上的第二个风扇。

因此，要打破这一点:两个股票冷却器主 120 毫米风扇吹过前面，一个股票 120 毫米海盗船风扇吹过散热器，一个 Noctua 120mm 毫米风扇吹过散热器，以及一个 Noctua 140mm 毫米风扇排出机箱后面的废气。

我做的第一件事是翻转前面的两个风扇，以便它们将空气直接吸入机箱，这将产生更多从机箱底部到顶部的直接气流。那会让箱子里面更凉快。(它也可能会将更多的灰尘吸入机箱，所以我必须记得更经常地检查和清洁它。)不过，在我这么做之后，声音实际上变得更差了。前排的粉丝离我坐的地方更近，更容易听到。尽管如此，机箱内还是传来了风扇的嗡嗡声。

我开始隔离每个风扇，试图找出声音来自哪里。是饮水机本身的问题吗？前面的粉丝？或者可能是我 GPU 上的粉丝？我做了个测试。打开电脑，我打开机箱，把手伸进去，让每个风扇停止转动，看看每个风扇发出多少声音。(我知道在电脑运行时接触机箱通常是不可取的，但我没有风扇控制盒，这种方法比拔掉每个风扇要快得多，也容易得多。尝试自担风险！)

由于我暂时停止了粉丝的奔跑，很快就明白了股票粉丝是我的问题。两个 Noctua 球迷几乎是沉默的，但股票冷却主和海盗船球迷是*的方式*大声。那些是我的罪犯。

我又订购了三个 120 毫米的 Noctua 风扇，并调换了它们。有了新风扇，我的电脑几乎没有声音了。这是一个显著的变化，它让人们开始关注以前的喧嚣。

如果一年前你问我我的电脑有多吵，我会说它相当安静。我已经习惯了。这种低水平的声音污染在我们的家中越来越常见，因为它们充满了处理器，冷却设备，风扇，电器和通风系统。我们经常被持续不断的低频噪音所包围，虽然我们没有意识到这一点，但它仍然会产生微妙而普遍的响度。

自从换了我的风扇，我也注意到了更好的散热效果。尽管我的电脑听起来好像根本没有开机，但它运行起来明显更凉爽了。(值得注意的是，我还降低了 CPU 的超频电压，这降低了 CPU 的发热程度。)

我买的扇子每把大约 20 美元，所以这五把新扇子总共花了我大约 100 美元。这并不便宜，但如果你已经在购买昂贵的电脑部件，这也不完全是倾家荡产。至少在我看来，这是值得的。实际上我不知道 Noctua 粉丝是绝对的*最好的*粉丝还是什么，所以我并不是要对他们的竞争对手做出具体的认可。我总是看到他们在各种硬件网站获得高分，尤其是他们运行时的安静。我相信还有其他品牌也很好。

说真的，这只是 [另一篇直截了当的帖子](https://kotaku.com/seriously-though-maybe-get-a-switch-stand-1823760550) 关于一项相对平淡无奇的硬件投资，它让我的游戏生活明显变得更好。我甚至没有意识到那三个股票迷把我的电脑弄得和他们一样吵。将它们换成更好、更安静的风扇对我的日常电脑游戏有了显著的改善。
