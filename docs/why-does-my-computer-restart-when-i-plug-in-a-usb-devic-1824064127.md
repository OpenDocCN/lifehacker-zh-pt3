# 当我插入 USB 设备时，为什么我的计算机会重新启动？

> 原文：<https://lifehacker.com/why-does-my-computer-restart-when-i-plug-in-a-usb-devic-1824064127>

是时候再来一剂高科技 911 了。本周，一位读者对 [USB 设备](https://lifehacker.com/rip-usb-these-are-ports-and-connectors-to-care-about-i-1796823560) 又有了新的问题。具体来说，当他们将东西插入 USB 端口时，他们的系统有时会重新启动。从一到“不好”的范围来看，我们更接近右边而不是左边，但是他们可以尝试一些技巧来缓解一下。



每周一，在我们新的技术建议专栏 Tech 911 上，我们会努力解决你在使用台式机、笔记本电脑或其他喜爱的设备时遇到的所有问题。我们还提供软件建议，帮助您完成您想要做的任何事情。

### 我的电脑可能不应该随机重置，对不对？

Lifehacker 读者[**LichelMaforge**](http://lifehacker.com/1823776795)T5】写道:

“多年来，我一直在制造个人电脑。它们都工作得非常完美，但它们通常都有一个小缺陷:有时，而且很少，当我插入 USB 设备时，似乎我短路了系统，它会重新启动。帮我弄清楚为什么！”

我承认，这让我有点挠头。我的第一个绝对猜测是硬件问题——尤其是你提到你制造个人电脑。(我不是在批评你的技能，但是也许你的机箱内部而不是你的操作系统内部发生了一些事情。)然而，重置的不规则性也让我远离了硬件问题；我觉得如果有什么东西短路或者你有一个错误的 USB 端口或连接，你会看到这个问题更多。

在我进入故障诊断之前，请确保您尝试了不同的设备和不同的 USB 端口。当一个耗电的 USB 设备连接到一个特定的端口时，它可能会以某种方式造成崩溃，要么是它自己，要么是它也插入了，比如说，你的机箱的前面板端口以及许多其他设备。

### 开始时的一些硬件建议

一些 Lifehacker 的读者已经附和了我的第一个硬件建议:重新审视你的系统。确保您的主板正确安装在其支架上；三重检查你的布线(特别是你的电源到主板和 USB 头到主板)；确保没有奇怪的电线，碎片，或者谁知道会导致某种短路的东西。

> “可能是你的主板和机箱短路了。如果需要，请确保安装了支架，并且不要将固定主板的螺钉拧得太紧。”—[**【tux rocks**](http://lifehacker.com/1823779490)

> *“确保你在主板上使用了正确的竖板(固定螺丝)。”* — [**飘渺**](http://lifehacker.com/1823777440)

> “实际上，我在 2002 年生产的一台电脑上也遇到了同样的问题！我一直没弄清楚是什么原因导致的，但它只影响了前面的 USB 端口，所以我用胶带盖住了它，再也没有使用过，问题再也没有出现。”—[**bwong 2**](http://lifehacker.com/1823778727)

(将第三个建议放在“最后手段”一栏中。)

一旦你完成了对你的主板的快速检查——移动几根电线或者移除并重新连接你的 USB 接头——也要考虑联系你的主板制造商来检查 [一个更新的 BIOS](https://lifehacker.com/flash-not-that-kind-of-flash-your-bios-288017) 来下载。制造商可能已经发现了一些与 USB 相关的问题，并在更新中修复了它。

下一次你的 USB 问题出现时——假设它出现了，如果你快速查看你的主板没有解决问题——你可以用你的故障诊断变得有点疯狂。尝试拔下系统的所有电缆，然后重新连接。卸下并重新拔插显卡、内存和硬盘。(你大概不需要为你的 CPU 费心，但如果你想彻底，那就是你了。)清除 CMOS，方法是拔掉系统插头，取出主板电池，等待几分钟，再插回电池，然后通电，然后重置您之前定制的任何 BIOS 设置。

如果你仍然不能解决这个问题，你可以试着绕过它。拿起一个 PCI Express USB 卡，希望它至少能给你一个将 USB 设备连接到计算机背面的方法。也有可能是你的电源造成的，但这是一个更昂贵的替代品，一个更大的周末项目，一个可能不会真正解决你的问题的想法。对于您的系统来说也是如此——如果您的所有版本都使用相同的 ol’机箱，那么它的前面板 USB 接头可能有问题(假设您在使用它们时经历了这些随机复位)。

### 是软件惹的祸(又来了？)

有可能是一些 Windows 问题导致了你随机的 USB 重启。为此，我向你推荐我在上周的 [Tech 911](https://lifehacker.com/why-wont-my-usb-show-up-on-my-pc-1823887066) 专栏中建议的同一批技巧:

*   当您插入导致这些重置的设备时，检查设备管理器是否有任何奇怪的事情(如果是这样的话/您甚至有机会检查)。
*   考虑对产生这些随机复位的任何设备进行固件更新(如果它们被隔离到单个设备*和*，如果该设备甚至有可用的固件更新)。
*   拔下所有设备，并尝试卸载所有通用串行总线控制器。重新启动你的电脑，Windows 应该重新安装这些驱动程序。
