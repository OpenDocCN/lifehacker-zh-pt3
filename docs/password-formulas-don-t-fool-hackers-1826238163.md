# 密码公式骗不了黑客

> 原文:[https://life hacker . com/password-formulas-don-t-fool-hackers-1826238163](https://lifehacker.com/password-formulas-don-t-fool-hackers-1826238163)

每次我们在 Lifehacker 上写关于密码的文章时，都会有一些读者分享他们创建密码的秘密公式。据密码管理公司 [、达什莱恩](https://www.dashlane.com/) 的高级经理瑞安·麦钱特称，这些公式很容易被破解。

Watch

[Dashlane 最近分析了来自多年大规模数据泄露的 6100 万个密码](https://blog.dashlane.com/virginia-tech-passwords-study/)——许多安全研究人员、黑客甚至公众都可以获得这些密码。Dashlane 最大的缺点是人们不是很有创意。即使是用公式的也不行。

在像 *iloveyou* 、 *ferrari* 和 *starwars* 这样显而易见的常见密码中，Dashlane 发现了像“密码行走”这样的常见公式，这涉及到敲击相邻的键来创建看起来随机的东西，但实际上是令人难以置信的可猜测的。“行走”密码有 *1q2w3e4r* 、 *zaq12wsx* 、*！qaz@wsx* 。这些很常见，黑客可能会将它们包括在针对随机账户的“字典攻击”中。

也许，像 [one Lifehacker 读者](https://lifehacker.com/1602002905#_ga=2.24978556.770591559.1526910908-1297080755.1497980211) 一样，你“使用了一个基于网站名称的公式”你仍然处于危险之中，Merchant 说:“如果(一个黑客)知道某人的‘基本密码’，预测它会有什么变化并不困难。”尤其是因为黑客知道每个网站的密码要求。所以当你的一个公式密码暴露了，就可以全部暴露了。如果你只是在你的 Reddit 密码后面加上“tidder ”,黑客就会知道在你的脸书密码后面加上“koobecaf”。黑客还可以猜测您可能会用其他符号替换哪些符号。字母和数字可能会变成标点符号。将每一个 *i* 改为*！*，雷布斯风格，不会忽悠他们。

所以*请*，放弃你的公式，使用一个密码管理器，它会为你创建实际上是随机的密码，然后记住它们，这样你就不必学习它们了。你可以用 Dashlane。我个人喜欢 1Password。我们已经在这里 [列出了我们最喜欢的五个密码管理器](https://lifehacker.com/the-five-best-password-managers-5529133) 。我甚至评论了一个更新更可爱的选项，叫做[remember](https://lifehacker.com/remembear-is-a-good-password-manager-for-beginners-1820888270)。

你无法阻止账户被攻破；这取决于储存它们的公司和组织。你所能做的就是控制损失，让你的密码不那么容易被猜到。密码的作用是保护你的数据安全，而不是让你觉得自己很聪明。
