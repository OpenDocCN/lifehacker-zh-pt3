# 三种成分的欢乐时光:经典的，醉酒的夜宵

> 原文:[https://life hacker . com/3-ingredient-happy-hour-the-classic-boozy-vesper-1797339960](https://lifehacker.com/3-ingredient-happy-hour-the-classic-boozy-vesper-1797339960)

周末快乐，欢迎回到 [*三料欢乐时光*](http://lifehacker.com/tag/3-ingredient-happy-hour) ，每周饮品专栏，提供超级简单又美味的酒水。这个周末我们将啜饮一杯经典鸡尾酒，它的核心就是一杯清澈、冰冷的酒:维斯帕马提尼。

Watch

如果你熟悉詹姆斯·邦德佳能，你可能听说过晚祷。而不是纠结于“伏特加还是杜松子酒？”晚祷者通过说“去他的”和使用两种烈酒来回避整个争论。(公平地说，杜松子酒要多得多。)除了活力二人组之外，这种饮料还含有一点 Cocchi Americano ord Lillet，尽管我偏爱 Cocchi。

就像我说的，这只是冷酒，但它是第三种成分，使整个事情令人惊讶的是，特别是如果你习惯于啜饮马提尼。大多数食谱只需要半盎司的 Cocchi 或 Lillet，但我喜欢用 3/4，因为我认为 Cocchi 非常美味。要制作晚祷，你需要:

*   3 盎司杜松子酒
*   1 盎司伏特加
*   3/4 盎司 Cocchi Americano 或 Lillet

将所有东西倒入加冰的品脱玻璃杯(或搅拌杯)中，搅拌，搅拌，搅拌。搅拌超过你认为你需要的量，因为稀释是非常重要的。(如果你真的想拥抱邦德，你可以握手，但你会喝一杯过度激动的饮料。)搅拌不足的啤酒会有一股刺鼻的酒精味，你不需要那种噪音。穿上你最喜欢的衣服，滤入一个双座玻璃杯，边喝边喝。(如果你想把配料数量增加到——呃——四种，你可以用一大条柠檬皮装饰。你不说我也不说。)
