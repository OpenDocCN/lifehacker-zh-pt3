# 用这个必备的扩展删除你的旧脸书活动

> 原文：<https://lifehacker.com/social-book-post-manager-is-a-must-have-for-pruning-you-1823952493>

尽管每个人都在谈论 [删除他们的脸书账户](https://lifehacker.com/dont-delete-facebook-just-be-smarter-on-facebook-1823922407) ，我们怀疑你的大多数朋友可能像往常一样开始他们的一天——在早上通勤时查看脸书。但是如果你真的想清理你的脸书生活，Chrome 扩展 [社交图书管理员](https://chrome.google.com/webstore/detail/social-book-post-manager/ljfidlkcmdmmibngdfikhffffdmphjae?hl=en-US) 是一个很好的工具。



虽然你必须给 Social Book Post Manager 在脸书上“阅读和更改你的数据”的许可——这可能不会让那些希望减少第三方访问他们脸书信息的人感到兴奋——但这是你能够让扩展发挥其魔力的唯一方式。

从那里，拉起脸书。然后去倒一杯美味的咖啡或放一些舒缓的音乐，因为你将要移除(或隐藏)你的许多数字生活。Social Book Post Manager 就像它的名字所暗示的那样:它通过你的活动 feed 找到与你提供的特定关键字匹配的内容，然后允许你删除或私有化这些帖子。

如果你每年都在脸书上非常活跃，或者如果你过滤掉特别删除或隐藏关于你仍然讨厌的前任的内容，社交图书发布管理器会花一点时间工作，但这比试图手动清理你的活动订阅要快得多，即使你必须多次运行该扩展以确保你得到了所有内容。

(如果你只是打算全部删除你的脸书账户，你可能不需要使用社交图书发布管理器来手动删除你所有的旧内容——只需删除你的脸书账户，然后就到此为止。)
