# 使用 Bechdel 测试打破你的网飞常规

> 原文：<https://lifehacker.com/use-the-bechdel-test-to-break-out-of-your-netflix-rut-1828090189>

全能的推荐算法以神秘的方式工作，而且不总是最好的。如果你的网飞主页已经变成了一个回音室，一点点角度的改变可以帮助你找到你新喜欢的电影。

大多数人可能都熟悉贝克德尔测验[，这是对 1985 年艾莉森·贝克德尔的漫画《T4》](https://lifehacker.com/the-bechdel-test-and-other-media-representation-tests-1819324045)的诠释很简单:如果一部电影中有两个女人在谈论除了男人以外的事情，这部电影就通过了。尽管在过去的 33 年里增加了更多的规定，比如女性应该有自己的名字，但基本要点还是一样的。它设置了令人难以置信的低门槛，但大量主流电影仍然无法清除它——这使得 Bechdel 测试成为一种寻找你可能错过的电影的好方法。

网飞最近在推特上发布了一系列通过贝克德尔测试的图书 ，这是一个很好的起点。每个人都有适合自己的东西:情感时代剧(*、*卡罗尔*)、标志性的合奏喜剧(*湿热的美国夏天*)、令人愉快的 90 年代浪漫喜剧(*滑动门*)和一部强大的优秀星球大战(*最后的绝地*)。我认为值得一提的其他网飞产品包括*橘子*、*影子舞者*、*明显的孩子*和汉娜·盖斯比的*纳内特*特辑。出于某种原因，这些作品大多没有得到广泛发行或丰厚的营销预算，这使得它们更难找到有欣赏力的观众。*

*我并不是说一部电影必须通过贝克德尔测试才是好的、伟大的，甚至是值得你花费时间的；毕竟，大多数伟大的电影和电视节目都失败了。这只是挑战电影和电视中占主导地位的男性叙事的一种有用但不完美的方式。下次你想找一部周五晚上上映的电影，试试通过贝克德尔测试的电影——你可能会爱上它。*
