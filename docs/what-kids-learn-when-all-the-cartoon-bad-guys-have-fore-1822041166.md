# 当所有的卡通坏蛋都有外国口音时，孩子们能学到什么

> 原文：<https://lifehacker.com/what-kids-learn-when-all-the-cartoon-bad-guys-have-fore-1822041166>

经过去年的网络大混乱之后，证实了*狮子王* [的刀疤和木法沙确实是兄弟](http://time.com/4920182/the-lion-king-mufasa-scar-brothers/) 。我很高兴事情解决了。现在唯一令人挠头的是:为什么 Scar 有英国口音，而周围没有其他狮子有英国口音？他是在英国狮子寄宿学校受的教育吗？看了一大堆英国猫电影？最有可能的原因是这样的:电影人经常用外国口音和不标准的方言给“坏”角色配音。

Watch

伊莎贝尔·法塔尔(Isabel Fattal)在《大西洋月刊》(The Atlantic)中探讨了这一现象，她写了一篇分析文章 [“为什么卡通人物说话会带外国口音？”](https://www.theatlantic.com/education/archive/2018/01/why-do-cartoon-villains-speak-in-foreign-accents/549527/) 刀疤、*菲尼亚斯和费伯中的汉斯·杜芬舒斯博士、*阿拉丁中的贾法尔、迪士尼*罗宾汉*中的邪恶国王都是她所说的“坏人”中的少数例子。这是我一直注意到的事情之一，但从未给予太多关注。Fattal 认为父母应该。不管我们是否知道，在每一个电影制作决策中，文化偏见都可能会溜进来。孩子们会发现它们。

社会语言学家卡尔文·吉德尼 [和塔夫茨大学的朱莉·多布罗教授一起研究了儿童娱乐动画中的](http://www.jstor.org/stable/1049446?seq=1#page_scan_tab_contents) 语言模式，他向*大西洋*解释说，虽然电影中的外国口音不同，但共同点是“像我们”和“不像我们”的二元区别。吉德尼告诉法塔尔，在《狮子王》中，“我认为有必要从听起来像英国人的邪恶狮子，加上听起来像非洲裔美国人和拉丁美洲人的鬣狗那里‘夺回丛林’，这真的很令人不安。"

法塔尔写道:

> 从《刀疤》到《阿拉丁》中的贾法尔，研究发现英国口音是恶棍最常用的外国口音。德国和斯拉夫口音也是常见的反派声音。恶棍的追随者或助手经常说与低社会经济地位相关的方言，包括东欧工人阶级的方言或美国地区的方言，如“意大利裔美国黑帮”(就像《T2》中的 Claude 在《星球大战 T3》中说“tuh-raining”而不是“training”。)在被研究的样本中，似乎没有一个反派说的是标准的美式英语；当他们用美国口音说话时，总是用与低社会经济地位相关的地方方言。

Fattal 解释说，由于“电视是儿童文化信息的主要来源”(即许多人看了大量的电视)，外国口音与恶棍的关联“可能会对美国教育儿童参与多样性的方式产生令人担忧的影响。”吉德尼和多布罗正在做进一步的研究，以检验究竟是如何做到的。

在此之前，父母可以意识到这些模式的存在，并告诉他们的孩子。一起看表演，问问题。以下是一些让屏幕时间对孩子有教育意义的方法，给他们一个探索所见所闻之外的机会。
