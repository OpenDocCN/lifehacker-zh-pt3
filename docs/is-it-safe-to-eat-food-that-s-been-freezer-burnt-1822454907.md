# 吃冷冻过的食物安全吗？

现在，在你冰箱的后面，有一袋你两年前打开的豌豆，或者是上次打折时匆忙包好的鸡胸肉。霜模糊了，粘在一起，失去了味道，那些传递过来的包裹正在死亡，就像无数被遗忘的食物在冰箱里慢慢燃烧一样。

冰柜烫伤的祸害是由 [两种效应](http://onlinelibrary.wiley.com/doi/10.1111/j.1541-4329.2009.00072.x/pdf) 造成的:脱水和氧化。前者——水分流失——使冷冻食品变干，失去味道，质地变得坚韧。后者是一种化学反应，其中氧分子与其他分子结合形成新的化合物，导致不可逆的颜色和味道变化。让我们依次来看两者。

冷冻室中的脱水主要是由升华引起的，升华是固体直接蒸发成气体的过程。为了了解这是如何工作的，考虑一下冷冻牛排表面的冰晶。看着牛排，那些水分子看到了一个富含水分的环境，在那里它们可以与其他水分子结合形成固体。然而，看向寒冷的空气，这些分子看到的是一个缺乏水分的环境，在那里没有东西可抓。虽然水*想让*附着在牛排上，但偶尔会有一个分子自由摆动，然后一去不复返。随着这些分子的离开，表面变得越来越粗糙，暴露在空气中的表面积越来越大，加速了损失。如果有足够的时间，即使被困在牛排深处的水也会找到出路——带着味道出去。

另一方面，氧化有点像这个过程的逆过程。冷冻室的空气中充满了四处流动的氧分子，而牛排的表面却是空的，因此当氧分子与食物碰撞时，它们会被捕获。例如，牛排可爱的红色变成了棕色，因为肌红蛋白变成了它的氧化表亲偏肌红蛋白，丰富的脂肪氧化并变形为“平淡”或变味的版本。

冷冻食品吃起来是安全的，但是它们已经失去了活力，而且也没有什么可以修复的了。你可以修剪受影响的区域，冲洗掉坚硬的冰屑，但从根本上说，脱水和氧化的化学变化是不可逆的。相反，你如何从一开始就避免冷冻烧伤？

最重要的因素是确保食物是妥善包装的。不要让表面暴露在外，尽量减少食物与空气的接触。对于肉类，用保鲜膜或蜡状冷冻纸紧紧包裹一次，然后再用铝箔包裹一次。(保鲜膜本身并不密封，杂货店陈列柜中用来包装肉类的大多数塑料收缩膜也不密封。)液体应该放在有紧盖的塑料罐里，蔬菜可以放在可密封的冷冻袋里。只要有可能，尽可能多地去除空气——挤出袋子，紧紧包裹，避免罐子里有额外的顶部空间——或者使用真空密封器来完成这项工作。

然而，也许最好的建议是，在冰箱声称它是 之前吃掉你的食物 [。最好的包装工作和最高端的真空封口机只能推迟不可避免的事情——你所有的冷冻食品总有一天会被烧毁。](https://www.foodsafety.gov/keep/charts/storagetimes.html)
