# 这些航空公司提供免费的机上信息服务

> 原文：<https://lifehacker.com/these-airlines-offer-free-in-flight-messaging-1829421448>

在过去的几年里，航空公司之间的竞争给乘客带来了一些实实在在的好处。国内航班上的免费餐(或者至少是升级的小吃选择)开始卷土重来，现在许多航空公司也开始在航班上提供免费信息。



它不如免费 WiFi 好，但免费信息可以让你在空中使用 Facebook Messenger、WhatsApp 和 iMessage 等应用程序与朋友联系。它不止一次地帮助我安排了一次接送，让一个朋友知道我落后(或领先)，了，并在总体上与下面的世界保持联系，而无需为此支付大量现金。

不幸的是，并不是所有的航空公司都提供免费信息。

本周，西南航空开始在部分航班上免费提供机上短信服务。不是每个航班都有，但数量不少。

其他提供这种选择的航空公司有:阿拉斯加、联合航空和达美航空。

像所有事情一样，你的里程可能会有所不同，尤其是取决于你搭乘的飞机的年龄。尽管如此，如果你将来要驾驶其中一架飞机，看看你是否能免费连接还是值得的。
