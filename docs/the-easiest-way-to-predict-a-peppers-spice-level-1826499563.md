# 预测辣椒辣度的最简单方法

> 原文：<https://lifehacker.com/the-easiest-way-to-predict-a-peppers-spice-level-1826499563>

有些辣椒你知道会很辣(habaneros)，有些辣椒你知道不会辣(bell)，但也有些辣椒可能是温和的、辣的，或者“请把我的舌头从嘴里拿出来，放在一杯冰牛奶里。”幸运的是，有一种简单的方法可以预测辛辣的结果，而且它一直就在我们的眼皮底下。



正如贾斯汀·查普尔在《T2》一书中的 [这一集里解释的那样，要估计一个辣椒的辣度，你所要做的就是闻一闻它:](http://www.cookerybythebook.com/home/2018/5/31/96-just-cook-it)

> 实际上，我是从我在《纽约时报》工作的朋友梅丽莎·克拉克那里学到这个技巧的。她说，你要做的是把墨西哥胡椒或辣椒切成两半，然后闻一下。如果它闻起来像甜椒，那么它就更甜。但是如果它闻起来真的很辣，那么你知道这将是一个非常辣的辣椒，你可能应该在添加更多之前先少用一些。她教我的，我就想，为什么我从来不知道？这真是个绝妙的把戏。

像贾斯汀一样，我也有点失望，我不知道。这和我脸上的鼻子一样明显，我有一个大鼻子。
