# 如何赢得弹球比赛

弹球作为一种赌博游戏曾经被禁止。纽约市警察局长四处奔走 [敲打弹球机为](https://www.history.com/news/that-time-america-outlawed-pinball) 拍照。如果你以业余方式玩，只是想防止球从板上掉下来，那么你可能还会认为弹球是一种随机的游戏，玩几下就变得无聊了。但是如果你按照专业人士的方式来玩——即使你真的不擅长——你会得到更多的乐趣。

Watch

在上面的视频中，Vox 解释了弹球游戏的各个部分，以及如何发现隐藏在所有铃铛和哨子中的目标。遵循以下原则:

1.  在你玩之前检查一下棋盘。
2.  捕捉并瞄准球。

3.  将球指向闪烁的灯。
4.  观察物镜的数字显示。
5.  阅读规则卡。
6.  去打多球。

一旦你注意了，你会发现弹球游戏是一组迷你游戏，它们凝聚成一条弧线，几乎是一个情节。沿着那条弧线前进比仅仅让球保持运动状态要令人满意得多。

如果你不太擅长玩，避开四分之一游戏机，找一个弹球博物馆或其他所有你能玩的街机。当你不必担心不断地把钱喂给机器时，弹球会有趣得多。此外，你还可以四处寻找一款适合你的游戏。几十年来，弹球发生了很大的变化；你可能更喜欢旧的机械机器，或者内置高清电脑屏幕的新机器，如 [*《霍比特人》*](http://store.jerseyjackpinball.com/Games/The-Hobbit/The-Hobbit-Pinball-Limited-Edition_2.html) 。弹球:不都一样！
