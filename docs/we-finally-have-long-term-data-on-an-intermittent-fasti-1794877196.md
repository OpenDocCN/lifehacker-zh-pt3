# 我们终于有了间歇性禁食饮食的长期数据

> 原文:[https://life hacker . com/we-finally-have-long-term-data-on-an-intermittent-fasti-1794877196](https://lifehacker.com/we-finally-have-long-term-data-on-an-intermittent-fasti-1794877196)

如果你对 [、5:2 饮食](https://lifehacker.com/starving-yourself-two-days-a-week-is-actually-not-a-bad-1793788417) 或其他形式的 [间歇禁食](http://vitals.lifehacker.com/how-to-free-yourself-from-food-cravings-with-intermitte-1702108722) 感兴趣，你可能想知道这些饮食是否长期有效。直到最近，我们还没有太多的数据。现在，我们有一点:每隔一天禁食证明交付相同的结果作为定期节食。

这项名为 [的研究发表在《美国医学会杂志内科学](http://jamanetwork.com/journals/jamainternalmedicine/fullarticle/2623528) 上，研究跟踪了人们六个月的减肥尝试和六个月的维持饮食。在最初的六个月里，100 名受试者中有三分之一可以想吃什么就吃什么；三分之一的人一天吃三顿饭，占他们热量需求的 75%(所以，如果他们正常情况下吃 2000 卡路里，一天 1500 卡路里)；禁食组在 25%(500 卡路里)的一天和 125%(2500 卡路里)的一天之间交替。

最后，两组都保持了相同的体重(仅 5-6%，对于一个 200 磅的人来说是 10-12 磅)，并且在血压、心率、胆固醇、胰岛素抵抗、空腹血糖等方面都有相似的数据。

群体之间最大的区别？辍学率。禁食组失去了 34 人中的 13 人(38%)，其中 5 人说他们戒烟是因为他们讨厌这种饮食。坚持节食的那一组只有 29%的成员离开，而且没有人把节食作为原因。对照组失去了 26%的人。请记住，这些人都必须与研究人员保持一年的联系，退出的人数包括那些刚刚昏倒的人。上面的平均值，比如减掉的 10-12 磅，*包括了*辍学的人。因此，这意味着减肥可能对坚持减肥的间歇性禁食者更有效。

这些饮食最终比预想的更加相似。人们最终吃的不仅仅是提供的食物，他们在斋戒日吃得太多，在盛宴日吃得太少。这是人们很难坚持这种饮食的另一种方式。

那么这项研究证明间歇性禁食没什么特别的吗？是也不是。受试者是“新陈代谢健康”的肥胖女性，而 IF 的支持者经常说它的价值在于修复受损的新陈代谢。他们吃的食物相当标准，碳水化合物含量高:55%碳水化合物，30%脂肪，15%蛋白质。许多间歇性禁食者将该方案与低碳水化合物食物相结合，依靠蛋白质、脂肪和纤维提供大部分热量。

最后，这只是禁食的一种。5:2 饮食让你在禁食之间有更多的时间。另一种常见的禁食方式是每天 18 小时不吃东西:基本上，不吃早餐，从昨天的晚餐到今天的午餐不吃任何东西。我们不知道这些其他形式是否会比隔日禁食更好或更差，但你可以随时尝试看看。

底线是，间歇性禁食不足以击败传统节食。但是值得一试，如果你认为这对你有用的话。
