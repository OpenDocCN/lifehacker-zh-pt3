# 艾滋病毒和人乳头瘤病毒的区别

> 原文：<https://lifehacker.com/the-difference-between-hiv-and-hpv-1826141768>

我们的总统显然不知道人乳头瘤病毒 [和艾滋病](https://www.nbcnews.com/politics/donald-trump/bill-gates-says-trump-asked-difference-between-hiv-hpv-n875286) 的区别，但我不在这里作出判断。你可能记得在性教育课上，如果你认为你有一个愚蠢的问题，你应该问，因为你可能不是唯一一个。



*   [**HIV**](https://www.cdc.gov/hiv/default.html) **，人类免疫缺陷病毒**，因是导致艾滋病的病毒而闻名。它是通过性传播的，会损害你的免疫系统，尽管有药物可以控制这种病毒。艾滋病毒影响了全球数百万人，包括 4%的非洲人。(比尔·盖茨向我们的总统解释了这些病毒之间的区别，他管理着一个基金会，该基金会在撒哈拉以南非洲地区提供艾滋病诊断、预防和治疗。根据疾病控制中心的数据，110 万美国人感染了艾滋病病毒，但是七分之一的人不知道。幸运的是， [艾滋病病毒检测由 ACA 合规保险计划免费承保。](https://www.hiv.gov/federal-response/policies-issues/the-affordable-care-act-and-hiv-aids)
*   [](https://www.cdc.gov/std/hpv/stdfact-hpv.htm)****是人类乳头瘤病毒**，因引发宫颈癌而闻名。如果你有子宫颈，并且每隔几年做一次巴氏涂片，你的妇科医生正在寻找这种病毒感染的迹象。人乳头瘤病毒病毒也可以通过性传播，并且如此普遍，以至于我们大多数人都在某个时候感染了它(并且通常不知道)。因为它偶尔会导致癌症，所以每个人在性行为活跃之前接种疫苗是有意义的。该疫苗推荐给所有性别的青春期前儿童，也可以给年轻人接种。**

**这两种疾病都值得了解，如果不治疗，这两种疾病都可能是毁灭性的。但它们是非常非常不同的东西。**
