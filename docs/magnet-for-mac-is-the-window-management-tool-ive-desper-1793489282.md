# Mac 的磁铁是我迫切需要的窗口管理工具

> 原文：<https://lifehacker.com/magnet-for-mac-is-the-window-management-tool-ive-desper-1793489282>

这些年来，我们已经在 Mac 上 [看到了几个不同的工具](http://lifehacker.com/hyperdock-brings-window-previews-other-dock-enhancemen-5649024)[Windows 风格的窗口管理](http://lifehacker.com/snap-any-window-to-half-the-screen-size-in-windows-7-5077728) ，但是没有一个适合我的工作流程。 [Magnet](http://magnet.crowdcafe.com/) 是一个已经存在了一段时间的应用程序，但最近的几次更新终于使它成为我需要的应用程序。

Watch

一眼看去， [磁铁](http://georiot.co/AhK) (售价 4.99 美元，但目前正以 99 英镑出售)就像任何窗口捕捉工具一样。您可以整齐地并排组织窗口，以全屏幕、四分之一、三分之一或任何您想要的组合。您可以通过拖动窗口、设置键盘快捷键或使用菜单栏来管理窗口。这对于像我这样的人来说很有帮助，我通常会打开几十个大小不一的窗口，它们以一种杂乱无章的方式堆叠在一起，这让我在应用切换器上花费了比我想要的更多的时间。如果我真的费心去手动调整一个窗口的大小，那通常只是为了让它大到足以从另一堆应用中跳出。

让 Magnet 从其他选择中脱颖而出的是大量的抛光。它不会被多显示器弄糊涂(事实上支持多达六个外部显示器)，并且您可以定制键盘快捷键来满足您的需求。它还支持窗口区域的任意组合。例如，你可以把一个窗口塞在右上角，另一个塞在右下角，然后展开一个窗口占据另一半屏幕。Magnet 还增加了左/中/右三分之一的选项，这是我真正期待的功能。

磁铁特别好的一点是，它不会强迫你做任何特定的动作。无论你是键盘快捷键型的人，菜单型的人，还是鼠标型的人，你都可以成为 Magnet 的用户。让我们来看看所有这些不同的选项是如何工作的。

### 通过拖动来整理窗口

如果你喜欢用鼠标在 Mac 上导航，Magnet 有一个系统，你可以激活屏幕的不同角落来锁定一个窗口:

*   将一个窗口移动到四个角中的任何一个，它将在那个角缩小到四分之一大小。

*   向左或向右移动一个窗口，它将垂直占据半个屏幕。
*   如果你移动一个窗口到顶部或底部，右边或左边，它会水平扩展到屏幕的一半。
*   将窗口移动到中上边缘以全屏显示。
*   您可以将窗口移动到屏幕底部的三个部分，将它分成三部分。

拖放窗口需要一点时间来适应，但是一旦你掌握了它，它就足够直观了。它实现得足够好，感觉像是 macOS 中的一个标准功能。

### 用键盘快捷键组织窗口

比拖动窗口更好的是键盘快捷键。您可以根据需要定制这些选项，但是默认选项很容易适应:

*   **Ctrl+Alt+D/F/G** 将窗口一分为三。
*   **Ctrl+Alt+E/T** 将一个窗口捕捉成三分之二。
*   **Ctl+Alt+U/I/J/K** 将一个窗口分成四等分。
*   **Ctrl+Alt+右箭头/左箭头**垂直对折。
*   **Ctrl+Alt+上箭头/下箭头**水平对折。

一旦你掌握了键盘快捷键，你就可以轻松地管理窗口了。

### 从菜单栏组织窗口

最后，如果你似乎不记得键盘快捷键或拖动位置，你可以点击菜单栏上的一些东西。正如你所期望的，你可以通过点击菜单栏中相应的选项来移动窗口。

我一直在寻找像 Magnet 这样的窗口管理工具，但总是被大多数选项只支持将屏幕或象限减半这一事实所推迟。更糟糕的是，许多其他的实用程序感觉像被黑了一样，而 Magnet 感觉更像是 macOS 本身的一部分。

一般来说，我会在左边并排打开两个 Chrome 窗口，在右边打开第三个窗口，用于各种聊天应用。如果我在制作视频，将 Screenflow 放在三分之二的窗口中，同时在另外三分之一的窗口中关注与工作相关的内容也是不错的。这里有数不清的选项，所以你需要做一点实验来找到最适合你的，但是最终清理你的屏幕的最小努力是值得的。还有许多其他类似的选项，包括[【BetterSnapTool】](https://www.boastr.net/)(免费)、 [【奇观】](https://www.spectacleapp.com/) (免费)、 [和 HyperDock](http://go.redirectingat.com/?id=33330X911647&site=lifehacker.com&xs=1&isjs=1&url=https%3A%2F%2Fbahoom.com%2Fhyperdock%2F&xguid=dfd58d23c9bf19548887a9d5da079381&xuuid=d36fefecca02a6d86b87e2e5172f293e&xsessid=e21f3dfcd59e5e186c858ff846e63fb7&xcreo=0&xed=0&sref=http%3A%2F%2Flifehacker.com%2Fmagnet-for-mac-is-the-window-management-tool-ive-desper-1793489282&pref=http%3A%2F%2Flifehacker.com%2F&xtz=420)(9.95 美元)，它们都令人钦佩地完成了自己的工作。磁铁是最终吸引我的那个。
