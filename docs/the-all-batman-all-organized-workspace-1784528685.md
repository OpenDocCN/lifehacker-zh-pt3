# 全蝙蝠侠，全组织工作空间

> 原文：<https://lifehacker.com/the-all-batman-all-organized-workspace-1784528685>

在过去的几周里，我们一直在强调简单、干净的工作空间，这一个也不例外。当然，有空间展开和工作，但实际的计算机设置本身是很好的，最小化，并向我们最喜欢的专营权之一致敬。看看海报和这些额外的照片。



Redditor [crj3012](https://www.reddit.com/user/crj3012) 将此提交给了战斗站 subreddit，但我们真正挖掘的是这里的齿轮。他是一个机械键盘的收藏家(正如你将在稍后的照片中看到的)，这在我们的书中已经值得很多加分(他把所有的权利都给了 [机械键盘 subreddit](https://www.reddit.com/r/MechanicalKeyboards) ，他是 [机械市场 subreddit](https://www.reddit.com/r/mechmarket) 的卖家)，但那些装裱和悬挂的经典蝙蝠侠漫画书封面艺术真的达成了交易。这是工作区的另一张照片，突出了这些海报:

但既然我们是为了设备而来，这里就来拍一张桌面本身的照片，包括外壳、绿色背光和 crj3012 用作日常驱动程序的键盘:

他没有在 reddit 的帖子中具体介绍这款装备，但他提到这款装备是一款[Corsair Carbide Air 240](https://www.amazon.com/dp/B00LA6WXEO/ref=twister_B00M1AL2EU?_encoding=UTF8&asc_campaign=InlineText&asc_refurl=https://lifehacker.com/the-all-batman-all-organized-workspace-1784528685&asc_source=&psc=1&tag=kinjalifehackerlink-20)，下面链接中的一些其他照片透露了一些其他规格。至于那些键盘，看看这张照片:

他指出:

> 我是一个键盘爱好者。每个都有不同的布局和开关，所以感觉都不一样。当我登机时，通常会在它们之间循环。我也带一个去上班。
> 
> 我对漫画也有同感。不确定他们的最终位置。

明白了吗？

无论如何，创造性地使用电缆存储的命令挂钩和专用于械甲怪的书架是一个很好的触摸，正如他办公室的其他触摸一样，如咆哮者瓶和带框的精酿啤酒标签系列和书架顶部所有的蝙蝠侠黑胶。点击下面的链接查看完整的图片库，并前往 reddit 主题，这一切都是从这里开始的。
