# 让哈里王子教你做超级香脆培根

> 原文：<https://lifehacker.com/let-prince-harry-teach-you-to-make-super-crispy-bacon-1820935673>

好像哈里王子这一周做得还不够，用好消息淹没我们的时间表，把梅根汗·马克尔从“谁？”对于现实生活中的黑公主(我知道她会成为公爵夫人，不要@我)，他也给我们带来了一个制作超脆培根的技巧。

Watch

殿下显然是在迪士尼世界学到了这个诀窍——他对美国之美的热爱根深蒂固——当他只有八岁时，回到白金汉宫，坚持让皇家厨师用同样的方式准备他的培根。

前皇家厨师麦蒂告诉 [美食&](http://www.foodandwine.com/news/prince-harry-favorite-foods-recipes):

> 哈里王子对我说，“我们在美国的迪斯尼世界吃过它，它非常脆，你可以把它咬断。你必须先把它烤熟，然后你必须把它放在一些纸巾里，放在微波炉里加热一分钟。”我想，是的，好的，谢谢你，你这个小鬼，教我如何烹饪。但是他一走，我就试了一下，得到了这种超脆的培根。

从那以后，麦迪就一直这样做培根。

虽然用微波炉加热培根并不是什么新鲜事，但哈里王子先烤培根的技巧意味着大部分培根脂肪在进入微波炉之前就被烹饪掉了，这样你就可以省去一大堆麻烦(和一大堆纸巾)。
