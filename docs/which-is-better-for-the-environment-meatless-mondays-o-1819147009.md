# 哪个对环境更有利:无肉星期一还是#无肉十月？

> 原文:[https://life hacker . com/哪个对环境更有利-无肉星期一-o-1819147009](https://lifehacker.com/which-is-better-for-the-environment-meatless-mondays-o-1819147009)

作为一个十年的素食主义者，我支持任何可能减少美国人对动物产品依赖的倡议。虽然我的选择主要是出于动物福利的立场，但我也认识到牲畜，尤其是红肉的影响正在对我们的环境造成损害。

Watch

《大众科学》主编乔·布朗在本周的一篇文章中写道 “家畜排放了 12%的温室气体，而红肉是罪魁祸首:尤其是奶牛，它们将草转化为人类食物的效率低得可怜——它们饲料的 1%转化为我们可以消耗的热量，这还不到一只鸡的十分之一——所以饲养它们需要大量的水和能量。我们吃得越多，农民就越需要繁殖它们。”

(旁注:因为我们喜欢传递有趣的信息，所以值得分享的是，虽然普遍认为是牛放屁导致甲烷水平增加破坏了我们的大气，但 [牛打嗝](http://metro.co.uk/2017/03/25/cow-burps-rather-than-farts-are-destroying-the-earths-atmosphere-6531638/) 实际上在这个问题上扮演了一个主要的，可能是主要的角色。还有他们处理粪便的方式。)

为了鼓励减少红肉的摄入量，从普通美国人一年吃 54 磅牛肉的开始，PopSci 的人发起了一项新的运动#NoRedOctober。

当然，另一位 PopSci 作家 Sophie Bushwick 在今天的帖子中认为，对于食肉的美国人来说，已经有了更好的挑战: [无肉星期一](http://www.meatlessmonday.com/) 。她的数据显示，美国人平均每年摄入 77 磅牛肉，每周一(非节假日)不吃红肉一个月与选择不吃肉的区别是巨大的:“不管你在#NoRedOctober 期间避免食用多少头牛或多少磅牛肉，”她写道，“选择周一不吃肉，你会多节省 61%的动物或肉。”

当然，我们都是为了更大的影响(因为 10 月也是 [素食宣传月](https://worldvegetarianday.navs-online.org/) ，我可以把这个 [“无肉不吃”](https://worldvegetarianday.navs-online.org/veg-pledge/enter-the-contest/) 的 1000 美元现金奖励竞赛抛到选项里吗？).但这里的底线是:当谈到改变习惯时，首先是你，而不是统计数据。无论你是被每天、每周还是每月的过程所吸引(或者是一次性挑战)，你真正坚持的那个过程将会产生最大的影响。
