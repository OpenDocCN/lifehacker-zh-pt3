# 用这个 Chrome 扩展快速删除脸书多年的帖子

> 原文:[https://life hacker . com/quick-delete-years-of-Facebook-posts-with the-chrome-1828148008](https://lifehacker.com/quickly-delete-years-of-facebook-posts-with-this-chrome-1828148008)

当你在大学的时候，上传你做一个史诗般的小桶倒立的视频很酷。既然你已经从法学院毕业，正在寻找一家公司来加入。

明智的做法是回到过去，将所有这些罪证照片、视频和帖子保密。更聪明的举动？完全删除它们。虽然当你竞选国会议员时，它们肯定还会再次出现在网络空间，但删除帖子至少会保护你不被勤奋的人力资源代表发现。

一个接一个的回去删帖子会很讨厌。我们最近在 Lifehacker 总部偶然发现了一个 Chrome 扩展 ，它可以为你处理脏活累活。

这款名为 [社交书籍帖子管理器](https://chrome.google.com/webstore/detail/social-book-post-manager/ljfidlkcmdmmibngdfikhffffdmphjae) 的扩展可以让你删除特定日期之前你在社交网络上的所有帖子，或者通过 word search 指定你想要删除的帖子类型。例如，你可以搜索包含你前任名字或“杂草”的帖子

该工具将显示所有符合您特定标准的帖子，然后您可以选择保留哪些帖子，删除哪些帖子(或将其设为私有)。

一旦你删除了那些罪证照片，小心你将来在网上发布的内容总是一个好建议。那个醉醺醺的帖子，不管多搞笑，大概永远是个恐怖的想法。
