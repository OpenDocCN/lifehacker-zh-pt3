# 接下来我们应该做什么？

各位网友大家好，欢迎来到全新的 [*选题会，会吗？*](http://lifehacker.com/tag/will-it-sous-vide) ，列在那里我用我的沉浸循环器做任何你想让我做的事。

龙虾尾绝对是美味的成功，但现在我们必须转向另一种食物。我心中没有一个具体的主题，只是一些随意的想法。

*   **鲜奶油取样器:**我们可以做一大堆蛋羹，把它倒入四个罐子里，给每个罐子加上不同的味道。当然会有火炬。
*   **土豆:**有没有好的苏司维土豆应用？我还没有找到一个让我印象特别深刻的。你们有什么想法吗？
*   蘑菇:我想我们可以在这里得到一个额外的蘑菇状蘑菇，但显然我们必须在锅里完成才能得到那些脆脆的边缘。
*   通过低火慢煮，我们也许可以跳过腌制步骤，仍然可以得到大量的浓缩风味。

很明显，我对你任何原创的真空土豆概念都非常开放，尤其是如果你对真空土豆有好的想法，因为我不相信有这样的想法。
