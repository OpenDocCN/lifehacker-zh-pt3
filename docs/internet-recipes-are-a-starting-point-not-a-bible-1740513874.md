# 互联网食谱是一个起点，而不是一本圣经

> 原文：<https://lifehacker.com/internet-recipes-are-a-starting-point-not-a-bible-1740513874>

学习烹饪通常始于在网上找到一些食谱，然后在厨房里尝试。很好，但不要就此打住。互联网食谱是一个很好的起点，但它们也有局限性。以下是其中的一些，告诉你如何超越它们，在厨房中发挥真正的创造力。



做饭并不容易，尤其是如果你还没有养成习惯的话 。对一些人来说，这很有趣，也很自然。对于其他人来说，T5 需要工作 T7T9】需要时间和精力 T11】。然而，回报是巨大的:你每天都可以以一种有意义的(更不用说美味的)方式发挥创造力，并且你可以掌控自己健康最重要的方面。

但是当你盲目遵循食谱时，你就限制了自己。你可能正在学习基础知识，但你没有学到烹饪中最重要的部分:尝试各种口味，找到你和你的家人都会喜欢的菜肴的终极版本。

### **网络菜谱长期销售不足**

大多数食谱，尤其是来自大型食谱网站的食谱，都是为了迎合大众口味。他们想尽可能不惹人讨厌。这并不意味着他们总是坏的，但是他们从来就不是真正的好的 T2。你很少会想到“哇，我真的能尝出 X”或者“这道菜里的 Y 真的出来了。”它们是为最普通、中庸的味蕾而设计的，除非你把它们混合起来，否则你会得到一道不折不扣的普通、中庸的菜。

正因为如此，他们几乎是单方面的淡季。这意味着如果你不做点什么的话，你的食物也会被吃掉。当然，很多食谱自豪地宣称它们的“味道好”、“草药味”和“辣味”，但我发现很少有*真的*符合宣传。

解决方法很简单:给食物调味。说真的，就这么简单。有一个古老的厨师规则，如果你在给一道菜调味，加入你认为需要的盐。然后再补充一点。

每当你看到一个本该供八个人食用的食谱，并要求“两瓣大蒜”，那都是废话，你可能应该再加两瓣或三瓣(除非你讨厌大蒜。)也许我只是喜欢味道，但在炖肉中加入“少量辣椒粉”并不会给人留下深刻印象，一茶匙生姜也不会给一个四口之家的炒菜食谱增添任何有意义的味道。(烹饪电视节目也因这类东西而臭名昭著。)

在你真正开始烹饪之前，看看你的配料和它们的数量，问问你自己，“我真的能尝出这个味道吗？如果不是，为什么会在这里？”烹饪的时候也要调味，品尝一下，看看你的口味是如何发展的。上菜前再尝一遍，在饭菜上桌前做些调整。在我刚刚谈到的那些烹饪节目中，你总能看到这一点:一些厨师在紧要关头做些东西，端上来，味道很糟糕。法官或者任何大喊“你端上来之前尝过这个吗？”他们结结巴巴地回答道:“我只有几分钟时间，我没有机会，我本打算……”你知道要点了。别做那种人。

不要误解我，有些地方一点点就能走很长的路。像柑橘汁这样的酸就是一个很好的例子。微妙的成分，如藏红花或温和的香草，需要温和的菜肴才能发光，是另一个原因。像阿多宝中的辣椒或烈性辣酱这样的超辣食材，轻点处理比重手处理更好。再一次，这归结为*制作*食谱和*在你开始烹饪前考虑*食谱之间的区别。

### 食谱让你忽略了自己的口味

当你刚刚开始烹饪时——我指的是刚刚开始——你不知道一点和一点的区别，所以严格按照食谱做没有错。一旦你学会了基础知识，明白了你喜欢什么和不喜欢什么，你应该更怀疑地阅读这些食谱。

当你阅读食谱时，想想你喜欢的口味，并考虑如何将它们融合在一起。例如，如果你想在下一次游戏日派对上尝试自制沙拉，但你讨厌芫荽叶，无论如何不要用它，因为食谱说你应该这样做。问问你自己它扮演什么角色(剧透:它是一种辛辣的药草)，用一种你真正喜欢的药草代替，也许是泰国罗勒或意大利欧芹。你喜欢辛辣的食物吗？你可以在任何食物中加入红辣椒片。如果萨尔萨酱需要半个墨西哥胡椒籽，而你喜欢辣味的，那就把整个都扔进去。 [就这么做](https://www.youtube.com/watch?v=ZXsQAXx_ao0) 。

这只是几个例子，但是如果你认为“食谱是这么说的，所以它一定是对的”，你就在给自己帮倒忙。阅读每一份食谱，问自己“为什么这个在里面？给党带来什么？”一旦你习惯了这样做，你就能把你不喜欢的成分换成更美味的。

然后，问问你自己如何改进它以适合你的口味。你的改变可能并不总是奏效，但尝试并找到奏效的方法比吃一些容易忘记的东西要好，或者更糟:一些让你希望你只是点了外卖的东西。

### 食谱网站完全是一场赌博(但是你可以找到更好的)

很多人从搜索引擎和社交网络上获取食谱，比如 [、Pinterest](https://pinterest.com/) 、[yummy、](http://www.yummly.com/)[all recipes](http://allrecipes.com/)，甚至 [Bing](https://blogs.bing.com/search/2015/10/14/bing-makes-it-easier-to-find-recipes/) 。它们都是很好的来源，尤其是如果你是一个新厨师，或者如果你需要一些厨房灵感。然而，麻烦的是，虽然在那里很容易找到灵感，但你并不总是知道这些食谱是否经过测试并确实有效，或者味道是否与它们附带的美味照片相匹配。

是的，批判性地阅读这些食谱(就像我们上面讨论的)会有帮助，但是你也可以找到更好的来源。你的新闻不是单一来源的，对吧？所以也不要单一来源你的食物。你想要专家的意见和不同的观点。你希望有人替你尝试，这样你就不会浪费时间。(其实这也是 [美试厨](http://americastestkitchen.com/) 所依据的一个前提！)

深入挖掘。探索独立的美食博客。不要忽略那些在 Pinterest 上发布自己的东西的家庭博主:他们通常会链接到自己的食谱，这样你就可以了解他们是如何做每件事的，结果如何，他们最初从哪里获得食谱，以及他们是如何调整它们的。它们是无价的。访问有自己的测试厨房的烹饪杂志网站——几乎所有的网站都有食谱部分(或“测试厨房”部分),在那里他们可以尝试一些东西，并告诉你如何在家里获得很好的效果。

当你开始列出自己的网站和资源时，寻找那些你信任其观点和口味的个人美食博主和厨师。你可能已经知道像阿尔顿·布朗、j .肯吉·洛佩兹-Alt 和林恩·罗塞托·卡斯帕这样的名字。想想你喜欢的烹饪节目和他们背后的人。

不要害怕在厨房里玩耍。不要把任何人的一句话当成食谱，或者你的食物应该是什么味道。你在为你和你所喂养的人做饭——他们的口味和需求比菜谱上说的更重要；进去做一些你知道你会喜欢的东西，而不是根据别人的直觉去掷骰子。
